# 데이터 수집
## 데이터 마이닝이란 무엇입니까?

데이터 마이닝은 기업에서 원시 데이터를 유용한 정보로 변환하는 데 사용하는 프로세스입니다. 소프트웨어를 사용하여 대량의 데이터에서 패턴을 찾음으로써 기업은 고객에 대해 더 많이 알게 되어 보다 효과적인 마케팅 전략을 개발하고 판매를 늘리며 비용을 절감할 수 있습니다. 데이터 마이닝은 [효과적인 데이터 수집](/big-data),. [창고 보관](/warehousing) 및 컴퓨터 처리에 달려 있습니다.

## 데이터 마이닝 작동 방식

데이터 마이닝에는 의미 있는 패턴과 추세를 수집하기 위해 큰 정보 블록을 탐색하고 분석하는 작업이 포함됩니다. 데이터베이스 마케팅, 신용 위험 관리, 사기 탐지, 스팸 이메일 필터링과 같은 다양한 방법으로 사용되거나 사용자의 감정이나 의견을 식별하는 데에도 사용할 수 있습니다.

데이터 마이닝 프로세스는 5단계로 나뉩니다. 첫째, 조직은 데이터를 수집하여 데이터 웨어하우스에 로드합니다. 다음으로 사내 서버나 클라우드에서 데이터를 저장하고 관리합니다. 비즈니스 분석가, 관리 팀 및 정보 기술 전문가는 데이터에 액세스하고 데이터를 구성할 방법을 결정합니다. 그런 다음 응용 프로그램 소프트웨어는 사용자의 결과를 기반으로 데이터를 정렬하고 마지막으로 최종 사용자는 데이터를 그래프나 표와 같이 공유하기 쉬운 형식으로 표시합니다.

## 데이터 웨어하우징 및 마이닝 소프트웨어

데이터 마이닝 프로그램은 사용자가 요청한 내용을 기반으로 데이터의 관계와 패턴을 분석합니다. 예를 들어, 회사는 데이터 마이닝 소프트웨어를 사용하여 정보 클래스를 생성할 수 있습니다. 예를 들어, 레스토랑에서 데이터 마이닝을 사용하여 특정 스페셜을 제공해야 하는 시기를 결정하려고 한다고 가정해 보겠습니다. 수집한 정보를 살펴보고 고객이 언제 방문하고 무엇을 주문하는지에 따라 클래스를 만듭니다.

다른 경우에 데이터 마이너는 논리적 관계를 기반으로 정보 클러스터를 찾거나 연관 및 순차적 패턴을 보고 소비자 행동의 경향에 대한 결론을 도출합니다.

웨어하우징은 데이터 마이닝의 중요한 측면입니다. 웨어하우징은 회사에서 데이터를 하나의 데이터베이스 또는 프로그램으로 중앙 집중화하는 것입니다. 데이터 웨어하우스를 통해 조직은 특정 사용자가 분석하고 사용할 데이터 세그먼트를 분리할 수 있습니다.

그러나 다른 경우에는 분석가가 원하는 데이터로 시작하여 해당 사양을 기반으로 [데이터 웨어하우스 를 생성할 수 있습니다.](/data-warehousing) 기업 및 기타 주체가 데이터를 구성하는 방식에 관계없이 경영진의 의사 결정 프로세스를 지원하는 데 데이터를 사용합니다.

## 데이터 마이닝 및 소셜 미디어

데이터 마이닝의 가장 수익성 있는 응용 프로그램 중 하나는 소셜 미디어의 응용 프로그램이었습니다. Facebook(Meta 소유), TikTok, Instagram 및 Twitter와 같은 플랫폼은 타겟 마케팅 광고를 보내기 위해 개인 사용자에 대한 데이터를 수집하여 선호도를 추론합니다. 이 데이터는 또한 소비자 제품에 대한 것이든 선거에서 투표할 대상이든 사용자 행동에 영향을 미치고 선호도를 변경하는 데 사용됩니다.

소셜 미디어의 데이터 마이닝은 여러 조사 보고서를 통해 큰 논쟁거리가 되었으며 마이닝 사용자의 데이터가 얼마나 사악한지 보여줍니다.

> Cambridge Analytica 스캔들은 소셜 미디어 회사가 사용자를 희생시키면서 데이터 마이닝을 사용할 수 있는 방법의 대표적인 예입니다.

>

## 데이터 마이닝의 예

식료품점은 데이터 마이닝 기술의 잘 알려진 사용자입니다. 많은 슈퍼마켓은 비회원에게는 제공되지 않는 할인된 가격으로 이용할 수 있는 무료 [로열티 카드 를 고객에게 제공합니다.](/loyalty-program) 이 카드를 사용하면 상점에서 누가 무엇을 언제, 얼마에 구매하는지 쉽게 추적할 수 있습니다. 데이터를 분석한 후 매장은 이 데이터를 사용하여 고객에게 구매 습관을 대상으로 하는 쿠폰을 제공하고 상품을 판매할 때 또는 정가로 판매할 때를 결정할 수 있습니다.

데이터 마이닝은 회사가 특정 가설을 증명하기 위해 전체 샘플 그룹을 대표하지 않는 선택된 정보만 사용하는 경우 우려의 원인이 될 수 있습니다.

> 데이터 마이닝 프로세스는 검색 엔진 기술 및 웹사이트 추천 프로그램을 비롯한 애플리케이션을 구동하는 기계 학습 모델을 구축하는 데 사용됩니다.

>

## 하이라이트

- 데이터 마이닝은 고객이 무엇에 관심이 있거나 구매를 원하는지 파악하는 것부터 사기 탐지 및 스팸 필터링에 이르기까지 기업에서 사용할 수 있습니다.

- 데이터 마이닝 프로그램은 사용자가 요청하거나 제공하는 정보를 기반으로 데이터의 패턴과 연결을 분해합니다.

- 데이터 마이닝은 추세와 패턴을 식별하기 위해 대량의 정보를 분석하는 프로세스입니다.

- 소셜 미디어 회사는 수익을 창출하기 위해 데이터 마이닝 기술을 사용하여 사용자를 상품화합니다.

- 이러한 데이터 마이닝의 사용은 최근 비판을 받고 있습니다. 사용자는 특히 선호도에 영향을 미치는 데 사용되는 경우 개인 정보에서 발생하는 데이터 마이닝을 종종 인식하지 못합니다.

## 자주하는 질문

### 데이터 마이닝은 어떻게 이루어지나요?

데이터 마이닝은 빅 데이터와 기계 학습 및 기타 인공 지능(AI)을 포함한 고급 컴퓨팅 프로세스에 의존합니다. 목표는 구조화되지 않았거나 대규모 데이터 세트에서 추론이나 예측으로 이어질 수 있는 패턴을 찾는 것입니다.

### 데이터 마이닝의 또 다른 용어는 무엇입니까?

데이터 마이닝은 또한 덜 사용되는 용어 지식 발견 데이터 또는 KDD로 진행됩니다.

### 누가 데이터 마이닝을 사용합니까?

데이터 마이닝 응용 프로그램은 금융 부문에서 시장의 패턴을 찾기 위해 잠재적인 보안 위협을 식별하려는 정부에 이르기까지 다양합니다. 기업, 특히 온라인 및 소셜 미디어 회사는 사용자에 대한 데이터 마이닝을 사용하여 특정 사용자 집합을 대상으로 하는 수익성 있는 광고 및 마케팅 캠페인을 만듭니다.
