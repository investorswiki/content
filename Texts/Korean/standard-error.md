# 표준 에러
## 표준 오차란 무엇입니까?

통계의 표준 오차(SE)는 통계적 표본 모집단의 대략적인 표준 편차입니다. 표준 오차는 [표본 분포](/sampling-distribution) 가 모집단을 나타내는 정확도를 표준 편차를 사용하여 측정하는 통계 용어입니다 . 통계에서 표본 평균은 모집단의 실제 평균에서 벗어납니다. 이 편차는 평균의 표준 오차입니다.

## 표준오차 이해하기

"표준 오차"라는 용어 는 평균 또는 중앙값과 같은 다양한 표본 통계 의 [표준 편차 를 나타내는 데 사용됩니다.](/standarddeviation) 예를 들어, "평균의 표준 오차"는 모집단에서 가져온 표본 평균 분포의 표준 편차를 나타냅니다. 표준 오차가 작을수록 표본이 전체 모집단에서 더 많이 대표됩니다.

표준 오차와 표준 편차 사이의 관계는 주어진 표본 크기에 대해 표준 오차를 표본 크기의 제곱근으로 나눈 표준 편차와 같습니다. 표준 오차는 또한 표본 크기에 반비례합니다. 표본 크기가 클수록 통계가 실제 값에 가까워지기 때문에 표준 오차가 작아집니다.

표준 오차는 추론 통계의 일부로 간주됩니다. 데이터 세트 내 평균의 표준 편차를 나타냅니다. 이것은 확률 변수에 대한 변동의 척도로 사용되어 산포에 대한 측정을 제공합니다. 스프레드가 작을수록 데이터 세트가 더 정확합니다.

> 표준오차와 표준편차는 변동성의 척도이고 중심경향 척도는 평균, 중앙값 등을 포함합니다.

>

## 표준 오차에 대한 요구 사항

[모집단이](/representative-sample) [샘플링](/representative-sample) 되면 평균 또는 평균이 일반적으로 계산됩니다. 표준 오차에는 계산된 모집단 평균과 알려진 것으로 간주되거나 정확한 것으로 간주되는 평균 간의 변동이 포함될 수 있습니다. 이는 샘플 수집과 관련된 부수적인 부정확성을 보정하는 데 도움이 됩니다.

여러 표본을 수집하는 경우 각 표본의 평균이 다른 표본과 약간 다를 수 있으므로 변수 간에 분산이 생성됩니다. 이 산포는 데이터 세트 전체의 평균 간의 차이를 설명하는 표준 오차로 가장 자주 측정됩니다.

평균 계산에 관련된 데이터 포인트가 많을수록 표준 오차는 작아지는 경향이 있습니다. 표준 오차가 작을 때 데이터가 실제 평균을 더 잘 대표한다고 합니다. 표준 오차가 큰 경우 데이터에 눈에 띄는 불규칙성이 있을 수 있습니다.

표준 편차는 각 데이터 포인트의 산포를 나타냅니다. 표준 편차는 각 표준 편차 수준에 표시된 데이터 포인트 수를 기반으로 데이터의 유효성을 결정하는 데 사용됩니다. 표준 오차는 평균 내 편차를 분석하여 표본의 정확도 또는 여러 표본의 정확도를 결정하는 방법으로 더 많이 사용됩니다.

## 하이라이트

- 표준 오차는 통계 표본 모집단의 대략적인 표준 편차입니다.

- 표준 오차에는 계산된 모집단 평균과 알려진 것으로 간주되거나 정확한 것으로 간주되는 평균 간의 변동이 포함될 수 있습니다.

- 평균 계산에 관련된 데이터 포인트가 많을수록 표준 오차가 작아지는 경향이 있습니다.
