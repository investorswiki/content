# 소스 코드
소스 코드는 결정론적 지침 및 명령문 목록을 기반으로 프로그램(소프트웨어)이 작동하는 방식을 정의하는 컴퓨터 코드 행의 모음을 나타냅니다. 본질적으로 소스 코드는 원래 작성된 순간의 소프트웨어의 기본 상태입니다.

따라서 소스 코드는 Java, JavaScript, C, Python, PHP, Go, Rusty, Solidity 및 Haskell과 같은 다양한 컴퓨터 프로그래밍 언어로 작성할 수 있는 일련의 명령으로 구성됩니다. 이러한 모든 프로그래밍 언어에는 공통된 특성이 있습니다. 특정 작업을 수행하고 특정 출력을 생성하기 위해 컴퓨터가 따라야 하는 일련의 명령으로 구성됩니다.

생성된 소스 코드는 종종 컴퓨터가 이해하고 실행할 수 있는 기계어로 "번역"되거나 컴파일됩니다. 그러나 어떤 상황에서는 소스 코드가 컴파일 없이 바로 실행됩니다(이러한 컴퓨터 프로그램을 인터프리터라고 함).

컴퓨터 소프트웨어 개발 중 소스 코드 단계는 프로그램이 최종적으로 컴퓨터 기계에 의해 실행되기 전에 프로그래머가 읽고, 주석을 달고, 편집할 수 있는 초기 작업 단계입니다. 소스 코드 라이브러리는 코드 라인에 주석을 포함할 수 있기 때문에 소프트웨어 엔지니어, 컴퓨터 과학자 및 프로그래머의 작업을 훨씬 쉽게 만듭니다(예: 설명적이고 통찰력 있는 주석이 포함된 읽을 수 있는 일반 텍스트).
