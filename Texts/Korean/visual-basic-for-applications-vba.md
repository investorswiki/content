# VBA(Visual Basic for Applications)
VBA(Visual Basic for Applications)는 Microsoft Corporation(NASDAQ: MSFT) 레거시 소프트웨어인 Visual Basic의 일부로, Microsoft가 Windows 운영 체제용 프로그램 작성을 돕기 위해 구축했습니다. Visual Basic for Applications는 Access, Excel, PowerPoint, Publisher, Word 및 Visio와 같은 Microsoft Office(MS Office, Office) 응용 프로그램에서 내부 프로그래밍 언어로 실행됩니다.

VBA를 사용하면 도구 모음, 메뉴, 대화 상자 및 양식과 같은 GUI(그래픽 사용자 인터페이스) 기능을 조작하여 MS Office 호스트 응용 프로그램에서 일반적으로 사용할 수 있는 것 이상으로 사용자 지정할 수 있습니다. VBA는 독립 실행형 프로그램이 아닙니다. VBA를 사용하여 UDF(사용자 정의 함수)를 만들고 Windows [API(응용 프로그래밍 인터페이스 )에 액세스](/application-programming-interface) 하고 특정 컴퓨터 프로세스 및 계산을 자동화할 수 있습니다.

## 응용 프로그램용 Visual Basic에 대한 추가 정보

VBA는 이벤트 기반 도구입니다. 즉, 이를 사용하여 작업 또는 일련의 작업을 시작하도록 컴퓨터에 지시할 수 있습니다. 이렇게 하려면 편집 모듈에 명령을 입력하여 사용자 정의 매크로(매크로 명령의 줄임말)를 만듭니다.

매크로는 본질적으로 특정 컴퓨팅 작업을 수행하는 다른 일련의 문자(해당 출력)를 입력으로 하는 일련의 문자입니다. VBA는 Microsoft Office 2010과 함께 제공되는 Visual Basic 버전이므로 VBA 소프트웨어를 구입할 필요가 없습니다.

> Visual Basic for Applications는 여전히 Microsoft에서 판매 및 지원하는 유일한 VB 6 버전이며 Office 프로그램의 내부 구성 요소로만 사용됩니다.

>

## VBA는 어떻게 사용됩니까?

### 우리 대부분에게

MS Office 응용 프로그램 내에서 Visual Basic for Applications를 통해 사용자는 단순한 워드 프로세싱 및 스프레드시트 작업을 넘어서는 무수한 기능을 수행할 수 있습니다. 일반 사용자의 경우 VBA는 매크로를 통해 빈번한 일상 작업을 덜 반복적으로 수행하는 데 도움이 됩니다.

매크로는 맞춤형 차트 및 보고서 생성, 워드 및 데이터 처리 기능 수행과 같은 거의 모든 작업을 자동화할 수 있습니다. 예를 들어, 한 번의 클릭으로 Excel이 스프레드시트의 일련의 [회계](/accounting) 항목에서 전체 대차대조표를 생성하도록 하는 매크로를 작성할 수 있습니다 .

### 컴퓨터 전문가용

그러나 프로그래머는 큰 코드 조각 복제, 기존 프로그램 기능 병합, 특정 언어 설계와 같은 보다 복잡한 방식으로 매크로를 사용합니다.

### 회사 및 조직용

VBA는 컴퓨터 경계를 넘어 명령이 상호 작용할 수 있도록 하는 COM 인터페이스****라는 기술을 사용하여 Microsoft가 아닌 외부 설정에서도 작동할 수 있습니다. AutoCAD, ArcGIS, CATIA, Corel, raw 및 SolidWorks를 포함하여 많은 회사에서 독점 및 상업용 자체 응용 프로그램 내에서 VBA를 구현했습니다.

[이자율](/interestrate) 및 퇴직까지의 년 수와 같은 기타 요인을 기반으로 [투자 포트폴리오](/portfolio) 에서 100만 달러를 벌기 위해 얼마나 걸릴지 식별하는 것과 같은 고유한 목적에 맞게 Excel을 사용자 지정할 수 있습니다 .

## 금융 시장에서 VBA의 예

### VBA가 금융에서 유비쿼터스인 방법

금융의 핵심은 엄청난 양의 데이터를 조작하는 것입니다. 따라서 VBA는 금융 서비스 부문에 고유합니다. 재무 부서에서 일하는 경우 VBA는 인식 여부에 관계없이 매일 사용하는 응용 프로그램 내에서 실행될 가능성이 높습니다. 해당 분야의 일부 작업에는 VBA에 대한 사전 지식이 필요하고 일부는 그렇지 않습니다.

어느 쪽이든, 재무 분야에서 경력을 쌓고 싶다면 해당 영역의 최신 기술 동향과 일상 활동에서 자동화를 사용하는 방법을 아는 것이 중요합니다. VBA는 사용자에게 직관적이기 때문에 컴퓨터 프로그래밍 지식이 거의 또는 전혀 없는 사람들도 쉽게 배울 수 있습니다.

### 재무 전문가가 VBA를 사용하는 방법

- 매크로를 사용하면 회계사, 상업 은행가, [투자 은행가](/investmentbanker),. 연구 분석가, 영업 사원, 거래자, 포트폴리오 관리자, 사무원 또는 관리자와 같은 금융 전문가가 막대한 양의 데이터를 빠르게 분석하고 조정할 수 있습니다.

- Excel에서 VBA를 사용하여 복잡한 거래, 가격 책정 및 위험 관리 모델을 만들고 유지 관리하고 매출 및 수익을 예측하고 재무 비율을 생성할 수 있습니다.

[투자 시나리오](/scenario_analysis) 를 만들 수 있습니다 .

- VBA를 사용하여 고객 이름 또는 기타 콘텐츠 목록을 생성할 수도 있습니다. 송장, 양식 및 차트 생성 과학 데이터를 분석하고 예산 및 예측을 위한 데이터 표시를 관리합니다.

##하이라이트

- MS Office 응용 프로그램 내의 VBA 기능 독립형 제품이 아닙니다.

- VBA를 사용하면 매크로를 생성하여 반복적인 워드 및 데이터 처리 기능을 자동화하고 사용자 정의 양식, 그래프 및 보고서를 생성할 수 있습니다.

- Visual Basic for Applications는 Microsoft가 개발하고 소유한 컴퓨터 프로그래밍 언어입니다.
