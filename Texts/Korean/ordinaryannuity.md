# 보통연금
## 보통연금이란?

일반 연금은 고정된 기간 동안 연속된 기간이 끝날 때 지급되는 일련의 동일 금액입니다. 일반 연금의 지불은 매주만큼 자주 이루어질 수 있지만 실제로는 일반적으로 월별, 분기별, 반기별 또는 매년 지급됩니다. 일반 연금의 반대는 각 기간이 시작될 때 지불되는 만기 연금입니다. 이 두 가지 일련의 지불은 관련되어 있지만 연금으로 알려진 금융 상품과 동일하지 않습니다.

## 일반 연금의 작동 원리

보통 연금의 예로는 일반적으로 반기마다 발행되는 채권의 이자 지급과 수년간 안정적인 지급 수준을 유지한 주식의 분기 배당금이 있습니다. 일반 연금 의 [현재 가치](/presentvalue) 는 일반적으로 적용되는 이자율에 크게 좌우됩니다.

[화폐의 시간가치](/timevalueofmoney) 때문에 금리가 오르면 일반 연금의 현재 가치가 낮아지고 금리가 떨어지면 현재 가치가 높아집니다. 연금의 가치는 다른 곳에서 돈을 벌 수 있는 수익을 기반으로 하기 때문입니다. 다른 곳에서 더 높은 이율을 받을 수 있다면 해당 연금의 가치는 하락합니다.

## 보통연금의 현재 가치 예시

일반 연금의 현재 가치 공식은 세 가지 변수를 고려합니다. 그것들은 다음과 같습니다:

- PMT = 현금결제 기간

- r = 기간당 이자율

- n = 기간의 총 수

이러한 변수가 주어지면 일반 연금의 현재 가치는 다음과 같습니다.

- **현재 가치** = PMT x ((1 - (1 + r) ^ -n ) / r)

예를 들어, 일반 연금이 5년 동안 연간 $50,000를 지불하고 이자율이 7%인 경우 현재 가치는 다음과 같습니다.

- **현재 가치** = $50,000 x ((1 - (1 + 0.07) ^ -5) / 0.07) = $205,010

> 일반 연금은 만기 연금보다 현재 가치가 낮으며 다른 모든 조건은 동일합니다.

>

## 만기 연금의 현재 가치 예

일반 연금의 경우 투자자는 기간이 끝날 때 지급금을 받습니다. 이는 투자자가 기간 초에 지급금을 받는 [만기](/annuitydue) [연금](/annuitydue) 과 대조됩니다 . 일반적인 예는 세입자가 일반적으로 다음 달에 대해 집주인에게 선불로 지불하는 임대료입니다. 이러한 지불 시기의 차이는 연금의 가치에 영향을 미칩니다. 연체료의 공식은 다음과 같습니다.

- **연금액의 현재 가치** = PMT + PMT x ((1 - (1 + r) ^ -(n-1) / r)

위의 예에서 연금이 만기 연금인 경우 현재 가치는 다음과 같이 계산됩니다.

- **연금액의 현재 가치** = $50,000 + $50,000 x ((1 - (1 + 0.07) ^ -(5-1) / 0.07) = $219,360.

다른 모든 조건이 동일할 때 지불해야 하는 연금은 항상 일반 연금보다 더 가치가 있습니다. 왜냐하면 돈이 더 일찍 수령되기 때문입니다.

##하이라이트

- 보통연금은 월별 또는 분기별과 같이 각 기간 말에 일련의 정기 지급금을 말합니다.

- 일관된 분기별 주식 배당금은 보통 연금의 한 예입니다. 월 임대료는 만기 연금의 예입니다.

- 이에 반해 만기가 도래한 연금은 매 기간 초에 지급합니다.
