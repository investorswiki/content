# 내부수익률(IRR) 규칙
## 내부수익률(IRR) 규칙이란 무엇입니까?

내부 수익률(IRR) 규칙은 프로젝트 또는 투자의 IRR이 최소 [요구 수익률(](/requiredrateofreturn) [허들 율](/hurdlerate) 이라고도 함 )보다 크면 프로젝트 또는 투자를 추구해야 한다고 명시합니다 .

## 내부수익률(IRR) 규칙 이해

본질적으로 IRR 규칙은 프로젝트 또는 투자를 진행할지 여부를 결정하는 지침입니다. 프로젝트의 예상 IRR이 높을수록(그리고 [자본 비용을 초과하는 금액이 클수록) 프로젝트에서 회사에](/costofcapital) 더 많은 [순 현금](/net-cash) 이 생성됩니다. 즉, 이 경우 프로젝트가 수익성이 있어 보이고 경영진이 진행해야 합니다. 반면에 IRR이 자본 비용보다 낮으면 규칙은 프로젝트 또는 투자를 포기하는 것이 최선의 조치라고 선언합니다.

수학적으로 IRR은 미래 현금 흐름의 [순 현재 가치](/npv) (NPV)가 정확히 0이 되는 비율입니다.

투자자와 기업은 IRR 규칙을 사용하여 [자본 예산 책정 에서 프로젝트를 평가](/capitalbudgeting) 하지만 항상 엄격하게 시행되는 것은 아닙니다. 일반적으로 IRR이 높을수록 좋습니다. 그러나 회사는 더 큰 전략 계획에 기여하거나 경쟁을 방해하는 것과 같은 다른 무형의 이점이 있기 때문에 더 낮은 IRR을 가진 프로젝트를 선호할 수 있습니다. 회사는 또한 더 큰 프로젝트에서 생성되는 더 높은 현금 흐름 때문에 더 높은 IRR을 가진 훨씬 더 작은 프로젝트보다 더 낮은 IRR을 가진 더 큰 프로젝트를 선호할 수 있습니다.

> 회사는 일반적으로 내부 수익률(IRR) 규칙에 의해 제공되는 결론을 따르지만 프로젝트의 규모, 프로젝트가 회사의 더 큰 전략이나 목표에 기여하는지 여부와 같은 기타 고려 사항은 IRR이 낮은 프로젝트를 진행하기로 결정했습니다.

>

## IRR 규칙의 예

회사에서 두 개의 프로젝트를 검토하고 있다고 가정합니다. 경영진은 프로젝트 중 하나, 둘 다 또는 둘 다 진행하지 않을지 여부를 결정해야 합니다. 자본 비용은 10%이며 각각의 현금 흐름 패턴은 다음과 같습니다.

**프로젝트 A**

- 초기 지출 = $5,000

- 1년차 = $1,700

- 2년차 = $1,900

- 3년차 = $1,600

- 4년차 = $1,500

- 5년차 = $700

**프로젝트 B**

- 초기 지출 = $2,000

- 1년차 = $400

- 2년차 = $700

- 3년차 = $500

- 4년차 = $400

- 5년차 = $300

회사는 각 프로젝트에 대한 IRR을 계산해야 합니다. 초기 지출(기간 = 0)은 음수입니다. IRR을 푸는 것은 다음 방정식을 사용하는 반복 프로세스입니다.

$0 = Σ CF~t~ ÷ (1 + IRR)^t^

어디:

- CF = 순 현금 흐름

- IRR = 내부수익률

- t = 기간(0에서 마지막 기간까지)

-또는-

$0 = (초기 지출 * -1) + CF~1~ ÷ (1 + IRR)^1^ + CF~2~ ÷ (1 + IRR)^2^ + ... + CF~X~ ÷ (1 + IRR)^X^

위의 예를 사용하여 회사는 각 프로젝트에 대한 IRR을 다음과 같이 계산할 수 있습니다.

**IRR 프로젝트 A:**

$0 = (-$5,000) + $1,700 ÷ (1 + IRR)^1^ + $1,900 ÷ (1 + IRR)^2^ + $1,600 ÷ (1 + IRR)^3^ + $1,500 ÷ (1 + IRR)^4^ + $700 ÷ (1 + IRR)^5^

IRR 프로젝트 A = **16.61 %**

**IRR 프로젝트 B:**

$0 = (-$2,000) + $400 ÷ (1 + IRR)^1^ + $700 ÷ (1 + IRR)^2^ + $500 ÷ (1 + IRR)^3^ + $400 ÷ (1 + IRR)^4^ + $300 ÷ (1 + IRR)^5^

IRR 프로젝트 B = **5.23 %**

회사의 자본 비용이 10%인 경우 경영진은 프로젝트 A를 진행하고 프로젝트 B를 거부해야 합니다.

## 하이라이트

- 내부수익률(IRR) 규칙은 IRR이 최소 요구수익률(허들율이라고도 함)보다 클 경우 프로젝트 또는 투자를 추진해야 한다고 명시합니다.

- 프로젝트에 덜 유형적인 다른 이점이 있는 경우 회사는 IRR 규칙을 엄격하게 따르지 않을 수 있습니다.

- IRR Rule은 기업이 프로젝트 진행 여부를 결정하는 데 도움이 됩니다.

## 자주하는 질문

### IRR을 사용하는 것은 현금 흐름할인법을 사용하는 것과 동일합니까?

예. IRR을 사용하여 순 현재 가치를 구하는 것은 재무 분석의 현금 흐름 할인 방법으로 알려져 있습니다. IRR(내부 수익률)은 일련의 현금 흐름(양수 및 음수)을 0의 순 현재 가치(NPV)(또는 현금 투자). 투자자와 기업은 IRR을 사용하여 프로젝트에 대한 투자가 정당화될 수 있는지 여부를 평가합니다.

### IRR 규칙은 어떻게 사용됩니까?

본질적으로 IRR 규칙은 프로젝트 또는 투자를 진행할지 여부를 결정하는 지침입니다. IRR이 자본 비용을 초과하는 한 프로젝트의 예상 IRR이 높을수록 회사의 순 현금 흐름이 높아집니다. 반면에 IRR이 자본 비용보다 낮으면 규칙은 프로젝트 또는 투자를 포기하는 것이 최선의 조치라고 선언합니다.

### 기업은 항상 IRR 규칙을 따를까요?

IRR 규칙이 항상 엄격하게 시행되는 것은 아닙니다. 일반적으로 IRR이 높을수록 좋습니다. 그러나 회사는 더 큰 전략 계획에 기여하거나 경쟁을 방해하는 것과 같은 다른 무형 이익이 있기 때문에 여전히 자본 비용을 초과하는 한 더 낮은 IRR을 가진 프로젝트를 선호할 수 있습니다. 궁극적으로 기업은 프로젝트 진행 여부를 결정할 때 여러 요소를 고려합니다. IRR 규칙보다 중요한 요소가 있을 수 있습니다.
