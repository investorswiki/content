# 매크로 바이러스
## 매크로 바이러스란 무엇입니까?

매크로 바이러스는 소프트웨어 프로그램을 감염시키고 프로그램이 열릴 때 일련의 작업이 자동으로 시작되도록 합니다. 일반 [매크로 처럼 작동](/macro-environment) 하며 종종 기존 매크로 대신 자체적으로 설치됩니다.

## 매크로 바이러스 이해

매크로 바이러스는 인터넷에서 시작되어 이미 사람의 컴퓨터에 중단된 프로그램에 침투합니다. 그러나 매크로 바이러스가 무엇인지 완전히 이해하려면 먼저 "매크로"가 무엇인지 이해하는 것이 중요합니다.

매크로는 소프트웨어 프로그램 내에서 특정 기능을 자동으로 트리거하는 일련의 명령입니다. 매크로는 Microsoft Word와 같은 프로그램에 설치하여 프로그램이 자동으로 수행할 수 없는 복잡한 작업을 수행할 수 있습니다. 예를 들어, 회사는 매크로를 사용하여 디자인된 레터헤드 또는 미리 디자인된 표를 Microsoft Word 페이지 템플릿에 자동으로 삽입하거나 프로그램 자체에서는 사용할 수 없는 사용자 지정 페이지 형식을 사용할 수 있습니다.

매크로 바이러스는 바이러스가 사람을 감염시키고, 자신을 복제하고, 다른 사람에게 퍼뜨리는 방식과 마찬가지로 퍼질 수 있는 능력으로 인코딩됩니다. 매크로 바이러스는 사용자 모르게 또는 동의 없이 컴퓨터에 자료를 복제하고 설치할 수 있습니다. 이메일 스팸을 통해 전송된 경우 해당 사용자의 주소록에 있는 모든 사람에게 자동으로 전송되는 경우가 많습니다.

## 매크로 바이러스는 어떻게 컴퓨터를 감염시키나요?

매크로 바이러스는 인터넷에서 시작되며 스팸 이메일에서 자주 발생하며 무해한 매크로를 모방하여 Microsoft Office와 같은 사람의 컴퓨터에 설치된 프로그램에 침투합니다. 미리 설치된 매크로를 대신하는 경우가 많으며 일반 매크로가 실행될 때 활성화되지만 사용자 모르게 프로그램을 사용하지 않을 때도 작동할 수 있습니다. 그들이 트리거하는 자동 작업은 문서에 제어할 수 없는 텍스트 추가에서 사용자 주소록에 있는 사람들에게 스팸 메시지 보내기에 이르기까지 다양합니다.

매크로 바이러스는 종종 컴퓨터나 프로그램 자체에 무해하지만 사용자에게 혼란과 좌절을 안겨줍니다. 컴퓨터에 설치되면 포함되지 않은 경우 여러 프로그램에 퍼질 수 있습니다.

## 매크로 바이러스를 예방하는 방법

오늘날 대부분의 바이러스 백신 소프트웨어 패키지는 컴퓨터에 있는 기존 매크로 바이러스를 찾아 파괴하고 새로운 바이러스가 뿌리를 내리는 것을 방지하도록 설계되었습니다. 그러나 모든 매크로 바이러스가 바이러스 백신 소프트웨어에 의해 탐지되는 것은 아니며 모든 바이러스 백신 소프트웨어가 동일하지는 않습니다. 바이러스 백신 소프트웨어를 최신 상태로 유지하는 것도 중요하지만 인터넷에서 다운로드하거나 여는 항목을 인식하는 것도 마찬가지로 중요합니다.

예를 들어 모르는 주소에서 보낸 이메일의 첨부 파일을 열면 안 됩니다. 이메일 스팸은 쉽게 식별할 수 있지만 속임수를 사용하여 콘텐츠가 합법적이라고 생각하게 만들 수도 있습니다.

## 하이라이트

- 이 매크로 바이러스는 시스템에 혼란을 주는 것 외에도 다른 시스템을 감염시키기 위해 복제 및 확산되도록 프로그래밍되어 있습니다.

- 매크로 바이러스는 가짜 웹 링크나 이메일 첨부 파일을 통해 온라인으로 가장 자주 퍼집니다.

- 안티바이러스 소프트웨어는 일반적으로 매크로 바이러스가 해를 입히기 전에 탐지하고 제거하는 데 능숙합니다.

- 매크로 바이러스는 일련의 소프트웨어 기능을 자동으로 실행하여 종종 해로운 영향을 미치는 일종의 컴퓨터 악용 또는 맬웨어입니다.
