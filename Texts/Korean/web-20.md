# 웹 2.0
## 웹 2.0이란?

Web 2.0은 이전 버전인 Web 1.0에 비해 더 많은 사용자 생성 콘텐츠와 최종 사용자의 유용성을 제공하는 인터넷의 현재 상태를 설명합니다. 일반적으로 웹 2.0은 [닷컴 거품](/dotcom-bubble) 의 여파로 디지털 시대를 변화시킨 21세기 인터넷 애플리케이션을 의미합니다 .

## 웹 2.0의 이해

Web 2.0은 인터넷에 대한 특정 기술 업그레이드를 의미하지 않습니다. 그것은 단순히 21세기에 인터넷이 사용되는 방식의 변화를 나타냅니다. 새로운 시대에는 참가자 간의 정보 공유 및 상호 연결 수준이 높아집니다. 이 새로운 버전을 통해 사용자는 정보를 받아들이는 수동적인 뷰어 역할을 하는 대신 경험에 능동적으로 참여할 수 있습니다.

## 웹 1.0 대 웹 2.0

웹 1.0은 인터넷의 첫 번째 단계를 설명하는 데 사용됩니다. 이 시점에서 콘텐츠 제작자는 거의 없었습니다. 인터넷을 사용하는 사람들의 대부분은 소비자였습니다. 정적 페이지는 특정 코딩이나 언어를 사용하여 대화형 및 애니메이션 웹사이트를 통합하는 동적 [HTML](/html) 보다 더 일반적이었습니다 . 이 단계의 콘텐츠는 데이터베이스 관리 시스템이 아닌 서버의 파일 시스템에서 가져왔습니다. 사용자는 온라인 방명록에 서명할 수 있었고 HTML 양식은 이메일로 전송되었습니다.

Web 1.0으로 분류되는 인터넷 사이트의 예로는 Britannica Online, 개인 웹 사이트 및 mp3.com이 있습니다. 일반적으로 이러한 웹 사이트는 정적이며 기능과 유연성이 제한적입니다.

웹 2.0이라는 용어는 1999년 인터넷이 사용자를 적극적으로 참여시키는 시스템으로 전환하면서 처음 사용되었습니다. 사용자는 콘텐츠를 보는 것보다 콘텐츠를 제공하도록 권장되었습니다. 사람들은 이제 기사와 댓글을 게시할 수 있게 되었고, 다른 사이트에서 사용자 계정을 만드는 것이 가능해지면서 참여도가 높아졌습니다. Web 2.0은 또한 웹 앱, [WordPress 와 같은 자체 출판 플랫폼](/wordpress-cms) 및 [소셜 미디어](/social-media) 사이트 를 탄생시켰습니다 .

Web 2.0 사이트의 예로는 Wikipedia, Facebook, Twitter 및 다양한 블로그가 있으며 모두 동일한 정보를 공유하고 전달하는 방식을 변화시켰습니다.

인터넷의 사회적 측면은 특히 변형되었습니다. 일반적으로 소셜 미디어를 통해 사용자는 생각, 관점 및 의견을 공유하여 서로 참여하고 상호 작용할 수 있습니다. 사용자는 태그, 공유, 트윗 등을 할 수 있습니다.

## 웹 2.0의 장점과 단점

기술의 발전으로 사용자는 자신의 생각과 의견을 다른 사람들과 공유할 수 있게 되었고, 이는 다른 사람들을 조직하고 연결하는 새로운 방법을 만들어 냈으며 더 큰 수준의 협업을 촉진했습니다.

그러나 인터넷이 공개 포럼처럼 작동하는 데에는 많은 단점이 있습니다. 소셜 미디어의 확장을 통해 우리는 온라인 스토킹, 사이버 괴롭힘, [독싱](/personally-identifiable-information-pii),. [신분 도용](/identitytheft) 및 기타 온라인 범죄가 증가하는 것을 보았습니다. 또한 오픈 소스 정보 공유 사이트나 소셜 미디어를 통해 사용자 간에 잘못된 정보가 확산될 위험이 있습니다.

Web 1.0과 마찬가지로 Web 2.0은 인터넷 진화의 또 다른 과도기적 단계입니다. Web 3.0은 모든 사용자의 요구에 보다 직관적으로 맞춤화될 것이기 때문에 Semantic Web이라고 불릴 것으로 예상됩니다.

##하이라이트

- 인터넷의 새로운 시대에 참여자 간의 정보 공유 및 상호 연결 수준이 높아집니다.

- Web 2.0은 인터넷에 대한 특정 기술 업그레이드를 의미하지 않습니다. 인터넷 사용 방식의 변화를 의미합니다.

- Web 2.0은 이전 버전인 Web 1.0에 비해 더 많은 사용자 생성 콘텐츠와 최종 사용자를 위한 유용성을 제공하는 인터넷의 현재 상태를 설명합니다.
