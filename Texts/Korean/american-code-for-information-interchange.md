# 정보 교환을 위한 미국 코드(ASCII)
## 정보 교환을 위한 미국 코드(ASCII)란 무엇입니까?

일반적으로 ASCII라고 하는 미국 정보 교환 코드는 1963년 컴퓨터 간에 정보를 전송하는 표준이 된 텍스트용 컴퓨터 언어입니다.

ASCII 시스템은 영숫자 문자와 기호를 컴퓨터가 읽을 수 있는 코드로 변환합니다. 원래 ASCII는 영어 전용이었습니다. 이제 두 가지 유형의 ASCII 코드가 있습니다. 7비트 인코딩 시스템을 사용하는 표준 코드와 8비트 시스템을 사용하는 확장 코드가 있습니다.

인터넷 통신에서 ASCII는 모든 언어에서 사용할 수 있고 ASCII와 호환되는 유니코드 표준으로 점차 대체되었습니다. 그러나 ASCII는 여전히 개인용 컴퓨터 및 전자 장치에 텍스트를 입력하는 데 사용됩니다.

## 정보 교환을 위한 미국 코드(ASCII) 이해

ASCII의 가장 초기 버전은 Bell 데이터 서비스가 텔레프린터에 메시지를 전달하는 데 사용하는 텔레그래픽 코드에서 찾을 수 있습니다.

ASCII는 데이터 처리를 위한 최초의 주요 문자 인코딩 시스템이었으며 1963년에 컴퓨팅 표준으로, 1969년에 인터넷 전송 표준으로 채택되었습니다. 월스트리트 거래자와 중개인이 사용하는 주문 입력 컴퓨터 시스템에서 오랫동안 사용되었습니다.

그것은 점차적으로 인터넷 표준으로 유니코드 세계 문자 표준(Unicode)으로 대체되었습니다. 그러나 유니코드의 처음 128자는 ASCII와 동일하기 때문에 두 시스템이 공존한다고 할 수 있습니다.

> Alt 키를 누른 상태에서 문자 코드 번호를 입력하면 Windows 호환 컴퓨터를 사용하여 문서에 ASCII 기호를 입력할 수 있습니다. 예를 들어 ALT 키를 누른 상태에서 156을 입력하면 영국 파운드 기호인 £가 표시됩니다.

>

### ASCII란?

ASCII가 표준으로 채택되기 전에 모든 컴퓨터 제조업체는 자체 코드를 사용했으며 일부 회사는 서로 다른 제품에 둘 이상의 코드를 사용했습니다. 컴퓨터가 서로 통신할 수 없었습니다.

ASCII는 1963년 American National Standards에 의해 표준으로 채택되었지만, 1968년까지 미국에서도 완전히 구현되지 않았습니다. 컴퓨터는 서로 통신할 수 있습니다.

ASCII는 원래 128개의 알파벳 문자, 숫자 및 기호를 포함했으며 영어를 기반으로 했습니다. 다른 언어를 수용하기 위해 1970년대에 확장된 양식이 추가되었습니다.

곧 다양한 버전의 ASCII가 사용되었습니다. 결국 255자를 포함했습니다. 이 버전을 확장 ASCII라고 합니다.

오늘날 사용자는 Alt 키를 누르고 NUM 잠금 키가 켜진 상태에서 숫자 키패드에 문자 코드를 입력하여 저작권 기호와 같은 ASCII 또는 유니코드 기호를 문서에 입력할 수 있습니다.

##하이라이트

- ASCII는 계속 존재하지만 대부분의 언어를 인코딩하는 데 사용할 수 있는 유니코드로 대체되었습니다.

- ASCII는 원래 128개의 영어 문자와 기호만 포함했지만 나중에 다른 언어에서 사용되는 문자를 포함하여 추가 문자를 포함하도록 확장되었습니다.

- 미국 정보 교환 코드(ASCII)는 컴퓨터 간에 텍스트를 전송하는 데 사용하도록 설계된 문자 인코딩 형식입니다.

##자주하는 질문

### ASCII는 무엇입니까? 유니코드?

유니코드는 ASCII의 범용 버전으로 볼 수 있습니다. ASCII는 결국 American Code for Information Interchange이고 첫 번째 반복에는 해당 언어에서 사용되는 영어 알파벳과 기호가 포함되었습니다. 이후 버전에는 다른 많은 언어에서 사용되는 기호가 포함되었습니다. 반면 유니코드는 ASCII에서 사용되는 것과 동일한 128개의 기호로 시작하지만 현재 버전에는 144,697개의 문자가 포함되어 있습니다.

### 아스키란?

정보 교환을 위한 미국 표준 코드(ASCII)는 텍스트의 전자 전송을 위한 문자 인코딩 형식입니다. 모든 문자는 고유 번호로 표시됩니다. ASCII의 첫 번째 버전에는 대문자와 소문자, 쉼표와 같은 일반적으로 사용되는 기호를 나타내는 알파벳 문자를 나타내는 128자만 포함되었습니다. 이후 버전에서는 다음과 같은 추가 기호를 포함하여 ASCII를 255자로 확장했습니다. 영국 파운드 기호(£)와 스페인어 텍스트에 사용되는 거꾸로 된 물음표(¿).

### ASCII 문자란 무엇입니까?

확장 ASCII 테이블에서 문자 코드 0 ~ 31은 텍스트 시작 및 백스페이스와 같은 제어 코드입니다. 문자 코드 32 ~ 127은 대부분 알파벳 문자(대문자 및 소문자)와 별표 및 달러 기호와 같은 기호입니다.Character 코드 128 - 255는 여러 버전으로 제공되며, 각 버전은 거꾸로 느낌표, 엔 기호 및 그루지야어 쉼표와 같은 원래 영어 전용 ASCII에 추가된 사항을 반영합니다.
