# 목표 추구
## 목표 추구란 무엇입니까?

목표 탐색은 출력만 알고 있을 때 올바른 입력 값을 찾는 과정입니다. 목표 추구 기능은 Microsoft Excel과 같은 다양한 컴퓨터 소프트웨어 프로그램에 내장될 수 있습니다.

## 목표 추구 이해

목표 추구는 이미 알려진 출력 값을 기반으로 입력 값을 알아내는 것과 관련된 프로세스를 설명하는 데 사용되는 일반적인 용어입니다. 이 프로세스에는 컴퓨터 소프트웨어를 사용하여 계산할 수 있는 공식에서 특정 연산자를 사용하는 것이 포함됩니다.

목표 추구는 컴퓨터 소프트웨어 프로그램에 대한 "가정 분석"에 사용되는 도구 중 하나입니다. [가정 분석 은 이러한 변경 사항이 워크시트의 수식 결과](/sensitivityanalysis) 에 [어떤](/sensitivityanalysis) 영향을 미치는지 확인하기 위해 (Microsoft Excel) 셀의 값을 변경하는 프로세스입니다. 목표를 추구할 때 주어진 값 또는 출력에 대해 가정 분석을 수행하고 있습니다. 따라서 본질적으로 "출력이 X이면 어떻습니까?" 또는 기본적으로 원인과 결과 상황을 묻는 시나리오를 만드는 것입니다.

좀 더 복잡한 문제의 경우 사람들은 종종 컴퓨터 소프트웨어를 사용합니다. Microsoft Excel과 같은 스프레드시트 프로그램에는 목표 찾기 도구가 내장되어 있습니다. 출력 값이 이미 알려진 경우 사용자가 공식에 대해 원하는 입력 값을 결정할 수 있습니다. 이 기능은 사용자 가 매달 지불할 수 있는 금액(출력)만 알고 있는 경우 차용인이 자격을 갖추는 데 필요한 [이자율 (입력)과 같은 사항을 결정하는 데 도움이 될 수 있습니다.](/interestrate)

그러나 목표 추구 소프트웨어를 사용할 때 한 가지 주의할 점이 있습니다. 하나의 입력 값이 있는 경우에만 작동합니다. 두 개 이상의 입력 값을 파악해야 하는 경우 작동하지 않습니다. 위의 예를 들자면 대출 총액과 월납입금을 파악하고 싶다면 목표추적 소프트웨어가 작동하지 않습니다. 여러 변수를 파악하려면 애드온이 필요할 것입니다.

### Microsoft Excel에서 목표 추구가 작동하는 방식

위에서 언급했듯이 목표 탐색 소프트웨어는 이미 출력 값(또는 결과)을 알고 있지만 하나의 입력 값을 결정하려는 경우에만 작동합니다. 다음은 프로그램에서 목표 찾기 기능을 사용하려는 경우 따라야 할 몇 가지 주요 단계입니다.

- 새 스프레드시트 열기

- 열에 레이블을 지정합니다. 이렇게 하면 워크시트의 모든 내용을 더 쉽게 읽을 수 있습니다. 따라서 위의 예를 사용하면 첫 번째 열은 "대출 금액", 두 번째 열은 " 개월 단위 [기간](/term) ", 세 번째 열은 "이자율", 마지막 열은 "지급"이 됩니다.

- 알고 있는 값을 입력하십시오.

- 목표 계산 공식을 입력합니다. 이 경우에는 지불입니다. 공식에서 가정할 이자율은 현재 0%입니다.

이제 이자율을 결정해야 합니다. Excel의 Goal Seek 기능을 사용하여 이 작업을 수행하고 이미 가지고 있는 값을 입력할 수 있습니다.

## 목표 추구의 예

목표 추구 [기업가](/entrepreneur) 는 궁극적인 목표에 도달할 수 있는 방법을 결정하기 위해 목표 추구를 사용하는 사람입니다. 예를 들어, 기업가는 1년에 총 10만 달러를 벌기 위해 시간당 얼마를 벌어야 하는지 물을 수 있습니다. 그들은 원하는 출력 값($100,000)을 알고 있으므로 최적의 입력을 알아내기 위해 다시 작업해야 합니다. 즉, 한 해 동안 일할 수 있는 시간(또는 기꺼이 할 의사가 있는)을 결정해야 하므로 목표를 달성하기 위해 시간당 얼마를 벌 수 있는지 결정해야 합니다.

## 하이라이트

- Goal-seek 소프트웨어는 입력 값이 하나만 있는 경우에만 작동합니다.

- 목표 탐색은 출력만 알고 있을 때 올바른 입력을 찾는 것입니다.

- 이것은 종종 Microsoft Excel의 목표 탐색 기능이나 컴퓨터 프로그램을 통한 가상 분석을 사용하여 수행됩니다.
