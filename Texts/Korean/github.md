# 깃허브
GitHub는 사용자가 자신의 계정에 파일, 문서 및 컴퓨터 코드를 업로드할 수 있는 웹 기반 오픈 소스 개발 플랫폼 및 호스팅 서비스입니다. GitHub를 사용하면 등록된 사용자와 등록되지 않은 사용자 모두 업로드된 파일과 소스 코드 저장소를 보고, 다운로드하고, 기여할 수 있습니다. 저렴한 비용과 기능으로 인해 GitHub는 세계에서 가장 큰 소스 코드 플랫폼이 되었으며 많은 개발자가 협업 작업을 수행하는 방법으로 사용하는 동시에 다양한 소프트웨어, 파일 및 문서를 공유할 수 있습니다.

#### GitHub 작동 방식

GitHub 계정이 등록되면 사용자는 저장소라고 하는 파일 및 콘텐츠의 고유한 디렉터리를 만들 수 있습니다. 리포지토리를 생성한 후에는 소위 마스터 브랜치를 생성할 수 있습니다. GitHub 마스터 브랜치는 모든 프로젝트의 단일 시작점을 나타냅니다. 프로젝트를 누구나 사용할 수 있도록 하거나(공개 저장소) 프로젝트에 대한 액세스를 제한하고 특정 사용자와만 공유(개인 저장소)하는 것은 사용자의 몫입니다. 마스터 브랜치에서 서브 브랜치를 생성할 수 있으며, 기본적으로 사용자가 프로젝트의 다양한 측면에서 개별적으로 작업할 수 있습니다. 공개 리포지토리에서 다른 사용자는 코드에 대한 변경 사항을 제안할 수 있으며 결국에는 새 버전의 파일을 업로드할 수 있습니다. 제안은 pull request라는 기능을 통해 이루어지며, 이를 통해 개발자는 잠재적인 변경 사항을 효과적으로 적용하기 전에 논의하고 검토할 수 있습니다. GitHub 플랫폼은 개발자가 변경 사항을 쉽게 적용하거나 되돌리고 전반적인 진행 상황을 추적할 수 있는 다양한 기능과 도구를 제공합니다.

#### 풀 리퀘스트가 무엇인가요?

pull 요청을 통해 사용자는 특정 분기 또는 저장소에 대한 변경 사항을 다른 사람에게 알릴 수 있습니다. 풀 리퀘스트가 생성되면 사용자는 코드에 대한 모든 변경 사항을 쉽게 검토하고 이전 버전과 새 버전을 비교할 수 있는 개요 화면으로 이동합니다. 따라서 모든 변경 사항은 문서화되고 기록되며 사용자는 주석, 레이블 및 이정표를 추가하거나 특정 작업을 다른 기여자에게 할당할 수도 있습니다.

#### GitHub는 누가 사용합니까?

유연한 플랫폼으로 인해 GitHub에는 공공 및 민간 부문의 사용자가 있습니다. 소프트웨어 회사, 앱 개발자, 개인 프로그래머 및 암호 화폐 개발자는 일부 예입니다. 오픈 소스 프로젝트로서 [Bitcoin GitHub](/bitcoin) 에는 Bitcoin 소스 코드의 검토 및 개선에 기여하는 많은 사람들과 함께 전 세계에 많은 개발자가 있습니다.

#### GitHub는 무료인가요?

GitHub는 모든 오픈 소스 및 상용 개발 프로젝트에 대해 무료입니다. 그러나 무료 계획은 계정에서 생성할 수 있는 개인 및 공용 리포지토리의 수를 제한합니다. 무료 플랜은 또한 프로젝트가 가질 수 있는 공동 작업자의 수를 제한합니다. 따라서 더 큰 프로젝트는 필요에 따라 유료 GitHub 플랜으로 업그레이드해야 할 수 있습니다.
