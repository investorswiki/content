# 기술 통계
## 기술 통계란 무엇입니까?

기술 통계는 주어진 데이터 세트를 요약하는 간단한 기술 계수로, 전체 모집단을 나타내거나 모집단 표본이 될 수 있습니다. 기술 통계는 중심 경향 측정과 변동성 측정(확산)으로 나뉩니다. 중심 경향 측정에는 평균, 중앙값 및 최빈값이 포함되고 변동성 측정에는 표준 편차, 분산, 최소 및 최대 변수, [첨도](/kurtosis) 및 [왜도 가 포함](/skewness) 됩니다.

## 기술 통계의 이해

간단히 말해서 기술 통계는 데이터의 표본 및 측정값에 대한 간단한 요약을 제공하여 특정 데이터 세트의 기능을 설명하고 이해하는 데 도움이 됩니다. 기술 통계의 가장 잘 알려진 유형은 거의 모든 수준의 수학 및 통계에서 사용되는 [평균](/mean),. [중앙값](/median) 및 [모드 의 중심 측정입니다.](/mode) 평균 또는 평균은 데이터 세트 내의 모든 수치를 더한 다음 세트 내의 수치 수로 나누어 계산됩니다.

예를 들어, 다음 데이터 세트의 합은 20입니다: (2, 3, 4, 5, 6). 평균은 4(20/5)입니다. 데이터 세트의 모드는 가장 자주 나타나는 값이며 중앙값은 데이터 세트의 중간에 위치한 수치입니다. 데이터 세트 내에서 높은 수치와 낮은 수치를 구분하는 수치입니다. 그러나 여전히 매우 중요한 기술 통계의 덜 일반적인 유형이 있습니다.

사람들은 기술 통계를 사용하여 대규모 데이터 세트에서 이해하기 어려운 정량적 통찰력을 한 입 크기의 설명으로 변경합니다. 예를 들어, 학생의 평균 학점(GPA)은 기술 통계에 대한 좋은 이해를 제공합니다. GPA의 개념은 광범위한 시험, 수업 및 성적에서 데이터 포인트를 가져와 함께 평균화하여 학생의 전반적인 학업 성취도에 대한 일반적인 이해를 제공한다는 것입니다. 학생의 개인 GPA는 평균 학업 성취도를 반영합니다.

## 기술 통계의 유형

모든 기술 통계는 중심 경향의 측정 또는 분산 측정이라고도 하는 변동성의 측정 [입니다](/variability) [.](/variability)

###중앙 경향

중심 경향 측정은 데이터 세트의 평균 또는 중간 값에 초점을 맞추는 반면 변동성 측정은 데이터 분산에 초점을 맞춥니다. 이 두 가지 측정값은 그래프, 표 및 일반 토론을 사용하여 사람들이 분석된 데이터의 의미를 이해하도록 돕습니다.

중심 경향 측정은 데이터 세트에 대한 분포의 중심 위치를 설명합니다. 사람은 분포에서 각 데이터 포인트의 빈도를 분석하고 분석된 데이터 세트의 가장 일반적인 패턴을 측정하는 평균, 중앙값 또는 최빈값을 사용하여 설명합니다.

### 변동성 측정

변동성 측정(또는 확산 측정)은 데이터 집합에 대해 분포가 얼마나 분산되어 있는지 분석하는 데 도움이 됩니다. 예를 들어, 중심 경향의 측정은 사람에게 데이터 세트의 평균을 제공할 수 있지만 데이터가 세트 내에서 어떻게 분포되어 있는지는 설명하지 않습니다.

따라서 데이터의 평균은 100개 중 65개일 수 있지만 여전히 1과 100에 데이터 포인트가 있을 수 있습니다. 가변성 측정은 데이터 세트의 모양과 확산을 설명하여 이를 전달하는 데 도움이 됩니다. 범위, [사분위수](/quartile),. 절대 편차 및 분산은 모두 변동성 측정의 예입니다.

다음 데이터 세트를 고려하십시오. 5, 19, 24, 62, 91, 100. 해당 데이터 세트의 범위는 95이며, 가장 높은 값(100)에서 데이터 세트의 가장 낮은 숫자(5)를 빼서 계산됩니다.

##하이라이트

- 기술통계는 중심경향 측정과 변동성(또는 확산) 측정이라는 두 가지 기본 범주의 측정으로 구성됩니다.

- 기술 통계는 데이터 세트의 특성을 요약하거나 설명합니다.

- 변동성 또는 산포도는 집합 내 데이터의 산포를 나타냅니다.

- 중심 경향 측정은 데이터 집합의 중심을 나타냅니다.

##자주하는 질문

### 기술 통계를 사용하여 추론이나 예측을 할 수 있습니까?

아니요. 이러한 설명은 데이터 속성을 이해하는 데 도움이 되지만 데이터 세트에서 변수가 서로 상호 작용하는 방식을 이해하려면 통계의 별도 분기인 추론 통계 기술이 필요합니다.

### 평균과 표준편차는 무엇인가요?

이들은 일반적으로 사용되는 두 가지 기술 통계입니다. 평균은 일부 데이터에서 관찰된 평균 수준이고 표준 편차는 분산 또는 해당 변수에서 관찰된 데이터가 평균 주위에 분포되어 있는 정도를 나타냅니다.

### 단순히 데이터를 설명하는 통계가 왜 필요한가요?

기술 통계는 변수의 평균, 표준 편차 또는 빈도와 같은 샘플 또는 데이터 세트의 특성을 설명하거나 요약하는 데 사용됩니다. 추론 통계는 데이터 샘플 요소의 집합적 속성을 이해하는 데 도움이 될 수 있습니다. 변수의 표본 평균, 분산 및 분포를 알면 주변 세계를 이해하는 데 도움이 됩니다.
