# 하이퍼텍스트 마크업 언어 – HTML
## HTML(HyperText Markup Language)이란 무엇입니까?

HTML(HyperText Markup Language)은 인터넷에 표시하기 위해 파일에 삽입된 마크업 기호 또는 코드의 집합입니다. 마크업은 웹 브라우저에 웹 페이지의 단어와 이미지를 표시하는 방법을 알려줍니다.

각 개별 조각 마크업 코드("<"와 ">" 사이에 위치)를 요소라고 하지만 많은 사람들이 이를 태그라고도 합니다. 일부 요소는 일부 디스플레이 효과가 시작되고 종료될 때를 나타내는 쌍으로 제공됩니다.

## HTML 설명

HyperText Markup Language는 웹사이트 생성을 용이하게 하는 컴퓨터 언어입니다. 다른 언어와 마찬가지로 코드 단어와 구문이 있는 언어는 상대적으로 이해하기 쉽고 시간이 지남에 따라 누군가가 만들 수 있는 능력이 점점 더 강력해집니다. HTML은 언어를 설계하고 유지 관리하는 조직인 World Wide Web Consortium을 가장하여 인터넷의 요구 사항과 요구 사항을 충족하도록 계속 발전하고 있습니다. 예를 들어 [Web 2.0](/web-20) 으로의 전환과 함께 .

하이퍼텍스트는 인터넷 사용자가 웹을 탐색하는 방법입니다. 하이퍼링크라는 특수 텍스트를 클릭하면 사용자가 새 페이지로 이동합니다. 하이퍼를 사용한다는 것은 선형적이지 않다는 것을 의미하므로 사용자는 사용 가능한 링크를 클릭하기만 하면 인터넷의 어느 곳이든 갈 수 있습니다. 마크업은 HTML 태그가 태그 안의 텍스트에 수행하는 작업입니다. 그들은 그것을 특정 유형의 텍스트로 표시합니다. 예를 들어, 마크업 텍스트는 단어나 구에 대한 특정 주의를 끌기 위해 볼드체 또는 이탤릭체의 형태로 제공될 수 있습니다.

## 하이퍼텍스트 마크업 언어 기초

기본적으로 HTML은 텍스트 파일에 입력된 일련의 짧은 코드입니다. 이들은 HTML의 기능을 강화하는 태그입니다. 텍스트는 HTML 파일로 저장되고 웹 브라우저를 통해 볼 수 있습니다. 브라우저는 파일을 읽고 작성자가 가시적 렌더링이 되는 것을 작성하는 데 사용한 코드의 지시에 따라 텍스트를 가시적 형식으로 변환합니다. HTML을 작성하려면 작성자의 비전을 만들기 위해 태그를 올바르게 사용해야 합니다.

태그는 HTML 코드에서 일반 텍스트를 구분하는 것입니다. 태그는 웹 페이지에 그래픽, 이미지 및 표를 표시할 수 있도록 하는 꺾쇠 괄호 사이의 단어입니다. 다른 태그는 다른 기능을 수행합니다. 가장 기본적인 태그는 텍스트에 서식을 적용합니다. 웹 인터페이스가 보다 동적으로 변해야 하므로 CSS(Cascading Style Sheets) 및 JavaScript 응용 프로그램을 사용할 수 있습니다. CSS는 웹 페이지의 접근성을 높이고 JavaScript는 기본 HTML에 기능을 추가합니다.

## HTML 대 XML

HTML과 달리 Extensible Markup Language 또는 XML을 사용하면 사용자가 자신의 마크업을 정의할 수 있습니다. 예를 들어, XML을 사용하여 한 사용자는 <footnote> 태그로 각주를 표시하도록 선택할 수 있고 다른 사용자는 <fn>을 선택할 수 있습니다.

HTML을 사용하면 하나의 미리 결정된 태그만 특정 유형의 정보를 나타내는 데 사용할 수 있습니다. XML 문서는 사용자 정의 태그를 포함하고 문서가 마크업과 콘텐츠로만 구성되어 있기 때문에 읽기 쉽습니다.

## 하이라이트

- HTML(HyperText Markup Language)은 웹 브라우저에서 월드 와이드 웹에서 페이지를 렌더링하는 데 사용하는 기본 스크립팅 언어입니다.

- HTML의 초기 버전은 정적(Web 1.0)이었지만, 최신 버전은 상당한 동적 유연성(Web 2.0, 3.0)을 특징으로 합니다.

- 마크업은 두 개의 뾰족한 대괄호(예: <각주>) 사이에 나타나는 텍스트이고 내용은 그 외의 모든 것입니다.

- 하이퍼텍스트를 사용하면 사용자가 링크를 클릭하고 해당 링크에서 참조하는 새 페이지로 리디렉션할 수 있습니다.
