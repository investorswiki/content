# 파일 전송 프로토콜(FTP)
## 파일 전송 프로토콜(FTP)이란 무엇입니까?

FTP(파일 전송 프로토콜)라는 용어는 네트워크를 통해 장치 간에 파일을 전송하는 프로세스를 나타냅니다. 이 프로세스는 한 당사자가 다른 당사자가 인터넷을 통해 파일을 보내거나 받는 것을 허용할 때 작동합니다. 원래 사용자가 두 개의 물리적 장치 간에 정보를 교환하고 통신하는 방법으로 사용되었지만 이제는 일반적으로 원격으로 보관되는 안전한 위치인 [클라우드 에 파일을 저장하는 데 일반적으로 사용됩니다.](/cloud-computing)

FTP는 [기업](/business) 이나 개인이 한 컴퓨터 시스템에서 다른 컴퓨터 시스템으로 파일을 전송하거나 웹사이트에서 서버에서 파일을 업로드하거나 다운로드하는 데 사용할 수 있습니다.

## 파일 전송 프로토콜(FTP) 작동 방식

파일 전송 프로토콜을 사용하면 개인과 기업이 같은 공간에 있지 않고도 전자 파일을 다른 사람과 공유할 수 있습니다. 이것은 FTP 클라이언트를 사용하거나 클라우드를 통해 수행할 수 있습니다. 옵션에 관계없이 양쪽 모두 작동하는 인터넷 연결이 필요합니다.

대부분의 [웹](/web-20) 브라우저에는 사용자가 컴퓨터에서 서버로 또는 그 반대로 파일을 전송할 수 있는 FTP 클라이언트가 함께 제공됩니다. 많은 사용자가 추가 기능을 제공하기 때문에 일부 사용자는 타사 FTP 클라이언트를 사용하려고 할 수 있습니다. 무료로 다운로드할 수 있는 FTP 클라이언트의 예로는 FileZilla Client, FTP Voyager, WinSCP, CoffeeCup Free FTP 및 Core FTP가 있습니다.

많은 사람들이 자신도 모르는 사이에 FTP를 사용했습니다. 웹 페이지에서 파일을 다운로드한 적이 있다면 FTP를 사용해 본 적이 있을 것입니다. 첫 번째 단계는 자동으로 또는 사용자 이름과 비밀번호를 수동으로 입력하여 로그인하는 것입니다. FTP를 사용하려면 특정 포트 번호를 통해 FTP 서버에 액세스해야 합니다. FTP 클라이언트를 통해 FTP 서버에 액세스하면 이제 파일을 전송할 수 있습니다. 일부 서버에서는 익명으로 액세스할 수 있으므로 모든 공용 FTP 서버에 로그인이 필요한 것은 아닙니다.

위에서 언급했듯이 FTP는 원래 두 대의 물리적 컴퓨터 간에 파일을 보내고 받는 방법으로 개발되었습니다. 그러나 기술의 변화로 사용자는 클라우드를 통해 파일 전송을 실행할 수 있습니다. 클라우드를 사용하면 비교적 저렴한 비용으로 편리하고 안전하게(개인 및 회사를 [데이터 침해 로부터 보호할 수 있음) 전송을 수행할 수 있습니다.](/data-breach)

> FTP 클라이언트라는 용어는 파일을 다른 사람에게 전송할 수 있는 소프트웨어를 의미합니다.

>

## 특별 고려 사항

파일 전송 프로토콜은 컴퓨터와 컴퓨팅 시스템이 인터넷에서 작동하는 방식을 지시하는 다양한 프로토콜 중 하나입니다. 기타 이러한 프로토콜에는 다음이 포함됩니다.

- **하이퍼텍스트 전송 프로토콜(HTTP):** 웹을 통해 데이터를 전송하도록 설계됨

- **Internet Message Access Protocol(IMAP):** 공유 서비스에서 게시판 또는 이메일 메시지에 대한 액세스를 제공합니다.

- **네트워크 시간 프로토콜(NTP):** 네트워크를 통해 컴퓨터의 시계 시간을 동기화합니다.

FTP를 사용하면 인터넷 상의 컴퓨터가 파일을 주고받을 수 있습니다. 따라서 오늘날 웹 사이트를 구축하고 유지 관리하는 데 필수적인 도구입니다.

### FTP 클라이언트에서 찾아야 할 사항

개별 FTP 클라이언트는 사용자가 파일을 업로드 및 다운로드하는 방식을 수정할 수 있는 다양한 기능을 제공합니다. 예를 들어, FileZilla를 사용하는 경우 프로그램을 통해 파일에 대한 [대역폭](/bandwidth) 제한을 설정할 수 있습니다. 이를 통해 업로드 및 다운로드 속도를 제어할 수 있으며, 이는 한 번에 여러 파일 전송을 관리하는 경우에 유용할 수 있습니다.

FTP 클라이언트에서 찾을 수 있는 다른 기능으로는 공개 키 인증, 파일 압축 수준 설정 기능 또는 파일 마스크를 사용하여 서버를 검색할 수 있는 도구가 있습니다.

## 파일 전송 프로토콜(FTP)의 예

FTP 소프트웨어는 설정이 비교적 간단합니다. FileZilla는 무료로 다운로드할 수 있는 FTP 클라이언트입니다. 접속하고자 하는 서버의 주소, 포트, 접속 비밀번호를 입력합니다.

액세스 권한이 부여되면 로컬 시스템의 사용자 파일과 액세스된 서버가 표시됩니다. 사용자는 서버에서 로컬 시스템으로 파일을 다운로드하거나 로컬 시스템에서 서버로 파일을 업로드할 수 있습니다. 또한 적절한 권한이 있는 한 서버의 파일을 변경할 수도 있습니다.

##하이라이트

- FTP는 웹사이트를 구축하고 유지 관리하는 사람들에게 필수적인 도구입니다.

- 파일 전송 프로토콜은 인터넷과 컴퓨터 시스템 간에 파일을 한 위치에서 다른 위치로 다운로드, 업로드 및 전송하는 방법입니다.

- FTP를 사용하면 컴퓨터 간에 또는 클라우드를 통해 파일을 주고받을 수 있습니다.

- 사용자는 FTP 전송을 실행하기 위해 인터넷 연결이 필요합니다.

- 대부분의 웹사이트에 이미 FTP가 내장되어 있지만 많은 FTP 클라이언트는 무료로 다운로드할 수 있습니다.
