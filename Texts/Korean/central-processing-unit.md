# 중앙 처리 장치(CPU)
컴퓨터 프로그램의 명령을 해석하고 해당 명령에 따라 기본 작업을 실행하는 컴퓨터의 전자 회로입니다. 기본 연산에는 산술, 논리, 제어 및 입출력(I/O)이 포함됩니다. CPU(중앙 처리 장치)라는 용어는 1960년대 초반부터 컴퓨터 산업에서 널리 사용되었습니다.

CPU는 기본적으로 4가지 기능 단위로 구성됩니다.

- 제어 장치: CPU 내에서 명령 및 데이터의 흐름을 제어하는 역할을 합니다.

- 산술 논리 장치(ALU): CPU의 모든 산술 및 논리 연산 계산을 수행합니다.

- 레지스터: 매우 빠르게 액세스할 수 있는 내부 메모리 셀입니다. 이러한 구성 요소는 변수(데이터, 주소) 또는 산술/논리 연산의 중간 결과를 저장하는 데 사용됩니다.

- 캐시: 주 메모리에 대한 액세스를 최소화하여 결과적으로 CPU 성능을 향상시키는 작고 빠른 메모리입니다.

이 장치는 클록 속도에 의해 동기화되고 세 가지 유형의 버스로 연결됩니다.

- 데이터 버스: 데이터 전달을 담당합니다.

- 주소 버스: 읽거나 쓸 메모리 주소를 전달합니다.

- 제어 버스: 다른 구성 요소 및 I/O 장치를 관리할 수 있습니다.

CPU 아키텍처는 실행할 수 있는 명령어 세트도 특징입니다. 일반적으로 명령어 세트 아키텍처에는 두 가지 유형이 있습니다.

- CISC(복합 명령어 집합 컴퓨터): 여러 클록 주기에서 산술 연산, 메모리 액세스 또는 주소 계산과 같은 여러 저수준 연산을 수행할 수 있는 광범위한 복잡한 명령어 집합입니다.

- RISC(Reduced Instruction Set Computer): 하나의 클록 주기에서 단일 하위 수준 연산을 수행하는 축소된 명령어 집합입니다.
