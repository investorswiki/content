# 데이터 분석
## 데이터 분석이란 무엇입니까?

데이터 분석은 원시 데이터를 분석하여 해당 정보에 대한 결론을 내리는 과학입니다. 데이터 분석의 많은 기술과 프로세스는 인간이 소비하기 위해 원시 데이터를 처리하는 기계적 프로세스와 [알고리즘 으로 자동화되었습니다.](/algorithm)

## 데이터 분석 이해

데이터 분석은 다양한 유형의 데이터 분석을 포괄하는 광범위한 용어입니다. 모든 유형의 정보에 데이터 분석 기술을 적용하여 개선하는 데 사용할 수 있는 통찰력을 얻을 수 있습니다. 데이터 분석 기술을 사용하면 대량의 정보에서 잃어버릴 수 있는 추세와 메트릭을 밝힐 수 있습니다. 그런 다음 이 정보를 사용하여 프로세스를 최적화하여 비즈니스 또는 시스템의 전반적인 효율성을 높일 수 있습니다.

예를 들어, [제조](/manufacturing) 회사는 종종 다양한 기계의 런타임, 가동 중지 시간 및 작업 대기열을 기록한 다음 데이터를 분석하여 작업 부하를 더 잘 계획하여 기계가 최대 용량에 가깝게 작동하도록 합니다.

프로덕션의 [병목 현상 을](/bottleneck) 지적하는 것 이상을 수행할 수 있습니다 . 게임 회사는 데이터 분석을 사용하여 대다수의 플레이어를 게임에서 계속 활성 상태로 유지하는 플레이어에 대한 보상 일정을 설정합니다. 콘텐츠 회사는 다른 보기 또는 다른 클릭을 얻기 위해 콘텐츠를 계속 클릭, 시청 또는 재구성하기 위해 동일한 데이터 분석을 많이 사용합니다.

데이터 분석은 기업이 성과를 최적화하는 데 도움이 되기 때문에 중요합니다. 이를 비즈니스 모델에 구현한다는 것은 기업이 보다 효율적인 비즈니스 수행 방법을 식별하고 많은 양의 데이터를 저장함으로써 비용 절감에 도움을 줄 수 있음을 의미합니다. 기업은 또한 데이터 분석을 사용하여 더 나은 비즈니스 결정을 내리고 고객 동향과 만족도를 분석하여 새롭고 더 나은 제품과 서비스를 제공할 수 있습니다.

> ### SQL

> 현대 데이터 분석의 초창기 중 일부는 SQL 덕분입니다. 1979년에 만들어진 이 컴퓨팅 언어를 사용하면 관계형 데이터베이스를 쿼리하고 결과 데이터 세트를 더 쉽게 분석할 수 있습니다. SQL은 오늘날에도 여전히 널리 사용됩니다.

### 데이터 분석 단계

데이터 분석과 관련된 프로세스에는 다음과 같은 여러 단계가 포함됩니다.

1. 첫 번째 단계는 데이터 요구 사항 또는 데이터 그룹화 방법을 결정하는 것입니다. 데이터는 연령, 인구 통계, 소득 또는 성별로 구분될 수 있습니다. 데이터 값은 숫자이거나 범주별로 나눌 수 있습니다.

1. 데이터 분석의 두 번째 단계는 데이터를 수집하는 프로세스입니다. 이것은 컴퓨터, 온라인 소스, 카메라, 환경 소스 또는 직원과 같은 다양한 소스를 통해 수행할 수 있습니다.

1. 데이터가 수집되면 분석할 수 있도록 구성해야 합니다. 이는 통계 데이터를 가져올 수 있는 스프레드시트 또는 기타 형태의 소프트웨어에서 발생할 수 있습니다.

1. 그런 다음 분석 전에 데이터를 정리합니다. 이는 중복이나 오류가 없는지, 불완전하지 않은지 확인하기 위해 스크러빙 및 검사됨을 의미합니다. 이 단계는 분석할 데이터 분석가에게 넘어가기 전에 오류를 수정하는 데 도움이 됩니다.

## 데이터 분석 유형

데이터 분석은 네 가지 기본 유형으로 나뉩니다.

1. **기술적 분석:** 주어진 기간 동안 발생한 일을 설명합니다. 조회수가 올랐나요? 이번 달 매출이 지난 달보다 더 강해졌습니까?

1. **진단 분석:** 문제가 발생한 이유에 더 중점을 둡니다. 여기에는 더 다양한 데이터 입력과 약간의 가설이 포함됩니다. 날씨가 맥주 판매에 영향을 미쳤습니까? 최신 마케팅 캠페인이 판매에 영향을 미쳤습니까?

1. **예측 분석:** 이것은 가까운 장래에 일어날 가능성이 높은 것으로 이동합니다. 우리가 더운 여름을 보냈을 때 판매는 어떻게 되었습니까? 얼마나 많은 기상 모델이 올해 더운 여름을 예측합니까?

1. **처방적 분석:** 이것은 행동 방침을 제안합니다. 이 5가지 기상 모델의 평균으로 더운 여름이 올 가능성이 58% 이상으로 측정되면 양조장에 야간 근무를 추가하고 추가 탱크를 임대하여 생산량을 늘려야 합니다.

[Six Sigma](/six-sigma) 프로그램 을 포함하여 금융 세계의 많은 품질 관리 시스템을 뒷받침합니다 . 무게나 생산 라인의 백만 개당 결함 수 등을 적절하게 측정하지 않으면 최적화가 거의 불가능합니다.

데이터 분석 사용을 채택한 일부 [부문 에는 빠른 처리가 가능한 여행 및 접객 산업이 포함됩니다.](/sector) 이 업계는 고객 데이터를 수집하고 문제가 있는 경우 문제가 있는 위치와 해결 방법을 파악할 수 있습니다.

헬스케어는 대량의 정형 및 비정형 데이터 사용을 결합하고 데이터 분석을 사용하여 신속한 결정을 내립니다. 유사하게, 소매 산업은 끊임없이 변화하는 쇼핑객의 요구를 충족시키기 위해 방대한 양의 데이터를 사용합니다. 소매업체가 수집 및 분석하는 정보는 트렌드를 파악하고 제품을 추천하며 수익을 높이는 데 도움이 됩니다.

> 2021년 12월 기준으로 미국 데이터 분석가의 평균 총액은 93,000달러를 약간 넘었습니다.

## 데이터 분석 기법

데이터 분석가가 데이터를 처리하고 정보를 추출하는 데 사용할 수 있는 여러 가지 분석 방법과 기술이 있습니다. 가장 인기 있는 방법이 아래에 나열되어 있습니다.

- [회귀 분석](/regression) 은 종속 변수 간의 관계를 분석하여 하나의 변화가 다른 변수의 변화에 어떻게 영향을 미칠 수 있는지 결정하는 것을 수반합니다.

- [요인 분석](/random-factor-analysis) 은 큰 데이터 세트를 가져와 더 작은 데이터 세트로 축소하는 것을 수반합니다. 이 책략의 목표는 그렇지 않았다면 더 보기 어려웠을 숨겨진 경향을 발견하려고 시도하는 것입니다.

- **동질 집단 분석**은 데이터 세트를 유사한 데이터 그룹으로 나누는 프로세스이며, 종종 고객 인구 통계로 나뉩니다. 이를 통해 데이터 분석가 및 기타 데이터 분석 사용자는 데이터의 특정 하위 집합과 관련된 수치를 더 자세히 알아볼 수 있습니다.

- [몬테카를로 시뮬레이션](/montecarlosimulation) 은 다양한 결과가 발생할 확률을 모델링합니다. 종종 위험 완화 및 손실 방지에 사용되는 이러한 시뮬레이션은 여러 값과 변수를 통합하며 다른 데이터 분석 접근 방식보다 더 나은 예측 기능을 제공하는 경우가 많습니다.

- **시계열 분석**은 시간 경과에 따른 데이터를 추적하고 데이터 포인트 값과 데이터 포인트 발생 간의 관계를 확고히 합니다. 이 데이터 분석 기법은 일반적으로 주기적인 추세를 파악하거나 재무 예측을 예측하는 데 사용됩니다.

## 데이터 분석 도구

데이터 분석은 숫자를 계산하기 위한 광범위한 수학적 및 통계적 접근 방식과 함께 기술 능력 면에서 빠르게 발전했습니다. 오늘날 데이터 분석가는 데이터 수집, 정보 저장, 데이터 처리 및 결과 보고에 도움이 되는 광범위한 소프트웨어 도구를 보유하고 있습니다.

데이터 분석은 항상 스프레드시트 및 Microsoft Excel과 느슨한 관계를 유지해 왔습니다. 이제 데이터 분석가는 종종 원시 프로그래밍 언어와 상호 작용하여 데이터베이스를 변환하고 조작합니다. Python과 같은 [오픈 소스 언어가 자주 활용됩니다.](/open-source) R과 같은 데이터 분석을 위한 보다 구체적인 도구는 통계 분석 또는 그래픽 모델링에 사용할 수 있습니다.

데이터 분석가는 또한 결과를 보고하거나 전달할 때 도움을 받습니다. Tableau와 Power BI는 모두 정보를 컴파일하고, 데이터 분석을 수행하고, 대시보드와 보고서를 통해 결과를 배포하는 데이터 시각화 및 분석 도구입니다.

데이터 분석가를 지원하는 다른 도구도 등장하고 있습니다. SAS는 [데이터 마이닝](/datamining) 을 지원할 수 있는 분석 플랫폼 이며 Apache Spark는 대규모 데이터 집합을 처리하는 데 유용한 오픈 소스 플랫폼입니다. 이제 데이터 분석가는 회사에 제공하는 가치를 더욱 향상시킬 수 있는 광범위한 기술 능력을 갖게 되었습니다.

## 결론

정보에 점점 더 의존하고 통계를 수집하는 세상에서 데이터 분석은 개인과 조직이 데이터를 확인하는 데 도움이 됩니다. 다양한 도구와 기술을 사용하여 일련의 원시 숫자를 의사 결정과 사려 깊은 관리를 주도하는 유익하고 교육적인 통찰력으로 변환할 수 있습니다.

## 하이라이트

- 데이터 분석은 원시 데이터를 분석하여 해당 정보에 대한 결론을 내리는 과학입니다.

- 데이터 분석에 대한 다양한 접근 방식에는 발생한 일(기술적 분석), 발생한 이유(진단 분석), 일어날 일(예측 분석) 또는 다음에 수행해야 할 작업(처방적 분석)이 포함됩니다.

- 데이터 분석은 기업이 성과를 최적화하고, 더 효율적으로 수행하고, 수익을 극대화하거나, 보다 전략적으로 결정을 내리는 데 도움이 됩니다.

- 데이터 분석의 기술과 프로세스는 인간이 소비하기 위해 원시 데이터에 대해 작동하는 기계적 프로세스 및 알고리즘으로 자동화되었습니다.

- 데이터 분석은 최고의 데이터 조작을 위해 스프레드시트, 데이터 시각화 및 보고 도구, 데이터 마이닝 프로그램 또는 오픈 소스 언어에 이르는 다양한 소프트웨어 도구에 의존합니다.

## 자주하는 질문

### 데이터 분석이 왜 중요한가요?

데이터 분석은 기업이 성과를 최적화하는 데 도움이 되기 때문에 중요합니다. 이를 비즈니스 모델에 구현한다는 것은 기업이 비즈니스를 수행하는 보다 효율적인 방법을 식별하여 비용을 절감하는 데 도움이 될 수 있음을 의미합니다. 기업은 또한 데이터 분석을 사용하여 더 나은 비즈니스 결정을 내리고 고객 동향과 만족도를 분석하여 새롭고 더 나은 제품과 서비스를 제공할 수 있습니다.

### 데이터 분석의 4가지 유형은 무엇입니까?

데이터 분석은 네 가지 기본 유형으로 나뉩니다. 기술 분석은 주어진 기간 동안 발생한 일을 설명합니다. 진단 분석은 문제가 발생한 이유에 더 중점을 둡니다. 예측 분석은 가까운 시일 내에 일어날 가능성이 있는 것으로 이동합니다. 마지막으로 처방적 분석은 행동 방침을 제시합니다.

### 누가 데이터 분석을 사용하고 있습니까?

데이터 분석은 턴어라운드가 빠른 여행 및 숙박 산업과 같은 여러 부문에서 채택되었습니다. 이 업계는 고객 데이터를 수집하고 문제가 있는 경우 문제가 있는 위치와 해결 방법을 파악할 수 있습니다. 의료는 대량의 정형 및 비정형 데이터 사용을 결합한 또 다른 분야이며 데이터 분석은 빠른 결정을 내리는 데 도움이 될 수 있습니다. 유사하게, 소매 산업은 끊임없이 변화하는 쇼핑객의 요구를 충족시키기 위해 방대한 양의 데이터를 사용합니다.
