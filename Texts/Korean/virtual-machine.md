# 가상 기기
가상 머신(VM)은 컴퓨터 시스템을 에뮬레이트하는 프로그램입니다. 가상 CPU, 메모리 및 스토리지가 있으며 외부에서 동일한 하드웨어를 사용하는 물리적 시스템과 다르지 않은 것처럼 보입니다.

## 가상 머신은 어떻게 작동합니까?

가상 머신에는 운영 체제, 파일 및 애플리케이션이 있습니다. 실제 컴퓨터처럼 작동합니다. 그것은 웹사이트를 호스팅하고, 프로그램을 실행하고, 다른 시스템과 네트워크에 존재할 수 있습니다. 화면에서 볼 경우 가상 머신은 해당 컴퓨터의 자체 운영 체제 내에서 실행되는 창으로 나타납니다.

또한 CPU, 메모리 및 스토리지가 있지만 물리적이 아닌 시뮬레이션입니다. 이 때문에 여러 가상 머신이 하나의 서버에 공존할 수 있습니다. 이 경우 서버는 하이퍼바이저라는 소프트웨어를 사용하여 관리합니다. 하이퍼바이저는 VM의 가상 하드웨어를 서버의 물리적 구성 요소에 매핑합니다.

이 시나리오에서 하나의 물리적 서버에는 2, 3, 4 또는 10개의 가상 머신이 실행되고 있을 수 있습니다. 그러나 그 수는 서버의 리소스에 의해 제한됩니다. VM이 너무 많으면 성능이 저하됩니다.

## 가상 머신이 사용되는 이유는 무엇입니까?

가상 머신은 시스템의 나머지 부분과 별도로 코드를 실행할 수 있는 가상 공간의 일부인 샌드박스입니다. 따라서 VM은 새 소프트웨어를 테스트하거나 악성일 수 있는 코드를 실행하는 데 이상적입니다.

가상 머신은 사용자가 컴퓨터 운영 체제에서 작동하지 않는 소프트웨어를 실행할 수 있도록 다른 또는 이전 OS를 실행할 수 있습니다. 예를 들어 누군가가 VM을 사용하여 Linux 장치에서 Windows 소프트웨어를 실행할 수 있습니다. 이를 달성하기 위해 Linux 시스템에서 Windows 운영 체제를 실행하는 가상 머신을 실행합니다.

## 주요 예: 이더리움 가상 머신(EVM)

이더리움 네트워크의 노드는 모두 스마트 계약을 실행하는 데 사용되는 이더리움 가상 머신(EVM)을 실행합니다. 이는 별도의 샌드박스에서 발생하기 때문에 이더리움 플랫폼에 상당한 보안 이점을 제공하여 많은 해커가 선택하는 전략인 DDoS(분산 서비스 거부 공격)로부터 면역이 됩니다.
