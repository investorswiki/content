# 연속 컴파운딩
## 연속 합성이란 무엇입니까?

연속 복리는 이론적으로 무한한 기간 동안 계산되어 계정 잔액에 재투자되는 경우 복리 [이자 가 도달할 수 있는 수학적 한계입니다.](/interest) 이것이 실제로는 불가능하지만 연속 복리 이자의 개념은 금융에서 중요합니다. 대부분의 이자가 월별, 분기별 또는 반기별로 복리 처리되기 때문에 [복리](/compounding) 의 극단적인 경우입니다 .

## 연속 합성의 공식 및 계산

연간 또는 월간과 같이 유한한 기간에 대한 이자를 계산하는 대신 연속 복리는 무한한 기간에 걸쳐 일정한 복리를 가정하여 이자를 계산합니다. 유한한 기간에 대한 복리 이자의 공식은 다음 네 가지 변수를 고려합니다.

- PV = 투자의 현재 가치

- i = [명시된](/stated-annual-interest-rate) 이자율

- n = 복리 기간의 수

- t = 년 단위의 시간

연속 복리 계산 공식은 이자 발생 투자의 미래 가치 공식에서 파생됩니다.

미래 가치(FV) = PV x [1 + (i / n)]^(nxt)^

n이 무한대에 접근할 때 이 공식의 한계를 계산하면(연속 복리의 정의에 따라) 연속 복리 이자의 공식이 나옵니다.

**FV = PV xe(ixt)**, 여기서 e는 2.7183에 가까운 수학 상수입니다.

## 연속 합성이 알려줄 수 있는 것

이론적으로 연속 복리 이자는 계정 잔액이 지속적으로 이자를 얻고 이 이자를 잔액에 다시 공급하여 이자를 얻음을 의미합니다.

연속 복리는 이자가 무한한 기간 동안 복리될 것이라는 가정 하에 이자를 계산합니다. 연속 복리가 필수적인 개념이지만 현실 세계에서는 이자가 계산되고 지급되는 기간이 무한대로 존재하는 것이 불가능합니다. 결과적으로 이자는 일반적으로 월별, 분기별 또는 연간과 같은 고정 기간을 기준으로 복리됩니다.

> 투자금액이 매우 높아도 연속복리로 얻는 총 이자의 차이는 기존 복리기간에 비해 그리 크지 않습니다.

>

## 연속 합성을 사용하는 방법의 예

예를 들어 $10,000 투자가 내년에 15%의 이자를 얻는다고 가정합니다. 다음 예는 이자가 연, 반기, 분기, 월, 일 및 지속적으로 복리될 때 투자의 최종 가치를 보여줍니다.

- **연간 복리:** FV = $10,000 x (1 + (15% / 1)) ^(1 x 1)^ = $11,500

- **반기 복리:** FV = $10,000 x (1 + (15% / 2)) ^(2 x 1)^ = $11,556.25

- **분기 복리:** FV = $10,000 x (1 + (15% / 4)) ^(4 x 1)^ = $11,586.50

- **월별 복리:** FV = $10,000 x (1 + (15% / 12)) ^(12 x 1)^ = $11,607.55

- **일일 복리:** FV = $10,000 x (1 + (15% / 365)) ^(365 x 1)^ = $11,617.98

- **연속 복리:** FV = $10,000 x 2.7183 ^(15% x 1)^ = $11,618.34

일일 복리로 얻은 총 이자는 $1,617.98이고 연속 복리로 얻은 총 이자는 $1,618.34로 한계 차이입니다.

## 하이라이트

- 연속복리이자를 계산하는 공식은 4가지 변수를 고려합니다.

- 대부분의 이자는 반기, 분기 또는 월 단위로 복리됩니다.

- 연속복리이자의 개념은 현실적으로 불가능하지만 금융에서는 중요하다.

- 연속 복리 이자는 이자가 복리되어 무한대로 잔액에 다시 추가된다고 가정합니다.
