# 선 그래프
## 선 그래프란 무엇입니까?

선 그래프(선 그림 또는 선 차트라고도 함)는 선을 사용하여 개별 데이터 요소를 연결하는 그래프입니다. 선 그래프는 지정된 시간 간격에 대한 정량적 값을 표시합니다. 금융에서 꺾은선형 차트는 일반적으로 자산 또는 유가 증권의 역사적 가격 움직임을 묘사하는 데 사용됩니다.

[막대 차트](/barchart),. 파이 차트 및 (거래 중) [촛대 차트](/candlestick) 를 비롯한 데이터의 다른 시각화와 비교할 수 있습니다 .

## 선 그래프 이해하기

선 그래프는 직선으로 연결된 데이터 포인트 "마커"를 사용합니다. 직선으로 연결된 이러한 데이터 포인트는 시각화에 도움이 됩니다. 선 그래프는 다양한 목적을 위해 다양한 필드에서 사용되지만 시간 경과에 따른 값의 변화를 그래픽으로 표현해야 할 때 특히 유용합니다.

선 그래프는 [증권 가격](/security),. 회사 수익 시트 및 주요 주가 [지수 의 이력을 포함하여 시간 경과에 따른 가치의 시각적 표현을 만들기 위해 재무에서 자주 사용됩니다](/index). 그들은 또한 다른 유가 증권을 비교하는 데 유용합니다.

[기술 분석](/technicalanalysis) 분야와 관련하여 투자자는 선 그래프를 사용하여 추세를 시각화하여 분석에 큰 도움을 줄 수 있습니다.

선 그래프에는 몇 가지 제한 사항이 있습니다. 예를 들어, 선 그래프는 데이터 포인트가 너무 많으면 명확성을 잃는 경우가 많습니다. 또한 특정 효과를 얻기 위해 시각적으로 쉽게 조작할 수 있습니다. 예를 들어, 축의 데이터 포인트 범위를 조정하여 겉보기 변화 정도를 시각적으로 조작할 수 있습니다.

> 선 그래프는 수동으로 구성하거나 Microsoft Excel과 같은 소프트웨어를 사용하여 구성할 수 있으므로 최종 제품의 속도와 정확도가 크게 향상됩니다.

>

## 선 그래프 만들기

선 그래프는 x축(가로)과 y축(세로)의 두 축으로 구성됩니다. 각 축은 서로 다른 데이터 유형을 나타내며 교차하는 지점은 (0,0)입니다. x축은 해당 값이 측정된 항목에 종속되지 않기 때문에 독립 축입니다. y축은 해당 값이 x축의 값에 따라 달라지기 때문에 종속 축입니다.

각 축은 해당 축을 따라 측정된 데이터에 따라 레이블이 지정되어야 합니다. 그런 다음 각 축을 적절한 증분으로 나누어야 합니다(예: 첫째 날, 둘째 날 등). 예를 들어, 이전 2주 동안의 주식 가격 변동을 측정하는 경우 x축은 측정된 시간(기간 내 거래일)을 나타내고 y축은 [주가 를 나타냅니다](/stock).

선 그래프를 사용하여 주식 가격을 추적할 때 가장 일반적으로 사용되는 데이터 포인트는 주식의 [종가](/closingprice) 입니다.

예를 들어, 거래 첫날에 주어진 주식의 가격이 $30이고 데이터 포인트가 (1, $30)에 있다고 가정합니다. 거래 2일차에 주식 가격은 $35였으며 데이터 포인트는 (2, $35)였습니다.

각 데이터 포인트는 시간에 따른 값의 변화를 시각적으로 보여주는 선으로 그려지고 연결됩니다. 주식의 가치가 매일 증가하면 선이 위쪽으로 오른쪽으로 기울어집니다. 반대로, 주식의 가격이 꾸준히 하락하고 있다면 선은 오른쪽으로 아래로 기울어질 것입니다.

## 하이라이트

- 투자에서 기술적 분석 분야에서 선 그래프는 사용자가 추세를 시각화할 수 있도록 하는 데 매우 유용합니다.

- 선 그래프는 다양한 목적으로 여러 분야에서 사용되지만 가장 일반적인 기능은 시간 경과에 따른 값의 변화를 그래픽으로 표현하는 것입니다.

- 선 그래프는 x축(가로)과 y축(세로)의 두 축으로 구성되며 그래픽으로 (x,y)로 표시됩니다.

- 선 그래프는 일반적으로 지정된 시간 간격 동안 정량적 값을 표시하는 개별 데이터 요소를 연결합니다.

- 금융에서 선 그래프는 유가 증권 가격의 변화를 포함하여 시간 경과에 따른 가치의 시각적 표현을 만드는 데 사용됩니다.

## 자주하는 질문

### 선 그래프는 금융에서 어떻게 유용합니까?

선 그래프는 시간 경과에 따른 추세를 시각적으로 표현하는 데 매우 효과적이기 때문에 재무에 유용합니다. 이러한 이유로 특정 기간 동안 주식의 성과를 나타내는 데 자주 사용됩니다.

### Excel에서 선 그래프를 어떻게 만드나요?

Excel에서 선 그래프를 사용하여 시간 경과에 따른 추세를 표시할 수 있습니다. Excel에서 가로축(x축)에 텍스트 레이블, 날짜 또는 몇 개의 숫자 레이블이 있는 경우 선 그래프가 적합합니다. Excel에서 선 그래프를 만드는 단계는 다음과 같습니다. (숫자 레이블을 사용하는 경우 꺾은선형 차트를 만들기 전에 A1 셀을 비워둡니다):1. 값을 입력한 후 범위(해당 값을 포함하는 범위)를 선택합니다. 예: A1:D7.1. 삽입 탭의 차트 그룹에서 선 기호("선 차트 삽입")1를 클릭합니다. "마커가 있는 선"을 클릭합니다.

### 그래프 이론에서 선 그래프란 무엇입니까?

그래프 이론은 수학 분야입니다. 그래프 이론은 특히 개체 간의 쌍 관계를 모델링하는 데 사용되는 수학적 구조인 그래프를 연구합니다. 간선 그래프, 교환 그래프, 인접 그래프 및 파생 그래프입니다. 이것은 선 그래프의 공식적인 정의입니다. 주어진 그래프(**G)**, 해당 선 그래프(**L**(** G**))는 **L**(**G**)의 각 꼭지점이 **G**의 간선을 나타내고, **L**(**G**의 두 꼭짓점을 나타내는 그래프입니다. ) 해당 모서리가 **G**에서 공통 끝점을 공유하는 경우 인접합니다. 선 그래프는 **G** 가장자리의 교차 그래프이며 두 끝점 집합으로 각 가장자리를 나타냅니다.

### Google 스프레드시트에서 선 그래프를 만들려면 어떻게 해야 하나요?

Google 스프레드시트에서 선 그래프를 만들면 시간 경과에 따른 데이터 추세를 식별하는 데 도움이 됩니다. 다음은 선 그래프를 만들 수 있도록 Google 스프레드시트에서 데이터 형식을 지정하는 단계입니다.1. 첫 번째 열에 데이터를 설명하는 레이블을 입력합니다. 첫 번째 열의 레이블이 가로 축1에 표시됩니다. 범주 이름을 포함하려면 각 열의 첫 번째 행에 범주 이름을 입력합니다. 첫 번째 행의 항목은 범례1에 레이블로 표시됩니다. 다른 모든 열에는 숫자 데이터를 입력합니다. 카테고리 이름1을 추가하도록 선택할 수도 있습니다. 다른 셀의 경우 표시하려는 데이터 포인트를 입력1. 시트의 모든 행은 차트의 다른 선을 나타냅니다. 꺾은선형 차트를 맞춤설정하는 방법에 대한 자세한 내용은 Google 스프레드시트를 비롯한 모든 도구에 대한 도움말 센터를 제공합니다.

### 선 그래프를 어떻게 설명합니까?

선 그래프는 시간이 지남에 따라 지속적으로 변경되는 정보를 그래픽으로 표시한 것입니다. 선 그래프에는 데이터 요소가 나타내는 값의 지속적인 변화를 나타내는 직선으로 연결된 다양한 데이터 요소가 있습니다.

### 선 그래프는 무엇에 사용됩니까?

선 그래프는 다양한 기간 동안의 변화를 추적하는 데 사용됩니다. 선 그래프는 비교 도구로 사용할 수도 있습니다. 즉, 둘 이상의 그룹에 대해 같은 기간 동안의 변화를 비교하는 것입니다.
