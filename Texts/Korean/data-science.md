# 데이터 과학
## 데이터 과학이란 무엇입니까?

[빅 데이터](/big-data) 를 기반으로 유용한 정보를 제공하는 응용 수학과 [통계 의 한 분야입니다](/statistics).

데이터 과학 또는 데이터 기반 과학은 의사 결정 목적을 위해 데이터 뭉치를 해석하기 위해 계산의 도움으로 다양한 분야의 측면을 결합합니다.

## 데이터 과학 이해

데이터는 휴대폰, 소셜 미디어, 전자 상거래 사이트, 의료 설문 조사 및 인터넷 검색을 포함한 다양한 부문, 채널 및 플랫폼에서 가져옵니다. 사용 가능한 데이터의 양이 증가하면서 빅 데이터를 기반으로 하는 새로운 연구 분야, 즉 모든 [부문](/sector) 에서 더 나은 운영 도구를 만드는 데 기여하는 방대한 데이터 세트가 열립니다 .

기술 및 수집 기술의 발전으로 인해 지속적으로 증가하는 데이터 액세스가 가능합니다. 개인의 구매 패턴과 행동을 모니터링하고 수집된 정보를 기반으로 예측할 수 있습니다.

그러나 끊임없이 증가하는 데이터는 비정형이며 효과적인 의사 결정을 위해 구문 분석이 필요합니다. 이 프로세스는 기업에게 복잡하고 시간이 많이 소요되므로 데이터 과학이 등장했습니다.

> ### 데이터 과학의 목적

> 데이터 과학 또는 데이터 기반 과학은 빅 데이터와 머신 러닝을 사용하여 의사 결정을 위해 데이터를 해석합니다.

>

## 데이터 과학의 간략한 역사

"데이터 과학"이라는 용어는 "컴퓨터 과학"과 동의어로 사용되었던 1960년대 초부터 사용되었습니다. 나중에 이 용어는 다양한 응용 프로그램에서 사용되는 데이터 처리 방법에 대한 조사를 정의하기 위해 구별되었습니다.

2001년 William S. Cleveland는 독립적인 분야를 지칭하기 위해 "데이터 과학"이라는 용어를 처음으로 사용했습니다. Harvard Business Review는 2012년에 "21세기의 가장 섹시한 직업"으로 데이터 과학자의 역할을 설명하는 기사를 발표했습니다.

## 데이터 과학이 적용되는 방식

데이터 과학은 데이터 세트를 수집하고, 처리하고, 데이터 세트에서 통찰력을 도출하고, 세트에서 의미 있는 데이터를 추출하고, 의사 결정을 위해 해석하기 위해 여러 분야의 도구를 통합합니다. 데이터 과학 분야를 구성하는 학문 분야에는 마이닝, 통계, 기계 학습, 분석 및 프로그래밍이 포함됩니다.

[데이터 마이닝](/datamining) 은 복잡한 데이터 세트에 알고리즘을 적용하여 패턴을 드러낸 다음 세트에서 유용하고 관련성 높은 데이터를 추출하는 데 사용됩니다. 통계 측정 또는 [예측 분석](/predictive-analytics) 은 이 추출된 데이터를 사용하여 데이터가 과거에 일어난 일을 기반으로 미래에 발생할 가능성이 있는 이벤트를 측정합니다.

기계 학습은 인간이 평생 처리할 수 없는 대량의 데이터를 처리하는 인공 지능 도구입니다. 머신 러닝은 이벤트가 발생할 가능성을 예측된 시간에 실제로 발생한 것과 일치시켜 예측 분석에서 제시된 의사 결정 모델을 완성합니다.

[알고리즘](/algorithm) 을 사용하여 기계 학습 단계에서 구조화된 데이터를 수집하고 처리합니다 . 분석가는 의사 결정 팀이 이해할 수 있는 응집력 있는 언어로 데이터를 해석, 변환 및 요약합니다. 데이터 과학은 거의 모든 상황에 적용되며 데이터 과학자의 역할이 진화함에 따라 데이터 아키텍처, 데이터 엔지니어링 및 데이터 관리를 포함하도록 해당 분야가 확장될 것입니다.

> ### 빠른 사실

> 미국 노동 통계국(US Bureau of Labor Statistics)에 따르면 컴퓨터 및 정보 연구 과학자에 대한 수요는 2019년에서 2029년까지 15% 증가할 것으로 예상되며 이는 다른 직종보다 훨씬 빠릅니다.

>

## 데이터 과학자

데이터 과학자는 많은 경우 회사 운영을 개선하기 위해 많은 양의 데이터를 수집, 분석 및 해석합니다. 데이터 과학자 전문가는 데이터를 분석하고 데이터 세트의 패턴, 추세 및 관계를 감지하는 통계 모델을 개발합니다. 이 정보는 소비자 행동을 예측하거나 비즈니스 및 운영 위험을 식별하는 데 사용할 수 있습니다.

데이터 과학자의 역할은 종종 의사 결정자에게 문제 해결에 적용할 수 있고 이해할 수 있는 방식으로 데이터 통찰력을 제시하는 스토리텔러의 역할입니다.

## 오늘날의 데이터 과학

기업은 소비자에게 가치를 제공하기 위해 일상 활동에 빅 데이터와 데이터 과학을 적용하고 있습니다. 은행 기관은 사기 탐지 성공을 향상시키기 위해 빅 데이터를 활용하고 있습니다. 자산 관리 회사는 빅 데이터를 사용하여 지정된 시간에 유가 증권 가격이 오르거나 내릴 가능성을 예측합니다.

Netflix와 같은 회사는 빅 데이터를 마이닝하여 사용자에게 제공할 제품을 결정합니다. 또한 Netflix는 알고리즘을 사용하여 시청 기록을 기반으로 사용자에게 맞춤 추천을 제공합니다. 데이터 과학은 빠른 속도로 진화하고 있으며 그 응용 프로그램은 미래의 삶을 계속 변화시킬 것입니다.

## 하이라이트

- 데이터 과학은 기계 학습 및 인공 지능과 같은 기술을 사용하여 의미 있는 정보를 추출하고 미래의 패턴 및 행동을 예측합니다.

- 기술이 발전하고 빅데이터 수집 및 분석 기술이 고도화됨에 따라 데이터 사이언스 분야가 성장하고 있습니다.

- 기술의 발전, 인터넷, 소셜 미디어 및 기술의 사용은 모두 빅 데이터에 대한 액세스를 증가시켰습니다.

## 자주하는 질문

### 데이터 과학의 단점은 무엇입니까?

[Cambridge Analytica](/cambridge-analytica) 와 같은 여러 스캔들에 비추어 비판을 받았습니다.

### 데이터 과학은 무엇에 유용합니까?

데이터 과학은 구조화되지 않았거나 관련이 없어 보이는 데이터에서 패턴을 식별하여 추론과 예측을 가능하게 합니다. 사용자 데이터를 수집하는 기술 회사는 기술을 사용하여 수집된 정보를 유용하거나 수익성 있는 정보의 소스로 전환할 수 있습니다.

### 모든 과학이 데이터를 사용하지 않습니까?

네, 모든 경험과학은 데이터를 수집하고 분석합니다. 데이터 과학을 구분하는 것은 빅 데이터 세트를 처리하고 분석하기 위해 정교한 계산 방법과 기계 학습 기술을 사용하는 것을 전문으로 한다는 것입니다. 종종 이러한 데이터 세트는 너무 크거나 복잡하여 기존 방법을 사용하여 적절하게 분석할 수 없습니다.
