# 블록 보상
블록 보상은 채굴자가 새로운 블록을 성공적으로 검증했을 때 채굴자에게 보상되는 암호화폐를 말합니다. 블록 보상은 블록 보조금과 거래 수수료의 두 가지 구성 요소로 구성됩니다. 블록 보조금은 새로 생성된 코인으로 구성되며 블록 보상의 가장 큰 부분을 차지합니다. 다른 부분은 [블록](/block) 에 포함된 트랜잭션에 의해 지불된 모든 수수료로 구성 됩니다.

블록 보상은 거의 전적으로 블록 보조금으로 이루어지기 때문에 블록 보상으로 언급하면서 블록 보조금에 대해 이야기하는 사람들을 보는 것은 매우 일반적입니다. 따라서 대중적인 용어로 "블록 보상"이라는 용어는 수수료를 설명하지 않습니다.

[비트코인](/bitcoin) 의 경우 블록 보조금은 50 BTC에서 시작하여 210,000 블록(약 4년에 한 번)마다 절반으로 감소하고 있습니다. 이러한 프로세스를 비트코인 [반감기 라고](/halving) 합니다. 비트코인의 블록 보조금은 2012년 25 BTC, 2016년 12.5 BTC로 줄었습니다. 다음 [반감기](/halving) 는 2020년 5월에 있을 것으로 예상됩니다.

새로 생성된 코인은 코인베이스 트랜잭션이라는 특수한 종류의 트랜잭션에 의해 생성됩니다. 일반적으로 코인베이스 트랜잭션은 블록에 추가되는 첫 번째 트랜잭션이며, 코인이 하나의 공백 입력에서 나오기 때문에 기본적으로 무에서 코인을 생성합니다.

##하이라이트

- 금액은 2140년경에 0에 도달할 것으로 예상됩니다.

- 블록 보상은 해당 통화 블록을 성공적으로 채굴할 경우 얻는 비트코인 수를 나타냅니다.

- 210,000 블록 생성 후 또는 대략 4년마다 보상 금액이 절반으로 줄어듭니다.
