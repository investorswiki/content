# 아크 탄성
## 아크탄성이란?

호 탄성은 주어진 두 점 사이에서 다른 변수에 대한 한 변수 의 [탄성 입니다.](/elasticity) 두 변수 간의 관계를 정의하는 일반적인 기능이 없을 때 사용합니다.

호 탄성은 곡선의 두 점 사이의 탄성으로도 정의됩니다. 이 개념은 수학과 경제학에서 모두 사용됩니다.

## 수요의 아크 가격 탄력성을 계산하는 방법

제품 가격이 10달러에서 8달러로 하락하여 수요량이 40개에서 60개로 증가하면 수요의 가격 탄력성은 다음과 같이 계산할 수 있습니다.

- **수요량의 변화율** = (Qd~2~ – Qd~1~) / Qd~1~ = (60 – 40) / 40 = 0.5

- **가격 변동률** = (P~2~ – P~1~) / P~1~ = (8 – 10) / 10 = -0.2

- 따라서 **PEd** = 0.5 / -0.2 = 2.5

의 [절대값](/absolute-value) 을 고려하기 때문에 음수 부호는 무시됩니다. 가격이 $10에서 $8로 하락할 때 이 재화의 가격 탄력성은 2.5라는 결론을 내릴 수 있습니다.

## 아크 탄성은 무엇을 말합니까?

경제학에서 수요의 탄력성을 계산하는 두 가지 방법이 있습니다. 수요의 가격(또는 점) 탄력성과 수요의 탄력성의 호입니다. 수요 의 아크 [가격 탄력성은 가격에 대한 수요량](/priceelasticity) 의 반응성을 측정합니다. [수요 곡선](/demand-curve) 의 특정 지점 또는 곡선의 두 지점 사이의 수요 탄력성을 취합니다.

## 수요의 아크탄력성

수요 공식의 가격탄력성의 문제점 중 하나는 가격이 오르느냐 내리느냐에 따라 다른 값을 준다는 것이다. 위의 예에서 다른 시작점과 끝점을 사용하는 경우(즉, 가격이 $8에서 $10으로 증가하고 수요량이 60에서 40으로 감소했다고 가정하면 Pe~d~는 다음과 같습니다.

- **수요량의 변화율** = (40 – 60) / 60 = -0.33

- **가격 변동률** = (10 – 8) / 8 = 0.25

- **PEd** = -0.33 / 0.25 = 1.32로 2.5와 많이 다릅니다.

이 문제를 해결하기 위해 아크 탄성을 사용할 수 있습니다. 호탄력성은 두 점 사이의 중간점을 이용하여 수요곡선에서 선택된 두 점 사이의 중간점에서의 탄력성을 측정한다. 수요의 호탄력성은 다음과 같이 계산할 수 있습니다.

- **Arc Ed** = [(Qd~2~ – Qd~1~) / 중점 Qd] ÷ [(P~2~ – P~1~) / 중점 P]

위에 제시된 예에 따라 호 탄성을 계산해 보겠습니다.

- **중간점 Qd** = (Qd~1~ + Qd~2~) / 2 = (40 + 60) / 2 = 50

- **중간 가격** = (P~1~ + P~2~) / 2 = (10 + 8) / 2 = 9

- **수요량의 % 변화** = (60 – 40) / 50 = 0.4

- **가격 변동률** = (8 – 10) / 9 = -0.22

- **호 에드** = 0.4 / -0.22 = 1.82

호탄력성을 사용하면 가격이 오르든 내리든 상관없이 호탄력성이 같은 값을 주기 때문에 어느 지점이 시작점이고 어느 지점이 끝점인지에 대해 걱정할 필요가 없습니다. 따라서 가격에 큰 변화가 있을 때 가격탄력성보다 호탄력성이 더 유용하다.

##하이라이트

- 호탄력성은 더 큰 가격 변화에 더 유용하며 가격이 오르든 내리든 동일한 탄력적 결과를 제공합니다.

- 호탄력성의 개념에서 탄력성은 그래프상의 수요곡선의 호에 걸쳐 측정됩니다.

- 아크 탄성 계산은 두 점 사이의 중간점을 사용하여 탄성을 제공합니다.
