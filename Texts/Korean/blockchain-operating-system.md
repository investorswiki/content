# 블록체인 운영 체제
## 블록체인 운영 체제란 무엇입니까?

블록체인 운영 체제는 컴퓨터 시스템 또는 플랫폼의 배경에서 실행되는 지원 시스템으로 블록체인을 사용합니다. 예를 들어, Android 모바일 또는 Windows PC는 스마트폰의 메모리 또는 PC의 하드 디스크에 해당 운영 체제(OS)의 로컬 설치가 필요하며 모든 트랜잭션 및 명령은 로컬에서 실행됩니다. 블록체인 기반 OS는 인증, 실행 및 기록이 블록체인에서 발생하는 사용자 장치의 모든 명령과 트랜잭션을 캡처합니다.

## 블록체인 운영 체제 이해

[비트코인](/bitcoin) 의 표준 결제 처리 시스템을 넘어 [암호 화폐](/cryptocurrency),. [블록체인](/blockchain) 은 기술 스택 전체에서 광범위하게 사용됩니다. [분산 원장 기술](/distributed-ledger-technology-dlt) 의 기술 개발 중 떠오르는 유행 은 블록체인 운영 체제입니다.

블록체인은 기본적으로 원장으로 전환된 트랜잭션 처리 엔진으로 작동합니다. 지불 처리가 필요하거나 [이더리움](/ethereum) 플랫폼의 최신 가제트로 크립토키티를 무장해야 하거나 포도원에서 VeChain 블록체인의 문앞까지 고가의 와인 배송을 추적하려는 경우 이러한 모든 애플리케이션 블록체인은 인증, 기록 및 처리 트랜잭션을 기반으로 합니다.

Microsoft Windows, Apple Mac 또는 [Android](/android-operating-system) 또는 [iOS 와 같은 모바일 시스템과 같은 모든 표준 운영 체제](/apple-ios) 도 모든 작업이 장치에서 로컬로 완료되는 마우스 클릭 또는 화면 탭을 통해 실행되는 사용자 명령을 기반으로 트랜잭션을 실행합니다. 동일한 개념이 장치 OS용 블록체인의 사용으로 확장되며, 여기서 운영 체제로 작동하는 데 사용하는 것이 더 효율적인 OS로 간주됩니다.

## 블록체인 OS 노력의 예

블록체인 기반 OS를 구축하려는 초기 시도는 모바일 및 스마트폰용으로 처음 등장했으며 클라우드 기반 가상 시스템이었습니다. 필요한 모든 트랜잭션 처리는 클라우드 호스팅 블록체인 기반 데이터 센터에서 발생하며 사용자는 장치 터치스크린의 탭을 통해 필요한 명령만 실행합니다.

예를 들어 홍콩에 본사를 둔 NYNJA Group Ltd. 블록체인 기반 NYNJA 가상 운영 체제(vOS)를 위해 Amgoo 스마트폰 제조업체와 전략적 협력 관계를 맺고 있습니다. 두 회사는 라틴 아메리카의 통신 사업자와 협력하여 활성화 시 NYNJA vOS 사용자에게 초기 데이터 블록을 제공합니다. vOS는 텍스트, 음성, 화상 회의 및 프로젝트 관리 도구를 제공하는 통신 계층, 상업 거래를 위한 보안 지불 계층, 비트코인, 이더리움 및 모든 ERC-20 호환 토큰을 지원하는 다중 통화 지갑을 지원합니다.

[공연 경제 직원을](/gig-economy) 할당하는 것과 같은 상업 활동을 위한 시장과 사용자가 상품을 사고 팔 수 있는 시장을 지원합니다. vOS는 NYNJAcoin 또는 NYN이라는 기본 암호화폐에 의해 지원됩니다.

## 특별 고려 사항

블록체인의 모든 이점과 이점은 블록체인 OS 사용자에게 제공될 것으로 기대됩니다.

사용자가 Android 또는 iOS 모바일, Windows 또는 Mac PC에서 수행하는 모든 작업은 OS 로그에 모든 사용자 활동을 기록할 수 있는 OS 제조업체뿐만 아니라 해당 앱, ISP에 의해 캡처되는 경향이 있습니다. 블록체인 기반 OS는 보안 및 개인 정보 보호, 규제 완화, 분산 OS 사용의 이점을 제공합니다.

이 개념은 여전히 발전하고 있으며 실제 사용은 제한적입니다. 그러나 장치 OS의 원활하고 어수선한 작업을 제공하는 데 성공하면 이러한 블록체인 OS에서 실행되는 장치가 점점 더 많아지는 것이 그리 멀지 않을 수 있습니다.

##하이라이트

- 처음에는 휴대폰 또는 기타 연결된 스마트 장치용 OS로 목표로 삼았던 블록체인 운영 체제는 높은 수준의 데이터 보안 및 사용자 익명성을 강조합니다.

- OS로 기능하기 위해서는 블록체인 프로토콜이 사용자의 컴퓨터 코드 및 명령 실행을 허용해야 합니다.

- 블록체인 운영 체제는 블록체인 원장 기술을 활용하여 컴퓨터 시스템 또는 네트워크를 전체 또는 부분적으로 실행합니다.
