# IP 주소
## IP 주소란 무엇입니까?

IP 주소는 인터넷 프로토콜 주소를 나타냅니다. 특정 컴퓨터 또는 컴퓨터 네트워크와 연결된 식별 번호입니다. 인터넷에 연결되면 IP 주소를 통해 컴퓨터가 정보를 보내고 받을 수 있습니다.

## IP 주소 작동 방식

IP 주소는 컴퓨터가 인터넷을 통해 데이터를 주고받을 수 있도록 합니다. 대부분의 IP 주소는 순전히 숫자이지만 인터넷 사용이 증가함에 따라 일부 주소에 문자가 추가되었습니다.

IP 주소에는 공개, 비공개, 정적 및 동적의 네 가지 유형이 있습니다. public 및 private은 네트워크의 위치를 나타내는 반면(private는 네트워크 내부에서 사용되고 public은 네트워크 외부에서 사용됨) 정적 및 동적은 영구성을 나타냅니다.

고정 IP 주소는 할당되지 않고 수동으로 생성된 주소입니다. 정적 주소도 변경되지 않는 반면 동적 IP 주소는 DHCP(Dynamic Host Configuration Protocol) 서버에 의해 할당되었으며 변경될 수 있습니다. 동적 IP 주소는 가장 일반적인 유형의 인터넷 프로토콜 주소입니다. 동적 IP 주소는 일정 시간 동안만 활성화되며 이후 만료됩니다. 컴퓨터는 자동으로 새 임대를 요청하거나 컴퓨터가 새 IP 주소를 받을 수 있습니다.

는 할당된 컴퓨터나 사용자에게 완전히 고유하므로 [사회 보장 번호 (SSN)](/ssn) 와 비교할 수 있습니다 . 이러한 번호를 생성하면 라우터가 인터넷에서 정보를 보내는 위치를 식별할 수 있습니다. 또한 올바른 장치가 전송되는 내용을 수신하고 있는지 확인합니다. 우체국이 소포를 배달하기 위해 우편 주소가 필요하듯이 라우터는 요청된 웹 주소로 배달하기 위해 IP 주소가 필요합니다.

## IP 주소의 예

다크 웹은 기존의 검색 엔진에 의해 인덱싱되지 않는 암호화된 온라인 콘텐츠를 나타냅니다. 다크 웹에는 범죄자들이 불법 및 불법 상품을 거래할 수 있는 불법 온라인 시장이 있습니다. 이러한 교환의 대부분은 온라인 암호 화폐 비트코인을 사용하여 이루어지므로 당국이 이러한 거래에 참여하는 사람들을 추적하고 포착하기가 더 어렵습니다.

2018년 1년간의 정부 작전 끝에 국토안보부 소속 요원들이 불법 무기 구매를 시도한 용의자의 컴퓨터에 접근하기 위해 무기 딜러로 위장했습니다. 이를 통해 다크 웹을 사용하는 추가 용의자의 지리적 위치를 추적하는 데 사용한 IP 주소에 액세스할 수 있었습니다.

IP 주소가 체포된 것은 이번이 처음이 아닙니다. 2012년에 경찰은 IP 주소를 사용하여 해킹 그룹 Lulzsec의 구성원을 추적하고 체포했습니다. [인터넷 서비스 제공자(ISP)](/isp) 로부터 정보를 얻기 위해 영장을 사용하여 법 집행 요원은 해커의 물리적 주소를 추적하고 불법 인터넷 활동으로 그들을 체포할 수 있었습니다.

## 하이라이트

- 인터넷 프로토콜(IP) 주소는 컴퓨터가 정보를 주고받을 수 있도록 합니다.

- IP 주소를 사용하면 올바른 당사자가 정보를 보내고 받을 수 있습니다. 즉, 경우에 따라 사용자의 물리적 위치를 추적하는 데 사용할 수도 있습니다.

- IP 주소에는 공용, 사설, 정적 및 동적의 4가지 유형이 있습니다.

## 자주하는 질문

### IPv6 주소란 무엇입니까?

IPv4 주소는 점으로 구분된 1~3자리 숫자 4개로 구성됩니다. IPv6은 128비트 영숫자 값을 사용하여 엔드포인트 장치를 온라인으로 식별하는 최신 버전의 인터넷 주소 지정입니다. IPv4 주소는 208.80과 비슷할 수 있습니다. 154.224 및 IPv6 주소는 다음과 같을 수 있습니다. 2001:0db8:85a3:0000:0000:8a2e:0370:7334.IPv6을 사용하면 주소를 복제하지 않고도 훨씬 더 많은 연결된 장치가 인터넷에 존재할 수 있으므로 문제가 발생할 수 있습니다.

### 누군가가 자신의 IP 주소를 어떻게 "스푸핑"할 수 있습니까?

VPN(가상 사설망)을 통해 다른 곳에서 온 것처럼 IP 주소를 만들 수 있습니다. 이러한 서비스는 기본적으로 설정한 네트워크에 가상으로 연결하여 컴퓨터의 인터넷 주소를 마스킹합니다. 따라서 귀하의 IP 주소가 그들의 것으로 나타납니다. 또한 해커는 신뢰할 수 있는 소스를 모방하고 이를 수락하도록 악성 데이터 패킷 헤더의 소스 주소를 수정하는 보다 정교한 기술을 사용하여 IP 주소를 스푸핑할 수 있습니다.

### 내 IP 주소는 어떻게 알 수 있나요?

IP 주소를 확인하는 방법에는 여러 가지가 있습니다. 아마도 가장 간단한 방법은 Google과 같은 인터넷 검색 엔진에 "내 IP 주소는 무엇입니까?"를 입력하는 것입니다. Windows 사용자의 경우 [시작 > 설정 > 네트워크 및 인터넷]을 열고 연결된 Wi-Fi 네트워크를 선택하여 IP를 찾을 수 있습니다. 속성에서 "IPv4 주소" 옆에 나열된 IP 주소를 찾습니다.
