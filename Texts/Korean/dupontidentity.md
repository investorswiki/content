# 듀폰 아이덴티티
## 듀폰의 아이덴티티는 무엇입니까?

DuPont의 아이덴티티는 회사의 자기자본이익률(ROE)이 이윤, 총 자산 회전율, 자기자본 승수라는 세 가지 다른 비율의 곱으로 표현될 수 있음을 보여주는 표현입니다.

## 듀폰의 아이덴티티 이해하기

일반적으로 DuPont 분석으로 알려진 DuPont의 정체성은 1920년대에 이 아이디어를 사용하기 시작한 DuPont Corporation에서 유래했습니다. DuPont의 정체성은 ROE가 다음 세 가지에 의해 영향을 받는다는 것을 알려줍니다.

1. 이익률로 측정되는 [운영 효율성 ;](/efficiencyratio)

2. 총자산 회전율로 측정한 [자산이용](/assetturnover) [효율](/assetturnover) 그리고

3. 자본 승수로 측정되는 [재무 레버리지 .](/timevalueofmoney)

DuPont ID의 공식은 다음과 같습니다.

ROE = 이익 마진 x 자산 회전율 x 자본 승수

이 공식은 다음과 같이 더 세분화될 수 있습니다.

ROE = (순이익/매출) x (매출/총자산) x (총자산/주주)

> ROE가 불만족스러운 경우 DuPont의 아이덴티티는 분석가와 경영진이 비즈니스에서 성과가 저조한 부분을 찾는 데 도움이 됩니다.

>

## 듀폰 아이덴티티 계산 예

회사가 2년 동안 다음 재무 데이터를 보고한다고 가정합니다.

첫해 순이익 = $180,000

첫해 수익 = $300,000

첫해 총 자산 = $500,000

1년차 주주 자본 = $900,000

2년차 순이익 = $170,000

2년차 수익 = $327,000

2년차 총 자산 = $545,000

2년차 주주 자본 = $980,000

DuPont ID를 사용하여 각 연도의 ROE는 다음과 같습니다.

ROE 1년차 = ($180,000 / $300,000) x ($300,000 / $500,000) x ($500,000 / $900,000) = 20%

ROE 2년차 = ($170,000 / $327,000) x ($327,000 / $545,000) x ($545,000 / $980,000) = 17%

약간의 반올림으로 위의 두 ROE 계산은 다음과 같이 나뉩니다.

ROE 1년차 = 60% x 60% x 56% = 20%

ROE 2년차 = 52% x 60% x 56% = 17%

ROE가 2년차에 하락했음을 분명히 알 수 있습니다. 한 해 동안 순이익, 수익, 총 자산 및 주주 지분은 모두 가치가 변경되었습니다. 분석가나 관리자는 DuPont ID를 사용하여 이러한 감소의 원인을 분석할 수 있습니다. 여기에서 그들은 자기자본 배수와 총 자산 회전율이 2년 동안 정확히 일정하게 유지되었음을 알 수 있습니다. 이는 ROE 하락의 원인으로 이익률만 남게 된다. 수익 마진은 60%에서 52%로 떨어졌지만 실제로 수익은 2년 차에 증가했다는 사실은 회사가 일년 내내 비용과 비용을 처리하는 방식에 문제가 있음을 나타냅니다. 그런 다음 관리자는 이러한 통찰력을 사용하여 다음 해를 개선할 수 있습니다.
