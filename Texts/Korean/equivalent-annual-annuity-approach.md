# 등가 연간 연금 접근법(EAA)
## 등가 연간 연금 접근법이란 무엇입니까?

등가 연간 연금 접근법은 [자본 예산 책정 에서](/capitalbudgeting) [상호 배타적인](/mutuallyexclusive) 프로젝트와 불평등한 삶 을 비교 하는 데 사용되는 두 가지 방법 중 하나입니다. EAA 접근 방식은 연금인 경우 프로젝트 수명 기간 동안 프로젝트에서 생성된 일정한 연간 현금 흐름을 계산합니다. 삶이 불평등한 프로젝트를 비교할 때 투자자는 EAA가 더 높은 프로젝트를 선택해야 합니다.

## 등가 연간 연금 접근법(EAA) 이해

EAA 접근 방식은 3단계 프로세스를 사용하여 프로젝트를 비교합니다. 일정한 연간 현금 흐름의 현재 가치는 프로젝트의 [순 현재 가치 와 정확히 같습니다](/npv). 분석가가 가장 먼저 하는 일은 수명 기간 동안 각 프로젝트의 NPV를 계산하는 것입니다. 그 후, 그들은 연금의 현재 가치가 프로젝트의 NPV와 정확히 같도록 각 프로젝트의 EAA를 계산합니다. 마지막으로 분석가는 각 프로젝트의 EAA를 비교하여 EAA가 가장 높은 것을 선택합니다.

예를 들어 가중 평균 자본 비용이 10%인 회사가 [A](/wacc) [와](/wacc) B라는 두 프로젝트를 비교한다고 가정합니다. 프로젝트 A의 NPV는 $3백만이고 예상 수명은 5년이고 프로젝트 B의 NPV는 $2입니다. 백만과 3년의 예상 수명. 재무 계산기를 사용하여 프로젝트 A의 EAA는 $791,392.44이고 프로젝트 B의 EAA는 $804,229.61입니다. EAA 접근 방식에서 회사는 연간 연금 가치가 더 높기 때문에 프로젝트 B를 선택합니다.

## 특별 고려 사항

### 등가 연간 연금 접근법 계산

종종 분석가는 일반적인 현재 가치와 [미래 가치](/futurevalue) 함수를 사용하여 EAA를 찾는 재무 계산기를 사용합니다. 분석가는 스프레드시트 또는 일반 비재무 계산기에서 다음 공식을 사용하여 정확히 동일한 결과를 얻을 수 있습니다.

- C = (rx NPV) / (1 - (1 + r)^-n^ )

어디에:

- C = 등가 연금 현금 흐름

- NPV = 순 현재 가치

- r = 기간당 이자율

- n = 기간 수

예를 들어, 두 개의 프로젝트를 고려하십시오. 하나는 7년 임기와 NPV가 $100,000입니다. 다른 하나의 임기는 9년이고 NPV는 $120,000입니다. 두 프로젝트 모두 6%의 할인율로 할인됩니다. 각 프로젝트의 EAA는 다음과 같습니다.

- EAA 프로젝트 1 = (0.06 x $100,000) / (1 - (1 + 0.06)^-7^ ) = $17,914

- EAA 프로젝트 2 = (0.06 x $120,000) / (1 - (1 + 0.06)^-9^ ) = $17,643

프로젝트 1이 더 나은 옵션입니다.

##하이라이트

- 등가 연간 연금 접근법은 자본 예산 책정에서 상호 배타적 인 프로젝트와 불평등 한 삶을 비교하는 데 사용되는 두 가지 방법 중 하나입니다.

- 종종 분석가는 EAA를 찾기 위해 일반적인 현재 가치 및 미래 가치 함수를 사용하여 재무 계산기를 사용합니다.

- 불평등한 삶을 가진 프로젝트를 비교할 때 투자자는 더 높은 등가의 연간 연금을 선택해야 합니다.
