# 최소 제곱 기준
## 최소 제곱 기준은 무엇입니까?

최소 제곱 기준은 직선을 생성하는 데 사용된 데이터를 묘사할 때 직선의 정확도를 측정하는 데 사용되는 공식입니다. 즉, 공식이 가장 잘 맞는 선을 결정합니다.

이 수학 공식은 종속 변수의 동작을 예측하는 데 사용됩니다. 접근 방식은 최소 자승 회귀선이라고도 합니다.

## 최소 제곱 기준 이해하기

최소 제곱 기준은 수학 함수에 의해 생성된 [제곱합 을 최소화하여 결정됩니다.](/sum-of-squares) 제곱은 데이터 포인트와 회귀선 또는 데이터 세트의 평균값 사이의 거리를 제곱하여 결정됩니다.

최소 제곱 분석은 그래프에 그려진 데이터 포인트 세트로 시작합니다. 독립변수는 가로 x축에 표시되고 종속변수는 세로 y축에 표시됩니다. 분석가는 최소 제곱 공식을 사용하여 독립 변수와 종속 변수 간의 관계를 설명하는 가장 정확한 직선을 결정합니다.

### 최소 제곱의 일반적인 용도

새로운 금융 공학 기술과 더불어 컴퓨팅 성능의 발전으로 최소 자승법의 사용이 증가하고 기본 원칙이 확장되었습니다.

최소 제곱 및 관련 통계 방법은 수혜자가 항상 사용을 인식하지 못하는 경우에도 금융, 경제 및 투자 전반에 걸쳐 일반화되었습니다.

예를 들어, 현재 많은 투자 플랫폼에서 사용하는 로보어드바이저는 몬테카를로 시뮬레이션 기술을 사용하여 포트폴리오를 관리하지만, 이는 배후에서 이를 사용하는 계정 소유자의 시야에서 이루어지지 않습니다.

다른 응용 프로그램에는 수익 분포의 시계열 분석, 경제 예측 및 정책 전략, 고급 옵션 모델링이 포함됩니다.

### 최소제곱은 무엇을 말합니까?

수학자들은 방정식을 정확히 풀려고 하는 대신 [최소제곱법](/least-squares-method) 을 사용하여 근사값에 도달합니다. 이를 최대 가능성 추정이라고 합니다.

최소 제곱 접근 방식은 함수와 함수가 설명하는 데이터 포인트 사이의 거리를 제한합니다. 곡선이 데이터 세트에 맞는 [비선형 회귀 모델링에서](/nonlinear-regression) 종종 회귀 분석에 사용됩니다 .

> 수학자들은 최소 제곱법을 사용하여 최대 가능성 추정값에 도달합니다.

>

최소 제곱 접근 방식은 회귀 방정식을 결정하는 데 널리 사용되는 방법이며 응답 변수와 예측 변수 간의 관계를 알려줍니다.

함수를 곡선에 피팅할 때 자주 사용되는 모델링 방법에는 직선 방법, [다항식 방법](/polynomial_trending),. 로그 방법 및 가우스 방법이 있습니다.

선형 또는 일반 최소 제곱은 관측 및 실험 데이터를 분석하기 위해 가장 간단하고 가장 일반적으로 사용되는 선형 회귀 추정기입니다. 주어진 데이터 포인트 세트를 통해 가장 잘 맞는 직선을 찾습니다.

## 하이라이트

- 최소 제곱 결과를 사용하여 데이터를 요약하고 동일한 그룹 또는 시스템에서 관련되지만 관찰되지 않은 값에 대해 예측할 수 있습니다.

- 금융, 경제, 투자 전반에 걸쳐 최소제곱기준법을 사용합니다.

- 선을 만드는 데 사용된 데이터를 묘사할 때 선의 정확도를 추정하는 데 사용됩니다.
