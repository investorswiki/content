# Cifrar
En [criptografía](/cryptography),. un cifrado es una serie de instrucciones definidas que se pueden seguir para cifrar o descifrar un mensaje de texto. El proceso de [cifrado](/encryption) consiste en convertir información clara en una versión ilegible o inaccesible. El texto original que se puede entender claramente se conoce como texto sin formato, mientras que su forma cifrada se denomina texto cifrado. Ambos textos contienen la misma información, la única diferencia es que el texto cifrado está escrito en un formato que solo pueden leer o acceder aquellos que tienen el [mecanismo de](/decryption) descifrado [correcto .](/decryption)

La mayoría de los algoritmos de cifrado implican el uso de una pieza particular de información secreta, que comúnmente se conoce como clave criptográfica. El esquema de cifrado varía según el modelo de clave, y los algoritmos de cifrado se pueden definir como simétricos o asimétricos. Los cifrados simétricos usan solo una clave para el cifrado y el descifrado, mientras que los cifrados asimétricos usan diferentes claves para cada operación.

Aunque la mayoría de las técnicas modernas de encriptación son realizadas por computadoras, las cifras ya se usaban para codificar mensajes desde antes de los antiguos griegos, alrededor del año 400 a. El conocido político romano Julio César hizo un gran uso de los cifrados de sustitución, reemplazando cada letra en un mensaje con la letra que estaba ubicada tres lugares más abajo en el alfabeto.

Por ejemplo, si usamos la misma técnica para encriptar la palabra BINANCE, el texto cifrado resultante sería ELQDQFH. Este mensaje parece no tener sentido para un tercero, pero el destinatario puede descifrarlo fácilmente, ya que ya conoce el número de turno empleado por el remitente para cifrarlo.
