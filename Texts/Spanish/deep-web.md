# Red profunda
La web profunda es la sección de la World Wide Web (WWW) que está algo escondida. Contiene páginas que no están indexadas por los motores de búsqueda web tradicionales, como Google, Bing y DuckDuckGo.

La web profunda es el término opuesto a la “web superficial”. Como tal, también puede denominarse "web oculta" o "web invisible".

## Más allá de la superficie

La web profunda representa más del 99% del contenido en la web. Es una parte de la WWW diferente a la llamada “web de superficie”, que es la parte visible de la web. La red de superficie es la capa de más fácil acceso. Incluye todas las páginas de índices (o búsquedas).

Aunque la cantidad de contenido que se puede descubrir a través de los motores de búsqueda es enorme, la web superficial contiene mucha menos información que la web profunda. Un estudio publicado en 2001 indicó que menos del 0,05% de todo el contenido está disponible en la web superficial, mientras que el otro ~99,95% se encuentra en la web profunda.

## ¿Qué es la Deep Web?

Por definición, la web profunda incluye páginas que son oscuras o a las que solo se puede acceder a través de un método de autenticación específico. Ser oscuro significa que requieren una URL directa para ser encontrados (la búsqueda en Google no funciona). Aún así, la mayoría del contenido de la web profunda se encuentra en la última categoría.

En ese sentido, podemos considerar como parte de la web profunda cualquier sitio web que solicite un nombre de usuario y una contraseña. En otras palabras, muchas de las páginas a las que la gente accede todos los días son en realidad parte de la web profunda. Esto incluye perfiles de redes sociales, bandejas de entrada de correo electrónico y cuentas bancarias.

Esto puede sonar confuso al principio porque la terminología popular dice que la web profunda es la web oscura. Pero dado que nuestras cuentas personales no están indexadas en las páginas de los motores de búsqueda, también se consideran parte de la web profunda.

Aún así, la mayor parte de la web profunda consiste en estadísticas y bases de datos privadas. Por ejemplo, las agencias gubernamentales de EE. UU. como la [Comisión de Comercio de Futuros de Productos Básicos](/cftc) [(CFTC)](/cftc),. la Administración Nacional Oceánica y Atmosférica (NOAA) y la [Comisión de Bolsa y Valores (SEC)](/sec) representan una cantidad significativa de estos datos digitales.

## Internet profunda versus web oscura

También es común ver confusión sobre las diferencias entre la deep web y la dark web. En resumen, la web oscura es una parte de la web profunda que es anónima.

Por lo tanto, a menudo se asocia con sitios web que facilitan actividades ilegales o que brindan algún tipo de beneficio de privacidad para activistas y denunciantes (incluidos los [piratas informáticos de sombrero negro](/hacker) y los hacktivistas). Se estima que la web oscura representa menos del 0,01% de la web profunda y solo es accesible a través de plataformas especializadas, como Tor.

## Reflejos

- Quizás la crítica más seria a la deep web es que socava la apertura y la igualdad de internet.

- La web profunda se refiere a partes de Internet a las que no se puede acceder por completo a través de motores de búsqueda estándar como Google, Yahoo y Bing.

- La web profunda brinda a los usuarios acceso a mucha más información de la que estaría disponible en Internet, y también aumenta la privacidad.

- La web profunda incluye páginas que no fueron indexadas, sitios de pago por servicio, bases de datos privadas y la web oscura.

## PREGUNTAS MÁS FRECUENTES

### ¿La Deep Web y la Dark Web son lo mismo?

La web oscura es un pequeño componente de la web profunda. La web profunda se refiere a todo el Internet que no es fácilmente accesible o indexado por los motores de búsqueda (lo que se denomina "web de superficie"). La web oscura consiste en sitios web profundos que se ocultan intencionalmente, a menudo para llevar a cabo actividades ilegales o transacciones ilícitas en línea.

### ¿Es ilegal acceder a la web profunda?

No. El simple hecho de acceder a sitios que no están indexados o que no están disponibles públicamente no es ilegal. Sin embargo, podría ser ilegal hacer un uso indebido o robar información que se encuentra en sitios web profundos.

### ¿Cómo puedo encontrar sitios en la web profunda?

Debido a que la web profunda no está completamente indexada con los principales motores de búsqueda, a menudo la única forma de encontrar dichos sitios es conocer la dirección de Internet exacta que se debe usar. Hay algunos sitios web especializados o motores de búsqueda que catalogan algunos sitios web profundos. Por ejemplo, se pueden encontrar recursos académicos en la web profunda utilizando plataformas como PubMed, LexisNexis, Web of Science o Project MUSE.
