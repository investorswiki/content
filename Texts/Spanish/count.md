# Contar
## ¿Qué es contar?

El término recuento se refiere a una forma de [análisis técnico](/technicalanalysis) que emplea [gráficos de punto y figura (P&F)](/pointandfigurechart) para evaluar el movimiento vertical de los precios de las acciones. Diseñado para ser utilizado para el análisis de inversiones a largo plazo, los gráficos de puntos y figuras se consideran una de las formas más fáciles para que los comerciantes determinen los puntos de entrada y salida. Los comerciantes que emplean el análisis de conteo trazan los aumentos y disminuciones de precios que se utilizan para identificar los precios objetivo.

## Comprender el conteo

El análisis técnico es un tipo de disciplina de negociación que utiliza tablas y gráficos para evaluar las [tendencias pasadas](/trend) de los valores, como el precio y [el volumen de negociación](/volumeoftrade),. a fin de determinar el rendimiento futuro. Los comerciantes emplean técnicas de análisis técnico para encontrar los puntos de [entrada](/entry-point) y [salida](/exit-point) que optimizarán las ganancias. Algunos operadores utilizan gráficos de puntos y figuras sin tener en cuenta el paso del tiempo. Esta técnica se conoce como análisis de conteo.

El análisis de conteo traza Xs para representar aumentos de precios y Os para disminuciones de precios en su gráfico. Los analistas basan los cálculos de conteo en movimientos de precios laterales históricos y los usan para determinar la probabilidad de que se pueda alcanzar un [precio objetivo .](/pricetarget) Los análisis de conteo Xs y Os se utilizan con una escala tradicional y una cantidad de reversión determinada previamente. Los comerciantes usan esto para determinar si ciertas posiciones son rentables. Los inversores pueden revisar la secuencia de fluctuaciones de precios para estimar cómo se moverán los precios en el futuro. Hay varios métodos de conteo, como el método de conteo de ruptura que se usa para encontrar un objetivo de precio alcista que se debe usar con una señal de compra P&F activa.

> Como inversionista, puede revisar la secuencia de fluctuaciones de precios para estimar cómo es probable que se muevan los precios en el futuro.

>

## Tipos de análisis de conteo

Como se señaló anteriormente, existen diferentes métodos para el análisis de conteo, como conteos de ruptura, inversión y horizontales.

### Análisis de recuento de rupturas

El método de conteo [de ruptura](/breakout) se usa para encontrar un objetivo de precio alcista que se debe usar con una señal de compra P&F activa. Hay cuatro pasos para este método.

En primer lugar, la señal de venta más activa, conocida como desglose de [doble fondo](/doublebottom),. debe encontrarse en el gráfico P&F, trabajando de derecha a izquierda. El segundo paso consiste en trabajar a la derecha de esta señal para encontrar la siguiente señal de compra o la ruptura del [doble](/doubletop) techo. La columna que produce esta señal es clave porque se convierte en la columna de medida. Luego, la altura de la columna de medida debe calcularse y multiplicarse por la cantidad de inversión de la caja. Finalmente, el total de este cálculo debe sumarse al mínimo de la columna a la izquierda de la columna de medida.

### Análisis de recuento de inversión

El método de conteo de [reversión](/reversal) se puede usar para encontrar objetivos de precios [alcistas](/bull) y [bajistas .](/bear) Hay tres pasos que se utilizan para encontrar precios alcistas. El conteo de reversión debe usarse con una señal de compra P&F activa. Primero, trabajando de izquierda a derecha, se debe encontrar la señal de venta P&F más reciente. La columna X junto a la señal de venta se convierte en la columna de medida. A continuación, se debe calcular la altura de la columna y multiplicarla por la [cantidad de inversión de la caja](/reversalamount). Luego, el total debe agregarse al mínimo de la columna a la izquierda de la columna de medida.

### Análisis de conteo horizontal

Para el método de conteo horizontal, se debe formar un patrón de [congestión](/congestion) o inversión en un gráfico P&F. El patrón de congestión debe tener un mínimo de cinco columnas de ancho y debe tener una columna que rompa la congestión. Las columnas en este patrón deben contarse. Este es el ancho. Después de que se produce la columna de ruptura en el gráfico de P&F, los analistas pueden multiplicar el ancho por el [tamaño del cuadro](/boxsize) y el monto de reversión para estimar la extensión del precio. La extensión se agrega al mínimo del patrón para un precio objetivo.

## Reflejos

- Hay varios métodos de conteo, como el método de conteo de ruptura, el método de conteo inverso y el método de conteo horizontal.

- El análisis de conteo usa X para representar aumentos de precios y Os para disminuciones de precios para identificar precios objetivo.

- Los cálculos de conteo se basan en movimientos de precios laterales históricos, que se utilizan para determinar la probabilidad de alcanzar un precio objetivo.

- Count es una forma de análisis técnico que utiliza gráficos de puntos y figuras para evaluar el movimiento vertical de los precios de las acciones.
