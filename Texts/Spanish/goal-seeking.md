# Búsqueda de objetivos
## ¿Qué es la búsqueda de objetivos?

La búsqueda de metas es el proceso de encontrar el valor de entrada correcto cuando solo se conoce la salida. La función de búsqueda de objetivos puede integrarse en diferentes tipos de programas informáticos como Microsoft Excel.

## Comprender la búsqueda de objetivos

La búsqueda de objetivos es un término general que se usa para describir el proceso involucrado en calcular el valor de entrada en función de un valor de salida ya conocido. El proceso implica el uso de un operador específico en una fórmula, que se puede calcular mediante un software de computadora.

La búsqueda de objetivos es una de las herramientas utilizadas en el "análisis hipotético" de los programas informáticos. Un [análisis hipotético es un proceso de cambio de valores en celdas (Microsoft Excel) para ver](/sensitivityanalysis) [cómo](/sensitivityanalysis) estos cambios afectarán los resultados de la fórmula en la hoja de trabajo. Cuando busca un objetivo, está realizando un análisis hipotético sobre un valor dado o el resultado. Entonces, en esencia, estaría creando un escenario preguntando "¿qué pasaría si el resultado fuera X?", o básicamente, una situación de causa y efecto.

Para algunos de los problemas más complejos, las personas a menudo usan software de computadora. Un programa de hoja de cálculo como Microsoft Excel tiene una herramienta de búsqueda de objetivos incorporada. Permite al usuario determinar el valor de entrada deseado para una fórmula cuando ya se conoce el valor de salida. Esta característica puede ayudar al usuario a determinar cosas como la [tasa de interés para la que](/interestrate) un prestatario debe calificar (la entrada) si solo sabe cuánto puede pagar cada mes (la salida).

Pero hay una advertencia sobre el uso de software de búsqueda de objetivos: solo funciona si hay un valor de entrada. Si necesita averiguar dos o más valores de entrada, entonces no funcionará. Si tomamos el ejemplo anterior, si desea calcular el monto total del préstamo y el pago mensual, el software de búsqueda de objetivos no funcionará. Probablemente necesite un complemento para descubrir múltiples variables.

### Cómo funciona la búsqueda de objetivos en Microsoft Excel

Como se mencionó anteriormente, el software de búsqueda de objetivos solo funcionará si ya conoce el valor de salida (o el resultado) pero desea determinar un valor de entrada. Aquí hay algunos pasos clave que puede seguir si desea utilizar la función de búsqueda de objetivos en el programa:

- Abrir una nueva hoja de cálculo

- Etiqueta tus columnas. Esto hará que sea más fácil para usted leer todo en la hoja de trabajo. Entonces, usando el ejemplo anterior, la primera columna será "Cantidad del préstamo", la segunda columna será " [Plazo](/term) en meses", la tercera será "Tasa de interés" y la última será "Pago".

- Escriba los valores que conoce.

- Introduzca la fórmula para calcular el objetivo, en este caso, el pago. Puede ignorar la tasa de interés por ahora, que la fórmula supondrá que es 0 por ciento.

Ahora tendrás que determinar la tasa de interés. Puede hacer esto usando la función Goal Seek en Excel e ingrese los valores que ya tiene.

## Ejemplo de búsqueda de objetivos

Un [emprendedor](/entrepreneur) que busca objetivos es alguien que utiliza la búsqueda de objetivos para determinar cómo puede alcanzar sus objetivos finales. Por ejemplo, un empresario puede preguntar cuánto necesitará ganar por hora para obtener $100,000 brutos en un año. Conocen el valor de salida deseado ($100 000) y, por lo tanto, tendrán que volver a trabajar para determinar la entrada óptima. Eso significa que tendrán que determinar cuántas horas podrán (o estarán dispuestos a) trabajar durante el año y, por lo tanto, cuánto ganarán por cada hora para alcanzar su meta.
