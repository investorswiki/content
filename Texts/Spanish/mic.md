# Código de identificación del mercado (MIC)
## ¿Qué es un código de identificación de mercado (MIC)?

Un código de identificación de mercado (MIC, por sus siglas en inglés) es un código de cuatro caracteres que se utiliza para identificar los mercados de [valores y otras](/stockmarket) [bolsas](/stockmarket) comerciales dentro de los sistemas informáticos de referencia y comercio global. La primera letra de cualquier MIC se asigna aleatoriamente seguida de un código alfanumérico de tres dígitos para el mercado en el que se realiza la operación. Los remitentes pueden proponer una combinación de cuatro caracteres, pero la autoridad de registro sigue siendo la responsable final de la asignación. Una vez asignado, un código MIC no se puede modificar .

El código MIC se usa para procesar y liquidar operaciones y está siendo empujado hacia la aceptación global a medida que las industrias de valores avanzan hacia el procesamiento directo (STP). La [Bolsa de Valores de Londres](/lse) utiliza MIC como parte de sus sistemas de identificación de valores [SEDOL](/sedol),. que es una alternativa al sistema de identificación CUSIP basado en EE. UU .﻿﻿.

## Cómo se utilizan los códigos de identificación del mercado

Los MIC han sido presentados por ISO como un estándar internacional para identificar mercados, plataformas de negociación e intercambios para permitir el procesamiento automatizado de transacciones. Con el uso de MIC como el estándar de identificación universal, podría ser un paso significativo para acercar el procesamiento directo más allá de la fase de concepto.

Hay diferentes tipos de MIC. Un segmento de mercado MIC identifica una sección de una de las entidades cubiertas por el código que se especializa en uno o más instrumentos específicos o está regulada de manera diferente. Los MIC de segmento de mercado se instituyeron para una mayor precisión, según ISO .

Para cada MIC de segmento de mercado, hay un MIC principal, también conocido como MIC operativo .

Un MIC operativo, a su vez, identifica a la entidad que opera una bolsa, una plataforma de negociación, un mercado regulado o no regulado o una instalación de información comercial en un país específico .

Las organizaciones de mercado pueden solicitar un MIC presentando una solicitud a la autoridad de registro. Por lo general, un MIC de segmento de mercado se genera cuando la demanda amerita la necesidad de identificar el segmento. Dependiendo de las circunstancias, puede suceder que solo segmentos específicos de un mercado reciban identificación con un segmento de mercado MIC .

## Reflejos

- Se utiliza un código de identificación de mercado (MIC) para especificar los mercados de valores y otras bolsas en las que se negocian valores.

- Los MIC se utilizan como estándar de identificación internacional y universal, con códigos definidos por la Organización Internacional de Normalización (ISO).

- El código es único, incluye cuatro caracteres, comenzando con un carácter alfanumérico asignado aleatoriamente, seguido de un código de tres dígitos para indicar el mercado, como XNAS para el mercado Nasdaq.
