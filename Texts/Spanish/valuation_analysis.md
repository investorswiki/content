# Análisis de valoración
## ¿Qué es el análisis de valoración?

El análisis de valoración es un proceso para estimar el valor o valor aproximado de un activo, ya sea un negocio, acciones, [valores de renta fija](/fixed-incomesecurity),. materias primas, bienes inmuebles u otros activos. El analista puede usar diferentes enfoques para el análisis de valoración para diferentes tipos de activos, pero el hilo común será observar los fundamentos subyacentes del activo.

## Comprender el análisis de valoración

El análisis de valoración es principalmente ciencia (procesamiento de números), pero también hay un poco de arte involucrado porque el analista se ve obligado a hacer suposiciones para las entradas del modelo. El valor de un activo es básicamente el [valor presente](/presentvalue) (PV) de todos los flujos de efectivo futuros que se prevé que produzca el activo. Inherente al modelo de estimación de una empresa, por ejemplo, hay una miríada de supuestos relacionados con el crecimiento de las ventas, los márgenes, las opciones de financiación, los gastos de capital, las tasas impositivas, la tasa de descuento para la fórmula PV, etc.

Una vez que se configura el modelo, el analista puede jugar con las variables para ver cómo cambia la valoración con estos diferentes supuestos. No existe un modelo único para todas las clases de activos. Mientras que la valoración de una empresa manufacturera puede ser adecuada para un modelo [DCF de varios años](/dcf),. y una empresa inmobiliaria se modelaría mejor con [el ingreso operativo neto](/noi) (NOI) actual y la [tasa de](/capitalizationrate) [capitalización](/capitalizationrate) (tasa de capitalización), materias primas como el mineral de hierro, el cobre o la plata estarían sujetos a un modelo centrado en las previsiones de oferta y demanda mundiales.

## Cómo se utiliza el análisis de valoración

El resultado del análisis de valoración puede tomar muchas formas. Puede ser un solo número, como una empresa con una valoración de aproximadamente $ 5 mil millones, o podría ser un rango de números si el valor de un activo depende en gran medida de una variable que a menudo fluctúa, como un bono corporativo con un alta duración con un rango de valoración entre la par y el 90% de la par dependiendo del rendimiento del [bono del Tesoro a 30 años](/30-yeartreasury). La valoración se puede expresar como un múltiplo de precio. Por ejemplo, mientras que una acción de tecnología se negocia a un múltiplo de precio-beneficio (P/E) de 40x, una acción de telecomunicaciones se valora a 6x [el valor empresarial](/enterprisevalue) - [beneficio antes de intereses, impuestos, depreciación y amortización](/ebitda) (EV/EBITDA). ) o un banco cotiza a una relación [precio/valor contable](/price-to-bookratio) (P/B) de 1,3x [.](/price-to-bookratio) El análisis de valoración también puede tomar la forma final de valor de activo por acción o valor de activo neto (NAV) por acción.

## Valoración y Valor Intrínseco

El análisis de valoración es importante para que los inversores [estimen](/intrinsicvalue) [los valores intrínsecos](/intrinsicvalue) de las acciones de la empresa a fin de tomar decisiones de inversión mejor informadas. Los valores razonables de los bonos no se desvían mucho, si es que lo hacen, de los valores intrínsecos, pero las oportunidades surgen de vez en cuando en el caso de estrés financiero de una empresa muy endeudada. El análisis de valoración es una herramienta útil para comparar empresas dentro del mismo sector o estimar el rendimiento de una inversión durante un período de tiempo determinado.

## Reflejos

- El análisis de valoración busca estimar el valor razonable o el valor intrínseco de un activo, como un negocio o un valor.

- Se emplearán diferentes procesos de valuación dependiendo del tipo de activo que se considere, si ese activo produce flujos de efectivo y cuál es el propósito de la valuación.

- El análisis de valoración se basa en varias metodologías y modelos diferentes para llegar a un precio único basado en diferentes entradas o variables.
