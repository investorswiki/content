# Dirección IP
## ¿Qué es una dirección IP?

La dirección IP significa dirección de protocolo de Internet; es un número de identificación que está asociado con una computadora o red informática específica. Cuando está conectado a Internet, la dirección IP permite que las computadoras envíen y reciban información.

## Cómo funciona una dirección IP

Una dirección IP permite que las computadoras envíen y reciban datos a través de Internet. La mayoría de las direcciones IP son puramente numéricas, pero a medida que crece el uso de Internet, se han agregado letras a algunas direcciones.

Hay cuatro tipos diferentes de direcciones IP: públicas, privadas, estáticas y dinámicas. Mientras que el público y el privado son indicativos de la ubicación de la red (privado se usa dentro de una red mientras que el público se usa fuera de una red), estático y dinámico indican permanencia.

Una dirección IP estática es aquella que se creó manualmente, en lugar de haber sido asignada. Una dirección estática tampoco cambia, mientras que una dirección IP dinámica ha sido asignada por un servidor de Protocolo de configuración dinámica de host (DHCP) y está sujeta a cambios. Las direcciones IP dinámicas son el tipo más común de direcciones de protocolo de Internet. Las direcciones IP dinámicas solo están activas durante un cierto período de tiempo, luego de lo cual expiran. La computadora solicitará automáticamente una nueva concesión o la computadora puede recibir una nueva dirección IP.

Una dirección IP se puede comparar con un [Número de Seguro Social](/ssn) (SSN) ya que cada uno es completamente único para la computadora o el usuario al que está asignado. La creación de estos números permite a los enrutadores identificar dónde están enviando información en Internet. También se aseguran de que los dispositivos correctos reciban lo que se envía. Al igual que la oficina de correos necesita una dirección de correo para entregar un paquete, un enrutador necesita una dirección IP para entregar a la dirección web solicitada.

## Ejemplo de una dirección IP

La web oscura se refiere al contenido en línea encriptado que no está indexado por los motores de búsqueda convencionales. La web oscura tiene un mercado ilegal en línea donde los delincuentes pueden comerciar con bienes ilegales e ilícitos. Muchos de estos intercambios se realizan utilizando la criptomoneda en línea bitcoin, lo que dificulta que las autoridades rastreen y capturen a las personas que participan en estas transacciones.

En 2018, después de una operación gubernamental de un año, los agentes que trabajaban con el Departamento de Seguridad Nacional se hicieron pasar por traficantes de armas para obtener acceso a las computadoras de los sospechosos que intentaban comprar armas ilegalmente. Esto les permitió obtener acceso a direcciones IP, que usaron para rastrear las ubicaciones geográficas de sospechosos adicionales que estaban usando la web oscura.

Esta no es la primera vez que las direcciones IP han dado lugar a arrestos. En 2012, la policía usó direcciones IP para rastrear y arrestar a miembros del grupo de piratería Lulzsec. Usando órdenes judiciales para obtener información del [proveedor de servicios de Internet (ISP)](/isp),. el personal encargado de hacer cumplir la ley pudo rastrear las direcciones físicas de los piratas informáticos y arrestarlos por su actividad ilegal en Internet.

## Reflejos

- Una dirección de protocolo de Internet (IP) permite que las computadoras envíen y reciban información.

- Una dirección IP permite que las partes correctas envíen y reciban información, lo que significa que también se puede usar para rastrear la ubicación física de un usuario en algunos casos.

- Hay cuatro tipos de direcciones IP: públicas, privadas, estáticas y dinámicas.

## PREGUNTAS MÁS FRECUENTES

### ¿Qué es una dirección IPv6?

Las direcciones IPv4 constan de cuatro números de uno a tres dígitos separados por un punto. IPv6 es la versión más nueva de direccionamiento de Internet que utiliza un valor alfanumérico de 128 bits para identificar un dispositivo de punto final en línea. Una dirección IPv4 puede parecerse a 208.80. 154.224, y una dirección IPv6 puede verse como: 2001:0db8:85a3:0000:0000:8a2e:0370:7334.IPv6 permite que existan muchos más dispositivos conectados en Internet sin duplicar direcciones, lo que puede causar problemas.

### ¿Cómo puede alguien "falsificar" su dirección IP?

Puede hacer que su dirección IP parezca que proviene de otro lugar a través de una red privada virtual (VPN). Estos servicios enmascaran esencialmente la dirección de Internet de su computadora al unirse virtualmente a la red que establecen. Por lo tanto, su dirección IP aparecerá como la de ellos. Los piratas informáticos también pueden falsificar direcciones IP utilizando técnicas más sofisticadas que modifican la dirección de origen en un encabezado de paquete de datos malicioso para imitar una fuente confiable y aceptarla.

### ¿Cómo averiguo mi dirección IP?

Hay varias formas de determinar su dirección IP. Quizás el más simple es escribir "cuál es mi dirección IP" en un motor de búsqueda de Internet como Google. Para los usuarios de Windows, puede encontrar su IP abriendo [Inicio> Configuración> Red e Internet] y luego seleccione la red Wi-Fi a la que está conectado. En Propiedades, busque su dirección IP junto a "Dirección IPv4".
