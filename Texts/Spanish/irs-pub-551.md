# Publicación 551 del IRS - Base de los activos
## ¿Qué es la Publicación 551?

La Publicación 551 del IRS, titulada Base de activos, es un documento informativo publicado por el [Servicio de Impuestos Internos (IRS)](/irs) que describe cómo determinar la [base de costos](/costbasis) para inversiones, bienes raíces y activos comerciales.﻿La base de costos se utiliza para determinar qué cantidad de la ganancia o pérdida se obtiene de una venta y representa el costo original de la inversión o propiedad. También se utiliza para determinar la depreciación y amortización de una propiedad .

La Publicación 551 del IRS se puede encontrar en el sitio web del IRS.

## Entendiendo la Publicación 551 del IRS

La base del costo de una propiedad suele ser el costo de compra; sin embargo, la base puede aumentar con el tiempo si el propietario realiza mejoras en la propiedad. Para inversiones, como acciones y bonos, la base del costo también incluye las tarifas de negociación .

A efectos fiscales, el método utilizado por el Servicio de Rentas Internas (IRS, por sus siglas en inglés) es [el primero en entrar, el primero en salir](/fifo) (FIFO, por sus siglas en inglés) para quienes están familiarizados con el método de seguimiento de inventario para empresas. En otras palabras, cuando se realiza una venta, primero se usaría la base del costo de la compra original y seguiría una progresión a través del historial de compras .

A partir de 2018, las pequeñas empresas no están sujetas a las reglas uniformes de capitalización si los ingresos brutos anuales promedio son de $25 millones o menos durante los 3 años fiscales anteriores y la empresa no es un paraíso fiscal. Las reglas de capitalización uniforme especifican los costos que agrega a la base en ciertas circunstancias. Puede encontrar más información sobre cómo establecer la base de costos para las inversiones en la [Publicación 550](/irs-pub-550) del IRS .

## Base de costos de declaración de impuestos

Aunque las casas de bolsa están obligadas a informar al IRS el precio pagado por los valores sujetos a impuestos, para algunos [valores](/security),. como los que se mantienen durante un largo período de tiempo o los que se transfieren de otra casa de bolsa, la base del [costo histórico](/historical-cost) deberá ser proporcionada por el inversor . Todo lo cual pone la responsabilidad de la información precisa sobre la base de costos en los inversores.

Determinar la base del costo inicial de valores y [activos financieros](/financialasset) para una sola compra inicial es muy sencillo. En realidad, puede haber compras y ventas posteriores a medida que un inversionista toma decisiones para implementar [estrategias comerciales específicas](/trading-strategy) y maximizar el potencial de ganancias para impactar una cartera general. Con todos los diversos tipos de inversiones, incluidas acciones, bonos y opciones, calcular la base de costos con precisión a efectos fiscales puede resultar complicado.

En cualquier transacción entre un comprador y un vendedor, el precio inicial pagado a cambio de un producto o servicio calificará como [base de costo](/costbasis). La base del costo de capital es el costo total para un inversionista; esta cantidad incluye el precio de compra por acción más [reinvertido](/reinvestment) [dividendos](/dividend) y comisiones. La base del costo de capital no solo se requiere para determinar cuántos impuestos, si corresponde, deben pagarse sobre una inversión, sino que es fundamental para rastrear las ganancias o pérdidas de la inversión para tomar decisiones informadas de compra o venta .

## Reflejos

- El IRS requiere el método de primeras entradas, primeras salidas (FIFO) para calcular los impuestos y la base de costos, lo que significa que las tenencias más antiguas se venden primero a efectos fiscales.

- La base de costo es el valor original o precio de compra de un activo o inversión para efectos fiscales.

- La Publicación 551 del IRS informa a los contribuyentes cómo obtener la base del costo de un activo o inversión.

- Base se utiliza para calcular la tasa de impuesto a las ganancias de capital, que es la diferencia entre la base de costo del activo y el valor de mercado actual.
