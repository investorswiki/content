# Ad hoc
La frase "ad hoc" es una frase de origen latino que se usa en inglés moderno para significar "para este propósito" o "específicamente para esto". La mayoría de los diccionarios dan la traducción directa de la frase latina como "por esto".

El término "ad hoc" se puede utilizar para implicar que una solución encontrada para un problema o problema en particular no se debe considerar como una solución predeterminada que se utilizará en aplicaciones futuras.

Por ejemplo, un juez puede dictar un fallo "ad hoc" sobre un caso revisando los detalles del caso y los detalles específicos de la situación, pero con miras a que este fallo no se considere jurisprudencial para resolver incidentes de similar naturaleza. naturaleza en el futuro.

En el desarrollo de software, el término "ad hoc" se usa a menudo en la frase "solución ad hoc" para referirse a una solución específica y personalizada desarrollada para manejar las necesidades de un sistema o usuario que se presenta.
