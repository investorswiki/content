# Fuente abierta
El software de código abierto es un tipo de software cuyo [código fuente](/source-code) está publicado y disponible para que cualquiera lo vea, cambie o redistribuya. El software de código abierto generalmente se desarrolla de manera abierta y colaborativa entre muchos participantes diferentes. Dichos proyectos adoptan principios de transparencia, intercambio abierto, participación libre, creación rápida de prototipos y desarrollo colaborativo orientado a la comunidad.

Si bien todas las licencias de software de código abierto permiten que cualquier persona estudie, modifique y redistribuya el código fuente, algunas requieren que el código fuente original se publique junto con el código fuente modificado.

Lo opuesto al software de código abierto es el software de código cerrado, cuyo código fuente no se publica, y solo la persona, el equipo o la organización que lo creó tiene derecho a modificarlo y distribuirlo.

El software de código abierto puede tener ventajas sobre otros tipos de software. Si bien promueve la filosofía de colaboración abierta que puede beneficiar tanto a los desarrolladores como a los usuarios, también puede conducir a un producto de mayor calidad debido a la convocatoria abierta de participación que reúne a muchos expertos diferentes de diversos campos. La solución de problemas también puede ser más rápida y de menor costo que en el caso del software de código cerrado, ya que potencialmente una gran cantidad de participantes tiene el derecho, el acceso y la experiencia para atender esos problemas.

Por otro lado, si el software de código abierto se desarrolla de manera descentralizada, sin una sola entidad que coordine el proceso de desarrollo, los diferentes participantes pueden tardar mucho tiempo en acordar decisiones estratégicas. Muchos proyectos de código abierto implementan tipos especiales de modelos de gobernanza para abordar este problema.

En términos más amplios, el movimiento que tiene como objetivo hacer que todo o la mayoría del software utilice una licencia de código abierto se denomina movimiento de software de código abierto. Los objetivos de este movimiento también incluyen mejorar la tecnología de código abierto con la producción de código de alta calidad hecho de forma cooperativa por personas con ideas similares. Los desarrolladores que apoyan el movimiento del software de fuente abierta escriben e intercambian código voluntariamente como una contribución a la filosofía de la colaboración abierta.

## Reflejos

- El código abierto se refiere a un proyecto, a menudo un proyecto de desarrollo de software o TI, con código disponible gratuitamente y permiso para realizar ediciones, correcciones de errores, alteraciones y mejoras.

- El código abierto se basa en la participación de la comunidad, la confianza y las estrategias de concesión de licencias permisivas.

- Los esfuerzos de software de código abierto han estado detrás de algunas de las aplicaciones y sistemas operativos más populares disponibles, incluido el sistema operativo Android para teléfonos inteligentes y el navegador web Firefox.
