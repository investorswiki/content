# Declaración jurada de pérdida
## ¿Qué es la declaración jurada de pérdida?

Una declaración jurada de pérdida es un documento que declara la pérdida de un [valor,](/security) generalmente por robo o destrucción. La declaración jurada contiene todos los detalles sobre la pérdida, como el nombre del propietario y cualquier información relacionada con la seguridad. Esa información puede incluir un número de serie o la fecha de emisión del valor. Realizada la declaración, se puede emitir una carta de indemnización solicitando la reposición del valor.

## Entendiendo la declaración jurada de pérdida

Siempre existe la posibilidad de que se pierda un registro físico de un valor, como un certificado de acciones. Si el registro es robado, extraviado o destruido, el propietario debe jurar una declaración jurada relacionada con la pérdida del artículo. Este es un juramento que indica que, según el leal saber y entender de la persona, el registro realmente se ha ido. Si la copia perdida aparece después de que el propietario recibe un valor de reemplazo, el registro original debe enviarse a la empresa para su eliminación.

### Otros usos

Hay declaraciones comparables para documentos financieros faltantes que no son valores. Una declaración jurada de pagaré perdido se presenta cuando se destruye o se pierde un [pagaré](/promissorynote) u otro registro de deuda. Este tipo de declaración jurada incluiría el nombre del deudor, el saldo pendiente, la tasa de interés de la deuda, detalles sobre el historial de pago y la lista de cualquier propiedad que se haya ofrecido como garantía.

Se puede presentar una declaración jurada de pérdida para otros tipos de documentos, como los relacionados con la propiedad de una casa, un pasaporte, un diploma o incluso una tarjeta de cajero automático. Las licencias de matrimonio faltantes pueden requerir dicho documento.

Cada vez que una persona pierde un registro que no se puede reemplazar de inmediato con una copia, es posible que deba presentar una declaración jurada de pérdida. Por ejemplo, el [título](/title),. el registro o las placas de un vehículo podrían destruirse, robarse o desfigurarse hasta el punto de que ya no se puedan usar. Algunos estados requieren una declaración jurada para registrar su pérdida. El propietario o titular del título debe proporcionar información como la marca, el modelo y el número de identificación del vehículo asociado con los documentos perdidos.

Las reglas y requisitos con respecto a la información requerida al presentar una declaración jurada de pérdida varían según la jurisdicción. Algunos requieren una declaración jurada de pérdida para ser notariada.

## Reflejos

- Una declaración jurada de pérdida es un documento que declara que un valor, como un certificado de acciones, se perdió o se destruyó.

- El emisor del valor requiere una declaración jurada para indemnizar al tenedor o reemplazar el certificado.

- Las personas físicas pueden presentar declaraciones juradas de pérdida para otros tipos de documentos, como un pasaporte o un diploma.
