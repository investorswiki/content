# Gráfico de barras
## ¿Qué es un gráfico de barras?

Un gráfico de barras es un cuadro que traza datos usando columnas o barras rectangulares (llamadas bins) que representan la cantidad total de observaciones en los datos para esa categoría. Los gráficos de barras se pueden mostrar con columnas verticales, barras horizontales, barras comparativas (barras múltiples para mostrar una comparación entre valores) o barras apiladas (barras que contienen múltiples tipos de información).

Los gráficos de barras se usan comúnmente en el análisis financiero para mostrar datos. Un gráfico de [volumen de existencias](/volume) es un tipo de gráfico de barras verticales de uso común. Un [histograma](/histogram) es un ejemplo de un gráfico de barras utilizado en el análisis estadístico que representa una distribución de probabilidad en algunos datos o muestras.

## Comprender un gráfico de barras

El propósito de un gráfico de barras es transmitir información relacional rápidamente, ya que las barras muestran la cantidad de una categoría en particular. El eje vertical del gráfico de barras se llama eje y, mientras que la parte inferior de un gráfico de barras se llama eje x.

Al interpretar un gráfico de barras, la longitud de las barras/columnas determina el valor como se describe en el eje y.

El eje x podría ser cualquier variable, como el tiempo, o la categoría que se mide, como las [ganancias por acción](/eps) (EPS), los ingresos y/o [el flujo de efectivo](/cashflow). En el comercio, los gráficos de barras se utilizan a menudo para representar el volumen de negociación y aparecerán en un panel debajo del gráfico de precios de un valor.

## Características del gráfico de barras

Un gráfico de barras típico tiene una etiqueta o título, eje x, eje y, escalas o incrementos para el eje y barras. Algunos gráficos también pueden tener una leyenda que especifica lo que representan varios colores, como en un gráfico de barras apiladas.

Los gráficos de barras son ideales para comparar dos o más valores, o valores a lo largo del tiempo. Los datos se muestran horizontal o verticalmente. Los gráficos de barras individuales se utilizan para transmitir valores discretos de un elemento dentro de una categoría. Por ejemplo, un gráfico de barras podría mostrar la cantidad de hombres con un determinado rasgo para edades específicas. El valor discreto, o el número de instancias en las que un individuo tiene un determinado rasgo, se muestra variando la longitud de la barra. Más instancias significan una barra más larga y menos instancias significan una barra más corta. En este ejemplo, se establece una barra diferente para cada edad o grupo de edad.

En [el análisis técnico](/technicalanalysis),. un gráfico de volumen muestra cuánto volumen hubo en cada día en particular. El eje x muestra los días, mientras que una barra que se extiende desde ese día muestra cuánto volumen había por eje y.

Cuando un gráfico tiene un punto cero bien definido y el conjunto de datos tiene valores tanto positivos como negativos en relación con este punto, aún se pueden mostrar barras. Las barras por encima de la línea cero suelen representar valores positivos (consulte la escala), mientras que las barras por debajo de la línea cero suelen mostrar valores negativos.

Los datos se pueden mostrar horizontal o verticalmente. Para cambiar la orientación, se cambian los ejes x e y.

## Tipos de gráficos de barras

### Gráficos de barras agrupadas

Los gráficos de barras agrupadas, también llamados gráficos de barras agrupadas, representan valores discretos para más de un elemento que comparten la misma categoría.

Un gráfico de barras podría mostrar el número de individuos, hombres y mujeres, con un determinado rasgo para edades específicas. El número agregado de instancias podría combinarse en una barra. Alternativamente, las instancias podrían permanecer segregadas por género; una barra para todas las instancias masculinas y una barra para todas las instancias femeninas se colocarían una al lado de la otra para cada edad o grupo de edad.

### Gráficos de barras apiladas

Los gráficos de barras apiladas o los gráficos de barras compuestos dividen un total agregado en partes. Estas partes generalmente se identifican utilizando diferentes colores para cada sección. En el ejemplo anterior, el agregado de instancias tanto para hombres como para mujeres se puede combinar en una barra, pero la barra se puede dividir en múltiples secciones representadas por diferentes colores.

Las barras apiladas requieren una leyenda o un etiquetado específico para identificar lo que muestran las diversas partes de la barra.

## Gráficos de barras en análisis técnico

Algunas formas de análisis técnico utilizan gráficos de barras. Por ejemplo, los comerciantes pueden emplear un histograma de [divergencia de convergencia de promedio móvil](/macd) (MACD), que es un indicador técnico popular que ilustra la diferencia entre la línea MACD y la [línea de señal](/signal_line).

## Gráfico de barras frente a gráfico de barras

Un gráfico de barras muestra datos en columnas, mientras que un [gráfico de barras](/barchart) es un término de análisis técnico que describe la visualización de los precios de apertura, máximo, mínimo y cierre (a veces se omite la apertura) para un [valor en particular](/security) durante un período de tiempo específico mediante una barra vertical. . Pequeñas líneas horizontales se extienden a la izquierda y derecha de la barra vertical para mostrar los precios de apertura y cierre.

A diferencia del gráfico de barras, la barra de precios solo cubre los precios relevantes y no se extiende desde el eje x.

## Limitaciones del gráfico de barras

Un gráfico de barras es una forma de mostrar información. La forma en que se eligen los datos para mostrar podría afectar su interpretación. Por ejemplo, si se elige una escala demasiado grande, los datos pueden parecer insignificantes, cuando en realidad podrían ser muy significativos, pero la escala no permite una comparación adecuada.

Los gráficos de barras también pueden hacer que los datos parezcan convincentes cuando en realidad podrían carecer de sustancia. Al igual que con todos los datos, verifique la fuente de la que provienen y asegúrese de que provenga de un grupo o muestra lo suficientemente grande.

Por ejemplo, mirar solo unos pocos días de datos de volumen en una acción no proporciona mucha información relevante. Sin embargo, observar cómo se compara el volumen reciente con el volumen del último año proporcionará al comerciante técnico más información para analizar el volumen.

## Reflejos

- Los datos se presentan a través de columnas verticales u horizontales.

- Los gráficos de barras tienen un eje x e y y se pueden usar para mostrar una, dos o muchas categorías de datos.

- Las columnas pueden contener múltiples variables etiquetadas (o solo una), o pueden agruparse (o no) con fines comparativos.

- Se pueden crear gráficos de barras para mostrar datos de múltiples maneras muy visuales.
