# Período base
## ¿Qué es un Período Base?

Un período base es un punto en el tiempo para el cual se recopilan datos y se utilizan como punto de referencia frente a los datos económicos de otros períodos para interpretarlos sobre una base común. Los períodos base se utilizan a menudo en aplicaciones financieras y económicas, como la medición de la inflación u otras variables sujetas a cambios en función del paso del tiempo.

El período base también puede denominarse "período de referencia", "período base" o "período índice".

## Comprender los períodos base

El período base se puede considerar como un criterio común para los datos económicos. En lugar de indicar cada punto de datos de una serie como un número sin procesar, se puede establecer como una proporción o porcentaje del valor de los datos en el período base. El valor para el período base se establece habitualmente como la unidad de medida, generalmente 1 o 100, y todos los demás puntos de datos se reexpresan como valores decimales, fraccionarios o porcentuales del valor de los datos para ese período.

Comparar cada punto de datos con el período base puede ser una forma conveniente de manejar series de datos que consisten en números grandes o complejos. Cada punto de datos en la serie indexada se puede interpretar fácilmente como la proporción, el cambio porcentual o la tasa de crecimiento de la serie de datos subyacente a lo largo del tiempo, en relación con el período base.

## Ejemplo

Por ejemplo, si un [índice de precios](/consumerpriceindex) tiene como [año base](/base-year-analysis) 1990, y los precios actuales se comparan con los precios en ese período de tiempo para construir un índice de serie de tiempo, entonces el nivel de precios en todos los demás años se establecería como un porcentaje de los años 90. nivel de precios. Al índice de precios de 1990 se le podría asignar un valor de 100 y los niveles de precios de otros años tendrían valores proporcionalmente mayores (o menores que) 100 en proporción a la relación de los niveles de precios reales de esos años.

Alternativamente, aunque con menos frecuencia, el período base puede referirse a comparar cada punto de datos con un valor de datos pasado utilizando un intervalo de tiempo constante en lugar de un período base constante. Esta técnica no crea una comparación de índice consistente a lo largo del tiempo, pero puede ayudar a eliminar el efecto de las fluctuaciones estacionales o de corto plazo en los datos. Las comparaciones [año tras año](/year-over-year) o mes tras mes son ejemplos del uso de datos anteriores en un intervalo constante como base para la comparación con los datos actuales.

> El uso de períodos base para indexar datos no está limitado a aplicaciones financieras. Muchas ciencias naturales también utilizan regularmente un período base como parte de sus procesos analíticos. Por ejemplo, para medir los cambios en los patrones climáticos globales, se deben establecer años base.

## Período Base y el Efecto Base

Al construir un índice, la elección de una base de comparación puede influir en cómo se pueden interpretar los datos y debe elegirse cuidadosamente para iluminar el objetivo deseado para el que se utilizan los datos. Valores atípicos o condiciones anormales en un período base pueden dar lugar a comparaciones que distorsionan las tendencias en una serie de datos. Esta distorsión a veces se denomina [efecto base](/base-effect).

Por ejemplo, suponga que la ciudad de Nueva York instituye nuevos códigos de construcción que entran en vigencia el 1 de junio de un año determinado. En el mes de mayo, los constructores se apresuran a iniciar nuevos proyectos de construcción para evitar el gasto de cumplir con los nuevos códigos. Esto podría conducir a una situación en la que los datos de inicios de construcción muestren un valor anormalmente alto para mayo y un valor anormalmente bajo para junio, ya que los constructores adelantan sus cronogramas de construcción, lo que no refleja ninguna tendencia subyacente en los datos, solo uno. fuera del cambio regulatorio.

En este caso, elegir el punto de datos de mayo o junio como año base en una comparación o construir un índice de serie de tiempo conduciría a resultados muy distorsionados en los datos de transformación resultantes, ya que cada punto de datos en el índice comparará los datos actuales con un valor anormalmente alto (o bajo) en el denominador. Un analista haría bien en elegir un valor más típico como período base para la comparación de puntos de datos posteriores.

## Reflejos

- La comparación de otros puntos de datos con un período base constante permite a los analistas detectar cambios y distinguir las tendencias a largo plazo de las fluctuaciones a corto plazo.

- El período base se refiere al punto de referencia contra el cual se miden los datos económicos de otros períodos.

- La elección de un período base puede influir en la perspectiva de un observador sobre los datos, lo que se conoce como efecto base.
