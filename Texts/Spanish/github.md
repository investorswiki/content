# GitHub
GitHub es una plataforma de desarrollo de código abierto basada en la web y un servicio de hospedaje que permite a los usuarios cargar archivos, documentos y códigos de computadora en sus cuentas. GitHub permite a los usuarios registrados y no registrados ver, descargar e incluso contribuir a los archivos cargados y repositorios de código fuente. Debido a su bajo costo y características, GitHub se convirtió en la plataforma de código fuente más grande del mundo, siendo utilizada por muchos desarrolladores como una forma de realizar trabajo colaborativo, al mismo tiempo que podía compartir una variedad de software, archivos y documentación.

#### Cómo funciona GitHub

Una vez que se registra una cuenta de GitHub, el usuario puede crear su propio directorio de archivos y contenido, que se denomina repositorio. Después de crear un repositorio, uno puede crear la llamada rama maestra. Las ramas maestras de GitHub representan el único punto de partida de cualquier proyecto. Depende del usuario hacer que un proyecto esté disponible para cualquier persona (repositorio público) o restringir el acceso al mismo y compartirlo solo con ciertos usuarios (repositorio privado). A partir de una rama maestra, se pueden crear subramas, lo que básicamente permite al usuario trabajar en varios aspectos del proyecto por separado. En un repositorio público, otros usuarios pueden sugerir cambios en el código y eventualmente cargar nuevas versiones de los archivos. Las sugerencias se realizan a través de una función denominada solicitud de extracción, que permite a los desarrolladores discutir y revisar los posibles cambios antes de que se apliquen de manera efectiva. La plataforma GitHub ofrece una variedad de funciones y herramientas que facilitan a los desarrolladores realizar o revertir cambios y realizar un seguimiento de su progreso general.

#### ¿Qué es una solicitud de extracción?

Una solicitud de extracción permite a los usuarios informar a otros sobre los cambios que han realizado en una determinada rama o repositorio. Cuando se crea una solicitud de extracción, se lleva al usuario a una pantalla de descripción general donde puede revisar fácilmente todos los cambios realizados en el código, comparando la versión anterior y la nueva. Por lo tanto, todos los cambios se documentan y registran y los usuarios también pueden agregar comentarios, etiquetas e hitos o incluso asignar tareas específicas a otros colaboradores.

#### ¿Quién usa GitHub?

Debido a su plataforma flexible, GitHub tiene usuarios tanto del sector público como privado. Las empresas de software, los desarrolladores de aplicaciones, los programadores individuales y los desarrolladores de criptomonedas son solo algunos ejemplos. Como proyecto de código abierto, [Bitcoin GitHub](/bitcoin) tiene muchos desarrolladores en todo el mundo, con muchas personas que contribuyen a la revisión y mejora del código fuente de Bitcoin.

#### ¿GitHub es gratuito?

GitHub es gratuito para todos los proyectos de desarrollo comercial y de código abierto. Sin embargo, los planes gratuitos limitan la cantidad de repositorios privados y públicos que una cuenta puede crear. Los planes gratuitos también limitan el número de colaboradores que puede tener un proyecto. Por lo tanto, es posible que los proyectos más grandes deban actualizarse a un plan pago de GitHub, según sus necesidades.
