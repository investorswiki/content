# ICONO
## ¿Qué es ICONO (ICX)?

ICON es una plataforma que pretende facilitar las interacciones de cadenas de bloques independientes, también conocidas como comunidades. Dentro de la plataforma ICON, una comunidad se define como una red de nodos que siguen un único sistema de gobierno. (Un nodo es una computadora que se conecta a una red de criptomonedas).

Las redes de criptomonedas (como Bitcoin y Ethereum) se consideran comunidades, pero los bancos, las empresas, los hospitales, las escuelas y los gobiernos también podrían participar como comunidades dentro de este ecosistema. ICON es compatible con un [token](/cryptocurrency) de criptomoneda, ICX. El Proyecto ICON fue fundado en 2017 por Icon Foundation, una organización con sede en Corea del Sur.

El objetivo del proyecto ICON es facilitar una plataforma donde las entidades de una amplia gama de industrias (financiera, seguridad, seguros, salud, educación y comercio) puedan interactuar y realizar transacciones en una sola red.

## Entendiendo el ICONO (ICX)

Una oferta inicial de monedas (ICO) en septiembre de 2017 para la criptomoneda de Icon, ICX, recaudó aproximadamente $43 millones. Esta venta incluyó el 50% del suministro de tokens de ICON. El proyecto lanzó oficialmente su cadena de bloques en enero de 2018 y los participantes en la ICO recibieron su criptomoneda ICX en junio de 2018.

En 2019, ICON lanzó su propio estándar de token, IRC16, que permite a los usuarios usar el software ICON como infraestructura para crear sus propios activos y valores tokenizados. Estos activos y valores tokenizados son tokens digitales (o "monedas") que representan la propiedad de capital unidades. Con el capital tokenizado, una empresa puede recaudar capital emitiendo sus acciones en forma de activos digitales.

Así es como ICON describe su objetivo: establecer una “nación digital”, en la que diferentes actores económicos puedan emitir y controlar sus propias formas de valor, bajo los sistemas de reglas que seleccionen. Según su sitio web, el objetivo del Proyecto ICON es "hiperconectar el mundo" mediante la "construcción de una de las redes descentralizadas más grandes del mundo " .

El proyecto ICON se inspiró para su proyecto en la forma en que se estructuran las economías del mundo real; las empresas, las organizaciones sin fines de lucro y las instituciones gubernamentales aprovechan un medio común de intercambio, una moneda nacional, sin dejar de ser actores económicos distintos pero interoperables . un punto central, creando una red blockchain interconectada.

Una cadena de [bloques](/blockchain) es un tipo de base de datos. Se diferencia de una base de datos típica en la forma en que almacena los datos. Los datos se almacenan en bloques en orden cronológico: a medida que ingresan nuevos datos, se ingresan en un nuevo bloque. Una vez que el bloque está lleno de datos, ese bloque se encadena al bloque anterior.

Si bien las cadenas de bloques se asocian más comúnmente con las criptomonedas, hay áreas adicionales donde se puede aplicar esta tecnología porque se pueden almacenar diferentes tipos de información en una cadena de bloques. Si bien el uso más común hasta ahora ha sido como libro de transacciones, cualquier intercambio de información se puede agregar a una cadena de bloques. Las transacciones que involucren valores, divisas, contratos comerciales, escrituras, préstamos, propiedad intelectual e identificación personal podrían tokenizarse.

Cada cadena de bloques es administrada por una red específica. Por esta razón, en el pasado, no se consideraba una posibilidad conectar diferentes cadenas de bloques. El proyecto ICON es un intento de tal conexión. A diferencia de otros sistemas de pago centralizados, que requieren que las empresas se adhieran a sus propias políticas y pautas, ICON permite que las comunidades individuales mantengan el control de sus propias políticas. De esta forma, ICON busca reducir las posibles barreras a la adopción.

## Objetivos de ICON

ICON tiene como objetivo crear una red más grande, compuesta por redes individuales, que funcione con criptomonedas. Las cadenas de bloques que participan en el proyecto ICON pueden intercambiar monedas utilizando el intercambio descentralizado (DEX) de la plataforma.

El DEX establece reservas de divisas para cada comunidad de blockchain para que las comunidades puedan intercambiar valor en tiempo real. Los tipos de cambio se establecen para estas transacciones utilizando un modelo de análisis de IA y la moneda ICX nativa se utiliza como moneda intermediaria para el intercambio.

El proyecto ICON consta de cinco componentes principales: ICON Republic, comunidades ICON, representantes comunitarios (C-Rep), nodos comunitarios (C-Nodes) y nodos ciudadanos. Las comunidades ICON son una red de nodos dentro de un único sistema de gobierno; pueden operar con sus propias estructuras de gobierno, número de nodos y características. Las comunidades también pueden variar en sus enfoques para la toma de decisiones. Por ejemplo, mientras que bitcoin adopta un enfoque basado en el consenso, otras comunidades, como las instituciones financieras, pueden seguir un enfoque jerárquico.

Los nodos comunitarios son la infraestructura de cada comunidad; mantienen la cadena de bloques de una comunidad. Los representantes de la comunidad son nodos que se eligen dentro de cada comunidad y se confía en ellos para interactuar con ICON Republic. A través del representante de la comunidad, la cadena de bloques de cada comunidad está vinculada a ICON Republic. Representantes de la comunidad reciben ICX por su trabajo.

ICON Republic es la estructura de gobierno de la red. También es el comité de decisión para las operaciones de la cadena de bloques. El gobierno de ICON Republic está determinado por los votos de los representantes de la comunidad. Una analogía útil es imaginar ICON Republic como los Estados Unidos e imaginar cada comunidad ICON como un estado separado.

Las acciones de ICON Republic no determinan el gobierno de las propias comunidades separadas. Sin embargo, ICON Republic rige el proceso mediante el cual se acuña la nueva criptomoneda ICX.

Los nodos ciudadanos son participantes en la cadena de bloques que no tienen derecho a voto pero pueden realizar transacciones dentro de una comunidad, con otras comunidades y con Icon Republic.

La cadena de bloques de ICON Republic se conoce como el "nexo". La tecnología que impulsa el nexo es un algoritmo llamado loopchain. Loopchain conecta comunidades que forman parte de ICON Republic. Estas comunidades, cuando se agrupan, se denominan consorcios, determinan un conjunto común de reglas que permiten que las diferentes cadenas de bloques funcionen juntas de manera efectiva.

El conjunto de reglas que rigen cómo interactúan las cadenas de bloques independientes con la cadena de bloques de ICON Republic, el nexo, se denomina Protocolo de transmisión de cadena de bloques (BTP).

Finalmente, al igual que con [Ethereum](/ethereum) [,](/ethereum) cualquiera puede usar ICON para crear aplicaciones descentralizadas, llamadas DAPP, a través del canal público de la cadena de bloques. Además, después de que estas aplicaciones se incluyan en la tienda DAPP de ICON, cualquier persona con un nodo ciudadano puede descargarlas y usarlas.

## Críticas a ICON

Si bien ICON tiene planes ambiciosos para conectar diferentes comunidades, enfrenta muchos desafíos para lograr este objetivo. En primer lugar, muchos inversores están acostumbrados a los intercambios de criptomonedas centralizados tradicionales. Si bien es cierto que estos intercambios centralizados tradicionales no encajan con la ideología descentralizada de las criptomonedas, en 2021, estos intercambios siguen siendo una parte integral del mercado de criptomonedas. Muchos inversores solo compran criptomonedas que pueden comprar a través de sus intercambios favoritos. Esto podría crear problemas para ICX. Dados los objetivos de ICON, es posible que muchos intercambios no quieran ofrecer ICX a sus clientes.

Los altos niveles de competencia aumentan el riesgo de que una criptomoneda en particular no tenga éxito. Al igual que otras criptomonedas, ICON enfrenta una dura competencia de otras criptomonedas.

## Reflejos

- Las comunidades están conectadas a ICON Network a través de un intercambio descentralizado.

- La posible oposición de los intercambios de criptomonedas centralizados tradicionales es un desafío importante que enfrenta ICON.

- ICON es una tecnología de cadena de bloques y un marco de red diseñado para permitir que las cadenas de bloques independientes interactúen entre sí.
