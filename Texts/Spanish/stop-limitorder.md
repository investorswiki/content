# Orden de límite de parada
## ¿Qué es una orden stop-limit?

Una orden stop-limit es una operación condicional durante un período de tiempo determinado que combina las características de stop con las de una orden limitada y se utiliza para mitigar el riesgo. Está relacionado con otros tipos de órdenes, incluidas las órdenes limitadas (una orden para comprar o vender una cantidad específica de acciones a un precio determinado o mejor) y las órdenes stop-on-quote (una orden para comprar o vender un valor después de su vencimiento). precio ha superado un punto especificado).

## Cómo funcionan las órdenes Stop-Limit

Una orden stop-limit requiere el establecimiento de dos puntos de precio:

1. Stop: El inicio del precio objetivo especificado para la operación.

1. Límite: El exterior del precio objetivo para la operación.

También se debe establecer un marco de tiempo durante el cual la orden stop-limit se considera ejecutable.

El principal beneficio de una orden stop-limit es que el [comerciante](/trader) tiene un control preciso sobre cuándo se debe ejecutar la orden.

La [desventaja](/downside),. como ocurre con todas las órdenes limitadas, es que no se garantiza que la operación se ejecute si la acción o la materia prima no alcanza el precio tope durante el período de tiempo especificado.

La orden de límite de parada se ejecutará a un precio específico, o mejor, después de que se haya alcanzado un precio de parada determinado. Una vez que se alcanza el precio de detención, la orden de límite de detención se convierte en una [orden de límite](/limitorder) para comprar o vender al precio de límite o mejor. Este tipo de orden es una opción disponible con casi todos los corredores en línea.

## Características de las órdenes Stop y Limit

Una [orden stop](/stoporder) es una orden que se vuelve ejecutable una vez que se alcanza un precio establecido y luego se completa al precio de mercado actual. Una orden stop tradicional se completará en su totalidad, independientemente de cualquier cambio en el precio de mercado actual a medida que se completen las operaciones.

Una orden limitada es aquella que se fija a un precio determinado. Sólo es ejecutable en momentos en que la operación puede realizarse al precio límite oa un precio que se considere más favorable que el precio límite. Si la actividad comercial hace que el precio se vuelva desfavorable con respecto al precio límite, entonces cesará la actividad relacionada con la orden.

Al combinar las dos órdenes, el [inversor](/investor) tiene una precisión mucho mayor en la ejecución de la operación.

Una orden stop se completa [al](/atthemarket) precio de mercado después de que se haya alcanzado el precio stop, independientemente de si el precio cambia a una posición desfavorable. Esto puede llevar a que las transacciones se completen a precios inferiores a los deseables si el mercado se ajusta rápidamente. La combinación de la orden stop con las características de una orden limitada garantiza que la orden no se ejecutará una vez que el precio se vuelva desfavorable, según el límite del inversionista. Por lo tanto, en una orden stop-limit, después de que se activa el precio stop, la orden límite entra en vigor para garantizar que la orden no se complete a menos que el precio sea igual o mejor que el precio límite especificado por el inversor.

## Ejemplo del mundo real de una orden de límite de parada

Por ejemplo, suponga que Apple Inc. (AAPL) cotiza a $ 155 y que un inversionista quiere comprar las acciones una vez que comience a mostrar un impulso alcista importante. El inversionista ha puesto una orden stop-limit para comprar con el precio stop en $160 y el precio límite en $165. Si el precio de AAPL se mueve por encima del precio stop de $160, la orden se activa y se convierte en una orden limitada. Siempre que la orden se pueda completar por debajo de $ 165, que es el precio límite, la operación se completará. Si las brechas de existencias superan los $ 165, entonces el pedido no se completará.

Las órdenes stop-limit de compra se colocan por encima del precio de mercado en el momento de la orden, mientras que las órdenes stop-limit de venta se colocan por debajo del precio de mercado.

## Reflejos

- Las órdenes stop-limit son una operación condicional que combina las características de un stop loss con las de una orden limitada para mitigar el riesgo.

- Los comerciantes a menudo usan órdenes de límite de parada para asegurar las ganancias o limitar las pérdidas a la baja.

- Las órdenes stop-limit permiten a los operadores tener un control preciso sobre cuándo se debe completar la orden, pero no se garantiza que se ejecuten.

## PREGUNTAS MÁS FRECUENTES

### ¿Cuánto duran las órdenes stop-limit?

Las órdenes stop-limit se pueden establecer como órdenes diarias, en cuyo caso vencerían al final de la sesión de mercado actual, o [como órdenes válidas hasta que se cancelen (GTC)](/gtc),. que se transfieren a futuras sesiones de negociación. Las diferentes plataformas de operaciones y corredurías tienen distintos vencimientos para las órdenes GTC, así que verifique el período de tiempo en el que su orden GTC será válida.

### ¿Las órdenes stop-limit funcionan fuera de horario?

Las órdenes de stop-loss solo se activarán durante el horario estándar del mercado, que generalmente es de 9:30 a. m. a 4:00 p. m., hora del Este. No se ejecutarán durante sesiones de horario extendido o cuando el mercado esté cerrado los fines de semana y días festivos.

### ¿Cuál es la diferencia entre una orden stop-loss y una orden stop-limit?

Una orden stop-loss asegura la ejecución, mientras que una orden stop-limit asegura una ejecución al precio deseado. La decisión sobre qué tipo de orden usar depende de una serie de factores. Una orden de límite de pérdida se activará al precio de mercado una vez que se haya incumplido el nivel de límite de pérdida. Un inversor con una posición larga en un valor cuyo precio se está desplomando rápidamente puede descubrir que el precio al que se ejecutó la orden de límite de pérdidas está muy por debajo del nivel en el que se estableció el límite de pérdidas. Esto puede ser un riesgo importante cuando una acción tiene una brecha a la baja, por ejemplo, después de un informe de ganancias, para una posición larga; por el contrario, una brecha alcista puede ser un riesgo para una posición corta. Una orden stop-limit combina las características de una orden stop-loss y una orden limitada. El inversor especifica el precio límite, asegurando así que la orden stop-limit solo se ejecutará al precio límite o mejor. Sin embargo, como con cualquier orden de límite, el riesgo aquí es que la orden no se complete en absoluto, dejando al inversor atrapado en una posición de pérdida de dinero.

### ¿Cuál es un ejemplo de una orden stop-limit utilizada para una posición corta?

Una posición [corta](/short) necesitaría una orden límite de compra-stop para limitar las pérdidas. Por ejemplo, si un comerciante tiene una posición corta en la acción ABC a $50 y le gustaría limitar las pérdidas entre un 20 % y un 25 %, puede ingresar una orden de límite de parada para comprar a un precio de $60 y un precio límite de $62,50. Si las acciones cotizan a un precio de $60 a $62,50, entonces se ejecutará la orden de límite de parada, limitando la pérdida del comerciante en la posición corta en el rango deseado de 20% a 25%. Sin embargo, si las acciones aumentan, por ejemplo, a $ 65, entonces la orden de límite de parada no se ejecutará y la posición corta permanecerá abierta.
