# Retorno anual
## ¿Qué es una declaración anual?

El rendimiento anual es el [rendimiento](/return) que proporciona una inversión durante un período de tiempo, expresado como un porcentaje anual ponderado en el tiempo. Las fuentes de rendimiento pueden incluir dividendos, rendimientos de capital y revalorización del capital. La tasa de rendimiento anual se mide con respecto al monto inicial de la inversión y representa una [media geométrica](/geometricmean) en lugar de una simple [media aritmética](/arithmeticmean).

## Comprender la rentabilidad anual

El método de facto para comparar el rendimiento de las inversiones con la [liquidez](/liquidity),. se puede calcular un rendimiento anual para varios activos, que incluyen acciones, bonos, fondos, materias primas y algunos tipos de [derivados](/derivative). Este proceso es un método preferido, considerado más preciso que una declaración simple, ya que incluye ajustes por intereses compuestos. Se considera que diferentes clases de activos tienen diferentes estratos de rendimientos anuales.

## Rendimientos anuales de las acciones

También conocido como [rendimiento anualizado](/annualized-total-return),. el rendimiento anual expresa el aumento de valor de las acciones durante un período de tiempo designado. Para calcular un rendimiento anual, se requiere información sobre el precio actual de la acción y el precio al que se compró. Si se han producido divisiones, el precio de compra debe ajustarse en consecuencia. Una vez que se determinan los precios, primero se calcula el porcentaje de retorno simple, y finalmente se anualiza esa cifra. La devolución simple es solo el precio actual menos el precio de compra, dividido por el precio de compra.

El rendimiento anualizado varía del promedio típico y muestra la ganancia o pérdida real de una inversión, así como la dificultad para recuperar las pérdidas. Por ejemplo, perder el 50 % en una inversión inicial requiere una ganancia del 100 % el próximo año para compensar la diferencia. Debido a la considerable diferencia en las ganancias y pérdidas que pueden ocurrir, los rendimientos anualizados ayudan a nivelar los resultados de inversión para una mejor comparación.

> Las estadísticas de rendimiento anual se citan comúnmente en materiales promocionales para fondos mutuos, [ETF](/etf) y otros valores individuales.

## Rendimientos anuales en un 401K

El cálculo difiere cuando se determina el rendimiento anual de un [401K](/401kplan) durante un año específico. En primer lugar, se debe calcular el rendimiento total. Se necesita el valor inicial para el período de tiempo que se examina, junto con el valor final. Antes de realizar los cálculos, se debe restar del valor final cualquier aporte a la cuenta durante el período de tiempo en cuestión.

Una vez determinado el valor final ajustado, se divide por el saldo inicial. Finalmente, reste 1 del resultado y multiplique esa cantidad por 100 para determinar el porcentaje de retorno total.

## Reflejos

- Se puede determinar un rendimiento anual para una variedad de activos, incluidas acciones, bonos, fondos mutuos, ETF, materias primas y ciertos derivados.

- Un rendimiento anual o anualizado es una medida de cuánto ha aumentado una inversión en promedio cada año, durante un período de tiempo específico.

- Un rendimiento anual puede ser más útil que un simple rendimiento cuando desea ver el rendimiento de una inversión a lo largo del tiempo o comparar dos inversiones.

- El rendimiento anualizado se calcula como un promedio geométrico para mostrar cómo sería el rendimiento anual compuesto.
