# Internet oscura
## ¿Qué es la web oscura?

El término web oscura se refiere al contenido en línea encriptado que no está indexado por los motores de búsqueda convencionales. El acceso a la web oscura solo se puede hacer usando navegadores específicos, como el navegador TOR. Hay una gran cantidad de privacidad y anonimato que conlleva el uso de la web oscura en comparación con los sitios web tradicionales.

Como tal, la mayor parte de la atención se centra en los [mercados en línea](/market) de drogas, [intercambios](/exchange) de datos robados y otras actividades ilegales cuando las personas piensan en la web oscura. A pesar de esto, a menudo hay razones muy legítimas por las que las personas eligen usar la web oscura, incluidos los disidentes políticos y las personas que desean mantener cierta información privada.

## Comprender la web oscura

Como su nombre lo indica, la web oscura es una red secreta que existe bajo tierra. Se compone de una serie de sitios web que están ocultos al público en general. Esto significa que no son accesibles a través de los motores de búsqueda tradicionales, como Google.

Los motores de búsqueda tradicionales devuelven resultados porque contienen índices de enlaces a sitios web. Estos se clasifican según las palabras clave y la relevancia. La web oscura, por otro lado, utiliza información que no está disponible en estos otros motores de búsqueda, como contenido de cuentas individuales, como correo electrónico, [redes sociales](/social-media),. banca, junto con bases de datos personales y profesionales, y documentos (legales y médico).

También llamada darknet, la web oscura es muy parecida a la web más amplia en sus inicios a fines del siglo XX. Hay mucho material sobre cómo hacerlo funcionar, y no mucho que hacer una vez que uno llega allí. Gran parte del contenido de la web oscura es muy amateur. Por otro lado, es mucho más fácil para las personas iniciar sitios y llamar la atención. Los gigantes tecnológicos y las grandes organizaciones de medios tienen muy poca influencia en la web oscura a partir de 2020.

Al igual que con los inicios de Internet, la web oscura tiene la reputación de ser un refugio para actividades ilegales y con frecuencia está implicada en actividades ilícitas y delictivas. Si bien la web oscura ha jugado un papel en las transacciones ilegales y poco éticas, también proporciona una salida social para las personas que, de lo contrario, podrían ser perseguidas por sus identidades o creencias políticas. También brinda a las autoridades legales las herramientas adicionales que necesitan para detener a los perpetradores de actividades poco éticas.

> La web oscura todavía es un trabajo en progreso, y aún no se conocen sus costos y beneficios totales.

>

## Consideraciones Especiales

La gente suele confundir la web oscura con las [criptomonedas](/cryptocurrency),. que suelen utilizarse para realizar compras allí. Pero hay una clara diferencia. La web oscura facilita la configuración y el acceso a sitios web que ofrecen un alto grado de anonimato para todos los involucrados.

Muchos sitios web oscuros contienen solo información en lugar de la capacidad de comprar o vender algo. Sin embargo, es cierto que las criptomonedas, como [Bitcoin](/bitcoin) y [Monero](/monero),. a menudo se usan para transacciones en la web oscura. Pero uno no tiene que usar la web oscura para usar criptomonedas.

## Historia de la Web Oscura

La web oscura apareció oficialmente por primera vez a principios de la década de 2000 junto con la creación de Freenet, que fue desarrollada por Ian Clarke para proteger a los usuarios contra la intervención del gobierno y los ataques cibernéticos. El sistema, que todavía está disponible hoy, permite a los usuarios expresarse libremente sin ser rastreados en línea.

El Laboratorio de Investigación Naval de EE. UU. financió un proyecto llamado The Onion Router (TOR). TOR ofreció a las fuentes de inteligencia una forma de comunicarse de manera fácil y segura, especialmente en áreas hostiles donde la seguridad personal es clave. Ahora es uno de los navegadores más comunes que se utilizan para acceder a la web oscura y utiliza bases de datos para ayudar a las personas a navegar y encontrar la información que necesitan.

El auge de las criptomonedas aumentó la popularidad de la dark web, especialmente para los ciberdelincuentes. Esto se debe a que las monedas digitales a menudo brindan una gran cantidad de anonimato para las personas que compran y venden en la web oscura.

Debido a su asociación con ciertas actividades ilícitas, ha habido llamados para la regulación de la web oscura. Por ejemplo, el [G20](/g-20) y el [Grupo de Acción Financiera Internacional (FATF, por sus siglas en inglés)](/financial-action-task-force-fatf) han pedido a las empresas de criptomonedas que brinden información sobre compradores y vendedores en transacciones realizadas en línea. Esto es especialmente cierto, dicen, para ayudar a las fuerzas del orden público a rastrear organizaciones criminales y actividades ilícitas.

> Ross Ulbricht fue investigado y arrestado en 2013 por crear y administrar el sitio web oscuro The Silk Road, un sitio web oscuro que, según las autoridades, se usaba para comprar y vender narcóticos.

>

## Web oscura versus Web profunda

La dark web y la deep web a menudo también se usan erróneamente de manera intercambiable. La web oscura es, de hecho, solo una parte de la web profunda. La web profunda es lo que se encuentra bajo tierra y no es solo el contenido que se considera oscuro. Por eso también se le llama la telaraña invisible o la telaraña oculta.

La información que se encuentra en la web profunda normalmente está encriptada y no se encuentra en los índices como en la web tradicional. Incluye todas las páginas que no aparecen cuando realiza una búsqueda web. También contiene todo lo que requiere un inicio de sesión, como contenido de:

- Banca en línea

- Sitios web de pago, como Netflix y Amazon Prime

- Servicios de alojamiento de archivos, como Dropbox y sus competidores

- Bases de datos privadas

La información que figura en la web oscura también puede incluir cualquier cosa en bases de datos solo para suscriptores en lugar de cualquier cosa ilegal.

## Ventajas y desventajas de la web oscura

### Ventajas

La web oscura ayuda a las personas a mantener la privacidad y expresar libremente sus puntos de vista. La privacidad es esencial para muchas personas inocentes aterrorizadas por acosadores y otros delincuentes. La creciente tendencia de los empleadores potenciales a rastrear las publicaciones en las redes sociales también puede dificultar la participación pública en discusiones honestas.

Finalmente, la popularidad de la web oscura entre los delincuentes la convierte en una forma perfecta para que los policías encubiertos se comuniquen.

### Desventajas

Algunas personas inevitablemente abusarán del poder asociado con el uso de la web oscura al facilitar la participación en actividades delictivas. Por ejemplo, la combinación de la web oscura y las criptomonedas teóricamente hace que sea mucho más fácil contratar a alguien para cometer ciertos delitos.

Si bien la web oscura promete privacidad a sus usuarios, también puede usarse para violar la privacidad de otros. Las fotos privadas, los registros médicos y la información financiera han sido robados y compartidos en la web oscura.

## Ejemplo de la web oscura

Digamos que desea ingresar a la web oscura para ver qué información suya, si la hay, está flotando. Puede instalar el navegador TOR en su sistema utilizando una red privada virtual. Una VPN permite que las personas usen una red pública como si estuvieran transmitiendo información en una privada. Básicamente te da privacidad en una red pública.

Mientras navega, probablemente notará que las direcciones oscuras de los sitios web terminan con la extensión .onion en lugar de los tradicionales .com, .org, .edu, etc. Sin embargo, una palabra de advertencia. Asegúrese de tener algún software de seguridad instalado porque existe una buena posibilidad de que pueda entrar en contacto con malware y estafas de phishing.

## La línea de fondo

La web oscura es un concepto relativamente nuevo, al menos si se considera la historia de Internet en su conjunto. Puede usarlo para permanecer en el anonimato sin temor a las repercusiones si desea expresarse libremente. Pero esta cantidad de privacidad lo ha ayudado a ganar una mala reputación como foro para aquellos que quieren usarlo para actividades ilícitas. Si está buscando usar la web oscura, es tan simple como instalar un navegador privado, usar una VPN y asegurarse de que su computadora permanezca segura y protegida.

## Reflejos

- Se requieren navegadores específicos, como Tor Browser, para acceder a la dark web.

- También tiene reputación de estar asociado con actividades ilícitas y poco éticas.

- La web oscura abre sitios que utilizan información que no está indexada en línea, como cuentas bancarias, cuentas de correo electrónico y bases de datos.

- Las personas que usan la web oscura pueden mantener su privacidad y expresar libremente sus opiniones.

## PREGUNTAS MÁS FRECUENTES

### ¿Cómo entras en la Dark Web?

Puede acceder a la web oscura instalando navegadores específicos y anónimos, como TOR. Una vez instalado, el navegador funciona de la misma forma que los tradicionales. Pero puede ser un poco difícil acceder a la información porque no utiliza un índice para ubicar la información deseada. Asegúrese de instalar un software de seguridad para asegurarse de que su computadora y su información personal estén seguras.

### ¿Qué haces si tu información está en la Dark Web?

Puede tomar una serie de pasos para ayudar a proteger su información. Asegúrese de cambiar todas sus contraseñas, revise sus informes de crédito para ver si hay discrepancias e informe a su(s) banco(s). También puede congelar su crédito para evitar que se vea más comprometido.

### ¿Cómo saber si su información está en la Dark Web?

Su información personal, como la banca, las redes sociales y las cuentas de correo electrónico, junto con su número de Seguro Social, pueden estar a la venta en la web oscura. Si desea ver si parte de su información está flotando en la web oscura, intente ejecutar un escaneo de la web oscura. Alternativamente, puede registrarse en un servicio de monitoreo de la web oscura. Estas herramientas le advierten cada vez que encuentran algo de su información.

### ¿Es ilegal acceder a la Dark Web?

A pesar de lo que su nombre indica, no es ilegal acceder a la dark web. En realidad, brinda a las personas privacidad y anonimato que los sitios web tradicionales no ofrecen a las personas. Por ejemplo, las personas pueden ingresar a la web oscura y publicar sus pensamientos sobre la actividad política sin temor a ser reprendidos por funcionarios gubernamentales y otros grupos.
