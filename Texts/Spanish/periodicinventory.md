# Inventario Periódico
## ¿Qué es el inventario periódico?

El sistema de inventario periódico es un método de valoración de [inventario](/inventory) con fines de información financiera en el que se realiza un recuento físico del inventario a intervalos específicos. Este método de contabilidad toma el inventario al comienzo de un período, agrega nuevas compras de inventario durante el período y deduce el inventario final para derivar el costo de los bienes vendidos (COGS).

## Comprender el inventario periódico

Bajo el sistema de inventario periódico, una empresa no conocerá sus niveles de inventario unitario ni COGS hasta que se complete el proceso de conteo físico. Este sistema puede ser aceptable para una empresa con un bajo número de [SKU](/stock-keeping-unit-sku) en un mercado de movimiento lento, pero para todos los demás, el sistema de [inventario perpetuo](/perpetualinventory) se considera superior por las siguientes razones principales:

1. El sistema perpetuo actualiza continuamente el [libro mayor de activos de inventario](/asset-ledger) en el sistema de base de datos de una empresa, brindando a la gerencia una vista instantánea del inventario; el sistema periódico requiere mucho tiempo y puede producir números obsoletos que son menos útiles para la administración.

1. El sistema perpetuo mantiene actualizados los COGS a medida que ocurren los movimientos de inventario; el sistema periódico no puede dar cifras COGS precisas entre períodos de conteo.

1. El sistema perpetuo realiza un seguimiento de los artículos de inventario individuales para que, en caso de que haya artículos defectuosos, por ejemplo, se pueda identificar rápidamente el origen del problema; lo más probable es que el sistema periódico no permita una resolución rápida.

1. El sistema perpetuo está basado en tecnología y los datos se pueden respaldar, organizar y manipular para generar informes informativos; el sistema periódico es manual y más propenso a errores humanos, y los datos pueden extraviarse o perderse.

## Consideraciones especiales: COGS

El costo de los bienes vendidos, comúnmente conocido como COGS, es una cuenta fundamental del estado de resultados, pero una empresa que utiliza un sistema de inventario periódico no sabrá el monto de sus registros contables hasta que se complete el conteo físico.

> El COGS variará drásticamente con los niveles de inventario, ya que a menudo es más barato comprar a granel, si tiene espacio de almacenamiento para acomodar.

>

Suponga que una empresa tiene un inventario inicial de $500 000 el 1 de enero. La empresa compra $250 000 de inventario durante un período de tres meses y, después de una cuenta de inventario físico, determina que tiene un inventario final de $400 000 al 31 de marzo, que se convierte en el inventario inicial. cantidad para el próximo trimestre. El COGS para el primer trimestre del año es de $350 000 ($500 000 al principio + $250 000 de compras - $400 000 al final).

Debido a las discrepancias de tiempo, se convierte en responsabilidad del gerente o propietario de la empresa responsable de monitorear el inventario del período si tiene sentido para su resultado final asignar horas para contar el inventario diariamente, semanalmente, mensualmente o anualmente.
