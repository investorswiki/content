# BEP-721
BEP-721 es un estándar de token Binance Smart Chain (BSC) que permite la creación de tokens no fungibles (NFT). Es una extensión de ERC-721 de Ethereum, uno de los estándares NFT más utilizados y es compatible con Ethereum Virtual Machine (EVM). Cada NFT es único y no se intercambia con ningún otro token.

Los tokens BEP-721 le permiten tokenizar la propiedad de los datos y adjuntarles un identificador único. Este aspecto hace que el token sea único y muy diferente de los tokens BEP-20. Con BEP-20, los desarrolladores pueden crear múltiples tokens idénticos dentro de un contrato inteligente. Sin embargo, con BEP-721, a cada token se le asigna una ID de token diferente.

A través de estas identificaciones únicas, los tokens BEP-721 se pueden usar para representar artículos coleccionables no fungibles. Los usuarios pueden intercambiar y transferir estos tokens en función de su valor de mercado, que depende de la rareza o utilidad del token.

Los tokens BEP-721 pueden representar:

1. Arte digital y físico

1. Coleccionables

1. Elementos del juego

1. Propiedad física e inmobiliaria

1. Billetes de lotería

Al igual que otros tokens en Binance Smart Chain, las transferencias de tokens BEP-721 requieren BNB para las tarifas de gas. Los tokens BEP-721 se pueden crear en varias plataformas en el ecosistema BSC, incluido Binance NFT Marketplace, Featured By Binance, BakerySwap y Juggerworld.

## Funciones del token BEP-721

BEP-721 tiene una lista de funciones que rigen cómo interactúa el token con Binance Smart Chain. Algunos de estos son bastante estándar y también están presentes en el estándar BEP-20:

- nombre: Define el nombre del token BEP-721, por el cual otros contratos lo identificarán

- Símbolo: un nombre más corto para el token similar a un símbolo de cotización

- balanceOf: Muestra el saldo de fichas de una dirección específica

- totalSupply: Define el número total de tokens creados.

También hay un conjunto de funciones básicas de propiedad, algunas de las cuales son opcionales de incluir. Una característica única de las NFT es la función de metadatos:

tokenMetadata: esta función permite la inclusión de metadatos en un token utilizado para vincular a un archivo de arte u otro aspecto coleccionable del NFT.

Por ejemplo, cada CryptoPunk tiene metadatos contenidos dentro de la NFT que lo señalan a un punk en particular en una cuadrícula de 10,000 punks. Aunque CryptoPunks no usa el estándar de token BEP-721, su uso de metadatos es el mismo.
