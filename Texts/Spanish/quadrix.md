# cuadrix
## ¿Qué es Quadrix?

Quadrix es un sistema de clasificación de acciones que utiliza más de 90 variables en siete categorías principales para determinar el valor de una acción. El sistema Quadrix es producido y mantenido por Horizon Publishing Company. Las siete categorías principales son impulso, calidad, valor, fortaleza financiera, estimaciones de ganancias, rendimiento y reversión. El puntaje general para una acción en particular está determinado por un promedio ponderado de las 90 variables.

El sistema Quadrix fue desarrollado por el director de inversiones de Horizon Investment Services, Rich Moroney, y ha estado publicando puntajes para acciones desde el año 2000.

## Cómo funciona Quadrix

Quadrix es una herramienta de evaluación de acciones que califica las acciones en función de puntajes variables que se clasifican en una de siete categorías. Las siete categorías de variables utilizadas en Quadrix son impulso, calidad, valor, solidez financiera, ganancias previstas, rendimiento y reversión. El sistema Quadrix también se puede utilizar como una herramienta para realizar un seguimiento del rendimiento del grupo industrial.

Quadrix es una primera pantalla de acciones para crear carteras. El sistema analiza el conjunto dado de 90 variables en un universo de más de 4.000 acciones. Su objetivo es reducir el universo de acciones aceptables bajo la consideración de un inversor. No es un creador de carteras basado en el riesgo como algunos asesores robóticos.

## Ejemplos y usos de Quadrix

Quadrix es utilizado internamente por Horizon Publishing para evaluar las acciones de los clientes y está disponible para los suscriptores de los productos de boletines de Horizon. Como se mencionó anteriormente, no es un creador de carteras basado en el riesgo y no se basa en [la Teoría Moderna de Carteras](/modernportfoliotheory) (MPT) para ponderar acciones o evaluar el riesgo versus el rendimiento para el inversor.
