# Protocolo 0x
## ¿Qué es el protocolo 0x?

El protocolo 0x (cero x) es un protocolo que permite el intercambio de activos entre pares en la cadena de bloques de Ethereum. El protocolo 0x fue lanzado en 2017 por 0x Labs, una organización con sede en San Francisco enfocada en crear nuevos mercados en el ecosistema 0x.

El protocolo en sí no es, por definición, un intercambio descentralizado; permite la creación de intercambios descentralizados que se pueden utilizar en aplicaciones amplias.

## Entendiendo el Protocolo 0x

En el mundo de las monedas digitales, un protocolo es un conjunto de reglas estándar que puede usar un sistema (o varias partes que realizan transacciones) para comunicarse entre sí sin problemas. El protocolo 0x es esencialmente un formato de mensajería estándar que las entidades que realizan transacciones pueden usar para intercambiar activos digitales o tokens.

El sistema de mensajería SWIFT estándar que se usa en la industria bancaria es similar a un protocolo (excepto que se usa en un contexto diferente). Los bancos de todo el mundo utilizan el sistema de mensajería [SWIFT estándar](/swift) para comunicarse sobre transferencias de dinero.

El sistema tiene un conjunto estándar de campos de mensajes definidos para todos los usuarios; sus valores correspondientes se utilizan en el sistema SWIFT para transmitir de forma segura detalles entre dos partes, como el remitente, el destinatario, el monto, la moneda, la sucursal de origen y la sucursal de destino, entre otros.

> SWIFT crea un formato estándar y universalmente aceptable que permite operaciones más fluidas y una mayor eficiencia.

>

Debido a que cada banco se ha adaptado a las reglas estándar del sistema de mensajería SWIFT, pueden realizar transacciones entre ellos directamente. La situación saldría mal si cada banco siguiera su propio protocolo distinto porque entonces tendrían que adherirse a un canal de comunicación uno a uno con todos los demás bancos.

## Goles de 0x

El objetivo de 0x Labs es crear la infraestructura necesaria para la economía de criptomonedas emergente y permitir la creación de mercados que no podrían haber existido antes.

El protocolo 0x intenta funcionar de manera similar a SWIFT, pero en el contexto de operar intercambios descentralizados para intercambiar tokens digitales y activos que se ejecutan en la cadena de bloques Ethereum. Construido sobre la base de los estándares de token de Ethereum, el protocolo 0x actúa como la capa de infraestructura clave para la creciente cantidad de aplicaciones e instrumentos financieros que se crean utilizando la tecnología blockchain y el comercio en formas digitales.

Cada día que pasa, los tokens se están convirtiendo en una parte cada vez mayor del valor financiero mundial. En consecuencia, el requisito de comercializar tokens y activos digitales de forma segura y eficiente está creciendo. Con sus formatos de mensaje claramente definidos y contratos inteligentes, el protocolo 0x intenta responder a esta necesidad.

> La palabra "ficha" a menudo se usa indistintamente con "moneda". Sin embargo, una ficha es diferente de una moneda porque representa un activo, mientras que una moneda se usa como medio de intercambio.

>

Se utiliza en una amplia gama de mercados, como juegos y financiación, que permite a los usuarios intercambiar tokens y activos.

ZRX es el token nativo de gobernanza y participación de 0x. Los propietarios de ZRX tienen algo que decir sobre cómo evoluciona el protocolo, y los poseedores de tokens también pueden apostar sus tokens para obtener recompensas de liquidez de ETH (el token ether de Ethereum).

## Consideraciones Especiales

El formato de mensaje del protocolo 0x se compone de un conjunto de campos de datos que contienen información vital, como el activo digital o el token que se comercializará, el valor del precio de la transacción, el tiempo de vencimiento y las identidades definidas de las partes que realizan la transacción.

Los contratos inteligentes se encargan de la lógica comercial necesaria para generar, enviar, recibir y procesar los datos vinculados a la actividad comercial. También deja espacio para actualizaciones esenciales, si las hay, en el futuro. Las disposiciones para las actualizaciones son necesarias si es necesario realizar cambios para cumplir con las regulaciones modificadas o cualquier modificación relacionada con el funcionamiento intrínseco de la red blockchain de Ethereum.

El sistema también utiliza retransmisores, que actúan como agregadores de pedidos y transmiten pedidos de participantes del mercado designados al mercado o bolsa.

## Reflejos

- El protocolo 0x es un protocolo abierto que permite el intercambio de activos entre pares en la cadena de bloques de Ethereum.

- Construido sobre la base de los estándares de token de Ethereum, el protocolo 0x actúa como la capa de infraestructura crítica para el creciente número de aplicaciones financieras que implementan la pila de tecnología blockchain.

- 0x Labs, una organización enfocada en crear nuevos mercados en el ecosistema 0x, creó el protocolo 0x.

## PREGUNTAS MÁS FRECUENTES

### ¿Es 0x un DEX?

Es similar a un intercambio descentralizado pero tiene diferencias únicas. Una de las diferencias es que DEX almacena pedidos en su cadena de bloques, mientras que 0x no.

### ¿Puedo minar 0x?

0x es un protocolo para la cadena de bloques de Ethereum. Facilita los intercambios entre pares de activos creados con Ethereum, por lo que no hay token para minar.

### ¿Qué es la moneda Zerox?

Creado por la empresa 0x (cero x), ZRX es un token creado en la cadena de bloques de Ethereum para impulsar los intercambios de criptomonedas.
