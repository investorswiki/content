# Base de arrastre
## ¿Qué es una base de transferencia?

Una base de transferencia es un método para determinar la [base fiscal](/costbasis) de un activo cuando se transfiere de un individuo a otro. A menudo se utiliza una base de traspaso cuando una de las partes deja bienes o propiedades a otra persona como regalo. En esta situación, la base a menudo sigue siendo la misma que cuando el donante tenía el activo, pero la base puede ajustarse para contabilizar los impuestos sobre donaciones que se pagaron.

## Comprender la base de transferencia

La base de traspaso difiere de [la base de incremento](/stepupinbasis). Se utiliza una base de traspaso durante la vida del donante, mientras que una base progresiva se utiliza cuando un activo se hereda después de que el donante fallece. En un escenario de base step-up, el valor de los activos que se transfieren se ajusta a su valor de mercado actual.

La base del costo de una inversión es el monto total invertido originalmente, más cualquier [comisión](/commission) o tarifa involucrada en la compra. Esto puede describirse en términos del monto en dólares de la inversión o el precio efectivo por acción de la inversión.

Determinar la base de costo correcta de una inversión, también conocida como base impositiva, es importante, especialmente si reinvirtió distribuciones de dividendos y [ganancias de capital](/capitalgainsdistribution) en lugar de tomar las ganancias de la inversión en efectivo. Cuando reinvierte dividendos o distribuciones, aumenta la base imponible de su inversión. Este aumento debe contabilizarse para que pueda declarar ganancias de capital más bajas y, por lo tanto, pagar menos impuestos. Si no declara la base imponible más alta, podría terminar pagando impuestos más altos.

Cuando se le entregan acciones como obsequio, su base de costo es la base de costo del tenedor original del activo que le dio el obsequio. Si las acciones se negocian a un precio más bajo que cuando se regalaron las acciones, la tasa más baja es la base del costo. Si las acciones se le otorgaron como parte de una herencia, la base del costo de las acciones para el heredero es el [precio](/market-price) de mercado de las acciones en la fecha de la muerte del propietario original.

En el proceso de planificación patrimonial, la base de transferencia ayuda a determinar el valor inicial de la herencia de uno, por lo que la base de transferencia ayuda a determinar la [tasa de impuestos](/taxrate) que los herederos de una herencia deben pagar sobre [las ganancias de capital](/capitalgain) cuando venden cualquier activo asociado con esa herencia.

## Base de traspaso e impuestos sobre donaciones

[impuestos sobre donaciones](/gifttax) son un componente crítico para determinar la base de transferencia de un activo. Este es un impuesto federal que se aplica a escenarios en los que la parte receptora no paga al donante el valor total del obsequio (aunque es posible que pague una cantidad menor).

El dador del regalo es el que lleva la peor parte del impuesto sobre donaciones. En general, los obsequios al cónyuge oa una organización política, o los obsequios valorados en menos de la exclusión anual del impuesto sobre donaciones, junto con los gastos médicos y educativos, están excluidos del impuesto sobre donaciones.

Para 2021, el máximo del impuesto sobre donaciones es de $15,000 por persona por año ($16,000 para 2022). Esto significa que una persona puede regalar a otra persona $15,000 o menos por año sin incurrir en un impuesto sobre donaciones.

El impuesto sobre donaciones difiere del [impuesto sobre sucesiones](/estatetax),. que se aplica a la porción heredada de un heredero de una [sucesión](/estate). En 2021, el límite de exclusión para el valor del patrimonio fue la combinación de activos brutos y donaciones sujetas a impuestos anteriores que superen los $11,7 millones ($12,06 millones en 2022). Esto significa efectivamente que un patrimonio de $11.7 millones no estaría obligado a presentar una declaración de [impuestos](/taxreturn) y estaría exento de pagar el impuesto al patrimonio.

Si bien el impuesto al patrimonio se aplica principalmente a los bienes que se dejan a los herederos, no se aplica a la transferencia de bienes a un cónyuge sobreviviente. El derecho de los cónyuges a dejarse cualquier cantidad se conoce como [deducción marital ilimitada](/unlimited-marital-deduction).

## Reflejos

- En general, la base de transferencia es la misma que la base de costo original.

- Una base de transferencia se refiere a la base del costo de un activo recibido de otra persona.

- El hecho de que el bien haya sido transmitido a título de donación o de herencia afectará su base imponible y su base de cálculo.
