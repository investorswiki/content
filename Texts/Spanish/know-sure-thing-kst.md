# Saber algo seguro (KST)
## ¿Qué es saber con seguridad (KST)?

Know Sure Thing (KST) es un [oscilador de momento](/oscillator) desarrollado por Martin Pring para hacer que las lecturas de la tasa de cambio sean más fáciles de interpretar para los comerciantes.

## Cálculo de Know Sure Thing (KST)

El KST se calcula tomando el [promedio móvil simple](/sma) (SMA) de cuatro períodos [de tasa de cambio](/rateofchange) (ROC) diferentes, sumándolos para obtener el KST y creando una línea de señal tomando el SMA de 9 períodos del KST.

Finalmente, la [línea de señal](/signal_line) se calcula tomando la SMA de 9 períodos del valor KST.

## Comprender la Cosa Saber Seguro (KST)

El indicador KST se puede usar de la misma manera que muchos otros osciladores de impulso, como el conocido [índice de fuerza relativa](/rsi) (RSI). Las señales comerciales se generan cuando el KST cruza la línea de señal, pero los operadores también pueden buscar convergencia y divergencia con el precio, condiciones de sobrecompra o sobreventa, o [cruces](/crossover) de la línea central.

Muchos comerciantes combinan el indicador KST con otras formas de análisis técnico para maximizar sus probabilidades de éxito. Por ejemplo, los comerciantes pueden buscar otros indicadores que no sean de impulso, patrones de gráficos o patrones de [velas](/candlestick) japonesas para ayudarlos en la toma de decisiones.

> En un artículo de **Acciones y materias primas** de 1992, Pring se refirió al indicador como "Tasa de cambio sumada (KST)", pero el término KST se quedó con [los analistas técnicos](/technical-analyst).

Esta información también podría usarse para evitar realizar operaciones menos rentables sugeridas por el indicador KST.

## Reflejos

- Know Sure Thing (KST) es un oscilador de impulso destinado a interpretar datos de precios de tasa de cambio.

- Las señales comerciales se generan cuando el KST cruza la línea de señal, pero los operadores también buscan condiciones de sobrecompra o sobreventa.

- Los comerciantes también combinan el KST con otros análisis técnicos para maximizar sus probabilidades de una operación exitosa.
