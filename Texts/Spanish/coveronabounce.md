# Cubrir en un rebote
## ¿Qué es Cover On A Bounce?

Cobertura en un rebote es un término de negociación de acciones que significa cubrir una posición negociando después de que el precio de la acción haya rebotado en un nivel de soporte. La estrategia consiste en esperar a que la acción baje lo suficiente como para alcanzar un [nivel de soporte](/support),. luego rebote brevemente, luego baje ligeramente para corregir el rebote y cierre la posición corta en este punto bajo.

El comerciante o inversor utiliza el rebote como indicador de que el precio bajará ligeramente, para corregir el rebote, pero no significativamente más bajo. El peligro de usar esta estrategia es que es posible que el movimiento hacia arriba, después de alcanzar el nivel de soporte más bajo, sea una reversión total, no simplemente un rebote predecible.

## Cómo funciona Cover On A Bounce

Cobertura en un rebote significa cerrar una [posición corta](/short) comprando una acción después de que el precio cae lo suficiente como para alcanzar un nivel de soporte, luego rebota brevemente y luego corrige. Esta es una estrategia comercial que utiliza el rebote para indicar que la acción ha dejado de caer, lo que permite al inversionista o comerciante cerrar su posición corta al precio más bajo o casi más bajo posible.

Un nivel de soporte es el precio más bajo al que tradicionalmente ha llegado una acción, lo que significa que es poco probable que la acción baje de ese precio. Un comerciante o inversor podría simplemente esperar hasta que el precio caiga a ese nivel de soporte y luego comprar para cerrar la posición corta. Sin embargo, si el nivel de soporte no se mantiene y el precio baja aún más, eso puede causar un impulso y el precio caerá de manera mucho más significativa, y el comerciante o inversor debe esperar para cerrar la posición.

No hay forma de saber si el nivel de soporte se mantendrá hasta que lo haga o no. Esto significa que un rebote por encima del nivel de soporte es la señal de que el nivel de soporte se mantendrá, por lo que el rebote es el mejor indicador para el inversor o comerciante de que este es el precio más bajo que obtendrán. Tan pronto como el precio se corrija desde el rebote, el comerciante o inversor cerrará la posición.

## Ejemplo de portada en un rebote

Una acción puede comenzar en $ 90 por acción y comenzar a caer. Tan pronto como el comerciante o inversor utilice indicadores para concluir que la acción está en una tendencia bajista y seguirá cayendo, abrirá una posición corta vendiendo. En este caso, pueden venderse a $80. En un mundo perfecto, la acción seguirá cayendo, hasta que alcance un nivel de soporte de $48. La acción ha llegado a este precio antes, pero no ha bajado en más de un año. El comerciante o inversionista espera para ver si la acción se romperá y bajará, o si se mantendrá en el nivel de soporte de $48. Si la acción rebota hasta $53, el comerciante o inversor sabe que es probable que se mantenga el nivel de soporte. Esperan a que el precio se corrija por el rebote a $49 y compran para cerrar la posición corta. La ganancia bruta de esta operación es de $31 por acción.

## Reflejos

- El comerciante o inversor utiliza el rebote como indicador de que el precio bajará ligeramente, para corregir el rebote, pero no significativamente más bajo.

- Cobertura en un rebote significa cubrir una posición negociando después de que el precio de las acciones haya rebotado en un nivel de soporte.
