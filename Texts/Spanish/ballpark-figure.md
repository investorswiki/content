# Figura de estadio de béisbol
## ¿Qué es una cifra aproximada?

Una cifra aproximada es una estimación numérica aproximada o una aproximación del valor de algo que de otro modo se desconoce. Las cifras aproximadas son comúnmente utilizadas por [contadores](/accountant),. vendedores y otros profesionales para estimar resultados actuales o futuros. Un corredor de bolsa podría usar una cifra aproximada para estimar cuánto dinero podría tener un cliente en algún momento en el futuro, dada una cierta tasa de crecimiento. Un vendedor podría usar una cifra aproximada para estimar cuánto tiempo podría ser viable un producto que un cliente estaba pensando en comprar.

Una cifra aproximada es esencialmente un marcador de posición establecido con el fin de especular cuál podría ser la cantidad o el total de algo para que las partes involucradas puedan avanzar en cualquier negociación o planificación que esté en marcha. Como concepto, tiene aplicaciones en estimaciones empresariales, así como en la vida cotidiana, según las circunstancias.

> Una cifra aproximada es una estimación numérica amplia de lo que podría llegar a ser algo si se midiera con precisión, si se evaluara con el fin de negociar negocios, hacer tratos o una lluvia de ideas en general.

>

## Comprender las cifras aproximadas

Las cifras aproximadas son estimaciones que se utilizan para hacer avanzar una discusión o un trato cuando aún no se puede determinar la medida exacta del tamaño o la cantidad de algo.

Las cifras aproximadas se pueden usar para fines cotidianos, como estimar la cantidad de alimentos y bebidas que se necesitarán para una parrillada o cuántos meses se necesitarán para pagar una nueva compra.

Las cifras aproximadas también se utilizan en todas partes en el mundo de los negocios, como estimar cuánto podría costar expandirse a un determinado mercado, o cuántos años podría tomar para que una empresa sea rentable o para que las ventas justifiquen una gran compra. También se puede usar para estimar la adopción pública de un concepto, tecnología o producto, como cuántas personas probablemente comprarán un determinado teléfono y cuánto tiempo les llevará actualizar ese teléfono, una vez comprado.

## Consideraciones Especiales

Si bien las cifras aproximadas se usan con frecuencia y pueden ser útiles para establecer una línea de base para la discusión, deben tratarse como nada más que estimaciones; no son números duros. Estas cifras son frecuentemente exageradas por los vendedores y otros profesionales que deben usar la persuasión para generar ingresos o cerrar tratos. Las principales decisiones comerciales y financieras probablemente no deberían tomarse en función de estos números; sin embargo, podrían servir primero como estimaciones para ser refinadas a través de un análisis más completo.

Una teoría popular del término postula que probablemente tenga una historia similar a la del modismo "en el mismo estadio", que significa "aproximadamente la misma cantidad".

## Reflejos

- Las cifras aproximadas se utilizan en la vida diaria y en muchos aspectos de los negocios; sin embargo, es importante recordar que es solo una estimación, no una lectura precisa de algo.

- Una cifra aproximada es una estimación de lo que algo podría representar numéricamente cuando se evalúa un número más preciso, como el costo de un producto.

- Las cifras aproximadas pueden ser útiles para establecer un marcador de posición, con fines de conversación o estimación, cuando no se dispone de un número más preciso.
