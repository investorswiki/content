# Control de inventario de dos contenedores
## ¿Qué es el control de inventario de dos contenedores?

El control de inventario de dos contenedores es un sistema que se utiliza para determinar cuándo se deben reabastecer los artículos o materiales utilizados en la producción. Cuando se han agotado los artículos del primer contenedor, se realiza un pedido para rellenarlos o reemplazarlos. Se supone que el segundo contenedor tiene suficientes artículos para durar hasta que llegue el pedido del primer contenedor. En resumen, el primer contenedor tiene un mínimo de stock de trabajo y el segundo contenedor mantiene el stock de reserva o material restante.

El método de control de inventario de dos contenedores también se denomina a veces [kanban](/kanban),. que está fuertemente asociado con el método [justo a tiempo](/jit) (JIT) de un proceso de fabricación.

## Cómo funciona el control de inventario de dos contenedores

La gestión eficaz de los niveles de stock es uno de los mayores desafíos a los que se enfrentan las empresas. No tener suficiente [inventario](/inventory) puede resultar en la pérdida de oportunidades de ventas y la pérdida frente a la competencia. Tener demasiadas existencias, por otro lado, aumenta la posibilidad de daño, deterioro, robo y ser víctima de cambios en [la](/demand) [demanda](/demand). También significa mayores costos de almacenamiento y demoras en la recuperación del dinero de los bienes comprados para reinvertir en el negocio.

El sistema de control de inventario de dos contenedores es una técnica básica utilizada para garantizar que las empresas reduzcan estos riesgos y siempre tengan, más o menos, el nivel de stock adecuado para satisfacer la demanda sin excederse.

En su forma más simple, el proceso se puede desglosar así:

- El primer contenedor se coloca encima o delante del segundo contenedor.

- Se coloca una tarjeta de pedido en la parte inferior de ambos contenedores.

- El stock se toma de la primera papelera más accesible.

- Cuando el primer contenedor está vacío, se reemplaza por el segundo contenedor.

- La tarjeta de pedido se utiliza para reabastecer el primer contenedor.

- Cuando llega el stock pedido, se coloca en la papelera vacía y el proceso se repite.

Este sistema se emplea ampliamente en diferentes industrias que involucran operaciones de fabricación y también es efectivo para el control de inventario de hospitales.

## Consideraciones Especiales

El control de inventario de dos contenedores casi siempre se usa para artículos pequeños o de bajo valor que se pueden comprar y almacenar fácilmente a granel. Por el contrario, los artículos de mayor valor están sujetos al sistema de [inventario perpetuo](/perpetualinventory).

Además, dependiendo de los patrones históricos de variación en la tasa de agotamiento de las existencias de trabajo (segmento n.° 1), se puede ajustar la cantidad solicitada para las existencias de reserva (sector n.° 2).

Es esencial que el nuevo pedido realizado después de que se vacíe el primer contenedor llegue antes de que se vacíe el segundo contenedor; de lo contrario, el proceso no funcionará según lo previsto. El enfoque de inventario utilizado para ambos contenedores es [primero en entrar, primero en salir](/fifo) (FIFO), dado que el inventario colocado en el primer contenedor es también el inventario que se vende primero.

En general, el siguiente cálculo se usa para determinar cuánto inventario mantener en la ubicación de reserva:

**(Tasa de uso diario * plazo de entrega) + inventario de seguridad**

## Ejemplo de control de inventario de dos contenedores

La empresa A es un pequeño fabricante que utiliza varios tipos de tuercas y tornillos para ensamblar sus productos. Los sujetadores se encuentran entre los muchos artículos que pide a proveedores externos. Utiliza aproximadamente 800 de ellos por semana, o 160 por día, con un tiempo de entrega, el período entre el comienzo y la finalización de un proceso de producción, de tres días.

De acuerdo con el primer cálculo anterior, el contenedor de reserva de la empresa A debe almacenar al menos 480 sujetadores. Sin embargo, la gerencia también es consciente de que los niveles de uso a veces pueden fluctuar hasta en un 15 %, por lo que, como medida de precaución, elige agregar algunos sujetadores más a su contenedor de almacenamiento de reserva. Este stock de seguridad podría ser útil si la demanda aumenta y las tasas de producción aumentan, como ha ocurrido en el pasado.

## Reflejos

- El control de inventario de dos contenedores casi siempre se usa para artículos pequeños o de bajo valor que se pueden comprar y almacenar fácilmente a granel.

- El control de inventario de dos contenedores es un sistema que se utiliza para determinar cuándo se deben reabastecer los artículos o materiales utilizados en la producción.

- Cuando se agotan los artículos del primer contenedor, se realiza un pedido para reemplazarlos. Durante la espera, se utilizan artículos del segundo contenedor.

- El sistema de control de inventario de dos contenedores garantiza que las empresas reduzcan los riesgos de inventario y siempre tengan el nivel adecuado de existencias para satisfacer la demanda.

- Las tarjetas bin y las tarjetas del libro mayor de la tienda se utilizan para registrar el inventario.
