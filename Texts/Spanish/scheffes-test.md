# Prueba de Scheffe
## ¿Qué es una prueba de Scheffé?

Una prueba de Scheffé es una prueba estadística que es una prueba post-hoc utilizada en el análisis estadístico. Lleva el nombre del estadístico estadounidense Henry Scheffé. La prueba de Scheffé se utiliza para realizar comparaciones no planificadas, en lugar de comparaciones planificadas previamente, entre las medias de los grupos en un experimento [de análisis de varianza](/anova) (ANOVA).

Una comparación no planificada es una comparación realizada dentro de un conjunto de datos después de que se haya ejecutado una prueba de ANOVA, por lo que los parámetros de la comparación no se integran en el experimento de ANOVA. La prueba de Scheffé se puede utilizar para determinar si las medias individuales difieren o si el promedio de un grupo de medias difiere del promedio de otro grupo de medias.

## Comprender la prueba de Scheffé

Mientras que la prueba de Scheffé tiene la ventaja de darle al experimentador la flexibilidad de probar cualquier comparación que parezca interesante, el inconveniente de esta flexibilidad es que la prueba tiene un poder estadístico relativamente más bajo que las pruebas que están diseñadas para comparaciones planificadas previamente.

Si bien se pueden realizar comparaciones planificadas previamente mediante pruebas como las pruebas t o las pruebas F, estas pruebas no son adecuadas para comparaciones post hoc o no planificadas. Para tales comparaciones, son apropiadas las pruebas de comparación múltiple, como la prueba de Scheffé, el método de Tukey-Kramer o la [prueba de Bonferroni](/bonferroni-test).
