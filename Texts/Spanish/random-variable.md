# Variable aleatoria
## ¿Qué es una variable aleatoria?

Una variable aleatoria es una variable cuyo valor se desconoce o una función que asigna valores a cada uno de los resultados de un experimento. Las variables aleatorias a menudo se designan con letras y se pueden clasificar como [discretas](/discrete-distribution),. que son variables que tienen valores específicos, o continuas, que son variables que pueden tener cualquier valor dentro de un rango continuo.

Las variables aleatorias se utilizan a menudo en análisis [econométricos](/econometrics) o de [regresión](/regression) para determinar las relaciones estadísticas entre sí.

## Comprender una variable aleatoria

En probabilidad y [estadística](/statistics),. las variables aleatorias se utilizan para cuantificar los resultados de una ocurrencia aleatoria y, por lo tanto, pueden tomar muchos valores. Se requiere que las variables aleatorias sean medibles y, por lo general, son números reales. Por ejemplo, la letra X puede designarse para representar la suma de los números resultantes después de lanzar tres dados. En este caso, X podría ser 3 (1 + 1 + 1), 18 (6 + 6 + 6) o entre 3 y 18, ya que el número más alto de un dado es 6 y el número más bajo es 1.

Una variable aleatoria es diferente de una variable [algebraica .](/algebraic-method) La variable en una ecuación algebraica es un valor desconocido que se puede calcular. La ecuación 10 + x = 13 muestra que podemos calcular el valor específico para x que es 3. Por otro lado, una variable aleatoria tiene un conjunto de valores, y cualquiera de esos valores podría ser el resultado resultante como se ve en el ejemplo. de los dados de arriba.

En el mundo corporativo, se pueden asignar variables aleatorias a propiedades como el precio promedio de un activo durante un período de tiempo determinado, el [rendimiento de la inversión](/returnoninvestment) después de un número específico de años, la tasa de rotación estimada en una empresa dentro de los siguientes seis meses, etc. Los analistas de riesgos asignan variables aleatorias a los modelos de riesgo cuando quieren estimar la probabilidad de que ocurra un evento adverso. Estas variables se presentan utilizando herramientas como tablas de [análisis de sensibilidad y escenarios](/sensitivityanalysis) que los administradores de riesgos utilizan para tomar decisiones relacionadas con la mitigación de riesgos.

## Tipos de variables aleatorias

Una variable aleatoria puede ser discreta o continua. Las variables aleatorias discretas toman un número contable de valores distintos. Considere un experimento en el que se lanza una moneda tres veces. Si X representa el número de veces que la moneda sale cara, entonces X es una variable aleatoria discreta que solo puede tener los valores 0, 1, 2, 3 (desde ninguna cara en tres lanzamientos sucesivos de moneda hasta todas las caras). No es posible ningún otro valor para X.

Las variables aleatorias continuas pueden representar cualquier valor dentro de un rango o intervalo específico y pueden tomar un número infinito de valores posibles. Un ejemplo de una variable aleatoria continua sería un experimento que implique medir la cantidad de lluvia en una ciudad durante un año o la altura promedio de un grupo aleatorio de 25 personas.

Basándose en esto último, si Y representa la variable aleatoria para la altura promedio de un grupo aleatorio de 25 personas, encontrará que el resultado resultante es una figura continua ya que la altura puede ser 5 pies o 5.01 pies o 5.0001 pies. es un número infinito de valores posibles para la altura.

Una variable aleatoria tiene una [distribución de probabilidad](/probabilitydistribution) que representa la posibilidad de que ocurra cualquiera de los valores posibles. Digamos que la variable aleatoria, Z, es el número en la cara superior de un dado cuando se lanza una vez. Los posibles valores de Z serán, por lo tanto, 1, 2, 3, 4, 5 y 6. La probabilidad de cada uno de estos valores es 1/6, ya que todos tienen la misma probabilidad de ser el valor de Z.

Por ejemplo, la probabilidad de obtener un 3, o P (Z=3), cuando se lanza un dado es 1/6, y también lo es la probabilidad de obtener un 4 o un 2 o cualquier otro número en las seis caras de un dado. morir. Tenga en cuenta que la suma de todas las probabilidades es 1.

## Ejemplo de una variable aleatoria

Un ejemplo típico de una variable aleatoria es el resultado de lanzar una moneda. Considere una distribución de probabilidad en la que los resultados de un evento aleatorio no tienen la misma probabilidad de ocurrir. Si la variable aleatoria Y es el número de caras que obtenemos al lanzar dos monedas, entonces Y podría ser 0, 1 o 2. Esto significa que podríamos no tener cara, una cara o ambas caras en un lanzamiento de dos monedas.

Sin embargo, las dos monedas caen de cuatro maneras diferentes: TT, HT, TH y HH. Por lo tanto, P(Y=0) = 1/4 ya que tenemos una posibilidad de no obtener cara (es decir, dos cruces [TT] cuando se lanzan las monedas). De manera similar, la probabilidad de obtener dos caras (HH) también es 1/4. Tenga en cuenta que obtener una cara tiene la probabilidad de ocurrir dos veces: en HT y TH. En este caso, P (Y=1) = 2/4 = 1/2.

## Reflejos

- Una variable aleatoria es una variable cuyo valor se desconoce o una función que asigna valores a cada uno de los resultados de un experimento.

- Una variable aleatoria puede ser discreta (con valores específicos) o continua (cualquier valor en un rango continuo).

- Los analistas de riesgos utilizan variables aleatorias para estimar la probabilidad de que ocurra un evento adverso.

- El uso de variables aleatorias es más común en probabilidad y estadística, donde se utilizan para cuantificar los resultados de sucesos aleatorios.

## PREGUNTAS MÁS FRECUENTES

### ¿Qué es una variable aleatoria continua?

Una variable aleatoria continua representa cualquier cantidad dentro de un rango específico o conjunto de puntos y puede reflejar una cantidad infinita de valores potenciales, como la precipitación promedio en una región.

### ¿Qué es una variable aleatoria discreta?

Una variable aleatoria discreta es un tipo de variable aleatoria que tiene un número contable de valores distintos que se le pueden asignar, como en el lanzamiento de una moneda.

### ¿Qué es una variable aleatoria mixta?

Una variable aleatoria mixta combina elementos de variables aleatorias discretas y continuas.
