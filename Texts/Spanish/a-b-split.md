# División AB
## ¿Qué es una división AB?

Una prueba dividida de AB es una forma de comparar dos versiones de una campaña de [marketing](/marketing),. una aplicación móvil, un sitio web u otros medios medibles para determinar cuál funciona mejor. En una prueba dividida AB, los usuarios se dividen aleatoriamente en un grupo de control y un grupo de variación.

A estos dos conjuntos de usuarios se les muestra esencialmente el mismo medio con la excepción de una sola variable, por ejemplo, el tamaño de un botón de [llamada a la acción "Pedir ahora"](/call-action-cta) en una página de comercio electrónico. Los resultados se miden para determinar el éxito de la variable.

La división AB también se conoce como prueba A/B, pruebas de cubo o prueba de ejecución dividida.

## Comprender una división AB

La división AB se ha utilizado durante décadas en campañas de [correo directo .](/direct-mail) También se ha adaptado con éxito a [los medios interactivos](/interactive-media) para probar las ráfagas de correo electrónico, [los anuncios publicitarios](/banneradvertising),. el sitio web y la funcionalidad de la aplicación móvil (además de otros usos).

Las audiencias se dividen en dos grupos: el grupo de control y el grupo de variación. Por ejemplo, un editor de boletines podría querer probar la efectividad de un llamado a la acción como "Suscríbase dentro de las 48 horas para recibir un 20% de descuento".

El grupo de variación recibe el boletín con la llamada a la acción, mientras que al grupo de control no se le hace ninguna oferta. Esto le permite al editor determinar si la llamada a la acción es efectiva y si la respuesta es suficiente para justificar el 20 % de descuento.

Aunque las pruebas divididas de AB se han utilizado durante mucho tiempo en marketing, Internet permite a los profesionales diseñar e implementar pruebas más rápidamente, lo que acelera significativamente el proceso de iteración del diseño. Las pruebas divididas de AB se pueden realizar de forma continua a un costo relativamente bajo, lo que permite un ajuste fino constante de las campañas de marketing, las actualizaciones del sitio web y el desarrollo de herramientas en línea. En efecto, las pruebas divididas de AB dan como resultado una guía procesable sobre la optimización del sitio web que utiliza datos en la toma de decisiones.

## Consideraciones Especiales

### Pasos de prueba de división AB

Los siguientes son pasos para crear y ejecutar una prueba dividida AB:

1. **Recopilar datos** para identificar oportunidades de mejora.

1. **Identifique objetivos** como mejorar [las tasas de clics](/clickthroughrates) o los registros de correo electrónico.

1. **Formule una hipótesis** para una idea de prueba dividida AB.

1. **Cree una variación de prueba** con una sola variable, como el color de un botón clave.

1. **Ejecute la prueba** para medir las interacciones de los usuarios.

1. **Analizar los datos** para determinar si los resultados son estadísticamente significativos para actuar.
