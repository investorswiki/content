# Sistema de Información Contable (AIS)
## ¿Qué es un Sistema de Información Contable (SIA)?

Un sistema de información contable (AIS) implica la recopilación, el almacenamiento y el procesamiento de datos financieros y contables que utilizan los usuarios internos para informar a los [inversores](/investor),. [acreedores](/creditor) y autoridades fiscales. Por lo general, es un método basado en computadora para rastrear la actividad contable junto con los recursos de tecnología de la información. Un AIS combina [las prácticas contables tradicionales](/accounting-practice),. como el uso de los Principios de Contabilidad [Generalmente](/gaap) [Aceptados (GAAP)](/gaap),. con recursos modernos de tecnología de la información.

## Cómo se utiliza un Sistema de Información Contable (AIS)

Un sistema de información contable contiene varios elementos importantes en el [ciclo](/accounting-cycle) [contable](/accounting-cycle). Aunque la información contenida en un sistema varía entre industrias y tamaños de empresas, un AIS típico incluye datos relacionados con [ingresos](/revenue),. [gastos](/expense),. información de clientes, información de empleados e información fiscal. Los datos específicos incluyen órdenes de venta e informes de análisis, solicitudes de compra, facturas, registros de cheques, inventario, nómina, libro mayor, balance de prueba e información de estados financieros.

Un sistema de información contable debe tener una estructura de base de datos para almacenar información. Esta estructura de base de datos generalmente se programa con un lenguaje de consulta que permite la manipulación de tablas y datos. Un AIS tiene numerosos campos para ingresar datos, así como para editar datos previamente almacenados. Además, los sistemas de información contable suelen ser plataformas altamente seguras con medidas preventivas contra virus, piratas informáticos y otras fuentes externas que intentan recopilar información. [La ciberseguridad](/cybersecurity) es cada vez más importante a medida que más y más empresas almacenan sus datos electrónicamente.

Las diversas salidas de un sistema de información contable ejemplifican la versatilidad de sus capacidades de manipulación de datos. Un AIS produce informes que incluyen informes de antigüedad de [cuentas por cobrar basados en la información del cliente, calendarios de](/accountsreceivable) [depreciación](/depreciation) para activos fijos y balances de prueba para informes financieros. También se pueden reproducir listas de clientes, cálculos de impuestos y niveles de inventario. Sin embargo, la correspondencia, los memorandos o las presentaciones no se incluyen en el AIS porque estos elementos no están directamente relacionados con la contabilidad o los informes financieros de una empresa.

## Beneficios de los Sistemas de Información Contable

### Interfaz interdepartamental

Un sistema de información contable se esfuerza por interactuar entre múltiples departamentos. Dentro del sistema, el departamento de ventas puede cargar el presupuesto de ventas. El equipo de gestión de inventario utiliza esta información para realizar recuentos de inventario y comprar materiales. Tras la compra de inventario, el sistema puede notificar al departamento de [cuentas por pagar](/accountspayable) de la nueva factura. Un AIS también puede compartir información sobre un nuevo pedido para que los departamentos de fabricación, envío y servicio al cliente estén al tanto de la venta.

### Controles internos

Una parte integral de los sistemas de información contable se relaciona con [los controles internos](/internalcontrols). Las políticas y los procedimientos se pueden colocar dentro del sistema para garantizar que la información confidencial de clientes, proveedores y negocios se mantenga dentro de una empresa. Mediante el uso de aprobaciones de acceso físico, requisitos de inicio de sesión, registros de acceso, autorizaciones y segregación de funciones, los usuarios pueden estar limitados a solo la información relevante necesaria para realizar su función comercial.
