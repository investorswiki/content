# Variable manifiesta
## ¿Qué es una variable manifiesta?

Una variable manifiesta es una variable o factor que se puede medir u observar directamente. Es lo opuesto a una variable latente, que es un factor que no se puede observar directamente y que necesita que se le asigne una variable manifiesta como indicador para comprobar si está presente.

Una variable manifiesta también se conoce como variable observable o variable medida.

## Comprender la variable manifiesta

Los estadísticos utilizan varias técnicas de análisis cuando examinan variables manifiestas y variables latentes. Los cuatro modelos más utilizados son [el análisis factorial](/random-factor-analysis),. el análisis de rasgos latentes, el análisis de perfiles latentes y el análisis de clases latentes. El modelo que se use en última instancia depende de si las variables manifiestas son continuas o categóricas, y si las variables latentes también son continuas o categóricas.

Las variables manifiestas se utilizan en modelos estadísticos de variables latentes, que prueban las relaciones entre un conjunto de variables manifiestas y un conjunto de variables latentes. Una variable latente, que no se puede observar directamente, también se conoce como factor o constructo.

La comparación de variables manifiestas y latentes puede ayudar a las empresas a evaluar factores tan aparentemente intangibles como la satisfacción del cliente, la lealtad del producto o la confiabilidad de la empresa. Los estadísticos también usan modelos de variables latentes para determinar si una empresa o acción es una buena inversión.

> Las variables manifiestas, o aquellas que se pueden observar, son útiles como modelos de variables latentes o aquellas que están ocultas y son difíciles de determinar.

>

## Ejemplo de uso de variables de manifiesto

Las variables manifiestas y latentes se pueden usar para medir factores en el negocio que parecen difíciles de evaluar por sí solos, como la satisfacción del cliente. La satisfacción real del cliente es un factor oculto o latente, que solo se puede medir en comparación con una variable manifiesta o un factor observable.

Suponga que el minorista Home Delight quiere tener una idea de si sus clientes están satisfechos con su nueva línea de cojines. Home Delight podría contratar una empresa de investigación estadística o realizar su propia investigación interna para tratar de determinar la satisfacción del cliente. Puede realizar encuestas, mirar comentarios en sitios minoristas como Amazon donde se vende el producto o realizar investigaciones más específicas utilizando variables manifiestas. La empresa podría optar por estudiar variables observables, como números de ventas, el precio por venta, tendencias regionales de compra, el género del cliente, la edad del cliente, el porcentaje de clientes que regresan y qué tan alto clasificó un cliente el producto en varios todos los sitios en la búsqueda del factor latente, es decir, la satisfacción del cliente.

## Reflejos

- Una variable manifiesta es lo contrario de una variable latente, que es un atributo que está oculto y por lo tanto no se puede observar directamente.

- Las variables manifiestas son utilizadas por investigadores que buscan analizar y categorizar diferentes modelos financieros o científicos.

- Una variable manifiesta puede ser estudiada directamente por un investigador o estadístico cuando se grafica en un diagrama.

- Los modelos de variables latentes utilizan variables manifiestas como factor para determinar si existen variables latentes.
