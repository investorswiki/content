# Código estadounidense para el intercambio de información (ASCII)
## ¿Qué es el Código estadounidense para el intercambio de información (ASCII)?

El Código Estadounidense para el Intercambio de Información, comúnmente llamado ASCII, es un lenguaje informático para texto que se convirtió en el estándar para transmitir información entre computadoras en 1963.

El sistema ASCII traduce caracteres y símbolos alfanuméricos en un código legible por computadora. El ASCII original era solo en inglés. Ahora hay dos tipos de códigos ASCII; el código estándar que usa un sistema de codificación de siete bits y un código extendido que usa un sistema de ocho bits.

En las comunicaciones por Internet, ASCII ha sido reemplazado gradualmente por el estándar Unicode, que puede usarse con cualquier idioma y es compatible con ASCII. Sin embargo, ASCII todavía se usa para ingresar texto en computadoras personales y dispositivos electrónicos.

Se pronuncia ASK-y.

## Entendiendo el Código Americano para el Intercambio de Información (ASCII)

La iteración más antigua de ASCII se puede encontrar en un código telegráfico utilizado por los servicios de datos de Bell para enviar mensajes a los teleimpresores.

ASCII fue el primer sistema de codificación de caracteres importante para el procesamiento de datos y se adoptó como estándar para la informática en 1963 y para la transmisión por Internet en 1969. Se utilizó durante muchos años en los sistemas informáticos de entrada de pedidos utilizados por los comerciantes y corredores de Wall Street.

Ha sido reemplazado gradualmente como estándar para Internet por el estándar mundial de caracteres Unicode (Unicode). Sin embargo, dado que los primeros 128 caracteres de Unicode son los mismos que los de ASCII, se puede decir que los dos sistemas coexisten.

> Puede ingresar un símbolo ASCII en un documento utilizando cualquier computadora compatible con Windows manteniendo presionada la tecla ALT mientras escribe el número del código del carácter. Por ejemplo, si mantiene presionada la tecla ALT mientras escribe 156, obtendrá £, el signo de la libra esterlina.

### ¿Qué hay en ASCII?

Antes de que se adoptara ASCII como estándar, cada fabricante de computadoras usaba su propio código y algunas empresas usaban más de uno para diferentes productos. Las computadoras no podían comunicarse entre sí.

Aunque ASCII fue adoptado como estándar por los Estándares Nacionales Estadounidenses en 1963, no se implementó por completo ni siquiera en los EE. Las computadoras podían comunicarse entre sí.

ASCII originalmente contenía 128 caracteres alfabéticos, números y símbolos y estaba basado en el idioma inglés. Se agregaron formularios extendidos durante la década de 1970 para adaptarse a otros idiomas.

Pronto, varias versiones de ASCII estaban en uso. Eventualmente incluyó 255 caracteres. Esta versión se conoce como Extended ASCII.

Hoy en día, los usuarios pueden ingresar símbolos ASCII o Unicode, como un símbolo de derechos de autor, en los documentos presionando la tecla alt e ingresando el código de carácter en el teclado numérico con la tecla de bloqueo NUM activada.

Unicode podría verse como una versión universal de ASCII. Las versiones posteriores incluyeron símbolos utilizados en muchos otros idiomas. Unicode, por otro lado, comienza con los mismos 128 símbolos utilizados en ASCII, pero las versiones actuales contienen 144.697 caracteres.

### ¿Qué es ASCII?

El Código estándar estadounidense para el intercambio de información, o ASCII, es un formato de codificación de caracteres para la transmisión electrónica de texto. Cada carácter está representado por un número único. La primera versión de ASCII contenía solo 128 caracteres, que representaban las letras del alfabeto, en mayúsculas y minúsculas, además de una serie de símbolos de uso común como la coma. Las versiones posteriores ampliaron ASCII a 255 caracteres, incluidos símbolos adicionales como el El símbolo de la libra esterlina (£) y el signo de interrogación al revés que se usa en el texto en español (¿).
