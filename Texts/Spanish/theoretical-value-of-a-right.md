# Valor teórico (de un derecho)
## ¿Qué es el Valor Teórico (De Un Derecho)?

El valor teórico (de un derecho) es el valor de un [derecho de suscripción](/subscription_right). Durante el período de tiempo en que se anuncia una nueva [oferta de derechos](/rightsoffering) hasta tres días antes de que caduquen los derechos de suscripción (conocido como el período de [derechos](/cumrights) acumulados), el valor del derecho es específico y se puede calcular fácilmente. Para calcular el valor de un derecho durante la ventana en la que es efectivo, se debe informar al inversor el precio de suscripción y la cantidad de derechos necesarios para comprar una acción. Con esa información se puede calcular el valor del derecho mediante la siguiente fórmula:

(Precio de la acción - Precio de suscripción de derechos por acción) / (Número de derechos necesarios para comprar una acción + 1)

## Comprender el valor teórico (de un derecho)

El valor teórico de un derecho y el [valor de mercado](/marketvalue) de un derecho son generalmente iguales o muy similares. También se le conoce como el [valor intrínseco](/intrinsicvalue) del derecho. Debido a que el valor de las acciones que tienen derechos adjuntos durante el período de los derechos acumulados puede diferir de las acciones regulares sin tales derechos, los inversionistas desean conocer este valor teórico.

## Ejemplo del mundo real del valor teórico de un derecho

A modo de ejemplo, el precio actual de una acción es de $40, el precio de [ejercicio](/exerciseprice) (o precio de suscripción) es de $35 y se requieren cuatro derechos para comprar una acción. El valor teórico del derecho es:

($40 - $35) / (4 + 1) = $1.

El período de tiempo de aproximadamente tres días antes del vencimiento se denomina período de ejercicio de los derechos. Estos son los últimos días para ejercer los derechos, pero un período demasiado pronto para adquirir nuevas acciones con derechos, ya que la operación se liquidará antes de la fecha de registro, el día en que expiran los derechos. El valor teórico durante el período de ejercicio de los derechos, cuando los derechos se negocian independientemente de las acciones, difiere del valor durante el período de los derechos acumulados.

El cálculo del valor durante el período de ejercicio de los derechos es:

(Precio de la acción - Precio de suscripción del derecho) / Número de derechos necesarios para comprar una acción.

Continuando con el ejemplo anterior, el precio de la acción durante el período sin derechos es de $38, el valor teórico del derecho durante el período de ejercicio de los derechos sería ($38 - $35) / 4 = $0,75.

El valor de un derecho se calcula usando los mismos parámetros que se usan para fijar el precio de las opciones, incluido el precio de suscripción de los derechos, las tasas de interés vigentes, el tiempo hasta el vencimiento y el precio de la acción subyacente, teniendo en cuenta el nivel de su [volatilidad](/volatility). La principal diferencia es que los derechos tienen un [valor de tiempo significativamente menor](/timevalue) que la mayoría de las opciones debido a su vida útil comparativamente corta.

## Precio cero pagado teórico

Si un inversor elige vender su derecho directamente en el mercado, o si decide dejar que el derecho caduque, lo que puede resultar en un cargo administrativo mínimo, recibirá el precio cero teórico pagado del derecho. Este valor se calcula determinando la diferencia entre el precio de suscripción que pagó el inversor y el precio teórico ex-derecho.

Teniendo en cuenta el ejemplo utilizado anteriormente, el cálculo de un precio de pago cero teórico se ve así: $40 - $38 = $2. Así, el monto que el inversionista recibiría por el derecho es el doble del valor del derecho durante el período cum rights e incluso mayor que el valor del derecho durante el período ex-rights.

## Reflejos

- A estos mismos inversores se les dice cuántos derechos son necesarios para comprar una acción.

- A partir de esta información se puede calcular el precio teórico.

- A los inversores con derechos de suscripción se les informa el precio por el que pueden comprar las acciones, generalmente con un descuento sobre el precio de mercado actual.

- El valor teórico de un derecho puede calcularse durante el período cum derechos.
