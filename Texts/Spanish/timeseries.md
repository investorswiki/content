# Series de tiempo
## ¿Qué es una serie temporal?

Una serie de tiempo es una secuencia de puntos de datos que ocurren en orden sucesivo durante un período de tiempo. Esto se puede contrastar con [los datos transversales](/cross_sectional_analysis),. que capturan un punto en el tiempo.

Al invertir, una serie de tiempo rastrea el movimiento de los puntos de datos elegidos, como el precio de un valor, durante un período de tiempo específico con puntos de datos registrados a intervalos regulares. No hay una cantidad mínima o máxima de tiempo que deba incluirse, lo que permite que los datos se recopilen de manera que brinden la información que busca el inversionista o analista que examina la actividad.

## Comprender las series temporales

Se puede tomar una serie de tiempo sobre cualquier variable que cambie con el tiempo. Al invertir, es común usar una serie de tiempo para rastrear el precio de un valor a lo largo del tiempo. Esto se puede rastrear a corto plazo, como el precio de un valor a la hora en el transcurso de un día hábil, o a largo plazo, como el precio de un valor al cierre del último día de cada mes durante el curso de cinco años.

El análisis de series de tiempo puede ser útil para ver cómo cambia un activo, un valor o una variable económica dados con el tiempo. También se puede utilizar para examinar cómo los cambios asociados con el punto de datos elegido se comparan con los cambios en otras variables durante el mismo período de tiempo.

Las series temporales también se utilizan en varios contextos no financieros, como medir el cambio en la población a lo largo del tiempo. La siguiente figura muestra una serie de tiempo de este tipo para el crecimiento de la población de EE. UU. durante el siglo 1900-2000.

## Análisis de series temporales

Suponga que desea analizar una serie temporal de los precios de cierre diarios de las acciones de una determinada acción durante un período de un año. Obtendría una lista de todos los precios de cierre de las acciones de cada día durante el último año y los enumeraría en orden cronológico. Esta sería una serie de tiempo de [precio de cierre diario de un año](/closingprice) para la acción.

Profundizando un poco más, puede analizar datos de series temporales con herramientas de análisis técnico para saber si la serie temporal de las acciones muestra alguna estacionalidad. Esto ayudará a determinar si la acción pasa por picos y valles en momentos regulares cada año. El análisis en esta área requeriría tomar los precios observados y correlacionarlos con una temporada elegida. Esto puede incluir temporadas de calendario tradicionales, como verano e invierno, o temporadas comerciales, como las temporadas de vacaciones.

Como alternativa, puede registrar los cambios en el precio de las acciones en relación con una variable económica, como la [tasa de desempleo](/unemploymentrate). Al correlacionar los puntos de datos con información relacionada con la variable económica seleccionada, puede observar patrones en situaciones que muestran dependencia entre los puntos de datos y la variable elegida.

> Un problema potencial con los datos de series temporales es que, dado que cada variable depende de su estado o valor anterior, puede haber una gran [autocorrelación](/autocorrelation),. lo que puede sesgar los resultados.

## Pronóstico de series de tiempo

La previsión de series temporales utiliza información sobre valores históricos y patrones asociados para predecir la actividad futura. En la mayoría de los casos, esto se relaciona con el análisis de tendencias, el análisis de fluctuaciones cíclicas y cuestiones de estacionalidad. Como con todos los métodos de pronóstico, el éxito no está garantizado.

El [modelo Box-Jenkins](/box-jenkins-model),. por ejemplo, es una técnica diseñada para pronosticar rangos de datos basados en entradas de una serie de tiempo específica. Pronostica datos usando tres principios, [autorregresión](/autoregressive),. diferenciación y [promedios móviles](/movingaverage). Estos tres principios se conocen como p, d y q respectivamente. Cada principio se utiliza en el análisis de Box-Jenkins y juntos se muestran colectivamente como un [promedio móvil integrado autorregresivo](/autoregressive-integrated-moving-average-arima),. o ARIMA (p, d, q). ARIMA se puede utilizar, por ejemplo, para pronosticar los precios de las acciones o el crecimiento de las ganancias.

otro método, conocido como [análisis de rango reescalado](/rescaled-rangeanalysis),. para detectar y evaluar la cantidad de persistencia, aleatoriedad o [reversión a la media](/meanreversion) en los datos de series temporales. El rango reescalado se puede usar para extrapolar un valor futuro o un promedio de los datos para ver si una tendencia es estable o es probable que se revierta.

## Análisis transversal frente a series de tiempo

El análisis transversal es uno de los dos métodos de comparación generales para el análisis de acciones. El análisis transversal analiza los datos recopilados en un solo punto en el tiempo, en lugar de durante un período de tiempo. El análisis comienza con el establecimiento de objetivos de investigación y la definición de las variables que un analista quiere medir. El siguiente paso es identificar la sección transversal, como un grupo de pares o una industria, y establecer el punto específico en el tiempo que se está evaluando. El paso final es realizar un análisis, basado en la sección transversal y las variables, y llegar a una conclusión sobre el desempeño de una empresa u organización. Esencialmente, el análisis transversal le muestra a un inversionista qué compañía es la mejor según las métricas que le interesan.

El análisis de series de tiempo, conocido como [análisis de](/trendanalysis) [tendencias](/trendanalysis) cuando se aplica al comercio técnico, se enfoca en un solo valor a lo largo del tiempo. En este caso, el precio se juzga en el contexto de su desempeño pasado. El análisis de series de tiempo muestra a un inversionista si la compañía está mejor o peor que antes según las medidas que le interesan. A menudo, estos serán clásicos como [ganancias por acción](/eps) (EPS), deuda a capital, [flujo de caja libre](/freecashflow) (FCF), etc. En la práctica, los inversores suelen utilizar una combinación de análisis de series temporales y análisis transversal antes de tomar una decisión. Por ejemplo, observar el EPS a lo largo del tiempo y luego verificar el EPS de referencia de la industria.

## Reflejos

- Aunque los datos transversales se consideran lo opuesto a las series temporales, en la práctica a menudo se usan juntos.

- Los métodos de pronóstico que utilizan series temporales se utilizan tanto en el análisis fundamental como en el técnico.

- El análisis de series temporales puede ser útil para ver cómo cambia un determinado activo, valor o variable económica a lo largo del tiempo.

- Una serie temporal es un conjunto de datos que rastrea una muestra a lo largo del tiempo.

- En particular, una serie de tiempo permite ver qué factores influyen en ciertas variables de un período a otro.

## PREGUNTAS MÁS FRECUENTES

### ¿Cómo se analizan los datos de series temporales?

Las técnicas estadísticas se pueden utilizar para analizar datos de series temporales de dos maneras clave: para generar inferencias sobre cómo una o más variables afectan alguna variable de interés a lo largo del tiempo, o para pronosticar tendencias futuras. A diferencia de los datos transversales, que son esencialmente una porción de una serie de tiempo, la flecha del tiempo le permite al analista hacer afirmaciones causales más plausibles.

### ¿Cómo se utilizan las series temporales en la minería de datos?

[La minería de datos](/datamining) es un proceso que convierte montones de datos sin procesar en información útil. Al utilizar software para buscar patrones en grandes lotes de datos, las empresas pueden aprender más sobre sus clientes para desarrollar estrategias de marketing más efectivas, aumentar las ventas y reducir los costos. Las series de tiempo, como un registro histórico de registros corporativos o estados financieros, son particularmente útiles aquí para identificar tendencias y patrones que pueden pronosticarse en el futuro.

### ¿Cuáles son algunos ejemplos de series temporales?

Una serie de tiempo se puede construir con cualquier dato que se mida a lo largo del tiempo a intervalos espaciados uniformemente. Los precios históricos de las acciones, las ganancias, el PIB u otras secuencias de datos financieros o económicos se pueden analizar como una serie temporal.

### ¿Cuál es la distinción entre datos transversales y de series temporales?

Una sección transversal analiza un solo punto en el tiempo, lo cual es útil para comparar y analizar el efecto de diferentes factores entre sí o para describir una muestra. Las series temporales implican muestreos repetidos de los mismos datos a lo largo del tiempo. En la práctica, se utilizan comúnmente ambas formas de análisis; y cuando están disponibles, se usan juntos.
