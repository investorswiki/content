# Monedero determinista jerárquico (HD)
## ¿Qué es una billetera determinista jerárquica (HD)?

Una billetera determinista jerárquica (HD) es una [billetera digital](/digital-wallet) que se usa comúnmente para almacenar las claves digitales para los titulares de criptomonedas como Bitcoin y Ethereum. Cualquiera que tenga una copia de la clave pública y privada similar a una contraseña puede controlar la [criptomoneda](/cryptocurrency) en la cuenta.

## Comprender las billeteras deterministas jerárquicas (HD)

En el mundo de las criptomonedas, las billeteras contienen llaves en lugar de monedas. Una billetera de criptomonedas tiene dos claves: una clave o dirección pública, que actúa como un número de cuenta, y una [clave privada](/private-key) que el titular usa para transferir fondos a otras cuentas. La clave privada es muy parecida a una contraseña. Alguien que transfiera [Bitcoin](/bitcoin) desde su cuenta, por ejemplo, usaría su clave privada para autorizar la transacción.

Esta combinación de claves públicas y privadas está diseñada para garantizar la seguridad frente a los piratas informáticos, así como el anonimato en las transacciones. Debido a que el par de claves permite la transferencia de la criptomoneda de alguien, es fundamental que la clave privada permanezca segura. Para ello, las claves se generan aleatoriamente.

Se debe hacer una copia de seguridad de cada clave en la billetera para evitar que su propietario y los fondos a los que está vinculada se pierdan irrevocablemente. Pero para garantizar la privacidad, uno de los propósitos que impulsan la creación de Bitcoin y otras criptomonedas, y limitar el uso de claves públicas a una transacción cada una, sería necesario crear un nuevo par de claves para cada transacción. Esto puede volverse difícil de manejar con el tiempo.

Las billeteras deterministas se crearon para ofrecer una solución, una en la que todas las claves se pueden rastrear hasta una semilla aleatoria original, generalmente un conjunto de palabras aleatorias y una [función hash](/hash). Con una billetera determinista, la semilla original es suficiente para recuperar todas las claves públicas y privadas, por lo que solo se requiere una única copia de seguridad en el momento de la creación.

## Monederos deterministas frente a deterministas jerárquicos

Las billeteras HD son el tipo más avanzado de billetera determinista. Contienen claves en una estructura de árbol, en la que las claves principales pueden generar claves secundarias, las cuales pueden generar claves secundarias, y así sucesivamente, infinitamente. El titular de la criptomoneda puede utilizar la estructura de árbol para organizar las transacciones por tipo de transacción o por entidad involucrada, como departamentos o subsidiarias.

Al igual que las billeteras deterministas simples, todas las billeteras HD se crean a partir de una semilla raíz maestra única, generalmente representada por una secuencia de palabras mnemotécnicas, lo que facilita la transcripción y el almacenamiento de los titulares de cuentas. Pero las billeteras HD también ofrecen la opción de crear claves públicas sin tener que acceder a las claves privadas correspondientes. Esto significa que se pueden usar en servidores inseguros o en un modo de solo recepción.

## Reflejos

- Para evitar la piratería, estas claves deben generarse aleatoriamente y respaldarse en la billetera.

- Una billetera determinista jerárquica es una billetera digital comúnmente utilizada para almacenar las claves para los titulares de criptomonedas como Bitcoin y Ethereum.

- Las billeteras HD permiten crear una serie de pares de claves a partir de una semilla aleatoria, lo que brinda comodidad y capacidad de administración, así como seguridad de alto nivel.
