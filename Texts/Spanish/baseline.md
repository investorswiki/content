# Base
## ¿Qué es una línea base?

Una línea de base es un punto fijo de referencia que se utiliza con fines de comparación. En los negocios, el éxito de un proyecto o producto a menudo se mide contra un número de referencia de costos, ventas o cualquier número de otras variables. Un proyecto puede exceder un número de referencia o no cumplirlo.

Por ejemplo, una empresa que quiere medir el éxito de una línea de productos puede usar la cantidad de unidades vendidas durante el primer año como referencia para medir las ventas anuales subsiguientes. La línea de base sirve como punto de partida contra el cual se miden todas las ventas futuras.

## Comprender una línea de base

Una línea de base puede ser cualquier número que sirva como un punto de partida razonable y definido para propósitos de comparación. Puede usarse para evaluar los efectos de un cambio, seguir el progreso de un proyecto de mejora o medir la diferencia entre dos períodos de tiempo.

Por ejemplo, una [empresa pública](/publiccompany) hará un seguimiento del desempeño de cada línea de productos eligiendo un año como referencia y midiendo todos los años subsiguientes contra él.

Por lo general, se utiliza una línea base cuando se prepara un estado financiero o un análisis presupuestario. La declaración o análisis utiliza los ingresos y gastos existentes como referencia para evaluar si un nuevo proyecto se implementa con éxito.

### La línea de base en el análisis de estados financieros

Un análisis de estados financieros que usa una línea de base se llama [análisis](/horizontalanalysis) [horizontal](/horizontalanalysis). Compara la información financiera histórica de una empresa durante una serie de períodos de informes que pueden ser mensuales, trimestrales o anuales.

El primer período en un análisis horizontal se denota como el período de referencia. Todos los períodos subsiguientes se miden luego como un porcentaje de la línea de base. Por lo tanto, un período que tenga los mismos ingresos que la línea de base tendría un 100 % de ingresos.

> En tecnología de la información, existen tres puntos de referencia comúnmente utilizados: costo, alcance y cronograma.

>

Este ejercicio es útil para detectar tendencias, observar áreas de crecimiento o declive y evaluar el desempeño financiero en general. Las proporciones como el margen de beneficio también se comparan horizontalmente con el año de referencia para sacar conclusiones sobre el desempeño continuo de una empresa.

### La línea de base en la elaboración de presupuestos

La presupuestación del proyecto funciona a partir de lo que se conoce como línea base de costos. La línea de base del costo es el presupuesto aprobado para el proyecto, generalmente desglosado con algún detalle por categoría de costo y período de tiempo del costo.

Si una empresa abre un nuevo almacén, por ejemplo, y la base de costos se ha establecido en $100 000 por mes cada mes durante 10 meses, cualquier costo mensual que supere los $100 000 es una [señal de alerta](/redflag) para el analista de presupuesto.

Sin embargo, los costos del proyecto inevitablemente fluctúan con respecto a las cifras de referencia, ya que se realizan gastos desconocidos e inesperados o incluso, en algunos casos, se obtienen ahorros. La línea base de costos se puede actualizar para reflejar los costos reales del proyecto.

### La línea de base en tecnología de la información

En la gestión de la tecnología de la información, se puede establecer una línea de base para los niveles máximos o previstos de rendimiento. Hay tres puntos de referencia de uso común: costo, [alcance](/scope) y cronograma.

Las aplicaciones de software utilizadas por los profesionales de la gestión de proyectos suelen estar diseñadas para mantener y realizar un seguimiento de estas tres medidas críticas de referencia.

## Reflejos

- En el análisis financiero horizontal, las cifras del primer período de información sirven como referencia para la comparación de períodos posteriores.

- En la presupuestación de proyectos, las cifras del presupuesto aprobado son las líneas de base para la comparación de los gastos reales.

- En la gestión de la tecnología de la información, la línea de base es el nivel máximo o previsto de rendimiento.
