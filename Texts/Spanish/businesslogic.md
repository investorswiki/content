# Lógica de negocios
## ¿Qué es la lógica empresarial?

La lógica empresarial son las reglas o [algoritmos personalizados](/algorithm) que manejan el intercambio de información entre una base de datos y la interfaz de usuario. La lógica comercial es esencialmente la parte de un programa de computadora que contiene la información (en forma de reglas comerciales) que define o restringe cómo opera una empresa. Tales reglas comerciales son políticas operativas que generalmente se expresan en binarios verdaderos o falsos. La lógica de negocio se puede ver en los flujos de trabajo que soportan, como en secuencias o pasos que especifican en detalle el flujo adecuado de información o datos, y por lo tanto la toma de decisiones. La lógica empresarial también se conoce como "lógica de dominio".

## Comprender la lógica empresarial

Dicho de otra manera, la lógica comercial son reglas comerciales del mundo real puestas en un código de computadora y mostradas en un programa de computadora a través de una interfaz de usuario. La lógica empresarial es más evidente en su papel en la creación de flujos de trabajo que pasan datos entre usuarios y sistemas de software. La lógica empresarial determina cómo se pueden mostrar, almacenar, crear y modificar los datos. Proporciona un sistema de reglas que guía cómo los objetos comerciales (partes del software que controlan cómo se transportan los datos) funcionan entre sí. La lógica comercial también guía cómo se accede y actualiza los objetos comerciales dentro del software. Existe en un nivel más alto que el tipo de código que se utiliza para mantener la infraestructura informática básica, como la forma en que se muestra una base de datos a un usuario o como infraestructura básica del sistema.

Los algoritmos involucrados en la lógica empresarial realizan un procesamiento de datos entre bastidores que es invisible para el usuario pero que es fundamental para que las cosas funcionen sin problemas en una economía moderna.

## Lógica empresarial frente a reglas empresariales

Las reglas comerciales son inútiles sin la lógica comercial para determinar cómo se calculan, modifican y transmiten los datos a los usuarios y al software. Pero sin reglas comerciales para crear un marco, la lógica comercial no puede existir. La lógica comercial es cualquier parte de una empresa comercial que constituye un sistema de procesos y procedimientos, mientras que cualquier otra cosa es un ejemplo de una regla comercial.

## Ejemplo de lógica empresarial

La lógica empresarial del emisor de una tarjeta de crédito puede especificar que las [transacciones con tarjeta de crédito fuera del estado que superen](/transaction) un cierto límite, digamos $500, se [marquen](/flag) como sospechosas y se contacte al emisor lo antes posible para confirmar la autenticidad de la transacción. La política de marcar una transacción de este tipo es un ejemplo de una regla comercial; el proceso real de marcar la transacción es un ejemplo de lógica comercial. Dado que cada día se realizan millones de transacciones con tarjetas de crédito, la lógica comercial permite verificar y procesar dichas transacciones de manera eficiente y oportuna.

## Reflejos

- **La lógica de negocios existe en un nivel más alto que el tipo de código que se usa para mantener la infraestructura informática básica.**

- **La lógica empresarial se puede ver en los flujos de trabajo que soportan, como en secuencias o pasos que especifican en detalle el flujo adecuado de información o datos**

- **La lógica empresarial hace referencia a la lógica y los algoritmos que sirven como base del código en el software empresarial.**
