# Software comercial
## ¿Qué es el software comercial?

El software comercial facilita el comercio y el análisis de productos financieros, como acciones, opciones, futuros o [divisas](/currency).

Hay una amplia variedad de paquetes de software comercial disponibles en todos los niveles de experiencia comercial y adaptados a diferentes mercados (por ejemplo, acciones frente a divisas).

## Comprender el software comercial

A menudo, [las firmas de corretaje](/brokerage-company) brindan a sus clientes software comercial para realizar transacciones y administrar sus cuentas. El software puede descargarse y ejecutarse desde una computadora de escritorio o un dispositivo móvil, o puede estar basado en la web donde el comerciante accede al software a través de un sitio web en el que inicia sesión.

Los comerciantes también pueden comprar software comercial de terceros que complemente o mejore el software proporcionado por los corredores.

Debido a la caída de los costos de las comisiones a lo largo de los años, más comerciantes e inversores se han movido para realizar al menos algunas de sus propias operaciones y análisis utilizando cuentas de operaciones [autodirigidas .](/self-directed-ira) Esto ha aumentado la demanda de software que proporcione capacidades comerciales, así como recursos de análisis e información dentro del software.

El software comercial puede proporcionar a los usuarios información de precios para activos, tipos de órdenes especiales, [datos fundamentales](/fundamentalanalysis),. gráficos, indicadores de análisis técnico, estadísticas, salas de chat y otras herramientas o funciones patentadas que los corredores y desarrolladores de software utilizan para atraer a los comerciantes a su servicio.

La disponibilidad de interfaces de programación de aplicaciones, o [API](/application-programming-interface),. también ha ayudado a impulsar la industria del software comercial. Las API permiten vincular dos piezas más de software comercial, funcionando como una sola. Esto permite a los usuarios acceder a los beneficios de múltiples piezas de software. Las API no siempre son necesarias, ya que un usuario podría simplemente ejecutar dos o más programas de forma independiente en su computadora, aunque los programas no se comunicarán entre sí.

## Tipos de software comercial

Existen diferentes tipos de software comercial con diferentes características proporcionados tanto por corredores como por desarrolladores externos.

Algunas de las características más comunes incluyen:

- **Realización de operaciones**: la mayoría del software de operaciones tiene la capacidad de realizar operaciones, incluidas órdenes de [mercado](/marketorder),. órdenes limitadas y otros tipos de órdenes avanzadas, así como la capacidad de buscar cotizaciones en tiempo real y ver el libro de órdenes [de nivel 2](/level2). Algunos programas también realizarán un seguimiento de las estadísticas comerciales, como la [tasa](/win-loss-ratio) de ganancia y la ganancia/pérdida promedio en operaciones cerradas.

- **Análisis técnico**: la mayoría del software comercial incluye capacidades de gráficos interactivos, incluidos patrones de gráficos como líneas de [tendencia](/trendline) y formas, así como [indicadores técnicos](/technicalindicator) como [promedios móviles](/movingaverage) u osciladores de impulso.

- **Análisis fundamental**: Algunos software de negociación brindan acceso a información fundamental, incluidos [estados financieros](/financial-statements),. calificaciones de analistas y otras herramientas patentadas diseñadas para que los inversores simplifiquen su [diligencia debida](/duediligence).

- **Negociación programática**: el software comercial avanzado permite a los comerciantes desarrollar sistemas comerciales que se pueden ejecutar automáticamente en lugar de tener que hacer clic manualmente en un botón. Además, estas soluciones de software pueden proporcionar una funcionalidad de [backtesting](/backtesting) diseñada para ayudar a los operadores a ver cómo se habrían desempeñado sus sistemas de negociación automatizados en el pasado.

- **Negociación en papel**: algunos programas de negociación incluyen la capacidad de realizar transacciones sin dinero real sin riesgo, lo que se conoce como [negociación en papel](/papertrade). Los comerciantes pueden probar sus habilidades para ver cómo se desempeñarían antes de comprometer capital real. Esta característica es especialmente común entre los corredores del [mercado de divisas](/forex-market).

## Elección del software comercial

Antes de decidirse por el software comercial, los comerciantes e inversores deben considerar cuidadosamente qué características necesitan. [Los operadores activos](/active-trading) que confían en los sistemas de negociación automatizados pueden elegir un software de negociación completamente diferente al de un inversor que solo busca la capacidad de realizar transacciones.

Las aplicaciones de software pueden tener diferentes [estructuras de](/fee-structure) [tarifas](/fee-structure),. características de rendimiento y otros factores que afectan la rentabilidad.

La mayoría de los corredores y desarrolladores de software permiten que los clientes potenciales prueben su software antes de comprometerse a comprarlo o abrir una cuenta con el corredor. Aproveche esto probando varias piezas de software. Vea qué herramientas y funciones le gustan y utiliza. Luego evalúe los pros y los contras del corredor (si corresponde) y sus comisiones.

Si le gusta un corredor en particular, debido a su estructura de tarifas bajas, por ejemplo, pero no le gusta su software, aún puede encontrar software de terceros que puede utilizar a través de una API o de forma independiente.

Por ejemplo, si no le gustan las capacidades de gráficos de su bróker, puede suscribirse a un servicio/software de gráficos de terceros que le gusten y utilizarlo junto con las capacidades comerciales de su bróker.

## Ejemplos de software comercial de terceros

La mayoría de los corredores tienen su propio software comercial, aunque algunos ofrecen software de terceros. Por ejemplo, en la industria de divisas, muchos corredores tienen su propio software, pero muchos también ofrecen MetaTrader4 y/o MetaTrader5, que es una plataforma de negociación de terceros de uso común.

En el mercado de valores, la mayoría de los corredores ofrecen su propio software. Aquí hay algunos grandes corredores y su software.

- Fidelity proporciona Active Trader Pro.

- Interactive Brokers proporciona TWS y una estructura de tarifas por acción de bajo costo.

- Charles Schwab proporciona Streetsmart Edge y operaciones bursátiles de $4,95.

- TradeStation proporciona TradeStation y es popular entre los comerciantes diarios y los comerciantes activos.

- TD Ameritrade proporciona la plataforma de negociación thinkorswim y operaciones bursátiles a $6,95.

- También hay varias plataformas comerciales y de software de terceros que están ampliamente disponibles.

- La plataforma NinjaTrader proporciona capacidades de gráficos, análisis y negociación y se puede vincular con varios corredores.

- TradingView y [StockCharts proporcionan herramientas](/stockcharts-technical-rank-sctr) [gráficas](/stockcharts-technical-rank-sctr) técnicas y fundamentales. Estas herramientas pueden complementar las capacidades de gráficos proporcionadas por las plataformas comerciales.

## Reflejos

- Las características comunes del software comercial incluyen la colocación de pedidos, el análisis técnico, el análisis fundamental, el comercio automatizado y el comercio en papel.

- Los comerciantes autodirigidos necesitan utilizar y aprender a usar de manera efectiva su software comercial, además de aprender a operar o invertir.

- El software comercial se utiliza para el comercio electrónico y el análisis de valores.
