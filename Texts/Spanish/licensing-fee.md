# Cuota de licencia
## ¿Qué es una tarifa de licencia?

El término "tarifa de licencia" se puede usar en varios contextos, donde se usa más comúnmente para describir una cantidad de dinero pagada a una entidad por un determinado derecho o capacidad. Una tarifa de licencia puede ser una cantidad de dinero pagada por un individuo o empresa a una agencia gubernamental por el privilegio de realizar un determinado servicio o participar en una línea específica de negocios. Las tarifas de licencia se pueden pagar por marcas registradas, derechos de autor o patentes, entre otros.

## Cómo funciona una tarifa de licencia

Muchos tipos de profesiones requieren licencias para participar en la profesión. Estas licencias generalmente se emiten después de aprobar un examen de certificación o de completar las horas de capacitación obligatoria. Una tarifa de licencia también puede describir una suma pagada para usar un elemento con derechos de autor, como una fotografía o un logotipo, que es propiedad de otra persona. Una vez que un profesional tiene su licencia, es un [licenciatario](/licensee).

Las tarifas de licencia, combinadas con las tarifas para obtener la capacitación necesaria para calificar para la licencia, crean [barreras para ingresar](/barrierstoentry) a las ocupaciones con licencia. Los restaurantes deben obtener una licencia de licor y pagar una tarifa de licencia de licor para servir licores. Los peluqueros deben pagar una tarifa de licencia y obtener una licencia de cosmetología para operar su negocio. Estas barreras de entrada se pueden eludir fácilmente si la parte que busca unirse a una profesión se somete al proceso de certificación necesario. Sin embargo, el tiempo y la inversión necesarios para obtener la certificación y la tarifa asociada al hacerlo pueden aumentar los costos de los servicios que ofrecen los proveedores con licencia.

## Cuota de licencia frente a regalías

Las regalías son pagos basados en el uso por usar un activo o propiedad. Por lo general, es un porcentaje de los ingresos brutos o la ganancia neta. Mientras tanto, una tarifa de licencia es dinero pagado por alguien que usa la propiedad de alguien, pero esta tarifa generalmente es una cantidad fija.

Se pueden cobrar regalías por cosas que también tienen licencia, como las patentes. El propietario de una empresa puede ofrecer una parte de sus ingresos al propietario de una obra protegida por derechos de autor oa los franquiciadores. Por otro lado, se puede usar una tarifa de licencia para usar un software o similar, que se puede pagar anualmente.

## Ejemplo de una tarifa de licencia

Un ejemplo de una tarifa de licencia pagada para usar un artículo con derechos de autor sería el propietario de una empresa de camisetas que desea vender camisetas impresas con el logotipo de un equipo de béisbol de las Grandes Ligas. El propietario de la empresa de camisetas tendría que obtener el permiso de Major League Baseball y posiblemente el permiso del propio equipo, además de pagar una tarifa de licencia para obtener la licencia necesaria para usar la imagen.

## Reflejos

- Una tarifa de licencia es un dinero pagado por un derecho o capacidad de usar una propiedad o activo. Las cosas que se pueden licenciar incluyen software, patentes y obras con derechos de autor.

- Las tarifas de licencia son generalmente una cantidad fija, mientras que las regalías son pagos basados en el uso de un activo o propiedad.

- Otros tipos de honorarios incluyen los requeridos para ciertas profesiones, como los cosmetólogos.
