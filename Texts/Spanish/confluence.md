# Confluencia
Originalmente, el término confluencia se usa para describir un punto geográfico donde dos o más ríos se unen para formar un solo cuerpo de agua. Pero siguiendo la misma lógica, también puede usarse en el contexto de las finanzas, para describir la confluencia de múltiples estrategias de inversión o señales comerciales.

Cuando se trata de inversiones a largo plazo, la confluencia se puede lograr cuando un inversor, asesor o administrador de carteras crea una cartera basada en varias estrategias, normalmente invirtiendo en múltiples clases de activos. En la mayoría de los casos, esto conduciría a lo que llamamos [diversificación de cartera](/diversification). Tenga en cuenta que la confluencia se relaciona con el uso combinado de diferentes estrategias, mientras que la diversificación se refiere esencialmente a una cartera que tiene distintos tipos de activos.

En lo que respecta al análisis comercial y técnico (TA), la confluencia podría describirse como el desarrollo de un plan o estrategia comercial que tiene en cuenta diferentes métodos comerciales o indicadores TA. Además, el término también puede usarse para describir el uso combinado de múltiples señales comerciales, como una forma de confirmar la validez de una posible señal de compra o venta.

Por ejemplo, imagine que un comerciante detectó una posible zona de precio de reversión basada en los niveles de [resistencia](/resistance) y [soporte](/support). Pero antes de realizar la operación (es decir, abrir una posición), el comerciante también podría verificar la posición de los promedios móviles para ver si alguno de ellos sugiere la misma zona de reversión. Aparte de eso, también podrían usar el método Ichimoku Cloud para intentar confirmar aún más la validez de su análisis.

Entonces, podemos decir que una tendencia del mercado o un nivel de reversión de precios puede confirmarse mediante el uso de múltiples fuentes de datos y señales comerciales. Esto es lo que llamamos una confluencia de análisis técnico. Como consecuencia, un comerciante tendría más razones para abrir o cerrar una posición o simplemente para esperar un mejor momento para actuar.

## Reflejos

- La confluencia ocurre cuando múltiples ideas o estrategias se usan juntas para formar una sola idea o estrategia coherente.

- En el asesoramiento de inversión, la confluencia se basa en la creación de una estrategia en capas a partir de varios tipos de análisis o teorías que pueden lograr los objetivos de los inversores dentro de su perfil de riesgo.

- En el análisis técnico, la confluencia resulta del empleo de varios indicadores o señales para generar puntos de entrada y salida o detectar reversiones en una tendencia.
