# Estándar de valor
## ¿Qué es el estándar de valor?

El estándar de valor es un valor acordado para una transacción en el [medio de cambio de un país](/mediumofexchange),. como el dólar estadounidense o el peso mexicano. Un estándar de valor permite que todos los comerciantes y entidades económicas establezcan precios uniformes para bienes y servicios. Este estándar es necesario para mantener una economía estable.

## Comprender el estándar de valor

Hasta el siglo XX, el oro era el estándar de valor en muchos países. Estados Unidos abandonó el patrón [oro a](/goldstandard) [nivel](/goldstandard) nacional en 1934 e internacionalmente en 1971. En su lugar, ahora se utiliza un sistema de tipos de [cambio flotantes para la moneda.](/floatingexchangerate)

Por lo general, un estándar de valor se basa en una [mercancía](/commodity) que es ampliamente conocida y utilizada, lo que le permite servir como medida para otras mercancías. Por ejemplo, metales como el oro, la plata, el cobre y el bronce se han utilizado a lo largo de la historia como formas de [moneda](/currency) y patrones de valor. Dar un valor fijo a una cantidad específica de oro, y luego enmarcar otras mercancías como múltiplos o fracciones de ese valor, permite otorgar valor a otros artículos dispares dentro de la misma economía.

## Cómo se aplica un estándar de valor

Mediante el uso de dichos estándares, el valor de otros bienes y servicios puede determinarse de manera relativamente uniforme, independientemente de las diferencias entre esos bienes y servicios. El valor de un automóvil de lujo, por ejemplo, puede establecerse tan fácilmente como el valor de un par de zapatillas para correr. La escala de valor de estos artículos es drásticamente diferente, al igual que su función y uso. El establecimiento de un estándar de valor para la moneda, en particular, permite un fácil intercambio entre individuos, comerciantes y clientes, y empresas.

Si una economía carece de tal estándar de valor, puede ser común ver un sistema de trueque empleado para gobernar el comercio y el comercio. Esto puede significar que el valor asignado de los bienes o servicios podría ser muy subjetivo y variable. Por ejemplo, sin un estándar de valor, un agricultor que produce verduras puede tener que hacer trueques directamente para adquirir los bienes que necesita, como madera o fertilizantes. El valor de las hortalizas ofrecidas en el comercio requeriría algún tipo de acuerdo entre las partes, ya que no se dispone de un estándar de valor para establecer parámetros para el intercambio.

Incluso con un estándar de valor, el valor asignado de una mercancía aún puede fluctuar. Sin embargo, la existencia de la norma mantiene un grado de cohesión y consistencia en todo el sistema económico, salvo una influencia altamente disruptiva en el mercado.

## Reflejos

- Se necesita un estándar de valor para que el valor de los bienes y servicios pueda determinarse de manera consistente.

- Un estándar de valor es un valor acordado para una transacción en un medio de intercambio, como el dólar estadounidense o el oro.

- Sin un estándar de valor, pueden surgir otras formas de intercambiar bienes, como un sistema de trueque.
