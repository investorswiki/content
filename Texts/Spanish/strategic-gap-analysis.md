# Análisis de brechas estratégicas
## ¿Qué es el análisis de brechas estratégicas?

El análisis de brechas estratégicas es una técnica de gestión comercial que requiere una evaluación de la diferencia entre el mejor resultado posible de un esfuerzo comercial y el resultado real. Incluye recomendaciones sobre los pasos que se pueden tomar para cerrar la brecha.

[análisis de brechas](/gap-analysis) estratégicas tiene como objetivo determinar qué pasos específicos puede tomar una empresa para lograr un objetivo en particular. Una gama de factores que incluyen el marco de tiempo, el desempeño de la gestión y las restricciones presupuestarias se analizan críticamente para identificar las deficiencias.

El análisis debe ser seguido por un plan de implementación.

## Comprender el análisis de brechas estratégicas

Un análisis de brechas estratégicas es un método que se utiliza para ayudar a una empresa o cualquier otra organización a determinar si está obteniendo el mejor rendimiento de sus recursos. Identifica la brecha entre el statu quo y el mejor resultado posible. Realizar un análisis de brechas estratégicas puede señalar áreas potenciales de mejora e identificar los recursos que se requieren para que una organización logre sus objetivos estratégicos.

El análisis de brechas estratégicas surge de una variedad de evaluaciones de desempeño, más notablemente de la evaluación comparativa. Cuando se conoce el nivel de rendimiento de una industria o un proyecto, ese punto de referencia se puede utilizar para medir si el rendimiento de una empresa es aceptable o si necesita mejorar. Tal comparación informa un análisis de brechas estratégicas.

A partir de ese momento, la organización puede determinar qué combinación de recursos, como dinero, tiempo y personal, se necesita para obtener un mejor resultado.

> Muchas empresas no planifican estratégicamente; pueden alcanzar sus objetivos comerciales básicos, pero no logran desarrollar todo su potencial.

>

Muchas empresas no planifican estratégicamente. Tienen los recursos y las competencias para lograr sus objetivos comerciales básicos, pero no logran desarrollar todo su potencial. Un análisis de brechas estratégicas podría ayudar a una empresa de este tipo a cerrar la brecha entre sus niveles de rendimiento actuales y potenciales.

## Ejemplo de análisis de brechas estratégicas

Un pequeño restaurante [familiar](/momandpop) en un pueblo costero tiene una clientela leal de lugareños, pero sus propietarios anhelan servir también a la multitud de vacaciones de verano. Un análisis de brechas estratégicas identifica los cambios necesarios para que el restaurante alcance sus objetivos.

Estos cambios pueden incluir mudarse a una calle más concurrida, permanecer abierto más tarde para atraer a los vacacionistas y actualizar el menú. Los dueños de restaurantes no tienen que tomar ninguna de estas recomendaciones. Pero podría hacerlo si quiere alcanzar ese nivel más alto de éxito empresarial.

## Reflejos

- El análisis identifica los pasos que se deben tomar para cerrar esa brecha.

- El análisis de brechas estratégicas mide la diferencia entre un resultado ideal y el resultado real.

- Para una empresa u otra organización, el análisis puede conducir a un plan de acción para un mayor éxito.
