# ActiveX
## ¿Qué es ActiveX?

ActiveX es un marco de software de Microsoft (MSFT) que permite que las aplicaciones compartan funcionalidades y datos entre sí a través de navegadores web, independientemente del lenguaje de programación en el que estén escritos. Los complementos de ActiveX permitieron que los primeros navegadores web incrustaran archivos multimedia o entregaran actualizaciones de software a los usuarios.

Microsoft introdujo ActiveX en 1996. Muchos controles ActiveX se ejecutan solo en Windows y con productos de Microsoft como Internet Explorer, Word y Excel. JavaScript, otro compilador de idiomas cruzados y otras plataformas similares se usan más ampliamente que ActiveX.

## Comprender ActiveX

Los controles ActiveX son software precodificado similar a los complementos del navegador web. Por ejemplo, una página web que muestra un archivo Flash puede requerir que un usuario descargue un control Flash ActiveX para que el archivo se pueda reproducir directamente en el navegador sin abrir una nueva aplicación. Los controles ActiveX amplían la funcionalidad de un navegador, lo que permite que el navegador realice tareas que de otro modo no podría ejecutar de forma innata. Es particularmente útil para reproducir videos y otro contenido multimedia, omitiendo el paso de abrir un reproductor multimedia por separado.

ActiveX todavía se utiliza en Internet Explorer 11, pero no es compatible con el último navegador de Microsoft, Edge . Navegadores como Mozilla Firefox, Google Chrome, Safari de Apple y Opera utilizan otros tipos de complementos de navegador, como JavaScript o lenguajes multiplataforma similares.

ActiveX sigue siendo útil para los usuarios de Microsoft y se incluye con Windows 10. Esto se debe a que ActiveX aún permite que el software independiente reciba actualizaciones, interactúe entre programas y mejore la funcionalidad. Por ejemplo, ActiveX permite a los usuarios crear documentos más interactivos en Word o crear formularios para completar en Excel.

## ActiveX y Seguridad Informática

Microsoft advierte que los controles ActiveX a veces pueden funcionar mal o dar a los usuarios contenido que no quieren. Los controles ActiveX también se pueden usar para instalar [spyware](/adware),. virus y malware, o dañar los datos de su computadora. Por este motivo, es importante que solo instale controles ActiveX de fuentes en las que confíe .

En parte debido al uso malicioso generalizado de los controles ActiveX, muchos navegadores deshabilitan los controles ActiveX de forma predeterminada o no los admiten en absoluto. Por ejemplo, los usuarios de Google Chrome deben habilitar ActiveX en la configuración de seguridad de Chrome o descargar una extensión de Chrome .

Incluso Microsoft parece estar alejándose de ActiveX. Edge, su nuevo navegador que reemplaza a Internet Explorer en los sistemas operativos Windows, no es compatible con ActiveX .

## Reflejos

- Microsoft advierte que los controles ActiveX pueden usarse para espiar su computadora, dañar datos o instalar software malicioso .

- ActiveX es una plataforma para desarrollar pequeños programas o complementos que brindan mayor funcionalidad a los navegadores web y productos de Microsoft.

- Muchos navegadores no son compatibles con ActiveX de forma predeterminada. Edge, el nuevo navegador de Microsoft, no ofrece soporte para controles ActiveX .
