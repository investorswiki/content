# Orden de caja superior
## ¿Qué es un pedido Box-Top?

Una orden box-top es una orden de compra o venta realizada al mejor [precio del mercado](/market-price). Es una forma anticuada de referirse a una [orden de mercado](/marketorder).

Una orden de mercado se encuentra entre las transacciones más comunes y directas que se observan en los mercados financieros. Está destinado a ejecutarse lo más rápido posible al precio de venta actual y, a menudo, es el tipo de orden predeterminado en la mayoría de las plataformas de corretaje.

## Comprender los pedidos de Box-Top

Las órdenes box-top ejecutan órdenes de mercado al mejor precio actual. Si la orden no se puede ejecutar por completo, se coloca una orden limitada para las acciones restantes al precio al que se ejecutó la parte ejecutada. Una [orden limitada](/limitorder) es una orden colocada para ejecutar una transacción de compra o venta en un número determinado de acciones y a un precio límite especificado o mejor.

Por ejemplo, si un comerciante ingresó una orden box-top para comprar 1,000 acciones al precio de mercado actual de $50, y solo la mitad de las acciones se negocian a ese precio, entonces se coloca una orden limitada de compra para las otras 500 acciones. Si en algún momento durante la vigencia de la orden el precio vuelve a $50, la orden limitada se activa y las acciones restantes se negociarán a $50.

## Órdenes Limitadas y Órdenes Stop

Las órdenes limitadas permiten a un inversionista limitar el tiempo que una orden puede estar pendiente antes de ser cancelada. Las órdenes limitadas permiten que las órdenes box-top se ejecuten de manera efectiva. La ejecución de una orden de límite no siempre está garantizada, pero ayuda a garantizar que el inversor no pierda la oportunidad de comprar o vender a un precio objetivo. Estas órdenes establecen un máximo o un mínimo en el que un comerciante está dispuesto a comprar o vender una acción en particular. Un inversor puede establecer condiciones específicas para las órdenes de límite, como exigir que todas las acciones deseadas se compren o vendan al mismo tiempo si se va a ejecutar la operación, lo que se denomina orden [de todo o nada](/aon).

Por otro lado, una [orden stop-loss](/stop-lossorder) asegura que no se realice una transacción a un precio peor que el objetivo indicado. Se puede utilizar para vender un instrumento existente o para realizar una nueva transacción. Con una orden de detención, una operación solo se ejecuta cuando el valor alcanza un precio particular conocido como precio de detención. Para los inversores que no pueden controlar sus acciones durante un cierto período de tiempo, las órdenes de detención son particularmente ventajosas. Los corredores a veces incluso establecen órdenes de detención sin cargo.

Si bien las órdenes stop pueden no incluir un cargo de corretaje, las órdenes limitadas pueden tener una comisión más alta. La ventaja de una orden de límite es que garantiza que se realizará una operación a un precio determinado; sin embargo, es posible que la orden no se ejecute si no se alcanza el precio límite.

## Reflejos

- Una orden de mercado es una instrucción para comprar o vender un valor inmediatamente al precio actual.

- Una orden box top es una forma poco común de referirse a una orden de mercado.

- Una orden de límite es, en cambio, una instrucción para comprar o vender solo a un precio especificado por el inversor.
