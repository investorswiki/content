# Brace Gatarek Musiela (BGM) Modelo
## ¿Qué es el modelo Brace Gatarek Musiela (BGM)?

El modelo Brace Gatarek Musiela (BGM) es un modelo financiero no lineal que utiliza la tasa de oferta interbancaria de Londres (LIBOR) para cotizar derivados de tasas de interés. El modelo Brace Gatarek Musiela (BGM) valora los valores mediante el examen de las tasas cotizadas en el mercado. Se utiliza con más frecuencia cuando se fijan precios de [swaptions](/swaption) y caplets (una opción call sobre LIBOR) en el mercado LIBOR.

> El modelo Brace Gatarek Musiela también se conoce como modelo de mercado LIBOR.

>

## Comprender el modelo Brace Gatarek Musiela (BGM)

A diferencia del [modelo Hull-White](/hullwhite-model),. que utiliza la tasa corta instantánea, o el modelo [Heath-Jarrow-Morton](/hjmmodel) (HJM), que utiliza la tasa forward instantánea, el modelo Brace Gatarek Musiela Model (BGM) solo utiliza tasas que son observables: forward Tasas LIBOR. El modelo BGM también es consistente con el modelo de Black, que es una variante del modelo derivado de Black-Scholes ampliamente utilizado.

> Intercontinental Exchange, la autoridad responsable del LIBOR, dejará de publicar el LIBOR en USD a una semana y a dos meses después del 31 de diciembre de 2021. Todos los demás LIBOR dejarán de publicarse después del 30 de junio de 2023 .

>

## Usos del modelo BGM

El modelo BGM puede determinar un precio para una inversión si el pago se puede desglosar en tasas a plazo (rendimientos), ya que las tasas a plazo se aplican a un marco de tiempo específico y se correlacionan con otras tasas a plazo. Los inversores pueden ejecutar simulaciones utilizando las diversas volatilidades y correlaciones, y luego determinar el valor razonable mediante el descuento de cupones.

La tasa de oferta interbancaria de Londres es el promedio de las tasas de interés estimadas por cada uno de los principales bancos de Londres que se cobraría si tomara prestado de otros bancos. Por lo general, se abrevia como Libor o LIBOR.
