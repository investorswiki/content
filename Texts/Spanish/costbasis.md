# Base de costo
## ¿Qué es la base de costos?

La base de costo es el valor original de un activo a efectos fiscales, generalmente el precio de compra, ajustado por división de [acciones](/stocksplit),. [dividendos](/dividend) y [devolución de](/returnofcapital) distribuciones de capital. Este valor se utiliza para determinar la ganancia de capital, que es igual a la diferencia entre la base de costo del activo y el [valor de mercado actual](/cmv). El término también se puede utilizar para describir la diferencia entre el precio de [contado](/cashprice) y el precio de [futuros](/futures) de un producto determinado.

## Comprender la base de costos

En el nivel más básico, la base del costo de una inversión es el monto total invertido originalmente, más cualquier [comisión](/commission) o tarifa involucrada en la compra. Esto puede describirse en términos del monto en dólares de la inversión o el precio efectivo por acción pagado por la inversión.

Usar la base de costo correcta, también conocida como base impositiva, es importante especialmente si reinvirtió dividendos y [distribuciones de ganancias de capital](/capitalgainsdistribution) en lugar de tomar las ganancias en efectivo. La reinversión de las distribuciones aumenta la base imponible de su inversión, que debe contabilizar para reportar una [ganancia de capital más baja](/capitalgain) y, por lo tanto, pagar menos impuestos. Si no utiliza la base imponible más alta, podría terminar pagando impuestos dos veces sobre las distribuciones reinvertidas. Determinar la base de costo correcta también es el primer paso al calcular las ganancias y pérdidas después de que se vende una acción.

> La reinversión de dividendos aumenta la base de costos de una acción porque los dividendos se usan para comprar más acciones.

>

Los inversionistas suelen utilizar el [método de base de costo promedio](/averagecostbasismethod) para la declaración de impuestos de fondos mutuos. Se informa un método de base de costo con la firma de corretaje donde se mantienen sus activos. Muchas firmas de corretaje utilizan por defecto el método de base de costo promedio. Los inversores también pueden elegir entre otros métodos, incluidos: primero en entrar, primero en salir ( [FIFO](/fifo) ), último en entrar, primero en salir ( [LIFO](/lifo) ), alto costo, bajo costo y más. Una vez que se determina un método de base de costos para un fondo mutuo específico, debe permanecer vigente. Las casas de bolsa proporcionarán a los inversionistas la documentación fiscal anual adecuada sobre las ventas de fondos mutuos en función de sus elecciones de método de base de costos. El concepto de base de costos es básicamente sencillo, pero puede complicarse de muchas maneras. La base de costos de seguimiento es necesaria para fines fiscales, pero también es necesaria para ayudar a rastrear y determinar el éxito de la inversión. La clave es mantener buenos registros y simplificar la estrategia de inversión cuando sea posible.

## Ejemplo de base de costos

Por ejemplo, si se compraron 100 acciones de una acción por $1,000 el año pasado, con el primer año de dividendos por un monto de $100 y los dividendos del segundo año por un monto de $200, todo lo cual se reinvirtió, la ley fiscal aplicable considera que estas ganancias reinvertidas son ingresos. Para fines de cálculo de impuestos, la base de costo ajustado cuando se vendan las acciones se registrará en $1,300 en lugar del [precio de compra original](/purchaseprice) de $1,000. Por lo tanto, si el precio de venta es de $1500, la [ganancia imponible](/taxablegain) sería solo de $200 ($1500 - $1300) en lugar de $500 ($1500 - $1000). Si la base del costo se registra incorrectamente como $ 1,000, esto resulta en una obligación tributaria más alta de lo que normalmente se adeudaría.

## Comparaciones de la base de costos

La comparación de la base de costos puede ser una consideración importante. Suponga que un inversionista realizó las siguientes compras consecutivas de fondos en una cuenta imponible: 1500 acciones a $20, 1000 acciones a $10 y 1250 acciones a $8. La base del costo promedio del inversionista se calcula dividiendo $50,000/3,750 acciones. El costo promedio es de $13.33.

Suponga que el inversionista luego vende 1,000 acciones del fondo a $19. El inversionista tendría una ganancia de capital de $5,670 usando el método de base de costo promedio.

- Ganancia/pérdida usando la base del costo promedio: ($19 - $13.33) x 1,000 acciones = $5,670

Los resultados pueden variar significativamente según el costo.

- Primero en entrar, primero en salir: ($19 - $20) x 1,000 acciones = - $1,000

- Último en entrar, primero en salir: ($19 - $8) x 1,000 = $11,000

- Alto costo: ($19 - $20) x 1,000 acciones = - $1,000

- Bajo costo: ($19 - $8) x 1,000 = $11,000

En este caso, el inversionista estaría mejor si hubiera seleccionado el método FIFO o el método de alto costo para determinar la base del costo antes de vender las acciones. Estos métodos no generarían impuestos sobre una pérdida de $1,000. Con el método de base de costo promedio, el inversionista debe pagar un impuesto a las ganancias de capital sobre la ganancia de $5,670.

## Cómo afectan las divisiones de acciones a la base de costos

Si la empresa divide sus acciones, esto afectará su base de costo por acción, pero no el valor real de la inversión original o la inversión actual. Continuando con el ejemplo anterior, suponga que la empresa emite una [división de acciones de 2:1 en la](/stocksplit) que una acción antigua le otorga dos acciones nuevas. Puede calcular su base de costo por acción de dos maneras:

- Tome el monto de la inversión original ($10 000) y divídalo por la nueva cantidad de acciones que posee (2000 acciones) para llegar a la nueva base de costo por acción ($10 000/2000 = $5,00).

- Tome su costo base anterior por acción ($10) y divídalo por el factor de división de 2:1 ($10.00/2 = $5.00).

## Base de costo de acciones regaladas o heredadas

En el caso de que las acciones le fueran entregadas como regalo, su base de costo es la base de costo del titular original que le dio el regalo. Si las acciones se negocian a un precio más bajo que cuando se regalaron las acciones, la tasa más baja es la base del costo. Si las acciones se le otorgaron como herencia, la base del costo de las acciones para usted como heredero es el [precio de mercado actual](/market-price) de las acciones en la fecha de la muerte del propietario original.

Hay muchos factores que afectarán su base de costos y eventualmente sus impuestos cuando decida vender. Si su base de costo real no está clara, consulte a un [asesor financiero](/financial-advisor),. contador o abogado fiscal.

## Base de costos y contratos de futuros

En lo que respecta a los futuros, la base del costo es la diferencia entre el precio al contado local de un producto básico y su precio de futuros asociado. Por ejemplo, si un contrato de futuros de maíz en particular se cotiza a $ 3.50, mientras que el precio de mercado actual de la [materia prima](/commodity) es de $ 3.10, hay una base de costo de 40 centavos. Si fuera cierto lo contrario, con el contrato de futuros negociando a $3.10 y el precio al contado siendo $3.50, la base de costo sería 40 centavos negativos, ya que la base de costo puede ser positiva o negativa dependiendo de los precios involucrados.

El precio al contado local representa el precio prevaleciente del activo subyacente, mientras que el precio que figura en un contrato de futuros se refiere a una tasa que se daría en un momento específico en el futuro. Los precios de los futuros varían de un contrato a otro según el mes en que vencen.

Al igual que con otros mecanismos de inversión, el precio al contado fluctúa según las condiciones actuales del mercado local. A medida que se acerca la fecha de entrega, el precio de los futuros y el precio al contado se acercan más.

## Reflejos

- La base de costo es el precio original al que se adquirió un activo, para efectos fiscales.

- Las ganancias de capital se calculan calculando la diferencia entre el precio de venta y el costo.

- Existen varios métodos de contabilidad para ajustar la base de costos para que sea más favorable, pero tenga cuidado de seguir las pautas del IRS.
