# Protocolo de transferencia de archivos (FTP)
## ¿Qué es el Protocolo de transferencia de archivos (FTP)?

El término protocolo de transferencia de archivos (FTP) se refiere a un proceso que implica la transferencia de archivos entre dispositivos a través de una red. El proceso funciona cuando una de las partes permite que otra envíe o reciba archivos a través de Internet. Originalmente utilizado como una forma para que los usuarios se comuniquen e intercambien información entre dos dispositivos físicos, ahora se usa comúnmente para almacenar archivos en la [nube](/cloud-computing),. que generalmente es una ubicación segura que se mantiene de forma remota.

FTP puede ser utilizado por una [empresa](/business) o individuo para transferir archivos de un sistema informático a otro o por sitios web para cargar o descargar archivos de sus servidores.

## Cómo funciona el Protocolo de transferencia de archivos (FTP)

El protocolo de transferencia de archivos permite que las personas y las empresas compartan archivos electrónicos con otros sin tener que estar en el mismo espacio. Esto se puede hacer usando un cliente FTP o a través de la nube. Independientemente de la opción, ambas partes requieren una conexión a Internet que funcione.

La mayoría de los navegadores [web](/web-20) vienen con clientes FTP que permiten a los usuarios transferir archivos desde su computadora a un servidor y viceversa. Es posible que algunos usuarios deseen utilizar un cliente FTP de terceros porque muchos de ellos ofrecen funciones adicionales. Ejemplos de clientes FTP que se pueden descargar gratis incluyen FileZilla Client, FTP Voyager, WinSCP, CoffeeCup Free FTP y Core FTP.

Mucha gente ha usado FTP antes sin siquiera darse cuenta. Si alguna vez ha descargado un archivo de una página web, ha utilizado FTP. El primer paso es iniciar sesión, lo que puede ocurrir automáticamente o ingresando manualmente un nombre de usuario y una contraseña. FTP también requerirá que acceda a un servidor FTP a través de un número de puerto específico. Una vez que acceda al servidor FTP a través de su cliente FTP, ahora puede transferir archivos. No todos los servidores FTP públicos requieren que inicie sesión porque algunos servidores le permiten acceder a ellos de forma anónima.

Como se señaló anteriormente, FTP se desarrolló originalmente como una forma de enviar y recibir archivos entre dos computadoras físicas. Pero con los cambios en la tecnología, los usuarios pueden ejecutar transferencias de archivos a través de la nube. El uso de la nube permite que las transferencias se realicen de manera conveniente y segura (lo que podría proteger a individuos y empresas de [violaciones de datos](/data-breach) ) y con un costo relativamente bajo.

> El término cliente FTP se refiere al software que le permite transferir archivos a otra parte.

>

## Consideraciones Especiales

El protocolo de transferencia de archivos es uno de los muchos protocolos diferentes que dictan cómo se comportan las computadoras y los sistemas informáticos en Internet. Otros protocolos de este tipo incluyen:

- **Protocolo de transferencia de hipertexto (HTTP):** Diseñado para transmitir datos a través de la web

- **Protocolo de acceso a mensajes de Internet (IMAP):** Brinda acceso a tableros de anuncios o mensajes de correo electrónico desde un servicio compartido

- **Protocolo de hora de red (NTP):** Sincroniza la hora del reloj en las computadoras a través de una red

FTP permite que las computadoras en Internet transfieran archivos de un lado a otro. Como tal, es una herramienta esencial para quienes construyen y mantienen sitios web en la actualidad.

### Qué buscar en un cliente FTP

Los clientes FTP individuales brindan diferentes funciones que permiten a los usuarios modificar la forma en que cargan y descargan archivos. Por ejemplo, si usa FileZilla, el programa le permite establecer límites de [ancho de banda](/bandwidth) para los archivos. Esto le permite controlar las velocidades de carga y descarga, lo que puede ser útil si administra varias transferencias de archivos a la vez.

Otras características que puede querer buscar en un cliente FTP incluyen autenticación de clave pública, la capacidad de establecer niveles de compresión de archivos o herramientas que le permitan buscar un servidor usando máscaras de archivos.

## Ejemplo de Protocolo de transferencia de archivos (FTP)

El software FTP es relativamente sencillo de configurar. FileZilla es un cliente FTP descargable y gratuito. Escriba la dirección del servidor al que desea acceder, el puerto y la contraseña para acceder al servidor.

Una vez que se haya otorgado el acceso, los archivos del usuario en su sistema local, así como el servidor al que se accedió, serán visibles. El usuario puede descargar archivos del servidor al sistema local o cargar archivos del sistema local al servidor. También pueden realizar cambios en los archivos del servidor, siempre que tengan la autorización adecuada para hacerlo.

## Reflejos

- FTP es una herramienta esencial para aquellos que construyen y mantienen sitios web.

- El protocolo de transferencia de archivos es una forma de descargar, cargar y transferir archivos de una ubicación a otra en Internet y entre sistemas informáticos.

- FTP permite la transferencia de archivos entre computadoras o a través de la nube.

- Los usuarios requieren una conexión a Internet para ejecutar transferencias FTP.

- Muchos clientes FTP se pueden descargar gratis, aunque la mayoría de los sitios web ya tienen el FTP integrado.
