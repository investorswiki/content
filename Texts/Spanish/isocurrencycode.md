# Código de moneda ISO
## ¿Qué es un código de moneda ISO?

Los códigos de moneda ISO son los códigos alfabéticos de tres letras que representan las diversas [monedas](/currency) utilizadas en todo el mundo. Cuando se combinan en pares, forman los símbolos y tipos de cambio [cruzados](/crossrate) utilizados en el comercio de divisas.

Cada uno de los códigos alfabéticos de tres letras específicos de cada país también tiene un código numérico de tres dígitos correspondiente. Estos códigos están identificados por la [Organización Internacional de Normalización](/international-organization-for-standardization-iso) (ISO), una organización no gubernamental que proporciona estándares para la fabricación, el comercio, la tecnología y la comunicación. Para las monedas, el documento rector se denomina ISO 4217:2015.

## Comprender los códigos de moneda ISO

Los códigos de divisas ISO son fundamentales para los [pares de divisas](/currencypair),. que son las estructuras de cotización y fijación de precios de las divisas negociadas en el [mercado de divisas](/forex-market). El valor de una moneda es una tasa y se determina por su comparación con otra moneda.

El primer código de tres dígitos que se utiliza para designar una divisa en la cotización de un par de divisas se denomina [divisa base](/basecurrency),. y la segunda divisa se denomina [divisa de cotización](/quotecurrency). El par de divisas indica cuánto de la divisa cotizada se necesita para comprar una unidad de la divisa base.

Por ejemplo, [EUR/USD](/eur-usd-euro-us-dollar-currency-pair) es la cotización del euro frente al dólar estadounidense. EUR es el código de moneda ISO de tres letras para el euro y USD es el código para el dólar estadounidense. Un precio de cotización para este par de 1,2500 significa que un euro se cambia por 1,2500 dólares estadounidenses porque, en este caso, [EUR](/euro) es la moneda base y USD es la moneda de cotización (o moneda contraria). Esto significa que 1 euro se puede cambiar por 1,25 dólares estadounidenses. Otra forma de ver esto es que le costará USD $125 comprar EUR 100.

ISO no participó en transacciones de divisas hasta 1973, cuando el organismo de elaboración de normas decidió que sería útil participar. Después de cinco años de colaboración y deliberación, los primeros códigos de moneda estandarizados se publicaron en 1978, con un estándar sobre cómo deberían cambiar.

## Consideraciones Especiales

Según el sitio web de ISO, "ISO 4217:2015 especifica la estructura de un código alfabético de tres letras y un código numérico equivalente de tres dígitos para la representación de monedas. Para aquellas monedas que tienen unidades menores, también muestra la relación decimal entre tales unidades y la propia moneda". Por ejemplo, el código numérico de tres dígitos para el dólar estadounidense es 840 y el código numérico para el euro es 978. Pero no vería monedas cotizadas usando números como este (978/840).

Aunque, como explica el documento de ISO, "ISO 4217:2015 está diseñado para su uso en cualquier aplicación comercial, comercial y bancaria, donde se requiera describir las monedas y, cuando corresponda, los fondos. Está diseñado para ser igualmente adecuado para usuarios manuales y para aquellos que emplean sistemas automatizados". Si fuera útil hacerlo, los algoritmos comerciales o de procesamiento de pedidos podrían usar códigos numéricos para un procesamiento más efectivo.

## Códigos de divisas principales

El sitio web de ISO ofrece una lista completa de códigos de moneda en formatos XML y XMS. Todos los principales pares de divisas tienen [mercados muy líquidos](/liquidmarket) que operan las 24 horas del día, todos los días hábiles, y tienen [márgenes muy estrechos](/spread).

Los principales pares de divisas son los siguientes:

- EUR/USD - euro / dólar estadounidense

- [GBP/USD](/gbp-usd-british-pound-us-dollar-currency-pair) - [Libra esterlina (Gran Bretaña)](/gbp) / Dólar estadounidense

- USD/JPY - dólar estadounidense / [yen japonés](/jpy-japanese-yen)

- USD/CHF - dólar estadounidense / [franco suizo](/chf)

> El valor negociado frente al dólar estadounidense es el principal criterio para la clasificación como los principales pares de divisas.

>

Otras monedas importantes incluyen:

- AUD - [dólar australiano](/aud-australian-dollar)

- CAD - [dólar](/cad-canadian-dollar) [canadiense](/cad-canadian-dollar)

- CNY - [Yuan renminbi chino](/cny-china-yuan-renminbi)

- NZD - [dólar neozelandés](/nzd-new-zealand-dollar)

- INR - [rupia india](/inr)

- BZR - [Real brasileño](/brl-brazilian-real)

- SEK - [corona sueca](/sek-swedish-krona)

- ZAR - [Rand sudafricano](/zar-south-african-rand)

- HKD - [Dólar de Hong Kong](/hkd-hong-kong-dollars)

## Reflejos

- Los códigos de moneda ISO designan la base y cotizan las monedas en las cotizaciones de precios de divisas.

- Los comités de estándares ISO establecieron códigos de moneda estandarizados en 1978.

- Los códigos de moneda ISO son los códigos alfabéticos de tres letras que representan las diversas monedas utilizadas en todo el mundo.

- ISO también ha designado equivalentes numéricos menos conocidos para los códigos de moneda de tres letras.
