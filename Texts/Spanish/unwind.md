# Relajarse
## ¿Qué es deshacer una posición?

Relajarse es cerrar una posición comercial, y el término tiende a usarse cuando la operación es compleja o grande. Revertir también se refiere a la corrección de un error comercial, ya que corregir un error comercial puede ser complejo o requerir múltiples pasos u operaciones. Por ejemplo, un [corredor](/broker) vende por error parte de una posición cuando un inversor quería agregarla. El corredor tendría que deshacer la transacción comprando primero las acciones vendidas y luego comprando las acciones que deberían haberse comprado en primer lugar.

## Cómo funciona la relajación

La reversión se utiliza para referirse a las operaciones de cierre que requieren múltiples pasos, operaciones o tiempo. Si un inversor toma una posición [larga](/long) en acciones y, al mismo tiempo, vende [opciones de venta](/put) sobre el mismo tema, deberá cancelar esas operaciones en algún momento. Esto implica cubrir las opciones y vender las acciones subyacentes. Un corredor seguiría un proceso similar al intentar corregir un error de compra o venta.

La reversión es un proceso de reversión o cierre de una operación al participar en una transacción de compensación.

## Cerrar una posición

Cerrar una posición es el proceso requerido para eliminar una inversión particular de una [cartera](/portfolio). En el caso de los valores, cuando un inversor quiere cerrar la posición, la acción más común es vender el valor. En el caso de los [cortos](/short),. un inversor necesitaría volver a comprar las acciones cortas para cerrar la posición. Es más probable que se utilice el término reversión cuando la compra o venta se produce en varias transacciones, y no solo en una. Relajarse es un proceso.

## Relajación para corregir errores comerciales

Si un corredor realiza accidentalmente una acción incorrecta con los fondos de un inversionista, como comprar más de un valor en particular cuando la instrucción era venderlo, el corredor debe revender el valor que se compró accidentalmente para corregir el error. Luego deben realizar la venta original solicitada. Si el corredor experimenta una pérdida durante este proceso de corrección de errores, el corredor es responsable de la diferencia, no el inversor.

Otras actividades que pueden considerarse un error comercial incluyen la compra o venta de un valor distinto al especificado, la compra o venta de una cantidad incorrecta de un valor o la negociación de valores prohibidos. Los errores que se detectan antes de que se procesen por completo y que se cancelan con éxito no requieren corrección.

## Riesgo de Liquidación y Liquidez

[liquidez](/liquidity) puede tener efectos negativos en la capacidad de un inversionista o de un corredor para deshacer una transacción. La liquidez se refiere a la facilidad con la que se puede comprar o vender un activo en particular. Si un activo es menos líquido, es más difícil encontrar un comprador o vendedor adecuado, por lo que el riesgo de liquidez es elevado. Independientemente de si una transacción se completó intencional o accidentalmente, todos los riesgos asociados con el valor en particular aún se aplican al intentar cancelarlo.

## Reflejos

- Deshacer una posición es cerrarla.

- En algunos casos, la estrategia de desconexión también se utiliza para corregir errores comerciales.

- En general, las operaciones grandes y complejas son candidatas para deshacer una posición.
