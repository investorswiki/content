# Interfaz de pago unificado (UPI)
## ¿Qué es una interfaz de pago unificada (UPI)?

Una interfaz de pago unificado (UPI) es una aplicación para teléfonos inteligentes que permite a los usuarios transferir dinero entre cuentas bancarias. Es un sistema de pago móvil de ventanilla única desarrollado por la Corporación Nacional de Pagos de la India (NPCI). Elimina la necesidad de ingresar datos [bancarios](/bank) u otra información confidencial cada vez que un [cliente](/customer) inicia una transacción.

La interfaz de pago unificado es un sistema de pago en tiempo real. Está diseñado para permitir [transferencias](/transfer) interbancarias entre pares a través de un único proceso de autenticación de factor de dos clics. La interfaz está regulada por el [Banco de la Reserva de la India (RBI)](/rbi),. el banco central de la India. Funciona transfiriendo dinero entre dos cuentas bancarias junto con una plataforma móvil.

Se dice que el sistema es un método seguro para transferir dinero entre dos partes y elimina la necesidad de realizar transacciones con [efectivo físico](/cash) o a través de un banco. El sistema piloto se lanzó en India el 11 de abril de 2016. Los bancos de todo el país comenzaron a cargar su interfaz en agosto de 2016.

## Cómo funciona la interfaz de pagos unificados (UPI)

UPI utiliza los sistemas existentes, como el Servicio de Pago Inmediato (IMPS) y el Sistema de Pago Habilitado de Aadhaar (AEPS), para garantizar una liquidación sin problemas en todas las cuentas. Facilita las transacciones push (pagar) y pull (recibir) e incluso funciona para pagos [en ventanilla](/otc) o con código de barras, así como para múltiples pagos recurrentes, como facturas de servicios públicos, cuotas escolares y otras suscripciones.

> Según NPCI, 134 bancos estaban usando la interfaz a partir de febrero de 2019. Más de [Rs](/inr). 270 mil millones se canjearon a través de la UPI ese mismo mes.

>

Una vez que se establece un identificador único, el sistema permite que se entreguen pagos móviles sin el uso de tarjetas de crédito o débito, operaciones bancarias netas o cualquier necesidad de ingresar detalles de la cuenta. Esto no solo garantizaría una mayor seguridad de la información confidencial, sino que también conectaría a las personas que tienen cuentas bancarias a través de [teléfonos inteligentes](/smartphone) para realizar transacciones sin problemas.

En general, UPI implica menos transacciones en efectivo y potencialmente reduce la población [no bancarizada](/unbanked).

## Enviar dinero frente a recibir dinero

El envío de dinero a través de la UPI se denomina "empuje". Para enviar dinero, el usuario inicia sesión en la interfaz y selecciona la opción Enviar dinero/Pago. Después de ingresar la identificación virtual del destinatario y el monto deseado, elige la cuenta de la cual se debitará el dinero. Luego, el usuario ingresa un [número de identificación personal especial (PIN)](/personal-identification-number) y recibe una confirmación.

Recibir dinero a través del sistema se denomina "atracción". Una vez que el usuario ha iniciado sesión en el sistema, selecciona la opción de cobrar dinero. Luego, el usuario debe ingresar la identificación virtual del remitente, el monto a cobrar y la cuenta en la que depositará los fondos. Luego, un mensaje va al pagador con la solicitud de pago. Si decide realizar el pago, ingresa su PIN UPI para autorizar la transacción.

Una vez completada la transferencia, tanto el remitente como el destinatario reciben una confirmación por mensaje de texto en sus teléfonos inteligentes.

## Ejemplo de servicios ofrecidos por UPI

UPI ofrece una serie de funciones clave. Los usuarios pueden acceder a saldos e historiales de transacciones además de enviar y recibir dinero. Para enviar dinero, los usuarios necesitan un número de cuenta, el Código del Sistema Financiero Indio (o IFSC, que es un código alfanumérico que facilita las transferencias electrónicas), el número de teléfono móvil del destinatario y una identificación virtual o número Aadhaar (que es como un Social Numero de seguridad).

## Reflejos

- Una interfaz de pago unificado (UPI) es una aplicación de teléfono inteligente para la banca en la India.

- La interfaz está regulada por el Banco de la Reserva de la India (RBI), el banco central de la India, y los fondos se envían entre cuentas.

- Esta aplicación elimina la necesidad de ingresar datos bancarios u otra información confidencial cada vez que un cliente inicia una transacción, lo que la convierte en una forma segura de realizar operaciones bancarias.
