# tiempo de bloque
## ¿Qué es el tiempo de bloque?

El tiempo de bloque es la medida del tiempo que les toma a los mineros o validadores dentro de una red verificar las transacciones dentro de un bloque y producir un nuevo bloque en esa cadena de [bloques](/blockchain).

Las cadenas de bloques fueron popularizadas por primera vez por Bitcoin cuando se introdujo en 2009. La tecnología ha crecido a medida que se crean más criptomonedas, cada una de las cuales puede usar diferentes o las mismas cadenas de bloques, métodos de validación y técnicas para crear nuevos bloques.

## Comprender el tiempo de bloqueo

Una cadena de bloques es una base de datos distribuida que registra todas las transacciones dentro de una red de criptomonedas. Puede pensar en un bloque dentro de la base de datos como una celda en una hoja de cálculo donde se almacena la información de la transacción. Los mineros verifican las transacciones, lo que lleva tiempo porque encontrar la solución al bloque requiere que las computadoras realicen una gran cantidad de cálculos de prueba y error.

Esto se [denomina hash:](/hash) utiliza un algoritmo para verificar todas las transacciones dentro de un bloque, lo que valida la autenticidad de las transacciones y la información almacenada. Cuando se encuentra la solución de bloque, se crea un nuevo bloque. La cantidad de tiempo para encontrar la solución y crear un nuevo bloque es el tiempo de bloque.

Aquí hay algunos puntos clave para recordar si está tratando de entender el tiempo de bloqueo:

- Un bloque es un archivo que registra un número de las transacciones de criptomonedas más recientes.

- Cada bloque contiene una referencia al bloque que lo precedió (por eso es teóricamente imposible alterar la criptomoneda).

- Los "mineros" de criptomonedas compiten entre sí para resolver el hash, que es el número hexadecimal generado que verifica las transacciones. El ganador recibe una moneda criptográfica.

## ¿En qué se diferencia el tiempo de bloque de Bitcoin del de Ethereum?

Cada criptomoneda tiene un tiempo de bloque diferente: Bitcoin tarda unos 10 minutos, mientras que Ethereum solo tarda unos 14 segundos. La cantidad exacta de tiempo que lleva la generación de bloques varía y depende de la dificultad del hash (el número hexadecimal generado por el algoritmo hash). En otras palabras, los tiempos de bloqueo no siempre serán los mismos.

Existen mecanismos de consenso para permitir que una red acuerde que una transacción es válida. Las criptomonedas pueden utilizar diferentes mecanismos de consenso que, entre otros factores, afectan el tiempo que se tarda en verificar transacciones y crear nuevos bloques. La prueba de trabajo y la prueba de participación son dos tipos de mecanismos de consenso que utilizan diferentes métodos para verificar una transacción. Ethereum está en transición a un mecanismo de consenso de prueba de participación a lo largo de 2022, mientras que Bitcoin permanece en el mecanismo de prueba de trabajo más popular y que consume mucha energía.

## Reflejos

- Un bloque es verificado por mineros, que compiten entre sí para verificar las transacciones y resolver el hash, lo que crea otro bloque.

- Bajo el mecanismo de consenso de prueba de trabajo, la criptomoneda es recompensada por resolver el hash de un bloque y crear un nuevo bloque.

- El tiempo de bloque es el tiempo que se tarda en crear un nuevo bloque en una cadena de bloques de criptomonedas.

## PREGUNTAS MÁS FRECUENTES

### ¿Cómo obtengo un bloque de Bitcoin?

En realidad, nunca recibe un bloque de Bitcoin, ya que es parte del marco de Bitcoin. En cambio, recibe un Bitcoin cuando su minero resuelve el hash y crea otro bloque.

### ¿Cuántos Bitcoins se crearán alguna vez?

Bitcoin tiene un límite de 21 millones. Hay casi 19 millones de Bitcoins en circulación, y la cantidad de Bitcoins creados por año se reduce a la mitad cada cuatro años. Esto ralentiza la creación de Bitcoin.

### ¿Cuántos Ethereum se crearán alguna vez?

Ethereum, a diferencia de Bitcoin, no tiene un límite superior en la cantidad de monedas que se crearán.
