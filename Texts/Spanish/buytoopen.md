# Comprar para abrir
## ¿Qué es comprar para abrir?

"Comprar para abrir" es un término utilizado por las casas de bolsa para representar el establecimiento de una nueva (apertura) posición larga de [compra](/calloption) o [venta](/putoption) en [opciones](/option). Si un nuevo inversionista en opciones quiere comprar una opción call o put, ese inversionista debe comprar para abrir. Una orden de compra para abrir indica a los participantes del mercado que el comerciante está estableciendo una nueva posición en lugar de cerrar una posición existente. La orden [de venta para cerrar](/selltoclose) se utiliza para salir de una posición tomada con una orden de compra para abrir.

El establecimiento de una nueva posición corta se denomina [venta para abrir](/selltoopen),. que se cerraría con una orden [de compra para cerrar](/buytoclose). Si un nuevo inversionista en opciones quiere vender una opción call o put, ese inversionista debe vender para abrir.

## Comprender las órdenes de compra para abrir

La terminología de compra y venta para el comercio de opciones no es tan sencilla como lo es para el comercio de acciones. En lugar de simplemente colocar una orden de compra o venta como lo harían con las acciones, [los operadores de opciones](/trader) deben elegir entre "comprar para abrir", "comprar para cerrar", "vender para abrir" y "vender para cerrar".

Una posición de compra para abrir puede indicar a los participantes del mercado que el comerciante que inicia la orden cree algo sobre el mercado o tiene algo que hacer. Eso es particularmente cierto si el pedido es grande. Sin embargo, ese no tiene por qué ser el caso. De hecho, los comerciantes de opciones frecuentemente se involucran en actividades de [difusión](/spreadoption) o [cobertura](/hedge) donde una compra para abrir en realidad puede compensar las posiciones existentes.

> Comprar para abrir una opción de venta fuera del dinero al comprar una acción es una excelente manera de limitar el riesgo.

>

El intercambio puede declarar que solo se pueden realizar órdenes de cierre durante condiciones específicas del mercado, por lo que es posible que no se ejecute una orden de compra para abrir. Eso podría suceder si una acción con opciones disponibles está programada para ser [eliminada de la lista](/delisting) o si la bolsa detiene la negociación de la acción por un tiempo prolongado.

## Consideraciones sobre existencias

El término "comprar para abrir" también se puede aplicar a las acciones. Cuando un inversionista decide establecer una nueva posición en una acción en particular, la primera transacción de compra se considera compra para abrir porque abre la posición.

Al abrir la posición, la acción se establece como una participación en la cartera. La posición permanece abierta hasta que se cierra vendiendo todas las acciones. Eso se conoce como vender para cerrar porque cierra la posición. Vender una posición parcial significa que se han vendido algunas acciones, pero no todas. Una posición se considera cerrada cuando no queda más de una acción en particular en una cartera.

Las órdenes de compra para cerrar también entran en juego cuando se cubre una posición [de venta corta](/shortsale). Una posición de venta corta toma prestadas las acciones a través del corredor y se cierra recomprando las acciones en el mercado abierto. La última transacción para cerrar completamente la posición se conoce como orden de compra para cerrar. Esta transacción elimina la exposición por completo. La intención es recomprar las acciones a un precio más bajo para generar una ganancia de la diferencia entre el precio de venta al descubierto y el precio de compra para cerrar.

En los casos en que el precio de la acción suba considerablemente, un vendedor en corto puede tener que comprar para cerrar con pérdidas para evitar que ocurran pérdidas aún mayores. En el peor de los casos, el corredor puede ejecutar una liquidación forzada como resultado de una [llamada de margen](/margincall). Luego, el corredor exigiría que el inversionista coloque dinero en la cuenta de margen debido a un déficit. Eso generaría una orden [de compra para cubrir](/buytocover) para cerrar la posición con pérdidas debido a un capital insuficiente en la cuenta.

## Comprar para abrir vs. Comprar para cerrar

Si un inversionista quiere comprar una opción call o put para beneficiarse de un movimiento de precios del valor subyacente, entonces ese inversionista debe comprar para abrir. Comprar para abrir inicia una posición de opciones largas que le da al especulador el potencial de obtener una ganancia extremadamente grande con un riesgo muy bajo. Por otro lado, la seguridad debe moverse en la dirección correcta dentro de un tiempo limitado, o la opción perderá todo su valor debido al [tiempo de deterioro](/timedecay).

Los vendedores de opciones tienen una ventaja sobre los compradores debido al decaimiento del tiempo, pero aún pueden querer comprar para cerrar sus posiciones. Cuando un inversionista vende opciones, el inversionista permanece obligado por los términos de esas opciones hasta la fecha de vencimiento. Sin embargo, los movimientos en el precio del valor pueden permitir que los vendedores de opciones obtengan la mayor parte de sus ganancias mucho antes o los motiven a reducir las pérdidas.

Supongamos que alguien vende [en las opciones de venta de dinero](/atthemoney) que duran un año, y luego la acción subyacente sube un 10% después de tres meses. El vendedor de opciones puede comprar para cerrar y obtener la mayor parte de las ganancias de inmediato. Si la acción cae un 10% después de tres meses, el vendedor de opciones tendrá que pagar más para comprar para cerrar y limitar las pérdidas potenciales.

## Ejemplo de comprar para abrir

Supongamos que un comerciante ha realizado un análisis y cree que el precio de las acciones de XYZ pasará de $40 a $60 el próximo año. El comerciante podría comprar para abrir una llamada de XYZ. El [precio de ejercicio](/strikeprice) podría ser de $ 50 con una [fecha de vencimiento](/expirationdate) de aproximadamente un año a partir de ahora.

## Reflejos

- Una posición de compra para abrir en opciones crea la oportunidad de grandes ganancias con pérdidas mínimas, pero tiene un alto riesgo de expirar sin valor.

- Los comerciantes suelen utilizar una orden de compra para abrir para abrir posiciones en una opción o acción determinada.

- Comprar para abrir una posición de opciones puede compensar o cubrir otros riesgos en una cartera.
