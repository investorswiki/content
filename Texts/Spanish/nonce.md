# Mientras tanto
Un nonce se refiere a un número o valor que solo se puede usar una vez. Los nonces se utilizan a menudo en protocolos de autenticación y funciones hash criptográficas. En el contexto de la tecnología blockchain, un nonce se refiere a un número pseudoaleatorio que se utiliza como contador durante el proceso de minería.

Por ejemplo, los mineros de Bitcoin deben tratar de adivinar un nonce válido mientras realizan múltiples intentos para calcular un hash de bloque que cumpla con ciertos requisitos (es decir, que comience con una cierta cantidad de ceros). Al competir para extraer un nuevo bloque, el primer minero que encuentra un nonce que da como resultado un hash de bloque válido tiene el derecho de agregar el siguiente bloque a la cadena de [bloques](/blockchain),. y es recompensado por hacerlo.

En otras palabras, el proceso de minería consiste en que los mineros realicen una miríada de funciones hash con muchos valores nonce diferentes hasta que se produzca una salida válida. Si la salida de hashing de un minero cae por debajo del umbral predeterminado, el bloque se considera válido y se agrega a la cadena de bloques. Si la salida no es válida, el minero sigue intentando con diferentes valores de nonce. Cuando un nuevo bloque se extrae y valida con éxito, el proceso comienza de nuevo.

Dentro de Bitcoin, y la mayoría de los sistemas de Prueba de trabajo, el nonce es solo un número aleatorio que los mineros usan para iterar el resultado de sus cálculos de hash. Los mineros emplean un enfoque de prueba y error, en el que cada cálculo toma un nuevo valor de nonce. Lo hacen porque la probabilidad de adivinar con precisión un nonce válido es cercana a cero.

El protocolo ajusta automáticamente la cantidad promedio de intentos de hashing para garantizar que cada nuevo bloque se genere, en promedio, cada 10 minutos. Este proceso se conoce como ajuste de dificultad y es lo que determina el umbral de minería (es decir, con cuántos ceros debe comenzar el hash del bloque para que se considere válido). La dificultad de extraer un nuevo bloque está relacionada con la cantidad de poder de hash (tasa de hash) comprometida con un sistema de cadena de bloques. Cuanto más poder de hash se dedique a la red, mayor será el umbral, lo que significa que se requerirá más poder de cómputo para ser un minero competitivo y exitoso. Por el contrario, si los mineros deciden dejar de minar, la dificultad se ajustará y el umbral bajará, por lo que se requerirá menos potencia de hash para minar, pero el protocolo se asegurará de que la generación de bloques siga un programa de 10 minutos, independientemente .

## Reflejos

- Los mineros abren nuevos bloques generando un nonce que hace que el hash que crean sea menor que el hash del bloque que están validando.

- Nonce, un "número que solo se usa una vez", se refiere al número que un minero de blockchain necesita descubrir antes de encontrar un bloque en la blockchain.

- Una vez que el minero resuelve el hash, recibe la recompensa del bloque.

## PREGUNTAS MÁS FRECUENTES

### ¿Cómo se usan los nonces?

En criptomoneda, los nonces se usan como el número que se usa en un hash para verificar las transacciones y otros datos contenidos en un bloque.

### ¿Qué significa Nonce?

En criptomoneda, un nonce es una abreviatura de "número que solo se usa una vez", que es un número que se agrega a un bloque cifrado o cifrado en una cadena de bloques que, cuando se repite, cumple con las restricciones del nivel de dificultad. El nonce es el número que los mineros de blockchain están resolviendo para recibir la recompensa del bloque.

### ¿Qué es un Nonce de Blockchain?

Un nonce de cadena de bloques es un número que se agrega a un bloque con hash o cifrado en una cadena de bloques.
