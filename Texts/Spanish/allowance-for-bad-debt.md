# Provisión para deudas incobrables
## ¿Qué es una provisión para deudas incobrables?

Una provisión para deudas incobrables es una cuenta de valoración utilizada para estimar el monto de las cuentas por [cobrar de una empresa](/receivables) que, en última instancia, pueden ser incobrables. También se conoce como [provisión para cuentas de cobro dudoso](/allowancefordoubtfulaccounts). Cuando un prestatario incumple un préstamo, la provisión para la cuenta de [deudas incobrables](/baddebt) y el saldo de cuentas por cobrar del préstamo se reducen por el [valor en libros](/bookvalue) del préstamo.

## Cómo funciona una provisión para deudas incobrables

Los prestamistas usan una provisión para deudas incobrables porque el [valor nominal de las](/facevalue) [cuentas por cobrar](/accountsreceivable) totales de una empresa no es el saldo real que finalmente se cobra. En última instancia, una parte de las cuentas por cobrar no se pagará. Cuando un cliente nunca paga el monto de capital o interés adeudado en una cuenta por cobrar, la empresa finalmente debe cancelarla por completo.

## Métodos para estimar una provisión para deudas incobrables

Hay dos formas principales de calcular la asignación para deudas incobrables. Un método se basa en las ventas, mientras que el otro se basa en las cuentas por cobrar.

### Método de venta

El método de ventas estima la provisión para deudas incobrables como un porcentaje de las ventas a crédito a medida que ocurren. Suponga que una empresa gana $1,000,000 en ventas a crédito pero sabe por experiencia que el 1.5% nunca paga. Entonces, la estimación del método de venta de la provisión para deudas incobrables sería de $15,000.

### Método de cuentas por cobrar

El método de cuentas por cobrar es considerablemente más sofisticado y aprovecha la [antigüedad de las cuentas por cobrar](/accounts-receivable-aging) para proporcionar mejores estimaciones de la provisión para cuentas incobrables. La idea básica es que cuanto más tiempo no se pague una deuda, más probable es que nunca se pague. En este caso, quizás solo se agregaría el 1% de las ventas iniciales a la provisión para deudas incobrables.

Sin embargo, el 10% de las cuentas por cobrar que no se hubieran pagado después de 30 días podría agregarse a la provisión para deudas incobrables. Después de 90 días, podría subir al 50%. Finalmente, las deudas podrían cancelarse después de un año.

## Requisitos para una provisión por deudas incobrables

De acuerdo con los principios de contabilidad generalmente aceptados ( [GAAP](/gaap) ), el requisito principal para una provisión para deudas incobrables es que refleje con precisión el historial de cobro de la empresa. Si $2,100 de $100,000 en ventas a crédito no se pagaron el año pasado, entonces 2.1% es un método de ventas adecuado para estimar la provisión para deudas incobrables este año. Este proceso de estimación es fácil cuando la empresa ha estado operando durante algunos años. Las nuevas empresas deben usar promedios de la industria, [reglas generales](/rule-of-thumb) o números de otra empresa.

> Es necesaria una estimación precisa de la provisión para deudas incobrables para determinar el valor real de las cuentas por cobrar.

>

## Consideraciones predeterminadas

Cuando un prestamista confirma que el saldo de un préstamo específico está en mora, la empresa reduce la provisión para el saldo de cuentas incobrables. También reduce el saldo de las cuentas por cobrar del préstamo, porque el incumplimiento del préstamo ya no es simplemente parte de una estimación de la deuda incobrable.

## Consideraciones de ajuste

La provisión para deudas incobrables siempre refleja el saldo actual de los préstamos que se espera que incumplan, y el saldo se ajusta con el tiempo para mostrar ese saldo. Suponga que un prestamista estima que $2 millones del saldo del préstamo están en riesgo de incumplimiento y la cuenta de reserva ya tiene un saldo de $1 millón. Luego, la entrada de ajuste al gasto por deudas incobrables y el aumento a la cuenta de provisión es $1 millón adicional.

## Reflejos

- Las principales formas de estimar la provisión para cuentas incobrables son el método de ventas y el método de cuentas por cobrar.

- Una provisión para deudas incobrables es una cuenta de valoración utilizada para estimar el monto de las cuentas por cobrar de una empresa que, en última instancia, pueden ser incobrables.

- Los prestamistas utilizan una provisión para deudas incobrables porque el valor nominal de las cuentas por cobrar totales de una empresa no es el saldo real que finalmente se cobra.

- De acuerdo con los principios de contabilidad generalmente aceptados (PCGA), el requisito principal para una provisión para deudas incobrables es que refleje con precisión el historial de cobranza de la empresa.
