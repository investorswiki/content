# Licencia publica general
GPL, o Licencia pública general (GPL), se refiere a una licencia gratuita con copyleft para software. Otorga a cualquier persona los derechos de usar, leer, copiar, compartir, modificar y distribuir libremente un programa de computadora u otro tipo de trabajo. GPL se escribió originalmente para el Proyecto GNU y fue la primera licencia copyleft que se adoptó para uso general. Fue creado a principios de 1989 por Richard Stallman, el fundador de la Free Software Foundation (FSF).

A diferencia de los derechos de autor, el término copyleft significa que la GPL permite que se publiquen trabajos derivados, pero requiere que se distribuyan bajo los mismos términos de licencia que el trabajo original. Por lo tanto, los usuarios no pueden usar software con licencia GPL y publicar un trabajo derivado bajo otro tipo de licencia. Este no es el caso de otros tipos de licencias de software libre, como Berkley Software Distribution (BSD) y las licencias MIT.

Las licencias BSD y MIT caen dentro de la categoría de licencias permisivas. Si bien tanto las licencias copyleft como las permisivas permiten a los usuarios copiar, cambiar y distribuir software, sus condiciones son algo diferentes.

Por un lado, las licencias copyleft garantizan que el software de código abierto permanezca disponible para todos. También evita que otra persona se beneficie de un trabajo que se puso a disposición de forma gratuita. Los defensores del copyleft tienden a estar más preocupados por mantener cierto control sobre su trabajo.

Por otro lado, las licencias permisivas permiten que el software se use ampliamente, siempre que se haga referencia a los desarrolladores originales y se atribuya su trabajo. En otras palabras, una licencia permisiva le permite a cualquier persona copiar, cambiar y distribuir una obra, bajo cualquier tipo de licencia. El único requisito es dar créditos a los creadores originales.

Actualmente, la licencia GPL es la forma más utilizada para distribuir software libre y de código abierto. Los programas populares de software libre como GNU Emacs, GNU Compiler Collection (GCC) e incluso el kernel de [Linux](/linux) están sujetos a esta licencia.
