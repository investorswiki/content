# Moneda de nombre (NMC)
## ¿Qué es Namecoin (NMC)?

Namecoin es "un sistema de nombres peer-to-peer basado en Bitcoin". Es un token para una cadena de bloques inicialmente bifurcada de la cadena de bloques de Bitcoin. Una bifurcación es un cambio en una cadena de bloques que hace que la nueva rama no sea compatible con la rama original. Namecoin se desarrolló como una cadena de bloques y un token para un sistema de nombres de dominio (DNS) descentralizado.

Los creadores de Namecoin lo describen como "una tecnología experimental de código abierto que mejora la [descentralización](/decentralizedmarket),. la seguridad, la resistencia a la censura, la privacidad y la velocidad de ciertos componentes de la infraestructura de Internet, como el DNS y las identidades".

## Comprender Namecoin (NMC)

Namecoin utiliza el concepto de pares de nombre/valor de datos adjuntos a una moneda. Un par nombre/valor es un valor de datos y un nombre que se utiliza para identificar el valor en una base de datos. En este caso, el par nombre/valor es una dirección numérica legible por computadora con un nombre legible por humanos. Se crea un token para el par nombre/valor y el titular del token es propietario del par nombre/valor. Por lo tanto, una Namecoin se utiliza para establecer la propiedad de un nombre de dominio.

### Historial de Namecoin

La idea de desarrollar un sistema de nombre de dominio tokenizado se presentó por primera vez en 2010 en una sala de Internet Relay Chat (IRC), #bitcoin-dev, donde los usuarios discutieron los méritos de un sistema DNS similar a bitcoin que se conoció como bitdns. Namecoin fue presentado en abril de 2011 por Vincent Durham (un nombre anónimo) en el mismo foro de Bitcoin utilizado para anunciar Bitcoin.

> Namecoin cuenta con un equipo activo de desarrolladores y personal de soporte que trabaja para continuar mejorando la cadena de bloques, los casos de uso y atraer a los usuarios.

>

## ¿En qué se diferencia Namecoin de Bitcoin?

Namecoin presenta varias similitudes con Bitcoin, actualmente la principal moneda digital por capitalización de mercado. La bifurcación de Namecoin se inició para crear una nueva forma de establecer la propiedad de las direcciones legibles por humanos asignadas a las direcciones legibles por máquina utilizadas en la web: los nombres de dominio.

La principal diferencia entre Bitcoin y Namecoin es el propósito de la tecnología. Los creadores de Bitcoin querían una moneda alternativa viable; Los creadores de Namecoin querían un sistema de nombres de dominio. Debido a esto, existen diferentes reglas de consenso y protocolo que existen dentro de cada proyecto. Ciertas características son necesarias al realizar transacciones financieras que no son necesarias al completar registros de nombres para nuevos dominios de Internet.

Debido a que Namecoin utiliza el mismo algoritmo de [prueba](/proof-work) [de trabajo](/proof-work) (PoW) que Bitcoin, puede fusionar y extraer Namecoins sin hardware adicional (y sin electricidad adicional) si ya está extrayendo Bitcoin. Tanto Bitcoin como Namecoin están limitados a un total de 21 millones de monedas. La diferencia fundamental en Namecoin es que los tokens se agotan a medida que se registran los nombres de dominio. El costo de un nombre de dominio para el suministro utilizable de Namecoin es .01 NMC. Esto significa que hay 2100 millones de nombres de dominio (u otros usos) disponibles con Namecoin.

## Objetivos de Namecoin

Los desarrolladores de Namecoin indicaron varios usos y aplicaciones potenciales para este token experimental. En primer lugar, los desarrolladores esperan proteger los derechos de libertad de expresión en línea haciendo que la web sea más resistente a la censura.

> Namecoin es un sistema de registro y transferencia de pares clave/valor basado en tecnología Bitcoin. Esto significa que Namecoin se puede utilizar para registrar y transferir nombres o claves arbitrarias de forma segura. También puede adjuntar datos a estos nombres.

>

Hay miles de formas en que Namecoin intenta hacer esto. Por ejemplo, se puede usar para adjuntar información de identificación, como direcciones de correo electrónico, direcciones públicas de criptomonedas o claves específicas, a varias identidades según lo determine el usuario. También se puede utilizar como un medio para proporcionar una validación de certificado de sitio web descentralizada.

Namecoin se puede usar en Tor y otras plataformas web anonimizadoras para generar dominios Tor .onion significativos para los humanos. En el futuro, el token y su tecnología subyacente también podrían usarse para firmas de archivos, asegurar procedimientos de votación, servicios notariales y establecer pruebas de existencia para personas y entidades.

Debido a sus vínculos con la red Namecoin, estos nombres son difíciles de censurar o incautar, lo que significa que son resistentes a la influencia externa. Además, los creadores de Namecoin especifican que las búsquedas no generan tráfico de red. El resultado de esto es que Namecoin también ofrece capacidades de privacidad mejoradas.

## Reflejos

- Namecoin intenta reemplazar la tecnología de servidor de nombres de dominio existente con tecnología blockchain.

- Namecoin es una cadena de bloques con un token bifurcado inicialmente del software Bitcoin, que se utiliza para representar la propiedad de un nombre de dominio.

- Fue desarrollado para descentralizar la infraestructura de Internet, reducir la censura y aumentar la privacidad y la seguridad.

## PREGUNTAS MÁS FRECUENTES

### ¿Es Namecoin una Altcoin?

La definición de la comunidad de criptomonedas de una altcoin es una que no es Bitcoin. Algunos también incluyen Ethereum en la definición, por lo que Namecoin es una moneda alternativa.

### ¿Cuánto vale Namecoin?

Los desarrolladores de Namecoin recomiendan que no compre NMC con fines de inversión o moneda, ya que no está diseñado para usarse de esta manera. Sin embargo, NMC ha oscilado entre $.20 y $2.81 desde 2017.

### ¿Todavía existe Namecoin?

Namecoin sigue siendo un token válido. Puede comprar NMC en algunos intercambios descentralizados, pero el token se usa mejor para el propósito previsto.
