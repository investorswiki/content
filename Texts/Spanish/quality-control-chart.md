# Gráfico de control de calidad
## ¿Qué es un gráfico de control de calidad?

Un gráfico de control de calidad es un gráfico que muestra si los productos o procesos muestreados cumplen con las especificaciones previstas. De lo contrario, el gráfico mostrará el grado en que varían de las especificaciones. Un gráfico de control de calidad que analiza un atributo específico de un producto se denomina gráfico univariante, mientras que un gráfico que mide las variaciones en varios atributos del producto se denomina gráfico multivariante. Los productos seleccionados al azar se prueban para los atributos dados que el gráfico está rastreando.

## Comprender los gráficos de control de calidad

[El control de calidad](/quality-control) (QC) es un conjunto de procesos mediante los cuales una empresa garantiza que la calidad del producto se mantenga o mejore. El control de calidad requiere que la empresa cree un entorno en el que tanto la dirección como los empleados se esfuercen por alcanzar la perfección. Esto se hace capacitando al personal, creando puntos de [referencia](/benchmark) para la calidad del producto y probando productos para verificar variaciones [estadísticamente significativas .](/statistically_significant) Un aspecto importante del control de calidad es el establecimiento de controles bien definidos. Estos controles ayudan a estandarizar tanto la producción como las reacciones a los problemas de calidad. Limitar el margen de error al especificar qué actividades de producción deben completarse y qué personal reduce la posibilidad de que los empleados participen en tareas para las que no tienen la capacitación adecuada.

Los gráficos de control de calidad son un tipo de control que los ingenieros suelen utilizar para evaluar el rendimiento de los procesos o productos terminados de una empresa. Si se detectan problemas, se pueden comparar fácilmente con su ubicación en el gráfico para la depuración o el control de errores. En otras palabras, proporciona un modelo heurístico para mantener el control de calidad.

Una forma común del gráfico de control de calidad es el gráfico de barra x (indicado como x̅**)**, donde el eje y del gráfico rastrea el grado en que la varianza del atributo probado es aceptable. El eje x rastrea las muestras analizadas. El análisis del patrón de [variación](/variance) representado por un gráfico de control de calidad puede ayudar a determinar si los defectos se producen de forma aleatoria o sistemática.

El gráfico R (rango) es un gráfico de control de calidad que se utiliza para monitorear la variación de un proceso basado en muestras pequeñas tomadas en momentos específicos.

Un gráfico de control de calidad también puede ser univariante o multivariante, lo que significa que puede mostrar si un producto o proceso se desvía de uno o más de un resultado deseado.

> Se utilizan diferentes tipos de gráficos de control de calidad, como gráficos de barras X, gráficos S y gráficos Np, según el tipo de datos que deben analizarse.

>

## Ejemplo de un gráfico de control de calidad

Por ejemplo, Bob quiere saber si su prensa de widgets está creando widgets que cumplen con los estándares. Decide probar la densidad de una muestra aleatoria de widgets para ver si el sistema de inyección de aire a presión funciona correctamente y si mezcla suficiente aire en la masa de widgets. Un lote adecuadamente aireado de masa para widgets hará que el widget terminado flote en el agua. Bob crea un gráfico de barras X para realizar un seguimiento del grado de flotabilidad de cada widget seleccionado al azar.

## Reflejos

- Si aparecen problemas, la tabla de control de calidad se puede utilizar para identificar el grado en que se diferencian de esas especificaciones y ayudar en la corrección de errores.

- Una representación común del gráfico de control de calidad es el gráfico de barras x, donde el eje y rastrea la varianza del atributo probado y es aceptable de forma univariante o multivariante.

- Un gráfico de control de calidad es una representación gráfica de si los productos o procesos de una empresa cumplen con las especificaciones previstas.
