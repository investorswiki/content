# GitHub
GitHub to internetowa platforma programistyczna typu open source i usługa hostingowa, która umożliwia użytkownikom przesyłanie plików, dokumentów i kodu komputerowego na swoje konta. GitHub umożliwia zarówno zarejestrowanym, jak i niezarejestrowanym użytkownikom przeglądanie, pobieranie, a nawet współtworzenie przesłanych plików i repozytoriów kodu źródłowego. Ze względu na niski koszt i funkcje, GitHub stał się największą platformą kodu źródłowego na świecie, wykorzystywaną przez wielu programistów jako sposób na wykonywanie pracy zespołowej, a jednocześnie umożliwiającą udostępnianie różnych programów, plików i dokumentacji.

#### Jak działa GitHub

Po zarejestrowaniu konta GitHub użytkownik może stworzyć własny katalog plików i treści, który nazywa się repozytorium. Po utworzeniu repozytorium można stworzyć tzw. master branch. Gałęzie główne GitHub reprezentują pojedynczy punkt początkowy każdego projektu. Do użytkownika należy udostępnienie projektu każdemu (repozytorium publiczne) lub ograniczenie dostępu do niego i udostępnienie go tylko wybranym użytkownikom (repozytorium prywatne). Z głównej gałęzi można tworzyć podgałęzie, co w zasadzie pozwala użytkownikowi pracować oddzielnie nad różnymi aspektami projektu. W publicznym repozytorium inni użytkownicy mogą sugerować zmiany w kodzie, ostatecznie przesyłając nowe wersje plików. Sugestie są tworzone za pomocą funkcji zwanej pull request, która umożliwia programistom omawianie i przeglądanie potencjalnych zmian przed ich skutecznym zastosowaniem. Platforma GitHub oferuje różnorodne funkcje i narzędzia, które ułatwiają programistom wprowadzanie lub cofanie zmian oraz śledzenie ich ogólnego postępu.

#### Co to jest żądanie ściągnięcia?

Żądanie ściągnięcia umożliwia użytkownikom poinformowanie innych o zmianach, które wprowadzili w określonej gałęzi lub repozytorium. Po utworzeniu pull requesta użytkownik zostaje przeniesiony do ekranu przeglądu, na którym może łatwo przejrzeć wszystkie zmiany wprowadzone w kodzie, porównując starą i nową wersję. Dlatego wszystkie zmiany są dokumentowane i rejestrowane, a użytkownicy mogą również dodawać komentarze, etykiety i kamienie milowe, a nawet przydzielać określone zadania innym współtwórcom.

#### Kto korzysta z GitHub?

Ze względu na elastyczną platformę GitHub ma użytkowników zarówno z sektora publicznego, jak i prywatnego. Firmy programistyczne, twórcy aplikacji, indywidualni programiści i twórcy kryptowalut to tylko kilka przykładów. Jako projekt open source, [Bitcoin GitHub](/bitcoin) ma wielu programistów na całym świecie, z wieloma osobami przyczyniającymi się do przeglądu i ulepszania kodu źródłowego Bitcoin.

#### Czy GitHub jest bezpłatny?

GitHub jest bezpłatny dla wszystkich projektów deweloperskich typu open source i komercyjnych. Jednak bezpłatne plany ograniczają liczbę prywatnych i publicznych repozytoriów, które można utworzyć za pomocą konta. Darmowe plany ograniczają również liczbę współpracowników, których może mieć projekt. Dlatego większe projekty mogą wymagać uaktualnienia do płatnego planu GitHub, zgodnie z ich potrzebami.
