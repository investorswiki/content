# Web 2.0
## Co to jest Web 2.0?

Web 2.0 opisuje obecny stan Internetu, który ma więcej treści generowanych przez użytkowników i użyteczności dla użytkowników końcowych w porównaniu do wcześniejszego wcielenia, Web 1.0. Ogólnie rzecz biorąc, Web 2.0 odnosi się do aplikacji internetowych XXI wieku, które przekształciły erę cyfrową w następstwie [bańki dotcomów](/dotcom-bubble).

## Zrozumienie Web 2.0

Web 2.0 nie odnosi się do żadnych konkretnych technicznych aktualizacji Internetu. Odnosi się po prostu do zmiany sposobu korzystania z Internetu w XXI wieku. W nowej erze istnieje wyższy poziom wymiany informacji i wzajemnych powiązań między uczestnikami. Ta nowa wersja pozwala użytkownikom aktywnie uczestniczyć w doświadczeniu, a nie tylko działać jako pasywni widzowie, którzy przyswajają informacje.

## Sieć 1.0 a Web 2.0

Web 1.0 służy do opisu pierwszego etapu Internetu. W tym momencie było niewielu twórców treści; większość korzystających z Internetu to konsumenci. Strony statyczne były bardziej popularne niż dynamiczny [HTML](/html),. który zawiera interaktywne i animowane witryny z określonym kodem lub językiem. Zawartość na tym etapie pochodziła z systemu plików serwera, a nie z systemu zarządzania bazą danych. Użytkownicy mogli wpisywać się do księgi gości online, a formularze HTML wysyłane były pocztą elektroniczną.

Przykładami witryn internetowych sklasyfikowanych jako Web 1.0 są Britannica Online, witryny osobiste i mp3.com. Ogólnie rzecz biorąc, te strony internetowe są statyczne i mają ograniczoną funkcjonalność i elastyczność.

Termin Web 2.0 pojawił się po raz pierwszy w 1999 roku, gdy Internet przesunął się w kierunku systemu, który aktywnie angażował użytkownika. Użytkowników zachęcano do dostarczania treści, a nie tylko do ich przeglądania. Ludzie mogli teraz publikować artykuły i komentarze, a także tworzyć konta użytkowników w różnych witrynach, zwiększając w ten sposób uczestnictwo. Web 2.0 dało również początek aplikacjom internetowym, platformom do samodzielnego publikowania, takim jak [WordPress](/wordpress-cms),. a także serwisom [społecznościowym](/social-media).

Przykładami witryn Web 2.0 są Wikipedia, Facebook, Twitter i różne blogi, które zmieniły sposób udostępniania i dostarczania tych samych informacji.

Szczególnie zmienił się społeczny aspekt Internetu; ogólnie rzecz biorąc, media społecznościowe pozwalają użytkownikom na interakcję i interakcję ze sobą poprzez dzielenie się przemyśleniami, perspektywami i opiniami. Użytkownicy mogą oznaczać, udostępniać, tweetować i lubić.

## Zalety i wady Web 2.0

Rozwój technologii umożliwił użytkownikom dzielenie się swoimi przemyśleniami i opiniami z innymi, co stworzyło nowe sposoby organizowania i łączenia się z innymi ludźmi oraz promowało większy stopień współpracy.

Internet ma jednak wiele wad, które działają bardziej jak otwarte forum. Dzięki ekspansji mediów społecznościowych zaobserwowaliśmy wzrost liczby prześladowań online, cybernękania, [doxingu](/personally-identifiable-information-pii),. [kradzieży tożsamości](/identitytheft) i innych przestępstw internetowych. Istnieje również zagrożenie rozpowszechniania się dezinformacji wśród użytkowników, czy to za pośrednictwem witryn udostępniających informacje o otwartym kodzie źródłowym, czy w mediach społecznościowych.

Podobnie jak Web 1.0, Web 2.0 jest kolejną fazą przejściową w ewolucji Internetu. Przewiduje się, że sieć 3.0 będzie nazywana siecią semantyczną, ponieważ zostanie dostosowana do potrzeb każdego użytkownika, aby była bardziej intuicyjna.

##Przegląd najważniejszych wydarzeń

- W nowej erze Internetu istnieje wyższy poziom wymiany informacji i wzajemnych powiązań między uczestnikami.

- Web 2.0 nie odnosi się do żadnych konkretnych technicznych aktualizacji Internetu; odnosi się do zmiany sposobu korzystania z Internetu.

- Web 2.0 opisuje obecny stan Internetu, który ma więcej treści tworzonych przez użytkowników i użyteczności dla użytkowników końcowych w porównaniu do wcześniejszego wcielenia, Web 1.0.
