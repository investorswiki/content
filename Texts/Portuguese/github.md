# GitHubGenericName
O GitHub é uma plataforma de desenvolvimento de código aberto baseada na Web e serviço de hospedagem que permite aos usuários fazer upload de arquivos, documentos e códigos de computador para suas contas. O GitHub permite que usuários registrados e não registrados visualizem, baixem e até contribuam para os arquivos carregados e repositórios de código-fonte. Devido ao seu baixo custo e recursos, o GitHub tornou-se a maior plataforma de código-fonte do mundo, sendo utilizado por muitos desenvolvedores como forma de realizar trabalhos colaborativos, além de poder compartilhar uma variedade de softwares, arquivos e documentação.

#### Como o GitHub funciona

Uma vez que uma conta do GitHub é registrada, o usuário pode criar seu próprio diretório de arquivos e conteúdo, que é chamado de repositório. Depois de criar um repositório, pode-se criar o chamado branch master. As ramificações principais do GitHub representam o único ponto de partida de qualquer projeto. Cabe ao usuário disponibilizar um projeto para qualquer pessoa (repositório público) ou restringir o acesso a ele e compartilhar apenas com determinados usuários (repositório privado). A partir de um branch master, podem ser criados sub-branchs, o que basicamente permite que o usuário trabalhe em vários aspectos do projeto separadamente. Em um repositório público, outros usuários podem sugerir alterações no código, eventualmente fazendo upload de novas versões dos arquivos. As sugestões são feitas por meio de um recurso chamado pull request, que permite que os desenvolvedores discutam e revisem as possíveis alterações antes de serem efetivamente aplicadas. A plataforma GitHub oferece uma variedade de recursos e ferramentas que facilitam para os desenvolvedores fazer ou reverter alterações e acompanhar seu progresso geral.

#### O que é um pull request?

Uma solicitação pull permite que os usuários informem aos outros sobre as alterações que fizeram em uma determinada ramificação ou repositório. Quando um pull request é criado, o usuário é levado a uma tela de visão geral onde pode revisar facilmente todas as alterações feitas no código, comparando a versão antiga com a nova. Portanto, todas as alterações são documentadas e registradas e os usuários também podem adicionar comentários, rótulos e marcos ou até mesmo atribuir tarefas específicas a outros colaboradores.

#### Quem usa o GitHub?

Por causa de sua plataforma flexível, o GitHub tem usuários dos setores público e privado. Empresas de software, desenvolvedores de aplicativos, programadores individuais e desenvolvedores de criptomoedas são apenas alguns exemplos. Como um projeto de código aberto, o [Bitcoin GitHub](/bitcoin) tem muitos desenvolvedores em todo o mundo, com muitas pessoas contribuindo para a revisão e melhoria do código-fonte do Bitcoin.

#### O GitHub é gratuito?

O GitHub é gratuito para todos os projetos de desenvolvimento comercial e de código aberto. No entanto, os planos gratuitos limitam o número de repositórios privados e públicos que podem ser criados por uma conta. Os planos gratuitos também limitam o número de colaboradores que um projeto pode ter. Portanto, projetos maiores podem precisar atualizar para um plano pago do GitHub, de acordo com suas necessidades.
