# Análise Discriminante Múltipla (MDA)
## O que é Análise Discriminante Múltipla (MDA)?

O termo análise discriminante múltipla (MDA) refere-se a uma técnica estatística usada por [planejadores financeiros](/financialplanner),. [consultores](/investmentadvisor) de investimentos e analistas para avaliar investimentos potenciais quando muitas variáveis estão em jogo. A MDA permite aos profissionais financeiros a viabilidade de investir em diversos títulos do mercado, estudando diferentes fatores ou variáveis, como a volatilidade. Este é um ramo da análise discriminante, que é utilizado por pesquisadores e estatísticos que fazem classificações de indivíduos e dados com base em diferentes variáveis.

## Entendendo a Análise de Discriminação Múltipla (MDA)

A análise discriminante múltipla é uma técnica que distingue conjuntos de dados uns dos outros com base nas características observadas por um profissional. Ele é usado em [finanças](/finance) para comprimir a [variação](/variance) entre títulos enquanto seleciona várias variáveis.

Ao usar a técnica MDA, os profissionais financeiros reduzem as diferenças entre certas variáveis para que possam ser classificadas em vários grupos maiores e depois comparadas com outra variável. Na maioria dos casos, os profissionais que usam o MDA geralmente tentam agrupar os dados em pelo menos três, se não mais, grupos diferentes.

Um analista que está considerando um número de ações pode usar a análise discriminante múltipla como uma ferramenta para se concentrar nos pontos de dados que são os mais importantes. Isso simplifica as demais diferenças entre as [ações](/stock) sem descartá-las totalmente. Por exemplo, um analista que deseja selecionar títulos com base em valores que medem [volatilidade](/volatility) e consistência histórica pode usar o MDA para fatorar outras variáveis, como preço.

> Outras variáveis que os analistas podem usar ao empregar análises discriminantes múltiplas incluem diferentes índices financeiros.

>

A principal razão pela qual os profissionais utilizam esta técnica é desenvolver [conjuntos eficientes de Markowitz](/markowitzefficientset). Essas [carteiras de investimento](/portfolio) são desenvolvidas com base em retornos que são maximizados para um determinado nível de risco. Estes foram nomeados em homenagem ao economista Harry Markowitz, que também é considerado o pai da [moderna teoria de portfólio (MPT)](/modernportfoliotheory).

## Considerações Especiais

Como observado acima, a análise discriminante múltipla está relacionada à análise discriminante, que é comumente usada por estatísticos e outros pesquisadores. A MDA também é conhecida, pelo menos para os estatísticos, como análise de variáveis canônicas ou análise discriminante canônica.

É um tipo de análise discriminante, que é amplamente utilizado por pesquisadores que [analisam dados](/data-analytics) em muitos campos. A análise discriminante ajuda pesquisadores e estatísticos a classificar diferentes conjuntos de dados definindo uma regra ou selecionando um valor que fornecerá a separação mais significativa.

##Destaques

- MDA é um ramo da análise discriminante, que é comumente usado por estatísticos e outros pesquisadores.

- Esta técnica é usada para comprimir a variância entre os títulos durante a triagem de várias variáveis.

- Os profissionais financeiros costumam usar o MDA como forma de desenvolver conjuntos eficientes de Markowitz, um tipo de portfólio que maximiza os retornos com base em determinados níveis de risco.

- Um analista que está considerando um número de ações pode usar a análise discriminante múltipla para se concentrar nos pontos de dados que são mais importantes para a decisão em questão.

- A análise discriminante múltipla é usada por planejadores financeiros para avaliar investimentos potenciais quando várias variáveis devem ser levadas em consideração.
