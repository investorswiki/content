# 0x Protocolo
## Qual é o protocolo 0x?

O protocolo 0x (zero x) é um protocolo que permite a troca ponto a ponto de ativos na blockchain Ethereum. O protocolo 0x foi lançado em 2017 pela 0x Labs, uma organização sediada em São Francisco focada na criação de novos mercados no ecossistema 0x.

O protocolo em si não é, por definição, uma troca descentralizada; ele permite a criação de trocas descentralizadas que podem ser usadas em amplas aplicações.

## Entendendo o protocolo 0x

No mundo das moedas digitais, um protocolo é um conjunto de regras padrão que podem ser usadas por um sistema (ou por várias partes transacionais) para se comunicarem sem problemas. O protocolo 0x é essencialmente um formato de mensagens padrão que as entidades transacionais podem usar para trocar ativos digitais ou tokens.

O sistema de mensagens SWIFT padrão usado no setor bancário é semelhante a um protocolo (exceto que é usado em um contexto diferente). Bancos em todo o mundo usam o sistema de mensagens [SWIFT padrão para se comunicar sobre transferências de dinheiro.](/swift)

O sistema possui um conjunto padrão de campos de mensagens definidos para todos os usuários; seus valores correspondentes são usados no sistema SWIFT para transmitir com segurança detalhes entre duas partes, como remetente, destinatário, valor, moeda, filial de origem e filial de destino, entre outros.

> SWIFT cria um formato padrão e universalmente aceitável que permite operações mais suaves e maior eficiência.

>

Como cada banco se adaptou às regras padrão do sistema de mensagens SWIFT, eles podem realizar transações diretamente entre si. A situação daria errado se cada banco seguisse seu próprio protocolo distinto, porque então eles teriam que aderir a um canal de comunicação individual com todos os outros bancos.

## Gols de 0x

O objetivo do 0x Labs é criar a infraestrutura necessária para a economia emergente de criptomoedas e permitir a criação de mercados que não poderiam existir antes.

O protocolo 0x tenta funcionar de forma semelhante ao SWIFT, mas no contexto da operação de trocas descentralizadas para negociação de tokens digitais e ativos que são executados no blockchain Ethereum. Construído com base nos padrões de token Ethereum, o protocolo 0x atua como a principal camada de infraestrutura para o crescente número de aplicativos e instrumentos financeiros que estão sendo criados usando a tecnologia blockchain e negociando em formulários digitais.

A cada dia que passa, os tokens estão se tornando uma parte crescente do valor financeiro do mundo. consequentemente, a necessidade de negociar ativos e tokens digitais de forma segura e eficiente está crescendo. Com seus formatos de mensagem claramente definidos e contratos inteligentes, o protocolo 0x tenta responder a essa necessidade.

> A palavra "token" é frequentemente usada como sinônimo de "moeda". No entanto, um token é diferente de uma moeda porque representa um ativo, enquanto uma moeda é usada como meio de troca.

>

É usado em uma ampla variedade de mercados, como jogos e financiamento, que permite que os usuários negociem tokens e ativos.

ZRX é o token de governança e staking nativo da 0x. Os proprietários do ZRX têm uma opinião sobre como o protocolo evolui, e os detentores de tokens também podem apostar seus tokens para ganhar recompensas de liquidez ETH (o Ethereum token ether).

## Considerações Especiais

O formato de mensagem do protocolo 0x é composto por um conjunto de campos de dados que carregam informações vitais, como o ativo digital ou token a ser negociado, o valor do preço da transação, o prazo de validade e as identidades definidas das partes transacionais.

Os contratos inteligentes cuidam da lógica de negócios necessária para gerar, enviar, receber e processar os dados vinculados à atividade de negociação. Também permite espaço para atualizações essenciais, se houver, no futuro. As provisões para atualizações são necessárias se quaisquer alterações forem necessárias para aderir aos regulamentos modificados ou quaisquer modificações relacionadas ao funcionamento intrínseco da rede blockchain Ethereum.

O sistema também usa retransmissores, que atuam como agregadores de ordens e transmitem ordens de participantes do mercado designados para o mercado ou bolsa.

##Destaques

- O protocolo 0x é um protocolo aberto que permite a troca ponto a ponto de ativos na blockchain Ethereum.

- Construído com base nos padrões de token Ethereum, o protocolo 0x atua como a camada de infraestrutura crítica para o crescente número de aplicativos financeiros que implementam a pilha de tecnologia blockchain.

- 0x Labs, uma organização focada na criação de novos mercados no ecossistema 0x, construiu o protocolo 0x.

##PERGUNTAS FREQUENTES

### 0x é um DEX?

É semelhante a uma troca descentralizada, mas tem diferenças únicas. Uma das diferenças é que um DEX armazena pedidos em seu blockchain, enquanto 0x não.

### Posso minerar 0x?

0x é um protocolo para o blockchain Ethereum. Ele facilita as trocas peer-to-peer de ativos construídos usando Ethereum, portanto, não há token para minerar.

### O que é a moeda Zerox?

Criado pela empresa 0x (zero x), o ZRX é um token construído na blockchain Ethereum para alimentar as trocas de criptomoedas.
