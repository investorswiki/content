# Baseando
## O que é base?

Basing é um termo usado por [analistas técnicos](/technical-analyst) [que](/technical-analyst) se refere a uma consolidação no preço de um título, geralmente após uma tendência de baixa, antes de iniciar sua fase de alta. O padrão de preço resultante parece plano ou ligeiramente arredondado.

## Entendendo a Base

Basear é uma ocorrência comum depois que um título, ou o mercado, está em um longo declínio ou está no meio de um avanço significativo. Em outras palavras, o mercado está fazendo uma pausa. Alguns títulos, como [ações](/stock),. podem formar uma base que dura vários anos antes que a tendência se reverta. Os períodos de base são acompanhados de queda de volume e há um equilíbrio entre oferta e demanda. A volatilidade também se contrai quando uma ação é negociada lateralmente.

Os títulos que estão se baseando estabelecem níveis claros de suporte e resistência à medida que os touros e os ursos lutam pelo controle. Os traders institucionais podem usar um período de base para [acumular](/accumulation) uma grande posição em nome de seus clientes. Muitos analistas técnicos acreditam que a base é crucial, especialmente para ações que tiveram um rápido declínio, antes que uma reversão significativa possa começar. A base também pode ser vista como a 'pausa que atualiza' que permite que um título retome seu movimento de alta.

## Baseando Estratégias de Negociação

**Continuação da tendência:** Os traders que estão usando um período base para encontrar um ponto de entrada em um mercado de tendência devem fazer uma negociação quando o preço ultrapassar o máximo do intervalo consolidado (para uma [posição comprada](/long) ). A fuga deve ocorrer em volume acima da média para mostrar a participação no movimento. Idealmente, uma média móvel comumente usada, como a de 20 dias ou 50 dias, atua como suporte na parte inferior do período base; isso permite que a média móvel alcance o preço. A média móvel atua como resistência para uma posição curta.

A faixa estreita de uma formação de base permite uma relação risco/recompensa saudável. Os comerciantes podem colocar uma ordem stop-loss abaixo do preço mais baixo negociado no período base. Como a expectativa é que o mercado volte a ter tendência, [metas de lucro](/profit-target) que são muitos múltiplos do valor de parada podem ser definidas para capturar a maior parte do movimento.

**Reversão de tendência:** traders contrários podem usar um período de base para encontrar fundos ou topos em potencial em um título. Se um mercado estiver se consolidando por um longo tempo, uma [quebra](/breakout) na direção oposta à tendência anterior geralmente aciona ordens de stop-loss e atrai traders, levando a um ambiente propício para uma reversão. Tal como acontece com a estratégia de continuação de tendência, a negociação deve ser encerrada se o preço ultrapassar o preço mais baixo negociado durante o período de base. Os comerciantes podem usar retrocessos da tendência anterior para definir metas de lucro.
