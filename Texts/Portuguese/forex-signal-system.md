# Sistema de Sinais Forex
## O que é um sistema de sinais Forex?

Um sistema de sinal forex é um conjunto de análises que um trader forex usa para gerar sinais para determinar se deve comprar ou vender um par de moedas a qualquer momento. Os sistemas de sinais Forex podem ser baseados em ferramentas de gráficos [de análise técnica](/technicalanalysis) ou eventos baseados em notícias. Um sistema de sinais para negociação forex geralmente é composto de vários sinais diferentes que trabalham juntos para criar uma decisão de compra ou venda. Os sistemas de sinais de negociação podem estar disponíveis gratuitamente, por uma taxa, ou são desenvolvidos internamente pelos próprios traders.

## Entendendo os Sistemas de Sinais Forex

Um [sinal de negociação](/trade-signal) é um gatilho para ação; seja para comprar ou vender um título ou outro ativo, gerado pela análise. Essa análise pode ser gerada por humanos usando indicadores técnicos ou pode ser gerada usando [algoritmos matemáticos](/algorithm) baseados na ação do mercado, possivelmente em combinação com outros fatores de mercado, como indicadores econômicos.

Um número crescente de indicadores técnicos está disponível para os traders estudarem, incluindo aqueles de domínio público, como uma média móvel ou o [oscilador estocástico](/stochasticoscillator),. bem como indicadores proprietários disponíveis comercialmente. Padrões de gráficos como [cabeça e ombros](/head-shoulders),. [fundos arredondados](/roundingbottom),. [bandeiras](/flag) e [flâmulas](/pennant) também podem ser empregados. Além disso, muitos traders desenvolvem seus próprios indicadores exclusivos, às vezes com a ajuda de um programador qualificado.

> A maioria dos indicadores tem variáveis definidas pelo usuário que permitem que os traders adaptem as principais entradas, como o "período de retrospectiva" (quantos dados históricos serão usados para formar os cálculos) para atender às suas necessidades.

>

Não há limite para o quão complexo um sinal de negociação pode ser. No entanto, [os comerciantes](/trader) tendem a manter as coisas simples usando apenas um punhado de insumos. Para fins práticos, é muito mais fácil gerenciar um gerador de sinal simples e testá-lo periodicamente para ver quais componentes precisam ser ajustados ou substituídos.

Muitas entradas introduziriam complexidade, exigindo mais tempo do que um trader tem a oferecer. E como os mercados mudam com o tempo, muitas vezes com grande velocidade, estratégias complexas podem se tornar obsoletas antes mesmo de os testes serem concluídos.

Desenvolver uma abordagem de sinal para negociação ou investimento é o precursor do desenvolvimento de um sistema de negociação totalmente [algorítmico](/algorithmictrading) ou mesmo automatizado. [A negociação de alta frequência](/high-frequency-trading) (HFT) é um exemplo. Esses sistemas usam sinais gerados na escala de milissegundos para negociar dentro e fora de várias posições centenas a milhares de vezes por dia.

## Usando sistemas de sinais Forex

Os sistemas de sinais Forex podem criar negociações de compra ou venda que são manuais ou automatizadas. Um sistema manual envolve um trader sentado na tela do computador, procurando sinais e interpretando se deve comprar ou vender.

Um sistema de negociação automatizado envolve o trader "ensinando" ao software quais sinais procurar e como interpretá-los. Pensa-se que a negociação automatizada remove o elemento psicológico que é prejudicial para muitos comerciantes.

Os sinais podem vir de outros pares de moedas, preços de títulos, preços de [commodities](/commodity) e preços de ações. Por exemplo, um day trader pode ter um sinal de que, quando um determinado índice de ações é negociado em sua média móvel de 100 ou 200 dias, seu sinal diz que eles devem vender um determinado par de moedas.

Os sinais de Forex enviam suas informações através de uma variedade de plataformas, dependendo do software usado e das preferências do trader. Por exemplo, um trader pode receber os sinais por e-mail, WhatsApp, Skype e outros aplicativos semelhantes. Os melhores sistemas de sinais forex não apenas transmitem a um trader para comprar ou vender, mas também fornecem informações detalhadas sobre por que a ação é a correta no momento.

##Destaques

- Os sistemas de sinais Forex são baseados em análises técnicas, eventos baseados em notícias ou ambos, que incluem vários sinais diferentes para determinar uma decisão de compra ou venda.

- Os sistemas de sinais Forex podem criar negociações de compra ou venda que são manuais ou automatizadas. Os manuais envolvem um trader esperando por um sinal e agindo, enquanto os automatizados são configurados para que o sistema aja por conta própria.

- Um sistema de sinal forex é um conjunto de análises que os comerciantes forex usam para gerar sinais de negociação para determinar se deve comprar ou vender um par de moedas em um determinado momento.

- Há uma variedade de sistemas de sinais forex que podem ser usados gratuitamente, por uma taxa ou desenvolvidos pelo próprio trader.
