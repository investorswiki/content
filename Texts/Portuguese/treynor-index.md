# Índice Treynor
## O que é o Índice Treynor?

O Índice Treynor mede o desempenho ajustado ao risco de uma carteira de investimentos analisando o excesso de retorno de uma carteira por unidade de risco. No caso do Índice Treynor, o retorno excedente refere-se ao retorno obtido acima do retorno que poderia ter sido obtido em um investimento livre de risco. (Embora esta seja uma especulação teórica porque não existem investimentos verdadeiramente livres de risco.)

Para o Índice Treynor, a medida de risco de mercado usada é [beta](/beta),. que é uma medida de risco geral de mercado ou risco sistemático. Beta mede a tendência do retorno de um portfólio mudar em resposta a mudanças no retorno do mercado geral. Quanto maior o Índice Treynor, maior o retorno excedente gerado pela carteira por cada unidade de risco geral de mercado.

O Índice Treynor também é conhecido como [Índice de Treynor](/treynorratio) ou índice de recompensa para volatilidade.

## O que o Índice Treynor pode lhe dizer

O Índice Traynor indica quanto retorno um investimento, como uma carteira de ações, um fundo mútuo ou [um fundo negociado em bolsa](/etf),. ganhou para a quantidade de risco que o investimento assumiu. Um Índice Treynor mais alto significa que um portfólio é um investimento mais adequado. O índice é uma métrica de desempenho que expressa essencialmente quantas unidades de recompensa um investidor recebe para cada unidade de volatilidade que experimenta.

Assim como o [índice de Sharpe](/sharperatio) – que usa [o desvio padrão em](/standarddeviation) vez do beta como medida de risco – a premissa fundamental por trás do Índice Treynor é que o desempenho do investimento deve ser ajustado ao risco para transmitir uma imagem precisa do desempenho. O Índice Traynor foi desenvolvido pelo [economista](/economist) Jack Treynor, um economista americano que também foi um dos inventores do Capital Asset Pricing Model (CAPM).

Embora um Índice Treynor mais alto possa indicar um investimento adequado, é importante que os investidores tenham em mente que um índice não deve ser o único fator confiável para decisões de investimento. Mais importante ainda, como o Índice Treynor é baseado em dados históricos, as informações que ele fornece não indicam necessariamente o desempenho futuro.

## Exemplo do Índice Treynor

Por exemplo, suponha que o Portfolio Manager A alcance um [retorno](/portfolio-return) de portfólio de 8% em um determinado ano, quando a [taxa de retorno livre de risco](/risk-freerate) for de 5%; a carteira teve um beta de 1,5. No mesmo ano, o Portfolio Manager B obteve um retorno do portfólio de 7%, com um beta do portfólio de 0,8.

O Índice Treynor é, portanto, 2,0 para o Portfolio Manager A e 2,5 para o Portfolio Manager B. Enquanto o Portfolio Manager A excedeu o desempenho do Portfolio Manager B em um ponto percentual, o Portfolio Manager B realmente teve o melhor desempenho em uma base ajustada ao risco.

##Destaques

- Retorno excedente refere-se ao retorno obtido acima do retorno que poderia ter sido obtido em um investimento livre de risco.

- O Índice Treynor mede o desempenho ajustado ao risco de uma carteira de investimentos analisando o excesso de retorno de uma carteira por unidade de risco.

- Para o Índice Treynor, a medida de risco de mercado utilizada é beta, que é uma medida de risco geral de mercado ou risco sistemático.
