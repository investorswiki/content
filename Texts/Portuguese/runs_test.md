# Executa o teste
## O que é um teste de execuções?

Um teste de execuções é um procedimento estatístico que examina se uma [sequência](/data-analytics) [de dados](/data-analytics) está ocorrendo aleatoriamente de uma [distribuição específica](/normaldistribution). O teste de execuções analisa a ocorrência de eventos semelhantes que são separados por eventos diferentes.

Ao investir, um teste de execução pode ser importante para os investidores determinarem se o conjunto de dados que estão usando é gerado aleatoriamente ou se é impactado por uma variável subjacente. Os comerciantes que se concentram na análise técnica podem usar um teste de execução para ajudar a analisar a [ação do preço](/price-action) de um título.

## Entendendo um teste de execuções

Uma corrida é uma série de valores crescentes ou decrescentes, geralmente representados em um gráfico por símbolos de mais (+) ou menos (-). Nas estatísticas, um teste de execução ajuda a determinar a [aleatoriedade dos dados](/random-factor-analysis),. descobrindo quaisquer variáveis que possam afetar os padrões de dados.

Por exemplo, uma lista de números de um dígito verdadeiramente aleatórios deve ter apenas algumas instâncias em que uma sequência de números é crescente numericamente. No entanto, em muitos casos, é difícil afirmar a aleatoriedade dos dados em que existem milhares de sequências em uma sequência de dados. Assim, o teste de corridas foi criado como um [método objetivo](/objective-probability) de determinar a aleatoriedade.

## Tipos de Testes de Execuções

O teste de corridas é uma versão abreviada do nome completo: o teste de corridas Wald-Wolfowitz, assim chamado em homenagem aos matemáticos Abraham Wald e Jacob Wolfowitz. O teste Wald-Wolfowitz é um [teste estatístico não paramétrico](/nonparametric-statistics),. o que significa que os dados analisados não precisam atender a certas suposições ou parâmetros. O teste de Wald-Wolfowitz pode ser usado para examinar a [hipótese](/hypothesistesting) de que as variáveis na sequência de dados são mutuamente independentes.

Alguns estatísticos acreditam que outro tipo de teste de execuções - o teste de Kolmogorov-Smirnov - é uma ferramenta melhor do que o teste de Wald-Wolfowitz para detectar diferenças entre distribuições. O teste de Kolmogorov-Smirnov é um tipo de [teste de ajuste](/goodness-of-fit) que demonstra se os dados da amostra testados representam [padrões de distribuição normal](/normaldistribution) ou se os dados estão de alguma forma distorcidos. O teste estabelece a discrepância entre os valores dos dados amostrais e o modelo de distribuição normal.

## Benefícios de um teste de execuções

O modelo de teste de execuções é importante para determinar se um resultado de um ensaio é realmente aleatório, especialmente nos casos em que dados aleatórios versus sequenciais têm implicações para teorias e análises subsequentes. Um teste de execução pode ser uma ferramenta valiosa para investidores que empregam [análise técnica](/technicalanalysis) para tomar suas decisões de negociação. Esses traders analisam tendências estatísticas, como movimento e [volume de preços](/volume),. para identificar oportunidades de negociação potencialmente lucrativas. É importante que esses traders entendam as variáveis subjacentes que podem afetar o movimento dos preços e um teste de execução pode ajudar com isso.

Duas maneiras poderosas que os traders podem usar um teste de corridas incluem:

1. Testando a aleatoriedade da distribuição, pegando os dados na ordem dada e marcando com mais (+) os dados maiores que a [mediana](/median) e com menos (-) os dados menores que a mediana (os números iguais à mediana são omitido.)

1. Testar se uma função se ajusta bem a um conjunto de dados, marcando os dados que excedem o valor da função com + e os outros dados com −. Para este uso, o teste de corridas, que leva em consideração os sinais, mas não as distâncias, é complementar ao teste do [qui-quadrado](/chi-square-statistic),. que leva em consideração as distâncias, mas não os sinais.

## Destaques

- Um teste de execução é uma análise estatística que ajuda a determinar a aleatoriedade dos dados, revelando quaisquer variáveis que possam afetar os padrões de dados.

- Um teste de corrida, também conhecido como teste de corrida Wald–Wolfowitz, foi desenvolvido pelos matemáticos Abraham Wald e Jacob Wolfowitz.

- Os comerciantes técnicos podem usar um teste de execução para analisar tendências estatísticas e ajudar a identificar oportunidades comerciais lucrativas.

- Por exemplo, um investidor interessado em analisar o movimento do preço de uma determinada ação pode realizar um teste de execução para obter informações sobre uma possível ação futura do preço dessa ação.
