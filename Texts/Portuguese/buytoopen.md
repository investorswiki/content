# Comprar para abrir
## O que é comprar para abrir?

"Comprar para abrir" é um termo usado pelas corretoras para representar o estabelecimento de uma nova (abertura) de uma posição comprada ou [vendida](/putoption) em [opções](/calloption) [.](/option) Se um novo investidor de opções quiser comprar uma compra ou uma venda, esse investidor deve comprar para abrir. Uma ordem de compra para abrir indica aos participantes do mercado que o trader está estabelecendo uma nova posição em vez de fechar uma posição existente. A ordem [de venda para fechamento](/selltoclose) é usada para sair de uma posição tomada com uma ordem de compra para abrir.

O estabelecimento de uma nova posição curta é chamado [de venda para abertura](/selltoopen),. que seria encerrada com uma ordem [de compra para fechamento](/buytoclose). Se um novo investidor de opções quiser vender uma compra ou uma venda, esse investidor deve vender para abrir.

## Entendendo Comprar para Abrir Pedidos

A terminologia de compra e venda para negociação de opções não é tão direta quanto para negociação de ações. Em vez de simplesmente colocar uma ordem de compra ou venda como fariam para ações, [os operadores de opções](/trader) devem escolher entre "comprar para abrir", "comprar para fechar", "vender para abrir" e "vender para fechar".

Uma posição de compra para abrir pode indicar aos participantes do mercado que o trader que inicia o pedido acredita em algo sobre o mercado ou tem um machado para moer. Isso é particularmente verdadeiro se o pedido for grande. No entanto, isso não tem que ser o caso. Na verdade, os traders de opções frequentemente se envolvem em atividades de [spread](/spreadoption) ou [hedge](/hedge),. onde uma compra para abrir pode realmente compensar as posições existentes.

> Comprar para abrir uma aposta fora do dinheiro ao comprar uma ação é uma excelente maneira de limitar o risco.

>

A bolsa pode declarar que apenas ordens de fechamento podem ocorrer durante condições específicas de mercado, portanto, uma ordem de compra para abrir pode não ser executada. Isso pode acontecer se uma ação com opções disponíveis estiver agendada para [deslistagem](/delisting) ou a bolsa interromper a negociação da ação por um período prolongado.

## Considerações sobre ações

O termo "comprar para abrir" também pode ser aplicado a ações. Quando um investidor decide estabelecer uma nova posição em uma determinada ação, a primeira transação de compra é considerada comprar para abrir porque abre a posição.

Ao abrir a posição, a ação é estabelecida como uma participação na carteira. A posição permanece aberta até que seja fechada com a venda de todas as ações. Isso é conhecido como vender para fechar porque fecha a posição. Vender uma posição parcial significa que algumas ações, mas não todas, foram vendidas. Uma posição é considerada fechada quando não há mais de uma determinada ação em uma carteira.

Ordens de compra para fechar também entram em jogo ao cobrir uma posição [de venda a](/shortsale) descoberto. Uma posição de venda a descoberto empresta as ações através do corretor e é fechada comprando de volta as ações no mercado aberto. A última transação para fechar completamente a posição é conhecida como ordem de compra para fechamento. Esta transação remove completamente a exposição. A intenção é recomprar as ações a um preço mais baixo para gerar lucro com a diferença do preço de venda a descoberto e o preço de compra para fechar.

Nos casos em que o preço da ação sobe acentuadamente, um vendedor a descoberto pode ter que comprar para fechar com prejuízo para evitar que perdas ainda maiores ocorram. Na pior das hipóteses, o corretor pode executar uma liquidação forçada como resultado de uma [chamada de margem](/margincall). Então, o corretor exigiria que o investidor colocasse dinheiro na conta de margem devido a um déficit. Isso geraria uma ordem [de compra para cobrir](/buytocover) para fechar a posição com prejuízo devido ao patrimônio líquido insuficiente da conta.

## Compre para abrir vs. Compre para fechar

Se um investidor quiser comprar uma chamada ou uma venda para lucrar com um movimento de preço do título subjacente, esse investidor deve comprar para abrir. Comprar para abrir inicia uma posição longa de opções que dá ao especulador o potencial de obter um lucro extremamente grande com risco muito baixo. Por outro lado, o título deve se mover na direção certa dentro de um tempo limitado, ou a opção perderá todo o seu valor para [o decaimento do tempo](/timedecay).

Os vendedores de opções têm uma vantagem sobre os compradores devido à deterioração do tempo, mas eles ainda podem querer comprar para fechar suas posições. Quando um investidor vende opções, o investidor permanece obrigado pelos termos dessas opções até a data de vencimento. No entanto, os movimentos no preço do título podem permitir que os vendedores de opções obtenham a maior parte de seus lucros muito mais cedo ou motivá-los a reduzir as perdas.

Suponha que alguém venda [no money](/atthemoney) put com duração de um ano e, em seguida, a ação subjacente suba 10% após três meses. O vendedor de opções pode comprar para fechar e obter a maior parte dos lucros imediatamente. Se a ação cair 10% após três meses, o vendedor de opções terá que pagar mais para comprar para fechar e limitar as perdas potenciais.

## Exemplo de Comprar para Abrir

Suponha que um trader tenha feito alguma análise e acredite que o preço da ação XYZ passará de $ 40 para $ 60 no próximo ano. O trader pode comprar para abrir uma chamada para XYZ. O [preço de exercício](/strikeprice) pode ser de US$ 50 com uma [data de vencimento](/expirationdate) daqui a cerca de um ano.

## Destaques

- Uma posição de compra para abrir em opções cria a oportunidade de grandes ganhos com perdas mínimas, mas tem um alto risco de expirar sem valor.

- Uma ordem de compra para abrir é geralmente usada por traders para abrir posições em uma determinada opção ou ação.

- Comprar para abrir uma posição de opções pode compensar ou cobrir outros riscos em um portfólio.
