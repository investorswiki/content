# Número de conta bancária internacional (IBAN)
## O que é um número de conta bancária internacional (IBAN)?

Um IBAN, ou número de conta bancária internacional, é um sistema de numeração internacional padrão desenvolvido para identificar uma conta bancária no exterior. O número começa com um código de país de dois dígitos, depois dois números, seguidos por vários outros caracteres alfanuméricos. Observe que um IBAN não substitui a numeração de conta do próprio banco, pois serve apenas para fornecer informações adicionais que ajudam na identificação de pagamentos no exterior.

## Como funcionam os números de contas bancárias internacionais (IBANs)

O número IBAN consiste em um código de país de duas letras, seguido por dois dígitos de verificação e até trinta e cinco caracteres alfanuméricos. Esses caracteres alfanuméricos são conhecidos como o número básico da conta bancária (BBAN). Cabe à associação bancária de cada país determinar qual BBAN selecionará como padrão para as contas bancárias daquele país.

Um número IBAN será usado ao enviar transferências interbancárias ou transferir [dinheiro de um banco](/wiretransfer) para outro, especialmente através de fronteiras internacionais. No cadastro de países que atualmente utilizam o sistema IBAN, vários exemplos são os seguintes:

- Albânia: AL47 2121 1009 0000 0002 3569 8741

- Chipre: CY 17 002 00128 0000001200527600

- Kuwait: KW81CBKU0000000000001234560101

- Luxemburgo: LU 28 001 9400644750000

- Noruega: NO 93 8601 1117947

> Os EUA e o Canadá são os dois principais países que não utilizam o sistema IBAN; no entanto, eles reconhecem o sistema e processam os pagamentos de acordo com o sistema.

>

## Códigos IBAN vs. SWIFT

Existem dois métodos padronizados e reconhecidos internacionalmente de identificação de contas bancárias quando uma transferência está sendo feita de um país para outro: o número de conta bancária internacional (IBAN) e o código da [Sociedade para Telecomunicações Financeiras Interbancárias Mundiais](/swift) (SWIFT). A diferença entre os dois métodos está no que eles identificam.

Um código SWIFT é usado para identificar um banco específico durante uma transação internacional, enquanto o IBAN é usado para identificar uma conta individual envolvida na transação internacional. Ambos desempenham um papel essencial no bom funcionamento do mercado financeiro internacional.

O sistema SWIFT é anterior às tentativas de padronizar as transações bancárias internacionais por meio do IBAN. Continua sendo o método pelo qual a maioria das transferências internacionais de fundos são feitas. Uma das principais razões para isso é que o sistema de mensagens SWIFT permite que os bancos compartilhem uma quantidade significativa de dados financeiros.

Esses dados incluem o status da conta, valores de débito e crédito e detalhes relacionados à transferência de dinheiro. Os bancos geralmente usam o código de identificação do banco (BIC) em vez do código SWIFT. No entanto, os dois são facilmente intercambiáveis; ambos contêm uma mistura de letras e números e geralmente têm entre oito e 11 caracteres.

## Requisitos para números de contas bancárias internacionais

O IBAN foi desenvolvido a partir de padrões nacionais divergentes para identificação de contas bancárias. Os usos variados de formulários alfanuméricos para representar bancos, agências, códigos de roteamento e números de contas específicos geralmente levavam a interpretações errôneas e/ou omissões de informações críticas de pagamentos.

Para facilitar esse processo, a Organização Internacional de Padronização (ISO) publicou a ISO 13616:1997 em 1997. Pouco depois, o Comitê Europeu de Padrões Bancários (ECBS) publicou uma versão menor, acreditando que a flexibilidade original permitida na versão ISO era impraticável. Na versão do ECBS, eles permitiam apenas letras maiúsculas e um IBAN de tamanho fixo para cada país.

Desde 1997, uma nova versão, a ISO 13616:2003, substituiu a versão inicial do ECBS. Uma versão posterior em 2007 estipulou que os elementos IBAN devem facilitar o processamento de dados internacionalmente, tanto em ambientes financeiros como entre outros setores; no entanto, não especifica quaisquer procedimentos internos, incluindo, mas não se limitando a técnicas de organização de arquivos, mídia de armazenamento ou idiomas.

## Destaques

- Um número de conta bancária internacional (IBAN) é um sistema de numeração internacional padrão para contas bancárias individuais em todo o mundo.

- Bancos na Europa desenvolveram originalmente o sistema para simplificar transações envolvendo contas bancárias de outros países.

- Um IBAN é usado para identificar uma conta individual envolvida em uma transação internacional.

- O IBAN também atua como um método para verificar se os detalhes da transação estão corretos.

## PERGUNTAS FREQUENTES

### Por que o IBAN foi criado?

O IBAN foi desenvolvido para reduzir erros e melhorar a verificação de pagamentos internacionais, reduzindo pagamentos rejeitados, atrasos nas transferências e encargos bancários e taxas associadas.

### Quem usa um IBAN?

O IBAN foi criado para facilitar pagamentos eletrônicos entre bancos na zona do euro. Desde então, ele se expandiu em todo o mundo, embora nem todos os bancos e nem todas as regiões tenham aderido ao padrão e você ainda precise confiar em um sistema alternativo como o SWIFT. Os países norte-americanos, australianos e asiáticos não usam o IBAN para transferências domésticas de dinheiro e só o farão ao enviar um pagamento para um país que adotou o IBAN.

### Como é um número IBAN?

Um número IBAN contém até 34 caracteres alfanuméricos. Ele é precedido por um código de país de dois caracteres, dois dígitos de verificação e um número básico de conta bancária (BBAN) que contém detalhes específicos do banco e da conta. O formato da parte BBAN varia de país para país, que normalmente inclui um código de banco e código de agência.

### Como posso obter um IBAN?

Você pode solicitar um IBAN se for cliente de um banco em uma região IBAN. Observe que um IBAN só pode ser usado para receber pagamentos e não é usado para saques.
