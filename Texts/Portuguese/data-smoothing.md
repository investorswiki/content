# Suavização de dados
## O que é suavização de dados?

A suavização de dados é feita usando um algoritmo para remover o [ruído](/noise) de um conjunto de dados. Isso permite que padrões importantes se destaquem mais claramente.

A suavização de dados pode ser usada para ajudar a prever tendências, como as encontradas nos preços dos títulos, bem como na análise econômica. A suavização de dados destina-se a ignorar valores discrepantes únicos e levar em conta os efeitos da [sazonalidade](/seasonal-adjustment).

## Entendendo a suavização de dados

Quando os dados são compilados, eles podem ser manipulados para remover ou reduzir qualquer [volatilidade](/volatility) ou qualquer outro tipo de ruído. Isso é chamado de suavização de dados.

A ideia por trás da suavização de dados é que ela pode identificar mudanças simplificadas para ajudar a prever diferentes tendências e padrões. Ele atua como uma ajuda para estatísticos ou comerciantes que precisam analisar muitos dados - que muitas vezes podem ser complicados de digerir - para encontrar padrões que não veriam de outra forma.

Para explicar com uma representação visual, imagine um gráfico de um ano para as ações da Empresa X. Cada ponto alto individual no gráfico para a ação pode ser reduzido enquanto aumenta todos os pontos mais baixos. Isso faria uma curva mais suave, ajudando um investidor a fazer previsões sobre como a ação pode se comportar no futuro.

> Os dados suavizados são geralmente preferidos pelos economistas porque identificam melhor as mudanças nas tendências em comparação com os dados não suavizados, que podem parecer mais erráticos e criar sinais falsos.

>

## Considerações Especiais

### Métodos para suavização de dados

Existem diferentes métodos nos quais a suavização de dados pode ser feita. Algumas delas incluem o método de randomização, usando um [passeio aleatório](/random-walk-index),. calculando uma [média móvel](/movingaverage) ou conduzindo uma das várias técnicas de suavização exponencial.

> Uma [média móvel simples](/sma) (SMA) coloca peso igual nos preços recentes e históricos, enquanto uma [média móvel exponencial](/ema) (EMA) coloca mais peso nos dados de preços recentes.

>

O modelo de passeio aleatório é comumente usado para descrever o comportamento de instrumentos financeiros, como ações. Alguns investidores acreditam que não há relação entre o movimento passado no preço de um título e seu movimento futuro. A suavização de caminhada aleatória assume que os pontos de dados futuros serão iguais ao último ponto de dados disponível, mais uma variável aleatória. Analistas técnicos e fundamentais discordam dessa ideia; eles acreditam que movimentos futuros podem ser extrapolados examinando tendências passadas.

Frequentemente usada em [análise técnica](/technicalanalysis),. a média móvel suaviza a ação do preço enquanto filtra a volatilidade dos movimentos aleatórios dos preços. Esse processo é baseado em preços passados, tornando-se um indicador de acompanhamento de tendências - ou de atraso. Como pode ser visto no gráfico de preços abaixo, a média móvel (EMA) tem a forma geral e a tendência dos dados de preços diários subjacentes, representados pelas velas. Quanto mais dias incorporados à média móvel, mais suavizada a linha se torna.

## Vantagens e desvantagens da suavização de dados

A suavização de dados pode ser usada para ajudar a identificar tendências na economia, em títulos, como ações e sentimento do consumidor. A suavização de dados também pode ser usada para outros fins comerciais.

Por exemplo, um economista pode suavizar os dados para fazer [ajustes sazonais](/seasonal-adjustment) para determinados indicadores, como vendas no varejo, reduzindo as variações que podem ocorrer a cada mês, como feriados ou preços do gás.

Há quedas para usar esta ferramenta, no entanto. A suavização de dados nem sempre fornece uma explicação das tendências ou padrões que ajuda a identificar. Também pode levar a que certos pontos de dados sejam ignorados, enfatizando outros.

## Exemplo de suavização de dados na contabilidade financeira

Um exemplo frequentemente citado de suavização de dados na contabilidade empresarial é fazer uma [provisão para contas duvidosas](/allowancefordoubtfulaccounts) alterando [as despesas com dívidas incobráveis](/bad-debt-expense) de um período de relatório para outro. Por exemplo, uma empresa espera não receber o pagamento de certas mercadorias em dois períodos contábeis; US$ 1.000 no primeiro período do relatório e US$ 5.000 no segundo período do relatório.

Se se espera que o primeiro período de relatório tenha uma receita alta, a empresa pode incluir o valor total de $ 6.000 como provisão para devedores duvidosos nesse período de relatório. Isso aumentaria a despesa com dívidas incobráveis na demonstração de resultados em $ 6.000 e reduziria o lucro líquido em $ 6.000. Isso suavizaria um período de alta renda, reduzindo a renda. É importante que as empresas usem métodos de julgamento e contabilidade legal ao ajustar quaisquer contas.

## Destaques

- A suavização de dados usa um algoritmo para remover o ruído de um conjunto de dados, permitindo que padrões importantes se destaquem.

- Embora a suavização de dados possa ajudar a prever certas tendências, ela resultará inerentemente em menos informações na amostra, o que pode levar a que determinados pontos de dados sejam ignorados.

- Diferentes modelos de suavização de dados incluem o método aleatório o uso de médias móveis.

- A suavização de dados pode ser usada para prever tendências, como as encontradas nos preços dos títulos.
