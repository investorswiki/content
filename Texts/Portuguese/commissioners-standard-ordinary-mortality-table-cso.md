# Tábua de Mortalidade Ordinária Padrão dos Comissários
## O que é a Tábua de Mortalidade Ordinária Padrão dos Comissários?

[tábua de mortalidade](/mortality-table) ordinária padrão do comissário é uma tábua atuarial utilizada para calcular os valores mínimos de [não cassação das apólices de](/nonforfeiture-clause) [seguro de vida](/lifeinsurance) ordinária . A tabela de mortalidade ordinária padrão dos comissários (CSO) reflete a probabilidade de que pessoas em várias faixas etárias morram em um determinado ano.

## Entendendo a Tabela de Mortalidade Ordinária Padrão dos Comissários

As tabelas de mortalidade ordinária padrão dos comissários (CSO) são divididas por homens versus mulheres, bem como fumantes versus não fumantes. A última tabela de mortalidade da OSC foi concluída em 2017. Para todas as apólices emitidas a partir de 1º de janeiro de 2020, as seguradoras de vida devem agora usar as tabelas atualizadas de 2017. A atualização de 2017 foi a primeira atualização das tabelas desde 2001. A idade terminal de 121 ainda é usado na atualização de 2017, como foi para 2001 .

A atualização de 2017 foi emitida como resultado de mais [dados disponíveis](/data-science),. incluindo o fato de que as pessoas agora estão vivendo mais. A grande diferença do CSO de 2001 é que a atualização de 2017 tem mais que o dobro dos dados de empresas que fornecem dados de experiência, além de mais informações sobre fumantes versus não fumantes. A quantidade de dados de exposição de seguros usados na criação do CSO de 2017 é de US$ 30,7 trilhões de 51 seguradoras, bem acima dos US$ 5,7 trilhões de 21 seguradoras em 2001 .

A atualização da tabela de mortalidade das OSCs foi examinada com várias métricas em mente. Por exemplo, as modificações dos números de 2001 para os números de 2017 foram inspecionadas para determinar a suavidade da tabela. [Os valores de reserva](/valuation-reserve) foram calculados e examinados para relacionamentos apropriados. [As reservas estatutárias](/statutory-reserves) produzidas pela tabela foram comparadas às reservas de cheques para garantir que a tabela proposta forneceria reservas estatutárias suficientes para a maioria das empresas. As provisões de deficiência não foram consideradas porque as premissas de prêmio bruto não estavam disponíveis para servir de base para essa estimativa, o que tornou uma estimativa mais conservadora.

## Como a Tabela de Mortalidade Ordinária Padrão dos Comissários é Usada

A tabela de mortalidade ordinária padrão dos comissários (CSO) é usada para calcular os requisitos de reserva para uma determinada seguradora. É a tabela legalmente exigida para cálculo de reservas obrigatórias e valores de não confisco para companhias de seguros de vida. Em outras palavras, as seguradoras de vida devem olhar para a idade de seus segurados e então calcular quanto dinheiro eles devem manter em reservas para pagar os benefícios futuros da apólice, usando as taxas de mortalidade da OSC. Isso também significa que o CSO é a base para determinar os valores em dinheiro garantidos e outros benefícios de não confisco. Estes são os montantes disponíveis para os tomadores de seguros que [renunciem](/cashsurrendervalue) aos seus contratos de seguro de vida.

A expectativa de vida é a estatística que mais importa para as seguradoras. Existem milhares de [subscritores de seguro de vida](/underwriter) trabalhando em todo o país, tentando adivinhar com precisão a expectativa de vida de alguém, e as condições de saúde e o histórico familiar são usados para ajustar uma taxa cotada para cima ou para baixo. Cada companhia de seguros de vida tem suas próprias tabelas sofisticadas de expectativa de vida, que são usadas para escrever apólices. As empresas consideram o CSO como fator ou base para esses cálculos.

## Tábua de Mortalidade Ordinária Padrão dos Comissários vs. Tábua de Mortalidade Industrial

As tabelas de mortalidade ordinárias padrão (CSO) dos comissários contrastam com as tabelas de mortalidade industrial para apólices de seguro de vida industrial. As apólices de seguro de vida industrial exigem prêmios mais baixos dos proprietários de apólices do que as apólices de CSO porque o valor nominal do seguro de vida industrial é menor, geralmente não superior a US$ 10.000. Além disso, os prêmios de seguro de vida industrial geralmente são pagos [semanalmente](/weekly-premium-insurance),. em vez de mensalmente. Ambos os tipos de tabelas devem ser aprovados pela Associação Nacional de Comissários de Seguros (NAIC).

## Destaques

- A atualização da tabela de mortalidade das OSCs de 2017 foi resultado de mais dados de experiência disponíveis e inclui maior expectativa de vida em geral.

- A tábua de mortalidade das OSCs foi atualizada em 2017 pela primeira vez desde 2001.

- A tábua de mortalidade normal padrão do comissário (CSO), usada pelas seguradoras de vida, estabelece a probabilidade de que pessoas de uma certa idade venham a morrer em um determinado ano.

- A tábua de mortalidade CSO é usada para calcular os requerimentos de reserva para uma determinada seguradora.
