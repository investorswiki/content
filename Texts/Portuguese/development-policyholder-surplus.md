# Desenvolvimento para o Excedente do Segurado
## O que é o Excedente do Desenvolvimento para o Segurado?

O excedente de desenvolvimento para o segurado é a razão entre o desenvolvimento da reserva para perdas de uma [seguradora e o](/insurance) [excedente do segurado](/policyholder-surplus). O índice de superávit de desenvolvimento para segurado mostra se uma empresa está reservando uma quantidade adequada de fundos como reservas para perdas.

O índice também é um indicador para saber se o superávit do segurado (o patrimônio líquido de uma seguradora) está superestimado ou subestimado.

## Compreendendo o Excedente do Desenvolvimento para o Segurado

O excedente do segurado é a diferença entre os ativos de uma seguradora menos seus [passivos](/liability). O excedente do segurado ajuda a medir a [saúde financeira](/financial-health) de uma seguradora. As seguradoras fazem reservas para o caso de precisarem pagar [sinistros](/insurance_claim) a seus clientes ou segurados.

Uma reclamação de seguro é uma solicitação de um segurado para ser compensado por uma perda financeira de um evento coberto dentro da apólice de seguro. O excedente do segurado é considerado um buffer adicional de capital ou dinheiro que pode ser usado para pagar sinistros se não houver reservas suficientes.

O excedente de desenvolvimento para o segurado ajuda a determinar se uma seguradora tem uma quantidade excessiva de reservas ou se a empresa tem reservas inadequadas. O índice de superávit de desenvolvimento para segurado é frequentemente calculado em vários períodos de tempo para ver se uma seguradora está superestimando ou subestimando consistentemente suas reservas.

Se o índice de superávit de desenvolvimento para segurado estiver aumentando de ano para ano, pode ser uma indicação de que a seguradora está fortalecendo intencionalmente suas reservas para perdas (exagerar), enquanto uma diminuição no índice pode indicar que suas reservas estão sendo subavaliadas.

O excedente de um segurado refere-se ao restante dos ativos de uma seguradora, após dedução de todos os seus passivos para poder proporcionar os benefícios esperados aos segurados. É o patrimônio líquido da seguradora, conforme demonstrado em suas demonstrações financeiras. O excedente também é considerado um apoio financeiro que protege os segurados contra situações inesperadas. Algumas empresas incluem as seguintes contas no superávit de seus segurados:

- [Interesses minoritários](/minorityinterest),. que são participações ou participação inferior a 50% em outra empresa

- [Patrimônio líquido](/stockholdersequity) composto por ações ordinárias, outros resultados abrangentes, [capital](/additionalpaidincapital) [integralizado adicional](/additionalpaidincapital) (ou dinheiro pago de investidores por ações) e [lucros acumulados](/retainedearnings) (ou lucro acumulado); no entanto, o patrimônio não deve incluir participações minoritárias

- Um substituto de capital, especificamente capital híbrido, que pode ter elementos de ações e títulos

## Benefícios do Desenvolvimento para o Excedente do Segurado

Os reguladores ficam de olho nas seguradoras para garantir que elas não corram o risco de se tornarem [insolventes](/insolvency),. e um dos métodos que usam para monitorar um grande número de seguradoras é a revisão dos índices financeiros. As seguradoras têm, assim, um incentivo para garantir que seus índices não sejam considerados incomuns e, assim, gerenciarão suas perdas de reservas de forma a não chamar atenção.

Subestimar as reservas para perdas resultará em mais receita do superávit dos segurados, mas menos receita da reserva. A gestão da reserva para perdas de uma seguradora ajuda a empresa a suavizar sua receita e chama menos atenção dos reguladores. Os erros de reserva de perdas (exagerar e subestimar) estão correlacionados com as atividades de receita da seguradora. As seguradoras envolvidas em atividades de investimento mais arriscadas são mais propensas a relatar mais erros de reserva de perdas.

Analisar uma companhia de seguros envolve revisar seus índices financeiros para determinar como os índices mudaram ao longo do tempo, bem como como os índices se comparam a companhias de seguros semelhantes. Se o índice de superávit de desenvolvimento para segurado de uma empresa for baixo, uma análise mais aprofundada deve se concentrar em quais linhas de negócios são as mais problemáticas. O índice pode ser recalculado para cada linha de negócios.

##Destaques

- O excedente de desenvolvimento para o segurado é a razão entre o desenvolvimento da reserva para perdas de uma seguradora e o excedente do segurado.

- O excedente do segurado é a diferença entre os ativos de uma seguradora menos seus passivos.

- O índice de superávit do desenvolvimento para o segurado mostra se uma seguradora está reservando uma quantia apropriada de fundos como reservas para perdas.
