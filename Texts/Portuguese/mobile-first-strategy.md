# Estratégia Mobile First
## O que é uma estratégia Mobile-First?

A expressão "estratégia mobile-first" refere-se a uma abordagem de desenvolvimento de sites em que a versão do site para dispositivos móveis tem prioridade sobre a versão para desktop.

As empresas estão priorizando cada vez mais as versões móveis de suas páginas da web, devido à crescente popularidade da navegação na Internet por telefone e das compras on-line entre os consumidores.

## Como funcionam as estratégias Mobile-First

Ao buscar uma estratégia mobile-first, uma empresa lançará primeiro uma versão habilitada para dispositivos móveis de seu site, antes de investir tempo ou recursos substanciais no desenvolvimento de uma versão para desktop. Muitas vezes, alguns sites de desktop rudimentares serão lançados juntamente com a versão móvel, mas esse site de desktop pode consistir em pouco mais do que uma página de destino com links para um aplicativo móvel ou para informações básicas, como uma página de perguntas frequentes (FAQ).

Existem vários fatores subjacentes que impulsionam a popularidade das estratégias mobile-first. Para começar, a porcentagem de tráfego da web atribuível a [smartphones](/smartphone) em comparação com computadores de mesa cresceu de forma constante ao longo do tempo. De acordo com o StatCounter, a navegação móvel era essencialmente igual à navegação em desktop, com cada uma com 48% de participação em janeiro de 2021. A Perficient estimou que 61% de todo o tráfego da Web nos EUA se originou de dispositivos móveis em 2020.

Outro fator importante que incentiva as empresas a buscar uma estratégia mobile-first diz respeito aos dados e recursos adicionais que os sites móveis podem fornecer. Como os smartphones agora normalmente utilizam interfaces de tela sensível ao toque, é possível que os proprietários de sites habilitados para dispositivos móveis obtenham dados detalhados da experiência do usuário (UX) sobre exatamente como o usuário é clicado fisicamente ou rolado pela página. Em alguns casos, os proprietários de sites podem até obter dados sobre os movimentos dos olhos do usuário, conforme determinado usando a câmera frontal do telefone.

Embora essas métricas avançadas de dados do usuário sejam relativamente raras, outros tipos de dados avançados do usuário são bastante comuns. Um exemplo é o rastreamento de localização, no qual as coordenadas GPS dos telefones dos usuários são passadas automaticamente para o provedor do site. Esse tipo de informação pode ser fundamental para informar futuras [campanhas de marketing](/marketing-strategy) ou planejar esforços de desenvolvimento de negócios, como a abertura de novas lojas de varejo ou armazéns de distribuição.

## Exemplo do mundo real de uma estratégia Mobile-First

No entanto, talvez o fator mais significativo que impulsiona a adoção de estratégias mobile-first seja a estratégia de "indexação mobile-first" do Google. Essa decisão, que foi colocada em prática em julho de 2019, deixou claro que, daqui para frente, o algoritmo de busca da empresa daria prioridade às versões mobile dos sites ao indexar inicialmente um novo site.

Na prática, isso significa que novos sites desenvolvidos sem uma estratégia mobile-first provavelmente terão resultados de otimização de mecanismo de busca (SEO) abaixo da média. Considerando a enorme influência que o Google desfruta no ecossistema global de mecanismos de busca, parece provável que as estratégias mobile-first sejam adotadas como uma prática recomendada para a maioria, senão todos os novos projetos de desenvolvimento de sites daqui para frente.

##Destaques

- No futuro, as estratégias mobile-first provavelmente se tornarão a norma, em parte devido à decisão dos principais mecanismos de pesquisa de priorizar as versões móveis dos sites.

- Uma estratégia mobile-first é aquela em que a versão mobile de um site tem prioridade sobre sua versão desktop.

- Essa prática era relativamente rara no passado, mas tem se tornado cada vez mais comum.
