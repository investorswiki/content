# Gráfico de barras
## O que é um gráfico de barras?

Um gráfico de barras é um gráfico que plota dados usando barras ou colunas retangulares (chamadas caixas) que representam a quantidade total de observações nos dados para essa categoria. Os gráficos de barras podem ser exibidos com colunas verticais, barras horizontais, barras comparativas (várias barras para mostrar uma comparação entre valores) ou barras empilhadas (barras contendo vários tipos de informações).

Os gráficos de barras são comumente usados na análise financeira para exibir dados. Um gráfico de [volume de ações](/volume) é um tipo comumente usado de gráfico de barras verticais. Um [histograma](/histogram) é um exemplo de gráfico de barras usado em análise estatística que descreve uma distribuição de probabilidade em alguns dados ou amostra.

## Entendendo um gráfico de barras

O objetivo de um gráfico de barras é transmitir informações relacionais rapidamente, pois as barras exibem a quantidade de uma categoria específica. O eixo vertical do gráfico de barras é chamado de eixo y, enquanto a parte inferior de um gráfico de barras é chamada de eixo x.

Ao interpretar um gráfico de barras, o comprimento das barras/colunas determina o valor conforme descrito no eixo y.

O eixo x pode ser qualquer variável, como tempo, ou a categoria que está sendo medida, como [lucro por ação](/eps) (EPS), receita e/ou [fluxo de caixa](/cashflow). Na negociação, os gráficos de barras são frequentemente usados para representar o volume de negociação e aparecerão em um painel abaixo do gráfico de preços de um título.

## Recursos do gráfico de barras

Um gráfico de barras típico tem um rótulo ou título, eixo x, eixo y, escalas ou incrementos para o eixo e barras. Alguns gráficos também podem ter uma legenda que especifica o que as várias cores representam, como em um gráfico de barras empilhado.

Os gráficos de barras são ideais para comparar dois ou mais valores, ou valores ao longo do tempo. Os dados são exibidos horizontalmente ou verticalmente. Gráficos de barra simples são usados para transmitir valores discretos de um item dentro de uma categoria. Por exemplo, um gráfico de barras pode exibir o número de homens com uma determinada característica para idades específicas. O valor discreto, ou o número de instâncias em que um indivíduo tem uma determinada característica, é exibido variando o comprimento da barra. Mais instâncias significam uma barra mais longa e menos instâncias significam uma barra mais curta. Neste exemplo, uma barra diferente é estabelecida para cada idade ou faixa etária.

Na [análise técnica](/technicalanalysis),. um gráfico de volume mostra quanto volume havia em cada dia específico. O eixo x mostra os dias, enquanto uma barra que se estende a partir desse dia mostra quanto volume havia no eixo y.

Quando um gráfico possui um ponto zero bem definido e o conjunto de dados possui valores positivos e negativos em relação a este ponto, as barras ainda podem ser exibidas. As barras acima da linha zero normalmente representam valores positivos (verifique a escala), enquanto as barras abaixo da linha zero normalmente apresentam valores negativos.

Os dados podem ser exibidos horizontalmente ou verticalmente. Para alternar a orientação, os eixos x e y são alternados.

## Tipos de gráficos de barras

### Gráficos de barras agrupados

Os gráficos de barras agrupados, também chamados de gráficos de barras agrupados, representam valores discretos para mais de um item que compartilham a mesma categoria.

Um gráfico de barras poderia exibir o número de indivíduos, masculinos e femininos, com uma determinada característica para idades específicas. O número agregado de instâncias pode ser combinado em uma barra. alternativamente, as instâncias poderiam permanecer segregadas por gênero; uma barra para todas as instâncias masculinas e uma barra para todas as instâncias femininas seriam colocadas lado a lado para cada idade ou faixa etária.

### Gráficos de barras empilhadas

Gráficos de barras empilhados ou gráficos de barras compostos dividem um total agregado em partes. Essas peças são normalmente identificadas utilizando cores diferentes para cada seção. No exemplo acima, o agregado de instâncias para machos e fêmeas pode ser combinado em uma barra, mas a barra pode ser dividida em várias seções representadas por cores diferentes.

Barras empilhadas requerem uma legenda ou rotulagem específica para identificar o que as várias partes da barra estão mostrando.

## Gráficos de Barras em Análise Técnica

Algumas formas de análise técnica utilizam gráficos de barras. Por exemplo, os comerciantes podem empregar um histograma de [divergência de convergência média móvel](/macd) (MACD), que é um indicador técnico popular que ilustra a diferença entre a linha MACD e a [linha de sinal](/signal_line).

## Gráfico de barras vs. gráfico de barras

Um gráfico de barras mostra os dados em colunas, enquanto um [gráfico de barras](/barchart) é um termo de análise técnica que descreve a exibição dos preços de abertura, alta, baixa, fechamento (às vezes a abertura é omitida) para um determinado [título](/security) durante um período de tempo específico usando uma barra vertical . Pequenas linhas horizontais se estendem à esquerda e à direita da barra vertical para mostrar os preços de abertura e fechamento.

Ao contrário do gráfico de barras, a barra de preços cobre apenas os preços relevantes e não se estende até o eixo x.

## Limitações do gráfico de barras

Um gráfico de barras é uma maneira de exibir informações. A forma como os dados são escolhidos para serem exibidos pode afetar sua interpretação. Por exemplo, se for escolhida uma escala muito grande, os dados podem parecer insignificantes – quando na verdade podem ser muito significativos, mas a escala não permite uma comparação apropriada.

Os gráficos de barras também podem fazer com que os dados pareçam atraentes quando, na verdade, podem estar faltando substância. Tal como acontece com todos os dados, verifique a fonte de onde vem e certifique-se de que é de um pool ou amostra grande o suficiente.

Por exemplo, observar apenas alguns dias de dados de volume em uma ação não fornece muitas informações relevantes. No entanto, observar como o volume recente se compara ao volume do ano passado fornecerá a um trader técnico mais informações para analisar o volume.

##Destaques

- Os dados são apresentados através de colunas verticais ou horizontais.

- Os gráficos de barras têm eixos x e y e podem ser usados para mostrar uma, duas ou muitas categorias de dados.

- As colunas podem conter múltiplas variáveis rotuladas (ou apenas uma), ou podem ser agrupadas (ou não) para fins comparativos.

- Gráficos de barras podem ser criados para mostrar dados de várias maneiras altamente visuais.
