# Provisão para Perdas de Empréstimos
## O que é uma provisão para perdas com empréstimos?

Uma provisão para perdas com empréstimos é uma despesa da [demonstração do resultado](/incomestatement) reservada como uma provisão para empréstimos não cobrados e pagamentos de empréstimos. Essa provisão é usada para cobrir diferentes tipos de perdas com empréstimos, como [empréstimos inadimplentes](/nonperformingloan),. falência de clientes e [empréstimos renegociados](/renegotiated-loan) que incorrem em pagamentos abaixo do estimado anteriormente. As provisões para perdas com empréstimos são então adicionadas às reservas para perdas com empréstimos, um item do [balanço patrimonial](/balancesheet) que representa o valor total das perdas com empréstimos subtraídas dos empréstimos de uma empresa .

## Como funciona uma provisão para perdas com empréstimos

Os credores do setor bancário geram receita a partir dos juros e despesas que recebem dos produtos de empréstimo. Os bancos emprestam para uma ampla gama de clientes, incluindo consumidores, pequenas empresas e grandes corporações.

Os padrões de empréstimo e os requisitos de relatórios estão mudando constantemente, e as restrições foram rigorosamente apertadas desde o auge da crise financeira de 2008. Regulações aprimoradas para bancos resultantes da [Lei Dodd-Frank](/dodd-frank-financial-regulatory-reform-bill) focadas no aumento dos padrões de empréstimos, que exigiram mutuários de maior qualidade de crédito e também aumentaram os requisitos de [liquidez de capital](/liquidity) para o banco .

Apesar dessas melhorias, os bancos ainda precisam contabilizar [inadimplências](/default2) e despesas que ocorrem como resultado de empréstimos. As provisões para perdas com empréstimos são um ajuste contábil padrão feito nas reservas para perdas com empréstimos de um banco incluídas nas demonstrações financeiras dos bancos. As provisões para perdas com empréstimos são consistentemente feitas para incorporar as mudanças nas projeções para perdas dos produtos de crédito do banco. Embora os padrões para empréstimos tenham melhorado muito, os bancos ainda enfrentam atrasos no pagamento de empréstimos e inadimplência .

> Como a provisão para perdas com empréstimos aparece na demonstração do resultado como uma despesa, ela reduzirá os lucros operacionais.

>

## Reservas para perdas com empréstimos na contabilidade

As reservas para perdas com empréstimos são normalmente contabilizadas no [balanço patrimonial de um banco](/balancesheet),. que pode aumentar pelo valor da provisão para perdas com empréstimos ou diminuir pelo valor das [baixas líquidas a](/chargeoff) cada trimestre.

Provisões para créditos de liquidação duvidosa são feitas constantemente para atualizar estimativas e cálculos baseados em estatísticas de inadimplência de clientes do banco. Essas estimativas são calculadas com base nas taxas médias históricas de inadimplência por diferentes níveis de tomadores. As perdas de crédito por atrasos de pagamento e despesas de cobrança também são incluídas nas estimativas de provisão para devedores duvidosos e são calculadas usando uma metodologia semelhante, que leva em consideração as estatísticas de pagamento anteriores dos clientes de crédito de um banco.

Em geral, ao reservar reservas para perdas com empréstimos e atualizar constantemente as estimativas por meio de provisões para perdas com empréstimos, os bancos podem garantir que estão apresentando uma avaliação precisa de sua posição financeira geral. Esta posição financeira é frequentemente divulgada publicamente através das demonstrações financeiras trimestrais do banco.

## Destaques

- As provisões para perdas com empréstimos são adicionadas às reservas para perdas com empréstimos, um item do balanço que mostra as perdas totais com empréstimos.

- Uma provisão para perdas com empréstimos é uma despesa da demonstração do resultado reservada para permitir empréstimos não cobrados e pagamentos de empréstimos.

- Os bancos são obrigados a contabilizar possíveis inadimplências e despesas de empréstimos para garantir que estejam apresentando uma avaliação precisa de sua saúde financeira geral.
