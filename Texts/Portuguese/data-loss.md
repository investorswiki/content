# Perda de dados
## O que é perda de dados?

A perda de dados ocorre quando informações valiosas ou confidenciais em um computador são comprometidas devido a roubo, erro humano, vírus, malware ou falha de energia. Também pode ocorrer devido a danos físicos ou falhas mecânicas ou equipamentos de um edifício.

As maiores razões para a perda de dados incluem roubo de laptop, exclusão acidental ou substituição de arquivos, quedas e picos de energia, líquidos derramados e desgaste ou falha súbita de discos rígidos. O backup regular de arquivos possibilita a recuperação de dados em caso de perda de dados. Para dados sem backup, os serviços de recuperação profissional podem restaurar os dados perdidos.

Os servidores também podem sofrer perda de dados, assim como computadores e dispositivos individuais.

## Entendendo a perda de dados

Ataques de engenharia social e phishing são fontes comuns de vírus de computador e infecções por malware que se infiltram nos computadores e levam à perda de dados. Os invasores também podem [criptografar dados](/encryption) e mantê-los como reféns até que o usuário [pague um resgate](/ransomware) para obter a chave de descriptografia. Manter o software antivírus atualizado ajuda a proteger contra vulnerabilidades de software, mas ataques de [phishing](/phishing) e [engenharia social](/social-engineering) podem não ser evitáveis pelo software antivírus.

Portanto, uma boa defesa é educar as pessoas sobre a natureza desses ataques para que sejam menos propensas a se tornarem vítimas. Por exemplo, ter cuidado ao clicar em links de e-mail e baixar anexos pode ajudar a evitar a perda de dados.

## Perda de dados: causas comuns

Picos de energia e quedas de energia prejudicam os computadores, fazendo com que os sistemas operacionais desliguem repentinamente sem seguir os procedimentos adequados. A corrupção de arquivo que pode resultar pode impossibilitar a reinicialização do computador. Derramamentos de líquidos nos teclados dos laptops podem penetrar na caixa e danificar os componentes internos, especialmente no caso de bebidas ácidas ou açucaradas, por isso é uma boa ideia manter os líquidos longe dos laptops ou usar uma caneca de viagem à prova de derramamento.

Os discos rígidos têm partes móveis que podem sofrer uma falha mecânica devido ao desgaste, superaquecimento, descarga eletrostática ou queda. Eles também podem falhar devido à corrupção de arquivos, formatação imprópria da unidade ou corrupção de software. Os discos rígidos podem falhar e sofrer perda de dados repentinamente, ou podem mostrar sinais de falha lenta, como travar repetidamente, ficar cada vez mais lento ou fazer ruídos incomuns.

A criação de backups regulares de dados do disco rígido ajuda a proteger contra essa forma de perda de dados. Por exemplo, um indivíduo pode fazer backup de seus arquivos pessoais de um computador desktop para um disco rígido externo e para [a nuvem](/cloud-storage). Ter os dados armazenados em três locais que enfrentam riscos diferentes minimiza o risco de perda total de dados.

## Perda de dados: o elemento humano

Uma grande ameaça de perda de dados para as empresas vem de funcionários que não estão cientes dos riscos que estão correndo. As empresas precisam de uma maneira de controlar como seus dados são compartilhados, monitorando e protegendo documentos de negócios sempre e onde os funcionários os estão usando, armazenando ou transmitindo, seja em anexos de e-mail, via smartphone, laptops, pen drives ou armazenamento em nuvem, para proteger contra perda de dados.

Prevenir a perda de dados é importante para as empresas protegerem sua privacidade e propriedade intelectual, bem como cumprirem as regulamentações governamentais. As organizações podem empregar recursos de prevenção contra perda de dados (DLP) em softwares de fornecedores como Google e Microsoft para se proteger contra perda de dados. Há também pacotes de prevenção de perda de dados de provedores como Clearswift, Symantec, Digital Guardian, Forcepoint e McAfee, entre outros.

## Destaques

- A perda de dados pode ser causada por fatores externos, como falta de energia, roubo ou um ataque de phishing de base ampla.

- As empresas podem se proteger usando procedimentos de prevenção de perda de dados em software e adotando protocolos para funcionários que permitem que eles trabalhem e compartilhem documentos comerciais com segurança.

- A perda de dados é a destruição de informações importantes ou privadas que foram armazenadas em um computador ou rede.

- Os indivíduos podem se proteger com proteção atualizada contra vírus e malware e evitando abrir e-mails desconhecidos.

- A perda de dados pode ser causada por erro humano, como quando uma pessoa abre um e-mail com um vírus anexado, ou quando o software antivírus expirou, ou quando uma pessoa deixa cair um computador ou derrama líquido nele.
