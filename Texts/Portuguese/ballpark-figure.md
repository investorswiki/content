# Valor aproximado
## O que é uma figura aproximada?

Um valor aproximado é uma estimativa numérica aproximada ou aproximação do valor de algo que de outra forma é desconhecido. Os números aproximados são comumente usados por [contadores](/accountant),. vendedores e outros profissionais para estimar resultados atuais ou futuros. Um corretor da bolsa poderia usar um número aproximado para estimar quanto dinheiro um cliente pode ter em algum momento no futuro, dada uma certa taxa de crescimento. Um vendedor poderia usar um número aproximado para estimar por quanto tempo um produto que um cliente estava pensando em comprar poderia ser viável.

Uma figura aproximada é essencialmente um marcador de posição estabelecido para fins de especulação sobre o valor ou o total de algo, para que as partes envolvidas possam avançar em qualquer negociação ou planejamento que esteja em andamento. Como conceito, tem aplicações em estimativas de negócios, bem como na vida cotidiana, dependendo das circunstâncias.

> Um número aproximado é uma estimativa numérica ampla do que algo poderia significar se fosse medido com precisão, avaliado para fins de negociações comerciais, negociação ou brainstorming geral de ideias.

>

## Entendendo os números aproximados

Números aproximados são estimativas usadas para mover uma discussão ou acordo quando a medida exata do tamanho ou quantidade de algo ainda não pode ser determinada.

Os números aproximados podem ser usados para fins do dia-a-dia, como estimar a quantidade de comida e bebida que pode ser necessária para um churrasco ou quantos meses provavelmente levará para pagar uma nova compra.

Os números aproximados também são usados em todos os lugares do mundo dos negócios, como estimar quanto pode custar a expansão em um determinado mercado ou quantos anos pode levar para uma empresa ser lucrativa ou para vendas justificar uma grande compra. Também pode ser usado para estimar a adoção pública de um conceito, tecnologia ou produto, como quantas pessoas provavelmente comprarão um determinado telefone e quanto tempo levará para atualizar esse telefone, uma vez adquirido.

## Considerações Especiais

Embora os números aproximados sejam usados com frequência e possam ser úteis para estabelecer uma linha de base para discussão, eles devem ser tratados como nada mais do que estimativas; não são números rígidos. Esses números são frequentemente desproporcionados por vendedores e outros profissionais que precisam usar a persuasão para gerar renda ou fechar negócios. As principais decisões de negócios e financeiras provavelmente não devem ser tomadas com base nesses números; no entanto, eles poderiam primeiro servir como estimativas a serem refinadas por meio de uma análise mais completa.

Uma teoria popular do termo postula que provavelmente tem uma história semelhante à do idioma "no mesmo estádio", que significa "aproximadamente a mesma quantidade".

##Destaques

- Os números aproximados são usados na vida cotidiana e em muitos aspectos dos negócios; no entanto, é importante lembrar que é apenas uma estimativa, não uma leitura precisa de algo.

- Um número aproximado é uma estimativa do que algo pode representar numericamente quando um número mais preciso é avaliado, como o custo de um produto.

- Números aproximados podem ser úteis para estabelecer um espaço reservado, para fins de conversação ou estimativa, quando um número mais preciso não estiver disponível.
