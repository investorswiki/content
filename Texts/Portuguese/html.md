# Linguagem de marcação de hipertexto – HTML
## O que é linguagem de marcação de hipertexto (HTML)?

HyperText Markup Language (HTML) é o conjunto de símbolos ou códigos de marcação inseridos em um arquivo destinado a exibição na Internet. A marcação informa aos navegadores da Web como exibir as palavras e imagens de uma página da Web.

Cada código de marcação de peça individual (que ficaria entre os caracteres "<" e ">") é referido como um elemento, embora muitas pessoas também se refiram a ele como uma tag. Alguns elementos vêm em pares que indicam quando algum efeito de exibição deve começar e quando deve terminar.

## HTML explicado

HyperText Markup Language é a linguagem de computador que facilita a criação de sites. A linguagem, que tem palavras de código e sintaxe como qualquer outra linguagem, é relativamente fácil de compreender e, com o passar do tempo, cada vez mais poderosa no que permite que alguém crie. O HTML continua a evoluir para atender às demandas e requisitos da Internet sob o disfarce do World Wide Web Consortium, a organização que projeta e mantém a linguagem; por exemplo, com a transição para a [Web 2.0](/web-20).

HyperText é o método pelo qual os usuários da Internet navegam na web. Ao clicar em um texto especial chamado hyperlinks, os usuários são levados a novas páginas. O uso do hiper significa que não é linear, de modo que os usuários podem ir a qualquer lugar da Internet simplesmente clicando nos links disponíveis. Marcação é o que as tags HTML fazem com o texto dentro delas; eles o marcam como um tipo específico de texto. Por exemplo, o texto de marcação pode vir na forma de negrito ou itálico para chamar a atenção específica para uma palavra ou frase.

## Fundamentos da linguagem de marcação de hipertexto

Em sua essência, o HTML é uma série de códigos curtos digitados em um arquivo de texto. Essas são as tags que potencializam os recursos do HTML. O texto é salvo como um arquivo HTML e visualizado por meio de um navegador da web. O navegador lê o arquivo e traduz o texto em uma forma visível, conforme indicado pelos códigos que o autor usou para escrever o que se torna a renderização visível. Escrever HTML requer que as tags sejam usadas corretamente para criar a visão do autor.

As tags são o que separam o texto normal do código HTML. Tags são as palavras entre o que é conhecido como colchetes angulares, que permitem que gráficos, imagens e tabelas apareçam na página da web. Tags diferentes executam funções diferentes. As tags mais básicas aplicam formatação ao texto. Como as interfaces da web precisam se tornar mais dinâmicas, podem ser usadas folhas de estilo em cascata (CSS) e aplicativos JavaScript. O CSS torna as páginas da Web mais acessíveis e o JavaScript adiciona poder ao HTML básico.

## HTML vs. XML

Ao contrário de HTML, Extensible Markup Language ou XML, permite que os usuários definam sua própria marcação. Por exemplo, usando XML, um usuário pode optar por denotar uma nota de rodapé com a tag <footnote>, enquanto outro usuário pode optar por <fn>.

Usando HTML, apenas uma tag pré-determinada pode ser usada para denotar um tipo específico de informação. Os documentos XML devem ser fáceis de ler, pois contêm tags definidas pelo usuário e os documentos consistem apenas em marcação e conteúdo.

## Destaques

- HyperText Markup Language (HTML) é a linguagem de script básica usada por navegadores da Web para renderizar páginas na World Wide Web.

- As primeiras versões do HTML eram estáticas (Web 1.0), enquanto as iterações mais recentes apresentam uma grande flexibilidade dinâmica (Web 2.0, 3.0).

- Marcação é o texto que aparece entre dois colchetes pontiagudos (por exemplo, <footnote>), e conteúdo é todo o resto.

- HyperText permite que um usuário clique em um link e seja redirecionado para uma nova página referenciada por esse link.
