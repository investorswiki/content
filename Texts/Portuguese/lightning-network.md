# Rede Lightning
Uma segunda camada operando em cima de um blockchain, permitindo maior velocidade de transação entre os nós participantes. Esta é uma solução de dimensionamento proposta.

## Destaques

- Assim como o blockchain, a rede relâmpago desintermedia instituições centrais, como bancos, que são responsáveis por rotear a maioria das transações hoje.

- A rede relâmpago é uma solução tecnológica destinada a resolver o problema de velocidade de transação no blockchain bitcoin, introduzindo transações fora do livro.

- A Lightning Network foi formalmente elaborada pela primeira vez em um artigo de Joseph Poon e Thaddeus Dryja em 2015.
