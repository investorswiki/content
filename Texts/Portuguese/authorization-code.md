# Código de autorização
## O que é um Código de Autorização?

Um código de autorização é uma senha alfanumérica que autoriza seu usuário a comprar, vender ou transferir itens ou inserir informações em um espaço protegido por segurança. Um código de autorização é normalmente uma sequência de letras, números ou uma combinação de ambos, que valida a identidade de uma pessoa, aprova uma transação ou fornece acesso a uma área segura.

O uso mais comum de códigos de autorização são aqueles enviados a um comerciante por emissores de [cartão de](/creditcard) crédito, para confirmar que o cartão de crédito do cliente tem crédito suficiente disponível para autorizar a transação.

> Diferentes plataformas podem impor requisitos específicos ao solicitar que os usuários criem senhas; por exemplo, eles podem exigir o uso de um sinal de pontuação - letras maiúsculas e minúsculas - ou limites de comprimento mínimo.

>

## Entendendo os códigos de autorização

Os códigos de autorização são usados para qualquer transação ou entrada que tenha restrições sobre quais usuários têm direito de acesso. Por exemplo, um código de autorização de cartão de crédito é um código de cinco ou seis números do banco emissor para o fornecedor que autoriza a venda. Se o cartão de crédito utilizado for falsificado ou se o cartão estiver acima do limite pré-determinado, a administradora do cartão de crédito recusará automaticamente a venda. Se aprovado, o código de autorização é anexado à transação do cartão de crédito. Isso sinaliza ao lojista que a transação é legítima, além de ajudar a identificar a transação em exames de acompanhamento, como devolução de mercadoria ou disputas de compra.

Os códigos de autorização são transmitidos digitalmente e são usados para acelerar [o processamento do cartão de crédito](/batch-credit-card-processing). Se os fornecedores tivessem que ligar para o emissor para obter um código de autorização verbal, a fim de concluir cada transação, isso reduziria drasticamente a velocidade do comércio.

## Códigos de Autorização em Aprovações de Despesas e Segurança de Dados

Os códigos de autorização também desempenham um papel nos [controles financeiros corporativos](/internalcontrols). A equipe pode receber diferentes códigos de autorização para compras e despesas. Isso permite que as empresas acompanhem as compras e os gastos em áreas específicas até o nível do funcionário.

Esses códigos de autorização também recebem limites de transação específicos. Se um funcionário estiver tentando gastar algo além do limite de seu subsídio, será necessário que um código de autorização seja fornecido por um gerente, supervisor ou outro funcionário superior na hierarquia organizacional. Nesse sentido, os códigos de autorização são mecanismos de controle integral que podem ser usados para ajudar a combater a fraude de funcionários ou a apropriação indébita de fundos.

Os códigos de autorização também se tornaram comumente usados em locais de trabalho profissionais para manter a segurança da informação. O acesso a servidores ou VPNs pode ser gerenciado usando códigos de autorização vinculados a IDs de usuário exclusivos, para controlar quem tem permissão para acessar bancos de dados confidenciais.

Embora esses códigos de autorização possam ser usados permanentemente durante o mandato de um funcionário, eles são atualizados com mais frequência periodicamente – semelhante à atualização de controles de senha. Há também códigos ou tokens de autorização de uso único que duram apenas a duração de uma única sessão.

##Destaques

- **Os códigos de autorização são senhas alfanuméricas que autorizam os usuários a realizar várias transações.**

- **No mundo corporativo, os funcionários podem receber diferentes códigos de autorização para compras e despesas que contêm limites de transação específicos.**

- **Embora os códigos de autorização possam ser usados permanentemente durante o mandato de um funcionário, eles geralmente exigem atualização de rotina.**
