# Rede de Valor Agregado (VAN)
## O que é uma rede de valor agregado (VAN)?

Uma rede de valor agregado (VAN) é um serviço privado hospedado que fornece às empresas uma maneira segura de enviar e compartilhar dados com suas [contrapartes](/counterparty). As redes de valor agregado eram uma maneira comum de facilitar o intercâmbio eletrônico de dados (EDI) entre empresas. Como a Internet criou concorrência para este serviço com o advento do e-mail seguro, as VANs responderam expandindo suas ofertas de serviços para incluir coisas como criptografia de mensagens, e-mail seguro e relatórios de gerenciamento.

Uma rede de valor agregado simplifica o processo de comunicação reduzindo o número de partes com as quais uma empresa precisa se comunicar. A VAN faz isso agindo como intermediária entre parceiros de negócios que compartilham dados baseados em padrões ou proprietários. As VANs são configuradas com recursos de auditoria para que os dados trocados sejam formatados corretamente e validados antes de serem transferidos para a próxima parte. Às vezes, as VANs são chamadas de redes de valor agregado ou linhas de comunicação prontas para uso.

## Como funciona uma rede de valor agregado (VAN)

As redes de valor agregado geralmente são usadas por grandes empresas para o gerenciamento eficiente [da cadeia de suprimentos](/scm) com seus fornecedores, ou por consórcios industriais ou empresas de telecomunicações. As VANs geralmente operam em uma configuração de caixa postal, enquanto uma empresa envia uma transação para uma VAN, e a VAN a coloca na caixa postal do destinatário. O receptor entra em contato com a VAN e pega a transação, e então envia uma transação própria.

O sistema é semelhante ao e-mail, exceto que é usado para dados estruturados padronizados em vez de texto não estruturado.

## VANs na era da Internet

A onipresença da Internet diminuiu a atração de VANs, em grande parte devido a considerações de custo. Simplificando, geralmente é mais econômico mover dados pela Internet do que pagar as taxas mensais mínimas e cobranças por caractere incluídas nos contratos típicos de VAN. As VANs enfrentaram o desafio da Internet concentrando-se em setores específicos do setor, como saúde, varejo e manufatura. Esses setores têm preocupações exclusivas com integridade de dados e segurança que tornam as VANs uma verdadeira solução de valor agregado.

> As VANs simplificam o processo de comunicação, permitindo que a empresa se comunique com menos partes.

>

Os dados trocados por meio da VAN podem ser formatados para ir diretamente para o aplicativo de software da organização receptora, um pacote de [planejamento de recursos empresariais](/erp) (ERP), por exemplo. Essa troca direta entre duas empresas aumenta a velocidade do comércio e, ao mesmo tempo, reduz as chances de erros humanos que ocorrem com a entrada manual de dados.

As VANs também podem fornecer ferramentas de visibilidade que mostram o status de entrega de dados e alguns fluxos de trabalho correspondentes, permitindo que as empresas coordenem melhor as atividades dependentes por meio do sistema, em vez de trocar telefonemas e e-mails. O uso de uma VAN não é apenas mais eficiente e preciso, mas também economiza o custo de contratação de profissionais de entrada de dados humanos para a troca de informações.

Como muitas tecnologias pré-Internet, as VANs tiveram que se reinventar para permanecer relevantes no futuro. Hoje, as VANs oferecem serviços que vão além das caixas de correio para [troca](/interchange) e recuperação EDI, autenticação de mensagens e arquivamento de transações anteriores. As VANs modernas criam valor para as empresas, oferecendo backups automáticos de dados EDI, acesso flexível a esses dados por meio de portais da Web seguros e pacotes de preços de dados ilimitados.

##Destaques

- As VANs são importantes para gerenciar as cadeias de suprimentos.

- As redes de valor agregado são frequentemente usadas para intercâmbio eletrônico de dados entre empresas.

- As VANs facilitam o processo de comunicação com a comunicação entre menos partes.
