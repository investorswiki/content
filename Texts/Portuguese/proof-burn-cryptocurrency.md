# Prova de Queima (Criptomoeda)
## O que é uma prova de queima para criptomoeda?

A prova de queima é um dos vários algoritmos de [mecanismo de consenso implementados por uma rede](/consensus-mechanism-cryptocurrency) [blockchain](/blockchain) para garantir que todos os nós participantes cheguem a um acordo sobre o estado verdadeiro e válido da rede blockchain. Este algoritmo é implementado para evitar a possibilidade de qualquer gasto duplo de moedas criptográficas.

A prova de queima segue o princípio de “queimar” as moedas detidas pelos mineradores que lhes concedem direitos minerários.

## Entendendo os Sistemas de Prova

O blockchain é o banco de dados primário de uma [criptomoeda](/cryptocurrency). Ele contém todas as informações relacionadas à transação em [blocos](/block-bitcoin-block) e esses blocos atuam como unidades de armazenamento de dados do blockchain. Um bloco é escrito apenas quando os nós do blockchain concordam com um conjunto de transações que os nós consideram válidos.

Devido à natureza autônoma e descentralizada da rede blockchain, é necessário um mecanismo automatizado para garantir que os nós participantes concordem apenas com transações válidas. Essa importante tarefa é realizada por algoritmos de mecanismo de consenso.

### Comprovante de Trabalho

O tipo mais conhecido de algoritmos geradores de consenso é chamado [de prova de trabalho](/proof-work) (POW). Em um sistema POW, os mineradores são recompensados por atualizar o blockchain. Isso envolve o uso do poder de computação para resolver uma equação matemática e resulta em uma recompensa monetária. Bitcoin, a criptomoeda original e mais popular, usa um sistema POW.

Quanto mais um minerador paga pelo equipamento de computação necessário para resolver o quebra-cabeça criptográfico, maior a chance de obter o direito de minerar os blocos. No entanto, uma abordagem POW requer dispositivos de hardware de mineração caros, e esse método é dificultado pelo alto consumo de energia.

Como o método POW consome muitos recursos, não é muito eficiente. Consequentemente, moedas POW, incluindo Bitcoin, não são muito úteis como um instrumento fungível.

### Prova de participação

[A prova de participação](/proof-stake-pos) (POS) é outro algoritmo que atribui direitos de mineração aos mineradores proporcionais às suas participações na criptomoeda.

Nesse sistema, o blockchain é mantido por um grupo de validadores selecionados aleatoriamente que “apostam” os tokens de rede nativos, bloqueando-os no blockchain para produzir e aprovar blocos.

Infelizmente, a construção mais complexa dos sistemas POS os torna mais vulneráveis a ataques e, como os benefícios fluem cada vez mais para os maiores detentores de moedas, em um sistema POS, quanto mais rico você é, mais rico você fica.

### Prova de Queimadura

A prova de queima (POB) é um algoritmo de consenso alternativo que tenta resolver o problema de alto consumo de energia de um sistema POW.

O POB é frequentemente chamado de sistema POW sem desperdício de energia. Ele opera com o princípio de permitir que os mineradores “queimem” tokens de moeda virtual. Eles recebem então o direito de escrever blocos na proporção das moedas queimadas.

Iain Stewart, o inventor do algoritmo POB, usa uma analogia para descrever o algoritmo: moedas queimadas são como plataformas de mineração. Nesta analogia, um minerador queima suas moedas para comprar um equipamento de mineração virtual que lhe dá o poder de minerar blocos. Quanto mais moedas forem queimadas pelo minerador, maior será o seu "rig" de mineração virtual.

Para queimar as moedas, os mineradores as enviam para um endereço verificável que não pode ser gasto. Esse processo não consome muitos recursos (além das moedas queimadas) e garante que a rede permaneça ativa e ágil. Dependendo da implementação, os mineradores podem queimar a moeda nativa ou a moeda de uma cadeia alternativa, como Bitcoin. Em troca, eles recebem uma recompensa no token de moeda nativa do blockchain.

Você pode enviar transações para a rede que queimará suas próprias moedas de criptomoeda. Outros participantes podem minerar/queimar em cima do seu bloco, e você também pode pegar as transações de outros participantes para adicioná-las ao seu bloco. Essencialmente, toda essa atividade de queima mantém a rede ágil, e os participantes são recompensados por suas atividades (tanto queimando suas próprias moedas quanto queimando as moedas de outras pessoas).

Para evitar a possibilidade de vantagens injustas para os early adopters, o sistema POB implementou um mecanismo que promove a queima periódica de criptomoedas para manter o poder de mineração. O poder das moedas queimadas “decai” ou reduz parcialmente cada vez que um novo bloco é extraído. Isso promove a atividade regular dos mineradores, em vez de um investimento único e antecipado. Para manter uma vantagem competitiva, os mineradores também podem precisar investir periodicamente em melhores equipamentos à medida que a tecnologia avança.

## Exemplo de Prova de Queima

A implementação do POB pode ser personalizada. Por exemplo, a Slimcoin, uma rede de moeda virtual que usa POB, permite que um minerador queime moedas que não apenas lhe dá o direito de competir pelo próximo bloco, mas também lhe dá a chance de receber blocos durante um período de tempo mais longo, pelo menos um ano.

Essencialmente, a implementação POB do Slimcoin combina três algoritmos: POW, POS e o conceito central de POB. O processo de queima de moedas utiliza POW; quanto mais moedas se queima, mais chances se tem de minerar, garantindo assim o POS; e todo o ecossistema segue o conceito POB.

## Destaques

- A prova de queima é a terceira tentativa de criar um sistema para impedir atividades fraudulentas em uma blockchain, além de melhorar o funcionamento da blockchain como ferramenta para transações.

- As criptomoedas usam vários métodos para validar os dados armazenados em suas blockchains, incluindo um método chamado "prova de queima".

- Prova de trabalho e prova de participação também são métodos para prevenir atividades fraudulentas em uma blockchain; prova de trabalho é o sistema empregado pela criptomoeda original e mais popular, Bitcoin.
