# regressão não linear
A regressão não linear é uma forma de análise de regressão na qual os dados são ajustados a um modelo e, em seguida, expressos como uma função matemática. A [regressão](/regression) linear simples relaciona duas variáveis (X e Y) com uma linha reta (y = mx + b), enquanto a regressão não linear relaciona as duas variáveis em uma relação não linear (curva).

O objetivo do modelo é tornar a [soma dos quadrados](/sum-of-squares) o menor possível. A soma dos quadrados é uma medida que rastreia até que ponto as observações Y variam da função não linear (curva) usada para prever Y.

Ele é calculado primeiro encontrando a diferença entre a função não linear ajustada e cada ponto Y de dados no conjunto. Então, cada uma dessas diferenças é elevada ao quadrado. Por último, todas as figuras quadradas são somadas. Quanto menor a soma desses números ao quadrado, melhor a função ajusta os pontos de dados no conjunto. A regressão não linear usa funções logarítmicas, funções trigonométricas, funções exponenciais, funções de potência, curvas de Lorenz, funções gaussianas e outros métodos de ajuste.

A modelagem de regressão não linear é semelhante à modelagem de regressão linear, pois ambas buscam rastrear uma resposta específica de um conjunto de variáveis graficamente. Modelos não lineares são mais complicados do que modelos lineares para desenvolver porque a função é criada por meio de uma série de aproximações (iterações) que podem resultar de tentativa e erro. Os matemáticos usam vários métodos estabelecidos, como o método de Gauss-Newton e o método de Levenberg-Marquardt.

Muitas vezes, os modelos de regressão que parecem não lineares à primeira vista são, na verdade, lineares. O procedimento de estimativa de curva pode ser usado para identificar a natureza das relações funcionais em jogo em seus dados, para que você possa escolher o modelo de regressão correto, seja linear ou não linear. Os modelos de regressão linear, embora normalmente formem uma linha reta, também podem formar curvas, dependendo da forma da equação de regressão linear. Da mesma forma, é possível usar a álgebra para transformar uma equação não linear de modo que ela imite uma equação linear – essa equação não linear é chamada de “intrinsecamente linear”.

> A regressão linear relaciona duas variáveis com uma linha reta; a regressão não linear relaciona as variáveis usando uma curva.

>

## Exemplo de regressão não linear

Um exemplo de como a regressão não linear pode ser usada é prever o crescimento populacional ao longo do tempo. Um gráfico de dispersão de dados populacionais em mudança ao longo do tempo mostra que parece haver uma relação entre o tempo e o crescimento populacional, mas que é uma relação não linear, exigindo o uso de um modelo de regressão não linear. Um modelo de crescimento populacional logístico pode fornecer estimativas da população para períodos que não foram medidos e previsões de crescimento populacional futuro.

As variáveis independentes e dependentes usadas na regressão não linear devem ser quantitativas. Variáveis categóricas, como região de residência ou religião, devem ser codificadas como variáveis binárias ou outros tipos de variáveis quantitativas.

Para obter resultados precisos do modelo de regressão não linear, você deve certificar-se de que a função especificada descreve com precisão a relação entre as variáveis independentes e dependentes. Bons valores iniciais também são necessários. Valores iniciais ruins podem resultar em um modelo que não converge ou em uma solução que é apenas ótima localmente, em vez de globalmente, mesmo que você tenha especificado a forma funcional correta para o modelo.

##Destaques

- A regressão não linear pode mostrar uma previsão do crescimento populacional ao longo do tempo.

- A regressão não linear é uma função curva de uma variável X (ou variáveis) que é usada para prever uma variável Y

- Tanto a regressão linear quanto a não linear predizem respostas Y de uma variável X (ou variáveis).
