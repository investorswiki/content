# Critério dos Mínimos Quadrados
## Qual é o critério dos mínimos quadrados?

O critério dos mínimos quadrados é uma fórmula usada para medir a precisão de uma linha reta na representação dos dados que foram usados para gerá-la. Ou seja, a fórmula determina a linha de melhor ajuste.

Esta fórmula matemática é usada para prever o comportamento das variáveis dependentes. A abordagem também é chamada de linha de regressão de mínimos quadrados.

## Entendendo o Critério dos Mínimos Quadrados

O critério dos mínimos quadrados é determinado minimizando a [soma dos quadrados](/sum-of-squares) criada por uma função matemática. Um quadrado é determinado pelo quadrado da distância entre um ponto de dados e a linha de regressão ou valor médio do conjunto de dados.

Uma análise de mínimos quadrados começa com um conjunto de pontos de dados plotados em um gráfico. As variáveis independentes são plotadas no eixo x horizontal, enquanto as variáveis dependentes são plotadas no eixo y vertical. O analista usa a fórmula dos mínimos quadrados para determinar a linha reta mais precisa que explicará a relação entre uma variável independente e uma variável dependente.

### Usos Comuns dos Mínimos Quadrados

Avanços no poder de computação, além de novas técnicas de engenharia financeira, aumentaram o uso de métodos de mínimos quadrados e estenderam seus princípios básicos.

Mínimos quadrados e métodos estatísticos relacionados tornaram-se comuns em finanças, economia e investimentos, mesmo que seus beneficiários nem sempre estejam cientes de seu uso.

Por exemplo, os robo-advisors agora usados por muitas plataformas de investimento empregam técnicas de simulação de Monte Carlo para gerenciar portfólios, embora isso seja feito nos bastidores e fora da vista dos correntistas que os utilizam.

Outras aplicações incluem análise de séries temporais de distribuições de retorno, previsão econômica e estratégia de política e modelagem avançada de opções.

### O que os mínimos quadrados dizem a você?

Em vez de tentar resolver uma equação exatamente, os matemáticos usam o [método dos mínimos quadrados](/least-squares-method) para chegar a uma aproximação. Isto é referido como uma estimativa de probabilidade máxima.

A abordagem dos mínimos quadrados limita a distância entre uma função e os pontos de dados que a função explica. É usado em análise de regressão, muitas vezes em modelagem de [regressão não linear](/nonlinear-regression) em que uma curva é ajustada em um conjunto de dados.

> Os matemáticos usam o método dos mínimos quadrados para chegar a uma estimativa de máxima verossimilhança.

>

A abordagem dos mínimos quadrados é um método popular para determinar equações de regressão e informa sobre a relação entre as variáveis de resposta e as variáveis de previsão.

Os métodos de modelagem que são frequentemente usados ao ajustar uma função a uma curva incluem o método da linha reta, o [método polinomial](/polynomial_trending),. o método logarítmico e o método gaussiano.

Os mínimos quadrados lineares ou ordinários são o estimador de regressão linear mais simples e mais comumente usado para analisar dados observacionais e experimentais. Ele encontra uma linha reta de melhor ajuste através de um conjunto de pontos de dados fornecidos.

## Destaques

- Os resultados dos mínimos quadrados podem ser usados para resumir dados e fazer previsões sobre valores relacionados, mas não observados, do mesmo grupo ou sistema.

- O método do critério dos mínimos quadrados é usado em finanças, economia e investimentos.

- É usado para estimar a precisão de uma linha na representação dos dados que foram usados para criá-la.
