# Índice de ajuste
## O que é um índice de ajuste?

[índice de](/index) ajuste refere-se a uma modificação que é aplicada a um conjunto de dados para torná-lo uma melhor representação das condições externas. Sem o uso de um índice de ajuste, os dados em questão podem ser distorcidos. Um índice de ajuste pode ser uma modificação baseada em fórmula ou um único número derivado de um conjunto externo de observações. É frequentemente usado ao relatar [indicadores econômicos](/economic_indicator),. como emprego sazonal.

## Como funcionam os índices de ajuste

Quase todo mundo depende de dados financeiros e econômicos para tomar decisões importantes. [Os investidores](/investor) precisam dessas informações para tomar decisões informadas sobre seus investimentos. Corporações e governos dependem dele para ajudar seus negócios a crescer e garantir o florescimento da [economia .](/economy) É por isso que é tão importante que as informações fornecidas sejam precisas. É aqui que o índice de ajuste é útil.

Índice de ajuste é um termo que tem aplicações em uma ampla gama de contextos. Por si só, refere-se a uma alteração numérica de dados específicos para melhorar a precisão ou utilidade de um conjunto de dados. As melhorias podem tentar remover distorções como fluxos e refluxos sazonais em um determinado conjunto de dados ou levar em conta um tamanho [de amostra relativamente pequeno](/sample).

Mas isso não é tudo. Outras aplicações do índice de ajuste podem ser usadas para atualizar um dado desatualizado para melhor representar as condições atuais. Também pode melhorar a comparabilidade de conjuntos de dados distintos. Em última análise, um índice de ajuste pode fornecer contexto para um conjunto de dados autônomo e, assim, maximizar a aplicabilidade dessas informações. Os índices fazem isso em uma enorme variedade de situações.

Nas transações comerciais, as partes podem usar um índice de ajuste para permitir modificações com base nas condições de [mercado vigentes](/market). Governos e [economistas](/economist) podem alterar os dados para contabilizar os fluxos sazonais de gastos e empregos, a fim de obter uma melhor representação das condições econômicas. Por exemplo, os economistas dos EUA fazem ajustes regulares nos números de empregos do país para contabilizar contratações adicionais durante a temporada de Natal. Sem modificações, a [taxa de desemprego](/unemploymentrate) seria distorcida e inflada artificialmente, à medida que mais pessoas voltassem à força de trabalho para esses empregos temporários durante a temporada.

> Dados econômicos, como o relatório de empregos, geralmente são ajustados para fatores sazonais para evitar que sejam inflados artificialmente.

>

## Exemplos de um índice de ajuste

### Hipotecas de taxa ajustável (ARMs)

Talvez o índice de ajuste mais conhecido seja aquele que os credores usam para redefinir [hipotecas de taxa ajustável](/arm) após o período inicial ter expirado. Isso normalmente ocorre em qualquer lugar entre três a 10 anos de vida de um ARM. Nesse ponto, o credor usa um índice de ajuste para conciliar a taxa inicial do empréstimo com as taxas de mercado vigentes. A taxa mais utilizada é a [London Interbank Offered](/libor) [Rate](/libor) (LIBOR). O credor pega esse índice e adiciona uma [margem](/margin) para definir uma nova [taxa de juros](/interestrate) para o empréstimo.

### Índice de Desenvolvimento Humano (IDH)

Um segundo exemplo demonstra como os pesquisadores podem usar um índice de ajuste para comparar uma variedade de conjuntos de dados. O Programa das Nações Unidas para o Desenvolvimento (PNUD) mantém um [Índice de Desenvolvimento Humano](/human-development-index-hdi) para acompanhar as realizações dos países em saúde, educação e [renda](/income) [.](/income)

O IDH de vários países pode ser comparado para demonstrar os níveis relativos de progresso desses países nessas medidas. No entanto, este índice consiste em medidas agregadas de desenvolvimento para cada país e originalmente não continha informações sobre como os benefícios do desenvolvimento são distribuídos igualmente dentro de cada país.

Partindo do pressuposto de que a desigualdade necessariamente degrada o verdadeiro nível de desenvolvimento humano de um país, o PNUD decidiu que esta informação era relevante para a medição do IDH. Para resolver esse problema, o PNUD desenvolveu um índice de desigualdade em 2010. Ele aplicou esse índice ao IDH para criar um IDH ajustado à desigualdade. Esse índice de ajuste permitiu que o PNUD a cada ano ajustasse o índice de forma a potencializar o índice de desenvolvimento humano em países com maior igualdade.

### Índice de Preços ao Consumidor (IPC)

Outra cláusula de ajuste permite que as partes de um contrato comercial ou pessoal modifiquem esse acordo de acordo com variáveis econômicas externas. O [Índice de Preços ao Consumidor](/consumerpriceindex) (CPI), publicado mensalmente pelo [Bureau of Labor Statistics](/bls) (BLS), é um índice de ajuste comumente usado que as partes de um contrato usarão para estruturar uma cláusula de escalação. Isso é comum em uma ampla variedade de acordos, desde escalas salariais sindicais até arrendamentos comerciais e pagamentos de pensão alimentícia. À medida que o IPC aumenta ou diminui, a [obrigação](/obligation) financeira do pagador também aumenta e diminui.

##Destaques

- Eles podem aumentar a utilidade de dados passados e presentes, tornando-os mais precisos ou mais consistentes ao longo do tempo.

- Os índices de ajuste podem ser aplicados a coisas como taxas de juros de contratos específicos, preços, salários ou medidas gerais de condições econômicas ou de mercado.

- Um índice de ajuste é um fator ou fórmula usada para ajustar um conjunto de dados ou métrica para refletir melhor medição, nova metodologia ou mudanças nas condições do mundo real.
