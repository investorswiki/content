# Software de gráficos Forex
## O que é o software de gráficos Forex?

O software de gráficos Forex é um kit de ferramentas de visualização analítica baseado em computador usado para ajudar os comerciantes de moeda com análise de negociação [de câmbio (FX).](/forex) Essas ferramentas de software produzem gráficos de preços interativos para vários pares de moedas, juntamente com vários indicadores técnicos e sobreposições.

Muitos comerciantes usam pacotes de software de gráficos forex para determinar a direção provável em um determinado par de moedas em conjunto com outras tecnologias, como [software de previsão preditiva](/forex-forecast-and-forecasting-software) e [negociação on-line](/electronic-currency-trading) para obter uma vantagem nos mercados forex.

## Entendendo o software de gráficos Forex

O software de gráficos Forex oferece aos usuários uma interface gráfica que exibe os movimentos no valor dos pares de moedas durante um determinado período. Vários tipos de gráficos diferentes estão disponíveis para o usuário, como gráficos de barras, gráficos de linhas e velas, entre outros. Essas informações podem ser tão simples quanto um gráfico básico de preços históricos ou podem incluir resmas de informações adicionais úteis para [análise técnica](/technicalanalysis) de pares de moedas. Esses dados ajudam um trader a identificar os pares e prazos de negociação mais vantajosos, juntamente com os pontos de entrada e saída.

Muitos [corretores forex](/currency-trading-forex-brokers) agora oferecem algum tipo de software de gráficos para os comerciantes que usam sua plataforma, gratuitamente ou sob um serviço de assinatura. Escolher o melhor software de gráficos geralmente depende das necessidades e experiências pessoais de um trader. As considerações incluem o tipo de análise técnica que desejam realizar, o volume ou a duração de suas negociações e o tipo de dispositivo que desejam usar para visualizar os gráficos. Traders experientes anotarão as fontes de dados das quais as soluções de gráficos extraem seus preços e outras informações, garantindo que essas fontes estejam atualizadas, confiáveis e precisas.

## Exibições de software de gráficos Forex

Os gráficos de Forex geralmente exibem informações como um gráfico de [linhas, gráfico](/linechart) de [barras](/barchart) ou gráfico de velas, dependendo das preferências ou necessidades do trader. O software de gráficos Forex normalmente mostra preços de [fechamento, preços](/closingprice) de [abertura](/openingprice),. preço alto e pontos de preço baixo. Os gráficos de barras e [velas](/candlestick) exibem informações sobre os preços de abertura e fechamento de um par de moedas, bem como os preços altos e baixos do par de moedas durante esse período. Muitas vezes, os comerciantes confiam no gráfico de velas porque exibe uma quantidade maior de informações.

Dependendo dos indicadores e padrões que um trader de FX espera detectar, eles podem optar por exibir intervalos de intervalo por minuto ou hora, enquanto outros preferem intervalos diários, semanais, mensais ou mesmo anuais ou plurianuais.

## Tipos de gráficos básicos

Um gráfico de linhas é uma representação gráfica da ação histórica do preço de um ativo que conecta uma série de pontos de dados com uma linha contínua. Este é o tipo mais básico de gráfico usado em finanças e normalmente mostra apenas os preços de fechamento de um título ao longo do tempo. Os gráficos de linhas podem ser usados em qualquer período de tempo, mas na maioria das vezes usando mudanças de preços diárias.

Os gráficos de barras exibem informações usando uma linha vertical simples definida em relação a um eixo x/y. O preço é exibido como o eixo y com o tempo representado no eixo x. Marcas horizontais que se estendem para a esquerda ou direita da linha mostram os preços de abertura e fechamento.

Os gráficos de velas apresentam as mesmas informações com mais variedade visual. Este tipo de gráfico usa duas cores diferentes para indicar a direção da mudança ao longo do tempo, uma cor para cima e outra para baixo. Uma linha fina representa o intervalo de preços oferecidos ao longo do dia com uma barra mais grossa preenchendo o espaço entre os preços de abertura e fechamento. Os comerciantes podem determinar se o preço de abertura é superior ao preço de fechamento com base na cor da barra. Normalmente, cores mais claras indicam que um par fechou mais alto do que abriu, enquanto cores mais escuras indicam uma queda no preço entre a abertura e o fechamento.

##Destaques

- Um gráfico forex mostra graficamente o comportamento histórico de uma moeda paga em vários períodos de tempo, juntamente com padrões técnicos, indicadores e sobreposições.

- O software de gráficos Forex pode ser uma ferramenta poderosa que os usuários podem personalizar e negociar diretamente nos mercados forex eletrônicos.

- O software de gráficos de Forex fornece aos comerciantes de FX uma interface gráfica interativa para ajudar na análise técnica e nas negociações de tempo.
