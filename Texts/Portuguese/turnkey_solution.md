# Solução chave na mão
## O que é uma solução chave na mão?

Uma solução pronta para uso é um tipo de sistema construído [de ponta a ponta](/end-to-end) para um cliente que pode ser facilmente implementado em um processo de negócios atual. Ele está imediatamente pronto para uso após a implementação e é projetado para cumprir um determinado processo, como fabricação (em parte ou todo), cobrança, design de site, treinamento ou gerenciamento de conteúdo.

Um sistema de solução turnkey é diferente de um sistema personalizado ou projetado. Quando uma empresa ou negócio não está procurando por vários sinos e assobios, uma solução pronta para uso pode ser uma opção acessível e simples.

Em uma solução turnkey, um comprador corporativo só precisa "girar" uma "chave" para iniciar uma atividade comercial na estrutura recém-construída. A "solução" evoluiu no conceito chave na mão à medida que as economias se tornaram mais baseadas em serviços.

Uma possível desvantagem desse método, ao contrário de projetar um processo interno ou um sistema personalizado, é que as especificações importantes para uma determinada empresa podem não estar suficientemente integradas à solução. Decisões internas ou personalizadas versus decisões prontas para uso levam em consideração a complexidade de um processo de negócios.

> Com os rápidos avanços na tecnologia de software e hardware, as soluções prontas para uso são populares para empresas que desejam velocidade de lançamento no mercado.

>

## Outros tipos de termos chave na mão

Tradicionalmente, turnkey, como termo, é comumente aplicado a projetos de construção, como armazéns, outros edifícios construídos especificamente ou casas. A ideia é que o empreiteiro finalize um produto que o comprador possa utilizar imediatamente.

Uma propriedade turnkey é geralmente um prédio de apartamentos recém-reformado e atualizado ou uma casa pronta para morar para compradores ou locatários. Empresas imobiliárias, vendedores e compradores de imóveis e investidores imobiliários geralmente descrevem novas casas ou casas como edifícios prontos para uso "chave na mão".

Um negócio turnkey pode se referir a um negócio que pode ser adquirido sem a necessidade de fazer alterações no modelo de negócios ou nos produtos vendidos. Por exemplo, uma franquia como a Taco Bell é um tipo de negócio pronto para uso.

## Prós e Contras das Soluções Chave na Mão

Uma solução pronta para uso pode ser a maneira mais acessível de construir algo de ponta a ponta que seja benéfico para vários usuários. No entanto, soluções prontas para uso podem não levar em conta as necessidades individuais de uma empresa. Uma solução pronta para uso de software de gerenciamento de conteúdo oferecida por uma empresa terceirizada pode ser mais barata de comprar, mas pode não cobrir todas as necessidades do comprador.

## Exemplo de uma solução chave na mão

Uma empresa que pretenda implementar um recurso de cobrança online em seu site pode usar provedores terceirizados para processar cada [transação](/transaction) e lidar com todos os problemas técnicos relacionados a esse processo. A utilização de uma solução chave na mão, neste caso, permite à empresa evitar as dores de cabeça de programar a ferramenta internamente e as aprovações associadas ao desenvolvimento de um sistema de faturação de raiz.

##Destaques

- Soluções chave na mão podem economizar tempo e dinheiro de uma empresa quando implementadas com sucesso.

- Turnkey é uma palavra que pode ser usada para descrever uma solução, um negócio ou uma propriedade.

- Soluções turnkey são criadas para serem implementadas em um processo de negócios atual sem interrupções.

- Uma solução pronta para uso é o oposto de uma solução personalizada.
