# Média ajustada
## O que é uma média ajustada?

A média ajustada surge quando as médias estatísticas devem ser corrigidas para compensar desequilíbrios de dados e grandes [variações](/variance). Os valores discrepantes presentes nos conjuntos de dados geralmente serão removidos para determinar o ajustado, pois podem ter um grande impacto nas médias calculadas de pequenas populações. Uma média ajustada pode ser determinada removendo esses valores discrepantes por meio de análise de regressão. As médias ajustadas também são chamadas de médias [dos mínimos quadrados](/least-squares).

## Entendendo as médias ajustadas

As médias ajustadas são usadas com mais frequência em finanças quando há pontos de dados discrepantes que têm um impacto desproporcional na linha de tendência de um conjunto de dados. Um analista pode optar por remover totalmente os valores atípicos, mas isso normalmente é feito apenas nos casos em que os valores por trás dos valores atípicos são conhecidos ou uma previsão aproximada de uma tendência é um motivo adequado.

Para pesquisadores e profissionais que desejam remover valores discrepantes, as equações de regressão múltipla são o método preferido. A análise de regressão fornece um resultado mais preciso e dados mais confiáveis na conclusão de um estudo. Além da análise de regressão, também existem maneiras mais básicas de ajustar uma média.

Uma maneira de ajustar uma média é adicionar variáveis categóricas que separam os dados com mais precisão. Por exemplo, imagine um estudo analisando o consumo de álcool na profissão contábil que descobre que os contadores hoje bebem metade do que os contadores bebiam há 50 anos. Embora isso possa parecer positivo, após uma análise mais profunda, descobre-se que o estudo não foi ajustado para o gênero. Quando o sexo é levado em consideração, verifica-se que os contadores do sexo masculino bebem um pouco menos do que os contadores de 50 anos atrás, mas a maior parte da mudança é o crescimento do número total de contadores do sexo feminino. Em média, o estudo mostra que as mulheres contabilistas bebem quase o mesmo que suas contrapartes femininas há 50 anos. As contabilistas também bebem muito menos do que os contabilistas do sexo masculino hoje e há 50 anos. Mas as contadoras são mais numerosas do que nunca, ajudando efetivamente a reduzir o nível geral de consumo de álcool na profissão, embora seus colegas homens tenham permanecido relativamente estáticos em hábitos de consumo.

As variáveis adicionais, neste caso, contam uma história mais precisa sobre os dados e podem ser combinadas novamente em uma média geral, adicionando um valor para sexo que reflita a proporção de homens para mulheres em cada grupo de amostragem. Isso mostraria uma queda mais modesta no consumo geral de bebidas na profissão. No entanto, fazer uma análise mais aprofundada desses dados pode sugerir que uma média integrada não é a melhor maneira de apresentar esses dados.

Neste exemplo, o sexo dos participantes seria considerado [covariável](/covariance),. um tipo de variável que o pesquisador não pode controlar, mas que impacta os resultados de um experimento. Usar uma média ajustada é uma forma de compensar as covariáveis: qual o efeito da atividade ou comportamento se não houvesse diferenças entre os gêneros? Os mesmos tipos de ajustes são feitos para outros dados demográficos, como idade, etnia, status socioeconômico etc.

## Exemplo de uma média ajustada

Em 2009, o Financial Accounting Standards Board (FASB) esclareceu a [regra de marcação a mercado](/marktomarket) [para](/marktomarket) aliviar a pressão e melhorar instantaneamente os balanços dos grandes bancos. Se um analista estivesse revisando as tendências na solidez do balanço patrimonial em 2010 para os últimos dez anos usando medidas publicadas pelo banco, a média média seria problemática e imprecisa porque incluiria a mudança de regra de 2009.

Uma maneira de corrigir isso é criar um coeficiente de variação para as diferenças médias entre os valores do balanço patrimonial e os valores de mercado na época, para um subconjunto de ativos comuns em grandes bancos. Na prática, no entanto, os analistas do setor bancário continuaram a calcular valores de marcação a mercado rigorosos após o esclarecimento da regra, de modo que a solução simples seria usar esses conjuntos de dados. Mais importante, os bancos sempre tiveram um pouco de discrição sob as regras de marcação a mercado, de modo que os valores do balanço patrimonial para ativos mantidos devem sempre ser vistos com ceticismo e verificados independentemente quando possível.

##Destaques

- A média ajustada é usada para corrigir médias estatísticas que incluem desequilíbrios óbvios devido a discrepâncias no conjunto de dados.

- O método preferido de calcular uma média ajustada para a maioria dos profissionais que se baseiam em estatísticas é por meio de regressões múltiplas.

- Uma maneira mais simples de calcular uma média ajustada é adicionar variáveis categóricas que separam os dados com mais precisão, como sexo.
