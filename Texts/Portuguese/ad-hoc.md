# Ad hoc
A frase "ad hoc" é uma frase de origem latina que é usada no inglês moderno para significar "para este propósito" ou "especificamente para isso". A tradução direta da frase latina é dada pela maioria dos dicionários como "para isso".

O termo "ad hoc" pode ser usado para indicar que uma solução encontrada para um determinado problema ou questão não deve ser considerada como uma solução padrão a ser usada em aplicativos futuros.

Por exemplo, um juiz pode tomar uma decisão "ad hoc" sobre um caso analisando os detalhes do caso e detalhes específicos da situação, mas com a visão de que essa decisão não será considerada como jurisprudência para resolver incidentes de natureza semelhante. natureza no futuro.

No desenvolvimento de software, o termo "ad hoc" é frequentemente usado na frase "solução ad hoc" para significar uma solução específica e sob medida desenvolvida para lidar com as necessidades de um sistema ou usuário que é apresentado.
