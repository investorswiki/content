# Plano Patrocinado pelo Empregador (ESP)
## O que é um plano patrocinado pelo empregador?

Um plano patrocinado pelo empregador é um tipo de plano de benefícios oferecido aos funcionários sem custo ou com custo relativamente baixo. Esses planos, como um 401(k) ou HSA, cobrem uma série de serviços, incluindo poupança para aposentadoria e assistência médica. Os funcionários que se inscrevem nesses programas capitalizam o benefício de receber serviços com desconto.

Por outro lado, os empregadores que oferecem esses planos normalmente se beneficiam de incentivos fiscais. Além disso, patrocinar benefícios é visto como uma forma de recrutar e reter funcionários valiosos.

## Entendendo os Planos Patrocinados pelo Empregador

Os planos patrocinados pelo empregador abrangem uma vasta geografia de serviços que incluem diferentes tipos de planos de saúde em grupo e planos de poupança para aposentadoria com vantagens fiscais preferenciais.

Planos de poupança patrocinados pelo empregador, como os planos 401(k) e [Roth 401(k)](/roth401k),. fornecem aos funcionários uma maneira automática de economizar para a aposentadoria enquanto se beneficiam de incentivos fiscais. A recompensa para os funcionários que participam desses programas é que eles recebem essencialmente dinheiro grátis quando seus empregadores oferecem contribuições correspondentes. Devido ao aumento do custo dos serviços de saúde, os planos de saúde em grupo são um benefício adicional para os colaboradores que beneficiam de custos mais baixos.

## Vantagens fiscais dos planos patrocinados pelo empregador

Contribui para um plano 401(k), é feito usando dólares "antes de impostos". Antes de impostos significa que o dinheiro flui diretamente do contracheque para o plano antes da dedução dos impostos. Como resultado, menos de sua renda acaba sendo tributada. A avaliação do imposto ocorre no momento em que os fundos são retirados da conta, geralmente a uma taxa mais baixa.

Com um Roth 401(k), você contribui com dinheiro para o plano "após os impostos". Após os impostos significa que o dinheiro flui do seu cheque para o plano após as deduções fiscais. A desvantagem é que as retiradas qualificadas mais tarde são isentas de impostos. Um plano Roth 401(k) é uma ferramenta particularmente útil se você acabar em uma faixa de imposto mais baixa na aposentadoria.

Alguns tipos de planos de saúde patrocinados pelo empregador também oferecem certas vantagens fiscais. Um exemplo é uma [Conta Poupança Saúde](/hsa) (HSA), que está associada a um [plano de saúde de alta franquia](/hdhp) (HDHP).

Um HSA é um tipo de conta poupança para despesas médicas qualificadas. As contribuições são "antes de impostos", os juros crescem isentos de impostos e as retiradas feitas para cobrir despesas médicas qualificadas também são isentas de impostos. E, ao contrário da [Conta de Despesas Flexível](/flexiblespendingaccount) (FSA), o dinheiro em sua HSA rola de ano para ano.

Algumas HSAs funcionam como contas básicas de poupança com juros. Dependendo do provedor com o qual seu empregador trabalha, você pode direcionar seu dinheiro para contas de investimento da HSA, oferecendo diferentes opções de fundos mútuos, assim como faria com um plano 401(k). No entanto, a maioria das empresas exige que um membro invista uma quantia definida em uma conta HSA básica antes de direcionar o dinheiro para uma conta de investimento HSA. Nesse sentido, o HSA funciona como um 401(k) para despesas médicas. O dinheiro retirado para outros fins é tributado.

##Destaques

- Patrocínio não significa que um empregador contribua com fundos para os planos, embora eles possam igualar certas contribuições de funcionários.

- Esses planos são muitas vezes com vantagens fiscais para os funcionários.

- Os planos patrocinados pelo empregador referem-se a benefícios aos funcionários oferecidos por uma organização.

- Os empregadores instalam esses planos de benefícios para atrair e reter trabalhadores, além de receber incentivos fiscais e outros incentivos.
