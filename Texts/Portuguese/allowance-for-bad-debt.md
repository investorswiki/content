# Provisão para dívidas incobráveis
## O que é uma provisão para dívidas incobráveis?

Uma provisão para dívidas incobráveis é uma conta de avaliação usada para estimar o valor dos [recebíveis de uma empresa](/receivables) que podem ser incobráveis. Também é conhecido como [provisão para devedores duvidosos](/allowancefordoubtfulaccounts). Quando um mutuário deixa de pagar um empréstimo, a provisão para [devedores duvidosos](/baddebt) e o saldo a receber do empréstimo são reduzidos pelo [valor contábil](/bookvalue) do empréstimo.

## Como funciona uma provisão para dívidas incobráveis

Os credores usam uma provisão para dívidas incobráveis porque o [valor nominal](/facevalue) do total de [contas a receber de uma empresa](/accountsreceivable) não é o saldo real que é finalmente cobrado. Em última análise, uma parcela dos recebíveis não será paga. Quando um cliente nunca paga o valor do principal ou dos juros devidos sobre um recebível, a empresa deve eventualmente anulá-lo totalmente.

## Métodos de estimativa de provisão para dívidas incobráveis

Existem duas maneiras principais de calcular a provisão para dívidas incobráveis. Um método é baseado em vendas, enquanto o outro é baseado em contas a receber.

### Método de vendas

O método de vendas estima a provisão para devedores duvidosos como uma porcentagem das vendas a crédito conforme elas ocorrem. Suponha que uma empresa ganhe $ 1.000.000 em vendas a crédito, mas saiba por experiência que 1,5% nunca paga. Então, a estimativa do método de vendas da provisão para devedores duvidosos seria de $ 15.000.

### Método de contas a receber

O método de contas a receber é consideravelmente mais sofisticado e aproveita o [envelhecimento dos recebíveis](/accounts-receivable-aging) para fornecer melhores estimativas da provisão para devedores duvidosos. A ideia básica é que quanto mais tempo uma dívida não for paga, mais provável é que a dívida nunca seja paga. Nesse caso, talvez apenas 1% das vendas iniciais fosse adicionado à provisão para devedores duvidosos.

No entanto, 10% dos recebíveis que não foram pagos após 30 dias podem ser adicionados à provisão para devedores duvidosos. Após 90 dias, pode subir para 50%. Finalmente, as dívidas podem ser amortizadas após um ano.

## Requisitos para uma provisão para dívidas incobráveis

De acordo com os princípios contábeis geralmente aceitos ( [GAAP](/gaap) ), o principal requisito para uma provisão para devedores duvidosos é que ela reflita com precisão o histórico de cobranças da empresa. Se $ 2.100 de $ 100.000 em vendas a crédito não foram pagos no ano passado, então 2,1% é uma estimativa adequada do método de vendas da provisão para devedores duvidosos este ano. Esse processo de estimativa é fácil quando a empresa está operando há alguns anos. Novos negócios devem usar médias [do setor, regras práticas](/rule-of-thumb) ou números de outro negócio.

> Uma estimativa precisa da provisão para devedores duvidosos é necessária para determinar o valor real das contas a receber.

>

## Considerações padrão

Quando um credor confirma que um determinado saldo de empréstimo está inadimplente, a empresa reduz a provisão para devedores duvidosos. Também reduz o saldo a receber do empréstimo, porque a inadimplência do empréstimo não é mais simplesmente parte de uma estimativa de inadimplência.

## Considerações de ajuste

A provisão para devedores duvidosos sempre reflete o saldo atual dos empréstimos que devem entrar em inadimplência, e o saldo é ajustado ao longo do tempo para mostrar esse saldo. Suponha que um credor estime que US$ 2 milhões do saldo do empréstimo esteja em risco de inadimplência e que a conta de provisão já tenha um saldo de US$ 1 milhão. Então, a entrada de ajuste para despesas com dívidas incobráveis e o aumento na conta de provisão é um adicional de $ 1 milhão.
