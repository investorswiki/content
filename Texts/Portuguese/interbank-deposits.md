# Depósitos interbancários
## O que são depósitos interbancários?

O termo depósito interbancário refere-se a um acordo entre dois bancos em que um mantém fundos em uma [conta](/account) de outra instituição. O acordo de depósito interbancário exige que o banco detentor abra uma [conta a pagar](/due-to-account) para o outro. Esta é uma conta do [razão geral](/generalledger) com fundos a pagar a outra parte. No arranjo, o banco correspondente é quem aguarda o depósito.

## Entendendo os Depósitos Interbancários

Os depósitos interbancários fazem parte do [mercado interbancário](/interbankmarket). O mercado interbancário é um sistema usado por bancos e outras instituições financeiras para negociar moedas. Este sistema exclui [investidores de varejo](/retailinvestor) — indivíduos que compram e vendem títulos para sua conta pessoal em vez de para outra empresa ou organização — e outras partes comerciais menores. A maioria das negociações interbancárias realizadas no mercado é proprietária, o que significa que os bancos o fazem entre e entre si. Há casos, porém, em que esse tipo de operação bancária ocorre para grandes clientes institucionais.

> O sistema interbancário exclui pequenos investidores de retalho e outros pequenos parceiros comerciais.

>

No mercado interbancário, os bancos tomam emprestado e emprestam dinheiro entre si para administrar a [liquidez](/liquidity) e cumprir as [exigências de reservas](/requiredreserves) que os reguladores impõem a eles. A exigência de reserva é a quantidade de dinheiro que um banco deve manter em seus cofres. Depósitos, assim como empréstimos, estão entre os muitos tipos de transações que ocorrem entre bancos que os ajudam a cumprir essas condições. Essas transações também proporcionam ao mercado uma grande liquidez.

Quando dois bancos fazem um acordo para um depósito interbancário, o banco titular cria uma conta a pagar para o banco correspondente – a instituição que faz o depósito. A conta devida é uma conta de depósito, também conhecida como conta a pagar.

Os bancos usam uma taxa de juros especial em depósitos e empréstimos de curto prazo. Essa taxa é conhecida como [taxa interbancária](/interbankrate). A taxa interbancária depende do vencimento, das condições de mercado e das [classificações de crédito](/creditrating) das instituições envolvidas. A Intercontinental Exchange [London Interbank Offered Rate](/libor) (ICE LIBOR) é uma taxa de [referência](/benchmark),. que alguns dos principais bancos do mundo cobram uns dos outros por empréstimos de curto prazo.

## Considerações Especiais

Conforme mencionado acima, o banco para o qual a conta em dívida é mantida é referido como o banco correspondente. Esta designação é geralmente realizada para depósitos que ocorrem entre bancos domésticos. Mas os termos mudam quando o [banco correspondente](/correspondent-bank) é uma instituição estrangeira. Nesse caso, a conta devida é uma conta [nostro](/nostroaccount) — derivada da palavra nosso em latim — para o banco que detém o depósito. Simplificando, esta é uma conta mantida por um banco em moeda estrangeira em outra instituição. Isso está em contrato com uma conta [vostro](/vostroaccount) — a palavra latina para seu — para o banco correspondente estrangeiro. Uma conta vostro é o termo que o banco usa para descrever contas que outras empresas têm em seus livros em sua moeda local. Assim, o banco correspondente chamará sua conta no banco titular de conta nostro, enquanto o banco titular a chamará de conta vostro.

Segue um exemplo para facilitar o entendimento. Digamos que o Banco A faça um depósito interbancário no Banco B, que está em um país diferente. A conta é chamada de conta nostro - nossa conta em seu livro-razão - para o Banco A, enquanto é uma conta vostro ou sua conta para o Banco B.

## Destaques

- Um depósito interbancário é um acordo entre dois bancos em que um mantém fundos em uma conta de outra instituição.

- A maioria das negociações interbancárias realizadas no mercado é proprietária - os bancos o fazem entre e para o outro.

- O acordo exige que o banco titular abra uma conta a pagar para o outro.
