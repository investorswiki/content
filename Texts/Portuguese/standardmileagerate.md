# Taxa de milhagem padrão
## Qual é a taxa de quilometragem padrão?

A taxa de quilometragem padrão é uma taxa definida pelo IRS que um contribuinte pode usar para calcular os custos dedutíveis de impostos de operar seu veículo para determinados fins. As taxas de milhagem padrão para 2017 são: 53,5 centavos por milha para milhas de negócios percorridas; 17 centavos por milha para fins médicos ou para mudança; e 14 centavos por quilômetro rodado servindo instituições de caridade.

## Definição mais profunda

A cada ano, o IRS define uma taxa de milhagem padrão. As milhas dedutíveis do imposto devem ter sido percorridas para uma ou mais das seguintes finalidades:

- O negócio

- Serviço de caridade

- Médico

- Mudança

Se um contribuinte optar por usar a taxa de quilometragem padrão para calcular o custo de seu veículo para o ano fiscal, ele não poderá deduzir outras despesas relacionadas à condução, como trocas de óleo e outros custos de manutenção de rotina. Muitos contribuintes optam por deduzir a quilometragem padrão porque é mais simples do que rastrear e registrar recibos de gás e manutenção.

Para reivindicar a quilometragem padrão de um veículo, o contribuinte deve possuí-lo ou alugá-lo. Um contribuinte pode reivindicar a taxa de quilometragem padrão para até quatro carros. Cinco ou mais carros se enquadram na categoria de operação de frota.

## Exemplo de taxa de quilometragem padrão

Tammy possui um Honda Accord 2014 e o usa para fins comerciais. Ela reivindica a quilometragem que ela coloca no veículo como uma dedução quando ela declara seus impostos. Para reivindicar a milhagem de negócios, Tammy registra as milhas que dirige para o trabalho em um livreto que mantém no porta-luvas do carro.

Quando ela declarar seus impostos de 2017, Tammy usará a taxa de quilometragem padrão para dirigir a negócios, que é de US$ 0,535. Supondo que Tammy dirija 3.000 milhas para fins comerciais durante 2017, sua dedução total de milhagem padrão seria de US$ 1.605 (0,535 x 3.000).

## Destaques

- As taxas são revistas anualmente pelo IRS.

- A dedução para uso de veículo pessoal como despesa de mudança foi eliminada, exceto para militares da ativa.

- Existem três taxas de milhagem padrão usadas para calcular a dedução, uma para uso de um veículo pessoal a negócios, para fins beneficentes e para fins médicos.
