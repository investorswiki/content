# Contrato Construir-Operar-Transferir
## O que é um contrato Build-Operate-Transfer (BOT)?

Um contrato build-operate-transfer (BOT) é um modelo usado para financiar grandes projetos, normalmente projetos de infraestrutura desenvolvidos por [meio de parcerias público-privadas](/public-private-partnerships).

O esquema BOT refere-se à concessão inicial de uma entidade pública, como um governo local, a uma empresa privada para construir e operar o projeto em questão. Após um prazo definido, normalmente duas ou três décadas, o controle sobre o projeto é devolvido ao ente público.

## Como funcionam os contratos de construção-operação-transferência

Sob um contrato de construção-operação-transferência (BOT), uma entidade – geralmente um governo – concede uma concessão a uma empresa privada para financiar, construir e operar um projeto. A empresa opera o projeto por um período de tempo (talvez 20 ou 30 anos) com o objetivo de recuperar seu investimento, então transfere o controle do projeto para o governo.

Os projetos BOT são normalmente projetos de infraestrutura greenfield de grande escala que, de outra forma, seriam financiados, construídos e operados exclusivamente pelo governo. Exemplos incluem uma rodovia no Paquistão, uma estação de tratamento de águas residuais na China e uma usina nas Filipinas.

Em geral, os empreiteiros BOT são empresas de propósito específico formadas especificamente para um determinado projeto. Durante o período do projeto – quando o empreiteiro está operando o projeto que construiu – as receitas geralmente vêm de uma única fonte, um comprador offtake. Esta pode ser uma empresa governamental ou estatal.

Os contratos de compra de energia, nos quais uma concessionária do governo atua como offtaker e compra eletricidade de uma usina de propriedade privada, são um exemplo desse arranjo. Sob uma concessão tradicional, a empresa venderia diretamente aos consumidores sem intermediário do governo. Os acordos BOT geralmente estipulam preços mínimos que o offtaker deve pagar.

## Variações no BOT

Existem várias variações do modelo BOT básico. Nos contratos de construção, operação e transferência (BOOT), o empreiteiro é o proprietário do projeto durante o período do projeto. Nos contratos de construção-arrendamento-transferência (BLT), o governo arrenda o projeto do empreiteiro durante o período do projeto e se encarrega da operação. Outras variações têm o projeto do empreiteiro, bem como a construção do projeto. Um exemplo é um contrato de design-build-operate-transfer (DBOT).
