# Protocolo Interledger
## O que é o Protocolo Interledger?

O protocolo Interledger é um protocolo [blockchain](/blockchain) usado para pagamentos em diferentes redes de pagamento. O protocolo [de código aberto](/open-source) conecta livros de dois ou mais bancos diferentes, removendo intermediários e autoridades centrais do sistema. Promete reduzir os custos e o tempo necessário para processar transações internacionais.

O Interledger é utilizado pelo Ripple Labs para conectar sistemas bancários através das fronteiras, onde o token [Ripple](/ripple-cryptocurrency) (XRP) funciona como uma camada de liquidação padronizada entre bancos globais, tornando o Ripple um pouco semelhante a um serviço [hawala digital](/hawala).

## Entendendo o Protocolo Interledger

O processo atual para transferências internacionais entre bancos envolve a navegação de vários protocolos de pagamento usados pelos livros digitais para processar transações. Os protocolos se comunicam usando conectores, mas as ferramentas e padrões usados nesses sistemas são fragmentados.

Por exemplo, o conjunto atual de conectores não possui modos padronizados para comunicação. A introdução de bancos intermediários para facilitar as transações entre entidades que não têm uma relação direta entre si complica ainda mais o processo. Isso ocorre porque ele multiplica o número de saltos para concluir uma transação, tornando-a mais cara e demorada. Também aumenta os riscos de segurança porque os pagamentos precisam passar por vários sistemas para chegar ao destinatário final.

O Interledger Protocol usa o conceito de [escrow criptográfico](/escrow) para permitir a passagem de fundos através de conectores ou nós na rede. O whitepaper do protocolo Interledger define o depósito criptográfico como o equivalente financeiro de um protocolo de confirmação de duas fases. O último protocolo consiste em duas etapas:

1. O primeiro passo consiste em definir um conjunto de condições para que uma transação avance ou aborte.

1. A segunda etapa define o processamento da transação uma vez que as condições sejam atendidas.

O depósito criptográfico para ledgers é o bloqueio condicional de fundos entre duas partes. Os fundos são liberados somente após determinadas condições, incluindo as baseadas no tempo para que a transação ocorra. Caso contrário, a transação é anulada. No final de uma transação de pagamento bem-sucedida, o remetente recebe um recibo criptográfico do destinatário. Alternativamente, os fundos garantidos são devolvidos às respectivas partes.

## Modo Atômico vs. Modo Universal

O protocolo Interledger pode ser implementado em dois modos: modo atômico e modo universal. No modo atômico, os notários são incorporados ao sistema. Eles são um grupo ad-hoc que é usado para verificar e validar transações. Normalmente, os modos atômicos ocorrem entre nós de conectores confiáveis entre bancos ou empresas de serviços financeiros que podem se relacionar entre si .

O modo universal não requer notários e pode funcionar entre conectores não confiáveis. Ele usa a criptomoeda interna da Ripple, XRP, para facilitar as transferências. A transferência é acompanhada por restrições de tempo. Se não ocorrer dentro de um determinado prazo, a transação é anulada .

## Destaques

- Interledger é um conjunto de protocolos de código aberto para enviar pagamentos em diferentes livros.

- Com base na tecnologia blockchain, o objetivo é remover intermediários para criar uma rede de pagamentos interbancários segura, descentralizada e transfronteiriça.

- O Interledger não está vinculado a uma única empresa, blockchain ou moeda, embora seja apresentado com destaque com a criptomoeda Ripple.
