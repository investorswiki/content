# eXtensible Business Reporting Language (XBRL)
## O que é a Linguagem Extensível de Relatórios de Negócios?

XBRL ou eXtensible Business Reporting Language é um padrão de software desenvolvido para melhorar a forma como os dados financeiros são comunicados, facilitando a compilação e o compartilhamento desses dados. Notavelmente, eXtensible Business Reporting Language é uma implementação de XML (extensible markup language), que é uma especificação usada para organizar e definir dados online.

O XBRL usa tags para identificar cada parte dos dados financeiros, o que permite que eles sejam usados programaticamente por um programa compatível com XBRL. XBRL permite fácil transmissão de dados entre empresas.

## Entendendo a Linguagem Extensível de Relatórios de Negócios

Imagine que você está vendo as [demonstrações financeiras de uma empresa](/financial-statements) [on](/financial-statements) -line no site da empresa. Tradicionalmente, essas declarações seriam simplesmente em texto simples. Se você quisesse colocar esses números em um arquivo de planilha para executar a análise dos extratos, teria que digitar manualmente ou copiar e colar cada conta e o número correspondente na planilha.

No entanto, se os dados no site estivessem disponíveis em eXtensible Business Reporting Language (XBRL), você poderia simplesmente converter esses dados do site em um programa de planilha (geralmente instantaneamente) compatível com XBRL.

Devido à natureza padronizada das etiquetas de identificação e do próprio idioma, os dados financeiros de um país, que estabeleceu padrões contábeis como o US [GAAP](/gaap),. podem ser facilmente compilados nos padrões contábeis aceitos de outro país, mesmo que sejam drasticamente diferentes. O relatório de dados financeiros em XBRL não é exigido por todas as empresas, mas como se tornou predominante, foi sugerido que não demorará muito para que todas as empresas tenham que relatar seus dados financeiros neste idioma. iXBRL, onde **i** significa inline é uma atualização que permite que metadados XBRL sejam incorporados em um documento HTML.

O XBRL foi desenvolvido em 1998 com a versão 1.0 pelo American Institute of Certified Public Accountants ( [AICPA](/american-institute-of-certified-public-accountants) ). A versão mais recente dos padrões, v2.1, foi formalizada no ano de 2003. Embora o padrão v2.1 tenha permanecido estável desde então, vários módulos XBRL foram desenvolvidos que podem ser plugados para obter novas funcionalidades ou operabilidade.

##Destaques

- XBRL é baseado em codificação XML e é uma forma padronizada de transmissão de registros financeiros em todo o mundo.

- eXtensible Business Reporting Language (XBRL) é uma estrutura global disponível gratuitamente de padrões contábeis usados para troca de informações comerciais.

- A versão mais atual do XBRL, v2.1, foi publicada em 2003 e permaneceu estável. Vários módulos opcionais existem como complementos.
