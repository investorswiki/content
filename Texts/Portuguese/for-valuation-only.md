# Apenas para avaliação (FVO)
## O que é apenas para avaliação (FVO)

For Valuation Only (FVO) é uma notação incluída em uma [cotação nominal](/nominalquotation) de um título. Os formadores de mercado usam cotações de FVO para ajudar a estabelecer o valor de um título. Quando uma notação FVO aparece na frente de uma cotação de preço, indica que a cotação é meramente para fins informativos, e não uma oferta da parte emissora.

## Entendimento apenas para avaliação (FVO)

For Valuation Only (FVO) aparecerá como uma notação em uma cotação de preço nominal de um título como cortesia aos investidores, indicando que não é um convite para negociação. Tais cotações nominais incluirão SAV ou FYI.

O objetivo de uma cotação nominal é permitir que um trader avalie uma participação atual onde pode ser difícil estabelecer uma referência atual e indicar um preço para fins informativos sem obrigar um corretor a entrar em uma negociação. Essas cotações são úteis para determinar uma posição de margem, fornecendo informações necessárias para determinar o valor atual de um determinado ativo, mas não servem como convites para negociação. As cotações nominais diferem das [cotações firmes](/firmquote),. que são convites para negociar a preços firmes e não estão sujeitas a cancelamento.

## Como as Cotações Nominais e as Cotações Firmes Diferem

Tanto as cotações nominais quanto as cotações firmes são ferramentas-chave dos formadores de mercado, tanto as corretoras quanto os intermediários individuais, cujo objetivo é permitir o fluxo suave dos mercados financeiros. Como os corretores e formadores de mercado lidam com pedidos para seus clientes, bem como para suas próprias contas, eles devem cumprir as regras [da Securities and Exchange Commission](/sec) sobre publicação de cotações e tratamento de pedidos de clientes, sob o [Securities Exchange Act de 1934](/seact1934).

Sob essas regras, quando um corretor publica uma cotação firme, ela não é negociável, e o corretor é obrigado a executar ordens ao preço publicado. Qualquer corretor que não honrar o lance cotado está violando os regulamentos do setor. Essa violação é conhecida como [recuar](/backingaway).

A notação FVO permite que um trader forneça informações de avaliação em uma cotação nominal, respeitando as regras da SEC. Os corretores que anotam um preço de títulos com FVO garantem que todas as partes entendam que a cotação foi dada por cortesia e não representa um convite para negociar.

Por exemplo, uma cotação nominal do FVO pode ser apresentada quando um trader considera comprar um contrato em uma bolsa de futuros, apenas para descobrir que nenhum formador de mercado fez uma oferta firme e, portanto, um preço para o contrato não foi estabelecido por um longo período de tempo. Nesse caso, o trader pode solicitar uma cotação nominal para ter uma noção do que os formadores de mercado podem oferecer nas condições atuais antes de decidir se deve ou não buscar um contrato real. Tal cotação seria necessariamente anotada FVO ou FYI.

No contexto de uma cotação nominal, o FVO não deve ser confundido com a Fair Value Option, que se refere a uma ferramenta contábil para avaliar os valores dos instrumentos financeiros de um negócio.
