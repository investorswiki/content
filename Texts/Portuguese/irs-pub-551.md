# Publicação IRS 551 - Base de Ativos
## O que é a publicação 551?

A publicação 551 do IRS, intitulada Base de ativos, é um documento informativo publicado pelo [Internal Revenue Service (IRS) que descreve como](/irs) determinar a [base de custo](/costbasis) para investimentos, imóveis e ativos comerciais. ganho ou perda é realizado a partir de uma venda e representa o custo original do investimento ou propriedade. Também é usado para determinar a depreciação e amortização de uma propriedade .

A publicação 551 do IRS pode ser encontrada no site do IRS.

## Compreendendo a publicação 551 do IRS

A base de custo para uma propriedade é normalmente o custo de compra, no entanto, a base pode aumentar ao longo do tempo se o proprietário fizer melhorias na propriedade. Para investimentos, como ações e títulos, a base de custo também inclui taxas de negociação .

Para fins fiscais, o método usado pelo Internal Revenue Service (IRS) é [o primeiro a entrar, primeiro a sair](/fifo) (FIFO) para aqueles familiarizados com o método de rastreamento de estoque para empresas. Em outras palavras, quando uma venda é feita, a base de custo da compra original seria usada primeiro e seguiria uma progressão no histórico de compras .

A partir de 2018, as pequenas empresas não estão sujeitas às regras uniformes de capitalização se a receita bruta anual média for de US$ 25 milhões ou menos para os 3 anos fiscais anteriores e a empresa não for um abrigo fiscal. As regras de capitalização uniforme especificam os custos que você adiciona à base em determinadas circunstâncias.﻿Mais informações sobre como estabelecer a base de custo para investimentos podem ser encontradas na [Publicação 550 do IRS](/irs-pub-550).

## Base de custos de relatórios fiscais

Embora as corretoras sejam obrigadas a informar ao IRS o preço pago pelos títulos tributáveis, para alguns [títulos](/security),. como os mantidos por um longo período de tempo ou os transferidos de outra corretora, a base de [custo histórico](/historical-cost) precisará ser fornecida pelo investidor . Tudo isso coloca o ônus de relatórios precisos de base de custos sobre os investidores.

Determinar a base de custo inicial de títulos e [ativos financeiros](/financialasset) para apenas uma compra inicial é muito simples. Na realidade, pode haver compras e vendas subsequentes à medida que um investidor toma decisões para implementar [estratégias de negociação específicas](/trading-strategy) e maximizar o potencial de lucro para impactar um portfólio geral. Com todos os vários tipos de investimentos, incluindo ações, títulos e opções, calcular a base de custo com precisão para fins fiscais pode ser complicado.

Em qualquer transação entre um comprador e um vendedor, o preço inicial pago em troca de um produto ou serviço será qualificado como [base de custo](/costbasis). A base de custo de capital é o custo total para um investidor; este valor inclui o preço de compra por ação mais o valor [reinvestido](/reinvestment) [dividendos](/dividend) e comissões. A base de custo de capital não é apenas necessária para determinar quanto, se houver, os impostos precisam ser pagos em um investimento, mas é fundamental para rastrear os ganhos ou perdas no investimento para tomar decisões informadas de compra ou venda .

## Destaques

- O IRS exige o método first-in, first-out (FIFO) para cálculo de impostos e base de custo, o que significa que as participações mais antigas são vendidas primeiro para fins fiscais.

- A base de custo é o valor original ou preço de compra de um ativo ou investimento para fins fiscais.

- A publicação 551 da Receita Federal informa aos contribuintes como obter a base de custo de um bem ou investimento.

- Base é utilizada para calcular a alíquota do imposto sobre ganhos de capital, que é a diferença entre a base de custo do ativo e o valor atual de mercado.
