# Código Identificador de Mercado (MIC)
## O que é um Código Identificador de Mercado (MIC)?

Um código identificador de mercado (MIC) é um código de quatro caracteres usado para identificar [bolsas de](/stockmarket) [valores](/stockmarket) e outras bolsas de negociação dentro de sistemas de computador de referência e negociação global. A primeira letra de qualquer MIC é alocada aleatoriamente seguida por um código alfanumérico de três dígitos para o mercado em que a negociação ocorre. Os remetentes podem propor uma combinação de quatro caracteres, mas a Autoridade de Registro permanece responsável pela atribuição. Uma vez alocado, um código MIC não pode ser modificado .

O código MIC é usado para processar e liberar negociações e está sendo empurrado para a aceitação global à medida que as indústrias de valores mobiliários avançam para o processamento direto (STP). A [Bolsa de Valores de Londres](/lse) usa MICs como parte de seus sistemas de identificação de segurança [SEDOL](/sedol),. que é uma alternativa ao sistema de identificação CUSIP baseado nos EUA .

## Entendendo o Código Identificador de Mercado (MIC)

[Direto através do processamento](/straightthroughprocessing),. uma abordagem conceitual para melhorar a velocidade de processamento de transações é considerada o santo graal da negociação global de títulos. Para que isso seja possível, será necessário haver uma riqueza de códigos consistentes para o mercado de origem, moeda e identificação de títulos. Tal como está agora, existem vários sistemas diferentes usados por diferentes países e para diferentes tipos de títulos.

Um padrão global terá que surgir ao longo do tempo, e todos os softwares e sistemas de processamento terão que apoiá-lo, fazendo as mudanças técnicas necessárias ao longo do caminho. Esse processo levará tempo à medida que as indústrias de valores mobiliários avançam em direção ao objetivo final de um mercado de "qualquer título, em qualquer lugar, a qualquer hora".

## Como os códigos identificadores de mercado são usados

Os MICs foram apresentados pela ISO como um padrão internacional para identificar mercados, plataformas de negociação e trocas para permitir o processamento automatizado de negociações. Com os MICs sendo usados como o padrão de identificação universal, pode ser um passo significativo para aproximar o processamento direto de ser realizado além da fase de conceito.

Existem diferentes tipos de MIC. Um segmento de mercado MIC identifica uma seção de uma das entidades abrangidas pelo código que se especializa em um ou mais instrumentos específicos ou é regulamentada de forma diferente. Os MICs do segmento de mercado foram instituídos para maior precisão, de acordo com a ISO .

Para cada MIC de segmento de mercado, há um MIC pai, também conhecido como MIC operacional .

Um MIC operacional, por sua vez, identifica a entidade que opera uma bolsa, plataforma de negociação, mercado regulamentado ou não regulamentado ou um mecanismo de relatório de negociação em um país específico .

As organizações de mercado podem solicitar um MIC enviando uma solicitação à autoridade de registro. Normalmente, um MIC de segmento de mercado é gerado quando a demanda justifica a necessidade de identificar o segmento. Dependendo das circunstâncias, pode acontecer que apenas segmentos específicos de um mercado recebam identificação com um segmento de mercado MIC .

##Destaques

- Um código identificador de mercado (MIC) é usado para especificar os mercados de ações e outras bolsas nas quais os títulos são negociados.

- MICs são usados como um padrão internacional e universal de identificação, com códigos definidos pela International Organization for Standardization (ISO).

- O código é único, inclui quatro caracteres, começando com um caractere alfanumérico atribuído aleatoriamente, seguido por um código de três dígitos para indicar o mercado, como XNAS para o mercado Nasdaq.
