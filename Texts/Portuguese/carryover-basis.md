# Base de Transferência
## O que é uma base de transferência?

Uma base de transporte é um método para determinar a [base fiscal](/costbasis) de um ativo quando ele é transferido de um indivíduo para outro. Uma base de transferência é frequentemente usada quando uma parte deixa ativos ou propriedades para outra pessoa como presente. Nessa situação, a base geralmente permanece a mesma de quando o doador detinha o ativo, mas a base pode ser ajustada para contabilizar quaisquer impostos sobre doações que foram pagos.

## Entendendo a base de transição

A base de transporte difere de uma base [step-up](/stepupinbasis). Uma base de transferência é usada durante a vida do doador, enquanto uma base de aumento é usada quando um ativo é herdado após o falecimento do doador. Em um cenário de base step-up, o valor dos ativos sendo transferidos é ajustado ao seu valor de mercado atual.

A base de custo de um investimento é o valor total originalmente investido, mais quaisquer [comissões](/commission) ou taxas envolvidas na compra. Isso pode ser descrito em termos do valor em dólares do investimento ou do preço efetivo por ação do investimento.

Determinar a base de custo correta de um investimento - também conhecida como base fiscal - é importante especialmente se você reinvestiu dividendos e [distribuições de ganhos de capital](/capitalgainsdistribution) em vez de receber os ganhos do investimento em dinheiro. Quando você reinveste dividendos ou distribuições, a base tributária de seu investimento aumenta. Esse aumento deve ser contabilizado para que você possa relatar ganhos de capital menores e, portanto, pagar menos impostos. Se você não informar a base tributária mais alta, poderá acabar pagando impostos mais altos.

Quando as ações são dadas a você como presente, sua base de custo é a base de custo do detentor original do ativo que lhe deu o presente. Se as ações estão sendo negociadas a um preço mais baixo do que quando as ações foram doadas, a taxa mais baixa é a base de custo. Se as ações foram dadas a você como parte de uma herança, a base de custo das ações para o herdeiro é o [preço](/market-price) de mercado das ações na data da morte do proprietário original.

No processo de planejamento patrimonial, a base de transferência ajuda a determinar o valor inicial da propriedade, portanto, a base de transferência ajuda a determinar a [taxa de imposto](/taxrate) que os herdeiros de uma propriedade devem pagar sobre [ganhos de capital](/capitalgain) quando vendem quaisquer ativos associados a essa propriedade.

## Base de Transferência e Impostos sobre Presentes

[impostos sobre doações](/gifttax) são um componente crítico na determinação da base de transferência de um ativo. Este é um imposto federal que se aplica a cenários em que a parte receptora não paga ao doador o valor total do presente (embora possa pagar um valor menor).

O doador do presente é aquele que arca com o peso do imposto sobre doações. Em geral, presentes para o cônjuge ou para uma organização política, ou presentes com valor inferior à exclusão anual do imposto sobre doações, juntamente com despesas médicas e educacionais, estão excluídos do imposto sobre doações.

Para 2021, o imposto sobre doações máximo é de US$ 15.000 por pessoa por ano (US$ 16.000 para 2022). Isso significa que um indivíduo pode presentear outro indivíduo com US$ 15.000 ou menos por ano sem incorrer em um imposto sobre doações.

O imposto sobre doações difere do [imposto sobre herança](/estatetax),. que é cobrado sobre a parte herdada de um herdeiro de uma [propriedade](/estate). Em 2021, o limite de exclusão para o valor da propriedade foi combinado de ativos brutos e presentes tributáveis anteriores superiores a US$ 11,7 milhões (US$ 12,06 milhões em 2022). Isso significa efetivamente que uma propriedade de US $ 11,7 milhões não seria obrigada a apresentar uma declaração de [imposto](/taxreturn) e estaria isenta de pagar o imposto de propriedade.

Embora o imposto sobre herança incida principalmente sobre os bens deixados aos herdeiros, ele não se aplica à transferência de bens para um cônjuge sobrevivente. O direito dos cônjuges de deixar qualquer quantia um para o outro é conhecido como [dedução conjugal ilimitada](/unlimited-marital-deduction).

## Destaques

- Em geral, a base de transporte é a mesma que a base de custo original.

- Uma base de transporte refere-se à base de custo de um ativo recebido de outro indivíduo.

- Se o bem foi transferido como doação ou por herança afetará sua situação tributável e cálculo de base.
