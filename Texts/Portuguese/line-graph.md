# Gráfico de linha
## O que é um gráfico de linhas?

Um gráfico de linhas — também conhecido como gráfico de linhas ou gráfico de linhas — é um gráfico que usa linhas para conectar pontos de dados individuais. Um gráfico de linhas exibe valores quantitativos em um intervalo de tempo especificado. Em finanças, os gráficos de linhas são comumente usados para descrever a ação histórica do preço de um ativo ou título.

Os gráficos de linha podem ser comparados com outras visualizações de dados, incluindo gráficos de [barras , gráficos](/barchart) de pizza e (na negociação) [gráficos de velas](/candlestick),. entre outros.

## Entendendo os gráficos de linha

Os gráficos de linha usam "marcadores" de pontos de dados, que são conectados por linhas retas. Esses pontos de dados, conectados por linhas retas, auxiliam na visualização. Embora os gráficos de linha sejam usados em muitos campos diferentes para diferentes finalidades, eles são especialmente úteis quando é necessário criar uma representação gráfica das alterações nos valores ao longo do tempo.

Gráficos de linha são frequentemente usados em finanças para criar representações visuais de valores ao longo do tempo, incluindo mudanças nos preços de [títulos](/security),. planilhas de receitas da empresa e históricos dos principais [índices de ações](/index). Eles também são úteis para comparar títulos diferentes.

No investimento, especificamente no que diz respeito à área de [análise técnica](/technicalanalysis),. os gráficos de linhas são utilizados pelos investidores para visualizar tendências, o que pode auxiliá-los muito em suas análises.

Existem algumas limitações para gráficos de linha. Por exemplo, gráficos de linha geralmente perdem clareza quando há muitos pontos de dados. Também é fácil manipulá-los visualmente para obter certos efeitos. Por exemplo, o grau aparente de mudança pode ser manipulado visualmente ajustando o intervalo de pontos de dados nos eixos.

> Os gráficos de linha podem ser construídos manualmente ou usando software, como o Microsoft Excel, que melhora muito a velocidade e a precisão do produto final.

## Construindo um gráfico de linha

Os gráficos de linha consistem em dois eixos: eixo x (horizontal) e eixo y (vertical). Cada eixo representa um tipo de dados diferente e os pontos em que eles se cruzam são (0,0). O eixo x é o eixo independente porque seus valores não dependem de nada medido. O eixo y é o eixo dependente porque seus valores dependem dos valores do eixo x.

Cada eixo deve ser rotulado de acordo com os dados medidos ao longo desse eixo. Em seguida, cada eixo deve ser dividido em incrementos apropriados (por exemplo, dia um, dia dois, etc.). Por exemplo, se medir as mudanças nos preços de uma ação nas duas semanas anteriores, o eixo x representaria o tempo medido (dias de negociação dentro do período) e o eixo y representaria os preços das [ações .](/stock)

Ao usar gráficos de linha para rastrear o preço de uma ação, o ponto de dados mais comumente usado é o [preço de fechamento](/closingprice) da ação.

Por exemplo, suponha que, no primeiro dia de negociação, o preço de uma determinada ação era de $ 30, resultando em um ponto de dados em (1, $ 30). No segundo dia de negociação, o preço da ação era de $ 35, resultando em um ponto de dados em (2, $ 35).

Cada ponto de dados é plotado e conectado por uma linha que mostra visualmente as mudanças nos valores ao longo do tempo. Se o valor da ação aumentasse diariamente, a linha se inclinaria para cima e para a direita. Por outro lado, se o preço da ação estivesse diminuindo constantemente, a linha se inclinaria para baixo e para a direita.

## PERGUNTAS FREQUENTES

Os gráficos de linha são úteis em finanças porque são muito eficazes na criação de representações visuais de tendências ao longo do tempo. Por esse motivo, eles são frequentemente usados para descrever o desempenho de uma ação em um período específico de tempo.

### O que é o gráfico de linhas na teoria dos grafos?

A teoria dos grafos é uma disciplina matemática. A teoria dos grafos estuda especificamente gráficos, estruturas matemáticas que são usadas para modelar relações de pares entre objetos. o gráfico de arestas, o gráfico de intercâmbio, o gráfico adjunto e o gráfico derivado. Esta é a definição formal de um gráfico de linha: Dado um gráfico (**G)**, seu gráfico de linha (**L**(** G**)) é um grafo tal que cada vértice de **L**(**G**) representa uma aresta de **G,** e dois vértices de **L**(**G** ) são adjacentes se suas arestas correspondentes compartilham uma extremidade comum em **G**. Um gráfico de linha é o gráfico de interseção das arestas de **G**, representando cada aresta pelo conjunto de suas duas extremidades.

### Como você explica um gráfico de linhas?

Um gráfico de linha é uma exibição gráfica de informações que mudam continuamente ao longo do tempo. Dentro de um gráfico de linhas, existem vários pontos de dados conectados entre si por uma linha reta que revela uma mudança contínua nos valores representados pelos pontos de dados.

### Para que serve um gráfico de linha?

Os gráficos de linha são usados para rastrear alterações em diferentes períodos de tempo. Os gráficos de linha também podem ser usados como uma ferramenta de comparação: para comparar as mudanças no mesmo período de tempo para mais de um grupo.
