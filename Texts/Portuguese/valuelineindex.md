# Índice Composto de Linha de Valor
## O que é Índice Composto de Linha de Valor?

O Value Line Composite Index é um índice de ações contendo aproximadamente 1.700 empresas da [NYSE](/nyse),. [American Stock Exchange](/amex),. Nasdaq, Toronto e mercados de balcão. O Value Line Composite Index tem duas formas: O Value Line Geometric Composite Index (o índice original igualmente ponderado) e o Value Line Arithmetic Composite Index (um índice que reflete as mudanças se uma carteira detivesse quantidades iguais de ações). Esses índices são normalmente publicados na Value Line Investment Survey, criada por Arnold Bernhard, o fundador, e CEO da Value Line Inc.

## Entendendo o Índice Composto de Linha de Valor

A "Linha de Valor" onde o índice recebe seu homônimo refere-se a um múltiplo de fluxo de caixa que Bernhard sobreporia a um gráfico de preços para normalizar o valor de diferentes empresas. A Value Line é uma das mais respeitadas empresas de pesquisa de investimentos. Seu histórico de desempenho tem sido extremamente forte. Na verdade, os portfólios de modelos da empresa geralmente superam o mercado no [longo prazo](/longrun).

O Value Line Composite Index é composto pelas mesmas empresas do Value Line Investment Survey, excluindo os fundos fechados .

O número de empresas no Value Line Index flutua com base em fatores que incluem a adição ou saída das empresas nas próprias bolsas, fusões, aquisições, falências e as decisões de cobertura tomadas pela Value Line para o Value Line Index. As decisões da Value Line sobre quais empresas incluir são perseguidas com a intenção de criar uma ampla representação do mercado acionário norte-americano. Além disso, o número de empresas listadas em qualquer bolsa pode variar, pois uma empresa pode passar de uma bolsa para outra ou ser adicionada ou removida da lista. No entanto, a saída ou movimentação de empresas nas bolsas não são fatores na metodologia do Value Line Index, independentemente de serem empregados cálculos geométricos ou aritméticos.

## O Índice Composto Geométrico da Linha de Valor

Este é o índice original, introduzido em 30 de junho de 1961. É um índice igualmente ponderado usando uma média geométrica. A variação diária do preço do Value Line Geometric Composite Index é encontrada multiplicando-se a razão entre o preço de fechamento de cada ação e seu preço de fechamento anterior e elevando esse resultado ao recíproco do número total de ações .

## O Índice Composto Aritmético da Linha de Valor

Esse índice foi estabelecido em 1º de fevereiro de 1988, usando a média aritmética para imitar mais de perto a mudança no índice se você detivesse uma carteira de ações em quantidades iguais . a variação percentual diária de todas as ações e depois dividindo pelo número total de ações .

##Destaques

- Existem duas formas para o índice—o Índice Composto Geométrico da Linha de Valor e o Índice Composto Aritmético da Linha de Valor.

- O Geometric Composite Index tem peso igual, usa uma média geométrica e tem uma variação diária mais próxima da variação mediana do preço das ações.

- O Índice Composto Aritmético utiliza uma média aritmética, com a variação diária do índice refletindo uma carteira composta por ações em quantidades iguais .

- O índice Value Line Composite contém um mix de aproximadamente 1.700 ações dos principais índices do mercado norte-americano .
