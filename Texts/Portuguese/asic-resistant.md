# Resistente a ASIC
**Envio da comunidade** - Autor: **John Ma**

Resistente a ASIC é a propriedade de uma criptomoeda que é “imune” à mineração [ASIC .](/asic)

ASICs são [circuitos integrados](/integrated-circuit) criados para atender a um caso de uso específico, realizando uma tarefa de computação específica. No mundo das criptomoedas, os dispositivos ASIC são projetados para participar do processo de [mineração de](/mining) Bitcoin (ou outras criptomoedas). Como tal, o Bitcoin é um exemplo de criptomoeda que não pode ser considerada resistente a ASIC.

Uma criptomoeda resistente a ASIC tem seu protocolo e algoritmo de mineração configurados de tal forma que o uso de máquinas ASIC para minerar a moeda é impossível ou não traz nenhum benefício significativo quando comparado à mineração tradicional de GPU. Em alguns casos, usar ASICs em criptomoedas resistentes a ASIC pode ser ainda pior do que usar o hardware mais convencional.

Como a mineração envolve várias tentativas de encontrar uma solução para um tipo de problema matemático, o trabalho de um ASIC é realizar tantas tentativas quanto possível (ou seja, tantas funções de hash por segundo quanto possível). Isso significa que usar ASICs para minerar Bitcoin ou outras criptomoedas de Prova de Trabalho é muito melhor do que usar um hardware de uso geral, como uma placa de GPU.

No entanto, o processo de tornar uma criptomoeda resistente a ASIC é um jogo defensivo, que requer desenvolvimento e modificações contínuos. Isso se deve ao motivo pelo qual os designers e fabricantes de ASIC estão constantemente produzindo novos modelos de mineradores ASIC e, às vezes, os modelos mais novos são capazes de contornar a resistência ao ASIC de certas criptomoedas.

Vale a pena notar que as redes blockchain que dependem de outros métodos de obtenção de consenso (como PoS, dPoS e PoA) são resistentes ao ASIC por design. Em relação às criptomoedas Proof of Work (PoW), algumas são resistentes e outras não (depende do algoritmo de mineração implementado).
