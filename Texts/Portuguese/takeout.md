# Tirar
## O que é para viagem?

No contexto das finanças, o termo takeout pode se referir a:

1. Um empréstimo de longo prazo que substitui outro empréstimo, geralmente de curto prazo.

1. Uma gíria para a compra de uma empresa por meio de uma aquisição, fusão ou aquisição, tirando assim a empresa-alvo do jogo.

## Entendendo os empréstimos para viagem

Takeout é um termo que tem vários usos no setor financeiro, mas os dois principais usos para esse termo são como um tipo de financiamento ou compra de uma empresa.

Um empréstimo para viagem é um método de financiamento pelo qual um empréstimo adquirido posteriormente é usado para substituir o empréstimo inicial. Mais especificamente, um empréstimo para viagem, ou financiamento para viagem, é um financiamento de longo prazo que o credor promete fornecer em uma data específica ou quando critérios específicos para a conclusão de um projeto forem atendidos.

Empréstimos para viagem são comumente usados no desenvolvimento de propriedades. Um desenvolvedor pode garantir um empréstimo de curto prazo para demolir uma estrutura existente e pagar uma equipe para construir uma nova. Uma vez que a nova estrutura esteja pronta ou uma parte significativa dela esteja concluída, o desenvolvedor pode obter financiamento de longo prazo para pagar o empréstimo original.

### Empréstimo para viagem

Um [credor takeout](/take-out-lender) é uma instituição financeira que fornece empréstimos hipotecários de longo prazo para substituir o financiamento de curto prazo usado para financiar a compra de terrenos ou o desenvolvimento e construção de grandes edifícios, como imóveis comerciais.

Esses credores oferecem financiamento de longo prazo e taxas de juros mais baixas em troca de pagamentos de hipoteca, uma parte dos pagamentos de aluguel e [ganhos de capital](/capitalgain) se a propriedade for vendida.

Um [compromisso de retirada](/take-out-commitment) é uma garantia escrita por um credor para fornecer financiamento permanente para substituir um empréstimo de curto prazo em uma data futura especificada, se o projeto atingiu um determinado estágio.

## Retirada por aquisição

Takeout, como um termo coloquial, pode se referir à compra de uma empresa, seja por meio de uma [aquisição](/acquisition),. [fusão](/merger) ou outra forma de compra. A natureza do takeover não importa para que seja um takeout, e o termo é usado em todos os contextos. Assim, um takeout pode se referir a uma aquisição hostil, uma fusão amigável ou uma aquisição alavancada ou de gestão. O que importa é que a empresa-alvo seja "tirada do jogo".

Diz-se que uma empresa está " [em jogo](/inplay) " se for provável que seja adquirida no futuro ou se atualmente tiver ofertas de compradores. Uma retirada refere-se, portanto, à retirada da empresa do jogo, o que ocorre quando a aquisição foi finalizada (ou se o negócio não se concretizar).

##Destaques

- Uma aquisição takeout refere-se a uma empresa sendo retirada do jogo, o que ocorre quando o negócio foi finalizado.

- Um empréstimo para viagem, que é bastante comum em desenvolvimento imobiliário, é um financiamento de longo prazo que o credor promete fornecer em uma determinada data ou quando critérios específicos para a conclusão de um projeto forem atendidos.

- Takeout pode se referir a um empréstimo que substitui outro empréstimo ou, como uma gíria, à compra de uma empresa por meio de uma aquisição ou compra.
