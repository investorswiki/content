# Busca de golos
## O que é a busca de metas?

A busca de metas é o processo de encontrar o valor correto de entrada quando apenas a saída é conhecida. A função de busca de metas pode ser incorporada a diferentes tipos de programas de computador, como o Microsoft Excel.

## Entenda a busca de metas

A busca de metas é um termo geral usado para descrever o processo envolvido em descobrir seu valor de entrada com base em um valor de saída já conhecido. O processo envolve o uso de um operador específico em uma fórmula, que pode ser calculada usando um software de computador.

A busca de metas é uma das ferramentas usadas na "análise de hipóteses" em programas de computador. Uma [análise hipotética é um processo de alteração de valores em células (Microsoft Excel) para ver](/sensitivityanalysis) [como](/sensitivityanalysis) essas alterações afetarão os resultados da fórmula na planilha. Quando você está buscando uma meta, está realizando uma análise hipotética em um determinado valor ou na saída. Então, em essência, você estaria criando um cenário perguntando "e se a saída fosse X" - ou basicamente, uma situação de causa e efeito.

Para alguns dos problemas mais complexos, as pessoas costumam usar software de computador. Um programa de planilha como o Microsoft Excel tem uma ferramenta de busca de metas embutida. Ele permite que o usuário determine o valor de entrada desejado para uma fórmula quando o valor de saída já é conhecido. Esse recurso pode ajudar o usuário a determinar coisas como a [taxa de juros que](/interestrate) um mutuário precisa para se qualificar (a entrada) se ele souber apenas quanto pode pagar por mês (a saída).

Mas há uma ressalva ao usar o software de busca de metas: ele só funciona se houver um valor de entrada. Se você precisar descobrir dois ou mais valores de entrada, não funcionará. Se pegarmos o exemplo acima, se você quiser descobrir o valor total do empréstimo e o pagamento mensal, o software de busca de metas não funcionará. Você provavelmente precisaria de um complemento para descobrir várias variáveis.

### Como a busca por metas funciona no Microsoft Excel

Como mencionado acima, o software de busca de metas só funcionará se você já souber o valor de saída (ou o resultado), mas quiser determinar um valor de entrada. Aqui estão algumas etapas importantes que você pode seguir se quiser usar o recurso de busca de metas no programa:

- Abra uma nova planilha

- Rotule suas colunas. Isso tornará mais fácil para você ler tudo na planilha. Assim, usando o exemplo acima, a primeira coluna será “Valor do empréstimo”, a segunda coluna será “ [Prazo](/term) em meses”, a terceira será “Taxa de juros” e a última será “Pagamento”.

- Digite os valores que você conhece.

- Digite a fórmula para calcular a meta — neste caso, o pagamento. Você pode ignorar a taxa de juros por enquanto, que a fórmula assumirá como 0%.

Agora você terá que determinar a taxa de juros. Você pode fazer isso usando a função Goal Seek no Excel e inserir os valores que você já possui.

## Exemplo de busca de metas

Um [empreendedor em busca de objetivos](/entrepreneur) é alguém que usa a busca de objetivos para determinar como eles podem alcançar seus objetivos finais. Por exemplo, um empreendedor pode perguntar quanto precisará ganhar por hora para faturar US$ 100.000 em um ano. Eles sabem o valor de saída desejado – $ 100.000 – e, portanto, terão que trabalhar de volta para descobrir a entrada ideal. Isso significa que eles terão que determinar quantas horas poderão (ou estão dispostos a) trabalhar durante o ano e, portanto, quanto ganharão por hora para atingir sua meta.

## Destaques

- No entanto, o software Goal-seek só funciona se houver apenas um valor de entrada.

- A busca de metas é encontrar a entrada correta quando apenas a saída é conhecida.

- Isso geralmente é feito usando a função de busca de metas no Microsoft Excel ou análise de hipóteses por meio de programas de computador.
