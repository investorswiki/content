# Código Americano para Intercâmbio de Informações (ASCII)
## O que é o Código Americano para Intercâmbio de Informações (ASCII)?

O Código Americano para Intercâmbio de Informações, comumente chamado de ASCII, é uma linguagem de computador para texto que se tornou o padrão para transmissão de informações entre computadores em 1963.

O sistema ASCII traduz caracteres alfanuméricos e símbolos em um código legível por computador. O ASCII original era somente em inglês. Existem agora dois tipos de códigos ASCII; o código padrão que usa um sistema de codificação de sete bits e um código estendido que usa um sistema de oito bits.

Nas comunicações pela Internet, o ASCII foi sendo gradualmente substituído pelo padrão Unicode, que pode ser usado com qualquer idioma e é compatível com o ASCII. No entanto, o ASCII ainda é usado para inserir texto em computadores pessoais e dispositivos eletrônicos.

É pronunciado ASK-y.

## Entendendo o Código Americano para Intercâmbio de Informações (ASCII)

A primeira iteração do ASCII pode ser encontrada em um código telegráfico usado pelos serviços de dados da Bell para entregar mensagens a teleimpressoras.

ASCII foi o primeiro grande sistema de codificação de caracteres para processamento de dados e foi adotado como padrão para computação em 1963 e para transmissão pela Internet em 1969. Foi usado por muitos anos nos sistemas de computador de entrada de pedidos usados por corretores e corretores de Wall Street.

Ele foi gradualmente substituído como padrão para a Internet pelo Unicode Worldwide Character Standard (Unicode). No entanto, como os primeiros 128 caracteres do Unicode são iguais aos do ASCII, pode-se dizer que os dois sistemas coexistem.

> Você pode inserir um símbolo ASCII em um documento usando qualquer computador compatível com Windows mantendo pressionada a tecla ALT enquanto digita o número do código do caractere. Por exemplo, segurar a tecla ALT enquanto digita 156 lhe dará £, o sinal da libra britânica.

### O que há em ASCII

Antes de o ASCII ser adotado como padrão, cada fabricante de computador usava seu próprio código e algumas empresas usavam mais de um para produtos diferentes. Os computadores não conseguiam se comunicar uns com os outros.

Embora o ASCII tenha sido adotado como padrão pelos American National Standards em 1963, ele não foi totalmente implementado mesmo nos EUA até 1968, quando o presidente Lyndon B. Johnson determinou sua adoção pelo governo federal para garantir que os computadores do governo e a indústria de telecomunicações computadores podem se comunicar uns com os outros.

O ASCII originalmente continha 128 caracteres alfabéticos, números e símbolos e foi baseado no idioma inglês. Formulários estendidos foram adicionados na década de 1970 para acomodar outros idiomas.

Logo, várias versões do ASCII estavam em uso. Eventualmente incluiu 255 caracteres. Esta versão é conhecida como Extended ASCII.

Hoje, os usuários podem inserir símbolos ASCII ou Unicode, como um símbolo de copyright em documentos, pressionando a tecla alt e digitando o código de caractere no teclado numérico com a tecla NUM lock ativada.

##Destaques

- O ASCII continua a existir, mas foi amplamente substituído pelo Unicode, que pode ser usado para codificar qualquer idioma.

- O ASCII originalmente continha apenas 128 letras e símbolos em inglês, mas foi posteriormente expandido para incluir caracteres adicionais, incluindo aqueles usados em outros idiomas.

- O Código Americano para Intercâmbio de Informações, ou ASCII, é um formato de codificação de caracteres projetado para uso na transmissão de texto entre computadores.

##PERGUNTAS FREQUENTES

### O que é ASCII vs. Unicode?

O Unicode pode ser visto como uma versão universal do ASCII. ASCII é, afinal, o Código Americano para o Intercâmbio de Informações, e sua primeira iteração incluiu o alfabeto do idioma inglês e os símbolos usados no idioma. Versões posteriores incluíam símbolos usados em muitos outros idiomas. O Unicode, por outro lado, começa com os mesmos 128 símbolos usados em ASCII, mas as versões atuais contêm 144.697 caracteres.

### O que é ASCII?

O American Standard Code for Information Interchange, ou ASCII, é um formato de codificação de caracteres para a transmissão eletrônica de texto. Cada caractere é representado por um número único. A primeira versão do ASCII continha apenas 128 caracteres, representando as letras do alfabeto, maiúsculas e minúsculas, além de vários símbolos comumente usados, como a vírgula. Versões posteriores estenderam o ASCII para 255 caracteres, incluindo símbolos adicionais, como o Símbolo da libra britânica (£) e o ponto de interrogação de cabeça para baixo usado no texto em espanhol (¿).

### Quais são os caracteres ASCII?

Na tabela ASCII estendida, os códigos de caractere de 0 a 31 são códigos de controle, como início de texto e backspace. Os códigos de caractere de 32 a 127 são principalmente letras do alfabeto (maiúsculas e minúsculas) e símbolos como o asterisco e o cifrão. os códigos de 128 a 255 vêm em várias versões, cada uma delas refletindo acréscimos ao ASCII original somente em inglês, como um ponto de exclamação invertido, o sinal do iene e a vírgula georgiana.
