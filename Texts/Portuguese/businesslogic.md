# Logíca de negócios
## O que é lógica de negócios

A lógica de negócios são as regras ou [algoritmos personalizados](/algorithm) que lidam com a troca de informações entre um banco de dados e a interface do usuário. A lógica de negócios é essencialmente a parte de um programa de computador que contém as informações (na forma de regras de negócios) que definem ou restringem como uma empresa opera. Essas regras de negócios são políticas operacionais que geralmente são expressas em binários verdadeiros ou falsos. A lógica de negócios pode ser vista nos fluxos de trabalho que eles suportam, como em sequências ou etapas que especificam em detalhes o fluxo adequado de informações ou dados e, portanto, a tomada de decisões. A lógica de negócios também é conhecida como "lógica de domínio".

## Entendendo a lógica de negócios

Dito de outra forma, a lógica de negócios são regras de negócios do mundo real colocadas em código de computador e mostradas em um programa de computador por meio de uma interface de usuário. A lógica de negócios é mais evidente em seu papel na criação de fluxos de trabalho que passam dados entre usuários e sistemas de software. A lógica de negócios determina como os dados podem ser mostrados, armazenados, criados e alterados. Ele fornece um sistema de regras que orienta como os objetos de negócios (partes do software que controlam como os dados são transportados) funcionam uns com os outros. A lógica de negócios também orienta como os objetos de negócios dentro do software são acessados e atualizados. Ele existe em um nível mais alto do que o tipo de código usado para manter a infraestrutura básica do computador, como como um banco de dados é exibido para um usuário ou como infraestrutura básica do sistema.

Os algoritmos envolvidos na lógica de negócios executam o processamento de dados nos bastidores que é invisível para o usuário, mas é fundamental para manter as coisas funcionando sem problemas em uma economia moderna.

## Lógica de Negócios vs. Regras de Negócios

As regras de negócios são inúteis sem lógica de negócios para determinar como os dados são calculados, alterados e transmitidos para usuários e software. Mas sem regras de negócios para criar uma estrutura, a lógica de negócios não pode existir. A lógica de negócios é qualquer parte de uma empresa de negócios que compõe um sistema de processos e procedimentos, enquanto qualquer outra coisa é um exemplo de regra de negócios.

## Exemplo de lógica de negócios

A lógica de negócios de um emissor de cartão de crédito pode especificar que [transações de cartão de crédito fora do estado](/transaction) acima de um certo limite, digamos $ 500, sejam [sinalizadas](/flag) como suspeitas e o emissor contatado o mais rápido possível para confirmar a autenticidade da transação. A política de sinalizar tal transação é um exemplo de regra de negócios; o processo real de sinalizar a transação é um exemplo de lógica de negócios. Dado que milhões de transações de cartão de crédito são realizadas todos os dias, a lógica de negócios permite que essas transações sejam verificadas e processadas de maneira eficiente e oportuna.
