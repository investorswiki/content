# BEP-2
BEP-2 é um padrão técnico para emissão e implementação de tokens na Binance Chain. Esse padrão define um conjunto de regras que os tokens devem seguir para funcionar no ecossistema da Binance Chain. Um padrão de token comparável para emissão de tokens na blockchain Ethereum é o [ERC-20](/erc-20).

BEP-2 não é um software, mas sim um conjunto de especificações técnicas. Essas regras garantem que os tokens no ecossistema da Binance Chain possam interagir perfeitamente entre si. Os tokens BEP-2 podem representar uma ampla gama de ativos digitais e bens comercializáveis. O padrão garante que todos eles possam se integrar sem problemas entre si e com o ecossistema circundante.

O padrão BEP-2 também define alguns dos recursos básicos e essenciais dos tokens emitidos na Binance Chain. Embora o padrão defina as regras, tokens diferentes ainda terão implementações diferentes para os casos de uso específicos para os quais foram criados. Ainda assim, criar um padrão como esse diminui significativamente a quantidade de esforço necessária dos desenvolvedores para emitir tokens.

O ativo nativo da Binance Chain é o BNB, que é usado para pagar taxas (semelhante ao gás). Isso significa que, quando você quiser transferir seus tokens BEP-2, também precisará ter uma pequena quantidade de BNB em sua carteira.

Os tokens BEP-2 podem ser negociados na Binance DEX, uma [exchange descentralizada](/decentralized-exchange) que está sendo executada em cima da Binance Chain.

Você pode armazenar tokens BEP-2 em várias carteiras diferentes. A Trust Wallet é uma excelente opção se você quiser usar uma carteira móvel, mas muitas outras carteiras de software também suportam o padrão. O padrão BEP-2 também é suportado por carteiras de hardware, como Ledger Nano X, Ledger Nano S, Trezor Model T ou Coolwallet S.

Você pode navegar por todos os tokens que foram emitidos na Binance Chain no explorador de blocos da Binance Chain.

Você pode emitir tokens BEP-2 usando este portal. É altamente recomendável ler primeiro as especificações técnicas mais detalhadas no GitHub da Binance Chain. Se você quiser brincar com as ideias de token BEP-2, você pode fazer isso na rede de testes da Binance Chain.

Você encontrará guias técnicos detalhados sobre a emissão de tokens e muitos tópicos relacionados na Documentação da Binance Chain.
