# Opções Roll Up
## O que é um conjunto de opções?

Um roll up de opções refere-se ao fechamento de uma posição de [opções existente](/option) enquanto abre uma nova posição na mesma opção a um [preço de exercício mais alto](/strikeprice).

É o oposto de uma rolagem de opções, onde um investidor simultaneamente fecha uma posição e abre outra com um preço de exercício mais baixo.

## Entendendo um conjunto de opções

Um roll up de opções, que é a abreviação de "rolar uma opção até um preço de exercício mais alto", refere-se ao aumento do preço de exercício de uma posição de opção fechando o contrato inicial e abrindo um novo contrato para o mesmo [ativo subjacente](/underlying-asset) a um preço mais alto. preço de greve. O trader executa ambas as pernas simultaneamente para reduzir a [derrapagem](/slippage) ou a erosão do lucro, devido a uma mudança no preço do ativo subjacente que pode ocorrer ao colocar a estratégia.

Quer a posição existente seja de [venda](/put) ou de [compra](/calloption),. o procedimento de roll up é o mesmo. Um roll up em uma opção de compra é uma estratégia de alta porque você está apostando que o preço continuará subindo para o novo preço de exercício mais alto. Também é uma negociação de alta ao rolar opções de venda, uma vez que passar para uma greve mais alta indica que você não acredita que o preço cairá mais.

Ao rolar uma opção de compra, o trader deve garantir que o custo líquido da nova posição seja menor do que o lucro gerado pelo fechamento da posição antiga, dado que o [prêmio de call mais alto de strike,](/callpremium) [out-of-the-money](/outofthemoney) (OTM) deve ser menos. Por outro lado, novos contratos de venda também custariam mais em um roll up do que os antigos contratos de venda.

Dependendo se as posições antigas e novas são [longas](/long) ou [curtas](/short),. o resultado de um roll up pode ser um débito ou um crédito na conta. Quanto depende do diferencial de preço das opções roladas.

## Como funciona um conjunto de opções

Para iniciar um roll up de opções, o trader pode configurar ordens simultâneas de " [vender para fechar](/selltoclose) " e " [comprar para abrir](/buytoopen) " para sair de uma posição longa existente enquanto abre uma nova posição longa com um preço de exercício mais alto, ou configurar ordens simultâneas de "comprar para abrir" fechar" e "vender para abrir" ordens para sair de uma posição curta existente enquanto abre uma nova posição curta com um preço de exercício mais alto.

Existem várias razões pelas quais um trader acumularia uma posição, inclusive para evitar o exercício em posições de compra curtas ou simplesmente aumentar a alta para uma posição de compra longa. Lembre-se de que uma chamada longa [in-the-money](/inthemoney) (ITM) perde a maior parte de seu [valor de tempo](/timevalue),. portanto, rolar para uma chamada OTM daria ao trader lucros parciais e, possivelmente, mais retorno pelo dinheiro, graças ao preço mais baixo do novo chamadas.

Uma posição [de venda longa](/long_put) pode rolar para um preço mais alto se o ativo subjacente subir de preço, mas o trader ainda acredita que acabará caindo. Desta forma, a posição permanece no lugar com perdas reduzidas.

> Os traders devem observar que os [spreads](/spread) entre os preços das opções com diferentes strikes variam. Algumas condições de mercado não serão tão propícias para o roll up quanto outras.

>

## Outros tipos de rolos de opções

Os traders de opções usam várias estratégias de rolagem para responder às mudanças nas condições do mercado, garantir lucros, limitar perdas e [gerenciar riscos](/riskmanagement).

comerciantes também podem [rolar para baixo](/rolldown) uma posição da mesma maneira que podem rolar. Esta estratégia envolve simplesmente fechar a posição original e abrir uma nova posição com o mesmo ativo subjacente e [data de vencimento](/expirationdate),. mas a um preço mais baixo.

Além disso, os comerciantes podem [avançar](/rollforward) uma posição mantendo o mesmo preço de exercício, estendendo-se para uma data de vencimento mais longa. Se o novo contrato envolver um preço de exercício mais alto e uma data de vencimento posterior, a estratégia é chamada de "roll-up and forward". Por outro lado, se o novo contrato tiver um preço de exercício mais baixo e uma data de vencimento posterior, ele é chamado de "roll-down and forward".

## Destaques

- Uma estratégia de roll-up de opções é normalmente implantada em resposta às mudanças nas condições do mercado.

- Um roll up de opções fecha uma posição de opções em um exercício para abrir uma nova posição no mesmo tipo de opção a um preço de exercício mais alto.

- Um roll up de uma opção de compra ou de uma opção de venda é uma estratégia de alta, enquanto uma rolagem de uma opção de compra ou de venda é uma estratégia de baixa.
