# Restrição de conclusão do projeto
## O que é restrição de conclusão do projeto?

Uma restrição de conclusão de projeto é uma cláusula frequentemente encontrada em [escrituras de títulos municipais](/indenture) que exige que a parte emissora venda títulos de dívida adicionais (normalmente títulos de receita) para financiar a conclusão total de um projeto.

[títulos](/bond) municipais reembolsa um credor usando a receita resultante de um projeto concluído. Caso um projeto encontre obstáculos que o impeçam de atingir a fase de geração de receita – por exemplo, devido a custos de construção maiores do que o previsto – uma restrição de conclusão do projeto exigiria que o emissor assumisse dívidas adicionais para concluir o projeto.

## Entendendo a restrição de conclusão do projeto

A restrição de conclusão do projeto é uma cláusula destinada a proteger os interesses dos obrigacionistas. No caso de um projeto gerador de receita ser abandonado ou interrompido de outra forma antes da conclusão – por exemplo, devido a derrapagens de custos – a cláusula forçaria o emissor a garantir financiamento de dívida adicional. Isso garante que o projeto seja concluído e comece a gerar a receita necessária para cumprir suas obrigações de pagamento de títulos.

Para efeitos de títulos municipais, uma escritura garante as especificações legais e vinculativas do contrato que detalham as principais características do título. Isso inclui a data de vencimento, quando os pagamentos de juros são devidos e os juros reais a serem cobrados, juntamente com quaisquer termos e condições. Uma restrição de conclusão do projeto é um exemplo de um termo que pode ser incluído em uma escritura de emissão de títulos para proteger os [detentores de títulos](/bondholder) e ajudar a garantir que eles recuperem seu investimento.

## Exemplo de uma restrição de conclusão do projeto

Aqui está um exemplo de como uma restrição de conclusão de projeto pode funcionar. Imagine uma cidade que está construindo uma nova estrada com pedágio. Para financiar o projeto, que custará US$ 5 milhões, a cidade emite um valor equivalente em títulos para pagar a construção.

No entanto, no meio do projeto, a cidade encontra um grande obstáculo que eleva o preço da construção para US$ 10 milhões. Como a escritura da venda de títulos original continha uma restrição de conclusão do projeto, a cidade é obrigada a fornecer os US$ 5 milhões adicionais para concluir o projeto da estrada com pedágio. Graças à restrição de conclusão do projeto, os detentores de títulos estão protegidos de perder seu investimento.

## Destaques

- A restrição de conclusão de projeto é uma cláusula que exige que a parte emissora venda títulos de dívida para financiar a conclusão total de um projeto.

- A restrição protege os detentores de títulos, pois os emissores são obrigados a garantir o financiamento necessário para concluir um projeto e gerar receita para pagar os investidores.

- Restrições de conclusão de projetos são frequentemente encontradas em escrituras de títulos municipais.
