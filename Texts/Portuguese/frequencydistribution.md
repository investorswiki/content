# distribuição de frequência
## O que é uma distribuição de frequência?

Uma distribuição de frequência é uma representação, em formato gráfico ou tabular, que exibe o número de observações dentro de um determinado intervalo. O tamanho do intervalo depende dos dados que estão sendo analisados e dos objetivos do analista. Os intervalos devem ser mutuamente exclusivos e exaustivos. As [distribuições de](/distribution) frequência são normalmente usadas dentro de um contexto estatístico. Geralmente, as distribuições de frequência podem ser associadas ao gráfico de uma [distribuição normal](/normaldistribution).

## Entendendo uma distribuição de frequência

Como ferramenta [estatística](/statistics),. uma distribuição de frequência fornece uma representação visual da distribuição de observações dentro de um teste específico. Os analistas costumam usar uma distribuição de frequência para visualizar ou ilustrar os dados coletados em uma amostra. Por exemplo, a altura das crianças pode ser dividida em várias categorias ou intervalos diferentes.

Ao medir a altura de 50 crianças, algumas são altas e outras baixas, mas há uma alta probabilidade de maior frequência ou concentração na faixa intermediária. Os fatores mais importantes para a coleta de dados são que os intervalos utilizados não devem se sobrepor e devem conter todas as observações possíveis.

## Representação Visual de uma Distribuição de Frequência

Tanto os histogramas quanto os gráficos de barras fornecem uma exibição visual usando colunas, com o eixo y representando a contagem de frequência e o eixo x representando a variável a ser medida. Na altura dos filhos, por exemplo, o eixo y é o número de filhos e o eixo x é a altura. As colunas representam o número de crianças observadas com as alturas medidas em cada intervalo.

Em geral, um gráfico de [histograma](/histogram) normalmente mostrará uma distribuição normal, o que significa que a maioria das ocorrências cairá nas colunas do meio. Distribuições de freqüência podem ser um aspecto chave para traçar distribuições normais que mostram probabilidades de observação divididas entre [desvios padrão](/standarddeviation).

As distribuições de frequência podem ser apresentadas como uma tabela de frequência, um histograma ou um gráfico de barras. Abaixo está um exemplo de uma distribuição de frequência como uma tabela.

## Distribuição de frequência na negociação

As distribuições de frequência não são comumente usadas no mundo dos investimentos; no entanto, os traders que seguem Richard D. Wyckoff, um trader pioneiro do início do século 20, usam uma abordagem de negociação que envolve distribuição de frequência.

As casas de investimento ainda usam a abordagem, que requer prática considerável, para ensinar os comerciantes. O gráfico de frequência é chamado de gráfico de ponto e figura e foi criado a partir da necessidade de [os operadores de pregão](/floortrader) observarem a ação do preço e identificarem as tendências.

O eixo y é a variável medida e o eixo x é a contagem de frequência. Cada mudança na ação do preço é indicada em Xs e Os. Os traders interpretam isso como uma tendência de alta quando surgem três Xs; neste caso, a demanda superou a oferta. Na situação inversa, quando o gráfico mostra três O's, indica que a oferta superou a demanda.

## A linha de fundo

Uma distribuição de frequência é usada para exibir o número de observações dentro de um determinado intervalo. Esse método, embora nem sempre seja comumente usado em investimentos, ainda é usado por alguns traders. Nesse caso, o gráfico de frequência é chamado de gráfico de ponto e figura e é usado para identificar tendências por meio da observação da ação do preço.

##Destaques

- As distribuições de frequência são particularmente úteis para distribuições normais, que mostram as observações de probabilidades divididas entre desvios padrão.

- Uma distribuição de frequência em estatística é uma representação que exibe o número de observações dentro de um determinado intervalo.

- A representação de uma distribuição de frequência pode ser gráfica ou tabular para facilitar o entendimento.

- Em finanças, os comerciantes usam distribuições de frequência para tomar nota da ação do preço e identificar tendências.

##PERGUNTAS FREQUENTES

### Quais são os tipos de distribuição de frequência?

Os tipos de distribuição de frequência são distribuição de frequência agrupada, distribuição de frequência não agrupada, distribuição de frequência acumulada, distribuição de frequência relativa e distribuição de frequência acumulada relativa.

### Qual é a importância de uma distribuição de frequência?

Uma distribuição de frequência é um meio de organizar uma grande quantidade de dados. Ele pega dados de uma população com base em certas características e organiza os dados de forma compreensível para um indivíduo que deseja fazer suposições sobre uma determinada população.

### Como posso construir uma distribuição de frequência?

Para construir uma distribuição de frequência, primeiro, observe as classes específicas determinadas por intervalos em uma coluna e, em seguida, some os números em cada categoria isolada com base em quantas vezes ela aparece. A frequência pode então ser anotada na segunda coluna.
