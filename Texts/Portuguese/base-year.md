# Ano base
## O que é um ano base?

Um ano base é o primeiro de uma série de anos em um índice econômico ou financeiro. Normalmente, é definido para um nível arbitrário de 100. Anos-base novos e atualizados são introduzidos periodicamente para manter os dados atualizados em um índice específico. Qualquer ano pode servir como ano base, mas os analistas normalmente escolhem os anos recentes.

## Entendendo o ano base

Um [ano base](/base-year-analysis) é usado para comparação na medida de uma atividade de negócios ou índice econômico. Por exemplo, para encontrar a taxa de [inflação](/inflation) entre 2013 e 2018, 2013 é o ano base ou o primeiro ano no tempo definido. O ano base também pode descrever o ponto de partida a partir de um ponto de crescimento ou de uma linha de [base](/baseline) para calcular [as vendas nas mesmas lojas](/samestoresales).

## Ano Base e Taxas de Crescimento

Muitos [índices financeiros](/ratioanalysis) são baseados no crescimento porque os analistas querem saber quanto um determinado número muda de um período para outro. A equação da taxa de crescimento é (Ano Atual - Ano Base) / Ano Base. O passado, na análise de índices, é o período base. A análise de crescimento é uma maneira comumente usada para descrever o desempenho da empresa, principalmente para vendas. Se a empresa A aumentar as vendas de $ 100.000 para $ 140.000, isso implica que a empresa aumentou as vendas em 40%, onde $ 100.000 representa o valor do ano base.

## Ano Base e Cálculos de Vendas nas Mesmas Lojas

As empresas estão sempre procurando maneiras de aumentar as vendas. Uma maneira de as empresas aumentarem as vendas é abrindo novas lojas ou filiais. As novas lojas têm taxas de crescimento mais altas porque começam do zero, e cada venda de nova loja é uma venda incremental. Como resultado, os analistas analisam fatores adicionais, como quanto as vendas cresceram em uma base de vendas mesmas lojas. Isso também é conhecido como medição [de lojas comparáveis](/comparable-store-sales) ou [vendas de lojas comparáveis](/comps).

No cálculo das vendas de lojas comp, o ano base representa o ponto de partida para o número de lojas e a quantidade de vendas que essas lojas geraram. Por exemplo, se a empresa A tem 100 lojas que venderam $ 100.000 no ano passado, cada loja vendeu $ 10.000. Este é o ano base. Seguindo este método, o ano base determina as vendas base e o número base de lojas. Se a empresa A abrir mais 100 lojas no ano seguinte, essas lojas geram $ 50.000, mas as vendas nas mesmas lojas diminuem em valor em 10%, de $ 100.000 para $ 90.000. A empresa pode relatar um crescimento de 40% nas vendas de US$ 100.000 a US$ 140.000, mas analistas experientes estão mais interessados no declínio de 10% nas vendas nas mesmas lojas.
