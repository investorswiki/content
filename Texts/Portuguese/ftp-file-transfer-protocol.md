# Protocolo de transferência de arquivos (FTP)
## O que é o protocolo de transferência de arquivos (FTP)?

O termo protocolo de transferência de arquivos (FTP) refere-se a um processo que envolve a transferência de arquivos entre dispositivos em uma rede. O processo funciona quando uma parte permite que outra envie ou receba arquivos pela internet. Originalmente usado como uma forma de os usuários se comunicarem e trocarem informações entre dois dispositivos físicos, agora é comumente usado para armazenar arquivos na [nuvem](/cloud-computing),. que geralmente é um local seguro que é mantido remotamente.

O FTP pode ser usado por uma [empresa](/business) ou indivíduo para transferir arquivos de um sistema de computador para outro ou por sites para fazer upload ou download de arquivos de seus servidores.

## Como funciona o protocolo de transferência de arquivos (FTP)

O protocolo de transferência de arquivos permite que indivíduos e empresas compartilhem arquivos eletrônicos com outras pessoas sem precisar estar no mesmo espaço. Isso pode ser feito usando um cliente FTP ou através da nuvem. Independentemente da opção, ambas as partes precisam de uma conexão com a Internet em funcionamento.

A maioria dos navegadores [da Web](/web-20) vem com clientes FTP que permitem aos usuários transferir arquivos de seu computador para um servidor e vice-versa. Alguns usuários podem querer usar um cliente FTP de terceiros porque muitos deles oferecem recursos extras. Exemplos de clientes FTP gratuitos para download incluem FileZilla Client, FTP Voyager, WinSCP, CoffeeCup Free FTP e Core FTP.

Muitas pessoas já usaram FTP antes mesmo sem perceber. Se você já baixou um arquivo de uma página da Web, você usou o FTP. O primeiro passo é fazer o login, que pode ocorrer de forma automática ou digitando manualmente um nome de usuário e senha. O FTP também exigirá que você acesse um servidor FTP por meio de um número de porta específico. Depois de acessar o servidor FTP por meio de seu cliente FTP, agora você pode transferir arquivos. Nem todos os servidores FTP públicos exigem que você entre porque alguns servidores permitem que você os acesse anonimamente.

Como observado acima, o FTP foi originalmente desenvolvido como uma forma de enviar e receber arquivos entre dois computadores físicos. Mas com as mudanças na tecnologia, os usuários podem executar transferências de arquivos pela nuvem. O uso da nuvem permite que as transferências sejam feitas de forma conveniente e segura (o que pode proteger indivíduos e empresas de [violações de dados](/data-breach) ) e com custo relativamente baixo.

> O termo cliente FTP refere-se ao software que permite transferir arquivos para outra parte.

>

## Considerações Especiais

O protocolo de transferência de arquivos é um dos muitos protocolos diferentes que ditam como os computadores e sistemas de computação se comportam na Internet. Outros protocolos incluem:

- **Hypertext Transfer Protocol (HTTP):** Projetado para transmitir dados pela web

- **Internet Message Access Protocol (IMAP):** Fornece acesso ao quadro de avisos ou mensagens de e-mail de um serviço compartilhado

- **Network Time Protocol (NTP):** Sincroniza os horários do relógio em computadores em uma rede

O FTP permite que computadores na Internet transfiram arquivos para frente e para trás. Como tal, é uma ferramenta essencial para quem constrói e mantém sites hoje.

### O que procurar em um cliente FTP

Clientes FTP individuais fornecem recursos diferentes que permitem que os usuários modifiquem a maneira como fazem upload e download de arquivos. Por exemplo, se você usa o FileZilla, o programa permite definir limites de [largura de banda para arquivos.](/bandwidth) Isso permite controlar as velocidades de upload e download, o que pode ser útil se você gerenciar várias transferências de arquivos de uma só vez.

Outros recursos que você pode procurar em um cliente FTP incluem autenticação de chave pública, a capacidade de definir níveis de compactação de arquivo ou ferramentas que permitem pesquisar um servidor usando máscaras de arquivo.

## Exemplo de protocolo de transferência de arquivos (FTP)

O software FTP é relativamente simples de configurar. FileZilla é um cliente FTP gratuito para download. Digite o endereço do servidor que você deseja acessar, a porta e a senha para acessar o servidor.

Uma vez concedido o acesso, os arquivos do usuário em seu sistema local, bem como o servidor acessado, ficarão visíveis. O usuário pode fazer download de arquivos do servidor para o sistema local ou fazer upload de arquivos do sistema local para o servidor. Eles também podem fazer alterações em arquivos no servidor, desde que tenham a devida autorização para fazê-lo.

##Destaques

- O FTP é uma ferramenta essencial para quem constrói e mantém sites.

- O protocolo de transferência de arquivos é uma maneira de baixar, fazer upload e transferir arquivos de um local para outro na internet e entre sistemas de computador.

- O FTP permite a transferência de arquivos entre computadores ou através da nuvem.

- Os usuários precisam de uma conexão com a Internet para executar transferências FTP.

- Muitos clientes FTP são gratuitos para download, embora a maioria dos sites já tenha o FTP embutido.
