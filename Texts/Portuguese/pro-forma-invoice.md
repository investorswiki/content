# Fatura proforma
## O que é uma fatura pro forma?

Uma fatura pro forma é uma [nota de venda preliminar](/bill-of-sale) enviada aos compradores antes de uma remessa ou entrega de mercadorias. A fatura normalmente descreve os itens comprados e outras informações importantes, como o peso de envio e as despesas de transporte.

## Entendendo as faturas pro forma

As faturas pro forma geralmente entram em jogo com transações internacionais, especialmente para fins alfandegários nas importações. Uma [fatura](/invoice) pró-forma é diferente de uma simples cotação de preço, pois é um contrato vinculativo, embora os termos de venda estejam sujeitos a alterações.

Empresas em praticamente todos os setores usam faturas pro forma para satisfazer seu processo interno de aprovação de compra. As faturas pro forma simplificam o processo de vendas, eliminando idas e vindas adicionais após a conclusão de uma venda, pois todos os termos foram acordados antecipadamente.

A maioria das faturas pro forma fornecem ao comprador um preço de venda preciso. Inclui uma estimativa de quaisquer [comissões](/commission) ou taxas, como impostos aplicáveis ou custos de envio. Embora a fatura pró-forma possa estar sujeita a alterações, ela representa uma estimativa de boa fé para evitar expor o comprador a quaisquer encargos inesperados e significativos assim que a transação for finalizada.

[quaisquer](/deliverables) [entregas](/deliverables) acordadas ou com os itens enviados. Embora contenha detalhes de custo exatos associados à venda, não é uma exigência oficial de pagamento. Nenhuma diretriz dita a apresentação ou o formato exato de uma fatura pro forma, e ela pode ou não se assemelhar a outras faturas comerciais.

> Se uma empresa usa uma fatura pró-forma para alfândega dos EUA, ela deve apresentar uma fatura comercial no prazo de 120 dias.

## Considerações Especiais

A documentação específica é necessária antes que as mercadorias possam passar pela alfândega nos EUA Uma fatura comercial tradicional deve listar informações substanciais, como o comprador e o vendedor, uma descrição das mercadorias, a quantidade, o valor de todos os itens enviados e a localização de a compra.

Uma fatura pró-forma requer apenas informações suficientes para permitir que a alfândega determine os [direitos](/duty) necessários a partir de um exame geral das mercadorias incluídas. Se uma empresa usa uma fatura pró-forma para alfândega, deve apresentar uma fatura comercial no prazo de 120 dias.

## Exemplo de fatura pro forma

Se as partes relevantes já concluíram uma venda, uma fatura comercial a registra. Caso a transação ainda não tenha ocorrido, a empresa vendedora pode utilizar uma fatura pró-forma. Uma empresa também pode usar uma fatura pró-forma se os termos do contrato de venda especificarem que o pagamento total não é devido até que o comprador receba certas mercadorias.

Um cliente de padaria, por exemplo, pode concordar com o preço de um bolo em uma fatura pró-forma e o padeiro entregará o bolo assim que estiver pronto. O cliente pagará por isso quando receber a fatura. O valor final da fatura será o mesmo ou próximo ao valor da fatura pro forma.
