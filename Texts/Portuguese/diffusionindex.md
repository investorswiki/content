# Índice de difusão
## O que é um índice de difusão?

Um índice de difusão é uma métrica usada para mostrar a dispersão da mudança em relação a um ponto de referência, e uma série de dados pode demonstrar se há tendências ascendentes ou descendentes.

Os índices de difusão são comumente usados para destacar os resultados de uma pesquisa. As respostas estão na forma de valores e, dependendo desses valores, um número base é designado. Muitos indicadores usam uma linha de base de 0, 50 ou 100. Uma série de dados mostra a distância dos resultados da linha de base. Enquanto um ponto de dados não muito distante do número base indicaria pouca mudança, um ponto de dados distante da base poderia sugerir uma mudança significativa.

Na economia, o índice de difusão é utilizado na formulação de indicadores, como [os índices dos gerentes de compras](/pmi) e o [Índice de Manufatura de Richmond](/richmond-manufacturing-index). Diferentemente dos indicadores econômicos, nos quais os dados são baseados em preços ou unidades, aqueles que utilizam um índice de difusão são baseados em pesquisas. E dependendo de como as perguntas da pesquisa são enquadradas, o índice de difusão pode ser usado como ferramenta de previsão, da mesma forma que o PMI é considerado um indicador econômico líder.

## Como calcular um índice de difusão

Existem diferentes maneiras de calcular um índice de difusão, e a fórmula pode ser longa ou curta.

###Fórmula da Reserva Federal

O [Federal Reserve](/federal-reserve) usa a fórmula básica para calcular um índice de difusão. Essa fórmula é usada no cálculo de índices para muitos indicadores entre os 12 bancos regionais do Fed, incluindo pesquisas dos Feds de Richmond e Kansas City.

Índice de difusão = [100 * (I − D)] / (I + N + D)

I = Número de Inquiridos que Relatam Aumentos

N = Número de entrevistados que relatam nenhuma alteração

D = Número de entrevistados que relatam diminuições

### Fórmula IHS Markit

O IHS Markit, grupo de pesquisa que publica índices de gerentes de compras nos EUA e em outros países, tem fórmula própria, diferente da do Fed.

Índice de difusão do IHS Markit = (% Reportando 'Para cima') + [0,5 * (% Reportando 'o mesmo')] + [0,0 * (% Reportando 'Para baixo')]

Isso pode ser expresso em forma curta como:

Índice = Porcentagem de Respostas Positivas + 0,5 (Porcentagem de Respostas Neutras)

## Exemplo de um índice de difusão

Como exemplo, para uma pergunta de pesquisa de amostra sobre se a economia entrará em [recessão](/recession) em breve, os resultados mostraram que 70% dos entrevistados responderam que havia uma alta probabilidade, 20% responderam que não havia probabilidade e 10% responderam que não havia era uma probabilidade baixa.

Se os resultados forem inseridos na fórmula do Fed para o índice de difusão, ele seria calculado da seguinte forma:

[100 * (0,70 - 0,10)] / (0,70 + 0,20 + 0,10) = 60 / 1 = 60

Se calculado usando a fórmula do IHS Markit, a resposta seria:

0,70 + (0,5 * 0,20) + (0,0 * 0,10) = 0,70 + 0,10 + 0,00 = 0,80

Os resultados diferem para ambos, dependendo da fórmula usada. Atribuir uma linha de base de 0 ou 100 seria ideal neste exemplo, pois a resposta pode ser manipulada como dois dígitos.

## Como interpretar um índice de difusão

Uma série de dados pode mostrar impulso e revelar se há tendências ascendentes e descendentes. Os dados acima da linha de base indicariam expansão, enquanto os dados abaixo indicariam contração. Dados próximos ou na linha de base significariam pouca ou nenhuma mudança.

## Quais são as limitações de um índice de difusão?

Uma difusão de índice pode representar uma visão unilateral. Assim, comparar o índice de difusão com outros conjuntos de dados pode fornecer confirmação adicional ou refutar uma conclusão feita usando um índice de difusão. O uso de outras métricas, como o [coeficiente de correlação](/correlationcoefficient),. pode mostrar se um índice de difusão é uma ferramenta de previsão confiável.

##Destaques

- Um índice de difusão refere-se à tendência comum dentro de um grupo de números ou estatísticas.

- O índice de difusão pode ser usado para detectar divergências, que podem sinalizar força ou fraqueza que não é visível ao observar o índice de ações por conta própria.

- Um índice de difusão decrescente mostra que há menos ações subindo, o que indica que a pressão de compra está enfraquecendo e/ou a pressão de venda está aumentando no índice de ações.

- No mercado de ações, um índice de difusão crescente significa que há um número crescente de ações em alta, o que é positivo para índices de ações em alta.
