# período base
## O que é um Período Base?

Um período base é um ponto no tempo para o qual os dados são coletados e usados como referência em relação aos dados econômicos de outros períodos para interpretá-los em uma base comum. Os períodos base são frequentemente usados em aplicações financeiras e econômicas, como medir a inflação ou outras variáveis sujeitas a alterações com base na passagem do tempo.

O período de base também pode ser referido como "período de referência", "período de base" ou "período de índice".

## Entendendo os Períodos Base

O período base pode ser pensado como um critério comum para dados econômicos. Em vez de declarar cada ponto de dados em uma série como um número bruto, ele pode ser declarado como uma proporção ou porcentagem do valor dos dados no período base. O valor do período base costuma ser definido como a unidade de medida, geralmente 1 ou 100, e todos os outros pontos de dados são reapresentados como valores decimais, fracionários ou percentuais do valor dos dados para esse período.

Comparar cada ponto de dados com o período base pode ser uma maneira conveniente de lidar com séries de dados que consistem em números grandes ou complexos. Cada ponto de dados na série indexada pode ser facilmente interpretado como a proporção, alteração percentual ou taxa de crescimento da série de dados subjacente ao longo do tempo, em relação ao período base.

## exemplo

Por exemplo, se um [índice de preços](/consumerpriceindex) tem um [ano-base](/base-year-analysis) de 1990 e os preços atuais estão sendo comparados com os preços daquele período para construir um índice de série temporal, então o nível de preços em todos os outros anos seria declarado como uma porcentagem da década de 1990 nível de preço. O índice de preços para 1990 pode ter um valor de 100 e os níveis de preços para outros anos teriam valores proporcionalmente maiores (ou menores que) 100 em proporção à proporção dos níveis de preços reais desses anos.

Alternativamente, embora menos comumente, o período de base pode se referir à comparação de cada ponto de dados com um valor de dados passado usando um intervalo de tempo constante em vez de um período de base constante. Essa técnica não cria uma comparação de índice consistente ao longo do tempo, mas pode ajudar a eliminar o efeito de flutuações sazonais ou de curto prazo nos dados. As comparações [ano a ano](/year-over-year) ou mês a mês são exemplos de uso de dados passados em um intervalo constante como base para comparação com dados atuais.

> O uso de períodos base para indexar dados não está restrito a aplicações financeiras. Muitas ciências naturais também usam regularmente um período base como parte de seus processos analíticos. Por exemplo, para medir as mudanças nos padrões climáticos globais, devem ser estabelecidos anos-base.

>

## Período base e o efeito base

Ao construir um índice, a escolha de uma base de comparação pode influenciar como os dados podem ser interpretados e devem ser escolhidos com cuidado para iluminar o objetivo desejado para o qual os dados estão sendo usados. Valores atípicos ou condições anormais em um período base podem levar a comparações que distorcem as tendências em uma série de dados. Essa distorção às vezes é chamada de [efeito base](/base-effect).

Por exemplo, suponha que a cidade de Nova York institui novos códigos de construção que entram em vigor em 1º de junho de um determinado ano. No mês de maio, as construtoras correm para iniciar novos projetos de construção para evitar os gastos com o cumprimento dos novos códigos. Isso pode levar a uma situação em que os dados para o início da construção mostrem um valor anormalmente alto para maio e um valor anormalmente baixo para junho, à medida que os construtores avançam seus cronogramas de construção, o que não reflete nenhuma tendência subjacente nos dados, apenas o um- fora da mudança regulatória.

Nesse caso, escolher o ponto de dados de maio ou junho como o ano base em uma comparação ou construir um índice de série temporal levaria a resultados altamente distorcidos nos dados de transformação resultantes, já que cada ponto de dados no índice comparará os dados atuais com um valor anormalmente alto (ou baixo) no denominador. Um analista seria bem aconselhado a escolher um valor mais típico como o período base para a comparação de pontos de dados posteriores.

##Destaques

- A comparação de outros pontos de dados com um período base constante permite que os analistas identifiquem mudanças e distingam tendências de longo prazo de flutuações de curto prazo.

- O período base refere-se ao benchmark contra o qual os dados econômicos de outros períodos são medidos.

- A escolha de um período base pode influenciar a perspectiva de um observador sobre os dados, o que é conhecido como efeito base.
