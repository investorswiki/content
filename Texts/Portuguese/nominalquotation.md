# Cotação Nominal
## O que é cotação nominal?

Uma cotação nominal é um preço hipotético pelo qual uma ação ou algum outro título pode ser negociado. O oposto de uma cotação nominal é uma [cotação firme](/firmquote),. que representa uma oferta obrigatória para negociar a um preço específico.

## Entendendo a cotação nominal

As cotações nominais são fornecidas pelos formadores de [mercado](/marketmaker) para ajudar os traders a estimar o valor de uma transação proposta antes de tomar uma decisão. Eles não representam ofertas reais para comprar ou vender o título. Para evitar confusão, os símbolos nas cotações nominais são precedidos pelos prefixos FYI ou FVO (For Valuation Only).

O método exato para calcular as cotações nominais difere dependendo do formador de mercado. Geralmente, os formadores de mercado produzem cotações nominais referenciando a precificação histórica e teórica desse título,

As cotações nominais destinam-se a permitir que os [comerciantes](/trader) estimem o valor de um ativo antes de iniciar uma negociação. Eles são comumente usados nos mercados de [derivativos](/derivative),. incluindo os mercados de [futuros](/futures),. [opções](/option) e [câmbio (forex)](/foreign-exchange). Em diferentes, as cotações nominais podem ser referidas como cotações nominais ou preços nominais.

O oposto de uma cotação nominal é uma [cotação firme](/firmquote),. que é uma oferta real para comprar ou vender um título. Cotações firmes não estão sujeitas a cancelamento. Na verdade, a [Securities and Exchange Commission](/sec) (SEC) pune os formadores de mercado que não cumprem as ofertas firmes, uma ofensa conhecida como recuar de uma transação.

As cotações nominais são especialmente importantes para os comerciantes que compram com [margem](/margin). Na negociação de margem, um investidor toma dinheiro emprestado de uma [corretora](/brokerage-company) para comprar uma quantidade maior do título. A garantia para o dinheiro emprestado consiste nos ativos adquiridos em margem, além das reservas de caixa do próprio trader.

Tanto o comerciante de margem quanto a corretora têm um forte incentivo para monitorar de perto o valor da [garantia](/collateral) na conta de um comerciante de margem. Assim, as corretoras fornecem cotações nominais para os ativos da conta, permitindo o acompanhamento de seu valor.

## Exemplos de cotação nominal

- Os operadores de câmbio usam um tipo de cotação nominal conhecida como [cotação](/indicativequote) [indicativa](/indicativequote). Este é um tipo de cotação de moeda não vinculativa, fornecida por um formador de mercado a uma [contraparte](/counterparty). Quando um formador de mercado oferece uma cotação indicativa a um comerciante, o criador de mercado não é obrigado a negociar o par de moedas fornecido pelo preço ou pela quantidade indicada na cotação. Se um trader ou cliente solicitar uma cotação para um par de moedas, mas não especificar o valor para negociação, ou se houver alguma dúvida quanto à capacidade do formador de mercado de negociar o par de moedas na oferta ou pedido cotado, o formador de mercado emitirá uma citação indicativa.

- Os comerciantes [de títulos municipais](/municipalbond) usam o que é chamado de [indicação viável](/workable-indication) para estimar o preço e a oferta de uma determinada emissão de títulos. A indicação praticável é outro tipo de cotação nominal. Ou seja, permite que um comerciante concorde provisoriamente com os termos da indicação viável, mantendo o direito de revisar o pedido por um período de tempo especificado. Por outro lado, uma cotação firme é imediatamente vinculativa.

##Destaques

- O oposto de uma cotação nominal é uma cotação firme, que representa uma oferta atual e vinculativa para fazer uma transação a esse preço.

- Os símbolos nas cotações nominais são precedidos pelos prefixos FYI ou FVO (For Valuation Only).

- As cotações nominais são fornecidas pelos formadores de mercado para ajudar os traders a estimar o valor de uma transação proposta antes de tomar uma decisão.
