# Shell Corporation
## O que é uma Shell Corporation?

Uma corporação de fachada é uma corporação sem operações comerciais ativas ou ativos significativos. Esses tipos de corporações não são necessariamente ilegais, mas às vezes são usados de forma ilegítima, como para disfarçar a propriedade da empresa da aplicação da lei ou do público. Razões legítimas para uma corporação de fachada incluem coisas como uma [startup](/startup) usando a entidade comercial como um veículo para arrecadar fundos, realizar uma [aquisição hostil](/hostiletakeover) ou abrir o capital.

## Entendendo a Shell Corporation

Por exemplo, além das razões legais acima, as empresas de fachada atuam como veículos de evasão fiscal para negócios legítimos, como é o caso das entidades corporativas da Apple sediadas no Reino Unido. Eles também são usados para obter diferentes formas de [financiamento](/financing).

No entanto, a elisão fiscal às vezes é vista como uma [brecha](/loophole) para [a evasão fiscal](/taxevasion),. pois essas empresas são conhecidas por serem usadas em atividades de mercado negro ou cinza. É natural suspeitar de uma corporação de fachada e é importante entender os vários cenários em que elas surgem.

## Razões para configurar legitimamente uma Shell Corporation

A razão número um para uma empresa doméstica criar uma empresa de fachada é realizar um [paraíso fiscal](/taxhaven) no exterior. Grandes corporações, como no exemplo da Apple, decidiram transferir empregos e lucros para o [exterior](/offshore),. aproveitando-se de códigos fiscais mais flexíveis. Este é o processo de "offshoring" ou "terceirização" do trabalho que antes era realizado internamente.

Para permanecer dentro dos limites legais internacionalmente, as corporações americanas criarão empresas de fachada nos países estrangeiros em que estão terceirizando o trabalho. Isso é legalmente permitido pelos Estados Unidos e alguns dizem que é o próprio código tributário dos EUA que está forçando as empresas domésticas a criar corporações de fachada no exterior.

Outra forma de as empresas de fachada ajudarem com os impostos envolve a necessidade de [as instituições](/financialinstitution) financeiras realizarem atividades financeiras em mercados estrangeiros. Isso permite que eles invistam em [mercados de capitais](/capitalmarkets) fora das fronteiras domésticas e obtenham economias fiscais potenciais.

## Formas que as pessoas abusam das empresas Shell

Embora existam razões legítimas para criar uma empresa de fachada, muitos indivíduos ricos abusam das empresas de fachada para ganho pessoal. A tributação progressiva dentro dos Estados Unidos, ou seja, as faixas de [impostos](/taxbracket),. lentamente fez com que as pessoas procurassem paraísos fiscais pessoais. Pessoas com renda significativamente alta se estabeleceram como empresas de fachada em um ou vários locais, como as Ilhas Cayman. Esta é uma área cinzenta de evasão fiscal onde as pessoas canalizam os ganhos através de empresas de fachada de tal forma que não são contabilizados na [renda pessoal](/personalincome).
