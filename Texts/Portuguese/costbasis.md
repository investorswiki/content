# Base de Custo
## O que é base de custo?

A base de custo é o valor original de um ativo para fins fiscais, geralmente o preço de compra, ajustado para [desdobramentos de ações](/stocksplit),. [dividendos](/dividend) e distribuições [de retorno de capital](/returnofcapital). Esse valor é utilizado para determinar o ganho de capital, que é igual à diferença entre a base de custo do ativo e o [valor atual de mercado](/cmv). O termo também pode ser usado para descrever a diferença entre o preço à [vista](/cashprice) e o preço [futuro](/futures) de uma determinada mercadoria.

## Entendendo a base de custo

No nível mais básico, a base de custo de um investimento é o valor total originalmente investido, mais quaisquer [comissões](/commission) ou taxas envolvidas na compra. Isso pode ser descrito em termos do valor em dólares do investimento ou do preço efetivo por ação pago pelo investimento.

Usar a base de custo correta, também conhecida como base tributária, é importante especialmente se você reinvestiu dividendos e [distribuições de ganhos de capital](/capitalgainsdistribution) em vez de receber os ganhos em dinheiro. Reinvestir distribuições aumenta a base tributária de seu investimento, que você deve contabilizar para relatar um [ganho de capital menor](/capitalgain) e, portanto, pagar menos impostos. Se você não usar a base tributária mais alta, poderá acabar pagando impostos duas vezes sobre as distribuições reinvestidas. Determinar a base de custo correta também é o primeiro passo ao calcular ganhos e perdas após a venda de uma ação.

> O reinvestimento de dividendos aumenta a base de custo de uma ação porque os dividendos são usados para comprar mais ações.

>

O [método de base de custo médio](/averagecostbasismethod) é comumente usado por investidores para relatórios de impostos de fundos mútuos. Um método de base de custo é relatado com a corretora onde seus ativos são mantidos. Muitas corretoras adotam o método de base de custo médio. Os investidores também podem escolher entre outros métodos, incluindo: first in first out ( [FIFO](/fifo) ), last in first out ( [LIFO](/lifo) ), alto custo, baixo custo e muito mais. Uma vez determinado um método de base de custo para um fundo mútuo específico, ele deve permanecer em vigor. As corretoras fornecerão aos investidores a documentação fiscal anual apropriada sobre vendas de fundos mútuos com base em suas escolhas de método de base de custo. O conceito de base de custo é basicamente simples, mas pode se tornar complicado de várias maneiras. A base de custo de rastreamento é necessária para fins fiscais, mas também é necessária para ajudar a rastrear e determinar o sucesso do investimento. A chave é manter bons registros e simplificar a estratégia de investimento sempre que possível.

## Exemplo de base de custo

Por exemplo, se 100 ações de uma ação foram compradas por $ 1.000 no ano passado, com o primeiro ano de dividendos no valor de $ 100 e os dividendos do segundo ano no valor de $ 200, todos os quais foram reinvestidos, a lei tributária aplicável considera esses lucros reinvestidos como renda. Para fins de cálculo de impostos, a base de custo ajustada quando a ação for vendida será registrada em $ 1.300 em vez do [preço de compra original](/purchaseprice) de $ 1.000. Assim, se o preço de venda for $ 1.500, o [ganho tributável](/taxablegain) seria apenas $ 200 ($ 1.500 - $ 1.300) em vez de $ 500 ($ 1.500 - $ 1.000). Se a base de custo for registrada incorretamente como $ 1.000, isso resultará em um passivo fiscal mais alto do que normalmente seria devido.

## Comparações de base de custo

A comparação de base de custo pode ser uma consideração importante. Suponha que um investidor fez as seguintes compras consecutivas de fundos em uma conta tributável: 1.500 ações a $ 20, 1.000 ações a $ 10 e 1.250 ações a $ 8. A base de custo médio do investidor é calculada dividindo $ 50.000/3.750 ações. O custo médio é de US$ 13,33.

Suponha que o investidor venda 1.000 ações do fundo a $ 19. O investidor teria um ganho de capital de $ 5.670 usando o método de base de custo médio.

- Ganho/perda com base no custo médio: (US$ 19 - US$ 13,33) x 1.000 ações = US$ 5.670

Os resultados podem variar significativamente por base de custo.

- First in first out: ($ 19 - $ 20) x 1.000 ações = - $ 1.000

- Último a entrar primeiro a sair: (US$ 19 - US$ 8) x 1.000 = US$ 11.000

- Alto custo: (US$ 19 - US$ 20) x 1.000 ações = - US$ 1.000

- Baixo custo: (US$ 19 - US$ 8) x 1.000 = US$ 11.000

Nesse caso, o investidor estaria em melhor situação se tivesse selecionado o método FIFO ou o método de alto custo para determinar a base de custo antes de vender as ações. Esses métodos não resultariam em nenhum imposto sobre uma perda de $ 1.000. Com o método de base de custo médio, o investidor deve pagar um imposto sobre ganhos de capital sobre o ganho de $ 5.670.

## Como as divisões de ações afetam a base de custo

Se a empresa dividir suas ações, isso afetará sua base de custo por ação, mas não o valor real do investimento original ou o investimento atual. Continuando com o exemplo acima, suponha que a empresa emita um [desdobramento de ações](/stocksplit) de 2:1 em que uma ação antiga dá a você duas novas ações. Você pode calcular sua base de custo por ação de duas maneiras:

- Pegue o valor do investimento original (US$ 10.000) e divida-o pelo novo número de ações que você possui (2.000 ações) para chegar à nova base de custo por ação (US$ 10.000/2.000 = US$ 5,00).

- Pegue sua base de custo anterior por ação (US$ 10) e divida-a pelo fator de divisão de 2:1 (US$ 10,00/2 = US$ 5,00).

## Base de Custo de Ações Doadas ou Herdadas

Caso as ações tenham sido dadas a você como presente, sua base de custo é a base de custo do detentor original que lhe deu o presente. Se as ações estão sendo negociadas a um preço mais baixo do que quando as ações foram doadas, a taxa mais baixa é a base de custo. Se as ações foram dadas a você como herança, a base de custo das ações para você como herdeiro é o [preço](/market-price) de mercado atual das ações na data da morte do proprietário original.

Existem muitos fatores que afetarão sua base de custos e, eventualmente, seus impostos quando você decidir vender. Se sua verdadeira base de custo não for clara, consulte um [consultor financeiro](/financial-advisor),. contador ou advogado tributário.

## Base de Custo e Contratos Futuros

Em relação aos futuros, a base de custo é a diferença entre o preço à vista local de uma mercadoria e seu preço futuro associado. Por exemplo, se um determinado contrato futuro de milho estiver sendo negociado a $ 3,50, enquanto o preço de mercado atual da [commodity](/commodity) hoje é $ 3,10, há uma base de custo de 40 centavos. Se o inverso fosse verdadeiro, com o contrato futuro sendo negociado a $ 3,10 e o preço à vista sendo $ 3,50, a base de custo seria negativa em 40 centavos, pois a base de custo pode ser positiva ou negativa dependendo dos preços envolvidos.

O preço à vista local representa o preço prevalecente para o ativo subjacente, enquanto o preço listado em um contrato futuro refere-se a uma taxa que seria dada em um ponto específico no futuro. Os preços dos futuros variam de contrato para contrato, dependendo do mês em que expiram.

Tal como acontece com outros mecanismos de investimento, o preço à vista flutua dependendo das condições atuais do mercado local. À medida que a data de entrega se aproxima, o preço dos futuros e o preço à vista se aproximam.

## Destaques

- A base de custo é o preço original pelo qual um ativo foi adquirido, para fins fiscais.

- Os ganhos de capital são calculados calculando a diferença do preço de venda para a base de custo.

- Existem vários métodos contabilísticos para ajustar a base de custos para que seja mais favorável, mas tenha o cuidado de seguir as orientações do IRS.
