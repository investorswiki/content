# BEP-721
BEP-721 é um padrão de token Binance Smart Chain (BSC) que permite a criação de tokens não fungíveis (NFTs). É uma extensão do ERC-721 da Ethereum, um dos padrões NFT mais usados e é compatível com a Ethereum Virtual Machine (EVM). Cada NFT é único e não é trocado por nenhum outro token.

Os tokens BEP-721 permitem tokenizar a propriedade dos dados e anexar a eles um identificador exclusivo. Esse aspecto torna o token único e muito diferente dos tokens BEP-20. Com o BEP-20, os desenvolvedores podem criar vários tokens idênticos em um contrato inteligente. No entanto, com o BEP-721, cada token recebe um ID de token diferente.

Por meio desses IDs exclusivos, os tokens BEP-721 podem ser usados para representar itens colecionáveis e não fungíveis. Os usuários podem negociar e transferir esses tokens com base em seu valor de mercado, que depende da raridade ou utilidade do token.

Os tokens BEP-721 podem representar:

1. Arte digital e física

1. Colecionáveis

1.Itens do jogo

1. Propriedade física e imóveis

1. Bilhetes de loteria

Como outros tokens na Binance Smart Chain, as transferências de token BEP-721 exigem BNB para taxas de gás. Os tokens BEP-721 podem ser criados em várias plataformas no ecossistema BSC, incluindo o Binance NFT Marketplace, Featured By Binance, BakerySwap e Juggerworld.

## Funções do token BEP-721

O BEP-721 tem uma lista de funções que regem como o token interage com a Binance Smart Chain. Alguns deles são bastante padrão e também estão presentes no padrão BEP-20:

- nome: Define o nome do token BEP-721, que outros contratos irão identificá-lo por

- Símbolo: Um nome mais curto para o token semelhante a um símbolo de ação

- balanceOf: Mostra o saldo do token de um endereço específico

- totalSupply: Define o número total de tokens criados.

Há também um conjunto de funções básicas de propriedade, algumas das quais são opcionais para incluir. Um recurso exclusivo para NFTs é a função de metadados:

tokenMetadata: Esta função permite a inclusão de metadados em um token usado para vincular a um arquivo de arte ou outro aspecto colecionável do NFT.

Por exemplo, cada CryptoPunk tem metadados contidos no NFT que o apontam para um determinado punk em uma grade de 10.000 punks. Embora os CryptoPunks não usem o padrão de token BEP-721, o uso de metadados é o mesmo.
