# Pro Forma
## O que é pró-forma?

Pro forma é uma maneira de descrever declarações emitidas antes de algumas transações financeiras ou anúncios comerciais. Uma frase latina que significa “como uma questão de forma”, ações pró-forma são aquelas que sustentam convenções básicas em uma interação. Enquanto as ações pró-forma têm importância social, nos negócios elas são uma cortesia esperada que ajuda a contextualizar e facilitar uma transação.

## Definição mais profunda

Em praticamente todos os contextos, as práticas pró-forma são consideradas uma formalidade. Eles têm aplicabilidade em várias circunstâncias diferentes, incluindo lei, governo e negócios. “Pro forma” pode simplesmente descrever uma norma ou a realização de alguma ação esperada que atenda aos requisitos mínimos para uma relação social ou profissional entre duas partes.

Nos negócios, as declarações pró-forma relatam as finanças de uma empresa enquanto oferecem uma perspectiva hipotética sobre seu futuro projetado. Essas declarações são úteis para investidores que possam estar interessados no desempenho de uma empresa ou para ajudar a facilitar uma fusão ou aquisição porque indicam [fluxo de caixa](/cashflow) e receita líquida. Eles também são úteis para os bancos quando uma empresa precisa solicitar um empréstimo ou linha de crédito. Geralmente, as declarações pró-forma não detalham explicitamente despesas ou vendas incomuns.

Semelhante a uma [carta de intenção](/letterofintent),. os documentos pró-forma são usados como uma espécie de precursor de uma fatura para manifestar interesse em fazer uma transação, como uma cotação de venda. Nos EUA, os documentos pró-forma que relatam os lucros de uma empresa são regidos pelos Princípios Contábeis Geralmente Aceitos (GAAP) da Securities and Exchange Commission (SEC).

## Exemplo pro forma

Ações pró-forma podem ser tão simples como cumprimentar colegas de trabalho pela manhã ou emitir um anúncio superficial. Eles também são usados com frequência pelo Congresso dos EUA para impedir que seus membros adiem por muito tempo; por uma questão de política, nenhum negócio é conduzido, mas uma sessão pró-forma pode impedir o presidente de fazer nomeações de recesso ou vetar projetos de bolso. A prática pró-forma em ambientes profissionais funciona da mesma maneira: uma ação destinada a definir os termos de um relacionamento futuro entre duas partes.

## Destaques

- Eles também podem ser usados internamente pela administração para auxiliar nas decisões de negócios.

- As finanças pro forma podem não ser compatíveis com GAAP, mas podem ser emitidas ao público para destacar determinados itens para investidores em potencial.

- Pro forma, latim para “por uma questão de forma” ou “por uma questão de forma”, é um método de cálculo de resultados financeiros usando certas projeções ou presunções.

- É ilegal que empresas de capital aberto engane investidores com resultados financeiros pro forma que não utilizam as estimativas mais conservadoras possíveis de receitas e despesas.

## PERGUNTAS FREQUENTES

### Você pode comparar declarações pro forma de diferentes empresas entre si?

Talvez, mas não é aconselhável. Como as definições de pro forma das empresas variam de acordo com seus métodos internos de previsão e elaboração de suposições, você deve ter cuidado ao comparar valores pro forma entre diferentes empresas. Se você não sabe como as empresas definem seus números pro forma, pode estar comparando inadvertidamente maçãs com laranjas.

### Qual é a diferença entre finanças pro forma e GAAP?

Não existem regras universais que as empresas devem seguir ao relatar lucros pro forma. É por isso que é importante para os investidores distinguir entre os lucros pro forma e aqueles reportados usando os princípios contábeis geralmente aceitos (GAAP). O GAAP impõe diretrizes rígidas quando as empresas relatam lucros, mas os números pro forma são melhor considerados como lucros "hipotéticos", calculados de acordo com a relevância de certos eventos ou condições. Por esta razão, os investidores devem examinar não apenas os lucros pro forma, mas também os lucros GAAP, e nunca confundir um com o outro.

### O que é uma fatura pro forma?

Uma [fatura pro forma é uma](/pro-forma-invoice) [nota de venda](/bill-of-sale) preliminar enviada aos compradores antes de uma remessa ou entrega de mercadorias. A fatura normalmente descreve os itens comprados e outras informações importantes, como o peso de envio e as despesas de transporte. Uma fatura pró-forma requer apenas informações suficientes para permitir que a alfândega determine os direitos necessários a partir de um exame geral das mercadorias incluídas.

### O que é uma Demonstração Financeira Pro Forma?

Demonstrações financeiras pro forma incorporam valores, previsões ou estimativas hipotéticas, incorporadas aos dados para fornecer uma "imagem" dos lucros de uma empresa se determinados itens não recorrentes forem excluídos. Estes são muitas vezes destinados a ser financeiros preliminares ou ilustrativos que não seguem as práticas contábeis padrão. Basicamente, as empresas usam seu próprio critério no cálculo de lucros pro forma, incluindo ou excluindo itens dependendo do que eles sentem que refletem com precisão o verdadeiro desempenho da empresa. Como as previsões pro forma são hipotéticas por natureza, elas podem se desviar dos resultados reais, às vezes de forma significativa.
