# Saiba com certeza (KST)
## O que é o Know Sure Thing (KST)?

O Know Sure Thing (KST) é um [oscilador de momento](/oscillator) desenvolvido por Martin Pring para tornar as leituras de taxa de variação mais fáceis para os traders interpretarem.

## Calculando o Know Sure Thing (KST)

O KST é calculado tomando a [média móvel simples](/sma) (SMA) de quatro diferentes períodos [de taxa de variação](/rateofchange) (ROC), somando-os para chegar ao KST e criando uma linha de sinal tomando o SMA de 9 períodos do KST.

## Entendendo a coisa certa (KST)

O indicador KST pode ser usado da mesma maneira que muitos outros osciladores de momento, como o conhecido [índice de força relativa](/rsi) (RSI). Os sinais de negociação são gerados quando o KST cruza a linha de sinal, mas os comerciantes também podem procurar convergência e divergência com o preço, condições de sobrecompra ou sobrevenda ou [cruzamentos](/crossover) da linha central.

Muitos traders combinam o indicador KST com outras formas de análise técnica para maximizar suas chances de sucesso. Por exemplo, os comerciantes podem olhar para outros indicadores sem impulso, padrões gráficos ou padrões de [velas](/candlestick) para ajudar na tomada de decisões.

> Em um artigo de 1992 **Stocks and Commodities**, Pring se referiu ao indicador como "Summed Rate of Change (KST)", mas o termo KST ficou com [os analistas técnicos](/technical-analyst).

## Destaques

- O Know Sure Thing (KST) é um oscilador de momento destinado a interpretar dados de preços de taxa de variação.

- Os sinais de negociação são gerados quando o KST cruza a linha de sinal, mas os comerciantes também procuram condições de sobrecompra ou sobrevenda.

- Os comerciantes também combinam o KST com outras análises técnicas para maximizar suas chances de uma negociação bem-sucedida.
