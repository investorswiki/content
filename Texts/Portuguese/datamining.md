# Mineração de dados
## O que é mineração de dados?

A mineração de dados é um processo usado pelas empresas para transformar dados brutos em informações úteis. Ao usar o software para procurar padrões em grandes lotes de dados, as empresas podem aprender mais sobre seus clientes para desenvolver estratégias de marketing mais eficazes, aumentar as vendas e diminuir os custos. A mineração de dados depende da [coleta de dados](/big-data),. [armazenamento](/warehousing) e processamento de computador eficazes.

## Como funciona a mineração de dados

A mineração de dados envolve explorar e analisar grandes blocos de informações para obter padrões e tendências significativos. Ele pode ser usado de várias maneiras, como marketing de banco de dados, gerenciamento de risco de crédito, detecção de fraudes, filtragem de e-mail de spam ou até mesmo para discernir o sentimento ou opinião dos usuários.

O processo de mineração de dados se divide em cinco etapas. Primeiro, as organizações coletam dados e os carregam em seus data warehouses. Em seguida, eles armazenam e gerenciam os dados, seja em servidores internos ou na nuvem. Analistas de negócios, equipes de gerenciamento e profissionais de tecnologia da informação acessam os dados e determinam como desejam organizá-los. Em seguida, o software aplicativo classifica os dados com base nos resultados do usuário e, finalmente, o usuário final apresenta os dados em um formato fácil de compartilhar, como um gráfico ou tabela.

## Software de Data Warehousing e Mineração

Os programas de mineração de dados analisam relacionamentos e padrões nos dados com base no que os usuários solicitam. Por exemplo, uma empresa pode usar um software de mineração de dados para criar classes de informações. Para ilustrar, imagine que um restaurante queira usar mineração de dados para determinar quando deve oferecer determinados pratos especiais. Ele analisa as informações coletadas e cria classes com base em quando os clientes visitam e no que eles pedem.

Em outros casos, os mineradores de dados encontram agrupamentos de informações com base em relacionamentos lógicos ou observam associações e padrões sequenciais para tirar conclusões sobre tendências no comportamento do consumidor.

O armazenamento é um aspecto importante da mineração de dados. Warehousing é quando as empresas centralizam seus dados em um banco de dados ou programa. Com um data warehouse, uma organização pode separar segmentos de dados para usuários específicos analisarem e usarem.

No entanto, em outros casos, os analistas podem começar com os dados que desejam e criar um [data warehouse](/data-warehousing) com base nessas especificações. Independentemente de como as empresas e outras entidades organizam seus dados, eles os usam para apoiar os processos de tomada de decisão da administração.

## Mineração de dados e mídias sociais

Uma das aplicações mais lucrativas da mineração de dados tem sido a das mídias sociais. Plataformas como Facebook (de propriedade da Meta), TikTok, Instagram e Twitter coletam resmas de dados sobre usuários individuais para fazer inferências sobre suas preferências para enviar anúncios de marketing direcionados. Esses dados também são usados para tentar influenciar o comportamento do usuário e alterar suas preferências, seja para um produto de consumo ou em quem ele votará em uma eleição.

A mineração de dados nas mídias sociais tornou-se um grande ponto de discórdia, com vários relatórios investigativos e exposições mostrando o quão nefastos os dados dos usuários de mineração podem ser.

> O escândalo da Cambridge Analytica é um excelente exemplo de como as empresas de mídia social podem usar a mineração de dados às custas de seus usuários.

## Exemplo de mineração de dados

As mercearias são usuários bem conhecidos de técnicas de mineração de dados. Muitos supermercados oferecem [cartões de fidelidade gratuitos](/loyalty-program) aos clientes que lhes dão acesso a preços reduzidos não disponíveis para não membros. Os cartões facilitam para as lojas rastrearem quem está comprando o quê, quando está comprando e a que preço. Depois de analisar os dados, as lojas podem usar esses dados para oferecer aos clientes cupons direcionados aos seus hábitos de compra e decidir quando colocar os itens à venda ou quando vendê-los pelo preço total.

A mineração de dados pode ser motivo de preocupação quando uma empresa usa apenas informações selecionadas, que não são representativas do grupo geral da amostra, para provar uma determinada hipótese.

> Os processos de mineração de dados são usados para criar modelos de aprendizado de máquina que potencializam aplicativos, incluindo tecnologia de mecanismo de pesquisa e programas de recomendação de sites.

## Destaques

- A mineração de dados pode ser usada por corporações para tudo, desde aprender sobre o que os clientes estão interessados ou desejam comprar até detecção de fraude e filtragem de spam.

- Os programas de mineração de dados quebram padrões e conexões nos dados com base nas informações que os usuários solicitam ou fornecem.

- A mineração de dados é o processo de análise de um grande lote de informações para discernir tendências e padrões.

- As empresas de mídia social usam técnicas de mineração de dados para mercantilizar seus usuários, a fim de gerar lucro.

- Esse uso de mineração de dados tem sido criticado ultimamente, pois os usuários muitas vezes desconhecem que a mineração de dados acontece com suas informações pessoais, especialmente quando são usadas para influenciar preferências.

## PERGUNTAS FREQUENTES

### Como é feita a mineração de dados?

A mineração de dados depende de big data e processos avançados de computação, incluindo aprendizado de máquina e outras formas de inteligência artificial (IA). O objetivo é encontrar padrões que possam levar a inferências ou previsões de conjuntos de dados grandes ou não estruturados.

### Qual é outro termo para mineração de dados?

A mineração de dados também atende pelo termo menos usado, descoberta de conhecimento em dados, ou KDD.

### Quem usa mineração de dados?

As aplicações de mineração de dados vão desde o setor financeiro à procura de padrões nos mercados até os governos que tentam identificar potenciais ameaças à segurança. Corporações, e especialmente empresas de mídia online e social, usam mineração de dados em seus usuários para criar campanhas de publicidade e marketing lucrativas que visam conjuntos específicos de usuários.
