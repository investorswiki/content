# Linux
## O que é Linux?

Linux é um popular sistema operacional (SO) de código aberto e altamente personalizável. Ele facilita a comunicação entre software e hardware, alimentando grande parte do mundo de hoje - de computadores a carros.

Criado em 1991 por Linus Torvalds, o Linux é um software de código aberto que agora faz parte de uma grande variedade de sistemas. Como código aberto, o Linux está disponível gratuitamente para uso comercial e não comercial. Qualquer pessoa pode usar, modificar e distribuir o código-fonte sob os termos de suas licenças.

Devido à sua flexibilidade, o Linux está presente em uma ampla gama de aplicações, incluindo computadores desktop, laptops, servidores web, [supercomputadores](/supercomputer),. smartphones, televisores, roteadores, câmeras digitais, instrumentos musicais e veículos.

Tecnicamente falando, o Linux é um kernel: um programa de computador que é o elemento central de um sistema operacional. Simplificando, o kernel é o que traduz o código do computador em dados binários que o hardware pode entender.

Usar o kernel Linux sozinho não se parece em nada com a experiência de computador à qual a maioria das pessoas está acostumada. Assim, desenvolvedores de todo o mundo adicionaram componentes e aplicativos adicionais para torná-lo mais fácil de usar. Assim, o Linux geralmente é distribuído como pacotes (coleções de ferramentas de software). Essas versões modificadas do Linux são chamadas de distros (abreviação de Linux Distributions).

### Quais são os tipos de distribuições Linux?

Ao longo do tempo, comunidades e empresas trabalharam com o Linux para criar sistemas que atendam às suas preferências ou necessidades. Há uma longa lista de diferentes distribuições Linux, de todo o mundo. Enquanto alguns são totalmente baseados na comunidade (como Debian ou Arch Linux), outros têm suporte corporativo (como Fedora ou openSUSE). Diferentes distros visam fazer coisas diferentes. Por exemplo, eles podem ser focados em privacidade (TAILS), usabilidade (Ubuntu) ou personalização (Arch).

### Por que alguém iria querer usar o Linux?

No geral, o Linux é um sistema versátil, eficiente e seguro. Pode ser personalizado com base nas preferências do usuário. Sua infraestrutura leve permite velocidades de computador mais rápidas e pode até ser usada para revitalizar uma máquina antiga. As muitas distribuições Linux permitem que os usuários escolham um sistema que atenda às suas necessidades - seja simplicidade, segurança, privacidade ou personalização avançada.
