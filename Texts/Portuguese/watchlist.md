# Lista de observação
## O que é uma lista de observação?

Uma lista de observação é um conjunto de [títulos](/security) que um investidor monitora para potenciais oportunidades de negociação ou investimento.

Muitas plataformas de corretagem e financeiras permitem a fácil construção e visualização de listas de observação. Uma lista de observação bem organizada pode ajudar a identificar oportunidades de negociação, acompanhar o desempenho do portfólio ou monitorar ações quentes ou populares.

## Entendendo as listas de observação

Uma lista de observação é basicamente o que parece - uma lista de ações que um investidor observa com o objetivo de aproveitar os preços se caírem o suficiente para criar uma situação interessante de desvalorização. Isso leva a lista "seguida de perto" um passo adiante. Esses são nomes que um investidor estaria disposto a comprar e possuir pelo preço certo ou com o catalisador certo (um sinal de que o crescimento voltou a funcionar, por exemplo).

Um investidor ou trader pode criar uma lista de observação de várias, dezenas ou até centenas de instrumentos de negociação para tomar decisões de investimento mais informadas. Uma lista de observação pode ajudar um investidor a rastrear empresas e ficar a par de notícias financeiras ou outras que possam afetar esses instrumentos.

Normalmente, o investidor monitora a lista, esperando que certos critérios sejam atendidos, como negociação, acima de um determinado volume, quebra de um [intervalo de 52 semanas](/52-week-range) ou movimentação acima de sua média móvel de 200 dias, antes de colocar ordens de negociação.

## Tipos de listas de observação

A maioria das plataformas de negociação permite que os usuários criem suas próprias listas de observação para os títulos em que estão interessados.

Fidelity, uma das principais plataformas de negociação online, permite aos usuários criar até quinze listas de observação diferentes, com 50 símbolos cada. Os usuários também podem configurar suas listas para alertá-los sobre quaisquer novos sinais de negociação, como mudanças repentinas de preço. Um investidor bem organizado pode usar essa funcionalidade para criar listas de observação separadas para ações, títulos, fundos mútuos ou quaisquer outros ativos negociáveis.

As listas de observação também são usadas no comércio de criptomoedas, onde oscilações acentuadas de preços às vezes podem oferecer breves oportunidades de altos lucros. Além das métricas de negociação, uma lista de observação para criptomoedas pode rastrear tokens com um próximo lançamento de [fork](/hard-fork) ou mainnet.

## Quando usar uma lista de observação

Um investidor, por exemplo, pode estar interessado em comprar ações de um determinado setor. Mas se esse setor é geralmente supervalorizado, pode oferecer poucas ações com preços atraentes.

Um investidor poderia criar uma lista de todas as ações desse setor que acompanharia várias medidas de [avaliação](/valuation),. incluindo índices como [preço-lucro](/price-earningsratio),. preço-vendas e preço-livro, entre outros. Quando uma empresa na lista atendesse a um critério de avaliação específico, como um índice de PE inferior a 15, saberia então que essa ação era uma possível candidata a investimento. Muitos sites orientados para investimentos e plataformas de corretagem permitem que visitantes ou clientes criem listas de observação online, muitas vezes gratuitamente.

Se você estiver interessado em acompanhar quaisquer valores mobiliários, poderá criar sua própria lista de observação na maioria das plataformas de corretagem.

> Muitos investidores iniciantes são tentados a rastrear o maior número possível de ações, mas uma lista de observação com centenas de entradas é muito ampla para qualquer investidor. Não tente rastrear tudo de uma vez.

## Considerações Especiais

Como regra geral, as listas de observação fornecidas pelas plataformas de corretagem podem acomodar cerca de 25 a 75 nomes, dependendo do espaço ocupado por gráficos, scanners, tickers de notícias e janelas [de profundidade de mercado .](/marketdepth) Uma lista de observação mais longa de, digamos, 200 ações provavelmente será ampla demais para quase qualquer investidor monitorar ou manter. Os investidores devem atualizar esta lista pelo menos duas vezes por mês. Afinal, isso geralmente é uma lista de nomes que um investidor está simplesmente esperando para ficar barato o suficiente para comprar.

É uma boa ideia dedicar pelo menos uma tela inteiramente aos tickers da lista de observação, com cada entrada exibindo apenas dois ou três campos, incluindo último preço, variação líquida e variação percentual. Alguns traders adicionam um único gráfico a este que liga os tickers para permitir uma rápida revisão dos padrões de preços durante o dia de negociação.

## Exemplo de uma lista de observação

Para exemplos de listas de observação de ações, **Yahoo! O Finance** oferece várias listas de observação selecionadas com critérios predefinidos, como "Most Active Penny Stocks" e "Most Shorted Stocks". Essas listas têm a vantagem adicional de atualizações automáticas para que os usuários não precisem adicionar ou remover manualmente os estoques que não atendem mais aos critérios da lista.

## A linha de fundo

As listas de observação são uma maneira fácil para os investidores rastrearem o mercado em busca de sinais e oportunidades de negociação. Agora existem muitas ferramentas para os investidores criarem suas próprias listas de observação ou seguir listas de observação selecionadas por outras partes. Ao restringir o mercado a uma lista de títulos-chave, os traders podem concentrar sua atenção para observar de perto os títulos nos quais estão interessados.

##Destaques

- Uma lista de observação é um inventário de símbolos de ticker que são monitorados para oportunidades potenciais ou para acompanhar seu desempenho.

- Algumas plataformas de negociação podem oferecer listas de observação selecionadas, de acordo com critérios que os especialistas acreditam serem importantes para os usuários da plataforma.

- As listas de observação podem ser gerenciadas e monitoradas proativamente ou visualizadas passivamente ao longo do tempo.

- Usando um screener de ações, é possível criar uma watchlist automatizada focada em uma métrica específica, como ganhos ou desempenho de longo prazo.

- A maioria das corretoras online e portais financeiros permitem a fácil construção de listas de observação.

##PERGUNTAS FREQUENTES

### O que é uma boa ferramenta de lista de ações?

Em termos de funcionalidade, as ferramentas de lista de observação mais versáteis vêm como parte de um produto de investimento pago. O TC2000, o Wealth Lab e o Trade Ideas da Worden oferecem grandes bancos de dados para clientes pagantes. Plataformas de negociação gratuitas, como Fidelity ou Robinhood, também vêm com a capacidade de criar listas de observação e acompanhar ações, embora possam rastrear menos métricas do que as versões pagas. Muitos sites de finanças on-line, como Marketwatch e TradingView, também oferecem listas de observação gratuitas e telas de ações.

### O que é uma lista de ações com curadoria?

Uma lista de observação de ações com curadoria é uma lista de observação que foi criada por um corretor ou plataforma de negociação para o benefício de seus clientes. Essas listas de observação têm a conveniência adicional de manutenção automática, de modo que o usuário final não precisa adicionar ou remover estoques que não atendem mais aos critérios da lista.

### Como você cria uma lista de observação de ações?

A maioria das plataformas de negociação on-line oferece funcionalidade para que os usuários criem suas próprias listas de observação, permitindo que eles rastreiem facilmente qualquer segurança que desperte seu interesse. Para criar uma lista de observação, você deve primeiro identificar seus principais critérios de investimento e decidir que tipos de investimentos você está procurando. Em seguida, usando um [rastreador de ações](/stockscreener) ou ferramenta semelhante, pesquise ações que se encaixem nesses critérios e adicione-as à sua lista de observação.
