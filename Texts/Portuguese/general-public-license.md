# Licença Pública Geral
GPL, ou General Public License (GPL), refere-se a uma licença copyleft gratuita para software. Ele concede a qualquer pessoa o direito de usar, ler, copiar, compartilhar, modificar e distribuir livremente um programa de computador ou outro tipo de trabalho. A GPL foi originalmente escrita para o Projeto GNU e foi a primeira licença copyleft a ser adotada para uso geral. Foi criado no início de 1989 por Richard Stallman, fundador da Free Software Foundation (FSF).

Ao contrário do copyright, o termo copyleft significa que a GPL permite a publicação de trabalhos derivados, mas exige que eles sejam distribuídos sob os mesmos termos de licença do trabalho original. Assim, os usuários não podem usar software licenciado pela GPL e liberar um trabalho derivado sob outro tipo de licença. Este não é o caso de outros tipos de licenças de software livre, como as licenças Berkley Software Distribution (BSD) e MIT.

As licenças BSD e MIT se enquadram na categoria de licenças permissivas. Embora as licenças copyleft e permissiva permitam que os usuários copiem, alterem e distribuam software, suas condições são um pouco diferentes.

Por um lado, as licenças copyleft garantem que o software de código aberto permaneça disponível para todos. Também evita que outra pessoa lucrar com um trabalho que foi disponibilizado gratuitamente. Os defensores do copyleft tendem a se preocupar mais em manter algum controle sobre seu trabalho.

Por outro lado, as licenças permissivas permitem que o software seja amplamente utilizado, desde que os desenvolvedores originais sejam referenciados e atribuídos por seu trabalho. Em outras palavras, uma licença permissiva permite que qualquer pessoa copie, altere e distribua um trabalho, sob qualquer tipo de licença. O único requisito é dar créditos aos criadores originais.

Atualmente, a licença GPL é a forma mais utilizada de distribuição de software livre e de código aberto. Programas populares de software livre como o GNU Emacs, o GNU Compiler Collection (GCC) e até mesmo o kernel [Linux](/linux) são licenciados sob esta licença.
