# Trimestre até a data (QTD)
## O que é Trimestral Até a Data (QTD)?

Trimestre até a data (QTD) é um intervalo de tempo que captura todas as atividades relevantes da empresa que ocorreram entre o início do [trimestre atual](/quarter) e o ponto em que os dados foram coletados posteriormente no trimestre. As informações trimestrais geralmente são coletadas em situações em que todo o período trimestral ainda não terminou e podem permitir que a administração veja como o trimestre está se configurando.

## Entendendo o Trimestre Até a Data (QTD)

Na linguagem financeira, um trimestre refere-se a um período de três meses durante um [ano fiscal](/fiscalyear). Como há 12 meses em um ano, há quatro trimestres: primeiro trimestre (Q1), segundo trimestre (Q2), terceiro trimestre (Q3) e quarto trimestre (Q4).

Para determinar o desempenho de uma empresa durante um determinado trimestre, a administração pode querer extrair dados do início do trimestre para uma determinada área para avaliar como ela se saiu até agora.

Por exemplo, uma empresa pode ter uma meta de [receita](/revenue) de US$ 5 milhões para o trimestre. Falta um mês para o final do trimestre e a administração gostaria de ver a receita do trimestre até a data para determinar se eles estão no caminho certo para atingir a meta de US$ 5 milhões.

Dependendo do resultado, as informações do QTD ajudam a administração a tomar a decisão de permanecer no curso, pois a empresa está no caminho certo, ou ajustar o curso porque perderá o número alvo.

As informações de QTD permitem que o gerenciamento investigue por que uma determinada métrica não está no alvo e corrija o problema. Isso é particularmente útil ao comparar os mesmos períodos com exercícios anteriores ou com outros trimestres.

## Análise de dados do trimestre até a data

Muitas empresas gastam muito tempo preparando seus relatórios trimestrais. Todas as informações devem estar limpas e livres de erros. Os relatórios de QTD usados, precisos e oportunos podem ajudar uma empresa a tomar medidas para melhorar seu desempenho.

Uma análise de QTD é mais eficaz no final do trimestre, pois há dados mais significativos disponíveis para avaliar a qualidade dos resultados. Dito isto, as mudanças necessárias são mais impactantes no início do trimestre, pois há mais tempo para efetuar as mudanças. A administração deve encontrar um equilíbrio entre os dois.

Uma análise de QTD normalmente é apenas para uso interno, e não para uso externo, porque a [Securities and Exchange Commission](/sec) (SEC) não exige que as empresas relatem informações antes do término de um trimestre. A informação QTD é uma ferramenta de gestão para determinar o andamento do negócio.

Como tal, não há muitas comparações de QTD entre as empresas porque a administração pode extrair informações em momentos diferentes dentro de um trimestre. Dito isto, os resultados trimestrais finais são muito comparáveis entre as empresas.

Ao comparar os resultados trimestrais, é importante observar que nem todas as empresas utilizam o ano civil como ano fiscal. Garantir que você está comparando exatamente o mesmo período de tempo entre as empresas e, portanto, evitando fatores [sazonais](/seasonality),. permitirá uma comparação de maçãs com maçãs.

## Destaques

- Trimestre até a data (QTD) é um intervalo de tempo que captura as informações financeiras de uma empresa desde o início de um trimestre até um ponto posterior no trimestre.

- As informações trimestrais são principalmente para uso interno e não para uso externo.

- As informações para uma análise trimestral são solicitadas antes do término de um trimestre para avaliar se a empresa está no caminho certo para cumprir os resultados trimestrais.

- As mudanças necessárias com base nos dados do trimestre, no entanto, são mais impactantes no início do trimestre.

- Uma análise trimestral é mais útil no final do trimestre, quando houver tempo suficiente para coletar dados significativos.

- As informações trimestrais permitem que a administração corrija o curso se não estiver no caminho certo para atingir as metas financeiras.
