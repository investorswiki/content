# Inventário Periódico
## O que é Inventário Periódico?

O sistema de inventário periódico é um método de avaliação de [inventário](/inventory) para fins de relatórios financeiros em que uma contagem física do inventário é realizada em intervalos específicos. Esse método contábil faz o estoque no início de um período, adiciona novas compras de estoque durante o período e deduz o estoque final para derivar o custo das mercadorias vendidas (CPV).

## Entendendo o Inventário Periódico

Sob o sistema de inventário periódico, uma empresa não conhecerá seus níveis de estoque unitário nem o CPV até que o processo de contagem física seja concluído. Esse sistema pode ser aceitável para um negócio com um número baixo de [SKUs](/stock-keeping-unit-sku) em um mercado de baixa movimentação, mas para todos os outros, o sistema de [estoque permanente](/perpetualinventory) é considerado superior pelos seguintes motivos principais:

1. O sistema perpétuo atualiza continuamente o [registro de ativos de estoque](/asset-ledger) no sistema de banco de dados de uma empresa, dando ao gerenciamento uma visão instantânea do estoque; o sistema periódico é demorado e pode produzir números obsoletos que são menos úteis para o gerenciamento.

1. O sistema perpétuo mantém o CPV atualizado à medida que ocorrem as movimentações de estoque; o sistema periódico não pode fornecer valores CPV precisos entre os períodos de contagem.

1. O sistema permanente rastreia itens de estoque individuais para que, caso haja itens defeituosos, por exemplo, a origem do problema possa ser identificada rapidamente; o sistema periódico provavelmente não permitiria uma resolução imediata.

1. O sistema perpétuo é baseado em tecnologia e os dados podem ser copiados, organizados e manipulados para gerar relatórios informativos; o sistema periódico é manual e mais propenso a erros humanos, e os dados podem ser extraviados ou perdidos.

## Considerações Especiais: CPV

O custo das mercadorias vendidas, comumente referido como CPV, é uma conta fundamental da demonstração do resultado, mas uma empresa que usa um sistema de inventário periódico não saberá o valor de seus registros contábeis até que a contagem física seja concluída.

> O CPV varia drasticamente com os níveis de estoque, pois geralmente é mais barato comprar a granel - se você tiver espaço de armazenamento para acomodar.

>

Suponha que uma empresa tenha um estoque inicial de $ 500.000 em 1º de janeiro. A empresa compra $ 250.000 em estoque durante um período de três meses e, após uma conta de estoque físico, determina que possui um estoque final de $ 400.000 em 31 de março, que se torna o estoque inicial montante para o próximo trimestre. O CPV para o primeiro trimestre do ano é de $ 350.000 ($ 500.000 inicial + $ 250.000 compras - $ 400.000 final).

Devido às discrepâncias de tempo, torna-se ônus do gerente ou proprietário da empresa responsável pelo monitoramento do estoque do período se fizer sentido para seus resultados alocar horas para contar o estoque diariamente, semanalmente, mensalmente ou anualmente.
