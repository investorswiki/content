# linha de tendência
## O que é uma linha de tendência?

As linhas de tendência são linhas facilmente reconhecíveis que os traders desenham em gráficos para conectar uma série de preços ou mostrar o melhor ajuste de alguns dados. A linha resultante é então usada para dar ao trader uma boa ideia da direção em que o valor de um investimento pode se mover.

Uma linha de tendência é uma linha desenhada sobre as máximas do pivô ou sob as mínimas do pivô para mostrar a direção predominante do preço. As linhas de tendência são uma representação visual de suporte e resistência em qualquer período de tempo. Eles mostram a direção e a velocidade do preço e também descrevem padrões durante os períodos de contração dos preços.

## O que as Trendlines dizem para você?

A linha de tendência está entre as ferramentas mais importantes utilizadas pelos analistas técnicos. Em vez de olhar para o desempenho dos negócios anteriores ou outros [fundamentos](/fundamentalanalysis),. os analistas técnicos procuram tendências na [ação do preço](/price-action). Uma linha de tendência ajuda os analistas técnicos a determinar a direção atual dos preços de mercado. Os analistas técnicos acreditam que a tendência é sua amiga, e identificar essa tendência é o primeiro passo no processo de fazer uma boa negociação.

Para criar uma linha de tendência, um analista deve ter pelo menos dois pontos em um gráfico de preços. Alguns analistas gostam de usar prazos diferentes, como um minuto ou cinco minutos. Outros olham para [gráficos diários](/dailychart) ou gráficos semanais. Alguns analistas reservam o tempo completamente, optando por visualizar as tendências com base em intervalos de [ticks em](/tick) vez de intervalos de tempo. O que torna as linhas de tendência tão universais em uso e apelo é que elas podem ser usadas para ajudar a identificar tendências, independentemente do período de tempo, período de tempo ou intervalo usado.

Se a empresa A estiver negociando a $ 35 e passar para $ 40 em dois dias e $ 45 em três dias, o [analista](/analyst) tem três pontos para traçar em um gráfico, começando em $ 35, depois passando para $ 40 e depois para $ 45. Se o analista traçar uma linha entre os três pontos de preço, eles terão uma tendência de alta. A linha de tendência desenhada tem uma inclinação positiva e, portanto, está dizendo ao analista para comprar na direção da tendência. Se o preço da empresa A passar de $ 35 para $ 25, no entanto, a linha de tendência tem uma inclinação negativa e o analista deve vender na direção da tendência.

As linhas de tendência são relativamente fáceis de usar. Um trader simplesmente precisa traçar os dados de preço normalmente, usando abertura, fechamento, alta e baixa. Nesse caso, um trader pode optar por entrar em uma posição longa perto da linha de tendência e depois estendê-la no futuro. Se a ação do preço violar a linha de tendência no lado negativo, o trader pode usar isso como um sinal para fechar a posição. Isso permite que o trader saia quando a tendência que eles estão seguindo começa a enfraquecer.

As linhas de tendência são, obviamente, um produto do período de tempo. No exemplo acima, um trader não precisa redesenhar a linha de tendência com muita frequência. Em uma escala de minutos, no entanto, as linhas de tendência e as negociações podem precisar ser reajustadas com frequência.

## A diferença entre linhas de tendência e canais

Mais de uma linha de tendência pode ser aplicada a um gráfico. Os comerciantes geralmente usam uma linha de tendência conectando altas por um período, bem como outra para conectar baixas, a fim de criar [canais](/channel). Um canal adiciona uma representação visual de suporte e resistência para o período de tempo que está sendo analisado. Semelhante a uma única linha de tendência, os traders estão procurando um pico ou uma [quebra](/breakout) para tirar a ação do preço do canal. Eles podem usar essa violação como um ponto de saída ou um ponto de entrada, dependendo de como estão configurando sua negociação.

## Limitações de uma linha de tendência

As linhas de tendência têm limitações compartilhadas por todas as ferramentas de gráficos, pois precisam ser reajustadas à medida que mais dados de preços chegam. Uma linha de tendência às vezes dura muito tempo, mas eventualmente a ação do preço se desviará o suficiente para precisar ser atualizada. Além disso, os traders geralmente escolhem diferentes pontos de dados para se conectar. Por exemplo, alguns traders usarão os mínimos mais baixos, enquanto outros podem usar apenas os preços de fechamento mais baixos por um período. Por último, as linhas de tendência aplicadas em prazos menores podem ser sensíveis ao volume. Uma linha de tendência formada em baixo volume pode ser facilmente quebrada à medida que o volume aumenta ao longo de uma sessão.

##Destaques

- As linhas de tendência indicam o melhor ajuste de alguns dados usando uma única linha ou curva.

- O período de tempo que está sendo analisado e os pontos exatos usados para criar uma linha de tendência variam de trader para trader.

- As linhas de tendência podem ser aplicadas aos altos e baixos para criar um canal.

- Uma única linha de tendência pode ser aplicada a um gráfico para fornecer uma imagem mais clara da tendência.
