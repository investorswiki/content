# Frase Semente
Uma frase de semente é uma coleção de palavras que podem ser usadas para acessar sua carteira de criptomoedas.

Você ouvirá frequentemente os termos semente mnemônica ou frase mnemônica usados para descrever a mesma ideia. Alguns acreditam que estes são descritores pobres, no entanto, pois implicam que a frase precisa ser memorizada.

Introduzidas no Bitcoin com BIP39, as frases iniciais fornecem aos usuários uma maneira mais fácil de fazer backup de suas carteiras. Se você já usou uma carteira anteriormente, provavelmente foi solicitado a escrever de 12 a 24 palavras aleatórias como as seguintes:

> **esqueça a asa siga o flip andorinha alcançar a visão correta testemunha do jantar híbrido orgulhoso**

Se você perder o acesso à sua carteira, eles podem ser carregados em qualquer carteira compatível com BIP32 para recuperar seus fundos. Existem 2048 palavras na lista de palavras BIP39, o que significa que uma string de 12 palavras teria 128 bits de segurança. Isso significa que um invasor precisaria realizar 2128 operações para adivinhar uma semente de 12 palavras. Pode parecer um número pequeno, mas certamente não é. É praticamente inviável para os invasores com mais recursos quebrar essa sequência (por enquanto). Quanto mais longa a semente, mais segura ela é.

As frases semente – ao contrário das chaves privadas – deixam menos espaço para erros humanos quando se trata de backups, pois são mais fáceis de gravar e transmitir. Melhor ainda, eles são capazes de gerar não apenas uma, mas uma quantidade colossal de chaves a partir de uma chave mestre. Isso permite que os usuários evitem a reutilização de endereços, pois, com uma única semente, eles podem criar um número aparentemente infinito de endereços de recebimento.

As frases de semente não são específicas de criptomoeda. Eles podem ser usados em centenas, para que uma semente possa realmente ser usada para recuperar um portfólio inteiro de moedas e tokens. A maioria das carteiras líderes permite que o usuário recupere seus fundos com uma frase inicial, desde que a tenha registrado corretamente.
