# Moeda base
## O que é a Moeda Base?

No mercado [cambial](/forex),. os preços unitários da moeda são cotados como pares de moedas. A moeda base – também chamada de moeda de transação – é a primeira moeda que aparece na cotação de um par de moedas, seguida pela segunda parte da cotação, chamada de moeda de [cotação](/quotecurrency) ou moeda de balcão. Para fins contábeis, uma empresa pode usar a moeda base como moeda nacional ou moeda [contábil](/accounting-currency) para representar todos os lucros e perdas.

## QUEBRANDO A Moeda Base

No forex, a moeda base representa quanto da moeda de cotação é necessária para você obter uma unidade da moeda base. Por exemplo, se você estivesse olhando para o par de moedas CAD/USD, o dólar canadense seria a moeda base e o dólar americano seria a moeda de cotação.

As abreviaturas usadas para moedas são prescritas pela [Organização Internacional para Padronização (ISO).](/international-organization-for-standardization-iso) Esses códigos são fornecidos no padrão ISO 4217. Os pares de moedas usam esses códigos feitos de três letras para representar uma moeda específica. As moedas que constituem um par de moedas às vezes são separadas por uma barra. A barra pode ser omitida ou substituída por um ponto, um traço ou nada.

Os principais códigos de moeda incluem USD para o dólar americano, EUR para o [euro](/euro),. JPY para o iene japonês, [GBP](/gbp) para a libra esterlina, AUD para o dólar australiano, CAD para o dólar canadense e CHF para o franco suíço.

## Partes de um par de moedas

No forex, os pares de moedas são escritos como XXX/YYY ou simplesmente XXXYYY. Aqui, XXX é a moeda base e YYY é a moeda de cotação. Exemplos desses formatos são GBP/AUD, EUR/USD, USD/JPY, GBPJPY, EURNZD e EURCHF.

Quando fornecido com uma [taxa](/exchangerate) de câmbio , os pares de moedas indicam quanto da moeda de cotação é necessária para comprar uma unidade da moeda base fornecida. Por exemplo, ler EUR/USD = 1,55 significa que _1 é igual a $ 1,55. Isso diz diretamente que, para comprar _1, o comprador deve pagar US$ 1,55. A cotação do par de moedas é lida da mesma maneira ao vender a moeda base. Se um [vendedor](/seller) quiser vender _1, ele receberá $ 1,55 por isso.

## Movimento simultâneo

As cotações de Forex são apresentadas em pares porque os investidores compram e vendem moedas simultaneamente. Por exemplo, quando um comprador compra EUR/USD, significa basicamente que ele está comprando euros e vendendo dólares americanos ao mesmo tempo. Os investidores compram o par se acharem que a moeda base ganhará valor em comparação com a moeda de cotação. Por outro lado, eles vendem o par se acharem que a moeda base perderá valor em comparação com a moeda cotada.
