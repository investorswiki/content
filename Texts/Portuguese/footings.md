# fundamentos
## O que são sapatas?

Na contabilidade, um footing é o saldo final ao somar todos os débitos e créditos. Os débitos são computados, seguidos pelos créditos, e os dois são compensados para calcular o saldo da conta. As sapatas são comumente usadas na contabilidade para determinar os saldos finais a serem colocados nas [demonstrações financeiras](/financial-statements).

## Entendendo os fundamentos

À medida que as transações comerciais são registradas, os contadores inserem os valores para cada tipo de transação em uma das duas colunas. Cada transação é registrada como um débito ou crédito. As colunas de débito e crédito são somadas, sendo o total de cada coluna um rodapé.

Os dois fundamentos são compensados juntos para calcular o saldo da conta para o período. O saldo da conta é o valor que é transportado para as demonstrações financeiras. O processo é repetido para cada tipo de transação. O termo "rodapé" é apropriado porque os totais estão localizados no final de cada coluna.

##Destaques

- As sapatas são comumente usadas na contabilidade para determinar os saldos finais das contas, que são relatados nas demonstrações financeiras de uma empresa.

- Uma vez que os débitos e créditos são totalizados, os dois números - ou fundamentos - são compensados para calcular o saldo da conta.

- Na contabilidade, um footing é o saldo final ao somar todos os débitos e créditos.
