# Lista de Materiais (BOM)
## O que é uma lista de materiais (BOM)?

Uma lista de materiais (BOM) é uma extensa lista de [matérias-primas](/rawmaterials),. componentes e instruções necessárias para construir, fabricar ou reparar um produto ou serviço. Uma lista de materiais geralmente aparece em um formato hierárquico, com o nível mais alto exibindo o produto acabado e o nível inferior mostrando componentes e materiais individuais.

Existem diferentes tipos de listas de materiais específicos para engenharia usados no processo de projeto; eles também são específicos para a fabricação usada no processo de montagem.

## Entendendo uma lista de materiais (BOM)

Uma lista de materiais (BOM) é uma fonte centralizada de informações usada para fabricar um produto. É uma lista dos itens necessários para criar um produto, bem como as instruções sobre como montar esse produto. Os fabricantes que constroem produtos iniciam o processo de montagem criando uma BOM.

Criar uma lista de materiais precisa (BOM) é vital porque garante que as peças estejam disponíveis quando necessário, além de garantir que o processo de montagem seja o mais eficiente possível. Se a BOM não for precisa, pode causar a interrupção da produção, o que aumenta [os custos operacionais](/operating-cost),. pois é necessário tempo para localizar peças faltantes, iniciar outra ordem de produção ou até que o processo correto de montagem seja determinado.

Os diferentes tipos de listas de materiais (BOMs) dependem do tipo de projeto e das necessidades do negócio. As áreas comuns que utilizam BOMs são engenharia, design, operações, fabricação e muito mais. Uma lista técnica de fabricação é essencial para projetar sistemas de [planejamento de recursos empresariais](/erp) (ERP) e [planejamento de requisitos de materiais](/mrp) (MRP).

Os itens incluídos em uma BOM são o número da peça, nome da peça, quantidade, unidade de medida, referências de montagem, método de construção das peças e notas adicionais.

## Exibições de listas de materiais (BOMs)

Uma lista técnica exibe suas informações de duas maneiras: uma exibição de explosão ou uma exibição de implosão. Uma explosão de lista de materiais (BOM) exibe uma montagem no nível mais alto dividida em seus componentes individuais e peças no nível mais baixo, enquanto uma implosão de BOM exibe a ligação de peças individuais no nível inferior a uma montagem no nível superior.

Por exemplo, um computador é explodido em discos rígidos, chips de computador, painéis de memória de acesso aleatório e processadores. Cada processador é explodido em uma unidade aritmética, uma unidade de controle e um registrador. Os requisitos para a unidade aritmética, unidade de controle e registro são implodidos nos requisitos do processador, que são implodidos nos requisitos de todo o computador.

## Tipos de listas de materiais (BOMs)

Uma lista de BOM é necessária ao construir um produto e solicitar peças de reposição e reduz possíveis problemas se forem necessários reparos no produto. Ajuda a planejar os pedidos de aquisição e reduz a possibilidade de erros. Os dois principais tipos de BOMs são BOMs de engenharia e BOMs de fabricação.

Uma lista de materiais de engenharia define o design do produto acabado. Inclui todos os números de peças alternativos e [substitutos](/substitute) e peças contidas nas notas de desenho. Cada linha da lista de materiais (BOM) inclui o código do produto, nome da peça, número da peça, revisão da peça, descrição, quantidade, unidade de medida, tamanho, comprimento, peso e especificações ou recursos do produto.

A lista técnica de engenharia geralmente é organizada por engenheiros com base em um desenho de projeto auxiliado por computador (CAD). Para um produto acabado, pode haver mais de uma BOM de engenharia criada. Isso faz parte do [gerenciamento do ciclo de vida do produto](/product-life-cycle-management).

Uma lista de materiais de fabricação (BOM) compreende todas as montagens e peças necessárias para construir um item acabado pronto para ser enviado. Também incorpora os materiais de embalagem necessários para enviar o produto ao cliente. Ele contém processos que exigem execução no produto antes da conclusão e armazena todas as informações necessárias para [as atividades de fabricação](/manufacturing-production).

##Destaques

- As listas de materiais (BOMs) podem ser apresentadas como um display de explosão ou um display de implosão.

- Os dois principais tipos de listas de materiais (BOMs) são as listas de materiais de fabricação (BOMs) e as listas de materiais de engenharia (BOMs).

- Uma lista de materiais (BOM) é uma fonte centralizada de informações contendo uma lista de itens usados para fabricar um produto e as instruções sobre como fazê-lo.

- Frequentemente mostrada de forma hierárquica, uma lista de materiais (BOM) lista o produto acabado no topo, até os componentes e materiais individuais.
