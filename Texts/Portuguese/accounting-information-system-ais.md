# Sistema de Informação Contábil (AIS)
## O que é um Sistema de Informação Contábil (AIS)?

Um sistema de informações contábeis (AIS) envolve a coleta, armazenamento e processamento de dados financeiros e contábeis usados por usuários internos para relatar informações a [investidores](/investor),. [credores](/creditor) e autoridades fiscais. Geralmente é um método baseado em computador para rastrear a atividade contábil em conjunto com os recursos de tecnologia da informação. Um AIS combina [práticas contábeis tradicionais](/accounting-practice),. como o uso de Princípios Contábeis [Geralmente](/gaap) [Aceitos (GAAP)](/gaap),. com recursos modernos de tecnologia da informação.

## Como um sistema de informação contábil (AIS) é usado

Um sistema de informação contábil contém vários elementos importantes no [ciclo](/accounting-cycle) [contábil](/accounting-cycle). Embora as informações contidas em um sistema variem entre setores e tamanhos de negócios, um AIS típico inclui dados relacionados a [receitas](/revenue),. [despesas](/expense),. informações de clientes, informações de funcionários e informações fiscais. Dados específicos incluem pedidos de vendas e relatórios de análise, requisições de compra, faturas, registros de cheques, estoque, folha de pagamento, razão, balancete e informações de demonstrações financeiras.

Um sistema de informações contábeis deve ter uma estrutura de banco de dados para armazenar informações. Essa estrutura de banco de dados normalmente é programada com linguagem de consulta que permite a manipulação de tabelas e dados. Um AIS tem vários campos para entrada de dados, bem como para editar dados armazenados anteriormente. Além disso, os sistemas de informações contábeis geralmente são plataformas altamente seguras com medidas preventivas tomadas contra vírus, hackers e outras fontes externas que tentam coletar informações. [A cibersegurança](/cybersecurity) é cada vez mais importante à medida que mais e mais empresas armazenam seus dados eletronicamente.

As várias saídas de um sistema de informações contábeis exemplificam a versatilidade de seus recursos de manipulação de dados. Um AIS produz relatórios incluindo relatórios de vencimento de [contas a receber](/accountsreceivable) com base nas informações do cliente, cronogramas de [depreciação](/depreciation) para ativos fixos e balancetes para relatórios financeiros. Listas de clientes, cálculos de impostos e níveis de estoque também podem ser reproduzidos. No entanto, correspondências, memorandos ou apresentações não são incluídos no AIS porque esses itens não estão diretamente relacionados aos relatórios financeiros ou à escrituração de uma empresa.

## Benefícios dos Sistemas de Informação Contábil

### Interface interdepartamental

Um sistema de informações contábeis se esforça para interagir entre vários departamentos. Dentro do sistema, o departamento de vendas pode carregar o orçamento de vendas. Essas informações são usadas pela equipe de gerenciamento de estoque para realizar contagens de estoque e comprar materiais. Após a compra do estoque, o sistema pode notificar o departamento de [contas a pagar](/accountspayable) sobre a nova fatura. Um AIS também pode compartilhar informações sobre um novo pedido para que os departamentos de fabricação, expedição e atendimento ao cliente estejam cientes da venda.

###Controles internos

Uma parte integrante dos sistemas de informação contábil diz respeito aos [controles internos](/internalcontrols). Políticas e procedimentos podem ser colocados dentro do sistema para garantir que informações confidenciais de clientes, fornecedores e negócios sejam mantidas dentro de uma empresa. Por meio do uso de aprovações de acesso físico, requisitos de login, logs de acesso, autorizações e segregação de funções, os usuários podem ser limitados apenas às informações relevantes necessárias para desempenhar sua função comercial.
