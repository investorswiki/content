# Linha de base
## O que é uma linha de base?

Uma linha de base é um ponto de referência fixo usado para fins de comparação. Nos negócios, o sucesso de um projeto ou produto geralmente é medido em relação a um número de linha de base para custos, vendas ou qualquer outra variável. Um projeto pode exceder um número de linha de base ou não atingi-lo.

Por exemplo, uma empresa que deseja medir o sucesso de uma linha de produtos pode usar o número de unidades vendidas durante o primeiro ano como base para medir as vendas anuais subsequentes. A linha de base serve como ponto de partida contra o qual todas as vendas futuras são medidas.

## Entendendo uma linha de base

Uma linha de base pode ser qualquer número que sirva como um ponto de partida razoável e definido para fins de comparação. Ele pode ser usado para avaliar os efeitos de uma mudança, acompanhar o progresso de um projeto de melhoria ou medir a diferença entre dois períodos de tempo.

Por exemplo, uma [empresa de capital aberto](/publiccompany) acompanhará o desempenho de cada linha de produtos escolhendo um ano como linha de base e medindo todos os anos subsequentes em relação a ele.

Uma linha de base é normalmente usada quando uma demonstração financeira ou análise orçamentária é preparada. A declaração ou análise usa receitas e gastos existentes como base para avaliar se um novo projeto foi implementado com sucesso.

### A Linha de Base na Análise das Demonstrações Financeiras

Uma análise de demonstração financeira que usa uma linha de base é chamada de [análise](/horizontalanalysis) [horizontal](/horizontalanalysis). Ele compara as informações financeiras históricas de uma empresa em vários períodos de relatório que podem ser mensais, trimestrais ou anuais.

O primeiro período em uma análise horizontal é indicado como o período de linha de base. Todos os períodos subsequentes são então medidos como uma porcentagem da linha de base. Portanto, um período que tenha a mesma receita que a linha de base teria 100% de receita.

> Em tecnologia da informação, existem três pontos de referência comumente usados: custo, escopo e cronograma.

>

Este exercício é útil para identificar tendências, observar áreas de crescimento ou declínio e avaliar o desempenho financeiro geral. Índices como margem de lucro também são comparados horizontalmente com o ano de referência para tirar conclusões sobre o desempenho contínuo de uma empresa.

### A linha de base no orçamento

O orçamento do projeto funciona a partir do que é conhecido como linha de base de custos. A linha de base de custo é o orçamento aprovado para o projeto, geralmente dividido em alguns detalhes por categoria de custo e período de custo.

Se uma empresa abre um novo depósito, por exemplo, e a linha de base de custo foi definida em US$ 100.000 por mês todos os meses durante 10 meses, qualquer custo mensal superior a US$ 100.000 é uma [bandeira vermelha](/redflag) para o analista de orçamento.

No entanto, os custos do projeto flutuam inevitavelmente em relação aos números da linha de base como despesas desconhecidas e inesperadas ou até mesmo, em alguns casos, economias são realizadas. A linha de base de custo pode ser atualizada para refletir os custos reais do projeto.

### A Linha de Base em Tecnologia da Informação

No gerenciamento de tecnologia da informação, uma linha de base pode ser definida para níveis de desempenho antecipados ou máximos. Existem três pontos de linha de base comumente usados: custo, [escopo](/scope) e cronograma.

Os aplicativos de software usados por profissionais de gerenciamento de projetos geralmente são projetados para manter e rastrear essas três medidas críticas de linha de base.

##Destaques

- Na análise financeira horizontal, os números do primeiro período do relatório servem como linhas de base para comparação dos períodos subsequentes.

- No orçamento do projeto, os números do orçamento aprovado são as linhas de base para comparação das despesas reais.

- Na gestão de tecnologia da informação, a linha de base é o nível de desempenho previsto ou máximo.
