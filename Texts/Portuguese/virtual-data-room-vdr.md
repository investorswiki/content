# Sala de dados virtual (VDR)
## O que é uma sala de dados virtual?

Uma sala de dados virtual (VDR), também conhecida como sala de negócios, é um repositório online seguro para armazenamento e distribuição de documentos. Normalmente, é utilizado durante o processo de due diligence que antecede uma fusão ou aquisição para revisar, compartilhar e divulgar a documentação da empresa.

## Noções básicas sobre salas de dados virtuais

As salas de dados virtuais têm substituído cada vez mais as salas de dados físicas tradicionalmente usadas para divulgar e compartilhar documentos. Com a [globalização](/globalization) dos negócios e o aumento do escrutínio para reduzir custos, as salas de dados virtuais são uma alternativa atraente às salas de dados físicas. As salas de dados virtuais são amplamente acessíveis, imediatamente disponíveis e mais seguras.

À medida que as preocupações com a segurança crescem e os incidentes com violações aumentam, os provedores de VDR estão desenvolvendo bancos de dados mais sofisticados e confiáveis. [Ofertas públicas iniciais](/ipo) (IPOs), operações de auditoria e parcerias ou outros negócios que devem trabalhar em conjunto e compartilhar informações usarão salas de dados virtuais.

## Usos de salas de dados virtuais

[fusões e aquisições](/mergersandacquisitions) (M&A) são o uso mais comum de VDRs. Esses repositórios fornecem um local para a devida diligência necessária durante a finalização do negócio. Essas transações comerciais envolvem grandes quantidades de documentos, muitos dos quais são confidenciais e contêm informações confidenciais. O uso de um VDR é uma maneira segura e confiável para todas as partes interessadas revisarem e trocarem documentos à medida que se envolvem em negociações.

As empresas muitas vezes trabalham umas com as outras para produzir e fabricar produtos durante a construção de um edifício e para oferecer serviços. Formar e manter essas relações comerciais requer contratos e a transmissão frequente de dados. As salas de dados virtuais permitem o armazenamento desses contratos e disponibilizam prontamente os documentos necessários para a continuidade das parcerias comerciais. Como exemplo, as alterações feitas nas plantas de uma estrutura por um engenheiro ficam imediatamente disponíveis para todos os empreiteiros envolvidos no projeto.

[A auditoria](/audit) das práticas, conformidade e contas da empresa é uma prática comum em todos os negócios. Este processo é frequentemente um problema, pois os trabalhadores devem interagir com reguladores e ajustadores externos. Além disso, hoje muitas empresas têm escritórios em locais remotos e em todo o mundo em vários fusos horários.

O uso de um data room virtual permite que advogados, contadores, reguladores internos e externos e demais interessados tenham um ponto de acesso centralizado. Fornecer um sistema central reduz erros e tempo. Além disso, prevê transparência na comunicação. Dependendo do tipo de auditoria, o nível de acesso e a autoridade variam.

Oferecer uma oferta pública inicial (IPO) é uma tarefa assustadora que exige uma quantidade inconcebível de papelada. Assim como as auditorias, a transparência é essencial. As empresas devem criar, trocar, reter e gerenciar grandes volumes de documentos. Devido à natureza da transação, a maioria dos usuários terá acesso restrito, como "somente visualização". A capacidade de copiar, encaminhar ou imprimir pode ser proibida.

## Alternativa a um VDR

Embora as salas de dados virtuais ofereçam muitos benefícios, elas não são adequadas para todos os setores. Por exemplo, alguns governos podem optar por continuar usando salas de dados físicas para trocas de informações altamente confidenciais. Os danos de possíveis ataques cibernéticos e [violações de dados](/data-breach) excedem os benefícios oferecidos pelas salas de dados virtuais. Os resultados de tais eventos podem ser cataclísmicos se as partes ameaçadoras acessarem informações classificadas. Nesses casos, o uso de um VDR não será considerado.

##Destaques

- Geralmente, ações como copiar, imprimir e encaminhar são desabilitadas em VDRs.

- Os VDRs são comumente usados por empresas quando estão se fundindo, trabalhando em um projeto ou outra joint venture que requer acesso a dados compartilhados.

- Os VDRs são considerados mais seguros do que os documentos físicos, pois não há risco de perda durante o trânsito ou de serem destruídos acidentalmente.

- Salas de dados virtuais, ou VDRs, existem como uma maneira segura de armazenar documentos que várias pessoas precisam acessar simultaneamente.
