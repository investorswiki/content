# Significado estatístico
## O que é significância estatística?

A significância estatística é uma determinação feita por um analista de que os resultados nos dados não são explicáveis apenas por acaso. O teste estatístico [de hipóteses](/hypothesistesting) é o método pelo qual o analista faz essa determinação. Este teste fornece um [valor p](/p-value),. que é a probabilidade de observar resultados tão extremos quanto os dos dados, assumindo que os resultados são realmente devidos apenas ao acaso. Um valor de p de 5% ou inferior é frequentemente considerado estatisticamente significativo.

## Entendendo a significância estatística

A significância estatística é uma determinação sobre a [hipótese nula](/null_hypothesis),. o que sugere que os resultados são devidos apenas ao acaso. Um conjunto de dados fornece significância estatística quando o valor p é suficientemente pequeno.

Quando o valor-p é grande, então os resultados nos dados são explicáveis apenas por acaso, e os dados são considerados consistentes (embora não comprovem) a hipótese nula.

Quando o valor de p é suficientemente pequeno (tipicamente 5% ou menos), os resultados não são facilmente explicados pelo acaso e os dados são considerados inconsistentes com a hipótese nula. Nesse caso, a hipótese nula do acaso apenas como explicação dos dados é rejeitada em favor de uma explicação mais sistemática.

> A significância estatística é frequentemente usada para novos testes de medicamentos farmacêuticos, para testar vacinas e no estudo de patologia para testes de eficácia e para informar os investidores sobre o sucesso da empresa no lançamento de novos produtos.

>

## Exemplos de significância estatística

Suponha que Alex, um analista financeiro, esteja curioso para saber se alguns investidores tinham conhecimento prévio da falência repentina de uma empresa. Alex decide comparar a média dos retornos diários do mercado antes do fracasso da empresa com aqueles depois para ver se há uma diferença estatisticamente significativa entre as duas médias.

O valor-p do estudo foi de 28% (>5%), indicando que uma diferença tão grande quanto a observada (-0,0033 a +0,0007) não é incomum sob a explicação apenas do acaso. Assim, os dados não forneceram evidências convincentes de conhecimento prévio da falha. Por outro lado, se o valor de p fosse 0,01% (muito menos de 5%), então a diferença observada seria muito incomum sob a explicação apenas do acaso. Nesse caso, Alex pode decidir rejeitar a hipótese nula e investigar mais a fundo se alguns traders tinham conhecimento prévio.

A significância estatística também é usada para testar novos produtos médicos, incluindo medicamentos, dispositivos e vacinas. Relatórios de significância estatística disponíveis publicamente também informam os investidores sobre o sucesso da empresa no lançamento de novos produtos.

Digamos, por exemplo, que um líder farmacêutico em medicamentos para diabetes relatou que houve uma redução estatisticamente significativa no diabetes tipo 1 quando testou sua nova insulina. O teste consistiu em 26 semanas de terapia randomizada entre pacientes com diabetes, e os dados deram um valor de p de 4%. Isso significa para investidores e agências reguladoras que os dados mostram uma redução estatisticamente significativa no diabetes tipo 1.

> Os preços das ações das empresas farmacêuticas são frequentemente afetados por anúncios da significância estatística de seus novos produtos.

>

##Destaques

- A significância estatística é usada para fornecer evidências sobre a plausibilidade da hipótese nula, que levanta a hipótese de que não há nada mais do que chance aleatória em ação nos dados.

- O teste de hipótese estatística é usado para determinar se o resultado de um conjunto de dados é estatisticamente significativo.

- A significância estatística é uma determinação de que uma relação entre duas ou mais variáveis é causada por algo diferente do acaso.

- Geralmente, um valor de p de 5% ou menos é considerado estatisticamente significativo.

##PERGUNTAS FREQUENTES

### Como a significância estatística é determinada?

O teste de hipótese estatística é usado para determinar se os dados são estatisticamente significativos. Em outras palavras, se o fenômeno pode ou não ser explicado apenas como um subproduto do acaso. A significância estatística é uma determinação sobre a hipótese nula, que postula que os resultados são devidos apenas ao acaso. A rejeição da hipótese nula é necessária para que os dados sejam considerados estatisticamente significativos.

### O que é P-Value?

Um valor p é uma medida da probabilidade de que uma diferença observada poderia ter ocorrido apenas por acaso. Quando o valor de p é suficientemente pequeno (por exemplo, 5% ou menos), então os resultados não são facilmente explicados pelo acaso e a hipótese nula pode ser rejeitada. Quando o valor-p é grande, então os resultados nos dados são explicáveis apenas por acaso, e os dados são considerados consistentes com (enquanto provam) a hipótese nula.

### Como a significância estatística é usada?

A significância estatística é frequentemente usada para testar a eficácia de novos produtos médicos, incluindo medicamentos, dispositivos e vacinas. Relatórios de significância estatística disponíveis publicamente também informam os investidores sobre o sucesso da empresa no lançamento de novos produtos. Os preços das ações das empresas farmacêuticas são frequentemente fortemente afetados pelos anúncios da significância estatística de seus novos produtos.
