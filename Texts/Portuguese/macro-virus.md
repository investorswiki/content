# Vírus de macro
## O que é um vírus de macro?

Um vírus de macro infecta um programa de software e faz com que uma série de ações comece automaticamente quando o programa é aberto. Ele funciona como uma [macro](/macro-environment) normal e geralmente se instala no lugar de uma macro existente.

## Entendendo os vírus de macro

Os vírus de macro se originam na internet e se infiltram em programas já parados no computador de uma pessoa. Para entender completamente o que é um vírus de macro, no entanto, é importante primeiro entender o que é uma "macro":

Uma macro é uma série de comandos que aciona automaticamente uma função específica dentro de um programa de software. Macros podem ser instaladas em programas como o Microsoft Word para executar tarefas complexas que o programa não conseguiria fazer automaticamente. Por exemplo, uma empresa pode usar uma macro para inserir automaticamente o papel timbrado projetado ou tabelas pré-projetadas em modelos de página do Microsoft Word ou usar formatos de página personalizados que não estão disponíveis com o programa por si só.

Os vírus de macro são codificados com a capacidade de se espalhar - da mesma forma que um vírus infecta uma pessoa, se replica e se espalha para outras pessoas. Um vírus de macro pode replicar e instalar material em um computador sem o conhecimento ou consentimento do usuário. Se enviado por e-mail de spam, ele geralmente se envia automaticamente para todos no catálogo de endereços desse usuário.

## Como um vírus de macro infecta um computador?

Os vírus de macro se originam na Internet, geralmente em e-mails de spam, e se infiltram em programas instalados no computador de uma pessoa, como o Microsoft Office, imitando uma macro benigna. Eles geralmente substituem macros pré-instaladas e são ativados quando a macro normal é executada, mas podem operar mesmo quando o programa não está em uso sem o conhecimento do usuário. As ações automáticas que eles acionam podem variar desde a adição de texto incontrolável a um documento até o envio de mensagens de spam para pessoas nos catálogos de endereços do usuário.

Os vírus de macro geralmente são inofensivos para o computador ou para o próprio programa, mas causam confusão e frustração para o usuário. Uma vez instalados em um computador, eles podem se espalhar por vários programas se não estiverem contidos.

## Como prevenir um vírus de macro

A maioria dos pacotes de software antivírus atuais são projetados para localizar e destruir qualquer vírus de macro existente em um computador e impedir que novos vírus criem raízes. No entanto, nem todos os vírus de macro são detectados pelo software antivírus e nem todos os softwares antivírus são iguais. É importante manter o software antivírus atualizado, mas é igualmente importante estar ciente do que você baixa ou abre da Internet.

Por exemplo, você não deve abrir um anexo em um e-mail enviado de um endereço que você não conhece. O spam de e-mail pode ser facilmente identificável, mas também pode usar truques para fazer uma pessoa pensar que o conteúdo é legítimo.

## Destaques

- Além de causar estragos em um sistema, esses vírus de macro também são programados para se replicar e se espalhar para infectar outros sistemas.

- Os vírus de macro geralmente se espalham on-line por meio de links falsos da Web ou como anexos de e-mail.

- O software antivírus normalmente é bom para detectar e remover vírus de macro antes que eles possam causar danos.

- Um vírus de macro é um tipo de exploração de computador ou malware que aciona automaticamente uma série de funções de software, muitas vezes com efeito deletério.
