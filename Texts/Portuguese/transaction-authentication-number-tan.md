# Número de autenticação de transação (TAN)
## O que é um Número de Autenticação de Transação (TAN)

Um número de autenticação de transação é um código de uso único usado no processamento de transações online. Um número de autorização de transação (TAN) representa uma camada adicional de segurança além de uma senha para fazer login com segurança em uma conta ou realizar uma transação.

## DESCRIÇÃO DO Número de Autenticação da Transação (TAN)

Os números de autenticação de transação (TANs) fornecem segurança extra. Comerciantes e empresas de cartões de pagamento têm um incentivo para melhorar a segurança das transações, pois uma melhor segurança reduz a possibilidade de ocorrência de fraudes. Organizações como o Payment Card Industry Security Standards Council criam padrões que os [adquirentes de cartões de pagamento](/acquirer) e as empresas de processamento devem seguir para criptografar as informações do cartão no ponto de interação (POI) e, posteriormente, descriptografar e processar a transação.

Os números de autorização de transação são uma maneira de as instituições financeiras reduzirem a possibilidade de fraude. Eles são números de uso único e fornecem uma autenticação de dois fatores de uma transação. O primeiro nível de autenticação pode incluir um [número de identificação pessoal](/personal-identification-number) (PIN) ou senha para acessar uma conta, enquanto o segundo nível de autenticação pode ser o TAN.

As instituições financeiras geralmente fornecem uma lista de senhas ou senhas que podem ser usadas para autenticar uma transação, sendo que cada TAN é válido apenas para um único uso. A instituição financeira que fornece a lista de TANs mantém um banco de dados no qual associa cada TAN a um determinado usuário.

TANs são mais comumente usados em verificações de transações online. Quando um indivíduo ou empresa inicia a transação, pode ser fornecido o TAN em um e-mail, em uma mensagem de texto SMS ou por outro meio. O método de entrega normalmente foi autenticado previamente por meio de uma interação anterior, como um banco enviando uma mensagem de texto confirmando que um determinado número de telefone está vinculado a uma conta. Quando uma transação estiver sendo realizada, o usuário receberá uma mensagem com o código TAN e será solicitado a inserir esse código em um campo em um formulário baseado na web. Se o código corresponder corretamente, a transação será processada.

## Números de autenticação de transação e autenticação de dois fatores

À medida que os americanos moveram mais suas atividades on-line, o uso de números de autenticação de transações se espalhou das instituições financeiras para muitas outras áreas da vida. É aconselhável, por exemplo, que os usuários de e-mail entrem em suas contas de e-mail usando [autenticação de dois fatores](/twofactor-authentication-2fa),. em que é preciso inserir a senha junto com um número de autenticação de transação. Eles geralmente são mantidos pelo usuário em uma lista de códigos de uso único ou o número é recuperado por mensagem de texto, e-mail ou telefonema.
