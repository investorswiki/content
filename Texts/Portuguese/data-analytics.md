# Análise de dados
## O que é análise de dados?

A análise de dados é a ciência de analisar dados brutos para tirar conclusões sobre essas informações. Muitas das técnicas e processos de análise de dados foram automatizados em processos mecânicos e [algoritmos](/algorithm) que funcionam sobre dados brutos para consumo humano.

## Entendendo a análise de dados

A análise de dados é um termo amplo que abrange muitos tipos diversos de análise de dados. Qualquer tipo de informação pode ser submetida a técnicas de análise de dados para obter insights que podem ser usados para melhorar as coisas. As técnicas de análise de dados podem revelar tendências e métricas que, de outra forma, seriam perdidas na massa de informações. Essas informações podem ser usadas para otimizar processos para aumentar a eficiência geral de um negócio ou sistema.

Por exemplo, as empresas de [manufatura](/manufacturing) geralmente registram o tempo de execução, o tempo de inatividade e a fila de trabalho de várias máquinas e analisam os dados para planejar melhor as cargas de trabalho para que as máquinas operem mais perto da capacidade máxima.

A análise de dados pode fazer muito mais do que apontar [gargalos](/bottleneck) na produção. As empresas de jogos usam a análise de dados para definir cronogramas de recompensas para os jogadores que mantêm a maioria dos jogadores ativos no jogo. As empresas de conteúdo usam muitas das mesmas análises de dados para mantê-lo clicando, assistindo ou reorganizando o conteúdo para obter outra visualização ou outro clique.

A análise de dados é importante porque ajuda as empresas a otimizar seus desempenhos. Implementá-lo no modelo de negócios significa que as empresas podem ajudar a reduzir custos identificando formas mais eficientes de fazer negócios e armazenando grandes quantidades de dados. Uma empresa também pode usar a análise de dados para tomar melhores decisões de negócios e ajudar a analisar as tendências e a satisfação do cliente, o que pode levar a novos e melhores produtos e serviços.

### Etapas de análise de dados

O processo envolvido na análise de dados envolve várias etapas diferentes:

1. O primeiro passo é determinar os requisitos de dados ou como os dados são agrupados. Os dados podem ser separados por idade, demografia, renda ou sexo. Os valores dos dados podem ser numéricos ou divididos por categoria.

1. A segunda etapa na análise de dados é o processo de coletá-los. Isso pode ser feito por meio de uma variedade de fontes, como computadores, fontes on-line, câmeras, fontes ambientais ou por meio de pessoal.

1. Uma vez coletados os dados, eles devem ser organizados para que possam ser analisados. Isso pode ocorrer em uma planilha ou outra forma de software que possa coletar dados estatísticos.

1. Os dados são então limpos antes da análise. Isso significa que ele é limpo e verificado para garantir que não haja duplicação ou erro e que não esteja incompleto. Esta etapa ajuda a corrigir quaisquer erros antes de irem para um analista de dados para serem analisados.

## Tipos de análise de dados

A análise de dados é dividida em quatro tipos básicos.

1. **Análise descritiva:** descreve o que aconteceu em um determinado período de tempo. O número de visualizações aumentou? As vendas estão mais fortes este mês do que no passado?

1. **Análise de diagnóstico:** isso se concentra mais no motivo pelo qual algo aconteceu. Isso envolve entradas de dados mais diversas e um pouco de hipótese. O clima afetou as vendas de cerveja? Essa última campanha de marketing impactou as vendas?

1. **Análise preditiva:** isso se move para o que provavelmente acontecerá no curto prazo. O que aconteceu com as vendas da última vez que tivemos um verão quente? Quantos modelos climáticos prevêem um verão quente este ano?

1. **Análise prescritiva:** isso sugere um curso de ação. Se a probabilidade de um verão quente for medida como uma média desses cinco modelos climáticos está acima de 58%, devemos adicionar um turno da noite à cervejaria e alugar um tanque adicional para aumentar a produção.

A análise de dados sustenta muitos sistemas de controle de qualidade no mundo financeiro, incluindo o sempre popular programa [Six Sigma](/six-sigma). Se você não está medindo corretamente algo – seja seu peso ou o número de defeitos por milhão em uma linha de produção – é quase impossível otimizá-lo.

Alguns dos [setores](/sector) que adotaram o uso de análise de dados incluem o setor de viagens e hospitalidade, onde os retornos podem ser rápidos. Esse setor pode coletar dados de clientes e descobrir onde estão os problemas, se houver, e como corrigi-los.

Healthcare combina o uso de grandes volumes de dados estruturados e não estruturados e usa análise de dados para tomar decisões rápidas. Da mesma forma, o setor de varejo usa grandes quantidades de dados para atender às demandas em constante mudança dos compradores. As informações que os varejistas coletam e analisam podem ajudá-los a identificar tendências, recomendar produtos e aumentar os lucros.

## Técnicas de análise de dados

Existem vários métodos e técnicas analíticas diferentes que os analistas de dados podem usar para processar dados e extrair informações. Alguns dos métodos mais populares estão listados abaixo.

- [A análise de regressão](/regression) envolve analisar a relação entre variáveis dependentes para determinar como uma mudança em uma pode afetar a mudança em outra.

- [A análise fatorial](/random-factor-analysis) envolve pegar um grande conjunto de dados e reduzi-lo a um conjunto de dados menor. O objetivo dessa manobra é tentar descobrir tendências ocultas que, de outra forma, seriam mais difíceis de ver.

- **Análise de coorte** é o processo de dividir um conjunto de dados em grupos de dados semelhantes, geralmente divididos em um grupo demográfico do cliente. Isso permite que analistas de dados e outros usuários de análise de dados mergulhem ainda mais nos números relacionados a um subconjunto específico de dados.

- [Simulações de Monte Carlo](/montecarlosimulation) modelam a probabilidade de diferentes resultados acontecerem. Frequentemente usadas para mitigação de riscos e prevenção de perdas, essas simulações incorporam vários valores e variáveis e geralmente têm maiores recursos de previsão do que outras abordagens de análise de dados.

- **Análise de série temporal** rastreia os dados ao longo do tempo e solidifica a relação entre o valor de um ponto de dados e a ocorrência do ponto de dados. Essa técnica de análise de dados geralmente é usada para identificar tendências cíclicas ou projetar previsões financeiras.

## Ferramentas de análise de dados

Além de uma ampla gama de abordagens matemáticas e estatísticas para processar números, a análise de dados evoluiu rapidamente em recursos tecnológicos. Hoje, os analistas de dados têm uma ampla gama de ferramentas de software para ajudar a adquirir dados, armazenar informações, processar dados e relatar descobertas.

A análise de dados sempre teve laços soltos com planilhas e Microsoft Excel. Agora, os analistas de dados também costumam interagir com linguagens de programação brutas para transformar e manipular bancos de dados. Linguagens [de código aberto](/open-source),. como Python, são frequentemente utilizadas. Ferramentas mais específicas para análise de dados como R podem ser usadas para análise estatística ou modelagem gráfica.

Os analistas de dados também têm ajuda ao relatar ou comunicar descobertas. Tanto o Tableau quanto o Power BI são ferramentas de visualização e análise de dados para compilar informações, realizar análises de dados e distribuir resultados por meio de painéis e relatórios.

Outras ferramentas também estão surgindo para auxiliar os analistas de dados. O SAS é uma plataforma de análise que pode auxiliar na [mineração de dados](/datamining),. enquanto o Apache Spark é uma plataforma de código aberto útil para processar grandes conjuntos de dados. Os analistas de dados agora têm uma ampla gama de recursos tecnológicos para aprimorar ainda mais o valor que entregam à sua empresa.

## A linha de fundo

Em um mundo cada vez mais dependente de informações e coleta de estatísticas, a análise de dados ajuda indivíduos e organizações a garantir seus dados. Usando uma variedade de ferramentas e técnicas, um conjunto de números brutos pode ser transformado em insights informativos e educacionais que impulsionam a tomada de decisões e o gerenciamento cuidadoso.

## Destaques

- A análise de dados é a ciência de analisar dados brutos para tirar conclusões sobre essas informações.

- Várias abordagens para análise de dados incluem observar o que aconteceu (análise descritiva), por que algo aconteceu (análise de diagnóstico), o que vai acontecer (análise preditiva) ou o que deve ser feito em seguida (análise prescritiva).

- A análise de dados ajuda uma empresa a otimizar seu desempenho, ter um desempenho mais eficiente, maximizar o lucro ou tomar decisões mais estrategicamente orientadas.

- As técnicas e processos de análise de dados foram automatizados em processos mecânicos e algoritmos que funcionam sobre dados brutos para consumo humano.

- A análise de dados conta com uma variedade de ferramentas de software que vão desde planilhas, visualização de dados e ferramentas de relatórios, programas de mineração de dados ou linguagens de código aberto para a maior manipulação de dados.
