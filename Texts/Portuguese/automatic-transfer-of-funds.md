# Transferência automática de fundos
## O que é uma Transferência Automática de Fundos?

Uma transferência automática de fundos é um acordo bancário permanente pelo qual as transferências entre duas (ou mais) contas de um cliente são feitas de forma regular e periódica sob condições especificadas. As transferências automáticas são executadas sem mais instruções ou ações do cliente; uma maneira comum de execução de transferências automáticas é por meio de instruções de "varredura", em que todos os fundos em excesso em uma conta são varridos para outra conta. A transferência automática de fundos é uma oferta central dos [bancos](/onlinebanking) [comerciais](/commercialbank) e online .

## Como funciona uma transferência automática de fundos

As transferências automáticas são frequentemente utilizadas para o movimento regular de fundos de uma [conta corrente](/checkingaccount) para uma [conta poupança](/savingsaccount). As transferências automáticas podem ser usadas para transferir dinheiro entre as contas de dois cônjuges ou entre um pai e um filho. Configurar transferências automáticas para pagar contas é uma ferramenta de orçamento útil porque elas podem ser usadas para pagamentos periódicos iguais, como hipotecas ou outros pagamentos de empréstimos.

Outro uso comum dessas transferências é para [proteção de cheque especial](/overdraft-protection),. em que os fundos são movidos de uma conta com juros mais altos para cobrir pagamentos devidos em outra conta.

As empresas com várias subsidiárias às vezes fazem uso de uma conta de saldo zero (ZBA). Uma ZBA geralmente organiza a transferência automática de fundos porque é uma conta corrente na qual é mantido um saldo zero. Quando os fundos são necessários na ZBA, a quantidade exata de dinheiro necessária é movida para a conta por meio de uma transferência automática de fundos de uma conta central ou principal.

As empresas também podem usar um ZBA se seus funcionários tiverem cartões de crédito ou débito da empresa, pois permite maior controle em relação à distribuição de fundos da empresa (além de limitar saldos excedentes). Uma conta ZBA pode ajudar a garantir que os gerentes aprovem previamente todas as atividades nos cartões de débito/crédito da empresa. Após a aprovação, é iniciada uma transferência automática de fundos de uma conta máster em valores suficientemente grandes para cobrir os encargos apresentados.

## Considerações Especiais

Entre os bancos online, a cibersegurança tornou-se especialmente crítica para evitar tornar as informações eletrônicas vulneráveis a danos ou roubos. Quando as informações são transferidas entre redes, como durante uma transferência automática de fundos, podem ocorrer ataques cibernéticos.

Os ataques cibernéticos podem ocorrer de várias formas, incluindo ataques de backdoor, nos quais um ladrão explora um método alternativo de acesso a um sistema; ataques de negação de serviço, que impedem um usuário legítimo de acessar um sistema; e ataques de acesso direto, inclui bugs e vírus, que obtêm acesso a um sistema e copiam suas informações e/ou modificam o sistema.

##Destaques

- As transferências automáticas podem ser usadas para mover dinheiro de uma conta bancária para outra, como de uma conta corrente para uma conta poupança.

- Uma transferência automática de fundos é um acordo bancário permanente pelo qual as transferências da conta de um cliente são feitas de forma regular e periódica.

- Quase todos os bancos físicos e bancos online oferecem serviços de transferência automática para seus clientes.
