# Sistema de Informações Regulatórias de Seguros (IRIS)
## O que é o Sistema de Informações Regulatórias de Seguros (IRIS)?

O Insurance Regulatory Information System (IRIS) é um conjunto de bases de dados e ferramentas utilizadas para analisar as [demonstrações financeiras](/financial-statements) das seguradoras. Gerenciado pela [National Association of Insurance Commissioners](/nainsurancec) (NAIC), o Insurance Regulatory Information System (IRIS) está disponível desde 1972 e é empregado principalmente pelos reguladores para determinar a solvência das seguradoras.

## Como funciona o Sistema de Informações Regulatórias de Seguros (IRIS)

O Sistema de Informações Regulatórias de Seguros (IRIS) explora as informações financeiras arquivadas pelas [seguradoras](/insurance) para calcular índices que podem ser usados para estabelecer quais seguradoras correm o risco de não cumprir suas dívidas de longo prazo e outras obrigações financeiras.

O sistema produz automaticamente esses índices a partir das demonstrações financeiras que as seguradoras são obrigadas a apresentar aos seus reguladores. Uma vez extraídas as informações relevantes, são gerados relatórios que listam cada seguradora analisada, os índices financeiros derivados para elas e as faixas em que cada índice financeiro deve se enquadrar.

As empresas que estão fora da faixa usual são levadas ao conhecimento dos departamentos de seguros estaduais responsáveis por regulá-las. A partir daí, esses reguladores podem decidir examinar mais detalhadamente os culpados e, se necessário, colocá-los sob estreita supervisão.

De acordo com o NAIC, cujos membros consistem no chefe do departamento de seguros de cada estado, todas as seguradoras são obrigadas a apresentar demonstrações financeiras no estado em que estão licenciadas para operar. O Insurance Regulatory Information System (IRIS), que foi desenvolvido pelos reguladores estaduais de seguros em conjunto com o NAIC, é projetado para fazer o resto.

> ### Importante

> Muitos departamentos de seguros estaduais disponibilizam ao público dados financeiros sobre as seguradoras.

>

## Benefícios do Sistema de Informações Regulatórias de Seguros (IRIS)

O Insurance Regulatory Information System (IRIS) melhora a eficiência dos reguladores de seguros estaduais com recursos limitados, funcionando como uma ferramenta útil que pode ser usada junto com os bancos de dados computadorizados de cada estado projetados para capturar, processar e analisar as demonstrações financeiras das companhias de seguros.

A NAIC enfatiza que nenhum estado é capaz de revisar minuciosamente a [condição financeira](/financial-health) das seguradoras licenciadas imediatamente após o recebimento das demonstrações financeiras. Em vez disso, cabe ao Sistema de Informações Regulamentares de Seguros (IRIS) cumprir esse papel. A NAIC alega que o sistema “ajuda fornecendo ferramentas e bancos de dados de solvência que destacam as seguradoras que merecem a maior prioridade na alocação dos recursos dos reguladores, direcionando esses recursos para o melhor uso possível”.

Até certo ponto, o Insurance Regulatory Information System (IRIS) também pode ser benéfico para as seguradoras. Em vez de perder um tempo precioso esperando que o regulador aja, a empresa pode pegar as informações fornecidas pelo sistema e usá-las para examinar e corrigir imediatamente quaisquer problemas financeiros destacados antes que eles saiam do controle.

## Considerações Especiais

Índices fora da faixa padrão não indicam necessariamente que uma seguradora está com problemas financeiros.

Alguns índices são baseados em fatores fora do controle direto da empresa, como o desempenho da economia ou do [mercado de ações](/stockmarket). Como investem os [prêmios](/insurance-premium) que obtêm das apólices de subscrição, é possível que uma seguradora tenha vários índices fora da norma.

Em última análise, isso significa que os reguladores são obrigados a fazer algumas escavações extras para determinar se os relatórios gerados pelo Sistema de Informações Regulamentares de Seguros (IRIS) são motivo de preocupação. O próprio sistema serve principalmente como um guia, destacando possíveis problemas que podem ou não precisar ser resolvidos, ao mesmo tempo em que fornece aos reguladores uma solução rápida para ver como as empresas se comparam.

## Destaques

- Gerenciado pela NAIC, é empregado principalmente pelos reguladores para determinar a solvência das seguradoras.

- O sistema produz automaticamente índices a partir das demonstrações financeiras que as seguradoras são obrigadas a apresentar aos seus reguladores.

- O IRIS é um conjunto de bases de dados e ferramentas utilizadas para analisar as demonstrações financeiras das seguradoras.

- Esses índices são então usados para estabelecer quais seguradoras estão com problemas de saúde financeira e potencialmente merecem um escrutínio mais minucioso.
