# Cifra
Em [criptografia](/cryptography),. uma cifra é uma série de instruções definidas que se pode seguir para criptografar ou descriptografar uma mensagem de texto. O processo de [criptografia](/encryption) consiste em converter informações claras em uma versão ilegível ou inacessível. O texto original que pode ser claramente entendido é conhecido como texto simples, enquanto sua forma criptografada é chamada de texto cifrado. Ambos os textos contêm a mesma informação, a única diferença é que o texto cifrado é escrito em um formato que só pode ser lido ou acessado por aqueles que possuem o mecanismo [de](/decryption) [descriptografia correto.](/decryption)

A maioria dos algoritmos de cifra envolve o uso de uma determinada informação secreta, que é comumente chamada de chave criptográfica. O esquema de criptografia varia de acordo com o modelo de chave, e os algoritmos de cifras podem ser definidos como simétricos ou assimétricos. As cifras simétricas usam apenas uma chave para criptografia e descriptografia, enquanto as cifras assimétricas usam chaves diferentes para cada operação.

Embora a maioria das técnicas modernas de criptografia sejam feitas por computadores, as cifras já eram usadas para codificar mensagens desde antes dos gregos antigos, por volta de 400 aC. O conhecido político romano Júlio César fez grande uso de cifras de substituição, substituindo cada letra em uma mensagem pela letra que estava localizada três lugares mais abaixo no alfabeto.

Por exemplo, se usarmos a mesma técnica para criptografar a palavra BINANCE, o texto cifrado resultante seria ELQDQFH. Esta mensagem parece sem sentido para um terceiro, mas pode ser facilmente descriptografada pelo destinatário pretendido, pois ele já conhece o número de turno empregado pelo remetente para criptografá-la.
