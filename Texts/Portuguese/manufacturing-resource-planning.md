# Planejamento de Recursos de Manufatura
## O que é Planejamento de Recursos de Manufatura?

O Manufacturing Resource Planning (MRP II) é um sistema de informação integrado utilizado pelas empresas. O Manufacturing Resource Planning (MRP II) evoluiu dos primeiros sistemas de [Materials Requirement Planning (MRP)](/mrp),. incluindo a integração de dados adicionais, como funcionários e necessidades financeiras.

O sistema foi projetado para centralizar, integrar e processar informações para uma tomada de decisão eficaz em programação, engenharia de projeto, [gerenciamento de estoque](/inventory-management) e [controle de custos](/cost-control) na fabricação.

Tanto o MRP quanto o MRP II são vistos como predecessores do [Enterprise resource planning (ERP)](/equityriskpremium),. que é um processo pelo qual uma empresa, geralmente um fabricante, gerencia e integra as partes importantes de seus negócios.

Um sistema de informação gerencial ERP integra áreas como planejamento, compras, estoque, vendas, marketing, finanças e [recursos humanos](/humanresources). O ERP é usado com mais frequência no contexto de software, com muitos aplicativos grandes sendo desenvolvidos para ajudar as empresas a implementar o ERP.

## Entendendo o Planejamento de Recursos de Manufatura (MRP II)

O MRP II é um sistema baseado em computador que pode criar cronogramas de produção detalhados usando dados em tempo real para coordenar a chegada de materiais de componentes com disponibilidade de máquinas e mão de obra. O MRP II é amplamente usado por si só, mas também é usado como um módulo de sistemas de [planejamento de recursos empresariais (ERP) mais extensivos](/erp).

O MRP II é uma extensão do sistema de planejamento de necessidades de materiais original (MRP I). O planejamento de requisitos de materiais (MRP) é um dos primeiros sistemas de informação integrados baseados em software projetados para melhorar a [produtividade](/productivity) das empresas.

Um sistema de informações de planejamento de necessidades de materiais é um sistema baseado em previsão de vendas usado para programar [entregas](/delivery) e quantidades de matéria-prima, dadas suposições de máquinas e unidades de mão de obra necessárias para atender a uma previsão de vendas.

Na década de 1980, os fabricantes perceberam que precisavam de um software que também pudesse se conectar aos seus sistemas de contabilidade e prever os requisitos de estoque. O MRP II foi fornecido como uma solução, que incluiu essa funcionalidade além de todos os recursos oferecidos pelo MRP I.

### MRP I vs. MRP II

Para todos os efeitos, o MRP II substituiu efetivamente o software MRP I. A maioria dos sistemas MRP II oferece todas as funcionalidades de um sistema MRP. Mas, além de oferecer programação mestre de produção, lista de materiais (BOM) e rastreamento de estoque, o MRP II oferece funcionalidade em logística, marketing e finanças em geral.

Por exemplo, o MRP II é capaz de levar em conta variáveis que o MRP não é — incluindo capacidade de máquina e pessoal — fornecendo uma representação mais realista e holística das capacidades operacionais de uma empresa. Muitas soluções MRP II também oferecem recursos de simulação que permitem que os operadores insiram variáveis e vejam o efeito downstream. Devido à sua capacidade de fornecer feedback sobre uma determinada operação, o MRP II às vezes é chamado de sistema de malha fechada.

O MRP I incluiu as três principais funcionalidades a seguir:

- Programação mestre de produção

- lista de materiais

- acompanhamento de estoque

O MRP II inclui esses três, mais o seguinte:

- programação de capacidade da máquina

- previsão de demanda

- Garantia da Qualidade

- conta Geral

Os sistemas MRP II ainda são amplamente utilizados por empresas de manufatura hoje e podem ser encontrados como soluções independentes ou como parte de um sistema de planejamento de recursos empresariais (ERP). Os sistemas de software Enterprise Resources Planning (ERP) são considerados os sucessores do software MRP II.

As suítes de ERP incluem aplicativos bem fora do escopo da fabricação. Isso pode incluir tudo, desde recursos humanos e gerenciamento de relacionamento com clientes até gerenciamento de ativos corporativos.

## Destaques

- Tanto o MRP quanto o MRP II são vistos como predecessores do Enterprise resource planning (ERP).

- Manufacturing Resource Planning (MRP II) é um sistema de informação integrado utilizado pelas empresas.

- MRP II é uma extensão do planejamento de necessidades de materiais (MRP).
