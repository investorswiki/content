# Mediana
## Qual é a mediana?

A mediana é o número do meio em uma lista ordenada, crescente ou decrescente de números e pode ser mais descritiva desse conjunto de dados do que a média. É o ponto acima e abaixo do qual metade (50%) dos dados observados cai e, portanto, representa o ponto médio dos dados.

A mediana é frequentemente comparada com outras estatísticas descritivas, como a [média](/mean) (média), [moda](/mode) e [desvio padrão](/standarddeviation).

## Entendendo a mediana

A mediana é o número do meio em uma lista ordenada de números. Para determinar o valor mediano em uma sequência de números, os números devem primeiro ser ordenados, ou organizados, em ordem de valor do menor para o maior ou do maior para o menor. A mediana pode ser usada para determinar uma média aproximada, ou [média](/mean),. mas não deve ser confundida com a média real.

- Se houver uma quantidade ímpar de números, o valor da mediana é o número que está no meio, com a mesma quantidade de números abaixo e acima.

- Se houver uma quantidade par de números na lista, o par do meio deve ser determinado, somado e dividido por dois para encontrar o valor da mediana.

> A mediana às vezes é usada em oposição à média quando há valores discrepantes na sequência que podem distorcer a média dos valores. A mediana de uma sequência pode ser menos afetada por outliers do que a média.

>

## Exemplo de mediana

Para encontrar o valor da mediana em uma lista com uma quantidade **ímpar** de números, deve-se encontrar o número que está no meio com uma quantidade igual de números em ambos os lados da mediana. Para encontrar a mediana, primeiro organize os números em ordem, geralmente do menor para o maior.

Por exemplo, em um conjunto de dados de {3, 13, 2, 34, 11, 26, 47}, a ordem classificada se torna {2, 3, 11, 13, 26, 34, 47}. A mediana é o número no meio {2, 3, 11, **13**, 26, 34, 47}, que neste caso é 13, pois há três números de cada lado.

Para encontrar o valor da mediana em uma lista com uma quantidade **par** de números, deve-se determinar o par do meio, somá-los e dividir por dois. Novamente, organize os números em ordem do menor para o maior.

Por exemplo, em um conjunto de dados de {3, 13, 2, 34, 11, 17, 27, 47}, a ordem classificada se torna {2, 3, 11, 13, 17, 27, 34, 47}. A mediana é a média dos dois números no meio {2, 3, 11, **13**, **17**, 26 34, 47}, que neste caso é quinze {(13 + 17) ÷ 2 = 15}.

> A mediana está intimamente associada aos [quartis](/quartile),. ou seja, a divisão dos dados observados em quatro partes iguais. A mediana seria o ponto central, com os dois primeiros quartis caindo abaixo dele e os dois segundos acima dele. Outras formas de agrupar dados incluem [quintis](/quintile) (em cinco seções) e [decis](/decile) (em 10 seções).

##Destaques

- A mediana é o número do meio em uma lista ordenada de números e pode ser mais descritiva desse conjunto de dados do que a média.

- A mediana às vezes é usada em oposição à média quando há valores discrepantes na sequência que podem distorcer a média dos valores.

- Se houver uma quantidade par de números na lista, o par do meio deve ser determinado, somado e dividido por dois para encontrar o valor da mediana.

- Em uma distribuição normal, a mediana é igual à média e à moda.

- Se houver uma quantidade ímpar de números, o valor da mediana é o número que está no meio, com a mesma quantidade de números abaixo e acima.

##PERGUNTAS FREQUENTES

### Onde está a mediana em uma distribuição normal?

Na [distribuição normal](/normaldistribution) ("curva em sino"), a mediana, a média e a moda têm o mesmo valor e caem no ponto mais alto no centro da curva.

### Como você calcula a mediana?

A mediana é o valor médio em um conjunto de dados. Primeiro, organize e ordene os dados do menor para o maior. Para encontrar o valor do ponto médio, divida o número de observações por dois. Se houver um número ímpar de observações, arredonde esse número para cima e o valor nessa posição será a mediana. Se o número de observações for par, tire a média dos valores encontrados acima e abaixo dessa posição.

### Quando a média e a mediana são diferentes?

Em um conjunto de dados distorcido, a média e a mediana normalmente serão diferentes. A média é calculada somando todos os valores nos dados e dividindo pelo número de observações. Se houver discrepâncias consideráveis ou se os dados se agruparem em torno de determinados valores, a média (média) não será o ponto médio dos dados. Por exemplo, em um conjunto de dados {0, 0, 0, 1, 1, 2, 10, 10} a média seria 124/8 = 3. A mediana, no entanto, seria 1 (o valor do ponto médio). É por isso que muitos economistas preferem a mediana para relatar a renda ou riqueza de uma nação, pois é mais representativa da a distribuição real de renda.
