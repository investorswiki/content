# Código de moeda ISO
## O que é um código de moeda ISO?

Os códigos de moeda ISO são os códigos alfabéticos de três letras que representam as várias [moedas](/currency) usadas em todo o mundo. Quando combinados em pares, eles compõem os símbolos e [as taxas cruzadas](/crossrate) usadas na negociação de moedas.

Cada um dos códigos alfabéticos de três letras específicos do país também possui um código numérico de três dígitos correspondente. Esses códigos são identificados pela [Organização Internacional para Padronização](/international-organization-for-standardization-iso) (ISO), uma organização não governamental que fornece padrões para fabricação, comércio, tecnologia e comunicação. Para moedas, o documento regulador é chamado ISO 4217:2015.

## Entendendo os códigos de moeda ISO

Os códigos de moeda ISO são centrais para [os pares de moedas](/currencypair),. que são as estruturas de cotação e preço das moedas negociadas no [mercado cambial](/forex-market). O valor de uma moeda é uma taxa e é determinado pela sua comparação com outra moeda.

O primeiro código de três dígitos usado para designar uma moeda em uma cotação de par de moedas é chamado de [moeda base](/basecurrency) e a segunda moeda é chamada de [moeda de cotação](/quotecurrency). O par de moedas indica quanto da moeda de cotação é necessária para comprar uma unidade da moeda base.

Por exemplo, [EUR/USD](/eur-usd-euro-us-dollar-currency-pair) é a cotação do euro em relação ao dólar americano. EUR é o código de moeda ISO de três letras para o euro e USD é o código para o dólar americano. Um preço cotado para este par de 1,2500 significa que um euro é trocado por 1,2500 dólares americanos porque, neste caso, [EUR](/euro) é a moeda base e USD é a moeda de cotação (ou contra moeda). Isso significa que 1 euro pode ser trocado por 1,25 dólares americanos. Outra maneira de ver isso é que custará US $ 125 para comprar 100 euros.

A ISO não estava envolvida em transações monetárias até 1973, quando o órgão normativo decidiu que seria útil estar envolvido. Após cinco anos de colaboração e deliberação, os primeiros códigos de moeda padronizados foram publicados em 1978, com um padrão sobre como eles deveriam mudar.

## Considerações Especiais

De acordo com o site da ISO, "A ISO 4217:2015 especifica a estrutura de um código alfabético de três letras e um código numérico equivalente de três dígitos para a representação de moedas. Para as moedas com unidades menores, também mostra a relação decimal entre tais unidades e a própria moeda." Por exemplo, o código numérico de três dígitos para o dólar americano é 840 e o código numérico para o euro é 978. Mas você não veria moedas cotadas usando números como este (978/840).

Embora, como explica o documento ISO, "a ISO 4217:2015 seja destinada ao uso em qualquer aplicação de comércio, comércio e serviços bancários, onde moedas e, quando apropriado, fundos devem ser descritos. Ela foi projetada para ser igualmente adequada para usuários manuais e para aqueles que empregam sistemas automatizados." Se isso fosse útil, os algoritmos de negociação ou processamento de pedidos poderiam usar códigos numéricos para um processamento mais eficaz.

## Principais códigos de moeda

O site da ISO oferece uma lista completa de códigos de moeda nos formatos XML e XMS. Todos os principais pares de moedas têm [mercados muito líquidos](/liquidmarket) que negociam 24 horas por dia todos os dias úteis e têm [spreads muito estreitos](/spread).

Os principais pares de moedas são os seguintes:

- EUR/USD - euro/dólar americano

- [GBP/USD](/gbp-usd-british-pound-us-dollar-currency-pair) - [Libra esterlina (libra esterlina)](/gbp) / Dólar americano

- USD/JPY - Dólar americano / [Iene japonês](/jpy-japanese-yen)

- USD/CHF - Dólar americano / [Franco suíço](/chf)

> Valor negociado em relação ao dólar americano é o principal critério para classificação como principais pares de moedas.

>

Outras moedas importantes incluem:

- AUD - [Dólar australiano](/aud-australian-dollar)

- CAD - [dólar](/cad-canadian-dollar) [canadense](/cad-canadian-dollar)

- CNY - [China yuan renminbi](/cny-china-yuan-renminbi)

- NZD - [Dólar neozelandês](/nzd-new-zealand-dollar)

- INR - [rupia indiana](/inr)

- BZR - [real brasileiro](/brl-brazilian-real)

- SEK - [coroa sueca](/sek-swedish-krona)

- ZAR - [rand sul-africano](/zar-south-african-rand)

- HKD - [Dólar de Hong Kong](/hkd-hong-kong-dollars)

## Destaques

- Códigos de moeda ISO designam as moedas base e de cotação em cotações de preços forex.

- Os comitês de padrões ISO estabeleceram códigos de moeda padronizados em 1978.

- Os códigos de moeda ISO são os códigos alfabéticos de três letras que representam as várias moedas usadas em todo o mundo.

- A ISO também designou equivalentes numéricos menos conhecidos para os códigos de moeda de três letras.
