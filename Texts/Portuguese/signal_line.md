# Linha de sinal
## O que é uma linha de sinal

As linhas de sinal são usadas em indicadores técnicos, especialmente [osciladores](/oscillator),. para gerar sinais de compra e venda ou sugerir uma mudança de tendência. Eles são chamados de linhas de sinal porque quando outro indicador ou linha os cruza, é um sinal para negociar ou que algo potencialmente importante está acontecendo com o preço de um ativo. Pode ser que o preço estava em tendência, [recuou](/pullback) e agora está começando a tendência ou novamente, ou pode sinalizar que uma nova tendência de alta ou baixa está começando.

As linhas de sinal geralmente são médias móveis de um indicador técnico, como a [convergência-divergência da média móvel](/macd) (MACD) ou [oscilador estocástico](/stochasticoscillator). A linha de sinal é aplicada ao indicador para gerar mais sinais de negociação do que estaria disponível sem a linha de sinal.

Uma linha de sinal também é comumente conhecida como "linha de disparo".

## A fórmula para uma linha de sinal é

A linha de sinal não é um indicador, portanto, será calculada de forma diferente para cada indicador em que for usada. Isso ocorre porque o criador de um indicador geralmente cria uma fórmula para a linha de sinal ao fazer o indicador. A linha de sinal faz parte do indicador.

As linhas de sinais geralmente são cálculos simples. Por exemplo, a linha de sinal para o MACD é uma [média móvel exponencial](/ema) de nove períodos (EMA) do valor MACD.

A linha de sinal para o oscilador estocástico é uma [média](/sma) [móvel simples de três períodos](/sma) (SMA) do estocástico (chamada %K neste caso).

## Como calcular uma linha de sinal

1. Encontre a fórmula para a linha de sinal do indicador que você está usando. Normalmente, a fórmula é uma EMA ou SMA do indicador.

1. Calcule o indicador ou adicione-o a um gráfico no software de gráficos.

1. Calcule a EMA ou SMA do indicador. Alternativamente, aplique a média móvel aplicável ao indicador no gráfico para criar a linha de sinal.

## O que uma linha de sinal lhe diz?

As linhas de sinal podem ser aplicadas a muitos indicadores técnicos diferentes, mas a convergência-divergência da média móvel (MACD) e os osciladores estocásticos são os dois mais populares. A maioria das linhas de sinal é criada usando uma média móvel dos valores do indicador. Essas médias móveis geralmente são SMAs ou EMAs.

As linhas de sinal também podem ser usadas para indicar uma mudança no [momento](/momentum) de uma tendência. Por exemplo, se um indicador cruzar acima da linha de sinal, isso indica que o preço está começando a subir. Se um indicador cruzar abaixo da linha de sinal, isso indica que o preço está começando a cair.

Esses sinais são normalmente usados em conjunto com outras informações. Por exemplo, se o preço estiver visivelmente em uma [tendência de alta](/uptrend) de longo prazo , um trader pode considerar apenas realizar negociações longas em cruzamentos de alta. Eles podem vender quando há um cruzamento de baixa, mas não entrariam em uma posição curta porque isso iria contra a tendência de alta de longo prazo.

Outras informações que sinalizam cruzamentos de linhas são frequentemente usadas em conjunto com outras formas de análise técnica, como indicadores técnicos, padrões de gráficos ou padrões de [velas](/candlestick) que fornecem confirmação. Como outro exemplo, os comerciantes podem usar [pontos de pivô](/pivotpoint) para identificar possíveis pontos de virada e, em seguida, procurar cruzamentos MACD para confirmação de uma reversão.

## A diferença entre uma linha de sinal e uma média móvel

Uma [média móvel](/movingaverage) pode ser a média de qualquer coisa, mas na análise técnica a maioria das pessoas considera uma média móvel baseada no preço ou, às vezes, no volume, como a média móvel de 200 dias do preço. Na maioria dos casos, uma linha de sinal é uma média móvel, mas é chamada de linha de sinal em vez de uma média móvel para ajudar a evitar confusão com as médias móveis baseadas em preço. As linhas de sinal são uma média móvel de um cálculo de indicador e, portanto, são usadas para gerar sinais de negociação apenas para esse indicador.

## Limitações das Linhas de Sinal

As linhas de sinal geralmente são apenas uma média móvel de um indicador. Desta forma, a linha de sinal [fica atrás](/laggingindicator) dos movimentos do indicador. À medida que o indicador reage às mudanças de preço, ele se move mais rápido que a linha de sinal e isso gera um cruzamento.

Embora as linhas de sinal possam produzir bons sinais às vezes, o que resulta em grandes mudanças de tendência e movimentos de preços, muitos cruzamentos serão [sinais falsos](/false-signal). Um sinal falso é quando o indicador cruza a linha de sinal, mas o preço não se move na direção esperada. O preço também pode cruzar a linha de sinal, resultando em vários sinais que perdem o dinheiro do trader se os sinais de negociação forem negociados.

Por essas razões, os cruzamentos de linha de sinal raramente são usados isoladamente. Outras formas de [análise](/fundamentalanalysis) técnica ou fundamental são usadas para confirmar sinais de negociação ou descartar certas negociações de linha de sinal.

## Destaques

- Geralmente é uma média móvel de um indicador que é aplicado ao indicador para que a linha de sinal e o indicador possam cruzar para gerar sinais de negociação.

- As linhas de sinal podem ser usadas de diferentes maneiras para diferentes indicadores, mas normalmente quando o indicador cruza acima da linha de sinal é interpretado como [alta](/bull) para o preço e quando o indicador cruza abaixo da linha de sinal é [baixa](/bear) para o preço.

- Uma linha de sinal não é um indicador técnico por si só.
