# Associado em Seguros Pessoais (API)
## Qual é a designação de Associate in Personal Insurance (API)?

O Associate in Personal Insurance (API) é uma certificação profissional no [setor de seguros](/insurance). Como o próprio nome sugere, a API é focada no [seguro de linhas pessoais](/personal-insurance-lines),. que é o tipo de seguro que protege contra danos pessoais, morte ou perda de propriedade. É uma das muitas designações conferidas pelos The Institutes, uma organização dedicada a fornecer credenciamento e educação continuada para profissionais de seguros.

## Como funciona o design da API

A designação API é popular entre os profissionais que desejam iniciar ou avançar na carreira no ramo de seguros pessoais. Ao obter a API, os profissionais podem demonstrar que têm conhecimento de produtos de seguros não comerciais, incluindo habilidades críticas, como [subscrição de seguros](/underwriting),. [definição de prêmios](/insurance-premium),. marketing e gerenciamento de portfólio.

Para profissionais que desejam obter certificações mais avançadas, como a [designação Chartered Property Casualty Underwriter (CPCU)](/chartered-property-casualty-underwriter-cpcu),. a API pode ser uma opção popular. Isso ocorre porque muitas das condições necessárias para obter a API também contam para os requisitos da CPCU. Especificamente, dois dos três exames associados à API podem ser reutilizados para o credenciamento da CPCU, que é uma das designações de maior prestígio no setor de acidentes de propriedade.

Para obter o API, os candidatos devem desenvolver amplo conhecimento sobre os tipos de apólices de seguro vendidas no mercado de seguros pessoais. Isso inclui uma ampla educação em assuntos como endossos de contratos, condições e critérios de subscrição e regulamentos do setor. A API é frequentemente recomendada para corretores de seguros, agentes, avaliadores de sinistros, subscritores e pessoal de atendimento ao cliente que atende clientes de seguros pessoais.

## Exemplo do mundo real da designação da API

Existem dois caminhos que os candidatos podem seguir ao concluir a API. O primeiro deles inclui cursos sobre práticas de [subscrição](/underwriting) e marketing na indústria de seguros pessoais, gestão de carteiras, seguros de propriedade e [responsabilidade](/liability_insurance) e seguros pessoais. O segundo caminho consiste em três cursos, abrangendo subscrição e marketing, [gestão de risco](/riskmanagement) e responsabilidade patrimonial e planejamento financeiro pessoal.

Normalmente, os candidatos levam entre 12 e 18 meses para concluir esses materiais do curso, que são acessados por autoatendimento. Para aqueles que preferem aulas presenciais, as aulas também podem estar disponíveis offline em regiões selecionadas. Depois de passarem nos exames, os titulares da API não precisam atender aos requisitos educacionais contínuos para manter sua designação.

##Destaques

- A API é uma designação profissional popular no setor de seguros de linhas pessoais.

- É administrado pelos Institutos e normalmente leva entre 12 e 18 meses para ser concluído.

- Muitos candidatos optam por adicionar à sua designação de API buscando uma credencial mais avançada, como a CPCU.
