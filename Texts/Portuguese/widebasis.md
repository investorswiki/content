# ampla base
## O que é uma base ampla?

Uma base ampla é uma condição encontrada em mercados [futuros](/futures) em que o preço à vista local de uma mercadoria está relativamente longe de seu preço futuro. É o oposto de uma [base estreita](/narrowbasis),. na qual o [preço à vista](/spotprice) e os preços futuros são muito próximos.

É normal que haja alguma diferença entre os preços à vista e os preços futuros, devido a fatores como custos de transporte e [manutenção](/holding-costs),. [taxas de juros](/interestrate) e clima incerto. Isso é conhecido como [base](/basis). Por mais ampla que seja, essa lacuna normalmente converge à medida que a [data de vencimento](/expirationdate) do contrato de futuros se aproxima.

## Entendendo a base ampla

Em última análise, uma base ampla indica um descompasso entre [oferta e demanda](/law-of-supply-demand). Se a oferta de curto prazo for relativamente baixa devido a fatores como clima excepcionalmente ruim, os preços à vista locais podem subir em relação aos preços futuros. Se, por outro lado, a oferta de curto prazo for relativamente alta, como no caso de uma safra excepcionalmente grande, os preços à vista locais podem cair em relação aos preços futuros.

Qualquer uma dessas situações daria origem a uma base ampla, onde "base" é simplesmente o preço à vista local menos o preço do contrato futuro. Essa lacuna deve desaparecer gradualmente à medida que os contratos futuros se aproximam de sua data de vencimento, pois, caso contrário, os investidores poderiam simplesmente explorar uma oportunidade de [arbitragem](/arbitrage) entre os preços à vista locais e os preços futuros.

Quando a base diminui de um número negativo como -$1, para um número menos negativo como $-0,50, essa mudança é conhecida como base de fortalecimento. Por outro lado, quando a base encolhe de um número positivo maior para um número positivo menor, isso é conhecido como base enfraquecida.

> ###Importante

> De um modo geral, uma base estreita é consistente com um mercado muito [líquido](/liquidity) e [eficiente](/marketefficiency),. enquanto uma base ampla está associada a um mercado relativamente ilíquido e ineficiente. No entanto, alguma variação entre os preços locais à vista e os preços futuros é normal e esperada.

>

## Exemplo do mundo real de uma base ampla

Suponha que você seja um trader de futuros de commodities interessado no mercado de petróleo. Você observa que o preço à vista local do [petróleo bruto](/crude-oil) é de US$ 40,71, enquanto o preço dos futuros de petróleo bruto com vencimento em dois meses é de US$ 40,93. Nesse cenário, você observa que a base entre esses dois preços é relativamente pequena, de apenas -US$ 0,22 (preço à vista de US$ 40,71 menos o preço futuro de US$ 40,93). Essa base estreita faz sentido, considerando que o contrato é muito negociado e faltam apenas dois meses para o vencimento do contrato.

Olhando para o futuro, no entanto, você começa a encontrar alguns contratos com uma base ampla. O mesmo contrato para entrega em nove meses, por exemplo, tem preço futuro de $ 42,41. Esse spread relativamente amplo de -$1,70 pode ser devido a muitos fatores diferentes. Por exemplo, os comerciantes podem esperar que o preço do petróleo suba como resultado da diminuição da oferta ou do aumento da atividade econômica. Não importa o motivo, a base quase certamente diminuirá à medida que a data do contrato se aproxima.

##Destaques

- Tal divergência entre o preço à vista e o preço futuro pode estar associada a iliquidez ou altos custos de carregamento.

- A base inevitavelmente diminui à medida que o contrato futuro se aproxima de sua data de vencimento; qualquer lacuna que produzisse oportunidades para lucros de arbitragem.

- Uma base ampla é uma condição de mercado na qual a diferença entre os preços à vista e os preços futuros é relativamente grande.
