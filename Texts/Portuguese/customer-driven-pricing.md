# Preços com base no cliente
## O que é preço orientado ao cliente?

Uma decisão importante que as empresas enfrentam é a fixação de preços. As três principais estratégias de precificação são precificação baseada em custo, precificação baseada na concorrência e determinação do [cliente](/customer) ou [precificação baseada no valor do cliente](/valuebasedpricing). A precificação orientada para o cliente é a prática de estabelecer preços de acordo com o valor percebido pelos clientes dos bens ou serviços de uma empresa.

A base do pressuposto para este modelo é que um cliente está disposto a pagar um determinado preço quando o valor entregue excede esse custo. A precificação orientada para o cliente pode funcionar quando um produto ou serviço é personalizável em vez de [comoditizado](/commoditize) (por exemplo, milho), pois haverá muitas opções se houver muitos concorrentes oferecendo produtos ou serviços semelhantes.

A estratégia de preços também pode ser aplicada em diferentes mercados geográficos, onde as forças de oferta e demanda são tendenciosas para o vendedor e o cliente pode estar disposto a pagar mais do que um cliente em outro lugar. A precificação orientada para o cliente é contrastada com a precificação baseada na concorrência - definição de preços com base nos preços e estratégias dos concorrentes - e precificação baseada no custo, na qual uma empresa se concentra em atingir uma meta de [margem](/profitmargin) com pouca atenção à análise do valor percebido pelos clientes.

## Como funciona o preço orientado ao cliente

Para otimizar os preços, as empresas precisam considerar a melhor forma de [segmentar o mercado](/market-segment) para que os preços reflitam as diferenças de valor discernidas pelos diferentes tipos de consumidores. Para fazer isso, as empresas devem realizar uma compreensão abrangente de como um cliente valoriza um produto ou serviço.

Se for um negócio, esse produto ou serviço é parte integrante de suas operações, algo absolutamente necessário para manter o negócio funcionando sem problemas? Proporciona benefícios adicionais ou diferenciados a um consumidor comum? Nesse caso, a empresa pode definir preços com prêmios para aumentar as vendas agregadas.

Se a oferta da empresa estiver disponível para clientes de vários concorrentes em um determinado mercado e não tiver diferenciação significativa, é improvável que a precificação orientada para o cliente tenha sucesso como estratégia para aumentar as receitas. Onde a concorrência pode ser menor, mesmo que seu produto ou serviço não se destaque, uma empresa pode se engajar em preços orientados para o cliente porque a oferta pode ser restrita em relação à demanda.

> A estratégia de preços orientada para o cliente funciona bem para produtos que atraem as necessidades emocionais dos consumidores e em nichos de mercado.

>

## Preços orientados para o cliente na era do comércio eletrônico

Em épocas anteriores, quando a informação não era tão fluida como hoje, as empresas tinham mais liberdade para variar os preços de bens e serviços entre os diferentes grupos de clientes. Os atributos de produtos e serviços, assim como os preços, não eram tão transparentes como são hoje. Isso apresenta desafios para as empresas definirem preços orientados para o cliente porque as vantagens da informação foram erodidas.

Ainda é possível, no entanto, que as empresas que se mantêm atualizadas sobre as necessidades de clientes de longa data retenham [o poder de precificação](/pricingpower) quando fornecem valor de produto ou serviço que excede o custo para os clientes.

## Destaques

- Para serem eficazes, as empresas devem considerar a melhor forma de segmentar o mercado para que os preços reflitam as percepções de valor desses segmentos.

- A precificação orientada para o cliente é uma estratégia de precificação na qual uma empresa define os preços de acordo com o valor percebido pelos clientes de seus produtos e serviços.
