# Unidade de Manutenção de Estoque (SKU)
## O que é uma unidade de manutenção de estoque?

Uma unidade de manutenção de estoque (SKU) é um código de barras digitalizável, mais frequentemente visto impresso em etiquetas de produtos em uma loja de varejo. A etiqueta permite que os fornecedores rastreiem automaticamente o movimento do estoque. O SKU é composto por uma combinação alfanumérica de cerca de oito caracteres. Os caracteres são um código que rastreia o preço, os detalhes do produto e o fabricante. Os SKUs também podem ser aplicados a produtos intangíveis, mas faturáveis, como unidades de tempo de reparo em uma oficina de automóveis ou [garantias](/warranty).

## Entendendo as unidades de manutenção de estoque (SKUs)

Os SKUs são usados por lojas, catálogos, fornecedores de comércio eletrônico, prestadores de serviços, depósitos e centros de atendimento de produtos para rastrear os níveis de estoque. SKUs escaneáveis e um sistema POS significam que é fácil para os gerentes determinarem quais produtos precisam ser reabastecidos. Quando um cliente compra um item no ponto de venda (POS), o SKU é digitalizado e o sistema POS remove automaticamente o item do estoque, além de registrar outros dados, como o preço de venda. Os SKUs não devem ser confundidos com números de modelo, embora as empresas possam incorporar números de modelo nos SKUs.

> Ao adicionar SKUs a cada produto, os lojistas podem rastrear facilmente a quantidade de produtos disponíveis. Os proprietários podem criar limites para avisá-los quando novos pedidos de compra devem ser feitos.

>

As empresas criam diferentes SKUs para seus produtos e serviços. Por exemplo, uma loja que vende sapatos cria SKUs internos que mostram os detalhes de um produto, como cor, tamanho, estilo, preço, fabricante e marca. Por exemplo, o SKU para botas Ugg roxas no estilo Bailey Bow, tamanho 6, pode ser "UGG-BB-PUR-06".

### A importância das unidades de manutenção de estoque

Os SKUs permitem que os compradores comparem características de itens semelhantes. Por exemplo, quando um comprador compra um DVD específico, os varejistas on-line podem exibir filmes semelhantes comprados por outros clientes com base nas informações de SKU. Este método pode desencadear compras adicionais pelo cliente, aumentando assim a receita de uma empresa. Os SKUs também permitem a coleta de dados sobre vendas. Por exemplo, uma loja pode ver quais itens estão vendendo bem e quais não são baseados nos SKUs digitalizados e nos dados do PDV.

## Unidades de manutenção de estoque vs. Códigos de produtos universais

Como as empresas criam internamente SKUs para rastrear o estoque, as SKUs para produtos idênticos variam entre as empresas. Diferentes SKUs ajudam os varejistas a projetar campanhas publicitárias sem interferência de outros fornecedores.

Por exemplo, se uma empresa fornecer o SKU para anunciar um determinado refrigerador com desconto, os compradores não poderão visualizar facilmente o mesmo refrigerador em outros vendedores com base apenas no SKU. Isso impede os concorrentes de igualar os preços anunciados e roubar clientes. Por outro lado, os códigos de produtos universais (UPCs) são idênticos, independentemente de qual empresa esteja vendendo os itens.

## Exemplos de SKUs no mundo moderno

Os SKUs estão tornando a experiência de compra mais eficiente do que nunca. Por exemplo, ao fazer compras de sapatos no passado, os balconistas teriam que vasculhar visualmente o estoque dos fundos e procurar um modelo específico de sapatos no tamanho correto. Hoje, muitos varejistas estão equipados com scanners portáteis que permitem que os vendedores verifiquem o estoque da loja simplesmente digitalizando uma amostra do chão. Este é um dos muitos benefícios do sistema SKU moderno.

##Destaques

- Os SKUs ajudam os fornecedores a determinar quais produtos precisam ser reordenados e fornecem dados de vendas.

- SKUs também são usados para unidades de tempo de reparo, serviços e garantias.

- Uma unidade de manutenção de estoque (SKU) é um código de barras digitalizável para ajudar os fornecedores a rastrear automaticamente o movimento do estoque.
