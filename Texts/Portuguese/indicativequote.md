# Cotação Indicativa
## O que é uma cotação indicativa?

Uma cotação indicativa é uma estimativa razoável do preço de mercado atual de uma [moeda](/currency) que é fornecida por um [formador de mercado](/marketmaker) a um investidor mediante solicitação. No entanto, essa taxa não pode ser negociada, daí a palavra indicativa.

Em outras palavras, quando um formador de mercado fornece uma cotação indicativa a um trader, o formador de mercado não é obrigado a honrar o preço declarado nessa cotação se a contraparte optar por realizar transações nesse [par de moedas](/currencypair).

## Entendendo a cotação indicativa

Os formadores de mercado normalmente fornecerão uma cotação indicativa se uma contraparte, geralmente um trader ou cliente, manifestar interesse em negociar um par de moedas, mas não fornecer nenhuma informação adicional, como o volume a ser transacionado. [Os revendedores](/dealer) também podem fornecer cotações indicativas se as condições voláteis do mercado limitarem sua capacidade de negociar às taxas de mercado prevalecentes para essa moeda. O ponto chave a ser observado é que o formador de mercado não é obrigado a honrar uma cotação indicativa.

Geralmente. os clientes solicitarão cotações para ter uma ideia de onde uma moeda está sendo negociada antes de tomar a decisão de agir. Isso é especialmente verdadeiro para grandes negócios porque há mais em jogo para cada movimento incremental de preço, ou [pip](/pip).

O que eles procuram é uma aproximação, embora bastante precisa, da taxa de mercado de um par de moedas. Como o cliente manifestou interesse em realizar a transação, um revendedor fornecerá a melhor cotação não garantida de onde esse par de moedas pode ser comprado ou vendido. Na linguagem do mercado, isso é conhecido como cotação indicativa

Uma cotação indicativa está em contraste direto com uma [cotação firme](/firmquote). Com este tipo de cotação, um revendedor informa o preço, volume e quaisquer outros detalhes que sejam pertinentes para concluir uma transação. É inegociável e garantido, o que significa que será honrado se a contraparte optar por transacionar esta cotação.

## Exemplo de uma cotação indicativa

Uma empresa norte-americana está prestes a adquirir uma empresa francesa por 200 milhões de [euros](/euro). Eles solicitam uma cotação para avaliar a quantidade de [dólares americanos](/usd-united-states-dollar) que serão necessários para serem trocados para concluir esta transação. A taxa interbancária para [EUR/USD](/eur-usd-euro-us-dollar-currency-pair) é 1,1520/1,1525, o que significa que 1,1525 USD serão necessários para comprar um EUR. O formador de mercado, ciente do interesse da empresa em comprar uma grande quantidade de Euros, cota-lhes uma taxa indicativa de 1,1535. A partir daqui, o cliente pode pedir uma cotação firme ou passar e voltar mais tarde.

A [linha inferior](/bottomline) é que os comerciantes podem confiar em cotações indicativas para ter uma boa ideia da taxa de [câmbio](/exchangerate) na qual eles podem entrar em uma transação de câmbio, embora não haja nenhum compromisso do formador de mercado em honrar essas cotações.

## Destaques

- O ponto chave a ser observado é que o formador de mercado não é obrigado a honrar uma cotação indicativa.

- Uma cotação indicativa é uma estimativa razoável do preço de mercado atual de uma moeda que é fornecida por um formador de mercado a um investidor mediante solicitação.

- Uma cotação indicativa está em contraste direto com uma cotação firme, que é garantida pelo formador de mercado.
