# Análise Intermercado
## O que é análise intermercado?

A análise intermercado é um método de análise de mercados examinando as correlações entre diferentes [classes de ativos](/assetclasses). Em outras palavras, o que acontece em um mercado pode, e provavelmente afeta, outros mercados, então um estudo do(s) relacionamento(s) pode ser benéfico para o trader.

## Entendendo a Análise Intermercado

A análise intermercado analisa mais de uma classe de ativos ou mercado financeiro relacionado para determinar a força ou fraqueza dos mercados financeiros ou classes de ativos que estão sendo considerados.

Em vez de olhar para os mercados financeiros ou classes de ativos individualmente, a análise intermercado analisa vários mercados fortemente correlacionados, ou classes de ativos, como [ações](/stock),. [títulos](/bondmarket),. [moedas](/currency-exchange) e [commodities](/commodity). Esse tipo de análise se expande simplesmente olhando para cada mercado individual ou ativo isoladamente, observando também outros mercados ou ativos que tenham uma forte relação com o mercado ou ativo sendo considerado.

Por exemplo, ao estudar o mercado dos EUA, vale a pena observar o mercado de títulos dos EUA, os preços das commodities e o [dólar americano](/usd-united-states-dollar). As mudanças nos mercados relacionados, como os preços das commodities, podem ter um impacto no mercado de ações dos EUA e precisariam ser compreendidas para obter uma maior compreensão da direção futura do mercado de ações dos EUA.

A análise intermercado deve ser considerada [análise fundamental,](/fundamentalanalysis) pois se baseia mais em relacionamentos para fornecer um senso geral de direção, mas muitas vezes é classificada como um ramo da [análise técnica](/technicalanalysis). Existem diferentes abordagens para a análise intermercado, incluindo mecânica e baseada em regras.

## Correlações de Análise Intermercado

Realizar uma análise das relações intermercados é relativamente simples, onde seriam necessários dados, amplamente disponíveis e gratuitos nos dias de hoje, e uma planilha ou programa de gráficos. Um estudo de correlação simples é o tipo de análise intermercado mais fácil de realizar. Esse tipo de análise é quando uma variável é comparada com uma segunda variável em um conjunto de dados separado.

Uma correlação positiva pode chegar a +1,0, o que representa uma correlação perfeita entre os dois conjuntos de dados. Uma correlação inversa perfeita (negativa) representa um valor tão baixo quanto -1,0. Leituras próximas à linha zero indicariam que não há correlação discernível entre as duas amostras.

Uma correlação perfeita entre quaisquer dois mercados por um período de tempo muito longo é rara, mas a maioria dos analistas provavelmente concordaria que qualquer leitura sustentada acima do nível +0,7 ou abaixo do nível -0,7 (o que equivaleria a aproximadamente 70% de correlação) é estatisticamente significativo. Além disso, se uma correlação passar de positiva para negativa, o relacionamento provavelmente seria instável e provavelmente inútil para negociação.

[correlação](/correlation) mais aceita é a correlação inversa entre os preços das ações e as taxas de juros, que postula que à medida que as taxas de juros sobem, os preços das ações caem e, inversamente, à medida que as taxas de juros caem, os preços das ações sobem.

## Destaques

- A análise intermercado é um método de análise de mercados examinando as correlações entre diferentes classes de ativos.

- Um estudo de correlação simples é o tipo de análise intermercado mais fácil de realizar, onde os resultados variam de -1,0 (correlação negativa perfeita) a +1,0 (correlação positiva perfeita).

- A correlação mais aceita é a correlação inversa entre os preços das ações e as taxas de juros, que postula que, à medida que as taxas de juros sobem, os preços das ações caem e vice-versa.
