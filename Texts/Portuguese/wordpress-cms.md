# WordPress (CMS)
## O que é WordPress?

O WordPress é um popular sistema de gerenciamento de conteúdo de código aberto (CMS). Embora originalmente associado principalmente a blogs pessoais, desde então passou a ser usado para uma ampla variedade de sites, incluindo publicações profissionais e plataformas [de comércio](/ecommerce) eletrônico.

Desde o seu lançamento em 2003, o WordPress cresceu para alimentar cerca de um terço dos sites em todo o mundo. Hoje, é o lar de uma grande comunidade de desenvolvedores que produziu mais de 50.000 plugins.

## Como o WordPress funciona

Os usuários do WordPress podem selecionar entre duas versões básicas do serviço. A primeira delas é a versão “hospedada” do WordPress, que pode ser acessada em WordPress.com. Esta versão permite que os usuários iniciem um site hospedado nos próprios servidores do WordPress. Como tal, muitas vezes é favorecido por usuários não técnicos, que podem desejar evitar a complexidade de hospedar um site em seu próprio servidor privado. Como benefício adicional, a versão hospedada do WordPress é atualizada automaticamente para a versão mais recente do software WordPress, o que pode ajudar a mitigar possíveis violações de segurança.

Para usuários mais avançados, também é possível simplesmente baixar a versão mais recente do WordPress do WordPress.org. Ao selecionar esta opção, o usuário deve fazer seus próprios arranjos para hospedar o software em seu próprio servidor privado ou de um provedor de hospedagem [de terceiros .](/third-party) O usuário é então livre para personalizar o software conforme se adequa, mas também deve ser responsável por fazer o upload manual de quaisquer atualizações subsequentes.

Devido à sua grande base de usuários, os sites do WordPress são um alvo popular entre os hackers, que geralmente procuram explorar versões desatualizadas do software que podem possuir vulnerabilidades de segurança conhecidas. Por esse motivo, é prudente evitar a auto-hospedagem do serviço, a menos que você esteja confiante em sua capacidade de proteger e atualizar o serviço juntamente com quaisquer plug-ins adicionais que você possa ter instalado.

> ### Personalizações WordPress

> O WordPress foi projetado desde o início para ser altamente personalizável. Sua [interface de programação de aplicativos (API)](/application-programming-interface) abrange tudo, desde seu banco de dados até modificações e personalizações de temas. Mantendo tudo padronizado, os desenvolvedores podem se concentrar em construir funcionalidades em cima do WordPress, sabendo que ele funcionará em todas as instalações que forem mantidas atualizadas.

>

## Exemplo do mundo real do WordPress

Existem muitas empresas diferentes que fornecem plugins e temas do WordPress projetados para estender sua funcionalidade. Por exemplo, o WooCommerce se tornou uma das ferramentas de comércio eletrônico mais populares para WordPress, permitindo que os usuários aceitem pedidos, rastreiem remessas e adicionem outras funcionalidades de comércio eletrônico ao seu site WordPress existente .

Entre os muitos outros exemplos de plugins para WordPress estão aqueles preocupados com otimização de mecanismos de busca (SEO), formulários de contato, marketing de newsletter, mídia social no site e recursos de comentários e vários recursos de design projetados para tornar os sites mais visualmente atraentes.

##Destaques

- WordPress é uma plataforma CMS de código aberto líder.

- É conhecido por sua natureza altamente personalizável, ostentando uma grande comunidade de desenvolvedores de plugins.

- Devido à sua grande base de usuários, o WordPress é frequentemente alvo de hackers. Para reduzir esse risco, os usuários devem garantir que seu software WordPress seja mantido atualizado o tempo todo.
