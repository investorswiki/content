# Dados sociais
Dados sociais são informações que os usuários de mídia social compartilham publicamente, o que inclui metadados como a localização do usuário, idioma falado, dados biográficos e/ou links compartilhados. Os dados sociais são valiosos para profissionais de marketing que procuram insights de clientes que possam aumentar as vendas ou, no caso de uma campanha política, ganhar votos. Existem muitos tipos de dados sociais, incluindo tweets do Twitter, posts no Facebook, pins no Pinterest, posts no Tumblr e check-ins no Foursquare e Yelp. O Facebook for Business e o Twitter Ads são dois programas que ajudam os anunciantes a usar [dados sociais para comercializar](/social-media-marketing-smm) para [usuários-alvo](/target-market) que provavelmente estarão interessados em seus anúncios.

## Detalhamento de dados sociais

Os usuários voluntariamente tornam públicos muitos de seus dados sociais, permitindo que as empresas tenham acesso fácil e gratuito a eles. Se uma empresa que vende ingressos para eventos esportivos perceber que um usuário segue vários times esportivos, essa empresa pode segmentar anúncios para esse usuário para tentar convencê-lo a comprar ingressos para ver seu time favorito jogar. Outra maneira pela qual uma empresa pode usar dados sociais é fornecer anúncios oportunos com base em postagens recentes, como anúncios de eletrodomésticos para alguém que compartilhou que está comprando uma casa.

Com dados sociais de alta qualidade agregados e analisados corretamente, as empresas podem direcionar anúncios para as pessoas com maior probabilidade de comprar seus produtos ou serviços. Os dados sociais também podem ajudar as empresas a determinar os locais mais eficazes para anunciar. As empresas podem refinar ainda mais sua publicidade ao restringir seu público-alvo por gênero, idioma falado, dispositivo eletrônico usado, idade, interesses, localização e outros fatores. Os dados sociais não apenas ajudam as empresas a adquirir novos clientes, mas também as ajudam a se envolver ainda mais com os clientes existentes.

## Analisando dados sociais

Normalmente, há duas etapas para analisar dados sociais. A primeira é coletar os dados gerados pelos usuários em sites de rede e depois analisar esses dados. O processo de análise geralmente ocorre em tempo real — e isso é usado para determinar influência, alcance, relevância e outras considerações. As empresas que usam esse tipo de análise de dados precisam manter várias coisas em mente, incluindo como distinguir entre dados sociais e sentimentos, relevância de tempo (o que é relevante hoje pode não ser amanhã), qualidade (o impacto de determinadas mensagens e comentários de pessoas específicas ), e como a atividade viral começa e se espalha.

## Limitações dos Dados Sociais

Os dados sociais são imperfeitos por várias razões. Limita-se às informações que os usuários decidem compartilhar sobre si mesmos. Por exemplo, alguns usuários podem não compartilhar sua localização ou gênero, dando aos anunciantes um perfil incompleto para trabalhar. Outro problema é que muitos usuários nas [mídias sociais](/social-media) não são usuários reais, mas contas falsas de robôs, ou bots. Mesmo com usuários reais, tentar avaliar seus sentimentos sobre uma marca ou candidato político (chamado “análise de sentimentos”) com base nos comentários que eles fazem nem sempre é possível porque muitos de seus comentários são neutros e os [algoritmos](/algorithm) podem classificar incorretamente os comentários como positivos quando eles são negativos e vice-versa. Além disso, muitos comentários positivos e negativos disponíveis são extremos, tornando difícil avaliar com precisão como os consumidores em geral se sentem em relação a um produto, serviço, marca ou candidato político.
