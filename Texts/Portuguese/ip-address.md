# Endereço de IP
## O que é um endereço IP?

Endereço IP significa endereço de protocolo de internet; é um número de identificação que está associado a um determinado computador ou rede de computadores. Quando conectado à internet, o endereço IP permite que os computadores enviem e recebam informações.

## Como funciona um endereço IP

Um endereço IP permite que os computadores enviem e recebam dados pela Internet. A maioria dos endereços IP são puramente numéricos, mas à medida que o uso da Internet cresce, letras foram adicionadas a alguns endereços.

Existem quatro tipos diferentes de endereços IP: público, privado, estático e dinâmico. Enquanto o público e o privado são indicativos da localização da rede – o privado sendo usado dentro de uma rede enquanto o público é usado fora de uma rede – estático e dinâmico indicam permanência.

Um endereço IP estático é aquele que foi criado manualmente, em vez de ter sido atribuído. Um endereço estático também não muda, enquanto um endereço IP dinâmico foi atribuído por um servidor DHCP (Dynamic Host Configuration Protocol) e está sujeito a alterações. Os endereços IP dinâmicos são o tipo mais comum de endereços de protocolo da Internet. Os endereços IP dinâmicos ficam ativos apenas por um determinado período de tempo, após o qual expiram. O computador solicitará automaticamente uma nova concessão ou poderá receber um novo endereço IP.

Um endereço IP pode ser comparado a um [Social Security Number](/ssn) (SSN), pois cada um é totalmente exclusivo para o computador ou usuário ao qual está atribuído. A criação desses números permite que os roteadores identifiquem para onde estão enviando informações na internet. Eles também garantem que os dispositivos corretos estejam recebendo o que está sendo enviado. Assim como os correios precisam de um endereço de correspondência para entregar um pacote, um roteador precisa de um endereço IP para entregar ao endereço da Web solicitado.

## Exemplo de um endereço IP

A dark web refere-se ao conteúdo online criptografado que não é indexado pelos mecanismos de pesquisa convencionais. A dark web tem um mercado online ilegal onde os criminosos podem negociar mercadorias ilegais e ilícitas. Muitas dessas trocas ocorrem usando a criptomoeda online bitcoin, o que torna mais difícil para as autoridades rastrear e capturar as pessoas que participam dessas transações.

Em 2018, após uma operação governamental de um ano, agentes que trabalhavam com o Departamento de Segurança Interna se passaram por traficantes de armas para obter acesso aos computadores de suspeitos que tentavam comprar armas ilegalmente. Isso permitiu que eles tivessem acesso a endereços IP, que eles usaram para rastrear as localizações geográficas de suspeitos adicionais que estavam usando a dark web.

Esta não é a primeira vez que endereços IP levam a prisões. Em 2012, a polícia usou endereços IP para rastrear e prender membros do grupo de hackers Lulzsec. Usando mandados para obter informações do [Provedor de Serviços de Internet (ISP)](/isp),. os agentes da lei conseguiram rastrear os endereços físicos dos hackers e prendê-los por sua atividade ilegal na Internet.

## Destaques

- Um endereço de protocolo de Internet (IP) permite que os computadores enviem e recebam informações.

- Um endereço IP permite que as informações sejam enviadas e recebidas pelas partes corretas, o que significa que também pode ser usado para rastrear a localização física de um usuário em alguns casos.

- Existem quatro tipos de endereços IP: público, privado, estático e dinâmico.

## PERGUNTAS FREQUENTES

### O que é um endereço IPv6?

Os endereços IPv4 consistem em quatro numerais de um a três dígitos separados por um ponto. O IPv6 é a versão mais recente do endereçamento da Internet que usa um valor alfanumérico de 128 bits para identificar um dispositivo terminal online. Um endereço IPv4 pode ser algo como 208.80. 154.224, e um endereço IPv6 pode ter a seguinte aparência: 2001:0db8:85a3:0000:0000:8a2e:0370:7334.IPv6 permite que muito mais dispositivos conectados existam na Internet sem duplicar endereços, o que pode causar problemas.

### Como alguém pode "falsificar" seu endereço IP?

Você pode fazer seu endereço IP parecer que vem de outro lugar por meio de uma rede privada virtual (VPN). Esses serviços basicamente mascaram o endereço de internet do seu computador ao ingressar virtualmente na rede que eles estabelecem. Portanto, seu endereço IP aparecerá como deles. Os hackers também podem falsificar endereços IP usando técnicas mais sofisticadas que modificam o endereço de origem em um cabeçalho de pacote de dados maliciosos para imitar uma fonte confiável e aceitá-la.

### Como faço para descobrir meu endereço IP?

Existem várias maneiras de determinar seu endereço IP. Talvez o mais simples seja digitar "qual é o meu endereço IP" em um mecanismo de pesquisa da Internet como o Google. Para usuários do Windows, você pode encontrar seu IP abrindo [Iniciar > Configurações > Rede e Internet] e, em seguida, selecione a rede Wi-Fi à qual está conectado. Em Propriedades, encontre seu endereço IP listado ao lado de "Endereço IPv4".
