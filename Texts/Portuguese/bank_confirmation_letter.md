# Carta de confirmação bancária (BCL)
## O que é uma carta de confirmação bancária (BCL)?

Uma carta de confirmação bancária (BCL) é uma carta de um banco ou [instituição financeira](/financialinstitution) confirmando a existência de um empréstimo ou uma [linha de crédito](/lineofcredit) que foi estendida a um mutuário. A carta atesta oficialmente o fato de que o mutuário - normalmente um indivíduo, empresa ou organização - é elegível para emprestar uma quantia específica de fundos para um propósito específico.

## Como funciona uma carta de confirmação bancária (BCL)

O objetivo de uma carta de confirmação bancária é garantir a um terceiro, geralmente um vendedor, que o mutuário tem acesso a recursos financeiros suficientes para concluir uma transação, como a compra de mercadorias. A carta de confirmação – às vezes conhecida como [carta de conforto –](/comfort_letter) não é uma garantia de pagamento, mas apenas uma garantia dos recursos financeiros do mutuário para efetuar o pagamento.

As cartas de confirmação bancária normalmente exigem a assinatura de representantes do banco ou da instituição financeira que estão autorizados a emitir tal correspondência.

Como uma carta de confirmação é emitida em relação a uma transação ou projeto específico, ela não é transferível para uma transação ou projeto diferente. Se o cliente do banco decidir entrar em um negócio ou compra diferente, o cliente geralmente é solicitado a obter uma nova carta de confirmação.

Por exemplo, um potencial comprador de uma casa decide comprar uma casa diferente da especificada na carta de confirmação bancária; seria necessário um novo BCL.

As regulamentações variam de país para país em termos de se e até que ponto uma carta de confirmação deve indicar a finalidade específica para a qual um empréstimo ou linha de crédito está sendo concedido ao mutuário.

## Usos comuns de uma carta de confirmação bancária

As cartas de confirmação bancária são mais comumente preparadas para um cliente empresarial do banco, atestando a existência de uma linha de crédito específica. As cartas muitas vezes servem para tranquilizar os vendedores de um grande número de mercadorias.

Eles também podem ser emitidos para uma empresa que está entrando em um [projeto de joint venture](/jointventure) com outra empresa. Embora a carta não garanta o pagamento ou o fornecimento de fundos, ela fornece uma garantia de alta probabilidade de a empresa receber o pagamento do cliente do banco.

> Uma carta de confirmação bancária serve para assegurar a todas as partes envolvidas em uma transação comercial que o cliente do banco (o mutuário) possui, ou dispõe, dos recursos financeiros necessários para concluir a transação.

>

O uso mais comum de uma carta de confirmação bancária por um indivíduo é durante a compra de uma casa ou terreno. Nesses casos, a carta fornece confirmação ao vendedor ou corretor de imóveis de que o cliente do banco está aprovado para uma [hipoteca](/mortgage) de até um valor especificado para uma compra proposta.

A carta não é um compromisso de compra do imóvel; é apenas uma garantia de que o cliente do banco tem acesso a fundos para concluir uma compra. Na maioria das situações, um possível comprador não poderá fechar um imóvel sem ter uma carta de confirmação bancária em mãos.

## Perguntas frequentes sobre a carta de confirmação bancária

### Como faço para obter uma carta de confirmação bancária?

Uma carta de confirmação bancária pode ser recebida do seu banco mediante solicitação. O banco emitirá a carta com as assinaturas apropriadas e a fornecerá a você.

### O que é uma carta de certificação bancária?

Uma carta de certificação bancária é uma carta emitida por um banco que confirma que um indivíduo tem uma conta nesse banco e o valor total dos fundos na conta.

### O que é uma carta de verificação bancária?

Uma carta de verificação bancária é o mesmo que uma carta de certificação bancária; uma carta de um banco confirmando que um indivíduo tem uma conta nesse banco com o valor total dos fundos na conta.

### Como recebo uma carta de confirmação bancária do meu banco?

Para obter uma carta de confirmação bancária do seu banco, você pode solicitar pessoalmente em uma agência bancária de um dos banqueiros, por telefone para o banco e, dependendo da instituição financeira, por meio de sua plataforma online.

##Destaques

- Uma carta de confirmação bancária (BCL) valida que um banco tem uma linha de crédito com um de seus clientes.

- Cartas de confirmação bancária também podem ser emitidas para uma empresa que está participando de um projeto de joint venture com outra empresa.

- As cartas de confirmação bancária são normalmente emitidas para clientes empresariais, atestando sua solvência.

- O BLC não é uma garantia de pagamento, mas uma garantia de recursos financeiros do mutuário para concluir uma compra.

- As pessoas físicas podem solicitar um BCL durante a compra de uma casa ou terreno para garantir uma hipoteca ou estabelecer a credibilidade com o vendedor.
