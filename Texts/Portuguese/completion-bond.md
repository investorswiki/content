# Título de Conclusão
## O que é um título de conclusão?

Um título de conclusão é um contrato que [garante](/financial-guarantee) uma compensação monetária se um determinado projeto não for concluído. Ele fornece proteção se o empreiteiro ficar sem dinheiro ou quaisquer outros problemas orçamentários surgirem durante o projeto. Muitas empresas usam títulos de conclusão, incluindo filmes, videogames e projetos de construção.

Uma garantia de conclusão também pode ser conhecida como garantia de conclusão.

## Entendendo as obrigações de conclusão

Um título de conclusão é um tipo específico de [título de](/surety) [garantia](/surety). A fiança é uma garantia financeira de que uma compensação será paga a uma determinada parte se um contrato não for cumprido de forma satisfatória ou completa. A fiança é um contrato celebrado por pelo menos três partes. O primeiro é o devedor, que é o cliente, proprietário ou parte que exige que o título seja lançado para sua proteção. O segundo é o principal, que é a parte principal que promete concluir o projeto ou contrato. O terceiro é o fiador ou [devedor](/obligor),. que assegura ao credor que a tarefa ou projeto pode ser cumprido até o fim.

Os títulos de conclusão são frequentemente usados para projetos complexos que envolvem grandes somas de dinheiro ou vários investidores. Para garantir o financiamento necessário, um contratante fará uma garantia de empréstimo a uma instituição de crédito na forma de uma garantia de conclusão. O vínculo garante que o projeto será concluído no prazo, dentro do orçamento e livre de [ônus](/construction-lien). Um fiador terceirizado avaliará o risco para a conclusão do projeto e cobrará um prêmio para segurar os riscos específicos de um determinado projeto. Assim, um título de conclusão garante que um credor ainda receba o principal e os juros, mesmo que o projeto não seja concluído.

## Garantia de conclusão vs. Garantia de execução

Um título de conclusão oferece mais cobertura do que um [vínculo de desempenho](/performancebond). Uma garantia de desempenho é uma [garantia de indenização](/indemnity) que garante a conclusão satisfatória do trabalho do contrato por um empreiteiro. Enquanto as obrigações de conclusão criam uma garantia entre o devedor e seu credor como devedor, as obrigações de desempenho criam uma garantia entre o devedor e o devedor contratual.

O devedor recebe uma compensação por quaisquer perdas incorridas se o devedor violar os termos contratuais do acordo. Múltiplos títulos de conclusão podem ser necessários para cada contrato dentro de um projeto.

## Exemplos de títulos de conclusão

As razões pelas quais os projetos não são concluídos diferem entre [os setores](/industry). Naturalmente, os títulos de conclusão também devem funcionar de forma ligeiramente diferente para projetos em vários negócios.

### Construção

Como os projetos de construção podem levar muitos meses ou até anos para serem concluídos, os riscos para os investidores podem ser altos. Os investidores são muito mais propensos a se envolver se uma garantia de conclusão for fornecida. Dessa forma, eles sabem que receberão seu dinheiro de volta com juros se o projeto não for concluído.

### Entretenimento

Os títulos de conclusão são uma tradição de longa data no negócio do entretenimento. Muitas variáveis podem entrar em jogo que podem afetar a conclusão de um grande projeto de filme. Nesse caso, os produtores do filme fornecerão uma fiança de conclusão a um banco para financiar o projeto do filme. Em troca de garantir o [reembolso](/repayment) do empréstimo, os produtores geralmente não precisam fazer nenhum reembolso do empréstimo até que o projeto seja concluído. Todos os profissionais que trabalham no filme se beneficiam do vínculo de conclusão porque os produtores são desencorajados a encerrar o projeto antes da conclusão.

### Hipoteca

Um título de conclusão pode ser parte de um acordo de financiamento hipotecário e protege tanto o devedor [quanto o credor](/mortgagor) [hipotecário](/mortgagee). Um financiador terceirizado, geralmente uma empresa de garantia de conclusão, normalmente se envolve no negócio. O terceiro fornece um suporte financeiro se o financiamento original for insuficiente para concluir o projeto.

## Destaques

- Os títulos de conclusão são usados com mais frequência para projetos de construção, mas também são encontrados na indústria do entretenimento e em algumas hipotecas.

- Os títulos de conclusão são mais adequados para projetos multifacetados envolvendo vários financiadores.

- Uma garantia de conclusão é um contrato financeiro que paga integralmente quando um projeto de construção ou infraestrutura é concluído, oferecendo uma cobertura maior do que uma garantia de desempenho.
