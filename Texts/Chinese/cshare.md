# C类股
## 什么是 C 类股票？

C 类股票是一类共同基金份额，其特点是水平负荷，包括基金营销、分销和服务的年度费用，设定为固定百分比。这些费用相当于公司或个人的佣金，帮助投资者决定拥有哪只基金。费用按年收取。

相比之下，前端负载在购买股票时收取费用，而后端负载在投资者出售股票时评估费用；空头基金完全不收取佣金，费用仅计入基金的资产净值（NAV）。

## C 类股票的基础知识

与其他共同基金股票类别相比，C 类股票的费用比率通常低于[B 类股票](/classbshares)。但是，它们的费用比率高于 A 类股。费用比率是运营共同基金的总体年度管理成本。因此，对于那些计划将共同基金保留几年的相对短期投资者来说，C 类股票可能是一个不错的选择。

构成 C 股级别负载的持续收费正式称为 12b-1 费用，命名自[1940 年《投资公司法》的一部分](/investmentcompanyact)。 12b-1 总费用的上限为每年 1%。在这 1% 的费用中，分销和营销费用最高可达 0.75%，而服务费最高可达 0.25%。尽管指定用于营销，12b-1 费用主要用于奖励出售基金股份的中介。从某种意义上说，这是投资者每年向共同基金支付的佣金，而不是交易佣金。

其他共同基金份额类别也有 12b-1 费用，但程度不同。对 A 类股票收取的费用通常较低，以补偿该类别支付的高额前期佣金。 C 股往往支付最高 1% 的费用，并且由于 12b-1 费用计入共同基金的整体费用比率，它们的存在可以将 C 类股东的年度费用比率推高至 2% 以上。

与 A 股不同，C 类股票没有前端负载，但它们通常带有小的后端负载，正式称为[或有递延销售费用](/cdsc)(CDSC)，就像 B 类股票一样。然而，C 股的这些负担要小得多，通常只有 1% 左右，而且一旦投资者持有共同基金一年，它们通常就会消失。

## 谁应该投资 C 类股票？

由于短期[赎回收取的后端负担](/redemption)，计划在一年内提取资金的投资者可能希望避免 C 股。另一方面，与 C 股相关的较高持续费用使其成为长期投资者不太理想的选择。

持有不同费用的投资在很长一段时间内（例如，在退休基金中）的最终价值差异可能是巨大的。例如，投资 50,000 美元的基金，该基金的回报率为 6%，每年收取 2.25% 的运营费，持有 30 年。投资者将获得的最终金额将等于 145,093.83 美元。投资金额相同、年回报率相同但年运营费为 0.45% 的基金将为投资者提供更多收益，最终价值为 250,832.55 美元。

C 类股票最适合计划将基金保留有限的中间期的投资者，最好是一年以上但少于三年。这样，您可以坚持足够长的时间来避免 CDSC，但不会太久以至于高费用比率会对基金的整体回报造成重大影响。

## C 类股票的真实世界示例

Calamos 成长基金是同时拥有 A 类和 C 类股票的基金的一个例子。 A 类股票的费用率为 1.40%。其中 0.25% 是 12b-1 费用。他们最多有 4.75% 的前端负载，该负载会根据投资金额而减少。该基金的 C 类股票没有前端负担，但持有不到一年的股票最多可持有 1% 的 CDSC。 C 类股还征收最高 1% 的 12b-1 费用，将基金的整体费用率推高至 2.15%。
