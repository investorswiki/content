# 实时交易报告
## 什么是实时交易报告？

实时交易报告是指[做市商](/marketmaker)(MM) 在每笔交易完成后立即公开报告的监管要求。实时交易报告提高了市场的效率和透明度。

实时交易报告不应与[实时报价](/real-time-quote)(RTQ) 混淆，后者提供有关资产价格的买卖信息，不报告交易细节或交易活动。

## 了解实时交易报告

实时交易报告是做市商在交易执行后 90 秒内公开报告交易的要求。交易股票须接受实时­交易报告，此类报告由金融业监管机构 ( [FINRA](/finra) ) 执行，该机构前身为全国证券交易商协会 ( [NASD](/nasd) )。

FINRA 是一家私营公司，充当自律组织。它是一个非政府组织，负责监管会员交易所市场和经纪公司。作为证券行业最终监管机构（包括 FINRA）的政府机构是证券交易委员会 ( [SEC](/sec) )。

实时交易报告记录在交易报告和合规引擎 ( [TRACE](/trace) ) 中。 TRACE 为个人投资者和市场专业人士提供访问几乎所有[场外](/otc)(OTC) 公共和私人交易活动的信息。 TRACE 计划整合了公共和私人公司债券和机构债务的交易数据，其中包括资产支持证券和抵押贷款支持证券。

TRACE 系统要求执行时间以军事格式报告为东部时间。 TRACE 规则还要求监管报告在东部时间进行，即使这意味着将执行时间和日期都转换为东部时间。但是，公司无需向其客户确认东部时间的交易日期或执行时间。

## 为什么实时报告很重要

实时交易报告增强了市场的[价格透明度](/pricetransparency)。价格透明度是指有关特定股票的[买入价](/bid)和卖出价以及交易数量的信息的可用性。价格透明度很重要，因为了解其他人的出价、询价和交易情况有助于确定证券、商品或服务的供需情况，并进一步确定其真实价值。如果信息被证明不充分或无法访问，则该特定市场可能被视为效率低下。

[市场效率](/marketefficiency)的核心是衡量市场信息的可用性，以在不增加[交易](/transaction)成本的情况下为证券买卖双方提供最大数量的机会进行交易。缺乏价格透明度使消费者和投资者处于不利地位。例如，在医疗保健领域，患者通常不知道特定医疗程序的实际成本，这使他们没有太多机会（如果有的话）协商更好的价格。

＃＃ 强调

- TRACE 等自动化电子系统也允许实时报告场外交易。

- 近乎即时的交易数据提高了证券交易所的市场透明度、问责制和效率。

- 实时交易报告是一项规定，要求做市商和交易所专家在执行后 90 秒内向公众发布交易细节。
