# X-Mark 签名
## 什么是 X-Mark 签名？

X-mark 签名是由一个人代替实际签名制作的。由于文盲或残疾，一个人可能无法在文件上附加完整的签名，以证明其内容已经过审查和批准。为了具有法律效力，X-mark 签名必须有见证。

## 了解 X-Mark 签名

X 标记签名之所以如此命名，是因为历史上签署文件的人只是简单地制作一个类似于字母“X”的交叉影线标记，而不是他们完整的习惯签名。 X 标记的实际形式实际上可能不是字母“X”，并且可能采用其他形式的难以辨认的标记，旨在证明该人理解并同意所述条款。

由于存在明显的[欺诈可能性](/fraud)，可能会对使用 X-mark 签名的文件的有效性和可执行性产生疑问。在某些州，主要法律可以要求法院使带有 X 的遗嘱无效，除非立[遗嘱](/will)人身体或精神上无法签署自己的全名。

如果个人在事故中受伤并需要批准法律文件但不能在物理上形成完整的签名，则他们可能会使用 X 标记签名。例如，立遗嘱人可能需要在医院接受治疗时向责任方授予[授权书。](/powerofattorney)

X-mark 签名可以用于签署支票、商业合同和[期票](/promissorynote)，即使签名者没有身心障碍[。](/promissorynote)

## X-Mark 签名要求

各州可能需要一名以上的证人才能使 X 标记签名有效。 X 标记签名也可能需要经过[公证](/notarize)，以使文件具有可执行性。

使用 X 标记签名的个人可能还需要在签署文件时出示一些身份证明。无论个人情况如何，都可以执行该规定。

证人可能需要也可能不需要提供其身份证明。他们可能被要求提供他们的地址并输入或打印他们的名字。在有争议的遗嘱的情况下，可以传唤证人出庭就立遗嘱人签署文件作证。

## X-Mark 签名的法律挑战

如果对文件提出法律要求，则每个证人与签署文件的人的关系可能会受到质疑。例如，如果 X-mark 签名的唯一见证人是可以从签署的遗嘱中受益的个人，则可能会引起对欺诈的担忧。

签署 X 标记签名的个人的心理能力可能会导致对文件有效性的法律挑战。如果个人患有可能会影响他们对所签署内容的理解的情况，则该文件可能被视为不可执行。如果个人患有痴呆症或其他限制他们理解文件法律效力的能力的疾病，则可能会发生这种情况。
