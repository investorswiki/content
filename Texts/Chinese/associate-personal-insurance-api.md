# 个人保险副学士 (API)
## 什么是个人保险 (API) 助理？

人身保险协会 (API) 是[保险行业的专业认证](/insurance)。顾名思义，API 专注于[个人线路保险](/personal-insurance-lines)，这是一种保护人身伤害、死亡或财产损失的保险。它是 The Institutes 授予的众多称号之一，The Institutes 是一家致力于为保险专业人士提供认证和继续教育的组织。

## API 名称的工作原理

API 称号在希望在个人线路保险领域开始或提升其职业生涯的专业人士中很受欢迎。通过获取 API，专业人士可以证明他们具备非商业保险产品的知识，包括[保险承保](/underwriting)、[保费设置](/insurance-premium)、营销和投资组合管理等关键技能。

对于希望获得更高级认证的专业人士，例如[特许财产保险商 (CPCU)认证，](/chartered-property-casualty-underwriter-cpcu) API 可能是一个受欢迎的选择。这是因为获取 API 所需的许多条件也计入 CPCU 的要求。具体来说，与 API 相关的三项考试中的两项可以重复用于 CPCU 认证，这是财产险行业最负盛名的认证之一。

要获得 API，候选人必须对个人保险市场上出售的保险单种类有广泛的了解。这包括对合同背书、条件和承保标准以及行业法规等主题的广泛教育。 API 通常推荐给为个人保险客户提供服务的保险经纪人、代理人、理赔员、承销商和客户服务人员。

## API 名称的真实示例

候选人在完成 API 时可以采取两种途径。其中第一个包括关于个人保险行业的[承保和营销实践、投资组合管理、财产和](/underwriting)[责任保险](/liability_insurance)以及个人保险的课程。第二条路径包括三门课程，涵盖承销和营销、[风险管理](/riskmanagement)和财产责任以及个人财务规划。

通常，考生需要 12 到 18 个月才能完成这些课程材料，这些材料可以自助访问。对于那些喜欢面对面教学的人，也可以在特定地区离线上课。一旦他们通过了考试，API 的持有者不需要满足持续的教育要求来保持他们的指定。

＃＃ 强调

- API 是个人线路保险行业流行的专业名称。

- 它由研究所管理，通常需要 12 到 18 个月才能完成。

- 许多候选人选择通过追求更高级的证书（例如 CPCU）来增加他们的 API 称号。
