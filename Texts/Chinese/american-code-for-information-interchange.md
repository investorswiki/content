# 美国信息交换代码 (ASCII)
## 什么是美国信息交换代码 (ASCII)？

美国信息交换代码，通常称为 ASCII，是一种计算机文本语言，于 1963 年成为计算机之间传输信息的标准。

ASCII 系统将字母数字字符和符号转换为计算机可读的代码。最初的 ASCII 是纯英文的。现在有两种 ASCII 码；使用七位编码系统的标准代码和使用八位系统的扩展代码。

在 Internet 通信中，ASCII 已逐渐被 Unicode 标准所取代，该标准可以与任何语言一起使用并与 ASCII 兼容。但是，ASCII 仍用于在个人计算机和电子设备上输入文本。

发音为 ASK-y。

### 什么是 ASCII 与 Unicode？

Unicode 可以被视为 ASCII 的通用版本。毕竟，ASCII 是美国信息交换代码，它的第一次迭代包括英语字母表和该语言中使用的符号。后来的版本包括许多其他语言中使用的符号。另一方面，Unicode 以 ASCII 中使用的相同 128 个符号开头，但当前版本包含 144,697 个字符。

###什么是ASCII？

美国信息交换标准代码 (ASCII) 是一种用于文本电子传输的字符编码格式。每个字符都由一个唯一的数字表示。 ASCII 的第一个版本仅包含 128 个字符，代表字母表中的字母，大写和小写，加上一些常用符号，如逗号。后来的版本将 ASCII 扩展为 255 个字符，包括附加符号，如英镑符号 (£) 和西班牙语文本中使用的倒置问号 (¿)。

### ASCII 字符是什么？

在扩展 ASCII 表中，字符代码 0 到 31 是控制代码，例如文本开头和退格。字符代码 32 到 127 主要是字母表中的字母（大写和小写）以及星号和美元符号等符号。字符代码 128 到 255 有多个版本，每个版本都反映了对原始英文 ASCII 的添加，例如倒置感叹号、日元符号和格鲁吉亚逗号。
