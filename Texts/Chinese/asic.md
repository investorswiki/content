# 专用集成电路 (ASIC) 矿工
顾名思义，ASIC 是专为特定用例而设计的[集成电路](/integrated-circuit)，与为我们的计算机和移动设备供电的 CPU 等通用电路不同。

在某些情况下，一些简单的计算任务不需要通用 CPU 带来的财务和计算开销。相反，ASIC 可以用作更简单和有效的替代方案（在成本和能源方面）。

在加密货币领域，ASIC 一词被广泛用于指代比特大陆和下龙矿业等公司正在开发和定期改进的专用硬件。这些硬件设计的唯一目的是挖掘[比特币](/bitcoin)（或其他[加密货币](/cryptocurrency)）。有一些硬币无法使用 ASIC 矿工进行有效开采，因此可能被称为[抗 ASIC](/asic-resistant)加密货币。

简而言之，[挖掘](/mining)是一个过程，包括执行无数的散列函数，直到产生有效的散列输出。找到有效哈希的矿工将其用作他们工作的证明，这授予他们验证下一个交易区块并收集区块奖励的权利。

尽管 ASIC 可以非常高效，但受限于特定用例会使它们在做任何其他事情时完全无用。此外，加密货币领域的持续技术进步带来了新的 ASIC 模型，这些模型很快使旧设计完全无利可图。

关于由 ASIC 引起的算力集中化也存在激烈的争论。一方面，它们为保护和验证区块链提供了急需的算力，但它们也将采矿权集中在少数几家有能力购买数千个 ASIC 来建立和运营大型[采矿场的采矿公司手中](/mining-farm)和矿池。

＃＃ 强调

- 比特币矿工审查和验证以前的比特币交易并创建新区块，以便将数据添加到区块链中。

- 通常，专用集成电路 (ASIC) 被优化为仅计算单个函数或一组相关函数。

- 专用集成电路 (ASIC) 矿工是一种计算机化设备或硬件，仅使用 ASIC 来挖掘比特币或其他加密货币。

＃＃ 常问问题

### 什么是抗 ASIC 硬币？

ASIC 抗性硬币是具有 ASIC 抗性算法的加密货币。用 ASIC 采矿设备开采这种加密货币几乎是不可能的，即使尝试这样做，回报也将非常微不足道。抗 ASIC 硬币的主要理由是保持其区块链的去中心化，这是支持比特币创建的指导原则之一。目前，少数私人矿场和挖矿合约商是比特币全网算力很大一部分的主要来源，这与比特币去中心化的核心原则背道而驰。

### ASIC 挖矿和 GPU 挖矿有什么区别？

ASIC 矿机的开发目的是为了挖掘特定的加密货币，例如比特币或莱特币。 GPU 挖掘涉及使用诸如 NVIDIA 或 AMD 出售的 GPU 来挖掘加密货币。 GPU挖矿的明显优势是硬件明显比ASIC挖矿所需的设备便宜，功耗也更低。然而，由于 GPU 在游戏和计算机显示方面还有其他应用，因此它们在挖掘加密货币方面的效率远低于 ASIC 矿工。

### 什么是比特币挖矿？

挖矿是管理区块链并创建新比特币的过程。比特币矿工的工作是审查和验证以前的比特币交易，然后创建一个新区块，以便将信息添加到区块链中。挖掘过程涉及使用与包含交易数据的[块](/block-bitcoin-block)相关联的内在哈希函数来解决复杂的数学问题。各种比特币矿工之间激烈竞争，以解决一个必要的数学难题。第一个找到难题解决方案的矿工能够授权交易或将比特币添加到区块中。比特币挖矿“彩票”中的每个获胜者都会获得奖励（一定数量的比特币）。奖励包括该区块中交易的所有交易费用，这会激励矿工将尽可能多的交易收集到一个区块中，以增加他们的奖励。
