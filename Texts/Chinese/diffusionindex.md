# 扩散指数
## 什么是扩散指数？

扩散指数是用于显示变化相对于参考点的分散程度的指标，一系列数据可以证明是否存在上升或下降趋势。

扩散指数通常用于突出调查结果。响应是值的形式，并且根据这些值，指定一个基数。许多指标使用 0、50 或 100 的基线。一系列数据显示结果与基线的差距。虽然离基数不远的数据点表明变化不大，但离基数远的数据点可能表明发生了重大变化。

在经济学中，扩散指数用于制定指标，例如[采购经理人指数](/pmi)和[里士满制造业指数](/richmond-manufacturing-index)。与数据基于价格或单位的经济指标不同，使用扩散指数的经济指标基于调查。根据调查问题的框架，扩散指数可以用作预测工具，就像 PMI 被认为是领先的经济指标一样。

## 如何计算扩散指数

计算扩散指数的方法有多种，公式可以是长格式也可以是短格式。

### 美联储公式

[美联储](/federal-reserve)使用基本公式计算扩散指数。该公式用于计算美联储 12 家地区银行中许多指标的指数，包括里士满和堪萨斯城联储的调查。

扩散指数 = [100 * (I - D)] / (I + N + D)

I = 报告增加的受访者人数

N = 报告无变化的受访者人数

D = 报告减少的受访者人数

### IHS Markit 公式

发布美国和其他国家采购经理人指数的研究机构 IHS Markit 有自己的公式，与美联储的不同。

IHS Markit 的扩散指数 = (% 报告“上升”) + [0.5 * (% 报告“相同”)] + [0.0 * (% 报告“下降”)]

这可以简写为：

指数 = 正面回应的百分比 + 0.5（中性回应的百分比）

## 扩散指数示例

例如，对于经济是否会在短期内进入[衰退的抽样调查问题](/recession)，结果显示，70%的受访者回答很有可能，20%回答不太可能，10%回答说可能性很小。

如果将结果输入美联储的扩散指数公式，则计算如下：

[100 * (0.70 - 0.10)] / (0.70 + 0.20 + 0.10) = 60 / 1 = 60

如果使用 IHS Markit 的公式计算，答案将是：

0.70 + (0.5 * 0.20) + (0.0 * 0.10) = 0.70 + 0.10 + 0.00 = 0.80

结果因使用的公式而异。在此示例中分配 0 或 100 的基线将是理想的，因为答案可以被操纵为两位数。

## 如何解释扩散指数

一系列数据可以显示动能，并揭示是否存在上升和下降趋势。高于基线的数据表示扩张，而低于基线的数据表示收缩。接近或处于基线的数据意味着变化很小或没有变化。

## 扩散指数的局限性是什么？

扩散指数可以代表单方面的观点。因此，将扩散指数与其他数据集进行比较可以提供额外的确认或反驳使用扩散指数得出的结论。使用其他指标，例如[相关系数](/correlationcoefficient)，可以显示扩散指数是否是可靠的预测工具。

＃＃ 强调

- 扩散指数是指一组数字或统计数据中的共同趋势。

- 扩散指数可用于发现分歧，这可能表明通过查看股票指数本身看不到的强弱。

- 扩散指数下降表明走高的股票数量减少，这表明股票指数的买盘压力正在减弱和/或卖盘压力正在增加。

- 在股市中，扩散指数上升意味着越来越多的股票走高，这对股票指数走高是积极的。
