# 复合回报
##什么是复合回报？

复合回报[率是回报率](/rateofreturn)，通常以百分比表示，表示一段时间内一系列收益或损失对原始资本金额的累积影响。[复合回报](/compounding)通常以年度形式表示，这意味着报告的百分比数字代表资本随时间复合的年化率。

当以年表示时，复合回报可以称为[复合](/cagr)年增长率（CAGR）。

如果[投资基金](/investment-fund)声称在过去五年中产生了 10% 的年复合回报率，这意味着在其第五年末，该基金的资本规模已增长到与现有资金相同的规模每年年初的收入正好是每年年底的 10%。

## 了解复合回报

复合回报被视为比平均回报更准确地衡量投资回报随时间变化的表现。这是因为[平均年回报率](/aar)没有生效，这导致投资者实际回报的严重错误陈述。平均回报要么高估要么低估回报的增长或下降。实际上，复合回报确保在计算中考虑到波动性，这可能会增加或缩小回报。

## 复合回报示例

例如，假设您以 1,000 美元的初始投资开始。如果您将 1,000 乘以 1.1 五次，即 $1,000 x (1.1)^5^，您最终将得到大约 $1,611。如果 1,000 美元的投资在五年结束时最终价值 1,611 美元，则该投资可以说在该五年期间产生了 10% 的年复合回报。

这是数学：

- 第 1 年：1,000 美元 x 10% = 1,100 美元

- 第 2 年：1,100 美元 x 10% = 1,210 美元

- 第 3 年：1,210 美元 x 10% = 1,331 美元

- 第 4 年：1,331 美元 x 10% = 1,464.10 美元

- 第 5 年：1,464 美元 x 10% = 1,610.51 美元

然而，这并不意味着投资在五年内每年实际升值 10%。五年后最终价值达到 1,611 美元的任何增长模式都相当于 10% 的年化回报率。假设该投资在前四年没有赚到任何钱，然后在最后一年赚了 611 美元（当年的回报率为 61.1%）。这仍然相当于五年测量期间 10% 的年复合回报率，因为最终金额仍然等于 1,000 美元如果每年稳定升值 10% 所达到的水平。

如果上面示例中描述的投资回报是使用平均回报计算的，那么它最终会得到一个不正确的百分比。如果上述投资在前四年没有任何收益，但在第五年获得了 61.1% 的收益，则平均收益为：(0% + 0% + 0% + 0% + 61.1%) / 5 = 12.22%

＃＃ 强调

- 复合回报是资本在一系列时间累积的回报率。

- 与平均回报相比，复合回报是一种更准确的衡量标准，用于计算一段时间内投资的增长或下降。
