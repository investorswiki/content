# IntraFi® 网络存款
## 什么是 IntraFi® 网络存款？

IntraFi® 网络存款（以前称为存款证明账户注册服务 (CDARS)）的目的是帮助投资[存款证明](/certificateofdeposit)(CD) 的人在保持资金投保的同时保持在联邦存款保险公司（ FDIC）每家银行的每位存款人的保险限额为 250,000 美元。使用 IntraFi® 网络存款，投资者可以将资金分散到各种当地的 FDIC 保险银行。

IntraFi® 网络存款还为消费者提供了一种获取 FDIC 保险的方式，以获取其 CD 投资本金和利息，从而为他们提供了一种在[银行倒闭时将风险外包的方式](/bank-failure)。

## 了解 IntraFi® 网络存款

IntraFi® Network Deposits 是由 Promontory Interfinancial Network（由一群前金融监管机构于 2003 年创立）运营的营利性服务。 IntraFi® Network Deposits 包括一个由 3,000 多家美国银行和储蓄机构组成的网络。它被个人、公共基金、企业、非营利组织、信用合作社和财务顾问使用。

IntraFi® Network Deposits 消除了消费者从一家银行到另一家银行存款的需要，并使他们能够将资金投资于期限从一个月到五年不等的 CD。它还为消费者提供 CD 级别的利率，通常比货币市场共同基金和国库券的利率要好。

用户在通过 IntraFi® Network Deposits 进行 CD 投资时协商每个期限的一种利率。这消除了为每张 CD 手动计算支出或协商每个到期日的多个利率的需要。

登记处的每家参与的当地银行都设置自己的利率，并按全部存款金额支付。当地银行还充当 IntraFi 存款的[保管人](/custodian)。 IntraFi® 网络存款的分托管人是纽约梅隆银行。

使用 IntraFi® 网络存款需要首先找到当地的参与银行，然后通过专门针对 IntraFi 存款的单独存款安排协议进行存款。然后，当地参与银行将资金分散到几家成员银行，确保每家银行的资金数额永远不会超过 FDIC 每家银行每位存款人 250,000 美元的限额。

> ### 50 亿美元的 SRI

> 2020 年，通过 IntraFi Network Deposits 存入了近 50 亿美元的社会责任存款。

## 特别注意事项

作为 IntraFi® 网络存款计划的参与者，消费者只能与一家当地银行开展业务。他们会收到一份包含每个帐户信息的合并报表（而不是在多个银行保留多个帐户并维护多个登录名和季度报表）。由于银行付费参与 IntraFi® 网络存款登记，因此消费者直接向其主要银行支付任何适用费用。

消费者还可以通过开设个人账户或使用经纪 CD 来避免超过联邦存款保险公司 (FDIC) 的保险限额。然而，这些方法更复杂，需要更多的时间来执行。

＃＃ 强调

- IntraFi® 网络存款（前身为存款证明账户注册服务 (CDARS)）帮助拥有大额存款的消费者将其资金保持在 FDIC 每家银行每位存款人 250,000 美元的保险限额以下。

- 使用 IntraFi 存款的客户可以在一家本地银行（而不是在网络上的许多银行）开立账户。

- IntraFi® 网络存款通过在其由 3,000 多家机构组成的庞大网络中的各种当地 FDIC 保险银行开设账户来运作。

- 它有效地允许超过 250,000 美元的存款有效地成为 FDIC 保险。

＃＃ 常问问题

### 哪些组织认可了 IntraFi® 网络存款？

IntraFi® Network Deposits 已获得美国银行家协会 (ABA) 以及各州级银行协会的专业认可。它还与美国独立社区银行家、FiServ 和纽约梅隆银行结成了联盟。

### 为什么使用 IntraFi 存款？

使用 IntraFi® 网络存款，您可以通过与您选择的当地银行建立单一关系，将现金余额存入活期存款账户、货币市场存款账户和 CD，所有有息账户都可以获得数百万 FDIC 保险.

### IntraFi 网络存款如何允许单笔存款超过 250,00 美元的 FDIC 覆盖？

当您在一家具有单独保管能力的银行存款时，您可以在该银行获得最高 250,000 美元的 FDIC 保险。通过使用 IntraFi 网络存款，您可以通过单一银行关系获得数百万美元的 FDIC 保护，因为较大的存款分为标准 FDIC 保险最高 250,000 美元的金额，并存入其他网络银行的存款账户，可通过您的主要银行。
