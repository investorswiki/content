# 价格变化率指标 (ROC)
## 什么是价格变化率 (ROC) 指标

价格变化率 (ROC) 是一种基于动量的[技术指标](/technicalindicator)，用于衡量当前价格与一定时期前价格之间的价格百分比变化。 ROC 指标针对零绘制，如果价格变化向上，则指标向上移动到正区域，如果价格变化向下，则指标移动到负区域。

该指标可用于发现[背离、超买和](/divergence)[超卖](/divergence)情况以及中心线交叉。

## 如何计算价格变化率指标

计算 ROC 的主要步骤是选择“n”值。短线交易者可能会选择较小的 n 值，例如 9。长期投资者可以选择一个值，例如 200。n 值是当前价格与多少个时期前的比较。较小的值将使 ROC 对价格变化的反应更快，但这也可能意味着更多的[错误信号](/fakeout)。较大的值意味着 ROC 反应较慢，但信号在出现时可能更有意义。

1. 选择一个 n 值。它可以是 12、25 或 200 之类的任何值。短期交易者通常使用较小的数字，而长期投资者则使用较大的数字。

1. 查找最近一期的[收盘](/closingprice)[价](/closingprice)。

1. 求 n 个周期前该周期的收盘价。

1. 将第二步和第三步的价格代入 ROC 公式。

1. 每个周期结束时，计算新的 ROC 值。

与大多数动量振荡器一样，ROC 出现在价格图表下方单独窗口中的图表上。 ROC 是针对区分正值和负值的零线绘制的。正值表示向上的购买压力或动力，而低于零的负值表示卖压或向下的动力。在任一方向（正或负）增加值表示动量增加，向零移动表示动量减弱。

零线[交叉](/crossover)可用于指示趋势变化。根据所使用的 n 值，这些信号可能在趋势变化的早期出现（小 n 值）或在趋势变化的很晚出现（较大的 n 值）。中华民国很容易出现[洗盘](/whipsaw)，尤其是在零线附近。因此，此信号通常不用于交易目的，而只是提醒交易者可能正在发生趋势变化。

还使用了超买和超卖水平。这些水平不是固定的，但会因交易的资产而异。交易者希望了解过去导致价格反转的 ROC 值是什么。通常交易者会发现价格有规律地反转的正值和负值。当 ROC 再次达到这些极端读数时，交易者将保持高度警惕并观察价格开始反转以确认 ROC 信号。随着 ROC 信号到位，**和**价格反转以确认 ROC 信号，可以考虑进行交易。

ROC 也通常用作指示可能即将发生的趋势变化的背离指标。[当](/divergence)股票或另一种资产的价格向一个方向移动而其 ROC 向相反方向移动时，就会出现背离。例如，如果股票价格在一段时间内上涨，而 ROC 逐渐走低，则 ROC 表示与价格的看跌背离，这表明趋势可能转向下行。如果价格下跌而 ROC 上涨，则同样的概念也适用。这可能预示着价格上涨。背离是一个出了名的糟糕的[时机](/markettiming)信号，因为背离可以持续很长时间并且不会总是导致价格反转。

## 价格变化率与动量指标的区别

这两个指标非常相似，如果在每个指标中使用相同的 n 值，将产生相似的结果。主要区别在于 ROC 将当前价格与 n 个周期前的价格之间的差值除以 n 个周期前的价格。这使它成为一个百分比。大多数动量指标的计算不这样做。相反，价格差异只需乘以 100，或者当前价格除以 n 个周期前的价格，然后乘以 100。这两个指标最终都讲述了类似的故事，尽管一些交易者可能略微偏爱其中一个，因为他们可以提供略有不同的读数。

## 使用价格变化率指标的局限性

使用 ROC 指标的一个潜在问题是，尽管一些技术分析师认为最近的价格走势对确定可能的未来价格更为重要，但它的计算对最近价格和 n 个时期前的价格给予同等权重移动。

该指标也容易出现洗盘，尤其是在零线附近。这是因为当价格盘整时，价格变化收缩，将指标移向零。这样的时间可能会导致[趋势交易出现多个错误信号](/trendtrading)，但确实有助于确认价格盘整。

虽然指标可用于发散信号，但信号通常出现得太早。当 ROC 开始背离时，价格仍然可以在趋势方向运行一段时间。因此，背离不应被视为交易信号，但如果其他指标和分析方法出现其他反转信号，则可用于帮助确认交易。

＃＃ 强调

- 价格变化率 (ROC) 振荡器是一种无界动量指标，用于技术分析，以零水平中点为基准。

- 价格盘整时，ROC 将徘徊在零附近。在这种情况下，交易者关注整体价格趋势很重要，因为除了确认盘整之外，中华民国将提供很少的洞察力。

- ROC 上升至高于零通常确认上升趋势，而 ROC 下降至低于零则表明下降趋势。
