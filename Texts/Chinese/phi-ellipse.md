# 菲椭圆
## 什么是 Phi 椭圆？

phi-椭圆是一种用于技术分析的投资工具，也称为斐波那契椭圆。 phi-椭圆得名于这样一个事实，即[斐波那契](/fibonaccilines)数列是 phi 数学概念的自然导数。

一般来说，phi椭圆用于检测[价格模式](/pattern)。这些模式可以帮助一些交易者分析市场并确定何时买入或卖出证券。这些模式可用于所有市场，例如股票、外汇或期货。

## Phi-Ellipse 告诉你什么？

与试图确定阻力位和支撑位的标准斐波那契技术工具相比，交易者使用 phi 椭圆来识别一般市场趋势。一旦在图表上确定了三个点，就可以绘制 phi 椭圆。 phi-椭圆是一个三波价格点，可以帮助说明主要趋势。

要绘制 phi 椭圆，必须首先选择三个点：第一个点只是一个任意点（但可能是价格[摆动低点](/swinglow)），第二个点是出现在该任意点之后的峰值，并通过反转来识别从高峰。第三点是趋势的另一个反转回到原来的方向。第三点也是最后一点也证实了市场趋势。绘制椭圆后，平分该椭圆的线用于指示趋势。

斐波那契交易工具用于帮助投资者识别价格走势。它的主要用途之一是通过分析椭圆的变化形状来识别价格变动的基本结构。由于其复杂性，phi-椭圆通常由计算机程序绘制。

## 如何使用 Phi-Ellipse 的示例

使用 phi 椭圆或其他斐波那契交易工具，包括 phi 螺旋线，可以帮助说明肉眼更难看到的价格模式，以指导交易决策。

为了解释 phi 椭圆的结果，投资者将查看椭圆的角度。价格在 phi 椭圆之外移动表明潜在的趋势变化。线在斐椭圆底部/顶部平分的地方可能是一个反转点。

同一方向的多个 phi 椭圆通常可以包含在一个更大的 phi 椭圆中，这将为整个趋势提供趋势和反转信息。

phi-椭圆在大多数交易和图表平台上并不常见，因为它不像其他工具那样受欢迎，例如[斐波那契回撤](/fibonacciretracement)或[斐波那契扩展](/fibonacciextensions)。

## Phi 椭圆和斐波那契弧之间的区别

[斐波那契弧线](/fibonacciarc)连接高点和低点。弧线从高点和低点向右延伸，表示价格未来可能达到的潜在支撑（上升趋势）或潜在阻力（下降趋势）水平。这些水平可能会提供交易机会。

## 使用 Phi-Ellipse 的限制

phi-椭圆不是一个常见的指标，因此在大多数交易软件上都很难找到。大多数图表平台中可用的椭圆绘图工具不一定与斐波那契比率一致。

phi-椭圆根据最近的波动高点和低点确定趋势方向。这些可以用肉眼看到，因此该指标在确定当前趋势方向方面几乎没有增加。它可能会提供潜在的反转点，类似于断裂的趋势线，但预测并不总是准确的。价格可能会移出 phi 椭圆，但随后会横向移动或继续沿原始趋势方向移动。

该指标最好与[价格行为分析](/price-action)和其他技术分析工具结合使用，以帮助确认 phi 椭圆信号和分析。

＃＃ 强调

- 该指标不常见，可能并非在所有图表平台上都可用。

- 椭圆绘图工具与 phi 椭圆不同，因为它可能与斐波那契比率不一致。

- Phi-ellipse 是一种基于斐波那契比率的三点技术分析工具，连接摆动高点和摆动低点，以确定趋势方向和可能的反转点。
