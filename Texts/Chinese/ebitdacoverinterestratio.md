# EBITDA-利息覆盖率
## 什么是 EBITDA 与利息覆盖率？

EBITDA 与利息覆盖率是一种财务比率，用于评估公司的财务耐久性，方法是检查公司是否至少有足够的利润来使用税前收入支付利息费用。具体来说，它旨在查看可用于此目的的[利息、税项、折旧和摊销前利润(EBITDA) 的比例。](/ebitda)

EBITDA 与利息的覆盖率也简称为 EBITDA 覆盖率。 EBITDA 覆盖率和[利息覆盖率](/interestcoverageratio)之间的主要区别在于，后者使用[收入和](/ebit)税前利润 (EBIT)，而不是更广泛的 EBITDA。

> - EBITDA 与利息覆盖率或 EBITDA 覆盖率用于查看公司支付未偿债务利息的难易程度。

> - 该公式将利息、税项、折旧和摊销前的收益除以总利息支付，使其比标准利息覆盖率更具包容性。

> - 覆盖率越高越好，但理想的覆盖率可能因行业而异。

## 了解 EBITDA 与利息的覆盖率

EBITDA 与利息覆盖率首先被[杠杆收购](/leveragedbuyout)银行家广泛使用，他们将其用作确定新重组公司是否能够偿还其[短期债务的第一个筛选条件](/shorttermdebt)。比率大于 1 表示公司有足够的利息覆盖率来支付利息费用。

虽然该比率是评估公司是否能够支付其利息相关费用的一种非常简单的方法，但该比率的应用也受到使用 EBITDA（息税折旧和摊销前利润）作为代表的相关性的限制。各种财务数据。例如，假设一家公司的 EBITDA 对利息的覆盖率为 1.25；这可能并不意味着它能够支付利息，因为该公司可能需要将大部分利润用于更换旧设备。由于 EBITDA 不考虑折旧相关费用，因此 1.25 的比率可能不是财务持久性的明确指标。

## EBITDA-利息覆盖率计算和示例

用于 EBITDA 与利息覆盖率的两个公式略有不同。根据被分析的公司的不同，分析师可能对哪一种更适用有不同的看法。它们如下：

>

> EBITDA-利息覆盖率 = (EBITDA + 租赁付款) / (贷款利息付款 + 租赁付款)

>

和

>

> 利息覆盖率，即息税前利润/利息费用。

>

例如，考虑以下内容。一家公司报告的销售收入为 1,000,000 美元。工资支出报告为 250,000 美元，而水电费报告为 20,000 美元。租金为 100,000 美元。该公司还报告了 50,000 美元的[折旧和 120,000 美元的](/depreciation)[利息支出](/interestexpense)。要计算 EBITDA 与利息的覆盖率，分析师首先需要计算 EBITDA。 EBITDA 的计算方法是将公司的[EBIT](/ebit) （息税前利润）加回折旧和摊销金额。

在上述示例中，公司的 EBIT 和 EBITDA 计算如下：

- 息税前利润 = 收入 - 运营费用 - 折旧 = 1,000,000 美元 - (250,000 美元 + 20,000 美元 + 100,000 美元) - 50,000 美元 = 580,000 美元

- EBITDA = EBIT + 折旧 + 摊销 = $580,000 + $50,000 + $0 = $630,000

接下来，使用包含租赁付款期限的 EBITDA 对利息覆盖率的公式，公司的 EBITDA 对利息覆盖率是：

- EBITDA 对利息的覆盖率 = ($630,000 + $100,000) / ($120,000 + $100,000)

- = 730,000 美元 / 220,000 美元

- = 3.32
