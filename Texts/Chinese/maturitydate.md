# 成人礼
##什么是到期日？

到期日是指存款证明（CD）或债券等投资到期并偿还给投资者的日期。此时，投资停止支付利息，投资者可以赎回累积的利息和他们的资本而不会受到惩罚。

## 更深层次的定义

存款证 (CD) 是银行用来筹集资金的债务工具。 CD 的主要特点是：

- **到期日** — CD 有一个固定的到期日，从一个月到五年不等。

- **利率** - 利息按购买之日固定的年利率计算。

- **利息支付** — 根据 CD 的不同，利息可能会累积并在到期日支付，也可以每月、每季度或每年定期支付。

- **提前取款** — 可以在到期日之前提取资金，但在大多数情况下，提前取款会受到处罚。

CD 被认为是一种安全的投资，是在一段时间内储存资金并赚取利息的理想方式。在 CD 上赚取的利息高于您在储蓄账户上赚取的利息。 CD 的期限越长，您从中获得的利息就越高。

当一张 CD 即将到期时，您的银行会通知您，并为您提供赎回或将其转入另一张 CD 的选项。

期限到期日也适用于公司债券和国债。它们的操作方式与 CD 类似，但有一个重要例外。你不能提早取出你的钱。你必须等待债券成熟。

## 到期日示例

查尔斯以 2.25% 的利率向当地银行投资了 10,000 美元购买了一张为期 5 年的 CD。 CD 将在几个月后到期，因为他可能在今年晚些时候需要这笔钱，所以他通知银行他想将 CD 转换为 6 个月的 CD。

＃＃ 强调

- 到期日是指固定收益工具的本金必须偿还给投资者的时间。

- 一旦到期日，由于债务协议不再存在，定期支付给投资者的利息将停止。

- 到期日同样指借款人必须全额偿还分期贷款的到期日。

- 到期日用于将债券分为三大类：短期（1 至 3 年）、中期（10 年或更长）和长期（通常为 30 年期国债）。
