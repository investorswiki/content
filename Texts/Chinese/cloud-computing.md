# 云计算
## 什么是云计算？

云计算是通过互联网提供不同的服务。这些资源包括工具和应用程序，如数据存储、服务器、数据库、网络和软件。

[基于云的存储](/cloud-storage)不是将文件保存在专有硬盘驱动器或本地存储设备上，而是可以将它们保存到远程数据库中。只要电子设备可以访问网络，它就可以访问数据和运行它的软件程序。

云计算是个人和企业的一种流行选择，原因有很多，包括节省成本、提高生产力、速度和效率、性能和安全性。

## 了解云计算

云计算之所以如此命名，是因为正在访问的信息是在云或虚拟空间中远程找到的。提供云服务的公司使用户能够将文件和应用程序存储在远程服务器上，然后通过 Internet 访问所有数据。这意味着用户无需在特定位置即可访问它，从而允许用户远程工作。

云计算将处理和处理数据所涉及的所有繁重工作从您随身携带或坐着工作的设备中解放出来。它还将所有这些工作转移到远离网络空间的巨大计算机集群中。互联网变成了云，瞧——您的数据、工作和应用程序可通过您可以连接到互联网的任何设备在世界任何地方使用。

云计算既可以是公共的，也可以是私有的。公共云服务通过 Internet 提供服务是有偿的。另一方面，私有云服务只为一定数量的人提供服务。这些服务是提供托管服务的网络系统。还有一个混合选项，它结合了公共和私人服务的元素。

## 云服务的类型

无论是哪种服务，云计算服务都为用户提供了一系列功能，包括：

- 电子邮件

- 存储、备份和数据检索

- 创建和测试应用程序

- 分析数据

- 音频和视频流

- 按需交付软件

云计算仍然是一项相当新的服务，但正在被许多不同的组织使用，从大公司到小企业，从非营利组织到政府机构，甚至是个人消费者。

## 部署模型

有各种类型的云，每一种都不同。公共云在互联网上的服务器和存储上提供服务。这些由处理和控制所有硬件、软件和通用基础设施的第三方公司运营。客户通过几乎任何人都可以访问的帐户访问服务。

私有云是为特定客户保留的，通常是一个企业或组织。公司的数据服务中心可以托管云计算服务。许多私有云计算服务是在私有网络上提供的。

顾名思义，混合云是公共服务和私有服务的组合。这种类型的模型为用户提供了更大的灵活性，并有助于优化用户的基础设施和安全性。

> 较新的云计算服务形式包括社区云、大数据云和多云。

## 云计算的类型

云计算不是像微芯片或手机那样的单一技术。相反，它是一个主要由三种服务组成的系统：[软件即服务](/software-as-a-service-saas)(SaaS)、基础设施即服务 (IaaS) 和平台即服务 (PaaS)。

1. **软件即服务 (SaaS)** 涉及向客户授予软件应用程序许可。许可证通常通过即用即付模式或按需提供。这种类型的系统可以在 Microsoft Office 的 365 中找到。

1. **基础设施即服务 (IaaS)** 涉及一种通过基于 IP 的连接作为按需服务的一部分交付从操作系统到服务器和存储的一切的方法。客户可以避免购买软件或服务器，而是通过[外包](/outsourcing)的按需服务采购这些资源。 IaaS 系统的流行示例包括 IBM Cloud 和 Microsoft Azure。

1. **平台即服务 (PaaS)** 被认为是三层云计算中最复杂的。 PaaS 与 SaaS 有一些相似之处，主要区别在于它不是在线交付软件，它实际上是一个用于创建通过 Internet 交付的软件的平台。该模型包括 Salesforce.com 和 Heroku 等平台。

##云计算的优势

基于云的软件为各行各业的公司提供了许多好处，包括能够通过本机应用程序或浏览器从任何设备使用软件。因此，用户可以以完全无缝的方式将他们的文件和设置转移到其他设备上。

云计算不仅仅是访问多个设备上的文件。借助云计算服务，用户可以在任何计算机上查看电子邮件，甚至可以使用 Dropbox 和 Google Drive 等服务存储文件。云计算服务还使用户能够[备份](/back-up)他们的音乐、文件和照片，确保这些文件在硬盘驱动器崩溃时立即可用。

它还为大型企业提供了巨大的成本节约潜力。在云成为可行的替代方案之前，公司需要购买、构建和维护成本高昂[的](/information-management-technology-imt)[编队管理技术](/information-management-technology-imt)和基础设施。公司可以将昂贵的服务器中心和 IT 部门换成快速的 Internet 连接，让员工与云在线交互以完成他们的任务。

云结构允许个人节省台式机或笔记本电脑上的存储空间。它还可以让用户更快地[升级软件，因为软件公司可以通过网络提供他们的产品，而不是通过涉及光盘或闪存驱动器的更传统的、有形的方法。](/upgrade)例如，Adobe 客户可以通过基于 Internet 的订阅访问其 Creative Cloud 中的应用程序。这允许用户轻松下载新版本和修复程序。

##云的缺点

随着云计算带来的所有速度、效率和创新，自然存在风险。

安全性一直是云计算的一个大问题，尤其是在涉及敏感医疗记录和财务信息时。虽然法规强制云计算服务支持其安全和合规措施，但它仍然是一个持续存在的问题。加密保护重要信息，但如果该加密密钥丢失，数据就会消失。

云计算公司维护的服务器也可能成为自然灾害、内部错误和断电的受害者。云计算的地理范围是双向的：加利福尼亚州的停电可能会使纽约的用户瘫痪，而德克萨斯州的一家公司可能会丢失其数据，如果某些事情导致其位于缅因州的供应商崩溃。

与任何技术一样，员工和经理都有学习曲线。但是，由于许多人通过单个门户访问和操作信息，无意中的错误可能会在整个系统中传播。

##商业世界

企业可以以不同的方式使用云计算。一些用户在云上维护所有应用程序和数据，而另一些用户则使用混合模型，将某些应用程序和数据保存在私有服务器上，而另一些则保存在云上。

在提供服务方面，企业计算领域的主要参与者包括：

- 谷歌云

- 亚马逊网络服务 (AWS)

- 微软天青

- IBM 云

- 阿里云

Amazon Web Services 是 100% 公开的，包括现收现付的外包模式。进入平台后，您可以注册应用程序和其他服务。 Microsoft Azure 允许客户在他们自己的站点上保存一些数据。同时，阿里云是阿里巴巴集团的子公司。

- 云存储在需要更大存储空间的个人和寻求高效异地数据备份解决方案的企业中越来越受欢迎。

＃＃ 常问问题

### 什么是云计算的例子？

今天，有几个企业和个人都使用的云计算应用程序示例。一种类型的云服务是音频或视频的流媒体平台，实际的媒体文件远程存储在其中。另一个是数据存储平台，如 Google Drive、Dropbox、OneDrive 或 Box。

### 云计算安全吗？

由于软件和数据远程存储在云计算中，数据安全和平台安全是一个大问题。[云安全](/cloud-security)是指为保护存储在基于云的服务上的数字资产和数据而采取的措施。保护这些数据的措施包括双因素授权 (2FA)、VPN 的使用、安全令牌、数据加密和防火墙服务等。

### 云计算的主要类型有哪些？

云计算服务的主要类型包括基础设施即服务 (IaaS)、平台即服务 (PaaS) 和软件即服务 (SaaS)。用户通过互联网，通常与无服务器计算相关联。- PaaS 为最终用户提供软件和硬件，这些最终用户通常是软件开发人员。 PaaS 允许用户开发、运行和管理他们自己的应用程序，而无需构建和维护基础设施。- SaaS 是一种软件许可模式，它允许使用外部服务器以订阅方式访问软件，而无需下载和安装它们本地。
