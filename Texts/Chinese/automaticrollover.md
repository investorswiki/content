# 自动翻转
## 什么是自动翻转？

1. 自动展期是将合格的退休计划分配转入[个人退休账户](/ira)(IRA)，账户持有人无需采取任何行动。当公司在离开公司后从公司赞助的退休计划中删除少量余额的员工时，就会发生这种情况。余额较大的员工可以选择留在计划中。

2. 自动展期也指在存款证（CD）到期时将利息和[本金再投资，无需账户持有人采取任何行动。](/principal)当 CD 到期时，证书持有者有一定天数将收益转移到另一个账户。如果他们什么都不做，金融机构会自动将收益再投资到与原始 CD 具有相同期限的新 CD 中。

## 了解自动翻转

1. 自动展期是[安全港](/safeharbor)规则的一部分，该规则要求公司向受影响的员工提供必要的披露信息、如何再投资的说明，并提前 60 天通知他们将从退休计划中删除。一旦此通知期过去，员工的资金将进入另一个称为安全港 IRA 的投资工具，该工具投资于货币市场基金或其他低风险投资。如果计划持有人希望发生不同的事情，其他选择包括现金分配或转存到特定的退休账户。作为 2001 年经济增长和税收减免和解法案的一部分，安全港 IRA 规则于 2005 年生效。

2. CD 的自动展期，也称为“自动续订”，几乎总是以与原始投资相同的期限再投资 CD。但是，利率通常会有所不同，具体取决于当前[收益率](/yield)。

## 自动翻转的优缺点

1. 自动展期帮助公司从 401(k)s 和其他退休计划中删除小额账户。拥有太多小账户既是管理负担，也是额外费用。移除大量小账户有助于降低计划中其他人的成本。不采取行动的员工的缺点是，如果他们多年来自动留在低收益投资中，他们的退休储蓄将跟不上通货膨胀。

2. 自动续订可以简化CD持有者的再投资流程。考虑一个持有一个月 CD 的投资者。如果没有自动续订，该投资者需要每月购买一张新 CD，才能继续投资。然而，长期 CD 持有者的一个缺点是，这类投资者可能更愿意将资金投入其他领域，如果他们在自动展期开始前的几天内不采取行动，他们将面临现金处罚早点出新CD。此外，自动续订有时会使投资者以不利的利率购买收益率较低的 CD。
