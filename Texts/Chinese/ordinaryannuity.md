# 普通年金
##什么是普通年金？

普通年金是在固定时间段内连续期间结束时支付的一系列等额付款。虽然普通年金的支付频率可以每周支付一次，但实际上它们通常是每月、每季度、每半年或每年支付一次。与普通年金相反的是到期年金，其中在每个时期的开始支付。这两个系列的支付与被称为年金的金融产品不同，尽管它们是相关的。

## 普通年金如何运作

普通年金的例子是债券的利息支付，通常每半年支付一次，以及多年来保持稳定支付水平的股票的季度股息。普通年金的[现值](/presentvalue)很大程度上取决于现行利率。

由于货币的[时间价值](/timevalueofmoney)，利率上升会降低普通年金的现值，而利率下降会增加其现值。这是因为年金的价值是基于您的钱可以在其他地方获得的回报。如果您可以在其他地方获得更高的利率，那么相关年金的价值就会下降。

## 普通年金示例的现值

普通年金的现值公式考虑了三个变量。它们如下：

- PMT = 期间现金支付

- r = 每期利率

- n = 总周期数

给定这些变量，普通年金的现值为：

- **现值** = PMT x ((1 - (1 + r) ^ -n ) / r)

例如，如果普通年金每年支付 50,000 美元，为期 5 年，利率为 7%，则现值为：

- **现值** = $50,000 x ((1 - (1 + 0.07) ^ -5) / 0.07) = $205,010

> 在其他条件相同的情况下，普通年金的现值将低于到期年金。

>

## 年金到期示例的现值

回想一下，对于普通年金，投资者在时间段结束时收到付款。这与[到期年金](/annuitydue)形成对比，在该[年金](/annuitydue)中，投资者在期初收到付款。一个常见的例子是租金，租客通常会提前一个月向房东支付租金。付款时间的这种差异会影响年金的价值。年金到期计算公式如下：

- **到期年金的现值** = PMT + PMT x ((1 - (1 + r) ^ -(n-1) / r)

如果上述示例中的年金是到期年金，则其现值将计算为：

- **到期年金的现值** = $50,000 + $50,000 x ((1 - (1 + 0.07) ^ -(5-1) / 0.07) = $219,360。

在其他条件相同的情况下，到期的年金总是比普通年金更有价值，因为钱是更早收到的。

＃＃ 强调

- 普通年金是在每个期末支付的一系列定期付款，例如每月或每季度。

- 一致的季度股票股息是普通年金的一个例子；月租是应付年金的一个例子。

- 相反，在到期年金中，付款是在每个期初进行的。
