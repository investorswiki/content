# 零息存单 (CD)
## 什么是零息存单 (CD)？

零息存单 (CD) 是一种[在](/certificateofdeposit)期限内不支付利息的存单。相反，零息 CD 通过以低于其[面值的价格出售来提供回报](/facevalue)。这意味着一旦 CD[到期](/maturitydate)，投资者将获得比初始投资更多的收益。即使在到期日之前没有支付利息，这也为投资者提供了[投资回报 (ROI) 。](/returnoninvestment)

相比之下，传统的 CD 在整个期限内定期支付利息，通常是每年一次。零息 CD 和常规 CD 都是规避风险的投资者的热门选择，因为它们提供有保证的本金保护。然而，零息存单可能对那些在投资期间并不特别关心产生现金流的投资者特别有吸引力。

## 零优惠券 CD 的工作原理

零息 CD 是以大幅折扣出售的 CD，但在到期时支付全部面值。例如，一张面值为 100 美元的零息 CD 可能仅售 90 美元，这意味着投资者在期限结束时将获得 10 美元的利润。 “零息”一词源于这些投资没有每年支付利息的事实，也称为“息[票](/coupon)”。

零息 CD 被视为低风险投资。只要他们在期限结束前不提取资金，投资者就可以在预定的时间段内获得特定的回报。而且由于零息 CD 通常由银行发行，这意味着只要发行 CD 的银行由 FDIC 投保，它们就会得到[联邦存款保险公司 (FDIC)的支持。](/fdic)

零息 CD 的主要优势在于，与传统 CD 相比，它们往往提供略高的回报。但是，它们也有其缺点。例如，即使零息 CD 每年不支付利息，但每年赚取的[应计利息](/accruedinterest)仍被视为应纳税收入，即使这些资金直到期限结束时才真正收到。这意味着投资者必须提前计划以确保他们有足够的资金来支付这些税款。除了待遇之外，零息 CD 的另一个潜在重大缺点是它们可以被构造为可赎回投资。这意味着它们可以在到期前由开证行收回，然后以当前较低的利率重新发行。当然，零息存单不提供年度利息支付，这对于以现金流为导向的投资者来说可能不方便。

## 零优惠券 CD 的真实示例

为了说明这一点，考虑一张面值为 5,000 美元的 5 年期零息 CD 以 4,000 美元的价格出售的情况。要购买 CD，投资者只需支付 4,000 美元。在 5 年结束时，他们将收到全部 5,000 美元。但是，与此同时，该工具不会支付任何利息。

在这种情况下，从投资中获得的 1,000 美元利润相当于 5 年每年 200 美元的应计收入。从投资者的角度来看，这相当于 5% 的年利率，但需要注意的是，这些资金实际上要到第 5 年年底才能收到。此外，由于应计利息被视为应税收入，因此投资者将需要确保他们有足够的资金来支付到期日前几年的税收费用。

考虑到所有这些因素，投资者可能会认为这是一项有吸引力的投资，因为另类固定收益投资的收益率低于 5%，并且如果投资者在 5 年期限内不需要定期现金流.

＃＃ 强调

- 零息 CD 是一种在整个期限内不支付利息的 CD。

- 零息存单通常比传统存单提供更高的回报，以补偿投资者缺乏利息收入。

- 相反，投资者通过在到期时收到高于工具购买价格的面值来获得补偿。
