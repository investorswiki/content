# 基地我
## 什么是基础 I

Base I 是第一个用于信用卡支付的电子授权系统。它由美国银行于 1973 年开发。

## 破坏基地 I

Base I 于 1973 年首次开发，作为信用卡交易的电子实时授权系统。它由美国银行（BankAmericard 的发行商）开发，作为 VisaNet 系统的一部分。 Base 是 Bank America System Engineering 的首字母缩写词。今天，BankAmericard 作为 Visa 卡销售，而 Base I 是 VisaNet 系统两个阶段中的第一个。第二阶段称为 Base II。

在 Base I 系统开发之前，信用卡处理随着国家信用卡系统的发展而发展。第一张卡是闭环系统，原生于特定零售商或与特定银行有业务联系的本地化商家群体。在最早的案例中，交易是通过商家打给当地银行的电话记录下来的，当地银行为持卡人的月结单编制记录。

卡系统在 1950 年代稳步增长，美国银行的 BankAmericard 在 1960 年代初期主导了加利福尼亚市场。 1966 年，随着银行同业卡协会的成立，允许在广阔地理区域内相互竞争的银行之间进行交易的开环系统首次出现。这个银行联盟将很快采用万事达卡品牌，并激励美国银行成立自己的银行卡1970 年，NBI 的竞争对手网络。大约在这个时候，第三方公司出现了，以支持向无纸化交易处理迈进。曾经这样的公司是 VisaNet。 NBI 于 1973 年收购了 VisaNet，并推出了 Visa 卡以与 MasterCard 竞争。 1970 年代初的诉讼允许成员银行加入这两个网络。

## Base I 和 VisaNet 系统如何处理交易

Base I 系统的开发大致与 1970 年代中期 Visa 卡的推出相吻合。 Base I 是指实时授权系统，商家通过该系统将交易批准请求发送到银行。该请求将包括卡号和美元金额。作为回应，银行将发送简单的批准消息或拒绝消息并附上解释。

Base II 提供了一个结算流程来处理 Base I 系统生成的交易的日终对账。 Base II 是一个批处理系统 - 与 Base I 的实时活动不同，结算会定期进行，并向商户收取结算费用。

随着 NBI 和 VisaNet 推出并更新其系统，万事达卡也推出了类似的两部分处理平台，包括一个称为 INAS 的系统来处理交易和[INET](/interbank-network-for-electronic-transfer-inet)来结算和清算余额。
