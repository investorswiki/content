# 参考资产
## 什么是参考资产？

参考资产是[信用衍生品](/creditderivative)中用于保护债务持有人免受潜在风险借款人影响的基础资产。参考资产也称为参考实体、参考义务或涵盖义务。参考资产可以是资产，如债券、票据或其他债务支持证券。

## 参考资产的工作原理

参考资产是一种债务支持的证券。当一个实体发行债务或借钱时，它总是有可能无法偿还资金，这被称为[违约风险](/defaultrisk)。债务持有人因借款人违约的可能性而固有地面临风险。为了对冲这种违约风险，债务持有人可以签订信用衍生品，例如[总回报](/totalreturn)或[信用违约掉期](/creditdefaultswap)（CDS）。这些信用衍生工具将风险转移给第三方以应对违约风险。

信用违约掉期 (CDS) 是最常用的信用衍生品类型，它允许债务持有人将他们面临的风险转移给第三方，该第三方通常是另一个贷方。使用这种方法，他们可以在不出售资产本身的情况下分配风险。债务持有人将向第三方支付一次性或持续费用，称为溢价。如果借款人拖欠债务，债务持有人有权获得一部分参考资产。

## 参考资产示例

### 信用违约掉期 (CDS)

信用违约互换 (CDS) 中的参考资产，也可以称为信用衍生品合约，通常包括[市政债券](/municipalbond)、新兴市场债券、[抵押贷款支持证券](/mbs)(MBS) 或[公司债券等资产](/corporatebond)，由借款人或参考实体。

例如，假设银行 A 投资于 X 公司的债券，尽管 X 公司享有风险借款人的声誉。为了保护自己免受 X 公司违约债券的风险，银行 A 决定与银行 B 进行信用违约掉期 (CDS)。根据 CDS，银行 A 将向银行 B 支付溢价以承担一些风险。但是，A 银行仍然正式拥有 X 公司债券。如果 X 公司违约，银行 A 将从 B 银行收到部分或全部原始债券（参考资产）的价值。如果 X 公司最终没有拖欠贷款，银行 B从银行 A 支付的溢价中获得的利润以换取其承担的风险。

＃＃ 强调

- 当一个实体发行债务或借钱时，它总是有可能不会偿还资金。为了防范这种违约风险，债务持有人可能会签订信用衍生品，例如总回报或信用违约掉期 (CDS)。

- 参考资产是信用衍生品中用于保护债务持有人免受潜在风险借款人影响的基础资产。

- 这些信用衍生工具将风险转移给第三方以应对违约风险。
