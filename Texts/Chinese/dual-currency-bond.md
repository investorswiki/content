# 双币债券
## 什么是双币债券？

双币债券是一种[以](/coupon)一种货币计价而[本金以另一种货币计价](/principal)的[债务工具](/debtinstrument)。这种债券会使持有人面临汇率风险。

双币债券可以与[双币掉期](/dual-currency-swap)和[双币存款相比较](/dualcurrencydeposit)。

## 了解双币债券

双币债券是一种合成证券，以一种货币赎回，而债券有效期内的利息支付则以另一种货币支付。例如，以美元 ( [USD](/usd) ) 发行并以日元 ( [JPY](/jpy-japanese-yen) ) 支付利息的债券被视为双币种债券。

的[货币](/currency)，称为基础货币，通常是支付利息的货币。发行债券时，本金和金额是固定的，也可以标明汇率。双币种债券的[票面利息](/coupon-rate)通常设定为高于可比较[的直接固定利率债券的利率](/fixedrate-bond)，并以较弱或较低利率的货币支付。

两种最常见的双币债券类型是：

1. **传统双币债券：**以投资者本币支付利息，本金以发行人本币计价。

1、**反向双币债券：**以发行人本币支付利息，本金以投资者本币计价。

双币债券发行最常由跨国公司和交易员在[欧元债券](/eurobond)[市场上发起](/eurobond)。双币债券的变种有[幕府债券](/shogunbond)、日元挂钩债券、多币种条款债券、外国付息债券和天堂与地狱债券。

## 双币债券示例

假设发行的债券[面值](/parvalue)为 1,000 美元，到期日为一年。利息以美元支付，到期本金偿还将以欧元支付。假设的[即期汇率](/spotexchangerate)[为](/spotexchangerate)EUR [/USD](/eur-usd-euro-us-dollar-currency-pair) 1.24。因此，每张债券的本金偿还价值设定为 (USD1000 x EUR1) / USD1.24 = EUR806.45。

那么，在第一年年底，该债券的现金流为 1,000 美元 + 806.45 欧元。如果美元市场的一年期市场利率为 4%，欧元市场为 7%，则应发行债券的利率为：

- 1000 = (1000r / 1.04) + 1.24 (806.45 / 1.07)

- 1000 = (1000r / 1.04) + 934.58

- 1040 = 1000r + 971.96

在哪里：

- r = 0.068，或 6.8%

## 特别注意事项

与票息和本金相关的汇率可以在[信托契约发行债券时指定](/trust_indenture)。发行人也可以决定在支付息票和本金时根据即期汇率进行支付。

双币种债券发行人通常会设定一个汇率，允许以更强势货币支付的款项升值更多。此外，指定的到期本金还款额允许强势货币的汇率有所升值。

双币债券须承受汇率风险。如果偿还本金的货币升值，债券持有人就会赚钱；如果它贬值，他们就会赔钱。投资者可以使用发行时固定汇率的[双币互换来抵消双币债券的汇率风险。](/dual-currency-swap)双币债券也用于直接对冲汇率风险，无需在[外汇市场进行任何交易](/foreign-exchange-markets)。

＃＃ 强调

- 双币债券是一种债务工具，其票面支付以一种货币计价，本金以另一种货币计价，可能使持有人面临汇率风险。

- 最常见的两种双币债券是传统双币债券和反向双币债券。

- 双币债券发行最常由跨国公司和交易员在欧元债券市场发起。
