# ABLE（实现更美好的生活体验）帐户
## 什么是 ABLE（实现更好的生活体验）帐户？

ABLE 账户是一个税收优惠的储蓄账户，可供 26 岁之前被诊断患有严重残疾的个人使用。受益人、朋友或家人可以向该账户供款，但年度供款总额不能超过一定限额，即与赠与税豁免挂钩。在 2020 年和 2021 年，此限额为 15,000 美元。到 2022 年，此限额将增加到 16,000 美元。

[州所得税](/state-income-tax)中扣除，但捐款本身并不打算用于减税。但是，帐户内的资金免税增长。

## ABLE 账户如何运作

ABLE 账户由 2014 年《实现更好的生活体验 (ABLE) 法案》创建，作为帮助残疾人维持公共福利的一种方式，例如[补充保障收入](/ssi)(SSI) 和医疗补助，这要求受益人的储蓄和其他资产低于特定水平保持资格。在大多数情况下，ABLE 账户中的资金不计入个人参加这些计划的资格。

ABLE 帐户的功能类似于[529 计划](/529plan)帐户。只要分配用于包括教育在内的合格残疾费用，ABLE账户中的资金就可以免税投资和增长；住房;运输;就业培训和支持；辅助技术及相关服务；个人支持服务；健康、财务管理和行政服务；法律费用； ABLE 账户监督和监控的费用；葬礼和埋葬；和基本生活费。

与 529 一样，ABLE 计划由各个州制定。四个州——爱达荷州、北达科他州、南达科他州和威斯康星州——没有有效的 ABLE 计划，但如果一个州接受外部居民加入其计划，无论您住在哪里，您都可以在那里开设 ABLE 账户。

> 当 ABLE 账户的受益人死亡时，该人居住的州可以对账户中剩余的部分或全部资金提出索赔，以补偿 Medicaid 费用。

>

## 税收改革如何影响 ABLE 账户

《减税和就业法案》对 ABLE 账户进行了几项关键变更，并于 2018 年生效。

如果受益人有收入，他们可以将该收入存入其 ABLE 账户，超过 15,000 美元的限额。额外供款仅限于以下较小者：受益人在纳税年度的补偿，或单人家庭的贫困线金额。 2021 年，美国大陆和华盛顿特区为 12,880 美元，阿拉斯加为 16,090 美元，夏威夷为 14,820 美元。

但是，如果他们的雇主代表他们向工作场所退休计划供款，则帐户受益人没有资格获得此额外供款。

同样由于税法的变化，ABLE 账户受益人现在有资格获得储蓄者信贷，旨在帮助中低收入工人。可以在 8880 表（合格退休储蓄供款信用）上从 ABLE 帐户的收入中获得高达 2,000 美元的额外供款。这种抵免额可以减少一个人所欠的税款或增加他们的退税额。

2017 年的税收改革还使得将部分资金从 529 大学储蓄账户转入 ABLE 账户成为可能。 ABLE 账户的受益人也必须是学费账户的指定受益人或符合条件的家庭成员。

## ABLE 账户与信托

在 ABLE 法案之前，许多家庭不得不使用[特殊需求信托](/special-needs-trust)将资产留给残疾家庭成员，而不会导致他们没有资格获得政府服务。但是，创建信托通常需要法律援助，这可能会很昂贵。

信托确实有一个主要优势：可以对其做出的贡献没有上限。相比之下，ABLE 帐户的每个计划都有状态限制。许多州将此限制设置为 500,000 美元以上，只有前 100,000 美元免于影响补充保障收入 (SSI) 的资格。

对于许多家庭来说，ABLE 帐户是他们可以用来帮助确保残疾儿童的财务未来的额外工具，而不是替代特殊需求信托。

＃＃ 强调

- 通过在 ABLE 账户中存钱，个人能够保持获得 SSI 和 Medicaid 等政府计划的资格。

- ABLE 帐户类似于 529 教育帐户，它们由各州管理，而不是由美国联邦政府管理。

- ABLE 账户是为在 26 岁生日前被诊断患有严重残疾的人提供的税收优惠储蓄账户。
