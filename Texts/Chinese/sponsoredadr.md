# 赞助的 ADR
## 什么是赞助 ADR？

保荐 ADR 是银行代表其股权作为[标的资产的外国公司发行的](/underlying-asset)[美国存托凭证 (ADR)](/adr) 。保荐的 ADR 在 ADR 和外国公司之间建立了法律关系，吸收了发行证券的成本。非赞助的[ADR](/unsponsoredadr)只能在场外交易市场 (OTC) 交易，而赞助的 ADR 可以在主要交易所上市。 ADR 是美国投资者将国际公司股票添加到其投资组合中的一种简单方式。

## 了解赞助的 ADR

外国公司利用美国存托凭证进入外国[资本市场](/capitalmarkets)。通常专注于国内上市公司的投资者有机会从增长更快的新兴市场（例如中国或印度的市场）中[获得](/emergingmarketeconomy)[回报](/emergingmarketeconomy)。尽管在美国上市，使用赞助 ADR 的公司仍将以其本国货币计价其收入和利润。

保荐存托凭证分为三个级别。一级赞助的 ADR 只能在[场外](/over-the-countermarket)交易 (OTC) 交易，不能在美国交易所上市。但是，这种类型的 ADR 更容易为外国公司设立，不需要相同的披露，也不需要公司遵守[公认会计原则](/gaap)(GAAP)。因此，这些证券的可用信息较少。

II 级保荐 ADR 可以在交易所上市，因此对更广泛的市场可见。然而，二级 ADR 要求公司遵守[美国证券交易委员会的规定](/sec)。 III 级赞助的 ADR 允许公司发行股票以筹集资金，但要求最高级别的合规和披露。

## 赞助的 ADR 和其他外国投资方式

外国投资可以带来可观的回报，但通常风险更高。通常，投资者通过购买外国股票来获得其投资组合中的外国股票的[敞口](/equity)。然而，获得敞口的另一种选择是通过[外国直接投资](/fdi)(FDI)。当公司将业务扩展到新兴经济体时，就会发生这种情况。外国直接投资可以采取在发展中国家开设新的特许经营权或地区总部的形式，并依靠当地和外籍员工的混合。

公司也可以开设[子公司](/subsidiary)或[联营公司](/associate-company)。这可能涉及收购现有外国公司的控股权或与外国公司合并或创建合资企业。

一般而言，公司在更[开放的经济体中从事 FDI，这些经济体](/open-market)提供熟练的劳动力和强劲的增长前景、更少的法规和更少的政治不稳定。 2018年，布鲁金斯学会发表了《在非洲的竞争：中国、欧盟和美国》，指出美国是非洲大陆最大的投资者，对外直接投资总额为540亿美元。

＃＃ 强调

- 银行代表以股权作为标的资产的外国公司发行保荐 ADR。

- 保荐 ADR 是 ADR 与外国公司之间的法律关系，外国公司负责发行证券的费用。

- 赞助的 ADR 在主要交易所上市，而非赞助的 ADR 只能在场外交易 (OTC) 市场交易。
