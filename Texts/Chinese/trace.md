# 贸易报告和合规引擎 (TRACE)
## 什么是贸易报告和合规引擎 (TRACE)？

交易报告和合规引擎 (TRACE) 是由[美国证券交易商协会](/nasd)(NASD)（现称为[FINRA ）开发的一个程序](/finra)，它允许报告与符合条件的[固定资产相关的](/fixed-incomesecurity)[场外](/otc)交易 (OTC) 交易[收入证券](/fixed-incomesecurity)。作为 FINRA 成员并处理特定固定收益证券的经纪人，必须按照美国证券交易委员会 (SEC) 的规则报告其交易。

## 了解贸易报告和合规引擎 (TRACE)

根据 FINRA 的说法，TRACE 计划的使用旨在为所有市场参与者建立一个公平的竞争环境。这是通过为他们提供全面的实时访问债券价格信息来实现的。及时发布交易数据旨在确保能够平等获取有关公司、机构和结构性产品的可靠债券信息。

从 1998 年到 2001 年，SEC 批准了 NASD 就所有美国公司债券和二级 OTC 固定收益交易中的交易采用的规则。这些规则旨在为债券市场带来更大的[价格透明度。](/pricetransparency)随后，TRACE 在 2002 年开始发挥作用，以遵守新批准的规则。该程序取代了自 1994 年以来使用的以前的固定收益定价系统 (FIPS)。

TRACE 目前由金融业监管局 (FINRA) 运营。该计划为个人投资者和市场专业人士提供了访问几乎所有场外公共和私人固定收益交易活动的信息。 TRACE 计划整合了公共和私人公司债券、机构债务和证券化产品的交易数据。这包括资产支持证券和抵押支持证券。

## TRACE 向市场和监管机构提供什么

FINRA 表示，TRACE 的服务进一步增强了市场的完整性。投资者可以访问此类实时数据，以便更好地了解和衡量其经纪交易的表现。此外，FINRA 声称 TRACE 协助监管机构监控市场、定价和执行质量。

通过 TRACE 可以提供不同级别的服务，有偿和无偿。对于交易信息的实时数据显示，个人、非商业访问是免费的。个人专业用户的访问权限或这些显示器的企业级许可证需要付费。通过主要市场数据供应商和一些金融网站访问显示的 TRACE 数据。

## 其他 TRACE 数据

通过 TRACE 提供的其他数据服务和馈送包括日终交易和活动报告、市场活动和绩效指标以及增强的历史数据。

例如，TRACE 的增强历史数据包括[交易级别的信息](/transaction)，例如交易价格、执行日期和时间、交易规模和收益。数据还包括买卖指标和交易对手信息等信息。这些细节以前不向公众提供。

- 由于场外交易市场通常比在交易所上市交易更缺乏流动性、更少监管和更不透明，因此 TRACE 提供了急需的信息，以提供更加透明和公平的市场。

- TRACE 由 FINRA（前身为 NASD）运营，于 2002 年推出，取代了之前的 FIPS 报告系统。

- 交易报告和合规引擎 (TRACE) 是一种用于报告场外债券市场交易和数据的工具。
