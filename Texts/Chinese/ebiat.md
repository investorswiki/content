# 税后利息前收益 (EBIAT)
## 什么是税后利息前利润 (EBIAT)？

税后利息前利润 (EBIAT) 是用于评估公司一个季度或一年的经营业绩的众多财务指标之一。

EBIAT 衡量公司的盈利能力而不考虑其[资本结构](/capitalstructure)，即反映在[债转股中的债务和股票发行的组合](/debtequityratio)。 EBIAT 是一种衡量公司在考虑税收时从其业务中产生收入的能力的方法。

## 了解 EBIAT

EBIAT 在[财务分析中的使用](/financial-analysis)不像其他指标那样普遍，特别是[利息、税收、折旧和摊销](/ebitda)(EBITDA)。它主要被监控为跟踪公司有多少现金可用于支付其债务义务的一种方式。如果公司没有太多的折旧或摊销，EBIAT 可能会受到更密切的关注。

EBIAT 将税收视为公司无法控制的持续费用，特别是在公司盈利的情况下。 EBIAT 的计算消除了可能从[债务融资中获得的任何税收优惠](/debtfinancing)。因此，该措施通过消除可能增强或降低其财务实力的因素，准确地反映了公司的财务状况。

## EBIAT 计算示例

EBIAT 的计算很简单。它是公司的 EBIT x (1 - 税率)。公司的息税前利润按以下方式计算：

EBIT = 收入 - 营业费用 + 营业外收入

例如，考虑以下内容。 X 公司报告该年度的销售收入为 1,000,000 美元。同期，该公司报告的非营业收入为 30,000 美元。该公司的商品销售成本为 200,000 美元，而折旧和摊销报告为 75,000 美元。销售、一般和管理费用为 150,000 美元，其他杂项费用为 20,000 美元。该公司还报告了一年的一次性特殊费用 50,000 美元。

在本例中，息税前利润将计算为：

息税前利润 = $1,000,000 - ($200,000 + $75,000 + $150,000 + $20,000 + $50,000) + $30,000 = $535,000

如果 X 公司的税率为 30%，则 EBIAT 的计算公式为：

EBIAT = EBIT x (1 - 税率) = $535,000 x (1 - 0.3) = $374,500

一些分析师会争辩说，特殊费用不应该包括在计算中，因为它不是经常性的。是否包括它由进行计算的分析师自行决定。

该决定可能取决于特殊费用的大小，但这些类型的项目可能会产生重大影响。在此示例中，如果从计算中排除一次性特殊费用，则会产生以下数字：

不计特殊费用的息税前利润 = 585,000 美元

不含特殊费用的 EBIAT = 409,500 美元

不包括特殊费用，X 公司的 EBIAT 高出 9.4%，这可能会对决策者产生影响。

＃＃ 强调

- 税后利息前利润 (EBIAT) 衡量公司在特定时期或一段时间内的经营业绩。

- EBIAT 显示公司在清算时有多少现金可用于支付其债权人。

- EBIAT 忽略了公司的资本结构作为一个因素。
