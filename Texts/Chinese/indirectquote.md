# 间接报价
## 什么是间接引用？

间接报价一词是[外汇市场](/foreign-exchange-markets)上的一种货币报价，表示买入或卖出一个单位本币所需的可变外币数量。间接报价也称为“数量报价”，因为它表示购买单位本币所需的外币数量。换句话说，本币是间接报价中的[基础货币](/basecurrency)，而外币是[相对货币](/counter-currency)。

## 理解间接引用

间接报价是直接报价的相反或倒数，[也](/directquote)称为“报价”，它以可变数量的本币单位表示一单位外币的价格。

由于美元 ( [USD](/usd-united-states-dollar) ) 是全球外汇市场的主要货币，因此惯例通常使用直接报价，以美元为基础货币和其他货币，如加元 ( [CAD](/cad-canadian-dollar) )、日元 ( [JPY )](/jpy-japanese-yen) ) 和印度卢比 ( [INR](/inr) ) — 作为对价货币。此规则的例外情况是[欧元](/euro)和英联邦货币，如英镑 (GBP)、澳元 ( [AUD](/aud-usd-australian-dollar-us-dollar-currency-pair) ) 和新西兰元 ( [NZD](/nzd-new-zealand-dollar) )，它们通常以间接形式报价（例如1[英镑](/gbp)= 1.30 美元）。

考虑加元的例子，我们假设它对美元的交易价格为 1.2500。在加拿大，此报价的间接形式为 1 加元 = 0.8000 美元（即 1/1.2500）。但是，[外汇市场的常规报价](/foreign-exchange-markets)是1.2500，这是从美国角度来看的间接报价，因为它表明需要多少外币（CAD）才能获得1美元。相反，0.8000 美元将是直接报价。

在间接报价中，较低的[汇率](/exchangerate)意味着本国货币正在贬值或变弱。继续上面的例子，如果美元/加元的报价现在变为 1 美元 = 1.2300 加元（间接报价），那么这意味着美元（本国货币）变得更弱，因为获得 1 美元需要更少的加元。直接报价为 0.8130 (1/1.2300)，表明 1 加元将为您带来 0.8130 美元，而不是 0.8000。
