# 无保险存款证明
## 什么是无保险存款证明？

未投保[的](/certificateofdeposit)[存款](/certificateofdeposit)证明是没有为损失投保的 CD。由于缺乏保险，这些 CD 产生更高的[利率](/interestrate)，因为购买者承担了所有[风险](/risk)。如果发行 CD 的金融机构或实体破产，则购买者将失去投资。

## 了解无保险 CD

大多数 CD 由[联邦存款保险公司 (FDIC)](/fdic)或在信用合作社的情况下由[国家信用合作社管理局 (NCUA)提供保险](/ncua)。如果贷款金融机构无力偿债，这些机构将向 CD 持有人支付一定的限额。但是，也有无保险的 CD，例如[离岸](/offshore)CD 和[代理 CD](/brokered-cd) 。

> 离岸 CD 将您的钱存入外国机构的银行证书中。诱惑是利率是您在美国进行类似投资所能获得的利率的倍数。然而，危险在于押注外国银行的安全，如果您的资金以该国货币而非美元存放，您将面临[货币风险](/currencyrisk)。

>

## 特别注意事项

FDIC 保险账户是符合 FDIC 所涵盖要求的[银行](/bank)或[储蓄机构（储蓄和贷款协会）账户。](/thriftbank)可以接受 FDIC 保险的账户类型包括[可协商的提款指令 (NOW)](/nowaccount) 、支票、储蓄、货币市场存款账户和 CD。合格账户中的最高保险金额为每位存款人、每位成员机构 250,000 美元。这意味着，如果您在银行账户中存有高达该金额而银行倒闭，那么 FDIC 将使您免于遭受任何损失。

其他类别的 CD 是由投资公司组合在一起的外来产品。寻求[收益的投资者](/yield)有时会在没有意识到它们不受政府担保的情况下购买这些产品。它们可能具有较高的预告利率、较长的锁定期、可变利率、与股票或债券市场等指数相关[的回报率](/rateofreturn)，甚至与没有公开披露价格的资产相关的可变利率。

一些经纪 CD 可能部分没有保险。其他形式的 CD 是公牛 CD、熊 CD 和[洋基 CD](/yankeecd) 。牛市 CD 的利率与其基础[市场指数的价值直接相关](/marketindex)。当某人投资于牛市 CD 时，他们会根据相关的市场指数获得最低回报率和额外的指定百分比。牛市 CD 持有者在 CD 的有效期内收到的利率随着市场指数价值的增加而增加。

＃＃ 强调

- CD 以及储蓄账户和货币市场账户是您可以在当地银行或信用合作社投资的储蓄工具。

- 未投保的存款证明是一种 CD，它没有受到联邦存款保险公司 (FDIC) 或国家信用合作社管理局 (NCUA) 的损失保险。

- 无保险 CD 的示例是洋基 CD、牛市 CD 和熊市 CD。

- 大多数 CD 由 FDIC 或 NCUA 投保。

- 未投保的 CD 通常具有较高的利率，因为 CD 的购买者承担了与它们相关的所有风险。

＃＃ 常问问题

### 投资无保险的存款证 (CD) 是否安全？

没有保险的存款证 (CD) 存在风险。投资者一直将他们的资金置于共同基金、年金、人寿保险单、股票和债券等未投保的期权中。每个人都必须决定较高的利率是否值得冒险。

### CD 是否由联邦存款保险公司 (FDIC) 投保？

大多数 CD 是通过银行或信用合作社提供的，这些银行期权由联邦存款保险公司 (FDIC) 提供高达 250,000 美元的保险。国家信用合作社管理局 (NCUA) 为信用合作社选项提供类似的保护。有一些无保险的选择，通常通过[经纪公司提供](/brokerage-company)。这些选项包括离岸 CD、牛市 CD、熊市 CD 和洋基 CD。

### 没有保险的 CD 有什么好处？

虽然没有保险的 CD 存在风险，但最大的好处是您可以赚更多的钱。随着时间的推移，更高的利率将带来更高的回报率。如果您对市场充满信心，那么一张没有保险的 CD 对您来说可能很有意义。
