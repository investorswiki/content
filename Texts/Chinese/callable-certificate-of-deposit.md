# 可赎回存款证 (CD)
## 什么是可赎回存款证 (CD)？

可赎回存款证 (CD) 是 FDIC 保险的[CD](/certificateofdeposit) ，包含与其他类型的[可赎回](/callable-security)固定收益证券类似的赎回[功能。](/callprovision)可赎回 CD 可由发行银行在其规定的到期日之前提前赎回（赎回），通常是在给定的时间范围内并以预设的赎回价格。这通常在利率下降时进行，允许发行银行停止向 CD 持有人支付高于现行利率的费用。

由于 CD 可能在到期前被赎回，这将导致利息收入的损失和[再投资风险](/reinvestmentrisk)，因此可赎回 CD 的利率通常高于普通 CD 的利率。

## 了解可调用 CD

可赎回 CD 有两个特点：存款证明和CD 发行人拥有的嵌入式[看涨期权。](/calloption)发行人通常会在利率下降时寻求收回 CD，因为这将阻止发行人支付高于现行市场利率的固定利率。然后银行可能会以较低的利率重新发行新的 CD。

CD 本质上是银行向投资者发行的[定期存款](/timedeposit)，投资者购买 CD 以赚取固定期限内的投资利息，该利息可能高于[活期存款支付的利息](/demanddeposit)。这些金融产品在到期之前支付利息，此时投资者可以使用资金。尽管仍然可以在到期日之前从 CD 中提取资金，但此操作通常会导致[提前](/withdrawal-penalty)[提取惩罚](/withdrawal-penalty)。 CD 通常比标准储蓄账户提供更高的回报率，因为资金的流动性较低，但也被认为是低风险投资，因为它通常由[联邦](/fdic)[存款保险公司 (FDIC)](/fdic)或[国家保险公司](/ncua)投保高达 250,000 美元[信用社管理局 (NCUA)](/ncua) 。

可赎回证券是一种可以由发行人提前赎回的证券，允许发行人对其有息证券进行再融资。银行在 CD 中添加了调用功能，因此如果利率下降，它就不必继续向 CD 持有人支付更高的利率。可赎回 CD 通常在提前赎回时向投资者支付赎回[溢价](/callpremium)，以激励投资者承担与投资相关的[赎回风险](/callrisk)。

## 特别注意事项

[赎回溢价](/callpremium)是为补偿投资者被赎回风险所需的高于 CD 面值的金额，[通常](/parvalue)随着 CD 接近到期日而降低。它通常被定价为 CD 对投资者的收益增加，并在向潜在投资者规定 CD 条款的披露声明中明确披露。

[赎回日期是银行可以收回其未偿CD](/calldate)的日期，它也包含在披露声明中。

在 CD 中增加赎回条款会给投资者带来再投资风险。这是定期存款可能提前收回的风险，迫使投资者将他或她的收益再投资于支付较低利息的CD。

> 随着 CD 到期日的临近，赎回保费金额通常会减少。在投资可调用 CD 之前阅读细则是明智的。

>

## 可调用 CD 示例

如果银行发行传统 CD 向投资者支付 4.5%，并且利率下降到银行可以仅以 3.5% 的价格向其他人发行同一张 CD，则银行将支付高 1% 的利率。 CD 的持续时间。通过使用可赎回 CD，银行可以选择为其再融资并以 3.5% 的收益率重新发行新的 CD。

如果银行发行可赎回 4.5% 的 CD 并在两年内到期，但将其第一个赎回日期设定为自发行之日起 6 个月后，则在这六个月过去之前，它将无法收回其 CD。该锁定期向投资者提供了至少半年支付4.5%利息的保证。如果银行决定在那个时候赎回 CD，银行向 CD 持有人支付的一次性赎回溢价将在一定程度上减轻较高利率的损失。

## 底线

可赎回 CD 将利率从银行转移给您。对于更高的风险，您将获得比使用具有相似到期日的传统 CD 更高的回报。在投资之前，您应该比较两种产品的价格。然后，考虑一下您认为未来利率的发展方向。如果您担心再投资风险并喜欢简单，可赎回 CD 可能不适合您。

＃＃ 强调

- 可赎回存款证 (CD) 可以选择在到期前由 CD 发行人以预设价格赎回。

- 银行可能会选择发行可赎回 CD，这样当利率下降时，它就不会在 CD 的期限内支付更高的利息。

- 可赎回存款证的收益率将略高于传统存款证，这是为了弥补投资者被赎回的风险。

＃＃ 常问问题

### 可赎回 CD 是否为 FDIC 投保？

可赎回 CD 通常由联邦存款保险公司 (FDIC) 或国家信用合作社管理局 (NCUA) 投保高达 250,000 美元。

### 可调用 CD 的缺点是什么？

在 CD 中增加赎回条款会给投资者带来再投资风险。这是定期存款可能提前收回的风险，迫使投资者将他或她的收益再投资于支付较低利息的CD。

### 可赎回 CD 是否比传统存款证明更具优势？

通常他们提供更好的利率。这是因为存单有可能在到期前被赎回，这将导致利息收入的损失，并带来再投资风险。
