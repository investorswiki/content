# 面值
## 什么是标称值？

证券的名义价值，通常称为面值或面值，是其[赎回](/redemption)价格，通常在该证券的正面注明。对于债券和股票，它是已发行证券的规定价值，而不是其市场价值。在经济学中，名义价值是指未经调整的利率或当前价格，不考虑通货膨胀或其他因素，而不是[实际价值](/real-value)，后者是根据一般价格水平随时间的变化进行调整的。

## 了解标称值

名义价值是许多债券和优先股计算的关键组成部分，包括利息支付、市场价值、折扣、溢价和收益率。由于供需考虑，普通股的名义价值通常会远低于其市场价值，而优先股的名义价值应更符合其市场价值。债券的名义价值将根据市场利率与其市场价值有所不同。

[名义](/nominal)和实际价值在经济学中也起着至关重要的作用，无论是考虑[名义 GDP](/nominalgdp)与实际 GDP 还是[名义利率](/nominalinterestrate)与实际利率。实际价值因素会影响购买力的变化。虽然[名义回报率](/nominal-rate-of-return)反映了投资者的收益占其初始投资的百分比，但[实际回报率](/realrateofreturn)将通货膨胀和投资者收益的实际购买力考虑在内。

## 债券面值

对于债券，面值是[面值，即](/facevalue)[到期](/maturity)偿还给债券持有人的金额。公司债券、市政债券和政府债券的面值通常分别为 1,000 美元、5,000 美元和 10,000 美元。

如果债券的[到期收益](/yieldtomaturity)[率](/yieldtomaturity)(YTM) 高于其名义利率（[票面利率](/coupon-rate)），则债券的实际价值将低于其面值（名义）价值，并且该债券以低于面值的价格出售，或[低于标准杆](/below-par)。相反，如果 YTM 低于其名义利率，则债券的实际价值高于其面值，据说它以[高于面值或面值的溢价出售，](/abovepar)如果它们相同，则债券以名义或面值出售。零息债券总是以低于名义价值的价格出售，因为投资者在债券到期之前不会收到利息。债券市值计算公式为：

**债券价格 = SUM(付息) / (1 + 市场收益率) ^ i + 面值 / (1 + 市场收益率) ^ n****其中：付息 = 面值 * 票面利率； i = 每年； n = 总年数**

例如，发行面值为 1000 美元、票面利率为 10% 的 3 年期公司债券。每年的息票支付为 100 美元（1000 美元 * 10%）。如果市场利率 (YTM) 高于票面利率，例如 12%，那么债券的市场价值将以低于面值的价格出售（低于 1000 美元）。**﻿**

**债券价格 = $100 / (1+12%) + $100 / (1+12%)2 + $100 / (1+12%)3 + $1000 / (1+12%)3**

债券价格 = 89.29 美元 + 79.72 美元 + 71.18 美元 + 711.79 美元 = 951.98 美元

##股票的名义价值

公司股票的面值或面值是公司发行[股本时为资产负债表目的而分配的任意值](/sharecapital)，通常为 1 美元或更少。它对股票的市场价格几乎没有影响。例如，如果一家公司获得了筹集 500 万美元的授权，并且其股票的面值为 1 美元，则它可以发行和出售最多 500 万股股票。股票面值和售价之间的差额称为[股份溢价](/sharepremiumaccount)，可能相当大，但从技术上讲，它不包含在股本中，也不受法定资本限制的限制。因此，如果股票售价为 10 美元，则 500 万美元将被记录为实缴股本，而 4500 万美元将被视为额外实收资本。

[优先股](/preferredstock)是支付股息的混合资产，可以转换为普通股。名义（面值）价值在这里非常重要，因为这是用于计算股息的金额。例如，一家发行 5% 优先股、面值（名义）为 50 美元的公司每年将支付每股 2.50 美元（5% * 50 美元）的股息。优先股的价格将取决于市场对所提供股息百分比的评估，在这种情况下为 5%。如果市场对 5% 感到满意，那么股票将在其名义（面值）附近交易。如果股息百分比高于或低于市场预期，则优先股的价格将以高于或低于其名义价值的价格交易。

## 经济学中的名义价值

在经济学中，名义价值是指当前的货币价值，不会根据通货膨胀的影响进行调整。随着时间的推移比较值时，这使得标称值有点无用。正是出于这个原因，投资者更喜欢将通货膨胀因素考虑在内的实际价值，以提供更准确和易于理解的相对比较。实际利率是名义利率减去通货膨胀率。

**实际利率 = 名义利率 - 通货膨胀率**

例如，如果某一年的名义[国内生产](/gdp)总值 (GDP) 增长率为 5.5%，相关的年通货膨胀率为 2%，则该年的实际 GDP 增长率为 3.5%。

## 名义汇率与实际汇率

名义[汇率](/exchangerate)是可以购买一单位给定外币的本币单位数。实际汇率定义为国外物价水平与国内物价水平的比值，其中国外物价水平通过现行名义汇率折算为本币单位。与名义汇率相比，实际汇率始终是浮动的，因为即使在固定汇率制度下，实际汇率也会随着通货膨胀的变化而变化。

在考察一个国家的出口竞争力时，重要的是实际汇率。[名义有效汇率（NEER）是一国货币兑换一篮子多种外币](/neer)的未经调整的加权平均汇率，是衡量一国在外汇市场上的国际竞争力的指标。但是可以调整 NEER 以补偿本国的通货膨胀率相对于其贸易伙伴的通货膨胀率，从而产生[实际](/reer)[有效汇率](/reer)（REER）。
