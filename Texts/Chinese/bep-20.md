# BEP-20
BEP-20 是币安智能链上的代[币](/token)标准，扩展了最常见的以太坊代币标准 ERC-20。您可以将其视为代币的蓝图，定义了如何使用、谁可以使用以及其他使用规则。由于它与币安链的[BEP-2](/bep-2)和以太坊的 ERC-20 相似，因此兼容两者。

BEP-20 被认为是币安智能链的技术规范，旨在为开发人员提供一种灵活的格式来推出一系列不同的代币。这些可以代表任何东西，从企业的股票到存储在银行金库中的美元（即稳定币）。

当然，同样可以创建一种原生资产作为 BEP-20 代币，甚至可以将来自其他区块链的代币与币安智能链挂钩。这就是“Peggy”硬币所做的，它本质上是其他加密资产（如 LINK 或 XRP）的 BEP-20 版本。

与币安链上的 BEP-2 代币一样，BEP-20 代币的转移也是由 BNB 推动的。这为验证者提供了将交易包含在区块链中的激励，因为他们将收取 BNB 作为其麻烦的费用。

您可能知道 Binance Smart Chain 被设想为 Binance Chain 的扩展。采用双链架构，两条链相辅相成——币安智能链迎合去中心[化应用](/decentralized-applications-dapps)，不拥塞原链，优化超快速交易。

由于这种架构，重点放在了跨链兼容性上。出于这个原因，BEP-2 代币可以交换为 BEP-20 等价物。最简单的方法可能是通过币安链钱包扩展，尽管随着时间的推移，更多的方法无疑会出现。

要更详细地了解 BEP-20 标准，请务必查看 GitHub 上的提案草案。
