# 区块链操作系统
## 什么是区块链操作系统？

区块链操作系统使用区块链作为支持系统，在计算机系统或平台的后台运行。例如，您的 Android 手机或 Windows PC 需要在智能手机的内存或 PC 的硬盘上本地安装相应的操作系统 (OS)，并且所有事务和命令都在本地执行。基于区块链的操作系统从用户设备中捕获所有命令和事务，在区块链上对它们进行身份验证、执行和记录。

## 了解区块链操作系统

超越流行[比特币的标准支付处理系统](/bitcoin) [加密货币](/cryptocurrency)，[区块链](/blockchain)在整个技术堆栈中得到广泛使用。[分布式账本](/distributed-ledger-technology-dlt)技术发展中的新兴时尚是区块链操作系统。

区块链本质上是一个账本转换的交易处理引擎。无论您需要处理付款，还是需要使用[以太坊](/ethereum)平台上的最新小工具武装您的加密猫，或者如果您想在唯链区块链上跟踪从葡萄园到您家门口的高成本葡萄酒运输，所有这些应用程序区块链基于身份验证、记录和处理交易。

任何标准操作系统，无论是 Microsoft Windows、Apple Mac 还是[Android](/android-operating-system)或[iOS等移动系统](/apple-ios)，也可以根据通过鼠标点击或屏幕点击发出的用户命令执行交易，所有任务都在设备上本地完成。同样的概念扩展到设备操作系统的区块链的使用，它作为操作系统工作被视为更高效的操作系统。

## Blockhain OS 努力的例子

构建基于区块链的操作系统的早期尝试首先出现在移动设备和智能手机上，它是一个基于云的虚拟系统。所有必要的交易处理都发生在云托管的基于区块链的数据中心上，用户只需通过设备触摸屏上的点击发出必要的命令。

例如，总部位于香港的 NYNJA Group Ltd. 与 Amgoo 智能手机制造商就其基于区块链的 NYNJA 虚拟操作系统 (vOS) 进行了战略合作。两家公司将与拉丁美洲的电信运营商合作，在激活时为 NYNJA vOS 用户提供初始数据块。 vOS 支持提供文本、语音、视频会议和项目管理工具的通信层、用于商业交易的安全支付层，以及支持比特币、以太坊和所有 ERC-20 兼容代币的多币种钱包。

OS 平台还支持商业活动市场——例如将熟练的[零工经济](/gig-economy)工人分配给用户的特定工作需求，以及用户买卖商品的市场。 vOS 由其称为 NYNJAcoin 或 NYN 的本机加密货币支持。

## 特别注意事项

预计区块链的所有好处和优势都将提供给区块链操作系统用户。

无论用户在其 Android 或 iOS 手机、Windows 或 Mac PC 上做什么，都容易被相应的应用程序、ISP 以及操作系统制造商捕获，这些制造商可能会在操作系统日志中记录所有用户活动。基于区块链的操作系统提供了安全和隐私的好处，以及对操作系统的去监管、去中心化的使用。

这个概念仍在不断发展，现实世界的使用是有限的。但是，如果它成功地提供了设备操作系统的流畅和整洁的工作，那么看到越来越多的设备在这种区块链操作系统上运行可能不会太远。

＃＃ 强调

- 最初旨在作为手机或其他连接智能设备的操作系统，区块链操作系统吹捧高水平的数据安全性和用户匿名性。

- 为了作为操作系统运行，区块链协议必须允许执行来自用户的计算机代码和命令。

- 区块链操作系统利用区块链分类账技术来全部或部分运行计算机系统或网络。
