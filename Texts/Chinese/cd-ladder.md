# 光盘阶梯
## 什么是 CD 梯形图？

CD阶梯是一种策略，在该策略中，投资者将一笔资金分成相等的数量，并将其投资于具有不同[到期日](/maturitydate)[的存款证（CD）](/certificateofdeposit) 。这种策略降低了利率和[再投资风险](/reinvestmentrisk)。

CD 是一种投资产品，在特定时期内提供固定利率。投资资金由[联邦存款保险公司 (FDIC)](/fdic)投保高达 250,000 美元，由开证行锁定，直至 CD 到期日。这些储蓄工具的到期日通常设置为三个月、六个月、一年或五年。承诺资金的期限越长，支付的[利息](/interest)就越高。为了利用不同时期提供的各种[利率](/interestrate)，投资者可以遵循一种称为 CD 阶梯的策略。

## 如何构建 CD 梯形图

假设您有 20,000 美元要投资，并且想要建立一个为期四年的 CD 阶梯。

### 步骤 1：打开单独的 CD

与其将所有资金都放在一张 CD 中，不如将 5,000 美元放入将在一年、两年、三年和四年后到期的四张 CD 中的每张中。在投资资金之前，您希望找到 CD 上利率最优惠的银行。你开始的是：

- 一年 CD 5,000 美元

- 5,000 美元的两年 CD

- 5,000 美元的三年期 CD

- 5,000 美元的四年 CD

### 步骤 2：在到期时更新和转换每张 CD

随着每张 CD 的成熟，您可以将其更新为 4 年的 CD。通过这样做，四年后您将拥有四张为期四年的 CD，但其中只有一张 CD 每年会成熟。

如果您在 2021 年 1 月打开了所有 CD，则设置阶梯将如下所示：

- 2022 年 1 月：将一年期 CD 更新为四年期 CD

- 2023 年 1 月：将两年期 CD 更新为四年期 CD

- 2024 年 1 月：将三年期 CD 更新为四年期 CD

- 2025 年 1 月：将四年期 CD 更新为四年期 CD

这将允许您在建立阶梯的同时利用长期 CD 的更高利率，并每年从阶梯中提取 25% 的资金，而不会因每年有一张 CD 到期而受到处罚。

## 迷你 CD 梯子

迷你 CD 阶梯与常规 CD 阶梯的概念相同，但 CD 期限较短。您可以用三个月、六个月、九个月和一年的 CD 构建一个迷你 CD 阶梯来部署相同的策略。但请记住，通过使用短期 CD 建立阶梯，您可以获得的利率会更低。

## CD 阶梯的好处

重视本金和收入安全的投资者遵循 CD 阶梯策略。它还为您提供稳定的现金流，因为 CD 将在不同时间到期。通过将投资分散到不同期限的 CD 上，您可以从较长期 CD 的较高利率中受益，并且不必反复更新持有您所有资金的短期 CD。

如果银行资不抵债，CD 还提供 FDIC 违约保险。除了同样由联邦政府支持的美国[国债（T-bonds）外，没有其他工具可以为固定收益投资者提供同等保护。](/treasurybond)

此外，通过分级您的 CD，您可以自定义它们的总（或总）利率，通常是[向上的](/upside)。例如，如果您只购买三个月的 CD 来产生季度现金流，您将实现相对较低[的](/rateofreturn)[回报率](/rateofreturn)。但是，使用阶梯式 CD 投资组合，您仍然可以实现季度付款，但总投资组合回报率要高得多，因为期限较长的 CD 通常支付更高的利息。

如果您将所有资金存入一张 CD，您可能会错过在资金被锁定期间发生的利率上升。但是，使用 CD 阶梯，您可以通过将到期 CD 的收益再投资到利率更高的新 CD 中来利用短期利率。另一方面，如果利率下降，您仍然可以享受现有长期 CD 提供的高利率带来的好处。

因此，CD 阶梯提供了在 CD 到期时进行现金再投资的定期机会，同时降低了[利率风险](/interestraterisk)。如果发生紧急情况并且您需要现金，阶梯式策略可确保您始终拥有到期的 CD，从而降低[流动性风险](/liquidityrisk)。

## 底线

与任何其他投资一样，使用阶梯式 CD 的做法完全取决于您的个人财务目标。一般来说，它们非常适合那些想要[资金安全](/capital)、可预测的现金流和简单的人。 CD 非常易于理解、访问和构建，可以满足您的财务目标。

另一方面，由于其提供的安全性，CD 的回报率通常较低。此外，它们没有提供特殊的税收待遇来节省地方、州或联邦税收。因此，如果您处于高[税率范围](/taxbracket)内，则很难证明其合理性。如果您处于低税率范围内，它们会更有意义。

请记住，使用 CD 最安全的做法是通过您的银行并确保您的存款在 FDIC 保险限额范围内。如果您决定通过经纪行，请了解您引入了各种风险变量，包括佣金、销售人员（经纪人）的可疑动机以及[本金的潜在损失](/principal)。保持简单以获取好处。

＃＃ 常问问题

### CD 阶梯是一项安全的投资吗？

通过受保银行购买的 CD 梯子是一项安全的投资。联邦存款保险公司 (FDIC) 为 CD 阶梯提供高达 250,000 美元的保护。只要投资者不提早取款，大多数 CD 阶梯都能清楚地显示预期回报。

### CD 阶梯有什么好处？

CD 阶梯具有普通 CD 的所有优点；这是一项安全的投资，可以随着时间的推移带来预期的回报。 CD阶梯的好处在于提前取款费用的风险较小，因为投资者将在更早的日期获得部分资金。

###谁应该制作存款证明（CD）阶梯？

存款证明 (CD) 阶梯对于寻求安全投资的个人来说是一项很好的投资。 CD 阶梯需要耐心，因为投资者将因任何提前提款而受到惩罚。此外，高级别的安全性意味着 CD 阶梯不一定会有巨大的回报，所以请记住这一点。
