# 远期折扣
## 什么是远期折扣？

远期折扣是一个术语，表示一种[货币的远期或预期未来价格](/currency)低于[即期价格的情况](/spotprice)。市场表明，当前的国内汇率兑另一种货币将下跌。这种远期折扣是通过将当前现货价格与现货价格加上给定时间段内的净利息支付与相同时间段内的远期外汇合约价格进行比较来衡量的。如果远期合同价格低于现货加上预期的利息支付，则存在远期折价的条件。

## 远期折扣如何运作

虽然经常发生，但远期折扣并不总是导致货币汇率下降。由于现货、远期和[期货](/futures)定价的一致性，这仅仅是预期会发生。通常，它反映了所涉两国货币之间的[利率差异可能引起的变化。](/interestrate)

远期货币汇率通常不同于货币的即期汇率。如果一种货币的远期汇率高于即期汇率，则该货币存在[溢价](/premium)。当远期汇率低于即期汇率时，就会出现折扣。负溢价相当于折扣。

## 计算远期折扣的示例

计算远期汇率的基础需要货币对的当前现货价格和两国的利率（见下文）。考虑这个[日元](/jpy-japanese-yen)和美元之间的兑换示例。

- 90 天日元兑美元 (¥/$) 远期汇率为 109.50。

- 即期汇率 ¥ / $ 汇率 = 109.38。

年化[远期保费计算](/forwardpremium)= (109.50-109.38÷109.38) x (360 ÷ 90) x 100% = 0.44%

在这种情况下，美元相对于日元是“强”的，因为美元的远期价值比即期价值高出每美元 0.12 日元的溢价。日元将以折价交易，因为其对美元的远期价值低于其[即期汇率](/spot_rate)。

要计算日元的远期折扣，您首先需要根据美元兑日元的关系计算日元的远期汇率和即期汇率。

- ¥/$ 远期汇率为 (1÷109.50 = 0.0091324)。

- ¥/$ 即期汇率为 (1÷109.38 = 0.0091424)。

以美元计的日元年化远期折扣 = ((0.0091324 - 0.0091424) ÷ 0.0091424) × (360 ÷ 90) × 100% = -0.44%

对于一年以外的期间的计算，您将输入天数，如下例所示。对于三个月远期汇率：远期汇率 = 即期汇率乘以（1 + 国内汇率乘以 90/360 / 1 + 国外汇率乘以 90/360）。

要计算远期利率，请将即期利率乘以利率比率并调整到期时间。

远期利率=即期利率x（1+国外利率）/（1+国内利率）。

例如，假设当前美元兑[欧元](/euro)汇率为 1.1365 美元。国内利率或美国利率为5%，国外利率为4.75%。将这些值代入方程得到：F = $1.1365 x (1.05 / 1.0475) = $1.1392。在这种情况下，它反映了远期溢价。

## 什么是远期合约？

远期合约是两方之间在特定未来日期以确定价格购买或出售货币的协议。它类似于[期货合约](/futurescontract)，主要区别在于它在[场外交易 (OTC)](/otc)市场进行交易。交易对手之间直接创建远期合约，而不是通过正式的交易所。

远期合约的好处包括定制条款、金额、价格、[到期日](/expirationdate)和交割基础。交割可以是现金，也可以是[标的资产的实际](/underlying-asset)[交割](/underlying-asset)。未来合约的缺点包括二级市场缺乏流动性。另一个缺陷是集中[清算所](/clearinghouse)导致更高程度的违约风险。因此，散户投资者不像期货合约那样容易获得远期合约。

合约远期价格可能与现货价格相同，但通常更高，从而产生溢价。如果现货价格低于[远期价格](/forwardprice)，则产生远期折扣。

投资者或机构持有远期合约以[对冲](/hedge)或推测货币走势。与客户进行远期合约投资的银行或其他金融机构消除了[利率掉期](/interestrateswap)市场中由此产生的货币风险。

＃＃ 强调

- 远期合约类似于期货，但没有标准化，并在场外交易中由两个特定方执行。

- 远期溢价是存在于远期外汇合约与货币现货价格之间的比较条件。

- 要了解这种情况，您首先需要了解什么是远期合约。

- 远期折扣意味着签订远期合约的人预计他们打算兑换的货币在未来某个时候会减少。
