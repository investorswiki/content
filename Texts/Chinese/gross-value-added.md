# 总增加值 (GVA)
## 什么是总增加值 (GVA)？

总增加值 (GVA) 是一种经济[生产力](/productivity)指标，用于衡量公司子公司、公司或市政当局对经济、生产者、部门或地区的贡献。

直接归因于该生产的所有投入和[原材料](/rawmaterials)的成本提供了一个美元价值。因此，GVA通过补贴和税收（关税）对产品的影响来调整[国内生产总值（GDP）。](/gdp)

## 了解总增加值 (GVA)

GVA 是该国的产出减去中间消耗，即总产出与净产出之差。 GVA 很重要，因为它用于计算 GDP，这是一个国家整体[经济状况的关键指标](/economy)。它还可用于查看特定地区、州或省增加（或损失）多少价值。

在国家层面，GVA 有时被视为衡量经济总产出和增长超过 GDP 或[国民](/gnp)生产总值 (GNP) 的指标。 GVA 通过产品税和产品[补贴与 GDP 相关](/subsidy)。它增加了政府给予某些经济部门的补贴，并减去了对其他部门征收的税款。

在公司层面，通常计算该指标以代表公司当前生产或提供的特定产品、服务或公司单位的 GVA。一旦减去[固定资本](/fixed-capital)的消耗和[折旧](/depreciation)的影响，公司就知道特定业务为其底线增加了多少净价值。换句话说，GVA 数字揭示了该特定产品对公司利润的贡献。

## 总增加值 (GVA) 示例

- 私人消费 = 5000 亿美元

- 总投资 = 2500 亿美元

- 政府投资 = 1500 亿美元

- 政府支出 = 2500 亿美元

- 出口总额 = 1500 亿美元

- 进口总额 = 1250 亿美元

- 产品总税额 = 10%

- 产品补贴总额 = 5%

使用这些数据，可以计算 GVA。第一步是计算GDP。回想一下，GDP 计算为私人消费 + 总投资 + 政府投资 + 政府支出 +（[出口](/export)- 进口）：

- **GDP** = 5000 亿美元 + 2500 亿美元 + 1500 亿美元 + 2500 亿美元 +（1500 亿美元 - 1250 亿美元）= 1.175 万亿美元

接下来，我们计算产品的补贴和[税收](/tariff)。为简单起见，假设所有私人消费都是对产品的消费。在这种情况下，补贴和税收如下：

- **产品补贴** = 5000 亿美元 x 5% = 250 亿美元

- **产品税** = 5000 亿美元 x 10% = 500 亿美元

有了这个，GVA可以计算如下：

- **总增加值** = 1.175 万亿美元 + 250 亿美元 - 500 亿美元 = 1.15 万亿美元

＃＃ 强调

- 总增加值 (GVA) 是衡量公司子公司、公司或市政当局对经济、生产者、部门或地区的贡献的经济生产力指标。

- GVA 很重要，因为它用于调整 GDP，这是衡量一个国家整体经济状况的关键指标。

- GVA 是该国的产出减去中间消耗，即总产出与净产出之差。

- 它也可以用来衡量产品或服务为满足公司的固定成本贡献了多少钱。
