# 通胀挂钩存款证
## 什么是通胀挂钩存款证？

顾名思义，与通货膨胀挂钩[的存款证 (CD)](/certificateofdeposit)是一种 CD，其[利率](/interestrate)与[通货膨胀率挂钩](/inflation)。为了换取这种额外的保护，与传统 CD 相比，与通胀挂钩的 CD 通常提供略低的利率。

## 与通胀挂钩的存款证如何运作

与房地产或公开交易的[股票等其他资产不同](/shares)，债务工具和储蓄账户在投资时以美元计价。因此，如果通货膨胀在投资期内增加，这可能会侵蚀投资期结束时返还本金的实际价值。出于这个原因，担心通胀的投资者可能希望考虑具有内置[通胀风险保护的投资工具](/inflationrisk)。

与通胀挂钩的 CD 通过将 CD 的利率与[消费者价格指数 (CPI)挂钩，从而为投资者提供这种保护，CPI](/consumerpriceindex)是一种广泛使用的通胀衡量指标。如果在 CD 期限内通货膨胀增加，该增加将被添加到工具的利率中，从而保护投资者免受利率风险。这些证券还由[联邦存款保险公司 (FDIC)](/fdic)投保高达 250,000 美元，使与通胀挂钩的 CD 成为投资者可获得的最安全资产之一。

与投资中经常发生的情况一样，这些风险保护的权衡是与通胀挂钩的 CD 通常提供非常低的收益率，通常略低于传统 CD。投资者提高其与通胀挂钩的 CD 的利率的一种方法是同意将其资金锁定更长的投资期限。通常，与通胀挂钩的 CD 的期限在六个月到六年之间，期限越长，利率越高。

## 与通胀挂钩的存款证明的真实示例

Michaela 是一名房地产投资者，经常买卖房产。她计划在六个月内开始建设一个新的房地产开发项目，但担心从现在到那时通胀可能会上升。因为她担心通货膨胀可能会侵蚀她持有的现金的实际价值，Michaela 决定将她的一部分现金投资到一张期限为 6 个月的与通货膨胀挂钩的 CD。

在学期结束时，Michaela 将提取她的资金，以帮助她为即将到来的建设项目提供资金。与此同时，与通胀挂钩的 CD 将根据 CPI 的任何潜在增长增加她的利息支付，从而保护她免受通胀风险。为了换取这种担保，Michaela 必须承诺将她的资金保留在 CD 中六个月，并且还必须接受比传统 CD 略低的起始利率。

＃＃ 强调

- 与通胀挂钩的 CD 是一种收益率相对较低的工具，但可以成为多元化投资组合的有用补充，尤其是对风险意识更强的投资者而言。

- 与通胀挂钩的 CD 是一种投资工具，可提供通胀风险保护。

- 这是通过将投资利率与通胀指标（通常是 CPI）挂钩来实现的。
