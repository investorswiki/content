# 中央处理器 (CPU)
是计算机的电子电路，负责解释计算机程序的指令并根据这些指令执行基本操作。基本操作包括算术、逻辑、控制和输入/输出（I/O）。自 1960 年代初以来，中央处理器 (CPU) 一词已广泛用于计算机行业。

CPU 主要由四个功能单元组成：

- 控制单元：负责控制 CPU 内的指令和数据流。

- 算术逻辑单元 (ALU)：执行 CPU 的所有算术和逻辑运算计算。

- 寄存器：是可以非常快速地访问的内部存储单元。这些组件用于存储变量（数据、地址）或算术/逻辑运算的中间结果。

- 高速缓存：是一种更小更快的内存，可以最大限度地减少对主内存的访问，从而提高 CPU 性能。

这些单元通过时钟速率同步并通过三种类型的总线连接：

- 数据总线：负责传送数据。

- 地址总线：传送要读取或写入的内存地址。

- 控制总线：允许管理其他组件和 I/O 设备。

CPU 架构的特征还在于它可以执行的指令集。通常，有两种类型的指令集架构：

- CISC（复杂指令集计算机）：一组广泛的复杂指令，可以在几个时钟周期内执行多个低级操作，例如算术运算、内存访问或地址计算。

- RISC（精简指令集计算机）：在一个时钟周期内执行单个低级操作的精简指令集。
