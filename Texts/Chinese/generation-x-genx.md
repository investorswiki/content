# X世代（X世代）
## 什么是 X 代（X 代）？

X 代，有时简称为 X 代，是 1960 年代中期至 1980 年代初出生的美国一代的名称。

构成 X 一代的确切年份各不相同。一些研究人员——例如人口统计学家威廉·施特劳斯和尼尔·豪——将确切的出生年份放在 1961 年至 1981 年之间，而盖洛普将出生年份置于 1965 年至 1979 年之间。但所有人都同意 X 一代跟随婴儿潮一代而先于 Y 一代或千禧一代。

## 了解 X 一代

“X 世代”这个名称来自于道格拉斯·库普兰 (Douglas Coupland) 于 1991 年出版的小说《X 世代：加速文化的故事》**。虽然它对营销比社会学更有用，但世代理论——假设人们出生在可以将同一时间框架视为具有相似观点、价值观、品味和习惯的群体——[代沟的概念](/generation-gap)在美国已获得广泛接受

该理论涵盖的美国几代人是：

-[最伟大的一代](/the_greatest_generation)（约 1901 年至 1924 年出生）

- 沉默的一代（大约 1925 年至 1945 年）

-[婴儿潮一代](/baby_boomer)（大约 1946 年至 1964 年）

- X 一代（大约 1965 年至 1985 年）

-[千禧](/millennial)一代（大约 1985 年至 2000 年）

- Z 世代（后千禧一代）（2001-2020）

X 一代的人数约为 6500 万，而婴儿潮一代和千禧一代各有约 7200 万会员。 X 一代有时也被称为“锁匙一代”，因为他们经常在放学后在家中无人看管，直到他们的父母下班回家。

与沉默的一代一样，X 一代被定义为“中间”一代。该集团的盈利能力和储蓄首先受到[网络泡沫破灭的影响](/dotcom-bubble)，其次受到 2008 年金融危机和[大萧条的影响](/great-recession)。在社会和政治权力方面，X 一代夹在婴儿潮一代之间，他们在越南和里根时代长大，而奥巴马时代则是千禧一代。

> X 世代与另一个称为[三明治世代的群体重叠](/sandwichgeneration)。每一代现代人都在这个位置上占有一席之地，它被用来描述中年人——由于寿命更长和晚年生孩子——发现自己同时支持年迈的父母和成长中的孩子。

>

## X 一代与婴儿潮一代和千禧一代

2020 年发布的第 20 届全美员工退休年度调查对 X 一代、婴儿潮一代和千禧一代进行了比较。在其发现中：

- X 一代认为，与千禧一代 (77%) 和婴儿潮一代 (73%) 相比，他们将比父母 (80%) 更难实现财务安全。

- 在三代人中，X 一代更有可能背负信用卡债务（52%），千禧一代更有可能拥有学生贷款（26%），而婴儿潮一代更有可能没有债务（25%）。

- 与千禧一代 (42%) 和婴儿潮一代 (45%) 相比，X 一代最不可能使用财务顾问 (37%)。

## X 世代的财务状况

在接下来的几十年里，财富将发生重大转移——总计约 48 万亿美元——从婴儿潮一代向包括他们的 X 一代子女在内的年轻一代转移。他们将需要它。

根据美联储的最新数据，X 一代仅占美国财富的 29%，而婴儿潮一代的财富仅占一半以上（51%）。 2008 年，中位年龄为 35 岁的 X 一代仅拥有全国 7.4% 的财富，不到婴儿潮一代 35 岁时财富的一半。

> X 一代的著名成员包括杰夫·贝佐斯、泰格·伍兹和已故的库尔特·科本。

## 退休储蓄

在 Transamerica 调查中，近 60% 的 X 一代受访者“强烈同意”或“有点同意”他们正在创造足够大的储备金。 X 世代平均有 64,000 美元的退休储蓄。毫不奇怪，婴儿潮一代拥有最多，144,000 美元，而千禧一代最少，为 23,000 美元。百分之九的 X 一代根本没有退休储蓄。

这远远达不到每一代人所期望的退休所需的水平。 X 世代和婴儿潮一代估计，要获得财务安全感，他们需要 50 万美元，而千禧一代需要 30 万美元。

## 市场时机对 X 一代的影响

平均而言，X 一代家庭在投资回报率低于婴儿潮一代的时期开始工作、储蓄和投资。许多 X 世代家庭在市场估值较高的时期开始积累储蓄，例如 1990 年代后期的[技术](/tech_bubble)泡沫和网络泡沫以及 2008 年全球金融危机的前夕。随之而来的[熊市](/tech_bubble)[的影响](/bearmarket)仍然沉重地压在他们的投资组合上。

根据 Transamerica 的调查，只有 44% 的 X 一代工人表示他们已经完全康复或没有受到大衰退的影响，而婴儿潮一代的这一比例为 50%。

此外，Xer 一代经历了一个特别低的利率环境，这对其增加金融资产价值的能力产生了不利影响。与此同时，X 一代投资者早期经历的市场大幅下跌似乎让他们更加[厌恶风险](/riskaverse)。

## X 世代面临的其他挑战

X 世代相对较低的财富水平将使他们难以维持父母的消费模式，教育、医疗和房地产成本上升。还有三明治综合症——这一代人已经到了抚养和教育孩子的年龄，同时还要照顾年迈的父母。

根据 Experian 的研究，X 一代现在拥有最高的平均债务。在 2019 年至 2020 年间，他们的平均债务增加了约 3.5%，即 4,802 美元，达到 140,643 美元。抵押债务占比最高，其次是HELOC贷款、学生贷款、汽车贷款、个人贷款和信用卡债务。

## 重塑 X 一代的退休生活

X世代的退休环境与其父母不同。曾经很常见的私营部门养老金计划很少见，并已被固定缴款计划所取代，例如[401(k)](/401kplan) 。 X 一代也不指望[社会保障](/socialsecurity)来资助他们的退休生活。

根据 Transamerica 的调查，婴儿潮一代 (37%) 更有可能期望社会保障成为他们退休收入的主要来源，而 X 一代的这一比例仅为 26%。事实上，41% 的 X 世代“强烈同意”他们退休时可能没有社会保障，而 26% 的婴儿潮一代也有同感。”

Transamerica 发现，总体而言，三代人在 65 岁以上工作的财务和健康老龄化相关原因都相同。但婴儿潮一代比其他几代人更有可能这样做，因为他们想要收入。 X 一代将继续工作，因为他们没有足够的储蓄而无法退休。

## X 一代的财务规划

财务压力的可能性可能很大，但可以采取措施减轻压力、平衡预算并减轻计划外生活事件的影响。以下是对 X 一代的一些建议，以使他们的财务生活井然有序，并应对世代三明治的各个层面：孩子、父母和他们自己。

### 制定遗产计划

如果您有受抚养的孩子并且还没有遗嘱或其他必要文件，这一点至关重要。您不希望您的家属或您的财物的命运由[遗嘱认证](/probate)法庭的法官决定。因此，现在是与[遗产规划律师](/estateplanning)预约的时候了，以获得您的遗嘱、生前遗嘱、医疗和持久授权书——也许还有生前信托——以确保您所有家属的顺利和快速转移、财产和对继承人的责任。

而且由于遗产清算可能是一个情感上微妙的过程，因此现在这样做可以让您和您的家人从冷静、合乎逻辑的角度思考应该如何进行。

### 获得全面的财务计划

当您 20 多岁时，管理您的财务是一个相当简单的事情，即养成良好的财务习惯，例如储蓄和预算。现在您的财务状况可能有点复杂，一个财务变量，例如您为公司的 401(k) 计划供款的金额，可能会以难以计算或难以计算的方式影响其他几个领域。以任何准确度进行预测。

这种可变的影响可能意味着现在是时候聘请专业的财务规划师或财务顾问了，他们可以将您的现金流、[资产负债表](/balancesheet)、风险承受能力、投资目标、时间范围和[税](/taxbracket)级纳入复杂的财务规划计划。这至少可以让您了解您的实际财务状况以及您需要做什么才能在退休年龄达到您想要的位置。只是准备好在最后看到一些令人不快的数字，这些数字可能表明您将无法像您希望的那样尽快退休。

### 管理你的债务

如果您正在考虑买房，最好先看看 15 年期[固定利率抵押贷款](/fixed-rate_mortgage)。利率可能再也不会这么低了，至少在 X 一代的一生中，15 年期贷款的利息仅为 30 年期抵押贷款的三分之一。如果您的债务负担变得难以管理，请找一家合法的债务管理公司来帮助您控制债务。

### 在大学规划上抢占先机

尽管大多数专家警告父母不要将退休储蓄转移到孩子的大学基金中，但现在是开设[Coverdell 教育储蓄账户](/coverdellesa)或[529 计划](/529plan)基金（如果不存在）的时候了。你的孩子和你一样可以为这些基金捐款，你从已故父母或其他亲属那里继承的钱也可以成为大学资助的来源。为他们开设个人退休账户可能是另一个不错的选择，只要您确信他们不会出于其他目的提取供款。

### 从父母那里获取财务状况

诚然，父母和孩子之间关于金钱的谈话可能很尴尬。但是，如果您还没有与您的父母谈论过他们的健康状况和财务状况，那么可能是时候在这方面开展工作了。如果您父母的健康状况不佳并且他们没有制定遗产计划，那么如果他们同意，最好自己掏出现金来完成这项工作。

如果您在处理管理式医疗问题时需要帮助，请咨询年长的律师寻求建议，并选择指定的兄弟姐妹作为处理这些问题的关键人物。年迈父母的孩子常犯的一个错误是高估了[Medicare](/medicare) 、Medigap 和[Medicaid的](/medicaid)覆盖范围。了解需要自费支付的费用可以确定购买[长期护理保险](/ltcinsurance)（如果仍然可行）和补充保险政策是否有益。

###让返校的孩子做出贡献

照顾年迈父母的压力可能会乘以赡养成年子女的费用。要求大学毕业后回家的后代帮助支付家庭开支——包括支付房租、购买杂货或协助照顾老人——可以减轻与抚养多代人相关的一些压力。它还可以为孩子们提供一些关于财务和财政责任的人生课程。

＃＃ 常问问题

### Z 世代或 X 世代年龄更大吗？

X 世代比 Z 世代早几十年。

### X 世代中的“X”代表什么？

字母“X”指的是一个未知变量（如数学中的），或者指代不被定义的愿望。

### 为什么 X 一代有时被称为迷失的一代？

也被称为“迷失的一代”、“被遗忘的一代”或“隐形一代”，X 一代被贴上了这些绰号，因为在他们崛起期间发生的社会价值观发生了变化，见证了离婚、单亲家庭、和孤独。今天，GenX 再次被认为是“迷路”，因为它夹在婴儿潮一代和千禧一代之间。
