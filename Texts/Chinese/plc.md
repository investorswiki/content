# 公共有限公司（PLC）
## 什么是公共有限公司 (PLC)？

公共有限公司（PLC）是英国的一家上市公司。 PLC 相当于带有 Inc. 或公司名称的美国上市公司。在公司名称后使用 PLC 缩写是强制性的，并且向投资者和与公司打交道的任何人传达它是一家上市公司。

## 公共有限公司 (PLC) 的运作方式

PLC 指定向公众提供股票的公司。这些股票的买家承担有限责任。这意味着，他们不对超过他们为股票支付的金额的任何商业损失负责。

在英国，PLC 的运营方式与美国的上市公司类似。它的运营受到监管，并且需要定期向股东和潜在股东发布关于其真实财务状况的报告。

## PLC 的要求

英国公司法规定，PLC 必须在公司名称和最低股本 50,000 英镑之后具有 PLC 名称。与美国的上市公司一样，PLC 提供各种类型的股票，例如普通股和累积优先股。 PLC的普通股类似于美国公司发行的普通股。

累积优先股类似于美国的优先股 PLC 的其他关键要求包括发行股份、任命董事和遵守注册要求。 PLC 还必须有 PLC 或公共有限公司作为名称的一部分。

## PLC 的优缺点

成立公众有限公司（PLC）的最大优势在于它赋予了通过发行公众股票筹集资金的能力。在公共证券交易所上市会吸引对冲基金、共同基金、专业交易员以及个人投资者的兴趣。与私人有限公司相比，这往往会导致增加对公司投资的资本。

> 最大的 PLC 组成了金融时报证券交易所 100 指数，即 Footsie。

>

另一方面，英国对 PLC 的监管要多得多，就像美国对上市公司一样。他们必须举行对所有股东开放的年度股东大会，并遵守更高的会计透明度标准。因为它们是公开的，它们也容易受到来自股东的压力和来自竞争对手的收购要约。

通过成为 PLC，公司可以获得更多的资本，并为股东提供流动性。这些是美国公司上市的类似好处。不利的一面是，成为 PLC 意味着更多的审查和要求的报告。公司将拥有更多股东，公司的价值可能会变得更加波动，因为它是由金融市场决定的。

## 公共有限公司（PLC）与私人有限公司（LTD）

PLC在英国是一家上市公司同时，还有私人有限公司（LTD），它们是英国的私人公司，私人有限公司的股份不向公众发售。

私人公司仍然成立，通常在公司大楼。这些公司仍然需要有法律文件才能成立业务。私人公司必须至少有一名董事。

要通过在英国的公共投资筹集资金，公司必须是 PLC。 PLC 与 LTD 类似，只是它们是公开交易的，其股票可以在证券交易所自由出售和交易。同时，PLC必须至少有两名董事并举行年度股东大会。

## PLC 示例

根据定义，所有在[伦敦证券交易所上市的公司都是 PLC。](/lse)时装零售商 Burberry 是 Burberry Group PLC。汽车制造商劳斯莱斯是劳斯莱斯控股有限公司。伦敦证券交易所的 100 家最大的 PLC 被归为一个名为金融时报证券交易所 100 (FTSE 100) 或通俗地称为 Footsie 的指数。

该组中的公司代表了整个英国的经济。 Footsie 与美国道琼斯工业平均指数 (DJIA) 相当。 截至 2021 年 3 月，Footsie 中市值最大的 PLC 包括联合利华、汇丰银行和阿斯利康。

荷兰皇家壳牌、汇丰控股、BP、葛兰素史克和英美烟草。所有这些公司的正式名称都包括 PLC 名称。并非所有 PLC 都在证券交易所上市。公司可能会选择不在交易所上市，也可能不符合上市要求。

## 公共有限公司 (PLC) 常见问题解答

### 成为一家上市有限公司 (PLC) 意味着什么？

PLC 是英国的一家上市公司。这些公司的名称后面必须有 PLC 或“公共有限公司”字样。例如，石油和天然气公司 BP plc 是一家英国上市公司，总部位于英国伦敦。

### 谁是公共有限公司的所有者？

与总部位于美国的上市公司一样，PLC 由股东所有。这些公司在交易所和股票上交易，个人、公司、共同基金等可以公开买卖股票。

### PLC 的主要特点是什么？

PLC的主要特点是它位于英国并且是公开交易的。公司名称后还必须有 PLC 或“公共有限公司”的名称。

### 公共和私人有限公司有什么区别？

PLC是一家上市公司，而私人有限公司也是一家英国公司，只是它是私人的。两者之间还有其他显着差异，例如私人有限公司只需要一名董事，而 PLC 必须有两名董事。
