# 超文本标记语言 - HTML
## 什么是超文本标记语言 (HTML)？

超文本标记语言 (HTML) 是一组标记符号或代码，插入到旨在在 Internet 上显示的文件中。标记告诉网络浏览器如何显示网页的文字和图像。

每个单独的片段标记代码（位于“<”和“>”字符之间）被称为一个元素，尽管许多人也将它称为一个标签。一些元素成对出现，指示某种显示效果何时开始以及何时结束。

## HTML 解释

超文本标记语言是促进网站创建的计算机语言。这种语言与任何其他语言一样具有代码字和语法，相对容易理解，并且随着时间的推移，它允许人们创造的东西越来越强大。在设计和维护语言的组织万维网联盟的幌子下，HTML 不断发展以满足 Internet 的需求和要求；例如，随着向[Web 2.0的过渡](/web-20)。

超文本是 Internet 用户浏览 Web 的方法。通过单击称为超链接的特殊文本，用户将被带到新页面。使用 hyper 意味着它不是线性的，因此用户只需单击可用链接即可访问 Internet 上的任何地方。标记是 HTML 标签对其中的文本所做的；他们将其标记为特定类型的文本。例如，标记文本可以以粗体或斜体的形式出现，以引起对单词或短语的特别注意。

## 超文本标记语言基础

HTML 的核心是输入到文本文件中的一系列短代码。这些是支持 HTML 功能的标签。文本保存为 HTML 文件并通过 Web 浏览器查看。浏览器读取文件并将文本转换为可见形式，按照作者用来编写可见渲染的代码的指示。编写 HTML 需要正确使用标签来创建作者的愿景。

标签是将普通文本与 HTML 代码分开的地方。标签是所谓的尖括号之间的词，它允许图形、图像和表格出现在网页上。不同的标签执行不同的功能。最基本的标签将格式应用于文本。随着 Web 界面需要变得更加动态，可以使用级联样式表 (CSS) 和 JavaScript 应用程序。 CSS 使网页更易于访问，而 JavaScript 为基本的 HTML 增添了力量。

## HTML 与 XML

与 HTML 不同，可扩展标记语言或 XML 允许用户定义自己的标记。例如，使用 XML，一个用户可以选择使用标签 <footnote> 来表示脚注，而另一个用户可以选择 <fn>。

使用 HTML，只能使用一个预先确定的标签来表示特定类型的信息。 XML 文档旨在易于阅读，因为它们包含用户定义的标签，并且文档仅包含标记和内容。

＃＃ 强调

- 超文本标记语言 (HTML) 是 Web 浏览器用来在万维网上呈现页面的基本脚本语言。

- HTML 的早期版本是静态的（Web 1.0），而较新的迭代具有很大的动态灵活性（Web 2.0、3.0）。

- 标记是出现在两个尖括号之间的文本（例如，<footnote>），而内容则是其他一切。

- 超文本允许用户单击链接并被重定向到该链接引用的新页面。
