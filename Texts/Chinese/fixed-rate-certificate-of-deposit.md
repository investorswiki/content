# 固定利率存款证 (CD)
## 什么是固定利率存款证 (CD)？

固定利率[存款证 (CD)](/certificateofdeposit)是一种在整个期限内具有固定利率的投资工具。 CD 通常以三个月至一年为增量提供期限，然后切换为两年、三年和五年期限。固定利率CD的期限越长，[固定利率就越高](/fixedinterestrate)。大型和小型零售银行都提供固定利率的 CD。

## 了解固定费率 CD

对投资保守的储户被固定利率 CD 所吸引，这为他们提供了已知的收入流，直到[到期为止](/maturity)。此外，由于[联邦存款保险公司 (FDIC)为 CD 提供](/fdic)高达 250,000 美元（每个账户持有人、每个发行人）的担保，因此将资金投入这些工具的投资者对资产价值的安全性感到放心。固定利率 CD 支付的利息可能不及其他固定收益证券，但保守的储户接受以较低[利率](/interest)换取较低[资本风险的权衡](/capitalrisk)。

[提前从 CD 中提取资金](/earlywithdrawal)通常会受到处罚，因此 CD 持有人几乎总是将资金留在工具中，直到其到期。到期后，根据个人的财务需要，他们可能会将已到期的 CD 转入另一张 CD。然而，新的固定利率很可能与刚刚成熟的不同。经济中的一般[利率](/interestrate)环境决定了发行银行如何设置固定利率 CD。

## 固定利率 CD 与可变利率 CD

浮动利率 CD 与固定利率 CD 一样具有固定期限，但利息支付可能会波动，因为 CD 的利率与某个指数挂钩，例如最[优惠利率](/primerate)指数、[消费者价格指数 (CPI)](/consumerpriceindex)或国库券速度。支付的金额基于初始指数值和最终指数值之间的百分比差异。浮动利率 CD 的投资者比固定利率 CD 购买者的风险厌恶程度低，并且个人通过将资金存入可变利率 CD，可能会表达他们的信念，即经济中的利率将在整个期限内上升的光盘。如果这被证明是正确的，那么 CD 将比固定利率 CD 产生更多的利息。

[税率](/taxbracket)为他们赚取的利息缴纳联邦税。

## 固定费率 CD 示例

银行提供保证 5% 利率回报的固定利率 CD。 CD 的期限为六个月。 Tatiana 在 CD 上投资了 1,000 美元。六个月后，她可以选择提取 1,050 美元或将其转入另一张 CD。她选择后一种选择，并在年底时提取 1,100 美元。她将对她 100 美元的收入缴税。

Tatiana 的朋友 Marc 也在同一张 CD 上投资了 1000 美元，但由于家庭紧急情况，三个月后被迫提取全部金额。提前退出的罚款是三个月的利息。 Marc 为提前退出支付了 12.50 美元的罚款。

＃＃ 强调

- 固定利率存款证 (CD) 是一种在整个期限内具有固定利率的投资工具。

- 与浮动利率 CD 不同，固定利率 CD 的利率保持不变。

- CD 到期后，持有人可以提取全部金额或将其转入另一张 CD。

- 通常情况下，较长期的固定利率 CD 支付更高的利率，并且提前从 CD 中提取资金会受到处罚。

＃＃ 常问问题

### 在经济衰退期间，我应该把钱存入存款证 (CD) 吗？

在经济衰退期间，人们想要最安全的投资选择。固定利率存款证 (CD) 是一种安全的选择，因为它们由联邦存款保险公司 (FDIC) 提供高达 250,000 美元的保险。请记住，在 CD 到期之前，存入 CD 的资金在没有提款费用的情况下不会轻易被清算。

### 使用固定利率 CD 或流动 CD 会赚更多吗？

[液体 CD](/liquid-certificate-of-deposit)高得多的利率。固定利率 CD 还为投资较大初始存款并长期投资的个人提供更高的利率。流动性 CD 的利率较低，可以灵活地提早取款。

### 如果我从固定利率 CD 中提取资金，是否会受到处罚？

是的，在到期前从固定利率 CD 中提取资金会受到处罚。如果您担心在到期日之前可能需要您的资金，请考虑投资[CD 阶梯](/cd-ladder)。这种投资策略可确保您在不同时间段取回部分资金，希望避免任何提前取款。
