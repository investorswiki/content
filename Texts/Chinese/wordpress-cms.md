# WordPress (CMS)
## 什么是 WordPress？

WordPress 是一种流行的开源内容管理系统 (CMS)。虽然它最初主要与个人博客相关，但后来被广泛用于各种网站，包括专业出版物和[电子商务](/ecommerce)平台。

自 2003 年推出以来，WordPress 已经发展到为全球大约三分之一的网站提供支持。今天，它是一个大型开发者社区的所在地，该社区已经制作了超过 50,000 个插件。

## WordPress 的工作原理

WordPress 的用户可以从该服务的两个基本版本中进行选择。其中第一个是 WordPress 的“托管”版本，可在 WordPress.com 上访问。此版本允许用户启动托管在 WordPress 自己的服务器上的网站。因此，它通常受到非技术用户的青睐，他们可能希望避免在自己的私人服务器上托管网站的复杂性。作为一个额外的好处，WordPress 的托管版本会自动更新到最新版本的 WordPress 软件，这有助于缓解潜在的安全漏洞。

对于更高级的用户，也可以简单地从 WordPress.org 下载最新版本的 WordPress。通过选择此选项，用户必须自行安排将软件托管在他们自己的私人服务器上或从[第三方](/third-party)托管服务提供商处托管。然后，用户可以自由定制他们认为合适的软件，但还必须负责手动上传任何后续更新。

由于其庞大的用户群，WordPress 网站是黑客的热门目标，他们经常寻求利用可能具有已知安全漏洞的过时版本的软件。出于这个原因，避免自托管服务可能是谨慎的做法，除非您有信心保护和更新服务以及您可能安装的任何其他插件。

> ### WordPress 自定义

> WordPress 从头开始设计为高度可定制的。它的[应用程序编程接口 (API)](/application-programming-interface)涵盖了从数据库到主题修改和定制的所有内容。通过保持一切标准化，开发人员可以专注于在 WordPress 之上构建功能，因为知道它将适用于所有保持最新的安装。

>

## WordPress 的真实世界示例

有许多不同的公司提供旨在扩展其功能的 WordPress 插件和主题。例如，WooCommerce 已成为 WordPress 最受欢迎的电子商务工具之一，使用户能够接受订单、跟踪发货，并在其现有的 WordPress 网站上添加其他电子商务功能。

在 WordPress 插件的许多其他示例中，涉及搜索引擎优化 (SEO)、联系表格、新闻通讯营销、现场社交媒体和评论功能以及旨在使网站更具视觉吸引力的各种设计功能。

＃＃ 强调

- WordPress 是领先的开源 CMS 平台。

- 它以其高度可定制的特性而闻名，拥有庞大的插件开发者社区。

- 由于其庞大的用户群，WordPress 经常成为黑客的目标。为了降低这种风险，用户应确保他们的 WordPress 软件始终保持最新。
