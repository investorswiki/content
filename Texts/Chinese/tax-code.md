# 税法
## 什么是税码？

税法是联邦政府的文件，通常有数千页，详细说明了个人和企业在将一定比例的收入汇给联邦或州政府时必须遵守的规则。税法由负责为公众解释的税务律师用作来源。

## 税法如何运作

在美国，美国国会制定税法并在联邦一级制定规则。[国税局](/irs)( [IRS](/irs) ) 实施既定规则，并通过税法解释它们如何应用于不同的场景。在州一级，这些法律由州、地方或县政府制定，这些政府使用税法来授权任何经过投票和商定的税收。实际上，税法有时被称为[国内税收法](/internal-revenue-code)(IRC)，是由联邦、州和地方政府当局颁布的税法集合。

每个通过的税法都被分配了一个代码，该代码被添加到 IRC 出版物中现有税法的集合中。由于普通人不容易理解税法，国税局提供了详细的说明，分解每个代码以及如何应用它们。 IRS 提供的所有税率、排除、扣除、抵免、养老金和福利计划、[个人豁免](/personal-exemption)等均取自联邦税法。 IRC 中的税码按部分组织和引用。例如，《国内税收法》第 1 节对美国公民和居民以及遗产和信托的应税收入征收联邦所得税。 IRC 第 11 条规定征收企业所得税。

<h5><a href="">TTT</a></h5>

一些二手资料，例如编号的美国国税局出版物、税收裁决和大众市场所得税书籍，试图将税法以通俗易懂的语言呈现给纳税人。 IRS 出版物可从 IRS 网站免费获得印刷版或在线版。

另一个试图解释税法的次要来源是[美国](/ustreasury)[财政部](/ustreasury)在大多数税法部分发布的财政部法规或税收法规，以提供有关如何使用法律的更长解释和示例。这些法规发布在联邦法规 (26 CFR) 第 26 篇中，也可在 GPO 网站上在线获取。

联邦法规电子代码是 CFR 的不断更新版本。纳税人通常可以通过遵循这些二次出版物中列出的指南正确遵守税收规则，但对于更复杂的情况，可能需要直接查阅税法，以破译税法。
