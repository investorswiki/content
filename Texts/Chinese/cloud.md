# 云
在计算机科学中，术语“云”是指共享资源池，通过 Internet 可供多个用户使用。这些资源通常与数据存储和计算能力有关，但也可能包括不同类型的服务、应用程序、网络和服务器。

通常，基于云的资源可以轻松方便地访问，并且可以动态重新配置以适应多种用途，并具有不同级别的可扩展性。如今，云计算技术广泛分布于人类生活的多个领域。一些服务和应用程序是在云上构建的。

例如，像 Netflix 这样的点播流媒体服务正在利用云计算功能来有效扩展并提供尽可能好的用户体验。 Skype 和 WhatsApp 等流行的在线消息和语音通话应用程序也在使用云计算来实现用户之间的高质量通信。其他示例包括 Microsoft Office 365 和 Google G Suite 提供的基于云的解决方案，它们正在帮助全球数百万人，让您随时随地轻松地进行实时工作和协作。

美国国家标准与技术研究院 (NIST) 的 Peter Mell 和 Timothy Grance 将云计算描述为一种由三种服务模型和四种部署模型组成的技术。

三种云计算服务模式包括：

- 基础设施即服务 (IaaS)：提供基础计算资源，如数据处理、数据存储和网络功能。这些可以基于 Amazon EC2、Microsoft Azure 和 Google Compute Engine 等云计算服务。

- 平台即服务 (PaaS)：提供平台，允许客户开发和部署获得的或消费者创建的应用程序到云计算平台。 AWS Elastic Beanstalk、Heroku 和 Google App Engine 是 PaaS 模型的一些示例。

- 软件即服务 (SaaS)：结合了基础架构和在云中运行的软件。用户可以访问提供商的软件应用程序和数据库，而云提供商则完全管理运行应用程序的底层基础设施和平台。 Salesforce、Microsoft Office 365 和 Slack 是 SaaS 模型的示例。

四种云计算部署模型包括：

- 公共云：可能由企业、政府机构或其他第三方云服务提供商拥有和运营的云。公共云是为公众开放使用而设计的。

- 私有云：虚拟化和分发 IT 基础架构以供单个组织及其消费者或业务部门独家使用的云（即不向公众开放）。

- 社区云：为具有兼容目标和关注点（例如，安全要求、策略、合规性考虑等）的特定消费者群体或社区虚拟化和分发 IT 基础架构的云。

- 混合云：两种或多种不同云基础架构（公共、社区或私有）的组合。
