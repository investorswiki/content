# 巨灾债券
## 什么是巨灾债券 (CAT)？

巨灾债券 (CAT) 是一种高收益[债务工具](/debtinstrument)，旨在在发生自然灾害时为保险业公司筹集资金。 CAT[债券](/bond)允许[发行人](/issuer)仅在发生地震或龙卷风等特定条件时从债券中获得资金。如果受债券保护的事件导致向保险公司支付利息，支付利息和偿还本金的义务要么被推迟，要么被完全免除。

CAT 债券的[到期日较短](/maturitydate)，为 3 至 5 年。这些证券的主要投资者是[对冲基金](/hedgefund)、养老基金和其他机构投资者。

## 了解巨灾债券

财产险和意外险保险公司以及[再保险公司使用巨灾债券](/reinsurance)将风险转移给投资者。这些债券于 1990 年代首次上市，为保险公司和再保险公司提供了另一种抵消与承保政策相关的风险的方式。机构投资者可以从 CAT 债券获得比大多数其他[固定收益证券](/fixed-incomesecurity)更高的利率。

CAT 债券是一种保险相关证券 (ILS)——与预先指定的事件或保险相关风险相关的金融证券的总称。仅当受债券保护的灾难发生时，才向保险公司支付 CAT 债券。

## CAT 债券支付如何运作

发行 CAT 债券时，投资者筹集的资金会进入一个安全的[抵押](/collateral)账户，在那里他们可以投资于各种其他低风险证券。向投资者支付的利息来自安全的抵押账户。

可以构建 CAT 债券，以便仅当自然灾害总成本在指定的承保期内超过特定的美元金额时才会支付。债券也可以与风暴或地震的强度或事件的数量挂钩，例如袭击德克萨斯州的超过五次命名飓风。一系列自然灾害将触发向保险公司支付的款项，这些资金也将来自安全抵押账户。

如果承保的自然灾害的成本超过债券发行筹集的总金额，投资者将失去[本金。](/principal)但是，如果在债券的存续期内，应对灾难的成本不超过规定的金额，投资者将在债券到期时获得本金返还。投资者还可以从定期支付的利息中受益，以换取持有债券。

## CAT 债券的好处和风险

### 潜在好处

的[利率](/interestrate)通常与金融市场或经济状况无关。通过这种方式，即使在利率较低且传统债券收益率较低的情况下，CAT 债券也能为投资者提供稳定的利息支付。此外，[机构投资者](/institutionalinvestor)可以使用 CAT 债券来帮助[分散](/diversification)投资组合，以防范经济和市场风险。这种潜在风险降低的原因是这些投资不一定与经济表现或股市走势相关。

[派息股票](/dividend)相比，CAT 债券的收益率具有竞争力。 CAT 债券的投资者在债券有效期内收到固定利息。此外，由于这些债券的期限通常是短期的，因此触发支付的事件发生的可能性较小。

CAT 债券有利于保险业，因为筹集的资金降低了他们在自然灾害保险方面的自付费用。 CAT债券还在保险公司最需要的时候为他们提供现金，这可以防止他们因自然灾害而需要申请破产。

### 潜在风险

虽然 CAT 债券可以降低保险公司的风险，但风险由证券的购买者承担。债券到期时间较短，在一定程度上减轻了损失投资本金的风险。

根据保险信息研究所的数据，2019 年全球自然灾害造成的总体损失总计 1500 亿美元，与过去 30 年经通胀调整后的平均水平大致持平，低于 2018 年的 1860 亿美元。2019 年，820 起事件造成损失，而 2018 年为 850 起。2019 年事件的保险损失总额为 520 亿美元，低于 2018 年的 860 亿美元。因此，尽管 2019 年的总体损失与 2018 年相比有所下降，但损失的成本可能会高达数十亿美元，持有 CAT 债券的投资者面临损失全部或部分投资的风险。投资者需要权衡 CAT 债券提供的诱人收益率的风险与回报。

CAT 债券可以分散经济和[市场风险](/marketrisk)，因为自然灾害通常与经济事件和股市走势无关。但是，如果自然灾害导致经济衰退并随后导致股市下跌，则可能存在例外情况。如果灾难性事件促使向保险公司付款，持有 CAT 债券的投资者将面临损失本金的风险。但是，如果灾难性事件发生在经济衰退期间，如果一些投资者在失去对 CAT 债券的投资的同时也失去了收入来源（工作），后果可能会更加复杂。

## 巨灾债券示例

假设美国最大的[互助保险公司之一 State Farm Insurance](/mutual-insurance-company)发行了 CAT 债券。该债券[面值](/facevalue)1000美元，两年到期，年利率6.5%。购买 CAT 债券的投资者每年将获得 65 美元，本金将在到期时返还。发行 CAT 债券筹集了 1 亿美元的收益，并存放在一个特别账户中。

- 债券的结构使得只有在两年内自然灾害总成本超过 3 亿美元时才会向 State Farm 支付款项。任何剩余资金将在债券到期时返还给投资者。

- 在第二年期间，发生了一系列自然灾害，总损失达 5.5 亿美元。这激活了对 State Farm 的支付，1 亿美元从特别账户转移到保险公司。

- 持有 1,000 美元 CAT 债券的投资者在第一年赚取了 65 美元的利息，然后在第二年失去了本金。 State Farm 通过发行 CAT 债券将自然灾害的成本从 5.5 亿美元降低到 4.5 亿美元。

＃＃ 强调

- 投资者可以在债券的有效期内获得高于大多数固定收益证券的利率。

- 如果确实发生了特殊事件，引发了支付，支付利息和返还本金的义务要么被推迟，要么被完全免除。

- 巨灾债券 (CAT) 是一种[高收益](/high_yield_bond)债务工具，旨在在发生自然灾害时为保险业公司筹集资金。

- CAT 债券允许发行人仅在发生特定事件（例如地震或龙卷风）时才能收到付款。
