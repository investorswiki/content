# 交货地点
## 什么是交货点？

[期货合约](/futurescontract)或其他衍生品中，交割点是合约标的实物[商品](/commodity)的交割地点。持仓的期货合约买家必须准备好接受交割，并为实物商品支付商定的价格。但是，交割点仅适用于规定资产实物交割的未来合约。以[现金结算的](/cashsettlement)合约不进行实物交割。

## 了解交货点

交割点是编写期货合约的重要因素。选择的交割点会影响[标的资产](/underlying-asset)的净[交割价格](/deliveryprice)或成本。交货条款保证所交付货物的价值。对于实物交割，商品的价格因地点而异，这是由于将商品从货源地运输到交货点的成本。因此，要为合同目的指定商品的单一价格，交货点是必不可少的细节。

大多数[期货市场](/futuresmarket)参与者进行投机交易，并且大多数人不考虑在期货合约中接受实物商品的交割。这些投机买家购买期货合约是因为他们相信基础商品的价格会上涨，而不是因为他们有兴趣接收几千桶石油或几千头牛。接收交付需要有资源来存储这些资源并将其推销给潜在的最终买家。

希望获得实物商品的期货合约买家经常希望锁定他们在生产中使用的商品的价格。通过购买期货合约，他们希望降低资产价格不利变动的风险。

## 交货点示例

交货点通常是所交易商品的重要交通枢纽。由交货点引起的价格变化在汽油价格中很容易观察到。如果您在城市之间进行公路旅行，您通常会注意到汽油平均价格的逐渐变化。大型炼油中心附近的价格最低。

- 大多数天然气和液化天然气 (LNG) 纽约商品交易所 ( [NYMEX](/nymex) ) 执行期货的交割点是位于路易斯安那州埃拉斯的天然气管道[亨利枢纽。](/henry_hub)

- 由芝加哥期货交易所 ( [CBOT](/cbot) ) 执行的玉米和大豆期货将运往伊利诺伊河 204 英里段的码头

- 活牛和生猪到达分布在五个地区或领地的各个牲畜场和屠宰场

- 在纽约商品交易所交易的西德克萨斯中质原油 (WTI) 原油期货，在俄克拉荷马州的库欣枢纽设有交割点

## 特别注意事项

当交付发生时，代表特定地点商品的特定数量和质量的[仓单或不记名收据从卖方转手给买方。](/warrant)然后发生全额支付。买方有权将商品从仓库中取出。通常，购买者会将产品留在存储地点并支付定期存储费。交易所还为交付过程的许多方面设定费用。

谨慎选择商品交割点至关重要。国内地点可能会提出一组法律、税收和监管要求，这些要求与将国外地点设置为交货点所产生的要求有很大不同。这些差异可能足以使给定的交易变得不切实际甚至不可能。

＃＃ 强调

- 期货合约将指定在特定交割点（或多个地理区域存在的点）交割的标准化商品。

- 交割点是一个约定的地点，与到期衍生品合约相关的标的资产从空头实际交付到多头。

- 根据要交付的商品，存储和交付成本会有所不同，并反映在衍生品合约的价格中。
