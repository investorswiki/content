# 可变利率存款证 (CD)
## 什么是可变利率存款证 (CD)？

浮动利率[存款证 (CD)](/certificateofdeposit)是银行和信用合作社提供的一种产品，具有固定期限但利率波动。有几个因素决定了这张 CD 的利率，例如最[优惠利率](/primerate)、[消费者价格指数 (CPI)](/consumerpriceindex) 、[国库券](/treasurybill)或[市场指数](/marketindex)。支付金额的[基础是起始指数和最终指数之间的百分比差异。](/basis-value)联邦[存款保险公司 (FDIC)](/fdic)保护可变利率和其他 CD。

## 了解可变速率 CD

浮动利率 CD 允许投资者将他们的资金存入一个安全、受保护的账户，在其期限内将获得相对适度的利息。在 CD 到期之前，账户持有人通常无法获得所赚取的[利息。](/interest)一些发行人确实提供了允许[提前提取](/earlywithdrawal)资金的免罚金 CD。但是，[利率](/interestrate)可能低于不提供此选项的 CD。

浮动利率 CD 支付的利率可以在证券的整个生命周期内上下浮动。决定浮动利率 CD 利率的确切因素因机构而异。相比之下，[固定利率 CD](/fixed-rate-certificate-of-deposit)具有锁定利率，其基础来自 CD 起源。这意味着利率在整个学期内保持不变。

CD 通常被认为是[更安全](/cd-ladder)[的投资方式之一](/cd-ladder)，尤其是 FDIC 的保护支持其中的大多数。总体而言，CD 是可用的最可靠、低风险的投资选择之一。它们吸引保守、规避风险的储户和投资者。投资 CD 也是[分散](/diversification)[投资组合](/portfolio)风险的绝佳方式。对于新的或谨慎的投资者来说，固定利率 CD 可能是更好的起点，但那些对稍微增加[风险感到满意的人可能想要考虑浮动利率 CD。](/risk)

> Federal Deposit Insurance Corp. 在 FDIC 保险的银行和储蓄协会为每位存款人提供高达 250,000 美元的 CD 保护。

## 可变速率 CD 的特殊注意事项

在考虑具有可变利率的 CD 时，需要记住一些事项。首先，请记住，这些 CD 通常在低利率时期具有最显着的利润潜力。如果您在利率较低时购买浮动利率 CD，那么利率很可能会在整个期限内上升。相比之下，如果 CD 开盘时利率很高，那么它们可能很快就会下降。

此外，请考虑哪些功能对您最重要。对提早提款有高额罚款的浮动利率 CD 可能不如提早提款政策更宽松的固定利率产品那么有吸引力。

尽管听起来很吸引人，但可变速率 CD 也存在某些缺陷。例如，长期的低利率可能会对您的回报产生不利影响，即使以后利率会增加。相比之下，在这种情况下，固定利率 CD 的利润更高。

可变利率 CD 收益也容易受到[通货膨胀的影响](/inflation)。在高通胀时期尤其如此。 CD 本质上会在一段时间内锁定您的资金。如果在那段时间通货膨胀率飙升，而你的回报跟不上它，那么你所持资产的价值总体上会下降。

## 可变速率 CD 示例

假设 CD 基于最优惠利率，这是商业银行向其最有[信誉的](/credit-worthiness)客户收取的费用。该 CD 的发行期限为三年，并有[本金](/principal)偿还的保证。在此期间，最优惠利率从 4% 降至 1%。发行时间和[到期日之间的最优惠利率差](/maturity)（在这种情况下为-3%）是应付给持有人的金额。如果最优惠利率向相反方向移动，从 1% 增加到 4%，那么持有人将从 CD 中获利。

## 底线

浮动利率 CD 为寻求安全、受保护投资的人们提供了多种好处，这些投资将获得相对适度的利息。请记住，在 CD 到期之前，帐户持有人通常无法获得所赚取的利息。尽管听起来很吸引人，但可变速率 CD 也存在某些缺陷。例如，长时间的低利率可能会对您的回报产生不利影响，即使利率稍后会增加。相比之下，在这种情况下，固定利率 CD 的利润更高。

### 什么决定了可变速率 CD 的速率？

有几个因素决定了这张 CD 的利率，例如最优惠利率、消费者价格指数 (CPI)、国库券或市场指数。支付金额的基础是起始指数和最终指数之间的百分比差异。

### 如果我在 CD 到期之前兑换 CD，会发生什么？

通常，提前退出会导致经济处罚。在 CD 到期之前，账户持有人通常无法获得所赚取的利息。一些发行人确实提供了允许提前提取资金的免罚金 CD。但是，利率可能低于不提供此选项的 CD。

### 浮动利率存款证 (CD) 是否由政府投保？

存款证明 (CD) 是更安全的投资方式之一，特别是联邦存款保险公司 (FDIC) 的保护支持其中的大部分。 FDIC 在受 FDIC 保险的银行和储蓄协会为每位存款人提供高达 250,000 美元的 CD 保护。
