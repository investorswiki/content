# 杜邦标识
## 杜邦标识是什么？

杜邦恒等式表示公司的股本回报率 (ROE) 可以表示为其他三个比率的乘积：利润率、总资产周转率和股本乘数。

## 了解杜邦的身份

杜邦身份，俗称杜邦分析，来自杜邦公司，该公司在 1920 年代就开始使用这一理念。杜邦恒等式告诉我们ROE受三件事影响：

1 、[经营效率](/efficiencyratio)，以利润率衡量；

2 、[资产使用](/assetturnover)[效率](/assetturnover)，以总资产周转率衡量；和

3.[财务杠杆](/timevalueofmoney)，以权益乘数衡量。

杜邦恒等式的公式为：

ROE = 利润率 x 资产周转率 x 权益乘数

反过来，这个公式可以进一步分解为：

ROE =（净收入/销售额）x（收入/总资产）x（总资产/股东权益）

> 如果 ROE 不令人满意，杜邦标识可以帮助分析师和管理层找到表现不佳的业务部分。

>

## 杜邦身份示例计算

假设一家公司报告了两年的以下财务数据：

第一年净收入 = 180,000 美元

第一年收入 = 300,000 美元

第一年总资产 = 500,000 美元

第一年股东权益 = 900,000 美元

第二年净收入 = 170,000 美元

第二年收入 = 327,000 美元

第二年总资产 = 545,000 美元

第二年股东权益 = 980,000 美元

使用杜邦恒等式，每年的 ROE 为：

第一年的 ROE = ($180,000 / $300,000) x ($300,000 / $500,000) x ($500,000 / $900,000) = 20%

第二年 ROE = ($170,000 / $327,000) x ($327,000 / $545,000) x ($545,000 / $980,000) = 17%

稍作四舍五入，上述两个 ROE 计算分解为：

第一年 ROE = 60% x 60% x 56% = 20%

第二年 ROE = 52% x 60% x 56% = 17%

您可以清楚地看到 ROE 在第二年有所下降。在这一年中，净收入、收入、总资产和股东权益都发生了变化。通过使用杜邦的身份，分析师或经理可以分析导致这种下降的原因。在这里，他们看到股权倍数和总资产周转率在第二年保持不变。这仅留下利润率作为较低 ROE 的原因。看到利润率从 60% 下降到 52%，而收入实际上在第二年有所增加，这表明公司全年处理费用和成本的方式存在问题。然后，管理人员可以利用这些见解来改进下一年。
