# 交易 ID (TXID)
交易 ID (TXID) 或交易哈希是赋予每个经过验证并添加到区块链的交易的唯一字符串。换句话说，TXID 是一个标识号，用于标记区块链上的每笔交易。

当您进行交易并将其添加到区块链时，它会被赋予一个交易 ID 作为唯一标识符。根据使用的区块链，您可以通过搜索交易 ID 在区块浏览器中找到它。这对于验证基本交易信息很有用，例如发送的金额、转账日期、发送/接收地址以及交易收到的网络确认数量。

例如，这是中本聪发送给 Hal Finney 的第一笔比特币交易的 TXID：

**f4184fc596403b9d638783cf57adfe4c75c605f6356fbc91338530e9831e9e16**

这是著名的[比特币披萨](/bitcoin-pizza)交易的 TXID：

**cca7507897abc89628f450e8b1e0c6fca4ec3f7b34cccf55f3f531c659ff4d79**

[blockchair.com](/bitcoin)或许多其他网站）中自己查找这些交易。

当您从 Binance 等中心化交易所 (CEX) 提取加密货币时，平台会显示您的提款交易的 TXID。在币安，你可以在你的交易历史页面上找到它。如果您单击给定提款的 TXID，它将带您进入该区块链的区块浏览器中的相关交易。

如果您将资金发送到错误的地址（或区块链），则您的提款历史记录中的 TXID 对于开始调查您是否可以收回资金至关重要。即便如此，这也不是保证，最好联系您要退出的平台上的支持人员。
