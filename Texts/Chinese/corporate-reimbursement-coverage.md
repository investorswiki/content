# 公司报销范围
## 什么是企业报销范围？

公司报销保险是公司购买的一种[责任保险](/liability_insurance)，以保护自己免受因针对其董事和高级管理人员的法律诉讼而造成的损失。它是[董事和高级职员 (D&O) 责任保险的三个组成部分之一](/directors-and-officers-liability-insurance)。 （董事和高级职员 (D&O) 责任险还包括 A 面、B 面和 C 面险。公司报销险称为 B 面险。）

## 了解企业报销范围

公司报销范围是董事和高级职员 (D&O) 责任保险的一部分。这种类型的责任保险主要是为了保护个人高管免受损失，但公司报销功能还涵盖公司本身因对个人采取法律行动而可能遭受的任何损失。

对 B 面保险的需求是由公司为其高管承担的赔偿义务驱动的。通常，该义务在公司章程或公司[章程](/articlesofincorporation)中明确规定。该条款要求公司保护因履行对公司的职责而面临法律诉讼的高管或为其支付法律代表费用。这项义务本质上是一般性的，高管们在加入公司时经常会协商他们的赔偿细节，作为个人合同的一部分。这一点很重要，因为公司 D&O 政策的 B 面部分只能涵盖针对个人高管提出的索赔损失，而不是公司本身。

此类法律诉讼的常见原因包括：

- 违反对股东的信托义务。

- 未能满足工作场所安全规定。

- 从竞争公司或前雇主那里盗窃客户。

- 虚假陈述公司资产或财务状况。

## 董事和高级职员 (D&O) 责任范围的类型

D&O 保险的另外两个组成部分称为 A 面和 C 面。当公司无法履行其赔偿义务时，A 面将承担高管的财务损失。这种无能在[破产](/bankruptcy)中最为常见，而 A 面的承保范围迫使保险公司为法律辩护提供资金。

C 面是 D&O 责任保险的三个组成部分中最不常见的，通常只由[上市公司购买](/publiccompany)。 C 面特别保护这些公司免受与公司证券相关的索赔。

投资者经常就其证券价值起诉公司及其经理，声称存在某种形式的管理不善或虚假陈述。发生这种情况时，公司将向 B 方提出索赔，以支付为其高管辩护的费用。假设公司拥有 C 方保单，它还将采取 C 方行动来弥补因对公司本身的诉讼而造成的任何损失。

＃＃ 强调

- 公司报销范围是董事和高级职员 (D&O) 责任保险的一部分。

- 公司购买公司报销保险，以保护自己免受因针对其董事和高级职员的法律诉讼而造成的损失。

- 虽然董事和高级职员 (D&O) 责任保险的结构主要是为了保护个人高管免受损失，但公司报销功能还涵盖了公司本身可能因对个人采取法律行动而遭受的任何损失。
