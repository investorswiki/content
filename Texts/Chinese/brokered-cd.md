# 经纪存单 (CD)
## 什么是代理 CD？

经纪 CD 是由称为经纪人的中间人出售的存款证明。金融机构使用经纪人推销他们的 CD 以帮助他们获得存款。

经纪 CD 的利率往往非常具有竞争力，因为金融机构直接与其他机构竞争您的存款。

## 更深层次的定义

[CD](/certificateofdeposit)是您存放在银行或其他金融机构的存款。它受特定时间的保护，并且在该时间过去之前无法访问。

经纪 CD 遵循相同的概念，但您通过经纪人而不是与您的金融机构合作。经纪人在规定的时间内持有 CD。经纪 CD 实际上是金融机构持有的较大 CD 的一部分。

从您那里购买经纪 CD 的经纪人投资了大量资金，然后在该投资中出售了一定数量的股票。每个购买其中一张经纪 CD 的人都会购买大量的投资资金。

代理 CD 的好处是双重的：

- 作为经纪 CD 的购买者，由于在银行投资的金额较大，您将获得更多的投资利息。在 CD 期限结束时，更多的应计利息等于更多的钱。

- 银行还从经纪 CD 中受益，因为与传统 CD 相比，它一次收到大笔存款，在一天或一段时间内可能不会增加太多。

如果您正在考虑投资经纪 CD，请务必与信誉良好的公司合作，因为联邦存款保险公司不为经纪 CD 提供保险。此外，利率波动或希望在期限结束前退出 CD 的经纪人可能会给投资者带来问题。

## 代理 CD 示例

如果您通过信誉良好的经纪人购买代理 CD，则您是在向经纪人选择的银行或金融机构购买较大 CD 的一部分。您与其他投资者一起购买这张 CD，并在 CD 结束时或其他指定时间从支付中受益。

＃＃ 强调

- 经纪 CD 通常比普通 CD 收益更高，因为它们处于竞争更加激烈的市场中。

- 一家银行仍启动经纪 CD，但将其外包给试图寻找潜在投资者的公司。

- 经纪 CD 通常比传统的银行 CD 提供更大的灵活性。

- 经纪 CD 的灵活性可以让投资者更容易犯错。

- 经纪 CD 是投资者通过经纪公司或销售代表而不是直接从银行购买的 CD。

＃＃ 常问问题

### 银行 CD 比经纪 CD 更好吗？

购买长期经纪 CD 会使投资者面临利率风险。如果投资者在利率上升几年后不得不在二级市场上出售，那么 20 年期经纪 CD 的价格可能会大幅下降。利率下降时存在不同的风险。许多经纪 CD 都是可赎回的 CD，因此如果利率下降，发行人可能会想要赎回并再融资。

### 经纪 CD 比银行 CD 好吗？

这取决于您的财务需求。经纪 CD 的收益率通常高于标准银行 CD。此外，经纪 CD 通常比传统的银行 CD 提供更大的灵活性。例如，经纪 CD 的期限可能比银行 CD 长得多，在某些情况下长达 20 到 30 年。

### 经纪 CD 是否有 FDIC 保险？

答案有点棘手：经纪 CD 在技术上不受 FDIC 保险。但是，经纪人从银行购买的基础 CD 是有保险的。这使得从财务状况良好的公司购买它们变得至关重要。 CD 由联邦存款保险公司在每家银行为每人提供高达 250,000 美元的保险。
