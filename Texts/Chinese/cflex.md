# CFLEX
## 什么是 CFLEX？

CFLEX 是一个用于定制期权的电子交易平台，自 2007 年起由芝加哥期权交易所 ( [CBOE](/cboe) ) 运营。它允许交易者以电子和匿名的方式自动交易具有可定制变量的期权，而不是因为独特的条款而不得不手动处理每笔交易每个选项的。

CFLEX 是 CBOE 灵活期权交易 ( [FLEX](/flexoption) ) 系统的扩展，用于在交易大厅输入和报告非标准期权头寸。

## 了解 CFLEX

CFLEX 由 CBOE 运营，这是一家首屈一指的期权交易所，始于 1973 年，是交易者和投资者交易在交易所上市的衍生证券的第一个市场。 1993 年，芝加哥期权交易所创建了其灵活交易所 (FLEX) 期权产品，能够在期权交易中选择自己的特定条款。

在 FLEX 之前，所有自定义期权交易都必须手动和[场外](/otc)交易 (OTC) 进行，因为它们是非标准化的，并且每个都有自己独特的条款。除了允许买卖双方根据自己的喜好定制合同条款外，FLEX 选项还提供其他好处。这些好处包括防止与场外交易相关的[交易对手风险。](/counterpartyrisk)交易由期权清算公司 ( [OC](/occ) [C](/occ) ) 担保，其他交易所交易的期权也是如此。

CFLEX 允许交易者通过互联网轻松简单地交易复杂的期权。要通过 CFLEX 进行交易，交易者需要与 CFLEX 签署用户协议，然后开始使用基于互联网的平台作为基于浏览器的应用程序或作为互联网 API 进行交易。

CFLEX 通过价格时间匹配算法和实时订单簿为交易者提供实时匿名交易的能力。它还托管一个二级市场来更改条款或用另一个订单取消订单，该功能用于平仓功能，但对于期权，有时在技术上无法像股票那样平仓。

## 自定义期权合约

期权是衍生品合约的一种，[因为](/derivative)[它们](/derivative)来源于股票和债券等直接证券。您直接购买或出售直接证券。购买衍生品赋予您在特定时间以一定金额购买或出售直接证券的权利，但没有义务，而出售衍生品则出售您这样做的权利。衍生品合约的细节远比股票等传统资产的简单买价或卖价复杂得多。交易所交易的期权在这些方面都是标准化的，以保持高效和透明的交易。

具有自定义细节的销售选项需要手动处理每笔销售，因为它需要找到买家，在某些情况下需要找到卖家，并匹配所有这些自定义细节。当一些期权以标准化的细节在交易所上市时，它们就可以进行电子交易。然而，直到 2007 年，技术和期权交易者的想象力才赶上使用电子交易交易定制期权的可能性。
