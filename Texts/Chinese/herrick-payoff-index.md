# 赫里克收益指数
## 什么是赫里克收益指数？

Herrick Payoff Index 是一种技术分析工具，可跟踪价格、交易量和[未平仓合约，](/openinterest)以识别[期货](/futures)和[期权](/option)市场的潜在趋势和逆转。交易者经常使用该指标作为人群心理的衡量标准，并跟踪资金流向以做出前瞻性决策。

## 了解赫里克收益指数

Herrick Payoff 指数考虑了衍生合约的价格、交易量和未平仓合约以产生看涨和看跌[信号](/trade-signal)。由于计算中使用未平仓合约，因此技术指标只能用于期权和期货市场。大多数交易者使用赫里克收益指数来衡量期货和期权市场中的人群心理。在这些市场中，[流动性](/liquidity)低于股票市场，随着时间的推移，[波动的可能性更大。](/volatility)

由于交易者越来越多地买入合约，当价格和持仓量上升时，就会产生看涨的[延续信号。](/continuationpattern)此外，当价格和未平仓合约同时下降时，合约可能会出现看涨[逆转](/reversal)，因为随着价格变得越来越有吸引力，卖压正在下降。

当价格下跌且未平仓头寸增加时，就会产生看跌的延续信号，因为交易员越来越多地进行看跌押注。此外，当价格上涨且未平仓合约下降时，合约可能会出现看跌逆转，这表明看涨交易者正在失去动力。

一般来说，当指标高于中心线时，看涨交易者处于控制之中，而当指标低于中心线时，看跌交易者处于控制之中。但是，交易者应将该指标与其他技术指标或图表[模式结合使用，](/pattern)以最大限度地提高成功交易的几率。

> 使用其他技术指标进行[确认](/confirmation)可以减少 Herrick Payoff Index 虚假信号的影响。

>

## Herrick 回报指数的好处

Herrick Payoff Index 的一个特点是能够在趋势仍在发生时产生退出信号。随着未平仓头寸的下降，赫里克收益指数表明持续的价格趋势可能会逆转。有了这个指标，交易者就有可能在价格开始下跌之前退出。这与许多其他技术信号非常不同，后者是[滞后指标](/laggingindicator)。最值得注意的是，任何基于移动平均线的指标，例如[MACD](/macd) ，总是会滞后于市场的价格走势。

使用 Herrick Payoff Index 在价格下跌之前退出交易会产生更大的收益，因为它的所在地位于高度波动的期货和期权市场。由于这些[衍生品](/derivative)使用了如此高的[杠杆率](/leverage)，即使[基础证券](/underlying-security)价格的小幅下跌也很容易造成重大损失。例如，股价仅下跌 5% 就可能导致[看涨期权](/calloption)的损失超过 25% 。

## Herrick 回报指数的缺点

然而，由于其反转信号的前瞻性，Herrick 回报指数通常是一个不太可靠的进入交易指南。假设价格仍在下降，但持仓量也在下降，如 Herrick Payoff Index 所示。虽然这可能表明下行压力正在放缓，但这并不意味着[反弹](/rally)迫在眉睫。相反，该资产可以稳定价格并在那里停留很长时间。著名交易员杰西·利弗莫尔（Jesse Livermore）称此类证券为“无精打采的流浪者”，他实际上更不喜欢它们，而不是他通常会立即出售的直接损失。

然而，普通交易者通常更害怕损失，这在基于 Herrick Payoff 指数进行交易时是真正的可能性。在价格仍在下跌时购买证券违背了投机者的普遍看法，即不应试图抓住[下跌的匕首](/fallingknife)。事实上，[价值投资者](/valueinvesting)经常试图这样做。然而，价值投资者很少使用像 Herrick Payoff Index 这样的技术指标。

即使有退出信号，赫里克收益指数也存在过早变动的风险。当滞后指标发出虚假信号时，他们的追随者至少有减少损失的安慰。由于 Herrick Payoff Index 在价格仍在上涨时发出退出信号，因此兴趣可能会复苏，而证券可能会继续畅通无阻地上涨。放弃这样的后起之秀会让许多交易者感到沮丧，尤其是那些经验有限的交易者。

＃＃ 强调

- 但是，由于它具有前瞻性，该指数也可能产生误报，因此应与其他技术指标结合使用。

- Herrick Payoff 是一种工具，用于确认衍生品市场的价格趋势或反转，使用价格和数量信息来跟踪资金流动。

- Herrick Payoff Index 的一个优点是即使在趋势市场中也能产生信号。
