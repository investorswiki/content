# 图标
## 什么是 ICON (ICX)？

ICON 是一个旨在促进独立区块链（也称为社区）交互的平台。在 ICON 平台内，社区被定义为遵循单一治理系统的节点网络。 （节点是连接到加密货币网络的计算机。）

加密货币网络（例如比特币和以太坊）被认为是社区，但银行、企业、医院、学校和政府也可以作为社区参与到这个生态系统中。通过[加密货币令牌](/cryptocurrency)ICX支持 ICON 。 ICON 项目由位于韩国的组织 Icon Foundation 于 2017 年创立。

ICON 项目的目标是促进一个平台，使来自不同行业（金融、安全、保险、医疗保健、教育和商业）的实体可以在单个网络上进行交互和交易。

## 了解 ICON (ICX)

2017 年 9 月，Icon 的加密货币 ICX 的首次代币发行 (ICO) 筹集了大约 4300 万美元。此次销售包括 ICON 代币供应的 50%。该项目于 2018 年 1 月正式启动其区块链，ICO 的参与者于 2018 年 6 月收到了他们的 ICX 加密货币。

2019 年，ICON 发布了自己的代币标准 IRC16，允许用户使用 ICON 软件作为基础设施来创建自己的代币化资产和证券。这些代币化资产和证券是代表股权所有权的数字代币（或“硬币”）单位。借助代币化股权，企业可以通过以数字资产形式发行股票来筹集资金。

这就是 ICON 描述其目标的方式：建立一个“数字国家”，其中不同的经济参与者可以在他们选择的规则体系下发布和控制自己的价值形式。根据其网站，Project ICON 的目标是通过“建立世界上最大的去中心化网络之一”来“超连接世界” 。

ICON 项目的灵感来自于现实世界经济的结构方式；企业、非营利组织和政府机构都利用一种通用的交换媒介——一种国家货币——同时仍然保持独特但可互操作的经济参与者。区块链技术和随附的加密货币令牌，称为 ICX，允许分散系统中的参与者在一个中心点，创建一个互连的区块链网络。

[区块链](/blockchain)是一种数据库。它在存储数据的方式上与典型的数据库不同。数据按时间顺序存储在块中：随着新数据的进入，它被输入到一个新的块中。一旦块被数据填充，该块将链接到前一个块。

虽然区块链最常与加密货币相关联，但由于可以将不同类型的信息存储在区块链上，因此该技术还可以应用在其他领域。虽然迄今为止最常见的用途是作为交易账本，但任何信息交换都可以添加到区块链中。涉及证券、货币、商业合同、契约、贷款、知识产权和个人身份的交易可以被标记化。

每个区块链都由特定的网络管理。出于这个原因，在过去，连接不同的区块链被认为是不可能的。 ICON 项目就是对这种联系的一次尝试。与其他要求企业遵守自己的政策和准则的集中支付系统不同，ICON 允许各个社区保留对自己政策的控制权。通过这种方式，ICON 力求减少采用的潜在障碍。

## ICON 的目标

ICON 旨在创建一个更大的网络——由个人网络组成——由加密货币提供支持。参与 ICON 项目的区块链可以使用平台的去中心化交易所 (DEX) 进行货币兑换。

DEX为每个区块链社区设置货币储备，以便社区可以实时交换价值。使用 AI 分析模型为这些交易设置汇率，并使用原生 ICX 硬币作为交易所的中介货币。

ICON项目由五个主要组成部分组成：ICON共和国、ICON社区、社区代表（C-Rep）、社区节点（C-Nodes）和公民节点。 ICON 社区是一个单一治理系统内的节点网络；他们可以使用自己的治理结构、节点数量和特征进行操作。社区的决策方法也可能不同。例如，虽然比特币采用共识驱动的方法，但其他社区（如金融机构）可能会采用分层方法。

社区节点是每个社区的基础设施；他们维护社区的区块链。社区代表是从每个社区中选出的节点，并且受信任可以与 ICON 共和国进行交互。通过社区代表，每个社区的区块链都链接到 ICON 共和国。社区代表因其工作而获得 ICX。

ICON 共和国是网络的管理结构。它也是区块链运营的决策委员会。 ICON 共和国治理由社区代表投票决定。一个有用的类比是将 ICON 共和国想象成美国，并将每个 ICON 社区想象成一个独立的州。

ICON 共和国的行动并不能决定独立社区本身的治理。但是，ICON 共和国确实管理着新 ICX 加密货币的铸造过程。

公民节点是区块链中的参与者，他们没有投票权，但可以在社区内、与其他社区以及 Icon Republic 进行交易。

ICON Republic 的区块链被称为“nexus”。为连接提供动力的技术是一种称为循环链的算法。循环链连接作为 ICON 共和国一部分的社区。这些社区——当它们组合在一起时，它们被称为一个联盟——确定了一组共同的规则，允许不同的区块链有效地协同工作。

管理独立区块链如何与 ICON 共和国的区块链（nexus）交互的一组规则称为区块链传输协议（BTP）。

最后，与[以太](/ethereum)[坊一样](/ethereum)，任何人都可以通过区块链的公共渠道使用 ICON 构建去中心化应用程序，称为 DAPP。此外，这些应用程序在 ICON 的 DAPP 商店上架后，任何拥有公民节点的人都可以下载和使用它们。

## ICON 的批评

虽然 ICON 制定了连接不同社区的雄心勃勃的计划，但它在实现这一目标方面面临许多挑战。首先，许多投资者习惯于传统的中心化加密货币交易所。虽然这些传统的中心化交易所确实不符合加密货币的去中心化意识形态，但到 2021 年，这些交易所仍然是加密货币市场不可或缺的一部分。许多投资者只购买他们可以通过他们最喜欢的交易所购买的加密货币。这可能会给 ICX 带来问题。鉴于 ICON 的目标，许多交易所可能不想向其客户提供 ICX。

高水平的竞争会增加特定加密货币无法流行的风险。与其他加密货币一样，ICON 面临来自其他加密货币的激烈竞争。

＃＃ 强调

- ICON 是一种区块链技术和网络框架，旨在允许独立的区块链相互交互。
