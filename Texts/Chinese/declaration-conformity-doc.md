# 符合性声明 (DoC)
## 什么是符合性声明 (DoC)？

符合性声明 (DoC) 是一份文件，说明产品（通常是电子产品）符合其必须合法遵守的标准，例如安全法规。

## 了解符合性声明 (DoC)

DoC 证明消费品已由认可的实验室或测试机构使用批准的方法进行测试，以确保其在销售前完全可操作且安全。例如，它可以证明产品不含致癌物质，不会分解，并且如果它是供儿童使用的，则不会造成窒息危险。

从经济上讲，DoC 为用户和消费者提供了对产品质量的一些保证，同时为卖家提供了关于产品的一般适销性的一些可信赖的东西——当然，前提是它实际上符合标准。这降低了购买和销售商品的[交易成本](/transactioncosts)，否则消费者可能由于产品的[信息不对称而无法信任](/asymmetricinformation)，并且在某些情况下为生产者提供了与消费者面临的某些风险相关的安全网或责任盾。

正式的 DoC 记录了所有考虑到批准产品的评估。如果事实证明先前批准的产品实际上不符合其标准，则 DoC 会给出一张准确评估的地图以及由谁评估的地图。这使得追查哪里出了问题以及谁（如果有的话）应该承担责任变得很容易。

> 产品测试可以由政府机构自己进行，也可以由官方认可的第三方测试机构进行，具体取决于产品及其制造或销售的司法管辖区。

>

## 符合性声明 (DoC) 要求

DoC 通常有两种形式。首先是详细说明产品标准和测试的正式文件或官方认可的报告，其次是产品、包装或营销材料上的印章、标志或标记，表明产品符合客户和购买公众的标准.

在美国，一些联邦法律和许多州法律为消费品制定了安全标准和其他标准。美国国会于 1972 年颁布了《消费品安全法》(CPSA)，该法案成立了美国[消费品安全委员会](/consumer-product-safety-commission)(CPSC)，并确定了其制定适当产品安全标准的权力。 2008 年，乔治·W·布什总统签署了《消费品安全改进法案》(CPSIA)，以促进安全法规，特别是儿童产品，并对不合规的制造商实施更严厉的处罚。

### 符合性声明中包含的组件

DoC 由产品的制造商或分销商创建，并应由有权代表制造商做出决定并分配必要资源以确保正确完成过程的人员签署。除了日期、制造商名称和地址等基本信息外，DoC 通常还包括以下项目：

- 产品的具体型号和/或序列号

- 适用于该产品且必须遵守的指令的完整列表

- 用于评估产品的所有标准的日期列表

- 产品符合必要标准的声明

- 授权签字以及签字人的姓名和职位

根据产品和相应监管或测试机构设定的要求，可能需要其他信息。 DoC 必须被翻译成任何销售该产品的国家的语言。

## 符合性声明 (DoC) 示例

在美国，[联邦通信委员会](/fcc)(FCC) 的标志被视为 DoC，出现在符合其监管标准的电子产品上。虽然官方 DoC 是制造商创建的用于显示其测试细节并证明其有效性的文件，但 FCC 标签本身被认为是美国公众的 DoC，因为它只印在通过这些测试的产品上.

在[欧盟](/europeanunion)，通常使用的 DoC 是欧洲符合性或 CE 标志。就像 FCC 印章一样，产品上的 CE 标志证明它已通过批准的测试，因此人们会相信它可以安全使用。

＃＃ 强调

- 在美国，联邦通信委员会 (FCC) 负责颁发 DoC。

- 官方文档是产品批准之前完成的所有评估的记录，因此可以轻松跟踪故障的责任和位置。

- 符合性声明 (DoC) 证明消费品已由认可的实验室或测试机构进行测试，以确保其在销售前完全可操作且安全。
