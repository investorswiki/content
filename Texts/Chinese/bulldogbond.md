# 斗牛犬债券
## 什么是斗牛犬债券？

牛头犬债券是一种由非英国公司发行的[外国债券，旨在](/foreignbonds)从英国投资者那里以英镑 ( [GBP ) 筹集资金。](/gbp)

## 了解斗牛犬债券

如果一家公司认为它会在这个市场上获得有吸引力的利率或如果它需要外汇，它可能会选择进入一个外国市场。当一家公司决定进入国外市场时，它可以通过发行[外国债券来实现](/foreignbonds)，这些债券是以目标市场的货币计价的债券。简单地说，外国债券是外国实体在国内市场以国内市场[货币发行的一种](/currency)[债券，](/bond)作为筹集资金的手段。外国债券主要用于为[发行人](/issuer)提供进入其自身以外的另一个资本市场以筹集资金的机会[。](/issuer)

牛头犬债券是一种由非英国公司发行的外国债券，旨在从英国投资者那里筹集英镑资金。例如，一家希望在英国[债券市场获得投资资金的加拿大公司](/bondmarket)可能会选择发行牛头犬债券。如果相关费用（债务）也是英镑，而收入可以抵消它们，那么汇率风险就会降低。

鉴于英国斗牛犬是英格兰的国家象征，这些以英镑计价的债券被称为斗牛犬债券。根据国际清算银行的数据，英镑在全球交易量最大的货币和[储备货币](/reservecurrency)类别中均排名第四，仅次于美元 ( [USD](/usd) )、欧元 ( [EUR](/euro) ) 和日元 ( [JPY](/jpy-japanese-yen) ) 。

## Bulldog 债券特征

- 牛头犬债券由一家银行或国内银行财团承销，以英镑计价。

- 当英国的利率低于外国公司的国内利率时，发行牛头犬债券。发行牛头犬债券可以降低发行人的利息支出或[借贷成本](/costofdebt)。

- 寻求在地域上分散其投资组合的美国投资者可以购买该债券，但这样做他们会承担[外汇](/foreignexchangerisk)[风险](/foreignexchangerisk)，即英镑相对于美元价值发生不利变化的风险。但是，交易所的有利走势可能会为投资者带来经济收益。

- 牛头犬债券类似于[洋基债券](/yankeebond)，以美元计价，非美国公司可以在美国出售这些债券以筹集美元资金。

- 其他外国债券包括[袋鼠债券](/kangaroo_bond)、[枫叶债券](/maple_bond)、斗牛士债券、[武士债券](/samuraibond)和伦勃朗债券。

＃＃ 强调

- 这些以英镑计价的外国债券被称为牛头犬债券，因为英国斗牛犬是英格兰的国家象征。

- Bulldog 债券是一种在英国交易的债券，由有兴趣从英镑赚取收入流的买家购买。

- 牛头犬债券是一种由寻求从英国投资者那里筹集英镑资金的非英国公司发行的外国债券。
