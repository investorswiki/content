# 那恰
##什么是Nacha？

Nacha 是电子系统的管家，该系统连接所有美国银行账户并促进它们之间的资金流动。据该组织称，2019 年通过其[自动票据交换所](/ach)(ACH) 网络转移了 55.8 万亿美元。

Nacha 以前称为国家自动票据交换所协会，是一个由使用其网络的金融机构资助的非营利性协会。 Nacha 和 Interactive Financial eXchange (IFX) 论坛于 2018 年合并，后者是一个开发金融数据系统规范的国际行业协会。

## 了解纳查

ACH 网络支持数十亿电子金融交易，包括直接存款、社会保障和政府福利报表、电子账单支付、个人对个人 (P2P) 和[企业](/btob)[对企业](/btob)(B2B) 支付。

> Nacha 于 2018 年与 Interactive Financial eXchange (IFX) 论坛合并。

>

通过其监管和规则制定功能，Nacha 为电子支付系统的有效运行奠定了基础，同时致力于更新技术和实施新的支付系统。

##那恰的历史

Nacha 成立于 1974 年，由几个地区机构合并而成。它最初是美国银行家协会的一部分。

它在直接工资存款、电子福利存款和自动信用卡交易等创新的发展和标准化方面发挥了重要作用。

最近，它接管了根据 2010 年平价医疗法案处理 B2B 健康保险支付的任务。 Nacha 现在负责管理医疗保健电子资金转账 (EFT) 标准，该标准支持健康计划和提供者之间符合 HIPAA 的交易，允许信息随付款一起传输，从而简化了提供者的会计程序。

## ACH 网络

ACH 网络通过一个无处不在的支付系统连接所有美国金融机构，该系统旨在安全有效地将资金和信息从一个银行账户转移到另一个银行账户。

Nacha 制定商业惯例的规则和准则，并参与新应用程序的开发。它还制定和监控质量和风险管理控制。

尽管它不是政府机构，但 Nacha 与包括美联储、美国财政部和州银行当局在内的各种政府机构密切合作，以确保美国金融机构使用的电子支付系统的完整性。

2014 年，Nacha 成立了支付创新联盟，作为支付行业和 ACH 网络的代言人。﻿该联盟由全球支付生态系统中的数百家公司和组织组成。它提供有关支付系统现代化、趋势、标准、安全性和持续创新等主题的讨论、辩论、教育和网络。

##更快的付款

2019 年 11 月，Nacha 推出了一项名为 Faster Payments Playbook 的计划，旨在让消费者“以近乎即时的资金可用性随时随地向任何人付款” 。

Nacha 还提供教育和认证服务；与金融机构、企业和政府的行业合作；和宣传资源。

Nacha 的 API 标准化行业组 (ASIG) 支持美国金融服务行业内标准化[应用程序编程接口(API) 的进步和使用。](/application-programming-interface)
