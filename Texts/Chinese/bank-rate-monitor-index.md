# 银行利率监测指数
## 什么是银行利率监控指数？

Bankrate Monitor Index 是美国银行和信用合作社[存款账户](/deposit)支付的货币市场[利率指数。](/interestrate)这些账户包括支票账户、储蓄账户、货币市场账户和 CD 账户。

1996 年，Bankrate Monitor Index 上线并发展成为一个受欢迎的个人理财网站 Bankrate.com。

## Bankrate Monitor 指数的工作原理

银行利率监测指数由 Robert K. Heady 创建，以响应联邦政府通过 1982 年的《加恩-圣日耳曼存款机构法案》，该法案放松了对储蓄和贷款协会的管制，并允许银行发行[可调整利率的抵押贷款](/arm)和货币市场账户。该指数的目的是帮助消费者了解存款利率的普遍趋势，以便他们可以用自己的钱购买最优惠的价格。

1996 年上线后，该公司于 2000 年更名为 Bankrate Inc.。2011 年，它进行了[首次公开募股 (IPO)](/ipo) ，并在[纽约证券交易所 (NYSE) 上市](/nyse)，股票代码为 RATE。 2017 年 11 月，该公司在被在线营销公司 Red Ventures 收购后恢复私有制。

今天，Bankrate 是金融行业领先的利率和个人理财新闻研究和数据聚合器之一。它向纽约时报、今日美国和华尔街日报等主要媒体发布每日金融研究和市场状况。通过其个人理财网站 Bankrate.com，它提供了一系列面向消费者的内容，例如抵押贷款计算器、财务规划工具和其他资源。

## Bankrate Monitor Index 的真实示例

Bankrate Monitor Index 的现代继任者是 Bankrate 通过其网站为消费者发布的各种利率调查和其他功能。其中包括对信用卡、抵押贷款、储蓄账户和其他流行金融产品的评级。

例如，Bankrate 定期汇总有关不同[信用卡的信息](/creditcard)，并根据其年费、信用要求、利率和会员福利等特征对它们进行排名。读者可以指定哪些功能对他们最重要，并根据他们的分数过滤掉不同的信用卡提供商。对于抵押贷款，读者可以访问类似的信息，同时还可以根据他们的邮政编码和他们的首付预算来过滤报价。

＃＃ 强调

- Bankrate Monitor Index 是一个跟踪流行金融工具利率的指数。

- 它后来发展成为 Bankrate.com，一个专注于个人理财资源的网站。

- 今天的网站为信用卡、抵押贷款、信用额度和汽车保险等产品提供利率数据、计算器和其他资源。
