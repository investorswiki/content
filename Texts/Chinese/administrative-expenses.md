# 行政费用
## 什么是行政费用？

管理费用是组织产生的与制造、生产或销售等特定核心职能没有直接关系的费用。这些[间接费用](/overhead)与整个组织有关，而不是与个别部门或业务单位有关。

## 了解管理费用

行政开支可能包括高级管理人员的薪金以及与一般服务或用品相关的成本；例如，法律、会计、文书工作和信息技术。这些成本往往与企业的商品或服务生产没有直接关系，通常不计入毛利率。

公司为执行基本操作（例如，管理工资单或医疗保健福利）、提高监督和效率和/或遵守法律法规而产生管理费用。在[损益表中](/incomestatement)，管理费用低于销售[成本(](/cogs) [COGS](/cogs) )，并且可能与[其他费用（如一般费用或销售费用）合并显示](/general-and-administrative-expenses)。

一些管理费用在性质上是[固定](/fixedcost)的，因为它们是作为业务运营基础的一部分而产生的。无论发生的生产或销售水平如何，这些费用都会存在。其他管理费用是[半可变的](/semivariablecost)。例如，企业将始终使用一些最低水平的电力来保持灯亮。除此之外，它还可以采取措施减少电费。

由于企业可以在不直接影响其销售或生产的产品的情况下消除管理费用，因此这些成本通常是预算削减的首要因素。管理层有强烈的动机保持相对于其他成本的较低管理费用，因为这使企业能够更有效地利用[杠杆。](/leverage)销售与管理费用比率有助于公司衡量有多少销售收入用于支付管理成本。

公司可以从其纳税申报表中扣除业务运营所需的合理、普通和必要的管理费用。这些费用必须在正常业务过程中发生，并在发生当年扣除。

## 其他类型的行政费用

某些员工（例如会计和 IT 员工）的工资和福利被视为管理费用。所有高管薪酬和福利均被视为管理费用。建筑租赁、保险、订阅、公用事业和办公用品可归类为一般费用或管理费用。

根据要折旧的资产，[折旧](/depreciation)费用可分为一般费用、管理费用或销售（营销）费用。组织也可以选择将咨询和法律费用作为管理费用。但是，[研发](/randd)(R&D) 成本不被视为管理费用。

为了全面了解与运营某些业务部门相关的成本，公司可能会根据收入、费用、面积或其他指标的百分比向其每个部门分配管理费用。在内部，这允许管理层就扩大或减少单个业务部门做出决定。

## 管理费用示例

例如，如果 XYZ 公司每月在电费上花费 4,000 美元并将其记录为管理费用，它可能会根据每个单独部门占用的平方英尺来分配成本。认为：

- 生产设施2,000平方英尺

- 制造设施占地 1,500 平方英尺

- 会计办公室750平方英尺

- 售楼处750平方英尺

公司占地5000平方。电费可以按如下方式分配：

- 生产：1,600 美元或 (2,000 / 5,000) x 4,000 美元

- 制造：1,200 美元或 (1,500 / 5,000) x 4,000 美元

- 会计：600 美元或 (750 / 5,000) x 4,000 美元

- 销售额：600 美元或 (750 / 5,000) x 4,000 美元

＃＃ 强调

- 一定程度的管理费用将始终作为运营的必要组成部分。

- 管理费用是为支持业务运作而发生的费用，但与特定产品或服务的生产没有直接关系。

- 行政费用通常是最先被确定为削减预算的费用，因为它们不会直接影响公司的主要业务功能。

- 管理层可以根据收入、费用或其他措施的百分比向其业务部门分配管理费用。
