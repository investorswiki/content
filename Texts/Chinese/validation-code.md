# 验证码
## 什么是验证码？

[信用卡](/creditcard)正面或背面的一系列三个或四个数字。它旨在为在线或通过电话进行的信用卡交易提供额外的安全保障。

大多数信用卡发卡机构将其验证码放在卡背面签名面板的最右侧。但是，在美国运通 (AXP) 卡上，验证码印在卡的正面。

## 验证码如何工作

随着在线购物的不断普及，身份盗用和其他形式的信用卡欺诈的威胁变得越来越严重。为降低这种风险而采取的一项措施是在进行信用卡购买时使用验证码。

在典型的交易中，客户将被要求提供他们的姓名、账单地址、卡号、到期日期和验证码。尽管其中许多细节，例如姓名和地址，可以从其他来源获得；卡号、有效期和验证码理论上只能从卡本身获得。作为一项附加措施，验证码通常印在卡的背面，这使得潜在的窃贼更难从一张信用卡照片中收集所有必要的信息。

为了进一步加强这些安全措施，消费者保护法禁止商家在购买后存储客户的验证码——尽管不道德的卖家仍可能非法记录这些信息。持卡人在使用[销售点 (POS)](/point-of-sale)终端进行付款时必须输入的个人识别码 (PIN) 提供了额外的保护措施。

## 验证代码的真实示例

尽管验证码等安全措施提高了身份盗用或使用被盗信用卡购物的难度，但它们不太可能阻止足够积极的小偷。在实践中，信用卡欺诈近年来持续攀升，2020 年报告的案件超过 393,207 起。美国是迄今为止受影响最严重的国家，占全球案件的近 34%。

客户购买后，商家不得存储卡安全码，这为信用卡盗窃提供了额外的保护。尽管如此，验证码仍可能被盗，持卡人应像保护卡号和有效期一样保护其卡的验证码。验证码是一个关键数据，可以让窃贼使用他人的卡进行欺诈交易。

但是，如果小偷确实使用了被盗的卡，根据公平信用计费法案 (FCBA)，持卡人的责任仅限于 50 美元，具体取决于报告盗窃的时间。客户如果发现自己的卡丢失，或检测到可疑或未经授权的购买或其他活动，应立即联系其信用卡发卡机构报告问题并提醒他们可能存在欺诈案件。然后，发卡机构可以取消或停用该卡。

＃＃ 强调

- 它由印在信用卡正面或背面的三个或四个字母的代码组成。

- 验证码是为减少信用卡欺诈而部署的安全措施之一。

- 根据最新的尼尔森报告，信用卡欺诈案件持续上升，2019 年达到近 290 亿美元，预计到 2027 年将增至 380 亿美元左右，其中美国占大部分，最近报道损失近 34%。
