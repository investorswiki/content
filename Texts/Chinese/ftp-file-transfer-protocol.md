# 文件传输协议 (FTP)
## 什么是文件传输协议 (FTP)？

术语文件传输协议 (FTP) 是指涉及通过网络在设备之间传输文件的过程。当一方允许另一方通过 Internet 发送或接收文件时，该过程有效。最初用作用户在两个物理设备之间进行通信和交换信息的一种方式，现在它通常用于将文件存储在[云](/cloud-computing)中，通常是远程保存的安全位置。

[企业或个人](/business)可以使用 FTP将文件从一个计算机系统传输到另一个计算机系统，或者由网站从其服务器上传或下载文件。

## 文件传输协议 (FTP) 的工作原理

文件传输协议允许个人和企业与他人共享电子文件，而不必在同一个空间。这可以使用 FTP 客户端或通过云来完成。无论选择哪种方式，双方都需要有效的互联网连接。

大多数[网络](/web-20)浏览器都带有 FTP 客户端，使用户能够将文件从他们的计算机传输到服务器，反之亦然。一些用户可能想要使用第三方 FTP 客户端，因为其中许多提供了额外的功能。可免费下载的 FTP 客户端示例包括 FileZilla Client、FTP Voyager、WinSCP、CoffeeCup Free FTP 和 Core FTP。

许多人以前使用过 FTP，甚至没有意识到这一点。如果您曾经从网页下载过文件，那么您就使用过 FTP。第一步是登录，可以自动登录，也可以手动输入用户名和密码。 FTP 还要求您通过特定端口号访问 FTP 服务器。通过 FTP 客户端访问 FTP 服务器后，您现在可以传输文件了。并非所有公共 FTP 服务器都要求您登录，因为某些服务器允许您匿名访问它们。

如上所述，FTP 最初是作为一种在两台物理计算机之间发送和接收文件的方式而开发的。但随着技术的变化，用户可以通过云端执行文件传输。使用云可以方便、安全地进行传输（这可以保护个人和公司免受[数据泄露](/data-breach)），并且成本相对较低。

> 术语 FTP 客户端是指允许您将文件传输给另一方的软件。

## 特别注意事项

文件传输协议是决定计算机和计算系统如何在 Internet 上运行的许多不同协议之一。其他此类协议包括：

- **超文本传输协议 (HTTP)：** 旨在通过网络传输数据

- **互联网消息访问协议 (IMAP)：** 提供对来自共享服务的公告板或电子邮件消息的访问

- **网络时间协议 (NTP)：** 通过网络同步计算机上的时钟时间

FTP 使 Internet 上的计算机能够来回传输文件。因此，它是当今构建和维护网站的必备工具。

### 在 FTP 客户端中寻找什么

各个 FTP 客户端提供不同的功能，允许用户修改他们上传和下载文件的方式。例如，如果您使用 FileZilla，该程序允许您设置文件的[带宽](/bandwidth)限制。这使您能够控制上传和下载速度，如果您同时管理多个文件传输，这将很有帮助。

您可能希望在 FTP 客户端中寻找的其他功能包括公钥身份验证、设置文件压缩级别的能力或使您能够使用文件掩码搜索服务器的工具。

## 文件传输协议 (FTP) 示例

FTP 软件的设置相对简单。 FileZilla 是一个免费的、可下载的 FTP 客户端。输入您要访问的服务器的地址、端口和访问服务器的密码。

一旦授予访问权限，用户在其本地系统上的文件以及访问的服务器将是可见的。用户可以从服务器下载文件到本地系统，也可以从本地系统上传文件到服务器。他们还可以更改服务器上的文件，只要他们有适当的授权即可。
