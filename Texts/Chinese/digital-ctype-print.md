# 数码 C 型打印
## 什么是数字 C 型打印？

数字 C 型或显色彩色照片是由数字曝光系统创建的任何照片照片，与传统的暗房或模拟技术不同。数码 C 型打印是通过将感光材料暴露于 LED 或激光来开发的，然后使用类似于传统摄影的方法对材料进行清洗。 C代表显色。

## 数字 C 型打印解释

在传统的暗房或模拟设置中，放大机将摄影底片的图像投射到一张相纸上，同时控制焦点、图像的强度和曝光时间。放大镜是一种光学仪器，是一种类似于幻灯机的摄影工具。

传统的暗房工艺不同于数字 c 型或显色印刷工艺。对于数字 c 型打印，通常由放大机完成的工作由计算机完成，技术人员控制相同的因素：焦点、强度和曝光时间。在这种情况下，纸张使用激光或 LED 而非传统灯泡进行曝光。 LED 代表发光二极管，是一种在电流流过时发出光的光源，与标准灯泡不同。

图像曝光后，无论是通过传统方式还是数字方式，下一步大致相同。它仍然包括所谓的湿化学过程。包含图像的纸张在照相显影剂中进行处理，然后经过漂白定影，最后在水中洗涤以去除处理化学物质。然后将图像晾干，然后可以对其进行扫描、裁剪、编辑和修改。

## 数字 C 型打印不同于 Giclée 打印

数字 C 型或显色照片是由数字文件而非底片制成的传统图片或照片照片。显色印花有时会与微喷印花相混淆，但它们是不同的。 Giclée 印刷品是在不使用任何化学物质或光敏性的情况下制作的。这种类型的打印将基于颜料的墨水与高质量的档案型纸张相结合，从而产生特别高质量的喷墨打印。

## 数字 C 型打印与喷墨打印

数字 c 型打印与喷墨打印不同，因为喷墨打印使用细小的墨滴而不是光源，例如激光。用于数字 C 型打印的机器可能比喷墨打印机贵得多，并且往往用于商业环境。数码 C 型打印的寿命估计也比基于颜料的打印更短，并且可以用这种工艺打印的材料数量更加有限。

用于数字 C 型打印的其他术语包括照片实验室、数字 C、激光显色、数字 RA-4、化学染料、实验室打印或 C 打印。

## 真实世界数字 C 型打印示例

数字 C 打印是摄影实验室打印。它们由所谓的 minilabs 生产。常见的例子包括 Fuji Frontier，这是一种看起来像传统计算机打印机但设计用于制作更高质量打印件的高端机器。 7700 系列每小时可打印多达 2,360 张高分辨率打印件，每小时可打印多达 620 张 8"X10" 打印件。其他能够生产高端数字 C 型打印件的处理器示例包括 LightJet 或 Lambda 等宽幅照片打印机，它们使用 Fuji Crystal Archive 或 Kodak Endura 等纸张。

＃＃ 强调

- 对于数字印刷，传统上由称为放大机的光学设备完成的工作由计算机完成。

- 数字 C 型或显色彩色打印是一种以数字方式创建的打印，与传统方式相反。

- 一旦图像被投射到一张纸上，纸就会在显影剂中处理，然后漂白，然后洗掉处理化学品。
