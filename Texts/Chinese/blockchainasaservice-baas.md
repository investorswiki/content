# 区块链即服务（BaaS）
## 什么是区块链即服务（BaaS）？

区块链即服务 (BaaS) 是第三方创建和管理基于云的网络，用于构建区块链应用程序业务的公司。这些第三方服务是区块链技术不断发展的领域中相对较新的发展。区块链技术的应用已经远远超出了其在加密货币交易中最著名的用途，并且已经扩展到解决各种安全交易。因此，需要托管服务。

## 了解区块链即服务（BaaS）

BaaS 基于[软件即服务 (SaaS)](/software-as-a-service-saas)模型并以类似的方式工作。它允许客户利用基于云的解决方案在区块链上构建、托管和运营自己的[区块链](/blockchain)应用程序和相关功能。同时，基于云的服务提供商保持了基础架构的敏捷性和可操作性。

作为更大区块链生态系统的发展，BaaS 被视为促进区块链在企业中的采用。

BaaS 领域的主要参与者包括：

- 微软与 ConsenSys 合作，于 2015 年在 Microsoft Azure 上推出以太坊区块链即服务。

- 亚马逊推出了 Amazon Managed Blockchain，这是一项使用包括以太坊和 Hyperledger Fabric 在内的开源框架“让创建和管理可扩展的区块链网络变得容易”的服务。

- R3，一个由全球金融机构组成的财团，生产了一个名为 Corda 的分布式财务分类账。

- PayStand，专门在公司之间发送和接收付款。

消费者和企业越来越愿意适应区块链技术。但是，创建、配置和运营区块链以及维护其基础设施所涉及的技术复杂性和运营开销通常会成为障碍。

BaaS 提供外部服务提供商来设置所有必要的区块链技术和基础设施，但需要付费。创建后，提供程序将继续为客户端处理复杂的后端操作。

BaaS 运营商通常提供支持活动，例如带宽管理、适当的资源分配、托管要求和数据安全功能。 BaaS 运营商让客户可以专注于核心工作：区块链的功能。

事实上，BaaS 提供商的角色类似于网络托管提供商的角色。网站创建者在他们自己的个人计算机上创建和运行所有网站内容。他们可能会雇用支持人员或注册外部托管服务提供商，如 Amazon Web Services 或 HostGator。这些第三方公司负责基础设施和维护问题。

BaaS 可能是促使区块链技术更广泛、更深入地渗透到各个行业部门和企业的催化剂。与其创建和运行自己的区块链，企业，无论大小，现在都可以简单地将技术复杂的工作外包并专注于其核心活动。

＃＃ 强调

- 区块链即服务 (BaaS) 是指第三方基于云的基础设施和管理，用于构建和运营区块链应用程序的公司。

- BaaS 可能是导致区块链技术广泛采用的催化剂。

- BaaS 的功能类似于一种网络主机，为基于区块链的应用程序或平台运行后端操作。
