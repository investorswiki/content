# 甜原油
## 什么是甜原油？

低硫原油是指提取的[原油](/crude-oil)，发现其硫含量非常低。由于硫会降低各种精炼石油产品（如汽油、柴油甚至塑料）的产量，因此它被认为是一种有价值且有效的原油来源。

低硫原油可以与不太理想的[含硫原油形成对比](/sourcrude)，后者具有高水平的硫含量。

## 甜原油的工作原理

原油主要由碳和氢组成，分别占其化学成分的 85% 和 12%。其余元素由硫、氧、氮和氦组成。根据微量元素的组成，[精炼原油的过程在精炼](/oil-refinery)后剩余的副产品（或“渣油”）的量方面可能会有很大差异。

如果油的硫含量保持非常低，则油最容易使用。为了帮助制定这种优质油的行业标准定义， [NYMEX](/nymex)交易所提出了原油低硫油的定义，规定其硫含量不得超过 0.42%。

尽管今天可以使用精确的技术来评估石油样品的硫含量，但过去几代石油勘探者会通过亲自品尝样品来推断样品的硫含量。如果样品尝起来略带甜味并散发出令人愉悦的气味，则意味着它的硫含量较低。带有臭鸡蛋气味的酸味表明硫含量高。正是通过这种古老的测试方法，才有了“甜原油”和“酸原油”这两个术语。

## 其他注意事项

除了控制硫含量外，原油的质量还受其密度的影响。根据[美国石油学会 (API)](/american-petroleum-institute)制定的密度标准，原油也可以根据其相对于水的密度被识别为“轻质”或“重质”。例如，密度等于水的原油被定义为“API 比重”为 10。

API 的密度等级是倒置的，这意味着 API 比重大于 10 的油会漂浮在水面上，被称为“轻质原油”。相反，API 比重低于 10 的油会沉入水中，被称为“重质原油”。尽管并非所有轻质原油都是甜的，但大多数重质原油都是酸的，因为它通常含有大量的硫以及镍等金属。低密度（或“轻”）油是可取的，因为它更容易提炼、蒸馏和运输。

## 甜原油的真实示例

将这些元素结合在一起，我们可以看到最理想的原油类型是轻质低硫原油。这是因为它非常高效，在其精制过程中留下的残留物最少，同时由于其低密度而易于蒸馏和运输。

轻质低硫原油最著名的例子可能是[西德克萨斯中](/wti)质原油或 WTI 原油，它[比布伦特原油](/northseabrentcrude)和迪拜原油等其他主要类型的原油更轻、更甜。它在北美受欢迎的部分原因是它在北美生产和精炼，同时也在纽约商品交易所等商品期货市场上活跃交易。

＃＃ 强调

- 当与低密度相结合时，低硫原油是一种备受追捧的石油，价格高，被称为轻质低硫原油。

- 低硫原油是一种按其低硫含量分类的原油。

- 在北美，低硫原油的一个流行例子是西德克萨斯中质原油 (WTI) 原油，该原油在纽约商品交易所 (NYMEX) 交易活跃。
