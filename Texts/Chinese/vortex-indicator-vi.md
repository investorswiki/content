# 涡流指示器 (VI)
## 什么是涡流指示器 (VI)？

漩涡指标 (VI) 是由两条线组成的指标 - 一条上升趋势线 (VI+) 和一条下降趋势线 (VI-)。这些线通常分别为绿色和红色。涡旋指标用于发现趋势[反转](/reversal)并确认当前趋势。

## 了解涡流指示器 (VI)

漩涡指标最初由 Etienne Botes 和 Douglas Siepman 开发，他们在 2009 年版的“股票和商品技术分析”中引入了这一概念。漩涡指标基于两条趋势线：VI+ 和 VI-。

## 涡流指示器计算

该指标的计算分为四个部分。

1.真实范围（TR）是最大的：

- 电流高减去电流低

- 当前高点减去前一收盘价

- 当前低点减去前一收盘价

2.[上升](/uptrend)趋势和下降趋势：

- VM+ = 当前高点的绝对值减去之前的低点

- VM- = 当前低点减去先前高点的绝对值

3.参数长度（n）

- 确定参数长度（通常在 14 到 30 天之间）

- 总结最后 n 个周期的真实范围，VM+ 和 VM-：

- 最后 n 个周期的真实范围之和 = SUM TRn

- 最后 n 个周期的总和 VM+ = SUM VMn+

- 最后 n 个周期的总和 VM- = SUM VMn−

4. 创建趋势线 VI+ 和 VI-

- 总和 VMn+/总和 TRn = VIn+

- 总和 VMn-/总和 TRn = VIn−

- 每天重复此过程形成 VI+ 和 VI- 趋势线。

当价格走势波动时，使用 VI- 和 VI+ 交叉的传统应用会导致许多错误的交易信号。增加指标中使用的周期数以减少这一点，例如，使用 25 个周期而不是 14 个周期。

## 推论

漩涡指标通常与其他反转趋势模式结合使用，以帮助支持反转信号。它被集成到大多数[技术分析](/technicalanalysis)软件程序中。 VI+ 和 VI- 通常在烛台图下方独立绘制。下图提供了一个示例，其中的线条指示[烛台](/candlestick)图表上不断变化的趋势信号。

当 VI+ 低于 VI- 然后越过 VI- 以在趋势线中占据顶部位置时，就会出现上升趋势或买入信号。当 VI- 低于 VI+ 并穿过 VI+ 以在趋势线中占据顶部位置时，就会出现下降趋势或卖出信号。总体而言，顶部位置的趋势线通常决定了[证券](/security)是处于上升趋势还是下降趋势。
