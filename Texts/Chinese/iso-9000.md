# ISO 9000
## 什么是 ISO 9000？

ISO 9000 是一套质量管理标准，作为国际公认的企业和其他组织绩效基准而制定。它由国际标准化组织 (ISO) 在来自许多国家的标准专业人士的意见下创建。

质量管理是监督所有过程的行为，这些过程在产品或服务的创建和交付中实现和保持所需的卓越水平。这包括确定质量政策、创建和实施质量计划和保证，以及质量控制和改进。它也被称为[全面质量管理](/total-quality-management-tqm)（TQM）。

> - ISO 9000 是一套国际公认的质量保证和管理标准。

> - 由国际标准化组织出版，旨在鼓励生产符合全球可接受质量水平的商品和服务。

> - ISO 9000 列出了质量管理体系的最佳实践、指南和标准词汇。

> - ISO 9001 是 ISO 9000 的一部分，由寻求 ISO 认证的企业或其他组织的行动项目组成。

## 了解 ISO 9000

制定 ISO 9000 标准是为了帮助制造商有效记录质量体系要素，这些要素需要实施以维持高效的质量体系。它们越来越多地应用于任何组织或行业。

ISO 9001 现在被用作服务部门、教育和政府部门的质量管理基础，以帮助组织满足其客户、满足法规要求并实现持续改进。

ISO 9000 系列或标准系列最初由[国际标准化组织](/international-organization-for-standardization-iso)(ISO) 于 1987 年发布。它们首先在欧洲流行，然后在 1990 年代传播到美国。随着世界质量保证观的演变，标准也进行了修订。

当前版本的 ISO 9000 和 ISO 9001 于 2015 年 9 月发布。

> ISO 9000 列出了质量管理体系的最佳实践、指南和标准词汇。

>

## ISO 9000 标准

ISO 9000 详细列出了质量管理体系的基本原理和词汇。

ISO 9000 的可操作部分是 ISO 9001。获得认证的组织被认为符合 ISO 9001。获得该认证是一个需要一年多的过程，并且需要大量文件来证明符合标准。

ISO 9000 系列包含以下标准：

- ISO 9001:2015：质量管理体系——要求

- ISO 9000:2015：质量管理体系——基础知识和词汇（定义）

- ISO 9004:2009：质量管理体系——管理组织的持续成功（持续改进）

- ISO 19011:2011：审核管理系统指南

## 如何获得 ISO 9001 认证

ISO 认证由第三方组织提供，而不是由 ISO 本身提供。

寻求认证的公司或其他组织将首先实施 ISO 9000 中包含的质量管理建议。然后进行内部审核或获得外部审核员的服务，以评估其是否符合标准。然后它会召集一个被授权审查其合规性并颁发认证的组织。

ISO 网站提供有关选择认证机构的信息。

在美国，ANSI 国家认可委员会 (ANAB) 提供有关其已认可开展认证服务以符合 ISO 和其他质量标准的公司的信息。

> 全球约有 100 万个组织已获得 ISO 9001 认证。

>

## ISO 9000 的好处

采用 ISO 标准完全是自愿的。获得认证可能是一项耗时且昂贵的工作。然而，满足 ISO 标准有很多好处。

该组织列举了获得认证的几个好处，包括增加消费者和利益相关者的信心，保持竞争优势，以及在公司开展业务的任何司法管辖区更容易满足政府规定的法规。

例如，Toro 公司在其网站上有一个部分专门介绍它所获得的用于生产其商业产品和灌溉系统的管理实践的 ISO 9001 认证。

它是全球约 100 万家经认证符合 ISO 9000 标准的公司之一。在线数据库提供对大多数认证组织的访问。

## ISO 9000 的历史

ISO 9000 可能是国际标准化组织制定的最著名和最广泛采用的标准集。

ISO 成立于 1947 年，是一个独立的非政府机构，根据 160 个成员国的意见制定和发布技术、工业和商业标准。

其标准旨在建立管理实践，以全球可接受的水平生产安全、可靠和高质量的产品和服务。

该组织已发布了超过 24,000 项推荐的标准。最值得注意的是，它为希望采用有效环境管理体系的公司和组织发布了一套环境标准[ISO 14001 。](/iso-14001)

其他标准分为许多类别，包括健康和安全、能源管理、食品安全和信息技术安全。

## 什么是质量管理？

一般来说，质量管理通过实施短期举措来关注长期目标。

质量控制 (QC) 是质量管理和保证的关键部分，企业通过它寻求确保在减少或零错误的情况下保持或改进产品质量。

质量控制要求企业创造一个管理层和员工都力求完美的环境。这是通过培训人员、为产品质量创建基准以及测试产品以检查统计上显着的变化来完成的。

质量控制的一个主要方面是建立明确的控制。这些控制有助于[标准化](/standardization)生产和对质量问题的反应。通过指定由哪些人员完成哪些生产活动来限制错误空间，从而减少员工参与他们没有接受过足够培训的任务的机会。

＃＃ 常问问题

### ISO 9000 和 ISO 9001 有什么区别？

标题经常互换使用。 ISO 9001 是 ISO 9000 的一部分，由达到 ISO 9000 合规性的主要行动项目组成。

### ISO 9000 和 ISO 14000 有什么区别？

ISO 9000 和 ISO 14000 都是由国际标准化组织创建和发布的标准集。 - ISO 9000 致力于质量管理。它旨在帮助公司和其他组织确保他们创建的产品和服务以及他们用来创建它们的流程符合高质量和诚信的高标准。 - IDO 14000 是一套标准，适用于希望他们使用的流程和他们创造的产品对环境产生最小不利影响的公司和其他组织。

### ISO 9000 的要求是什么？

ISO 9000 要求涉及任何组织的质量管理基础。它们详细规定了在产品或服务的规划、生产和交付过程中必须满足的标准。ISO 9000 中包含的主要主题领域包括制造流程、设备维护、记录保存、员工培训和客户关系。 ISO 9000 背后的组织是为了满足国际商业的需求而成立的，该标准基于来自许多国家/地区的监管机构的意见。其目的是鼓励生产符合任何购买国家标准的产品和服务。

### ISO 9000 的成本是多少？

寻求 ISO 9000 认证的小型组织可能会花费 10,000 到 15,000 美元准备和获得认证。成本包括 ISO 9000 手册（约 240 美元）和审核员费用（每天约 1,300 美元）。成本取决于组织的规模和复杂性，以及他们已经建立质量管理体系的程度。
