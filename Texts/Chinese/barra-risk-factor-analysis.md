# 巴拉风险因素分析
## 什么是 Barra 风险因素分析？

Barra 风险因素分析是由 Barra Inc. 创建的多因素模型，用于衡量与市场相关的证券的总体风险。 Barra 风险因素分析包含 40 多个数据指标，包括收益增长、股票换手率和高级债务评级。然后，该模型衡量与三个主要组成部分相关的风险因素：行业风险、不同投资主题的风险和公司特定风险。

## 了解 Barra 风险因素分析

[投资者](/investor)和投资组合经理在评估市场或投资组合时仔细审查的一个因素是投资风险。在决定投资什么资产时，识别和衡量投资风险是最重要的步骤之一。这是因为所承担的风险水平决定了资产或资产组合在交易周期结束时的回报水平.因此，最广泛接受的财务原则之一是[风险和回报之间的权衡](/riskreturntradeoff)。

投资组合经理可能用来衡量投资风险的一种方法是评估一系列广泛因素对各种资产或证券表现的影响。使用因子模型，证券的收益产生过程是由各种常见的[基本因子的](/fundamentals)[存在](/fundamentals)以及资产对每个因子的独特敏感性驱动的。由于几个重要因素可以在很大程度上解释投资的预期风险和回报，因此可以使用因素模型来评估投资组合的回报有多少可归因于每个共同因素敞口。因子模型可以分为单因子模型和多因子模型。一种可用于衡量投资组合风险的多因素模型是 Barra 风险因素分析模型。

Barra 风险因素分析由 Barra Inc. 的创始人 Bar Rosenberg 开创，并在 Grinold 和 Kahn (2000)、Conner 等人 (2010) 和 Cariño 等人 (2010) 中进行了详细讨论。它在其模型中包含了许多可用于预测和控制风险的因素。多因素风险模型使用许多代表投资特征的关键基本面因素。其中一些因素包括收益率、盈利增长、波动性、流动性、[动量](/momentum)、规模、[市盈率](/price-earningsratio)、杠杆和增长[；](/price-earningsratio)用于描述投资组合或资产的风险或回报的因素，从定量但未指定的因素转变为易于识别的基本特征。

Barra 风险因素分析模型使用单个[风险](/var)价值 (VaR) 数字衡量证券的相对风险。这个数字代表 0 到 100 之间的百分位等级，相对于美国市场，0 是波动最小的，100 是波动[最大](/volatility)[的](/volatility)。例如，风险价值数为 80 的证券被计算为比市场及其特定行业中 80% 的证券具有更大的价格波动水平。因此，如果亚马逊的 VaR 为 80，则意味着其股票的价格波动幅度大于 80% 的股票市场或公司所在行业。

＃＃ 强调

- Barra 风险因素分析是一个多因素模型，由 Barra Inc. 创建，用于衡量与证券相关的整体风险，相对于市场。

- Barra 风险因素分析包含 40 多个数据指标，包括盈利增长、股票换手率和高级债务评级。
