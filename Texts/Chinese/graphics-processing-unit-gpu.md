# 图形处理单元 (GPU)
## 什么是图形处理单元 (GPU)？

图形处理单元 (GPU) 是能够渲染图形以在电子设备上显示的芯片或电子电路。 GPU 于 1999 年被引入更广泛的市场，以其用于提供消费者在现代视频和游戏中所期望的流畅图形而闻名。

## 图形处理单元 (GPU) 的工作原理

视频和 gs 中的图形由多边形坐标组成，这些坐标转换为位图（称为“渲染”的过程），然后转换为显示在屏幕上的信号。这种转换需要图形处理单元 (GPU) 具有强大的处理能力，这也使得 GPU 在[机器学习](/machine-learning)、人工智能和其他需要大量复杂和精密计算的任务中非常有用。

## 图形处理单元 (GPU) 的历史

1999 年，Nvidia 推出了 Geforce 256，这是第一个广泛使用的 GPU。 Nvidia 将 GPU 定义为“具有集成变换、照明、三角形设置/裁剪和渲染引擎的单芯片处理器，每秒能够处理至少 1000 万个多边形。” GeForce 256 通过优化 3D 游戏性能改进了其他处理器的技术。

虽然 Nvidia 在 GPU 市场上仍然占据主导地位，但该技术已经有了很大的改进。在 2000 年代，Nvidia 发布了 GeForce 8800 GTX，其纹理填充率高达每秒 368 亿次。

如今，GPU 重新流行起来。由于人工智能和加密货币的出现，它们的使用已扩展到新的行业。 GPU 在建立更广泛地访问更高质量的虚拟现实游戏方面也发挥了作用。

## GPU 与 CPU

在 1990 年代后期 GPU 出现之前，图形渲染由中央处理器 (CPU) 处理。当与 CPU 结合使用时，GPU 可以通过从 CPU 承担一些计算密集型功能（例如渲染）来提高计算机性能。这加快了应用程序的处理速度，因为 GPU 可以同时执行许多计算。这种转变还允许开发更先进和资源密集型的软件。

在 GPU 或 CPU 中处理数据由内核处理。处理单元的核心越多，计算机完成任务的速度就越快（并且可能更高效）。 GPU 使用数千个内核来并行处理任务。 GPU 的并行结构与 CPU 的并行结构不同，CPU 使用更少的内核来顺序处理任务。 CPU 可以比 GPU 更快地执行计算，这使得它在基本任务上表现得更好。

## 特别注意事项

“GPU”一词经常与“显卡”互换使用，尽管两者是不同的。显卡是一种硬件，包含一个或多个 GPU、子板和其他允许显卡运行的电子组件。

但是，GPU 可以集成到主板中，也可以在显卡的子板上找到。最初，高端电脑是唯一配备显卡的电脑。今天，大多数台式计算机通常使用带有 GPU 的单独显卡来提高性能，而不是依赖内置在主板中的 GPU。

## GPU 和加密货币挖掘

虽然 GPU 最初受到视频编辑和电脑游戏爱好者的欢迎，但加密货币的快速增长[创造](/cryptocurrency)了一个新市场。这是因为加密货币挖掘需要数千次计算才能将交易添加到[区块链](/blockchain)中，这可以通过访问 GPU 和廉价的电力供应来获利。

近年来，两家著名的显卡制造商 Nvidia Corp. (NVDA) 和 Advanced Micro Devices Inc. (AMD) 的销售额和收入都因加密货币挖矿而迅速增长。

这产生了令非采矿客户感到沮丧的副作用，他们看到价格上涨和供应枯竭。因此，零售商有时会限制个人可以购买的显卡数量。虽然[比特币](/bitcoin)等更受欢迎的加密货币的矿工已转向使用称为专用集成电路 (ASIC) 的专用且更具成本效益的芯片组，但图形处理单元仍用于挖掘鲜为人知的货币。

> 加密货币的普及导致 GPU 的大量短缺。来自 Verge 的报道计算得出，在 eBay 等网站上，GPU 的售价是其市价的两到三倍。

## GPU 公司的例子

Advanced Micro Devices (AMD) 和 Nvidia (NVDA) 是 GPU 市场上的两大巨头。下面我们来看看这两家公司。

### Advanced Micro Devices (AMD)

AMD 是最值得信赖的显卡生产商之一。这家制造商于 1969 年在硅谷作为一家初创公司起步，开发高性能计算和可视化产品。 AMD 于 2006 年收购领先的视频卡制造商 ATI 进入 GPU 市场。从那时起，AMD 和 Nvidia 一直是 GPU 市场的主导者。截至 2021 年 5 月，AMD 的市值为 973 亿美元。自 2013 年以来，AMD 已出货超过 5 亿颗 GPU，控制着 17% 的 GPU 市场份额。

AMD 将 GPU 市场的重点放在 PC 游戏上，是全球游戏玩家的最爱。

### 英伟达 (NVDA)

英伟达是 1999 年第一家将 GPU 推向世界的公司。历史上第一款 GPU 被称为 Geforce 256。1999 年也是英伟达以每股 12 美元的价格[首次公开募股(IPO) 的一年。](/ipo)截至 2021 年 5 月，该股票的交易价格约为每股 645 美元。

英伟达的市值为 4048 亿美元，控制着 13% 的 GPU 市场份额。

英伟达在高级 GPU 市场拥有相当大的影响力。根据英伟达的网站，“世界排名前 10 位的超级计算机中有八台现在使用英伟达 GPU、InfiniBand 网络，或两者兼而有之。英伟达为最新列表中的 TOP500 系统中的 346 台提供动力。”

英伟达自己的超级计算机Selene排名世界第五，是世界上最快的工业超级计算机。

## 图形处理单元 (GPU) 常见问题解答

### GPU和VGA有什么区别？

GPU 是能够用于渲染图形以在电子设备上显示的芯片或电子电路，而 VGA 或视频图形阵列连接器是用于传输视频信号和计算机视频输出的物理设备。

###你如何超频你的GPU？

在超频之前，请确保彻底清洁您的设备并为您的软件安装任何更新和错误修复。由于技术更新，超频相当简单。只需安装 Afterburner 等软件，然后让系统开始工作。安装完成后，运行游戏基准测试以测试新软件。

### 什么是 GPU 缩放？

GPU 缩放功能使用户能够根据显示器的分辨率调整游戏的纵横比。一些用户认为这种调整纵横比将进一步提高显示器的图像质量。
