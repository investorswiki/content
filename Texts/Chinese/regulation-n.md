# 条例 N
## 什么是规则 N？

条例 N 是由消费者金融保护局 (CFPB) 和联邦贸易委员会 (FTC) 制定的规则，强制遵守2009 年[信用卡责任和责任与](/credit-card-accountability-responsibility-and-disclosure-act-of-2009)[披露法案](/credit-card-accountability-responsibility-and-disclosure-act-of-2009)(CARD Act) 和[多德-弗兰克华尔街法案2010 年](/dodd-frank-financial-regulatory-reform-bill)[改革和消费者金融保护法案](/dodd-frank-financial-regulatory-reform-bill)（多德-弗兰克法案）。

## 了解法规 N

[抵押信贷产品的](/mortgage)金融产品进行了规范，将它们定义为由住宅或其他不动产担保的任何信贷产品，提供给消费者供家庭、个人或家庭使用。它还进一步规范了抵押贷款经纪人如何向消费者展示其抵押信贷产品。[联邦贸易委员会](/ftc)(FTC)监督对 N 条例的遵守情况。

规则 N 也称为抵押行为和实践广告规则或 MAP 规则，因为它规定了抵押贷款人、服务商、[经纪人](/mortgagebroker)、广告代理和其他人如何宣传抵押服务。该规则禁止抵押贷款经纪人、贷方、服务和广告机构向消费者发送的抵押广告和其他商业通讯中的欺骗性索赔。

欺骗性声明的示例包括宣传低固定利率，但未具体说明该利率仅适用于非常短暂的介绍期，以及将支付利率与[利率混为一谈](/interestrate)。或者，贷方可能无法让消费者知道付款利率可能无法支付每月到期的利息，从而导致负[摊销](/amortization)，即由于每月未付利息被添加到本金中，贷款金额会随着时间的推移而增加。

欺骗性的抵押贷款广告也可能无法讨论重要的[贷款](/loan)条款，或者暗示所涉及的抵押贷款人与政府机构有关联，但实际上并非如此。

N 条例旨在杜绝这种行为，并根据 2008 年金融危机和随后的[大萧条后通过的法律实施](/great-recession)。房地产信贷的过度扩张，包括所谓的[掠夺性借贷](/predatory_lending)行为，是导致这场危机的 2000 年代房地产泡沫的主要组成部分，促使立法者、消费者权益团体和金融业人士采取行动并进行改革，以防止类似情况发生将来不会发生滥用行为。

> 被发现违反 N 条例的抵押贷款人和广告商可能面临民事处罚。

>

## N 条例禁止的欺诈性抵押贷款索赔示例

N 条例与 FTC 法案的第 5 条相平行，该条禁止虚假广告和广告中的误导性声明。条例 N 禁止的欺骗性声明的一些示例包括以下虚假陈述：

- 与抵押产品相关的消费费用的性质、金额或存在；

- 提供的抵押贷款类型；

- 抵押协议的条款、付款、金额或其他要求，包括与[保险](/insurance)和税收相关的要求；

- 利率、付款金额、期限长度和其他抵押条款的可变性；

- 每月还款的百分比将用于支付利息、贷款金额或到期总金额；

[再融资](/refinance)或修改抵押贷款或其条款的可能性，或消费者这样做的能力；

抵押产品可能带来的任何[提前还款罚款；](/prepaymentpenalty)

[违约](/default2)的可能性以及哪些情况构成违约；

- 消费者居住在所购买住宅的权利；

- 就抵押信贷产品向客户提供的任何专家建议或咨询服务的性质、内容和可用性；

- 有关抵押产品的商业通讯或广告的来源。

＃＃ 强调

- 违反条例 N 包括虚假广告和广告中的误导性声明。

- Reg N 适用于受 FTC 监管的人，但 FTC 和 CFPB 共享执法权力。

- CFPB 和 FTC 制定了 N 条例，以执行 CARD 法案和多德-弗兰克法案。
