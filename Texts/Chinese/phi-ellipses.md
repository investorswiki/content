# 披椭圆
## 什么是 Phi 椭圆？

Phi 椭圆是一种斐波那契交易工具，用于帮助投资者识别价格模式、趋势和潜在的趋势反转。 phi 椭圆类似于一个椭圆，围绕价格波绘制，突出价格的路径。当价格在 phi 椭圆之外移动时，一个新的价格模式或趋势方向正在开始。

## Phi 椭圆告诉你什么？

Phi 椭圆是一种鲜为人知的交易工具，用于检测价格模式，因此[投资者](/investor)可以监控当前和过去的价格模式，以确定何时买卖。 Phi 椭圆用于股票、股指、货币、期货，也可能适用于其他市场。

phi 椭圆的主要用途之一是通过分析椭圆的变化形状来识别价格变动的基本结构。 Phi 椭圆也可以链接在一起。例如，在长期上升趋势中，每个主要的上升波都可以包含在一个菲利普椭圆中，每个[回调也可以包含在这个椭圆中](/pullback)。当价格突破趋势波 phi 椭圆时，表明可能开始回调。当价格走出回调 phi 椭圆时，表明价格可能开始向趋势方向移动。然后可以在整个趋势周围绘制一个 phi 椭圆，以指示长期趋势何时可能结束——当价格移动到 phi 椭圆之外时。

phi 椭圆由计算机或交易/图表软件包绘制，因为它基于精确测量。

椭圆是交易平台上常用的绘图工具。它们本质上是椭圆形的，但不一定基于斐波那契比率。椭圆通过 Fischer 变换数学术语转换为 phi 椭圆，在 Robert Fischer 和 Jens Fischer 的**Candlesticks、Fibonacci 和 Chart Pattern Trading Tools** 和 **The New Fibonacci Trader** 中讨论。

## 如何使用 Phi 椭圆的示例

phi 椭圆的主要功能是突出当前市场模式，例如包括趋势方向移动和回调的趋势。如果 phi 椭圆围绕上升趋势绘制，则当价格突破 phi 椭圆底部时，可能会退出多头交易。

当价格高于 phi 椭圆顶部时，交易者可以退出[空头头寸。](/short)

在趋势期间，可能需要绘制和调整许多 phi 椭圆，以适应价格走势的展开。这意味着使用此工具可以成为一门艺术。

Phi 椭圆在交易或图表平台上并不常见。下面的标准普尔 500 ETF 信托 (SPY) 日线图显示了椭圆，只是为了突出该指标的一般用途。这些是**不是** phi 椭圆。

## 趋势线和 Phi 椭圆的区别

[趋势](/trendline)线也可以在潜在趋势变化发生时发出信号。趋势线是沿着上升趋势的[摆动低点](/swinglow)或下降趋势的摆动高点绘制的。当价格超过趋势线时，一旦绘制，它可能预示着可能的趋势变化。像 phi 椭圆一样，趋势线通常需要调整，并且可以根据短期和长期趋势绘制。

## Phi 椭圆的优缺点

Phi 椭圆能够在一次分析中整合价格和时间。 phi 椭圆可以根据价格变动动态调整。 Phi 椭圆可以在所有时间范围内使用，但由于不断的绘制和调整，[当日交易时可能会变得很麻烦。](/daytrader)

当价格在 phi 椭圆角度的相反方向上移出 phi 椭圆时（上升趋势和价格跌破 phi 椭圆下方），它有助于发出潜在趋势变化或多头退出点的信号。这可能发生在其他技术指标发出趋势变化信号之前。

Phi 椭圆随着时间的推移而发展。尽管所有 phi 椭圆都有一个共同的外观，但它们的最终形式各不相同，变粗、变细、变长或变短。通常，投资者可以发现较小的趋势，这些趋势已经是较大趋势的一部分。因此，一旦确定了更大的趋势，phi 椭圆就可以用作其中的进入点和退出点。

在 phi 椭圆的不利方面，交易者无法将其编程为完全自动化的[交易信号](/trade-signal)；交易者需要大量的纪律才能有效地使用该工具，并且交易者必须熟悉将 phi 椭圆应用于价格图表，以便利用他们提供的信息。该工具需要调整，并且可能需要根据交易的时间框架频繁绘制或重绘。用 phi 椭圆进行交易似乎很简单，但要绘制它们并正确交易它们需要大量练习。

＃＃ 强调

- 当价格上升到向下倾斜的 phi 椭圆之上时，它发出买入信号。

- Phi 椭圆是在许多图表平台上发现的椭圆绘图工具的数学修改版本。

- 可以在所有价格波上绘制 phi 椭圆，较大的 phi 椭圆可能包含较小的 phi 椭圆。

- 当价格跌破向上倾斜的 phi 椭圆时，表示卖出。

- Phi 椭圆在大多数图表平台上并不常见，因为它们基于精确的比率。
