# 远期外汇合约 (FEC)
## 什么是远期交易合约 (FEC)？

远期外汇合约 (FEC) 是一种特殊类型的[场外](/otc)交易 (OTC) 外币 (forex) 交易，用于兑换外汇市场上不常交易的货币。这些可能包括次要货币以及[被冻结](/blockedcurrency)或不可兑换的货币。涉及这种被冻结货币的 FEC 称为不可交割远期或 NDF。

从广义上讲，[远期合约](/forwardcontract)是两方之间在未来特定时间交换一对货币的合约协议。这些交易通常发生在[现货合约](/spottrade)结算日期之后的日期，用于保护买方免受货币价格波动的影响。

## 了解远期交易合约 (FEC)

远期外汇合约（FEC）不在交易所交易[，](/exchange)标准数量的[货币](/currency)不在这些协议中交易。尽管如此，除非双方同意，否则不能取消。

参与合同的各方通常对对冲[外汇](/foreign-exchange)[头寸](/foreign-exchange)或采取[投机](/speculation)头寸感兴趣。所有 FEC 均规定了货币对、名义金额、结算日期和交割汇率，并规定以定价日的现行即期汇率完成交易。

因此，合同的汇率是固定的，并指定了未来的特定日期，从而使相关各方能够更好地为未来的金融项目进行预算，并提前准确地了解他们在特定的未来日期从交易中获得的收入或成本。 FEC 的性质保护双方免受货币未来[即期汇率的意外或不利变动](/forex-spot-rate)。

[货币对](/currencypair)的远期汇率通常可以在未来长达 12 个月内获得，或者对于四个“主要货币对”而言长达 10 年。

>

一般来说，大多数货币对的远期[汇率](/exchangerate)可以在未来长达 12 个月内获得。有四对货币被称为“[主要货币对](/majors)”。这些是美元和欧元；美元和日元；美元和英镑；还有美元和瑞士法郎。对于这四对，可以获得长达 10 年的汇率。

许多供应商也提供短至几天的合同时间。尽管可以定制合同，但大多数实体不会看到 FEC 的全部好处，除非将最低合同金额设置为 30,000 美元。

## 特别注意事项

最大的远期外汇市场是人民币 ( [CNY](/cny-china-yuan-renminbi) )、印度卢比 ( [INR](/inr) )、韩元 ( [KRW](/krw-korean-won) )、新台币 ( [TWD](/twd-taiwan-dollar) )、巴西雷亚尔 ( [BRL](/brl-brazilian-real) ) 和俄罗斯卢布 ( [RUB](/rub-russian-ruble) )。与此同时，最大的场外交易市场在伦敦，纽约、新加坡和香港也有活跃的市场。除了活跃的[NDF市场](/ndf)外，包括韩国在内的一些国家还限制了在岸远期市场。

FEC 交易的最大部分是针对美元 ( [USD](/usd) ) 进行的。也有使用欧元 ( [EUR](/euro) )、日元 ( [JPY](/jpy-japanese-yen) ) 以及在较小程度上使用英镑 ( [GBP](/gbp) ) 和瑞士法郎 ( [CHF](/chf) ) 的活跃市场。

## 远期汇率计算和示例

可以使用四个变量来计算合约的远期汇率：

- S = 货币对的当前即期汇率

- r(d) = 本币[利率](/interestrate)

- r(f) = 外币利率

- t = 合同时间，以天为单位

远期汇率的公式为：

>

> 远期利率 = S x (1 + r(d) x (t / 360)) / (1 + r(f) x (t / 360))

>

例如，假设美元 (USD) 和加元 ( [CAD](/cad-canadian-dollar) ) 的即期汇率为 1 加元买入 0.80 美元。美国三个月利率为0.75%，加拿大三个月利率为0.25%。在这种情况下，三个月美元/加元 FEC 汇率将计算为：

>

> 三个月远期利率 = 0.80 x (1 + 0.75% * (90 / 360)) / (1 + 0.25% * (90 / 360)) = 0.80 x (1.0019 / 1.0006) = 0.801

>

由于超过 90 天的费率造成的差异是百分之一美分。

＃＃ 强调

- FEC 以可定制的条款和条件在场外交易，很多时候参考非流动性、被封锁或不可兑换的货币。

- FEC 被用作对冲风险，因为当外汇交易无法进行时，它可以保护双方免受货币未来即期汇率意外或不利变动的影响。

- 远期外汇合约 (FEC) 是两方之间为实现货币交易而达成的协议，通常涉及在外汇市场上不易获得的货币对。
