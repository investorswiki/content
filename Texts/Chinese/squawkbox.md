# 电话亭
## 什么是 Squawk Box？

squawk box 是经纪公司的分析师或交易员在交易大厅或办公桌上使用的对讲机扬声器的术语。在[投资银行](/investment-banking)、[股票经纪公司](/brokerage-company)和交易所大厅都可以找到 Squawk 盒子。 squawk box 允许公司的分析师和交易员与公司的经纪人进行交流。

在过去的几年里，squawk box 对讲机已经被在线即时通讯平台所取代，用于实时传播信息和交易订单。

## 了解 Squawk 框

也被称为“hoot-n-hollers”，金融公司使用 squawk box 向他们的经纪人通报当前分析师的建议、市场事件和大宗交易的信息。这种沟通渠道有助于让经纪人了解重要的市场因素，并允许公司指导其经纪人的交易。

与电话不同，squawk box 为在线的各方提供持续、始终在线的音频连接，而无需拨入。参与者当然可以决定将他们的麦克风静音，以防止环境音频通过扬声器。

尽管技术带来了许多其他形式的交流，但电话亭仍然是许多投资银行和经纪公司的必备工具。但是，它们不再是**物理**盒子，而是电子渠道或横幅通知。

## Squawk Box 和分析师建议

许多人通过收听 squawk box 来了解分析师评级的当前趋势。传统上，分析师对证券的建议范围包括“买入”、“持有”和“卖出”。买入表明该证券被低估并且很划算，而卖出表明它可能被高估了。除了“表现不佳”和“跑赢大盘”评级之外，每个评级现在都存在多个术语（“卖出”可以是“强力卖出”，而“买入”可以是“强力买入”）。

分析师将研究公开财务报表，听取公司电话会议，并直接与经理以及公司客户交谈，以深入了解他们的内部运作和现状。

分析师经常使用[贴现现金流](/dcf)(DCF) 模型来支持他们的定性分析。 DCF 是一种估值方法，它依赖于公司未来的自由现金流预测。分析师将使用所需的年利率对这些进行折现。然后使用现值估计来评估投资潜力。如果分析师通过 DCF 分析得出的价值高于当前的投资成本，那么机会可能是一个很好的机会。

## Squawk Box 和大宗交易

收听 squawk box 的另一个原因是了解当前大宗交易的情况。大宗[交易](/blocktrade)或大宗订单是提交出售或购买大量证券的文件。大宗交易经常发生在各方之间，通常在公开市场之外，以减少它们对证券价格的影响。一般来说，10,000 股股票（不包括低价股）或价值 200,000 美元的债券被视为大宗交易。了解谁在进行大宗交易或大宗订单可以帮助银行员工了解[新发行](/newissue)的供需情况。

## 底线

Squawk box 大多是在过去，但你可能会偶尔看到一个。如今，交易者们都盯着屏幕，通常只戴一个耳机，并且可以轻松收听内部“squawk”频道或观察他们的屏幕以获取交易通知。

＃＃ 常问问题

### 什么是最好的股票不和谐？

随着在线聊天室和论坛如此受欢迎，这些工具被用于众包信息以用于交易也就不足为奇了。 Discord 和 Reddit 是许多最大的在线股票市场社区的所在地，讨论从金融新闻到组织社区交易的方方面面。最好的聊天室是您找到专业人士并仔细审查他们的建议的地方。这些地方通常更适合讨论，而不适合可靠的财务信息。

### 什么是 Squawk 服务？

squawk 服务是持续的音频或文本流，通常基于订阅，让听众及时了解财务信息、大宗交易以及服务想要传达给听众的任何其他信息，以便他们快速访问财务数据。

### Squawk Box 是什么意思？

CNBC 上有一个名为“Squawk Box”的热门节目，它的名字来源于一个用于在交易员和分析师之间交流内部交易的物理盒子。尽管多年来随处可见，但横幅通知和屏幕交易警报等更快服务的出现已经使 squawk box 过时了。
