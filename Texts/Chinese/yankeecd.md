# 洋基存款证明（CD）
## 什么是洋基存款证明 (CD)？

洋基[存款证明 (CD)](/certificateofdeposit)是一种由外国银行分行在美国发行的 CD。 Yankee CD 以美元计价，被外国银行用来从美国投资者那里筹集资金。

## Yankee CD 的工作原理

在美国经营的外国银行通常需要获取美元，以便向美国客户提供信贷或履行以[美元 (USD)](/usd)计价的义务。为了帮助筹集这些美元资本，外国银行有时会通过称为洋基债券的特殊 CD 接受美国客户的[存款](/deposit)。

与传统的 CD 一样，Yankee CD 是一种储蓄账户，在特定投资期结束时返还初始投资之前支付利息。尽管投资者通常可以在此日期之前提取资金，但这样做可能会招致[提前提取的罚款](/withdrawal-penalty)。一般而言，CD 的期限介于 1 个月至 5 年之间，期限较长的账户支付的[利息较高。](/interest)

除了它们由外国银行提供之外，洋基 CD 和普通 CD 之间的另一个主要区别是它们的最低投资规模。通常，洋基 CD 的最低面值为 100,000 美元，因此适合较大的投资者。此外，Yankee CD 仅提供不到一年的短期[到期。](/maturity)因为它们不是由美国机构发行，洋基 CD 不受[联邦存款保险公司 (FDIC)的保护](/fdic)，通常要求投资者在整个到期期间“锁定”他们的资金。

## Yankee CD 的真实世界示例

Yankee CD 通常由在美国设有分支机构的外国银行在纽约发行。它们要么由外国银行自己直接出售，要么通过一个或多个注册[经纪商](/broker-dealer)[间接出售](/broker-dealer)。提供洋基 CD 的外国银行最常见的原产国是日本、加拿大、英国和西欧国家。这些银行通常使用通过 Yankee CD 筹集的资金向其美国企业客户提供信贷。

据里士满联储称，洋基 CD 于 1970 年代初首次发行，最初的收益率高于国内 CD。当时的外资银行知名度不高，由于会计规则不同，财务信息匮乏，其信用质量难以评估。

随着投资者对外国银行的认知和熟悉程度的提高，外国银行为其洋基 CD 支付的溢价有所下降。这种资金成本差异被外国银行免除美联储[准备金要求部分抵消](/requiredreserves)，直到 1978 年国际银行法生效。

豁免还有助于建立在 1980 年代初期稳步增长的洋基 CD 市场。在 1990 年代初期，洋基 CD 迅速增长，因为 1990 年 12 月取消了对期限少于 18 个月的非个人定期存款的准备金要求。此前，外国银行为美元贷款提供 3% 的美联储准备金要求给持有洋基 CD 的美国借款人。

＃＃ 强调

- Yankee CD 的到期期限比普通 CD 短，通常不到一年。在此期间，客户可能无法在不面临高额提早提款罚款的情况下提款。

- 它们由寻求从美国储户筹集资金的外国银行发行。

- Yankee CD 是一种面向较大投资者的储蓄工具。
