# 活动X
## 什么是 ActiveX？

ActiveX 是 Microsoft (MSFT) 的一个软件框架，它允许应用程序通过 Web 浏览器彼此共享功能和数据，而不管它们是用什么编程语言编写的。ActiveX 附加组件允许早期的 Web 浏览器嵌入多媒体文件或交付向用户提供软件更新。

Microsoft 于 1996 年推出了 ActiveX。许多ActiveX 控件仅在 Windows 上运行，并且与 Internet Explorer、Word 和 Excel 等 Microsoft 产品一起运行。 JavaScript、另一种跨语言编译器和其他类似平台比 ActiveX 使用更广泛。

## 了解 ActiveX

ActiveX 控件是类似于 Web 浏览器插件的预编码软件。例如，显示 Flash 文件的网页可能需要用户下载 Flash ActiveX 控件，以便可以直接在浏览器中播放该文件，而无需打开新的应用程序。 ActiveX 控件扩展了浏览器的功能，允许浏览器执行原本无法执行的任务。它对于播放视频和其他多媒体内容特别有用，可以跳过打开单独的媒体播放器的步骤。

ActiveX 仍在 Internet Explorer 11 中使用，但不受 Microsoft 最新浏览器 Edge 的支持。 Mozilla Firefox、Google Chrome 、Apple 的 Safari 和 Opera 等浏览器使用其他类型的浏览器插件，例如 JavaScript 或类似的跨平台语言。

ActiveX 对 Microsoft 用户仍然有用，并包含在 Windows 10 中。这是因为 ActiveX 仍然允许独立软件接收更新、跨程序接口和增强功能。例如，ActiveX 允许用户在 Word 中创建更具交互性的文档或在 Excel 中创建可填写的表单。

## ActiveX 和计算机安全

微软警告说，ActiveX 控件有时会出现故障或向用户提供他们不想要的内容。 ActiveX 控件还可用于安装[间谍软件](/adware)、病毒和恶意软件，或损坏计算机上的数据。因此，仅从您信任的来源安装 ActiveX 控件非常重要。

部分由于 ActiveX 控件的广泛恶意使用，许多浏览器要么默认禁用 ActiveX 控件，要么根本不支持它们。例如，Google Chrome 用户必须在 Chrome 的安全设置中启用 ActiveX 或下载 Chrome 扩展程序。

甚至微软似乎也开始远离 ActiveX。 Edge，它的新浏览器正在取代 Windows 操作系统上的 Internet Explorer，不支持 ActiveX 。

＃＃ 强调

- Microsoft 警告说，ActiveX 控件可用于监视您的计算机、损坏数据或安装恶意软件。

- ActiveX 是一个用于开发小程序或插件的平台，这些小程序或插件可为 Web 浏览器和 Microsoft 产品提供更多功能。

- 许多浏览器默认不支持 ActiveX。 Microsoft 的新浏览器 Edge 不提供对 ActiveX 控件的支持。
