# 统一托管账户 (UMA)
## 什么是统一托管账户？

统一管理账户 (UMA) 是专业管理的私人投资账户，可以在一个账户中包含多种类型的投资。投资可能包括共同基金、股票、债券和交易所交易基金。统一托管账户通常会按指定的时间表重新平衡。

## 了解统一托管帐户 (UMA)

统一管理账户是[高净值投资者](/hnwi)管理其资产的少数选择之一。统一管理账户是独立管理账户的演变，其相似之处在于它是一个经常重新平衡的专业管理账户。然而，单独管理的账户通常不以汇集具有不同目标的多种投资和投资工具而闻名。单独管理账户是一种高净值投资选择，通常由投资经理提供，通常侧重于作为投资者单独账户管理的目标策略。如果投资者想要跨多种策略进行投资，他们可能必须开设多个单独管理的账户。

对于寻求合并多项投资的投资者而言，统一管理账户通常是更好的选择。 UMA 消除了拥有多个账户的需要，并且可以将投资者的所有资产合并到一个账户中。

> UMA 投资者根据管理的总资产（AUM）支付年度管理费；费用随着资产管理规模的增加而下降。

>

## 通过统一托管账户进行投资

相同的银行和经纪公司通常提供统一管理账户作为单独管理账户。他们的服务范围也扩大到包括注册投资顾问和私人财富管理公司。技术一直是支持其扩张的驱动因素。一个统一的托管账户提供商承担着更大的整体信托责任，因为他们作为众多投资的监督者，其中可能包括股票头寸、员工股票期权计划、第三方独立账户管理等。

UMA 提供商与高净值投资者合作，整合客户的所有资产。汇总资产后，UMA 提供商将以多种方式与客户合作。 UMA 提供商可以检查整个投资组合以制定全面的计划。 UMA 账户规划可以包括一个覆盖策略，该策略旨在通过有针对性的资产分配多样化方法来管理投资组合。 UMA 提供商还为投资者提供新的选择，包括投资者可能希望随着时间的推移投资的附属公司和产品。鉴于组合资产所创造的全面、有效的前沿，UMA 提供者通常会分析投资组合以符合现代投资组合理论。 UMA 提供商的替代选项可以帮助客户更好地调整他们的总投资组合以优化风险回报。

UMA 提供商还为高净值客户提供更简化的投资报告，并为全面的税务规划提供更大的支持。 UMA 提供商还与客户一起确定适合其整体投资策略的[再平衡计划。](/rebalancing)

UMA 标准因提供商而异，投资者通常会签署一份协议，详细说明账户的管理、费用以及允许的投资和结构。 UMA 投资者通常根据管理的总资产支付年度管理费。费用通常会随着[管理资产的增加而降低，](/aum)每年的幅度从 1.50% 到 3% 不等。

＃＃ 强调

- UMA 通常会定期重新平衡，以确保所持资产反映正确的投资平衡。

- 统一管理账户 (UMA) 是一种私人管理的投资账户，可以包括多种投资，例如共同基金、股票和债券。

- UMA 提供商与投资者合作设计一个包含多项投资的综合投资组合，为税务规划提供支持，并制定有效的再平衡计划。

- 这是一种经常受到高净值投资者青睐的账户类型，他们希望获得比单独管理账户所能获得的更大范围的投资。
