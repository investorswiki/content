# 拨号和微笑
## 什么是拨号和微笑？

拨号和微笑是一种[电话营销](/telemarketing)技术，在这种技术中，会主动向潜在客户拨打产品或服务的电话。这是[冷呼唤](/coldcalling)的俚语。无论使用哪种名称，这种做法都具有负面含义。

[锅炉房](/boilerroom)运营中不道德的股票发起人的高压销售活动有关。这些销售人员用乐观和积极的语气向毫无戒心的投资者出售高风险甚至虚假的投资。

拨号和微笑也称为拨打美元。

## 了解拨号和微笑

拨号和微笑通常涉及呼叫潜在客户，而不是现有客户。股票[经纪公司](/brokerage-company)可能会使用从潜在客户[数据库中生成的潜在](/sales-lead)客户来赢得新客户。

拨号和微笑依赖于线索，最好是那些经过审查的净资产、收入、职业、邮政编码或其他线索，以了解目标购买所推销产品的能力。可以根据用户的需要购买或建立这样的列表。一些拨号和微笑的从业者挖掘已发布的公司目录或在下班后拨打公司的主要电话线以挖掘他们的电子电话分机目录以获取姓名和头衔。

### 对抗拨号和微笑

美国证券交易委员会 ( [SEC](/sec) ) 制定了限制证券公司打电话的规则。例如，这些规则限制了可以拨打电话的时间，并禁止来电者做出虚假陈述。他们还敦促消费者向金融业监管局 ( [FINRA](/finra) ) 报告违规行为。

一些司法管辖区制定了法规来限制电话推销行为的滋扰。

### 拨号和微笑以及请勿呼叫注册表

对手机进行电话推销已经很普遍，但实际上它们违反了美国联邦通信委员会 ( [FCC](/fcc) ) 的规定，该规定禁止在未经用户事先同意的情况下自动拨打手机。电话推销员通常使用自动呼叫，只有当呼叫被应答。

请勿来电 (DNC) 注册表的引入，其中列出了不希望接到电话推销员主动来电的人，减少了此类电话的发生率。虽然因违反 DNC 规则而处以的高额罚款对合法企业具有威慑作用，但对于可能从事非法或非法活动的锅炉房经营者而言，它们的威慑力较小。

> 对手机进行电话推销违反了 FCC 规则。

>

您可以在联邦贸易委员会网站的“请勿致电”列表中输入您的号码。它应该适用于固定电话和手机号码。

### 拨号和微笑策略

虽然这些规则对最具争议的冷电话方法产生了寒蝉效应，但拨号和微笑的最佳做法可能适用于许多类型的销售和客户寻找工作。例如：

- 不要害怕“不”这个词。大多数拨号和微笑的呼叫者每天都会被拒绝多次。他们很快就知道唯一的失败是没有打下一个电话。

- 做好准备。呼叫者必须了解潜在客户的前景、需求和需求，并知道如何解决这些问题。应该练习投球。如果可能，请在致电前发送一封介绍性电子邮件。

- 调整自己的节奏。将通话清单分成可管理的部分，但一定要保持势头。当您在滚动时，通话更容易。

＃＃ 强调

- 拨号和微笑是冷呼叫的行话。

- 消费者可以通过登录 Do Not Call 注册表来躲避拨号和微笑策略。

- 在投资行业，这种销售策略受美国证券交易委员会监管。
