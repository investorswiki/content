# 先提交规则
## 什么是先归档规则？

首先归档规则可能意味着不同的事情，具体取决于上下文。在联邦法院诉讼程序的背景下，首先提起诉讼的规则规定，无论谁是第一个提起诉讼的一方，都将获得其本国法院的审判或法律诉讼地点。

在[知识产权](/intellectualproperty)和[专利](/patent)申请的背景下，在先申请规则确保了第一个申请发明专利的人通常获得销售该发明的权利。

## 了解先提交规则

首先归档规则可能意味着不同的事情，具体取决于上下文。

### 联邦诉讼先提交规则

第一个提起诉讼的规则规定，首先提起诉讼的一方将获得其本国法院的审判或法律诉讼地点。这提供了一个优势，因为诉讼律师可能熟悉法官或法院，并且不会产生额外的费用，例如差旅费。在审判期间，诉讼律师也不必离开家住在酒店。先备案规则只是一般规则。这不是规定所有程序的法律，并且可以例外。

### 知识产权和专利申请

专利申请很复杂，许多投资者聘请律师来帮助他们完成这个过程。它需要研究以确保不存在类似创作的专利。第一个为发明申请专利的人在被接受后可以获得专利。即使这个人不是第一个创造发明的人，他们也是专利的所有者，因为他们是第一个提出申请的人。

该专利允许发明人出售该作品。专利局要求申请具有竞争优势，才能在专利有效期内销售产品。虽然最先申请的人可能会先获得专利，但在此过程中申请可能会被多次拒绝。这可能意味着另一位申请人击败了他们并获得了专利。

因此，不建议延迟[知识产权申请。](/intellectualproperty)在先申请规则在申请人获得特定和个人专利之间的宽限期相对较短，发明人应在披露发明的任何细节之前提交申请，包括提供待售物品。这对于避免模仿者使用该设计至关重要。

## 聘请律师优先提交

为避免法律纠纷或为先申请专利规则提供帮助，发明人可能需要专利律师的帮助。法律专业人员将致力于保护发明人的权利，律师可以通过专利申请程序协助或支持发明人。

＃＃ 强调

- 先申请规则也适用于专利申请，授予第一个申请专利的人对其知识产权主张权利。

- 第一个提交规则断言，第一个提起诉讼的一方将被授予他们的家庭所在地以进行审判或法律程序。

- 鼓励发明人迅速申请专利，以免模仿者篡夺他们的新发明或新工艺。
