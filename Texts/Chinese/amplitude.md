# 振幅
##什么是幅度？

幅度是证券价格在一段时间内从其波动周期的谷底（底部）到其价格变动的波峰或峰值之间的差异。当计算看涨[回撤](/retracement)（从低谷到峰值计算时）时，振幅为正，而在计算看跌回撤时（从峰值到低谷计算时），振幅为负。

## 了解幅度

幅度允许估计特定证券的波动性。幅度越大，无论是正的还是负的，证券被判断的波动性越大。波动性水平也可以表示特定投资中存在的风险量。

### 什么构成高峰或低谷

峰值被确定为特定证券在特定时间段内达到的最高价格点。有了这种理解，峰值可能会根据检查的时间段而有所不同。

波谷是波峰的倒数。它代表证券在同一时期内价格最低的点。当与一个国家的[国内生产](/gdp)总值 (GDP) 相关时，低谷代表经济萧条期间紧接向上转向复苏之前的最低点。

### 确定与波峰和波谷相关的幅度

幅度表示一个时间段内波峰中点与波谷中点之间的差值。每个中点是通过找出极端值（例如上述峰值或谷值）与中线之间的差异来确定的。在可能存在正值和负值的情况下，中线可能位于零。

在其他情况下，在不允许负值的情况下，中线可能代表证券的平均价格。幅度是通过从另一个中点减去一个中点来计算的。

## 将幅度计算为公式

为了计算幅值，a 的值可以使用以下公式，假设 b 的值是波峰的中点，c 的值是波谷的中点。

对于看涨回撤，应使用公式 b - c = a，其中 x 轴上 c 位于 b 之前。这将产生一个正振幅 a，表示上升趋势。

对于看跌回撤，应使用公式 c - b = a，其中 b 在 x 轴上位于 c 之前。这将导致负幅值 a 表示下降趋势。

＃＃ 强调

- 从底部到顶部查看证券的路径时，幅度为正；从上往下看，幅度为负。

- 幅度是通过从另一个中点减去一个中点来计算的，看涨回撤的公式与看跌回撤的公式不同。

- 金融振幅衡量特定证券价格在一段时间内的变化。

- 通常，幅度着眼于证券价格从其波浪周期波谷或底部到顶部或波峰的移动。

- 查看价格随时间的变化可以让市场参与者了解证券的波动水平，这有助于市场时机和其他交易策略。
