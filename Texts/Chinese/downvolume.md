# 降低音量
##什么是成交量下降？

当证券的价格随着大量交易而下跌时，就会出现成交量下降。成交量下降是一种交易场景，也可以称为成交量下降。

这可以与[up volume形成对比](/upvolume)。

## 了解下音量

成交量下降是指对股票产生负面影响的大量交易。下跌交易量与上涨交易量相反，其中证券的价格随着交易量的增加而上涨。成交量下跌表示[看跌](/bear)交易，而成交量上涨表示[看涨](/bull)交易。

如果证券价格下跌，但仅在低交易量时，除了真正的熊市转折之外，可能还有其他因素在起作用。例如，一些[做市商](/marketmaker)[或](/marketmaker)其他参与者正在休假，导致流动性比平时少，或者买家在等待价格略微走低后再进行投标。无论哪种方式，成交量下降都反映了价格随着交易量的增加而下跌的情况，这证实了价格走低。

[噪音交易者](/noisetrader)往往是大量交易的重要贡献者。在某些情况下，股票可能会因公司内部刚刚向公众发布的积极发展而上涨。如果这个消息出乎意料，随着股票修正并上涨，它可能会导致[机构投资者](/institutionalinvestor)和[散户投资者的大量交易。](/retailinvestor)通常，噪音交易者会极大地促进交易量大的交易日，因为这些投资者会跟随趋势并根据情绪情绪进行大量交易。

大多数[技术分析师](/technical-analyst)和机构投资者将关注他们正在考虑投资的证券数量。交易量飙升通常是由值得关注的重要市场催化剂引起的。许多技术分析师认为，成交量也可能是价格向看涨或看跌方向突破的信号。

＃＃ 体积

[成交量](/volume)是影响证券价格的市场因素之一。交易量由在指定时间段内交易的证券股份数量定义。一般来说，交易者会每天观察证券的成交量，将价格下跌的日子称为成交量下跌的日子。

有几个因素会影响音量。成交量下降的日子通常受到直接有关股票的负面消息或间接影响股票的消息的影响。低于预期的收益报告或有关公司销售、管理或管理决策的负面消息可能导致大量交易，即所谓的[抛售](/sell-off)。

在抛售中，大部分的成交量都是下跌的，这意味着投资者正在迅速卖出而不是买入，这对价格产生负面影响。一般来说，交易量大的日子可能会受到噪音交易者的严重影响，他们是非专业交易者，当有关公司的高调新闻发布时，他们往往会更频繁地交易。噪音交易者的交易可能会对股票造成不必要的负面影响，这有时会从超卖中创造买入机会。

交易者可以观察几个指标来解释交易量并了解其对证券价格的影响。三个最受欢迎的指标是[成交量加权平均价格](/vwap)(VWAP)、[正成交量指数 (](/pvi) [PVI](/pvi) ) 和[负成交量指数](/nvi)(NVI)。

成交量加权平均价格是从以下计算的移动平均线绘制的趋势线：

VWAP = (买入的证券股份 x 证券股份的价格) / 买入的证券股份

监控交易量对价格影响的交易者通常会关注 VWAP 交叉。当 VWAP 交叉飙升至下行并越过证券的[烛台](/candlestick)形态时，这是成交量下降的迹象。如果检测到这种模式，它可能是证券价格看跌趋势的早期指标。交易者通常会通过卖出以利用价格下跌来从这一信号中获益。

## PVI 和 NVI

正负交易量指数（PVI 和 NVI）由 Paul Dysart 于 1936 年首次开发，以帮助投资者辨别市场交易量的一些影响。PVI和 NVI 在计算扩展到个人之后在 1970 年代变得更加流行证券。

PVI：如果当前交易量大于前一天的交易量，则 PVI = 上一个 PVI + {[(今天的收盘价-昨天的收盘价)/昨天的收盘价)] x 上一个 PVI}。如果当前交易量低于前一天的交易量，则 PVI 不变。

NVI：如果当前交易量小于前一天的交易量，则 NVI = 上一个 NVI + {[(今天收盘价-昨天收盘价)/昨天收盘价)] x 上一个 NVI}。如果当前交易量高于前一天的交易量，则 NVI 不变。

这些指数值提供了有关价格如何随交易量波动的见解。在成交量上升的趋势中，PVI 将随着成交量的增加而走高。因此，寻求在看涨交易量交易中获利的投资者可以使用 PVI 作为潜在价格信号的一项指标。

＃＃ 强调

- 成交量下跌是指证券价格下跌伴随着高交易量或增加交易量。

- 负成交量指数有助于跟踪成交量下降，以确认价格下跌确实可能预示着情绪的长期转变。

- 成交量下降可能表明转向修正或熊市。
