# 相关值的基准
## 什么是相关值的基准？

相关值的基准是一个基准或特定参考点，投资基金或个人投资者用来衡量其投资组合的重要相关值，例如[beta](/beta) ，它衡量证券对整个市场的[波动性，](/volatility)或[R-squared](/r-squared) ，一种统计量度，显示因变量的方差有多少可以由自变量解释。

## 了解相关值的基准

基准相关值很重要，因为它们表明特定基金的表现与其市场的相关程度，使用[基准](/benchmark)作为该市场的代理。例如，如果基金的投资主题与基准密切相关，则通常认为与基金基准的高度相关性对该基金有利。

相关值的基准取决于特定基金的投资授权。例如，[大型](/large-cap)美国股票基金可能会使用[标准普尔 500](/sp500)作为其相关值的基准，而大型加拿大股票基金可能会使用[标准普尔/多伦多证券交易所综合指数](/sp-tsx-composite-index)作为其基准。

可以使用相关系数来衡量基金特定指标与其基准指标之间的相关性。[相关系数](/correlationcoefficient)是衡量两个变量之间关系有多强的统计量。

如果值的范围在 -1.0 和 1.0 之间，则 -1.0 的相关性表示完全[负相关](/negative-correlation)；这意味着这两个变量根本没有对齐，而相关性为 1.0 则显示出完美的[正相关性](/positive-correlation)，表明这些变量彼此紧密相关。相关性为 0 表示两个变量的移动之间的关系为零或没有关系。

## 相关值基准的重要性

了解您的投资如何关联对于了解如何管理特定投资组合的[风险非常重要](/risk)。如果您的投资策略旨在遵循特定基准（例如指数）的策略，请检查您的投资组合中的财务指标与基准中的财务指标相比如何。这将使您能够衡量您的投资是否正常、投资组合的风险状况以及其他重要因素。

相关值的基准可作为指导，如果需要对投资组合进行任何调整，可以通知投资组合经理。它还将表明投资组合未来的表现如何，这有助于为任何损失做好准备。

## 投资组合中资产的相关性

相关性是基于不同资产价格之间的关系。它衡量两种资产价格一起移动的可能性，并在 -1 到 1 的范围内移动。例如，如果两种资产的相关性均为 1，则它们是正相关的，并且将始终沿相同的方向向上或向下移动。

因此，如果您只投资科技股，其相关性很可能为 1，而政府通过的新法规损害了科技股的业务增长，您的整个投资组合都会受到负面影响。

具有负相关性（值为-1）的资产始终朝相反的方向移动。相关性为 0 的资产有 50% 的时间朝同一方向移动。

如果您的太多投资高度相关，那么如果其中一项遭受损失，那么许多其他或全部投资也会遭受损失。

## 多样化以减少相关值

根据经验，资产的相关性范围在 -0.5 到 0.5 左右之间通常被认为是谨慎的，但实际数字会因投资者的[风险承受能力而异](/risktolerance)。例如，[规避风险的](/riskaverse)投资者会希望尽可能少的相关性。这也是[多元化背后的理念](/diversification)。

多元化的投资组合包含彼此之间几乎没有相关性的资产。可能有一定数量的资产确实相关，但也有足够多的资产不相关，因此一个领域的不利市场变动可能不会影响另一个领域，从而最大限度地减少损失。

＃＃ 强调

- 根据基准测量的常见相关性指标包括 beta 和 R 平方。

- 多元化有助于降低投资组合中资产之间的相关性，这有助于减轻特定市场低迷时期的损失。

- 相关值的基准是投资基金用来衡量财务指标相关性的参考点。

- 基准相关值用于显示投资组合的表现与其市场相关的程度，特别是作为市场或基金预期投资策略的代表的基准。

- 相关系数用于衡量两个变量之间的关系有多强。

- 最常见的相关系数值介于 -1.0 和 1.0 之间，-1.0 表示相关性最低，1.0 表示相关性最高。
