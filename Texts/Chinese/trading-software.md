# 交易软件
## 什么是交易软件？

交易软件有助于金融产品的交易和分析，例如股票、期权、期货或[货币](/currency)。

有各种各样的交易软件包可供各种级别的交易经验使用，并针对不同的市场（例如，股票与外汇）量身定制。

## 了解交易软件

通常，[经纪公司会](/brokerage-company)为他们的客户提供交易软件来进行交易和管理他们的账户。该软件可以从桌面或移动设备下载和启动，或者它可以是基于网络的，交易者通过他们登录的网站访问该软件。

交易者还可以购买第三方交易软件来补充或增强券商提供的软件。

[自主](/self-directed-ira)交易账户进行至少一些自己的交易和分析。这增加了对提供交易功能的软件以及软件内的分析和信息资源的需求。

交易软件可以为用户提供资产定价信息、特殊订单类型、[基本数据](/fundamentalanalysis)、图表、技术分析指标、统计数据、聊天室，以及经纪人和软件开发商用来吸引交易者服务的其他专有工具或功能。

应用程序编程接口或[API的可用性](/application-programming-interface)也有助于推动交易软件行业的发展。 API 允许将另外两个交易软件链接起来，作为一个。这允许用户访问多个软件的好处。 API 并不总是需要，因为用户可以简单地在他们的计算机上独立运行两个或多个程序，尽管这些程序不会相互通信。

## 交易软件类型

经纪商和第三方开发商提供了不同类型的具有不同功能的交易软件。

一些最常见的功能包括：

- **下单交易**：大多数交易软件都可以下单，包括[市价单](/marketorder)、限价单和其他高级订单类型，以及查询实时报价和查看[二级](/level2)订单簿的能力.一些软件还会跟踪交易统计数据，例如平仓交易的[赢率](/win-loss-ratio)和平均损益。

- **技术分析**：大多数交易软件都包含交互式图表功能，包括[趋势线](/trendline)和形状等图表模式，以及[移动平均线](/movingaverage)或动量振荡器等[技术指标。](/technicalindicator)

- **基本面分析**：一些交易软件提供对基本面信息的访问，包括[财务报表](/financial-statements)、分析师评级和其他专为投资者简化[尽职调查而设计的专有工具](/duediligence)。

- **程序化交易**：先进的交易软件使交易者能够开发可以自动执行的交易系统，而不必手动点击按钮。此外，这些软件解决方案可能会提供回[测](/backtesting)功能，旨在帮助交易者了解他们的自动交易系统过去的表现。

- **纸面交易**：一些交易软件包括进行无风险无现金交易的能力，即所谓的[纸面交易](/papertrade)。交易者可以在投入实际资金之前测试他们的技能，看看他们的表现如何。此功能在[外汇市场](/forex-market)的经纪商中尤为常见。

## 选择交易软件

在决定交易软件之前，交易者和投资者应该仔细考虑他们需要哪些功能。依赖自动交易系统的[活跃交易者](/active-trading)可能会选择与只寻求下单能力的投资者完全不同的交易软件。

软件应用程序可能具有不同的[费用](/fee-structure)[结构](/fee-structure)、性能特征和其他影响盈利能力的因素。

大多数经纪人和软件开发商允许潜在客户在承诺购买或在经纪人处开设账户之前测试他们的软件。通过尝试几个软件来利用这一点。查看您喜欢和使用的工具和功能。然后权衡经纪人（如果适用）及其佣金的利弊。

如果您喜欢某个特定的经纪人，例如因为他们的低费用结构，但您不喜欢他们的软件，您仍然可以找到可以通过 API 或独立使用的第三方软件。

例如，如果您不喜欢经纪商的图表功能，您可以订阅您喜欢的第三方图表服务/软件，并将其与经纪商的交易功能结合使用。

## 第三方交易软件示例

大多数经纪人都有自己的交易软件，尽管有些提供第三方软件。例如，在外汇行业，很多经纪商都有自己的软件，但也有很多还提供 MetaTrader4 和/或 MetaTrader5，这是常用的第三方交易平台。

在股票市场中，大多数经纪人都提供自己的软件。这里有一些大型经纪人和他们的软件。

- Fidelity 提供 Active Trader Pro。

- 盈透证券提供交易平台和低成本的每股费用结构。

- Charles Schwab 提供 Streetsmart Edge 和 4.95 美元的股票交易。

- TradeStation 提供 TradeStation，在日内交易者和活跃交易者中很受欢迎。

- TD Ameritrade 提供thinkorswim 交易平台和6.95 美元的股票交易。

- 还有一些广泛可用的第三方软件和交易平台。

- NinjaTrader 平台提供图表、分析和交易功能，并可与多家经纪商联动。

- TradingView 和[St](/stockcharts-technical-rank-sctr) [ockCharts](/stockcharts-technical-rank-sctr)提供技术和基本图表工具。这些工具可以补充交易平台提供的图表功能。

＃＃ 强调

- 交易软件的常见功能包括下单、技术分析、基本面分析、自动交易和纸面交易。

- 除了学习如何交易或投资之外，自主交易者还需要利用和学习如何有效地使用他们的交易软件。

- 交易软件用于证券的电子交易和分析。
