# 条例 C
## 什么是规则 C？

条例 C 是实施1975 年《[房屋抵押披露法》](/home-mortgage-disclosure-act-hmda)的条例。条例 C 要求许多金融机构每年披露有关其提供住宅抵押的社区的贷款数据。因此，监管机构可以评估[贷方](/lender)是否充分满足了该社区潜在借款人的需求。

## 规则 C 的工作原理

所有由政府以任何身份支持的[抵押贷款提供者](/mortgage)必须每年披露过去一年内提供的所有抵押贷款的数量和金额。这些贷款必须按房产所在的人口普查区进行细分。

自 2022 年 1 月 1 日起，任何总资产为 5,000 万美元或以下的贷款机构都免于遵守 C 条例规定的数据收集要求。这包括低于 5,000 万美元门槛的银行、储蓄协会和信用合作社。门槛会定期提高，以跟上消费者价格指数 (CPI) 衡量的通货膨胀。这意味着在 2023 年及以后，门槛可能会再次提高。

条例 C 旨在提供可用于：

- 帮助确定银行、储蓄协会和信用合作社是否在满足各自社区居民的住房需求

- 帮助公职人员分配公共部门投资，以吸引私人投资到需要的地区

- 协助识别可能的歧视性贷款模式并强制遵守联邦反歧视法规

> ### 重要

> 法规 C 并非旨在鼓励不健全的贷款做法或不当的信贷分配。

>

随着新的最终规则的发布，C 规则指南可能会发生变化。例如，消费者金融保护局（Bureau）修订了 C 条例，在 2020 年 4 月提高了封闭式抵押贷款数据的报告门槛。作为该规则变更的一部分，开放式贷款数据的报告门槛自 2022 年 1 月 1 日起，在之前的 500 个开放式信贷额度临时门槛到期后，信用额度设置为 200 个。

## 条例 C 涵盖哪些内容？

条例 C 涵盖了与数据收集和贷款发放有关的许多问题。以下是该法规如何运作的详细信息。

### 数据编译

需要遵守法规 C 的金融机构必须在每个日历年报告其数据。这些数据被分解为人口普查区，以显示抵押贷款的发放、房屋购买和家庭装修贷款。

条例 C 要求这些机构还提供有关未产生贷款的贷款申请的[数据](/origination)。这包括撤回的申请、拒绝贷款、因不完整而被驳回的申请以及获得批准但未被接受的申请。

收集此类数据应该为当局提供一种筛选贷款歧视事件的方法。这些信息与人口普查区的地理位置和人口统计信息有关。如果重复出现拒绝向特定人群提供融资的模式，金融机构可能会面临当局的处罚。例如，一家银行可能会不断拒绝向特定种族或特定地区的人提供融资，尽管其他方面是合格的。此类活动将引起监管机构的注意。

### 报告和披露

条例 C 要求受此规则约束的金融机构进行年度报告。必须在收集和记录数据的下一个日历年的 3 月 1 日之前提交报告。金融机构必须将年度贷款/申请登记册的副本保留至少三年。

同时，报告至少 60,000 笔担保贷款和申请的金融机构（不包括购买的担保贷款）也需要按季度报告数据。该报告应在每个日历季度结束后的 60 个日历日内完成。

联邦金融机构审查委员会 (FFIEC) 必须根据每个金融机构上一日历年提交的数据提供披露声明。一旦发布这些披露声明，金融机构必须向公众提供书面通知，说明这些声明可以在 CFPB 的网站上查看，作为《房屋抵押披露法》的一部分。

＃＃＃ 保持记录中

根据条例 C，受保机构必须记录受保抵押贷款申请和贷款的数据。然后必须将该信息报告给适当的联邦监管机构。具体而言，“机构必须在机构采取最后行动的日历季度结束后的 30 个日历日内记录每个申请或贷款的数据。”

> ### 注意

> 金融机构可以以纸质或电子形式保存其年度 HMDA 记录。

>

＃＃＃ 抵押

条例 C 涵盖的机构必须收集和报告“任何由住宅担保的抵押贷款，包括开放式信贷额度，无论贷款的目的如何。”但某些类型的贷款被排除在外，包括无担保的房屋装修贷款、主要用于商业或商业目的的住宅担保贷款、农业用途贷款和其他明确排除的贷款。

收集的数据点包括：

- 有关申请人或借款人的详细信息，例如年龄和信用评分

- 有关贷款定价的信息，包括借款人获得抵押贷款的总成本、临时介绍利率和借款人支付的初始费用

- 有关贷款特征的详细信息，例如贷款期限、提前还款罚款或非摊销特征

- 用于担保贷款的房产信息，包括房产类型和价值

> ### 提示

> 法规 C 还要求金融机构收集和报告有关借款人的种族、种族和性别的信息。

>

## 特别注意事项

消费者金融保护局继续修订条例 C。迄今为止，该政策的更新包括增加新的报告要求，以符合 2010 年的《[多德-弗兰克华尔街改革和消费者保护法](/dodd-frank-financial-regulatory-reform-bill)》。多德-弗兰克还转让了从联邦储备委员会到消费者金融保护局的家庭抵押披露法案 (HMDA) 的规则制定机构。

## 底线

条例 C 在确保整个抵押贷款行业的公平做法方面发挥着重要作用。如果您计划申请抵押贷款以购买房屋、为现有抵押贷款再融资或申请房屋净值贷款或信用额度，那么了解该法规对您的影响以及您的贷方可能会收集哪些信息非常重要。

＃＃ 强调

- 条例 C 旨在帮助公职人员确定公共部门投资的分配计划，以此吸引更多私人投资到有需要的地区。

- 法规 C 要求许多金融机构每年披露其提供住宅抵押贷款的社区的贷款数据。

- 所有由政府以任何身份提供支持的抵押贷款提供者必须每年披露过去一年内提供的所有抵押贷款的数量和金额。

＃＃ 常问问题

### 什么是抵押贷款条例 C？

条例 C，也称为家庭抵押披露，管理数据的收集和有关抵押相关活动的某些信息的披露。资产超过一定门槛的金融机构，包括银行、储蓄协会和信用合作社，必须根据 C 规则汇编数据。

### 谁受法规 C 的约束？

法规 C 涵盖封闭式和开放式消费贷款或由房屋担保的信贷额度。因此，这可以包括第一和第二抵押贷款、房屋净值贷款和房屋净值信贷额度。

### 什么是涵盖贷款条例 C？

担保贷款是封闭式抵押贷款或开放式信贷额度，不被视为明确排除在法规 C 报告要求之外。涵盖的贷款包括由不动产担保的消费者抵押贷款，其中贷款超过特定的年利率或积分和费用的百分比。
