# EIA 石油状况报告
## 什么是 EIA 石油状况报告？

EIA 石油状况报告每周三由美国能源部下属机构美国[能源信息署](/energy-information-admin)(EIA) 发布。它详细说明了美国持有的[原油](/crude-oil)储备水平，以及它在国内和国外生产的原油和相关产品的数量。

## 了解 EIA 石油状况报告

EIA 石油状况报告详细介绍了美国现有的原油库存，以及精炼石油产品（如汽油、取暖油和柴油）的库存水平。报告的主要亮点部分细分了炼油厂的原油投入、这些炼油厂的[产能利用率](/capacityutilizationrate)以及新汽油和馏分油的产量。

通常，该报告还包括一个表格，其中包含原油和石油产品的净进口或美国进口总额减去出口。此外，该报告还详细介绍了[西德克萨斯](/wti)[中质](/wti)(WIT) 原油、普通无铅汽油、2 号取暖油和丙烷的每加仑价格。

原油价格由[供需决定](/law-of-supply-demand) [技术，](/technicalanalysis)与其他商品一样。当储备下降时，价格会上涨，反之亦然。因此，EIA 石油状况报告反映了当前的原油价格趋势。报告的作者并没有试图解释这些趋势的基本驱动因素。

投资分析师和期货交易员将研究该报告，并利用其数据对能源价格和生产水平做出预测。同样，一些为能源公司工作的分析师将利用新闻从报告中收集数据，以帮助他们制定长期战略和业务决策。媒体还经常使用 EIA 报告来报道直接影响消费者的能源行业和价格，例如取暖油和汽油。

## EIA 石油状况报告的详细信息

美国能源信息署在其网站上提供状态报告。用户可以使用在线工具查看报告，包括按主题的分析和预测以及按类别排序的数据。完整的报告以 PDF 格式提供，并以可导出的 Microsoft Excel 和逗号分隔值 (CSV) 电子表格的形式提供，以供进一步分析。

- EIA 石油状况报告重点部分提供了大多数行项目过去 4 周的生产数据。同样，一年前和前一周的比较也使用了过去四个星期的数据。

- 生产数据的基础来自指定日期的[收盘价](/closingprice)。

- 报告中的表格显示原油、车用汽油、燃料乙醇和石油产品的每周库存或供应估计。这些数据可以按地理区域进一步细分，因为新英格兰的价格可能与阿拉斯加或下大西洋各州的价格存在显着差异。

- 报告下方是一张包含石油产品库存和价格的表格，这些产品的交易频率较低，包括航空燃料、煤油和超低硫柴油。

- 原油、汽油和取暖油的价格部分包含现货（或短期）和期货定价以及航空燃料、丙烷和柴油燃料的价格。

- 最后，该报告提供了美国以及地理区域的**零售**汽油和公路柴油燃料的月度价格。

## EIA 报告与 API 报告

美国[石油协会 (API)](/american-petroleum-institute)是一个成立于 1919 年的组织，其目标是支持强大的美国石油和天然气行业，同时促进整个行业的安全。 API 为参与生产、加工、精炼和分销能源产品的美国公司提供服务。

API 从其近 600 名成员中进行研究，收集成员和非成员的数据，并在每周统计公报中发布他们的统计数据和行业指标。该报告包含有关能源产品、进口、出口、成本、钻井活动和完井的供需数据。

API 还有助于开发和维护 700 多项标准和实践，其中许多已被纳入地方和联邦法规。 API 代表石油和天然气行业向联邦政府（包括国会、行政部门和州政府）宣传。

API 还有助于与监管机构进行谈判，包括在法律诉讼中代表行业。 API 的每周统计公报通常在每周 EIA 报告之前的周二发布。

EIA 与 API 非常相似，因为它是一个收集、分析和发布有关美国石油和天然气行业信息的独立组织。 EIA 在周三发布其每周石油状况报告。该报告包含原油和精炼产品的库存水平以及供需数据。但是，与 API 不同，EIA 不会积极游说任何政策和监管变化。

## EIA 报告常见问题

＃＃ 强调

- EIA 石油状况报告每周三由美国能源信息署 (EIA) 发布。

- EIA 石油状况报告详细说明了美国现有原油库存和精炼石油产品（如汽油、取暖油和柴油燃料）的库存水平。

- EIA 石油状况报告可以显示原油价格趋势，因为原油库存增加可能表明需求下降，从而导致油价下跌，反之亦然。

＃＃ 常问问题

###环评报告什么时候发布？

环评报告在每周三上午 10:30 后发布。

###环评报告可靠吗？

EIA 报告被广泛认为是石油和天然气行业供需的可靠指标。但是，该报告并不保证或预测供需或价格趋势将以一种或另一种方式变化。

### 为什么环评报告很重要？

EIA 石油状况报告很重要，因为它显示了原油等能源产品的价格和库存数据。 EIA报告是石油和天然气行业供需的重要指标。例如，如果原油库存增加，则可能表明石油需求正在增加，从而导致价格上涨。
