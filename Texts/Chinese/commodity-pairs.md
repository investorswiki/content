# 商品对
##什么是商品对？

商品对或商品货币是来自拥有大量商品储备的国家的外汇[货币对。](/foreign-currency-pairs)由于这些国家生产和出口各种商品，因此这些货币对与商品价格的变化高度相关。希望获得商品价格波动风险敞口的交易者和投资者通常持有商品货币对的头寸，作为购买[商品的代理投资](/commodity)。

商品对包括美元 ( [USD](/usd) ) 与加元 ( [CAD](/cad-canadian-dollar) )、澳元 ( [AUD](/aud-australian-dollar) ) 和新西兰元 ( [NZD](/nzd-new-zealand-dollar) ) 的配对。俄罗斯卢布 ( [RUB](/rub-russian-ruble) )、巴西雷亚尔 ( [BRL](/brl-brazilian-real) ) 和沙特里亚尔 ( [SAR](/sar-saudi-riyal) ) 也是对商品价格敏感的货币。

## 了解商品对

[外汇 (FX)](/foreign-exchange)市场上交易最广泛的货币对之一。外汇交易是在各种货币的汇率中建立头寸的行为。外汇交易涉及以现行汇率买卖外币，目标是汇率对投资者有利。

以 0.7500 的现行汇率买入澳元兑美元汇率 ( [AUD/USD )，即 75 美分兑一澳元。](/aud-usd-australian-dollar-us-dollar-currency-pair)如果交易所升至 0.8500，交易者将预订抵消交易以平仓并赚取 0.0100 美分的利润（减去经纪人费用或费用）。在外汇交易中，没有实际的货币交割。相反，抵消交易平仓，净赚或亏的金额通过交易者的经纪账户进行交换和处理。

### 流动性

商品对市场往往[流动性很强](/liquidmarket)，这是一个可以轻松执行交易的市场，因为有很多买家和卖家可供选择。相反，[流动](/illiquid)[性差的](/illiquid)市场可能导致交易者由于缺乏市场参与者而难以平仓。增加商品对流动性的还有其经济的稳定性，这支持货币。这些特征使商品对对希望在快速进出交易的同时获得利润潜力的交易者具有吸引力。

### 其他商品国家

尽管有许多国家拥有大量自然资源和商品储备，例如俄罗斯、沙特阿拉伯和委内瑞拉，但其中许多国家的商品通常受到本国政府的高度监管或交易稀少。

## 商品对交易中的主要货币

构成商品对三重奏中非美国部分的三个国家都具有使其货币和商品资源吸引投资者的特定品质。

### 加元

加元兑美元 ( [USD/CAD](/usd-cad-us-dollar-canadian-dollar-currency-pair) ) 的价值与包括石油在内的商品价格高度相关。加拿大广阔的地区拥有相对未受破坏的景观，这意味着该国拥有丰富的自然资源，例如木材和燃料。加拿大经济严重依赖商品的生产和销售。例如，石油和燃料占该国出口的最大份额。因此，石油价格是经济健康的主要驱动力。交易这对货币也称为交易“加元”。加拿大靠近美国意味着这两个经济体紧密相连，这从出口总额中可以看出，2019 年加拿大有 75% 的出口流向美国。

### 澳元

交易澳元兑美元汇率也被称为交易“澳元”。澳大利亚是全球最丰富的煤炭和铁矿石出口国。澳大利亚也拥有广阔的郁郁葱葱的自然景观，是世界上资源最丰富的国家之一。该国还出口石油和黄金，因此其货币严重依赖这些商品价格。

[经济表现如何的](/economicgrowth)见解，因为中国是澳大利亚最大的出口客户。在中国上。强劲的中国经济对世界其他地区以及澳大利亚来说都是好兆头，由于对澳大利亚商品的需求增加，这可能会推高澳元汇率。

### 新西兰元

新西兰是世界上最大的浓缩牛奶出口国，还出口其他乳制品、肉类和羊毛。新西兰与黄金有着密切的联系，并将对商品价格的变动做出反应。将新西兰元兑换成美元 ( [NZD/USD](/nzd-usd-new-zealand-dollar-us-dollar-currency-pair) ) 也称为交易“猕猴桃”。

另一个推动奇异果需求的因素是新西兰的利率通常高于许多其他国家。因此，投资者经常将资金汇往该国以获得更高的收益。其中一些投资者在日本等低利率国家借钱，然后将借来的资金（从[日元](/jpy-japanese-yen)）兑换成新西兰元，投资于新西兰的银行。这种通过从低收益国家借款以投资于高收益国家的方式为投资提供资金的过程称为[货币套利交易](/currencycarrytrade)。

＃＃ 强调

- 商品对是指对商品价格变化敏感的经济体中的货币，这些经济体通常是依赖商品出口来实现其 GDP 的国家。

- 示例包括澳大利亚元、加拿大元和新西兰元以及石油生产国的货币。

- 投资者交易商品货币，部分是为了利用推动这些国家汇率的商品价格波动。
