# 货币市场账户 (MMA)
##什么是货币市场账户？

货币市场账户是您可以在银行和信用合作社开立的计息账户。它们与储蓄账户非常相似，但它们也提供了一些支票账户功能。货币市场账户不同于货币市场共同基金。

货币市场账户支付有竞争力的利率，是您省钱的安全场所。如果您想要可预测的收益和联邦保险账户，您可能需要开设货币市场账户。

与传统或高收益储蓄账户相比，货币市场账户支持更多将资金转出账户的方式。但是，您不应该像支票账户那样使用货币市场账户。您每个月也只能从货币市场账户提款六次，就像将钱从储蓄账户中转出一样。与储蓄账户相比，货币市场账户可能需要更大的最低存款和余额。

##货币市场账户的利弊

总是有例外，没有产品是完美无瑕的。确保了解权衡取舍。

＃＃＃ 优点

- **您可以赚取利息：** 目前，最好的货币市场账户支付大约 0.6%。

- **您的存款已投保：**您的资金在银行或信用合作社的每个账户所有者的保险金额高达 250,000 美元。

- **您的现金可以取用：**您的帐户通常带有借记卡和/或实物支票。

### 缺点

- **大的最低存款要求：**货币市场账户可能需要比传统储蓄账户更大的存款。

- **低于** **收益率** **比其他银行产品：** 存款证可能会支付更具竞争力的收益率。

- **提款有限制：** 虽然您可能会开支票或使用借记卡将资金从您的货币市场账户中转出，但由于联邦授权，存在限制。

##我如何选择最好的货币市场账户？

首先，你应该货比三家。在您当地的银行或信用合作社开始您的搜索，但也将您的搜索扩展到在线银行。在线机构往往提供更高的费率。

在您进行研究时，要考虑的最重要因素之一是货币市场账户的年收益率。年收益率（APY）表示您将在一年中通过复利赚取多少。换句话说，它是您的第一笔存款所赚取的利息，以及在其他利息收入之上所赚取的利息。数字越高，你的钱就越多。

接下来，查看所有帐户限制。检查货币市场账户要求是否使赚取收益或规避费用变得过于困难。大额余额要求（例如 25,000 美元）的情况并不少见。

此外，请务必查看费用，包括如果您在开户后三个月内关闭该帐户是否会收取罚款。注意月费、转账费、运费、非活跃账户费用和其他罚款。

##货币市场账户与其他账户有什么区别

### 货币市场账户与支票账户

虽然货币市场账户可能提供支票写入特权，但这些账户并非旨在像支票账户那样使用。根据联邦授权，您可以从这些账户中提取资金的次数有限制。然而，最好的货币市场账户利率远高于支票账户。

### 货币市场账户与储蓄账户

储蓄账户和货币市场账户有更多的共同点：它们支付利息，并且旨在让您保持储蓄。但是有一些区别。一般来说，您将不得不在货币市场账户中存放比在储蓄账户中更多的钱。使用货币市场账户，您可以获得支票。不要指望您的储蓄账户中有这个工具。

### 货币市场账户与 CD

CD 可以为您支付比货币市场账户更具竞争力的利率，但您的资金在货币市场账户中的流动性比 CD 中的更高。如果您在货币市场账户和 CD 之间做出决定，请在继续之前评估您的目标。

##货币市场账户和货币市场基金有什么区别？

虽然货币市场账户和货币市场基金名称相似，但它们却大不相同。最值得注意的是，货币市场基金不提供 FDIC 保险，您可能会失去本金。以下是它们主要区别的细分。

＃＃ 常问问题

###货币市场账户有什么好处？

MMA 的一些好处包括更高的利率、保险保护、支票书写和借记卡特权。利率高于储蓄账户的诱惑是 MMA 的主要吸引力之一。他们能够提供更高的利率，因为他们被允许投资存款证、政府证券和商业票据，而储蓄账户则无法做到。这些账户还提供了轻松访问资金以及在同一机构的多个账户之间转移资金的灵活性。与储蓄账户不同，许多 MMA 提供一些支票写入特权，并在账户中提供借记卡，就像普通支票账户一样。

### 货币市场账户安全吗？

银行的货币市场账户由联邦政府的独立机构联邦存款保险公司承保。 FDIC 涵盖某些类型的账户，包括 MMA，每家银行的每位存款人最高 250,000 美元。如果存款人在同一家银行有其他可保账户（支票、储蓄、存款证明），它们都计入 250,000 美元的保险限额。对于想要为超过 250,000 美元投保的储户而言，实现这一目标的最简单方法是在多个银行或信用合作社开设账户。联名账户的保险金额为 500,000 美元。

### MMA 的缺点是什么？

潜在的缺点包括有限的交易、费用、提款限制和最低余额要求。银行和信用合作社一般要求客户存入一定数额的资金才能开立账户，并将账户余额保持在一定水平以上。如果余额低于最低限度，许多人将收取月费。虽然一些 MMA 提供有吸引力的利率，但大多数将无法与其他收益率更高的替代品竞争。银行和信用合作社提供多种类型的账户，其中一些具有可以使它们与货币市场账户竞争或优于货币市场账户的功能。尽管美联储修改了提款限制，但银行可能会限制储户从其 MMA 中取款的次数。
