# 价值线综合指数
## 什么是价值线综合指数？

价值线综合指数是一个股票指数，包含来自[纽约证券交易所](/nyse)、[美国证券交易所](/amex)、纳斯达克、多伦多和场外交易市场的大约 1,700 家公司。价值线综合指数有两种形式：价值线几何综合指数（最初的等权指数）和价值线算术综合指数（如果投资组合持有相同数量的股票，则反映变化的指数。）这些指数通常发布在由创始人阿诺德·伯恩哈德创建的价值线投资调查中， Value Line Inc.的首席执行官

## 了解价值线综合指数

该指数获得同名的“价值线”是指伯恩哈德将叠加在价格图表上以使不同公司的价值正常化的现金流的倍数。 Value Line 是最受尊敬的投资研究公司之一。它的业绩记录非常强劲。事实上，从[长远来看，该公司的模型投资组合总体上跑赢了市场](/longrun)。

价值线综合指数由与价值线投资调查相同的公司组成，不包括封闭式基金。

Value Line Index 中的公司数量会根据以下因素而波动，这些因素包括交易所本身的公司增加或退市、合并、收购、破产以及 Value Line 对 Value Line Index 做出的覆盖决策。 Value Line 关于纳入哪些公司的决定旨在创造北美股票市场的广泛代表性。此外，在任何给定交易所上市的公司数量可能会有所不同，因为公司可能会从一个交易所转移到另一个交易所，或者被添加或退市。然而，无论是采用几何还是算术计算，公司在交易所的退市或流动都不是价值线指数方法中的因素。

## 价值线几何综合指数

这是 1961 年 6 月 30 日推出的原始指数。它是使用几何平均值的等权重指数。价值线几何综合指数的每日价格变化是通过将每只股票的收盘价与其前一个收盘价的比率相乘并将该结果与股票总数的倒数相乘而得出的。

## 价值线算术综合指数

该指数于 1988 年 2 月 1 日建立，如果您持有等量的股票投资组合，则使用算术平均值更接近地模拟指数的变化。价值线算术综合指数的每日价格变化是通过添加所有股票的每日百分比变化，然后除以股票总数。

＃＃ 强调

- 指数有两种形式——价值线几何综合指数和价值线算术综合指数。

- 几何综合指数是等权重的，使用几何平均值，每日变化最接近股票价格变化的中值。

- 算术综合指数使用算术平均值，指数的每日变化反映了由等量股票组成的投资组合。

- 价值线综合指数包含来自北美主要市场指数的大约 1,700 只股票。
