# 货币历史
## 什么是货币历史？

货币历史是一个术语，指[外汇市场](/forex)上各种[货币对的历史表现](/foreign-currency-pairs)，代表汇率随时间的差异。

外汇交易者使用这些历史货币汇率来掌握货币对在过去相对于当前趋势的交易情况。对这种历史关系的合理解释有助于预测这些货币的未来价格走势。

## 了解货币历史

汇率的货币历史数据对于外汇交易者以及在海外开展业务并需要将一种货币转换为另一种货币的跨国公司非常有价值。与过去的货币汇率相比，了解今天的汇率是一种有价值的计划和交易工具。

有许多零售外汇经纪商免费提供这些信息。其他人有时会为这些数据收取象征性的费用。在决定从何处获取历史货币汇率数据之前，进行自己的研究非常重要。

历史价格信息至关重要的一个原因是因为在外汇市场交易的[技术分析师数量众多。](/technical-analyst)这种交易方式着眼于过去的表现，以识别可以表示市场趋势、反转以及支撑或阻力位的模式和其他交易信号。这些数据可以合并到[外汇预测](/forex-forecast-and-forecasting-software)[软件包](/forex-forecast-and-forecasting-software)中，以帮助外汇交易者通过技术图表和指标进行交易分析。

## 获取货币历史信息

大多数在线货币经纪商和交易平台都会提供一定程度的价格历史记录，但这些历史记录可能不完整，或者可能无法及时回溯到您的目的。可以从零售外汇经纪商处收集其他信息，但这些信息可能会收取数据费用。

虽然大多数人可能都知道美国国税局 ( [IRS](/irs) ) 是您向其缴纳税款的组织，但投资者和旅行者也可以在这里深入了解货币历史数据。美国国税局发布了有关年平均汇率的信息表，并附有简单的说明：“要将外币兑换成美元，请将外币金额除以适用的年平均汇率。 ”

以下是查找货币历史的其他一些地方：

- 国际货币基金组织 ( [IMF](/imf) ) 还提供大量历史货币数据以及汇率查询工具，以帮助研究过去的关系和趋势。

- 彭博有多种资源可帮助追踪货币、分析它们之间的历史关系并评估其潜在的未来价值。

- OANDA 历史货币转换器提供超过 38,000 种货币对的 25 年历史汇率，并能够将数据下载为有用的格式以进行“抽查、分析和报告” 。

- 缅因大学汇编了一系列额外的资源，可用于获取历史外汇汇率数据。

- 每个国家的中央银行或其他货币当局对于查找历史外汇信息至关重要。可以通过国际清算银行 ( [BIS](/bis) ) 找到中央银行的名单。这些银行通常保留官方汇率记录。

- 外汇交易者在技术分析和外汇图表的过程中利用货币历史来进行更明智的交易。

- 许多在线外汇经纪商向用户提供货币历史记录，但还有其他一些官方资源可用于查找几十年前的汇率数据。
