# 第一次做对（漂移）
## 什么是第一次做对（漂移）？

第一次做对 (DRIFT) 是一种[管理会计](/managerialaccounting)技术或实践，其中心是减少浪费和提高生产过程的效率。第一次做对是库存管理的一部分，只订购需要的库存材料以降低库存成本。第一次做对可以帮助企业减少生产延迟并提高效率。然而，DRIFT 也有其缺点，包括它可能会阻止公司利用对公司产品需求激增的机会。

## 理解第一次做对（漂移）

第一次做对 (DRIFT) 的重要性源于降低闲置库存或原材料成本的目标。 DRIFT 与[即时](/jit)(JIT) 库存相关，这是一个仅接收所需材料的过程，旨在降低库存成本并改善生产管理。换句话说，在 JIT 下，公司在记录销售额之前不会开始生产，从而使库存水平保持在低水平。 DRIFT 背后的理念是，管理层希望构成 JIT 理念的所有流程都能正确有效地完成，因此生产过程中不会出现任何延误。

DRIFT 试图解决 JIT 库存系统的局限性和潜在缺陷。例如，如果在某个生产阶段出现最轻微的错误，就会影响整个生产过程。通过“第一次做对”，公司能够运行平稳的生产过程，而无需携带过多的库存，这将有助于降低生产成本。因此，DRIFT 要求公司建立有效的通信系统来记录销售情况、进行相应的库存采购并根据需要调整生产计划。

## DRIFT 的批评

利用 DRIFT 的公司可以体验到更低的成本和更高的[利润率](/profitmargin)。利润率是每一美元收入产生的利润量。利润率是一个重要的指标，因为它说明了费用控制和收入增长。利润或净收入可以随着收入的增加而增加，但如果费用以更快的速度增长，利润就会受到侵蚀，从而导致利润率下降。在制造过程中，DRIFT 有助于解决费用管理问题并提高利润。但是，DRIFT 和 JIT 生产策略存在一些潜在缺陷，可能导致**降低**利润。

### 防止规模经济

使用 DRIFT 和 JIT 系统的公司失去了实现[规模经济的机会](/economiesofscale)。当产量增加但平均投入成本下降时，就会出现规模经济。产量增加导致的成本降低是由于设备等固定成本保持不变或基本不变。

使用 DRIFT 和 JIT 的公司在购买耗材时也会放弃基于数量的折扣。因此，该公司可能会为每件商品支付更多费用，因为它会进行更小、更频繁的供应订单，而这些订单不符合供应商的价格优惠条件。缺乏折扣会导致更高的单位供应成本并侵蚀利润率。

### 无库存

由于没有库存或材料库存，任何[供应链问题](/scm)或成品需求的意外激增都可能导致向最终客户的交付延迟。延长的延误可能导致客户不满意和订单丢失。

使用 JIT 和 DRIFT 的按需生产也意味着公司必须找到愿意运送频繁的小订单的供应商。如果发生任何中断，例如自然灾害，如果供应商无法交付材料，公司可能会遇到生产延误。批量购买虽然比按需购买成本更高，但可以让公司拥有充足的库存，以应对供应链中断。

### 增加运输成本

频繁向供应商下订单也会导致额外的运输和处理费用。结果可能会增加商品的单位成本，并最终降低公司的利润率。换句话说，额外的运输成本可能会抵消 DRIFT 生产方法旨在创造的利润率增长。

＃＃ 强调

- 尽管 DRIFT 可以降低成本并提高利润率，但公司可能会错过对其商品的需求激增。

- 第一次做对是库存管理的一部分，只订购需要的库存材料以降低库存成本。

- 第一次做对 (DRIFT) 用于管理会计，旨在减少浪费并提高生产效率。
