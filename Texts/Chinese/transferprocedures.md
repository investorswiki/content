# 转移程序
## 什么是转移程序？

转让程序是股票（或其他证券）的所有权从一方转移到另一方的方式。此过程由[转让代理执行，该代理](/transferagent)遵循由证券交易委员会 ( [SEC](/sec) ) 管理的详细、记录在案的一系列步骤，以确保交易已完成。每当买方和卖方之间进行交易（资产从卖方的托管人转移到买方的托管人）时，或者当资产的所有者更换经纪公司或在他们控制的一个或多个经纪账户之间转移资产时，都会使用转移程序。

在帐户转移过程中会同时发生许多事件。即使采用当今的现代技术，从一个客户的帐户成功转移到另一个客户的帐户可能需要长达一周的时间，尽管最好提前计划任何潜在的延迟。在美国，股票被规定在 T+2 个交易日内清算。清算时间从 T+3 年前开始减少，并且随着时间的推移可能会减少。

## 转移程序如何运作

以下有关转移程序的信息由美国金融监管机构[FINRA提供： 如今，经纪账户中持有的大多数资产都是](/finra)通过自动化电子流程在[经纪自营商之间转移的。](/broker-dealer)国家证券清算公司 ( [NSCC](/nscc) ) 运营自动客户账户转账服务 ( [ACATS](/acat) )，以促进客户账户从一个经纪交易商转移到另一个经纪交易商。涉及最常见资产类别的转让，即现金、股票、境内公司发行的公司债券和上市期权，都可以通过ACATS轻松转让。

ACATS 充当转让代理人，记录股票所有者的个人详细信息。当股份所有权发生变化时，转让代理人注销卖方的股票凭证（或其电子记录），并为买方制作新的股票凭证。虽然是自动化的，但账户转账过程有些复杂，并受某些因素和法规的影响，其中最重要的将在下面讨论。

一旦接收公司获得交易信息，它将某些客户数据，包括账户名称、社会安全号码和交付公司的帐号输入到 ACATS。输入数据后不久，自动功能允许交付公司查看已提出转移帐户的请求。一旦客户账户信息正确匹配，并且接收公司决定接受账户，交付公司将需要大约三天的时间将资产转移到新公司。这称为交付过程。总体而言，验证过程和交付过程通常需要大约六天时间才能完成。通常，交付实体不是经纪交易商（例如银行、共同基金或信用合作社）的转账将需要更多时间。此外，转移需要托管人的账户，例如个人退休账户 ( [IRA](/ira) ) 或未成年子女的托管账户，可能需要额外的时间。
