# 股息税收抵免
## 什么是股息税收抵免？

股息税收抵免是加拿大居民对其从加拿大公司收到的股息的总额部分的[纳税义务应用的金额。](/taxliability)总额和股息税收抵免适用于个人，而不是公司。

## 了解股息税收抵免

截至 2018 年，个人从加拿大公司收到的合格股息“总额”增加了 38%。要使[股息](/dividend)被正式确认为合格股息，必须由支付股息的公司指定为合格股息。截至 2019 年，不符合条件的股息的[总](/gross-up)增长率为 15%。将总增长率视为对适用税款的增加。

例如，如果一家公司支付每股 20 美元的股息，投资者将获得每股 20 美元 x 1.38 = 27.60 美元，这意味着他们的税后股息将为每股 20 美元。累计金额作为[应纳税所得额计入](/taxableincome)纳税人的所得税表中。然后，加拿大联邦和省政府都给予个人相当于总额百分比的[税收抵免，这有助于减少实际](/taxcredit)[应纳税额。](/incometaxpayable)

> ### 38%

> 截至 2018 年，个人从加拿大公司获得的符合条件的股息金额已“累计”。

>

例如，假设 Susan Smith 的[有效税率](/effectivetaxrate)为 25%。在 2018 纳税年度，她收到 250 美元的合格股息和 200 美元的不合格股息。要计算联邦股息税收抵免，她必须按[加拿大税务局 (CRA)](/ccra)指定的百分比合计她收到的总股息。在这种情况下，合格股息的百分比为 38%，不合格股息的百分比为 15%。

- = ($250 x 1.38) + ($200 x 1.15)

- = 345 美元 + 230 美元

- = 575 美元

这意味着苏珊报告了 575 美元作为应税收入。由于她的有效税率为 25%，因此她对该收入的税款为：

- = 575 美元 x 0.25

- = 143.75 美元

对于符合条件的股息，联邦股息税收抵免占应税股息的百分比为 15.0198%，不符合条件的股息为 9.0301%。她在联邦一级的股息税收抵免将是：

- = ($345 x 0.150198) + ($230 x 0.090301)

- = 51.82 美元 + 20.77 美元

- = 72.59 美元

因此，税收抵免将苏珊的原始纳税义务减少到 143.75 美元 - 72.59 美元 = 71.16 美元。

请注意，有联邦和省税收抵免。例如，如果苏珊住在阿尔伯塔省，她可以申请 10% 的省级税收抵免，当应用于她的股息时，可以进一步减少她的纳税义务。

股息税收抵免是为了抵消[双重征税而实施](/double_taxation)[的不可退还的抵免，](/nonrefundabletaxcredit)因为股息是通过公司的税后利润支付给股东的，股东收到的股息也被征税。从外国公司收到的股息不受总额和股息税收抵免机制的约束。因此，您将为外国公司的股息支付更高的税率。

＃＃ 强调

- 总额和股息税收抵免仅适用于个人。

- 加拿大居民针对从加拿大公司收到的股息总额部分的税收负债申请股息税收抵免。

- 通常有联邦和省税收抵免。
