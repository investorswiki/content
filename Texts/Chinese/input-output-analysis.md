# 投入产出分析
## 什么是输入输出分析？

投入产出分析（IO）是一种基于不同经济[部门](/sector)或行业之间相互依存关系的宏观经济分析形式。这种方法通常用于估计正面或负面经济冲击的影响，并分析整个经济体的连锁反应。 IO 经济分析最初由[Wassily Leontief](/wassily-leontief) (1906-1999) 开发，他后来因其在该领域的工作而获得[诺贝尔经济学奖。](/nobel-memorial-prize-in-economic-sciences)

## 理解输入输出分析

IO分析的基础涉及输入输出表。此类表格包括一系列数据行和列，用于量化经济体所有部门的[供应链](/supplychain)。行业列在每行和每列的标题中。每列中的数据对应于该行业生产函数中使用的投入水平。

例如，汽车制造栏显示了制造汽车所需的资源（例如，钢、铝、塑料、电子产品等的数量）。 IO 模型通常包括单独的表格，显示每单位投资或生产所需的劳动力数量。

[新古典经济学或西方的政策顾问](/neoclassical)通常不使用投入产出分析，但它已被用于依赖[中央计划者的协调经济的](/centrally-planned-economy)[马克思主义经济分析](/marxian-economics)。

## 三种类型的经济影响

IO 模型估计三种类型的影响：直接的、间接的和诱导的。这些术语是指当改变给定的投入水平时，在整个经济中波及的初始、二次和三次影响的另一种方式。通过使用 IO 模型，[经济学家](/economist)可以估计由于一个或多个特定行业的投入变化而导致的跨行业产出变化。

- 经济冲击的直接影响是支出的初始变化。例如，建造一座桥梁需要在水泥、钢铁、建筑设备、劳动力和其他投入上的支出。

- 间接或次要影响是由于投入品供应商雇用工人来满足需求。

- 诱发性或第三性影响将来自供应商的工人购买更多的商品和服务供[个人消费](/pce)。这种分析也可以反向运行，看看对输入的影响可能是观察到的输出变化的原因。

## 输入输出分析示例

这是 IO 分析如何工作的示例。当地政府想要建造一座新桥，需要证明投资成本是合理的。为此，它聘请了一名经济学家进行 IO 研究。

这位经济学家与工程师和建筑公司交谈，以估计这座桥的成本、所需的物资以及建筑公司将雇用多少工人。

经济学家将这些信息转换为美元数字，并通过 IO 模型运行数字，从而产生三个层次的影响。直接影响只是输入模型的原始数字，例如[原始输入](/rawmaterials)（水泥、钢铁等）的值。

间接影响是供应公司创造的就业机会，如水泥和钢铁公司。这些公司需要雇用工人来完成项目。他们要么有资金这样做，要么不得不借钱这样做，这将对银行产生另一个影响。

引发的影响是新工人为自己和家人在商品和服务上花费的金额。这包括食物和衣服等基本生活，但现在他们有更多的[可支配收入](/disposableincome)，这也与享受的商品和服务有关。

IO 分析研究了当地政府想要建造一座新桥对经济各个部门的连锁反应。这座桥可能需要政府利用税收来支付一定的成本，但 IO 分析将显示该项目通过雇佣公司所产生的好处，这些公司雇佣了在经济中花费的工人，帮助它发展。

＃＃ 强调

- 在投入产出分析中模拟了三种类型的影响。它们是直接影响、间接影响和诱发影响。

- 投入产出分析用于估计正面或负面经济冲击的影响，并分析整个经济的连锁反应。

- 投入产出表是投入产出分析的基础，描绘了量化经济所有部门供应链的数据行和列。

- 这些对经济的影响是在某些投入水平发生变化时确定的。

- 投入产出分析的使用在西方世界或新古典经济学中并不常见，但在需要对经济进行中央计划时经常在马克思主义经济学中使用。

- 投入产出分析是基于不同经济部门或行业之间相互依存关系的宏观经济分析。
