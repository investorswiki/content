# 子弹贷款
## 什么是子弹贷款？

子弹贷款通常被称为气球贷款，需要[气球付款](/balloon-payment)，通常是大笔余额，以及在贷款到期时到期的最终付款。

在整个借贷期间，唯一支付的款项（如果有的话）是利息费用，而最初借入的本金将在借贷期限结束时支付。子弹贷款在商业房地产中比在住宅房地产中更常见。

## 子弹贷款如何运作

大多数子弹贷款是针对房地产开发商的土地合同发放的。子弹贷款不会在票据期限内完全摊销，因此到期时会有大量本金余额。 “子弹”一词是指大笔一次性付款，通常是本金的全部价值，在贷款到期时到期。

[承销程序](/underwriting)确定的最高本金金额的批准。然后可以通过几种不同的方式构建贷款，具体取决于借款人希望如何偿还。

子弹贷款借款人可能会在贷款期限内获得零还款选项或仅支付利息。

> 子弹贷款要求借款人在其期限结束时一次性支付大笔款项。

>

当提供零还款贷款时，利息将根据贷款条款累积，通常为每月或每年，借款人将被要求在到期时以大笔本金和应计利息的形式支付总余额。

在只付利息的子弹贷款中，借款人需要定期支付利息。这将到期时的子弹或气球付款减少到贷款总本金的金额。

子弹贷款通常被认为是短期融资，并且可以提供不同的期限，具体取决于借款人预计多久还款。由于向借款人提供的灵活性，贷方通常对子弹贷款收取更高的利率。

子弹贷款可以有担保或无担保。通常，子弹或气球贷款用于购买未开发的土地，与完全开发的房产相比，它提供的抵押品更少。一些开发商可能会选择使用子弹贷款购买单片土地，而另一些开发商可能会使用子弹贷款来开发具有多片土地的整个细分市场。

##子弹贷款的利弊

子弹头贷款具有较低的仅利息或零还款额和贷款结构灵活性的优势。但是，子弹贷款也可能具有相对较高的利率和贷款期末还款额较大的缺点。

开发商通常最初从建筑项目的子弹贷款中受益，并根据他们对项目完成所需时间的预期来安排贷款期限。然而，开发商可能无法从项目中获得现金流来支持定期贷款支付，直到项目完成，当他们有不动产可供出售以支付贷款成本时。子弹付款也可能在现金流开始之前到期。

许多建筑商选择[外卖贷款](/take-out-loan)为债务再融资。在外卖贷款中，借款人提供新建成的建筑物作为新贷款的[抵押品](/collateral)，然后用这笔钱偿还现有的子弹贷款。

＃＃ 常问问题

###子弹贷款和摊销贷款有什么区别？

典型的摊销贷款时间表要求在借款期限内逐步偿还贷款本金。但是，子弹贷款需要在到期日一次性偿还贷款本金。

### 谁有资格获得子弹贷款？

子弹贷款遵循相同的承销流程，可能包括预先批准和信用评分、收入和资产验证，以及类似于定期摊销期权的财产评估。

### 如何计算子弹支付？

- 付款 = (A * i * (1 + i)ⁿ) / ((1 + i)ⁿ - 1)其中：- 付款 = 每月付款- A = 贷款金额- i = 定期利率- n = 期数计算在子弹或气球贷款期限后到期的余额： - B = (A * (1 + i)ⁿᵇ) - Pmt / i * ((1 + i)ⁿᵇ - 1)其中：- B = Bullet payment- nb = 子弹贷款期数
