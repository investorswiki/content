# 经营现金流量保证金
## 什么是营运现金流量保证金？

经营现金流量利润率是衡量[经营活动现金](/cash-flow-from-operating-activities)占给定期间总销售收入百分比的现金流量比率。

与营业利润率一样，它是衡量公司盈利能力和效率以及盈利质量的可靠指标。

## 了解经营现金流量保证金

营运现金流利润率衡量公司将销售转化为现金的效率。这是一个很好的[收益质量指标，](/qualityofearnings)因为它只包括涉及实际资金转移的交易。

因为[现金流](/cashflow)是由收入、间接费用和运营效率驱动的，所以它非常有说服力，尤其是在将业绩与同行业的竞争对手进行比较时。运营现金流是否转为负数，因为公司正在对其运营进行投资以使其更有利可图？或者公司是否需要注入外部资金来争取时间继续经营，拼命扭转业务？

正如公司可以通过更有效地利用[营运资金来提高运营现金流边际一样](/workingcapital)，他们也可以通过延迟应付应付账款的支付、追逐客户付款或减少库存来暂时压平运营现金流边际。但如果一家公司的经营现金流量利润率逐年上升，则表明其[自由现金流](/freecashflow)（FCF）正在改善，其扩大资产基础和为股东创造长期价值的能力也在改善。

## 营运现金流量保证金与营运保证金

营运现金流量边际不同于[营运边际](/operatingmargin)。营业利润率包括折旧和摊销费用。然而，运营现金流边际增加了非现金费用，例如折旧。

营业利润率的计算方法是营业收入除以收入。除了使用营业收入外，这类似于营业现金流量利润率。营业现金流量利润率使用营业现金流量而不是营业收入。

自由现金流边际是另一种现金边际衡量指标，它还增加了资本支出。在资本密集型行业，固定成本与可变成本的比例很高，由于[经营杠杆的作用](/operatingleverage)，销售额的小幅增长可以导致经营现金流的大幅增加。

## 经营现金流量保证金示例

经营现金流 = 净收入 + 非现金费用（折旧和摊销）+ 营运资金变动

假设 ABC 公司记录了 2018 年业务活动的以下信息：

- 销售额 = 5,000,000 美元

- 折旧 = 100,000 美元

- 摊销 = 125,000 美元

- 其他非现金费用 = 45,000 美元

- 营运资金 = 1,000,000 美元

- 净收入 = 2,000,000 美元

并记录了2019年经营活动的以下信息：

- 销售额 = 5,300,000 美元

- 折旧 = 110,000 美元

- 摊销 = 130,000 美元

- 其他非现金费用 = $55,000

- 营运资金 = 1,300,000 美元

- 净收入 = 2,100,000 美元

我们将 2019 年经营活动产生的现金流量计算为：

- 经营活动产生的现金流量 = 2,100,000 美元 + (110,000 美元 + 130,000 美元 + 55,000 美元) + (1,300,000 美元 - 1,000,000 美元) = 2,695,000 美元

为了得出营运现金流边际，这个数字除以销售额：

- 营运现金流利润率 = 2,695,000 美元 / 5,300,000 美元 = 50.8%

＃＃ 强调

- 营运现金流边际的计算方法是营运现金流除以收入。

- 这就是它与营业利润率的区别，营业利润率使用不包括折旧等费用的营业收入。

- 营运现金流利润率揭示了公司将销售转化为现金的效率，是衡量盈利质量的良好指标。

- 该比率使用营运现金流，增加了非现金费用。

＃＃ 常问问题

### 经营现金流利润率越高还是越低越好？

更高的比率总是更好，因为它表明更大比例的收入正在转化为现金流。

### 营运现金流利润率与营运利润率有何不同？

### 什么是运营现金流？

也称为来自经营活动的现金流量，或缩写为 CFO，这个数字表示流经公司与其核心业务活动相关的资金量。
