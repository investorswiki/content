# 收益调用
## 什么是收益率？

赎回收益率 (YTC) 是一个金融术语，是指债券持有人在债券持有至赎回日期之前获得的回报，赎回日期发生在债券到期之前的某个时间。这个数字可以在数学上计算为债券未来票面支付的现值和赎回价格等于债券当前市场价格的复合利率。

收益率适用于[可赎回债券](/callablebond)，这是一种债务工具，可让债券投资者在所谓的赎回日以赎回价格赎回债券（或债券发行人回购债券）。根据定义，债券的赎回日按时间顺序发生在[到期日之前](/maturitydate)。

一般来说，债券可以在几年内赎回。尽管确切的[赎回价格](/callprice)基于现行市场价格，但它们通常以高于其面值的溢价被赎回。

## 理解收益率调用

许多债券是可赎回的，尤其是市政债券和公司发行的债券。如果利率下降，发行债券的公司或市政当局可能会选择偿还未偿债务并以更低的成本获得新的融资。

计算赎回此类债券的收益率很重要，因为它揭示了投资者将获得的回报率，假设：

1. 债券在可能的最早日期被赎回

1.债券以当前市场价格购买

1. 债券持有至赎回日

[到期收益率](/yieldtomaturity)更准确地估计债券的预期回报。

## 计算收益率

虽然用于计算调用收益率的公式乍一看有点复杂，但实际上非常简单。

计算赎回收益率的完整公式是：

**P = (C / 2) x {(1 - (1 + YTC / 2) ^ -2t) / (YTC / 2)} + (CP / (1 + YTC / 2) ^ 2t)**

在哪里：

P = 当前市场价格

=年[付息](/coupon)

CP = 买入价

t = 距离调用日期剩余的年数

YTC = 调用收益率

根据这个公式，调用的收益不能直接求解。如果手动进行计算，则必须使用迭代过程来找到要调用的产量。幸运的是，许多计算机软件程序都具有“求解”功能，只需单击鼠标即可计算这些值。

## Yield-To-Call 示例

例如，考虑一张[面值](/facevalue)为 1,000 美元的可赎回债券，每半年支付 10% 的息票。该债券目前的定价为 1,175 美元，五年后可选择以 1,100 美元的价格赎回。请注意，到期前的剩余年份对于此计算无关紧要。

使用上述公式，计算将设置为：

$1,175 = ($100 / 2) x {(1- (1 + YTC / 2) ^ -2(5)) / (YTC / 2)} + ($1,100 / (1 + YTC / 2) ^ 2(5))

通过一个迭代过程，可以确定该债券的赎回收益率为 7.43%。

＃＃ 强调

- 术语“赎回收益率”是指债券持有人在到期日之前持有证券直至赎回日所获得的回报。

- 可以使用计算机程序以数学方式计算调用收益率。

- 赎回收益率适用于可赎回债券，即让债券投资者以赎回价格提前赎回债券（或债券发行人回购债券）的证券。
