# IP地址
## 什么是 IP 地址？

IP地址代表互联网协议地址；它是与特定计算机或计算机网络相关联的识别号。当连接到互联网时，IP 地址允许计算机发送和接收信息。

## IP 地址的工作原理

IP 地址允许计算机通过 Internet 发送和接收数据。大多数 IP 地址都是纯数字的，但随着 Internet 使用量的增长，一些地址已经添加了字母。

有四种不同类型的 IP 地址：公共、私有、静态和动态。虽然公共和私有表示网络的位置——私有在网络内部使用，而公共在网络外部使用——但静态和动态表示永久性。

静态 IP 地址是手动创建的，而不是分配的。静态地址也不会更改，而动态 IP 地址已由动态主机配置协议 (DHCP) 服务器分配并且可能会更改。动态 IP 地址是最常见的 Internet 协议地址类型。动态 IP 地址仅在一定时间内处于活动状态，之后它们就会过期。计算机将自动请求新的租约，或者计算机可能会收到一个新的 IP 地址。

可以将 IP 地址与[社会安全号码](/ssn)(SSN) 进行比较，因为每个地址对于分配给它的计算机或用户来说都是完全唯一的。这些数字的创建允许路由器识别它们在互联网上发送信息的位置。他们还确保正确的设备正在接收正在发送的内容。就像邮局需要一个邮寄地址来投递包裹一样，路由器也需要一个 IP 地址来投递到所请求的网址。

## IP 地址示例

暗网是指未被传统搜索引擎索引的加密在线内容。暗网有一个非法的在线市场，犯罪分子可以在其中交易非法和非法商品。其中许多交易都是使用在线加密货币比特币进行的，这使得当局更难追踪和捕获参与这些交易的人。

2018 年，经过长达一年的政府行动，与国土安全部合作的特工冒充武器经销商，以获取试图非法购买武器的嫌疑人的计算机。这使他们能够访问 IP 地址，用于追踪使用暗网的其他嫌疑人的地理位置。

这不是 IP 地址第一次导致逮捕。 2012 年，警方使用 IP 地址追踪并逮捕了黑客组织 Lulzsec 的成员。利用搜查令从[互联网服务提供商 (ISP)获取信息](/isp)，执法人员能够追踪黑客的实际地址，并以他们的非法互联网活动逮捕他们。

＃＃ 强调

- 互联网协议 (IP) 地址允许计算机发送和接收信息。

- 有四种类型的 IP 地址：公共、私有、静态和动态。

＃＃ 常问问题

### 什么是 IPv6 地址？

IPv4 地址由四个以点分隔的一到三位数字组成。 IPv6 是较新版本的 Internet 寻址，它使用 128 位字母数字值来识别在线端点设备。Ipv4 地址可能类似于 208.80。 154.224，而 IPv6 地址可能如下所示：2001:0db8:85a3:0000:0000:8a2e:0370:7334。IPv6 允许在 Internet 上存在更多连接的设备，而无需重复地址，这可能会导致问题。

### 有人如何“欺骗”他们的 IP 地址？

您可以通过虚拟专用网络 (VPN) 使您的 IP 地址看起来像是来自其他地方。这些服务实质上通过虚拟加入它们建立的网络来掩盖您的计算机的互联网地址。因此，您的 IP 地址将显示为他们的。黑客还可能使用更复杂的技术来欺骗 IP 地址，这些技术会修改恶意数据包标头中的源地址，以模仿可信源并接受它。

### 如何找到我的 IP 地址？

有几种方法可以确定您的 IP 地址。也许最简单的方法是在像谷歌这样的互联网搜索引擎中输入“我的 IP 地址是什么”。对于 Windows 用户，您可以通过打开 [开始 > 设置 > 网络和互联网] 找到您的 IP，然后选择您连接的 Wi-Fi 网络。在“属性”下，找到“IPv4 地址”旁边列出的 IP 地址。
