# 股票挂钩证券 (ELKS)
股票挂钩证券是一种与股票市场基准挂钩的可变支付[债务](/debtinstrument)[工具。](/debtinstrument)这些证券是另一种固定收益投资——结构性产品，最常被创建为[债券](/bond)。股票挂钩证券通常用于私人市场公司资本融资，并提供给投资者以筹集公司资本。因此，它们不在金融市场交易所交易。

## 什么是股票挂钩证券 (ELKS)？

股票挂钩证券类似于股票和债券。因此，尽管它们可能是债务证券，但与股票挂钩的证券提供的回报与某种形式的[标的](/underlying)股票挂钩——因此得名。这种股权通常是[普通股](/commonstock)。这意味着回报与标的股票的向上和向下运动有关。

ELKS 通常在一年内成熟。他们支付的收益率通常高于基础证券的收益率。他们还在成熟之前向投资者进行两次支付或分配，这就是投资者更喜欢这类投资的原因。

> 股票挂钩证券通常在一年内到期。

>

## 了解股票挂钩证券 (ELKS)

与股票挂钩的证券发行为公司提供了另一种安排向投资者支付[利息的方式。](/interest)发行人可以根据一系列股票市场产品支付担保利息，包括股票、一组股票或股票指数。

他们还可以限制或支付基准回报的特定部分。标准的股票挂钩证券结构为债券将提供与股票[基准](/benchmark)挂钩的可变利息支付和到期时的本金回报。 ELKS 为发行人提供受控[利率](/interestrate)产品。

## 股票挂钩证券的类型

投资者可能有机会从几个不同的发行人那里投资 ELKS。他们还可能会发现 ELKS 被宣传为与市场挂钩。以下是[市面上](/market)能买到的几种ELKS 。

###企业ELKS

公司通常与投资银行合作，以支持为资本融资构建与股票挂钩的证券产品。加拿大皇家银行 (RBC) 是[结构性融资](/structuredfinance)股票挂钩证券的主要来源。 RBC 与公司合作，构建具有各种条款的股票挂钩证券产品。

### 银行提供的 ELKS

散户投资者可能会看到银行提供的股票挂钩证券以及存款证明。股票挂钩证券可以是任何类型的投资，其利息支付与股票基准挂钩。联合银行将股票挂钩 CD 宣传为其市场挂钩 CD 产品的一个组成部分。 CD 的利息与股票指数挂钩。最低投资额为 4,000 美元。

### 市场挂钩证券

整个投资行业都提供与市场基准相关的支付证券。与市场挂钩的证券可以将支付与任何类型的市场基准挂钩。发行人可以构建与市场挂钩的证券，以根据股票基准进行支付。他们还可以使用任何其他市场基准，例如黄金或[货币](/currency)。

对于证券发行人，与市场挂钩的产品提供了通过选择特定基准来控制向投资者付款的机会。对于投资者来说，他们可以提供一个简单的替代方案来投资基准本身。与黄金挂钩的 CD 的投资者通常会寻求获得与黄金相同的回报率。发行人可以通过多种方式构建与市场挂钩的产品。与市场挂钩的产品也被认为是非[流动性](/illiquid)的，并且在投资期间不可交易或赎回而不受惩罚。

### 股票挂钩证券的例子有哪些？

ELKS 的一些示例是公司 ELKS、银行提供的 ELKS 以及通过存单或其他代表一揽子证券的工具提供的与市场挂钩的证券。

### 股票挂钩票据 (ELN) 如何运作？

ELN 以执行价格购买，这是现货价格的折扣。 ELN 发行人在达到或达到行使价时将股票交付给投资者。

### 股票挂钩票据是股票证券吗？

股票挂钩票据支付与标的证券表现挂钩的回报，而股票挂钩证券支付固定利率。
