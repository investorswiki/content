# Linux
##什么是Linux？

Linux 是一种流行的开源、高度可定制的操作系统 (OS)。它促进了软件和硬件之间的通信，为当今世界的大部分提供动力——从计算机到汽车。

Linux 由 Linus Torvalds 于 1991 年创建，是一种开源软件，现在已成为各种系统的一部分。作为开源，Linux 可免费用于商业和非商业用途。任何人都可以根据其许可条款使用、修改和分发源代码。

由于其灵活性，Linux 存在于广泛的应用程序中，包括台式计算机、笔记本电脑、Web 服务器、[超级计算机](/supercomputer)、智能手机、电视、路由器、数码相机、乐器和车辆。

从技术上讲，Linux 是一个内核：一个计算机程序，它是操作系统的核心元素。简而言之，内核是将计算机代码翻译成硬件可以理解的二进制数据。

单独使用 Linux 内核看起来不像大多数人习惯的计算机体验。因此，世界各地的开发人员都添加了额外的组件和应用程序，以使其更加用户友好。因此，Linux 通常作为包（软件工具的集合）分发。这些修改过的 Linux 版本称为发行版（Linux 发行版的缩写）。

### Linux 发行版有哪些类型？

随着时间的推移，社区和公司已经使用 Linux 来创建适合他们偏好或需求的系统。来自世界各地的不同 Linux 发行版的列表很长。虽然有些完全基于社区（如 Debian 或 Arch Linux），但有些则有企业支持（如 Fedora 或 openSUSE）。不同的发行版旨在做不同的事情。例如，它们可能专注于隐私（TAILS）、可用性（Ubuntu）或定制（Arch）。

### 为什么有人要使用 Linux？

总的来说，Linux 是一个多功能、高效且安全的系统。它可以根据用户的喜好进行定制。它的轻量级基础架构可以提高计算机速度，甚至可以用来振兴旧机器。许多 Linux 发行版使用户能够选择满足他们需求的系统 - 无论是简单性、安全性、隐私性还是高级个性化。
