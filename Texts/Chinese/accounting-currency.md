# 会计货币
## 什么是会计货币？

记账本位币是在公司[总账中记录交易时使用的货币单位](/generalledger)，通常也称为公司的“账簿”或会计记录。记帐货币也可以称为[报告货币](/reportingcurrency)。

会计（报告）货币不一定与[功能](/functional-currency)或交易货币相同。功能货币是员工和客户在进行交易（例如销售）时使用的货币。对于在许多不同国家开展业务的大型跨国公司而言，这种差异尤为重要。

## 了解会计货币

在多个国家/地区开展业务通常需要使用多种[货币进行](/currency)[业务交易](/transaction)。在这种情况下，编制[财务报表](/financial-statements)的公司总部或母公司的货币被视为记账货币。对于在使用美元 ( [USD](/usd) )、欧元 ( [EUR](/euro) ) 或英镑 ( [GBP](/gbp) ) 等主要货币的国家/地区运营的公司，记账货币可能与功能货币相同。在较小的市场中以“次要”货币经营的公司更有可能拥有本国记账货币和外币功能货币。

例如，一家位于东京的日本电子公司可能会使用日元 ( [JPY](/jpy-japanese-yen) ) 作为其记账货币，因为这是公司总部和运营所在的当地货币。公司在记录交易时可能会使用其本国货币或当地货币，即使销售是以外币计价的。因此，在中国开展业务的日本公司将使用日元作为记账货币，即使销售交易使用的是人民币 ( [CNY](/cny-china-yuan-renminbi) )。

## 转换为会计货币

对于管理多种货币的公司或投资者而言，[外汇](/foreign-exchange)汇率和兑换的相互作用会使会计记录的维护成为一项复杂的任务。在日常运营中使用不同货币的其他分支机构或子公司必须将其财务报表转换为会计货币，以便合并报表。这是使用[货币换算的](/currency-translation)[时间](/temporalmethod)或[当前汇率](/currentratemethod)方法完成的。

### 时间方法

在时间法（也称为历史法）中，资产和负债分为货币和非货币两类。现金、投资和应收账款等高流动性资产被视为货币资产。同样，应在短期内支付的负债，例如应付账款和应付工资，被视为货币负债。

在这种方法下，货币资产和负债采用资产负债表日的汇率折算。另一方面，非货币[资产](/functional-currency)和[负债的](/liability)[汇率价值](/exchangerate)是基于这些资产和负债的获得或发生时间。非货币资产的一个例子是购买固定资产，例如一件设备或一块土地。

### 当前费率方法

[表](/balancesheet)日的汇率折算。这可能会产生更高程度的[换算风险](/translationrisk)，因为当前汇率可能会在会计期结束之前发生巨大变化。

＃＃ 强调

- 记账货币是用于公司官方记账的货币。

- 时间法和现行汇率法是国外子公司货币换算为母公司货币的两种常用方法。

- 记账货币通常与公司总部的当地货币相同，但可能与使用的交易货币不同。

- 在日常运营中使用不同货币的子公司必须将其财务报表转换为记账货币，以便合并财务报表。
