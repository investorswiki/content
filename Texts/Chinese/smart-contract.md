# 智能合约
智能合约是一种计算机软件，被设计为自动执行的合约，这意味着它在满足预定条件后触发某些动作。例如，智能合约可以用作在两方之间交换加密货币（或任何其他数字资产）的数字协议。一旦设定了协议的条款，智能合约就会验证它们的履行情况，并根据这些条款分配资产。

换句话说，智能合约基本上是在满足特定条件后执行特定功能的代码行。代码通常遵循触发预定和可预测动作的“if...then...”语句。

例如，在线商店可能会实施智能合约，以确保“如果收到付款，则交付产品”——这将使整个过程更加高效，更不容易出现人为错误。

尽管智能合约在区块链和加密货币的背景下变得流行起来，但这个概念是由美国密码学家尼克·萨博在 1994 年首次描述的，比比特币的诞生早了很多年。

智能合约在区块链空间和加密货币市场中发挥着重要作用，特别是关于 ERC-20 代币，它代表了在以太坊网络上创建的遵循[ERC-20](/erc-20)标准的一类代币。这些代币通常通过首次代币发行 ( [ICO](/initial-coin-offering-ico) ) 活动分发，并且使用智能合约可以在销售期间实现无需信任且具有成本效益的资金交换。它们的使用还可以促进去中心化应用程序 ( [DApps](/decentralized-applications-dapps) ) 或去中心化交易所 ( [DEX )](/decentralized-exchange)的支付处理。

智能合约适用的另一个领域是金融服务行业。例如，该技术可用于自动化交易的清算和结算、债券息票的支付，甚至保险索赔的计算和支付。

尽管智能合约在金融领域有明显的应用，但其用途广泛，几乎适用于任何需要在各方之间转移资金、数字资产或任何类型的数字信息的行业。例如，设备租赁行业已经在现实世界中广泛使用这些合同来提高租赁协议的效率。

在医疗保健领域，正在探索该技术作为临床试验中数据操纵的对策。智能合约甚至可以通过建立共享所有权的明确记录并相应地分配知识产权的所有版税和收益来执行知识产权协议。
