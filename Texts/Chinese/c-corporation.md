# C公司
## 什么是 C 公司？

AC 公司（或 C-corp）是公司的法律结构，其中所有者或股东与实体分开征税。 C公司是最普遍的公司，也需要缴纳企业所得税。对企业利润征税是在公司和个人层面上进行的，造成了双重征税的情况。

C-corps 可以与[S 公司](/subchapters)和有限责任公司 (LLC) 等进行比较，它们也将公司的资产与其所有者分开，但具有不同的法律结构和税收待遇。一种较新的组织类型是 B 型公司（或福利公司），它是一家营利性公司，但在目的、责任和透明度方面与 C 型公司不同，但在征税方式上没有区别。

## C 公司如何运作

公司在将剩余金额以股息的形式分配给股东之前，先对收益缴纳公司税。然后，个人股东就其收到的股息缴纳个人所得税。尽管双重征税是不利的结果，但能够以较低的公司税率将利润再投资于公司是一个优势。

AC 公司必须每年至少为股东和董事举行一次会议。必须保留会议记录以显示业务运营的透明度。 AC 公司必须保存公司董事的投票记录以及所有者姓名和所有权百分比的列表。此外，企业必须在主要营业地点的场所拥有公司章程。 C 公司将提交年度报告、财务披露报告和财务报表。

### 组织 C 公司

成立 C 公司的第一步是选择并注册一个未注册的企业名称。注册人将根据该州的法律向州务卿提交[公司章程。](/articlesofincorporation) C 公司向股东提供股票，股东在购买后成为公司的所有者。股票的发行是在企业成立时发行的。

所有 C 类公司都必须提交 SS-4 表格以获得雇主识别号 (EIN)。尽管不同司法管辖区的要求有所不同，但 C 公司必须提交州税、收入税、工资税、失业税和残疾税。除了注册和税务要求外，公司还必须成立董事会来监督整个公司的管理和运营。任命董事会旨在解决[委托代理困境](/principal-agent-problem)，即当代理人代表委托人工作时会出现道德风险和利益冲突。

> C 公司是最常见的公司类型，而不是 S 公司或 LLC。

>

## C 公司的好处

C 公司限制董事、股东、雇员和管理人员的个人责任。这样，企业的法律义务就不能成为与公司相关的任何个人的个人债务义务。随着所有者的变化和管理层成员的更换，C 公司继续存在。

AC 公司可能有许多所有者和股东。但是，在达到特定阈值时需要向证券交易委员会 (SEC) 注册。提供股票的能力使公司能够获得大量资金，这些资金可以为新项目和未来的扩张提供资金。

＃＃ 强调

- C公司限制了投资者和公司所有者的责任，因为他们在企业倒闭中可能损失的最大金额是他们投资的金额。

- C 公司被授权举行年度会议，并拥有由股东投票选出的董事会。

- AC Corporation 在法律上将所有者或股东的资产和收入与公司的资产和收入分开。
