# 稀有物品
##什么是稀有物品？

不寻常的项目是不被视为正常业务运营一部分的非经常性或一次性收益或损失。不寻常的收益或损失可以作为持续经营收入的单独组成部分记录在损益表中，或者可以在财务报表的[脚注或](/footnote)年度报告的管理层讨论和分析 ( [MD&A ) 部分中识别。](/mdanalysis)

## 了解不寻常的物品

单独报告异常项目对于确保财务报告的透明度非常重要。由于不寻常的项目不太可能再次发生，因此将这些项目分开——无论是在[损益表中](/incomestatement)，还是在管理层讨论和分析 (MD&A) 或脚注中——可以让投资者更好地评估核心业务活动的创收能力。

不寻常的项目可能包括：

- 重组费用，包括遣散费和工厂关闭

-[资产减值](/impairedasset)费用或[冲销](/impairedasset)

[终止经营](/discontinued-operations)造成的损失

- 提前偿还债务造成的损失

-[并购](/mergersandacquisitions)或[资产剥离](/divestiture)相关费用

- 出售资产的收益或损失

- 诉讼的收益或损失

- 自然灾害造成的损失成本或运营放缓

- 会计政策变更产生的费用

负责发布公认会计原则 ( [GAAP ) 的独立非营利组织](/gaap)财务会计准则委员会 ( [FASB )](/fasb)已给予管理层回旋余地，以便在适当的时候在损益表中提供更具描述性的单独项目，例如“飓风造成的损失”到办公楼。”

## 特别注意事项

对不寻常项目的处理对公司业绩分析和股票估值、信贷协议和高管薪酬计划有几个影响。分析师必须对损益表进行调整，以生成“干净的” [EBIT](/ebit) 、 [EBITDA](/ebitda)和净收入数据，以此计算[价格倍数](/pricemultiples)。债务协议必须明确排除如何计算某些契约。高管薪酬计划也需要解释薪酬公式中如何处理不寻常的项目。
