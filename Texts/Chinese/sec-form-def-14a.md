# SEC 表格 DEF 14A
## SEC 表格 DEF 14A 是什么？

SEC 表格 DEF 14A 是向[证券交易委员会 (SEC)提交的文件，在需要](/sec)[股东](/shareholder)投票时必须由注册人或代表注册人提交。 SEC 表格 DEF 14A 最常与年度会议代理一起使用。该表格应向证券持有人提供足够的信息，以使他们能够在即将举行的证券持有人会议上进行知情投票或授权[代理人](/proxy)代表他们投票。

SEC 表格 DEF 14A 包括有关证券持有人会议的日期、时间和地点、代理人的可撤销性、异议人的评估权、提出请求的人、某些人在待采取行动的事项中的直接或间接利益、修改等信息或交换证券、投票程序和其他敷衍细节。普通投资者经常忽略表格 DEF 14A。它包含有关公司治理的关键细节，列在下一节中，受到[激进主义者](/activist-investor)和志同道合的投资者的认真审查。

## 了解 SEC 表格 DEF 14A

[根据 1934 年证券交易法](/seact1934)[第](/seact1934)14(a) 条，SEC 表格 DEF 14A，也称为“最终代理声明” 。该表格在向股东提供最终[代理声明时提交给 SEC](/proxystatement) ，有助于 SEC 确保股东的权利得到维护。委托书有助于股东在为拟议项目投票时了解公司治理实践。

在表格 DEF 14A 的开头，列出了可供投票的项目。它们通常包括批准重新选举董事、批准以咨询为基础的高管薪酬（所谓的“薪酬决定权”）、批准审计费用以及批准审计公司的持续参与。在许多情况下，代理备案将要求批准新的或修订的高管薪酬计划。有时，投票选票上会出现股东对某个特定事项的投票。一个例子是取消从使用抗生素或激素饲养牲畜的农场采购肉类。

## 揭开公司治理实践的面纱

SEC 表格 DEF 14A 是股东的主要文件，用于了解[董事会的组成](/boardofdirectors)以及他们如何监督公司的管理。董事会负责委员会的组建和运作，其中最重要的是薪酬委员会。代理文件的大部分内容专门用于讨论高管薪酬实践和理念，以及高管和董事薪酬组成表格。还显示了主要股东所有权百分比表。高管薪酬增长最近引起了激烈的争论。股东查看代理文件以确定薪酬水平是否可以接受。

与这场辩论相关的问题是大型被动指数基金，包括先锋、贝莱德、道富和其他持有美国企业大量所有权的公司，在投票中是否过于被动。这些重量级人物的过往记录表明，他们绝大多数时间都是根据董事会的建议投票的。当激进投资者发现某些公司治理做法令人反感时，他们发挥了重要作用。
