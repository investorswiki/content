# 美国石油学会
## 什么是美国石油学会？

美国石油协会 (API) 是代表石油和天然气行业的贸易协会。它专注于影响公共政策和[游说](/lobby)国会、政府行政部门、州政府和媒体。它与[监管机构进行谈判](/federal-agencies)并在法律诉讼中代表行业。

API 在美国拥有 600 多个生产、加工和分销石油和天然气的成员。它成立于 1919 年，是标准制定组织。它的其他活动包括制定环境保护标准、健康和安全法规以及培训和认证计划。该研究所还收集和分发有关石油和天然气生产各个方面的统计数据，这些数据发表在其**每周统计公报**中。

##美国石油学会活动

API 开展的研究范围从经济分析到毒理学测试。它收集和发布有关美国工业运营的数据，包括各种[产品](/export)的供需、进出口[、](/import)钻井活动和成本以及完井情况。它的出版物**每周统计公报**被视为[能源信息管理局](/energy-information-admin)（EIA）发布的类似数据的前奏。

API 引领了石油、天然气和石化设备和操作标准的发展。它制定了 700 多项标准，并致力于在全球范围内推广这些标准。

API 还提供质量、环境和职业健康与安全管理体系认证。它组织关于公共政策问题的研讨会、讲习班和会议。通过 API 培训，该研究所为石油和天然气业务提供有关监管要求和行业标准的培训和材料。

## 美国石油学会历史

该研究所的起源可以追溯到第一次世界大战，当时成立了国家石油战争服务委员会，以确保向武装部队供应石油供应线的稳定流动。最大的石油生产和炼油公司的负责人在该委员会任职，该委员会是在美国商会下成立的，后来成为一个准政府机构。战后，美国石油协会成立，以继续促进与政府的行业合作，促进国内和国际贸易，促进其利益。 1969 年，API 将其办公室从纽约市迁至华盛顿特区

1920 年，API 开始每周发布[原油](/crude-oil)[产量](/crude-oil)统计数据。 1924年，该研究所发布了第一个油田设备标准。在第一次世界大战期间，生产商汇集了钻井设备以克服供应短缺，但经常发现由于缺乏统一性，设备无法协同工作。
