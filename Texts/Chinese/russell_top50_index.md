# 罗素 50 强指数
## 什么是罗素 50 强超级股指数？

罗素 50 大超级股指数是一个市值加权指数，涵盖广泛的[罗素 3000指数美国股票中的 50 只最大](/russell_3000)股票。因此，Top 50 指数是一个超大盘指数，可以与包含 30只[蓝筹股的道琼斯工业平均指数 (](/bluechip) [DJIA ) 相提并论](/djia)。

## 了解罗素 50 大超级股指数

罗素 50 强超级股指数仅包含 50 只股票，但由于现代[超级股](/megacap)公司规模如此之大，它仍然占美国所有股票的很大比例。罗素美国指数是美国领先的机构投资者股票基准。这种广泛的美国指数使投资者能够按特定规模、投资风格和其他市场特征跟踪当前和历史市场表现。

所有罗素美国指数都是[罗素 3000](/russell_3000)指数的子集，包括著名的大盘[罗素 1000](/russell_1000index)指数和小盘[罗素 2000](/russell2000)指数。罗素美国指数被设计为广泛的金融产品的组成部分，例如指数跟踪基金、衍生品和交易所交易基金 (ETF)，并发挥其作为绩效[基准的作用](/benchmark)。该指数可以被认为是大盘股的代表，因为会员的平均市值超过 1750 亿美元。该指数每年都会重新编制，以考虑新的和成长中的成员公司。

罗素 50 强超级股指数通常支付高于平均水平的[股息收益率](/dividendyield)，这反映了最大交易公司的总体安全性和现金流生成。
