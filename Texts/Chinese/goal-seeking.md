# 目标寻求
## 目标是什么？

目标寻找是在仅知道输出的情况下找到正确输入值的过程。目标搜索功能可以内置到不同类型的计算机软件程序中，如 Microsoft Excel。

## 了解目标寻求

目标寻找是一个通用术语，用于描述根据已知输出值计算输入值所涉及的过程。该过程涉及在公式中使用特定的运算符，可以使用计算机软件进行计算。

目标搜索是计算机软件程序“假设分析”中使用的工具之一。[假设](/sensitivityanalysis)分析是更改 (Microsoft [Excel](/sensitivityanalysis) ) 单元格中的值的过程，以查看这些更改将如何影响工作表上的公式结果。当您寻求目标时，您正在对给定值或输出执行假设分析。因此，从本质上讲，您将通过询问“如果输出是 X 会怎样”来创建一个场景——或者基本上是一个因果情况。

对于一些比较复杂的问题，人们会经常使用计算机软件。像 Microsoft Excel 这样的电子表格程序有一个内置的目标搜索工具。当输出值已知时，它允许用户确定公式的所需输入值。如果用户只知道自己每月可以支付多少（输出），则此功能可以帮助用户确定借款人需要符合条件的[利率（输入）。](/interestrate)

但是使用目标搜索软件有一个警告：它只有在有一个输入值时才有效。如果您需要计算出两个或更多输入值，那么它将不起作用。如果我们以上面的例子为例，如果你想计算贷款总额和每月还款额，目标搜索软件是行不通的。您可能需要一个附加组件才能找出多个变量。

### 如何在 Microsoft Excel 中寻找目标

如上所述，目标搜索软件只有在您已经知道输出值（或结果）但想要确定一个输入值的情况下才能工作。如果您想在程序中使用目标搜索功能，可以遵循以下一些关键步骤：

- 打开一个新的电子表格

- 标记您的列。这将使您更容易阅读工作表中的所有内容。因此，使用上面的示例，第一列将是“贷款金额”，第二列将是“以月为单位的[期限](/term)”，第三列将是“利率”，最后一列将是“付款”。

- 输入您知道的值。

- 输入公式来计算目标——在本例中为付款。您现在可以忽略利率，公式将假定利率为 0%。

现在您必须确定利率。您可以使用 Excel 中的目标搜索功能并输入已有的值来执行此操作。

## 寻求目标的例子

追求目标的[企业家](/entrepreneur)是使用目标寻求来确定他们如何实现最终目标的人。例如，一位企业家可能会问他们每小时需要赚多少钱才能在一年内赚到 100,000 美元。他们知道他们想要的输出值——100,000 美元——因此，他们必须重新计算以找出最佳输入。这意味着他们必须确定他们在一年中能够（或愿意）工作多少小时，因此，为了实现目标，他们每小时能赚多少钱。

＃＃ 强调

- 但是，目标搜索软件仅在只有一个输入值时才有效。

- 目标寻找是在只知道输出的情况下找到正确的输入。

- 这通常使用 Microsoft Excel 中的目标搜索功能或通过计算机程序进行假设分析来完成。
