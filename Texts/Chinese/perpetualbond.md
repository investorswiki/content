# 永续债
## 什么是永续债券？

永续债券，也称为“consol bond”或“prep”，是一种没有[到期日的固定收益证券](/maturitydate)。这种类型的债券通常被认为是一种[股权](/equity)，而不是债务。这些类型的债券的一个主要缺点是它们不可赎回。然而，他们的主要好处是他们永远支付稳定的利息支付。

## 了解永续债

[债券市场](/bondmarket)的一小部分。这主要是因为很少有实体足够安全，可以让投资者投资于本金永远不会偿还的债券。

现存的一些值得注意的永续债券是英国财政部为第一次世界大战和 1720 年南海泡沫而发行的债券。美国的一些人认为联邦政府应该发行永续债券，这可能有助于避免再融资成本与有到期日的债券发行有关。

## 永久债券示例

由于永续债券支付类似于股票股息支付，因为它们都提供了无限期的某种回报，因此它们的定价方式相同是合乎逻辑的。

因此，永续债券的价格是固定利息支付或票面金额除以某个恒定贴现率，该贴现率代表货币随时间贬值的速度（部分原因是通货膨胀）。随着时间的推移，[贴现率](/discountrate)分母会降低名义上固定的[票面金额的实际价值](/coupon)，最终使该值为零。因此，永续债券即使永远支付利息，也可以被赋予一个有限的价值，这反过来又代表了它们的价格。

## 永续债现值公式

现值 = D / r

在哪里：

D = 债券的定期付息

r = 适用于债券的贴现率

例如，如果永续债券每年支付 10,000 美元的永续性，并且假设贴现率为 4%，则现值为：

现值 = 10,000 美元 / 0.04 = 250,000 美元

请注意，永续债券的现值对假设的贴现率高度敏感，因为付款被认为是事实。例如，使用上面的示例，贴现率分别为 3%、4%、5% 和 6%，现值为：

现值 (3%) = $10,000 / 0.03 = $333,333

现值 (4%) = $10,000 / 0.04 = $250,000

现值 (5%) = $10,000 / 0.05 = $200,000

现值 (6%) = $10,000 / 0.06 = $166,667

＃＃ 强调

- 永续债虽然不可赎回，但永远支付源源不断的利息。

- 永续债券，也称为 perps 或 consol 债券，是没有到期日的债券。

- 由于这些债券的性质，它们通常被视为一种股权而非债务。
