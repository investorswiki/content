# BEP-2
BEP-2 是币安链上代币发行和实施的技术标准。该标准定义了代币在币安链生态系统中发挥作用应遵循的一套规则。在以太坊区块链上发行代币的可比代币标准是[ERC-20](/erc-20) 。

BEP-2 不是一个软件，而是一套技术规范。这些规则确保币安链生态系统中的代币可以无缝交互。 BEP-2 代币可以代表广泛的数字资产和可交易商品。该标准确保它们都可以顺利地与彼此以及周围的生态系统集成。

BEP-2 标准还定义了币安链上发行的代币的一些基本和最基本的特征。虽然标准定义了规则，但对于创建它们的特定用例，不同的令牌仍然会有不同的实现。尽管如此，创建这样的标准显着减少了开发人员发行代币所需的工作量。

Binance Chain的原生资产是BNB，用于支付费用（类似于gas）。这意味着当您想要转移您的 BEP-2 代币时，您的钱包中还需要有少量的 BNB。

BEP-2 代币可以在 Binance DEX 上交易，这是一个运行在 Binance Chain 之上的去中心[化交易所。](/decentralized-exchange)

您可以将 BEP-2 代币存储在许多不同的钱包中。如果您想使用移动钱包，Trust Wallet 是一个很好的选择，但许多其他软件钱包也支持该标准。硬件钱包也支持 BEP-2 标准，例如 Ledger Nano X、Ledger Nano S、Trezor Model T 或 Coolwallet S。

您可以在币安链区块浏览器中浏览在币安链上发行的所有代币。

您可以使用此门户发布 BEP-2 令牌。强烈建议先阅读币安链 GitHub 上更详细的技术规范。如果您想尝试 BEP-2 代币创意，您可以在币安链测试网上进行。

您将在币安链文档中找到有关代币发行的详细技术指南以及许多相关主题。
