# 展示书
## 什么是展示书？

Display Book 是[纽约证券交易所](/nyse)(NYSE) 的专有跟踪工具，用于显示、记录和执行市场订单。交易和订单数据包括特定证券的订单类型、价格、时间和数量。纽约证券交易所附属交易所的专家为他们交易的每种证券使用该系统。纽约证券交易所于 2012 年用其通用交易平台 (UTP) 取代了 Display Book 。

## 了解展示书

纽约证券交易所是世界上最大的[股票交易所](/equity)。交易所[专家](/specialist)使用纽约证券交易所展示簿来执行订单和管理订单流。专家管理股票的库存以及买卖交易的订单和定价。如果市场变得不平衡，专家就会介入，利用他们的库存来管理库存需求的突然变化。

Display Book 帮助专家跟踪特定证券的市场活动，并将数据连续发布到合并磁带。 Display Book 本质上充当了以电子方式提交给专家的订单的匹配引擎。订单执行后，系统立即将交易确认发送回发起订单的经纪交易商。

## 通用交易平台 (UTP)

多年来，随着技术的进步和市场需求的增加，需要一个新的系统来取代 Display Book。导致通用交易平台 (UTP) 实施的一些不断变化的市场动态包括：

- 交易所的现金股票产品数量增加

地整合全球股票和[衍生品市场](/derivative)

- 计算能力和连接性的快速发展

- 客户要求更快的执行速度

UTP 是 Display Book 的涡轮增压版本，具有更低的延迟、更高的吞吐量和更强大的功能。 UTP 还结合了 Display Book 和 Super Display Book 的功能，这是 Display Book 的配套数据库系统。

NYSE 及其家族成员交易所完成了从 Display Book 到 UTP 的迁移。这些交易所包括：

- [NYSE Arca](/nyse-arca) ，这是美国的电子证券交易所，[交易所交易产品 (ETP)](/exchange-traded-products-etp)和股票交易，包括[交易所交易基金 (ETF)](/etf) 。 ETF 是包含跟踪或反映指数的一篮子证券的基金，例如[标准普尔](/sp500)[500 指数](/sp500)。

- NYSE [Euronext](/euronext) ，这是 2007 年 NYSE 和[Euronext合并的结果](/euronext)。

- ICE Futures Europe，在欧洲交易期货和期权，但在新加坡和加拿大以及整个欧洲设有办事处。

- NYSE American，这是一个专为小型、成长型公司设计的交易所，前身为[美国证券交易所](/amex)。
