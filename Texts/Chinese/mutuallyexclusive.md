# 互斥
##什么是互斥的？

互斥是一个[统计](/statistics)术语，描述不能同时发生的两个或多个事件。它通常用于描述一种结果的发生取代另一种结果的情况。例如，战争与和平不能同时共存。这使它们相互排斥。

## 理解互斥

互斥事件是不能同时发生的事件，但不应被视为独立事件。独立事件对其他选项的可行性没有影响。举个基本的例子，考虑掷骰子。你不能在一个骰子上同时掷出 5 和 3。但是，您绝对可以在两个骰子上掷出 5 和 3。同时掷出 5 和 3 意味着这个结果是相互排斥的。将 5 投到 1 和 3 投到其他意味着它们不是相互排斥的结果。

＃＃＃ 机会成本

当面临相互排斥的选项之间的选择时，公司必须考虑[机会成本](/opportunitycost)，这是公司为追求每个选项而放弃的成本。机会成本和互斥性的概念具有内在联系，因为每个互斥选项都需要牺牲通过选择替代选项可能产生的任何利润。

货币[时间价值](/timevalueofmoney)(TVM) 和其他因素使互斥分析更加复杂。为了进行更全面的比较，公司使用[净现值](/npv)(NPV) 和[内部收益率](/irr)(IRR) 公式在数学上确定在两个或多个互斥选项之间进行选择时哪个项目最有利。

## 互斥示例

互斥性的概念经常应用于[资本预算](/capitalbudgeting)。公司可能必须在完成后为公司增值的多个项目之间进行选择。其中一些项目是相互排斥的。

例如，假设一家公司的扩建项目预算为 50,000 美元。如果可用的项目 A 和 B 各花费 40,000 美元，而项目 C 仅花费 10,000 美元，则项目 A 和 B 是互斥的。如果公司追求 A，它也负担不起追求 B，反之亦然。项目 C 可以被认为是独立的。无论追求哪个其他项目，公司仍然可以负担得起追求C。接受 A 或 B 不会影响 C 的可行性，接受 C 不会影响任何其他项目的可行性。

此外，在查看机会成本时，请考虑对项目 A 和 B 的分析。假设项目 A 的潜在回报为 100,000 美元，而项目 B 的潜在回报仅为 80,000 美元。由于 A 和 B 互斥，选择 B 的机会成本等于最有利可图的选项（在这种情况下为 A）的利润减去所选选项（B）产生的利润；也就是说，100,000 美元 - 80,000 美元 = 20,000 美元。因为选项 A 是最有利可图的选项，所以选择选项 A 的机会成本为 0 美元。

## 底线

相互排斥的事情不可能同时发生。在商业中，这通常涉及项目的实施或预算分配。如果两件事不是相互排斥的，那意味着一件的存在和发生并不一定意味着另一件不能共存。

＃＃ 强调

- 当事件不能同时发生时，它们被认为是互斥的。

- 如果考虑相互排斥的选项，公司必须权衡机会成本，或者通过选择每个选项将放弃什么。

- 这个概念经常出现在商业世界的预算和交易评估中。

- 不相互排斥意味着两个实例或结果可以同时发生，并且一个结果不限制另一个结果的可能。

- 在两个相互排斥的选择之间做出决定时，通常会考虑货币时间价值 (TVM)。

＃＃ 常问问题

###金融中的互斥意味着什么？

通常，这涉及预算和付款。如果一家公司有 1.8 亿美元可花费，那么它就不能通过对业务进行再投资和向高层管理人员提供奖金来花费这 1.8 亿美元。在这种情况下，这两个选项是互斥的。如果公司只能在一个国家/地区保留许可，这意味着他们不应尝试在两个不同的国家/地区获得许可，因为它们是相互排斥的。

### 如果项目是互斥的，这意味着什么？

在商业中，经理和董事经常需要计划资源分配。如果一家公司正在建造一座桥梁和一座摩天大楼，并且这两个项目都需要非常专业的设备，并且世界上只有一个，这意味着这些项目是互斥的，因为这两个项目不能同时使用该设备同时。这个想法可以扩展到考虑专业人士、软件系统（不能同时运行 Mac 和 Windows）和分配的预算。

### 独立和互斥有什么区别？

为了说明独立和相互排斥之间的区别，请考虑前面的战争与和平示例。法国可能会发生战争，意大利可能会发生和平。这是两个独立的国家，因此每个国家都可以处于自己的和平状态。然而，法国不可能有战争，法国不可能有和平。由于它们不能共存，这使它们相互排斥。
