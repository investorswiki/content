# 点数 (P&F) 图表
## 什么是点数 (P&F) 图表？

点数图绘制股票、债券、商品或[期货的价格变动，](/futures)而不考虑时间的流逝。

与其他一些类型的图表相反，例如烛台，它标志着资产在设定时间段内的移动程度，P&F 图表使用由堆叠的 X 或 O 组成的柱，每个柱代表一定数量的价格变动。 X 代表价格上涨，而 O 代表价格下跌。

在查看 P&F 图表时，技术分析师仍会使用支撑和阻力等概念以及其他模式。一些人认为，支撑位和阻力位以及[突破](/breakout)在 P&F 图表上的定义更加清晰，因为它过滤掉了微小的价格变动并且不太容易受到[虚假突破的影响](/failedbreak)。

## 如何计算点数 (P&F) 图表

点数图不需要计算，但它们确实需要设置至少两个变量。

一个变量是盒子的大小。盒子大小可以是特定的美元金额，例如 1 美元，百分比，例如当前价格的 3%，或者它可以基于[平均真实范围](/atr)(ATR)，这意味着盒子大小将根据波动性而波动。

还需要设置反转量。反转量通常是盒子大小的三倍。例如，如果盒子大小为 1 美元，则反转金额为 3 美元。反转可以设置为交易者想要的任何值，例如盒子大小的 1 倍或盒子大小的 5.5 倍。

一个可选变量是是使用标的资产的高价和低价还是使用[收盘价](/closingprice)。使用最高价和最低价将意味着创建更多的 X 和 O，而仅使用收盘价（与最高和最低价相比计算的变动更少）将意味着创建更少的 X 和 O。

## 点数 (P&F) 图表告诉您什么？

与传统烛台或条形图相比，点数图通常为[技术分析师提供不同的交易和趋势信号。](/technicalanalysis)虽然一些分析师更依赖点数图表，但其他分析师则使用这些图表来确认传统图表提供的信号，以避免错误的突破。

点数图表的关键是盒子大小，或者价格变动量，它决定了图表中是否添加了新的 X 或 O。例如，假设盒子大小是 3 美元。如果最后一个 X 发生在 15 美元的价格，当价格上涨到 18 美元时，一个新的将添加到 X 的当前列中。

值得注意的是，如果价格继续上涨并且没有突破预定的反转量，那么 X 的线在同一列中继续存在，此时，新的 O 列开始。

[下跌](/bearmarket)的市场中，一列 O 也是如此；该列一直持续到股票达到反转量，此时新的 X 列开始。

当价格不再移动到足以在当前 X 或 O 列中放置另一个 X 或 O，然后价格在相反方向移动至少三个盒子大小（如果这是选择的反转量）时，就会发生反转。当发生反转时，会同时画出几个 X 或 O。例如，在价格上涨或 X 列之后，如果发生反转并且反转量是三个盒子大小，当反转发生时，将从最高 X 下方的一个位置开始绘制三个 O。

交易者使用 P&F 图表的方式与其他图表类似。交易者仍在关注支撑位和阻力位。突破可以预示主要的趋势变化。根据盒子的大小，列本身可以代表重要的趋势，并且当列发生变化（从 O 到 X，或 X 到 O）时，可能预示着重要的趋势反转或[回调](/pullback)。

##点和图分析师

**《华尔街日报》**的创始人查尔斯·道 (Charles Dow) 因开发点数图表作为确定供需失衡的一种方式而受到[赞誉](/law-of-supply-demand)。

Tom Dorsey 是专注于点数图表的最重要的技术分析师之一，他于 1987 年创立了研究公司 Dorsey, Wright & Associates。他撰写了几本关于该主题的书籍，包括 **Point & Figure Charting: The Essential Application用于预测和跟踪市场价格。** 纳斯达克于 2015 年收购了 Dorsey, Wright & Associates 。

Dorsey 帮助普及了点数图表与更传统的技术指标的使用，例如[移动平均线](/movingaverage)、相对强度和[上涨/下跌线](/advancedeclineline)。

## Point-and-Figure (P&F) 和 Renko 图的区别

[砖形图](/renkochart)也基于盒子大小，当价格按盒子大小移动时，它会创建一个向上或向下的砖块，该砖块与前一个砖块呈 45 度角移动。砖形图从来没有相邻的砖块。因此，如果价格向相反方向移动两个框量，就会发生反转。

图表类型之间的主要区别在于外观。 P&F 图表是 X 和 O 的并排列，而 renko 图表是由一系列以 45 度角随时间分布的框创建的。

## 使用点数 (P&F) 图表的限制

P&F 图表对价格变化的反应可能很慢。例如，突破必须移动箱量才能发出突破发生的信号。这可能会使一些交易者受益，因为它可能会减少错误的突破信号，但价格已经将盒子数量（或更多）移动到突破点之外。对于一些交易者来说，在价格已经波动这么大之后获得信号可能无效。

此外，虽然 P&F 图表可能有助于减少错误突破的数量，但仍然会发生错误突破。看似突破的情况可能仍会在短时间内逆转。

P&F 图表擅长让交易者保持强劲的趋势，因为许多小的反趋势运动被过滤掉了。然而，当发生逆转时，它可能会严重抹去利润或导致巨额亏损。因为反转量通常非常大，如果交易者只使用 P&F 图表，他们将不会看到反转，直到价格显着不利于他们。

使用 P&F 图表时，建议同时关注资产的实际价格，以便实时监控风险。这可以通过监控烛台或[开高低收](/ohlcchart)(OHLC) 图表来完成。

＃＃ 强调

- X 和 O 分别堆叠在一起，通常会形成一系列 X 或 O。

- 当价格上涨一定数量（称为盒子大小）时，会创建一个 X。当价格降低盒子大小量时，会创建一个 O。

- 盒子大小是根据资产的价格和投资者的偏好设置的。

- 当价格走势与其当前趋势相反时，会形成一个新的 X 或 O 柱，并且超出反转量。
