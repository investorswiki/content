# 远期保费
## 什么是远期溢价？

远期溢价是一种货币的远期或预期未来价格高于[现货价格的情况](/spotprice)。市场表明，当前的国内汇率兑另一种货币将会上涨。

这种情况可能会令人困惑，因为汇率上升意味着[货币贬值](/currency-depreciation)。

## 了解远期保费

远期溢价经常被衡量为当前[即期汇率](/spot_rate)和[远期汇率之间的差额](/forwardrate)，因此可以合理地假设未来即期汇率将等于当前期货汇率。根据远期预期的汇率理论，当前的即期期货汇率就是未来的即期汇率。该理论植根于[实证研究](/empiricalprobability)，是长期时间范围内的合理假设。

> 通常，远期溢价反映了所涉两国两种货币之间的利率差异可能引起的变化。

>

远期货币[汇率](/exchangerate)通常不同于货币的即期汇率。如果一种货币的远期汇率高于即期汇率，则该货币存在溢价。当远期汇率低于即期汇率时，就会出现折扣。

## 远期利率溢价计算

计算远期汇率的基础需要货币对的当前现货价格和两国的利率（见下文）。考虑一下日元和美元之间的兑换示例：

- 90 天日元兑美元 (¥/$) 远期汇率为 109.50。

- 即期汇率 ¥ / $ 汇率 = 109.38。

- **年化远期保费计算** = ((109.50-109.38÷109.38) x (360 ÷ 90) x 100% = 0.44%

在这种情况下，美元相对于日元是“强”的，因为美元的远期价值比即期价值高出每美元 0.12 日元的溢价。日元将以折价交易，因为其对美元的远期价值低于其即期汇率。

要计算日元的远期折扣，您首先需要根据美元兑日元的关系计算日元的远期汇率和即期汇率。

- ¥/$ 远期汇率为 (1÷109.50 = 0.0091324)。

- ¥/$ 即期汇率为 (1÷109.38 = 0.0091424)。

- **日元的年化远期折扣，以美元计** = ((0.0091324 - 0.0091424) ÷ 0.0091424) × (360 ÷ 90) × 100% = -0.44%

对于一年以外的期间的计算，您将输入天数，如下例所示。三个月远期利率等于即期利率乘以（1 + 国内利率乘以 90/360 / 1 + 国外利率乘以 90/360）。

要计算远期利率，请将即期利率乘以利率比率并调整到期时间。因此，远期利率等于即期利率 x (1 + 国内利率) / (1 + 国外利率)。

例如，假设当前美元兑欧元汇率为 1.1365 美元。国内利率，或美国利率为5%，国外利率为4.75%。将这些值代入方程得到：F = $1.1365 x (1.05 / 1.0475) = $1.1392。在这种情况下，它反映了远期溢价。

＃＃ 强调

- 远期溢价经常被衡量为当前即期汇率和远期汇率之间的差额。

- 远期溢价是一种货币的远期或预期未来价格高于现货价格的情况。

- 当远期溢价为负时，是否相当于折价。
