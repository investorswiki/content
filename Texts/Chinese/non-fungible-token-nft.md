# 不可替代代币 (NFT)


不可替代代币 (NFT) 是一种代表独特资产的加密代币。 NFT 是数字或现实世界资产的代币化版本。它们充当区块链网络中真实性和所有权的可验证证明。 NFT 不能相互互换，并且给数字世界带来了稀缺性。

[可替代性](/fungibility)是指资产的属性，其单个单元可互换且彼此基本无法区分。例如，所有[法定货币](/fiatmoney)都是可替代的。为了充当交换媒介，每个单独的单元必须可以与任何其他等效的单独单元互换。一美元钞票可以与任何其他真正的一美元钞票互换。

[化应用程序 (DApp)](/decentralized-applications-dapps)可以使用 NFT 来创建和拥有独特的数字物品和收藏品。虽然 NFT 可以在连接买家和卖家的开放市场上交易，但值得注意的是，每个市场的价值都是独一无二的。

已经创建了各种框架来促进 NFT 的发行。其中最突出的是[ERC-721](/erc-721) ，它是在以太坊区块链上发行和交易不可替代资产的标准。一个更新的改进标准是 ERC-1155，它使单个合约能够包含可替代和不可替代的代币。

NFT 的标准化允许更高程度的[互操作性](/interoperability)，这意味着可以相对轻松地在应用程序之间转移独特的资产。

NFT 有可能成为新的区块链驱动的数字经济的关键组成部分之一。它们可以用于许多不同的领域，例如视频游戏、数字身份、许可、证书或美术——甚至允许对物品进行部分所有权。将所有权和身份数据存储在区块链上将提高数据完整性和隐私性，而对这些资产进行简单、[无需信任](/trustless)的转移和管理可以减少贸易和全球经济中的摩擦。
