# 源代码
源代码是指计算机代码行的集合，它们负责根据确定性指令和语句的列表定义程序（软件）将如何运行。从本质上讲，源代码是软件在最初编写时的主要状态。

因此，源代码由一组指令组成，这些指令可以用各种不同的计算机编程语言编写，例如 Java、JavaScript、C、Python、PHP、Go、Rusty、Solidity 和 Haskell。所有这些编程语言都有一个共同的特点：它们由一组指令组成，计算机要遵循这些指令来执行某些操作并产生特定的输出。

创建源代码后，通常会被“翻译”或编译成计算机能够理解和执行的机器语言。然而，在某些情况下，源代码无需编译即可立即执行（这些计算机程序称为解释器）。

在计算机软件的开发过程中，源代码阶段是程序员在程序最终确定并最终由计算机机器执行之前能够阅读、评论和编辑的初始工作阶段。源代码库使软件工程师、计算机科学家和程序员的工作变得更加容易，因为它们能够在代码行中包含注释（例如，带有解释性和洞察力的注释的可读纯文本片段）。
