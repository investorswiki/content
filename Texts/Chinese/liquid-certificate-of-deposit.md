# 流动存单 (CD)
## 什么是流动性存款证明 (CD)？

流动[性存款证明](/certificateofdeposit)是一种 CD，可让您在不产生罚款的情况下进行提款。账户中的资金在产品的整个生命周期内都可以使用——与大多数传统 CD 不同，后者会因[提早提款](/earlywithdrawal)而收取高额罚款，从而降低了当时所赚取的[利息。](/interest)然而，投资者也不能有他们的蛋糕并吃掉它。液体 CD 通常提供比传统 CD 更低的费率，这意味着它们会牺牲[产量](/yield)以换取灵活性。

## 了解液体 CD

流动性 CD 允许您在[到期日之前提款，](/maturitydate)但必须提前通知机构。这不像从支票账户中提款那么简单。

一些银行或信用合作社需要一周的通知，而另一些则需要长达 30 天。在很多情况下，还有一个初始锁定期：您不能在基金的第一周内进行提款。这可以防止股票市场中经常出现的那种当日交易行为。

此外，液体 CD 通常对在给定时间点可以提取的数量有限制。一些银行允许在一次交易中提取 100% 的资金，而其他银行则有更严格的要求，提取金额要么是初始存款的最低百分比，要么只是最低美元金额。

通过流动性 CD 销售的免罚金提款也可能成为吸引投资者的诡计。例如，流动 CD 通常会限制免罚金提款的次数。与其他[流动性](/liquidasset)相对较低的工具相比，它们还具有更高的最低投资额，但不能提供相同的收益。

违反这些规则可能会导致与传统 CD 相当的提款处罚。如果您对流动 CD 感兴趣，请记住阅读产品的所有条款和条件，其中明确说明了不同的提款要求和最小账户规模。

> 从流动 CD 中提款可以受到合同限制，这些提款的金额也是如此。

>

## 液体 CD 的替代品

一些投资者喜欢流动性 CD 的灵活性。它提供了在紧急情况下快速、轻松地获得计息资金的途径。这样一来，您就不一定需要在储蓄账户中存放大量闲置现金。

不过，液体 CD 并不适合所有人。还有许多其他解决方案可以满足各种财务目标。一种选择是购买回报更高但灵活性更低的传统 CD。如果您有可靠的[应急基金](/emergency_fund)并且不需要快速获得现金，那么您将从传统 CD 中受益更多。

[阶梯](/laddering)式投资是一种流行的 CD 投资方法，可以[保证定期获得稳定的收入](/cd-ladder)。另一种方法是开设货币市场账户，它比 CD 提供更大的灵活性。

## 液体 CD 示例

一家银行在一年（2% 的回报率）到五年（5% 的回报率）的时间段内提供[利率](/interestrate)从 2% 到 5% 的流动 CD。这些 CD 的最低投资额因利率而异。

例如，利率为 2% 的流动性 CD 需要 10,000 美元的最低投资额，而提供 5% 回报率的流动性 CD 则需要 25,000 美元的最低投资额。提供 2% 的 CD 的免罚金提款仅限于一次提款，而回报率为 5% 的 CD 则在整个期限内仅限于 3 次提款。

＃＃ 强调

- 市场上流动性 CD 的替代品包括货币市场账户和银行储蓄账户。

- 流动 CD 的利率低于传统 CD。

- 流动性存款证明是一种 CD，允许投资者进行提款而不会受到处罚。

＃＃ 常问问题

### 我可以从液体 CD 中取出多少内容是否有限制？

您可以从流动 CD 中提取多少以及您可以多快获取资金通常是有限制的。尽管规则因银行而异，但许多流动性 CD 账户要求投资者发出通知并将其提款限制在一定金额内。虽然可以使用您的资金，但它并不像清算普通银行账户那么容易。

### 流动性存款证明（流动性CD）值得吗？

流动性存款证明（流动性 CD）提供某些好处，但并不是每个人的理想选择。虽然流动 CD 提供安全和灵活的提款，但它们通常承诺较低[的回报率](/rateofreturn)。此外，液体 CD 通常需要更高的最低存款额。

### 液体CD 是我储蓄的好地方吗？

流动性 CD 可以充当有限的支票账户，但它不是存放您的储蓄的最佳场所。应急基金的最佳位置是完全可访问的账户，例如支票或高收益储蓄账户。如果您将积蓄保存在流动的 CD 中，您可能无法立即完全使用您的资金。
