# Bump-Up 存款证明（Bump-Up CD）
当利率上升时，提高存款证明可以帮助储户利用收益率上升的优势。

在利率不断提高的环境中，加息 CD 让消费者可以选择提高 CD 的年收益率，而无需更改其条款。通常，加息 CD 只允许在期限内提高一次利率，这表明它们可能更适合了解当前利率环境以及利率可能会上升或下降的消费者。一些加价 CD 的条款允许客户多次提高利率。

自 2020 年 3 月以来，美联储一直将利率维持在接近零的水平，当时 COVID-19 大流行对美国经济造成了严重打击，几乎没有迹象表明美联储将很快采取行动加息。尽管如此，利率还是会随着时间的推移而波动，储户可以通过增加 CD 等策略为自己做好准备。

## Bump-up CD 的工作原理

CD 是具有指定到期日和规定利率的储蓄账户。通常，利率在 CD 的整个生命周期内保持不变，但有一些选项允许更改利率。 Bump-up CD 允许持有人在 CD 继续成熟的同时提高他们的回报率。

升级 CD 的期限各不相同，但大多数是两三年。由于 CD 赋予持有人提高利率的能力，因此大多数 Bump-up CD 的利率略低于传统 CD。不同金融机构所需的最低 CD 金额也有所不同，但有些最低可低至 500 美元。

许多加价 CD 允许单次加息，但有些（尤其是长期的）允许多次加价。可能有关于您一次可以提高多少利率的规则，并且始终最好与您的金融机构核实其条件是什么。

另请记住，如果您在到期日之前从 CD 中取出资金，您可能会支付罚金，除非它是无罚金 CD。

## Bump-up CD 示例

假设您想留出 10,000 美元用于家居装修。您预计您将在几年内不需要资金，但希望看到资金在此期间安全增长并利用利率的任何上涨。在这种情况下，如果您预计在期限结束之前不需要资金，则为期两年的临时 CD 可能是理想的选择。

现在假设您投资了一张提供初始 0.5% APY 的增值 CD。然后，一年后，利率提高，将增加 CD 的 APY 提高到 0.8%。因此，您可以使用一次加价来利用新的、更高的费率。

如果事实证明您在两年期间需要资金，请务必了解提前取款罚款的费用，以确定增补 CD 是否适合您。

## Bump-up CD 的优点

加价 CD 的最大优势是能够在到期期间提高利率。如果利率在期限内增加，您将能够利用它们，帮助您更快地赚钱。其他类型的 CD 将您锁定在您的 APY 中，要求您在使用更高的利率之前完成该期限，或者对在 CD 到期之前关闭 CD 施加提前取款罚款。使用 Bump-up CD，您可以轻松地提高利率。

## 增强型 CD 的缺点

Bump-up CD 的吸引力在于它的灵活性，但也有一些缺点。与其他类型的 CD 相比，Bump-up CD 通常以较低的 APY 开始，从而使银行能够对冲未来 APY 的增长。根据市场情况，标准 CD 在整个期限内可能比加价 CD 更能让您感兴趣。

也不能保证在 CD 任期内利率会上升。或者，如果利率继续攀升并继续攀升，您可能会过早行动，错过更大的 APY 增长。

## 如何打开增强型 CD

开设临时 CD 与开设其他类型的银行账户类似，但提供此类账户的银行较少。请务必研究银行产品的详细信息，包括到期期限、涨价频率、初始 APY、初始存款和提前取款费用。您可能会在网上银行找到更优惠的价格，但您也可以通过货比三家从传统金融机构找到一些优惠。找到适合您需求的银行和账户后，您需要填写申请表，选择账户并阅读账户材料。

## 升级 CD 的替代品

您可能需要考虑几种替代 CD 的方法，具体取决于您从储蓄中寻找什么：

- 传统CD：具有特定期限和利率的固定账户。

- 升压CD：就像一个以设定的时间间隔自动提高你的APY的升压CD。

- CD 阶梯：通过使多张 CD 成熟来最大限度地节省成本的策略。

＃＃ 常问问题

### 什么是升压存款证明 (CD)？

随着市场利率的上升，升压存款证 (CD)，也称为升息 CD，为投资者提供了一次提高利率的特殊选择。这是建立比常规 CD 期权更高[回报率的](/rateofreturn)绝佳机会。

### 升级 CD 值得吗？

如果市场利率在您的投资期限内上涨，那么购买 CD 是值得的。如果利率没有增加，那么普通 CD 是更好的选择，因为它们的起始利率高于加价 CD。这完全取决于市场和投资者对其投资的态度。

### 我可以多次提高 CD 的利率吗？

定期 CD 不允许您提高利率。 Bump-up CD 通常会提供一次机会来提高您的费率。多次加息通常只允许用于期限较长的临时 CD。
