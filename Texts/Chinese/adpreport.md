# ADP全国就业报告
**正式名称**：就业情况

**究竟是什么**？衡量创造的净新就业机会。还衡量失业率、平均小时收入和平均工作周的长度。

**来源**：劳工处

**频率**：每月

**什么时候发布**？每个月的第一个星期五东部时间上午 8:30。上个月的数据。

**市场重要性**：高。几乎总是在推动市场。非常及时。包含有关工作和工资增长的信息，被认为是衡量经济健康状况的唯一最佳指标。就业报告的基调通常为整个月发布的其他经济指标定下基调。

**其他说明**： (a) 本报告的两个关键部分——失业率和平均时薪——出现在许多通货膨胀模型中。该报告的各个部分用于帮助预测许多其他经济指标。平均小时收入用于帮助预测个人收入以及就业成本指数的工资和薪金部分。总制造时间指数用于帮助预测工业生产。建筑工作岗位的变化用于帮助预测房屋开工和建筑支出。 (b) 对于平均小时收入，标题数字是与上个月相比的百分比变化，但我们也绘制了同比变化图。这样您就可以看到收入增加或减少的速度。

＃＃ 强调

- ADP 就业报告于 2006 年推出。

- ADP 全国就业报告是跟踪美国非农私营就业的经济数据的月度报告

- 发布报告的公司 Automatic Data Processing Inc. 负责处理美国约五分之一的私人雇员的工资单

- ADP 全国就业报告分为四个单独的版本。

- ADP 全国就业报告被视为对更详细的劳工统计局就业情况报告的有用预览。

＃＃ 常问问题

### 为什么 ADP 就业报告的数字高于 BLS 报告？

从 2017 年开始，经济学家开始观察到 ADP 和 BLS 的全国就业数据之间存在显着差异，ADP 的估计值一直高于政府数据。这可能是由于两个机构在方法和抽样方面的细微差别。 ADP 的数据基于其客户公司的工资单，这意味着必须调整其数据以反映国家经济数据。

### ADP 全国就业报告何时发布？

ADP 就业报告通常在劳工统计局发布每月就业报告的前两天发布，每月的第一个星期五发布。由于 ADP 报告首先出现，因此它通常被用作政府机构更全面统计数据的预览。

### ADP 就业报告与劳工统计局报告有何不同？

ADP 就业报告与 BLS 官方报告之间的主要区别在于，ADP 仅涵盖非农业、私人雇员。作为一个政府机构，劳工统计局的调查还包括政府雇员。此外，虽然 ADP 仅发布一组数字，但 BLS 报告已更新，以包括在报告发布后发送调查回复的公司。两份报告都有非常相似的方法，并且往往会产生非常相似的结果。
