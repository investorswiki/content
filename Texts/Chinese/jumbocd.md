# 巨额存款证明 (CD)
## 什么是巨型 CD？

巨型 CD 是一种储蓄凭证，具有指定的利率、固定的到期日和 100,000 美元的最低投资要求。

## 更深层次的定义

[存款证明是](/certificateofdeposit)一种储蓄账户，需要在一定时期内投资一定数量的资金。 CD 提供比普通储蓄账户更高的利率。大多数 CD 的期限从几个月到几年不等，期限越长，利率越高。 CD储户会因在到期日之前提取资金而受到处罚。

巨型 CD 需要比普通 CD 更大的投资。当 CD 到期时，客户可以无罚金提取资金并收取 CD 赚取的利息。许多人喜欢CD，因为它们是低风险投资。

有两种类型的巨型 CD：可转让和不可转让。

- **可转让 CD** 在二级市场交易，通常以不记名形式进行。这意味着所有权的唯一证明是文件的实际拥有。销售不记名凭证的机构不具备所有权记录。

- **不可转让的 CDs** 由开证行以存款形式持有。它们带有购买者的名字。

[联邦存款保险公司 (FDIC)投保](/fdic)的银行购买的所有 CD 的保险金额最高为 250,000 美元。[国家信用合作社管理局 (NCUA)](/ncua)为信用合作社的 CD 提供保险。

## 巨型 CD 示例

Jake 决定将 100,000 美元投入当地银行的巨型 CD 中，因为他对 CD 投资的安全性感到满意。该银行提供 2.5% 的五年期债券。当巨型 CD 成熟时，Jake 的 100,000 美元投资价值 113,140.82 美元。

＃＃ 强调

- 巨型 CD 向投资者支付固定利率，通过部分抵消市场风险帮助稳定投资组合的回报。

- 巨型 CD 通常有 100,000 美元的最低余额要求。

- 虽然巨型 CD 的最低余额要求高于传统 CD，但作为回报，它们支付更高的利率。
