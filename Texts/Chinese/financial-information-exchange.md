# 财务信息交换 (FIX)
## 什么是财务信息交换 (FIX)？

[证券](/security)交易信息的国际实时交换。该协议由 FIX 社区使用，该社区包括包括所有主要[投资银行在内的近 300 家成员公司](/investmentbank)。

FIX 已成为交易前、交易和交易后通信以及美国监管报告的事实上的消息传递标准。它与几乎所有常用的网络技术兼容。 FIX Protocol, Ltd. 拥有并维护 FIX 系统。该公司的成立完全是为了实现这一目的并确保系统保持在公共领域。

## 了解财务信息交换 (FIX)

FIX 的通信包括短信和电子邮件、证券交易分配、新闻、订单提交和更改、交易广告和[执行](/execution)报告。它主要用于[企业对企业](/btob)(B2B) 交互，旨在改善业务消息和交易流。

FIX 通过最大限度地减少冗余并减少在电话通信、书面消息、交易和文档上花费的时间来实现这一目标。对基金、[投资经理](/investment-manager)和投资银行公司来说，好处尤其明显。 FIX 系统通过和跨证券交易所传输有关证券交易的准确和及时的财务信息。

于 1992 年推出，用于[所罗门兄弟](/salomon-brothers)和富达投资之间的股权交易，旨在提供更高效和负责任的交易和记录保存，取代主要通过电话处理的系统。在旧系统下，感兴趣的迹象通常会“搁置”丢失或发送给错误的交易者。

此后，FIX 已成为股票市场交易前、交易和交易后通信的标准电子协议，并且也越来越多地用于其他市场。

>[环球银行金融电信协会](/swift)(SWIFT) 是[后台](/backoffice)消息传递的标准，而 FIX 是[前台](/frontoffice)消息传递的标准。

>

## 财务信息交换 (FIX) 用户

[买方](/buyside)（机构）和[卖方](/sellside)（经纪人/交易商）中都很受欢迎。用户包括共同基金、投资银行、经纪人、股票和期货交易所以及其他[电子通信网络](/ecn)(ECN)。它主要用于股票交易，虽然它可以处理债券、外汇和衍生品交易。

FIX 交易社区成员公司维护并继续开发 FIX 消息传递标准。社区成员包括全球多家领先的[金融机构](/financialinstitution)。这些成员公司所做的工作确保标准继续发展以满足新的和新兴的交易要求。

他们的行动还促进了 FIX 在全球范围内的采用。 FIX 协议本身是一种非专有、免费和开放的标准，由其成员公司不断开发。

## FIX 的下一步是什么？

FIX 是一个不断变化的实体，力求与行业和技术的变化保持同步。近年来，成员们一直在讨论当前的问题和挑战，包括[网络安全](/cybersecurity)、[数字货币](/cryptocurrency)和[区块链](/blockchain)、执行透明度和性能改进。

任何考虑使用 FIX 的公司都可能希望从 FIX 交易社区网站下载 FIX 实施指南。

- 金融信息交换 (FIX) 是一种信息和数据协议，用于在投资银行和经纪交易商之间传播价格和交易信息。
