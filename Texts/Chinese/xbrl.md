# 可扩展业务报告语言 (XBRL)
## 什么是可扩展业务报告语言？

XBRL 或可扩展业务报告语言是一种软件标准，旨在改进财务数据的通信方式，从而更容易编译和共享这些数据。值得注意的是，可扩展业务报告语言是 XML（可扩展标记语言）的一种实现，它是一种用于在线组织和定义数据的规范。

XBRL 使用标签来识别每条财务数据，然后允许 XBRL 兼容程序以编程方式使用这些数据。 XBRL 允许在企业之间轻松传输数据。

## 了解可扩展业务报告语言

想象一下，您正在公司网站上在线查看公司的[财务报表](/financial-statements)[。](/financial-statements)传统上，这些陈述只是纯文本。如果您想将这些数字放入电子表格文件中以对报表进行分析，则必须手动键入或将每个帐户和相应的数字复制并粘贴到电子表格中。

但是，如果网站上的数据以可扩展业务报告语言 (XBRL) 的形式提供，您可以简单地将网站上的数据转换为与 XBRL 兼容的电子表格程序（通常是即时的）。

由于识别标签和语言本身的标准化性质，来自一个制定了美国[公认会计准则等会计标准的国家的财务数据](/gaap)可以很容易地编译成另一个国家公认的会计标准，即使它们有很大的不同。并非所有公司都要求使用 XBRL 报告财务数据，但由于它已变得普遍，因此有人建议不久所有公司都必须以这种语言报告其财务数据。 iXBRL，其中 **i** 代表内联，是一种允许将 XBRL 元数据嵌入 HTML 文档的更新。

XBRL 是由美国注册会计师协会 ( [AICPA](/american-institute-of-certified-public-accountants) ) 于 1998 年开发的 1.0 版。最新版本的标准 v2.1 于 2003 年正式确定。虽然 v2.1 标准自那时以来一直保持稳定，但已经开发了几个 XBRL 模块，可以插入以实现新的功能或可操作性。

＃＃ 强调

- XBRL 基于 XML 编码，是一种在世界范围内传输财务记录的标准化方式。

- 可扩展业务报告语言 (XBRL) 是一个免费提供的全球会计标准框架，用于交换业务信息。

- XBRL 的最新版本 v2.1 于 2003 年发布并保持稳定。有几个可选模块作为附加组件存在。
