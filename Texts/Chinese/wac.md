# 加权平均息票 (WAC)
## 什么是加权平均息票 (WAC)？

加权平均息票 (WAC) 是衡量作为抵押贷款支持证券 (MBS) 出售给投资者的抵押贷款池的回报率。基础抵押贷款的偿还期限不同，因此 WAC 代表其在发行时的回报，并且可能与以后的 WAC 有所不同。

## 了解加权平均优惠券 (WAC)

银行通常会在二级抵押贷款市场上出售他们发行的抵押贷款。买家是机构投资者，例如对冲基金和投资银行。这些买家将抵押贷款打包成有价证券，这些证券可以作为[抵押贷款支持证券](/mbs)（MBS）在公开市场上交易给投资者。

> 在加权平均计算中，每笔抵押贷款的本金余额被用作其权重因子。

>

MBS 持有人收到的利息或[息票支付](/coupon)是按支持 MBS 的抵押贷款的相关票息的加权平均值计算的。

### 计算 WAC

加权平均息票 (WAC) 的计算方法是取 MBS 的基础抵押贷款所欠利率的总额，并根据每个抵押贷款所代表的证券百分比对其进行加权。

WAC代表不同利率的不同抵押贷款池的平均利率。在加权平均计算中，每笔标的抵押贷款的本金余额被用作加权因子。

为了计算 WAC，每个抵押贷款或 MBS 的票面利率乘以其剩余的本金余额。结果相加，总和除以余额。

计算加权平均票面利率的另一种方法是获取每个抵押池的权重，乘以它们各自的票面利率，然后将结果相加得到 WAC。

例如，假设 MBS 由三个不同的抵押贷款池组成，本金余额为 1100 万美元。第一个抵押贷款捆绑包或部分包括价值 400 万美元的抵押贷款，收益率为 7.5%。第二个池有 500 万美元的抵押贷款余额，利率为 5%。第三个池有价值 200 万美元的抵押贷款，利率为 3.8%。

使用上述第一种方法：

WAC = [(400 万美元 x 0.075) + (500 万美元 x 0.05) + (200 万美元 x 0.038)] / 1100 万美元

WAC = (300,000 美元 + 250,000 美元 + 76,000 美元) / 1100 万美元

WAC = 626,000 美元/1100 万美元 = 5.69%

或者，WAC 可以通过首先评估每个抵押贷款部分的权重来计算：

池 1 权重：400 万美元/1100 万美元 = 36.36%

池 2 权重：500 万美元/1100 万美元 = 45.45%

第三组权重：200 万美元/1100 万美元 = 18.18%

权重之和为 100%。因此，WAC 计算如下：

WAC = (36.36 x 0.075) + (45.45 x 0.05) + (18.18 x 0.038)

WAC = 2.727 + 2.2725 + 0.6908 = 5.69%

由于不同的抵押贷款持有人以不同的利率和不同的时间表偿还抵押贷款，加权平均票面利率可能会在 MBS 的整个生命周期内发生变化。

### 当 MSB 有风险时

如果没有提到 2007-2008 年的金融危机，那么就没有提到抵押贷款支持证券是不完整的，这在很大程度上归咎于他们。

那个时期的许多 MBS 投资都得到了全国房地产泡沫期间发行的抵押贷款的支持，并且在许多情况下，发行给了无力偿还贷款的借款人。当泡沫破裂时，许多借款人被迫违约，这些资产证券化的价值消失了。

事实上，它们以次级贷款为抵押。

＃＃ 强调

- WAC 将随着证券基础抵押贷款的偿还而改变。

- 这些投资的分析师使用抵押贷款支持证券的 WAC 来估计其预付特征。

- WAC 是抵押贷款支持证券发行时标的抵押贷款的平均总利率。
