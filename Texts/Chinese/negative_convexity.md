# 负凸性
## 什么是负凸性？

[收益率曲线的形状](/yieldcurve)是凹形时，就存在负凸性。债券的凸性是其久[期的变化率，](/rateofchange)它以债券价格相对于收益率的二阶[导数来衡量。](/derivative)大多数抵押债券是负凸的，可赎回债券通常在收益率较低时表现出负凸。

## 理解负凸性

债券久[期](/duration)是指债券价格受利率上升和下降影响的程度。凸性表明债券的久期如何随着利率的变化而变化。通常，当利率下降时，债券的价格会上涨。然而，对于具有负凸性的债券，价格会随着利率下降而下降。

例如，对于可赎回债券，随着利率下降，发行人按[面值赎回债券的动机会](/at-par)增加；因此，它的价格不会像不可赎回债券的价格那样快速上涨。随着债券被赎回的可能性增加，[可赎回债券](/callablebond)的价格实际上可能会下降。这就是为什么可赎回债券的价格相对于收益率曲线的形状是凹的或负凸的。

## 凸度计算示例

由于久期是一个不完美的价格变化估计量，投资者、分析师和交易员计算债券的凸度。凸性是一种有用的风险管理工具，用于衡量和管理投资组合的市场风险敞口。这有助于提高价格走势预测的准确性。

虽然凸度的精确公式相当复杂，但可以使用以下简化公式找到凸度的近似值：

凸近似 = (P(+) + P(-) - 2 x P(0)) / (2 x P(0) x dy ^2)

在哪里：

P(+) = 利率下降时的债券价格

P(-) = 加息时的债券价格

P(0) = 债券价格

dy = 十进制形式的利率变化

例如，假设债券目前定价为 1,000 美元。如果利率降低 1%，债券的新价格为 1,035 美元。如果利率提高 1%，债券的新价格为 970 美元。近似凸度为：

凸性近似 = ($1,035 + $970 - 2 x $1,000) / (2 x $1,000 x 0.01^2) = $5 / $0.2 = 25

在使用久期来估计债券价格时，必须使用[凸性调整。](/convexity-adjustment)凸度调整公式为：

凸度调整 = 凸度 x 100 x (dy)^2

在本例中，凸度调整为：

凸度调整 = 25 x 100 x (0.01)^2 = 0.25

最后，使用久期和凸性来估计给定利率变化的债券价格，投资者可以使用以下公式：

债券价格变动=久期x收益率变动+凸度调整

＃＃ 强调

- 评估债券的凸度是衡量和管理投资组合的市场风险敞口的好方法。

- 当债券价格和利率下降时存在负凸性，导致收益率曲线呈凹形。
