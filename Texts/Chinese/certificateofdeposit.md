# 存款证明（CD）
##

什么是存款证明？

存款证明 (CD) 是一种通常由商业银行发行的储蓄账户，它限制您使用您投资的资金，但提供的利率远高于与常规储蓄账户相关的利率。存款在商定的期限内增值，但如果在该时间结束之前提取，可能需要支付费用。

## 更深层次的定义

当客户在银行开立 CD 账户时，她会在一段时间内投资特定数量的资金。发行人定期支付利息，直到到期日，此时账户持有人收到她的原始投资以及所有[利息](/interest)。到期时间较短的 CD 将为您提供可观的回报，但投资于通常具有更高收益率的较长到期 CD 是值得的。

与普通储蓄账户相比，CD 提供更高的收益来弥补流动性损失。它们也代表了低风险的[投资机会](/investmentincome)，因为账户持有人几乎不需要了解投资市场，并且他们由 FDIC 投保高达 250,000 美元。一些银行允许浮动利率，而另一些银行可能会将 CD 与股票市场或其他指数挂钩。利率几乎总是与通货膨胀挂钩。

[到期日](/maturitydate)之前提取本金存款的 CD 客户进行处罚，但一些银行允许 CD 所有者从 CD 期限内的应计利息中提取，尽管这会降低收益。一些 CD 会自动展期，将利息应用于本金，从而使所有者的收益复利，而另一些 CD 在到期后停止收取利息，可以手动续订或全额提取。

##存款证明示例

加菲猫从经营一家成功的千层面餐厅获得了一些额外的现金。作为一只猫，他对投资知之甚少，但他想以最小的努力来增加他的钱。他打开了一张价值 20,000 美元、期限为两年、利率为 3% 的 CD，在两年结束时它已经到期，现在价值 21,218 美元。

＃＃ 强调

- 与股票和债券相比，CD 是一种更安全、更保守的投资，提供的增长机会较低，但具有非波动性、有保证的回报率。

- 几乎每家银行、信用合作社和经纪公司都提供 CD 选项菜单。

- 虽然您在打开 CD 时锁定了一个期限，但如果您遇到紧急情况或计划更改，可以选择提前退出。

- 支付最高的存款证 (CD) 支付的利率高于最好的储蓄和货币市场账户，以换取将资金存放一段固定的时间。

- 全国最高的 CD 费率通常比每个学期的行业平均水平高出三到五倍，因此货比三家可以带来显着收益。

＃＃ 常问问题

### 存款证明 (CD) 如何运作？

存款证 (CD) 是银行和信用合作社提供的一种简单且流行的储蓄工具。当存款人购买 CD 时，他们同意将一定数量的钱存入银行一段时间，例如一年。作为交换，银行同意向他们支付预定的利率，并保证在期限结束时偿还其本金。例如，在一年期 5% 的证书上投资 1,000 美元意味着在一年内获得 50 美元的利息，加上您最初投资的 1,000 美元。

### CD 的优点和缺点是什么？

一些储蓄者喜欢 CD，因为它们提供的安全性以及它们完全可以预测的事实。另一方面，存款证通常承诺的回报率非常低，尤其是在近年来联邦基金利率处于历史低位的情况下。如果所提供的利率低于当前的[通货膨胀](/inflation)率，那么当根据通货膨胀调整后的基础来衡量时，CD 的投资者实际上会在投资上亏损。出于这个原因，注重收益的投资者可能更喜欢风险更高但潜在回报更高的投资。

###你会在CD上赔钱吗？

实际上，在 CD 上赔钱几乎是不可能的，原因有二。首先，它们由提供它们的银行或信用合作社担保，这意味着它们在法律上必须向您支付约定的利息和本金金额。其次，它们通常也由联邦政府投保，这意味着即使银行或信用合作社破产，您的本金也很可能仍会得到偿还。由于这些原因，CD 被认为是最安全的投资之一。
