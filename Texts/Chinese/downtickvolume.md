# 下行音量
## 什么是下行交易量？

下跌交易量是指以低于前一交易价格的价格交易的给定证券的股票数量。 Downtick 交易量用于表示市场的活跃程度。

## Downtick 成交量告诉你什么

Downtick 交易量是分析师和交易者用来了解资产价格当前走势的指标。当一笔交易的价格高于前一笔交易时，即为[上涨](/uptick)。当一笔交易以低于前一笔交易的价格发生时，即为[下跌](/downtick)。通过扩展，下跌交易量是下跌交易时有多少股票或合约易手。

下行交易量可以全天累计计算。或者，它可以跨多只股票进行计算，从而表明抛售兴趣是否广泛分布于多只股票，或者强（或弱）抛售是否与特定股票隔离。

还可以（在任何时间范围内）查看交易量，以评估在此期间交易量是否增加或减少。如果价格在一段时间内下跌，高于之前的交易量，则表明销售力增加。如果价格下跌发生在低于平均成交量的情况下，则表明卖盘可能并不强劲，但不幸的是，即使成交量低，价格也可能继续下跌。

## 下跌成交量作为市场指标

将上涨股票与下跌股票进行比较的常见指标是[Tick 指数](/tickindex)。该指标创建了纽约证券交易所股票上涨与下跌的比率。

当该比率超过或 +1,000 或 –1,000 时，该比率被视为极端。如果刻度线达到 –1,000，则观察反转。有很多销售可能很快就会耗尽。短线交易者将关注他们的图表，寻找潜在的上行趋势。也就是说，不能保证价格会很快或持续多久。

交易者还可以观察下跌的开始。任何时候 Tick Index 开始下跌都表明更多股票开始出现下跌（卖出）。这可以用作退出多头头寸或进入空头头寸的短期指标。

Tick Index 是如何将下行趋势纳入交易策略的一个示例。[武器指数(TRIN) 有点相似，](/arms)它查看有多少股票上涨与下跌，以及交易量。由于该指标包含交易量，因此通常与分时指数结合使用。

## Downtick 和 Tick 的区别

下跌是指交易以低于先前交易价格的价格发生。价格[变动](/tick)是价格在任一方向上可以做出的最小变动。但是，下跌不限于仅移动一个刻度。交易可能相距甚远，代表着许多变动。

## 使用 Downtick Volume 和 Uptick Volume 的限制

下跌和上涨量是衡量价格走势强度的非常短期的指标。它显示了逐个交易发生的情况，因此作为一个指标可能非常不稳定。

信息是历史的。它告诉我们发生了一笔交易，或者累积地向我们展示了发生了什么，但除非在数据中找到非常具体的模式，否则拥有这些信息可能不一定会导致知道何时是买入或卖出的好时机。像任何指标一样，由交易者决定是否可以获利。

下行交易量最好与其他形式的[技术分析结合使用](/technicalanalysis)。例如，如果价格处于下跌趋势中，则下跌交易量以及 Tick Index 和 Arms Index 可能有助于坚持做空交易，直到有证据表明股票或整个股市正在失去下行动力.随着价格走势和这些指标开始发出积极信号，相互确认，那么做多可能是个好主意。

＃＃ 强调

- 下跌时的高成交量表明交易者强烈希望卖出。

- 下跌交易量是指以低于先前交易价格的价格易手的股票或合约数量。

- Downtick 成交量提供了对卖压强度的洞察。
