# 本金汇率挂钩证券 (PERL)
## 什么是本金汇率挂钩证券 (PERL)？

本金汇率挂钩证券 (PERL) 是一种债务投资，每半年支付一次利息，收益率[与](/linked-exchangerate-system)外汇汇率挂钩。即本金还款金额由到期还款时某种货币对美元的汇率决定。

PERL 的许多买家都是将此类[债务证券](/debtsecurity)视为对冲外汇汇率波动的一种手段的公司。它们也可能被投机者购买，他们认为他们知道特定外币的价格走势。

## 了解本金汇率挂钩证券 (PERL)

PERL 是在两方之间买卖的债务证券或债务工具。他们每半年向买方付款一次，金额由特定货币对基础货币（通常是美元）的汇率决定。

这使得 PERL 成为一种[双币债券](/dual-currency-bond)，它以基础货币支付[息票](/coupon)和本金，同时本金支付根据设定的赎回公式而变化。通过这个公式，变量与所选货币相对于基础货币美元的走势相关联。

PERL通常以美元计价，其利息以美元支付，但其还款价值取决于一定时间范围内美元与特定外币之间的汇率[。](/exchangerate)

随着外币相对于美元升值，本金支付增加。随着外币兑美元贬值，支付减少。

希望开展全球业务的公司可以通过购买 PERL 来更安全地开展业务，这允许货币与美元保持联系。

### 反向 PERL

还有一个反向PERL。这以一种货币计价，但以另一种货币支付利息。

对于反向 PERL，本金支付随着基础货币相对于外币升值而增加，而支付随着基础货币贬值而减少。

反向 PERL 的一个例子是以日元计价的债券，它以美元支付利息。如果美元兑日元升值，投资者的[收益率](/yield)会增加，但如果美元贬值，收益率就会下降。

＃＃ 强调

- 如果美元兑其他货币升值，PERL 的收益率将会下降。

- 还有一个反向 PERL，如果美元兑其他货币升值，收益率会增加。

- PERL 是一种以美元购买并以美元支付利息但最终还款金额以第二种货币确定的债券。
