# 资本化比率
##什么是资本化比率？

资本化比率是衡量公司[资本结构中债务比例的指标](/capitalstructure)。它们是用于评估公司财务状况的更有意义的债务比率之一。

资本化比率包括债务权益比率、[长期债务资本化比率](/longtermdebt-capitalization)和总债务资本化比率。每个比率的公式是：

- 债务权益比率 = 总债务 / 股东权益

- 长期债务资本化=长期债务/（长期债务+股东权益）

- 总债务资本化 = 总债务 /（总债务 + 股东权益）

资本化比率也称为[杠杆比率](/leverageratio)。

## 了解资本化比率

基本上，资本化比率处理公司如何筹集资金或资本。债务和股权是公司可以用来为其运营融资的两种主要方法。

债务有一些优势。利息支付可以免税。债务也不会像发行额外股票那样稀释公司的所有权。当利率低时，进入债务市场很容易，并且有资金可以借出。债务可以是长期的，也可以是短期的，可以包括发行债券的银行贷款。股权可能比债务更昂贵。通过发行更多股票来筹集额外资本可以稀释公司的所有权。

另一方面，股权不必偿还。负债过多的公司可能会发现其行动自由受到债权人的限制和/或高利率支付损害了其盈利能力。所有情况中最糟糕的情况是在不利的经济条件下难以按时支付运营和债务负债。最后，一家竞争激烈的公司，如果因高额债务而步履蹒跚，就会发现其竞争对手利用其问题抢占更多市场份额。

> 比较公司的资本化比率与同行业公司的比率相比更有效。

>

## 资本化比率示例

即使是同一家公司，不同的比率也会产生不同的结果。

让我们考虑一家[短期债务](/shorttermdebt)为 500 万美元、长期债务为 2500 万美元、股东权益为 5000 万美元的公司。公司的资本化比率计算如下：

- 债务权益比率 = (500 万美元 + 2500 万美元) / 5000 万美元 = 0.60 或 60%

- 长期债务资本化 = 2500 万美元/（2500 万美元 + 5000 万美元）= 0.33 或 33%

- 总债务资本化 = (500 万美元 + 2500 万美元) / (500 万美元 + 2500 万美元 + 5000 万美元) = 0.375 或 37.5%

## 资本化比率的意义

由于债务的[税收保护，](/taxshield)高资本化比率可以增加[股本回报率](/returnonequity)，但较高的债务比例会增加公司破产的风险。

但是，公司可接受的资本化比率水平取决于其经营所在的行业。公用事业、管道和电信等[资本密集型行业](/capitalintensive)和具有可预测[现金流](/cashflow)的公司通常具有较高的资本化比率。相反，在技术和零售等行业，可以质押的资产相对较少的公司[的](/collateral)债务水平较低，因此资本化比率也较低。

公司可接受的债务水平取决于其现金流是否足以偿还此类债务。[利息覆盖率](/interestcoverageratio)是另一种流行的[杠杆比率](/leverageratio)[，](/interestcoverageratio)衡量公司的息税前利润 ( EBIT [)](/ebit)与其[利息费用的比率](/interestexpense)。例如，比率为 2 表示公司每产生 1 美元的利息费用就产生 2 美元。

与所有比率一样，应随着时间的推移跟踪公司的资本化比率，以确定它们是否稳定。还应将它们与同行公司的类似比率进行比较，以确定公司相对于同行的杠杆地位。

＃＃ 强调

- 资本化比率衡量公司资本基础中的债务比例、来自贷方和股东的资金。

- 资本化比率包括债务权益比率、长期债务资本化比率和总债务资本化比率。

- 一家公司可接受的资本化比率不是绝对的，而是取决于其经营所在的行业。
