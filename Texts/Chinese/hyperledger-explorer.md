# 超级账本资源管理器
## 什么是超级账本资源管理器？

Hyperledger Explorer 是一个开源区块[链](/blockchain)实用程序模块，允许用户创建一个用户友好的基于 Web 的应用程序，用户可以使用它查看、启动、组织或查询构成区块链网络不可或缺的一部分的各种工件和开发。它专门用于部署使用 Hyperledger 保护伞创建的区块链。

## 了解超级账本资源管理器

Hyperledger Explorer 应运而生，因为[Hyperledger 项目](/hyperledger)自 2016 年成立以来一直在不断发展，现在在[开源](/open-source)Linux 基金会的赞助下，全球 130 多个组织正在使用和贡献。由于具有如此广泛的适应性，因此需要一种易于使用、人类可解释的实用程序，该实用程序提供区块链事件的仪表板视图。因此出现了 Hyperledger Explorer，最初由 IBM 和 Intel 等技术支持者以及领先的金融部门清算和结算公司 DTCC 贡献。他们每个人最初都开发了自己的 Explorer 版本，后来在 Hyperledger 项目下合并他们的努力以实现共同目标。

与 Windows 资源管理器和任务管理器的标准实用程序一样，Hyperledger Explorer 可以被视为一种使用界面的简便方法，该界面允许用户查看区块链的必要网络信息。它包括名称、状态和网络节点列表、[块的详细信息](/block-bitcoin-block)、交易和相关数据、交易系列、链代码以及可能存储在区块链上的任何其他相关详细信息。

由于所有此类原始区块链数据通常采用人类难以阅读的格式，Hyperledger Explorer 试图通过使用图形、图表、图片和模板以及通常的搜索和监控工具来提供简单的可视化。

## Hyperledger Explorer 的工作原理

它的架构包括一个运行在后端的 Web 服务器，负责与所有其他组件交互并维护必要的查询服务器响应。 Web 套接字用于在服务器和 Hyperledger Explorer 的各种客户端组件之间进行通信。 RethinkDB 数据库用于存储有关区块链组件的必要详细信息，例如有关区块、交易和智能合约的信息，并且可以查询任何必要的信息。安全存储库负责确保仅维护安全且经过授权的访问以访问 Hyperledger Explorer。

Hyperledger Explorer 允许统一的企业级可视化，这可能是在区块链上开发特定功能或组件的区块链开发人员、寻求研究历史发展的研究人员或负责的区块链运营商实时需要的用于管理区块链，或由高层管理人员。
