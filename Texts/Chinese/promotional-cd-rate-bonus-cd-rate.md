# 存款证 (CD) 促销利率
## 什么是促销存款证 (CD) 利率？

促销[存款证明 (CD)利率，也称为奖金 CD 利率，是银行和](/certificateofdeposit)[信用合作社](/creditunion)为吸引新存款而提供的高于正常存款证明的回报率。通常，此促销率仅限于某些存款金额或某些短期内。

## 促销 CD 费率如何运作

促销存款证利率一般只针对短期存单提供，并且需要更高的[最低投资额](/minimum_investment)。与所有 CD 一样，它们保证最低回报率，并为[联邦存款保险公司 (FDIC)](/fdic)提供高达每人 250,000 美元的银行保险保障。股票是 CD 的信用合作社版本，风险也很低，因为它们通过[国家信用合作社管理局 (NCUA)获得了相同金额的保险](/ncua)。

[到期](/maturity)时，促销 CD 会更新为具有相同到期时间的标准 CD，并使用标准发布的 CD 费率而不是促销费率。但是，机构可能会通过提供比新 CD 收益率更高的展期率来激励投资者继续投资。促销价格用于吸引新客户或诱使现有客户购买更多 CD。

## 存款证 (CD) 解释

存款证是一种具有固定[到期日](/maturitydate)和[固定利率的储蓄凭证，](/fixedinterestrate)考虑到最低投资要求，以任何面额发行。期限可以短至几天或长达十年，但标准范围是三个月至五年，期限越长，利率越高。 CD 支付的利率高于[储蓄账户](/savingsaccount)。利率较高的 CD 可以获得更高的收益。网上银行往往具有最具竞争力的利率。

> 大多数 CD 都有固定利率，这意味着年收益率在期限内被锁定。

>

CD 可以在到期时自动更新，或者在到期时，本金加利息可用于提取。 CD 是一种[定期存款](/timedeposit)，限制持有人按需提取资金。提前[取款的罚款](/withdrawal-penalty)取决于 CD 的期限和发行机构。典型的提前取款罚款等于既定的利息金额。 FDIC 和 NCUA 保险不包括因提前取款而产生的罚款。

大多数 CD 都有固定利率，这意味着年收益率在期限内被锁定。例如，一张[年](/apy)[收益率 (APY)为](/apy)2.50% 的五年期 CD只需 5,000 美元的押金即可赚取约 625 美元。在一个利率为 1.50% 的储蓄账户中，相同的存款金额将赚取大约 375 美元。在这种情况下，CD 的收入将是高收益储蓄账户的 1.5 倍以上。

＃＃ 强调

- 到期时，促销 CD 通常会以标准发布的 CD 费率更新为具有相同期限的标准 CD。

- 促销 CD 费率通常适用于短期 CD，并且需要更高的最低投资。

- FDIC 为银行 CD 的每笔存款投保最高 250,000 美元。
