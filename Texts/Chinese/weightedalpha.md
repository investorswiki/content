# 加权阿尔法
## 什么是加权 Alpha？

加权阿尔法衡量证券在特定时期（通常是一年）内的表现，但与早期表现相比，近期活动相对更重要。

[Alpha](/alpha) (α) 是用于描述投资策略击败市场的能力或其“优势”的术语。阿尔法因此也经常被称为“[超额回报](/excessreturn)”或“[异常回报率](/abnormalreturn)”，它指的是市场是有效的，因此没有办法系统地获得超过整个大盘的回报。

## 了解加权 Alpha

顾名思义，加权阿尔法是衡量证券（例如股票）在特定时期（通常是一年）内上涨或下跌的[加权指标。](/weighted)通常，通过将更高的权重分配给较晚的绩效测量而不是分配给早期测量的权重，更多地强调最近的活动。这有助于给出一个更关注当前时期的回报数字，这在分析该证券时应该被证明更相关。这个指标在[技术分析师](/technical-analyst)和那些倾向于依靠分析来增强他们的交易决策的人中非常流行。

加权 alpha 使用加权数学计算得出[alpha](/alpha)性能数据。 Alpha 是相对于基准的风险调整绩效的衡量标准。在[资产管理领域](/assetmanagement)，阿尔法通常被认为是基金经理技能的代表。在分析股票时，这种推理也是有效的，这在一定程度上反映了公司管理团队的有效性。

例如，回报与基准相当的股票，根据假设的风险水平进行调整，其 alpha 为零。正 alpha 表示股票产生的回报高于基准，而负 alpha 表示相反。

## 加权 Alpha 计算

加权计算根据各种因素给出分配的权重。指数使用权重按价格或市值赋予证券更高的权重。在加权 alpha 计算中，通常会为时间序列中较近期的时间段回报赋予更高的权重。

[证券](/security)回报的一年。一般来说，如果证券具有正加权阿尔法，投资者可以假设其价格在过去一年中一直在上涨。相反，如果证券价格的加权 alpha 为负，投资者可以假设一年的价格回报较低。

## 加权 Alpha 推理

各种投资者使用加权阿尔法。最常见的技术分析师将使用加权 alpha 作为支持买入和卖出信号的指标。技术分析师使用这一指标来识别过去一年表现出强劲趋势的公司，更具体地说，将注意力集中在势头正在增强的公司上。当加权 alpha 为正时，它可以支持看涨的买入信号。当加权 alpha 为负时，它可以支持看跌的卖出信号。

例如，考虑一只股票在过去一年中通过看涨和看跌的趋势模式经历了几个高点和低点。使用[Bol](/bollingerbands) [linger Band](/bollingerbands)通道的技术分析师可能会看到价格正在接近其支撑趋势线。如果股票的加权 alpha 为正，则可以肯定该股票的价格在过去一年中基本上一直在上涨，从而支持另一次[看涨](/bull)推高。

在另一种情况下，交易者可能会在布林带通道中看到股票价格达到并开始超过其阻力带。这通常是反转的信号，并表示卖出信号。但是，如果该证券具有正加权 alpha，则它更有可能突破其阻力位并走高。因此，在这种情况下，加权 alpha 可以支持买入交易。

＃＃ 强调

- 加权 alpha 衡量证券在特定时期（通常是一年）内的表现，最近的活动更为重要。

- 加权 alpha 可以识别在过去一年中表现出强劲趋势的公司，更具体地说，可以识别出势头正在增强的公司。

- 正加权 alpha 表明证券产生的回报高于基准；负数表示相反。
