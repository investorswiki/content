# Räkna
## Vad är greve?

Termen count hänvisar till en form av [teknisk analys](/technicalanalysis) som använder [punkt- och siffror (P&F) diagram](/pointandfigurechart) för att utvärdera den vertikala rörelsen av aktiekurser. Designad för att användas för långsiktig investeringsanalys, punkt- och figurdiagram anses vara ett av de enklaste sätten för handlare att bestämma ingångs- och utgångspunkter. Handlare som använder räkningsanalys plottar prishöjningar och sänkningar som används för att fastställa riktpriser.

## Förstå Count

Teknisk analys är en typ av handelsdisciplin som använder diagram och grafer för att utvärdera tidigare [trender](/trend) för värdepapper, såsom pris och [handelsvolym](/volumeoftrade),. för att bestämma framtida resultat. Handlare använder tekniska analystekniker för att hitta [ingångs-](/entry-point) och [utgångspunkterna](/exit-point) som kommer att optimera vinsten. Vissa handlare använder punkt- och figurdiagram utan att ta hänsyn till tidens gång. Denna teknik kallas en räkneanalys.

Räkneanalys plottar Xs för att representera prishöjningar och Os för prissänkningar i diagrammet. Analytiker baserar räkneberäkningar på historiska sidledes prisrörelser och använder dem för att bestämma sannolikheten för att ett [prismål](/pricetarget) kan nås. Räkneanalys Xs och Os används med en traditionell skala och ett tidigare bestämt reverseringsbelopp. Traders använder detta för att avgöra om vissa positioner är lönsamma. Investerare kan granska sekvensen av prisfluktuationer för att uppskatta hur priserna kommer att röra sig i framtiden. Det finns flera räknemetoder, som t.ex. breakout count-metoden som används för att hitta ett hausseartat prismål måste användas med en aktiv P&F-köpsignal.

> Som investerare kan du granska sekvensen av prisfluktuationer för att uppskatta hur priserna sannolikt kommer att röra sig i framtiden.

>

## Typer av räkneanalys

Som nämnts ovan finns det olika metoder för räkningsanalys, såsom breakout, reversering och horisontella räkningar.

### Breakout Count Analysis

Metoden för [breakouträkning](/breakout) används för att hitta ett hausseartat prismål som måste användas med en aktiv P&F-köpsignal. Det finns fyra steg för denna metod.

Först måste den mest aktiva säljsignalen, känd som en [dubbelbottenuppdelning](/doublebottom),. hittas på P&F-diagrammet, som arbetar från höger till vänster. Det andra steget innebär att arbeta till höger om denna signal för att hitta nästa köpsignal eller [Double Top](/doubletop) Breakout. Kolumnen som producerar denna signal är nyckeln eftersom den blir måttkolumnen. Sedan måste höjden på måttkolumnen beräknas och multipliceras med lådans återföringsbelopp. Slutligen måste summan av denna beräkning läggas till den nedre delen av kolumnen till vänster om måttkolumnen.

### Analys av omkastningsräkning

Metoden för [reversering](/reversal) kan användas för att hitta [hausseartade](/bull) och [baisseartade](/bear) prismål. Det finns tre steg som används för att hitta hausseartade priser. Reverseringsräkningen måste användas med en aktiv P&F-köpsignal. Först, från vänster till höger, måste den senaste P&F-säljsignalen hittas. X-kolumnen bredvid säljsignalen blir måttkolumnen. Därefter måste höjden på kolumnen beräknas och multipliceras med lådans [återföringsbelopp](/reversalamount). Sedan måste summan läggas till den nedre delen av kolumnen till vänster om mätkolumnen.

### Horisontell räkningsanalys

För den horisontella räkningsmetoden måste ett [trängselmönster](/congestion) eller omkastning bildas på ett P&F-diagram. Trängselmönstret måste vara minst fem kolumner brett och måste ha en kolumn som bryter trängseln. Kolumnerna i detta mönster ska räknas. Detta är bredden. Efter att breakout-kolumnen inträffar på P&F-diagrammet kan analytiker multiplicera bredden med [boxstorleken](/boxsize) och återföringsbeloppet för att uppskatta prisförlängningen. Förlängningen läggs till den lägsta av mönstret för ett prismål.

## Höjdpunkter

– Det finns flera räknemetoder, som räknemetoden för breakout, räknemetoden för omkastning och den horisontella räknemetoden.

- Räkneanalys använder Xs för att representera prishöjningar och Os för prissänkningar för att fastställa målpriser.

- Räkneberäkningar baseras på historiska sidledes prisrörelser, som används för att bestämma sannolikheten för att nå ett prismål.

– Räkna är en form av teknisk analys som använder punkt- och figurdiagram för att utvärdera aktiekursernas vertikala rörelse.
