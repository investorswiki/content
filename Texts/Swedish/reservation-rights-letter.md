# Rättighetsförbehåll brev
## Vad är ett brev om rättighetsförbehåll?

Ett rättighetsförbehåll lämnas av ett [försäkringsbolag](/insurance) till en försäkrad part som anger att en skada kanske inte täcks av en försäkring. Rättsförbehållsbrev bestrider inte ett krav. Brevet indikerar dock att försäkringsgivaren utreder skadeanspråket och förbehåller sig rätten att avslå anspråket efter det att utredningen avslutats.

## Hur ett brev om rättighetsförbehåll fungerar

Förbehåll av rättigheter från ett försäkringsbolag är en avsiktsförklaring om att de förbehåller sig sina fulla lagliga rättigheter. Detta fungerar som ett meddelande om att de inte avsäger sig sina lagliga rättigheter att vidta åtgärder vid ett senare tillfälle. Ett brev skickas som ett meddelande om att en försäkringsgivare förbehåller sig sina rättigheter, och om den beslutar sig för att neka täckning senare kan den åberopa sitt ursprungliga rättighetsförbehåll som en varning.

Försäkringsgivare som utfärdar ett rättighetsförbehåll kan i slutändan avslå ett krav, eller så kan de besluta att försvara den försäkrade mot ett anspråk som riktas mot den. För att få en bättre förståelse för vad som hänt måste försäkringsgivaren göra en egen utredning. Meddelandet om att den har för avsikt att genomföra en utredning är ett rättighetsförbehåll. Dessa brev är obligatoriska eftersom när en försäkringsgivare får en skadeanmälan, innehåller den bara en liten mängd information om vad som hände, vad som orsakade skadorna och vem som var ansvarig.

Att ta emot brevet fungerar som en indikator för den försäkrade att anspråket kan avslås, eller att informationen i det ursprungliga anspråket utlöste frågor som behöver utvärderas ytterligare. Påståendet kan till exempel vara ofullständigt eller innehålla motstridiga uppgifter.

Försäkringsbolagen skickar ett förbehåll av rättighetsbrev eftersom att inte göra det kan betraktas som ett avstående från sina rättigheter vid ett senare tillfälle. För det mesta förekommer skrivelser om rättighetsförbehåll som generiska formbrev. De ska dock inte tas lätt på. Åtminstone alla som tar emot en sådan bör kontakta sitt försäkringsbolag för att se varför de tror att skadeanmälan kanske inte täcks. Ofta kommer de att berätta att de bara täcker sina baser.

> Enligt en ansvarsförsäkring kan din försäkringsgivare ha en bredare skyldighet att försvara den försäkrade än att faktiskt säkra mot skador.

>

## Krav för ett brev om rättighetsförbehåll

Rättighetsförbehållsbrevet innehåller specifik information om skadeanspråket, inklusive försäkringen i fråga, anspråket mot försäkringen och den del av anspråket som eventuellt inte täcks. Försäkrade som får ett rättighetsförbehåll bör kontakta sin försäkringsgivare för att få mer information om skadeanmälan och utredningsprocessen. Försäkringsgivaren kan ge viss inledande information om vilka aspekter av skadeanmälan som den undersöker. Den försäkrade kan överväga att kontakta en advokat om det verkar som att försäkringsgivaren har för avsikt att avslå kravet.

Även om en försäkringsgivare kan skicka ett brev om rättighetsförbehåll, är det fortfarande ansvarigt för att svara på stämningar i samband med ett krav medan det utför sin utredning. Försäkringsgivare skickar brevet för att visa att de förbehåller sig sina rättigheter, eftersom underlåtenhet att skicka brevet kan betraktas som ett avstående från rättigheter.

##Höjdpunkter

- Försäkringsbolag kommer att utfärda ett rättighetsförbehåll till den försäkrade för att tjäna som ett meddelande om att de genomför en utredning av skadeanmälan

- Förbehållsbrev kan verka generiska men är en formell indikator på att medan försäkringsbolaget går vidare med en skadeanmälan, kan vissa förluster inte täckas.

– Försäkringsgivare kan inte behålla sin reservation av rättigheter på obestämd tid, en försäkrad kan driva på deras beslut att ge eller neka täckning.
