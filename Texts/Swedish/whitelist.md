# Vitlista
Termen vitlista hänvisar till en lista över tillåtna och identifierade individer, institutioner, datorprogram eller till och med kryptovalutaadresser. I allmänhet är vitlistor relaterade till en viss tjänst, händelse eller information. Som sådan kan vitlistor presentera distinkta betydelser beroende på sammanhanget de används.

Till exempel, när användare registrerar sig för ett företags e-postlista, uppmanas de ofta att lägga till företagets e-post till sin vitlista, så att meddelandena inte hamnar direkt i användarnas skräppostmappar. I vissa fall kan företag också betala avgifter för att bli vitlistade av internetleverantörer, så att deras e-postmeddelanden inte automatiskt markeras som skräppost. Så genom att vitlista en e-postadress kan användare försäkra sig om att framtida e-postmeddelanden kommer att nå deras inkorg.

Vitlistor kan också användas i samband med nätverkssäkerhet. Till exempel kan lokala nätverk (LAN) skapa en lista över betrodda MAC-adresser så att en utomstående inte kan ansluta sig till den. På samma sätt kan trådlösa internetroutrar använda vitlistor så att endast identifierade användare kan dra nytta av internetanslutningen.

Utöver det kan begreppet vitlistning vara relaterat till en lista över datorprogram som anses vara säkra att använda. Många antivirusprogram har en fördefinierad lista över betrodda applikationer som inte kommer att riktas mot under systemgenomsökningar, men användare kan också skapa vitlistor manuellt.

Slutligen, i blockchain- och kryptovalutamiljön är vitlistor antingen relaterade till ICO-händelser (Initial Coin Offering) eller till uttagsadresser. I det första fallet kan kryptovalutaprojekt erbjuda en vitlistningsfas för investerare som är villiga att delta i den offentliga försäljningen av sina tokens. Så alla investerare som vill delta i ICO skulle behöva tillhandahålla sin personliga information innan de blir vitlistade (detta görs vanligtvis genom ett KYC-förfarande).

När det gäller uttagsadresser hänvisar vitlista till en lista över kryptovalutaadresser som användarna definierar som pålitliga. I så fall skulle de bara kunna ta ut pengar från sitt växlingskonto till de adresser som tidigare var vitlistade. På Binance är detta även känt som adresshantering eller vitlista för uttagsadresser, och det är ett effektivt sätt för användare att skydda sina pengar från hackare.
