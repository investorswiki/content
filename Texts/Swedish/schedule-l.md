# Schema L
## Vad är schema L: Transaktioner med intresserade personer?

Schema L är ett [skatteschema](/taxschedule) kopplat till IRS [Form 1040](/1040) som används för att beräkna standardavdraget för vissa skatteregistrerare. Schema L används endast av skattebetalare som ökar sitt standardavdrag genom att rapportera statliga eller lokala fastighetsskatter, skatter från köp av ett nytt motorfordon eller från en nettokatastrofförlust som rapporterats på [formulär 4684](/form-4684).

Schema L används också för dem som lämnar in formulär 990 eller formulär 990-EZ för att ge information om finansiella transaktioner och arrangemang mellan organisationen som lämnade in formulären och diskvalificerade personer enligt avsnitt 4958, eller andra intresserade personer. Schema L används också för att särskilja medlemmar i en organisations styrande organ som oberoende medlemmar .

## Förstå schema L: Transaktioner med intresserade personer

Schema L är nödvändig eftersom återbetalningar och rabatter som redan erhållits på fastighetsskatter minskar mängden ytterligare [standardavdrag](/standarddeduction) som en skattebetalare kan vara berättigad till. Schablonavdraget är den del av inkomsten som inte är skattepliktig som kan användas för att minska din totala skattekostnad. När det gäller motorfordon kan skattebetalare i stater som inte tar ut försäljningsskatt utan tar ut en annan avgift vid köp av ett nytt fordon behandla dessa avgifter som en skatt för detta formulär. Skattebetalare bör kontrollera om ett ökat schablonavdrag ger samma skatteförmån som att specificera avdrag.

Schema L används också för att rapportera partnerskapsinkomster till IRS. Så, till exempel, om du är en del av ett affärspartnerskap som delar ut inkomster till sina medlemmar, föreskriver federal skattelag att partnerskapet inte beskattas på vinsten, utan partnerna redovisar inkomsten på sina egna individuella avkastningar. [Blankett](/form-1065) [1065](/form-1065) används för att först deklarera affärspartnerskapsinkomst till IRS och sedan används schema L för att detaljera detaljerna i partnerskapets balansräkning .

Balansräkningen inkluderar alla affärstillgångar, eget kapital och kapital samt skulder och den ger en finansiell översikt över verksamheten. Balansräkningen anger också om det krävs andra former baserat på det totala ekonomiska beloppet, eftersom mer pappersarbete behövs om de totala partnerskapstillgångarna överstiger totalt 10 miljoner dollar.

Schema L finns tillgängligt på IRS webbsida .

## Vem kan spara schema L: Transaktioner med intresserade personer?

IRS webbplats listar specifika instruktioner angående användning och arkivering av ett schema L. Formuläret kan användas för de flesta organisationer, med specifika arkiveringsinstruktioner baserade på deras skattestruktur eller medlemskap. Schema L kan också användas för att kräva en nettokatastrofförlust [om](/disasterloss) du bor i ett område som drabbats av en federal katastrof.

##Höjdpunkter

– De flesta skattebetalare som använder schablonavdraget istället för att specificera gör det för att de inte behöver hålla koll på kvalificerade utgifter eller att de inte har tillräckligt med poster att dra av.

- Schema L är ett formulär som bifogas till amerikanska inkomstdeklarationer som används för att beräkna en skattebetalares standardavdrag.

- Schema L kan också användas för organisationer i samband med Form 990.

– Alla skattebetalare är inte berättigade till schablonavdraget.
