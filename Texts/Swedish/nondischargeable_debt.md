# Ej avskrivningsbar skuld
## Vad är skulder som inte kan frigöras?

Ej avskrivningsbar skuld är en typ av skuld som inte kan elimineras genom ett [konkursförfarande](/bankruptcy). Sådana skulder inkluderar, men är inte begränsade till, studielån; de flesta federala, statliga och lokala skatter; pengar lånade på ett kreditkort för att betala dessa skatter; och barnbidrag och underhållsbidrag.

## Ej avskrivningsbar skuld förklaras

Vissa ytterligare skulder kan vara oavslutbara, men endast om borgenären motsätter sig friskrivningen. Dessa inkluderar skulder som härrör från en krigsförlikning eller skilsmässa; skulder som uppstått på grund av bedrägliga handlingar; skulder från uppsåtliga och illvilliga handlingar till en annan person eller annan persons egendom; och skulder från förskingring, stöld eller brott mot förtroendeansvar.

## Sätt att fastställa skulder som inte kan avskrivas

Många oavslutbara skulder anses så eftersom de kan härröra från [handlingar av förseelse](/malfeasance) från gäldenärens sida. Det kan också finnas fel av utelämnanden som faller i denna kategori. Till exempel kan oplanerade skulder, som hänvisar till alla skulder som inte fanns med i konkursansökan, vara oavslutbara. Undantag från detta är möjliga, särskilt om borgenärerna hade kännedom om konkursansökan och inte vidtog några åtgärder.

Enligt [kapitel 7 i](/chapter7) konkursbalken inkluderar andra typer av oavslutbara skulder betalningar för personskada som orsakats av gäldenären när han var berusad och körde ett motorfordon. Borgenärer kan också bestrida avgifter som gäldenären vill ta bort genom konkurs. Om domstolen godkänner sådana invändningar kommer dessa skulder att bli oavslutbara.

Detta inkluderar kreditkortsköp som är skyldiga en enda fordringsägare för lyxvaror som överstiger vissa dollarbelopp som anskaffades inom 90 dagar efter konkursansökan. Men om gäldenären kan bevisa att han hade för avsikt att betala tillbaka borgenären eller att köpen inte var lyxvaror, kan domstolen medge att skulden frigörs. Kontantförskott över ett visst dollarbelopp som mottogs inom 70 dagar efter en konkursansökan kan också vara skuld som inte kan frigöras.

Det finns ytterligare omständigheter enligt kapitel 7 under vilka domstolen kan förklara skulder som oavslutbara. Detta inkluderar tillfällen då gäldenären förstör register över sin ekonomi, överför egendom i ett försök att dölja den för borgenärer, om gäldenären inte fullföljer genomgången av en kurs i privatekonomistyrning eller om gäldenären inte kan redogöra för vissa saknade tillgångar.

Det kan finnas tillfällen där en gäldenär tidigare ansökt om konkurs och fått [sina skulder avskrivna](/debt-discharge). Detta kan vara skäl under deras senaste konkurs för att deras skulder ska förklaras oavslutbara, beroende på vilken typ av konkurs som lämnades in och tidsramen.
