# Back to Back Engagemang
## Vad är ett back-to-back-åtagande?

Ett back-to-back-åtagande är ett åtagande att göra ett andra take-out-lån som piggybackar ett annat lån. Med ett back-to-back-åtagande, när villkoren för det första lånet är uppfyllda, kommer de att rullas in i det andra lånet.

## Förstå ett back-to-back-åtagande

Det bästa exemplet på ett back-to-back-åtagande är när en bank ger ett bygglån för att bygga ett hus. När huset är byggt och ett inflyttningsbevis utfärdat kommer banken att ge ett nytt lån, troligen ett bottenlån, för att ta bygglånet. Bankens åtagande kommer att specificera de villkor som måste uppfyllas för att åtagandet att finansiera det andra lånet ska vara giltigt. Termen "back-to-back-åtagande" kan också användas för att beskriva ett avtal om att köpa ett bygglån vid ett senare tillfälle.

## Fördelar med ett back-to-back-åtagande

Back-to-back-åtaganden hjälper långivare att begränsa sin risk. Till exempel, om en bank utfärdar ett lån med överenskommelsen om att en andra bank kommer att köpa ut det vid ett senare tillfälle, minskar ursprungsbanken risken genom att endast vara ansvarig under en kort period av lånets löptid. Ansvaret övergår till att banken köper upp lånet efter en förutbestämd period.

När ett back-to-back-åtagande används för att rulla ett bygglån till ett [hypotekslån](/mortgage),. minskar långivaren risken genom att få tillgång till säkerheter för att ytterligare säkra lånet om låntagaren fallerar. Ett bygglån ger inte långivaren tillgång till mycket säkerhet. Men om lånet rullas in i ett hypotekslån när bygget har slutförts, kan långivaren använda den nya strukturen som säkerhet.

## Exempel på ett back-to-back-åtagande

En låntagare får ett bygglån från Bank A för att bygga en ny restaurang. Bank A går med på lånet under förutsättning att ett back-to-back-åtagande görs med Bank B, där Bank B går med på att köpa ut bygglånet om ett år.
