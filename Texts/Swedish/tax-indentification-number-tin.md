# Tax Identification Number (TIN)
## Vad är ett skatteregistreringsnummer?

Ett Tax Identification Number (TIN) är ett niosiffrigt spårningsnummer som tilldelas och används för att identifiera en skattebetalare.

Om du får en inkomst, eller har någon rapporteringspliktig eller på annat sätt skattepliktig verksamhet (när det gäller en enhet eller ett företag), förväntar sig regeringen att du betalar skatt på det. För att göra det kräver IRS att du först tillhandahåller ett skatteidentifikationsnummer.

Ett personnummer är en form av TIN och används av de flesta skattebetalare när de lämnar in sina federala och statliga skatter.

Ett TIN krävs också för icke-skatterelaterade ändamål, såsom att öppna ett bankkonto, ansöka om ett lån, ansöka om ett jobb, köpa eller hyra fastigheter och kräva förmåner från staten.

## Andra typer av skatteidentifikationsnummer

### Personnummer

Den vanligaste typen av skatteidentifikationsnummer är personnummer (SSN) utfärdat av Social Security Administration. Ett personnummer tilldelas amerikanska medborgare, permanent bosatta och vissa icke-invandrade arbetstagare. Föräldrar måste ansöka om ett personnummer för sina barn innan de kan göra anspråk på dem som anhöriga.

Ett personnummer är obligatoriskt för individer som vill använda statliga tjänster, ansöka om förmåner, söka arbete eller ansöka om skatt. Att ansöka om ett personnummer (eller något annat TIN för den delen) är gratis.

### Arbetsgivarens identifikationsnummer

Employer Identification Number (EIN) är det TIN som utfärdas till företag, ideella organisationer, dödsbon, truster och alla andra företag som är skyldiga att betala skatt. Detta nummer används vid redovisning av årsinkomst för beskattning.

IRS erbjuder en lista med ja eller nej-frågor för att hjälpa dig att avgöra om du behöver ett EIN.

### Individuell skattebetalares identifieringsnummer

Om en utlänning eller bosatt utlänning (eller deras make eller anhöriga) inte är berättigad till ett personnummer, får de ett niosiffrigt IDIN (Individual Taxpayer Identification Number). Detta liknar ett vanligt niosiffrigt TIN, men för att erhålla detta måste individen skicka in formulär W-7 tillsammans med dokument som styrker deras invånarstatus.

### Adoptionsskattenummer

När det gäller inhemsk adoption av barn, om adoptivföräldrarna inte kan få barnets personnummer för skatteändamål, kan de ansöka om ett adoptionsskattenummer (ATIN). Vissa villkor måste uppfyllas, till exempel att adoptionen måste vara under behandling och att barnet ska vara amerikansk medborgare eller permanent bosatt.

### Förberedarens skatteidentifikationsnummer

Utfärdat till skatteförberedare måste ett PTIN eller Preparer Tax Identification Number inkluderas på alla deklarationer som fylls i av en betald skatteexpert.

## Hur får man ett skatte-ID-nummer

IRS erbjuder information om hur du ansöker om olika skatteidentifikationsnummer.

- **Personnummer:** Fyll i formulär SS-5, ansökan om socialförsäkringskort i PDF-format och skicka in bevis på din identitet, ålder och amerikanskt medborgarskap eller laglig utlänningsstatus, säger IRS.

- **Arbetsgivaridentifikationsnummer:** Det finns ett antal sätt att ansöka om ett EIN, inklusive online, fax, post och telefon.

- **Individuellt skattebetalares identifieringsnummer:** Fyll i IRS-formulär W-7, IRS-ansökan om individuellt skattebetalaridentifikationsnummer. Du måste skicka in dokumentation av din utlänning eller bosatt utlänning med formuläret W-7. Du kan posta materialet, ta det till ett IRS-kontor eller skicka in det via en acceptansagent auktoriserad av IRS.

- **Taxidentifikationsnummer för adoption:** Fyll i blankett W-7A, ansökan om skattebetalares identifieringsnummer för pågående amerikanska adoptioner.

- **Förberedarens skatteidentifikationsnummer:** Skatteberedare kan ansöka om ett PTIN genom IRS-registreringssystemet. Du kan också fylla i blankett W-12, IRS Paid Preparer Tax Identification Number Application. IRS noterar dock att pappersansökan tar fyra till sex veckor att behandla.

## Höjdpunkter

– Det finns olika typer av skatteregistreringsnummer, till exempel arbetsgivarens identifikationsnummer, individuellt skatteregistreringsnummer och adoptionsskattenummer.

– TIN krävs även när man ansöker om kredit och anställning.

- Skattenummer är niosiffriga spårningsnummer som används av IRS enligt skattelagar.

- IRS utfärdar alla amerikanska skatte-ID-nummer förutom personnummer, som utfärdas av Social Security Administration.

- Skatteregistratorer måste ange sitt skatteregistreringsnummer på skatterelaterade dokument och när de begär förmåner.

## Vanliga frågor

### Hur kan jag få ett TIN?

Du kan få ett skatteregistreringsnummer genom att ansöka direkt till rätt byrå. Du kan till exempel få ett personnummer via Social Security Administration. Om du behöver ett identifieringsnummer för enskild skattebetalare eller ett för ett företag, kanske du kan få ett direkt från Internal Revenue Service.

### Är ett TIN detsamma som ett personnummer?

Ett personnummer är ett exempel på ett TIN. Det låter dig söka arbete i USA, för att få kredit hos banker och andra långivare, och du kan lämna in dina årliga skattedeklarationer till IRS med ditt SSN.

### Är ett TIN detsamma som ett skatte-ID?

Ett TIN är detsamma som ett skatte-ID eftersom det står för skatteregistreringsnummer eller skattebetalarens identifikationsnummer.

### Hur vet jag om jag behöver ett TIN?

Du behöver ett TIN om du är auktoriserad att arbeta i USA eller om du planerar att anmäla skatt till IRS. Du behöver också ett TIN om du vill dra nytta av förmåner eller tjänster som erbjuds av regeringen. Om du driver ett företag eller annan organisation behöver du också ett TIN för att sköta din dagliga verksamhet och för att rapportera dina skatter.

### Hur hittar jag mitt TIN online?

Byrån som utfärdar ditt TIN ger dig ett officiellt dokument med din unika identifierare. Eftersom det är en känslig information kan du inte gå online för att hämta den. Om du inte kommer ihåg eller förlorar dina dokument bör du kontakta den utfärdande myndigheten, till exempel SSA, för att få ditt TIN igen.
