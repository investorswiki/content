# Philadelphia Stock Exchange (PHLX)
## Vad är Philadelphia Stock Exchange (PHLX)?

Philadelphia Stock Exchange (PHLX), som ligger i Philadelphia, Pennsylvania, var den första officiella amerikanska värdepappersbörsen, som bildades 1790. Men den fokuserar för närvarande på aktie-, valuta- och indexoptioner snarare än aktiehandel.

## Förstå Philadelphia Stock Exchange (PHLX)

Ursprungligen känd som Board of Brokers, var PHLX den första formella värdepappersbörsen i USA och föregick dess mycket större kusin, [New York Stock Exchange (NYSE)](/nyse),. med två år. Det var också en av de första börserna som anammade elektronisk handel, när den 1975 introducerade ett system för routing och exekvering av aktieorder som heter PACE (Philadelphia Automated Communication and Execution system). Detta system kopplade ihop datorer och möjliggjorde omedelbar elektronisk orderexekvering.

1982 erbjöd PHLX valutaoptioner och inom sex år handlade de så mycket som 4 miljarder dollar per dag i underliggande värde.

2004 blev börsen den första våningsbaserade börsen som förvandlades från att vara ett sätesbaserat kooperativ till ett aktiebaserat vinstdrivande företag.

2008 köpte [Nasdaq OMX Group](/nasdaq),. nu Nasdaq Inc., PHLX och ändrade börsens namn till Nasdaq OMX PHLX, nu Nasdaq PHLX, och dess fokus till optioner. Det handlar för närvarande med mer än 2 600 optioner avräknade i US-dollar, [sektorindexoptioner](/indexoption) och aktieoptioner. Dessutom är det för närvarande den tredje största optionsmarknaden i USA med en andel på 17 %.

## Hem för populära index

PHLX har ett omfattande bibliotek med sektorindex, inklusive det mycket följt [PHLX KBW Bank Index (BKX)](/kbw-bank-index),. PHLX Gold/ Silver Sector Index (XAU) och PHLX Semiconductor Sector Index (SOX). Medan investerare inte kan köpa och sälja dessa, och de dussintals andra index som spåras på börsen, kan de handla med dem via optioner.

Börsen beräknar och sprider intradagsprisinformation på följande sektorindex, listade med namn och tickersymbol. Dessa symboler fungerar också som rotsymboler för sina respektive alternativ.

- PHLX KBW Insurance Index (KIX)

- PHLX KBW Regional Banking Index (KRX)

- PHLX Chemical Sector Index (XCM)

- PHLX Defence Sector Index (DFX)

- PHLX Drug Sector Index (RXS)

- PHLX Europe Sector Index (XEX)

- PHLX Housing Sector Index (HGX)
