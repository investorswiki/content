# Tidsvarierande volatilitet
## Vad är tidsvarierande volatilitet?

Tidsvarierande volatilitet avser fluktuationer i [volatilitet](/volatility) över olika tidsperioder. Investerare kan välja att studera eller överväga volatiliteten hos ett underliggande [värdepapper](/security) under olika tidsperioder. Till exempel kan volatiliteten för vissa tillgångar vara lägre under sommaren när handlare är på semester. Användningen av tidsvarierade volatilitetsmått kan påverka förväntningarna på investeringar.

## Hur tidsvarierande volatilitet fungerar

Tidsvarierande volatilitet kan studeras i vilken tidsram som helst. I allmänhet kräver volatilitetsanalys matematisk modellering för att generera volatilitetsnivåer som ett mått på risken för ett underliggande värdepapper. Denna typ av modellering genererar historisk volatilitetsstatistik.

Historisk volatilitet kallas generellt för [standardavvikelsen](/standarddeviation) för priser för ett finansiellt instrument, och därmed ett mått på dess risk. Med tiden förväntas ett värdepapper ha varierande volatilitet som är föremål för stora prissvängningar, med aktier och andra finansiella instrument som uppvisar perioder med hög volatilitet och låg volatilitet vid olika tidpunkter.

Analytiker kan också använda matematiska beräkningar för att generera implicit volatilitet. Implicit volatilitet skiljer sig från historisk volatilitet genom att den inte baseras på historiska data utan snarare en matematisk beräkning som ger ett mått på marknadens uppskattade volatilitet baserat på aktuella marknadsfaktorer.

## Historisk volatilitet

[Historisk volatilitet](/historicalvolatility) kan analyseras efter tidsperioder baserat på tillgången på data. Många analytiker försöker först modellera volatilitet med så mycket tillgänglig data som möjligt för att hitta volatiliteten i säkerheten under hela dess livstid. I denna typ av analys är volatilitet helt enkelt standardavvikelsen för ett värdepappers pris runt dess medelvärde.

Att analysera volatilitet efter angivna tidsperioder kan vara till hjälp för att underskatta hur ett värdepapper har betett sig under vissa marknadscykler, kriser eller riktade händelser. Tidsserievolatilitet kan också vara till hjälp för att analysera volatiliteten hos ett värdepapper under de senaste månaderna eller kvartalen jämfört med längre tidsramar.

Historisk volatilitet kan också vara en variabel i olika marknadsprissättningar och kvantitativa modeller. Till exempel kräver [Black-Scholes Option Pricing Model](/blackscholes) den historiska volatiliteten hos ett värdepapper när man försöker identifiera dess optionspris.

## Implicit volatilitet

Volatilitet kan också extraheras från en modell som Black-Scholes-modellen för att identifiera marknadens nuvarande antagna volatilitet. Med andra ord kan modellen köras bakåt med det observerade marknadspriset för en option som indata för att imputera vad volatiliteten hos den underliggande tillgången måste vara för att uppnå detta pris.

Generellt är implicit volatilitets tidsram baserad på tiden till förfall. Sammantaget kommer optioner med en längre tid till förfall att ha högre volatilitet medan optioner som löper ut inom en kortare tid kommer att ha lägre implicit volatilitet.

## Nobelpriset i ekonomi 2003

År 2003 vann ekonomerna Robert F. Engle och Clive Granger Nobels minnespris i ekonomi för deras arbete med att studera tidsvarierande volatilitet. Ekonomerna utvecklade modellen [Autoregressive Conditional Heteroskedasticity](/autoregressive-conditional-heteroskedasticity) (ARCH). Denna modell ger insikt för att analysera och förklara volatilitet över olika tidsperioder. Dess resultat kan sedan användas i prediktiv riskhantering som kan bidra till att minska förluster i en mängd olika scenarier.

## Höjdpunkter

- Volatilitetsanalys kräver användning av finansiella modeller för att lösa statistiska skillnader i prisfluktuationer över olika tidsramar.

- Tidsvarierande volatilitet beskriver hur prisvolatiliteten på en tillgång kan förändras givet olika tidsperioder.

- Volatilitet tenderar att vara medelåtergående, därför kan perioder med hög volatilitet följas av perioder med låg och vice versa.
