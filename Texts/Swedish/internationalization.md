# Internationalisering
### Vad är internationalisering?

Internationalisering beskriver processen att designa produkter för att möta behoven hos användare i många länder eller designa dem så att de enkelt kan modifieras för att uppnå detta mål. Internationalisering kan innebära att designa en webbplats så att den estetiska layouten fortfarande fungerar korrekt när den översätts från engelska till spanska. Detta kan vara svårt att uppnå eftersom många ord på spanska har fler tecken än sina engelska motsvarigheter. De kan alltså ta mer plats på sidan på spanska än på engelska.

I [ekonomisammanhang](/economics) kan internationalisering hänvisa till ett företag som vidtar åtgärder för att öka sitt fotavtryck eller ta större marknadsandelar utanför sitt hemland genom att förgrena sig till internationella marknader. Den globala företagstrenden mot internationalisering har hjälpt till att driva världsekonomin in i ett tillstånd av [globalisering](/globalization),. där ekonomier över hela världen blir starkt sammankopplade på grund av gränsöverskridande handel och finans. Som sådana påverkas de i hög grad av varandras nationella aktiviteter och ekonomiska välbefinnande.

## Förstå internationalisering

När ett företag försöker sälja sina varor utomlands kan det upptäcka att det finns flera vägspärrar i vägen. Vissa kan vara tekniska hinder som måste övervinnas; till exempel olika spänningar på hushållselen eller olika stickproppsformer som finns runt om i världen. Dessa kan åtgärdas genom tekniska anpassningar. Andra hinder kan vara kulturella, till exempel i Indien äter många hinduer inte nötkött. Detta innebär att för att internationalisera sig måste Mcdonald's fokusera på kyckling, fisk och andra menyalternativ utan nötkött som bättre överensstämmer med lokal sed och kultur. Att kunna anpassa sig flexibelt ger en ökad internationalisering.

Det finns många incitament som kan inspirera företag att sträva efter internationalisering. Till exempel i USA kan företag som betalar orimliga omkostnader dra ner på utgifterna genom att sälja produkter i länder med relativt svagare valutor eller i länder som har lägre levnadskostnader. Företag kan också dra nytta av internationalisering genom att sänka kostnaderna för företagande via minskade arbetskostnader som läggs ut på utländska marknader där varor kommer att säljas. Internationalisering kan alltså leda till produktinternationalisering eftersom produkter som säljs av multinationella företag numera ofta används i flera olika länder.

Från och med 2019 kom över 50 % av intäkterna från företag i det amerikanska [S&P 500-indexet](/sp500) från källor utanför USA. Detta är ett tydligt tecken på att stora amerikanska företag bedriver en stor del av sin verksamhet internationellt.

> Företag som vill intensifiera internationaliseringsarbetet bör vara medvetna om potentiella handelshinder som kan begränsa deras utsikter för handel utomlands.

>

## Exempel på internationalisering

När ett företag producerar varor för ett brett spektrum av kunder i olika länder, måste de produkter som internationaliseras ofta vara lokaliserade för att passa behoven hos ett visst lands konsumenter.

Till exempel måste ett internationaliserat program vara lokaliserat så att det visar datumkonventionen som "14 november" i USA, men som "14 november" i England. På samma sätt mäts enheter i Amerika i fot eller miles, medan de i Europa och Kanada använder det metriska systemet. Det betyder att bilar som säljs på dessa marknader snabbt måste kunna växla mellan miles och kilometer.

## Höjdpunkter

– Den här processen används av företag som vill expandera sitt globala fotavtryck bortom sin egen inhemska marknad och förstår att konsumenter utomlands kan ha olika smaker eller vanor.

– Internationalisering kräver ofta att man modifierar produkter för att passa de tekniska eller kulturella behoven i ett visst land, som att skapa kontakter som är lämpliga för olika typer av eluttag.

- Internationalisering beskriver att designa en produkt på ett sätt så att den lätt kan konsumeras i flera länder.
