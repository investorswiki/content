# Återförsäljarincitament
## Vad är återförsäljarincitament?

Bilhandlare erbjuder ofta sina kunder incitament att köpa från dem. Dessutom erbjuder biltillverkarna incitament till återförsäljarna, och dessa incitament sänker återförsäljarnas kostnader för att köpa fordonen från tillverkaren.

## Djupare definition

Biltillverkare använder ibland återförsäljarincitament för att öka försäljningen av en specifik modell, och erbjuder ofta olika incitament i specifika regioner i landet.

Till exempel, om vissa modeller inte säljer lika bra som andra, kan tillverkaren erbjuda ett cashback-incitament för var och en som gör det. Detta kan skapa konkurrens mellan återförsäljare som vill sälja mer av den modellen än andra så att de skördar de ekonomiska frukterna.

Återförsäljare publicerar inte dessa incitament som de gör med andra incitament som de erbjuder sina kunder. Och även om de kan överföra besparingarna till sina kunder, är de inte skyldiga att göra det, och vissa gör det inte.

Men om en kund känner till ett incitament kan han eller hon använda det för att förhandla fram ett lägre pris eller övergripande bättre affär på fordonet. Incitament listas ibland online eller kan upptäckas genom att jämföra prisofferter från flera återförsäljare för att se vem som erbjuder den brantaste rabatten.

Förutom incitament för att sälja specifika modeller av fordon, kan tillverkare erbjuda återförsäljarincitament till specifika återförsäljare för att hjälpa dem att öka försäljningen när de öppnar. Eller om de har fordon som de vill lasta av kan de erbjuda incitament till återförsäljare som går med på att ta fler av dem.

Vissa incitament gäller inte att sälja specifika fordon, utan att uppfylla långsiktiga kvoter. Till exempel kan en återförsäljare erbjuda ett reducerat pris till en kund för ett fordon så att det kan uppfylla ett kommande försäljningsmål och tjäna ett incitament.

Vissa incitament erbjuder så mycket till återförsäljarna att de är villiga att ta en förlust på en viss transaktion för att få den rejäla belöningen.

## Exempel på återförsäljarincitament

En tillverkare kan erbjuda återförsäljare pengar tillbaka på vissa nivåer eller nivåer, och erbjuda mer pengar när de säljer fler fordon. När återförsäljaren når högre nivåer tjänar den mer per fordon än den gjorde på den tidigare nivån. Så om en specifik transaktion skulle få återförsäljaren till nästa nivå, kan den erbjuda en rabatt till kunden så att den kan nå nästa incitamentsnivå och tjäna mer på varje fordon den säljer efter det.

## Höjdpunkter

- Ett återförsäljarincitament är en finansiell strategi som används av tillverkare för att motivera återförsäljare att sälja sina produkter genom att erbjuda rabatter på dessa produkter.

- Återförsäljarincitament används för att stimulera försäljning av långsammare säljande modeller, för att justera lager, eller efter att specifika månatliga försäljningsmål har uppnåtts för att motivera säljare att fortsätta sälja

- Återförsäljarincitament kan ta formen av ett reducerat inköpspris för återförsäljaren, en kontant betalning eller ett kontantincitament, såsom en rabatt till konsumenten.
