# Mainnet Swap
I huvudsak består ett mainnet-byte av att byta från ett blockchain-nätverk till ett annat. I de flesta fall sker bytet när ett kryptovalutaprojekt migrerar från en tredjepartsplattform (t.ex. Ethereum) till deras eget inhemska blockkedjenätverk. Vid denna tidpunkt ersätts deras kryptovaluta-tokens gradvis av nyutgivna mynt och all blockchain-aktivitet flyttas till den nya kedjan.

Låt oss ta [BNB](/binance-coin-bnb) som ett exempel. Efter [mainnet-](/mainnet) lanseringen av Binance Chain uppmuntrades användare att migrera från Ethereum blockchain till Binance Chain. Därför började [ERC-20](/erc-20) BNB-tokeninnehavare att ersätta sina tokens med det nyutgivna BEP2 BNB-myntet (det ursprungliga myntet i Binance Chain). Mainnet-bytet följde ett förhållande på 1:1 så att 1 ERC-20 BNB hade samma värde som 1 BEP2 BNB. Efter bytet brändes alla återstående ERC-20 BNB-tokens, så nu kan bara BNB för den nya kedjan användas.

Därför sker ett mainnet swap när ett blockchain-projekt ersätter tidigare utgivna tokens med deras nya kryptovaluta, som vanligtvis körs på deras eget blockchain-nätverk. Denna process kan också kallas "token-migrering". Vanligtvis börjar bytet av mainnet direkt efter mainnet-starten.

Det bör noteras att migreringen av tokens inte alltid enbart förknippas med lanseringen av en ny blockchain. Det kan också ske i situationer där projekt helt enkelt flyttas från ett protokoll till ett annat. Till exempel utförde Storj ett mainnet-byte, och migrerade deras tokens från ett Bitcoin-baserat protokoll till Ethereum-nätverket som ett sätt att undvika skalbarhetsproblem.

Varje huvudnätsbyte har sina egna särdrag när det gäller utförande. Vissa projekt tilldelar migreringsperioder, med en fördefinierad deadline för användare att byta ut sina gamla tokens mot de nya. Om de inte gör det kan de sluta med att förlora tillgången till sina pengar eftersom de gamla tokens vanligtvis förstörs eller fryses.

Däremot gav mainnet-bytet som utfördes av Binance användare friheten att lämna sina BNB-tokens på sitt Binance-konto, så bytet kunde göras automatiskt av Binance-teamet. Faktum är att bytet utfördes genom uttagsbegäranden. Med andra ord, genom att behålla sin ERC-20 BNB på sitt Binance-konto kan användare senare ta ut den nyutgivna BEP2 BNB i lika stora mängder, utan att behöva oroa sig för att manuellt byta sina tokens.
