# Skattefusk
## Vad är ett skattefusk?

Termen "skattefusk" syftar på en individ eller grupp som inte betalar den skatt som krävs enligt lag. Beroende på användningen kan termen också syfta på individer som använder aggressiva [skatteflyktsstrategier](/tax_avoidance) trots att de tekniskt följer lagens bokstav.

## Förstå ett skattefusk

Regeringar är beroende av skatteintäkter för att betala för utgifter som [sjukvård](/health_care_sector),. polis, offentlig utbildning, trafikinfrastruktur och militären. För många skattebetalare utgör dock skatter en stor kostnad, vilket skapar ett starkt incitament att minska sin [skatteskuld](/taxliability) när det är möjligt.

Medan vissa program tillåter skattebetalare att sänka sina skatter lagligt – till exempel genom att bidra med pengar till en arbetsgivarsponsrad pensionsplan som en [401(k)](/401kplan),. andra tillåtna avdrag eller legitima [skatteskydd](/taxshelter) – använder vissa människor också tvivelaktiga skatteskydd och andra olagliga medel för att slippa betala skatt.

Varje individ som använder strategier för att undvika att betala skatt eller betala färre skatter än de borde på ett olagligt sätt (eller på ett lagligt sätt som anses oetiskt) anses vara ett skattefusk.

## Stoppa ett skattefusk

Eftersom skatteintäkter är avgörande för att regeringen ska kunna finansiera sina utgifter, har [Internal Revenue Service](/irs) (IRS) olika program utformade för att motverka, upptäcka och bestraffa skattefusk. Till exempel har IRS ett program på plats där [whistleblowers](/whistleblower) kan rapportera individer eller företag som de misstänker fuskar sin skatt. Som ett incitament att rapportera sådant [bedrägeri](/fraud) erbjuder IRS en potentiell belöning till whistleblowers, som betalas om rapporten leder till ett bekräftat fall av bedrägeri .

IRS har också den lagliga makten att utdöma betydande straff för skattefusk, inklusive höga böter och tid i fängelse .

Ibland kan individer fuska på sina skatter utan att ens vara medvetna om det. Detta kan uppstå på grund av den avsevärda komplexiteten i modern skattelagstiftning, som ofta kräver att professionella revisorer och advokater hjälper individer och företag att ge råd om deras verkliga skatteförpliktelser.

För att undvika oavsiktligt fusk tillhandahåller IRS en rad sökbara onlineresurser för dem som vill utbilda sig om det amerikanska skattesystemet. Dessutom finns det olika populära mjukvarupaket som kan hjälpa människor genom processen att betala sin skatt .

## Exempel på skattefusk

Det finns många sätt som en individ eller organisation kan klassificeras som ett skattefusk. Till exempel, vare sig avsiktligt eller av misstag, kan en person arbeta på enbart kontantbasis och avstå från att deklarera hela eller delar av sin [inkomst](/income) när de lämnar in sina skatter varje år.

Om deras arbetsgivare inte förde korrekta register över dessa kontantbetalningar, kan det vara omöjligt för IRS att spåra dessa transaktioner. Till exempel, om en arbetare fick 500 $ i veckan men bara rapporterade att de fick 250 $ i veckan, skulle de betala färre skatter genom att tillhandahålla denna falska information, vilket gör dem till ett skattefusk.

Icke desto mindre skulle underlåtenheten att avslöja kontantinkomsten tekniskt sett vara ett exempel på skattebedrägeri, vilket potentiellt skulle göra skattefusket sårbart för böter eller andra påföljder om individen skulle fångas.

Andra exempel på hur skattefusk kan förekomma är att överskatta värdet av donationer till välgörenhet för att få ett högt inkomstskatteavdrag, att betala anställda "under bordet" utan ordentliga [löneskatteavdrag](/payrolltax) och att inte rapportera spelvinster eller andra oväntade summor.

## Höjdpunkter

- IRS försöker identifiera, bötfälla och eller fängsla skattefusk och har vissa metoder och program för att göra det.

– "Skattefusk" är en vardaglig term som syftar på individer eller organisationer som inte betalar sin skatt.

– Vanliga exempel på skattefusk är de som underlåter att redovisa inkomster som betalats kontant, eller som betalar sina anställda utan att göra nödvändiga löneskatteavdrag.

– "Skattefusk" kan också syfta på de som sänker sina skatter lagligt men på sätt som anses oetiskt.

– Det används i allmänhet för att hänvisa till människor som medvetet undandrar sina skatter, även om det också kan syfta på de som gör det av misstag.
