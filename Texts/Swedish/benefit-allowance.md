# Bidragsbidrag
## Vad är en förmånsersättning?

En förmånsersättning är pengar som ett företag eller en statlig myndighet tillhandahåller en anställd för ett specifikt ändamål, såsom transport, sjukvårdskostnader eller ett [flexibelt utgiftskonto](/flexiblespendingaccount). Förmåner som administreras till anställda kan fördelas genom ordinarie [löneadministration](/payroll).

## Hur förmånsbidrag fungerar

Arbetsgivare kan använda en förmånsersättning för att ge anställda flexibilitet att skapa ett förmånspaket som bäst motsvarar deras behov. Istället för att till exempel ålägga alla anställda en viss hälsovårdsplan, kan arbetsgivaren erbjuda en basplan plus ett förmånsbidrag.

Arbetsgivare kan inrätta ett förmånsbidrag som också inkluderar täckning för friskvårdsprogram, såsom gymmedlemskap, som bidrar till den anställdes allmänna hälsa. Den anställde kan använda förmånsersättningen till kompletterande förmåner som tandvårdsförsäkring eller täckning för [anhöriga](/dependent).

Arbetsgivare kan också tillåta sina anställda att lägga en förmånsersättning till [livförsäkring](/lifeinsurance),. handikappförsäkring, synvård eller valfritt antal andra förmåner. Anställda får därmed skräddarsydda förmåner och arbetsgivare kan erbjuda ett konkurrenskraftigt förmånspaket som hjälper dem att rekrytera och behålla topptalanger.

## Sätt att strukturera förmånsersättningar

Små företag som kanske inte har resurser att erbjuda sjukförsäkring och andra förmånsplaner till sina anställda kan använda en förmånsersättning som ett alternativ för att skapa tillgång till täckning för sina anställda.

Förmånsbidrag kan erbjudas på en mängd olika sätt. Arbetsgivare kan skapa skattepliktiga [stipendier](/stipend) genom att ge anställda skattepliktiga höjningar. Detta ger den anställde ett fast stipendium för köp av sjukförsäkring. Den anställde kommer att få pengarna oavsett om de använder dem för upphandling av sjukförsäkring. Vanligtvis kommer den anställde att få ett formulär som beskriver hur mycket av stipendiet som ska redovisas som inkomst med sin skattedeklaration.

> Förmåner hjälper arbetsgivare att skapa skräddarsydda och konkurrenskraftiga förmånspaket som kan hjälpa dem att behålla sina bästa anställda.

>

Företag kan också erbjuda en skattefri ersättningsplan för att ge ett förmånsbidrag. Enligt detta alternativ skulle den anställde få ett fast belopp att lägga till sjukförsäkring; medlen utbetalas dock endast om försäkringen köps. För att få bidraget måste den anställde lämna in bevis på att han har köpt en sjukförsäkring för sig själv. Återbetalningarna beviljas dem sedan skattefritt.

## Särskilda överväganden

Tillsammans med förmånsersättningen kan en arbetsgivare utse en part som ska fungera som sjukförsäkringsmäklare för att hjälpa anställda att välja en plan.

Vissa arbetsgivare kan tidigare ha betalat ut förmånsersättningar för sjukförsäkring till anställda utan en formell plan på plats, men sådan praxis var inte i linje med reformerna i Affordable Care Act.

##Höjdpunkter

- En arbetsgivare kan ge sina anställda en förmånsersättning för en specifik användning, som för barnomsorg, transport, livförsäkring eller hälsoförmåner.

– Förmånsersättningar fördelas via ett företags ordinarie lönelista.

– Arbetsgivare kan också erbjuda en skattefri ersättningsplan för att ge sina anställda en förmånsersättning.

- Arbetsgivare använder ofta en förmånsersättning för att tillåta anställda att skapa ett användbart förmånspaket snarare än en plan som passar alla.
