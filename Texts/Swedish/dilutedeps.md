# Vinst per aktie efter utspädning (utspädd EPS)
## Vad är vinst per aktie efter utspädning?

Vinst per aktie efter utspädning avser ett företags vinst per aktie baserat på antalet utestående stamaktier och de stamaktier som kan konverteras från andra värdepapper om dessa värdepapper utnyttjades.

## Vad är skillnaden mellan grundläggande och utspädd vinst per aktie?

Resultat [per aktie före utspädning](/basic-earnings-per-share) står endast för antalet [utestående stamaktier](/outstandingshares) för perioden, medan vinst efter utspädning står för alla stamaktier inklusive de som eventuellt kan konverteras från andra värdepapper.

## Hur man beräknar utspädd vinst per aktie

Vinst per aktie efter utspädning kan beräknas genom att dividera [nettoresultatet](/netincome) med antalet stamaktier om alla teckningsoptioner och aktieoptioner utnyttjats och alla konvertibla skuldebrev och [preferensaktier](/preferredstock) konverterats. Om ett företag redovisade en förlust för perioden, skulle förlusten divideras med alla stamaktier, inklusive de potentiellt utspädande värdepapper som är utestående.

### Formel för utspädd vinst per aktie

Vinst per aktie efter utspädning = Nettoinkomst hänförlig till stamaktier / (vägt genomsnitt av utestående stamaktier under perioden + [Warranter + aktieoptioner + konvertibla obligationer + preferensaktier som potentiellt kan konverteras till stamaktier])

## Utspädd vinst per aktie Exempel: Apple (NASDAQ: AAPL)

I exemplet nedan för Apple skiljer sig den utspädda vinsten per aktie inte mycket i pris från grundsiffran. Apples beräkning för utspädd vinst visar ett något högre antal utestående stamaktier, inklusive de som kan konverteras från värdepapper, även om det inte gav någon uppdelning.

## Varför är utspädd vinst per aktie viktigt?

Utspädd vinst per aktie är viktig eftersom det är det accepterade vinsttalet som analytiker skulle publicera uppskattningar på och med vilket investerare och analytiker beräknar intjäningskvoter, inklusive [pris-till-vinst-kvoter](/price-earningsratio).

Den visas längst ner i [resultaträkningen](/incomestatement) och kommer sannolikt att vara den sista posten, efter grundvinst per aktie. För börsnoterade företag kan resultaträkningen hittas i den finansiella rapport som lämnas in kvartalsvis och årligen till Securities and Exchange Commission.

##Höjdpunkter

- Konvertering av dessa värdepapper minskar EPS, sålunda tenderar utspädd EPS alltid att vara lägre än EPS.

- Utspädande värdepapper är inte stamaktier, utan värdepapper som kan konverteras till stamaktier.

- Utspädd EPS anses vara ett konservativt mått eftersom det indikerar ett värsta scenario när det gäller EPS.

- Vinst per aktie efter utspädning (utspädd EPS) beräknar ett företags vinst per aktie om alla konvertibla värdepapper konverterats.
