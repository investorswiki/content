# Säkerhetsavtal
## Vad är ett säkerhetsavtal?

Ett säkerhetsavtal avser ett dokument som ger en långivare ett [säkerhetsintresse](/security-interest) i en angiven tillgång eller egendom som ställs som säkerhet. Villkoren fastställs i samband med att säkerhetsavtalet upprättas. Säkerhetsavtal är en nödvändig del av affärsvärlden, eftersom långivare aldrig skulle ge [kredit](/credit) till vissa företag utan dem. I händelse av att låntagaren fallerar kan den ställda säkerheten tas i beslag av långivaren och säljas.

## Förstå säkerhetsavtal

Företag och människor behöver pengar för att driva och finansiera sin verksamhet. Det finns sällan fall där enheter kan finansiera sig själva, varför de vänder sig till banker och andra investeringskällor för [kapital](/capital). Vissa långivare kräver mer än bara goda ord och räntebetalningar. Det är där säkerhetsavtal kommer in. Dessa är viktiga utkast till dokument mellan båda parter vid den tidpunkt då lånet förskotteras.

Säkerhetsavtal innehåller ofta covenants som beskriver bestämmelser för förskottering av medel, ett återbetalningsschema eller [försäkringskrav](/insurance). Låntagaren kan också tillåta långivaren att hålla [säkerheten](/collateral) för lånet tills återbetalning. Säkerhetsavtal kan även avse immateriell egendom såsom patent eller fordringar.

Ett säkrat skuldebrev kan innehålla ett säkerhetsavtal som en del av dess villkor. Om ett säkerhetsavtal listar en affärsegendom som säkerhet, kan långivaren lämna in ett [UCC-1-uttalande](/ucc-1-statement) för att fungera som en panträtt i fastigheten.

> Ett säkerhetsavtal minskar risken för fallissemang som långivaren står inför.

>

Förekomsten av ett säkerhetsavtal och en eventuell panträtt i den säkerheten kan påverka låntagarens möjlighet att få mer [finansiering](/financing) från andra långivare. Den egendom som användes som säkerhet kommer att bindas till den första långivarens villkor, vilket skulle innebära att säkrande av ytterligare ett lån mot samma egendom skulle leda till korssäkerhet.

## Särskilda överväganden

Många långivare är ovilliga att engagera sig i arrangemang som skulle ifrågasätta deras förmåga att få lämplig ersättning om låntagaren hamnar i [fallissemang](/default2). Företagare som söker finansiering från flera källor kan hamna i utmanande positioner om låntagare kräver säkerhetsavtal för sina tillgångar. Särskilt små företag kan ha få delar av egendom eller tillgångar som kan användas som säkerhet för att säkra lån.

Låntagaren kan ha begränsade möjligheter att tillhandahålla säkerheter som skulle tillfredsställa långivare. Även om ett säkerhetsavtal endast ger ett partiellt säkerhetsintresse i fastigheten, kan långivare vara ovilliga att erbjuda finansiering mot den fastigheten. Möjligheten skulle finnas kvar för [korssäkerhet](/cross-collateralization),. vilket skulle tvinga fastigheten att [likvideras](/liquidate) för att försöka låsa upp dess värde och ge kompensation till långivarna.

Egendom som kan tas upp som säkerhet enligt ett säkerhetsavtal inkluderar produktinventering, inredning, utrustning som används av ett företag, inventarier och [fastigheter som](/realestate) ägs av företaget. Låntagaren ansvarar för att säkerheten hålls i gott skick vid fallissemang. Den egendom som står som säkerhet får inte tas bort från lokalen om inte fastigheten behövs i den ordinarie verksamheten.

##Höjdpunkter

– Ett säkerhetsavtal är ett dokument som ger en långivare en säkerhetsandel i en angiven tillgång eller egendom som ställs som säkerhet.
