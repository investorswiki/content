# Defensiv intervallkvot (DIR)
## Vad är DIR (Defensive Interval Ratio)?

Det defensiva intervallkvoten (DIR), även kallad defensiv intervallperiod (DIP) eller grundläggande försvarsintervall (BDI), är ett finansiellt mått som anger antalet dagar som ett företag kan arbeta utan att behöva komma [åt anläggningstillgångar](/noncurrent-assets),. långsiktigt tillgångar vars fulla värde inte kan erhållas inom innevarande räkenskapsår, eller ytterligare externa finansiella resurser.

Alternativt kan detta ses som hur länge ett företag kan verka medan det endast förlitar sig på likvida tillgångar. DIR ses ibland som ett ekonomiskt [effektivitetsförhållande](/efficiencyratio) men anses oftast vara ett [likviditetsförhållande](/liquidityratios).

## Förstå det defensiva intervallförhållandet (DIR)

DIR anses av vissa marknadsanalytiker vara ett mer användbart likviditetsförhållande än det vanliga [snabba förhållandet](/quickratio) eller [nuvarande förhållandet](/currentratio) på grund av det faktum att det jämför tillgångar med kostnader snarare än att jämföra tillgångar med skulder. DIR används vanligtvis som en kompletterande finansiell analyskvot, tillsammans med det aktuella eller snabba förhållandet, för att utvärdera ett företags ekonomiska hälsa, eftersom det kan finnas väsentligt olika DIR- och snabba eller aktuella kvotvärden om till exempel ett företag har en stor antal utgifter men liten eller ingen skuld.

DIR kallas det defensiva intervallförhållandet eftersom dess beräkning involverar ett företags [omsättningstillgångar](/currentassets),. som också kallas defensiva tillgångar. Defensiva tillgångar består av kontanter, likvida medel, såsom obligationer eller andra investeringar, och andra tillgångar som lätt kan omvandlas till kontanter, såsom [kundfordringar](/accountsreceivable).

Till exempel, om ett företag har 100 000 USD kontanter till hands, 50 000 USD i [omsättbara värdepapper](/marketablesecurities) och 50 000 USD i kundfordringar, har det totalt 200 000 USD i defensiva tillgångar. Om företagets dagliga driftskostnader är lika med 5 000 USD är DIR-värdet 40 dagar: 200 000 / 5 000.

Naturligtvis anses ett högre DIR-tal vara bra, eftersom det inte bara visar att ett företag kan lita på sin egen ekonomi, utan det ger också ett företag tillräckligt med tid för att utvärdera andra meningsfulla alternativ för att betala sina utgifter. Som sagt, det finns inget specifikt nummer som anses vara det bästa eller rätta numret för en DIR. Det är ofta värt att jämföra DIR för olika företag i samma bransch för att få en uppfattning om vad som är lämpligt, vilket också skulle hjälpa till att avgöra vilka företag som skulle kunna vara bättre investeringar.

## Formel för defensiv intervallkvot (DIR)

Formeln för att beräkna DIR är:

**DIR (uttryckt som antal dagar) = omsättningstillgångar / dagliga driftskostnader**

var

**Omsättningstillgångar = kontanter + omsättbara värdepapper + nettofordringar**

**Dagliga driftskostnader = (årliga driftskostnader - icke-kontanta avgifter) / 365**

## Fördelar med det defensiva intervallförhållandet (DIR)

DIR är ett användbart verktyg för att utvärdera ett företags ekonomiska hälsa eftersom det ger en verklig statistik i antal dagar. På detta sätt vet ett företag exakt hur länge det kan fortsätta sin verksamhet genom att täcka dagliga driftskostnader utan att hamna i några ekonomiska svårigheter som sannolikt skulle kräva att det får tillgång till ytterligare medel genom antingen ny aktieinvestering, ett banklån eller försäljning av långa -fristiga tillgångar. Detta är oerhört viktigt för att hantera sin ekonomiska hälsa, eftersom den kan hantera sin balansräkning innan den måste ta på sig oönskade skulder.

I det avseendet kan det anses vara ett mer användbart likviditetsmått att undersöka än den nuvarande kvoten, som, samtidigt som den ger en tydlig jämförelse mellan ett företags tillgångar och dess skulder, inte ger någon definitiv indikation på hur länge ett företag kan fungera ekonomiskt utan stöter på betydande problem när det gäller den enkla dagliga verksamheten.

## Höjdpunkter

- Omsättningstillgångar jämförs med dagliga utgifter för att bestämma det defensiva intervallförhållandet.

- Den defensiva intervallkvoten (DIR) syftar till att beräkna hur många dagar ett företag kan verka medan det bara förlitar sig på likvida tillgångar.

- Den defensiva intervallkvoten kan ses över tid för att avgöra om ett företags likviditetsbuffert för att klara sina kostnader ökar eller minskar.

- Många analytiker ser det defensiva intervallförhållandet (DIR) som mer användbart än det snabba förhållandet eller det nuvarande förhållandet eftersom det jämför tillgångar med faktiska kostnader snarare än skulder.

- Även om ett högre DIR-nummer är att föredra, finns det inget specifikt nummer som indikerar vad som är rätt eller bättre att sikta på.
