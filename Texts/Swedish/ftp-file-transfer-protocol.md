# File Transfer Protocol (FTP)
## Vad är FTP (File Transfer Protocol)?

Termen filöverföringsprotokoll (FTP) hänvisar till en process som involverar överföring av filer mellan enheter över ett nätverk. Processen fungerar när en part tillåter en annan att skicka eller ta emot filer över internet. Ursprungligen användes det som ett sätt för användare att kommunicera och utbyta information mellan två fysiska enheter, men det används nu ofta för att lagra filer i [molnet](/cloud-computing),. vilket vanligtvis är en säker plats som hålls på distans.

FTP kan användas av ett [företag](/business) eller individ för att överföra filer från ett datorsystem till ett annat eller av webbplatser för att ladda upp eller ladda ner filer från deras servrar.

## Hur File Transfer Protocol (FTP) fungerar

Filöverföringsprotokollet tillåter individer och företag att dela elektroniska filer med andra utan att behöva vara i samma utrymme. Detta kan göras med hjälp av en FTP-klient eller via molnet. Oavsett alternativ kräver båda parter en fungerande internetanslutning.

De flesta [webbläsare](/web-20) kommer med FTP-klienter som gör det möjligt för användare att överföra filer från sin dator till en server och vice versa. Vissa användare kanske vill använda en tredjeparts FTP-klient eftersom många av dem erbjuder extra funktioner. Exempel på FTP-klienter som är gratis att ladda ner inkluderar FileZilla Client, FTP Voyager, WinSCP, CoffeeCup Free FTP och Core FTP.

Många människor har använt FTP tidigare utan att ens inse det. Om du någonsin har laddat ner en fil från en webbsida har du använt FTP. Det första steget är att logga in, vilket kan ske automatiskt eller genom att manuellt mata in ett användarnamn och lösenord. FTP kräver också att du kommer åt en FTP-server via ett specifikt portnummer. När du kommer åt FTP-servern via din FTP-klient kan du nu överföra filer. Inte alla offentliga FTP-servrar kräver att du loggar in eftersom vissa servrar gör att du kan komma åt dem anonymt.

Som nämnts ovan utvecklades FTP ursprungligen som ett sätt att skicka och ta emot filer mellan två fysiska datorer. Men med förändringar i tekniken kan användare utföra filöverföringar via molnet. Genom att använda molnet kan överföringar göras bekvämt och säkert (vilket skulle kunna skydda individer och företag från [dataintrång](/data-breach) ) och med relativt låg kostnad.

> Termen FTP-klient syftar på programvaran som låter dig överföra filer till en annan part.

>

## Särskilda överväganden

Filöverföringsprotokoll är ett av många olika protokoll som dikterar hur datorer och datorsystem beter sig på internet. Andra sådana protokoll inkluderar:

- **Hypertext Transfer Protocol (HTTP):** Designad för att överföra data över webben

- **Internet Message Access Protocol (IMAP):** Ger tillgång till anslagstavla eller e-postmeddelanden från en delad tjänst

- **Network Time Protocol (NTP):** Synkroniserar klocktider på datorer över ett nätverk

FTP gör det möjligt för datorer på internet att överföra filer fram och tillbaka. Som sådan är det ett viktigt verktyg för dem som bygger och underhåller webbplatser idag.

### Vad du ska leta efter i en FTP-klient

Enskilda FTP-klienter tillhandahåller olika funktioner som gör det möjligt för användare att ändra hur de laddar upp och laddar ner filer. Om du till exempel använder FileZilla låter programmet dig ställa in [bandbreddsgränser](/bandwidth) för filer. Detta gör att du kan kontrollera uppladdnings- och nedladdningshastigheter, vilket kan vara användbart om du hanterar flera filöverföringar samtidigt.

Andra funktioner du kanske vill leta efter i en FTP-klient inkluderar autentisering med publik nyckel, möjligheten att ställa in filkomprimeringsnivåer eller verktyg som gör att du kan söka på en server med filmasker.

## Exempel på File Transfer Protocol (FTP)

FTP-programvara är relativt enkel att installera. FileZilla är en gratis nedladdningsbar FTP-klient. Skriv in adressen till servern du vill komma åt, porten och lösenordet för åtkomst till servern.

När åtkomst har beviljats kommer användarens filer på sitt lokala system samt den åtkomstserver att vara synliga. Användaren kan ladda ner filer från servern till det lokala systemet, eller ladda upp filer från det lokala systemet till servern. De kan också göra ändringar i filer på servern, så länge de har rätt behörighet att göra det.

##Höjdpunkter

– FTP är ett viktigt verktyg för dem som bygger och underhåller webbplatser.

- Filöverföringsprotokoll är ett sätt att ladda ner, ladda upp och överföra filer från en plats till en annan på internet och mellan datorsystem.

- FTP möjliggör överföring av filer fram och tillbaka mellan datorer eller genom molnet.

- Användare kräver en internetanslutning för att kunna utföra FTP-överföringar.

- Många FTP-klienter är gratis att ladda ner, även om de flesta webbplatser redan har FTP inbyggd.
