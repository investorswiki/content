# Fundamentalt vägt index
## Vad är ett fundamentalt viktat index?

Ett fundamentalt viktat index är en typ av aktieindex där komponenterna väljs utifrån grundläggande kriterier i motsats till börsvärde. Fundamentalt viktade index kan basera sin konstruktion på en rad grundläggande mätvärden, såsom intäkter, [utdelningssatser](/dividendrate),. resultat eller [bokfört värde](/bookvalue). Fundamentalt viktade index utgör ett riktmärke för passivt förvaltade fonder som erbjuds investerare som söker exponering mot aktier baserat på grundläggande egenskaper.

## Så fungerar fundamentalt viktade index

Fundamentalt viktade index växte från investerarnas intresse för passiv förvaltning. De blev vanligare 2004 efter att forskning om dem introducerades av Research Affiliates. Intresset för fundamentalt viktade index har fortsatt att öka i takt med att fler fondbolag har byggt skräddarsydda index som representerar specifika investeringsaspekter på investeringsmarknaden.

Passivt förvaltade fundamentalt viktade index är en del av en ny våg av trackerfonderbjudanden. Skräddarsydda [spårningsfonder är passivt förvaltade](/trackerfund) [indexfonder](/trackerfund) som går längre än vanliga indexerbjudanden, som försöker replikera skräddarsydda index konstruerade utifrån ett brett utbud av egenskaper.

Fundamentalt viktade index är några av de mest framträdande skräddarsydda indexen som används av [passivt förvaltade](/passivemanagement) trackerfonder. Fondbolag kommer ofta att skapa sina egna skräddarsydda grundindex för att bygga en replikerad portfölj runt det för emission till allmänheten som en strukturerad fond. Genom att använda skräddarsydda fundamentalt viktade index kan investeringsbolagen avsevärt minska kostnaderna och förbättra effektiviteten genom lägre transaktionskostnader och årlig ombalansering.

Förespråkare av dessa index hävdar att de kan erbjuda en högre potentiell avkastning baserat på aggregerade grundläggande mått på marknaden kontra börsvärde. Över hela branschen kan de konstrueras med hjälp av ett brett spektrum av grundläggande faktorer som historiskt har visat sig vara framgångsrika mått för att identifiera topppresterande investeringar över tid. Vanligtvis är investerare som tenderar att föredra fundamentalt viktade index mer ivriga investerare som söker denna viktade strategi.

## Exempel på ett fundamentalt viktat index: FTSE RAFI

Financial Times Stock Exchange (FTSE) i samarbete med Research Affiliates har många fundamentalt viktade index. Index viktas med hjälp av fundamentala faktorer som total kontantutdelning, fritt kassaflöde, total försäljning och bokfört eget kapitalvärde.

Invesco FTSE RAFI US 1000 ETF är en fond som förvaltas till ett FTSE RAFI-index. Fonden strävar efter att replikera innehaven och resultatet för [FTSE RAFI US 1000 Index](/ftse-rafi-us-1000-index).
