# Åldrandeschema
## Vad är ett åldrandeschema?

Ett åldrandeschema är en redovisningstabell som visar ett företags kundfordringar, sorterade efter deras förfallodatum. Ofta skapat av [bokföringsprogram](/accounting-software),. ett åldrande schema kan hjälpa ett företag att se om dess kunder betalar i tid. Det är en uppdelning av fordringar efter åldern på den utestående fakturan, tillsammans med kundens namn och förfallna belopp.

## Hur ett åldrandeschema fungerar

Ett åldrande schema kategoriserar ofta konton som aktuella (under 30 dagar), 1-30 dagar förfallna, 30-60 dagar förfallna, 60-90 dagar förfallna och mer än 90 dagar förfallna. Företag kan använda åldringsscheman för att se vilka räkningar som är försenade och vilka kunder de behöver skicka betalningspåminnelser till eller, om de ligger för långt efter, skicka till inkasso. Ett företag vill att så många av sina konton ska vara så aktuella som möjligt eftersom ju längre kontot är förfallna, desto mer sannolikt är det att det aldrig kommer att betalas, vilket leder till en förlust.

Ett företag kan uppleva ekonomiska problem om det har ett betydande antal förfallna konton. Det kan behöva låna pengar för att hålla sig flytande på grund av de obetalda kontona. Det kommer att påverka företagets resultat ytterligare eftersom det kommer att vara ansvarigt för att betala ränta på pengarna det lånar. Varje dag en betalning är försenad kommer att ha någon slags inverkan på ett företags finansiella ställning, och varje konto som är försenat multiplicerar det.

> Ju längre förfallodagen ett konto går desto mer tveksamt är det att betalningen kommer att tas emot. Åldrande scheman tillåter företag att hålla sig på toppen av A/R i hopp om att begränsa tveksamma konton.

## Fördelar med åldrande scheman

Åldrandescheman används ofta av chefer och analytiker för att bedöma ett företags operativa och finansiella resultat. De är särskilt användbara för hantering av rörelsekapital. Åldrandescheman kan hjälpa företag att förutsäga sitt [kassaflöde](/cashflow) genom att klassificera utestående skulder efter förfallodagen från tidigast till senast och genom att klassificera förväntade intäkter efter antalet dagar sedan fakturor skickades ut.

Kassaflöde är viktigt för ett företag eftersom många företag misslyckas på grund av negativt kassaflöde. Det är därför att spåra kassaflödet är en avgörande faktor för att upprätthålla en sund och framgångsrik verksamhet. Förutom deras interna användning kan åldringsscheman också användas av borgenärer för att utvärdera om de ska låna ut ett företag pengar.

Dessutom kan [revisorer](/auditor) använda åldringsscheman för att utvärdera värdet av ett företags fordringar. Om samma kunder upprepade gånger dyker upp som förfallna i ett schema för åldrande av kundfordringar, kan företaget behöva omvärdera om det ska fortsätta att göra affärer med dem. Ett åldrandeschema för kundfordringar kan också användas för att uppskatta dollarbeloppet eller procentandelen av [fordringar](/receivables) som troligen inte kan samlas in. Det kan tillåta ett företag att vara proaktiv istället för reaktiv.

Genom att veta hur stor andel av fordringarna som kan vara oindrivbara kan företaget leta efter lösningar på sina kassaflödesproblem innan problemet går utom kontroll. För vissa branscher, såsom detaljhandel eller tillverkning, kan åldrande scheman spela en betydande roll för att fastställa kreditstandarder. Om ett företag märker att det har ett konsekvent problem med ett stort antal förfallna konton, kan det titta på att höja sina standarder när det kommer till en kunds kreditvärdighet.

##Höjdpunkter

- Åldrandescheman är redovisningstabeller som företag använder för att se om betalningar görs eller ser mottagna i rätt tid.

- Dessa scheman kan anpassas för att inkludera vilken tidsram företaget vill spåra, men inkluderar vanligtvis under 30 dagar, 1-30 dagar förfallna, 30-60 dagar förfallna och mer än 90 dagar förfallna.

– Att använda åldrandescheman kan hjälpa företag att upptäcka kassaflödesproblem innan de blir en ännu större fråga.

– Åldrandescheman kan hjälpa företag att upptäcka problem med sina kreditpolicyer.
