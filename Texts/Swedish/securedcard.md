# Säkert kreditkort
## Vad är ett säkert kreditkort?

Ett säkert kreditkort är ett kort som backas upp av ett sparkonto. Pengarna på kontot fungerar som en säkerhet för kreditkortet, och beloppet hjälper till att bestämma utgiftsgränsen för kortet. Låntagare kan inte komma åt pengarna på sparkontot medan de använder kortet.

## Djupare definition

Vanligtvis har ett säkert kreditkort en gräns som motsvarar 50 procent till 100 procent av insättningsbeloppet. Det betyder att om sparkontot som säkrar kreditkortet har $1 000 på sig, är kreditgränsen någonstans mellan $500 och $1 000.

När du har upprättat en positiv betalningshistorik kan den utfärdande banken förlänga kortets gräns utöver din insättning. Du använder ett säkert kreditkort precis som du gör ett vanligt, osäkrat kreditkort. Ett säkert kreditkort kan användas på företag som accepterar större kreditkort.

Om du har bra kredit, bör du inte ansöka om ett säkert kreditkort eftersom du hittar bättre alternativ någon annanstans.

Men konsumenter med dålig eller ingen kredit är bra kandidater för säkra kort. Säkerheten gör att korten har lägre kreditkrav än osäkrade kreditkort.

Även om säkrade kreditkort är användbara verktyg för låntagare som vill bygga upp eller bygga om krediter, måste konsumenterna göra sin forskning och läsa det finstilta innan de ansöker.

Säkrade kreditkort kan komma med avgifter och höga räntor. Nästan alla har årliga avgifter, och vissa har månatliga underhållsavgifter. En ny kortinnehavare kan behöva teckna en kostsam försäkring innan kortet används.

Räntekostnaderna gör ett säkert kreditkort till ett dåligt alternativ för någon som planerar att ha ett saldo. Betala av saldot varje månad för att minimera dina räntekostnader.

## Exempel på ett säkert kreditkort

Om du har slagit upp din kredit kan ett säkert kort hjälpa till att läka det.

Helst bör du betala av kortet varje månad, men om du måste ha ett saldo, begränsa det till högst 20 procent av ditt korts gräns. Det betyder att om ditt kort har en gräns på $500, håll saldot under $100 för att säkerställa att kortet hjälper din kreditpoäng så mycket som möjligt.

Innan du ansöker om ett säkert kreditkort, fråga det utfärdande företaget om det kommer att rapportera din betalningshistorik till de tre stora kreditbyråerna. Om svaret är "nej", fortsätt shoppa runt. Ett säkert kreditkort hjälper inte din kredit om inte emittenten rapporterar din information.

Betala alltid kreditkortet före förfallodagen. Detta hjälper inte bara till att skapa en bra betalningshistorik, det minimerar också avgifter och räntor i samband med sena betalningar.

Med tiden kommer din kredit att förbättras med ansvarsfull användning av det säkrade kreditkortet, till den grad att du kan ansöka om ett osäkrat kort.

De flesta större banker erbjuder säkra kreditkort. Om du har problem med att bli godkänd för ett kort eller att hitta ett med förmånliga villkor, kolla din lokala kreditförening. De flesta kreditföreningar erbjuder sina medlemmar säkrade kreditkort med rimliga avgifter och räntor.

##Höjdpunkter

- Bortsett från insättningen fungerar säkrade kreditkort som vilket kreditkort som helst.

- Säkrade kreditkort har vanligtvis lägre kreditgränser och högre avgifter än osäkra kreditkort.

- Konsumenter skaffar vanligtvis säkrade kreditkort för att förbättra sina kreditvärden eller upprätta en kredithistorik.

##FAQ

### Hur kan jag ändra ett säkert kreditkort till ett osäkrat kort?

Om du regelbundet uppfyller dina betalningar på ditt säkrade kreditkort, bör din kreditvärdering gradvis förbättras. Du kan kontrollera din kreditpoäng online med jämna mellanrum så att du vet när du har goda chanser att bli godkänd för ett vanligt kreditkort. Ibland kommer din kortutgivare att göra detta åt dig och automatiskt konvertera ditt säkra kort till ett osäkrat kort (och vanligtvis höja din kreditgräns också). Då bör du räkna med att göra regelbundna betalningar i några månader innan du godkänns för ett osäkert kort. kreditkort.

### Skapar säkrade kreditkort kredit?

Dom kan. Säkrade kreditkort riktar sig till personer med begränsad eller dålig kredithistorik och kan vara ett bra sätt att förbättra din kreditvärdering. Genom att göra regelbundna, pålitliga betalningar på ett säkert kreditkort kan du förbättra din kreditvärdighet och få tillgång till billigare kreditformer.

### Hur skiljer sig ett säkert kreditkort från ett osäkrat kreditkort?

Med ett vanligt, osäkrat kreditkort krävs ingen insättning. Med ett säkert kreditkort täcks pengarna som du lånar från din kortutgivare av en insättning. Denna insättning fungerar som säkerhet på kontot och ger kortutgivaren säkerhet ifall kortinnehavaren inte kan göra betalningar. Detta minskar risken för kortutgivaren, vilket i sin tur innebär att säkrade kort är tillgängliga för låntagare med dålig eller begränsad kredithistorik.

### Hur stänger jag ett säkert kreditkort?

Du kan i allmänhet stänga ett säkert kreditkort när som helst, så länge du är uppdaterad med dina betalningar och det inte finns något utestående saldo på kortet. För att göra detta, kontakta din kortutgivare När du stänger ett säkert kreditkort bör du få tillbaka din insättning, minus eventuella avgifter som din kortutgivare tar ut. Alternativt kan din kortutgivare erbjuda dig att konvertera ditt säkra kreditkort till ett standardkort om du regelbundet uppfyller dina betalningar.
