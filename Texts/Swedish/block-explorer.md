# block explorer
Kort sagt är en blockutforskare ett verktyg som tillhandahåller detaljerad analys om ett [blockkedjenätverk](/blockchain) sedan dess första dag i [genesis-blocket](/genesis-block). Vi kan säga att en blockutforskare fungerar som en sökmotor och webbläsare där användare kan hitta information om enskilda block, offentliga adresser och transaktioner associerade med en specifik kryptovaluta.

Vissa blockutforskare tillhandahåller även realtidsstatistik och marknadsdiagram, såväl som data om gruvpooler, väntande transaktioner, nätverkshashhastighet, rik lista, blockvaliderare, [föräldralösa block](/orphan-block-cryptocurrency),. hårda gafflar och mycket mer.

När det gäller väntande transaktioner kan blockutforskare vara användbara för användare som väntar på blockbekräftelser. Till exempel ger många börser sina användare transaktions-ID för deras insättnings- eller uttagsbegäran så att de kan spåra rörelsen av sina medel i realtid.

Beroende på typen av blockchain kan blockutforskare också fungera som ett allmänt informationsnav. Till exempel finns det tusentals [ERC-20-](/erc-20) tokens som körs ovanpå Ethereum-blockkedjan, och användare kan hitta data om dem genom att kontrollera sina [smarta kontrakt](/smart-contract) på Etherscan eller annan Ethereum-blockutforskare.

Blockutforskare är viktiga i processen att övervaka det aktuella tillståndet för ett kryptovalutanätverk. När det gäller Binance Chain kan användare kontrollera den aktuella statusen för BNB på [Binance Chain Explorer](/binance-coin-bnb),. inklusive myntförbränningstransaktioner och det aktuella [totala utbudet](/total-supply).

Även om vi kan ha flera blockutforskare på en enda webbplats, är varje utforskare relaterad till en viss blockkedja. Ändå kan det finnas mer än en blockutforskare för samma blockkedja (olika organisationer som tillhandahåller en liknande tjänst). Vissa företag erbjuder också kryptoplånbokstjänster tillsammans med blockutforskare. Förutom det tillhandahåller många webbplatser ett [API](/application-programming-interface) för utvecklare att använda blockchain-data på andra sätt.
