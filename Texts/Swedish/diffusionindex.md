# Diffusionsindex
## Vad är ett diffusionsindex?

Ett diffusionsindex är ett mått som används för att visa förändringens spridning i förhållande till en referenspunkt, och en serie data kan visa om det finns uppåtgående eller nedåtgående trender.

Diffusionsindex används ofta för att lyfta fram resultat från en undersökning. Svaren är i form av värden, och beroende på dessa värden anges ett bastal. Många indikatorer använder en baslinje på 0, 50 eller 100. En serie data visar hur långt resultaten är från baslinjen. Medan en datapunkt inte långt från basnumret skulle indikera liten förändring, kan en datapunkt långt från basen antyda en signifikant förändring.

Inom ekonomi används diffusionsindex vid formuleringen av indikatorer, såsom [inköpschefsindex](/pmi) och [Richmond Manufacturing Index](/richmond-manufacturing-index). Till skillnad från ekonomiska indikatorer, där uppgifterna baseras på priser eller enheter, baseras de som använder ett spridningsindex på undersökningar. Och beroende på hur enkätfrågorna är utformade kan diffusionsindexet användas som ett verktyg för att förutsäga, på samma sätt som PMI anses vara en ledande ekonomisk indikator.

## Hur man beräknar ett diffusionsindex

Det finns olika sätt att beräkna ett diffusionsindex, och formeln kan vara antingen i lång eller kort form.

###Federal Reserve Formel

Federal [Reserve](/federal-reserve) använder den grundläggande formeln för att beräkna ett diffusionsindex. Denna formel används för att beräkna index för många indikatorer bland Feds 12 regionala banker, inklusive undersökningar från Richmond och Kansas City Feds.

Diffusionsindex = [100 * (I − D)] / (I + N + D)

I = Antal respondenter som rapporterar ökningar

N = Antal svarande som rapporterar ingen förändring

D = Antal respondenter som rapporterar minskningar

### IHS Markit Formula

IHS Markit, en forskargrupp som publicerar inköpschefsindex på USA och andra länder, har sin egen formel som skiljer sig från Feds.

IHS Markits diffusionsindex = (% rapporterar 'upp') + [0,5 * (% rapporterar 'detsamma')] + [0,0 * (% rapporterar 'ner')]

Det kan uttryckas i kort form som:

Index = Procent av positiva svar + 0,5 (Procent av neutrala svar)

## Exempel på ett diffusionsindex

Som ett exempel, för en provundersökningsfråga om huruvida ekonomin kommer att gå in i [recession](/recession) någon gång snart, visade resultaten att 70 procent av de tillfrågade svarade att det var en hög sannolikhet, 20 procent svarade att det inte fanns någon sannolikhet och 10 procent svarade att det fanns en stor sannolikhet. var en låg sannolikhet.

Om resultaten matas in i Fed:s formel för diffusionsindex, skulle det beräknas som sådant:

[100 * (0,70 − 0,10)] / (0,70 + 0,20 + 0,10) = 60 / 1 = 60

Om det beräknas med IHS Markits formel skulle svaret vara:

0,70 + (0,5 * 0,20) + (0,0 * 0,10) = 0,70 + 0,10 + 0,00 = 0,80

Resultaten skiljer sig för båda beroende på vilken formel som används. Att tilldela en baslinje på 0 eller 100 skulle vara idealiskt i det här exemplet eftersom svaret kan manipuleras som tvåsiffriga.

## Hur man tolkar ett diffusionsindex

En serie data kan visa momentum och avslöja om det finns uppåtgående och nedåtgående trender. Data som ligger över baslinjen skulle indikera expansion, medan data nedan skulle indikera kontraktion. Data nära eller vid baslinjen skulle innebära liten eller ingen förändring.

Ett exempeldiagram nedan illustrerar en dataserie som visar expansion och kontraktion. Ju längre data är från baslinjen, desto större är sannolikheten för antingen expansion eller kontraktion över en given period.

## Vilka är begränsningarna för ett diffusionsindex?

En indexspridning kan representera en ensidig vy. Så att jämföra diffusionsindexet med andra datauppsättningar kan ge ytterligare bekräftelse eller motbevisa en slutsats som gjorts med hjälp av ett diffusionsindex. Användning av andra mätvärden, såsom [korrelationskoefficienten](/correlationcoefficient),. kan visa om ett diffusionsindex är ett tillförlitligt prediktionsverktyg.

##Höjdpunkter

- Ett diffusionsindex avser den vanliga tendensen inom en grupp av siffror eller statistik.

- Diffusionsindexet kan användas för att upptäcka divergenser, vilket kan signalera styrka eller svaghet som inte syns genom att titta på aktieindexet på egen hand.

- Ett sjunkande diffusionsindex visar att det är färre aktier som går uppåt, vilket tyder på att köptrycket försvagas och/eller att säljtrycket ökar på aktieindexet.

– På aktiemarknaden innebär ett stigande diffusionsindex att det finns ett ökande antal aktier som går uppåt, vilket är positivt för aktieindex som går uppåt.
