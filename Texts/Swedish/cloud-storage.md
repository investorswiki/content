# Molnlagring
## Vad är molnlagring?

Molnlagring är ett sätt för företag och konsumenter att spara data säkert online så att den kan nås när som helst från vilken plats som helst och enkelt delas med dem som har tillstånd. Molnlagring erbjuder också ett sätt att säkerhetskopiera data för att underlätta återställning utanför platsen.

Idag har individer tillgång till flera gratis molntjänster som Google Drive, Dropbox och Box, som alla kommer med uppgraderade prenumerationspaket som erbjuder större lagringsstorlekar och ytterligare molntjänster.

## Molnlagring förklaras

Molnlagring erbjuder ett enkelt sätt att lagra och/eller flytta data på ett säkert och säkert sätt. Det tillåter individer och företag att hålla sina filer lagrade hos molntjänstleverantören för åtkomst på begäran på vilken som helst av deras enheter. Molnlagring kan också användas för att arkivera data som kräver långtidslagring men som inte behöver nås ofta, till exempel vissa finansiella poster. Allt oftare används filer som lagras "i molnet" för gruppsamarbete.

Molnlagring fungerar genom att en klientdator, surfplatta eller smartphone kan skicka och hämta filer online till och från en fjärrdataserver. Samma data lagras vanligtvis på mer än en server samtidigt så att klienter alltid kan komma åt sina data även om en server är nere eller förlorar data. En bärbar datorägare kan till exempel lagra personliga foton både på sin hårddisk och i molnet om den bärbara datorn skulle bli stulen.

Ett molnlagringssystem kan specialisera sig på att lagra en viss typ av data som digitala foton eller musikfiler, eller kan tillhandahålla allmän lagring av alla typer av data som foton, ljudfiler, textdokument, presentationer och kalkylblad.

Molnlagring tros ha uppfunnits av datavetaren Dr. Joseph Carl Robnett Licklider på 1960-talet. Cirka två decennier senare började CompuServe erbjuda sina kunder små mängder diskutrymme för att lagra några av deras filer. I mitten av 1990-talet lanserade AT&T den första helt webbaserade lagringstjänsten för personlig och affärskommunikation. Sedan dess har en rad olika tjänster fått draghjälp. Några av de mest populära molnlagringsleverantörerna är Apple (iCloud), Amazon (Amazon Web Services), Dropbox och Google.

## Hur molnlagring gynnar företag

Molnlagring hjälper företag med stora datalagringsbehov att spara en betydande mängd utrymme och pengar genom att eliminera behovet av datalagringsinfrastruktur i affärslokalerna. Molnlagringsleverantören äger och underhåller all nödvändig hårdvara och mjukvara så att molnanvändarna inte behöver göra det. Att köpa pågående molnlagring kan kosta mer i längden, men det kan vara betydligt billigare i förväg. Vidare kan företag nästan omedelbart skala upp eller ner hur mycket molnlagring de har tillgång till när deras lagringsbehov förändras.

Molnet gör det också möjligt för anställda att samarbeta med kollegor – och arbeta på distans och utanför ordinarie kontorstid – samtidigt som det underlättar smidigt dokumentsamarbete genom att ge behöriga medarbetare enkel åtkomst till den mest uppdaterade versionen av en fil. På det personliga planet tillåter molnlagring mobil data och möjliggör digitalt liv på det holistiska sätt vi lever det idag. Utan molnet skulle smarta telefoner inte kunna vara gränssnittet för så mycket data (foton, dokument, information på språng). Att använda molnet för att lagra filer kan också ha en positiv effekt på miljön eftersom det minskar energiförbrukningen

## Cloud Storage Security

Det är så mycket uppmärksamhet kring molnlagring idag i den digitala eran eftersom så mycket av våra känsliga personuppgifter lagras i molnet oavsett om vi frivilligt lagrar dem där eller om ett företag vi gör affärer med bestämmer sig för att lagra dem där. Som ett resultat har [molnsäkerhet](/cloud-security) blivit ett stort problem. Användare undrar om deras information är säker, och ökande [dataintrång](/data-breach) har visat att det ibland inte är det. Användare är också oroade över om den data de har lagrat i molnet kommer att vara tillgänglig när de behöver den.

Även om molnlagring kan tyckas sårbar på grund av förekomsten av [hackning ,](/cybersecurity) [har](/cybersecurity) alternativen, som lagring på plats, också säkerhetsbrister. Företag tillhandahållen molnlagring kan faktiskt förbättra säkerheten genom att ge anställda ett alternativ till att använda sina personliga konton för att säkerhetskopiera och överföra filer som de behöver komma åt utanför kontoret.

En bra leverantör av molnlagring kommer att ha dataredundans och lagra samma filer på flera fysiska platser så att den överlever mänskliga fel, utrustningsfel eller naturkatastrofer. En ansedd leverantör kommer också att lagra och överföra data säkert så att ingen kan komma åt den utan tillstånd. Vissa användare kan också kräva att data lagras på ett sådant sätt att de bara kan läsas men inte ändras; den här funktionen är också tillgänglig via molnlagring.

## Höjdpunkter

- Molnlagring har blivit allt mer populär bland individer som behöver större lagringsutrymme och för företag som söker en effektiv off-site-lösning för säkerhetskopiering av data.

- Molnlagring tillåter privatpersoner och företag att lagra och hämta datorfiler via en internetansluten enhet.

- På grund av molnlagrings ökande popularitet och användning har molnsäkerhet blivit ett stort problem för att skydda dataintegriteten, förhindra hackningsförsök och undvika fil- eller identitetsstöld.
