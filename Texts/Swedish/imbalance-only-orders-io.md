# Endast obalans (IO) order
## Vad är order om endast obalans (IO)?

Order endast med obalans (IO) är limitorder som ger likviditet under [öppnings-](/opening-cross) och stängningsrotationen på Nasdaq-börsen. Dessa kan kategoriseras som "obalans endast öppna order" eller "obalans endast stängningsorder".

IO-ordrar placeras alltså för att kompensera för en [orderobalans](/order-imbalance) i öppnings- eller stängningskrysset.

## Förstå order endast om obalans (IO).

Order med endast obalans (IO) kommer endast att utföras på öppningskorset eller stängningskorset beroende på vilken beställningstyp som placeras (öppning eller stängning). IO-order kan vara att köpa eller sälja. IO-köporder utförs endast vid eller över [budpriset](/bid) 9:30 eller 16:00 , medan IO-säljorder endast utförs till eller under erbjudande- eller [säljpriset](/ask).

Innan öppnings- och stängningskorsningar utförs, prissätts köp- eller sälj-IO-order till det bästa köp- respektive säljpriset på Nasdaq-boken. IO-order måste nödvändigtvis vara limitorder; IO-order [på marknaden](/marketorder) är inte tillåtna.

Beställningarna kompenserar [på-stäng-](/limitoncloseorder) och på-öppna beställningar. Eftersom IO-order endast är exekverbara under öppningskrysset eller stängningskorset, riskerar de inte att utföras innan marknaden öppnar eller stänger. På så sätt skiljer de sig från vanliga [limitordrar](/limitorder).

## Obalans Endast beställningar Timing och överväganden

Obalansrapporter publiceras vid vissa tillfällen på handelsdagen. Dessa rapporter ger uppdateringar om köp- och säljorder vid öppet eller vid stängning som redan har lagts på plats. En köpobalans indikerar ett överskott av köporder kopplade till ett visst pris jämfört med relevanta säljorder. I det här scenariot kan aktiekursen stiga tills köp- och säljorder balanserar ut.

IO (och andra) beställningar accepteras på Nasdaq från kl. 04.00. Obalansinformation släpps först för dagen kl. 9.28, strax före öppningskrysset kl. 9.30.

Marknadsaktörer kan inte uppdatera eller annullera IO-order för öppningskrysset efter 9:28, eller uppdatera eller annullera IO-order för stängningskrysset efter 15:58. Men i båda fallen kan nya IO-order fortfarande läggas in efter dessa tidsfrister.

För det avslutande korset, sprider Nasdaq information om obalans mellan 15:55 och 16:00. Stängningsorder kan läggas in när som helst under dagen.

> Observera att IO-order kan sälja kort. Kortförsäljnings-IO-order och sälj-IO-order prissatta till eller under bästa köpkurs prissätts om till det bästa erbjudandepriset kl. 16.00. Under det avslutande korset, utförs kortförsäljnings-IO-order med en downtick endast om slutkursen är bättre än det bästa budet; dessa beställningar kommer inte att delta om slutkursen är på eller under det bästa budet.

>

## Exempel på användning av en obalansorder på den avslutande (eller öppnande) auktionen

Antag att en daytrader eller någon som redan äger Apple Inc. är intresserad av att sälja på det avslutande krysset.

Klockan 15:55 kommer obalansinformationen att spridas för Nasdaq-aktier, inklusive AAPL. Obalansinformationen kan påverka hur aktien handlas under de senaste fem minuterna av handeln, eftersom priset kan flytta sig för att kompensera för obalansen. Till exempel, om det finns en stor köpobalans, kan handlare köpa aktien och förvänta sig att priset ska gå upp mot slutet på grund av köpobalansen.

Antag att det finns en köpobalans på två miljoner aktier. Priset kommer sannolikt att öka för att locka tillräckligt många säljare för att sälja två miljoner aktier och kompensera för obalansen. Även om detta inte alltid är fallet. Obalansen kan vända och bli en säljobalans när fler beställningar kommer in för korset. Andra marknadskrafter kan också driva aktien i en annan riktning än förväntat, eller så kanske aktien inte rör sig mycket alls.

Handlaren bestämmer sig för att ange en säljorder för endast obalans (IO) för 100 aktier med ett gränspris på $220. Aktien handlas för närvarande på $220 med ungefär en minut kvar (15:59) tills marknaden stänger.

IO-ordern är en limitorder, så den kommer inte att [fyllas](/fill) om inte aktiekursen är över gränsen vid tidpunkten för utförande. Om priset är under 220 USD kommer säljordern inte att verkställas. Om slutkursen är över 220 $, kommer ordern att utföras till korspriset.

## Höjdpunkter

– IO-order kan användas för att sälja eller köpa och måste ha ett limitpris kopplat till sig. De kan inte vara marknads-IO-ordrar.

- Information om obalans på Nasdaq sprids mellan 9:28 och 9:30 för öppettider och 15:55 och 16:00 för stängning.

- Order med endast obalans (IO) kan användas vid stängning eller öppning av korsningar för att kompensera för obalanser som skapats av beställningar på öppna eller stängda.
