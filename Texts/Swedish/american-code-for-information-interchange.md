# American Code for Information Interchange (ASCII)
## Vad är den amerikanska koden för informationsutbyte (ASCII)?

American Code for Information Interchange, vanligen kallad ASCII, är ett datorspråk för text som blev standarden för att överföra information mellan datorer 1963.

ASCII-systemet översätter alfanumeriska tecken och symboler till en datorläsbar kod. Den ursprungliga ASCII var endast engelska. Det finns nu två typer av ASCII-koder; standardkoden som använder ett sjubitars kodningssystem och en utökad kod som använder ett åttabitarssystem.

Inom internetkommunikation har ASCII gradvis ersatts av Unicode-standarden, som kan användas med alla språk och är kompatibel med ASCII. Men ASCII används fortfarande för att skriva in text på persondatorer och elektroniska enheter.

Det uttalas ASK-y.

## Förstå den amerikanska koden för informationsutbyte (ASCII)

Den tidigaste iterationen av ASCII kan hittas i en telegrafisk kod som används av Bells datatjänster för att leverera meddelanden till teleskrivare.

ASCII var det första större teckenkodningssystemet för databehandling och antogs som standard för datoranvändning 1963 och för internetöverföring 1969. Det användes under många år i datorsystem för orderingång som användes av Wall Street-handlare och mäklare.

Den har gradvis ersatts som standard för internet av Unicode Worldwide Character Standard (Unicode). Men eftersom de första 128 tecknen i Unicode är desamma som de i ASCII, kan de två systemen sägas samexistera.

> Du kan ange en ASCII-symbol i ett dokument med vilken Windows-kompatibel dator som helst genom att hålla ned ALT-tangenten medan du skriver teckenkodnumret. Om du till exempel håller ned ALT-tangenten medan du skriver 156 får du £, det brittiska pundtecknet.

### Vad finns i ASCII

Innan ASCII antogs som standard använde varje datortillverkare sin egen kod, och vissa företag använde mer än en för olika produkter. Datorer kunde inte kommunicera med varandra.

Även om ASCII antogs som standard av American National Standards 1963, implementerades den inte helt ens i USA förrän 1968, när president Lyndon B. Johnson gav mandat att den skulle antas av den federala regeringen för att säkerställa att statliga datorer och telekommunikationsindustrin datorer kan kommunicera med varandra.

ASCII innehöll ursprungligen 128 alfabetiska tecken, siffror och symboler och var baserad på det engelska språket. Utökade formulär lades till under 1970-talet för att rymma andra språk.

Snart användes olika versioner av ASCII. Det inkluderade så småningom 255 tecken. Denna version är känd som Extended ASCII.

Idag kan användare ange ASCII- eller Unicode-symboler som en copyright-symbol i dokument genom att trycka på alt-tangenten och ange teckenkoden på det numeriska tangentbordet med NUM-låsknappen på.

##FAQ

### Vad är ASCII vs. Unicode?

Unicode kan ses som en universell version av ASCII.ASCII är trots allt American Code for Information Interchange, och dess första iteration inkluderade det engelskspråkiga alfabetet och symboler som används i språket. Senare versioner inkluderade symboler som används på många andra språk. Unicode, å andra sidan, börjar med samma 128 symboler som används i ASCII men nuvarande versioner innehåller 144 697 tecken.

### Vad är ASCII?

American Standard Code for Information Interchange, eller ASCII, är ett teckenkodningsformat för elektronisk överföring av text. Varje tecken representeras av ett unikt nummer. Den första versionen av ASCII innehöll endast 128 tecken, representerande bokstäverna i alfabetet, versaler och gemener, plus ett antal vanliga symboler som kommatecken. Senare versioner utökade ASCII till 255 tecken, inklusive ytterligare symboler som t.ex. Brittiskt pundsymbol (£) och det upp och nedvända frågetecknet som används i spansk text (¿).

### Vad är ASCII-tecken?

I den utökade ASCII-tabellen är teckenkoderna 0 till 31 kontrollkoder som början av text och backsteg. Teckenkoderna 32 till 127 är oftast bokstäver i alfabetet (versaler och gemener) och symboler som asterisken och dollartecknet. koderna 128 till 255 finns i flera versioner, var och en av dem återspeglar tillägg till den ursprungliga engelska ASCII som ett inverterat utropstecken, yen-tecknet och det georgiska kommatecken.
