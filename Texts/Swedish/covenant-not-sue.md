# Förbund att inte stämma
## Vad är ett förbund att inte stämma?

Ett förbund att inte stämma är ett juridiskt avtal som ålägger en part som kan begära skadestånd att avstå från att stämma den part som den har skäl mot. Ett [förbund](/covenant) att inte stämma kan indikera att den potentiella käranden inte kommer att stämma i all framtid, eller kan indikera att käranden kan skjuta upp en rättegång under en bestämd tidsperiod.

## Förstå ett förbund att inte stämma

Ett förbund att inte stämma ålägger en part som skulle kunna inleda en rättsprocess att inte göra det. Förbundet upprättas uttryckligen mellan två parter, och varje [tredje part](/third-party) som vill göra ett anspråk har lagligen rätt att göra det.

Covenants att inte stämma används för att lösa specifika juridiska frågor utanför domstolssystemet. Parter kan ingå den här typen av avtal för att förhindra en utdragen, dyr rättegång. I utbyte mot avtalet kan den part som skulle kunna begära skadestånd ges ersättning eller ges försäkringar om att den andra parten i avtalet kommer att genomföra en specifik åtgärd.

## Exempel på ett förbund att inte stämma

Föreställ dig till exempel att en statlig regerings miljöreglerande organ bestämmer att ett tillverkningsföretag inte hanterar farliga avfallsprodukter korrekt. Man skulle kunna inleda en stämningsansökan och begära skadestånd från tillverkaren, men vill istället tvinga tillverkaren att sanera det farliga materialet och se till att man gör sig av med material på ett korrekt sätt i framtiden.

Miljötillsynsmyndigheten kan erbjuda ett avtal om att inte stämma tillverkaren men förbehåller sig rätten att stämma om tillverkaren inte ändrar sina rutiner för avfallshantering. Detta är ett villkorligt förbund att inte stämma och utfärdas inte för all framtid.

[Patentinnehavare](/patent) kan också gå med på ett avtal om att inte stämma med företag som de licensierar sina patent till. Patentinnehavaren kan gå med på att inte stämma licenstagaren om en tredje part använder patentet utan tillstånd, men kan förbehålla sig rätten att själv stämma vilken tredje part som helst.

## Förbund att inte stämma vs. frigörande av ansvar

Ett förbund om att inte stämma är helt annorlunda än en frigörelse av ansvar. Ett frisläppande är ett [avstående eller](/waiver) [avstående](/waiver) från en känd rättighet. Ett frisläppande av ansvar kommer att avstå från eller förstöra den skadelidandes talan.

Ett förbund att inte stämma, å andra sidan, är inte ett avstående från en känd rättighet; ingenting överges eller förstörs. Ett förbund att inte väcka talan bevarar förekomsten av talan, men sätter avtalsenliga begränsningar på den skadelidandes rätt att väcka talan.

## Höjdpunkter

- Ett avtal om att inte väcka talan bevarar förekomsten av talan men sätter avtalsenliga begränsningar på den skadelidandes rätt att väcka talan.

– Ett förbund att inte stämma är ett juridiskt avtal där den som söker skadestånd går med på att inte stämma den part som den har skäl mot.

- Covenants att inte stämma används för att lösa specifika juridiska frågor utanför domstolssystemet.
