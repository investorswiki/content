# Förskottsförlustförsäkring (ALOP).
## Vad är förskottsförlustförsäkring (ALOP)?

Förskottsförlustförsäkring (ALOP) ger täckning för ekonomiska förluster på grund av förseningar i bygg- och infrastrukturprojekt.

ALOP kommer också att ge en [utbetalning](/payout) om företag står inför högre kostnader eller förlorade vinster när ett projekt tar längre tid än förväntat att slutföra. ALOP kallas ofta också för försenat färdigställande eller försenad startförsäkring (DSU).

## Förstå Advance Loss Of Profit (ALOP) försäkring

Stora byggprojekt köper förskottsförlustförsäkring eftersom de står inför flera risker som kan resultera i försenat slutförande av projektet. En hård vinter kan till exempel försena starten av ett projekt och därmed slutdatumet, eller kanske är byggarbetsplatsen mer instabil än vad ingenjörer ursprungligen uppskattade. De möjliga orsakerna till förseningar är många och ofta oväntade.

Sådana förseningar kan allvarligt påverka ekonomin för företag som förlitar sig på att ett byggprojekt slutförs i tid. Dessutom kan företag som använder sig av [skuldfinansiering](/debtfinancing) få svårt att betala tillbaka skulder som uppstått för att hyra eller köpa byggutrustning.

Företag som planerar att flytta in i en ny byggnad kan förlora pengar eftersom de inte kan öppna för affärer. Förseningar av vissa projekt, såsom hamnar, flygplatser, broar och tunnlar, kan påverka många företag negativt över ett stort geografiskt område.

Förskottsförlustförsäkring ger en skydd mot förluster i samband med dessa typer av risker, och de företag som köper ALOP-täckning kan spela en mängd olika roller i ett byggprojekt. [Investerare](/investor) i projektet kan köpa en ALOP-försäkring för att täcka kostnaden för att inte kunna tjäna hyror från byggnadshyresgäster.

Byggnadsentreprenörer kan köpa försäkringen för att täcka kostnaden för att behöva hyra byggutrustning och betala anställda för längre tid än förväntat. Företag som hyr ut utrustning som används i bygget kan också använda försäkringen för att täcka kostnaderna för att inte kunna hyra utrustningen för andra projekt.

## Förskottsförlustförsäkring och bruttovinst

Förskottsförlustförsäkring täcker endast den faktiska förlusten av [bruttovinst](/grossprofit) som härrör från ett försenat projekt. De typer av händelser som utlöser täckning beskrivs på försäkringsspråket, men det kanske inte täcker alla händelsetyper. Problem kan uppstå från oklarheter i definitionen av bruttovinst.

Mäklare och försäkringsgivare bör testa sin föreslagna definition av bruttovinst genom att köra olika scenarier innan försäkringen tecknas. Om du gör det säkerställer du att det återspeglar deras avsikter i händelse av en förlust. Tid som spenderas på att få dessa poäng rätt kommer att hjälpa till att minimera missförstånd senare och säkerställa att förväntningarna uppfylls när en förlust inträffar.
