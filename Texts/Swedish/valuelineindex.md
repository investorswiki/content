# Value Line Composite Index
## Vad är Value Line Composite Index?

Value Line Composite Index är ett aktieindex som innehåller cirka 1 700 företag från [NYSE](/nyse),. [American Stock Exchange](/amex),. Nasdaq, Toronto och marknader över disk. Value Line Composite Index har två former: Value Line Geometric Composite Index (det ursprungliga lika viktade indexet) och Value Line Arithmetic Composite Index (ett index som speglar förändringar om en portfölj innehar lika stora mängder aktier.) Dessa index publiceras vanligtvis i Value Line Investment Survey, skapad av Arnold Bernhard, grundaren, och VD för Value Line Inc

## Förstå Value Line Composite Index

"Värdelinjen" där indexet får sin namne hänvisar till en multipel av kassaflöde som Bernhard skulle lägga över ett prisdiagram för att normalisera värdet på olika företag. Value Line är ett av de mest respekterade investeringsanalysföretagen. Dess prestationsrekord har varit extremt starkt. Faktum är att företagets modellportföljer i allmänhet har slagit marknaden på [lång sikt](/longrun).

Value Line Composite Index är sammansatt av samma företag som The Value Line Investment Survey, exklusive slutna fonder .

Antalet företag i Value Line Index varierar baserat på faktorer inklusive tillägg eller avnotering av företagen på själva börserna, fusioner, förvärv, konkurser och de täckningsbeslut som Value Line tagit för Value Line Index. Value Lines beslut om vilka bolag som ska inkluderas tas i avsikt att skapa en bred representation av den nordamerikanska aktiemarknaden. Dessutom kan antalet företag som är noterade på en given börs variera, eftersom ett företag kan flytta från en börs till en annan eller läggas till eller avnoteras. Avnotering eller förflyttning av företag på börserna är dock inte faktorer i Value Line Index-metoden, oavsett om de geometriska eller aritmetiska beräkningarna används.

## Värdelinjen Geometric Composite Index

Detta är det ursprungliga indexet, som introducerades den 30 juni 1961. Det är ett lika viktat index som använder ett geometriskt medelvärde. Den dagliga prisförändringen för Value Line Geometric Composite Index hittas genom att multiplicera förhållandet mellan varje akties stängningskurs och dess tidigare stängningskurs och höja resultatet till det ömsesidiga av det totala antalet aktier .

## Värdelinjen Arithmetic Composite Index

Detta index etablerades den 1 februari 1988, med hjälp av det aritmetiska medelvärdet för att närmare efterlikna förändringen i indexet om du hade en portfölj med aktier i lika stora mängder. Den dagliga prisförändringen för Value Line Arithmetic Composite Index beräknas genom att lägga till den dagliga procentuella förändringen av alla aktier och sedan dividera med det totala antalet aktier

## Höjdpunkter

- Det finns två former för indexet - Value Line Geometric Composite Index och Value Line Arithmetic Composite Index.

- Det geometriska sammansatta indexet är likavikt, använder ett geometriskt medelvärde och har en daglig förändring närmast medianaktiekursförändringen.

- Arithmetic Composite Index använder ett aritmetiskt medelvärde, där den dagliga förändringen i indexet återspeglar en portfölj som består av aktier i lika stora mängder .

- Value Line Composite-indexet innehåller en blandning av cirka 1 700 aktier från de stora nordamerikanska marknadsindexen .
