# Tillgängliga Seat Miles (ASM)
## Vad är tillgängligt Seat Miles (ASM)

Available seat miles (ASM) är ett mått på ett flygplans lastkapacitet som är tillgänglig för att generera intäkter. Tillgängliga sätemil hänvisar till hur många sätesmil som faktiskt finns att köpa på ett flygbolag. Seat miles beräknas genom att multiplicera antalet miles som ett givet flygplan kommer att flyga med antalet tillgängliga platser för en given flygning.

## Förstå tillgängliga Seat Miles (ASM)

ASM är helt enkelt ett mått på flygets intäktsgenererande förmåga baserat på trafik. För [investerare](/investor) som analyserar flygbolag är ASM ett mycket viktigt mått för att avgöra vilka flygbolag som är bäst på att generera [intäkter](/revenue) från tillgången på säten till kunderna. Om alla platser på planet inte säljs, fungerar flygbolagets ASM under kapaciteten.

> Långa tillfällen av otillgängliga platser på ett flygbolag kan kosta ett flygbolag miljontals dollar.

>

## Hur tillgänglig Seat Miles-statistik används

Även känd som tillgängliga sittkilometer på vissa marknader, används detta mått av flygbolag och statistikhållare på grund av variationerna i flygplan och deras säteskonfigurationer. Mätvärdet kan inte bara mäta enskilda flygbolags prestanda utan den kommersiella flygbranschen som helhet. Bureau of Transportation Statistics, till exempel, håller ett löpande månatligt och årligt register över inhemska, internationella och totala tillgängliga sittplatser för alla flygbolag och flygplatser.

ASM i sig kanske inte ger den fullständiga bilden av ett lufttrafikföretags ekonomiska resultat. Även om måttet representerar flygplanens kapacitet och beläggning, är [intäktspassagerarmil](/revenue-passenger-mile-rpm) och [inkomst per tillgängliga platsmil](/revenue-per-available-seat-mile-rasm) andra sätt att beräkna pengarna som genereras av flygningar. Detta kan jämföras med kostnaden per tillgänglig platsmil för att fastställa lönsamheten för varje flygning.

Inte varje plats på ett plan som är upptagen genererar intäkter för flygbolaget. Säten som används av ett flygbolags personal, till exempel för att få kontakt med en flygbesättning de kommer att arbeta med, är icke-inkomstpassagerare. Vissa typer av beredskapspassagerare kan också ockupera platser, men inte generera intäkter för flygbolaget. Flygbolag kan behöva transportera icke-inkomstpassagerare ut av nödvändigheten att flytta flygbesättningar runt eller för att uppfylla skyldigheter gentemot passagerare som har garanterats passage på flygbolaget. Hur flygbolag balanserar sådana icke-inkomstpassagerare mot sina platser som upptas av betalande kunder kan direkt påverka lönsamheten för varje flygning.

Prissättning av säten och flygplanets kapacitet är alla viktiga för att förstå ett flygbolags hälsa; dock behövs sådana element som kostnader för bränsle, underhåll och andra resurser för att ytterligare illustrera flygbolagets prestation.

##Höjdpunkter

- Platsmil beräknas genom att multiplicera de tillgängliga platserna för ett visst plan med antalet mil som planet kommer att flyga för en given flygning.

- Bärkapaciteten för ett flygplan som är tillgänglig för att skapa intäkter mäts som tillgängliga sätesmil.

– För investerare som analyserar flygbolag är måttet på ett flygbolags ASM en viktig datapunkt att ta hänsyn till.

- Det finns andra mätvärden som används för att beräkna de pengar som genereras av flygbolagen, inklusive intäktspassagerarmil och intäkt per tillgängliga platsmil.
