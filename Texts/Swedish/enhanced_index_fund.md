# Enhanced Index Fund (EIF)
## Vad är en Enhanced Index Fund (EIF)?

En förbättrad indexfond är en fond som strävar efter att förbättra avkastningen för ett index genom att använda aktiv förvaltning för att modifiera [innehavens vikter](/holdings) för ytterligare avkastning.

## Förstå Enhanced Index Funds (EIF)

Förbättrade indexfonder är begränsade till att investera i värdepapper från det index som de benchmarkar. Dessa strategier kan använda olika typer av investeringsanalyser. De kan använda kvalitativa och kvantitativa metoder för att identifiera och [övervikta](/overweight) topppresterande aktier. I vissa fall kan de också använda hävstång och derivat för att förbättra avkastningen.

## Förbättrade indexfondstrategier

Förbättrade indexfonder kan jämföras med vilket [index som helst](/index) i världen. De börjar med jämförelseindex som grund för investeringar. Genom kvalitativ och kvantitativ analys försöker portföljförvaltare identifiera topppresterande aktier, som sedan får större vikt i portföljen. Vissa fonder kan använda hävstångseffekter och derivat, vilket gör att fonderna kan öka vikten av aktier de väljer att köpa och minska vikten av aktier de väljer att sälja.

Med hävstång kan fonder ta djupare långa positioner i aktier de föredrar. Hävstång och derivat ger också fondförvaltaren möjlighet att shorta aktier som de tror kommer att trenda lägre. Fondförvaltare får inte heller ta någon position i en aktie, vilket ger den en vikt på 0 % i portföljen.

Teoretiskt sett borde förmågan att ta både långa och [korta positioner](/short) hjälpa en fond att generera ytterligare [alfa](/alpha) från potentiella aktievinster och -förluster. Användningen av hävstång och derivat kan dock lägga till ytterligare kostnader och öka risken för förluster. Därför förlitar sig de flesta förbättrade indexfonder på aktiva förvaltningsmetoder byggda kring ett specifikt indexuniversum utan användning av alternativa investeringar.

## Förbättrade indexfondinvesteringar

Medan förbättrade indexfonder använder samma indexuniversum för investeringar som passiva fonder, kommer deras investeringsegenskaper att vara mycket olika. Förbättrade indexfonder har vanligtvis högre förvaltningsavgifter och högre transaktionskostnader än jämförbara indexfonder. Riskerna kan också vara högre beroende på hävstång och använda [derivat](/derivative).

Investerare kommer att hitta förbättrade indexfonderbjudanden från investeringsförvaltare i hela branschen, med de flesta av de största kapitalförvaltarna som erbjuder ett brett utbud av förbättrade indexfondprodukter. Fidelity Large Cap Value Enhanced Index Fund ger ett exempel.

## Fidelity Large Cap Value Enhanced Index Fund

Fidelity Large Cap Value Enhanced Index Fund strävar efter att förbättra avkastningen för Russell 1000 Value Index. Fonden använder kvantitativ fundamental analys i sina investeringsbeslut och investerar minst 80 % av tillgångarna i Russell 1000 Value-aktier. Fonden förlitar sig inte väsentligt på hävstångseffekter eller derivat i sin förvaltningsstrategi. Fonden har stadigt överträffat Russell 1000 Value sedan starten med en avkastning på 7,08 % jämfört med 6,85 % för indexet.
