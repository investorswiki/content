# Aktivitetskvoter
## Vad är ett aktivitetsförhållande?

En aktivitetskvot är en typ av finansiellt mått som indikerar hur effektivt ett företag utnyttjar tillgångarna i sin balansräkning för att generera intäkter och kontanter. Vanligtvis kallade [effektivitetskvoter](/efficiencyratio),. aktivitetskvoter hjälper analytiker att bedöma hur ett företag hanterar [lagerhantering](/inventory-management),. vilket är nyckeln till dess operativa smidighet och övergripande finansiell hälsa.

## Förstå aktivitetsförhållanden

Aktivitetskvoter är mest användbara när de används för att jämföra två konkurrerande företag inom samma bransch för att avgöra hur ett visst företag står sig bland sina kollegor. Men aktivitetskvoter kan också användas för att spåra ett företags finanspolitiska framsteg över flera inspelningsperioder, för att upptäcka förändringar över tid. Dessa siffror kan kartläggas för att ge en framåtblickande bild av ett företags framtida prestation.

Aktivitetskvoterna kan delas upp i följande underkategorier:

### Omsättningskvot för kundfordringar

Kundfordringarnas [omsättningskvot](/receivableturnoverratio) avgör en enhets förmåga att samla in pengar från sina kunder. Den totala kreditförsäljningen divideras med den genomsnittliga kundreskontra för en viss period. En låg andel tyder på en brist i insamlingsprocessen.

### Varulager Omsättningskvot

Varulagrets omsättningskvot mäter hur ofta lagersaldot säljs under en [räkenskapsperiod](/accountingperiod). Kostnaden [för sålda varor](/cogs) divideras med det [genomsnittliga lagret](/average-inventory) för en viss period. Högre beräkningar tyder på att ett företag kan flytta sitt lager relativt enkelt.

### Total Assets Omsättningskvot

Den totala omsättningsgraden för tillgångar mäter hur effektivt ett företag använder sina tillgångar för att anbuda en försäljning. Total försäljning divideras med totala tillgångar för att dechiffrera hur skickligt ett företag använder sina tillgångar. Mindre nyckeltal kan tyda på att ett företag kämpar för att flytta sina produkter.

### Avkastning på eget kapital

Ett prestationsmått kallas avkastning på eget kapital (ROE) och mäter intäkterna från eget kapital. ROE beräknas genom att dividera nettovinsten med alla utestående aktieaktier på marknaden.

### Omsättningskvot för tillgångar

Ett mått som kallas [tillgångens omsättningsförhållande](/assetturnover) mäter mängden intäkter ett företag genererar per dollar av tillgångar. Denna siffra, som helt enkelt beräknas genom att dividera ett företags försäljning med dess totala tillgångar, avslöjar hur effektivt ett företag använder sina tillgångar för att generera försäljning.

## Aktivitetskvoter vs. Lönsamhetskvoter

Aktivitetskvoter och [lönsamhetskvoter](/profitabilityratios) är båda grundläggande analytiska verktyg som hjälper investerare att utvärdera olika aspekter av ett företags finanspolitiska styrka. Lönsamhetsförhållanden skildrar ett företags vinstgenerering, medan effektivitetskvoter mäter hur väl ett företag använder sina resurser för att generera dessa vinster. Lönsamhetskvoter kan hjälpa analytiker att jämföra ett företags vinster med dess industrikonkurrenter, samtidigt som de spårar samma företags framsteg under flera olika rapporteringsperioder.

##Höjdpunkter

– En aktivitetskvot beskriver i stora drag alla typer av finansiellt mått som hjälper investerare och forskningsanalytiker att mäta hur effektivt ett företag använder sina tillgångar för att generera intäkter och kontanter.

- Aktivitetskvoter kan delas in i omsättningskvoter för varulager, omsättningskvoter för totala tillgångar, mätningar av avkastning på eget kapital och ett spektrum av andra mätvärden.

- Aktivitetskvoter kan användas för att jämföra två olika verksamheter inom samma sektor, eller så kan de användas för att övervaka ett enskilt företags finanspolitiska hälsa över tid.
