# Tillgångsbaserad utlåning
## Vad är tillgångsbaserad utlåning?

Tillgångsbaserad utlåning är verksamheten att låna ut pengar i ett avtal som är säkrat med säkerhet. Ett [tillgångsbaserat lån](/assetbasedfinance) eller kredit kan säkras av inventarier, kundfordringar, utrustning eller annan egendom som ägs av låntagaren.

Den tillgångsbaserade lånebranschen tjänar företag, inte konsumenter. Det är också känt som tillgångsbaserad finansiering.

## Hur tillgångsbaserad utlåning fungerar

Många företag behöver ta lån eller skaffa krediter för att möta rutinmässiga [kassaflödeskrav](/cashflow). Till exempel kan ett företag få en kreditram för att se till att det kan täcka sina lönekostnader även om det är en kort försening i betalningar som det förväntar sig att få.

Om företaget som söker lånet inte kan visa tillräckligt med kassaflöde eller kassatillgångar för att täcka ett lån, kan långivaren erbjuda sig att godkänna lånet med dess fysiska tillgångar som säkerhet. Till exempel kan en ny restaurang bara kunna få ett lån genom att använda dess utrustning som säkerhet.

Villkoren för ett tillgångsbaserat lån beror på typen och värdet av de tillgångar som erbjuds som säkerhet. Långivare föredrar mycket likvida säkerheter såsom värdepapper som lätt kan konverteras till kontanter om låntagaren inte betalar. Lån med fysiska tillgångar anses mer riskfyllda, så det maximala lånet kommer att vara betydligt mindre än tillgångarnas bokförda värde. Räntorna som tas ut varierar kraftigt, beroende på den sökandes kredithistorik, kassaflöde och hur länge du har gjort affärer.

> Räntorna på tillgångsbaserade lån är lägre än räntorna på lån utan säkerhet eftersom långivaren kan ta tillbaka de flesta eller alla sina förluster i händelse av att låntagaren fallerar.

>

## exempel

Säg till exempel att ett företag söker ett lån på 200 000 USD för att utöka sin verksamhet. Om bolaget pantsätter de mycket likvida [omsättbara värdepapperen](/marketablesecurities) i sin balansräkning som säkerhet, kan långivaren bevilja ett lån motsvarande 85 % av värdepapperens nominella värde. Om företagets värdepapper värderas till 200 000 USD, kommer långivaren att vara villig att låna 170 000 USD. Om företaget väljer att pantsätta mindre likvida tillgångar, såsom fastigheter eller utrustning, kan det endast erbjudas 50 % av sin erforderliga finansiering, eller 100 000 USD.

I båda fallen representerar rabatten kostnaderna för att konvertera säkerheten till kontanter och dess potentiella förlust i [marknadsvärde](/marketvalue).

## Särskilda överväganden

Små och medelstora företag som är stabila och som har fysiska tillgångar av värde är de vanligaste tillgångsbaserade låntagarna.

Men även stora företag kan ibland söka tillgångsbaserade lån för att täcka kortsiktiga behov. Kostnaden och lång ledtid för att ge ut ytterligare aktier eller obligationer på [kapitalmarknaderna](/capitalmarkets) kan vara för hög. Kontantefterfrågan kan vara extremt tidskänslig, till exempel vid ett större förvärv eller ett oväntat utrustningsköp.

##Höjdpunkter

- Likvida säkerheter är att föredra i motsats till illikvida eller fysiska tillgångar som utrustning.

- Tillgångsbaserad utlåning används ofta av små till medelstora företag för att täcka kortsiktiga krav på kassaflöden.

– Tillgångsbaserad utlåning innebär att man lånar ut pengar med låntagarens tillgångar som säkerhet.
