# GitHub
GitHub är en webbaserad utvecklingsplattform med öppen källkod och värdtjänst som tillåter användare att ladda upp filer, dokument och datorkod till sina konton. GitHub tillåter både registrerade och icke-registrerade användare att se, ladda ner och till och med bidra till de uppladdade filerna och källkodsförråden. På grund av dess låga kostnader och funktioner har GitHub blivit den största källkodsplattformen i världen, som används av många utvecklare som ett sätt att utföra samarbete, samtidigt som de kan dela en mängd olika mjukvara, filer och dokumentation.

#### Hur GitHub fungerar

När ett GitHub-konto väl är registrerat kan användaren skapa sin egen katalog med filer och innehåll, som kallas repository. Efter att ha skapat ett arkiv kan man skapa den så kallade mastergrenen. GitHub-huvudgrenar representerar den enda startpunkten för alla projekt. Det är upp till användaren att göra ett projekt tillgängligt för vem som helst (offentligt arkiv) eller att begränsa åtkomsten till det och endast dela med vissa användare (privat arkiv). Från en mastergren kan undergrenar skapas, vilket i princip gör att användaren kan arbeta med olika aspekter av projektet separat. På ett offentligt arkiv kan andra användare föreslå ändringar av koden och så småningom ladda upp nya versioner av filerna. Förslag görs genom en funktion som kallas pull request, som gör det möjligt för utvecklarna att diskutera och granska de potentiella förändringarna innan de tillämpas effektivt. GitHub-plattformen erbjuder en mängd funktioner och verktyg som gör det enkelt för utvecklare att göra eller återställa ändringar och att spåra deras övergripande framsteg.

#### Vad är en pull-begäran?

En pull-begäran tillåter användare att berätta för andra om ändringarna de har gjort i en viss gren eller arkiv. När en pull-begäran skapas tas användaren till en översiktsskärm där han enkelt kan granska alla ändringar som gjorts i koden, jämföra den gamla och den nya versionen. Därför dokumenteras och registreras alla ändringar och användare kan också lägga till kommentarer, etiketter och milstolpar eller till och med tilldela specifika uppgifter till andra bidragsgivare.

#### Vem använder GitHub?

På grund av sin flexibla plattform har GitHub användare från både den offentliga och privata sektorn. Mjukvaruföretag, apputvecklare, enskilda programmerare och kryptovalutautvecklare är bara några exempel. Som ett projekt med öppen källkod har [Bitcoin GitHub](/bitcoin) många utvecklare runt om i världen, med massor av människor som bidrar till granskning och förbättring av Bitcoin-källkoden.

#### Är GitHub gratis?

GitHub är gratis för alla öppen källkod och kommersiella utvecklingsprojekt. De kostnadsfria planerna begränsar dock antalet privata och offentliga arkiv som kan skapas av ett konto. De kostnadsfria planerna begränsar också antalet medarbetare som ett projekt kan ha. Därför kan större projekt behöva uppgradera till en betald GitHub-plan, enligt deras behov.
