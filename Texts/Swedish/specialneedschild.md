# Barn med särskilda behov
Ett barn med särskilda behov är en ungdom som har bestämt sig för att kräva särskild uppmärksamhet och särskilda behov som andra barn inte behöver. Staten kan förklara denna status i syfte att erbjuda förmåner och hjälp för barnets välbefinnande och tillväxt. Särskilda behov kan också vara en juridisk beteckning, särskilt inom adoptions- och fosterhemsgemenskapen, där barnet och [vårdnadshavaren](/guardian) får stöd för att hjälpa dem båda att leva produktiva liv.

## Förstå innebörden av barn med särskilda behov

Vårdnadshavare för barn med särskilda behov får vanligtvis någon form av ytterligare [skattelättnad](/taxcredit) eller avdrag för att ta itu med de extra kostnader och insatser som kan kräva barnomsorg och välbefinnande.

## Vad kvalificerar sig som ett specialbehov?

En mängd olika tillstånd och funktionsnedsättningar kan klassificeras som särskilda behov. De kan innefatta kroniska och terminala sjukdomar, fysiska funktionsnedsättningar och kognitiva eller psykiatriska problem.

Den assistans och läkarvård som höjer livskvaliteten för barn med särskilda behov kan resultera i långsiktiga och eskalerande kostnader. Omfattningen av barnets tillstånd kan kräva långtgående medicinskt stöd för att barnet ska kunna leva och frodas. Ett barn med ett försvagande eller livshotande tillstånd som är permanent kan till exempel behöva konstant medicinskt stöd under hela livet.

De kan behöva övervakas regelbundet om deras besvär förvärras. Stödutrustning kan behövas för att ge barnet rörlighet runt bostaden och upphandling av stöddjur som specialtränade hundar kan också behövas.

> Ett barn med särskilda behov kan ha ett livshotande tillstånd eller allvarliga inlärningssvårigheter. Hur som helst är de ett barn som kräver särskild uppmärksamhet och omsorg som andra barn inte gör.

>

## Särskilda överväganden

Ett barn med särskilda behov kan behöva alternativa tillvägagångssätt för utbildning som inte bara tillgodoser deras förutsättningar utan också arbetar för att skapa sätt för dem att främja sin egen förmåga att lära och utvecklas.

Till exempel kan ett barn med nedsatt fysisk rörlighet eller kommunikationsutmaningar genom traditionella verbala signaler behöva tränas på andra sätt, och de kan också behöva utbildning i hur man tillämpar dessa färdigheter i klassrummet och i verkliga miljöer.

I fall där ett barn med särskilda behov har kognitiva funktionsnedsättningar, kan deras utbildning kräva expertis för att ta itu med sådana problem och hitta metoder för att få kontakt med dem. Några exempel på detta kan inkludera att de tar betydligt mer tid och ansträngning för att säkerställa att de inte bara kan förstå lektionen utan också gå vidare till ett annat inlärningsstadium.

Även om den offentliga utbildningen är öppen för alla elever, är inte alla lärare utbildade för att presentera lektioner på ett sätt som motsvarar dessa behov. Detta kan leda till att barn med särskilda behov undervisas privat, i klasser eller skolor som är dedikerade till att tillgodose dessa behov.

##Höjdpunkter

- Definitionen av särskilda behov som hänför sig till ett barn inkluderar en mängd olika tillstånd, inklusive fysiska åkommor, inlärningssvårigheter och dödlig sjukdom.

– Vissa barn med särskilda behov kan gå i offentliga skolor som erbjuder ett brett utbud av pedagogiska och känslomässiga stödprogram, såsom arbetsterapi och en-mot-en lärarhjälp i klassrummet.

– Föräldrar och vårdnadshavare till barn med särskilda behov får vanligtvis skattelättnader eller avdrag för att kompensera kostnaderna för att uppfostra ett barn med särskilda behov.

##FAQ

### Hur gynnar ABLE-konton ett barn med särskilda behov?

[Achieving a Better Life Experience (ABLE)](/able-account) -planer eller 529As är skattefördelaktiga planer som motsvarar en collegesparplan för barn under 26 år med särskilda behov. Pengar som sätts in på ett ABLE-konto kan investeras, växa skattefritt, och användas för kvalificerande utgifter. Liksom förtroendet för särskilda behov påverkar inte ABLE-planer barnets förmåga att kvalificera sig för SSI eller andra offentliga stödprogram. Till skillnad från ett förtroende för särskilda behov finns det insättningsgränser på ett ABLE-konto - $15 000 per år, med ett tak som varierar beroende på stat, men i genomsnitt upp till $300 000 per barn.

### Hur kan du bygga en ekonomisk plan för barn med särskilda behov?

Även om det i bästa fall är osäkert att förutsäga vad vård kommer att kosta i framtiden, kan föräldrar öppna ett [förtroende för särskilda behov](/special-needs-trust) som gör det möjligt för föräldrar eller nära och kära att bidra ekonomiskt till en fond som kan betalas ut för kostnader som sjukvårdskostnader, vårdnadshavare eller transport. Den viktiga detaljen med en stiftelse för särskilda behov är att de pengar som finns där inte kommer att påverka några offentliga stödprogram som barnet kan få. Särskilda behov trusts övervakas av en förtroendeman. I kombination med program som SSI, bidragsprogram och kost- och hälsovårdsprogram kan föräldrar bygga en hållbar plan för sitt barns vård.

### Är barn med särskilda behov berättigade till kompletterande trygghetsinkomst (SSI)?

Barn med särskilda behov som är under 18 år kan kvalificera sig för SSI om de uppfyller flera parametrar utifrån sin inkomstnivå och typ av funktionsnedsättning. Betalningar kan variera från stat till stat, och villkoren omvärderas vart tredje år i de flesta fall. Efter att barnet fyllt 18 år kan de vara berättigade till SSI-handikappersättning men skulle bedömas annorlunda.

### Vilken ekonomisk hjälp finns tillgänglig för föräldrar med barn med särskilda behov?

Det finns ett antal statliga program som syftar till att lätta den ekonomiska bördan av att stödja ett barn med särskilda behov. Barn med särskilda behov är berättigade till subventionerade hälsovårdsprogram som Medicaid och Children's Health Insurance Program (CHIP). Det finns också program som erbjuder direktbetalningar eller hjälp med att säkra mat och tak över huvudet, såsom Supplemental Security Income (SSI), Temporary Assistance for Nedy Families (TANF) och Supplemental Nutrition Assistance Program (SNAP). Det finns också bidrag tillgängliga från olika ideella organisationer som kan hjälpa till att kompensera kostnaderna.

### Kan ett barn lita på både förmåga och särskilda behov?

Ja, ett barn med särskilda behov kan ha både ett KURS och ett förtroende för särskilda behov. Eftersom det finns specifika kvalificerande utgifter som kan betalas med varje typ av konto, kan det vara fördelaktigt att ha både konton öppna och finansierade. Eftersom det inte finns någon övre gräns för ett förtroende för särskilda behov, erbjuder det mer spelrum för föräldrar som säkrar sitt barns framtid bortom dem, men ABLE-planer kan täcka grundläggande levnadskostnader som hyra, medan ett förtroende för särskilda behov inte kanske.

### Är "särskilda behov" okänsligt?

Vissa handikappförespråkare motsätter sig häftigt termen "särskilda behov" och skulle eliminera den från amerikanskt tal om de kunde. Grupper som National Council on Disabilities säger att undvikande av termen "handikappade" bara leder till stigmatisering.
