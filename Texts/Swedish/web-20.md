# Webb 2.0
## Vad är Web 2.0?

Web 2.0 beskriver det nuvarande tillståndet för internet, som har mer användargenererat innehåll och användbarhet för slutanvändare jämfört med dess tidigare inkarnation, Web 1.0. I allmänhet hänvisar Web 2.0 till 2000-talets internetapplikationer som har förändrat den digitala eran i efterdyningarna av [dotcom-bubblan](/dotcom-bubble).

## Förstå Web 2.0

Web 2.0 hänvisar inte till några specifika tekniska uppgraderingar av internet. Det hänvisar helt enkelt till en förändring i hur internet används under 2000-talet. I den nya tiden finns det en högre nivå av informationsutbyte och sammanlänkning mellan deltagarna. Den här nya versionen tillåter användare att aktivt delta i upplevelsen istället för att bara agera som passiva tittare som tar in information.

## Web 1.0 vs. Web 2.0

Web 1.0 används för att beskriva det första steget av Internet. Vid det här laget fanns det få innehållsskapare; de flesta av dem som använde Internet var konsumenter. Statiska sidor var vanligare än dynamisk [HTML](/html),. som innehåller interaktiva och animerade webbplatser med specifik kodning eller språk. Innehållet i detta skede kom från en servers filsystem snarare än ett databashanteringssystem. Användare kunde signera gästböcker online och HTML-formulär skickades via e-post.

Exempel på webbplatser som klassificeras som Web 1.0 är Britannica Online, personliga webbplatser och mp3.com. I allmänhet är dessa webbplatser statiska och har begränsad funktionalitet och flexibilitet.

Termen Web 2.0 kom i bruk först 1999 då Internet svängdes mot ett system som aktivt engagerade användaren. Användare uppmuntrades att tillhandahålla innehåll istället för att bara titta på det. Folk kunde nu publicera artiklar och kommentarer, och det blev möjligt att skapa användarkonton på olika sajter, vilket ökade deltagandet. Web 2.0 gav också upphov till webbappar, självpublicerande plattformar som [WordPress](/wordpress-cms),. samt webbplatser för [sociala medier](/social-media).

Exempel på Web 2.0-webbplatser inkluderar Wikipedia, Facebook, Twitter och olika bloggar, som alla har förändrat hur samma information delas och levereras.

Den sociala aspekten av Internet har förändrats särskilt; i allmänhet tillåter sociala medier användare att engagera sig och interagera med varandra genom att dela tankar, perspektiv och åsikter. Användare kan tagga, dela, tweeta och gilla.

## Fördelar och nackdelar med Web 2.0

Teknikutvecklingen har gjort det möjligt för användare att dela sina tankar och åsikter med andra, vilket har skapat nya sätt att organisera och knyta an till andra människor och främjat en högre grad av samarbete.

Men det finns många nackdelar med att internet agerar mer som ett öppet forum. Genom expansionen av sociala medier har vi sett en ökning av stalking online, nätmobbning, [doxning](/personally-identifiable-information-pii),. [identitetsstöld](/identitytheft) och andra onlinebrott. Det finns också hot om att desinformation sprids bland användare, vare sig det är genom sajter för informationsdelning med öppen källkod eller på sociala medier.

Precis som Web 1.0 är Web 2.0 en annan övergångsfas i utvecklingen av Internet. Web 3.0 förutspås kallas den semantiska webben eftersom den kommer att skräddarsys för att bli mer intuitiv för varje användares behov.

## Höjdpunkter

– I Internets nya tidsålder finns en högre nivå av informationsdelning och sammanlänkning bland deltagarna.

- Web 2.0 hänvisar inte till några specifika tekniska uppgraderingar av internet; det hänvisar till en förändring i hur internet används.

- Web 2.0 beskriver det nuvarande tillståndet för internet, som har mer användargenererat innehåll och användbarhet för slutanvändare jämfört med dess tidigare inkarnation, Web 1.0.
