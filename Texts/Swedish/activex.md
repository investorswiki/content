# ActiveX
## Vad är ActiveX?

ActiveX är ett mjukvaruramverk från Microsoft (MSFT) som tillåter applikationer att dela funktionalitet och data med varandra via webbläsare, oavsett vilket programmeringsspråk de är skrivna på. ActiveX-tillägg gjorde det möjligt för tidiga webbläsare att bädda in multimediafiler eller leverera programuppdateringar till användare.

Microsoft introducerade ActiveX 1996. Många ActiveX-kontroller körs endast på Windows och med Microsoft-produkter som Internet Explorer, Word och Excel. JavaScript, en annan kompilator för flera språk och andra liknande plattformar används mer än ActiveX.

## Förstå ActiveX

ActiveX-kontroller är förkodad programvara som liknar plugin-program för webbläsare. Till exempel kan en webbsida som visar en Flash-fil kräva att en användare laddar ner en Flash ActiveX-kontroll så att filen kan spelas upp direkt i webbläsaren utan att öppna ett nytt program. ActiveX-kontroller utökar en webbläsares funktionalitet, vilket gör att webbläsaren kan utföra uppgifter som den annars inte skulle kunna utföra medfödd. Det är särskilt användbart för att spela upp videor och annat multimediainnehåll och hoppa över steget att öppna en separat mediaspelare.

ActiveX används fortfarande i Internet Explorer 11, men stöds inte av Microsofts senaste webbläsare, Edge . Webbläsare som Mozilla Firefox, Google Chrome, Apples Safari och Opera använder andra typer av plugin-program för webbläsare, som JavaScript eller liknande plattformsoberoende språk.

ActiveX är fortfarande användbart för Microsoft-användare och ingår i Windows 10. Detta beror på att ActiveX fortfarande tillåter fristående programvara att ta emot uppdateringar, gränssnitt över program och förbättra funktionaliteten. Till exempel tillåter ActiveX användare att skapa mer interaktiva dokument i Word eller skapa ifyllbara formulär i Excel.

## ActiveX och datorsäkerhet

Microsoft varnar ActiveX-kontroller kan ibland fungera felaktigt eller ge användarna innehåll som de inte vill ha. ActiveX-kontroller kan också användas för att installera [spionprogram](/adware),. virus och skadlig programvara, eller skada data på din dator. Av denna anledning är det viktigt att endast installera ActiveX-kontroller från källor du litar på .

Delvis på grund av utbredd skadlig användning av ActiveX-kontroller, inaktiverar många webbläsare antingen ActiveX-kontroller som standard eller stöder dem inte alls. Till exempel måste Google Chrome-användare aktivera ActiveX i Chromes säkerhetsinställningar eller ladda ner ett Chrome-tillägg .

Till och med Microsoft verkar vända sig bort från ActiveX. Edge, dess nya webbläsare som ersätter Internet Explorer på Windows-operativsystem, stöder inte ActiveX .

##Höjdpunkter

- Microsoft varnar för att ActiveX-kontroller kan användas för att spionera på din dator, skada data eller installera skadlig programvara .

– ActiveX är en plattform för att utveckla små program eller plug-ins som ger webbläsare och Microsoft-produkter ökad funktionalitet.

- Många webbläsare stöder inte ActiveX som standard. Edge, Microsofts nya webbläsare, erbjuder inte stöd för ActiveX-kontroller .
