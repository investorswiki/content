# Intyg om förlust
## Vad är intyg om förlust?

En förlustbekräftelse är ett dokument som förklarar förlusten av ett [värdepapper](/security) vanligtvis genom stöld eller förstörelse. Intyget innehåller alla detaljer om förlusten, såsom ägarens namn och all information som hänför sig till säkerheten. Den informationen kan inkludera ett serienummer eller värdepapperets utfärdandedatum. Efter att utlåtandet har gjorts kan ett skadeståndsbrev utfärdas med begäran om utbyte av säkerheten.

## Förståelseförklaring om förlust

Det finns alltid möjligheten att ett fysiskt register över ett värdepapper som ett aktiecertifikat kan gå förlorat. Om dokumentet blir stulet, förlagt eller förstört, måste ägaren svära en ed om förlusten av föremålet. Det här är en ed som beskriver att såvitt personen vet är skivan verkligen borta. Om det förlorade exemplaret dyker upp efter att ägaren erhållit en ersättningssäkerhet ska originalanteckningen skickas till företaget för omhändertagande.

### Andra användningsområden

Det finns jämförbara uppgifter för saknade finansiella dokument som inte är värdepapper. En försäkran om förlorad sedel lämnas när ett [skuldebrev](/promissorynote) eller annan skulduppteckning förstörs eller går förlorad. Denna typ av intyg skulle innehålla gäldenärens namn, det utestående saldot, räntesatsen på skulden, detaljer om återbetalningshistoriken och lista över alla fastigheter som ställdes upp som säkerhet.

Ett intyg om förlust kan lämnas in för andra typer av dokument, till exempel de som är relaterade till ägande av ett hus, ett pass, ett diplom eller till och med ett bankomatkort. Saknade äktenskapstillstånd kan kräva ett sådant dokument.

Varje gång en person förlorar en post som inte omedelbart kan ersättas med en kopia, kan de behöva lämna in en intyg om förlust. Till exempel kan [titeln](/title),. registreringen eller skyltarna på ett fordon förstöras, stjälas eller förstöras i den utsträckning de inte längre kan användas. Vissa stater kräver ett intyg för att registrera sin förlust. Ägaren eller rättighetshavaren måste tillhandahålla sådan information som märke, modell och fordonsidentifikationsnummer för fordonet som är kopplat till de saknade dokumenten.

Reglerna och kraven angående information som krävs vid inlämnande av en försäkran om förlust varierar beroende på jurisdiktion. Vissa kräver ett intyg om förlust för att bli attesterat.

##Höjdpunkter

– En förlustbevis är en handling som förklarar att ett värdepapper, till exempel ett aktiebrev, är förlorat eller förstört.

- Värdepappersutgivaren kräver en bekräftelse för att hålla innehavaren skadeslös eller ersätta certifikatet.

– Individer kan lämna in intyg om förlust för andra typer av handlingar, som pass eller examensbevis.
