# Klicka på Bedrägeri
## Vad är klickbedrägeri?

Klickbedrägeri är handlingen att illegalt klicka på betala-per-klick-annonser (PPC) för att öka webbplatsens [intäkter](/revenue) eller för att förbruka ett företags [annonseringsbudget](/budget). Klickbedrägeri skiljer sig från ogiltiga klick (de som upprepas eller görs av annonsens värd/utgivare) genom att det är avsiktligt, skadligt och inte har någon potential för att annonsen ska resultera i en försäljning.

Klickbedrägeri sker med betal-per-klick-annonsering och kan involvera antingen en människa, ett datorprogram eller ett automatiserat skript som utger sig för att vara en legitim användare och klickar på betald sökannonsering utan avsikt att köpa något.

## Förstå klickbedrägeri

Många webbplatser och onlineplattformar är värd för annonser och betalas av de företag som annonserar för att vara värd för deras annonser. Ett av betalningssystemen är att värdwebbplatsen kommer att betalas beroende på hur många besökare på webbplatsen som klickar på annonserna. Klickbedrägeri uppstår när en annons klickas på ett dubbelt sätt utan någon egentlig avsikt eller intresse av att besöka webbplatsen eller köpa tjänsten eller produkten som annonseras.

### Klicka på Bedrägeri för att minska konkurrensen

Klickbedrägeri begås av två huvudsakliga skäl: för att minska konkurrensen mellan annonsörer eller för att generera intäkter genom att spela PPC-reklamsystemet. Annonsör A kan till exempel delta i klickbedrägeri för att använda upp annonsör B:s annonsbudget och utrymme på irrelevanta klick, vilket gör att annonsör A blir den enda annonsören. Detta är ett exempel på ett klickbedrägeri som inte är avtalsslutande part.

Som ett annat exempel kan en illvillig angripare med uppsåt försöka få det att se ut som att en utgivare klickar på sina egna annonser, vilket kan få ett annonsnätverk att avsluta sin relation med den utgivaren och tro att de handlar i ond tro. Eftersom PPC-reklamintäkter är den primära [inkomstkällan](/income) för vissa utgivare, kan denna praxis sätta en utgivare i konkurs.

Klickbedrägeri kan också i vissa fall begås helt enkelt för att vandalisera utan ett särskilt ekonomiskt motiv eller när vänner, familj eller fans till en utgivare klickar på annonser på en webbplats för att generera mer intäkter för utgivaren. Båda formerna kan vara svåra att upptäcka.

### Klicka på Bedrägeri för att på konstgjord väg öka intäkterna

Ett annat skäl skulle vara att webbplatsägare (publicister) begår klickbedrägerier för att öka sina annonsintäkter. Detta arrangemang involverar tre parter: ett annonsnätverk (som Googles AdSense eller Yahoo! Search Marketing) som placerar annonsen; utgivaren som publicerar annonsen; och en annonsör som skapar annonsen och avtalar med annonsnätverket för att placera annonsen.

Klickbedrägeri under den här infrastrukturen sker när utgivare klickar på annonser som har placerats på deras egna webbplatser för att generera intäkter.

## Klicka på Bedrägeri i praktiken

Det enklaste och minst upptäckbara sättet att begå klickbedrägerier är att skapa en webbplats som är värd för [bannerannonser](/banneradvertising) och klicka på dessa annonser så mycket som möjligt för att generera intäkter. Vissa företag kommer att anställa lågkostnadsanställda – ofta utomlands – för att manuellt klicka på annonser hela dagen. Dessa är kända som klickgårdar.

Andra kommer att skriva eller använda skript för att automatiskt klicka på annonser. Båda dessa metoder är lätta att spåra om inte användaren eller skriptet maskerar datorns sanna [IP-adress](/ip-address),. vilket är lätt att göra genom att använda ett VPN. Ett annat populärt sätt är att använda datorvirus för att i smyg ta över ett stort antal datorer och låta dessa datorer klicka på annonser.

## Bestämma och eliminera klickbedrägeri

Ett företag måste veta när klickbedrägerier inträffar så att deras annonseringsbudget inte slösas bort. En annonsbudget är viktig eftersom den [hjälper till att generera försäljning](/marketing-strategy) och att sälja en produkt eller tjänst är målet för ett företag. Om ett företags reklambudget slösas bort genom klickbedrägerier och inte når faktiska potentiella kunder kan det påverka företaget ekonomiskt.

Det finns några sätt som klickbedrägeri kan fastställas. En är genom kontinuerliga klick från [ISP-](/isp) servrar som verkar likna varandra, en annan är genom ökningar i kostnader som inte är i linje med tidigare annonskampanjer eller statistik, brist på konverteringar (människor som köper din produkt) trots att annonskostnaderna ökar, och andra udda händelser som inte kan förklaras.

Många leverantörer hjälper till att förebygga klickbedrägeri men ett företag kan sträva efter att själv fastställa [bedrägeri](/fraud). Ett företag kan välja att annonsera på välkända webbplatser av hög kvalitet, blockera vissa IP-adresser, använda programvara som förhindrar klickbedrägeri och ständigt övervaka sina loggar och budgetar.

## Höjdpunkter

- Klickbedrägeri inträffar när en människa eller ett program (skript) klickar på annonser som utger sig för att vara legitima användare men utan avsikt att fullfölja ett köp.

- Klickbedrägeri kan upptäckas och förebyggas på en mängd olika sätt, såsom specialiserad övervakningsprogramvara som kan identifiera anomalier eller misstänkta klickaktiviteter.

- Klickbedrägeri är den olagliga metoden att spoofa betala-per-klick-annonser (PPC) för att öka webbplatsens intäkter eller för att tömma ett företags reklambudget.

– Klickbedrägeri kan ibland utföras av en webbplatss egna ägare för att på konstgjord väg öka dess annonsintäkter.
