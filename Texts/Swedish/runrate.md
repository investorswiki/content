# Run Rate
## Vad är körfrekvensen?

Run rate hänvisar till ett företags finansiella resultat baserat på att använda aktuell finansiell information som en förutsägelse av framtida resultat. Run rate fungerar som en extrapolering av nuvarande finansiella resultat och antar att nuvarande förhållanden kommer att fortsätta.

Körhastigheten kan också hänvisa till den genomsnittliga årliga utspädningen från företagets aktieoptionstilldelningar under den senaste treårsperioden som registrerats i årsredovisningen.

## Förstå körhastigheten

I samband med att extrapolera framtida prestanda tar körhastigheten aktuell prestandainformation och sträcker den över en längre period. Till exempel, om ett företag har [intäkter](/revenue) på 100 miljoner dollar under det senaste kvartalet, kan [VD :](/ceo) n dra slutsatsen att företaget, baserat på det senaste kvartalet, har en drifthastighet på 400 miljoner dollar. När data används för att skapa en årlig prognos för potentiell prestanda, kallas processen för annualisering.

En run rate kan vara till hjälp vid skapandet av prestationsuppskattningar för företag som har varit verksamma under korta tidsperioder, till exempel mindre än ett år, såväl som nyskapade avdelningar eller resultatenheter. Detta kan vara särskilt sant för ett företag som upplever sitt första lönsamma kvartal. Dessutom kan körhastigheten vara till hjälp i fall där en grundläggande affärsverksamhet ändrades på något sätt som förväntades påverka alla framtida resultat för den associerade verksamheten.

## Risker med att använda körhastigheten

Körhastigheten kan vara ett mycket vilseledande mått, särskilt i säsongsbetonade branscher. Ett bra exempel på detta är en återförsäljare som undersöker vinst efter vinterhelgen, eftersom detta är en tid då många återförsäljare upplever högre försäljningsvolymer. Om information baserad på semestersäsongens försäljning användes för att skapa en körhastighet, kan uppskattningar av framtida resultat bli förhöjda.

Dessutom baseras körhastigheten i allmänhet endast på de senaste uppgifterna och kanske inte korrekt kompenserar för omständighetsförändringar som kan orsaka en felaktig helhetsbild. Som ett exempel upplever vissa tekniktillverkare som Apple och Microsoft högre försäljning i samband med en ny produktsläpp. Att endast använda data från perioden omedelbart efter en stor produktsläpp kan leda till skev data.

Vidare tar körhastigheter inte hänsyn till stora engångsförsäljningar. Till exempel, om en tillverkare får ett stort kontrakt som betalas i förskott, oavsett tidpunkten för leveransen av varorna eller tjänsterna, kan detta göra att försäljningssiffrorna blir onormalt höga under en rapporteringsperiod baserat på detta onormala köp.

### Hur kommer körfrekvensen fram?

Inom finans extrapolerar körhastigheten ett företags nuvarande prestation för att förutsäga framtida resultat med antagandet att nuvarande förhållanden kommer att bestå. Prognoserna är vanligtvis för ett helt år, vilket är anledningen till att denna process ibland kallas "annualisering". Till exempel, om ett företag har intäkter på 100 miljoner dollar under det senaste kvartalet, kan VD:n dra slutsatsen att företaget arbetar med en årlig körhastighet på 400 miljoner dollar.
