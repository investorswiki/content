# Junior Hypotek
## Vad är ett juniorlån?

Ett juniorbolån är ett bolån som är underordnat ett första eller tidigare (senior) inteckningslån. Ett juniorlån hänvisar ofta till ett [andra inteckningslån](/secondmortgage),. men det kan också vara ett tredje eller fjärde inteckning (t.ex. bostadslån eller kreditlinjer (HELOC)). Vid utmätning betalas [senior (första) inteckning först.](/first_mortgage)

## Förstå Junior Mortgage

Ett juniorlån är ett [underordnat](/subordinate-financing) inteckning som görs medan ett originallån fortfarande gäller. I händelse av fallissemang skulle det ursprungliga inteckningslånet få alla intäkter från [likvidationen](/liquidation) av fastigheten tills det hela är betalat. Eftersom juniorbolån skulle få återbetalning först när det första bolånet har betalats, tenderar räntan som tas ut för ett juniorbolån att vara högre och det lånade beloppet blir lägre än det första bolånet.

Vanliga användningsområden för yngre inteckningar inkluderar piggy-back-inteckningar (80-10-10 inteckningar) och [bostadslån](/homeequityloan). Piggy-back inteckningar ger låntagare med mindre än 20 % handpenning ett sätt att undvika kostsamma privata bolåneförsäkringar. Bostadslån används ofta för att extrahera eget kapital för ett hem för att betala ner andra skulder eller göra ytterligare inköp. Varje upplåningsscenario bör analyseras noggrant och grundligt.

## Restriktioner och begränsningar för att fullfölja juniorlån

En junior inteckning kanske inte tillåts av innehavaren av den ursprungliga inteckningen. Om det finns villkor i ett bolån som tillåter att juniorbolån kan stiftas kan det finnas krav som låntagaren måste uppfylla innan man gör det. Till exempel kan en viss summa av seniorbolånet behöva betalas av innan ett juniorbolån kan tas. Långivaren kan också begränsa antalet yngre inteckningar som låntagaren kan ta på sig.

Ökad risk för fallissemang är ofta förknippad med juniorbolån. Detta har lett till att långivare tar ut högre räntor för juniorbolån jämfört med seniorbolån. Införandet av mer skulder genom ett juniorlån kan innebära att låntagaren är skyldig mer pengar på sitt hus än vad det är värderat på marknaden.

Om låntagaren inte kan hålla jämna steg med sina betalningar och huset förfaller till avskärmning, kan långivaren som tillhandahållit juniorlånet riskera att inte få tillbaka sina medel. Till exempel kan utbetalningen till innehavaren av ett seniorbolån förbruka hela eller större delen av tillgångarna. Det skulle innebära att långivaren för juniorlånet skulle kunna gå obetalt.

## Andra överväganden

Låntagare kan söka juniorlån för att betala av kreditkortsskulder eller för att täcka köpet av en bil. Till exempel kan en låntagare ta ett juniorlån med en 15-årig löptid för att ha medel för att betala av ett billån som har en löptid på fem år. Eftersom ny skuld införs genom juniorbolån är det möjligt att låntagaren inte kan betala tillbaka sina förpliktelser. Eftersom bostaden fungerar som säkerhet, även om de betalar av seniora inteckningar, kan låntagare möta [avskärmning](/foreclosure) av juniora inteckningar som förfaller till standard.

## Höjdpunkter

– Ett juniorbolån är ett bostadslån som görs utöver fastighetens primärbolån.

- Juniorbolån har ofta högre räntor och lägre lånebelopp och kan vara föremål för ytterligare restriktioner och begränsningar.

- Bostadslån och HELOC används ofta som andralån.

- Husägare kan söka ett juniorlån för att finansiera stora inköp som en ombyggnad av hemmet, collegeavgifter eller ett nytt fordon.
