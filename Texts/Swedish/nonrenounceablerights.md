# Ej avsägbara rättigheter
## Vad är icke-avsägbara rättigheter?

En icke-avsägbar företrädesemission avser ett erbjudande från ett företag till aktieägare om att köpa fler aktier i företaget (vanligtvis med rabatt).

Till skillnad från en rättighet som kan [avsägas](/renounceablerights),. är en icke-avsägbar rättighet inte överlåtbar och kan därför inte köpas eller säljas.

## Förstå icke-avsägbara rättigheter

Att ge ut fler aktier späder ut värdet på utestående aktier. Men eftersom företrädesemissionen tillåter befintliga aktieägare att köpa den nyemitterade aktien med rabatt, kompenseras de för den förestående [aktieutspädningen](/dilution). Den ersättning som företrädesemissionen ger dem motsvarar kostnaden för aktieutspädning. Däremot kommer aktieägare som inte utnyttjar rättigheterna genom att köpa den rabatterade aktien att förlora pengar eftersom deras befintliga innehav kommer att drabbas av utspädningen.

### Avsägbara rättigheter

En avsägbar rätt är också en inbjudan till ett bolags befintliga aktieägare att köpa ytterligare nya [aktier](/shares) i bolaget. Medan aktieägare har befogenhet att köpa fler aktier, kan de avsäga sig det privilegiet och handla med sina rättigheter på den öppna marknaden.

## Varför företag erbjuder icke-avsägbara rättigheter

Genom att erbjuda icke-avsägbara rättigheter, sätter företaget ett snävt [fönster av möjlighet](/window-of-opportunity) för aktieägare att köpa aktier med rabatt. Att erbjuda sådana rättigheter kan ses som mer fördelaktigt för bolaget än för befintliga aktieägare, trots att rabatt erbjuds. Om aktieägarna inte har tillräckliga medel vid den tidpunkt då de icke avsägbara rättigheterna kan utnyttjas, kan de förlora möjligheten att köpa till diskonteringsräntan. Oavsett vilka åtgärder de befintliga aktieägarna vidtar kommer bolaget att fortsätta med att emittera mer aktier.

Ett företag kan erbjuda icke-avsägbara rättigheter på aktier om det finns en tidsram och ett kapitalmål som verksamheten behöver uppfylla. Det kan behöva samla in pengar för ett förvärv, utöka sin verksamhet genom att anställa eller lägga till nya platser, betala tillbaka skulder eller hur många saker som helst. Företaget kan komma att gå i konkurs om det inte tar itu med sina kapitalförhållanden.

Om bolaget måste anskaffa kapital för att bibehålla sina framtidsutsikter som en fortsatt drift, kan det bli nödvändigt att emittera aktier oavsett potentiell utspädning av befintliga aktier. Icke avsägbara rättigheter är ett sätt för företaget att ge befintliga aktieägare chansen att behålla sin andel i verksamheten samtidigt som de kontrollerar det utrymme som finns för dem att dra nytta av rabatterna. För aktieägare kan detta ses som ett mindre önskvärt alternativ än att erbjudas rättigheter de kan tänkas sälja på marknaden och själva se avkastning.

## Höjdpunkter

- En rättighet som kan avsägas tillåter å andra sidan handel med rättigheterna.

- Företag kan erbjuda icke-avsägbara nyemissioner om ett akut behov av kontanter uppstår.

- En företrädesemission som inte kan avsägas gör det möjligt för befintliga aktieägare att köpa fler aktier i ett företag med rabatt. Dessa aktier kan inte handlas.
