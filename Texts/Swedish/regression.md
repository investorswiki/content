# regression
## Vad är regression?

Regression är en statistisk metod som används inom finans, investeringar och andra discipliner som försöker bestämma styrkan och karaktären hos sambandet mellan en beroende variabel (vanligtvis betecknad med Y) och en serie andra variabler (kända som oberoende variabler).

Regression hjälper investerings- och finansförvaltare att värdera tillgångar och förstå sambanden mellan variabler, såsom [råvarupriser](/commodity) och aktierna hos företag som handlar med dessa råvaror.

## Regression förklaras

De två grundläggande typerna av regression är enkel linjär regression och multipel linjär regression, även om det finns icke-linjära regressionsmetoder för mer komplicerad data och analys. Enkel linjär regression använder en oberoende variabel för att förklara eller förutsäga resultatet av den beroende variabeln Y, medan multipel linjär regression använder två eller flera oberoende variabler för att förutsäga resultatet.

Regression kan hjälpa finans- och investeringsproffs såväl som proffs i andra företag. Regression kan också hjälpa till att förutsäga försäljning för ett företag baserat på väder, tidigare försäljning, BNP-tillväxt eller andra typer av förhållanden. Kapitaltillgångsprissättningsmodellen (CAPM) är en ofta använd regressionsmodell inom finans för att [prissätta](/capm) tillgångar och upptäcka kapitalkostnader.

Den allmänna formen för varje typ av regression är:

- **Enkel linjär regression:** Y = a + bX + u

- **Multipel linjär regression:** Y = a + b~1~X~1~ ^+^ b~2~X~2~ + b~3~X~3~ + ... + b~t~ X~t~ + u

Var:

- Y = variabeln som du försöker förutsäga (beroende variabel).

- X = variabeln som du använder för att förutsäga Y (oberoende variabel).

- a = skärningen.

-b = lutningen.

- u = regressionsresten.

> Det finns två grundläggande typer av regression: enkel linjär regression och multipel linjär regression .

>

Regression tar en grupp [slumpvariabler](/random-variable),. som tros förutsäga Y, och försöker hitta ett matematiskt samband mellan dem. Detta förhållande är vanligtvis i form av en rät linje (linjär regression) som bäst approximerar alla individuella datapunkter. Vid multipel regression differentieras de separata variablerna genom att använda subskript.

## Ett verkligt exempel på hur regressionsanalys används

Regression används ofta för att bestämma hur många specifika faktorer som priset på en vara, räntor, särskilda branscher eller sektorer som påverkar prisrörelsen för en tillgång. Ovannämnda CAPM är baserat på regression och används för att projicera förväntad avkastning för aktier och för att generera kapitalkostnader. En akties avkastning regresseras mot avkastningen från ett bredare index, såsom S&P 500, för att generera en beta för den specifika aktien.

Beta är aktiens risk i förhållande till marknad eller index och återspeglas som lutningen i CAPM-modellen. Avkastningen för aktien i fråga skulle vara den beroende variabeln Y, medan den oberoende variabeln X skulle vara marknadsriskpremien.

Ytterligare variabler som börsvärde för en aktie, värderingskvoter och senaste avkastning kan läggas till CAPM-modellen för att få bättre uppskattningar av avkastningen. Dessa ytterligare faktorer är kända som de Fama-franska faktorerna, uppkallade efter professorerna som utvecklade den multipellinjära regressionsmodellen för att bättre förklara tillgångsavkastningen.
