# Kreditkortsfinansiering
## Vad är kreditkortsfinansiering?

Kreditkortsfinansiering är möjligheten att elektroniskt finansiera ett nytt konto, företag eller annan satsning genom att använda ett kreditkort. [Kreditkortsfinansiering](/creditcard) gör det möjligt för en individ eller ett företag att använda en lättillgänglig källa till medel, även om medlen lånas och därmed har en ränta.

## Förstå kreditkortsfinansiering

Små företag kan ha svårt att få startkapital för att köpa inventarier, göra en hyresdeposition eller andra funktioner som kräver kontanter. Om företagaren inte har sparpengar till hands och inte kan skaffa ett lån, kan kreditkortsfinansiering vara ett alternativ. Detta är särskilt fallet när ett minimum av finansiering krävs för att hålla ett konto öppet.

Investerare är en grupp som också använder sina kreditkort för att placera initiala medel på ett konto. Detta är ofta ett mer populärt alternativ på [valutakonton](/forex),. även om regler kan begränsa eller förbjuda användningen av kreditkort för att finansiera konton som är involverade med mer högriskinvesteringar, såsom derivat och valutor.

## Hur kreditkortsfinansiering kan användas för att etablera andra konton

Vissa banker kan tillåta att kreditkortsfinansiering används när man öppnar ett bankkonto. Detta kan göras i syfte att uppfylla minimikraven för saldo för att upprätta ett bankkonto. Det kan också vara ett sätt för kreditkortsinnehavaren att nå utgifterna för att tjäna en registreringsbonus eller andra belöningar för sina kort. Dessutom kan detta göras för att tjäna samtidiga bonusar från båda kontona, vilket kan inkludera pengar tillbaka för att uppfylla kraven under registreringsfasen. En sådan strategi kräver vanligtvis en plan för att också betala av kreditkortssaldot för att undvika att betala ränteavgifter och andra kostnader.

Vissa institutioner som accepterar vissa former av elektronisk finansiering kanske inte accepterar kreditkortsfinansiering, men kan acceptera finansiering via betalkort. Detta beror på att medlen från ett [betalkort](/debitcard) endast kommer att överföras om de finns på kortinnehavarens konto, vilket innebär att kortinnehavaren inte sätter in lånade medel som kräver att de betalar ränta. För riskabla satsningar, såsom investeringar och spekulationer, är användningen av kreditkort begränsad eller förbjuden eftersom kortinnehavaren kan förlora de insatta medlen och inte kunna betala tillbaka dem.

Kreditkortsinnehavare bör granska sitt kortavtal för att avgöra om kortföretaget anser att kreditkortsfinansiering är ett [kontantförskott](/cashadvance). Företag tar ofta ut olika räntor på lånade medel beroende på typen av transaktion, med köpkurser som vanligtvis är lägre än räntan på saldoöverföringar och kontantförskott.

## Höjdpunkter

– Kreditkortsfinansiering är ett alternativ för företagare som kanske inte kan få ett traditionellt lån. Det kan också användas av en grupp investerare för att kollektivt finansiera ett konto.

- Kreditkortsfinansiering innebär att man använder ett kreditkort för att finansiera ett nytt konto, företag eller annan satsning.

– Vissa riskabla satsningar förbjuder kreditkortsfinansiering, medan andra tillåter det men kan ta ut högre räntor.
