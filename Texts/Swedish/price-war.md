# Priskrig
## Vad är ett priskrig?

Ett priskrig är ett konkurrensutsatt utbyte mellan rivaliserande företag som sänker prispoängen på sina produkter, i ett strategiskt försök att underskatta varandra och ta större marknadsandelar. Ett priskrig kan användas för att öka intäkterna på kort sikt, eller så kan det användas som en långsiktig strategi.

Priskrig kan förhindras genom strategisk prishantering, som bygger på icke-aggressiv prissättning, en grundlig förståelse för konkurrensen och till och med robust kommunikation med konkurrenter.

> Priskrig bör inledas med försiktighet eftersom prissättningen har störst inverkan på ett företags resultaträknings resultat; ett prisfall på 1 % kan minska vinsten med mer än 10 %.

>

## Förstå priskrig

När ett företag försöker öka marknadsandelar är det enklaste sättet vanligtvis att sänka priserna, vilket sedan ökar produktförsäljningen. Konkurrensen kan tvingas följa efter om den säljer liknande produkter. Och när priserna sjunker ökar försäljningsmängden, vilket gynnar kunderna.

Så småningom nås en prisnivå som bara ett företag har råd att erbjuda, samtidigt som det fortfarande är lönsamt. Vissa företag kommer till och med att sälja med förlust i ett försök att eliminera konkurrensen helt.

## Speciella överväganden: Vad kan utlösa ett priskrig

Priskrig kan drivas av konkurrens mellan företag som är lokala i förhållande till varandra, som vill dominera det geografiska fotavtryck de ömsesidigt upptar. Med onlineföretag kan priskrig startas via onlineplattformar som vill ta affärer från fysiska företag som riktar sig mot samma konsumentdemografi och försöker sälja liknande produkter.

Företag som deltar i priskrig gör ett gemensamt val att minska eller eliminera sina nuvarande vinstmarginaler, i ett försök att locka fler kunder. För att mildra dessa effekter kan ett företag odla ett avtal med sina leverantörer för att anskaffa material eller färdiga produkter till en stor rabatt, jämfört med de priser som leverantörerna tar ut konkurrerande företag. Denna praxis gör det möjligt för företaget att drastiskt sänka sina priser till kunder under längre tidsperioder än konkurrenterna.

I sådana scenarier kan leverantören faktiskt lida förlusten, snarare än företaget som är engagerat i priskriget. Men företag som flyttar stora mängder produkter kan ha köpkraft att utnyttja sådana avtal.

Till exempel kan en nationell big-box-återförsäljare som säljer stora volymer av en produkt genom sina platser över hela landet ha ett avtal med leverantören om att fylla sitt lager med rabatt. Det skulle låta denna återförsäljare flytta produkten till lägre priser än marknadspriser.

Som svar kan lokala konkurrenskraftiga återförsäljare försöka erbjuda kortsiktiga rabatter för att locka kunder. Big-box-återförsäljaren kan sedan eskalera situationen till ett priskrig, och sänka sina priser ännu lägre än vad de lokala återförsäljarna kan matcha. Sådan praxis, om den upprätthålls under längre perioder, kan så småningom tvinga lokala återförsäljare att lägga ner sin verksamhet.

## Höjdpunkter

– Företag som deltar i priskrig gör ett uttryckligt val att dra ned nuvarande vinstmarginaler, i ett försök att locka fler kunder på kort sikt.

– För att hålla sig lönsamt under ett priskrig kan ett företag ordna med att köpa material från leverantörer till betydande rabatter.

- Ett priskrig avser agerandet av två konkurrerande företag som båda sänker priserna på produkter, i ett försök att underbyta varandra och ta större marknadsandelar.
