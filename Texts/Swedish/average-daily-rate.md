# Genomsnittlig dagspris (ADR)
## Vad är den genomsnittliga dagliga räntan (ADR)?

Den genomsnittliga dagliga priset (ADR) är ett mått som ofta används inom besöksnäringen för att indikera den genomsnittliga intäkten för ett upptaget rum en viss dag. Den genomsnittliga dagskursen är en av branschens [nyckelprestandaindikatorer (KPI).](/kpi)

Ett annat KPI-mått är [beläggningsgraden](/occupancy-rate),. som i kombination med ADR, omfattar [intäkt per tillgängligt rum](/revpar) (RevPAR), som alla används för att mäta driftsprestanda för en logienhet som ett hotell eller motell.

## Förstå den genomsnittliga dagliga räntan (ADR)

Det genomsnittliga dagliga priset (ADR) visar hur mycket intäkter som görs per rum i genomsnitt. Ju högre ADR, desto bättre. En stigande ADR tyder på att ett hotell ökar pengarna det tjänar på att hyra ut rum. För att öka ADR bör hotellen undersöka sätt att höja priset per rum.

Hotelloperatörer försöker öka ADR genom att fokusera på prissättningsstrategier. Detta inkluderar merförsäljning, korsförsäljningskampanjer [och](/cross-sell) kostnadsfria erbjudanden som gratis transferservice till den lokala flygplatsen. Den övergripande ekonomin är en stor faktor för att fastställa priser, med hotell och motell som försöker anpassa rumspriserna för att matcha nuvarande efterfrågan.

För att bestämma driftresultatet för ett boende kan ADR mätas mot ett hotells historiska ADR för att leta efter trender, såsom säsongspåverkan eller hur vissa kampanjer presterade. Det kan också användas som ett mått på relativ prestanda eftersom måttet kan jämföras med andra hotell som har liknande egenskaper, såsom storlek, kundkrets och läge. Detta hjälper till att prissätta rumsuthyrning korrekt.

## Exempel på den genomsnittliga dagskursen (ADR)

Om ett hotell har 50 000 USD i rumsintäkter och 500 sålda rum skulle ADR vara 100 USD (50 000 USD/500). Rum som används för internt bruk, till exempel de som är avsatta för hotellanställda och gratisrum, exkluderas från beräkningen.

## Exempel på verkliga världen

Tänk på Marriott International (MAR), en stor börsnoterad hotellägare som rapporterar ADR tillsammans med beläggningsgrad och RevPAR. För 2019 ökade Marriotts ADR med 2,1 % från 2018 till 202,75 USD i Nordamerika. Uthyrningsgraden var ganska statisk på 75,8 %. Att ta ADR och multiplicera den med beläggningsgraden ger RevPAR. I Marriotts fall motsvarar $202,75 gånger 75,8% en RevPAR på $153,68, vilket var en ökning med 2,19% från 2018.

## Skillnaden mellan den genomsnittliga dagliga priset (ADR) och intäkten per tillgängligt rum (RevPAR)

Det genomsnittliga dagspriset (ADR) behövs för att beräkna intäkterna per tillgängligt rum (RevPAR). Det genomsnittliga dagspriset talar om för ett logiföretag hur mycket de tjänar per rum i genomsnitt under en viss dag. Samtidigt mäter RevPAR ett boendes förmåga att fylla sina tillgängliga rum till medelpriset. Om beläggningsgraden inte är 100 % och RevPAR är under ADR, vet en hotelloperatör att den förmodligen kan sänka det genomsnittliga priset per rum för att öka beläggningen.

## Begränsningar för att använda den genomsnittliga dagliga räntan (ADR)

ADR berättar inte hela historien om ett hotells intäkter. Det inkluderar till exempel inte de avgifter som ett logiföretag kan ta ut om en gäst inte dyker upp. Siffran drar inte heller av poster som provisioner och rabatter som erbjuds kunder om det uppstår problem. En fastighets ADR kan öka till följd av prishöjningar, men detta ger begränsad information isolerat. Beläggningen kunde ha sjunkit, vilket gjort att de totala intäkterna blivit lägre.

##Höjdpunkter

- Driftsresultatet för ett hotell eller annan logiverksamhet kan bestämmas med hjälp av ADR.

- Den genomsnittliga dagliga taxan (ADR) mäter den genomsnittliga hyresintäkt som tjänas in för ett upptaget rum per dag.

- Att multiplicera ADR med beläggningsgraden motsvarar intäkterna per tillgängligt rum.

– Hotell eller motell kan öka ADR genom prishantering och kampanjer.
