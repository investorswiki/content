# Broker Booth Support System (BBSS)
## Vad är Broker Booth Support System (BBSS)?

Broker Booth Support System (BBSS) var ett omfattande elektroniskt system som användes av [New York Stock Exchange](/nyse) (NYSE) för att skicka order mellan mäklare och handelsbås på börsens golv. BBSS var ett extremt effektivt sätt att styra orderflödet under de första dagarna av elektronisk handel.

Före dess tillkomst använde mäklare pappersformulär och skickade löpare för att ge order till [golvhandlare](/floortrader) att utföra. BBSS tillät NYSE att hantera dagliga handelsvolymer på över en miljard aktier. 2004 ersattes Broker Booth Support System (BBSS) av NYSE Tradeworks, ett nytt elektroniskt system implementerat av NYSE.

## Förstå Broker Booth Support System (BBSS)

Med tillkomsten av elektronisk handel på 90-talet började börser dra nytta av ny teknik för att förbättra hur finansiell handel gick till. Målet var att göra utförandet av [köp- och säljorder](/marketorder) snabbare, mer exakt och effektivare.

NYSE implementerade Broker Booth Support System (BBSS), ett egenutvecklat elektroniskt orderhanteringssystem som installerades vid alla handelsbås på golvet. [Floor mäklare skrev](/floorbroker) in all information i systemet och lämnade digitala spår och tidsstämplar för att bevara handelsprocessens integritet under hela dagen.

Med varje knapptryckning inloggad i systemet kunde affärer bekräftas och tvister kunde lösas snabbt. Användare kunde skapa skräddarsydda rapporter från all data som sammanställts och organiserats i systemet. Även om BBSS var snabb och effektiv, var den inte felsäker. Det fanns tillfällen under dess tjänstgöring när systemet gick ner, vilket orsakade partiella stopp i orderhanteringen.

Mäklare på golvet fick återgå till den gammaldags metoden att skriva order på papper och köra över dem till handelsbodar tills de tillfälliga tekniska problemen var lösta. Robusta redundansfunktioner implementerades sedan för att förhindra framtida systemproblem.

## Extra BBSS-enheter

Broker Booth Support System (BBSS) var inte den enda moderna implementeringen av elektronisk handel som NYSE implementerade. För ökad effektivitet och bekvämlighet rullade NYSE ut e-Broker handhållna enheter för golvanvändning, så att mäklare kunde vandra runt på golvet fritt utan att behöva sitta i en bås.

NYSE installerade också X-terminaler för att sitta på kundplatser. X-terminalerna, levererade av NYSE men installerade av kunder, var anslutna till BBSS på börsgolvet, vilket gav fjärranvändare hela utbudet av BBSS-tjänster.

## NYSE Tradeworks

2004, efter 10 års användning av Broker Booth Support System (BBSS), beslutade NYSE att uppgradera sitt system med en ny plattform som heter NYSE Tradeworks. NYSE Tradeworks skapades i samarbete med IBM med målet att avskaffa driftstopp och möjliggöra skalbarhet för att hantera ständigt ökande volymer.

Enligt den dåvarande tekniska chefen (CTO) för NYSE, Roger Burkhardt, "satte BBSS standarden för att tillhandahålla effektiva och effektiva orderhanteringsmöjligheter till stöd för medlemsföretagens handelsaktiviteter på golvet. Med Tradeworks kommer NYSE att erbjuda samma funktionalitet, lägga till nya funktioner och ett förbättrat användargränssnitt."

NYSE Tradeworks kunde också tillhandahålla information i [realtid](/real_time),. vilket säkerställde att handlare inte bara gjorde affärer snabbare utan också mer exakt med tanke på den lätta tillgången till information av hög kvalitet. Lehman Brothers, nu nedlagd, var en av de första institutionerna som testade och använde NYSE Tradeworks eftersom det långsamt rullades ut under en 18-månadersperiod.

##Höjdpunkter

- Broker Booth Support System (BBSS) var ett elektroniskt system som användes av New York Stock Exchange (NYSE) för att skicka order mellan mäklare och handelsbås på börsens golv.

- 2004 ersatte NYSE Broker Booth Support System (BBSS) med ett nytt elektroniskt meddelandesystem känt som NYSE Tradeworks.

- Med implementeringen av Broker Booth Support System (BBSS) blev orderflödet snabbare och mycket mer effektivt, vilket möjliggör högre handelsvolymer.

- NYSE Tradeworks tillät samma funktionalitet som Broker Booth Support System (BBSS), men syftade till att avskaffa all driftstopp och möjliggjorde skalbarhet för att hantera ständigt växande handelsvolymer.

- Före användningen av Broker Booth Support System (BBSS) använde mäklare pappersformulär och skickade löpare för att ge order till golvhandlare för utförande.

##FAQ

### Använde alla finansiella börser Broker Booth Support System (BBSS)?

Nej, Broker Booth Support System (BBSS) var ett proprietärt elektroniskt system som endast användes av New York Stock Exchange (NYSE). Andra börser utvecklade och använde sina egna elektroniska handelssystem.

### Använder NYSE fortfarande Broker Booth Support System (BBSS)?

Nej, NYSE använder inte längre Broker Booth Support System (BBSS). Systemet implementerades 1994 och slutade användas 10 år senare då det ersattes med ett nytt system med förbättrad funktionalitet. Det nya systemet kallades NYSE Tradeworks.

### Varför var Broker Booth Support System Important (BBSS)?

Broker Booth Support System (BBSS) var viktigt eftersom det flyttade NYSE in i den elektroniska handelseran, vilket tog bort behovet för mäklare att skriva sina handelsorder på papperslappar för att levereras fysiskt av löpare till golvhandlare. Med BBSS hanterades hela processen elektroniskt, vilket förbättrade effektiviteten och noggrannheten i handeln.
