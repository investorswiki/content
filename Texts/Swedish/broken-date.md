# Trasig datum
## Vad är ett brutet datum?

Brutna datum är en term som används för att beskriva ett icke-standardiserat förfallodatum för alla typer av finansiell leverans. Brutna datum kan förekomma med optioner, terminer, obligationer och andra handelsinstrument.

Trasiga datum kan också kallas [udda datum](/odd-date).

## Förstå brutna datum

Ett brutet datum avser alla icke-standardiserade [förfallodagar](/maturitydate) som tilldelas ett finansiellt instrument. Finansiella instrument med en specificerad livslängd kan ibland avvika från deras slutliga förväntade förfallodag. Avvikelse kan förekomma på grund av helgdagar, veckodagsscheman eller tidpunkter som ställts in av administratören.

## Traktat datum överväganden

Brutna datum kan vara viktiga att känna igen för likviditetssyfte. Ett företag som äger ett finansiellt instrument bör ägna stor uppmärksamhet åt dess faktiska förfallodatum, eftersom brutna datum kan förekomma och instrumentet kanske inte alltid levereras på den exakta förväntade förfallodagen.

I vissa fall kan en emittent också tilldela en löptid som inte följer ett standardiserat schema. Alla typer av icke-standardiserad löptid kan kallas ett brutet datum eller udda datum. En investerare bör vara medveten om det sista förfallodatumet och aldrig anta ett datum baserat på standardiserad livslängd.

Medvetenhet om den slutliga förfallodagen eller [utgångsdatumet](/expirationdate) är viktigt för en investerare eftersom det påverkar handelspriset. För [terminskontrakt](/futurescontract) [kommer](/futurescontract) leveransdatumet att vara detsamma som utgången. För optioner bör en investerare vara medveten om det exakta utgångsdatumet på ett [optionskontrakt](/option),. men de kan vanligtvis utnyttja sin option för leverans när som helst. Obligationer är också ett annat vanligt instrument där ett brutet eller udda datum kan inträffa.

Många finansiella instrument kontrakt är noterade med perioder på en månad, tre månader, sex månader, ett år, två år, etc. Bara för att en period är noterad för ett finansiellt instrument betyder det inte att den kommer att förfalla på det exakta tidsschemat på grund av arbetsdag och andra administrativa faktorer.

## Broken Date Expiration

En investerare som köper ett Bitcoin-terminskontrakt förväntar sig att handeln avslutas den sista fredagen i kontraktsmånaden. Förfallodag vid vilket annat datum som helst skulle anses vara ett brutet datum eller udda datum. Om ett brutet datum inträffar skulle kontraktet lösas på det brutna utgångsdatumet.

Traditionella optionskontrakt på [S&P tredje 500-indexet](/sp500) löper ut på fredagen i utgångsmånaden. Om kontraktet av någon anledning löper ut på ett alternativt datum så skulle det betraktas som ett brutet datum eller udda datum.

##Höjdpunkter

- Ett brutet datum avser varje icke-standardiserad förfallodag som tilldelats ett finansiellt instrument.

- Till exempel, om en option vanligtvis löper ut den tredje fredagen i kontraktsmånaden, men marknaderna är stängda på grund av en helgdag, kan förfallodagen ske den föregående arbetsdagen, vilket skulle vara ett "trasigt datum".

- Investerare bör vara medvetna om eventuella brutna datum eftersom de påverkar priset på det finansiella instrument de handlar med.
