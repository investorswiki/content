# Intraday Intensity Index
## Vad är Intraday Intensity Index?

Intraday Intensity Index är en [volymbaserad](/volume) teknisk indikator som integrerar volym med ett värdepappers pris. Handlare kan använda Intraday Intensity Index för att följa hur toppar och dalar inom dagen rör sig med volymen i jämförelse med föregående dags stängningskurs.

## Förstå Intraday Intensity Index

Intraday Intensity Index har utvecklats av Dave Bostian. Det är en av flera indikatorer som kan användas för att följa hur volymen påverkar ett [värdepappers](/security) pris. Det ger en kontinuerlig volymfokuserad indikator genom att använda ett värdepappers senaste stängning, hög och låg i sin beräkning samtidigt som volymen tas i beräkning.

Täljaren subtraherar värdepapperens högsta och lägsta värde från två gånger det senaste [stängningspriset](/closingprice). Nämnaren multiplicerar volymen gånger skillnaden mellan värdepapprets höga och låga handelspris. Sammantaget blir värdepapperens toppar och dalar under dagen viktiga faktorer som driver indexets värde när de rör sig över stängningskursen med ökad volym.

Generellt, i indexets beräkning, när topparna och dalarna under dagen rör sig över slutkursen med volym, kommer indexet att röra sig kraftigt mot negativt territorium. Således kan Intraday Intensity Index vara till hjälp för att identifiera betydande förändringar i priset orsakade av volym. Eftersom höga volymer vanligtvis drivs av institutionella affärer kan det också anses vara ett sätt att följa hur institutionella investeringar påverkar priset.

Intraday Intensity Index är vanligtvis tillgängligt genom avancerad kartläggningsprogramvara. Det följs vanligtvis i ett diagramfönster under ett [ljusstakediagram](/candlestick). Ofta blir det också ett överlägg mot volym. Många handlare använder Intraday Intensity Index i samband med [Bollinger Bands](/bollingerbands) eller andra kuvertkanaler, eftersom betydande rörelser i Intraday Intensity Index kan hjälpa till att stödja handelsplaner kring ett värdepappers motstånd och stödja trendlinjer.

Intraday Intensity Index är en av några populära indikatorer för att följa volymens effekt på priset. Andra populära [indikatorer](/indicator) inkluderar det [volymvägda](/indicator) [genomsnittspriset](/vwap) (VWAP), det [positiva volymindexet](/pvi) och det [negativa volymindexet](/nvi) och Chaikens pengaflöde.

## Höjdpunkter

- Intraday Intensity Index används för att spåra hur volymen påverkar ett värdepappers pris.

- Intraday Intensity Index används ofta med andra indikatorer för att stödja handelsplaner kring ett värdepappers motstånd och stödja trendlinjer.

– Till exempel när intradags toppar och dalar rör sig över stängningskursen med volym, då kommer indexet att röra sig kraftigt negativt.

- Indexet använder ett värdepappers senaste stängning, hög och låg i sin beräkning samtidigt som den tar hänsyn till volym.
