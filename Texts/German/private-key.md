# Privat Schlüssel
Im Zusammenhang mit Kryptowährung ist ein privater Schlüssel eine Zahl, die es Benutzern ermöglicht, Transaktionen zu signieren und Empfangsadressen zu [generieren](/address).

Kryptowährungen verlassen sich stark auf die Kryptografie mit öffentlichen Schlüsseln, um die Gelder der Benutzer zu schützen. Tatsächlich kommt ihr Name daher.

Bei der Public-Key-Kryptographie generieren Benutzer eine sehr große Zahl (einen privaten Schlüssel), die praktisch unmöglich zu erraten ist. Typischerweise wird diese Nummer als eine lange Folge von Buchstaben und Zahlen dargestellt. Unten ist ein Beispiel für einen Bitcoin-Schlüssel:

> **L2hjTJNhjpUTdAVMArh3UqmnTXEVx6J6Faui8cUXCPpyQMUEkJ54**

Ein Benutzer kann einen öffentlichen Schlüssel erstellen, indem er eine Operation mit dem privaten Schlüssel durchführt. Normalerweise führen wir eine weitere Operation mit dem öffentlichen Schlüssel durch, um eine öffentliche Adresse zu erhalten. Dies ist, was Sie an andere Benutzer austeilen, wenn Sie möchten, dass sie Ihnen Geld senden.

Wichtig ist, dass Sie mit einem privaten Schlüssel auch Daten digital signieren können, z. B. eine Nachricht, die besagt, dass ich X an Adresse Y bezahle. Letztendlich ist das alles, was eine Transaktion ist – eine signierte Erklärung auf der Blockchain, die Coins einer neuen Adresse zuweist.

Wenn Sie eine Nachricht (dh eine Transaktion) mit Ihrem privaten Schlüssel signieren, können andere Ihren entsprechenden öffentlichen Schlüssel verwenden, um seine Authentizität zu überprüfen. Sie verwenden Ihren öffentlichen Schlüssel, um zu überprüfen, ob die Nachricht wirklich von Ihnen signiert wurde, und um sicherzustellen, dass sie danach nicht geändert wurde.

Beachten Sie, dass es notwendig (und absolut sicher) ist, Ihren öffentlichen Schlüssel mit anderen zu teilen. Aber Sie sollten niemals Ihren privaten Schlüssel preisgeben. Wenn jemand Zugriff darauf erhält, kann er Ihr Geld ausgeben, indem er Transaktionen in Ihrem Namen unterzeichnet.

Heutzutage sind Benutzer selten privaten Schlüsseln wie dem oben gezeigten Bitcoin ausgesetzt. Die meisten Kryptowährungs - [Wallets](/bitcoin-wallet) unterstützen heute [Seed Phrases](/seed-phrase),. bei denen es sich um menschenlesbare Backups handelt, die eine praktisch unbegrenzte Menge an privaten Schlüsseln wiederherstellen können.

Die Geheimhaltung Ihrer privaten Schlüssel ist von größter Bedeutung. Es gibt verschiedene Möglichkeiten, sie vor neugierigen Blicken zu schützen, obwohl viele zustimmen würden, dass Hardware Wallets die sicherste Option sind. Für den täglichen Gebrauch sind Software-Wallets vielleicht eine bequemere Alternative.

## Höhepunkte

- Eine Kryptowährungs-Wallet besteht aus einer Reihe von öffentlichen Adressen und privaten Schlüsseln. Jeder kann Kryptowährung in einer öffentlichen Adresse hinterlegen, aber ohne den entsprechenden privaten Schlüssel können keine Gelder von einer Adresse entfernt werden.

- Ein privater Schlüssel ist eine große, zufällig generierte Zahl mit Hunderten von Ziffern. Der Einfachheit halber werden sie normalerweise als Zeichenfolgen aus alphanumerischen Zeichen dargestellt.

- Private Schlüssel stellen die endgültige Kontrolle und das Eigentum an der Kryptowährung dar. Es ist von entscheidender Bedeutung, zu verhindern, dass die privaten Schlüssel verloren gehen oder kompromittiert werden.

- Ein privater Schlüssel ist eine Geheimzahl, die in der Kryptographie und Kryptowährung verwendet wird.

## FAQ

### Wie funktionieren private Schlüssel?

Ein privater Schlüssel ist eine extrem große Zahl, die in der Kryptographie verwendet wird, ähnlich wie ein Passwort. Private Schlüssel werden verwendet, um digitale Signaturen zu erstellen, die leicht verifiziert werden können, ohne den privaten Schlüssel preiszugeben. Private Schlüssel werden auch bei Kryptowährungstransaktionen verwendet, um den Besitz einer Blockchain-Adresse anzuzeigen.

### Sollten Sie einer Custodial Wallet vertrauen?

Eine Custodial Wallet ist ein Drittanbieterdienst, der es Benutzern ermöglicht, Kryptowährung ähnlich wie bei einer Bank aufzubewahren. Auf diese Weise können Benutzer die komplizierte Speicherung privater Schlüssel überspringen und sich stattdessen auf das technologische Know-how des Unternehmens verlassen, das den Dienst anbietet. Es gibt jedoch Kompromisse. Depot-Wallets sind oft Ziele für Hacker oder Phishing-Betrug, und sie können auch von Justizbehörden beschlagnahmt oder eingefroren werden. Die beste Lösung besteht darin, festzustellen, welche Art von Brieftasche zu Ihrer individuellen Risikotoleranz und Ihren technologischen Fähigkeiten passt.

### Wie speichert man Private Keys am besten?

Private Schlüssel können auf Computern oder Mobiltelefonen, USB-Laufwerken, einer speziellen Hardware-Wallet oder sogar auf einem Blatt Papier gespeichert werden. Die ideale Form der Speicherung hängt davon ab, wie oft Sie Ihre Kryptowährung verwenden möchten. Ein passwortgeschütztes Mobiltelefon oder ein Computer ist die bequemste Art, Kryptowährung für den täglichen Gebrauch aufzubewahren. Für die langfristige oder „kalte“ Aufbewahrung sollten private Schlüssel immer offline aufbewahrt werden, idealerweise auf Geräten, die noch nie mit dem Internet in Berührung gekommen sind. Sogar Drucker können kompromittiert werden. Hardware-Wallets können Cold Storage erleichtern, indem sie Transaktionen so signieren, dass die privaten Schlüssel nicht gefährdet werden.
