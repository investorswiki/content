# Einheitliche Zahlungsschnittstelle (UPI)
## Was ist eine einheitliche Zahlungsschnittstelle (UPI)?

Ein Unified Payment Interface (UPI) ist eine Smartphone-Anwendung, mit der Benutzer Geld zwischen Bankkonten überweisen können. Es ist ein mobiles Ein-Fenster-Zahlungssystem, das von der National Payments Corporation of India (NPCI) entwickelt wurde. Dadurch entfällt die Notwendigkeit, [Bankdaten](/bank) oder andere sensible Informationen jedes Mal einzugeben, wenn ein [Kunde](/customer) eine Transaktion initiiert.

Das Unified Payment Interface ist ein Echtzeit-Zahlungssystem. Es wurde entwickelt, um Peer-to-Peer- [Überweisungen zwischen Banken](/transfer) durch einen einzigen Zwei-Klick-Faktor-Authentifizierungsprozess zu ermöglichen. Die Schnittstelle wird von der [Reserve Bank of India (RBI)](/rbi),. Indiens Zentralbank, reguliert. Es funktioniert durch die Überweisung von Geld zwischen zwei Bankkonten zusammen mit einer mobilen Plattform.

Das System soll eine sichere Methode für den Geldtransfer zwischen zwei Parteien sein und macht Transaktionen mit physischem [Bargeld](/cash) oder über eine Bank überflüssig. Das Pilotsystem wurde am 11. April 2016 in Indien gestartet. Banken im ganzen Land haben im August 2016 damit begonnen, ihre Schnittstelle hochzuladen.

## So funktioniert Unified Payments Interface (UPI).

UPI verwendet bestehende Systeme wie Immediate Payment Service (IMPS) und Aadhaar Enabled Payment System (AEPS), um eine nahtlose Abrechnung über Konten hinweg zu gewährleisten. Es erleichtert Push- (Bezahlen) und Pull- (Empfangen) Transaktionen und funktioniert sogar für [Over-the-Counter-](/otc) oder Barcode-Zahlungen sowie für mehrere wiederkehrende Zahlungen wie Stromrechnungen, Schulgebühren und andere Abonnements.

> Laut NPCI nutzten im Februar 2019 134 Banken die Schnittstelle. Mehr als [Rs](/inr). 270 Milliarden wurden im selben Monat über die UPI ausgetauscht.

>

Sobald eine einzige Kennung eingerichtet ist, ermöglicht das System die Bereitstellung mobiler Zahlungen ohne die Verwendung von Kredit- oder Debitkarten, Net-Banking oder die Notwendigkeit, Kontodaten einzugeben. Dies würde nicht nur eine größere Sicherheit sensibler Informationen gewährleisten, sondern auch Personen mit Bankkonten über [Smartphones verbinden](/smartphone),. um problemlose Transaktionen durchzuführen.

Insgesamt impliziert UPI weniger Bargeldtransaktionen und reduziert möglicherweise die Bevölkerung [ohne Bankkonto .](/unbanked)

## Geld senden vs. Geld empfangen

Das Senden von Geld über den UPI wird als „Push“ bezeichnet. Um Geld zu senden, loggt sich der Benutzer in die Schnittstelle ein und wählt die Option Geld senden/Zahlung. Nach Eingabe der virtuellen ID des Empfängers und des gewünschten Betrags wählt dieser das Konto aus, von dem das Geld abgebucht werden soll. Der Benutzer gibt dann eine spezielle [persönliche Identifikationsnummer (PIN)](/personal-identification-number) ein und erhält eine Bestätigung.

Das Empfangen von Geld durch das System wird als „Pull“ bezeichnet. Sobald sich die Benutzerin im System angemeldet hat, wählt sie die Option zum Geldsammeln aus. Der Benutzer muss dann die virtuelle ID für den Überweisenden, den einzuziehenden Betrag und das Konto eingeben, auf das er das Geld einzahlen wird. Anschließend geht eine Nachricht mit der Zahlungsaufforderung an den Zahler. Entscheidet er sich für die Zahlung, gibt er seine UPI-PIN ein, um die Transaktion zu autorisieren.

Sobald die Übertragung abgeschlossen ist, erhalten sowohl der Absender als auch der Empfänger eine Bestätigung per SMS auf ihr Smartphone.

## Beispiel für von UPI angebotene Dienstleistungen

UPI bietet eine Reihe wichtiger Funktionen. Benutzer können auf Salden und Transaktionsverläufe zugreifen sowie Geld senden und empfangen. Um Geld zu senden, benötigen Benutzer eine Kontonummer, den Indian Financial System Code (oder IFSC, ein alphanumerischer Code, der elektronische Überweisungen erleichtert), die Handynummer des Empfängers und eine virtuelle ID oder Aadhaar-Nummer (die wie eine Social Sicherheitsnummer).

## Höhepunkte

- Ein Unified Payment Interface (UPI) ist eine Smartphone-Anwendung für Bankgeschäfte in Indien.

- Die Schnittstelle wird von der Reserve Bank of India (RBI), Indiens Zentralbank, reguliert und Gelder werden zwischen Konten gesendet.

- Diese App eliminiert die Notwendigkeit, Bankdaten oder andere sensible Informationen jedes Mal einzugeben, wenn ein Kunde eine Transaktion initiiert, was es zu einem sicheren Weg zum Bankgeschäft macht.
