# Spezielle Situation
Wirklich jedes außergewöhnliche Ereignis, das den Kurs einer Aktie beeinflussen könnte, wie die Ankündigung einer Fusion. Tatsächlich gibt es da draußen ganze Klassen von Anlegern, die bestimmte Aktien beobachten und nur darauf warten, dass etwas passiert.

## Höhepunkte

- Es gibt Fonds für Sondersituationen, die versuchen, solche Ereignisse auszunutzen.

- Eine Sondersituation ist ein einmaliges Ereignis, das sich auf eine Aktie oder einen anderen Vermögenswert auswirkt.

- Eine beliebige Anzahl positiver oder negativer Ereignisse kann dazu führen, dass der Kurs einer Aktie explodiert oder ihren Kurs drückt.
