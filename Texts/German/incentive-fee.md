# Incentive-Gebühr
## Was ist eine Incentive-Gebühr?

Eine Erfolgsgebühr ist eine Gebühr, die von einem [Fondsmanager](/fundmanager) auf der Grundlage der Wertentwicklung eines Fonds über einen bestimmten Zeitraum erhoben wird. Die Gebühr wird in der Regel mit einer Benchmark verglichen. Beispielsweise kann ein Fondsmanager eine Anreizgebühr erhalten, wenn sein Fonds den [S&P 500](/sp500) Index über ein Kalenderjahr übertrifft, und kann mit zunehmender Outperformance steigen.

## Leistungsprämien verstehen

Eine Incentive-Gebühr, auch Performance-Gebühr genannt, ist in der Regel an die Vergütung eines Managers und sein Leistungsniveau, genauer gesagt an seine finanzielle Rendite, gebunden. Solche Gebühren können auf verschiedene Weise berechnet werden. Bei separaten Konten kann die Gebühr beispielsweise an die Veränderung der [realisierten und nicht realisierten](/realizedprofit) Nettogewinne oder der erzielten Nettoerträge gekoppelt werden.

Bei [Hedge-Fonds](/hedgefund),. bei denen Anreizgebühren häufiger sind, wird die Gebühr im Allgemeinen auf der Grundlage des Wachstums des [Nettoinventarwerts](/nav) (NAV) des Fonds oder des Kontos berechnet. Für Hedgefonds ist eine Anreizgebühr von 20 % obligatorisch.

Obwohl sie selten sind, verwenden einige Fonds eine „ [Stoßdämpfer](/demandshock) “-Struktur, bei der ein Fondsmanager vor dem Anleger für eine Abwärtsbewegung der Performance bestraft wird.

In den Vereinigten Staaten wird die Verwendung von Anreizgebühren durch [registrierte Anlageberater](/ria) (RIAs) durch den Investment Advisers Act von 1940 abgedeckt und darf nur unter besonderen Bedingungen erhoben werden. Manager, die US-Pensionsfonds als Leistungsprämien verwenden möchten, müssen sich an den Employee Retirement Income Security Act (ERISA) halten.

## Beispiel für Anreizgebühren

Ein Investor nimmt eine 10-Millionen-Dollar-Position bei einem Hedgefonds ein und nach einem Jahr ist der NAV um 10 % (oder 1 Million US-Dollar) gestiegen, wodurch diese Position 11 Millionen US-Dollar wert ist. Der Manager wird 20 % dieser 1 Million Dollar Wechselgeld oder 200.000 Dollar verdient haben. Diese Gebühr reduziert den NAV auf 10,8 Millionen US-Dollar, was einer Rendite von 8 % entspricht, unabhängig von anderen Gebühren.

Der höchste Wert eines Fonds über einen bestimmten Zeitraum wird als [High-Water-Mark bezeichnet](/highwatermark). Im Allgemeinen fällt keine Anreizgebühr an, wenn ein Fonds so hoch fällt. Manager erheben in der Regel nur dann eine Gebühr, wenn sie die Höchstgrenze überschreiten.

Eine [Hürde](/hurdlerate) wäre ein vorgegebenes Renditeniveau, das ein Fonds erfüllen muss, um eine Anreizgebühr zu verdienen. Hürden können die Form eines Indexes oder einer Menge mit einem vorher festgelegten Prozentsatz haben. Wenn beispielsweise ein NAV-Wachstum von 10 % einer Hürde von 3 % unterliegt, würde eine Anreizgebühr nur auf die Differenz von 7 % erhoben. Hedgefonds waren in den letzten Jahren so beliebt, dass heute weniger von ihnen Hürden anwenden als in den Jahren nach der Großen Rezession.

## Besondere Erwägungen für Incentive-Gebühren

Kritiker von Anreizgebühren wie Warren Buffett behaupten, dass ihre verzerrte Struktur – bei der ein Manager an den Gewinnen, aber nicht an den Verlusten eines Fonds beteiligt wird – Manager nur dazu ermutigt, übermäßige Risiken einzugehen, um die Renditen zu drosseln.

## Höhepunkte

- Kritiker dieser Gebühren schlagen vor, dass sie Manager ermutigen, übermäßige Risiken einzugehen, um die Renditen zu steigern.

- Ein Fondsmanager erhält möglicherweise eine Leistungsprämie, wenn sich ein Fonds über einen bestimmten Zeitraum gut entwickelt.

- Die Höhe der Gebühr kann auf den realisierten Nettogewinnen, den nicht realisierten Nettogewinnen oder den erzielten Nettoerträgen basieren.

- Eine Anreizgebühr von 20 % ist typisch für Hedgefonds.
