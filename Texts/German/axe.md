# Axt
## Was ist eine Axt?

Eine Axt (oder „Axe to grind“) ist das Interesse, das ein Händler beim Kauf oder Verkauf eines Wertpapiers zeigt, das normalerweise bereits in den Büchern steht. Wenn ein Händler eine Long-Position hält, aber kurzfristige Bedenken hat, kann die Axt dieses Händlers in Richtung kurzfristiger [Put -](/putoption) Optionen von Bedeutung sein. Wenn ein Händler einem Zinsanstiegsrisiko ausgesetzt ist, hat er möglicherweise eine Axt, um sich gegen dieses Risiko [abzusichern .](/hedge)

Oft hält ein Händler mit einer Axt diese Informationen geheim. Denn wenn andere Marktteilnehmer die eigenen Beweggründe erkennen, können sie sich diese Informationen zunutze machen, indem sie ungünstige Preise anbieten oder bestimmte Geschäfte zurückhalten, um die Situation auszunutzen.

## Eine Axt verstehen

Der Begriff "Axt" leitet sich von dem Ausdruck "axe to grind" ab, was bedeutet, einen Hintergedanken oder ein egoistisches Ziel zu haben. Der Ausdruck bedeutete historisch, eine Beschwerde mit jemandem zu haben, besonders wenn man das Bedürfnis verspürt, Vergeltung zu suchen. Der Ausdruck stammt wahrscheinlich vom Schärfen einer Axt mit einer Schleifscheibe mit der Absicht (in dieser Definition), sich an jemandem zu rächen, indem man ihn verstümmelt oder tötet.

Händler verwenden häufig den Begriff „Axt“, um das besondere Interesse einer Person am Kauf oder Verkauf eines Wertpapiers, das sich bereits in ihrem Bestand befindet, oder an der Absicherung dagegen darzustellen. Der Begriff wurde in der Vergangenheit verwendet, um Anleihebestände zu bezeichnen, aber Händler haben die Verwendung auf alle [Wertpapiere ausgeweitet](/security). Im Gespräch wird der Begriff oft verwendet, um über den Plan eines Händlers in Bezug auf ein Wertpapier zu spekulieren, das er hält.

Ax sollte nicht mit „ax“ verwechselt werden, einem [Market Maker](/marketmaker),. der für die Preisbewegung eines bestimmten Wertpapiers von zentraler Bedeutung ist.

## Wie eine Axt in der Praxis verwendet wird

Der Begriff "Axt" kann auf viele verschiedene Arten verwendet werden, weshalb es wichtig ist, den Kontext des Gesprächs zu berücksichtigen.

Angenommen, ein Händler hat eine große Position in einem bestimmten Wertpapier. Wenn dieser Händler nach Angeboten sucht, um den Einsatz zu verkaufen, kann der Händler, der das Angebot bereitstellt, im Nachteil sein, wenn er nicht weiß, dass der erste Händler in Bezug auf das Wertpapier eine Axt hat. Der zweite Händler kann fragen: „Haben sie eine Axt an diesem Wertpapier?“ was bedeutet "Haben sie Pläne, dieses Wertpapier zu verkaufen?"

Händler können den Begriff auch verwenden, um Wertpapiere darzustellen, die sich auf die von ihnen gehaltenen Wertpapiere beziehen. Zum Beispiel kann ein Trader eine [Long-Position halten](/long) und eine Axt an Put-Optionen haben, wenn er wegen der kurzfristigen Aussichten der Aktie nervös ist.

Der Besitz einer Axt wird oft geheim gehalten, weil das Wissen um diese Informationen von anderen Marktteilnehmern genutzt werden kann, um die Situation zu ihrem eigenen Vorteil und auf Kosten des Axtbesitzers auszunutzen. Allerdings können sich Trader mit guter Beziehung direkt gegenseitig fragen, ob sie eine bestimmte Axt haben, in der Hoffnung, dass die Axt(en) des anderen Traders ihren eigenen entgegengesetzt sind – auf diese Weise können sie einen Trade oder Trades miteinander in einem beeinflussen gegenseitig vorteilhafte Weise.

## Höhepunkte

- Eine Axt (oder „axe to grind“) bezieht sich auf eine gewünschte Position, die ein Trader einnehmen möchte.

- Im Gespräch wird der Begriff oft verwendet, um über die Pläne eines Händlers in Bezug auf ein gehaltenes Wertpapier zu spekulieren.

- Wenn ein Trader eine Axt zu schleifen hat, kann die Geheimhaltung dieser Informationen andere daran hindern, einen Vorteil daraus zu ziehen.

- Ax wurde in der Vergangenheit als Referenz für Anleihenmärkte verwendet, umfasst jetzt aber alle Arten von Wertpapieren.
