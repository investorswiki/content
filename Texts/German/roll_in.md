# Einrollen
## Was ist Roll-In?

Ein Roll-In bezieht sich auf die Aktion, bestimmte Gebühren in eine [Hypothek einzubeziehen](/mortgage),. anstatt sie separat zu zahlen. Viele Kreditnehmer rechnen bestimmte Gebühren in ihre Hypotheken ein, um hohe Kosten im Voraus zu vermeiden. Sie können sich dafür entscheiden, weil sie zu Beginn des Darlehens einfach nicht über die Mittel verfügen, oder weil sie die Gebühren lieber [amortisieren möchten](/amortization),. indem sie kleinere Beträge über einen längeren Zeitraum zahlen.

Viele Kreditnehmer wälzen die Gebühren notgedrungen in eine Hypothek um. Wenn sie jedoch über die Mittel verfügen, um die Gebühren im Voraus zu bezahlen, sparen sie dadurch normalerweise eine erhebliche Menge Geld. Dies liegt daran, dass diese Gebühren zum [Hauptbetrag der Hypothek hinzugefügt werden](/principal),. auf die der Käufer dann für eine bestimmte Anzahl von Jahren Zinsen zahlt.

## Roll-In verstehen

### Gebühren, die einfließen können

Roll in kann austauschbar mit „to roll“ oder „rolling“ verwendet werden. Der Prozess kann für eine Vielzahl unterschiedlicher Gebühren gelten. Kreditgebühren, wie z. B. [Kreditvergabegebühren](/mortgage_originator),. können im Allgemeinen in eine Hypothek einfließen. Regierungsgebühren, die je nach Region variieren, können ebenfalls einfließen. Dazu können Anmeldegebühren und Verwaltungskosten gehören und bestimmte Steuern.An Immobilientransaktionen sind in der Regel Rechtsanwälte beteiligt, deren Gebühren auch in eine Hypothek umgewandelt werden können.

### "Roll-In" bei der Refinanzierung

Wenn ein Kreditnehmer eine Hypothek [refinanziert](/refinance),. ist die Refinanzierung oft mit bestimmten Gebühren verbunden. Wenn der Kreditnehmer über genügend Eigenkapital im Haus verfügt, kann der Kreditgeber zulassen, dass die Kosten der Refinanzierung in die neue Hypothek gerollt werden.

## Kosten, die nicht berücksichtigt werden können

Rolling-in-Kosten können dabei helfen, indem sie die Vorabkosten senken. Allerdings können nicht alle Kosten im Zusammenhang mit dem Kauf eines Eigenheims in die Hypothek gerollt werden. So genannte Prepaid- Kosten müssen im Voraus bezahlt werden und dürfen nicht eingerollt werden. Dies liegt häufig daran, dass Prepaid-Kosten auf ein [Treuhandkonto](/prepaidexpense) [gehen müssen](/escrow).

Vorauszahlungen können Grundsteuern, Hausratversicherungen und private [Hypothekenversicherungen umfassen](/mortgage-insurance). Sie werden als Prepaid bezeichnet, weil sie bezahlt werden, bevor sie tatsächlich fällig sind. Beispielsweise dürfen Grundsteuern nur einmal pro Jahr an die Gemeinde eines Hauses fällig werden. Ein Kreditgeber wird diese Steuern jedoch rechtzeitig vor diesem Datum einziehen und die Zahlung zur Zahlung bei Fälligkeit auf einem Treuhandkonto aufbewahren. Die Hinterlegung dieses Geldes schützt den Kreditgeber für den Fall, dass der Kreditnehmer in Zukunft mit Zahlungen in [Verzug gerät .](/default2)

## Höhepunkte

- Zu den Gebührenarten, die hinzugerechnet werden können, gehören Kreditgebühren, wie z. B. Kreditvergabegebühren; Regierungsgebühren wie Anmeldegebühren, Verwaltungskosten und bestimmte Steuern; und Anwaltskosten.

- Ein Roll-In bezieht sich auf die Aktion, bestimmte Gebühren in eine Hypothek einzubeziehen, anstatt sie separat zu zahlen.

- Viele Kreditnehmer rechnen bestimmte Gebühren in ihre Hypotheken ein, um hohe Kosten im Voraus zu vermeiden.
