# T-Konto
## Was ist ein T-Konto?

Ein T-Konto ist ein informeller Begriff für eine Reihe von [Finanzunterlagen](/accounting-records),. die die [doppelte Buchführung verwenden](/double-entry). Der Begriff beschreibt das Aussehen der Buchhaltungseinträge. Zuerst wird ein großer Buchstabe T auf eine Seite gezeichnet. Der Titel des Kontos wird dann direkt über der oberen horizontalen Linie eingegeben, während darunter links die [Belastungen und rechts die](/debit) [Gutschriften aufgeführt](/credit) werden, getrennt durch die vertikale Linie des Buchstabens T.

Ein T-Konto wird auch Sachkonto genannt.

## T-Konto verstehen

In [der doppelten Buchführung](/double-entry),. einer weit verbreiteten [Buchführungsmethode](/accountingmethod),. werden alle Finanztransaktionen so betrachtet, dass sie mindestens zwei Konten eines Unternehmens betreffen. Ein Konto erhält eine Belastungsbuchung, während das zweite eine Gutschriftsbuchung erhält, um jede auftretende Transaktion aufzuzeichnen.

Die Gutschriften und Belastungen werden in einem [Hauptbuch erfasst](/generalledger),. wo alle Kontostände übereinstimmen müssen. Das optische Erscheinungsbild des Hauptbuchjournals von Einzelkonten ähnelt einer T-Form, weshalb ein Hauptbuchkonto auch als T-Konto bezeichnet wird.

Ein T-Konto ist die grafische Darstellung eines Hauptbuchs, das die Transaktionen eines Unternehmens aufzeichnet. Es besteht aus Folgendem:

- Ein Kontotitel in der oberen horizontalen Zeile des T

- Eine Sollseite auf der linken Seite

- Eine Habenseite auf der rechten Seite

## T-Kontoaufzeichnung

Bei verschiedenen Konten können Belastungen und Gutschriften zu Erhöhungen oder Verringerungen führen, aber die Belastungsseite muss immer links von der T-Kontur liegen und die Gutschriften müssen auf der rechten Seite erfasst werden. Die Hauptbestandteile der [Bilanz](/balancesheet) – [Vermögenswerte](/asset),. [Verbindlichkeiten](/liability) und [Eigenkapital](/shareholdersequity) (SE) – können nach jeder Finanztransaktion in einem T-Konto ausgewiesen werden.

Die Belastungsbuchung eines Aktivkontos bedeutet eine Erhöhung des Kontos, während die rechte Seite des Aktiv-T-Kontos eine Verringerung des Kontos darstellt. Das bedeutet, dass ein Unternehmen, das beispielsweise Bargeld erhält, das Vermögenskonto belastet, bei einer Barauszahlung jedoch dem Konto gutschreibt.

Die Verbindlichkeiten und das Eigenkapital (SE) in einem T-Konto haben Einträge auf der linken Seite, um eine Verringerung der Konten widerzuspiegeln, und jede Gutschrift bedeutet eine Erhöhung der Konten. Ein Unternehmen, das [Aktien](/issuedshares) im Wert von 100.000 $ ausgibt, wird auf seinem T-Konto eine Erhöhung seines Vermögenskontos und eine entsprechende Erhöhung seines Eigenkapitalkontos aufweisen:

T-Konten können auch verwendet werden, um Veränderungen in der [Gewinn- und Verlustrechnung zu erfassen](/incomestatement),. wo Konten für [Einnahmen](/revenue) [(](/revenue) Gewinne [)](/profit) und [Ausgaben](/expense) (Verluste) eines Unternehmens eingerichtet werden können. Bei den Ertragskonten verringern Belastungsbuchungen das Konto, während eine Gutschrift das Konto erhöht. Andererseits erhöht eine Belastung ein Ausgabenkonto, und eine Gutschrift verringert es.

## T-Account-Vorteile

T-Konten werden üblicherweise zur Erstellung von [Ausgleichsbuchungen verwendet](/adjusting-journal-entry). Das Matching-Prinzip in der [periodengerechten Rechnungslegung](/accrualaccounting) besagt, dass alle Ausgaben mit den während der Periode erzielten Einnahmen übereinstimmen müssen. Das T-Konto leitet Buchhalter an, was sie in ein Hauptbuch eintragen müssen, um einen Anpassungssaldo zu erhalten, damit die Einnahmen den Ausgaben entsprechen.

Ein Geschäftsinhaber kann T-Konten auch verwenden, um Informationen zu extrahieren, wie z. B. die Art einer Transaktion, die an einem bestimmten Tag stattgefunden hat, oder den Saldo und die Bewegungen jedes Kontos.

## Höhepunkte

- Ein T-Konto ist ein informeller Begriff für eine Reihe von Finanzunterlagen, die die doppelte Buchführung verwenden.

- Der Kontotitel erscheint direkt über dem T. Darunter werden Belastungen auf der linken Seite und Gutschriften auf der rechten Seite aufgezeichnet, getrennt durch eine Linie.

- Das T-Konto leitet Buchhalter an, was sie in ein Hauptbuch eintragen müssen, um einen Ausgleichssaldo zu erhalten, damit die Einnahmen den Ausgaben entsprechen.

- Es wird als T-Konto bezeichnet, weil die Buchhaltungseinträge in einer Art und Weise angeordnet sind, die einer T-Form ähnelt.
