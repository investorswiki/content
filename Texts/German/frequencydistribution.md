# Häufigkeitsverteilung
## Was ist eine Häufigkeitsverteilung?

Eine Häufigkeitsverteilung ist eine grafische oder tabellarische Darstellung, die die Anzahl der Beobachtungen innerhalb eines bestimmten Intervalls anzeigt. Die Intervallgröße hängt von den analysierten Daten und den Zielen des Analysten ab. Die Intervalle müssen sich gegenseitig ausschließen und vollständig sein. Häufigkeitsverteilungen [werden](/distribution) typischerweise in einem statistischen Kontext verwendet. Im Allgemeinen können Häufigkeitsverteilungen mit der Darstellung einer [Normalverteilung in Verbindung gebracht werden](/normaldistribution).

## Eine Häufigkeitsverteilung verstehen

Als [statistisches](/statistics) Werkzeug bietet eine Häufigkeitsverteilung eine visuelle Darstellung der Verteilung von Beobachtungen innerhalb eines bestimmten Tests. Analysten verwenden häufig eine Häufigkeitsverteilung, um die in einer Stichprobe gesammelten Daten zu visualisieren oder zu veranschaulichen. Beispielsweise kann die Körpergröße von Kindern in mehrere verschiedene Kategorien oder Bereiche unterteilt werden.

Bei der Messung der Größe von 50 Kindern sind einige groß und andere klein, aber es besteht eine hohe Wahrscheinlichkeit einer höheren Häufigkeit oder Konzentration im mittleren Bereich. Die wichtigsten Faktoren für die Datenerhebung sind, dass sich die verwendeten Intervalle nicht überschneiden dürfen und alle möglichen Beobachtungen enthalten müssen.

## Visuelle Darstellung einer Häufigkeitsverteilung

Sowohl Histogramme als auch Balkendiagramme bieten eine visuelle Anzeige unter Verwendung von Säulen, wobei die y-Achse die Häufigkeitszählung darstellt und die x-Achse die zu messende Variable darstellt. Bei der Größe der Kinder ist beispielsweise die y-Achse die Anzahl der Kinder und die x-Achse die Größe. Die Spalten stellen die Anzahl der beobachteten Kinder dar, wobei die Körpergröße in jedem Intervall gemessen wurde.

Im Allgemeinen zeigt ein [Histogrammdiagramm](/histogram) normalerweise eine Normalverteilung, was bedeutet, dass die Mehrheit der Vorkommnisse in die mittleren Spalten fällt. Häufigkeitsverteilungen können ein wichtiger Aspekt bei der grafischen Darstellung von Normalverteilungen sein, die Beobachtungswahrscheinlichkeiten geteilt durch [Standardabweichungen zeigen](/standarddeviation).

Häufigkeitsverteilungen können als Häufigkeitstabelle, Histogramm oder Balkendiagramm dargestellt werden. Nachfolgend ein Beispiel einer Häufigkeitsverteilung als Tabelle.

## Häufigkeitsverteilung im Handel

Häufigkeitsverteilungen werden in der Welt der Investitionen nicht häufig verwendet; Händler, die Richard D. Wyckoff folgen, einem bahnbrechenden Händler des frühen 20. Jahrhunderts, verwenden jedoch einen Trading-Ansatz, der die Häufigkeitsverteilung beinhaltet.

Investmenthäuser verwenden immer noch den Ansatz, der viel Übung erfordert, um Trader zu unterrichten. Das Häufigkeitsdiagramm wird als Point-and-Figure-Diagramm bezeichnet und wurde aus der Notwendigkeit heraus erstellt, dass Parketthändler [Preisbewegungen](/floortrader) zur Kenntnis nehmen und Trends erkennen müssen.

Die y-Achse ist die gemessene Variable und die x-Achse ist die Häufigkeitszählung. Jede Änderung der Kursbewegung wird mit X und Os gekennzeichnet. Trader interpretieren es als Aufwärtstrend, wenn drei X auftauchen; In diesem Fall hat die Nachfrage das Angebot übertroffen. In der umgekehrten Situation, wenn das Diagramm drei O zeigt, zeigt dies an, dass das Angebot die Nachfrage übertroffen hat.

## Das Endergebnis

Eine Häufigkeitsverteilung wird verwendet, um die Anzahl der Beobachtungen innerhalb eines bestimmten Intervalls anzuzeigen. Diese Methode wird zwar nicht immer häufig beim Investieren verwendet, wird aber immer noch von einigen Händlern verwendet. In diesem Fall wird das Frequenzdiagramm als Point-and-Figure-Diagramm bezeichnet und wird verwendet, um Trends durch die Beobachtung der Preisbewegung zu identifizieren.

## Höhepunkte

- Häufigkeitsverteilungen sind besonders nützlich für Normalverteilungen, die die Beobachtungen von Wahrscheinlichkeiten geteilt durch Standardabweichungen zeigen.

- Eine Häufigkeitsverteilung in der Statistik ist eine Darstellung, die die Anzahl der Beobachtungen innerhalb eines bestimmten Intervalls anzeigt.

- Die Darstellung einer Häufigkeitsverteilung kann zur besseren Übersichtlichkeit grafisch oder tabellarisch erfolgen.

- Im Finanzbereich verwenden Trader Häufigkeitsverteilungen, um Kursbewegungen zu beobachten und Trends zu erkennen.

## FAQ

### Welche Arten von Häufigkeitsverteilungen gibt es?

Die Arten der Häufigkeitsverteilung sind gruppierte Häufigkeitsverteilung, nicht gruppierte Häufigkeitsverteilung, kumulative Häufigkeitsverteilung, relative Häufigkeitsverteilung und relative kumulative Häufigkeitsverteilung.

### Was ist die Bedeutung einer Häufigkeitsverteilung?

Eine Häufigkeitsverteilung ist ein Mittel, um eine große Datenmenge zu organisieren. Es nimmt Daten aus einer Population basierend auf bestimmten Merkmalen und organisiert die Daten in einer Weise, die für eine Person verständlich ist, die Annahmen über eine bestimmte Population treffen möchte.

### Wie kann ich eine Häufigkeitsverteilung konstruieren?

Um eine Häufigkeitsverteilung zu konstruieren, notieren Sie zuerst die spezifischen Klassen, die durch Intervalle in einer Spalte bestimmt werden, und summieren Sie dann die Zahlen in jeder isolierten Kategorie basierend darauf, wie oft sie auftaucht. Die Häufigkeit kann dann in der zweiten Spalte notiert werden.
