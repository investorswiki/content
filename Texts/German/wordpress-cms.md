# Wordpress (CMS)
## Was ist WordPress?

WordPress ist ein beliebtes Open-Source-Content-Management-System (CMS). Obwohl es ursprünglich hauptsächlich mit persönlichen Blogs in Verbindung gebracht wurde, wird es seitdem für eine Vielzahl von Websites verwendet, einschließlich professioneller Publikationen und [E-Commerce-](/ecommerce) Plattformen.

Seit seiner Einführung im Jahr 2003 ist WordPress auf etwa ein Drittel der Websites weltweit angewachsen. Heute ist es die Heimat einer großen Entwickler-Community, die über 50.000 Plugins produziert hat.

## So funktioniert WordPress

Benutzer von WordPress können aus zwei Basisversionen des Dienstes wählen. Die erste davon ist die „gehostete“ Version von WordPress, die unter WordPress.com zugänglich ist. Diese Version ermöglicht es Benutzern, eine Website zu starten, die auf den eigenen Servern von WordPress gehostet wird. Daher wird es oft von technisch nicht versierten Benutzern bevorzugt, die möglicherweise die Komplexität des Hostens einer Website auf ihrem eigenen privaten Server vermeiden möchten. Als zusätzlichen Vorteil wird die gehostete Version von WordPress automatisch auf die neueste Version der WordPress-Software aktualisiert, was dazu beitragen kann, potenzielle Sicherheitsverletzungen zu mindern.

Für fortgeschrittene Benutzer ist es auch möglich, die neueste Version von WordPress einfach von WordPress.org herunterzuladen. Bei Auswahl dieser Option muss der Benutzer selbst Vorkehrungen treffen, um die Software entweder auf seinem eigenen privaten Server oder bei einem Hosting [-Drittanbieter zu](/third-party) hosten. Der Benutzer kann die Software dann nach Belieben anpassen, muss aber auch für das manuelle Hochladen aller nachfolgenden Updates verantwortlich sein.

Aufgrund ihrer großen Benutzerbasis sind WordPress-Websites ein beliebtes Ziel von Hackern, die häufig versuchen, veraltete Versionen der Software auszunutzen, die möglicherweise bekannte Sicherheitslücken aufweisen. Aus diesem Grund ist es wahrscheinlich ratsam, den Dienst nicht selbst zu hosten, es sei denn, Sie sind sich sicher, dass Sie den Dienst zusammen mit zusätzlichen Plugins, die Sie möglicherweise installiert haben, sichern und aktualisieren können.

> ### WordPress-Anpassungen

> WordPress ist von Grund auf so konzipiert, dass es hochgradig anpassbar ist. Die [Anwendungsprogrammierschnittstelle (API)](/application-programming-interface) deckt alles ab, von der Datenbank bis hin zu Themenänderungen und -anpassungen . Indem alles standardisiert bleibt, können sich Entwickler darauf konzentrieren, Funktionen auf WordPress aufzubauen, in dem Wissen, dass es auf allen Installationen funktioniert, die auf dem neuesten Stand gehalten werden.

>

## Beispiel für WordPress aus der Praxis

Es gibt viele verschiedene Unternehmen, die WordPress-Plugins und -Themen anbieten, um die Funktionalität zu erweitern. Zum Beispiel hat sich WooCommerce zu einem der beliebtesten E-Commerce-Tools für WordPress entwickelt, das es Benutzern ermöglicht, Bestellungen anzunehmen, Sendungen zu verfolgen und andere E-Commerce-Funktionen zu ihrer bestehenden WordPress-Website hinzuzufügen .

Zu den vielen anderen Beispielen für Plugins für WordPress gehören solche, die sich mit Suchmaschinenoptimierung (SEO), Kontaktformularen, Newsletter-Marketing, sozialen Medien vor Ort und Kommentarfunktionen sowie verschiedenen Designfunktionen befassen, die darauf abzielen, Websites optisch ansprechender zu gestalten.

## Höhepunkte

- WordPress ist eine führende Open-Source-CMS-Plattform.

- Es ist bekannt für seine hochgradig anpassbare Natur und verfügt über eine große Community von Plugin-Entwicklern.

- Aufgrund seiner großen Benutzerbasis ist WordPress häufig das Ziel von Hackern. Um dieses Risiko zu reduzieren, sollten Nutzer darauf achten, dass ihre WordPress-Software stets auf dem neuesten Stand ist.
