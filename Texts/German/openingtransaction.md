# Eröffnungstransaktion
## Was ist eine Eröffnungstransaktion?

Eine Eröffnungstransaktion, ein Begriff, der typischerweise mit derivativen Produkten in Verbindung gebracht wird, bezieht sich auf den anfänglichen Kauf oder Verkauf, der eine neue Position begründet oder eröffnet. Man kann [kaufen,](/buytoopen) um eine Long-Position [zu eröffnen, oder verkaufen, um](/selltoopen) eine Short-Position zu eröffnen. Sobald eine Eröffnungstransaktion stattgefunden hat, existiert somit eine [offene Position](/open-position).

Das Gegenteil einer Eröffnungstransaktion wird passenderweise als [Schlusstransaktion bezeichnet](/closeposition). In diesem Fall könnte man [verkaufen, um](/selltoclose) eine bestehende Long- [Position zu schließen, oder kaufen, um](/buytoclose) eine bestehende Short-Position zu schließen.

##Eröffnungstransaktionen verstehen

Einfach ausgedrückt ist eine Eröffnungstransaktion der Akt der Initiierung eines neuen Handels. Es kann das Eingehen einer neuen Position in einem bestimmten [Wertpapier](/security) oder den Einstieg in eine Vielzahl unterschiedlicher Derivatkontraktpositionen umfassen, die für einen bestimmten Zeitrahmen offen bleiben. Der Begriff wird häufig mit dem [Optionshandel in Verbindung gebracht](/option). Optionsstrategien, wie das Schreiben einer Option short oder der Kauf einer Option long, wären Beispiele für eine Eröffnungstransaktion.

Eine Eröffnungstransaktion ist der erste Schritt bei der Platzierung eines Trades und beinhaltet den Kauf eines Vermögenswerts oder Finanzinstruments. Es handelt sich im Allgemeinen – aber nicht immer – um eine Abschlusstransaktion zu einem späteren Zeitpunkt, der am selben Tag für einen [Intraday-](/intraday) Handel oder Tage, Wochen oder Monate später für eine längerfristige Anlage erfolgen kann. Eine Eröffnungstransaktion kann für verschiedene Arten von Anlagen unterschiedliche Überlegungen beinhalten, und diese Überlegungen unterscheiden sich erheblich bei öffentlich gehandelten Wertpapieren und [Derivaten](/derivative).

Weniger häufig kann sich eine Eröffnungstransaktion auch auf den ersten Handel für ein bestimmtes Wertpapier an einem bestimmten Handelstag beziehen. Konkret bezieht sich dies auf den Handelspreis dieses Wertpapiers, der für Anleger von Bedeutung ist, da er ihnen einen Vergleich mit dem [Schlusskurs](/closingprice) des vorherigen Handelstages ermöglicht.

[Open Interest](/openinterest) eines Optionskontrakts zeigt an, wie viele Positionen darin derzeit bestehen.

## Öffentlich gehandelte Wertpapiere

Anleger können sich aus verschiedenen Gründen dafür entscheiden, über eine Eröffnungstransaktion in ein öffentlich gehandeltes Wertpapier zu investieren. Im Allgemeinen kaufen Anleger ein Wertpapier aufgrund seines Kapitalzuwachs- oder Ertragspotenzials. Anleger können aufgrund ihrer Wachstums- oder Wertmerkmale im Laufe der Zeit ein langfristiges Potenzial in einem Wertpapier sehen. Diese Motivationen können von den [Umsatzschätzungen](/revenue),. dem Ertragspotenzial oder den fundamentalen Kennzahlen eines Wertpapiers bestimmt werden.

Anleger und insbesondere Daytrader oder [technische Analysten](/technical-analyst) können sich dafür entscheiden, eine Wertpapierposition über eine Eröffnungstransaktion für ihre kurzfristigen Gewinne einzugehen. Kurzfristige Anleger gehen in der Regel eine Anlage mit einem genauer definierten Zeitrahmen ein und versuchen, die Position relativ schnell zu schließen, um von der günstigen kurzfristigen [Volatilität zu profitieren](/volatility). In diesem Szenario kann ein Anleger eine Transaktion innerhalb von Stunden, Tagen oder Wochen eröffnen und schließen.

## Derivatpositionen

Eine Eröffnungstransaktion, die einen Anleger in einen Derivatkontrakt einbindet, hat eine relativ wichtigere Bedeutung als eine Eröffnungstransaktion für ein öffentlich gehandeltes Wertpapier. Wenn ein Anleger eine Derivatposition eingeht, hat er eine bestimmte Zeitspanne Zeit, um aus der Investition [Gewinne zu erzielen.](/profit) Dies erfordert, dass sie die Position während ihrer gesamten Lebensdauer genauer überwachen.

In einem [amerikanischen Optionskontrakt](/americanoption) [hat](/americanoption) ein Anleger nach einer Eröffnungstransaktion das Recht, diesen Kontrakt jederzeit bis zum Ablauf auszuüben. Nach Ablauf gilt der Vertrag als geschlossen. Bei einer [europäischen Option](/europeanoption) kann der Optionsinhaber die Option nur am Verfalltag ausüben. Sowohl für amerikanische als auch für europäische Optionen kann der Anleger seine Option auch auf dem Markt handeln, um die Position zu schließen.

Bei einem Terminkontrakt kauft ein Anleger das Derivat zur Ausführung an einem bestimmten Datum. Sie können den Kontrakt bis zum Ablauf jederzeit auf dem freien Markt verkaufen. Wenn sie den Vertrag bis zum Ablauf halten, sind sie verpflichtet, die Forderungen aus dem Vertrag zu erfüllen, wozu auch die Lieferung [gehören](/physicaldelivery) [kann](/physicaldelivery).

## Höhepunkte

- Eine Eröffnungstransaktion kann sich auch auf den ersten Handel für ein bestimmtes Wertpapier an einem bestimmten Handelstag beziehen, der den Eröffnungspreis festlegt.

- Eine Eröffnungstransaktion ist eine, die eine neue Position einleitet, entweder Long oder Short, normalerweise im Zusammenhang mit den Derivatemärkten.

- Eröffnungstransaktionen werden schließlich durch Schließungstransaktionen ergänzt, die die offene Position entfernen oder schließen.
