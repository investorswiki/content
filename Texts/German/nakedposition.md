# Nacktstellung
## Was ist eine Nacktstellung?

Im Wertpapierhandel im Allgemeinen bezieht sich eine nackte Position auf eine Wertpapierposition, Long oder Short, die nicht gegen das Marktrisiko [abgesichert ist.](/hedge) Sowohl der potenzielle Gewinn als auch das potenzielle Risiko sind größer, wenn eine Position offen ist, anstatt auf irgendeine Weise gedeckt oder abgesichert zu sein. Im Optionshandel bezieht sich dieser Ausdruck speziell auf eine Option, die von einem Händler ohne etablierte Position in dem zugrunde liegenden Wertpapier verkauft wird.

## Eine nackte Position verstehen

Eine nackte Aktienposition hat nicht die Absicherung, die mit einer [Call-](/calloption) oder Put-Option oder vielleicht einer entgegengesetzten Position in einer verwandten Aktie verbunden ist. Zum Beispiel eine Long-Position in Coke und eine Short-Position in Pepsi.

Eine nackte Position ist von Natur aus riskant, da es keinen Schutz gegen eine nachteilige Bewegung gibt. Die meisten Anleger halten den Besitz von Aktien nicht für übermäßig riskant, insbesondere weil es in den meisten Fällen einfach ist, die Position an den Markt zurückzuverkaufen. Ein rückläufiger Markt für einen Anleger, der eine [Long-Position](/long) in einer Aktie hält, kann jedoch immer noch erhebliche Verluste mit sich bringen. In diesem Fall könnte das Halten einer [Put-Option](/putoption) gegen die Long-Aktienposition die Verluste zu einem geringen Preis auf ein überschaubares Maß begrenzen.

Das Gewinnpotenzial des Anlegers vor [Provisionen](/commission) würde um die [Prämie](/premium) oder die Kosten der Option reduziert. Betrachten Sie es als eine Versicherungspolice, die der Investor niemals zu nutzen hofft.

Anleger [, die Aktien](/shortselling) ohne Absicherung leerverkaufen, sind einem noch größeren Risiko ausgesetzt, da das Aufwärtspotenzial einer Aktie theoretisch unbegrenzt ist. In diesem Fall würde der Besitz eines Calls auf die zugrunde liegende Aktie dieses Risiko begrenzen.

## Naked-Optionen

Auf dem Optionsmarkt sind auch [ungedeckte](/uncovered-option) oder nackte Calls und Puts riskant. In diesem Fall ist es der Optionsverkäufer oder [Stillhalter](/writing-an-option),. der keine Absicherung gegen eine [Zuteilung hat](/assignment). Optionskäufer riskieren nur den für den Kauf der Optionen gezahlten Betrag, der normalerweise deutlich unter dem Betrag liegt, der zum Kauf tatsächlicher Aktien oder eines anderen zugrunde liegenden Vermögenswerts erforderlich ist.

Optionsverkäufer hingegen können ein unbegrenztes Risiko eingehen, wenn sie nicht abgesichert sind. Beispiel: Ein Investor verkauft eine Call-Option auf eine Aktie und diese Aktie steigt vor [Ablauf im Preis an](/expirationdate). Der Optionskäufer könnte wahrscheinlich die Option [ausüben](/exercise) und den Verkäufer zwingen, auf den [freien Markt](/open-market) zu gehen, um die Aktie zum höheren Preis zu kaufen, um sie an den Optionskäufer zu liefern. Wenn der Optionsverkäufer eine Gegenposition in der zugrunde liegenden Aktie besäße, wäre sein Risiko begrenzt.

Put-Verkäufer hätten ein nahezu unbegrenztes Risiko, sollte das [zugrunde liegende Wertpapier](/underlying-security) gegen Null fallen. Eine entsprechende Short-Position in der zugrunde liegenden Aktie würde dieses Risiko begrenzen.

Praktisch gesehen wird der Verkäufer von ungedeckten Puts oder Calls diese jedoch wahrscheinlich zurückkaufen, lange bevor sich der Preis des zugrunde liegenden Wertpapiers nachteilig zu weit vom [Ausübungspreis entfernt](/strikeprice),. basierend auf seiner Risikotoleranz und seinen [Stop-Loss -](/stop-lossorder) Einstellungen.

Fortgeschrittenere Optionshändler können das Risiko mit mehreren Positionen von Puts und Calls, sogenannten [Kombinationen , absichern](/combination).

## Höhepunkte

- Dieser Ausdruck wird häufiger mit Leerverkäufen von Aktien in Verbindung gebracht.

- Eine nackte Aktienposition ist eine Position, die nicht abgesichert ist.

- Eine ungedeckte Position wird auch häufig verwendet, um eine Option zu bezeichnen, die ohne eine Position in dem zugrunde liegenden Wertpapier verkauft wird, um sich gegen das Risiko einer Optionszuteilung abzusichern.
