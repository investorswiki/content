# Marktgewicht
## Was ist Marktgewicht?

Das Marktgewicht bezieht sich auf das Wertrating, das einem [festverzinslichen](/fixedincome) Instrument gegeben wird, wenn sein [Kreditspread den](/creditspread) Markterwartungen entspricht.

## Marktgewicht verstehen

Das Marktgewichtungs-Rangsystem liefert eine subjektive Schätzung der Genauigkeit des aktuellen Credit Spread eines festverzinslichen Instruments, die dann von einem Anleger verwendet werden kann, um festzustellen, ob dieses Instrument eine attraktive Anlage ist. Das System umfasst drei Ränge – Marktgewichtung, [Übergewichtung](/overweight) und [Untergewichtung](/underweight). Das Marktgewichtungsrating gibt an, dass der aktuelle Kreditspread eines Instruments den Markterwartungen entspricht. Im Wesentlichen wird gesagt, dass ein festverzinsliches Wertpapier, das als marktgewichtet gilt, einen Kreditspread bietet, der dem Konsens des Marktes entspricht oder nahe daran liegt.

So wie Aktien eine Kauf-, Verkaufs- oder Halteempfehlung haben können, bewertet dieses Bonitätsbewertungssystem einen Schuldtitel als übergewichtet, untergewichtet oder marktgewichtet. Eine Marktgewichtung entspricht einem Hold-Rating, während eine Über- oder Untergewichtung den Kauf- bzw. Verkaufstiteln entspricht. Analysten werden feststellen, ob der aktuelle Kreditspread ein angemessenes Risikomaß für die Anlage ist, und eine entsprechende Empfehlung abgeben.

Genau wie Aktienwerte werden festverzinsliche Instrumente in mehrere Kategorien unterteilt. Zu diesen Faktoren gehören unter anderem [das Kreditrisiko](/creditrisk) (oder Kreditrating), die Geographie, die Branche, [die Rendite](/yield) und die [Laufzeit](/maturity). Festverzinsliche Wertpapiere fügen eine weitere Betrachtungsebene für Eventualitäten hinzu, wie z. B. Call-Optionen und Konvertibilität, die Entscheidungen zur Portfoliogewichtung weiter beeinflussen könnten.

Festverzinsliche Instrumente wie Investment-Grade-Anleihen könnten als mit Marktgewicht gehalten beschrieben werden, was bedeutet, dass ein Portfolio Investment-Grade-Anleihen im Vergleich zu einer gemeinsamen Benchmark weder übergewichtet noch untergewichtet (allokiert). Wenn ein Portfoliomanager eine bestimmte Ansicht vertritt, beispielsweise eine optimistische Haltung gegenüber Anleihen aus dem Industriesektor, kann ein Portfolio vom Marktkonsens abweichen, indem das Portfolio in eine übergewichtete Position in Industrieanleihen übergewichtet wird.

## Höhepunkte

- Das Marktgewichtungs-Rangsystem liefert eine subjektive Schätzung der Genauigkeit des aktuellen Kreditspreads eines festverzinslichen Instruments, die dann von einem Anleger verwendet werden kann, um festzustellen, ob dieses Instrument eine attraktive Anlage ist.

- Marktgewicht bezieht sich auf das Wertrating, das einem festverzinslichen Instrument gegeben wird, wenn sein Kreditspread den Markterwartungen entspricht.

- Es gibt drei primäre Ränge, die verwendet werden, um festverzinsliche Instrumente zu bewerten – Marktgewichtung, Übergewichtung und Untergewichtung.
