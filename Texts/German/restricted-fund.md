# Eingeschränkter Fonds
## Was ist ein eingeschränkter Fonds?

Ein eingeschränkter Fonds ist ein Reservekonto, das Geld enthält, das nur für bestimmte Zwecke verwendet werden kann. Zweckgebundene Mittel geben den Spendern die Gewissheit, dass ihre Beiträge in der von ihnen gewählten Weise verwendet werden. Sie treten am häufigsten im Zusammenhang mit Fonds auf, die von bestimmten gemeinnützigen Organisationen, Universitäten oder Versicherungsunternehmen gehalten werden.

## Arten von gemeinnützigen eingeschränkten Fonds

[gemeinnützigen Organisation](/non-profitorganization) Geld gibt , kann er angeben, wie ein Geschenk auf drei grundlegende Arten verwendet werden soll:

- **Uneingeschränkter Fonds** – Das darin enthaltene Geld kann für jeden Zweck verwendet werden, den die Organisation für richtig hält.

- **Vorübergehend zweckgebundener Fonds** – Das Geld muss für einen bestimmten, angegebenen Zweck verwendet werden.

- **Permanent Restricted Fund** – Die Spende soll auf [Dauer](/perpetuity) als [Hauptbetrag gehalten](/principal) werden, für den [Zinsen](/interest) verdient werden können, wobei nur die Zinsen ausgegeben werden dürfen.

Wenn ein Spender eine gemeinnützige Organisation beschränkt, Gelder für einen bestimmten Zweck zuzuweisen, ist sie gesetzlich dazu verpflichtet. Eine Nichteinhaltung kann dazu führen, dass der Spender rechtliche Schritte einleitet und die gemeinnützige Organisation dem Büro des US-Generalstaatsanwalts meldet.

[Zustiftungen](/endowment) sind in der Regel dauerhaft zweckgebundene Mittel. In den meisten Fällen kann ihr Kapital nicht ausgegeben werden, und nur ein bestimmter Prozentsatz der Zinsen, die sie verdienen, kann pro Jahr ausgegeben werden. Darüber hinaus gibt es Beschränkungen, wie die Zinsen ausgegeben werden können. Beispielsweise kann eine Zustiftung an eine Universität auf die Finanzierung von Stipendien und Professuren beschränkt sein.

> Ein Spender von zweckgebundenen Mitteln an eine gemeinnützige Organisation gibt in der Regel in einem schriftlichen Dokument, dem sogenannten Schenkungsinstrument, an, wofür das Geld verwendet werden kann.

## Wie werden zweckgebundene Mittel ausgewiesen?

Der Spender bestimmt, ob die Mittel zu sperren sind. Typischerweise wird die Bezeichnung des Fonds schriftlich in dem sogenannten Geschenkinstrument angegeben. Stiftungen, die zweckgebundene Mittel vergeben, beschreiben oft, wie sie ihr Geld verwenden möchten, wenn sie den Preis verteilen.

Gemeinnützige Organisationen können Verwirrung darüber vermeiden, wie sie die Gelder eines Spenders ausgeben wollen, indem sie eine Auswahl an Bezeichnungen anbieten. Eine gemeinnützige Organisation für Krebsforschung könnte beispielsweise Spendern die Möglichkeit geben, ihre Gelder für klinische Studien zu Brust-, Haut- oder Hirnkrebs einzusetzen.

## Eingeschränkte Fondsverwaltung für gemeinnützige Organisationen

In der Regel müssen zweckgebundene Mittel nicht auf einem getrennten Bankkonto eingezahlt werden, sie müssen jedoch separat in den [Abschlüssen einer gemeinnützigen Organisation ausgewiesen werden](/financial-statements). Bei der [Budgetierung](/budget) sollten gemeinnützige Organisationen zweckgebundene und nicht zweckgebundene Mittel trennen, damit sie das Geld, das sie ausgeben müssen, korrekt zuordnen. Sie können ein internes System implementieren, das das Management benachrichtigt, wenn die Verpflichtungen für beschränkte Fonds erfüllt wurden. Sobald der Wunsch des Spenders erfüllt ist, kann überschüssiges Geld in freie Fonds übertragen werden.

Gemeinnützige Mitarbeiter sollten darin geschult werden, Ausgaben zu identifizieren, die zweckgebundenen Fonds zugewiesen werden müssen. Wenn die Mitarbeiter Gelder richtig zuteilen, sorgt das für zufriedene Spender und hilft, Rechtsstreitigkeiten zu vermeiden.

## Höhepunkte

- Oft verbunden mit Geldern, die durch Spenden an gemeinnützige Organisationen oder Stiftungen gehalten werden, stellen zweckgebundene Mittel sicher, dass Spender allein die Verwendung dieser Vermögenswerte steuern können.

- Die Nichteinhaltung von Beschränkungen oder die unbefugte Verwendung von eingeschränkten Mitteln kann rechtliche Schritte nach sich ziehen.

- Ein gebundenes Guthaben ist jedes Barguthaben, das für eine bestimmte oder begrenzte Verwendung vorgesehen ist.

## FAQ

### Was ist ein eingeschränkter Fonds?

Ein beschränkter Fonds enthält Gelder, die ein Spender verfügt hat, kann nur für einen bestimmten Zweck verwendet werden. Ist das Geld vorübergehend zweckgebunden, kann der Überschuss nach Zweckerfüllung frei werden. Wenn das Geld dauerhaft gebunden ist, muss es in Form einer Zustiftung erhalten bleiben, in der Regel auf ewig, und nur die Zinsen, die durch die Anlage der Zustiftung erzielt werden, dürfen für den Zweck verwendet werden.

### Benötigen zweckgebundene Fonds eigene Bankkonten?

Nein. Es ist nicht notwendig, das Geld zu trennen, indem Sie es auf ein eigenes Konto einzahlen. Stattdessen sollte die Trennung durch Rechnungslegungspraktiken im Jahresabschluss der gemeinnützigen Organisation behandelt werden.

### Was passiert, wenn zweckgebundene Mittel zweckentfremdet ausgegeben werden?

Die Gelder sind gesetzlich beschränkt, so dass ein Spender, wenn sie nicht für den vorgesehenen Zweck verwendet werden, rechtliche Schritte einleiten und ihre Rückgabe verlangen kann. Der Spender kann die gemeinnützige Organisation auch dem Büro des US-Generalstaatsanwalts melden.
