# Machen Sie einen Anruf
## Was ist ein Put auf einen Anruf?

Ein Put auf einen Call (PoC) ist eine Art [zusammengesetzte Option , bei der](/compoundoption) eine [Put -Option](/putoption) auf eine [Call-Option geschrieben wird](/calloption). Daher gibt es zwei Ausübungspreise und zwei Ausübungstage.

Wenn der Optionsinhaber die Put-Option ausübt, wird er eine Call-Option leerverkaufen, eine Option, die dem Inhaber das Recht, aber nicht die Verpflichtung gibt, einen bestimmten Vermögenswert zu einem festgelegten Preis innerhalb eines festgelegten Zeitraums zu kaufen.

Der Wert eines Put auf einen Call ändert sich umgekehrt proportional zum Aktienkurs. Das bedeutet, dass der Wert sinkt, wenn der Aktienkurs steigt, und steigt, wenn der Aktienkurs fällt. Ein Put auf einen Anruf kann auch als Split-Fee-Option bezeichnet werden.

## Wie ein Anruf funktioniert

Wenn der Inhaber einen Put auf einen Call ausübt, der als Overlying-Option bezeichnet wird, muss er die zugrunde liegende Call-Option an den Verkäufer liefern und eine Prämie auf der Grundlage des Ausübungspreises der Overlying-Put-Option kassieren. Diese Prämie wird als Rückvergütung bezeichnet.

Alternativ muss der Inhaber, wenn er eine Compound-Call-Option ausübt, dem Verkäufer der zugrunde liegenden Option eine Prämie zahlen, die auf dem Ausübungspreis der darüber liegenden Call-Option basiert.

Es ist üblicher, Compound [-Optionen auf Devisen- oder Rentenmärkten zu sehen](/fixedincome),. wo Ungewissheit bezüglich der Risikoschutzfähigkeiten der Option besteht. Die Vorteile von Compound-Optionen bestehen darin, dass sie eine große Hebelwirkung ermöglichen und billiger sind als reine Optionen. Wenn jedoch beide Optionen ausgeübt werden, ist die Gesamtprämie höher als die Prämie einer einzelnen Option [.](/premium)

Auf dem [Hypothekenmarkt](/mortgage) sind PoC-Optionen nützlich, um das Risiko von Zinsänderungen zwischen dem Zeitpunkt der Hypothekenzusage und dem geplanten Lieferdatum auszugleichen.

##Reale Anwendung

Während Spekulationen an den Finanzmärkten immer einen großen Teil der Aktivität von Compound-Optionen ausmachen, könnten sie für Geschäftsunternehmen nützlich sein, wenn sie ein großes Projekt planen oder Gebote abgeben. In einigen Fällen müssen sie die Finanzierung oder Lieferungen sicherstellen, bevor sie das Projekt tatsächlich starten oder gewinnen können. Wenn sie das Projekt nicht bauen oder gewinnen, könnten sie mit der Finanzierung zurückbleiben, die sie nicht benötigen. In diesem Fall bieten Compound-Optionen eine Versicherungspolice.

Dasselbe gilt auch für Institutionen, die die Finanzierung bereitstellen, da sie versuchen, ihr Risiko abzusichern, falls sie sich verpflichten, das Geld bereitzustellen, das von Unternehmen für ihre Projekte benötigt wird, und diese Unternehmen ihre Geschäfte nicht gewinnen.

## Compound-Optionen

Eine zusammengesetzte Option ist eine Option, bei der der Basiswert eine andere Option ist. Daher gibt es zwei Ausübungspreise und zwei Ausübungstage. Sie sind für jede Kombination von Calls und Puts verfügbar. Beispielsweise ein Put, bei dem der Basiswert eine Call-Option ist, oder ein Call, bei dem der Basiswert eine Put-Option ist.

Folgende Verbundoptionen stehen zur Verfügung:

- [Call auf einen Put](/call-on-a-put) : CoP (CaPut) – Dies ist eine Call-Option auf eine zugrunde liegende Put-Option. Der Eigentümer, der die Call-Option ausübt, erhält eine Put-Option.

- [Call auf einen Call](/call-on-a-call) : CoC (CaCall) – Der Anleger kauft eine Call-Option mit dem Recht, eine andere Call-Option auf dasselbe zugrunde liegende Wertpapier zu kaufen.

- [Put auf einen Put](/put-on-a-put) : PoP—Der Anleger muss die zugrunde liegende Put-Option an den Verkäufer liefern und eine Prämie basierend auf dem Ausübungspreis der darüber liegenden Put-Option kassieren.

- Put on a Call: PoC—Der Anleger muss die zugrunde liegende Call-Option an den Verkäufer liefern und eine Prämie basierend auf dem Ausübungspreis der darüber liegenden Put-Option kassieren.

Wenn der Inhaber eine zusammengesetzte Kaufoption ausübt, die als Overlying-Option bezeichnet wird, muss er dem Verkäufer der zugrunde liegenden Option eine Prämie zahlen, die auf dem Ausübungspreis der zusammengesetzten Option basiert. Diese Prämie wird als [Rückvergütung bezeichnet](/backfee). Alternativ muss der Inhaber, wenn er eine Compound-Put-Option ausübt, die zugrunde liegende Option an den Verkäufer der Compound-Option liefern.

Trader können zusammengesetzte Optionen verwenden, um die Lebensdauer einer bärischen Optionsposition zu verlängern, da es möglich ist, einen Put mit einer kürzeren Restlaufzeit für einen anderen Put mit einer längeren Restlaufzeit zu kaufen. Mit anderen Worten, sie können an den Verlusten des Basiswerts partizipieren, ohne ihn bei Verfall in voller Höhe kaufen zu müssen. Die Einschränkung ist, dass zwei Prämien gezahlt werden und höhere Kosten anfallen.

Es ist üblicher, Compound-Optionen auf Devisen- oder Rentenmärkten zu sehen, wo Ungewissheit bezüglich der Risikoschutzfähigkeiten der Option besteht. Die Vorteile von Compound-Optionen bestehen darin, dass sie eine große [Hebelwirkung ermöglichen](/leverage) und billiger sind als reine Optionen. Wenn jedoch beide Optionen ausgeübt werden, ist die Gesamtprämie höher als die Prämie einer einzelnen Option.

## Höhepunkte

- Ein Put auf eine Call-Option ist eine zusammengesetzte Option, die dem Inhaber das Recht gibt, eine Call-Option zu verkaufen

- Daher gäbe es zwei Ausübungspreise: einen für den Put und einen für den zugrunde liegenden Call.

- Ein Put auf eine Call-Option kann von einem Investor verwendet werden, um seine Absicherung eines zugrunde liegenden Vermögenswerts zu geringen Kosten zu erweitern, und er kann auch in der Immobilienentwicklung verwendet werden, um aus Eigentumsrechten auszusteigen, ohne an das Geschäft gebunden zu sein.
