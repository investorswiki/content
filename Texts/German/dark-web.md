# Dunkles Netz
## Was ist das Dark Web?

Der Begriff Dark Web bezeichnet verschlüsselte Online-Inhalte, die von herkömmlichen Suchmaschinen nicht indexiert werden. Der Zugriff auf das Dark Web kann nur mit bestimmten Browsern wie dem TOR-Browser erfolgen. Die Nutzung des Dark Web bietet im Vergleich zu herkömmlichen Websites ein hohes Maß an Privatsphäre und Anonymität.

Daher wird die meiste Aufmerksamkeit auf Online- [Marktplätze](/market) für Drogen, [den Austausch](/exchange) gestohlener Daten und andere illegale Aktivitäten gelegt, wenn die Leute an das Dark Web denken. Trotzdem gibt es oft sehr legitime Gründe, warum Menschen sich für die Nutzung des Dark Web entscheiden, einschließlich politischer Dissidenten und Menschen, die bestimmte Informationen geheim halten möchten.

## Das Dark Web verstehen

Wie der Name schon sagt, ist das Dark Web ein geheimes Netzwerk, das im Untergrund existiert. Es besteht aus einer Reihe von Websites, die der Öffentlichkeit verborgen bleiben. Das bedeutet, dass sie nicht über herkömmliche Suchmaschinen wie Google zugänglich sind.

Herkömmliche Suchmaschinen geben Ergebnisse zurück, weil sie Indizes mit Links zu Websites enthalten. Diese werden nach Schlüsselwörtern und Relevanz geordnet. Das Dark [Web](/social-media) hingegen verwendet Informationen, die in diesen anderen Suchmaschinen nicht verfügbar sind, wie z medizinisch).

Das Darknet, auch Darknet genannt, ist dem breiteren Web in seinen Anfängen im späten 20. Jahrhundert sehr ähnlich. Es gibt eine Menge Material darüber, wie man es zum Laufen bringt, und nicht viel zu tun, wenn man erst einmal dort ist. Viele Inhalte im Dark Web sind sehr amateurhaft. Auf der anderen Seite ist es für Einzelpersonen viel einfacher, Websites zu starten und Aufmerksamkeit zu erlangen. Tech-Giganten und große Medienorganisationen haben ab 2020 nur noch sehr wenig Einfluss auf das Dark Web.

Wie das frühe Internet hat das Darknet den Ruf, ein Paradies für illegale Aktivitäten zu sein, und ist häufig in illegale und kriminelle Aktivitäten verwickelt. Während das Dark Web bei illegalen und unethischen Transaktionen eine Rolle gespielt hat, bietet es auch eine soziale Möglichkeit für Menschen, die andernfalls wegen ihrer Identität oder politischen Überzeugung verfolgt werden könnten. Außerdem gibt es den Justizbehörden die zusätzlichen Werkzeuge an die Hand, die sie benötigen, um die Täter unethischer Aktivitäten festzunehmen.

> Das Dark Web ist immer noch in Arbeit und seine vollen Kosten und Vorteile sind noch nicht bekannt.

>

## Besondere Überlegungen

Leute verwechseln das Dark Web oft mit [Kryptowährungen](/cryptocurrency),. die dort oft verwendet werden, um Einkäufe zu tätigen. Aber es gibt einen deutlichen Unterschied. Das Dark Web erleichtert die Einrichtung und den Zugriff auf Websites, die ein hohes Maß an Anonymität für alle Beteiligten bieten.

Viele Dark-Web-Websites enthalten nur Informationen und nicht die Möglichkeit, etwas zu kaufen oder zu verkaufen. Es stimmt jedoch, dass Kryptowährungen wie [Bitcoin](/bitcoin) und [Monero](/monero) häufig für Transaktionen im Dark Web verwendet werden. Aber man muss nicht das dunkle Web benutzen, um Kryptowährungen zu verwenden.

## Geschichte des Dark Web

Das Dark Web erschien erstmals offiziell in den frühen 2000er Jahren zusammen mit der Gründung von Freenet, das von Ian Clarke entwickelt wurde, um Benutzer vor staatlichen Eingriffen und Cyberangriffen zu schützen. Das System, das heute noch verfügbar ist, ermöglicht es Benutzern, sich frei zu äußern, ohne online verfolgt zu werden.

Das US Naval Research Laboratory finanzierte ein Projekt namens The Onion Router (TOR). TOR bot Geheimdienstquellen eine Möglichkeit, einfach und sicher zu kommunizieren, insbesondere in feindlichen Gebieten, in denen die persönliche Sicherheit von entscheidender Bedeutung ist. Es ist heute einer der am häufigsten verwendeten Browser für den Zugriff auf das Dark Web und verwendet Datenbanken, um den Menschen zu helfen, sich zurechtzufinden und die benötigten Informationen zu finden.

Der Aufstieg von Kryptowährungen erhöhte die Popularität des Dark Web, insbesondere bei Cyberkriminellen. Das liegt daran, dass digitale Währungen oft ein hohes Maß an Anonymität für Menschen bieten, die im Dark Web kaufen und verkaufen.

Aufgrund seiner Verbindung mit bestimmten illegalen Aktivitäten wurden Forderungen nach einer Regulierung des Dark Web laut. Beispielsweise haben die [G20](/g-20) und die [Financial Action Task Force (FATF)](/financial-action-task-force-fatf) beide Kryptowährungsunternehmen aufgefordert, Informationen über Käufer und Verkäufer bei online durchgeführten Transaktionen bereitzustellen. Dies gilt insbesondere, um den Strafverfolgungsbehörden dabei zu helfen, kriminelle Organisationen und illegale Aktivitäten aufzuspüren.

> Ross Ulbricht wurde 2013 untersucht und verhaftet, weil er die dunkle Website The Silk Road erstellt und betrieben hatte, eine Darknet-Website, von der die Behörden sagten, dass sie zum Kauf und Verkauf von Betäubungsmitteln verwendet wurde.

>

## Dark Web vs. Deep Web

Das Dark Web und das Deep Web werden auch oft fälschlicherweise synonym verwendet. Tatsächlich ist das Dark Web nur ein Teil des Deep Web. Das Deep Web liegt im Untergrund und ist nicht nur der Inhalt, der als dunkel gilt. Deshalb wird es auch das unsichtbare Netz oder das verborgene Netz genannt.

Die im Deep Web gefundenen Informationen sind normalerweise verschlüsselt und werden nicht wie im traditionellen Web in Indizes gefunden. Es enthält alle Seiten, die nicht angezeigt werden, wenn Sie eine Websuche ausführen. Es enthält auch alles, was eine Anmeldung erfordert, wie z. B. Inhalte von:

- Online-Banking

- Bezahlen Sie Websites wie Netflix und Amazon Prime

- Datei-Hosting-Dienste wie Dropbox und seine Konkurrenten

- Private Datenbanken

Die im Dark Web aufgelisteten Informationen können auch alles enthalten, was nur für Abonnenten zugänglich ist, und nicht alles, was illegal ist.

## Vor- und Nachteile des Dark Web

### Vorteile

Das Dark Web hilft den Menschen, ihre Privatsphäre zu wahren und ihre Meinung frei zu äußern. Privatsphäre ist für viele unschuldige Menschen, die von Stalkern und anderen Kriminellen terrorisiert werden, von entscheidender Bedeutung. Auch die zunehmende Tendenz potenzieller Arbeitgeber, Beiträge in sozialen Medien zu verfolgen, kann es schwierig machen, öffentlich ehrliche Diskussionen zu führen.

Schließlich macht die Popularität des Darknets bei Kriminellen es zu einer perfekten Kommunikationsmöglichkeit für Undercover-Polizisten.

### Nachteile

Einige Leute werden unweigerlich die Macht missbrauchen, die mit der Nutzung des Dark Web verbunden ist, indem sie es einfacher machen, sich an kriminellen Aktivitäten zu beteiligen. Zum Beispiel macht es die Kombination aus Dark Web und Kryptowährungen theoretisch viel einfacher, jemanden mit der Begehung bestimmter Verbrechen anzuheuern.

Während das Dark Web seinen Nutzern Privatsphäre verspricht, kann es auch dazu verwendet werden, die Privatsphäre anderer zu verletzen. Private Fotos, Krankenakten und Finanzinformationen wurden alle gestohlen und im Darknet geteilt.

## Beispiel für das Dark Web

Nehmen wir an, Sie möchten ins Dark Web gehen, um zu sehen, welche Ihrer Informationen, falls vorhanden, herumschwirren. Sie können den TOR-Browser auf Ihrem System installieren, indem Sie ein virtuelles privates Netzwerk verwenden. Ein VPN ermöglicht es Menschen, ein öffentliches Netzwerk so zu nutzen, als würden sie Informationen über ein privates übertragen. Es gibt Ihnen im Grunde genommen Privatsphäre in einem öffentlichen Netzwerk.

Während Sie surfen, werden Sie wahrscheinlich feststellen, dass dunkle Website-Adressen mit der Erweiterung .onion und nicht mit den traditionellen .com, .org, .edu usw. enden. Ein Wort der Warnung jedoch. Stellen Sie sicher, dass Sie Sicherheitssoftware installiert haben, da die Wahrscheinlichkeit groß ist, dass Sie mit Malware und Phishing-Betrug in Kontakt kommen.

## Das Endergebnis

Das Dark Web ist ein relativ neues Konzept, zumindest wenn man die Geschichte des Internets als Ganzes betrachtet. Sie können es verwenden, um ohne Angst vor Konsequenzen anonym zu bleiben, wenn Sie sich frei äußern möchten. Aber dieses Maß an Privatsphäre hat dazu beigetragen, dass es einen schlechten Ruf als Forum für diejenigen erlangt hat, die es für illegale Aktivitäten nutzen wollen. Wenn Sie das Dark Web nutzen möchten, ist es so einfach wie die Installation eines privaten Browsers, die Verwendung eines VPN und die Gewährleistung, dass Ihr Computer sicher und geschützt bleibt.

## Höhepunkte

- Bestimmte Browser, wie z. B. Tor Browser, sind erforderlich, um das Dark Web zu erreichen.

- Es hat auch den Ruf, mit illegalen und unethischen Aktivitäten in Verbindung gebracht zu werden.

- Das Dark Web ruft Websites auf, die Informationen verwenden, die nicht online indiziert sind, wie Bankkonten, E-Mail-Konten und Datenbanken.

- Personen, die das Dark Web nutzen, können ihre Privatsphäre wahren und ihre Meinung frei äußern.

- Als Dark Web werden verschlüsselte Online-Inhalte bezeichnet, die von herkömmlichen Suchmaschinen nicht indexiert werden.

## FAQ

### Wie kommt man ins Dark Web?

Sie können auf das Dark Web zugreifen, indem Sie bestimmte, anonyme Browser wie TOR installieren. Einmal installiert, funktioniert der Browser genauso wie herkömmliche. Der Zugriff auf Informationen kann jedoch etwas schwierig sein, da kein Index verwendet wird, um die gewünschten Informationen zu finden. Stellen Sie sicher, dass Sie Sicherheitssoftware installieren, um sicherzustellen, dass Ihr Computer und Ihre persönlichen Daten sicher sind.

### Was tun Sie, wenn sich Ihre Informationen im Dark Web befinden?

Sie können eine Reihe von Maßnahmen ergreifen, um Ihre Daten zu schützen. Stellen Sie sicher, dass Sie alle Ihre Passwörter ändern, überprüfen Sie Ihre Kreditauskünfte auf Abweichungen und informieren Sie Ihre Bank(en). Sie können Ihr Guthaben auch einfrieren, um zu verhindern, dass es weiter gefährdet wird.

### Wie finden Sie heraus, ob sich Ihre Informationen im Dark Web befinden?

Ihre persönlichen Daten wie Bank-, Social-Media- und E-Mail-Konten sowie Ihre Sozialversicherungsnummer können im Dark Web zum Verkauf angeboten werden. Wenn Sie sehen möchten, ob Ihre Informationen im Dark Web herumschwirren, versuchen Sie, das Dark Web zu scannen. Alternativ können Sie sich für einen Dark-Web-Überwachungsdienst anmelden. Diese Tools warnen Sie, wenn sie Ihre Informationen finden.

### Ist es illegal, auf das Dark Web zuzugreifen?

Ungeachtet dessen, was der Name vermuten lässt, ist der Zugriff auf das Dark Web nicht illegal. Es bietet Einzelpersonen tatsächlich Privatsphäre und Anonymität, die herkömmliche Websites Einzelpersonen nicht bieten. Zum Beispiel können Menschen ins Dark Web gehen und ihre Gedanken über politische Aktivitäten posten, ohne befürchten zu müssen, von Regierungsbeamten und anderen Gruppen gerügt zu werden.
