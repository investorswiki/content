# Anfängliches Umtauschangebot (IEO)
Ein Initial Exchange Offering, allgemein als IEO bezeichnet, ist eine Spendenaktion, die von einer Börse verwaltet wird. Im Gegensatz zu einem Initial Coin Offering (ICO), bei dem das Projektteam selbst das Fundraising durchführt, bedeutet ein Initial Exchange Offering, dass das Fundraising auf einer Fundraising-Plattform einer bekannten Börse wie Binance Launchpad durchgeführt wird, auf der Benutzer Token kaufen können Gelder direkt aus ihrer eigenen Wechselgeldbörse.

Das ICO-Fieber im Jahr 2017 erwies sich als ein sehr risikoreiches Umfeld für Personen, die an neuen Blockchain-Projekt-Token-Veröffentlichungen teilnehmen wollten, da versehentlich Gelder an die falsche Brieftasche gesendet wurden oder einige Projektteams mit Geldern davonkamen.

### Was sind die Vorteile eines IEO?

Für einen Benutzer ist es einfach, an einem IEO teilzunehmen, da er keine On-Chain-Transaktionen mit verschiedenen Wallets auf verschiedenen Blockchains verwalten muss. Stattdessen benötigt ein Benutzer nur ein Konto an der Börse und etwas Geld auf seinem Konto und kann vollständig über die Schnittstelle der vertrauenswürdigen Website teilnehmen. Darüber hinaus setzt die Börse ihren Ruf hinter die Projekte auf ihrer Plattform und bietet ein höheres Maß an Vertrauen in das Projekt.

Für ein Projekt, das Spenden sammeln möchte, bietet ein IEO das Versprechen einer sofortigen Benutzerbasis, die ihr Produkt sehen kann, und je nach Größe des Publikums der Börse könnte das bedeuten, dass das Projekt seine externen Marketingtrichter für die Mittelbeschaffung reduzieren kann, was es ihnen ermöglicht konzentrieren sich nur auf die Entwicklung ihres Produkts.

Wenn Sie vorhaben, an einem IEO auf Binance Launchpad teilzunehmen, wird empfohlen, dass Sie den gesamten Binance Research-Bericht für den spezifischen Token, den Sie kaufen möchten, durchlesen.
