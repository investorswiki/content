# Zeichen
Token sind im Allgemeinen nicht abbaubare digitale Werteinheiten, die als Registrierungseinträge in Blockchains existieren.

Token gibt es in vielen verschiedenen Formen – sie können als Währungen für bestimmte Ökosysteme verwendet werden oder einzigartige Daten kodieren (siehe A Guide to Crypto Collectibles and Non-Fungible Tokens). Darüber hinaus können einige Token gegen Vermögenswerte außerhalb der Kette (z. B. Gold, Immobilien, Aktien) eingelöst werden.

Token werden im Allgemeinen von Unternehmen ausgegeben, die bestehende Blockchains von Drittanbietern wie die Ethereum-Blockchain verwenden, wie die vielen [ERC-20-](/erc-20) Token zeigen, die 2017 über ICOs ausgegeben und verkauft wurden. Genau genommen sind Token keine Kryptowährungen wie Bitcoin oder Ether, aber übertragbare Werteinheiten, die auf einer Blockchain ausgegeben werden.

Es gibt verschiedene Token-Klassifizierungen basierend auf den verschiedenen Eigenschaften der Token. Die Hauptklassifizierung verwendet die Funktionalität, um Token in Utility-Token und Security-Token zu unterteilen. Utility Token stellen im Allgemeinen den Zugang zu einer Dienstleistung dar oder können als Tauschmittel innerhalb eines Ökosystems fungieren.

Ein Beispiel für einen Utility-Token ist [BNB](/binance-coin-bnb),. der in erster Linie als Discount-Token zur Zahlung von Handelsgebühren an der Binance-Börse fungiert. Es kann aber auch zum Bezahlen von Waren und Dienstleistungen verwendet werden.

Security Token hingegen stellen finanzielle Vermögenswerte dar. Beispielsweise könnte ein Unternehmen während eines ICO Token-Aktien ausgeben, die dem Inhaber Eigentumsrechte und Dividenden gewähren. Diese wären rechtlich mit herkömmlich ausgeschütteten Aktien identisch.

Eine andere Klassifizierung bewertet Merkmale, um zwischen [fungiblen](/fungibility) und nicht fungiblen Token zu unterscheiden. Wenn Sie einen Dollarschein nehmen und ihn gegen einen anderen Dollarschein tauschen, behalten Sie den gleichen Wert. Es spielt keine Rolle, welche Einheit Sie besitzen, da sie demselben Zweck dienen. Auf der anderen Seite können Sie kein einzigartiges Kunstwerk nehmen und es durch ein anderes Kunstwerk ersetzen.

Das gleiche Prinzip gilt für Token. Am Beispiel von BNB spielt es keine Rolle, welche Einheiten Sie besitzen. Sie sind austauschbar. So etwas wie ein CryptoKitty hat jedoch einzigartige Eigenschaften, und jede Einheit muss anders behandelt werden.
