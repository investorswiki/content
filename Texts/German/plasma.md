# Plasma
Mit wenigen Worten, Plasma ist als Skalierungslösungstechnik für das Ethereum-Netzwerk konzipiert. Der Hauptvorschlag von Plasma besteht darin, als Rahmen von Seitenketten zu arbeiten, die so sparsam wie möglich mit der Ethereum-Blockchain (der Hauptkette) interagieren. Die Plasma-Struktur soll als [Blockchain](/blockchain) -Baum fungieren, der hierarchisch so angeordnet ist, dass viele „Kindketten“ über der Hauptkette erstellt werden.

Das Plasma-Framework wird durch die Verwendung von Smart Contracts und [Merkle Trees aufgebaut](/merkle-tree) und ermöglicht die Erstellung einer unbegrenzten Anzahl von Seitenketten – die im Wesentlichen kleinere Kopien der Ethereum-Blockchain sind.

Tatsächlich sind die untergeordneten Ketten so konzipiert, dass sie einen maßgeschneiderten Smart Contract ausführen, der es Unternehmen ermöglicht, die Plasmastruktur gemäß ihren individuellen Bedürfnissen einzusetzen. Daher können verschiedene Plasma Smart Contracts für verschiedene Anwendungsfälle erstellt werden. Durch die Nutzung der von der Hauptkette bereitgestellten Sicherheit ist Plasma in der Lage, eine Vielzahl verschiedener untergeordneter Ketten einzusetzen, die auf eine vorbestimmte Weise auf bestimmte Ziele hinarbeiten (die nicht unbedingt mit denen der Hauptkette zusammenhängen). Daher wäre es weniger wahrscheinlich, dass die Haupt-Blockchain von Ethereum überlastet wird.

Kurz gesagt, Plasma ist eine [Off-the-Main](/off-chain) [-Chain-](/off-chain) Lösung, die darauf abzielt, die Effizienz des Ethereum-Netzwerks (oder jeder anderen Blockchain) drastisch zu steigern, indem der Großteil der Verarbeitungsaufgaben von der Main-Chain abgezogen und auf a umverteilt wird Reihe kleinerer, funktioneller Ketten.

Plasma wurde von Vitalik Buterin und Joseph Poon im August 2017 als Skalierungslösung für Ethereum vorgeschlagen, aber das Konzept kann auch in anderen Blockchains angepasst und implementiert werden. Poon war auch einer der Idealisierer des Lightning Network-Vorschlags, was die Ähnlichkeiten von Plasma und LN als Skalierungslösungen erklärt (Plasma für Ethereum und Lightning Network für Bitcoin). Beachten Sie jedoch, dass sie unterschiedliche Ansätze und Mechanismen haben.

Das Ethereum Plasma ist ein Open-Source-Projekt und das öffentliche Repository ist auf ihrem GitHub zu finden. Ausführlichere und technische Informationen finden Sie auch im offiziellen Plasma-Whitepaper. Obwohl sich das Projekt noch in einem frühen Entwicklungsstadium befindet, ist die Idee in der Tat sehr interessant. Bei erfolgreicher Implementierung kann Plasma die Effizienz des Ethereum-Netzwerks drastisch verbessern. Darüber hinaus kann es auch ein nützlicher Ausgangspunkt für andere Blockchain-Netzwerke sein, die in Zukunft möglicherweise eine Skalierungslösung benötigen.

Beachten Sie jedoch, dass Plasma allein kein Projekt ist, sondern ein Framework zum Erstellen skalierbarer Anwendungen, die von verschiedenen Forschungsgruppen oder Unternehmen auf unterschiedliche Weise implementiert werden können.
