# Spot-Rückversicherung
## Was ist Spot-Rückversicherung?

Der Begriff Spot-Rückversicherung bezieht sich auf einen [Versicherungsvertrag , der das Risiko](/insurance) für ein einzelnes Ereignis von einer Versicherungsgesellschaft auf einen [Rückversicherer verteilt.](/reinsurer) Bei einem Spot-Rückversicherungsvertrag wird die Deckung vom ursprünglichen Versicherer auf den Rückversicherer übertragen. Als solches ist es eine Form der Versicherung für Versicherungsunternehmen. Eine Versicherungsgesellschaft kann eine Spot-Rückversicherung abschließen, wenn ein Teilbereich ihres Gesamtportfolios erheblich mehr Risiken birgt als ihr gesamtes Portfolio.

## Spot-Rückversicherung verstehen

Rückversicherung ist eine Form der Versicherung, die Versicherungsunternehmen schützt. Viele Versicherungsunternehmen können sich zusammenschließen, um das Risiko zu teilen, indem sie Policen von anderen Versicherern kaufen. Dadurch können sie das Verlustrisiko im Falle einer größeren Katastrophe oder eines Ereignisses, das zu einer hohen [Auszahlung führen würde, mindern](/payout).

Versicherungsunternehmen beim Abschluss von Rückversicherungsverträgen auf einen Teil der von ihren Versicherungsnehmern gezahlten [Prämien .](/premium) Auf diese Weise bezahlen sie effektiv einen anderen Versicherer, um einige der versicherungstechnischen Risiken aus ihren Büchern zu nehmen.

Ein Rückversicherungsvertrag kann einen ganzen Geschäftszweig oder bestimmte Arten von Policen abdecken. Im Rahmen von Spot-Rückversicherungsverträgen erklärt sich der Rückversicherer bereit, einige oder alle Risiken zu übernehmen, die mit einem einzelnen Ereignis oder Ereignis aus einem katastrophalen Verlust wie Überschwemmungen, Bränden und Naturkatastrophen verbunden sind, insbesondere wenn die Policen ein erhebliches Risiko tragen.

Die Rückversicherung kann es dem Rückversicherer ermöglichen, hinsichtlich der von ihm übernommenen Gefahren selektiv vorzugehen. Dies wird als [fakultative Rückversicherung bezeichnet](/facultative-reinsurance). Oder es kann erforderlich sein, dass der Versicherer automatisch eine Gefahr akzeptiert. Dies wird als [Vertragsrückversicherung bezeichnet](/treaty-reinsurance). Auf diese beiden Arten der Rückversicherung gehen wir im Folgenden näher ein.

## Besondere Überlegungen

Rückversicherung existiert, weil bestimmte Risiken einfach zu hoch sind, als dass sie von einem Unternehmen übernommen werden könnten. Daher suchen Versicherungsunternehmen häufig Rückversicherer auf, um das Risiko zu streuen. Rückversicherer sind große Versicherungsunternehmen, die Versicherungsunternehmen finanziellen Schutz bieten, was bedeutet, dass sie keine Finanzdienstleistungen oder Schutz für Einzelpersonen anbieten. Wie oben erwähnt, ermöglicht die Spot-Rückversicherung Versicherungsunternehmen, das Risiko, das sich aus einem einzelnen Ereignis ergibt, mit einem Rückversicherer zu teilen.

## Fakultative Rückversicherung vs. Vertragsrückversicherung

Die fakultative Rückversicherung ist eine der einfachsten Möglichkeiten für Versicherungsunternehmen, Rückversicherungsschutz zu erhalten. Diese Art von Vereinbarung ermöglicht es einem Rückversicherer, selektiv vorzugehen, indem er ein einzelnes [Risikoprofil](/risk-profile) oder eine bestimmte Gruppe von Risiken wählt. Rückversicherungsunternehmen zeichnen normalerweise die Policen, die sie rückversichern möchten, selbst.

Beispielsweise kann eine Versicherungsgesellschaft Überschwemmungsversicherungspolicen in einem großen geografischen Gebiet abschließen, sich aber dafür entscheiden, nur eine kleine Anzahl von Versicherungsnehmern zu übernehmen. Diese geringe Zahl kann das [Gesamtrisiko des Unternehmens](/aggregate-risk) über seine Grenze bringen, was dazu führt, dass es diese Policen rückversichert.

Die Vertragsrückversicherung hingegen ermöglicht es dem Versicherungsunternehmen, alle mit einer Reihe von Policen verbundenen Risiken an den Rückversicherer abzutreten. Der Rückversicherer zeichnet keine der Policen selbst und verpflichtet sich, den [Zedenten](/indemnity) von allen Risiken freizustellen.

Fakultative Rückversicherungsverträge können teurer sein als Vertragsrückversicherung. Denn die Vertragsrückversicherung deckt ein ganzes Buch oder eine ganze Kategorie von Risiken ab. Dies ist ein Hinweis darauf, dass die Beziehung zwischen dem Versicherer und dem Rückversicherer langfristiger ist, als wenn der Rückversicherer nur einmalige [Transaktionen zur](/transaction) Deckung einzelner Risiken abwickeln würde.

> Während die erhöhten Kosten eine Belastung darstellen, kann eine fakultative Rückversicherungsvereinbarung es dem Versicherer ermöglichen, Kunden zu übernehmen, die er andernfalls möglicherweise nicht annehmen könnte.

## Spot-Rückversicherung kaufen

Versicherungsunternehmen können Spot-Rückversicherungen kaufen, um Policen mit einem anderen Limit als dem, das für ihr Portfolio als Ganzes gewährt wird, abzudecken. Diese Art von Versicherung kann abgeschlossen werden, um eine bestimmte Gefahr oder eine Reihe von Policen an einem bestimmten Ort abzudecken, oder sie kann so spezifisch zugeschnitten werden, dass sie eine einzelne Police abdeckt.

Beispielsweise kann ein Unternehmen, das [Autoversicherungspolicen zeichnet, eine](/auto-insurance) Spot-Rückversicherung erwerben, um einen einzelnen Fahrer abzudecken, der als viel riskanter identifiziert wird als die anderen Fahrer, die es versichert. Indem das mit dem unfallträchtigeren Fahrer verbundene Risiko getrennt wird, verringert der Versicherer die Wahrscheinlichkeit, dass sein allgemeines Policenportfolio gegen seine Deckungsgrenze stößt.

## Höhepunkte

- Versicherungsunternehmen können eine Spot-Rückversicherung erhalten, wenn ein Teil ihres Portfolios ein höheres Risiko birgt als das gesamte Portfolio.

- Versicherungsunternehmen können Spot-Rückversicherungen für Policen erwerben, die ein anderes Limit als das für ihr gesamtes Portfolio gewährte Limit verwenden.

- Versicherer können Spot-Rückversicherung für Policen kaufen, die ein anderes Limit als das für ihr gesamtes Portfolio gewährte Limit verwenden.

- Spot-Rückversicherung ist ein Versicherungsvertrag, der das mit einem einzelnen Ereignis verbundene Risiko von einer Versicherungsgesellschaft auf einen Rückversicherer verteilt.

- Bei der fakultativen Rückversicherung kann der Rückversicherer die Risiken auswählen, zu deren Deckung er sich bereit erklärt, während die Vertragsrückversicherung vom Rückversicherer verlangt, dass er bestimmte Risiken abdeckt.

## FAQ

### Wie verdienen Rückversicherer Geld?

Rückversicherungsunternehmen bieten anderen Versicherern Versicherungsschutz für Policen, von denen sie glauben, dass sie vorhersehbare Risiken aufweisen. Dies bedeutet, dass Rückversicherer eher dazu neigen, Policen abzudecken, die mit nicht spekulativen Risiken verbunden sind. Zum Beispiel ist es weniger wahrscheinlich, dass ein Rückversicherer die Risiken übernimmt, die mit einer Autoversicherungspolice verbunden sind, die einen Fahrer mit hohem Risiko abdeckt. Stattdessen ist es wahrscheinlicher, dass sie Policen oder Fahrer mit sauberen Fahraufzeichnungen rückversichern. Als Gegenleistung für die Deckung tritt der ursprüngliche Versicherer einen Teil oder alle Prämien im Zusammenhang mit den Policen ab, die dem Rückversicherungsvertrag zugeordnet sind.

### Welche zwei Arten von Rückversicherung gibt es?

Fakultative und vertragliche Rückversicherung sind die beiden Hauptarten von Rückversicherungsverträgen, die Versicherungsunternehmen zur Verfügung stehen. Die fakultative Rückversicherung bietet Deckung durch den Rückversicherer für ein einzelnes oder eine Reihe von im Vertrag ausgehandelten Risiken. Die vertragliche Rückversicherung deckt einige oder alle Risiken ab, die ein Versicherungsunternehmen trägt entstehen können. Dies erfolgt normalerweise für einen festgelegten Zeitraum oder auf Vertragsbasis.

### Was ist die größte Rückversicherungsgesellschaft?

Das größte Rückversicherungsunternehmen der Welt ist die Münchener Rückversicherung mit Sitz in Deutschland. Das Unternehmen verzeichnete für das Gesamtjahr 2020 Nettoprämien in Höhe von 43,1 Milliarden US-Dollar. Es folgten Swiss Reinsurance mit Prämien in Höhe von 34,29 Milliarden US-Dollar und Hannover Rück mit Prämien in Höhe von 26,23 Milliarden US-Dollar.

### Was ist der Unterschied zwischen Versicherung und Rückversicherung?

Sowohl Versicherungen als auch Rückversicherungen bieten Deckung gegen Verluste, die aus bestimmten Risiken resultieren. Der Hauptunterschied zwischen den beiden besteht jedoch darin, wer versichert ist. Die Versicherung deckt Einzelpersonen, Unternehmen und andere Einrichtungen gegen Verluste ab, die durch ein Ereignis wie Feuer oder Naturkatastrophen entstehen. Oder sie kann als Schadensersatz für Schäden dienen, die von einer anderen Person wie einem Fahrer bei einem Autounfall verursacht wurden. Die Rückversicherung hingegen schützt Versicherungsunternehmen vor Verlusten. Ein Rückversicherer übernimmt einige oder alle Risiken, die mit einem einzelnen Risiko oder einer Reihe von Risiken verbunden sind, die den Policen zugeordnet sind, gegen einen Teil oder alle Versicherungsprämien.
