# Weiße Liste
Der Begriff Whitelist bezieht sich auf eine Liste zugelassener und identifizierter Personen, Institutionen, Computerprogramme oder sogar Kryptowährungsadressen. Im Allgemeinen beziehen sich Whitelists auf einen bestimmten Dienst, ein Ereignis oder eine Information. Daher können Whitelists je nach Kontext, in dem sie verwendet werden, unterschiedliche Bedeutungen haben.

Wenn sich Benutzer beispielsweise für die Mailingliste eines Unternehmens anmelden, werden sie häufig aufgefordert, die E-Mail-Adresse des Unternehmens zu ihrer Whitelist hinzuzufügen, damit die Nachrichten nicht direkt in den Spam-Ordnern der Benutzer landen. In einigen Fällen können Unternehmen auch Gebühren zahlen, um von Internetdienstanbietern auf die weiße Liste gesetzt zu werden, damit ihre E-Mails nicht automatisch als Spam markiert werden. Durch das Whitelisting einer E-Mail-Adresse können Benutzer also sicherstellen, dass zukünftige E-Mails ihren Posteingang erreichen.

Whitelists können auch im Rahmen der Netzwerksicherheit verwendet werden. Beispielsweise können lokale Netzwerke (LAN) eine Liste vertrauenswürdiger MAC-Adressen einrichten, sodass ein Außenstehender nicht in der Lage ist, sich ihr anzuschließen. In ähnlicher Weise können drahtlose Internet-Router Whitelists verwenden, um nur identifizierten Benutzern zu ermöglichen, von der Internetverbindung zu profitieren.

Abgesehen davon kann sich das Konzept der Whitelist auf eine Liste von Computerprogrammen beziehen, deren Verwendung als sicher gilt. Viele Antivirenprogramme verfügen über eine vordefinierte Liste vertrauenswürdiger Anwendungen, die bei Systemscans nicht anvisiert werden, aber Benutzer können Whitelists auch manuell erstellen.

Schließlich beziehen sich Whitelists in der Blockchain- und Kryptowährungsumgebung entweder auf Initial Coin Offering (ICO)-Ereignisse oder auf Auszahlungsadressen. Im ersten Fall können Kryptowährungsprojekte eine Whitelisting-Phase für Investoren anbieten, die bereit sind, sich am öffentlichen Verkauf ihrer Token zu beteiligen. Daher müsste jeder Investor, der am ICO teilnehmen möchte, seine persönlichen Daten angeben, bevor er auf die Whitelist gesetzt wird (dies erfolgt normalerweise durch ein KYC-Verfahren).

In Bezug auf Auszahlungsadressen bezieht sich Whitelist auf eine Liste von Kryptowährungsadressen, die Benutzer als vertrauenswürdig definieren. In diesem Fall könnten sie nur Geld von ihrem Börsenkonto an die Adressen abheben, die zuvor auf der weißen Liste standen. Bei Binance wird dies auch als Adressverwaltung oder Whitelist für Auszahlungsadressen bezeichnet und ist eine effiziente Möglichkeit für Benutzer, ihre Gelder vor Hackern zu schützen.
