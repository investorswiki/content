# Poisson-Verteilung
## Was ist eine Poisson-Verteilung?

In der [Statistik](/statistics) ist eine Poisson-Verteilung eine [Wahrscheinlichkeitsverteilung](/probabilitydistribution),. die verwendet wird, um anzuzeigen, wie oft ein Ereignis wahrscheinlich über einen bestimmten Zeitraum auftritt. Mit anderen Worten, es handelt sich um eine Zählverteilung. Poisson-Verteilungen werden häufig verwendet, um unabhängige Ereignisse zu verstehen, die innerhalb eines bestimmten Zeitintervalls mit einer konstanten Rate auftreten. Es wurde nach dem französischen Mathematiker Siméon Denis Poisson benannt.

Die Poisson-Verteilung ist eine diskrete Funktion, was bedeutet, dass die Variable nur bestimmte Werte in einer (potenziell unendlichen) Liste annehmen kann. Anders ausgedrückt, die Variable kann nicht alle Werte in einem kontinuierlichen Bereich annehmen. Bei der Poisson-Verteilung kann die Variable nur ganzzahlige Werte (0, 1, 2, 3 usw.) annehmen, ohne Brüche oder Dezimalzahlen.

## Poisson-Verteilungen verstehen

Eine Poisson-Verteilung kann verwendet werden, um abzuschätzen, wie wahrscheinlich es ist, dass etwas „X“ Mal passiert. Wenn zum Beispiel die durchschnittliche Anzahl der Personen, die an einem Freitagabend in einem einzelnen Restaurant Cheeseburger von einer Fast-Food-Kette kaufen, 200 beträgt, kann eine Poisson-Verteilung Fragen beantworten wie: „Wie hoch ist die Wahrscheinlichkeit, dass mehr als 300 Personen dies tun Burger kaufen?" Die Anwendung der Poisson-Verteilung ermöglicht es Managern somit, optimale Scheduling-Systeme einzuführen, die beispielsweise mit einer [Normalverteilung nicht funktionieren würden](/normaldistribution).

Eine der berühmtesten historischen, praktischen Anwendungen der Poisson-Verteilung war die Schätzung der jährlichen Zahl der durch Pferdetritte getöteten preußischen Kavalleriesoldaten. Moderne Beispiele umfassen die Schätzung der Anzahl von Autounfällen in einer Stadt einer bestimmten Größe; In der Physiologie wird diese Verteilung häufig verwendet, um die Wahrscheinlichkeitshäufigkeiten verschiedener Arten von Neurotransmittersekreten zu berechnen. Oder wie hoch wäre die Wahrscheinlichkeit, dass 600 Kunden an einem bestimmten Freitagabend kommen, wenn ein Videogeschäft jeden Freitagabend durchschnittlich 400 Kunden hat?

## Die Poisson-Verteilung im Finanzwesen

Die Poisson-Verteilung wird auch häufig verwendet, um finanzielle Zähldaten zu modellieren, bei denen die Zählung klein ist und oft null beträgt. Als ein Beispiel im Finanzbereich kann es verwendet werden, um die Anzahl der Trades zu modellieren, die ein typischer Investor an einem bestimmten Tag tätigen wird, was 0 (oft) oder 1 oder 2 usw. sein kann.

Als weiteres Beispiel kann dieses Modell verwendet werden, um die Anzahl der "Erschütterungen" des Marktes vorherzusagen, die in einem bestimmten Zeitraum, beispielsweise über ein Jahrzehnt, auftreten werden.

## Höhepunkte

- Eine Poisson-Verteilung, benannt nach dem französischen Mathematiker Siméon Denis Poisson, kann verwendet werden, um abzuschätzen, wie oft ein Ereignis wahrscheinlich innerhalb von "X" Zeiträumen auftritt.

- Poisson-Verteilungen werden verwendet, wenn die interessierende Variable eine diskrete Zählvariable ist.

- Viele Wirtschafts- und Finanzdaten erscheinen als Zählvariablen, z. B. wie oft eine Person in einem bestimmten Jahr arbeitslos wird, und bieten sich daher für die Analyse mit einer Poisson-Verteilung an.

## FAQ

### Wann sollte die Poisson-Verteilung verwendet werden?

Die Poisson-Verteilung wird am besten auf die statistische Analyse angewendet, wenn die betreffende Variable eine Zählvariable ist. Wie oft X beispielsweise vorkommt, basierend auf einer oder mehreren erklärenden Variablen. Zum Beispiel, um abzuschätzen, wie viele fehlerhafte Produkte bei unterschiedlichen Inputs vom Fließband kommen.

### Welche Annahmen macht die Poisson-Verteilung?

Damit die Poisson-Verteilung genau ist, sind alle Ereignisse unabhängig voneinander, die Ereignisrate im Laufe der Zeit ist konstant und Ereignisse können nicht gleichzeitig auftreten. Außerdem sind der Mittelwert und die Varianz einander gleich.

### Ist die Poisson-Verteilung diskret oder stetig?

Da sie diskrete Zählwerte misst, ist die Poisson-Verteilung auch eine [diskrete Verteilung](/discrete-distribution). Dies kann der stetigen Normalverteilung gegenübergestellt werden.
