# Permanentes Portfolio
## Was ist ein permanentes Portfolio?

Das permanente Portfolio ist ein [Anlageportfolio,](/portfolio-investment) das so konzipiert ist, dass es unter allen wirtschaftlichen Bedingungen gut abschneidet. Es wurde in den 1980er Jahren vom Investmentanalysten für den [freien Markt , Harry Browne, entwickelt.](/freemarket)

Das permanente Portfolio besteht zu gleichen Teilen aus Aktien, [Anleihen](/bond),. Gold und Bargeld oder Schatzwechseln.

## Ein permanentes Portfolio verstehen

Das permanente Portfolio wurde von Harry Browne so konstruiert, dass es seiner Meinung nach in jedem Wirtschaftsklima ein sicheres und profitables Portfolio ist. Unter Verwendung einer Variation der effizienten Marktindizierung erklärte Browne, dass ein Portfolio, das zu gleichen Teilen auf Wachstumsaktien, Edelmetalle, Staatsanleihen und Schatzwechsel aufgeteilt ist, eine ideale Anlagemischung für Anleger wäre, die Sicherheit und Wachstum suchen.

## Vor- und Nachteile eines permanenten Portfolios

Browne argumentierte, dass der Portfolio-Mix in allen Arten von wirtschaftlichen Situationen profitabel wäre: Wachstumsaktien würden in expansiven Märkten gedeihen, Edelmetalle in [inflationären](/inflation) Märkten – was steigende Preise bedeutet – Anleihen in Rezessionen und Schatzwechsel in Depressionen.

Browne schuf schließlich den sogenannten Permanent Portfolio Fund mit einer Vermögensmischung, die seinem theoretischen Portfolio im Jahr 1982 ähnelte. Von 1976 bis 2016 hätte ein hypothetisches permanentes Portfolio eine jährliche Rendite von 8,65 % erwirtschaftet, was einer Gesamtrendite von 2.600 % entspricht. Ein standardmäßigeres 60/40-Portfolio von Aktien zu Anleihen hätte eine jährliche Rendite von 10,13 % bei einer Gesamtrendite von 5.050 % erzielt.

Das permanente Portfolio hatte in dieser Zeit jedoch einige Vorteile. Das 60/40-Portfolio hatte eine [Standardabweichung](/standarddeviation) von 9,6, verglichen mit 7,2 für das permanente Portfolio. Während des Marktcrashs im Oktober [1987](/stock-market-crash-1987) wäre das 60/40-Portfolio um 13,4 % im Wert gesunken, während das dauerhafte Portfolio nur um 4,5 % gesunken wäre. Das permanente Portfolio hätte langfristig niedrigere Renditen generiert, wäre aber viel reibungsloser verlaufen. Das macht das Dauerportfolio zu einer attraktiven Option für risikoaverse Anleger.

## Beispiel für ein permanentes Portfolio

Angesichts der Vielzahl der verfügbaren Anlagemöglichkeiten gibt es viele Möglichkeiten, ein dauerhaftes Portfolio aufzubauen. Nachfolgend finden Sie einen Vorschlag, wie Sie diese ausgewogene Mischung erreichen können:

- 25 % in US-Aktien, um in Zeiten des Wohlstands eine starke Rendite zu erzielen. Für diesen Teil des Portfolios empfiehlt Browne einen einfachen [S&P 500-Indexfonds](/sp500) wie den Vanguard 500 Index Fund Admiral Shares (VFIAX).

- 25 % in langfristigen US-Staatsanleihen, die in Zeiten des Wohlstands und in Zeiten der Deflation – oder niedrigerer Preise – gut abschneiden, aber in anderen Wirtschaftszyklen schlecht abschneiden.

- 25 % in bar zur Absicherung gegen Zeiten „knappen Geldes“ oder [Rezession](/recession). In diesem Fall bedeutet „Bargeld“ kurzfristige US-Schatzwechsel.

- 25 % [Edelmetalle](/preciousmetal) (Gold) zum Schutz in Inflationszeiten. Browne empfiehlt Anlagemünzen aus Gold.

Browne empfiehlt, das Portfolio einmal jährlich [neu auszurichten](/rebalancing),. um die Zielgewichtungen von 25 % beizubehalten.

## Höhepunkte

- Der Vorteil besteht darin, dass ein permanentes Portfolio Verluste bei Marktabschwüngen reduziert, was für bestimmte Anleger von Vorteil sein kann.

- Die historische Performance hat gezeigt, dass ein dauerhaftes Portfolio langfristig gut abschneidet, aber nicht so gut wie ein traditionelles 60/40-Aktien-Anleihen-Portfolio.

- Ein permanentes Portfolio besteht zu gleichen Teilen aus Aktien, Anleihen, Gold und Bargeld.

- Ziel eines permanenten Portfolios ist es, durch Diversität unter allen wirtschaftlichen Bedingungen eine gute Leistung zu erbringen.
