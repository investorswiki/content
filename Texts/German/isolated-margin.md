# Isolierte Marge
Isolierte Margin ist der Margin-Saldo, der einer einzelnen Position zugewiesen wird. Der isolierte Margin-Modus ermöglicht es Händlern, ihr Risiko für ihre einzelnen Positionen zu steuern, indem sie die Höhe der jeder einzelnen Position zugewiesenen Marge einschränken. Das zugewiesene Margin-Guthaben für jede Position kann individuell angepasst werden.

Wenn die Position eines Händlers im Isolated Margin-Modus liquidiert wird, wird anstelle seines gesamten Margin-Saldos nur das Isolated Margin-Saldo liquidiert.

Nehmen wir zum Beispiel an, Alice geht eine Long-Position in BTC im Wert von 1000 USD mit 10-facher Hebelwirkung ein. Sie hat ein Margin-Guthaben von 2000 USD, möchte aber nur einen Teil davon für eine einzelne Position riskieren. Sie legt die Isolierte Marge für die Position auf 100 USD fest. Wenn ihre Position liquidiert wird, verliert sie nicht mehr als 100 USD.

Der Betrag der isolierten Marge kann für offene Positionen angepasst werden. Wenn eine Position im Isolated Margin-Modus kurz vor der Liquidation steht, kann die Liquidation verhindert werden, indem der Position zusätzliche Margin zugewiesen wird.

Andererseits ist es nicht möglich, den mit einer Position verknüpften Margin-Modus anzupassen, nachdem sie bereits eröffnet wurde. Es wird dringend empfohlen, die Einstellungen des Margin-Modus zu überprüfen, bevor Sie eine Position eingeben.

Ein weiterer häufig verwendeter Margin-Modus auf Handelsplattformen ist Cross Margin. Im Cross-Margin-Modus wird der gesamte Margin-Saldo auf offene Positionen aufgeteilt, um eine Liquidation zu vermeiden. Wenn Cross Margin aktiviert ist, riskiert der Trader, im Falle einer Liquidation sein gesamtes Margin-Guthaben zusammen mit allen offenen Positionen zu verlieren. Jeder realisierte PnL aus einer anderen Position kann einer Verlustposition helfen, die kurz vor der Liquidation steht.

Typischerweise ist Cross Margin die Standardeinstellung auf den meisten Handelsplattformen, da dies der einfachere Ansatz ist, der für Anfänger geeignet ist. Isolierte Margen können jedoch auch für spekulativere Positionen nützlich sein, die strenge Abwärtsbegrenzungen erfordern.
