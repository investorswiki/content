# Downgraden
## Was ist ein Downgrade?

Eine Herabstufung ist eine negative Änderung der Bewertung der erwarteten Wertentwicklung einer Aktie, die von einem Analysten für ein Finanzdienstleistungsunternehmen herausgegeben wird. Der Analyst weist darauf hin, dass sich die Zukunftsaussichten des Unternehmens verschlechtert haben.

Viele Finanzdienstleistungsunternehmen beschäftigen Analysten, um ihren Kunden Recherchen zur Verfügung zu stellen, einschließlich der Bewertung von Aktien mit einem Kauf-, Halte- oder Verkaufsrating. Eine Herabstufung würde das Rating eines Analysten von „Kaufen“ auf „Halten“ oder von „Halten“ auf „Verkaufen“ ändern. Ein [Upgrade](/upgrade) ist ein Schritt in die entgegengesetzte Richtung.

Schulden haben ein eigenes Bewertungssystem. Eine von mehreren [Ratingagenturen](/bond-rating-agencies) ordnet Schulden basierend auf einer Bewertung der Fähigkeit des Unternehmens, seine Schulden zu begleichen, Briefklassen zu. Wenn eine Anleihe herabgestuft wird, kann sie von einem „A“ [-Rating](/rating) auf ein „BBB“-Rating steigen.

## So funktionieren Downgrades

Analysten geben Aktienempfehlungen ab, um ihren Kunden oder Anlegern eine allgemeine Vorstellung von der erwarteten Wertentwicklung dieses Wertpapiers in der Zukunft zu geben. Die Empfehlungen werden angepasst, wenn sich die Grundlage der Empfehlung ändert.

Die Gründe für eine Herabstufung oder Heraufstufung können eine wichtige Ankündigung des Unternehmens, eine unerwartete Zahl in seinem [Jahresabschluss](/financial-statements) oder ein Nachrichtenereignis sein, das Auswirkungen auf das Unternehmen oder seine Branche hat.

### Herabstufungen von Anleihen

Im Falle von Anleihen gibt es mehrere Ratingagenturen, deren einzige Aufgabe darin besteht, Schuldemittenten zu recherchieren und ihren verschiedenen Schuldtiteln Ratings zuzuweisen. Die wichtigsten Ratingagenturen für Anleihen sind S&P Global, Moody's Investors Service und Fitch Ratings.

Anleihenkäufer achten genau auf ihre Ratings, und viele Rentenfonds investieren nur in Investment-Grade-Anleihen. [Schuldtitel mit einem Rating von](/corporate-credit-rating) „BBB“ und höher gelten als „ [Investment Grade “](/investmentgrade).

Es kann schwerwiegende Auswirkungen auf den Kurs und die Aussichten eines Unternehmens oder einer Regierung haben, wenn die von ihm begebenen Anleihen von „BBB“, was Investment Grade ist, auf „BB“, was unter Investment Grade liegt, herabgestuft werden. Ein Rating unter Investment Grade weist auf sich verschlechternde Fundamentaldaten des ausgebenden Unternehmens oder der Regierung hin.

## Gründe für die Herabstufung

Ein Analyst kann eine Aktie von einem Kauf auf einen Verkauf herabstufen, nachdem das emittierende Unternehmen Informationen über eine Untersuchung der Securities and Exchange Commission über die Geschäftstätigkeit des Unternehmens veröffentlicht hat.

Eine Aktie kann auch herabgestuft werden, weil sich die Fundamentaldaten des ausgebenden Unternehmens verschlechtern oder weil das aktuelle Markt- oder Makroumfeld eine ungünstige Wendung genommen hat.

Bei Aktien und Anleihen führt eine Herabstufung in der Regel zu einer negativen Berichterstattung in den Medien.

Hinter den Kulissen besteht der größte Nachteil einer Herabstufung in höheren Kapitalkosten, sowohl für Fremd- als auch für Eigenkapital. So wie eine Person nach einer Erhöhung der Kreditwürdigkeit möglicherweise einen Kredit zu einem niedrigeren Zinssatz aufnehmen kann, können Unternehmen nach einem positiven Upgrade häufiger und günstiger Geld leihen. Downgrades haben den gegenteiligen Effekt.

### Warnzeichen einer Herabstufung

Ratingagenturen und Aktienanalysten veröffentlichen Beobachtungslisten mit Aktien oder Unternehmen, die von einer Herabstufung bedroht sind. Investoren und Gläubiger behalten diese Listen genau im Auge.

Eine Herabstufung oder Heraufstufung kann von einer Vorhersage für ein bestimmtes Kursziel begleitet werden, das die Aktie voraussichtlich erreichen wird. Beispielsweise könnte ein Analyst eine Aktie von „Kaufen“ auf „Halten“ herabstufen und ein Kursziel festlegen, das einem Rückgang von 2,5 % entspricht. Wenn der Analyst genau verfolgt wird, wird die Aktie in den Stunden nach dieser Ankündigung um 2,5 % im Kurs fallen.

## Höhepunkte

- Eine Herabstufung einer Aktie ist eine Reaktion auf ein unerwartetes negatives Ereignis für ein Unternehmen oder die Branche, in der es tätig ist.

- Eine Herabstufung einer Anleihe ist ein Hinweis auf ein erhöhtes Risiko, dass das Unternehmen oder der Staat, die sich Geld leihen, seine Schulden nicht zurückzahlen kann.

- Eine Herabstufung ist eine negative Änderung des Ausblicks eines Aktienanalysten für eine Aktie oder des Ausblicks einer Ratingagentur für Anleihen.
