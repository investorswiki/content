# Formular 1040-X, geänderte US-Einzeleinkommensteuererklärung
## Was ist das Formular 1040-X, geänderte US-Einkommensteuererklärung für natürliche Personen?

Das 1040-X, Amended US Individual US Income Tax Return ist ein Formular, das von Steuerzahlern verwendet wird, die einen Fehler in einer zuvor eingereichten Bundessteuererklärung korrigieren müssen. Zu den häufigen Fehlern, die mit diesem Formular korrigiert werden, gehören Fehler im Anmeldestatus des Steuerzahlers oder der Anzahl der Familienangehörigen oder Auslassungen von Gutschriften oder Abzügen. Das Formular 1040-X sollte nicht verwendet werden, um einfache mathematische Fehler in einer Steuererklärung zu korrigieren, da der Internal Revenue Service (IRS) solche Fehler routinemäßig während der Bearbeitung von Steuererklärungen korrigiert.

## Wer kann das Formular 1040-X einreichen?

Ein 1040-X-Formular sollte von jedem eingereicht werden, der bereits eine Steuererklärung eingereicht hat und diese aus einem der folgenden Gründe ändern muss:

- Um Formular [1040](/1040),. 1040-SR oder 1040-NR zu korrigieren. Die meisten Steuerzahler verwenden eines dieser drei Formulare, um ihre jährliche Steuererklärung einzureichen.

- Um nach Ablauf der Frist eine Wahl zu treffen. Zum Beispiel kann ein verheirateter Steuerzahler wählen, ob er als Einzelperson oder gemeinsam mit einem Ehepartner einreichen möchte.

- Um einen Betrag zu ändern, der zuvor vom IRS angepasst wurde

- Zur Geltendmachung eines Rücktragsanspruchs aufgrund eines Verlustes oder ungenutzten Guthabens

## So reichen Sie das Formular 1040-X ein

> Sie können bis zu drei "akzeptierte" geänderte Erklärungen elektronisch einreichen. Nach dem dritten werden alle anderen abgelehnt.

>

Dieses Formular ist eine detaillierte, zeilenweise Beschreibung aller möglichen Anpassungen, sodass der Steuerzahler die genaue Art und Höhe jeder Änderung klar erfassen und eine kurze Beschreibung dessen eingeben kann, was geändert wird und warum.

Der Antragsteller muss nicht nur den 1040-X, sondern eine überarbeitete Version des gesamten 1040 oder 1040-SR, einschließlich aller beigefügten Formulare und Anlagen, auch wenn sie nicht geändert wurden, beifügen.

Steuerzahler können das Formular 1040-X per E-Mail einreichen oder per Post an den IRS senden. Formular 1040-X kann nur eingereicht werden, nachdem eine Jahreserklärung eingereicht wurde.

Um eine Gutschrift zu erhalten, müssen Steuerzahler innerhalb von drei Jahren nach Einreichung der ursprünglichen Erklärung oder innerhalb von zwei Jahren nach Zahlung der Steuer ein 1040-X-Formular einreichen, je nachdem, welcher Zeitpunkt später liegt.

Der IRS gibt Steuerzahlern die folgenden Tipps zum Formular 1040-X:

- Warten Sie auf die erste Rückerstattung, falls eine fällig war. Wenn Ihre ursprüngliche Steuererklärung eine Rückerstattung enthielt, sollten Sie warten, bis Sie diese erhalten, bevor Sie das Formular 1040-X einreichen. Der IRS wird eine separate, zweite Rückerstattung ausstellen, wenn dies gerechtfertigt ist, also können Sie den ersten Scheck gerne einlösen.

- Reichen Sie innerhalb der dreijährigen Änderungsfrist ein 1040-X-Formular ein. Wenn sich die Änderung auf Ihren Erstattungsbetrag auswirkt, müssen Sie das Formular innerhalb von drei Jahren ab dem Datum Ihrer ursprünglichen Steuererklärung oder innerhalb von zwei Jahren ab dem Datum der Zahlung des Steuerbetrags einreichen, je nachdem, welcher Zeitpunkt später liegt. Alle vorzeitig eingereichten Erklärungen gelten als am offiziellen Stichtag für die Steuererklärung eingereicht, der normalerweise der 15. April des Kalenderjahres ist.

## Höhepunkte

- Um eine Rückerstattung zu erhalten, muss das Formular innerhalb von drei Jahren nach Einreichung der ursprünglichen Erklärung oder innerhalb von zwei Jahren nach Zahlung der Steuer eingereicht werden, je nachdem, welcher Zeitpunkt später liegt.

- Sie können das Formular elektronisch einreichen, wenn Sie die entsprechende Steuererklärung elektronisch eingereicht haben. Ansonsten einsenden.

- Das IRS-Formular 1040-X wird von Steuerzahlern eingereicht, um eine zuvor eingereichte jährliche Steuererklärung zu ändern.

- Formular 1040-X sollte verwendet werden, um materielle Änderungen zu korrigieren, aber nicht, um mathematische Fehler zu beheben.

## FAQ

### Müssen Sie das Formular 1040-X zusammen mit dem Formular 1040 einreichen?

Sie müssen das Formular 1040-X einreichen, nachdem Sie Ihre 1040, 1040-SR oder 1040-NR eingereicht haben. Sie ergänzen Ihre frühere Steuererklärung mit den korrekten Informationen.

### Wann sollte ich meine geänderte Steuererklärung erhalten?

Laut IRS kann es bis zu 16 Wochen dauern (20 Wochen ab 2022 aufgrund von Verzögerungen aufgrund der Coronavirus-Pandemie), um eine geänderte Steuererklärung zu bearbeiten, selbst wenn sie elektronisch eingereicht wird. Ungefähr drei Wochen nachdem Sie das 1040-X eingereicht haben, können Sie den Status Ihrer Einreichung auf der IRS-Website unter Where's My Amended Return? Seite.

### Wohin senden Sie das Formular 1040-X?

Wenn Sie Ihre Jahreserklärung elektronisch eingereicht haben, können Sie jetzt ein 1040-X mit einer Gewerbesteuer-Vorbereitungssoftware elektronisch einreichen. Das ist der schnellste Weg, um Ihr Geld zurückzubekommen, falls Ihnen etwas zusteht. Wenn Sie möchten, können Sie es per Post senden. Der IRS listet vier separate Adressen auf, die von Einwohnern verschiedener Regionen und von im Ausland lebenden Amerikanern verwendet werden können.

### Können Erstattungen nach Formular 1040-X direkt eingezahlt werden?

Nein. Eine direkte Einzahlung ist derzeit nicht für Rückerstattungen auf der Grundlage geänderter Rückgaben verfügbar. Sie erhalten einen Scheck per Post vom IRS.

### Wofür wird ein 1040-X-Formular verwendet?

Das 1040-X ermöglicht es einem Steuerzahler, eine zuvor eingereichte Steuererklärung zu ändern oder zu korrigieren. Wenn Sie vergessen haben, eine Gutschrift zu beantragen oder einen Abzug zu erfassen, verwenden Sie dieses Formular. Wenn Sie einen mathematischen Fehler gemacht haben, vergessen Sie ihn. Der IRS führt immer die Nummern durch und korrigiert diese Art von Fehlern.
