# Bestätigung auf einem Diagramm
## Was ist eine Bestätigung auf einem Diagramm?

Die Bestätigung auf einem Chart beschreibt ein Chartmuster, das eine nachhaltige Gelegenheit zum Aktienhandel zeigt, die aufgrund ihrer Beständigkeit bestätigt wird (bei Glaubwürdigkeit). Dies erfordert normalerweise mindestens drei Tage, die aus mehreren Datenpunkten bestehen, bevor bestätigt wird, dass ein neuer Trend oder eine neue Musterbildung im Gange ist.

## Wie die Bestätigung auf einem Diagramm funktioniert

Die Bestätigung auf einem Chart ist einer von vielen Indikatoren, denen technische Analysten folgen. Technische Anleger interessieren sich hauptsächlich für Charttrends und weniger für Aktienfundamentaldaten [wie](/fundamentalanalysis) Unternehmensverkäufe und Cashflow. Technische Analysten verwenden die Bestätigung auf einem Chart als unterstützenden Beweis, wenn sie ihre Kauf- und Verkaufsempfehlungen abgeben. Händler zeichnen oft mehrere Indikatoren gleichzeitig auf, um so viele Daten wie möglich zu liefern, wenn sie überlegen, ob sie eine Aktie kaufen oder verkaufen sollen. Es ist üblich, dass technische Trader auf einem Chart von drei Charts nach einer Bestätigung suchen, um ihre Überzeugung zu untermauern.

Beim technischen Investieren durch die Verwendung von Charts geht es darum, Muster zu verstehen und zu erkennen. Sobald Sie ein Muster visualisieren und benennen können, können Sie viele Jahre zurückblicken, um festzustellen, wie effektiv dieses bestimmte Muster bei der Bestimmung quantifizierbarer Trends war. Was wie ein Chartmuster aussieht, ist oft nur eine weitere Seitwärtsbewegung innerhalb einer laufenden Handelszone, was bedeutet, dass keine bestimmte Richtung erkannt wurde. Die Bestätigung auf einem Diagramm erfolgt, wenn die vorhergesagte Bewegung tatsächlich eintritt. Das Lexikon der Namen von Diagrammmustern ist umfangreich, mit einer Vielzahl unterhaltsamer Namen, die von [verlassenem Baby](/bullish-abandoned-baby) bis zu [dunkler Wolke reichen](/darkcloud). Jedes dieser Muster hat eine charakteristische Form.

## Candlesticks mit vier Datenpunkten bestätigen

Candlestick-Muster verwenden normalerweise vier Datenpunkte, um ihre Formen zu definieren. Dies sind insbesondere der Eröffnungskurs der Aktie oder des Vermögenswerts, das Tageshoch, das Tagestief und der Schlusskurs. Zusammengenommen beschreiben diese vier Informationen ein bestimmtes Preisbewegungsmuster für einen bestimmten Tag. In der Praxis können Candlesticks über eine Reihe von Tagen kombiniert werden, um Handelsentscheidungen zu treffen.

Ein Beispiel für einen Candlestick wird [Hammer genannt](/hammer),. die Form, die entsteht, wenn der Aktienkurs deutlich nach unten öffnet, sich dann aber auf ein neues Hoch erholt. Das Gegenteil gilt auch, wie beim Muster des [hängenden Mannes zu sehen ist.](/hangingman)

[Candlestick-Muster](/candlestick) werden von technischen Händlern genau beobachtet, die hoffen, dass sich die Ergebnisse im Laufe der Zeit wiederholen. Der Doji ist das Muster, das gebildet wird, wenn eine Aktie zum nahezu gleichen Preis eröffnet und schließt. Die [Doji-](/doji) Zahl sieht aus wie ein Kerzenkreuz oder umgekehrtes Kreuz und zeigt an, dass Unentschlossenheit die Hauptursache für das Fehlen einer nachhaltigen Bewegung einer Aktie sein kann.

## Bestätigung auf einem Chart als ein Tool in der Toolbox

Der technische Handel funktioniert gut, wenn die Zeiten ziemlich stabil sind. Aber umsichtige Anleger wissen, dass sie die stärkeren Winde im Auge behalten müssen, die in einer Wirtschaft seismische Verschiebungen verursachen können, die nichts mit dem Wert einer bestimmten Aktie oder den Chartbewegungen zu tun haben. Eine Analogie ist die eines Maurers, der seine Ziegel entlang einer neuen Mauer positioniert, ohne zu bemerken, dass die im Bau befindliche Kathedrale auf einem sich verschiebenden Fundament steht. In dieser Analogie ist die Kathedrale die Summe aller wirtschaftlichen Kräfte, die während eines bestimmten Zeitraums am Werk waren, und die Mauer ist ein einzelnes Bauteil. Erfahrene Anleger wissen, dass sie den größeren Kräften, die eine Wirtschaft umgestalten können, große Aufmerksamkeit schenken müssen, wenn sie ihre vielen kurzfristigen Charting-Tools verwenden.

## Höhepunkte

- Candlestick-Muster werden an nur einem Handelstag anhand der Eröffnungs-, Schluss-, Höchst- und Tiefstkurse bestätigt, können aber für unterschiedliche Analysen über mehrere Tage kombiniert werden.

- Die Bestätigung erfordert mehrere Datenpunkte, typischerweise im Laufe von mindestens drei Handelstagen.

- Bestätigung auf einem Chart bezieht sich auf mehrere Datenpunkte, die die Gültigkeit eines technischen Musters oder Trends auf einem Preischart bestätigen oder Glaubwürdigkeit verleihen.
