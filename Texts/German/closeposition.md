# Position schließen
## Was ist eine geschlossene Position?

Das Schließen einer Position bezieht sich auf die Ausführung einer Wertpapiertransaktion, die genau das Gegenteil einer [offenen Position](/open-position) ist, wodurch sie annulliert und das anfängliche Risiko eliminiert wird. Das Schließen einer Long-Position in einem Wertpapier würde den Verkauf bedeuten, während das Schließen einer Short-Position in einem Wertpapier den Rückkauf bedeuten würde. Das Eingehen von Gegenpositionen in Swaps ist ebenfalls sehr üblich, um das Risiko vor Fälligkeit zu beseitigen.

Das Schließen einer Position wird auch als „Position Squaring“ bezeichnet.

## Close-Positionen verstehen

Wenn Trades und Investoren auf dem Markt Geschäfte tätigen, öffnen und schließen sie Positionen. Die anfängliche Position, die ein Anleger bei einem Wertpapier eingeht, ist eine offene Position, und dies kann entweder eine [Long-Position](/long) oder eine [Short-Position](/short) auf den Vermögenswert sein. Um aus der Position herauszukommen, muss sie geschlossen werden. Ein Long wird [verkaufen, um zu schließen](/selltoclose) ; ein kurzer wird [kaufen, um zu schließen](/buytoclose).

Das Schließen einer Position beinhaltet somit die entgegengesetzte Aktion, die die Position ursprünglich eröffnet hat. Ein Investor, der beispielsweise Microsoft (MSFT)-Aktien gekauft hat, hält diese Wertpapiere auf seinem Konto. Wenn er die Aktien verkauft, schließt er die Long-Position auf MSFT.

Die Differenz zwischen dem Preis, zu dem die Position in einem Wertpapier eröffnet wurde, und dem Preis, zu dem sie geschlossen wurde, stellt den [Bruttogewinn](/grossprofit) oder -verlust aus dieser Wertpapierposition dar. Positionen können aus einer Vielzahl von Gründen geschlossen werden – um Gewinne mitzunehmen oder Verluste einzudämmen, das Risiko zu reduzieren, Barmittel zu generieren usw. Ein Anleger, der beispielsweise seine [Kapitalertragssteuerschuld ausgleichen möchte](/capital_gains_tax),. wird seine Position auf ein verlierendes Wertpapier der Reihe nach schließen einen Verlust realisieren oder [ernten](/taxgainlossharvesting).

Der Zeitraum zwischen Eröffnung und Schließung einer Position in einem Wertpapier gibt die [Haltedauer](/holdingperiod) für das Wertpapier an. Diese Haltedauer kann je nach Präferenz des Anlegers und Art des Wertpapiers stark variieren. Beispielsweise schließen [Daytrader](/daytrader) Handelspositionen im Allgemeinen am selben Tag, an dem sie eröffnet wurden, während ein langfristig orientierter Investor eine Long-Position in einer [Blue-Chip-Aktie](/bluechipstock) viele Jahre nach Eröffnung der Position glattstellen kann.

Es ist möglicherweise nicht erforderlich, dass der Anleger Positionen für Wertpapiere schließt, die begrenzte Fälligkeits- oder Ablaufdaten haben, wie z. B. Anleihen und Optionen. In solchen Fällen wird die Schlussposition automatisch bei Fälligkeit der Anleihe oder Verfall der Option generiert.

## Besondere Überlegungen

Während die meisten Positionen nach Ermessen der Anleger geschlossen werden, werden Positionen manchmal unfreiwillig oder gewaltsam geschlossen. Beispielsweise kann eine Long-Position in einer Aktie, die in einem [Margin-Konto gehalten](/marginaccount) wird, von einer Maklerfirma glattgestellt werden, wenn die Aktie stark fällt und der Anleger nicht in der Lage ist, die [erforderliche zusätzliche Margin aufzubringen](/maintenancemargin). Ebenso kann eine Short-Position im Falle eines [Short Squeeze einem](/shortsqueeze) [Buy-In unterliegen](/buyin).

Eine geschlossene Position kann teilweise oder vollständig sein. Wenn das Wertpapier [illiquide ist](/illiquid),. kann der Anleger möglicherweise nicht alle seine Positionen auf einmal zum angegebenen [Grenzpreis schließen](/limitorder). Außerdem kann ein Anleger absichtlich nur einen Teil seiner Position schließen. Beispielsweise kann ein Kryptohändler, der eine offene Position auf drei XBT (Token für Bitcoin) hat, seine Position auf nur einem Token schließen. Dazu gibt er eine Verkaufsorder für einen XBT ein, wodurch er zwei offene Positionen in der Kryptowährung hat.

## Beispiel einer geschlossenen Position

Angenommen, ein Investor hat eine Long-Position auf die Aktie ABC eingegangen und erwartet, dass der Kurs ab dem Datum seiner Investition um das 1,5-fache steigen wird. Der Anleger schließt seine Investition, nachdem der Preis das gewünschte Niveau erreicht hat, indem er die Aktie verkauft.

## Höhepunkte

- Eine Glattstellungstransaktion wird im Allgemeinen von einem Händler initiiert, kann aber in einigen Fällen auch von Maklerfirmen zwangsweise geschlossen werden, wenn bestimmte Bedingungen erfüllt sind.

- Das Schließen einer Position bezieht sich auf das Aufheben einer bestehenden Position auf dem Markt, indem die entgegengesetzte Position eingenommen wird.

- Bei einem Leerverkauf würde dies den Rückkauf des Wertpapiers bedeuten, während eine Long-Position den Verkauf des Wertpapiers beinhaltet.
