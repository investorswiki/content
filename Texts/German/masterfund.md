# Master-Fonds
## Was ist ein Master-Fonds?

Ein Master-Fonds ist ein kollektiver Pool von Vermögenswerten, der in einer Master-Feeder-Anlagestruktur verwendet wird, die den Vorteil reduzierter Betriebskosten und Handelskosten bietet.

## Master-Fonds erklärt

Ein Master-Fonds ist ein Investmentfonds, der für den Handel mit Wertpapieren verwendet wird, wenn eine Master-Feeder-Struktur verwendet wird. Eine Master-Feeder-Struktur baut auf dem Konzept auf, Portfolios aus einem gemeinsamen Anlagepool zu verwalten. Es ermöglicht einer Fondsgesellschaft, die zahlreiche Feeder-Fonds mit ähnlichen Zielen verwaltet, umfassender von transaktionalen [Skaleneffekten zu profitieren](/economiesofscale).

Master-Feeder-Anordnungen können mit vielen verschiedenen Poolfonds verwendet werden. Die Entscheidung für eine Master-Feeder-Struktur wird von Fall zu Fall getroffen, da es zahlreiche Vorteile und oft viele Faktoren gibt.

## Offene Investmentfonds

Ein [offener Investmentfonds](/open-endfund) wird als kollektiver Anlagepool verwaltet, der mehrere [Anteilsklassen anbietet](/share_class),. die für verschiedene Arten von Anlegern strukturiert sind. Die Vermögenswerte werden gemeinsam in einem einzigen Portfolio gepoolt, und die Differenzierung nach Anteilsklassen wird von den Buchhaltern des Fonds verwaltet.

Ein offener Investmentfonds kann sich dafür entscheiden, seiner gemeinsamen Anlage eine zusätzliche Schicht hinzuzufügen, indem er eine Master-Feeder-Struktur verwendet. Dies kann für eine Fondsgesellschaft effizient sein, wenn die Investmentgesellschaft mehrere offene Fonds mit ähnlichen Anlagezielen und Beteiligungen verwaltet. In diesem Fall würde eine Master-Feeder-Struktur einen Fund-of-Funds-Ansatz verwenden, bei dem mehrere offene Fonds [Feeder-Fonds sind](/feederfund),. die Vermögenswerte umfassend in einem Master-Fonds bündeln.

Bei dieser Art von Dachfondsstruktur werden alle Transaktionen vom Master-Fonds aus getätigt. Im Master-Fonds können Wertpapiertransaktionen aggregiert und damit kostengünstiger gehandelt werden.

## Master-Feeder-Alternativen

Andere Arten von Fonds, außerhalb traditioneller offener Fonds, können sich ebenfalls für den Aufbau einer Master-Feeder-Struktur entscheiden. Ein Master-Fonds kann mit Feeder-Fonds eingerichtet werden, wenn eine erhebliche Differenzierung für verschiedene Arten von Anlegern erforderlich ist. Beispielsweise kann ein Fonds mit US- und [Offshore](/offshore) -Investoren eine Master-Feeder-Struktur nutzen, indem er zwei Feeder-Fonds anbietet, die eine differenzierte Anlage von US- und Offshore-Investoren ermöglichen.

## Regulierung und Offenlegung

Master-Feeder-Strukturen können von allen Fondsarten genutzt werden. Wenn eine Master-Feeder-Struktur in einem in den USA regulierten Fonds verwendet wird, wird dies im Verkaufsprospekt eines Fonds offengelegt, wobei die Bedingungen der Master-Feeder-Struktur im Detail angegeben werden. Master-Feeder-Fonds können eine nützliche Möglichkeit sein, die Kosteneffizienz eines Fonds zu verbessern, die etwas kompliziertere Struktur kann sie jedoch in Ländern mit minimalen Vorschriften zu einer risikoreicheren Anlage machen.
