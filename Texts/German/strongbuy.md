# Starker Kauf
## Was ist ein starker Kauf?

Ein starker Kauf ist eine Empfehlung von Analysten für eine Aktie, von der erwartet wird, dass sie die durchschnittliche Marktrendite und/oder die Rendite vergleichbarer Aktien im selben Sektor oder der gleichen Branche [deutlich übertrifft .](/outperform) Es stellt die nachdrückliche [Bestätigung](/endorsement) einer Aktie durch einen Analysten dar.

Einem starken Kauf kann ein [starker Verkauf gegenübergestellt werden](/strongsell).

## Einen starken Kauf verstehen

Es wird erwartet, dass eine Aktie mit einem „starken Kauf“-Rating die Märkte kurzfristig deutlich übertreffen wird. Ein starker Kauf ist die stärkste Empfehlung, die ein [Analyst](/analyst) zum Kauf einer Aktie geben kann. Wie bei jeder Art von Analystenrating ist das Rating nur so lange relevant, bis ein wesentliches Ereignis eintritt, das dazu führt, dass der Analyst seinen Ausblick auf das Unternehmen ändert. Ein „starker Kauf“ bedeutet, dass der Analyst glaubt, dass das der Aktie zugrunde liegende Unternehmen eine positive [finanzielle Performance](/financialperformance) und/oder günstige Marktbedingungen aufweist oder bald erfahren wird.

Ein starkes Kaufrating zeigt an, dass ein Analyst Grund zu der Annahme hat, dass die Aktie in den kommenden Monaten drastisch höher gehandelt wird. Was genau diese Gründe sind, kann sehr unterschiedlich sein, aber im Allgemeinen prognostiziert der Analyst den potenziellen Gewinn normalerweise in Erwartung eines positiven Ereignisses, wie z. B. der Rückkehr zur Rentabilität oder der Einführung eines neuen Produkts. Ein starkes Kaufschreiben wird normalerweise von einem äußerst optimistischen Kursziel für die Aktie begleitet, wie z. B. einem Gewinn von 30 % bis 50 % in den kommenden 12 Monaten.

Die starke Kaufempfehlung wird längst nicht mehr so hoch geschätzt wie noch vor wenigen Jahren. Daher halten viele Investmentbanken derzeit lieber an der weniger emotionalen Kaufempfehlung fest.

> Es gibt kein universelles Rankingsystem, daher ist es wichtig, die Ratingskala eines Unternehmens zu verstehen, um zu wissen, was sie mit den verwendeten Begriffen meint.

>

## Mögliche Gründe für starke Kaufempfehlungen

- **Anstieg des Aktienkurses:** Wenn die Aktienkursentwicklung eines Unternehmens einen beeindruckenden Lauf zeigt, suchen Analysten nach Gründen, die zu diesem Lauf beitragen. Wenn sie feststellen, dass diese Gründe auf eine anhaltend starke Performance hindeuten, können sie motiviert sein, eine starke Kaufempfehlung auszusprechen.

- **Schätzungsrevisionen:** Wenn ein Unternehmen seine Umsatzschätzungen für einen bestimmten Zeitraum anpasst, um eine stärkere erwartete Leistung anzuzeigen, können Analysten diese Einschätzung wiederholen, indem sie ihre Empfehlung für die Aktie auf einen starken Kauf zurücksetzen.

- **Eine Geschichte positiver Gewinnüberraschungen:** Wenn ein Unternehmen eine starke Erfolgsbilanz darin hat, seine eigenen Leistungsprognosen zu übertreffen, können Analysten eine Aktie als überlegene Option im Vergleich zu Wettbewerbern betrachten, was ein starkes Kaufrating angemessen machen würde.

- **Starke Wachstumsaussichten:** Wenn ein Unternehmen in einer starken Branche tätig ist und ein starkes Wachstum prognostiziert, kann dies ein Grund für eine starke Kaufempfehlung sein.

- **Wachstumstreiber:** Wenn ein Unternehmen bedeutende technologische Fortschritte in einem Sektor erzielt oder dazu beiträgt, kann dies den Respekt eines Analysten und eine nachfolgende starke Kaufempfehlung einbringen.

## Höhepunkte

- Ein starkes Kaufrating wird normalerweise von einem äußerst optimistischen Kursziel für die Aktie begleitet, z. B. einem Kursgewinn von 30 % bis 50 % in den kommenden 12 Monaten.

- Die Kaufempfehlung ist längst nicht mehr so hoch wie noch vor wenigen Jahren. Daher halten viele Investmentbanken derzeit lieber an der weniger emotionalen Kaufempfehlung fest.

- Ein starker Kauf ist die Empfehlung eines Analysten, Aktien eines Unternehmens zu kaufen, von dem aufgrund einer Analyse erwartet wird, dass es kurz- bis mittelfristig eine deutliche Outperformance erzielen wird.
