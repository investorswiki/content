# Geschenk im Vertrauen
## Was ist eine Vertrauensgabe?

Eine treuhänderische Schenkung ist eine besondere rechtliche und [treuhänderische](/fiduciary) Gestaltung, die eine indirekte [Vermögensvererbung](/bequest) an einen Begünstigten ermöglicht. Zweck einer treuhänderischen Schenkung ist es, die Besteuerung von Schenkungen zu vermeiden, die die [jährliche Schenkungssteuerfreigrenze](/annual-exclusion) überschreiten. Diese Art von Vertrauen wird häufig verwendet, um Vermögen an die nächste Generation zu übertragen.

## Ein Vertrauensgeschenk verstehen

[Treuhandfonds](/trust-fund) für ihre Kinder oder Enkel einrichten möchten . Die Errichtung eines Trusts ist eine [Nachlassplanungsstrategie](/estateplanning),. die verwendet werden kann, um Vermögenswerte oder Vermögen vom [Grantor](/estateplanning) – dem Eigentümer der Vermögenswerte, die das Trust gründen – an einen Begünstigten zu übertragen, der das Vermögen erhält [.](/grantor) Das Vermögen kann auf Wunsch des Spenders übertragen werden, was bedeutet, dass Beschränkungen festgelegt werden können, sodass der Empfänger nur dann auf das Geld zugreifen kann, wenn die im Trust festgelegten Anweisungen erfüllt wurden. Beispielsweise könnte ein Elternteil einen Trust gründen, in dem das Kind bis zu seinem 21. Geburtstag nicht auf die Gelder zugreifen kann.

### Jährlicher Schenkungssteuerfreibetrag

Typischerweise besteuert der [Internal Revenue Service (IRS)](/irs) die Schenkung von Vermögenswerten von einer Person an eine andere. Die Schenkungssteuer wird jedoch nur ausgelöst, wenn der Wert des zu übertragenden Vermögens einen bestimmten Betrag übersteigt, der als jährlicher Schenkungssteuerfreibetrag bezeichnet wird. Der jährliche Ausschlussbetrag für Geschenke beträgt 15.000 USD für das Steuerjahr 2021 und 16.000 USD für 2022.

### Crummy-Vertrauen

Eine treuhänderische Schenkung ist eine praktikable Methode, um Steuern auf Schenkungen zu vermeiden, die die jährliche Schenkungssteuer-Ausschlussgrenze überschreiten. Schenker können über den jährlichen Ausschluss hinaus Geschenke machen, ohne Steuern zahlen zu müssen, indem sie eine besondere Art von Treuhand gründen, wie z. B. einen [Crumsey Trust](/crummey-trust). Eine Schenkung an einen Crummey Trust ermöglicht es dem Begünstigten, das Schenkungsvermögen für eine begrenzte Zeit abzuheben, wodurch die Schenkung als gegenwärtiges Interesse angesehen wird und für die Schenkungssteuerbefreiung in Frage kommt. Wenn das Geschenk dieses zeitlich begrenzte Widerrufsrecht nicht hätte, würde es als zukünftiges Interesse betrachtet und der [Schenkungssteuer unterliegen](/gifttax).

Beispielsweise könnte der Trust so eingerichtet werden, dass der Begünstigte innerhalb eines festgelegten Zeitraums, z. B. innerhalb von 60 oder 90 Tagen, Auszahlungen vornehmen kann. Danach fallen die im Trust gehaltenen Schenkungsgelder unter die festgelegten Auszahlungsregeln, die vom Stifter des Trusts festgelegt wurden. Nehmen wir in unserem Beispiel an, dass ein Elternteil festlegt, dass ein Kind bis zu seinem 21. Lebensjahr keinen Zugriff auf das Treuhandgeld hat. Selbst wenn das Kind beschließt, das Treuhandvermögen sofort in Anspruch zu nehmen, hat es nur Zugriff auf das letzte Geschenk, wie auf alle vorherigen Geschenkgelder innerhalb des Treuhandkontos geschützt bleiben.

> Eine [Crommey-Bestimmung](/crummeypower) kann auch in einer anderen Art von Treuhand untergebracht werden. Zum Beispiel enthalten traditionelle Lebensversicherungstrusts oft eine Crommey-Bestimmung.

## Vor- und Nachteile einer Vertrauensgabe

Neben steuerlichen Vorteilen ist eine treuhänderische Schenkung eine Möglichkeit, ein finanzielles Polster für künftige Generationen aufzubauen. Die Übertragung von Vermögen von einer Generation auf die nächste per [Testament](/will) oder anderen Erbmitteln ist sowohl logistisch als auch emotional ein kompliziertes Unterfangen. Gleichzeitig können diese Regeln Einzelpersonen, Familien und Gemeinschaften enorme Vorteile bringen. Das Verständnis der Nuancen des Schenkens kann sowohl für Spender als auch für Begünstigte einen Mehrwert bringen.

Ein potenzieller Nachteil einer treuhänderischen Schenkung besteht darin, dass die Bereitstellung von sofortigem Zugang zu beträchtlichen Beträgen für Begünstigte – insbesondere Kinder – die Fähigkeit des Fonds gefährden kann, langfristiges Vermögen anzuhäufen. Einige Familien umgehen dies, indem sie Einschränkungen festlegen, wie z. B. die Begrenzung der Höhe oder Häufigkeit von Auszahlungen oder die Beendigung zukünftiger Geschenke an Empfänger, die sofort Geld abheben.

## Höhepunkte

- Eine Art von treuhänderischer Schenkung ist eine treuhänderische Schenkung nach Crumsey, die die Gewährung von Schenkungen für einen bestimmten Zeitraum zulässt, wobei die Schenkungen als gegenwärtiges Interesse gelten und für die Schenkungssteuerbefreiung in Frage kommen.

- Ein Nachteil einer treuhänderischen Schenkung besteht darin, dass sie ohne Einschränkungen errichtet wird und es einem Begünstigten, beispielsweise einem Kind, ermöglicht, große Beträge abzuheben, wodurch die finanzielle Lebensfähigkeit des Fonds gefährdet wird.

- In der Regel besteuert der IRS den Wert eines Geschenks, das übertragen wird, bis zum jährlichen Freibetrag für die Schenkungssteuer.

- Eine treuhänderische Schenkung ist eine Möglichkeit, Steuern auf Schenkungen zu vermeiden, die den jährlichen Freibetrag für die Schenkungssteuer übersteigen.

- Treuhandgeschenke werden üblicherweise verwendet, um Vermögen von einer Generation an die nächste weiterzugeben, indem ein Treuhandfonds eingerichtet wird.
