# Minderleistung
## Was ist Underperformance?

Wenn eine Anlage unterdurchschnittlich abschneidet, kann sie nicht mit anderen Wertpapieren Schritt halten. In einem steigenden Markt beispielsweise entwickelt sich eine Aktie unterdurchschnittlich, wenn sie keine Gewinne verzeichnet, die dem Anstieg des [S&P 500 Index entsprechen oder darüber hinausgehen](/sp500). In einem fallenden Markt ist eine Aktie, die schneller fällt als der [breitere Markt](/broad-basedindex),. ein Underperformer. „Underperform“ ist auch eine Analystenempfehlung, die einer Aktie zugewiesen wird, wenn erwartet wird, dass Aktien etwas schlechter abschneiden als die Marktrendite. Die Bezeichnung ist auch als „moderates Verkaufen“ oder „schwaches Halten“ bekannt.

## Verständnis der Underperformance-Bezeichnung

Die genauen Definitionen variieren zwischen den Brokern, aber ein „Underperform“ [-Rating](/rating) ist im Allgemeinen schlechter als „Neutral“, aber besser als „Verkaufen“ oder „ [Starker Verkauf](/strongsell) “.

- Neutral wird einer Aktie zugewiesen, von der erwartet wird, dass sie Ergebnisse liefert, die dem breiteren Markt entsprechen.

- Underperform ist eine Aktie, die wahrscheinlich leicht unterdurchschnittlich abschneiden wird: größere Verluste in einem fallenden Markt und unterdurchschnittliche Gewinne in einem steigenden Markt.

- Eine Verkaufsempfehlung wird einer Aktie gegeben, von der erwartet wird, dass sie an Wert verliert.

- Starker Verkauf spiegelt Bedenken wider, dass das Unternehmen in großen Schwierigkeiten steckt und die Aktie erhebliche Verluste erleiden könnte.

Ein Wertpapier kann die Underperform-Bezeichnung erhalten, wenn es eine Metrik, mit der es verglichen wird, nicht erfüllt oder überschreitet. Der Vergleich kann mit dem Gesamtmarkt, einem konkurrierenden Unternehmen oder einem [Index erfolgen](/index). Eine Vielzahl anderer Probleme könnte das Underperform-Rating nach sich ziehen, wie z. B. Bedenken hinsichtlich der Verschuldung des Unternehmens, des [Kurs-Gewinn-Verhältnisses](/price-earningsratio) oder des Verlusts von Marktanteilen.

## Beispiele für Underperformance-Bewertungen

Eine Branche könnte als leistungsschwach bezeichnet werden. Beispielsweise könnte die [Versorgungsbranche](/utilities_sector) diese Bezeichnung erhalten, weil das Wirtschaftswachstum die Branche ankurbeln kann, die Inflation jedoch zu höheren Zinssätzen führen könnte, was für den Versorgungssektor negativ wäre. In ähnlicher Weise könnte der Immobilienmarkt gesehen haben, wie niedrige Zinsen Investitionen in [Real Estate Investment Trusts vorangetrieben](/reit) haben, aber steigende Zinsen können diese Dynamik ändern. Diese Faktoren könnten dazu führen, dass eine Branche nicht das volle Renditepotenzial erwirtschaftet und ein Underperformance-Rating gerechtfertigt ist.

Einer bestimmten Aktie wird von einem Analysten ein Underperformance-Rating zugewiesen, wenn Bedenken bestehen, dass Aktien aus verschiedenen Gründen nicht mit anderen mithalten können, diese Bedenken jedoch kein absolutes Verkaufsrating rechtfertigen. Obwohl ein Unternehmen beispielsweise Wachstum oder positive Gewinne für ein Quartal oder für das Jahr verzeichnet, sind diese Renditen möglicherweise nicht auf dem Marktniveau. Wenn also ein Automobilhersteller für sein Geschäftsjahr eine Gesamtrendite von 12 % meldet, während der S&P 500 für dieses Jahr eine Gesamtrendite von 23 % sieht, könnte der Autohersteller als Underperformer eingestuft werden.

Je nach Maklerhaus kann ein Outlook-Rating von Underperform unterschiedliche Bedeutung haben. Bei Charles Schwab zum Beispiel beinhaltet ein Ausblick auf eine Underperformance auch eine [Verkaufsempfehlung](/guidance). Wenn ein Unternehmen von der Firma einen „stark unterdurchschnittlichen“ Ausblick erhält, erhält es auch die Verkaufsempfehlung. Diese Ratings können bedeuten, dass erwartet wird, dass die Aktien die Benchmarks nicht erfüllen werden.

## Höhepunkte

- Ein Analyst wird ein Underperform-Rating vergeben, wenn erwartet wird, dass eine Aktie nicht mit dem Markt Schritt halten wird, aber die Sorgen kein Verkaufsrating rechtfertigen.

- Eine unterdurchschnittliche Aktie hält nicht mit dem breiteren Markt Schritt.

- Das Underperform-Rating kann je nach Maklerunternehmen, das das Rating herausgibt, unterschiedliche Bedeutungen haben; es wird manchmal als schwaches Halten oder moderates Verkaufen bezeichnet.

- Es gibt eine Reihe von Gründen, warum eine Aktie ein Underperformance-Rating erhalten könnte, aber meistens ist dies das Ergebnis des Vergleichs von Unternehmenskennzahlen mit denen von Mitbewerbern oder dem Gesamtmarkt.
