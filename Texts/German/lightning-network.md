# Lightning-Netzwerk
Eine zweite Schicht, die auf einer Blockchain operiert und eine erhöhte Transaktionsgeschwindigkeit zwischen den teilnehmenden Knoten ermöglicht. Dies ist eine vorgeschlagene Skalierungslösung.

## Höhepunkte

- Ähnlich wie Blockchain trennt das Blitznetzwerk zentrale Institutionen wie Banken, die heute für das Routing der meisten Transaktionen verantwortlich sind.

- Das Lightning-Netzwerk ist eine technologische Lösung, die das Problem der Transaktionsgeschwindigkeit in der Bitcoin-Blockchain durch die Einführung von Off-Ledger-Transaktionen lösen soll.

- Das Blitznetzwerk wurde erstmals 2015 in einem Artikel von Joseph Poon und Thaddeus Dryja offiziell ausgearbeitet.
