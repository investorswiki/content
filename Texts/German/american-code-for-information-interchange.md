# Amerikanischer Code für den Informationsaustausch (ASCII)
## Was ist der amerikanische Code für den Informationsaustausch (ASCII)?

Der American Code for Information Interchange, allgemein ASCII genannt, ist eine Computersprache für Text, die 1963 zum Standard für die Übertragung von Informationen zwischen Computern wurde.

Das ASCII-System übersetzt alphanumerische Zeichen und Symbole in einen computerlesbaren Code. Das ursprüngliche ASCII war nur auf Englisch. Es gibt jetzt zwei Arten von ASCII-Codes; der Standardcode, der ein 7-Bit-Codierungssystem verwendet, und ein erweiterter Code, der ein 8-Bit-System verwendet.

In der Internetkommunikation wurde ASCII nach und nach durch den Unicode-Standard ersetzt, der mit jeder Sprache verwendet werden kann und mit ASCII kompatibel ist. ASCII wird jedoch immer noch zum Eingeben von Text auf PCs und elektronischen Geräten verwendet.

## Den American Code for Information Interchange (ASCII) verstehen

Die früheste Iteration von ASCII findet sich in einem telegrafischen Code, der von Bell-Datendiensten verwendet wird, um Nachrichten an Fernschreiber zu liefern.

ASCII war das erste große Zeichencodierungssystem für die Datenverarbeitung und wurde 1963 als Standard für die Datenverarbeitung und 1969 für die Internetübertragung übernommen. Es wurde viele Jahre lang in den von Wall Street-Händlern und -Brokern verwendeten Computersystemen zur Auftragseingabe verwendet.

Als Standard für das Internet wurde er nach und nach durch den Unicode Worldwide Character Standard (Unicode) abgelöst. Da jedoch die ersten 128 Zeichen von Unicode mit denen von ASCII identisch sind, kann man sagen, dass die beiden Systeme koexistieren.

> Sie können mit jedem Windows-kompatiblen Computer ein ASCII-Symbol in ein Dokument einfügen, indem Sie die ALT-Taste gedrückt halten, während Sie die Zeichencodenummer eingeben. Wenn Sie beispielsweise die ALT-Taste gedrückt halten, während Sie 156 eingeben, erhalten Sie £, das britische Pfundzeichen.

### Was ist in ASCII

Bevor ASCII als Standard eingeführt wurde, verwendete jeder Computerhersteller seinen eigenen Code, und einige Unternehmen verwendeten mehr als einen für verschiedene Produkte. Computer konnten nicht miteinander kommunizieren.

Obwohl ASCII 1963 von den American National Standards als Standard übernommen wurde, wurde es selbst in den USA erst 1968 vollständig implementiert, als Präsident Lyndon B. Johnson seine Annahme durch die Bundesregierung anordnete, um sicherzustellen, dass die Computer- und Telekommunikationsindustrie von der Regierung verwaltet wird Computer könnten miteinander kommunizieren.

ASCII enthielt ursprünglich 128 alphabetische Zeichen, Zahlen und Symbole und basierte auf der englischen Sprache. Erweiterte Formulare wurden in den 1970er Jahren hinzugefügt, um andere Sprachen aufzunehmen.

Bald wurden verschiedene Versionen von ASCII verwendet. Es umfasste schließlich 255 Zeichen. Diese Version ist als Extended ASCII bekannt.

Heutzutage können Benutzer ASCII- oder Unicode-Symbole wie ein Copyright-Symbol in Dokumente eingeben, indem sie die Alt-Taste drücken und den Zeichencode auf dem Ziffernblock bei aktivierter NUM-Taste eingeben.

## Höhepunkte

- ASCII existiert weiterhin, wurde jedoch weitgehend durch Unicode ersetzt, mit dem jede Sprache codiert werden kann.

- ASCII enthielt ursprünglich nur 128 englischsprachige Buchstaben und Symbole, wurde aber später um zusätzliche Zeichen erweitert, darunter auch solche, die in anderen Sprachen verwendet werden.

- Der American Code for Information Interchange oder ASCII ist ein Zeichencodierungsformat, das für die Übertragung von Text zwischen Computern entwickelt wurde.

## FAQ

### Was ist ASCII vs. Unicode?

Unicode könnte als universelle Version von ASCII angesehen werden. ASCII ist schließlich der amerikanische Code für den Informationsaustausch, und seine erste Iteration umfasste das englischsprachige Alphabet und die in der Sprache verwendeten Symbole. Spätere Versionen enthielten Symbole, die in vielen anderen Sprachen verwendet wurden. Unicode hingegen beginnt mit den gleichen 128 Symbolen, die in ASCII verwendet werden, aber aktuelle Versionen enthalten 144.697 Zeichen.

### Was ist ASCII?

Der American Standard Code for Information Interchange oder ASCII ist ein Zeichenkodierungsformat für die elektronische Übertragung von Text. Jedes Zeichen wird durch eine eindeutige Nummer dargestellt. Die erste Version von ASCII enthielt nur 128 Zeichen, die die Buchstaben des Alphabets, Groß- und Kleinbuchstaben, sowie eine Reihe häufig verwendeter Symbole wie das Komma darstellten. Spätere Versionen erweiterten ASCII auf 255 Zeichen, einschließlich zusätzlicher Symbole wie z Britisches Pfund-Symbol (£) und das umgekehrte Fragezeichen, das im spanischen Text verwendet wird (¿).

### Was sind die ASCII-Zeichen?

In der erweiterten ASCII-Tabelle sind die Zeichencodes 0 bis 31 Steuercodes wie Textanfang und Rücktaste. Die Zeichencodes 32 bis 127 sind hauptsächlich Buchstaben des Alphabets (Groß- und Kleinbuchstaben) und Symbole wie das Sternchen und das Dollarzeichen. B. ein umgekehrtes Ausrufezeichen, das Yen-Zeichen und das georgische Komma.
