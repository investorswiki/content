# Gewichtetes Alpha
## Was ist gewichtetes Alpha?

Das gewichtete Alpha misst die Wertentwicklung eines Wertpapiers über einen bestimmten Zeitraum, normalerweise ein Jahr, wobei jedoch der jüngsten Aktivität im Vergleich zur früheren Wertentwicklung eine relativ größere Bedeutung beigemessen wird.

[Alpha](/alpha) (α) ist ein Begriff, der verwendet wird, um die Fähigkeit einer Anlagestrategie zu beschreiben, den Markt oder seinen „Rand“ zu schlagen. Alpha wird daher auch oft als „ [Excess Return](/excessreturn) “ oder „ [Anormal Rate of Return](/abnormalreturn) “ bezeichnet, was sich auf die Vorstellung bezieht, dass Märkte effizient sind und es daher keine Möglichkeit gibt, systematisch Renditen zu erwirtschaften, die den breiten Markt insgesamt übersteigen.

## Gewichtetes Alpha verstehen

Wie der Name schon sagt, ist das gewichtete Alpha ein [gewichtetes](/weighted) Maß dafür, wie stark ein Wertpapier, beispielsweise eine Aktie, über einen bestimmten Zeitraum, normalerweise ein Jahr, gestiegen oder gefallen ist. Im Allgemeinen wird neuere Aktivität stärker betont, indem späteren Leistungsmessungen höhere Gewichtungen zugewiesen werden als solchen, die früheren Messungen zugewiesen wurden. Dies trägt dazu bei, eine Renditezahl zu erhalten, die sich stärker auf den aktuellsten Zeitraum konzentriert, was sich bei der Analyse dieses Wertpapiers als relevanter erweisen sollte. Diese Metrik ist sehr beliebt bei [technischen Analysten](/technical-analyst) und denjenigen, die sich auf Analysen verlassen, um ihre Handelsentscheidungen zu verbessern.

Gewichtetes Alpha verwendet gewichtete mathematische Berechnungen, um zu einer [Alpha -](/alpha) Performance-Zahl zu gelangen. Alpha ist ein Maß für die risikoadjustierte Performance im Vergleich zu einer Benchmark. Im Bereich der [Vermögensverwaltung](/assetmanagement) wird Alpha oft als Indikator für die Fähigkeiten des Fondsmanagers angesehen. Diese Argumentation kann auch bei der Analyse einer Aktie gelten, was teilweise die Effektivität des Managementteams einer Firma widerspiegelt.

Beispielsweise hat eine Aktie, deren Rendite auf dem Niveau der Benchmark lag, bereinigt um das eingegangene Risiko, ein Alpha von null. Ein positives Alpha zeigt an, dass die Aktie eine Rendite erzielt hat, die über der Benchmark liegt, während ein negatives Alpha das Gegenteil anzeigt.

## Gewichtete Alpha-Berechnung

Gewichtete Berechnungen ergeben eine zugewiesene Gewichtung basierend auf verschiedenen Faktoren. Indizes verwenden eine Gewichtung, um Wertpapieren nach Preis oder Marktkapitalisierung ein höheres Gewicht zu verleihen. Bei einer gewichteten Alpha-Berechnung wird in der Regel den Renditen jüngerer Zeiträume über eine Zeitreihe hinweg ein höheres Gewicht beigemessen.

Rendite eines [Wertpapiers für ein Jahr.](/security) Wenn ein Wertpapier ein positives gewichtetes Alpha aufweist, kann ein Anleger im Allgemeinen davon ausgehen, dass sein Kurs im vergangenen Jahr gestiegen ist. Wenn der Kurs eines Wertpapiers hingegen ein negatives gewichtetes Alpha aufweist, können Anleger davon ausgehen, dass die einjährige Kursrendite niedriger ist.

Bei einer gewichteten Alpha-Berechnung können die Gewichtungen je nach Präferenzen oder Softwareprogrammen für technische Analysen variieren. Einige gewichtete Alpha-Berechnungen können Gewichtungen nach Quartilen zuweisen, während andere eine Standardmethodik abnehmender Gewichtung verwenden.

> Alpha wird oft in Verbindung mit [Beta (dem griechischen Buchstaben β) verwendet, das die](/beta) [Gesamtvolatilität oder](/volatility) das Gesamtrisiko des breiten Marktes misst , bekannt als [systematisches Marktrisiko](/systematicrisk).

## Gewichtete Alpha-Inferenzen

Gewichtetes Alpha wird von einer Vielzahl von Anlegern verwendet. Am häufigsten verwenden technische Analysten das gewichtete Alpha als Indikator zur Unterstützung von Kauf- und Verkaufssignalen. Technische Analysten verwenden diese Kennzahl, um Unternehmen zu identifizieren, die im vergangenen Jahr einen starken Trend gezeigt haben, und insbesondere, um ihre Aufmerksamkeit auf Unternehmen zu richten, deren Momentum zunimmt. Wenn das gewichtete Alpha positiv ist, kann es ein bullisches Kaufsignal unterstützen. Wenn das gewichtete Alpha negativ ist, kann es ein rückläufiges Verkaufssignal unterstützen.

Betrachten Sie als Beispiel eine Aktie, die im letzten Jahr mehrere Hochs und Tiefs durch sowohl bullische als auch bärische Trendmuster erlebt hat. Ein technischer Analyst, der einen [Bollinger -](/bollingerbands) [Band](/bollingerbands) -Kanal verwendet, kann sehen, dass sich der Preis seiner Unterstützungstrendlinie nähert. Wenn die Aktie ein positives gewichtetes Alpha hat, kann dies eine Bestätigung dafür sein, dass der Kurs der Aktie im letzten Jahr stark gestiegen ist, was einen weiteren [Aufwärtstrend unterstützt.](/bull)

In einem anderen Szenario kann ein Händler sehen, wie der Kurs einer Aktie sein Widerstandsband in einem Bollinger-Band-Kanal erreicht und zu überschreiten beginnt. Oft ist dies ein Signal für eine Umkehrung und würde ein Verkaufssignal anzeigen. Wenn dieses Wertpapier jedoch ein positives gewichtetes Alpha hat, ist es wahrscheinlicher, dass es sein Widerstandsniveau überschreitet und höher steigt. Daher könnte das gewichtete Alpha in diesem Szenario einen Kauf-Trade unterstützen.

## Höhepunkte

- Das gewichtete Alpha misst die Wertentwicklung eines Wertpapiers über einen bestimmten Zeitraum, normalerweise ein Jahr, wobei den jüngsten Aktivitäten mehr Bedeutung beigemessen wird.

- Gewichtetes Alpha kann Unternehmen identifizieren, die im vergangenen Jahr einen starken Trend gezeigt haben, und insbesondere Unternehmen, deren Momentum zunimmt.

- Ein positives gewichtetes Alpha zeigt, dass das Wertpapier eine Rendite erzielt hat, die über der Benchmark liegt; ein negatives Maß zeigt das Gegenteil an.
