# Eigenkapitalverpflichtungserklärung (ECN)
## Was ist eine Equity Commitment Note (ECN)?

Eine Equity Commitment Note ist eine Art [Pflichtwandelanleihe,](/mandatoryconvertible) die von einer Bank ausgegeben wird und bei Fälligkeit gegen Aktien eingetauscht werden kann. Die Schuldverschreibung ist durch den Verkauf von Aktien durch ihren Emittenten rückzahlbar.

Es handelt sich im Wesentlichen um eine Art von Unternehmensanleihen, die mit Einschränkungen für den Kreditnehmer einhergeht, die festlegen, wie das Geld zur Rückzahlung der Schulden aufgebracht wird.

## Verstehen der Equity Commitment Note (ECN)

Die Equity Commitment Note ist eine Form der Unternehmensschuld. Es wird durch den Verkauf oder die Ausgabe von Wertpapieren zu einem späteren Zeitpunkt durch die Bank oder das Kreditinstitut, das es ausgibt, zurückgezahlt.

Eine Equity Commitment Note unterscheidet sich von einer Equity Contract Note darin, dass der Anleger die Wertpapiere nicht kaufen muss, um die Note einzulösen. Die Equity Commitment Note kann zu einem späteren Zeitpunkt durch den Verkauf von Stamm- oder Vorzugsaktien zurückgezahlt werden.

In den Vereinigten Staaten legt die Federal Reserve Bank eine maximale Laufzeit von 12 Jahren fest und verlangt, dass das ausgebende Unternehmen alle vier Jahre ein Drittel des Eigenkapitals finanziert .

## Höhepunkte

- Die Note kann bei Fälligkeit gegen Aktien eingetauscht werden.

- Eine Equity Commitment Note ist eine Art Pflichtwandelanleihe, die von einer Bank oder einem anderen Kreditinstitut ausgegeben wird.

- Im Wesentlichen ist eine Equity Commitment Note eine Unternehmensanleihe.
