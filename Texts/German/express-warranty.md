# Express-Garantie
## Was ist eine ausdrückliche Garantie?

Eine ausdrückliche Garantie ist eine Vereinbarung eines [Verkäufers](/seller),. innerhalb eines bestimmten Zeitraums nach dem Kauf Reparaturen oder einen Ersatz für ein fehlerhaftes Produkt, eine Komponente oder eine Dienstleistung bereitzustellen. Käufer verlassen sich auf diese Versprechen oder Garantien und kaufen manchmal Artikel deswegen.

## So funktioniert eine ausdrückliche Garantie

Eine [Garantie](/warranty) ist eine Zusicherung, dass ein Artikel den Versprechungen des Verkäufers entspricht. Gemäß dem Magnuson-Moss Warranty Act, der 1975 vom Kongress verabschiedet wurde, unterliegt ein Unternehmen, das eine ausdrückliche schriftliche Garantie gewährt, den Bundesrichtlinien und muss das Gesetz einhalten. Das Gesetz bietet Verbraucherschutz für den Fall, dass ein Unternehmen seine schriftliche Garantie nicht einhält.

Eine ausdrückliche Garantie kann auf viele verschiedene Arten formuliert werden. Es kann etwa lauten: „Wir garantieren ein Jahr lang für alle Möbel gegen Konstruktionsfehler. Wenn wir auf einen Konstruktionsfehler aufmerksam werden, reparieren oder ersetzen wir ihn.“

Die meisten ausdrücklichen Garantien stammen vom [Hersteller](/manufacturing) oder sind im Vertrag des Verkäufers enthalten. Sie können auch durch eine einfache Erklärung auf einer Anzeige oder einem Schild in einem Geschäft erstellt werden.

## Besondere Überlegungen

Details zu einem Produkt oder einer Dienstleistung, die in einer Werbung beschrieben werden, können den Präzedenzfall für eine ausdrückliche Garantie darstellen. In der Werbung gemachte Angaben über Qualität, Funktionalität, Lebensdauer und Wirksamkeit eines Produktes können eine ausdrückliche Garantie darstellen.

Wenn das Produkt nicht den in der Werbung angegebenen Standards entspricht oder innerhalb eines festgelegten Zeitraums einen Ausfall erleidet, hat der Kunde möglicherweise Anspruch auf einen kostenlosen Reparaturservice oder, wenn möglich, auf einen vollständigen Ersatz.

Allerdings ist nicht jeder Anspruch des Verkäufers im Gewährleistungsrecht verankert. Übertriebene Aussagen, die manchmal in der Werbung erscheinen, stellen nicht unbedingt ausdrückliche Garantien dar.

Wenn beispielsweise ein Autohersteller behauptet, sein Auto sei „das beste der Welt“, und der Käufer nach mehreren Autofahrten dieser Aussage nicht zustimmt, hat er nicht unbedingt Anspruch auf eine Rückerstattung, sofern nicht ausdrücklich anders angegeben.

## Beispiele für ausdrückliche Garantien

### E-Commerce

[E-Commerce-](/ecommerce) Unternehmen schließen in der Regel ausdrückliche Garantien für die von ihnen verkauften Waren ein, teilweise aufgrund der Art und Weise, wie Online-Einkäufe durchgeführt werden. Der Kunde kann Waren, die er kaufen möchte, nicht anprobieren oder physisch untersuchen.

Wie das Produkt bei Erhalt funktioniert und aussieht, kann sich dramatisch von dem unterscheiden, was sich der Kunde beim Surfen im Internet vorgestellt hat. Die Einbeziehung einer ausdrücklichen Garantie gibt ihnen ein gewisses Gefühl der Gewissheit, dass Probleme mit dem Kauf in irgendeiner Weise behoben werden.

Wenn ein Verbraucher beispielsweise online ein Business-Sakko kauft, der Artikel jedoch die falsche Größe oder Farbe hat oder Knöpfe fehlen, kann eine ausdrückliche Garantie den Verbraucher zu einer Rückerstattung oder einem Ersatz berechtigen. In solchen Fällen trägt in der Regel der Online-Verkäufer die Rechnung für etwaige zusätzliche Versandkosten.

### Autoverkauf

Autohändler werben in der Regel mit ausdrücklichen Garantiebedingungen für Reparaturen an den von ihnen verkauften Fahrzeugen. Dies kann Bestimmungen zu Laufleistung und Besitzdauer umfassen, die den Umfang dieser Deckung einschränken. Nachdem das Fahrzeug eine bestimmte Zeit lang im Besitz war oder über die Kilometergrenze hinaus gefahren wurde, wäre die ausdrückliche Garantie nicht mehr anwendbar.

## Ausdrückliche Garantie vs. stillschweigende Garantie

Ausdrückliche Garantien sind spezifische Zusagen, die ein Verkäufer einem Käufer mündlich oder schriftlich macht. In Ermangelung mitgeteilter Garantien kann eine [stillschweigende Garantie](/implied-warranty) in Kraft treten.

Stillschweigende Garantien sind ungeschriebene Garantien, dass ein Produkt oder eine Dienstleistung wie erwartet funktionieren sollte. Wenn Sie beispielsweise einen Kopfhörer kaufen, würden Sie erwarten, dass er funktioniert, wenn Sie ihn zum ersten Mal verwenden – es sei denn, Ihnen wurde beim Kauf etwas anderes gesagt.

[Der Uniform Commercial Code](/uniform-commercial-code) (UCC) bezieht sich auf eine „stillschweigende Gewährleistung der Marktgängigkeit“ und besagt, dass jede Ware, die in einer Transaktion verkauft wird, für die üblichen Zwecke geeignet sein muss, für die sie typischerweise verwendet wird.

**Korrektur** – **8. März 2022:** In einer früheren Version dieses Artikels wurde fälschlicherweise angegeben, dass das Magnuson-Moss-Gewährleistungsgesetz von Unternehmen die Ausstellung von Garantien verpflichte. Das Gesetz schreibt keine Garantien vor, sondern legt Bundesregeln dafür fest, wann Garantien angeboten werden.

## Höhepunkte

- Details zu einem Produkt oder einer Dienstleistung, die in einer Werbung beschrieben werden, können den Präzedenzfall für eine ausdrückliche Garantie darstellen.

- Eine ausdrückliche Garantie ist eine Vereinbarung eines Verkäufers, innerhalb eines bestimmten Zeitraums Reparaturen oder einen Ersatz für ein fehlerhaftes Produkt, eine fehlerhafte Komponente oder einen fehlerhaften Service bereitzustellen.

- Gemäß dem Magnuson-Moss Warranty Act muss ein Unternehmen, das eine ausdrückliche schriftliche Garantie gewährt, die Bundesrichtlinien einhalten.

- In Ermangelung kommunizierter Garantien kann eine stillschweigende Garantie in Kraft treten.
