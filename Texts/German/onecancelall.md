# One-Cancels-All-Bestellung
## Was ist eine One-Cancels-All (OCA)-Bestellung?

Eine One-Cancel-All-Order (OCA) ist ein Bündel von mindestens drei Aktien- oder Optionslimit-Orders, die zusammen platziert werden. Wenn eine dieser Orders ausgeführt wird, werden die verbleibenden Orders storniert. Es handelt sich um eine fortgeschrittene Order, die von Broker-Plattformen erhältlich ist, die sich an erfahrene Trader richten. Die beiden allgemeinen Strategien zur Verwendung einer OCA-Order umfassen die Optimierung eines Einstiegspreises innerhalb einer Aktie oder die Optimierung der Auswahl einer Aktie aus mehreren Optionen. Diese Orderart wird nicht von allen Brokern angeboten.

## So funktioniert eine One-Cancel-All-Bestellung (OCA).

Die gleichzeitig von einem OCA gesendeten Orders können [Limit-Orders](/limitorder),. [Stop -](/stoporder) [Orders](/stoporder) oder [Stop-Limit-Orders sein](/stoporder). auf einen oder mehrere Aktien oder Aktienoptionskontrakte platziert. Wenn einer der im OCA enthaltenen Trades ausgelöst wird, wird dieser Auftrag ausgeführt und die anderen Aufträge werden sofort zur Stornierung angewiesen. Sobald eine Bestellung aufgegeben wurde, kann ein Maklersystem diese Bestellung nur dann stornieren, wenn sie nicht bereits ausgeführt wird. Dies bedeutet, dass eine OCA-Order ein sehr geringes Risiko birgt, dass mehr als eine Order tatsächlich innerhalb des Bruchteils einer Sekunde ausgeführt werden könnte, nachdem die erste Order ausgeführt wurde, aber bevor alle anderen Orders storniert werden können.

Je nachdem, wie das System des Brokers programmiert ist, können, wenn die ausgelöste Order nur teilweise ausgeführt werden kann, die anderen Orders offen bleiben, aber entsprechend dem Anteil des teilweise ausgeführten Trades angepasst werden. Storniert ein Investor eine der Orders, werden automatisch auch die restlichen Orders zurückgezogen.

Die Aufträge können alle auf die gleiche zugrunde liegende Aktie oder auf separate Ausgaben platziert werden. Optionskontrakte können auch Teil eines gebündelten OCA sein. [Time-in-Force-](/timeinforce) Richtungen wie „Gültig bis auf Widerruf“ (GTC) können auf OCA-Aufträge angewendet werden.

## Wann eine One-Cancel-All (OCA)-Bestellung verwendet werden sollte

Es gibt viele Gründe, warum eine OCA-Bestellung sinnvoll sein kann. Typischerweise verwenden Anleger die OCA-Order, um zwischen einigen Optionen in die optimalste Aktie zu investieren, den Preispunkt zu optimieren, zu dem sie in eine einzelne Aktie einsteigen, und ihre Gewinne zu sichern und sich vor Verlusten zu schützen.

### Optimieren Sie die Auswahl mehrerer Aktien

Angenommen, ein Investor möchte im kommenden Quartal in den Einzelhandel investieren. Angenommen, sie haben 15.000 US-Dollar an verfügbarem Kapital, mit dem sie arbeiten können. Sie recherchieren den Sektor und wählen drei Aktien aus, die sie in Betracht ziehen möchten, aber sie wollen sehen, welche in der nächsten Woche die besten Schnäppchen bieten wird.

Sie platzieren drei Limit-Orders, eine für jede Aktie, zu einem Preis, der mehr als 1 % unter den letzten Notierungen liegt. Sie geben die richtige Anzahl von Aktien für jede Aktie an, um die Anzahl der Aktien zu maximieren, die sie zu den in der OCA-Order angegebenen Preisen kaufen könnten. Welche Aktie die jeweilige Limit-Order zuerst erreicht, ist die Aktie, die der Anleger ausgewählt hat, und die anderen beiden Orders werden storniert.

### Einstiegspunkt in Einzelaktien optimieren

Eine andere Strategie besteht darin, mehrere Wege zu einer einzigen Investition in Betracht zu ziehen. Beispielsweise kann ein Händler, der eine bestimmte Aktie besitzen möchte, einen Standard-Limit-Kaufauftrag für die Aktie mit einem oder mehreren [Optionskontrakten bündeln](/optionscontract),. die ihm andere Rechte zum Kauf der Aktie zu einem günstigen Preis geben könnten.

>

> OCA-Orders werden manchmal als Entweder-Oder-Oder-Trades bezeichnet. Sie können auch mit [One-Cancels-Other (OCO)-Aufträgen](/oco) verwechselt werden , die nur zwei Aufträge beinhalten. Dies sind alles komplizierte Transaktionen, und einige Maklerfirmen oder Online-Handelsplattformen bieten sie ihren Kunden nicht an. Diejenigen, die in der Regel Sondergebühren erheben.

>

### Schützen Sie sich vor Verlusten und sichern Sie Gewinne ab

Ein dritter Grund für eine OCA-Order, die manchmal als „ [Bracked](/bracketedbuyorder) Order“ bezeichnet wird, soll den Gewinn im Falle eines eskalierenden Kurses einer Aktie sicherstellen und sich vor Abwärtsverlusten schützen. Bei dieser alternativen Order platziert ein Investor eine Marktkauforder für Aktien des Unternehmens X und klammert diese Order mit zwei Verkaufsorders ein.

Die erste ist eine limitierte Verkaufsorder zu einem Ausübungspreis, der höher ist als der ursprüngliche Marktkaufpreis; Die zweite ist eine Stop-Loss-Order, die mit einer akzeptablen Marge unter dem Markteinkaufspreis festgelegt wird. Die Struktur dieses OCA garantiert dem Anleger einen minimalen Verlust durch Abwärtsbewegungen und einen garantierten Gewinn, wenn die Preise auf ein akzeptables Niveau steigen. Auf diese Weise werden auch One-cancels-the-other-Orders verwendet.

## Höhepunkte

- Das Risiko dieses Auftragstyps besteht darin, dass mehr als ein Auftrag nach dem ersten ausgeführt werden könnte, bevor die Stornierungsanweisungen erfüllt werden.

- Sobald die erste dieser potenziellen Bestellungen ausgeführt ist, werden die verbleibenden Bestellungen sofort zur Stornierung angewiesen.

- Ein One-Cancels-All-Auftragstyp erstellt mehrere potenzielle Aufträge basierend auf festgelegten Bedingungen.

- Häufig verwendete Strategien zur Verwendung dieses Auftragstyps umfassen die Verbesserung der Preisausführung oder die Verbesserung der Aktienauswahl.

- Die meisten Broker bieten eine ähnliche Funktion namens „bedingte Aufträge“ an.
