# Instamin
## Was ist ein Instamin?

Ein Instamine tritt auf, wenn eine große Menge an Kryptowährungstoken auf einmal ins Leben gerufen wird. Dies steht im Gegensatz zur geplanten Veröffentlichung der meisten Kryptowährungen auf der Grundlage eines Konsensprotokolls wie [Proof-of-Work](/proof-work) (PoW).

Instamining macht es effektiv zu einfach, Coins für eine [Kryptowährung](/cryptocurrency) zu einem bestimmten Zeitpunkt zu produzieren, was oft zu einer unfairen oder ungleichmäßigen Verteilung von Coins führt, die sich auf Entwickler oder Gründer eines Projekts konzentrieren. Typischerweise bietet eine Kryptowährung, die eine Instamine-Phase durchläuft, frühzeitig einen großen Teil ihrer Coins an, wenn das Interesse der Anleger wahrscheinlich hoch ist.

Ein Instamine kann mit Premining verglichen [werden](/premining).

##Instamine verstehen

Das Instaminieren kann unbeabsichtigt oder absichtlich erfolgen. Einige Kryptowährungen haben unmittelbar nach dem Start eine Angebotsschwemme erlebt. Dies kann aufgrund unvollkommener Mining-Algorithmen passieren, die den mit der Generierung neuer Coins verbundenen Schwierigkeitsgrad nicht richtig anpassen.

Ein Instamin kann aufgrund der unbeabsichtigten Folgen neuer Funktionen auftreten. Neue Kryptowährungen versuchen fast immer, eine oder mehrere Besonderheiten zu bieten, die potenzielle Benutzer oder Investoren ansprechen. Diese Funktionen reichen von strukturellen Elementen bis hin zu effekthascherischen Angeboten, mit allem dazwischen. Manchmal machen sie es zu einfach, eine Kryptowährung zu schürfen.

Instamining kann auch aufgrund schändlicher Codierung seitens eines oder mehrerer Entwickler stattfinden. Einige Kryptowährungen haben sogar mit der Idee gespielt, eine Instamine-Periode in die Einführung der Währung einzubauen. Ein solches System könnte als Anreiz für frühe Investoren dienen.

## Instaminierung vs. Premining

Instamining wird gelegentlich synonym mit dem Begriff Premining verwendet, obwohl dies etwas unterschiedliche Konzepte sind.

Instaminierte Coins können aufgrund absichtlicher oder versehentlicher Elemente ihrer Programmierung kurz nach dem Start leicht abgebaut werden. Premined Coins hingegen wurden generiert, bevor der Launch selbst stattfand.

Die meisten Kryptowährungen werden bis zu einem gewissen kontrollierten Umfang vorgemint, typischerweise von Entwicklern, die zu Recht einen Teil des Münzangebots beim Start behalten wollen. Vorminierungen können jedoch auch unter anderen Bedingungen erfolgen. Beispielsweise könnte eine Börse eine neue Kryptowährung bitten, vorgeminte Coins im Austausch für einen Platz in ihrer Liste der angebotenen Währungspaare bereitzustellen.

Wie kann ein Investor feststellen, ob eine bestimmte Kryptowährung vorgemint oder instaminiert wurde? Leider kann es schwierig sein, die Situation zum Zeitpunkt des Starts einzuschätzen. Die Hektik, die um die mit Spannung erwarteten Markteinführungen herrscht, macht dies zu einer besonders günstigen Zeit für Personen, die Fehler in Mining-Protokollen ausnutzen möchten.

Typischerweise kommt das Instamining erst im Nachhinein zum Vorschein, wenn die Münze versucht, sich langfristig zu etablieren. Coins, die einen massiven Verkauf von Token erfahren und dann unmittelbar danach im Preis fallen, könnten Instamining-Aktivitäten ausgesetzt gewesen sein. In vielen anderen Fällen kann es für einen Investor unmöglich sein, festzustellen, ob Instamining ein Faktor für die allgemeine Robustheit einer neuen Kryptowährung war.

> Instamining ist nur eines von mehreren Risiken im Zusammenhang mit Kryptowährungsinvestitionen. Wer in Erwägung zieht, in Kryptowährungen zu investieren, sollte über eine hohe [Risikobereitschaft](/risktolerance) oder ein [diversifiziertes](/diversification) Portfolio verfügen.

>

## Kritik an Instamining

Es gibt mehrere potenzielle Probleme mit der Instaminierung. Erstens nennen Investoren und Analysten es als möglichen Bereich für betrügerische Aktivitäten oder unlautere Geschäftspraktiken. Angenommen, eine bestimmte Gruppe kann sich einen großen Teil der Token im Verhältnis zu der Menge an Zeit, Energie und anderen Ressourcen sichern, die sie für das Mining-Verfahren aufwenden. Dann kann diese Gruppe umkehren und diese Token für einen unfairen Gewinn verkaufen.

Angenommen, das Instamining erfolgt kurz nach einem [Initial Coin](/initial-coin-offering-ico) [Offering](/initial-coin-offering-ico) (ICO). In diesem Fall kann dieser große Inhaber bei hoher Nachfrage eine beträchtliche Menge an Token zu einem hohen Preis abgeben. Das könnte sich negativ auf den Zustand des Gesamtmarktes für die Kryptowährung auswirken.

In anderen Fällen können die Begünstigten eines Instamines große Mengen dieser Coins über einen längeren Zeitraum halten. Sie könnten warten, bis die Preise höher sind, um die Coins zu verkaufen. Nach einem so massiven Verkauf ist es nicht ungewöhnlich, dass der Preis der Münze steil fällt und möglicherweise sogar den Untergang dieser Kryptowährung herbeiführt.

## Beispiel aus der Praxis für ein Instamine

Der vielleicht berühmteste Fall von Instamining ereignete sich nach dem Start von Dash. Die Algorithmen, die für die Anpassung der Mining-Schwierigkeit für Dash verantwortlich sind, funktionierten nicht wie geplant. Infolgedessen wurden in den zwei Tagen nach dem Start der Kryptowährung etwa 2 Millionen Münzen ausgegeben. Zwei Millionen Münzen machten etwa 15 % des gesamten Dash-Angebots aus, das jemals ausgegeben wurde.

Da das Angebot an Dash die Anleger überforderte, wurden die meisten Coins über verschiedene Börsen verkauft, oft zu extrem niedrigen Preisen. In diesem Fall hat ein einzelner Verkäufer keinen großen Bestand an Dash-Münzen abgeladen, sodass der Gesamtschaden für den Kryptowährungsmarkt und das Ökosystem minimal war.

Während Dash relativ unbeschadet aus seinem unbeabsichtigten Instamining-Fiasko hervorging, kann man das von einigen seiner Konkurrenten nicht behaupten. In der Tat kann das Instamining beim Start jeder neuen Kryptowährung stattfinden.

## Höhepunkte

- Wenn eine Münze instaminiert ist, steigt ihr Vorrat dramatisch an. Dies kann getan werden, um frühe Entwickler oder Unterstützer vor einem Börsengang zu belohnen, aber es kann den Preis zum Einsturz bringen, wenn es später in der Existenz einer Blockchain auftritt.

- Ein Instamine tritt auf, wenn Kryptowährungs-„Coins“ als eine einzige große Charge erstellt werden, anstatt durch den regulären Prozess des „Mining“.

- Dash ist das berühmteste Beispiel für ein versehentliches Instamine, aber es erholte sich später von diesem frühen Fehler.
