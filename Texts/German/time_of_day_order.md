# Reihenfolge der Tageszeit
## Was ist eine Tageszeitbestellung?

Der Begriff Time-of-Day-Order bezieht sich auf eine Handelsart, die es einem [Anleger ermöglicht](/investor),. eine bestimmte Tageszeit für die Ausführung einer Order zu beantragen. Tageszeitaufträge können entweder für Kauf- oder Verkaufsanträge für Aktien oder andere Arten von Finanzinstrumenten ausgeführt werden. Das Timing ist sehr wichtig, wenn es um den Kauf und Verkauf von [Aktien](/stock) geht, einschließlich der Tageszeit, daher ist es sehr wichtig, dass Anleger verstehen, wann sie ihre Orders ausführen müssen. Tageszeitaufträge können Händlern helfen, die Vermögenswerte auf der Grundlage von Informationen kaufen oder verkaufen möchten, von denen sie glauben, dass sie die Preise zu bestimmten Zeiten beeinflussen werden.

## So funktionieren tageszeitliche Bestellungen

Eine Time-of-Day-Order ist eine Art Ordertyp oder eine Aufforderung zum Kauf oder Verkauf einer Aktie, einer [Anleihe](/bond) oder eines anderen Finanzinstruments. Diese Art von Order ist eine Order, die mit spezifischen Anweisungen hinsichtlich des Zeitraums während des Handelstages versehen ist, in dem der Anleger die Ausführung wünscht. Dies ist eine besonders nützliche Strategie für [Daytrader](/daytrader) und andere Anleger, die ihren Ein- oder Ausstieg in eine Aktie oder ein anderes Finanzinstrument zeitlich festlegen möchten.

Tageszeitaufträge sind Strategien, die häufig Anleger ansprechen, die mit bestimmten Informationen handeln, die zu einem bestimmten Zeitpunkt offengelegt werden können. Nehmen wir zum Beispiel an, Sie erwarten, dass eine Regierungsankündigung am Horizont steht, die besagt, dass die Wirtschaft im vorangegangenen Quartal geschrumpft ist. Sie können einen Tagesauftrag zum Kauf von [Schatzwechseln](/treasurybill) (T-Bills) erteilen, wenn Sie glauben, dass Anleger auf die schlechten Nachrichten reagieren und in die Sicherheit von Staatsanleihen fliehen werden.

Die Arten von Aufträgen, die Börsen den Anlegern anbieten, sind wichtig, aber durchschnittliche Anleger verstehen nicht immer die Auswirkungen bestimmter Marktstrukturen. Zum Beispiel hat das Aufkommen des [Hochfrequenzhandels](/high-frequency-trading) die Aufmerksamkeit auf verschiedene Auftragsarten (z. B. Hide not Slide) gelenkt, die [algorithmischen Händlern zur Verfügung gestellt wurden](/algorithmictrading).

Ob ein Trader eine Time-of-Day-Order erteilen kann oder nicht, hängt von den Regeln der jeweiligen Börse oder des Marktes ab, an dem er handelt. Daher können Händler möglicherweise andere Arten von Ordertypen erstellen, z. B. Limit-Orders, [Stop-Orders](/stoporder) und [Stop-Loss-Orders](/stop-lossorder).

## Besondere Überlegungen

Time-of-Day-Orders können oft mit anderen Ordertypen gepaart werden. Beispielsweise kann es sinnvoll sein, die Time-of-Day-Order mit einer [Immediate-or-Cancel](/immediateorcancel) (IOC)-Order zu koppeln. Eine Immediate-or-Cancel-Order fordert, dass nur der Teil der Order ausgeführt wird, der ausgeführt werden kann, und der Rest storniert wird. Die IOC-Order wird üblicherweise für große Orders ausgeführt. Dies kann Sie schützen, wenn Sie glauben, dass ein Teil Ihrer Bestellung später zu einem höheren Preis ausgeführt wird.

## Beispiel für tageszeitliche Bestellungen

Hier ist ein hypothetisches Beispiel, um zu zeigen, wie Tageszeitaufträge funktionieren. Nehmen wir an, Sie möchten Apple (AAPL)-Aktien kaufen, weil Sie glauben, dass das kürzlich veröffentlichte iPhone beliebter ist, als das Unternehmen oder die Analysten vorhersagen. Sie glauben, dass die Aktie deutlich im Kurs steigen wird, wenn Apple am nächsten Tag seine iPhone-Verkaufsdaten veröffentlicht. Da die Ankündigung von Apple für morgen um 14:00 Uhr geplant ist, erteilen Sie eine Tageszeitbestellung für 13:59 Uhr

Wie oben erwähnt, kann es in diesem Fall sinnvoll sein, eine IOC-Bestellung aufzugeben. Dies gilt insbesondere, wenn Sie erwarten, dass die Apple-Aktien nach 14:00 Uhr steigen werden. Dies würde Sie daran hindern, einige Ihrer Bestellungen zu einem höheren Preis auszuführen, als Sie zuvor ausgeführt haben.

## Höhepunkte

- Ein Time-of-Day-Auftrag ist eine Handelsart, die es einem Anleger ermöglicht, eine bestimmte Tageszeit für die Ausführung eines Auftrags anzufordern.

- Diese Arten von Orders können entweder für Kauf- oder Verkaufsanfragen für Aktien oder andere Finanzinstrumente ausgeführt werden.

- Ob ein Anleger eine Tageszeitorder erteilen kann, hängt von den Regeln der jeweiligen Börse oder des Marktes ab, an dem er handelt.

- Time-of-Day-Orders können oft mit anderen Ordertypen gekoppelt werden, wie z. B. einer Immediate-or-Cancel-Order.

- Tageszeitaufträge können Anlegern helfen, die mit den spezifischen Informationen handeln möchten, von denen sie glauben, dass sie zu einem bestimmten Zeitpunkt bekannt werden.
