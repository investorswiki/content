# Frontgebühr
## Was ist eine Frontgebühr?

Die Ausgabegebühr ist die [Optionsprämie](/option-premium),. die ein Anleger beim erstmaligen Kauf einer [zusammengesetzten Option zahlt](/compoundoption). Eine zusammengesetzte Option ist eine Option, bei der der zugrunde liegende Vermögenswert ebenfalls eine Option ist; es ist eine Option über eine Option. Der Ausgabeaufschlag gibt dem Anleger das Recht, aber nicht die Pflicht, die ursprüngliche Option auszuüben, um die zugrunde liegende Option zu erhalten. Bei Ausübung ist für die Prämie der zugrunde liegenden Option eine weitere Gebühr zu entrichten, die als „ [Rückvergütung “ bezeichnet wird.](/backfee) Beachten Sie, dass die zweite Option ausgeübt werden kann, um eine endgültige Transaktion mit dem zugrunde liegenden Vermögenswert selbst durchzuführen.

Zum Beispiel könnten Sie eine zusammengesetzte Option kaufen, um einen Call auf [Euro zu kaufen](/euro). Wenn Sie die Compound-Option ausüben, haben Sie dann das Recht, Euro zu einem festgelegten Wechselkurs zu oder vor Ablauf eines bestimmten Zeitpunkts zu kaufen. Wenn Sie diese zweite Option ausüben, erhalten Sie die Euros.

## Vordergebühren verstehen

Compound-Optionen werden in Situationen eingesetzt, in denen Ungewissheit hinsichtlich der Notwendigkeit einer Risikominderung besteht. Beispielsweise kann ein Unternehmen ein Angebot für ein Projekt im Ausland abgeben. Im Erfolgsfall würde das Projekt erhebliche Einnahmen in einer Fremdwährung generieren, die möglicherweise gegen [Wechselkursrisiken](/foreignexchangerisk) abgesichert werden müssten . Eine zusammengesetzte Option wäre in diesem Fall sinnvoll, da der zu zahlende Ausgabeaufschlag niedriger wäre als die bei einem Devisenoptionskontrakt zu zahlende Prämie. Sie kaufen eine Option, um eine Fremdwährungsoption zu kaufen, und müssen dann nur die ursprüngliche Option verwenden (um die zugrunde liegende Option zu erhalten), wenn dies erforderlich und sinnvoll ist.

Verbundoptionen gibt es in vier Konfigurationen:

- [Call auf einen Put](/call-on-a-put),. CoP (CaPut) genannt, ist das Recht, eine bestimmte Put-Option zu kaufen.

- [Call on a Call](/call-on-a-call),. CoC (CaCall) genannt, ist das Recht, eine bestimmte Call-Option zu kaufen.

- [Put auf einen Put](/put-on-a-put),. PoP genannt, ist das Recht, eine bestimmte Put-Option zu verkaufen.

- [Put on a Call](/put-on-a-call),. PoC genannt, ist das Recht, eine bestimmte Call-Option zu verkaufen.

Die mit dem Kauf einer dieser vier Vereinbarungen verbundene Optionsprämie wäre die Ausgabegebühr und wird an den Verkäufer der zusammengesetzten Option gezahlt. Wenn beispielsweise ein Call auf einen Put [ausgeübt](/exercise) wird, kauft der Optionsinhaber einen Put und zahlt die Prämie für diese Option als Rückzahlungsgebühr.

## Beispiel für eine Ausgabegebühr bei einem Aktienhandel mit Compound-Optionen

Ein Beispiel für eine Frontgebühr wäre die Zahlung von 6 USD für zwei Call-on-a-Call-Optionskontrakte bei Apple Inc. (AAPL). Die Verträge berechtigen zum Kauf von AAPL 250 Strike Calls mit einer Laufzeit von drei Monaten. Die Aktie wird derzeit bei 225 $ gehandelt. Für die beiden Kontrakte betragen die Kosten 1.200 $ (6 $ x 200 Aktien, da jeder Kontrakt 100 Aktien umfasst).

Dies ist eine billigere Strategie, wenn die Wahrscheinlichkeit gering ist, dass die Aktie innerhalb der nächsten drei Monate über 250 $ steigt. Die Prämie, die für die dreimonatigen 250 Aufrufe direkt gezahlt werden müsste, würde 27 USD pro Kontrakt oder 5.400 USD (27 USD x 200 Aktien) betragen.

Typischerweise werden zusammengesetzte Optionen nicht in Optionen auf Aktien oder Aktienindizes verwendet. Sie werden hauptsächlich an Devisen- oder [Rentenmärkten eingesetzt](/fixedincome),. wo Unternehmen Unsicherheit oder Ungewissheit hinsichtlich der Notwendigkeit des Risikoschutzes einer Option haben.

Eine weitere übliche Geschäftsanwendung, für die Compound-Optionen verwendet werden, ist die [Absicherung von](/hedge) Angeboten für Geschäftsprojekte, die akzeptiert oder nicht akzeptiert werden können.

## Höhepunkte

- Die Option-auf-eine-Option bietet niedrigere Vorabkosten und mehr Flexibilität als der direkte Kauf von Schutz (der möglicherweise nicht benötigt wird).

- Eine zusammengesetzte Option ist effektiv eine Option zum Kauf oder Verkauf einer anderen Option, wie z. B. ein Call auf einen Put oder ein Put auf einen Put.

- Die Prämie auf den Kontrakt der zugrunde liegenden Option oder Rückvergütung wird nur gezahlt, wenn die zusammengesetzte Option ausgeübt wird.

- Die Ausgabegebühr bezieht sich auf die Prämienausgabe für den anfänglichen Kauf einer Compound-Option.
