# Kürzung
## Was ist Kürzung?

Kürzung ist der Akt, etwas einzuschränken, zu reduzieren oder zu kürzen. Das Wort wird oft in Geschäftsankündigungen verwendet und hat mehrere Verwendungen in der Hypothekenbranche:

- Ein Hypothekendarlehen kann durch Kürzung getilgt werden, wenn der Eigenheimbesitzer den Restbetrag vorzeitig zurückzahlt.

- Kapitalkürzung einer Hypothek tritt auf, wenn ein Kreditnehmer eine zusätzliche Zahlung auf den geschuldeten Kapitalbetrag leistet, um den ausstehenden Restbetrag zu reduzieren. Dies wird manchmal als teilweise Kürzung bezeichnet.

- Eine vollständige Hypothekenkürzung liegt vor, wenn der Restbetrag des Darlehens vorzeitig mit einer Summe getilgt wird.

## Kürzungen verstehen

Das Wort **Einschränkung** kann auf eine absichtliche Einschränkung oder Verringerung einer beliebigen Aktivität angewendet werden und ist in Bezug auf Geschäftsvorgänge allgemein gebräuchlich geworden.

Als Reaktion auf kurzfristige Marktbedingungen kann ein Unternehmen die Produktion drosseln. Dies ist besonders häufig in der Öl- und Gasindustrie der Fall, die schnell auf Änderungen bei [Angebot und Nachfrage reagieren muss](/law-of-supply-demand).

Ein Unternehmen kann sich als Reaktion auf finanzielle Probleme für eine Einschränkung einiger seiner Geschäftsaktivitäten entscheiden. Dies kann die vorübergehende oder dauerhafte Schließung einiger Betriebsteile beinhalten, um sich auf das Kerngeschäft zu konzentrieren.

> Eine Kürzung weist auf eine vorübergehende oder dauerhafte Reduzierung eines Produktionsniveaus hin.

>

Das Wort Kürzung wird auch häufig von Energieunternehmen verwendet, um eine Reduzierung der Lieferungen an Kunden aufgrund einer vorübergehenden Versorgungsknappheit zu beschreiben.

Eine Reihe neuerer Meldungen beziehen sich auf angekündigte Kürzungen, darunter die folgenden:

- Das kalifornische State Water Resources Control Board erließ im August 2021 eine „Notfalleinschränkungsanordnung“, die die Nutzung von Wasser aus den meisten Flüssen des Bundesstaates aufgrund von Dürrebedingungen auf unbestimmte Zeit untersagte.

- Canfor, ein Holzunternehmen, kündigte eine geplante Einschränkung der Produktionskapazität in seinen kanadischen Sägewerken im dritten Quartal 2021 aufgrund von Herausforderungen in der Lieferkette und extremen Waldbrandbedingungen an.

– Die indische Regierung kündigte Verzögerungen bei einer zoologischen Studie des Subkontinents aufgrund einer „Mittelkürzung“ für das Projekt während der COVID-19-Pandemie an.

Trotz seiner häufigen Verwendung zur Beschreibung von Geschäftskürzungen und Leistungsbeschränkungen stoßen [Verbraucher](/customer) höchstwahrscheinlich auf das Wort Kürzung im Kleingedruckten in Hypothekenpapieren.

Zumindest in diesem Zusammenhang ist eine Kürzung für den Verbraucher immer ein gutes Ergebnis.

## Höhepunkte

- Eine Einschränkung kann sich auf eine vorübergehende oder dauerhafte Einschränkung einer Tätigkeit beziehen.

- Hypothekenkürzung bedeutet Vorauszahlung eines Teils des Kapitalbetrags oder Rückzahlung des gesamten Darlehens früher als geplant.

- Das Wort wird häufig in Geschäftsankündigungen verwendet, insbesondere um Produktionskürzungen oder Begrenzungen der Energieabgabe zu melden.
