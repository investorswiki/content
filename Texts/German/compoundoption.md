# Compound-Option
## Was ist eine zusammengesetzte Option?

Eine zusammengesetzte Option ist eine Option, deren Basiswert eine weitere Option ist. Daher gibt es zwei [Ausübungspreise](/strikeprice) und zwei Ausübungstage.

Sie sind für jede Kombination von Calls und Puts verfügbar. Beispielsweise ein Put, bei dem der Basiswert eine [Kaufoption ist,](/calloption) oder ein Call, bei dem der Basiswert eine [Verkaufsoption ist](/putoption).

Jedes Paar hat eine Abkürzung:

- [Call bei einem Put](/call-on-a-put) - CoP (CaPut)

- [Anruf bei einem Anruf](/call-on-a-call) - CoC (CaCall)

- [Setzen Sie auf einen Put](/put-on-a-put) - PoP

- [Anruf](/put-on-a-call) tätigen - PoC

Compound-Optionen können als Split-Fee-Optionen bezeichnet werden.

## Compound-Optionen verstehen

Wenn der Inhaber eine zusammengesetzte Kaufoption [ausübt](/exercise),. die als Overlying-Option bezeichnet wird, muss er dem Verkäufer der zugrunde liegenden Option eine Prämie zahlen, die auf dem Ausübungspreis der zusammengesetzten Option basiert. Diese Prämie wird als [Rückvergütung bezeichnet](/backfee).

Angenommen, ein Investor möchte einen Put kaufen, um 100 Aktien zu 50 $ zu verkaufen. Die Aktie wird derzeit bei 55 $ gehandelt. Der Anleger könnte einen CaPut kaufen, der es ihm ermöglicht, jetzt einen Call zu kaufen, für beispielsweise 1 USD pro Aktie (100 USD), der es ihm ermöglicht, in Zukunft einen Put mit einem Ausübungspreis von 50 USD zu kaufen. Sie zahlen jetzt 1 US-Dollar pro Aktie, müssen aber nur dann die Gebühr für die zweite Option zahlen, wenn sie die erste Option ausüben, was dazu führt, dass sie die zweite Option erhalten.

Die Compound-Option gibt dem Anleger jetzt ein gewisses Engagement in der Put-Option, aber ohne die Kosten für die Zahlung einer langfristigen Put-Option jetzt. Wenn sie jedoch die anfängliche Call-Option ausüben und den Put erhalten, werden die gezahlten Prämien wahrscheinlich teurer sein, als wenn sie überhaupt erst einen Put gekauft hätten.

Im Falle eines PoP oder PoC bietet die Compound-Option das Recht, einen Put oder Call als Basiswert zu verkaufen.

## Variationen von Compound-Optionen

- **Call auf Put**: Dies ist eine Call-Option auf eine zugrunde liegende Put-Option. Der Eigentümer, der die Call-Option ausübt, erhält eine Put-Option. Ein Call auf eine Call-Option kann von einem Investor verwendet werden, um sein Engagement in einem zugrunde liegenden Vermögenswert zu geringen Kosten zu erweitern, und er kann in der Immobilienentwicklung verwendet werden, um Eigentumsrechte zu sichern, ohne sich verpflichten zu müssen.

- **Call on a Call**: Bei dieser Option kauft der Anleger eine weitere Call-Option mit individuellen Konditionen. Diese Bestimmungen beinhalten das Recht, eine Plain-Vanilla-Call-Option auf ein zugrunde liegendes Wertpapier zu kaufen. Unternehmen können während eines Bieterverfahrens für einen potenziellen Arbeitsvertrag einen Call auf einen Put verwenden.

- **Put on a Call**: Der Anleger muss die zugrunde liegende Call-Option an den Verkäufer liefern und eine Prämie basierend auf dem Ausübungspreis der darüber liegenden Put-Option kassieren. Ein Put auf eine Call-Option kann von einem Investor verwendet werden, um seine Absicherung eines zugrunde liegenden Vermögenswerts zu geringen Kosten zu erweitern, und es kann in der Immobilienentwicklung verwendet werden, um aus Eigentumsrechten auszusteigen, ohne an das Geschäft gebunden zu sein.

- **Put auf einen Put**: Ein Put wird auf einen Put-Kontrakt gekauft und gewinnt an Wert, wenn der Put-Kontrakt an Wert verliert, dh wenn das zugrunde liegende Wertpapier, auf dem die zweite Option basiert, steigt. Ein Put auf eine Put-Option wird verwendet, wenn ein bullischer Trader eine Hebelwirkung einsetzen möchte.

> Compound-Optionen sind auf den europäischen Derivatemärkten häufiger als auf den amerikanischen Derivatemärkten.

>

## Besondere Überlegungen

Es ist üblicher, Compound [-Optionen auf Devisen- oder Rentenmärkten zu sehen](/fixedincome),. wo Ungewissheit bezüglich der Risikoschutzfähigkeiten der Option besteht. Die Vorteile von Compound-Optionen bestehen darin, dass sie eine große [Hebelwirkung ermöglichen](/leverage) und anfänglich billiger sind als reine Optionen. Wenn jedoch beide Optionen ausgeübt werden, ist die Gesamtprämie höher als die Prämie einer einzelnen Option [.](/premium)

Auf dem [Hypothekenmarkt](/mortgage) sind CaPut-Optionen sinnvoll, um das [Zinsänderungsrisiko](/nominalinterestrate) zwischen dem Zeitpunkt der Hypothekenzusage und dem geplanten Liefertermin auszugleichen.

Trader können zusammengesetzte Optionen verwenden, um die Laufzeit einer Optionsposition zu verlängern, da es beispielsweise möglich ist, einen Call mit einer kürzeren Laufzeit für einen anderen Call mit einer längeren Laufzeit zu [kaufen](/expirationdate). Das heißt, sie können an der Wertentwicklung des Basiswerts partizipieren, ohne gleich zu Beginn den vollen Betrag für den Kauf aufzubringen. Der Vorbehalt ist, dass zwei Prämien gezahlt werden und höhere Kosten anfallen, wenn die zweite Option ausgeübt wird.

Während [Spekulationen](/speculation) an den Finanzmärkten immer einen großen Teil der Aktivität von Compound-Optionen ausmachen, könnten sie für Geschäftsunternehmen nützlich sein, wenn sie ein großes Projekt planen oder Gebote abgeben. In einigen Fällen müssen sie die Finanzierung oder Lieferungen sicherstellen, bevor sie das Projekt tatsächlich starten oder gewinnen können. Wenn sie das Projekt nicht bauen oder gewinnen, könnten sie mit der Finanzierung zurückbleiben, die sie nicht benötigen. Compound-Optionen stellen in diesem Fall eine Art [Versicherungspolice dar](/insurance-coverage).

## Beispiel für die Verwendung einer zusammengesetzten Option

Beispielsweise bietet ein Unternehmen an, ein großes Projekt abzuschließen. Wenn sie die Ausschreibung gewinnen, benötigen sie eine Finanzierung in Höhe von 200 Millionen US-Dollar für zwei Jahre. Die Formel, die sie bei der Berechnung verwenden, berücksichtigt jedoch die aktuellen Zinssätze. Daher ist das Unternehmen zwischen dem Vertragsangebot und dem möglichen Gewinn möglicherweise höheren Zinssätzen ausgesetzt. Sie könnten eine zweijährige [Zinsobergrenze ab](/interestratefloor) dem Datum der Auftragsvergabe kaufen, aber dies könnte sehr teuer werden, wenn sie den Auftrag nicht gewinnen.

Stattdessen könnte das Unternehmen eine Kaufoption auf eine zweijährige Zinsobergrenze kaufen. Wenn sie den Zuschlag erhalten, dann üben sie die Option auf den Zinsdeckel zum vorher festgelegten Aufschlag aus, weil sie ihn für das Projekt benötigen. Und wenn sie den Kontrakt nicht gewinnen, können sie die Option verfallen lassen, weil sie den Basiswert nicht mehr benötigen. Der Vorteil ist ein geringerer Anfangsaufwand [und ein](/outlaycost) geringeres Risiko.

## Höhepunkte

- Der Basiswert wird als zweite Option bezeichnet, während die anfängliche Option als Überlagerung bezeichnet wird.

- Zusammengesetzte Optionen können zwei Ausübungspreise und zwei Ablaufdaten umfassen.

- Eine zusammengesetzte Option ist eine Option, eine andere Option als Basiswert zu erhalten.

- Bei Ausübung der Compound-Option fallen zwei Prämien an.
