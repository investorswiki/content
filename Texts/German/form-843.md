# Formular 843
## Was ist Formular 843: Erstattungsantrag und Minderungsantrag?

Das Formular 843 ist ein Mehrzweck-Steuerdokument, das vom [Internal Revenue Service](/irs) (IRS) ausgestellt wird und von Steuerzahlern verwendet wird, um einen Anspruch auf Erstattung bestimmter veranlagter Steuern geltend zu machen oder die Herabsetzung von Zinsen oder irrtümlich angewendeten Strafen zu beantragen.

## Wofür wird Formular 843: Antrag auf Rückerstattung und Antrag auf Minderung verwendet?

Das Formular 843 kann verwendet werden, um beim IRS eine Ermäßigung bestimmter Steuern außer Einkommen zu beantragen, darunter:

- [Erbschafts-](/estatetax) oder [Schenkungssteuer](/gifttax)

- Zinsen oder Strafen aufgrund von IRS-Fehlern oder -Verzögerungen oder falscher schriftlicher Auskunft des IRS

- Um eine Rückerstattung von Sozialversicherungs- oder [Medicare](/medicare) -Steuern zu beantragen, die irrtümlicherweise einbehalten wurden und für die der Arbeitgeber die Übererhebung nicht berichtigen wird

Formular 843 kann nicht verwendet werden, um eine zuvor eingereichte Einkommens- oder Lohnsteuererklärung zu ändern in:

- Fordern Sie eine Rückerstattung der Vereinbarung, Gebühren für Kompromissangebote oder Pfandgebühren an

- Fordern Sie eine Ermäßigung von Schenkungs- oder Nachlasssteuern an

- Fordern Sie eine Rückerstattung oder Ermäßigung der Steuer nach dem [Federal Insurance Contributions Act](/fica) (FICA) an

- Fordern Sie eine Rückerstattung oder Ermäßigung der Railroad Retirement Tax Act-Steuer an

- Fordern Sie eine Rückerstattung oder Ermäßigung des Einkommensteuerabzugs [an](/withholdingtax)

## Wer kann Formular 843 einreichen: Erstattungsanspruch und Minderungsantrag?

Es gibt mehrere Gründe, warum ein Steuerzahler das Formular 843 einreichen kann. Dazu gehört, wenn der Arbeitgeber eines Steuerzahlers zu viel Einkommens-, Sozialversicherungs- oder Medicare-Steuern von einem Gehaltsscheck einbehält und keine Anpassungen vornimmt. Dieses Formular kann auch von einem bevollmächtigten Vertreter des Steuerzahlers eingereicht werden.

Ein weiterer Grund für die Einreichung dieses Formulars kann ein IRS-Fehler oder eine Verzögerung sein. Dies geschieht, wenn einem Steuerpflichtigen Zinsen, Bußgelder oder Steuerzuschläge, die nicht geschuldet sind, falsch eingeschätzt werden. In diesen Fällen kann ein Steuerzahler verlangen, dass der IRS den/die Fehler behebt, indem er einen Antrag auf Erstattung oder Minderung stellt.

Grundsätzlich ist für jede Steuer- bzw. Abgabenart und für jedes Steuerjahr ein separates Formular 843 einzureichen.

> Wenn der IRS der Meinung ist, dass ein Steuerzahler einen überhöhten Erstattungsbetrag fordert, kann er eine Strafe in Höhe von 20 % des als überhöht ermittelten Betrags verhängen.

>

## So reichen Sie das Formular 843 ein: Erstattungsanspruch und Minderungsantrag

Das Formular 843 erfordert grundlegende Informationen wie Name, Adresse, Sozialversicherungsnummer, Steuerzeitraum, Steuertyp und Rückgabetyp. Es erfordert eine Darlegung der Tatsachen und Probleme, warum Sie Anspruch auf eine Rückerstattung oder Minderung haben. Bei Anträgen auf Strafminderung müssen Sie die Abschnittsnummer des Internal Revenue Code (IRC) der Strafe in Zeile 4 schreiben. Sie finden diese Abschnittsnummer auf der IRS-Mitteilung, die Sie erhalten haben. Dann müssen Sie in Abschnitt 5a einen Grund für Ihre Anfrage auswählen. Zur Auswahl stehen:

- Zinsen wurden aufgrund von IRS-Fehlern oder -Verzögerungen festgesetzt

- Fehlerhafte schriftliche Informationen vom IRS

- Angemessener Grund oder ein anderer Grund als eine fehlerhafte schriftliche Beratung

Sie können in Abschnitt 7 eine Erläuterung Ihrer Anfrage schreiben. Denken Sie daran, Ihre Gründe mit Beweisen und Berechnungen zu untermauern. Das IRS ermöglicht es Ihnen, zusätzliche Seiten anzuhängen, wenn Sie mehr Platz benötigen.

Das Formular muss innerhalb von zwei Jahren ab dem Datum, an dem Sie die Steuern bezahlt haben, oder drei Jahren ab dem Datum der Einreichung der Steuererklärung eingereicht werden, je nachdem, welcher Zeitpunkt später liegt.

## Besondere Überlegungen für Formular 843

Wenn der IRS Ihre Forderung ablehnt, indem er eine gesetzlich vorgeschriebene Mitteilung über die Ablehnung der Forderung sendet – oder wenn sechs Monate ohne Maßnahmen vergehen –, können Sie beim US-Bezirksgericht oder beim US-Bundesgerichtshof einen Antrag stellen . Sie können auch vor Ablauf der [Verjährungsfrist eine Schutzklage erheben](/statute-of-limitations),. um Ihren Anspruch auf Erstattung zu wahren. Schutzansprüche und dingliche Ansprüche haben die gleiche Rechtswirkung.

## Höhepunkte

- Das Formular muss innerhalb von zwei Jahren ab dem Datum der Steuerzahlung oder drei Jahren ab dem Datum der Einreichung der Steuererklärung eingereicht werden, je nachdem, welcher Zeitpunkt später liegt.

- Das Formular 843 wird verwendet, um eine Rückerstattung bestimmter veranlagter Steuern zu beantragen oder um eine Ermäßigung von Zinsen oder Strafen zu beantragen, die irrtümlicherweise vom IRS verhängt wurden.

- Für jede Steuer- oder Abgabenart und für jedes Steuerjahr ist ein separates Formular einzureichen.
