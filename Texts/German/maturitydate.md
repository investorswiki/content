# Fälligkeitstag
## Was ist ein Fälligkeitsdatum?

Das Fälligkeitsdatum bezieht sich auf das Datum, an dem eine Anlage, z. B. ein Einlagenzertifikat (CD) oder eine Anleihe, fällig wird und an den Anleger zurückgezahlt wird. An diesem Punkt zahlt die Investition keine Zinsen mehr und Anleger können die angesammelten Zinsen und ihr Kapital ohne Strafe zurückzahlen.

## Tiefere Definition

Ein Einlagenzertifikat (CD) ist ein Schuldinstrument, das von Banken verwendet wird, um Geld zu beschaffen. Die wichtigsten Merkmale einer CD sind:

- **Fälligkeitsdatum** — Eine CD hat ein festes Fälligkeitsdatum, das von einem Monat bis zu fünf Jahren variieren kann.

- **Zinssatz** — Zinsen fallen zu einem am Kaufdatum festgelegten jährlichen Zinssatz an.

- **Zinszahlungen** — Je nach CD können die Zinsen angesammelt und am Fälligkeitsdatum ausgezahlt werden, oder sie können regelmäßig auf monatlicher, vierteljährlicher oder jährlicher Basis ausgezahlt werden.

- **Vorzeitige Auszahlung** — Es ist möglich, Kapital vor dem Fälligkeitsdatum abzuheben, aber in den meisten Fällen wird eine Vorfälligkeitsentschädigung erhoben.

Eine CD gilt als sichere Anlage und ist eine ideale Möglichkeit, Geld für einen bestimmten Zeitraum aufzubewahren und gleichzeitig Zinsen zu verdienen. Die Zinsen auf einer CD sind höher als die Zinsen, die Sie auf Sparkonten verdienen können. Je länger die Laufzeit der CD, desto höher die Zinsen, die Sie dafür verdienen.

Wenn eine CD fällig wird, wird Ihre Bank Sie benachrichtigen und Ihnen die Möglichkeit geben, Ihr Geld einzulösen oder in eine andere CD umzuwandeln.

Das Fälligkeitsdatum wird auch auf Unternehmens- und Staatsanleihen angewendet. Diese funktionieren ähnlich wie CDs, mit einer wichtigen Ausnahme. Sie können Ihr Geld nicht vorzeitig abheben. Sie müssen warten, bis die Bindung reift.

## Beispiel für Fälligkeitsdatum

Charles hat 10.000 US-Dollar in eine CD mit 5 Jahren Laufzeit bei einer örtlichen Bank zu einem Zinssatz von 2,25 Prozent investiert. Die CD läuft in ein paar Monaten ab und da er das Geld später im Jahr benötigen könnte, teilt er der Bank mit, dass er die CD in eine 6-Monats-CD umwandeln möchte.

## Höhepunkte

- Das Fälligkeitsdatum bezieht sich auf den Zeitpunkt, zu dem der Kapitalbetrag eines festverzinslichen Instruments an einen Anleger zurückgezahlt werden muss.

- Mit Erreichen des Fälligkeitsdatums entfallen die regelmäßig an die Anleger gezahlten Zinszahlungen, da der Schuldvertrag nicht mehr besteht.

- Das Fälligkeitsdatum bezeichnet ebenfalls das Fälligkeitsdatum, an dem ein Kreditnehmer einen Ratenkredit vollständig zurückzahlen muss.

- Das Fälligkeitsdatum wird verwendet, um Anleihen in drei Hauptkategorien einzuteilen: kurzfristig (ein bis drei Jahre), mittelfristig (10 oder mehr Jahre) und langfristig (in der Regel 30-jährige Staatsanleihen).
