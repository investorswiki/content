# Schöpfungseinheit
## Was ist eine Erstellungseinheit?

Eine Creation Unit ist ein Block neuer [Aktien](/shares),. die von einem börsengehandelten Fonds (ETF)-Unternehmen an einen Broker-Dealer zum Verkauf auf dem freien Markt verkauft werden. Creation-Unit-Blöcke haben in der Regel eine Größe zwischen 25.000 und 600.000 Anteilen. Broker-Dealer können die Anteile entweder gegen Barzahlung oder durch Sachleistungen kaufen.

## Wie Creation Units funktionieren

Eine ETF-Erstellungseinheit ist von zentraler Bedeutung für den Prozess, den ein Emittent börsengehandelter Fonds durchläuft, wenn er dem Markt neue ETF-Anteile zur Verfügung stellt. ETF-Emittenten arbeiten mit ETF-Vertreibern zusammen, um neue Anteile in Creation Units an [Broker-Dealer auszugeben](/broker-dealer).

Creation Units werden in der Regel an Broker-Dealer verkauft, die die Anteile in verschiedenen Formen bezahlen können. Beim Verkauf werden die Anteile zum [Nettoinventarwert](/nav) (NAV) des Fonds bewertet. Emittenten von börsengehandelten Fonds können jedoch mit Broker-Händlern über die Verkaufsbedingungen verhandeln und erhalten für die Transaktion in der Regel Bargeld oder Sachleistungen. In-Kind-Aktien sind eine gängige Methode, um die Anteile an einer Creation Unit zu bezahlen. Aktientransaktionen in Form von Sachleistungen erfordern, dass der Käufer ein Portfolio von Wertpapieren zusammenstellt, die dann als eine Art Handel an die ausgebende Gesellschaft übertragen werden. Aufgrund dieses Austauschs können Verkäufe von Creation Units vor der Börsennotierung Gewinne und Verluste für den Broker-Dealer generieren. Diese Gewinne und Verluste können steuerliche Auswirkungen haben. Broker-Dealer sind daher beim Kauf neuer Creation Units erheblichen Risiken ausgesetzt.

## Autorisierte Teilnehmer

Emittenten von börsengehandelten Fonds verwenden Creation Units für die Ausgabe neuer Anteile durch Broker-Dealer. Sie haben auch Beziehungen zu Broker-Dealer-Trading-Desks, die als [autorisierte Teilnehmer fungieren](/authorizedparticipant). Autorisierte Teilnehmer gibt es nur bei Open-End-ETFs und sie dienen dazu, erhebliche Abweichungen vom NAV eines Fonds auf dem Handelsmarkt zu überwachen. Da ETFs den ganzen Tag über aktiv an Finanzmarktbörsen gehandelt werden, bieten sie Prämien und Abschläge auf ihren buchhalterischen NAV. Autorisierte Teilnehmer folgen diesem Marktmechanismus und helfen bei der Verwaltung von Prämien und Abschlägen für den NAV-Handel.

## Gemeinsame Praktiken für Erstellungseinheiten

[SPDR-Serie von](/spiders) State Street Global Advisors ist eine der größten auf dem Markt für Investitionen im ETF-Sektor und bietet ein gutes Beispiel dafür, wie Creation Units verwendet werden. Zunächst einmal haben alle Fonds der Serie die gleichen Bestimmungen für die Erstellungseinheit. Zu verschiedenen Zeiten gibt die Serie nach Ermessen der Fondsgesellschaft Creation Units jedes Sektorfonds an Market Maker oder andere Broker-Dealer aus. Für die SPDR-Serie werden Creation Units in Blöcken von 50.000 Aktien ausgegeben. Finanzinstitute stellen Sachwertpapiere und/oder Barmittel für den Marktwert der Schöpfungseinheit bereit. Transaktionen von Creation Units werden von ALPS Portfolio Solutions Distributor, Inc. ermöglicht.

## Höhepunkte

- Eine Creation Unit ist ein Block neuer Aktien, die von einem börsengehandelten Fonds (ETF)-Unternehmen an einen Broker-Dealer zum Verkauf auf dem freien Markt verkauft werden.

- Zum Nettoinventarwert (NAV) des Fonds bewertet, bergen Creation Units erhebliche Risiken für Broker-Dealer, da sie entweder Gewinne oder Verluste erwirtschaften.

- Creation Units können unterschiedlich groß sein, wobei die meisten jeweils zwischen 25.000 und 600.000 ETF-Anteile enthalten. Sie können entweder durch den Erhalt von Bar- oder Sachaktien für die Transaktion bezahlt werden.
