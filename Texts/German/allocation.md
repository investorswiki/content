# Zuweisung
Die Zuteilung ist eine Zuteilung von Token oder Eigenkapital, die für einen bestimmten Investor, ein Team, eine Gruppe, eine Organisation oder eine andere verbundene Einheit verdient, gekauft oder reserviert werden kann.

Für ein Krypto-Team in den Whitepaper-Phasen kann das Führungsteam oder die Community daran arbeiten, zu entscheiden, welche Zuordnungen von Token auf verschiedene Initiativen und Abteilungen wie Entwicklung, Marketing, Betriebskosten usw. aufgeteilt werden sollen. Wenn ein Team eine Stiftung oder eine andere Einheit hat, die die Kontrolle über die Mittel hat, kann es auch entscheiden, eine Zuweisung für eine Token-Schatzkammer zu erstellen, die wie vom Team oder der Gemeinschaft festgelegt verwendet werden soll.

Investoren können auch Zuteilungen in Investitionsrunden erhalten. Beispielsweise kann ein Team Zuteilungen einer festgelegten „Ticketgröße“ oder eines Höchstbetrags an Frühinvestoren in einer privaten Verkaufsrunde verkaufen. In diesem Fall würde jede dieser einzelnen investierenden Parteien eine Zuteilung des Gesamtbetrags besitzen, der in dieser bestimmten Verkaufsrunde angeboten wird. Ein einzelnes Unternehmen kann möglicherweise Zuteilungen aus mehreren Verkaufsrunden halten, was bedeutet, dass es schließlich an verschiedenen Phasen eines Initial Coin Offering (ICO) oder Token-Verkaufsereignisses teilnehmen könnte, mit einer vordefinierten Zuteilung für jede Phase.

Teammitglieder, die an einem bestimmten Coin, Protokoll oder Projekt arbeiten, können auch einen Teil einer Teamzuteilung als Belohnung für ihre Arbeit erhalten. Beispielsweise könnten diese Zuteilungen zu einem bestimmten Zeitpunkt auf einmal ausgezahlt werden – beispielsweise am Datum eines Token-Generierungsereignisses (TGE) – oder nach einem vordefinierten Zeitplan über die Zeit verteilt werden.

In bestimmten Situationen werden die Zuteilungen im Rahmen einer Blockprämie über die Zeit oder über eine Vesting Period oder Cliff verteilt.
