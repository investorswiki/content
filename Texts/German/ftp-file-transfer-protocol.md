# Dateiübertragungsprotokoll (FTP)
## Was ist das File Transfer Protocol (FTP)?

Der Begriff File Transfer Protocol (FTP) bezieht sich auf einen Prozess, bei dem Dateien zwischen Geräten über ein Netzwerk übertragen werden. Der Prozess funktioniert, wenn eine Partei einer anderen erlaubt, Dateien über das Internet zu senden oder zu empfangen. Ursprünglich als Möglichkeit für Benutzer zur Kommunikation und zum Austausch von Informationen zwischen zwei physischen Geräten verwendet, wird es heute häufig zum Speichern von Dateien in der [Cloud verwendet](/cloud-computing),. die normalerweise ein sicherer Ort ist, der entfernt gehalten wird.

FTP kann von einem [Unternehmen](/business) oder einer Einzelperson verwendet werden, um Dateien von einem Computersystem auf ein anderes zu übertragen, oder von Websites, um Dateien von ihren Servern hoch- oder herunterzuladen.

## Funktionsweise des File Transfer Protocol (FTP).

Das Dateiübertragungsprotokoll ermöglicht es Einzelpersonen und Unternehmen, elektronische Dateien mit anderen zu teilen, ohne sich im selben Raum befinden zu müssen. Dies kann über einen FTP-Client oder über die Cloud erfolgen. Unabhängig von der Option benötigen beide Parteien eine funktionierende Internetverbindung.

Die meisten [Webbrowser](/web-20) verfügen über FTP-Clients, mit denen Benutzer Dateien von ihrem Computer auf einen Server und umgekehrt übertragen können. Einige Benutzer möchten möglicherweise einen FTP-Client eines Drittanbieters verwenden, da viele von ihnen zusätzliche Funktionen bieten. Beispiele für kostenlos herunterladbare FTP-Clients sind FileZilla Client, FTP Voyager, WinSCP, CoffeeCup Free FTP und Core FTP.

Viele Leute haben FTP schon einmal verwendet, ohne es überhaupt zu merken. Wenn Sie jemals eine Datei von einer Webseite heruntergeladen haben, haben Sie FTP verwendet. Der erste Schritt ist die Anmeldung, die automatisch oder durch manuelle Eingabe eines Benutzernamens und Passworts erfolgen kann. FTP erfordert außerdem, dass Sie über eine bestimmte Portnummer auf einen FTP-Server zugreifen. Sobald Sie über Ihren FTP-Client auf den FTP-Server zugreifen, können Sie jetzt Dateien übertragen. Nicht bei allen öffentlichen FTP-Servern müssen Sie sich anmelden, da Sie auf einigen Servern anonym darauf zugreifen können.

Wie oben erwähnt, wurde FTP ursprünglich entwickelt, um Dateien zwischen zwei physischen Computern zu senden und zu empfangen. Aber mit technologischen Änderungen können Benutzer Dateiübertragungen über die Cloud ausführen. Die Nutzung der Cloud ermöglicht bequeme und sichere Übertragungen (was Einzelpersonen und Unternehmen vor [Datenschutzverletzungen schützen könnte](/data-breach) ) und das zu relativ geringen Kosten.

> Der Begriff FTP-Client bezieht sich auf die Software, mit der Sie Dateien an eine andere Partei übertragen können.

>

## Besondere Überlegungen

Das Dateiübertragungsprotokoll ist eines von vielen verschiedenen Protokollen, die bestimmen, wie sich Computer und Computersysteme im Internet verhalten. Andere solche Protokolle umfassen:

- **Hypertext Transfer Protocol (HTTP):** Entwickelt, um Daten über das Internet zu übertragen

- **Internet Message Access Protocol (IMAP):** Bietet Zugriff auf Schwarzes Brett oder E-Mail-Nachrichten von einem gemeinsam genutzten Dienst

- **Network Time Protocol (NTP):** Synchronisiert Uhrzeiten auf Computern über ein Netzwerk

FTP ermöglicht es Computern im Internet, Dateien hin und her zu übertragen. Als solches ist es ein unverzichtbares Werkzeug für diejenigen, die heute Websites erstellen und pflegen.

### Worauf in einem FTP-Client zu achten ist

Einzelne FTP-Clients bieten verschiedene Funktionen, mit denen Benutzer die Art und Weise ändern können, wie sie Dateien hochladen und herunterladen. Wenn Sie beispielsweise FileZilla verwenden, können Sie mit dem Programm [Bandbreitenbeschränkungen](/bandwidth) für Dateien festlegen. Auf diese Weise können Sie die Upload- und Download-Geschwindigkeiten steuern, was hilfreich sein kann, wenn Sie mehrere Dateiübertragungen gleichzeitig verwalten.

Andere Funktionen, nach denen Sie vielleicht in einem FTP-Client suchen möchten, sind die Authentifizierung mit öffentlichen Schlüsseln, die Möglichkeit, Dateikomprimierungsstufen festzulegen, oder Tools, mit denen Sie einen Server mithilfe von Dateimasken durchsuchen können.

## Beispiel für File Transfer Protocol (FTP)

FTP-Software ist relativ einfach einzurichten. FileZilla ist ein kostenloser, herunterladbarer FTP-Client. Geben Sie die Adresse des Servers, auf den Sie zugreifen möchten, den Port und das Passwort für den Zugriff auf den Server ein.

Sobald der Zugriff gewährt wurde, sind die Dateien des Benutzers auf seinem lokalen System sowie der Server, auf den zugegriffen wird, sichtbar. Der Benutzer kann Dateien vom Server auf das lokale System herunterladen oder Dateien vom lokalen System auf den Server hochladen. Sie können auch Änderungen an Dateien auf dem Server vornehmen, sofern sie die entsprechende Berechtigung dazu haben.

## Höhepunkte

- FTP ist ein unverzichtbares Tool für diejenigen, die Websites erstellen und pflegen.

- Das Dateiübertragungsprotokoll ist eine Möglichkeit zum Herunterladen, Hochladen und Übertragen von Dateien von einem Ort zum anderen im Internet und zwischen Computersystemen.

- FTP ermöglicht die Übertragung von Dateien hin und her zwischen Computern oder über die Cloud.

- Benutzer benötigen eine Internetverbindung, um FTP-Übertragungen auszuführen.

- Viele FTP-Clients können kostenlos heruntergeladen werden, obwohl die meisten Websites FTP bereits integriert haben.
