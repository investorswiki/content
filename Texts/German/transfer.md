# Transfer
## Was ist eine Überweisung?

Eine Übertragung beinhaltet die Übertragung von Vermögenswerten, Geldmitteln und/oder Eigentumsrechten von einem Konto auf ein anderes. Eine Übertragung kann einen Austausch von Geldern erfordern, wenn es sich um einen Eigentumswechsel handelt, z. B. wenn ein Investor einen [Immobilienbesitz verkauft](/realestate). In diesem Fall erfolgt eine Eigentumsübertragung vom Verkäufer auf den Käufer und gleichzeitig eine Geldüberweisung in Höhe des ausgehandelten Preises vom Käufer an den Verkäufer.

Der Begriff Überweisung kann sich auch auf die Übertragung eines Kontos von einer Bank oder einem Broker zu einer anderen beziehen.

## Überweisungen verstehen

Eine Übertragung ist ein Begriff, der in verschiedenen Branchen und Transaktionstypen eine breite Konnotation hat. Nachfolgend finden Sie einige Beispiele dafür, wie eine Übertragung für eine Einzelperson, eine Gruppe oder ein Unternehmen ausgelöst wird.

In allen Fällen beinhaltet eine Übertragung die Bewegung von Geldern oder Vermögenswerten und die Möglichkeit, neues Eigentum zu übertragen.

## Banküberweisungen

Wenn ein Kontoinhaber Gelder von einem Konto auf ein anderes überweist, beispielsweise von einem [Girokonto](/checkingaccount) auf ein [Sparkonto](/savingsaccount) mit einem höheren Zinssatz oder von einem Sparkonto auf ein [IRA -](/ira) [Konto](/ira),. hat eine Überweisung stattgefunden. Die Überweisung muss nicht innerhalb derselben Bank erfolgen. Es kann sich um eine Interbankenüberweisung von einem Konto bei Bank A zu einem anderen bei Bank B handeln. Innerhalb der Bankenbranche können Gelder auch grenzüberschreitend durch [Überweisungen](/wiretransfer) von einem inländischen Konto auf ein ausländisches Konto und umgekehrt überwiesen werden. Die empfangenden Konten könnten von demselben Kontoinhaber gehalten werden oder einer anderen Person oder Firma gehören. Gelder werden normalerweise unter anderem zu Zwecken der Finanzplanung überwiesen, um bessere Investitionsraten zu nutzen, Zahlungen für Waren oder Dienstleistungen zu leisten, das erforderliche Guthaben eines anderen Kontos aufzubringen, jemandem oder einer Organisation zu schenken oder um Geld zu sparen Gründe dafür.

## Maklerüberweisungen

Anleger transferieren normalerweise Gelder und Vermögenswerte von innerhalb oder außerhalb ihrer [Maklerkonten](/brokerageaccount). Ein Anleger, der sein Anlagekonto aufstocken muss, damit er mehr Anteile kaufen kann, kann die Übertragung von einem anderen Anlagekonto vornehmen, das bei demselben Makler oder bei einem anderen Makler geführt wird. Die meisten Vermögenswerte wie [Unternehmensaktien](/stock),. [Anleihen](/bond),. [Einlagenzertifikate](/certificateofdeposit) (CDs), [Investmentfonds](/mutualfund) usw. können [in Form](/distributioninkind) von Sachwerten von einem Anlagekonto auf ein anderes übertragen werden. Diese Vermögenswerte können auch als Schenkung an eine andere Person oder gemeinnützige Organisation übertragen werden. Geldüberweisungen können auch vom Bankkonto eines Anlegers auf sein Maklerkonto und umgekehrt erfolgen.

## Zahlungen und Kryptowährung

Im Zahlungsbereich beinhaltet eine Überweisung das Verschieben von Geld von einem Konto auf ein anderes. Interne Überweisungen können Geldtransfers von einem Konto auf ein anderes umfassen, das derselben Person oder Organisation gehört. Überweisungen zwischen Personen umfassen Überweisungen, aber auch elektronische Zahlungen wie PayPal und Venmo.

In der Kryptoökonomie werden Gelder und Kryptowährungen [häufig](/cryptocurrency) zwischen Benutzern an öffentliche Adressen übertragen, wo auf die Gelder mit einem privaten Schlüssel zugegriffen werden kann, der für jeden Benutzer einzigartig ist. Wenn Waren und Dienstleistungen gehandelt werden, würde der Käufer beispielsweise [Bitcoins](/bitcoin) von seinen Beständen an die digitale Adresse des Verkäufers transferieren. Dies ist ähnlich wie bei jedem anderen Markt, ob stationär oder E-Commerce, wo Transfers zwischen Verkäufern und Käufern das sind, was einen Markt definiert.

Kryptowährungen können auch von einer Kryptowährungsbörse zu einer anderen Börse übertragen werden, wo sie auf einem Konto hinterlegt werden, das ebenfalls vom Absender oder einer anderen Person geführt wird, die der Verkäufer verschenkt oder mit der er Geschäfte abwickelt. Auch der Handel mit Kryptos gegen [Fiat-Geld](/fiatmoney) löst eine Art Überweisung aus, bei der der Benutzer Geld von seinem Bankkonto an die Börse überweisen und mit dem Geld Coins kaufen kann.

## Titelübertragungen von Vermögenswerten

Eigentumsrechte an Vermögenswerten wie Autos, Grundstücken und Häusern können übertragen werden, wenn sie an eine Einzelperson oder ein Unternehmen verkauft oder verschenkt werden. Wenn ein Hausbesitzer sein Haus an einen anderen verkauft, muss er die [Quitclaim-Urkunde](/quitclaimdeed) oder andere Formulare ausfüllen, die zur Übertragung des Eigentumstitels erforderlich sind. Ein Landbesitzer kann seinen Titel auf jeden oder jede Körperschaft übertragen, wenn er möchte. Eine Eigentumsübertragung kann durch den Verkauf des Grundstücks, die Schenkung, die Übereignung des Eigentums an einen [Begünstigten](/beneficiary),. nach einem Gerichtsbeschluss oder durch Zwangsvollstreckung erfolgen. Sogar das Eigentum an einem Telefon kann übertragen werden, wenn der Mobilfunkanbieter dies zulässt. In diesem Fall können das Mobiltelefon, die Telefonnummer und der Vertrag an eine andere Stelle übertragen werden, die das Eigentum daran übernimmt und die rechtliche Verantwortung für die Erfüllung der Rechnungszahlungen bei Fälligkeit übernimmt.

## Kreditübertragung

Auch Darlehen können übertragbar sein. Beispielsweise kann ein Hausbesitzer mit einem [übertragbaren Darlehen](/assumablemortgage) die Hypothek auf eine andere Person wie den Käufer übertragen, wenn er/sie für das Darlehen in Frage kommt. Dies könnte eine Win-Win-Lösung für beide an der Transaktion beteiligten Parteien sein. Beim Verkauf eines Autos kann der Verkäufer das Eigentum zusammen mit dem [Autokredit](/car-title-loan) auf den Käufer übertragen, wenn dieser für kreditwürdig befunden wird.

## Höhepunkte

- Eine Übertragung ist die Bewegung von Vermögenswerten, Geldern oder Eigentumsrechten von einem Ort zum anderen.

- Bankgeschäfte, Maklergeschäfte, Kryptowährungen, Vermögenstitel und Darlehensübertragungen sind einige Beispiele für Domänen und Transaktionstypen, bei denen Übertragungen stattfinden.

- Eine Übertragung wird auch verwendet, um den Prozess zu beschreiben, durch den das Eigentum an Geldern oder Vermögenswerten auf einen neuen Eigentümer übertragen wird
