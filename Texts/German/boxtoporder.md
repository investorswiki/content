# Box-Top-Bestellung
## Was ist eine Box-Top-Bestellung?

Eine Box-Top-Order ist eine Kauf- oder Verkaufsorder zum besten [Marktpreis](/market-price). Es ist eine veraltete Art, sich auf eine [Marktorder zu beziehen](/marketorder).

Eine Market Order gehört zu den häufigsten und unkompliziertesten Transaktionen an den Finanzmärkten. Es soll so schnell wie möglich zum aktuellen Angebotspreis ausgeführt werden und ist häufig die Standardauftragsart auf den meisten Brokerage-Plattformen.

## Box-Top-Bestellungen verstehen

Box-Top-Orders führen Marktorders zum aktuell besten Preis aus. Kann die Order nicht vollständig ausgeführt werden, wird für die verbleibenden Aktien eine Limitorder zu dem Preis platziert, zu dem der ausgeführte Teil ausgeführt wurde. Eine [Limit-Order](/limitorder) ist eine Order, die platziert wird, um eine Kauf- oder Verkaufstransaktion zu einer festgelegten Anzahl von Aktien und zu einem festgelegten Limitpreis oder besser auszuführen.

Wenn ein Händler beispielsweise eine Box-Top-Order zum Kauf von 1.000 Aktien zum aktuellen Marktpreis von 50 $ eingegeben hat und nur die Hälfte der Aktien zu diesem Preis gehandelt wird, wird eine Kauf-Limit-Order für die anderen 500 Aktien platziert. Wenn der Preis zu irgendeinem Zeitpunkt während der Laufzeit der Order auf 50 $ zurückkehrt, tritt die Limit-Order ein und die verbleibenden Aktien werden zu 50 $ gehandelt.

## Limit-Orders und Stop-Orders

Limit-Orders ermöglichen es einem Anleger, die Zeitdauer zu begrenzen, für die eine Order ausstehend sein kann, bevor sie storniert wird. Limit-Orders ermöglichen die effektive Ausführung von Box-Top-Orders. Die Ausführung einer Limit-Order ist nicht immer garantiert, trägt aber dazu bei, dass der Anleger keine Gelegenheit verpasst, zu einem bestimmten Preispunkt zu kaufen oder zu verkaufen. Diese Aufträge legen ein Maximum oder Minimum fest, zu dem ein Händler bereit ist, eine bestimmte Aktie zu kaufen oder zu verkaufen. Ein Anleger kann bestimmte Bedingungen für Limitaufträge festlegen, z. B. den gleichzeitigen Kauf oder Verkauf aller gewünschten Aktien, wenn der Handel ausgeführt werden soll, was als [Alles-oder-Nichts-](/aon) Auftrag bezeichnet wird.

Andererseits stellt eine [Stop-Loss-Order](/stop-lossorder) sicher, dass eine Transaktion nicht zu einem schlechteren Preis als dem angegebenen Ziel stattfindet. Es kann verwendet werden, um ein bestehendes Instrument zu verkaufen oder eine neue Transaktion einzugehen. Bei einer Stop-Order wird ein Trade nur ausgeführt, wenn das Wertpapier einen bestimmten Preis erreicht, der als Stop-Preis bekannt ist. Für Anleger, die ihre Aktien über einen bestimmten Zeitraum nicht überwachen können, sind Stop-Orders besonders vorteilhaft. Maklerfirmen legen manchmal sogar kostenlose Stop-Orders fest.

Während Stop-Orders möglicherweise keine Maklergebühr beinhalten, können Limit-Orders eine höhere Provision beinhalten. Der Vorteil einer Limit-Order besteht darin, dass sie garantiert, dass ein Trade zu einem bestimmten Preis ausgeführt wird; Es ist jedoch möglich, dass der Auftrag nicht ausgeführt wird, wenn der Limitpreis nicht erreicht wird.

## Höhepunkte

- Eine Marktorder ist eine Anweisung, ein Wertpapier sofort zum aktuellen Preis zu kaufen oder zu verkaufen.

- Eine Box-Top-Order ist eine ungewöhnliche Art, sich auf eine Marktorder zu beziehen.

- Eine Limit-Order ist stattdessen eine Anweisung, nur zu einem vom Anleger festgelegten Preis zu kaufen oder zu verkaufen.
