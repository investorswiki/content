# One-Cancels-the-Other (OCO)-Auftrag
Eine „One Cancels the Other“-Order (OCO) besteht aus zwei gleichzeitig erstellten Orders, von denen jedoch nur eine ausgeführt werden kann. Das bedeutet, sobald eine der Bestellungen ganz oder teilweise ausgeführt wird, wird die andere automatisch storniert. Obwohl seltener, können OCO-Orders auch als Order Cancels Order bezeichnet werden.

Im Wesentlichen ist ein OCO eine bedingte Order, die eine Limit-Order mit einer Stop-Limit-Order kombiniert, was sie zu einer grundlegenden Form der Handelsautomatisierung macht. Mit anderen Worten, eine OCO-Order gibt Ihnen die Möglichkeit, zwei Limit-Orders gleichzeitig zu platzieren. Dies macht die OCO-Funktion zu einem großartigen Handelsinstrument zur Verbesserung der Erfolgsquote (Gewinnmitnahme) und Minimierung potenzieller Verluste (Stop-Loss).

OCO-Orders können auch nützlich sein, wenn Sie versuchen, Positionen einzugeben. Wenn BNB beispielsweise zwischen 35 $ und 40 $ handelt, können Sie eine OCO-Order erstellen, die entweder bei einem [Widerstandsausbruch](/resistance) (über 40 $) kauft oder kauft, wenn der Preis auf das [Unterstützungsniveau von 35 $ fällt](/support). Es ist jedoch erwähnenswert, dass, wenn eine der Orders ausgeführt wird, die andere storniert wird. Abhängig von der Situation möchten Sie also möglicherweise eine neue Bestellung aufgeben, nachdem Ihr OCO ausgelöst wurde.

Zusammenfassend lässt sich sagen, dass Sie mit einer OCO-Order sicherer handeln können, indem Sie entweder potenzielle Gewinne sperren oder Risiken begrenzen. Es bietet auch mehr Vielseitigkeit, da Sie Positionen eingehen oder verlassen können, ohne zwischen einer [bullischen](/bullmarket) oder [bärischen Tendenz wählen zu müssen](/bearmarket). Abgesehen davon können OCO-Orders Händlern, die nicht wollen (oder keine Zeit haben), die Marktaktivitäten täglich zu verfolgen, Seelenfrieden bringen.

## Höhepunkte

- One-cancels-the-other (OCO) ist eine Art bedingter Auftrag für ein Auftragspaar, bei dem die Ausführung des einen automatisch den anderen storniert.

- Auf vielen Handelsplattformen können mehrere bedingte Orders platziert und andere Orders storniert werden, wenn eine ausgeführt wurde.

- Händler führen im Allgemeinen OCO-Aufträge für volatile Aktien aus, die über eine große Preisspanne gehandelt werden.
