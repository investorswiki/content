# Warnung vor Kreditbetrug
## Was ist eine Warnung vor Kreditbetrug?

Eine Kreditbetrugswarnung ist eine an eine [Kreditauskunftei gesendete Mitteilung,](/creditbureau) dass die Identität eines Verbrauchers möglicherweise gestohlen wurde und ein Antrag auf einen neuen Kredit im Namen dieses Verbrauchers möglicherweise nicht legitim ist. Eine Kreditbetrugswarnung kann Sie und Ihren Kredit davor schützen, dass jemand betrügerische Kreditkonten unter Ihrem Namen eröffnet. Sie sollten eine der großen Kreditauskunfteien (Experian, TransUnion und Equifax) informieren, wenn Ihre Karte gestohlen wurde, und sie können eine Warnung vor Kreditbetrug herausgeben.

## Kreditbetrugswarnung verstehen

Eine Kreditbetrugswarnung kann von einer Einzelperson bei den Kreditauskunfteien kostenlos für die Person, die sie einreicht, erlassen werden. Um diesen Vorgang abzuschließen, muss die Person einen Identitätsnachweis vorlegen, damit die Kreditauskunftei bestätigen kann, dass die Anfrage gültig ist.

## Arten von Kreditbetrugswarnungen

Es gibt drei Arten von Kreditbetrugswarnungen – anfängliche, erweiterte und aktive militärische Warnungen.

### Erste Warnung

Eine Erstwarnung ist 90 Tage gültig und kann danach für 90 Tage verlängert werden.

### Erweiterte Warnung

Eine erweiterte Warnung ist sieben Jahre lang gültig. Sie müssen den Kreditauskunfteien einen Polizeibericht vorlegen, in dem sie darüber informiert werden, dass Sie Opfer eines Identitätsdiebstahls geworden sind und die Straftat den Behörden gemeldet haben.

### Aktiver militärischer Alarm

Ein aktiver Militäralarm ist ein Jahr lang gültig und kann dazu beitragen, Ihr Guthaben während Ihres Einsatzes zu schützen. Menschen reichen häufig Kreditbetrugswarnungen ein, wenn sie glauben, Opfer eines [Identitätsdiebstahls geworden zu sein,](/identitytheft) oder wenn ihre Informationen im Rahmen einer Datenschutzverletzung kompromittiert wurden.

> Wenn Sie sicher sind, dass Ihre Identität gestohlen wurde, ziehen Sie für einen noch größeren Schutz eine [Kreditsperre in Betracht](/credit-freeze).

>

## Besondere Überlegungen

Wenn Sie glauben, dass jemand Ihre persönlichen oder finanziellen Daten gestohlen haben könnte und sie dazu verwenden könnte, betrügerische Konten in Ihrem Namen zu eröffnen, wenden Sie sich an eine der drei großen Kreditauskunfteien: Experian, Equifax und TransUnion. Fordern Sie an, dass sie eine Kreditbetrugswarnung für Ihr Konto platzieren. Sie können den Vorgang normalerweise online abschließen, aber Sie können ihn auch per Post oder Telefon erledigen. Das Büro, das Sie kontaktieren, soll dann die anderen beiden über die Betrugswarnung informieren, aber Sie möchten vielleicht alle drei selbst kontaktieren, um Ihre Grundlagen zu decken. Wenn Sie eine Kreditbetrugswarnung einrichten, haben Sie Anspruch auf kostenlose Kreditauskünfte von jeder großen Kreditauskunftei.

Untersuchen Sie Ihre Kreditauskunft auf Anzeichen von Betrug, z. B. Konten, die Sie nicht kennen. Während die Kreditbetrugswarnung in Kraft ist, wird erwartet, dass das Finanzinstitut, das die Kreditanfrage erhält, zusätzliche Schritte unternimmt, um die Identität des Antragstellers zu überprüfen, wenn jemand, einschließlich Ihnen, versucht, einen Kredit in Ihrem Namen zu beantragen. Daher kann eine Betrugswarnung ein wenig Ärger verursachen, wenn Sie selbst ein neues Konto eröffnen möchten, aber es kann auch genug Ärger machen, um einen Dieb daran zu hindern, ein betrügerisches Konto in Ihrem Namen zu eröffnen.

## Höhepunkte

- Initial, Extended und Active Military sind die drei Arten von Kreditbetrugswarnungen.

- Während die Kreditbetrugswarnung in Kraft ist, wird von dem Kreditgeber, der Kreditanfragen erhält, erwartet, dass er zusätzliche Schritte unternimmt, um zu überprüfen, ob die Anfrage echt ist.

- Eine Kreditbetrugswarnung wird an Kreditauskunfteien gesendet, um sie darüber zu informieren, dass die Identität eines Verbrauchers möglicherweise gestohlen wurde.

- Um eine Kreditbetrugswarnung abzuschließen, muss der Besitzer der gestohlenen Karte einen Identitätsnachweis vorlegen, um zu bestätigen, dass die Anfrage gültig ist.
