# Verschlüsselung
## Was ist Verschlüsselung?

Verschlüsselung ist eine Methode der Datensicherheit, die gespeicherte oder von einem Computer oder Gerät zu einem anderen übertragene Informationen auf eine Weise verschlüsselt, die nur von der Person oder dem Computer mit dem Verschlüsselungsschlüssel verschlüsselt werden kann. Die gängigsten Techniken, die von führenden Finanzinstituten zum Schutz vertraulicher personenbezogener Daten verwendet werden, beruhen auf Datenverschlüsselungstechnologie.

Aufgrund der Menge sensibler Informationen, die täglich über das Internet ausgetauscht werden, setzen auch immer mehr Websites und Online-Handelsdienste ausgeklügelte Verschlüsselungssysteme ein.

## Tiefere Definition

Es gibt mehrere Möglichkeiten, Daten zu verschlüsseln.

- Verschlüsselung mit privatem Schlüssel bedeutet, dass jeder Computer in einem Netzwerk über einen geheimen Code oder Schlüssel verfügt, der verwendet wird, um Informationen zu verschlüsseln, bevor er sie an einen anderen Computer sendet. Damit die Verschlüsselung mit privatem Schlüssel funktioniert, müssen Sie die Computer kennen, die miteinander „sprechen“, und auf denen der Schlüssel installiert sein muss. Das Besitzen eines privaten Schlüssels ähnelt einem geheimen Code, der zwischen zwei Computern geteilt wird und nur ihnen bekannt ist, um zwischen ihnen gesendete Nachrichten zu entschlüsseln. Wenn die Nachricht von jemand anderem gesehen wird, sieht es aus wie bedeutungslose Informationen.

- Die Verschlüsselung mit öffentlichem Schlüssel verwendet sowohl einen öffentlichen als auch einen privaten Schlüssel. Nur Ihr Computer kennt seinen privaten Schlüssel, aber er gibt seinen öffentlichen Schlüssel an jeden Computer weiter, der eine sichere Kommunikation mit ihm erfordert. Der Computer, der die Nachricht empfängt, verwendet seinen eigenen privaten Schlüssel und den vom Ursprungscomputer empfangenen öffentlichen Schlüssel, um verschlüsselte Nachrichten zu entschlüsseln.

Für Unternehmen, die auf die Übertragung sensibler Daten angewiesen sind, insbesondere in der Finanzbranche, bedeutet ein Datenleck aufgrund fehlender Verschlüsselung das Verhängnis. Abgesehen davon, dass Sie sich mit einem beschädigten Ruf auseinandersetzen müssen, können erhebliche finanzielle und Arbeitskosten anfallen, um den entstandenen Schaden zu beheben.

Wenn sensible Finanzinformationen auf einem Gerät wie einem Laptop oder Smartphone gespeichert sind, werden sie oft von Benutzern übersehen, die davon ausgehen, dass die Daten sicher sind. Diese Systeme können jedoch immer noch gehackt werden, und es kann auf vertrauliche Informationen zugegriffen werden. Die Verschlüsselung von auf Geräten gespeicherten Daten ist notwendig, um sie vor Hackern oder Dieben zu schützen.

## Verschlüsselungsbeispiel

Jedes Mal, wenn Sie sich bei einem webbasierten E-Mail-Dienst oder einer anderen auf Berechtigungen basierenden Website anmelden, geben Sie einen Benutzernamen und ein Passwort ein, die dann vom Browser verschlüsselt werden. Der Server, auf dem die Website gehostet wird, und Ihr Computer oder Mobilgerät einigen sich dann auf das Protokoll, das sie verwenden, um verschlüsselte Informationen online hin und her zu übertragen.

Die Verwendung verschlüsselter Online-Kommunikation bietet Ihnen mehrere Vorteile, darunter:

- Reduzieren Sie die Anzahl der Personen, die wissen, was Sie tun.

- Erschweren Sie Hackern den Zugriff auf Ihre Anmeldedaten.

- Verschleierung Ihrer Browserdaten.

- Verringern der Wahrscheinlichkeit, zum Besuch einer schädlichen Website getäuscht zu werden.

Ein Großteil der Zeit, die Menschen im Internet verbringen, besteht darin, ihre Informationen an andere zu senden. Wenn Sie online etwas kaufen oder ein Anmeldeformular ausfüllen, um ein Online-Konto zu eröffnen, geben Sie viele persönliche Informationen ein. Daher ist eine Verschlüsselung unerlässlich. Ob Ihre Online-Kommunikation sicher verschlüsselt ist, erkennen Sie in Ihrem Browser an einem Vorhängeschloss-Symbol in der URL-Leiste und die Website-Adresse enthält „https://“.

## Höhepunkte

- Verschlüsselung ist ein Mittel zum Sichern von Daten, indem sie mathematisch so verschlüsselt werden, dass sie nur von Personen mit dem richtigen Schlüssel oder Chiffre gelesen oder entschlüsselt werden können.

- Digitale Verschlüsselungsverfahren übersetzen Daten mit einem Algorithmus, der die Originalinformationen außer für autorisierte Benutzer unlesbar macht.

- Verschlüsselung ist in einer digital vernetzten Welt von entscheidender Bedeutung, um private Informationen, Nachrichten und Finanztransaktionen privat und sicher zu halten.
