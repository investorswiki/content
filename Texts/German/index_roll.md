# Indexrolle
## Was ist eine Indexrolle?

Der Begriff Index-Roll bezieht sich auf eine passive [Indexanlagestrategie](/index-investing),. die eine Kombination aus Indexfonds und [langfristigen Aktienantizipationspapieren](/leaps) (LEAPS) verwendet. Ein Anleger, der diese Strategie anwendet, muss eine Reihe von LEAP-Optionen verlängern, um sich an einer [langfristigen](/longterm) Bewegung in einem Index zu beteiligen. Die [Hebelwirkung](/leverage) der Optionen ermöglicht es dem Anleger, die Gewinne zu vergrößern, und kann langfristig zu einer Outperformance eines Index führen.

## Wie Indexrollen funktionieren

Anlegern steht eine breite Palette von Anlagestrategien zur Verfügung – sowohl aktive als auch passive Anlageansätze. Aktives Investieren verfolgt einen praktischen Ansatz, indem versucht wird, die durchschnittlichen [Renditen des Marktes zu übertreffen,](/return) indem kurzfristige Preisänderungen ausgenutzt werden. Passives Investieren hingegen verfolgt einen langfristigeren Ansatz, indem es sich in den Hintergrund rückt.

Im Wesentlichen verfolgen diejenigen, die passiv investieren, einen [Buy-and-Hold](/buyandhold) -Ansatz und verfolgen normalerweise einen Index, um zu versuchen, seine Renditen nachzuahmen. Index Rolling ist eine der passiven Anlagestrategien, die Anleger tendenziell anwenden.

Index Rolling ist eine langfristige passive Anlagestrategie, die den Anlegern zur Verfügung steht. Personen, die sich diesen Ansatz zunutze machen, können das gleiche [Engagement erzielen, als](/financial-exposure) würden sie in eine Standard- [Benchmark investieren](/benchmark),. jedoch oft mit weniger [Kapital](/capital) aufgrund des Engagements aus der Option „Long-Term Equity Anticipation Securities“ (LEAP).

Ein Anleger kann einen Roll Forward mit demselben [Ausübungspreis](/strikeprice) für den neuen wie für den alten durchführen oder einen neuen Ausübungspreis festlegen. Ein neuer Vertrag mit einem höheren Ausübungspreis als der ursprüngliche Optionsvertrag wird als Roll-up bezeichnet, während ein neuer Vertrag mit einem niedrigeren Ausübungspreis die Strategie zu einem Roll-down macht.

Im Laufe der Zeit hat die Position sehr ähnliche Auszahlungsmerkmale wie eine reguläre Indexierungsstrategie. Aber die Renditen sind tendenziell etwas höher. Das liegt an der Exposition gegenüber der Option in den frühen Phasen des Setups. [Die Volatilität](/volatility) bestimmt die Optionspreise, wobei eine geringere Volatilität zu niedrigeren Kosten für den Kauf von Optionen führt.

> Sie können eine höhere Rendite erzielen, indem Sie eine Index-Rolling-Strategie anstelle einer traditionellen Strategie anwenden, da die Option in den frühen Phasen des Setups exponiert ist.

>

## Besondere Überlegungen

Viele Buy-and-Hold-Anleger bevorzugen LEAPs. Dabei handelt es sich um öffentlich gehandelte [Wertpapiere](/security) mit Verfalldaten, die über ein Jahr hinausgehen. LEAPs ermöglichen es dem Käufer, den zugrunde liegenden [Vermögenswert](/asset) vor dem Ablaufdatum zu einem vorher festgelegten Preis zu kaufen oder zu verkaufen.

LEAP [-Call-Optionen](/calloption) können durch Call-Optionen mit späteren Ablaufdaten ersetzt werden, die es dem Anleger im Wesentlichen ermöglichen, seine Beteiligung an dem der Option zugrunde liegenden Vermögenswert auf unbestimmte Zeit zu verlängern. LEAP-Call-Optionen können eine höhere Kapitaleffizienz fördern, da sie weniger Kapital erfordern als der Kauf des Vermögenswerts selbst – in diesem Fall ein [börsengehandelter Aktienfonds](/etf) (ETF) –, indem sie eine LEAP-Strategie anwenden, die als Roll-Forward-Option bezeichnet wird.

## Nachteile einer Indexrolle

einem [Roll Forward wird](/rollforward) ein Vertrag über sein ursprüngliches Ablaufdatum hinaus verlängert. Dies geschieht in der Regel kurz vor Vertragsablauf. Anleger sollten die Kosten des Rollierens von Optionen berücksichtigen, bevor sie diese Strategie umsetzen, da das Rollieren erfordert, dass der Anleger eine Optionsposition glattstellt, möglicherweise mit Verlust, während er eine neue Position kauft.

Eine Index-Roll-Strategie verwendet LEAP-Call-Optionen auf einen bestimmten Aktienindex-ETF. LEAPS sind jedoch nicht für alle ETFs verfügbar. Dadurch wird das Spektrum der Anlageklassen für die Strategie eingegrenzt. Eine Liste der LEAPS-Optionen finden Sie bei der [Chicago Board Options Exchange](/cboe) (CBOE).

## Höhepunkte

- Anleger, die diesen Ansatz verwenden, erhalten das gleiche Engagement, als wenn sie in eine Standard-Benchmark investieren würden, jedoch mit weniger Kapital.

- Roll Forwards können den gleichen Ausübungspreis für den neuen wie für den alten verwenden, oder sie können einen neuen Ausübungspreis festlegen.

- Ein Index-Roll ist eine passive Indexanlagestrategie, die eine Kombination aus Indexfonds und Wertpapieren mit langfristiger Aktienerwartung verwendet.

- Eine Index-Roll-Position hat ähnliche Auszahlungsmerkmale wie eine reguläre Indexierungsstrategie, mit etwas höheren Renditen.
