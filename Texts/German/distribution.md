# Verteilung
## Was ist eine Verteilung?

Ausschüttung ist die Entnahme von Geld, das der Inhaber von einem Rentenkonto macht. Steuerzahler müssen den Ausschüttungsbetrag dem IRS melden, unabhängig davon, ob er steuerpflichtig ist oder nicht. Der Begriff bezieht sich auch auf Geld, das in Form einer Dividende an die Aktionäre ausgezahlt wird.

## Tiefere Definition

Steuerzahler sollten den Unterschied zwischen den drei Arten von Ausschüttungen kennen: normale, vorzeitige und erforderliche Mindestausschüttung. Eine normale Ausschüttung ist Geld, das vom Konto abgezogen wird, nachdem Sie mindestens 59,5 Jahre alt sind, während eine vorzeitige Ausschüttung Geld ist, das vor diesem Zeitpunkt abgehoben wird.

Sechs Monate nach Ihrem 70. Lebensjahr müssen Sie mit der Auszahlung der erforderlichen Mindestausschüttung beginnen.

Dies bedeutet nicht, dass Sie Ihr gesamtes Geld gleichzeitig abheben müssen. Manche Leute entscheiden sich für eine Pauschalausschüttung, die das Konto komplett leert. Andere entscheiden sich für regelmäßige Ausschüttungen und erhalten eine monatliche oder vierteljährliche Zahlung. Eine weitere Option besteht darin, das Geld auf dem Konto auf ein individuelles Rentenkonto oder IRA zu übertragen.

Einige Rentenkonten sind steuerbegünstigt, was bedeutet, dass Steuerzahler Steuern auf die Ausschüttungen zahlen, die sie von den Konten erhalten. Zusätzlich zur Steuer zahlen Personen, die sich für eine vorzeitige Ausschüttung entscheiden, auch eine Strafe an die IRS.

Steuerzahler melden die Ausschüttung auf dem Formular 1099-R für Ausschüttungen aus Renten, Annuitäten, Ruhestand, Gewinnbeteiligungsplänen, IRAs oder Versicherungsverträgen.

## Verteilungsbeispiel

Eine der Entscheidungen, die Sie treffen müssen, wenn Sie in den Ruhestand gehen, ist die Frage, wie Sie mit Ihrer Altersvorsorge umgehen. Wenn Sie Ihre Altersvorsorge auf mehrere Arten von Konten verteilen, können Sie die Ausschüttungen zu unterschiedlichen Zeitpunkten vornehmen.

Wenn Sie beispielsweise Ihr Geld in Ersparnissen und in einem vom Unternehmen geförderten Pensionsplan haben, können Sie sich entscheiden, die Ersparnisse zur Deckung der Lebenshaltungskosten zu verwenden und warten, bis Sie beginnen müssen, die erforderliche Mindestausschüttung aus der Rente und der Sozialversicherung zu beziehen.

## Höhepunkte

- Ausschüttungen von Investmentfonds bestehen aus Nettokapitalgewinnen aus dem gewinnbringenden Verkauf von Portfoliovermögen sowie aus Dividendenerträgen und Zinserträgen aus diesen Vermögenswerten.

- Eine Ausschüttung bezieht sich im Allgemeinen auf die [Auszahlung](/disbursement) von Vermögenswerten eines Fonds, Kontos oder einzelnen Wertpapiers an einen Anleger.

- Eine Kapitalausschüttung ist eine Barauszahlung, die auf einmal ausgezahlt wird, im Gegensatz zu einer Auszahlung in stetigen Raten.

- Steuerbegünstigte Rentenkonten beinhalten erforderliche Mindestausschüttungen – obligatorische Abhebungen, nachdem der Kontoinhaber ein bestimmtes Alter erreicht hat.

- Bei Wertpapieren wie Aktien oder Anleihen ist eine Ausschüttung eine Zins-, Kapital- oder Dividendenzahlung des Emittenten des Wertpapiers an die Anleger.
