# Füllen
## Was ist eine Füllung?

Ein Fill ist eine [ausgeführte](/execution) Order. Es ist die Handlung, eine Bestellung für ein Wertpapier oder eine Ware abzuschließen oder zu erfüllen. Die Ausführung von Aufträgen und das Melden von Erfüllungen ist ein grundlegender Akt bei der Transaktion von Aktien, Anleihen oder anderen Arten von Wertpapieren. Wenn zum Beispiel ein Trader eine Kauforder für eine Aktie zu 50 $ platziert und ein Verkäufer dem Preis zustimmt, findet der Verkauf statt und die Order wird ausgeführt. Der Preis von $50 ist der Ausführungs- oder Ausführungspreis.

## So funktionieren Füllungen

Es gibt verschiedene Möglichkeiten, wie Anleger versuchen können, einen Wertpapierauftrag auszuführen. Der erste und einfachste Ansatz ist die [Marktorder](/marketorder). In diesem Szenario beauftragt ein Anleger einen Makler, eine Anlage sofort zum besten verfügbaren [aktuellen Preis zu kaufen oder zu verkaufen](/currentprice).

Dies ist normalerweise eine Standardoption auf der Handelsplattform eines Anlegers und wird mit hoher Wahrscheinlichkeit ausgeführt. Eine Marktorder wird manchmal auch als uneingeschränkte Order bezeichnet und hat im Durchschnitt niedrige Provisionen, da es an Anforderungen, Logistik und Aufwand mangelt, um sie abzuschließen.

## Arten von Ausführungsaufträgen

Im Gegensatz dazu ist eine [Limit-Order](/limitorder) eine Anweisung, eine bestimmte Menge eines Finanzinstruments zu einem bestimmten Preis oder besser zu kaufen oder zu verkaufen. Eine Limit-Order wird möglicherweise nicht ausgeführt, wenn der vom Anleger festgelegte Preis während des Zeitraums, in dem die Order offen bleibt, nicht erreicht wird. Limit-Orders können in diesem Fall storniert werden.

Limit Orders garantieren, dass ein Anleger keine Kauf- oder Verkaufschance verpasst, wenn das Wertpapier sein gewünschtes Kursziel erreicht. [Kauf-Limit-Orders](/buy-limit-order) legen eine Obergrenze für den Preis fest, über dem ein Investor nicht zahlen wird, während Verkaufs-Limit-Orders ein Ziel für den günstigsten Preis festlegen, zu dem der Investor verkaufen wird.

Eine [Stop-Order](/stoporder) (auch Stop-Loss-Order genannt) ist eine Limit-Order, die zu einer Market-Order wird, sobald das Kursziel erreicht ist. Wenn beispielsweise eine [Kauf-Stopp-Order](/buystoporder) zu einem Preis von 20 USD (über dem aktuellen Marktpreis) eingegeben wird und die Aktie diesen Preis erreicht, werden bestimmte Aktien automatisch zum nächsten verfügbaren Marktpreis (z. B. 20,05 USD) gekauft.

Umgekehrt, wenn eine Stop-Verkaufsorder für 20 $ eingegeben wird und die Aktie fällt, wenn sie 20 $ erreicht, wird sie zu einer Verkaufsorder zum nächsten verfügbaren Marktpreis, der 19,98 $ betragen könnte.

## Besondere Überlegungen

Anlegeraufträge werden je nach Auftragstyp, der in das Maklersystem eingegeben wird, auf verschiedene Weise ausgeführt. Während die meisten Orders automatisch ausgeführt werden, wenn der Preis ausgelöst oder erreicht wird, können bestimmte Algorithmen manchmal festlegen, dass eine Order über einen festgelegten Zeitraum und/oder basierend auf dem Handelsvolumen eines Wertpapiers ausgeführt wird.

Wenn eine Order eine Bestimmung oder Bedingung wie einen Limitpreis hat, kann die Order nur teilweise ausgeführt werden. Eine teilweise Ausführung würde beispielsweise aus nur 200 Aktien resultieren, die zu einem Limitpreis von 53,00 $ ausgeführt werden, wenn die vollständige Order 1.000 Aktien umfasst.

Dies kann passieren, wenn zu diesem Limitpreis immer nur diese geringere Anzahl von Aktien geboten wird, während die Order noch besteht. Limitierte und zeitlich begrenzte Orders werden teilweise ausgeführt, während Marktorders fast immer vollständig ausgeführt werden.

## Höhepunkte

- Ein Fill ist das Ergebnis einer Auftragsausführung zum Kauf oder Verkauf von Wertpapieren auf dem Markt.

- Ein Fill meldet die Preise, Zeitstempel und das Volumen einer Order, die über einen Broker oder ein automatisiertes Handelssystem an den Markt gesendet wurde.

- Teilausführungen sind Aufträge, die aufgrund von Bedingungen, die an den Auftrag gestellt wurden, wie z. B. ein Limitpreis, nicht vollständig ausgeführt wurden.

## FAQ

### Wie lange dauert es, eine Marktorder auszuführen?

Bei aktiv gehandelten Aktien werden Marktaufträge fast sofort ausgeführt. Ein ungewöhnlich hohes Volumen kann den Handel jedoch verzögern.

### Werden Limit Orders sofort ausgeführt?

Limit-Orders werden nur ausgeführt, wenn der festgelegte Preis (oder besser) verfügbar ist. Daher werden Limit-Orders nur ausgeführt, wenn ein Wertpapier einen bestimmten Preis erreicht. Es gibt keine Garantie, dass die Bestellung sofort oder überhaupt ausgeführt wird.

### Warum wird meine vorbörsliche Bestellung nicht ausgeführt?

Vorbörsliche Aufträge werden während des vorbörslichen Handels (4.00 bis 9.30 Uhr EST) möglicherweise nicht ausgeführt, wenn nicht genügend Aktien vorhanden sind, um Ihren Auftrag auszuführen. Große Orders für Aktien mit geringem Volumen sind schwieriger auszuführen, insbesondere vorbörslich.
