# Gehaltskarte
## Was ist eine Lohnkarte?

Während Arbeitgeber früher nur per Scheck oder direkter Einzahlung auf das Bankkonto eines Mitarbeiters bezahlten, setzen einige auf neue Technologien und verwenden stattdessen Gehaltskarten. Das Gehalt des Mitarbeiters wird ähnlich einer Debit- oder Kreditkarte auf eine Plastikkarte geladen.

## Tiefere Definition

Eine Gehaltsabrechnungskarte funktioniert genauso wie eine Debit- oder Kreditkarte oder eine Debitkarte und wird von denselben Zahlungsabwicklern ausgestellt, die auch Kreditkarten wie MasterCard und Visa ausstellen.

Gehaltskarten werden überall dort akzeptiert, wo diese Kreditkarten akzeptiert werden. Sobald Ihr Arbeitgeber Ihnen eine Gehaltsabrechnungskarte ausstellt, wird Ihr Gehalt an jedem Zahltag darauf geladen, was bedeutet, dass Sie sich nicht für jede Zahlungsperiode eine neue Karte besorgen müssen.

Auf diese Weise ähnelt eine Lohnkarte einer Prepaid-Karte. Sie können Bargeld von Ihrer Karte erhalten, indem Sie sie in eine Bank bringen und dem Bankangestellten mitteilen, wie viel Geld Sie möchten, ähnlich wie einen Barvorschuss von einer Kreditkarte.

Arbeitgeber können nicht verlangen, dass Sie eine Gehaltsabrechnungskarte anstelle eines Schecks oder einer direkten Einzahlung verwenden. Wenn Sie die Karte verwenden, fallen normalerweise keine Gebühren an, abgesehen von möglichen Geldautomatenzuschlägen, wenn Sie Ihre Karte verwenden, um an einem dieser Automaten Bargeld zu erhalten. Dies ist eine Gebühr vom Besitzer des Geldautomaten, nicht von der Karte oder Ihrem Arbeitgeber.

Möglicherweise müssen Sie jedoch für den Ersatz Ihrer Karte bezahlen, wenn Sie sie verlieren oder sie beschädigt wird. Während für die grundlegende Nutzung der Karte in der Regel keine Gebühren anfallen, können Gebühren für bestimmte Aktionen anfallen, z. und in einigen Fällen sogar eine Gebühr zum Aufladen der Karte oder eine monatliche Gebühr für den Besitz der Karte.

##Beispiel für eine Gehaltsabrechnung

Wenn Sie sich für eine Lohnkarte entscheiden, können Sie diese direkt an der Kasse eines beliebigen Geschäfts verwenden, genau wie beim Einkaufen mit Ihrer Kreditkarte. Da einige Unternehmen für mehrere Transaktionen in einem Zahlungszeitraum Gebühren erheben, können Sie einfach das gesamte oder einen Teil des Bargelds von der Karte abheben, indem Sie es in eine Bank bringen oder zu einem Geldautomaten gehen und das Bargeld nach Bedarf verwenden.

## Höhepunkte

- Für Mitarbeiter umfassen die Vorteile von Lohnkarten die Möglichkeit, Rechnungen online zu bezahlen, online einzukaufen, automatische Rechnungszahlungen vorzunehmen und Bargeld an einem Geldautomaten zu erhalten.

- Zu den Nachteilen gehören die Möglichkeit monatlicher Wartungsgebühren, Gebühren für Geldautomaten außerhalb des Netzwerks und Gebühren für Kontostandsabfragen.

- Eine Gehaltsabrechnungskarte ist eine Prepaid-Karte, die einige Arbeitgeber verwenden, um die Löhne oder Gehälter ihrer Mitarbeiter an jedem Zahltag zu bezahlen.

- Gehaltskarten helfen Arbeitgebern, Geld zu sparen, da sie keine gedruckten Schecks ausstellen müssen, und ermöglichen es ihnen auch, Mitarbeitern, die kein Bankkonto haben, Karten anzubieten.
