# Festzins-Einlagenzertifikat (CD)
## Was ist ein festverzinsliches Einlagenzertifikat (CD)?

Ein festverzinsliches [Certificate of Deposit (CD)](/certificateofdeposit) ist ein Anlageinstrument, das über die gesamte Laufzeit einen festen Zinssatz aufweist. CDs bieten im Allgemeinen Laufzeiten in Schritten von drei Monaten bis zu einem Jahr an und wechseln dann zu zwei-, drei- und fünfjährigen Laufzeiten. Je länger die Laufzeit der Festzins-CD, desto höher der [Festzins](/fixedinterestrate). Sowohl große als auch kleine Privatkundenbanken bieten Festzins-CDs an.

## Eine festverzinsliche CD verstehen

Sparer, die mit ihren Anlagen konservativ sind, werden von festverzinslichen CDs angezogen, die ihnen bekannte Einkommensströme bis [zur Fälligkeit verschaffen](/maturity). Da CDs außerdem von der [Federal Deposit Insurance Corp. (FDIC)](/fdic) bis zu einer Höhe von 250.000 USD (pro Kontoinhaber und Emittent) garantiert sind, fühlen sich Anleger, die ihr Geld in diese Instrumente investieren, in Bezug auf die Sicherheit des Vermögenswerts sicher. Festverzinsliche CDs zahlen möglicherweise nicht so viel Zinsen wie andere festverzinsliche Wertpapiere, aber konservative Sparer akzeptieren den Kompromiss zwischen niedrigeren [Zinsen und einem](/interest) geringeren [Kapitalrisiko](/capitalrisk).

Normalerweise gibt es eine Strafe für [die vorzeitige Abhebung](/earlywithdrawal) von Geldern von einer CD, sodass ein CD-Inhaber das Geld fast immer im Instrument belässt, bis es fällig wird. Bei Fälligkeit können sie, abhängig von den finanziellen Bedürfnissen der Person, die ausgereifte CD in eine andere überführen. Die neue Festverzinsung dürfte sich jedoch von der gerade fällig gewordenen unterscheiden. Das allgemeine [Zinsumfeld](/interestrate) in der Wirtschaft bestimmt, wie festverzinsliche CDs von den emittierenden Banken festgelegt werden.

## CD mit festem Zinssatz vs. CD mit variablem Zinssatz

Eine CD mit variablem Zinssatz hat wie die CD mit festem Zinssatz eine feste Laufzeit, aber die Zinszahlungen können schwanken, da der Zinssatz des CDs an einen bestimmten Index gebunden ist, z. B. den [Leitzinsindex](/primerate),. den [Verbraucherpreisindex (VPI)](/consumerpriceindex) oder die Schatzwechsel Bewertung. Der Auszahlungsbetrag basiert auf einer prozentualen Differenz zwischen Indexanfangswert und Indexendwert. Ein Anleger in eine variabel verzinsliche CD ist weniger risikoscheu als ein Käufer einer festverzinslichen CD, und die Person kann durch das Einlegen von Geld in eine variabel verzinsliche CD ihre Überzeugung zum Ausdruck bringen, dass die Zinssätze in der Wirtschaft über die Laufzeit steigen werden der CD. Wenn sich das als richtig herausstellt, wird die CD mehr Zinsen generiert haben als eine Festzins-CD.

> CD-Besitzer müssen Bundessteuern auf die Zinsen zahlen, die sie zu ihrem [Steuerklassensatz verdienen](/taxbracket).

>

## Beispiel einer Festpreis-CD

Eine Festzins-CD, die eine Verzinsung von 5 % garantiert, wird von einer Bank angeboten. Die Laufzeit der CD beträgt sechs Monate. Tatiana investiert 1.000 Dollar in die CD. Nach sechs Monaten hat sie die Möglichkeit, die 1.050 $ abzuheben oder in eine andere CD umzuwandeln. Sie wählt die letztere Option und zieht am Ende eines Jahres bei Fälligkeit 1.100 $ ab. Sie wird Steuern auf ihre Einkünfte in Höhe von 100 Dollar schulden.

Tatianas Freund Marc hat ebenfalls 1.000 Dollar in dieselbe CD investiert, muss aber aufgrund eines familiären Notfalls nach drei Monaten den gesamten Betrag abheben. Die Vorfälligkeitsentschädigung beträgt drei Monatszinsen. Marc zahlt eine Strafe von 12,50 $ für eine vorzeitige Auszahlung.

## Höhepunkte

- Ein festverzinsliches Certificate of Deposit (CD) ist ein Anlageinstrument mit fester Verzinsung über die gesamte Laufzeit.

- Anders als bei einer CD mit variablem Zinssatz bleibt der Zinssatz bei einer CD mit festem Zinssatz konstant.

- Bei Fälligkeit einer CD kann der Inhaber entweder den gesamten Betrag abheben oder in eine andere CD umwandeln.

- In der Regel zahlen festverzinsliche CDs mit längerer Laufzeit höhere Zinssätze, und es gibt eine Strafe für die vorzeitige Abhebung von Geldern aus einer CD.

## FAQ

### Sollte ich mein Geld während einer Rezession in ein Einlagenzertifikat (CD) anlegen?

Während einer Rezession wollen die Menschen die sichersten Optionen für ihre Investitionen. Einlagenzertifikate (CDs) mit festem Zinssatz sind eine sichere Option, da sie von der Federal Deposit Insurance Corp. (FDIC) für bis zu 250.000 USD versichert sind. Denken Sie daran, dass Gelder, die in eine CD investiert werden, bis zum Zeitpunkt der Fälligkeit der CD nicht einfach ohne Abhebungsgebühren liquidiert werden können.

### Verdiene ich mit einer Festzins-CD oder einer Liquid-CD mehr?

Eine festverzinsliche CD bietet in der Regel eine viel höhere Rate als eine [liquide CD](/liquid-certificate-of-deposit). Festzins-CDs bieten auch einen höheren Zinssatz für Personen, die eine größere Ersteinzahlung investieren und ihr Geld über einen langen Zeitraum angelegt halten. Liquid CDs bieten einen niedrigeren Kurs, mit der flexiblen Option, Geld vorzeitig abzuheben.

### Gibt es eine Strafe, wenn ich mein Geld von einer festverzinslichen CD abhebe?

Ja, es gibt eine Strafe für das Abheben von Geld von einer festverzinslichen CD vor Fälligkeit. Wenn Sie befürchten, dass Sie Ihr Geld vor dem Fälligkeitsdatum benötigen könnten, sollten Sie in Erwägung ziehen, in eine [CD-Ladder zu investieren](/cd-ladder). Diese Anlagestrategie stellt sicher, dass Sie zu unterschiedlichen Zeitpunkten einen Teil Ihres Geldes zurückerhalten und hoffentlich vorzeitige Abhebungen vermeiden.
