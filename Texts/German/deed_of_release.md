# Freistellungsurkunde
## Was ist eine Freistellungsurkunde?

Eine Freigabeurkunde ist ein Rechtsdokument, das einen früheren Anspruch auf einen Vermögenswert aufhebt. Es dokumentiert die Entlassung aus einer verbindlichen Vereinbarung. Eine Freigabeurkunde kann enthalten sein, wenn ein Kreditgeber das Eigentum an einer [Immobilie](/realestate) nach Befriedigung der Hypothek auf den Hausbesitzer überträgt. Eine Freistellungsurkunde entbindet die Parteien buchstäblich von bisherigen Verpflichtungen.

## Wie funktionieren Freistellungsurkunden?

Ein guter Weg, um zu verstehen, wie eine Freigabeurkunde funktioniert, besteht darin, sie in einer Situation zu betrachten, die für viele Menschen üblich ist: Hauskauf. Die meisten Menschen kaufen ihre Häuser mit Hilfe einer Hypothek, die von einem Finanzinstitut bereitgestellt wird. Die Bank, die die Mittel für die Hypothek bereitstellt, verleiht das Geld nicht nur in gutem Glauben – sie nimmt einen Rechtsanspruch gegen das Haus als Sicherheit, bis das Darlehen zurückgezahlt ist.

Eine Hypothekenfreigabeurkunde wird dann erstellt, wenn der Kreditnehmer alle Hypothekenzahlungsbedingungen erfüllt oder eine vollständige Vorauszahlung leistet, um das Darlehen zu begleichen. Der Kreditgeber hält bis zu diesem Zeitpunkt das [Eigentum](/title) an der Immobilie und ist formell ein eingetragener Pfandgläubiger an der Immobilie, bis die vollständige und endgültige Zahlung erfolgt ist. Der Titel bietet eine gesicherte Sicherheit für die Darlehenszahlungen während der Laufzeit des Darlehens, wodurch das Ausfallrisiko für den Kreditgeber verringert wird.

Der Rechtsbeistand des kreditgebenden Instituts erstellt in der Regel die Freigabeurkunde, wenn das Darlehen befriedigt wurde. Es meldet, dass das Darlehen zu den erforderlichen Bedingungen vollständig zurückgezahlt wurde. Es besagt auch, dass das Pfandrecht entfernt wurde und das vollständige Eigentum auf den Hausbesitzer übertragen wurde.

Der Hausbesitzer besitzt den Vermögenswert frei und klar, nachdem ihr der Titel und die Freigabeurkunde zur Verfügung gestellt wurden. Sie unterliegen keinen Bedingungen oder Verpflichtungen des Kreditgebers mehr. Das Leihkonto wird geschlossen.

> Stellen Sie sicher, dass die Freigabeurkunde bei derselben Agentur eingetragen wird, die die ursprüngliche Hypothek eingetragen hat, damit Sie absolut sicher sein können, dass alle Pfandrechte entfernt sind.

>

## Arten von Freistellungsurkunden

Arbeitsverträge sind ein weiteres Szenario, in dem eine Freistellungsurkunde verwendet werden könnte. Das Dokument kann sowohl den Arbeitgeber als auch den Arbeitnehmer von allen Verpflichtungen befreien, die sie im Rahmen ihres Arbeitsvertrags hatten. In einigen Fällen kann eine Freistellungsurkunde einem Mitarbeiter eine bestimmte Zahlung gewähren. Dies kann im Falle einer Abfindung der Fall sein.

Die Entlassungsurkunde kann die Bedingungen der Abfindung enthalten, einschließlich Zahlung und Dauer der Zahlungen nach der Entlassung. Es kann auch vertrauliche Informationen identifizieren, die der Mitarbeiter nach der Kündigung nicht weitergeben kann, oder [Zurückhaltungsklauseln](/noncompete-agreement),. die einen ausscheidenden Mitarbeiter daran hindern, ein ähnliches Unternehmen zu gründen oder Kunden oder Klienten zu werben.

> ### Tipp

> Wenn Ihnen am Arbeitsplatz eine Freistellungsurkunde vorgelegt wird, lesen Sie diese unbedingt sorgfältig durch, um sicherzustellen, dass Sie verstehen, womit Sie einverstanden sind.

>

## Beispiel einer Freistellungsurkunde

Das häufigste Beispiel für eine Freigabeurkunde kann auftreten, wenn jemand ein Haus kauft. Wenn Sie eine Hypothek zum Kauf einer Immobilie aufnehmen, dient die Immobilie als Sicherheit für das Darlehen. In der Zwischenzeit hält der Kreditgeber das Pfandrecht auf das Haus als die Entität, der Sie Geld schulden. Wenn Sie Ihre Hypothekenbedingungen erfüllt und das Darlehen vollständig zurückgezahlt haben, kann eine Hypothekenfreigabeurkunde erstellt werden.

Ein weiteres Beispiel für eine Freistellungsurkunde sind [persönliche Bürgschaften](/personal-guarantee). Eine persönliche Bürgschaft bedeutet, dass Sie die persönliche Verantwortung für eine Schuld übernehmen, selbst wenn diese Schuld von einer Geschäftseinheit geschuldet wird, die Sie besitzen. Wenn Sie sich bereit erklären, eine persönliche Bürgschaft zu beenden, unabhängig davon, ob das Darlehen vollständig bezahlt wurde oder nicht, können Sie dies durch eine Freigabeurkunde tun.

> ### Hinweis

> Die Vereinbarung eines UCC-Pfandrechts (Uniform Commercial Code) für ein Geschäftsdarlehen schafft andere rechtliche und finanzielle Verpflichtungen als eine persönliche Bürgschaft.

>

Es ist möglich, online Mustererklärungen für Freigabeformulare zu finden, obwohl Sie möglicherweise eine Gebühr zahlen, um sie herunterzuladen. Es ist auch wichtig sicherzustellen, dass jedes von Ihnen verwendete Formular für die Freistellungsurkunde den gesetzlichen Richtlinien Ihres Bundesstaates für eine Freistellungsurkunde entspricht.

### Besondere Überlegungen

Wenden Sie sich an die [Federal Deposit Insurance Corp.](/fdic) (FDIC), wenn Sie keine Freigabeurkunde erhalten haben, weil Ihre kreditgebende Bank gescheitert ist und in die Zwangsverwaltung der FDIC gegangen ist. Die FDIC gibt an, dass sie das Problem normalerweise für Sie lösen kann.

Die Aufsichtsbehörde bietet ein [interaktives Suchtool](/lien) an, mit dem Sie herausfinden können, ob Ihre Bank von der Regierung durch Konkursverwaltung übernommen wurde.

## Höhepunkte

- Der Kreditgeber behält das Eigentum an Immobilien, bis die Bedingungen der Hypothek erfüllt sind, wenn üblicherweise eine Freigabeurkunde abgeschlossen wird.

- Freistellungserklärungen können auch zur Entlastung persönlicher Bürgschaftspflichten bei Geschäftskrediten verwendet werden.

- Eine Freistellungsurkunde kann einen Arbeitgeber und einen Arbeitnehmer auch von allen Verpflichtungen befreien, die sie im Rahmen ihres Arbeitsvertrags hatten, z. B. im Fall einer Abfindung.

- Eine Freistellungsurkunde entbindet die Vertragsparteien buchstäblich von früheren Verpflichtungen, wie z. B. Zahlungen im Rahmen einer Hypothek, weil das Darlehen abbezahlt wurde.
