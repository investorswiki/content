# Schattenbewertung
## Was ist eine Shadow-Bewertung?

Ein Schattenrating in einem inoffiziellen Rating, das einer Anleihe oder einem Emittenten von einer Kreditagentur erteilt wird [,](/credit-agency) jedoch ohne öffentliche Bekanntgabe des Ratings. Die Schattenbewertung kann zwei Zwecken dienen. Erstens kann es für ein Unternehmen hilfreich sein, das erwägt, bewertete Schuldtitel auf dem Markt zu emittieren, sich aber nicht sicher ist, wie es aufgenommen werden würde. Zweitens kann es als grobe Orientierung für Emissionen oder Emittenten dienen, die nicht formal von einer Kreditagentur bewertet wurden.

[Staatsanleihen](/sovereignbond) ) ohne offizielles Kreditrating interessiert sind, können das Schattenrating in ihren Anlageentscheidungsprozess integrieren.

## Schattenbewertungen verstehen

Ein Emittent, der noch nie offiziell von einer Kreditagentur bewertet wurde, kann sich an S&P, [Moody's](/moodys) oder eine andere Agentur wenden, um Schattenratings für sich selbst und eine [Schuldtitelemission](/debt-issue) zu erhalten, um das Wasser zu testen, oder um einen groben Hinweis darauf zu erhalten, wie sie auf den öffentlichen Märkten angesehen werden könnte. Die Kreditagentur würde ihre Schritte wie in einem formellen Ratingprozess durchlaufen, um ein Kreditrating zu vergeben. Dieses Schattenrating würde dem potenziellen Emittenten hilfreiche Informationen über die Preisgestaltung des Schuldinstruments (dh die ungefähren Zinskosten) auf dem aktuellen Markt und die potenzielle Nachfrage bei bekannten Gruppen von Schuldtitelinvestoren liefern.

Wenn das Schattenrating günstig ist, kann der Emittent ermutigt werden, mit dem Angebot fortzufahren und ein offizielles Rating der Kreditagentur einzuholen. Wenn das Schattenrating die Erwartungen des Emittenten nicht erfüllt, kann er mit einem formellen Rating und der Emission von Schuldtiteln auf dem Markt warten. Der Optionswert eines Schattenratings ist klar, und nachdem eine Kreditagentur den Emittenten oder die potenzielle Schuldverschreibung überprüft hat, hat der Emittent ein besseres Verständnis dafür, was erforderlich ist, um ein bestimmtes gewünschtes Rating zu erreichen, sei es, wie das Unternehmen Cashflows generiert oder wie man ein Debt Offering strukturiert.

## Andere Überlegungen

Wenn Anleger an gehandelten Schuldtiteln interessiert sind, die keine Kreditratings von einer Agentur tragen, können sie sich bei ihrer Bewertung der Kreditwürdigkeit der angestrebten Anlage an Schattenratings orientieren. Anleger können sich an eine Kreditagentur wenden, um eine Analyse durchzuführen, sie können die Analyse jedoch auch selbst unter Verwendung [vergleichbarer Analysemethoden durchführen](/comparables). Beispielsweise könnte ein nicht bewerteter Staat, der ähnliche Eigenschaften wie ein bewertetes Land besitzt, die gleiche Bewertungsbezeichnung wie dieser Comp erhalten. Das Schattenrating würde dann bei der Bestimmung durch einen Investor helfen, ob eine Schuldtitelemission eine akzeptable Investition ist.

## Höhepunkte

- Ein Schattenrating wird von einer Ratingagentur an einen Emittenten vergeben, jedoch auf inoffizielle Weise und ohne öffentliche Ankündigung.

- Das Schattenrating kann von einem Emittenten vorläufig eingeholt werden, während er darauf wartet, dass sein vollständiges Kreditrating gezeichnet wird.

- Das Schattenrating kann auch von einem Emittenten verwendet werden, um sich vor der Entscheidung zur Emission von Anleihen zu testen, ohne dass die Anleger reagieren. Bei einem ungünstigen Schattenrating kann das Unternehmen einfach auf die Anleiheemission verzichten.
