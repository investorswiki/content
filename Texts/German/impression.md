# Eindruck
## Was ist eine Impression?

Eine Impression ist eine Metrik, die verwendet wird, um die Anzahl der digitalen Aufrufe oder Interaktionen mit einem Inhalt, normalerweise einer Anzeige, einem digitalen Beitrag oder einer Webseite, zu quantifizieren. Impressionen werden auch als "Anzeigenansicht" bezeichnet. Sie werden in der [Online](/digital-marketing) -Werbung verwendet, die häufig pro Impression bezahlt wird. Das Zählen von Impressions ist für die Abrechnung und Bezahlung von Webwerbung im Suchmaschinenmarketing sowie für die Messung der Leistung von Social-Media-Kampagnen unerlässlich. Impressionen sind kein Maß dafür, ob auf eine Anzeige geklickt wurde, sondern wie oft sie angezeigt wurde oder potenzielle „Augäpfel“ darauf hatte, was zu einigen Diskussionen darüber führt, wie genau die Metrik ist.

## Wie Impressionen funktionieren

Im Großen und Ganzen entspricht eine Impression jedem Vorkommen einer Webseite, Anzeige oder eines Inhalts, der gefunden und geladen wird. Da es sowohl zum Messen als auch zum Verstehen zugänglich ist, ist es die bequemste und wirtschaftlichste Methode geworden, um festzustellen, ob eine Werbung gesehen wird oder nicht. Aber wie genau diese Zahl interpretiert wird, steht zur Debatte. Einige Online-Werbeexperten glauben, dass es keine exakte Methode zum Zählen von Impressions gibt, da eine Zählung beispielsweise dadurch verzerrt werden kann, dass eine einzelne Person dieselbe Anzeige bei mehreren Seitenaufrufen registriert. Es gibt mehrere weitere Möglichkeiten, die Gesamtzahl der Impressionen zu verzerren, was dazu führt, dass Werbetreibende jede Impressionszahl mit etwas Skepsis betrachten. Im Allgemeinen entscheiden die meisten Werbetreibenden und Publisher im Voraus, wie Impressions gezählt und verbucht werden. Werbetreibende können auf der Grundlage einer anderen Form der Berichterstattung, z. B. des Engagements (im Allgemeinen, wie ein Anzeigenbetrachter mit einer Anzeige interagiert), entscheiden, ob eine Kampagne erfolgreich ist oder nicht.

## Impression Accounting

[Cost-per-Mille (CPM)](/cpm) gemessen , wobei sich Mille auf 1.000 Impressionen (oder Cost-per-Tausend) bezieht. Eine Bannerwerbung kann einen CPM von 5 $ haben, was bedeutet, dass der Websitebesitzer jedes Mal 5 $ erhält, wenn eine Anzeige auf seiner Website 1.000 Mal angezeigt wird.

Der Eigentümer einer Website kann für jede Anzeigenimpression bezahlt werden. Andere Werbearrangements zahlen dem Websitebesitzer möglicherweise nur dann, wenn ein Besucher auf die Anzeige klickt oder auf die Anzeige klickt und einen Kauf tätigt. In der Regel zahlen Werbetreibende weniger für eine [Anzeigenkampagne, die ausschließlich auf](/marketing-campaign) [Impressionen](/marketing-campaign) basiert, und mehr für Kampagnen, die auf [Klicks](/clickthroughrates) und [Conversions basieren](/conversion-rate). Der Grund für diese unterschiedlichen Vergütungssätze liegt darin, dass eine Anzeige, die ihren Betrachter zu einer Aktion veranlasst, die zu einem Verkauf führt, für den Werbetreibenden wertvoller ist als eine, die dies nicht tut. Dennoch sind Eindrücke nützlich, wenn Sie PR-Kampagnen durchführen, die darauf abzielen, ein Image aufzubauen oder das Bewusstsein für ein Unternehmen oder Produkt zu schärfen.

Die genaue Art und Weise, wie Impressionen gezählt werden, ist etwas technisch. Ad-Server stellen ein kaum sichtbares Bild (oder „Pixel“) bereit, das auf jeder Publisher-Seite zu finden ist. Wenn eine Seite mit diesem Pixelbild geladen wird, wird ein Eindruck hinterlassen.

## Eindrucksbetrug

Mehrere Dinge können die Anzahl der Impressionen verzerren. Zum einen gehen Schätzungen davon aus, dass rund 60 % des gesamten Webverkehrs von Bots stammt. Impressionszählungen unterscheiden nicht zwischen einem menschlichen Anzeigenbetrachter oder einem Bot. Anzeigen können auch nicht geladen werden oder es kann die falsche Anzeige geladen werden. Solche Fehler können berücksichtigt werden oder nicht. Es gibt auch offenen Betrug, bei dem skrupellose Website-Entwickler verschiedene Methoden anwenden, um das System auszutricksen (eine Schätzung besagt, dass ein Viertel des Online-Werbemarkts betrügerisch ist). Nichtsdestotrotz bleiben Impressionen eine beliebte Methode, um das Engagement zu messen, sei es in der Werbung, in sozialen Medien oder bei der Analyse des Webverkehrs.

## Höhepunkte

- Das Zählen von Impressionen kann oft in einen Graubereich fallen, z. B. ob dies doppelte Aufrufe, Interaktionen von Bots umfasst oder ob Impressionen sogar eine effektive Methode sind, um den Erfolg einer digitalen Marketingkampagne zu messen.

- Impressionen werden verwendet, um die Anzahl der digitalen Aufrufe oder Interaktionen mit einem Inhalt, normalerweise einer Anzeige, einem digitalen Post oder einer Webseite, zu quantifizieren.

- Technisch gesehen liefern Adserver ein kaum sichtbares Bild (oder „Pixel“), das auf jeder Publisher-Seite zu finden ist. Wenn eine Seite mit diesem Pixelbild geladen wird, wird ein Eindruck hinterlassen.
