# Rufen Sie einen Anruf an
## Was ist ein Anruf bei einem Anruf?

Ein Call on a Call (CoC oder CaCall) ist eine Art [exotische Option](/exoticoption),. die dem Inhaber das Recht gibt, eine Call-Option auf denselben Basiswert zu kaufen. Im Wesentlichen ist ein Call auf eine Call-Option eine Option zum Kauf einer Option. Es wird zwei Ausübungspreise und zwei Ablaufdaten haben. Einer ist für die Compound-Call-Option und der andere für die zugrunde liegende [Vanilla](/vanillaoption) [-Option](/vanillaoption).

Der Call auf einen Call ist eine von vier Arten von [zusammengesetzten Optionen](/compoundoption),. die auch als Split-Fee-Optionen oder Optionen auf Optionen bekannt sind. Die anderen sind: [Call bei einem Put](/call-on-a-put) (CaPut); [einen Put anlegen](/put-on-a-put) ; und [ruf an](/put-on-a-call).

## So funktioniert ein Anruf bei einem Anruf

Call-on-Call-Optionen sind eine Art exotische Option mit angepassten Bedingungen, die an alternativen Börsen gehandelt werden. Bei einem Call auf eine Call-Option hat der Inhaber eine sekundäre Call-Option, die ihn berechtigt, aber nicht verpflichtet, eine Plain-Vanilla [-Call-Option](/calloption) mit festgelegten Bedingungen zu einem festgelegten Preis zu kaufen.

Ein Call-on-Call kann für einen Anleger vorteilhaft sein, wenn er zu einem optimalen Preis angeboten wird. Der Inhaber des Secondary Call hat das Recht, aber nicht die Pflicht, einen Plain Vanilla Call zu kaufen. Im Allgemeinen sind Call-on-Call-Optionen mit amerikanischer Ausübung strukturiert. Somit hat der Anleger bis zum [Ablaufdatum](/expirationdate) [Zeit](/expirationdate),. den Secondary Call auszuüben. Der Secondary Call und der Plain Vanilla Call können gleichzeitig oder getrennt ausgeübt werden.

Wenn der Investor den zusammengesetzten Call gleichzeitig ausübt, dann glaubt er, dass der Plain Vanilla Call im Geld und auf seinem profitabelsten Höhepunkt ist. Wenn beide zusammengesetzten Optionen ausgeübt werden, erhält der Anleger die zugrunde liegende Kaufoption, die dann sofort ausgeübt wird, um das zugrunde liegende [Wertpapier zu erhalten](/security). Wenn der Inhaber der zusammengesetzten Option sich dafür entscheidet, nur das erste Teil des Kontrakts auszuführen, erhält er die Plain-Vanilla-Call-Option zum angegebenen Verfall- und Ausübungspreis für die zukünftige Inkraftsetzung.

In einigen Fällen kann ein Anleger mit einem Call auf eine Call-Option sein Engagement in einem zugrunde liegenden Vermögenswert zu geringen Kosten erweitern. Viele Optionen ermöglichen eine Roll-Funktion, die ein erweitertes Engagement ermöglicht, aber ein Call auf eine Call-Option kann dies zu geringeren Kosten ermöglichen. Wenn der Inhaber eines Compound-Call beispielsweise den Preis der zugrunde liegenden Call-Option für seine Anlagepläne vorteilhaft findet, kann er den ersten Teil der Option ausüben, um die Plain-Vanilla-Option mit einem zukünftigen Verfallsdatum zu erhalten.

## Preise für einen Anruf für einen Anruf

Vor Ablauf hängt der Wert der Plain-Vanilla-Option vom Wert des Vermögenswerts ab, den die zugrunde liegende Option darstellt. Auf dem Optionsmarkt können Anleger verschiedene Methoden anwenden, um den Wert ihrer Option zu berechnen. Das [Merton-Modell](/mertonmodel) ist eine Methode, die für zusammengesetzte [Optionen eingeführt wurde](/option).

Ein zusammengesetzter Call auf eine Call-Option ist eine komplexe Option, die mit höheren Kosten (Prämien) verbunden ist als Vanilla-Optionen. Der Anleger muss außerdem Transaktionskosten zahlen, die bei der Ausführung beider Optionen berücksichtigt werden. Es ist wichtig, die Kosten der Compound-Option zu berücksichtigen, da sie die Rentabilität der Investition insgesamt verringern können.

## Andere Verbindungsoptionen

Die anderen drei Arten von zusammengesetzten Optionen sind:

- **Call auf Put**: Dies ist eine Call-Option auf eine zugrunde liegende Put-Option. Der Eigentümer, der die Call-Option ausübt, erhält eine Put-Option.

- **Call on a Call**: Bei dieser Option kauft der Anleger eine weitere Call-Option mit individuellen Konditionen. Diese Bestimmungen beinhalten das Recht, eine Plain-Vanilla-Call-Option auf ein zugrunde liegendes Wertpapier zu kaufen.

- **Put on a Call**: Der Anleger muss die zugrunde liegende Call-Option an den Verkäufer liefern und eine Prämie basierend auf dem Ausübungspreis der darüber liegenden Put-Option kassieren.

Diese Optionen werden auch als Split-Fee-Optionen bezeichnet.

## Anwendung in der realen Welt

Während Spekulationen an den Finanzmärkten immer einen großen Teil der Aktivität von Compound-Optionen ausmachen, könnten sie für Geschäftsunternehmen nützlich sein, wenn sie ein großes Projekt planen oder Gebote abgeben. In einigen Fällen müssen sie die Finanzierung oder Lieferungen sicherstellen, bevor sie das Projekt tatsächlich starten oder gewinnen können. Wenn sie das Projekt nicht bauen oder gewinnen, könnten sie mit Finanzierungen zurückbleiben, die sie nicht benötigen. In diesem Fall bieten Compound-Optionen eine Versicherungspolice.

Dasselbe gilt auch für Kreditgeber, da sie versuchen, ihr Risiko abzusichern, falls sie sich verpflichten, das Geld bereitzustellen, das Unternehmen für ihre Projekte benötigen, und diese Unternehmen ihre Geschäfte nicht gewinnen.

## Höhepunkte

- Ein Call auf eine Call-Option ist eine exotische Option, die dem Inhaber das Recht gibt, vor Ablauf des Kontrakts eine Plain-Vanilla-Option zu kaufen.

- Ein Call auf eine Call-Option kann von einem Investor genutzt werden, um sein Engagement in einem zugrunde liegenden Vermögenswert kostengünstig zu erweitern, und kann in der Immobilienentwicklung zur Sicherung von Eigentumsrechten verwendet werden, ohne sich verpflichten zu müssen.

- Ein Call auf einen Call ist also eine Art zusammengesetzte Option auf eine Option, die einen Call beinhaltet, um einen Call zu kaufen.
