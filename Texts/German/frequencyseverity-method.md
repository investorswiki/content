# Häufigkeits-Schwere-Methode
## Was ist die Häufigkeits-Schwere-Methode?

Die Frequency-Severity-Methode ist eine versicherungsmathematische Methode zur Bestimmung der erwarteten Anzahl von Schadensfällen, die ein Versicherer während eines bestimmten Zeitraums erhalten wird, und der Höhe der durchschnittlichen Kosten eines Versicherungsfalls.

Die Häufigkeits-Schwere-Methode verwendet historische Daten, um die durchschnittliche Anzahl von Ansprüchen und die durchschnittlichen Kosten für jeden Anspruch zu schätzen. Das Verfahren multipliziert die durchschnittliche Anzahl von Ansprüchen mit den durchschnittlichen Kosten eines Anspruchs.

## Verständnis der Häufigkeits-Schwere-Methode

Bei der Frequency-Severity-Methode bezieht sich die Häufigkeit auf die Anzahl der Schäden, die ein Versicherer in einem bestimmten Zeitraum erwartet. Wenn die Häufigkeit hoch ist, bedeutet dies, dass mit einer großen Anzahl von Schadensfällen zu rechnen [ist](/incidence-rate).

Der Schweregrad bezieht sich auf die Kosten eines Anspruchs. Ein Anspruch mit hohem Schweregrad ist teurer als ein durchschnittlicher Anspruch, und ein Anspruch mit niedrigem Schweregrad ist weniger teuer als ein durchschnittlicher Anspruch. Die durchschnittlichen Schadenkosten werden auf der Grundlage historischer Daten geschätzt.

Betrachten Sie zum Beispiel einen potenziellen Hauskäufer, der den Kauf eines Strandhauses in Miami in Betracht zieht. Dieser Teil der Küste Floridas wird durchschnittlich von einem Hurrikan pro Jahr heimgesucht. Da das Potenzial für eine vollständige Zerstörung so hoch und so häufig ist, würde die Häufigkeits-Schwere-Methode darauf hinweisen, dass eine Versicherungsgesellschaft es vermeiden sollte, eine Police für dieses Strandhaus abzuschließen.

## Häufigkeits-Schwere-Methode und andere Risikomodelle

Versicherer verwenden ausgefeilte Modelle, um die Wahrscheinlichkeit zu bestimmen, dass sie einen Schaden bezahlen müssen. Im Idealfall würde der Versicherer es vorziehen, [Prämien](/premium) für [das Abschließen](/underwriting) neuer Versicherungspolicen zu erhalten, ohne jemals einen Schaden auszahlen zu müssen, aber dies ist ein sehr unwahrscheinliches Szenario.

Stattdessen entwickeln die Versicherer Schätzungen darüber, wie viele Schadensfälle sie erwarten können und wie teuer die Schadensfälle sein werden, basierend auf den Arten von Policen, die sie Versicherungsnehmern anbieten. Die Frequency-Severity-Methode ist eine Möglichkeit, die Versicherer nutzen, um Modelle zu entwickeln.

Die Häufigkeit bezieht sich auf die Anzahl der Schäden, die ein Versicherer erwartet. Hohe Frequenz bedeutet, dass mit einer großen Anzahl von Schadensfällen zu rechnen ist.

Die durchschnittlichen Schadenkosten können auf der Grundlage [historischer Kostenzahlen geschätzt werden](/historical-cost). Da die Frequency-Severity-Methode vergangene Jahre bei der Bestimmung der Durchschnittskosten für zukünftige Jahre berücksichtigt, wird sie weniger von volatileren jüngsten Zeiträumen beeinflusst. Damit ist sie nicht auf [Schadenentwicklungsfaktoren](/loss-development) aus jüngeren Jahren angewiesen.

Dies bedeutet jedoch auch, dass sich das Verfahren langsamer an steigende [Volatilitäten anpasst](/volatility). Beispielsweise wird sich ein Versicherer, der eine [Hochwasserversicherung anbietet](/flood-insurance),. langsamer an eine Zunahme der Schwere oder Häufigkeit von Hochwasserschäden anpassen, die durch den jüngsten Anstieg der Wasserstände verursacht wurden.

## Höhepunkte

- Schweregrad bezieht sich auf die Kosten eines Anspruchs – ein Anspruch mit hohem Schweregrad ist teurer als ein durchschnittlicher Anspruch, und ein Anspruch mit niedrigem Schweregrad ist weniger teuer.

- Die Häufigkeit bezieht sich auf die Anzahl der Schäden, die ein Versicherer in einem bestimmten Zeitraum erwartet.

- Die Frequency-Severity-Methode ist eine Möglichkeit, die Versicherer nutzen, um Modelle zu entwickeln.

- Die Frequency-Severity-Methode ist eine versicherungsmathematische Methode zur Bestimmung der erwarteten Anzahl von Schadensfällen, die ein Versicherer während eines Zeitraums erhalten wird, sowie der durchschnittlichen Schadenshöhe.
