# Shareholder Value Transfer (SVT)
## Was ist Shareholder Value Transfer (SVT)?

Der Shareholder Value Transfer (SVT) ist eine Kennzahl, die den Aktionären als Richtlinie dienen soll, wie viel Eigenkapitalvergütung Mitarbeitern und Führungskräften [börsennotierter Unternehmen zugesprochen werden sollte](/publiccompany). Shareholder Value Transfer wird als Gesamtwert der Aktienzuteilungen geteilt durch die Marktkapitalisierung des Unternehmens berechnet. Dies ergibt einen Prozentsatz, auf den bestehende Aktionäre im Rahmen eines bestimmten Aktienvergütungsplans verwässert würden.

## Shareholder Value Transfer (SVT) verstehen

Die Shareholder Value Transfer-Metrik wurde von Institutional Shareholder Services (ISS) entwickelt, einem Forschungsunternehmen, das Aktionäre berät, wie sie über Aktionärsanträge abstimmen können. ISS berechnet den Shareholder-Value-Transfer für die Top-Unternehmen in jeder Branche und entscheidet über einen maximalen „Cap“-Betrag, den jedes Unternehmen an Shareholder-Value-Transfer zahlen muss, um gut abzuschneiden. ISS rät den Anlegern dann in der Regel, gegen jeden Vorschlag für eine Aktienvergütung zu stimmen, der die Obergrenze für die Übertragung des Aktionärswerts überschreiten würde.

Laut Institutional Shareholder Services bezieht sich Shareholder Value Transfer auf eine Schätzung des Werts, den das Unternehmen seinen Mitarbeitern und Direktoren über bestimmte aktienbasierte Vergütungsprogramme übertragen wird, gemessen zu einem bestimmten Datum auf der Grundlage eines Standardsatzes von Inputs. Das proprietäre Vergütungsmodell von ISS berechnet für jedes Unternehmen – basierend auf seiner Marktkapitalisierung, Branche und relevanten Leistungskennzahlen im Vergleich zu Mitbewerbern – eine Benchmark für die Aktionärswertübertragung, die zur Bewertung der Aktionärswertübertragung des Unternehmens verwendet wird.

Um eine Schätzung der Shareholder Value Transfer zu finden, verwenden die Berechnungen von Institutional Shareholder Service eine Kombination aus Daten von Drittanbietern für ein Optionspreismodell sowie unternehmensspezifische Daten (einschließlich ausstehender Zuteilungen und verbleibender Aktien für zukünftige Zuteilungen), die im Allgemeinen im jährlichen [10- K-](/10-k) oder Proxy-Einreichung.

## Höhepunkte

- Um eine SVT-Schätzung zu finden, verwenden die Berechnungen von ISS eine Kombination aus Daten von Drittanbietern und Unternehmensdaten, die in Steuerformularen angegeben sind.

- Shareholder Value Transfer wird als Gesamtwert der Aktienzuteilungen geteilt durch die Marktkapitalisierung des Unternehmens berechnet.

- Shareholder Value Transfer (SVT) ist eine Kennzahl, die Aktionären dabei helfen soll, wie viel Eigenkapitalvergütung Mitarbeitern und Führungskräften börsennotierter Unternehmen zugesprochen werden sollte.

- Die Kennzahl wurde von Institutional Shareholder Services (ISS) erstellt, einem Forschungsunternehmen, das Aktionäre bei der Abstimmung über Aktionärsanträge berät.

- ISS berechnet eine „Obergrenze“ für SVT pro Branche und leitet Unternehmen an, wie ihre Vorschläge zur Eigenkapitalvergütung aussehen sollten.
