# Setzen Sie auf einen Put
## Was ist ein Put auf einen Put?

Ein Put auf einen Put ist ein [Optionskontrakt](/optionscontract),. der dem Inhaber das Recht gibt, einen zugrunde liegenden Put-Optionskontrakt zu verkaufen. Der Put auf einen Put-Trade ist eine von vier Arten von [Compound-Optionen](/compoundoption).

Im Wesentlichen ist ein Put auf eine Put-Option eine Option zum Verkauf einer Option. Der zugrunde liegende Vermögenswert des Put auf eine Put-Option ist die ursprüngliche Option. Put-on-Put-Optionen sind an europäischen Börsen häufiger als in den Vereinigten Staaten.

## Setzen auf einen Put erklärt

Der Käufer einer Put-Option ist ein Händler, der erwartet, dass der Wert, auf dem die Option basiert, im Preis fallen wird. Der Händler erwirbt eine Verkaufsoption, normalerweise für 100 Aktien, die es ermöglicht, die Aktien zu einem bestimmten Preis (dem [Ausübungspreis](/strikeprice) ) bis zu einem bestimmten [Ablaufdatum zu verkaufen](/expirationdate). Wenn der Händler richtig liegt und der Vermögenswert fällt, ist die Option [im Geld](/inthemoney) und kann ausgeübt werden, wobei der Händler von der Differenz zwischen dem Optionspreis und dem Marktpreis profitiert.

Ein Put auf eine Put-Option beinhaltet zwei Put-Optionen, eine über der anderen. Ein Put auf einen Put hat zwei Ausübungspreise und zwei Verfalldaten. Einer ist für die Compound-Put-Option und der andere für die zugrunde liegende [Vanilla-Option](/vanillaoption).

Beachten Sie, dass zusammengesetzte Optionen in Europa häufiger sind und [europäische Optionen](/europeanoption) nur am Verfallsdatum ausgeübt werden können. Eine [amerikanische Option](/americanoption) kann am oder jederzeit vor dem Ablaufdatum ausgeübt werden.

Da eine der Variablen, die die Kosten einer Option bestimmen, der Preis des Basiswerts ist, sind die Kosten eines Puts auf eine Put-Option im Allgemeinen niedriger als die Kosten eines Vanilla Puts auf den entsprechenden Vermögenswert. Daher kann es dem Optionshändler eine gewisse Hebelwirkung verleihen.

## Wann man einen Put auf einen Put verwendet

[Hebelwirkung](/leverage) einsetzen möchte . Der Trader wird auch in Bezug auf den zugrunde liegenden Vermögenswert moderat optimistisch sein. Der Wert eines Put auf einen Put ändert sich direkt proportional zum Preis des Basiswerts. Das bedeutet, dass der Wert steigt, wenn der Vermögenspreis steigt, und sinkt, wenn der Vermögenspreis sinkt.

## Andere Verbindungsoptionen

Die anderen drei Arten von zusammengesetzten Optionen sind:

- **Call auf Put**: Dies ist eine Call-Option auf eine zugrunde liegende Put-Option. Der Eigentümer, der die Call-Option ausübt, erhält eine Put-Option.

- **Call on a Call**: Bei dieser Option kauft der Anleger eine weitere Call-Option mit individuellen Konditionen. Diese Bestimmungen beinhalten das Recht, eine Plain-Vanilla-Call-Option auf ein zugrunde liegendes Wertpapier zu kaufen.

- **Put on a Call**: Der Anleger muss die zugrunde liegende Call-Option an den Verkäufer liefern und eine Prämie basierend auf dem Ausübungspreis der darüber liegenden Put-Option kassieren.

Diese Optionen werden auch als Split-Fee-Optionen bezeichnet.

## Höhepunkte

- Ein Put auf einen Put ist ein Optionskontrakt, der seinem Käufer das Recht gibt, einen zugrunde liegenden Optionskontrakt zu verkaufen.

- Ein Put auf einen Put ist im Wesentlichen eine Option zum Verkauf einer Option.

- Der Basiswert des Puts auf eine Put-Option ist eine Vanilla-Put-Option.
