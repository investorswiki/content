# Reaktion
## Was ist eine Reaktion?

Eine Reaktion im Zusammenhang mit den Finanzmärkten ist eine plötzliche, aber normalerweise kurzlebige Aufwärts- oder Abwärtsbewegung des Aktienkurses. Technische Analysten beschreiben oft eine Abwärtsbewegung des Kurses einer Aktie nach einer Phase der Aufwärtsbewegung als Reaktion.

Reaktionen sind typischerweise Reaktionen auf Nachrichten oder Daten, die für das Unternehmen, das die Aktie ausgegeben hat, oder die Branche, in der es tätig ist, relevant sind. Die Kursänderung ist in der Regel gering.

Eine Reaktion ähnelt einer [Korrektur](/correction) oder [Umkehrung , hat](/reversal) aber nicht die gleiche Intensität oder Dauer.

## Eine Reaktion verstehen

Reaktionen werden im Allgemeinen als positives und normales Ereignis in einem gesunden Markt angesehen. Unendliche Preiserhöhungen können zu einem noch größeren Preisverfall führen, wenn ein Unternehmen die Erwartungen nicht erfüllt oder auf andere Hindernisse stößt.

Tatsächlich verhindert eine gelegentliche Reaktion ein Ereignis wie einen Run auf eine Aktie oder einen großvolumigen Ausverkauf zu einem späteren Zeitpunkt.

Eine [Überreaktion](/overreaction) ist eine extreme Reaktion auf neue Informationen. Im Finanzwesen und beim Investieren ist es eine emotionale Reaktion auf ein Wertpapier wie eine Aktie oder eine andere Investition, die entweder von Gier oder Angst geleitet wird. Anleger, die auf Nachrichten überreagieren, führen dazu, dass das Wertpapier entweder überkauft oder überverkauft wird, bis es wieder seinen [inneren Wert erreicht](/intrinsicvalue).

## Gute Nachrichten und schlechte Nachrichten

Eine Abwärtsreaktion ist oft eine Reaktion auf negative Nachrichten. Diese Nachrichten können ein schlechter [Gewinnbericht](/earnings),. eine kritische Geschichte über das Unternehmen, wirtschaftliche und politische Unsicherheit und jedes unerwartete und unglückliche Ereignis sein. Einige oder alle davon können Verkaufsdruck und einen Rückgang des Aktienkurses verursachen.

Positive Nachrichten führen normalerweise zu Kurssteigerungen, wenn auch nur kurz. Die Ankündigung eines neuen Produkts, eine Akquisition oder die Veröffentlichung eines optimistischen Wirtschaftsindikators kann eine positive Reaktion auf den Aktienkurs auslösen.

Diese Ereignisse können wirklich kurzlebig sein. Ein Hurrikan, der sich der Landung nähert, kann zu einem Rückgang der Aktien von Versorgungsunternehmen und Versicherungen führen. Stunden später kann eine Ankündigung, dass der Sturm abgezogen ist, die Preise umkehren.

> Eine Reaktion kann einen Einstiegspunkt für einen Trader bieten, der eine Position eingehen möchte, wenn andere technische Indikatoren bullisch bleiben.

## Reaktionen vs. Umkehrungen

Reaktionen können mit einem Achselzucken abgetan werden, insbesondere von Anlegern, die langfristig dabei sind. Umkehrungen sind schwerwiegender und können langanhaltend sein. Händler müssen in der Lage sein, zwischen den beiden zu unterscheiden.

Die meisten Umkehrungen beinhalten eine Änderung der zugrunde liegenden Fundamentaldaten eines Wertpapiers, die den Markt dazu zwingt, seinen Wert neu zu bewerten. Wenn ein Unternehmen ein katastrophales Quartal meldet, werden die Anleger den Kapitalwert der Aktie neu berechnen und entsprechend handeln. Oder die Veröffentlichung eines bahnbrechenden neuen Produkts durch einen Konkurrenten kann den Wert einer Aktie langfristig schädigen.

Ereignisse, die sich als bedeutsam herausstellen werden, erscheinen zunächst wie eine Reaktion. Aber wenn sie sich über mehrere Sitzungen erstrecken, könnte eine echte Umkehrung im Gange sein.

Aus diesem Grund verwenden Händler gleitende Durchschnitte, Trendlinien und Handelsbänder, um den Punkt zu markieren, an dem die Reaktion Gefahr läuft, in den Umkehrbereich einzutreten.

## Höhepunkte

- Eine Reaktion kann nur wenige Sitzungen dauern, bevor sie zum vorherrschenden Trend zurückkehrt.

- Eine echte Trendwende oder Preiskorrektur ist tiefer und länger andauernd als eine kurzlebige und gedämpfte Reaktion.

- Eine Reaktion ist eine kurze Preisbewegung, häufig als Reaktion auf Nachrichten oder die Veröffentlichung neuer Daten.
