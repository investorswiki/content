# Kontoanfrage
## Was ist eine Kontoabfrage?

Eine Kontoabfrage ist eine Überprüfung jeder Art von Finanzkonto, sei es ein Depotkonto oder ein Kreditkonto. Die Abfrage kann sich auf frühere Aufzeichnungen, Zahlungen und andere spezifische Transaktionen oder auf andere Einträge beziehen, die sich auf das Konto beziehen.

## Kontoabfrage verstehen

Die meisten Finanzinstitute haben eine formelle Abteilung, die sich mit Kontoanfragen befasst.

Eine Kontoabfrage könnte auf Anfrage einer Bank, eines [Kreditgebers](/lender) oder eines anderen Finanzinstituts erfolgen, das eine Kopie eines Kontoverlaufs benötigt, wenn eine Einzelperson, ein Unternehmen oder eine andere Einheit einen Kredit oder ein [Darlehen beantragt](/loan). Der Begriff wird oft verwendet, um eine [Kreditanfrage zu bezeichnen, dh](/credit-inquiry) wenn eine Anfrage an oder von einer Kreditagentur zu einem bestimmten Verbraucher gestellt wird. Auskunfteien können Kontoauskünfte an Kreditkartenunternehmen oder andere Kreditgeber erteilen, um zu beurteilen, ob Einzelpersonen die Rechnungen, für die sie verantwortlich sind, auf dem aktuellen Stand halten.

## Der Zweck einer Kontoabfrage

Eine Kontoauskunft wird typischerweise eingeleitet, wenn eine Person versucht, neue Schulden aufzunehmen, insbesondere in Verbindung mit einer größeren Anschaffung wie dem Erwerb von Immobilien. Bevor der Antragsteller für die neuen Schulden zugelassen wird, möchte der Kreditgeber seine Erfolgsbilanz bei der ordnungsgemäßen Begleichung seiner Schulden sehen. Dieser Schritt ist entscheidend für die Beurteilung der Gesamtkreditwürdigkeit eines potenziellen Kreditnehmers sowie für die Strukturierung der Zinssätze, die ihnen für das Darlehen angeboten werden können.

Ein Kontoinhaber kann in seinem eigenen Namen eine Kontoabfrage einleiten, insbesondere wenn fragwürdige Aktivitäten vermutet werden – wie z. Indem Sie verlangen, dass die Bank oder der Gläubiger eine Untersuchung solcher Transaktionen einleitet, könnte die Untersuchung ein Anfang zur Identifizierung von Sicherheitsverletzungen oder Betrugsfällen sein.

Es können auch Kontoabfragen durchgeführt werden, um sicherzustellen, dass Zahlungen verrechnet oder zumindest rechtzeitig geliefert wurden.

## Besondere Überlegungen für eine Kontoanfrage

[Kreditwürdigkeit](/creditrating) des Verbrauchers auswirken . Wenn eine Person in kurzer Zeit mehrere Kreditkarten beantragt, löst jede Beantragung normalerweise eine separate Kontoabfrage in der Zahlungshistorie aus. Wenn diese Anfragen an Kreditauskunfteien gemeldet werden, kann dies als Grund angesehen werden, die Kreditwürdigkeit des Verbrauchers zumindest vorübergehend zu verringern, da das Verhalten als Person interpretiert werden könnte, die Geld braucht oder kurz vor einem Kaufrausch steht viele Schulden machen.

> Auskunfteien lassen Kreditsuchende etwas locker: Sie lassen nicht zu, dass die Anfragen mehrerer Kreditgeber die Kreditwürdigkeit beeinträchtigen, vorausgesetzt, die Anfragen erfolgen innerhalb von 45 Tagen.

>

Es gibt jedoch eine Ausnahme für hypothekenbezogene Anfragen. Kreditagenturen verstehen, dass die Leute normalerweise nach dem besten Angebot suchen, mehrere Kreditgeber besuchen und dass alle verschiedenen Anfragen für denselben einzigen Kredit gelten (es sei denn, eine Familie plant zufällig, drei verschiedene 500.000-Dollar-Häuser zu kaufen). Sie zählen also nicht eine Flut von Anfragen gegen Sie, besonders wenn sie alle innerhalb von ein paar Monaten sind.

## Höhepunkte

- Exzessive Kontoabfragen in kurzer Zeit können der Kreditwürdigkeit einer Person schaden.

- Eine Kontoabfrage kann auch von einer Einzelperson an eine Institution gestellt werden, die Einzelheiten in ihrem eigenen Namen anfordert.

- Eine Überprüfung der Aktivität und Leistung einer beliebigen Art von Finanzkonto wird als Kontoabfrage bezeichnet.

- Eine Kontoauskunft wird typischerweise eingeleitet, wenn eine Person versucht, neue Schulden aufzunehmen.

- Eine Kontoauskunft erfolgt häufig auf Anfrage einer Bank, eines Kreditgebers oder eines anderen Finanzinstituts, wie z. B. einer Kreditauskunftei (auch als Kreditauskunft bekannt).
