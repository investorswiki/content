# Transaktions-ID (TXID)
Eine Transaktions-ID (TXID) oder ein Transaktions-Hash ist eine eindeutige Zeichenkette, die jeder verifizierten und der Blockchain hinzugefügten Transaktion zugewiesen wird. Mit anderen Worten, eine TXID ist eine Identifikationsnummer, die jede Transaktion in der Blockchain kennzeichnet.

Wenn Sie eine Transaktion durchführen und diese zur Blockchain hinzugefügt wird, erhält sie eine Transaktions-ID als eindeutige Kennung. Je nach verwendeter Blockchain können Sie diese dann in einem Block-Explorer finden, indem Sie nach der Transaktions-ID suchen. Dies kann nützlich sein, um grundlegende Transaktionsinformationen zu überprüfen, wie z. B. den gesendeten Betrag, das Überweisungsdatum, die Sende-/Empfangsadresse und die Anzahl der Netzwerkbestätigungen, die die Transaktion erhalten hat.

Dies ist zum Beispiel die TXID der allerersten Bitcoin-Transaktion, die von Satoshi Nakamoto an Hal Finney gesendet wurde:

**f4184fc596403b9d638783cf57adfe4c75c605f6356fbc91338530e9831e9e16**

Dies ist die TXID der berühmten [Bitcoin Pizza -](/bitcoin-pizza) Transaktion:

**cca7507897abc89628f450e8b1e0c6fca4ec3f7b34cccf55f3f531c659ff4d79**

Sie können diese Transaktionen selbst nachschlagen, indem Sie die TXIDs in einen Bitcoin-Block-Explorer wie blockchain.com, blockchair.com [oder](/bitcoin) viele andere kopieren.

Wenn Sie Kryptowährung von einer zentralen Börse (CEX) wie Binance abheben, zeigt Ihnen die Plattform die TXID Ihrer Abhebungstransaktion. Auf Binance finden Sie dies auf Ihrer Transaktionsverlaufsseite. Wenn Sie für eine bestimmte Auszahlung auf eine TXID klicken, gelangen Sie zur entsprechenden Transaktion im Block-Explorer dieser Blockchain.

Wenn Sie Geld an die falsche Adresse (oder Blockchain) senden, ist die TXID in Ihrem Auszahlungsverlauf unerlässlich, um zu prüfen, ob Sie Ihr Geld zurückerhalten können. Trotzdem ist es keine Garantie, und es ist am besten, den Support auf der Plattform zu kontaktieren, auf die Sie sich zurückziehen.
