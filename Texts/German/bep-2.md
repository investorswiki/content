# BEP-2
BEP-2 ist ein technischer Standard für die Ausgabe und Implementierung von Tokens in der Binance Chain. Dieser Standard definiert eine Reihe von Regeln, denen Token folgen sollten, um im Ökosystem der Binance Chain zu funktionieren. Ein vergleichbarer Token-Standard für die Ausgabe von Token auf der Ethereum-Blockchain ist [ERC-20](/erc-20).

BEP-2 ist keine Software, sondern eine Reihe technischer Spezifikationen. Diese Regeln stellen sicher, dass Token im Ökosystem der Binance Chain nahtlos miteinander interagieren können. BEP-2-Token können eine breite Palette von digitalen Vermögenswerten und handelbaren Gütern darstellen. Der Standard stellt sicher, dass sie sich alle reibungslos ineinander und in das umgebende Ökosystem integrieren lassen.

Der BEP-2-Standard definiert auch einige der grundlegenden und wichtigsten Merkmale von Token, die auf Binance Chain ausgegeben werden. Während der Standard die Regeln definiert, haben verschiedene Token immer noch unterschiedliche Implementierungen für die spezifischen Anwendungsfälle, für die sie erstellt wurden. Dennoch verringert die Schaffung eines solchen Standards den Aufwand, der von Entwicklern für die Ausgabe von Token erforderlich ist, erheblich.

Der native Vermögenswert von Binance Chain ist BNB, das zur Zahlung von Gebühren verwendet wird (ähnlich wie Gas). Das bedeutet, dass Sie, wenn Sie Ihre BEP-2-Token übertragen möchten, auch eine kleine Menge BNB in Ihrer Brieftasche haben müssen.

BEP-2-Token können auf Binance DEX gehandelt werden, einer [dezentralen Börse](/decentralized-exchange),. die auf Binance Chain läuft.

Sie können BEP-2-Token in vielen verschiedenen Wallets aufbewahren. Trust Wallet ist eine ausgezeichnete Option, wenn Sie eine mobile Brieftasche verwenden möchten, aber auch viele andere Software-Wallets unterstützen den Standard. Der BEP-2-Standard wird auch von Hardware-Wallets wie dem Ledger Nano X, Ledger Nano S, Trezor Model T oder dem Coolwallet S unterstützt.

Sie können alle Token, die auf Binance Chain ausgegeben wurden, im Block-Explorer von Binance Chain durchsuchen.

Über dieses Portal können Sie BEP-2-Token ausstellen. Es wird dringend empfohlen, zuerst die detaillierteren technischen Spezifikationen auf dem Binance Chain GitHub durchzulesen. Wenn Sie mit BEP-2-Token-Ideen herumspielen möchten, können Sie dies im Testnetz von Binance Chain tun.

Detaillierte technische Leitfäden zur Ausgabe von Token und viele verwandte Themen finden Sie in der Binance Chain-Dokumentation.
