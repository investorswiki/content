# Bereitstellungsgebühr
## Was ist eine Bereitstellungsgebühr?

Eine Bereitstellungsgebühr ist eine Gebühr, die ein Kreditgeber einem Kreditnehmer berechnen kann, dem er zugestimmt hat, einen Kredit zu gewähren. Die Bereitstellungsgebühr wird in der Regel für noch nicht in Anspruch genommene Kreditlinien erhoben und ist eine Garantie dafür, dass die Bank die Mittel bereithält.

## Tiefere Definition

Finanzinstitute erheben manchmal eine Bereitstellungsgebühr, weil sie zugestimmt haben, eine Kreditlinie verfügbar zu halten, aber keine Zinsen berechnen können, weil der Kreditnehmer sie noch nicht in Anspruch genommen hat. Einige Kreditgeber berechnen eine Pauschalgebühr für die Aufrechterhaltung einer Kreditlinie, während andere einen Prozentsatz des gesamten Kreditbetrags berechnen.

Banken können die Bereitstellungsgebühr nur einmal erheben, wenn der Vertrag abgeschlossen ist, oder sie können sie periodisch erheben, solange die Kreditlinie offen bleibt. Kreditgeber können die Bereitstellungsgebühr jährlich erheben. Wenn sie einen Prozentsatz anstelle einer Pauschale verwenden, kann der Betrag zwischen 0,25 Prozent und 1,5 Prozent der gesamten Kreditsumme liegen.

Finanzinstitute verlangen möglicherweise die Bereitstellungsgebühr im Voraus statt regelmäßig, und einige können die Bereitstellungsgebühr erstatten, sobald der Kreditgeber das Darlehen vollständig zurückgezahlt hat.

## Beispiel für Verpflichtungsgebühr

Wenn eine Bank zustimmt, Ihnen 500.000 US-Dollar zu leihen, Sie das Geld aber noch nicht angenommen haben, wird sie Ihnen wahrscheinlich eine Verpflichtungserklärung zusenden, in der die vollständigen Bedingungen des Kreditvertrags aufgeführt sind. Wenn die Bank eine Pauschalgebühr erhebt, kann es sein, dass Sie diese Gebühr im Voraus bezahlen müssen, bevor Sie das Darlehensgeld erhalten.

Die Bank kann jedoch stattdessen einen Prozentsatz des gesamten Kreditbetrags berechnen; Beispielsweise würde ein Darlehen von 500.000 USD mit einer Bereitstellungsgebühr von 0,25 Prozent insgesamt 1.250 USD betragen.

## Höhepunkte

- Die Kosten einer Bereitstellungsgebühr variieren je nach Kreditgeber.

- Wenn eine Bereitstellungsgebühr anfällt, handelt es sich um ein zukünftiges oder nicht ausgezahltes Darlehen, im Gegensatz zu Zinsgebühren, die auf der Grundlage einer bereits ausgegebenen oder geliehenen Summe berechnet werden.

- Kreditgeber verwenden Bereitstellungsgebühren, um sicherzustellen, dass sie für garantierte Finanzdienstleistungen entschädigt werden.

- Wenn Sie ein Hypothekendarlehen aufnehmen, können Bereitstellungsgebühren in die Abschlusskosten einfließen.
