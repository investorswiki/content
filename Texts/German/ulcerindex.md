# Ulkusindex (UI)
## Was bedeutet Ulkusindex?

Der Ulcer Index (UI) ist ein technischer Indikator, der das [Abwärtsrisiko](/downsiderisk) sowohl in Bezug auf die Tiefe als auch die Dauer von Kursrückgängen misst. Der Index steigt an Wert, wenn sich der Preis weiter von einem jüngsten Hoch entfernt und fällt, wenn der Preis auf neue Höchststände steigt. Der Indikator wird normalerweise über einen Zeitraum von 14 Tagen berechnet, wobei der Ulcer Index den prozentualen Verlust angibt, den ein Händler von dem Hoch in diesem Zeitraum erwarten kann.

Je höher der Wert des Ulcer Index ist, desto länger dauert es, bis eine Aktie wieder auf das frühere Hoch zurückkehrt. Einfach gesagt, es ist nur ein Maß für die Volatilität auf der Unterseite.

## Ulcer Index (UI) verstehen

Der Ulcer Index wurde 1987 von Peter Marin und Byron McCann zur Analyse von Investmentfonds entwickelt. Marin und McCann veröffentlichten es erstmals 1989 in ihrem Buch **The Investor's Guide to Fidelity Funds**. Der Indikator betrachtet nur das Abwärtsrisiko, nicht die Gesamtvolatilität. Andere Volatilitätsmaße, wie die Standardabweichung, behandeln Aufwärts- und Abwärtsbewegungen gleich, aber ein Trader hat normalerweise nichts gegen Aufwärtsbewegungen; es ist die Kehrseite, die Stress und Magengeschwüre verursacht, wie der Name des Index schon sagt.

## Berechnung des Ulcus-Index

Der Indikator wird in drei Schritten berechnet:

- Prozentualer Drawdown = [(Schlusskurs - 14-Perioden High Close)/14-Perioden High Close] x 100

- Quadratischer Durchschnitt = (14-Perioden-Summe des prozentualen Drawdowns zum Quadrat)/14

- Ulkusindex = Quadratwurzel des quadrierten Durchschnitts

Welches Kurshoch bei der Berechnung des Ulcer Index verwendet wird, wird durch Anpassung des Rückblickzeitraums bestimmt. Ein 14-tägiger Ulkusindex misst den Rückgang vom höchsten Punkt in den letzten 14 Tagen. Ein 50-Tage-Geschwürindex misst den Rückgang vom 50-Tage-Hoch. Ein längerer Rückblickzeitraum bietet Anlegern eine genauere Darstellung der langfristigen Kursrückgänge, denen sie möglicherweise ausgesetzt sind. Eine kürzerfristige Rückblickperiode bietet Händlern einen Anhaltspunkt für die jüngste Volatilität.

## Verwendung des Ulcus-Index

Martin empfiehlt den Ulcer Index als Risikomaß in verschiedenen Kontexten, in denen normalerweise die [Standardabweichung](/standarddeviation) verwendet wird. Der Ulcer Index kann auch im Zeitverlauf dargestellt und als eine Art Indikator für die technische Analyse verwendet werden, um zu zeigen, dass Aktien in den Bereich der Geschwürbildung gehen, oder um die Volatilität verschiedener Aktien zu vergleichen.

Anleger können den Ulcer Index nutzen, um verschiedene Anlagemöglichkeiten zu vergleichen. Ein niedrigerer durchschnittlicher Ulcer Index bedeutet ein geringeres Drawdown-Risiko im Vergleich zu einer Anlage mit einem höheren durchschnittlichen UI. Die Anwendung eines gleitenden Durchschnitts auf den Ulcer Index zeigt, welche Aktien und Fonds insgesamt eine geringere Volatilität aufweisen.

Das Beobachten von Spitzen im Ulcer-Index, die über das „Normale“ hinausgehen, kann auch verwendet werden, um auf Zeiten mit übermäßigem Abwärtsrisiko hinzuweisen, die Anleger möglicherweise vermeiden möchten, indem sie Long-Positionen verlassen.
