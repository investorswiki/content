# Punkte
## Was sind Punkte?

Punkte können ein Prozentsatz einer Zahl oder ein Maß für die Änderung einer Zahl sein. Punkte werden in verschiedenen Zusammenhängen in Finanzangelegenheiten verwendet. Sie können den Zinssatz für eine Hypothek im Verhältnis zum Leitzins oder die Gesamthöhe der mit einer Hypothek verbundenen Gebühren angeben. Sie geben die prozentuale Veränderung der Rendite einer Anleihe an. Sie werden auch verwendet, um die Preisbewegungen nach oben oder unten von Aktien zu melden.

Punkte werden manchmal auch als [Basispunkte](/basispoint) (bps) bezeichnet.

## Punkte verstehen

Ein Punkt drückt immer eine Menge von etwas aus. Von dort aus variieren seine Definition und seine Verwendung. Ein Punkt gibt manchmal, aber nicht immer, einen Prozentsatz an.

- Bei Anleihen und Schuldverschreibungen entspricht ein einzelner Punkt einer Veränderung des Nennwerts der Anlage um 1 %. Wenn eine Anleihe zu zwei Punkten unter Nennwert verkauft wird, wurde sie zu 2 % unter ihrem Nennwert verkauft.

- Bei Aktien werden Bewegungen von Aktien oder Aktienindizes oft in Punkten angegeben, wobei ein Punkt 1 $ entspricht. Wenn eine Aktie eine Sitzung bei 23 $ eröffnet und den Tag bei 25 $ beendet, soll sie um zwei Punkte gestiegen sein.

- Bei Terminkontrakten ist ein Punkt eine Preisänderung von einem Hundertstel Cent.

### Punkte in Hypotheken und anderen Darlehen

Im Bankwesen kann sich ein Punkt auf die prozentuale Differenz zwischen einer Hypothek oder einem anderen Darlehen und dem zu diesem Zeitpunkt geltenden Leitzins beziehen.

Beispielsweise kann eine Kreditkarte zu einem niedrigen Einführungszinssatz angeboten werden, der sich in sechs Monaten in einen Zinssatz von 12,99 Punkten über dem Leitzinssatz umwandelt. Mitte 2020 lag der Leitzins bei 3,25 %, sodass der Zinssatz für die Karte 16,24 % betragen würde.

Wenn eine Bank den Zinssatz einer Hypothek als Prime plus zwei Punkte bewirbt, bedeutet dies, dass der Zinssatz des Darlehens 2 % plus den Prime Rate beträgt. Wenn der Leitzins 3,25 % beträgt, beträgt der Hypothekenzins 5,25 %.

Bei Hypotheken kann ein Punkt auch die Höhe der vom Kreditgeber erhobenen Kreditvergabegebühr angeben. Jeder Punkt entspricht 1 % des Darlehensbetrags. Wenn eine Bank eine Hypothek in Höhe von 200.000 USD mit einer dreistufigen Bearbeitungsgebühr anbietet, wird für das Darlehen eine Bearbeitungsgebühr in Höhe von 6.000 USD erhoben.

### Punkte in Aktien und anderen Wertpapieren

Nur um die Verwirrung zu erhöhen, hat ein Punkt eine andere Definition, wenn er verwendet wird, um die Preisbewegung einer Anleihe, eines Futures-Kontrakts oder einer Aktie zu beschreiben.

- Ein Anstieg des Preises einer Anleihe um zwei Punkte weist auf eine Wertänderung von 2 % hin, z. B. einen Anstieg von 10.000 USD auf 10.200 USD.

- Eine Erhöhung um zwei Punkte bei einem [Futures-Kontrakt](/futurescontract) entspricht einer Erhöhung um zwei Hundertstel Cent, was 2 % eines Pennys entspricht.

- Eine Erhöhung des Preises einer Stammaktie um zwei Punkte entspricht einer Erhöhung um 2 USD, z. B. wenn eine Aktie von 100 USD auf 102 USD steigt.

## Höhepunkte

- Ein Punkt entspricht immer einem.

- Ein Hypothekenpunkt kann den Prozentsatz der mit dem Darlehen verbundenen Gebühren oder die Prämie des Darlehens gegenüber dem Hauptzinssatz angeben.

- Es kann ein Prozent (wie bei einer Änderung des Anleihekurses) oder 1 $ (für einen Aktienkurs) betragen.
