# Zielsuche
## Was ist Zielsuche?

Die Zielsuche ist der Prozess, den richtigen Eingabewert zu finden, wenn nur die Ausgabe bekannt ist. Die Funktion der Zielsuche kann in verschiedene Arten von Computersoftwareprogrammen wie Microsoft Excel eingebaut werden.

## Zielsuche verstehen

Zielsuche ist ein allgemeiner Begriff, der verwendet wird, um den Prozess zu beschreiben, der zur Ermittlung Ihres Eingabewerts auf der Grundlage eines bereits bekannten Ausgabewerts erforderlich ist. Der Prozess beinhaltet die Verwendung eines bestimmten Operators in einer Formel, die mit einer Computersoftware berechnet werden kann.

Die Zielsuche ist eines der Werkzeuge, die bei der "Was-wäre-wenn-Analyse" in Computersoftwareprogrammen verwendet werden. Eine [Was](/sensitivityanalysis) [-wäre-wenn-Analyse](/sensitivityanalysis) ist ein Prozess, bei dem Werte in (Microsoft Excel-)Zellen geändert werden, um zu sehen, wie sich diese Änderungen auf die Formelergebnisse auf dem Arbeitsblatt auswirken. Bei der Zielsuche führen Sie eine Was-wäre-wenn-Analyse für einen bestimmten Wert oder die Ausgabe durch. Im Wesentlichen würden Sie also ein Szenario erstellen, indem Sie fragen: „Was wäre, wenn die Ausgabe X wäre“ – oder im Grunde eine Ursache-Wirkungs-Situation.

Für einige der komplexeren Probleme wird häufig Computersoftware verwendet. Ein Tabellenkalkulationsprogramm wie Microsoft Excel verfügt über ein integriertes Tool zur Zielsuche. Es ermöglicht dem Benutzer, den gewünschten Eingabewert für eine Formel zu bestimmen, wenn der Ausgabewert bereits bekannt ist. Diese Funktion kann dem Benutzer helfen, Dinge wie den [Zinssatz zu bestimmen, für den sich](/interestrate) ein Kreditnehmer qualifizieren muss (die Eingabe), wenn er nur weiß, wie viel er jeden Monat zahlen kann (die Ausgabe).

Aber es gibt eine Einschränkung bei der Verwendung von Zielsuchsoftware: Sie funktioniert nur, wenn es einen Eingabewert gibt. Wenn Sie zwei oder mehr Eingabewerte herausfinden müssen, funktioniert es nicht. Wenn wir das obige Beispiel nehmen, wenn Sie den Gesamtbetrag des Darlehens und die monatliche Zahlung herausfinden möchten, wird eine zielsuchende Software nicht funktionieren. Sie benötigen wahrscheinlich ein Add-On, um mehrere Variablen herauszufinden.

### Wie die Zielsuche in Microsoft Excel funktioniert

Wie oben erwähnt, funktioniert eine Zielsuchsoftware nur, wenn Sie den Ausgabewert (oder das Ergebnis) bereits kennen, aber einen Eingabewert bestimmen möchten. Hier sind einige wichtige Schritte, die Sie befolgen können, wenn Sie die Zielsuchfunktion im Programm verwenden möchten:

- Öffnen Sie eine neue Tabelle

- Beschriften Sie Ihre Spalten. Das erleichtert es dir, alles auf dem Arbeitsblatt zu lesen. Wenn Sie also das obige Beispiel verwenden, lautet die erste Spalte „Kreditbetrag“, die zweite Spalte „ [Laufzeit](/term) in Monaten“, die dritte „Zinssatz“ und die letzte Spalte „Zahlung“.

- Geben Sie die Ihnen bekannten Werte ein.

- Geben Sie die Formel ein, um das Ziel zu berechnen – in diesem Fall die Zahlung. Sie können den Zinssatz vorerst ignorieren, da die Formel davon ausgeht, dass er 0 Prozent beträgt.

Jetzt müssen Sie den Zinssatz bestimmen. Sie können dies tun, indem Sie die Goal Seek-Funktion in Excel verwenden und die Werte eingeben, die Sie bereits haben.

## Beispiel für Zielsuche

Ein zielorientierter [Unternehmer](/entrepreneur) ist jemand, der mithilfe von Zielstreben bestimmt, wie er seine endgültigen Ziele erreichen kann. Beispielsweise kann ein Unternehmer fragen, wie viel er pro Stunde verdienen muss, um in einem Jahr 100.000 US-Dollar brutto zu verdienen. Sie kennen ihren gewünschten Ausgabewert – 100.000 $ – und müssen daher zurückarbeiten, um den optimalen Input herauszufinden. Das heißt, sie müssen bestimmen, wie viele Stunden sie im Laufe des Jahres arbeiten können (oder wollen) und wie viel sie für jede Stunde verdienen, um ihr Ziel zu erreichen.

## Höhepunkte

- Zielsuchsoftware funktioniert jedoch nur, wenn es nur einen Eingabewert gibt.

- Zielsuche ist das Finden der richtigen Eingabe, wenn nur die Ausgabe bekannt ist.

- Dies erfolgt häufig über die Zielsuchfunktion in Microsoft Excel oder die Was-wäre-wenn-Analyse über Computerprogramme.
