# Cloud Computing
## Was ist Cloud-Computing?

Cloud Computing ist die Bereitstellung verschiedener Dienste über das Internet. Zu diesen Ressourcen gehören Tools und Anwendungen wie Datenspeicherung, Server, Datenbanken, Netzwerke und Software.

Anstatt Dateien auf einer proprietären Festplatte oder einem lokalen Speichergerät zu speichern, ermöglicht die [Cloud-basierte Speicherung](/cloud-storage),. sie in einer entfernten Datenbank zu speichern. Solange ein elektronisches Gerät Zugriff auf das Internet hat, hat es Zugriff auf die Daten und die Softwareprogramme, um es auszuführen.

Cloud Computing ist aus einer Reihe von Gründen, darunter Kosteneinsparungen, gesteigerte Produktivität, Geschwindigkeit und Effizienz, Leistung und Sicherheit, eine beliebte Option für Privatpersonen und Unternehmen.

## Cloud Computing verstehen

Cloud Computing wird so genannt, weil die Informationen, auf die zugegriffen wird, entfernt in der Cloud oder einem virtuellen Raum gefunden werden. Unternehmen, die Cloud-Dienste anbieten, ermöglichen es Benutzern, Dateien und Anwendungen auf entfernten Servern zu speichern und dann über das Internet auf alle Daten zuzugreifen. Dies bedeutet, dass der Benutzer nicht an einem bestimmten Ort sein muss, um darauf zuzugreifen, sodass der Benutzer aus der Ferne arbeiten kann.

Cloud Computing nimmt dem Gerät, das Sie mit sich herumtragen oder an dem Sie sitzen und arbeiten, all die schwere Arbeit ab, die mit dem Knirschen und Verarbeiten von Daten verbunden ist. Es verlagert auch all diese Arbeit auf riesige Computercluster weit entfernt im Cyberspace. Das Internet wird zur Cloud, und voilà – Ihre Daten, Arbeit und Anwendungen sind von jedem Gerät verfügbar, mit dem Sie sich mit dem Internet verbinden können, überall auf der Welt.

Cloud Computing kann sowohl öffentlich als auch privat sein. Öffentliche Cloud-Dienste stellen ihre Dienste gegen Gebühr über das Internet bereit. Private Cloud-Dienste hingegen stellen Dienste nur einer bestimmten Anzahl von Personen zur Verfügung. Diese Dienste sind ein Netzwerksystem, das gehostete Dienste bereitstellt. Es gibt auch eine hybride Option, die Elemente der öffentlichen und privaten Dienste kombiniert.

## Arten von Cloud-Diensten

Unabhängig von der Art des Dienstes bieten Cloud-Computing-Dienste Benutzern eine Reihe von Funktionen, darunter:

- Email

- Speicherung, Sicherung und Datenwiederherstellung

- Erstellen und Testen von Apps

- Daten analysieren

- Audio- und Video-Streaming

- Bereitstellung von Software nach Bedarf

Cloud Computing ist immer noch ein ziemlich neuer Dienst, wird aber von einer Reihe verschiedener Organisationen genutzt, von großen Konzernen bis hin zu kleinen Unternehmen, gemeinnützigen Organisationen, Regierungsbehörden und sogar einzelnen Verbrauchern.

## Bereitstellungsmodelle

Es gibt verschiedene Arten von Wolken, die sich alle voneinander unterscheiden. Public Clouds stellen ihre Dienste auf Servern und Speichern im Internet bereit. Diese werden von Drittfirmen betrieben, die die gesamte Hardware, Software und die allgemeine Infrastruktur handhaben und kontrollieren. Kunden greifen über Konten auf Dienste zu, auf die nahezu jeder zugreifen kann.

Private Clouds sind bestimmten Kunden vorbehalten, in der Regel einem Unternehmen oder einer Organisation. Das Rechenzentrum des Unternehmens kann den Cloud-Computing-Dienst hosten. Viele private Cloud-Computing-Dienste werden in einem privaten Netzwerk bereitgestellt.

Hybrid Clouds sind, wie der Name schon sagt, eine Kombination aus öffentlichen und privaten Diensten. Diese Art von Modell ermöglicht dem Benutzer mehr Flexibilität und hilft, die Infrastruktur und Sicherheit des Benutzers zu optimieren.

> Zu den neueren Formen von Cloud-Computing-Diensten gehören die Community Cloud, die Big Data Cloud und die Multicloud.

## Arten von Cloud Computing

Cloud Computing ist keine einzelne Technologie wie ein Mikrochip oder ein Mobiltelefon. Vielmehr handelt es sich um ein System, das hauptsächlich aus drei Diensten besteht: [Software-as-a-Service](/software-as-a-service-saas) (SaaS), Infrastructure-as-a-Service (IaaS) und Platform-as-a-Service (PaaS).

1. **Software-as-a-Service (SaaS)** umfasst die Lizenzierung einer Softwareanwendung an Kunden. Lizenzen werden in der Regel über ein Pay-as-you-go-Modell oder On-Demand bereitgestellt. Diese Art von System findet sich in Microsoft Office 365.

1. **Infrastructure-as-a-Service (IaaS)** umfasst eine Methode zur Bereitstellung von allem, von Betriebssystemen bis hin zu Servern und Speicher, über IP-basierte Konnektivität als Teil eines On-Demand-Dienstes. Kunden können den Kauf von Software oder Servern vermeiden und diese Ressourcen stattdessen in einem [ausgelagerten](/outsourcing) On-Demand-Service beschaffen. Beliebte Beispiele für das IaaS-System sind IBM Cloud und Microsoft Azure.

1. **Platform-as-a-Service (PaaS)** gilt als die komplexeste der drei Ebenen des Cloud-basierten Computing. PaaS hat einige Ähnlichkeiten mit SaaS, der Hauptunterschied besteht darin, dass es sich nicht um die Online-Bereitstellung von Software, sondern um eine Plattform zur Erstellung von Software handelt, die über das Internet bereitgestellt wird. Dieses Modell umfasst Plattformen wie Salesforce.com und Heroku.

## Vorteile von Cloud Computing

Cloudbasierte Software bietet Unternehmen aus allen Branchen eine Reihe von Vorteilen, darunter die Möglichkeit, Software von jedem Gerät aus entweder über eine native App oder einen Browser zu verwenden. Dadurch können Benutzer ihre Dateien und Einstellungen völlig nahtlos auf andere Geräte übertragen.

Cloud Computing ist weit mehr als nur der Zugriff auf Dateien auf mehreren Geräten. Dank Cloud-Computing-Diensten können Benutzer ihre E-Mails auf jedem Computer abrufen und sogar Dateien mit Diensten wie Dropbox und Google Drive speichern. Cloud-Computing-Dienste ermöglichen es Benutzern auch, [ihre](/back-up) Musik, Dateien und Fotos zu sichern und sicherzustellen, dass diese Dateien im Falle eines Festplattencrashs sofort verfügbar sind.

Es bietet auch großen Unternehmen ein enormes Kosteneinsparungspotenzial. Bevor die Cloud zu einer praktikablen Alternative wurde, mussten Unternehmen kostspielige [Informationsmanagement](/information-management-technology-imt) [-Technologien](/information-management-technology-imt) und -Infrastrukturen kaufen, aufbauen und warten . Unternehmen können kostspielige Serverzentren und IT-Abteilungen gegen schnelle Internetverbindungen austauschen, wo Mitarbeiter online mit der Cloud interagieren, um ihre Aufgaben zu erledigen.

Die Cloud-Struktur ermöglicht es Einzelpersonen, Speicherplatz auf ihren Desktops oder Laptops zu sparen. Es ermöglicht Benutzern auch , Software schneller zu [aktualisieren](/upgrade),. da Softwareunternehmen ihre Produkte über das Internet anbieten können, anstatt über traditionellere, konkrete Methoden mit Discs oder Flash-Laufwerken. Beispielsweise können Adobe-Kunden über ein internetbasiertes Abonnement auf Anwendungen in der Creative Cloud zugreifen. Dies ermöglicht es Benutzern, neue Versionen und Fixes für ihre Programme einfach herunterzuladen.

## Nachteile der Cloud

Bei all der Geschwindigkeit, Effizienz und Innovation, die Cloud Computing mit sich bringt, gibt es natürlich Risiken.

Sicherheit war schon immer ein großes Anliegen bei der Cloud, insbesondere wenn es um sensible Krankenakten und Finanzinformationen geht. Während Vorschriften Cloud-Computing-Dienste dazu zwingen, ihre Sicherheits- und Compliance-Maßnahmen zu verstärken, bleibt dies ein anhaltendes Problem. Die Verschlüsselung schützt wichtige Informationen, aber wenn dieser Verschlüsselungsschlüssel verloren geht, verschwinden die Daten.

Auch Server, die von Cloud-Computing-Unternehmen betrieben werden, können Opfer von Naturkatastrophen, internen Fehlern und Stromausfällen werden. Die geografische Reichweite von Cloud Computing ist in beide Richtungen begrenzt: Ein Stromausfall in Kalifornien könnte Benutzer in New York lahmlegen, und eine Firma in Texas könnte ihre Daten verlieren, wenn etwas ihren in Maine ansässigen Anbieter zum Absturz bringt.

Wie bei jeder Technologie gibt es eine Lernkurve für Mitarbeiter und Manager. Da jedoch viele Personen über ein einziges Portal auf Informationen zugreifen und diese bearbeiten, können unbeabsichtigte Fehler auf ein gesamtes System übertragen werden.

## Die Welt der Wirtschaft

Unternehmen können Cloud Computing auf unterschiedliche Weise einsetzen. Einige Benutzer verwalten alle Apps und Daten in der Cloud, während andere ein Hybridmodell verwenden, bei dem bestimmte Apps und Daten auf privaten Servern und andere in der Cloud gespeichert werden.

Wenn es um die Bereitstellung von Dienstleistungen geht, gehören zu den großen Akteuren im Bereich Corporate Computing:

- Google-Cloud

- Amazon-Webdienste (AWS)

-Microsoft Azure

- IBM-Cloud

- Alibaba-Cloud

Amazon Web Services ist zu 100 % öffentlich und umfasst ein ausgelagertes Pay-as-you-go-Modell. Sobald Sie auf der Plattform sind, können Sie sich für Apps und zusätzliche Dienste anmelden. Microsoft Azure ermöglicht es Kunden, einige Daten an ihren eigenen Standorten zu speichern. Inzwischen ist Alibaba Cloud eine Tochtergesellschaft der Alibaba Group.

## Höhepunkte

- Cloud-Sicherheit ist zu einem immer wichtigeren Bereich in der IT geworden.

- Cloud Computing ist die Bereitstellung verschiedener Dienste über das Internet, einschließlich Datenspeicherung, Server, Datenbanken, Netzwerke und Software.

- Cloud-basierter Speicher ermöglicht es, Dateien in einer entfernten Datenbank zu speichern und sie bei Bedarf abzurufen.

- Dienste können sowohl öffentlich als auch privat sein – öffentliche Dienste werden online gegen eine Gebühr bereitgestellt, während private Dienste in einem Netzwerk für bestimmte Kunden gehostet werden.

- Cloud-Speicher wird immer beliebter bei Einzelpersonen, die mehr Speicherplatz benötigen, und bei Unternehmen, die eine effiziente Offsite-Datensicherungslösung suchen.

## FAQ

### Was ist ein Beispiel für Cloud Computing?

Heutzutage gibt es mehrere Beispiele für Cloud-Computing-Anwendungen, die sowohl von Unternehmen als auch von Einzelpersonen verwendet werden. Eine Art von Cloud-Diensten wären Streaming-Plattformen für Audio oder Video, bei denen die eigentlichen Mediendateien entfernt gespeichert werden. Eine andere wären Datenspeicherplattformen wie Google Drive, Dropbox, OneDrive oder Box.

### Ist Cloud Computing sicher?

Da Software und Daten beim Cloud Computing remote gespeichert werden, sind Datensicherheit und Plattformsicherheit ein großes Anliegen. [Cloud-Sicherheit](/cloud-security) bezieht sich auf die Maßnahmen, die ergriffen werden, um digitale Assets und Daten zu schützen, die in Cloud-basierten Diensten gespeichert sind. Zu den Maßnahmen zum Schutz dieser Daten gehören unter anderem die Zwei-Faktor-Autorisierung (2FA), die Verwendung von VPNs, Sicherheitstoken, Datenverschlüsselung und Firewall-Dienste.

### Was sind die wichtigsten Arten von Cloud Computing?

Zu den wichtigsten Arten von Cloud-Computing-Diensten gehören Infrastructure-as-a-Service (IaaS), Platforms-as-a-Service (PaaS) und Software-as-a-Service (SaaS). Benutzer über das Internet und wird allgemein mit Serverless Computing in Verbindung gebracht. PaaS stellt sowohl Software als auch Hardware für Endbenutzer bereit, die im Allgemeinen Softwareentwickler sind. PaaS ermöglicht es dem Benutzer, seine eigenen Apps zu entwickeln, auszuführen und zu verwalten, ohne die Infrastruktur aufbauen und warten zu müssen. - SaaS ist ein Softwarelizenzmodell, das den Zugriff auf Software auf Abonnementbasis über externe Server ermöglicht, ohne sie herunterladen und installieren zu müssen örtlich.
