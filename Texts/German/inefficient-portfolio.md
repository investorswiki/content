# Ineffizientes Portfolio
## Was ist ein ineffizientes Portfolio?

Ein ineffizientes Portfolio ist eines, das eine erwartete Rendite liefert, die für das eingegangene Risiko zu niedrig ist. Umgekehrt bezieht sich ein ineffizientes Portfolio auch auf eines, das für eine gegebene erwartete Rendite zu viel Risiko erfordert. Im Allgemeinen hat ein ineffizientes Portfolio ein schlechtes Risiko-Rendite-Verhältnis.

## Ein ineffizientes Portfolio verstehen

Ein ineffizientes [Portfolio](/portfolio) setzt einen Anleger einem höheren Risiko aus als nötig, um eine Zielrendite zu erreichen. Beispielsweise würde ein Portfolio aus Hochzinsanleihen, von dem erwartet wird, dass es nur die risikofreie Rendite bietet, als ineffizient bezeichnet. Ein Anleger könnte die gleiche Rendite erzielen, indem er [Schatzwechsel kauft](/treasurybill),. die als eine der sichersten Anlagen der Welt gelten (im Gegensatz zu Hochzinsanleihen, die per Definition als riskante Anlagen eingestuft werden).

## Effiziente Portfolios vs. ineffiziente Portfolios

In einem effizienten Portfolio werden investierbare Vermögenswerte so kombiniert, dass sie die bestmögliche erwartete Rendite für ihr Risikoniveau erzielen – oder das niedrigste Risiko für eine Zielrendite. Die Linie, die all diese effizienten Portfolios verbindet, wird als [Effizienzgrenze bezeichnet](/efficientfrontier). Die Effizienzgrenze stellt jene Portfolios dar, die für jedes gegebene Risikoniveau die maximale Rendite aufweisen. Das Letzte, was Anleger wollen, ist ein Portfolio mit einer niedrigen erwarteten Rendite und einem hohen Risikoniveau.

Kein Punkt auf der Effizienzgrenze ist besser als jeder andere Punkt. Anleger müssen ihre eigenen Risiko-Rendite-Präferenzen prüfen, um zu bestimmen, wo sie an der Effizienzgrenze investieren sollten. Dieses Konzept wurde erstmals 1952 von [Harry Markowitz](/harrymarkowitz) formuliert . In seinem Artikel „Portfolio Selection“, der 1952 im Journal of Finance veröffentlicht wurde, leistete Markowitz Pionierarbeit für das Konzept der [Modern Portfolio Theory](/modernportfoliotheory) (MPT). Die Risiko- und Renditeeigenschaften einer Anlage sollten laut MPT nicht allein betrachtet werden. Stattdessen sollten sie danach bewertet werden, wie sich die Anlage auf das Risiko und die Rendite des Gesamtportfolios auswirkt.

MPT geht davon aus, dass Anleger risikoavers sind, was bedeutet, dass Anleger bei zwei Portfolios, die die gleiche erwartete Rendite bieten, das weniger riskante bevorzugen. Daher geht ein Anleger nur dann ein erhöhtes [Risiko ein](/risk),. wenn er durch höhere erwartete Renditen kompensiert wird; Ein Investor, der höhere erwartete Renditen will, muss mehr Risiko eingehen. Die Annahme ist, dass ein rationaler Investor nicht in ein Portfolio investiert, wenn ein zweites Portfolio mit einem günstigeren Risiko-Erwartungs-Rendite-Profil existiert.

In der Praxis kann MPT verwendet werden, um ein Portfolio aufzubauen, das das Risiko für ein bestimmtes erwartetes Renditeniveau minimiert; Es ist sehr nützlich für Anleger, die versuchen, effiziente Portfolios mit börsengehandelten Fonds (ETFs) aufzubauen. Effiziente Portfolios nutzen die moderne Portfoliotheorie. Der Theorie zufolge können Sie die Volatilität Ihres Portfolios begrenzen, indem Sie Ihr Risiko auf verschiedene Arten von Anlagen verteilen. Mit dieser Idee könnte ein Portfolio riskanter Aktien insgesamt ein geringeres Risiko aufweisen als ein Portfolio, das nur eine konzentrierte Position hält, selbst wenn es sich um eine relativ sichere Position handelt.

## Höhepunkte

- Im Allgemeinen hat ein ineffizientes Portfolio ein schlechtes Risiko-Rendite-Verhältnis; es setzt einen Anleger einem höheren Risiko aus, als zur Erreichung einer Zielrendite erforderlich ist.

- Ein ineffizientes Portfolio ist eines, das eine erwartete Rendite liefert, die für das eingegangene Risiko zu niedrig ist.

- In einem effizienten Portfolio werden investierbare Vermögenswerte so kombiniert, dass sie die bestmögliche erwartete Rendite für ihr Risikoniveau erzielen – oder das niedrigste Risiko für eine Zielrendite.
