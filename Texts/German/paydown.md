# Auszahlung
Amortisation ist der Prozess, bei dem der geschuldete Betrag einer Hypothek oder eines anderen Darlehens im Laufe der Zeit reduziert wird, indem Teilzahlungen auf die Schulden geleistet werden. Eine Abzahlung kann sich auf jede Schuld beziehen, wie z. B. einen Autokredit, eine Kreditkartenschuld oder einen Schulkredit.

## Tiefere Definition

Eine Rückzahlung bezieht sich oft auf die Reduzierung nur des Kapitalbetrags [eines](/principal) Darlehens. Der Vorteil der Rückzahlung des Kapitalbetrags besteht darin, dass mit dem Rückgang auch die Zinszahlungen sinken. Die Rückzahlung des Kapitalbetrags reduziert, wie viel vom Kreditnehmer insgesamt ausgezahlt wird.

Eine Rückzahlung bezieht sich auch darauf, dass eine Regierung oder ein Unternehmen mehr für eine Schuld zahlt als den geliehenen Betrag. Dies kann vorkommen, wenn eine Organisation unbezahlte Schulden für weniger als die erste Ausgabe dieser Schuld neu ausgibt.

## Auszahlungsbeispiel

Jessica kauft ein Haus für 150.000 Dollar. Sie zahlt 20 Prozent an und leiht sich die restlichen 120.000 Dollar von einer Bank in Form eines 30-jährigen Darlehens. Anstatt nur ihre festgelegten monatlichen Zahlungen zu zahlen, wendet sie jeden Monat einen zusätzlichen Betrag auf das Kapital an. Indem sie ihre Rückzahlung beschleunigt, reduziert Jessica die Zinsen, die sie für die Hypothek zahlen muss. Dies hilft ihr auch, ihre Hypothek früher abzubezahlen.

## Höhepunkte

- Verbraucher können eine Amortisation erzielen, indem sie mehr als den monatlich fälligen Mindestbetrag für eine Schuld, z. B. eine Hypothek, zahlen.

- Unternehmen erreichen eine Rückzahlung, indem sie eine neue Schuldenrunde ausgeben, die kleiner ist als eine vorherige Runde, die Fälligkeit erreicht hat.

- Eine Rückzahlung ist eine Reduzierung des geschuldeten Kapitalbetrags für ein Darlehen oder eine andere Schuld.
