# Laufrate
## Was ist die Laufrate?

Die Laufrate bezieht sich auf die finanzielle Leistung eines Unternehmens, basierend auf der Verwendung aktueller Finanzinformationen als Prädiktor für die zukünftige Leistung. Die Run Rate fungiert als Extrapolation der aktuellen finanziellen Leistung und geht davon aus, dass die aktuellen Bedingungen anhalten.

Die Run Rate kann sich auch auf die durchschnittliche jährliche Verwässerung aus der Gewährung von Aktienoptionen des Unternehmens während des letzten im Jahresbericht verzeichneten Dreijahreszeitraums beziehen.

## Die Laufrate verstehen

Im Rahmen der Extrapolation zukünftiger Performance nimmt die Run Rate aktuelle Performance-Informationen und verlängert diese über einen längeren Zeitraum. Wenn ein Unternehmen beispielsweise im letzten Quartal einen [Umsatz](/revenue) von 100 Millionen US-Dollar erzielt hat, könnte der [CEO](/ceo) daraus schließen, dass das Unternehmen auf der Grundlage des letzten Quartals mit einer Run Rate von 400 Millionen US-Dollar arbeitet. Wenn die Daten verwendet werden, um eine jährliche Prognose für die potenzielle Leistung zu erstellen, wird der Prozess als Annualisierung bezeichnet.

Eine Run Rate kann bei der Erstellung von Performance-Schätzungen für Unternehmen hilfreich sein, die erst seit kurzer Zeit tätig sind, z. B. weniger als ein Jahr, sowie neu geschaffene Abteilungen oder Profitcenter. Dies kann insbesondere für ein Unternehmen gelten, das sein erstes profitables Quartal erlebt. Darüber hinaus kann die Run Rate in Fällen hilfreich sein, in denen ein grundlegender Geschäftsbetrieb in irgendeiner Weise geändert wurde, von der erwartet wurde, dass sie sich auf alle zukünftigen Leistungen des zugehörigen Unternehmens auswirkt.

## Risiken bei der Verwendung der Run Rate

Die Laufrate kann eine sehr trügerische Kennzahl sein, insbesondere in saisonalen Branchen. Ein gutes Beispiel hierfür ist ein Einzelhändler, der den Gewinn nach der Winterferiensaison untersucht, da dies eine Zeit ist, in der viele Einzelhändler höhere Verkaufsvolumina verzeichnen. Wenn Informationen auf der Grundlage von Feiertagsverkäufen verwendet wurden, um eine Laufrate zu erstellen, können Schätzungen der zukünftigen Wertentwicklung zufällig überhöht sein.

Darüber hinaus basiert die Laufrate im Allgemeinen nur auf den aktuellsten Daten und kann umstandsbedingte Änderungen, die zu einem ungenauen Gesamtbild führen können, möglicherweise nicht richtig kompensieren. Beispielsweise verzeichnen bestimmte Technologiehersteller wie Apple und Microsoft höhere Umsätze in Korrelation mit einer neuen Produktversion. Die Verwendung von Daten nur aus dem Zeitraum unmittelbar nach einer großen Produktveröffentlichung kann zu verzerrten Daten führen.

Darüber hinaus berücksichtigen Laufraten keine großen, einmaligen Verkäufe. Wenn ein Hersteller beispielsweise einen Großauftrag erhält, der im Voraus bezahlt wird, kann dies unabhängig vom Zeitpunkt der Lieferung der Waren oder Dienstleistungen dazu führen, dass die Verkaufszahlen für einen Berichtszeitraum aufgrund dieses anomalen Einkaufs ungewöhnlich hoch sind.

## Höhepunkte

- Run Rate kann sich auch auf die durchschnittliche jährliche Verwässerung aus der Gewährung von Aktienoptionen des Unternehmens während des letzten im Jahresbericht verzeichneten Dreijahreszeitraums beziehen.

- Laufraten sind hilfreich bei der Formulierung von Leistungsschätzungen für Unternehmen, die erst seit kurzer Zeit tätig sind.

- Die Laufrate geht davon aus, dass die aktuellen Bedingungen anhalten.

- Run Rate ist die finanzielle Leistung eines Unternehmens, wobei aktuelle Finanzinformationen als Prädiktor für die zukünftige Leistung verwendet werden.

## FAQ

### Wie kann die Verwendung der Laufrate hilfreich sein?

B. weniger als ein Jahr, sowie neu geschaffene Abteilungen oder Profitcenter. ### Was sind einige Nachteile der Run Rate?

Die Laufrate kann eine sehr trügerische Metrik sein, insbesondere in saisonalen Branchen, in denen Schätzungen der zukünftigen Leistung möglicherweise falsch überhöht sind. Da sie außerdem im Allgemeinen nur auf den aktuellsten Daten basiert, kann sie umstandsbedingte Änderungen, die zu einem ungenauen Gesamtbild führen können, möglicherweise nicht richtig kompensieren. Darüber hinaus berücksichtigen Laufraten keine großen, einmaligen Ereignisse, die Prognosen verzerren können.

### Wie kommt die Run Rate zustande?

Im Finanzbereich extrapoliert die Run Rate die aktuelle Leistung eines Unternehmens, um die zukünftige Leistung unter der Annahme vorherzusagen, dass die aktuellen Bedingungen bestehen bleiben. Die Projektionen beziehen sich normalerweise auf ein ganzes Jahr, weshalb dieser Vorgang manchmal als "Annualisierung" bezeichnet wird. Wenn ein Unternehmen beispielsweise im letzten Quartal einen Umsatz von 100 Millionen US-Dollar erzielt hat, könnte der CEO daraus schließen, dass das Unternehmen mit einer jährlichen Laufrate von 400 Millionen US-Dollar arbeitet.
