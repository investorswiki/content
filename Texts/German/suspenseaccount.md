# Konto gesperrt
Ein Zwischenkonto ist ein Abschnitt eines [Hauptbuchs](/generalledger),. in dem eine Organisation mehrdeutige Einträge aufzeichnet, die noch einer weiteren Analyse bedürfen, um ihre ordnungsgemäße Klassifizierung und/oder ihren korrekten Bestimmungsort zu bestimmen. Im Zusammenhang mit Investitionen ist ein Suspense-Konto ein Maklerkonto, auf dem ein Anleger sein Geld vorübergehend parkt, bis er dieses Geld für den Kauf neuer Investitionen einsetzen kann.

## Suspense-Konten verstehen

Zwischenkonten enthalten Einträge, bei denen Unsicherheiten oder Unstimmigkeiten bestehen. Wenn beispielsweise eine Person eine Einzahlung tätigt, aber versehentlich eine Kontonummer falsch notiert, wird dieses Geld auf einem Zwischenkonto aufbewahrt, bis der Fehler behoben ist. In einem anderen Szenario kann ein Kunde eine Zahlung vornehmen, aber nicht angeben, welche Rechnung er mit diesen Geldern begleichen möchte.

Ungeachtet der fraglichen Unsicherheiten werden Zwischenkonten gelöscht, sobald die Verwirrung gelöst ist, und zu diesem Zeitpunkt werden die Gelder umgehend auf die korrekt bezeichneten Konten umgeschichtet. Zu diesem Zeitpunkt sollte das verdächtige Konto theoretisch einen Saldo von null Dollar erreichen. Während es keinen endgültigen Zeitplan für die Durchführung eines Räumungsprozesses gibt, versuchen viele Unternehmen, dies regelmäßig auf monatlicher oder vierteljährlicher Basis durchzuführen.

## Hypotheken-Zwischenkonten

Ein Hypothekenverwalter kann Zwischenkonten verwenden, um Gelder zu halten, wenn ein Kreditnehmer eine erforderliche monatliche Kreditrückzahlung nicht leistet, wobei das Zwischenkonto als Auffangbehälter fungiert, der die Mittel sichert. Zu diesem Zeitpunkt kann sich der Hypothekenverwalter dafür entscheiden, den erhaltenen Teilbetrag auf verschiedene Silos zu verteilen, einschließlich des geschuldeten Kapitalbetrags, [aufgelaufener Zinsen](/accruedinterest),. Grundsteuerzahlungen und Versicherungsprogramme für Hausbesitzer.

In einigen Fällen leistet ein Kreditnehmer bewusst Teilzahlungen, indem er seine monatliche Rate absichtlich in zwei Teile aufteilt. In solchen Fällen können Hypothekenverwalter Zwischenkonten verwenden, um die erste Teilzahlung unterzubringen, bis die zweite Zahlung erfolgt ist. Nach Erhalt ausreichender Mittel für eine vollständige Zahlung überweist der Hypothekenverwalter den kombinierten Saldo auf das richtige Konto.

## Brokerage-Suspense-Konten

Ähnlich wie Business-Suspense-Konten halten Makler-Suspense-Konten vorübergehend Gelder, während Transaktionen abgeschlossen werden. Wenn ein Investor beispielsweise eine Gruppe von Wertpapieren im Wert von 500 USD verkauft, aber plant, diesen Geldbetrag schnell in eine andere Gruppe von Investitionen zu investieren, werden die 500 USD aus dem Verkauf auf ein Zwischenkonto verschoben, bis sie zugewiesen werden können der Neukauf.

> Der Betrag der auf dem schwebenden Konto gehaltenen Gelder wird als „schwebendes Guthaben“ bezeichnet.

>

## Höhepunkte

- Sperrkonten werden routinemäßig gelöscht, sobald die Art der gesperrten Beträge geklärt ist, und anschließend auf die korrekt bezeichneten Konten umgeschichtet.

- Ein Zwischenkonto ist ein Sammelkonto in einem Hauptbuch, das von Unternehmen verwendet wird, um mehrdeutige Einträge zu erfassen, die einer Klärung bedürfen.

- Beim Investieren halten Zwischenkonten das Geld eines Investors, bis es reinvestiert werden kann.

- Zwischenkonten werden häufig von Hypothekenbanken verwendet, wenn ein Kreditnehmer versehentlich eine monatliche Zahlung nicht leisten kann oder wenn ein Kreditnehmer die monatliche Zahlungsverpflichtung in Teilbeträge aufteilen möchte.
