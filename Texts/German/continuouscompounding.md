# Kontinuierliches Mischen
## Was ist kontinuierliches Compoundieren?

Kontinuierliche Verzinsung ist die mathematische Grenze, die Zinseszinsen [erreichen](/interest) können, wenn sie berechnet und über eine theoretisch unendliche Anzahl von Perioden in den Kontostand reinvestiert werden. Während dies in der Praxis nicht möglich ist, ist das Konzept des kontinuierlichen Zinseszinses im Finanzwesen wichtig. Es ist ein extremer Fall der [Aufzinsung](/compounding),. da die meisten Zinsen auf monatlicher, vierteljährlicher oder halbjährlicher Basis verzinst werden.

## Formel und Berechnung der kontinuierlichen Compoundierung

Anstatt die Zinsen für eine endliche Anzahl von Perioden zu berechnen, z. B. jährlich oder monatlich, berechnet die kontinuierliche Verzinsung die Zinsen unter der Annahme einer konstanten Verzinsung über eine unendliche Anzahl von Perioden. Die Formel für den Zinseszins über endliche Zeiträume berücksichtigt vier Variablen:

- PV = Barwert der Investition

- i = der [angegebene](/stated-annual-interest-rate) Zinssatz

- n = die Anzahl der Zinsperioden

- t = die Zeit in Jahren

Die Formel für die kontinuierliche Aufzinsung leitet sich aus der Formel für den zukünftigen Wert einer verzinslichen Anlage ab:

Zukünftiger Wert (FV) = PV x [1 + (i / n)]^(nxt)^

Die Berechnung des Limits dieser Formel, wenn n gegen unendlich geht (gemäß der Definition der kontinuierlichen Verzinsung), ergibt die Formel für die kontinuierliche Verzinsung:

**FV = PV xe (ixt)**, wobei e die mathematische Konstante ist, angenähert als 2,7183.

## Was Continuous Compounding Ihnen sagen kann

Theoretisch bedeutet kontinuierlicher Zinseszins, dass ein Kontostand ständig verzinst wird und diese Zinsen wieder in den Kontostand einfließen, sodass auch er Zinsen verdient.

Kontinuierliche Verzinsung berechnet die Zinsen unter der Annahme, dass die Zinsen über eine unendliche Anzahl von Perioden verzinst werden. Obwohl die kontinuierliche Verzinsung ein wesentliches Konzept ist, ist es in der realen Welt nicht möglich, eine unendliche Anzahl von Perioden für die Berechnung und Zahlung von Zinsen zu haben. Infolgedessen werden die Zinsen typischerweise auf der Grundlage einer festen Laufzeit, wie monatlich, vierteljährlich oder jährlich, verzinst.

> Selbst bei sehr hohen Anlagesummen ist der Unterschied in der Gesamtverzinsung durch kontinuierliche Aufzinsung im Vergleich zu traditionellen Aufzinsungsperioden nicht sehr hoch.

>

## Beispiel für die Verwendung von Continuous Compounding

Nehmen wir als Beispiel an, dass eine Investition von 10.000 $ im nächsten Jahr 15 % Zinsen einbringt. Die folgenden Beispiele zeigen den Endwert der Anlage, wenn die Zinsen jährlich, halbjährlich, vierteljährlich, monatlich, täglich und fortlaufend verzinst werden.

- **Jährliche Aufzinsung:** FV = 10.000 $ x (1 + (15 % / 1)) ^(1 x 1)^ = 11.500 $

- **Halbjährliche Aufzinsung:** FV = 10.000 $ x (1 + (15 % / 2)) ^(2 x 1)^ = 11.556,25 $

- **Vierteljährliche Aufzinsung:** FV = $10.000 x (1 + (15% / 4)) ^(4 x 1)^ = $11.586,50

- **Monatliche Aufzinsung:** FV = $10.000 x (1 + (15% / 12)) ^(12 x 1)^ = $11.607,55

- **Tägliche Aufzinsung:** FV = 10.000 $ x (1 + (15 % / 365)) ^(365 x 1)^ = 11.617,98 $

- **Kontinuierliche Aufzinsung:** FV = 10.000 $ x 2,7183 ^(15 % x 1)^ = 11.618,34 $

Bei täglicher Aufzinsung beträgt der Gesamtzinsertrag 1.617,98 USD, während bei kontinuierlicher Aufzinsung der Gesamtzinsertrag 1.618,34 USD beträgt, ein marginaler Unterschied.

## Höhepunkte

- Die Formel zur Berechnung des kontinuierlichen Zinseszinses berücksichtigt vier Variablen.

- Die meisten Zinsen werden halbjährlich, vierteljährlich oder monatlich verzinst.

- Das Konzept der kontinuierlichen Zinseszinsen ist im Finanzwesen wichtig, auch wenn es in der Praxis nicht möglich ist.

- Kontinuierlich verzinste Zinsen gehen davon aus, dass die Zinsen unendlich oft verzinst und wieder in die Bilanz aufgenommen werden.
