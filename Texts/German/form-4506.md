# Formular 4506, Antrag auf Kopie der Steuererklärung
## Was ist Formular 4506, Antrag auf Kopie der Steuererklärung?

Das Formular 4506, Antrag auf Kopie der Steuererklärung, wird von Steuerzahlern eingereicht, um genaue Kopien einer oder mehrerer zuvor eingereichter Steuererklärungen und Steuerinformationen beim [Internal Revenue Service](/irs) (IRS) anzufordern. Kopien können erforderlich sein, um eine Steuererklärung für das laufende Jahr auszufüllen, eine Steuererklärung für das Vorjahr zu ändern, einen Anspruch auf [Rückerstattung](/refund) oder Ermäßigung einzureichen, staatliche Leistungen zu beantragen, staatliche Studienbeihilfen zu beantragen, das Einkommen während eines Kreditantragsverfahrens zu überprüfen oder eine IRS-Prüfung verteidigen.

Für die Anfertigung einer vollständigen Kopie einer Steuererklärung wird eine Gebühr erhoben. In manchen Fällen kann eine kostenlose Abschrift – ein kürzerer Ausdruck der Steuererklärung – genauso gut den Bedürfnissen eines Steuerzahlers dienen.

## Wer kann Formular 4506 einreichen, Antrag auf Kopie der Steuererklärung?

Steuerzahler, die Kopien früher eingereichter Steuererklärungen anfordern, können das Formular 4506, Antrag auf Kopie der Steuererklärung, einreichen. Sie können auch verlangen, dass eine Kopie an Dritte gesendet wird. Zu den Dritten kann ein Hypothekengeber gehören, der Ihr Einkommen beim IRS überprüft. In diesem Fall unterschreiben und reichen Sie das Formular 4506 ein, und der IRS sendet die Kopie der Steuererklärung direkt an Ihren Kreditgeber.

## So reichen Sie das Formular 4506 ein, Antrag auf Kopie der Steuererklärung

Das Formular 4506 kann eingereicht werden, wenn eine exakte Kopie einer zuvor eingereichten Steuererklärung benötigt wird. Es kann eine Reihe von Steuererklärungen angefordert werden, darunter die Steuererklärungen der [Form 1040](/1040) für Einzelpersonen, die Körperschaftsteuererklärungen der Serie 1120, die Steuererklärungen für Partnerschaften ( [Form 1065](/form-1065) ) und die Steuererklärungen für Erbschaften oder Trusts (Form 1041). Das Formular 4506 muss ausgefüllt und an die IRS an die auf dem Formular angegebene Postanschrift geschickt werden.

Für jeden angeforderten Steuerzeitraum wird eine Gebühr von 43 USD erhoben. Kopien sind in der Regel für Erklärungen verfügbar, die im laufenden Jahr sowie in den vorangegangenen sechs Jahren eingereicht wurden.

## Andere relevante Formulare

Andere Formulare in der 4506-Serie umfassen das Formular 4506-T, Request for Transcript of Tax Return und das Formular 4506-T-EZ, Short Form Request for Individual Tax Return Transcript. Sie ermöglichen es Ihnen, eine Abschrift – einen Ausdruck ohne Anhänge – einer zuvor eingereichten Steuererklärung anzufordern.

Eine Abschrift einer zuvor eingereichten Steuererklärung kann durch Herunterladen, Ausfüllen und Versenden des Formulars 4506-T angefordert werden.

Formular 4506-T kann eingereicht werden, wenn Sie eine Abschrift Ihrer Steuererklärung oder Steuerkontoinformationen, [Formular W-2-](/w2form) Informationen und 1099-Informationen benötigen. Steuererklärungsprotokolle sind für das aktuelle und die letzten drei Steuerjahre und nur für die folgenden Erklärungen verfügbar: Formular 1040-Serie, Formular 1120-Serie und Formular 1065. Wenn Sie sich online beim IRS registrieren, können Sie es sofort und kostenlos erhalten .

Eine individuelle Steuererklärungsabschrift Ihrer zuvor eingereichten und bearbeiteten Steuererklärung der Serie 1040 kann ebenfalls kostenlos mit dem vereinfachten Formular 4506-T-EZ angefordert werden.

> Ein Steuererklärungsprotokoll ist ein Computerausdruck Ihrer Einkommensteuererklärungsinformationen. Es zeigt Anmeldedaten, Anmeldestatus, Angehörige, [bereinigtes Bruttoeinkommen](/agi) und Steuerpflicht. Das Transkript zeigt die meisten Einzelposten in Ihrer Rücksendung so, wie es ursprünglich eingereicht wurde, obwohl es immer noch kürzer ist als eine vollständige Kopie mit allen Anhängen. Es werden keine Änderungen angezeigt, die nach der Bearbeitung der Rücksendung vorgenommen wurden.

>

Wenn Steuerinformationen dringend benötigt werden, kann eine Abschrift kostenlos bereitgestellt werden, indem Sie das Online-Tool Get Transcript des IRS verwenden oder den IRS unter 1-800-908-9946 anrufen. Abschriften werden als Ersatz für eine exakte Kopie einer Steuererklärung von den United States Citizenship and Immigration Services, von Agenturen für Studentendarlehen und Hypothekendarlehen und anderen Dritten akzeptiert, die versuchen, das Einkommen des Steuerzahlers zu überprüfen. Tatsächlich wurde das Steuererklärungsprotokoll speziell erstellt, um die Forderungen der Hypothekenbanken nach den Steuererklärungsinformationen ihrer Kreditnehmer zu erfüllen.

Ab 2019 verfügt der IRS über ein neues Transkriptformat, das vor Identitätsdiebstahl und Betrug schützt. Beispielsweise sind nur die letzten vier Ziffern Ihrer [Sozialversicherungsnummer](/ssn) und Telefonnummer sichtbar. Steuerzahler können weiterhin eine unmaskierte Version anfordern, diese wird jedoch nur an ihre eingetragene Adresse gesendet.

## Besondere Überlegungen bei der Einreichung von Formular 4506

Bei der Einreichung eines der Formulare der Serie 4506 in Papierform können Kopien und Abschriften von [gemeinsam eingereichten](/jointreturn) Steuererklärungen von beiden Ehepartnern angefordert werden, und es ist nur eine Unterschrift erforderlich. Bei der Verwendung von Get Transcript online kann entweder der primäre oder der sekundäre Ehepartner bei einer gemeinsamen Erklärung den Antrag stellen. Bei Verwendung von Get Transcript per Post oder Telefon kann nur der Hauptsteuerzahler der Steuererklärung die Anfrage stellen.

## Höhepunkte

- Wenn Sie keine exakte Kopie Ihrer Steuererklärung benötigen, können Sie mit den Formularen 4506-T und 4506-T-EZ kostenlose Abschriften einer zuvor eingereichten Steuererklärung anfordern.

- Es kostet derzeit 43 $, eine Kopie Ihrer Rücksendung zu erhalten.

- Sie können eine Reihe verschiedener Arten von bereits eingereichten Steuererklärungen anfordern, die je nach Formular sechs Jahre oder länger zurückreichen.

- Formular 4506 wird von Steuerzahlern eingereicht, um genaue Kopien von zuvor eingereichten Steuererklärungen und Steuerinformationen anzufordern.

## FAQ

### Wie lange dauert es, bis Sie Dokumente erhalten, nachdem Sie ein Formular 4506 oder 4506-T eingereicht haben?

Wenn Sie die Option „Get Transcript“ auf der IRS-Website verwenden, können Sie Ihre Formulare sofort herunterladen. Wenn Sie eine Abschrift per Post anfordern, kann es 5 bis 10 Kalendertage dauern, bis sie an Ihrer beim IRS hinterlegten Adresse ankommt. Wenn Sie das Formular 4506 ausfüllen, um eine exakte Kopie Ihrer Steuererklärung anzufordern, können Sie davon ausgehen, dass es bis zu 75 Kalendertage dauern kann.

### Warum möchte mein Kreditgeber ein Formular 4506-T?

Hypothekengeber verlangen möglicherweise, dass Sie das Formular 4506-T als Teil Ihres Hypothekenantrags ausfüllen. Es ermöglicht ihnen, eine Abschrift Ihrer Steuererklärungsinformationen direkt vom IRS zu erhalten. Sie verwenden es, um die anderen Einkommensunterlagen zu überprüfen, die Sie während des Kreditantragsverfahrens vorgelegt haben.

### Reicht eine Abschrift für meinen Hypothekengeber?

Transkripte können in der Regel als Ersatz für eine exakte Kopie einer Erklärung von Hypothekenbanken verwendet werden. Tatsächlich wurden sie speziell geschaffen, um Hypothekengeber mit den Steuererklärungsinformationen ihrer Kreditnehmer zu versorgen.

### Was ist das IRS-Formular 4506 und 4506-T?

Das IRS-Formular 4506 kann von Steuerzahlern eingereicht werden, um eine exakte Kopie einer zuvor eingereichten Erklärung und Steuerinformationen anzufordern. Für Situationen, in denen keine exakte Kopie einer Steuererklärung erforderlich ist, können die Formulare 4506-T und 4506-T-EZ eingereicht werden, um kostenlose Abschriften einer zuvor eingereichten Steuererklärung zu erhalten.
