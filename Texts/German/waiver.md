# Verzicht
## Was ist eine Verzichtserklärung?

Ein Verzicht ist der freiwillige Verzicht auf ein gesetzliches Recht oder einen Anspruch einer Person in einer Rechtssituation. Durch die Unterzeichnung einer Verzichtserklärung entfernt eine Person eine tatsächliche oder potenzielle Haftung von einer anderen juristischen Person.

## Tiefere Definition

**Haftungsverzicht:** Wenn Personen sich an Aktivitäten beteiligen, die zu einer Verletzung oder zum Tod führen könnten, müssen sie möglicherweise eine Verzichtserklärung unterzeichnen, die eine andere Partei von der Haftung befreit, wenn sie verletzt werden oder sterben.

Die Teilnahme an Sportarten wie Fallschirmspringen, Seilrutschen und Bungie-Jumping sind Aktivitäten, bei denen die teilnehmende Person aufgefordert wird, eine Verzichtserklärung zu unterzeichnen, wodurch das mit der Aktivität verbundene Unternehmen von der Haftung befreit wird.

**Verzicht auf materielle Güter:** Wenn Eigentum rechtmäßig von einer Partei auf eine andere übertragen wird, können Personen auf ihr Recht verzichten, Ansprüche im Zusammenhang mit dem Eigentum geltend zu machen. Die rechtsgültige Übergabe eines Fahrzeugs dient dem Verkäufer als Verzichtserklärung. Durch die Übertragung des Fahrzeugs an einen anderen Elternteil ist der Verkäufer beispielsweise nicht mehr für Handlungen verantwortlich, die am Fahrzeug vorgenommen werden, und hat keine Rechte mehr an dem Fahrzeug.

**Hinweisverzicht:** Durch die Unterzeichnung eines Kündigungsverzichts verzichtet die rechtliche Partei auf das Recht auf formelle Benachrichtigung eines Gerichtsverfahrens. Verzichte auf Mitteilungen werden manchmal in Nachlassverfahren und während Notfallsitzungen verwendet, die von einem Vorstand abgehalten werden. Sie werden häufig verwendet, um ein zeitnahes Einleiten von Gerichtsverfahren zu ermöglichen.

## Verzichtsbeispiel

Amys Tochter macht mit ihrer Klasse einen einwöchigen Ausflug in ein Camp. Die Kinder werden Boot fahren, schwimmen und wandern.

Damit die Kinder teilnehmen können, müssen die Eltern der Kinder eine Verzichtserklärung unterzeichnen, die das Camp von der Haftung befreit, falls das Kind verletzt wird oder stirbt. Amy unterschreibt die Verzichtserklärung und entbindet das Camp von der Verantwortung, wenn ihre Tochter verletzt oder schlimmer wird.



## Höhepunkte

- Ein Verzicht ist eine rechtsverbindliche Bestimmung, bei der sich eine Vertragspartei bereit erklärt, freiwillig auf einen Anspruch zu verzichten, ohne dass die andere Partei haftbar ist.

- Verzichtserklärungen werden unterzeichnet, um das Risiko zu mindern.

- Verzichtserklärungen können entweder in schriftlicher Form oder in Form einer Klage erfolgen.

- Verzichtserklärungen sind üblich, wenn Klagen abgeschlossen werden, da eine Partei nicht möchte, dass die andere sie weiterverfolgt, nachdem ein Vergleich übertragen wurde.

- Beispiele für Verzichtserklärungen sind der Verzicht auf elterliche Rechte, der Haftungsverzicht, der Verzicht auf materielle Güter und der Verzicht auf Unzulässigkeitsgründe.

## FAQ

### Was ist ein Verzicht auf Subrogation?

Ein [Forderungsverzicht](/waiver-of-subrogation) ist ein Verzicht, der eine Person oder ein Unternehmen daran hindert, Schadenersatz von einem Dritten zu verlangen. Verzichtserklärungen finden sich häufig in Bauverträgen, Mietverträgen und Sachversicherungsverträgen. Versicherungsunternehmen fügen üblicherweise Gründe hinzu, die verhindern, dass eine Partei eine Versicherungsanspruchsregulierung erhält, wenn sie auf den Forderungsübergang verzichtet.

### Was ist ein Medicaid-Verzicht?

Ein Medicaid-Verzicht ist ein Verzicht, der von dem Staat unterzeichnet ist, der auf bestimmte Medicaid-Berechtigungsvoraussetzungen verzichten kann. Dies würde dazu führen, dass Menschen Pflege angeboten wird, die sonst möglicherweise keinen Anspruch auf Medicaid hätten. Die Verzichtserklärungen können auf bestimmte Weise eingeschränkt werden und könnten auf bestimmte medizinische Diagnosen beschränkt sein oder geografische Beschränkungen erlassen.

### Was ist ein GAP-Verzicht?

Ein GAP-Verzicht, der für Guaranteed Asset Protection Waiver steht, ist ein Verzicht, der eine Person von den verbleibenden Zahlungen für einen zerstörten Vermögenswert, typischerweise ein Auto, befreit. Dies bedeutet, dass jemand, der Geld für ein Auto schuldet und das Auto über die Bergung hinaus beschädigt wurde, nicht für die verbleibenden Zahlungen verantwortlich ist. Ein GAP-Verzicht könnte auch als Schuldenerlass angesehen werden.

### Was ist ein Pfandrechtsverzicht?

Ein [Zurückbehaltungsverzicht](/lien-waiver) ist ein Verzicht, der das Recht einer Gegenpartei verwirkt, ein Pfandrecht auf das Eigentum oder die Waren eines Zahlers zu stellen. Diese sind in der Baubranche in verschiedenen Bauphasen üblich. Ein Pfandverzicht ist ähnlich wie eine Quittung und kann verhindern, dass ein Pfandrecht hinterlegt wird.

### Was ist eine Gebührenbefreiung?

Ein Gebührenverzicht ist ein Verzicht, der unterzeichnet wird, um den Gebührenbetrag von jemandem, der typischerweise eine Zeit finanzieller Notlage durchmacht, entweder teilweise oder vollständig zu reduzieren. Sie können auch verwendet werden, um einen Käufer oder Dienstleister zu locken, wenn die Gebühr abschreckend sein und den Unterschied zwischen Abschluss oder Verlust des Verkaufs ausmachen könnte.
