# Versatz
## Was ist ein Offset?

Bei einem Offset wird eine entgegengesetzte Position in Bezug auf eine ursprüngliche Eröffnungsposition an den Wertpapiermärkten eingenommen. Wenn Sie beispielsweise 100 Aktien von XYZ long halten, wäre der Verkauf von 100 Aktien von XYZ die Gegenposition. Eine Gegenposition kann auch durch [Sicherungsinstrumente](/hedge) wie Futures oder [Optionen entstehen](/optionscontract).

Auf den [Derivatemärkten geht ein Händler](/derivative) zum Ausgleich einer [Futures](/futures) -Position eine gleichwertige, aber gegensätzliche Transaktion ein, die die [Lieferverpflichtung](/physicaldelivery) des physischen Basiswerts aufhebt. Das Ziel der Aufrechnung besteht darin, die Nettoposition eines Anlegers in einer Anlage auf null zu reduzieren, sodass aus dieser Position keine weiteren Gewinne oder Verluste entstehen.

In der Wirtschaft kann sich ein Ausgleich auf den Fall beziehen, in dem Verluste eines Geschäftsbereichs durch Gewinne eines anderen kompensiert werden. In ähnlicher Weise können Unternehmen den Begriff auch in Bezug auf Enterprise Risk Management ( [ERM](/enterprise-risk-management) ) verwenden, bei dem Risiken, die in einer Geschäftseinheit auftreten, durch gegensätzliche Risiken in einer anderen ausgeglichen werden. Beispielsweise kann eine Einheit dem Risiko eines fallenden Schweizer Frankens ausgesetzt sein, während eine andere von einem fallenden Franken profitieren kann.

## Grundlagen eines Offsets

Die Aufrechnung kann in einer Vielzahl von Transaktionen verwendet werden, um Verbindlichkeiten zu beseitigen oder zu begrenzen. In der Buchhaltung kann eine Buchung durch eine gleiche, aber entgegengesetzte Buchung ausgeglichen werden, die die ursprüngliche Buchung annulliert. Im Bankwesen bietet das Aufrechnungsrecht Finanzinstituten die Möglichkeit, Schuldnervermögen im Falle einer Zahlungsunfähigkeit einzustellen oder eine Pfändung zu beantragen, um geschuldete Gelder einzutreiben. Für Anleger, die an einem Futures-Kontrakt beteiligt sind, beseitigt eine Gegenposition die Notwendigkeit, eine physische Lieferung des zugrunde liegenden Vermögenswerts oder Rohstoffs zu erhalten, indem die zugehörigen Waren an eine andere Partei verkauft werden.

Unternehmen können sich dafür entscheiden, Verluste in einem Geschäftsbereich auszugleichen, indem sie die Gewinne aus einem anderen Bereich umverteilen. Dadurch kann die Rentabilität einer Aktivität die andere unterstützen. Wenn ein Unternehmen auf dem Smartphone-Markt erfolgreich ist und beschließt, ein Tablet als neue Produktlinie zu produzieren, können die durch den Smartphone-Verkauf erzielten Gewinne dazu beitragen, Verluste im Zusammenhang mit der Expansion in eine neue Arena auszugleichen.

Im Jahr 2016 verzeichnete BlackBerry Ltd. erhebliche Verluste bei seinen Mobilitätslösungen und Dienstzugangsgebühren. Die damit verbundenen Rückgänge wurden durch Gewinne in den Bereichen Software und andere Serviceangebote ausgeglichen, was die Gesamtauswirkung auf das Endergebnis von BlackBerry verringerte .

## Aufrechnung in Derivatkontrakten

Anleger gleichen Futures-Kontrakte und andere Anlagepositionen aus, um sich von damit verbundenen [Verbindlichkeiten zu befreien](/liability). Nahezu alle Futures-Positionen werden ausgeglichen, bevor die Bedingungen des [Futures-Kontrakts](/futurescontract) realisiert werden. Auch wenn die meisten Positionen kurz vor der Lieferfrist ausgeglichen werden, werden die Vorteile des Futures-Kontrakts als Absicherungsmechanismus immer noch realisiert.

Der Zweck der Aufrechnung eines Futures-Kontrakts auf eine Ware besteht für die meisten Anleger darin, zu vermeiden, dass sie die mit dem Kontrakt verbundenen Waren physisch erhalten müssen. Ein Futures-Kontrakt ist eine Vereinbarung zum Kauf eines bestimmten Rohstoffs zu einem bestimmten Preis an einem zukünftigen Datum. Wenn ein Kontrakt bis zum vereinbarten Datum gehalten wird, könnte der Anleger für die Annahme der physischen Lieferung des betreffenden Rohstoffs verantwortlich werden.

Auf Optionsmärkten versuchen Trader oft, bestimmte Risiken auszugleichen, die manchmal als ihre „ [Griechen](/greeks) “ bezeichnet werden. Wenn beispielsweise ein Optionsbuch einem Rückgang der impliziten Volatilität (Long- [Vega )](/vega) ausgesetzt ist , kann ein Händler entsprechende Optionen verkaufen, um dieses Risiko auszugleichen. Ebenso kann ein Trader, wenn eine Optionsposition einem direktionalen Risiko ausgesetzt ist, das zugrunde liegende Wertpapier kaufen oder verkaufen, um [deltaneutral zu werden](/deltaneutral). Dynamisches Hedging (oder [Delta-Gamma-Hedging](/deltagamma-hedging) ) ist eine Strategie, die von Derivatehändlern angewendet wird, um regelmäßig Ausgleichspositionen in ihren Büchern zu halten.

## Beispiel für Verrechnungspositionen

Wenn die ursprüngliche Investition ein Kauf war, wird ein Verkauf getätigt, um die Position zu neutralisieren; Um einen anfänglichen Verkauf auszugleichen, wird ein Kauf getätigt, um die Position zu neutralisieren.

Bei auf Aktien bezogenen Futures können Anleger durch Hedging eine gegensätzliche Position einnehmen, um das mit dem Futures-Kontrakt verbundene Risiko zu steuern. Wenn Sie beispielsweise eine [Long-Position](/long) in einer Aktie ausgleichen möchten, können Sie eine identische Anzahl von Aktien leerverkaufen.

## Höhepunkte

- Die Aufrechnung ist als Strategie bei Aktien- und Derivatekontrakten üblich.

- Bei einer Ausgleichsposition nimmt ein Händler eine gleichwertige, aber entgegengesetzte Position ein, um die Nettoposition auf Null zu reduzieren. Der Zweck des Eingehens einer Gegenposition besteht darin, Verbindlichkeiten zu begrenzen oder zu eliminieren.
