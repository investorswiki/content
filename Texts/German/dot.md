# Designated Order Turnaround (DOT (SuperDOT))
## Was ist der Designated Order Turnaround (DOT (SuperDOT))?

Designated Order Turnaround ist ein elektronisches System , das die Effizienz erhöht , indem Orders für börsennotierte [Wertpapiere](/security) direkt an einen Spezialisten auf dem Handelsparkett weitergeleitet werden , statt über einen [Makler](/broker). Designated Order Turnaround wird auch als DOT oder SuperDOT bezeichnet.

## Den Designated Order Turnaround (DOT (SuperDOT)) verstehen

Designated Order Turnaround ist ein früher von der [New York Stock Exchange (NYSE) verwendetes Order-Routing-System,](/nyse) bei dem Orders unter Umgehung des Brokers direkt an einen Spezialisten auf dem Handelsparkett gesendet werden. Seit den 1970er Jahren wurden die meisten Orders an der NYSE über das DOT elektronisch an die Bildschirme der Spezialisten übermittelt. Das DOT-System wird häufig bei Eingaben von kleinen Aufträgen, wie z. B. [Limitaufträgen](/limitorder),. und Korb- und Programmgeschäften verwendet.

Automatisierte Handelssysteme wie SuperDOT sind in der Lage, Orders sowohl schnell als auch genau auszuführen, und diese Systeme tragen dazu bei, die Anzahl der Fehler zu verringern, indem sie menschliche Eingriffe in den Orderabwicklungsprozess eliminieren. Automatisierte Handelssysteme bieten auch eine weitere Sicherheitsebene gegen Betrug und helfen so, das Risiko zu kontrollieren. Das System ermöglicht auch ein größeres Volumen auf dem Parkett, indem Provisionsmakler umgangen werden.

Innerhalb des DOT-Systems gibt der Benutzer, entweder ein Investor oder ein Makler, die Order direkt in das System ein, die dann sofort den Spezialisten erreicht. Sobald der Auftrag ausgeführt ist, erhält der Benutzer in Echtzeit einen Bestätigungsbericht der [Transaktion .](/transaction) Die meisten Privatanleger haben keinen direkten Zugang zum SuperDOT-System, sie greifen jedoch indirekt über Software oder Online-Dienste auf das System zu, die von [Maklerunternehmen angeboten werden](/brokerage-company),. die dann die Kundenaufträge in SuperDOT platzieren.

## Das Super-Schaubuch

Das DOT oder SuperDOT wurde 2009 durch ein System ersetzt, das als Super Display Book (SDBK) bekannt ist. Das SDBK ist das automatisierte System der NYSE, das Aufträge für Wertpapiere anzeigt, aufzeichnet und ausführt.

Das SDBK-Order-Routing-System ist ein ausgeklügeltes Computerprogramm, das die Übertragung von Market- [und](/market) Limit-Orders direkt an den Handelsposten und die designierten [Market Maker ermöglicht](/marketmaker),. wo ein bestimmtes Wertpapier gehandelt wird. Dieses System ermöglicht eine effizientere Transaktion, da die Order direkt an den Market Maker geliefert werden kann, anstatt zu einem Parketthändler telefoniert und manuell ausgeführt zu werden. Automatisierte Systeme wie SDBK führen Aufträge schnell und genau aus und tragen zur Risikokontrolle bei.

[Parkettmakler](/floorbroker) bearbeiten selten Aufträge, die von einzelnen Anlegern aufgegeben werden. Stattdessen werden diese Orders zur sofortigen Ausführung über die SDBK direkt an einen Market Maker geleitet. Parkettmakler bearbeiten normalerweise die größeren, komplexeren institutionellen [Handelsaufträge](/order).
