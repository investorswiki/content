# Null-Kapitalertragsrate
## Was ist eine Null-Kapitalgewinnrate?

Ein Kapitalertragssatz von null impliziert einen Steuersatz von 0 % auf Kapitalerträge. Dieser Satz von 0 % kann Einzelpersonen in Rechnung gestellt werden, die Immobilien innerhalb einer sogenannten „ [Unternehmenszone](/enterprise-zone) “ verkaufen. Eine solche Zone ist ein geografisches Gebiet, dem besondere Steuervergünstigungen, behördliche Ausnahmen oder andere öffentliche Beihilfen gewährt wurden, um die private wirtschaftliche Entwicklung und die Schaffung von Arbeitsplätzen zu fördern. Sie werden am häufigsten eingesetzt, um die Revitalisierung eines Stadtteils zu fördern.

Der Null - [Kapitalgewinnsatz](/capitalgain) kann von einer bestimmten Regierungsebene angewendet werden, um Investitionen in einem bestimmten Bereich anzuregen.

## Eine Kapitalgewinnrate von Null verstehen

Im Jahr 2004 verabschiedete der US-Kongress das Working Families Tax Relief Act, und der Präsident genehmigte es. Das Gesetz enthält Bestimmungen, die die Kapitalertragssteuer von 0 % auf bestimmte Immobilien ausdehnen, die innerhalb bestimmter Gewerbezonen verkauft werden USA in den 1970er Jahren, um die Flucht von Menschen und Unternehmen aus den städtischen Zentren in die Vororte umzukehren. Die Programme können verwendet werden, um ein privates Unternehmen zu ermutigen, in einer Nachbarschaft zu bleiben, dort zu expandieren oder dorthin umzuziehen.

Die Logik hinter diesem Gesetz besteht darin, Einzelpersonen einen Anreiz zu geben, in diesem Bereich zu investieren. Der Satz ist nicht exklusiv für eine Region, ein Bundesland oder eine Gemeinde. Gesetzgeber, die Arbeitsplätze schaffen und Investitionen in eine Gemeinde lenken wollen, erlassen häufig einen Kapitalertragssteuersatz von Null oder führen andere steuerliche Anreize in diesem Bereich ein.

Ein Steuergesetz aus dem Jahr 2012 machte den Kapitalertragssatz von 0 % für die meisten Antragsteller dauerhaft, vorausgesetzt, sie sind entweder Singles mit einem steuerpflichtigen Einkommen von weniger als 37.950 USD oder Paare mit einem steuerpflichtigen Einkommen von weniger als 75.900 USD. Dennoch müssen einige dieser Antragsteller mit bescheidenen Steuersätzen von 25 rechnen % auf 30 %, wenn sie zusätzliches Einkommen erzielen, das zu normalen Sätzen besteuert wird, wodurch ihre langfristigen Gewinne oder qualifizierten [Dividendenerträge](/dividend) von der 0 %-Klammer in die 15 %-Klammer für Kapitalerträge verschoben werden.

Auf der anderen Seite können Einzelabzüge das gewöhnliche Einkommen reduzieren, Einzelpersonen unter die 15 %-Grenze bringen und somit die Kapitalgewinne oder Dividenden erhöhen, die mit 0 % besteuert werden. Dies hilft zu erklären, warum Steuerzahler hohe bereinigte Bruttoeinkommen haben können, aber dennoch mit 0 % Steuern auf ihre langfristigen Kapitalgewinne rechnen müssen.

## Beispiel: Die DC Enterprise Zone

Im Rahmen dieses Programms hat jede Unternehmenszone ihre eigenen besonderen Regeln, die sich ändern können, wenn die Gesetzgebung erweitert oder geändert wird. Bei der DC-Unternehmenszone müssen beispielsweise die folgenden Mandate erfüllt werden:

- Das Eigentum muss während dieser Zeit des Eigentums wesentlich verbessert worden sein.

- Die Immobilie muss seit dem Erwerbsdatum mindestens fünf Jahre bestehen

- Mindestens 80 % des gesamten Bruttoeinkommens, das sich aus dem Immobilienbesitz ergibt, muss aus Geschäften stammen, die aktiv innerhalb der DC-Unternehmenszone betrieben werden.

- Handelt es sich bei der betreffenden Immobilie um eine gewerbliche Vermietung, müssen mindestens 50 % der Mieteinnahmen von Unternehmen stammen, die sich innerhalb der DC-Unternehmenszone befinden.

- Die ursprüngliche Nutzung des Grundstücks muss beim Steuerzahler beginnen; diese Voraussetzung gilt als erfüllt, wenn wesentliche Verbesserungen an der Liegenschaft vorgenommen worden sind.

## Höhepunkte

- Ein Satz von 0 % auf den Verkauf von Immobilien wird meistens mit Unternehmenszonen in Verbindung gebracht, bei denen es sich um spezielle Gebiete handelt, denen von einer Regierung ein Sonderstatus gewährt wird, um Entwicklung und Wirtschaftswachstum zu fördern.

- Um eine Kapitalgewinnrate von Null beizubehalten, müssen Immobilienbesitzer bestimmte Qualifikationen und Anforderungen erfüllen, die je nach Unternehmenszone variieren können.

- Bei einem Kapitalertragssatz von null fallen keine Steuern auf den Verkauf von Vermögenswerten oder Immobilien an, die andernfalls einen Kapitalertrag aufweisen würden.
