# Kaution für Hypothekenzinsen
## Was ist eine Kaution mit Hypothekenzinssperre?

Eine Hypothekenzinssicherung ist eine Gebühr, die ein Kreditgeber erhebt, um einen [Hypothekenzinssatz](/mortgageinterest) für einen bestimmten Zeitraum zu fixieren, mit der Erwartung, dass die Hypothek des Kreditnehmers innerhalb dieses Zeitraums finanziert wird.

Je länger die Sperrzeit, desto höher die erforderliche Sperrkaution. Die Sperrkaution wird dem Kreditnehmer bei der [Hypothekenfinanzierung wieder gutgeschrieben](/mortgage). Wenn der Kreditnehmer den Hypotheken- und Sperrvertrag verlässt, verliert er seine Kaution.

Die Gebühr für eine Ratensperre kann zwischen 0,25 % und 0,5 % des Betrags Ihrer Hypothek liegen. Bei einem Hypothekendarlehen in Höhe von 450.000 US-Dollar zum Beispiel würde eine Kaution von 0,25 % mit Zinsbindung 1.125 US-Dollar betragen.

## Was bewirkt die Kaution mit Hypothekenzinssperre?

Eine Hypothekenzinssperre schützt den Kreditnehmer davor, einen höheren [effektiven Jahreszins](/apr) auf sein Hypothekendarlehen zahlen zu müssen, falls die Zinsen in der Zeit zwischen Kreditgenehmigung und Hypothekenfinanzierung steigen sollten. Kreditnehmer warten oft, bis sie ein Haus zum Kauf gefunden haben, bevor sie eine Anzahlung leisten, um ihre Rate zu fixieren. Sie tun dies, weil die Zeit, die für die Wohnungssuche und die Annahme eines Angebots benötigt wird, ungewiss ist.

Kreditgeber verwenden Hypothekenzinssicherungseinlagen mit [festverzinslichen Hypotheken](/fixed-rate_mortgage),. bei denen die Zinsen an die Renditen von US-Staatsanleihen gebunden sind. 15-jährige Hypotheken sind an die Rendite der 10-jährigen Staatsanleihe gebunden, während 30-jährige Hypotheken der Rendite der 30-jährigen Staatsanleihe entsprechen.

Hypothekenzinsen können beeinträchtigt werden, wenn die Renditen dieser Wertpapiere steigen, entweder aufgrund einer Anhebung der kurzfristigen Zinssätze durch die [Federal Reserve](/federalreservesystem) oder aufgrund von Trends wie einem schnelleren Wirtschaftswachstum oder steigenden Preisen, die dazu führen, dass Anleiheinvestoren in Erwartung einer Inflation höhere Renditen verlangen.

## Beispiel für die Verwendung einer Kaution mit Hypothekenzinssperre

Einlagen mit Hypothekenzinssperre sichern einen bestimmten Zinssatz für ein Darlehen und werden auf der Grundlage eines Zinssatzes von etwa 0,25 % bis 0,50 % des Hypothekenbetrags berechnet. Für eine Hypothek in Höhe von 300.000 USD wäre beispielsweise eine Kaution in Höhe von 750 bis 1.500 USD erforderlich.

Ratensperren dauern normalerweise 30 bis 60 Tage, aber einige Kreditgeber verlängern eine Ratensperre auf 120 Tage oder mehr. Bestimmte Kreditgeber bieten möglicherweise eine kostenlose Ratensperre für einen bestimmten Zeitraum an, erheben dann aber Gebühren für die Verlängerung der Sperre. Kreditnehmer können erst nach ihrer ersten Hypothekengenehmigung einen Zinssatz festlegen.

## Einschränkungen einer Kaution mit Hypothekenzinssperre

Eine Einlage mit Hypothekenzinssperre kann Kreditnehmern Hunderte, wenn nicht Tausende von Dollar an Hypothekenzinsen in Zeiten schnell steigender Zinssätze ersparen, aber der Prozess birgt auch Risiken.

Eine zu frühe Bindung kann dazu führen, dass ein Kreditnehmer einen besseren Zinssatz verpasst, der möglicherweise vor dem Abschluss verfügbar ist. Darüber hinaus kann ein Kreditnehmer gezwungen sein, eine zusätzliche Kaution zu zahlen, um die Sperre nach Ablauf zu verlängern. Eine Zinssperre kann auch aufgehoben werden, wenn sich die finanziellen Verhältnisse des Kreditnehmers vor dem Abschluss ändern, wie [z](/dti).

## Höhepunkte

- Wenn die Zinssätze sinken, nachdem Sie bezahlt haben, um einen bestimmten Zinssatz festzulegen, kann Ihr Kreditgeber Ihnen eine zusätzliche Gebühr für den Wechsel zu einem niedrigeren Zinssatz in Rechnung stellen, oder Sie könnten auf dem höheren Zinssatz und dem Verlust Ihrer Einlage sitzen bleiben, wenn Sie gehen.

- Die Verwendung einer Kaution mit Hypothekenzinssperre kann Ihnen Sicherheit geben.

- Eine Ratensperre lässt Sie wissen, wie hoch Ihre Hypothekenzahlungen sein werden, und hilft Ihnen, ein entsprechendes Budget für den Kauf eines neuen Eigenheims zu erstellen.
