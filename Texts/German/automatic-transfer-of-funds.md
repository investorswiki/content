# Automatischer Geldtransfer
## Was ist eine automatische Geldüberweisung?

Eine automatische Geldüberweisung ist eine bestehende Bankvereinbarung, bei der Überweisungen zwischen den zwei (oder mehr) Konten eines Kunden regelmäßig und periodisch unter festgelegten Bedingungen durchgeführt werden. Automatische Überweisungen werden ohne weitere Anweisung oder Handlung des Kunden ausgeführt; Eine übliche Art und Weise, wie automatische Überweisungen ausgeführt werden, sind "Sweep"-Anweisungen, bei denen alle überschüssigen Gelder auf einem Konto auf ein anderes Konto übertragen werden. Der automatische Geldtransfer ist ein Kernangebot sowohl von [Geschäfts-](/commercialbank) als auch von [Online-Banken](/onlinebanking).

## So funktioniert eine automatische Überweisung

Automatische Überweisungen werden häufig für die regelmäßige Überweisung von Geldern von einem [Girokonto](/checkingaccount) auf ein [Sparkonto verwendet](/savingsaccount). Automatische Überweisungen können verwendet werden, um Geld zwischen den Konten zweier Ehepartner oder zwischen einem Elternteil und einem Kind zu überweisen. Das Einrichten automatischer Überweisungen zur Begleichung von Rechnungen ist ein nützliches Budgetierungstool, da sie für regelmäßige gleiche Zahlungen verwendet werden können, z. B. für Hypotheken oder andere Darlehenszahlungen.

Eine weitere häufige Verwendung dieser Überweisungen ist der [Überziehungsschutz](/overdraft-protection),. bei dem Gelder von einem Konto mit höheren Zinsen verschoben werden, um fällige Zahlungen auf einem anderen Konto zu decken.

Unternehmen mit mehreren Tochtergesellschaften verwenden manchmal ein Nullkonto (ZBA). Eine ZBA veranlasst häufig die automatische Überweisung, da es sich um ein Girokonto handelt, auf dem ein Saldo von Null geführt wird. Wenn in der ZBA Gelder benötigt werden, wird der exakt benötigte Geldbetrag durch eine automatische Überweisung von Geldern von einem zentralen oder Hauptkonto auf das Konto überwiesen.

Unternehmen können auch eine ZBA verwenden, wenn ihre Mitarbeiter über Kredit- oder Debitkarten des Unternehmens verfügen, da dies eine größere Kontrolle in Bezug auf die Verteilung von Unternehmensgeldern ermöglicht (zusammen mit der Begrenzung von Überschussguthaben). Ein ZBA-Konto kann dabei helfen, sicherzustellen, dass Manager alle Aktivitäten mit Debit-/Kreditkarten des Unternehmens vorab genehmigen. Nach der Genehmigung wird eine automatische Überweisung von Geldern von einem Hauptkonto in Höhe von Beträgen initiiert, die gerade groß genug sind, um die angezeigten Gebühren zu decken.

## Besondere Überlegungen

Bei Online-Banken ist die Cybersicherheit besonders wichtig geworden, um zu verhindern, dass elektronische Informationen anfällig für Beschädigung oder Diebstahl werden. Wenn Informationen über Netzwerke übertragen werden, z. B. bei einer automatischen Geldüberweisung, können Cyberangriffe auftreten.

Cyberangriffe können in einer Reihe von Formen auftreten, darunter Backdoor-Angriffe, bei denen ein Dieb eine alternative Methode für den Zugriff auf ein System ausnutzt; Denial-of-Service-Angriffe, die einen rechtmäßigen Benutzer daran hindern, auf ein System zuzugreifen; und Angriffe mit direktem Zugriff, einschließlich Bugs und Viren, die sich Zugang zu einem System verschaffen und dessen Informationen kopieren und/oder das System modifizieren.

## Höhepunkte

- Automatische Überweisungen können verwendet werden, um Geld von einem Bankkonto auf ein anderes zu überweisen, z. B. von einem Girokonto auf ein Sparkonto.

- Eine automatische Geldüberweisung ist eine bestehende Bankvereinbarung, bei der regelmäßig Überweisungen vom Konto eines Kunden vorgenommen werden.

- Fast alle stationären Banken und Online-Banken bieten ihren Kunden automatische Überweisungsdienste an.
