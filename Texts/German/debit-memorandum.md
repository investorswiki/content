# Lastschrift
## Was ist eine Lastschrift?

Eine Lastschrift oder "Lastschrift" ist ein Dokument, das einen Kunden über [Belastungsanpassungen aufzeichnet und benachrichtigt](/debit),. die auf seinem individuellen Bankkonto vorgenommen wurden. Die auf dem Konto vorgenommenen Anpassungen verringern das Guthaben auf dem Konto, werden jedoch für bestimmte Zwecke vorgenommen und nur für Anpassungen außerhalb normaler Belastungen verwendet. Die Gründe für die Ausstellung einer Lastschrift beziehen sich auf [Bankgebühren](/bank-fees),. zu wenig berechnete Rechnungen oder die Korrektur versehentlich positiver Salden auf einem Konto. Das Gegenteil einer Lastschrift ist eine Gutschrift.

## Eine Lastschrift verstehen

Eine Belastungsanzeige wird für drei allgemeine Fälle ausgestellt: eine Verringerung des Kontostands eines Bankkunden aufgrund von Gebühren und anderen damit zusammenhängenden Gebühren, eine Unterfakturierung von Waren oder Dienstleistungen von einem Verkäufer an einen Käufer oder eine interne Verrechnung mit einem geringfügigen Guthaben in ein Kundenkonto. Diese Arten von Situationen werden typischerweise als Banktransaktionen, inkrementelle Abrechnung bzw. interne Verrechnungen bezeichnet.

### Banktransaktionen

Beim [Privatkundengeschäft](/retailbanking) erhält ein Kontoinhaber eine Belastungsanzeige, aus der hervorgeht, dass ein Kontostand aus einem anderen Grund als einer Bargeldabhebung, einem eingelösten Scheck oder der Verwendung einer Debitkarte verringert wurde. Belastungsanzeigen können aufgrund von Bankgebühren, Gebühren für ungedeckte [Schecks](/bouncedcheck) oder Gebühren für das Drucken weiterer Schecks entstehen. Die Mahnungen werden typischerweise zusammen mit den monatlichen Kontoauszügen an Bankkunden versandt und die Lastschrift wird durch ein negatives Vorzeichen neben der Belastung gekennzeichnet.

### Inkrementelle Abrechnung

Im Business-to-Business-Geschäft wird eine Lastschrift als Berichtigungsverfahren nach einer versehentlichen Unterabrechnung von Waren oder Dienstleistungen an einen Kunden ausgestellt. Es soll einen Abrechnungsfehler korrigieren. Im formellen Sprachgebrauch teilt es einem Kunden mit, dass die Lastschrift seine [Verbindlichkeiten erhöhen wird](/accountspayable). Wenn beispielsweise ABC Co. eine Bestellung für XYZ Inc. ausführt und dem Kunden einen Betrag in Rechnung stellt, der unter dem vereinbarten Betrag liegt, stellt ABC Co. XYZ Inc. eine Lastschrift aus, um den fälligen Restbetrag anzugeben und zu erläutern.

### Interner Offset

Innerhalb einer Firma kann eine Lastschrift erstellt werden, um ein Guthaben zu verrechnen, das auf einem Kundenkonto vorhanden ist. Wenn ein Kunde absichtlich oder unabsichtlich mehr als einen Rechnungsbetrag zahlt, kann das Unternehmen eine Lastschrift ausstellen, um die Gutschrift auszugleichen und den positiven Saldo zu eliminieren. Wenn das Guthaben als wesentlich angesehen wird, würde das Unternehmen dem Kunden höchstwahrscheinlich eine Rückerstattung gewähren, anstatt eine Lastschrift zu erstellen.

## Höhepunkte

- Banktransaktionen beziehen sich auf Gebühren oder Servicegebühren, inkrementelle Abrechnung erfolgt, wenn einem Kunden versehentlich zu wenig berechnet wurde, und interne Verrechnungen dienen zum Ausgleich positiver Salden.

- Die drei Hauptgründe für die Ausstellung einer Lastschrift sind Banktransaktionen, Teilrechnungen oder interne Verrechnungen.

- Eine Belastungsanzeige ist eine Benachrichtigung an einen Kunden, dass eine Belastungskorrektur auf seinem Konto vorgenommen wurde, wodurch der Wert der verfügbaren Mittel verringert wird.

- Belastungsanzeigen werden nicht für normale Belastungsvorgänge auf einem Konto ausgestellt, wie z. B. das Einlösen eines Schecks oder die Verwendung einer Debitkarte.
