# Schnäppchenverkauf an eine gemeinnützige Organisation
## Was ist ein Schnäppchenverkauf?

Ein Schnäppchenverkauf bezieht sich auf den Verkauf einer Ware oder Dienstleistung für einen Betrag, der unter dem [fairen](/fairmarketvalue) [Marktwert liegt](/fairmarketvalue). Es kann als Spende an eine Wohltätigkeitsorganisation angesehen werden und zu einer steuerlichen Abschreibung führen.

## Tiefere Definition

Ein Schnäppchenverkauf ist ein Vertrag, bei dem Waren zu niedrigeren Preisen als normal verkauft werden. Sie werden normalerweise von Spendern verwendet, die beabsichtigen, Sachspenden an Wohltätigkeitsorganisationen zu leisten. Gelegentlich kann die übertragene Immobilie gegen eine ähnliche Immobilie mit einem niedrigeren Wert eingetauscht werden. Die Differenz gilt als Geschenk.

Wenn die getauschte Immobilie einen geschätzten Wert hat, muss die Kostenbasis proportional zwischen dem gehandelten Teil der Immobilie und dem gespendeten Rest aufgeteilt werden. Der Gewinn aus dem verkauften Eigentum wird als Einkommen ausgewiesen, während der gewährte Teil als Spende abgeschrieben wird (im Rahmen der Beiträge an Wohltätigkeitsorganisationen für geschätztes Eigentum).

Es gibt keine festgelegten Grenzen für die Art von Vermögenswerten, die sich für den Verkauf zu einem günstigen Preis qualifizieren. Ein Spender, der im Rahmen dieses Programms eine Immobilie an eine gemeinnützige Organisation verkauft, hat Anspruch auf einen Steuerabzug für die Differenz zwischen dem geschätzten Wert und dem Verkaufspreis. Zu den Hauptvorteilen der Durchführung von Schnäppchengeschäften gehören:

- Der Verkauf bietet eine schnelle Methode, um Bargeld zu erhalten, das in andere profitablere Immobilien und Unternehmungen reinvestiert werden kann.

- Die Abzüge vom gespendeten Vermögen können die Steuerbelastung des Verkäufers senken.

- Wohltätigkeitsorganisationen profitieren von den gespendeten Vermögenswerten, die ihre Mission und Aktivitäten fördern.

- Der Verkäufer zahlt keine Kapitalertragssteuer auf das Geschenk an die Wohltätigkeitsorganisation.

## Schnäppchenverkauf Beispiel

Ein Lebensmittelgeschäft verkauft Lebensmittel für 10.000 US-Dollar an ein gemeinnütziges Kinderheim, aber die Produkte sind 20.000 US-Dollar wert. In diesem Fall soll das Lebensmittelgeschäft ein Schnäppchen gemacht haben, weil es die Ware zu einem niedrigeren Preis als dem Marktwert verkauft hat. Da die Lebensmittel an ein gemeinnütziges Kinderheim verkauft werden, gilt der Differenzbetrag als Spende für wohltätige Zwecke.

Hier sind einige Tipps, wie Sie Steuerabzüge erhalten, wenn Sie einen Schnäppchenverkauf durchführen.

## Höhepunkte

- Ein gewöhnlicher Schnäppchenverkauf ist eine Übertragung von Immobilien an eine Wohltätigkeitsorganisation.

- Ein Schnäppchenverkauf verringert die Steuerschuld des Spenders, da er als steuerlich absetzbar gilt.

- Ein Schnäppchenverkauf an eine Wohltätigkeitsorganisation ist der Verkauf einer Ware oder Dienstleistung an eine Wohltätigkeitsorganisation zu einem Betrag, der unter dem angemessenen Marktwert liegt.

## FAQ

### Ist die Spende aus einem Schnäppchenverkauf steuerlich absetzbar?

Ja, es kann sein, aber es unterliegt bestimmten Regeln. Wenn beispielsweise eine zu verkaufende Immobilie seit dem Erwerb durch den Verkäufer an Wert gewonnen hat, muss die Kostenbasis anteilig zwischen dem verkauften und dem geschenkten Teil aufgeteilt werden. Das Internal Revenue Code legt fest, wie dies zu tun ist, um sich für einen Steuerabzug zu qualifizieren.

### Was ist ein Schnäppchenverkauf an eine gemeinnützige Organisation?

Ein Schnäppchenverkauf an eine Wohltätigkeitsorganisation liegt vor, wenn eine Person, die eine Sachspende für wohltätige Zwecke leisten möchte, ein Gut oder eine Dienstleistung unter ihrem angemessenen Marktwert an die Organisation verkauft. Die Preisdifferenz bildet die Spende.

### Muss es immer ein Verkauf sein?

Nicht immer. Ein Spender könnte eine Immobilie im Austausch dafür überschreiben, dass die Wohltätigkeitsorganisation eine andere Immobilie, die sie besitzt, an den Spender überschreibt. Solange das Eigentum des Spenders mehr wert war als das der Wohltätigkeitsorganisation, würde der Tausch einen Schnäppchenverkauf darstellen.
