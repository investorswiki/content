# Manager der Manager (MoM)
## Was ist Manager of Managers (MoM)?

Ein Manager-of-Managers-Ansatz (MoM) ist eine Art Aufsichtsanlagestrategie, bei der ein Manager Manager für ein Anlageprogramm auswählt und ihre Leistung regelmäßig überwacht.

## Manager of Managers verstehen (MoM)

Ein Manager-of-Manager-Ansatz wird typischerweise in institutionellen Anlageprogrammen verwendet. Sie unterscheidet sich von einer [Dachfondsstrategie dadurch](/fundsoffunds),. dass es sich um umfassende Anlageprogramme und nicht um einzelne Investmentfondsprodukte handelt.

Institutionelle Programmmanager können für die Verwaltung von Vermögenswerten für eine Reihe unterschiedlicher Zwecke verantwortlich sein. Pensionsfonds und Altersvorsorgepläne sind einige der häufigsten Programme. Im Allgemeinen setzen die meisten institutionellen Kunden eine Manager-of-Manager-Strategie ein. Institutionelle Manager von Manager-Investitionsprogrammen auf dem Markt werden von Altersvorsorgeplänen, [Stiftungen](/endowment),. Stiftungen, Regierungen und Unternehmen genutzt.

Eine Manager-of-Manager-Strategie ermöglicht es dem Manager, einen definierten Rahmen für Vermögensanlagen festzulegen. Institutionelle Manager, die institutionelle Anlageprogramme beaufsichtigen, können dann aus einer breiten Palette von Angeboten auf dem Markt auswählen, um sie an bestimmte Portfolioallokationen anzupassen.

## Institutionelle Investitionsprogramme

Die meisten institutionellen Anlageprogramme verwenden eine Manager-of-Manager-Strategie, um Vermögenswerte umfassend zu verwalten. In der Regel handelt es sich dabei um ein Kuratorium, das von der Institution als Manager eingesetzt wird. Eine Manager-of-Manager-Strategie ermöglicht es einer Institution, mit mehreren institutionellen Anlageverwaltern zusammenzuarbeiten, um ein Anlageengagement für ein vorgegebenes Vermögensallokationsprogramm zu erreichen.

Institutionelle Kunden, die diese Strategie anwenden, investieren in [institutionelle Anteilsklassen](/institutionalshares) und institutionelle Fonds, die von Anlageverwaltern angeboten werden. Sie können auch mit einem Anlageverwalter zusammenarbeiten, um Vermögenswerte auf einem separaten Konto zu verwalten. Im Rahmen einer Manager-of-Managers-Strategie trifft sich der institutionelle Kundenverwalter mit den Anlageverwaltern zu Sitzungen der Regulierungsbehörden und erhält auch Statusberichte zu den Anlagen. Institutionelle Manager verfolgen die Leistung jedes [Anlageverwalters](/investment-manager) und haben die Befugnis, leistungsschwache Manager zu ersetzen oder Änderungen an der Anlageallokation auf der Grundlage des umfassenden Programms vorzunehmen.

## Beispiel für den Manager of Managers (MoM)-Ansatz

Betrachten Sie als Beispiel für einen Manager-of-Manager-Ansatz eine Lehrergewerkschaft. Diese Gruppe hat ein Kuratorium, das das Investitionsprogramm für den [Pensionsplan der Gewerkschaft beaufsichtigt](/pensionplan). Der Stiftungsrat gilt als Geschäftsführer. Sie legen ein angemessenes Portfolio mit Allokationen auf verschiedene Sektoren und Marktsegmente fest. Der Manager der Manager stellt dann eine Reihe von Anlageverwaltern ein, um Vermögenswerte in den verschiedenen Kategorien zu verwalten, und investiert Teile des Pensionsfondsvermögens bei verschiedenen Anlageverwaltern. Fonds können mehreren Kategorien zugeordnet werden, darunter Geldmarktfonds, Rentenfonds und Aktienfonds.

Jeder Manager ist für die Verwaltung des jeweiligen Investmentfonds verantwortlich, für den er Dienstleistungen erbringt. Der Manager der Manager ist dafür verantwortlich, dass sie so effektiv wie möglich eingesetzt werden. Da kein einzelner Manager ein Experte für Investitionen in alle Anlageklassen ist, ermöglicht die Verwendung einer Manager-of-Manager-Strategie den Kunden, jederzeit einen erfahrenen Vermögensverwalter an jedem Aspekt einer Anlage zu haben.
