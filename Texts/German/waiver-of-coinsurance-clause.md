# Verzicht auf die Mitversicherungsklausel
## Was ist eine Klausel zum Verzicht auf die Mitversicherung?

Eine Klausel zum Verzicht auf Mitversicherung ist eine Bestimmung in einem [Versicherungsvertrag](/insurance),. die besagt, dass der Versicherer unter bestimmten Bedingungen nicht verlangt, dass der Versicherungsnehmer eine [Mitversicherung](/coinsurance) oder einen Prozentsatz des Gesamtanspruchs zahlt.

Diese Klauseln sind am häufigsten in der [Sachversicherung](/property-insurance) zu finden, können aber auch für die [Krankenversicherung](/healthinsurance) und in ziemlich seltenen Fällen für andere Versicherungsarten gelten.

## Wie eine Klausel zum Verzicht auf die Mitversicherung funktioniert

Eine Person oder ein Unternehmen mit einer Sachversicherung erhält möglicherweise nur eine Deckung von 80 %, was bedeutet, dass sie die restlichen 20 % der Mitversicherung zahlen muss, falls etwas mit ihrem Eigentum passiert und sie berechtigt ist, einen gültigen [Anspruch](/insurance_claim) auf Entschädigung geltend zu machen. Ein Verzicht auf die Mitversicherungsklausel verzichtet auf diese Verpflichtung des Versicherungsnehmers, die Last zu tragen und einen Teil der entstandenen Kosten aus eigener Tasche zu zahlen.

Im Allgemeinen neigen Versicherungsunternehmen dazu, nur bei kleineren Schadensfällen auf eine Mitversicherung zu verzichten. Teilweise können Policen aber auch einen Verzicht auf die Mitversicherung im Falle eines [Totalschadens beinhalten](/actual-total-loss).

Die spezifische Sprache, die Versicherungsunternehmen in schriftlichen Verzichtsklauseln verwenden, kann variieren, obwohl sie theoretisch alle ähnlich sind. Typischerweise können Verbraucher damit rechnen, höhere [Versicherungsprämien](/insurance-premium) für Policen mit Verzicht auf die Mitversicherungsklausel zu zahlen, da die Versicherungsgesellschaft dadurch stärker [haftbar gemacht wird.](/liability)

> ### Wichtig

> Versicherungen verzichten in der Regel nur bei kleineren Schäden auf eine Mitversicherung.

>

## Beispiel für eine Klausel zum Verzicht auf die Mitversicherung

Ein Mitversicherungsverzicht ist für einen Versicherungsnehmer im Falle eines Totalschadens besonders wertvoll. Angenommen, eine Mitversicherungsklausel verlangt, dass ein Versicherungsnehmer mindestens 80 % des tatsächlichen Werts der Immobilie versichert. Wenn also ein Gebäude 200.000 US-Dollar wert ist, sollte der Eigentümer eine Versicherung im Wert von mindestens 160.000 US-Dollar abschließen.

Im Falle eines Totalschadens würde die Police die 160.000 US-Dollar auszahlen und der Gebäudeeigentümer wäre für die verbleibenden 40.000 US-Dollar verantwortlich. Das würde sich natürlich ändern, wenn die Police einen Verzicht auf die Mitversicherungsklausel enthalten würde, in diesem Fall würde die Versicherungsgesellschaft die Rechnung für die gesamten 200.000 US-Dollar übernehmen.

## Besondere Überlegungen

Wie bereits erwähnt, kann eine Klausel zum Verzicht auf die Mitversicherung manchmal auf die Krankenversicherung sowie gelegentlich auf andere Arten von Versicherungsprodukten angewendet werden.

Einige Krankenversicherungen sind 80/20-Pläne, was bedeutet, dass der Versicherte 20 % der medizinischen Kosten übernimmt, während die Versicherungsgesellschaft die restlichen 80 % übernimmt – vorausgesetzt, der Kunde zahlt die Selbstbeteiligung.

In dem seltenen Szenario, dass eine Klausel zum Verzicht auf die Mitversicherung angewendet wird, würde dies die erforderliche Zahlung von 20 % durch den Versicherten in bestimmten Situationen eliminieren. Mit anderen Worten, sollte ein Patient eine Operation im Wert von 80.000 US-Dollar benötigen, würde ein Verzicht auf die Mitversicherung, die dieses Verfahren abdeckt, den Patienten davor bewahren, 16.000 US-Dollar für die Mitversicherung auszugeben.

Ähnlich wie bei der Sachversicherung deckt ein Selbstbehaltsverzicht im Gesundheitswesen jedoch oft weitaus geringere Beträge ab. Sie kommen typischerweise zum Einsatz, wenn Patienten zum Zeitpunkt der Entbindung bestimmte, relativ kostengünstige Leistungen im Voraus bezahlen.

## Höhepunkte

- Policen mit Verzicht auf Mitversicherungsklauseln haben tendenziell höhere Versicherungsprämien.

- Diese Klauseln können für Sachversicherungen, Krankenversicherungen oder andere Versicherungsarten gelten.

- Eine Klausel zum Verzicht auf Mitversicherung bezieht sich auf eine Formulierung in einer Versicherungspolice, die Bedingungen festlegt, unter denen Versicherungsnehmer einen Teil eines Anspruchs nicht bezahlen müssen.
