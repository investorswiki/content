# Total Return
Return on investment incorporating price appreciation with reinvested dividends or income over a specific period of time.

## Features
- Total return is the genuine rate of return of an investment or a pool of investments over a period.
- Total return is communicated as a percentage of the amount invested.
- Total return is a strong measure of an investment's overall performance.
- Total return incorporates interest, capital gains, dividends, and realized distributions.
