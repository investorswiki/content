# GitHub
GitHub ialah platform pembangunan sumber terbuka berasaskan web dan perkhidmatan pengehosan yang membolehkan pengguna memuat naik fail, dokumen dan kod komputer ke akaun mereka. GitHub membenarkan pengguna berdaftar dan tidak berdaftar untuk melihat, memuat turun dan juga menyumbang kepada fail yang dimuat naik dan repositori kod sumber. Oleh kerana kos dan cirinya yang rendah, GitHub telah menjadi platform kod sumber terbesar di dunia, digunakan oleh banyak pembangun sebagai cara untuk melaksanakan kerja kolaboratif, di samping dapat berkongsi pelbagai perisian, fail dan dokumentasi.

#### Cara GitHub berfungsi

Setelah akaun GitHub didaftarkan, pengguna boleh mencipta direktori fail dan kandungan mereka sendiri, yang dipanggil repositori. Selepas mencipta repositori, seseorang dapat mencipta cawangan induk yang dipanggil. Cawangan induk GitHub mewakili titik permulaan tunggal bagi mana-mana projek. Terpulang kepada pengguna untuk menyediakan projek kepada sesiapa sahaja (repositori awam) atau menyekat akses kepadanya dan berkongsi hanya dengan pengguna tertentu (repositori peribadi). Daripada cawangan induk, cawangan kecil boleh dibuat, yang pada asasnya membolehkan pengguna bekerja pada pelbagai aspek projek secara berasingan. Pada repositori awam, pengguna lain boleh mencadangkan perubahan pada kod, akhirnya memuat naik versi baharu fail. Cadangan dibuat melalui ciri yang dipanggil permintaan tarik, yang membolehkan pembangun membincangkan dan menyemak kemungkinan perubahan sebelum ia digunakan dengan berkesan. Platform GitHub menawarkan pelbagai ciri dan alatan yang memudahkan pembangun membuat atau mengembalikan perubahan dan menjejaki kemajuan keseluruhan mereka.

#### Apakah permintaan tarik?

Permintaan tarik membolehkan pengguna memberitahu orang lain tentang perubahan yang telah mereka buat pada cawangan atau repositori tertentu. Apabila permintaan tarik dibuat, pengguna dibawa ke skrin gambaran keseluruhan di mana dia boleh menyemak semua perubahan yang dibuat pada kod dengan mudah, membandingkan versi lama dan baharu. Oleh itu, semua perubahan didokumenkan dan direkodkan dan pengguna juga boleh menambah ulasan, label dan peristiwa penting atau bahkan memberikan tugasan khusus kepada penyumbang lain.

#### Siapa yang menggunakan GitHub?

Oleh kerana platformnya yang fleksibel, GitHub mempunyai pengguna dari kedua-dua sektor awam dan swasta. Syarikat perisian, pembangun aplikasi, pengaturcara individu dan pembangun mata wang kripto hanyalah beberapa contoh. Sebagai projek sumber terbuka, [Bitcoin GitHub](/bitcoin) mempunyai banyak pembangun di seluruh dunia, dengan ramai orang menyumbang kepada semakan dan penambahbaikan kod sumber Bitcoin.

#### Adakah GitHub percuma?

GitHub adalah percuma untuk semua projek pembangunan sumber terbuka dan komersial. Walau bagaimanapun, pelan percuma mengehadkan bilangan repositori peribadi dan awam yang boleh dibuat oleh akaun. Pelan percuma juga mengehadkan bilangan kolaborator yang boleh dimiliki oleh projek. Oleh itu, projek yang lebih besar mungkin perlu menaik taraf kepada pelan GitHub berbayar, mengikut keperluan mereka.
