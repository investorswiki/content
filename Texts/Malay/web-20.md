# Web 2.0
## Apakah itu Web 2.0?

Web 2.0 menerangkan keadaan semasa internet, yang mempunyai lebih banyak kandungan yang dijana pengguna dan kebolehgunaan untuk pengguna akhir berbanding dengan penjelmaan sebelumnya, Web 1.0. Secara umum, Web 2.0 merujuk kepada aplikasi Internet abad ke-21 yang telah mengubah era digital selepas [gelembung dotcom](/dotcom-bubble).

## Memahami Web 2.0

Web 2.0 tidak merujuk kepada sebarang peningkatan teknikal khusus kepada internet. Ia hanya merujuk kepada anjakan dalam cara internet digunakan pada abad ke-21. Pada zaman baharu, terdapat tahap perkongsian maklumat dan kesalinghubungan yang lebih tinggi di kalangan peserta. Versi baharu ini membolehkan pengguna mengambil bahagian secara aktif dalam pengalaman dan bukannya hanya bertindak sebagai penonton pasif yang mengambil maklumat.

## Web 1.0 lwn. Web 2.0

Web 1.0 digunakan untuk menerangkan peringkat pertama Internet. Pada ketika ini, terdapat sedikit pencipta kandungan; kebanyakan mereka yang menggunakan Internet adalah pengguna. Halaman statik adalah lebih biasa daripada [HTML dinamik](/html),. yang menggabungkan tapak web interaktif dan animasi dengan pengekodan atau bahasa tertentu. Kandungan dalam peringkat ini datang daripada sistem fail pelayan dan bukannya sistem pengurusan pangkalan data. Pengguna dapat menandatangani buku pelawat dalam talian, dan borang HTML dihantar melalui e-mel.

Contoh tapak Internet yang dikelaskan sebagai Web 1.0 ialah Britannica Online, tapak web peribadi dan mp3.com. Secara amnya, laman web ini adalah statik dan mempunyai fungsi dan fleksibiliti terhad.

Istilah Web 2.0 mula digunakan pada tahun 1999 apabila Internet berputar ke arah sistem yang melibatkan pengguna secara aktif. Pengguna digalakkan untuk menyediakan kandungan, bukannya hanya melihatnya. Orang ramai kini dapat menerbitkan artikel dan ulasan, dan menjadi mungkin untuk membuat akaun pengguna di tapak yang berbeza, oleh itu meningkatkan penyertaan. Web 2.0 turut melahirkan apl web, platform penerbitan sendiri seperti [WordPress](/wordpress-cms),. serta tapak [media sosial](/social-media).

Contoh tapak Web 2.0 termasuk Wikipedia, Facebook, Twitter, dan pelbagai blog, yang semuanya telah mengubah cara maklumat yang sama dikongsi dan dihantar.

Aspek sosial Internet telah diubah terutamanya; secara amnya, media sosial membenarkan pengguna untuk melibatkan diri dan berinteraksi antara satu sama lain dengan berkongsi pemikiran, perspektif dan pendapat. Pengguna boleh tag, kongsi, tweet dan suka.

## Kelebihan dan Kelemahan Web 2.0

Perkembangan teknologi telah membolehkan pengguna berkongsi pemikiran dan pendapat mereka dengan orang lain, yang telah mencipta cara baharu untuk mengatur dan berhubung dengan orang lain dan menggalakkan tahap kerjasama yang lebih tinggi.

Tetapi terdapat banyak keburukan Internet bertindak lebih seperti forum terbuka. Melalui pengembangan media sosial, kami telah melihat peningkatan dalam penjejakan dalam talian, buli siber, [doxing](/personally-identifiable-information-pii),. [kecurian identiti](/identitytheft) dan jenayah dalam talian yang lain. Terdapat juga ancaman penyebaran maklumat salah dalam kalangan pengguna, sama ada melalui laman perkongsian maklumat sumber terbuka atau di media sosial.

Seperti Web 1.0, Web 2.0 adalah satu lagi fasa peralihan dalam evolusi Internet. Web 3.0 diramalkan dipanggil Web Semantik kerana ia akan disesuaikan untuk menjadi lebih intuitif kepada setiap keperluan pengguna.
