# lokap token
Istilah lokap token merujuk kepada tempoh masa tertentu di mana token mata wang kripto tidak boleh diurus niaga atau didagangkan. Lazimnya, lokap ini digunakan sebagai strategi pencegahan untuk mengekalkan nilai jangka panjang yang stabil bagi aset tertentu. Ini boleh membantu untuk menghalang pemegang beg besar untuk menjual token mereka sekaligus di pasaran, yang mungkin menyebabkan harga meningkat dengan [cepat](/tank).

Adalah perkara biasa untuk melihat penjualan besar-besaran selepas Tawaran Syiling Permulaan (ICO) di mana pelabur awal (atau bahkan pasukan projek) akhirnya menjual pegangan mereka sejurus selepas mata wang kripto mencecah pasaran, menyebabkan kejatuhan besar-besaran dalam harga. Jadi lokap token digunakan untuk mengelakkan perkara ini daripada berlaku dan ia membawa tahap keyakinan tambahan kepada bakal peserta [jualan token](/token-sale).

Lokap token juga boleh dipanggil tempoh letak hak. Ini selalunya ditetapkan sebagai satu atau dua tahun selepas pelancaran mata wang kripto. Contohnya, jika syarikat permulaan mencipta mata wang kripto dan melancarkannya melalui ICO, mereka mungkin menentukan tempoh lokap untuk pasukan selama dua tahun, bermakna tiada ahli pasukan akan dapat mengakses token mereka sebelum tempoh lokap tamat. Ini membawa sentimen positif tentang projek dan pasukan kerana ia berkemungkinan akan membuatkan mereka bermotivasi untuk fokus pada kerja jangka panjang, tanpa perlu risau tentang nilai pasaran token mereka.

Perlu diingat bahawa token terkunci (atau [syiling](/coin) ) bukan sebahagian daripada [bekalan yang beredar](/circulating-supply) dan oleh itu tidak dipertimbangkan dalam analisis teknikal yang dilakukan oleh carta dan pedagang.
