# Kredit Cukai Arak Hitam
## Apakah Kredit Cukai Arak Hitam?

Apa yang dipanggil kredit cukai minuman keras hitam adalah kelemahan cukai dalam Kredit Campuran Bahan Api Alternatif (AFMC) yang telah dieksploitasi oleh beberapa syarikat dalam industri hasil hutan mulai tahun 2005.

Kredit cukai persekutuan bertujuan untuk menggalakkan syarikat menggunakan [biofuel](/biofuel) dengan memberi ganjaran kepada mereka dengan pencampuran bahan api fosil. Kelemahan itu membenarkan syarikat kertas, yang sudah menggunakan biofuel yang dikenali sebagai minuman keras hitam, melakukan sebaliknya daripada apa yang dimaksudkan oleh rang undang-undang ituâ€”menambahkan diesel kepada minuman keras hitam mereka untuk melayakkan diri mendapat kredit cukai berbilion dolar. Kredit telah dilanjutkan sekurang-kurangnya sehingga penghujung tahun 2020 tetapi perubahan dalam bahasa undang-undang menutup kelemahan itu .

Arak hitam ialah hasil sampingan biojisim pengeluaran pulpa kayu.

## Memahami Kredit Cukai Arak Hitam

Kredit Campuran Bahan Api Alternatif direka untuk menggalakkan syarikat menghasilkan dan menggunakan lebih banyak biofuel. Ia memberi kredit kepada syarikat untuk menghasilkan bahan api yang merupakan campuran petrol dan sumber alternatif seperti biodiesel untuk kegunaan sendiri atau untuk dijual .

Syarikat diberi kredit 50 sen setiap gelen untuk setiap gelen bahan api alternatif yang mereka gunakan .

Sama ada secara sengaja atau tidak, AFMC membenarkan kredit cukai dituntut oleh syarikat yang telah menggunakan biofuel, hasil sampingan pengeluaran pulpa kayu, tetapi boleh menambah beberapa bahan api konvensional untuk layak mendapat kredit.

Satu lagi kesan sampingan AFMC ialah memesongkan pasaran kertas global dengan menjadikan produk kertas AS lebih murah. Ini menyebabkan penggubal undang-undang Kanada mewujudkan subsidi yang sama untuk terus bersaing dengan syarikat Amerika.

### Akta Dasar Tenaga

Presiden AS George W. Bush menandatangani Akta Dasar Tenaga pada Ogos 2005, dan ia dilanjutkan pada 2007. Akta itu menangani pelbagai isu yang mengelilingi pengeluaran tenaga negara, termasuk kecekapan, sumber tenaga boleh diperbaharui, minyak dan gas, dan pengeluaran arang batu.

Undang-undang menyediakan jaminan pinjaman untuk syarikat yang menggunakan atau membangunkan teknologi yang mengelakkan produk sampingan gas rumah hijau. Ia juga meningkatkan jumlah biofuel yang ditetapkan yang diperlukan untuk dicampur dengan petrol di negara ini .

Syarikat yang mencampurkan bahan api tradisional dan biodiesel layak untuk AFMC di bawah bil tenaga .

Syarikat kertas dan kayu menemui kelemahan dalam rang undang-undang yang membolehkan mereka layak menerima kredit cukai dengan mencampurkan sejumlah kecil diesel dengan minuman keras hitam. Arak hitam ialah hasil sampingan biojisim pengeluaran pulpa kayu, yang biasanya digunakan untuk loji kuasa dan kilang - [kilang](/credit).

### Berakhirnya Kredit Cukai Arak Hitam

Di bawah beberapa siri lanjutan undang-undang, AFMC terus menjadi sebahagian daripada Kod Cukai AS melalui kelulusan Akta Peruntukan Disatukan Selanjutnya 2020. Kredit cukai kekal tetapi Akta tersebut mengubah takrifan bahan api yang layak. Itu akhirnya mengecualikan sebarang gas yang dihasilkan daripada biojisim.

Syarikat yang memfailkan pada atau sebelum Jan. 8, 2018, tidak lagi layak untuk kredit .

> IRS menggariskan peraturan untuk syarikat membuat tuntutan sekali sahaja untuk kredit untuk tahun cukai 2018 dan 2019 di bawah Notis 2020-8 .

>

## Pertimbangan Khas

Hasrat Kongres dalam mewujudkan Kredit Campuran Bahan Api Alternatif dan kredit cukai yang mendahuluinya, [kredit cukai bahan api motor alkohol](/alternative-fuels-credit),. adalah untuk mewujudkan insentif bagi industri untuk mencipta bahan api motor cecair daripada biojisim. Memandangkan pemprosesan pulpa kayu sentiasa meninggalkan biojisim, mengubahnya menjadi bahan api cecair yang boleh digunakan akan berguna dari segi ekonomi dan alam sekitar, dan kredit cukai bahan api motor alkohol bertujuan untuk mempercepatkan penyelidikan dan penukaran kepada bahan api biojisim. Pada Jun 2009, minuman keras hitam layak untuk AFMC yang boleh dibayar balik. Tetapi seperti yang dinyatakan di atas, ia dikecualikan daripada senarai bahan api yang layak.

##Sorotan

- Kredit Campuran Bahan Api Alternatif bertujuan untuk menggalakkan syarikat menggunakan biofuel dengan mencampurkannya dengan bahan api fosil.

- Melihat kelemahan, beberapa syarikat kertas dan kayu mencampurkan produk sampingan yang dipanggil minuman keras hitam dengan diesel untuk layak mendapat kredit berbilion dolar.

- Semakan kepada undang-undang telah mengubah syarat kelayakan untuk kredit pada tahun 2020.
