# Web Dalam
Deep web ialah bahagian World Wide Web (WWW) yang agak tersembunyi. Ia mengandungi halaman yang tidak diindeks oleh enjin carian web tradisional - seperti Google, Bing dan DuckDuckGo.

Web dalam ialah istilah yang bertentangan dengan "web permukaan". Oleh itu, ia juga boleh dirujuk sebagai "web tersembunyi" atau "web tidak kelihatan".

## Di luar permukaan

Web dalam menyumbang lebih daripada 99% kandungan di web. Ia adalah sebahagian daripada WWW yang berbeza daripada apa yang dipanggil "web permukaan", iaitu bahagian web yang boleh dilihat. Web permukaan adalah lapisan yang paling mudah diakses. Ia termasuk semua halaman indeks (atau boleh dicari).

Walaupun jumlah kandungan yang boleh ditemui melalui enjin carian adalah luas, web permukaan mengandungi lebih sedikit maklumat daripada web dalam. Satu kajian yang diterbitkan pada tahun 2001, menunjukkan bahawa kurang daripada 0.05% daripada semua kandungan tersedia dalam web permukaan, manakala satu lagi ~99.95% ditemui dalam web dalam.

## Apakah itu Deep Web?

Mengikut definisi, web dalam merangkumi kedua-dua halaman yang tidak jelas atau yang hanya boleh diakses melalui kaedah pengesahan tertentu. Menjadi tidak jelas bermakna mereka memerlukan URL langsung untuk ditemui (mencari di Google tidak berfungsi). Namun, sebahagian besar kandungan web dalam berada di bawah kategori kedua.

Di sepanjang garis itu, kami mungkin menganggap sebagai sebahagian daripada web dalam mana-mana tapak web yang meminta log masuk dan kata laluan. Dalam erti kata lain, kebanyakan halaman yang diakses oleh orang setiap hari sebenarnya adalah sebahagian daripada web dalam. Ini termasuk profil media sosial, peti masuk e-mel dan akaun bank.

Ini mungkin kedengaran mengelirukan pada mulanya kerana istilah popular menyatakan bahawa web dalam ialah web yang tidak jelas. Tetapi memandangkan akaun peribadi kami tidak diindeks pada halaman enjin carian, ia juga dianggap sebagai sebahagian daripada web dalam.

Namun, bahagian terbesar web dalam terdiri daripada statistik dan pangkalan data peribadi. Sebagai contoh, agensi kerajaan AS seperti misi [Commodity Futures Trading Com](/cftc) [(CFTC)](/cftc),. National Oceanic and Atmospheric Administration (NOAA) dan [Securities and Exchange Commission (SEC)](/sec) menyumbang sejumlah besar data digital ini.

## Web dalam lwn web gelap

Ia juga biasa untuk melihat kekeliruan mengenai perbezaan antara web dalam dan web gelap. Ringkasnya, web gelap ialah sebahagian daripada web dalam yang tidak dikenali.

Oleh itu, ia sering dikaitkan dengan tapak web yang memudahkan aktiviti haram atau yang menyediakan beberapa jenis manfaat privasi untuk aktivis dan pemberi maklumat (termasuk penggodam topi hitam [dan](/hacker) penggodam). Web gelap dianggarkan menyumbang kurang daripada 0.01% daripada web dalam dan hanya boleh diakses melalui platform khusus, seperti Tor.

## Sorotan

- Mungkin kritikan paling serius terhadap web dalam ialah ia menjejaskan keterbukaan dan kesaksamaan internet.

- Web dalam merujuk kepada bahagian internet yang tidak boleh diakses sepenuhnya melalui enjin carian standard seperti Google, Yahoo dan Bing.

- Web dalam memberikan pengguna akses kepada lebih banyak maklumat daripada yang mungkin tersedia di internet, dan ia juga meningkatkan privasi.

- Web dalam termasuk halaman yang tidak diindeks, tapak bayaran untuk perkhidmatan, pangkalan data peribadi dan web gelap.

## Soalan Lazim

### Adakah Web Dalam dan Web Gelap adalah Perkara yang Sama?

Web gelap ialah komponen kecil web dalam. Deep web merujuk kepada semua internet yang tidak mudah diakses atau diindeks oleh enjin carian (yang dipanggil "surface web.") Dark web terdiri daripada laman web dalam yang sengaja disembunyikan, selalunya untuk menjalankan haram atau transaksi haram dalam talian.

### Adakah Haram Mengakses Web Dalam?

Tidak. Hanya mengakses tapak yang tidak diindeks atau tersedia secara umum adalah tidak menyalahi undang-undang. Walau bagaimanapun, menyalahgunakan atau mencuri maklumat yang terdapat di tapak web yang mendalam mungkin menyalahi undang-undang.

### Bagaimana Saya Boleh Cari Tapak di Web Dalam?

Oleh kerana web dalam tidak diindeks sepenuhnya dengan enjin carian arus perdana, selalunya satu-satunya cara untuk mencari tapak sedemikian ialah mengetahui alamat internet yang tepat untuk digunakan. Terdapat beberapa laman web khusus atau enjin carian yang mengkatalogkan beberapa laman web dalam. Sebagai contoh, sumber akademik di web dalam boleh didapati menggunakan platform seperti PubMed, LexisNexis, Web of Science atau Project MUSE.
