# mod
## Apakah Mod?

Mod ialah nilai yang paling kerap muncul dalam set data. Satu set data mungkin mempunyai satu mod, lebih daripada satu mod, atau tiada mod langsung. Ukuran popular kecenderungan memusat lain termasuk [min](/mean),. atau purata set, dan [median](/median),. nilai tengah dalam set.

## Memahami Mod

Dalam [statistik](/statistics),. data boleh diedarkan dalam pelbagai cara. Taburan yang paling kerap disebut ialah taburan [normal](/normaldistribution) klasik (lengkung loceng). Dalam hal ini, dan beberapa taburan lain, nilai min (purata) jatuh pada titik tengah, yang juga merupakan kekerapan puncak nilai yang diperhatikan.

Untuk pengedaran sedemikian, min, median dan mod adalah semua nilai yang sama. Ini bermakna nilai ini ialah nilai purata, nilai tengah dan juga modâ€”nilai yang paling kerap berlaku dalam data.

Mod paling berguna sebagai ukuran kecenderungan memusat apabila memeriksa data kategori, seperti model kereta atau perisa soda, yang mana nilai median purata matematik berdasarkan pesanan tidak boleh dikira.

## Contoh Mod

Contohnya, dalam senarai nombor berikut, 16 ialah mod kerana ia muncul lebih banyak kali dalam set berbanding nombor lain:

- 3, 3, 6, 9, **16, 16, 16**, 27, 27, 37, 48

Satu set nombor boleh mempunyai lebih daripada satu mod (ini dikenali sebagai **bimodal** jika terdapat dua mod) jika terdapat berbilang nombor yang berlaku dengan kekerapan yang sama, dan lebih banyak kali daripada yang lain dalam set.

- **3, 3, 3**, 9, **16, 16, 16**, 27, 37, 48

Dalam contoh di atas, kedua-dua nombor 3 dan nombor 16 adalah mod kerana masing-masing berlaku tiga kali dan tiada nombor lain berlaku lebih kerap.

Jika tiada nombor dalam set nombor berlaku lebih daripada sekali, set itu tidak mempunyai mod:

- 3, 6, 9, 16, 27, 37, 48

Satu set nombor dengan dua mod ialah **bimodal**, satu set nombor dengan tiga mod ialah **trimodal**, dan mana-mana set nombor dengan lebih daripada satu mod ialah **multimodal**.

> Apabila saintis atau ahli statistik bercakap tentang pemerhatian modal, mereka merujuk kepada pemerhatian yang paling biasa.

## Kelebihan dan Kelemahan Mod

### Kelebihan:

- Mod ini mudah difahami dan dikira.

- Mod tidak dipengaruhi oleh nilai melampau.

- Mod ini mudah dikenal pasti dalam set data dan dalam [taburan frekuensi diskret](/frequencydistribution).

- Mod ini berguna untuk data kualitatif.

- Mod boleh dikira dalam jadual frekuensi terbuka.

- Mod boleh didapati secara grafik.

### Kelemahan:

- Mod tidak ditakrifkan apabila tiada ulangan dalam set data.

- Mod tidak berdasarkan semua nilai.

- Mod tidak stabil apabila data terdiri daripada sebilangan kecil nilai.

- Kadangkala data mempunyai satu mod, lebih daripada satu mod, atau tiada mod langsung.

##Sorotan

- Untuk taburan normal, mod juga adalah nilai yang sama dengan min dan median.

- Dalam statistik, mod ialah nilai yang paling biasa diperhatikan dalam set data.

- Dalam kebanyakan kes, nilai modal akan berbeza daripada nilai purata dalam data.

##Soalan Lazim

### Apakah Mod dalam Statistik Dengan Contoh?

Mod dalam statistik merujuk kepada nombor dalam set nombor yang paling kerap muncul. Sebagai contoh, jika satu set nombor mengandungi digit berikut, 1, 1, 3, 5, 6, 6, 7, 7, 7, 8, mod itu ialah 7, kerana ia kelihatan paling banyak daripada semua nombor dalam set itu.

### Bagaimana Saya Mengira Mod?

Mengira mod adalah agak mudah. letakkan semua nombor dalam set yang diberikan mengikut tertib; ini boleh dari terendah ke tertinggi atau tertinggi ke terendah, dan kemudian hitung berapa kali setiap nombor muncul dalam set. Yang paling kelihatan ialah mod.

### Apakah Perbezaan Antara Mod dan Min?

Mod ialah nombor dalam set nombor yang paling kerap muncul. Min bagi satu set nombor ialah jumlah semua nombor dibahagikan dengan bilangan nilai dalam set itu. Min juga dikenali sebagai purata.
