# Kredit Cukai Bahan Api
## Apakah Kredit Cukai Bahan Api?

Kredit untuk Cukai Persekutuan Dibayar Ke Atas Bahan Api (Kredit Cukai Bahan Api) ialah program yang membolehkan sesetengah perniagaan mengurangkan dolar [pendapatan bercukai mereka](/taxableincome) untuk dolar berdasarkan jenis kos bahan api tertentu. Kredit Cukai Bahan Api menggalakkan pengeluaran dan penggunaan bahan api daripada [sumber yang boleh diperbaharui](/renewable_resource).

Kredit ini secara amnya tidak tersedia kepada pembayar cukai individu, kerana ia terhad kepada kegunaan termasuk penggunaan perniagaan di luar lebuh raya, pertanian dan pertanian, serta bot dan bas.

## Memahami Kredit Cukai Bahan Api

Kredit Cukai Bahan Api adalah untuk mengimbangi cukai yang dikenakan oleh kerajaan AS ke atas bahan api seperti petrol dan diesel dalam keadaan tertentu. Perkhidmatan Hasil [Dalam Negeri (IRS)](/irs) mengenakan cukai bahan api ini terutamanya untuk membiayai penyelenggaraan lebuh raya, mengenakan cukai apabila bahan api itu dibeli. Oleh kerana tidak mungkin untuk mengasingkan penggunaan bahan api yang boleh dikenakan cukai daripada penggunaan bahan api tidak boleh cukai pada masa pembelian, hampir semua orang membayarnya. Walau bagaimanapun, tidak semua orang menggunakan bahan api untuk tujuan bercukai.

### Ia Memupuk Penggunaan Sumber Boleh Diperbaharui

Oleh kerana cukai ini akan membayar balik cukai bahan api dolar untuk dolar, sesetengah syarikat mungkin melihat manfaat untuk menambah bahan api daripada sumber boleh diperbaharui kepada formula bahan api sedia ada mereka untuk memanfaatkan kredit. Walau bagaimanapun, apabila teknologi dan industri auto terus berubah dan menyesuaikan diri, jenis bahan api tertentu yang layak untuk kredit cukai juga akan berubah—bersama-sama dengan kegunaan yang dimaksudkan.

> IRS menyenaraikan Kredit Cukai Bahan Api sebagai salah satu kredit cukai yang paling kerap disalahgunakan atau disalahgunakan; penipuan yang melibatkan kredit ini boleh mengakibatkan penalti $5,000 atau lebih dan masa penjara.

>

### Siapa yang Layak?

Kriteria kelayakan untuk kredit ini termasuk pelbagai jenis bahan api—termasuk jenis bahan api yang digunakan oleh nelayan komersial untuk menjalankan bot mereka dan ketegangan bahan api yang digunakan oleh gudang untuk mengendalikan forkliftnya, contohnya. Syarikat bas sekolah juga mungkin layak. Jika anda organisasi bukan untung, anda mungkin layak, walaupun anda menggunakan kenderaan di jalan raya .

### Adakah Perniagaan Anda Layak?

Anda boleh menyemak tapak web IRS, atau berunding dengan profesional cukai berlesen untuk mengetahui sama ada anda layak. Setelah mereka menentukan bahawa mereka layak mendapat kredit, syarikat dan sesetengah individu boleh memohon Kredit Cukai Bahan Api dengan menggunakan Borang Cukai Persekutuan 4136. Pemfail juga boleh menggunakan borang ini untuk menuntut [kredit bahan api alternatif](/alternative-fuels-credit).

> ### Penipuan Kredit Cukai Bahan Api

> Pada Mac 2020, mahkamah Colorado mensabitkan seorang penipu atas tiga pertuduhan pengubahan wang haram—menjatuhi hukuman penjara tujuh tahun, ditambah empat tahun pembebasan dalam pengawasan, dan memerintahkan dia membayar balik IRS kira-kira $7.2 juta.

>

### Syarikat Mana Yang Boleh Mendapat Manfaat?

Pemilik syarikat landskap, pertanian, pembuatan dan pembinaan membeli bahan api untuk menggerakkan peralatan yang diperlukan untuk menjalankan perniagaan mereka, tetapi mereka sering mengabaikan kredit ini. Contohnya, pemilik perniagaan landskap yang menggunakan petrol dalam mesin pemotong rumputnya akan mempunyai tuntutan yang sah terhadap kredit tersebut, kerana ia layak untuk kategori "penggunaan luar lebuh raya".

## Kredit Cukai Bahan Api dan IRS

### Tuntutan Yang Sering Disalahgunakan

Walaupun Kredit Cukai Bahan Api tidak tersedia kepada kebanyakan pembayar cukai, IRS mendapati bahawa banyak pemfail meningkatkan jumlah dolar bayaran balik yang diminta dengan membuat tuntutan kredit yang salah. Tuntutan tidak wajar untuk Kredit Cukai Bahan Api boleh datang dalam dua bentuk: Individu atau perniagaan boleh membuat kesilapan pada penyata cukai mereka yang sah, atau pencuri identiti boleh memfailkan tuntutan palsu, selalunya sebagai sebahagian daripada skim [penipuan yang lebih luas.](/fraud)

### "Dirty Dozen" IRS

Setiap tahun, IRS menyusun senarai Dirty Dozennya, yang mewakili penipuan cukai "terburuk daripada yang paling teruk". The Dirty Dozen berulang kali menyenaraikan Kredit Cukai Bahan Api sebagai salah satu kredit cukai yang paling kerap disalahgunakan atau disalahgunakan. Ia sama ada diambil secara tersilap atau dalam percubaan sedar untuk menipu kerajaan. IRS menganggap penipuan Kredit Cukai Bahan Api sebagai "tuntutan cukai remeh", yang boleh mengakibatkan penalti $5,000 atau lebih ditambah dengan masa penjara.

## Kredit Cukai Bahan Api dalam Berita

Pada 2 Mac 2020, IRS melaporkan di laman webnya bahawa mahkamah Colorado menjatuhkan hukuman penjara 83 bulan kepada Matthew Taylor kerana peranannya dalam skim penipuan kredit cukai [biodiesel .](/biodiesel) Taylor mengaku bersalah atas satu pertuduhan masing-masing berkomplot untuk menipu Amerika Syarikat, berkomplot untuk melakukan pengubahan wang haram, dan melakukan pengubahan wang haram. Sebagai tambahan kepada hukuman penjara tujuh tahun, hakim memerintahkan Taylor menjalani empat tahun lagi pembebasan dalam pengawasan dan membayar balik kerajaan AS kira-kira $7.2 juta.

Mahkamah memutuskan bahawa Taylor dan konspirator bersamanya menipu kerajaan AS dengan memfailkan tuntutan palsu untuk kredit cukai. (Dalam kes khusus ini, penipu akan menggunakan IRS Borang 8864 Biodiesel dan Kredit Bahan Api Diesel Boleh Diperbaharui untuk memohon kredit dan bukannya Borang IRS 4136.) Penipu mencipta syarikat palsu, Shintan Inc., yang mengaku berada dalam perniagaan itu. mencipta bahan api daripada sumber yang boleh diperbaharui. Untuk mengelakkan pendedahan, penipu memindahkan dana yang diperoleh secara haram melalui beberapa siri akaun bank milik Shintan dan [syarikat shell lain](/shellcorporation).

Dari 2010 hingga 2013, Taylor dan rakan sejenayahnya memohon dan menerima daripada IRS lebih daripada $7.2 juta dalam bentuk kredit cukai—yang mana Taylor menerima $4.5 juta secara peribadi—untuk bahan api boleh diperbaharui yang didakwa dihasilkan oleh Shintan. Sudah tentu, Shintan tidak menghasilkan bahan api boleh diperbaharui yang layak, mahupun bahan api sama sekali.

Penyiasat khas IRS mengenai kes itu menyatakan bahawa dengan tindakannya, Taylor menipu bukan sahaja kerajaan tetapi semua pembayar cukai AS.

##Sorotan

- Kredit Cukai Bahan Api membolehkan perniagaan mengurangkan dolar pendapatan bercukai mereka untuk dolar berdasarkan penggunaan jenis kos bahan api tertentu.

- Kredit Cukai Bahan Api menggalakkan penggunaan bahan api sumber yang boleh diperbaharui.

- Kredit ini hanya tersedia untuk sesetengah individu, kerana ia terhad kepada penggunaan perniagaan di luar lebuh raya dan pemilihan penggunaan yang ditetapkan dengan ketat.

- Syarikat landskap, perladangan, pembuatan dan pembinaan, antara lain, boleh mendapat manfaat daripada Kredit Cukai Bahan Api.
