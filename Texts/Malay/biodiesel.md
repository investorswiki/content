# biodiesel
## Apakah Itu Biodiesel?

Biodiesel ialah sejenis bahan api mudah terbakar yang diperbuat daripada minyak organik, seperti minyak sayuran. Ia dilihat oleh ramai sebagai alternatif yang lebih [mesra alam](/sustainability) kepada [petroleum](/petroleum).

Sejak Akta Dasar Tenaga diluluskan pada tahun 2005, penggunaan dan pengeluaran biofuel telah meningkat di Amerika Syarikat. Aplikasi biasa termasuk penggunaannya sebagai bahan api kenderaan, bahan api pesawat dan minyak pemanas.

## Memahami Biodiesel

Biofuel dihasilkan dengan menggabungkan sejenis alkoholâ€”seperti metanolâ€”dengan sejenis minyak sayuran, seperti yang terdapat dalam kacang soya atau minyak sawit. Bahan lain, seperti lemak haiwan atau gris masak kitar semula, juga boleh digunakan. Sebaik sahaja biojisim yang terhasil ditukar kepada bahan api cecair, ia boleh digunakan untuk membekalkan pelbagai keperluan tenaga, bersaing dengan sumber tenaga tradisional seperti petrol dan [minyak mentah](/crude-oil).

> ### Bahan Api Hibrid

> Walaupun biodiesel boleh dilihat sebagai alternatif kepada bahan api tradisional seperti petrol dan diesel, ia juga boleh diadun dengan bahan api berasaskan hidrokarbon tradisional untuk mengurangkan pelepasan kenderaan.

>

Menurut Jabatan Tenaga AS, biodiesel ialah sumber tenaga yang agak cekap, dengan biodiesel kacang soya menghasilkan kira-kira 4.56 unit untuk setiap satu unit tenaga bahan api fosil yang digunakan sepanjang kitaran hayatnya.

Walaupun biodiesel hari ini masih jauh kurang digunakan secara meluas berbanding bahan api konvensional, jurang harga antara diesel tradisional dan biodiesel dijangka mengecil dalam dekad akan datang, disebabkan oleh peningkatan kekurangan dan peraturan yang mempengaruhi produk berasaskan petroleum. Faktor tambahan, seperti kehadiran [subsidi pertanian dan alam sekitar](/subsidy),. juga boleh membantu meningkatkan daya saing biodiesel dan sumber bahan api alternatif lain.

## Contoh Biodiesel

Hari ini, biodiesel digunakan di seluruh dunia sebagai sumber bahan api alternatif untuk kereta dan kenderaan lain. Penyokongnya menyebut kesan positifnya terhadap kualiti udara berbanding bahan api fosil tradisional. Ada juga yang melihat biodiesel sebagai bahan api yang lebih selamat kerana ia kurang mudah terbakar berbanding diesel petroleum.

Sebaliknya, pengkritik biodiesel berpendapat bahawa penggunaan bahan api alternatif berasaskan produk pertanian boleh menyumbang kepada kekurangan makanan dengan menggalakkan petani beralih daripada tanaman yang lebih berkhasiat kepada yang bertujuan untuk digunakan hanya sebagai sumber bahan api.

Masalah tempatan yang ditimbulkan oleh perubahan sedemikian mungkin paling ketara dirasai dalam ekonomi membangun di mana pengguna makanan tempatan mungkin tidak mempunyai [kuasa beli](/purchasingpower) atau infrastruktur negara untuk [mengimport](/import) sumber makanan alternatif secara besar-besaran.

##Sorotan

- Pengkritiknya menunjukkan kesannya yang berpotensi mengganggu pengeluaran makanan global.

- Penyokong biodiesel menyatakan sumbangannya kepada kelestarian alam sekitar.

- Biodiesel ialah sejenis bahan api alternatif yang dibuat dengan menggabungkan alkohol dengan minyak sayuran.

- Biodiesel kurang berkemungkinan terbakar berbanding diesel.

- Ia boleh digunakan sebagai sumber tenaga untuk kenderaan dan aplikasi industri, di samping atau sebagai pengganti produk petroleum tradisional.
