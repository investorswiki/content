# Strategi Pertama Mudah Alih
## Apakah Strategi Diutamakan Mudah Alih?

Frasa "strategi diutamakan mudah alih" merujuk kepada pendekatan untuk membangunkan tapak web di mana versi laman web yang didayakan mudah alih diberi keutamaan berbanding versi desktopnya.

Syarikat semakin mengutamakan versi mudah alih halaman web mereka, disebabkan peningkatan populariti pelayaran internet berasaskan telefon dan membeli-belah dalam talian di kalangan pengguna.

## Cara Strategi Diutamakan Mudah Alih Berfungsi

Apabila melaksanakan strategi mengutamakan mudah alih, syarikat akan mengeluarkan versi tapak webnya yang didayakan mudah alih, sebelum melaburkan masa atau sumber yang banyak untuk membangunkan versi desktop. Selalunya, beberapa tapak web desktop asas akan dikeluarkan bersama versi mudah alih, tetapi tapak web desktop ini mungkin mengandungi lebih sedikit daripada halaman pendaratan yang memaut ke aplikasi mudah alih atau kepada maklumat asas seperti halaman soalan lazim (FAQ).

Terdapat beberapa faktor asas yang mendorong populariti strategi mudah alih. Sebagai permulaan, peratusan trafik web yang boleh dikaitkan dengan [telefon pintar](/smartphone) berbanding komputer meja telah berkembang dengan stabil dari semasa ke semasa. Menurut StatCounter, penyemakan imbas mudah alih pada asasnya sama dengan penyemakan imbas desktop dengan masing-masing mengambil bahagian 48% pada Januari 2021. Perficient menganggarkan bahawa 61% daripada semua trafik web AS berasal daripada peranti mudah alih pada 2020.

Satu lagi faktor utama yang menggalakkan syarikat untuk meneruskan strategi mengutamakan mudah alih adalah berkaitan dengan data dan ciri tambahan yang boleh disediakan oleh tapak web mudah alih. Oleh kerana telefon pintar kini biasanya menggunakan antara muka skrin sentuh, pemilik tapak web berdaya mudah alih boleh mendapatkan data pengalaman pengguna (UX) terperinci mengenai cara pengguna diklik atau ditatal melalui halaman secara fizikal. Dalam sesetengah kes, pemilik tapak web malah boleh mendapatkan data mengenai pergerakan mata pengguna, seperti yang ditentukan menggunakan kamera menghadap hadapan telefon mereka.

Walaupun metrik data pengguna lanjutan ini agak jarang berlaku, jenis data pengguna kaya yang lain agak biasa. Satu contoh sedemikian ialah penjejakan lokasi, di mana koordinat GPS telefon pengguna dihantar secara automatik ke pembekal tapak web. Maklumat jenis ini boleh memainkan peranan penting dalam memaklumkan [kempen pemasaran masa hadapan](/marketing-strategy) atau merancang usaha pembangunan perniagaan seperti pembukaan kedai runcit baharu atau gudang pengedaran.

## Contoh Dunia Sebenar bagi Strategi Diutamakan Mudah Alih

Namun mungkin faktor paling penting yang mendorong penggunaan strategi diutamakan mudah alih ialah strategi "pengindeksan diutamakan mudah alih" daripada Google. Keputusan ini, yang diamalkan pada Julai 2019, menjelaskan bahawa, pada masa hadapan, algoritma carian syarikat akan memberi keutamaan kepada versi mudah alih tapak web apabila mengindeks tapak web baharu pada mulanya.

Secara praktiknya, ini bermakna tapak web baharu yang dibangunkan tanpa strategi mengutamakan mudah alih berkemungkinan mengalami hasil pengoptimuman enjin carian (SEO) sub-par. Memandangkan pengaruh besar yang Google nikmati dalam ekosistem enjin carian global, nampaknya strategi yang mengutamakan mudah alih akan diterima sebagai amalan terbaik untuk kebanyakan jika tidak semua projek pembangunan tapak web baharu pada masa hadapan.

##Sorotan

- Melangkah ke hadapan, strategi mengutamakan mudah alih berkemungkinan menjadi kebiasaan, sebahagiannya disebabkan oleh keputusan enjin carian utama untuk mengutamakan versi mudah alih tapak web.

- Strategi mengutamakan mudah alih ialah strategi di mana versi mudah alih tapak web diberi keutamaan berbanding versi desktopnya.

- Amalan ini agak jarang berlaku pada masa lalu, tetapi telah menjadi semakin biasa.
