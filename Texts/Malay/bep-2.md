# BEP-2
BEP-2 ialah standard teknikal untuk pengeluaran dan pelaksanaan token pada Rantaian Binance. Piawaian ini mentakrifkan satu set peraturan yang harus dipatuhi token untuk berfungsi dalam ekosistem Rantaian Binance. Standard token yang setanding untuk mengeluarkan token pada blockchain Ethereum ialah [ERC-20](/erc-20).

BEP-2 bukanlah satu perisian, melainkan satu set spesifikasi teknikal. Peraturan ini memastikan bahawa token dalam ekosistem Binance Chain boleh berinteraksi dengan lancar antara satu sama lain. Token BEP-2 boleh mewakili pelbagai jenis aset digital dan barangan boleh diniagakan. Piawaian ini memastikan bahawa mereka semua boleh berintegrasi dengan lancar antara satu sama lain dan ekosistem sekeliling.

Piawaian BEP-2 juga mentakrifkan beberapa ciri asas dan paling penting bagi token yang dikeluarkan pada Binance Chain. Walaupun piawaian mentakrifkan peraturan, token yang berbeza masih akan mempunyai pelaksanaan yang berbeza untuk kes penggunaan khusus yang ia dicipta untuknya. Namun, mencipta standard seperti ini mengurangkan jumlah usaha yang diperlukan daripada pembangun untuk mengeluarkan token dengan ketara.

Aset asli Binance Chain ialah BNB, yang digunakan untuk membayar yuran (serupa dengan gas). Ini bermakna apabila anda ingin memindahkan token BEP-2 anda, anda juga perlu mempunyai sejumlah kecil BNB dalam dompet anda.

Token BEP-2 boleh didagangkan pada Binance DEX, [pertukaran terdesentralisasi](/decentralized-exchange) yang berjalan di atas Rantaian Binance.

Anda boleh menyimpan token BEP-2 dalam banyak dompet yang berbeza. Trust Wallet ialah pilihan yang sangat baik jika anda ingin menggunakan dompet mudah alih, tetapi banyak dompet perisian lain turut menyokong standard tersebut. Standard BEP-2 juga disokong oleh dompet perkakasan, seperti Ledger Nano X, Ledger Nano S, Trezor Model T, atau Coolwallet S.

Anda boleh menyemak imbas semua token yang dikeluarkan pada Binance Chain dalam penjelajah blok Binance Chain.

Anda boleh mengeluarkan token BEP-2 menggunakan portal ini. Adalah sangat disyorkan untuk membaca spesifikasi teknikal yang lebih terperinci pada Binance Chain GitHub terlebih dahulu. Jika anda ingin bermain-main dengan idea token BEP-2, anda boleh melakukannya di testnet Binance Chain.

Anda akan menemui panduan teknikal terperinci tentang pengeluaran token, dan banyak topik berkaitan dalam Dokumentasi Rantaian Binance.
