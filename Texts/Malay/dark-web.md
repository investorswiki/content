# Web Gelap
## Apakah itu Web Gelap?

Istilah web gelap merujuk kepada kandungan dalam talian yang disulitkan yang tidak diindeks oleh enjin carian konvensional. Mengakses web gelap hanya boleh dilakukan menggunakan penyemak imbas tertentu, seperti Penyemak Imbas TOR. Terdapat banyak privasi dan tidak dikenali yang datang dengan menggunakan web gelap berbanding tapak web tradisional.

Oleh itu, kebanyakan perhatian diberikan pada [pasaran dalam talian](/market) untuk dadah, [pertukaran](/exchange) untuk data yang dicuri dan aktiviti haram lain apabila orang ramai memikirkan web gelap. Walaupun begitu, selalunya terdapat sebab yang sangat sah mengapa orang memilih untuk menggunakan web gelap, termasuk penentang politik dan orang yang ingin merahsiakan maklumat tertentu.

## Memahami Web Gelap

Seperti namanya, web gelap adalah rangkaian rahsia yang wujud di bawah tanah. Ia terdiri daripada satu siri tapak web yang disembunyikan daripada orang awam. Ini bermakna mereka tidak boleh diakses melalui enjin carian tradisional, seperti Google.

Enjin carian tradisional mengembalikan hasil kerana ia mengandungi indeks pautan ke tapak web. Ini disenaraikan berdasarkan kata kunci dan perkaitan. Web gelap, sebaliknya, menggunakan maklumat yang tidak tersedia pada enjin carian lain ini, seperti kandungan daripada akaun individu, seperti e-mel, [media sosial](/social-media),. perbankan, bersama-sama dengan pangkalan data peribadi dan profesional serta dokumen (undang-undang dan perubatan).

Juga dipanggil darknet, web gelap adalah sama seperti web yang lebih luas pada zaman awalnya pada akhir abad ke-20. Terdapat banyak bahan tentang membuatnya berfungsi, dan tidak banyak yang perlu dilakukan sebaik sahaja seseorang tiba di sana. Banyak kandungan di web gelap sangat amatur. Sebaliknya, lebih mudah bagi individu untuk memulakan tapak dan mendapat perhatian. Gergasi teknologi dan organisasi media besar mempunyai pengaruh yang sangat kecil pada web gelap pada tahun 2020.

Seperti internet awal, web gelap mempunyai reputasi sebagai syurga untuk aktiviti haram dan sering terlibat dalam aktiviti haram dan jenayah. Walaupun web gelap telah memainkan peranan dalam transaksi haram dan tidak beretika, ia juga menyediakan saluran sosial untuk orang yang mungkin dianiaya kerana identiti atau kepercayaan politik mereka. Ia juga memberi pihak berkuasa undang-undang alat tambahan yang mereka perlukan untuk menangkap pelaku aktiviti tidak beretika.

> Web gelap masih dalam proses dan kos dan faedah penuhnya masih belum diketahui.

>

## Pertimbangan Khas

Orang ramai sering mengelirukan web gelap dengan mata [wang kripto](/cryptocurrency),. yang sering digunakan untuk membuat pembelian di sana. Tetapi terdapat perbezaan yang ketara. Web gelap menjadikannya lebih mudah untuk menyediakan dan mengakses tapak web yang menawarkan tahap kerahasiaan tinggi untuk semua orang yang terlibat.

Banyak tapak web web gelap hanya mengandungi maklumat dan bukannya keupayaan untuk membeli atau menjual sesuatu. Memang benar, walaupun, mata wang kripto, seperti [Bitcoin](/bitcoin) dan [Monero](/monero),. sering digunakan untuk transaksi di web gelap. Tetapi seseorang tidak perlu menggunakan web gelap untuk menggunakan mata wang kripto.

## Sejarah Web Gelap

Web gelap mula-mula muncul secara rasmi pada awal 2000-an bersama-sama dengan penciptaan Freenet, yang dibangunkan oleh Ian Clarke untuk melindungi pengguna daripada campur tangan kerajaan dan serangan siber. Sistem itu, yang masih tersedia hari ini, membolehkan pengguna meluahkan perasaan mereka secara bebas tanpa dijejaki dalam talian.

Makmal Penyelidikan Tentera Laut AS membiayai projek yang dipanggil The Onion Router (TOR). TOR menawarkan sumber risikan cara untuk berkomunikasi dengan mudah dan selamat, terutamanya di kawasan bermusuhan di mana keselamatan peribadi adalah kunci. Ia kini merupakan salah satu penyemak imbas yang paling biasa digunakan untuk mengakses web gelap, menggunakan pangkalan data untuk membantu orang ramai membuat laluan mereka dan mencari maklumat yang mereka perlukan.

Kebangkitan mata wang kripto meningkatkan populariti web gelap, terutamanya untuk penjenayah siber. Ini kerana mata wang digital sering memberikan banyak kerahasiaan untuk orang yang membeli dan menjual di web gelap.

Disebabkan kaitannya dengan aktiviti terlarang tertentu, terdapat seruan untuk mengawal selia web gelap. Sebagai contoh, [G20](/g-20) dan [Pasukan Petugas Tindakan Kewangan (FATF)](/financial-action-task-force-fatf) telah meminta syarikat mata wang kripto untuk memberikan maklumat tentang pembeli dan penjual dalam transaksi yang dijalankan dalam talian. Ini adalah benar, kata mereka, untuk membantu penguatkuasaan undang-undang mengesan organisasi jenayah dan aktiviti haram.

> Ross Ulbricht telah disiasat dan ditangkap pada 2013 kerana mencipta dan menjalankan laman web gelap The Silk Road, kata pihak berkuasa laman web darknet digunakan untuk membeli dan menjual narkotik.

## Web Gelap lwn. Web Dalam

Web gelap dan web dalam juga sering tersilap digunakan secara bergantian. Web gelap, sebenarnya, hanya satu bahagian daripada web dalam. Web dalam ialah apa yang terletak di bawah tanah dan bukan hanya kandungan yang dianggap gelap. Itulah sebabnya ia juga dipanggil web halimunan atau web tersembunyi.

Maklumat yang ditemui di web dalam biasanya disulitkan dan tidak ditemui pada indeks seperti di web tradisional. Ia termasuk semua halaman yang tidak muncul apabila anda menjalankan carian web. Ia juga mengandungi semua yang memerlukan log masuk, seperti kandungan daripada:

- Perbankan atas talian

- Bayar tapak web, seperti Netflix dan Amazon Prime

- Perkhidmatan pengehosan fail, seperti Dropbox dan pesaingnya

- Pangkalan data peribadi

Maklumat yang disenaraikan di web gelap juga boleh termasuk apa-apa pada pangkalan data pelanggan sahaja dan bukannya apa-apa yang menyalahi undang-undang.

## Kelebihan dan Kelemahan Web Gelap

### Kelebihan

Web gelap membantu orang ramai mengekalkan privasi dan bebas menyatakan pandangan mereka. Privasi adalah penting bagi ramai orang yang tidak bersalah yang diganas oleh penguntit dan penjenayah lain. Peningkatan kecenderungan majikan yang berpotensi untuk menjejaki siaran di media sosial juga boleh menyukarkan untuk terlibat dalam perbincangan yang jujur secara terbuka.

Akhirnya, populariti web gelap dengan penjenayah menjadikannya cara yang sempurna untuk pegawai polis yang menyamar untuk berkomunikasi.

### Kelemahan

Sesetengah orang pasti akan menyalahgunakan kuasa yang berkaitan dengan menggunakan web gelap dengan memudahkan untuk terlibat dalam aktiviti jenayah. Sebagai contoh, gabungan web gelap dan mata wang kripto secara teori menjadikannya lebih mudah untuk mengupah seseorang untuk melakukan jenayah tertentu.

Walaupun web gelap menjanjikan privasi kepada penggunanya, ia juga boleh digunakan untuk melanggar privasi orang lain. Foto peribadi, rekod perubatan dan maklumat kewangan semuanya telah dicuri dan dikongsi di web gelap.

## Contoh Web Gelap

Katakan anda ingin pergi ke web gelap untuk melihat maklumat anda, jika ada, yang terapung-apung. Anda boleh memasang pelayar TOR pada sistem anda dengan menggunakan rangkaian peribadi maya. VPN membenarkan orang ramai menggunakan rangkaian awam seolah-olah mereka sedang menghantar maklumat pada rangkaian peribadi. Ia pada asasnya memberi anda privasi pada rangkaian awam.

Semasa anda melayari, anda mungkin akan perasan bahawa alamat tapak web gelap berakhir dengan sambungan .onion dan bukannya .com, .org, .edu, dsb. tradisional. Satu perkataan amaran, walaupun. Pastikan anda telah memasang beberapa perisian keselamatan kerana terdapat kemungkinan besar anda mungkin tersentuh dengan perisian hasad dan penipuan pancingan data.

## Garisan bawah

Web gelap adalah konsep yang agak baru, sekurang-kurangnya apabila anda mempertimbangkan sejarah internet secara keseluruhan. Anda boleh menggunakannya untuk kekal tanpa nama tanpa rasa takut akan kesannya jika anda ingin meluahkan perasaan anda secara bebas. Tetapi jumlah privasi ini telah membantunya mendapat reputasi buruk sebagai forum bagi mereka yang ingin menggunakannya untuk aktiviti haram. Jika anda ingin menggunakan web gelap, ia semudah memasang penyemak imbas peribadi, menggunakan VPN dan memastikan komputer anda kekal selamat dan terjamin.

## Sorotan

- Penyemak imbas tertentu, seperti Pelayar Tor, diperlukan untuk mencapai web gelap.

- Ia juga mempunyai reputasi kerana dikaitkan dengan aktiviti haram dan tidak beretika.

- Web gelap menarik tapak menggunakan maklumat yang tidak diindeks dalam talian, seperti akaun bank, akaun e-mel dan pangkalan data.

- Orang yang menggunakan web gelap boleh mengekalkan privasi mereka dan menyatakan pandangan mereka secara bebas.

- Web gelap merujuk kepada kandungan dalam talian yang disulitkan yang tidak diindeks oleh enjin carian konvensional.

## Soalan Lazim

### Bagaimana Anda Dapatkan di Web Gelap?

Anda boleh mengakses web gelap dengan memasang penyemak imbas tanpa nama khusus, seperti TOR. Setelah dipasang, penyemak imbas berfungsi dengan cara yang sama yang dilakukan oleh pelayar tradisional. Tetapi agak sukar untuk mengakses maklumat kerana ia tidak menggunakan indeks untuk mencari maklumat yang dikehendaki. Pastikan anda memasang perisian keselamatan untuk memastikan komputer dan maklumat peribadi anda disimpan selamat.

### Apa Yang Anda Lakukan Jika Maklumat Anda Terdapat di Web Gelap?

Anda boleh mengambil beberapa langkah untuk membantu melindungi maklumat anda. Pastikan anda menukar semua kata laluan anda, semak laporan kredit anda untuk sebarang percanggahan dan maklumkan kepada bank anda. Anda juga boleh membekukan kredit anda untuk mengelakkannya daripada dikompromi lagi.

### Bagaimana Anda Mengetahui Jika Maklumat Anda Ada di Web Gelap?

Maklumat peribadi anda, seperti akaun perbankan, media sosial dan e-mel, bersama dengan nombor Keselamatan Sosial anda, mungkin dijual di web gelap. Jika anda ingin melihat sama ada mana-mana maklumat anda terapung di sekitar web gelap, cuba jalankan imbasan web gelap. Sebagai alternatif, anda boleh mendaftar untuk perkhidmatan pemantauan web gelap. Alat ini memberi amaran kepada anda apabila mereka menemui sebarang maklumat anda.

### Adakah Haram Mengakses Web Gelap?

Walaupun apa yang dimaksudkan oleh namanya, ia tidak menyalahi undang-undang untuk mengakses web gelap. Ia sebenarnya menyediakan individu dengan privasi dan kerahasiaan yang tidak ditawarkan oleh tapak web tradisional kepada individu. Sebagai contoh, orang ramai boleh pergi ke web gelap dan menyiarkan pemikiran mereka tentang aktiviti politik tanpa rasa takut ditegur oleh pegawai kerajaan dan kumpulan lain.
