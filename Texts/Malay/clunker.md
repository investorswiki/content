# Clunker
## Apa Itu Clunker?

Clunker ialah istilah popular untuk kenderaan lama yang dicipta pada tahun 1930-an. Baru-baru ini, istilah ini digunakan untuk merujuk kepada kenderaan yang didagangkan untuk kenderaan yang lebih baharu dan lebih jimat bahan api dalam program " [cash-for-clunkers](/cash-for-clunkers) " kerajaan AS, yang dilancarkan pada 2009. Untuk "clunker" layak untuk program ini , ia mesti memenuhi empat syarat:

1. Ia mestilah dalam keadaan boleh dipandu

1. Ia mesti diinsuranskan secara berterusan selama satu tahun sebelum pertukaran masuk

1. Ia mesti dikeluarkan kurang daripada 25 tahun sebelum tarikh tukar beli

1. Ia mesti mempunyai gabungan kecekapan bahan api 18 batu setiap gelen atau kurang

## Memahami Clunker

Program cash-for-clunkers di AS menawarkan pemandu "clunkers" lama sehingga baucar $4,500 untuk berdagang dalam kereta lama mereka untuk kenderaan yang lebih baharu dan lebih jimat bahan api. Jika kenderaan lama bernilai lebih daripada $4,500, maka program itu tidak akan memberi manfaat kerana pemilik kenderaan boleh menjual kereta mereka kepada peniaga.

Penyokong program itu berhujah bahawa ia adalah satu kejayaan kerana ia memberi rangsangan kepada ekonomi dan menggantikan banyak kenderaan yang tidak cekap bahan api dengan kenderaan yang lebih jimat bahan api yang menghasilkan kurang pencemaran. Program itu, kata penyokong, mengeluarkan kira-kira 700,000 kereta yang tidak cekap bahan api dari jalan raya.

## Kritikan terhadap Program Cash-For-Clunkers

Ramai ahli ekonomi, bersama beberapa agensi kerajaan persekutuan dan kumpulan alam sekitar, mengkritik program itu. Beberapa ahli ekonomi memanggil program itu sebagai contoh kesilapan "tingkap pecah", yang berpendapat bahawa perbelanjaan mencipta kekayaan. Mereka berhujah bahawa program itu gagal disebabkan oleh kesan tersembunyi dan akibat yang tidak dapat dilihat daripada program itu dan program itu mewujudkan [kekurangan](/shortage) kenderaan terpakai, menyebabkan harga kereta terpakai melonjak dan memudaratkan golongan berpendapatan rendah. Mereka juga berhujah bahawa program itu menelan belanja [pembayar cukai](/taxpayer) $3 bilion dan program itu tidak banyak membantu merangsang ekonomi ASâ€”walaupun dalam jangka pendekâ€”kerana ia membantu pengeluar kereta asing dengan mengorbankan pengeluar domestik.

Kajian 2017 menggunakan data daripada jualan di Texas untuk menilai program tersebut. Texas adalah salah satu pasaran utama untuk program ini dan bertanggungjawab untuk 6% daripada keseluruhan jualan. Kajian mendapati bahawa 60% daripada subsidi diberikan kepada pengguna yang akan membeli kereta baru tanpa mengira. Walaupun selepas program itu tamat, tiada perbezaan yang ketara dalam tingkah laku pembelian atau pemilikan kereta di negeri ini. Jika wang tunai untuk klunk telah berjaya, maka akan berlaku penurunan mendadak dalam pemilikan atau pembelian kereta. Kajian itu juga mendapati bahawa program itu mendorong pelanggan untuk membeli kenderaan jimat bahan api yang lebih murah untuk memenuhi kriterianya, sekali gus memesongkan pasaran bagi kenderaan jimat bahan api.

Pada hakikatnya, [Biro Penyelidikan Ekonomi Kebangsaan](/nber) menyatakan bahawa kesan positif program itu adalah sederhana dan jangka pendek dan kebanyakan transaksi yang didorongnya akan berlaku juga. Satu kajian oleh Edmunds mendakwa bahawa program itu mendorong 125,000 pembelian kenderaan yang tidak akan berlaku pada masa itu, dengan kos purata kira-kira $24,000 setiap transaksi. Kajian lain bersetuju dengan kesan bersih negatif, memandangkan membuang kenderaan yang didagangkan memerlukan sejumlah besar bahan kimia toksik dan enjin terpaksa dihantar ke tapak pelupusan sampah atau peleburan.

## Sorotan

- Clunker adalah rujukan kepada program tunai-untuk-clunker yang dimulakan oleh pentadbiran Obama pada 2009 untuk merangsang perbelanjaan pengguna untuk kenderaan jimat bahan api.

- Ia menawarkan kepada pemandu "clunkers" lama sehingga baucar $4,500 untuk berdagang dalam kereta lama mereka yang menelan minyak untuk kenderaan yang lebih baru dan lebih jimat minyak.

- Konsensus umum ialah program itu tidak berjaya dan tidak banyak memberi insentif untuk perbelanjaan kenderaan jimat bahan api.
