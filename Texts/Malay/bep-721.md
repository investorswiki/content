# BEP-721
BEP-721 ialah piawaian token Binance Smart Chain (BSC) yang membenarkan penciptaan token bukan fungible (NFT). Ia adalah lanjutan daripada ERC-721 Ethereum, salah satu piawaian NFT yang paling biasa digunakan dan serasi dengan Mesin Maya Ethereum (EVM). Setiap NFT adalah unik dan tidak ditukar dengan mana-mana token lain.

Token BEP-721 membenarkan anda untuk menandakan pemilikan data dan melampirkan padanya pengecam unik. Aspek ini menjadikan token satu jenis dan jauh berbeza daripada token BEP-20. Dengan BEP-20, pembangun boleh mencipta berbilang token yang sama dalam satu kontrak pintar. Walau bagaimanapun, dengan BEP-721, setiap token diberikan ID token yang berbeza.

Melalui ID unik ini, token BEP-721 boleh digunakan untuk mewakili item yang boleh dikumpulkan dan tidak boleh digunakan. Pengguna boleh berdagang dan memindahkan token ini berdasarkan nilai pasaran mereka, yang bergantung pada kelangkaan atau utiliti token.

Token BEP-721 boleh mewakili:

1. Seni digital dan fizikal

1. Barang koleksi

1.Item dalam permainan

1. Harta fizikal dan hartanah

1. Tiket loteri

Seperti token lain pada Binance Smart Chain, pemindahan token BEP-721 memerlukan BNB untuk yuran gas. Token BEP-721 boleh dibuat pada pelbagai platform dalam ekosistem BSC, termasuk Binance NFT Marketplace, Ditampilkan Oleh Binance, BakerySwap dan Juggerworld.

## Fungsi token BEP-721

BEP-721 mempunyai senarai fungsi yang mengawal cara token berinteraksi dengan Rantaian Pintar Binance. Sebahagian daripada ini adalah agak standard dan juga terdapat dalam piawaian BEP-20:

- nama: Mentakrifkan nama token BEP-721, yang mana kontrak lain akan mengenal pastinya

- Simbol: Nama yang lebih pendek untuk token yang serupa dengan simbol ticker

- balanceOf: Menunjukkan baki token alamat tertentu

- totalSupply: Mentakrifkan jumlah bilangan token yang dicipta.

Terdapat juga satu set fungsi pemilikan asas, beberapa daripadanya adalah pilihan untuk disertakan. Ciri unik untuk NFT ialah fungsi metadata:

tokenMetadata: Fungsi ini membenarkan kemasukan metadata ke dalam token yang digunakan untuk memaut ke fail karya seni atau aspek koleksi NFT yang lain.

Sebagai contoh, setiap CryptoPunk mempunyai metadata yang terkandung dalam NFT yang menunjukkannya kepada punk tertentu dalam grid 10,000 punk. Walaupun CryptoPunks tidak menggunakan standard token BEP-721, penggunaan metadata mereka adalah sama.
