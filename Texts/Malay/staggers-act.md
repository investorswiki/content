# Akta Staggers
## Apakah Akta Staggers?

Akta Rel Staggers 1980 ialah undang-undang persekutuan yang mengawal industri kereta api dengan ketara [di](/deregulate) [Amerika](/deregulate) Syarikat . diluluskan untuk menggantikan struktur terkawal tinggi sistem perkapalan rel AS yang telah wujud sejak penggunaan Akta Perdagangan Antara Negeri 1887, yang mengawal hampir semua kadar yang boleh dikenakan oleh kereta api kepada pengirim .

Akta itu dinamakan sempena Ahli Kongres Harley O. Staggers, yang merupakan pengerusi Jawatankuasa Perdagangan Antara Negeri dan Asing Dewan .

## Memahami Akta Staggers

Akta Staggers menggantikan infrastruktur kawal selia untuk landasan kereta api yang telah wujud sejak penggunaan Akta Perdagangan Antara Negeri pada tahun 1887. Akta ini meletakkan landasan kereta api di bawah [Suruhanjaya Perdagangan Antara Negeri](/interstate-commerce-commission) (ICC), yang menubuhkan sistem untuk menetapkan kadar perkapalan yang tidak dapat bersaing. dengan kemajuan dalam teknologi dalam pasca Perang Dunia II Amerika .

## Akta Revitalisasi dan Pembaharuan Kawal Selia Kereta Api

Akta Staggers mengikut Akta Revitalisasi Kereta Api dan Pembaharuan Kawal Selia (4R) 1976, yang berusaha untuk melonggarkan sekatan pengawalseliaan ke atas kereta api untuk membolehkan mereka lebih bebas dalam menetapkan kadar untuk kontrak dan perkhidmatan, dan kebebasan yang lebih besar untuk masuk atau keluar dari pelbagai rel. pasaran. Apabila landasan kereta api beralih daripada membuat kadar kolektif, mereka memerlukan undang-undang yang akan menyokong lebih fleksibiliti bagi syarikat penerbangan kereta api untuk tawar-menawar dengan pengirim .

## Perubahan kepada Sistem Rel di bawah Akta Staggers

Akta Staggers memperuntukkan perubahan utama berikut kepada industri rel di AS:

- Ia membenarkan pengangkut kereta api mengenakan sebarang kadar tertentu yang mereka pilih untuk perkhidmatan melainkan ICC menentukan tiada persaingan untuk perkhidmatan tersebut wujud

- Ia mengalih keluar pelarasan kadar seluruh industri

- Ia menetapkan bahawa akses mesti diberikan oleh satu landasan kereta api kepada rel kereta api yang lain sekiranya satu kereta api mempunyai kawalan kesesakan lalu lintas kereta api

- Ia membenarkan pengangkut kereta api mewujudkan kontrak tanpa semakan ICC melainkan ICC menentukan bahawa kontrak tersebut akan mengganggu keupayaan pengangkut untuk menyediakan perkhidmatan bersama

- Ia mengesahkan pembongkaran infrastruktur pembuatan kadar kolektif di kalangan landasan kereta api

Mengikuti Akta tersebut, kajian mendapati bahawa industri telah menurunkan kos dan harga untuk perkhidmatan, memihak kepada prospek masa depan kedua-dua industri rel dan pelanggannya .

## Sorotan

- Akta Kereta Api Staggers 1980 menyahkawalseliakan kadar kereta api di AS

- Akta ini menggalakkan penetapan kadar berdasarkan persaingan pasaran. Sehingga itu, kadar telah ditetapkan oleh Suruhanjaya Perdagangan Antara Negeri.

- Kajian mencadangkan bahawa penyahkawalseliaan menyebabkan kos pengangkutan rel lebih rendah
