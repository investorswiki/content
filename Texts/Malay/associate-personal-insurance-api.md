# Bersekutu dalam Insurans Peribadi (API)
## Apakah Jawatan Rakan Bersekutu dalam Insurans Peribadi (API)?

Associate in Personal Insurance (API) ialah pensijilan profesional dalam [industri insurans](/insurance). Seperti namanya, API tertumpu pada [insurans talian peribadi](/personal-insurance-lines),. iaitu jenis insurans yang melindungi daripada kecederaan peribadi, kematian atau kehilangan harta benda. Ia adalah salah satu daripada banyak jawatan yang diberikan oleh The Institutes, sebuah organisasi yang berdedikasi untuk menyediakan akreditasi dan pendidikan berterusan kepada profesional insurans.

## Cara Reka Bentuk API Berfungsi

Penamaan API adalah popular di kalangan profesional yang ingin memulakan atau memajukan kerjaya mereka dalam bidang talian insurans peribadi. Dengan mendapatkan API, profesional boleh menunjukkan bahawa mereka mempunyai pengetahuan tentang produk insurans bukan komersial, termasuk kemahiran kritikal seperti [pengunderaitan insurans](/underwriting),. [tetapan premium](/insurance-premium),. pemasaran dan pengurusan portfolio.

Bagi profesional yang ingin mendapatkan pensijilan yang lebih maju, seperti sebutan [Penaja Jamin Harta Bertauliah (CPCU)](/chartered-property-casualty-underwriter-cpcu),. API boleh menjadi pilihan yang popular. Ini kerana banyak syarat yang diperlukan untuk mendapatkan API juga dikira dalam keperluan CPCU. Khususnya, dua daripada tiga peperiksaan yang dikaitkan dengan API boleh digunakan semula ke arah akreditasi CPCU, yang merupakan salah satu jawatan paling berprestij dalam industri kemalangan harta benda.

Untuk mendapatkan API, calon mesti membangunkan pengetahuan yang luas tentang jenis polisi insurans yang dijual dalam pasaran insurans peribadi. Ini termasuk pendidikan yang luas dalam mata pelajaran seperti pengesahan kontrak, syarat dan kriteria pengunderaitan, dan peraturan industri. API sering disyorkan untuk broker insurans, ejen, penyelaras tuntutan, penaja jamin dan kakitangan perkhidmatan pelanggan yang memberi perkhidmatan kepada pelanggan insurans peribadi.

## Contoh Dunia Nyata Penetapan API

Terdapat dua laluan yang boleh diambil oleh calon apabila melengkapkan API. Yang pertama termasuk kursus mengenai amalan [pengunderaitan](/underwriting) dan pemasaran dalam industri insurans peribadi, pengurusan portfolio, insurans harta dan [liabiliti](/liability_insurance),. dan insurans peribadi. Laluan kedua terdiri daripada tiga kursus, meliputi pengunderaitan dan pemasaran, [pengurusan risiko](/riskmanagement) dan liabiliti harta, dan perancangan kewangan peribadi.

Biasanya, calon mengambil masa antara 12 dan 18 bulan untuk melengkapkan bahan kursus ini, yang diakses secara layan diri. Bagi mereka yang lebih suka arahan secara bersemuka, kelas juga mungkin tersedia di luar talian di kawasan terpilih. Setelah mereka lulus peperiksaan mereka, pemegang API tidak perlu memenuhi keperluan berterusan pendidikan untuk mengekalkan jawatan mereka.

##Sorotan

- API ialah jawatan profesional yang popular dalam industri insurans talian peribadi.

- Ia ditadbir oleh The Institutes, dan biasanya mengambil masa antara 12 dan 18 bulan untuk disiapkan.

- Ramai calon memilih untuk menambah penetapan API mereka dengan mengikuti kelayakan yang lebih maju, seperti CPCU.
