# Laporan Status Petroleum EIA
## Apakah Laporan Status Petroleum EIA?

Laporan Status Petroleum EIA diterbitkan setiap hari Rabu oleh [Pentadbiran Maklumat Tenaga AS](/energy-information-admin) (EIA), sebuah agensi dalam Jabatan Tenaga AS. Ia memperincikan tahap rizab [minyak mentah](/crude-oil) yang dipegang oleh AS, serta jumlah minyak mentah dan produk berkaitan yang dihasilkannya, dalam dan luar negara.

## Memahami Laporan Status Petroleum EIA

Laporan Status Petroleum EIA memperincikan stok minyak mentah AS sedia ada, serta tahap inventori produk petroleum ditapis seperti petrol, minyak pemanas dan bahan api diesel. Bahagian sorotan utama laporan itu membahagikan input minyak mentah kepada kilang penapisan, [penggunaan kapasiti](/capacityutilizationrate) kilang penapisan tersebut, dan jumlah pengeluaran petrol dan sulingan baharu.

Lazimnya, laporan itu juga termasuk jadual dengan import bersih, atau jumlah import AS tolak eksport, kedua-dua produk mentah dan petroleum. Juga, laporan itu memperincikan harga per-gelen bagi minyak mentah [West Texas](/wti) [Intermediate](/wti) (WIT), petrol tanpa plumbum biasa, No. 2 minyak pemanas, dan propana.

Harga minyak mentah ditentukan oleh [penawaran dan permintaan](/law-of-supply-demand) [teknikal,](/technicalanalysis) seperti halnya dengan komoditi lain. Apabila rizab turun, harga akan naik dan begitu juga sebaliknya. Oleh itu, Laporan Status Petroleum EIA adalah gambaran aliran harga minyak mentah semasa. Penulis laporan tidak cuba mentafsirkan pemacu asas arah aliran tersebut.

Penganalisis pelaburan dan peniaga niaga hadapan akan mengkaji laporan itu, dan menggunakan datanya untuk membuat unjuran mengenai harga tenaga dan tahap pengeluaran. Begitu juga, beberapa penganalisis yang bekerja untuk syarikat tenaga akan menggunakan berita untuk mengumpulkan data daripada laporan untuk membantu memaklumkan strategi jangka panjang dan keputusan perniagaan mereka. Media juga kerap menggunakan laporan EIA dalam melaporkan tentang industri tenaga dan harga, yang secara langsung memberi kesan kepada pengguna, seperti memanaskan minyak dan petrol.

## Butiran Laporan Status Petroleum EIA

Pentadbiran Maklumat Tenaga AS menyediakan laporan status daripada tapak webnya. Pengguna boleh menyemak laporan menggunakan alatan dalam talian, termasuk analisis dan unjuran mengikut topik dan data yang diisih mengikut kategori. Laporan penuh tersedia dalam format PDF dan sebagai hamparan Microsoft Excel dan comma-separated-value (CSV) yang boleh dieksport untuk analisis lanjut.

- Bahagian sorotan Laporan Status Petroleum EIA menyediakan data pengeluaran untuk kebanyakan item baris pada asas empat minggu di belakang. Begitu juga, perbandingan tahun lalu dan minggu sebelumnya juga menggunakan angka empat minggu tertinggal.

- Asas untuk data pengeluaran datang daripada [harga penutupan](/closingprice) untuk hari tertentu yang dinyatakan.

- Jadual dalam laporan menunjukkan stok mingguan atau anggaran bekalan untuk minyak mentah, petrol motor, etanol bahan api dan produk petroleum. Data boleh dipecahkan lagi mengikut wilayah geografi, kerana harga di New England boleh berbeza secara bermakna daripada harga di Alaska atau negeri Atlantik Bawah.

- Di bawah laporan itu ialah jadual dengan inventori dan harga untuk produk berasaskan minyak, yang berdagang kurang kerap, termasuk bahan api jet, minyak tanah dan diesel sulfur ultra rendah.

- Bahagian harga minyak mentah, petrol dan minyak pemanas mengandungi harga spot (atau jangka pendek) dan niaga hadapan bersama-sama dengan harga bahan api jet, propana dan bahan api diesel.

- Akhir sekali, laporan itu menyediakan harga bulanan untuk **runcit** petrol dan bahan api diesel di lebuh raya untuk AS, serta mengikut wilayah geografi.

## Laporan EIA lwn. Laporan API

Institut [Petroleum Amerika (API)](/american-petroleum-institute) ialah sebuah organisasi yang diasaskan pada tahun 1919 dengan matlamat untuk menyokong industri minyak dan gas AS yang kukuh sambil mempromosikan keselamatan di seluruh industri. API memberi perkhidmatan kepada syarikat AS yang terlibat dalam pengeluaran, pemprosesan, penapisan dan pengedaran produk tenaga.

Daripada hampir 600 ahlinya, API menjalankan penyelidikan, mengumpul data daripada ahli dan bukan ahli, dan menerbitkan statistik dan penunjuk industri mereka dalam Buletin Statistik Mingguan. Laporan itu mengandungi data mengenai bekalan dan permintaan untuk produk tenaga, import, eksport, kos, aktiviti penggerudian dan penyiapan telaga.

API juga membantu membangunkan dan mengekalkan lebih 700 piawaian dan amalan, kebanyakannya telah dimasukkan ke dalam peraturan tempatan dan persekutuan. API bertindak sebagai peguam bela bagi pihak industri minyak dan gas asli kepada kerajaan Persekutuan, termasuk Kongres, Cawangan Eksekutif dan kerajaan negeri.

API juga membantu berunding dengan agensi kawal selia, termasuk mewakili industri dalam prosiding undang-undang. Buletin Statistik Mingguan API biasanya dikeluarkan pada hari Selasa sebelum laporan EIA mingguan.

EIA sangat serupa dengan API kerana ia adalah organisasi bebas yang mengumpul, menganalisis dan menerbitkan maklumat tentang industri minyak dan gas AS. EIA menerbitkan Laporan Status Petroleum Mingguannya pada hari Rabu. Laporan itu mengandungi tahap inventori minyak mentah dan produk ditapis serta data bekalan dan permintaan. Walau bagaimanapun, EIA tidak melobi secara aktif untuk sebarang perubahan dasar dan peraturan, tidak seperti API.

## Soalan Lazim Laporan EIA

##Sorotan

- Laporan Status Petroleum EIA diterbitkan setiap hari Rabu oleh Pentadbiran Maklumat Tenaga AS (EIA).

- Laporan Status Petroleum EIA memperincikan stok minyak mentah AS sedia ada dan tahap inventori produk petroleum ditapis seperti petrol, minyak pemanas dan bahan api diesel.

- Laporan Status Petroleum EIA boleh menunjukkan trend harga minyak mentah memandangkan peningkatan dalam inventori minyak mentah boleh menunjukkan permintaan yang lebih rendah, yang membawa kepada kejatuhan harga minyak dan sebaliknya.

##Soalan Lazim

### Pukul Berapa Laporan EIA Dikeluarkan?

Laporan EIA dikeluarkan selepas 10:30 pagi pada hari Rabu setiap minggu.

### Adakah Laporan EIA Boleh Dipercayai?

Laporan EIA dianggap secara meluas sebagai penunjuk bekalan dan permintaan yang boleh dipercayai untuk industri minyak dan gas. Walau bagaimanapun, laporan itu tidak menjamin atau meramalkan bahawa bekalan dan permintaan atau aliran harga akan bergerak satu arah atau yang lain.

### Mengapa Laporan EIA Penting?

Laporan Status Petroleum EIA adalah penting kerana ia menunjukkan harga dan data stok inventori untuk produk tenaga seperti minyak mentah. Laporan EIA ialah penunjuk penting bekalan dan permintaan dalam industri minyak dan gas. Sebagai contoh, jika stok minyak mentah meningkat, ia mungkin memberi isyarat bahawa permintaan minyak meningkat, membawa kepada harga yang lebih tinggi.
