# Kredit Cukai Bahan Api Alternatif
## Apakah Kredit Cukai Bahan Api Alternatif?

Kredit bahan api alternatif ialah [kredit cukai tidak boleh dikembalikan yang](/nonrefundabletaxcredit) diberikan kepada [pembayar cukai](/taxpayer) yang menggunakan bahan api alternatif bukan alkohol yang sama ada dijual secara komersial oleh pembayar cukai atau digunakan dalam kenderaan pembayar cukai untuk perniagaan .

## Memahami Kredit Cukai Bahan Api Alternatif

Kredit bahan api alternatif, yang diperincikan dalam [Kod Hasil Dalam Negeri (IRC)](/internal-revenue-code),. berpunca daripada idea bahawa AS harus mengurangkan pergantungannya kepada minyak import dan menerima bahan api dan teknologi kenderaan termaju .

Terutamanya berpengaruh dalam pembangunan pandangan IRC mengenai bahan api alternatif ialah Akta Penciptaan Pekerjaan Amerika 2004 (Bil Pekerjaan), Akta Ekuiti Pengangkutan yang Selamat, Bertanggungjawab, Fleksibel, Cekap: Warisan untuk Pengguna (Bang Lebuh Raya), dan Dasar Tenaga. Akta 2005 (EPAct 2005). Kredit bahan api untuk [biodiesel](/biodiesel) atau kredit diesel boleh diperbaharui, serta campuran bahan api, termasuk campuran etanol/petrol, telah diwujudkan dalam bahagian Kredit Cukai Eksais Volumetrik (VEETC) dalam Rang Undang-Undang Pekerjaan. Ia bertujuan untuk menyediakan insentif untuk menghasilkan, menjual dan menggunakan bahan api ini .

Perkhidmatan Hasil Dalam Negeri menganggap bahan api alternatif sebagai gas petroleum cecair (LPG), gas asli termampat (CNG), gas asli cecair (LNG), hidrogen cecair, bahan api cecair yang diperoleh daripada arang batu (termasuk gambut) melalui proses Fischer-Tropsch, cecair hidrokarbon yang diperoleh daripada biojisim, dan bahan api Siri-P. Ambil perhatian bahawa [hidrokarbon](/hydrocarbon) termasuk cecair yang mengandungi oksigen, hidrogen dan karbon, dan hidrokarbon cecair yang diperoleh daripada biojisim termasuk etanol, biodiesel dan diesel boleh diperbaharui.

Walau bagaimanapun, IRS tidak memasukkan bahan api ini dalam kategori bahan api alternatif. [Insentif cukai](/tax-benefit) untuk bahan api ini dilindungi di bawah kategori bahan api petrol dan diesel, masing-masing .

> Kredit ini ditetapkan untuk tamat tempoh pada 31 Disember 2020 melainkan Kongres memanjangkan kredit .

>

## Ketersediaan Kredit Bahan Api Alternatif

Antara mereka yang mungkin boleh mengambil kesempatan daripada kredit bahan api alternatif ialah operator armada, pembekal bahan api, dan pengisar bahan api. Untuk layak mendapat [kredit cukai](/taxcredit),. komposisi bahan api, cara bahan api digunakan, dan prosedur pembayaran cukai eksais mesti dinyatakan dengan jelas. Pembekal dan pengisar juga mesti mendaftar dengan Perkhidmatan Hasil Dalam Negeri. Borang 8849, 4136, 6478, atau 8864 boleh digunakan untuk membuat tuntutan atau [bayaran balik](/tax-refund) untuk alkohol, biodiesel atau diesel boleh diperbaharui, atau bahan api alternatif yang digunakan untuk menghasilkan campuran .
