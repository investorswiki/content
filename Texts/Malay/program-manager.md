# Pengurus Program
## Apakah itu Pengurus Program?

Pengurus program menyelia pengurusan program tertentu, secara amnya dalam perniagaan [kad kredit](/creditcard) atau teknologi maklumat. Dalam kawasan kad kredit atau kad korporat, pengurus program mengawasi pengeluaran dan pembatalan kad, berhubung dengan pelbagai jabatan dan memantau serta melaporkan metrik prestasi utama. Dalam teknologi maklumat, pengurus program menyelia kumpulan projek berkaitan yang diuruskan oleh pengurus projek individu.

## Memahami Pengurus Program

Peranan pengurus program ialah kepimpinan yang memerlukan gabungan kemahiran. Fungsi kerja pengurus program kad yang pelbagai termasuk mengenal pasti peluang perniagaan, rundingan [vendor](/vendor),. mitigasi risiko untuk meminimumkan kerugian kredit dan pematuhan. Ia juga melibatkan mewujudkan dasar dan prosedur serta perkhidmatan sokongan pemegang kad.

Dalam bidang teknologi maklumat, terdapat beberapa perbezaan utama antara pengurusan program dan pengurusan [projek](/project-management). Walaupun prestasi pengurus projek adalah berdasarkan masa, kos dan [skop](/scope) projek, pengurus program dinilai secara kumulatif untuk semua projek dalam program mereka.

Selalunya, pengurus program bertanggungjawab ke atas hasil inisiatif tertentu kerana ia terpakai kepada keseluruhan syarikat. Oleh itu, pengurus program mengawasi pengurus projek dan mengurus strategi keseluruhan untuk inisiatif tertentu. Inisiatif boleh merangkumi pelaksanaan sistem inventori baharu, pelancaran produk baharu dan pengembangan yang dicapai melalui pembukaan kedai baharu.

Ini memerlukan pengurus program untuk mempertimbangkan faktor lain selain daripada [hasil projek jangka pendek](/deliverables) yang menjadi tumpuan pengurus projek, seperti keberkesanan program jangka panjang, kesannya terhadap sasaran korporat, dsb.

## Pensijilan

Institut Pengurusan Projek, persatuan keahlian profesional bukan untung untuk pengurusan projek dan program, menawarkan beberapa pensijilan yang diiktiraf dalam bidang ini. Kelayakan Profesional Pengurusan Projek (PMP) ialah kelayakan yang diiktiraf di peringkat global yang menunjukkan kecekapan pengurus program untuk menyelia pelbagai projek yang berkaitan dan memperuntukkan sumber untuk mencapai matlamat perniagaan strategik.

Untuk layak mendapat pensijilan PMP, seseorang memerlukan ijazah empat tahun, 36 bulan pengalaman mengetuai projek, dan 35 jam latihan; atau seseorang memerlukan diploma sekolah tinggi atau ijazah dua tahun, 60 bulan pengalaman mengetuai projek, dan 35 jam latihan.

## Tanggungjawab Pengurus Program

Menurut Institut Pengurusan Projek, berikut adalah tanggungjawab utama pengurus program:

- Pengurusan program harian sepanjang kitaran hayat program

- Tentukan tadbir urus program

- Rancang keseluruhan program dan pantau kemajuan

- Menguruskan bajet program

- Mengurus risiko dan isu dan mengambil ukuran pembetulan

- Menyelaraskan projek dan saling bergantungan mereka

- Mengurus dan menggunakan sumber merentas projek

- Mengurus komunikasi pihak berkepentingan

- Selaraskan penyampaian kepada hasil program

- Menguruskan dokumen program utama

## Bagaimana untuk Menjadi Pengurus Program

Titik permulaan untuk menjadi pengurus program ialah memiliki ijazah sarjana muda dalam pentadbiran perniagaan, sains komputer, komunikasi atau pemasaran. Bergantung pada industri yang anda ingin sertai, kelayakan lanjut, seperti program sarjana atau sijil mungkin perlu dilengkapkan. Kursus pengurusan dan kepimpinan bermanfaat untuk menjadi pengurus program

Gaji purata untuk pengurus program ialah $87,415. Gaji berkisar antara $53,000 dan $132,000, bergantung pada wilayah, pekerjaan tertentu dan industri. Walau apa pun, menjadi pengurus program adalah kerjaya yang berpendapatan tinggi. Sebagai tambahan kepada gaji asas, kerja pengurus program disertakan dengan bonus dan kemungkinan [rancangan perkongsian keuntungan](/profitsharingplan) bergantung pada syarikat yang bekerja.

## Sorotan

- Dalam teknologi maklumat, pengurus program menyelia kumpulan projek berkaitan yang diuruskan oleh pengurus projek individu.

- Dalam dunia kad kredit, pengurus program menyelia pengeluaran dan pembatalan kad, berhubung dengan pelbagai jabatan, dan memantau serta melaporkan metrik prestasi utama.

- Pengurus program dan pengurus projek berbeza sedikit dalam sektor teknologi maklumat. Pengurus program dinilai secara kumulatif untuk semua projek dalam program mereka; bukan hanya kos dan skop projek seperti yang digunakan untuk pengurus projek.

- Pengurus program ialah individu yang mengawasi pengurusan program tertentu, selalunya dalam sektor kad kredit atau teknologi maklumat.

- Institut Pengurusan Projek menawarkan beberapa pensijilan yang diiktiraf dalam bidang pengurus program dan pengurusan projek, seperti kelayakan Profesional Pengurusan Projek (PMP).
