# Code fiscal
## Qu'est-ce qu'un code fiscal ?

Un code fiscal est un document du gouvernement fédéral, comptant généralement des milliers de pages, qui détaille les règles que les particuliers et les entreprises doivent suivre pour verser un pourcentage de leurs revenus au gouvernement fédéral ou d'État. Le code des impôts est utilisé comme source par les avocats fiscalistes qui ont la responsabilité de l'interpréter pour le public.

## Comment fonctionne un code fiscal

Aux États-Unis, le Congrès américain rédige les lois fiscales et établit les règles au niveau fédéral. L' [Internal](/irs) [Revenue Service](/irs) (IRS) applique les règles établies et explique comment elles s'appliquent dans différents scénarios par le biais du code des impôts. Au niveau de l'État, ces lois sont établies par un gouvernement d'État, local ou de comté qui utilise les codes fiscaux pour autoriser toute taxation votée et convenue. En effet, le code des impôts, parfois appelé code fiscal [interne](/internal-revenue-code) (IRC), est un ensemble de lois fiscales promulguées par les autorités fédérales, étatiques et locales.

Chaque loi fiscale adoptée se voit attribuer un code qui est ajouté à la collection de lois fiscales existantes dans la publication IRC. Étant donné que le code des impôts n'est pas facilement compris par la personne moyenne, l'IRS fournit des instructions détaillées qui décomposent chaque code et comment ils doivent être appliqués. Tous les taux d'imposition, exclusions, déductions, crédits, régimes de retraite et d'avantages sociaux, [exemptions personnelles](/personal-exemption),. etc. fournis par l'IRS sont tirés des codes fiscaux fédéraux. Les codes fiscaux dans l'IRC sont organisés et référencés par sections. Par exemple, la section 1 de l'Internal Revenue Code relaie l'impôt fédéral sur le revenu sur le revenu imposable des citoyens et résidents américains, ainsi que sur les successions et fiducies. L'article 11 de l'IRC impose l'impôt sur le revenu des sociétés.

Un certain nombre de sources secondaires, telles que les publications numérotées de l'Internal Revenue Service, les décisions fiscales et les livres d'impôt sur le revenu grand public, tentent de mettre le code des impôts en langage clair pour les contribuables. Les publications de l'IRS sont disponibles gratuitement en version imprimée ou en ligne sur le site Web de l'IRS.

Une autre source secondaire qui cherche à interpréter les codes fiscaux est la réglementation du Trésor ou la réglementation fiscale, émise par le [Trésor](/ustreasury) [américain](/ustreasury) sur la plupart des sections du code fiscal pour fournir des explications plus longues et des exemples de la manière dont la loi est utilisée. Ces réglementations sont publiées dans le titre 26 du Code of Federal Regulations (26 CFR) et sont également disponibles en ligne sur le site Web du GPO.

Le Code électronique des réglementations fédérales est une version continuellement mise à jour du CFR. Les contribuables peuvent souvent se conformer correctement aux règles fiscales en suivant les directives énoncées dans ces publications secondaires, mais pour des situations plus complexes, il peut être nécessaire de consulter directement le code des impôts, pour décrypter les lois fiscales.

## Points forts

- Aux États-Unis, le Congrès américain rédige les lois fiscales et fixe les règles au niveau fédéral.

- Un code fiscal est un document du gouvernement fédéral, comptant généralement des milliers de pages, qui détaille les règles que les particuliers et les entreprises doivent suivre pour remettre un pourcentage de leurs revenus au gouvernement fédéral ou d'État.

- L'Internal Revenue Service (IRS) applique les règles établies et explique comment elles s'appliquent dans différents scénarios par le biais du code des impôts.
