# BEP-2
BEP-2 est une norme technique pour l'émission et la mise en œuvre de jetons sur la chaîne Binance. Cette norme définit un ensemble de règles que les jetons doivent suivre pour fonctionner dans l'écosystème Binance Chain. Une norme de jeton comparable pour l'émission de jetons sur la blockchain Ethereum est [ERC-20](/erc-20).

BEP-2 n'est pas un logiciel, mais plutôt un ensemble de spécifications techniques. Ces règles garantissent que les jetons de l'écosystème Binance Chain peuvent interagir de manière transparente les uns avec les autres. Les jetons BEP-2 peuvent représenter un large éventail d'actifs numériques et de biens échangeables. La norme garantit qu'ils peuvent tous s'intégrer en douceur les uns aux autres et à l'écosystème environnant.

La norme BEP-2 définit également certaines des caractéristiques de base et les plus essentielles des jetons émis sur Binance Chain. Bien que la norme définisse les règles, différents jetons auront toujours des implémentations différentes pour les cas d'utilisation spécifiques pour lesquels ils sont créés. Pourtant, la création d'une norme comme celle-ci réduit considérablement la quantité d'efforts requis des développeurs pour émettre des jetons.

L'actif natif de Binance Chain est le BNB, qui est utilisé pour payer les frais (similaires au gaz). Cela signifie que lorsque vous souhaitez transférer vos jetons BEP-2, vous aurez également besoin d'avoir une petite quantité de BNB dans votre portefeuille.

Les jetons BEP-2 peuvent être échangés sur Binance DEX, un [échange décentralisé](/decentralized-exchange) qui fonctionne au-dessus de Binance Chain.

Vous pouvez stocker des jetons BEP-2 dans de nombreux portefeuilles différents. Trust Wallet est une excellente option si vous souhaitez utiliser un portefeuille mobile, mais de nombreux autres portefeuilles logiciels prennent également en charge la norme. La norme BEP-2 est également prise en charge par les portefeuilles matériels, tels que Ledger Nano X, Ledger Nano S, Trezor Model T ou le Coolwallet S.

Vous pouvez parcourir tous les jetons qui ont été émis sur Binance Chain dans l'explorateur de blocs Binance Chain.

Vous pouvez émettre des jetons BEP-2 à l'aide de ce portail. Il est fortement recommandé de lire d'abord les spécifications techniques plus détaillées sur le Binance Chain GitHub. Si vous souhaitez jouer avec les idées de jetons BEP-2, vous pouvez le faire sur le testnet Binance Chain.

Vous trouverez des guides techniques détaillés sur l'émission de jetons et de nombreux sujets connexes dans la documentation de la chaîne Binance.
