# Code d'éthique
## Qu'est-ce qu'un code d'éthique ?

Un code d'éthique est un guide de principes conçu pour aider les professionnels à mener leurs activités avec honnêteté et intégrité. Un document de code d'éthique peut décrire la mission et les valeurs de l'entreprise ou de l'organisation, la façon dont les professionnels sont censés aborder les problèmes, les principes éthiques basés sur les valeurs fondamentales de l'organisation et les normes auxquelles le professionnel est tenu.

Un code d'éthique, également appelé « code d'éthique », peut englober des domaines tels que [l'éthique des affaires](/business-ethics),. un code de pratique professionnelle et un code de conduite des employés.

## Comprendre les codes d'éthique

L'éthique des affaires fait référence à la manière dont les principes éthiques guident les opérations d'une entreprise. Les problèmes courants qui relèvent de l'éthique des affaires comprennent les relations employeur-employé, la discrimination, les problèmes environnementaux, la corruption, [le délit d'initié](/insidertrading) et la responsabilité sociale.

Bien qu'il existe de nombreuses lois pour établir des normes éthiques de base au sein de la communauté des affaires, l'élaboration d'un code d'éthique dépend en grande partie de la direction d'une entreprise.

Les entreprises et les organisations commerciales ont généralement une sorte de code d'éthique que leurs employés ou membres sont censés suivre. Le non-respect du code d'éthique peut entraîner la résiliation ou le renvoi de l'organisation. Un code d'éthique est important car il énonce clairement les règles de comportement et fournit les bases d'un avertissement préventif.

> Bien qu'un code d'éthique ne soit souvent pas requis, de nombreuses entreprises et organisations choisissent d'en adopter un, ce qui aide à identifier et caractériser une entreprise pour les [parties prenantes](/stakeholder).

>

Compte tenu de l'importance du changement climatique et de la façon dont le comportement humain a eu un impact important sur le climat, de nombreuses entreprises ont pris l'initiative d'inclure les facteurs climatiques dans leur code d'éthique. Ces principes incluent les manières dont l'entreprise s'engage à fonctionner de manière durable ou comment elle s'y adaptera.

Dans de nombreux cas, cet engagement envers la [durabilité](/sustainability) augmente les coûts d'une entreprise, mais comme les consommateurs se concentrent de plus en plus sur les types d'entreprises avec lesquelles ils choisissent de s'engager, cela vaut souvent le coût de maintenir une bonne image publique.

Quelle que soit leur taille, les entreprises comptent sur leur personnel de direction pour établir une norme de conduite éthique que les autres employés doivent suivre. Lorsque les administrateurs adhèrent au code d'éthique, cela envoie le message que la conformité universelle est attendue de chaque employé.

## Types de codes d'éthique

Un code d'éthique peut prendre diverses formes, mais l'objectif général est de s'assurer qu'une entreprise et ses employés respectent les lois nationales et fédérales, se conduisent avec un idéal qui peut être exemplaire et s'assurent que l'entreprise menée est bénéfique. pour toutes les parties prenantes. Voici trois types de codes d'éthique que l'on retrouve dans les affaires.

### Code d'éthique basé sur la conformité

Pour toutes les entreprises, les lois réglementent des questions telles que l'embauche et les normes de sécurité. Les codes d'éthique basés sur la conformité établissent non seulement des lignes directrices pour la conduite, mais déterminent également les sanctions en cas de violation.

Dans certains secteurs, dont le secteur bancaire, des lois spécifiques régissent la conduite des affaires. Ces industries formulent des codes d'éthique basés sur la conformité pour faire respecter les lois et les réglementations. Les employés suivent généralement une formation formelle pour apprendre les règles de conduite. Étant donné que la non-conformité peut créer des problèmes juridiques pour l'entreprise dans son ensemble, les travailleurs individuels au sein d'une entreprise peuvent faire face à des sanctions pour ne pas avoir suivi les directives.

Pour s'assurer que les objectifs et les principes du code d'éthique sont respectés, certaines entreprises désignent un [responsable de la conformité](/compliance-officer). Cette personne est chargée de se tenir au courant des modifications apportées aux codes de réglementation et de surveiller la conduite des employés afin d'encourager la conformité.

Ce type de code d'éthique est basé sur des règles claires et des conséquences bien définies plutôt que sur un contrôle individuel du comportement personnel. Malgré le strict respect de la loi, certains codes de conduite basés sur la conformité ne favorisent donc pas un climat de responsabilité morale au sein de l'entreprise.

### Code d'éthique basé sur les valeurs

Un code d'éthique fondé sur les valeurs aborde le système de valeurs fondamentales d'une entreprise. Il peut définir des normes de conduite responsable en ce qui concerne le bien public et l'environnement. Les codes d'éthique fondés sur des valeurs peuvent exiger un plus grand degré d'autoréglementation que les codes fondés sur la conformité.

Certains codes de conduite contiennent un langage qui traite à la fois de la conformité et des valeurs. Par exemple, une chaîne d'épiceries pourrait créer un code de conduite qui épouse l'engagement de l'entreprise envers les réglementations en matière de santé et de sécurité plutôt que le gain financier. Cette chaîne d'épicerie pourrait également inclure une déclaration sur le refus de contracter avec des [fournisseurs](/vendor) qui nourrissent le bétail avec des hormones ou élèvent des animaux dans des conditions de vie inhumaines.

### Code d'éthique entre professionnels

Les conseillers financiers inscrits auprès de la [Securities and Exchange](/sec) [Commission (SEC)](/sec) ou d'un organisme de réglementation d'État sont liés par un code de déontologie connu sous le nom d'obligation fiduciaire. Il s'agit d'une exigence légale et aussi d'un code de loyauté qui les oblige à agir dans le meilleur intérêt de leurs clients.

Les comptables publics agréés, qui ne sont généralement pas considérés comme des fiduciaires pour leurs clients, doivent toujours suivre des normes éthiques similaires, telles que l'intégrité, l'objectivité, la véracité et la prévention des conflits d'intérêts, selon l' [American Institute of Certified Public Accountants (AICPA)](/american-institute-of-certified-public-accountants).

## Exemple de code d'éthique

De nombreuses entreprises et organisations ont adopté un code d'éthique. Un bon exemple vient du CFA Institute (CFAI), qui accorde le titre d'analyste financier agréé (CFA) et créateur des examens CFA. Les titulaires de la [charte CFA](/cfa) comptent parmi les professionnels de la finance les plus respectés et les plus reconnus à l'échelle mondiale. Selon le site Web du CFAI, les membres du CFA Institute, y compris les détenteurs de la charte CFA, et les candidats à la désignation CFA doivent adhérer au code de déontologie suivant :

- Agir avec intégrité, compétence, diligence, respect et de manière éthique avec le public, les clients, les clients potentiels, les employeurs, les employés, les collègues de la profession d'investissement et les autres participants aux marchés financiers mondiaux.

- Placer l'intégrité de la profession d'investisseur et les intérêts des clients au-dessus de leurs propres intérêts personnels.

- Faire preuve d'une diligence raisonnable et exercer un jugement professionnel indépendant lors de la réalisation d'analyses d'investissement, de la formulation de recommandations d'investissement, de la prise d'actions d'investissement et de l'engagement dans d'autres activités professionnelles.

- Pratiquer et encourager les autres à pratiquer de manière professionnelle et éthique, ce qui fera honneur à eux-mêmes et à la profession.

- Promouvoir l'intégrité et la viabilité des marchés mondiaux des capitaux pour le bénéfice ultime de la société.

- Maintenir et améliorer leurs compétences professionnelles et s'efforcer de maintenir et d'améliorer les compétences des autres professionnels de l'investissement.

## FAQ sur le code d'éthique

### Quels sont les cinq codes d'éthique ?

Toutes les entreprises auront un code d'éthique différent avec différents domaines d'intérêt, en fonction de l'industrie dans laquelle elles sont impliquées, mais les cinq domaines sur lesquels les entreprises se concentrent généralement incluent l'intégrité, l'objectivité, la compétence professionnelle, la confidentialité et le comportement professionnel.

### Qu'est-ce qu'un code d'éthique en entreprise ?

Un code d'éthique en entreprise est un ensemble de principes directeurs visant à garantir qu'une entreprise et ses employés agissent avec honnêteté et intégrité dans toutes les facettes de leurs opérations quotidiennes et à ne s'engager que dans des actes qui favorisent un avantage pour la société.

### Qu'est-ce qu'un code d'éthique pour les enseignants ?

Un code d'éthique pour les enseignants définit les principales responsabilités d'un enseignant envers ses élèves et le rôle de l'enseignant dans la vie de l'élève. Les enseignants sont tenus de faire preuve d'impartialité, d'intégrité et d'un comportement éthique en classe.

### Qu'est-ce qu'un exemple de code d'éthique ?

Un exemple de code d'éthique serait une entreprise qui rédige un code décrivant toutes les manières dont l'entreprise doit agir avec honnêteté et intégrité dans ses opérations quotidiennes, de la façon dont ses employés se comportent et interagissent avec les clients, aux types de les personnes avec lesquelles elle fait affaire, y compris les fournisseurs et les agences de publicité.

### Quelle est la différence entre un code d'éthique et un code de conduite ?

Un code d'éthique est plus large dans sa nature, décrivant ce qui est acceptable pour l'entreprise en termes d'intégrité et comment elle fonctionne. Un code de conduite est de nature plus ciblée et indique comment les employés d'une entreprise doivent agir au quotidien et dans des situations spécifiques.

## L'essentiel

Un code d'éthique est un ensemble de principes directeurs visant à demander aux professionnels d'agir d'une manière honnête et bénéfique pour toutes les parties prenantes impliquées. Un code d'éthique est rédigé par une entreprise et adapté à l'industrie spécifique à portée de main, exigeant que tous les employés de cette entreprise adhèrent au code.

Les choix moraux des entreprises ont évolué, de l'ère industrielle à l'ère moderne. Dans le monde dans lequel nous vivons aujourd'hui, les conditions de travail, l'impact d'une entreprise sur l'environnement et la manière dont elle traite les inégalités sont autant de domaines que la société juge importants alors qu'il y a peut-être deux siècles, elle ne l'était pas autant. Un code d'éthique aide à garantir que les entreprises agiront toujours avec intégrité.

## Points forts

- Dans certains secteurs, dont la banque et la finance, des lois spécifiques régissent la conduite des affaires. Dans d'autres, un code d'éthique peut être volontairement adopté.

- Les principaux types de codes de déontologie comprennent un code de déontologie basé sur la conformité, un code de déontologie basé sur les valeurs et un code de déontologie entre professionnels.

- Un code d'éthique définit les directives éthiques d'une organisation et les meilleures pratiques à suivre pour l'honnêteté, l'intégrité et le professionnalisme.

- Pour les membres d'une organisation, la violation du code d'éthique peut entraîner des sanctions pouvant aller jusqu'au congédiement.

- L'accent mis sur le changement climatique est devenu une partie intégrante des codes d'éthique des entreprises, détaillant leur engagement envers le développement durable.
