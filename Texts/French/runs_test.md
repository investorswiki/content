# Exécute le test
## Qu'est-ce qu'un test d'exécution ?

Un test d'exécution est une procédure statistique qui examine si une [chaîne](/data-analytics) [de données](/data-analytics) se produit de manière aléatoire à partir d'une [distribution spécifique](/normaldistribution). Le test d'exécution analyse l'occurrence d'événements similaires séparés par des événements différents.

En investissement, un test de séquences peut être important pour les investisseurs afin de déterminer si l'ensemble de données qu'ils utilisent est généré de manière aléatoire ou s'il est impacté par une variable sous-jacente. Les traders qui se concentrent sur l'analyse technique peuvent utiliser un test d'exécution pour aider à analyser l' évolution du [prix](/price-action) d'un titre.

## Comprendre un test d'exécution

Une course est une série de valeurs croissantes ou décroissantes, souvent représentées sur un graphique par des symboles plus (+) ou moins (-). Dans les statistiques, un test d'exécution aide à déterminer le [caractère aléatoire des données](/random-factor-analysis) en découvrant toutes les variables susceptibles d'avoir un impact sur les modèles de données.

Par exemple, une liste de nombres à un chiffre vraiment aléatoires ne devrait avoir que quelques cas où une séquence de nombres augmente numériquement. Cependant, dans de nombreux cas, il est difficile d'affirmer le caractère aléatoire des données dans lesquelles il existe des milliers de séquences dans une chaîne de données. Ainsi, le test des séries a été créé comme une [méthode objective](/objective-probability) de détermination du caractère aléatoire.

## Types de tests d'exécution

Le test d'exécution est une version abrégée du nom complet : le test d'exécution de Wald-Wolfowitz, ainsi nommé d'après les mathématiciens Abraham Wald et Jacob Wolfowitz. Le test de Wald-Wolfowitz est un [test statistique non paramétrique](/nonparametric-statistics),. ce qui signifie que les données analysées ne doivent pas nécessairement répondre à certaines hypothèses ou paramètres. Le test de Wald-Wolfowitz peut être utilisé pour examiner l' [hypothèse selon](/hypothesistesting) laquelle les variables de la chaîne de données sont mutuellement indépendantes.

Certains statisticiens pensent qu'un autre type de test de séquences - le test de Kolmogorov – Smirnov - est un meilleur outil que le test de Wald-Wolfowitz pour détecter les différences entre les distributions. Le test de Kolmogorov-Smirnov est un type de [test d'adéquation](/goodness-of-fit) qui démontre si les données d'échantillon testées représentent [des modèles de distribution normaux](/normaldistribution) ou si les données sont en quelque sorte biaisées. Le test établit l'écart entre les valeurs des données d'échantillon et le modèle de distribution normale.

## Avantages d'un test d'exécution

Le modèle de test des séries est important pour déterminer si le résultat d'un essai est vraiment aléatoire, en particulier dans les cas où les données aléatoires par rapport aux données séquentielles ont des implications pour les théories et l'analyse ultérieures. Un test d'exécution peut être un outil précieux pour les investisseurs qui utilisent [l'analyse technique](/technicalanalysis) pour prendre leurs décisions de trading. Ces commerçants analysent les tendances statistiques, telles que le mouvement des prix et le [volume](/volume),. pour repérer les opportunités commerciales potentiellement rentables. Il est important pour ces commerçants de comprendre les variables sous-jacentes qui pourraient avoir un impact sur le mouvement des prix et un test d'exécution peut aider à cela.

Deux manières puissantes pour les traders d'utiliser un test d'exécution incluent :

1. Tester le caractère aléatoire de la distribution, en prenant les données dans l'ordre donné et en marquant d'un plus (+) les données supérieures à la [médiane](/median) et d'un moins (-) les données inférieures à la médiane (les nombres égaux à la médiane sont omis.)

1. Tester si une fonction correspond bien à un ensemble de données, en marquant les données dépassant la valeur de la fonction avec + et les autres données avec -. Pour cette utilisation, le test des courses, qui tient compte des signes mais pas des distances, est complémentaire du test du [chi carré](/chi-square-statistic),. qui tient compte des distances mais pas des signes.

## Points forts

- Un test d'exécution est une analyse statistique qui aide à déterminer le caractère aléatoire des données en révélant toutes les variables susceptibles d'affecter les modèles de données.

- Un test d'exécution, également connu sous le nom de test d'exécution de Wald-Wolfowitz, a été développé par les mathématiciens Abraham Wald et Jacob Wolfowitz.

- Les traders techniques peuvent utiliser un test d'exécution pour analyser les tendances statistiques et aider à repérer les opportunités de trading rentables.

- Par exemple, un investisseur intéressé par l'analyse du mouvement des prix d'une action particulière pourrait effectuer un test de séquences pour avoir un aperçu de l'action future possible des prix de cette action.
