# Mettez un put
## Qu'est-ce qu'un put sur un put ?

Une option de vente sur une option de vente est un [contrat d'options](/optionscontract) qui donne à son détenteur le droit de vendre un contrat d'options de vente sous-jacent. Le put sur un put trade est l'un des quatre types d' [options composées](/compoundoption).

Essentiellement, un put sur une option de vente est une option de vente d'une option. L'actif sous-jacent de l'option de vente d'une option de vente est l'option d'origine. Les options de vente sur un put sont plus courantes sur les bourses européennes qu'aux États-Unis.

## Mettez un put expliqué

L'acheteur d'une option de vente est un trader qui s'attend à ce que le prix de l'actif sur lequel l'option est basée baisse. Le commerçant achète une option de vente, généralement pour 100 actions, qui permet de vendre les actions à un certain prix (le [prix d'exercice) à une](/strikeprice) [date d'expiration](/expirationdate) spécifiée . Si le trader a raison et que l'actif baisse, l'option est [dans la monnaie](/inthemoney) et peut être exercée, le trader profitant de la différence entre le prix de l'option et le prix du marché.

Une option de vente sur une option de vente implique deux options de vente, l'une sur l'autre. Un put sur un put a deux prix d'exercice et deux dates d'expiration. L'un est pour l'option de vente composée et l'autre pour l' [option vanille sous-jacente](/vanillaoption).

Notez que les options composées sont plus courantes en Europe et que [les options européennes ne](/europeanoption) peuvent être exercées qu'à la date d'expiration. Une [option américaine](/americanoption) peut être exercée à la date d'expiration ou à tout moment avant celle-ci.

Comme l'une des variables qui déterminent le coût d'une option est le prix de l'actif sous-jacent, le coût d'un put sur une option put sera généralement inférieur au coût d'un put vanille sur l'actif correspondant. Ainsi, cela peut fournir un effet de levier au trader d'options.

## Quand utiliser un put sur un put

Une option de vente sur une option de vente est utilisée lorsqu'un trader souhaite utiliser un effet de [levier](/leverage). Le trader sera également modérément haussier sur l'actif sous-jacent. La valeur d'un put sur un put change en proportion directe avec le prix de l'actif sous-jacent. Cela signifie que la valeur augmente à mesure que le prix de l'actif augmente et diminue à mesure que le prix de l'actif diminue.

## Autres options composées

Les trois autres types d'options composées sont :

- **Call on a put** : il s'agit d'une option d'achat sur une option de vente sous-jacente. Le propriétaire qui exerce l'option d'achat reçoit une option de vente.

- **Call on a call** : dans cette option, l'investisseur achète une autre option d'achat avec des dispositions personnalisées. Ces dispositions comprennent le droit d'acheter une option d'achat classique sur un titre sous-jacent.

- **Put on a call** : l'investisseur doit livrer l'option d'achat sous-jacente au vendeur et percevoir une prime basée sur le prix d'exercice de l'option de vente sous-jacente.

Ces options sont également connues sous le nom d'options à frais partagés.

## Points forts

- Un put sur un put est un contrat d'options qui donne à son acheteur le droit de vendre un contrat d'options sous-jacent.

- Un put sur un put est essentiellement une option de vente d'une option.

- L'actif sous-jacent de l'option de vente d'une option de vente est une option de vente vanille.
