# Publicité mobile
## Qu'est-ce que la publicité mobile ?

Le terme publicité mobile fait référence à toute forme de publicité qui apparaît sur des appareils mobiles tels que les smartphones et les tablettes. Les entreprises font de la publicité sur ces appareils via des annonces textuelles par SMS ou via [des bannières publicitaires](/banneradvertising) qui apparaissent intégrées sur un site Web mobile. Ils peuvent également être trouvés via des applications téléchargées, y compris des jeux mobiles.

## Comment fonctionne la publicité mobile

La technologie moderne a offert aux consommateurs un plus large éventail d'options en ce qui concerne la façon dont ils consomment les médias. En fait, les gens passent désormais plus de temps sur leurs [smartphones](/smartphone),. tablettes et autres appareils mobiles qu'ils ne le font assis devant la télévision.﻿C'est en raison de la facilité et du coût de ces appareils, sans parler de la disponibilité des connexions sans fil permettant aux gens de taper dedans.

Afin de suivre l'évolution des habitudes de consommation, les entreprises ont adapté leurs campagnes publicitaires en adoptant des stratégies de publicité mobile. En effet, la probabilité que les clients nouveaux et existants voient une publicité via un appareil mobile est plus élevée qu'ils ne le feraient via les voies traditionnelles. Étant donné que les appareils mobiles ont généralement des écrans plus petits que les ordinateurs ou les ordinateurs portables, cette forme de publicité numérique est généralement optimisée pour les petits écrans.

La première forme de publicité mobile a eu lieu via des messages texte SMS. Mais les campagnes publicitaires pour mobile se sont rapidement transformées en publicités sur le Web mobile et [dans les applications](/inapp-purchasing). L'un des modèles populaires de publicité mobile est connu sous le nom de coût par installation (CPI), où le paiement est basé sur l'utilisateur installant une application sur son appareil mobile. Les réseaux publicitaires mobiles CPI fonctionnent soit comme incitatifs, soit comme non incitatifs. Dans le modèle d'incitation, l'utilisateur reçoit des points virtuels ou des récompenses pour installer le jeu ou l'application.

De nombreuses applications proposent une version gratuite qui peut être téléchargée gratuitement, mais qui est payée en plaçant des publicités dans l'application. Les versions mobiles des sites Web contiennent également des publicités optimisées pour les affichages mobiles, plus petites qu'elles n'apparaîtraient sur la version complète du même site Web.

> Les consommateurs peuvent supprimer les publicités en achetant des versions complètes ou premium d'une application.

>

## Considérations particulières

Les entreprises adaptent leurs publicités en fonction des goûts des consommateurs et/ou de leur historique de navigation. Par exemple, les publicités qui apparaissent sur des sites et des applications comme Facebook sont basées sur l'historique de navigation Web d'un individu, sa situation géographique et d'autres données telles que les habitudes d'achat en ligne.

Bien que la publicité mobile aide à maintenir les [revenus de l'entreprise](/revenue),. la vie privée des consommateurs suscite des inquiétudes. En effet, les entreprises utilisent [l'exploration de données](/datamining) et d'autres moyens pour collecter des informations sur les consommateurs lorsqu'ils utilisent leurs appareils. Les partisans soutiennent que les entreprises peuvent partager ou même vendre les données qu'elles collectent et utilisent avec d'autres.

## Types de publicité mobile

La publicité mobile peut prendre différentes formes. Ceux-ci inclus:

- Notifications push : ce sont des pop-ups qui apparaissent sur un appareil mobile. Ceux-ci sont livrés aux consommateurs à tout moment. Cela signifie que les utilisateurs n'ont pas besoin d'être sur une application pour recevoir une notification.

- Annonces textuelles et bannières illustrées : les utilisateurs qui cliquent sur leurs annonces sont redirigés vers la page de l'annonceur en l'ouvrant sur un navigateur.

- Annonces Click-to-download : lorsqu'un consommateur clique sur ces annonces, il les achemine vers l'application Google ou l'App Store d'Apple. La destination dépend du système d'exploitation et de l'appareil du consommateur.

- Annonces click-to-call : les annonceurs permettent aux utilisateurs de cliquer sur leurs annonces afin de les appeler directement avec leur smartphone.

- Annonces click-to-message : un consommateur qui clique sur ce type d'annonce est invité à contacter directement l'annonceur par SMS.

## Publicité mobile contre marketing mobile

La publicité mobile peut sembler similaire au [marketing mobile](/mobile-marketing),. mais les deux sont intrinsèquement différents. Le marketing mobile est un terme plus général qui englobe la publicité mobile. Bien qu'il utilise les données personnelles collectées, le marketing mobile utilise également des technologies telles que les services de localisation pour personnaliser les publicités en fonction des préférences, des habitudes ou de l'emplacement de l'utilisateur. Cela signifie que certaines publicités mobiles peuvent n'apparaître que lorsqu'un utilisateur mobile se trouve à proximité d'un certain magasin ou fournisseur de services.

Le placement d'annonces mobiles fonctionne au moyen d'un processus d'enchères programmatiques pour le placement d'annonces, dans lequel les annonceurs enchérissent en temps réel pour le droit de placer une annonce sur un appareil mobile. L'infrastructure qui permet ce processus est connue sous le nom de plate-forme côté demande (DSP). L'utilisation de telles plateformes permettra aux annonceurs d'optimiser leurs performances en fonction d'un certain nombre d' [indicateurs de performance clés](/kpi) (KPI), tels que le [coût par clic](/cpc) effectif (eCPC) et le coût par action effectif (eCPA).

## Points forts

- La publicité mobile est toute forme de publicité qui apparaît sur des appareils mobiles tels que les smartphones et les tablettes utilisant des connexions sans fil.

- Les entreprises font de la publicité par le biais d'annonces textuelles par SMS ou par le biais de bannières publicitaires qui apparaissent intégrées sur un site Web mobile.

- Les publicités sont personnalisées en fonction des goûts des consommateurs et/ou de l'historique de navigation à l'aide de l'exploration de données et d'autres techniques de collecte d'informations.
