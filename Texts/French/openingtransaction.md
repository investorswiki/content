# Transaction d'ouverture
## Qu'est-ce qu'une transaction d'ouverture ?

Une transaction d'ouverture, un terme généralement associé aux produits dérivés, fait référence à l'achat ou à la vente initial qui établit ou ouvre une nouvelle position. On peut [acheter pour ouvrir](/buytoopen) pour établir une position longue ou [vendre pour ouvrir](/selltoopen) une position courte. Une fois qu'une transaction d'ouverture a eu lieu, il existe donc une [position ouverte](/open-position).

L'opposé d'une transaction d'ouverture est appelé, de manière assez appropriée, une [transaction de clôture](/closeposition). Dans ce cas, on pourrait [vendre pour fermer](/selltoclose) une position longue existante ou [acheter pour fermer](/buytoclose) une position courte existante.

## Comprendre les transactions d'ouverture

En termes simples, une transaction d'ouverture est l'acte d'initier une nouvelle transaction. Cela peut impliquer la prise d'une nouvelle position sur un [titre spécifié](/security) ou l'entrée dans une variété de différentes positions de contrats dérivés qui restent ouvertes pendant une période de temps spécifiée. Le terme est généralement associé au trading d' [options .](/option) Les stratégies d'options, telles que la vente d'une option courte ou l'achat d'une option longue, seraient des exemples d'une transaction d'ouverture.

Une transaction d'ouverture est la première étape lors de la passation d'une transaction et implique l'achat d'un actif ou d'un instrument financier. Cela implique généralement, mais pas toujours, une transaction de clôture à un moment ultérieur, qui peut avoir lieu le même jour pour une transaction [intrajournalière](/intraday),. ou des jours, des semaines ou des mois plus tard pour un investissement à plus long terme. Une opération d'ouverture peut avoir des considérations différentes pour différents types d'investissements, et ces considérations seront considérablement différentes pour les titres cotés en bourse par rapport aux [dérivés](/derivative).

Plus rarement, une transaction d'ouverture peut également faire référence à la première transaction d'un titre spécifique un jour de bourse donné. Plus précisément, cela fait référence au prix négocié de ce titre, qui est important pour les investisseurs car il leur donne un moyen de comparaison avec le [cours de clôture](/closingprice) du jour de bourse précédent.

[intérêt ouvert](/openinterest) d'un contrat d'options montre combien de positions existent actuellement dans celui-ci.

## Titres cotés en bourse

Les investisseurs peuvent choisir d'investir dans un titre coté en bourse par le biais d'une transaction d'ouverture avec diverses motivations. Généralement, les investisseurs achètent un titre pour son potentiel de plus-value du capital ou de revenu. Les investisseurs peuvent voir un potentiel à long terme dans un titre en raison de ses caractéristiques de croissance ou de valeur au fil du temps. Ces motivations peuvent être motivées par les estimations de [revenus](/revenue),. le potentiel de bénéfices ou les ratios fondamentaux d'un titre.

Les investisseurs et, plus spécifiquement, les day traders ou [les analystes techniques](/technical-analyst) peuvent choisir d'entrer dans une position sur un titre par le biais d'une transaction d'ouverture pour ses gains à court terme. Les investisseurs à court terme entrent généralement dans un investissement avec un échéancier plus défini, cherchant à clôturer la position relativement rapidement pour profiter d'une [volatilité à court terme favorable](/volatility). Dans ce scénario, un investisseur peut ouvrir et clôturer une transaction en quelques heures, jours ou semaines.

## Positions sur dérivés

Une transaction d'ouverture qui engage un investisseur dans un contrat dérivé a une signification relativement plus importante à considérer qu'une transaction d'ouverture pour un titre coté en bourse. Lorsqu'un investisseur entre dans une position dérivée, il dispose d'un laps de temps spécifié pour générer des [bénéfices](/profit) à partir de l'investissement. Cela les oblige à suivre de plus près la position tout au long de sa vie.

Dans un contrat d' [option américain](/americanoption) [,](/americanoption) après une opération d'ouverture, un investisseur a le droit d'exercer ce contrat à tout moment jusqu'à l'expiration. Après expiration, le contrat est considéré comme fermé. Avec une [option européenne](/europeanoption),. le détenteur de l'option ne peut exercer l'option qu'à la date d'expiration. Pour les options américaines et européennes, l'investisseur peut également négocier son option sur le marché pour clôturer la position.

Dans un contrat à terme, un investisseur achète le dérivé pour exécution à une date spécifiée. Ils peuvent toujours vendre le contrat sur le marché libre jusqu'à l'expiration. S'ils détiennent le contrat jusqu'à son expiration, ils sont alors tenus de respecter les exigences du contrat, qui peuvent inclure [la](/physicaldelivery) [livraison](/physicaldelivery).

## Points forts

- Une transaction d'ouverture peut également faire référence à la première transaction d'un titre spécifique un jour de bourse donné qui établit le cours d'ouverture.

- Une transaction d'ouverture est une transaction qui initie une nouvelle position, longue ou courte, généralement dans le contexte des marchés dérivés.

- Les transactions d'ouverture sont finalement accompagnées de transactions de clôture qui enlèvent ou clôturent la position ouverte.
