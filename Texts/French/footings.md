# Semelles
## Que sont les semelles ?

En comptabilité, un footing est le solde final lors de l'addition de tous les débits et crédits. Les débits sont comptés, suivis des crédits, et les deux sont compensés pour calculer le solde du compte. Les semelles sont couramment utilisées en comptabilité pour déterminer les soldes finaux à mettre sur [les états financiers](/financial-statements).

## Comprendre les semelles

Au fur et à mesure que les transactions commerciales sont enregistrées, les comptables saisissent les montants pour chaque type de transaction dans l'une des deux colonnes. Chaque transaction est enregistrée en débit ou en crédit. Les colonnes de débit et de crédit sont additionnées, le total de chaque colonne constituant un pied.

Les deux semelles sont compensées ensemble pour calculer le solde du compte pour la période. Le solde du compte est le montant qui est reporté aux états financiers. Le processus est répété pour chaque type de transaction. Le terme "footing" est approprié car les totaux sont situés à la fin de chaque colonne.

## Points forts

- Les semelles sont couramment utilisées en comptabilité pour déterminer les soldes finaux des comptes, qui sont déclarés dans les états financiers d'une entreprise.

- Une fois que les débits et les crédits sont chacun totalisés, les deux nombres - ou semelles - sont compensés pour calculer le solde du compte.

- En comptabilité, un footing est le solde final lors de l'addition de tous les débits et crédits.
