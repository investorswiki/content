# Prix multiple
## Qu'est-ce qu'un multiple de prix ?

Un multiple de prix est un ratio qui utilise le prix de l'action d'une entreprise en conjonction avec une mesure financière spécifique par action pour un instantané de l'évaluation. Le prix de l'action est généralement divisé par une métrique par action choisie pour former un ratio. Les multiples de prix permettent aux investisseurs d'évaluer la valeur marchande des actions d'une entreprise par rapport à une [mesure fondamentale](/metrics),. telle que les bénéfices, les flux de trésorerie ou la valeur comptable.

## Comprendre les multiples de prix

Un multiple de prix donne aux investisseurs la possibilité de faire une [évaluation simple](/valuation) d'une entreprise. Les multiples de prix sont compris par les investisseurs du monde entier et sont acceptés comme une norme par toutes les parties intéressées par une action.

Les investisseurs expriment généralement un ratio multiple de prix dans le format suivant : multiple de prix = prix de l'action / métrique par action.

Le numérateur du ratio est le prix de l'action, qui est le prix auquel se vend une seule action d'une société à un moment donné. Le prix de l'action d'une entreprise est facilement déterminé simplement en regardant un tableau des prix des actions de l'entreprise.

Le dénominateur est la mesure par action utilisée pour le calcul du multiple de prix spécifique. La métrique mesure certains aspects de la performance d'une entreprise. Les investisseurs peuvent calculer ces mesures en utilisant les données [des états financiers d'une entreprise](/financial-statements) ou en trouvant les mesures dans le cadre des données historiques d'une entreprise sur un site de courtage.

## Types de multiples de prix

L'analyse des ratios aide les investisseurs à déterminer la santé financière d'une entreprise en évaluant ses performances au fil du temps. Les multiples de prix révèlent aux investisseurs des informations qui les aident à comparer différentes sociétés en tant qu'opportunités d'investissement potentielles. Certains multiples de prix courants sont le ratio cours/ [bénéfice](/price-earningsratio) (P/E), le ratio [cours/bénéfice à terme (P/E à terme), le ratio cours](/forwardpe) /valeur [comptable](/price-to-bookratio) (P/B) et le ratio cours/ventes ( rapport P/S).

Les investisseurs et les analystes utilisent certains ratios pour prédire les bénéfices et les performances futures. Les autres ratios incluent le ratio [prix/valeur comptable](/ptbv) (P/TBV), le ratio prix/ [cash flow](/price-to-cash-flowratio) (P/CF), le ratio prix/EBITDA (P/EBITDA) et le ratio prix/ [flux](/pricetofreecashflow) [de trésorerie disponible](/pricetofreecashflow) (P/FCF ). Ces multiples de prix sont faciles à calculer en surface, mais les investisseurs doivent prendre soin d'analyser les composants du dénominateur pour s'assurer que les chiffres sont exacts.

> Lorsqu'ils utilisent un multiple de prix pour évaluer la performance d'une entreprise, les investisseurs doivent s'assurer qu'il n'y a pas d'éléments extraordinaires, ponctuels ou de facteurs non récurrents susceptibles de fausser la mesure financière.

>

## Avantages des multiples de prix

Les multiples de prix jouent un rôle important en fournissant un aperçu statique et prospectif de la valorisation d'une action. Les multiples sont utilisés pour comparer les multiples de valorisation présents et futurs (prévisionnels) d'une entreprise avec ses chiffres historiques et avec ceux de ses pairs.

Les multiples de prix peuvent aider les investisseurs à déterminer si une action est [surévaluée](/overvalued),. sous-évaluée ou équitablement évaluée. Ces ratios séduisent les investisseurs car ils sont généralement faciles à comprendre et à utiliser. Les multiples de prix aident les investisseurs à déterminer ce qu'une action achète en termes de mesure de valeur, comme les flux de trésorerie ou les bénéfices.

## Considérations particulières

Les investisseurs ne doivent utiliser que les multiples de prix pertinents pour un secteur donné. Le ratio AP / E serait une mesure d'évaluation appropriée pour une entreprise technologique, mais pas nécessairement pour une entreprise de services publics à forte intensité de capital qui impute un montant important d'amortissement aux bénéfices. Dans ce cas, la charge hors trésorerie réduira les bénéfices selon [les principes comptables généralement reconnus](/gaap) (GAAP) et donc [le bénéfice par action](/eps) (EPS), ce qui peut donner une fausse impression sur l'évaluation de l'entreprise. Parfois, le P/E est jeté par la fenêtre.

Par exemple, les non-propriétaires d'actions Amazon qui recherchent un P/E raisonnable depuis des années peuvent avoir réalisé que le multiple P/E, ou son absence, dans le cas d'Amazon n'avait pas d'importance d'un iota. Cela pourrait se produire à l'avenir, mais les propriétaires d'actions Amazon qui ont ignoré ce multiple de prix sont clairement gagnants.

## Où trouver les multiples de prix

La plupart des sites Web financiers affichent des multiples de prix de base tels que P/E, P/B ou P/S. Les ratios sont généralement calculés sur la base [des douze derniers mois (TTM)](/ttm) ou de la dernière période civile. Pour l'investisseur plus sérieux, des calculs manuels de multiples pertinents pour un secteur particulier peuvent être effectués avec les données fournies par les entreprises dans leurs rapports financiers.

## Points forts

- Les investisseurs et les analystes utilisent des multiples de prix pour avoir un aperçu de la valorisation d'une entreprise dans le cadre du processus d'examen d'une entreprise pour un investissement potentiel.

- Les multiples de prix courants comprennent les ratios cours/bénéfice (P/E), les ratios cours/ventes (P/S) et les ratios cours/flux de trésorerie (P/CF).

- Un multiple de cours est un ratio qui utilise le cours de l'action d'une entreprise en combinaison avec une mesure financière par action.
