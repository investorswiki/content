# Table de mortalité actuarielle
## Qu'est-ce qu'une table de mortalité actuarielle ?

Une table de mortalité actuarielle est un tableau ou une feuille de calcul qui montre la probabilité qu'une personne d'un certain âge décède avant son prochain anniversaire. Il est souvent utilisé par les compagnies d'assurance-vie pour calculer l'espérance de vie restante des personnes à différents âges et stades, ainsi que la probabilité de survivre à une année d'âge donnée.

Étant donné que les hommes et les femmes ont des taux de mortalité différents, une table de mortalité actuarielle est calculée séparément pour les hommes et les femmes. Une table de mortalité actuarielle est également appelée table de [mortalité](/mortality-table),. table de mortalité ou table actuarielle.

## Comment fonctionne une table de mortalité actuarielle

Les compagnies d'assurance utilisent des tables de mortalité actuarielles pour établir le prix des produits et prévoir les futurs événements assurés. Les tables de mortalité actuarielles basées sur les mathématiques et les statistiques aident les compagnies d'assurance en montrant les probabilités d'événements, tels que le décès, la maladie et l'invalidité.

Une table de mortalité actuarielle peut également inclure des facteurs permettant de différencier des risques variables tels que le tabagisme, la profession, le statut socio-économique, voire le jeu, et [l'endettement](/debt-load). La modélisation prédictive informatisée permet aux actuaires de calculer pour une grande variété de circonstances et de résultats probables.

## La science actuarielle

La science actuarielle utilise principalement deux types de tables de mortalité. Premièrement, la table de mortalité périodique est utilisée pour déterminer les taux de mortalité pour une période de temps spécifique d'une certaine population. L'autre type de table de mortalité actuarielle est appelée table de mortalité par cohorte, également appelée table de mortalité par génération. Il est utilisé pour représenter les taux de mortalité globaux de la vie entière d'une certaine population.

> Les tables de mortalité actuarielles pour les hommes et les femmes sont calculées différemment en raison de l'écart des espérances de vie pour chaque sexe.

>

La sélection de la population doit être née pendant le même intervalle de temps spécifique. Une table de mortalité de cohorte est plus couramment utilisée car elle tente de prédire tout changement attendu des taux de mortalité d'une population dans le futur.

Un tableau de cohorte analyse également les schémas de mortalité observables au fil du temps. Les deux types de tables de mortalité actuarielles sont basées sur des populations réelles du présent et sur des prévisions éclairées du futur proche d'une population. D'autres types de tables de mortalité peuvent être basées sur des enregistrements historiques. Ces types de tables de mortalité sous-estiment souvent les nourrissons et sous-estiment la mortalité infantile.

Les compagnies d'assurance utilisent des tables de mortalité actuarielles pour faire principalement deux types de prévisions : la probabilité de survivre à une année d'âge donnée et l' [espérance de vie restante](/lifeexpectancy) pour les personnes d'âges différents.

## Autres utilisations des tables de mortalité actuarielles

Les tables de mortalité actuarielles jouent également un rôle important dans les sciences de la biologie et de l'épidémiologie. En outre, la Social Security Administration aux États-Unis utilise des tables de mortalité actuarielles pour examiner les taux de mortalité des personnes bénéficiant de la sécurité sociale afin d'éclairer certaines décisions ou actions politiques.

Les tables de mortalité actuarielles sont également importantes dans la gestion du cycle de vie des produits et pour le calcul des pensions.

## Points forts

- Les compagnies d'assurance utilisent des tables de mortalité actuarielles dans leur travail.

- Ces tables peuvent porter des noms différents comme une table de mortalité, une table actuarielle ou une table de mortalité.

- Les statistiques d'une table de mortalité actuarielle calculent, entre autres, la probabilité de survivre à une année d'âge donnée.

- La science actuarielle utilise principalement deux types de tables de mortalité.

## FAQ

### Que fait un actuaire ?

Les actuaires disent qu'ils sont des gestionnaires de risques et « des experts dans l'évaluation de la probabilité d'événements futurs ».

### Comment les tables actuarielles sont-elles utilisées ?

Généralement, ils sont utilisés par les compagnies d'assurance-vie pour calculer l'espérance de vie restante des personnes à différents âges et stades, ainsi que la probabilité de survivre à une année d'âge donnée.

### Quels sont les deux types de tables actuarielles ?

Les deux tables sont la table de mortalité de période (pour déterminer les taux de mortalité pour une période spécifique d'une population définie) et la table de mortalité de cohorte (utilisée pour représenter les taux de mortalité globaux de la vie entière d'une certaine population).
