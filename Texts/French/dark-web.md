# Web sombre
## Qu'est-ce que le Dark Web ?

Le terme dark web fait référence à un contenu en ligne crypté qui n'est pas indexé par les moteurs de recherche conventionnels. L'accès au dark web ne peut se faire qu'à l'aide de navigateurs spécifiques, tels que le navigateur TOR. Il y a beaucoup de confidentialité et d'anonymat qui accompagnent l'utilisation du Web sombre par rapport aux sites Web traditionnels.

En tant que tel, la majeure partie de l'attention est portée sur les [marchés en ligne](/market) de la drogue, [les échanges](/exchange) de données volées et d'autres activités illégales lorsque les gens pensent au dark web. Malgré cela, il existe souvent des raisons très légitimes pour lesquelles les gens choisissent d'utiliser le dark web, y compris les dissidents politiques et les personnes qui souhaitent garder certaines informations privées.

## Comprendre le Dark Web

Comme son nom l'indique, le dark web est un réseau secret qui existe sous terre. Il est composé d'une série de sites Web qui sont cachés au grand public. Cela signifie qu'ils ne sont pas accessibles via les moteurs de recherche traditionnels, tels que Google.

Les moteurs de recherche traditionnels renvoient des résultats car ils contiennent des index de liens vers des sites Web. Ceux-ci sont classés en fonction des mots clés et de la pertinence. Le dark web, quant à lui, utilise des informations qui ne sont pas disponibles sur ces autres moteurs de recherche, telles que le contenu de comptes individuels, tels que les e-mails, [les réseaux sociaux](/social-media),. les banques, ainsi que des bases de données personnelles et professionnelles, et des documents (juridiques et médical).

Aussi appelé darknet, le dark web ressemble beaucoup au web plus large à ses débuts à la fin du 20e siècle. Il y a beaucoup de matériel pour le faire fonctionner, et pas grand-chose à faire une fois sur place. Une grande partie du contenu sur le dark web est très amateur. D'un autre côté, il est beaucoup plus facile pour les particuliers de démarrer des sites et d'attirer l'attention. Les géants de la technologie et les grandes organisations médiatiques ont très peu d'influence sur le dark web à partir de 2020.

Comme pour les débuts d'Internet, le dark web a la réputation d'être un refuge pour les activités illégales et est fréquemment impliqué dans des activités illicites et criminelles. Bien que le dark web ait joué un rôle dans des transactions illégales et contraires à l'éthique, il offre également un débouché social aux personnes qui pourraient autrement être persécutées pour leur identité ou leurs convictions politiques. Il donne également aux autorités judiciaires les outils supplémentaires dont elles ont besoin pour appréhender les auteurs d'activités contraires à l'éthique.

> Le dark web est encore un travail en cours, et ses coûts et avantages ne sont pas encore connus.

>

## Considérations particulières

Les gens confondent souvent le dark web avec les [crypto -monnaies](/cryptocurrency),. qui sont souvent utilisées pour y effectuer des achats. Mais il y a une nette différence. Le dark web facilite la configuration et l'accès à des sites Web qui offrent un haut degré d'anonymat pour toutes les personnes impliquées.

De nombreux sites Web sombres ne contiennent que des informations plutôt que la possibilité d'acheter ou de vendre quelque chose. Il est vrai, cependant, que les crypto-monnaies, telles que [Bitcoin](/bitcoin) et [Monero](/monero),. sont souvent utilisées pour les transactions sur le dark web. Mais il n'est pas nécessaire d'utiliser le dark web pour utiliser des crypto-monnaies.

## Histoire du Dark Web

Le dark web est officiellement apparu pour la première fois au début des années 2000 avec la création de Freenet, qui a été développé par Ian Clarke pour protéger les utilisateurs contre les interventions gouvernementales et les cyberattaques. Le système, qui est toujours disponible aujourd'hui, permet aux utilisateurs de s'exprimer librement sans être suivis en ligne.

Le US Naval Research Laboratory a financé un projet appelé The Onion Router (TOR). TOR a offert aux sources de renseignement un moyen de communiquer facilement et en toute sécurité, en particulier dans les zones hostiles où la sécurité personnelle est essentielle. C'est maintenant l'un des navigateurs les plus couramment utilisés pour accéder au dark web, utilisant des bases de données pour aider les gens à se repérer et à trouver les informations dont ils ont besoin.

L'essor des crypto-monnaies a accru la popularité du dark web, en particulier pour les cybercriminels. En effet, les monnaies numériques offrent souvent un grand anonymat aux personnes qui achètent et vendent sur le dark web.

En raison de son association avec certaines activités illicites, des appels à la réglementation du dark web ont été lancés. Par exemple, le [G20](/g-20) et le [Groupe d'action financière (GAFI)](/financial-action-task-force-fatf) ont tous deux appelé les sociétés de crypto-monnaie à fournir des informations sur les acheteurs et les vendeurs dans les transactions effectuées en ligne. Cela est particulièrement vrai, disent-ils, pour aider les forces de l'ordre à suivre les organisations criminelles et les activités illicites.

> Ross Ulbricht a fait l'objet d'une enquête et a été arrêté en 2013 pour avoir créé et géré le site Web sombre The Silk Road, un site Web darknet qui, selon les autorités, était utilisé pour acheter et vendre des stupéfiants.

## Web sombre contre Web profond

Le dark web et le deep web sont également souvent utilisés à tort de manière interchangeable. Le dark web n'est en fait qu'une partie du deep web. Le Web profond est ce qui se cache sous terre et ce n'est pas seulement le contenu qui est considéré comme sombre. C'est pourquoi on l'appelle aussi le Web invisible ou le Web caché.

Les informations trouvées sur le Web profond sont normalement cryptées et ne se trouvent pas dans les index comme elles le sont sur le Web traditionnel. Il comprend toutes les pages qui ne s'affichent pas lorsque vous effectuez une recherche sur le Web. Il contient également tout ce qui nécessite une connexion, comme le contenu de :

- Services bancaires en ligne

- Sites Web payants, tels que Netflix et Amazon Prime

- Services d'hébergement de fichiers, tels que Dropbox et ses concurrents

- Bases de données privées

Les informations répertoriées sur le dark web peuvent également inclure tout ce qui se trouve dans des bases de données réservées aux abonnés plutôt que tout ce qui est illégal.

## Avantages et inconvénients du Dark Web

### Avantages

Le dark web aide les gens à préserver leur vie privée et à exprimer librement leurs opinions. La vie privée est essentielle pour de nombreuses personnes innocentes terrorisées par les harceleurs et autres criminels. La tendance croissante des employeurs potentiels à suivre les publications sur les réseaux sociaux peut également rendre difficile l'engagement public dans des discussions honnêtes.

Enfin, la popularité du dark web auprès des criminels en fait un moyen idéal pour les policiers en civil de communiquer.

### Désavantages

Certaines personnes abuseront inévitablement du pouvoir associé à l'utilisation du dark web en facilitant la participation à des activités criminelles. Par exemple, la combinaison du dark web et des crypto-monnaies facilite théoriquement l'embauche de quelqu'un pour commettre certains crimes.

Alors que le dark web promet la confidentialité à ses utilisateurs, il peut également être utilisé pour violer la vie privée des autres. Des photos privées, des dossiers médicaux et des informations financières ont tous été volés et partagés sur le dark web.

## Exemple du Dark Web

Disons que vous voulez aller sur le dark web pour voir quelles informations vous concernant, le cas échéant, circulent. Vous pouvez installer le navigateur TOR sur votre système en utilisant un réseau privé virtuel. Un VPN permet aux gens d'utiliser un réseau public comme s'ils transmettaient des informations sur un réseau privé. Cela vous donne essentiellement la confidentialité sur un réseau public.

En surfant, vous remarquerez probablement que les adresses de sites Web sombres se terminent par l'extension .onion plutôt que par les traditionnels .com, .org, .edu, etc. Un mot d'avertissement, cependant. Assurez-vous d'avoir installé un logiciel de sécurité, car il y a de fortes chances que vous soyez en contact avec des logiciels malveillants et des escroqueries par hameçonnage.

## L'essentiel

Le dark web est un concept relativement nouveau, du moins si l'on considère l'histoire d'Internet dans son ensemble. Vous pouvez l'utiliser pour rester anonyme sans crainte de représailles si vous souhaitez vous exprimer librement. Mais cette quantité de confidentialité l'a aidé à acquérir une mauvaise réputation en tant que forum pour ceux qui veulent l'utiliser pour des activités illicites. Si vous cherchez à utiliser le dark web, c'est aussi simple que d'installer un navigateur privé, d'utiliser un VPN et de vous assurer que votre ordinateur reste sûr et sécurisé.

## FAQ

### Comment arrivez-vous sur le Dark Web ?

Vous pouvez accéder au dark web en installant des navigateurs spécifiques et anonymes, tels que TOR. Une fois installé, le navigateur fonctionne de la même manière que les navigateurs traditionnels. Mais il peut être un peu difficile d'accéder aux informations car il n'utilise pas d'index pour localiser les informations souhaitées. Assurez-vous d'installer un logiciel de sécurité pour vous assurer que votre ordinateur et vos informations personnelles sont en sécurité.

### Que faites-vous si vos informations se trouvent sur le Dark Web ?

Vous pouvez prendre un certain nombre de mesures pour aider à protéger vos informations. Assurez-vous de changer tous vos mots de passe, vérifiez vos rapports de solvabilité pour toute divergence et informez votre ou vos banques. Vous pouvez également geler votre crédit pour éviter qu'il ne soit compromis davantage.

### Comment savoir si vos informations se trouvent sur le Dark Web ?

Vos informations personnelles, telles que les comptes bancaires, les réseaux sociaux et les comptes de messagerie, ainsi que votre numéro de sécurité sociale, peuvent être en vente sur le dark web. Si vous voulez voir si l'une de vos informations flotte sur le dark web, essayez d'exécuter une analyse du dark web. Alternativement, vous pouvez vous inscrire à un service de surveillance du dark web. Ces outils vous avertissent chaque fois qu'ils trouvent l'une de vos informations.

### Est-il illégal d'accéder au Dark Web ?

Malgré ce que son nom l'indique, il n'est pas illégal d'accéder au dark web. Il offre en fait aux individus une confidentialité et un anonymat que les sites Web traditionnels n'offrent pas aux individus. Par exemple, les gens peuvent aller sur le dark web et publier leurs réflexions sur l'activité politique sans craindre d'être réprimandés par des représentants du gouvernement et d'autres groupes.
