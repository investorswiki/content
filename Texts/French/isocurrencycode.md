# Code de devise ISO
## Qu'est-ce qu'un code de devise ISO ?

Les codes de devise ISO sont les codes alphabétiques à trois lettres qui représentent les différentes [devises](/currency) utilisées dans le monde. Lorsqu'ils sont combinés par paires, ils constituent les symboles et [les taux croisés](/crossrate) utilisés dans le commerce des devises.

Chacun des codes alphabétiques à trois lettres spécifiques à un pays a également un code numérique à trois chiffres correspondant. Ces codes sont identifiés par l' [Organisation internationale de normalisation](/international-organization-for-standardization-iso) (ISO), une organisation non gouvernementale qui fournit des normes pour la fabrication, le commerce, la technologie et la communication. Pour les devises, le document directeur est appelé ISO 4217:2015.

## Comprendre les codes de devise ISO

Les codes de devise ISO sont au cœur des [paires de devises](/currencypair),. qui sont les structures de cotation et de prix des devises négociées sur le [marché des changes](/forex-market). La valeur d'une devise est un taux et est déterminée par sa comparaison avec une autre devise.

Le premier code à trois chiffres utilisé pour désigner une devise dans une cotation d'une paire de devises est appelé [devise de base](/basecurrency),. et le deuxième code est appelé [devise de cotation](/quotecurrency). La paire de devises indique la quantité de devise de cotation nécessaire pour acheter une unité de la devise de base.

Par exemple, [EUR/USD](/eur-usd-euro-us-dollar-currency-pair) est la cotation de l'euro par rapport au dollar américain. EUR est le code de devise ISO à trois lettres pour l'euro et USD est le code pour le dollar américain. Un prix coté pour cette paire de 1,2500 signifie qu'un euro est échangé contre 1,2500 dollars américains car, dans ce cas, l' [EUR](/euro) est la devise de base et l'USD est la devise de cotation (ou contre-devise). Cela signifie que 1 euro peut être échangé contre 1,25 dollar américain. Une autre façon de voir cela est qu'il vous en coûtera 125 USD pour acheter 100 EUR.

L'ISO n'a pas été impliquée dans les transactions monétaires jusqu'en 1973, lorsque l'organisme de normalisation a décidé qu'il serait utile d'être impliqué. Après cinq ans de collaboration et de délibération, les premiers codes monétaires normalisés ont été publiés en 1978, avec une norme sur la façon dont ils devraient changer.

## Considérations particulières

Selon le site Web de l'ISO, "ISO 4217: 2015 spécifie la structure d'un code alphabétique à trois lettres et d'un code numérique à trois chiffres équivalent pour la représentation des devises. Pour les devises ayant des unités mineures, il montre également la relation décimale entre ces unités et la monnaie elle-même. Par exemple, le code numérique à trois chiffres pour le dollar américain est 840 et le code numérique pour l'euro est 978. Mais vous ne verriez pas de devises citées en utilisant des nombres comme celui-ci (978/840).

Bien que, comme l'explique le document ISO, "l'ISO 4217:2015 est destinée à être utilisée dans toute application du commerce, du commerce et de la banque, où les devises et, le cas échéant, les fonds doivent être décrits. Il est conçu pour être également adapté pour utilisateurs manuels et pour ceux qui utilisent des systèmes automatisés. » S'il était utile de le faire, les algorithmes de négociation ou de traitement des commandes pourraient utiliser des codes numériques pour un traitement plus efficace.

## Principaux codes de devise

Le site Web de l'ISO propose une liste complète des codes de devise aux formats XML et XMS. Toutes les principales paires de devises ont [des marchés très liquides](/liquidmarket) qui se négocient 24 heures sur 24, tous les jours ouvrables, et leurs [écarts sont très étroits](/spread).

Les principales paires de devises sont les suivantes :

- EUR/USD - euro/dollar américain

- [GBP/USD](/gbp-usd-british-pound-us-dollar-currency-pair) - [Livre britannique (sterling)](/gbp) / dollar américain

- USD/JPY - Dollar américain / [Yen japonais](/jpy-japanese-yen)

- USD/CHF - Dollar américain / [Franc suisse](/chf)

> La valeur échangée par rapport au dollar américain est le principal critère de classification en tant que paires de devises majeures.

>

Les autres devises importantes comprennent:

- AUD - [dollar australien](/aud-australian-dollar)

- CAD - [dollar](/cad-canadian-dollar) [canadien](/cad-canadian-dollar)

- CNY - [Renminbi yuan chinois](/cny-china-yuan-renminbi)

- NZD - [Dollar néo-zélandais](/nzd-new-zealand-dollar)

- INR - [Roupie indienne](/inr)

- BZR - [réal brésilien](/brl-brazilian-real)

- SEK - [couronne suédoise](/sek-swedish-krona)

- ZAR - [Rand sud-africain](/zar-south-african-rand)

- HKD - [Dollar de Hong Kong](/hkd-hong-kong-dollars)

## Points forts

- Les codes de devise ISO désignent les devises de base et de cotation dans les cotations de prix forex.

- Les comités de normalisation ISO ont établi des codes monétaires normalisés en 1978.

- Les codes de devise ISO sont les codes alphabétiques à trois lettres qui représentent les différentes devises utilisées dans le monde.

- L'ISO a également désigné des équivalents numériques moins connus des codes de devise à trois lettres.
