# Peau dans le jeu
## Qu'est-ce que la peau dans le jeu ?

Skin in the game est une expression rendue populaire par le célèbre investisseur Warren Buffett faisant référence à une situation dans laquelle des initiés de haut rang utilisent leur propre argent pour acheter des actions de la société qu'ils dirigent. Le dicton est particulièrement courant dans les affaires, la finance et les jeux de hasard et est également utilisé en politique.

## Comprendre la peau dans le jeu

Dans les affaires et le financement, le terme peau dans le jeu est utilisé pour désigner les propriétaires ou les [mandants](/principal) ayant une participation importante dans un véhicule d'investissement, comme les actions d'une société, dans laquelle des investisseurs extérieurs sont sollicités pour investir. Dans cette phrase, "peau" est une figure de style pour la personne ou l'argent impliqué, et "jeu" est la métaphore des actions sur le terrain de jeu en cours de discussion.

Il n'est pas rare qu'un dirigeant reçoive des actions en guise de rémunération ou exerce [des options](/stockoption) d'achat d'actions pour acheter une action à prix réduit. Ce qui est moins courant, c'est qu'un dirigeant risque son propre argent dans l'entreprise dans laquelle il travaille. Lorsqu'un dirigeant met sa peau dans le jeu, cela est considéré comme un signe de bonne foi ou une preuve de confiance dans l'avenir de l'entreprise. , et il est considéré comme un signe positif par les investisseurs extérieurs.

Si les directeurs ou les propriétaires ont également investi leur propre argent dans le véhicule d'investissement, les investisseurs potentiels et existants traduiront cette décision par un vote de confiance. La peau dans le jeu - ou la propriété d'initiés - indique également aux investisseurs que l'entreprise fera probablement de son mieux pour générer des rendements pour ses investisseurs.

L'idée derrière les dirigeants qui mettent leur peau dans le jeu est de s'assurer que les entreprises sont gérées par des personnes partageant les mêmes idées qui partagent une participation dans l'entreprise. Les dirigeants peuvent parler tant qu'ils veulent, mais le meilleur vote de confiance est de mettre son propre argent en jeu, tout comme les investisseurs extérieurs.

## Limitations de la peau dans le jeu

Cependant, certaines limites existent lorsque les propriétaires et les cadres supérieurs sont invités à investir leur propre argent dans un titre. De nombreuses banques et autres institutions financières interdisent aux employés d'avoir une "peau" où le capital des clients est géré. La restriction résout le problème du [front running](/frontrunning),. c'est-à-dire lorsqu'un dirigeant entre dans une transaction - avec des informations privilégiées ou non publiques - juste avant un événement ou une annonce pour obtenir un avantage économique.

Il existe également des restrictions sur les [fonds mixtes](/commingledfund),. c'est-à-dire la mise en commun des ressources ou le mélange de fonds privés et de ressources d'entreprise dans les actions ou les obligations de l'entreprise. Il existe des cas où les dirigeants doivent rester objectifs dans leur prise de décision et se voient interdire d'investir dans l'entreprise qu'ils gèrent.

## Exigences de divulgation pour la peau dans le jeu

La [Securities](/sec) [and Exchange Commission](/sec) (SEC) exige que les fonds divulguent chaque année le montant d'argent que chaque [gestionnaire de portefeuille](/portfoliomanager) a investi dans le fonds. En utilisant ces informations publiques, les partisans soutiennent que trouver des gestionnaires de fonds qui mettent leur argent là où ils le disent peut être un moyen fiable d'identifier les gestionnaires de fonds susceptibles de battre le marché à long terme. Les partisans de skin in the game soutiennent que l'engagement de capitaux est le moyen le plus important d'aligner les intérêts des investisseurs et des gestionnaires.

La SEC exige également des entreprises qu'elles fassent rapport sur la propriété ou les transactions d'initiés sur les titres d'une entreprise. Les rapports sont nécessaires car les transactions effectuées par les dirigeants, les administrateurs et les dirigeants peuvent avoir un impact sur le cours des actions de la société. Il existe différents types de formulaires que les dirigeants doivent déposer auprès de la SEC. Les investisseurs peuvent accéder à ces déclarations de propriété d'initiés et les utiliser pour prendre une décision plus éclairée quant à l'opportunité d'investir ou non dans l'entreprise.

## Points forts

- Skin in the game fait référence aux propriétaires, cadres ou dirigeants ayant une participation significative dans les actions de l'entreprise qu'ils gèrent.

- La SEC exige des entreprises qu'elles fassent rapport sur la propriété ou la négociation d'initiés sur les titres d'une entreprise, qui sont mis à la disposition du public.

- La peau dans le jeu est importante pour les investisseurs car elle montre que les dirigeants partagent un intérêt dans le succès de l'entreprise.
