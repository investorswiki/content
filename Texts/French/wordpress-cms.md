# Wordpress (CMS)
## Qu'est-ce que WordPress ?

WordPress est un système de gestion de contenu (CMS) open source populaire. Bien qu'à l'origine, il était principalement associé aux blogs personnels, il est depuis devenu utilisé pour une grande variété de sites Web, y compris des publications professionnelles et des plateformes [de commerce électronique](/ecommerce).

Depuis son lancement en 2003, WordPress s'est développé pour alimenter environ un tiers des sites Web dans le monde. Aujourd'hui, il abrite une grande communauté de développeurs qui a produit plus de 50 000 plugins.

## Comment fonctionne WordPress

Les utilisateurs de WordPress peuvent choisir parmi deux versions de base du service. La première d'entre elles est la version "hébergée" de WordPress, accessible sur WordPress.com. Cette version permet aux utilisateurs de lancer un site Web hébergé sur les propres serveurs de WordPress. En tant que tel, il est souvent préféré par les utilisateurs non techniques, qui souhaitent éviter la complexité de l'hébergement d'un site Web sur leur propre serveur privé. Comme avantage supplémentaire, la version hébergée de WordPress est automatiquement mise à jour vers la dernière version du logiciel WordPress, ce qui peut aider à atténuer les failles de sécurité potentielles.

Pour les utilisateurs plus avancés, il est également possible de simplement télécharger la dernière version de WordPress depuis WordPress.org. En sélectionnant cette option, l'utilisateur doit prendre ses propres dispositions pour héberger le logiciel soit sur son propre serveur privé, soit auprès d'un fournisseur d'hébergement [tiers .](/third-party) L'utilisateur est alors libre de personnaliser le logiciel comme il l'entend, mais doit également être responsable du téléchargement manuel des mises à jour ultérieures.

En raison de sa large base d'utilisateurs, les sites Web WordPress sont une cible populaire parmi les pirates, qui cherchent souvent à exploiter des versions obsolètes du logiciel qui pourraient posséder des vulnérabilités de sécurité connues. Pour cette raison, il est probablement prudent d'éviter d'auto-héberger le service, sauf si vous êtes confiant dans votre capacité à sécuriser et à mettre à jour le service avec les plugins supplémentaires que vous avez éventuellement installés.

> ### Personnalisations WordPress

> WordPress est conçu dès le départ pour être hautement personnalisable. Son [interface de programmation d'application (API)](/application-programming-interface) couvre tout, de sa base de données aux modifications et personnalisations de thème. En gardant tout standardisé, les développeurs peuvent se concentrer sur la création de fonctionnalités sur WordPress, sachant que cela fonctionnera sur toutes les installations qui sont tenues à jour.

>

## Exemple réel de WordPress

Il existe de nombreuses entreprises différentes qui fournissent des plugins et des thèmes WordPress conçus pour étendre ses fonctionnalités. Par exemple, WooCommerce est devenu l'un des outils de commerce électronique les plus populaires pour WordPress, permettant aux utilisateurs d'accepter des commandes, de suivre les expéditions et d'ajouter d'autres fonctionnalités de commerce électronique à leur site Web WordPress existant .

Parmi les nombreux autres exemples de plugins pour WordPress figurent ceux concernés par l'optimisation des moteurs de recherche (SEO), les formulaires de contact, le marketing par newsletter, les médias sociaux sur site et les fonctionnalités de commentaire, ainsi que diverses fonctionnalités de conception conçues pour rendre les sites plus attrayants visuellement.

## Points forts

- WordPress est une plate-forme CMS open source de premier plan.

- Il est connu pour sa nature hautement personnalisable, bénéficiant d'une grande communauté de développeurs de plugins.

- En raison de sa large base d'utilisateurs, WordPress est fréquemment la cible de pirates. Pour réduire ce risque, les utilisateurs doivent s'assurer que leur logiciel WordPress est constamment mis à jour.
