# Réseau Lightning
Une deuxième couche fonctionnant au-dessus d'une blockchain, permettant une vitesse de transaction accrue entre les nœuds participants. Il s'agit d'une solution de mise à l'échelle proposée.

## Points forts

- Tout comme la blockchain, le Lightning Network désintermédie les institutions centrales, telles que les banques, qui sont aujourd'hui responsables de l'acheminement de la plupart des transactions.

- Le Lightning Network est une solution technologique destinée à résoudre le problème de vitesse de transaction sur la blockchain bitcoin en introduisant des transactions hors registre.

- Le réseau de foudre a été formellement élaboré pour la première fois dans un article de Joseph Poon et Thaddeus Dryja en 2015.
