# Code d'autorisation
## Qu'est-ce qu'un code d'autorisation ?

Un code d'autorisation est un mot de passe alphanumérique qui autorise son utilisateur à acheter, vendre ou transférer des articles, ou à saisir des informations dans un espace sécurisé. Un code d'autorisation est généralement une séquence de lettres, de chiffres ou une combinaison des deux, qui valide l'identité d'une personne, approuve une transaction ou donne accès à une zone sécurisée.

L'utilisation la plus courante des codes d'autorisation est celle envoyée à un commerçant par les émetteurs de [cartes de](/creditcard) crédit, pour confirmer que la carte de crédit du client dispose d'un crédit suffisant pour autoriser la transaction.

> Différentes plates-formes peuvent imposer des exigences spécifiques lorsqu'elles demandent aux utilisateurs de créer des mots de passe ; par exemple, ils peuvent nécessiter l'utilisation d'un signe de ponctuation (majuscules et minuscules) ou des seuils de longueur minimale.

>

## Comprendre les codes d'autorisation

Les codes d'autorisation sont utilisés pour toute transaction ou entrée qui a des restrictions sur lesquelles les utilisateurs ont le droit d'accéder. Par exemple, un code d'autorisation de carte de crédit est un code à cinq ou six chiffres de la banque émettrice au vendeur, qui autorise la vente. Si la carte de crédit utilisée est contrefaite ou si la carte dépasse sa limite prédéterminée, la société de carte de crédit refusera automatiquement la vente. S'il est approuvé, le code d'autorisation est joint à la transaction par carte de crédit. Cela signale au commerçant que la transaction est légitime, tout en aidant à identifier la transaction lors d'examens de suivi, tels que le retour de marchandises ou les litiges d'achat.

Les codes d'autorisation sont transmis numériquement et sont utilisés pour accélérer [le traitement des cartes de crédit](/batch-credit-card-processing). Si les vendeurs devaient appeler l'émetteur pour obtenir un code d'autorisation verbal, afin de compléter chaque transaction, cela réduirait considérablement la vitesse du commerce.

## Codes d'autorisation dans les approbations de dépenses et la sécurité des données

Les codes d'autorisation jouent également un rôle dans [les contrôles financiers des entreprises](/internalcontrols). Le personnel peut recevoir différents codes d'autorisation pour les achats et les dépenses. Cela permet aux entreprises de suivre les achats et les dépenses dans des domaines spécifiques jusqu'au niveau des employés.

Ces codes d'autorisation sont également dotés de seuils de transaction spécifiques. Si un employé tente de dépenser quelque chose au-delà de son seuil d'allocation, il faudra qu'un code d'autorisation soit donné par un responsable, un superviseur ou un autre membre du personnel plus haut dans la hiérarchie organisationnelle. En ce sens, les codes d'autorisation font partie intégrante des mécanismes de contrôle qui peuvent être utilisés pour aider à lutter contre la fraude des employés ou le détournement de fonds.

Les codes d'autorisation sont également devenus couramment utilisés dans les lieux de travail professionnels pour maintenir la sécurité des informations. L'accès aux serveurs ou aux VPN peut être géré à l'aide de codes d'autorisation liés à des ID utilisateur uniques, afin de contrôler qui est autorisé à accéder aux bases de données sensibles.

Bien que ces codes d'autorisation puissent être utilisés de manière permanente pendant toute la durée du mandat d'un employé, ils sont plus souvent actualisés périodiquement, de la même manière que l'actualisation des contrôles de mot de passe. Il existe également des codes d'autorisation à usage unique ou des jetons qui ne durent que le temps d'une seule session.

## Points forts

- **Les codes d'autorisation sont des mots de passe alphanumériques qui autorisent les utilisateurs à effectuer diverses transactions.**

- **Dans le monde de l'entreprise, les membres du personnel peuvent recevoir différents codes d'autorisation pour les achats et les dépenses qui contiennent des seuils de transaction spécifiques.**

- ** Bien que les codes d'autorisation puissent être utilisés en permanence pendant toute la durée du mandat d'un employé, ils nécessitent le plus souvent un rafraîchissement de routine. **
