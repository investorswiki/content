# Dans le réservoir
## Que signifie "dans le réservoir" ?

"Dans le réservoir" est un terme d'argot faisant référence à de mauvaises performances prolongées. Dans le réservoir peut être utilisé dans le contexte d'un titre spécifique comme dans les actions d'une entreprise, un secteur ou une industrie en particulier, et même une économie nationale ou le marché dans son ensemble. Un stock est considéré comme étant dans le réservoir lorsqu'il s'est très mal comporté pendant un certain temps.

Notez qu'en politique, « dans le réservoir » en est venu à signifier à l'appui de (c'est-à-dire, à l'appui d'un candidat ou d'une politique).

## Comprendre dans le réservoir

Dans le réservoir peut être utilisé pour décrire un stock qui a connu une forte baisse sur une courte période de temps, mais cette situation est généralement considérée comme un réservoir - le début de la diapositive qui envoie le stock dans le réservoir. Une action qui a connu une mauvaise performance trimestrielle est décrite comme s'effondrant si le prix commence à baisser peu de temps après que la nouvelle a été rendue publique. Les manques à gagner sont un coupable courant qui fait chuter un stock. Un investisseur pourrait dire que ses investissements sont dans le réservoir, ce qui signifie qu'ils ne se portent pas bien. De même, un investisseur pourrait qualifier ses investissements de tanking lorsque les positions se détériorent. Les commentaires généraux sur le marché utilisent également ce terme, en particulier dans les médias. Par exemple, un expert pourrait faire référence à la [zone euro](/eurozone) étant dans le réservoir.

Dans le réservoir suggère que le stock, le secteur ou un autre groupement référencé a enregistré de faibles performances pendant une longue période. "Mon portefeuille est dans le réservoir", suggère que vous avez perdu de l'argent sur vos avoirs pendant plus d'une journée, tandis que "mon portefeuille est en train de s'effondrer" peut faire référence à une seule journée de négociation.

## Que faites-vous lorsque les investissements sont dans le réservoir ?

L'une des plus grandes questions auxquelles les investisseurs individuels sont confrontés est de savoir quoi faire lorsqu'une action ou une partie de leur portefeuille perd de l'argent. Avoir un stock qui est dans le réservoir n'est jamais agréable. Faut-il vendre et arrêter l'hémorragie ? Devriez-vous attendre et espérer que la prochaine nouvelle du marché apportera un [rallye](/rally) ?

En fin de compte, c'est à l'investisseur de décider, mais le conseil le plus courant est de ne vendre que des actions qui sont dans le réservoir si vous avez fait une erreur dans votre analyse ou si l'évaluation n'est plus justifiée par le prix en raison d'une érosion rapide des [fondamentaux](/fundamentals) que vous pris votre décision d'investissement initiale. Il convient de noter que les investisseurs à contre-courant passent souvent au crible les titres qui sont dans le réservoir pour trouver leurs meilleurs achats.

## Points forts

- Dans le réservoir est utilisé familièrement pour désigner un investissement qui a mal performé pendant un certain temps et dont il est peu probable qu'il se redresse.

- Une économie qui est en récession prolongée peut également être considérée comme étant dans le réservoir, avec de sombres perspectives.

- Si vous détenez un investissement qui est vraiment dans le réservoir, la meilleure chose à faire est probablement de le laisser partir et de passer à autre chose avant qu'il ne saigne davantage.
