# Chiffrer
En [cryptographie](/cryptography),. un chiffrement est une série d'instructions définies que l'on peut suivre pour chiffrer ou déchiffrer un message texte. Le processus de [cryptage](/encryption) consiste à convertir une information claire en une version illisible ou inaccessible. Le texte original qui peut être clairement compris est appelé texte en clair, tandis que sa forme cryptée est appelée texte chiffré. Les deux textes contiennent les mêmes informations, la seule différence est que le texte chiffré est écrit dans un format qui ne peut être lu ou accessible que par ceux qui disposent du [mécanisme de](/decryption) décryptage [correct .](/decryption)

La plupart des algorithmes de chiffrement impliquent l'utilisation d'une information secrète particulière, communément appelée clé cryptographique. Le schéma de chiffrement varie selon le modèle de clé, et les algorithmes de chiffrement peuvent être définis comme symétriques ou asymétriques. Les chiffrements symétriques utilisent une seule clé pour le chiffrement et le déchiffrement, tandis que les chiffrements asymétriques utilisent des clés différentes pour chaque opération.

Bien que la plupart des techniques de cryptage modernes soient effectuées par des ordinateurs, les chiffrements étaient déjà utilisés pour coder les messages depuis avant les anciens Grecs, vers 400 av. Le célèbre politicien romain Jules César a fait un grand usage des chiffres de substitution, remplaçant chaque lettre d'un message par la lettre située trois endroits plus bas dans l'alphabet.

Par exemple, si nous utilisons la même technique pour chiffrer le mot BINANCE, le texte chiffré résultant serait ELQDQFH. Ce message semble dénué de sens pour un tiers mais peut être facilement déchiffré par le destinataire prévu puisqu'il connaît déjà le numéro de quart utilisé par l'expéditeur pour le chiffrer.
