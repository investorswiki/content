# Code américain pour l'échange d'informations (ASCII)
## Qu'est-ce que le code américain pour l'échange d'informations (ASCII) ?

Le code américain pour l'échange d'informations, communément appelé ASCII, est un langage informatique pour le texte qui est devenu la norme pour la transmission d'informations entre ordinateurs en 1963.

Le système ASCII traduit les caractères alphanumériques et les symboles en un code lisible par ordinateur. L'ASCII d'origine était uniquement en anglais. Il existe maintenant deux types de codes ASCII ; le code standard qui utilise un système de codage à sept bits et un code étendu qui utilise un système à huit bits.

Dans les communications Internet, l'ASCII a été progressivement remplacé par la norme Unicode, qui peut être utilisée avec n'importe quelle langue et est compatible avec l'ASCII. Cependant, ASCII est toujours utilisé pour saisir du texte sur des ordinateurs personnels et des appareils électroniques.

Il se prononce ASK-y.

## Comprendre le code américain pour l'échange d'informations (ASCII)

La première itération d'ASCII se trouve dans un code télégraphique utilisé par les services de données de Bell pour transmettre des messages aux téléimprimeurs.

ASCII a été le premier système majeur de codage de caractères pour le traitement des données et a été adopté comme norme pour l'informatique en 1963 et pour la transmission Internet en 1969. Il a été utilisé pendant de nombreuses années dans les systèmes informatiques de saisie des commandes utilisés par les commerçants et les courtiers de Wall Street.

Il a été progressivement remplacé comme norme pour Internet par l'Unicode Worldwide Character Standard (Unicode). Cependant, comme les 128 premiers caractères d'Unicode sont les mêmes que ceux d'ASCII, on peut dire que les deux systèmes coexistent.

> Vous pouvez entrer un symbole ASCII dans un document à l'aide de n'importe quel ordinateur compatible Windows en maintenant la touche ALT enfoncée tout en tapant le numéro de code du caractère. Par exemple, maintenir la touche ALT tout en tapant 156 vous donnera £, le signe de la livre sterling.

### Qu'y a-t-il en ASCII

Avant l'adoption de l'ASCII comme norme, chaque fabricant d'ordinateurs utilisait son propre code et certaines entreprises en utilisaient plusieurs pour différents produits. Les ordinateurs étaient incapables de communiquer entre eux.

Bien que l'ASCII ait été adopté comme norme par les normes nationales américaines en 1963, il n'a été pleinement mis en œuvre même aux États-Unis qu'en 1968, lorsque le président Lyndon B. Johnson a mandaté son adoption par le gouvernement fédéral afin de garantir que l'industrie des ordinateurs et des télécommunications du gouvernement les ordinateurs pouvaient communiquer entre eux.

L'ASCII contenait à l'origine 128 caractères alphabétiques, chiffres et symboles et était basé sur la langue anglaise. Des formulaires étendus ont été ajoutés dans les années 1970 pour accueillir d'autres langues.

Bientôt, diverses versions d'ASCII furent utilisées. Il comprenait finalement 255 caractères. Cette version est connue sous le nom d'ASCII étendu.

Aujourd'hui, les utilisateurs peuvent saisir des symboles ASCII ou Unicode tels qu'un symbole de copyright dans des documents en appuyant sur la touche alt et en saisissant le code de caractère sur le pavé numérique avec la touche de verrouillage NUM activée.

## Points forts

- ASCII continue d'exister mais a été largement remplacé par Unicode, qui peut être utilisé pour coder n'importe quelle langue.

- ASCII ne contenait à l'origine que 128 lettres et symboles de langue anglaise, mais a ensuite été étendu pour inclure des caractères supplémentaires, y compris ceux utilisés dans d'autres langues.

- Le code américain pour l'échange d'informations, ou ASCII, est un format de codage de caractères conçu pour être utilisé dans la transmission de texte entre ordinateurs.

## FAQ

### Qu'est-ce qu'ASCII ou Unicode ?

Unicode pourrait être considéré comme une version universelle d'ASCII. ASCII est, après tout, le code américain pour l'échange d'informations, et sa première itération comprenait l'alphabet de langue anglaise et les symboles utilisés dans la langue. Les versions ultérieures incluaient des symboles utilisés dans de nombreuses autres langues. Unicode, en revanche, commence par les mêmes 128 symboles utilisés en ASCII, mais les versions actuelles contiennent 144 697 caractères.

### Qu'est-ce que l'ASCII ?

L'American Standard Code for Information Interchange, ou ASCII, est un format de codage de caractères pour la transmission électronique de texte. Chaque caractère est représenté par un numéro unique. La première version d'ASCII ne contenait que 128 caractères, représentant les lettres de l'alphabet, majuscules et minuscules, ainsi qu'un certain nombre de symboles couramment utilisés tels que la virgule. Les versions ultérieures ont étendu l'ASCII à 255 caractères, y compris des symboles supplémentaires tels que le Symbole de la livre sterling (£) et le point d'interrogation à l'envers utilisé dans le texte espagnol (¿).

### Que sont les caractères ASCII ?

Dans la table ASCII étendu, les codes de caractère 0 à 31 sont des codes de contrôle tels que le début du texte et le retour arrière. les codes 128 à 255 sont disponibles en plusieurs versions, chacune d'elles reflétant des ajouts à l'ASCII original uniquement en anglais, tels qu'un point d'exclamation inversé, le signe yen et la virgule géorgienne.
