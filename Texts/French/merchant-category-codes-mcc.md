# Codes de catégorie de commerçant (MCC)
## Que sont les codes de catégorie de marchand (MCC) ?

Les codes de catégorie de commerçant (MCC) sont des numéros à quatre chiffres qu'un émetteur de carte de crédit utilise pour catégoriser les transactions que les consommateurs effectuent à l'aide d'une carte particulière. Les marques de paiement utilisent des codes de catégorie de marchands pour classer les marchands et les entreprises selon le type de biens ou de services fournis afin de suivre et de restreindre les transactions.

Les MCC peuvent être utilisés pour la déclaration fiscale, la promotion des échanges et la collecte d'informations sur le comportement d'achat des titulaires de carte.

Souvent, les MCC sont reconnus par tous les émetteurs de cartes de crédit ; cependant, tous les codes ne sont pas reconnus par tous. Les émetteurs ajoutent, éliminent ou modifient fréquemment les codes avec des améliorations régulières.

## Comprendre les codes de catégorie de marchand

Les codes de catégorie de commerçant ont plusieurs objectifs. Ils déterminent souvent les récompenses que les consommateurs reçoivent pour l'utilisation de leurs cartes de crédit et déterminent si une transaction commerciale doit être signalée à l' [IRS](/irs). De plus, ils déterminent le pourcentage de chaque transaction qu'une entreprise doit payer au processeur de carte de crédit. Les exemples suivants sont des utilisations courantes des codes de catégorie de commerçant.

- Si un consommateur détient une carte de crédit qui offre une remise de 5 % sur les compagnies aériennes, il devrait recevoir la récompense sur tout achat classé sous MCC 4511, qui concerne les compagnies aériennes et les transporteurs aériens.

- Les entreprises et les agences gouvernementales déclarent les achats de services à l'IRS afin que l'IRS puisse s'assurer que ces services paient tous les impôts sur le revenu dus. Si les entreprises effectuent ces achats à l'aide d'une carte de crédit, les entreprises peuvent utiliser les MCC pour déterminer quelles transactions sont classées comme des services.

- Une entreprise classée dans le MCC pour les stations-service paie parfois des [frais d'interchange différents](/interchange-rate) à son processeur de carte de crédit qu'une entreprise classée comme une société de location de voitures.

## Comment les MCC peuvent augmenter les récompenses des cartes de crédit

Les personnes possédant des cartes de récompenses gagnent généralement plus de récompenses si elles connaissent leurs MCC. Supposons que vous ayez une carte de crédit offrant 5 points par dollar dépensé dans les restaurants. La façon dont la société de carte de crédit détermine si les transactions par carte de crédit ont eu lieu dans un restaurant est en consultant les MCC. Si vous achetez un déjeuner dans un petit établissement familial qui combine un restaurant et une épicerie, et que le MCC classe l'établissement comme une épicerie, vous ne gagnerez pas 5 points par dollar sur ce que vous pensiez être un achat au restaurant. .

Cependant, si vous visitez fréquemment cet établissement et que vous connaissez son MCC associé, vous pouvez utiliser une carte de crédit différente, peut-être une qui vous donne 3 % de remise sur les achats à l'épicerie, pour maximiser votre remise en argent.

Une autre possibilité est qu'un émetteur de carte de crédit ne vous donne pas le montant correct de points ou de remise en argent même pour une transaction qui a un MCC qui devrait déclencher le bonus. Dans ce cas, vous pouvez contacter la compagnie de carte de crédit et lui demander de corriger l'erreur.

Pour savoir comment une transaction est catégorisée, il suffit de regarder votre relevé de carte de crédit. Sous "catégorie", vous verrez comment la société de carte de crédit a classé la transaction. Plutôt que d'afficher le MCC, la déclaration énumère généralement le nom de la catégorie (comme "épiceries" ou "pharmacies") afin que vous puissiez facilement comprendre les informations.

## Points forts

- Les MCC sont utilisés par les émetteurs de cartes de crédit pour identifier le type d'activité dans laquelle un commerçant est engagé.

- Les codes de catégorie de marchand (MCC) sont des nombres à quatre chiffres qui décrivent les principales activités commerciales d'un marchand.

- Les MCC sont utilisés pour suivre les habitudes de consommation et attribuer des points de carte de crédit pour les achats éligibles.
