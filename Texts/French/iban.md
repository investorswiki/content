# Numéro de compte bancaire international (IBAN)
## Qu'est-ce qu'un numéro de compte bancaire international (IBAN) ?

Un IBAN, ou numéro de compte bancaire international, est un système de numérotation international standard développé pour identifier un compte bancaire à l'étranger. Le numéro commence par un code de pays à deux chiffres, suivi de deux chiffres, suivis de plusieurs autres caractères alphanumériques. Notez qu'un IBAN ne remplace pas la numérotation de compte d'une banque, car il est uniquement destiné à fournir des informations supplémentaires qui aident à identifier les paiements à l'étranger.

## Comment fonctionnent les numéros de compte bancaire international (IBAN)

Le numéro IBAN se compose d'un code de pays à deux lettres, suivi de deux chiffres de contrôle et jusqu'à trente-cinq caractères alphanumériques. Ces caractères alphanumériques sont connus sous le nom de numéro de compte bancaire de base (BBAN). Il appartient à l'association bancaire de chaque pays de déterminer le BBAN qu'elle choisira comme norme pour les comptes bancaires de ce pays.

Un numéro IBAN sera utilisé lors de l'envoi de virements interbancaires ou de transferts [d'argent d'une banque](/wiretransfer) à une autre, en particulier au-delà des frontières internationales. Dans le registre des pays utilisant actuellement le système IBAN, plusieurs exemples sont les suivants :

- Albanie : AL47 2121 1009 0000 0002 3569 8741

- Chypre : CY 17 002 00128 0000001200527600

- Koweït : KW81CBKU0000000000001234560101

- Luxembourg : LU 28 001 9400644750000

- Norvège : N° 93 8601 1117947

> Les États-Unis et le Canada sont deux grands pays qui n'utilisent pas le système IBAN ; cependant, ils reconnaissent le système et traitent les paiements conformément au système.

>

## IBAN contre codes SWIFT

Il existe deux méthodes normalisées reconnues à l'échelle internationale pour identifier les comptes bancaires lorsqu'un virement est effectué d'un pays à un autre : le numéro de compte bancaire international (IBAN) et le code SWIFT ( [Society for Worldwide Interbank Financial Telecommunication ).](/swift) La différence entre les deux méthodes réside dans ce qu'elles identifient.

Un code SWIFT est utilisé pour identifier une banque spécifique lors d'une transaction internationale, tandis que l'IBAN est utilisé pour identifier un compte individuel impliqué dans la transaction internationale. Tous deux jouent un rôle essentiel dans le bon fonctionnement du marché financier international.

Le système SWIFT est antérieur aux tentatives de normalisation des transactions bancaires internationales via l'IBAN. Il reste la méthode par laquelle la majorité des transferts de fonds internationaux sont effectués. L'une des principales raisons à cela est que le système de messagerie SWIFT permet aux banques de partager une quantité importante de données financières.

Ces données comprennent l'état du compte, les montants débiteurs et créditeurs et les détails liés au transfert d'argent. Les banques utilisent souvent le code d'identification bancaire (BIC) au lieu du code SWIFT. Cependant, les deux sont facilement interchangeables ; les deux contiennent un mélange de lettres et de chiffres et comportent généralement entre huit et 11 caractères.

## Exigences pour les numéros de compte bancaire international

L'IBAN s'est développé à partir de normes nationales divergentes pour l'identification des comptes bancaires. Les utilisations variables des formulaires alphanumériques pour représenter des banques, des succursales, des codes d'acheminement et des numéros de compte spécifiques ont souvent conduit à des interprétations erronées et/ou à des omissions d'informations critiques sur les paiements.

Pour faciliter ce processus, l'Organisation internationale de normalisation (ISO) a publié la norme ISO 13616: 1997 en 1997. Peu de temps après, le Comité européen des normes bancaires (ECBS) a publié une version plus petite, estimant que la flexibilité initiale autorisée dans la version ISO était irréalisable. Dans la version de l'ECBS, ils n'autorisaient que les lettres majuscules et un IBAN de longueur fixe pour chaque pays.

Depuis 1997, une nouvelle version, l'ISO 13616:2003, a remplacé la version ECBS initiale. Une version ultérieure en 2007 stipulait que les éléments IBAN devaient faciliter le traitement des données à l'échelle internationale, tant dans les environnements financiers que dans d'autres secteurs ; cependant, il ne spécifie aucune procédure interne, y compris, mais sans s'y limiter, les techniques d'organisation des fichiers, les supports de stockage ou les langues.

## Points forts

- Un numéro de compte bancaire international (IBAN) est un système de numérotation international standard pour les comptes bancaires individuels dans le monde entier.

- Les banques en Europe ont initialement développé le système pour simplifier les transactions impliquant des comptes bancaires d'autres pays.

- Un IBAN est utilisé pour identifier un compte individuel impliqué dans une transaction internationale.

- L'IBAN sert également de méthode de vérification de l'exactitude des détails de la transaction.

## FAQ

### Pourquoi l'IBAN a-t-il été créé ?

L'IBAN a été développé pour réduire les erreurs et améliorer la vérification des paiements transfrontaliers en réduisant les paiements rejetés, les retards de transfert et les frais et commissions bancaires associés.

### Qui utilise un IBAN ?

L'IBAN a d'abord été créé pour faciliter les paiements électroniques entre les banques de la zone euro. Depuis lors, il s'est étendu dans le monde entier, bien que toutes les banques et toutes les régions n'aient pas rejoint la norme et que vous deviez peut-être encore vous fier à un système alternatif tel que SWIFT à la place. Les pays d'Amérique du Nord, d'Australie et d'Asie n'utilisent pas l'IBAN pour les transferts d'argent nationaux et ne le feront que lors de l'envoi d'un paiement vers un pays qui a adopté l'IBAN.

### À quoi ressemble un numéro IBAN ?

Un numéro IBAN contient jusqu'à 34 caractères alphanumériques. Il est précédé d'un code de pays à deux caractères, de deux chiffres de contrôle et d'un numéro de compte bancaire de base (BBAN) qui contient des informations bancaires et de compte spécifiques. Le format de la partie BBAN varie d'un pays à l'autre et comprend généralement un code bancaire et un code de succursale.

### Comment puis-je obtenir un IBAN ?

Vous pouvez demander un IBAN si vous êtes client d'une banque dans une région IBAN. Notez qu'un IBAN ne peut être utilisé que pour recevoir des paiements et n'est pas utilisé lors de retraits.
