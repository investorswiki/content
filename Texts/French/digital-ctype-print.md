# Impression numérique de type C
## Qu'est-ce qu'une impression numérique de type C ?

Une impression couleur numérique de type C ou chromogénique est toute impression photographique qui a été créée par un système d'exposition numérique, par opposition à une chambre noire traditionnelle ou à une technique analogique. Une impression numérique de type C est développée en exposant un matériau photosensible à des LED ou à des lasers, le matériau étant ensuite lavé à l'aide de méthodes similaires à la photographie traditionnelle. Le C signifie chromogénique.

## L'impression numérique de type C expliquée

Dans une chambre noire traditionnelle ou un réglage analogique, un agrandisseur projette l'image d'un négatif photographique sur une feuille de papier photographique, tout en contrôlant la mise au point, l'intensité de l'image et la durée d'exposition à la lumière. Un agrandisseur est un appareil optique, un outil photographique qui s'apparente à un projecteur de diapositives.

Le processus de chambre noire traditionnel est différent d'un processus d'impression numérique de type c ou chromogénique. Avec les tirages numériques de type C, le travail généralement effectué par un agrandisseur est plutôt effectué par un ordinateur, le technicien contrôlant les mêmes facteurs : mise au point, intensité et durée d'exposition à la lumière. Dans ce cas, le papier est exposé à l'aide de lasers ou de LED plutôt qu'avec une ampoule traditionnelle. LED signifie diode électroluminescente et est une source qui émet une lumière lorsqu'un courant la traverse, contrairement à une ampoule standard.

Une fois l'image exposée, que ce soit par des moyens traditionnels ou numériques, l'étape suivante est à peu près la même. Il comprend toujours ce qu'on appelle un procédé chimique par voie humide. Le papier qui contient l'image est traité dans un révélateur photographique, puis soumis à un blanchiment-fixage, avant d'être finalement lavé à l'eau pour éliminer les produits chimiques de traitement. L'image est ensuite laissée à sécher et peut être numérisée, recadrée, éditée et modifiée.

## L'impression numérique de type C diffère d'une impression Giclée

Une impression numérique de type C ou chromogénique est une image traditionnelle ou une épreuve photographique qui a été réalisée à partir d'un fichier numérique plutôt qu'un négatif. Une impression chromogénique est parfois confondue avec une impression Giclée, mais elles sont différentes. Une impression Giclée est réalisée sans utiliser de chimie ni de sensibilité à la lumière. Ce type d'impression combine des encres à base de pigments avec du papier de type archive de haute qualité, ce qui donne une impression jet d'encre d'une qualité particulièrement élevée.

## Impression numérique de type C par rapport aux impressions à jet d'encre

Les impressions numériques de type C diffèrent des impressions à jet d'encre car les impressions à jet d'encre utilisent de fines gouttelettes d'encre plutôt que des sources de lumière, comme un laser. Les machines utilisées pour les impressions numériques de type C peuvent être beaucoup plus chères que les imprimantes à jet d'encre et ont tendance à être utilisées dans des environnements commerciaux. La longévité des impressions numériques de type C est également estimée plus courte que celle de l'impression à base de pigments, et le nombre de matériaux pouvant être imprimés avec ce procédé est plus limité.

Les autres termes utilisés pour l'impression numérique de type C incluent laboratoire photo, C numérique, chromogénique laser, RA-4 numérique, colorant chimique, impression de laboratoire ou C-print.

## Exemple d'impression numérique de type C dans le monde réel

Les C-prints numériques sont des tirages photographiques de laboratoire. Ils sont produits par ce qu'on appelle des minilabs. Les exemples courants incluent le Fuji Frontier, une machine haut de gamme qui ressemble à une imprimante d'ordinateur traditionnelle mais qui est conçue pour produire des impressions de meilleure qualité. La série 7700 peut produire jusqu'à 2 360 impressions haute résolution par heure et jusqu'à 620 impressions 8"X10" par heure. D'autres exemples de processeurs capables de produire des impressions numériques de type C haut de gamme incluent les imprimantes photo grand format comme la LightJet ou la Lambda, qui utilisent des papiers tels que Fuji Crystal Archive ou Kodak Endura.

## Points forts

- Avec l'impression numérique, le travail effectué traditionnellement par un appareil optique appelé agrandisseur, est plutôt effectué par un ordinateur.

- L'impression couleur numérique de type C ou chromogénique est une impression qui a été créée numériquement, par opposition à la méthode traditionnelle.

- Une fois l'image projetée sur une feuille de papier, le papier est traité dans un révélateur, puis javellisé, puis débarrassé des produits chimiques de traitement.
