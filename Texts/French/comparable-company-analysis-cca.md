# Analyse des sociétés comparables (CCA)
## Qu'est-ce qu'une analyse d'entreprise comparable (CCA)

Une analyse d'entreprise comparable (CCA) est un processus utilisé pour évaluer la valeur d'une entreprise en utilisant les paramètres d'autres entreprises de taille similaire dans le même secteur. L'analyse des sociétés comparables repose sur l'hypothèse que des sociétés similaires auront des multiples de valorisation similaires, tels que [EV/EBITDA](/ebitda-ev-multiple). Les analystes compilent une liste des statistiques disponibles pour les sociétés examinées et calculent les multiples de valorisation afin de les comparer.

## Comprendre l'analyse des sociétés comparables (CCA)

L'une des premières choses que chaque banquier apprend est de savoir comment faire une analyse comparative ou une analyse d'entreprise comparable. Le processus de création d'une analyse d'entreprise comparable est assez simple. Les informations fournies par le rapport sont utilisées pour déterminer une estimation approximative de la valeur du cours de l'action ou de la valeur de l'entreprise.

## Analyse d'entreprise comparable

L'analyse des entreprises comparables commence par l'établissement d'un [groupe de pairs](/peer-group) composé d'entreprises similaires de taille similaire dans le même secteur ou la même région. Les investisseurs sont alors en mesure de comparer une entreprise particulière à ses concurrents sur une base relative. Ces informations peuvent être utilisées pour déterminer la [valeur d'entreprise](/enterprisevalue) (EV) d'une entreprise et pour calculer d'autres ratios utilisés pour comparer une entreprise à celles de son groupe de pairs.

## Analyse d'entreprise relative vs comparable

Il existe plusieurs façons d'évaluer une entreprise. Les approches les plus courantes sont basées sur les flux de [trésorerie](/cashflow) et la performance relative par rapport aux pairs. Les modèles basés sur la trésorerie, tels que le modèle d' [actualisation des flux de trésorerie](/dcf) (DCF), peuvent aider les analystes à calculer une [valeur intrinsèque](/intrinsicvalue) basée sur les flux de trésorerie futurs. Cette valeur est ensuite comparée à la valeur marchande réelle. Si la valeur intrinsèque est supérieure à la valeur de marché, le titre est sous-évalué. Si la valeur intrinsèque est inférieure à la valeur de marché, le titre est surévalué.

En plus de l'évaluation intrinsèque, les analystes aiment confirmer l'évaluation des flux de trésorerie avec des comparaisons relatives, et ces comparaisons relatives permettent à l'analyste de développer une référence ou une moyenne de l'industrie.

Les mesures d'évaluation les plus couramment utilisées dans l'analyse des entreprises comparables sont [la valeur d'entreprise par rapport aux ventes](/enterprisevaluesales) (EV/S), [le rapport prix](/price-earningsratio) [/bénéfices](/price-earningsratio) (P/E), [le prix comptable](/price-to-bookratio) (P/B) et [le prix par rapport aux ventes](/price-to-salesratio) (P/S). Si le ratio de valorisation de l'entreprise est supérieur à la moyenne des pairs, l'entreprise est surévaluée. Si le ratio de valorisation est inférieur à la moyenne des pairs, l'entreprise est sous-évaluée. Utilisés ensemble, les modèles d'évaluation intrinsèque et relative fournissent une mesure approximative de l'évaluation qui peut être utilisée pour aider les analystes à évaluer la valeur réelle d'une entreprise.

## Métriques d'évaluation et de transaction utilisées dans les comps

[comps](/comps) peuvent également être basés sur des multiples de transactions. Les transactions sont des acquisitions récentes dans le même secteur. Les analystes comparent les multiples en fonction du prix d'achat de l'entreprise plutôt que de l'action. Si toutes les entreprises d'un secteur particulier vendent pour une moyenne de 1,5 fois la valeur marchande ou 10 fois les bénéfices, cela donne à l'analyste un moyen d'utiliser le même nombre pour revenir à la valeur d'une entreprise homologue sur la base de ces références.

## Points forts

- L'analyse des entreprises comparables est le processus de comparaison des entreprises sur la base de mesures similaires pour déterminer leur valeur d'entreprise.

- Le ratio de valorisation d'une entreprise détermine si elle est surévaluée ou sous-évaluée. Si le ratio est élevé, alors il est surévalué. S'il est faible, l'entreprise est sous-évaluée.

-﻿Les mesures d'évaluation les plus couramment utilisées dans l'analyse des entreprises comparables sont la valeur d'entreprise par rapport aux ventes (EV/S), le rapport prix/bénéfice (P/E), le prix comptable (P/B) et le prix par rapport aux ventes (P/S).
