# Code d'identification du marché (MIC)
## Qu'est-ce qu'un code d'identification de marché (MIC) ?

Un code d'identification de marché (MIC) est un code à quatre caractères utilisé pour identifier [les marchés boursiers et autres](/stockmarket) [bourses](/stockmarket) de négociation au sein des systèmes informatiques mondiaux de négociation et de référencement. La première lettre de tout MIC est attribuée au hasard, suivie d'un code alphanumérique à trois chiffres pour le marché sur lequel une transaction a lieu. Les déposants peuvent proposer une combinaison de quatre caractères, mais l'autorité d'enregistrement reste ultimement responsable de l'attribution. Une fois attribué, un code MIC ne peut pas être modifié .

Le code MIC est utilisé pour traiter et compenser les transactions et est poussé vers une acceptation mondiale à mesure que les industries des valeurs mobilières évoluent vers le traitement direct (STP). La [Bourse de Londres](/lse) utilise les MIC dans le cadre de ses systèmes d'identification de sécurité [SEDOL](/sedol),. qui constituent une alternative au système d'identification CUSIP basé aux États-Unis .

## Comprendre le code d'identification du marché (MIC)

[Le traitement direct](/straightthroughprocessing),. une approche conceptuelle pour améliorer la vitesse de traitement des transactions est considérée comme le Saint Graal du commerce mondial des valeurs mobilières. Pour que cela soit possible, il faudra une multitude de codes cohérents pour le marché d'origine, la devise et l'identification des titres. À l'heure actuelle, il existe plusieurs systèmes différents utilisés par différents pays et pour différents types de titres.

Une norme mondiale devra émerger au fil du temps, et tous les logiciels et systèmes de traitement devront la soutenir, en apportant les modifications techniques nécessaires en cours de route. Ce processus prendra du temps à mesure que les industries des valeurs mobilières se dirigent vers l'objectif ultime d'un marché « n'importe quel titre, n'importe où, n'importe quand ».

> L'Organisation internationale de normalisation (ISO) publie la liste MIC de tous les pays le deuxième lundi du mois .

>

## Comment les codes d'identification du marché sont utilisés

Les MIC ont été présentés par l'ISO comme une norme internationale pour identifier les marchés, les plateformes de négociation et les bourses afin de permettre le traitement automatisé des transactions. Les MIC étant utilisés comme norme d'identification universelle, cela pourrait constituer une étape importante pour rapprocher le traitement direct de sa réalisation au-delà de la phase de conception.

Il existe différents types de MIC. Un segment de marché MIC identifie une section de l'une des entités couvertes par le code qui est spécialisée dans un ou plusieurs instruments spécifiques ou qui est réglementée différemment. Selon l'ISO, des MIC de segment de marché ont été institués pour plus de précision .

Pour chaque segment de marché MIC, il existe un MIC parent, également appelé MIC opérationnel .

Un MIC opérationnel, à son tour, identifie l'entité qui exploite une bourse, une plate-forme de négociation, un marché réglementé ou non réglementé ou une installation de déclaration des transactions dans un pays spécifique .

Les organisations de marché peuvent demander un MIC en adressant une demande à l'autorité d'enregistrement. En règle générale, un MIC de segment de marché est généré lorsque la demande justifie la nécessité d'identifier le segment. Selon les circonstances, il peut arriver que seuls des segments spécifiques d'un marché soient identifiés par un MIC de segment de marché .

## Points forts

- Un code d'identification de marché (MIC) est utilisé pour spécifier les marchés boursiers et autres bourses dans lesquelles les titres sont négociés.

- Les MIC sont utilisés comme norme internationale et universelle d'identification, avec des codes définis par l'Organisation internationale de normalisation (ISO).

- Le code est unique, comprend quatre caractères, commençant par un caractère alphanumérique attribué au hasard, suivi d'un code à trois chiffres pour signifier le marché, tel que XNAS pour le marché Nasdaq.
