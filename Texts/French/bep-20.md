# BEP-20
BEP-20 est une norme de [jeton](/token) sur Binance Smart Chain qui étend ERC-20, la norme de jeton Ethereum la plus courante. Vous pouvez le considérer comme un modèle pour les jetons qui définit comment ils peuvent être dépensés, qui peut les dépenser et d'autres règles pour leur utilisation. En raison de sa similitude avec le [BEP-2](/bep-2) de Binance Chain et l'ERC-20 d'Ethereum, il est compatible avec les deux.

BEP-20 a été conçu comme une spécification technique pour Binance Smart Chain, dans le but de fournir un format flexible aux développeurs pour lancer une gamme de jetons différents. Ceux-ci peuvent représenter n'importe quoi, des actions d'une entreprise aux dollars stockés dans un coffre-fort bancaire (c'est-à-dire un stablecoin).

Bien sûr, on pourrait également créer un actif natif en tant que token BEP-20, ou même ancrer des tokens d'autres blockchains pour les rendre utilisables sur Binance Smart Chain. C'est ce qui se passe avec les pièces "Peggy", qui sont essentiellement des versions BEP-20 d'autres actifs cryptographiques (tels que LINK ou XRP).

Comme les jetons BEP-2 sur Binance Chain, les transferts de jetons BEP-20 sont alimentés en BNB. Cela incite les validateurs à inclure les transactions dans la blockchain, car ils percevront le BNB en guise de frais pour leurs problèmes.

Vous savez peut-être que Binance Smart Chain a été envisagé comme une sorte d'extension de Binance Chain. Avec une architecture à double chaîne, les deux chaînes sont complémentaires - Binance Smart Chain s'adresse aux [applications décentralisées](/decentralized-applications-dapps) sans encombrer la chaîne d'origine, qui est optimisée pour le trading ultra-rapide.

En raison de cette architecture, l'accent a été mis sur la compatibilité inter-chaînes. Pour cette raison, les jetons BEP-2 peuvent être échangés contre leur équivalent BEP-20. Le moyen le plus simple de le faire est peut-être via l'extension Binance Chain Wallet, bien que d'autres méthodes émergeront sans aucun doute au fil du temps.

Pour un aperçu plus détaillé de la norme BEP-20, assurez-vous de consulter le projet de proposition sur GitHub.
