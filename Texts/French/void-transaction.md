# Annuler la transaction
## Qu'est-ce qu'une transaction annulée ?

Une transaction annulée est une transaction qui est annulée par un commerçant ou un [fournisseur](/vendor) [avant d'être réglée via le](/vendor) [compte de carte de débit ou de crédit](/creditcard) d'un consommateur .

Bien qu'une [transaction](/transaction) puisse être annulée, elle n'apparaît pas sur le relevé de compte du client. Il peut apparaître comme une transaction en attente lorsque le client vérifie son compte en ligne.

## Comprendre les transactions annulées

Lorsqu'une transaction a lieu, le commerçant glisse la carte de débit ou de crédit du client. S'il y a suffisamment de fonds dans le compte du client, le terminal autorise la transaction. Mais la transaction n'est pas entièrement réglée, car le paiement doit être libéré du compte du client vers le commerçant.

S'il y a un problème avec la transaction, elle peut être annulée, même si elle n'est pas réglée. Étant donné que la transaction est en attente et n'a pas été effacée du compte du client, cela signifie que la vente peut être empêchée de se concrétiser.

Afin d'annuler la transaction, le client doit contacter le commerçant et demander que la transaction soit annulée à moins que le commerçant ne remarque une erreur ou un problème et annule de manière proactive une transaction au point de vente. Une fois annulée, la transaction apparaîtra sur le compte du client en tant que transaction en attente, qui disparaîtra après un certain temps.

La retenue peut durer de 24 heures à plusieurs jours, ce qui gêne le client car il ne pourra pas accéder à l'argent pendant cette période.

> Une transaction annulée a généralement lieu le même jour que la transaction originale.

>

## Considérations particulières

### Annulation d'achats

Les erreurs peuvent être facilement corrigées en annulant les transactions si elles sont reconnues immédiatement. Par exemple, un acheteur peut constater qu'il a été facturé de manière incorrecte. Un client qui vient de payer ses articles dans une épicerie prend ses sacs et se rend compte que le caissier a accidentellement inclus certains des articles du client suivant dans son achat. Le caissier peut annuler la transaction, scanner à nouveau les bons articles et facturer le montant correct au client.

Certains commerçants peuvent accorder un certain délai pour annuler un achat. Cela se produit souvent avec les marchands [de commerce électronique](/ecommerce). Un acheteur peut souvent avoir la possibilité d'annuler un achat effectué en ligne dans les 24 heures. Si l'achat est annulé, le vendeur annule alors la transaction et l'acheteur n'est pas facturé pour l'achat.

### Annulation des transactions frauduleuses

Les frais frauduleux peuvent également être annulés. Les sociétés émettrices de cartes disposent de services de détection des fraudes pour signaler les transactions frauduleuses.

La plupart des entreprises mettent ces transactions en attente. Le client peut vérifier si une transaction est frauduleuse auprès de l'entreprise, qui annulera immédiatement la transaction. Si un [client](/customer) ne peut pas être joint pour vérification, de nombreuses sociétés émettrices de cartes annuleront automatiquement une transaction suspecte avant qu'elle ne soit réglée pour assurer la sécurité du client.

> Étant donné que les remboursements sont émis après que l'argent a déjà transité par le compte d'un client jusqu'au commerçant, le processus peut prendre plus de temps qu'une transaction annulée.

>

## Annuler les transactions par rapport aux remboursements

Les transactions annulées sont différentes des [remboursements](/refund). Avec les transactions annulées, aucun argent n'est réellement transféré de la société de carte de débit ou de crédit du client au commerçant. Mais les remboursements sont émis une fois qu'une transaction a été réglée et que le client a payé le bien ou le service.

Certains commerçants et systèmes de traitement des cartes de crédit peuvent en fait régler les transactions immédiatement. Lorsqu'une transaction est réglée immédiatement, le vendeur doit émettre un remboursement plutôt que d'annuler la transaction.

Contrairement aux transactions annulées, les remboursements peuvent prendre beaucoup plus de temps sur le compte d'un client. Certains remboursements prennent aussi peu que 48 heures pour être reflétés sur le compte d'un client, tandis que d'autres peuvent prendre jusqu'à 30 jours.

## Points forts

- Une transaction annulée est une transaction qui est annulée avant d'être réglée via le compte de carte de débit ou de crédit d'un consommateur.

- Lorsqu'une transaction est annulée, elle apparaît comme une transaction en attente sur le compte du client pendant une courte période, le temps que le processus soit terminé.

- Les frais erronés, les marchandises mal facturées et les achats frauduleux sont les plus susceptibles d'être annulés.

- Les transactions annulées sont différentes des remboursements, qui sont émis après que la transaction a été compensée par le compte du client.
