# Base de report
## Qu'est-ce qu'une base de report ?

Une base de report est une méthode pour déterminer la [base fiscale](/costbasis) d'un actif lorsqu'il est transféré d'un individu à un autre. Une base de report est souvent utilisée lorsqu'une partie laisse des actifs ou des biens à une autre personne en cadeau. Dans cette situation, la base reste souvent la même que lorsque le donateur détenait l'actif, mais la base peut être ajustée pour tenir compte de tout impôt sur les donations qui a été payé.

## Comprendre la base de report

La base de report diffère d'une [base progressive](/stepupinbasis). Une base de report est utilisée pendant la durée de vie du donateur, tandis qu'une base progressive est utilisée lorsqu'un actif est hérité après le décès du donateur. Dans un scénario progressif, la valeur des actifs transférés est ajustée à sa valeur marchande actuelle.

Le coût de base d'un investissement est le montant total initialement investi, plus les [commissions](/commission) ou frais liés à l'achat. Cela peut être décrit soit en termes de montant en dollars de l'investissement, soit en termes de prix effectif par action de l'investissement.

Il est important de déterminer la base de coût correcte d'un investissement, également appelée assiette fiscale, surtout si vous avez réinvesti les dividendes et [les distributions de gains en capital](/capitalgainsdistribution) au lieu de retirer les revenus de l'investissement en espèces. Lorsque vous réinvestissez des dividendes ou des distributions, l'assiette fiscale de votre investissement augmente. Cette augmentation doit être prise en compte afin que vous puissiez déclarer des gains en capital moins élevés et, par conséquent, payer moins d'impôts. Si vous ne déclarez pas la base d'imposition plus élevée, vous pourriez finir par payer des impôts plus élevés.

Lorsque des actions vous sont données en cadeau, votre base de coût est la base de coût du détenteur initial de l'actif qui vous a fait le cadeau. Si les actions se négocient à un prix inférieur à celui au moment où les actions ont été données, le taux le plus bas est la base du coût. Si les actions vous ont été données dans le cadre d'un héritage, le prix de base des actions pour l'héritier est le [prix](/market-price) du marché des actions à la date du décès du propriétaire initial.

Dans le processus de planification successorale, la base de report aide à déterminer la valeur initiale de sa succession, de sorte que la base de report aide à déterminer le [taux d'imposition](/taxrate) que les héritiers d'une succession doivent payer sur [les gains en capital](/capitalgain) lorsqu'ils vendent des actifs associés à cette succession.

## Base de report et taxes sur les donations

[donation](/gifttax) sont un élément essentiel pour déterminer la base de report d'un actif. Il s'agit d'une taxe fédérale qui s'applique aux scénarios dans lesquels la partie bénéficiaire ne paie pas la pleine valeur du don au donateur (bien qu'elle puisse payer un montant moindre).

Le donateur est celui qui supporte le poids de l'impôt sur les donations. En général, les cadeaux à son conjoint ou à une organisation politique, ou les cadeaux d'une valeur inférieure à l'exclusion annuelle de l'impôt sur les donations, ainsi que les dépenses médicales et éducatives, sont exclus de l'impôt sur les donations.

Pour 2021, le maximum de l'impôt sur les donations est de 15 000 $ par personne par année (16 000 $ pour 2022). Cela signifie qu'un individu peut offrir à un autre individu 15 000 $ ou moins par an sans encourir d'impôt sur les donations.

L'impôt sur les donations diffère de l' [impôt sur les successions](/estatetax),. qui est prélevé sur la part héritée d'un héritier d'une [succession](/estate). En 2021, la limite d'exclusion pour la valeur de la succession correspondait à l'actif brut combiné et aux dons imposables antérieurs supérieurs à 11,7 millions de dollars (12,06 millions de dollars en 2022). Cela signifie effectivement qu'une succession de 11,7 millions de dollars ne serait pas tenue de produire une [déclaration de revenus](/taxreturn) et serait exonérée du paiement de l'impôt sur les successions.

Alors que l'impôt sur les successions est principalement imposé sur les biens laissés aux héritiers, il ne s'applique pas au transfert d'actifs à un conjoint survivant. Le droit des époux de se léguer tout montant est connu sous le nom de [déduction conjugale illimitée](/unlimited-marital-deduction).

## Points forts

- En général, la base de report est la même que la base de coût d'origine.

- Une base de report fait référence à la base du coût d'un actif reçu d'un autre individu.

- Le fait que l'actif ait été transféré par donation ou par héritage affectera son statut fiscal et son calcul de base.
