# Actions institutionnelles
## Que sont les actions institutionnelles ?

[actions](/shares) de fonds communs de placement disponibles pour [les investisseurs institutionnels](/institutionalinvestor). Les catégories d'actions de fonds communs de placement institutionnels ont généralement les ratios de frais les plus bas parmi toutes les catégories d'actions d'un fonds commun de placement. Ils nécessitent généralement un investissement minimum d'environ 200 000 $ et peuvent nécessiter d'autres spécifications d'investissement.

## Comment fonctionnent les partages institutionnels

Les classes d'actions institutionnelles sont l'une des nombreuses classes d'actions proposées dans une offre de fonds communs de placement gérés. Les sociétés d'investissement offrent plusieurs catégories d'actions d'un fonds, donnant aux investisseurs la possibilité de choisir la catégorie d'actions qui correspond le mieux à leurs intérêts d'investissement.

Les classes d'actions de détail peuvent inclure les [classes A](/classashares),. B, C et R. Chaque classe d'actions de détail est associée à des frais différents, mais toutes les classes d'actions achetées sont regroupées dans un seul fonds géré par le [gestionnaire de portefeuille désigné](/portfoliomanager). Les sociétés d'investissement utilisent des structures de fonds mixtes pour gérer leurs offres de fonds afin de faciliter l'investissement dans des fonds communs de toutes les catégories d'actions.

Outre les offres d'actions de détail, les sociétés d'investissement proposent également des actions institutionnelles d'un fonds commun de placement. Les actions institutionnelles ont leur propre structure de frais et leurs propres spécifications d'investissement.

> Les actions institutionnelles ont généralement le ratio de dépenses le plus bas de toutes les classes d'actions.

>

## Considérations particulières

Les fonds communs de placement sont connus pour offrir plusieurs catégories d'actions avec plusieurs structures de frais. Les classes d'actions institutionnelles ont généralement le [ratio](/expenseratio) de dépenses le plus bas de toutes les classes d'actions proposées par un fonds commun de placement. Ils n'exigent généralement pas non plus de frais de vente.

Les faibles frais font des classes d'actions institutionnelles la classe de frais la plus attractive pour les investisseurs en fonds communs de placement. Pour cette raison, de nombreuses sociétés de fonds peuvent proposer différents types de classes d'actions institutionnelles, dont certaines sont adaptées aux investisseurs fortunés.

## Exigences pour les actions institutionnelles

Il existe un large éventail d'investisseurs institutionnels éligibles pour acheter des actions institutionnelles. Ces investisseurs maintiennent généralement d'importantes positions d'investissement de plus de 250 000 $. Dans la plupart des cas, un investisseur institutionnel sera un gestionnaire de fonds responsable des décisions d'investissement de grands programmes d'investissement.

Les investisseurs institutionnels comprennent les gestionnaires de placements de régimes de retraite, de fonds de dotation, de fondations, d'unités gouvernementales, de comptes d'assurance institutionnels, de comptes institutionnels d'entreprise et de fonds d'investissement institutionnels. Les investisseurs institutionnels peuvent également inclure des intermédiaires financiers cherchant à investir pour [des](/hnwi) clients fortunés.

## Exemple d'actions institutionnelles

Les actions Admiral de Vanguard en sont un exemple. Les ratios de dépenses d'Admiral Shares sont en moyenne de 0,14 %. Pour être admissible aux actions Admiral, l'investissement minimum requis pour les fonds indiciels est de 3 000 $. Les fonds gérés activement ont un investissement minimum de 50 000 $. Certains fonds indiciels sectoriels peuvent nécessiter un investissement minimum de 100 000 $.

## Points forts

- Les investisseurs institutionnels comprennent les gestionnaires de placements de régimes de retraite, de fonds de dotation, de fondations et de fonds d'investissement institutionnels, entre autres.

- Les fonds communs de placement offrent diverses catégories d'actions, les actions institutionnelles étant réservées aux investisseurs institutionnels et ayant généralement les ratios de frais les plus bas.

- Les actions institutionnelles ont leur propre structure de frais et leurs propres spécifications d'investissement, par rapport aux actions de détail.

- L'investissement minimum pour les actions institutionnelles est généralement de 200 000 $.

- Les actions Admiral de Vanguard, qui ont des exigences d'investissement minimales pour certains fonds, sont un exemple d'actions institutionnelles, ayant des ratios de frais moyens de 0,10 %.
