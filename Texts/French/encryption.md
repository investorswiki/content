# Chiffrement
## Qu'est-ce que le cryptage ?

Le cryptage est une méthode de sécurité des données qui code les informations stockées ou transmises d'un ordinateur ou d'un appareil à un autre d'une manière qui ne peut être codée que par la personne ou l'ordinateur disposant de la clé de cryptage. Les techniques les plus populaires utilisées par les principales institutions financières pour sécuriser les informations personnelles sensibles reposent sur la technologie de cryptage des données.

En raison du volume d'informations sensibles échangées quotidiennement sur Internet, de plus en plus de sites Web et de services de commerce en ligne adoptent également des systèmes de cryptage sophistiqués.

## Définition plus approfondie

Il existe plusieurs façons de chiffrer les données.

- Le chiffrement à clé privée signifie que chaque ordinateur d'un réseau possède un code secret ou une clé qui est utilisé pour chiffrer les informations avant de les envoyer à un autre ordinateur. Pour que le cryptage par clé privée fonctionne, vous devez connaître les ordinateurs qui « parleront » entre eux et sur lesquels la clé est installée. Avoir une clé privée s'apparente à un code secret partagé entre deux ordinateurs qui n'est connu que d'eux afin de décoder les messages échangés entre eux. Si le message est vu par quelqu'un d'autre, il ressemble à une information dénuée de sens.

- Le chiffrement à clé publique utilise à la fois une clé publique et une clé privée. Seul votre ordinateur connaît sa clé privée, mais il donne sa clé publique à tout ordinateur nécessitant une communication sécurisée avec lui. L'ordinateur qui reçoit le message utilise sa propre clé privée et la clé publique reçue de l'ordinateur d'origine pour décoder les messages chiffrés.

Pour les entreprises qui dépendent de la transmission de données sensibles, en particulier celles du secteur financier, subir une violation de données en raison d'un manque de cryptage est synonyme de perte. En plus d'avoir à faire face à une réputation endommagée, il peut y avoir des coûts financiers et de main-d'œuvre importants nécessaires pour réparer tout dommage qui a été causé.

Lorsque des informations financières sensibles sont stockées sur un appareil comme un ordinateur portable ou un smartphone, elles sont souvent ignorées par les utilisateurs qui supposent que les données sont en sécurité. Cependant, ces systèmes peuvent toujours être piratés et des informations confidentielles peuvent être consultées. Le cryptage des données stockées sur les appareils est nécessaire pour les tenir à l'écart des pirates ou des voleurs.

## Exemple de chiffrement

Chaque fois que vous vous connectez à un service de messagerie Web ou à un autre site Web basé sur des autorisations, vous entrez un nom d'utilisateur et un mot de passe qui sont ensuite cryptés par le navigateur. Le serveur hébergeant le site Web et votre ordinateur ou appareil mobile s'entendent ensuite sur le protocole qu'ils utiliseront pour transférer les informations cryptées dans les deux sens en ligne.

L'utilisation de communications en ligne cryptées vous offre plusieurs avantages, notamment :

- Réduire le nombre de personnes qui savent ce que vous faites.

- Rendre plus difficile pour les pirates d'accéder à vos informations de connexion.

- Masquage de vos données de navigation.

- Réduire le risque d'être trompé en visitant un site Web malveillant.

Une grande partie du temps que les gens passent sur Internet consiste à envoyer leurs informations à d'autres. Lorsque vous achetez quelque chose en ligne ou remplissez un formulaire d'inscription pour ouvrir un compte en ligne, vous saisissez de nombreuses informations personnelles. Ainsi, le cryptage est essentiel. Vous pouvez savoir si vos communications en ligne sont cryptées de manière sécurisée dans votre navigateur si une icône de cadenas s'affiche. dans la barre d'URL et l'adresse du site Web contient "https://".

## Points forts

- Le chiffrement est un moyen de sécuriser les données en les codant mathématiquement de sorte qu'elles ne puissent être lues ou déchiffrées que par ceux qui possèdent la clé ou le chiffrement correct.

- Les processus de cryptage numérique traduisent les données à l'aide d'un algorithme qui rend les informations d'origine illisibles, sauf pour les utilisateurs autorisés.

- Le cryptage est crucial dans un monde numériquement connecté pour garder les informations privées, les messages et les transactions financières privées et sécurisées.
