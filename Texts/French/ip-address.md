# Adresse IP
## Qu'est-ce qu'une adresse IP ?

L'adresse IP correspond à l'adresse de protocole Internet ; il s'agit d'un numéro d'identification associé à un ordinateur ou à un réseau informatique spécifique. Lorsqu'il est connecté à Internet, l'adresse IP permet aux ordinateurs d'envoyer et de recevoir des informations.

## Comment fonctionne une adresse IP

Une adresse IP permet aux ordinateurs d'envoyer et de recevoir des données sur Internet. La plupart des adresses IP sont purement numériques, mais à mesure que l'utilisation d'Internet se développe, des lettres ont été ajoutées à certaines adresses.

Il existe quatre types d'adresses IP différentes : publique, privée, statique et dynamique. Alors que le public et le privé indiquent l'emplacement du réseau - le privé étant utilisé à l'intérieur d'un réseau tandis que le public est utilisé à l'extérieur d'un réseau - statique et dynamique indiquent la permanence.

Une adresse IP statique est une adresse qui a été créée manuellement, au lieu d'avoir été attribuée. Une adresse statique ne change pas non plus, alors qu'une adresse IP dynamique a été attribuée par un serveur DHCP (Dynamic Host Configuration Protocol) et est susceptible de changer. Les adresses IP dynamiques sont le type le plus courant d'adresses de protocole Internet. Les adresses IP dynamiques ne sont actives que pendant un certain temps, après quoi elles expirent. Soit l'ordinateur demandera automatiquement un nouveau bail, soit l'ordinateur recevra une nouvelle adresse IP.

Une adresse IP peut être comparée à un [numéro de sécurité sociale](/ssn) (SSN) puisque chacun est complètement unique pour l'ordinateur ou l'utilisateur auquel il est attribué. La création de ces numéros permet aux routeurs d'identifier où ils envoient des informations sur Internet. Ils s'assurent également que les bons appareils reçoivent ce qui est envoyé. Tout comme le bureau de poste a besoin d'une adresse postale pour livrer un colis, un routeur a besoin d'une adresse IP pour livrer à l'adresse Web demandée.

## Exemple d'adresse IP

Le dark web fait référence au contenu en ligne crypté qui n'est pas indexé par les moteurs de recherche conventionnels. Le dark web a un marché en ligne illégal où les criminels peuvent vendre des biens illégaux et illicites. Bon nombre de ces échanges ont lieu à l'aide de la crypto-monnaie en ligne bitcoin, ce qui rend plus difficile pour les autorités de suivre et de capturer les personnes participant à ces transactions.

En 2018, après une opération gouvernementale d'un an, des agents travaillant avec le Department of Homeland Security se sont fait passer pour des trafiquants d'armes afin d'accéder aux ordinateurs de suspects qui tentaient d'acheter des armes illégalement. Cela leur a permis d'accéder aux adresses IP, qu'ils ont utilisées pour localiser les emplacements géographiques d'autres suspects qui utilisaient le dark web.

Ce n'est pas la première fois que les adresses IP entraînent des arrestations. En 2012, la police a utilisé des adresses IP pour retrouver et arrêter les membres du groupe de piratage Lulzsec. En utilisant des mandats pour obtenir des informations du [fournisseur de services Internet (FAI)](/isp),. les forces de l'ordre ont pu retrouver les adresses physiques des pirates et les arrêter pour leur activité illégale sur Internet.

## Points forts

- Une adresse de protocole Internet (IP) permet aux ordinateurs d'envoyer et de recevoir des informations.

- Une adresse IP permet aux informations d'être envoyées et reçues par les bonnes parties, ce qui signifie qu'elle peut également être utilisée pour localiser l'emplacement physique d'un utilisateur dans certains cas.

- Il existe quatre types d'adresses IP : publique, privée, statique et dynamique.

## FAQ

### Qu'est-ce qu'une adresse IPv6 ?

Les adresses IPv4 se composent de quatre chiffres de un à trois chiffres séparés par un point. IPv6 est la nouvelle version de l'adressage Internet qui utilise une valeur alphanumérique de 128 bits pour identifier un périphérique de point de terminaison en ligne. Une adresse Ipv4 peut ressembler à 208.80. 154.224, et une adresse IPv6 peut ressembler à : 2001:0db8:85a3:0000:0000:8a2e:0370:7334.IPv6 permet à beaucoup plus d'appareils connectés d'exister sur Internet sans dupliquer les adresses, ce qui peut causer des problèmes.

### Comment quelqu'un peut-il "usurper" son adresse IP ?

Vous pouvez donner l'impression que votre adresse IP provient d'ailleurs via un réseau privé virtuel (VPN). Ces services masquent essentiellement l'adresse Internet de votre ordinateur en rejoignant virtuellement le réseau qu'ils établissent. Par conséquent, votre adresse IP apparaîtra comme la leur. Les pirates peuvent également usurper des adresses IP en utilisant des techniques plus sophistiquées qui modifient l'adresse source dans un en-tête de paquet de données malveillant pour imiter une source fiable et l'accepter.

### Comment puis-je trouver mon adresse IP ?

Il existe plusieurs façons de déterminer votre adresse IP. Le plus simple est peut-être de taper "quelle est mon adresse IP" dans un moteur de recherche Internet comme Google. Pour les utilisateurs de Windows, vous pouvez trouver votre adresse IP en ouvrant [Démarrer > Paramètres > Réseau et Internet], puis sélectionnez le réseau Wi-Fi auquel vous êtes connecté. Sous Propriétés, recherchez votre adresse IP répertoriée à côté de "Adresse IPv4".
