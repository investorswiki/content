# Test
## Qu'est-ce qu'un test ?

Dans l'analyse technique et le trading, un test se produit lorsque le prix d'une action s'approche d'un niveau de support ou de résistance établi par le marché. Si le titre reste dans les niveaux de support et de résistance, le test réussit. Cependant, si le cours de l'action atteint de nouveaux creux et/ou de nouveaux sommets, le test échoue. En d'autres termes, pour l'analyse technique, les niveaux de prix sont testés pour voir si les modèles ou les signaux sont exacts.

Un test peut également faire référence à une ou plusieurs techniques [statistiques](/statistics) utilisées pour évaluer les différences ou les similitudes entre les valeurs estimées à partir de modèles ou de variables trouvées dans les données. Les exemples incluent le [test t](/t-test) et le test [z](/z-test).

## Comprendre les tests

Les indicateurs techniques populaires que les traders et les investisseurs utilisent pour tester les niveaux de support et de résistance incluent les lignes de tendance, les moyennes mobiles et les nombres ronds.

Par exemple, de nombreux investisseurs accordent une attention particulière à l'évolution des cours des principaux indices boursiers, tels que l'indice Standard & Poor's 500 (S&P 500), le Dow Jones Industrial Average (DJIA) et le Nasdaq Composite lorsqu'ils testent leur moyenne mobile sur 200 jours. ou une ligne de tendance à long terme. Des techniques plus avancées utilisées pour tester les niveaux de support et de résistance incluent l'utilisation de points de pivot, les niveaux de [retracement de Fibonacci](/fibonacciretracement) et les angles de Gann.

Les traders doivent surveiller de près le [volume](/volume) lorsque le prix d'une action s'approche des principales zones de support et de résistance. Si le volume augmente, il y a une plus grande probabilité que le prix échoue lorsqu'il teste ces niveaux en raison de l'intérêt accru pour la question. La baisse du volume, d'autre part, suggère que le test peut réussir car le stock peut ne pas avoir suffisamment de participation pour [atteindre](/breakout) un nouveau niveau.

> Une action peut tester les niveaux de support et de résistance à la fois sur un marché lié à une fourchette et sur un marché en tendance.

>

### Test de marché lié à la plage

Lorsqu'une action est [liée à une fourchette](/rangeboundtrading),. le prix teste fréquemment les limites supérieure et inférieure de la fourchette de négociation. Si les traders utilisent une stratégie qui achète du support et vend de la résistance, ils doivent attendre plusieurs tests de ces limites pour confirmer que le prix les respecte avant d'entrer dans une transaction.

Une fois en position, les traders doivent passer un ordre stop-loss au cas où le prochain test de support ou de résistance échouerait.

### Test de marché des tendances

Dans un marché à tendance haussière, la résistance précédente devient un support, tandis que dans un marché à tendance baissière, le support passé devient une résistance. Une fois que le prix atteint un nouveau sommet ou un nouveau creux, il [revient souvent](/retracement) pour tester ces niveaux avant de reprendre dans le sens de la tendance. Les traders Momentum peuvent utiliser le test d'un swing haut ou swing bas précédent pour entrer dans une position à un prix plus favorable que s'ils avaient chassé la cassure initiale.

Un [ordre stop-loss](/stop-lossorder) doit être placé directement sous la zone de test pour clôturer la transaction si la tendance s'inverse de manière inattendue.

## Tests statistiques

Les statistiques inférentielles utilisent les propriétés des données pour tester des hypothèses et tirer des conclusions. Le test d' [hypothèse](/hypothesistesting) permet de tester une idée à l'aide d'un [échantillon de données](/sample) par rapport à un paramètre de population. La méthodologie employée par l'analyste dépend de la nature des données utilisées et de la raison de l'analyse. En particulier, on cherche à rejeter l' [hypothèse nulle](/null_hypothesis),. ou l'idée qu'une ou plusieurs [variables aléatoires](/random-variable) n'ont aucun effet sur une autre. Si cela peut être rejeté, les variables sont susceptibles d'être associées les unes aux autres.

Il existe plusieurs outils utilisés pour effectuer des tests d'hypothèse, dont certains incluent :

- Un test t est un type de statistique inférentielle utilisé pour déterminer s'il existe une différence significative entre les moyennes de deux groupes, qui peuvent être liées dans certaines caractéristiques. Il est principalement utilisé lorsque les ensembles de données, comme l'ensemble de données enregistré comme le résultat de lancer une pièce 100 fois, suivraient une distribution normale et pourraient avoir des variances inconnues. Un test t est utilisé comme outil de test d'hypothèse, ce qui permet de tester une hypothèse applicable à une population. Les tests Z sont étroitement liés aux tests t, mais les tests t sont mieux réalisés lorsqu'une expérience a une taille d'échantillon plus petite.

- Le [test de Wilcoxon](/wilcoxon-test),. qui peut faire référence soit au test Rank Sum, soit à la version du test Signed Rank, est un test statistique non paramétrique qui compare deux groupes appariés.

- [Chi-carré](/chi-square-statistic) (**χ**^2^) est un test qui mesure comment un modèle se compare aux données réelles observées. Les données utilisées pour calculer une [statistique](/statistics) du chi carré doivent être aléatoires, brutes, [mutuellement exclusives](/mutuallyexclusive),. tirées de variables indépendantes et tirées d'un échantillon suffisamment grand. Par exemple, les résultats du lancer d'une pièce juste répondent à ces critères.

- Le test de [Bonferroni](/bonferroni-test) est un test statistique utilisé pour réduire l'occurrence d'un [faux positif](/type_1_error).

- Un [test de Scheffé](/scheffes-test) est une sorte de test d'analyse statistique post-hoc qui sert à faire des comparaisons non planifiées.

## Points forts

- Plusieurs tests techniques existent, dont ceux spécifiquement destinés aux marchés range-bound versus trending.

- Ces tests sont souvent utilisés pour confirmer les niveaux de résistance ou de support d'une action ou d'un autre actif.

- Les tests peuvent également faire référence à des méthodes statistiques pour évaluer des hypothèses ou des associations entre variables.

- Un test, en analyse technique, fait référence à la capacité d'un signal, d'un modèle ou d'un autre indicateur à rester ferme dans l'action ultérieure des prix.
