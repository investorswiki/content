# Facture proforma
## Qu'est-ce qu'une facture pro forma ?

Une facture pro forma est un [acte de vente préliminaire](/bill-of-sale) envoyé aux acheteurs avant l'expédition ou la livraison de marchandises. La facture décrira généralement les articles achetés et d'autres informations importantes, telles que le poids d'expédition et les frais de transport.

## Comprendre les factures pro forma

Les factures pro forma entrent souvent en jeu dans les transactions internationales, notamment à des fins douanières à l'importation. Une [facture](/invoice) pro forma est différente d'un simple devis en ce sens qu'il s'agit d'un accord contraignant, bien que les conditions de vente soient susceptibles d'être modifiées.

Les entreprises de pratiquement tous les secteurs utilisent des factures pro forma pour satisfaire leur processus d'approbation d'achat interne. Les factures pro forma rationalisent le processus de vente en éliminant les allers-retours supplémentaires après la conclusion d'une vente, car toutes les conditions ont été convenues à l'avance.

La plupart des factures pro forma fournissent à l'acheteur un prix de vente précis. Il comprend une estimation de toutes les [commissions](/commission) ou frais, tels que les taxes applicables ou les frais d'expédition. Bien que la facture pro forma puisse être sujette à modification, elle représente une estimation de bonne foi pour éviter d'exposer l'acheteur à des frais imprévus et importants une fois la transaction finale.

## Considérations particulières

Une documentation spécifique est requise avant que les marchandises puissent passer la douane aux États-Unis. Une facture commerciale traditionnelle doit énumérer des informations substantielles telles que l'acheteur et le vendeur, une description des marchandises, la quantité, la valeur de tous les articles expédiés et l'emplacement de l'achat.

Une facture pro forma ne nécessite que suffisamment d'informations pour permettre aux douanes de déterminer les [droits](/duty) nécessaires à partir d'un examen général des marchandises incluses. Si une entreprise utilise une facture pro forma pour les douanes, elle doit présenter une facture commerciale dans les 120 jours.

## Exemple de facture pro forma

Si les parties concernées ont déjà conclu une vente, une facture commerciale l'enregistre. Si la transaction n'a pas encore eu lieu, l'entreprise vendeuse peut utiliser une facture pro forma. Une entreprise peut également utiliser une facture pro forma si les termes du contrat de vente précisent que le paiement intégral n'est pas dû tant que l'acheteur n'a pas reçu certaines marchandises.

Un client de boulangerie, par exemple, peut accepter le prix d'un gâteau sur une facture pro forma, et le boulanger livrera le gâteau une fois qu'il sera prêt. Le client paiera alors à réception de la facture. Le montant final de la facture sera identique ou proche du montant de la facture pro forma.
