# Formulaire 1040-X, Déclaration de revenus des particuliers aux États-Unis modifiée
## Qu'est-ce que le formulaire 1040-X, déclaration de revenus des particuliers aux États-Unis modifiée ?

Le formulaire 1040-X, Amended US Individual US Income Tax Return est un formulaire utilisé par les contribuables qui ont besoin de corriger une erreur dans une déclaration de revenus fédérale déposée précédemment. Les erreurs courantes corrigées avec ce formulaire comprennent les erreurs dans le statut de dépôt du contribuable ou le nombre de personnes à charge ou les omissions de crédits ou de déductions. Le formulaire 1040-X ne doit pas être utilisé pour corriger de simples erreurs mathématiques dans une déclaration de revenus, car l'Internal Revenue Service (IRS) corrige régulièrement ces erreurs lors du traitement des déclarations.

## Qui peut remplir le formulaire 1040-X ?

Un formulaire 1040-X doit être déposé par toute personne qui a déjà déposé une déclaration de revenus et doit la modifier pour l'une des raisons suivantes :

- Pour corriger le formulaire [1040](/1040),. 1040-SR ou 1040-NR. La plupart des contribuables utilisent l'un de ces trois formulaires pour produire leur déclaration de revenus annuelle.

- Pour faire une élection après la date limite. Par exemple, un contribuable marié peut choisir de déposer en tant qu'individu ou conjointement avec un conjoint.

- Pour modifier un montant qui a été précédemment ajusté par l'IRS

- Pour faire une réclamation pour un report en arrière suite à une perte ou un crédit non utilisé

## Comment déposer le formulaire 1040-X

> Vous pouvez produire par voie électronique jusqu'à trois déclarations modifiées « acceptées ». Après le troisième, tous les autres seront rejetés.

>

Ce formulaire est une description détaillée, ligne par ligne, de tous les ajustements possibles, afin que le contribuable puisse clairement enregistrer le type et le montant exacts de chaque modification et entrer une brève description de ce qui est modifié et pourquoi.

Le déposant doit inclure non seulement le 1040-X, mais une version révisée de l'ensemble du 1040 ou du 1040-SR, y compris tous les formulaires et annexes joints, même s'ils n'ont pas été modifiés.

Les contribuables peuvent déposer un formulaire 1040-X par voie électronique ou l'envoyer à l'IRS par courrier. Le formulaire 1040-X ne peut être déposé qu'après le dépôt d'un rapport annuel.

Afin de recevoir un crédit, les contribuables doivent déposer un formulaire 1040-X dans les trois ans suivant le dépôt de la déclaration originale ou dans les deux ans suivant le paiement de la taxe, selon la dernière éventualité.

L'IRS fournit les conseils suivants aux contribuables concernant le formulaire 1040-X :

- Attendez le premier remboursement, si vous en deviez un. Si votre déclaration de revenus originale comprenait un remboursement, vous devez attendre de l'avoir reçu avant de remplir le formulaire 1040-X. L'IRS émettra un deuxième remboursement séparé si cela est justifié, alors n'hésitez pas à encaisser le premier chèque.

- Déposez un formulaire 1040-X dans le délai de modification de trois ans. Si le changement affecte le montant de votre remboursement, vous devez déposer le formulaire dans les trois ans à compter de la date de votre déclaration de revenus initiale ou dans les deux ans à compter de la date de paiement du montant de la taxe, selon la dernière éventualité. Toute déclaration déposée tôt sera considérée comme déposée à la date limite officielle de dépôt des impôts, qui est généralement le 15 avril de l'année civile.

## Points forts

- Pour obtenir un remboursement, le formulaire doit être déposé dans les trois ans suivant le dépôt de la déclaration initiale ou dans les deux ans suivant le paiement de la taxe, selon la dernière éventualité.

- Vous pouvez déposer le formulaire par voie électronique si vous avez déposé par voie électronique la déclaration de revenus correspondante. Sinon, envoyez-le par la poste.

- Le formulaire IRS 1040-X est déposé par les contribuables pour modifier une déclaration de revenus annuelle qui a été déposée précédemment.

- Le formulaire 1040-X doit être utilisé pour corriger les modifications importantes, mais pas pour corriger les erreurs mathématiques.

## FAQ

### Devez-vous remplir le formulaire 1040-X avec le formulaire 1040 ?

Vous devez déposer le formulaire 1040-X après avoir déposé votre 1040, 1040-SR ou 1040-NR. Vous modifiez votre déclaration de revenus précédente avec les informations correctes.

### Quand dois-je recevoir ma déclaration de revenus modifiée ?

L'IRS indique que le traitement d'une déclaration modifiée peut prendre jusqu'à 16 semaines (20 semaines à compter de 2022 en raison des retards causés par la pandémie de coronavirus) même si elle est déposée par voie électronique. Environ trois semaines après avoir déposé le 1040-X, vous pouvez commencer à vérifier l'état de votre dépôt sur le site Web de l'IRS sur son Où est ma déclaration modifiée ? page.

### Où envoyez-vous le formulaire 1040-X ?

Si vous avez déposé votre déclaration annuelle par voie électronique, vous pouvez désormais déposer par voie électronique un 1040-X à l'aide d'un logiciel de préparation des déclarations de revenus commercial. C'est le moyen le plus rapide de récupérer votre argent si vous en avez. Vous pouvez l'envoyer par la poste si vous préférez. L'IRS répertorie quatre adresses distinctes à l'usage des résidents de différentes régions et des Américains vivant à l'étranger.

### Les remboursements du formulaire 1040-X peuvent-ils être déposés directement ?

Non. Le dépôt direct n'est actuellement pas disponible pour les remboursements basés sur des déclarations modifiées. Vous recevrez un chèque par la poste de l'IRS.

### À quoi sert un formulaire 1040-X ?

Le 1040-X permet à un contribuable de modifier ou de corriger une déclaration de revenus déjà déposée. Si vous avez oublié de réclamer un crédit ou d'enregistrer une déduction, voici le formulaire à utiliser. Si vous avez fait une erreur mathématique, oubliez-la. L'IRS exécute toujours les chiffres et corrige ce genre d'erreur.
