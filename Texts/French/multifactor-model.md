# Modèle multifactoriel
## Qu'est-ce qu'un modèle multifactoriel ?

Un modèle multifactoriel est un [modèle financier](/financialmodeling) qui utilise plusieurs facteurs dans ses calculs pour expliquer les phénomènes de marché et/ou les prix d' [équilibre](/equilibrium) des actifs. Un modèle multifactoriel peut être utilisé pour expliquer soit un titre individuel, soit un [portefeuille](/portfolio) de titres. Pour ce faire, il compare deux ou plusieurs facteurs pour analyser les relations entre les variables et la performance qui en résulte.

## Comprendre un modèle multifactoriel

Les modèles multifactoriels sont utilisés pour construire des portefeuilles avec certaines caractéristiques, telles que le risque, ou pour suivre des [indices](/index). Lors de la construction d'un modèle multifactoriel, il est difficile de décider combien et quels facteurs inclure. De plus, les modèles sont jugés sur des chiffres historiques, qui pourraient ne pas prédire avec précision les valeurs futures.

Les modèles multifactoriels aident également à expliquer le poids des différents facteurs utilisés dans les modèles, indiquant quel facteur a le plus d'impact sur le prix d'un actif.

## Formule du modèle multifactoriel

Les facteurs sont comparés à l'aide de la formule suivante :

**Ri = ai + _i(m) * Rm + _i(1) * F1 + _i(2) * F2 +...+_i(N) * FN + ei**

Où:

**Ri** est le retour de la sécurité

**Rm** est le rendement du marché

**F**(1, 2, 3 ... N) est chacun des facteurs utilisés

**_** est le [bêta](/beta) par rapport à chaque facteur, y compris le marché (m)

**e** est le terme d'erreur

**a** est l'interception

## Types de modèles multifactoriels

Les modèles multifactoriels peuvent être divisés en trois catégories : les modèles macroéconomiques, les modèles fondamentaux et les modèles statistiques.

**Modèles macroéconomiques :** Les modèles macroéconomiques comparent le rendement d'un titre à des facteurs tels que l'emploi, [l'inflation](/inflation) et les taux d'intérêt.

**Modèles fondamentaux :** Les modèles fondamentaux analysent la relation entre le rendement d'un titre et ses données financières sous-jacentes, telles que [les bénéfices](/earnings),. [la capitalisation boursière](/marketcapitalization) et les niveaux d'endettement.

**Modèles statistiques :** Les modèles statistiques sont utilisés pour comparer les rendements de différents titres en fonction de la performance statistique de chaque titre en soi. Souvent, les données historiques sont utilisées dans ce type de modélisation.

## Construction de modèles multifactoriels

Les trois modèles les plus couramment utilisés pour construire un modèle multifactoriel sont un modèle combiné, un modèle séquentiel et un modèle intersectionnel.

**Modèle de combinaison :** Dans un modèle de combinaison, plusieurs modèles à facteur unique, qui utilisent un facteur unique pour distinguer les actions, sont combinés pour créer un modèle à facteurs multiples. Par exemple, les actions peuvent être triées en fonction du momentum uniquement lors du premier passage. Les passes suivantes utiliseront d'autres facteurs, tels que la [volatilité](/volatility),. pour les classer.

**Modèle séquentiel :** Un modèle séquentiel trie les actions en fonction d'un seul facteur de manière séquentielle pour créer un modèle multifactoriel. Par exemple, les actions d'une capitalisation boursière spécifique peuvent être analysées séquentiellement pour divers facteurs, tels que la valeur et le momentum, séquentiellement.

**Modèle intersectionnel :** Dans le modèle intersectionnel, les actions sont triées en fonction de leurs intersections pour les facteurs. Par exemple, les actions peuvent être triées et classées en fonction des intersections de la valeur et du [momentum](/momentum).

## Mesure du bêta

Le bêta d'un titre mesure le [risque systématique](/systemic-risk) d'un titre par rapport à l'ensemble du marché. Un bêta de 1 indique que le titre connaît théoriquement le même degré de volatilité que le marché et évolue en tandem avec le marché.

Un bêta supérieur à 1 indique que le titre est théoriquement plus volatil que le marché. A l'inverse, un bêta inférieur à 1 indique que le titre est théoriquement moins volatil que le marché.

Lorsque des modèles multifactoriels sont utilisés par les gestionnaires de placements pour évaluer le risque des investissements, le bêta est un facteur important qu'ils peuvent utiliser.

## Modèle Fama-français à trois facteurs

Un modèle multifactoriel largement utilisé est le modèle à trois facteurs de Fama-French. Le modèle Fama-French comporte trois facteurs : la taille des entreprises, les valeurs [book-to-market](/booktomarketratio) et les rendements excédentaires sur le marché. En d'autres termes, les trois facteurs utilisés sont SMB (petit moins grand), HML (élevé moins faible) et le rendement du portefeuille moins le [taux de rendement sans risque](/risk-freerate).

SMB représente les sociétés cotées en bourse avec de petites capitalisations boursières qui génèrent des rendements plus élevés, tandis que HML représente les [actions de valeur](/valuestock) avec des ratios livre-marché élevés qui génèrent des rendements plus élevés par rapport au marché.

## Points forts

- Le bêta d'un titre mesure le risque systématique d'un titre par rapport à l'ensemble du marché.

- Le modèle à trois facteurs Fama-French est un outil bien connu qui s'appuie sur le modèle d'évaluation des actifs financiers, qui se concentre uniquement sur le facteur de risque de marché, en incorporant des facteurs de taille et de valeur.

- Les portefeuilles multifactoriels peuvent être construits selon différentes méthodes : modélisation intersectionnelle, combinatoire et séquentielle.

- Les modèles multifactoriels révèlent les facteurs qui ont le plus d'impact sur le prix d'un actif.

- Un modèle multifactoriel est une stratégie de modélisation financière dans laquelle plusieurs facteurs sont utilisés pour analyser et expliquer les prix des actifs.
