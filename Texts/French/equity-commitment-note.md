# Note d'engagement sur actions (ECN)
## Qu'est-ce qu'un Equity Commitment Note (ECN) ?

Une note d'engagement sur actions est un type de dette [obligatoirement convertible](/mandatoryconvertible) émise par une banque qui peut être échangée contre des actions à l'échéance. Le billet est remboursable par la vente d'actions par son émetteur.

Il s'agit essentiellement d'un type de dette d'entreprise qui s'accompagne de restrictions sur l'emprunteur qui dictent comment l'argent pour rembourser la dette sera levé.

## Comprendre l'Equity Commitment Note (ECN)

La note d'engagement sur actions est une forme de dette d'entreprise. Il est remboursé par la vente ou l'émission de titres à une date future par la banque ou l'établissement de crédit qui l'a émis.

Un billet d'engagement sur actions diffère d'un contrat sur actions en ce sens que l'investisseur n'est pas tenu d'acheter les titres pour rembourser le billet. La note d'engagement sur actions peut être remboursée ultérieurement par la vente d'actions ordinaires ou privilégiées.

Aux États-Unis, la Federal Reserve Bank fixe une maturité maximale de 12 ans et exige que la société émettrice finance un tiers des fonds propres tous les quatre ans .

## Points forts

- Le billet peut être échangé contre des actions à l'échéance.

- Une note d'engagement sur actions est un type d'obligation convertible obligatoire émise par une banque ou un autre établissement de crédit.

- Essentiellement, une note d'engagement sur actions est une obligation d'entreprise.
