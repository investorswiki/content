# Ligne de base
## Qu'est-ce qu'une ligne de base ?

Une ligne de base est un point de référence fixe utilisé à des fins de comparaison. En affaires, le succès d'un projet ou d'un produit est souvent mesuré par rapport à un nombre de base pour les coûts, les ventes ou un certain nombre d'autres variables. Un projet peut dépasser un nombre de référence ou ne pas l'atteindre.

Par exemple, une entreprise qui souhaite mesurer le succès d'une gamme de produits peut utiliser le nombre d'unités vendues au cours de la première année comme référence par rapport à laquelle les ventes annuelles ultérieures sont mesurées. La ligne de base sert de point de départ par rapport auquel toutes les ventes futures sont mesurées.

## Comprendre une ligne de base

Une ligne de base peut être n'importe quel nombre qui sert de point de départ raisonnable et défini à des fins de comparaison. Il peut être utilisé pour évaluer les effets d'un changement, suivre l'avancement d'un projet d'amélioration ou mesurer la différence entre deux périodes de temps.

Par exemple, une [entreprise publique](/publiccompany) suivra la performance de chaque gamme de produits en choisissant une année comme référence et en mesurant toutes les années suivantes par rapport à celle-ci.

Une situation de référence est généralement utilisée lors de la préparation d'un état financier ou d'une analyse budgétaire. L'état ou l'analyse utilise les revenus et les dépenses existants comme référence pour évaluer si un nouveau projet est mis en œuvre avec succès.

### La ligne de base dans l'analyse des états financiers

Une analyse des états financiers qui utilise une ligne de base est appelée [analyse](/horizontalanalysis) [horizontale](/horizontalanalysis). Il compare les informations financières historiques d'une entreprise sur un certain nombre de périodes de reporting qui peuvent être mensuelles, trimestrielles ou annuelles.

La première période d'une analyse horizontale est appelée période de référence. Toutes les périodes suivantes sont ensuite mesurées en pourcentage de la ligne de base. Ainsi, une période qui a le même revenu que la ligne de base aurait un revenu de 100 %.

> En technologie de l'information, il existe trois points de base couramment utilisés : le coût, la portée et le calendrier.

>

Cet exercice est utile pour repérer les tendances, examiner les domaines de croissance ou de déclin et évaluer la performance financière globale. Des ratios tels que la marge bénéficiaire sont également comparés horizontalement à l'année de référence pour tirer des conclusions sur la performance continue d'une entreprise.

### La ligne de base dans la budgétisation

La budgétisation du projet fonctionne à partir de ce que l'on appelle une ligne de base des coûts. Le coût de référence est le budget approuvé pour le projet, généralement ventilé de manière assez détaillée par catégorie de coût et période de coût.

Si une entreprise ouvre un nouvel entrepôt, par exemple, et que le coût de base a été fixé à 100 000 $ par mois tous les mois pendant 10 mois, tout coût mensuel supérieur à 100 000 $ est un [signal d'alarme](/redflag) pour l'analyste budgétaire.

Cependant, les coûts du projet fluctuent inévitablement par rapport aux chiffres de référence, car des dépenses inconnues et imprévues ou même, dans certains cas, des économies sont réalisées. La ligne de base des coûts peut être mise à jour pour refléter les coûts réels du projet.

### La base en technologie de l'information

Dans la gestion des technologies de l'information, une ligne de base peut être établie pour les niveaux de performance anticipés ou maximaux. Il existe trois points de référence couramment utilisés : le coût, la [portée](/scope) et le calendrier.

Les applications logicielles utilisées par les professionnels de la gestion de projet sont généralement conçues pour maintenir et suivre ces trois mesures de base critiques.

## Points forts

- Dans l'analyse financière horizontale, les chiffres de la première période de déclaration servent de référence pour la comparaison des périodes suivantes.

- Dans la budgétisation du projet, les chiffres du budget approuvé sont les références pour la comparaison des dépenses réelles.

- En gestion des technologies de l'information, la ligne de base est le niveau de performance anticipé ou maximal.
