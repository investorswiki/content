# Stratégie mobile d'abord
## Qu'est-ce qu'une stratégie Mobile First ?

L'expression « stratégie mobile d'abord » fait référence à une approche de développement de sites Web dans laquelle la version mobile du site Web est prioritaire sur sa version de bureau.

Les entreprises accordent de plus en plus la priorité aux versions mobiles de leurs pages Web, en raison de la popularité croissante de la navigation sur Internet par téléphone et des achats en ligne parmi les consommateurs.

## Comment fonctionnent les stratégies Mobile-First

Lorsqu'elle poursuit une stratégie axée sur le mobile, une entreprise publiera d'abord une version mobile de son site Web, avant d'investir beaucoup de temps ou de ressources dans le développement d'une version de bureau. Souvent, un site Web de bureau rudimentaire sera publié parallèlement à la version mobile, mais ce site Web de bureau peut consister en un peu plus qu'une page de destination qui renvoie à une application mobile ou à des informations de base telles qu'une page de questions fréquemment posées (FAQ).

Plusieurs facteurs sous-jacents expliquent la popularité des stratégies axées sur le mobile. Pour commencer, le pourcentage du trafic Web attribuable aux [smartphones](/smartphone) par rapport aux ordinateurs de bureau a augmenté régulièrement au fil du temps. Selon StatCounter, la navigation mobile était essentiellement égale à la navigation sur ordinateur, chacune prenant une part de 48 % en janvier 2021. Perficient a estimé que 61 % de tout le trafic Web américain provenait d'appareils mobiles en 2020.

Un autre facteur majeur encourageant les entreprises à poursuivre une stratégie axée sur le mobile concerne les données et fonctionnalités supplémentaires que les sites Web mobiles peuvent fournir. Étant donné que les smartphones utilisent désormais généralement des interfaces à écran tactile, il est possible pour les propriétaires de sites Web mobiles d'obtenir des données détaillées sur l'expérience utilisateur (UX) concernant exactement la façon dont l'utilisateur a physiquement cliqué ou fait défiler la page. Dans certains cas, les propriétaires de sites Web peuvent même obtenir des données concernant les mouvements oculaires de l'utilisateur, déterminés à l'aide de la caméra frontale de leur téléphone.

Bien que ces métriques avancées de données utilisateur soient relativement rares, d'autres types de données utilisateur riches sont assez courantes. Un tel exemple est le suivi de localisation, dans lequel les coordonnées GPS des téléphones des utilisateurs sont automatiquement transmises au fournisseur du site Web. Ce type d'informations peut être déterminant pour éclairer les futures [campagnes de marketing](/marketing-strategy) ou planifier les efforts de développement commercial tels que l'ouverture de nouveaux magasins de détail ou d'entrepôts de distribution.

## Exemple concret d'une stratégie axée sur le mobile

Pourtant, le facteur le plus important à l'origine de l'adoption de stratégies axées sur le mobile est peut-être la stratégie "d'indexation axée sur le mobile" de Google. Cette décision, qui a été mise en pratique en juillet 2019, indiquait clairement qu'à l'avenir, l'algorithme de recherche de l'entreprise donnerait la priorité aux versions mobiles des sites Web lors de l'indexation initiale d'un nouveau site Web.

En pratique, cela signifie que les nouveaux sites Web développés sans stratégie mobile-first sont susceptibles de connaître des résultats d'optimisation des moteurs de recherche (SEO) inférieurs à la normale. Compte tenu de l'énorme influence dont jouit Google dans l'écosystème mondial des moteurs de recherche, il semble probable que les stratégies axées sur le mobile seront adoptées comme une pratique exemplaire pour la plupart sinon tous les nouveaux projets de développement de sites Web à l'avenir.

## Points forts

- À l'avenir, les stratégies mobiles d'abord deviendront probablement la norme, en partie en raison de la décision des principaux moteurs de recherche de donner la priorité aux versions mobiles des sites Web.

- Une stratégie mobile-first est une stratégie dans laquelle la version mobile d'un site Web est prioritaire sur sa version de bureau.

- Cette pratique était relativement rare dans le passé, mais est devenue de plus en plus courante.
