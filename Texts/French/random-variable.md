# Variable aléatoire
## Qu'est-ce qu'une variable aléatoire ?

Une variable aléatoire est une variable dont la valeur est inconnue ou une fonction qui attribue des valeurs à chacun des résultats d'une expérience. Les variables aléatoires sont souvent désignées par des lettres et peuvent être classées comme [discrètes](/discrete-distribution),. qui sont des variables qui ont des valeurs spécifiques, ou continues, qui sont des variables qui peuvent avoir n'importe quelle valeur dans une plage continue.

Les variables aléatoires sont souvent utilisées dans les analyses [économétriques](/econometrics) ou de [régression](/regression) pour déterminer les relations statistiques entre elles.

## Comprendre une variable aléatoire

Dans les probabilités et [les statistiques](/statistics),. les variables aléatoires sont utilisées pour quantifier les résultats d'une occurrence aléatoire et, par conséquent, peuvent prendre de nombreuses valeurs. Les variables aléatoires doivent être mesurables et sont généralement des nombres réels. Par exemple, la lettre X peut être désignée pour représenter la somme des nombres résultants après le lancement de trois dés. Dans ce cas, X pourrait être 3 (1 + 1 + 1), 18 (6 + 6 + 6), ou quelque part entre 3 et 18, puisque le nombre le plus élevé d'un dé est 6 et le nombre le plus bas est 1.

Une variable aléatoire est différente d'une variable [algébrique](/algebraic-method). La variable dans une équation algébrique est une valeur inconnue qui peut être calculée. L'équation 10 + x = 13 montre que nous pouvons calculer la valeur spécifique pour x qui est 3. D'autre part, une variable aléatoire a un ensemble de valeurs, et n'importe laquelle de ces valeurs pourrait être le résultat résultant comme on le voit dans l'exemple des dés ci-dessus.

Dans le monde de l'entreprise, des variables aléatoires peuvent être attribuées à des propriétés telles que le prix moyen d'un actif sur une période donnée, le [retour sur investissement](/returnoninvestment) après un certain nombre d'années, le taux de rotation estimé d'une entreprise dans les six mois suivants, etc. Les analystes des risques attribuent des variables aléatoires aux modèles de risque lorsqu'ils souhaitent estimer la probabilité qu'un événement indésirable se produise. Ces variables sont présentées à l'aide d'outils tels que des scénarios et des tableaux [d'analyse de sensibilité](/sensitivityanalysis) que les gestionnaires de risques utilisent pour prendre des décisions concernant l'atténuation des risques.

## Types de variables aléatoires

Une variable aléatoire peut être discrète ou continue. Les variables aléatoires discrètes prennent un nombre dénombrable de valeurs distinctes. Considérez une expérience où une pièce de monnaie est lancée trois fois. Si X représente le nombre de fois où la pièce tombe face, alors X est une variable aléatoire discrète qui ne peut avoir que les valeurs 0, 1, 2, 3 (de zéro tête dans trois lancers de pièces successifs à toutes les têtes). Aucune autre valeur n'est possible pour X.

Les variables aléatoires continues peuvent représenter n'importe quelle valeur dans une plage ou un intervalle spécifié et peuvent prendre un nombre infini de valeurs possibles. Un exemple de variable aléatoire continue serait une expérience qui consiste à mesurer la quantité de précipitations dans une ville sur une année ou la taille moyenne d'un groupe aléatoire de 25 personnes.

En s'appuyant sur ce dernier, si Y représente la variable aléatoire pour la taille moyenne d'un groupe aléatoire de 25 personnes, vous constaterez que le résultat résultant est un chiffre continu puisque la hauteur peut être de 5 pieds ou 5,01 pieds ou 5,0001 pieds. est un nombre infini de valeurs possibles pour la hauteur.

Une variable aléatoire a une [distribution de probabilité](/probabilitydistribution) qui représente la probabilité que l'une des valeurs possibles se produise. Disons que la variable aléatoire, Z, est le nombre sur la face supérieure d'un dé lorsqu'il est lancé une fois. Les valeurs possibles pour Z seront donc 1, 2, 3, 4, 5 et 6. La probabilité de chacune de ces valeurs est de 1/6 car elles sont toutes également susceptibles d'être la valeur de Z.

Par exemple, la probabilité d'obtenir un 3, ou P (Z=3), lorsqu'un dé est lancé est de 1/6, de même que la probabilité d'avoir un 4 ou un 2 ou tout autre nombre sur les six faces d'un mourir. Notez que la somme de toutes les probabilités est 1.

## Exemple de variable aléatoire

Un exemple typique de variable aléatoire est le résultat d'un tirage au sort. Considérons une distribution de probabilité dans laquelle les résultats d'un événement aléatoire ne sont pas également susceptibles de se produire. Si la variable aléatoire Y est le nombre de faces que nous obtenons en lançant deux pièces, alors Y pourrait être 0, 1 ou 2. Cela signifie que nous pourrions n'avoir aucune face, une face ou les deux face sur un lancer de deux pièces.

Cependant, les deux pièces atterrissent de quatre manières différentes : TT, HT, TH et HH. Par conséquent, le P(Y=0) = 1/4 puisque nous avons une chance de ne pas tomber sur pile (c'est-à-dire deux piles [TT] lorsque les pièces sont lancées). De même, la probabilité d'obtenir deux têtes (HH) est également de 1/4. Notez qu'avoir une tête a une probabilité de se produire deux fois : en ME et TH. Dans ce cas, P (Y=1) = 2/4 = 1/2.

## Points forts

- Une variable aléatoire est une variable dont la valeur est inconnue ou une fonction qui attribue des valeurs à chacun des résultats d'une expérience.

- Une variable aléatoire peut être soit discrète (ayant des valeurs spécifiques) soit continue (toute valeur dans une plage continue).

- Les analystes des risques utilisent des variables aléatoires pour estimer la probabilité qu'un événement indésirable se produise.

- L'utilisation de variables aléatoires est plus courante dans les probabilités et les statistiques, où elles sont utilisées pour quantifier les résultats d'occurrences aléatoires.

## FAQ

### Qu'est-ce qu'une variable aléatoire continue ?

Une variable aléatoire continue représente n'importe quelle quantité dans une plage ou un ensemble de points spécifique et peut refléter un nombre infini de valeurs potentielles, telles que les précipitations moyennes dans une région.

### Qu'est-ce qu'une variable aléatoire discrète ?

Une variable aléatoire discrète est un type de variable aléatoire qui a un nombre dénombrable de valeurs distinctes qui peuvent lui être attribuées, comme dans un tirage au sort.

### Qu'est-ce qu'une variable aléatoire mixte ?

Une variable aléatoire mixte combine des éléments de variables aléatoires discrètes et continues.
