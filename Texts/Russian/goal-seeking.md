# поиск цели
## Что такое поиск цели?

Поиск цели — это процесс нахождения правильного входного значения, когда известен только результат. Функция поиска цели может быть встроена в различные компьютерные программы, такие как Microsoft Excel.

## Поймите поиск цели

Поиск цели — это общий термин, используемый для описания процесса, связанного с определением вашего входного значения на основе уже известного выходного значения. Процесс включает использование определенного оператора в формуле, которую можно рассчитать с помощью компьютерного программного обеспечения.

Поиск цели - один из инструментов, используемых в анализе «что, если» в компьютерных программах. Анализ « [что,](/sensitivityanalysis) [если](/sensitivityanalysis) » — это процесс изменения значений в ячейках (Microsoft Excel), чтобы увидеть, как эти изменения повлияют на результаты формулы на листе. Когда вы ищете цель, вы выполняете анализ «что, если» для заданного значения или результата. Таким образом, по сути, вы создаете сценарий, спрашивая «что, если результат будет Х» — или, по сути, причинно-следственную ситуацию.

Для решения некоторых более сложных задач люди часто используют компьютерное программное обеспечение. Программа для работы с электронными таблицами, такая как Microsoft Excel, имеет встроенный инструмент для поиска целей. Это позволяет пользователю определить желаемое входное значение для формулы, когда выходное значение уже известно. Эта функция может помочь пользователю определить такие вещи, как [процентная ставка](/interestrate),. на которую заемщик должен претендовать (входные данные), если он знает только, сколько он может позволить себе платить каждый месяц (выходные данные).

Но есть одно предостережение при использовании программного обеспечения для поиска целей: оно работает только в том случае, если есть одно входное значение. Если вам нужно выяснить два или более входных значения, это не сработает. Если мы возьмем приведенный выше пример, если вы хотите выяснить общую сумму кредита и ежемесячный платеж, программное обеспечение для поиска целей не будет работать. Вам, вероятно, понадобится надстройка, чтобы выяснить несколько переменных.

### Как поиск цели работает в Microsoft Excel

Как упоминалось выше, программное обеспечение для поиска целей будет работать только в том случае, если вы уже знаете выходное значение (или результат), но хотите определить одно входное значение. Вот несколько ключевых шагов, которые вы можете выполнить, если хотите использовать функцию поиска цели в программе:

- Откройте новую таблицу

- Назовите свои столбцы. Это облегчит вам чтение всего в рабочем листе. Итак, в приведенном выше примере первый столбец будет «Сумма кредита», второй столбец будет « [Срок](/term) в месяцах», третий будет «Процентная ставка», а последний будет «Платеж».

- Введите значения, которые вы знаете.

- Введите формулу для расчета цели — в данном случае платежа. На данный момент вы можете игнорировать процентную ставку, которая будет считаться формулой равной 0 процентов.

Теперь вам нужно определить процентную ставку. Вы можете сделать это с помощью функции «Поиск цели» в Excel и ввести значения, которые у вас уже есть.

## Пример поиска цели

Целеустремленный [предприниматель](/entrepreneur) — это тот, кто использует поиск целей, чтобы определить, как они могут достичь своих конечных целей. Например, предприниматель может спросить, сколько ему нужно будет зарабатывать в час, чтобы получить 100 000 долларов в год. Они знают свою желаемую выходную стоимость — 100 000 долларов — и, следовательно, должны будут вернуться назад, чтобы вычислить оптимальные входные данные. Это означает, что им придется определить, сколько часов они смогут (или захотят) работать в течение года и, следовательно, сколько они будут зарабатывать за каждый час, чтобы достичь своей цели.

## Особенности

- Однако программное обеспечение поиска цели работает только в том случае, если есть только одно входное значение.

- Поиск цели - это поиск правильного входа, когда известен только результат.

- Это часто делается с помощью функции поиска цели в Microsoft Excel или анализа «что, если» с помощью компьютерных программ.
