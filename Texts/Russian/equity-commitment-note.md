# Примечание об обязательствах по капиталу (ECN)
## Что такое нота об обязательствах по акциям (ECN)?

Обязательство по выпуску акций — это тип [обязательного конвертируемого](/mandatoryconvertible) долга, выпущенного банком, который может быть обменен на акции при наступлении срока погашения. Вексель погашается за счет продажи акций его эмитентом.

По сути, это тип корпоративного долга, который имеет ограничения для заемщика, определяющие, как будут собираться деньги для погашения долга.

## Понимание примечания об обязательствах по капиталу (ECN)

Записка об обязательствах по акционерному капиталу является формой корпоративного долга. Он погашается путем продажи или выпуска ценных бумаг в будущем банком или кредитным учреждением, которое их выпускает.

Нота об обязательстве по долевому участию отличается от ноты о договоре о долевом капитале тем, что инвестору не требуется покупать ценные бумаги, чтобы выкупить ноту. Записка об обязательстве по долевому участию может быть погашена позднее путем продажи обыкновенных или привилегированных акций.

В Соединенных Штатах Федеральный резервный банк устанавливает максимальный срок погашения в 12 лет и требует, чтобы компания-эмитент финансировала одну треть капитала каждые четыре года .

## Особенности

- Вексель можно обменять на акции по истечении срока.

- Обязательство по выпуску акций — это тип обязательной конвертируемой облигации, выпускаемой банком или другим кредитным учреждением.

- По сути, вексельное обязательство является корпоративной облигацией.
