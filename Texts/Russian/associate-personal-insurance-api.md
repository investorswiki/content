# Ассоциированный специалист по личному страхованию (API)
## Что такое партнер по личному страхованию (API)?

Associate in Personal Insurance (API) — это профессиональная сертификация в [страховой отрасли](/insurance). Как следует из названия, API ориентирован на [страхование личных линий](/personal-insurance-lines),. которое представляет собой тип страхования, защищающий от телесных повреждений, смерти или потери имущества. Это одно из многих званий, присуждаемых Институтами, организацией, занимающейся предоставлением аккредитации и повышением квалификации специалистов в области страхования.

## Как работает обозначение API

Обозначение API популярно среди профессионалов, которые хотят начать или продвинуться по карьерной лестнице в сфере страхования личных линий. Получив API, специалисты могут продемонстрировать, что они обладают знаниями о некоммерческих страховых продуктах, в том числе такими важными навыками, как [страховое андеррайтинг](/underwriting),. [установление премий](/insurance-premium),. маркетинг и управление портфелем.

Для профессионалов, желающих получить более продвинутые сертификаты, такие как обозначение [Chartered Property Emergency Underwriter (CPCU)](/chartered-property-casualty-underwriter-cpcu),. API может быть популярным вариантом. Это связано с тем, что многие условия, необходимые для получения API, также учитываются в требованиях CPCU. В частности, два из трех экзаменов, связанных с API, могут быть повторно использованы для получения аккредитации CPCU, которая является одной из самых престижных наград в сфере страхования от несчастных случаев.

Чтобы получить API, кандидаты должны получить обширные знания о видах страховых полисов, продаваемых на рынке личного страхования. Это включает широкое образование по таким предметам, как одобрение контрактов, условия и критерии андеррайтинга, а также отраслевые правила. API часто рекомендуется страховым брокерам, агентам, специалистам по урегулированию убытков, андеррайтерам и персоналу службы поддержки клиентов, которые обслуживают клиентов по личному страхованию.

## Реальный пример обозначения API

Есть два пути, которыми кандидаты могут воспользоваться при заполнении API. Первый из них включает курсы по [андеррайтингу](/underwriting) и маркетинговой практике в индустрии личного страхования, управлению портфелем, страхованию имущества и [ответственности](/liability_insurance) и личному страхованию. Второй путь состоит из трех курсов, охватывающих андеррайтинг и маркетинг, [управление рисками](/riskmanagement) и имущественной ответственностью, а также личное финансовое планирование.

Как правило, кандидатам требуется от 12 до 18 месяцев для изучения этих материалов курса, доступ к которым предоставляется на основе самообслуживания. Для тех, кто предпочитает очное обучение, в некоторых регионах занятия также могут быть доступны в автономном режиме. После того, как они сдали свои экзамены, владельцы API не обязаны соответствовать текущим образовательным требованиям, чтобы сохранить свое назначение.

## Особенности

- API — это профессиональное обозначение, популярное в сфере страхования личных линий.

- Он находится в ведении Институтов и обычно занимает от 12 до 18 месяцев.

- Многие кандидаты предпочитают добавлять к своему назначению API, преследуя более продвинутые учетные данные, такие как CPCU.
