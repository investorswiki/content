# ЗНАЧОК
## Что такое ICON (ICX)?

ICON — это платформа, предназначенная для облегчения взаимодействия независимых блокчейнов, также называемых сообществами. В рамках платформы ICON сообщество определяется как сеть узлов, которые следуют единой системе управления. (Узел — это компьютер, который подключается к криптовалютной сети.)

Криптовалютные сети (такие как Биткойн и Эфириум) считаются сообществами, но банки, предприятия, больницы, школы и правительства также могут участвовать в качестве сообществ в этой экосистеме. ICON поддерживается через [токен криптовалюты](/cryptocurrency) ICX. Проект ICON был основан в 2017 году организацией Icon Foundation из Южной Кореи.

Цель проекта ICON — создать платформу, на которой организации из самых разных отраслей — финансов, безопасности, страхования, здравоохранения, образования и торговли — могут взаимодействовать и совершать транзакции в единой сети.

## Понимание ICON (ICX)

Первоначальное размещение монет (ICO) в сентябре 2017 года для криптовалюты Icon, ICX, привлекло около 43 миллионов долларов. Эта продажа включала 50% токенов ICON. Проект официально запустил свой блокчейн в январе 2018 года, а участники ICO получили свою криптовалюту ICX в июне 2018 года.

В 2019 году ICON выпустила собственный стандарт токенов IRC16, который позволяет пользователям использовать программное обеспечение ICON в качестве инфраструктуры для создания собственных токенизированных активов и ценных бумаг . единицы. С токенизированным капиталом бизнес может привлекать капитал, выпуская свои акции в виде цифровых активов.

Вот как ICON описывает свою цель: создать «цифровую нацию», в которой различные экономические субъекты могут выпускать и контролировать свои собственные формы стоимости в соответствии с выбранными ими системами правил. Согласно веб-сайту, целью проекта ICON является «гиперсвязь мира» путем «построения одной из крупнейших децентрализованных сетей в мире » .

Проект ICON черпал вдохновение для своего проекта из того, как устроена экономика реального мира; предприятия, некоммерческие организации и государственные учреждения используют общее средство обмена — национальную валюту — при этом оставаясь отдельными, но взаимодействующими экономическими субъектами. центральная точка, создающая взаимосвязанную сеть блокчейнов.

Блокчейн — [это](/blockchain) тип базы данных. Она отличается от обычной базы данных способом хранения данных. Данные хранятся в блоках в хронологическом порядке: по мере поступления новых данных они вводятся в новый блок. Как только блок заполняется данными, этот блок присоединяется к предыдущему блоку.

Хотя блокчейны чаще всего ассоциируются с криптовалютами, существуют дополнительные области, в которых эта технология может применяться, поскольку в блокчейне могут храниться различные типы информации. Хотя до сих пор наиболее распространенным использованием было использование в качестве реестра транзакций, любой обмен информацией может быть добавлен в блокчейн. Транзакции с ценными бумагами, валютой, деловыми контрактами, документами, кредитами, интеллектуальной собственностью и личной идентификацией могут быть токенизированы.

Каждый блокчейн управляется определенной сетью. По этой причине в прошлом подключение разных блокчейнов не рассматривалось как возможность. Проект ICON — это попытка такой связи. В отличие от других централизованных платежных систем, которые требуют от компаний соблюдения их собственных политик и правил, ICON позволяет отдельным сообществам сохранять контроль над своими собственными политиками. Таким образом, ICON стремится уменьшить потенциальные барьеры для внедрения.

## Цели ICON

ICON стремится создать большую сеть, состоящую из отдельных сетей, которая работает на криптовалютах. Блокчейны, участвующие в проекте ICON, могут обменивать валюту с помощью децентрализованной биржи платформы (DEX).

DEX устанавливает валютные резервы для каждого сообщества блокчейнов, чтобы сообщества могли обмениваться ценностями в режиме реального времени. Курсы обмена устанавливаются для этих транзакций с использованием модели анализа ИИ, а нативная монета ICX используется в качестве промежуточной валюты для обмена.

Проект ICON состоит из пяти основных компонентов: Республика ICON, сообщества ICON, представители сообщества (C-Rep), узлы сообщества (C-Nodes) и узлы граждан. Сообщества ICON представляют собой сеть узлов в рамках единой системы управления; они могут работать со своими собственными структурами управления, количеством узлов и характеристиками. Сообщества также могут различаться по своим подходам к принятию решений. Например, в то время как биткойн использует подход, основанный на консенсусе, другие сообщества, такие как финансовые учреждения, могут следовать иерархическому подходу.

Узлы сообщества — это инфраструктура для каждого сообщества; они поддерживают блокчейн сообщества. Представители сообщества — это узлы, которые избираются внутри каждого сообщества и которым доверяют взаимодействие с ICON Republic. Через представителя сообщества блокчейн каждого сообщества связан с ICON Republic. Представители сообщества получают ICX за свою работу.

ICON Republic является управляющей структурой сети. Это также комитет по принятию решений в отношении операций блокчейна. ICON Управление республикой определяется голосами представителей сообщества. Полезная аналогия — представить республику ICON как Соединенные Штаты и представить каждое сообщество ICON как отдельный штат.

Действия ICON Republic не определяют управление отдельными общинами. Тем не менее, ICON Republic регулирует процесс чеканки новой криптовалюты ICX.

Узлы-граждане — это участники блокчейна, которые не имеют права голоса, но могут совершать транзакции внутри сообщества, с другими сообществами и с Icon Republic.

Блокчейн ICON Republic называется «нексус». Технология, на которой работает нексус, — это алгоритм, называемый loopchain. Петля соединяет сообщества, которые являются частью ICON Republic. Эти сообщества — когда они сгруппированы вместе, они называются консорциумом — определяют общий набор правил, которые позволяют различным блокчейнам эффективно работать вместе.

Набор правил, регулирующих взаимодействие независимых блокчейнов с блокчейном ICON Republic — нексусом, — называется протоколом передачи блокчейна (BTP).

Наконец, как и в случае с [Ethereum , любой может использовать ICON для создания децентрализованных приложений, называемых](/ethereum) [DAPP](/ethereum),. через общедоступный канал блокчейна. Кроме того, после того, как эти приложения будут перечислены в магазине ICON DAPP, любой пользователь, у которого есть гражданский узел, сможет загрузить и использовать их.

## Критика ICON

Несмотря на то, что ICON имеет амбициозные планы по объединению различных сообществ, на пути к этой цели она сталкивается со многими проблемами. Во-первых, многие инвесторы привыкли к традиционным централизованным биржам криптовалют. Хотя эти традиционные централизованные биржи действительно не соответствуют децентрализованной идеологии криптовалют, в 2021 году эти биржи по-прежнему являются неотъемлемой частью рынка криптовалют. Многие инвесторы покупают только те криптовалюты, которые они могут купить на своих любимых биржах. Это может создать проблемы для ICX. Учитывая цели ICON, многие биржи могут не захотеть предлагать ICX своим клиентам.

Высокий уровень конкуренции увеличивает риск того, что конкретная криптовалюта не завоюет популярность. Как и другие криптовалюты, ICON сталкивается с жесткой конкуренцией со стороны других криптовалют.

## Особенности

- Сообщества подключены к сети ICON через децентрализованный обмен.

- Потенциальное противодействие со стороны традиционных централизованных криптовалютных бирж является серьезной проблемой, стоящей перед ICON.

- ICON — это технология блокчейна и сетевая структура, разработанная для того, чтобы независимые блокчейны могли взаимодействовать друг с другом.
