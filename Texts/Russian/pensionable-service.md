# Пенсионная служба
## Что такое пенсионная служба?

Засчитываемый для пенсии стаж относится к количеству времени, в течение которого работник накапливает кредит в отношении [пенсионного плана,](/pensionplan) в котором он зарегистрирован. Согласно канадскому законодательству, человек может накопить в общей сложности до 35 лет пенсионного стажа.

## Понимание пенсионного обслуживания

Засчитываемый для пенсии стаж — это количество времени, которое работник, зарегистрированный в пенсионном плане, накапливает в рамках этого плана во время своей работы. Обычно сообщаемый в отчетах годовой показатель, засчитываемый для пенсии стаж является одним из основных факторов при определении пенсионных пособий работника, наряду с самой высокой средней заработной платой.

Стоимость зачитываемой для пенсии службы, рассчитанная для каждого работника, чаще всего основывается на учете рабочего времени работодателя, выраженного в часах, годах или других интервалах. Каждый пенсионный план различается по расчету засчитываемого для пенсии стажа, поэтому участникам плана необходимо ознакомиться с конкретными условиями своего плана.

Период, в течение которого работник вносит засчитываемый для пенсии стаж непосредственно в пенсионный план, известен как «текущий стаж».

В дополнение к начислению текущего стажа физическое лицо может добавить соответствующий критериям [предыдущий стаж](/pastservice) для увеличения своей пенсии, что может быть сделано путем приобретения засчитываемого для пенсии стажа за соответствующие периоды предыдущей работы или путем перевода стоимости накопленных пенсионных пособий из плана другого работодателя через пенсионный план. ## Соглашение о переводе пенсии

Соглашение о переводе пенсии — это соглашение, заключенное между правомочным работодателем и правительством Канады, которое облегчает перевод между пенсионными планами.

Работодатель должен заключить официальное соглашение с правительством Канады, чтобы сотрудник мог воспользоваться Соглашением о переводе пенсии. По состоянию на 2021 год более 90 пенсионных планов участвуют в соглашениях о переводе пенсий в Канаде. Сотрудник, чья пенсия не имеет действующего Соглашения о переводе пенсии, может обратиться к своему работодателю с просьбой принять соглашение или он может изучить вариант программы выкупа услуг.

## Услуга выкупа, за которую начисляется пенсия

Для некоторых канадцев показатели зачитываемой для пенсии службы могут также включать периоды [оплачиваемой службы](/purchased-service) в качестве пособия по пенсионному плану государственной службы. Приобретение засчитываемых для пенсии услуг, также известное как выкуп услуг, должно быть рассчитано с использованием корректировки пенсии за прошлые периоды службы и подтверждено Налоговой службой Канады перед подсчетом.

Виды зачитываемой для пенсии службы, которые могут быть приобретены, включают предыдущую государственную службу, периоды службы, в течение которых участник не делал взносов в пенсионный план, и службу, признаваемую другими пенсионными планами. Такие покупки регулируются правилами [Налогового управления Канады](/ccra),. и условия варьируются в зависимости от планов и провинций.

Дополнительные требования и затраты могут применяться к обратному выкупу услуг, и применяется 35-летний предел накопления засчитываемых для пенсии услуг.

## Особенности

- Период зачитываемого для пенсии стажа именуется «текущий стаж».

- Для некоторых канадцев зачитываемый для пенсии стаж может также включать периоды платной службы в качестве льготы по пенсионному плану государственной службы.

- Работники могут добавить предыдущую службу к нарастающей текущей службе, чтобы увеличить свою пенсию.

- В соответствии с канадским законодательством, человек может накопить в общей сложности до 35 лет пенсионного стажа.

- Зачитываемый для пенсии стаж – это время, начисляемое по пенсионному плану.
