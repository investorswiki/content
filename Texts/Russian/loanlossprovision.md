# Резерв на возможные потери по ссудам
## Что такое резерв на возможные потери по ссуде?

Резерв на покрытие убытков по ссудам представляет собой расход в [отчете о прибылях и убытках](/incomestatement),. отложенный в качестве резерва на непогашенные ссуды и платежи по ссудам. Этот резерв используется для покрытия различных видов убытков по кредитам, таких как [неработающие кредиты](/nonperformingloan),. банкротство клиентов и [пересмотренные кредиты](/renegotiated-loan),. по которым платежи ниже, чем предполагалось ранее. Затем резервы на потери по ссудам добавляются к резервам на потери по ссудам, статье [баланса](/balancesheet),. которая представляет собой общую сумму убытков по ссудам за вычетом ссуд компании .

## Как работает резерв на покрытие убытков по ссуде

Кредиторы банковской отрасли получают доход от процентов и расходов, которые они получают от кредитных продуктов. Банки кредитуют широкий круг клиентов, включая потребителей, малый бизнес и крупные корпорации.

Стандарты кредитования и требования к отчетности постоянно меняются, а ограничения жестко ужесточаются после пика финансового кризиса 2008 года. Усовершенствованные правила для банков, вытекающие из [закона Додда-Франка,](/dodd-frank-financial-regulatory-reform-bill) были сосредоточены на повышении стандартов кредитования, которые требовали от заемщиков более высокого кредитного качества, а также повышали требования к [ликвидности капитала](/liquidity) для банка .

Несмотря на эти улучшения, банкам по-прежнему приходится учитывать дефолты по кредитам [и](/default2) расходы, возникающие в результате кредитования. Резервы на возможные потери по ссудам представляют собой стандартную бухгалтерскую корректировку резервов на потери по ссудам, включенных в финансовую отчетность банков. Резервы на возможные потери по ссудам постоянно создаются с учетом изменяющихся прогнозов потерь от кредитных продуктов банка. Хотя стандарты кредитования значительно улучшились, банки по-прежнему сталкиваются с просроченными платежами по кредитам и неплатежами по кредитам .

> Поскольку резерв на возможные потери по ссудам отображается в отчете о прибылях и убытках как расход, он снизит операционную прибыль.

>

## Резервы на возможные потери по ссудам в бухгалтерском учете

Резервы на возможные потери по ссудам обычно учитываются в [балансе банка](/balancesheet),. который может увеличиваться на сумму резерва на возможные потери по ссудам или уменьшаться на сумму чистых [списаний](/chargeoff) каждый квартал.

Резервы на возможные потери по ссудам постоянно создаются для обновления оценок и расчетов на основе статистики неплатежей клиентов банка. Эти оценки рассчитываются на основе средних исторических показателей дефолтов по разным уровням заемщиков. Кредитные убытки по просроченным платежам и расходы по взысканию также включаются в оценки резервов на возможные потери по ссудам и рассчитываются с использованием аналогичной методологии, которая учитывает предыдущую статистику платежей кредитных клиентов банка.

В целом, откладывая резервы на возможные потери по ссудам и постоянно обновляя оценки с помощью резервов на потери по ссудам, банки могут обеспечить точную оценку своего общего финансового положения. Это финансовое положение часто раскрывается публично через ежеквартальные финансовые отчеты банка.

## Особенности

- Резервы на возможные потери по ссудам добавляются к резервам на возможные потери по ссудам, статье балансового отчета, показывающей общие потери по ссудам.

- Резерв на покрытие убытков по ссудам представляет собой расход в отчете о прибылях и убытках, предназначенный для покрытия непогашенных ссуд и платежей по ссудам.

- Банки обязаны учитывать потенциальные дефолты по кредитам и расходы, чтобы обеспечить точную оценку своего общего финансового состояния.
