# Флип-флоп Примечание
## Что такое Flip-Flop Note?

Флип-флоп — это ценная бумага с [фиксированным доходом](/fixed-incomesecurity),. обеспеченная двумя разными источниками долга — один с [фиксированной процентной ставкой,](/fixedinterestrate) а другой с [переменной процентной ставкой](/variableinterestrate),. что позволяет держателю выбирать поток платежей.

## Понимание Flip Flop Note

Заметка с флип-флопом действует как [встроенный вариант](/embeddedoption) для инвесторов. Другими словами, держатель флип-флоп-ноты может выбирать между переменной процентной ставкой и фиксированной процентной ставкой в зависимости от того, какая из них имеет более высокую [доходность](/yield) за период.

Все флип-флоп-ноты технически являются долговыми ценными бумагами, состоящими из двух типов долговых обязательств. Переменная процентная ставка — это ставка по кредиту или ценной бумаге, которая колеблется с течением времени из-за того, что она основана на базовом [контрольном показателе](/benchmark) или индексе, который периодически меняется. Ценная бумага с фиксированной процентной ставкой выплачивает определенную процентную ставку, которая не меняется в течение срока действия инструмента. Как и в случае с любой другой ценной бумагой с фиксированным доходом, номинальная стоимость флип-флоп-ноты возвращается инвестору, когда ценная бумага достигает [даты погашения](/maturitydate).

Большинство флип-флоп-нот ограничиваются, когда держатель может выбрать, какая процентная ставка является основой для его процентных платежей, до конкретных дат. Переключение между двумя ставками позволяет держателю получать доход от процентной ставки в зависимости от того, какой тип долга приносит больше процентов в данный момент.

В соответствии со **Словарем по управлению финансовыми рисками**, срочная нота также может позволить заемщику сократить или удлинить срок ноты в зависимости от возможностей, доступных на дату сброса ноты.﻿Эта функция означает заемщик не может выбирать изменение процентной ставки по прихоти. Вместо этого это должно произойти, когда дата сброса облигации истекла. Когда наступает дата сброса, у инвестора есть возможность выбрать продление или сокращение срока инвестиции, будь то облигация или вексель, исходя из текущих процентных ставок.

## Пример флип-флоп-ноты

Например, типичная флип-флоп-нота может состоять из долга с фиксированной ставкой и облигации с [плавающим купоном](/frn). Если плавающая процентная ставка падает ниже фиксированного купона, инвестор может выбрать получение дохода от долга с фиксированной ставкой. И наоборот, когда плавающая ставка превышает фиксированный купон, инвестор переключается на заем с плавающей ставкой для получения дохода. В этой ситуации флип-флоп-нота аналогична облигации с плавающей [процентной ставкой и минимальной процентной](/interestratefloor) [ставкой](/interestratefloor).

Заметка с триггером также может позволить инвестору переключаться между двумя типами ценных бумаг для своих инвестиций. Например, флип-флоп-нота может использоваться для переключения с долгосрочной облигации на краткосрочную ноту с фиксированной процентной ставкой или фиксированную ценную бумагу. В некоторых случаях для переключения с нот на акции также можно использовать флип-флоп.

## Особенности

- Большинство флип-флоп-нот ограничиваются, когда держатель может выбрать, какая процентная ставка является основой для его процентных платежей, до определенных дат.

- Облигация Flip-flop — это ценная бумага с фиксированным доходом, обеспеченная двумя разными источниками долга — один с фиксированной процентной ставкой, а другой с переменной процентной ставкой, что позволяет держателю выбирать поток платежей.

- Заметка флип-флоп действует как встроенный вариант для инвесторов.
