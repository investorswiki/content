# Коробка стиля с фиксированным доходом
## Что такое коробка стиля с фиксированным доходом?

Блок стиля с фиксированным доходом обеспечивает визуальное представление инвестиционных характеристик инвестиций с фиксированным доходом. Коробки с фиксированным доходом были созданы Morningstar и чаще всего используются для [взаимных фондов](/mutualfund). Они являются ценным инструментом, который инвесторы могут использовать при определении структуры риска и доходности своих инвестиций с [фиксированным доходом .](/fixedincome) Они также помогают инвесторам классифицировать и выбирать инвестиции на основе определенных критериев инвестирования.

Коробку стиля с фиксированным доходом можно противопоставить [ячейке стиля акций](/equity_stylebox).

## Понимание коробки стиля с фиксированным доходом

Коробка с фиксированным доходом состоит из девяти квадратов с вертикальной и горизонтальной осями, используемыми для определения инвестиционных характеристик. Morningstar использует чувствительность к процентным ставкам и [кредитное качество](/creditquality) в качестве двух основных характеристик для рассмотрения.

На горизонтальной оси инвесторы найдут три категории для классификации чувствительности к процентным ставкам: ограниченная, умеренная и обширная. Чувствительность к процентной ставке зависит от продолжительности существования фонда. Таким образом, краткосрочные фонды с фиксированной процентной ставкой относятся к ограниченной категории, в то время как долгосрочные фонды с фиксированной процентной ставкой относятся к расширенной категории.

На вертикальной оси рейтинги кредитного качества являются вторым фактором, используемым для классификации инвестиций в фонды с фиксированным доходом. Категории кредитного качества коробки стиля включают высокий, средний и низкий.

Morningstar предоставляет подробную разбивку параметров для классификации квадратов стилей. Классификация чувствительности к процентным ставкам определяется средней трехлетней дюрацией фонда по сравнению с индексом Morningstar Core Bond. Кредитное качество определяется по методике, включающей средневзвешенные кредитные рейтинги фонда. Средства с высоким кредитным качеством будут иметь средневзвешенный кредитный рейтинг AA- и выше. Фонды с низким кредитным качеством будут иметь средневзвешенный кредитный рейтинг ниже BBB-. Morningstar является основным разработчиком коробок с фиксированным доходом, однако существуют варианты от других поставщиков финансовой информации.

## Анализ коробки с фиксированным доходом

Инвестор с фиксированным доходом, использующий стиль инвестирования в коробку для определения инвестиций с фиксированным доходом, может быть заинтересован в фильтрации наиболее эффективных фондов с ограниченной чувствительностью к процентным ставкам и высоким кредитным качеством.

Инвесторы, стремящиеся к более высокой потенциальной доходности от инвестиций с более низким кредитным качеством, с постоянным акцентом на ограниченную чувствительность к процентным ставкам, захотят отфильтровать ограниченное и низкое качество.

## Особенности

- В блоке с фиксированным доходом в качестве двух основных характеристик для рассмотрения используются чувствительность к процентным ставкам и кредитное качество, а на горизонтальной оси показаны категории погашения фонда, представленные краткосрочными, среднесрочными и долгосрочными.

- Блок с фиксированным доходом состоит из девяти квадратов с вертикальной и горизонтальной осями, используемыми для определения инвестиционных характеристик, чтобы помочь инвесторам оценить ценную бумагу или фонд с фиксированным доходом.

- Коробка стиля — это визуальный метод оценки характеристик инвестиций, ставший популярным благодаря финансовой аналитической фирме Morningstar.
