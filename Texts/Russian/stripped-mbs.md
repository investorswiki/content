# Раздетый МБС
## Что такое очищенная MBS?

Ценная бумага с ипотечным покрытием (MBS) — это тип ценной бумаги, [обеспеченной ипотекой](/mbs),. которая делится на [только основную](/postrips) и [процентную](/iostrips) [.](/iostrips) [полоски](/strip). Они получают свои денежные потоки либо от выплаты основной суммы долга, либо от выплаты процентов по лежащим в основе ипотечным кредитам, в отличие от обычных MBS, которые генерируют доход из обоих этих двух источников.

Обрезанные MBS очень чувствительны к изменениям [процентных ставок](/interestrate),. что делает их популярными среди инвесторов, которые считают, что могут предсказать будущую стоимость займа.

## Понимание разделенной MBS

Ипотечные ценные бумаги (MBS) — это инвестиции, состоящие из набора ипотечных [кредитов](/loan),. купленных у выпустивших их банков. Инвестирование в MBS означает покупку прав на эти различные ипотечные кредиты. Если все идет по плану и покупатели жилья не объявляют [дефолт](/default2),. они должны выплачивать регулярные ежемесячные процентные платежи, сумму, которую кредитор взимает, чтобы занять деньги, а также фактическое погашение кредита, также известное как основная сумма.

Урезанные MBS были созданы для привлечения различных типов инвесторов, давая им возможность получить либо проценты, либо основную сумму, а не и то, и другое. Инвестор свободен решать, какой стратегии придерживаться, исходя из своих потребностей в доходах и перспектив рынка. В частности, это требует прогнозирования направления процентных ставок.

## Полосы только для основного капитала и полосы только для процентов

Существуют некоторые фундаментальные различия между полосами только для основной суммы и полосами только для процентов.

Полосы только для основной суммы состоят из известной суммы в долларах, но неизвестных сроков оплаты. Они продаются инвесторам со скидкой от [номинала](/facevalue),. которая определяется процентной ставкой и скоростью [досрочного погашения](/prepayment).

С другой стороны, процентные стрипы генерируют высокие уровни денежных потоков в первые годы и значительно меньшие денежные потоки в последующие годы. Инвесторы могут выбирать между полосами только для основной суммы и полосами только для процентов в зависимости от того, что, по их мнению, будут делать процентные ставки в будущем.

## Особые соображения

### Процентные ставки

Из-за своей структуры изменения процентной ставки оказывают противоположное влияние на стрипы только для основной суммы и только для процентов. Повышение ставок увеличивает [ставку дисконтирования](/discountrate) [,](/discountrate) применяемую к денежным потокам, снижая цену полос только для основной суммы.

Доходность [стрипов](/yield) только для основной суммы напрямую зависит от скорости досрочного погашения: чем быстрее происходит досрочное погашение основного долга, тем выше общая доходность для инвестора стрипа только для основного долга. Поскольку предоплата увеличивается по мере снижения процентных ставок, инвесторы, покупающие только основную сумму, стремятся к более низким процентным ставкам.

Напротив, процентные полоски растут в цене, когда процентные ставки повышаются. Более высокие процентные ставки также снижают уровень досрочного погашения, что приводит к тому, что ипотека длится дольше. В этих обстоятельствах полоски только для процентов будут расти в цене, потому что они будут приносить проценты в течение более длительного периода времени.

> Разделенный MBS может быть адаптирован таким образом, чтобы он содержал больше процентов или больше основной суммы, предлагая инвестору индивидуальную величину [риска процентной ставки](/interestraterisk).

>

Проще говоря, когда процентные ставки падают, стрипы только для основной суммы будут расти в цене, а стрипы только для процентов будут снижаться. И наоборот, когда процентные ставки растут, процентные стрипы растут в цене, а стрипы только для основной суммы падают. Другими словами, если инвестор считает, что процентные ставки растут, он будет покупать только процентные стрипы. Если вместо этого инвестор считает, что процентные ставки снизятся, он купит полоски только для основной суммы.

## Особенности

- Доход получается либо за счет основной суммы долга, либо за счет процентов по лежащим в его основе ипотечным кредитам, в отличие от обычных MBS, денежные потоки которых основаны на обоих.

- Прибыльность стрипов только для основной суммы и стрипов только для процентов зависит, прежде всего, от направления процентных ставок.

- Разделенная ценная бумага с ипотечным покрытием (MBS) разделяет основную и процентную части MBS на отдельные ценные бумаги.
