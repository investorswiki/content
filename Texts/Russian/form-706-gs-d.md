# Форма IRS 706-GS(D)
## Что такое форма 706-GS(D): налоговая декларация о передаче без учета генерации для распределений?

Форма 706-GS(D) — это налоговая форма, распространяемая [Налоговой службой](/irs) (IRS), которая используется для расчета налогов, причитающихся с трастовых распределений, облагаемых налогом на [передачу с пропуском поколения](/generation-skipping-transfer-tax) (GSTT).

Когда человек умирает и оставляет свое имущество наследникам, он может попытаться избежать крупной суммы налогов на наследство, передав активы наследникам, которые не являются их детьми. Для этого они могут передать большое количество активов своим внукам или другим наследникам. Если дети лица, предоставившего право, никогда лично не вступают во владение имуществом, [налог на наследство](/estatetax) не применяется. Один из способов, которым человек может сделать это, — через траст.

Пропуск поколений по-прежнему может дать некоторые преимущества детям лица, предоставившего право. Например, лицо, предоставляющее право, может указать, что его дети имеют право на любой доход, генерируемый активами в доверительном управлении, даже если сами активы оставлены в доверительном управлении для их внуков.

## Кто может подавать форму 706-GS(D): Налоговая декларация о передаче без учета поколения для распределений?

Налогоплательщики, получающие налогооблагаемые выплаты от траста, должны использовать Форму 706-GS(D): Налоговая декларация о передаче без учета генерации для выплат. В частности, форма должна быть заполнена распределителем пропуска, то есть лицом, получающим выгоду от траста пропуска поколения.

Налогоплательщики должны подать эту форму в период с 1 января до крайнего срока подачи налоговых деклараций года, следующего за распределением денег из траста. Если форма не может быть подана вовремя, можно запросить продление, заполнив [форму 7](/filing-extension) [004](/filing-extension).

> Если вы не можете подать форму вовремя, вы можете запросить автоматическое продление на шесть месяцев, заполнив форму 7004.

>

## Как подать форму 706-GS(D): Налоговая декларация о передаче без учета генерации для распределений

В части I формы требуется общая информация, включая имя и номер социального страхования распространителя, имя и адрес лица, подающего декларацию (если он отличается от получателя), а также идентификационный номер работодателя доверительного распространителя. В Части II заявитель описывает характер дистрибутивов. Затем следует Часть III, где рассчитывается сумма налога.

## Форма 706-GS(D) по сравнению с формой 706-GS(D-1) по сравнению с формой 706

Форма 706-GS(D-1), которую подают доверительные управляющие, когда траст передает им активы, является частью более крупной налоговой [формы 706](/form-706). Эта форма используется для подачи налогов на недвижимость в целом. Раздел о переводах без учета поколений включен в форму 706-GS(D).

## Особенности

- Любой, кто получает налогооблагаемое распределение от траста, должен использовать эту форму.

- Форма 706-GS(D) используется для расчета налогов, причитающихся с трастовых распределений, облагаемых налогом на передачу с пропуском поколения.

- Эта форма должна быть подана в период с 1 января до крайнего срока подачи в году, следующем за распределением денег из траста.
