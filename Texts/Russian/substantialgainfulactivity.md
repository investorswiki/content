# Существенная прибыльная деятельность (SGA)
## Что такое существенная прибыльная деятельность (SGA)?

Существенная доходная деятельность (SGA) представляет собой месячную пороговую заработную плату, используемую [Администрацией социального обеспечения США](/ssa) (SSA) для предоставления лицам права на получение пособий по инвалидности. SSA ежегодно обновляет сумму в долларах, чтобы отразить [инфляцию](/inflation) (или рост цен) и, как правило, поддерживает более высокий порог для официально слепых.

## Понимание существенной прибыльной деятельности (SGA)

Существенная доходная деятельность представляет собой сумму ежемесячного дохода, ниже которой человек получает право на получение пособия по нетрудоспособности в рамках системы [социального обеспечения](/socialsecurity). SSA использует размер SGA в качестве ключевого фактора, определяющего, считает ли он человека инвалидом для целей своих программ.

Лица, которые не могут заниматься деятельностью, приносящей им больше месячного порога SGA, имеют право на получение пособия по инвалидности. SSA не считает лиц, способных заниматься деятельностью, приносящей доход больше порогового значения, инвалидами для целей своих программ.

Пороговые суммы, используемые для расчета суммы SGA, различаются для слепых и не слепых людей. Те, кто соответствует установленному SSA определению слепоты, имеют более высокий порог SGA, чем те, кто этого не делает, что означает, что слепые люди обычно могут зарабатывать больше в месяц, чем не слепые, прежде чем они лишатся права на получение пособия по инвалидности.

### Месячный лимит дохода

SSA установило сумму SGA на 2022 год для не слепых в размере 1350 долларов в месяц. Это означает, что любое лицо, способное работать по найму с заработком 1350 долларов и более в месяц, не будет соответствовать критериям права на получение пособия по инвалидности в 2021 году. SSA установило порог SGA для слепых в размере 2260 долларов в 2022 году.

### Корректировка стоимости жизни (COLA)

получатели социального обеспечения и дополнительного дохода (SSI) получат 5,9% [корректировки на прожиточный минимум](/cola) (COLA). COLA добавляет к ежемесячным пособиям корректировку на рост цен, называемый [инфляцией](/inflation). Хотя ваш конкретный ежемесячный платеж будет зависеть от вашей ситуации, в среднем работник-инвалид, которому платили 1282 доллара в месяц до COLA, будет получать 1358 долларов в месяц в 2022 году после COLA.

## SSDI против SSI

SSA предоставляет выплаты по нетрудоспособности физическим лицам в рамках двух программ. Социальное страхование по нетрудоспособности (SSDI) покрывает лиц, которые внесли взносы в программу социального обеспечения за счет отчислений из заработной платы.

[Supplemental Security Income](/ssi) (SSI) выплачивает пособия лицам с ограниченными возможностями, которые соответствуют определенному набору финансовых требований, независимо от того, работали они ранее или нет. Для не-слепых SSA использует пороговое значение SGA для определения права на льготы по любой из программ.

Однако для официально слепых SSA использует SGA только для определения права на получение выплат в рамках программы SSDI. Для слепых, которые получают пособия по инвалидности в рамках программы SSI, SSA не использует пороговые значения SGA при первоначальном определении права на получение пособия.

Как только SSA утверждает пособия по нетрудоспособности для человека, оно позволяет этому лицу продолжать получать пособия в течение короткого периода после того, как это лицо сможет снова выйти на работу и ежемесячно зарабатывать больше, чем сумма SGA. Это дает инвалидам стимул к поиску оплачиваемой работы и, если возможно, к повторному выходу на рынок труда в другом качестве на длительный срок.

## Особенности

- Если человек зарабатывает меньше порога SGA, он имеет право на пособие по инвалидности.

- Существенная доходная деятельность (SGA) — это уровень заработной платы, который может быть получен, что позволяет человеку претендовать на пособие социального обеспечения по инвалидности.

- Сумма SGA на 2022 год для незрячих составляет 1350 долларов в месяц и 2260 долларов в месяц для слепых.

- Существует два типа пособий по социальному обеспечению — социальное страхование по инвалидности (SSDI) и дополнительный доход по социальному обеспечению (SSI).

- SSDI платит лицам, которые внесли взносы в программу социального обеспечения, посредством отчислений из заработной платы, в то время как SSI платит лицам, которые соответствуют финансовым требованиям, независимо от того, были ли они трудоустроены или нет.
