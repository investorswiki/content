# Только для исполнения
## Что такое только выполнение?

Только исполнение — это торговая услуга, которая ограничивается только выполнением сделок, без получения клиентом каких-либо рекомендаций относительно достоинств или рисков инвестиций или их пригодности.

Услуги, предназначенные только для выполнения, также называемые только агентскими услугами, в основном предоставляются через Интернет или по телефону и, как правило, стоят дешевле, чем более надежные предложения. Поскольку клиентам не предоставляются консультации, услуги только по исполнению больше всего подходят для опытных инвесторов, которые осведомлены о рисках и выгодах различных типов инвестиций.

## Понимание только выполнения

Чтобы работать более эффективно, сервисы только для исполнения используют [номинальные](/nominee) счета. Это означает, что инвестиции клиента регистрируются на имя брокерской компании, предоставляющей услуги только по исполнению, хотя клиент остается бенефициарным владельцем. В то время как брокерская компания занимается всей документацией, связанной с покупкой и продажей ценных бумаг, а также сбором дивидендов и зачислением их на счет клиента, она также будет направлять информацию, которая требует действий акционера, непосредственно клиенту.

Поскольку эти фирмы предлагают только исполнение сделок, улучшения в технологии [торговых платформ](/trading-platform) привели к резкому снижению затрат на такие услуги. Для розничных инвесторов эти услуги, предназначенные только для исполнения, предлагают очень экономичную торговую платформу, и многие из них предлагают торговлю акциями за комиссию, выражающуюся однозначным числом в долларах.

## Исследования, предлагаемые сервисами только для выполнения

Хотя они не дают советов или рекомендаций, предлагаемых [брокерами с полным спектром услуг](/fullservicebroker),. многие службы, предоставляющие только услуги по исполнению, которые являются дочерними компаниями более крупных финансовых учреждений, действительно предоставляют своим клиентам некоторые [инвестиционные исследования , проводимые их материнскими компаниями.](/investment-analysis) Таким образом, хотя этим клиентам в конечном итоге приходится принимать инвестиционное решение самостоятельно, они получают огромное количество информации, такой как рыночные данные в режиме реального времени, торговые идеи и анализ тенденций, чтобы помочь им в процессе принятия решения.

Брокерская компания, выполняющая сделку, не будет иметь мнения о том, следует ли заключать сделку, поэтому использование фирмы, занимающейся только исполнением, создает дополнительный уровень риска для каждого трейдера или инвестора.

## Стоимость услуг только для выполнения

Стоимость услуг, связанных только с исполнением, составляет лишь часть стоимости заключения сделки у [брокера с полным спектром услуг](/fullservicebroker) или даже у большинства дисконтных брокеров. Поэтому он в первую очередь подходит для опытных трейдеров, [дневных трейдеров](/daytrader) и технических трейдеров, которые могут совершать множество сделок в любой день. Тем не менее, трейдеры должны знать о рисках чрезмерной торговли, которые могут привести к потенциально большим убыткам, которые более чем компенсируют низкую стоимость торговли через сервис только для исполнения.

Используя эти услуги, инвесторы могут совершать сделки с различными видами ценных бумаг, включая акции, облигации, взаимные фонды, биржевые фонды (ETF) и иностранную валюту.

## Особенности

- Таким образом, «только исполнение» означает, что вспомогательные услуги, предлагаемые брокерскими компаниями с полным спектром услуг, такие как внутренние исследования или финансовые консультации и планирование, отсутствуют.

- Из-за этого торговые платформы только для исполнения, как правило, очень дешевы и экономичны для трейдеров и инвесторов, которые имеют представление о том, что они хотят делать на рынке.

- «Только исполнение» — это брокерская модель, которая включает в себя только выполнение и отчетность по заказам, переданным им клиентами.
