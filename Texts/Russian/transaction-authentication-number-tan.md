# Номер аутентификации транзакции (TAN)
## Что такое номер аутентификации транзакции (TAN)

Номер аутентификации транзакции — это одноразовый код, используемый при обработке онлайн-транзакций. Номер авторизации транзакции (TAN) представляет собой дополнительный уровень безопасности помимо пароля для безопасного входа в учетную запись или проведения транзакции.

## РАЗБИВКА Номер аутентификации транзакции (TAN)

Номера аутентификации транзакций (TAN) обеспечивают дополнительную безопасность. Продавцы и компании, выпускающие платежные карты, заинтересованы в повышении безопасности транзакций, поскольку более высокий уровень безопасности снижает вероятность мошенничества. Такие организации, как Совет по стандартам безопасности индустрии платежных карт, создают стандарты, которым должны следовать [эквайеры платежных карт](/acquirer) и процессинговые компании, чтобы шифровать информацию о карте в точке взаимодействия (POI), а затем расшифровывать и обрабатывать транзакцию.

Номера авторизации транзакций — это один из способов, с помощью которого финансовые учреждения могут снизить вероятность мошенничества. Это одноразовые номера, обеспечивающие двухфакторную аутентификацию транзакции. Первый уровень аутентификации может включать в себя [личный идентификационный номер](/personal-identification-number) (PIN) или пароль для доступа к учетной записи, тогда как вторым уровнем аутентификации может быть TAN.

Финансовые учреждения обычно предоставляют список паролей или парольных фраз, которые можно использовать для аутентификации транзакции, при этом каждый TAN действителен только для однократного использования. Финансовое учреждение, предоставляющее список TAN, ведет базу данных, в которой оно связывает каждый TAN с конкретным пользователем.

TAN чаще всего используются для онлайн-проверки транзакций. Когда физическое или юридическое лицо начинает транзакцию, ему может быть предоставлен TAN по электронной почте, в текстовом SMS-сообщении или другим способом. Способ доставки обычно аутентифицируется заранее посредством предыдущего взаимодействия, например, когда банк отправляет текстовое сообщение, подтверждающее, что конкретный номер телефона связан с учетной записью. Когда транзакция проводится, пользователь получит сообщение с кодом TAN и должен будет ввести этот код в поле веб-формы. Если код правильно подобран, транзакция будет обработана.

## Номера аутентификации транзакций и двухфакторная аутентификация

Поскольку американцы перенесли большую часть своей деятельности в онлайн, использование номеров аутентификации транзакций распространилось из финансовых учреждений на многие другие сферы жизни. Например, рекомендуется, чтобы пользователи электронной почты входили в свои учетные записи электронной почты, используя [двухфакторную аутентификацию](/twofactor-authentication-2fa),. при которой необходимо ввести свой пароль вместе с номером аутентификации транзакции. Обычно они сохраняются пользователем в списке одноразовых кодов или номер извлекается с помощью текстового сообщения, электронной почты или телефонного звонка.
