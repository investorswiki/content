# Бежевая книга
Отчет Федеральной резервной системы «Бежевая книга» обобщает неподтвержденную информацию, собранную окружными федеральными резервными банками от ведущих бизнесменов, экономистов, экспертов по рынку и из других источников. Используется на заседаниях Федерального комитета по открытым рынкам (подразделение ФРС, определяющее денежно-кредитную политику).

**Источник**: Один из 12 окружных федеральных резервных банков на ротационной основе.

**Частота**: восемь раз в год.

**Когда выйдет?** За две среды до каждого заседания ФРС в 14:00 по восточному поясному времени.

**Значение рынка**: Некоторое. Используется в сочетании с последними экономическими показателями для оценки силы экономики. Время от времени двигает рынки, но «Бежевая книга» не является комментарием взглядов членов ФРС.

## Особенности

- Данные в отчете собраны 12 районными банками об экономических условиях в их соответствующих региональных экономиках.

- Бежевая книга представляет собой качественный отчет, подготовленный и опубликованный Федеральной резервной системой.

- FOMC использует информацию из Бежевой книги для принятия важных решений о национальной экономике.

- Отчет публикуется восемь раз в год, за две недели до каждого заседания Федерального комитета по открытым рынкам (FOMC).
