# Служба множественного листинга (MLS)
## Что такое MLS?

MLS расшифровывается как служба множественных списков, которая представляет собой базу данных, которую агенты по недвижимости используют для обмена своими списками с другими агентами и для поиска домов для продажи. Каждая база данных MLS обслуживает определенные регионы США и доступна только для агентов, которые платят за членство.

## Более глубокое определение

Концепция службы множественных списков восходит к тому времени, когда агенты по недвижимости периодически встречались, чтобы поделиться информацией о свойствах, которые они выставили на продажу. Сегодня агенты обмениваются этой информацией через компьютер. Они платят членские взносы за доступ к базе данных MLS, которая охватывает регион, в котором они работают, и эти взносы поддерживают программу.

MLS является полезным инструментом для агентов по недвижимости. Когда они выставляют недвижимость на продажу и добавляют ее в MLS, все агенты в этом регионе, имеющие доступ к системе, имеют возможность просмотреть список и показать его потенциальным покупателям. Это дает продавцам больше возможностей на местном рынке недвижимости. Агенты покупателей, использующие MLS, имеют больше возможностей для демонстрации своим клиентам.

В дополнение к расширению возможностей для продавцов и покупателей MLS поощряет сотрудничество между агентами по недвижимости и позволяет небольшим агентствам недвижимости конкурировать с крупными франчайзинговыми фирмами. Вместо того, чтобы продавать только списки, которые есть в офисе, они имеют доступ к запасам других агентов в этом районе. В результате все покупатели и продавцы имеют одинаковый доступ к недвижимости и покупателям, независимо от размера офиса недвижимости.

## Пример МЛС

Сэм — агент по недвижимости, который ищет дома для нескольких потенциальных покупателей. Сэм платит ежемесячную плату за использование MLS, к которой он часто обращается для поиска недвижимости для продажи. Он вводит параметры поиска в базу данных, такие как количество спален, цена и районы, исходя из того, что ищут его клиенты. Затем он печатает отчет для каждой собственности, чтобы поделиться с покупателями. Покупатели используют эту информацию, чтобы сузить свои списки свойств и решить, на какие из них обратить внимание.

## Особенности

- Служба множественных списков (MLS) представляет собой базу данных, созданную сотрудничающими брокерами по недвижимости для предоставления данных о недвижимости для продажи.

- MLS позволяет брокерам просматривать списки объектов недвижимости, выставленных на продажу, с целью установления контактов между покупателями жилья и продавцами.

- MLS помогает агентам по недвижимости и брокерам связываться, объединяя и обмениваясь информацией при разделении комиссий.

## ЧАСТО ЗАДАВАЕМЫЕ ВОПРОСЫ

### Что такое номер MLS?

Номер MLS — это, по сути, серийный номер для каждого объекта недвижимости на рынке. Он был создан, чтобы упростить различение свойств и быстрый поиск свойств. Напротив, номер NMLS, хотя и похож на операции с недвижимостью, выдается Национальной системой лицензирования нескольких штатов и постоянно присваивается каждому лицу, работающему специалистом по [ипотечным кредитам](/mortgage_originator).

### Нужна ли мне лицензия на недвижимость для доступа к службе множественных списков?

Как правило, для просмотра списков в местной службе множественных списков (MLS) в качестве покупателя или продавца вам необходимо получить доступ через агента по недвижимости. Чтобы фактически размещать недвижимость в MLS, вам необходимо иметь лицензию на недвижимость. Если вы не хотите получать лицензию на недвижимость и не хотите работать с полноправным агентом продавца, в некоторых районах есть несколько услуг по фиксированной плате за листинг, которые сделают список вашей собственности для вас.

### Какие сборы связаны с MLS?

Существуют сотни организаций MLS по всей стране, и все они взимают различные сборы непосредственно с брокеров по недвижимости, которые принадлежат к их сети MLS. MLS не взимает плату напрямую с населения.
