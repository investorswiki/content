# Передача акционерной стоимости (SVT)
## Что такое передача акционерной стоимости (SVT)?

Передача акционерной стоимости (SVT) — это показатель, предназначенный для акционеров в определении того, какая компенсация в виде акций должна быть присуждена сотрудникам и руководителям [публичных компаний](/publiccompany). Передача акционерной стоимости рассчитывается как общая стоимость долевых грантов, деленная на рыночную капитализацию компании. Это дает процент, до которого существующие акционеры будут разводнены в соответствии с данным планом компенсации акционерного капитала.

## Понимание передачи акционерной стоимости (SVT)

Метрика передачи акционерной стоимости была создана Institutional Shareholder Services (ISS), исследовательской фирмой, которая консультирует акционеров о том, как голосовать по предложениям акционеров. ISS рассчитывает передачу акционерной стоимости для ведущих компаний в каждой отрасли и принимает решение о максимальной сумме «потолка», которую любая компания должна заплатить при передаче акционерной стоимости, чтобы работать хорошо. Затем ISS обычно советует инвесторам голосовать против любого предложения о компенсации акционерным капиталом, которое превысит лимит передачи акционерной стоимости.

Согласно Institutional Shareholder Services, передача акционерной стоимости относится к оценке стоимости, которую компания будет передавать своим сотрудникам и директорам через определенные программы компенсации, основанной на акциях, которая измеряется на определенную дату на основе стандартного набора исходных данных. Собственная модель вознаграждения ISS рассчитывает контрольный показатель передачи акционерной стоимости для каждой компании — на основе ее рыночной капитализации, отрасли и соответствующих показателей эффективности по сравнению с аналогами — который используется при оценке передачи акционерной стоимости компании.

Чтобы найти оценку передачи акционерной стоимости, в расчетах Службы институциональных акционеров используется комбинация сторонних данных для модели ценообразования опционов, а также данные по конкретной компании (включая непогашенные гранты и акции, оставшиеся для будущих грантов), обычно сообщаемые в годовом [отчете .](/10-k) [K](/10-k) или по доверенности.

## Особенности

- Чтобы получить оценку SVT, в расчетах ISS используется комбинация данных третьих сторон и данных компании, представленных в налоговых формах.

- Передача акционерной стоимости рассчитывается как общая стоимость долевых грантов, деленная на рыночную капитализацию компании.

- Передача акционерной стоимости (SVT) — это показатель, предназначенный для акционеров в определении того, какая компенсация в виде акций должна быть присуждена сотрудникам и руководителям публичных компаний.

- Метрика была создана Institutional Shareholder Services (ISS), исследовательской фирмой, которая консультирует акционеров о том, как голосовать по предложениям акционеров.

- ISS рассчитывает «потолок» для SVT по отраслям, указывая компаниям, какими должны быть их предложения по компенсации акционерным капиталом.
