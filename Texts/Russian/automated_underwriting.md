# Автоматизированный андеррайтинг
## Что такое автоматизированный андеррайтинг?

Автоматизированный андеррайтинг — это управляемый [технологиями процесс андеррайтинга,](/underwriting) который обеспечивает компьютерное решение по кредиту. Кредитная индустрия широко переходит на использование новых платформ для андеррайтинга кредитов, основанных на технологиях, чтобы сократить время обработки всех типов [кредитов](/loan).

## Объяснение автоматизированного андеррайтинга

Автоматизированный андеррайтинг используется в различных целях на рынке кредитования. Это фактор, усиливающий конкуренцию и, в частности, конкуренцию между новыми онлайн-кредиторами и традиционными банками. В то же время это также создает большие возможности для партнерских отношений в сфере [финансовых технологий](/fintech),. которые включают революцию в автоматизированных услугах по андеррайтингу и кредитованию.

Автоматизированный андеррайтинг может использоваться во всех типах кредитов. Он в основном используется с обычными кредитами, которые включают стандартную процедуру андеррайтинга и [базовый график амортизации](/amortization_schedule) для платежей в рассрочку. Многие автоматизированные приложения для андеррайтинга доступны для личных кредитов через онлайн-кредиторов, таких как LendingClub и Prosper, однако крупные банки также интегрируют автоматизированные платформы для андеррайтинга. Как правило, кредиторы могут предлагать автоматизированные заявки на кредит для кредитных карт, личных кредитов, автокредитов и ипотечных кредитов.

## Автоматизированные приложения для андеррайтинга

Автоматизированный андеррайтинг исторически использовался для андеррайтинга кредитных карт, однако он становится все более популярным среди обычных кредитов. Заявки на получение ссуды могут быть структурированы таким образом, чтобы получать основную информацию о заявке, включая адреса, номера социального страхования и сведения о доходах. В партнерстве с поставщиками информации автоматизированные платформы андеррайтинга затем используют базовую информацию о заявке на получение кредита для получения соответствующих данных, таких как [кредитная история заемщика](/credit-history). Оттуда автоматизированная платформа может обрабатывать информацию о заемщике с помощью запрограммированного процесса андеррайтинга, который мгновенно принимает решение о кредите.

Автоматизированный андеррайтинг делает первую фазу процесса андеррайтинга намного более эффективной. Он может предоставлять мгновенные результаты, которые обычно могут занимать до 60 дней при ручной обработке. Он также имеет возможность помечать заявки и направлять их на андеррайтинг вручную для определенных проверок на заключительных этапах процесса кредитования. В случае с обычными кредитами обычно требуется взаимодействие человека для проверки некоторых входных данных, таких как доход и активы, чтобы закрыть сделку.

## Кредитные операционные системы

Кредитные операционные системы на рынке кредитования также быстро развиваются, чтобы обслуживать все аспекты кредитного процесса. В дополнение к автоматизированному андеррайтингу можно создать кредитные операционные системы для создания графиков амортизации и платежей в рассрочку, предоставления автоматизированного портала для онлайн-платежей и предоставления сервисных уведомлений кредитору о просроченных платежах.

Существует множество платформ кредитных операционных систем от различных поставщиков финансовых технологий. Freddie Mac поддерживает и продает большой автоматизированный механизм андеррайтинга, известный как Loan Prospector, а у Fannie Mae есть автоматизированный механизм андеррайтинга, известный как Desktop Underwriter. Как правило, кредитные операционные системы могут быть построены с помощью различных интерфейсов прикладного программирования, которые позволяют использовать плагины из множества технологий для создания индивидуальной системы.

## Особенности

- Андеррайтинг — это процесс оценки риска, связанного с финансовой операцией, такой как выпуск облигаций, банковский кредит или страховой полис.

- Критики, однако, утверждают, что использование программного обеспечения для андеррайтинга может усугубить неравенство на рынке кредитования.

- Автоматизированный андеррайтинг использует алгоритмы вместо людей, чтобы принимать решения по андеррайтингу быстрее и менее подвержены ошибкам.
