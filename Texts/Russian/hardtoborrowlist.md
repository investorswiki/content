# Список труднодоступных
## Что такое список труднодоступных?

Список труднодоступных активов — это инвентарная запись, используемая брокерскими конторами для указания того, какие акции трудно взять взаймы для сделок с [короткими продажами .](/shortsale) Список труднодоступных активов брокерской фирмы представляет собой актуальный каталог акций, которые нелегко взять взаймы для использования в качестве короткой продажи.

Список труднодоступных можно сравнить со списком [легкодоступных брокерских компаний](/easytoborrowlist).

## Понимание списка трудных для заимствования

Короткая продажа акций основана на представлении о том, что отдельный трейдер или инвестор, желающий получить прибыль от снижения цены акции, может занять акции этой акции у брокера.

[брокерских](/brokerage-company) компаний есть множество способов предоставить доступ к акциям, которые можно продать без покрытия. Однако, независимо от их методов, существует конечное количество акций, доступных для продажи. Как только количество доступных акций будет близко к исчерпанию, брокер опубликует какую-либо нотацию на своей платформе. Это предупреждает владельцев счетов о том, что если они попытаются продать эту ценную бумагу без покрытия, их торговый ордер может быть отклонен.

> Дефицит предложения — не единственная причина, по которой ценные бумаги могут оказаться в списке труднодоступных. Он также может быть включен из-за высокой [волатильности](/volatility) или чего-то еще.

>

Чтобы войти в короткую продажу, клиент брокерской компании должен сначала одолжить акции у своего брокера. Для предоставления акций брокер может использовать собственные запасы или заимствовать средства с [маржинального счета](/marginaccount) другого клиента или другой брокерской фирмы. Заемщик (т. е. продавец коротких позиций) должен платить проценты и комиссию за заимствованные акции. Те, кто находится в списке труднодоступных, могут иметь более высокую [комиссию за кредит](/stock-loan-fee) под акции из-за меньшего предложения.

Инвесторы, которые вступают в короткие сделки, пытаются получить прибыль на падающем рынке. Например, [инвестор](/investor) может подумать, что акции Apple, скорее всего, упадут в цене. Инвестор может продать акции без покрытия и, если цена упадет, как они ожидают, выкупить их обратно с прибылью. Однако, если акции растут, инвестор теряет деньги.

## Требования к списку труднодоступных объектов

Брокерские фирмы ежедневно обновляют свои списки труднодоступных средств. Брокер должен быть в состоянии предоставить или определить местонахождение акций для кредита своему клиенту перед выполнением сделки короткой продажи клиента.

[Постановление SHO](/regsho),. введенное в действие 3 января 2005 г., содержит условие «найти», которое требует от брокеров обоснованной уверенности в том, что акции, предназначенные для продажи на понижение, могут быть заимствованы и доставлены продавцу. Регулирование предназначено для предотвращения открытых коротких продаж, практики, когда инвестор размещает короткие продажи, не владея акциями.

## Список трудно заимствованных против списка легко заимствованных

Список труднодоступных является противоположностью списка легкодоступных, который представляет собой инвентарный список ценных бумаг, доступных для коротких продаж. В целом инвестор может предположить, что ценные бумаги, не включенные в список труднодоступных, будут доступны для продажи без покрытия. В то время как список труднодоступных кредитов брокерской фирмы обычно является внутренним списком, который не предоставляется клиентам, клиенты фирмы обычно имеют доступ к списку легкодоступных кредитов.

Брокерским клиентам, возможно, придется платить комиссию, которую трудно получить, при определенных коротких продажах. Как правило, стоимость заимствования акций из списка труднодоступных для заимствования выше, чем для акций из списка легкодоступных. У крупных брокерских фирм обычно есть [отдел кредитования ценных бумаг](/securitieslending),. который помогает найти акции, которые трудно взять взаймы. Отдел кредитования ценных бумаг брокерской компании также предоставляет ценные бумаги другим фирмам.

## Особенности

- Короткие продавцы полагаются на брокеров, чтобы иметь акции доступны для заимствования.

- Если у брокера очень мало доступных акций, то эта акция помещается в список труднодоступных.

- Акции из списка труднодоступных не могут продаваться без покрытия или иметь более высокие комиссионные за ссуду.
