# Коэффициент покрытия срока кредита — LLCR
## Что такое коэффициент покрытия срока кредита (LLCR)?

Коэффициент покрытия срока кредита (LLCR) — это финансовый коэффициент, используемый для оценки платежеспособности фирмы или способности компании-заемщика погасить непогашенный кредит. LLCR рассчитывается путем деления [чистой приведенной стоимости (NPV)](/npv) денег, доступных для [погашения долга](/repayment),. на сумму непогашенного долга.

LLCR аналогичен [коэффициенту покрытия долга](/dscr) (DSCR), но чаще используется в проектном финансировании из-за его долгосрочного характера. DSCR фиксирует один момент времени, тогда как LLCR охватывает весь период [кредита](/loan).

## Как рассчитать коэффициент покрытия срока кредита

LLCR можно рассчитать с помощью приведенной выше формулы или с помощью более быстрого способа: деления чистой приведенной стоимости свободных денежных потоков проекта на текущую стоимость непогашенной задолженности.

В этом расчете средневзвешенная стоимость долга является ставкой дисконтирования для расчета чистой приведенной стоимости, а «денежные потоки» проекта — это более конкретно [денежные потоки, доступные для обслуживания долга](/cads) (CFADS).

## О чем говорит коэффициент покрытия срока кредита?

LLCR – коэффициент [платежеспособности](/solvencyratio). Коэффициент покрытия срока кредита — это показатель того, сколько раз денежные потоки проекта могут погасить непогашенную задолженность в течение срока действия кредита. Коэффициент 1,0x означает, что LLCR находится на уровне безубыточности. Чем выше коэффициент, тем меньше потенциальный риск для кредитора.

[кредитору](/lender) иногда требуется резервный счет для обслуживания долга . В таком случае числитель LLCR будет включать баланс резервного счета. Соглашения о проектном финансировании неизменно содержат ковенанты, предусматривающие уровни LLCR.

## Разница между LLCR и DSCR

В корпоративных финансах [коэффициент покрытия долга](/dscr) (DSCR) является мерой денежного потока, доступного для погашения текущих долговых обязательств. Коэффициент показывает чистую операционную прибыль как множитель долговых обязательств со сроком погашения в течение одного года, включая проценты, основную сумму, фонд погашения и арендные платежи. Однако DSCR фиксирует только один момент времени, в то время как LLCR допускает несколько периодов времени, что больше подходит для понимания ликвидности, доступной для кредитов со средним и долгосрочным горизонтом.

LLCR используется аналитиками для оценки жизнеспособности данной суммы долга и, следовательно, для оценки профиля риска и связанных с этим затрат. У него менее непосредственное объяснение по сравнению с DSCR, но когда значение LLCR больше единицы, это обычно является сильным утешением для инвесторов.

## Ограничения LLCR

Одним из ограничений LLCR является то, что он не улавливает слабые периоды, потому что в основном представляет собой дисконтированное среднее значение, которое может сгладить неровности. По этой причине, если у проекта есть устойчивые денежные потоки с историей погашения кредита, хорошее эмпирическое правило состоит в том, что LLCR должен быть примерно равен среднему коэффициенту покрытия обслуживания долга.

## Особенности

- Коэффициент покрытия срока кредита (LLCR) - это финансовый коэффициент, используемый для оценки платежеспособности фирмы или способности компании-заемщика погасить непогашенный кредит.

- Чем выше коэффициент, тем меньше потенциальный риск для кредитора.

- Коэффициент покрытия срока кредита является мерой того, сколько раз денежные потоки проекта могут погасить непогашенную задолженность в течение срока действия кредита.
