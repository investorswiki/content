# Считать
## Что такое счет?

Термин «подсчет» относится к форме [технического анализа , в](/technicalanalysis) которой используются [графики «крестики-нолики» (P&F)](/pointandfigurechart) для оценки вертикального движения цен на акции. Графики «крестики-нолики», разработанные для использования в долгосрочном инвестиционном анализе, считаются одним из самых простых способов для трейдеров определить точки входа и выхода. Трейдеры, использующие подсчетный анализ, строят графики роста и снижения цен, которые используются для точного определения целевых цен.

## Понимание количества

Технический анализ — это тип торговой дисциплины, в которой диаграммы и графики используются для оценки прошлых [тенденций](/trend) по ценным бумагам, таких как цена и [объем торгов](/volumeoftrade),. чтобы определить будущие результаты. Трейдеры используют методы технического анализа, чтобы найти точки [входа](/entry-point) и [выхода](/exit-point),. которые оптимизируют прибыль. Некоторые трейдеры используют графики «крестики-нолики» без учета времени. Этот метод называется счетным анализом.

Счетный анализ отображает Xs для представления роста цен и Os для снижения цен на своем графике. Аналитики основывают расчеты на исторических боковых движениях цены и используют их для определения вероятности того, что [целевая цена](/pricetarget) может быть достигнута. Xs и Os анализа счета используются с традиционной шкалой и заранее определенной суммой разворота. Трейдеры используют это, чтобы определить, являются ли определенные позиции прибыльными. Инвесторы могут просмотреть последовательность колебаний цен, чтобы оценить, как цены будут двигаться в будущем. Существует несколько методов подсчета, например, метод подсчета прорывов, который используется для нахождения цели бычьей цены, должен использоваться с активным сигналом покупки P&F.

> Как инвестор, вы можете просмотреть последовательность колебаний цен, чтобы оценить их вероятное изменение в будущем.

>

## Типы анализа подсчета

Как отмечалось выше, существуют различные методы анализа подсчета, такие как прорыв, разворот и горизонтальный подсчет.

### Анализ количества прорывов

Метод подсчета [прорывов](/breakout) используется для определения бычьей ценовой цели и должен использоваться с активным сигналом покупки P&F. Этот метод состоит из четырех шагов.

Во-первых, наиболее активный сигнал на продажу, известный как пробой [двойного дна](/doublebottom),. должен быть обнаружен на графике P&F справа налево. Второй шаг включает работу справа от этого сигнала, чтобы найти следующий сигнал на покупку или прорыв [двойной вершины .](/doubletop) Столбец, создающий этот сигнал, является ключевым, поскольку он становится столбцом меры. Затем необходимо рассчитать высоту столбца меры и умножить ее на величину разворота поля. Наконец, итог этого расчета должен быть добавлен к нижней части столбца слева от столбца показателей.

### Анализ подсчета разворотов

Метод подсчета [разворотов](/reversal) можно использовать для нахождения [бычьих](/bull) и [медвежьих](/bear) ценовых целей. Для поиска бычьих цен используются три шага. Счетчик разворота должен использоваться с активным сигналом покупки P&F. Во-первых, работая слева направо, необходимо найти самый последний сигнал продажи P&F. Столбец X рядом с сигналом на продажу становится столбцом показателей. Далее необходимо рассчитать высоту столбца и умножить ее на [величину разворота коробки](/reversalamount). Затем итог должен быть добавлен к нижней части столбца слева от столбца меры.

### Анализ горизонтального подсчета

Для метода горизонтального подсчета на графике P&F должна сформироваться модель [скопления или разворота.](/congestion) Шаблон перегрузки должен иметь ширину не менее пяти столбцов и должен иметь столбец, который нарушает перегрузку. Столбцы в этом шаблоне должны быть подсчитаны. Это ширина. После того, как столбец прорыва появится на графике P&F, аналитики могут умножить ширину на [размер поля](/boxsize) и величину разворота, чтобы оценить расширение цены. Расширение добавляется к минимуму модели для цели цены.

## Особенности

- Существует несколько методов подсчета, таких как метод подсчета прорывов, метод подсчета разворотов и метод подсчета по горизонтали.

- Анализ подсчета использует X для представления увеличения цены и O для снижения цены, чтобы точно определить целевые цены.

- Расчеты количества основаны на исторических боковых движениях цены, которые используются для определения вероятности достижения ценовой цели.

- Count — это форма технического анализа, в которой используются графики «крестики-нолики» для оценки вертикального движения цен на акции.
