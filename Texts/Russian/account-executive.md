# Делопроизводитель
## Кто такой менеджер по работе с клиентами?

Менеджер по работе с клиентами — это сотрудник, который несет основную повседневную ответственность за текущие деловые отношения с клиентом.

Должность менеджера по работе с клиентами наиболее распространена в сфере рекламы и связей с общественностью, а также в сфере финансовых услуг. Технологические компании, предоставляющие услуги поддержки аппаратного и программного обеспечения, назначают менеджеров по работе с клиентами для крупных клиентов.

## Понимание менеджера по работе с клиентами

Менеджер по работе с клиентами является основным контактным лицом между продавцом и покупателем. Менеджер по работе с клиентами обычно находится в центре внимания с самого начала деловых отношений. Предложив сделку и договорившись о контракте, менеджер по работе с клиентами несет ответственность за выполнение условий контракта и удовлетворение клиента.

Менеджер по работе с клиентами также может связаться с клиентом, чтобы представить новые продукты и услуги. Это может включать в себя организацию выезда технического, дизайнерского и другого вспомогательного персонала к клиенту для проведения демонстраций и помощи с коммерческим предложением.

### Применяются правила

Менеджерам по работе с клиентами рекомендуется регулярно поддерживать связь с клиентами.

В финансовом мире ожидается, что руководители счетов будут следовать этическим нормам регулирующих органов. В финансовом мире они включают Правило 3220 Управления по регулированию финансовой индустрии ( [FINRA](/finra) ), применимое к [брокерам-дилерам](/broker-dealer) и [инвестиционным консультантам](/investmentadvisor),. которое ограничивает выплату подарков и чаевых.

### Обязанности менеджера по работе с клиентами

Обычно ожидается, что менеджеры по работе с клиентами приведут новый бизнес, и им часто даются годовые квоты. Они также могут быть назначены клиентами компании.

В некоторых случаях менеджеру по работе с клиентами можно было назначить только одного очень важного клиента, которому предстояло пройти по красной ковровой дорожке. У других менеджеров по работе с клиентами есть несколько клиентов, которым нужно расставить приоритеты с точки зрения их важности для фирмы. Один менеджер по работе с клиентами может управлять рекламной кампанией компании по производству продуктов питания и напитков и сети быстрого питания.

> В финансовом мире менеджеры по работе с клиентами должны следовать рекомендациям, установленным FINRA.

>

## Где они найдены

В различных отраслях менеджеры по работе с клиентами могут заниматься аппаратным и программным обеспечением для кибербезопасности банка, фармацевтическими потребностями больничной группы, потребностями частного клиента в управлении активами или требованиями хедж-фонда к [основным брокерским услугам.](/primebrokerage)

Компенсация для менеджера по работе с клиентами обычно представляет собой базовый оклад с комиссией с продаж и бонусами за достижение или превышение целей продаж. По данным Glassdoor.com, по состоянию на май 2021 года средняя базовая заработная плата за работу в США составляла 57 504 доллара.

Менеджеры по работе с клиентами обладают широким спектром навыков и глубоким знанием отрасли. Эти навыки включают следующее:

- Навыки продаж для привлечения новых клиентов и представления существующих клиентов о новых продуктах и услугах

- Навыки управления проектами, чтобы быть в курсе работы, выполняемой для клиента

- Навыки управления персоналом для работы с различными отделами внутри компании по мере необходимости

- Исключительные коммуникативные навыки для поддержания отношений с клиентами

## Особенности

- Компенсация обычно включает в себя базовый оклад, комиссионные с продаж и бонусы за превышение плановых показателей.

- Менеджеры по работе с клиентами работают во многих отраслях, но чаще всего их можно найти в сфере рекламы, связей с общественностью и финансовых услуг.

- Работа может подразумевать работу только для одного важного клиента или нескольких клиентов.
