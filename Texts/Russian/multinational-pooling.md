# Многонациональное объединение
## Что такое многонациональный пул?

Многонациональное объединение — это метод, который глобальные компании используют для управления рисками своих пенсионных планов для сотрудников по всему миру. Различные программы вознаграждения сотрудников многонациональной компании объединяются в международный пул. Для компаний, которые выбирают этот метод, ожидаемым результатом является экономия финансовых средств и лучший контроль над рисками.

## Общие сведения о многонациональном пуле

Многонациональное объединение включает объединение политик из более чем одной страны в одну центральную программу объединения. Этот тип программы предполагает использование выплаты [дивидендов](/dividend) на многонациональном уровне.

Компании могут принять решение об участии в многонациональном объединении, чтобы снизить стоимость льгот, которые они предоставляют своим сотрудникам. Это может быть эффективным решением для небольших страховых пулов (например, менее 100 застрахованных лиц), которые недостаточно велики, чтобы их можно было оценить по [опыту](/experience-rating-insurance) (это означает, что нет достаточного количества исторических данных для эффективного расчета риска будущих претензий и, следовательно, выполнения значимая мера того, сколько должна стоить политика).

Многонациональное объединение позволяет даже самым маленьким застрахованным группам присоединиться к пулу, чтобы эффективно оценить риск и извлечь выгоду из любой возможной экономии.

## Многонациональное объединение по сравнению с местным уровнем

Другим способом описания многонационального пула является объединение финансовых результатов различных местных договоров группового страхования по всему миру в один пул с рейтингом опыта. Короче говоря, страхование, предоставляемое многонациональным пулом, работает точно так же на местном уровне, но в масштабе нескольких стран.

Администрация на местном уровне и условия политики не меняются в многонациональном пуле. Премии и претензии выплачиваются так же, как если бы пул не использовался. Дополнительные административные расходы для локальных пулов не требуются. По сути, многонациональный пул обеспечивает преимущества единого продукта, который предлагает преимущества [управления рисками](/riskmanagement),. масштаба и ценообразования гораздо большего пула.

Многонациональное объединение может использоваться с несколькими типами страхования, такими как медицинское страхование, страхование по инвалидности, несчастный случай, смерть и полностью застрахованные планы пенсионных сбережений (например, план с установленными выплатами на основе аннуитета).

## Типы многонациональных пулов

Существует два типа многонационального объединения: для конкретной компании и для нескольких клиентов. Объединение в пулы компаний используется транснациональными корпорациями с международными клиентами, которые достаточно велики, чтобы создавать пулы самостоятельно. Мультиклиентские пулы доступны для компаний, которые менее глобальны, но, тем не менее, могут сократить расходы, объединив усилия с другими компаниями.

Существует несколько преимуществ использования многонационального пула как для крупных, так и для малых компаний. Они включают:

- Экономия от масштаба и [покупательной способности](/purchasingpower)

- Глобальный рейтинг опыта

- Экономия финансовых затрат

- Улучшенные условия [андеррайтинга](/insurance-underwriter)

- Годовая отчетность

- Инструмент управления и информационная база

## Особенности

- Многонациональный пул также помогает компаниям контролировать риски.

- Многонациональный пул можно использовать со страховкой, пособиями от несчастных случаев и планами пенсионных сбережений.

- Компании используют многонациональные пулы для управления планами вознаграждения сотрудников, чтобы помочь сэкономить деньги.
