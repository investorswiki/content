# Фундаменты
## Что такое фундамент?

В бухгалтерском учете фундамент — это окончательный баланс при сложении всех дебетов и кредитов. Подсчитываются дебеты, за которыми следуют кредиты, и эти два счета суммируются для расчета баланса счета. Фундаменты обычно используются в бухгалтерском учете для определения окончательного баланса, который должен быть включен в [финансовую отчетность](/financial-statements).

## Понимание фундаментов

По мере регистрации бизнес-операций бухгалтеры вводят суммы для каждого типа операций в одну из двух колонок. Каждая транзакция записывается либо как дебет, либо как кредит. Столбцы дебета и кредита суммируются, при этом сумма каждого столбца представляет собой основание.

Два фундамента сводятся вместе для расчета баланса счета за период. Остаток по счету — это сумма, которая переносится в финансовую отчетность. Процесс повторяется для каждого типа транзакции. Термин «основа» уместен, потому что итоги расположены в конце каждого столбца.

## Особенности

- Фундаменты обычно используются в бухгалтерском учете для определения окончательных остатков по счетам, о которых сообщается в финансовых отчетах компании.

- После суммирования дебета и кредита два числа — или фундаменты — объединяются для расчета баланса счета.

- В бухгалтерском учете фундамент - это окончательный баланс при сложении всех дебетов и кредитов.
