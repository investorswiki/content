# Премия по амортизируемым облигациям
## Что такое амортизируемая премия по облигациям?

Премия по амортизируемой облигации — это налоговый термин, который относится к превышению цены, уплаченной за облигацию, сверх ее номинальной стоимости. В зависимости от типа облигации премия может не облагаться налогом и амортизироваться на пропорциональной основе в течение срока действия облигации.

## Понимание премии по амортизируемой облигации

Премия по облигациям возникает, когда цена облигации увеличивается на вторичном рынке из-за падения рыночных процентных ставок. Облигация, продаваемая [с премией](/at-a-premium) к номиналу, имеет рыночную цену выше [номинальной стоимости](/facevalue).

Разница между текущей ценой облигации (или балансовой стоимостью) и номинальной стоимостью облигации является премией по облигации. Например, облигация, номинальная стоимость которой составляет 1000 долларов, но продается за 1050 долларов, имеет премию в размере 50 долларов. Со временем, когда премия по облигации приближается к сроку погашения, стоимость облигации падает до тех пор, пока не достигнет номинала на дату погашения. Постепенное уменьшение стоимости облигации называется амортизацией.

### Основа затрат

Для инвестора в облигации премия, выплачиваемая за облигацию, представляет собой часть [базовой стоимости](/costbasis) облигации, что важно для целей налогообложения. Если по облигации выплачиваются налогооблагаемые проценты, держатель облигации может амортизировать премию, то есть использовать часть премии для уменьшения суммы процентного дохода, включенного в налоги.

Те, кто инвестирует в налогооблагаемые облигации с премией, обычно получают выгоду от амортизации премии, потому что амортизируемая сумма может быть использована для компенсации процентного дохода от облигации. Это, в свою очередь, уменьшит сумму [налогооблагаемого дохода](/taxableincome),. генерируемого облигацией, и, следовательно, любого подоходного налога, причитающегося с нее. Стоимость налогооблагаемой облигации уменьшается на сумму премии, амортизируемой каждый год.

В случае, когда по облигации выплачиваются [необлагаемые налогом](/tax_exempt) проценты, инвестор в облигации должен амортизировать премию по облигации. Хотя эта амортизированная сумма не подлежит вычету при определении налогооблагаемого дохода, налогоплательщик должен уменьшить свою базу в облигации на амортизацию за год. IRS требует, чтобы метод [постоянной доходности](/constantyieldmethod) использовался для ежегодной амортизации премии по облигациям.

## Амортизация премии по облигациям методом постоянной доходности

Метод постоянной доходности используется для определения амортизации премии по облигациям за каждый период начисления. Он амортизирует премию по облигациям, умножая скорректированную базу на доходность при выпуске, а затем вычитая [купонный процент](/coupon). Или в виде формулы:

- **Начисление** = База покупки x (Доходность к погашению / Периоды начисления в год) - Процент по купону

Первым шагом в расчете амортизационной премии является определение [доходности к погашению](/yieldtomaturity) (YTM), которая представляет собой [ставку дисконтирования](/discountrate),. приравнивающую текущую стоимость всех оставшихся платежей, подлежащих выплате по облигации, к базису облигации.

Например, рассмотрим инвестора, который купил облигацию на 10 150 долларов. Облигации имеют пятилетний срок погашения и номинальную стоимость 10 000 долларов США. Он выплачивает купон в размере 5% раз в полгода, а доходность к погашению составляет 3,5%. Рассчитаем амортизацию за первый период и второй период.

### Первый Период

Поскольку по данной облигации производятся полугодовые выплаты, первым периодом являются первые шесть месяцев, после которых производится выплата первого купона; второй период – следующие шесть месяцев, после чего инвестор получает второй купонный платеж и так далее. Поскольку мы предполагаем шестимесячный период начисления, доходность и купонная ставка будут разделены на 2.

Следуя нашему примеру, доходность, используемая для амортизации премии по облигациям, составляет 3,5%/2 = 1,75%, а выплата купона за период составляет 5% / 2 x 10 000 долларов США = 250 долларов США. Амортизация за период 1 выглядит следующим образом:

- Начисление~период1~ = (10 150 долларов США x 1,75 %) — 250 долларов США.

- Начисление~период1~ = $177,63 - $250

- Начисление~период1~ = -$72,38

### Второй период

Основой облигации для второго периода является цена покупки плюс начисление за первый период, то есть 10 150 долларов – 72,38 доллара = 10 077,62 доллара:

- Начисление~период2~ = (10 077,62 долл. США x 1,75%) - 250 долл. США

- Начисление~период2~ = $176,36 - $250

- Начисление~период2~ = -$73,64

Для оставшихся восьми периодов (есть 10 периодов начисления или выплаты для полугодовой облигации со сроком погашения пять лет) используйте ту же самую структуру, представленную выше, для расчета амортизируемой премии по облигации.

По сути, облигация, купленная с премией, имеет отрицательное начисление; другими словами, база амортизируется.

## Особенности

- Премия, уплаченная за облигацию, представляет собой часть [базовой стоимости](/costbasis) облигации и, таким образом, может не облагаться налогом по ставке, распределенной (амортизируемой) в течение срока действия облигации.

- Налоговый термин, премия по амортизируемой облигации относится к превышению цены (премии), уплаченной за облигацию, сверх ее номинальной стоимости.

- IRS требует, чтобы метод постоянной доходности использовался для расчета амортизируемой премии по облигациям каждый год.

- Амортизация премии может быть выгодной, поскольку налоговый вычет может компенсировать любой процентный доход, генерируемый облигацией, тем самым уменьшая налогооблагаемый доход инвестора в целом.
