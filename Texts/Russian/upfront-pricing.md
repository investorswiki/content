# Предварительные цены
## Что такое предварительная цена?

Предварительное ценообразование относится к процентным ставкам и лимитам, установленным для заемщика при андеррайтинге и выпуске кредитной карты.

[андеррайтинге](/underwriting) кредитных карт кредитор будет использовать автоматизированную технологию для установления всех условий ценообразования в начале отношений.

Условия авансового ценообразования формируются на основе индивидуальных методологий [ценообразования, основанных на оценке риска](/riskbased-pricing),. которые учитывают кредитоспособность заемщика и соотношение долга к доходу. Используя эти входные данные, кредитор заранее установит условия ценообразования по кредитной карте для кредитного соглашения. Условия ценообразования обычно сосредоточены на процентной ставке заемщика и кредитном лимите.

## Как работает авансовое ценообразование

Предварительное ценообразование — это метод, используемый для кредитных карт и основанный на методологии ценообразования на основе риска. Методология ценообразования на основе риска исторически использовалась на кредитном рынке для установления всех видов ценообразования по различным кредитным продуктам.

Компании, выпускающие кредитные карты, используют модифицированную версию этой методологии, чтобы прийти к условиям, генерируемым с помощью систем андеррайтинга, которые анализируют информацию из кредитной заявки заемщика по кредитной карте.

> Подробная информация об условиях авансового ценообразования заемщика включена в его кредитный договор.

>

## Стоимость кредитной карты

[Кредитные карты](/creditcard) имеют свои собственные системы ценообразования, которые могут отличаться от невозобновляемых долговых обязательств. Тем не менее, оба будут использовать ценообразование на основе риска в качестве основной методологии андеррайтинга для установления условий. Цены на кредитные карты обычно формируются сразу после подачи заявки, а условия предоставляются заемщику в режиме реального времени.

Большинство кредитных карт будут иметь переменные ставки, которые назначают заемщику маржу в зависимости от его кредитного профиля и отношения долга к доходу. Компании-эмитенты кредитных карт обычно указывают базовые процентные ставки для своих условий ценообразования в качестве маркетингового инструмента для заемщиков. Заемщики, исследующие кредитные карты, могут найти стандартные ставки кредитора на веб-сайте кредитора, который обычно находится под такими лозунгами, как «цены и условия», «информация о ценах» или «ставки, вознаграждения и другая информация».

### Переменные ставки и кредитные лимиты

Рыночные ставки кредитора будут служить основой для установления предварительных условий ценообразования в процессе андеррайтинга. Поскольку большинство кредитных карт имеют переменные ставки, они обычно основаны на [индексированной ставке кредитора](/indexed_rate) плюс маржа.

Это требует, чтобы технология андеррайтинга кредитных карт генерировала определенную маржу для каждого заемщика. В процессе андеррайтинга кредитор также устанавливает кредитный лимит. Кредиторы основывают кредитный лимит счета на информации о заявке заемщика.

Кредитные лимиты обычно различаются для каждого заемщика. Для большинства кредитных карт процентная ставка и кредитный лимит являются двумя ключевыми переменными ценообразования. Эти переменные обычно устанавливаются мгновенно после одобрения кредита, что также приводит к соглашению о кредитной карте, которое заемщик должен подписать, чтобы открыть счет.

## Соглашения о кредитных картах

Кредитные карты обычно обеспечивают немедленное решение о новом счете кредитной карты. Это действие требует, чтобы кредитор в значительной степени полагался на автоматизированную технологию андеррайтинга, которая обрабатывает автоматизированное приложение и немедленно предоставляет заемщику их предварительные условия ценообразования в соглашении о кредитной карте.

В соглашении о кредитной карте также подробно описаны другие важные факторы для заемщика, такие как [комиссия](/fee). Как правило, комиссии будут постоянными для всех счетов для конкретного продукта кредитной карты. Плата за кредитную карту может включать пеню за просрочку платежа, ежемесячную плату за обслуживание счета и годовую плату.

Заемщик может полагаться на соглашение о кредитной карте, чтобы предоставить всю информацию о счете. Договор кредитной карты служит договором займа. Он будет включать политики продукта в отношении просроченных платежей, просрочек и дефолтов. Он также будет включать все процедуры, связанные с продуктом, с подробным описанием того, как продукт будет взимать [проценты](/interest).

## Особенности

- Двумя примерами переменных ценообразования для кредитных карт являются процентная ставка и кредитный лимит.

- Термин авансовое ценообразование относится к процентным ставкам и лимитам, установленным для заемщика на основании андеррайтинга и выпуска кредитной карты.

- Метод авансового ценообразования представляет собой методологию ценообразования на основе риска, которую кредитный рынок использует для установления цен на различные кредитные продукты, такие как, например, кредитные карты и автокредиты.

- Кредиторы используют автоматизированные технологии для установления всех условий ценообразования в начале отношений с клиентом.
