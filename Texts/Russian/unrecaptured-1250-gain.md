# Невозвращенный доход по разделу 1250
## Что такое невозвращенная прибыль по разделу 1250?

Невосстановленная прибыль по разделу 1250 представляет собой налоговую резервацию [Службы внутренних доходов (IRS),.](/irs) когда ранее признанная [амортизация](/depreciation) возвращается в доход при реализации прибыли от продажи амортизируемого [недвижимого](/realestate) имущества. Невосстановленная прибыль по [разделу](/scheduled) [1250](/section1250) облагается налогом по максимальной налоговой ставке 25% или меньше в некоторых случаях с 2019 года. на номер налогоплательщика 1040 .

## Как работает невостребованная прибыль по разделу 1250

[Активы по](/asset) разделу 1231 включают все амортизируемые [капитальные активы](/capitalasset),. находящиеся в собственности налогоплательщика более одного года. Раздел 1231 является зонтиком для активов, относящихся к разделу 1245 и разделу 1250, и последний определяет налоговую ставку [возврата амортизации](/depreciationrecapture). Раздел 1250 касается только недвижимого имущества, такого как здания и земля. Личное имущество, такое как машины и оборудование, подлежит возврату амортизации в качестве [обычного дохода](/ordinaryincome) в соответствии с [разделом 1245](/section1245).

Невосстановленная прибыль по разделу 1250 реализуется только при наличии чистой прибыли по [разделу 1231](/section-1231). По сути, капитальные убытки по всем амортизируемым активам компенсируют невозвращенную прибыль по разделу 1250 от недвижимости. Следовательно, общий чистый убыток капитала сводит невостребованную прибыль по разделу 1250 до нуля .

> Невосстановленная прибыль по разделу 1250 может быть компенсирована капитальными убытками.

>

Прибыль по разделу 1250 возвращается при продаже амортизированной недвижимости, как и в случае любого другого актива; единственная разница заключается в ставке, по которой он облагается налогом. Основанием для получения прибыли является компенсация ранее использованных амортизационных отчислений. В то время как прибыль, относящаяся к накопленной амортизации, облагается налогом по ставке налога на возврат в соответствии с разделом 1250, любые оставшиеся прибыли облагаются только долгосрочной [ставкой прироста капитала в размере](/capital_gains_tax) 15% .

## Пример непогашенной прибыли по разделу 1250

Если недвижимость была первоначально приобретена за 150 000 долларов США, а владелец заявляет об амортизации в размере 30 000 долларов США, [скорректированная базовая стоимость](/adjustedcostbase) недвижимости считается равной 120 000 долларов США. Если недвижимость впоследствии будет продана за 185 000 долларов США, владелец признает общую прибыль в размере 65 000 долларов США по сравнению с базой скорректированной стоимости. Поскольку имущество было продано по цене, превышающей базу, которая была скорректирована с учетом амортизации, невосстановленная прибыль по разделу 1250 основана на разнице между базой скорректированной стоимости и первоначальной ценой покупки.

Таким образом, первые 30 000 долларов прибыли облагаются невостребованной прибылью по разделу 1250, а оставшиеся 35 000 долларов облагаются налогом в размере обычного долгосрочного прироста капитала. В результате 30 000 долларов будут облагаться более высокой ставкой налога на прирост капитала до 25%. Остальные 35 000 долларов будут облагаться налогом по долгосрочной ставке прироста капитала в размере 15%.

## Особые соображения

Поскольку непогашенная прибыль по разделу 1250 считается формой [прироста капитала](/capitalgain),. она может быть компенсирована потерями капитала. Для этого об убытках капитала необходимо сообщить через Форму 8949 и Приложение D, и стоимость убытка может варьироваться в зависимости от того, будет ли он определен как краткосрочный или долгосрочный по своему характеру. Чтобы компенсировать убыток капитала прирост капитала, они оба должны быть определены как краткосрочные или долгосрочные. Краткосрочный убыток не может компенсировать долгосрочную прибыль, и наоборот.

## Особенности

- Невосстановленная прибыль по разделу 1250 представляет собой резерв по налогу на прибыль, предназначенный для восстановления части прибыли, связанной с ранее использованными амортизационными отчислениями.

- Это применимо только к продаже амортизируемого недвижимого имущества.

- Невостребованная прибыль по разделу 1250 обычно облагается налогом по максимальной ставке 25%.

- Прибыль по разделу 1250 может быть компенсирована потерями капитала по разделу 1231.
