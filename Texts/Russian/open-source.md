# Открытый исходный код
Программное обеспечение с открытым исходным кодом — это тип программного обеспечения, [исходный код](/source-code) которого опубликован и доступен любому для просмотра, изменения или распространения. Программное обеспечение с открытым исходным кодом обычно разрабатывается открытым и совместным образом между множеством разных участников. Такие проекты включают в себя принципы прозрачности, открытого обмена, свободного участия, быстрого прототипирования и совместной разработки, ориентированной на сообщество.

Хотя все лицензии на программное обеспечение с открытым исходным кодом позволяют любому изучать, изменять и распространять исходный код, некоторые требуют, чтобы исходный исходный код был опубликован вместе с измененным исходным кодом.

Противоположностью программного обеспечения с открытым исходным кодом является программное обеспечение с закрытым исходным кодом, исходный код которого не публикуется, и только человек, группа или организация, создавшие его, имеют право изменять и распространять его.

Программное обеспечение с открытым исходным кодом может иметь преимущества перед другими типами программного обеспечения. Продвигая философию открытого сотрудничества, которая потенциально может принести пользу как разработчикам, так и пользователям, она также может привести к более качественному продукту благодаря открытому призыву к участию, привлекающему множество разных экспертов из разных областей. Устранение проблем также может быть быстрее и дешевле, чем в случае программного обеспечения с закрытым исходным кодом, поскольку потенциально большое количество участников имеет право, доступ и опыт для решения этих проблем.

С другой стороны, если программное обеспечение с открытым исходным кодом разрабатывается децентрализованно, без единой организации, координирующей процесс разработки, согласование стратегических решений между различными участниками может занять значительное время. Многие проекты с открытым исходным кодом реализуют специальные типы моделей управления для решения этой проблемы.

В более широком смысле, движение, которое стремится заставить все или большую часть программного обеспечения использовать лицензию с открытым исходным кодом, называется движением за программное обеспечение с открытым исходным кодом. В цели этого движения также входит совершенствование технологий с открытым исходным кодом за счет создания высококачественного кода, созданного совместно единомышленниками. Разработчики, поддерживающие движение за программное обеспечение с открытым исходным кодом, добровольно пишут код и обмениваются им, что является вкладом в философию открытого сотрудничества.

## Особенности

- Открытый исходный код относится к проекту, часто это проект разработки программного обеспечения или ИТ, со свободно доступным кодом и разрешением на внесение изменений, исправлений ошибок, изменений и улучшений.

- Открытый исходный код опирается на участие сообщества, доверие и разрешительные стратегии лицензирования.

- Усилия по разработке программного обеспечения с открытым исходным кодом лежат в основе некоторых из самых популярных доступных приложений и операционных систем, включая ОС Android для смартфонов и веб-браузер Firefox.
