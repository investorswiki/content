# Эквивалентный годовой аннуитет (EAA)
## Что такое эквивалентный годовой аннуитет?

Подход эквивалентного годового аннуитета является одним из двух методов, используемых при [составлении бюджета капиталовложений](/capitalbudgeting) для сравнения [взаимоисключающих](/mutuallyexclusive) проектов с неравным сроком службы. Подход EAA рассчитывает постоянный годовой денежный поток, генерируемый проектом на протяжении всего срока его службы, если бы он был аннуитетным. При использовании для сравнения проектов с неравной продолжительностью жизни инвестор должен выбрать тот, у которого выше EAA.

## Понимание подхода эквивалентного годового аннуитета (EAA)

Подход EAA использует трехэтапный процесс для сравнения проектов. Приведенная стоимость постоянных годовых денежных потоков точно равна [чистой приведенной стоимости проекта](/npv). Первое, что делает аналитик, — это рассчитывает чистую приведенную стоимость каждого проекта за время его существования. После этого они вычисляют EAA каждого проекта так, чтобы текущая стоимость аннуитетов точно равнялась чистой приведенной стоимости проекта. Наконец, аналитик сравнивает EAA каждого проекта и выбирает проект с самым высоким EAA.

Например, предположим, что компания со [взвешенной средней стоимостью капитала](/wacc) [, равной](/wacc) 10%, сравнивает два проекта, А и Б. Проект А имеет чистую приведенную стоимость в размере 3 млн. миллионов и расчетный срок службы три года. Используя финансовый калькулятор, EAA проекта A составляет 791 392,44 доллара США, а проекта B — 804 229,61 доллара США. В соответствии с подходом EAA компания выберет проект B, поскольку он имеет более высокую эквивалентную годовую аннуитетную стоимость.

## Особые соображения

### Расчет эквивалентного годового аннуитета

Часто аналитик будет использовать финансовый калькулятор, используя типичные функции текущей стоимости и [будущей стоимости](/futurevalue),. чтобы найти EAA. Аналитик может использовать следующую формулу в электронной таблице или в обычном нефинансовом калькуляторе с точно такими же результатами.

- C = (rx NPV) / (1 - (1 + r)^-n^ )

Где:

- C = эквивалентный аннуитетный денежный поток

- NPV = чистая приведенная стоимость

- r = процентная ставка за период

- n = количество периодов

Например, рассмотрим два проекта. Один имеет семилетний срок и NPV в размере 100 000 долларов. Другой имеет девятилетний срок и NPV в размере 120 000 долларов. Оба проекта дисконтированы по ставке 6 процентов. EAA каждого проекта:

- Первый проект EAA = (0,06 x 100 000 долларов США) / (1 - (1 + 0,06)^-7^ ) = 17 914 долларов США.

- Второй проект EAA = (0,06 x 120 000 долл. США) / (1 - (1 + 0,06)^-9^) = 17 643 долл. США

Первый проект - лучший вариант.

## Особенности

- Подход эквивалентного годового аннуитета является одним из двух методов, используемых в капитальном бюджетировании для сравнения взаимоисключающих проектов с неравным сроком службы.

- Часто аналитик будет использовать финансовый калькулятор, используя типичные функции текущей стоимости и будущей стоимости, чтобы найти EAA.

- При использовании для сравнения проектов с неравной продолжительностью жизни инвестору следует выбирать тот, у которого эквивалентная годовая рента выше.
