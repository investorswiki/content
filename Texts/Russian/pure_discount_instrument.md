# Чистый дисконтный инструмент
## Что такое чистый дисконтный инструмент?

Инструмент с чистым дисконтом — это тип ценной бумаги, по которой не выплачивается доход до [погашения](/maturity). По истечении срока держатель получает [номинальную стоимость](/facevalue) инструмента. Инструмент изначально продается по цене ниже номинальной — со скидкой — и выкупается по номинальной стоимости.

## Как работает чистый дисконтный инструмент

Некоторые долговые инструменты требуют, чтобы эмитент выплатил кредитору заемную сумму плюс проценты. Это влечет за собой периодические выплаты процентов кредитору до наступления срока погашения ценной бумаги, после чего кредитору выплачивается номинальная стоимость ценной бумаги. В остальных случаях по ценным бумагам не производятся плановые процентные выплаты. Вместо этого инвесторы могут приобрести ценные бумаги по цене ниже [номинальной](/parvalue) и получить номинальную стоимость по истечении срока. Эти ценные бумаги относятся к чисто дисконтным инструментам.

Чисто дисконтные инструменты могут принимать форму [бескупонных облигаций](/zero-couponbond) или [казначейских векселей](/treasurybill). Дисконт по этим ценным бумагам, то есть разница между ценой покупки и стоимостью погашения при наступлении срока погашения, представляет собой проценты, которые накапливаются по этим долговым инструментам. Если инструмент с чистым дисконтом удерживается до погашения, держатель облигации получит доход в долларах, равный дисконту.

> Разница между ценой покупки и номинальной стоимостью чистых дисконтных инструментов представляет собой процент, который зарабатывает инвестор.

>

## Пример чистого дисконтного инструмента

Например, предположим, что срок погашения казначейских векселей номинальной стоимостью 1000 долларов составляет 270 дней, и в настоящее время они продаются по цене 950 долларов. Если инвестор будет держать казначейский вексель до момента погашения, он получит положительную доходность в размере:

r = (Скидка / Номинальная стоимость) x (360 / t)

Где,

- r = доходность в годовом исчислении

- Скидка = Номинальная стоимость - Цена покупки

- 360 = банковское соглашение о количестве дней в году

- t = время до погашения

Следуя нашему примеру выше, доходность может быть рассчитана следующим образом:

г = (50 долл. США/1000 долл. США) х (360/270)

= 0,05 х 1,33

= 0,0665, или 6,65%.

Формула, использованная выше, называется [доходностью банковского дисконта](/bankdiscountbasis). Доходность по чистым дисконтным инструментам представляет собой годовой доход, который получается, когда облигации конвертируются в номинальную стоимость. Эта доходность также называется [спотовой процентной ставкой](/spot_rate). Процентная облигация с предсказуемыми денежными потоками или выплатой процентов может рассматриваться как портфель чистых дисконтных облигаций.

Купонные облигации оцениваются с использованием ставок спот путем отнесения доходности чистого дисконтного инструмента со сроком погашения через шесть месяцев к [купонной выплате](/coupon) через шесть месяцев, доходности однолетнего дисконтного инструмента к купонным выплатам через год, и так далее, пока доходность не будет присвоена всем денежным потокам по облигации.

Формула для этого расчета выглядит следующим образом:

Цена = C1/(1+r1) + C2/(1+r2)2 + C3/(1+r3)3 + … + Cn/(1+rn)n + F/(1+rn)n

Где,

- C = денежный поток за период n

- r = спотовая процентная ставка за период n

- F = номинальная стоимость при погашении

Пока инструменты с чистым дисконтом доступны на всех сроках погашения, ставки спот будут точно отражать [временную структуру процентных ставок](/termstructure).

## Особенности

- Чисто дисконтные инструменты не выплачивают купон, вместо этого выплачивается номинальная стоимость по истечении срока.

- Эти инструменты оцениваются с дисконтом к номиналу. Доходность по чистым дисконтным инструментам, также известная как спотовая процентная ставка, представляет собой годовой доход, который получается, когда облигации конвертируются в номинальную стоимость.

- Популярными чисто дисконтными инструментами являются бескупонные облигации и казначейские векселя.
