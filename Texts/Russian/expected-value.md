# Ожидаемое значение (EV)
## Что такое ожидаемая ценность (EV)?

Ожидаемая стоимость (EV) — это ожидаемая средняя стоимость инвестиций в какой-то момент в будущем. Инвесторы используют EV для оценки ценности инвестиций, часто в связи с их относительным риском. [Современная портфельная теория](/modernportfoliotheory) (MPT), например, пытается найти оптимальное распределение портфеля на основе ожидаемой стоимости инвестиций и стандартных отклонений (т. е. риска).

В [статистике](/statistics) и вероятностном анализе ожидаемое значение рассчитывается путем умножения каждого из возможных результатов на вероятность возникновения каждого результата, а затем суммирования всех этих значений. Рассчитывая ожидаемые значения, инвесторы могут выбрать сценарий, который, скорее всего, даст желаемый результат.

## Понимание ожидаемого значения

Сценарный анализ — это один из методов расчета ожидаемой стоимости (EV) инвестиционной возможности. Он использует оценочные вероятности с [многомерными моделями](/multivariate-model) для изучения возможных результатов предлагаемых инвестиций. [Сценарный анализ](/scenario_analysis) также помогает инвесторам определить, берут ли они на себя соответствующий уровень риска, учитывая вероятный результат инвестиций.

EV [случайной величины](/random-variable) дает меру центра распределения переменной. По сути, EV — это долгосрочное среднее значение переменной. Из-за [закона больших чисел](/lawoflargenumbers) среднее значение переменной сходится к EV по мере того, как количество повторений приближается к бесконечности. EV также известен как ожидание, среднее значение или первый момент. EV можно рассчитать для одной дискретной переменной, одной непрерывной переменной, нескольких дискретных переменных и нескольких непрерывных переменных. Для ситуаций с непрерывными переменными необходимо использовать интегралы.

## Особенности

- При инвестировании ожидаемая стоимость акций или других инвестиций является важным фактором и используется в анализе сценариев.

- Ожидаемое значение (EV) описывает долгосрочный средний уровень случайной величины на основе ее вероятностного распределения.

- Современная теория портфелей использует ожидаемую стоимость в сочетании с инвестиционным риском (стандартным отклонением) для создания оптимизированных портфелей.

## ЧАСТО ЗАДАВАЕМЫЕ ВОПРОСЫ

### Как ожидаемая стоимость акции используется в теории портфеля?

Современная теория портфеля (MPT) и связанные с ней модели используют оптимизацию средней дисперсии, чтобы найти наилучшее распределение портфеля с поправкой на риск. Риск измеряется как стандартное отклонение портфеля, а среднее значение представляет собой ожидаемую стоимость ( [ожидаемый доход](/expectedreturn) ) портфеля.

### Как найти ожидаемую стоимость акций, по которым не выплачиваются дивиденды?

Для недивидендных акций аналитики часто используют метод мультипликаторов, чтобы получить ожидаемую стоимость. Например. отношение цены к прибыли (P/E) часто используется и сравнивается с аналогами отрасли. Таким образом, если среднее значение P/E для технологической отрасли равно 25x, EV акций технологических компаний будет в 25 раз превышать прибыль на акцию.

### Какова ожидаемая стоимость дивидендных акций?

Ожидаемая стоимость акций оценивается как [чистая приведенная стоимость](/npv) (NPV) всех будущих дивидендов, которые выплачиваются по акциям. Если вы можете оценить скорость роста дивидендов, вы можете предсказать, сколько инвесторы готовы заплатить за акции, используя модель дисконтирования дивидендов, такую как [модель роста Гордона (](/gordongrowthmodel) [GGM](/gordongrowthmodel) ).
