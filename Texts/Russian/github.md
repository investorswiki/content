# Гитхаб
GitHub — это веб-платформа разработки с открытым исходным кодом и служба хостинга, которая позволяет пользователям загружать файлы, документы и компьютерный код в свои учетные записи. GitHub позволяет как зарегистрированным, так и незарегистрированным пользователям просматривать, загружать и даже вносить свой вклад в загруженные файлы и репозитории исходного кода. Благодаря своей низкой стоимости и возможностям GitHub стал крупнейшей платформой исходного кода в мире, используемой многими разработчиками для совместной работы, а также позволяющей обмениваться разнообразным программным обеспечением, файлами и документацией.

#### Как работает GitHub

После регистрации учетной записи GitHub пользователь может создать свой собственный каталог файлов и контента, который называется репозиторием. После создания репозитория можно создать так называемую основную ветку. Основные ветки GitHub представляют собой единую отправную точку любого проекта. Пользователь может сделать проект доступным для всех (общедоступный репозиторий) или ограничить доступ к нему и поделиться им только с определенными пользователями (частный репозиторий). Из основной ветки могут быть созданы ветки, что в основном позволяет пользователю работать над различными аспектами проекта по отдельности. В общедоступном репозитории другие пользователи могут предлагать изменения в коде, в конечном итоге загружая новые версии файлов. Предложения вносятся с помощью функции, называемой запросом на вытягивание, которая позволяет разработчикам обсуждать и анализировать потенциальные изменения, прежде чем они будут эффективно применены. Платформа GitHub предлагает множество функций и инструментов, которые позволяют разработчикам легко вносить или отменять изменения и отслеживать их общий прогресс.

#### Что такое запрос на вытягивание?

Запрос на вытягивание позволяет пользователям сообщать другим об изменениях, которые они внесли в определенную ветку или репозиторий. Когда создается запрос на вытягивание, пользователь попадает на обзорный экран, где он может легко просмотреть все изменения, внесенные в код, сравнив старую и новую версию. Таким образом, все изменения документируются и записываются, и пользователи также могут добавлять комментарии, метки и вехи или даже назначать определенные задачи другим участникам.

#### Кто использует GitHub?

Благодаря своей гибкой платформе у GitHub есть пользователи как из государственного, так и из частного сектора. Компании-разработчики программного обеспечения, разработчики приложений, отдельные программисты и разработчики криптовалюты — вот лишь некоторые примеры. Как проект с открытым исходным кодом, [Биткойн GitHub](/bitcoin) имеет множество разработчиков по всему миру, и многие люди вносят свой вклад в обзор и улучшение исходного кода Биткойн.

#### Является ли GitHub бесплатным?

GitHub бесплатен для всех проектов с открытым исходным кодом и коммерческих проектов. Однако бесплатные планы ограничивают количество частных и общедоступных репозиториев, которые могут быть созданы учетной записью. Бесплатные планы также ограничивают количество соавторов, которые могут быть в проекте. Поэтому более крупным проектам может потребоваться перейти на платный план GitHub в соответствии с их потребностями.
