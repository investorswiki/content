# Плазма
В двух словах, Plasma разработана как технология масштабирования сети Ethereum. Основное предложение Plasma состоит в том, чтобы работать в качестве каркаса сайдчейнов, которые будут максимально экономно взаимодействовать с блокчейном Ethereum (основной цепочкой). Структура Plasma предназначена для работы в качестве дерева [блокчейна](/blockchain),. иерархически организованного таким образом, что множество «дочерних цепочек» создается поверх основной цепочки.

Платформа Plasma построена с использованием смарт-контрактов и [деревьев Меркла](/merkle-tree) и позволяет создавать неограниченное количество боковых цепей, которые, по сути, являются уменьшенными копиями блокчейна Ethereum.

По сути, дочерние цепи предназначены для запуска индивидуального смарт-контракта, что позволяет компаниям использовать структуру Plasma в соответствии со своими индивидуальными потребностями. Таким образом, для разных вариантов использования могут быть созданы разные смарт-контракты Plasma. Используя безопасность, обеспечиваемую основной цепочкой, Plasma может развернуть множество различных дочерних цепочек, которые будут работать заранее определенным образом для достижения определенных целей (не обязательно связанных с целями основной цепи). Следовательно, основная цепочка блоков Эфириума с меньшей вероятностью будет перегружена.

Короче говоря, Plasma — это [внешнее решение](/off-chain) [n-цепи](/off-chain),. которое направлено на резкое повышение эффективности сети Ethereum (или любого другого блокчейна), снимая большую часть вычислительных обязанностей с основной цепи и перераспределяя их на ряд более мелких функциональных цепей.

Plasma была предложена Виталиком Бутериным и Джозефом Пуном в августе 2017 года в качестве решения для масштабирования Ethereum, но эта концепция может быть адаптирована и реализована и в других блокчейнах. Пун также был одним из идеализаторов предложения Lightning Network, что объясняет сходство Plasma и LN в качестве решений для масштабирования (Plasma для Ethereum и Lightning Network для биткойнов). Но имейте в виду, что у них разные подходы и механизмы.

Ethereum Plasma — это проект с открытым исходным кодом, общедоступный репозиторий которого можно найти на GitHub. Для получения более подробной и технической информации вы также можете обратиться к официальному техническому документу Plasma. Хотя проект все еще находится на ранней стадии разработки, идея действительно очень интересная. В случае успешного внедрения Plasma может значительно повысить эффективность сети Ethereum. Кроме того, это также может быть полезной отправной точкой для других сетей блокчейнов, которым в будущем может потребоваться решение для масштабирования.

Обратите внимание, однако, что Plasma сама по себе — это не проект, это фреймворк для создания масштабируемых приложений, которые могут быть реализованы разными способами разными исследовательскими группами или компаниями.
