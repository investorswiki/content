# Прогнозное моделирование
## Что такое прогнозное моделирование?

Прогнозное моделирование — это процесс использования известных результатов для создания, обработки и проверки модели, которую можно использовать для прогнозирования будущих результатов. Это инструмент, используемый в [прогнозной аналитике](/predictive-analytics),. методе интеллектуального анализа данных, который пытается ответить на вопрос «что может произойти в будущем?»

## Понимание прогнозного моделирования

Анализируя исторические события, компании могут использовать прогностическое моделирование для повышения вероятности прогнозирования событий, поведения клиентов, а также финансовых, экономических и рыночных рисков.

Быстрый переход к цифровым продуктам создал море данных, которые легко доступны для бизнеса. [Большие данные](/big-data) используются компаниями для улучшения динамики отношений между клиентами и бизнесом. Этот огромный объем данных в режиме реального времени извлекается из таких источников, как социальные сети, история просмотров в Интернете, данные мобильных телефонов и платформы [облачных вычислений .](/cloud-computing)

Однако данные обычно неструктурированы и слишком сложны для того, чтобы люди могли проанализировать их за короткий период времени. Из-за огромного объема данных компании используют инструменты прогнозного моделирования — часто с помощью компьютерных программ. Программы обрабатывают огромные объемы исторических данных для оценки и выявления закономерностей в данных. Оттуда модель может обеспечить историческую запись, а также оценку того, какое поведение или события могут произойти снова или в будущем.

> Прогнозное моделирование может использоваться спортивными командами для анализа вероятности успеха с использованием статистики игроков и ситуационного анализа.

>

## Приложения прогнозного моделирования

Прогнозная аналитика использует предикторы или известные функции для создания прогнозных моделей, которые будут использоваться для получения выходных данных. Прогностическая модель способна узнать, как разные точки данных связаны друг с другом. Двумя наиболее широко используемыми методами прогностического моделирования являются [регрессия](/regression) и [нейронные сети](/neuralnetwork).

В области статистики регрессия относится к линейной зависимости между входными и выходными переменными. Прогностическая модель с линейной функцией требует одного предиктора или функции, чтобы предсказать вывод или результат. Например, банк, который надеется обнаружить [отмывание денег](/moneylaundering) на его ранних стадиях, может использовать модель линейного прогнозирования.

Банк хочет определить, кто из его клиентов может заниматься отмыванием денег в определенный момент времени. Используя данные о клиентах банка, прогностическая модель строится на основе долларовой суммы денежных переводов, сделанных клиентами за определенный период времени.

Модель учат распознавать разницу между транзакцией по отмыванию денег и обычной транзакцией. Оптимальным результатом модели должна быть закономерность, которая сигнализирует о том, какой клиент отмывал деньги, а какой нет. Если модель обнаружит, что в отношении конкретного клиента возникает схема мошенничества, она подаст сигнал к действию, которым будет заниматься подразделение по предотвращению мошенничества банка.

## Инструменты прогнозного моделирования

Прогнозные модели также используются в нейронных сетях, таких как машинное обучение и глубокое обучение, которые являются областями искусственного интеллекта (ИИ). Нейронные сети вдохновлены человеческим мозгом и созданы с помощью сети взаимосвязанных узлов на иерархических уровнях, которая представляет собой основу для ИИ. Сила нейронных сетей заключается в их способности обрабатывать нелинейные отношения данных. Они способны создавать взаимосвязи и шаблоны между переменными, которые оказались бы невозможными или слишком трудоемкими для аналитиков-людей.

С одной стороны, банк может ввести известные переменные, такие как стоимость переводов, инициированных его клиентами, в свою модель, чтобы определить, кто, вероятно, занимается отмыванием денег. С другой стороны, нейронная сеть может создать более мощный шаблон, создав связь между входными переменными. Эти входные переменные могут включать время входа в систему, географическое местоположение пользователя, IP-адрес устройства пользователя, получателя или отправителя средств и любые другие переменные или поведение, которые могут быть связаны с деятельностью по отмыванию денег.

Другие методы прогнозного моделирования, используемые финансовыми компаниями, включают деревья решений, анализ данных временных рядов и байесовский анализ. Компании, которые используют преимущества больших данных с помощью методов прогнозного моделирования, лучше понимают, как их клиенты взаимодействуют с их продуктами, и могут определить потенциальные риски и возможности для компании.

## Особенности

- Прогнозное моделирование — это процесс использования известных результатов для создания, обработки и проверки модели, которую можно использовать для будущих прогнозов.

- Компании могут использовать прогностическое моделирование для прогнозирования событий, поведения клиентов, а также финансовых, экономических и рыночных рисков.

- Двумя наиболее широко используемыми методами прогнозного моделирования являются регрессия и нейронные сети.
