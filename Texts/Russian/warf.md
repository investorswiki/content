# Средневзвешенный рейтинговый фактор (WARF)
## Что такое средневзвешенный рейтинговый фактор (WARF)

Средневзвешенный рейтинговый фактор (WARF) — это показатель, который используется рейтинговыми компаниями для определения кредитного качества портфеля. Этот показатель объединяет кредитные рейтинги активов портфеля в единый рейтинг. WARF чаще всего рассчитываются для [обеспеченных долговых обязательств](/cdo) (CDO).

## Понимание средневзвешенного рейтингового фактора (WARF)

Чтобы рассчитать средневзвешенный рейтинговый фактор CDO, рейтинговые агентства должны сначала определить [кредитный рейтинг](/creditrating) для каждого инструмента, лежащего в основе CDO. Например, в таксономии Fitch Ratings этот рейтинг может варьироваться от чрезвычайно высокого кредитного качества (AAA) до низкого качества (CCC) и дефолта ( D [)](/fitch-ratings) [.](/fitch-ratings) Этот буквенный рейтинг соответствует числовому рейтинговому фактору, который, в свою очередь, соответствует 10-летней вероятности дефолта. WARF определяется путем вычисления средневзвешенного значения этих числовых факторов. Для расчета средневзвешенного значения условный баланс актива умножается на рейтинговый коэффициент, а затем эти значения суммируются. Затем эта сумма делится на общий условный баланс портфеля.
