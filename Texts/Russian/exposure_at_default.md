# Воздействие по умолчанию (EAD)
## Что такое риск по умолчанию (EAD)?

Риск при дефолте (EAD) — это общая стоимость, которой подвергается банк в случае дефолта по кредиту. Используя подход, основанный на внутренних рейтингах (IRB), финансовые учреждения рассчитывают свой риск. Банки часто используют внутренние модели управления рисками по умолчанию для оценки соответствующих систем EAD. За пределами банковской отрасли EAD известен как кредитный риск.

## Понимание воздействия по умолчанию

EAD — это прогнозируемая сумма убытков, которым может подвергнуться банк, когда должник не выполнит свои обязательства по кредиту. Банки часто рассчитывают значение EAD для каждого кредита, а затем используют эти цифры для определения общего риска дефолта. EAD — это динамический номер, который меняется по мере того, как заемщик выплачивает кредит кредитору.

Существует два метода определения экспозиции по умолчанию. Регуляторы используют первый подход, который называется базовым внутренним рейтингом (F-IRB). Второй метод, называемый [расширенным внутренним рейтингом](/airb) (A-IRB), является более гибким и используется банковскими учреждениями. Банки должны раскрывать свою подверженность риску. Банк будет основывать эту цифру на данных и внутреннем анализе, таких как характеристики заемщика и тип продукта. EAD, наряду с [убытком при дефолте](/lossgivendefault) (LGD) и [вероятностью дефолта](/defaultprobability) (PD), используются для расчета капитала кредитного риска финансовых учреждений.

## Особые соображения

### Вероятность дефолта и убытков при дефолте

Анализ PD — это метод, используемый крупными учреждениями для расчета ожидаемых убытков. PD присваивается каждой мере риска и представляет вероятность дефолта в процентах. PD обычно измеряется путем оценки просроченных кредитов. Он рассчитывается путем проведения миграционного анализа кредитов с аналогичным рейтингом. Расчет производится для определенного периода времени и измеряет процент невозвратных кредитов. Затем PD присваивается уровню риска, и каждый уровень риска имеет один процент PD.

LGD, уникальный для банковской отрасли или сегмента, измеряет ожидаемый убыток и отображается в процентах. LGD представляет собой сумму, не возмещенную кредитором после продажи базового актива, если заемщик не выполнит обязательства по кредиту. Точную переменную LGD может быть трудно определить, если потери портфеля отличаются от ожидаемых. Неточный LGD также может быть связан со статистически малым сегментом. Отраслевые LGD обычно можно получить у сторонних кредиторов.

Кроме того, числа PD и LGD обычно действительны в течение всего экономического цикла. Однако кредиторы будут проводить переоценку с изменениями рынка или состава портфеля. Изменения, которые могут вызвать переоценку, включают восстановление экономики, рецессию и слияния.

Банк может рассчитать свои ожидаемые убытки, умножив переменную EAD на PD и LGD:

- EAD x PD x LGD = **Ожидаемый убыток**

### Почему важна экспозиция по умолчанию

В ответ на кредитный кризис 2007–2008 годов банковский сектор принял международные правила, чтобы уменьшить риск дефолта. Целью Базельского комитета по банковскому надзору является улучшение способности банковского сектора справляться с финансовым стрессом. За счет улучшения управления рисками и прозрачности банков международное соглашение надеется избежать эффекта домино, связанного с банкротством финансовых учреждений.

## Особенности

- Подверженность дефолту (EAD) — это прогнозируемая сумма убытков, которым может подвергнуться банк, когда должник не выполнит свои обязательства по кредиту.

- Подверженность дефолту, убыток при дефолте и вероятность дефолта используются для расчета капитала кредитного риска финансовых учреждений.
