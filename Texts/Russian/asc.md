# Комитет по стандартам бухгалтерского учета (ASC)
## Что такое Комитет по стандартам бухгалтерского учета?

Комитет по стандартам бухгалтерского учета (ASC) был бывшей организацией при Консультативном комитете органов бухгалтерского учета (CCAB) в Соединенном Королевстве. В обязанности Комитета по бухгалтерским стандартам (ASC) входили разработка стандартов [финансовой отчетности и бухгалтерского учета](/financial-statements),. регистрация этих стандартов и распространение их через пресс-релизы и публикации. Он существовал с 1976 по 1990 год, когда его обязанности взял на себя Совет по стандартам бухгалтерского учета (ASB). Комитету предшествовал Руководящий комитет по стандартам бухгалтерского учета (ASSC).

До того, как были созданы регулирующие органы, бухгалтерские скандалы происходили довольно регулярно. Бухгалтерские скандалы в конце 1960-х - начале 1970-х годов побудили к созданию Комитета по стандартам бухгалтерского учета для выпуска стандартов бухгалтерского учета. В 1990 году его обязанности взял на себя Совет по стандартам бухгалтерского учета, который затем был заменен Советом по [международным стандартам бухгалтерского учета](/ias) (IASB) в 2001 году. Совет по международным стандартам бухгалтерского учета издает стандарты бухгалтерского учета в Соединенном Королевстве и сотрудничает с разработчиками стандартов бухгалтерского учета других стран. . В США есть [Совет по стандартам финансового учета (FASB)](/fasb),. базирующийся в Коннектикуте.
