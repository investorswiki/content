# Неамортизированная премия по облигациям
## Что такое неамортизированная премия по облигациям?

Премия по неамортизированной облигации представляет собой разницу между [номинальной стоимостью облигации](/facevalue) и ценой ее продажи. Если облигация продается со [скидкой](/discountbond),. например, по 90 центов за доллар, эмитент все равно должен выплатить полные 100 центов номинальной стоимости по [номиналу](/parvalue). Поскольку эта сумма процентов еще не выплачена держателям облигаций, она является обязательством [эмитента](/issuer).

## Понимание неамортизированной премии по облигациям

Премия по [облигации](/premiumbond) — это превышение цены [облигации](/bond) над ее номинальной стоимостью. Когда преобладающие [процентные ставки](/interestrate) в экономике снижаются, цена облигаций растет. Это связано с тем, что рыночная процентная ставка становится ниже, чем фиксированная [купонная ставка](/coupon-rate) по непогашенным облигациям.

Поскольку держатели облигаций владеют облигациями с более высокой процентной ставкой, им требуется премия в качестве компенсации на рынке. Неамортизированная премия по облигациям — это то, что осталось от премии по облигациям, которую эмитент еще не [списал](/write-off) в качестве процентных расходов.

Например, предположим, что при процентной ставке 5% эмитент облигаций продал облигации с фиксированным купоном в размере 5%, подлежащим выплате ежегодно. Через некоторое время процентные ставки снизились до 4%. Новые эмитенты облигаций будут выпускать облигации с более низкой процентной ставкой. Инвесторы, которые предпочли бы купить облигацию с более высоким купоном, должны будут заплатить премию держателям облигаций с более высоким купоном, чтобы побудить их продать свои облигации. В этом случае, если номинальная стоимость облигации составляет 1000 долларов, а облигация продается по цене 1090 долларов после снижения процентных ставок, разница между ценой продажи и номинальной стоимостью составляет неамортизированную премию по облигации (90 долларов).

Неамортизируемый доход по облигациям представляет собой часть дохода по облигациям, которая будет амортизирована (списана) за счет расходов в будущем. Амортизированная сумма этой облигации кредитуется как [процентный расход](/interestexpense). Если по облигации выплачиваются налогооблагаемые проценты, держатель облигации может [амортизировать премию](/amortizable-bond-premium),. то есть использовать часть премии для уменьшения суммы процентного дохода, включенного в налоги.

## Особые соображения

Те, кто инвестирует в облигации с налогооблагаемой премией, как правило, получают выгоду от амортизации премии, поскольку амортизируемая сумма может быть использована для компенсации процентного дохода от облигации, что уменьшит сумму налогооблагаемого дохода, который инвестор должен будет выплатить по облигации. Стоимость [налогооблагаемой](/costbasis) облигации уменьшается на сумму премии, амортизируемой каждый год.

В случае, когда по облигации выплачиваются необлагаемые налогом проценты, инвестор в облигации должен амортизировать премию по облигации. Хотя эта амортизированная сумма не подлежит вычету при определении налогооблагаемого дохода, налогоплательщик должен уменьшить свою базу в облигации на амортизацию за год.

Неамортизированный доход по облигациям учитывается как обязательство перед эмитентом облигаций. В балансовом отчете эмитента эта статья записывается на специальном счете, называемом счетом премии по неамортизированным облигациям. На этом счете отражается оставшаяся сумма премии по облигациям, которую эмитент облигаций еще не амортизировал или не списал на процентные расходы в течение срока действия облигации.

## Пример: расчет премии по неамортизированной облигации

Для расчета суммы, подлежащей амортизации за налоговый год, цена облигации умножается на [доходность к погашению](/yieldtomaturity) (YTM), результат которой вычитается из купонной ставки облигации. Используя приведенный выше пример, доходность к погашению составляет 4%.

- Умножение цены продажи облигации на YTM дает 1090 долларов x 4% = 43,60 доллара.

- Это значение при вычитании из суммы купона (ставка купона 5% x номинальная стоимость 1000 долларов США = 50 долларов США) дает 50 - 43,60 долларов США = 6,40 долларов США, что является амортизируемой суммой.

- Для целей налогообложения держатель облигаций может уменьшить свой процентный доход в размере 50 долларов США до 50-6,40 долларов США = 43,60 долларов США.

- Неамортизированная премия по истечении года составляет 90 долларов премии по облигациям - 6,40 долларов амортизируемой суммы = 83,60 долларов.

- Во втором налоговом году 6,40 долларов премии по облигациям уже амортизированы, поэтому базовая [стоимость облигации](/costbasis) составляет 1090 долларов - 6,40 долларов = 1083,60 долларов.

- Премиальная амортизация за 2-й год = 50 долларов США – (1 083,60 долларов США x 4 %) = 50 долларов – 43,34 долларов США = 6,64 долларов США.

- Премия, оставшаяся после второго года или неамортизированная премия, составляет 83,60 доллара – 6,64 доллара = 76,96 доллара.

Предполагая, что срок погашения облигации составляет пять лет, вы можете выполнить тот же расчет для оставшихся трех лет. Например, базовая стоимость облигации на третий год составит 1 083,60 долл. США - 6,64 долл. США = 1 076,96 долл. США.

## Особенности

- Неамортизированная премия по облигациям является обязательством для эмитентов, поскольку они еще не списали эти процентные расходы, но в конечном итоге наступят сроки их погашения.

- Премия по неамортизированным облигациям представляет собой чистую разницу в цене, по которой эмитент облигаций продает ценные бумаги, за вычетом фактической номинальной стоимости облигаций на момент погашения.

- В финансовых отчетах неамортизированная премия по облигациям отражается на счете обязательств, который называется «Счет неамортизированной премии по облигациям».
