# Кривая доходности номинала
## Что такое номинальная кривая доходности?

Кривая номинальной доходности представляет собой графическое представление доходности гипотетических казначейских ценных бумаг с ценой по номиналу. На кривой номинальной доходности купонная ставка будет равна [доходности к погашению](/yieldtomaturity) (YTM) ценной бумаги, поэтому казначейские облигации будут торговаться по [номинальной стоимости](/parvalue).

Кривую доходности номинала можно сравнить с [кривой доходности спот](/spot_rate_yield_curve) и кривой доходности форварда для казначейских облигаций.

## Понимание кривых доходности по номиналу

Кривая доходности представляет собой график, показывающий взаимосвязь между процентными ставками и доходностью облигаций с различными сроками погашения, от трехмесячных [казначейских векселей](/treasurybill) до 30-летних [казначейских облигаций](/treasurybond). График построен с осью Y, изображающей процентные ставки, и осью X, показывающей увеличение продолжительности времени.

Поскольку краткосрочные облигации обычно имеют более низкую доходность, чем долгосрочные облигации, кривая имеет наклон вверх вправо. Когда говорят о кривой доходности, обычно имеют в виду кривую доходности спот, в частности, кривую доходности спот для [безрисковых облигаций](/riskfreeasset). Однако в некоторых случаях упоминается другой тип кривой доходности — кривая номинальной доходности.

Кривая номинальной доходности отображает YTM купонных облигаций с разными сроками погашения. Доходность к погашению — это доход, который инвестор в облигации ожидает получить, предполагая, что облигация будет удерживаться до погашения. Облигация, выпущенная по [номиналу](/parvalue),. имеет доходность до погашения, равную [купонной ставке](/coupon-rate). Поскольку процентные ставки колеблются во времени, YTM либо увеличивается, либо уменьшается, чтобы отразить текущую среду процентных ставок.

Например, если процентные ставки снижаются после выпуска облигации, стоимость облигации будет увеличиваться, учитывая, что купонная ставка, прикрепленная к облигации, теперь выше, чем процентная ставка. В этом случае купонная ставка будет выше, чем YTM. По сути, YTM — это [ставка дисконтирования,](/discountrate) при которой сумма всех будущих денежных потоков от облигации (т. е. купонов и основной суммы долга) равна текущей цене облигации.

Номинальная доходность — это купонная ставка, при которой цена облигации равна нулю. Кривая номинальной доходности представляет собой облигации, которые торгуются по номиналу. Другими словами, кривая номинальной доходности представляет собой график зависимости доходности к погашению от [срока до погашения](/termtomaturity) для группы облигаций, оцененных по номинальной цене. Он используется для определения купонной ставки, которую будет выплачивать новая облигация с заданным сроком погашения, чтобы продать ее по номинальной стоимости сегодня. Кривая номинальной доходности дает доходность, которая используется для дисконтирования нескольких денежных потоков по облигации с купонным доходом. Он использует информацию из кривой спотовой доходности, также известной как кривая купона с нулевым процентом, для дисконтирования каждого купона по соответствующей спотовой ставке.

Поскольку дюрация спотовой кривой доходности больше, кривая всегда будет лежать выше кривой номинальной доходности, когда кривая номинальной доходности имеет восходящий наклон, и ниже кривой номинальной доходности, когда кривая номинальной доходности имеет нисходящий наклон.

## Получение кривой доходности по номиналу

Получение кривой номинальной доходности является одним из шагов к созданию теоретической кривой доходности спотовой ставки, которая затем используется для более точной оценки купонной облигации. Для получения безарбитражных [форвардных процентных ставок используется](/forwardrate) метод, известный как [бутст-](/bootstrapping) [рэп](/bootstrapping). Поскольку казначейские векселя, предлагаемые правительством, не содержат данных за каждый период, метод начальной загрузки используется в основном для заполнения недостающих цифр для получения кривой доходности. Например, рассмотрим эти облигации с номинальной стоимостью 100 долларов и сроком погашения шесть месяцев, один год, 18 месяцев и два года.

Поскольку купонные выплаты производятся раз в полгода, шестимесячная облигация имеет только один платеж. Таким образом, его доходность равна номинальной ставке, которая составляет 2%. Годовая облигация предусматривает два платежа через шесть месяцев. Первый платеж составит 100 долларов x (0,023/2) = 1,15 доллара. Этот процентный платеж должен быть дисконтирован на 2%, что является спотовой ставкой за шесть месяцев. Вторым платежом будет сумма купонного платежа и погашения основного долга = 1,15$ + 100$ = 101,15$. Нам нужно найти ставку, по которой этот платеж следует дисконтировать, чтобы получить номинальную стоимость в 100 долларов. Расчет:

- 100 долл. США = 1,15 долл. США/(1 + (0,02/2)) + 101,15 долл. США/(1 + (x/2)) ^2^

- 100 долл. США = 1,1386 + 101,15 долл. США/(1 + (х/2))^2^

- 98,86 долл. США = 101,15 долл. США/(1 + (х/2)) ^ 2 ^

- (1 + (х/2)) ^ 2 ^ = 101,15 долл. США / 98,86 долл. США

- 1 + (х/2) = √1,0232

- х/2 = 1,0115 – 1

- х = 2,302%

Это ставка с нулевым купоном для однолетней облигации или годовая спотовая ставка. Используя этот процесс, мы можем рассчитать спотовую ставку для других облигаций со сроком погашения через 18 месяцев и два года.

## Особенности

- Кривая номинальной доходности интерполирует кривую доходности казначейских ценных бумаг, основанную на том, что все сроки погашения являются ценами по номинальной стоимости.

- Номинальная доходность обычно падает ниже кривых доходности как спот, так и форвард при нормальных обстоятельствах.

- При номинальной стоимости процентная ставка должна быть идентична купонной ставке, выплачиваемой по облигации.
