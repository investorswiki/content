# Для этого случая
Фраза «ad hoc» — это фраза латинского происхождения, которая используется в современном английском языке для обозначения «для этой цели» или «специально для этого». Прямой перевод латинской фразы в большинстве словарей дается как «для этого».

Термин «специальный» может использоваться для обозначения того, что решение, найденное для конкретной проблемы или проблемы, не должно рассматриваться как решение по умолчанию, которое будет использоваться в будущих приложениях.

В разработке программного обеспечения термин «специальное» часто используется во фразе «специальное решение» для обозначения конкретного и индивидуального решения, разработанного для удовлетворения потребностей системы или пользователя, которые представлены.
