# Вордпресс (CMS)
## Что такое WordPress?

WordPress — популярная система управления контентом (CMS) с открытым исходным кодом. Хотя изначально он был связан в основном с личными блогами, с тех пор он стал использоваться для самых разных веб-сайтов, включая профессиональные публикации и платформы [электронной коммерции .](/ecommerce)

С момента своего запуска в 2003 году WordPress стал использоваться примерно на одной трети веб-сайтов по всему миру. Сегодня он является домом для большого сообщества разработчиков, которое выпустило более 50 000 плагинов.

## Как работает WordPress

Пользователи WordPress могут выбрать одну из двух основных версий сервиса. Первая из них — «размещенная» версия WordPress, доступная на WordPress.com. Эта версия позволяет пользователям запускать веб-сайт, размещенный на собственных серверах WordPress. Таким образом, его часто предпочитают нетехнические пользователи, которые могут пожелать избежать сложностей с размещением веб-сайта на собственном частном сервере. В качестве дополнительного преимущества размещенная версия WordPress автоматически обновляется до последней версии программного обеспечения WordPress, что может помочь смягчить потенциальные нарушения безопасности.

Для более продвинутых пользователей также можно просто загрузить последнюю версию WordPress с сайта WordPress.org. Выбирая эту опцию, пользователь должен принять собственные меры для размещения программного обеспечения либо на своем частном сервере, либо у [стороннего](/third-party) хостинг-провайдера. Затем пользователь может настроить программное обеспечение по своему усмотрению, но также должен нести ответственность за ручную загрузку любых последующих обновлений.

Из-за большой пользовательской базы веб-сайты WordPress являются популярной мишенью для хакеров, которые часто пытаются использовать устаревшие версии программного обеспечения, которые могут иметь известные уязвимости в системе безопасности. По этой причине, вероятно, разумно избегать самостоятельного размещения службы, если вы не уверены в своей способности защитить и обновить службу вместе с любыми дополнительными подключаемыми модулями, которые вы, возможно, установили.

> ### Настройки WordPress

> WordPress разработан с нуля, чтобы его можно было легко настроить. Его [интерфейс прикладного программирования (API)](/application-programming-interface) охватывает все, от базы данных до модификаций и настроек темы. Сохраняя все стандартизированным, разработчики могут сосредоточиться на создании функциональности поверх WordPress, зная, что он будет работать на всех установках, которые поддерживаются в актуальном состоянии.

>

## Реальный пример WordPress

Есть много разных компаний, которые предоставляют плагины и темы для WordPress, предназначенные для расширения его функциональности. Например, WooCommerce стал одним из самых популярных инструментов электронной коммерции для WordPress, позволяя пользователям принимать заказы, отслеживать поставки и добавлять другие функции электронной коммерции на свой существующий веб-сайт WordPress .

Среди многих других примеров плагинов для WordPress есть те, которые связаны с поисковой оптимизацией (SEO), контактными формами, маркетингом информационных бюллетеней, функциями социальных сетей и комментариев на сайте, а также различными функциями дизайна, предназначенными для того, чтобы сделать сайты более привлекательными.

## Особенности

- WordPress — ведущая платформа CMS с открытым исходным кодом.

- Он известен своими широкими возможностями настройки и может похвастаться большим сообществом разработчиков плагинов.

- Из-за большой пользовательской базы WordPress часто становится целью хакеров. Чтобы снизить этот риск, пользователи должны следить за тем, чтобы их программное обеспечение WordPress постоянно обновлялось.
