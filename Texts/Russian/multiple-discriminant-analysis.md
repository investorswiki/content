# Множественный дискриминантный анализ (MDA)
## Что такое множественный дискриминантный анализ (MDA)?

Термин множественный дискриминантный анализ (MDA) относится к статистическому методу, используемому специалистами по [финансовому планированию](/financialplanner),. [инвестиционными консультантами](/investmentadvisor) и аналитиками для оценки потенциальных инвестиций, когда на карту поставлено множество переменных. MDA позволяет финансовым специалистам инвестировать в различные рыночные ценные бумаги, изучая различные факторы или переменные, такие как волатильность. Это раздел дискриминантного анализа, который используется исследователями и статистиками, которые проводят классификации людей и данных на основе различных переменных.

## Понимание множественного дискриминантного анализа (MDA)

Множественный дискриминантный анализ — это метод, который отличает наборы данных друг от друга на основе характеристик, наблюдаемых профессионалом. Он используется в [финансах](/finance) для сжатия [разницы](/variance) между ценными бумагами при проверке нескольких переменных.

Используя метод MDA, финансовые специалисты уменьшают различия между определенными переменными, чтобы их можно было разделить на несколько более крупных групп, а затем сравнить с другой переменной. В большинстве случаев специалисты, использующие MDA, часто пытаются сгруппировать данные как минимум в три, если не больше, разных группы.

Аналитик, рассматривающий ряд акций, может использовать множественный дискриминантный анализ в качестве инструмента, чтобы сосредоточиться на наиболее важных точках данных. Это упрощает другие различия между [акциями,](/stock) не исключая их полностью. Например, аналитик, который хочет выбрать ценные бумаги на основе значений, которые измеряют [волатильность](/volatility) и историческую согласованность, может использовать MDA, чтобы исключить другие переменные, такие как цена.

> Другие переменные, которые аналитики могут использовать при проведении множественного дискриминантного анализа, включают различные финансовые коэффициенты.

>

Основная причина, по которой профессионалы используют эту технику, заключается в разработке [эффективных по Марковицу множеств](/markowitzefficientset). Эти инвестиционные [портфели](/portfolio) разрабатываются на основе доходности, которая максимизируется при определенном уровне риска. Они были названы в честь экономиста Гарри Марковица, который также считается отцом [современной портфельной теории (MPT)](/modernportfoliotheory).

## Особые соображения

Как отмечалось выше, множественный дискриминантный анализ связан с дискриминантным анализом, который обычно используется статистиками и другими исследователями. MDA также известен, по крайней мере, статистикам, как канонический анализ переменных или канонический дискриминантный анализ.

Это тип дискриминантного анализа, который широко используется исследователями, [анализирующими данные](/data-analytics) во многих областях. Дискриминантный анализ помогает исследователям и статистикам классифицировать различные наборы данных, устанавливая правило или выбирая значение, обеспечивающее наиболее значимое разделение.

## Особенности

- MDA - это раздел дискриминантного анализа, который обычно используется статистиками и другими исследователями.

- Этот метод используется для сжатия дисперсии между ценными бумагами при проверке нескольких переменных.

- Финансовые специалисты часто используют MDA как способ разработки эффективных наборов Марковица, типа портфеля, который максимизирует доход на основе определенных уровней риска.

- Аналитик, рассматривающий ряд акций, может использовать множественный дискриминантный анализ, чтобы сосредоточиться на точках данных, наиболее важных для рассматриваемого решения.

- Специалисты по финансовому планированию используют множественный дискриминантный анализ для оценки потенциальных инвестиций, когда необходимо учитывать ряд переменных.
