# Программа Medallion для агентов по переводу ценных бумаг
## ОПРЕДЕЛЕНИЕ программы Medallion для агентов по переводу ценных бумаг

Программа Medallion для агентов по переводу ценных бумаг (STAMP) — это система проверки, используемая многими различными учреждениями для [гарантии индивидуальных подписей](/medallionsignatureguarantee) на физических сертификатах ценных бумаг, требующих передачи. STAMP является одной из трех гарантийных программ для этой цели, двумя другими являются программа медальона фондовой биржи (SEMP) и программа подписи медальона Нью-Йоркской фондовой биржи (MSP). STAMP — это некоммерческая организация, которая является совладельцем и сооператором (наряду с Kemark Financial Service) официальной службы гарантии подписи, утвержденной Ассоциацией по переводу ценных бумаг.

## РАЗБИВКА Программа Medallion для агентов по переводу ценных бумаг

Комиссия по ценным бумагам и биржам (SEC) требует, чтобы подпись человека на физическом сертификате ценной бумаги была проверена до того, как ценная бумага будет принята [агентом по передаче](/transferagent). Более того, Правило SEC 17Ad-15 гласит, что агенты по передаче «способствуют установлению равного отношения к финансовым учреждениям, которые гарантируют подписи индоссантов ценных бумаг, повышают эффективность процесса передачи ценных бумаг и снижают риск, связанный с неспособностью гаранта подписи выполнить свои обязательства. .

Члены STAMP, в том числе коммерческие банки, кредитные союзы, сберегательные банки или [брокеры-дилеры](/broker-dealer),. должны иметь оборудование для печати Medallion и [гарантийное](/surety) покрытие для оказания услуги. Оттиск или печать медальона на сертификате указывает трансфер-агенту, что подпись гарантируется надежным финансовым учреждением, которое, предположительно, знает своего клиента. Трансфер-агент не примет сертификат без гарантии подписи из-за риска потери или ответственности, если позже обнаружится, что подпись была подделана. Гаранты, как правило, не взимают плату за услугу выходных данных, вместо этого предлагая их в качестве любезности за свои банковские или брокерские отношения с клиентом.
