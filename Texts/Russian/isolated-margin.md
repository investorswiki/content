# Изолированная маржа
Изолированная маржа — это баланс маржи, распределенный по отдельной позиции. Режим изолированной маржи позволяет трейдерам управлять своим риском по отдельным позициям, ограничивая размер маржи, выделяемой для каждой из них. Выделенный баланс маржи для каждой позиции можно настроить индивидуально.

Если позиция трейдера ликвидируется в режиме изолированной маржи, вместо всего баланса маржи ликвидируется только баланс изолированной маржи.

Например, допустим, Алиса открывает длинную позицию по BTC на сумму 1000 долларов США с кредитным плечом 10x. У нее есть баланс маржи в размере 2000 долларов США, но она хочет рискнуть только частью этой суммы для отдельной позиции. Она устанавливает изолированную маржу для позиции в размере 100 долларов США. Если ее позиция будет ликвидирована, она не потеряет более 100 долларов США.

Величина изолированной маржи может быть скорректирована для открытых позиций. Если позиция в режиме изолированной маржи близка к ликвидации, ликвидацию можно предотвратить, назначив для позиции дополнительную маржу.

С другой стороны, настройка режима маржи, связанного с позицией после того, как она уже открыта, невозможна. Перед входом в позицию настоятельно рекомендуется проверить настройки режима маржи.

Другим часто используемым режимом маржи на торговых платформах является кросс-маржа. В режиме кросс-маржи весь баланс маржи распределяется между открытыми позициями, чтобы избежать ликвидации. Если кросс-маржа включена, трейдер рискует потерять весь баланс маржи вместе с любыми открытыми позициями в случае ликвидации. Любой реализованный PnL из другой позиции может помочь проигрышной позиции, которая близка к ликвидации.

Как правило, кросс-маржа является настройкой по умолчанию на большинстве торговых платформ, так как это более простой подход, подходящий для начинающих трейдеров. Однако изолированная маржа также может быть полезна для более спекулятивных позиций, требующих строгих ограничений на понижение.
