# Алгебраический метод
## Что такое алгебраический метод?

Алгебраический метод относится к различным методам решения пары линейных уравнений, включая [построение графика](/line-graph),. подстановку и исключение.

## Что говорит вам алгебраический метод?

Метод построения графиков включает в себя построение графиков двух уравнений. Пересечение двух линий будет координатой x, y, которая является решением.

С помощью метода подстановки перестройте уравнения, чтобы выразить значение переменных x или y через другую переменную. Затем замените это выражение значением этой переменной в другом уравнении.

## Особенности

- Наиболее часто используемые алгебраические методы включают метод подстановки, метод исключения и метод построения графиков.

- Алгебраический метод представляет собой совокупность нескольких методов, используемых для решения пары линейных уравнений с двумя переменными.
