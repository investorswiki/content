# Квантовые вычисления
Квантовые вычисления работают с частицами, которые могут находиться в суперпозиции. Эти частицы представляют собой кубиты, а не биты, и могут принимать значения 1, 0 или оба значения одновременно.

## Особенности

- Это не похоже на классические компьютеры, где добавление дополнительных транзисторов увеличивает мощность только линейно.

- Квантовые вычисления состоят из кубитов.

- Мощность квантовых компьютеров растет экспоненциально с увеличением количества кубитов.

- В отличие от обычного компьютерного бита, который может быть 0 или 1, кубит может быть любым из них или суперпозицией 0 и 1.

- Квантовые вычисления — это изучение того, как использовать явления квантовой физики для создания новых способов вычислений.
