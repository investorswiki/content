# Виртуальная машина
Виртуальная машина (ВМ) — это программа, которая эмулирует компьютерную систему. Он имеет виртуальный ЦП, память и хранилище и внешне ничем не отличается от физической машины с тем же оборудованием.

## Как работает виртуальная машина?

Виртуальная машина имеет операционную систему, файлы и приложения. Он работает так же, как физический компьютер; он может размещать веб-сайты, запускать программы и существовать в сети с другими системами. При просмотре на экране виртуальная машина выглядит как окно, работающее в собственной операционной системе этого компьютера.

У него также есть ЦП, память и хранилище, но они моделируются, а не физически. Благодаря этому на одном сервере может сосуществовать несколько виртуальных машин. Когда это происходит, сервер использует часть программного обеспечения, называемого гипервизором, для управления ими. Гипервизор сопоставляет виртуальное оборудование виртуальной машины с физическими компонентами на сервере.

В этом сценарии на одном физическом сервере могут работать две, три, четыре или даже десять виртуальных машин. Однако количество ограничено ресурсами сервера — слишком много ВМ приведет к снижению производительности.

## Почему используются виртуальные машины?

Виртуальные машины — это песочницы — фрагменты виртуального пространства, в которых код может выполняться отдельно от остальной системы. Это делает виртуальные машины идеальными для тестирования нового программного обеспечения или запуска кода, который может быть вредоносным.

Виртуальная машина может работать под управлением другой или более старой ОС, чтобы пользователь мог запускать программное обеспечение, которое не будет работать в операционной системе компьютера. Например, кто-то может использовать виртуальную машину для запуска части программного обеспечения Windows на устройстве Linux. Для этого они запускали виртуальную машину с операционной системой Windows в своей системе Linux.

## Ключевой пример: виртуальная машина Ethereum (EVM)

Все узлы в сети Ethereum используют виртуальную машину Ethereum (EVM), которая используется для выполнения смарт-контрактов. Поскольку это происходит в отдельной песочнице, это обеспечивает значительные преимущества в плане безопасности для платформы Ethereum, делая ее невосприимчивой к DDoS (распределенным атакам типа «отказ в обслуживании», которые являются стратегией для многих хакеров).
