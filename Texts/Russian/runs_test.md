# Выполняет тест
## Что такое тест запуска?

Прогонный тест — это статистическая процедура, которая проверяет, является ли [строка](/data-analytics) [данных](/data-analytics) случайным образом полученной из определенного [распределения](/normaldistribution). Тест запусков анализирует возникновение похожих событий, разделенных событиями, которые отличаются друг от друга.

При инвестировании для инвесторов может быть важно провести тестирование, чтобы определить, генерируется ли набор данных, который они используют, случайным образом или на него влияет базовая переменная. Трейдеры, которые сосредотачиваются на техническом анализе, могут использовать тест прогонов, чтобы помочь проанализировать поведение [цены](/price-action) ценной бумаги.

## Понимание выполнения теста

Прогон — это серия возрастающих или убывающих значений, часто представленная на графике символами плюс (+) или минус (-). В статистике запуск тестов помогает определить [случайность данных](/random-factor-analysis),. обнаруживая любые переменные, которые могут повлиять на шаблоны данных.

Например, список действительно случайных однозначных чисел должен иметь только несколько экземпляров, в которых последовательность чисел численно возрастает. Однако во многих случаях трудно утверждать случайность данных, в которых есть тысячи последовательностей в строке данных. Таким образом, тест прогонов был создан как [объективный метод](/objective-probability) определения случайности.

## Типы запусков тестов

Тест запусков — это сокращенная версия полного названия: тест Вальда-Вольфовица, названный так в честь математиков Авраама Вальда и Джейкоба Вулфовица. Критерий Вальда-Вольфовица является [непараметрическим статистическим тестом](/nonparametric-statistics),. который означает, что анализируемые данные не должны соответствовать определенным предположениям или параметрам. Тест Вальда-Вольфовица можно использовать для проверки [гипотезы о том](/hypothesistesting),. что переменные в строке данных взаимно независимы.

Некоторые статистики считают, что другой тип теста серий — тест Колмогорова-Смирнова — является лучшим инструментом, чем тест Вальда-Вольфовица, для обнаружения различий между распределениями. Тест Колмогорова-Смирнова - это тип [теста согласия](/goodness-of-fit),. который демонстрирует, представляют ли проверяемые выборочные данные [нормальные модели распределения](/normaldistribution) или данные каким-то образом искажены. Тест устанавливает несоответствие между значениями выборочных данных и модели нормального распределения.

## Преимущества запуска теста

Модель серийных испытаний важна для определения того, действительно ли результат испытания является случайным, особенно в случаях, когда случайные данные, а не последовательные, имеют значение для последующих теорий и анализа. Рыночный тест может быть ценным инструментом для инвесторов, которые используют [технический анализ](/technicalanalysis) для принятия своих торговых решений. Эти трейдеры анализируют статистические тенденции, такие как движение цены и [объем](/volume),. чтобы определить потенциально прибыльные торговые возможности. Этим трейдерам важно понимать основные переменные, которые могут влиять на движение цены, и в этом может помочь тестирование прогонов.

Два эффективных способа, с помощью которых трейдеры могут использовать ранний тест, включают в себя:

1. Проверка случайности распределения путем взятия данных в заданном порядке и отметки плюсом (+) данные больше [медианы](/median),. а минусом (-) данные меньше медианы (числа равные медиане опущен.)

1. Проверка того, хорошо ли функция соответствует набору данных, путем пометки данных, превышающих значение функции, знаком +, а других данных — знаком -. Для этого использования тест прогонов, который учитывает знаки, но не расстояния, дополняет [тест хи-квадрат](/chi-square-statistic),. который учитывает расстояния, но не знаки.

## Особенности

- Прогонный тест — это статистический анализ, который помогает определить случайность данных, выявляя любые переменные, которые могут повлиять на шаблоны данных.

- Прогонный тест, также известный как прогонный тест Вальда-Вольфовица, был разработан математиками Авраамом Вальдом и Джейкобом Вулфовицем.

- Технические трейдеры могут использовать тест прогонов для анализа статистических тенденций и определения прибыльных торговых возможностей.

- Например, инвестор, заинтересованный в анализе движения цены конкретной акции, может провести тест на прогоне, чтобы получить представление о возможном будущем движении цены этой акции.
