# Идентификационный номер эмитента (ИИН)
## Что такое идентификационный номер эмитента (ИИН)?

Идентификационный номер эмитента (ИИН) представляет собой первые несколько цифр номера платежной карты, выпущенной [финансовым учреждением](/financialinstitution). Обычно это первые восемь цифр на кредитной, дебетовой или платежной карте другого типа.

Идентификационный номер эмитента уникален для эмитента и его партнерского сетевого провайдера. ИИН помогает определить сеть процессинга, используемую для транзакций по карте.

## Понимание идентификационных номеров эмитента

Первые шесть-восемь цифр на платежной карте, такой как кредитная или дебетовая карта, известны как идентификационный номер эмитента. ИИН используется только для идентификации карточной сети, а не держателя карты. Система нумерации позволяет идентифицировать кредитную, дебетовую или другую платежную карту как выпущенную через процессинговую сеть финансовым учреждением.

Каждый раз, когда карта используется в транзакции, процессор карты выполняет ряд шагов проверки, чтобы убедиться, что владелец карты авторизован для совершения транзакции и что на счету достаточно средств.

> По данным Международной организации по стандартизации (ISO), количество цифр ИИН увеличилось с шести до восьми, чтобы соответствовать растущему числу эмитентов карт.

>

Хотя номера платежных карт различаются по длине, обычно они состоят из 19 цифр. При выдаче клиенту платежной карты эмитент создает уникальный идентификатор, используемый для ведения учета и обработки транзакций.

## Общие идентификационные номера эмитента

Первые цифры платежной карты являются идентификационным номером эмитента. Эти цифры варьируются от одного до шести. Эмитенты установили уникальные идентификационные номера эмитента, по которым они признаны в отрасли.

Примеры идентификационных номеров эмитента включают следующее:

- [American Express](/american-express-card) : 34, 37

- Карта Discover: с 6011, с 622126 по 622925, с 624000 по 626999, с 628200 по 628899, 64, 65

- Mastercard: с 2221 по 2720, с 51 по 55

- Виза: 4

## ИИН по сравнению с номерами платежных карт

[основного номера счета](/primary-account-number-pan) клиента . Номер платежной карты — это дополнительный идентификатор, который также связан с учетной записью клиента. Каждый раз, когда клиенту выдается новый тип платежной карты, он будет иметь свой уникальный номер платежной карты.

Номер, указанный на лицевой стороне платежной карты, может содержать от восьми до 19 цифр, включая ИИН, и не является случайным номером. Они представляют собой код, который идентифицирует владельца карты и связывает его с финансовым учреждением, выпустившим карту. Первые несколько цифр представляют идентификационный номер эмитента. Остальные цифры на карте используются для идентификации держателя карты и привязки его к конкретному счету в финансовом учреждении.

При использовании карты в процессе транзакции проверяется как ИИН, так и номер платежной карты. Номер платежной карты позволяет процессинговым системам связываться с [финансовым учреждением](/financialinstitution),. чтобы убедиться в наличии средств на счете.

## Особенности

- У каждого эмитента кредитной карты будет свой уникальный ИИН, например, карты American Express будут начинаться с цифр «34» или «37».

- ИИН отображается как первые несколько цифр номера кредитной карты и используется для проверки подлинности и статуса карты.

- Идентификационные номера эмитента (ИИН) указывают, какая компания выпустила кредитную или дебетовую карту.
