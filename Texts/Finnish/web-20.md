# Web 2.0
## Mikä on Web 2.0?

Web 2.0 kuvaa Internetin nykytilaa, jossa on enemmän käyttäjien luomaa sisältöä ja käytettävyyttä loppukäyttäjille verrattuna sen aikaisempaan inkarnaatioon, Web 1.0:aan. Yleisesti ottaen Web 2.0 viittaa 2000-luvun Internet-sovelluksiin, jotka ovat muuttaneet digitaalista aikakautta [dotcom-kuplan jälkimainingeissa](/dotcom-bubble).

## Web 2.0:n ymmärtäminen

Web 2.0 ei viittaa mihinkään tiettyihin Internetin teknisiin päivityksiin. Se viittaa yksinkertaisesti muutokseen internetin käytössä 2000-luvulla. Uudella aikakaudella osallistujien kesken on korkeampi tiedon jakamisen ja keskinäisten yhteyksien taso. Tämän uuden version avulla käyttäjät voivat osallistua aktiivisesti kokemukseen sen sijaan, että he toimisivat passiivisina katsojina, jotka ottavat vastaan tietoja.

## Web 1.0 vs. Web 2.0

Web 1.0:aa käytetään kuvaamaan Internetin ensimmäistä vaihetta. Tässä vaiheessa sisällöntuottajia oli vähän; suurin osa Internetiä käyttävistä oli kuluttajia. Staattiset sivut olivat yleisempiä kuin dynaaminen [HTML](/html),. joka sisältää interaktiivisia ja animoituja verkkosivustoja tietyllä koodauksella tai kielellä. Tässä vaiheessa sisältö tuli palvelimen tiedostojärjestelmästä tietokannan hallintajärjestelmän sijaan. Käyttäjät pystyivät allekirjoittamaan online-vieraskirjoja, ja HTML-lomakkeita lähetettiin sähköpostitse.

Esimerkkejä Web 1.0:ksi luokitelluista Internet-sivustoista ovat Britannica Online, henkilökohtaiset verkkosivustot ja mp3.com. Yleensä nämä sivustot ovat staattisia ja niillä on rajoitetusti toimintoja ja joustavuutta.

Termi Web 2.0 otettiin käyttöön ensimmäisen kerran vuonna 1999, kun Internet kääntyi kohti järjestelmää, joka aktivoi käyttäjän aktiivisesti. Käyttäjiä kannustettiin tarjoamaan sisältöä sen sijaan, että he vain katsoisivat sitä. Ihmiset pystyivät nyt julkaisemaan artikkeleita ja kommentteja, ja käyttäjätilejä oli mahdollista luoda eri sivustoille, mikä lisäsi osallistumista. Web 2.0 synnytti myös verkkosovelluksia, itsejulkaisualustoja, kuten [WordPress](/wordpress-cms),. sekä [sosiaalisen median](/social-media) sivustoja.

Esimerkkejä Web 2.0 -sivustoista ovat Wikipedia, Facebook, Twitter ja erilaiset blogit, jotka kaikki ovat muuttaneet tapaa, jolla samoja tietoja jaetaan ja toimitetaan.

Internetin sosiaalinen puoli on muuttunut erityisen paljon; Yleisesti ottaen sosiaalisen median avulla käyttäjät voivat olla tekemisissä ja vuorovaikutuksessa toistensa kanssa jakamalla ajatuksia, näkökulmia ja mielipiteitä. Käyttäjät voivat merkitä, jakaa, twiitata ja tykätä.

## Web 2.0:n edut ja haitat

Teknologian kehitys on antanut käyttäjille mahdollisuuden jakaa ajatuksiaan ja mielipiteitään muiden kanssa, mikä on luonut uusia tapoja organisoida ja ottaa yhteyttä muihin ihmisiin sekä edistänyt yhteistyön lisääntymistä.

Mutta on monia haittoja Internetissä, joka toimii enemmän kuin avoin foorumi. Sosiaalisen median laajentumisen myötä olemme nähneet lisääntyneen verkkoväkittelyn, verkkokiusaamisen, [doxingin](/personally-identifiable-information-pii),. [identiteettivarkauksien](/identitytheft) ja muiden verkkorikosten määrän. Uhka on myös väärän tiedon leviäminen käyttäjien keskuudessa, joko avoimen lähdekoodin tiedonjakosivustojen kautta tai sosiaalisessa mediassa.

Kuten Web 1.0, Web 2.0 on toinen siirtymävaihe Internetin kehityksessä. Web 3.0:aa ennustetaan kutsuttavan semanttiseksi webiksi, koska se räätälöidään entistä intuitiivisemmaksi jokaisen käyttäjän tarpeisiin.

## Kohokohdat

- Internetin uudella aikakaudella osallistujien kesken on korkeampi tiedon jakaminen ja vuorovaikutus.

- Web 2.0 ei viittaa tiettyihin Internetin teknisiin päivityksiin; se viittaa Internetin käytön muutokseen.

- Web 2.0 kuvaa Internetin nykytilaa, jossa on enemmän käyttäjien luomaa sisältöä ja käytettävyyttä loppukäyttäjille verrattuna aikaisempaan inkarnaatioonsa Web 1.0:aan.
