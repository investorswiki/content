# Tumma Pilvipeite
## Mikä on Dark Cloud Cover?

Dark Cloud Cover on laskeva käänteinen [kynttilänjalkakuvio](/candlestick),. jossa alaspäin oleva kynttilä (tyypillisesti musta tai punainen) avautuu edellisen kynttilän (yleensä valkoinen tai vihreä) sulkemisen yläpuolelle ja sulkeutuu sitten kynttilän keskikohdan alapuolelle.

Kuvio on merkittävä, koska se osoittaa vauhdin siirtymisen ylhäältä alaspäin. Kuvio luodaan ylös kynttilällä, jota seuraa alaspäin oleva kynttilä. Kauppiaat odottavat hinnan jatkuvan alhaisempana seuraavalla (kolmannella) kynttilällä. Tätä kutsutaan [vahvistukseksi](/confirmation).

## Tumman pilven peittämisen ymmärtäminen

Dark Cloud Cover -kuvio sisältää suuren mustan kynttilän, joka muodostaa "tumman pilven" edellisen kynttilän päälle. Kuten [laskussa , ostajat](/bearishengulfingp) [nostavat](/bearishengulfingp) hintaa korkeammalle aukioloaikoina, mutta myyjät ottavat haltuunsa myöhemmin istunnon aikana ja painavat hintaa jyrkästi alemmas. Tämä siirtyminen ostamisesta myyntiin osoittaa, että hinta voi kääntyä laskuun.

Useimmat kauppiaat pitävät Dark Cloud Cover -kuviota hyödyllisenä vain, jos se tapahtuu [nousutrendin](/uptrend) tai yleisen hinnannousun seurauksena. Hintojen noustessa kuviosta tulee entistä tärkeämpi, jotta se merkitsee mahdollista siirtymistä alaspäin. Jos [hintatapahtuma on epätasainen](/choppymarket),. kuvio on vähemmän merkittävä, koska hinta pysyy todennäköisesti epätasaisena kuvion jälkeen.

Dark Cloud Cover -kuvion viisi kriteeriä ovat:

1. Olemassa oleva nouseva [nousutrendi](/uptrend).

1. Nouseva ( [nouseva](/bull) ) kynttilä tuossa nousutrendissä.

1. [Ero](/gap) seuraavana päivänä.

1. Rako ylös muuttuu alas ( [karhumainen](/bear) ) kynttilä.

1. Laskeva kynttilä sulkeutuu edellisen nousevan kynttilän keskipisteen alapuolelle.

Dark Cloud Cover -kuviolle on lisäksi tunnusomaista valkoiset ja mustat kynttilänjalat, joissa on pitkät [oikeat rungot](/realbody) ja suhteellisen lyhyet tai olemattomat [varjot](/shadow). Nämä ominaisuudet viittaavat siihen, että lasku oli sekä erittäin ratkaiseva että merkittävä hintaliikkeen kannalta. Kauppiaat voivat myös etsiä vahvistusta mallia noudattavan laskevan kynttilän muodossa. Hinnan odotetaan laskevan Dark Cloud Coverin jälkeen, joten jos se ei auta, kuvio saattaa [epäonnistua](/failedbreak).

Laskevan kynttilän sulkemista voidaan käyttää [pitkistä](/long) asennoista poistumiseen. Vaihtoehtoisesti kauppiaat voivat poistua seuraavana päivänä, jos hinta jatkaa laskuaan (malli vahvistettu). Jos [lyhennetään](/short) laskukynttilän sulkeutuessa tai seuraavalla jaksolla, [stop loss](/stop-lossorder) voidaan sijoittaa laskevan kynttilän korkeuden yläpuolelle.

Dark Cloud Cover -kuviolla ei ole [voittotavoitetta .](/profit-target) Kauppiaat käyttävät muita menetelmiä tai kynttilänjalkamalleja määrittääkseen, milloin heidän tulee poistua lyhyestä kaupasta Dark Cloud Coverin perusteella.

Kauppiaat voivat käyttää Dark Cloud Cover -kuviota muiden teknisten analyysimuotojen yhteydessä. Kauppiaat voivat esimerkiksi etsiä [suhteellista vahvuusindeksiä (RSI)](/rsi),. joka on suurempi kuin 70, mikä vahvistaa, että arvopaperi on yliostettu. Elinkeinonharjoittaja voi myös etsiä [hajoamista](/breakdown) keskeiseltä [tukitasolta](/support) Dark Cloud Cover -kuvion jälkeen merkkinä siitä, että [laskutrendi](/downtrend) saattaa olla tulossa.

Tässä esimerkissä Dark Cloud Cover tapahtuu, kun kolmatta nousevaa kynttilää seuraa laskeva kynttilä, joka avautuu korkeammalle ja sulkeutuu viimeisen nousevan kynttilän keskipisteen alapuolelle. Malli ennusti onnistuneesti laskusuhdanteen seuraavassa istunnossa, jossa hinta liikkui lähes seitsemän prosenttia alemmas. Tuo istunto antoi vahvistuksen.

Kauppiaat, jotka olivat pitkiä, saattoivat harkita poistumista lähellä laskevan kynttilän sulkeutumista tai seuraavana päivänä (vahvistuspäivä), jolloin hinta jatkoi laskuaan. Sijoittajat voivat myös tehdä lyhyitä positioita näissä tilanteissa.

Jos syötetään lyhyeksi, alkupysäytyshäviö voidaan sijoittaa laskevan kynttilän korkeuden yläpuolelle. Vahvistuspäivän jälkeen stop loss voidaan tässä tapauksessa pudottaa juuri vahvistuspäivän korkeimman yläpuolelle. Sijoittajat asettaisivat sitten alemman tuottotavoitteen tai jatkaisivat [suurimman tappionsa](/trailingstop) [perässä](/trailingstop),. jos hinta laskee edelleen.

## Kohokohdat

- Molempien kynttilöiden tulee olla suhteellisen suuria, ja niissä tulee osoittaa vahvasti kauppiaiden ja sijoittajien osallistuminen. Kun kuvio esiintyy pienten kynttilöiden kanssa, se on tyypillisesti vähemmän merkittävä.

- Dark Cloud Cover on kynttilänjalkakuvio, joka osoittaa vauhdin siirtymisen huonompaan suuntaan hinnannousun seurauksena.

- Kauppiaat tyypillisesti näkevät, näkyykö laskukynttilän jälkeisessä kynttilässä myös laskevia hintoja. Laskevan kynttilän jälkeistä hinnan laskua kutsutaan vahvistukseksi.

- Kuvio koostuu laskevasta kynttilästä, joka avautuu edellisen nousevan kynttilän yläpuolelle, mutta sulkeutuu sen jälkeen.
