# American Code for Information Interchange (ASCII)
## Mikä on American Code for Information Interchange (ASCII)?

American Code for Information Interchange, jota yleisesti kutsutaan ASCII:ksi, on tietokonekieli tekstille, josta tuli standardi tiedonsiirrossa tietokoneiden välillä vuonna 1963.

ASCII-järjestelmä kääntää aakkosnumeeriset merkit ja symbolit tietokoneella luettavaksi koodiksi. Alkuperäinen ASCII oli vain englanninkielinen. Nyt on olemassa kahdenlaisia ASCII-koodeja; vakiokoodi, joka käyttää seitsemän bitin koodausjärjestelmää, ja laajennettu koodi, joka käyttää kahdeksan bitin järjestelmää.

Internet-viestinnässä ASCII on vähitellen syrjäytynyt Unicode-standardilla, jota voidaan käyttää millä tahansa kielellä ja joka on yhteensopiva ASCII:n kanssa. ASCII:ta käytetään kuitenkin edelleen tekstin syöttämiseen henkilökohtaisissa tietokoneissa ja elektronisissa laitteissa.

Se lausutaan ASK-y.

## American Code for Information Interchange (ASCII) ymmärtäminen

ASCII:n varhaisin iteraatio löytyy sähkekoodista, jota Bell-datapalvelut käyttävät viestien toimittamiseen telekirjoittimille.

ASCII oli ensimmäinen suuri tietojenkäsittelyn merkkikoodausjärjestelmä, ja se otettiin käyttöön tietojenkäsittelyn standardina vuonna 1963 ja Internet-lähetyksen standardina vuonna 1969. Sitä käytettiin useiden vuosien ajan Wall Streetin kauppiaiden ja välittäjien käyttämissä tilausten kirjausjärjestelmissä.

Se on vähitellen korvattu Internetin standardina Unicode Worldwide Character Standardilla (Unicode). Koska Unicoden ensimmäiset 128 merkkiä ovat samat kuin ASCII:ssa, voidaan sanoa, että nämä kaksi järjestelmää ovat olemassa rinnakkain.

> Voit syöttää ASCII-symbolin asiakirjaan millä tahansa Windows-yhteensopivalla tietokoneella pitämällä ALT-näppäintä painettuna samalla, kun kirjoitat merkkikoodin numeron. Esimerkiksi, kun pidät ALT-näppäintä painettuna kirjoittaessasi 156, saat £, Ison-Britannian punnan merkin.

### Mitä ASCII sisältää

Ennen kuin ASCII otettiin käyttöön standardina, jokainen tietokonevalmistaja käytti omaa koodiaan, ja jotkut yritykset käyttivät useampaa kuin yhtä eri tuotteissa. Tietokoneet eivät pystyneet kommunikoimaan keskenään.

Vaikka American National Standards hyväksyi ASCII-standardin vuonna 1963, se otettiin kokonaan käyttöön edes Yhdysvalloissa vasta vuonna 1968, jolloin presidentti Lyndon B. Johnson valtuutti liittohallituksen hyväksymään sen varmistaakseen, että valtion tietokone- ja televiestintäteollisuus tietokoneet voisivat kommunikoida keskenään.

ASCII sisälsi alun perin 128 aakkosmerkkiä, numeroa ja symbolia ja perustui englannin kieleen. Laajennettuja lomakkeita lisättiin 1970-luvulla muiden kielten mukaiseksi.

Pian useita ASCII-versioita otettiin käyttöön. Lopulta se sisälsi 255 merkkiä. Tämä versio tunnetaan nimellä Extended ASCII.

Nykyään käyttäjät voivat syöttää ASCII- tai Unicode-symboleja, kuten tekijänoikeussymbolin, asiakirjoihin painamalla alt-näppäintä ja syöttämällä merkkikoodin numeronäppäimistöllä NUM-lukitusnäppäimen ollessa päällä.

##Kohokohdat

- ASCII on edelleen olemassa, mutta se on suurelta osin korvattu Unicodella, jota voidaan käyttää minkä tahansa kielen koodaamiseen.

- ASCII sisälsi alun perin vain 128 englanninkielistä kirjainta ja symbolia, mutta myöhemmin sitä laajennettiin lisäämään merkkejä, mukaan lukien muilla kielillä käytetyt.

- American Code for Information Interchange tai ASCII on merkkikoodausmuoto, joka on suunniteltu käytettäväksi tekstin siirtämisessä tietokoneiden välillä.

### Mikä on ASCII vs. Unicode?

Unicode voidaan nähdä ASCII:n yleisversiona. ASCII on loppujen lopuksi American Code for Information Interchange, ja sen ensimmäinen iteraatio sisälsi kielessä käytetyt englanninkieliset aakkoset ja symbolit. Myöhemmät versiot sisälsivät symboleja, joita käytetään monilla muilla kielillä. Unicode puolestaan alkaa samoilla 128 symbolilla, joita käytetään ASCII:ssä, mutta nykyiset versiot sisältävät 144 697 merkkiä.

### Mikä on ASCII?

American Standard Code for Information Interchange tai ASCII on merkkikoodausmuoto tekstin sähköiseen siirtoon. Jokaista merkkiä edustaa yksilöllinen numero. Ensimmäinen ASCII-versio sisälsi vain 128 merkkiä, jotka edustivat aakkosten kirjaimia, isoja ja pieniä kirjaimia sekä useita yleisesti käytettyjä symboleja, kuten pilkkua. Myöhemmissä versioissa ASCII laajennettiin 255 merkkiin, mukaan lukien lisäsymbolit, kuten Englannin punnan symboli (£) ja espanjankielisessä tekstissä käytetty ylösalaisin oleva kysymysmerkki (¿).

### Mitä ASCII-merkit ovat?

Laajennetussa ASCII-taulukossa merkkikoodit 0–31 ovat ohjauskoodeja, kuten tekstin alku ja askelpalautin.Merkkikoodit 32–127 ovat enimmäkseen aakkosten kirjaimia (isot ja pienet kirjaimet) ja symboleja, kuten tähti ja dollarimerkki. koodeja 128-255 on useita versioita, joista jokainen heijastaa lisäyksiä alkuperäiseen vain englanninkieliseen ASCII:hen, kuten käänteinen huutomerkki, jenimerkki ja Georgian pilkku.
