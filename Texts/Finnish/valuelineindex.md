# Arvorivin yhdistelmäindeksi
## Mikä on arvolinjan yhdistelmäindeksi?

Value Line Composite Index on osakeindeksi, joka sisältää noin 1 700 yritystä [NYSE :stä](/nyse),. [American Stock Exchangesta](/amex),. Nasdaqista, Torontosta ja OTC-markkinoista. Value Line Composite -indeksillä on kaksi muotoa: Value Line Geometric Composite Index (alkuperäinen yhtä painotettu indeksi) ja Value Line Aritmetic Composite Index (indeksi, joka heijastaa muutoksia, jos salkussa on yhtä paljon osakkeita.) Nämä indeksit julkaistaan yleensä Value Line Investment Surveyssa, jonka on luonut perustaja Arnold Bernhard, ja Value Line Inc :n toimitusjohtaja

## Arvorivin yhdistelmäindeksin ymmärtäminen

"Arvolinja", jossa indeksi saa kaimansa, viittaa kassavirran monikerrokseen, jonka Bernhard asettaisi hintakaavion päälle normalisoidakseen eri yritysten arvon. Value Line on yksi arvostetuimmista sijoitustutkimusyhtiöistä. Sen suorituskykyennätys on ollut erittäin vahva. Itse asiassa yrityksen mallisalkut ovat yleensä lyöneet markkinat [pitkällä aikavälillä](/longrun).

Value Line Composite -indeksi koostuu samoista yrityksistä kuin The Value Line Investment Survey, lukuun ottamatta suljettuja rahastoja .

Value Line -indeksissä olevien yritysten määrä vaihtelee tekijöiden mukaan, kuten yritysten lisääminen tai poistaminen pörssissä, sulautumiset, yrityskaupat, konkurssit ja Value Line -indeksin kattavuuspäätökset. Value Linen päätökset siitä, mitkä yhtiöt sisällytetään mukaan, tehdään tarkoituksena luoda laaja edustus Pohjois-Amerikan osakemarkkinoista. Lisäksi missä tahansa pörssissä listattujen yritysten määrä voi vaihdella, koska yritys voi siirtyä pörssistä toiseen tai tulla lisätyksi tai poistumaan listalta. Yritysten listaaminen tai liikkuminen pörssissä ei kuitenkaan ole Value Line Index -metodologiassa tekijöitä riippumatta siitä, käytetäänkö geometrisia vai aritmeettisia laskelmia.

> ### 1961

> Vuosi, jolloin alkuperäinen Value Line Geometric Composite -indeksi lanseerattiin.

>

## Arvoviivan geometrinen yhdistelmäindeksi

Tämä on alkuperäinen indeksi, joka otettiin käyttöön 30. kesäkuuta 1961. Se on yhtä painotettu indeksi, joka käyttää geometristä keskiarvoa. Value Line Geometric Composite -indeksin päivittäinen hinnanmuutos saadaan kertomalla kunkin osakkeen päätöskurssin suhde sen edelliseen päätöskurssiin ja nostamalla tämä tulos osakkeiden kokonaismäärän käänteisarvoon .

## Arvorivin aritmeettinen yhdistelmäindeksi

Tämä indeksi perustettiin 1. helmikuuta 1988 käyttämällä aritmeettista keskiarvoa jäljittelemään paremmin indeksin muutosta, jos omistat yhtä suuren osakesalkun.﻿Valuerivien aritmeettisen yhdistelmäindeksin päivittäinen hinnanmuutos lasketaan lisäämällä kaikkien osakkeiden päivittäinen prosentuaalinen muutos ja jakamalla sitten osakkeiden kokonaismäärällä

## Kohokohdat

- Indeksillä on kaksi muotoa: arvoviivan geometrinen yhdistelmäindeksi ja arvoviivan aritmeettinen yhdistelmäindeksi.

- Geometrinen yhdistelmäindeksi on tasapainotettu, käyttää geometristä keskiarvoa ja sen päivittäinen muutos on lähinnä osakekurssin mediaanimuutosta.

- Aritmeettinen yhdistelmäindeksi käyttää aritmeettista keskiarvoa, jolloin indeksin päivittäinen muutos heijastelee salkkua, joka koostuu yhtä suurista osakkeista .

- Value Line Composite -indeksi sisältää noin 1 700 osakkeen yhdistelmän tärkeimmistä Pohjois-Amerikan markkinaindekseistä .
