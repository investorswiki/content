# Visual Basic for Applications (VBA)
Visual Basic for Applications (VBA) on osa Microsoft Corporationin (NASDAQ: MSFT) vanhaa Visual Basic -ohjelmistoa, jonka Microsoft on rakentanut auttamaan ohjelmien kirjoittamisessa Windows-käyttöjärjestelmää varten. Visual Basic for Applications toimii sisäisenä ohjelmointikielenä Microsoft Office (MS Office, Office) sovelluksissa, kuten Access, Excel, PowerPoint, Publisher, Word ja Visio.

VBA:n avulla käyttäjät voivat muokata MS Office -isäntäsovelluksissa normaalisti käytettävissä olevia ominaisuuksia pidemmälle – VBA ei ole itsenäinen ohjelma – manipuloimalla graafisen käyttöliittymän (GUI) ominaisuuksia, kuten työkalurivejä ja valikoita, valintaikkunoita ja lomakkeita. Voit käyttää VBA:ta käyttäjän määrittämien funktioiden (UDF) luomiseen, Windowsin [sovellusliittymien (API) käyttämiseen](/application-programming-interface) ja tiettyjen tietokoneprosessien ja laskelmien automatisointiin.

## Lisätietoja Visual Basic for Applicationsista

VBA on tapahtumaohjattu työkalu, mikä tarkoittaa, että voit käyttää sitä käskemään tietokonetta käynnistämään toiminnon tai toimintosarjan. Voit tehdä tämän rakentamalla mukautettuja makroja – lyhenne sanoista makroohjeet – kirjoittamalla komennot muokkausmoduuliin.

Makro on pohjimmiltaan merkkijono, jonka syöttäminen johtaa toiseen merkkijonoon (sen tulos), joka suorittaa tiettyjä laskentatehtäviä. Sinun ei tarvitse ostaa VBA-ohjelmistoa, koska VBA on Visual Basicin versio, joka toimitetaan Microsoft Office 2010:n mukana.

> Visual Basic for Applications on ainoa VB 6:n versio, jota Microsoft edelleen myy ja tukee, ja vain Office-ohjelmien sisäisenä osana.

>

## Kuinka VBA:ta käytetään?

### Useimmille meistä

MS Office -sovelluksissa Visual Basic for Applications antaa käyttäjille mahdollisuuden suorittaa lukemattomia toimintoja, jotka ylittävät yksinkertaisen tekstinkäsittelyn ja taulukkolaskentatoiminnot. Tavalliselle käyttäjälle VBA auttaa vähentämään toistuvia päivittäisiä tehtäviä makrojen avulla.

Makrot voivat automatisoida lähes minkä tahansa tehtävän, kuten räätälöityjen kaavioiden ja raporttien luomisen sekä tekstin- ja tietojenkäsittelytoimintojen suorittamisen. Voit esimerkiksi kirjoittaa makron, joka saa Excelin luomaan yhdellä napsautuksella kokonaisen taseen [laskentataulukon kirjanpitomerkintöjen sarjasta](/accounting).

### Tietokoneammattilaisille

Ohjelmoijat käyttävät makroja kuitenkin monimutkaisemmilla tavoilla – kuten monistamalla suuria koodinpätkiä, yhdistämällä olemassa olevia ohjelmatoimintoja ja suunnittelemalla tiettyjä kieliä.

### Yrityksille ja organisaatioille

VBA voi toimia myös ulkoisissa eli ei-Microsoftin asetuksissa käyttämällä COM-liitäntää**,**, joka mahdollistaa komentojen vuorovaikutuksen yli tietokonerajojen. Monet yritykset ovat ottaneet käyttöön VBA:ta omissa sovelluksissaan, sekä yksityisissä että kaupallisissa sovelluksissa, mukaan lukien AutoCAD, ArcGIS, CATIA, Corel, raw ja SolidWorks.

Mikä tahansa yritys voi käyttää VBA:ta mukauttaakseen Exceliä ainutlaatuiseen tarkoitukseen, kuten arvioidakseen, kuinka kauan kestää ansaita miljoona dollaria [sijoitussalkussa](/portfolio) tietyn [koron](/interestrate) ja muiden tekijöiden, kuten eläkkeelle jäämiseen kuluvien vuosien, perusteella.

## Esimerkki VBA:sta rahoitusmarkkinoilla

### Kuinka VBA on kaikkialla rahoituksessa

Rahoitus on pohjimmiltaan valtavien tietomäärien manipuloimista; näin ollen VBA on endeeminen rahoituspalvelualalla. Jos työskentelet rahoitusalalla, VBA toimii todennäköisesti päivittäin käyttämissäsi sovelluksissa, olitpa tietoinen siitä tai et. Jotkut alan työt edellyttävät VBA:n aiempaa tuntemusta, ja jotkut eivät.

Joka tapauksessa, jos haluat jatkaa uraa rahoitusalalla, on tärkeää, että tunnet verkkotunnuksesi uusimmat teknologiatrendit ja kuinka voit käyttää automaatiota päivittäisessä toiminnassasi. Koska VBA on käyttäjän intuitiivinen, ne, joilla on vähän tai ei ollenkaan tietokoneohjelmointia, voivat oppia sen helposti.

### Tapoja, joilla rahoitusalan ammattilaiset käyttävät VBA:ta

- Makrot antavat rahoitusalan ammattilaisille – kirjanpitäjille, liikepankkiireille, [investointipankkiireille](/investmentbanker),. tutkimusanalyytikoille, myyntimiehille, kauppiaille, salkunhoitajille, virkailijoille tai järjestelmänvalvojille – analysoida ja muokata valtavia tietomääriä nopeasti.

- Excelin VBA:n avulla voit luoda ja ylläpitää monimutkaisia kaupankäynti-, hinnoittelu- ja riskienhallintamalleja, ennakoida myyntiä ja tuottoa sekä luoda taloudellisia tunnuslukuja.

- Visual Basic for Applications -sovelluksella voit luoda erilaisia salkunhallinta- ja [sijoitusskenaarioita](/scenario_analysis).

- Voit myös käyttää VBA:ta asiakkaiden nimiluetteloiden tai muun sisällön tuottamiseen; luoda laskuja, lomakkeita ja kaavioita; analysoida tieteellistä tietoa ja hallita datan näyttöä budjetteja ja ennusteita varten.

## Kohokohdat

- VBA-toiminnot MS Office -sovelluksissa; se ei ole erillinen tuote.

- VBA:lla voit luoda makroja automatisoidaksesi toistuvia tekstin- ja tietojenkäsittelytoimintoja sekä luoda mukautettuja lomakkeita, kaavioita ja raportteja.

- Visual Basic for Applications on Microsoftin kehittämä ja omistama tietokoneohjelmointikieli.
