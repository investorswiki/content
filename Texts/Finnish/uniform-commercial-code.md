# Uniform Commercial Code (UCC)
## Mikä on Uniform Commercial Code (UCC)?

Uniform Commercial Code (UCC) on [standardoitu](/standardization) joukko liiketoimia koskevia lakeja ja määräyksiä. UCC-koodi perustettiin vuonna 1953, koska yritysten oli yhä vaikeampaa käydä kauppaa osavaltioiden eri rajojen yli valtion eri lakien vuoksi.

Uniform Commercial Code (UCC) on tärkeä, koska se auttaa eri valtioiden yrityksiä asioimaan keskenään tarjoamalla standardin oikeudellisen ja sopimuskehyksen. Useimmat Yhdysvaltain osavaltiot ovat hyväksyneet UCC-lait. Vaikka osavaltioittain on joitain pieniä eroja, UCC-koodi koostuu yhdeksästä erillisestä artiklasta. UCC:n artiklat säätelevät erilaisia liiketoimia, mukaan lukien pankkitoiminta ja lainat.

## Kuinka yhtenäinen kauppakoodi (UCC) toimii

[UCC](/personalproperty) ) lait säätelevät [henkilökohtaisen omaisuuden myyntiä](/personalproperty) ja monia muita liiketoimia. Jos olet joskus ostanut yrityksen tai ajoneuvon aiemmin, olet todennäköisesti allekirjoittanut UCC-1-lausunnon. Omistusoikeus säilyy lainanantajan hallussa, kunnes laina on maksettu.

Uniform Commercial Code (UCC) -linjaukset keskittyvät pitkälti pienyritysten ja [yrittäjien toimintaan](/entrepreneur). Osa tarkoituksena on selvittää sekaannusta siitä, kuinka kukin valtio voisi erikseen säännellä tällaisia toimintoja.

> Vaikka UCC-säännöstö säätelee henkilökohtaista omaisuutta koskevia kauppasuhteita, se ei säätele kiinteistöä, kuten maata tai maa-alueisiin liittyviä rakenteita.

>

UCC-koodi asettaa standardit sekkien ja muiden yritystodistusten käsittelylle. Usein sitä sovelletaan pankin vakuuttamiin kiinteistöihin, joissa omistusoikeus säilyy, kunnes lainanottaja maksaa rahoituksen loppuosan.

Yritysten, jotka suorittavat liiketoimia kotivaltionsa ulkopuolella, on noudatettava sovellettavaa UCC-lakia, mukaan lukien laitteiden [vuokraaminen](/lease),. tavaroiden myynti, lainanotto ja sopimusten solmiminen.

## Uniform Commercial Code (UCC) -artikkelit

Alla on yleiskatsaus siitä, mitä Uniform Commercial Coden (UCC) yhdeksän eri artiklaa koskevat:

**Artikla 1:** Yleisissä määräyksissä vahvistetaan määritelmät ja tietyt parametrit Uniform Commercial Code (UCC) -säännöstön soveltamiselle. Se on viimeksi päivitetty vuonna 2001.

**Artikla 2/2a:** Tavaroiden myynti, ei kiinteistö- ja palvelusopimuksia. 2 a artikla kattaa henkilökohtaisen omaisuuden vuokrasopimukset.

**Artikla 3:** Sekit, vekselit ja muut siirtokelpoiset välineet, kuten setelit (lupaus maksaa rahaa). Kohde katsotaan siirtokelpoiseksi, jos se voidaan siirtää toiselle henkilölle ja se on edelleen täytäntöönpanokelpoinen alkuperäistä maksajaa vastaan.

**Artikla 4/4a:** Pankkitalletukset ja perintä, joka kattaa sekkien käsittelyä ja automaattista pankkien välistä perintää koskevat säännöt. 4 a artiklassa keskitytään varojen siirtoihin.

**Artikla 5**: Remburssit, joiden pankki on myöntänyt kaupan helpottamiseksi.

**Artikla 6:** Omaisuuden joukkomyynti, huutokaupat ja likvidaatiot. Useimmat osavaltiot uskovat tämän artiklan olevan vanhentunut, ja Uniform Law Commission (ULC) on suositellut kumoamista, jonka useimmat osavaltiot ovat hyväksyneet.

**Artikla 7:** Omistusasiakirjat, mukaan lukien varastokuitit, irtotavaramyynti ja konossementit (BoL).

**Artikla 8:** Sijoitusarvopaperit; erityisesti arvopapereiden hallussapito välittäjien kautta.

**Artikla 9:** Henkilökohtaista omaisuutta, maatalouden panttioikeutta, velkakirjoja, lähetyksiä ja vakuuksia koskevat vakuudelliset liiketoimet.

Uniform Commercial Code (UCC) käy läpi usein tarkistuksia, jotka koskevat tiettyjä artikkeleita.

## Uniform Commercial Code (UCC) -historia

Uniform Commercial Code (UCC) ei perustettu kongressin kautta. Sen ovat luoneet yksityiset organisaatiot, joihin kuuluvat Uniform Law Commission (ULC), joka tunnetaan myös nimellä National Conference of Commissioners on Uniform State Laws (NCCUSL), ja American Law Institute (ALI).

ULC perustettiin vuonna 1892 yhtenäisen [kauppaoikeuden luomiseksi](/commercial-code). Järjestö sääti useita lakeja perustamisestaan 1950-luvulle asti. 1950-luvulla ULC koonnut yhdessä ALI:n kanssa kaikki kaupalliset lait yhdeksi kaupallisiksi säännöstöiksi, joita osavaltiot voivat noudattaa.

UCC esiteltiin osavaltioille vuonna 1951, ja Pennsylvania oli ensimmäinen, joka otti UCC:n käyttöön vuonna 1953, ja muut osavaltiot ottivat sen käyttöön ajan myötä. Louisiana on nyt ainoa osavaltio, joka ei ole täysin ratifioinut koodia, vaikka se on hyväksynyt osan siitä.

## Erityisiä huomioita

Jokaisella osavaltiolla on mahdollisuus hyväksyä koodi sellaisena kuin se on kirjoitettu tai hyväksyä ja muuttaa sen määräyksiä.

Louisiana ei hyväksynyt Uniform Commercial Coden (UCC) artiklaa 2 sellaisena kuin se on kirjoitettu. Valtio ei myöskään hyväksynyt pykälää 2A, joka kattaa sellaisen henkilökohtaisen omaisuuden vuokraamisen, jota ei pidetä kiinteistönä.

Kalifornia on myös tehnyt joitain muutoksia toteuttamalla oman versionsa UCC-laeista. Kiinteistösopimukset ovat yksi poikkeuksista Kalifornian UCC:n hyväksymiseen. [Kiinteistön , kuten varaston,](/realestate) ostoa koskevat lait eivät ole Kalifornian kaupallisessa säännöstössä, vaan osavaltion kiinteistöjä koskevat lait ja määräykset.

Kalifornian palvelusopimukset eivät myöskään kuulu UCC:n piiriin. Palvelusopimukset sisältävät autokorjaukset, maalaustyöt, sisustustyöt ja niin edelleen. Nämä toiminnot kuuluvat valtion vakuutuslakien piiriin.

## Kohokohdat

- Uniform Commercial Code (UCC) on joukko yrityslakeja, jotka säätelevät rahoitussopimuksia ja liiketoimia, joita käytetään eri osavaltioissa.

- Useimmat osavaltiot ovat omaksuneet UCC-koodin täysin, ja muut ovat mukauttaneet sitä hieman.

- Yritysten, jotka suorittavat liiketoimia kotivaltionsa ulkopuolella, on noudatettava Uniform Commercial Code (UCC) -sääntöjä.

- UCC-koodi koostuu yhdeksästä erillisestä artiklasta, joista jokainen kattaa erilliset pankki- ja lainojen näkökohdat.

- Uniform Commercial Code (UCC) ei ole perustettu kongressin kautta, vaan yksityisten organisaatioiden toimesta.

## UKK

### Kuinka UCC:n panttioikeus toimii?

UCC:n panttioikeus, joka tunnetaan myös nimellä UCC-hakemus, on lomake, jonka velkoja jättää ilmoittaakseen, että heillä on intressi velallisen omaisuuteen, olipa kyse sitten henkilökohtaista tai liiketoiminnasta. UCC:n panttioikeuden yleisenä tarkoituksena on antaa velkojalle mahdollisuus vaatia rahoituksesta vakuuksia velalliselle. Luotonantajalla on oikeus panttioikeudessa olevaan omaisuuteen, kunnes velallinen on maksanut taloudellisen velvoitteensa.

### Ketä yhtenäinen kauppalaki suojaa?

Uniform Commercial Code (UCC) perustettiin suojelemaan kaikkia liiketoimintaa harjoittavia henkilöitä. Se luotiin osavaltioiden välisen kaupan standardoimiseksi riippumatta siitä, tapahtuuko kauppaa yksilöiden tai yritysten välillä.

### Mikä on UCC-ilmoitusmaksu?

UCC-hakemusmaksu on 40 dollaria paperihakemuksista ja 20 dollaria sähköisistä hakemuksista New Yorkissa.

### Mitä yhtenäisen kauppalain 2 ja 2A artikla kattaa?

Yhtenäisen kauppalain 2 § kattaa tavaroiden myynnin, lukuun ottamatta kiinteistö- ja palvelusopimuksia, kun taas 2 a artikla koskee henkilökohtaisen omaisuuden vuokraamista.
