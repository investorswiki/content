# Skyen
## Hvad er cloud computing?

Cloud computing er levering af forskellige tjenester via internettet. Disse ressourcer omfatter værktøjer og applikationer som datalagring, servere, databaser, netværk og software.

I stedet for at opbevare filer på en proprietær harddisk eller lokal lagerenhed, gør [skybaseret lagring](/cloud-storage) det muligt at gemme dem i en ekstern database. Så længe en elektronisk enhed har adgang til internettet, har den adgang til dataene og softwareprogrammerne til at køre den.

Cloud computing er en populær mulighed for mennesker og virksomheder af en række årsager, herunder omkostningsbesparelser, øget produktivitet, hastighed og effektivitet, ydeevne og sikkerhed.

## Forstå Cloud Computing

Cloud computing er navngivet som sådan, fordi den information, der tilgås, findes eksternt i skyen eller et virtuelt rum. Virksomheder, der leverer cloud-tjenester, gør det muligt for brugere at gemme filer og applikationer på fjernservere og derefter få adgang til alle data via internettet. Dette betyder, at brugeren ikke behøver at være et bestemt sted for at få adgang til det, hvilket giver brugeren mulighed for at arbejde eksternt.

Cloud computing tager alle de tunge løft, der er involveret i at knuse og behandle data væk fra den enhed, du bærer rundt på eller sidder og arbejder ved. Det flytter også alt det arbejde til enorme computerklynger langt væk i cyberspace. Internettet bliver til skyen, og voilà – dine data, dit arbejde og dine applikationer er tilgængelige fra enhver enhed, som du kan oprette forbindelse til internettet med, hvor som helst i verden.

Cloud computing kan være både offentlig og privat. Offentlige cloud-tjenester leverer deres tjenester over internettet mod et gebyr. Private cloud-tjenester leverer derimod kun tjenester til et vist antal mennesker. Disse tjenester er et system af netværk, der leverer hostede tjenester. Der er også en hybrid mulighed, som kombinerer elementer fra både offentlige og private tjenester.

## Typer af skytjenester

Uanset typen af tjeneste giver cloud computing-tjenester brugerne en række funktioner, herunder:

- E-mail

- Lagring, backup og datahentning

- Oprettelse og test af apps

- Analyse af data

- Streaming af lyd og video

- Levering af software on demand

Cloud computing er stadig en ret ny tjeneste, men den bliver brugt af en række forskellige organisationer fra store virksomheder til små virksomheder, nonprofitorganisationer til offentlige myndigheder og endda individuelle forbrugere.

## Implementeringsmodeller

Der er forskellige typer skyer, som hver især er forskellige fra hinanden. Offentlige skyer leverer deres tjenester på servere og lagring på internettet. Disse drives af tredjepartsvirksomheder, som håndterer og kontrollerer al hardware, software og den generelle infrastruktur. Kunder får adgang til tjenester via konti, som næsten alle kan få adgang til.

Private skyer er reserveret til specifikke kunder, normalt én virksomhed eller organisation. Firmaets dataservicecenter kan være vært for cloud computing-tjenesten. Mange private cloud computing-tjenester leveres på et privat netværk.

Hybride skyer er, som navnet antyder, en kombination af både offentlige og private tjenester. Denne type model giver brugeren mere fleksibilitet og hjælper med at optimere brugerens infrastruktur og sikkerhed.

> Nyere former for cloud computing-tjenester omfatter community cloud, big data cloud og multicloud.

>

## Typer af cloud computing

Cloud computing er ikke et enkelt stykke teknologi som en mikrochip eller en mobiltelefon. Det er snarere et system, der primært består af tre tjenester: [software-as-a-service](/software-as-a-service-saas) (SaaS), infrastruktur-as-a-service (IaaS) og platform-as-a-service (PaaS).

1. **Software-as-a-service (SaaS)** involverer licensudstedelse af en softwareapplikation til kunder. Licenser leveres typisk gennem en pay-as-you-go-model eller on-demand. Denne type system kan findes i Microsoft Office's 365.

1. **Infrastructure-as-a-service (IaaS)** involverer en metode til at levere alt fra operativsystemer til servere og lagring via IP-baseret tilslutning som en del af en on-demand-tjeneste. Klienter kan undgå behovet for at købe software eller servere og i stedet skaffe disse ressourcer i en [outsourcet](/outsourcing) on-demand-tjeneste. Populære eksempler på IaaS-systemet omfatter IBM Cloud og Microsoft Azure.

1. **Platform-as-a-service (PaaS)** betragtes som det mest komplekse af de tre lag af cloud-baseret databehandling. PaaS deler nogle ligheder med SaaS, den primære forskel er, at i stedet for at levere software online, er det faktisk en platform til at skabe software, der leveres via internettet. Denne model inkluderer platforme som Salesforce.com og Heroku.

## Fordele ved Cloud Computing

Cloud-baseret software tilbyder virksomheder fra alle sektorer en række fordele, herunder muligheden for at bruge software fra enhver enhed enten via en native app eller en browser. Som et resultat kan brugere overføre deres filer og indstillinger til andre enheder på en fuldstændig problemfri måde.

Cloud computing er langt mere end blot at få adgang til filer på flere enheder. Takket være cloud computing-tjenester kan brugere tjekke deres e-mail på enhver computer og endda gemme filer ved hjælp af tjenester som Dropbox og Google Drive. Cloud computing-tjenester gør det også muligt for brugere at [sikkerhedskopiere](/back-up) deres musik, filer og billeder, hvilket sikrer, at disse filer er umiddelbart tilgængelige i tilfælde af en harddisknedbrud.

Det giver også store virksomheder et enormt omkostningsbesparelsespotentiale. Før skyen blev et levedygtigt alternativ, var virksomhederne forpligtet til at købe, konstruere og vedligeholde dyrt [inden for](/information-management-technology-imt) [formation management teknologi](/information-management-technology-imt) og infrastruktur. Virksomheder kan bytte dyre servercentre og it-afdelinger ud med hurtige internetforbindelser, hvor medarbejderne interagerer med skyen online for at udføre deres opgaver.

Skystrukturen giver enkeltpersoner mulighed for at spare lagerplads på deres stationære eller bærbare computere. Det giver også brugere mulighed for at [opgradere](/upgrade) software hurtigere, fordi softwarevirksomheder kan tilbyde deres produkter via internettet i stedet for gennem mere traditionelle, håndgribelige metoder, der involverer diske eller flashdrev. For eksempel kan Adobe-kunder få adgang til applikationer i deres Creative Cloud gennem et internetbaseret abonnement. Dette giver brugerne mulighed for nemt at downloade nye versioner og rettelser til deres programmer.

## Ulemper ved skyen

Med al den hastighed, effektivitet og innovationer, der følger med cloud computing, er der naturligvis risici.

Sikkerhed har altid været et stort problem i skyen, især når det kommer til følsomme lægejournaler og økonomiske oplysninger. Mens regler tvinger cloud computing-tjenester til at styrke deres sikkerheds- og overholdelsesforanstaltninger, er det stadig et vedvarende problem. Kryptering beskytter vital information, men hvis denne krypteringsnøgle går tabt, forsvinder dataene.

Servere, der vedligeholdes af cloud computing-virksomheder, kan også blive ofre for naturkatastrofer, interne fejl og strømafbrydelser. Den geografiske rækkevidde af cloud computing skærer begge veje: Et blackout i Californien kan lamme brugere i New York, og et firma i Texas kan miste sine data, hvis noget får dens Maine-baserede udbyder til at gå ned.

Som med enhver teknologi er der en læringskurve for både medarbejdere og ledere. Men med mange enkeltpersoner, der får adgang til og manipulerer information gennem en enkelt portal, kan utilsigtede fejl overføres på tværs af et helt system.

## Forretningens verden

Virksomheder kan anvende cloud computing på forskellige måder. Nogle brugere vedligeholder alle apps og data i skyen, mens andre bruger en hybridmodel, der holder visse apps og data på private servere og andre i skyen.

Når det kommer til at levere tjenester, omfatter de store aktører i virksomhedens computersfære:

- Google Cloud

- Amazon Web Services (AWS)

- Microsoft Azure

- IBM Cloud

- Alibaba Cloud

Amazon Web Services er 100 % offentlige og inkluderer en outsourcet pay-as-you-go-model. Når du er på platformen, kan du tilmelde dig apps og yderligere tjenester. Microsoft Azure giver klienter mulighed for at opbevare nogle data på deres egne websteder. I mellemtiden er Alibaba Cloud et datterselskab af Alibaba Group.

## Højdepunkter

- Skysikkerhed er blevet et stadig vigtigere felt inden for IT.

- Cloud computing er levering af forskellige tjenester via internettet, herunder datalagring, servere, databaser, netværk og software.

- Cloud-baseret lagring gør det muligt at gemme filer til en ekstern database og hente dem efter behov.

- Tjenester kan være både offentlige og private - offentlige tjenester leveres online mod et gebyr, mens private tjenester hostes på et netværk til specifikke klienter.

- Cloud storage er blevet mere og mere populært blandt enkeltpersoner, der har brug for større lagerplads, og for virksomheder, der søger en effektiv off-site backup-løsning.

## Ofte stillede spørgsmål

### Hvad er et eksempel på cloud computing?

I dag er der flere eksempler på cloud computing-applikationer, der bruges af både virksomheder og enkeltpersoner. En type cloud-tjeneste ville være streamingplatforme til lyd eller video, hvor de faktiske mediefiler gemmes eksternt. En anden ville være datalagringsplatforme som Google Drive, Dropbox, OneDrive eller Box.

### Er cloud computing sikker?

Fordi software og data gemmes eksternt i cloud computing, er datasikkerhed og platformsikkerhed et stort problem. [Skysikkerhed](/cloud-security) refererer til de foranstaltninger, der træffes for at beskytte digitale aktiver og data, der er lagret på skybaserede tjenester. Foranstaltninger til at beskytte disse data omfatter blandt andet to-faktor-autorisation (2FA), brug af VPN'er, sikkerhedstokens, datakryptering og firewall-tjenester.

### Hvad er hovedtyperne af cloud computing?

De vigtigste typer af cloud computing-tjenester omfatter Infrastructure-as-a-Service (IaaS), Platforms-as-a-Service (PaaS) og Software-as-a-Service (SaaS).- IaaS leverer IT-infrastruktur til slut- brugere via internettet og er almindeligvis forbundet med serverløs computing.- PaaS serverer både software og hardware til slutbrugere, som generelt er softwareudviklere. PaaS giver brugeren mulighed for at udvikle, køre og administrere deres egne apps uden at skulle bygge og vedligeholde infrastrukturen.- SaaS er en softwarelicensmodel, som giver adgang til software på abonnementsbasis ved hjælp af eksterne servere uden at skulle downloade og installere dem lokalt.
