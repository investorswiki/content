# McClellan Summation Index
## Hvad er McClellan Summation Index

McClellan Summation Index er en langsigtet version af [McClellan Oscillator](/mcclellanoscillator),. som er en markedsbreddeindikator baseret på aktiefremgang og -fald. Sherman og Marian McClellan skabte og udviklede McClellan Summation Index.

Fortolkningen svarer til den for McClellan Oscillator, bortset fra at den er mere egnet til mellemstore tendenser og relaterede vendinger. McClellan Summation Index kan beregnes som summen af alle de daglige værdier af McClellan Oscillatoren.

## Forstå McClellan Summation Index

McClellan Summation Index bruges i teknisk analyse og kan bruges til at identificere bullish eller bearish bias, såvel som styrken af tendensen. Det er en anden måde at kvantificere bevægelserne i markedet på end at se på prisniveauerne for de forskellige indekser, såsom [S&P 500](/sp500) og [Dow Jones Industrial Average](/djia).

McClellan-summen har flere fortolkninger og betragtes som neutral ved en aflæsning på +1.000. I løbet af 1960'erne holdt McClellan Summation Index sig generelt inden for grænserne af 0 og +2.000; dog har udvidelsen af antallet af aktier handlet på NYSE resulteret i en udvidelse af tærsklerne for overkøbt og oversolgt korridor.

Nogle tommelfingerregler for McClellan Summation Index inkluderer følgende:

- Se efter store bunde under -1.300

- Se efter store toppe med en divergens over +1.600

- Begyndelsen af store tyreløb er nogle gange angivet, når McClellan Summation Index krydser over +1.900 efter at have flyttet +3.600 point fra dets tidligere laveste (f.eks. flytter McClellan Summation Index fra -1.550 til +1.950.
