# Deep Web
Det dybe web er den del af World Wide Web (WWW), der er noget skjult. Den indeholder sider, der ikke er indekseret af traditionelle websøgemaskiner - såsom Google, Bing og DuckDuckGo.

Det dybe spind er det modsatte udtryk for "overfladevævet". Som sådan kan det også blive omtalt som "skjult web" eller "usynligt web."

## Ud over overfladen

Det dybe web står for over 99 % af indholdet på nettet. Det er en del af WWW, der er anderledes end det såkaldte "overfladeweb", som er den synlige del af nettet. Overfladebanen er det lettest tilgængelige lag. Det inkluderer alle indekser (eller søgbare) sider.

Selvom mængden af indhold, der kan findes gennem søgemaskiner, er enorm, indeholder overfladenettet meget mindre information end det dybe web. En undersøgelse offentliggjort i 2001 indikerede, at mindre end 0,05% af alt indhold er tilgængeligt i overfladenettet, mens de andre ~99,95% findes i det dybe net.

## Hvad er Deep Web?

Per definition omfatter det dybe web både sider, der er uklare, eller som kun kan tilgås via en specifik godkendelsesmetode. At være obskur betyder, at de kræver en direkte URL for at blive fundet (at søge på Google virker ikke). Alligevel falder størstedelen af deep web-indholdet under sidstnævnte kategori.

På den måde kan vi betragte enhver hjemmeside, der beder om et login og en adgangskode, som en del af det dybe web. Med andre ord er mange af de sider, folk tilgår hver dag, faktisk en del af det dybe web. Dette inkluderer profiler på sociale medier, e-mail-indbakker og bankkonti.

Dette kan lyde forvirrende i starten, fordi den populære terminologi siger, at det dybe web er det obskure web. Men da vores personlige konti ikke er indekseret på søgemaskinesider, betragtes de også som en del af det dybe web.

Alligevel består den største del af det dybe web af statistikker og private databaser. For eksempel står amerikanske regeringsorganer som [Commodity Futures Trading Com](/cftc) [mission (CFTC)](/cftc),. National Oceanic and Atmospheric Administration (NOAA) og [Securities and Exchange Commission (SEC)](/sec) for en betydelig mængde af disse digitale data.

## Deep web vs. dark web

Det er også almindeligt at se forvirring om forskellene mellem det dybe spind og det mørke spind. Kort sagt er det mørke web en del af det dybe web, der er anonymt.

Det er således ofte forbundet med hjemmesider, der faciliterer ulovlige aktiviteter, eller som giver en form for privatlivsfordele for aktivister og whistleblowere (herunder black hat [hackere](/hacker) og hacktivister). Det mørke web anslås at udgøre mindre end 0,01% af det dybe web og er kun tilgængeligt via specialiserede platforme, såsom Tor.

## Højdepunkter

- Den måske mest alvorlige kritik af det dybe web er, at det underminerer internettets åbenhed og lighed.

- Det dybe web refererer til dele af internettet, der ikke er fuldt tilgængelige via standardsøgemaskiner som Google, Yahoo og Bing.

- Det dybe web giver brugerne adgang til langt mere information, end der ellers ville være tilgængelig på internettet, og det øger også privatlivets fred.

- Den dybe web omfatter sider, der ikke er indekseret, gebyr-for-service-websteder, private databaser og det mørke web.

## Ofte stillede spørgsmål

### Er Deep Web og Dark Web det samme?

Det mørke spind er en lille bestanddel af det dybe spind. Det dybe web refererer til alt det internet, der ikke er let tilgængeligt eller indekseret af søgemaskiner (som kaldes "overfladenettet"). Det mørke web består af dybe websteder, der med vilje er skjult, ofte for at udføre ulovlige eller ulovlige transaktioner online.

### Er det ulovligt at få adgang til Deep Web?

Nej. Det er ikke ulovligt at få adgang til websteder, der ikke er indekseret eller på anden måde offentligt tilgængelige. Det kan dog være ulovligt at misbruge eller stjæle information fundet på dybe websteder.

### Hvordan kan jeg finde websteder på Deep Web?

Fordi det dybe web ikke er fuldt indekseret med almindelige søgemaskiner, er den eneste måde at finde sådanne sider ofte på at kende den nøjagtige internetadresse, der skal bruges. Der er nogle specialiserede websteder eller søgemaskiner, der katalogiserer nogle dybe websteder. For eksempel kan akademiske ressourcer på det dybe web findes ved hjælp af platforme som PubMed, LexisNexis, Web of Science eller Project MUSE.
