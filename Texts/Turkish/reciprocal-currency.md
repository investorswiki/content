# Karşılıklı Para Birimi
## Karşılıklı Para Birimi Nedir?

[Döviz (forex) piyasasında](/foreign-exchange-markets) karşılıklı bir para birimi, bir döviz çiftinin ABD dolarını (USD) içerdiği, ancak USD'nin [temel para birimi olmadığı bir durumu tanımlar](/basecurrency) ; bunun yerine, [teklif para birimidir](/quotecurrency) (karşı para birimi olarak da bilinir).

Böylece karşılıklı bir para birimi, dolar başına para birimi yerine yabancı para birimi başına ABD doları cinsinden kote edilir. Yaygın bir örnek, [EUR/USD](/eur-usd-euro-us-dollar-currency-pair) döviz çiftleri olabilir; burada 1.20'lik bir fiyat teklifi, bir euro'nun 1.20 ABD doları satın aldığı anlamına gelir.

## Karşılıklı Para Birimlerini Anlama

USD döviz çiftlerinin çoğunda, bir FX teklifinde ilk olarak görünen temel para birimi olarak ABD doları bulunur. Örneğin, USD/JPY (dolar karşısında [Japon yeni](/jpy-japanese-yen) ) veya USD/CAD (dolar karşısında [Kanada doları](/cad-canadian-dollar) ).

Böyle bir teklifte, size bir ABD dolarının kaç birim döviz alabileceğini söylerdi. Örneğin, İsrail Şekeli (USD/ILS) için fiyat 3.25 ise, bir dolar 3.25 şekel satın alır.

Bununla birlikte, karşılıklı para birimleri bunun yerine yaygın olarak "Avrupa" terimleriyle ifade edilir, yani ABD doları dışındaki para birimi temel para birimidir.

"Karşılıklı para birimi" bu nedenle, ABD doları (USD) ve başka bir para biriminin eşlendiği, ancak USD'nin kote edilen ilk para birimi olmadığı döviz piyasasında kullanılan döviz çiftlerini tanımlar.

ABD'li bireyler veya firmalar için karşılıklı bir para birimi, doğrudan bir fiyat konvansiyonu olacaktır. Dolaylı bir [alıntı](/indirectquote) alternatif olarak zıt ifade olarak görünecektir.

## Alt çizgi

Döviz ticaretinde karşılıklı bir para birimi, USD'nin temel para birimi değil, karşı para birimi olduğu bir döviz çiftidir. Birim yabancı para birimi başına ABD doları cinsinden kote edilir. Karşılıklı para birimleri, genel karşılıklı para birimleri EUR/USD, GBP/USD ve AUD/USD olmak üzere küresel FX ticaretinin büyük bir bölümünü oluşturur.

## Öne Çıkanlar

- Karşılıklı para birimi, temel para birimi olarak USD kullanılmadan ABD dolarını (USD) içeren bir para birimi çiftidir.

- USD/JPY ve USD/CAD, ABD dolarının temel para birimi olduğu örneklerdir.

- Döviz kurları, 1'i cari döviz kuruna bölerek ters çevrilebilir.

- NZD/USD ve EUR/USD, USD bu çiftlerde temel para birimi olmadığından karşılıklı döviz çiftlerine örnektir.

- Bu alıntı gösterimi, USD'nin temel para birimi olarak hizmet ettiği ve bazen "Avrupa" teklifi olarak adlandırıldığı zamandan daha az yaygındır.

## SSS

### Karşılıklı Para Birimi Düzenlemesi Nedir?

Karşılıklı para birimi düzenlemesi, iki ülke arasında, birbirlerinin para birimlerinin belirli bir para arzını sürdürmek için yapılan bir anlaşmadır. Bu, ülkeler arasındaki ve küresel finans piyasalarındaki likiditeyi iyileştirir, daha verimli finansal işlemlere izin verir, zorunlu karşılıkları korur ve döviz kurlarını belirler. Karşılıklı para birimi düzenlemeleri, takas hatları olarak da bilinir.

### Karşılıklı Döviz Kurunu Nasıl Bulursunuz?

Karşılıklı bir döviz kuru, döviz kurunun tersi olacaktır. Ters ilişki için 1'i iki para biriminin mevcut döviz kuruna bölersiniz. Örneğin, USD/EUR döviz kuru 0,89 ise, EUR/USD'nin karşılıklı döviz kurunu bulmak için şu hesaplamayı yaparsınız: 1/0,89, 1,12'ye ulaşmak için.

### Bir Para Birimi Alıntılamanın İki Yolu Nedir?

Para birimleri, doğrudan alıntılar veya dolaylı alıntılar olarak kote edilebilir. Doğrudan kotasyon, bir birim yabancı para biriminin yerel para biriminin karşılık gelen birimlerinde kote edilmesidir. Dolaylı kotasyon, bir birim yerel para biriminin eşdeğer yabancı para birimi cinsinden kote edilmesidir.

### Döviz Çifti Nedir?

Bir döviz çifti, bir para biriminin değeri diğerinin değerine karşı kote edilen iki para biriminin değerini kote eder. Çiftteki ilk para birimi temel para birimidir ve listelenen ikinci para birimi, karşıt para birimidir. Para birimi çiftleri, bir birim temel para birimini satın almak için karşıt para biriminin ne kadarının gerekli olduğunu gösterir.

### Yabancı Para Birimi Seçenekleri Nasıl Kote Edilir?

Döviz opsiyonları içsel değer + zaman değeri olarak fiyatlandırılır. Bu, FX opsiyon primidir. Gerçek değer, grev oranı ve ileri oran kullanılarak dönüştürülen para birimleri arasındaki farktır.
