# Dolaşımdaki para
## Dolaşımdaki Para Birimi Nedir?

[para birimi](/currency),. bir ülkede tüketiciler ve işletmeler arasındaki işlemleri gerçekleştirmek için fiziksel olarak kullanılan kağıt para veya madeni para biçimindeki nakit miktarını ifade eder. Dolaşımdaki para birimi, bir ülkenin para otoritesi tarafından çıkarılan paranın tamamı, eksi sistemden çıkarılan nakittir. Dolaşımdaki para birimi, genel [para arzının](/moneysupply) bir kısmını temsil eder ve toplam arzın bir kısmı çek ve tasarruf hesaplarında saklanır.

## Dolaşımdaki Para Birimini Anlama

Dolaşımdaki para birimi, bir ülkenin ekonomisinde mal ve hizmet satın almak için kullanılan para olduğu için eldeki para birimi olarak da düşünülebilir. [Merkez bankalarının](/centralbank) para otoriteleri , en [likit varlık](/liquidasset) sınıflarından birini temsil ettiği için dolaşımdaki fiziki para miktarına dikkat etmektedir . Dolaşımdaki para birimi, diğer para türlerine (örneğin banka rezervleri) kıyasla merkez bankalarının para politikası için daha az önemlidir çünkü para biriminin miktarı nispeten daha az esnektir.

ABD'de yeni para birimi [Hazine Bakanlığı tarafından basılır](/ustreasury) ve Federal Rezerv Bankaları tarafından daha fazla para siparişi veren bankalara dağıtılır. Dolaşımdaki ABD para birimi miktarı, uluslararası piyasadan gelen talep sonucunda yıllar içinde artmıştır. Hazine Bakanlığı'na göre, dolaşımdaki ABD para biriminin yarısından fazlası yurtiçinden ziyade yurtdışında bulunuyor. ABD para birimine yönelik denizaşırı talep, kısmen, daha [değişken](/volatility) para birimi değerlerine sahip ülkelerle karşılaştırıldığında ABD para biriminin göreli istikrarından kaynaklanmaktadır .

[Elektronik fonlar](/electronic-money) birçok işlem türü için erişilebilir olsa da , bazı durumlarda dolaşımdaki fiziksel para birimi tercih edilebilir. Örneğin, doğal afetlerden sonra, hemen ihtiyaç duyulan hizmetler için ödeme aracı olarak fiziksel para birimi daha yaygın hale gelebilir. Ayrıca, felaketin doğası elektronik fonlara erişimi zorlaştırabilir veya imkansız hale getirebilir. Güç, yaygın alanlarda bulunmayabilir, örneğin, işlemleri yürütmek için tek yöntem olarak fiziksel para birimi veya kağıt çekler yapmak. Fiziki para biriminin teslimi, [varlıkların](/asset) kurumlar arasında transfer edilmesini beklemek yerine, fonları hemen ihtiyacı olanların eline verir .

## Dolaşımdaki Para Birimi Örneği

Amerika Birleşik Devletleri'nde, basılan ve dolaşımda kalan para birimlerinin çoğu, 1, 2, 5, 10, 20, 50 ve 100 dolarlık banknotları (dolaşımdaki madeni paralara ek olarak) içerir. Farklı dönemlerde Hazine Bakanlığı üretimi durdurmuş ve [Federal Rezerv Bankaları](/federalreservesystem) belirli para birimlerini dolaşımdan kaldırmıştır.

Örneğin, İkinci Dünya Savaşı'ndan sonra 500$, 1.000$, 5.000$ ve 10.000$'lık para birimlerinin basımı durduruldu. 1969'da Federal Rezerv Bankalarına bu kağıt para birimini dolaşımdan çıkarmaları emredildi. Bu mezhepler, büyük fon transferleri yapmak gibi amaçlar için kullanılmıştır. Ayrıca, [güvenli](/electronic-funds-transfer-act) [elektronik para transfer yöntemleri giderek daha fazla kullanılmaya başlandıkça](/electronic-funds-transfer-act),. bu tür büyük para birimlerine olan ihtiyaç ortadan kalktı. Böyle bir para birimi hala mevcut olsa da, Federal Rezerv Bankaları onları dolaşımdan çıkarmak ve ardından fiziksel para birimini yok etmek için aktif olarak çalışır.

## Öne Çıkanlar

- Dolaşımdaki para birimi, para otoriteleri tarafından çıkarılan para miktarından bir ekonomiden çıkarılan para biriminin çıkarılmasıyla elde edilen miktardır.

- Amerika Birleşik Devletleri'nde, elektronik fon transferlerini gerçekleştirme yeteneği, işlemler için daha büyük faturalara olan ihtiyacı azalttığından, para biriminin çoğunluğu 100 ABD Doları veya daha azdır.

- Federal Rezerv Bankaları, ABD Darphanesi'nden yeni para birimi sipariş eder ve gerektiğinde dolaşımdan çıkarır.

- Dolaşımdaki para, bir ülkenin para arzının önemli bir bileşenidir.
