# プリンシパル・エージェント問題
##プリンシパル・エージェント問題とは何ですか？

プリンシパルエージェント問題は、個人またはグループと、彼らに代わって行動する権限を与えられた代表者との間の優先順位の対立です。[エージェントは、](/agent)[プリンシパル](/principal)の最善の利益に反する方法で行動する場合があります。

プリンシパル・エージェント問題は、プリンシパルとエージェントの可能な役割と同じくらい多様です。これは、資産またはプリンシパルの所有権がその資産の直接制御を別の当事者またはエージェントに委任するあらゆる状況で発生する可能性があります。

##プリンシパルエージェント問題を理解する

プリンシパル・エージェント問題は、政治学と経済学の標準的な要因になっています。この理論は、1970年代にハーバードビジネススクールのマイケルジェンセンとロチェスター大学のウィリアムメックリングによって開発されました。 1976年に発表された論文で、彼らは、所有権と支配の分離として特定した、代理店の費用とその原因として定義したものを回避するように設計された所有権構造の理論を概説しました。

この制御の分離は、プリンシパルがエージェントを雇うときに発生します。プリンシパルは、ある程度の制御と決定を行う権利をエージェントに委任します。ただし、元本は資産の所有権と損失に対する責任を保持します。

たとえば、一部の所有者である会社の株式投資家は、会社の最高経営責任者（CEO）を代理人として信頼して戦略を実行するプリンシパルです。つまり、株価を上げるか、[配当を支払う](/dividend)か、またはその両方を望んでいます。 CEOが代わりにすべての利益を拡大に投入するか、マネージャーに大きな[ボーナス](/bonus)を支払うことを選択した場合、プリンシパルはエージェントに失望したと感じる可能性があります。

プリンシパル・エージェント問題には多くの救済策があり、それらの多くは期待を明確にし、結果を監視することを含みます。プリンシパルは通常、問題を修正できる、または修正する唯一の当事者です。

##代理店費用

論理的には、プリンシパルはエージェントのアクションを常に監視することはできません。代理人が責任を怠ったり、下手な決定を下したり、あるいは本人の最善の利益に反する方法で行動したりするリスクは、[代理店の費用として定義することができます](/agencycosts)。エージェントの行動から生じる問題に対処する際に、追加の代理店費用が発生する可能性があります。代理店費用は、[取引費用](/transactioncosts)の一部と見なされます。

代理店の費用には、代理人が特定の方法で行動することを奨励するための金銭的またはその他のインセンティブを設定する費用も含まれる場合があります。代理店の雇用による投資収益率の予想される増加が、代理店の費用を含む代理人の雇用の費用よりも大きい限り、プリンシパルはこれらの追加費用を負担する用意があります。

##プリンシパルエージェント問題の解決策

プリンシパルエージェント問題を解決する方法があります。責任は、エージェントがプリンシパルが望むように行動するためのインセンティブを作成するプリンシパルにあります。最初の例、[株主](/shareholder)とCEOの関係を考えてみましょう。

株主は、マネージャーを雇用する前後に、リスクを克服するための行動を取ることができます。第一に、彼らはマネージャーのインセンティブを株主のインセンティブと一致させる方法でマネージャーの契約を書くことができます。プリンシパルは、エージェントに定期的に結果を報告するように要求できます。彼らは情報を追跡するために外部のモニターや監査人を雇うことができます。最悪の場合、マネージャーを置き換えることができます。

###契約条項

近年、報酬と業績測定を可能な限り密接に結び付ける雇用契約に向かう傾向があります。企業の経営者にとってのインセンティブには、業績に基づく株式または[ストックオプションの報奨](/stockoption)、[利益分配計画](/profitsharingplan)、または経営者の給与と株価を直接結び付けることが含まれます。

根本的には、それは良いサービスのためのチップと同じ原則です。理論的には、チップは顧客またはプリンシパルとエージェントまたはウェイターの利益を一致させます。彼らの優先順位は現在一致しており、優れたサービスに焦点を合わせています。

##プリンシパルエージェント問題の例

プリンシパルエージェント問題は、金融の世界を超えた多くの日常的な状況で発生する可能性があります。弁護士を雇うクライアントは、弁護士が必要以上に請求可能な時間を壊してしまうのではないかと心配するかもしれません。住宅所有者は、市議会による納税者資金の使用を承認しない場合があります。住宅購入者は、不動産業者が購入者の懸念よりも手数料に関心があるのではないかと疑うかもしれません。

これらすべての場合において、校長はその問題に関してほとんど選択の余地がありません。仕事を成し遂げるにはエージェントが必要です。

##ハイライト

-この問題は、クライアントと弁護士の関係から、株主とCEOの関係まで、さまざまな状況で発生する可能性があります。

-プリンシパルエージェント問題は、資産の所有者と資産の管理が委任された人との間の優先順位の競合です。

-代理人が本人の最善の利益に反する方法で行動するリスクは、代理店の費用として定義することができます。

-プリンシパルエージェント問題を解決するには、優先順位を調整するか、情報の流れを改善するか、またはその両方を行うために、報酬のシステムを変更する必要がある場合があります。
