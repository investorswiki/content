# マルチアセットクラス
##マルチアセットクラスとは何ですか？

マルチアセットクラスは、マルチアセットクラスまたはマルチアセットファンドとも呼ばれ、投資として使用されるアセットクラス（現金、[株式](/equity)、債券など）の組み合わせです。マルチアセットクラスの投資には複数のアセットクラスが含まれるため、アセットのグループまたはポートフォリオが作成されます。クラスの重みと種類は、個々の投資家によって異なります。

##マルチアセットクラスの仕組み

マルチアセットクラスの[投資](/investment)は、投資を複数のクラスに分散することにより、ポートフォリオ全体の分散を促進します[。](/diversification)これにより、1つのクラスの資産を保有する場合に比べてリスク（ボラティリティ）が減少しますが、潜在的なリターンが隠される可能性もあります。たとえば、マルチアセットクラスの投資家は債券、株式、現金、[不動産を保有](/real-property)しているのに対し、シングルクラスの投資家は株式のみを保有している場合があります。 1つの[資産クラス](/assetclasses)が特定の期間にアウトパフォームする可能性がありますが、歴史的に、すべての期間にアウトパフォームする資産クラスはありません[。](/outperform)

##リスク許容基金

多くの投資信託会社は、投資家のリスク許容度に従って機能するように設計された資産配分ファンドを提供しています。資金は積極的なものから保守的なものまでさまざまです。アグレッシブスタイルのファンドは、エクイティへの配分がはるかに高く、おそらく100％にもなります。

Fidelity Asset Manager 85％ファンド（「FAMRX」）は、積極的なファンドの一例です。このファンドは、ファンドのエクイティへの配分の85％、債券と現金の15％を維持するように設計されています。保守的な投資家にとって、ファンドの配分は債券にかなり集中するでしょう。 Fidelity Asset Managerの20％ファンド（「FASIX」）には、株式が20％、債券が50％、短期金融市場ファンドが30％あります。

##目標日ファンド

目標日ファンドは、投資家の期間に応じて配分を変更するマルチアセットファンドです。投資家は、自分たちの期間を厳密に反映するファンドを選択します。たとえば、30年以上退職しない投資家は、2045年以降のターゲットファンドの1つを選択する必要があります。ファンドの日付が遅いほど、期間が長くなるため、ファンドはより積極的になります。 2050年の目標日ファンドは、株式が85〜90％を超え、残りは債券または短期金融市場にあります。

期間が大幅に短い投資家は、最近の満期を迎えるファンドの1つを選択します。 5年以内に引退する人は、全体的なリスクを軽減し、資本保全に集中するために、より高いレベルの債券を備えた目標日ファンドを持っているでしょう。

目標日ファンドは、適切な資産配分の選択に関与したくない投資家にとって有益です。投資家が年をとり、期間が短くなるにつれて、目標日ファンドのリスクレベルも低くなります。時間の経過とともに、ファンドは徐々に株式から債券およびマネーマーケットに自動的に移行します。

##マルチアセットクラスファンドのメリット

通常、ベンチマークを満たすか上回ることに焦点を当てるバランスの[取れたファンド](/balancedfund)とは異なり、マルチアセットクラスのファンドは、[インフレを超えるなどの特定の投資成果を達成するように構成されてい](/inflation)ます。証券、セクター、不動産、その他の種類の証券にまたがる幅広い投資オプションにより、目標を達成するための非常に高い柔軟性が得られます。

このタイプのファンドはまた、主に債券と株式を組み合わせる可能性のあるほとんどのバランスの取れたファンドよりも多様性を提供します。多くは[積極的に管理されてい](/activemanagement)ます。つまり、個人または人々のグループは、市場のダイナミクスに基づいて意思決定を行い、リターンを最大化し、リスクを制限します。

##ハイライト

-マルチアセットクラスの投資は、投資家の方向性に対応するために時間の経過とともに変化する可能性があります。この典型的な例は、目標日ファンドです。

-マルチアセットクラスは、主に、投資家のさまざまなセクターへのエクスポージャーを拡大することにより、下振れリスクを制限するために構築されています。

-一部のETFは、マルチアセットクラスの投資と見なすことができます。
