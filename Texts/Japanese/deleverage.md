# デレバレッジ
##デレバレッジとは何ですか？

[レバレッジ](/leverage)を減らそうとする場合です。言い換えれば、レバレッジ解除は債務の削減であり、レバレッジの反対です。事業体がレバレッジを削減するための最も直接的な方法は、[貸借対照表](/balancesheet)上の既存の債務および義務を直ちに返済することです。これができない場合、会社または個人は[デフォルトのリスクが高まる可能性があります](/defaultrisk)。

##デレバレッジを理解する

レバレッジ（または負債）には、控除された利子に対する税制上の優遇措置、繰延現金支出、株式の希薄化の回避などの利点があります。債務は私たちの社会の不可欠な側面になっています。最も基本的なレベルでは、企業は債務を運用の資金調達、拡張への資金提供、[研究開発への支払いに使用し](/randd)ます。

ただし、企業が多額の債務を引き受ける場合、その債務を返済するための利息の支払いまたは費用は、企業に経済的損害を与える可能性があります。その結果、企業は、資産を[清算](/liquidate)または売却したり[、債務を再構築したりすることによって、債務のレバレッジ解消または返済を余儀なくされることがあり](/debtrestructuring)ます。

適切に使用された場合、債務は企業がその長期的な成長に資金を供給するのを助けるための触媒になる可能性があります。負債を利用することで、企業は[資本](/equity)を増やすことなく請求書を支払うことができ、株主の利益の[希薄化](/dilution)を防ぐことができます。株式の希薄化は、企業が株式を発行するときに発生し、既存の株主または投資家の所有割合の低下につながります。企業は株式を発行することで資本や資金を調達することができますが、株式の希薄化により既存株主の株価が下がる可能性があるという欠点があります。

###債務の発行

別の方法は、企業がお金を借りることです。企業は、[債券](/bond)の形で投資家に直接債務を発行することができます。投資家は会社に債券の[元本](/principal)を前払いし、その見返りとして、定期的な[利息](/coupon-rate)の支払いと、債券の[満期日に元本を返済します](/maturitydate)。企業は銀行や債権者から借りることで資金を調達することもできます。

たとえば、投資家からの500万ドルの投資で会社が設立された場合、会社の資本は500万ドルになります。これは、会社が運営に使用するお金です。会社が2,000万ドルを借りて債務融資をさらに組み込む場合、会社は現在、[資本予算プロジェクトに投資するための2,500万ドルを持ち](/capitalbudgeting)、固定数の株主の価値を高める機会が増えています。

###債務のレバレッジ解消

多くの場合、企業は成長を開始するために過剰な金額の債務を引き受けます。ただし、レバレッジを使用すると、企業のリスクが大幅に高まります。レバレッジが計画どおりにさらに成長しない場合、リスクは企業が負担できないほど大きくなる可能性があります。このような状況では、会社ができることは、借金を返済することによってデレバーすることだけです。[デ](/redflag)レバレッジは、企業の成長を必要とする投資家にとっては危険信号となる可能性があります。

[負債](/liability)によって資金が供給される企業の貸借対照表の相対的な割合を減らすことです。基本的に、これは2つの方法のいずれかで実行できます。第一に、企業または個人は、事業運営を通じて現金を調達し、その余剰現金を使用して負債をなくすことができます。第二に、設備、株式、債券、不動産、ビジネスアームなどの既存の資産を売却することができ、その結果得られる収益を債務の返済に振り向けることができます。いずれの場合も、貸借対照表の負債部分は削減されます。

>個人貯蓄率は、レバレッジ解消の1つの指標です。人々はより多くのお金を節約するため、借り入れをしていません。

##レバレッジ解消がうまくいかない場合

[ウォール街](/wallstreet)は、レバレッジ解消の成功を好意的に迎えることができます。たとえば、大規模な一時解雇の発表は株価を上昇させる可能性があります。ただし、レバレッジ解消は必ずしも計画どおりに進むとは限りません。債務水準を下げるために[資本](/capital)を調達する必要があるため、企業が売却したくない資産を[売却](/firesale)価格で売却せざるを得ない場合、企業の株式の価格は一般的に短期的には悪化します。

さらに悪いことに、投資家が企業が不良債権を抱えており、レバレッジを削減できないと感じると、その債務の価値はさらに急落します。それから、企業はそれを売ることができれば、それを損失で売ることを余儀なくされます。債務を売却または返済できない場合、事業が失敗する可能性があります。破綻した企業の[有毒な債務を抱え](/toxic-debt)ている企業は、[債券市場](/fixedincome)が崩壊するにつれて、バランスシートに大きな打撃を与える可能性があります。これは、2008年の崩壊前にリーマンブラザーズの債務を抱えていた企業の場合でした。

##レバレッジ解消の経済効果

[経済成長](/economicgrowth)と企業拡大の不可欠な要素です。あまりにも多くの人々や企業が一度に借金を返済し、それ以上引き受けないことを決定した場合、経済は苦しむ可能性があります。レバレッジ解消は通常、企業にとっては良いことですが、[景気後退](/recession)や景気後退の際に発生した場合、経済の信用成長を制限する可能性があります。企業がレバレッジを削減して借入を削減するにつれて、経済の下降スパイラルが加速する可能性があります。

その結果、政府は資産を購入し、価格を下げたり、支出を奨励したりするために、介入して債務を引き受ける（レバレッジ）ことを余儀なくされています。この[財政刺激策は、](/fiscalpolicy)住宅価格を下支えし銀行融資を促進するために住宅[ローン担保証券](/mbs)を購入する、特定の証券の価値を下支えする政府支援保証を発行する、破綻した企業の財政状態をとる、税金を提供するなど、さまざまな形でもたらされる可能性があります消費者に直接リベートし、[税額控除](/taxcredit)を通じて家電製品や自動車の購入を助成するか、または同様の行動のホスト。

[連邦準備制度](/federalreservesystem)はまた、フェデラル[ファンド金利](/federalfundsrate)を引き下げて、銀行がお互いにお金を借りるのをより安くし、金利を押し下げ、銀行が消費者や企業に貸し出すことを奨励することができます。

>納税者は通常、政府が苦しみ、レバレッジ解消プロセスを経ている企業を救済する際に、連邦債務を返済する責任があります。

##レバレッジ解消と財務比率の例

たとえば、X社の資産が2,000,000ドルで、そのうち1,000,000ドルが負債で賄われ、1,000,000ドルが株式で賄われているとします。その年の間に、X社は50万ドルの[純利益](/net-income-after-taxes-niat)または純利益を稼いでいます。

企業の財務状態を測定するために利用できる財務比率は多数ありますが、使用する主要な比率の3つを以下に概説します。

-総資産利益[率（ROA）](/returnonassets)は、総資産を純利益で割ったものであり、企業が設備などの長期資産でどれだけ収益を上げているかを示します。

-[株主資本利益率（ROE）](/returnonequity)は、純利益を株主資本で割って計算されます。これは、企業が株式の発行から調達した資本を使用して、どれだけ利益を上げているかを示します。

-[負債資本（D / E）](/debtequityratio)は、会社の負債を株主資本で割って計算されます。デット・トゥ・エクイティは、企業がその成長にどのように資金を提供しているか、そしてその負債をカバーするのに十分なエクイティ株があるかどうかを示します。

以下は、X社の財務情報を使用した比率計算です。

-**総資産利益率**=$ 500,000 / $ 2,000,000 = 25％

-**株主資本利益率**=$ 500,000 / $ 1,000,000 = 50％

-**デット・トゥ・エクイティ** = $ 1,000,000 / $ 1,000,000 = 100％

上記のシナリオの代わりに、年の初めに会社が$800,000の資産を使用して$800,000の負債を返済することを決定したと仮定します。このシナリオでは、X社の資産は$ 1,200,000になり、そのうち$ 200,000は負債で賄われ、$1,000,000は株式で賄われます。会社がその年の間に同じ$500,000を稼いだ場合、総資産利益率、自己資本利益率、および負債資本利益率は次のようになります。

-**総資産利益率**=$ 500,000 / $ 1,200,000 = 41.7％

-**株主資本利益率**=$ 500,000 / $ 1,000,000 = 50％

-**デット・トゥ・エクイティ** = $ 200,000 / $ 1,000,000 = 20％

2番目の比率のセットは、会社がはるかに健全であることを示しているため、投資家または貸し手は2番目のシナリオがより有利であると考えるでしょう。

##ハイライト

-レバレッジ解消とは、新たな債務を負うことなく未払いの債務を削減することです。

-レバレッジ解消の目標は、負債によって資金調達された企業のバランスシートの相対的な割合を減らすことです。

-体系的なレバレッジ解消が多すぎると、景気後退と信用収縮につながる可能性があります。
