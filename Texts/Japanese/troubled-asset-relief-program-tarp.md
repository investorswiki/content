# 不良資産買取プログラム（TARP）
##不良資産買取プログラム（TARP）とは何ですか？

不良資産買取プログラム（TARP）は、2008年の金融危機を受けて、国の金融システムを安定させ、経済成長を回復し、[差し押さえを緩和するために米国財務省によって作成および](/foreclosure)[実行されたイニシアチブでした。](/herbert-m-allison-jr) TARPは、問題のある企業の資産と株式を購入することにより、これらの目標を達成しようとしました。

##不良資産買取プログラム（TARP）の仕組み

[フレディマック](/freddiemac)、アメリカンインターナショナルグループ（AIG）などのいくつかの主要な金融機関が深刻な財政問題を経験したため、世界の信用市場は2008年9月にほぼ停滞しました。[リーマンブラザーズ](/lehman-brothers)は破産し、投資会社のゴールドマンサックスとモルガンスタンレーは、資本状況を安定させるために、チャーターを商業銀行に[変更](/corporatecharter)[しました。](/corporatecharter)

状況が完全に暴走するのを防ぐために、財務長官の[ヘンリー・ポールソン](/henry-paulson)は不良資産買取プログラム（TARP）を開拓しました。 2008年10月3日、[緊急経済安定化法が可決され、ジョージWブッシュ大統領が法に署名しました](/emergency-economic-stability-act)。

TARPの本来の目的は、住宅ローン担保証券（MBS）を購入することにより、短期金融市場と[二次住宅ローン市場の流動性を高め](/secondary_mortgage_market)、それを通じて、それらを所有する機関の潜在的な損失を減らすことでした。

その後、TARPの目的は、政府が銀行やその他の金融機関の株式を購入できるようにわずかに変更されました。 TARPは当初、財務省に7,000億ドルの購買力を与えました。ドッド・フランク・ウォールストリート改革および消費者保護法（単に[ドッド・フランクと呼ばれる](/dodd-frank-financial-regulatory-reform-bill)）は、後に7000億ドルの認可を4750億ドルに減らしました。

> TARP資金は、銀行、保険会社、自動車メーカーの株式を購入したり、金融機関や住宅所有者に資金を貸したりするために使用されました。

>>

米国政府は、バンクオブアメリカ/メリルリンチ、バンクオブニューヨークメロン、シティグループ、ゴールドマンサックス、JPモルガン、モルガンスタンレー、ステートストリート、ウェルズファーゴの8つの銀行で[優先株を購入しました。](/preferredstock)銀行は政府に5％の配当を与えることを要求され、2013年には9％に増加し、銀行が5年以内に株式を買い戻すことを奨励しました。

プログラムの開始から2010年10月3日（資金延長の期限）まで、2450億ドルが銀行の安定化に使用され、270億ドルが信用の利用可能性を高めるプログラムに費やされ、800億ドルが米国の自動車産業（具体的にはGMとクライスラー）に費やされました。 、680億ドルはAIGを安定させるために使用され、460億ドルは住宅を手頃な価格にするなどの差し押さえ防止プログラムに使われました。

TARPの規定は、関係する企業が特定の税制上の優遇措置を失うことを要求し、多くの場合、役員報酬に制限を設け、ファンドの受領者が上位25名の最高賃金の役員にボーナスを与えることを禁じました。それでも、2009年までに、救済された企業は主要な人員に約200億ドルを支払いました。これは、皮肉なことに[TARPボーナスと呼ばれ](/tarp-bonuses)ます。

###TARPのレガシー

2013年12月、財務省はTARPをまとめ、政府はその投資が納税者のために110億ドル以上を稼いだと結論付けました。具体的には、TARPは4,264億ドルの投資から、合計4,417億ドルの資金を回収しました。政府はまた、TARPがアメリカの自動車産業の失敗を防ぎ、100万人以上の雇用を節約し、銀行を安定させ、個人や企業の信用力を回復させたと主張しました。

>TARPはまだ物議を醸しています。支持者は、それが米国の金融システムを救い、金融危機を短縮したと言いますが、批評家は、イニシアチブがウォール街に不必要な後押しを与えたと非難しています。

>>

それでも、経済学者、政治家、金融専門家は、TARPのメリットについて議論し、それが必要だったかどうか疑問に思っています。批評家たちは、このプログラムが住宅市場を助けるのにほとんど役立たなかったと非難している。政府が将来の慣行を管理するために救済していた金融会社の株式を主張すべきだったという人もいます。

代わりに、批評家は、TARPの無担保ローンは本質的に悪い行動に対する報酬として機能し、「無責任に行動し、私たちはあなたを助けます」というメッセージを送り、依存の危険な前例を確立すると意見を述べています。

TARPはまた、ウォール街がこれらの悪名高いボーナスを含む利益を享受し、個人が大不況の結果として債務、失業、差し押さえに苦しんでいたとしても、収益性に戻るのを見たアメリカ国民に政府を慕わなかった。

##ハイライト

-TARPは当時物議を醸しており、その有効性については引き続き議論されています。

-2008年から2010年にかけて、TARPは4,264億ドルを企業に投資し、4,417億ドルを回収しました。

-TARPは、政府に住宅ローン担保証券と銀行株を購入させることにより、金融システムを安定させました。

-不良資産買取プログラム（TARP）は、2008年の金融危機に続いて米国財務省によって開始されました。
