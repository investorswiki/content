# クラウドストレージ
##クラウドストレージとは何ですか？

クラウドストレージは、企業や消費者がオンラインでデータを安全に保存するための方法であり、いつでもどこからでもアクセスでき、許可を与えられた人と簡単に共有できます。クラウドストレージは、オフサイトでのリカバリを容易にするためにデータをバックアップする方法も提供します。

現在、個人は、Googleドライブ、Dropbox、Boxなどのいくつかの無料のクラウドコンピューティングサービスにアクセスできます。これらのサービスにはすべて、より大きなストレージサイズと追加のクラウドサービスを提供するアップグレードされたサブスクリプションパッケージが付属しています。

##クラウドストレージの説明

クラウドストレージは、安全で安全な方法でデータを保存および/または移動するための簡単な方法を提供します。これにより、個人や企業は、任意のデバイスでオンデマンドアクセスを行うために、ファイルをクラウドサービスプロバイダーに保存しておくことができます。クラウドストレージは、長期的なストレージを必要とするが、特定の財務記録など、頻繁にアクセスする必要のないデータをアーカイブするためにも使用できます。 「クラウドに」保存されたファイルは、グループコラボレーションに利用されることが増えています。

クラウドストレージは、クライアントコンピューター、タブレット、またはスマートフォンがリモートデータサーバーとの間でファイルをオンラインで送受信できるようにすることで機能します。通常、同じデータが複数のサーバーに同時に保存されるため、1つのサーバーがダウンしたりデータが失われたりした場合でも、クライアントは常にデータにアクセスできます。たとえば、ラップトップコンピュータの所有者は、ラップトップが盗まれた場合に備えて、個人の写真をハードドライブとクラウドの両方に保存する場合があります。

クラウドストレージシステムは、デジタル写真や音楽ファイルなどの特定の種類のデータの保存に特化したり、写真、オーディオファイル、テキストドキュメント、プレゼンテーション、スプレッドシートなどのあらゆる種類のデータの一般的な保存を提供したりできます。

クラウドストレージは、1960年代にコンピューター科学者のジョセフカールロブネットリックライダー博士によって発明されたと考えられています。約20年後、CompuServeは、ファイルの一部を保存するために、顧客に少量のディスク容量を提供し始めました。 1990年代半ば、AT＆Tは、個人およびビジネスのコミュニケーションのための最初のすべてのWebベースのストレージサービスを開始しました。それ以来、さまざまなサービスが注目を集めています。最も人気のあるクラウドストレージプロバイダーには、Apple（iCloud）、Amazon（Amazon Web Services）、Dropbox、およびGoogleがあります。

##クラウドストレージがビジネスにどのように役立つか

クラウドストレージは、主要なデータストレージが必要な企業が、事業所のデータストレージインフラストラクチャの必要性を排除することにより、大幅なスペースとお金を節約するのに役立ちます。クラウドストレージプロバイダーは、必要なすべてのハードウェアとソフトウェアを所有および保守しているため、クラウドユーザーはその必要がありません。継続的なクラウドストレージの購入は、長期的にはより多くの費用がかかる可能性がありますが、事前に大幅に安くなる可能性があります。さらに、企業は、ストレージのニーズの変化に応じて、アクセスできるクラウドストレージの量をほぼ瞬時にスケールアップまたはスケールダウンできます。

クラウドにより、従業員は同僚と共同作業を行うことができ、リモートで通常の営業時間外に作業することができます。また、許可された従業員がファイルの最新バージョンに簡単にアクセスできるようにすることで、スムーズなドキュメント共同作業が容易になります。個人レベルでは、クラウドストレージはモバイルデータを可能にし、今日の私たちの生活の全体的な方法でデジタルライフを可能にします。クラウドがなければ、スマートフォンはそれほど多くのデータ（写真、ドキュメント、外出先での情報）のインターフェースにはなり得ません。クラウドを使用してファイルを保存すると、エネルギー消費が削減されるため、環境にもプラスの効果があります。

##クラウドストレージのセキュリティ

今日のデジタル時代のクラウドストレージは、自主的に保存する場合でも、取引先の企業がクラウドに保存する場合でも、機密性の高い個人データの多くがクラウドに保存されるため、注目を集めています。その結果、[クラウドのセキュリティ](/cloud-security)が大きな関心事になっています。ユーザーは自分の情報が安全かどうか疑問に思っており、[データ侵害の増加により、安全で](/data-breach)ない場合があることが示されています。ユーザーは、クラウドに保存したデータに必要なときにアクセスできるかどうかについても懸念しています。

クラウドストレージはハッキングが蔓延しているため脆弱に見えるかもしれませんが[、](/cybersecurity)[オン](/cybersecurity)サイトストレージなどの代替手段にもセキュリティの脆弱性があります。会社が提供するクラウドストレージは、従業員がオフィスの外でアクセスする必要のあるファイルをバックアップおよび転送するために個人アカウントを使用する代わりの方法を従業員に提供することにより、実際にセキュリティを向上させることができます。

優れたクラウドストレージプロバイダーは、データの冗長性を備えており、同じファイルを複数の物理的な場所に保存して、人的エラー、機器の障害、または自然災害に耐えられるようにします。評判の良いプロバイダーは、データを安全に保存および送信するため、許可なくデータにアクセスすることはできません。一部のユーザーは、データの読み取りのみが可能で変更できないような方法でデータを保存することを要求する場合もあります。この機能も、クラウドストレージを通じて利用できます。
