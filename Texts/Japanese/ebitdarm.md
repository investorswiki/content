# EBITDARM
## EBITDARMとは何ですか？

[EBITDARM](/financialperformance) （利息、税金、減価償却、[償却、家賃、および管理手数料の前の収益）は、](/amortization)特定の企業の[財務実績](/financialperformance)を測定するために使用される選択的な収益指標です。 EBITDARMは、企業の家賃と管理費が通常よりも高い運用コストの割合を占める場合に、 [EBITDAなどのより一般的な指標と比較されます。](/ebitda)

##EBITDARMを理解する

投資家は、会社の収益性を分析するために自由に使えるいくつかの財務指標を持っています。多くは単純な[収益](/earnings)または[純利益](/netincome)に焦点を合わせています。また、パフォーマンスを測定するために特定の広告申込情報を含めたり除外したりすると便利な場合があります。

EBITDARMはEBITDAの拡張であり、利息、税金、減価償却、および償却前の収益の略です。これは、企業の業績と、資金調達や会計上の決定、または税務環境を考慮せずに収益を上げる能力を評価するために設計された公式です。費用は業務の一部とは見なされません。

EBITDARMが異なるのは、収益性を計算するときにレンタル料金と[管理料金も除外することです。](/managementfee)これは、そのような料金がかなりの[運用コストを占める企業を分析するときに役立ちます](/operating-cost)。

[不動産投資信託](/reit)（REIT）、収益を生み出す不動産を所有または資金提供している企業、およびヘルスケア企業（病院や介護施設の運営者など）は、これらの業界が使用するスペースをリースすることが多いため、このボックスにチェックマークを付け[ます](/rent-expense)。主要な運用コスト。 EBITDARMを使用すると、利益を生む避けられない固定費を取り除くことで、これらの企業の業績をより正確に把握できます。

>所有資産と賃貸資産に関連する費用を調整することで、賃貸または所有する資産の量に違いがある企業間で収益をより比較できるようになります。

>>

EBITDARMは、通常、次のように計算されます。

-** EBITDARM**=純利益+利息+税金+減価償却+償却+家賃と再編+管理費

###EBITDARM要件

すべての企業がEBITDARMを報告するわけではありません。この指標および他の同様のタイプの[調整後収益](/adjusted-earnings)の数値は、[一般に公正妥当と認められた会計原則](/gaap)（GAAP）に準拠していません。

必須ではありませんが、この指標は財務諸表に表示され、[証券取引委員会](/sec)（SEC）に報告方法に関するいくつかの規則を提示するよう促します。 SECは、企業がGAAPに基づいて収益を報告することを要求しています。 EBITDARMおよびその他の非GAAP財務指標も報告する場合は、これらの数値が最も直接的に比較可能なGAAP財務指標とどのように対照的であるかを示す必要があります。

###EBITDARMのメリット

[営業利益の](/operatingincome)調整を伴う措置は、純利益およびEBITDAや[EBIT](/ebit) （利息および税引前利益）などのより洗練された非GAAP措置と併せて検討された場合、投資家にとって最も有益です。また、同じ業界セクター内で事業を行っている企業の比較にも役立ちます。たとえば、資産を所有している企業とリースしている企業などです。

EBITDARMは、企業内での[資本配分の決定がどれほど効果的であるかを確認するために、賃料に対して測定される場合があります。また、特に](/capital_allocation)[信用格付け機関](/creditrating)（CRA）によって、企業の債務返済能力を確認するためにもよく使用されます。

この措置を提示する企業の多くは、高い[債務](/debt)負担を抱えています。[アナリスト](/analyst)と投資家は、EBITDARMの全体的なレベルと傾向を測定するだけでなく、EBITDARM対利息や債務対EBITDARMなどの債務返済カバレッジ比率を計算するために使用できます。

###EBITDARMへの批判

EBITDA、EBITDAR、EBITDARMなどの調整後の収益額に対する批判はたくさんあります。それらには、調整が企業の[キャッシュフロー](/cashflow)の正確な全体像を提供しないために歪曲的であり、操作が容易であり、[運転資本](/workingcapital)の変動を含む実際の費用の影響を無視するという懸念が含まれます。

[減価償却](/depreciation)費を追加することにより、企業やアナリストが[設備投資の経常費用を無視することへの](/capitalexpenditure)懸念を表明しています。

##ハイライト

-EBITDARMは、運用コストが大幅に異なる企業間で収益を比較できるようにするためによく使用されます。

-この指標は、家賃と管理費がかなりの運営費を占める企業を分析する際に役立ちます。

-EBITDARMなどの非GAAPメトリックを開示する企業は、これらの数値が最も直接的に比較可能なGAAP財務指標とどのように対照的であるかを示す必要があります。

-EBITDARMは、利息、税金、減価償却、償却、家賃、および管理手数料を差し引く前の収益を表し、財務実績の測定に使用される非GAAP収益指標です。
