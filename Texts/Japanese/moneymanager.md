# マネーマネージャー
##マネーマネージャーとは何ですか？

マネーマネージャーは、個人または機関投資家の証券[ポートフォリオを管理する個人または金融会社です。](/portfolio)通常、マネーマネージャーは、投資オプションの調査と選択から資産の監視と売却時期の決定に至るまで、さまざまな専門知識を持つ人々を雇用します。

手数料の見返りとして、マネーマネージャーは、適切な投資戦略の策定や、それらの目標を達成するための証券の売買など、クライアントのために慎重に投資を選択および管理するという基準上の義務を負っています[。](/fiduciary)マネーマネージャーは、「ポートフォリオマネージャー」、「アセットマネージャー」、または「投資マネージャー」とも呼ばれます。

##マネーマネージャーのしくみ

マネーマネージャーは、クライアントに個別のサービス、個別のポートフォリオ、および継続的な管理を提供します。取引ベースの管理とは対照的に、[手数料](/feebasedinvestment)ベースの管理では、クライアントとそのアドバイザーは同じ側にいます。つまり、クライアントは、証券を売買するブローカーの決定に疑問を呈する必要がなくなります。プロのマネーマネージャーは、取引の手数料を受け取らず、運用資産の割合に基づいて支払われます。したがって、ポートフォリオが成長するのを見るのは、マネーマネージャーとクライアントの両方にとって最大の利益です。

##マネーマネージャーを使用する理由

専門的な訓練を受けたマネーマネージャーは、クライアントのポートフォリオに最も適切な投資を選択するための専門知識を持っています。マネーマネージャーは通常、財務諸表を分析することで会社のファンダメンタルズを評価するのに役立つ[公認財務アナリスト（CFA）の指定を受けています。](/cfa)マネーマネージャーは、特定のセクターに関する専門知識を持っている場合もあります。たとえば、マネージャーは以前、自動車業界で自動車株を選択する際に優位に立つ役割を果たしていた可能性があります。

マネーマネージャーは、会社の幹部へのインタビュー、調査レポート、分析データ、高度な財務モデリングソフトウェアなど、多数の情報やツールにアクセスできます。これらのリソースを持つことで、マネーマネージャーは成功の可能性が高い投資決定を下すことができます。たとえば、マネーマネージャーは、CEOにインタビューした後、企業が競争力のある独自の利点を持っていることに気付く場合があります。

##マネーマネージャーはどのように支払われますか？

マネーマネージャーは通常、ポートフォリオのサイズに応じて、年間0.5％から2％の範囲の管理手数料を請求します。たとえば、資産運用会社は、100万ドルのポートフォリオに対して1％の管理手数料を請求する場合があります。ドルベースでは、これは10,000ドルの管理手数料に相当します。 （$ 1,000,000 x 1/100）。資産運用会社やヘッジファンドも、成功報酬を請求する場合があります。これは、プラスのリターンを生み出すための報酬です。成功報酬は通常、ファンドの利益の10％から20％の範囲です。たとえば、ファンドが10％の成功報酬を請求し、$ 250,000の利益を返す場合、クライアントは追加の$ 25,000の報酬（$ 250,000 x 10/100）を支払います。

##マネーマネージャーの実際の例

個人投資家の資金を受け入れる一流の資金管理会社の例には、Vanguard Group Inc.、Pacific InvestmentManagementCo.が含まれます。 （PIMCO）、およびJPモルガンアセットマネジメント。

有名な個人のマネーマネージャーには、[バークシャーハサウェイのウォーレンバフェットと](/berkshire-hathaway)フェアホルムファンドのブルースバーコウィッツが含まれます。

##ハイライト

-マネーマネージャーは、個人または機関投資家の証券ポートフォリオを管理する個人または金融会社です。

-マネーマネージャーには、クライアントの最初の利益を最後に、そして常に置く方法で投資を選択および管理する受託者責任があります。

-プロのマネーマネージャーは、取引の手数料を受け取りません。むしろ、運用資産の割合に基づいて支払われます。
