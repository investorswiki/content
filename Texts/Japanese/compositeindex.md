# 複合インデックス
##複合インデックスとは何ですか？

[株式](/equity)、証券、またはインデックスをグループ化する統計ツールです。通常、複合インデックスの要素は標準化された方法で組み合わされるため、大量のデータを簡単に表示できます。

##複合インデックスを理解する

複合インデックスは、投資分析を実施し、経済動向を測定し、市場活動を予測するために作成されます。

また、株式市場全体またはセクター全体に対する証券の価格変動を追跡するためのツールとしても使用されます。したがって、これらは投資家のポートフォリオを測定するための有用なベンチマークを提供します。[十分に分散された](/diversification)[ポートフォリオ](/diversification)の目標は、通常、主要な複合インデックスを上回ることです。米国で最もフォローされているインデックスの3つは、Nasdaq Composite、Dow Jones Industrial Average（Dow）、およびStandard＆Poor's 500 Index（S＆P 500）です。

##複合インデックスの種類

###ナスダック総合指数

[Nasdaq Compositeは](/nasdaqcompositeindex)、 1971年に50社のみで最初に設立されました。今日、それはナスダック株式市場に上場されている3,000以上の個人の一般的な株式を含むインデックスです。 Nasdaqコンポジットは、[市場資本](/marketcapitalization)（市場キャップ）加重手法（キャップ加重手法とも呼ばれます）を使用して計算されます。

### Standard＆Poor's 500 Index

[Standard＆Poor's 500 Index （S＆P 500）は](/sp500)、米国の大型株式の最高のバロメーターと広く見なされています。これには、[市場価値](/marketvalue)で500の最大の米国上場企業が含まれています。 S＆P 500は、キャップ加重インデックスでもあります。

###ダウジョーンズ工業平均

ダウジョーンズ工業平均（単に「ダウ」または「ダウジョーンズ」とも呼ばれる）は、価格加重複合指数です。 「市場が上がっている」というニュースを読むとき、彼らは一般的にダウを指しています。

##キャップ加重インデックスと価格加重インデックス

Dow（価格加重インデックス）とは異なり、NasdaqとS＆P500はどちらもキャップ加重インデックスです。

キャップ加重インデックスでは、各コンポーネントの総市場資本が比例してインデックスレベルを決定するために使用されます。この方法では、市場資本が高いコンポーネントはコンポジットでの重みが大きくなり、市場資本が低いコンポーネントはコンポジットでの重みが小さくなります。株式が上限加重指数の総市場資本に到達するためには、各企業の1株あたりの価格に、発行済み株式の総数を掛けます。

###キャップ加重複合インデックスの例

-株式A：1株あたりの価格は25ドルで、発行済み株式総数は1,000,000株です。

-株式B：1株あたりの価格は50ドルで、発行済み株式の合計は500,000ドルです。

-株式C：1株あたりの価格は50ドルで、発行済み株式総数は1,000,000株です。

それぞれの時価総額は次のようになります。

-ストックA=$ 25 x 1,000,000 = $ 25,000,000

-株式B=$ 50 x 500,000 = $ 25,000,000

-株式C=$ 50 x 1,000,000 = $ 50,000,000

したがって、コンポジットの時価総額は1億ドルになります。ストックAの重量は25％、ストックBの重量は25％、ストックCの重量は50％になります。通常、[インデックス除数](/indexdivisor)は、レポートの目的でインデックスを管理しやすくするために使用されます。この場合、除算器は$ 100,000になり、初期の複合レベルは$ 100,000,000 / $ 100,000=1,000に等しくなります。

[価格加重指数では、コンポーネント](/priceweightedindex)は価格で加重されます（市場資本や発行済み株式数ではありません）。各株は、1株あたりの価格に比例してインデックスに影響を与えます。価格の高い株は、価格の低い株よりも重みが大きくなるため、その特定の株は、インデックスの全体的なパフォーマンスに大きな影響を与えます。

###株価平均型複合指数の例

[株価平均型指数](/priceweightedindex)では、コンポーネントは時価総額や発行済み株式ではなく、価格で加重されます。各株式は、1株あたりの価格に比例してインデックスに影響を与えます。価格の高い株は、価格の低い株よりもウェイトが高くなるため、インデックスのパフォーマンスに大きな影響を与えます。

-株式A：価格は3ドルに等しい

-株式B：価格は6ドルに等しい

-株式C：価格は30ドルに等しい

-ストックD：価格は$10に等しい

-株式E：価格は1ドルに等しい

複合レベルは、コンポーネントを追加し、その合計をコンポーネントの数で割ることによって求められます。この場合、複合レベルは$ 10（$ 50/5 = $ 10）になります。

##ハイライト

-複合インデックスは、投資分析の実施、経済動向の測定、および市場活動の予測に使用されます。

-十分に多様化されたポートフォリオの目標は、通常、主要な複合インデックス（Nasdaq Composite、Dow、およびS＆P 500）を上回ることです。

-複合インデックスは、市場全体またはセクターのパフォーマンスの表現を作成するために、多くの異なる株式、証券、またはインデックスをグループ化する統計ツールです。
