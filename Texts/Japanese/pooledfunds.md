# プールされた資金
##プールされた資金とは何ですか？

プールされた資金は、投資の目的で集約された多くの個人投資家からのポートフォリオ内の資金です。相互基金、ヘッジ基金、為替取引基金、[年金基金](/pensionplan)、およびユニット投資信託はすべて、専門的に管理されたプール基金の例です。プールされたファンドへの投資家は、投資1ドルあたりの取引コストの削減と多様化を可能にする[規模の経済](/economiesofscale)から恩恵を受けます。

##プールされた資金の基本

投資クラブ、パートナーシップ、トラストなどのグループは、プールされた資金を使用して、株式、債券、相互資金に投資します。プールされた投資口座により、投資家は単一の口座保有者として扱われ、個別に購入できるよりも多くの株式をまとめて購入でき、多くの場合、より良い（割引された）価格で購入できます。

[相互資金](/mutualfund)は、プールされた資金の中で最もよく知られているものの1つです。専門家によって積極的に管理されていますが、インデックスファンドでない限り、さまざまな投資ビークルに持ち株を分散させ、単一またはクラスの証券がポートフォリオ全体に与える影響を軽減します。相互資金には数百または数千の証券が含まれているため、1つの証券のパフォーマンスが低下しても、投資家への影響は少なくなります。

プールされたファンドのもう1つのタイプは、ユニット投資信託です。これらのプールされた資金は、株式、債券、およびその他の証券に投資するために小規模な投資家からお金を受け取ります。ただし、相互ファンドとは異なり、ユニット投資信託はファンドの存続期間にわたってポートフォリオを変更せず、一定期間投資します。

##プールされた資金の長所と短所

###利点

プールされた資金を使用すると、投資家のグループは、大規模な投資家だけが通常利用できる機会を利用できます。さらに、投資家は取引コストを節約し、ポートフォリオをさらに多様化します。ファンドには数百または数千の影響を受ける証券が含まれているため、1つの証券の[パフォーマンスが](/underperform)[ミリ秒未満の場合、投資家は少なくなり](/underperform)ます。

専門的な管理は、投資家がファンドの目的に沿って仕事をしながら、最高のリスクとリターンのトレードオフを確実に受けられるようにするのに役立ちます。この管理は、自分の投資を完全に処理するための時間と知識が不足している可能性のある投資家を支援します。

特に相互資金は、非常に攻撃的で、やや攻撃的で、リスクを嫌う投資家にさまざまな投資オプションを提供します。相互資金は、追加の資金株を購入できる[配当と利息](/dividend)の再投資を可能にします。投資家は、ポートフォリオを拡大しながら、ファンドのポートフォリオバスケットに含まれるすべての証券を保持するための取引手数料を支払わないことでお金を節約します。

###デメリット

お金がグループファンドにプールされると、個人投資家は、グループの投資決定を単独で行う場合よりも、グループの投資決定を制御することができなくなります。すべてのグループ決定がグループ内の各個人に最適であるとは限りません。また、何を購入するかを決める前に、グループはコンセンサスに達する必要があります。市場が不安定な場合、合意に達するために時間と労力を費やすことは、迅速な利益または潜在的な損失を減らすための機会を奪う可能性があります。

専門的に管理されたファンドに投資する場合、投資家はそれを運営するマネーマネージャーにコントロールを放棄します。さらに、彼は管理費の形で追加費用を負担します。管理資産（AUM）のパーセンテージとして毎年請求される料金は、ファンドのトータルリターンを減少させます。

一部の投資信託はまた、負荷または販売料金を請求します。資金はこの料金が請求される時期によって異なりますが、最も一般的なのは、購入時に支払われるフロントエンドの負荷と、売却時に支払われるバックエンドの負荷です。

[キャピタルゲイン](/capitalgain)に対して申告し、税金を支払います。これらの利益はすべての投資家に均等に分配されますが、売却された持ち株から時間の経過とともに利益を得る機会を得られなかった新しい株主を犠牲にすることもあります。

ファンドが持ち株を頻繁に売却する場合、キャピタルゲインの分配が毎年行われ、投資家の課税所得が増加する可能性があります。

##プールされたファンドの例

バンガードグループ株式会社は、世界最大の投資運用会社および退職年金サービスのプロバイダーの1つです。同社は、世界中の投資家に何百もの異なる投資信託、ETF、およびその他のプールされたファンドを提供しています。

たとえば、カナダの子会社であるVanguard Investments Canadaは、カナダの投資家に多くのプールファンド製品を提供しています。これらの製品には、39のカナダETFと4つの相互ファンド、12のターゲット退職ファンドと8つのプールファンドが含まれます。後者の2つのグループは機関投資家が利用できます。

プールされたファンドの1つであるVanguardGlobalex-Canada Fixed Income Index Pooled Fund（CAD-hedged）は、外国の債券に投資しています。 2019年4月、カナダのポートフォリオ提供に中国政府の政策銀行債を含めることを利用するために、新しいベンチマークであるBloomberg Global Aggregate ex-CAD Float Adjusted andScaledIndexを採用しました。

##ハイライト

-プールされたファンドは、1つの巨大なポートフォリオとして投資し、多数の個人からの資本を集約します。

-相互資金やユニット投資信託（UIT）など、多くのプールされた資金は専門的に管理されています。

-プールされた資金により、個人は大規模な機関投資家だけが利用できる規模の機会にアクセスできます。
