# S＆amp; P/TSXコンポジットインデックス
## S＆P / TSXコンポジットインデックスとは何ですか？

S＆P / TSXコンポジットインデックスは、カナダの主要証券取引所である[トロント証券取引所](/toronto-stock-exchange-tsx)（TSX）に上場している最大の企業の業績を追跡する[資本化加重株式インデックスです。](/capitalizationweightedindex)

これは、米国の[S＆P 500インデックスに相当するため、カナダの投資家によって綿密に監視されています。](/sp500) S＆P / TSX総合指数は、カナダで最大かつ最も著名な企業で構成されているため、カナダ経済の健全性のバロメーターとしてよく使用されます。

## S＆P/TSXコンポジットインデックスを理解する

S＆P / TSXコンポジットインデックスは、 [Standard and Poor's](/sp) （S＆P）によって計算されます。 S＆P / TSXインデックスファミリーの中で最も広く、トロント証券取引所（世界で9番目に大きい取引所）に上場している約1,500社のうち、約230から250のカナダの上場企業が含まれています。ただし、これらの企業は、カナダの株式市場の約95％、TSXの市場資本全体の約70％を占めています。

2022年4月の時点で、インデックスの時価総額（米ドル）は2.75兆ドルを超えています。インデックスのメンバーの約3分の1は、銀行、投資会社、金融サービス会社、保険会社で構成される[金融セクターに属していました。](/financial_sector)インデックスに大きく表されている他のセクターには、[エネルギー](/energy_sector)、[材料](/basic_materials)、産業、および情報技術が含まれていました。

> S＆P / TSXコンポジットインデックスに表示される残りのセクターには、コミュニケーションサービス、ユーティリティ、コンシューマーステープル、コンシューマー裁量、不動産、ヘルスケアが含まれます。全部で11のセクターがあります。

資本化加重指数として、S＆P / TSX複合指数は、小規模企業よりも大規模な会員企業の影響を強く受けています。これはストック[インデックス](/nasdaqcompositeindex)を計算する一般的な方法であり、S＆P500と[NasdaqCompositeIndex](/nasdaqcompositeindex)はどちらもこのアプローチを使用しています。

ほとんどのベンチマークインデックスと同様に、S＆P / TSXには3つの主要な機能があります。国の公的企業の業績を簡単に把握できるスナップショットを提供すること、ファンドマネージャーが結果を比較できる基準を提供すること、[交換する形式化された構造を提供することです。](/etf)[トレードファンド](/etf)（ETF）とインデックスファンドがフォローできます。

## S＆P/TSX複合インデックスの適格性基準

[流動性](/liquidity)と[市場資本](/marketcapitalization)に関する一連の適格基準を満たさなければなりません。具体的には、メンバー企業の株価が指定された期間を超えて1ドルを下回った場合、メンバー企業はインデックスから削除されます。同様に、メンバーは、市場資本がインデックスの少なくとも0.04％のままであることを確認する必要があります。

十分な流動性も重要です。流動性は、過去12か月間に米国とカナダで取引された株式の総数を、期末に発行された浮動株調整後の[株式](/outstandingshares)数で割ったものとして測定され、新規参入者の場合は少なくとも0.50、少なくとも既存の構成要素の場合は0.25。[二元上場株式の](/duallisting)場合、カナダのボリュームのみを使用する場合は、少なくとも0.25、または既存のメンバーの場合は0.125の流動性が必要です。

含まれるためには、企業はカナダ人としての資格も必要です。つまり、カナダで設立、形成、または設立され、TSXに一次証券取引所に上場し、[財務諸表](/financial-statements)やその他の開示文書を現地の規制当局に提出し、国内に「実質的な存在」を持っている必要があります。本社または主要な執行部がカナダにあるか、国内の[固定資産](/fixedasset)と収益のかなりの部分を持っている。

## S＆P/TSX総合指数への投資方法

S＆P / TSX総合指数は、[ベンチマーク](/benchmark)と投資可能な指数の両方として機能します。

投資家は、もちろん、インデックス内の個々の株を選択することで、インデックスへのエクスポージャーを得ることができます。もちろん、200を超える株があることを考えると、少し非現実的です。または、少なくとも上位10株です。グローバルな取引とオンラインプラットフォームの出現を考えると、外国の株式を直接購入することはこれまでになく簡単になりましたが、米国を拠点とする投資家は依然として[米国の預託機関の領収書](/adr)（ADR）を購入する方が便利だと感じています。これらは通常、インデックスにあるような大規模なカナダの企業が利用できます。

ただし、最大の経済性と多様化のためには、S＆P/TSXコンポジットインデックスを追跡するファンドに投資する方がおそらくより現実的です。その1つがiSharesS＆P / TSX 60インデックスETF（TSX：XIU）で、その名前が示すように、インデックスで最大の60社を保持しています。

他の2つのETF（iShares S＆P / TSXキャップドコンポジットインデックス（TSX：XIC）およびBMO S＆P / TSXキャップドコンポジットインデックス（TSX：ZCN））は、サブセットインデックスであるS＆P/TSXキャップドコンポジットを追跡します。このインデックスは、S＆P / TSXコンポジットのすべての構成要素に10％の上限ウェイトを課して、揮発性を低減します（パフォーマンスを低下させる可能性もあります）。

S＆P/TSXコンポジットインデックスを追跡する相互ファンドもあります。たとえば、Scotia Canadian Equity Index Fundは、インデックスの株式に投資することにより、長期的な資本成長を達成することを目的としています。

## S＆P/TSXコンポジットインデックスの歴史

S＆P / TSXコンポジットインデックスは、以前のインデックスであるTSE 300から開発されました。TSE300は、1977年にトロント証券取引所によって発売されました（TSEは取引所の以前の略語でした）。 S＆P 500をモデルにしており、正確には300の固定数の株式が含まれていました。したがって、Standard＆Poor's（現在のS＆P Dow Jones Indices）という名前は、2002年にインデックスの管理を引き継ぎ、名前を変更しました。

何年にもわたって、インデックス内の正確な企業数は変動し、インデックスの構成は変化しました。当初は、資源が豊富なカナダを反映して、鉱業と石油会社が支配的でした。 21世紀になると、それは変化し始め、S＆P / TSX総合指数が2014年に17の資源会社を削除し、2015年に16のほとんど非資源会社を追加したとき、それは見出しになりました。 2016年には、金融会社がインデックスの20％を占めました。 2021年には、最大のセクターである31.7％を占めています。

インデックスは1977年の発売以来着実な成長を記録しました。近年、COVID-19の大流行の発生中に、特に2020年の初めを含め、いくつかの大きなクラッシュが発生しました。インデックスは最終的に2020年3月に底を打ち、それ以来前例のないレベルで実行されています。 2022年3月9日、S＆P/TSXコンポジットインデックスは過去最高の22,087に達しました。

##ハイライト

-これはカナダ経済のバロメーターと見なされており、米国のS＆P500インデックスに類似しています。

-S＆P / TSXコンポジットインデックスは、ベンチマークと投資可能なインデックスの両方として機能し、主にETFとそれを追跡する相互資金を通じて個人投資家がアクセスできます。

-企業は、インデックスの一部であり続けるために、厳格な流動性と市場資本の要件を維持する必要があります。

-S＆P / TSXコンポジットインデックスは、カナダ最大の公営企業約250社を追跡するベンチマークエクイティインデックスです。

-2022年第2四半期の時点で、インデックスの総市場上限（米ドル）は2.75兆ドルを超えており、トロント証券取引所全体の約70％に相当します。

### TSXに掲載されるのにどれくらいの費用がかかりますか？

トロント証券取引所（TSX）の上場料金は、10,000ドルから200,000ドルの範囲です。 TSXベンチャーエクスチェンジの場合は7,500ドルから40,000ドル。正確な金額は、上場時点での会社の市場価値によって決定されます。

### S＆P TSXコンポジットインデックスにはいくつの株がありますか？

S＆P / TSX総合指数は通常、一度に230から250の株式を保有しています。固定数を維持していません。 2022年第2四半期の時点で、239人の構成員がいました。

### TSXが上がるとはどういう意味ですか？

カナダの株式の全体的なパフォーマンスは、米国の主要な株式市場であり、北米で3番目に大きいトロント証券取引所（TSX）が上昇するか下降するかによって判断されます。人々が「TSXが稼働している」と言うとき、彼らは一般に、トロント証券取引所の250余りの最大の企業のバスケットであるS＆P/TSXコンポジットインデックスについて話します。
