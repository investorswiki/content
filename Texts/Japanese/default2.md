# デフォルト
##デフォルトとは何ですか？

デフォルトは、債務がローンであるか証券であるかにかかわらず、債務に対して必要な利息または元本の返済を行わないことです。個人、企業、さらには国でさえ、債務不履行に陥ることがあります。[デフォルトリスク](/defaultrisk)は債権者にとって重要な考慮事項です。

##デフォルトの説明

デフォルトは、住宅で担保された住宅ローンや会社の資産で担保されたビジネスローンなどの[担保付き](/secureddebt)[債務で発生する可能性があります。](/secureddebt)借り手がタイムリーな支払いを怠った場合、ローンはデフォルトになり、それを確保するために使用された資産は危険にさらされる可能性があります。同様に、その債券に必要なクーポン支払いを行うことができない会社はデフォルトになります。

デフォルトは、クレジットカードの残高などの[無担保債務でも発生する可能性があります。](/unsecureddebt)デフォルトは借り手の信用格付けを低下させ、将来借りる能力を制限する可能性があります。

##担保付債務と無担保債務のデフォルト

個人、企業、または国が債務不履行に陥った場合、その貸し手または投資家は資金を回収するために訴える可能性があります。彼らの回復の見通しは、債務が担保されているかどうかに一部依存します。

###担保付債務

借り手が住宅ローンの債務不履行に陥った場合、銀行は最終的に住宅ローンを担保する住宅を[差し押さえることが](/foreclosure)あります。借り手が自動車ローンをデフォルトにした場合、貸し手は車両を取り戻すことができます。これらは担保付ローンの例です。担保付きローンでは、貸し手はローンで取得した特定の資産に対する法的請求権を持っています。

、債権者との和解交渉の時間を提供し、[没収を](/forfeiture)[回避](/forfeiture)するために破産保護を申請することができます。

###無担保債務

デフォルトは、医療費やクレジットカードの残高などの無担保債務でも発生する可能性があります。無担保債務は資産で担保されていませんが、債務不履行が発生した場合でも貸し手は法的請求を行います。クレジットカード会社は、アカウントをデフォルトに戻す前に数か月待つことがよくあります。未払いの残高の支払いがない状態で6か月以上経過すると、債務は償却されます。つまり、貸し手はそれを損失として償却し、口座を閉鎖します。次に、債権者は、償却された債務を回収機関に売却することができ、回収機関は借り手から回収を試みます。

>債務不履行に無担保債務が含まれる場合、債務を購入する回収機関は、借り手の資産に対してリーエンまたは判決を下す場合があります。[判決リーエン](/judgment-lien)は、債務者が契約上の義務を履行しなかった場合に債権者に債務者の財産を所有する権利を与える裁判所の判決です。

##学生ローンのデフォルト

学生ローンは別の種類の無担保債務です。学生ローンの返済に失敗した場合、2016年に学生の借金に起因する逮捕状を持ったテキサス人男性のように、武装した米国元帥のチームが玄関先にいない可能性があります。しかし、それでも非常に悪い考えです。その借金を無視します。

ほとんどの点で、学生ローンの不履行は、クレジットカードの支払いを怠った場合と同じ結果になります。ただし、1つの重要な点で、それははるかに悪化する可能性があります。連邦政府はほとんどの学生ローンを保証しており、借金の回収者は連邦政府が雇用する権限を持つことを夢見ています。おそらくあなたのドアにいる武装した元帥ほど悪くはないでしょうが、それは非常に不快になる可能性があります。

###まず、あなたは「滞納している」

ローンの支払いが90日遅れると、正式に[滞納し](/delinquent)ます。その事実は、3つの主要な[信用調査機関すべてに報告されてい](/creditbureau)ます。あなたの信用格付けは下がります。つまり、クレジットの新規申請は拒否されるか、より高い金利でよりリスクの高い借り手にのみ承認される可能性があります。

悪い信用格付けは他の方法であなたをフォローすることができます。潜在的な雇用主は、特にセキュリティクリアランスを必要とする従業員の場合、申請者のクレジットスコアを確認することがよくあります。多くの家主もそうです。

###次に、あなたは'デフォルトで'

支払いが少なくとも270日遅れると、ローンはデフォルトになります。ほとんどのデフォルトの学生ローンは、米国教育省によって保持されています。

連邦学生援助局のデフォルト解決グループとローンリハビリ契約を締結していない借り手は、最終的には、税金の払い戻しやその他の連邦支払いの差し控え、および持ち帰り支払いの最大15％の保証の対象となる可能性があります。

財務オフセットプログラムとして知られるこのような収集は、連邦政府のCOVID-19救済措置の一環として、2023年2月28日まで停止されました。

>教育省によると、学生ローンが滞納している、またはデフォルトであるすべての借り手は、支払いが再開されたときに「新たなスタート」を提供されます。

>>

###デフォルトの代替

良い最初のステップは、あなたがあなたの支払いに追いつくのに問題があるかもしれないことに気づいたらすぐにあなたの貸し手に連絡することです。貸し手は、より達成可能な返済計画についてあなたと協力したり、ローンの支払いの延期や猶予を得るのを手伝ったりすることができるかもしれません。 COVID-19の救済措置として、学生ローンの支払いと未払いのローンに対する利息の蓄積は、2022年8月31日まで教育省によって停止されたことに注意してください。

>連邦学生ローンがデフォルトになっている場合は、連邦学生ローンのリハビリプログラムに参加するか、ローンの整理を使用できます。

>>

##ソブリンのデフォルト

[ソブリンのデフォルト](/sovereign-default)は、国が債務を返済しない場合に発生します。個人または企業の債務者とは異なり、デフォルトの国は、他のさまざまなリスクや問題に直面していますが、通常、裁判所による義務の履行を強制することはできません。

経済が不況に陥るか、通貨が[切り下げられる可能性があり](/devaluation)ます。債務不履行国は、今後数年間、債券市場から締め出される可能性があります。

ソブリンのデフォルトは、政情不安、経済の管理ミス、銀行危機など、さまざまな理由で発生する可能性があります。 2015年、ギリシャは、欧州連合からの追加の債務救済を確保する前に、[国際通貨基金（IMF）](/imf)への17億3000万ドルの支払いをデフォルトしました。

##先物契約のデフォルト

先物[契約の不履行](/futurescontract)は、一方の当事者が契約で定められた義務を履行しない場合に発生します。ここでのデフォルトは通常、必要な日付までに契約を決済できないことを伴います。先物契約は、特定の商品または資産を含む将来の取引に関する法的合意です。契約の一方の当事者は特定の日付と価格で購入することに同意し、もう一方の当事者は契約で指定されたマイルストーンで販売することに同意します。

##ローンをデフォルトするとどうなりますか？

借り手がローンの債務不履行に陥った場合、その結果には次のようなものがあります。

-借り手の[信用報告書に対する否定的な意見](/creditreport)と低い[信用スコア](/credit_score)、借り手の信用度の数値的尺度

-将来的にクレジットを取得する可能性が低くなります

-新しい債務のより高い金利

-賃金およびその他の罰則の付与。飾り付けとは、借り手の賃金または銀行口座から直接支払いを差し引くように第三者に指示する法的手続きを指します。

>クレジットビューローExperianによると、デフォルトはクレジットレポートに残り、7年間クレジットスコアに考慮されます。

>>

##デフォルトの実際の例

プエルトリコは、2015年に5800万ドルの債券支払いに対して628,000ドルしか支払わなかったときにデフォルトになりました。 2017年のハリケーンマリアによる被害は、島の経済と債務の危機を悪化させました。

2019年、プエルトリコは、米国史上最大の破産であった1,290億ドルから約860億ドルに債務を削減する計画を発表しました。破産申請は、議会で可決された2016年の法律に基づいて承認されました。プエルトリコの監視、管理、および経済的安定法（PROMESA）も、地域の公的財政を監視するための財務監視委員会を設立しました。

2022年初頭、米国の裁判官は、破産プロセスの一環として、プエルトリコの700億ドルの公的債務を約半分に削減するリストラ計画を承認しました。

##ハイライト

-デフォルトは借り手を法的請求にさらし、将来の信用へのアクセスを制限する可能性があります。

-債務不履行は、借り手が債務に対して必要な支払いを行うのをやめたときに発生します。

-米国政府は、COVID-19の救済措置として、2022年8月31日まで学生ローンの回収と利息の発生を一時停止しました。

-デフォルトは、住宅で担保された住宅ローンなどの担保付き債務、またはクレジットカードや学生ローンなどの無担保債務で発生する可能性があります。
