# 認識された損失
##認識された損失とは何ですか？

認識された損失は、投資または資産がその購入価格よりも低い価格で売却された場合に発生します。認識された損失は、[所得税](/incometax)の目的で報告され、その後、将来の期間に繰り越され、投資家が認識された利益に対して支払わなければならないキャピタルゲイン税を減らすことができます。

##認識された損失はどのように機能するか

個人または企業が[資本資産を購入する場合、](/capitalasset)その評価は時間の経過とともに変動し、購入価格に対して上昇または下降する可能性があります。認識された価値の変動は、処分されるまで利益または損失としてカウントされません。売却時に資産に[キャピタルロス](/capitalloss)が発生した場合、それに対して請求を行うことができます。

[税務計画](/tax-planning)戦略に使用できます。たとえば、投資家が特定の年に$ 10,500の課税対象の[キャピタルゲイン](/capitalgain)を持ち、別の投資の$ 2,500の損失を認識できる場合、この損失を課税対象のキャピタルゲインに適用できます。このような状況では、この投資家の年間の正味課税キャピタルゲインは、10,500ドルではなく、8,000ドルになります。

>投資損失は、毎年一定の限度額（現在は3,000ドル）までの投資利益またはその他の収入に対して償却することができ、これを超える金額は将来の使用のために繰り越すことができます。

>>

認識された損失は、特定の年数まで適用することもできます。つまり、企業または個人が特定の年に[課税所得](/taxableincome)を持たない場合、認識された損失は、代わりに将来の日付で利益に対する税金を相殺する可能性があります。

[税務上の損失の収穫は、認識されたキャピタルロスを使用して課税所得を相殺または削減する可能性があります。これは](/taxgainlossharvesting)、ポートフォリオを[多様化](/diversification)または[リバランス](/rebalancing)するために、望ましくない投資を売却し、より魅力的な投資に置き換えることをすでに計画している投資家にとって特に役立ちます。これには、パフォーマンスが低下したファンドの株式の売却が含まれる場合もあれば、負担となる不動産に関連する場合もあります。

>車や家などの個人使用の資産の売却による損失は、税控除の対象にはなりません。

>>

##認識された損失と実現された損失

投資または資産の処分後[の](/realizedloss)「認識された損失」と[実現された損失を区別することが重要です。](/realizedloss)意味は異なりますが、両方の用語は互いに混同されます。実現損失は、投資家が取引を完了した直後に実現されますが、税金には影響しません。キャピタルゲインから差し引かれるのは、認識された損失のみです。

ほとんどの投資資産の売却は、実現損失と認識損失の両方を同時に発生させます。通常は、取引の直後です。[内国歳入庁](/irs)（IRS）は、税法に具体的に記載されている特定の取引の税効果を遅らせ[ます](/tax-code)。売却によって税効果が遅れる場合、実現損失は発生しますが、認識損失は発生しません。

##特別な考慮事項

実現された認識されない損失を生み出す可能性のあるかなり一般的なトランザクションの1つは、[同種交換](/like-kind_exchange)です。これらの取引は、1031取引所またはスターカー取引所とも呼ばれ、2つの納税者が、2つの賃貸物件を相互に取引するなど、同様の資産を交換するときに発生します。

> 2017年12月、同種交換を不動産に限定する新しい規則が導入されました。以前は、企業間の有形および無形資産の交換も許可されていました。

>>

この手法は、納税者が故意に自分の財産を価値の低いものと交換するときに、意図的な将来の損失を導くために使用される可能性があります。ただし、認識されたキャピタルロスは、投資家が後で新しい資産を売却したときにのみ開始されます。

##ハイライト

-内国歳入庁（IRS）自体が、特定の取引の税効果を遅らせる可能性があります。

-認識された損失は将来にも適用できるため、個人や企業は、課税所得が多い時期に税額を減らすことができます。

-認識された損失とは、投資または資産が購入価格よりも低い価格で売却された場合です。

-売却時に資産にキャピタルロスが発生した場合、この損失はキャピタルゲイン税から差し引くことができます。
