# ちこうスパン（遅れスパン）
##チコウスパン（ラギングスパン）とは？

Chikouスパンは、 [Ichimoku Kinko Hyo](/ichimokuchart) 、またはIchimokuCloudインジケーターのコンポーネントです。 「ラグスパン」とも呼ばれ、資産の最新の終値から26期間遅れて[終値をプロットすることによって作成されます。](/closingprice) Chikouスパンは、トレーダーが現在のトレンドと以前のトレンドの関係を視覚化し、潜在的なトレンドの逆転を見つけることができるように設計されています。

チコウスパンが価格より上に表示される場合は上昇[傾向](/trendtrading)と見なされ、インジケーターが価格より下に表示される場合は下降傾向と見なされます。多くのトレーダーは、潜在的なトレンドの変化を示すために、チコウスパンが以前の価格と交差するのを監視しています。

##チコウスパン（ラグスパン）の計算方法

1.最後の終値をメモしてから、この値を26期間前にプロットします。

1.新しい終値ごとにこのプロセスを繰り返します。

1.すべての値を接続して、1本の線を作成します。

>デフォルト設定は26期間ですが、この数値を変更して、スパンと価格の間の距離を増減できます。

>>

##チコウスパンを理解する（ラグスパン）

[雲](/ichimoku-cloud)としても知られる一目の均衡表の5つの主要な路線の1つです。 1969年に日本のジャーナリスト細田剛一によって開発された一目の雲は、トレーダーが資産のトレンドと[勢いを測定するために使用する技術的指標です。](/momentum)他の要素は、天満線、紀順線、千光スパンA、千光スパンBです。

>「ちこう」は日本語で「地溝帯」を意味します。

>>

インジケーターを使用する重要な方法の1つは、現在の価格との関係を表示することです。価格がラインより上に表示されている場合、それは多くの場合、価格に弱点があることを示しています。価格がチコウスパンを下回っている場合、それは通常、価格に強みがあり、価格が上昇していることを示しています。

これは、価格がチコウスパンと前後に交差している場合には役に立ちません。トレンドがまだ存在するか、価格アクションが[不安定](/choppymarket)である可能性がありますが、一目の雲指標の他の要素は、トレンドの方向性についてのより良い洞察を提供する可能性があります。

上記のことを考えると、チコウスパンが価格を超えると、これはトレンドの[逆転を示す場合があり](/reversal)ます。理想的には、価格とチコウのスパンは、しばらくの間、それらの間にある程度の距離があります。上記のように、価格とチコウが絡み合っている場合、信号は信頼できません。

<！-6D7315B171A3E4AD88DC72110D29DDBF-->

チコウスパンが価格を超えたとき、価格の上昇傾向[が](/uptrend)始まったことを示す可能性があります。チコウが価格を上回れる唯一の方法であるため、価格はすでに上昇し始めています。同様に、チコウが（しばらく離れた後）価格を下回った場合、それは価格が下がり始め、下がっていることを示している可能性があります。

ほとんどの一目の均衡表戦略は、一目のスパンを勢いの指標として、また他の4つの一目のラインとの関係に基づく二次[確認ツールとして採用しています。](/confirmation)

チコウスパンのもう1つの用途は、[抵抗](/resistance)または[サポートのポイントを確認するのに役立つこと](/support)です。チコウスパンは終値の高値と安値に一致しますが、それらから相殺されるため、これは他の何よりも視覚的な確認です。

##チコウスパン（遅延スパン）と単純移動平均（SMA）

どちらも[遅れている指標](/laggingindicator)ですが、方法が異なります。

ちこうスパンは平均ではありません。過去にプロットされた終値です。一方、単純移動平均（SMA）は、複数の期間にわたる平均価格です[。](/sma)それは平均的であり、したがって価格変動に即座にそして完全に反応することができないので、それは遅れます。

最新のSMA値はチャートの右側と最新の価格に揃えられますが、チコウのスパンは最新の価格の左側の26期間です。

##チコウスパン（ラグスパン）の使用制限

遅れているスパンは、過去にプロットされた終値です。この式には本質的に予測できるものはありません。

[クロスオーバーはトレンドの変化](/crossover)を示すことができますが、多くの[誤ったシグナルがあります](/false-signal)。価格とチコウスパンは、意味のある価格変動やトレンドの変化なしに交差することがよくあります。このため、このインジケーターは一目の雲インジケーターの他の要素と組み合わせて使用する必要があります。

クロスオーバーがトレンドの変化をもたらす場合、クロスオーバーが発生した理由として、価格はすでにその方向に大幅に移動しています。信号が到着するまでに価格がすでに大幅に変動している場合、それは必ずしも価値のある取引機会ではない可能性があります。

トレーダーはまた、[価格行動](/price-action)と[傾向分析](/trendanalysis)、および基本的な分析と他の技術的指標を彼らの取引に取り入れたいと思うかもしれません。

##ハイライト

-一目の指標の他の要素と組み合わせて使用され、伝統的にそれ自体でトレードシグナルを生成するために使用されることはありません。

-これは、最後のローソク足/バーから26期間遅れて終値をプロットすることによって作成されます。

-資産の勢いを測定し、潜在的なトレンドの変化を特定するために使用されます。

-一目のスパンは、一目の均衡表の5つの要素の1つです。
