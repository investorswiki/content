# 免税証券
##免税証券とは何ですか？

免税証券とは、生み出された収入に連邦税、州税、地方税が含まれない投資です。ほとんどの免税証券は、州、準州、または地方自治体の義務を表す地方債の形で提供されます。一部の投資家にとって、米国貯蓄債券の利子は連邦所得税も免除される場合があります。

##免税セキュリティの仕組み

配当や利子などの免税証券の所得には、[連邦税](/federal_income_tax)は適用されません。投資家が住んでいる場所によっては、免税証券はすべての税金が免除される場合があります。州内居住者は通常、本国からの[一般義務公債](/generalobligationbond)に対する州および連邦の免税を受けます。地方債は非課税証券の最も一般的な参照ですが、地方債、[米国貯蓄債券、またはその他](/ussavingsbonds)[の非課税証券に投資する相互資金](/ussavingsbonds)も非課税ステータスを取得できます。連邦国債、すなわち米国貯蓄債券および[財務省インフレ保護証券](/tips)（TIPS）は、連邦レベルで課税されますが、州税および地方税は免除されます。

たとえば、地方自治体がレクリエーション公園の資金を調達するために地方債を発行するとします。発行状態にある投資家のジョン・スミスは、 2年で満期になり、毎年支払われる3％の[クーポン率を持つ5,000ドルの](/coupon-rate)[額面債券を購入します。](/parvalue) 2年のそれぞれの終わりに、投資家は3％x $ 5,000 =$150の利息収入を受け取ります。この収入は、連邦政府または州政府によって課税されません。債券が満期になると、ジョン・スミスは地方自治体から元の元本投資を受け取ります。

州政府、地方自治体、および非営利団体は、これらのプロジェクトの資金調達に使用される免税債が低金利であり、したがって[借入コストが低いことを考慮して、新しいプロジェクトに着手することが奨励されて](/costofdebt)います。投資家は、節税がこの低い利回りを補うのに十分重要であるかどうかを判断する必要があります。

投資家の[限界税率が高いほど、投資家](/marginaltaxrate)にとってより価値があり有益な免税証券です。免税証券は、投資家の税率によって決定されるように、多くの場合、現在の利回りよりも高い[税相当の利回りを運びます。](/taxequivalentyield)税相当利回りは、同じ税引後金利を提供するために必要となる課税対象金利です。免税債の税相当利回りは、次のように計算できます。

税相当利回り=免税利回り/（1 –限界税率）

たとえば、上記の例のJohn Smithが35％の課税範囲に該当する場合、3％の地方債の利回りは、利回りが次の課税対象の債券に相当します。

= 0.03 /（1 – 0.35）

= 0.03 / 0.65

= 0.046、つまり4.6％

ジョン・スミスが22％の税率であった場合はどうなりますか？税相当利回りは次のようになります。

= 0.03 / 0.78

= 0.038、つまり3.8％

税率が高いほど、税相当利回りも高くなります。これは、免税証券がより高い税率の証券に最も適していることを示しています。

ほとんどの場合、組織は、免税証券を発行する前に、[内国歳入法（IRC）の](/internal-revenue-code)[セクション501（c）（3）に基づいて登録する必要があります。](/501c3-organizations)

##ハイライト

-免税証券は、人が支払わなければならない税金が多ければ多いほど、より価値があり、有益です。

-州、準州、または地方自治体の義務を表す地方債は、免税証券の典型的な例です。

-免税証券では、所得は税負担なしで生み出されます。
