# テックストリート
## Tech Streetとは何ですか？

[テクノロジーセクター](/technology_sector)を指す用語であり、半導体、ソフトウェアとゲーム、パーソナルコンピューター、データストレージ、テレコム、ITサービス、インターネットサービス、その他多数のサブカテゴリに分類されます。 Tech Streetには、Meta（以前のFacebook）、Google、Apple、IBM、Microsoft、TexasInstrumentsなどの企業が含まれます。

##TechStreetを理解する

大規模な証券取引所での[ウォールストリート](/wallstreet)、[ベイストリート、ダラルストリート](/baystreet)などの用語の換喩的な使用をモデルにしています。これらは、それぞれ米国、カナダ、インドの証券取引所の本部を収容する実際の道路を指しますが、TechStreetは実際の場所を指しません。

金融ニュース組織は、見出しでTech Streetという用語を使用して、テクノロジーセクターの動きや出来事について語っています。

##テックストリートとその他の市場

Tech Streetは、グローバル市場と金融市場において重要なセクターです。今日の世界では、テクノロジーは世界貿易を促進し、投資家がマウスをクリックするだけで証券を売買できるようにします。

Tech Streetの企業は、企業だけでなく消費者にも重要なサービスを提供しています。何年にもわたって、TechStreetに代表される製品とサービスの範囲は劇的に拡大しました。今日、テクノロジーセクターは大きくて多様な業界であり、[クラウドコンピューティング](/cloud-computing)企業、テレビおよび家電メーカー、ゲームおよびアプリ企業、インターネット企業、ハードウェアメーカーが含まれます。

4つのビッグテックストックは[FANG](/fang-stocks-fb-amzn)で、Meta（META）、Amazon Inc.（AMZN）、Netflix Inc.（NFLX）、およびGoogleの親会社であるAlphabet Inc.（GOOG）で構成されています。 「MadMoney」のJimCramerは頭字語を作り出し、投資家はFANGのパフォーマンスを市場指数と頻繁に比較しています。

テクノロジーセクター全体と同様に、FANG株の動きが市場の動きを大きく左右します。つまり、FANGが上がると、市場は上がります。 FANGが下がると、市場は下がる。 [FAANG](/faang-stocks)は同じ4つの株で、Apple Inc.（AAPL）が追加されています。

##TechStreetが手に負えなくなった例

多くの新しく革新的な企業がテクノロジー分野にあります。これは、スペースに[成長株](/growthstock)がたくさんあることを意味します。したがって、テクノロジー株は高[株価](/price-earningsratio)[収益率](/price-earningsratio)（P / E）の倍数で取引される傾向があります。高いPEは1つのことですが、手に負えなくなる可能性があります。

[テクノロジーバブル](/tech_bubble)の間に、多くの会社の株が急騰しましたが、会社には売上も収益も、事業計画さえもありませんでした。ナスダック総合指数は、1995年半ばから2000年のピークまでの間に550％以上上昇しました。 2002年後半までに、75％以上減少しました。

ハイテクストリート企業は、セクシーな新製品やイノベーションを紹介することで知られています。 1990年代の魅力は、インターネットとそれがすべてをどのように変えるかということでした。インターネットは物事を劇的に変えました、しかしビジネスはまだ繁栄するために[ビジネスプラン](/business-plan)と成長する収入を必要とします。これらのない会社は存在しなくなりました

##ハイライト

-Tech streetはテクノロジー株を指します。これは、一般的な株を指すときにウォール街を指す場合と同様です。

-テクノロジー株はイノベーションの最前線にあることが多く、その多くは成長の可能性に基づいて高いP/E比で取引されます。

-Tech Street Stockには、情報配信、インターネットサービスプロバイダー、ワイヤレス通信など、さまざまな業界が含まれます。
