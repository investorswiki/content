# ドージコイン（DOGE）
##ドージコインとは何ですか？

Dogecoin（DOGE）は、ピアツーピアの[オープンソース](/open-source)暗号通貨です。それはアルトコインとほとんど皮肉なミームコインと見なされ[ます](/altcoin)。 2013年12月に発売されたドージコインは、柴犬をイメージしたロゴをロゴにしています。

一見冗談として作成されたように見えますが、Dogecoinの[ブロックチェーン](/blockchain)にはまだメリットがあります。その基盤となるテクノロジーはライトコインから派生しています。 scrypt[アルゴリズム](/algorithm)を使用するDogecoinの注目すべき機能は、その低価格と無制限の供給です。

##Dogecoinを理解する

ドージコインは冗談のようなものとして始まりましたが、作成された後、次のようになりました。 2017年後半までに、多くのコインの価値を大幅に引き上げた[暗号通貨バブルに参加していました。](/cryptocurrency) 2018年にバブルが崩壊した後、Dogecoinはその価値の多くを失いましたが、それを取引し、TwitterやRedditのコンテンツをチップするために使用するサポーターのコアがまだあります。

ユーザーはデジタル通貨取引所でDogecoinを売買できます。彼らはDogecoinを取引所またはDogecoin[ウォレットに保管することを選択できます](/digital-wallet)。

##ドージコインの歴史

###はじめに

オーストラリアのシドニーにあるAdobeInc.のオフィスのプロダクトマネージャーであるJacksonPalmerは、暗号通貨を取り巻く誇大宣伝を風刺する方法として、2013年にDogecoinを作成しました。パーマーは、新しいテクノロジーの「懐疑的な分析」オブザーバーとして説明されており、彼の新しい暗号通貨ベンチャーについての彼の最初のツイートは、冗談めかして行われました。しかし、ソーシャルメディアで肯定的なフィードバックを得た後、彼はドメインdogecoin.comを購入しました。

一方、オレゴン州ポートランドでは、[デジタル通貨の作成を望んで](/digital-currency)いたが、努力を促進するのに苦労したIBMのソフトウェア開発者であるBilly Markusが、ドージコインの話題を発見しました。 Markusは、実際のDogecoinの背後にあるソフトウェアを構築する許可を得るためにPalmerに連絡しました。

[Luckycoin](/proof-work)に基づいたDogecoinのコードであり、それ自体はLitecoinから派生しており、最初はブロックマイニングにランダム化された報酬を使用していましたが、2014年3月に静的な報酬に変更されました。 PoWコイン。

>プルーフオブワーク（PoW）は多くの暗号通貨の基礎を形成し、安全で分散型のコンセンサスを可能にします。

>>

パーマーとマーカスは12月にコインを発売しました。 2013年6月6日。2週間後の12月。 19、おそらく中国が銀行に暗号通貨への投資を禁止したために、ドージコインの価値は300％急上昇しました。

###ドージコインの台頭

ドージコインは、柴犬をロゴにした[ビットコインの「楽しい」バージョンとして売り出されました。](/bitcoin) Dogecoinのカジュアルなプレゼンテーションは、急成長している暗号コミュニティのムードに合っていました。その暗号化技術と無制限の供給は、ビットコインのより速く、より適応性があり、消費者に優しいバージョンの議論でした。

ドージコインは「インフレコイン」ですが、ビットコインのような暗号通貨は、作成されるコインの数に上限があるため、[デフレです。](/deflation) 4年ごとに、マイニング報酬を介して流通するビットコインの量は半分になり、すべてのコインがリリースされるまで、そのインフレーション率はそれに伴って半分になります。

2014年1月、ドージコインコミュニティは、ジャマイカのボブスレーチームのソチ冬季オリンピックへの旅行に資金を提供するために、約3万ドル相当の2700万ドージコインを寄付しました。その年の3月、ドージコインコミュニティは、ケニアに井戸を建設するために11,000ドル相当のドージコインを寄付し、NASCARドライバーのジョシュワイズを後援するために55,000ドルのドージコインを寄付しました。

>最初の誕生日までに、ドージコインの[時価総額](/marketcapitalization)は2,000万ドルで、熱心なファン層がありました。

>>

###論争はドージコインからいくつかの楽しみを取ります

ドージコインの自由奔放な楽しみは、一般的に暗号コミュニティがより深刻になり始めたため、2015年にその魅力の一部を失いました。ドージコインコミュニティのすべてがうまくいっていなかった最初の兆候は、「有毒なコミュニティ」がコインとそれが生み出していたお金の周りで成長したと言ったジャクソンパーマーの出発でした。

その有毒なコミュニティの1人のメンバーは、ムーラと呼ばれるドージコイン取引所を作成した英国市民であるアレックスグリーン、別名ライアンケネディでした。 Alex Green（彼の仮名）は、NASCARの資金調達者に1,500ドルではなく15,000ドルを誤って与えたと伝えられている、豪華なティッパーとしてコミュニティで知られていました。

[取引所](/exchange)の創設に資金を提供するために多額の寄付をするよう説得しましたが、後に彼がその寄付を使って150万ドル以上のビットコインを購入し、それが彼に贅沢なライフスタイルをもたらしたことが明らかになりました。これとは別に、ケネディは2016年に複数のレイプ罪で有罪判決を受け、懲役11年の刑を言い渡されました。

>ドージコインは、2017/18年のクリプトブルランで初めて時価総額10億ドルを超えました。

2019年の夏、Dogecoinは、他の暗号市場とともに価値がさらに上昇しました。ドージコイン愛好家は、暗号通貨取引所[Binance](/binance-exchange)がコインをリストしたときに満足し、多くの人がTesla、Incを考えました。 （TSLA）CEOのElon Muskは、謎めいたツイートでコインを承認しました。

##2020年代のDogecoin

Dogecoinのインフラストラクチャは、ボランティアであるコインの開発者にとって中心的な焦点ではありませんでした。 Dogecoinが運営と取引を継続する理由の1つは、活発な鉱夫のコミュニティです。 CryptoIQのZacharyMashiachは、次のように述べています。

>>

>多くのScryptマイナーは、他のScrypt PoW暗号通貨よりもDogecoin（DOGE）を依然として好んでいます。実際、Dogecoin（DOGE）ハッシュレートは約150 TH/sです。これは、ライトコイン（LTC）のハッシュレートである170 TH / sをわずかに下回っています。これは、ドージコイン（DOGE）をライトコイン（LTC）とマージマイニングできるためです。つまり、マイナーは同じ作業を使用して両方の暗号を同時にマイニングできます。基本的に、Litecoin（LTC）をマイニングする実質的にすべての人が、Dogecoin（DOGE）もマイニングすることを選択します。これは、Dogecoin（DOGE）のマージマイニングが利益を増やすためです。

>>

>>

マスクは2021年にドージコインを公然とサポートし、5月に彼がコインの開発者と協力して取引効率を改善しているとツイートしました。今年の初め、SpaceXの創設者は、ソーシャルメディアで、テスラが支払い方法としてDogecoinを受け入れるべきかどうかを尋ねる投票を実施しました。 10月、シネマチェーンAMCエンターテインメントホールディングス株式会社。 （AMC）は、年末までにデジタルギフトカードの購入にドージコインを受け入れ、ミームベースの暗号通貨にさらに有用性を追加すると発表しました。

10月現在2021年8月8日、Dogecoinの市場キャップランキングは10位で、市場資本は319億ドルで、前年の48ドルと3億3900万ドルの市場価値を大幅に上回りました。

＃＃よくある質問

###どうすればドージコインを購入できますか？

多くの暗号通貨取引所の1つでアカウントを確立することでドージコインを購入できます。 Dogecoinトランザクションをサポートする取引所には、Coinbase、Binance、Krakenなどがあります。暗号取引を可能にする伝統的な証券会社であるロビンフッドも、ドージコインをサポートしています。

###ドージコインはどこで使うことができますか？

あなたはそれを受け入れることを選択したどんな商人ともあなたのドージコインを使うことができます。イーロンマスクのSpaceXやダラスマーベリックスなど、さまざまな種類の企業がDOGEを受け入れています。多くのDogecoin保有者は、DOGEを使用して、Redditやその他のソーシャルメディアプラットフォームでコンテンツ作成者にチップを提供しています。

###ドージコインは安全ですか？

ドージコインは、ビットコインや他の暗号通貨と同じように、ブロックチェーンテクノロジーを使用しています。ブロックチェーン技術は、不可能ではありませんが、ハッキングするのは非常に難しいと考えられています。時価総額が最大で採用率が最も高い暗号通貨の1つであるドージコインは、比較的安全です。
