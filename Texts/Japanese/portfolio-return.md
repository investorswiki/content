# ポートフォリオリターン
##ポートフォリオリターンとは何ですか？

いくつかのタイプの投資を含む[投資ポートフォリオ](/portfolio-investment)によって実現される利益または損失を指します。ポートフォリオは、投資戦略の定められた目的、およびポートフォリオの対象となる投資家のタイプの[リスク許容度](/risktolerance)に基づいて利益をもたらすことを目的としています。

##ポートフォリオリターンを理解する

[ベンチマーク](/benchmark)を満たすことを目指しています。つまり、株式または債券の保有の分散された理論上のポートフォリオ、場合によっては2つの資産クラスの組み合わせを意味します。投資家は通常、投資の中に1つ以上のタイプのポートフォリオを持ち、時間の経過とともにバランスの[取れ](/balancedinvestmentstrategy)[た投資収益率を達成しようとし](/balancedinvestmentstrategy)ます。

[小型株ファンド](/small-cap)から、株式、債券、現金を組み合わせたバランスの[取れたファンド](/balancedfund)まで、さまざまな種類があります。多くのポートフォリオには国際株も含まれ、一部は地理的地域または新興市場のみに焦点を当てています。

多くの投資運用会社は、反対方向に動く傾向がある他のクラスの所有権を通じて、特定のクラスの投資の減少を相殺しようとするポートフォリオを選択します。たとえば、多くの投資マネージャーは、株式が急激なドローダウンを経験すると債券価格が上昇する傾向があるため、債券と株式の両方を混合する傾向があります。これは、ポートフォリオの望ましいリターンを長期にわたって達成し、変動性を滑らかにするのに役立ちます。

>株式や債券など、反対方向に移動する傾向のある資産クラスの組み合わせは、ポートフォリオのバランスを取るための賢い方法であることがよくあります。

##ポートフォリオの収益とリバランス

多くの投資家が従うベストプラクティスは、毎年の終わりにポートフォリオを確認し、投資目標を引き続き達成するために調整を行うことです。

たとえば、投資家は成長基金で例外的な年を過ごし、他の投資家が最終的に価値に戻る可能性があることを期待して、それらの利益の一部を価値基金に移すことを決定する場合があります。

##投資家がポートフォリオの収益に与える影響

投資家がポートフォリオからお金を引き出すことを意図している年齢は、適切な投資目的を選択する上で重要な要素であり続けます。たとえば、退職からわずか数年の投資家は、ポートフォリオの収益を保護したいと考えており、現金、マネーマーケット、および短期債の組み合わせに投資する可能性があります。

逆に、若い投資家は通常、比較的高いリスクを引き受けようとし、株式、ハイイールド債、そしておそらくマネージドフューチャーの組み合わせに投資します。これらはそれぞれ、時間の経過とともにインフレ率を超える可能性があります。

注目すべきことに、インターネット時代の到来により、投資家は市場の収益にほぼリアルタイムでアクセスできるようになり、相対的なパフォーマンスデータにも簡単にアクセスできるようになりました。[相互ファンドに投資](/mutualfund)する場合、投資家はチャートとファンドのリターンをベンチマークインデックスと比較して、ピアグループの平均（通常は10年以上前）、および特定のファンドの上位の[資産配分を引き出すことができます。](/assetallocation)

##ハイライト

-投資家は、時間の経過とともにバランスの取れた投資収益率を達成するために、投資の中にいくつかのタイプのポートフォリオを持っていることがよくあります。

-投資家向けのポートフォリオオプションには、小型株と大型株ファンド、株式対。債券、ETFおよびその他のさまざまな可能性。

-ポートフォリオの収益は、投資ポートフォリオが特定の期間にどれだけの利益または損失をもたらすかを示します。
