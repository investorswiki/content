# クイックレスポンス（QR）コード
##クイックレスポンス（QR）コードとは何ですか？

、デジタル機器で簡単に読み取れる[バーコード](/barcode)の一種で、情報を一連のピクセルとして正方形のグリッドに保存します。 QRコードは、[サプライチェーン](/supplychain)内の製品に関する情報を追跡するために頻繁に使用され、マーケティングおよび広告キャンペーンで頻繁に使用されます。

QRコードは、古い一次元バーコードからの進歩と見なされ、国際標準化[機構](/international-organization-for-standardization-iso)（ISO）によって2000年に国際標準として承認されました。

##クイックレスポンス（QR）コードを理解する

QRコードは、標準のバーコードよりも多くの情報を提供する方法として1990年代に開発されました。トヨタの[子会社で](/subsidiary)あるデンソーウェーブが、製造プロセス全体で自動車を追跡する方法として発明しました。平行線で跳ね返る光を必要とするバーコードとは異なり、QRコードは次のようなデバイスでデジタルスキャンできます。携帯電話として。

QRコードは、白い背景にグリッド（マトリックス）に配置された黒い正方形で構成され、マトリックスに存在するパターンからデータを抽出できる専用のソフトウェアによって読み取られます。これらのコードは、従来のバーコードよりも多くの情報を含むことができ、主に4つのデータモード（英数字、数値、2進数、漢字）を処理します。

データ容量が増えたにもかかわらず、QRコードは消費者に期待したほど人気がありませんでした。消費者が情報を共有するために作成するのではなく、最も一般的には広告主や[マーケティングキャンペーンに関連付けられてい](/marketing-campaign)ます。

##クイックレスポンス（QR）コードとバーコード

製品またはサービスについて伝達できる情報の量は、従来、製品のパッケージまたはその利点を宣伝する広告のスペースの量によって制限されていました。消費者が製品に関する詳細情報（入手可能性、価格、属性）を求めている場合は、営業担当者を探すか、追加のドキュメントを要求する必要があります。

バーコードは通常、製品パッケージの裏側にあり、さまざまな幅の平行線の組み合わせを使用してデータを伝達します。これは、光学スキャナーを備えたマシンで読み取ることができます。

[在庫と価格設定を](/inventory)管理する方法に革命をもたらし、 1960年代に、機器とコンテナを追跡するために米国の鉄道によって最初に実用化されました。従来の2次元バーコードは、1974年に米国の小売店で一般的に使用されるようになりました。現在、バーコードは、従業員IDバッジや病院のブレスレットから輸送用コンテナまで、あらゆるものに使用されています。

##クイックレスポンス（QR）コードの種類

さまざまなアイテムに使用できるQRコードタイプがいくつかあります。それらが含まれます：

-**マイクロQRコード**：スペースが限られている場合に使用される従来のQRコードの小さいバージョン。マイクロQRコードのサイズはさまざまですが、最小のものは11 x 11モジュールで、最大21文字の英数字をエンコードします。

-**モデル1のQRコード：**モデル1は、モデル2とマイクロQRのプロトタイプです。 1〜14のバージョンが、自動識別製造業者インターナショナル（AIMI）標準に登録されています。最大データ容量は468バイトで、最大707文字の英数字をエンコードできます。

-**モデル2のQRコード：**モデル2には、位置調整を改善するための位置合わせパターンがあり、モデル1よりもデータ密度が高くなっています。AIMI標準には1〜40のバージョンが登録されており、バージョン40では最大4,296文字の英数字を保存できます。文字。

-** IQRコード**：スペースや形状が問題になる場合は、正方形または長方形で作成できます。 61のフォーマットのいずれかになります。

-** SQRC **：個人情報を含めるための制限付き読み取り機能を備えています。

-**フレームQR**：グラフィック、イラスト、写真などの形式でより大きなデータを含めることができるカスタマイズ可能なフレーム。

##ハイライト

-従来のバーコードの機能強化であるQRコードは、はるかに多くの情報を保存でき、サプライチェーン管理から暗号通貨ウォレットアドレスまで、さまざまなアプリケーションで使用されます。

-クイックレスポンス（QR）コードは、コンピューターデバイスを使用してデータをエンコードし、すばやく取得するために使用される、暗いピクセルまたは明るいピクセルの正方形のマトリックスです。

-さまざまな目的に合わせてカスタマイズされた、またはより多くのデータを保存できるQRコードのバージョンとバリエーションがいくつか存在します。
