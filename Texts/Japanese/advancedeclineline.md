# 前進/下降（A / D）ライン
##前進/下降（A / D）ラインとは何ですか？

前進/下降線（またはA / D線）は、毎日の上昇株と下降株の数の差をプロットする[技術的な指標です。](/technicalindicator)インジケーターは累積的で、前の数に正の数が加算されます。負の数の場合は、前の数から減算されます。

A / Dラインは、上昇または下降する株がもっとあるかどうかをトレーダーに伝えるため、市場[センチメントを示すために使用されます。](/marketsentiment)主要なインデックスの価格トレンドを確認するために使用され、[発散](/divergence)が発生したときに反転を警告することもできます。

## A/Dラインの計算方法

1.当日高値で終了した株数から当日低値で終了した株数を差し引きます。これはあなたにネットアドバンスを与えるでしょう。

1.これが平均を計算するのが初めての場合、ネットアドバンスがインジケーターに使用される最初の値になります。

1.翌日、その日のネットアドバンスを計算します。正の場合は前日の合計に加算し、負の場合は減算します。

1.手順1と3を毎日繰り返します。

## A / Dラインは何を教えてくれますか？

A / Dラインは、現在のトレンドの強さと逆転の可能性を確認するために使用されます。指標は、株式の大部分が市場の方向性に参加しているかどうかを示します。

指数が上昇しているが、A / Dラインが下向きに傾斜している場合、弱気の[発散と呼ばれ](/divergence)、市場が幅を失い、方向を逆転させようとしていることを示しています。 A / Dラインの傾きが上向きで、市場が上昇傾向にある場合、市場は健全であると言えます。

逆に、指数が下落を続け、A / Dラインが上向きになっている場合は、強気の発散と呼ばれ、売り手が信念を失っていることを示している可能性があります。 A / Dラインと市場の両方が一緒に下落傾向にある場合、価格の下落が続く可能性が高くなります。

## A / Dラインとアームインデックス（TRIN）の違い

A / Dラインは通常、長期的な指標として使用され、時間の経過とともに上昇および下降している株式の数を示します。一方、 [Arms Index](/arms) （TRIN）は、通常、進行中の株式の比率と進行中のボリュームの比率を測定する短期的な指標です。計算と彼らが焦点を当てる時間枠が異なるため、これらの指標は両方ともトレーダーに異なる情報を伝えます。

## A/Dラインの使用に関する制限

[NASDAQ](/nasdaq)株に関して常に正確な測定値を提供するとは限りません。これは、NASDAQが頻繁に小規模な投機的企業をリストアップしており、その多くが最終的に失敗したり、[上場廃止になったりするため](/delisting)です。株式は取引所で上場廃止されますが、A/Dラインの以前に計算された値のままです。これは、累積前値に追加される将来の計算に影響します。このため、NASDAQ関連の指数が上昇している場合でも、A/Dラインが長期間低下することがあります。

注意すべきもう1つの点は、一部のインデックスは[時価総額加重であるということ](/wamc)です。これは、会社が大きくなればなるほど、インデックスの動きに与える影響が大きくなることを意味します。 A / Dラインは、すべての株に等しい重みを与えます。したがって、これは平均的な小型株から[中型株のより良いゲージであり、大型株または](/midcapstock)[メガキャップ株](/megacap)の数が少ないわけではありません。

##ハイライト

-主要指数が低下し、A / Dラインが上昇している場合、時間の経過とともに下落している株式は少なくなります。これは、指数が下落の終わりに近づいている可能性があることを意味します。

-[主要指数](/marketindex)が上昇しているとき、上昇傾向は強い参加を示している上昇傾向を確認します。

-前進/下降（A / D）線は、株式市場の上昇または下降に参加している株式の数を示すために使用される[幅の指標です。](/breadthindicator)

-主要指数が上昇しており、A / Dラインが低下している場合は、上昇している銘柄が少ないことを示しています。これは、指数が上昇の終わりに近づいている可能性があることを意味します。

-主要なインデックスが減少している場合、下降する前進/下降線は下降傾向を確認します。
