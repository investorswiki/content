# カウンター通貨
##カウンター通貨とは何ですか？

カウンター通貨という用語は、[通貨ペアの参照通貨または2番目の通貨を指し](/currencypair)ます。カウンター通貨と基本通貨は、通貨または外国為替（外国為替）市場の一部です。トレーダーまたは投資家は、最初の通貨または[基本通貨](/basecurrency)の1単位を購入するために販売する必要のあるカウンター[通貨の量を決定でき](/currency)ます。通貨トレーダーがISO通貨コードを調べる場合、カウンター通貨はペアの基本通貨の後にリストされます。

##カウンター通貨の仕組み

通貨または[外国為替](/foreign-exchange)市場は、世界で最大かつ最も[流動性の高い市場の1つ](/liquidmarket)です。投資家はこの市場で毎日数兆ドル相当の通貨を取引しています。これは、証券取引所のような一元化された場所ではなく、銀行、ブローカー、トレーダー、および機関で構成される電子ネットワークで構成されています。

通貨は外国為替市場にペアでリストされています。この組み合わせは通貨ペアと呼ばれます。最初の通貨は基本通貨または取引通貨と呼ばれ、2番目の通貨はカウンター通貨または[見積もり通貨と呼ばれます](/quotecurrency)。外国為替市場では、トレーダーは、最初の通貨または基本通貨の1単位を購入するために必要なカウンター通貨の量を決定します。 [ISO通貨コード](/isocurrencycode)を使用して通貨ペアを検索する場合、カウンター通貨は基本通貨に続く通貨です。

トレーダーと投資家は、外国為替取引を理解するために、通貨ペアがどのように構成されているかを理解する必要があります。最初の通貨または基本通貨は、1ドルや1[ユーロなどの1つの通貨単位に相当し](/euro)ます。 EUR / USD通貨ペアで1ユーロを購入すると、特定の[米ドル数を販売することで1ユーロを受け取ることになります](/usd-united-states-dollar)。この例では、ユーロが基本通貨であり、ドルがカウンター通貨です。

>投資家が通貨ペアを購入または長期化する場合、彼らはカウンター通貨を売りますが、通貨ペアをショートさせる場合、彼らはカウンター通貨を購入します。

##特別な考慮事項

通貨ペア（基本通貨とカウンター通貨の両方）は、さまざまな要因の影響を受けます。これらのいくつかには、経済活動、[中央銀行によって制定された金融および財政政策](/centralbank)、および[金利が含まれます](/interestrate)。

ユーロや米ドルなどの主要通貨は、特にエキゾチックな通貨での取引に関しては、通貨ペアのカウンター通貨ではなく、基本通貨である可能性が高くなります。 2021年に市場で最も一般的に取引された通貨ペアは次のとおりです。

-EUR / GBP

-EUR / USD

-GBP / USD

-USD / CHF

-米ドル/円

上記のように、これらのペアの最初の通貨は基本通貨であり、2番目の通貨（スラッシュの後）はカウンター通貨です。 GBP / USDのペアリングでは、ポンドは基本通貨または購入される通貨であり、ドルはカウンター通貨です。これは販売されているものです。

##カウンター通貨の例

トレーダーが米ドルを使用して400ポンドを購入したいとします。これには、GBP/USD通貨ペアを使用した取引が含まれます。取引を実行するために、彼らは£400を得るために彼らが売る必要があるUSD（カウンター通貨）の数を把握する必要があります。

2021年6月3日の取引日の終わりのペアの為替レートは1.4103でした。これは、トレーダーが£1を購入するのに$1.4103かかることを意味します。その日に[取引](/transaction)を完了するために、トレーダーは、基本通貨の400単位、または400ポンド（400 x 1.4103）で564.12ドルを取得するために、カウンター通貨の564.12単位を販売する必要がありました。

##ハイライト

-トレーダーは、最初の通貨または基本通貨の単一ユニットを購入するために販売する必要のあるカウンター通貨のユニット数を決定します。

-販売する必要のあるカウンター通貨の単位数を決定するには、基本通貨の合計単位数に為替レートを掛けます。

-カウンター通貨は、ISO通貨コードペアの基本通貨に従います。

-カウンター通貨は、通貨ペアの2番目または参照通貨です。

-特にエキゾチックな通貨を取引する場合、主要通貨がペアの基本通貨になる可能性が高くなります。
