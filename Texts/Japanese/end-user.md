# エンドユーザー
##エンドユーザーとは何ですか？

「エンドユーザー」という用語は、商品またはサービスの消費者を指し、多くの場合、消費者に固有の固有のノウハウを持っています。

文字通りの意味で、エンドユーザーという用語は、商品またはサービスを購入して使用する人と、その設計、開発、および製造の段階に関与する人を区別するために使用されます。

##エンドユーザーを理解する

成功する製品またはサービスを作成するために、それを作成、開発、テスト、および販売する人々は、自分自身のニーズではなく、エンドユーザーのニーズを考える必要があります。エンドユーザーへの配信は、前述のすべてのプロセスが完了した後の最後のステップであり、目標は、エンドユーザーが以前は不可能だったことを達成できるようにすることです。

たとえば、外貨を取引するためのソフトウェアプラットフォームを設計するコンピュータープログラマーは、インターフェイスの高度化のレベルとエンドユーザーが実行する手順について考える必要があります。この顧客が取引にどのようにアプローチするか、トレーダーが何を見る必要があるか、どのようにデータと情報にアクセスし、取引がどのように実行されるか、そして取引後に何をしなければならないか。

>エンドユーザーは、必ずしもプロバイダーから製品またはサービスを購入したユーザーではありません。

>>

##特別な考慮事項

エンドユーザーへの言及は[テクノロジー](/technology_sector)業界では一般的であり、これらの顧客には基本レベルの技術的専門知識が期待されていることを示唆しています。 2018年3月1日、[サイバーセキュリティ](/cybersecurity)会社であるProofpoint、Inc.は、セキュリティ意識向上トレーニング会社の買収を発表しました。 ProofpointのCEOは次のように述べています。

>>

>サイバー犯罪者は従業員を搾取する新しい方法を探し続けているため、企業はエンドユーザーの行動を変え、サイバーセキュリティ教育ソリューションでリスクを軽減することに注意する必要があります。

>>

ヘルスケア分野では、この表現が頻繁に使用されます。 「エンドユーザーレベルでは、当社の製品は成長しています」と、歯科用品会社であるDentsply SironaのCEOは、2016年の四半期ごとのプレスリリースでコメントしました。製品の使用中。

エンドユーザーは、使用する製品やサービスがどのように開発されたか、またはどのように機能するかについて、多くの知識を持っていないことが多いことに注意してください。たとえば、Microsoft Windowsなどのコンピュータオペレーティングシステムの平均的なエンドユーザーは、製品に組み込まれたコード行、ユーザーエクスペリエンスの調査、またはデバッグを理解していません。エンドユーザーが気にするのは、使いやすさ、機能性、安定性だけです。つまり、企業は通常、専門家ではなく、そのような一般のエンドユーザーを念頭に置いて製品を開発する必要があります。

>あらゆるタイプの企業にとって、変化するテクノロジーとエンドユーザーの進化するニーズを綿密に追跡することが重要です。そうすることで、企業が満足し、企業とのビジネスを継続できるようにするための措置を講じることができます。これは、高品質の製品と優れたカスタマーサービスを提供することで最もよく達成できます。

>>

##エンドユーザーエクスペリエンス

エンドユーザーはほとんどの場合、専門家の知識や特定のスキルを持たない一般の個人であるため、企業はユーザーエクスペリエンスがわかりやすく、直感的で、効果的であることを確認する必要があります。ユーザーエクスペリエンス（UX）は独自の職業になり、UXチームはさまざまな業界の多くの企業に採用され、さまざまな製品に携わっています。

ユーザーエクスペリエンスでは、定量的および定性的な調査方法の両方を利用して、バグ、設計上の欠陥、一般的なユーザーエラーを特定して解決し、更新されたバージョンの新しいアイデアをクラウドソースします。エンドユーザーエクスペリエンスをナビゲートするのが難しい場合、特に不満のある顧客がすべての人に不満を表明できるソーシャルメディアやオンラインレビューの時代では、製品が成功する可能性は低くなります。

##エンドユーザーとお客様

すでに述べたように、「エンドユーザー」という用語はテクノロジーに関しては一般的であり、より専門的な用語である「[収益を生み出すユニット](/revenuegeneratingunit)」またはRGUを指す場合もあります。

対照的に、顧客とは、テクノロジーの購入トランザクションを実行する人であり、最終的なエンドユーザーである場合とそうでない場合があります。たとえば、ビジネスの場所では、上司がコンピュータを購入する人である可能性があるため、上司が顧客ですが、実際のエンドユーザーはジュニアレベルの従業員になります。

##エンドユーザーの例

ハイテク企業には、数十億とまではいかなくても数百万のエンドユーザーがいます。たとえば、Microsoftには、Windowsマシンを所有または操作している人、またはWord、Excel、PowerPointなどのMSOfficeツールを使用している人を含むエンドユーザーがいます。 Appleのエンドユーザーには、iPhoneでiOSを実行しているユーザー、またはMacを使用しているユーザーが含まれます。

企業または製品のエンドユーザーの数は、需要、競争、革新、または外部からの圧力に応じて変化する可能性があります。たとえば、COVID-19の大流行中に、世界中の人々がロックダウンに入り、自宅で仕事をしたり、オンラインで授業を受けたりすることを余儀なくされたため、MicrosoftTeamsやZoomなどのビデオ会議プラットフォームのエンドユーザーの数が急増しました。

##エンドユーザーのよくある質問

###エンドユーザーライセンス契約とは何ですか？

エンドユーザーライセンス契約（EULA）には、利用規約、免責事項、およびさまざまなソフトウェアパッケージまたはオンラインサービスのユーザーが続行する前に同意する必要があることの確認が含まれています。多くの場合、多くのページが長く、技術的な専門用語や法律用語で表現されていますが、研究者は、クリックする前に実際にEULAを読んで、条件に同意する人はほとんどいないことを発見しました。

###エンドユーザー認証とは何ですか？

エンドユーザー認証とは、エンドユーザーがアクセスしようとしている製品またはサービスの使用を許可されていることを確認するために使用されるいくつかの手法のいずれかを指します。これらには、アクティブなライセンスまたはアクティベーションコード、2要素認証（2FA）、生体認証データ（指紋スキャンなど）、またはそれらの任意の組み合わせが含まれる場合があります。

###エンドユーザーコンピューティングとは何ですか？

エンドユーザーコンピューティング（EUC）は、コンピュータープログラミング、開発、または設計機能をプログラマー以外の人に提供します。ビジュアルまたはオブジェクト指向のプログラミング言語により、行固有のコードやマシン言語の知識を必要としない開発環境が可能になります。ドラッグアンドドロップモジュールまたはウィジェットとテンプレートを使用するWeb構築プラットフォームは、HTML、javascript、またはCSSを知らない人のためのもう1つの例です。

###エンドユーザーサポートとは何ですか？

エンドユーザーサポートは、カスタマーサポートとテクニカルサポートの側面を組み合わせたものです。目標は、ユーザーがソフトウェアまたはプラットフォームをナビゲートし、エラーを修正し、バグレポートを取得できるようにすることです。

##ハイライト

-エンドユーザーのエクスペリエンスとサポートは、ユーザー指向の製品とサービスの成功に不可欠です。

-顧客としての「エンドユーザー」への言及は、テクノロジー業界で最も一般的です。

-エンドユーザーとは、企業が生産する商品またはサービスを消費または利用する個人またはその他のエンティティです。

-このように、エンドユーザーは顧客とは異なる場合があります。製品またはサービスを購入するエンティティまたは個人が実際にそれを使用するものではない可能性があるためです。

-エンドユーザーへの納品は、多くの場合、製品の製造と販売の最終ステップです。
