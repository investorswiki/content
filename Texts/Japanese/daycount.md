# 日数コンベンション
##日数コンベンションとは何ですか？

日数の慣例は、債券やスワップなどの負債証券で使用されるシステムであり、次のクーポン支払いがクーポン期間全体よりも短い場合に、未収利息の金額または現在の値を計算します。

##日数の表記法を理解する

日数の規則は、債券だけでなく、スワップ、住宅ローン、金利先渡契約にも適用されます。日数の規則を適用するための規則と定義の多くは、幅広い金融取引の文書を提供する国際スワップディーラー協会によって定められています。

たとえば、合意された日数の規則を使用して、次のクーポンの支払いがクーポン期間全体よりも短い場合に、[未収利息の金額](/accruedinterest)または[現在の値（PV）を計算します。](/presentvalue)

最も一般的な規則には、30 / 360、30 / 365、actual / 360、actual / 365、actual/actualがあります。

--30 / 360-360日の年を使用して毎日の関心を計算し、それを30（標準化された月）で乗算します。

-30/365-365日の年を使用して毎日の関心を計算し、それを30（標準化された月）で乗算します。

--actual / 360-360日の年を使用して毎日の関心を計算し、それを各期間の実際の日数で乗算します。

--actual / 365-365日の年を使用して毎日の関心を計算し、それを各期間の実際の日数で乗算します。

--actual / actual-年間の実際の日数を使用して毎日の関心を計算し、それを各期間の実際の日数で乗算します。

各[債券市場](/bondmarket)と金融商品には独自の日数の慣習があり、それは商品の種類、金利が固定か変動か、発行国によって異なります。米国財務省が発行した[債券](/bond)および債券は、実際/実際に基づいて計算された利息を獲得します。これは、期間内のすべての日が等しい値を持つことを意味します。また、クーポン期間の長さと結果として生じる支払いが異なることも意味します。

ほとんどのマネーマーケット預金と[変動金利債の利息](/frn)は、実際/360日ベースで計算されます。主な例外は英国ポンド建てのもので、利息は実際/365ベースで計算されます。オーストラリア、ニュージーランド、香港のドルなど、英国のポンドに密接に関連している、または関連している通貨も、365日を使用します。

[スワップ](/interestrateswap)の固定レートレッグとほとんどの固定レートボンドは、30/360日の規則または30/365のいずれかを使用します[。](/interestrateswap)この規則では、月は常に30日として扱われ、年は常に360日または365日として扱われると規定されています。スワップの固定レートに30/360規則を使用するスワップ市場には、[米ドル](/usd-united-states-dollar)、[ユーロ](/euro)、[スイスフランが含まれ](/chf)ます。英国ポンドと[日本円のスワップは](/jpy-japanese-yen)通常、30/365規則を使用します。オーストラリア、ニュージーランド、香港は再びイギリスに続きます。

ほとんどの金利スワップの変動レートレッグは、360日または365日のいずれかに対して、実際の日数の変動を使用します。米ドル市場を含む固定レートレッグに30/360を使用する市場は、変動レートレッグにactual/360を使用します。固定相場制で30/365を使用するものは、変動相場制で実際/365を使用します。

[London InterBank Offering Rate](/libor) （ LIBOR）は、最も一般的に使用されるベンチマーク金利であり、ロンドン時間の午前11時45分に毎日掲載されます。

>LIBORの責任者であるIntercontinentalExchangeは、2021年12月31日以降、1週間および2か月のUSD LIBORの公開を停止します。他のすべてのLIBORは、2023年6月30日以降に廃止されます。

>>

ほとんどの通貨では、LIBORでの利息は実際/360日ベースで計算されます。主な例外は、実際の/365日ベースで計算される英国ポンドです。

##ハイライト

-日数の規則は、2つの日付の間の日数を計算するための標準化された方法です。

-ほとんどのマネーマーケット預金および変動金利手形の利息は、実際/ 360日カウントの規則に基づいて計算されますが、米国財務省によって発行された債券および手形は、実際/実際に基づいて計算された利息を獲得します。

-金利スワップの固定レートレッグとほとんどの固定レート債券は、30/360または30/365日カウントの規則を使用しますが、変動レートレッグは実際の/360または365日カウントのバリエーションを使用します大会。
