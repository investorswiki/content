# チャネルスタッフィング
##チャンネルスタッフィングとは何ですか？

が一般に販売できるよりも多くの製品を[流通チャネル](/distribution-channel)に沿って意図的に送ることにより、売上高と収益の数値を膨らませるために使用する欺瞞的なビジネス慣行です。チャネルの詰め込みは通常、四半期末または年末の直前に行われるため、経営陣は報酬への悪影響を恐れて「数を増やす」ことができます。

##チャネルスタッフィングの仕組み

チャネルスタッフィングとは、エンドユーザーが妥当な期間内に購入する可能性が高いよりも多くの商品を流通チャネルに沿って流通業者や小売業者に出荷する企業の慣行を指します。これは通常、大幅な割引、リベート、延長された支払い条件などの有利なインセンティブを提供して、現在のニーズを超える数量を購入するように販売業者や小売業者を説得することによって達成されます。

通常、販売業者は、最終的な販売が実際に行われたかどうかを疑問視する未販売の在庫を返品する権利を保持します。流通チャネルの「スタッフィング」は、短期的な収益と収益の目標を達成するために収益認識を加速するために企業が使用する慣行として、[証券取引委員会（SEC）](/sec)によって嫌われており、投資家を誤解させます。

チャネルスタッフィングにより、ディストリビューターは特定の期間の売上高と関連する利益測定値を一時的に増やします。このアクティビティは、[売掛金](/receivables)の人為的な増加も引き起こします。ただし、余剰商品を販売できない場合、小売業者は現金ではなく余剰商品を販売業者に返送します。販売業者は、売掛金（GAAP手順に準拠している場合）と最終的には収益を再調整する必要があります。

言い換えれば、スタッフィングは常に会社に追いつきます。なぜなら、スタッフィングの速度で売上を維持することができないからです。チャネルスタッフィングは、卸売業と小売業に限定されていません。それは、産業部門、ハイテク産業、および製薬産業でも発生する可能性があります。 Valeant Pharmaceuticalsは、2016年にチャネルスタッフィングで有罪判決を受けた企業のひどい例です。

チャンネルスタッフィングの告発は自動車業界にも課せられており、自動車業界は売上高を膨らませるために需要の令状よりも多くの新車をディーラーに送っています。

>この不正行為は通常、報酬目標を達成するため、または株式の価値を上げるため、または四半期または年次の結果が発表されたときに株価が下落するのを防ぐために行われます。

>>

##チャンネルスタッフィングの例

2004年8月、製薬会社のブリストルマイヤーズスクイブ（NYSE：BMY）は、SECによるチャネルスタッフィング訴訟を解決するために1億5,000万ドルを支払うことに合意しました。

裁判所の文書は次のことを明らかにしています。

>>

> 2年間、ブリストル-マイヤーズは市場をだまして、財務予測と市場の期待に応えていると信じ込ませました。実際、同社は主にチャネルスタッフィングと操作的な会計装置を通じてその数を増やしていました。ブリストルマイヤーズにその違反行為の責任を負わせ、ブリストルマイヤーズや他の上場企業が同様の計画に従事することを思いとどまらせるには、厳しい制裁が必要です。

>>

>>

> Bristol-Myersは、主に、需要に先んじて卸売業者に医薬品を販売することにより、目標を達成するのに十分な量の過剰在庫を四半期末近くに流通チャネルに詰め込むことにより、業績を膨らませました。ブリストル・マイヤーズは、チャネルの詰め込みの結果として、メディケイドと特定の主要ベンダー、それらの卸売業者から大量の医薬品を購入した卸売業者の顧客によるリベートの見越額を大幅に過小評価しました。

>>

ブリストル・マイヤーズは、数百万ドルの罰金を支払うことに加えて、003年3月に、過去の財務諸表を修正し、チャネルスタッフィング活動と不適切な会計を開示しました。

##ハイライト

-チャネルスタッフィングとは、エンドユーザーが妥当な期間内に購入する可能性が高いよりも多くの商品を流通チャネルに沿って流通業者や小売業者に出荷する企業の慣行を指します。

-チャネルスタッフィングにより、ディストリビューターは特定の期間の売上高と関連する利益測定値を一時的に増やします。

-規制当局はその慣行に憤慨し、それを欺くと考えています。場合によっては、問題のある会社に法的措置をとることができます。
