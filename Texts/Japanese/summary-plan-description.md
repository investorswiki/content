# 要約計画の説明
##サマリープランの説明とは何ですか？

要約プランの説明（SPD）は、雇用主が[従業員退職所得保障法](/erisa)の対象となる退職プランまたは健康保険プランに参加する従業員に無料で提供しなければならない文書です。 SPDは、プログラムが提供するメリットと計画の仕組みに関する詳細なガイドです。従業員がプランに参加する資格を得る時期、福利厚生の計算方法と支払い方法、福利厚生の請求方法、および福利厚生が[確定する時期について説明する必要があり](/vesting)ます。

SPDは、従業員が理解できるわかりやすい言葉である必要があります。プラン名とその[内国歳入庁](/irs)が割り当てた番号、雇用主の名前と住所、プラン管理者の名前と連絡先情報、[医療保険の相互運用性と説明責任に関する法律](/hipaa)の権利に関する声明、ERISAの開示、および従業員が提出できる方法に関するガイダンスを含める必要があります。不満や訴え。

##サマリープランの説明を理解する

あなたが最初に雇用されたとき、あなたは90日以内にあなたの新しい雇用主のヘルスケアと退職給付をカバーするSPDを受け取るべきです。職場でコンピュータを定期的に使用している場合、またはハードコピーとして使用している場合、会社はドキュメントを電子的に配布する場合があります。電子コピーのみを受け取る場合は、書面によるコピーを要求できます。

##要約計画の説明はあなたの質問に答えるべきです

計画には、次の質問に対する回答を含める必要があります。

-プランに参加するための最低年齢要件はありますか？

-プランに参加するための最低サービス要件はありますか？もしそうなら、それは何ですか、そしてそれはどのように計算されますか？

-計画年度はいつ始まり、いつ終わりますか？ 1月1日から12月31日まで実行されますか、それとも開始日と終了日が異なりますか？アカウント所有者は、毎年の[控除](/deductible)対象がいつリセットされるかを知る必要があるため、この情報は健康保険にとって重要です。

-私はプランに寄付をしますか、それともすべての寄付は私の雇用主からのものですか？

-リタイアメントプランの場合、プランは他のプランからの[ロールオーバー拠出を許可](/rollover)しますか？たとえば、以前の雇用主からの[401（k）をこのプランにロールオーバーできますか？](/401kplan)

-退職金制度の場合、雇用主と従業員の拠出金はどのように投資されますか？特定の投資オプションを選択しない場合にお金が行くデフォルトの投資はありますか？もしそうなら、彼らは何ですか？投資の選択を変更するにはどうすればよいですか？

-私はいつ退職プランに権利が確定しますか？私はすぐに[100％権利](/fully-vested)が確定しますか、それとも部分的または完全に権利が確定するために一定の年数の間会社で働く必要がありますか？

-退職金口座から借りることはできますか？もしそうなら、ルールは何ですか？

-障害者になった場合、給付はどうなりますか？会社を辞めたら？引退したら？私が死んだら？休職したら？

##要約プランの説明は、訴訟に対する雇用者の保護を提供します

ビジネス面では、雇用主はSPDが必要なすべてをカバーしていることを確認する必要があります。 SPDがない、またはSPDが不十分な場合、雇用主は従業員からの訴訟にさらされます。 ERISAガイドラインに従い、除外と制限を明確に示したSPDは、起こりうる法的措置から会社を保護します。

SPDは、さまざまな福利厚生を受ける資格のある従業員を定義するものと、そのような従業員に[独立請負業者](/independent-contractor)、一時労働者、配偶者、同棲相手、および子供が含まれるかどうかを記載する必要があります。

##特別な考慮事項

さらに、従業員の10％以上が英語以外の言語を話す場合は、それらの他の言語でもSPDを公開する必要があります。 ERISA法を理解している弁護士を雇って、配布する前にSPDを確認すると、文書が完全、徹底的、正確であり、州法および連邦法に準拠していることを確認できます。 SPDを最初から最後まで作成するのに数か月かかる場合があります。従業員がそれを簡単に理解できるようにすることで、混乱している従業員からの苦情、訴訟、および人材への質問を減らすことができます。

雇用主は、提供する特典を定期的に変更する場合があります。これが発生した場合、雇用主は、改訂されたSPDまたはSPDの変更を説明する重要な変更の要約を従業員に書面で通知する必要があります。会社は、適用範囲または利益が減少する場合、変更が有効になってから60日以内に通知を配布する必要があります。変更によって補償範囲や利益が減少しない場合は、変更が有効になった計画年の終了後210日以内に通知を配布する必要があります。

##ハイライト

-プランは、プラン名、プランのIRS割り当て番号、雇用主の名前と住所、健康と説明責任の権利の声明などの特定の質問に答える必要があります。

-計画では、プログラムのメリットと計画の仕組みについて説明します。

-従業員は、雇用主から要約計画の説明（SPD）を受け取る必要があります。
