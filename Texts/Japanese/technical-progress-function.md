# 技術進歩機能（TPF）
##技術進歩機能とは何ですか？

技術進歩機能（TPF）は、マクロ経済成長モデルの一部であり、社会が生み出すことができ、実際に生み出す経済的生産の総量に対する技術と技術の進歩の影響、および生産要因の生産性を説明します。社会は雇用しています。マクロ経済成長の以前のモデルは、天然資源の寄付、労働力の増加、または経済成長と発展を説明するための資本財と設備の蓄積などの要因に焦点を当てていました。 20世紀には、これらのさまざまな生産要素をより効率的に組み合わせて生産性を向上させるための技術的改善の役割が、経済成長の鍵として経済学者の間で広く認識されるようになりました。いくつかの異なるエコノミストによるマクロ経済成長のモデルへのTPFの組み込みは、この認識を正式な経済モデルにもたらしました。

特定のマクロ経済モデルのTPFは、技術の進歩と生産量の増加との関係を数学的に指定します。 TPFの特定の形式と構造は、マクロ経済モデルごとに異なる可能性がありますが、一般に、技術進歩の速度の増加が、全体的な生産性を促進する上で最も重要な要素、または最も重要な要素の1つであることを示しています。経済成長。技術の進歩は、生産方程式の入力側でより優れた技術を使用することで国がより多くの生産を行うのに役立つため、国の経済成長の重要な要素になる可能性があります。

これらのマクロ経済モデルに基づいて、経済的手法を使用して、回帰モデルを使用することにより、技術進歩が総経済生産量に与える影響を経験的に推定することができます。したがって、[経済生産の成長を純粋に投入配分効率の観点から](/economicgrowth)見るのではなく、技術進歩機能は、最終生産全体への貢献者としての技術進歩を測定する方法を提供します。

##TPFを理解する

TPFは、総生産量と、さまざまな変数が総生産量にどのように影響するかを理解するために使用される多因子回帰モデルのコンポーネントです。基本的な生産回帰では、出力は基本的な変数が生産に割り当てられる効率のレベルによって説明されます。たとえば、労働と機械は生産に影響を与える2つの基本的な変数です。

より詳細な分析により、経済統計学者は技術の進歩を2つの要素に分解しようとするかもしれません。通常、2つの主要な要素は次のとおりです。

-**具体化された技術進歩**：新しい機器への投資に起因する改善された技術。行われた新しい技術的変更は、機器に具体化されます。

-**非実体化された技術進歩**：新しい機器に投資することなく生産量の増加をもたらす改善された技術。

技術進歩関数は、生産回帰分析に追加された変数です。基本的に、それは方程式の追加機能であり、他の基本的な入力では説明されない、生産への技術的貢献に関する洞察を提供します。一般に、技術の進歩が進むにつれて、生産量の増加は生産方程式内の技術の進歩に起因し、他の変数には起因しません。

## Solow Residual

[ロバート・ソロー](/blackscholes)は、ソロー残差および全要素生産性（TFP）としても知られる、技術進歩機能の概念に関する研究でノーベル賞を受賞しました。 Solowは、生産性に影響を与えるさまざまな機能を詳細に説明したモデルを使用して、生産性を理解するために使用される成長モデルをレイアウトしました。 Solowのモデルには、資本、労働、および技術進歩の機能が含まれています。その後の研究者は、人的資本などの追加の変数を含めるためにSolowのモデルを変更しました。

Solowのモデルでは、TPFは、技術の進歩が総生産量にどの程度影響を与えているかを示しています。

米国で1909年から49年までのモデルを使用したとき、Solowは、米国での労働生産性の増加の8分の1だけが資本の増加に起因する可能性があることを発見しました。残りは、労働力と資本の使用方法の技術的進歩の結果でした。言い換えれば、アメリカはアメリカのノウハウと革新のおかげで素晴らしいものになりました。

全要素生産性は、さまざまな影響によって影響を受ける可能性があります。すべてが技術進歩の傘下にありますが、影響には、技術、文化的要因、および新しい経済効率が含まれる可能性があります。そのため、技術進歩機能とTFPを使用して、各国の技術的影響と技術進歩の違いを分析することもできます。

##ハイライト

-TPFは、ある国の技術進歩の革新に起因する経済成長の程度を測定します。

-TPFは、さまざまな要因が総生産にどのように影響するかを研究するマクロ経済モデルのコンポーネントです。

-技術の進歩は、新しい機器に具体化されているか、機器に関係のない新しいイノベーションによる生産性の向上に具体化されていないかのいずれかとして現れる可能性があります。
