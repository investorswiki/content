# 取得した資金手数料および費用（AFFE）
##取得した資金手数料（AFFE）とは何ですか？

取得したファンドの手数料と費用（AFFE）は、マルチマネージャーまたは[ファンドオブファンズ](/fundsoffunds)（FOF）の目論見書の項目であり、基になるファンドの営業費用を示します。これは2007年1月の時点で要件になりました。この項目は現在、「手数料と経費」の見出しと目論見書の下でファンドの手数料スケジュールに含まれています。

##取得した資金の手数料と費用を理解する

取得したファンドの手数料および費用は、より複雑な手数料体系を持つマルチマネージャーおよびファンド・オブ・ファンズのオプションに関連付けられています。これらの手数料は、ファンドの[年間総費用](/tafoe)を増加させ、複数のマネージャーに支払われる管理手数料を含みます。

ファンドオブファンド（FOF）は、ミューチュアルファンドやヘッジファンドなど、独自の投資を選択しないプールされた投資ファンドです。代わりに、これらのFOFは他の投資信託またはヘッジファンドに投資します。言い換えれば、そのポートフォリオには、独自のポートフォリオマネージャーによって管理されている他のファンドのさまざまな基礎となるポートフォリオが含まれています。これらの保有は、債券、株式、その他の種類の証券などの資産への直接投資に取って代わります。ファンド・オブ・ファンズ（FOF）戦略は、すべてが1つのポートフォリオにまとめられたさまざまな[ファンド・カテゴリー](/fundcategory)への投資により、幅広い[分散](/diversification)と適切な[資産配分を実現することを目的としています。](/assetallocation)

FOFを購入する投資家は、2つのレベルの手数料を支払う必要があります。個人ファンドと同様に、FOFは[管理手数料](/managementfee)と成功[報酬を](/performance-fee)請求する場合がありますが、管理のほとんどがサブファンド自体に委任されているという事実を反映して、成功報酬は通常、個人投資信託よりも低くなります。

##SECの規制と開示

2007年1月、証券取引委員会（SEC）は[、1940年の投資会社法に新しい規定を制定し始めまし](/investmentcompanyact)た。これにより、ファンド会社はファンドオブファンズオプションを簡単に登録できるようになりました。 SECは、マルチマネージャーファンドに関する1940年法のセクション12（d）（1）に基づく法律を拡大しました。 SECはまた、登録届出書を改訂して、これらの基金の費用に関する追加の詳細を含めました。具体的には、登録届出書では、目論見書に記載されている包括的な手数料スケジュールに含める必要があるマルチマネージャーの追加手数料開示要件として、ファンドマネージャーに「取得したファンド手数料および経費」を含めることが求められています。

2007年以前は、ファンドオブファンズへの投資は、 [SECによって承認された特定の状況下でのみ許可されていました](/sec)。ほとんどの場合、これらのファンドオブファンズ投資はゼロの[費用比率を報告します。](/expenseratio)開示は誤解を招くものであり、費用がないことを示し、ポートフォリオのさまざまな原資産によって発生する営業費用があると報告しました。

新しいAFFE要件により、株主が負担する関係と費用の組み合わせをより透明性の高い方法で開示できるようになりました。 AFFE項目は、ファンドの料金表に追加され、ファンドの他の標準経費に追加されます。 AFFEは、投資顧問が[マルチマネージャーに支払うことに同意した個別の手数料で構成される包括的な手数料として設定され](/multiple-managers)ます。 AFFEは、個々のマネージャーとの合意に応じて、0.02％から10％の範囲になります。

##例：Neuberger Berman Absolute Return Multi-Manager Fund

Neuberger Berman Absolute Return Multi-Manager Fundは、マルチマネージャーファンドに見られる手数料体系の一例を示しています。ファンドは、クラスA、クラスC、および機関投資家の株式を提供するオープンエンド型ミューチュアルファンドです。

ファンドには標準的な手数料が適用され、[管理手数料](/managementfee)は株式クラス全体で1.92％から1.81％の範囲です。クラスAおよびクラスCの株式には、それぞれ0.25％および1.00％の分配手数料が請求され、機関投資家の株式には分配手数料はかかりません。その他の総営業費用は1.04％から1.02％の範囲です。取得したファンドの手数料と費用は、ファンドの最後の手数料費用の項目を締めくくり、すべての株式クラスが0.05％の手数料を支払います。免除を伴う年間総費用は3.94％から2.83％の範囲です。

##ハイライト

-取得したファンド・オブ・ファンズと費用（AFFE）により、ファンド・オブ・ファンズ（FOF）の投資家は、FOFが投資するポートフォリオファンドに管理手数料としていくら支払っているのかを理解できます。

-通常のAFFEは、FOFが保有するファンドの種類とそれに関連する手数料に応じて、最大10％の範囲になります。

-AFFEは、ファンドの手数料スケジュールの必須項目として表示され、マルチマネージャー投資に伴うより複雑で階層化された手数料構造を認めています。
