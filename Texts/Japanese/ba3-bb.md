# Ba3 / BB-
## Ba3 / BB-とは何ですか？

Ba3 / BB-は、一般的に投機的と見なされる債券に与えられる債券レートです。 Ba3は、[ムーディーズの信用格付けサービス](/moodys)によって提供される長期債券格付けであり、BB-は、[スタンダード＆プアーズ](/sp)と[フィッチの両方の](/fitch-ratings)格付けサービスによって提供される並行格付けです。

## Ba3/BB-説明

債券に与えられた信用格付けは、証券のリスクと発行者が債務不履行に陥る可能性の尺度を提供します[。](/creditrating)債券の信用格付けは、社債または国債の信用力を表しています。元本投資を失うリスクを回避するために安全な債券投資を探しているリスク嫌悪の投資家は、国債またはAAAからBaa3/BBB-の格付けの投資適格社債を選ぶことができます。

投資適格債よりもリスクが高い債券は、[ジャンク債と呼ばれ](/junkbond)ます。投資家は、高水準のリスクを引き受けることに対する補償として、これらの債券を購入するためのより高い利回りを要求します。したがって、これらの債券は[高利回り債](/high_yield_bond)とも呼ばれます。[ジャンク債格付けスペクトル](/junkbond)のより安定した終わりに向かっている非投資適格債は、通常、信用格付け機関であるムーディーズ、スタンダード＆プアーズ、およびフィッチ格付けによってBa3/BB-の信用格付けが与えられます。これはハイイールド債のカテゴリー内で最高の格付けティアですが、Ba3 / BB-格付けは、悪化する経済状況や企業固有の動向が発行者の義務を果たす能力を妨げる可能性があるという懸念のレベルが高いことを示しています。 Ba2 / BBは、Ba3 / BB-の真上にある評価であり、B1 /B+は真下にあります。

##ジャンク債の格付け

Ba3 / BB-の信用格付けは、債券がある程度[投機的](/speculation)であり、リスクにさらされていることを示しています。 Ba3 / BB-と格付けされた債券は、特に米国政府、地方自治体、および最大のグローバル企業によって発行された格付けの高い債券をはるかに上回る[満期利回り](/yieldtomaturity)（YTM）または[コール利回りを提供します。](/yieldtocall)ただし、投資家は、この高いレートが、財政的に健全でなく、投資の損失につながる可能性のある企業または政府への投資に対する補償として機能することを認識することが重要です。

Ba3 / BB格付けは通常、発行体のバランスシートの強さ、[債務返済能力](/debtservice)、現在のビジネスおよび経済状況、発行会社の成長見通しなど、発行体との関係で特定の要因を分析した後に決定されます。 。企業が投資適格債の品質として評価され、一定期間後に企業の統計がレビューされた後、非投資適格債の品質に格下げされる可能性があります。同様に、ビジネスの見通しと財務諸表が力強い成長とリスクの低下を反映している場合、Ba3/BB-格付けの企業は投資適格にアップグレードされる可能性があります。

##ハイライト

-Ba3はB1レーティングを上回り、Ba2を下回りますが、BBはB+を上回りBBを下回ります。

--Ba3 / BB-は、ムーディーズ、S＆P、およびフィッチが発行済みの債務証券（通常は債券）または投資適格を下回るクレジットの発行者（つまり会社または企業）（つまり「ジャンク債」）に対して使用する信用格付けです。 ）。

-ムーディーズはBa3格付けを使用し、S＆PとフィッチはBB-を使用しています。
