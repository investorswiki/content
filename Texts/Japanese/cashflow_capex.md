# 資本的支出へのキャッシュフロー（CFからCAPEX）
##設備投資へのキャッシュフロー（CFからCAPEX）とは何ですか？

[資本的支出に対する](/capitalexpenditure)キャッシュフロー（CF / CapEX）は、フリーキャッシュフローを使用して[長期資産を取得する企業の能力を測定する比率です。](/longtermassets) CF / CapEX比率は、企業が大小の設備投資のサイクルを経るにつれて変動することがよくあります。 CF / CapEX比率が高いということは、新しい資本的支出への投資に資金を提供するのに十分な資本を持っている企業を示しています。

##資本的支出へのキャッシュフロー（CF / CAPEX）を理解する

アナリストは、実際のデータを使用して、会社に関する手がかりと洞察を見つけようとします。彼らは、市場が利益のために売買されるのを待っている潜在的に[過小評価](/undervalued)または[過大評価された証券でいっぱいであると信じています。](/overvalued)[ファンダメンタル分析](/fundamentalanalysis)の主要なツールは比率です。キャッシュフローと資本的支出（CF / CapEX）の比率は、他の比率と同様に、会社の業績に関する情報を提供します。具体的には、この比率は、会社が有形固定資産（PP＆E）などの資本的支出に投資した1ドルあたりの事業からどれだけの現金を生み出しているかをアナリストに示します。これは、[成長株](/growthstock)を探しているアナリストにとって非常に重要です。

## CF/CapEXの計算

CFからCAPEXは、次のように計算されます。

資本的支出へのキャッシュフロー=事業からのキャッシュフロー/資本的支出

[、営業活動によるキャッシュフローを](/cash-flow-from-operating-activities)設備投資で割って算出しています。これらの項目は両方とも、[キャッシュフロー計算書に記載されてい](/cashflowstatement)ます。設備投資は、将来の投資とみなされるため、投資によるキャッシュフローの項目です。

たとえば、ある会社の営業活動によるキャッシュフローが10,000ドルで、資本的支出に5,000ドルを費やしているとします。その場合、それは、事業から得られるすべてのドルの半分が設備投資に向けられていることを意味します。会社が設備投資に1,000ドルを費やすと、比率は10対1に減少します。つまり、事業から得られる1ドルの10％だけが設備投資に使われます。営業活動によるキャッシュフローがマイナスの場合、資本的支出は外部資金によって賄われています。

##資本的支出に対するキャッシュフローの解釈（CF / CapEX）

一般的に、CF / CapEX比率が高いことは良い指標であり、比率が低いことは成長の指標です。車を考えてみましょう。他のすべての条件が同じであれば、ガソリンで満たされた車は空の車よりも優れています。同様に、あなたのクレジットカードよりもあなたのポケットの現金からガスを支払う方が良いです。最良のシナリオは、最近ガスで満たされた車で、運転手のポケットに現金で支払われます。これは、CF/CapEX比率が高い企業に似ています。多くのアナリストは、資本的支出を利益成長の推進力と見なしているため、資本的支出への投資が少ない企業は、設備投資を埋めたばかりの企業までは行かない可能性があります。

##ハイライト

-一般的に、CF / CapEX比率が高いことは、新しい資本的支出への投資に資金を提供するのに十分な資本を持っている企業を反映しています。

-CF / CapEX比率は、企業が大小の資本的支出のサイクルを経るにつれて、時間とともに変化します。

-資本的支出へのキャッシュフロー（CF / CapEX）は、フリーキャッシュフローを使用して長期資産を購入する企業の能力を調べます。
