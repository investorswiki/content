# 総商品価値（GMV）
##総商品価値（GMV）とは何ですか？

[顧客間](/ctoc)（C2C）交換サイトを通じて特定の期間に販売された商品の合計値です。これは、ビジネスの成長または他者が所有する商品を販売するためのサイトの使用の尺度です。

総商品価値（GMV）は、 [eコマース](/ecommerce)サイトのビジネスの健全性を判断するためによく使用されます。これは、その収益が、販売された総商品と請求された料金の関数になるためです。これは、現在の四半期の値と前の四半期の値など、時間の経過に伴う比較尺度として最も役立ちます。

GMVは総商品量としても知られています。両方のフレーズは、総売上高の総金銭的価値を示します。

##総商品価値（GMV）を理解する

総商品価値（GMV）は、手数料または費用を差し引く前に計算されます。これは、小売企業が成長を測定するために使用できる情報を提供します。多くの場合、月ごとまたは年ごとに行われます。一般に、小売業は完了したすべての売上の総額を計算できますが、正確な計算を行うには、この数値から商品の返品を削除する必要がある場合があります。

>発生した料金と費用には、広告、配送、返品、割引が含まれる場合があります。

>>

GMVを計算するには、販売された商品の数に商品の販売価格を掛けるだけです。

##総商品価値（GMV）の長所と短所

###利点

[小売業者は販売する商品の生産者である場合とそうでない場合がある](/retail-sales)ため、すべての売上の総額を測定することで、会社の業績に関する洞察が得られます。これは、小売業者が実際にはどちらにも参加せずに買い手と売り手を接続するためのサードパーティメカニズムとして機能する顧客間市場で特に当てはまります。

[委託部門の小売業者は、正式に](/consignment)[在庫](/inventory)を購入することはないため、価値を提供する可能性があります。アイテムは多くの場合、会社の小売店内に保管されていますが、ビジネスは、他の人またはエンティティの商品または資産の認定再販業者として、多くの場合有料で機能します。一般的に、彼らはアイテムの真の所有者になることはありません。アイテムを委託した個人または団体は、そのように選択した場合、アイテムを返品して請求することができるからです。

###デメリット

GMVは、C2C取引所で販売された商品の合計値を表しますが、企業の収益性を正確に反映しているわけではありません。主に、企業が手数料から得る真の[収益。](/revenue)たとえば、ある会社のGMVがその月に500ドルだった場合、その500ドル全体が会社に支払われることはありません。その大部分は商品を販売した個人に送られます。会社の真の収入は、サイトの使用に対して請求する料金になります。料金が2％の場合、会社の実際の収益は$ 500 x 2％=$10になります。

eコマースサイトの種類によっては、GMVには他の欠点もあります。たとえば、企業が自社の商品を製造および販売するオンライン小売業者である場合、GMVは企業の収益を示しますが、それは多くの場合制限される可能性のある1つのメトリックにすぎません。来店したお客様の数やリピーターからの収益はわかりません。これは、[お客様の満足度](/customer-service)、ひいては会社の長期的な健全性の観点から重要な指標です。

##消費者間小売業者

Customer-to-Customer（C2C）小売業者は、売り手が在庫にあるアイテムをリストし、買い手が関心のあるアイテムを見つけるためのフレームワークまたはシステムを提供します。小売業者は仲介者として機能し、取引のどの時点でも実際に買い手または売り手になることなく、通常は有料で取引を促進します。

>「商品」という言葉は、「マーチャンド」または商人からの古いフランス語の「商品」に由来します。

>>

これらの顧客間の販売の多くでは、取引を促進する小売業者が実際の商品と接触することはありません。代わりに、販売の財務部分が完了すると、販売者はアイテムを購入者に直接送信します。

このモデルは、小売業者が生産者、製造業者、または流通業者から商品を購入し、基本的に会社が購入した商品の正規再販業者として機能する他の小売モデルとは大幅に異なる場合があります。

##総商品価値（GMV）の例

最もよく知られているC2Cサイトの2つは、eBayとEtsyです。たとえば、会計年度の第1四半期に、eBayは100個の商品を販売しました。簡単にするために、これらの商品はすべて5ドルで販売されていました。第1四半期の場合、eBayのGMVは100 X $ 5 =$500になります。

たとえば、同じ四半期にEtsyが80の商品を販売し、簡単にするために、すべての商品の価格が4ドルだったとします。第1四半期の場合、EtsyのGMVは80 x $ 4 =$320になります。

この例では、eBay（EBAY）のGMVはEtsy（ETSY）の320ドルよりも500ドルの方が優れています。ただし、これですべてがわかるわけではありません。これらのサイトでは、収益の一部は商品を販売した売り手に還元されなければなりません。 eBayとEtsyは、実際の収益である請求する料金のみを保持します。

この例では、eBayは2％の料金を請求するため、$ 10（$ 500 x 2％）になります。一方、Etsyは、より高い料金を請求します。この例では4％です。 Etsyは$12.80（$ 320 x 4％）をもたらします。この例では、Etsyは持ち帰り収入が多いため、実際にパフォーマンスが向上しました。

## GMVFAQ

### GMVとはどういう意味ですか？

GMVとは、総商品価値または総商品量を意味し、通常、顧客間（C2C）交換サイトを通じて特定の期間に販売された商品の合計値を指します。

### GMVは収益と同じですか？

eコマースサイトの種類にもよりますが、GMVは総収入と同じです。ただし、eBayのようなサイトの場合、これは販売された商品の合計金額を反映していますが、実際の収益の一部は商品の販売者向けであるため、実際の収益ではありません。 eBayが実際に稼ぐ収益は、eBayが販売に請求する料金から得られます。

###スタートアップのGMVとは何ですか？

スタートアップでは、GMVは総商品収益、つまり企業が商品やサービスの販売を通じて生み出す総収益です。 GMVは、控除を考慮した純売上高と併せて測定することが重要です。

### GMVはどのように計算されますか？

GMVは、特定の期間に販売された商品の合計金額に販売価格を掛けて計算されます。 GMV=商品の販売価格x販売された商品の数。

##結論

総商品価値（GMV）は、顧客間（C2C）交換サイトによって販売された商品の合計値ですが、このメトリックは他のタイプの小売業者に適用されることがよくあります。 GMVは、販売された商品の合計値を報告するため、計算するのに便利なメトリックですが、他のメトリック、特に手数料を通じて収益を生み出す企業の場合は、考慮する必要があります。

##ハイライト

-ある期間から別の期間にGMVを分析することで、経営陣とアナリストは会社の財務状態を判断できます。

-GMVは、収益の一部が元の売り手に支払われるため、企業の収益を正確に表すものではありません。

-商品総額は、手数料または費用を差し引く前に計算されます。

-総商品価値（GMV）は、顧客間またはeコマースプラットフォームを介して販売された商品の量を指します。

-これは、事業の成長または委託を通じて他者が所有する製品を再販するためのサイトの使用の尺度です。
