# USD / JPY（米ドル/日本円）
## USD / JPY（米ドル/日本円）とは何ですか？

USD / JPYは、米ドルと日本円の為替レートを表すために使用される略語です。[通貨ペアは](/currencypair)、 1米ドル（[基本通貨）を購入するために必要な](/basecurrency)日本円（[見積もり通貨）の数を示します](/quotecurrency)。[日本円（JPY）](/jpy-japanese-yen)の記号は¥です。

## USD / JPY（米ドル/日本円）ペアを理解する

USD / JPYペアの価値は、1米ドルあたりの日本円で表示されます。たとえば、ペアが150で取引されている場合、1米ドルを150円に交換できることを意味します。世界第3位の国民経済であり、主要な輸出国である日本の地位は、米ドル/円を世界で最も流動的で取引量の多い通貨ペアの1つにしています。

USD / JPYは、相互に、および他の通貨との関係で、米ドルと日本円の価値に影響を与える要因の影響を受けます。[連邦準備制度](/federalreservesystem)と[日本銀行（日銀）](/bankofjapan)の政策金利の金利差[は](/interest-rate-differential)、米ドル/円の為替レートに重要な影響を及ぼします。より高い金利は、その通貨建ての資産の所有者がより高い利回りを獲得できるようにするため、通貨を比較的魅力的にします。

たとえば、日銀の政策金利がゼロに近いまま、[フェデラルファンド金利](/federalfundsrate)がゼロ近くから2％に上昇した場合、投資家はドル建て短期金融市場で大幅に高い利回りを得ることができるため、ドルは円に対して上昇する傾向があります。 。実際、日銀が他の[中央銀行](/centralbank)の利上げを拒否した2022年半ばには、円は対ドルで24年ぶりの安値まで下落しました。日本の中央銀行と政府は、何十年にもわたって国を支配してきた[デフレを、](/deflation)エネルギー価格の上昇に起因する短期的な[インフレよりも大きな脅威と見なし続けました。](/inflation)

##安全な避難所

デフレの中での日本の低金利は、円を[安全な避難所](/safe-haven)通貨に変えました。これは、市場の混乱の時期にその価値が上昇する傾向にあったことを意味します。市場のストレス時には、米ドルなどの高利回りの外貨への日本の投資資金の流れが逆転する傾向があり、対ドルで円高となっています。これは大不況の際に明らかであり、これにより米ドル/円のレートは2007年の120から2009年までに90未満になりました。

逆に、金融市場のリスク欲求が高まると円安傾向にあります。[大不況](/great-recession)後の数年間、世界経済が回復するにつれ、円は米ドルに対してゆっくりと下落しました。日本銀行が大規模な[量的緩和に着手した2013年に弱体化が加速した](/quantitative-easing)。

##米ドル/円の相関関係

[USD/CHF](/usd-chf-us-dollar-swiss-franc-currency-pair)と[正の相関関係がある](/positive-correlation)傾向があります。これは、両方の通貨ペアが米ドルを[基本通貨として使用していることを除けば](/basecurrency)、スイスフランは投資家の間で伝統的な安全な魅力を持つもう1つの通貨であるためです。反対に、USD/JPYは金の価格と負の相関関係にあります。大不況の間に米ドル/円が下落したため、金価格は高騰しました。

> USD / JPY通貨ペアは、伝統的に米国財務省の利回りと密接かつ正の相関関係がありました。
