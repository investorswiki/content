# 優先株式
##優先シェアとは何ですか？

優先株式（より一般的には[優先株式と呼ばれる）](/preferredstock)は、普通株式の配当が発行される前に株主に支払われる配当を伴う会社の株式の株式です。会社が破産した場合、優先株主は普通株主の前に会社の資産から支払われる権利があります。

ほとんどの優先株には固定配当がありますが、普通株には通常ありません。優先株式株主も通常、議決権を持っていませんが、一般株主は通常、議決権を持っています。

##優先シェアを理解する

優先株は、累積優先株、非累積優先株、参加優先株、転換優先株の4つのカテゴリーに分類されます。

[累積優先株式](/cumulative_preferred_stock)には、一般株主が配当金を受け取る前に、過去に省略されたものを含め、すべての配当金を株主に支払うことを会社に要求する条項が含まれています。これらの配当金の支払いは保証されていますが、期日が到来したときに常に支払われるとは限りません。未払いの配当金には「延滞配当金」というモニカが割り当てられ、支払い時に現在の株式所有者に合法的に送金する必要があります。この種の優先株の保有者には、追加の報酬（利息）が与えられる場合があります。

>>

>四半期配当=[（配当率）x（パーバリュー）]÷4

>>

>>

>1株あたりの累積配当額=四半期配当金x未払いの支払い数

>>

非累積優先株は、省略または未払いの配当を発行しません。会社が特定の年に配当を支払わないことを選択した場合、非累積優先株式の株主は、将来いつでもそのような放棄された配当を請求する権利または権限を持ちません。

[参加する優先株式](/participatingpreferredstock)は、株主に、一般的に指定された優先配当の割合に等しい金額の配当に加えて、所定の条件に基づく追加の配当を支払う権利を与えます。この追加の配当は、通常、一般株主が受け取る配当額が所定の1株あたりの金額よりも多い場合にのみ支払われるように設計されています。会社が清算された場合、参加する優先株主は、株式の購入価格と、普通株主が受け取った残りの収益の比例配分分を返済する権利も持つ可能性があります。

転換優先株式には、株主が優先株式を一定数の普通株式に転換できるオプションが含まれています。これは通常、事前に設定された日付以降いつでも可能です。通常の状況では、転換可能な優先株式は、株主の要求に応じてこの方法で交換されます。ただし、会社は、株主または発行者が発行を強制することを可能にするそのような株式に関する規定を持っている場合があります。転換可能な普通株の価値は、最終的には[普通株の](/commonstock)パフォーマンスに基づいています。

##優先株とは何ですか？

優先株は、優先株とも呼ばれ、[普通株](/commonstock)と[固定収入の両方のセキュリティと同様の特性を提供するセキュリティの一種です](/fixed-incomesecurity)。会社が支払う配当に関しては、通常、優先株の保有者が優先されます。引き換えに、優先株は、普通株と同じレベルの議決権または上向きの参加を享受しないことがよくあります。

##優先シェアの主なタイプは何ですか？

優先シェアには、主に4つのタイプがあります。累積優先、非累積優先、参加優先、および変換可能です。累積優先株式の保有者は、過去の期間に支払われなかった配当に対して遡及的に配当を受け取る権利がありますが、非累積優先株式にはこの規定はありません。このため、累積優先株式は通常、非累積優先株式よりも高額になります。同様に、参加する優先株は、会社の利益が特定のレベルを超えるなど、特定の業績目標が達成された場合に追加の配当の利益を提供します。[転換社債の](/convertiblebond)ような転換優先株は、保有者が指定された[行使価格](/strikeprice)で優先株を普通株に転換することを可能にします。

##破産した会社の優先株を所有している場合はどうなりますか？

会社が破産した場合、その会社のさまざまな証券保有者が会社の資産を請求することになります。これらの証券保有者が資産のシェアを受け取る順序は、証券契約で彼らに与えられた特定の権利によって異なります。たとえば、優先株は一般的に普通株よりも優先されるため、普通株主よりも先に支払われます。ただし、優先株は一般に、社債、社債、またはその他の固定収入証券よりも優先度が[低く](/corporatebond)なり[ます](/debenture)。

##ハイライト

-優先株式には、累積（保証）、非累積、参加型、転換型の4種類があります。

-優先株式（優先株式）は、普通株式の配当が支払われる前に株主に配当が支払われる会社の株式です。

-優先株はリスクを嫌う投資家にとって理想的であり、呼び出し可能です（発行者はいつでもそれらを引き換えることができます）。
