# 債務雪崩
##債務雪崩とは何ですか？

債務のなだれは、加速され[た債務返済計画の一種です](/repayment)。基本的に、債務者は、各債務源に最小限の支払いを行うのに十分な資金を割り当て、残りの返済資金を最も高い[金利](/interestrate)の債務に充てます。債務雪崩アプローチを使用すると、最も高い金利の債務が完全に返済されると、追加の返済資金は次に高い有利子ローンに向けられます。このシステムは、すべての負債が完済されるまで続きます。

>成功するためには、借金雪崩プログラムに着手する前に、生活費と緊急事態の両方のために銀行に十分なお金を持っている必要があります。

>>

##債務雪崩はどのように機能しますか？

債務雪崩プログラムを開始する最初のステップは、債務の支払いに利用できる月収の金額を指定することです。この金額は、家賃、食料品、デイケア、輸送などの生活費を現在義務付けられていない資金から調達する必要があります。

たとえば、生活費を差し引いた後、毎月500ドルを利用して、負債の返済に充てることができるとします。現在のローンには次のものが含まれます。

[の年率（APR）](/apr)のクレジットカードで$ 1,000

-6％の金利で月額1,250ドルの車の支払い

-8％の金利で5,000ドル[のクレジットライン（LOC）](/lineofcredit)

簡単にするために、最低支払額が通常の毎月の分割払いである自動車ローンを除いて、各債務には[最低月額50ドルの支払いがあると仮定します。](/minimum-monthly-payment)

各ローンの最低月額支払い（$ 50 x 2）の支払いに$100を割り当てる必要があります。残りの400ドルは、あなたの最高利息の負債に充てられるお金に追加されます。この例では、20％の金利でクレジットカードの負債を決済するために450ドルを支払います。残高に追加料金を追加しないと仮定すると、クレジットカードの負債は4か月目までに全額返済されます。現在、追加の資金は、2番目に高い有利子負債であるLOCの廃止に向けられます。最後に、500ドルすべてが最低金利の債務である自動車ローンに充てられます。

##債務のなだれは関心を減らしますが、規律が必要です

###債務雪崩の利点

債務返済の債務雪崩法の利点は、計画に固執している限り、無借金の目標に向かって取り組んでいる間に支払う利息の額を最小限に抑えることです。また、蓄積される利息が少なくなるため、一貫した支払いを前提として、負債から抜け出すのにかかる時間が短縮されます。

[複合金利を](/compoundinterest)使用するため、これらの負債には利息が追加されます。複合利息が発生する割合は、調合期間の数が多いほど複合利息が大きくなるように、[調合](/compounding)の頻度に依存します。ほとんどのクレジットカードの残高は毎日利息を合成しますが、利息が毎月、半年ごと、または毎年複利になるローンがあります。

###債務雪崩のデメリット

借金のなだれは、規律とコミットメントを取り払う手法であるため、一部の人にとっては明らかに不利になる可能性があります。債務アバランシェシステムに固執するという最善の意図があっても、特に自動車や住宅の修理などの予期しない費用を経験した後は、すべての債務の最小支払いに簡単に戻すことができます。そのため、ほとんどのファイナンシャルプランナーは、加速された債務返済計画を試みる前に、まず[6か月の緊急資金を節約することを推奨しています。](/emergency_fund)

##すべての人のためではありませんが、機能します

借金を返済する借金雪崩の方法は、すべての人に適しているわけではありません。たとえば、それを見抜くには多大な規律が必要であり、日常生活費に十分なお金が必要であり、緊急事態に備えて銀行に余分なお金が必要です。ただし、コースを継続できる人にとっては、借金のなだれは、比較的安価かつ迅速に借金から抜け出すための良い方法です。

##債務スノーボールに似ていますが、異なります

、もう1つの加速された借金返済計画である[借金スノーボール](/snowball)とは異なります。債務の雪だるま式では、債務者は最小の支払いを超えたお金を使用して、最小の残高から最大の残高まで債務を返済します。この方法は、総利息の点でより多くの費用がかかりますが、借金スノーボール法は、いくつかの小さな借金を排除することによって動機を提供します。

##ハイライト

-債務雪崩は、債務を維持できる人にとって、比較的迅速かつ安価に債務を消去する体系的な方法です。

-債務のなだれを使用すると、債務の各ソースに対して最小の支払いを行い、残りの利用可能な資金を使用して、最も高い金利の債務に追加で支払うことができます。
