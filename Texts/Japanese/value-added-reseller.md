# 付加価値再販業者（VAR）
##付加価値再販業者（VAR）とは何ですか？

[エンドユーザー](/end-user)に再販するためにカスタマイズされた製品またはサービスを追加することにより、サードパーティ製品の価値を高める会社です。付加価値再販業者は、情報技術（IT）業界で重要な役割を果たし、コア製品に加えて、追加のハードウェア、インストールサービス、コンサルティング、トラブルシューティング、またはその他の関連製品やサービスを提供します。

>付加価値再販業者は、情報技術（IT）業界で重要かつ重要な役割を果たしています。

>>

##付加価値再販業者を理解する

付加価値再販業者が存在するのは、それらが製造業者、特にIT部門の製造業者にとって重要な流通チャネルを代表しているためです。付加価値再販業者は、コアハードウェアまたはソフトウェア製品またはシステム全体を受け取り、顧客向けに追加のシステムアドオンのパッケージをカスタマイズします。再販業者は機器の製造元ではありませんが、顧客のために適切にカスタマイズ、インストール、テスト、および保守するために、製品に関する完全な知識を持っていることが期待されます。

付加価値再販業者のメリットを認識しているIT企業は、通常、このチャネルを通じて売上を伸ばす手段として、製品の割引を提供します。これらの再販業者の中には、1つの会社に限定できるものもありますが、ほとんどの場合、顧客により多くの選択肢を提供するために、いくつかまたは複数のブランドを扱っています。

##付加価値再販業者の例

他の大手IT製品メーカーと同様に、シスコは、品質管理を確実にするために、メンバーの承認、認証、トレーニング、および監査を含む再販業者プログラムを育成しています。再販業者は、最初にシスコ製品の取り扱いを許可されている必要があります。これは、製品の販売をサポートするための人員とインフラストラクチャがあることを示しています。

サービスのレベルに応じて、シスコから「セレクト」、「プレミア」、または「ゴールド」の認定を受ける場合があります。シスコはさらに、エンタープライズネットワーク、サイバーセキュリティ、モノのインターネット（IoT）、データセンターなどの多くの分野に特化したリセラー向けのトレーニングを提供しています。付加価値再販業者は、定期的にシスコによる監査を受けて、リセラーネットワークのメンバーとしての継続的な価値を証明する必要があります。

##付加価値再販業者の利点

付加価値再販業者は、提供する付加価値のおかげで、リピートビジネスの可能性を高めることができます。 VARは、特定の製品およびソリューションの顧客に対する顧客サービス中心の単一の窓口としても機能します。多くの場合、VARは、顧客の課題を理解し、顧客満足を保証するような専門知識を提供するのに適しています。

VARマージンの大部分は、製品自体ではなく、付加価値のある製品やサービスに由来します。これらの製品自体は、通常、少量しかマークアップされていません。標準化されたVARプログラムのようなものはありません。すべての企業には、独自のビジネス環境と条件、条件、および方法があります。

##付加価値再販業者のデメリット

付加価値再販業者は、販売する製品のコストを実際に管理することはできず、再販プロセス全体が透過的でない場合があります。場合によっては、メーカーは再販業者に割引を提供することでこの問題を軽減しようとします。これにより、再販業者は顧客に請求する価格をより細かく制御できます。

また、再販業者は製品の品質や機能をほとんどまたはまったく制御できず、変化する顧客の要件に適応するために製造元に依存する必要があります。

##ハイライト

-付加価値再販業者は、販売している製品のコストを実際に管理することはできません。

-付加価値再販業者は、エンドユーザーに再販するためにコア製品にカスタマイズされた製品またはサービスを追加することにより、他社製品の価値を高めます。

-付加価値再販業者は、提供する付加価値を通じてリピートビジネスを増やすことができます。
