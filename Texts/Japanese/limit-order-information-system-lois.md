# 指値注文情報システム（LOIS）
##指値注文情報システムとは何ですか？

[株式市場](/stockmarket)の専門家が使用する電子システム（LOIS）です。指値注文情報システムは、取引所に上場されている証券の価格とサイズの見積もりを一覧表示します。このシステムによって生成されたデータは、証券が取引される場所、注文数量、およびビッドとオファーの価格に関する情報を配信するため、サブスクライバーとスペシャリストが取引を行うのに最適な市場を見つけるために使用されます。

##指値注文情報システム（LOIS）を理解する

指値注文情報システムは、利用可能な最良の証券価格を見つけるために使用され、ナスダックやニューヨーク証券取引所などの参加取引所で取引されている証券に関する情報を生成します。[制限](/limitorder)[注文](/limitorder)とは、特定の価格で株式を売買する注文であり、注文数量、買値、売値、証券が上場されている取引所などの情報が含まれます。指値注文は、株式の買い手または売り手が希望よりも高いまたは低い価格で売られるのを防ぐのに役立ちます。

##指値注文

指値注文は、銀行または証券会社に対して、指定された価格以上で一定量の金融商品を売買するために行われるテイクプロフィット注文です。指値注文は成行注文ではないため、注文が開いたままの期間中に[投資家が](/investor)設定した価格を満たせない場合、指値注文は実行されない場合があり[ます。](/marketorder)

指値注文の実行は保証されていませんが、市場で取引された場合、投資家が目標価格で売買する機会を逃さないことを保証します。ポジションの方向に応じて、指値注文は[買い指値](/buy-limit-order)[注文](/buy-limit-order)または売り指値注文と呼ばれることもあります。たとえば、買い手が1株あたり30ドルを超える支払いを希望しないことを規定する買い指値注文は、売り指値注文が行われるために少なくとも30ドルの株価を必要とする場合があります。

##注文管理システム

[注文管理システム](/oms)（ OMS）は、注文の入力と処理のために多くの業界で使用されているコンピューターソフトウェアシステムです。注文は、製品に応じて、企業、消費者、または両方の組み合わせから受け取ることができます。オファーと価格設定は、カタログ、Webサイト、またはブロードキャストネットワーク広告を介して行うことができます。

注文管理システムの別の用途は、証券の注文実行を容易にし、管理するソフトウェアベースのプラットフォームとしてです。注文管理システムは、金融市場ではトレードオーダー管理システムとして知られていますが、バイサイドとセルサイドの両方で使用されますが、バイサイドとセルサイドのOMSによって提供される機能はわずかに異なります。通常、エクスチェンジメンバーのみがエクスチェンジに直接接続できます。つまり、セルサイドOMSは通常エクスチェンジ接続を備えていますが、バイサイドOMSはセルサイド企業への接続に関係しています。
