# キャプティブ付加価値（CVA）
##キャプティブ付加価値（CVA）とは何ですか？

、親組織が所有および運営する[キャプティブ保険会社を設立](/captive-insurance-company)することにより、組織が実現できる経済的利益を指します。

##キャプティブ付加価値（CVA）を理解する

キャプティブ付加価値（CVA）は、組織のキャプティブ保険子会社が管理組織に利益をもたらす場合に発生します。キャプティブ保険会社を設立する主な理由は、キャプティブ保険会社の引受利益から親組織に利益をもたらしながら、所有者のリスクを保証することです。

組織構造の観点から、1つ以上の子会社を持つ会社は、完全所有の子会社としてキャプティブ保険会社を設立します。キャプティブ保険会社は資本化されており、キャプティブ対応法のある管轄区域で運営されており、認可された[保険会社として運営することができます](/insurance)。

キャプティブ保険会社は、所有者と参加者に特殊な形式の保険を提供します。参加者は、一般の人々よりも保険の適用範囲が少ないことがよくあります。これは、大規模な組織がリスクの一部を賄うために使用する可能性のある[自己保険と、](/selfinsurance)[責任ポリシー](/liability_insurance)などの市販の保険の両方とは異なります。

キャプティブ付加価値（CVA）は、キャプティブ保険会社を通じて、保険の引き受け、節税、およびより手頃な保険を取得することによる節約から得られる利益によって発生します。

##キャプティブ付加価値（CVA）の作成

キャプティブプログラムは、ほとんどの場合、大規模な組織内で見られます。これは、キャプティブプログラムがビジネス全体に与える機会の影響を評価する際に、通常、より多くの問題を抱えているため、キャプティブ[付加価値分析](/valueadded)を実施する能力が向上したことも一因です。大規模な組織は、悪い年に保険の損失をよりよく吸収することもできます。

キャプティブ保険会社を設立することにより、被保険者は自己資本をリスクにさらすことを選択します。従来の保険業界の外で活動するということは、被保険者を保護するために設計された規制を回避できることを意味し、トレードオフとしてそれらのコストを節約します。

キャプティブ保険と同様に、利益が実現したときに配当金が再投資される相互保険があります。[相互保険会社](/mutual-insurance-company)は余剰分を分配するのではなく蓄積する傾向があるため、キャプティブ保険子会社を設立すると、所有者の裁量で利益を分配することができます。

##キャプティブ付加価値（CVA）のリスクモデリング

被保険者のプールは組織全体に限定されているため、リスクモデリングは、より大きく、より多様な保険リスクプールよりも単純になる傾向があります。モデリングは、キャプティブ付加価値が実現される可能性が高いかどうか、および数年間でどれだけの利益が可能かを判断するのに役立ちます。

キャプティブ保険の潜在的な財務リスクを評価するために利用できるすべてのモデルの中で、人気のあるモデルは[リスクの価値](/value-risk-vor)（VOR）です。この手法では、特定のリスクが企業の目標達成にどのように役立つかという観点から、リスクのコストを確認します。リスクの価値は、[株主](/shareholder)と利害関係者が、従来とは異なるリスクを伴うことが知られている活動を行うことによって影響を受ける彼らの価値をどのように見るかを調べます。

リスクの大きさは、事業活動の種類と、会社が費用を回収できない可能性に依存し、1つの活動に費やすと[機会費用が発生するという追加の知識があります。](/opportunitycost)

企業が将来にリソースと資本をどのように投資するのが最善かを検討する場合、機会コストは常に重要な要素です。多くの組織は、コアビジネス目標に厳密な戦略的焦点を維持し、本質的でない活動に気を取られることを避けようとしています。

##ハイライト

-キャプティブ保険会社は、他の保険会社が提供する意思がない可能性のある親組織に組織が保険を提供することを許可します。

-リスクモデルは、キャプティブ保険の潜在的な財務リスクの評価に使用されます。人気のあるモデルは、バリューアットリスク（VOR）です。

-キャプティブ付加価値（CVA）は、キャプティブ保険会社を通じて、保険の引き受け、節税、およびより手頃な保険を取得することによる節約から得られる利益によって発生します。

-キャプティブ保険会社は、新しい子会社に資金を提供し、新しいビジネスからの潜在的な損失を吸収できる大規模な組織で最も一般的に見られます。

-キャプティブ付加価値（CVA）は、自社のキャプティブ保険会社を設立することで企業が達成できる経済的利益です。
