# 混合レート
##混合レートとは何ですか？

混合利率は、以前の利率と新しい利率の組み合わせを表す、ローンに請求される利率です。混合レートは通常、古いローンのレートよりも高いが、新しいローンのレートよりも低い金利が請求される既存のローンの借り換えを通じて提供されます。

このタイプのレートは、異なるレートの複数のローンの実際の債務または複数の利息の流れからの収益をよりよく理解するために、会計目的で計算されます。

>混合金利は、ローンの借り換え時に支払われる実際の金利を理解するためによく使用されますが、2番目の住宅ローンなどの追加の債務を追加するときにも使用できます。

>>

##混合レートの仕組み

混合レートは、借り手が既存の低金利ローンを借り換えることを奨励するために貸し手によって使用され、また[、資金のプールされたコストを計算するために使用されます](/pooled-cost-of-funds)。これらの利率は、社債の加重平均利率も表しています。結果として生じる利率は、社債の総利率と見なされます。

混合料金は、個人ローンまたは住宅ローンを借り換える個々の借り手にも適用されます。借り換え後の混合平均金利を計算するために消費者が利用できる無料のオンライン計算機がいくつかあります。

##混合レートの例

混合料金は、借り換えられた社債、または個人が借りた個人ローンに適用できます。混合利率の計算には、ローンの利率の加重平均を取ることが含まれます。

###社債

一部の企業には、複数の種類の社債があります。たとえば、会社に5％の利率で50,000ドルの負債があり、10％の利率で50,000ドルの負債がある場合、合計混合率は次のように計算されます。

>>

>（50,000 x 0.05 + 50,000 x 0.10）/（50,000 + 50,000）= 7.5％

>>

混合レートは、貸借対照表上の負債または投資収益を定量化するための原価会計でも使用されます。たとえば、ある会社に2つのローンがあり、1つは5％で1,000ドル、もう1つは6％で3,000ドルで、毎月利息を返済した場合、1,000ドルのローンは1年後に50ドルを請求し、3,000ドルのローンは請求します。 180ドル。したがって、混合率は次のようになります。

>>

>（50 + 180）/ 4,000 = 5.75％

>>

別の仮説の例として、A社が2020年第2四半期の結果を発表し、貸借対照表セクションの収益レポートに35億ドルの負債に対する会社の混合率を概説したと仮定します。当四半期の混合金利は3.76％でした。

＃＃＃ 個人ローン

銀行は混合レートを使用して顧客を維持し、実績のある信用できる顧客への融資額を増やします。たとえば、顧客が現在7％の利率で75,000ドルの住宅ローンを保有していて、現在の利率が9％のときに借り換えを希望する場合、銀行は8％の混合利率を提供する可能性があります。その後、借り手は、8％の混合率で150,000ドルの借り換えを決定できます。

##ハイライト

-混合利率は、以前の利率と新しい利率の組み合わせを表す、ローンに請求される利率です。

-混合利率を計算するには、ほとんどの場合、ローンの利率の加重平均を取ります。

-混合レートは、借り換えられた社債、または借り換えられた住宅ローンなどの消費者ローンに適用できます。
