# ホームオフィス経費
##ホームオフィスの費用とは何ですか？

ホームオフィスの費用は、主たる住居内での事業の運営または雇用関連の活動の遂行から発生する費用です。

##ホームオフィスの経費を理解する

[住宅ローン](/mortgage)に支払われる利子、固定資産税などの住宅費を年次確定申告で差し引くことができます。

IRSは、Publication587でホームオフィス控除に関する規則の詳細を提供しています。

>###ホームオフィス控除

>あなたがあなたの家で行う事業によって発生する費用は、あなたが自営業である場合にあなたの税額を下げることができるホームオフィス控除の基礎です。

>>

ビジネス目的でのみ使用されるサービスまたはユーティリティは、完全に差し引くことができます。これには、事務用品、電話回線、コンピューター機器が含まれます。許可される控除額は、住宅所有者が申告書を提出する方法や収益など、いくつかの要因によって異なりますが、通常の事業過程で発生する限り、ほとんどの場合、費用としていくつかの項目を請求できます。

ホームオフィスは、次の基準を満たしている場合にのみ、ホームオフィス控除の対象となります。

-それはあなたの主要なビジネスの場所として排他的かつ定期的に使用されます。

-これは、通常の取引またはビジネスの過程で、患者、クライアント、または顧客と会ったり、取引したりする場所として、排他的かつ定期的に使用されます。

-それはあなたの貿易やビジネスに関連して使用されるあなたの家に取り付けられていない別の構造です。

-特定のストレージ用途で定期的に使用されます。

-それはあなたのビジネスのためのレンタルとして使用されます。

-あなたはあなたの家をデイケア施設として使用しています。

経験則は「排他的かつ定期的」です。ホームオフィスとして指定する場所は、ホームオフィスとして排他的かつ定期的に使用する必要があります。 IRSの言葉によると、「スペースを恒久的なパーティションでマークする必要はありません。ビジネスと個人の両方の目的で問題のエリアを使用する場合、専用テストの要件を満たしていません。 」

家の隅にビジネスの取引にしか使用しない机がある場合、それはホームオフィススペースとしての資格がありますが、テレビを見ている家族の隣のソファに座ってラップトップでビジネスを行う場合、ソファをホームオフィスとして請求することはできません。

##ホームオフィスの費用を計算する方法

あなたの家のどれだけがホームオフィスを構成するか、そしてあなたの発生した費用のどれだけが控除可能であるかを計算することができるとIRSが言う2つの方法があります。最初の方法（「通常の」方法）は、実際の費用を計算することです。 2番目の方法は「簡略化された」方法と呼ばれ、より高速ですが、多くの控除が得られない場合があります。

###通常の方法

実際の費用の会計を使用する方法は、以下を計算することから始まります：

-あなたの事業費のどれが直接的、間接的、または無関係です。

-ビジネス目的で使用されるあなたの家の割合。

直接費用とは、業務に使用する地域でのみ塗装や修理を行うことです。間接費は、保険、公共料金、一般的な修理など、家全体の費用であり、ホームオフィスにもメリットがあります。

あなたのホームエリアの許容可能なビジネスの割合を把握するために、IRSは2つの方法を提供します：

1.ビジネスに使用する面積（長さに幅を掛けたもの）を家の総面積で割ります。

1.家の部屋がほぼ同じ大きさの場合、ビジネスに使用する部屋の数を家の部屋の総数で割ることができます。

通常の方法では正確な記録を保持する必要がありますが、IRSには、納税者を正しい道に導くための便利なワークシートがあります。

>###住宅費控除の制限

>あなたの費用があなたの家で行われる事業からのあなたの総収入より少ない場合、あなたの費用はすべて控除できますが、あなたの費用があなたの総収入を超える場合、それらの一部だけが差し引かれます。

>>

###簡略化された方法

簡略化された方法は、ホームオフィスの許容控除額の計算を簡略化するために2013年に導入されました。簡略化された方法で控除額を計算するには、次のことを知っておく必要があります。

-ビジネスを行う際に使用されるあなたの家の許容面積。自宅で一年中事業を行っていない場合や、年度中に面積が変わった場合は、使用した許容面積と毎月の事業日数を知る必要があります。

-あなたの家のビジネス利用からの総収入。

-あなたの家の使用に関係のないあなたの事業費の額。

-適格なビジネス用途が、定期的に（ただし排他的ではない）自宅のスペースを使用するデイケア施設である場合は、自宅の一部がデイケアに使用されている時間の割合も知る必要があります。

この情報を入手したら、次の手順で控除額を計算できます：

1.許容面積に5ドルを掛けます（資格のあるビジネス用途が、定期的ではあるが排他的ではない、自宅のスペースを使用するデイケアの場合は5ドル未満）。

1.住宅の事業利用に関連する総収入から、住宅の利用に関係のない事業からの費用を差し引く。これらの費用が家のビジネス使用からの総収入よりも大きい場合、あなたは家のこのビジネス使用の控除を受けることができません。

1.（1）と（2）の金額のうち小さい方を取ります。これは、簡略化された方法を使用して、あなたの家のこの適格なビジネス使用のために差し引くことができる金額です。

簡略化された方法を使用するファイラーの能力を制限する他の規則が適用されます。たとえば、スペースを他の人と共有する場合、両方が同じスペースを差し引くことはできません。その他の制限は、Publication587に記載されています。

##ホームオフィスの経費の実際の例

例として、自宅の外で自分のビジネスを運営しているフリーランスのライターを考えてみましょう。彼らは約200平方フィートの専用オフィススペース、仕事関連の電話にのみ使用される携帯電話、そして作家に社説の手がかりを提供する雑誌の購読を持っています。これらのアイテムはすべて、作家の家の200平方フィートを含むホームオフィスの費用として税控除の対象となります。これは、その部屋がビジネス目的で使用されるためです。

さらに、ライターは、契約書の印刷に使用するインク、署名された契約書を送信するために購入しなければならなかったオールインワンプリンターの全費用、および完了した業界関連のトレーニングを差し引くことができます。

>###不動産控除には上限があります

>減税および雇用法により、固定資産税の申告者が控除できる金額が制限されているため、10,000ドルを超える場合は、ホームオフィスの控除を使用して固定資産税の全額を控除することはできません。

>>

遠隔地の従業員であろうと自営業であろうと、人が家の外で働くときに差し引かれる可能性のあるさまざまな費用があります。認定された[税務](/tax-accounting)専門家は、利用可能な控除額を確認し、請求されているすべての項目が有効であることを確認できます。

たとえば、このフリーライターが専用のオフィススペースを持っておらず、代わりに毎日家のすぐ近くのコーヒーショップで働いていた場合、ユーティリティと住宅ローンに関連する費用を請求することはできません。ホームオフィス控除。彼らが店で働いている間に彼らが毎日購入するコーヒーとドーナツのように、彼らに利用可能な追加の控除があるかもしれません。

##ハイライト

-自宅で行う事業の運営にかかる費用は、一定の限度内で連邦税から控除できます。

-控除のルールは2017年の減税および雇用法で変更されたため、2018年以降に公開された控除に関する情報を必ず入手してください。

-あなたの費用と控除額を計算するために、あなたはあなたのすべての費用を集計してあなたのビジネスに捧げられたあなたの家のパーセントを計算するか、同じ計算をするために単純化された方法を使うことができます。
