# ALL（アルバニアレク）
##アルバニアレク（ALL）とは何ですか？

アルバニア語**lek**（ALL）は、アルバニア共和国の国の通貨です。複数形のレクはレクエで、1つのアルバニアのレクは100個の** qindarka **で構成されています。インフレにより、qindarka建ての硬貨は発行されなくなりましたが、法定[通貨として受け入れられてい](/currency)ます。

2020年12月の時点で、US $1.00は約101ALLに相当し、1ALLは約$0.01に相当します。

##アルバニア語レクを理解する

アルバニア共和国で使用されているAlbanianlek（ALL）は、1926年にアルバニア国立銀行によって最初に発行されました。この通貨の名前は、Alexander the Greatに由来します。これは、彼の名前の短縮形が**Leka**であるためです。アルバニア語。 1926年以前は、アルバニアは自国通貨を持っておらず、代わりに厳格な[金本位制を遵守する](/goldstandard)か、代わりに外貨に依存していました。第一次世界大戦前、オスマントルコの**ピアストル**はこの地域で広く流通していました。

第一次世界大戦後、国は法定通貨として自国通貨を課すさまざまな勢力による軍事占領の継承を見ました。金フランまたは**フラン生殖器**は、当時最も広く使用されていた通貨単位になりました。アルバニアには公的資金がなかったため、人々は国境を接する国の通貨を使用して金の為替レートを固定していました。

[欧州連合（EU）](/europeanunion)への加盟を申請しており、公式協議は2020年に始まります。承認された場合、アルバニアは一般的なユーロ（ [EUR](/euro) ）通貨に切り替える可能性があります。

##アルバニアレクの歴史

最初に鋳造されたレケは、5および10キンダルカの金種で導入されたブロンズコインと、1 / 4、1 / 2、および1レックの金種で発行されたニッケルコイン、および銀1、2、および5**フラン**でした。フランガは現在、時代遅れの通貨単位であり、当時は5レクに相当します。フランガコインは1926年から1939年まで使用され、アルバニア人の王ゾグー1世が描かれていました。

第二次世界大戦中のアルバニアのイタリアへの降伏中に、ベニート・ムッソリーニの指揮の下で新しい一連のコインの発行が発生しました。硬貨には、当時のイタリア王であったヴィットーリオエマヌエル3世の肖像画があり、1941年まで流通していました。1947年にアルバニアがナチスの占領から解放された後、共産党が国を支配しました。新しい当局は古いコインを流通から取り除き、社会主義国の紋章を表示する新しいコインを発行しました。新しい亜鉛コインは、1 / 2、1、2、および5レケの金種を持っています。この通貨はソビエトルーブルに固定され、1965年の通貨改革まで使用されていました。

1946年から1965年までのソビエト占領中、ルーブルが再建された後、1961年までは12.5レックから1ルーブル、1961年から1965年までは1.25レックから1ルーブルで[ソビエトルーブルにペグされました。](/rub-russian-ruble) 1965年以降、ルーブルの再評価は引き換えに不平等を生み出し、2番目のレクの問題を引き起こしました。レックの2番目のシリーズは、1つの新しいレックに対して10の古いレックの割合で最初のレックと交換されました。

最も一般的な紙幣の種類は、現在200、500、1,000、2,000、および5,000レケです。 2,000レケ紙幣は1996年にALLの第3号に付属していました。

##アルバニアの政治経済史

アルバニア共和国は、アドリア海の崖が並ぶ海岸線により、バルカン半島で戦略的な位置を占めています。この地域は1912年にオスマン帝国からの独立を宣言し、公国としての存在を開始しました。当初、文化的および宗教的に分離されたグループ間に不和があり、派閥間の一時的な戦いが発生しました。

1925年から1928年の間に、最初のアルバニア共和国は、この地域の安定を回復しようとする権威主義体制に陥りました。しかし、イタリア人は政権に貿易と海運に対するイタリアの影響を受け入れることを強制し、それが最初の共和国の短命を終わらせ、それを君主制に変えました。その後、この地域の支配を強化するために、イタリアは軍隊をこの地域に移し、1943年まで領土を占領しました。第二次世界大戦中、アルバニアは一時的にナチスドイツの支配下に置かれ、その間に戦闘は国とその人口を荒廃させました。

枢軸国が衰退するにつれて、ソビエトは入国し、国を解放した。アルバニアは共産主義国家、アルバニア人民共和国になりました。ソビエト統治時代、この地域は急速に工業化し、経済は成長しました。同時に、同盟国であるソビエト連邦、中国、ユーゴスラビアとの間で債務を抱えていた。

1980年代後半から1990年代にかけて共産主義の支配が崩壊したことで、アルバニアは1991年に第4共和国を形成しました。しかし、腐敗により、国の資金の多くが政府支援のポンジ計画と銀行詐欺に浪費されました。たとえば、多くの市民は、1996年に崩壊したこれらの詐欺に投資するために家を売り、ローンを組むことを強要されました。抗議が全国で勃発し、暴力的になり、現政権を追放しました。

2019年の時点で、[世界銀行](/worldbank)のデータによると、この国の年間国内総生産（GDP）は2.2％で、年間インフレ率は0.4％です。

##ハイライト

-国が政権交代と内戦を何度か繰り返したため、通貨は世界の通貨に対して時間とともに価値を失っています。

-アルバニアの**lek**（ALL）は、1926年から使用されているアルバニアの公式通貨です。

-レックの前は、国は通貨として金を使用していたか、オスマン帝国の**ピアストル**や**フランジャーミナル**などの外国の問題に依存していました。

-アルバニアはEU加盟を申請しており、承認された場合、公式通貨として一般的に使用されるユーロに切り替わる可能性があります。
