# 申請書（RFA）
##申請リクエスト（RFA）とは何ですか？

申請要求（RFA）は、組織が[助成金の資金提供](/government-grant)が可能であることを発表する一種の勧誘通知です。助成金は、プロジェクトまたは研究の利益のために連邦、州、または地方政府によって授与される金銭的賞です。 RFAは、研究者や他の組織に、資金の使用方法について[入札を提示する可能性があることを通知します。](/bid)申し込みのリクエストには、通常、対象となるプログラムの種類、期待される内容、申し込みの提出方法と審査方法の概要が記載されています。

##申請のリクエスト（RFA）を理解する

申請のリクエストは、一般的に[政府機関](/federal-agencies)や[非営利機関に関連付けられています](/non-profitorganization)。これらの組織は、非政府機関よりも助成金の形でお金を解放する可能性が高いためです。たとえば、政府機関はグリーンエネルギーの研究に資金を割り当てています。プロジェクトのガイドラインと制限、利用可能な金額、およびプロジェクトの範囲（この場合はグリーンエネルギー）を示すRFAをリリースします。

##助成プロセスの一環としての申請申請（RFA）

申請のリクエストは、助成金作成プロセスの1つの側面です。このプロセスには多くの段階があり、通常は資金調達機会の発表（FOA）から始まります。この発表では、通常、資金をめぐる競争の結果として、連邦政府機関が裁量的助成金または協力協定を授与する意向を公表します。 FOAの直後に、プログラムアナウンス（PA）が続きます。これは、優先順位または重点分野を開示し、機会が開いている期間のタイミングガイドラインを設定します。次のステップはRFAであり、次に契約提案の勧誘である[提案依頼書（RFP）が続き](/request-for-proposal)、最後に、ポリシーと手順、RFAまたはPAの発表の修正、およびその他の情報を発表する通知（NOT）が続きます。アイテム。

##申請リクエスト（RFA）の例

RFAには、問題のプロジェクトの概要、アプリケーションの評価方法、提出と評価の両方のプロセスのタイミング、およびその他の重要な情報が含まれます。

###米国環境保護庁（EPA）

たとえば、飲料水システムの容量を構築しようとするプログラムの米国環境保護庁からの最近の申請要求には、次のセクションが含まれています：

1.申請者の適格性

1.プロジェクトの適格性

1.しきい値の問題

1.評価の問題

1.タイミングとロジスティクス

1.予算の懸念

1.資金調達の明確化

1.その他（未分類の質問に対応）

###国立衛生研究所（NIH）

[米国保健福祉省（HHS）](/us-department-health-and-human-services-hhs)の一部である国立衛生研究所は、毎日発行されるNIH助成金および契約ガイドを通じて、同様の概要を提供しています。NIHは、 「サポートの申請を検討しています。基礎的または臨床的な生物医学、行動、および生物工学の研究の概要」は、次のような申請プロセスの要求について説明しています：

-1つ以上のNIH機関が助成金を授与するための資金を確保している、より狭く定義された領域を特定します。

-通常、RFAアナウンスで指定された日付の単一のレシート（以前に受信）があります。

-それらは通常、発行する授与コンポーネントによって召集された科学レビューグループによってレビューされます。

##ハイライト

-RFAには、提出と評価の両方のプロセスのタイミングと、より重要な情報も含まれます。

-申請要求（RFA）は、助成金が利用可能であることを組織が発表する一種の勧誘通知です。

-RFAには、問題のプロジェクトの概要と、アプリケーションの評価方法が含まれます。

-RFAは、研究者や他の組織に、資金の使用方法について入札を提示する可能性があることを通知します。
