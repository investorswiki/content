# 複数の価格
##価格の倍数とは何ですか？

価格の倍数は、評価のスナップショットとして、会社の株価を特定の1株あたりの財務指標と組み合わせて使用する比率です。株価は通常、選択した1株あたりの指標で割って比率を形成します。価格の倍数により、投資家は、収益、キャッシュフロー、帳簿価額などの基本的な[指標](/metrics)に関連して、企業の株式の市場価値を評価できます。

##価格の倍数を理解する

に会社の簡単な[評価を行う機会を与えます。](/valuation)価格の倍数は世界中の投資家に理解されており、株式のすべての利害関係者によって標準として受け入れられています。

投資家は通常、価格の倍数比を次の形式で表現します。倍数の価格=株価/1株あたりの指標。

比率の分子は株価です。これは、企業の株式の1株が特定の時間に販売される価格です。会社の株価は、会社の株価チャートを見れば簡単にわかります。

分母は、特定の価格の倍数の計算に使用される1株あたりの指標です。このメトリックは、企業のパフォーマンスのある側面を測定します。投資家は、会社の[財務諸表のデータを使用する](/financial-statements)か、証券会社のサイトで会社の履歴データの一部としてメトリックを見つけることにより、これらのメトリックを計算できます。

##価格の倍数の種類

比率分析は、投資家が企業の業績を経時的に評価することにより、企業の財務状態を判断するのに役立ちます。価格の倍数は、投資家がさまざまな企業を潜在的な投資機会として比較するのに役立つ洞察を明らかにします。一般的な価格の倍数には、[価格対収益](/price-earningsratio)（P / E）比率、[価格対先物収益](/forwardpe)（先物P / E）、[価格対帳簿](/price-to-bookratio)（P / B）比率、および価格対売上（ P/S比。

投資家とアナリストは、収益と将来のパフォーマンスを予測するためにいくつかの比率を使用します。その他の比率には、[価格対有形帳簿](/ptbv)（P / TBV）、[価格対キャッシュフロー](/price-to-cash-flowratio)（P / CF）、価格対EBITDA（P / EBITDA）、および[価格対フリーキャッシュ](/pricetofreecashflow)[フロー](/pricetofreecashflow)（P / FCF ）が含まれます。 ））。これらの価格の倍数は表面上は簡単に計算できますが、投資家は分母の構成要素を分析して数値が正確であることを確認するように注意する必要があります。

>価格の倍数を使用して会社の業績を評価する場合、投資家は、財務指標を歪める可能性のある特別な項目、1回限りの、または非定期的な要因がないことを確認する必要があります。

##価格の倍数の利点

価格の倍数は、株式の評価を静的かつ前向きに見る上で重要な目的を果たします。倍数は、会社の現在および将来の（予測された）評価倍数を、その過去の数値および同業他社の数値と比較するために使用されます。

[過大評価](/overvalued)されているか、過小評価されているか、または公正に評価されているかを判断するのに役立ちます。これらの比率は、一般的に理解と使用が容易であるため、投資家にとって魅力的です。価格の倍数は、投資家がキャッシュフローや収益などの価値の尺度の観点から何を購入するかを理解するのに役立ちます。

##特別な考慮事項

投資家は、特定の業界に関連する価格の倍数のみを使用する必要があります。 AP / E比は、テクノロジー企業にとって適切な評価基準ですが、収益にかなりの金額の減価償却を請求する資本集約的な公益事業会社にとっては必ずしもそうではありません。この場合、非現金費用は[一般に公正妥当と認められた会計原則](/gaap)（GAAP）の収益を低下させ、したがって[1株当たり利益](/eps)（EPS）を低下させ、会社の評価について誤った印象を与える可能性があります。時々P/Eは窓の外に投げ出されます。

たとえば、何年にもわたって妥当なP / Eを探していたAmazon株の非所有者は、Amazonの場合のP / Eの倍数、またはその欠如が1つのイオタに関係していないことに気付いたかもしれません。将来的にはそうなるかもしれませんが、この価格の倍数を無視したアマゾン株の所有者は明らかに勝者です。

##価格の倍数を見つける場所

ほとんどの金融ウェブサイトは、P / E、P / B、P/Sなどの基本的な株価収益率を表示します。比率は通常、最後の[12か月（TTM）](/ttm)または最後の暦期間に基づいて計算されます。より真面目な投資家の場合、特定の業界に関連する倍数の手計算は、企業が財務報告で提供するデータを使用して行うことができます。

##ハイライト

-投資家とアナリストは、価格の倍数を使用して、潜在的な投資について会社をレビューするプロセスの一部として、会社の評価についての洞察を得ます。

-一般的な株価収益率には、株価収益率（P / E）、株価収益率（P / S）、株価収益率（P / CF）が含まれます。

-価格の倍数は、会社の株価を1株あたりの財務指標と組み合わせて使用する比率です。
