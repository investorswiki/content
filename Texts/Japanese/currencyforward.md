# 通貨フォワード
##通貨フォワードとは何ですか？

[外国為替市場](/foreign-exchange-markets)における拘束力のある契約であり、将来の[通貨](/currency)の購入または売却の[為替レート](/exchangerate)を固定します。通貨先渡は、基本的に、前払いの[証拠金支払い](/initialmargin)を伴わないカスタマイズ可能なヘッジツールです。

通貨先物のその他の主な利点は、その条件が標準化されておらず、上場通貨先物とは異なり、特定の金額および任意の満期または納期に合わせて調整できることです。

##通貨先渡を理解する

[通貨先物](/currencyfuture)や[オプション契約](/optionscontract)などの他のヘッジメカニズム（それぞれ証拠金要件と保険料の支払いに前払いが必要）とは異なり、大企業や銀行が使用する場合、通貨先物は通常、前払いを必要としません。

ただし、通貨先渡には柔軟性がほとんどなく、拘束力のある義務があります。つまり、「ロックイン」レートが最終的に不利であることが判明した場合、契約の買い手または売り手は立ち去ることができません。したがって、先渡取引を行う[金融機関は](/financialinstitution)、不着や決済のリスクを補うために、取引関係のない[個人投資家や中小企業からの預金を要求する場合があります。](/retailinvestor)

オプションが相互に受け入れ可能であり、契約で事前に指定されている場合、通貨先渡決済は現金または配達ベースのいずれかで行うことができます。通貨先渡は、一元化された取引所で取引されないため、[店頭（OTC）証券であり、「完全先渡」とも呼ばれます。](/otc)

>輸入業者と輸出業者は通常、為替レートの変動をヘッジするために通貨先渡を使用します。

##通貨フォワードの例

、[通貨ペアの](/currencypair)[金利差に](/interest-rate-differential)依存します（両方の通貨が[外国為替市場](/forex)で自由に取引されていると仮定します）。

たとえば、カナダドルの現在の[スポットレート](/spot_rate)がUS $ 1 = C $ 1.0500、カナダドルの1年金利が3％、米ドルの1年金利が1.5％であると仮定します。

[金利平価](/interestrateparity)に基づいて、US $ 1 + 1.5％の利息は、C $ 1.0500 + 3％の利息に相当します。つまり、次のようになります。

-$ 1（1 + 0.015）= C $ 1.0500 x（1 + 0.03）

-US $ 1.015 = C $ 1.0815、またはUS $ 1 = C $ 1.0655

したがって、この場合の1年先物レートはUS $ = C$1.0655です。カナダドルは米ドルよりも金利が高いため、[グリーンバックに対して](/greenback)[先物割引で取引されることに注意してください](/forwarddiscount)。また、1年後のカナダドルの実際のスポットレートは、現在の1年先物レートとは相関関係がありません。

通貨先渡レートは、単に金利差に基づいており、実際の為替レートが将来どこにあるかについての投資家の期待を組み込んでいません。

##通貨先渡とヘッジ

通貨先渡はヘッジメカニズムとしてどのように機能しますか？カナダの輸出会社が100万米ドル相当の商品を米国の会社に販売しており、1年後に輸出収入を受け取ることを期待していると仮定します。輸出業者は、カナダドルが1年後の現在のレート（1.0500）から上昇した可能性があることを懸念しています。これは、1米ドルあたりのカナダドルの受け取りが少なくなることを意味します。したがって、カナダの輸出業者は、 1米ドル= 1.0655カナダドルの先物レートで、今から年間100万ドルを販売する[先渡契約を締結します。](/forwardcontract)

今から1年後の場合、スポットレートはUS $ 1 = C $ 1.0300です。これは、C $が輸出者の予想どおりに上昇したことを意味します。フォワードレートを固定することで、輸出者はC $ 35,500の利益を得ることができます（販売することにより）。スポットレートC$1.0300ではなく、C $1.0655で100万米ドル）。一方、1年後のスポットレートが1.0800カナダドル（つまり、輸出業者の予想に反してカナダドルが下落した場合）の場合、輸出業者の想定損失は14,500カナダドルになります。

##ハイライト

-上場通貨先物やオプション契約とは異なり、大企業や銀行が使用する場合、通貨先渡は前払いを必要としません。

-これらは一般的にヘッジに使用され、特定の想定元本や納期など、カスタマイズされた条件を持つことができます。

-通貨先物レートの決定は、問題の通貨ペアの金利差に依存します。

-通貨先渡は、通貨ペアの為替レートを固定する外国為替市場で取引されるOTC契約です。

＃＃ よくある質問

###通貨先物と通貨先物の違いは何ですか？

通貨先物と先物は非常に似ています。主な違いは、通貨先物は標準化された条件を持ち、 [Chicag](/cme) [o Mercantile Exchange](/cme) （CME）などの取引所で取引されるのに対し、先物はカスタマイズ可能な条件を持ち、店頭（OTC）で取引されることです。

###通貨先渡はどの通貨で書くことができますか？

それらはカスタマイズ可能でOTCを取引するため、通貨先渡は任意の数の通貨ペアに表示されます。どちらが取引に関与するカウンターパーティによって決定されます。

###通貨先渡が使用されるのはなぜですか？

通貨先渡は、一定期間の為替レートを固定するために使用されます。これは、外貨エクスポージャーを[ヘッジするためによく使用されます](/hedge)
