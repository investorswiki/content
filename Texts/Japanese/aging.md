# エージング
##老化とは何ですか？

[売掛金（AR）](/accountsreceivable)内の不規則性を評価および特定するために使用する方法です。アカウントは、請求書の未払い期間に応じて分類および検査されるため、個人は会社の[貸倒れ](/baddebt)や財務状況をよりよく把握できます。

[売掛金エージング](/accounts-receivable-aging)または[エージングスケジュール](/aging-schedule)とも呼ばれます。

##老化を理解する

ARは、顧客によって提供または使用されたがまだ支払われていない商品またはサービスの会社による残高です。[貸借対照表に](/balancesheet)[流動資産](/currentassets)として記載されており、クレジットでの購入に対して顧客が支払うべき金額を示しています。

エージングには、会社の未払いの顧客の[請求書](/invoice)と[クレジット](/credit)メモを日付範囲で分類することが含まれます。スケジュールはさまざまな時間枠でカスタマイズできますが、通常、これらのレポートには、期日から30日、31〜60日、61〜90日などの30日グループの請求書が一覧表示されます。エージングレポートは顧客名でソートされ、各請求書は番号または日付で分類されます。

>###重要

>一般に、売上請求書の未払い期間が長いほど、会社が未払い分を回収できない可能性が高くなります。

>>

企業は、この[会計](/accounting)プロセスを利用して、与信および回収機能の有効性を把握し、潜在的な貸倒れを見積もっています。経営陣は、[疑わしい口座の引当金を修正し、](/allowancefordoubtfulaccounts)しばしば不良債務となる期間ごとの請求額の過去の割合を決定し、その割合を最新の経年報告に適用します。

##エージングレポートの例

会社Aは通常、30日間で1％の不良債権、31日から60日間で5％の不良債権、61日以上で15％の不良債権を抱えています。最新のエージングレポートには、30日間で500,000ドル、31〜60日間で200,000ドル、61日以上で50,000ドルが含まれています。

計算（$ 500,000 x 1％）+（$ 200,000 x 5％）+（$ 50,000 x 15％）に基づいて、会社は$22,500の疑わしいアカウントの引当金を持っています。

##老化の利点

高齢化により、企業は不良債権の可能性のあるケースを認識し、未払いの請求書を把握し、未払いの請求書を最小限に抑えることが容易になります。[売掛金を](/receivables)監視し、適切に分類する管理チームは、どの顧客がコレクションに送信する必要があるかを特定するために適切に配置する必要があります。これには、フォローアップ請求書によるターゲティングが必要であり、会社が売掛金をあまりにもゆっくりと収集し、[信用リスクを負っているのかどうかを確認する必要があります](/creditrisk)。

多くの場合、エージングはクレジットと販売慣行を決定するための便利なツールです。たとえば、会社が借りているものを集めるのが困難な場合、連続した後払い者に現金のみで事業を拡大することを選択するかもしれません。

エイジングは、投資家やアナリストが外部で使用することもできます。会社の売掛金コレクションの減速の兆候は、ずさんな慣行を示唆している可能性があります。これらの問題を是正するための措置が迅速に講じられない場合、現金が枯渇し、債権者が会社の資金の貸付を延期する可能性があります。[キャッシュフロー](/cashflow)の問題は大きな危険信号です。請求書を投資して支払うための流動的な通貨がなければ、会社は、登録する収益と利益の量に関係なく、[破産するリスクがあります。](/insolvency)

##エイジングの制限

有用ではありますが、老化は決して欠陥がないわけではありません。多くの企業は月末に請求し、数日後にエージングレポートを実行するため、前月の未払いのアカウントが表示されます。一部の請求書の支払いが進行中ですが、売掛金が誤って悪い状態で表示さ[れ](/payment)ます[。](/payment)月末の請求の前にレポートを実行すると、ARが少なくなり、実際には多くの現金が未払いであるにもかかわらず、入ってきた現金はほとんど表示されません。

さらに、経営陣は特定の会社に特に長いまたは異常に短いクレジット期間を延長する場合があります。つまり、一部の請求書は、実際にはそうではないのに、経年変化レポートに極端に遅れているか、時間どおりに表示される場合があります。結果として、会社の[財務状態](/financial-health)を正確に表すには、会社の与信条件がレポートの期間と一致することが重要です。

レポートに適用されていないクレジットにも特別な注意が必要です。それらは、適用される請求書を見つけて、エージングレポートの延滞債権の量を減らすことによってクリーンアップできます。

##ハイライト

-経年劣化は、会計士や投資家が会社の売掛金（AR）内の不規則性を評価および特定するために使用する方法です。

-未払いの顧客の請求書とクレジットメモは、請求書が未払いになっている期間を決定するために、通常は30日の日付範囲で分類されます。

-企業は、エージングを適用して、クレジットおよび回収機能の有効性を理解し、潜在的な不良債権を見積もります。

-投資家はこれと同じ情報を使用して、潜在的なキャッシュフローの問題と破産のリスクを特定できます。
