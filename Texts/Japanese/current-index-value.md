# 現在のインデックス値
##現在のインデックス値とは何ですか？

は、[変動金利](/variableinterestrate)[ローンの基礎となる](/indexed_rate)[インデックスレート](/indexed_rate)の最新の値を指します。変動金利ローンは、借り手が支払う必要のある完全にインデックス化されたレートを計算するために、インデックス化されたレートとマージンに依存しています。この値は、一般的な市況と、市場で発生する状況に基づく変化を反映している必要があり[ます](/market)。

>現在の指数値は、一般的な市況および市場に基づく変化を反映しています。

>>

##現在のインデックス値を理解する

現在のインデックス値は、変動金利の[ローン](/loan)商品を計算するために貸し手によって使用されます。借り手がこれらのローンに支払うレートは、完全インデックスレートと呼ばれます。これは、インデックスレートと[マージンの両方の関数です](/margin)。貸し手は、さまざまなリセット時間で変化する完全にインデックス化されたレートのさまざまな変動金利ローン商品を提供する場合があります。

インデックス付きレートは貸し手によって設定され、次のようなさまざまなインデックスに基づくことができます。

-貸し手の[プライムレート](/primerate)

-[ロンドン](/libor)[銀行オファーレート](/libor)（LIBOR）

-資金コスト指数（COFI）

-貯蓄指数（COSI）の[コスト](/cosiindex)

-さまざまな[米国債](/ustreasury)

貸し手は、インデックス付きレートの構成要素を決定し、ローン契約の条件の概要を示します。ただし、選択したインデックスは、閉じた後も変更されません。

変動金利ローンのローン金利は、インデックス金利と借り手の証拠金を加算して計算されます。変動金利商品のローン引受プロセスでは、引受人は借り手に信用プロファイルまたは[信用格付け](/creditrating)に基づいて証拠金を割り当てます。借り手は完全にインデックス化されたレートを支払う必要があります。これは、基礎となるインデックス化されたレートの変化に応じて変化します。多くの変動金利のクレジット商品では、変動金利は[変動し](/volatility)ます。これは、いつでも変更できることを意味します。したがって、現在のインデックス値が変更されると、借り手のレートが変更されます。

##特別な考慮事項

[変動金利住宅ローン（ARM）には、固定金利と変動](/arm)[金利](/interest)の両方が組み込まれています。[ロンドン銀行間取引金利](/libor)（LIBOR）は、住宅ローンの変動金利として使用される一般的な指数です。 LOBORは2021年の終わりに存在しなくなります。ARMを取得する借り手は、指定されたリセット日が発生するまでの最初の数年間は固定料金を支払います。

この固定金利は通常、ローンの最初の5年間に適用され、その後は毎年金利がリセットされます。この種のローンは、 [5/1ハイブリッド住宅ローンとして知られてい](/5-1_arm)ます。借り手は、リセット日およびその後の該当する期間ごとに変動金利で利息が請求されます。変動金利住宅ローンの変動金利は、標準の変動金利商品と同じ方法で計算されます。借り手は、基礎となる指数レートと証拠金を支払います。

変動金利住宅ローンでは、変動金利は変動する可能性があり、基礎となる現在のインデックス値が変更されるたびに、または変動金利をスケジュールできるたびに変更されます。スケジュールされた変動金利では、借り手はスケジュールされた時間にリセットされる[完全にインデックス化されたレートを支払います。](/fullyindexedinterestrate)リセット日が予定されているほとんどの変動金利住宅ローンは、12か月ごとにリセットされます。変動金利がスケジュールに基づいている場合、借り手の金利は現在のインデックス値にその特定の日付の借り手のマージンを加えたものに変更され、完全にインデックス化されたレートは次のリセット日まで変更されません。

##ハイライト

-インデックスレートは、貸し手のプライムレート、LIBOR、米国債などのさまざまなインデックスに基づく場合があります。

-現在のインデックス値は、変動金利ローンの基礎となるインデックスレートの最新の値です。

-変動金利ローンは、借り手が支払わなければならない完全にインデックス化されたレートを計算するために、インデックス化されたレートとマージンに依存しています。
