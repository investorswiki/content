# シナリオ分析
##シナリオ分析とは何ですか？

、ポートフォリオの有価証券の価値の特定の変化、または金利の変化などの主要な要因が発生したと仮定して、一定期間後の[ポートフォリオ](/portfolio)の期待値を推定するプロセスです。

シナリオ分析は、一般的に、不利なイベントに応じたポートフォリオの価値の変化を推定するために使用され、理論上の最悪のシナリオを調べるために使用される場合があります。

>シナリオ分析は、アナリストが行った入力と仮定と同じくらい優れています。

>>

##シナリオ分析の仕組み

手法として、シナリオ分析では、投資期間内に再投資される期待収益に対してさまざまな再投資率を計算します。

[感度分析](/sensitivityanalysis)の原則に従って、さまざまな状況（シナリオと呼ばれる）の発生に基づいてポートフォリオの価値の変化を推定するプロセスを提供します。感度分析は、独立変数のさまざまな値が特定の条件下で従属変数にどのように影響するかを示しています。

これらの評価は、非常に可能性の高いものから非常に可能性の低いものまで、さまざまな潜在的なイベントに関連する特定の投資内に存在するリスクの量を調べるために使用できます。分析の結果に応じて、投資家は、存在するリスクのレベルが自分の快適ゾーン内にあるかどうかを判断できます。

##特別な考慮事項

###シナリオ分析の例

特に最悪のシナリオを調べるシナリオ分析の1つのタイプは、[ストレステスト](/stresstesting)です。ストレステストは、コンピュータシミュレーション技術を使用して、将来起こりうる重大な状況に対する機関や[投資ポートフォリオ](/portfolio-investment)の回復力をテストするためによく使用されます。このようなテストは、投資リスクと資産の妥当性を評価するために、金融業界で通常使用されています。

ストレステストは、内部プロセスと制御の評価にも使用されます。近年、規制当局は、金融機関に対して、自己資本やその他の資産が適切であることを確認するためのストレステストの実施を要求しています。

###シナリオ分析と投資戦略

シナリオ分析にアプローチする方法はたくさんあります。一般的な方法は、日次または月次の証券リターンの[標準偏差を決定し](/standarddeviation)、各証券が平均リターンの上下2または3標準偏差であるリターンを生成する場合にポートフォリオに期待される値を計算することです。このように、アナリストは、これらの極値をシミュレートすることにより、特定の期間中のポートフォリオの価値の変化に関して妥当な量の確実性を持つことができます。

検討中のシナリオは、新製品の発売の相対的な成功または失敗などの単一の変数、または競合他社の事業の活動の可能な変化と組み合わされた製品の発売の結果などの要因の組み合わせに関連する可能性があります。目標は、より極端な結果の結果を分析して、投資戦略を決定することです。

###個人および企業財務におけるシナリオ分析

[投資](/diversification)[シナリオ](/diversification)を調査するために使用されるのと同じプロセスを、理論的なシナリオに基づいて価値の変化を調査するために、他のさまざまな財務状況に適用できます。消費者側では、シナリオ分析を使用して、現金購入のために資金を節約するのではなく、クレジットでアイテムを購入することのさまざまな経済的結果を調べることができます。さらに、人は新しい求人を受け入れるかどうかを決定するときに発生する可能性のあるさまざまな経済的変化を見ることができます。

企業はシナリオ分析を使用して、ビジネスを運営できる2つの施設または店舗のいずれかを選択するなど、特定の決定の潜在的な財務結果を分析できます。これには、家賃、光熱費、保険の違い、または一方の場所には存在するが他方には存在しない可能性のある利益などの考慮事項が含まれる場合があります。

##シナリオ分析に関するFAQ

###シナリオ分析の利点は何ですか？

シナリオ分析の最大の利点は、考えられるすべての結果の詳細な調査として機能することです。このため、マネージャーは意思決定をテストし、特定の変数の潜在的な影響を理解し、潜在的なリスクを特定することができます。

###シナリオ分析の欠点は何ですか？

シナリオ分析の主な欠点は単純です。誤った仮定は、モデルがマークから大きく外れる、つまり「ガベージイン、ガベージアウト」につながる可能性があります。

シナリオ分析もユーザーのバイアスの影響を受けやすく、履歴データに大きく依存する傾向があります。

###リスク管理におけるシナリオ分析とは何ですか？

先に述べたように、シナリオ分析には、マイナス面を含む、考えられるさまざまな結果の徹底的な調査が含まれます。これにより、リスク管理者はリスクエクスポージャーを特定、準備、および管理できます。

###戦略的管理におけるシナリオ分析とは何ですか？

シナリオ分析は、ほとんどすべての経営上の決定、特に競争戦略に関連する決定に適用できます。言い換えると、シナリオ分析により、マネージャーは戦略的提案をテストできます。たとえば、小規模な競合他社を買収するかどうかなど、さまざまな条件下でどのようになるかを把握できます。

###シナリオ分析と感度分析の違いは何ですか？

シナリオ分析では、考えられるさまざまな結果を調べますが、すべての変数を同時に操作した場合の影響を分析します。結果は通常、ベースケースのシナリオ、ベストケースのシナリオ、およびワーストケースのシナリオになります。

一方、感度分析では、一度に**1つ**の変数を変更した場合の影響を評価します。

##結論

シナリオ分析は、いくつかの主要な変数を操作した後、ポートフォリオの期待値を推定するプロセスです。この方法は、投資戦略と企業財務の両方で使用できます。

投資家やマネージャーが利用するのに最適なツールですが、シナリオ分析は、ユーザーが行った仮定や入力と同じくらい優れています。
