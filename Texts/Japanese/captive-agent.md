# キャプティブエージェント
##キャプティブエージェントとは何ですか？

キャプティブエージェントは、1つの保険会社でのみ働く保険エージェントです。キャプティブエージェントは、通常、給与と[手数料](/commission)、および特典の組み合わせで、その1つの会社によって支払われます。彼らはフルタイムの従業員または[独立した請負業者である可能性があります](/independent-contractor)。

##キャプティブエージェントを理解する

[保険](/insurance)代理店は、キャプティブエージェントまたは独立エージェントのいずれかを選択できます。キャプティブエージェントは1つの保険会社でのみ機能しますが、[独立エージェント](/independent-agent)は特定の保険会社では機能せず、一連の企業からポリシーを販売できます。

キャプティブエージェントは1つの会社でしか働かないため、特定の会社の保険商品について深い知識を持っていますが、その会社の商品を必要としない、または資格を持たないクライアントを支援することはできません。

多くの場合、親会社は、キャプティブエージェントに特定のポリシーを販売したり、特定の販売割り当てを達成したりするようにプッシュする場合がありますが、多くの場合、顧客にとって最良の製品ではありません。エージェントはそのクライアントのニーズに最適なポリシーを探すことができるため、通常、独立したエージェントの方がクライアントに適しています。唯一の欠点は、独立したエージェントが特定の会社の製品に関する専門知識を持っていない可能性があることです。

キャプティブエージェントは、クライアントに並外れたレベルのサービスを提供することに優れていることがよくあります。これは、関係構築、事実調査、および顧客サービスにより多くの時間を費やす自由があるためです。ますますデジタル化されている保険市場では、これらはクライアントがますます得ていないタイプのものです。

##キャプティブエージェントの長所と短所

独立したエージェントの取り決めは、より多様な収入源を提供するため、エージェントにとってより良い場合がありますが、エージェントが独自の[資金を提供し](/startup-capital)、事業費を支払い、利益を手配する[必要がある場合があるため、リスクも高くなります。](/startup-capital)独立した代理人になるには、自分で事業を始める必要があります。これには、店舗の設立のすべての側面と、協力する保険会社を見つける必要があります。

一方、キャプティブエージェントはこれらの側面を処理し、作業を開始するために多額の[資本](/capital)を投入する必要はありません。したがって、保険代理店がキャプティブ代理店として働くことを好む主な理由は、経済的であり、ビジネスを行うことの容易さです。彼らの会社は通常、重要な全国[広告予算](/advertising-budget)は言うまでもなく、事務処理、継続的なトレーニング、重要なボーナス、およびその他の動機付けプログラムを処理するためのオフィス、管理スタッフを提供します。

キャプティブエージェントは通常、保険会社から見込み客の広範なリストを受け取ります。これは、消費者が広告に応答すると、会社が彼らをその地域のキャプティブ販売エージェントに誘導するためです。

キャプティブエージェントは通常、給与と手数料を支払われ、特典を受け取ります。独立した請負業者は通常、手数料を支払われるだけであり、彼ら自身に利益を提供しなければなりません。

キャプティブエージェントになることのいくつかの欠点は、面倒な契約に縛られ、あなたが働いている保険会社に義務を負い、ビジネスを行う方法に手を縛ることが多いことです。

一部の保険会社は、市場に出回っている競合製品と比較して劣っていても、製品の販売に割り当てを課しています。キャプティブエージェントとして、クライアントに最適なものを販売することはできませんが、保険会社が提供するものを販売することはできません。これは、クライアントが他の場所で受け取ることができる価格よりも高い場合もあります。キャプティブエージェントとしてのあなたの目標は、クライアントに最適なものを提供するのではなく、会社のビジネスを拡大することです。

##ハイライト

-キャプティブエージェントには通常、給与と手数料が支払われ、手当が支給されます。

-キャプティブエージェントは、1つの保険会社でのみ機能する保険エージェントです。

-キャプティブエージェントは、勤務している保険会社の製品のみを販売し、その提供以外の個人を支援することはできません。

-再び捕虜になることの不利な点には、面倒な契約、特定の製品のみの販売、およびクライアントの最善の利益にはならない可能性のある製品が含まれます。

-キャプティブエージェントの反対は、多くの保険会社で働く独立したエージェントです。

-キャプティブエージェントになることの利点には、管理タスク、全国的な広告予算、クライアントリストなどの会社で働くことの利点が含まれます。
