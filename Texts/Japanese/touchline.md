# タッチライン
##タッチラインとは何ですか？

タッチラインは、特定の[証券の購入者が](/security)入札する意思のある最高価格、または販売者が取引日の任意の時点で提供（または販売）する意思のある最低価格です。

したがって、タッチラインは証券の[ビッドアスクスプレッドによって表され](/bid-askspread)ます。ビッドとアスクが動くと、タッチラインも動きます。

##タッチラインを理解する

タッチラインは、任意の時点で最良の入札または特定のセキュリティを要求することを指定します。[流動性の高い証券は一般的に](/liquidasset)ビッドアスクのスプレッドが狭く、[流動性](/illiquid)の低い証券はスプレッドが広くなります。たとえば、EUR / USD通貨ペアは流動性が高いと見なされているため、ビッドアスクスプレッドは小さくなります[。](/eur-usd-euro-us-dollar-currency-pair)

証券に5ドル、5.10ドル、5.15ドルで入札しているさまざまなバイヤーがいる場合、タッチラインの入札価格は5.15ドルになります。これは、誰かが入札を投稿することをいとわない最高価格です。

同時に、証券に5.20ドル、5.30ドル、5.35ドルのさまざまな売り手がいる場合、タッチラインの売値は5.20ドルになります。これは、誰かがオファーを投稿することをいとわない最低価格です。この証券のビッドアスクスプレッドは、ビッドが$ 5.15、アスクが$5.20です。スプレッド、つまりビッドとアスクの差は$0.05です。

##ビッドアスクスプレッド

タッチラインは、特定の資産の最良のビッドおよび/またはアスク価格を表します。これは、ビッドアスクスプレッドに関連します。これは、アスク価格が市場の[資産のビッド価格を超える金額](/asset)です。ビッドアスクスプレッドは、基本的に、買い手が資産に対して支払う意思のある最高価格と、[売り手が資産](/seller)を売却するために受け入れる意思のある最低価格との差です。

ビッドアスクスプレッドは特定の資産の[需要と供給](/demand)を反映しており、ビッドは需要を表し、アスクは[供給を表し](/supply)ます。ビッドとアスクの深さはビッドアスクスプレッドに大きな影響を与える可能性があり、一方が他方を上回っている場合、または両方が堅牢でない場合、それを大幅に広げます。

[マーケットメーカー](/marketmaker)とトレーダーは、ビッドアスクスプレッドとビッドの深さを利用して、スプレッドの差を相殺するように求めます。これは、トレーダーやマーケットメーカーが利益をもたらす主な方法の1つです。たとえば、マーケットメーカーは5.10ドルで入札し、5.15ドルでオファーする場合があります。誰かがマーケットメーカーに売った場合、マーケットメーカーは5.10ドルで購入したことになります。誰かがマーケットメーカーから5.15ドルで購入した場合、マーケットメーカーは取引された1株あたり0.05ドルを稼ぎます。

ある資産から別の資産へのビッドアスクスプレッドのサイズは、主に各資産の[流動性の違いのために異なります。](/liquidity)特定の市場は他の市場より流動性があります。一部の流動性の低い資産には、入札価格の最大1％以上のスプレッドがある場合があります。

入札価格は、誰かがその特定の瞬間に投稿しようとしている最高の入札であり、質問は、誰かがその特定の瞬間に提供しようとしている最低の投稿価格です。この画像の場合、入札タッチラインは$ 40.12で、オファータッチラインは$40.13です。

ビッドとアスクの価格は静的ではありません。誰かが入札に直接売るか、オファーから直接購入する可能性があります。これにより、その価格で投稿された株式が削除されます。

この例では、1,000（10 x 100株、または10 x [Round Lot](/roundlot) ）株が$40.12で入札されています。誰かが1,000株を入札価格で売った場合、入札価格は40.12ドルではなく、次に高い入札になります。流動性のある株式では、次の入札またはオファーは通常、現在の入札より1セント低いか、現在のオファーより1セント高くなります。この場合、次の入札は通常の状況では40.11ドルになります。

##ハイライト

-ビッドとアスクの価格は静的ではありません。誰かがビッドに直接販売したり、オファーから直接購入したりする可能性があります。これにより、その価格で投稿された株式が削除されます。

-ビッド価格は、その特定の瞬間に誰かが喜んで投稿する最高の入札であり、アスクは、その特定の瞬間に誰かが喜んで提供する最低の投稿価格です。

-タッチラインは、証券の最もタイトなビッドアスクスプレッドによって表されるように、最高のビッドと最低のオファーです。
