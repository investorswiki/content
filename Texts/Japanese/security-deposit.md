# 保証金
##保証金とは何ですか？

敷金とは、家やアパートの家主、貸し手、売り手に、住居に引っ越して世話をする意図があることを証明するものとして与えられるお金です。保証金は、取引条件に応じて、返金または返金不可のいずれかになります。敷金は、受取人の安全を確保するためのものであり、損害賠償や遺失物の支払いにも使用できます。

敷金は、無形のセキュリティ手段として、または損害や財産の損失が発生した場合の有形のセキュリティの手段として機能します。

[エスクロー口座](/escrow)など、敷金がどこに保管されているか、および利息を徴収する必要があるかどうかについて、さまざまな法律があります。

>>

##保証金の仕組み

敷金は、入居前または物件を所有する前に支払われます。これらの敷金は通常、月額家賃と同じ金額です。賃貸人の行動により損害が発生した場合、賃貸ユニット内の電化製品の修理または交換に保証金が使用される場合があります。

たとえば、借家人が窓を壊したり、物件の床、壁、インフラに恒久的な損傷を与えたりした場合、家主は敷金を修理に使用できます。通常、物件の状態が良好で、賃貸人が退去する際に修理の必要がない場合、敷金は返金される場合があります。

##保証金の要件

敷金の額は通常1か月分の家賃ですが、それより高くなることもあります。物件の賃貸料が上がると、エスクローに預けられている敷金が足りなくなる場合があります。

敷金は、敷金の保有中に利息が発生する可能性がありますが、家賃の値上げ率がその利息を超える可能性があります。その後、賃借人は、保持されている敷金にさらにお金を追加する必要があります。

敷金は[課税所得とは見なされず](/taxableincome)、現地の法律では敷金を[信託基金として扱うことがよくあり](/trust-fund)ます。最終家賃の支払いとして使用される敷金は、前払いとして請求する必要があり、支払い時に課税されます。

##特別な考慮事項

一部の州では、家主は、他の方法では敷金を支払うことも、借家人によって引き起こされた損害を修復するために敷金を使用することもできない借家人からの家賃として敷金を適用する場合があります。各州は、物件の占有が終了したときに、敷金を使用して最終月の家賃を支払うことができるかどうかを規定する場合があります。地域の法律によっては、最終月の家賃と敷金が同じでない場合があり、別々に会計処理する必要があります。家主は、敷金を最終的な家賃として使用するために、賃貸人からの書面による承認を必要とする場合もあります。

特定の都市や近隣地域では、敷金に必要な金額に問題が生じる可能性があります。一部の地区には、周辺地域と比較して高い敷金率を請求する家主がいる可能性があります。これは、低所得の個人や家族がそれらの地域に住む場所を見つけることを強制する効果があります。敷金が不動産に請求される家賃に関連していくらになるかについて制限を設ける地方の法律が制定されるかもしれません。

##ハイライト

-敷金は、賃貸人によって損傷、紛失、または盗難された賃貸ユニット内の何かを修理または交換する手段として機能します。

-保証金は通常、物件が「合理的に」良好な状態で、通常の減価償却の時点まで残された場合、出発時に返金されます）

-通常、敷金は入居前に支払う必要があり、州法により、必要に応じて敷金をどのように適用するかが定められています。
