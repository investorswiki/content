# バリューネットワーク
##バリューネットワークとは何ですか？

バリューネットワークは、グループ全体に利益をもたらすために相互作用する組織や個人間の一連の接続です。バリューネットワークにより、会員は商品を売買したり、情報を共有したりすることができます。これらの[ネットワーク](/network-effect)は、ノード（メンバー）とコネクタ（関係）を表示するシンプルなマッピングツールで視覚化できます[。](/network-effect)

##バリューネットワークを理解する

ビジネスと商取引では、バリューネットワークは経済エコシステムの一例です。各メンバーは、成長を促進し、価値を高めるために互いに依存しています。バリューネットワークのメンバーは、外部メンバー（顧客など）または[研究開発](/randd)チームなどの内部メンバーで構成できます。

価値ネットワークは、イノベーション、社会福祉、環境、およびその他の多くの分野を強化します。 1つのノードの弱点は、ネットワーク全体に影響を与える可能性があります。たとえば、開発チームが弱い場合、生産チームは製品を作成するのに苦労し、購入者が出荷を待たされる可能性があります。

##バリューネットワークの種類

バリューネットワークの主なタイプには、Clayton Christensenネットワーク、Fjeldstad、Stabellsネットワーク、NormannとRamirezのコンステレーション、VernaAlleeのネットワークが含まれます。

###クレイトンクリステンセンネットワーク

クレイトンクリステンセンネットワークは、すでに外部に存在し、ネットワークへの新規参入者は現在のネットワークまたはビジネスモデルの形に合うように形成されるという関係を説明しています。新規参入者は、現在のネットワークに順応し、一致する可能性が高いため、新規参入者は、突破したり、新しいアイデアを提供したり、変更を実装したりするのに苦労します。

### Fjeldstad and Stabells Network

FjeldstadとStabellsは、ネットワークの最も重要な部分は、（1）顧客、（2）サービス、（3）サービスプロバイダー、および（4）サービスへのアクセスを許可する契約であると考えています。この理論は、顧客はネットワークに不可欠であり、顧客の関与が付加価値を提供すると述べています。最も一般的な例は、Facebook、YouTube、Instagram、TikTokなどのソーシャルメディアです。これらのメディアでは、顧客がサインアップし、契約条件に同意して、ネットワークに価値を付加します。

###ノルマンとラミレスの星座

NormannとRamirezのコンステレーションバリューネットワークは、ネットワークが絶え間ない変化と改善を可能にする流動的なセットアップであると信じています。現在の関係を分析し、価値を付加する方法として開口部と機会を探すのは、ネットワークのメンバー次第です。

###VernaAlleeのネットワーク

Verna Alleeのネットワークは、ネットワークが有形と無形の両方の価値を生み出すと信じており、[価値ネットワーク分析](/value-network-analysis)をビジネスのあらゆる側面に組み込んで、あらゆる段階で最大の価値を引き出す必要があります。

##バリューネットワークのメリット

バリューネットワークが提供するメリットは、企業または個人が、接続している他の人のリソース、影響力、および洞察を適用する方法からもたらされます。たとえば、新興企業は、投資家やメンターなどの外部とのつながりに目を向けて、ビジネスの発展と成長に取り組む方法について経験豊富なガイダンスを提供する場合があります。

多くの創設者は彼らが開発する製品やサービスを深く理解していますが、そのサービスを市場に出し、顧客を見つけ、ビジネスを拡大することは彼らにはなじみがないかもしれません。

この欠点を補うために、彼らはそのような問題についての経験を持つ信頼できる[利害関係者のアドバイスを求めるかもしれません](/stakeholder)。それは彼らの関係の無形の利益と考えられています。彼らはまた、潜在的なメンターや投資家への露出を増やすために、インキュベーターやアクセラレーターなどの[スタートアップ](/startup)を支援することを専門とするグループに目を向けるかもしれません。

##バリューネットワークの例

投資家は通常、支援しているスタートアップにガイダンスを提供します。リーダーがアイデアを具体的な企業に成長させるのを支援することで、利害関係者はスタートアップの発展から利益を得ることができるからです。そのガイダンスは、投資家が所有する専門知識の形をとることができます。

投資家は、スタートアップの創設者と彼らが計画を進めるために協力できる他のビジネスとの間の紹介を促進するかもしれません。たとえば、会社が自社製品のプロトタイプを作成する必要がある場合、投資家はそれらをオーダーメイドのプロトタイプを作成する別の会社に誘導できる可能性があります。同様に、スタートアップが量販店や流通業者を探している場合、彼らが受けるガイダンスは、各組織や個人のビジネスの増加を意味する可能性があるため、関係者全員に利益をもたらす可能性があります。

##ハイライト

-バリューネットワークは、ノード（メンバー）とコネクタ（関係）を介してマッピングツールで表現できます。

-バリューネットワークのメンバーは、相互に売買したり、重要で関連性のある情報を交換したりできます。

-バリューネットワークの主な利点には、企業または個人がリソースを適用する方法、影響力、およびネットワーク接続の洞察が含まれます。

-バリューネットワークは、個人または個人と企業の間のつながりであり、その相互作用がグループに利益をもたらします。

-バリューネットワークは、メンバーが価値を高めるのに役立ち、内部（研究開発など）と外部（顧客など）のリソースで構成されます。
