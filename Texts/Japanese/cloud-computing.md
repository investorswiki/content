# クラウドコンピューティング
##クラウドコンピューティングとは何ですか？

クラウドコンピューティングは、インターネットを介したさまざまなサービスの提供です。これらのリソースには、データストレージ、サーバー、データベース、ネットワーク、ソフトウェアなどのツールやアプリケーションが含まれます。

[クラウドベースのストレージでは](/cloud-storage)、ファイルを専用のハードドライブやローカルストレージデバイスに保存するのではなく、リモートデータベースに保存できます。電子デバイスがWebにアクセスできる限り、それを実行するためのデータとソフトウェアプログラムにアクセスできます。

クラウドコンピューティングは、コスト削減、生産性の向上、速度と効率、パフォーマンス、セキュリティなど、さまざまな理由から、人々や企業に人気のあるオプションです。

##クラウドコンピューティングを理解する

クラウドコンピューティングは、アクセスされる情報がクラウドまたは仮想空間でリモートに検出されるため、そのように名付けられています。クラウドサービスを提供する企業は、ユーザーがファイルやアプリケーションをリモートサーバーに保存し、インターネット経由ですべてのデータにアクセスできるようにします。これは、ユーザーがアクセスするために特定の場所にいる必要がないことを意味し、ユーザーがリモートで作業できるようにします。

クラウドコンピューティングは、持ち運びや座って作業するデバイスから、データの処理と処理に伴うすべての面倒な作業を取り除きます。また、そのすべての作業をサイバースペースのはるか遠くにある巨大なコンピュータークラスターに移動します。インターネットはクラウドになり、データ、仕事、アプリケーションは、世界中のどこからでもインターネットに接続できるあらゆるデバイスから利用できます。

クラウドコンピューティングは、パブリックとプライベートの両方にすることができます。パブリッククラウドサービスは、インターネットを介して有料でサービスを提供します。一方、プライベートクラウドサービスは、特定の数の人々にのみサービスを提供します。これらのサービスは、ホステッドサービスを提供するネットワークのシステムです。パブリックサービスとプライベートサービスの両方の要素を組み合わせたハイブリッドオプションもあります。

##クラウドサービスの種類

サービスの種類に関係なく、クラウドコンピューティングサービスはユーザーに次のような一連の機能を提供します。

- Eメール

-ストレージ、バックアップ、およびデータの取得

-アプリの作成とテスト

-データの分析

-オーディオおよびビデオストリーミング

-オンデマンドでソフトウェアを提供

クラウドコンピューティングはまだかなり新しいサービスですが、大企業から中小企業、非営利団体から政府機関、さらには個人の消費者まで、さまざまな組織で使用されています。

##展開モデル

雲にはさまざまな種類があり、それぞれが異なります。パブリッククラウドは、インターネット上のサーバーとストレージでサービスを提供します。これらは、すべてのハードウェア、ソフトウェア、および一般的なインフラストラクチャを処理および制御するサードパーティ企業によって運営されています。クライアントは、ほぼ誰でもアクセスできるアカウントを介してサービスにアクセスします。

プライベートクラウドは、特定の顧客、通常は1つの企業または組織のために予約されています。同社のデータサービスセンターは、クラウドコンピューティングサービスをホストする場合があります。多くのプライベートクラウドコンピューティングサービスは、プライベートネットワーク上で提供されます。

ハイブリッドクラウドは、その名前が示すように、パブリックサービスとプライベートサービスの両方を組み合わせたものです。このタイプのモデルは、ユーザーの柔軟性を高め、ユーザーのインフラストラクチャとセキュリティを最適化するのに役立ちます。

>新しい形式のクラウドコンピューティングサービスには、コミュニティクラウド、ビッグデータクラウド、マルチクラウドが含まれます。

##クラウドコンピューティングの種類

クラウドコンピューティングは、マイクロチップや携帯電話のような単一のテクノロジーではありません。むしろ、これは主に3つのサービスで構成されるシステムです。サービス[としてのソフトウェア](/software-as-a-service-saas)（SaaS）、サービスとしてのインフラストラクチャ（IaaS）、およびサービスとしてのプラットフォーム（PaaS）です。

1. ** Software-as-a-Service（SaaS）**には、顧客へのソフトウェアアプリケーションのライセンスが含まれます。ライセンスは通常、従量制モデルまたはオンデマンドで提供されます。このタイプのシステムは、MicrosoftOfficeの365にあります。

1。**Infrastructure-as-a-Service（IaaS）**には、オンデマンドサービスの一部として、IPベースの接続を介してオペレーティングシステムからサーバーおよびストレージまですべてを配信する方法が含まれます。クライアントは、ソフトウェアやサーバーを購入する必要をなくし、代わりに、[アウトソーシング](/outsourcing)されたオンデマンドサービスでこれらのリソースを調達できます。 IaaSシステムの一般的な例には、IBMCloudとMicrosoftAzureが含まれます。

1。**Platform-as-a-service（PaaS）**は、クラウドベースのコンピューティングの3つのレイヤーの中で最も複雑であると考えられています。 PaaSはSaaSといくつかの類似点を共有しています。主な違いは、ソフトウェアをオンラインで配信するのではなく、実際にはインターネット経由で配信されるソフトウェアを作成するためのプラットフォームであるということです。このモデルには、Salesforce.comやHerokuなどのプラットフォームが含まれます。

##クラウドコンピューティングの利点

クラウドベースのソフトウェアは、ネイティブアプリまたはブラウザーを介して任意のデバイスからソフトウェアを使用できるなど、あらゆるセクターの企業に多くのメリットをもたらします。その結果、ユーザーはファイルと設定を完全にシームレスな方法で他のデバイスに引き継ぐことができます。

クラウドコンピューティングは、複数のデバイス上のファイルにアクセスするだけではありません。クラウドコンピューティングサービスのおかげで、ユーザーは任意のコンピューターで自分のメールをチェックしたり、DropboxやGoogleドライブなどのサービスを使用してファイルを保存したりすることができます。クラウドコンピューティングサービスにより、ユーザーは音楽、ファイル、写真を[バックアップできるようになり、ハードドライブがクラッシュした場合にこれらのファイルをすぐに利用できるようになります。](/back-up)

また、大企業に大きなコスト削減の可能性を提供します。クラウドが実行可能な代替手段になる前は、企業[は](/information-management-technology-imt)[フォーメーション管理テクノロジー](/information-management-technology-imt)とインフラストラクチャでコストのかかる購入、構築、および保守を行う必要がありました。企業は、コストのかかるサーバーセンターとIT部門を高速インターネット接続に交換できます。このインターネット接続では、従業員がオンラインでクラウドとやり取りしてタスクを完了します。

クラウド構造により、個人はデスクトップまたはラップトップのストレージスペースを節約できます。また、ソフトウェア会社はディスクやフラッシュドライブを含む従来の具体的な方法ではなく、Webを介して製品を提供できるため、ユーザーはソフトウェアをより迅速に[アップグレードできます。](/upgrade)たとえば、アドビのお客様は、インターネットベースのサブスクリプションを通じてCreativeCloudのアプリケーションにアクセスできます。これにより、ユーザーはプログラムの新しいバージョンや修正を簡単にダウンロードできます。

##クラウドのデメリット

クラウドコンピューティングに伴うスピード、効率、革新のすべてに、当然のことながらリスクがあります。

特に機密性の高い医療記録や財務情報に関しては、セキュリティは常にクラウドにとって大きな懸念事項でした。規制により、クラウドコンピューティングサービスはセキュリティとコンプライアンスの対策を強化する必要がありますが、それは継続的な問題です。暗号化は重要な情報を保護しますが、その暗号化キーが失われると、データは消えます。

クラウドコンピューティング企業によって保守されているサーバーも、自然災害、内部バグ、および停電の犠牲になる可能性があります。クラウドコンピューティングの地理的な範囲は、両方の方法を削減します。カリフォルニアの停電はニューヨークのユーザーを麻痺させる可能性があり、テキサスの企業は、何かがメインベースのプロバイダーをクラッシュさせるとデータを失う可能性があります。

他のテクノロジーと同様に、従業員とマネージャーの両方に学習曲線があります。しかし、多くの個人が単一のポータルを介して情報にアクセスして操作しているため、不注意によるミスがシステム全体に伝わる可能性があります。

##ビジネスの世界

企業はさまざまな方法でクラウドコンピューティングを採用できます。一部のユーザーはすべてのアプリとデータをクラウド上に維持しますが、他のユーザーはハイブリッドモデルを使用して、特定のアプリとデータをプライベートサーバーに保持し、他のユーザーはクラウド上に保持します。

サービスの提供に関しては、企業のコンピューティング分野の大手企業には次のものがあります。

-Google Cloud

-アマゾンウェブサービス（AWS）

-Microsoft Azure

-IBM Cloud

-Alibaba Cloud

アマゾンウェブサービスは100％公開されており、従量制のアウトソーシングモデルが含まれています。プラットフォームにアクセスしたら、アプリや追加サービスにサインアップできます。 Microsoft Azureを使用すると、クライアントは自分のサイトに一部のデータを保持できます。一方、AlibabaCloudはAlibabaGroupの子会社です。

###クラウドコンピューティングの例は何ですか？

今日、企業と個人の両方で使用されているクラウドコンピューティングアプリケーションの例がいくつかあります。クラウドサービスの1つのタイプは、実際のメディアファイルがリモートに保存されるオーディオまたはビデオのストリーミングプラットフォームです。もう1つは、Googleドライブ、Dropbox、OneDrive、Boxなどのデータストレージプラットフォームです。

###クラウドコンピューティングは安全ですか？

ソフトウェアとデータはクラウドコンピューティングにリモートで保存されるため、データのセキュリティとプラットフォームのセキュリティは大きな懸念事項です。[クラウドセキュリティ](/cloud-security)とは、クラウドベースのサービスに保存されているデジタル資産とデータを保護するために実施される対策を指します。このデータを保護するための対策には、2要素認証（2FA）、VPN、セキュリティトークン、データ暗号化、ファイアウォールサービスなどが含まれます。

###クラウドコンピューティングの主な種類は何ですか？

クラウドコンピューティングサービスの主なタイプには、Infrastructure-as-a-Service（IaaS）、Platforms-as-a-Service（PaaS）、およびSoftware-as-a-Service（SaaS）が含まれます。-IaaSは、ITインフラストラクチャをエンドに提供します-ユーザーはインターネット経由で、一般的にサーバーレスコンピューティングに関連付けられています。-PaaSは、ソフトウェアとハードウェアの両方を、一般的にソフトウェア開発者であるエンドユーザーに提供します。 PaaSを使用すると、ユーザーはインフラストラクチャを構築および保守することなく、独自のアプリを開発、実行、および管理できます。-SaaSはソフトウェアライセンスモデルであり、外部サーバーをダウンロードしてインストールしなくても、サブスクリプションベースでソフトウェアにアクセスできます。ローカルで。
