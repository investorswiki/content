# 利益
##利益とは何ですか？

利益とは、事業活動から生み出される[収益](/revenue)が、問題の活動の維持に伴う費用、費用、および税金を超えた場合に実現される経済的利益を表します。

獲得した利益はすべて事業主に還元され、事業主は現金をポケットに入れるか、[配当として株主に分配する](/dividend)か、事業に再投資するかを選択します。

##利益はあなたに何を伝えますか？

利益は、すべての費用を考慮した後に企業が引き込むお金です。レモネードスタンドであろうと公営の多国籍企業であろうと、あらゆるビジネスの主な目標はお金を稼ぐことです。したがって、ビジネスのパフォーマンスはさまざまな形で収益性に基づいています。

トップラインの収益性に関心のあるアナリストもいれば、税金やその他の費用を差し引く前の収益性に関心のあるアナリストもいます。さらに、すべての費用が支払われた後にのみ収益性に関心を持つ人もいます。

利益の3つの主要なタイプは、粗利益、営業利益、および純利益です。これらはすべて、損益計算書に記載されています。各利益タイプは、特に他の競合他社や期間と比較した場合に、アナリストに会社の業績に関するより多くの情報を提供します。

>「利益」という言葉は、ラテン語の名詞** profectus **は「進歩」を意味し、動詞**proficere**は「前進する」を意味します。

>>

##粗利益、営業利益、純利益

収益性の最初のレベルは[粗利益](/grossprofit)です。これは、売上から販売した商品のコストを差し引いたものです。売上高は損益計算書の最初の項目であり、販売された商品のコスト（COGS）は通常そのすぐ下に表示されます。

>>

>粗利益=収益-売上原価

>>

たとえば、会社Aの売上高が$ 100,000で、COGSが$ 60,000の場合、総利益は$ 40,000、つまり$100,000から$60,000を引いたものになります。粗利益を売上で割って、[粗利益マージン](/gross_profit_margin)を40％、つまり$40,000を$100,000で割った値です。

[営業利益](/operating_profit)は、諸経費やその他の間接費などの営業費用と、[減価償却費などの会計費用を取り除き](/depreciation)ます。金利税引前利益、または[EBITと呼ばれることもあり](/ebit)ます。

>>

>営業利益=収益-売上原価（COGS）-営業費用-減価償却費

>>

純利益はさらに、事業によって支払われる利息と税金のコストを取り除きます。損益計算書の一番下にあるため、会社の「ボトムライン」と呼ばれることも[あり](/bottomline)[ます](/bottomline)。

>>

>純利益=EBIT-支払利息-税金

>>

収益は、ある期間の収益性と、配当および利益剰余金に利用できる金額を企業に示します。保持されたものは、負債の返済、プロジェクトへの資金提供、または会社への再投資に使用できます。

##ハイライト

-会計上の目的で、企業は粗利益、営業利益、および純利益（「収益」）を報告します。

-利益は、総収入から総費用を差し引いたものとして計算されます。

-収益性の高い企業は、利益が配当として株主に還元されるか、企業に再投資されて株式価値が高まるため、投資家にとって魅力的です。

＃＃ よくある質問

###会社の収益は何を教えてくれますか？

収益は、ある期間の収益性と、配当および留保利益に利用できる金額を企業に示します。保持されたものは、負債の返済、プロジェクトへの資金提供、または会社への再投資に使用できます。収益の増加は企業が成長していることを示していますが、収益の減少は危険信号である可能性があります。

###利益はどこから来るのですか？

企業が商品を売るために互いに競争する資本主義システムでは、利益がどこから来るのかという問題は、エコノミストの間で関心のあるものの1つでした。たとえば、[カール・マルクスは、](/karl-marx)事業主が労働者から引き出した[余剰労働力から利益が生じると主張した。](/marxian-economics)現代の思想家は、事業を始めるときに起業家が負うリスクを利益が補うことを示唆しています。他の人は、利益は[非効率的な市場](/inefficientmarket)と[不完全競争](/imperfect_competition)から生じると主張します。

###利益に対する法人税率はいくらですか？

米国では、利益に対する法人税率は現在21％です（2017年の減税および雇用法以降の35％から引き下げられました）。
