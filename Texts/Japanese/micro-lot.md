# マイクロロット
##マイクロロットとは何ですか？

マイクロロットは、[外国為替取引における](/forex)[基本通貨の1,000単位を表します](/basecurrency)。基本通貨は、[ペアの最初の通貨、](/currencypair)または購入または販売する通貨です。マイクロロットでの取引により、小売業者は比較的小さな増分で取引することができます。

外国為替トレーダーは、ミニロットとスタンダードロットで取引すること[も](/mini-lot)でき[ます](/standard-lot)。

##マイクロロットを理解する

投資家がマイクロロットを注文する場合、これは、売買される通貨の1,000単位を注文したことを意味します。たとえば、 [EUR / USD](/eur-usd-euro-us-dollar-currency-pair) （ユーロ対米ドル）の通貨ペアでは、ユーロが基本通貨であり、トレーダーは1,000ユーロを売買します。

マイクロロットは通常、外国為替トレーダーが取引できる通貨の最小ブロックであり、取引を開始しようとしているが潜在的なマイナス面を減らしたい初心者トレーダーによって使用されます。比較的まれですが、一部の外国為替ブローカーは、基本通貨の100単位であるナノロットを提供しています。

投資家は、ミニロットまたは標準ロットを取引したくない場合にマイクロロットを使用します。 10個のマイクロロットは1つのミニロット（10,000ユニット）に相当し、10個のミニロットは1つの標準ロット（基本通貨の100,000ユニット）に相当します。

マイクロロットでの取引は、トレーダーを制限する必要はありません。彼らは好きなだけ小さくまたは大きく取引することができます。 1マイクロロットを取引することも、1,000マイクロロットを取引することもできます。これは、1,000,000ユニット（10標準ロット）の通貨に相当します。マイクロロットでは、125マイクロロットなどの位置サイズの微調整が可能です。これは12.5ミニロットに相当します。トレーダーがミニロットしか取引できない場合は、12または13のミニロットを選択する必要がありますが、これは125マイクロロットほど微調整されていません。

ほとんどの小売仲介口座では、トレーダーは100ドルや500ドルなどの比較的少量の開始預金でマイクロロットを取引できます。

>ナノロットはさらに小さく、マイクロロットの10分の1のサイズです。ナノロットを取引する場合、米ドルベースの通貨ペアの1ピップはわずか0.01ドルに相当します。

>>

##ロットサイズの違い

ユニットサイズが小さいため、トレーダーはリスクをより適切に管理できます。たとえば、標準ロットでEUR / USDを1[ピップ](/pip)移動すると、トレーダーは10ドルの利益または損失になります。トレーダーのアカウントに500ドルしかない場合（200：1の[レバレッジが必要](/leverage)）、5ピップの動き（数秒で発生する可能性があります）は、アカウントの10％を失っていることを意味します。

ミニロット（20：1のレバレッジが必要）では、EUR / USDで1ピップ移動するごとに、1ドルの利益または損失が発生します。アカウントの10％を失うには、アカウントが50ピップス移動する必要があります。最後に、マイクロロット（2：1のレバレッジが必要）では、EUR/USDの動きの各ピップは$0.10の価値があります。トレーダーが取引でアカウントの10％を失うには、価格は500ピップス移動する必要があります。

これらの例は、マイクロロットのユニットサイズが小さいことは、取引の柔軟性が高く、レバレッジが減少する可能性があるため、アカウントが小さいトレーダーにとって非常に有益であることを示しています[。アカウントにあります](/margincall)。

500ドルのアカウントでは、1,000ユニットのマイクロロットを売買するのに約2：1のレバレッジしかかかりません。 500ドルのアカウントで標準ロットを購入すると、約200：1のレバレッジが発生し、わずか50ピップの動きでアカウント全体が一掃される可能性があります。 Forexのレバレッジは、米国および世界中の多くの国で50：1に制限されています。

##マイクロロットを使用した理想的な位置のサイジング

外国人トレーダーは、小さなアカウントでリスクを微調整するために、ポジションサイズを小さく保つためにマイクロロットを使用することがよくあります。

トレーダーが1.2250でGBP/USDを購入し、 1.2200で[ストップ](/gbp-usd-british-pound-us-dollar-currency-pair)[ロス](/stop-lossorder)を設定したいとします。彼らは50ピップの危険を冒しています。彼らは1,000ドルのアカウントを持っており、その2％、つまり20ドルのリスクを冒すことをいとわない。

マイクロロットで理想的な位置サイズを見つけるために、値を次の式に代入することができます：

>>

>リスクへのドル/（ピップのリスクxマイクロロットのピップ値）=マイクロロットの位置サイズ

>>

>>

> $ 20 /（$ 50 x $ 0.10）=4マイクロロット

>>

トレーダーが取引で20ドルのリスクを冒すことをいとわない、50ピップストップロスの理想的なポジションサイズは、4マイクロロットです。逆に言えば、トレーダーが4つのマイクロロットを購入し、1ピップの動きごとに$ 0.40（$ 0.10 x 4マイクロロット）の価値がある場合、トレーダーが4つのマイクロロットで50ピップを失うと、$20を失います。

ミニロットピップ値を入力してミニロット、または標準ロットピップ値を入力して標準ロットに調整できます。ピップスの値は、取引されている通貨ペアによって異なる場合があることに注意してください。

##ハイライト

-マイクロロットは、ミニロットまたは標準ロットよりも小さい位置および/または位置サイズのより大きな微調整を可能にします。

-その他のロットサイズには、ナノロット（100ユニット）、ミニロット（10,000ユニット）、および標準ロット（100,000ユニット）が含まれます。

-外国為替取引のマイクロロットは、通貨ペアの基本通貨の1,000単位です。
