# 大連商品取引所
## Dalian Commodities Exchangeとは何ですか？

大連商品取引所（DCE）は中国の大連にあり、さまざまな[商品の](/commodity)[先物契約を取引してい](/futurescontract)ます。この取引所は、世界最大の農業先物取引所の1つです。

##大連商品取引所を理解する

大連商品取引所は、鉄道や高速道路へのアクセスが可能なこの地域の戦略的な位置に一部起因して、国際的な農業ハブとしての北東中国の拡大の活性化の重要な要因となっています。

[取引所](/exchange)は、先物取引とオプション取引の場の提供、契約の作成と上場、取引の整理と監督、清算、決済など、いくつかの主要な機能を実行します。さらに、取引所は、市場の監視とルールの施行、[リスク管理](/riskmanagement)ルールの策定と実装、マーケティングと投資家の教育イベントの開催、市場データと情報サービスなどを処理します。

##大連交流の歴史

大連取引所は1993年2月28日に設立されました。中国の先物産業は60年後の1990年に復活し、その時点で大連取引所が設立されました。約200人の会員と16万人以上の投資家を擁する非営利の自主規制機関です。この取引所は、中国本土で栽培されている大豆の流通にとって重要な場所であるという事実もあり、中国で最も多くの商品取引所を持っています。 90年代を通じて、取引所は、市場における財務の完全性、リスク管理、機能性、および[透明性](/transparency)と流動性についての評判を獲得しました。

2013年、大連商品取引所は農産物取引所としての役割から、鉄鉱石やコークス炭などの産業を含むように拡大しました。現在、取引所には50万人近くの参加者がいます。 2015年、DCEは、[先物産業協会によって世界の主要なデリバティブ取引所の8位にランクされ](/fia)ました。また、石油、プラスチック、石炭、冶金コークス、鉄鉱石の最大の先物市場でもあります。

あまり知られていない取引商品の中で、交換取引は、線形低密度ポリエチレン、ポリプロピレン、パーム油先物、卵、ファイバーボード、大豆、大豆ミール、大豆油、遺伝子組み換え大豆、大豆ミールオプション先物、米、およびトウモロコシです。

2016年、先物産業協会（FIA）は、大連商品取引所が取引量で世界で8番目に大きい取引所であると報告しました。 2007年には国内市場シェアの半分を誇り、金融先物を含む世界の先物市場シェアの約2％を占めています。

##ハイライト

-取引所には、大豆、大豆ミール、パーム油などのさまざまな農産物の先物とスポット契約が記載されています。

-Dalian Commodities Exchange（DCE）は、中国本土にある商品およびデリバティブ取引所です。

-1993年に設立されたDCEは、農業生産と輸出のハブとしてこの地域の成長に大きく貢献し、世界最大の商品取引所の1つになりました。
