# 死に体
##レームダックとは何ですか？

レームダックは、債務不履行に陥った、または取引損失をカバーできないために破産したトレーダーに関連して使用される、使用されていない英国の用語です。

##レームダックを理解する

「レームダック」というフレーズは、[ロンドン証券取引所にさかのぼることができます](/lse)。決済日に請求に応じることができなかったメンバーは、「レームダック」と呼ばれ、交換のメンバーシップを失います。

「ラメダック」の最も初期の記録された使用法の1つは、1788年に発行された**下品な舌の古典的な辞書**に表示されます。彼は自分の損失や差額を支払うことができないか、支払わないでしょう。その場合、彼は借金が清算されて支払われるまで再びそこに現れることができないので、**路地から出て行く**と言われます。友愛に慌てて」

金銭的に負傷したトレーダーが取引所から**よろめく**というイメージは、このカラフルなフレーズがどのように使用されるようになったのかを説明するのに役立ちます。ブルとベアという用語は同じ期間のものです。

「レームダック」という用語は、特に市場が損失を被ったときから、新聞のアカウントによく登場しました。たとえば、このアカウントは1787年7月19日に記録されました。

>>

>証券取引所は、この日ここで行われたような、口座の決済に関するシーンを長年にわたって展示していませんでした。路地から**29羽のレームダック**が群がっていました。それらの不足は**25万ポンド**に達します。

>>

やがて、「ラメダック」は米国への道を見つけ、そこで最初に資金不足のビジネススキームの記述子になりました。また、効果がない、再選を求めないことを選択した、再選に立候補する資格がない、または選挙に敗れたが、選挙の勝者が職を支配するまで在職している政治家についても説明しています。

##ハイライト

-そのようなトレーダーは、取引所の路地から**よじ登った**ので、レームダックと呼ばれていました。

-レームダックは、決済日に請求に応じることができなかったロンドン証券取引所のメンバーを表すために使用される英国の用語でした。

-レームダックは、すべての借金が決済されて支払われるまで、再び取引することはできませんでした。
