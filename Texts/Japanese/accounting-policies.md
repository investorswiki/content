# 会計方針
##会計方針とは何ですか？

会計方針は、会社の経営陣によって実施される特定の原則と手順であり、[財務諸表の作成に使用され](/financial-statements)ます。これらには、会計方法、測定システム、および開示を提示するための手順が含まれます。会計方針は、原則が会計規則であり、方針がそれらの規則を順守する企業の方法であるという点で、[会計原則とは](/accounting-principles)異なります。

##会計方針の使用方法

会計方針は、会社が財務諸表を作成する方法を管理する一連の基準です。これらのポリシーは、[減価償却方法、](/depreciation)[のれん](/goodwill)の認識、研究開発（R＆D）コストの準備、在庫評価、財務会計の統合などの複雑な会計慣行を具体的に処理するために使用されます。これらのポリシーは会社ごとに異なる場合がありますが、すべての会計ポリシーは、[一般に公正妥当と認められた会計原則](/gaap)（GAAP）および/または国際財務報告基準（IFRS）に準拠する必要があります。

会計原則は、企業が運営することが期待されるフレームワークと考えることができます。ただし、フレームワークはある程度柔軟であり、会社の経営陣は、会社の財務報告に有利な特定の会計方針を選択できます。会計原則は時々寛大であるため、会社の特定の方針は非常に重要です。

[収益](/earnings)を報告する際に経営陣が保守的であるか積極的であるかを知ることができます。投資家は、収益の[質](/qualityofearnings)を評価するために収益レポートを確認する際に、これを考慮に入れる必要があります。また、会社の財務諸表をレビューするために雇われた外部監査人は、会社のポリシーをレビューして、GAAPに準拠していることを確認する必要があります。

>###重要

>会社の経営陣は、特定の在庫評価方法の選択など、自社の財務報告に有利な会計方針を選択できます。

>>

##会計方針の例

会計方針は、合法的に収益を操作するために使用できます。たとえば、企業は、[平均コスト](/averagecostmethod)、[先入れ先出し](/fifo)（FIFO）、または[後入れ先出し](/lifo)（LIFO）の会計方法を使用して在庫を評価することができます。平均原価法では、企業が製品を販売する場合、[会計期間中に生産または取得されたすべての在庫の加重平均原価を](/accountingperiod)使用して、売上原価（COGS）が決定され[ます](/cogs)。

FIFO在庫原価法では、企業が製品を販売する際に、最初に製造または取得した在庫の原価が販売されたと見なされます。 LIFO方式では、製品が販売されるときに、最後に生産された在庫のコストが販売されたと見なされます。在庫価格が上昇している時期には、企業はこれらの会計方針を使用して収益を増減させることができます。

たとえば、製造業の企業は、月の前半は1ユニットあたり10ドル、下半期は1ユニットあたり12ドルで在庫を購入します。同社は最終的に合計10ユニットを$10で購入し、10ユニットを$ 12で購入し、1か月で合計15ユニットを販売します。

会社がFIFOを使用している場合、売上原価は（10 x $ 10）+（5 x $ 12）=$160です。平均コストを使用する場合、売上原価は（15 x $ 11）=$165です。 LIFOを使用する場合、売上原価は（10 x $ 12）+（5 x $ 10）=$170です。したがって、売上原価を最小限に抑えて収益を増やすために、価格が上昇している期間にFIFO方式を使用することは有利です。

##ハイライト

-会計方針は、会社が財務諸表を作成するために使用する手順です。ルールである会計原則とは異なり、会計方針はそれらのルールに従うための基準です。

-会計方針における企業の選択は、経営陣がその収益を報告する際に積極的であるか保守的であるかを示します。

-会計方針は、合法的に収益を操作するために使用される場合があります。

-会計方針は、依然として一般に公正妥当と認められた会計原則（GAAP）に準拠する必要があります。
