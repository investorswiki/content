# 実質国内総生産（実質GDP）
##実質GDPとは何ですか？

実質GDPは、[インフレ](/inflation)調整後の[国内総生産の指標です](/gdp)。これを、インフレを調整せずに現在の価格を使用してGDPを測定する[名目GDPと比較してください。](/nominalgdp) 2つのインデックスは同じ出力を測定しますが、値の変化とボリュームの変化という非常に異なる目的で使用されます。

##より深い定義

GDPは、国の経済温度を把握するための最も重要な方法です。これは、特定の期間に生産されたすべての商品とサービスの金銭的価値から、生産に使用された商品とサービスの価値を差し引いたものです。企業も中小企業も、将来の計画をGDPに依存しています。投資家はこれを使用して、[利益率を見積もり](/margin)、財務上の決定を下します。エコノミストは、予測を支援し、経済への洞察を得るためにそれを使用します。

###名目GDP

価格の上昇と下降（主に上昇）の定期的なパルスは、名目GDPによって捉えられます。これは、時間の経過に伴う経済価値の成長を追跡します。国内総生産全体が1年で3％上昇し、同じ期間にインフレ率が2％で推移した場合、その年の名目GDPは+ 5％になります。

GDPをインフレ調整されていない他の経済データと比較する場合、名目GDPが好ましい数値です。たとえば、債務は常に名目金額として表示されるため、債務とGDPの比率は名目GDPデータを使用して計算されます。インフレがデータに組み込まれているため、名目GDPは経済成長の不正確な状況を提供する可能性があることに注意してください。

###実質GDP

実質GDPは、国の経済成長率のより正確な図を提供します。 GDPデフレーターは、インフレのデータを調整するために使用されます。これにより、価格の変化に関係なく、経済生産がどれだけ成長（または縮小）したかを理解できます。

実質GDPを計算するとき、インフレを管理するために基準年が選択されます。実質GDPの数値は、同じ基準年の価格を使用して、異なる年に生産された商品の量をキャプチャします。さまざまな年の異なる実質GDPの数値は、価値ではなく量の変化を反映しています。

##実質GDPの例

2016年の第1四半期には、米国の実質GDPは1.1％増加し、名目GDPは1.4％増加しました。コアPCEインフレ指数（実質GDPを調整するために使用されるインフレの尺度）は1.4％増加しました。 GDPデフレーターの計算に使用されるこの数値は、四半期の実質GDPと名目GDPの差を示しています。

##ハイライト

-実質GDPは、財とサービスの量と価値の両方の比較を示すため、年ごとおよび異なる年ごとのGDPの比較をより意味のあるものにします。

-実質国内総生産（実質GDP）は、特定の年に経済によって生産されたすべての財とサービスの価値を反映するインフレ調整後の指標です（基準年の価格で表されます）。そして、しばしば「一定価格」、「インフレ修正」、または「一定ドル」GDPと呼ばれます。

-実質GDPは、名目GDPをGDPデフレーターで割って計算されます。

＃＃ よくある質問

###実質GDPは何を測定しますか？

実質GDPは、1年間にわたる国の経済生産高のインフレ調整後の測定値です。米国のGDPは、主に支出アプローチに基づいて測定され、次の式を使用して計算されます：GDP = C + G + I + NX（ここで、C =消費、G =政府支出、I =投資、NX =純輸出）。

### GDPを使用することのいくつかの批評は何ですか？

多くのエコノミストは、GDPはインフォーマル経済を考慮せず、家庭での介護や家事労働を考慮せず、企業間の活動を無視し、数えるため、全体的な経済的成功の代用として使用すべきではないと主張しています。他の欠点の中でも、経済活動としてのコストと無駄。

###実質GDPの「実質」とはどういう意味ですか？

実質GDPは、数量を計算するがインフレに合わせて調整された一定の価格を使用して、商品とサービスの合計値を追跡します。これは、インフレを考慮しない名目GDPとは対照的です。一定の価格に調整することで、時間の経過や国間のリンゴ間の比較のための「実際の」経済的生産量の尺度になります。

###実質GDPの測定が重要なのはなぜですか？

GDPが大きい国では、その国で生成される商品やサービスの量が多くなり、一般的に生活水準が高くなります。このため、多くの市民や政治指導者は、GDP成長を国の成功の重要な尺度と見なしており、「GDP成長」と「経済成長」を同じ意味で参照することがよくあります。 GDPにより、政策立案者と中央銀行は、経済が縮小しているか拡大しているか、それが後押しまたは抑制を必要としているかどうか、そして景気後退やインフレなどの脅威が間近に迫っているかどうかを判断できます。インフレを考慮することにより、実質GDPは、ある期間から別の期間への生産レベルの変化をより正確に測定できます。

###実質GDPと名目GDPはどのように異なりますか？

インフレ期には、実質GDPは名目GDPよりも低くなります。デフレ時には、実質GDPは高くなります。たとえば、2000年に名目GDPが1,000億ドルで、2020年までに50％増加して1,500億ドルになった架空の国を考えてみましょう。同じ期間に、インフレによってドルの相対的な購入力が50％減少しました。名目GDPだけを見ると、経済は非常に好調であるように見えますが、2000ドルで表された実質GDPは、実際には750億ドルを示し、実際に経済成長の純全体的な低下が起こったことを示しています。経済パフォーマンスを測定する方法として実質GDPが経済学者に支持されているのは、この精度の高さによるものです。
