# ハイパーテキストマークアップ言語– HTML
##ハイパーテキストマークアップ言語（HTML）とは何ですか？

ハイパーテキストマークアップ言語（HTML）は、インターネット上での表示を目的としたファイルに挿入されるマークアップ記号またはコードのセットです。マークアップは、WebブラウザにWebページの単語や画像を表示する方法を指示します。

個々のピースマークアップコード（「<」と「>」の文字の間にある）は要素と呼ばれますが、多くの人はそれをタグとも呼びます。一部の要素はペアになっており、表示効果がいつ開始され、いつ終了するかを示します。

##HTMLの説明

ハイパーテキストマークアップ言語は、Webサイトの作成を容易にするコンピューター言語です。他の言語と同じようにコード語と構文を持っている言語は、比較的理解しやすく、時間が経つにつれて、誰かが作成できるものにおいてますます強力になります。 HTMLは、言語を設計および保守する組織であるWorld Wide Web Consortiumを装って、インターネットの要求と要件を満たすために進化し続けています。たとえば、 [Web2.0への移行に伴います](/web-20)。

ハイパーテキストは、インターネットユーザーがWebをナビゲートする方法です。ハイパーリンクと呼ばれる特別なテキストをクリックすると、ユーザーは新しいページに移動します。ハイパーの使用はそれが線形ではないことを意味するので、ユーザーは利用可能なリンクをクリックするだけでインターネット上のどこにでも行くことができます。マークアップは、HTMLタグがその中のテキストに対して行うことです。彼らはそれを特定のタイプのテキストとしてマークします。たとえば、マークアップテキストは、単語やフレーズに特別な注意を引くために、太字または斜体の形式で表示できます。

##ハイパーテキストマークアップ言語の基本

HTMLは、基本的に、テキストファイルに入力される一連のショートコードです。これらは、HTMLの機能を強化するタグです。テキストはHTMLファイルとして保存され、Webブラウザで表示されます。ブラウザはファイルを読み取り、作成者が表示レンダリングになるものを作成するために使用したコードの指示に従って、テキストを表示形式に変換します。 HTMLを作成するには、作成者のビジョンを作成するためにタグを正しく使用する必要があります。

タグは、通常のテキストをHTMLコードから分離するものです。タグは、グラフィック、画像、および表をWebページに表示できるようにする山かっこと呼ばれるものの間の単語です。異なるタグは異なる機能を実行します。最も基本的なタグは、テキストにフォーマットを適用します。 Webインターフェースをより動的にする必要があるため、カスケードスタイルシート（CSS）およびJavaScriptアプリケーションを使用できます。 CSSはWebページをよりアクセスしやすくし、JavaScriptは基本的なHTMLにパワーを追加します。

##HTMLとXML

HTMLとは異なり、Extensible Markup Language（XML）を使用すると、ユーザーは独自のマークアップを定義できます。たとえば、XMLを使用すると、あるユーザーはタグ<footnote>で脚注を示すことを選択でき、別のユーザーは<fn>を選択できます。

HTMLを使用すると、特定のタイプの情報を示すために使用できる事前定義されたタグは1つだけです。 XMLドキュメントは、ユーザー定義のタグが含まれており、ドキュメントはマークアップとコンテンツのみで構成されているため、読みやすいようになっています。

##ハイライト

-ハイパーテキストマークアップ言語（HTML）は、ワールドワイドウェブ上でページをレンダリングするためにWebブラウザによって使用される基本的なスクリプト言語です。

-HTMLの初期バージョンは静的（Web 1.0）でしたが、新しいイテレーションは非常に動的な柔軟性を備えています（Web 2.0、3.0）。

-マークアップは2つの角かっこ（<footnote>など）の間に表示されるテキストであり、コンテンツはそれ以外のすべてです。

-HyperTextを使用すると、ユーザーはリンクをクリックして、そのリンクによって参照される新しいページにリダイレクトできます。
