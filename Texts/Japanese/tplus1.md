# T + 1（T + 2、T + 3）
## T + 1（T + 2、T + 3）とは何ですか？

T + 1（T + 2、T + 3）は、証券取引の[決済日を表す略語です。](/settlementdate) 「T」は[取引](/transaction)日を表し、取引が行われる日です。数字の1、2、または3は、取引日から何日後に決済、つまり金銭と証券の所有権の譲渡が行われるかを示します。

## T + 1を理解する（T + 2、T + 3）

[株式市場](/stockmarket)が開かれている日のみです。 T + 1は、トランザクションが月曜日に発生する場合、決済は火曜日までに発生する必要があることを意味します。同様に、T + 3は、月曜日に発生するトランザクションは、これらの日の間に休日が発生しないと仮定して、木曜日までに決済する必要があることを意味します。ただし、金曜日にT + 3決済日で証券を売却する場合、所有権と送金は次の水曜日まで行う必要はありません。

株式の決済日を知ることは、配当を支払う会社に関心のある投資家や戦略的トレーダーにとっても重要です。決済日によって、どちらの当事者が配当を受け取るかが決まるからです。つまり、株式購入者が配当金を受け取るには、取引[は配当金の基準日より前に決済される必要があります。](/recorddate)

>取引から決済までの期間は、投資家が取引を取り消すことができるフレックスタイムではないことに注意してください。取引は取引日に行われます。転送は後日まで行われません。

>>

以前は、セキュリティトランザクションは電子的ではなく手動で行われていました。投資家は、実際の証明書である特定の証券の配達を待たなければならず、受領するまで支払いをしませんでした。納期は変動する可能性があり、価格は変動する可能性があるため、市場規制当局は、証券と現金を納品しなければならない期間を設定します。

何年も前、株式の決済日はT + 5、つまり取引日から5[営業日](/business-day)でした。最近まで、決済はT+3に設定されていました。今日はT+2です（つまり、取引日から2営業日後）。

決済日は証券の種類によって異なります。現在、すべての在庫はT+2です。ただし、債券、投資信託、および一部のマネーマーケットファンドは、T + 1、T + 2、およびT+3の間で異なります。

SECは最近、株式とETFの決済をT+1に短縮する提案を導入しました。承認されれば、新しい規則は2024年のある時期までに施行されるでしょう。

>決済日は、投資家が[記録の共有](/holderofrecord)[者となる日](/holderofrecord)です。週末と祝日は日数に含まれません。

>>

## T + 1の例（T + 2、T + 3）

T + 1（T + 2、T + 3）の決済日がどのように機能するかの例として、4月5日月曜日にMicrosoft（MSFT）の株式を購入する投資家を考えてみます。[注文が](/fill)完了した直後の投資のうち、マイクロソフトの株主としての投資家のステータスは、4月7日水曜日まで会社の記録簿に確定されません。

**訂正— 2022年5月5日：**この記事には、以前は投資信託の決済日のタイムラインに関するエラーが含まれていました。

##ハイライト

-株式は通常T+2であり、債券、投資信託、およびマネーマーケットファンドはT + 1、T + 2、およびT+3の間で異なります。

-文字「T」は取引日を示します。数字の1、2、または3は、取引日から何日後に決済が行われるかを示します。

-T + 1（またはT + 2、T + 3）は、取引の決済日を表す略語です。
