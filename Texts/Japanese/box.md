# ボストンオプションエクスチェンジ（BOX）
##ボストンオプションエクスチェンジ（BOX）とは何ですか？

[TMXグループ](/tmx-group)が所有および運営する[デリバティブ](/derivative)取引所を指します。交換は自動化されており、ボストンにあります。 2002年に設立され、2004年2月にBOXの取引が開始されました。この取引所は、既存のオプション市場に代わるものを提供するために、[モントリオール取引所、ボストン証券取引所、およびインタラクティブブローカーグループによる共同の取り組みとして開始されました。](/montrealexchange)この交換は現在、BOX交換と呼ばれています。

##ボストンオプションエクスチェンジ（BOX）の仕組み

BOX Exchangeは、2,000を超えるオプションクラスで取引する投資家に提供する自動取引所です。[モントリオール取引所](/montrealexchange)、ボストン証券取引所、Interactive Brokers Groupのパートナーシップの結果、2002年にボストンオプション取引所として作成されました。目的は、投資家に公開市場でオプションを取引する別の方法を提供することでした。取引は2月に正式に取引所で開始されました。 2004年6月6日。技術業務は、現在モントリオール取引所の親会社である[TMXグループによって処理されます。](/tmx-group) 2018年に社名をBOXExchangeに変更しました。

[価格改善](/priceimprovement)期間またはPIPと呼ばれるプロセスを通じてトレーダーに価格改善を提供する最初のオプション交換でした。すべての投資家はPIPを利用できますが、ファシリテーショントレードを提供する意欲と能力のあるブローカーが必要です。すべてのブローカーがこれをクライアントに提供しているわけではないため、一部の[投資家](/investor)はBOXで提供されている価格改善にアクセスできません。

取引オプションを提供します。[プット](/put)や[コール](/call)などのバニラオプションは、保有者に権利を与えますが、オプションの期限が切れる前に、[原資産を特定の価格（行使](/underlying-asset)[価格](/strikeprice)と呼ばれる）で売買する義務はありません。最も基本的な機能では、プットはロングポジションを[ヘッジ](/hedge)したり、原資産の価格下落を推測したりするために使用されます。コールは、原資産の価格上昇を推測したり、[ショート](/short)ポジションをヘッジしたりするために使用されます。

取引所は、参加者に匿名性と[透明性を備えた、各オプションの5つの最良のビッドとオファーを送信します](/transparency)。トレーダーは、高度な戦略のために複雑な注文を利用することもできます。参加者は、BOXに連絡してリスク管理戦略を調整することもできます。各企業は、個々のニーズを満たすために独自のリスクパラメータを設定できます。

##特別な考慮事項

[市場](/market)に新しいイノベーションをもたらそうとしています。 PIPに加えて、取引所は価格/時間優先[アルゴリズムを使用](/algorithm)して注文を照合し、すべての参加者が平等に扱われます。これにより、トレーダーは取引に低コストでアクセスでき、参加者は取引所で取引するために株式メンバーシップを必要としません。参加者は[ブローカーディーラー](/broker-dealer)であり、クライアントにBOX取引を提供できます。参加者は、自動取引システムでの低遅延取引の恩恵を受けます。[マーケットメーカー](/marketmaker)は、さまざまなオプションで流動性を提供します。

> 500契約以上の注文は、BOX Exchangeの他の大量注文に対して実行されるため、通常のビッドアンドアスクプロセスには影響しません。

>>

500以上の契約を取引する大規模なトレーダーは、[ブロックオーダー](/blocktrade)オークションにアクセスできます。これらにより、通常の[ビッドおよびオファー](/bid-and-ask)プロセスに影響を与えることなく、大量の注文を他の大量の注文に対して実行できます。これは、大量の注文が同様のサイズのブロック注文と一致しない場合、入札価格または要求価格に大きな影響を与える可能性があるため、不規則な価格変動を回避するのに役立ちます。

##ハイライト

-取引所は、各オプションについて5つの最良の入札とオファーを送信し、参加者に匿名性と透明性を提供します。

-ボストンオプション取引所は、TMXグループが所有および運営するデリバティブ取引所です。

-2002年に設立され、2004年に取引所で取引が開始されました。

-BOXは、価格改善期間と呼ばれるプロセスを通じてトレーダーに価格改善を提供した最初のオプション取引所でした。
