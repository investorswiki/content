# 貨幣の時間価値（TVM）
##お金の時間価値（TVM）とは何ですか？

、暫定[的に収益が得られる可能性がある](/earning-potential)ため、現在の金額は将来の同じ金額よりも価値があるという概念です。

これが財務の基本原則です。手持ちの金額は、将来支払われる同じ金額よりも大きな価値があります。

お金の時間価値は、現在の割引価値とも呼ばれます。

##お金の時間価値（TVM）を理解する

投資家は、一度投資されるとお金の合計が時間とともに増加するため、将来同じ金額ではなく、今日お金を受け取ることを好みます。たとえば、普通預金口座に預け入れられたお金は利息を稼ぎます。時間の経過とともに、利息は元本に追加され、より多くの利息を獲得します。それが複利の力です。

それが投資されない場合、お金の価値は時間とともに侵食されます。マットレスに1,000ドルを3年間隠すと、投資した場合にその期間に稼いだ可能性のある追加のお金を失うことになります。インフレによってその価値が低下したため、回収すると購買力がさらに低下します。

別の例として、今すぐ$ 10,000を受け取るか、2年後に$10,000を受け取るかを選択できるとします。額面価格は同じですが、遅延に関連する機会費用のため、今日の10,000ドルは、2年後よりも多くの価値と[有用性を持っています。](/utility)

言い換えれば、支払いの遅延は機会を逃したことです。

##お金の時間価値の公式

正確な状況によっては、お金の時間価値の計算式が若干変わる場合があります。たとえば、年金または[永久](/perpetuity)の支払いの場合、一般化された式には追加またはより少ない要素があります。ただし、一般的に、最も基本的なTVMの式では、次の変数が考慮されます。

-FV=将来のお金の価値

-PV=お金の現在価値

--i=金利

--n=1年あたりの複利計算期間の数

--t=年数

これらの変数に基づくと、TVMの式は次のとおりです。

** FV = PV x [1 +（i / n）]（nxt）**

##お金の時間価値の例

毎年合計10％の利息で1年間に合計10,000ドルが投資されると仮定します。そのお金の将来価値は次のとおりです。

FV = $ 10,000 x [1 +（10％/ 1）] ^（1 x 1）= $ 11,000

数式を再配置して、現在のドルで将来の合計の値を見つけることもできます。たとえば、今日から1年で5,000ドルの価値がある、7％の利息で毎年合成される現在のドル額は次のとおりです。

PV = $ 5,000 / [1 +（7％/ 1）] ^（1 x 1）= $ 4,673

###複利計算期間が将来価値に与える影響

複利計算期間の数は、TVMの計算に劇的な影響を及ぼします。上記の10,000ドルの例をとると、複利計算期間の数が四半期、月次、または日次に増加した場合、最終的な将来価値の計算は次のようになります。

-四半期複利：FV = $ 10,000 x [1 +（10％/ 4）] ^（4 x 1）= $ 11,038

-毎月の複利：FV = $ 10,000 x [1 +（10％/ 12）] ^（12 x 1）= $ 11,047

-毎日の複利：FV = $ 10,000 x [1 +（10％/ 365）] ^（365 x 1）= $ 11,052

これは、TVMが金利と期間だけでなく、複利計算が毎年計算される回数にも依存することを示しています。

##お金の時間価値は機会費用とどのように関連していますか？

[機会費用](/opportunitycost)は、お金の時間価値の概念の鍵です。お金は、時間をかけて投資され、プラスの利益を得る場合にのみ成長することができます。

投資されていないお金は、時間の経過とともに価値を失います。したがって、将来支払われると予想される金額は、それがどれほど自信を持って予想されても、その間に価値を失っています。

##お金の時間的価値が重要なのはなぜですか？

お金の時間価値の概念は、投資決定を導くのに役立ちます。

たとえば、投資家がプロジェクトAとプロジェクトBの2つのプロジェクトから選択できるとします。プロジェクトAが1年目に100万ドルの現金支払いを約束し、プロジェクトBが5年目に100万ドルの現金支払いを約束することを除いて、これらは同じです。

支払いは等しくありません。 1年後に受け取った100万ドルの支払いは、5年後に受け取った100万ドルの支払いよりも[現在価値が高くなります](/presentvalue)。

##お金の時間価値は、財務でどのように使用されますか？

お金の時間的価値が意思決定プロセスに影響を与えない単一の財務分野を見つけるのは難しいでしょう。

お金の時間価値は、投資機会を評価するための最も一般的で影響力のある方法の1つである[割引キャッシュフロー（DCF）分析の中心的な概念です。](/dcf)

また、財務計画およびリスク管理活動の不可欠な部分です。たとえば、年金基金のマネージャーは、口座保有者が退職時に十分な資金を確実に受け取れるように、お金の時間的価値を考慮します。

##ハイライト

-普通預金口座の場合、複利計算期間の数も重要な決定要因です。

-お金の時間価値は、お金の合計が将来の同じ金額よりも現在の価値があることを意味します。

-これは、お金は投資によってのみ成長できるからです。遅れた投資は機会を失います。

-お金の時間価値を計算するための式は、お金の量、その将来の価値、それが稼ぐことができる量、および時間枠を考慮します。
