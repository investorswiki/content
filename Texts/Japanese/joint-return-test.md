# ジョイントリターンテスト
##ジョイントリターンテストとは何ですか？

共同返還テストは、別の納税者がそのように主張するために潜在的な[扶養家族](/dependent)が合格しなければならない[IRSテストの1つです。](/irs)

>###重要

> IRSによると、「共同申告を行う場合、通常、既婚者を扶養家族として主張することはできません。 」

>>

共同返還テストでは、扶養家族は配偶者との共同返還を申請できず、親や保護者などの他の誰かの返還に依存していると主張することはできません。ただし、このルールには例外があります。

扶養家族を主張することは価値があるので、IRSは、扶養家族が二重に数えられていないことを確認するために、共同返還テストなどのいくつかのテストを実施します。

##ジョイントリターンテストを理解する

共同申告テストによると、共同申告を行う納税者は、「その人とその配偶者は、源泉徴収または推定納税額の還付を請求するためにのみ共同申告を行う」という1つの条件の下で扶養家族として請求できます。

納税者は、結婚して配偶者に申告した人を扶養家族として数えない場合があります。たとえその人が課税年度中にお金を稼がず、配偶者が共同申告で課税所得を報告した場合は納税者の家に住んでいる場合でも。

IRSは、次の例を示しています。「あなたは18歳の娘をサポートし、夫が軍隊にいる間、彼女は一年中あなたと一緒に暮らしました。彼は年間25,000ドルを稼ぎました。夫婦は共同で返還を申請します。娘を扶養家族だと主張しないでください。 」

別の例：「あなたの18歳の息子と彼の17歳の妻は、パートタイムの仕事から800ドルの賃金があり、他の収入はありませんでした。彼らは一年中あなたと一緒に住んでいました。どちらも確定申告をする必要はありません。彼らは子供がいません。息子の給料や妻の給料から税金は徴収されませんでした。ただし、彼らは共同で申告し、124ドルのアメリカの機会クレジットを請求し、その金額の払い戻しを受けます。 」

「アメリカ機会税額控除を主張することが申告の理由であるため、源泉徴収または推定納税の還付を受けるためだけに申告しているわけではありません。共同申告テストの例外は適用されないため、次のことができます。どちらも扶養家族だと主張しないでください。 」

##扶養家族を主張するための共同返還テスト

現代の[所得税](/federal_income_tax)は1913年に最初に導入され、扶養家族の控除が4年後に税法に追加されました。

議会が扶養家族の控除を長い間支持してきたことは、連邦所得税制度の全体的な累進性を維持しながら、大家族を持つという選択肢を支持したいという願望を反映しています。当初の所得税は非常に累進的であり、課税される所得の上位1％しかありませんでしたが、その累進性により、一般的に支援するためにより多くの所得を必要とする大家族に対するバイアスが生じました。

議会はそれ以来、扶養家族の控除を支援し続けており、2018年の税制改革法により、一部の納税者にとって扶養家族の請求をさらに有利にしています。

2018年以降、17歳未満の扶養家族を請求できる納税者は、以前の1,000ドルから、子供1人あたり2,000ドルの税額控除を受けることができます。さらに、議会は信用が段階的に廃止される所得水準を引き上げました。クレジットは現在、夫婦で40万ドル、独身で20万ドルの収入で段階的に廃止され始めています。これに対し、2017年の夫婦は110,000ドル、独身は75,000ドルでした。このメリットは、多くの申告者にとって特に価値のある税法の一部です。なぜなら、児童タックスクレジットは、課税所得を下げる控除ではなく、1ドルごとの納税義務の削減だからです。

>バイデン大統領が法に署名した2021年のアメリカ救済計画の結果、以前は2,000ドルだった児童税額控除の上限が、6歳から17歳の子供は3,000ドル、6歳未満の子供は3,600ドルに引き上げられました。クレジットも全額返金されます。以前は、1,400ドルしか返金できませんでした。これらの変更は、2021年の米国救済法の一部であり、追加の議会制定法によって延長されない限り、2021年の課税年度にのみ有効です。収入が75,000ドルを超えるシングル、および収入が150,000ドルを超えるカップルでは段階的に廃止されます。

>>

##ハイライト

-1つの例外は、扶養家族であると主張する人もその配偶者も課税対象となるのに十分な収入を得ていないが、源泉徴収された賃金の払い戻しを受けるために申告した場合です。

-ほとんどの場合、他の誰か（通常は配偶者）と共同で確定申告をしている扶養家族として誰かを主張することはできません。

-あなたが支援した家に住んでいて、お金を稼いでいない人が扶養家族であるかどうかを判断するときは、共同帰国テストを適用する必要があります。
