# 修正第16条
##修正第16条とは何ですか？

米国憲法修正第16条は1913年に批准され、議会は州間で、また国勢調査に関係なく、あらゆる源泉からの所得に税金を課すことができます。

##修正第16条を理解する

修正第16条の本文は次のとおりです。

>>

>議会は、いくつかの州の間で配分することなく、また国勢調査や列挙に関係なく、得られたあらゆる源泉からの収入に税金を課し、徴収する権限を有するものとします。

>>

議会は1909年7月に修正を求める共同決議を可決し、アラバマは1か月後にそれを批准しました。この改正は、1913年2月3日にデラウェア州、ワイオミング州、ニューメキシコ州が批准したときに発効しました。

最初の恒久的な連邦[所得税](/incometax)は1913年に徴収されました。スケジュールは7つの括弧で構成され、最初の2万ドルの所得に対する1％から50万ドルを超える所得に対する6％までの範囲の税率でした。政府は総額2,830万ドルを調達しました。 （これらの数値はインフレ調整されていません。）

> ### 1913

>最初の恒久的な連邦所得税が課された年。

>>

##修正第16条以前の連邦所得税

議会は、修正第16条の批准前に所得税を課していました。 [1862年の歳入法は](/revenue-tax-act-1862)、年収600ドル以上の市民に収入の3％を請求し、10,000ドル以上の市民は5％を支払いました。南北戦争に資金を提供するために税金が徴収されました。税率は1864年に引き上げられましたが、法律は1872年に失効することが許可されました。しかし、ほとんどの場合、連邦政府は1913年より前に物品税と関税からの収入の大部分を引き上げました。

議会は、1894年に、4,000ドルを超える所得に対して2％の別の国民所得税を課そうとしました。この税は、マサチューセッツ州の居住者であるCharles Pollockによって法廷で異議を申し立てられ、最高裁判所は**Pollockv。 1895年にFarmers'Loan＆Trust Co。**が、税金を大幅に引き下げました。

判決の根拠は、憲法第1条第2項第3項に由来します。

>>

>代表者と直接税は、それぞれの番号に応じて、この連合に含まれる可能性のあるいくつかの州に配分されるものとします...

>>

米国憲法では、「直接税」は「所有権を理由とする」財産に対する税金です。

** Pollock **において、最高裁判所は、この説明が原告のFarmers'Loan＆Trust Co.の10株からの収入、さらにはその資産から得られるすべての利息、[配当](/dividend)、および家賃に適用されるとの判決を下しました。 （裁判所は、労働からの収入が直接税であるとは裁定しなかったので、理論的には、連邦の不均衡な所得税の対象となる可能性があります。）直接税を徴収するために、議会はそれを州は、例えば、衆議院での代表に基づいて、それぞれに調達額を割り当てています。

修正第16条はその要件を削除しました。この変化は主に南部と西部の州によって支えられていました。当時、連邦政府の主要な収入源であった関税が、すでに急激な生活費の上昇を悪化させていました。

##修正第16条の定義に関するFAQ

###修正第16条は何と言っていますか？

>>

>修正第16条のテキストは、「議会は、いくつかの州の間で配分することなく、また国勢調査や列挙に関係なく、得られたあらゆる源泉からの[収入に税金を課し、徴収する権限を有するものとする」と述べています。](/income)

>>

###修正第16条は何を達成しましたか？

憲法修正第16条により、議会は最初の全国所得税を制定することができました。これは現在、連邦政府の最大の収入源です。それ以前は、連邦政府の収入のほとんどは関税によるものでした。

###修正第16条によると、収入の定義は何ですか？

修正第16条は、「あらゆる源泉からの収入」に言及しており、「収入」の意味を幅広く解釈することができます。後の事件では、最高裁判所は、「資本、労働、またはその両方から得られる利益」を意味するように収入を明確にしました。これには、「資本資産の売却または転換によって得られる利益」が含まれます。

###修正第16条は本当に合格しましたか？

米国下院によると、下院は5時間の討論の後、1909年7月12日に第16条改正案を可決し、賛成318票、反対14票でした。上院は77-0の投票で決議を承認した。しかし、修正案は、4年後の1913年まで、必要な数の州によって批准されませんでした。

##ハイライト

-憲法修正第16条以前は、憲法は直接税を各州の人口に比例させることを義務付けていました。連邦政府の収入のほとんどは、関税と物品税から来ていました。

-この変更は、一般的に南部と西部の州によってサポートされていました。

-最初の国民所得税は1894年に制定されましたが、** Pollockv。Farmers'Loan＆Trust Co。**（1895）の場合、最高裁判所によって取り下げられました。この訴訟に応じて、修正第16条が可決されました。

-米国憲法修正第16条は、1913年に批准され、議会があらゆる源泉からの所得に課税することを認めています。

-所得税は現在、連邦政府の最大の収入源です。
