# アテステーションサービス
##アテステーションサービスとは何ですか？

アテステーションサービス、またはアテステーションサービスは、公認会計士（CPA）によって実施される会社の財務諸表の独立したレビューです。 CPAは、データの信頼性に関する結論を含む認証レポートを提供します。

アテステーションサービスの基準は、専門サービスグループである[米国](/american-institute-of-certified-public-accountants)[公認会計士協会](/american-institute-of-certified-public-accountants)（AICPA）によって開発および公開されています。近年、財務報告とは関係のない幅広い証明サービスを反映するように基準が更新されました。たとえば、企業は消費者のプライバシーに関する声明で証明サービスを要求する場合があります。

##アテステーションサービスを理解する

法律では、証明とは、証人の立会いのもとで法的文書が適切に署名されたという証人による宣言です。基本的に、ドキュメントが有効であることを確認します。公証人は文書の証明を提供します。

財務では、証明サービスは、数値が正確で信頼できるというCPAの宣言です。サービスは独立した当事者によって完了するため、内部会計士によって作成された財務情報を検証します。

証明機能には、包括的審査、他者による審査の審査、特定の手続きの分析に限定された部分審査の3種類があります。

-包括的な調査は、企業の財務諸表の全体的な正確性と完全性についての[意見を表明するCPAで終了します。](/cpa)このレベルの精査は、会計監査に相当します。

-レビューはセカンドオピニオンに似ています。以前の監査の結果を確認するか、見逃された可能性のある問題を明らかにします。

-部分的な検査は、会社の簿記プロセスの特定の側面に限定されます。

3つの認証機能はすべて、監査手順、独立性、意見の表明に関してAICPAによって設定された基準に準拠している必要があります。

> CPAは、セキュリティやプライバシー管理ステートメントなどの非財務文書に対して証明サービスを実行するようにますます求められています。

>>

アテステーションレビューは、CPAの最も重要な義務の1つです。 2011年、アテステーションエンゲージメントの基準に関する声明（SSAE）No.監査基準（SAS）番号に関する16の置き換えられたステートメント。監査を実施する公認会計士のための信頼できるガイドとして70 。

この文書は、SSAENo.のATセクション801と呼ばれます。 16は、アテステーション機能の目標の概要を示しています。また、定義を広げます。 AICPAによると、公認会計士は、非財務手続きで証明サービスを実行するようにますます求められています。これらは多くの場合、複雑な政府の規制や要件への準拠を証明することを目的としています。いくつかの例を挙げれば、セキュリティ手順、プライバシー管理、温室効果ガス排出量のレポートなどが含まれる場合があります。

##ハイライト

-会社はまた、レビューまたは部分的な検査を求める場合があります。

-包括的なレビューの後、CPAはステートメント内の数値の整合性について意見を述べます。

-証明機能は、CPAによる会社の財務諸表のレビューです。
