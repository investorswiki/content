# 価値デフレ
##価値デフレとは何ですか？

価値のデフレ、または[シュリンクフレーションは、](/shrinkflation)小売業者やサービスプロバイダーがコストを削減し、より小さなパッケージを販売したり、より小さな部分を提供したり、同じステッカー価格を維持するために同じ価格でより少ない金額を提供したりするときに発生します。企業は、コストが上昇し、消費者が特に価格を意識しているときに、密かに価格を上げる方法としてこれを行う場合があります。

経済全体の価値デフレは、実際には、同じ価格水準で実質消費が低下する程度の価格インフレの一形態です。物価指数の計算で考慮されていない場合、価値デフレはインフレ率と生活費の過小評価につながる可能性があります。

##価値デフレを理解する

価値デフレは価格を上げる方法であるため、消費者は気づきにくく、一般的なパッケージの食品量の削減、レストランでの分量の削減、待ち時間の増加、顧客の削減という形をとることができます。サービスとサポート、または低コストの材料や材料への切り替え。

多くの買い物客は品質の変化よりも価格の変化に敏感であるため、これは成功する戦術になる可能性があります。マーケティングの観点からは、一貫した価格を維持するために、パッケージを縮小する方が価格を上げるよりも優れています。

しかし、クラフトが2016年にトブラローネバーを縮小し、英国で注目を集めたときに発見したように、価値デフレは裏目に出る可能性があります。英国の食品小売業者は、ポンド安と輸入原料のコストの上昇を補うために価値デフレを広範に利用しており、シュリンクフレーションが現象となっています。国家統計局によると、2012年から2017年にかけて2,500を超える製品が価値デフレの対象となりました。

[消費者物価指数](/consumerpriceindex)や[小売物価指数](/rpi)などのインフレ指標には、デフレが見られない場合があります。多くの経済統計機関は、品質調整プロセスを使用して、製品の重量や品質の変化から価格の動きを分離しているため、公式のインフレ統計では価格の上昇として表示されるはずです。

ただし、価値デフレの手法の多くは、設計上、測定が難しい場合があります。メーカーは、製品を大幅に変更することなく、低コストの入力に切り替えることができます。たとえば、ホットココアメーカーがより安価な甘味料に切り替えたり、粉チーズ製品のメーカーがその製品の木材パルプフィラー含有量を増やしたりする場合があります。これにより、一部のお客様の品質が低下する可能性がありますが、品質が低下しているにもかかわらず、動作を変更するだけでは不十分な場合があります。他の消費者は、変化にまったく気付かないかもしれません。これは、公式のデータや統計機関によって捕捉される場合とされない場合があります。

特に、サービスの削減や原材料や材料の品質の低下は、消費者や統計家が説明し、調整することが困難または不可能な場合があります。たとえば、ホテルが清掃スタッフに部屋ごとの清掃に費やす時間を減らすように指示して、清潔さを低下させたり、家電メーカーが低コストのカスタマーサポートプロバイダーに切り替えて通話待機時間を増やしたりする場合があります。ユーザーへの低品質のサービス。

価値のデフレが「完璧なビジネス犯罪」に相当するかどうかにかかわらず、世界中の消費者はこれらのパッケージングのトリックに注意を払う必要があります。問題は、[日用消費財の大手企業が、ステッカー価格の引き上げや](/fastmoving-consumer-goods-fmcg)[営業利益の](/operatingmargin)圧迫に直面する前に、どこまで価値のデフレを起こし、ブランドに損害を与えるリスクを冒すことができるかということです。

##ハイライト

-それは、パッケージまたは部分のサイズが同じ価格で縮小されるシュリンクフレーション、または微妙に安価な製品が古い製品と同等に販売される品質低下の形をとることができます。

-価値のデフレは、インフレ、特に統計機関によって説明されていないインフレに寄与する可能性があります。

-価値デフレとは、企業が販売価格を上げるのではなく、顧客に提供する価値を下げることです。
