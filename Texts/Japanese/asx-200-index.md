# S＆amp; P /ASX200インデックス
## S＆P / ASX 200インデックスとは何ですか？

S＆P / ASX 200インデックスは、オーストラリアのベンチマーク機関投資家向け株式市場インデックスであり、[浮動株調整後の](/freefloatmethodology)[時価総額で最大の200株で構成されてい](/capitalizationweightedindex)ます。これは、オーストラリア市場でS＆Pダウジョーンズが発行した多数のインデックスの1つですが（S＆P / ASXファミリーのインデックスと呼ばれます）、そのグループの主要なベンチマークと見なされています。

## S＆P /ASX200インデックスを理解する

[オーストラリア証券取引所（ASX）に上場](/asx)されている200の最大のインデックス適格株式のパフォーマンスを、浮動株調整後の時価総額で測定するように設計されています。インデックスの構成要素は、ASXにリストされている適格企業から抽出されます。企業を含めるにはASXに上場する必要がありますが、これらは一次上場または二次上場のいずれかになります（二次上場とは、別の国または別の取引所に一次上場している企業によるものです）。すべての普通株と優先株は含める資格がありますが、[ハイブリッド株](/hybridsecurity)（債券の特性を持つ証券）は含まれません。

## S＆P /ASX200インデックスの重要性

フロート調整時価総額を使用する理由は、取引可能なベンチマークインデックスを用意することです。したがって、大規模な機関投資家によるベンチマークとしての使用に適しています。フリーフロートが低い（つまり、取引が少ない）株式は取引が難しく、時価総額のベンチマークインデックスに含めるのに適切であるとは見なされません。インデックスが流動的であることを保証するために、定期的に取引されている株式のみが含まれる資格があります。したがって、インデックス発行者であるS＆Pダウ・ジョーンズは、S＆P / ASX 200が代表的で流動的で取引可能であるため、オーストラリアの卓越したベンチマークであると説明しています。

インデックスは2000年4月に開始され、インデックスに含まれる株式が適格基準を確実に満たすようにリバランスされています。 200株が含まれているにもかかわらず、インデックスは大企業によって支配されています。 2021年6月の時点で、インデックスの最大の10銘柄がインデックスの46％以上を占めています。これらの10銘柄のうち4銘柄は銀行グループであり、合計でインデックスの3分の1強を占めています。 2番目に大きなセクターは材料（資源）で、20.6％でした。 2021年6月の指数の株価収益[率](/price-earningsratio)は65.72で、[配当利回り](/dividendyield)は2.8％でした。

[上場](/etn)[投資信託](/etf)（ETF）と上場投資信託（ ETN）のほか、先物、オプション、および取引可能な先物のオプションがいくつかあります。 S＆P / ASX 200に関連する投資可能な商品のリストは、インデックスプロバイダーが発行する月次ファクトシートに記載されています。 S＆Pダウジョーンズも発行しているS＆P / ASX 200 VIXインデックスは、オーストラリアの株式市場の30日間の[インプライドボラティリティを測定します。](/iv)
