# ピップ
##ピップとは何ですか？

「ピップ」は、ポイントまたは価格金利ポイントのパーセンテージの頭字語です。ピップは、外国為替市場の慣習に基づいて、為替レートが行うことができる最小の全体の単価の動きです。

ほとんどの通貨ペアの価格は小数点以下4桁で、1ピップは小数点以下4桁です。したがって、ピップは1％の1/100または1[ベーシスポイントに相当し](/basispoint)ます。

たとえば、USD / CAD[通貨ペア](/currencypair)が行うことができる最小の全体の単位移動は、0.0001ドルまたは1ベーシスポイントです。

##ピップを理解する

ピップは外国為替（外国為替）の基本的な概念です。外国為替トレーダーは、価値が別の通貨に関連して表される通貨を売買します。これらの外国為替ペアのクォートは、小数点以下4桁まで正確なビッドおよびアスクスプレッドとして表示されます。

為替レートの動きはピップで測定されます。ほとんどの通貨ペアは小数点以下4桁まで引用されているため、これらのペアの最小の単位変更は1ピップです。

###ピップ値の計算

ピップの価値は、通貨ペア、為替レート、および取引額によって異なります。 EUR / USDペアのように、外国為替口座が米ドルで資金調達されており、USDがペアの2番目（または見積もり通貨）である場合、ピップは.0001に固定されます。

この場合、1ピップの値は、取引値（またはロットサイズ）に0.0001を掛けて計算されます。したがって、EUR / USDペアの場合、たとえば10,000ユーロの取引額に.0001を掛けます。ピップ値は$1です。 1.0801で1ドルに対して10,000ユーロを購入し、1.0811で販売した場合、10ピップまたは$10の利益が得られます。

一方、USD / CADペアのように、USDがペア（または基本通貨）の最初の場合、ピップ値には為替レートも含まれます。ピップのサイズを為替レートで割り、次に取引額を掛けます。

たとえば、.0001をUSD / CADの為替レート1.2829で割ってから、標準のロットサイズ100,000を掛けると、ピップ値は$7.79になります。カナダドルに対して1.2829で100,000米ドルを購入し、1.2830で販売した場合、1ピップまたは7.79ドルの利益が得られます。

日本円（JPY）のペアは、小数点以下2桁で引用され、小数点以下4桁の規則の顕著な例外を示しています。 EUR/JPYや[USD/JPY](/usd-jpy-us-dollar-japanese-yen-currency-pair)などの通貨ペアの場合、ピップの値は1/100を為替レートで割った値です。たとえば、EUR / JPYが132.62と見積もられている場合、1ピップは1/100÷132.62=0.0000754です。ロットサイズが100,000ユーロの場合、1ピップ（米ドル）の価値は7.54ドルになります。

>フラクショナルピップはピップよりも小さいため、より正確な測定が可能です。それらは、引用された為替レートの最後に上付きの数字として表示されます。分数ピップはピップの1/10です。

##ピップと収益性

通貨ペアの為替レートの動きは、トレーダーが一日の終わりに利益を上げるか損失を出すかを決定します。 EUR / USDを購入するトレーダーは、米ドルに対してユーロの価値が上がると利益を得ることができます。トレーダーが1.1835でユーロを購入し、1.1901で取引を終了した場合、取引で66ピップ（1.1901〜1.1835）になります。

ここで、112.06でUSD/JPYペアを売って日本円を買うトレーダーを考えてみましょう。トレーダーは、112.09でポジションをクローズすると、トレードで3ピップを失います。彼らが112.01でそれを閉じるならば、彼らは5ピップスで利益を上げます。

違いは小さいように見えるかもしれませんが、数兆ドルの外国為替市場では、利益と損失がすぐに加算される可能性があります。たとえば、112.01でクローズした1,000万ドルのポジションでは、トレーダーは500,000円を稼ぎます。米ドルでは、4,463.89ドル（500,000円/ 112.01円）です。

##実際のピップの例

[ハイパーインフレーション](/hyperinflation)と[切り下げ](/devaluation)の組み合わせは、為替レートを管理不能になるまで押し上げる可能性があります。大量の現金を持ち歩くことを余儀なくされている消費者に影響を与えることに加えて、これは取引を管理不能にする可能性があり、ピップの概念は意味を失います。

この有名な歴史的例は、ドイツのワイマール共和国で、1923年11月に為替レートが第一次世界大戦前の4.2マルク/ドルから4.2兆マルク/ドルに崩壊したときに起こりました。

もう1つの好例は、2001年に1ドルあたり160万のレベルに達したトルコリラであり、多くの貿易システムでは対応できませんでした。政府は為替レートから6つのゼロを削除し、新しいトルコリラに名前を変更しました。 2021年1月の時点で、平均為替レートは1ドルあたり7.3リラと合理的です。

##ハイライト

-ピップは1ベーシスポイントに相当します。

-日本円は例外で、為替レートは小数点以下2桁であり、小数点以下4桁ではありません。

-実際には、ピップは100分の1パーセント（1/100 x .01）であり、小数点以下第4位（0.0001）に表示されます。

-外国為替通貨ペアはピップで見積もられ、ポイントのパーセンテージの略です。

-外国為替相場のビッドアスクスプレッドはピップで測定されます。

＃＃よくある質問

###ピップはどのように使用されますか？

これらは、通貨ペアの為替レート市場相場の一部です。ピップは、あなたが取ったかもしれない市場でのポジションの見積もりと価値の変化を表しています。仮に、1.1356で通貨ペアを購入し、1.1360で販売したとします。あなたはあなたの取引で4ピップを作りました。次に、単一のピップの値を計算し、それをロットサイズで乗算して、利益のドル値を算出する必要があります。

###日本円の為替レートはピップを使用していますか？

はい、そうです。ただし、円は例外です。円の見積もりは通常、小数点以下2桁まで延長されます。したがって、単一の単位ピップは、他の通貨ペアの.0001ではなく.01です。

###ピップとは何ですか？

ピップは、外国為替相場でのビッドとアスクのスプレッドの差の最小の全体単位の測定値です。ピップは1％の1/100、つまり.0001に相当します。したがって、外国為替相場は小数点以下4桁まで拡張されます。小さい価格の増分は、小数のピップで測定されます。分数ピップはピップの1/10です。
