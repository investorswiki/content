# 産業連関分析
##入出力分析とは何ですか？

産業連関分析（IO）は、さまざまな経済[セクター](/sector)または産業間の相互依存性に基づくマクロ経済分析の一形態です。この方法は、一般的に、プラスまたはマイナスの経済ショックの影響を推定し、経済全体の波及効果を分析するために使用されます。 IO経済分析は、もともと[Wassily Leontief](/wassily-leontief) （1906–1999）によって開発されました。彼は、この分野での業績により、後に[ノーベル記念経済科学賞を受賞しまし](/nobel-memorial-prize-in-economic-sciences)た。

##入出力分析を理解する

IO分析の基盤には、入出力テーブルが含まれます。このような表には、経済のすべてのセクターの[サプライチェーンを](/supplychain)定量化する一連の行と列のデータが含まれています。産業は、各行と各列のヘッダーにリストされています。各列のデータは、その業界の生産機能で使用される入力のレベルに対応しています。

たとえば、自動車製造の列には、自動車の製造に必要なリソース（たとえば、鋼、アルミニウム、プラスチック、電子機器などの量）が表示されます。 IOモデルには通常、投資または生産の1ドル単位あたりに必要な労働量を示す個別の表が含まれています。

産業連関分析は、[新古典派経済学](/neoclassical)や西側の政策顧問によって一般的に利用されていませんが、[中央計画者](/centrally-planned-economy)に依存する協調経済の[マルクス経済分析で採用されてい](/marxian-economics)ます。

##3種類の経済効果

IOモデルは、直接、間接、および誘発の3つのタイプの影響を推定します。これらの用語は、特定の入力レベルに変更が加えられたときに経済全体に波及する初期、二次、および三次の影響を指す別の方法です。 IOモデルを使用することにより、[エコノミスト](/economist)は1つ以上の特定の業界でのインプットの変化による業界全体のアウトプットの変化を推定できます。

-経済ショックの直接的な影響は、支出の最初の変化です。たとえば、橋を建設するには、セメント、鉄鋼、建設機械、労働力、およびその他の投入物に支出する必要があります。

-間接的または二次的な影響は、需要を満たすために労働者を雇用する投入物の供給者によるものです。

-誘発された、または三次的な影響は、[個人消費のためにより多くの商品やサービスを購入するサプライヤーの労働者から生じるでしょう](/pce)。この分析は逆に実行することもでき、入力への影響が出力の観察された変化の原因である可能性が高いことを確認します。

##産業連関分析の例

これは、IO分析がどのように機能するかの例です。地方自治体は新しい橋を建設したいと考えており、投資のコストを正当化する必要があります。そのために、経済学者を雇ってIO調査を実施します。

エコノミストはエンジニアや建設会社と話し合い、橋の費用、必要な物資、建設会社に雇われる労働者の数を見積もります。

エコノミストは、この情報をドルの数値に変換し、IOモデルを介して数値を実行します。これにより、3つのレベルの影響が生じます。直接的な影響は、モデルに入力された元の数値、たとえば、[生の入力](/rawmaterials)（セメント、鋼など）の値です。

間接的な影響は、供給会社、つまりセメント会社や鉄鋼会社によって生み出された雇用です。これらの企業は、プロジェクトを完了するために労働者を雇う必要があります。彼らはそうするための資金を持っているか、そうするためにお金を借りなければならず、それは銀行に別の影響を与えるでしょう。

誘発される影響は、新しい労働者が自分自身とその家族のために商品やサービスに費やす金額です。これには、食品や衣料品などの基本的なものも含まれますが、[可処分所得が増え](/disposableincome)たため、楽しむための商品やサービスにも関連しています。

IO分析は、地方政府が新しい橋を建設したいと望んでいることによって引き起こされる経済のさまざまなセクターへの波及効果を研究します。橋は税金を利用して政府から一定の費用を必要とするかもしれませんが、IO分析は、経済に費やす労働者を雇う会社を雇い、それが成長するのを助けることによってプロジェクトが生み出す利益を示します。

##ハイライト

-産業連関分析では、3種類の影響がモデル化されています。それらは、直接的な影響、間接的な影響、および誘発された影響です。

-産業連関分析は、プラスまたはマイナスの経済ショックの影響を推定し、経済全体の波及効果を分析するために使用されます。

-産業連関表は産業連関分析の基盤であり、経済のすべてのセクターのサプライチェーンを定量化するデータの行と列を示しています。

-経済へのこれらの影響は、特定の投入レベルが変更されたときに決定されます。

-産業連関分析の使用は、西洋世界や新古典派経済学では一般的ではありませんが、経済の中央計画が必要な場合にマルクス経済学でよく使用されます。

-産業連関分析は、さまざまな経済セクターまたは産業間の相互依存性に基づくマクロ経済分析です。
