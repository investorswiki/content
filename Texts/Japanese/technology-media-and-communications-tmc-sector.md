# テクノロジー、メディア、テレコム（TMT）セクター
##テクノロジー、メディア、テレコム（TMT）セクターとは何ですか？

テクノロジー、メディア、およびテレコム（TMT）セクターは、新しいテクノロジーに焦点を当てた企業の大多数を含む業界グループです。 TMTと1990年代の[ニューエコノミーの考え方の間にはかなりの重複があり](/neweconomy)ます。 TMTセクターは、テクノロジー、メディア、および通信（TMC）と呼ばれることもあります。

経済における技術の役割が拡大したため、技術部門はもはやイノベーションに依存するすべての企業を保持することはできません。たとえば、Meta（META）（以前のFacebook）とNetflix（NFLX）はどちらも、Technology Select Sector SPDR Fund（XLK）ではなくCommunication Services Select Sector SPDR Fund（XLC）に含まれています。大手ハイテク企業がTMTセクターをますます支配しています。

##TMTセクターを理解する

[R＆D ）](/randd)に依存する幅広い企業が含まれます。彼らは特許やその他の[知的財産](/intellectualproperty)に焦点を当てており、企業の急速な成長を望んでいます。その結果、TMTセクターの投資家は、[企業価値対売上高](/enterprisevaluesales)（EV /売上高）を優先して、比較的高い株価収益率[（P / E）を許容することがよくあります。](/price-earningsratio)

TMTセクターは、市場を劇的に上回る潜在的な[テンバガーを](/tenbagger)探している[成長投資家にとって有益です。](/growthinvesting)新しい技術はTMTセクターで継続的に開発されており、一部の企業は最終的にこのセクターで有名になる可能性があります。成長投資家は、TMT業界の株式を早期に取得することで、より高いリターンを達成したいと考えています。

>多くのTMT株は、市場をアウトパフォームする可能性が高いですが、より大きな下振れリスクにも直面しています。

##特別な考慮事項

この業界セグメントは幅広い範囲をカバーしているため、TMTをハードウェア、半導体、ソフトウェア、メディア、テレコムなどのサブセクターに分割すると役立つことがよくあります。 TMTセクターには、新しいハイテク企業が含まれているため、多くの合併、買収、新規株式公開（ [IPO](/ipo) ）が発生しています。各サブセクターには、異なる成長指標と見通しもあります。たとえば、通信セクターはワイヤレスへの移行によって推進されています。

ハードウェア企業には、コンピューターメーカー（IBM、Dell、HP）だけでなく、サーバーシステム、モバイルデバイスの携帯電話、タブレット、ハードドライブやメモリなどのストレージデバイスのメーカーも含まれます。ハードウェアの中で、[半導体](/semiconductor)メーカーはあらゆる種類のアプリケーションで使用される集積回路とマイクロチップを開発および製造しています。代表的な企業には、Intel、AMD、Texas Instruments、Nvidiaなどがあります。

ソフトウェア会社は、個人と企業の両方のためにコンピューターまたはモバイルアプリケーションを作成します。 Microsoft、Adobe、SAPは、ソフトウェア企業のトップにランクされています。

さらに、メディアおよび通信会社もTMTセクターで重要な役割を果たしています。メディア会社は、テレビ、印刷物、およびオンラインでマルチメディアコンテンツを開発、制作、および配布しています。テレビネットワーク、ケーブルTVプロバイダー、制作スタジオ、ソーシャルメディア企業はすべてこのサブセクターに属しています。最後に、テレコムは、電話、テレビ、[インターネットサービスプロバイダーなどの通信関連ビジネスに焦点を当てています](/isp)。 AT＆TやVerizonなど、業界を支配するいくつかの重要な通信があります。

##テクノロジー、メディア、テレコムセクターの例

市場参加者はTMT企業をさまざまなサブセクターに分類する可能性があるため、複数のTMTサブセクターに属する企業の例は数多くあります。メタは、インターネット企業またはメディア企業のいずれかと見なすことができます。 Appleは、誰が会社を審査しているかに応じて、インターネット、ハードウェア、ソフトウェア、またはメディアのカテゴリに分類できます。

サブセクターをまたがる他の例は、Hulu、Amazon、Netflixです。場合によっては、TMTのあるサブセクターの企業が別のサブセクターを合併または買収して、製品の提供を[統合、多様化、および拡大することがあります。](/consolidation)

合併の例としては、2000年のAOLとTime Warnerの合併があります。2015年にAT＆TとDIRECTVが協力し、2016年にDell-EMCの合併が行われました。

##ハイライト

-TMT業界の株式を早期に取得することにより、成長投資家はより高いリターンを達成することを望んでいます。

-テクノロジー、メディア、テレコム（TMT）セクターは、新しいテクノロジーに焦点を当てた企業を含む業界グループです。

-TMTセグメントは非常に広いため、ハードウェア、半導体、ソフトウェア、メディア、テレコムなどのサブセクターに分割されることがよくあります。

-TMTセクターには、研究開発（R＆D）に依存する幅広い企業が含まれます。
