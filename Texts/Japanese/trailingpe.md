# トレーリング株価収益率（トレーリングP / E）
##株価収益率とは何ですか？

株価収益率（P / E）は、過去12か月の実際の収益に基づく相対的な評価の倍数です。これは、現在の株価を取得し、過去12か月の1株当たりのトレーリング利益（EPS）で割って計算されます。

トレーリングP/Eは、フォワードP /Eと対比することができます。[フォワードP/E](/forwardpe)は、代わりに、予測される将来の収益を使用して株価収益率を計算します。

##トレーリング株価収益率（P / E）を理解する

株価収益率（ [P / E比](/price-earningsratio)）は、企業の株価を直近の[会計年度の収益で割って計算され](/fiscalyear)ます。人々が一般的に株価収益率を指すとき、彼らは通常、末尾の株価収益率を指します。これは、現在の市場価値または株価を過去12か月間の[1株当たり利益（EPS）で割って計算されます。](/eps)

直近の会計年度の収益は、年次報告書の[損益計算書に記載されています。](/incomestatement)損益計算書の下部には、会社の会計年度全体のEPSの合計があります。会社の現在の株価をこの数値で割って、株価収益率を算出します。

>>

>トレーリングP/Eレシオ=現在の株価/トレーリング12か月EPS

>>

この測定値は、予想される将来のパフォーマンスではなく実際のパフォーマンスに基づいて計算されるため、信頼できると見なされます。ただし、企業の過去の収益は必ずしも将来の収益の良い予測因子ではないため、注意が必要です。

##アナリストがP/Eを使用するのはなぜですか？

アナリストは、相対的な収益のリンゴ間の評価を作成するため、株価収益率が好きです。したがって、株価収益率を使用して、市場での相対的な掘り出し物を探したり、株価が他の株と比較して高すぎる場合を判断したりできます。[経済的な濠](/economicmoat)が深いため、株価収益率を高くする価値のある企業もありますが、収益に比べて株価が高い企業の中には、単に高値を付けている企業もあります。同様に、一部の企業は非常にお買い得であるため、より低いP / Eに値しますが、他の企業は、財政的弱さのために低いP/Eで正当化されます。トレーリングP/Eは、アナリストが期間を一致させて、[相対値のより正確で最新の測定値を得るのに役立ちます](/relative-value)。

株価収益率のデメリットは、株価が常に変動している一方で、収益は固定されていることです。アナリストは、昨年末の収益ではなく、直近の4四半期の収益を使用する株価収益率を使用して、この問題に対処しようとしています。

##株価収益率の追跡例

たとえば、株価が50ドルで、12か月のトレーリングEPSが2ドルの企業は、トレーリングP / E比が25倍（25回読み取り）になります。これは、同社の株式が過去12か月の収益の25倍で取引されていることを意味します。

同じ例を使用すると、会社の株価が年の途中で40ドルに下がった場合、新しい株価収益率は20倍になります。これは、株価が収益の20倍で取引されていることを意味します。収益は変わっていませんが、株価は下がっています。

過去2四半期の収益も減少した可能性があります。この場合、アナリストは、会計年度の計算の最初の4分の2を、直近の2四半期に置き換えて株価収益率を下げることができます。直近の2四半期で表される上半期の収益が低下傾向にある場合、株価収益率は20倍を超えます。これはアナリストに、株価が実際には収益レベルの低下を考えると現在の価格で[過大評価されている可能性があることを示しています。](/overvalued)

##トレーリングvs.フォワードP/E

トレーリングP/E比率は、次の4四半期または次の予測される12か月の収益の収益見積もりまたは予測を使用するフォワードP/Eとは異なります。その結果、フォワードP / Eは、企業を評価する際に投資家にとってより適切な場合があります。それにもかかわらず、フォワードP / Eは推定される将来の収益に依存しているため、誤算やアナリストのバイアスが発生しやすくなります。また、企業は、次の四半期の収益レポートで[コンセンサスの見積もりP / E](/consensusestimate)を上回るために、収益を過小評価または誤解する可能性があります。

両方の比率は、取得時に役立ちます。トレーリングP/E比は、買収された会社の過去の業績の指標です。 Forward P / Eは、将来に向けた会社のガイダンスを表しています。通常、買収した会社の評価は後者の比率に基づいています。ただし、購入者は、目標の収益が達成された場合に追加の支払いを行うオプションを使用して、取得価格を下げるために[収益引当金を使用できます。](/earnout)

##ハイライト

-トレーリングP/Eは、期間間および企業間の相対株価を標準化および比較するための有用な指標と見なされます。

-トレーリングP/Eは広く使用されていますが、過去の収益が会社の現在または将来の収益状況を正確に反映していない可能性があるという点で制限されています。

-株価収益率のトレーリングは、過去1年間の1株当たり利益と比較した市場での企業の株価を示しています。
