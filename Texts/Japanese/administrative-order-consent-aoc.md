# 管理上の同意書（AOC）
##管理上の同意書（AOC）とは何ですか？

行政同意書（AOC）は、個人または企業と規制機関との間の合意であり、個人または企業は、違反によって生じた損害の支払いと、損害の発生を引き起こした活動の停止に同意します。

同意に基づく行政命令（AOC）は、ほとんどの場合、汚染などの環境被害に関連しています。ビジネスの文脈では、[環境保護庁](/environmental-protection-agency)（EPA）は通常、環境損傷をクリーンアップまたは修正するためにAOCを発行します。 AOCは、ビジネスが準拠していない場合、法廷で執行することができます。

##同意に基づく管理命令を理解する

行政同意書（AOC）は、州政府および連邦政府によって、個人および企業に、その活動によって引き起こされた環境損害の支払いを強制するために使用されます。

比較的迅速かつ簡単にクリーンアップできる小さな流出などの損傷は限定的である場合もあれば、スーパーファンドのクリーンアップや大規模な油流出などの広範囲にわたる場合もあります。 AOCは、個人または企業に行動を起こさせます。

同意に関する行政命令は、どのような損害が発生したか、および損害と浄化を軽減するためにどのような措置を講じる必要があるかを概説しています。

> AOCは、企業が準拠していない場合、法廷で執行することができます。

>>

##特別な考慮事項

AOCはまた、一般市民、企業、および利害関係者が、企業がとることが期待される提案された行動を検討することを可能にするコメント期間を提供します。コメント期間中、たとえば、ビジネスはクリーンアップのコストが高すぎると言うかもしれませんが、コミュニティのメンバーはビジネスに必要なアクションが十分に広範ではないと言うかもしれません。

政府および企業は、自主的に行政命令を締結します。ただし、これは、契約が不履行に陥った後、契約が拘束力を持たないことを意味するものではなく、訴訟につながる可能性があります。

>ほとんどの場合、政府と企業の両方が、それらに特に適用される契約の構成要素に対して共同で個別に責任を負います。同意に基づく行政命令に関与する事業とAOCを発行する政府の両方が、環境被害が費用効果が高くタイムリーに処理されることを保証することに関心を持っています。

>>

##同意に基づく管理命令の例

AOCの例として、ハワイ大学は2021年に、ハワイ学部との合意に従って、汚染限界を超えた廃棄物の流出のためにワイキキ水族館の水システム（大学が監督）をアップグレードすることに合意しました。健康の。合意されたAOCにより、大学は、施設をアップグレードし、特定された環境問題を修正する間、研究および公教育の目的で水族館を開いたままにすることができます。

環境衛生副局長の河岡義裕氏によると。 「AOCの条件は、ワイキー水族館が許可の条件に準拠し続けることを保証し、将来、沿岸水域を危険にさらす潜在的な水質汚染を防ぐのに役立ちます。」

##ハイライト

-ビジネスの文脈では、環境保護庁（EPA）は通常、環境損傷をクリーンアップまたは修正するためにAOCを発行します。

-行政同意書（AOC）は、個人または企業と規制機関との間の合意であり、違反によって生じた損害の支払いと、損害の発生を引き起こした活動の停止に同意します。

-AOCは、ほとんどの場合、汚染などの環境被害に関連しています。

-政府および企業は自主的にAOCを締結しますが、締結されると拘束力があり、不履行により訴訟が発生する可能性があります。

-ほとんどの場合、政府と企業の両方が、それらに適用される契約の構成要素に対して連帯責任を負います。したがって、企業と政府の両方が環境被害を軽減することに関心を持っています。

＃＃よくある質問

###スーパーファンドクリーンアップ契約とは何ですか？

スーパーファンド和解契約は、スーパーファンドサイトのクリーンアッププロセスを開始するために潜在的に責任のある当事者と政府の間で締結された行政同意書（AOC）です。このようなAOCは裁判所の承認を必要としませんが、AOCに到達できない場合、EPAは司法同意判決を求めることができます。

###スーパーファンドサイトとは何ですか？

スーパーファンドサイトは、人々にとって危険であると考えられている高度に汚染された場所の米国指定です。したがって、連邦政府によって部分的に資金提供された特別な除染と浄化の努力が必要です（特に単一の事業体が責任を問われることができない場合）。

###スーパーファンド決済契約の種類は何ですか？

1つ以上の潜在的責任者（PRP）がSuperfundサイトで特定された場合、AOCは次の4つの方法のいずれかで起草できます。-**管理契約：** PRPは、政府にすでに発生した費用と政府の請負業者が負担する将来の費用。-**「作業」の合意：** PRPはクリーンアップ作業を行い、すべての費用を支払います。-**費用回収契約：**管理契約と同様に、PRPは過去に発生した費用に対する政府。-**「キャッシュアウト」契約：** PRPは、将来行われる作業に先立って、推定サイト費用の「適切な金額」を支払います。
