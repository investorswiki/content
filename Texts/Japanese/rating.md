# 評価
##評価とは何ですか？

格付けは、アナリストまたは格付け機関によって株式または債券に割り当てられる評価ツールです。割り当てられた評価は、株式または債券の投資機会のレベルを示します。 3つの主要な評価機関は、 [Standard＆Poor's](/sp) 、 [Moody's Investors Service](/moodys) 、および[FitchRatings](/fitch-ratings)です。

##評価の仕組み

[バイサイド](/buyside)と[セルサイドの](/sellside)両方で株式を調査し、それらの株式について意見を書くアナリスト。多くの場合、「購入」、「保留」、「販売」などの評価が含まれます。一方、債券は3つの主要な[債券格付け機関によって格付けされてい](/bond-rating-agencies)ます。

>企業は、負債をできるだけ少なくし、企業内で突然の変化が発生した場合に警戒を怠らないことで、格付けスコアを向上させることができます。

##評価の種類

###アナリストの評価

バイサイドのアナリストは、ポートフォリオ管理の決定を通知する目的で、チームに意見を書きます。セルサイドのアナリストは、他の人に彼らの研究について教育し、クライアントに代わって特定の株を売ろうと試みるために意見を書きます。株式の場合、アナリストは「購入」、「保留」、または「販売」の評価と、株式に対してこのアクションを推奨する理由の説明を割り当てることができます。

主要なウォール街の銀行や機関に関しては、それらはすべて異なる用語と分類を使用しています。たとえば、モルガンスタンレーでは、「太りすぎ」、「等体重」、「低体重」という用語を使用しています。その評価のタイムラインは12から18ヶ月です。クレディ・スイスは、12か月の期間に基づいて、「アウトパフォーム」、「ニュートラル」、および「アンダーパフォーム」という用語を使用しています。これらの用語はすべて、「購入」、「保留」、および「販売」の評価のバリエーションです。

###格付け機関の格付け

債券の場合、格付け機関は、発行体の基本的な財務状況に基づいて債券の相対的な安全性を評価します。これにより、発行者が元本を返済し、利息を[支払う](/payment)[能力が精査され](/ability-to-repay)ます。

ムーディーズおよびS＆Pの格付けは、投資適格カテゴリーの最高から最低まで、Aaa / AAA、Aa1 / AA +、Aa2 / AA、Aa3 / AA-、A1 / A +、A2 / A、A3 / A-、Baa1 /BBB+です。 Baa2/BBBおよびBaa3/BBB-。

Standard＆Poor'sは、 [S＆P 500](/sp500)インデックスのプロバイダーであり、独立した信用格付けの主要なデータソースおよびインデックスプロバイダーでもあります。 S＆P 500インデックスは、米国株式市場の全体的な状態を判断するために広く利用されているゲージです。

ムーディーズは、政府および商業発行の債券に関する国際金融調査のプロバイダーです。ムーディーズは、格付けシステムを利用して借り手の信用力を判断しています。この評価尺度は、Aaa（最高品質）から始まり、C（最低品質）になります。

フィッチレーティングスは、国際的な信用格付け機関でもあります。このエージェンシーは、企業が内部の変化にどれほど敏感であるか、企業が保有する債務の種類などの要因に基づいて評価を行います。フィッチは、投資家がデフォルトしない投資のガイドとして使用され、その結果、確実なリターンにつながります。

さまざまな格付け機関によって割り当てられた格付けは、主に保険会社または発行者の信用力に基づいています。したがって、この格付けは、[デフォルト](/default2)の確率の直接的な尺度として解釈できます。ただし、信用の安定性と支払いの優先順位も格付けに考慮されます。

##ハイライト

-株式については、バイサイドとセルサイドのアナリストが調査を行い、カバーする株式について、「購入」、「保留」、「販売」などの評価を含む意見を書き込みます。

-3つの主要な債券格付け機関は、スタンダード＆プアーズ、ムーディーズインベスターズサービス、フィッチレーティングスです。

-格付けは、アナリストまたは格付け機関によって株式または債券に割り当てられる評価ツールです。

-債券格付けは、発行者または保険会社の信用力を評価します。これは、デフォルトの可能性の直接的な尺度として解釈できます。
