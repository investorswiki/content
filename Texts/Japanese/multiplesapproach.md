# 複数のアプローチ
##倍数アプローチとは何ですか？

倍数アプローチは、同様の資産が同様の価格で販売されるという考えに基づく[評価理論です。](/valuation)[営業利益](/operatingmargin)やキャッシュフローなど、企業の比較に使用される比率のタイプは、類似の企業間で同じであると想定しています。

投資家はまた、倍数アプローチを倍数分析または評価[倍数と呼びます](/multiple)。その際、株価収益率（P / E）などの財務比率を収益の倍数として参照する場合があります。

##倍数アプローチの基本

一般に、「倍数」は、株式の評価に使用できるさまざまな指標のクラスの総称です。倍数は、資産の市場または推定値を[財務諸表の特定の項目で割ることによって計算される単純な比率です](/financial-statements)。倍数アプローチは、同じ財務指標を使用して類似の企業を評価しようとする比較分析方法です。

評価アプローチを使用するアナリストは、特定の比率が適用可能であり、同じ業種または[業界内で事業を行っているさまざまな企業に適用されると想定しています](/industry)。言い換えれば、倍数分析の背後にある考え方は、企業が比較可能である場合、倍数アプローチを使用して、別の企業の価値に基づいて1つの企業の価値を決定できるということです。倍数アプローチは、企業または財務上の特性（たとえば、期待される成長）の多くを単一の数値でキャプチャし、特定の財務指標（たとえば、 [EBITDA](/ebitda) ）を掛けて、企業または株式の価値を生み出すことを目的としています。

##倍数アプローチで使用される一般的な比率

[企業価値の倍数](/ev-ebitda)と株式の倍数は、評価の倍数の2つのカテゴリです。企業価値の倍数には、[企業の](/enterprisevaluesales)[e-value-to-sales比率](/enterprisevaluesales)（EV /売上）、EV / EBIT、およびEV/EBITDAが含まれます。エクイティの倍数には、会社の株価と、収益、売上、簿価などの基礎となる会社の業績の要素との比率を調べることが含まれます。普通株式の倍数には、[株価収益率](/price-earningsratio)（P / E）比率、[株価収益率](/pegratio)（PEG）比率、[株価純資産倍率](/price-to-bookratio)（P / B）、および[株価純資産倍率](/price-to-salesratio)（P / S）比率が含まれます。 。

[企業](/enterprisevalue)[価値](/enterprisevalue)（EV）に変化がない場合でも、[資本構造](/capitalstructure)の変化によって株式の倍数が人為的に影響を受ける可能性があります。企業価値の倍数は、資本構造に関係なく、さまざまな企業を直接比較できるため、株式の倍数よりも優れた評価モデルであると言われています。さらに、分母は損益計算書の上位で計算されるため、企業評価の倍数は通常、会計上の差異による影響が少なくなります。ただし、エクイティの倍数は、簡単に計算でき、ほとんどの金融Webサイトや新聞からすぐに入手できるため、投資家によってより一般的に使用されます。

##倍数アプローチの使用

投資家は、類似の企業を特定し、それらの市場価値を評価することから、倍数アプローチを開始します。次に、比較可能な企業の倍数が計算され、[平均](/mean)や[中央値](/median)などの主要な統計指標を使用して標準化された数値に集約されます。さまざまな企業の中で重要な倍数として識別された値は、分析中の企業の対応する値に適用され、その値が推定されます。倍数を作成する場合、分母は過去の利益ではなく、利益の予測を使用する必要があります。

後向きの倍数とは異なり、前向きの倍数は、評価の原則と一致しています。特に、企業の価値は、過去の利益や[埋没費用ではなく、将来のキャッシュフローの](/sunkcost)[現在価値に等しいということ](/presentvalue)です。

-一般的に使用されるエクイティ倍数には、株価純資産倍率、PEG、株価純資産倍率、株価純資産倍率が含まれます。
