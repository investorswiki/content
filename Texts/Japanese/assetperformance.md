# 資産パフォーマンス
##資産パフォーマンスとは何ですか？

資産パフォーマンスとは、運用リソースを取得して管理し、収益性の高い収益を生み出す企業の能力を指します。企業は、資産からプラスのパフォーマンスを引き出すことができ、その結果、会社のプラスのパフォーマンスが得られます。

総資産利益率（ROA）などの比率、および企業が資産を使用して収益を生み出す効率と運用の効率を追跡するその他の指標は、資産のパフォーマンスの指標です。

##資産パフォーマンスを理解する

資産パフォーマンスとは、企業が運用リソースの使用を管理する方法を指します。特定のメトリックと比率は、リソースの使用を測定できます。アナリストは、これらの指標と比率を利用して、同じ業界の多くの企業の資産パフォーマンスを比較します。アナリストは、キャッシュコンバージョンサイクル、総資産利益率、固定資産回転率などの指標を使用して、企業の年間資産パフォーマンスを比較および評価します。

通常、資産パフォーマンスの向上は、企業が同じ量の資産を使用してより高い収益を得ることができるか、より少ない資産を使用して同じ量の収益を生み出すのに十分効率的であることを意味します。

##総資産利益率（ROA）

企業の資産パフォーマンスを決定する最も一般的な方法は、総[資産利益率](/returnonassets)（ROA）を調べることです。 ROAは、ある期間に報告された[純利益を調べ、それを総](/netincome)[資産で割り](/asset)ます。総資産を測定するには、同じ期間の開始資産値と終了資産値の平均を計算します。

>>

>総資産利益率（ROA）=純利益/総資産

>>

一部のアナリストは[、金利税](/ebit)引前利益（EBIT）を取得し、それらを総資産で除算します。

>>

>総資産利益率（ROA）=EBIT/総資産

>>

これは、経営陣の資金調達の決定に影響されることなく、資産から利益を生み出す企業の能力の純粋な尺度です。

###良いROAとは何ですか？

どちらの方法を使用しても、結果は[収益率として報告され](/rateofreturn)ます。総資産利益率が20％の場合、会社は資産に投資した5ドルごとに1ドルの利益を生み出します。 ROAは、ビジネスが投資1ドルあたりの利益を増やし続けているかどうかをすばやく示していることがわかります。投資家は、優れた経営陣がROAの向上に努め、自由に使える資産1ドルからより多くの利益を引き出すことを期待しています。

[成長企業](/growthstock)にとって、角を曲がった問題の確かな兆候です。売上高の伸びを目指すということは、[売掛金](/accountsreceivable)、在庫、生産設備、設備などの資産への大規模な先行投資を意味することがよくあります。需要が減少すると、組織は高額で乾燥した状態になり、請求書を支払うために売却できない資産に過剰に投資する可能性があります。その結果、金融危機になる可能性があります。

##ハイライト

-資産パフォーマンスは、貸借対照表に保持されている資産から利益または利益を生み出す企業の能力を測定します。

-資産パフォーマンスは通常、ある会社のパフォーマンスを経時的に、または競合他社と比較するために使用されます。

-ROAは、企業の資産パフォーマンスを測定するために最も広く使用されている指標です。

-強力な資産パフォーマンスを生み出すことは、アナリストが企業を優れた投資と見なすかどうかを判断するために使用される基準の1つです。
