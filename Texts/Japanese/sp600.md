# S＆amp; P 600
## S＆P 600とは何ですか？

[Standard＆Poor'sが管理する](/sp)[小型](/small-cap)株のインデックスです。特定の流動性と安定性の要件を満たす幅広い中小企業を追跡します。これは、他の要因の中でも、パブリックフロート、時価総額、財務的実行可能性などの特定のメトリックによって決定されます。

## S＆P600を理解する

、どちらも小型株のパフォーマンスを測定するという点で[ラッセル2000インデックスに](/russell2000)匹敵しますが、前者ははるかに狭い範囲の資産を対象としています。このため、S＆P 600は、米国の投資可能な株式全体の約3％〜4％しか監視していません。このインデックスには、平均時価総額が15億ドルの601株がリストされています。

[S＆P 500](/sp500)または中型株[S＆P 400](/sp-midcap-400-index)インデックスと重複しないようにするために、8億5,000万ドルから37億ドルの間でなければなりません。

セクター別の内訳は、上場企業の大部分が金融、産業、情報技術、ヘルスケア、および消費者の裁量で運営されていることを示しています。ユーティリティや通信サービスでビジネスを行っている企業は最も少ない。

## S＆P600への投資

インデックスを直接売買することはできませんが、S＆P 600の取引を検討している投資家向けに、いくつか[の取引所取引ファンド](/etf)（ETF）が存在します。最もアクティブなファンドは、BlackrockのiShares、StateStreetのSPDRETF、およびVanguardを流れます。

投資家がこれらのファンドを選択する理由の1つは、小型株が提供する大きな上振れの可能性を捉えることです。真実は、より成功した企業の多くは、限界を超えて成長し、より大きなインデックスの1つの基準を満たすと、ベンチマークから除外されるということです。インデックスを離れるその他の理由には、合併または証券取引所からの上場廃止が含まれます。

次のETFは、S＆P 600のパフォーマンスを追跡しようとします（2022年7月2日現在のデータ）。

-** iShares Core S＆P Small-Cap ETF（IJR）：** 2000年5月22日に開始されたIJRは、S＆P 600の投資結果を受動的に複製しようとしています。ETFは610億ドルを超える資産を誇り、30- 1日の平均ボリュームは約440万ドルで、費用比率は0.06％です。市場価値別のETFの最大の保有量には、BlackRock Cash Fund、Southwestern Energy、Agree Realty、Omnicell、Rodgersが含まれます。

-** SPDRポートフォリオS＆P 600小型株ETF（SPSM）：** 2013年に開始されたSPSMは、S＆P 600に投資する別の受動的な方法を提供します。ETFは39億ドルを超える資産を保有し、費用比率は0.05です。 ％。 IJRと同様に、SPSMの最大の保有物には、Southwestern Energy、Agree Realty、Omnicellなどがあります。

-** Invesco S＆P SmallCap Value With Momentum ETF（XSVM）：** 2005年に開始されたXSVMは、投資家にS＆P 600への投資に対するより「アクティブな」アプローチを提供します。具体的には、インデックスはS＆Pにリストされた122の証券で構成されます「価値スコア」と「勢いスコア」が最も高い600。 ETFの市場価値は5億9,800万ドルを超え、費用比率は0.39％です。その上位の保有物は、フレッシュデルモンテプロデュース、コンズ、オリンピックスチール、クロスカントリーヘルスケア、およびグループ1オートモーティブです。

-** Invesco S＆P SmallCap Momentum ETF（XSMO）：** 2005年に発売されたXSMOは、S＆P600にリストされている117の証券で構成されています。 ETFの資産は約1億3800万ドルで、費用比率は0.39％です。その最大の保有物には、SM Energy、ServisFirst Bancshares、ExlService Holdings、Civitas Resources、およびCoca-Colaが含まれます。

## S＆P600に投資するメリット

###成長する余地

スモールキャップに投資する主な理由は明らかです。スモールキャップよりも成長の余地があります。 Microsoft、Apple、Wal-Martなどの巨人は、1,000億ドル以上の収益を生み出しています。当然のことながら、これら[の優良](/bluechip)企業にとって、売上を迅速かつ大幅に拡大することははるかに困難です。

一方、毎年売上高が2倍または3倍になっている小規模企業を見つけることは非常に一般的です。なんで？彼らははるかに小さな販売基盤から働いているからです。

言い換えれば、ソフトウェアスモールキャップが今後10年間で「マルチバガー」リターンを提供することは、マイクロソフトよりもはるかに簡単です。

###ウォールストリートがフォローしている

スモールキャップが優れているもう1つの大きな理由は、プロのアナリストがスモールキャップをそれほど厳密にフォローしていないことです。これにより、投資家はウォールストリートのレーダーの下で飛行している間にこれらの企業に「参入」することができます。

>大規模な投資信託は通常、たとえば会社の10％を所有したり、1つの会社の株式に資金の5％を使用したりすることを防ぐ「制限」ルールで投資する必要があります。したがって、数十億の資産を持つ相互資金の場合、小型株は単純に針を動かしません。

>>

###多様化

小型株のもう1つの大きなメリットは、[分散投資です](/diversification)。

小型株は中型株や大型株とは特性が異なります。それらは異なった振る舞いをします。したがって、小型株は、大幅な多様化のメリットをもたらす可能性があります。

## S＆P600の制限

小規模企業への投資は、大型株よりも高い潜在的利益をもたらす可能性がありますが、いくつかの課題もあります。

S＆P 600にリストされている企業の多くは、地理的なフットプリントが小さく、ドルが下がると苦しむ傾向があります。

理論的には、これにより投資家は、国内で所有する小規模な企業から購入するのではなく、海外で取引するようになります。収益の伸びに打撃を与えると、株価にも打撃を与える可能性があります。

スモールキャピタルは、より少ない販売基盤で働いているため、成長する余地がたくさんあります。これは、これらの企業が若いスタートアップであることが多いためです。また、若い新興企業には、実証されていないビジネスモデル、経験の浅い管理チーム、限られた財源があります。

>これらの要因のために、小規模企業の将来を予測することは、優良企業（毎年何十億ドルも稼ぐ）よりもはるかに困難です。

##小規模投資に関連するより多くのリスク

###揮発性リスク

[揮発](/volatility)性リスクとは、資産の価格が上下に変動する程度を指します。ボラティリティが高いほど、投資のリスクは高くなります。

中小企業の長期的な将来を予測することは難しいため、小型株は大型株よりも激しく変動する傾向があります。小型株は、市場全体よりも一貫して変動が激しい。したがって、小さなキャップにさらしたい場合は、乱気流をナビゲートするのに十分な時間の範囲があることを確認してください。

###ビジネスリスクとデフォルトリスク

ビジネスリスクとは、収益と収益の低下につながる要因に企業がさらされることです。[デフォルトリスク](/defaultrisk)とは、企業が債務を支払うことができない可能性を指します。

この点で、小規模企業はビジネスのリスクとデフォルトのリスクの両方を抱えています。なんで？

AppleやDisneyのような大規模な優良企業は、何十年にもわたって存続するための資金力とブランド力を持っていますが、小規模な新興企業は通常、実証されていない[ビジネスモデル](/businessmodel)、経験の浅い管理チーム、限られた財源を持っています。これにより、中小企業は、経済の低迷、コストの急上昇、大企業との激しい競争などの要因の影響をはるかに受けやすくなります。

>デビッドとゴリアテのサクセスストーリーごとに、小さな男が押しつぶされる状況ははるかに多くあります。小規模投資家として、あなたはその事実を受け入れ、数字ゲームをプレイすることができなければなりません。

>>

###流動性リスク

[流動性リスク](/liquidityrisk)とは、資産の価格に大きな影響を与えることなく、資産を迅速に売買できる範囲を指します。

中小企業は大企業ほど関心を引き付けないため、小型株は大型株より流動性が低くなります。価格に影響を与えることなく、いつでも大量のMicrosoft株式を売買するのは簡単ですが、小型株の場合はそれほど単純ではありません。

多くの場合、小型株は、購入したいときに十分な供給がないか、販売したいときに十分な需要がありません。その結果、投資家は小型株を予想よりも高い価格で購入し、低い価格で販売することになります。

###カバレッジの欠如

小型株は、主にウォールストリートのアナリストや[機関投資家からの報道が不足しているため、お金を稼ぐための途方もない隠れた機会を提供し](/institutionalinvestor)ます。

しかし一方で、このカバレッジの欠如は、小規模企業に関する質の高い調査や情報を入手することを困難にしています。結果は2つあります。

1.広くフォローされている大型株企業よりも、小型株企業の方が経営能力の欠如や非倫理的な行動に気付くのは難しい。

1.投資家は、情報に基づいた決定を行うために、大型株よりも小型株の分析にはるかに多くの時間と労力を費やす必要があります。

>小さなキャップのスペースに飛び込みたい場合は、情報の不足に伴う不確実性を受け入れることができなければなりません。

＃＃ よくある質問

### S＆P 500とS＆P 600の違いは何ですか？

S＆P 500インデックスは、米国で最大の500株のゲージです。一方、S＆P 600は、米国株の小型株の範囲をカバーしています。

### S＆P 600のティッカーシンボルとは何ですか？

S＆P600自体にはティッカーシンボルがありません。ただし、S＆P 600のパフォーマンスを追跡しようとするETFには、iShares Core S＆P小型株ETF（ティッカーシンボルIJR）およびSPDRポートフォリオS＆P 600小型株ETF（ティッカーシンボルSPSM）が含まれます。

###どのS＆Pインデックスが大型および中型企業を追跡していますか？

S＆P 500は、米国で最大の500の公営企業を追跡します。一方、S＆P 400は、米国で公的に取引されている中型株の最も広く使用されている指標です。
