# 標準産業分類（SICコード）
##標準産業分類（SIC）コードとは何ですか？

Standard Industrial Classification（SIC）コードは、米国政府によって割り当てられた4桁の数値コードであり、企業が属する業界を分類すると同時に、事業活動ごとに業界を整理します。 SICコードは、さまざまな業界や政府機関全体の経済活動を分類および分析し、さまざまな政府機関によって収集された統計データの表示の均一性を促進するために、1937年に米国政府によって作成されました。SICコードは、米国、英国を含む

[北米産業分類システム（NAICS）](/naics)と呼ばれる6桁のコードのシステムにほとんど置き換えられました。NAICSコードは、カナダ、米国間の業界データの収集と分析を標準化するために部分的に採用されました。北米自由貿易協定を締結した州およびメキシコ。

置き換えられたにもかかわらず、政府機関や企業は、現在でもSIC標準コードを使用して、企業の事業活動を類似の企業と照合することにより、企業が属する業界を分類しています。

##標準産業分類（SIC）コードを理解する

SICシステムは、経済を11の主要な部門に分類します。

-農業、林業、および漁業

-鉱業

- 工事

-製造

-運輸および公益事業

- 卸売業

- 小売業

-金融、保険、不動産

-サービス

- 行政

-分類できない施設

次に、これらは83の2桁の主要グループに分割され、さらに416の3桁の業界グループに分割され、さらに1,000を超える4桁の業界に分割されます。

すべての企業には、主要な事業を示す主要なSICコードがあります。 SICコードの最初の2桁は主要な業界グループを識別し、3桁目は業界グループを識別し、4桁目は特定の業界を識別します。

[証券取引委員会（SEC ）は](/sec)、市場を規制する主要な政府機関です。 SECは引き続きSICコードを使用しています。 SICコードは、企業の業界を示すために、企業の電子データ収集、分析、および取得システムのファイリング（EDGARと呼ばれる）にリストされています。

たとえば、Apple Inc.のSECファイリングでは、SICコードが3571（電子コンピュータ）として示されています。最初の2桁（35）は、主要な産業グループを「産業および商業機械およびコンピューター機器」として識別し、3桁目（7）は、その産業グループを「コンピューターおよびオフィス機器」に絞り込み、最後の桁（1）は分類します。それは「電子計算機」として。最初の2桁が20〜39の場合、これらの企業は製造部門または経済のセグメントに分類されることに注意してください。

##SICコードとNAICSコード

SICコードシステムは、引き続き最も人気のある業界分類システムです。 NAICSコードシステムは、SICシステムの長い歴史と、SICコードを長期間使用してきた企業やその他の組織が新しい分類システムに切り替えることを躊躇しているため、完全に置き換えることができませんでした。

米国政府は1987年までSICコードの更新を停止しましたが、民間のデータ組織が介入し、非常に具体的な分類の追加を含め、SICシステムの更新を続けました。その結果、最も定義されたレベルでは、現在10,000を超える6桁のSICコードがあります（1,066のNAICSコードと比較して）。

NAFTAの設立後、北米経済内での統合が進んだことで、米国企業は現在、SICコードとNAICSコードの両方を持っています。

##SICコードの使用方法

企業と政府は、以下で説明するように、さまざまな方法でSICコードを使用します。

###ビジネス

-企業はSICコードを使用して、既存の顧客と潜在的な顧客を業界別に識別します。

-SICコードは、税務上の目的で会社を分類するために使用できます。

-銀行と債権者は、SICコードを使用して、信用供与を検討する際に企業が属する業界を識別します。

-SICコードは、専門家や企業がターゲットを絞ったマーケティングキャンペーンを作成するために使用します。

-企業は、SICコードを介して類似企業を検索することにより、業界または地域の競争を特定できます。

-企業はまた、政府の契約を識別するためにSICコードを使用します。

＃＃＃ 政府

-政府はSICコードを使用して、さまざまな連邦および州の機関や民間企業の主要な経済データを整理および標準化しています。

-SICコードを使用すると、異なる政府機関間でデータを簡単に比較できます。

-SICコードは、税分類にも使用されます。

-SECは、会社の提出書類を分類するときにSICコードを使用します。
