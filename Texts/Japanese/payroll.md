# 給与
## Payrollとは何ですか？

給与とは、企業が一定期間または特定の日に従業員に支払わなければならない報酬です。通常、会社の経理部門または[人事部門によって管理されます。](/humanresources)中小企業の給与は、所有者または従業員が直接処理する場合があります。

給与は、給与処理、従業員給付、保険、および源泉徴収などの会計タスクを処理する専門会社にますます外部委託されています。 Atomic、Bitwage、Finch、Pinwheel、Wagestreamなどの多くのペイロール[フィンテック企業は、テクノロジーを活用してペイロールプロセスを簡素化しています。](/fintech)これらのソリューションは、従業員に利便性とスピードを提供し、デジタル給与関連のドキュメントに、ギグやアウトソーシング経済で必要とされる革新的なテクノロジー対応サービスを提供します。

給与は、会社の従業員のリストと各従業員への報酬額を参照することもできます。給与はほとんどの企業にとって主要な費用であり、ほとんどの場合[控除可能です。つまり、](/deductible)[総収入](/grossincome)から費用を差し引いて、会社の課税所得を下げることができます。残業、病欠、その他の変動要因により、給与は給与期間ごとに異なる場合があります。

##給与を理解する

[直接預金](/directdeposit)または小切手による支払いの分配が含まれます。ただし、企業は、給与、源泉徴収税、ボーナス、残業代、病欠、休暇手当を記録するための会計機能も実行する必要があります。企業は、メディケア、社会保障、失業税のために政府に支払われる金額を脇に置いて記録する必要があります。

多くの企業は、ソフトウェアソリューションを使用して給与を管理しています。従業員はAPIを介して時間を入力し、支払いは処理されて銀行口座に振り込まれます。

多くの中規模および大規模企業は、プロセスを合理化するために給与サービスを[アウトソーシングしています。](/outsourcing)雇用主は、各従業員が働いている時間数を追跡し、この情報を給与サービスに中継します。給料日に、給与サービスは、給料期間中の労働時間または週数と給料率に基づいて、従業員に支払うべき総額を計算します。このサービスは、所得から税金やその他の源泉徴収を差し引いてから、従業員に支払います。

##特別な考慮事項

[総売上高](/grosssales)が50万ドル以上の雇用主は、1938年に可決された[公正労働基準法](/fair-labor-standards-act-flsa)（FLSA）の要件の対象となります。これは、特定の不公正な賃金慣行から労働者を保護する米国の法律です。 FLSAは、[最低賃金](/minimum_wage)、残業代の要件、児童労働の制限など、さまざまな労働規制を定めています。たとえば、FLSAの規則では、労働者が時間どおりに考慮される時期と、残業代を支払う時期を指定しています。

法律は、残業（週に40時間を超える労働時間）を通常の時給の1.5倍で支払うことを義務付けています。一部の従業員はFLSAを免除されており、[独立した請負業者](/independent-contractor)やボランティアは従業員とは見なされないため、この法律は適用されません。

一部の時間給労働者はFLSAの対象外ですが、他の規制の対象となります。たとえば、鉄道労働者は鉄道労働法に準拠しており、トラック運転手は自動車運送業者法の範囲に含まれます。

FLSAはまた、主にチップによって補償される仕事をどのように扱うかを定めています。チップを持ったサービスワーカーの場合、従業員がチップから月額30ドル以上を定期的に受け取らない限り、雇用主は従業員に最低賃金を支払わなければなりません。

##プロの給与サービスを使用することの長所と短所

給与申告サービスの主な利点の1つは、会計手順を簡素化し、企業が法務および税務申告の要件に準拠していることを確認するのに役立つさまざまなレポートを作成できることです。給与サービスは、従業員が使用した休暇または個人的な時間の記録を保持する場合もあります。

不利な点に関しては、企業が給与システムを外部委託する場合、正確な会計を行うために事業外の個人に依存する必要があります。エラーが発生した場合、会社の現場担当者は動揺した従業員に対処する必要があります。企業はまた、給与サービスによって行われたエラーに対して税制上の罰則に直面する可能性があります。

[社内で](/in-house)給与を実行するよりも費用がかかることです。サービスは、設定された月額料金を請求するか、サービスのさまざまな層に対してさまざまな支払い構造を提供する場合があります。コストがかかるため、運用予算が厳しい中小企業にとって、給与サービスは最適なオプションではない場合があります。

>ビジネスが成長するにつれて、その会計上のニーズはより複雑になります。大企業は、会計および給与機能のためにカスタムエンタープライズリソースプランニング（ERP）システムに投資する必要があるかもしれません。

##給与ソフトウェアプログラム

一部の企業は、特殊な給与サービスを使用する代わりに、給与ソフトウェアプログラムに依存することを選択しています。会社がソフトウェアを購入すると、追加の月額料金は発生しません。ソフトウェアプログラムには通常、印刷可能な納税申告書と[源泉徴収](/withholding)表が含まれています。

財務上の節約に加えて、内部給与システムは、企業が機密の財務情報を非公開に保つのに役立ちます。ただし、ソフトウェアプログラムには時間がかかる可能性があり、スタッフが少ない中小企業にとっては問題となる可能性があります。

中小企業の所有者は、未収勘定と未払勘定を追跡し、収益性を測定し、納税シーズンの準備をするのに役立つため、会計ソフトウェアの恩恵を受けます。中小企業とは、大規模なカスタマイズを必要とせずに、すぐに使用できるソフトウェアを使用できる企業です。ビジネスが成長するにつれて、その会計上のニーズはより複雑になり、カスタムの[エンタープライズリソースプランニング](/erp)（ERP）システムが必要になることがよくあります。

中小企業が利用できるクラウドベースの会計ソフトウェアにはさまざまな種類があります。業界の種類と従業員数は、どの会計ソフトウェアが適切かを決定する2つの要因です。たとえば、[フリーランサー](/freelancer)は、レストランのオーナーと同じ会計ソフトウェアの機能を必要としません。

##給与税はどのように計算しますか？

給与税の計算方法は、ビジネスと地域の法律によって異なります。ただし、QuickBooksが提供する一般的なガイドラインは次のとおりです。最初のステップは、従業員の総賃金を計算することです。

###1.従業員の総賃金を計算します

従業員の総賃金は、従業員の賃金率とスケジュールされた賃金期間を使用して決定できます。ほとんどの企業は、毎週、2週間ごと、または毎月従業員に支払います。時間給の従業員の総賃金を計算するには、賃金期間に働いた時間に時給を掛けます。式は次のとおりです。

>>

>時給x給与期間中の総労働時間=総賃金

>>

給与のある従業員の総給与を計算するには、その年俸をその年の給与期間数で割ります。式は次のとおりです。

>>

>年俸/年の給与期間数=総給与

>>

例えば。従業員は年間50,000ドルを稼ぎます。彼らの会社は、2週間ごとに合計26の給与期間で従業員に支払います。したがって、従業員の総賃金は$1,923.08です。

###2.税引前控除を行う

総賃金を決定した後、控除額を除外する必要があります。これらは税額控除ですが、他の税引前控除も適用される場合があります。税引前控除には以下が含まれます。

（ [k）](/401kplan)およびいくつかの退職計画

-健康保険プラン

-Health [Savings Account](/hsa) （HSA）またはFlexible Spending Account（FSA）の寄付

-いくつかの生命保険プラン

### 3.税金を差し引く（FICA、失業、所得税）

税引き前の控除を取得すると、残りの支払いが課税されます。 FICAの税率は7.65％で、メディケアの場合は1.45％、社会保障税の場合は6.2％です。その他の税率は、連邦、州、または地方の法律と従業員のW-4によって決定されます。

IRS税テーブルを使用して連邦所得税を計算します。ほとんどの場合、社会保障税とメディケア税を支払うときに連邦税を支払うことになります。すべての支払いをIRSフォーム941で報告してください。

従業員の総賃金から7.65％のFICA税を差し引きます。あなたは雇用主として、各従業員の貢献と一致しなければなりません。ビジネスは、社会保障とメディケアへの従業員と会社の両方の貢献を提出します。

たとえば、従業員は、最新の給与期間の総給与で1,923ドルを稼ぎます。従業員の社会保障税負担を計算するには、$1,923に.062を掛けて$119.26を取得します。従業員のメディケア税負担を計算するには、$1,923に.0145を掛けて$27.88を取得します。合計すると、従業員のFICA税負担は、雇用主が一致しなければならない支払期間で147.14ドルです。この場合、雇用主はIRSに294.28ドルを支払う必要があります。半分は会社への直接経費であり、残りの半分は従業員の給与から差し引かれます。

雇用主は所得税控除と一致しませんが、連邦失業税を支払います。 IRSの所得源泉徴収アシスタントは、従業員が支払うべき連邦所得税の額を決定するのに役立ちます。

### 4.自発的な控除は、残りの賃金から取らなければなりません。

これらには次のものが含まれます。

-Roth [401（k）](/roth401k)の貢献

-生命保険プラン

-長期障害保険プラン

-賃金の飾り

-連合会費

すべての税金と控除の後、残りの金額は、従業員が給料日に家に持ち帰る金額です。

##結論

給与の処理は複雑で時間のかかる作業であり、連邦および州の厳格な規則や規制を順守する必要があります。広範囲にわたる記録管理と細部への注意が必要です。中小企業は、クラウドベースのソフトウェアを使用して独自の給与を処理することがよくあります。他の企業は、給与機能をアウトソーシングするか、全体的な会計と給与を管理する統合ERPシステムに投資することを選択します。

##ハイライト

-企業は、専門的なサービスを使用して給与を外部委託するか、自分でやりたくない場合はクラウドベースのソフトウェアを使用できます。

-給与の計算には多くの要素が含まれ、複雑になる可能性があります。

-ただし、企業は、会計、記録管理を実行し、メディケア、社会保障、および失業税のための資金を確保する必要があります。

-給与プロセスには、従業員の労働時間の追跡、賃金の計算、および直接預金または小切手による支払いの分配が含まれます。

-給与とは、企業が一定期間および特定の日に従業員に支払わなければならない報酬です。

＃＃よくある質問

###給与税とは何ですか？

給与税には社会保障が含まれ、これはあなたの収入の6.2％を最大$132,900まで引き出します。給与税は、あなたの収入の1.45％を占めるメディケアにも支払われます。雇用主は給与税も支払います。彼らはあなたの収入の6.2％を支払うので、政府はあなたの総収入の12.4％を受け取り、あなたの雇用主はあなたの収入の1.45％をメディケアに支払います。

###給与減税とは何ですか？

給与減税は、社会保障税とメディケア税が源泉徴収され、給与から差し引かれることを意味します。アイデアは、労働者と企業が給料ごとに少し余分に家に持ち帰り、それが彼らがより多くを費やして経済を刺激することを奨励するということです。

###給与税の休日とは何ですか？

給与税の休日は、後日まで給与税の徴収を延期することであり、その時点でそれらの税金が支払われることになります。給与税の繰り延べは、持ち帰り賃金を一時的に引き上げることにより、労働者に一時的な経済的救済を提供することを目的としています。
