# 個人保険（API）のアソシエイト
##個人保険（API）指定のアソシエイトとは何ですか？

[保険業界](/insurance)の専門資格です。その名前が示すように、APIは[個人保険](/personal-insurance-lines)に焦点を当てています。これは、人身傷害、死亡、または財産の損失から保護する保険の一種です。これは、保険専門家に認定と継続教育を提供することに専念している組織であるインスティテュートによって付与された多くの指定の1つです。

##API設計の仕組み

APIの指定は、個人保険の分野でキャリアを開始または前進させたい専門家の間で人気があります。 APIを取得することにより、専門家は、[保険引受](/underwriting)、[保険料設定](/insurance-premium)、マーケティング、ポートフォリオ管理などの重要なスキルを含む、非商業保険商品の知識を持っていることを証明できます。

[チャータードプロパティカジュアルティアンダーライター（CPCU）の指定](/chartered-property-casualty-underwriter-cpcu)など、より高度な認定を取得したい専門家にとって、APIは人気のあるオプションです。これは、APIを取得するために必要な条件の多くが、CPCUの要件にもカウントされるためです。具体的には、APIに関連する3つの試験のうち2つは、損害保険業界で最も権威のある指定の1つであるCPCU認定に再利用できます。

APIを取得するには、候補者は個人保険市場で販売されている保険契約の種類に関する幅広い知識を身に付ける必要があります。これには、契約の承認、条件と引受基準、業界規制などの科目に関する幅広い教育が含まれます。 APIは、個人保険の顧客にサービスを提供する保険ブローカー、エージェント、損害査定人、引受人、および顧客サービス担当者に推奨されることがよくあります。

##API指定の実際の例

APIを完了するときに候補者が取ることができる2つのパスがあります。これらの最初のものには、個人保険業界、ポートフォリオ管理、財産および[賠償責任保険、および個人保険における](/liability_insurance)[引受およびマーケティング慣行](/underwriting)に関するコースが含まれています。 2番目のパスは、引受とマーケティング、[リスク管理](/riskmanagement)と財産責任、および個人の財務計画をカバーする3つのコースで構成されています。

通常、候補者は、セルフサービスベースでアクセスされるこれらのコース資料を完了するのに12〜18か月かかります。対面での指導を好む方のために、一部の地域ではクラスをオフラインで利用できる場合もあります。試験に合格すると、APIの所有者は、指定を維持するために継続的な教育要件を満たす必要はありません。

##ハイライト

-APIは、個人向け保険業界で人気のある専門的な指定です。

-それは研究所によって管理されており、通常、完了するまでに12〜18か月かかります。

-多くの候補者は、CPCUなどのより高度なクレデンシャルを追求することにより、API指定に追加することを選択します。
