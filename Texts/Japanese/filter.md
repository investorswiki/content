# フィルター
##フィルターとは何ですか？

[の証券ユニバース](/universeofsecurities)内から選択するオプションの数を絞り込むために使用される基準です。このプロセスは、証券の[スクリーニング](/stockscreener)とも呼ばれます。したがって、「フィルター」および「スクリーン」という用語は、この文脈では同義語として使用できます。

##フィルターについて

使用される特定のフィルターは、問題の投資家の戦略によって異なります。たとえば、[バリュー投資家は、](/valueinvesting)[バランスシートの強さ](/balancesheet)や[収益の質](/qualityofearnings)など、問題の会社の[基本的な強みに](/fundamentals)関連する要素を使用する可能性があります。一方、[テクニカルアナリスト](/technicalanalysis)は、200日移動平均を上回っているか下回っているかなど、最近の価格履歴に関連する要因に関心を持っている可能性があります。

投資家にとって、フィルタリングプロセスは通常、投資家のスタイルや目的に明らかに適合しない企業を除外するように設計されたパラメータの一般的なセットから始まります。たとえば、外国株の下取りを望まない北米の投資家は、アメリカまたはカナダの証券取引所に上場している会社を除くすべての会社を除外することから始める場合があります。

これらの一般的なパラメーターが設定されると、投資家は、残りの企業が選択した投資戦略に厳密に一致するように、ますます具体的なフィルターを適用できます。

##フィルターの例

エマは、明確に定義された投資戦略を持つバリュー投資家です。彼女は、[株価純資産倍率](/price-to-bookratio)[（P / B）](/price-to-bookratio)が1.00以下で取引されているカナダとアメリカの配当支払い会社のみを購入することを目指しています。彼女は投資する$30,000を持っており、投資ごとに$ 1,000を割り当てて、30の持ち株の[ポートフォリオを作成しようとしています。](/portfolio)

彼女は検索を開始するために、オンラインの株式スクリーニングソフトウェアツールを使用して、カナダまたは米国で取引されていないすべての企業を排除します。これにより膨大な数の企業が作成されるため、彼女は結果をさらに洗練するために追加の要素を追加します。少なくとも1％の[配当利回りを提供しないすべての企業を除外します。](/dividendyield)結果として得られる1,500のリストは大幅に削減されますが、それでも彼女が探している30社よりはるかに多くなっています。

次のステップとして、彼女はP / Bフィルターを追加し、比率が1.00を超えるすべての企業を削除します。これによりリストがさらに減り、約250人の候補者が残ります。

この時点で、エマは彼女が進むことができる2つの方法があると考えています。 1つは、数が30候補のカットオフレベル近くまで減少するまでフィルターを追加することです。もう1つは、250人の候補者を、彼女の要因の1つ、または追加の要因の観点からランク付けすることです。

彼女は、P / B比率の観点からランク付けし、比率が最も低い30の候補を選択することにより、既存の250のリストから30の投資を選択することにしました。これにより、平均株価純資産倍率が0.40、平均配当利回りが9％の30の投資ポートフォリオが作成されます。

##ハイライト

-フィルターを使用すると、投資家は事前に調整された候補のリストから投資を選択できるため、時間を大幅に節約できます。

-投資家は、投資戦略に応じて、さまざまなタイプのフィルターを使用します。

-フィルターは、投資候補を絞り込む、つまり「選別」するために使用される基準です。
