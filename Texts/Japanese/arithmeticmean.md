# 算術平均
##算術平均とは何ですか？

[平均](/mean)または平均の最も単純で最も広く使用されている尺度です。それは単に数のグループの合計を取り、次にその合計をシリーズで使用される数の数で割ることを含みます。たとえば、34、44、56、および78の数値を考えます。合計は212です。算術平均は212を4で割った値（53）です。

人々はまた、[幾何平均](/geometricmean)や[調和平均など、他のいくつかのタイプの平均を使用](/harmonicaverage)します。これらは、金融や投資の特定の状況で機能します。もう1つの例は、消費者物価指数（CPI）や個人消費支出（PCE）などの経済データを計算するときに使用されるトリム平均です。

##算術平均のしくみ

算術平均は、財務においてもその位置を維持します。たとえば、平均収益の見積もりは通常、算術平均です。特定の株式をカバーする16人の[アナリストの平均](/analyst)[収益期待値](/earnings)を知りたいとします。すべての推定値を合計し、16で割ると、算術平均が得られます。

特定の月の株式の平均終値を計算する場合も同様です。その月に23の取引日があるとしましょう。単純にすべての価格を取得し、それらを合計し、23で割って算術平均を取得します。

算術平均は単純で、少しでも財務と数学のスキルを持っているほとんどの人がそれを計算できます。また、数値のグループが大きい場合でも、有用な結果を提供する傾向があるため、中心傾向の有用な尺度にもなります。

##算術平均の制限

算術平均は必ずしも理想的とは限りません。特に、単一の外れ値が平均を大幅に歪める可能性がある場合はそうです。 10人の子供たちのグループの許容量を見積もりたいとしましょう。それらのうちの9つは、週に10ドルから12ドルの間の手当を受け取ります。 10人目の子供は60ドルの手当を受け取ります。その1つの外れ値は、16ドルの算術平均になります。これはグループをあまり代表していません。

この特定のケースでは、許容値の[中央値10がより適切な測定値である可能性があります。](/median)

算術平均は、投資ポートフォリオのパフォーマンスを計算する場合、特に[複利計算](/compounding)、または配当と収益の再投資を伴う場合にも、大きくありません。また、アナリストが見積もりを行う際に使用する現在および将来の[キャッシュフロー](/cashflow)の計算にも一般的に使用されません。そうすることは、誤解を招く数字につながることはほぼ確実です。

>###重要

>算術平均は、外れ値がある場合、または過去の収益を見る場合に誤解を招く可能性があります。幾何平均は、系列[相関](/serial-correlation)を示す系列に最も適しています。これは特に投資ポートフォリオに当てはまります。

##算術vs。幾何平均

これらのアプリケーションでは、アナリストは異なる方法で計算される幾何平均を使用する傾向があります。幾何平均は、系列相関を示す系列に最も適しています。これは特に投資[ポートフォリオ](/portfolio)に当てはまり[ます](/portfolio)。

債券の利回り、株式のリターン、[市場リスクのプレミアムなど、金融のほとんどのリターンには相関関係があります](/marketriskpremium)。[期間](/timehorizon)が長くなるほど、より重要な複利計算と幾何平均の使用が必要になります。揮発性の数値の場合、幾何平均は、前年比の複利を考慮に入れることにより、真のリターンのはるかに正確な測定値を提供します。

幾何平均は、シリーズ内のすべての数値の積を取り、シリーズの長さの逆数に引き上げます。手作業では手間がかかりますが、MicrosoftExcelでGEOMEAN関数を使用して簡単に計算できます。

幾何平均は、期間ごとに発生する複利を考慮に入れるため、計算方法が算術平均または算術平均とは異なります。このため、投資家は通常、幾何平均を算術平均よりも正確な[リターンの尺度と見なします。](/return)

##算術との例幾何平均

過去5年間の株式のリターンが、20％、6％、-10％、-1％、および6％であるとしましょう。算術平均は、それらを単純に合計して5で割ると、年間平均収益は4.2％になります。

代わりに、幾何平均は（1.2 x 1.06 x 0.9 x 0.99 x 1.06）^ 1/5 ^ -1 = 3.74％/年の平均収益として計算されます。この場合、より正確な計算である幾何平均は、常に算術平均よりも小さいことに注意してください。

##ハイライト

-算術平均は、単純な平均、または一連の数値の合計をその一連の数値の数で割ったものです。

-金融でより一般的に使用される他の平均には、幾何平均と調和平均が含まれます。

-金融の世界では、算術平均は通常、平均を計算するための適切な方法ではありません。特に、単一の外れ値が平均を大幅に歪める可能性がある場合はそうです。
