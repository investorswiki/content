# 蓄積/分布インジケーター（A / D）
##蓄積/分布インジケーター（A / D）とは何ですか？

累積/分配インジケーター（A / D）は、在庫が蓄積されているか分配されているかを評価するために[ボリュームと価格を使用する累積インジケーターです。](/volume) A / D指標は、株価とボリュームフローの[相違を特定することを目的としています。](/divergence)これにより、トレンドがどれほど強いかについての洞察が得られます。価格が上昇しているが指標が下落している場合は、購入または累積量が価格上昇をサポートするのに十分ではなく、価格下落が来る可能性があることを示唆しています。

>###重要なポイント

>-累積/分配（A / D）ラインは、期間の範囲内で価格が閉じた場所を調べ、それをボリュームで乗算することにより、資産または証券の需要と供給を測定します。

>-A / Dインジケーターは累積的です。つまり、1つの期間の値が最後の期間から加算または減算されます。

>-一般に、上昇するA / D線は上昇傾向を確認するのに役立ち、下降するA/D線は価格の下降傾向を確認するのに役立ちます。

A/Dラインの計算方法

1.乗数を計算することから始めます。計算する最新の期間の[終値](/closingprice)、高値、および安値に注意してください。

1.乗数と現在の期間のボリュームを使用して、[マネーフロー](/moneyflow)ボリュームを計算します。

1.最後のA/D値にマネーフローボリュームを追加します。最初の計算では、最初の値としてマネーフローボリュームを使用します。

1.各期間が終了するたびにプロセスを繰り返し、前の合計に新しいマネーフローボリュームを加算/減算します。これはA/Dです。

##蓄積/分布インジケーター（A / D）は何を教えてくれますか？

[、需要と供給](/law-of-supply-demand)の要因が価格にどのように影響しているかを示すのに役立ちます。 A / Dは、価格変更と同じ方向または反対方向に移動できます。

計算の乗数は、特定の期間における売買の強さのゲージを提供します。これは、価格がその[範囲の上部または下部のどちらでクローズされているかを判断することによって行われます](/range)。次に、これにボリュームを掛けます。したがって、在庫が期間の範囲の上限近くでクローズし、ボリュームが大きい場合、A/Dジャンプが大きくなります。あるいは、価格がレンジの高値近くで終了するがボリュームが低い場合、またはボリュームが高ければ価格がレンジの中央に向かって終了する場合、A/Dはそれほど上昇しません。

価格が期間の価格範囲の下部で終了する場合も、同じ概念が適用されます。ボリュームと期間の範囲内で価格が閉じる場所の両方が、A/Dがどれだけ低下するかを決定します。

<！-2D20C6024E6E625AE9F8BCB335A76FC4-->

A / Dラインは、価格動向を評価し、今後の反転を発見するために使用されます。 A / Dラインが上昇トレンドにあるときに[証券の](/security)価格が[下降トレンドにある](/downtrend)場合、インジケーターは買い圧力があり、証券の価格が上昇に逆行する可能性があることを示します。逆に、A / Dラインが下降トレンドにあるときに証券の価格が上昇トレンドにある場合、インジケーターは売り圧力またはより高い分布がある可能性があることを示します。これは、価格が下落による可能性があることを警告しています。

[トレンド](/trend)への洞察を提供します。大幅に上昇しているA/Dラインは、価格が大幅に上昇していることを示しています。同様に、価格が下落し、A / Dも下落している場合でも、流通は十分にあり、価格は下落し続ける可能性があります。

##蓄積/分布インジケーター（A / D）対。オンバランスボリューム（OBV）

これらのテクニカル指標はどちらも、多少異なりますが、価格と出来高を使用します。[オンバランスボリューム](/onbalancevolume)（OBV）は、現在の終値が前回の終値よりも高いか低いかを調べます。終値が高い場合、その期間のボリュームが追加されます。終値が低い場合、その期間のボリュームが差し引かれます。

A / Dインジケーターは、前回の終値を考慮せず、期間の範囲内で価格が終値を付けた場所に基づいて乗数を使用します。したがって、インジケーターは異なる計算を使用し、異なる情報を提供する場合があります。

##累積/分布インジケーター（A / D）の使用に関する制限

A / Dインジケーターは、ある期間から次の期間への価格の変化を考慮せず、現在の期間の範囲内で価格が閉じる場所にのみ焦点を当てます。これにより、いくつかの異常が発生します。

大量の在庫が20％減少する[とします。](/gap)価格は1日を通して変動し、1日の範囲の上限で終了しますが、それでも前回の終値から18％下落しています。そのような動きは実際にA/Dを上昇させるでしょう。株価はかなりの価値を失いましたが、それはその日次範囲の上部で終了しました。したがって、ボリュームが大きいため、インジケーターはおそらく劇的に増加します。[トレーダー](/trader)は価格チャートを監視し、これらのような潜在的な異常をマークする必要があります。これらはインジケーターの解釈に影響を与える可能性があるためです。

また、インジケーターの主な用途の1つは、相違を監視することです。発散は長時間続く可能性があり、タイミング信号としては不十分です。指標と価格の間に相違が見られる場合、それは[逆転](/reversal)が差し迫っていることを意味するものではありません。価格が逆転するまでに長い時間がかかる場合や、まったく逆転しない場合があります。

A / Dは、トレンド内の強みまたは弱みを評価するために使用できる1つのツールにすぎませんが、欠点がないわけではありません。 A / Dインジケーターを、[米の](/price-action)[行動](/price-action)分析、チャートパターン、[ファンダメンタル分析](/fundamentalanalysis)などの他の形式の分析と組み合わせて使用すると、株価を動かしているものをより完全に把握できます。
