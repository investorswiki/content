# Ba2 / BB
## Ba2 / BBとは何ですか？

Ba2 / BBは、信用格付け機関が信用発行者または発行者に対して使用する格付けデザインであり、格付けスペクトルのデフォルトリスクが高いことを示しています。ムーディーズインベスターズサービスはBa2を使用し、S＆PグローバルレーティングスとフィッチレーティングスはBBを使用しています。

Ba2 / BBは投資適格を下回る格付けですが、非投資適格（[ジャンク](/junkbond)またはハイイールド）のブラケットで2番目に高い格付けです。

## Ba2/BBを理解する

Ba2 / BBレーティング、および機関によって設定された他のすべてのレーティングには、説明的なガイドラインがあります。格付けは、発行される信用商品と信用商品の発行者の両方に適用されます。

###問題のBa2/BB評価

ムーディーズの場合、Ba2と格付けされた問題は投機的であると判断され、かなりの信用リスクにさらされます。修飾子「2」は、義務がその一般的な格付けカテゴリーの中央にランク付けされていることを示します。つまり、Ba1より1ノッチ下[、](/ba1-bb) Ba3より1ノッチ上です。

S＆Pの場合、BBと格付けされた問題は「重要な投機的特性」を持ち、「そのような義務にはある程度の品質と保護特性がある可能性がありますが、大きな不確実性または悪条件への大きなエクスポージャーがこれらを上回る可能性があります」と見なされます。 S＆PのBB格付けは、BB +より1ノッチ低く、 [BB-より1ノッチ高くなって](/ba3-bb)います。

フィッチは、そのBB格付けは、問題がビジネスまたは経済の変化の影響を受けやすいことを示していると述べています。フィッチレーティングスシステムはS＆Pに準拠しており、BBはBB +より1ノッチ下、BB-より1ノッチ上です。

###発行者のBa2/BB評価

ムーディーズの場合、Ba2と評価された発行者は投機的であると判断され、「シニアの特定の営業義務およびその他の契約上のコミットメントをデフォルトするという実質的なリスクにさらされています」。

>非投資適格と見なされますが、Ba2 / BB格付けは、投機的クラスで2番目に高い格付けであり、Ba1 /BB+に次ぐものです。

>>

S＆P債券格付け機関の場合、BB格付けの債務者は、「重大な継続的な不確実性と、不利なビジネス、財務、または経済状況へのエクスポージャーに直面します。フィッチのシステムによると、BBの格付けは、特にビジネスや経済状況が変化した場合にデフォルトのリスクがあることを示していますが、ビジネスには現在の義務を果たす柔軟性があります。

##特別な考慮事項

企業が多くの目的のいずれかのために資金を調達するために債券を発行したい場合、通常、格付け機関のサービスを求めて、債券発行および発行者自体に関する信用意見を指定します。格付けは、債券が投資家に販売される際の[価格発見プロセスに役立ちます。](/pricediscovery)

Ba2 / BB格付けは、投資適格を下回るか、高利回りまたはジャンクと呼ばれることもあります。したがって、債券の利回りは、一般的に投資適格証券よりも高く、債券投資家が引き受けている支払い不履行のより大きなリスクを補います。

発行者と発行者の格付けは通常同じですが、たとえば、投資家向けの追加の信用保護によって発行が強化された場合、それらは異なる可能性があります。債券の格付けはBa1/BB +で、発行者はBa2/BBのままです。

##ハイライト

-ムーディーズはBa2格付けを使用し、S＆PとフィッチはBBを使用しています。

-Ba2 / BBは、投資適格をわずかに下回る信用格付けであり、より投機的であると考えられています。

-Ba2はBa3の評価を上回り、Ba1を下回りますが、BBはBB-を上回り、BB+を下回ります。

-企業は通常、投資家の透明性と価格発見を支援するために、新しい問題の格付けのために信用格付け機関のサービスを求めています。
