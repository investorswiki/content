# 相互通貨
##相互通貨とは何ですか？

[外国為替（外国為替）市場で](/foreign-exchange-markets)は、相互通貨は、通貨ペアが米ドル（USD）を含むが、USDが[基本通貨ではない状況を表し](/basecurrency)ます。代わりに、それは[見積もり通貨](/quotecurrency)（カウンター通貨とも呼ばれます）です。

したがって、相互通貨は、1ドルあたりの通貨の単位ではなく、外貨の単位あたりの米ドルで見積もられます。一般的な例は[EUR/USD](/eur-usd-euro-us-dollar-currency-pair)通貨ペアで、1.20の見積もりは、1ユーロが1.20米ドルを購入することを意味します。

##相互通貨を理解する

米ドル通貨ペアの大部分は、FX見積もりで最初に表示される基本通貨として米ドルを特徴としています。たとえば、USD / JPY（ドル対[日本円](/jpy-japanese-yen)）またはUSD / CAD（ドル対[カナダドル](/cad-canadian-dollar)）。

そのような見積もりでは、1米ドルで何単位の外貨を購入できるかがわかります。たとえば、イスラエルシェケル（USD / ILS）の見積もりが3.25の場合、1ドルで3.25シェケルを購入します。

ただし、逆数通貨は、一般に「ヨーロッパ」用語と呼ばれる用語で引用されます。つまり、米ドル以外の通貨が基本通貨です。

したがって、「相互通貨」は、米ドル（USD）と別の通貨がペアになっている外国為替市場で使用される通貨ペアを表しますが、USDは最初に見積もられる通貨ではありません。

>[直接](/directquote)相場は、外貨が国内通貨の単位当たりで表される通貨ペア相場です。米国の個人または企業の場合、相互通貨は直接見積もりの慣例になります。あるいは、[間接引用](/indirectquote)は反対の表現として表示されます。

##相互通貨の例

相互通貨の例は、NZD/USDの相場です。この通貨ペアの基本通貨[はニュージーランドドル](/nzd-new-zealand-dollar)、見積もり通貨は米ドルです。

言い換えれば、NZD/USDの[為替レート](/exchangerate)をニュージーランドドル対米ドルとして見積もることになります。したがって、NZD / USDの見積もりが0.70の場合、1つのNZDを70セントに交換できることを意味します。

>世界で最も取引されている通貨は、米ドル、ユーロ、日本円、英ポンド、オーストラリアドルです。

##結論

外国為替取引における相互通貨は、米ドルが基本通貨ではなく、カウンター通貨である通貨ペアです。これは、外貨の単位あたりの米ドルで見積もられています。相互通貨は世界的なFX取引の大部分を占めており、一般的な相互通貨はEUR / USD、GBP / USD、およびAUD/USDです。

##ハイライト

-相互通貨は、米ドルを基本通貨として使用せずに米ドル（USD）を含む通貨ペアです。

-USD/JPYとUSD/CADは、米ドルが基本通貨である例です。

-為替レートは、1を現在の為替レートで割ることで反転できます。

-NZD/USDとEUR/USDは、USDがこれらのペアの基本通貨ではないため、相互通貨ペアの例です。

-この見積もり表記は、米ドルが基本通貨として機能する場合よりも一般的ではなく、「ヨーロッパ」見積もりと呼ばれることもあります。

＃＃ よくある質問

###相互通貨の取り決めとは何ですか？

相互通貨協定は、互いの通貨の特定のマネーサプライを維持するための2つの国間の合意です。これにより、国家間および世界の金融市場における流動性が向上し、より効率的な金融取引が可能になり、預金準備率が維持され、為替レートが設定されます。相互通貨の取り決めは、スワップラインとも呼ばれます。

###相互為替レートをどのように見つけますか？

相互為替レートは、為替レートの逆数になります。逆の関係では、1を2つの通貨の現在の為替レートで割ります。したがって、たとえば、USD / EURの為替レートが0.89の場合、EUR / USDの逆数の為替レートを見つけるには、次の計算を実行します。1/0.89で1.12に到達します。

###通貨を見積もる2つの方法は何ですか？

通貨は、直接見積もりまたは間接見積もりとして見積もることができます。直接見積もりとは、外貨の1単位が国内通貨の対応する単位で見積もられる場合です。間接見積りとは、国内通貨の1単位を同等の外貨で見積もる場合です。

###通貨ペアとは何ですか？

通貨ペアは、2つの通貨の値を引用し、一方の通貨の値はもう一方の通貨の値に対して引用されます。ペアの最初の通貨は基本通貨であり、2番目にリストされている通貨は引用された通貨です。通貨ペアは、基本通貨の1単位を購入するために必要な見積もり通貨の量を示します。

###外貨オプションはどのように見積もられますか？

外国為替オプションは、本源的価値+時間価値として価格設定されています。これはFXオプションプレミアムです。本質的価値は、ストライクレートとフォワードレートを利用した換算通貨の差です。
