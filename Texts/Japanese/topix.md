# 東京物価指数（TOPIX）
##東京物価指数（TOPIX）とは何ですか？

東京物価指数（一般にTOPIXと呼ばれる）は、[東京証券取引所](/tokyo)（TSE）[の株価の指標です。](/tokyo) TOPIXは[時価総額加重指数](/capitalizationweightedindex)であり、TSEの「最初のセクション」にあるすべての企業を一覧表示します。このセクションは、取引所のすべての大企業を1つのグループに編成します。 TSEの2番目のセクションでは、残りの小規模企業をすべてプールします。

##東京物価指数（TOPIX）を理解する

東京証券取引所（TSE）は、日本最大の証券取引所[で](/exchange)あり、TOPIXの計算と発行を担当しています。日経平均株価、または日本の[日経225株価平均](/nikkei)と比較すると、TOPIXは価格変動のより公正な描写を反映し、東証で取引されている最大の企業を含むため、すべての日本の株式市場のより適切な表現と見なされます。それに比べて、日経は価格で加重されており、東証にリストされている上位225社[の優良企業のみで構成されています。](/bluechip)

##TOPIXのセクター指数

TOPIXは、基準日（1968年1月4日）の時価総額を100ポイントとした企業の[時価総額を示しています。](/marketcapitalization)この指標は、株式市場の全体的な傾向を判断するために使用され、投資家の[ベンチマークとして使用されます。](/benchmark)

の[セクター](/sector)指数は、TOPIXの構成要素を33のカテゴリーに分けて作成された[指数で構成されています。](/index)これらのカテゴリーは、証券コード協議会によって定義された産業部門に従って決定されます。それらには、建設、繊維およびアパレル、非鉄金属、機械、電力およびガス、陸上および航空輸送、小売業、銀行、証券および商品先物、不動産、保険、農林業、製薬、と鉄と鋼。

TOPIXには、TSEによっても公開されているいくつかの[サブインデックスがあります。](/subindex)これらには、TOPIX New Indexシリーズ、サイズベースのTOPIXサブインデックス、TOPIXセクターインデックス、東京証券取引所複合インデックスシリーズ、東京証券取引所配当フォーカス100インデックス、東京証券取引所REITプロパティセクターインデックスシリーズが含まれますが、これらに限定されません。 、および東京証券取引所マザーズインデックス。

##フリーフロートインデックスとしてのTOPIX

一連の3つのフェーズで、TOPIXは、発行済み株式の総数に基づいて企業を重み付けするシステムから、取引可能な株式の総数に基づいて企業を重み付けするシステムに移行しまし[た](/outstandingshares)。これは[フリーフロートとして知られてい](/freefloatmethodology)ます。 TOPIXの移行は2005年に始まり、2006年の夏に完了しました。

この移行は本質的に技術的なものですが、インデックスに記載されている企業の実際のウェイトに大きな影響を与えています。これは、日本の大多数の企業が、高度で強力な業務提携を維持するためにビジネスパートナーの株式を大量に[保有](/holdings)しており、これらの株式がインデックスの企業ウェイト計算に含まれなくなったためです。

>ビジネスパートナー同士が緊密な関係を築き、時には株式を取得するという日本の慣習は系列と呼ばれ[、](/keiretsu)文字通り「ヘッドレスコンバイン」を意味します。

##特別な考慮事項

いくつかのTOPIXインデックスの1つは、 [TOPIXCore30インデックス](/topix-core-30-index)です。時価総額加重指数は、時価総額が大きく、[最も流動性の高い](/liquidmarket)TOPIX指数の30銘柄のパフォーマンスを追跡します。指数は1998年4月1日に開始され、基準値は1,000でした。 TOPIX Core 30の企業リストには、Honda Motor、Canon、Mitsubishi Corp.、Sony、ToyotaMotorなどの日本で最も有名な名前が含まれています。

投資家はインデックスを直接購入することはできません。ただし、投資家がTOPIXのパフォーマンスを追跡する証券のバスケットに投資できるようにする[上場投資信託（ETF）があります。](/etf)野村アセットマネジメント会社は、TOPIX Core 30上場投資信託（ETF）を運営しています。ファンドの投資目的は、TOPIX Core 30と構成が類似しているポートフォリオを構築し、対応する価格と[利回り](/yield)のパフォーマンスを提供することです。

##ハイライト

--TOPIXは、時価総額で日本最大の企業を代表する取引所の最初のセクションで国内企業を追跡します。

-TOPIXとして知られる東京物価指数は、東京証券取引所（TSE）によって計算および公開されている日本の株式市場指数です。

-投資家はインデックスを直接購入することはできませんが、TOPIX Core 30 ETFなど、TOPIXのパフォーマンスを追跡する証券バスケットに投資できるようにする上場投資信託（ETF）がいくつかあります。

-もう1つの日本の株価指数は、日経平均株価指数です。これは、東証で取引されている優良株上位225社で構成される株価加重指数です。
