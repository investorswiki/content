# ビットコインの過剰主義
##ビットコインの過剰主義とは何ですか？

ビットコインの最大主義者は、世界で最も人気のある[暗号通貨である](/cryptocurrency)[ビットコイン](/bitcoin)が、将来必要となる唯一のデジタル資産であると信じています。マキシマリストは、他のすべての[デジタル通貨](/virtual-currency)はビットコインより劣っていると信じています。過剰主義のイデオロギーは、他の暗号通貨は、2009年にビットコインを作成した仮名[の中本聡によって確立された理想と一致していないという見解を持っています。](/satoshi-nakamoto)

ビットコインは、中央当局によって管理されている[法定通貨と呼ばれる政府発行の通貨とは異なります。](/fiatmoney)代わりに、ビットコインは分散型であり、その[ブロックチェーン](/blockchain)は公的に分散された元帳です。つまり、トランザクションは参加者と共有され、透過的です。

しかし、取引量の多いデジタル資産としてのビットコインの人気にもかかわらず、それは他の多くの暗号通貨の作成にもつながりました。ビットコインの最大主義者は、アルトコインと呼ばれるこれらの他の暗号通貨は不要で劣っていると信じて[い](/altcoin)ます[。](/altcoin)

##ビットコインの過剰主義を理解する

ビットコインは分散型暗号通貨の最初の試みではなかったかもしれませんが、これまでのところ間違いなく最も成功しています。ビットコインの最大主義者は、ビットコインネットワークが投資家が将来デジタル通貨で望むすべてのものを提供するという信念を持っています。このように、最大主義者は、将来のある時点でビット[コイン](/monopoly)[モノポリ](/monopoly)を支持します（または少なくともその必然性について合意します） 。

[イーサリアム](/ethereum)の開発者であるVitalikButerinは、2014年にビットコインの最大化のアイデアについてコメントしました。ブテリンはビットコインの最大化を「ビットコインの優位性の最大化」と表現しました。ブテリンはさらに、最大主義者の見方を説明しました。

>>

>複数の競合する暗号通貨の環境は望ましくない、「さらに別のコイン」を起動するのは間違っている、そしてビットコイン通貨が暗号通貨シーンで独占的な位置を占めるようになることは正当かつ不可避であるという考え。

>>

ブテリンは、そのように最大主義哲学を区別しました：

>>

>ビットコインをサポートしてより良くしたいという単純な願望。そのような動機は間違いなく有益です...むしろ、ビットコイン上に何かを構築することが物事を行う唯一の正しい方法であり、他のことを行うことは非倫理的であるというスタンスです。ビットコインの最大主義者はしばしば「ネットワーク効果」を議論として使用し、それらと戦うことは無駄であると主張します。

>>

###ビットコインのブロックチェーン

[分散型台帳テクノロジーは](/distributed-ledgers)、ビットコインのブロックチェーンネットワークの中心です。分散型台帳は、記録されたデータのコピーを共有ネットワーク内の参加者に送信することでトランザクションを共有できるため、有益です。結果として、この透明性はセキュリティを強化し、不正を防止するのに役立ちます。悪意のある攻撃者がブロックチェーンの1つを変更した場合、元のトランザクションのコピーを持っている他の参加者は、詐欺師によって何が変更されたかを判断し、トランザクションの正当な部分を復元できます。

しかし、ビットコイン（BTC）の人気は、暗号通貨の時代を迎え、他の何百ものデジタル通貨の作成につながりました。これらの暗号通貨の多くは、何らかの方法で基本的なビットコイン構造から構築されていますが、他のデジタル通貨はブロックチェーンテクノロジーに基づいていますが、必ずしもビットコインのネットワークに基づいているとは限りません。言い換えれば、ビットコインの分散型台帳は、当初意図されていたピアツーピアの現金支払いシステムだけでなく、他の目的にも使用できるように変更されています。

###変更されたブロックチェーン

ブロックチェーンテクノロジーの人気の高まりにより、プライベートブロックチェーンと呼ばれるビットコインの分散型台帳の修正バージョンが生まれました。企業や政府は、検証または認証された後、選択された数の参加者のみがネットワークへのアクセスを許可されるプライベートブロックチェーンネットワークを作成できます。

これらのプライベートブロックチェーンは、許可されたネットワークまたは半許可されたネットワークにすることができ、パブリックとプライベートの両方の特性を混在させることができます。これらのネットワーク内には、特定の参加者に付与された指定されたアクセス許可があり、ネットワーク上で特定のアクティビティのみを実行できます。ネットワークは、読み取り専用アクセスや編集アクセスなど、参加者が許可する機能に制限を課す場合もあります。セミプライベートブロックチェーンネットワークの例としては、特定の納税者や企業が法的な称号や記録管理にアクセスできるようにする一方で、一般の人々がそれらの記録にアクセスすることを制限する地方自治体があります。

パブリックブロックチェーンは分散化されています。つまり、監視権限を持つ集中化された権限はありませんが、プライベートブロックチェーンには、ネットワークを管理および制御する政府や企業などの集中化されたエンティティがあります。

ビットコインの最大主義者は、分散化がビットコインのブロックチェーンネットワークの重要な特徴であると信じているためです。プライベート、セミプライベート、および許可されたブロックチェーンの使用は、オープンで透過的であり、集中型のガバナンスがないという点で、暗号通貨の目的に反して実行されます。

##ビットコインの過剰主義の理由

マキシマリストは、他のすべてのデジタル通貨よりもビットコインを支持するビットコインサポーターの声のグループです。以下は、マキシマリストがビットコインが他のすべての暗号を無効にするだろうと信じている理由のいくつかです。

###ビットコインのネットワーク

今日の多くのビットコインマキシマリストは、デジタル通貨の成功は基盤となるブロックチェーンネットワークに依存しているという考えを支持しています。他のデジタル通貨は、ビットコインネットワークに固有の問題に対処するように設計された元のビットコインの前提に変更を加える可能性がありますが、成功の究極のマーカーはブロックチェーンの長さと強さであるという考えを聞くのが一般的です。ビットコインの基盤となるネットワークはそれなりに強力であるため、考え方は変わりません。また、特定のデジタル通貨の機能を別のデジタル通貨で自由に採用できるため、ネットワーク自体が最も重要な要素です。

マキシマリストは、この原則の証拠として、[時価総額](/marketcapitalization)によるデジタル通貨のリーダーボードにおけるビットコインと[ビットコインキャッシュの優位性を指摘する可能性があります。](/bitcoin-cash)ビットコインキャッシュとビットコインゴールドは、多くの新しいアルトコインと比較して機能が制限されています。ただし、アルトコインはビットコインネットワークに接続されているため、より高い価値を維持します。富、ユーザーベースのサイズ、成功の歴史は、ビットコインネットワークを他のブロックチェーンとは一線を画す機能です。

###ビットコインは十分に確立されています

過剰主義の視点に対する別の議論は、新しい金融商品は投資家の信頼を構築する上で高い障壁に直面しなければならないという原則です。デジタル通貨の人気が飛躍的に高まったとしても、市場から撤退することを好む多くの主要な金融機関や個人投資家がいまだに存在します。

ビットコインの最大主義者は、デジタル通貨を主流の金融と投資の世界に完全に統合するプロセスは遅いものになると信じています。その結果、部外者は最も古く、最も人気があり、最も確立されたネットワークに最も注意を払う可能性があります。デジタル通貨の場合、最も確立されているのはビットコインです。

数十の新しい、テストされていない新しいデジタル通貨で、ビットコインは信頼性と成功を証明しているという点で強力な利点があります。他の暗号通貨ネットワークがハッキングや他の否定的な宣伝に苦しんでいるとき、ビットコインの最大主義者はこれを彼らの議論を支持するさらなる証拠と見なす傾向があります。

###ビットコインのアルトコインに対する取引の影響

過剰主義哲学の最後の議論は、暗号通貨またはより広範なポートフォリオ内の多様化と関係があります。ビットコインの価格はアルトコインの世界の価格に広く影響を与える傾向があるため、アルトコインへの投資は、暗号通貨の保有を多様化するための疑わしい方法である可能性があります。

その場合、投資家は、他のコインやトークンに投資することでお金を危険にさらすよりも、ビットコインなどの最善の資産に投資するほうがよいという議論が続きます。ただし、ビットコインの価格の上昇は必ずしもアルトコインを高くしているわけではありませんが、最大主義者はそれがアルトコインの品質の低さによるものであると主張するかもしれません。

##ビットコインの過剰主義に関する懸念

ビットコインが唯一のデジタル通貨になるためには、ビットコインの最大主義には克服すべきハードルがあります。ビットコインネットワークとその暗号通貨の制限のために、多くのアルトコインとそれに続くブロックチェーンネットワークのバリエーションが生まれました。ビットコインの課題と制限には、次のものがあります。

###スケーラビリティ

ビットコインのような暗号通貨は、[プルーフオブワーク](/proof-work)（PoW）プロセスを使用して、ブロックチェーンで実行されたトランザクションを検証します。トランザクションを検証し、それらが正確であることを確認する責任がある人は、マイナーと呼ばれます。鉱夫は、取引の正当性を検証することでネットワークの監査人として機能し、不正行為の防止に役立ちます。

新しいトランザクションがブロックチェーンに追加されると、コピーがネットワーク上の参加者とコンピューターであるすべてのノードに送信されます。ただし、ビットコインの人気が高まるにつれて、トランザクションの量も増えます。ブロックチェーンネットワークを共有データベースと考えると、追加されるデータが多いほど、システムが停止し、遅延が発生します。

その結果、増大するトランザクションの量を処理するには、膨大な量のエネルギーが必要になります。たとえば、ビットコインブロックチェーンを保護するために必要な電力量は非常に大きくなっているため、2021年の時点で、パキスタンの国の総エネルギー消費量を超えています。

ビットコインのブロックチェーン内の遅延または低速が、暗号通貨のスケーラビリティを妨げています。言い換えれば、ビットコインの[スケーラビリティの](/scalability)問題は、ボリュームを処理できないため、金融取引での広範な使用に受け入れられないことです。その結果、他のブロックチェーンネットワークとその暗号通貨が必要になり、ビットコインの最大主義イデオロギーに穴を開けます。

###取引のボラティリティ

ビットコインが広く使用されている支払い方法になるためのもう1つの課題は、暗号通貨の価格が変動しすぎることです。これはボラティリティと呼ばれます。価格の変動が大きすぎると、企業や個人が日常のビジネス取引[の交換手段](/mediumofexchange)として暗号を使用することが困難になります。

###スマートコントラクト

初期のビットコインはその使用が制限されており、他のブロックチェーンがサポートするように特別に設計された[スマートコントラクト](/smart-contracts)と分散型アプリケーション（dApps）を構築するためのメカニズムを提供していませんでした。スマートコントラクトは、コンピューターコードで記述された、買い手と売り手の間の契約条件を含む自己実行型の契約です。デジタルコードは、取引の条件と実行を制御します。

スマートコントラクトを使用すると、自動車の購入や販売など、2者間で取引を行うことができます。その結果、契約は両方の当事者が契約で必要な義務を実行する場合にのみ実行可能であるため、一元化された権限は必要ありません。

イーサリアムブロックチェーンネットワークで使用されるスマートコントラクトは、金融セクターで人気を博しています。ビットコインのブロックチェーンネットワークはスマートコントラクトを提供することでその機能を強化しましたが、金融取引ではイーサリアムに遅れをとっています。

###代替ブロックチェーン

過去数年にわたって、ブロックチェーンネットワークは企業や業界によって確立されてきました。これらの代替ブロックチェーンネットワークは、今日一般的に取引されている暗号通貨を必ずしも必要としません。代わりに、これらの企業は、特定の参加者グループが個人的に使用する独自のネットワークと暗号通貨を作成しています。

たとえば、スイスユニオン銀行（UBS）が率いる銀行グループは、サンドボックスを開発しました。これにより、金融セクター内での支払いにブロックチェーンテクノロジーを使用する方法を探ることができます。そうすることで、UBSは、他の大手銀行と協力して、Utility Settlement Coin（USC）と呼ばれ、Fnalityプロジェクトとして知られる独自の暗号通貨を作成しました。

USCは、1対1で米ドルなどの法定通貨に変換できるため、現金と同様の方法で動作します。 USCはまた、ビットコインの暗号通貨とはまったく対照的な中央銀行によって支援されます。言い換えれば、金融セクターは、顧客、企業間の支払い、および銀行間の送金に使用できる独自のネットワークを作成することにより、ビットコインのブロックチェーンと暗号通貨をバイパスしました。

##ビットコインの過剰主義の未来

ビットコインのマキシマリストは、ビットコインブロックチェーンの問題は解決可能であり、現在開発中であると主張します。政府、企業、投資家がビットコインのブロックチェーンを選択するのか、他の多くのオプションを選択するのかによって、ビットコインの最大主義者が最終的に勝つかどうかが決まる可能性があります。ただし、他のネットワークや暗号への投資を考えると、今後数年間は他の多くの暗号通貨が存在するように見えます。

##ハイライト

-ビットコインの最大主義者は、ビットコインの欠点は解決できると主張する可能性がありますが、代替ブロックチェーンへの投資は増え続けています。

-ビットコインの最大主義者は、ビットコインが将来必要となる唯一のデジタル資産であると信じています。

-ビットコインの最大主義者は、他のすべてのデジタル通貨はビットコインより劣っていると信じています。

-ただし、ビットコインのスケーラビリティの問題により、トランザクション量の増加を処理できる他のブロックチェーンネットワークが開発されました。
