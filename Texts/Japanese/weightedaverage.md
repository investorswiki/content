# 加重平均
##加重平均とは何ですか？

加重平均は、データセット内の数値のさまざまな重要度を考慮した計算です。加重平均の計算では、最終的な計算が行われる前に、データセット内の各数値に所定の重みが掛けられます。

加重平均は、データセット内のすべての数値に同じ重みが割り当てられる単純な平均よりも正確な場合があります。

##加重平均を理解する

単純平均または算術平均の計算では、すべての数値が等しく扱われ、等しい重みが割り当てられます。ただし、加重平均は、各データポイントの相対的な重要性を事前に決定する重みを割り当てます。

加重平均は、ほとんどの場合、データセット内の値の頻度を等しくするために計算されます。たとえば、調査では、統計的に有効と見なされるのに十分な回答をすべての年齢層から収集できますが、18〜34歳の年齢層は、人口の割合に比べて他のすべての年齢層よりも回答者が少ない場合があります。調査チームは、18〜34歳のグループの結果に重みを付けて、彼らの見解が比例して表されるようにすることができます。

ただし、データセット内の値は、発生頻度以外の理由で重み付けされる場合があります。たとえば、ダンスクラスの生徒がスキル、出席、マナーで評価される場合、スキルの評価は他の要素よりも重要視される場合があります。

いずれの場合も、加重平均では、各データポイントの値に割り当てられた重みが乗算され、合計されてデータポイントの数で除算されます。

加重平均では、最終的な平均数は各観測値の相対的な重要性を反映しているため、単純な平均よりもわかりやすくなります。また、データを平滑化し、その精度を高める効果もあります。

###株式ポートフォリオの重み付け

投資家は通常、数年の期間にわたって株式のポジションを構築します。そのため、これらの株式のコストベースとそれらの相対的な価値の変化を追跡することは困難です。

投資家は、株式に支払われた株価の加重平均を計算できます。これを行うには、各価格で取得した株式数にその価格を掛け、それらの値を加算してから、合計値を合計株式数で除算します。

>加重平均は、各データポイントの相対的な重要性を事前に決定することによって得られます。

>>

たとえば、投資家が1年目に100株の会社を10ドルで取得し、2年目に同じ株の50株を40ドルで取得するとします。支払った価格の加重平均を取得するには、投資家は1年目は100株に10ドル、2年目は50株に40ドルを掛け、その結果を加算して合計3,000ドルになります。次に、株式に支払われた合計金額（この場合は$ 3,000）を、両方の年に取得した株式数150で割って、加重平均支払額$20を求めます。

この平均は、絶対価格だけでなく、各価格で取得された株式数に関して加重されています。

##加重平均の例

加重平均は、ポートフォリオの収益、在庫会計、評価など、株価のほかに多くの財務分野で表示されます。

複数の証券を保有するファンドが年間で10％上昇している場合、その10％は、ファンドの各ポジションの価値に対するファンドのリターンの加重平均を表します。

在庫会計の場合、在庫の加重平均値は商品価格の変動を考慮しますが、 [LIFO （後入れ先出し](/lifo)）または[FIFO](/fifo) （先入れ先出し）方式は値よりも時間の方が重要です。

自社株の価格が正しいかどうかを見極めるために企業を評価する場合、投資家は[加重平均資本コスト（WACC）を使用](/wacc)して企業のキャッシュフローを割り引きます。 WACCは、企業の資本構造における負債と株式の市場価値に基づいて加重されます。

##ハイライト

-加重平均は、単純な平均よりも正確な場合があります。

-加重平均は、データセット内のいくつかの要因の相対的な重要性または頻度を考慮に入れています。

-株式投資家は、加重平均を使用して、さまざまな時期に購入された株式のコストベースを追跡します。

###加重平均はどのように計算されますか？

加重平均は、相対的な比率またはパーセンテージにその値を順番に掛け、それらの合計を合計することで計算できます。したがって、ポートフォリオが55％の株式、40％の債券、および5％の現金で構成されている場合、これらのウェイトに年間パフォーマンスを掛けて、加重平均リターンを取得します。したがって、株式、債券、および現金がそれぞれ10％、5％、および2％を返した場合、加重平均収益は（0.55 x 10％）+（0.40 x 5％）+（0.05 x 2％）= 7.6％になります。 。

###財務で使用される加重平均の例は何ですか？

財務には、[ボリューム加重平均価格（VWAP）](/vwap) 、加重平均資本コスト（WACC）、グラフ作成に使用される[指数移動平均（EMA）など、多くの加重平均があり](/ema)ます。ポートフォリオの重みの構築、およびLIFOとFIFOの在庫方法も、加重平均を利用します。

###加重平均は単純平均とどのように異なりますか？

加重平均は、平均化されるものの相対的な寄与、つまり重みを説明しますが、単純な平均はそうではありません。したがって、平均して比較的多く発生するアイテムにより多くの価値をもたらします。
