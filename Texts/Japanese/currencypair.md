# 通貨ペア
##通貨ペアとは何ですか？

通貨ペアは、2つの異なる通貨の見積もりであり、一方の通貨の値が他方に対して見積もられます。通貨ペアの最初にリストされている通貨は[基本通貨と呼ばれ、2番目の通貨は](/basecurrency)[見積もり通貨](/quotecurrency)と呼ばれます。

通貨ペアは、ある通貨の価値を別の通貨と比較します。つまり、基本通貨（または最初の通貨）と2番目の通貨または見積もり通貨を比較します。これは、基本通貨の1単位を購入するために必要な見積もり通貨の量を示します。通貨は、 [ISO通貨コード](/isocurrencycode)、または国際市場で関連付けられている3文字のアルファベットコードによって識別されます。したがって、米ドルの場合、ISOコードはUSDになります。

##通貨ペアを理解する

通貨ペアの取引は、[外国為替市場](/foreign-exchange-markets)としても知られる外国為替市場で行われます。これは、金融業界で最大かつ最も[流動性の高い市場です。](/liquidmarket)この市場では、通貨の売買、交換、投機が可能です。また、国際貿易と投資のための通貨の変換を可能にします。外国為替市場は24時間年中無休（ほとんどの休日を含む）で開かれており、大量の取引量が見られます。

すべての外国為替取引には、ある通貨の同時購入と別の通貨の販売が含まれますが、通貨ペア自体は単一のユニット、つまり売買される商品と考えることができます。外国為替ブローカーから通貨ペアを購入するときは、基本通貨を購入し、見積もり通貨を販売します。逆に、通貨ペアを売るときは、基本通貨を売って見積もり通貨を受け取ります。

[ビッド](/bid)（買い）と[アスク](/ask)価格（売り）に基づいて見積もられます。入札価格は、外国為替ブローカーが見積もりまたはカウンター通貨と引き換えにあなたから基本通貨を購入する価格です。アスク（オファーとも呼ばれます）は、ブローカーが見積もりまたはカウンター通貨と引き換えに基本通貨を販売する価格です。

通貨を取引するときは、ある通貨を売って別の通貨を買うことになります。逆に、商品や株を取引するときは、現金を使ってその[商品のユニットまたは](/commodity)特定の株の[株の](/shares)数を購入します。金利や経済成長、[国内総生産（GDP）などの通貨ペアに関連する経済データ](/gdp)は、トレーディングペアの価格に影響を与えます。

##主要通貨ペア

広く取引されている通貨ペアは、米ドルに対するユーロ、または[EUR/USDとして表示され](/eur-usd-euro-us-dollar-currency-pair)ます。実際、最も取引量が多いため、世界で最も流動性の高い通貨ペアです。EUR/ USD = 1.2500の見積もりは、1ユーロが1.2500米ドルに交換されることを意味します。この場合、EURは基本通貨であり、USDは見積もり通貨（カウンター通貨）です。これは、1ユーロを1.25米ドルに交換できることを意味します。これを別の見方をすると、100ユーロを購入するには125ドルかかるということです。

世界の通貨と同じ数の通貨ペアがあります。存在する通貨ペアの総数は、通貨が出入りするにつれて変化します。すべての通貨ペアは、ペアのために毎日取引されるボリュームに従って分類されます。

米ドルに対して最も多く取引される通貨は、主要通貨と呼ばれ、次のものが含まれます。

-EUR/USDまたはユーロ対米ドル

-[米ドル/円](/usd-jpy-us-dollar-japanese-yen-currency-pair)またはドル対日本円

-GBP [/USD](/gbp-usd-british-pound-us-dollar-currency-pair)または英国ポンド対ドル

-USD [/CHF](/usd-chf-us-dollar-swiss-franc-currency-pair)またはスイスフラン対ドル

-AUD [/USD](/aud-usd-australian-dollar-us-dollar-currency-pair)またはオーストラリアドルと米ドル

-USD [/CAD](/usd-cad-us-dollar-canadian-dollar-currency-pair)またはカナダドルと米ドル

最後の2つの通貨ペアは、カナダとオーストラリアの両方が商品が豊富であり、両方の国がそれらの価格の影響を受けるため、商品通貨として知られています。主要通貨ペアは最も流動性の高い市場を持ち、月曜日から木曜日まで24時間取引される傾向があります。通貨市場は日曜日の夜に開き、金曜日の米国東部時間午後5時に閉じます。

##マイナーとエキゾチックなペア

米ドルに関連付けられていない通貨ペアは、マイナー通貨または[クロスと呼ばれ](/crosscurrency)ます。これらのペアはわずかに広いスプレッドを持ち、メジャーほど流動的ではありませんが、それでも十分に流動的な市場です。最も取引量の多いクロスは、個々の通貨もメジャーである通貨ペアの中にあります。クロスの例としては、EUR / GBP、GBP / JPY、EUR/CHFなどがあります。

エキゾチックな通貨ペアには、[新興市場の通貨が含まれます](/emergingmarketeconomy)。これらのペアは流動性が低く、スプレッドははるかに広くなっています。エキゾチックな通貨ペアの例は、USD / SGD（米ドル/シンガポールドル）です。

##ハイライト

-通貨ペアの注文時に、最初にリストされた通貨または基本通貨が購入され、通貨ペアまたは見積もり通貨の2番目にリストされた通貨が販売されます。

-通貨ペアは、FX市場で取引される2つの異なる通貨の為替レートの価格見積もりです。

-EUR / USD通貨ペアは、世界で最も流動性の高い通貨ペアと見なされています。 USD / JPYは、世界で2番目に人気のある通貨ペアです。
