# フェザーベッド
##フェザーベッドとは何ですか？

フェザーベッドという用語は、労働組合の規制に準拠するために雇用主が労働力を変更することを要求する[労働組合の慣行を指します。](/labor-union)組合が羽毛布団に従事するとき、企業は一般にこれらの要求を満たすために[人件費を増やすことを余儀なくされます。](/cost-of-labor)これは、必要以上に多くの労働者を雇用するか、契約条項を満たすために生産を制限するという形でもたらされる可能性があります。

##フェザーベッドのしくみ

フェザーベッドは、北米で一般的に使用されている口語的な用語であり、英国での無人化に似ています。これは、労働組合が特定のタスクを完了するために必要以上に人件費を増やすことを雇用主に要求する場合に発生します。

フェザーベッドは、多くの場合、必要以上に、雇用主に追加の従業員を雇うことを要求するという形をとります。また、人件費を増加させる時間のかかる作業ポリシーと手順を追加したり、企業の生産レベルと全体的な[生産](/productivity)性を低下させる慣行を採用したりすることも意味します。

フェザーベッドは、不要になった従業員が組合によって保持される必要がある場合、または組合が雇用主に特定の職務に過剰な資格を持つ労働者を雇用することを要求した場合にも発生します。

>羽毛布団は、技術の進歩と発展に直面して人々を雇用し続けるための組合の方法として浮上してきました。

>>

[産業](/industry)が生産性を高めるための技術的進歩を発展させ、実施するにつれて、組合が労働者を維持する方法として浮上してきました。羽毛布団はしばしば否定的な見方で描かれるため、組合は通常その存在を否定しますが、一部の[エコノミスト](/economist)は、この慣行が実際には組織からの[余剰利益を失業している従業員に再分配するのに役立つと主張しています。](/surplus)

[効率](/efficiency)によって時代遅れになったものを促進すると主張している。

##特別な考慮事項

米国議会は、1935年に全国労働関係法（NLRA）を施行するために全国労働関係委員会（NLRB）を設立しました。この法律は、雇用者と労働者の両方の権利と利益を保護するために同年に成立しました。 NLRBは、NLRAの違反者に、雇用主または労働組合の誰であるかに関係なく、不当労働行為をやめるよう命じる権限を与えられています。

NLRBはまた、犯罪者に、金銭的補償を通じて不法行為によって被害を受けた従業員または団体に救済を提供するよう指示する場合があります。

NLRAは、雇用者と労働組合または従業員のグループとの間で行われる[団体交渉を奨励](/collective-bargaining)し、雇用条件を交渉し、[民間部門での不当労働行為を削減することによって労働者の権利を保護します](/private-sector)。 NLRAは、1947年のタフト[ハートリー法](/tafthartleyact)または労使関係法によって改正されました。タフトハートリー法は、労働組合の活動を制限し、管轄ストライキ、ワイルドキャットストライキ、二次ボイコット、閉店、金銭などの戦術を禁止しました。連邦の政治キャンペーンへの組合による貢献。

フェザーベッドは、法律のセクション8（b）（6）で具体的に取り上げられており、次のように書かれています。

>>

>組合は、実施されていないサービスの支払いを求めることはできません。

>>

>法のセクション8（b）（6）は、労働組織またはその代理人が「雇用者に金銭またはその他の価値のあるものを支払う、または提供する、または支払うまたは提供することに同意するようにさせる、または試みることを違法にしている。実行されない、または実行されないサービスの厳密さの性質。」

>>

このセクションでは、実行されていない作業または実行されることを意図していない作業に対して雇用主が支払うことを引き起こす慣行を具体的に禁止しますが、不要な実行されたサービスの[支払いを確保することは禁止しません。](/payment)

この規定は米国最高裁判所によって狭義に解釈されており、NLRAは、労働組合が実施されていない、または実施されないサービスの見返りとして雇用主からの支払いを厳密に行う状況のみを制限すると裁定しました。組合は、たとえ同じ時間内に同じように仕事をすることができた従業員が少なくても、雇用主の同意を得て、実際に従業員が行った仕事の支払いを要求することがあります。

##ハイライト

-羽毛布団の下では、企業は一般的にこれらの要求を満たすために人件費を増やすことを余儀なくされています。

-フェザーベッドは労働組合の慣行であり、雇用主は労働組合の規制に準拠するために労働力を変更する必要があります。

-雇用主は、必要以上に多くの従業員を雇用したり、人件費を増加させる時間のかかるポリシーや手順を追加したり、生産性を低下させる慣行を採用したりする必要がある場合があります。
