# USD / CHF（米ドル/スイスフラン）
## USD / CHF（米ドル/スイスフラン）とは何ですか？

USD / CHFは、米ドルとスイスフランの通貨ペアです。通貨ペアは、1米ドル（[基本通貨](/basecurrency)）を購入するために必要なスイスフラン（見積もり通貨）の数を示します。 USD / CHF通貨ペアの取引は、「スイス」の取引とも呼ばれます。

## USD / CHF（米ドル/スイスフラン）を理解する

CHFはスイスフランの一意の[通貨コード](/isocurrencycode)であり、USDは米ドルの通貨コードです。通貨はペアで見積もられ、一方の通貨のうち、もう一方の通貨を購入するのにかかる費用がわかります。

USD / CHFの価値は、1米ドルを購入するのに必要なフラン数です。たとえば、ペアが1.05で取引されている場合、1米ドルを購入するには1.05スイスフランが必要であることを意味します。レートが0.9850の場合、1米ドルを購入するのに0.9850フランかかることを意味します。

USD / CHFは、米ドルまたはスイスフランの相互および他の通貨との関係に影響を与える要因の影響を受けます。両国の雇用データと国内[総生産](/gdp)[（ GDP）は、通貨ペアに大きな影響を与えるいくつかの経済指標です。](/gdp)

[連邦準備制度（Fed）](/federalreservebank)と[スイス国立銀行](/swiss-national-bank)（SNB）の金利差も、この通貨ペアに影響を与えます。たとえば、FRBが[公開市場操作](/openmarketoperations)に介入して米ドルを上昇させると、スイスフランと比較して米ドルが上昇するため、USD/CHFの価値が上昇する可能性があります。

一方、スイス国立銀行が金利を引き上げた場合、それはより多くの投資家をフランに誘い込み、その価値を高める可能性があります。この場合、米ドルを購入するのに必要なフランが少なくなるため、米ドル/スイスフランのレートは下がります。

、EUR / USD（ユーロ/ USD）およびGBP / USD（英国ポンド/ USD）の通貨ペアと負の[相関関係がある](/correlation)傾向があります。これは、[ユーロ](/euro)、スイスフラン、英国ポンドの正の相関関係によるものです。

## USD/CHFの最近の価格履歴

スイスフランは安全な避難所の通貨です。つまり、世界的な経済的ストレスやボラティリティの高い時期には、フランはしばしば高く評価されます。スイスフランと[日本円](/jpy-japanese-yen)は、最も人気のある2つの安全な通貨取引と見なされています。これは、スイスが財政的および政治的に安定していると広く見なされているためです。

[大不況](/great-recession)の初期段階では、スイスフランは日本円を除くすべての主要な貿易相手国に対して高く評価されました。 2007年の初めから2008年半ばにかけて、米ドルはCHFに対して下落しましたが、2008年後半に米国株式市場での売りが拡大するにつれて、より多くの投資家が米ドルに逆流し始め、CHFよりも比較的安全な賭けと見なしました。

2009年以降、米ドルは再びCHFに対して下落し、ペアは2011年の安値である0.7066まで下落し続けました。その後、米ドルは上昇の勢いを増し、ペアは2012年から2021年の間に0.83から1.0344の間で取引されました。

2015年、SNBがEUR / CHFペアに設定したフロアを引き上げたとき、USD/CHFは1.20近くで取引されていました。 EUR / CHFは急落し、USD/CHFも0.83に急落しました。多くの世界的な外国為替ブローカーが破産したか、トレーダーの損失のために救済を必要としたため、急速な急落は通貨市場に大規模な激変を引き起こしました。この事件は、主に多くの国で利用可能な[レバレッジ](/leverage)の低下を中心に、広範な改革を引き起こしました。

## USD/CHFの価格変動の解釈

USD / CHFのレートが0.90で、レートが1.05まで上昇した場合、1米ドルを購入するのにより多くのCHFがかかるため、USDはCHFに対して高く評価されています。

一方、レートが1.03から0.99に下がると、1米ドルを購入するのにかかるCHFが少なくなるため、CHFは高く評価されるか、USDはCHFに比べて下落します。

USD / CHFレートは、1米ドルを購入するのに何スイスフランかかるかを示していますが、スイスに向かう旅行者は、1スイスフランを購入するのに何米ドルかかるか知りたい場合があります。これには、CHF / USDレートを知る必要があります（コードが反転します）。 CHF / USDレートを取得するには、1をUSD/CHFレートで割ります。

たとえば、USD / CHFレートが0.9350の場合、CHF / USDレートを取得するには、1を0.9350で除算します。結果は1.0695のCHF/USDレートです。この料金は、1スイスフランを購入するのに1.0695米ドルかかることを旅行者に知らせます。これは、USD / CHFレートから、1ドルの購入にかかる費用が1フラン未満であるため、CHFの方がUSDよりも価値があることがわかっているため、理にかなっています。

旅行中または物理的な通貨を取得するときは、銀行や[外貨両替所](/currency-exchange)は通常3〜5％を請求し、これを通貨の価格に反映させることに注意してください。したがって、スイスに向かう旅行者は、CHFごとに1.0695米ドルのリアルタイムの外国為替市場レート（0.9350のUSD / CHFレートに相当）を取得できない可能性があります。代わりに、為替レートは追加の4％を請求する可能性があります。たとえば、1スイスフランあたりのレートは1.0695ドルではなく1.1123ドルになります。

##ハイライト

-USD / CHFは米ドルとスイスフランの通貨であり、レートは1米ドルを購入するのに必要なフラン数を反映しています。

-USD / CHFは、EUR/USDおよびGBP/USDと負の相関関係にある傾向があります。

-スイスフランは、国の安定した政治的および財政的地位のために、しばしば安全な避難所と見なされています。
