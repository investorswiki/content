# 総付加価値（GVA）
##総付加価値（GVA）とは何ですか？

総付加価値（GVA）は、企業の子会社、企業、または自治体の経済、生産者、セクター、または地域への貢献度を測定する経済[生産性指標です。](/productivity)

から、その生産に直接起因するすべての投入物と[原材料](/rawmaterials)のコストを差し引いた金額を提供します。したがって、GVAは、製品に対する補助金と税金（関税）の影響によって[国内総生産（GDP）を調整します。](/gdp)

##総付加価値（GVA）を理解する

GVAは、国の生産量から中間消費量を差し引いたものであり、総生産量と純生産量の差です。 GVAは、国の[経済](/economy)全体の状態を示す重要な指標であるGDPの計算に使用されるため、重要です。また、特定の地域、州、または州からどれだけの価値が追加（または失われる）されるかを確認するためにも使用できます。

[総国家生産物](/gnp)（GNP）に対する総経済生産高および成長の尺度として好まれることがあります。 GVAは、製品に対する税金と製品[に対する補助金](/subsidy)を通じてGDPに関連しています。これは、政府が経済の特定のセクターに付与する補助金を追加し、他のセクターに課せられる税金を差し引きます。

企業レベルでは、このメトリックは、企業が現在生産または提供している特定の製品、サービス、または企業単位によってGVAを表すために計算されることがよくあります。[固定資本](/fixed-capital)の消費と[減価償却](/depreciation)の影響を差し引くと、会社は特定の事業が収益にどれだけの正味価値をもたらすかを知ることができます。言い換えると、GVA番号は、その特定の製品が会社の利益に貢献したことを示しています。

##総付加価値（GVA）の例

-個人消費=5,000億ドル

-総投資額=2500億ドル

-政府投資=1500億ドル

-政府支出=2500億ドル

-総輸出額=1500億ドル

-総輸入額=1250億ドル

-製品の総税額=10％

-製品への補助金の合計=5％

このデータを使用して、GVAを計算できます。最初のステップはGDPを計算することです。 GDPは、個人消費+総投資+政府投資+政府支出+（[輸出](/export)-輸入）として計算されることを思い出してください。

-** GDP ** =5,000億ドル+2,500億ドル+1500億ドル+2500億ドル+（1500億ドル-1250億ドル）=1.175兆ドル

次に、製品に対する補助金と[税金を計算](/tariff)します。簡単にするために、すべての個人消費は製品の消費であると仮定します。その場合の補助金と税金は次のとおりです。

-**製品への補助金**=5,000億ドルx5％=250億ドル

-**製品の税金**=5,000億ドルx10％=500億ドル

これにより、GVAは次のように計算できます。

-**総付加価値**=1.175兆ドル+250億ドル-500億ドル=1.15兆ドル

##ハイライト

-総付加価値（GVA）は、企業の子会社、企業、または自治体の経済、生産者、セクター、または地域への貢献度を測定する経済生産性指標です。

-GVAは、国の経済全体の状態の重要な指標であるGDPを調整するために使用されるため、重要です。

-GVAは、国の生産量から中間消費量を差し引いたものであり、総生産量と純生産量の差です。

-また、製品またはサービスが企業の固定コストを満たすためにどれだけのお金を提供したかを測定するために使用することもできます。
