# ロボアドバイザー
##ロボアドバイザーとは何ですか？

ロボアドバイザーは、人間による監督がほとんどまたはまったくない、自動化されたアルゴリズム主導のファイナンシャルプランニングサービスを提供するデジタルプラットフォームです。典型的なロボアドバイザーは、オンライン調査を通じてあなたの財政状況と将来の目標について質問します。次に、データを使用してアドバイスを提供し、自動的に投資します。

最高のロボアドバイザーは、簡単なアカウント設定、堅牢な目標計画、アカウントサービス、およびポートフォリオ管理を提供します。さらに、セキュリティ機能、行き届いたカスタマーサービス、包括的な教育、および低料金を提供します。

##ロボアドバイザーを理解する

2008年に立ち上げられた最初のロボアドバイザーであるBettermentは、当初は[目標日ファンド内の資産のバランスを取り直すことを目的としていました](/target-date_fund)。シンプルなオンラインインターフェースを通じて、受動的なバイアンドホールド投資の管理を支援しようとしました。 2022年、ベターメントは、投資家への提供を拡大するために暗号通貨ポートフォリオを構築および維持するロボアドバイザープラットフォームであるマカラを買収しました。

テクノロジー自体は新しいものではありませんでした。ヒューマンウェルスマネージャーは、2000年代初頭から、自動化された[ポートフォリオ配分](/assetallocation)ソフトウェアを使用してきました。しかし、Bettermentがリリースされるまで、テクノロジーを購入できるのは彼らだけだったため、クライアントはイノベーションの恩恵を受けるためにファイナンシャルアドバイザーを雇わなければなりませんでした。

[現代ポートフォリオ理論](/modernportfoliotheory)（MPT）のいくつかの変形を使用して最適化されたパッシブインデックス戦略を使用しています。一部のロボアドバイザーは、社会的責任投資（SRI）、ハラール投資、またはヘッジファンドを模倣する戦術戦略のために最適化されたポートフォリオを提供します。さらに、[減税の収穫](/taxgainlossharvesting)、投資の選択、[退職後の](/retirement-planning)[計画](/retirement-planning)など、はるかに高度なタスクを処理できます。

> 2021年、資産の面で最大のロボアドバイザーはVanguard Personal Advisor Servicesであり、運用資産（AUM）は2,310億ドルでした。

>>

業界は爆発的な成長を遂げました。ロボアドバイザーが管理する顧客資産は2020年に約1兆ドルに達し、2025年までに世界中で2.9兆ドルに達すると予想されています。

ロボアドバイザーのその他の一般的な呼称には、「自動投資アドバイザー」、「自動投資管理」、「デジタルアドバイスプラットフォーム」などがあります。名前に関係なく、それはすべて投資管理のための[fintech （金融テクノロジー）アプリケーションを](/fintech)指します。

##ポートフォリオのリバランス

ロボアドバイザーの大多数は、現代ポートフォリオ理論（またはいくつかの変形）を利用して、ユーザー向けのパッシブなインデックス付きポートフォリオを構築しています。一度確立されると、ロボアドバイザーはこれらのポートフォリオを監視し続け、市場が動いた後でも最適な資産クラスの重み付けが維持されるようにします。ロボアドバイザーは、リバランスバンドを使用してこれを実現します。

すべての[資産クラス](/assetclasses)、または個々のセキュリティには、目標の重みと対応する許容範囲が与えられます。たとえば、配分戦略には、新興市場の株式で30％、国内の優良株で30％、[国債](/government-bond)で40％を保有し、各資産クラスの回廊が±5％であるという要件が含まれる場合があります。

>これまで、このタイプのリバランスは時間がかかり、取引手数料が発生する可能性があるため、嫌われていました。ただし、ロボアドバイザーはこれを低料金で自動的に行うように設計されています。

>>

リバランスバンドを使用するということは、新興市場と国内の優良株の保有が25％から35％の間で変動する可能性がある一方で、ポートフォリオの35％から45％を国債に割り当てる必要があることを意味します。持ち株のウェイトが許容範囲外に跳ね上がると、ポートフォリオ全体が最初のターゲット構成を反映するように再調整されます。

ロボアドバイザーで一般的に見られ、アルゴリズムによって費用効果が高くなる別のタイプのリバランスは、減税の収穫です。税務上の損失の収穫は、同様の証券のキャピタルゲイン税債務を相殺するために損失で証券を売却することを含む戦略です。

この戦略は通常、短期のキャピタルゲインの認識を制限するために採用されます。ロボアドバイザーは、資産クラスごとに2つ以上の安定した上場投資信託（ETF）を維持することによってこれを行います。したがって、S＆P 500 ETFが価値を失った場合、キャピタルロスを固定するために自動的に売却されます。同時に、別のS＆P500ETFを購入します。

[ウォッシュセール](/washsale)違反を回避するために、ロボアドバイザーがETFを適切に選択するようにプログラムされていることを確認してください。

>>

##ロボアドバイザーを使用する利点

ロボアドバイザーの主な利点は、従来のアドバイザーに代わる低コストの代替手段であるということです。人的労力を排除することにより、オンラインプラットフォームはわずかなコストで同じサービスを提供できます。ほとんどのロボアドバイザーは、管理される特定の金額ごとに0.5％未満の年間定額[料金を請求します。これは、人間のファイナンシャルプランナーが請求する通常の1％から2％よりもはるかに少ないです（](/feebasedinvestment)[手数料ベース](/commission-broker)のアカウントの場合はそれ以上です）。

ロボアドバイザーもよりアクセスしやすくなっています。インターネットに接続している限り、24時間年中無休で連絡できます。さらに、アカウントの登録に必要な最小資産は通常数百から数千であるため、開始するのに必要な資本は大幅に少なくなります（$ 3,000〜 $ 5,000が標準のベースラインです）。最も人気のあるロボアドバイザーの1つであるBettermentには、標準的なサービスの最低アカウントがありません。

>多くの人間のアドバイザーは、投資可能な資産、特に現場で確立された資産に10万ドル以上の顧客を引き受けることを好みます。これらの富裕層は、さまざまな[ウェルスマネジメント](/wealthmanagement)サービスを必要としており、それらにお金を払う余裕があります。

>>

効率は、これらのオンラインプラットフォームが持つもう1つの重要な利点です。たとえば、ロボアドバイザーの前に、取引を実行したい場合は、ファイナンシャルアドバイザーに電話するか、物理的に会い、ニーズを説明し、彼らが取引を実行するのを待つ必要があります。今、あなたはあなたの家の快適さの中でいくつかのボタンをクリックするだけでそれらすべてを行うことができます。

一方、ロボアドバイザーを使用すると、個人投資家として選択できるオプションが制限されます。たとえば、投資する投資信託やETFを選択したり、アカウントで個別の株式や債券を購入したりすることはできません。ただし、市場を打ち負かすために個々の株を購入することは、悪い結果を生み出すことが繰り返し示されているため、これは有益かもしれません。平均して、通常の投資家は、インデックス戦略でより良い結果を得ることがよくあります。

##ロボアドバイザーを雇う

ロボアドバイザーを開くには、多くの場合、短いリスクプロファイリングアンケートに回答し、財務状況、期間、および個人投資の目標を評価する必要があります。多くの場合、ロボアドバイザリーアカウントにすばやく簡単に資金を提供するために、銀行口座を直接リンクする機会があります。

自動化されたアドバイザリーサービスの特徴は、オンラインアクセスの容易さです。しかし、多くのデジタルプラットフォームは、特定の人口統計を他のプラットフォームよりも引き付け、ターゲットにする傾向があります。つまり、テクノロジーに精通し、投資可能な資産を蓄積しているミレニアル世代とジェネレーションXの投資家です。

> SECは、2021年11月に、多くのロボアドバイザーのコンプライアンス問題に関して投資家にリスクアラートを発行しました。そのため、FINRAInvestorAlertsおよびSECDivisionof ExaminationのWebサイトで情報を確認することにより、常に情報を得ることができます。

>>

この人口は、個人情報をオンラインで共有し、資産管理などの重要なタスクをテクノロジーに委託する方がはるかに快適です。実際、ロボアドバイザー企業のマーケティング活動の多くは、これらの投資家にリーチするためにソーシャルメディアチャネルを採用しています。

##ロボアドバイザーと規制

ロボアドバイザーは、人間のアドバイザーと同じ法的地位を保持しています。したがって、[米国証券取引委員会（SEC）に登録する必要があり、従来](/sec)[のブローカーディーラー](/broker-dealer)と同じ証券法および規制の対象となります。

ほとんどのロボアドバイザーは、[金融業界](/finra)[規制当局](/finra)（FINRA）のメンバーです。 BrokerCheckを使用して、人間のアドバイザーと同じようにロボアドバイザーを調査できます。

ロボアドバイザーが管理する資産は、銀行預金ではなく投資目的で保有される証券であるため、[連邦預金保険公社（FDIC）によって保険がかけられていません。](/fdic)ただし、ブローカーディーラーが資産に保険をかけることができる他の多くの手段があるため、これは必ずしもクライアントが保護されていないことを意味するわけではありません。たとえば、米国のもう1つの著名なロボアドバイザーであるWealthfrontは、[証券投資者保護公社](/sipc)（SIPC）によって保険をかけられています。

##ロボアドバイザーがお金を稼ぐ方法

ほとんどのロボアドバイザーがお金を稼ぐ主な方法は、運用資産（AUM）に基づく[ラップフィーです。](/wrap-fee)従来の（人間の）ファイナンシャルアドバイザーは通常、AUMの年間1％以上を請求しますが、多くのロボアドバイザーは運用資産1,000ドルあたり年間約0.25％を請求します。

>ロボアドバイザーを使用した投資の見返りが、ロボアドバイザーの使用に関連する総コストを上回らない場合は、ロボアドバイザーを使用しない方がよい場合があります。

>>

管理費に加えて、ロボアドバイザーは他のいくつかの方法でお金を稼ぐことができます。 1つの方法は、現金残高（「現金管理」）で得られる利息です。これは、クライアントではなくロボアドバイザーに貸方記入されます。多くのロボアドバイスされたアカウントはポートフォリオにわずかな現金しか割り当てられていないため、これもまた、多くのユーザーがいる場合にのみ、重要な収入源になる可能性があります。

もう1つの収益源は[、注文フローの支払いです](/paymentoforderflow)。通常、ロボアドバイザーは、預金、利子、および配当から追加された資金を蓄積します。次に、これらを1日に1つか2つのポイントで実行される大きなブロック注文にまとめます。これにより、注文サイズが大きいため、実行する取引が少なくなり、より有利な条件を取得できます。

最後に、ロボアドバイザーは、住宅ローン、クレジットカード、保険証券など、対象を絞った金融商品やサービスを顧客に売り込むことでお金を稼ぐことができます。これらは多くの場合、広告ネットワークではなく戦略的パートナーシップを通じて行われます。

##クラス最高のロボアドバイザー

米国および世界中で数百のロボアドバイザーが利用可能です。それらの多くは毎年発売されます。それらはすべて、投資管理、退職計画、および一般的な財務アドバイスのいくつかの組み合わせを提供します。

これは、最大の市場シェアを持つ最も競争力のある製品をまとめたものです。

###スタンドアロンロボアドバイザー

これらの企業は、デジタルアドバイザリーテクノロジーの初期のパイオニアの一部です。彼らは最低からゼロまでのアカウントの最低額で最も競争力のある料金を持っています。現在投資資産がないクライアントは、これらのプラットフォームをゼロから始めることができます。

###ロボアドバイザーのレガシー製品

ますます多くの金融サービスおよび[資産運用](/assetmanagement)会社がロボアドバイザーを立ち上げています。これらのプラットフォームは通常、手数料と口座の最低額が高く、洗練された投資家を対象としています。これらは、すでにこれらの会社を資産管理者として使用しているクライアントにとって便利なオプションです。

##ロボアドバイザーの欠点

ロボアドバイザーの参入により、金融サービスの世界と平均的な消費者の間の従来の障壁のいくつかが打ち破られました。これらのオンラインプラットフォームにより、富裕層だけでなく、誰もが健全なファイナンシャルプランニングにアクセスできるようになりました。

それでも、業界の多くは、ウェルスマネジメントの万能ソリューションとしてのデジタルアドバイザーの実行可能性に疑問を抱いています。彼らの技術的能力の相対的な卑劣さと最小限の人間の存在を考えると、ロボアドバイザーは共感と洗練が不足していると批判されてきました。

アカウントが少なく、投資経験が限られている場合、これらは優れたエントリーレベルのツールです。資産計画、複雑な税務管理、信託基金の管理、退職後の計画などの高度なサービスが必要な場合は、それらが不足していることに気付くかもしれません。

>多くの人がパッシブインデックス戦略から離れ、市場をアンダーパフォームする可能性のあるよりリスクの高い分野に投資しているため、ロボアドバイザーが投資するものに注意を払ってください。

>>

自動化されたサービスは、予期しない危機や異常な状況に対処するための設備も整っていません。たとえば、彼らはあなたが仕事の合間にいるのか、予期しない出費を処理しているのかを知りません。デジタルアドバイザーに自動引き出しを設定していると、予期せずに資金が枯渇する可能性があります。

さらに、ロボアドバイザーは、あなたが目標を定義し、あなたの経済状況を明確に理解していることを前提に活動しています。多くの投資家にとって、そうではありません。 「リスク許容度は低いですか、中程度ですか、高いですか」などの質問を調査します。投資の概念と、選択した各オプションの実際の影響についての基本的な知識があることを前提としています。

##ハイライト

-ロボアドバイザーは、共感性と複雑さの欠如について批判されてきました。

-ロボアドバイザーは、最小限の人間の監督で自動化されたアルゴリズム投資サービスを提供するデジタルプラットフォームです。

-現代ポートフォリオ理論に基づいて、パッシブインデックス戦略を自動化および最適化することがよくあります。

-これらは従来の投資に最適であり、資産計画などのより複雑な問題には最適なオプションではありません。

-ロボアドバイザーは多くの場合安価であり、低い期首残高を必要とするため、個人投資家が利用できるようになっています。

＃＃ よくある質問

###ロボアドバイザーでお金を失うことはできますか？

はい、ロボアドバイザーを使用すると、特にリバランスのコスト、手数料、減税の収穫でお金を失う可能性があります。

###ロボアドバイザーは市場を打ち負かしますか？

ほとんどのロボアドバイザーは、潜在的に市場を打ち負かす可能性のある戦略を組み込むのではなく、現代ポートフォリオ理論に従って市場を複製しようとするパッシブインデックス戦略に投資するため、市場を打ち負かすことはありません。

###ロボアドバイザーはあなたにお金を稼ぐことができますか？

はい、他のファイナンシャルアドバイザーと同じように、ロボアドバイザーでお金を稼ぐことができます。

###ロボアドバイザーは何をしますか？

ロボアドバイザーは、人間の介入なしに自動化されたアルゴリズムを通じてファイナンシャルプランニングサービスを提供します。

###ロボアドバイザーはどのように機能しますか？

ロボアドバイザーは、最初にオンライン調査を通じてクライアントに関する情報を収集し、次にそのデータに基づいてクライアントに自動的に投資することで機能します。ロボアドバイザーは、パッシブインデックス投資戦略を使用することがよくあります。
