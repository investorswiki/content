# 調整頻度
##調整頻度とは何ですか？

調整頻度とは、最初の固定金利期間が満了したときに変動金利住宅ローン（ARM）の金利がリセットされる金利を指します。

この頻度は、ローンの全期間にわたって利息費用を大幅に増加させる可能性があります。借り手は、閉鎖する前に、住宅ローンのこの要素を知っておく必要があります。

##調整頻度を理解する

調整頻度は重要ですが、[変動金利住宅ローン（ARM）の見落とされがちな機能です](/arm)。各ARMは、いくつかの重要な変数を備えています。これらの住宅ローンには、金利が固定される導入期間が含まれ、その後、一般的な市場金利を反映するために金利が定期的に変動する第2段階が続きます。

市場金利は、最初の住宅ローン契約で特定された[指数金利に反映されます。](/indexed_rate)初期期間は3年から10年の範囲である傾向があります。レート調整は、初期調整とその後の調整の上限によって制限されます。各ARMには、ローン契約期間中の任意の時点でレートを管理する絶対レートキャップが設定される傾向があります。

調整頻度は、通常、1年に1回の調整に設定されます。一般的に、金利変更の間隔が長いほど、借り手にとって有利です。金利が調整される頻度が低いほど、借り手が選択された指数の上昇リスクにさらされる頻度は低くなります。

ARMの[初期利率は、通常、従来の30年住宅ローンの利率を下回って](/initial-interest-rate)いることに注意することが重要です。これは、借り手をローンに引き付けるのに役立ちます。より頻繁に調整が行われると、貸し手はローンの利率を一般的な利率とより迅速に一致させることができます。

>###重要

> ARMによる金利調整を回避する唯一の方法は、新しい固定金利のローンに借り換えることです。

>>

##調整率と周波数

調整頻度は、ARMの金利が定期的に調整される率です。調整レートは、後続の各調整期間の後にARMに支払う新しいレートを表します。繰り返しになりますが、これは、インデックスまたはベンチマークレートがどちらの方向に移動したかに応じて、ARMに関連付けられた初期金利よりも高い場合と低い場合があります。

前述のように、ARMには[レートキャップが組み込まれており](/capstructure)、チェックされていないレートが増加するのを防ぎます。キャップには、年間キャップとローンキャップの寿命の2種類があります。年間上限は、ローン期間中の任意の年にレートが変更できる金額を制限します。ローンの上限の有効期間は、ローンの有効期間中に支払う最小および最大のレートを設定します。

>###注

>利息のみの変動金利住宅ローンでは、最初のローン期間中に利息のみの支払いを行うことができますが、これらの支払いによってローンの元本が減ることはありません。

>>

##最適な調整頻度はどれくらいですか？

一般的に言えば、住宅所有者にとっては、調整頻度が長いほど、ローンの金利の潜在的な変化が少なくなるため、より良い結果が得られます。たとえば、レートが毎月調整されるARMは、毎年または5年ごとに1回だけ調整されるARMと比較して、ローンの全期間にわたってより多くの利息がかかる可能性があります。

ARMの最適な調整頻度は、住宅購入の予算に基づいて、最終的には余裕のある頻度です。一貫した安定した収入と金利が全体的に低い場合、より頻繁な金利調整は予算にそれほど負担をかけないかもしれません。一方、毎月の支払いが月ごとまたは[年ごとに変動している場合](/year-over-year)、それはあなたのローンの義務に追いつくのをより難しくする可能性があります。

>###警告

[支払いオプションARM](/poaminimumpayment)に注意することが重要です。これにより、毎月最低額しか支払うことができない場合があります。これにより、負の[償却が発生する可能性があり](/amortization)ます。

>>

##調整頻度の例

[調整上限](/periodiccap)が1％の5/1ARMについて考えてみます。これは、5年後に最初の調整が行われ、5年後に年に1回調整されるARMです。

5年間の初期期間中に、最初の調整ポイントで実勢レートが6％になるまで金利が上昇したと想定します。これにより、住宅ローンの6年目に借り手に4％の新しい利率がもたらされ、その年の終わりに別の調整が行われます。

このシナリオを、毎月の調整頻度のあるローンと比較してください。このようなローンは、6％に達するのに3か月しかかかりません。インデックスレートが高いままであると仮定すると、借り手は6か月間6％のレートを支払うことを余儀なくされますが、最初の例の借り手は通年で4％のままになります。最初の例の借り手は、大幅な節約の恩恵を受けるでしょう。

>###ヒント

[固定金利](/fixed-rate_mortgage)の住宅ローンを選択した場合の総利息費用を見積もるのに役立ちます。

>>

##ハイライト

-調整の頻度は、ローンの全期間にわたって利息費用を大幅に増加させる可能性があるため、借り手は、クロージングの前にこの住宅ローンの要素を認識しておく必要があります。

-調整頻度は、通常、1年に1回の調整に設定されます。

-調整頻度は、調整レートとは異なります。調整レートは、レートが調整された後にARMに支払う新しい金利を表します。

-調整頻度とは、変動金利住宅ローン（ARM）の金利が、最初の固定金利期間が満了したときにリセットされる率を指します。

＃＃よくある質問

###調整頻度とは何ですか？

調整頻度は、変動金利住宅ローン（ARM）の金利が、基礎となるベンチマーク金利の変更に伴って増減する割合です。 ARMの一般的な調整頻度は1年ですが、月ごとまたは数年ごとに調整できるものもあります。

###適切な調整頻度はどれくらいですか？

適切な調整頻度とは、毎月の住宅ローンの支払いと支払う金利に関してある程度の予測可能性を維持できる頻度です。調整が少ないということは、ローンの金利と支払いの変更が少ないことを意味しますが、調整が頻繁になると、時間の経過とともにローンのコストが大幅に変わる可能性があります。

### ARMはどのくらいの頻度で調整しますか？

ARMの調整頻度は、ローン条件によって異なります。典型的なARM構造は5/1で、住宅所有者は最初の5年間に1つの固定料金を支払い、その後は年次料金調整を行います。他のARM構造には、3 / 1、7 / 10、および10/1が含まれます。
