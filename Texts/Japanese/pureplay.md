# ピュアプレイ
##純粋な遊びとは何ですか？

純粋な遊びとは、1つのタイプの製品またはサービスのみに焦点を当てている会社です。一部の投資家は、分析が容易で、特定の[市場セグメント](/market-segment)へのエクスポージャーを最大化できるため、純粋なプレイに投資することを好みます。

純粋な遊びは、さまざまな業界にわたって多くの製品やサービスを提供する多部門の企業や[コングロマリットとは対照的です。](/conglomerate)

たとえば、米国の銀行株へのエクスポージャーを希望する投資家は、バークシャーハサウェイ（BRK.B）が銀行業務だけでなく他の多くの業界にも関与しているため、バンクオブアメリカ（BAC）の株式を購入することを好むかもしれません。とセクター。

##純粋な演劇を理解する

純粋なプレイ会社は、特定の製品または業界セグメントに特定の賭けをしたい特定のタイプの[アクティブな投資家に人気があります。](/activeinvesting)これらの投資家にとって、いくつかの多様な事業ラインを持つ会社を購入することは、彼らが投資したくない産業で不必要なリスクを冒すことを彼らに強います。

アナリストにとって、純粋なプレーは、比較可能な企業分析またはピア分析のためのより正確なデータを取得する機会を表しています。これらのレポートは、投資分析のための重要な情報源であり、[相対的な評価](/relative-valuation-model)[の基礎となります](/relative-valuation-model)。

相対評価では、[帳簿価格](/price-to-bookratio)（P / B）比率、[収益](/price-earningsratio)（P / E）比率、[販売価格](/price-to-salesratio)（P / S）比率、[価格などの指標を使用します。](/price-to-cash-flowratio) [-キャッシュフロー](/price-to-cash-flowratio)（P / CF）比。これらの各値は、投資アナリストが会社の相対的な価値を計算し、会社が[過大評価](/overvalued)[されている](/undervalued)か[過小評価されているかを評価するのに役立ちます](/overvalued)。純粋なプレイ会社は、互いにはるかに直接的に比較できるため、これらの分析への有益なインプットです。一方、コングロマリットは、その結果が多くの産業セクターを反映しているため、容易に比較することはできません。

>現実的には、純粋な遊びという用語は常に概算として使用されます。これは、今日の企業はほとんどの場合、業界を超えたエクスポージャーをある程度持っているためです。これは、大規模な上場企業を見る場合に特に当てはまります。

##純粋なプレイの実例

トレーダーは、米国の銀行セクターの分析を行っています。具体的には、PBとPEの比率に基づいて、さまざまな米国の銀行株の相対的な魅力を評価したいと考えています。以下の株式のリストは、分析のために編集されています。

-BB＆T Corporation：PB1.28およびPE12.98

-KeyCorp：PB1.06およびPE10.58

-SunTrust Banks：PB1.16およびPE11.88

-Citizens Financial Group：PBが0.75、PEが9.59

すべてのビジネスは複雑でユニークですが、この分析は、これらの4つのバンキングソックスが互いに比較的比較可能であることを明らかにしています。リージョナルバンキングは彼らのビジネスモデルの中心的な焦点であるため、これは予想されることです。そのため、それらは銀行セクターにとって「純粋な遊び」と見なされる可能性があります。

対照的に、この同じトレーダーは、金融セクターでの重要な役割のために、バークシャーハサウェイをリストに含めるように誘惑されました。しかし、これは、バークシャーの銀行以外の活動が数多くあり、銀行の純粋なプレーと直接比較することが困難であったため、反対に決定されました。

[セクター](/sector)、1つの製品、または1つの投資戦略に依存しているため、純粋なプレーには多くの場合、より高い[特定のリスクが伴います](/specificrisk)。これは、多様化することで軽減できます。

##ハイライト

-純粋な遊びとは、特定の業界またはニッチに焦点を当てた企業への投資を指します。

-今日、多くの企業がいくつかの製品ラインまたは市場セグメントに関与しているため、純粋なプレーを特定するのは難しい場合があります。

-一部の投資家は、分析のしやすさと特定のセクターへの露出のために純粋な演劇を好みます。
