# ロケーション値（CLV）を閉じる
##近接位置値（CLV）とは何ですか？

クローズロケーション値（CLV）は、[テクニカル分析で使用されるメトリック](/technicalanalysis)であり、証券の終値がその日の高値と安値に対してどこで下落するかを評価します。

近接位置の値の範囲は+1.0〜-1.0です。正の値が高いほど、終値がその日の高値に近いことを示し、負の値が大きいほど、終値がその日の安値に近いことを示します。

## Close Location Value（CLV）は何を教えてくれますか？

クローズロケーション値（CLV）は、高値から安値の範囲に関連する価格の位置を測定するテクニカル分析ツールです。 -1から+1の範囲で移動します。または、100を掛けると、-100％から+ 100％の範囲で移動します。

1（または100％）に近いクローズロケーション値（CLV）の読み取り値は、終値が高値に近く、強気の兆候と見なされることを示します。 -1（または-100％）に近いCLVの測定値は、終値がその安値に近く、弱気の兆候と見なされる可能性があることを示しています。ゼロに近いCLV読み取り値は、ニュートラルと見なされます。

## Close Location Value（CLV）の使用方法の例

近接位置値（CLV）自体は、ほとんどのトレーダーにとってそれほど重要であるとは考えられていません。このインジケーターは、主に他の技術方程式の変数として使用されます。

CLVは、たとえば、累積/配分（A / D）ライン（[累積/配分インジケーター[A / D]とも呼ばれます）の計算で顕著に機能します](/accumulationdistribution)。

##近接位置値（CLV）と累積/分布インジケーター（A / D）の違い

一般に、テクニカル分析の目標は、将来の価格変動を予測することです。これは、特定の証券における価格と出来高の活動を研究することによって達成されます。

蓄積/分布インジケーター（A / D）は、さまざまなボリュームインジケーターの1つです。ボリュームインジケーターは、最も一般的に使用されるチャートプラットフォームのいくつかで視覚的に表される数式です。出来高は[流動性](/liquidity)の指標と見なされることがよくあります。これは、出来高が最も多い株式または市場が最も流動性が高く、短期取引に最適であると考えられているためです。 （言い換えれば、さまざまな価格で取引する準備ができている多くの買い手と売り手がいます。）

蓄積/分配指標（A / D）は、[出来高](/volume)と価格を使用して、株式が蓄積または分配されているかどうかを評価し、株価と出来高フローの[相違を識別しようとします。](/divergence)

終値（CLV）は、日中の高値と安値に対する資産の終値を示します。クローズロケーション値（CLV）は、累積/分配（A / D）ラインの計算に使用されます。

## Close Location Value（CLV）の使用に関する制限

近接位置値（CLV）がそれ自体で有用であると見なされない理由のひとつは、ランダムな価格の急上昇または下落に非常に敏感であるためです。このボラティリティの高まりにより、多くの状況でほとんど役に立たなくなります。 （信頼性の高い高低関係メトリックとして、[ストキャス](/stochastic-modeling)ティクスは、不安定性が少なく、高低範囲の価格位置を決定するために別の式に依存するため、近接位置値（CLV）の代わりにしばしば好まれます。

##ハイライト

-CLVは、他のインジケーターと組み合わせて使用されます。

-正の値は終値がその日の高値に近いことを意味し、負の値は終値がその日の安値に近いことを意味します。

-終値（CLV）は、日中の高値と安値に対する資産の終値を示します。

-CLV値が+1の場合、終値はその日の高値と同じであり、-1はその日の安値であることを意味します。

＃＃ よくある質問

###蓄積/分配ラインをどのように読みますか？

累積/分配（A / D）ラインの目的は、価格トレンドを評価し、今後の反転を見つけるのに役立つことです。たとえば、A / Dラインが上昇トレンドにあるときに証券の価格が[下降トレンド](/downtrend)にある場合、指標は買い圧力があるかもしれないことを示します。証券の価格は上向きに逆行する可能性があります。逆に、A / Dラインが下降トレンドにあるときに証券の価格が上昇トレンドにある場合、インジケーターは売り圧力（またはより高い分布）がある可能性があることを示します。これは、価格が下落する可能性があることを警告します。一般に、上昇/分配（A / D）ラインは上昇トレンドを確認するのに役立ち、下降A/Dラインは価格下降トレンドを確認するのに役立ちます。

###テクニカル分析における蓄積とは何ですか？

トレーダーは、証券の価格の変化の重要性を評価する方法としてボリュームを使用することがよくあります。ボリュームとは、特定の期間に取引された株式の数を指します。ボリュームパターンを分析する場合、累積は（基本的に）株式が購入されたときであり、分配は（基本的に）特定の株式が売却されたときです。

###蓄積と分布をどのように特定しますか？

テクニカル分析では、累積/分布（A / D）は、価格とボリュームの変化の間に関係を作成する指標です。たとえば、価格変動に伴うボリュームが多いほど、価格変動はより重要になります（想定できます）。アナリストは、累積/分配（A / D）を使用して、価格に関連付けられたボリュームを比較することにより、価格の変化を確認します。ボリュームパターンを分析する場合、累積は（本質的に）買いであり、分配は（本質的に）売りです。株式の需要が高い場合は、蓄積されています。流通とは、株式が流通という形で需要よりも供給が多い場合を指します。
