# ベースI
##ベースIとは

Base Iは、クレジットカード決済のための最初の電子認証システムでした。 1973年にバンクオブアメリカによって開発されました。

##ベースIの内訳

Base Iは、クレジットカード取引用の電子リアルタイム認証システムとして1973年に最初に開発されました。これは、VisaNetシステムの一部として、BankAmericardの発行者であるBankofAmericaによって開発されました。 Baseは、Bank AmericaSystemEngineeringの頭字語です。現在、BankAmericardはVisaカードとして販売されており、BaseIはVisaNetシステムの2つのフェーズの最初のフェーズです。 2番目のフェーズはベースIIとして知られています。

Base Iシステムが開発される前は、クレジットカード処理は国のクレジットカードシステムの成長とともに進化していました。最初のカードは閉ループシステムであり、特定の小売業者または特定の銀行にビジネス接続しているローカライズされた商人のグループに固有のものでした。初期のケースでは、取引は加盟店から地元の銀行への電話で記録され、地元の銀行はカード所有者の月次明細書の記録をまとめました。

カードシステムは1950年代に着実に成長し、1960年代初頭にはバンクオブアメリカのBankAmericardがカリフォルニア市場を支配しました。広い地理的領域で競合する銀行間の取引を可能にするオープンループシステムは、1966年にInterbank Card Associationが設立されたときに初めて登場しました。この銀行の提携により、間もなくMasterCardブランドが採用され、BankofAmericaが独自のブランドを形成するようになります。ライバルネットワークであるNBI、1970年。この頃、紙を使わない取引処理への移行をサポートするサードパーティ企業が登場しました。かつてそのような会社はVisaNetでした。 NBIは1973年にVisaNetを買収し、MasterCardと競合するVisaカードを販売しました。 1970年代初頭の訴訟により、加盟銀行は両方のネットワークに参加することができました。

##BaseIとVisaNetシステムがトランザクションを処理する方法

Base Iシステムの開発は、1970年代半ばのVisaカードの発売とほぼ一致しました。ベースIは、マーチャントが取引承認要求を銀行に送信するためのリアルタイム承認システムを指します。リクエストには、カード番号と金額が含まれます。それに応じて、銀行は簡単な承認メッセージまたは説明が添付された拒否メッセージのいずれかを送信します。

Base IIは、BaseIシステムによって生成されたトランザクションの1日の終わりの調整を処理するための決済プロセスを提供します。 Base IIはバッチシステムです。BaseIのリアルタイムアクティビティとは異なり、決済は定期的に行われ、加盟店への決済手数料を査定します。

NBIとVisaNetがシステムを立ち上げて更新すると、MasterCardは、トランザクションを処理するINASと残高を決済して[清算するINETと呼ばれるシステムで構成される、同様の2つの部分からなる処理プラットフォームを採用しました。](/interbank-network-for-electronic-transfer-inet)
