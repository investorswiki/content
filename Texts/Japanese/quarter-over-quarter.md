# クォーターオーバークォーター（Q / Q）
##クォーターオーバークォーター（Q / Q）とは何ですか？

四半期ごと（Q / Q）は、ある[四半期](/quarter)から次の四半期への投資または企業の成長の尺度です。 Q / Q成長は、企業の利益または収益の成長を比較するために最も一般的に使用されますが、経済の[マネーサプライの変化を説明するためにも使用できます。](/moneysupply) [国内総生産](/gdp)（GDP）、またはその他の経済的測定。

##四半期ごとの理解（Q / Q）

は、会社の[財務状態](/financial-health)を評価するために、毎年または四半期ごとにリリースされる財務諸表を調べます。四半期ごとのステートメントは、[証券取引委員会](/sec)（SEC）または企業のWebサイトによって提供されるEDGARデータベースを通じて公開されており、 [10-Q](/10q)ステートメントと呼ばれます。アナリストは、四半期ごとの複数の期間にわたる企業のパフォーマンスを確認するときに、Q/Qの数値と変化を確認します。

Q / Qは、ある会計四半期と前の四半期の間のパフォーマンスの変化率です。四半期は通常3か月または90日です。 Q / Qは、ある期間から次の期間への[財務諸表](/financial-statements)にあるさまざまな財務数値と指標の成長率の変化を測定します。通常、比較は、会社の[会計年度の1四半期の](/fiscalyear)レポートと前の四半期のレポートを比較します。 Q/Qは次のように計算されます。

**（現在の四半期-前の四半期）/前の四半期**

特定の経済レポートは四半期ごとにリリースされ、前の四半期と比較して経済の成長または衰退を示します。たとえば、米国商務省[経済分析局](/bea)（BEA）によってリリースされた国内総生産（GDP）レポートは、四半期ごとにリリースされ、政府、企業、および個人の決定に影響を与えます。

レポートは、GDPが四半期ごとにどのように変化したかを示しており、景気後退は2四半期連続のGDPの低下と見なされるため、景気後退や不況などの経済的結果の可能性を示すことができます。四半期ごとのGDPの変化を分析することで、政策立案者は、たとえばGDPの低下を目撃している場合など、さらなる経済的影響を回避するために政策調整を行うことができます。

##四半期ごとの変動（Q / Q）

Q / Qの他のバリエーションは、月ごと（M / M）および[前年](/year-over-year)比（YOY）です。月ごとの成長は前月の成長を測定しますが、変化率は自然災害などの1回限りのイベントの影響を受けるため、Q/Qよりも変動しやすい傾向があります。 YOYは、前年と比較した1年間のパフォーマンスの変化を測定します。 YOYはより多くのデータを組み込んでいるため、基礎となるレポートの数値をより長期的に把握できます。 Q / Qの変化率は、通常、YOYの測定値よりも変動性が高くなりますが、M/Mの数値よりも変動性は低くなります。

インテルの収益は2018年の第1四半期から第2四半期にかけて11％増加しましたが、IBMの収益はQ / Qで41％も大幅に増加しました。ただし、2つの連続した四半期のみが調査されていることに注意してください。投資家は、他のいくつかの四半期を調べて、これらの変化がトレンドなのか、それとも季節的または一時的な調整なのかを確認します。

四半期の開始日が異なる企業間でQ/Q情報を比較すると、分析が歪む可能性があります。含まれる時間は異なる場合があり、季節要因が歪む可能性があります。投資家は、変化が進行中の傾向を反映しているか、外部要因の影響を受けているかを判断するために、一定期間にわたって数四半期を検討する必要があります。四半期の開始日が異なる企業を比較する場合、投資家は可能な限り[季節性](/seasonality)の影響を取り除くことが重要です。

##ハイライト

-アナリストは、四半期ごとの複数の期間にわたる企業の業績をレビューする際にQ/Qを考慮します。

-Q / Qは、国内総生産（GDP）などの他の重要な統計の変化を測定するためにも使用されます。

-Q / Qには、月ごとや前年ごとなど、他にもバリエーションがあります。

-四半期ごとの結果は、証券取引委員会（SEC）または企業のWebサイトで確認できます。

-四半期の開始日が異なる企業間でQ/Q情報を比較すると、季節的要因や一時的な環境条件のために分析が歪む可能性があります。

-四半期ごと（Q / Q）は、ある四半期から次の四半期への投資または企業の成長を測定します。
