# 通勤価値
##転流値とは何ですか？

[一括](/lump-sum-payment)で支払われる場合に組織が年金債務を履行するために必要な推定コストを指します。退職した従業員には、一括払いまたは通常の年金支払いのいずれかを選択することができます。支払いは、今日のドルを使用して計算された通勤価値に応じて[年金制度によって行われます。年金基金の義務は、主に、](/pensionplan)[死亡率表](/mortality-table)に基づく、長期金利とその受益者の期待寿命の積です。

##通勤価値を理解する

企業は、新入社員を引き付け、既存の労働力を維持するために、さまざまなメリットを提供します。これらの特典には、休暇、ボーナス、手数料、[従業員株式購入プラン](/espp)（ESPP）、および年金プランが含まれます。年金プランは、雇用主、従業員、または両方の当事者が、投資専門家によって管理される投資[ポートフォリオ](/portfolio)に（従業員に税引前のドルを使用して）貢献できるように設計されています。

従業員が退職する場合、年金制度からお金を受け取る方法について2つの選択肢があります。彼らは定期的な固定支払いを選ぶかもしれません、それは彼らに月収の源を保証します[、](/income)あるいは彼らは計画から一時金をとることができます。後者のオプションは、年金の通勤または現金化と呼ばれます。この一括払いの金額を通勤金額と呼びます。

年金基金のマネージャーは、彼らの支払い義務と彼らの計画の準備金要件を決定するために通勤価値を計算しなければなりません。このプロセスは、[資本予算](/capitalbudgeting)プロジェクトの[正味現在価値](/npv)（NPV）を計算するプロセスと似ています。転流値は、必然的に推定値です。これは、従業員が退職する年齢、その人の[期待寿命](/lifeexpectancy)、および一括払いを投資した場合に期待できる[収益率（RoR）に基づいて計算されます。](/rateofreturn)したがって、それは将来の利益の現在の価値です。

これらの要因は両方とも、将来の[金利に関する仮定に依存し](/interestrate)[ます](/interestrate)。予想される金利が高いほど、必要な金額は低くなり、逆もまた同様です。将来的にお金が必要になるほど、通勤価値は低くなり、逆もまた同様です。

>通勤価値の計算は、使用される金利の仮定に大きく依存します。

>>

##特別な考慮事項

金利は、従業員が一括払いまたは毎月の給付額を選択する際の重要な要素です。一括払いの従業員は、会社の予測で使用されている金利よりも高い投資[収益を得ることに賭けます。](/return)従業員がどのオプションを選択するかに関係なく、確固たる決定を下す前に数値を計算することは常に良い考えです。

しかし、従業員が退職する前に会社を辞め、他の誰かと仕事をする場合はどうなりますか？一部の雇用主は、個人に年金を無傷に保つオプションを提供する場合があります。しかし、これは非常にまれです。非従業員のアカウントの維持と支払いが必要になるためです。

大多数は、従業員が年金制度の金額を外部口座に送金するか、通勤した価値を受け取ることを要求する場合があります。もちろん、これは最後の手段となるはずです。個人が早期に[退職金制度を現金化するときに適用される税務上の影響がある可能性があるからです。](/retirement-planning)

>通勤価値は、将来の金融債務の正味現在価値を説明するためにも使用できます。

##通勤価値の例

架空の例を使用して、交換された値がどのように機能するかを示しましょう。 XYZ Corporationが従業員に[確定給付プラン](/definedbenefitpensionplan)を持っており、従業員Dが65歳で退職するとします。彼らは、残りの生涯にわたって毎年最終給与の80％を支払う年金を受け取る権利があります。現在の死亡率表に基づくと、従業員Dは85歳まで生きると予想されます。

彼らがXYZCorporationで働き始めて以来、同社はこの将来の[責任を見越して、従業員Dの給与の一部を年金基金に預けていました](/liability)。彼らが引退して支払いの受け取りを開始する準備ができたので、現在の投資収益率とファンドへの追加の支払いがないと仮定すると、残りの人生で期待される支払いの流れを生み出すのに十分なお金が節約されます。

これが通勤価値です。従業員Dは、年金制度にとどまり、支払いを受け取るか、通勤した金額を一時金として[引き出すことを選択できます。](/withdrawal)

##ハイライト

-年金基金の管理者は、支払い義務と準備金の要件を決定するために通勤価値を計算する必要があります。

-通勤価値とは、受益者が年金制度を通じて退職時に一括払いとして受け取る権利のある金額の合計です。

-この値は、受益者の将来の期待寿命などの要因に基づいて推定されています。

-従業員は、毎月の支払いで年金を受け取るオプションもあります。

-通勤価値を取得したり、年金を現金化したりする場合、特に早期に行う場合は、特定の納税義務が発生する可能性があります。
