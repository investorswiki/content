# スピンアウト
##スピンアウトとは何ですか？

スピンアウトは、部門を分離して新しい独立した企業を形成することを含む、一種の企業再編です。スピンアウト会社は、セグメントおよび関連する資産と負債の運用を引き受けます。

親会社は、[証券取引委員会](/sec)（SEC）から、[フォーム10-12B](/sec-form-f-10-12b)でスピンアウトの詳細を要求されます。このフォームには、新会社のスピンアウトの論理的根拠、長所、および短所を概説する実質的な情報レターまたは説明が含まれています。そしてその業界の見通し。スピンアウトは、通常、株主には非課税ですが、完了するまでに最大6か月かかる場合があります。

##スピンアウトを理解する

スピンアウトはさまざまな理由で発生する可能性があります。親会社は、会社全体とは異なるペースで成長している可能性のある組み込み部門の価値を解き放ちたいと思うかもしれません。通常、親よりも速く成長しているトラップまたは制約されたセグメントは、独立した会社としてより良いでしょう。

スピンアウトにより、スピンオフされた部門は、新会社の株式または会社の成長に資金を供給するための債券の形での負債を発行することにより、自己資本を調達することができます。統合された事業体では資金調達のための資金調達は不可能かもしれませんが、収益性の高い部門を分離することにより、分社化された部門は投資家や銀行を引き付ける可能性が高くなります。

スピンアウトは、運用管理、マーケティング、財務、人的資源などのさまざまな側面でさまざまなニーズを持つ可能性のあるセグメントにリソースを転用することなく、コア業務に集中できるようにすることで、親会社を支援することもできます。

また、スピンアウトされる部門は、ソフトウェアや必要なテクノロジーなどの補助サービスを作成するために設立された可能性があります。技術部門は収益性はありますが、親会社の業界に適合しない可能性があります。その結果、親会社と部門の事業計画や戦略が一致しない可能性があるため、分割したほうがよい場合があります。

>スピンアウトは通常、企業にとって前向きな兆候ですが、投資家はスピンアウト後に親会社に残っているものを気に入らず、その株式を売却する可能性があります。

>>

部門の収益性が親会社ほど高くない場合にも、スピンアウトが発生する可能性があります。別の会社を設立することで、苦労している部門の気を散らすものを取り除きます。また、スピンアウトにより、経営陣は資産を売却したり、新会社の合併や買収を探したりすることができます。

親会社は、多くの場合、スピンアウトの公平性を維持したり、製品やサービスの供給に関する契約関係に署名したりすることで、スピンアウトをサポートします。多くの場合、スピンアウトした会社の管理チームは親会社からも引き出されます。

##スピンアウトのいくつかの欠点

投資家は一般的にスピンアウトを支持しています。これは、さまざまなニーズと成長見通しを持つセグメントが単独でスピンアウトすることはビジネス上理にかなっているためです。時間の経過に[伴う評価が示している](/valuation)ように、分離された部分の合計は通常、投資家の全体よりも大きくなります。

ただし、スピンアウトプロセスは、管理時間と数か月間の気晴らしの点でコストがかかる可能性があります。経営陣の焦点は、会社の経営からスピンアウトの実行に移る可能性があります。また、スピンアウトを計画して完了するには、かなりの取引費用がかかる可能性があります。

もちろん、スピンアウトした部門がそれ自体で利益を上げるという保証はありません。スピンアウトした会社は、親の助けがなければ損失や収益の低下を招く可能性があります。逆に、スピンアウトによって収益性の高い部門を削除すると、親会社の収益が減少し、財務実績が低下する可能性があります。

##スピンアウトの例

スピンアウトは一般的であり、投資家はスピンアウトを推進する十分な理由があります。 2009年にブリストルマイヤーズスクイブからスピンアウトされたミードジョンソンニュートリション、2013年にファイザーからスピンアウトされたゾエティス、2016年にフィアットクライスラーからスピンアウトされたフェラーリなど、多くの注目すべきスピンアウトがあります。

###チポトルメキシカングリル

チポトルメキシカングリルは2006年にマクドナルドからスピンアウトしました。マクドナルドの理由は、デンバーポストが報告したように、「成長を促進し、主要な事業により多くのエネルギーを注ぐこと」でした。 Chipotleの株式は、最初の公募で22ドルで提供され、2006年に600万株が売却されました。2021年7月9日の取引終了時点で、Chipotleの株式は1株あたり1,592.25ドルで取引されていました。

### Delphi Technologies PLC

Delphi Automotive PLCは、2017年12月5日に45億ドルの事業体となったDelphi Technologies PLCをスピンアウトしました。新会社は、高度な推進システムを提供しています。コンピューティングパワーとスマートビークルアーキテクチャの飛躍的な向上によって可能になりました。」

DelphiAutomotiveはAptivPLCになり、パワートレイン事業を維持しました。スピンアウトしたDelphiTechnologiesPLCは、独自の運命を担っています。

###失敗したスピンアウト：旧海軍

衣料品小売業者のGapInc.（GAP）は、2019年の初めに、CNNの報告によると、同社が旧海軍の部門をスピンアウトする意向であると発表しました。オールドネイビーは独立した会社になります。バナナリパブリック、ヒルシティ、アスレタなどの他のブランドを含むギャップストアは1つの会社になります。

2018年、オールドネイビーは他のすべてのブランドとほぼ同じくらいの収益を生み出しました。ギャップと残りの店舗からの収益は90億ドルでしたが、売上高は80億ドルでした。スピンアウトの可能性の結果として、オールドネイビーは独自のビジネスプランと戦略の下でブランドを成長させるために解放されたでしょう。ギャップと残りの店舗は、過去数年間の売上高の伸びに苦戦しているため、統合されます。

しかし、このスピンアウトは実現しませんでした。 2020年、ギャップは計画を中止し、オールドネイビーを維持することを決定しました。ウォルマート、コストコ、ターゲットなどのアパレルを販売する大規模小売店との低コストの競争の中で、同社の独立した見通しの実行可能性が低下したためです。その発表を行った後、ギャップシェアは急上昇しました。

##ハイライト

-スピンアウト会社は、セグメントおよび関連する資産と負債の運用を引き受けます。

-スピンアウトにより、部門をスピンオフして、株式の発行を通じて独自の資本を調達し、独自のビジネス戦略を運用することができます。

-スピンアウトは、新しい独立した企業を形成するために部門を分離することを含む一種の企業再編です。
