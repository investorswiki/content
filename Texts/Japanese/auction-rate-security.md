# オークションレートセキュリティ（ARS）
##オークションレートセキュリティ（ARS）とは何ですか？

[オランダのオークション](/dutchauction)を通じて販売される変動金利証券の一種です。 ARSは通常、20年から30年の長期満期の債券、またはクローズドエンド型ファンドが発行する優先株式のいずれかです。 ARSは、可能な限り低い利回りで市場をクリアする金利で販売されます。これにより、ARSのすべての入札者が債務発行で同じ利回りを受け取ることが保証されます。 ARSの金利は、追加のオークションを通じて定期的にリセットされます。通常、7日、14日、28日、または35日ごとにリセットされます。

2008年の世界的な金融危機の間、オークションが清算率を確立するのに十分な入札者を引き付けることができなかったため、ARS市場は失敗しました。これは、多くの投資家が売却できない長期満期の投資を保有していることを意味しました。

##オークションレート証券（ARS）を理解する

[変動金利](/variableinterestrate)の柔軟性を求めている地方自治体および企業の発行者は、オークションレート証券（ARS）のルートをたどることができます。オークションレート証券は、オランダのオークションプロセスを通じて金利が決定される中長期の債務発行です。ある意味、ARSは、金利がほぼ毎月リセットされるため、短期的な問題であるかのように機能します。オランダのオークションは、すべての入札を取り込んで、オファリングの合計を販売できる最高価格を決定した後にオファリングの価格が設定される公募オークション構造です。

##ARSオークションの前

ARSのオークションは、7日、14日、28日、または35日ごとに開催され、その時点で料金がリセットされます。オークションの前に、ブローカーはクライアントと可能なARSレートの範囲について話し合います。 「[価格トーク」](/price_talk)と呼ばれるこの議論は、クライアントに予想レートの基礎を提供しますが、投資家はこの範囲外の入札を自由に提出できます。

投資家は、25,000ドルの額面で、購入する意思のある株式数と、債券から受け入れる意思のある最低金利を指定する入札を提出することにより、競争入札プロセスに入ります。

入札は締め切りまで受け付けられ、締め切り後はオークションエージェントが提出された入札に基づいて清算率を計算します。清算利率は、次のオークションまでに証券に支払われる利率です。

##ARSオークション後

投資家の入札レートが清算レートよりも低い場合、投資家は希望する入札の全部または一部を受け取ります。清算率を上回った入札は落札されません。[クーポン](/coupon)は、各オークション期間が終了した直後に支払われ、利回りは四半期ごとに決済されます[。](/coupon)投資家は、連邦税、州税、地方税が免除されていることに加えて、投資適格格付けが高いため、これらの証券に惹かれます。 ARSはまた、その複雑さとリスクの増大により、[マネーマーケット商品](/money-market-yield)よりもわずかに高い税引後利回りを提供します。

##ARS市場の崩壊

2月に。 2008年、ARS市場は、市場の4つの主要な投資銀行（シティグループ、UBS AG、ワコビア、メリルリンチ）が流動性の懸念から最後の手段の入札者として行動することを拒否したときに失敗しました。発行者に代わってこれらの証券を売却したブローカーは、買い手にそれらが流動的であると信じさせました。

[流動性がなくなっ](/illiquid)た長期投資を売却することができませんでした。事実上、オークションレートの証券の市場は存在しなくなりました。

ARS市場の崩壊後、[金融業界規制当局](/finra)（FINRA）、[証券取引委員会](/sec)（SEC）、および州の司法長官は、投資家に代わって主要なブローカーディーラーとの和解を交渉するために介入しました。バンクオブアメリカやシティグループなどの大手金融機関は、ARS投資のリスクを十分に開示していないと述べた投資家に400億ドル以上の返済を命じられました。

##ハイライト

-ARS市場は、2008年の世界的な金融危機の際に崩壊し、何万人もの投資家が売却できない長期投資を行ったままになりました。

-オークションレート証券（ARS）は、変動金利投資の一種であり、通常、長期満期の債券または優先株式のいずれかです。

-ARSの金利は通常、オランダのオークションを通じて7日、14日、28日、または35日ごとにリセットされます。

-オランダのオークションは、投資家が購入する意思のあるオファリングの金額と支払う意思のある価格に入札する公開オークションです。
