# Cloud on Title
##タイトルのクラウドとは何ですか？

所有権の雲とは、ある当事者から別の当事者への所有権の移転を妨げる可能性のある、資産に対する未解決の請求、リーエン、またはその他の負担です。 Cloud on titleは、譲渡を行う前に解決またはクリアする必要のあるタイトル証書の状態に関連する疑いがあることを指摘しています。

##より深い定義

住宅ローンが物件に持ち出されると、住宅ローンの所有者はその物件にリーエンを取ります。これは、所有者がタイトルをクリアせずに物件を売却できないことを意味します。リーエンは、他のさまざまな理由でタイトルの上に置くこともできます。これには、IRSリーエンや不動産に関連する法的判断が含まれます。このような状況では、タイトルに雲があるため、タイトルの譲渡はできません。

これが発生した場合、販売者には3つのオプションがあります。

-借金を返済することによってリーエンが解除されるように手配します。

-販売をキャンセルします。

-リーエンまたは負担をクリアするために静かなタイトル訴訟を開始します。

タイトルのすべてのクラウドが所有権の譲渡に影響を与えるわけではありません。土地が細分化されると、開発者はその土地に適用される条件と制限を記録することができます。また、電力線などのサービスには、ゾーニング要件と記録された地役権がある場合があります。これらは、タイトルの転送には影響しません。ただし、これらの地役権と条件は、プロパティの使用方法に影響を与える可能性があるため、購入者はこれらの地役権と条件を認識しておく必要があります。このため、バイヤーは記録の地役権の対象となる購入契約に署名することに注意する必要があり、すべての地役権はタイトルに文書化する必要があります。

##クラウドオンタイトルの例

ジムは不動産から不動産を購入しました。タイトル検索中に、エージェントは、45年前にさかのぼる古い住宅ローンが不動産に対して登録されていることを発見しました。このタイトルの雲はおそらく誤記であることに気づき、エージェントはタイトルを静めるための行動を開始しました。これは、住宅ローンが実際に返済されて閉鎖されたことを証明しましたが、何らかの理由で文書が適切に処理されませんでした。静かなタイトルアクションが成功した後、ジムは続行することができ、販売は終了しました。

##ハイライト

-タイトルの雲は、quitclaim証書によって解決されます。

-タイトルの雲は、不動産のタイトルを問題にするあらゆる障害です。

-抵当流れの例としては、差し押さえ手続き、財産のリーエン、検認、または不正なタイトルがあります。
