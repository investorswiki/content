# 所有者の収益実行率
##所有者の収益の実行率とは何ですか？

、定義された期間（通常は1年）にわたる所有者の収益（[フリーキャッシュフロー）](/freecashflow)の推定値です。

##所有者の収益の実行率を理解する

[実行率](/runrate)という2つの別個の要素で構成される用語です。それがどのように機能するかを理解するには、まず、それぞれが何を意味するのかを理解する必要があります。

###実行率

稼働率は、過去のデータに基づいて企業の将来の[業績を予測するための方法です。ある会社](/financialperformance)が前四半期に1億ドルの[収益を](/revenue)記録したとしましょう。この情報を将来のパフォーマンスの予測として使用すると、年間で4億ドルの売上を記録する、または4億ドルの実行率で稼働していると言えます。

###所有者の収益

次に、所有者の収益があります。投資の第一人者であるWarrenBuffettが好む評価方法です。[純利益](/netincome)（NI）は投資家から大きな注目を集めていますが、所有者に分配して[株主価値を高めるために企業が金庫に持っている実際の金額を常に完全に反映しているわけではありません](/shareholder-value)。

それが、所有者の収益が達成しようとしていることです。バフェットは、企業の価値は、事業の存続期間中に発生すると予想される純キャッシュフロー（所有者の収益）の合計から、[収益の再投資を差し引いたものにすぎないと述べまし](/earnings)た。 1986年のバークシャーハサウェイ年次株主レターで、バフェットは所有者の収益とその計算方法についていくつかの洞察を与えました。

>>

>「これらの質問を検討すると、「所有者の収益」と呼ばれるものについていくつかの洞察を得ることができます。これらは、（a）報告された収益に加えて、（b）減価償却、枯渇、償却、およびN社の項目（1）および（4）などの特定の他の非現金費用からプラントおよび設備などの平均年間資本支出額を差し引いたものを表します。 （競争力と単位量を維持するために追加の作業資本が必要な場合は、増分も（c）に含める必要があります。ただし、LIFOインベントリ方式に従う企業は、単位量が変わらない場合、通常、追加の作業資本を必要としません。 ）」

>>

言い換えれば、所有者の収益=報告された収益+減価償却、償却+/-その他の非現金費用–平均年間保守設備投資+/-[作業資本の変化](/workingcapital)。結果として得られた数字が私たちに伝えようとしているのは、会社が生み出している価値の量と、株主に還元されている量です。多くの場合、それはフリーキャッシュフロー（FCF）に似ています。つまり、企業が運用をサポートし、資本資産を維持するためにキャッシュアウトフローを考慮した後に生成するキャッシュです。

##所有者の収益の実行率の長所と短所

[財務状態](/financial-health)を測定するために使用できる重要な指標です。所有者の収益の増加は、企業のその後の収益が良好になるというシグナルとして機能する傾向があります。したがって、正確な所有者の収益実行率を評価することは、会社の長期的なパフォーマンスを予測する上で非常に重要になる可能性があります。

問題は、所有者の収益実行率が常に信頼できるとは限らないことです。つまり、会社の財務実績が期間を通じて一貫していることは当然のことです。たとえば、 [4分の3の後に](/quarter)、ある会社が所有者の収益900万ドルを計上したとします。業績が一定であると仮定すると、[会計年度](/fiscalyear)（FY）の会社の所有者の収益実行率は1200万ドル（四半期あたり300万ドル）になります。

[季節的な経験](/seasonality)をする業界で事業を行っているかどうかを評価するのが難しい場合があります。このような場合、ある期間の所有者の収益が全期間に適用されない場合があります。

>###重要

>四半期ごとに業績が変動する企業に適用した場合、所有者の収益実行率に欠陥があります。

>>

実行率は、新製品のリリースに関連する売上の増加、多くのテクノロジー企業で一般的な発生、または大規模な1回限りの売上も考慮していません。

##ハイライト

-所有者の収益実行率は、企業の財務が一貫していることを前提としているため、収益源が不安定な企業には適用できません。

-所有者の収益実行率は、定義された期間（通常は1年）にわたる所有者の収益（フリーキャッシュフロー）の推定値です。

-現在の財務データを使用して、企業が生産し、使用できると予想される実際のドル価値を示します。
