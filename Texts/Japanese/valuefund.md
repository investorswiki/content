# バリューファンド
##バリューファンドとは何ですか？

、基本的な特性に基づいて価格が[過小評価され](/undervalued)ているとみなされる株式に投資しようとします。バリュー投資は、成長の見通しが高い新興企業に焦点を当てる成長投資とは対照的です。

##バリューファンドの仕組み

バリューファンドと[バリュー投資](/valueinvesting)は、多くの場合、投資家のベンジャミングラハムとウォーレンバフェットによって開発された戦略と同義です。バリューマネージャーは、株式の本源的価値に関連する基本的な特性に基づいて、バリューファンドの株式を選択します。バリューファンドは通常、時間の経過とともに着実に成長する可能性のある長期的な投資配分として使用されます。したがって、バリューファンドへの投資は、投資のデューデリジェンスと忍耐に関連していることがよくあります。

ほぼすべての大規模なファンドファミリーがバリューファンドを提供しています。バリューファンドは、多くの場合、さまざまなコンポーネントによって分類されます。バリエーションで最も人気のあるカテゴリの1つは、[時価総額](/marketcapitalization)です。たとえば、投資家は、小型株、中型株、大型株のバリューファンドを含むファンドファミリーから選択することができます。

バリュー投資の背後にある前提は、市場にはいくつかの固有の非効率性があり、さまざまな理由で特定の企業が実際の価値を下回るレベルで取引することです。バリューファンドマネージャーは、これらの市場の非効率性を特定することに熟練しています。理論的には、市場がこれらの非効率性を修正すると、バリュー投資家は株価の上昇から利益を得るでしょう。多くの場合、[バリュー株](/valuestock)は、配当金の分配プログラムを実施している定評のある企業であるため、配当金の支払いにも関連しています。

>[ブレンドファンド（またはブレンドファンド）は、](/blendfund)[バリュー株](/valuestock)と[成長株](/growthstock)の両方を組み合わせたエクイティミューチュアルファンドの一種です。これらのファンドは、単一のポートフォリオでこれらの人気のある投資スタイルの間で投資家の分散を提供します。

##バリューファンドの例

以下は、投資市場のバリュー[ミューチュアルファンド](/mutualfund)と[上場投資信託（ETF）の4つの例です](/etf)。

### 1.ヴァンガードエクイティ-インカムファンド投資家株式（VEIPX）

Vanguard Equity-Income Fund Investor Sharesは、投資家に平均以上の配当を支払う大企業への投資に焦点を当てています。このファンドは、より高い利回りを求め、長期的な投資期間を持っている投資家に最適です。

### 2. ClearBridge Large Cap Value Fund（SAIFX）

ClearBridge Large Cap Value Fundは、価値に焦点を合わせた投資戦略を通じて資本の増価と収益を追求する、積極的に運用されているバリューファンドです。ファンドは複数のシェアクラスを提供しています。また、一貫した四半期配当も支払っています。

### 3. Invesco S＆P 500エンハンストバリューETF（SPVU）

インベスコS＆P 500エンハンスドバリューETFは、S＆P500エンハンスドバリューインデックスのパフォーマンスを追跡します。ファンドマネージャーは、ファンドの資産の少なくとも90％を、原資産指数の一部である株式に投資します。インデックスの株式は「バリュースコア」が高く、[ファンダメンタル分析](/fundamentalanalysis)に基づいて過小評価される傾向があります。

### 4. iシェアーズエッジMSCIUSAバリューファクターETF（VLUE）

iシェアーズエッジMSCIUSAバリューファクターETFはインデックスファンドです。これは、MSCI USA EnhancedValueIndexの保有とリターンを再現することを目的としています。インデックスには、比較的低い評価で取引される価値特性を持つ米国の大型株と中型株が含まれています。

##ハイライト

-バリュー投資戦略の背後にある前提は、市場がこれらの株式の真の価値を認識すると、株価が上昇し、バリューファンド投資家がこの上昇から利益を得るということです。

-バリュー株は、投資家に配当金を支払う老舗企業であることがよくあります。

-世界で最も成功している投資家の1人であるウォーレンバフェットは、バリュー投資家です。

-バリューファンドは、ファンダメンタル分析に基づいて過小評価されている株式に焦点を当てた戦略に従うプールされた投資です。
