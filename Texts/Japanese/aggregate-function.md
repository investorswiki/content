# 集計関数
##集計関数とは何ですか？

集計関数は、値の範囲を含む数学的な計算であり、その結果、派生元の蓄積されたデータの重要性を表す単一の値になります。集計関数は、[記述統計を](/descriptive_statistics)[導出するためによく使用されます](/descriptive_statistics)。

集計関数は、現在職場で一般的になっているデータベース、スプレッドシート、および統計ソフトウェアパッケージでよく使用されます。集計関数は、経済および金融で広く使用されており、経済の健全性または市場のパフォーマンスを表すキー数値を提供します。

##集計関数を理解する

集計関数は、データセットに対して実行される計算を参照して、基になるデータを正確に表す単一の数値を取得します。コンピューターの使用により、これらの計算の実行方法が改善され、集計関数で結果を非常に迅速に生成できるようになり、ユーザーのデータに対する信頼度に基づいて重み付けを調整することもできます。コンピューターのおかげで、集計関数はこれまで以上に大きく複雑なデータセットを処理できます。

一般的な集計関数には次のものがあります。

[算術平均](/mean)とも呼ばれます）

- カウント

-最大

-最小

- 範囲

--NaNmean（NaN値を無視する平均。「nil」または「null」とも呼ばれます）

-[中央値](/median)

[-モード](/mode)_

-合計

##経済モデリングにおける集計関数

集計関数の数学は、過去10年間の米国の平均[国内総生産（GDP）成長を求めるなど、非常に単純な場合があります。](/gdp)それ自体がデータセットの集計関数の積であるGDP値のリストを考えると、年ごとの差を見つけて、差を合計し、10で除算します。数学は鉛筆と紙で実行できますが、世界のすべての国のGDP値を含むデータセットに対してその計算を行おうと想像してみてください。この場合、Excelシートは処理時間を大幅に短縮し、モデリングソフトウェアのようなプログラムソリューションはさらに優れています。このタイプの処理能力は、エコノミストが大規模なデータセットに対して一連の集計関数を実行するのに大いに役立ちました。

[経済学](/econometrics)や分野内の他の分野では、毎日集計関数を使用しており、結果の図の名前でそれを認識することがあります。[総供給](/aggregatesupply)と[需要](/aggregatedemand)は、2つの集約関数の結果を視覚的に表したものです。1つは本番データセットで実行され、もう1つは支出データセットで実行されます。総需要曲線は、同様の支出データセットから作成され、期間にわたってプロットされたサブセットの総数を示して、時系列にわたる変化を示す曲線を作成します。このタイプの視覚化またはモデリングは、経済の現在の状態を示すのに役立ち、実際のポリシーおよびビジネス上の決定を通知するために使用できます。

##ビジネスにおける集計関数

[総時間など](/aggregate_hours)、多くの総計機能があります。とはいえ、財務で集計関数を使用するより興味深い方法の1つは、[集計リスクのモデル化](/aggregate-risk)です。

特に金融機関は、簡単に理解できるエクスポージャーの要約を提供する必要があります。これは、特定のカウンターパーティリスクとリスクのある総[価値を要約することを意味します](/var)。これらの数値を算出するために使用される計算は、それ自体がデータセットに基づく確率であるリスクを正確に反映している必要があります。

複雑さが増すと、間違った場所での日当たりの良い仮定がモデル全体を損なう可能性があります。この正確な問題は、リーマン・ブラザーズの崩壊をめぐるフォールアウトに影響を及ぼしました。

##ハイライト

-エコノミストは、データ集計の出力を使用して、時間の経過に伴う変化をプロットし、将来の傾向を予測します。

-集約されたデータから作成されたモデルは、ポリシーとビジネスの決定に影響を与えるために使用できます。

-集計関数は、より大きなデータセットを表す単一の数値を提供します。使用されている数値は、それ自体が集計関数の積である可能性があります。

-多くの記述統計は、集計関数の結果です。
