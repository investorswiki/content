# ノンデリバラブルスワップ（NDS）
##ノンデリバラブルスワップ（NDS）とは何ですか？

ノンデリバラブルスワップ（NDS）は、主要通貨とマイナー通貨の間の[通貨スワップのバリエーションであり、制限されているか、変換できません。](/currencyswap)これは、通貨フローの物理的な交換がある通常の通貨スワップとは異なり、スワップに関係する2つの通貨の実際の配信がないことを意味します。代わりに、NDSの定期的な決済は、現金ベースで、通常は米ドルで行われます。

決済額は、スワップ契約で指定された為替レートとスポットレートの差額に基づいており、一方の当事者が他方の当事者に差額を支払います。ノンデリバラブルスワップは、一緒にバンドルされた一連の[ノンデリバラブルフォワード](/ndf)と見なすことができます。

##ノンデリバラブルスワップ（NDS）を理解する

ノンデリバラブルスワップは、多国籍企業が通貨管理のために利益を本国に送金することを許可されない可能性があるリスクを軽減するために使用されます。また、NDSを使用して、[流動性](/liquidity)がほとんどない制限された通貨での突然の切り下げまたは[減価のリスクを](/depreciation)[ヘッジ](/hedge)し、現地市場で通貨を交換するための法外なコストを回避します。為替制限のある国の金融機関は、NDSを使用して外貨建てローンのエクスポージャーをヘッジしています。

NDSの主要な変数は次のとおりです。

-想定元本（つまり、取引金額）

-関係する2つの通貨（ノンデリバラブル通貨と決済通貨）

-決済日

-スワップの契約レート、および

-固定レートと日付–信頼できる独立した市場ソースからスポットレートが供給される特定の日付。

##NDSの例

、半年ごとに4％の固定金利で米国の[貸し手](/lender)から5年間の1,000万米ドルの融資を受けています。 LendExは、地元企業への貸付のために、米ドルを現在の為替レート5.4でアルゼンチンペソに換算しました。ただし、ペソの将来の下落が懸念されており、利息の支払いと元本の返済を米ドルで行う方が費用がかかることになります。したがって、以下の条件で海外[のカウンターパーティと通貨スワップを締結します。](/counterparty)

-**想定元本（N）** –利息の支払いに400,000米ドル、元本の返済に1,000万米ドル。

-**関連する通貨**–アルゼンチンペソと米ドル。

-**決済日**–合計10日。最初の日付は最初の利息の支払いと一致し、10番目と最後の日付は最終的な利息の支払いと元本の返済と一致します。

-**スワップの契約レート（F）** –簡単にするために、利息の支払いに6（1ドルあたりのペソ）、元本の返済に7の契約レートを言います。

-**固定料金と日付（S）** –決済日の2日前、ロイターから東部標準時正午12時に供給。

NDSを決定する方法は、次の式に従います。

>>

>利益=（NS-NF）/ S = N（1-F / S）

>>

この例でNDSがどのように機能するかを次に示します。最初の確定日（最初の利払い/決済日の2日前）に、スポット為替レートが米ドルに対して5.7ペソであると想定します。 LendExは6のレートでドルを購入する契約を結んでいるため、この契約レートとスポットレートの差に想定利息を掛けた金額をカウンターパーティに支払う必要があります。この正味決済額は米ドルで計算され、-$ 20,000 [つまり、（5.7-6.0）x 400,000 = -120,000 / 6=-$20,000]になります。

2番目の確定日に、スポット交換レートが6.5から米ドルであると想定します。この場合、スポット交換レートは契約レートよりも悪いため、LendExは33,333ドルの正味支払いを受け取ります[（6.5 – 6.0）x 400,000 = 200,000 / 6=33,333ドルとして計算]。

このプロセスは、最終返済日まで続きます。ここで注意すべき重要な点は、これはノンデリバラブルスワップであるため、カウンターパーティ間の決済はアルゼンチンペソではなく米ドルで行われるということです。

##ハイライト

-NDSは通常、基礎となる通貨を入手するのが難しい、流動性がない、または不安定な場合に使用されます。たとえば、発展途上国の通貨や、キューバや北朝鮮などの制限された通貨の場合です。

-ノンデリバラブルスワップ（NDS）は、スワップ自体に関係する2つの通貨ではなく、米ドル相当額で支払われ決済される通貨スワップの一種です。

-その結果、原通貨の現物引渡しがないため、スワップは転換不能（制限付き）と見なされます。
