# 基本通貨
##基本通貨とは何ですか？

[外国為替市場で](/forex)は、通貨単価は通貨ペアとして見積もられます。基本通貨（取引通貨とも呼ばれます）は、通貨ペアの見積もりに表示される最初の通貨であり、その後に[見積もり通貨](/quotecurrency)またはカウンター通貨と呼ばれる見積もりの2番目の部分が続きます。会計上の目的で、会社はすべての損益を表すために国内通貨または[会計通貨として基本通貨を使用する場合があります。](/accounting-currency)

##基本通貨の内訳

外国為替では、基本通貨は、基本通貨の1単位を取得するために必要な見積もり通貨の量を表します。たとえば、CAD / USD通貨ペアを見ると、カナダドルが基本通貨になり、米ドルが見積もり通貨になります。

通貨に使用される略語は、[国際標準化機構（ISO）によって規定されています。](/international-organization-for-standardization-iso)これらのコードは、標準のISO 4217で提供されています。通貨ペアは、特定の通貨を表すために3文字で構成されるこれらのコードを使用します。通貨ペアを構成する通貨は、スラッシュ文字で区切られることがあります。スラッシュは省略されるか、ピリオド、ダッシュ、または何も置き換えられない場合があります。

主要な通貨コードには、米ドルがUSD、ユーロがEUR [、](/euro)日本円がJPY 、英国ポンドが[GBP](/gbp) 、オーストラリアドルがAUD、カナダドルがCAD、スイスフランがCHFが含まれます。

##通貨ペアの一部

外国為替では、通貨ペアはXXX/YYYまたは単にXXXYYYと表記されます。ここで、XXXは基本通貨であり、YYYは見積もり通貨です。これらの形式のサンプルは、GBP / AUD、EUR / USD、USD / JPY、GBPJPY、EURNZD、およびEURCHFです。

[為替レート](/exchangerate)が提供されている場合、通貨ペアは、提供された基本通貨の1単位を購入するために必要な見積もり通貨の量を示します。たとえば、EUR / USD = 1.55を読み取ることは、_1が$1.55に等しいことを意味します。これは、_1を購入するには、購入者が$1.55を支払う必要があることを直接示しています。通貨ペアの見積もりは、基本通貨を販売する場合と同じ方法で読み取られます。[売り手](/seller)が_1を売りたい場合、彼はそれに対して$1.55を受け取ります。

##同時動作

投資家は同時に通貨を売買するため、外国為替相場はペアとして記載されています。たとえば、買い手がEUR / USDを購入する場合、それは基本的に、彼がユーロを購入し、同時に米ドルを販売していることを意味します。投資家は、基本通貨が相場通貨とは対照的に価値を得ると考える場合にペアを購入します。一方、基本通貨が相場通貨とは対照的に価値を失うと彼らが考える場合、彼らはペアを売ります。
