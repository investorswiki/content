# 保険契約者余剰の収益
##保険契約者余剰の利益とは何ですか？

率は、保険会社の[純利益](/netincome)と保険契約者[余剰の比率です](/policyholder-surplus)。保険契約者余剰は、保険契約者が所有する保険会社の資産から負債を差し引いたものです。保険契約者の余剰利益率の目標は、保険会社の財務状態を測定し、保険会社がどれだけの収益を利益に変えることができるかを判断することです。

##保険契約者余剰の収益を理解する

[保険会社が保険](/insurance)契約の引受と投資収益から生み出す収益額に対してどれだけの利益をもたらすことができるかを示し、保険契約者余剰は保険会社の資産が負債をどれだけ超えるかを表します。

この比率は、保険会社の税引後利益とキャピタルゲインを保険契約者の剰余金で割って計算され、保険契約者の剰余金は保険会社の資産の代わりになります。これは、他の業界で使用され[ている自己資本利益率](/returnonequity)（ROE）の測定値に似ており、保険会社の[財務状態の測定値です](/financial-health)。通常、パーセンテージで表されます。

##保険契約者余剰の収益に影響を与える要因

保険契約者の剰余金の返還は、引受された保険契約の種類、経済の状態、および[請求](/insurance_claim)が行われる可能性によって影響を受けます。市場での競争の欠如は、保険会社が[保険料の価格](/insurance-premium)を上げることを可能にし、それはより多くの収入をもたらすでしょう。

その後、この収益を証券に投資して、プラスのリターンを生み出すことができます。特に[株式市場](/stockmarket)のパフォーマンスの観点から、強い経済は、利益が実現されると純利益を増やすことができます。

保険会社はまた、大嵐などの大災害がないことから恩恵を受け、多くの保険契約者が同時に請求を提出することになります。たとえば、ハリケーンが保険会社がポリシーを作成した多くの町の多くの家を荒廃させた場合、これは保険会社の財務実績に大きな影響を与えます。

[再保険会社](/reinsurance)に割り当てることです。

保険会社の保険契約者余剰に対するリターンを検討する投資家は、特定の比率につながった要因の混合も検討する必要があります。株式市場のパフォーマンスは以前よりもはるかに良好でしたか？また、パフォーマンスは持続可能であると思われますか？たとえば、ドットコムバブルの前にテクノロジー株に投資している保険会社は、後から考えると成長は持続不可能でしたが、非常に高い純利益を得ることができました。

気候変動により、特定の地域はより自然災害に見舞われましたか？会社はどのような種類のポリシーを提供しており、それらのポリシーのリスクは適切に説明されていますか？たとえば、同社は干ばつ状態がますます起こりやすい地域で火災保険を提供することができます。

##保険契約者余剰データの収益の取得

[全米保険監督官協会](/nainsurancec)（NAIC）の[保険規制情報システム](/insurance-regulatory-information-system-iris)（IRIS）に基づく、ほとんどの州の公開データです。 IRISは、州の保険部門に、それぞれの州内で営業している保険会社の財政状態をスクリーニングおよび分析するための統合されたアプローチを提供するように設計された分析ソルベンシーツールおよびデータベースのコレクションです。

NAIC委員会の州の保険規制当局によって開発されたIRISは、州の保険部門が規制当局の注意を最も必要とする保険会社にリソースを集中させるのを支援することを目的としています。 NAICによると、このシステムは、各州の保険部門独自のソルベンシー監視の取り組みに取って代わるものではありません。

##ハイライト

-保険契約者余剰利益率は、保険会社の財務状態を評価するために保険業界で使用される比率です。

-保険契約者の余剰利益率は、保険会社が引受方針と投資収益から生み出す収益と比較して、どれだけの利益を上げているかを示すことを目的としています。

-保険契約者の余剰利益率に影響を与える要因には、株式市場のパフォーマンス、保険金支払いの割合、および保険契約書のリスクレベルが含まれます。

-ほとんどの州では、保険契約者の余剰利益は、全米保険監督官協会（NAIC）が管理する保険規制情報システム（IRIS）を通じて利用できます。

-税引後利益とキャピタルゲインを合計し、保険契約者余剰で割って、保険契約者余剰の収益を決定します。

-純収入は保険契約者の余剰と比較され、比率に到達します。
