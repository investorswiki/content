# 見積もり通貨
##見積もり通貨とは何ですか？

[外国為替（外国為替）](/foreign-exchange)では、一般に[カウンター通貨](/counter-currency)として知られている見積もり通貨は、直接通貨ペアと間接[通貨ペアの両方の2番目の通貨で](/currencypair)あり、基本通貨の値を決定するために使用されます。

[直接見積もり](/directquote)[では](/directquote)、見積もり通貨は外貨ですが、[間接見積もり](/indirectquote)では、見積もり通貨は国内通貨です。相場通貨は、為替レートが相場される場合、ペアの基本通貨の後に表示されます。最初の通貨または[基本通貨](/basecurrency)の1単位を購入するために、販売する必要のある見積もり通貨の金額を決定できます。

##見積もり通貨を理解する

[外国為替](/forex)市場で通貨を取引したい人にとって不可欠です。マーケットメーカーは、特定の通貨ペアを直接または間接のいずれかの方法で取引する傾向があります。つまり、見積もり通貨を理解することが最も重要です。

通貨ペアの為替レートは、基本通貨の1単位を購入/販売するために販売/購入する必要がある見積もり通貨の量を反映しています。通貨ペアのレートが上がると、ペアが直接か間接かにかかわらず、見積もり通貨の価値は下がります。

たとえば、米ドルとカナダドルのクロスレートは[USD / CAD](/usd-cad-us-dollar-canadian-dollar-currency-pair)で表され、直接見積もりです。これは、CADが見積もり通貨であり、USDが基本通貨であることを意味します。 CADは、1米ドルの価値を決定するための参照として使用されます。米国中心の観点から、CADは外貨です。

一方、 [EUR / USD](/eur-usd-euro-us-dollar-currency-pair)は、ユーロと米ドルのクロスレートを示し、間接的な見積もりです。これは、EURが基本通貨であり、USDが見積もり通貨であることを意味します。ここで、USDは国内通貨であり、1ユーロの価値を決定します。

##特別な考慮事項

通貨ペア（基本通貨と見積もり通貨の両方）は、さまざまな要因の影響を受けます。これらのいくつかには、経済活動、[中央銀行によって制定された金融および財政政策](/centralbank)、および[金利](/interestrate)[が含まれます](/interestrate)。

ユーロや米ドルなどの主要通貨は、特にエキゾチックな通貨での取引に関しては、通貨ペアの相場通貨ではなく基本通貨である可能性が高くなります。

2021年に市場で最も一般的に取引された通貨ペアは次のとおりです。

-EUR / GBP

-EUR / USD

-GBP / USD

-USD / CHF

-米ドル/円

上記のように、これらのペアの最初の通貨は基本通貨であり、2番目の通貨（スラッシュの後）は見積もり通貨です。 GBP / USDの組み合わせでは、ポンドは基本通貨または購入される通貨であり、ドルは見積もり通貨です。これは販売されているものです。

##見積もり通貨の例

トレーダーが米ドルを使用して400ポンドを購入したいとします。これには、GBP/USD通貨ペアを使用した取引が含まれます。取引を実行するために、彼らは£400を得るために彼らが売る必要があるUSD（見積もり通貨）の数を把握する必要があります。

2021年6月3日の取引日の終わりのペアの為替レートは1.4103でした。これは、トレーダーが£1を購入するのに$1.4103かかることを意味します。その日に[取引](/transaction)を完了するには、トレーダーは基本通貨の400単位、または£400 =（400 x 1.4103）で$ 564.12を取得するために、見積もり通貨の564.12単位を販売する必要がありました。

##ハイライト

-直接見積もりでは、見積もり通貨は外貨ですが、間接見積もりでは、見積もり通貨は国内通貨です。

-誰かが通貨ペアを購入する（長くなる）とき、彼らはカウンター通貨を売ります。彼らが通貨ペアをショートさせた場合、彼らはカウンター通貨を購入するでしょう。

-見積もり通貨（カウンター通貨）は、直接通貨ペアと間接通貨ペアの両方の2番目の通貨であり、基本通貨を評価するために使用されます。

-通貨クォートは、最初の（基本）通貨の1単位と交換する必要があるクォート通貨の多くの単位を示しています。
