# ミニサイズのダウオプション
##ミニサイズのダウオプションとは何ですか？

ミニサイズ（または「ミニ」または「E-ミニ」）ダウオプションは、原資産が[E-ミニ](/emini)ダウジョーンズ工業株平均（ [DJIA](/djia) ）先物契約である[インデックスオプション契約の一種です。](/indexoption)基礎となるE-miniダウは、ティックサイズが$5のDJIAの1/5の値で評価されます。ダウミニオプションは、[シカゴマーカンタイル取引所](/cme)（CME）の[Globex](/globex)システムを介して電子的に取引されます。

先物とオプションのミニサイズの契約は、トレーディングインデックスをより実用的にし、Nasdaq 100、S＆P 500、S＆P MidCap 400、Russell 2000などの他の幅広い[インデックス](/index)、および金や通貨などの商品にも存在します。ユーロなど。

##ミニサイズのダウオプションを理解する

ミニ先物およびオプション契約では、これらのミニサイズの商品のコストが標準の先物契約よりも安いため、投資家はエクスポージャーとポジションのサイズを微調整できます。 E-mini Dow先物契約の各ワンポイントの動きは、5ドルに相当します。オプショントレーダーは、ポジションの[デルタ](/delta)に関してこれを念頭に置いています。

[プットオプションの-1または](/putoption)[コールオプション](/calloption)の+1のデルタは、オプションが原資産のインデックスとポイントごとに移動することを示します。デルタがゼロに向かって移動すると、原資産の先物契約がポイントあたり5ドル移動しても、オプション契約は移動しない場合があります。

ダウジョーンズ工業株30種平均指数のE-miniオプションは[アメリカンスタイル](/americanoption)のオプションであり、満期前の任意の時点で行使できることを意味します。オプションを行使すると、原資産である[現金決済](/cashsettlement)のE-mini先物契約の対応するポジションが「物理的に」[配信されます。](/physicaldelivery)

>各ミニサイズのダウオプションは、1つの基礎となるE-ミニダウ先物契約を管理します。

>>

2020年の時点で、E-miniダウジョーンズ契約はGlobexで3番目に人気のあるミニ契約であり、2位のNasdaq 100 E-miniと、ボリュームで最も人気のあるS＆P500E-miniに次ぐものです。 E-mini Dowオプションでは、他のインデックスと比較して1日あたりの取引量が比較的少なくなっています。

ミニサイズのダウオプションはシンボルOYMで取引され、ダウの将来のミニはティッカーYMMで取引されます。彼らは3月、6月、9月、12月の経験があります。オプションの取引は、契約月の第3金曜日の東部標準時午前9時30分に終了します。

##E-ミニダウオプションの価格

ミニサイズのダウオプションの価格は、見積もり価格に乗数を掛けたものです。したがって、オプションの見積もり価格が300の場合、オプションの価格は300 x 5、つまり$1,500になります。これは、オプションに対して支払われる[プレミアムです。](/premium)支払われるプレミアムは、オプションの買い手（コールまたはプット）が失う可能性のあるものの中で最も多いものです。原資産先物を購入する人は、ポイントあたり5ドルの損失に直面します。これは、オプションプレミアムの固定損失よりも大幅に多くなる可能性があります。

原資産指数の価格が行使価格にオプションの価格を加えたものを上回った場合、E-miniダウコールオプションで利益が得られます。たとえば、オプションの行使価格が26,000で、オプション価格が800の場合、原資産指数が26,800を超えると、トレーダーは収益を上げます。

プットオプションの場合、同じ数字を使用して、インデックスがストライキからプレミアムを差し引いた値を下回ると、トレーダーはお金を稼ぎ始めます。この場合、26,000〜800、つまり25,200です。

##ミニサイズのダウオプション取引の例

6月に満了する基礎となるE-miniDow先物が25,648で取引されていると仮定します。トレーダーは、現在5月中旬で、来月には基礎となるE-miniDowの先物が大幅に上昇すると考えています。

彼らは25,650の行使価格で原資産のオプション契約を購入します。オプション価格は400に$5を掛けたもので、合計プレミアムコストは$ 2,000（および[手数料](/commission)）です。

で[損益](/breakevenpoint)分岐点に達するには、基礎が26,050（25,650 + 400）に上昇する必要があります。

-6月の[満了時](/expirationdate)に、原資産先物契約が25,650（行使価格）を下回る場合、コールオプションは無価値に失効し、トレーダーはオプションに支払った2,000ドルを失います（ただしそれ以上ではありません）。

-原資産が満期時に25,650から26,050の間にある場合、オプションは[金銭に含まれます](/inthemoney)が、取引は依然として全体的な損失をもたらします。基礎となるものが25,650に近いほど、2,000ドルのうち多くを失うことになります。基礎が満了時に26,050である場合、それらは損益分岐点になります。

-26,050を超えるポイントごとに、トレーダーはポイントごとに5ドルを稼ぎます。したがって、原資産が満期時に27,000である場合、オプションコールの買い手は

>>

> $ 4,750 =（（27,000-26,050）x $ 5）。

>>

別の方法で計算し、満期時の値から行使価格を差し引いた値に5ドルを掛けてから、オプションのコストを差し引きます。

>>

> 27,000-25,650 = 1,350 x $ 5 = $ 6,750-$ 2,000 =$4,750。

>>

##ハイライト

-ミニサイズのダウオプションは、基礎となる1つのE-ミニダウ先物契約を管理します。

-基礎となる先物契約は、それぞれ5ドル相当の1ポイントの増分で移動します。

-ミニサイズのダウオプションを購入するプレミアムは、オプションの価格に5ドルの乗数を掛けたものです。
