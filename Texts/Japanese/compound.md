# 化合物
##複合とは何ですか？

貯蓄者と投資家にとっての複合とは、投資された元本に収益を繰り返し追加することによって、時間の経過とともに指数関数的に成長する金額の能力を意味します。収益の各ラウンドは、次のラウンドの収益を生み出す元本に追加されます。貯蓄口座では、これは[複合利息と呼ばれ](/compoundinterest)ます。

対照的に、[単利](/simple_interest)は複利を反映していません。利息は元の残高にのみ支払われ、元の残高と以前の収益は支払われません。

##化合物を理解する

XYZ社に10,000ドルを投資するとします。初年度の[シェア](/shares)は20％上昇しました。あなたの投資は今$12,000の価値があります。その優れたパフォーマンスに基づいて、あなたは在庫を保持します。 2年目には、シェアはさらに20％上昇します。 12,000ドルの投資は14,400ドルに増えました。

初年度のように追加の$2,000（20％）を評価するのではなく、初年度に獲得した$ 2,000も20％増加したため、追加の$400を評価します。

プロセスを推定すると、以前の収益がさらに利益をもたらし始めるため、数値は非常に大きくなり始めます。実際、25年間年間20％で投資された10,000ドルは、ほぼ1,000,000ドルに成長します。これは、元の投資額にお金を追加することなく行われます。

複利の力は、アルバート・アインシュタインによって世界の8番目の不思議と呼ばれていました。彼はまた、「それを理解する者はそれを稼ぐ。理解しない者はそれを支払う」と宣言したと言われている。

##複利の計算方法

複利の計算式は次のとおりです。

>>

>複利=[元本](/principal)と将来の利息（または[将来価値）の合計から現在の元本（または](/futurevalue)[現在価値](/presentvalue)）を差し引いた金額

>>

>>

> = [P（1 + i）^ n ^] – P

>>

>>

> = P [（1 + i）^ n ^ – 1]

>>

>>

>ここで、P =元本、i =パーセンテージで表した[名目](/nominal)年利、n=複利期間の数。

>>

>複利計算期間が1年に1回を超える場合は、「i」と「n」を調整することを忘れないでください。

>>

##複利の例

毎年複合する5％の[金利](/interestrate)で$10,000の3年間のローンを取ります。興味のある金額はいくらですか？この場合、次のようになります。

>>

> $ 10,000 [（1 + 0.05）^ 3 ^] – 1 = $ 10,000 [1.157625 – 1] = $ 1,576.25

>>

複合利息を計算する場合、複合期間の数によって大きな違いが生じます。配合期間の数が多いほど、配合対象の量が多くなります。

複利計算期間が1年に1回を超える場合は、それに応じて「i」と「n」を調整する必要があります。 「i」は1年あたりの複利計算期間の数で割る必要があり、「n」は1年あたりの複利計算期間の数にローンまたは預金の[満期](/maturity)を年単位で掛けたものです。

米国証券取引委員会が運営するウェブサイトであるInvestor.govは、無料のオンライン複合利息計算機を提供しています。計算機は、元本に対して行われた毎月の預金の入力を可能にします。これは、通常の節約者に役立ちます。

##複利と単利

[元本残高](/principal)のみを考慮しますが、複合利息は元本残高と特定の期間に蓄積された利息を考慮します。

たとえば、個人が4年間で年利5％で15,000ドルを借りる場合、単純利息は15,000ドルでのみ計算されますが、複合利息は15,750ドル（15,000 x .05）になります。 1年目以降は16,537.5ドル（15,450 x .05）、2年目以降は17,364.4ドル（16,537.5 x .05）、3年目以降は17,364.4ドル（16,537.5 x .05）。

>個人でお金を借りる場合は、単純な利子ローンとしてローンを組む方がよいでしょう。節約しようとしている個人として、あなたの投資が複利であるならば、それはより良いです。

>>

単純な利息の場合、利息の合計額は15,000 x .05 x 3 = $ 2,250になり、未払いの合計額は$ 15,000 + $ 2,250 =$17,250になります。ローンが複合利息に基づいている場合よりも114ドル少なくなります。

##結論

コンパウンディングとは、時間の経過とともに初期投資に収益が繰り返し追加されるため、お金が指数関数的に成長する能力です。 1ラウンドの収益が全体の金額に追加され、より多くの金額を投資できるようになり、さらに多くの収益が生成されます。その後、このプロセスは時間の経過とともに継続的に発生し、節約を拡大できるようになります。これが、専門家ができるだけ早く投資するよう人々にアドバイスする理由です。

##ハイライト

-単純な利息は、投資された元の金額に対してのみ支払われ、時間の経過とともにゆっくりと成長します。

-複利計算期間の数が多いほど、複利の金額は大きくなります。

-元本は、新しい利息の支払いが追加されるたびに指数関数的に増加します。

-複利とは、一定期間にわたって投資された元本に利息の支払いを繰り返し追加することです。

-金融の専門家は、複利計算による時間のメリットが収益を大幅に増加させるため、個人に早期に貯蓄を開始するようアドバイスしています。

＃＃ よくある質問

###連続複利とは何ですか？

継続的な複合利息とは、利息が計算され、元本に継続的に加算される場合です。これは、週、月、または年のより一般的な間隔とは対照的に、非常に短い間隔で行われるため、最も極端な形の配合です。それは、無限の期間にわたって利息を複利にすることを目指しています。これは、実際の実用性ではなく、主に理論的な概念です。

###暗号の化合物とは何ですか？

暗号の複合は、暗号の借り入れと貸し出しを扱うプロトコルに関連しています。これは分散型のブロックチェーンベースのプロトコルであり、暗号の借用と貸し出しを容易にします。

###銀行は単利または複合利息を使用しますか？

銀行は、規制や商品の種類に応じて、複利と単利の両方を使用できます。単利はローンの元本のみで計算されますが、複利は元本と利息の両方で計算されます。お金を借りるには、単純な利子ローンを持っている方が良いです。節約するには、複利投資をする方が良いです。

###複合年間成長率とは何ですか？

年複利成長率は、投資が開始残高から終了残高まで成長するために必要な収益率である代表的な成長率です。これは、収益率が毎年同じであり、利益が毎年の終わりに再投資された場合に投資が増加したであろう率を示しています。結果をスムーズにするため、可能な投資間の比較ツールとして使用されます。

###ディスクリートコンパウンディングとは何ですか？

離散複利とは、利息が計算され、設定された間隔で元本に追加される場合です。関心が複合される一般的な間隔は、毎週、毎月、または毎年です。個別の調合は、対象が個別の調合よりも短い間隔で連続的に調合される連続的な調合とは対照的です。
