# 前進/下降指数
##前進/下降指数とは何ですか？

上昇/下降指数は、特定の指数内の上昇株と下降株の数の累積差を表す[市場幅の指標です。](/market_breadth) A / Dインデックス値の上昇は、市場が[勢いを増し](/momentum)ていることを示し、値の低下は、市場が勢いを失っている可能性があることを示しています。

前進/下降指数は、前進/下降線またはA/D指数または線とも呼ばれます。これは、現在の株価指数の[傾向を確認するために使用されます](/trend)。または、A/D指数が株価指数の方向と異なる場合に株価指数の逆転を予告することができ[ます](/divergence)。この指標は任意の時間枠で計算できますが、主に毎日の統計から計算されます。

##毎日の前進/下降指数の計算方法

1.取引[セッションの終了時に進行中の株式の数を集計し](/tradingsession)ます。

1.取引セッションの終了時に減少している株式の数を集計します。

1.前払いから下落を差し引きます。

1.ステップ3が負の場合、前のインデックス値から数値を差し引きます。ステップ3が正の場合、それを前のインデックス値に追加します。

1.初めて計算するときは、ステップ3の値のみを使用します（以前のインデックス値がないため）。これは、次の取引日の事前インデックス値として使用されます。

1.ステップ1から4を毎日繰り返します。

##前進/下降指数は何を教えてくれますか？

株価指数の上昇傾向が続く可能性を確認するために、上昇/下降指数値がよく使用されます。株価指数が上昇しているが、進行中の問題よりも下落している問題が多い場合（A / D指数が下落している場合）、通常、株価指数が幅を失い、下落する準備ができていることを示しています。

株価指数が下がると、A/D指数も下がる傾向があります。これは理にかなっています。なぜなら、株価が上昇するよりも下降する方が株価指数が低下するからです。

株価指数が下落している間にA/D指数が上昇している場合、これは強気の発散と呼ばれ、株価指数がすぐに上昇し始める兆候である可能性があります。株価は下落よりも上昇し始めているため、株価指数も間もなく上昇する可能性があります。

[テクニカルインディケータ](/technicalindicator)または[チャートパターン](/pattern)と組み合わせて前進/下降インデックスを使用して、より正確に特定のトレーディングシグナルを生成します。 A / Dインデックスは、それ自体でシグナルを提供または販売するものではありません。むしろ、株価指数の健全性について幅広い見方を示しています。

S＆P 500の前進/下降指数が現在1835であると想像してください。最終取引日の終わりに、300株が上昇（前進）し、200株が下降（下降）した場合、100が前進/下降に追加されます。インデックス値、1935にプッシュします。

##前進/下降指数と武器指数（TRIN）の違い

A / Dインデックスは、正味前進在庫数を測定する累積インデックスです。 [Arms Index](/arms) （ TRIN）は別の幅の指標ですが、ボリュームが含まれています。 TRINは、前進する株式の比率と前進する[ボリュームの比率を調べます](/volume)。これらの指標はさまざまな入力を使用しているため、株価指数の全体的な健全性を評価するために相互に組み合わせて使用できます。

##前進/下降指数の使用の制限

[Nasdaq](/nasdaq)関連の株価指数が上昇している場合でも、A/D指数は長期間低下する可能性があります。たとえば、ナスダックは[ニューヨーク証券取引所](/nyse)（NYSE）よりも投機的な株を持っている傾向があります。これらの投機的な株式は、破産したり、[上場廃止になったりする可能性が高くなり](/delisting)ます。彼らがそうする前に、彼らはA / Dインデックスを引き下げ、現在取引所に上場されている株が好調で上昇しているかもしれないとしても、彼らの悪影響は残っています。

A / Dインデックスは、常に逆転を予見するわけではありません。多くの場合、それは単に価格と同じパターンで動くだけです。株価指数が逆転するたびに発散が見られるわけではありません。

株価指数内の傾向が強いままであっても、A/Dラインは時々矛盾するシグナルを提供するかもしれません。または、A / Dラインが強くトレンドになっている可能性がありますが、株価指数の方向性は期待どおりに進んでいません。

##ハイライト

-上昇するA/D指数は、上昇する株価指数を確認するのに役立ち、下降するよりも上昇する株の方が多いため、強さを示します。

-A / Dラインの上昇に伴う株価指数の低下は[強気](/bull)の発散であり、より多くの株式が上昇し始めているため、株式市場が上昇する可能性があることを示しています。

-上昇/下降を伴う株価指数の上昇は[弱気の](/bear)発散であり、上昇に参加している株式が少ないため、株式市場の上昇が勢いを失っていることを示しています。

-下落しているA/Dインデックスは、下落している株価指数を確認するのに役立ちます。これは、上昇するよりも下降する株の方が多いため、弱さを示しています。
