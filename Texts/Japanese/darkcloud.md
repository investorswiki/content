# ダーククラウドカバー
##ダーククラウドカバーとは何ですか？

ダーククラウドカバーは弱気の反転[ローソク足](/candlestick)パターンで、ダウンキャンドル（通常は黒または赤）が前のアップキャンドル（通常は白または緑）の近くで開き、次にアップキャンドルの中間点の下で閉じます。

このパターンは、勢いが上向きから下向きにシフトしていることを示しているため、重要です。パターンは、アップキャンドルとそれに続くダウンキャンドルによって作成されます。トレーダーは、次の（3番目の）キャンドルで価格が下がり続けることを求めています。これは[確認と呼ばれ](/confirmation)ます。

##ダーククラウドカバーを理解する

ダーククラウドカバーパターンには、前のアップキャンドルの上に「ダーククラウド」を形成する大きな黒いキャンドルが含まれます。[弱気](/bearishengulfingp)[な飲み込みパターンと](/bearishengulfingp)同様に、買い手はオープン時に価格を高くしますが、売り手はセッションの後半に引き継ぎ、価格を大幅に下げます。この購入から販売への移行は、価格が下落に転じる可能性があることを示しています。

ほとんどのトレーダーは、ダーククラウドカバーパターンが上昇傾向または全体的な価格の上昇に続いて発生する場合にのみ有用であると考えてい[ます](/uptrend)。価格が上昇するにつれて、パターンは下振れへの潜在的な動きを示すためにより重要になります。価格アクションが[不安定な場合](/choppymarket)、パターンの後で価格が不安定なままになる可能性があるため、パターンはそれほど重要ではありません。

ダーククラウドカバーパターンの5つの基準は次のとおりです。

1.既存の強気の[上昇トレンド](/uptrend)。

1.その上昇トレンド内の上昇（[強気](/bull)）キャンドル。

1.翌日の[ギャップアップ。](/gap)

[弱気](/bear)）キャンドルに変わります。

1.弱気のキャンドルは、前の強気のキャンドルの中間点より下で閉じます。

[実体](/realbody)が長く、[影](/shadow)が比較的短いか存在しない白と黒のローソク足がさらに特徴的です。これらの属性は、下落が価格変動の観点から非常に決定的かつ重要であったことを示唆しています。トレーダーはまた、パターンに従って弱気のろうそくの形で確認を探すかもしれません。 Dark Cloud Coverに続いて価格が下がると予想されるため、そうでない場合はパターンが[失敗する可能性があり](/failedbreak)ます。

[長い](/long)位置を出るために使用されるかもしれません。あるいは、価格が下落し続ける場合、トレーダーは翌日終了する可能性があります（パターンが確認されました）。弱気のろうそくの終わり、または次の期間に[不足する](/short)場合は、弱気のろうそくの高さより上に[ストップロス](/stop-lossorder)を置くことができます。

ダーククラウドカバーパターンの[利益目標](/profit-target)はありません。トレーダーは、Dark Cloud Coverに基づいて、ショートトレードをいつ終了するかを決定するために、他の方法またはローソク足パターンを利用します。

トレーダーは、他の形式の技術分析と組み合わせてダーククラウドカバーパターンを使用できます。たとえば、トレーダーは70を超える[相対強度指数（RSI）を探す場合があります。これは、セキュリティが買われ過ぎであることを確認するものです。トレーダーは、](/rsi)[下降トレンド](/downtrend)が近づいている可能性があることを示すシグナルとして、ダーククラウドカバーパターンに続く主要な[サポートレベル](/support)からの[内訳](/breakdown)を探すこともできます。

この例では、3番目の強気のキャンドルの後に、最後の強気のキャンドルの中間点より高く開いて閉じる弱気のキャンドルが続くときに、ダーククラウドカバーが発生します。このパターンは、価格が7％近く下がった次のセッションでの下降をうまく予測しました。そのセッションは確認を提供しました。

長い間トレーダーは、弱気のろうそくの終わり近くか、価格が下がり続けた翌日（確認日）に退出することを検討することができました。トレーダーは、これらの分岐点でもショートポジションに入る可能性があります。

短く入ると、最初のストップロスは弱気のキャンドルの高さより上に配置される可能性があります。この場合、確認日の翌日には、ストップロスが確認日の高さのすぐ上まで下がる可能性があります。その後、トレーダーは下振れ利益目標を設定するか、価格が下落し続ける場合は[トップロスを](/trailingstop)[引き下げ続けます。](/trailingstop)

##ハイライト

-両方のキャンドルは比較的大きく、トレーダーと投資家の強い参加を示している必要があります。パターンが小さなキャンドルで発生する場合、通常はそれほど重要ではありません。

-Dark Cloud Coverは、価格の上昇に続いて勢いが下向きにシフトすることを示すローソク足のパターンです。

-トレーダーは通常、弱気のキャンドルに続くキャンドルも価格の下落を示しているかどうかを確認します。弱気のろうそくに続くさらなる価格の下落は確認と呼ばれます。

-パターンは、前の強気のキャンドルの中間点の上で開き、次に閉じる弱気のキャンドルで構成されています。
