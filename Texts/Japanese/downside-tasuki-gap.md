# 欠点たすきギャップ
##欠点たすきギャップとは何ですか？

Downside Tasuki Gapは、現在の[下降トレンド](/downtrend)の継続を示すために一般的に使用される[ローソク足の形成です](/candlestick)。このパターンは、一連のローソク足が次の特性を示したときに形成されます。

1.最初のろうそく足は、既存の下降トレンド内で赤または逆（下）です。

2. 2番目のろうそく足は前のバーの終値の下に[ギャップ](/gap)があり、これも赤（下）です。

3.最後のバーは、最初の2つのバーのギャップ内で**閉じる**白または緑（上）のローソク足です。白いろうそくがギャップを完全に閉じる必要はないことに注意することが重要です。

##欠点のたすきギャップは何を教えてくれますか？

欠点たすきギャップ（弱気たすきギャップとも呼ばれます）は、3つのキャンドルの[継続パターン](/continuationpattern)です。このパターンを見つけるには、次の基準に留意してください。

まず、明確な下降トレンドが存在する必要があり、大きな赤/ダウンキャンドルが存在する必要があります（キャンドルスティックチャートの色はカスタマイズ可能です）。

第二に、上記のろうそくに続いて、価格はギャップダウンし、別の大きな赤/黒のろうそくを形成する必要があります。

第三に、白/緑のキャンドルは赤/黒のキャンドルに続く必要があります。緑/白のキャンドルは、赤いキャンドルの実体の内側で開き、その上で閉じる必要があります。このキャンドルは、最初の2つのキャンドルの間のギャップを埋めてはなりません。

関係する3つのキャンドルのうち、最初の3分の2は赤/黒で、緑/白になります。資格を得るには、2番目と3番目のキャンドルが反対の色である必要があります。

ダウンサイドタスキギャップパターンは、下降トレンドの力を示しています。[クマはコントロールし](/bear)ていて、その強さを示しています。その後、この下向きの強さは増幅され、価格が下落し、新しい赤いろうそくが形成されることで示されます。しかし、ブルズが価格を押し上げようとするため、この動きに続いて一時停止します。価格がギャップを埋めることができず、価格が再び下落し始めた場合、強気筋は逃げる可能性が高く、既存の下降トレンドを再開することになります。

一部のトレーダーは、下降トレンドが続くと想定して、白いろうそくの終わり近くで[ショートに入るのを選びます。](/short)他の人は、価格が白いろうそくの安値または始値を下回るのを待つことを好みます。これにより、価格が再び下落し、下降トレンドが再開している可能性があることを確認できます。

Downside [Tasuki Gapには、対応するUpsideTasukiGapがあり](/upside-tasuki-gap)ます。上記と同じ基準を表示することで見つけることができますが、反対のフォーメーションで上昇傾向にあります。

##下振れのタスキギャップをトレードする方法の例

Nvidia Corp. （NVDA）チャートは、下向きのTasukiギャップパターンを示しています。パターンが表示されるとき、価格は短期的な下降傾向にあります。ダウンキャンドルがあり、その後に別のダウンキャンドルへのギャップが続きます。次に、ギャップに侵入するがそれを閉じないアップキャンドルがあります。

トレーダーは、その白いろうそくの終わり近くでショートに入る可能性があり、パターンの最初のろうそくの終わりまたは高さより上で[ストップロスが発生します。](/stop-lossorder)

アップキャンドルに続いて、価格は下がります。この場合、再びギャップダウンします。

ただし、下落の動きは短く、価格はすぐに元に戻ります。

##下側のたすきギャップと下側のギャップを並べた白い線のパターンの違い

たすきの隙間は部分的にアップキャンドルで埋められています。ギャップダウンの[白い線の](/updown-gap-sidebyside-white-lines)パターンは、ギャップダウンの後にキャンドルが並んでいるパターンです。このパターンは、現在のトレンドの継続を示しています。

##下側のたすきギャップの使用の制限

パターンは、他の価格バーの海にある3つのキャンドルです。このパターンに焦点を当てることにより、トレーダーはコンテキストを失う可能性があります。たとえば、このパターンが発生すると、短期的な傾向は低下する可能性がありますが、長期的な傾向は上昇する可能性があります。したがって、価格は、より長く、より支配的な傾向に沿って、パターンの直後に上昇する可能性があります。

このパターンはあまり一般的ではないため、取引の機会は限られています。

示されているように、このパターンではコンテキストが重要です。下降トレンドと売り圧力が強いほど、価格は下がり続ける可能性が高くなります。価格行動は[不安定](/choppymarket)、乱暴、または弱い傾向にありますが、成功の確率は下落パターンです。

パターンの後で、価格がどこまで下がるか、またはまったく下がるかどうかは示されていません。これには、別の形式の分析が必要です。

ローソク足パターンを取引する前に、勝者と敗者の両方を含む、パターンがどのように実行されたかの歴史的な例を探してください。

##ハイライト

-パターンは、パターンに従って価格がどこまで下落する可能性があるか（下落した場合）を示していません。

-パターンは下降トレンドで発生し、その下降トレンドの潜在的な継続を示します。

-それは、ダウンキャンドルがあり、ギャップが別のダウンキャンドルに下がってから、ギャップ内で閉じるアップキャンドルがあるときに形成されます。
