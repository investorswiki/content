# バリューネットワーク分析
##バリューネットワーク分析とは何ですか？

[ネットワーク](/value-network)内でのこれらのメンバーの相互作用です。バリューネットワーク分析は通常、チャートまたはWebを使用して関係を視覚化することによって行われます。

バリューネットワーク分析の参加者は、個別に評価され、ネットワークにもたらすメリットについて評価されます。バリューネットワーク分析では、業務の財務的側面と非財務的側面を含め、ビジネス全体を調べます。

##バリューネットワーク分析を理解する

価値ネットワーク分析は、ビジネスの財務的および非財務的価値と側面の両方を評価する方法を提供します。ほとんどの形式の分析は、通常、各ネットワークの異なるポイント間で発生する重要な関係とトランザクションの図またはマップを介して、視覚的な形式で実行されます。これらのポイントは一般に、個人、グループ、ビジネスユニット、さらには業界内の個々のビジネスなどの人々を表しています。

バリューネットワークは、製品の製造中またはサービスの提供中のメンバーとその相互作用で構成されます。これらのつながりは、強力な企業を特定し、企業の潜在的なリスクを見つける上で非常に重要です。

>バリューネットワーク分析は、企業の強みとビジネスのリスクを特定するのに役立ちます。

>>

たとえば、ネットワークメンバーが大きな影響力を持っている場合、そのメンバーの喪失はグループ全体を荒廃させる可能性があります。これは価値があるので本質的価値分析として知られていますが、値札を付けるのは難しいです。

##バリューネットワーク分析の適用

バリューネットワーク分析を通じて適用される方法論は、組織が内部および外部のバリューネットワークを最適化し、運用内のチームの相乗効果とともに外部との関係を最大限に活用するのに役立ちます。これには、組織に織り込まれた関係全体での知識、情報、専門知識の交換が含まれます。分析の目的は、ピーク時に運用し、全体的な生産性を向上させるために、すべての関係者とのコミュニケーションとコラボレーションを改善することです。

バリューネットワーク分析のアプリケーションは、組織の内部再構築、相互に関連する部門間のワークフローの改善、およびプロジェクト計画などのニーズに役立ちます。この分析は、統合が必要な新しい部門や業務とのつながりを深め、最大限に活用するために、合併や買収を行う組織を支援することもできます。

企業が[プロセスの再設計を行って](/business-process-redesign)おり、大規模なオーバーホールと新しいフレームワークを確立する必要がある場合は、バリューネットワーク分析を適用して、行う必要のある変更をより明確に把握できます。組織が新しいビジネスモデルを策定する必要がある場合は、バリューネットワーク分析アプローチを適用して、そのようなモデルの開発や、新しいモデルの今後の運用方法に関する新しい洞察を提供するために利用できるリソースを特定できます。

の[研究開発](/research-and-development-expenses)（R＆D）の側面も、新しいサービスや製品の作成に協力するために利用できる情報と専門知識を特定することにより、バリューネットワーク分析の恩恵を受けることができます。

##内部と外部のバリューネットワーク

前述のように、バリューネットワーク分析には内部と外部の2種類があります。名前が示すように、内部のブランチまたは要因はビジネス内にあります。これらには、従業員、経営陣、事業内のさまざまな部門、および社内で行われるプロセスと活動が含まれる場合があります。

場合によっては、これらのネットワークによって生み出される価値は、同じ目標のために一緒に働く2人の関係など、ビジネス以外の場合にも適用できます。内部ネットワークの価値は、ビジネス内のこれらの異なるポイント間の関係を分析することによって評価されます。

一方、外部価値ネットワーク分析は、ビジネス外の要因に依存しています。これには、そのサプライヤー、ビジネスパートナー、および会社の他の利害関係者、およびその顧客と他のエンドユーザーの分析が含まれる場合があります。外部価値ネットワーク分析が実施されるとき、それはこれらの外部要因によって生み出されたビジネスとの関係と価値をレビューします。

##ハイライト

-バリューネットワーク分析の参加者は、個別に評価され、ネットワークにもたらすメリットについても評価されます。

-バリューネットワーク分析は、組織のメンバーとバリューネットワーク内でのそれらの相互作用の評価です。

-バリューネットワークは、内部（ビジネス内の要因）または外部（ビジネス外の要因）の場合があります。

-分析は、一般的に視覚的に、一般的には図または地図の形で描かれます。
