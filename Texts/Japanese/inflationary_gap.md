# インフレギャップ
##インフレーションギャップとは何ですか？

[実質国内総生産](/realgdp)（GDP）のレベルと、経済が[完全な雇用](/fullemployment)で運営されている場合に存在するであろう[GDPと](/gdp)の差を測定するマクロ経済概念です。

##インフレギャップを理解する

[貿易](/trade)活動の増加、政府支出の増加などの要因により、財とサービスの需要が生産を上回った場合、インフレギャップが存在します。

このような背景から、実質GDPは潜在GDPを上回り、インフレギャップをもたらす可能性があります。実質GDPの相対的な上昇により[経済](/economy)が消費を増加させ、長期的には価格が上昇するため、インフレギャップはそのように名付けられています。

>ギャップがインフレと見なされるためには、現在の実質GDPは、完全雇用経済のGDP（潜在GDPとも呼ばれる）よりも高くなければなりません。

>>

インフレギャップは、経済が拡大している[ビジネスサイクルのポイントを表しています。](/businesscycle)経済の中で利用できる資金の数が多いため、消費者は商品やサービスを購入する傾向があります。商品やサービスの需要は増加しますが、生産はまだシフトを補っていないため、価格は上昇して市場の[均衡を回復します](/equilibrium)。

潜在的なGDPが実際のGDPよりも高い場合、そのギャップは代わりにデフレギャップと呼ばれます。もう1つのタイプの需給ギャップは[景気後退ギャップ](/recessionarygap)であり、これは経済が完全雇用均衡を下回って運営されていることを表しています。

##実質国内総生産（GDP）の計算

[マクロ経済](/macroeconomics)理論によれば、財市場は実質GDPの水準を決定し、それは次の関係で示されます。実質GDPを計算するには、最初に[名目GDPを計算し](/nominalgdp)ます。

Y = C + I + G + NX

どこ：

-Y=名目GDP

-C=消費支出

-I=投資

-G=政府支出

-NX=[純輸出](/netexports)

次に、実質GDP = Y / Dです。ここで、Dは[GDPデフレーター](/gdppricedeflator)であり、時間の経過とともにインフレが有効になります。

消費支出、投資、政府支出、または純輸出の増加は、短期的には実質GDPを上昇させます。実質GDPは、[インフレ](/inflation)または[デフレ](/deflation)の影響を補いながら、経済成長の尺度を提供します。これは、実際の経済成長と経済内の商品やサービスの価格の単純な変化との違いを説明する結果を生み出します。

##インフレギャップを管理するための財政および金融政策

政府は、多くの場合、経済内で循環する資金の数を減らすことによって、インフレギャップを減らすのを助けるために[財政政策を使用することを選択するかもしれません。](/fiscalpolicy)これは、政府支出の削減、増税、[債券](/bond)および証券の発行、および送金の削減を通じて達成できます。

[経済均衡](/economic-equilibrium)を回復するのに役立つ可能性があります。流通量が減少すると、商品やサービスの全体的な需要も減少し、インフレが減少します。

中央銀行はまた、インフレ活動と戦うためのツールを自由に利用できます。[連邦準備制度（FRB）が](/federalreservebank)[金利](/interestrate)を引き上げると、借入金はより高額になります。

[厳格な通貨政策](/tightmonetarypolicy)は、その後、ほとんどの消費者が利用できる金額を下げ、需要と価格の低下、またはインフレの後退を引き起こすはずです。均衡に達すると、連邦政府または他の[中央銀行](/centralbank)はそれに応じて金利をシフトすることができます。

##ハイライト

-ギャップがインフレと見なされるためには、現在の実質GDPが潜在的なGDPよりも高くなければなりません。

-インフレギャップは、実際のGDPの現在のレベルと、経済が完全な雇用で運営されている場合に存在するであろうGDPとの差を測定します。

-インフレギャップを減らすことができる政策には、政府支出の削減、増税、債券と証券の発行、金利の引き上げ、および送金の支払いの削減が含まれます。
