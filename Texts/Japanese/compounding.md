# 複利
##複利とは何ですか？

[キャピタルゲイン](/capitalgain)または[利息](/interest)のいずれかからの資産の収益を再投資して、時間の経過とともに追加の収益を生み出すプロセスです。指数関数を使用して計算されたこの成長は、投資が初期の元本と前の期間からの累積収益の両方から収益を生み出すために発生します。

したがって、複利は、各期間に元本のみが利息を得る線形成長とは異なります。

*

##複利を理解する

複利とは、通常、元本と累積利息の両方で得られる利息による資産の価値の増加を指します。この現象は、[時間価値（TMV）の概念](/timevalueofmoney)を直接実現したものであり、複合利益としても知られています。

複利は[資産](/asset)と負債の両方に作用します。コンパウンドは資産の価値をより迅速に高めますが、未払いの元本と以前の利息に利息が蓄積されるため、ローンの未払い額も増える可能性があります。

複利がどのように機能するかを説明するために、毎年5％の利息を支払うアカウントに10,000ドルが保持されているとします。最初の年または複利計算期間の後、アカウントの合計は$ 10,500に上昇しました。これは、$10,000の[元本](/principal)に$500の利息が追加されたことを単純に反映したものです。 2年目に、アカウントは元の元本と1年目の利息の500ドルの両方で5％の成長を実現し、2年目の利益は525ドル、残高は11,025ドルになります。 10年後、引き出しがなく、5％の安定した金利を想定すると、アカウントは$16,288.95に増加します。

##複利計算期間の延長

配合の頻度が高くなると、配合の効果が強くなります。 1年の期間を想定します。この1年間の配合期間が長いほど、投資の将来の価値は高くなります。したがって、当然、1年に2回の配合期間は、1年よりも優れており、1年に4回の配合期間は2よりも優れています。

この効果を説明するために、上記の式が与えられた次の例を考えてみましょう。 100万ドルの投資が年間20％を稼ぐと仮定します。さまざまな数の配合期間に基づく結果の将来の値は、次のとおりです。

-年間配合（n = 1）：FV = $1,000,000×[1+（20％/ 1）] ^（1 x 1）^ = $ 1,200,000

-半年ごとの調合（n = 2）：FV = $1,000,000×[1+（20％/ 2）] ^（2 x 1）^ = $ 1,210,000

-四半期ごとの複合（n = 4）：FV = $1,000,000×[1+（20％/ 4）] ^（4 x 1）^ = $ 1,215,506

-毎月の複利（n = 12）：FV = $1,000,000×[1+（20％/ 12）] ^（12 x 1）^ = $ 1,219,391

-毎週の複利（n = 52）：FV = $1,000,000×[1+（20％/ 52）] ^（52 x 1）^ = $ 1,220,934

-毎日の複利計算（n = 365）：FV = $1,000,000×[1+（20％/ 365）] ^（365 x 1）^ = $ 1,221,336

明らかなように、年間の配合期間の数が大幅に増加しても、将来の値はわずかなマージンで増加します。設定された期間にわたる複合の頻度は、投資の成長に限定的な影響を及ぼします。微積分に基づくこの制限は、[連続複利として知られ](/continuouscompounding)ており、次の式を使用して計算できます。

##複利の例

[金融において](/finance)非常に重要であり、その効果に起因する利益は多くの投資戦略の背後にある動機です。たとえば、多くの企業は、投資家が現金[配当を](/dividend)再投資して追加の株式を購入できるようにする配当再投資計画（DRIP）を[提供](/stock)し[てい](/dividendreinvestmentplan)ます。安定した配当を前提として、株式数の増加は配当支払いからの将来の収入を一貫して増加させるため、これらの配当支払い株式のより多くに再投資することは投資家の利益を悪化させます。

配当金の再投資に加えて配当成長株への投資は、この戦略に別の複合層を追加します。これは、一部の投資家が二重複合と呼んでいます。この場合、より多くの株式を購入するために配当が再投資されるだけでなく、これらの配当成長株は、1株あたりの支払いも増やしています。

##ハイライト

-したがって、複利は、利息に対する利息として解釈できます。その効果は、時間の経過とともに利息へのリターンを拡大することであり、いわゆる「複利の奇跡」です。

-銀行または金融機関が複利を貸方に記入する場合、年次、月次、または日次などの複利期間を使用します。

-複合とは、利息が既存の元本とすでに支払われた利息に貸方記入されるプロセスです。

＃＃ よくある質問

###調合に最適な平均のタイプはどれですか？

財務で使用される平均（[平均）計算](/mean)にはさまざまな種類があります。複合を含む投資または貯蓄口座の平均収益を計算するときは、[幾何学的平均を使用するのが最善](/geometricmean)です。金融では、これは[時間加重平均収益](/time-weightedror)または[複合年間成長率（CAGR）として知られることもあり](/cagr)ます。

###単純な関心と複合的な関心の違いは何ですか？

単純利息は、投資または預け入れられた元本の金額にのみ利息を支払います。たとえば、$ 1,000が5％の単純な利息で預け入れられた場合、毎年$50を獲得します。ただし、複合利息は「利息利息」を支払うため、1年目は50ドルを受け取り、2年目は52.5ドル（1,050ドル×0.05）を受け取ります。

###投資家はどのようにして複利収益を受け取ることができますか？

複利に加えて、投資家は配当を再投資することで複利を受け取ることができます。これは、配当金の支払いから受け取った現金を会社の追加の株式を購入するために使用することを意味します。これは、将来、それ自体が配当金を支払うことになります。

###複合的な関心を持つ72のルールとは何ですか？

[ルール72は](/ruleof72)、複合的な利益（または複合的な利益）がある場合に投資または節約の価値が2倍になる期間を見積もるために使用されるヒューリスティックです。規則では、2倍になるのにかかる年数は72を金利で割ったものであるとされています。したがって、配合で5％の場合、2倍になるまでに約14年5ヶ月かかります。
