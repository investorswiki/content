# 利息のクレジット方法
##インタレストクレジットの方法とは何ですか？

利息貸方法は、固定インデックス年金（FIA）への利息の変化をどのように測定するかを決定します。選択された利息クレジット方法は、年金保有者が特定の期間に受け取ることができる利息の量を測定します。

[年金](/annuity)契約には、利息の取り扱いに影響を与える多くの条項があります。ほとんどの連絡先には、上限（許可される最大利息）、参加率（契約に貸方記入される利息の割合）、およびスプレッドの組み合わせが含まれます。これらは、インデックス値の増加の可能性を制限します。

##利息のクレジット方法を理解する

[インデックス付き年金](/indexedannuity)は、S＆P 500インデックスなどのエクイティインデックスのパフォーマンスにリンクされたキャッシュフローを返しますが、クレジットされる最大リターンには上限があります。ポイントツーポイントのインタレストクレジット方式では、インデックスの値の増加は2つの時点から計算されます。これは計算するのに最も簡単な利息クレジット方法ですが、年金契約者に最大の利益を提供しない場合があります。たとえば、インデックスが期間の開始時に1,000と評価され、終了までに1,150に増加した場合、ポイントツーポイント方式ではこれを15％の増加（150/1000 x 100）と呼びます。インデックスの価値が下がった場合、契約はその価値を失うことはありませんが、利息は契約に追加されません。

月平均法は、各月末の指数の値を取り、それらを平均します。これは、月末の値をすべて合計し、12で割るのと同じくらい簡単です。たとえば、インデックスが1,000で年を開始し、平均インデックス値が1,200である場合、月間平均は20パーセント（200 / 1,000 x 100）になります。この方法は、不安定な市場で検討される可能性があります。

月次合計法は、毎月のインデックスの増加または減少のパーセンテージを取得し、それらを合計します。インデックスは、月ごとに正の領域に入るか、負の領域に分類される可能性があります。パーセンテージが合計されてプラスになる場合、利息は契約にクレジットされます。この方法は、揮発性に最も敏感です。

##インタレストクレジット方式のタイプ

他の種類の年金では、利息のクレジット方法が異なります。通常の定額年金は、 [T-billsレートにリンクされたレートで利息をクレジットし](/treasurybill)ます。一方、固定インデックス付き年金は、特定のインデックスの変更に基づく数式を使用して利息をクレジットします。クレジット方法は、年金にクレジットされる利息の額を決定します。クレジットのレートと頻度は、FIA契約の条件によって異なります。

保険会社が使用するこれらの代替の利息クレジット方法には、次のものがあります。

-年間のポイントツーポイント平均

-隔年のポイントツーポイント平均

-毎月のポイントツーポイント平均

-日次平均/月次平均

-後知恵指数戦略の月間平均

##ハイライト

-インデックス付き年金の場合、クレジット方式では、株価指数の価格変動の月平均をとることができる式が必要です。

-利息のクレジット方法は、インデックス付き年金の年金受給者によるキャッシュフローを決定するためのアプローチです。

-標準の定額年金は、政府が発行した国庫証券にリンクされた固定レートで年金受給者を信用します。
