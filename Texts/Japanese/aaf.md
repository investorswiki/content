# アセットアロケーションファンド
##アセットアロケーションファンドとは何ですか？

資産配分ファンドは、投資家にさまざまな資産クラスにわたる多様な投資ポートフォリオ[を提供するファンドです。](/diversification)ファンドの資産配分は、資産クラスの組み合わせの間で固定または変動する可能性があります。つまり、市場の状況に応じて、資産クラスの固定パーセンテージに保持されるか、一部でオーバーウェイトになる可能性があります。

資産配分ファンドの人気のある資産カテゴリーには、株式、債券、[現金同等物が含まれます。これら](/cashequivalents)は、さらに分散するために地理的に分散される場合もあります。

##アセットアロケーションファンドを理解する

アセットアロケーションファンドは、[現代ポートフォリオ理論から開発されまし](/modernportfoliotheory)た。現代ポートフォリオ理論は、投資家が[効率的なフロンティア](/efficientfrontier)に含まれる投資の分散ポートフォリオに投資することによって最適なリターンを達成できることを示しています。

現代ポートフォリオ理論投資の標準的なアプリケーションには、株式、債券、および現金同等物の効率的なフロンティアが含まれます。さらに、現代ポートフォリオ理論は、投資家の[リスク許容度](/risktolerance)に合わせてポートフォリオが資産構成をどのように変化させることができるかを概説しています。

##アセットアロケーションファンドの種類

資産配分ファンドは、投資家向けにさまざまな配分と資産の組み合わせを備えた現代ポートフォリオ理論の簡素化されたアプリケーションを提供します。

資産配分ファンドの最も一般的なタイプの1つは、バランスの[取れたファンド](/balancedfund)です。バランスの取れたファンドとは、60％の株式や40％の債券など、株式と[債券のバランスの取れた配分を意味します。](/fixedincome) 60/40ミックスは、幅広い市場の多様化を求める投資家に人気のある標準化された戦略となっているため、投資家は多数のファンドを見つけるでしょう。

アセットアロケーションファンドはまた、リスク許容度に基づいてさまざまなレベルの分散を提供します。 60/40を超える追加の投資カテゴリーを探している投資家は、保守的な配分ファンド、中程度の配分ファンド、積極的な配分ファンドなど、多くのオプションを見つけるでしょう­。

通常、[退職計画で使用される](/retirement-planning)ライフサイクルまたは[目標日ファンド](/target-date_fund)も、資産配分ファンドの一種と見なされます。これらのファンドは、リスクリターンの高いポジションから始まり、ファンドが目標の利用日に近づくにつれて徐々にリスクが低くなる、ターゲットを絞った資産クラスの組み合わせで管理されます。

対象となる資産配分を決定した後、ファンドはさまざまな方法で投資選択を管理できます。一部のファンドは、さまざまな市場エクスポージャーを表すために、さまざまな[上場投資信託（ETF）に投資することを選択する場合があります。他のファンドは、](/etf)[ファンダメンタル分析](/fundamentalanalysis)を使用して各資産クラスで最高のパフォーマンスを発揮する証券を選択することにより、より積極的なアプローチを取る可能性があります。

全体として、ほとんどのファンドは、変化する市況や経済環境に応じて、証券を積極的に監視し、配分またはリバランスします。

##人気のあるアセットアロケーションファンド

以下は、投資業界のトップアセットアロケーションファンドの例です。

-** iShares Core Aggressive Allocation ETF（AOA）**：iShares Core Aggressive Allocation ETFは、S＆P Target RiskAggressiveIndexのパフォーマンスを再現しようとする[トラッカーファンドです。](/trackerfund)ファンドは、インデックスを複製しようとするターゲットETFに投資します。インデックスは株式に重点を置いており、リスク許容度の高い投資家を対象としています。

-** iShares Core Conservative Allocation ETF（AOK）**：iShares Core Conservative Allocation ETFは、S＆P Target RiskConservativeIndexのパフォーマンスを再現しようとするトラッカーファンドです。ファンドは、インデックスを複製しようとするETFに投資します。インデックスは債券に大きく重点を置いており、より保守的なリスク許容度を持つ投資家を対象としています。

-** Vanguard Balanced Index（VBIAX）**：資産配分ファンドを探している投資家は、Vanguardで多くのオプションを見つけるでしょう。同社のVanguardBalancedIndexファンドは、約60％を株式に、40％を債券に投資しています。その保有は、CRSP US TotalMarketIndexとBloombergUSAggregate Float AdjustedBondIndexの2つのインデックスを追跡しようとしています。

##ハイライト

-最も一般的な資産配分ファンドには、バランスファンドと目標日ファンドが含まれます。

-アセットアロケーションファンドは、現代ポートフォリオ理論の産物です。

-資産配分ファンドは、投資家のリスク許容度を考慮して、最適なポートフォリオを作成しようとします。

-現代ポートフォリオ理論投資の標準的なアプリケーションには、株式、債券、および現金同等物の効率的なフロンティアが含まれます。

-アセットアロケーションファンドには、無限のバリエーションがあります。ファンドはすべて最適な分散を求めますが、それらはすべて資産クラスの異なる組み合わせを持ち、独自の内部ルールに従います。

-投資家は、投資信託または上場投資信託（ETF）の株式を購入することにより、資産配分ファンドにアクセスできます。
