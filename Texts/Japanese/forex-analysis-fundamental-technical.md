# 外国為替分析
##外国為替分析とは何ですか？

外国為替分析は、[通貨](/currencypair)[ペア](/currencypair)の価格の変化とそれらの価格の変化に影響を与える力を調べる慣行です。これは、利益を上げることを目的として通貨を売買する外国為替トレーダーによって使用されます。[基本](/fundamentalanalysis)的な分析方法とテクニカル分析方法の両方が外国為替取引で使用されており、多くのトレーダーは両方の技術を組み合わせたハイブリッドアプローチを使用しています。

##外国為替分析の仕組み

外国為替市場は世界最大の市場であり、毎日何兆ドルもの資金が手に渡っています。小売業者と金融会社はどちらも、この大きくて重要な市場から利益を得ようと定期的に外国為替分析を行っています。今日、外国為替市場は24時間年中無休で開かれており、ほとんどの活動はロンドン、ニューヨーク、東京などのグローバルな貿易ハブに集中しています。

投資家が個々の会社の株式を購入できる株式市場とは異なり、外国為替市場で取引される通貨は常にペアで取引されます。ペアの通貨の1つが購入されると、これは必然的にペアのもう1つの通貨が販売されることを意味します。世界で最も一般的に取引されている通貨ペアには、USD / CAD、 [EUR / USD](/eur-usd-euro-us-dollar-currency-pair) 、EUR/JPYなどがあります。

外国為替分析は、ペア内のどの通貨が特定の期間にわたってより強くなる可能性が高いかを判断する手法です。外国為替トレーダーは、この知識を使用して、価値が上がると予想される通貨を購入したり、価値が下がると思われる通貨を売ったりすることができます。これらの予測を支援するために、外国為替トレーダーはさまざまな情報源を調査します。これらには、各国の経済状況、石油などの関連[商品の価格](/commodity)、または国際経済に影響を与える主要なニュースイベントなどの基本的な要因が含まれます。過去の平均に対する通貨の最近の価格履歴など、技術的要因も考慮されます。

##外国為替分析の実際の例

説明のために、世界で最も取引量の多いペアの1つであるUSD/CAD通貨ペアの場合を考えてみましょう。カナダと米国は緊密な貿易相手国です。つまり、国境を越えた貿易、旅行、投資などの理由で、両者の間で大量の外国為替が発生しています。

この通貨ペアを見る際に、ファンダメンタル分析を使用する外国為替トレーダーは、両方の通貨の[金利](/interestrate)やインフレ見通しなどの要因を見る可能性があります。彼らはまた、将来の金利動向への洞察を提供するかもしれない雇用などの要因を検討します。強い経済は金利を押し上げる傾向があり、より多くの投資家をその通貨とその市場に引き付け、より高い金利の恩恵を受けるでしょう。カナダドルは石油価格とともに変動する傾向があるため、トレーダーは石油価格の将来の方向性についての期待も考慮に入れる可能性があります。

一方、テクニカルアナリストは、USD/CADの価格履歴内のトレンドと範囲を探す可能性があります。トレンドは通貨ペアが動いている全体的な方向性を提供し、[範囲](/range)は価格が近づいている支持または抵抗の領域を示す場合があります。また、価格変動のリズムやパターンを見つけるためにテクニカル指標を使用する場合もあります。これらのパターンをグラフ化することにより、アナリストは、価格がパターン内でどのように動くかを予測し、いつそれが過去の範囲から外れるかを予測することができます。

##ハイライト

-通貨はペアで取引され、為替レートは一方の通貨の他方の通貨の価格に基づいています。

-主要なタイプの分析には、技術的および基本的なものが含まれ、多くのトレーダーは両方のアプローチを組み合わせて使用しています。

-外国為替分析は、通貨ペアを購入するか、売却するか、取引を待つかを決定する研究です。
