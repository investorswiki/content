# マイニングプール
##マイニングプールとは何ですか？

[暗号通貨マイナー](/cryptocurrency)の共同グループであり、ネットワークを介して計算リソースを組み合わせて、[ブロックを見つける](/block-bitcoin-block)か、暗号通貨のマイニングに成功する可能性を高めます。

##マイニングプールの仕組み

個別に、マイニングプールの参加者は、ブロックを見つける作業に向けて処理能力を提供します。プールがこれらの取り組みに成功した場合、通常は関連する暗号通貨の形で報酬を受け取ります。

報酬は通常、グループ全体に対する各個人の処理能力または作業の割合に応じて、貢献した個人間で分割されます。場合によっては、個々の鉱夫は報酬を受け取るために[仕事の証拠を示さなければなりません。](/proof-work)

>報酬は通常、合意された条件と鉱業活動へのそれぞれの貢献に基づいて鉱夫の間で分割されます。

>>

暗号通貨マイニングを通じて利益を上げたい人は誰でも、自分の専用デバイスでソロになるか、複数のマイナーとそのデバイスを組み合わせて[ハッシュ](/hash)出力を強化するマイニングプールに参加するかを選択できます。たとえば、それぞれが335メガハッシュ/秒（MH / s）を提供する6つのマイニングデバイスを接続すると、累積2ギガハッシュのマイニングパワーを生成できるため、ハッシュ関数の処理が高速になります。

##マイニングプールメソッド

すべての暗号通貨マイニングプールが同じように機能するわけではありません。ただし、最も人気のあるマイニングプールの多くを管理する一般的なプロトコルがいくつかあります。

比例マイニングプールは最も一般的なものの1つです。このタイプのプールでは、プールの処理能力に貢献しているマイナーは、プールがブロックを見つけることに成功する時点まで共有を受け取ります。その後、鉱夫は保有する株式数に比例した報酬を受け取ります。

1株あたりの支払いプールは、各鉱夫が貢献のために株を受け取るという点で、いくぶん同じように機能します。ただし、これらのプールは、ブロックがいつ検出されたかに関係なく、即時の支払いを提供します。このタイプのプールに貢献している鉱夫は、いつでも比例支払いと株式を交換することができます。

一方、ピアツーピアマイニングプールは、プール構造が一元化されるのを防ぐことを目的としています。そのため、プール自体に関連する個別のブロックチェーンを統合し、単一の中心的な問題が原因でプール自体が失敗するのと同様に、プールのオペレーターが不正行為を行うのを防ぐように設計されています。

##マイニングプールのメリット

個々のマイニングで成功すると報酬の完全な所有権が付与されますが、電力とリソースの要件が高いため、成功する可能性は非常に低くなります。多くの場合、鉱業は個人にとって有益な事業ではありません。これらのデジタル通貨の人気が高まり、競争力のある鉱夫になるために必要な高価なハードウェアと電気に関連するコストが潜在的な報酬を上回ることが多いため、近年、多くの暗号通貨の採掘がますます困難になっています。

マイニングプールは、ハードウェアと電気のコストの点で個々の参加者の必要性が少なく、収益性の可能性を高めます。個々の鉱夫がブロックを見つけて鉱業報酬を受け取る可能性はほとんどないかもしれませんが、他の鉱夫とチームを組むと成功率が劇的に向上します。

##マイニングプールのデメリット

マイニングプールに参加することで、個人はマイニングプロセスの自律性の一部を放棄します。これらは通常、プール自体によって設定された条件に拘束されます。これにより、マイニングプロセスへのアプローチ方法が決まる場合があります。また、潜在的な報酬を分割する必要があります。つまり、プールに参加している個人の利益の割合は低くなります。

blockchain.comによると、AntPool、Poolin、F2Poolなどの少数のマイニングプールがビットコインマイニングプロセスを支配しています。多くのプールは[分散化に](/decentralizedmarket)努めていますが、これらのグループは[ビットコイン](/bitcoin)プロトコルを管理する権限の多くを統合しています。一部の暗号通貨の支持者にとって、少数の強力なマイニングプールの存在は、ビットコインや他の暗号通貨に固有の分散型構造に反します。

##ハイライト

-マイニングプールが成功して報酬を受け取った場合、その報酬はプールの参加者間で分配されます。

-暗号通貨マイニングプールは、計算リソースを共有するマイナーのグループです。

-マイニングプールは、これらの組み合わされたリソースを利用して、ブロックを見つけるか、暗号通貨のマイニングに成功する可能性を高めます。
