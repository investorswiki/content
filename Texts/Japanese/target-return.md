# ターゲットリターン
##ターゲットリターンとは何ですか？

目標収益は、投資家がその会社に投資した資本から何を作りたいかに基づいてビジネスに価格を設定する価格設定モデルです。

##ターゲットリターンを理解する

目標収益は、ベンチャーに投資された金額に、投資家が見返りとして見たい利益を加えたものとして計算され、金額の[時間価値（TVM）で調整され](/timevalueofmoney)ます。[投資収益率（ROI）方式](/returnoninvestment)として、目標収益価格設定では、投資家が[現在の価格に到達するために逆方向に作業する必要があり](/currentprice)ます。

この価格設定方法を使用する際の主な問題の1つは、投資家が合理的に達成できる収益と、目標収益に到達できる期間の両方を選択する必要があることです。高いリターンと短い期間を選択するということは、投資家が同じ期間でより低いリターン、またはより長い期間で同じリターンを期待する場合よりも、ベンチャーが短期的にはるかに収益性が高くなければならないことを意味します。

目標収益率は、企業が希望する利益を生み出すために製品販売に設定する必要のある価格を予測するためにも使用できます。このモデルは、会社が目標収益を達成するために予測販売量を達成できることを前提としています。実際の売上が不足した場合、目標を達成するために価格を調整する必要があります。

たとえば、懐中電灯会社は、新しい懐中電灯の開発に投資された1,000万ドルに対して15％の目標収益を設定する場合があります。ユニットあたりの製造コストは12ドルで、会社は指定された時間枠内に少なくとも70,000ユニットを販売することを期待しています。つまり、新しい懐中電灯はそれぞれ33.43ドル以上の価格で、求められている利益を実現する必要があります。

ターゲットリターンモデルは、価格の値上げが他の基準に基づいているコストプラス価格設定戦略とは多少異なります。製品の製造コストが主な要因であり、価格を高く設定することで追加の利益率が生まれます。時間と予想販売量は、この価格モデルには影響しません。代わりに、会社は、会社への投資や製品の開発を考慮せずに、販売する製品からどれだけ稼ぎたいかを決定します。

別のモデルである[価値ベースの価格設定](/valuebasedpricing)は、反対方向から機能します。これは、会社が製品に割り当てる価値から始まり、収益性を達成するために生産コストを調整するために機能します。

##ハイライト

-通常、投資家は予想される目標収益から逆戻りして現在の価格に到達します。

-目標収益率は、製品の製造コストを合計し、マークアップを追加するコストプラス価格設定モデルとは異なります。

-目標収益は、お金の時間的価値を考慮に入れるため、他の価格設定モデルとは異なります。

-目標収益とは、投資家が投資から期待する将来の価値または利益を指します。
