# 固定ドル値カラー
##固定ドルバリューカラーとは何ですか？

固定ドルの価値の首輪は、合併中に企業が取得した戦略が適用される可能性があることを指します。この方法により、買収企業の株価変動から会社を守ることができます。

カラーは、トレーダーがロングプットポジション、ショートコールポジションを保持し、原株のロングシェアであるオプション取引戦略を指します。保護メカニズムには、特定の株式の株式を保有すると同時に、保護プットを購入し、保有に対するコールオプションを販売することが含まれます。プットとコールはどちらも[アウトオブザマネー](/outofthemoney)（OTM）オプションであり、有効期限は同じで、購入した契約数は同じです。

##固定ドルバリューカラーの仕組み

[A](/mergersandacquisitions) （M＆A）取引で役立つ2種類のカラーの1つです。これは、対象会社の資産を保護することを目的としており、買収会社の株価が下落した場合でも、売り手の各株式に対して一貫したドル価値を提供します。固定ドルのバリューカラーの目的は、重大な損失を未然に防ぐ可能性のあるサーキットブレーカーになることです。買収取引の株式構成要素に下限と上限を設定することにより、購入を行う会社は、買収される会社の株式ごとに、その株式の固定ドル価値を提供することを約束します。

固定ドルのバリューカラーは、合併または買収取引の[交換比率を設定します。](/exchangeratio)この比率は、買収した会社の株式を買収した会社の株式と交換するために買収する会社の株式交換レベルを決定します。この交換比率は、戦略が交換の最小レベルと最大レベルであるフロアとキャップを提供するため、カラー内で変動します。

##固定ドル値カラーに関する考慮事項

ハーバードロースクールの2014年の記事では、M＆A戦略のいくつかの側面について説明しました。 「Cレベルスイートのますます強気な見通しや、より高く安定した株式市場の評価、意味のある株式コンポーネントの必要性などの取引固有の要因など、株式取引の新たな魅力には多くの要因が関係しています。税の逆転取引」と述べた。

「固定価値構造に内在する潜在的な落とし穴のいくつかはすぐに明らかになります。追加の保護がない場合、取得銀行の株価が署名から決算までの間に下落した場合、取得銀行は当初の予想よりも多くの株式を発行するという[希薄化の影響を受けるリスクがあります。」](/dilution)

専門家は、そのような希薄化効果により、買収が購入者にとって魅力的でなくなる可能性があると付け加えました。また、「証券取引所の議決権要件に基づく買い手株主の承認を必要とするのに十分な株式が発行されるか、債務、インセンティブエクイティ、またはその他の主要な契約の管理条項の変更をトリガーするのに十分な希薄化を引き起こす可能性があります。

「逆に、価値に基づいて変動する交換比率は、取得者の株価が大幅に上昇した場合にも問題を引き起こす可能性があります。たとえば、発行する株式数を、達成に関連するしきい値を下回るようにするなどです。望ましい税務上の扱い。」

##ハイライト

-固定ドルのバリューカラーは、買収企業の株価の変動をヘッジするために合併中に採用されたオプション戦略です。

-固定ドルのバリューカラーは、買収する会社の株式を買収する会社の株式と交換するために、買収する会社の交換レベルを設定します。

-プットオプションとコールオプションの両方を使用して、戦略は買収取引の株式コンポーネントに下限と上限の両方を設定します。
