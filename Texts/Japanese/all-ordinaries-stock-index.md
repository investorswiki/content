# オールオーディナリー株価指数
## All-Ordinaries株価指数とは何ですか？

All-Ordinaries（またはスラングの場合は「 [Kangaroos 」）は、](/kangaroos)[オーストラリア](/asx)[証券取引所](/asx)（ASX）の普通株で構成される株価指数です。 All-Ordinaries Indexは、オーストラリアの株式で最も引用されている[ベンチマークです。](/benchmark) ASXは、インデックスとそのリターンの計算と配布を担当しています。

##すべての普通の株価指数を理解する

All-Ordinaries Indexは市場加重であり、約500社が含まれています。 All-Ordinaries Indexは1980年1月に開始され、「AllOrds」としても知られています。これはオーストラリアで最も古い株式指数です。 All Ordsインデックスに含まれる企業の時価総額は、ASXに上場されているすべての株式の価値の95％を超えています。オーストラリアのAllOrdinariesの3文字の取引所ティッカーは「XAO」です。

企業がインデックスに含まれるためには、ASXによって確立された要件を満たしている必要があります。 All Ordsには、株主に支払われた配当の価値は含まれていないため、その期間中に株式市場への投資から得られた総収益は反映されていません。価格変動と配当の両方を考慮した指数を累積指数と呼びます。 All Ordinaries Accumulation Indexがありますが、これは金融メディアで定期的に引用されていません。

確立されたとき、AllOrdsの基本インデックスは500でした。これは、インデックスが現在5000ポイントである場合、All Ordsの株式の価値は1980年1月から10倍に増加したことを意味します。2000年4月3日に、All Ordsは時価総額で500の最大の企業をカバーするように再構築されました。これは、 S＆P / ASX 200などの新しいベンチマークインデックスの導入と同時に行われました。これらの新しいインデックスにより、AllOrdsの重要性は大幅に低下しました。

##すべての通常のインデックスの包含要件

All-Ordinariesインデックスに含まれるためには、企業はASXで上場されているすべての国内株式の少なくとも0.2％の市場価値を持ち、ASXでの月間上場株式の少なくとも0.5％の平均売上高を維持している必要があります。 。

これらの基準を満たす企業は多種多様であり、その市場価値は大きく異なります。これは、資本金が大きい企業の株価変動は、小規模な企業よりもAllOrdinariesIndexに大きな影響を与えることを意味します。

All Ordinaries Indexポートフォリオは、含まれる企業が引き続き含まれる基準を満たしていることを確認するために、月末に更新されます。また、上場廃止、追加、資本再建など、ポートフォリオ会社に変更があった場合は、月を通じて更新されます。

##ハイライト

--All-Ordinaries（XAO）は、オーストラリアの株式のベンチマーク株価指数です。

-「カンガルー」としても知られるこのインデックスは、オーストラリア証券取引所（ASX）によって構築および維持されています。
