# グレーディング証明書
##グレーディング証明書とは何ですか？

[商品先物市場で](/commodityfuturescontract)は、格付け証明書は、[先物契約](/futures)の基礎となる商品の品質と信頼性を確立する重要な文書です。

所与の商品が交換グレードであり、先物契約を尊重する目的で配達可能であることを確立するために、等級付け証明書が必要です。それらを入手するには、トレーダーは専門の検査官または試験会社の専門知識に頼らなければなりません。

##グレーディング証明書のしくみ

格付け証明書を発行できるのは、特定の許可された個人または組織のみです。それらを提供するために、これらの検査官は、個人的な観察や写真や化学分析などの正式な材料試験手順を含むさまざまな方法に依存しています。関連する商品によっては、さまざまな種類の専門知識が必要になる場合があります。たとえば、検査官はX線を使用して、金や銀などの貴金属のグレードをテストする場合があります。石油の場合、彼らは原油アッセイなどの化学試験法を使用します。

これらの商品格付け証明書は、先物市場で商品の購入者と販売者の両方に役立つ貴重な目的を果たします。この文書は、製品の価値、品質、および状態に関する公式の客観的な決定を提供します。これは、売却または交換のステータスを確立するだけでなく、ローンを取得したり、損失が発生した場合に請求を提出したりするために必要な貴重な文書としても役立ちます。問題の製品が非常に高品質である場合、その格付け証明書を使用して、より高い価格を正当化することができます。逆に、製品が標準以下と見なされる場合、それらはより低い価格をサポートする可能性があります。

[Chicago Mercantile Exchange （CME）](/cme)などの商品取引所は、取引する商品ごとに定義された品質基準を確立しているため、市場参加者は、期待される品質グレードを知ることができます。商品を配達する当事者は、これらの基準を満たす必要があります。そうしないと、契約の[不履行に見舞われるリスクがあります。](/default2)

##成績証明書には何が含まれていますか？

一般的に言えば、特定の格付け証明書の作成に伴う複雑さのレベルは、問題の商品の市場価値とほぼ一致します。貴金属は、カカオや大豆などの比較的安価な商品と比較して、比較的精巧で正確な試験基準を持っています。

たとえば、ココアの場合、一般的なグレーディング証明書には、豆の原産国、製品の一般的な説明、およびスモーキーかドライかなどの状態が示されます。また、欠陥のある豆の割合や1キログラムあたりのおおよその豆の数などの品質に関する解説も含まれます。

>商品が必要な基準グレード未満で評価された場合、売り手（配達人）はその決定を取引所にアピールすることができます。

##実際の例

[大陸間取引所](/intercontinentalexchange)（ICE）にリストされているコーヒー先物契約を考えることができます。

コーヒー先物契約は、37,500ポンドの配達を指定します：

>>

>「ブラジル、ブルンジ、コロンビア、コスタリカ、ドミニカ共和国、エクアドル、グアテマラ、ホンジュラス、インド、ケニア、メキシコ、パプアニューギニア、ニカラグア、パナマ、ペルー、ルワンダ、サルバドール、タンザニア、ウガンダの成長の洗浄アラビカコーヒー、またはベネズエラ。」

>>

さらに、ICEは以下を指定します。

>>

>「コーヒー'C'は、健全な状態で、カップ内のすべての未洗浄および熟成フレーバーがなく、良好な焙煎品質であり、取引所によって確立された基準に従った豆のサイズと色の1つの成長で構成されます。コロンビアのコーヒーの場合、10を超える完全な欠陥を含むコーヒーの配達は許可されないことを除いて、それぞれの成長の基準を下回る15を超える完全な欠陥を含むコーヒーの配達は許可されないものとします。それぞれの成長の基礎。欠陥は、取引所によって確立された格付けスケジュールに基づいて確立されるものとします。」

>>

配達時に、先物契約の所有者は、コーヒーの等級付けを交換に要求できます。次に、マスターサンプラー（適格で公平な採点者のリストからランダムに選択されます）が、1.2インチの側壁を備えた5x8インチのアルミニウムトレイから配達可能なコーヒーを検査します。バッチからサンプリングされた6カップ未満を検査し、採点者はその清潔さ、豆のサイズ、香り、風味、焙煎性、および欠陥（色落ち、臭い、殻の破れなど）を評価します。その後、マスターサンプラーがグレーディングレポートを作成し、対応する証明書が取引所から発行されます。

>商品の採点手数料は、採点証明書を請求された方が負担します。コーヒー契約を格付けする場合、ICEはコーヒー1袋あたり1.25ドルを請求し、最低料金は40ドルです。

>>

##ハイライト

-そうしないと、先物契約の売り手が契約を履行せず、取引所から罰せられる可能性があります。

-先物市場は、基準グレードと呼ばれる標準化された契約仕様を使用しており、これを満たす必要があります。

-配達される商品の品質を確立するために使用され、商品を検査する専門のサンプラーによって判断されます。

-市場参加者は、商品先物取引所が定めた品質基準を順守する必要があります。

-格付け証明書は、商品先物市場の参加者が使用する文書です。

＃＃ よくある質問

###ベーシスグレードとは何ですか？

[ベーシスグレード、またはパーグレード](/basisgrade)は、先物契約に基づく商品の最低基準です。

###先物商品の品質はどのように決定されますか？

商品の標準品質は、先物契約によってその基本グレードとして定義されています。納品された商品の実際の品質は、公平な専門家によって判断され、取引所と契約自体によって設定された事前に確立された基準に従って評価プロセスを介して確立されます。検査の結果、採点証明書が発行されます。

###先物と商品の違いは何ですか？

先物契約は、将来の日付でいくつかの基礎となる資産を提供する契約ですが、今日確立された価格で提供されます。その基礎となる資産は、株式インデックス、債券、通貨から商品まで、何でもかまいません。商品は、配達する必要のある物理的な商品であり、商品先物契約の価格に基づいています。例としては、石油、金、小麦、家畜などがあります。
