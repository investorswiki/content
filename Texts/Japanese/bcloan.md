# B/Cローン
## B / Cローンとは何ですか？

AB / Cローンは、[信用度](/loan)の低い借り手および信用履歴が最小限の借り手に対するローンです。個人消費者ローンや住宅ローンを含むこのタイプの資金調達は、通常、高金利と手数料を請求する代替の貸し手によって発行されます。彼らは、サブプライムまたは[シンファイル](/thin-file)の借り手にローン適格性の第2層を提供します。これは、従来の基準に従い、従来の金融機関によって発行された、Aラベルのローンの対象とならない種類の申請者です。

## B/Cローンを理解する

B / Cラベルの付いたローンのカテゴリの借り手は、多くの場合、支払い記録が不十分（支払いの失敗または遅延が多い）、[信用履歴](/credit-history)（[失敗](/bankruptcy)）があるか、過剰な金額の債務を抱えている可能性があります。ただし、業界がシンファイルの借り手と呼んでいるものでもあります。クレジットスコアを生成するための信用履歴がないか限られている消費者です。若い人や自分の名前でクレジットカードを使うのが初めての人は、このカテゴリに分類されることがよくあります。

あまり有利ではない、略奪的な条件にもかかわらず、B / Cラベルの付いたローンは、借り手が融資を受けるための良い方法であると同時に、[クレジットスコア](/credit_score)と信用履歴を改善することができます（忠実な返済を行うと仮定します）。これはすべて、将来、より有利な資金調達条件を受け取るのに役立ちます。

## B/Cローンの特徴

B / Cローンは、一般的に[サブプライムローンに分類でき](/subprimeloan)ます。借り手のクレジットスコアは一般に650以下であるため、貸し手にとって[デフォルトのリスク](/defaultrisk)が高くなります。これは、3つの信用格付け機関によって開発されたスコアリングシステムであるVantageScoreによって指定された、公正、貧弱、または非常に貧弱なカテゴリに分類されます。 、Equifax、TransUnion、およびExperian。 Experianのデータによると、借り手の69.10％がこれらのカテゴリに該当します。

これらのタイプの借り手にサービスを提供するために、ますます多くの代替信用機関および代替貸し手が信用市場で発展している。シンファイルの借り手に焦点を当てている貸し手と信用調査機関は、携帯電話の請求書、公共料金の請求書、家賃の支払い、さらには公共記録などの代替タイプの支払いデータを分析しようとします。

B / Cローンに関連する追加の信用リスクのため、貸し手は通常、Aラベルのプライムローンに義務付けられているものよりも高い手数料と金利を必要とします。個人向けB/Cローンの年間金利レベルは、通常25％から75％の範囲です。

しかし、B/Cローンは町で最も厳しいものではありません。実際、それらの金利は一般的にDラベルのローンよりも有利です。このカテゴリは、最大400％の年利を請求する貸し手からの[ペイデイローン](/payday-loans)をカバーできます。 B / Cローンは通常、Aラベルのローンよりも比較的高いが、Dラベルのローンよりも大幅に低い金利になります。

##特別な考慮事項

の[ドッドフランクウォールストリート](/dodd-frank-financial-regulatory-reform-bill)[改革および消費者保護法](/dodd-frank-financial-regulatory-reform-bill)は、すべての貸し手に新しい貸付要件を制定しました。基本的に、これらの業界全体のローン引受基準は厳しくなり、より質の高いローンに対してより大きなインセンティブが提供されました。同法はまた、適格住宅ローンを作成しました。これは、流通市場で特別な保護とより有利な条件を受け取ることができる特定の要件を満たす住宅ローンです。

その結果、Aラベルのローンはクレジット市場の大部分を占めています。この法律の規制は、消費者が資金を調達することを困難にする可能性がありますが、この法律はまた、略奪的貸付に対する保護を強化し、特定の場合には[前払いペナルティ](/prepaymentpenalty)を禁止し、一般に、ローンおよび住宅ローン契約においてより明確で透明性の高い条件を義務付けています。

B / Cローンでは、より完全な開示が特に重要になる可能性があります。多くの場合、借り手はこれらの代替ローンの1つから始めて、後でAラベルのローンの資格を得ようとしますが、借り換えを困難または経済的に不利にする特定の条件（前払いペナルティなど）を発見するだけです。

##ハイライト

-B / Cローンの金利と手数料は、特に標準的なローンと比較した場合、信用度の低い借り手への貸付のリスクを説明するために、通常は高くなります。

-2010年のドッド・フランクウォールストリート改革および消費者保護法は、略奪的貸付をより困難にする規制を導入しました。

-B / Cローンは、Aラベルのローンよりも有利ではありませんが、Dラベルのローンよりは優れています。

-標準的な市場の貸し手とは対照的に、代替の貸し手は、信用度の低い借り手にローンを提供します。

-AB / Cローンは、信用度の低い借り手または信用履歴がほとんどない借り手に提供されるローンです。
