# 完全な先物ポジション
##アウトライト先物ポジションとは何ですか？

完全な先物ポジションは、それ自体で行われるヘッジされていない先物取引であり、より大規模またはより複雑な取引の一部ではありません。

##完全な先物ポジションを理解する

[市場リスク](/marketrisk)から[ヘッジされ](/hedge)ていないロングまたはショートの取引です。潜在的な利益と潜在的なリスクの両方は、何らかの方法でカバーまたはヘッジされているポジションよりも、完全なポジションの方が大きくなります。完全なポジションとは、それ自体が独立したポジションであり、より大規模またはより複雑な取引の一部ではありません。購入後に価格が上昇した場合は[ロング](/long)アウトライト先物ポジションから、ショートが開始された後に価格が下落した場合は[ショート](/short)トレードから利益が得られます。

[先物契約](/futurescontract)の方向性に対する純粋なロングまたはショートの賭けです。そのポジションを別のポジションでヘッジまたはオフセットすることは、それがもはや完全なポジションではないことを意味します。

完全な先物は、投資家を市場の変動にさらすため、裸の先物とも呼ばれます。リスクを軽減するために、投資家は、保護[オフセット](/offset)オプション、先物の原証券のオフセットポジション、または関連市場の反対のポジションを購入することを選択できます。リスクをヘッジまたは相殺することは、それがもはや完全な先物ポジションではないことを意味します。

不利な動きに対する保護がないため、完全な先物ポジションは本質的にリスクがあります。ほとんどのトレーダーは、流動性のある先物契約の取引が過度にリスクがあるとは考えていません。特に、ほとんどの場合、ポジションを[カバー](/cover)したり、市場に売り戻したりするのは簡単だからです。先物市場でのほとんどの投機的ポジションは、完全なポジションです。

ただし、先物契約でロングポジションを保持している投資家の市場の衰退は、依然として重大な損失をもたらす可能性があります。この場合、長期先物ポジションに対して[プットオプションを保有すること](/putoption)で、損失を管理可能な金額に制限することができます。トレーダーの潜在的な利益は、オプションの[プレミアム](/premium)またはコストによって減少します。トレーダーが使用したくない保険証券であると考えてください。

[先物市場の上昇の可能性](/futuresmarket)は理論的に無制限であるため、ヘッジなしで先物を売るトレーダーはさらに大きなリスクに直面します。この場合、基礎となる先物契約のコールを所有することで、そのリスクを制限できます。

時にはリスクが少ない完全な先物ポジションの別の代替手段は、[垂直スプレッド](/verticalspread)取引でポジションを取ることです。これは潜在的な利益と損失のリスクの両方を制限し、原証券または[商品](/commodity)の限られた動きのみを期待するトレーダーにとっては良い選択です。

上記のすべてがリスクを制限しますが、利益を制限または減少させる傾向もあります。多くの投機的トレーダーは、その単純さ（1つのポジション）と、取引がタイミングよく行われ、その後の価格変動を利用するときに、より大きな利益を生み出す能力があるため、完全な先物ポジションを好みます。

##アウトライト先物ポジションの例

[S＆P500](/sp500)が今後数か月で上昇すると信じていると仮定します。現在は7月なので、トレーダーはシカゴマーカンタイル取引所（CME）にESのシンボルでリストされているE-Mini S＆P500の12月の契約を購入します。

トレーダーは[指値注文](/limitorder)を使用して$4321のポジションを入力し、1つの契約を購入します。この特定の先物契約は、各ポイントに4つの増分（[ティック](/tick)と呼ばれる）で0.25増分で移動します。各ティックは利益または損失で$12.50の価値があり、各ポイントは$ 50（4 x $ 12.50）の価値があります。

このポジションは、先物契約の価格上昇に対する純粋な方向性のある賭けです。トレーダーは、リスクを相殺したり、ポジションに関連する利益を拡大したりするために、他のポジションをとっていません。したがって、これは完全な先物ポジションです。

1か月の契約価格が4351ドルであると想定します。トレーダーは30ポイントまたは$1500（$ 50 x 30ポイント）、[手数料が少なくなり](/commission)ます。その後、価格は$4311に下がります。トレーダーは10ポイント、つまり$ 500（$ 50 x 10ポイント）下がっています。

##ハイライト

-ヘッジまたはオフセットポジションがアウトライトポジションに追加された場合、それはアウトライトポジションではなく、ヘッジまたは部分的にヘッジされたポジションになります。

-完全なポジションは理論的にはより大きな利益の可能性を持っていますが、完全なポジションはヘッジされたポジションよりも大きなリスクにトレーダーをさらします。

-完全な先物ポジションは、先物契約に対する一方向の賭けポジションであり、より大規模またはより複雑な戦略の一部ではありません。
