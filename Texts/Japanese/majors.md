# メジャーペア
##メジャーペアとは何ですか？

、[外国為替（FX）市場で](/forex-market)最も取引量の多い4つの[通貨ペア](/foreign-currency-pairs)です。現在の4つの主要なペアは、 [EUR / USD](/eur-usd-euro-us-dollar-currency-pair) 、 [USD / JPY](/usd-jpy-us-dollar-japanese-yen-currency-pair) 、 [GBP / USD](/gbp-usd-british-pound-us-dollar-currency-pair) 、 [USD/CHF](/usd-chf-us-dollar-swiss-franc-currency-pair)です。

これらの4つの主要な通貨ペアは成果物の通貨であり、 [Group of Ten](/groupoften) （G10）通貨グループの一部です。これらの通貨は、経済取引に関連するかなりの量に貢献します[が](/volume)、投機目的で最も取引量の多いペアの一部でもあります。

##メジャーペアを理解する

主要なペアは、世界の外国為替市場を牽引すると多くの人に考えられており、最も取引量が多いです。メジャーペアは4ペアのみで構成されていると広く見なされていますが、 [USD / CAD](/usd-cad-us-dollar-canadian-dollar-currency-pair) 、 [AUD / USD](/aud-usd-australian-dollar-us-dollar-currency-pair) 、および[NZD/USD](/nzd-usd-new-zealand-dollar-us-dollar-currency-pair)ペアもメジャーと見なされるべきであると考える人もいます。これらの3つのペアは、「[商品ペア](/commodity-pairs)」と呼ばれるグループに含まれています。

主要なペアを構成する5つの通貨（米ドル、ユーロ、日本円、英国ポンド、スイスフラン）はすべて、2021年の時点で最も取引されている通貨の上位7つに含まれています。

-EUR / USDは、世界で最も取引量の多い通貨ペアであり、すべての外国為替取引の20％以上を占めています。

-USD / JPYは2位で、GBP / USD、USD / CHFが続き、世界の外国為替市場でわずかなシェアを占めています。

>外国為替市場での取引の半分以上は米ドルに関係しています。

>>

[商品ベースの経済の](/commodity)ため、USD / CAD、AUD / USD、およびNZD / USDの取引量は、USD / CHFの取引量を超えることが多く、場合によってはGBP/USDを上回ります。

##トレーダーが主要なペアを取引する理由

ボリュームはより多くのボリュームを引き付ける傾向があります。これは、出来高が多いほど、買値と売値の間の[スプレッドが狭くなる傾向があるためです。](/spread)メジャーペアにはボリュームがたくさんあります。したがって、それらは[エキゾチックなペアよりもスプレッドが小さく](/exotic_currency)、ほとんどのトレーダーを引き付ける傾向があり、ボリュームを高く保ちます。

また、ボリュームが大きいということは、トレーダーが大きなポジションサイズで簡単に市場に出入りできることを意味します。ボリュームの少ないペアでは、価格を大幅に変動させることなく、大きなポジションを販売または購入することがより困難になる場合があります。

ボリュームが大きいということは、特定の時間に購入または販売する意欲のある人が増えることを意味します。その結果、スリップの可能性が[低く](/slippage)なり、スリップが発生した場合のスリップが小さくなります。それは、大きな滑りが主要なペアで起こり得ないということではありません。薄く取引されたエキゾチックなペアよりもはるかに少ないですが、それは可能です。

##主要なペアの価格はどのように決定されますか？

[な](/floatingexchangerate)ペアの通貨はすべて[浮動](/floatingexchangerate)であり、価格は[供給と需要によって決定され](/law-of-supply-demand)ます。中央銀行は価格を管理するために介入する場合がありますが、通常は、経済的損害を引き起こす可能性があるほど価格が上下するのを防ぐ必要がある場合に限ります。

需給は、各国の経済的または基本的な状況、[金利](/interestrate)、国/通貨に対する将来の期待、および現在のポジション（ある時点で撤退する必要のあるポジション）の影響を受けます。

##メジャーペアの見積もりと変動の例

通貨価格は絶えず変化しています—特に毎秒注文を行う参加者が非常に多いためメジャー—現在のレートは[通貨相場で示されます](/currencypair)。

EUR / USDの価格は1.15である可能性があります。これは、1ユーロを購入するのに1.15ドルかかることを意味します。レートが1.20まで上昇した場合、それは、1ユーロを購入するためにより多くのドル（1.20ドル）がかかるため、ユーロの価値が上昇したことを意味します。レートが1.10に下がると、ユーロを購入するのにかかる米ドルが少なくなるため、米ドルの価値が上がるか、ユーロの価値が下がります。

##ハイライト

-4つの主要な通貨ペアは、いわゆる商品通貨ペア（USD / CAD、AUD / USD、およびNZD / USD）とともに、世界で最も活発に取引されているペアの一部です。

-EUR / USDは、世界で群を抜いて最も取引量の多い通貨ペアであり、1日の量が多いため、投機家の間で人気があります。

-外国為替市場の主要な通貨ペアは、EUR / USD、USD / JPY、GBP / USD、およびUSD/CHFです。
