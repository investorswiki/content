# Web 2.0
## Web 2.0とは何ですか？

Web 2.0は、インターネットの現在の状態を説明します。インターネットは、以前のバージョンであるWeb 1.0と比較して、より多くのユーザー生成コンテンツとエンドユーザーの使いやすさを備えています。一般に、Web 2.0は、[ドットコムバブル](/dotcom-bubble)の余波でデジタル時代を変革した21世紀のインターネットアプリケーションを指します。

##Web2.0を理解する

Web 2.0は、インターネットに対する特定の技術的アップグレードについては言及していません。それは単に21世紀におけるインターネットの使用方法の変化を指します。新しい時代では、参加者間の情報共有と相互接続のレベルが高くなっています。この新しいバージョンでは、ユーザーは情報を取り込む受動的な視聴者として行動するだけでなく、積極的に体験に参加することができます。

##Web1.0とWeb2.0

Web 1.0は、インターネットの最初の段階を説明するために使用されます。この時点では、コンテンツクリエーターはほとんどいませんでした。インターネットを利用している人のほとんどは消費者でした。静的ページは、特定のコーディングまたは言語を使用したインタラクティブなアニメーションWebサイトを組み込んだダイナミック[HTML](/html)よりも一般的でした。この段階のコンテンツは、データベース管理システムではなく、サーバーのファイルシステムからのものでした。ユーザーはオンラインゲストブックに署名することができ、HTMLフォームは電子メールで送信されました。

Web 1.0として分類されるインターネットサイトの例は、Britannica Online、個人のWebサイト、およびmp3.comです。一般に、これらのWebサイトは静的であり、機能と柔軟性が制限されています。

Web 2.0という用語は、インターネットがユーザーを積極的に関与させるシステムに軸足を移したときに、1999年に最初に使用されました。ユーザーは、コンテンツを表示するだけでなく、コンテンツを提供するように促されました。記事やコメントを公開できるようになり、さまざまなサイトでユーザーアカウントを作成できるようになり、参加者が増えました。 Web 2.0は、Webアプリ、 [WordPress](/wordpress-cms)のような自費出版プラットフォーム、および[ソーシャルメディア](/social-media)サイトも生み出しました。

Web 2.0サイトの例には、ウィキペディア、フェイスブック、ツイッター、およびさまざまなブログが含まれます。これらはすべて、同じ情報の共有と配信の方法を変革しました。

インターネットの社会的側面は特に変化しています。一般に、ソーシャルメディアを使用すると、ユーザーは考え、視点、意見を共有することで、相互に関与し、対話することができます。ユーザーは、タグ付け、共有、ツイートなどを行うことができます。

##Web2.0の長所と短所

テクノロジーの発達により、ユーザーは自分の考えや意見を他の人と共有できるようになり、他の人を組織して接続する新しい方法が生まれ、より高度なコラボレーションが促進されました。

しかし、インターネットがオープンフォーラムのように機能することには多くの欠点があります。ソーシャルメディアの拡大を通じて、オンラインストーカー、ネットいじめ、 [doxing](/personally-identifiable-information-pii) 、[個人情報の盗難](/identitytheft)、およびその他のオンライン犯罪が増加しています。また、オープンソースの情報共有サイトやソーシャルメディアを介して、ユーザー間で誤った情報が拡散する恐れもあります。

Web 1.0と同様に、Web2.0はインターネットの進化におけるもう1つの移行段階です。 Web 3.0は、すべてのユーザーのニーズにより直感的になるように調整されるため、セマンティックWebと呼ばれると予測されています。

##ハイライト

-インターネットの新時代では、参加者間の情報共有と相互接続のレベルが高くなっています。

-Web 2.0は、インターネットに対する特定の技術的アップグレードについて言及していません。これは、インターネットの使用方法の変化を指します。

-Web 2.0は、インターネットの現在の状態を説明します。これは、以前のバージョンであるWeb 1.0と比較して、より多くのユーザー生成コンテンツとエンドユーザーの使いやすさを備えています。
