# 走る
##ランとは何ですか？

[テクニカル分析では、](/technicalanalysis)ランは、特定の[証券](/security)、[セクター](/sector)、または[インデックス](/marketindex)に対して同じ方向に発生する一連の連続した価格変動です。ランは、長期の上昇トレンドまたは下降トレンドで構成され、[毎日](/uptrend)の[利益](/downtrend)[または](/downtrend)損失が繰り返されることを特徴としています。たとえば、5回の取引セッションで株価が毎日上昇した場合、それはブルランであると言われ、ラリーとも呼ばれます。ベアランは、連続したダウン日で構成されます。

ベアランの直後にブルランが続くなど、特定のシーケンスで表示されるランは、トレードの開始または終了の技術レベルを識別するためのシグナルとして、チャート戦略でよく使用されます。ランを見るとき、トレーダーは、ランの強さの指標として、動きの背後にある基礎となるボリュームを考慮する必要があります。トレーダーは、他の[テクニカル指標](/technicalindicator)や[チャートパターン](/pattern)など、動きを取り巻く他の要因も考慮したいと思うかもしれません。

##ランを理解する

実行とは、特定の証券における一連の連続した価格の上昇または下降です。多くの場合、トレーダーはランを強気のラリーまたは弱気のラリーと呼びます。ランを分類する一定の日数はありませんが、従来、ほとんどのトレーダーは、3回以上の連続した値上げがランを増やすと考えています。

##実行の例

以下のチャートは、SPDR S＆P 500 ETF（SPY）に登場したブルランの顕著な例を示しています。 SPY株は、2017年1月中旬から2月下旬にかけて上昇しました。3人の白人兵士のパターン（下記参照）の後、その後の3週間で指数は上昇を続けました。この特定の実行は、6日間の連続した移動で構成されていました。インデックスが統合し始めたときに実行は最終的に終了し、その後の数週間でさらに混合された方法でさらに上昇しました。

##3人の白い兵士が走る

<！--013AA9977571A472569EF9D3F5067639-->

[3つ](/three_white_soldiers)の[白い兵士のローソク足パターンは、](/three_white_soldiers)前日のキャンドルの本体内で各セッションのオープンが発生し、前日よりも高く閉じた3つの連続したロングボディの[ローソク足](/candlestick)で構成されるタイプのランです。一般に、3人の白人兵士のパターンは、下降トレンドの後に発生する場合は強気の反転ですが、上昇トレンド中または統合期間の後に発生する場合は[継続パターン](/continuationpattern)としても機能する可能性があります。

他のいくつかのローソク足パターンも、実行を分析するときに役立つ場合があります。たとえば、[強気の巻き込み](/bullishengulfingpattern)は実行の開始を表し、同事の[星](/doji)は実行の終了を示す場合があります。トレーダーは、ランを分析するときにこれらのローソク足パターンを念頭に置く必要があります。
