# ファイルと一時停止
##ファイルと一時停止とは何ですか？

ファイルと一時停止は、完全な定年の夫婦が配偶者の利益を受け取り、同時に退職金を遅らせることを可能にする[社会保障](/socialsecurity)[請求戦略でした。](/socialsecurity) 2016年5月1日、11月に署名された2015年超党派予算法により終了しました。 2015年2月、オバマ大統領によるものであり、もはや実行可能な戦略ではありません。

##ファイルとサスペンドを理解する

申告と一時停止は、高収入の配偶者が完全な退職手当を申請しただけで、受け取り始めなかったとしても、低収入の配偶者が配偶者手当を受け取り始めることを可能にする戦略でした。これは、現在の66歳または67歳（人が生まれた時期によって異なります）の引退を超えて完全に遅らせるという利点を逃さずに、夫婦が配偶者給付規則の恩恵を受けるための方法でした。

、主たる受益者（高収入の配偶者）がすでに配偶者給付を請求している場合に限り、配偶者は配偶者給付を請求できますが、その後、将来の日付までそれらの給付の受け取りを遅らせます。これが起こったとき、受益者が技術的にまだ引退していないという事実にもかかわらず、彼または彼女はすぐに配偶者給付を申請し、受け取り始めることができました。その結果、主たる受益者の退職給付は、将来に押し進められるほど長くなり続けるでしょう。

##なぜファイルして一時停止するのですか？

カップルがファイルを使用して戦略を一時停止すると、配偶者の福利厚生がすぐに始まりました。配偶者給付は、高収入の配偶者の収入の半分であるため、配偶者が他の方法で受け取る給付よりも価値があることがよくあります。

一方、遅延引退クレジットは年々価値が高まり、最終的に償還されると月々の支払いははるかに大きくなります。退職給付は、延期された年ごとに元の金額の8％ずつ増加します。これは、69歳（現在の66歳の退職年齢から3年後）まで退職給付を延期した場合、毎月の給付額は、66歳で退職した場合よりも24％高くなります（毎年8％の延期）。

70歳を超えて退職給付を増やすことはできません。また、完全退職年齢は段階的であり、人が生まれた年によって異なることに注意してください。現在の世代の定年者の定年は66歳ですが、ほんの数歳年下の定年は67歳で完全定年に達します。

##ハイライト

-ファイルと一時停止は、夫婦が配偶者手当を受け取り、退職金を遅らせることを可能にする社会保障最大化戦略でした。

-低所得の配偶者は、自分の完全な引退を遅らせながら配偶者手当を受け取ることができるという考えでした。

-2015年に可決された新しい法律は、70歳を超えて退職給付を増やすことはできないと述べて、この戦略を大幅に排除しました。
