# 情報交換のためのアメリカのコード（ASCII）
##情報交換のためのアメリカのコード（ASCII）とは何ですか？

一般にASCIIと呼ばれる情報交換のためのアメリカのコードは、1963年にコンピュータ間で情報を送信するための標準となったテキスト用のコンピュータ言語です。

ASCIIシステムは、英数字と記号をコンピューターで読み取り可能なコードに変換します。元のASCIIは英語のみでした。現在、ASCIIコードには2つのタイプがあります。 7ビットエンコーディングシステムを使用する標準コードと、8ビットシステムを使用する拡張コード。

インターネット通信では、ASCIIは、どの言語でも使用でき、ASCIIと互換性のあるUnicode標準に徐々に取って代わられています。ただし、パーソナルコンピュータや電子機器でのテキスト入力には引き続きASCIIが使用されます。

ASK-yと発音されます。

##情報交換のためのアメリカのコード（ASCII）を理解する

ASCIIの最も初期の反復は、ベルデータサービスがテレプリンターにメッセージを配信するために使用する電信コードにあります。

ASCIIは、データ処理用の最初の主要な文字エンコードシステムであり、1963年にコンピューティングの標準として、1969年にインターネット送信の標準として採用されました。これは、ウォールストリートのトレーダーやブローカーが使用する注文入力コンピューターシステムで長年使用されていました。

インターネットの標準として、Unicode Worldwide Character Standard（Unicode）に徐々に置き換えられています。ただし、Unicodeの最初の128文字はASCIIの文字と同じであるため、2つのシステムは共存していると言えます。

###ASCIIの内容

ASCIIが標準として採用される前は、すべてのコンピューターメーカーが独自のコードを使用し、一部の企業は異なる製品に複数のコードを使用していました。コンピューターは相互に通信できませんでした。

ASCIIは、1963年に米国国家標準によって標準として採用されましたが、リンドンB.ジョンソン大統領が政府のコンピューターおよび通信業界を確保するために連邦政府による採用を義務付けた1968年まで、米国でも完全には実装されていませんでした。コンピュータは互いに通信できます。

ASCIIには元々128のアルファベット文字、数字、記号が含まれており、英語に基づいていました。他の言語に対応するために、1970年代に拡張フォームが追加されました。

間もなく、さまざまなバージョンのASCIIが使用されるようになりました。最終的には255文字が含まれました。このバージョンは拡張ASCIIとして知られています。

現在、ユーザーは、Altキーを押し、NUMロックキーをオンにして数値キーパッドに文字コードを入力することにより、著作権記号などのASCIIまたはUnicode記号をドキュメントに入力できます。

##ハイライト

-ASCIIは引き続き存在しますが、ほとんどの言語をエンコードするために使用できるUnicodeに置き換えられました。

-ASCIIには元々128の英語の文字と記号しか含まれていませんでしたが、後で他の言語で使用される文字を含む追加の文字を含むように拡張されました。

-American Code for Information Interchange（ASCII）は、コンピューター間でテキストを送信するために設計された文字エンコード形式です。

＃＃よくある質問

###ASCIIとは何ですか。 Unicode？

UnicodeはASCIIのユニバーサルバージョンと見なすことができます。ASCIIは結局のところ、情報交換のためのアメリカのコードであり、その最初の反復には、英語のアルファベットとその言語で使用される記号が含まれていました。それ以降のバージョンには、他の多くの言語で使用される記号が含まれていました。一方、Unicodeは、ASCIIで使用されるのと同じ128の記号で始まりますが、現在のバージョンには144,697文字が含まれています。

### ASCIIとは何ですか？

情報交換のためのアメリカ標準コード（ASCII）は、テキストの電子送信用の文字エンコード形式です。すべての文字は一意の番号で表されます。 ASCIIの最初のバージョンには、大文字と小文字のアルファベットの文字を表す128文字と、コンマなどの一般的に使用される記号が含まれていました。後のバージョンでは、ASCIIを255文字に拡張し、英国のポンド記号（£）とスペイン語のテキストで使用される逆さまの疑問符（¿）。

### ASCII文字とは何ですか？

拡張ASCIIテーブルでは、文字コード0〜31は、テキストの先頭やバックスペースなどの制御コードです。文字コード32〜127は、ほとんどがアルファベット（大文字と小文字）の文字と、アスタリスクやドル記号などの記号です。文字コード128から255にはいくつかのバージョンがあり、それぞれが、逆感嘆符、円記号、グルジア語のコンマなど、元の英語のみのASCIIへの追加を反映しています。
