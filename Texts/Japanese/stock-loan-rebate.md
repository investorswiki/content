# ストックローンリベート
##株式ローンのリベートとは何ですか？

を借りる必要のある[ショートセラー](/shortselling)に[現金担保](/cash-collateral)として株式を貸し出す顧客に証券会社が与える現金支払いです。

証券が貸し出されると、ローンに関連する利息とともに、[ローン料金が株式の借り手に請求されます。](/stock-loan-fee)貸与された証券の保有者は、仲介業者からのリベートとしてこの料金の一部を受け取ります。

##株式ローンのリベートを理解する

ショートセラーが株を借りてバイヤーに配達する場合、セラーはリベート料金を支払う必要があります。この料金は、売却の金額と市場での株式の入手可能性によって異なります。借りるのが難しい、または高額な場合、リベート料金は高くなります。

場合によっては、証券会社は、決済日より前に市場で証券を購入するようにショートセラーに強制します。これは、強制[バイインと呼ばれ](/buyin)ます。証券会社は、決済日に株式が利用できないと考えている場合、強制的なバイインを要求する場合があります。

>不足する前に、トレーダーはブローカーにその株式のショートセールリベート料金を確認する必要があります。料金が高すぎる場合は、在庫をショートする価値がない可能性があります。

>>

簡単に言えば、株式ローンのリベートは、マージンでの借入に対して請求される利息の反対側として、ブローカーから潜在的に利用可能な大規模な投資家への支払いです。マージンで株式を購入したことがない投資家にとって、これは外国の概念かもしれません。

マージンで株式を購入したり、売り切れたりするトレーダーは、マージンで株式を購入するときに、ブローカーがそれらの株式の購入に使用した資金の利息を請求することを知っている傾向があります。取引が数日間だけ行われる場合、料金は最小限に抑えられ、実際には気付かれることはありません。これは、通常、低料金のクレジットカードと比較して年額料金になるためです。

貸付ブローカーは、投資家がマージンで株式を購入するために使用したお金のすべての利息を引き続き発生させます。しかし、誰がそれらの利息の支払いを受ける資格がありますか？ほとんどの場合、それはブローカーですが、それらの支払いが他の誰かに送られる可能性がある別のシナリオが存在する可能性があります。

##なぜ株式ローンのリベートが提供されるのですか？

次のシナリオを考えてみましょう。アカウント残高が$100,000の投資家Aは、XYZの株式1000株を購入しますが、1株あたり$ 200で、マージンで購入する必要があり、その場で$100,000のローンに相当します。投資家Aが支払う利息は、年間6％に相当します。

次に、投資家BがたまたまXYZで500株のショートポジションを同時にオープンしたいと考えたとします。したがって、投資家Bが売り切れた500株は、投資家Aが購入した株式の半分です。このシナリオでは、投資家Bがショートポジションを開くために必要な現金担保を提供しているため、最終的には、投資家AがXYZで証拠金ポジションを取るために使用されるのは投資家Bからの現金です。

このシナリオに基づいて、投資家Bが彼らのショートポジションから利息の支払いを提供されるべきであるのは正しいように思われます。このシナリオは、ブローカーがより大規模な顧客の一部に株式ローンのリベートを提供するように駆り立てるものです。実際、彼らはしばしばそうしますが、それは選ばれた顧客のためだけであり、かなりの料金が取られた後ではありません。

##特別な考慮事項

非常に大きなアカウントを持たない小売業者や投資家は、短い取引を開始した場合、リベートが提供されない可能性がありますが、大規模な機関顧客は、かなりのアカウントまたは注文フローを引き付けるためにそのようなリベートを提供される場合があります。リベートの金額は、借り手と貸し手の間で確立された証券貸付契約によって決定され、リベートは通常、貸し手の株式貸付手数料の全部または一部を相殺します。

>株式貸付手数料とリベートの条件と規模は、証券会社が顧客に提供する証券貸付契約に詳しく記載されています。

>>

株式ローンのリベートは、証券貸付の甘味料です。証券貸付はショートセルの重要な特徴であり、投資家は証券を借りてすぐに売却し、後で低価格で買い戻すことで利益を得ることを望んでいます。貸し手は、証券の収益を高める料金によって補償されます。また、トランザクションの終了時にセキュリティが返されます。

通常、このタイプの取り決めは、小規模な[個人投資家には利用できません](/retailinvestor)。株式ローンのリベートは通常、プロのトレーダー、機関投資家、その他のブローカー/ディーラーなど、手元に十分な現金がある大規模なクライアントのみが利用できます。

さらに、担保として現金を使用しない借り手は、株式ローンのリベートを受ける資格がありません。他の種類の資産を担保として提供する借り手は、たとえその担保が財務省の債券や手形などの現金にほぼ匹敵する証券の形であっても、通常は貸し手の料金を負担します。

##株式ローンのリベートの例

ヘッジファンドが1株あたり20ドル相当の100万株の株式を30日間借りるシナリオを考えてみましょう。ローン契約では、このローンの担保は102％であると規定されているため、ヘッジファンドは$20,400,000を提供します。契約ローン手数料は3％で、リベートは.7％、[再投資率](/reinvestmentrate)は1％です。さらに、リベート後の純投資収益は分割され、60％が借り手に、40％が貸し手に支払われます。この例では、年間360日の期間を想定します。

したがって、30日間のローンの株式ローンのリベートは11,900ドルで、次のように計算されます。

>>

> [（2,000万ドルx 102％x 0.70％）] x（30÷360）= $ 11,900

>>

再投資収益は17,000ドルで、次のように計算されます。

>>

> [（2,000万ドルx 102％x 1.00％）] x（30÷360）=17,000ドル

>>

再投資収益からリベートを差し引くと、純投資収益は$5,100になります。次に、これらの収益は60/40に分割されます。つまり、3,060ドルが借り手に支払われ、貸し手は2,040ドルを保持します。

借り手はまた、3％の年間株式ローン手数料を負担します。この場合、30日間の手数料は$50,000です。純投資収益の彼らの部分はこの料金を相殺するので、この期間の借り手の月額料金は次のように計算すると$46,940になります。

>>

> $ 50,000-$ 3.060 = $ 46,940

>>

##ハイライト

-株式ローンのリベートは、ビジネスを引き付けて維持するために、特定の主要な顧客に提供される場合があります。

-株式ローンのリベートは、他の人に株式を貸している証券会社の顧客が受け取る支払いです。

-この支払いは、マージンの借り手によって支払われる利息の支払いとローン料金から行われます。

-一般に、個人のトレーダーや個人投資家は、取引口座に大量の株式を保有する必要があるため、多額のリベートの対象となるのは難しい場合があります。
