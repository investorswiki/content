# 経済的利益（または損失）
##経済的利益（または損失）とは何ですか？

経済的利益または損失は、アウトプットの販売から受け取った収益と使用されたすべてのインプットのコスト、および[機会費用の差](/opportunitycost)です。経済的利益の計算では、機会費用と[明示的費用](/explicitcost)が収益から差し引かれます。

オポチュニティコストは、経営陣によって決定される一種の[暗黙のコスト](/implicitcost)であり、さまざまなシナリオや視点に基づいて異なります。

##経済的利益（または損失）を理解する

[会計上の利益](/accountingprofit)と併せて分析されることがよくあります。会計上の利益は、会社が[損益計算書に表示する利益](/incomestatement)です。会計上の利益は、実際の流入と流出を測定し、企業に必要な財務の透明性の一部です。

一方、経済的利益は企業の財務諸表に記録されておらず、規制当局、投資家、または金融機関に開示する必要もありません。経済的利益は一種の「もしも」分析です。企業や個人は、生産レベルやその他の代替ビジネスを含む選択に直面したときに、経済的利益を検討することを選択できます。経済的利益は、過去の利益を考慮するための代用となる可能性があります。

経済的利益の計算は、エンティティおよびシナリオによって異なる場合があります。一般に、次のようにキャプチャできます。

>>

>経済的利益=収益-明示的なコスト-機会費用

>>

この式では、機会コストを除外すると、会計上の利益だけが得られますが、機会コストも差し引くと、実装できた可能性のある他のオプションと比較するためのプロキシを提供できます。

企業は、損益計算書に明示的なコストを透過的に示します。損益計算書の[一番下の行](/bottomline)の会計上の利益は、直接、間接、および資本コストを差し引いた後の[純利益です。](/netincome)[販売された商品](/cogs)のコストは、単位あたりのコストの分析に使用される最も基本的な明示的なコストです。したがって、上記の式では、企業は機会コストを単位ごとに分類して、単位あたりの経済的利益を得ることができます。

別のオプションを選択することで得られる可能性のある収入との比較を求める場合は、経済的利益を使用できます。自分のビジネスを始める個人は、ビジネスの最初の年の代用として経済的利益を使うかもしれません。大規模な事業体の場合、事業管理者は、事業運営のさまざまな段階での経済的利益と比較して、総利益、営業利益、純利益をより複雑に見る可能性があります。

##特別な考慮事項

###機会費用

機会費用は、特に代替案が利用可能な場合に、ビジネス上の意思決定をより深く分析するために使用できます。サイズが異なる場合があります

機会コストはやや恣意的であり、一種の暗黙のコストとして知られています。それらは、経営陣の見積もりや市場の状況によって異なります。一般に、機会コストは、別の選択を行うことで達成できた可能性のある会計上の利益になります。

##経済的利益の例

個人が事業を開始し、10万ドルの初期費用が発生します。運営の最初の年の間に、ビジネスは$120,000の収入を稼ぎます。これにより、会計上の利益は$20,000になります。しかし、その個人が前の仕事にとどまっていたとしたら、彼女は45,000ドルを稼いだでしょう。この例では、個人の経済的利益は次のようになります。

>>

> $ 120,000-$ 100,000-$ 45,000 =-$ 25,000

>>

この計算では、ビジネスの最初の年のみが考慮されます。初年度以降、コストが10,000に減少した場合、将来の経済的利益の見通しは改善されます。経済的利益がゼロになると、その会社は「[通常の利益](/normal_profit)」の状態にあると言われます。

粗利益と比較して経済利益を使用する場合、企業はさまざまなタイプのシナリオを検討する場合があります。この場合、粗利益が焦点となり、企業はユニットあたりの機会コストを差し引きます。

>>

>経済的利益=ユニットあたりの収益-ユニットあたりのCOGS-ユニットの機会コスト

>>

会社が1ユニットあたり5ドルのコストでTシャツを販売することで1ユニットあたり10ドルを生み出す場合、Tシャツの1ユニットあたりの総[利益](/profit)は5ドルになります。ただし、収益が10ドル、コストが2ドルのショートパンツを作成する可能性がある場合は、機会費用も8ドルになる可能性があります。

>>

> $ 10- $ 5-$ 8 =-$ 3

>>

すべてが同じであれば、Tシャツの代わりにショーツを製造していれば、会社は1ユニットあたり3ドル多く稼ぐことができたはずです。したがって、ユニットあたり-$ 3は、経済的損失と見なされます。

企業は、このタイプの分析を生産レベルの決定に使用できます。利益のより複雑なシナリオ分析では、ビジネスの実行に関連する支出やビジネスサイクルのさまざまなフェーズに応じて、間接コストまたは他のタイプの暗黙のコストも考慮に入れる場合があります。

##ハイライト

-機会コストは、代替案を選択する際に企業が見逃す利益です。

-経済的利益は内部分析に使用され、透明性のある開示には必要ありません。

-経済的利益は、収益から明示的コストと機会コストの両方を差し引いた結果です。
