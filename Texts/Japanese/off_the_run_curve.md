# オフザラン財務省イールドカーブ
##オフザラン財務省のイールドカーブとは何ですか？

オフザラン財務[イールドカーブは](/yieldcurve)、直近の発行前に発行された米国債の[満期と利回りを](/maturity)グラフで表したものです。

##オフザラン財務省のイールドカーブを理解する

オフザラン財務省とは、最近発行されていない、特定の満期の米国[国債を指します。](/government-bond)国債は、政府支出を支援するために発行される債務証券です。米国の連邦国債には、貯蓄債券、[財務省債](/treasurybond)（T債）、[財務省インフレ保護証券](/tips)（TIPS）が含まれます。

オフザランとは、それらが最近発行されたトレジャリーの一部ではないことを意味します。これはオンザラントレジャリーと呼ばれ、対応するイールドカーブはオンザ[ラントレジャリーイールドカーブと呼ばれ](/on_the_run_curve)ます。イールドカーブは、実際に債券の価格設定のベンチマークを決定するため、重要です。

実行中の財務省は、現在の需要の変動によって引き起こされる価格の歪みを起こしやすいです。オフザランの財務省は、[利回り](/yield)が新しい同業者よりも安定しているため、特に有用です。したがって、オフザランのトレジャリーイールドカーブは、オンザランのトレジャリーイールドカーブに固有の歪みを滑らかにする傾向があります。

実行中の財務省は、最も活発に取引されている財務省証券です。それらは、1日の取引量の半分以上を占めると推定されています。ただし、実際には、市場性のある発行済み財務省証券の5％未満しか占めていません。残りの財務省債務は、オフザラン財務省として知られています。実行中の証券と実行中の証券の金額の違いも、2つの証券間の価格設定に影響を与える可能性があります。

##オフザラン財務省のイールドカーブの例

実行中の財務イールドカーブは、債券の価格設定に使用される主要な[ベンチマークです。](/benchmark)ただし、投資家やトレーダーが実行する債券分析では、オフザランの財務省イールドカーブの基礎が使用されます。これらの投資家は、実行中の財務イールドカーブには、実行中の債券に対する現在の市場の需要によって引き起こされる価格の歪みがあると考えています。

オフザランの財務省のイールドカーブがどのように機能するかを想像するために、米国財務省が債券を発行するときのタイムラインを考えてみてください。 1年の1月に最初に10年債が発行されたとき、それらの債券は「実行中」の財務省と見なされます。このステータスは、それらが最も関連性の高い、または最新のエディションであるためです。しかし、その年の後半に、米国財務省が10年債の新しいバッチをリリースする必要がある場合、新しいバッチは実行中の問題になり、1月のバッチは実行外の財務省になります。次に、イールドカーブは、実行されていない財務省のみを使用して計算されます。

##ハイライト

-オフザランとは、それらが最近発行されたトレジャリーの一部ではないことを意味します。これはオンザラントレジャリーと呼ばれ、対応するイールドカーブはオンザラントレジャリーイールドカーブと呼ばれます。

-オフザラン財務イールドカーブは、直近の発行前に発行された米国債の満期と利回りをグラフで表したものです。
