# GBP / USD（英国ポンド/米ドル）
## GBP / USD（英国ポンド/米ドル）とは何ですか？

GBP / USD（British Pound / US Dollar）は、英国ポンドと米ドル[通貨のペア](/currencypair)またはクロスの略語です。通貨ペアは、1英国ポンド（基本通貨）を購入するために必要な米ドル（見積もり通貨）の数を読者に伝えます。

GBP / USDは3番目に大きな取引ペアであり、外国為替市場全体の約11％を占めています。 GBP / USD通貨ペアの取引は、「ケーブル」の取引とも呼ばれます。

## GBP / USD（英国ポンド/米ドル）を理解する

GBP / USDペアの値は、X米ドルあたり1英国ポンドとして見積もられています。たとえば、ペアが1.50で取引されている場合、1英国ポンドを購入するには1.5米ドルかかることを意味します。

GBP / USDは、世界で最も広く取引されている上位5つのペアの1つです。これは、相互および他の通貨との関係で、英国ポンドおよび/または米ドルの価値に影響を与える影響を受ける要因によるものです。このため、[イングランド銀行（BoE）](/boe)と[連邦準備制度の](/federalreservesystem)[金利差](/interest-rate-differential)は、これらの通貨を相互に比較した場合の価値に影響を与えます。

FRBが[公開市場](/open-market)活動に介入して米ドルを上昇させると、英ポンドと比較して米ドルが上昇するため、GBP/USDのクロスの価値が低下する可能性があります。

##大不況とBrexit

大不況の間、英国ポンドの価値は急激に下落しました。 2007年、GBP / USDペアは2.10ドルを超える史上最高値まで取引され、その後1.40ドルを下回り、投資家が米ドル（いわゆる安全な通貨）に群がったため、その価値の3分の1以上を失いました。大不況に続く5年ほどで、英国のポンドは米ドルに対して約1.6で取引されるように回復しました。

英国が欧州連合からの離脱を決議した2016年6月、GBP/USDはさらに急激に下落しました。 GBP / USDペアは1回の取引セッションで10％減少し、Brexit投票の前の月に20％近く減少しました。 EUを去る投票は、貿易取引の再交渉を余儀なくされるため、英国経済にとって否定的であると見なされ、この不確実性により、投資家は記録的なペースで英国から資金を引き出しました。

> 2008年の後半に、英国のポンドは2.10ドルから1.40ドルを下回り、その価値の3分の1以上を失いました。これは、投資家がドルを市場の変動に対する「安全な避難所」と見なしたためと考えられます。

＃＃ 歴史的なデータ

英国ポンドは歴史的に米ドルよりも強かったが、戦前の約5ドルから現在の1.31ドルまで着実に弱体化している。これは、英国の経済力が相対的に低下し、英国の海外植民地のほとんどが失われたことと、米国経済の力が増したことによるものと思われます。

全体的には下落しているものの、短期的にはポンドも上下に変動しています。 1972年には、一時的に最高値の2.65ドルを取り戻した後、次の10年で最低値の1.05ドルに下がりました。

21世紀において、ポンドは、1.90ドルの高値から、約1.30ドルの現在の値まで、下降傾向を続けています。コロナウイルスのパンデミックを取り巻く経済の不確実性は、ヨーロッパ市場の喪失と相まって、英国経済の見通しを全体的に弱めています。

##ハイライト

-投資家は、他の通貨と一緒に、任意の外国為替ブローカーを通じて、GBP/USDペアを取引することができます。

-英国ポンドは、第二次世界大戦後の数十年で着実に弱体化してきましたが、歴史的に米ドルよりも強力です。

-両国の中央銀行が通貨を引き上げたり価値を下げたりするのは、影響を受けた経済指標と行動によるものです。

-大不況後、投資家がドルに群がったため、ポンドはその価値のほぼ3分の1を失いました。

-GBP / USD通貨ペアは、世界で3番目に広く取引されている通貨ペアです。

＃＃よくある質問

###英国ポンドはなぜそんなに強いのですか？

ポンドは歴史的に米ドルよりも強く、過去20年間でポンドの値は1.20ドルから1.70ドルの範囲でした。通貨の強さは、金利、投資家の好み、および国の経済の相対的な強さと密接に関係しています。もう1つの重要な要素は、流通しているドルとポンドの数です。1英国ポンドは1ドル以上の価値がありますが、流通しているドルはポンドよりもはるかに多いのです。

###どうすればGBPとUSDを交換できますか？

外国為替ブローカーを通じて、GBPとUSD、およびその他の通貨の組み合わせを取引できます。外国為替ブローカーは、外国為替商品に焦点を当てていることを除けば、株式ブローカーと同じです。

### GBPとUSDを計算するにはどうすればよいですか？

英国ポンドを米ドルに換算するには、ポンド数に換算日のGBP/USD為替レートを掛けるだけです。たとえば、2022年3月16日に800英国ポンドを米ドルに換算する場合、£800 x 1.31 $ /£（その日の為替レート）を掛けると$1048USDになります。ドルからポンドに変換するには、乗算するのではなく、単純に為替レートで除算します。

###英ポンドは米ドルより強いですか？

英国ポンド（GBP）は歴史的に米ドルよりも強力でした。つまり、1ポンドは1ドルよりも価値があります。しかし、近年、ポンドは対ドルで下落しています。執筆時点では、英国ポンドは1.31米ドルの価値があります。
