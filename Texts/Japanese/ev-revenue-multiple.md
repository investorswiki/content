# 企業価値対収益（EV / R）複数
##企業価値対収益の倍数（EV / R）とは何ですか？

企業価値対収益の倍数（EV / R）は、企業の企業価値を収益と比較する株式の価値の尺度です。 EV / Rは、投資家が株式の価格が適正かどうかを判断するために使用するいくつかの基本的な指標の1つです。 EV / Rの倍数は、買収の可能性がある場合の企業の[評価を決定するためにもよく使用され](/valuation)ます。これは、[企業価値対売上高](/enterprisevaluesales)の倍数とも呼ばれます。

##企業価値対収益の倍数（EV / R）を理解する

企業価値対収益（EV / R）の倍数は、企業の収益を企業価値と比較するのに役立ちます。低いほど良い、つまり、企業が過小評価されているEV/R多重信号は低くなります。

一般的に評価[の倍数として使用され](/multiple)、EV/Rは取得時によく使用されます。取得者は、EV/Rの倍数を使用して適切な公正価値を決定します。企業価値は、取得者がそれぞれ引き受けて受け取るであろう負債を追加し、現金を引き出すために使用されます。

##企業価値対収益の倍数（EV / R）と企業価値対EBITDA（EV / EBITDA）の違い

企業価値対収益（EV / R）は企業の収益を生み出す能力に注目し、企業価値対EBITDA（EV / EBITDA）は[企業マルチプルとも呼ばれ、企業](/ev-ebitda)の営業能力を生み出す能力に注目します。キャッシュフロー。

EV / EBITDAは営業費用を考慮に入れていますが、EV/Rはトップラインのみを考慮しています。 EV / Rの利点は、初期のAmazon（AMZN）のように、まだ収入や利益を上げていない企業に利用できることです。

##企業価値対収益の倍数（EV / R）の使用に関する制限

企業価値対収益の倍数を使用して、同じ業界の企業を比較し、業界で最高の比率からの比率のベンチマークとして、比率が良いパフォーマンスを表すか悪いパフォーマンスを表すかを知る必要があります。

また、Yahoo！などですぐに利用できる時価総額とは異なります。財務、EV / Rの倍数は、企業価値を計算する必要があります。これには、負債を追加して現金を差し引く必要があり、拡張バージョンを使用する場合は追加の要因が含まれる可能性があります。

##ハイライト

-買収の可能性がある場合に企業の評価を決定するためによく使用されます。

-企業の企業価値を収益と比較する株式の価値の尺度。

-収入や利益を生み出さない企業に使用できます。
