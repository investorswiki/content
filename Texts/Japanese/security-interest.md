# 担保権
##担保権とは何ですか？

は、通常はローンを取得するために差し入れられ[た担保](/collateral)[に対する](/lien)強制力のある法的請求またはリーエンです。借り手は、貸し手に特定の資産の担保権を提供します。これにより、借り手がローンの支払いを停止した場合に、貸し手は資産の全部または一部を差し押さえる権利が与えられます。その後、貸し手は、差し押さえられた担保を売却してローンを返済することができます。

##担保権を理解する

ローンの利息を確保すると、貸し手のリスクが低下し、その結果、貸し手がより低い利息を請求できるようになり、それによって借り手[の資本コストが削減されます。](/costofcapital)担保権が付与される取引は「担保権取引」と呼ばれます。

担保権の付与は、自動車ローン、ビジネスローン、住宅ローンなどのローンの標準であり、まとめて[担保ローンと呼ばれ](/secureddebt)ます。ただし、クレジットカードは[無担保ローンに分類され](/unsecuredloan)ます。クレジットカード会社は、デフォルトのカードで購入した衣服、食料品、または休暇を取り戻すことはありません。[署名ローン](/signature_loan)は無担保ローンのもう一つの例です。これら2種類のローンの主な違いは、担保の有無です。

[ユニフォームコマーシャルコード（](/uniform-commercial-code) [UCC](/uniform-commercial-code) ）は、担保権が法的に有効であるための3つの要件、「添付ファイル」と呼ばれるプロセスを指定しています。

1.担保権には価値があります。

1.借り手が担保を所有します。

1.借り手は[セキュリティ契約に署名しました](/security-agreement)。

さらに、担保はセキュリティ契約に具体的に記載されている必要があります。たとえば、ローン契約に記載されている証券は、「借り手のすべての車両」ではなく、借り手の2013年ホンダアコードを指定している場合があります。

貸し手はまた、他の貸し手が同じ担保に対する権利を持たないようにするために、担保権を「完全」にする必要があります。完全な担保権とは、他の当事者が請求できない資産に対する担保権のことです。利息は、適切な法定当局に登録することで完成します。これにより、法的強制力が高まり、その資産に対するその後の請求には[ジュニアステータスが与えられます](/subordinateddebt)。注意として、再運搬の[証書は](/deed-of-reconveyance)、銀行がもはや不動産に対する担保権を持っていないことを証明しています。

>完全な担保権とは、借り手のみが所有する資産に対する担保権であり、適切な法定当局に登録する必要があります。

##担保権の例

シーラが車を買うために2万ドルを借りて、仕事を失ったためにローンの残高が1万ドルになったときに、支払いをやめたとしましょう。貸し手は自分の車を所有し、オークションで10,000ドルで販売します。これは、シーラのローン残高を満たします。シーラはもう車を持っていませんが、貸し手にお金を借りていません。貸し手はもはやその本に[悪いローン](/baddebt)を持っていません。

貸し手がローンを発行する前に資産の担保権を借り手に与えることを借り手に要求する可能性がある別の状況は、企業が機械設備を購入するためにお金を借りたい場合です。事業は銀行に機械の担保権を付与し、事業がローンの支払いを行うことができない場合、銀行は機械を取り戻し、貸したお金を取り戻すためにそれを売却します。破産により企業がローンの支払いを停止した場合、資産に対する請求を行う際には、担保付きの貸し手が無担保の貸し手よりも優先されます。

##ハイライト

-ローンの担保権は、借り手が提供する担保に対する法的請求であり、貸し手が担保を差し戻し、ローンが悪化した場合にそれを売却することを可能にします。

-担保権は貸し手のリスクを低下させ、ローンに対してより低い利子を請求できるようにします。

-低金利は、借り手の資本コストも削減されることを意味します。
