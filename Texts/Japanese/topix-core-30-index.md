# TopixCore30インデックス
## Topix Core 30インデックスとは何ですか？

Topix Core 30 Indexは、日本の東京証券取引所第1部（ [Topix ）に上場している大手企業30社で構成される](/topix)[市場指数](/marketindex)です。 Topix Core 30は、いくつかの異なるTopixインデックスの1つです。

Topix Core 30 Indexは、流動性が高く、時価総額が最大の30社のパフォーマンスを測定することを目的としています。インデックスは、企業のフリーフロートによって重み付けされています。

## TopixCore30インデックスを理解する

Topix Core 30は、Topixインデックスにリストされている1,500社を超える企業の中で最も流動性が高く資本の多い30銘柄のインデックスです。

頭字語であり、[東証株価指数](/topix)は東京証券取引所第1部のすべての日本企業をリストしています。東証株価指数は、東京証券取引所で広く支持されている2つのインデックスファミリーの1つであり、もう1つは[日経](/nikkei)です。

インデックスの計算方法と使用法に関して、Topixインデックスは、米国で使用されているS＆Pインデックスに類似していると考えることができます。日経指数は、米国のダウ工業株30種平均指数に最も類似しています。

トピックコア30は、時価総額によって選択および加重された株式の上場投資[信託（ETF）です。](/etf) Topix Core 30は、野村アセットマネジメントが運営し、東京証券取引所が運営しています。配当金は毎年7月15日に支払われます。

Topix Core 30は上位30銘柄のみに焦点を当てているため、日本経済の成長を示していますが、市場の詳細は示しておらず、業界内のイベントやトレンド、または日本経済全体。

##重み付けの変更

2005年から2006年にかけて、Topix Core 30での企業の加重方法の変更が段階的に導入されました。以前は、企業は会社の発行済み株式総数によって加重されていました。たとえ会社自体またはそのビジネスパートナーがそれらの株式のほとんどを所有していたとしても、それらが取引されることができなかったとしても、多くの株式を持っていたこの特権的な会社。

市場は、取引に利用できる株式数によって均等化会社に移行しました。これは「フリーフロート」と呼ばれ、投資家やトレーダーのために取引できる企業に特権を与えます。この変更により、Topix Core 30はより民主的で、アクセスしやすく、取引活動に対応できるようになりました。
