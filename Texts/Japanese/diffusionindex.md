# 拡散指数
##拡散指数とは何ですか？

拡散指数は、基準点に対する変化の分散を示すために使用されるメトリックであり、一連のデータは、上昇傾向または下降傾向があるかどうかを示すことができます。

拡散指数は、調査の結果を強調するために一般的に使用されます。応答は値の形式であり、それらの値に応じて、基数が指定されます。多くの指標は、0、50、または100のベースラインを使用します。一連のデータは、結果がベースラインからどれだけ離れているかを示します。ベース番号からそれほど遠くないデータポイントはほとんど変化を示さないでしょうが、ベースから遠く離れたデータポイントは重要な変化を示唆する可能性があります。

[購買管理者の指数](/pmi)や[リッチモンド製造業指数](/richmond-manufacturing-index)などの指標の策定に使用されます。データが価格や単位に基づいている経済指標とは異なり、拡散指数を使用するものは調査に基づいています。また、調査の質問がどのように構成されているかに応じて、PMIが主要な経済指標と見なされるのと同じように、拡散指数を予測のツールとして使用できます。

##拡散指数の計算方法

拡散指数を計算する方法はいくつかあり、式は長い形式でも短い形式でもかまいません。

###連邦準備制度

[連邦準備制度](/federal-reserve)は、拡散指数を計算するための基本的な公式を使用しています。この公式は、リッチモンドとカンザスシティのFRBによる調査を含む、FRBの12の地方銀行の多くの指標のインデックスを計算する際に使用されます。

拡散指数=[100*（I − D）] /（I + N + D）

I=増加を報告している回答者の数

N=変更なしを報告した回答者の数

D=減少を報告している回答者の数

###IHSMarkitフォーミュラ

米国やその他の国で購買マネージャーのインデックスを公開している研究グループであるIHSMarkitには、FRBとは異なる独自の公式があります。

IHS Markitの拡散指数=（％レポート'上'）+ [0.5 *（％レポート'同じ'）] + [0.0 *（％レポート'下'）]

これは、短い形式で次のように表すことができます。

インデックス=ポジティブレスポンスのパーセンテージ+0.5（ニュートラルレスポンスのパーセンテージ）

##拡散指数の例

一例として、経済がすぐに[不況に陥るかどうかに関するサンプル調査の質問で](/recession)は、回答者の70％が高い可能性があると回答し、20％が可能性がないと回答し、10％が可能性は低かった。

結果がFRBの拡散指数の公式に入力されると、次のように計算されます。

[100 *（0.70 − 0.10）] /（0.70 + 0.20 + 0.10）= 60/1 = 60

IHS Markitの式を使用して計算すると、答えは次のようになります。

0.70 +（0.5 * 0.20）+（0.0 * 0.10）= 0.70 + 0.10 + 0.00 = 0.80

結果は、使用する式によって異なります。この例では、回答を2桁で操作できるため、ベースラインに0または100を割り当てるのが理想的です。

##拡散指数の解釈方法

一連のデータは勢いを示し、上昇傾向と下降傾向があるかどうかを明らかにする可能性があります。ベースラインを上回っているデータは拡大を示し、下のデータは縮小を示します。ベースラインの近くまたはベースラインにあるデータは、ほとんどまたはまったく変化がないことを意味します。

##拡散指数の制限は何ですか？

インデックスの拡散は、一方的な見方を表す場合があります。したがって、拡散インデックスを他のデータセットと比較すると、追加の確認が得られたり、拡散インデックスを使用して行われた結論に反論したりする可能性があります。[相関係数](/correlationcoefficient)などの他のメトリックを使用すると、拡散指数が信頼できる予測ツールであるかどうかを示すことができます。

##ハイライト

-拡散指数は、数値または統計のグループ内の一般的な傾向を指します。

-拡散指数は、株価指数だけでは見えない強さや弱さを示す可能性のある発散を見つけるために使用できます。

-拡散指数の低下は、上昇する株式が少ないことを示しています。これは、株価指数に対する買い圧力が弱まっている、および/または売り圧力が高まっていることを示しています。

-株式市場では、拡散指数の上昇は、上昇する株式の数が増加していることを意味します。これは、株価指数が上昇するのにプラスです。
