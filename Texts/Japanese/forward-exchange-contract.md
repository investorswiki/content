# 先物為替契約（FEC）
##先物為替契約（FEC）とは何ですか？

先物為替契約（FEC）は、外国為替市場で頻繁に取引されない通貨を交換するために締結される特殊なタイプの[店頭](/otc)（OTC）外貨（外国為替）取引です。これらには、マイナー通貨だけでなく、[ブロックされた](/blockedcurrency)通貨やその他の方法で変換できない通貨が含まれる場合があります。このようなブロックされた通貨を含むFECは、ノンデリバラブルフォワードまたはNDFとして知られています。

大まかに言えば、[先渡契約](/forwardcontract)は、将来の特定の時間に通貨のペアを交換するための2つの当事者間の契約上の合意です。これらの取引は通常、[スポット契約が決済](/spottrade)された日以降に行われ、通貨価格の変動から買い手を保護するために使用されます。

##先物為替契約（FEC）を理解する

先物為替契約（FEC）は取引所で取引され[て](/exchange)おらず、これらの契約では標準金額の[通貨は取引されていません。](/currency)それでも、両当事者の相互合意がない限り、キャンセルすることはできません。

契約に関与する当事者は、一般的に、外国の取引所のポジションをヘッジする[か](/foreign-exchange)[、](/foreign-exchange)[投機的なポジション](/speculation)を取ることに関心があります。すべてのFECは、通貨ペア、想定元本、決済日、および配信レートを設定し、また、取引を完了するために固定日の実勢スポットレートを使用することを規定しています。

したがって、契約の為替レートは固定され、将来の特定の日付に指定されます。これにより、関係者は将来の金融プロジェクトの予算を改善し、指定された将来の日付での取引からの収入またはコストを事前に正確に知ることができます。 FECの性質は、通貨の将来の[スポットレート](/forex-spot-rate)の予期しないまたは不利な動きから両当事者を保護します。

>ほとんどの[通貨ペアの先物為替レート](/currencypair)は、通常、将来最大12か月間、または4つの「主要ペア」の場合は最大10年間取得できます。

>>

通常、ほとんどの通貨ペアの先物[為替レート](/exchangerate)は、最大12か月先まで取得できます。 「[メジャーペア](/majors)」と呼ばれる通貨のペアは4つあります。これらは米ドルとユーロです。米ドルと日本円。米ドルと英ポンド。米ドルとスイスフラン。これらの4つのペアについて、最大10年間の為替レートを取得できます。

数日という短い契約期間も多くのプロバイダーから入手できます。契約はカスタマイズできますが、最小契約額を$ 30,000に設定しない限り、ほとんどのエンティティはFECのメリットを十分に享受できません。

##特別な考慮事項

最大の先物為替市場は、人民元（ [CNY](/cny-china-yuan-renminbi) ）、インドルピー（ [INR](/inr) ）、韓国ウォン（ [KRW](/krw-korean-won) ）、ニュー台湾ドル（ [TWD](/twd-taiwan-dollar) ）、ブラジルルーブル（ [BRL](/brl-brazilian-real) ）、ロシアルーブル（ [RUB](/rub-russian-ruble) ）です。一方、最大のOTC市場はロンドンで開催され、ニューヨーク、シンガポール、香港でも活発な市場が開かれています。韓国を含む一部の国では、活発な[NDF](/ndf)市場に加えて、陸上の先物市場が制限されていますが制限されています。

FEC取引の最大のセグメントは、米ドル（ [USD](/usd) ）に対して行われます。ユーロ（ [EUR](/euro) ）、日本円（ [JPY](/jpy-japanese-yen) ）、および程度は低いものの英国ポンド（ [GBP](/gbp) ）とスイスフラン（ [CHF ）](/chf)を使用する活発な市場もあります。

##フォワードエクスチェンジの計算と例

契約の先物為替レートは、次の4つの変数を使用して計算できます。

-S=通貨ペアの現在のスポットレート

--r（d）=国内通貨[金利](/interestrate)

--r（f）=外貨金利

-t =契約時間（日数）

先物為替レートの計算式は次のようになります。

>>

>フォワードレート=Sx（1 + r（d）x（t / 360））/（1 + r（f）x（t / 360））

>>

たとえば、米ドル（USD）とカナダドル（ [CAD](/cad-canadian-dollar) ）のスポットレートが1CADで$0.80USDを購入するとします。米国の3か月の率は0.75％であり、カナダの3か月の率は0.25％です。この場合、3か月のUSD /CADFECレートは次のように計算されます。

>>

>3か月先物レート=0.80x（1 + 0.75％*（90/360））/（1 + 0.25％*（90/360））= 0.80 x（1.0019 / 1.0006）= 0.801

>>

90日間の料金による差は、100分の1セントです。

##ハイライト

-FECは、カスタマイズ可能な条件でOTCで取引され、流動性が低い、ブロックされている、または変換できない通貨を参照することがよくあります。

-FECは、FX取引が利用できない場合に、通貨の将来のスポットレートの予期しないまたは不利な動きから両当事者を保護するため、リスクに対するヘッジとして使用されます。

-先物為替契約（FEC）は、通貨取引を行うための2つの当事者間の合意であり、通常、外国為替市場では容易にアクセスできない通貨ペアが関係します。
