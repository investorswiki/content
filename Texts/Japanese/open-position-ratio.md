# オープンポジション比率
##オープンポジション比率とは何ですか？

[取引プラットフォームまたは取引所](/trading-platform)の主要通貨ペアごとに保有されている[オープンポジション](/open-position)のパーセンテージとして、そのプラットフォーム上のすべての主要ペアで保有されているポジションの総数に対して計算されます。

[に対する建玉](/openinterest)のローカル指標であり、さまざまな外国為替プラットフォームおよび取引所間で異なります。

##オープンポジション比率を理解する

オープンポジション比率は、[外国為替](/foreign-exchange)トレーダーが、投資家がどの通貨に注目しているかを把握するために使用されます。これは、1つの主要通貨ペアが他の通貨ペアとどのように比較されるかを示し、毎日数回更新されます。これは、この目的のために取引量活動と組み合わせて使用されます。

[ロングショート比率](/longshort-ratio)が存在する主要通貨ペアの合計ポジションに対するロングポジションまたはショートポジションのパーセンテージを示していません。

オープンポジション比率は特定のリテールトレーディングプラットフォーム上のトレーダーのポジションによってローカルに決定される傾向があるため、大規模な投資銀行が市場を支配しているはるかに広範な外国為替市場で起こっていることのほんの一部を表しています。[スポット取引](/spottrade)は外国取引のごく一部にすぎず、小売取引プラットフォームはそのごく一部にすぎません。オープンポジション比率が何らかの用途を持っている場合、それはどの小売取引が混雑しているのかを示すことであり、これは単に[群集行動を反映している可能性があります](/herdinstinct)。

##オープンポジション比率の例

例として、ユーロ対の通貨ペア。米ドル（EUR / USD）は、架空のFutureForexプラットフォームで25.8のオープンポジション比率を持つ可能性があります。これは単に、EUR / USDがその時点でFutureForexのすべてのオープンポジションの25.8％を占めることを意味します。

利用可能なすべてのオープンポジション比率のパーセンテージは、常に100％近くになります。これは、マイナーな通貨ペアが計算に含まれないことが多く、合計が100％未満になる可能性があるためです。主要通貨ペアには、 [FX](/eur-usd-euro-us-dollar-currency-pair)市場で最も取引量が多いと考えられる4つの外国為替ペアが含まれます。 [USD / JPY](/usd-jpy-us-dollar-japanese-yen-currency-pair) ; [GBP / USD](/gbp-usd-british-pound-us-dollar-currency-pair) ;および[USD/CHF](/usd-chf-us-dollar-swiss-franc-currency-pair) 。

##ハイライト

-この比率は、どの通貨ペアがプラットフォーム上で最もオープンな関心を持っているかについての全体的な印象を与えます。通貨ペアの特定のロングショート比率と混同しないでください。

-オープンポジション比率は、プラットフォームごとに異なるため、使用が制限されており、主に小売スポットの活動を示しているにすぎません。

-オープンポジション比率は、特定の外国為替取引プラットフォームで保持されているオープン通貨ポジションの割合を示します。
