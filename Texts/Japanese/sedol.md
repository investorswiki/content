# 証券取引所デイリーオフィシャルリスト（SEDOL）
##証券取引所デイリーオフィシャルリスト（SEDOL）とは何ですか？

証券取引所の日次公式リスト（SEDOL）は、[ロンドン証券取引所](/lse)および英国のさまざまな小規模取引所で取引される証券に割り当てられた7文字の識別コードです。 SEDOLコードは、[ユニットトラスト](/unittrust)、投資信託、保険リンク証券、および国内および海外の株式に使用されます。 SEDOLコードは、米国で取引されている株式の統一証券識別手順委員会によって発行されたコードである[CUSIP番号に](/cusipnumber)相当します。

##証券取引所の日次公式リスト（SEDOL）を理解する

[本社の変更](/corporate-headquarters)、企業の合併、新しい[ISINの発行](/isin)、買収、会社名の変更、株式の[再分類の発生](/reclassification)など、さまざまな理由で発行される可能性があります。

##SEDOL分類コードの構成

すべてのSEDOLコードは7文字で、2つの部分に分割されています。最初の6文字は英数字コードで、7番目の文字は末尾のチェックディジットです。英数字の部分では、BからZまでの文字が許可され、0から9までの数字は数字として許可されます。

2004年1月より前に発行されたSEDOLコードは、厳密に数字で構成されていました。 2004年1月26日以降、SEDOLコードは、B000009から始まり、数字と文字の両方で順番に発行されています。文字の位置ごとに、数字は常に文字の前に置かれ、母音は使用されません。したがって、2004年1月以降に発行されたSEDOLコードは、文字で始まります。

SEDOLコードのチェックディジットは、7文字すべての加重和を10の倍数にするために選択されます。この桁は、最初の6文字の加重和を使用して計算されます。このプロセスでは、文字に番号が割り当てられます。各文字は、9にアルファベットの位置に対応する番号を加えたものに等しくなります。たとえば、Cは12（9 + 3）になります。

##SEDOL分類コードの重要性

ロンドン証券取引所は、SEDOLを、世界的に認められ、国境を越えた貿易の失敗によって発生するコストを削減し、貿易および証券取引の効率を高める、重要でユニークな市場レベルのセキュリティ識別子として認めています。 SEDOLコードは、そのような失敗を減らし、取引プロセスを合理化することにより、英国の取引所がより高いレベルのサービスを提供するのに役立ちます。

SEDOLコードのいくつかの特徴により、SEDOLコードは重要で重要なものになっています。一意の割り当てられた国レベルの番号により、各国に1つずつ割り当てられた番号を簡単に識別できます。 SEDOLコードも迅速で、発行処理の時間枠が短縮されています。 SEDOLコードを重要なものにする最後の特徴の1つは、共通性です。コードは、上場証券と非上場証券のすべての国に割り当てられ、すべての資産クラスをカバーします。

取引所は取引されている資産を追跡するための安全で識別可能な方法を必要とするため、最終的に、SEDOLコードは今日の世界市場で適切です。ロンドンと英国は、資産の追跡における独自性と効率性、および投資家が正しい株式を購入することを保証する能力について、SEDOLコードに依存しています。

##SEDOL分類コードの例

銀行大手のHSBCは、1991年にロンドン証券取引所（LSE）に上場し、SEDOLコードは0540528です。報告されたSEDOL番号が正しいかどうかを確認するには、トレーダーは数字に割り当てられた重みを掛けて加算する必要があります。結果の数値が10の倍数である場合、コードは正しいです。

HSBCの場合、計算は次のようになります。

0 + 5X3 + 4X1 + 0 + 5X3 + 2X9 + 1X8 = 60、これは10の倍数です。

##ハイライト

-SEDOLコードは、ロンドン証券取引所および英国の他の小規模取引所で取引される証券に割り当てられた一意の7文字の英数字の識別子です。

-SEDOLコードには、最後に一意のチェックサム文字が割り当てられています。 7桁目は、最初の6文字の加重和です。

-SEDOLコードは、貿易コストを削減し、貿易および証券取引の効率を向上させたため、重要なイノベーションです。
