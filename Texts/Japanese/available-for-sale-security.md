# 売却可能有価証券
##売却可能証券とは何ですか？

売却可能有価証券（AFS）は、満期に達する前に売却すること、または満期日がない場合に長期間保有することを目的として購入された負債証券または株式証券です。会計基準では、企業が負債証券または株式証券への投資を[満期保有](/held-to-maturity-security)、[取引目的](/held-for-trading-security)、または売却可能として購入する場合に分類する必要があります。売却可能有価証券は公正価値で報告されています。会計期間間の価値の変化は、[貸借対照表](/balancesheet)の資本セクションの[その他の包括利益累計額に含まれてい](/accumulatedother)ます。

##販売可能なセキュリティの仕組み

売却可能（AFS）は、金融資産を説明および分類するために使用される会計用語です。これは、他の2種類の金融資産である、取引目的の有価証券または満期保有の有価証券として分類されていない負債証券または株式証券です。 AFS証券は非戦略的であり、通常、市場価格をすぐに利用できるようにすることができます。

（トレーディング投資からのものとは異なり）[純利益](/netincome)には反映されませんが、売却されるまで[その他の包括利益](/comprehensiveincome)（OCI）分類に表示されます。純利益は[損益計算書に報告され](/incomestatement)ます。したがって、AFS有価証券の未実現損益は損益計算書に反映されません。

純利益は、複数の会計期間にわたって貸借対照表の[利益剰余金に累積されます。](/retainedearnings)対照的に、AFS有価証券からの未実現損益を含むOCIは、会計期間の終了時に貸借対照表の「その他の包括利益累計額」に計上されます。その他の包括利益累計額は、貸借対照表の[資本セクションの](/shareholdersequity)利益剰余金のすぐ下に報告されます。

>###重要

>売却可能有価証券の未実現損益は、その他の包括利益累計額の貸借対照表に含まれています。

##販売可能vs.ホールドフォートレーディングvs。満期保有目的の有価証券

上記のように、有価証券には、売却可能有価証券、取引保有有価証券、および満期保有有価証券の3つの分類があります。[売買目的有価証券](/held-for-trading-security)は、主に短期的に購入および売却のために保有されます。目的は、長期投資ではなく、迅速な取引から利益を上げることです。スペクトルのもう一方の端には、[満期保有証券があり](/held-to-maturity-security)ます。これらは、会社が満期日まで保有することを計画している債務証書または株式です。例としては、満期日が設定された譲渡性預金（CD）があります。販売可能（AFS）は、真ん中にあるキャッチオールカテゴリです。同社がしばらく保有する予定であるが売却も可能であるのは、負債と資本の両方の有価証券を含む。

会計の観点から、これらのカテゴリーはそれぞれ異なる方法で扱われ、貸借対照表または損益計算書に利益または損失が表示されるかどうかに影響します。 AFS有価証券の会計処理は、トレーディング有価証券の会計処理と同様です。投資は短期的なものであるため、公正価値で計上されます。ただし、有価証券の取引については、公正市場価格に対する未実現損益は営業利益に計上され、損益計算書に表示されます。

売却可能有価証券の価値の変動は、その他の包括利益（OCI）の未実現損益として記録されます。損益計算書の下にOCI情報を含める会社もあれば、包括利益合計に含まれる内容を詳細に示す別のスケジュールを提供する会社もあります。

##売却可能有価証券の記録

会社が現金で売却可能有価証券を$100,000で購入した場合、現金への貸方と売却可能有価証券への借方を$100,000で記録します。有価証券の価値が次の報告期間までに50,000ドルに下落した場合、有価証券の公正市場価値の変化を反映するために投資を「評価減」する必要があります。この価値の減少は、売却可能有価証券への50,000ドルの貸方およびその他の包括利益への借方として記録されます。

同様に、投資額が翌月に増加した場合、その他の包括利益の増加として記録されます。価値の変化がOCIで認識されるために、証券を売却する必要はありません。このため、これらの損益は、有価証券が売却されるまで「未実現」と見なされます。

##ハイライト

-購入した負債証券またはエクイティ証券への投資は、満期保有、取引用保有、または売却可能として分類する必要があります。

-売却可能有価証券は公正価値で報告されています。

-売却可能有価証券（AFS）は、満期前に売却する目的で購入された負債証券または株式です。

-未実現損益は、貸借対照表の資本セクション内のその他の包括利益累計額に含まれています。
