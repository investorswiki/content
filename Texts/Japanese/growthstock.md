# 成長ストック
##成長株とは何ですか？

成長株とは、市場の平均成長率を大幅に上回る速度で成長すると予想される企業のシェアです。これらの株は一般的に配当を支払いません。これは、成長株の発行者は通常、短期的に成長を加速するために、発生した収益を再投資したい企業であるためです。投資家が成長株に投資するとき、彼らは将来彼らが最終的に彼らの株を売るときに彼ら[がキャピタルゲインを通してお金を稼ぐであろうと予想します。](/capitalgain)

##成長株を理解する

成長株は、あらゆるセクターまたは業界に出現する可能性があり、通常、高[収益（P / E）](/price-earningsratio)[比率で取引され](/price-earningsratio)ます。現時点では収益がない可能性がありますが、将来的には収益が見込まれます。

成長株への投資はリスクを伴う可能性があります。彼らは通常[配当](/stockdividend)を提供しないので、投資家が彼らの投資でお金を稼がなければならない唯一の機会は彼らが最終的に彼らの株を売るときです。会社がうまくいかない場合、投資家は売却の時期に株式を失うことになります。

成長株は、いくつかの共通の特徴を共有する傾向があります。たとえば、成長企業は独自の製品ラインを持つ傾向があります。彼らは特許を保有しているか、業界で他の人よりも優位に立つ技術にアクセスできる可能性があります。競合他社に先んじるために、彼らは利益を再投資して、より長期的な成長を確実にする方法として、さらに新しい技術と特許を開発します。

彼らの革新のパターンのために、彼らはしばしば彼らの業界で忠実な顧客基盤またはかなりの量の[市場シェア](/marketshare)を持っています。たとえば、コンピュータアプリケーションを開発し、新しいサービスを最初に提供する会社は、新しいサービスを提供する唯一の会社であるという市場シェアを獲得することにより、成長株になる可能性があります。他のアプリ会社が独自のバージョンのサービスで市場に参入した場合、最大数のユーザーを引き付けて保持することに成功した会社は、成長株になる可能性が高くなります。

多くの[小型](/small-cap)株は成長株と見なされます。ただし、一部の大企業は成長企業でもある可能性があります

>成長株は、あらゆる取引所や産業部門で取引されていますが、通常、最も急速に成長している産業や、ナスダックのようなより革新的な取引所で見つけることができます。

##成長株と価値株

[価値](/valuestock)株とは異なります。投資家は、基礎となる会社の力強い成長の結果として、成長株が大幅なキャピタルゲインを獲得することを期待しています。この予想により、これらの株式は一般的に高い価格対収益（P / E）比のために、過大評価されているように見える可能性があります。

それどころか、バリュー株は市場によって過小評価または無視されることがよくありますが、最終的には価値を得る可能性があります。投資家はまた、彼らが通常支払う配当から利益を得ようとします。バリュー株は、低価格対収益（P / E）比で取引される傾向があります。

一部の投資家は、[多様化のために、成長株とバリュー株の両方をポートフォリオに含めようとする場合があります](/diversification)。他の人は、価値や成長にもっと焦点を当てることによって専門化することを好むかもしれません。

一部のバリュー株は、単に収益報告が不十分であるか、メディアの注目が否定的であるために低価格になっています。しかし、彼らがしばしば持っている特徴の1つは、強力な配当支払い履歴です。強力な配当実績を持つバリューストックは、投資家に信頼できる収入を提供することができます。多くのバリューストックは、特に革新的でなくても、成長する準備ができていなくても、ビジネスを継続することを期待できる古い会社です。

##成長株の例

Amazon Inc.（AMZN）は、長い間成長株と見なされてきました。 2021年には、世界最大の企業の1つであり、しばらく前から存在しています。 2021年9月24日の時点で、Amazonは[時価総額](/marketcapitalization)で米国株の中で4番目にランクされています。

アマゾンの株式は、歴史的に高い株価収益率（P / E）で取引されてきました。 2020年6月から2021年9月までの間に、株式のP / Eは約58から106の範囲でした。会社の規模にもかかわらず、 2022年の[1株当たり利益](/eps)（EPS）の成長予測は67を超えています。

企業が成長すると予想される場合、投資家は（高いP / E比であっても）投資する意欲を持ち続けます。これは、数年後には現在の株価が後から安く見える可能性があるためです。リスクは、成長が期待どおりに継続しないことです。投資家は一つのことを期待して高い代償を払ってきましたが、それは得られませんでした。このような場合、成長株の価格は劇的に下落する可能性があります。

##ハイライト

-成長株は通常、配当を支払いません。

-成長株はしばしば高価に見え、高いP / E比で取引されますが、会社が急速に成長し続け、株価が上昇する場合、そのような評価は実際には安くなる可能性があります。

-成長株は、バリュー株とは対照的に置かれることがよくあります。

-投資家は期待に基づいて成長株に高い代償を払っているので、それらの期待が実現されない場合、成長株は劇的な減少を見ることができます。

-成長株とは、市場平均よりも速いペースで売上と収益を伸ばすことが期待される企業です。

＃＃ よくある質問

###株式が成長か価値かをどうやって知るのですか？

バリューストックは、将来の成長の可能性に目を向けるのではなく、実際の価値を下回って取引されると考えられているため、理論的には、株価がファンダメンタルズに追いつくため、優れたリターンを提供します。通常は配当を支払わない成長株とは異なり、バリュー株は平均[配当利回りよりも高いことがよくあります](/dividendyield)。バリュー株はまた、成長株とは逆に、比較的低い[帳簿価格（P / B）比と低いP / E値](/price-to-bookratio)を備えた、強力なファンダメンタルズを持つ傾向があります。

###成長株は危険ですか？

すべての投資と同様に、リスクとリターンの間には根本的なトレードオフがあります。成長株は将来のリターンの可能性が高いため、バリュー株や社債などの他の種類の投資よりもリスクが高くなります。主なリスクは、実現または期待される成長が将来にわたって継続しないことです。投資家は一つのことを期待して高い代償を払ってきましたが、それは得られませんでした。このような場合、成長株の価格は劇的に下落する可能性があります。

###成長株と見なされるものは何ですか？

株式に関して言えば、「成長」とは、会社に資本増価の余地が十分にあることを意味します。これらは、より新しく、より小規模な企業、および/またはテクノロジーやバイオテクノロジーなどの成長セクターの企業である傾向があります。成長株の収益は低いか、マイナスでさえある可能性があり、多くの場合、株価収益率が高くなります。

###成長株の例は何ですか？

架空の例として、成長株は、有望な新しい癌治療に取り組み始めたバイオテクノロジーの新興企業です。現在、この製品は臨床試験のフェーズIの段階にあり、FDAが薬剤候補をフェーズIIおよびIIIの試験に継続することを承認するかどうかは不明です。薬が合格し、最終的に使用が承認された場合、それは莫大な利益と資本の利益を意味する可能性があります。ただし、薬が計画どおりに機能しないか、深刻な副作用を引き起こす場合は、そのR＆D支出のすべてが無駄になっている可能性があります。
