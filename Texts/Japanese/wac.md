# 加重平均クーポン（WAC）
##加重平均クーポン（WAC）とは何ですか？

加重平均クーポン（WAC）は、住宅ローン担保証券（MBS）として投資家に販売される住宅ローンのプールの収益率の測定値です。基礎となる住宅ローンはさまざまな期間で返済されるため、WACは発行時のリターンを表し、後でWACとは異なる場合があります。

##加重平均クーポン（WAC）を理解する

銀行は、発行する住宅ローンを二次住宅ローン市場で定期的に販売しています。買い手は、ヘッジファンドや投資銀行などの機関投資家です。これらのバイヤーは、住宅ローンを市場性のある証券にパッケージ化し、公開市場で投資家に[住宅ローン担保証券](/mbs)（MBS）として取引することができます。

>加重平均の計算では、各住宅ローンの元本残高がその加重係数として使用されます。

###WACの計算

加重平均クーポン（WAC）は、MBSの基礎となる住宅ローンに支払うべき金利の総額を取得し、各住宅ローンが表す証券のパーセンテージに従ってそれらを重み付けすることによって計算されます。

WACは、さまざまな金利の住宅ローンのさまざまなプールの平均金利を表します。加重平均の計算では、基礎となる各住宅ローンの元本残高が加重係数として使用されます。

WACを計算するには、各住宅ローンまたはMBSのクーポンレートに残りの元本残高を掛けます。結果が合計され、合計が残りの残高で除算されます。

加重平均クーポンを計算する別の方法は、各住宅ローンプールの重みを取得し、それぞれのクーポンレートを掛けて、その結果を加算してWACを取得することです。

たとえば、MBSが、元本残高が1,100万ドルの3つの異なる住宅ローンプールで構成されているとします。最初の住宅ローンバンドル、またはトランシェは、7.5％の利回りの400万ドル相当の住宅ローンで構成されています。 2番目のプールには、5％の利率で500万ドルの住宅ローン残高があります。 3番目のプールには、3.8％の割合で200万ドル相当の住宅ローンがあります。

上で概説した最初の方法を使用する：

WAC = [（400万ドルx 0.075）+（500万ドルx 0.05）+（200万ドルx 0.038）]/11百万ドル

WAC =（$ 300,000 + $ 250,000 + $ 76,000）/ $1,100万

WAC =626,000ドル/11百万ドル=5.69％

あるいは、WACは、最初に各住宅ローントランシェの重量を評価することによって計算されます。

プール1の重み：400万ドル/1100万ドル=36.36％

プール2の重み：500万ドル/1100万ドル=45.45％

プール3の重み：200万ドル/1100万ドル=18.18％

重みの合計は100％です。したがって、WACは次のように計算されます。

WAC =（36.36 x 0.075）+（45.45 x 0.05）+（18.18 x 0.038）

WAC = 2.727 + 2.2725 + 0.6908 = 5.69％

さまざまな住宅ローン保有者がさまざまな金利とさまざまなタイムテーブルで住宅ローンを返済するため、加重平均クーポンレートはMBSの存続期間中に変化する可能性があります。

###MSBが危険にさらされたとき

住宅ローン担保証券についての言及は、2007年から2008年の金融危機への言及なしには完全ではありません。

その期間のMBS投資の多くは、全国的な住宅バブルの間に発行された住宅ローンによって支えられており、多くの場合、返済する余裕のない借り手に発行されました。バブルが崩壊したとき、これらの借り手の多くはデフォルトに追い込まれ、これらの資産の証券化の価値は溶けてしまいました。

実際、彼らはサブプライムローンで担保されていました。

##ハイライト

-証券の基礎となる住宅ローンが返済されると、WACは時間とともに変化します。

-住宅ローン担保証券のWACは、これらの投資のアナリストが前払いの特性を推定するために使用します。

-WACは、発行時の住宅ローン担保証券の基礎となる住宅ローンの平均総金利です。
