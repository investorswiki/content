# 経験豊富なセキュリティ
##ベテランセキュリティとは何ですか？

ベテラン証券は、新規株式公開による短期的な影響を排除するのに十分な期間、流通市場で上場されてきた金融商品です。また、ユーロマーケットで少なくとも40日間発行され、活発に取引されている証券も指します。

##経験豊富なセキュリティを理解する

新しい証券が新規株式公開（IPO）を通じて最初に提供されるとき、それらは上場直後にかなりのボラティリティを示す可能性があります。ベテラン証券はすでにしばらくの間市場に出回っており、価格と取引量の安定性により、新規上場証券よりも予測可能になっています。

###経験豊富なセキュリティ製品

[新規株式公開](/ipo)と同様の方法で引受会社によって管理されます。違いは、新株の価格設定が既存の発行済み株式の市場価格に基づいていることです。投資家は、経験豊富な証券提供の発表を経済的問題の指標として解釈することができます。このニュースにより、[発行済み株式](/outstandingshares)と新株の両方の価格が下落する可能性があります。

新しい株式を作成する経験豊富な証券の提供は、流通市場での株式の総量を増やすため、既存の株主の保有を大幅に希薄化する可能性があります。ただし、既存の株主からの経験豊富な問題は、既存の株主を希薄化するものではありません。そのため、ベテランの問題の売り手が誰であるかを知ることが重要です。

既存の株主からの経験豊富な証券提供には、会社の株式の全部または一部を売却する創設者または他のマネージャー（ベンチャーキャピタリストなど）が関与します。これは、会社の当初のIPOに「ロックアップ」期間が含まれていた状況でよく見られます。この期間中、設立株主は株式の売却を禁止されていました。

したがって、経験豊富な証券の提供は、設立株主が彼らの立場を現金化するための好ましい方法です。ベテランの証券オファリングは、企業の現金が不足していることを示す場合もあるため、投資家は、ベテランの証券オファリングの購入を検討する際に、企業の財務状況のさまざまな角度を考慮することが重要です。また、大量の株式、特に取引量の少ない株式を売却すると、株価に下落圧力がかかる可能性があります。

##経験豊富なセキュリティオファリングの例

新しい工場のために資金を調達するために、経験豊富な証券オファリングで追加の株式を売却したい公開会社であるABC社を考えてみてください。目標を達成するために、ABC社は引受人を雇って販売を処理し、SECに提供を登録します。売却が行われると、会社は有価証券の売却から資金を受け取ります。

個人投資家は、経験豊富なセキュリティ製品を作成することもできます。この種のベテラン発行では、個人投資家は公開会社ではなく株式の売却による収入を受け取りますが、発行済み株式を希薄化することもありません。

##ハイライト

-既存の株主からの経験豊富な証券提供は、他の既存の株主の保有を希薄化することはありませんが、新しい株式を作成する提供は、利用可能な株式の合計量を増やすことによって保有を希薄化します。

-ベテラン証券とは、ある程度の価格と取引の安定性を確立するのに十分な期間、流通市場で取引されている証券です。

-IPOと同様の方法で引受会社によって計画および実行されますが、経験豊富な証券オファリングは、既存の市場シェアの価格に基づいて価格設定されます。

-ベテラン証券は、新規株式公開（IPO）を通じて最初に提供された後、新しい証券がしばしば経験するボラティリティの傾向が少なくなります。
