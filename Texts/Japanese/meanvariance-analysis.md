# 平均分散分析
##平均分散分析とは何ですか？

平均分散分析は、期待収益に対して分散として表されるリスクを評価するプロセスです。投資家は、平均分散分析を使用して投資決定を行います。投資家は、さまざまなレベルの報酬と引き換えに、どれだけのリスクを喜んで引き受けるかを検討します。平均分散分析により、投資家は、特定のレベルのリスクで最大の報酬を見つけるか、[特定のレベルのリターンで最小のリスクを見つけることができます](/riskreturntradeoff)。

##平均分散分析を理解する

平均分散分析は[現代ポートフォリオ理論の一部で](/modernportfoliotheory)あり、投資家が完全な情報を持っていれば投資について合理的な決定を下すと想定しています。 1つの仮定は、投資家が低リスクと高報酬を求めているということです。平均分散分析には、分散と期待収益という2つの主要な要素があります。[分散](/variance)は、セット内の数値がどの程度変動または分散しているかを表す数値です。たとえば、分散は、特定の[証券のリターンが](/security)日次または週次ベースでどの程度分散しているかを示す場合があります。期待収益は、証券への投資の推定収益を表す確率です。 2つの異なる証券の期待収益が同じであるが、一方の分散が小さい場合は、分散が小さい方が適切です。同様に、2つの異なる証券の差異がほぼ同じである場合は、リターンが高い方が適切です。

[現代のポートフォリオ理論で](/modernportfoliotheory)は、投資家はさまざまなレベルの変動と期待される収益で投資するさまざまな証券を選択します。この戦略の目標は、投資を差別化することです。これにより、市場の状況が急速に変化した場合に壊滅的な損失が発生するリスクが軽減されます。

##平均分散分析の例

どの投資が最大の分散と期待収益を持っているかを計算することが可能です。次の投資が投資家のポートフォリオにあると仮定します。

投資A：金額= $ 100,000、期待収益は5％

投資B：金額= $ 300,000および10％の期待収益

ポートフォリオの合計値が$400,000の場合、各資産の重みは次のようになります。

投資Aウェイト=$100,000 / $ 400,000 = 25％

投資Bの重み=$300,000 / $ 400,000 = 75％

したがって、ポートフォリオの総期待収益は、ポートフォリオ内の資産のウェイトに期待収益を掛けたものです。

ポートフォリオの期待収益=（25％x 5％）+（75％x 10％）= 8.75％。ポートフォリオの分散は、投資の分散の単純な加重平均ではないため、計算がより複雑になります。 2つの投資間の相関は0.65です。投資Aの標準偏差または分散の平方根は7％であり、投資Bの標準偏差は14％です。

この例では、ポートフォリオの差異は次のとおりです。

ポートフォリオの差異=（25％^ 2 x 7％^ 2）+（75％^ 2 x 14％^ 2）+（2 x 25％x 75％x 7％x 14％x 0.65）= 0.0137

ポートフォリオの標準偏差は、答えの平方根です：11.71％。

##ハイライト

-2つの異なる証券の期待収益が同じで、一方の分散が小さい場合は、分散の小さい方が優先されます。

-期待収益は、証券への投資の推定収益を表す確率です。

-平均分散分析は、投資家が投資決定を比較検討するために使用するツールです。

[証券のリターンが](/security)日次または週次ベースでどの程度分散しているかを示します。

-分析は、投資家が特定のレベルのリスクで最大の報酬または特定のレベルのリターンで最小のリスクを決定するのに役立ちます。

-同様に、2つの異なる証券の分散がほぼ同じである場合は、リターンの高い方が優先されます。
