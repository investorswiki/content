# アービトラージ価格理論（APT）
##裁定価格理論（APT）とは何ですか？

アービトラージ価格理論（APT）は、資産の期待収益と体系的なリスクを捉える多数のマクロ経済変数との間の線形関係を使用して資産の収益を予測できるという考えに基づく[多要素資産価格モデルです。これは、一時的に誤った価格が設定されている可能性のある証券を特定するために、](/multifactor-model)[バリュー投資の](/valueinvesting)観点からポートフォリオを分析するための便利なツールです。

APTモデルのベータ係数は、線形回帰を使用して推定されます。一般に、過去の有価証券のリターンは、ベータを推定するための要因に基づいて回帰されます。

##裁定価格理論の仕組み

[資本資産価格モデル（CAPM）](/capm)の代替として、1976年にエコノミストのスティーブンロスによって開発されました。市場が完全に効率的であると想定するCAPMとは異なり、APTは、市場が最終的に修正されて証券が公正価値に戻る前に、市場が証券の価格を誤って設定することがあると想定しています。 APTを使用して、仲裁人は公正な市場価値からの逸脱を利用することを望んでいます。

[裁定取引](/arbitrage)の意味でのリスクのない操作ではありません。投資家は、リスクのない利益を固定するのではなく、モデルが正しく、方向性のある取引を行っていると想定しているためです。

##APTの数学モデル

APTはCAPMよりも柔軟性がありますが、より複雑です。 CAPMは1つの要素（市場リスク）のみを考慮しますが、APT式には複数の要素があります。また、セキュリティがさまざまなマクロ経済リスクに対してどれほど敏感であるかを判断するには、かなりの量の調査が必要です。

要因とその使用数は主観的な選択です。つまり、投資家は選択に応じてさまざまな結果を得ることになります。ただし、通常、4つまたは5つの要因が、セキュリティの利益の大部分を説明します。

APTファクターは、投資ポートフォリオの分散によって削減できない体系的なリスクです。価格予測因子として最も信頼できることが証明されているマクロ経済要因には、インフレの予想外の変化、[国民総生産](/gnp)（GNP）、社債のスプレッド、イールドカーブの変化などがあります。その他の一般的に使用される要因は[、国内総生産](/gdp)（GDP）、商品価格、市場指数、および為替レートです。

##裁定価格理論の使用例

たとえば、次の4つの要因は、株式のリターンを説明するものとして識別され、各要因に対する感度と、各要因に関連するリスクプレミアムが計算されています。

-国内総生産（GDP）の成長：**ß** = 0.6、RP = 4％

-インフレ率：**ß** = 0.8、RP = 2％

-金価格：**ß** = -0.7、RP = 5％

-Standard and Poorの500インデックスリターン：**ß** = 1.3、RP = 9％

-リスクフリーレートは3％です

APT式を使用して、期待収益は次のように計算されます。

-期待収益=3％+（0.6 x 4％）+（0.8 x 2％）+（-0.7 x 5％）+（1.3 x 9％）= 15.2％

-APTを使用して、仲裁人は公正な市場価値からの逸脱を利用することを望んでいます。

-裁定価格理論（APT）は、資産の期待収益と体系的なリスクを捉える多数のマクロ経済変数との間の線形関係を使用して資産の収益を予測できるという考えに基づく多要素資産価格モデルです。
