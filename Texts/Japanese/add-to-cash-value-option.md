# 現金価値オプションに追加
##現金価値オプションへの追加とは何ですか？

現金価値への追加オプションは、現金価値[生命保険契約に見られる契約条件](/lifeinsurance)です。現金価値への追加オプションを行使することにより、保険契約者は、保険契約者に支払われるのではなく、保険契約で得られた[配当を保険契約の現金価値に追加することを許可します。](/dividend)

##現金価値オプションへの追加のしくみ

現金価値生命保険の保有者は、保険の補償範囲に対して[保険料を支払い、その一部は](/insurance-premium)[現金価値口座](/cash-value-life-insurance)に振り向けられます。この現金価値は、保険会社が保険契約者に代わって投資し、その過程で[利息](/interest)と配当を累積することができます。保険契約者は、保険の月額保険料の支払いやローンの[担保の源泉など、さまざまな方法で口座の現金価値を使用できます。](/collateral)

現金価値をより迅速に構築するために、保険契約者は、自分の口座で得た配当金を口座の現金価値に自動的に再投資することを選択することもできます。これにより、短期的には保険契約者に支払われる収入が減少しますが、中長期的には保険契約者に利益をもたらす可能性があります。

最終的に、現金価値への追加オプションを行使するかどうかの決定は、保険契約者の短期的なキャッシュフローのニーズや保険契約内の長期的な投資収益率の見通しなどの要因に依存します。結局のところ、保険契約者は、蓄積された現金価値の全部または一部を引き出し、その収益を自分で投資することができます。したがって、配当金を現金価値に再投資するかどうかを決定する際、保険契約者はまず、将来の保険契約から合理的に期待できる[リターンのレベルを評価する必要があります。](/returnoninvestment)

##現金価値オプションへの追加の実際の例

ミカエラは最近生命保険を購入した若い専門家です。彼女の保険契約の条件の下で、彼女の毎月の保険料の一部は、彼女に代わって彼女の投資会社によって管理されている現金価値口座に発生します。保険会社は、この現金価値をさまざまな[投資手段](/investmentvehicle)に投資し、時間をかけて成長させることを目的としています。

彼女は30代前半であるため、ミカエラには長期的な[投資期間があり](/timehorizon)ます。したがって、彼女は保険契約で現金価値への追加オプションを行使することを決定し、現金価値口座で得た配当を保険に再投資できるようにします。そうすることでの彼女の意図は、政策の現金価値が中長期的にもっと急速に成長することを可能にすることです。

最終的に、ミカエラは、現金価値に対する借入、現金価値の引き出し、または現金価値を使用して毎月の保険料の一部またはすべてを支払うなどのアクションを通じて、この現金価値から利益を得ることができます。

##ハイライト

-これにより、保険契約者は、保険契約内で獲得した配当金を保険証券の現金価値勘定に再投資することができます。

-現金価値への追加オプションは、多くの生命保険契約に見られる条項です。

-これにより、将来的には、保険契約者が保険契約の現金価値に対して借り入れたり、引き出したりできるようになるなど、より大きなメリットがもたらされる可能性があります。

＃＃よくある質問

###現金価値への追加はどのように機能しますか？

保険契約者は、保険契約者に支払われるのではなく、保険契約で得られた配当を保険契約の現金価値に追加することを選択します。これにより、獲得した配当金をポリシーに再投資して、中長期的に現金価値をより迅速に成長させることができます。

###現金価値に追加オプションを選択する前に考慮すべきこと：

保険契約者の短期的なキャッシュフローのニーズや、保険契約内の長期的な投資収益率の見通しなどの要因を考慮する必要があります。保険契約者は、累積現金価値の全部または一部を引き出し、収益を自分で投資することを選択できます。保険契約者は、このオプションを選択する前に、今後の保険契約から合理的に期待できる[リターン](/returnoninvestment)のレベルを評価する必要があります。
