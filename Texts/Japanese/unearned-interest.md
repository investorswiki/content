# 不労所得
##不労所得とは何ですか？

不労所得とは、貸付機関がローンで回収したものの、まだ収益（または収益）として認識されていない利息です。代わりに、最初は負債として記録されます。ローンが早期に返済される場合、未稼得利息部分は借り手に返済されなければなりません。

不労利息は、不労割引とも呼ばれます。

##不労所得の内訳

貸付活動の結果として金融機関の帳簿に記録された利息は、稼得または未稼得のいずれかです。獲得利息は、その名前が示すように、貸し手に定期的な一連の義務付けられた支払いを支払う投資から特定の期間にわたって獲得される利息収入です。獲得した利息は、定められた期間の後に債券保有者に支払われる利息を通じて[債券から生成することができます。](/bond)

>不労利息は回収されましたが、収益（または収益）として認識されていません。最初は負債として記録されます。

>>

貸し手が受け取るすべての利息が得られるわけではありません。ほとんどの貸し手は、月の初めに行われるローンの支払いをスケジュールします。指定された期間に資金を貸し出すために貸し手に補償するために借り手が支払う利息は、貸し手への利息収入を表します。ただし、月初めに借り手が支払った利息は、月全体[の借り入れ費用に適用されるため、貸し手はこれを獲得していません。](/costofdebt)たとえば、借り手が毎月1日に、通常の1,200ドルの支払いを行い、そのうち240ドルが利息部分であると仮定します。 240ドルは、1か月全体でローンを利用するための借り手の費用です。彼が利息を前払いしたので、ローンの元本が十分に長く未払いでなかったので、240ドルは貸付機関によって獲得されません。この取引を認識するために、現金勘定が借方に記入され（現金が増加）、元帳の未稼得利息収入勘定に貸方記入されます。これは、銀行がそのような収入を記録しているが、利息部分を不労所得として認識していることを示しています。

ローンが早期に返済される場合、未稼得部分は借り手に返済されなければなりません。たとえば、借り手が車で36か月のローンを組むとします。彼女が30か月後にローン全体を完済した場合、彼女は6か月の未稼得利息を払い戻されます。これは、彼女がローンを早期に返済することによって節約できる金額です。

##不労所得の償却

不労所得は、長期の債券を扱うために貸付機関が使用する会計方法です。当初は負債として記録されていましたが、未稼得利息は、時間の経過とともに利息が発生するにつれて、最終的には貸付機関の帳簿に貸付期間中の収益として記録されます。この会計処理は、未稼得利息の[償却と呼ばれます。](/amortization)

不労所得を償却する場合、収益の一部は一度に1つの期間に割り当てられます。[前払利息](/prepaidinterest)を償却するために、前受利息勘定が借方に記入され、利息収入勘定が貸方に記入されます。

##不労所得の計算

[は、78](/ruleof78)のルールとして知られている方法を使用して推定することができます。規則78は、事前計算されたローン、つまり、ローンが行われる前に財務費用が計算されるローンを扱います。規則78は、ローンが早期に返済された場合に返済される財務費用または利息の金額を計算するために使用されます。不労所得の計算式は次のとおりです。

不労所得=Fx [k（k + 1）/ n（n + 1）]

ここで、F=総財務費用=nx M – P

M=定期的な毎月のローン支払い

P=元のローン金額

k=現在の支払い後のローン支払いの残りの数

n=元の支払い回数

たとえば、借り手は車に10,000ドルのローンを借りて、毎月48回の分割払いで310.00ドルを返済します。しかし、彼女は36か月後にローンを返済します。貸し手の不労所得は次のように計算できます。

F =（48 x $ 310）-$ 10,000

F = $ 4,880

不労所得=$4,880 x [（12 x 13）/（48 x 49）]

不労所得=$4,880 x（156/2352）

不労所得=$4,880 x 0.0663

不労所得=323.67ドル
