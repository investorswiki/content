# 比較可能な企業分析（CCA）
##比較可能な企業分析（CCA）とは

比較可能な企業分析（CCA）は、同じ業界内の同様の規模の他のビジネスのメトリックを使用して企業の価値を評価するために使用されるプロセスです。比較可能な企業分析は、 [EV / EBITDA](/ebitda-ev-multiple)のように、類似した企業が類似した評価の倍数を持つという仮定の下で機能します。アナリストは、レビュー対象の企業で利用可能な統計のリストを作成し、それらを比較するために評価の倍数を計算します。

##比較可能な企業分析（CCA）を理解する

すべての銀行家が最初に学ぶことの1つは、コンプ分析または同等の企業分析を行う方法です。比較可能な企業分析を作成するプロセスはかなり簡単です。レポートが提供する情報は、株価または会社の価値の価値の大まかな見積もりを決定するために使用されます。

##比較可能な企業分析

比較可能な企業分析は、同じ業界または地域の同様の規模の同様の企業で構成される[ピアグループを確立することから始まります。](/peer-group)その後、投資家は特定の会社を競合他社と相対ベースで比較することができます。この情報を使用して、企業の[企業価値](/enterprisevalue)（EV）を決定し、企業をそのピアグループの企業と比較するために使用される他の比率を計算できます。

##相対的対比較可能な企業分析

会社を評価する方法はたくさんあります。最も一般的なアプローチは、キャッシュフローと同業他社と比較した相対的なパフォーマンスに基づいてい[ます](/cashflow)。[割引キャッシュフロー（DCF）モデル](/dcf)など、現金に基づくモデルは、アナリストが将来のキャッシュフローに基づいて[固有の値を](/intrinsicvalue)計算するのに役立ちます。次に、この値が実際の市場価値と比較されます。本質的な価値が市場価値よりも高い場合、株式は過小評価されます。本質的な価値が市場価値よりも低い場合、株式は過大評価されます。

本質的な評価に加えて、アナリストは相対比較でキャッシュフロー評価を確認することを好みます。これらの相対比較により、アナリストは業界のベンチマークまたは平均を作成できます。

比較可能な企業分析で使用される最も一般的な評価指標は、[企業価値対売上高](/enterprisevaluesales)（EV / S）、[価格対](/price-earningsratio)[収益](/price-earningsratio)（P / E）、[価格対帳簿](/price-to-bookratio)（P / B）、および[価格対売上高](/price-to-salesratio)（P / S）です。会社の評価率が同業他社の平均よりも高い場合、会社は過大評価されます。評価率が同業他社の平均よりも低い場合、会社は過小評価されています。本質的および相対的な評価モデルを一緒に使用すると、アナリストが企業の真の価値を評価するのに役立つ評価のボールパーク測定値が提供されます。

##コンプで使用される評価とトランザクションのメトリック

[コンプ](/comps)は、トランザクションの倍数に基づくこともできます。トランザクションは、同じ業界での最近の買収です。アナリストは、株式ではなく会社の購入価格に基づいて倍数を比較します。特定の業界のすべての企業が平均1.5倍の市場価値または10倍の収益で販売している場合、アナリストはこれらのベンチマークに基づいて同じ数値を使用して同業他社の価値に戻すことができます。

##ハイライト

-比較可能な企業分析は、類似したメトリックに基づいて企業を比較し、企業価値を決定するプロセスです。

-企業の評価比率は、企業が過大評価されているか過小評価されているかを決定します。比率が高い場合、それは過大評価されています。それが低い場合、会社は過小評価されています。

-比較可能な企業分析で使用される最も一般的な評価指標は、企業価値対売上高（EV / S）、価格対収益（P / E）、価格対帳簿（P / B）、および価格対売上高（P / S）です。 。
