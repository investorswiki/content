# 残存価額
##サルベージバリューとは何ですか？

[耐用年数](/usefullife)の終わりに資産と引き換えに受け取ると予想するものに基づいて、[減価償却が完了し](/depreciation)た後の[資産](/asset)の推定簿価です。そのため、資産の見積残存価額は、減価償却スケジュールの計算において重要な要素です。

##サルベージバリューを理解する

見積残存価額は、会社が時間の経過とともに帳簿上で減価償却する資産について決定できます。すべての企業には、残存価額を見積もるための独自の基準があります。一部の企業は、資産の残存価額が非常に小さいため、資産を常に0ドルに減価償却することを選択する場合があります。一般に、残存価額は、減価償却費が全額計上された後の会社の帳簿上の資産の帳簿価額になるため、重要です。これは、会社が耐用年数の終わりに資産の売却から受け取ると期待する価値に基づいています。場合によっては、残存価額は、減価償却された運用不能な資産を部品として売却することで取得できると会社が信じている金額にすぎないことがあります。

###減価償却費と残存価額の仮定

企業は、資産の減価償却と残存価額を想定する際に、マッチングの原則を考慮に入れています。マッチングの原則は、関連する収益が得られるのと同じ期間に企業が費用を認識することを要求する[発生主義会計の概念です。](/accrualaccounting)企業が資産が長期間収益に貢献することを期待している場合、その資産は長く、耐用年数があります。

会社が資産の耐用年数を確信していない場合、完全な減価償却後に資産を帳簿に載せるか、資産をその残存価額で売却するために、より少ない年数とより高い残存価額を見積もることがあります。企業が減価償却費を前払いしたい場合は、より多くの減価償却費を前払いで差し引く[加速償却方法を使用できます。](/accelerateddepreciation)多くの企業は、資産の使用率がその費用認識とその耐用年数にわたる収益と完全に一致していると信じているため、0ドルの残存価額を使用しています。

##減価償却方法

減価償却スケジュールを作成するために必要ないくつかの仮定があります。財務会計士が選択できる減価償却の主な方法は、[定額法](/straightlinebasis)、[定率法](/decliningbalancemethod)、[倍額定率法](/double-declining-balance-depreciation-method)、[年数の合計](/sum-of-the-years-digits)、[生産単位の5つ](/unit-of-production-method)です。減価償却費、倍額定率法、および年数の合計の桁法は、加速償却法であり、初期の減価償却費は前払いで高くなっています。

これらの各方法では、残存価額を考慮する必要があります。資産の減価償却額は、すべての減価償却費が記録された後の減価償却[累計額の合計であり、取得](/accumulated-depreciation)[原価](/historical-cost)から残存価額を差し引いた結果でもあります。減価償却中の資産の簿価は、取得原価から現在までの減価償却累計額を差し引いたものです。

###定額法による減価償却

通常、定額法は最も基本的な減価償却方法です。これには、資産全体がその残存価額まで減価償却されるまで、耐用年数全体を通じて毎年均等な減価償却費が含まれます。

たとえば、ある会社が5,000ドルのコストでマシンを購入するとします。同社は、1,000ドルの残存価額と5年の耐用年数を決定します。これらの仮定に基づくと、定額法を使用した年間減価償却費は、（5,000ドルのコスト-1,000ドルの残存価額）/ 5年、つまり年間800ドルです。これにより、減価償却率は20％（$ 800 / $ 4,000）になります。

###残高の減少

定率法は加速償却法です。この方法では、毎年、定額法による減価償却率に残りの減価償却額を掛けた額で機械を減価償却します。資産の簿価は前年度の方が高いため、同じ割合で前年度の減価償却費は大きくなり、毎年減少します。

上記の例を使用すると、マシンのコストは5,000ドル、残存価額は1,000ドル、寿命は5年で、毎年20％で減価償却されるため、初年度の費用は800ドル（4,000ドルの減価償却額* 20％）になります。 2年目は640ドル（（4,000ドル-800ドル）* 20％）など。

###倍額定率

倍額定率法（DDB）方式では、定額法の2倍の減価償却率を使用します。機械の例では、減価償却率は20％です。したがって、DDB方式では、減価償却費が（20％x 2）または年間の残りの減価償却額の40％で記録されます。

残高の減少とDDBの両方で、会社は減価償却額を決定するために初期残存価額を設定する必要があります。

###年の合計の桁

この方法では、減価償却計算用の端数が作成されます。上記の例を使用すると、耐用年数が5年の場合、分母は5 + 4 + 3 + 2 + 1=15になります。分子は、資産の耐用年数の残りの年数です。その場合、5年間のそれぞれの減価償却費の割合は、5 / 15、4 / 15、3 / 15、2 / 15、および1/15になります。各端数に減価償却合計額を掛けます。

###生産単位

この方法では、資産が耐用年数にわたって生産する総ユニット数の見積もりが必要です。次に、減価償却費は、生産されたユニットの数に基づいて年間計算されます。この方法では、減価償却額に基づいて減価償却費も計算されます。

##ハイライト

-残存価額は、すべての減価償却費が全額計上された後の資産の簿価です。

-残存価額は、会社が減価償却スケジュールで使用する減価償却額の合計に影響します。

-資産の残存価額は、耐用年数の終了時に資産を売却または分割することと引き換えに企業が受け取ると予想されるものに基づいています。

-残存価額が非常に小さいため、企業は資産を完全に0ドルまで減価償却する場合があります。
