# プリンシパル為替レートリンクセキュリティ（PERL）
##プリンシパル為替レートリンクセキュリティ（PERL）とは何ですか？

、外国為替レートに[連動](/linked-exchangerate-system)する利回りを持つ債務への投資の一種です。つまり、元本返済額は、返済期日の米ドルと比較した特定の通貨の為替レートによって決定されます。

PERLの購入者の多くは、この種の[債務証券](/debtsecurity)を外国為替レートの変動をヘッジする手段と見なしている企業です。また、特定の外貨の価格がどちらの方向に動くかを知っていると考える投機家が購入することもあります。

##元本為替レート連動証券（PERL）を理解する

PERLは、2者間で売買される債務証券または債務証券です。彼らは半年ごとに、特定の通貨と基本通貨（通常は米ドル）の為替レートによって決定される金額を購入者に支払います。

これにより、PERLは、[クーポン](/coupon)と元本を基本通貨で支払い、元本の支払いは設定された償還式に従って変化する一種の[二重通貨建債になります。](/dual-currency-bond)この式により、変数は、基本通貨である米ドルと比較した、選択した通貨の動きにリンクされます。

PERLは通常米ドル建てで、利息は米ドルで支払われますが、返済額は特定の期間内のドルと特定の外貨との間の[為替レートによって決定されます。](/exchangerate)

外貨が米ドルに対して上昇するにつれて、元本の支払いは増加します。外貨がドルに対して下落すると、支払いは減少します。

グローバルビジネスを希望する企業は、通貨がドルへのリンクを維持できるようにするPERLを購入することで、より安全にビジネスを行うことができます。

###リバースPERL

逆PERLもあります。これはある通貨建てですが、別の通貨に利息を支払います。

逆PERLの場合、元本の支払いは外貨に対して基本通貨が高くなるにつれて増加し、支払いは基本通貨の減価とともに減少します。

逆PERLの例は、ドルで利息を支払う円建ての債券です。ドルが円に対して上昇すれば投資家の[利回りは上がるが、ドルの価値が下がると利回りは下がる。](/yield)

##ハイライト

-米ドルが他の通貨に対して上昇した場合、PERLの利回りは低下します。

-米ドルが他の通貨に対して上昇した場合に利回りが増加する逆PERLもあります。

--PERLは、米ドルで購入され、米ドルで利息を支払うタイプの債券ですが、最終的な返済額は2番目の通貨で決定されます。
