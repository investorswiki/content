# Know Sure Thing（KST）
## Know Sure Thing（KST）とは何ですか？

Know Sure Thing（KST）は、トレーダーが解釈しやすいように変化率の読み取り値を作成するためにMartinPringによって開発されたモメンタム[オシレーターです。](/oscillator)

## Know Sure Thing（KST）の計算

KSTは、4つの異なる[変化率（ROC）期間の](/rateofchange)[単純移動平均](/sma)（SMA）を取得し、それらを合計してKSTを作成し、9期間のSMAを取得して信号線を作成することによって計算されます。 KST。

## Know Sure Thing（KST）を理解する

[相対力指数](/rsi)（RSI）など、他の多くの運動量オシレーターと同じように使用できます。トレーディングシグナルは、KSTがシグナルラインを横切るときに生成されますが、トレーダーは、価格、買われ過ぎまたは売られ過ぎの状態、または中心線の[クロスオーバー](/crossover)との収束と発散を探すこともあります。

多くのトレーダーは、KSTインジケーターを他の形式のテクニカル分析と組み合わせて、成功の確率を最大化します。たとえば、トレーダーは、意思決定に役立つ他の非モメンタムインジケーター、チャートパターン、または[ローソク足パターンを調べる場合があります。](/candlestick)

> 1992年の**株式と商品**の記事で、プリンはこの指標を「合計変化率（KST）」と呼んでいましたが、KSTの用語は[テクニカルアナリストに固執していました](/technical-analyst)。

、成功する取引のオッズを最大化するために、他の形式の[テクニカル分析](/technicalanalysis)も検討した可能性があります。たとえば、上記のチャートを見ているトレーダーは、シグナルが発生した日のかなりの弱気な[ボリューム](/volume)、またはトレーディングシグナルに至るローソク足パターンを決定時に考慮した可能性があります。これらの洞察は、KST指標によって提案された収益性の低い取引を回避するためにも使用できます。

##ハイライト

--Know Sure Thing（KST）は、変化率の価格データを解釈することを目的としたモメンタムオシレーターです。

-取引シグナルは、KSTがシグナルラインを横切るときに生成されますが、トレーダーは買われ過ぎまたは売られ過ぎの状態も探します。

-トレーダーはまた、KSTを他のテクニカル分析と組み合わせて、取引が成功する確率を最大化します。
