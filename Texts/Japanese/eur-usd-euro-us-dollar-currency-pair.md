# 通貨ペア
##通貨ペアとは何ですか：EUR / USD（ユーロ/米ドル）？

通貨ペアEUR/USDは、米ドルに対するユーロの短縮形、または欧州連合（EU）と米国（USD）の通貨のクロスです。通貨ペアは、1ユーロ（基本通貨）を購入するために必要な米ドル（見積もり通貨）の数を示します。 EUR / USD通貨ペアの取引は、「ユーロ」の取引とも呼ばれます。 EUR / USDペアの価値は、1ユーロ/米ドルとして見積もられています。たとえば、ペアが1.50で取引されている場合、1ユーロを購入するには1.5米ドルかかることを意味します。

##通貨ペアの基本：EUR / USD（ユーロ/米ドル）

EUR / USDペアは、世界最大の2つの経済圏の組み合わせを表すため、世界で最も広く取引されているペアになりました。これは、相互に、および他の通貨との関係で、ユーロおよび/または米ドルの価値に影響を与える影響を受ける要因によるものです。このため、欧州中央銀行（ECB）と連邦準備制度（FRB）の[金利](/interest-rate-differential)[差](/interest-rate-differential)は、これらの通貨を相互に比較した場合の価値に影響を与えます。たとえば、FRBが米ドルを強くするために公開市場活動に介入した場合、ユーロに比べて米ドルが上昇するため、EUR/USDクロスの価値が下がる可能性があります。同じように、EU経済からの悪いニュースは、EUR/USDペアの価格に悪影響を及ぼします。イタリアとギリシャでの政府債務危機と移民流入のニュースはユーロの売り切りをもたらし、ペアの為替レートは急落しました。

##ユーロ通貨の簡単な歴史

[マーストリヒト条約](/maastricht-treaty)の結果として1992年に発足しました。もともとは1999年に会計通貨として導入されました。1月に。 2002年1月1日、ユーロはEU加盟国で流通し始め、数年の間に欧州連合の受け入れ通貨となり、最終的には多くの加盟国の通貨に取って代わりました。続いて、ユーロは統合され、多数のヨーロッパ経済を代表しています。これは、欧州連合のすべての加盟国の為替レートとボラティリティを安定させるのに役立ちます。また、ユーロは外国為替市場で最も取引量の多い通貨の1つであり、米ドルに次ぐものです。

2018年3月26日の時点で、欧州連合の28の加盟国のうち19がユーロを使用しています。 ECBによると、2017年1月1日の時点で、世界で1兆ユーロ以上が流通しています。

## EUR/USD価格チャートを読む

示された価格が株式の価格を直接表す株式の価格チャートとは異なり、通貨ペアの価格チャートにリストされている価格は、2つの通貨の[為替](/exchangerate)[レート](/exchangerate)を表します。したがって、チャートの方向指示は基本通貨に対応します。前の例を使用すると、トレーダーがEUR / USD通貨で1.50でロングポジションをとる場合、レートが1.70に上がると、ユーロの強さが増し（価格チャートに示されているように）、米ドルが弱まります。現在、同じユーロを購入するには$ 1.70（より多くのドル）が必要であり、ドルが弱くなり、ユーロが強くなります。

ただし、ペアの基本通貨は固定されており、常に1単位を表すことを理解することが重要です。したがって、強化および/または弱化の原因はレートに反映されません。ユーロが強くなったり、米ドルが弱くなったりするため、EUR/USDレートが上昇する可能性があります。どちらの結果も、レート（価格）の上昇とそれに対応する価格チャートの上昇を条件とします。

##ハイライト

-政府の政策と、ペアの通貨市場における需要と供給の経済性の影響を受けます。

-EUR / USDペアは、1ユーロを購入するために必要な米ドルの数を表します。
