# 無料のファイル入力可能な納税フォーム
##無料のファイル入力可能な納税フォームとは何ですか？

無料ファイルの記入可能な納税フォームは、[内国歳入庁（IRS）](/irs)がそのWebサイトで納税者が利用できるようにする電子フォームです。これらは、個人が所得[税申告書](/taxreturn)を提出するために使用するハードコピーフォームの正確なデジタルバージョンです。 IRSは、必要なフォームをすでに知っている上級ユーザー向けにデジタルフォームを設計しました。

IRSは、デジタルファイリングプロセスに移行する方法として、2009年に無料ファイルの入力可能なファックスフォームを最初にリリースしました。これらの電子フォームは、それ以来、所得税申告の誤りの数を最小限に抑えています。 4,600万人以上が無料ファイルの記入可能なフォームを使用しています。

個人または家族[の収入](/income)が73,000ドル以下の納税者は、IRSと提携してFreeFileを通じて無料でサービスを提供している多くの専門的な税務ソフトウェアプログラムのいずれかの支援を受けることもできます。

もちろん、すべての納税者は、IRS.govで入手できる納税フォームを無料でダウンロードできます。無料の税務準備ソフトウェアのみが収入によって制限されます。

2021税年度の確定申告は、1月からIRSによって受け入れられています。 2022年24日。

##無料のファイル入力可能な納税フォームの仕組み

[税の還付](/tax-refund)を受けるプロセスをスピードアップします。納税者は、IRSWebサイトの無料ファイルの記入可能な税務フォームにアクセスできます。

[納税者がフォームをダウンロード](/taxpayer)すると、フォームを印刷せずに、コンピューター、電話、またはタブレットに直接データを入力してフォームに入力できます。そこから、納税者はコンピューターでフォームに署名し、IRSに電子的にフォームを提出します。完了すると、最新の暗号化技術を使用して返品が代理店に送信されます。

>個人または家族の収入が2021年に73,000ドル以下だった場合は、IRSFreeFileサービスを通じてプロの税務準備ソフトウェアを無料で使用できます。

納税者の情報を体系的にチェックして納税者がすべての[控除を受けることを保証する多くの商業税準備ソフトウェアパッケージとは異なり](/deduction)、IRS Free Fileの記入可能なフォームは、提出の支援を提供せず、州の税務フォームを含みません。申請支援のためにIRSパートナーサイトを選択できます。州税フォームを含めることもできますが、手数料がかかる場合があります。

フリーファイルの資格を得るには、個人または家族が2021年の課税年度の時点で73,000ドル以下の課税所得を持っている必要があります。

ほとんどの納税者にとって、2021年の納税申告書は2022年4月18日に支払われることに注意してください。これは、4月15日のコロンビア特別区での解放日の休日のため1日後です。ただし、メイン州またはマサチューセッツ州に住む人々の場合、これらの州での愛国者の日の休日のため、提出期限は2022年4月19日です。

##無料ファイル入門

無料の記入可能な納税フォームを使用してデジタルで納税を提出するには、事前に特定の情報が必要です。フォームに正しく記入するには、前年の確定申告書に記載されている[調整済み総所得（AGI）、](/agi)[社会保障番号](/ssn)、および配偶者と扶養家族がいる場合はその番号が必要です。

[W-2](/w2form)のように、年収、領収書、社会保障給付や[失業](/unemployment)補償の文書など、年収に関する文書にアクセスする必要があります。中小企業や賃貸、[不動産](/realestate)、ロイヤルティ、パートナーシップ、 [S法人](/subchapters)、または信託からの収入がある場合は、公式の番号と適切な文書を手元に用意してください。

Free File Fillable税務フォームを使用して申告するには、[個人識別番号](/personal-identification-number)と5桁のPIN、または確定申告書に記載されている前年のAGIも必要です。この情報が見つからない場合、IRSはAGIを取得するためのツールをオンラインで提供しています。

##ハイライト

-電子的にファイリングすると、返品がより早く返され、払い戻しがより早く返されます。

-納税者は、2021年の収入が73,000ドル以下の場合、プロの納税準備ソフトウェアを無料で使用することもできます。

-無料ファイルの記入可能な税務フォームは、米国政府が納税者がデジタルで利用できるようにする税務フォームです。
