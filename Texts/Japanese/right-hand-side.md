# 右側（RHS）
##右側（RHS）とは何ですか？

右側（RHS）は、[通貨ペアでの](/currencypair)[オファー](/ask)価格を指し、誰かが[基本](/basecurrency)通貨を売っても構わないと思っている最低価格を示します。外国為替相場は、ペアで2番目にリストされている相場通貨のうち、ペアで最初にリストされている通貨である基本通貨を購入するのにかかる金額を反映しています。

##右側（RHS）を理解する

右側（RHS）は、文字通り、外国為替価格の見積もりの右側です。相場は左側に[ビッド、右側に](/bid)[オファー](/ask)を示しています。右側は現在のオファー価格です。これは、その価格で喜んで売り手がいるので、購入したい人が即座に取引できる場所です。

たとえば、通貨ペアの見積もりが1.2500 x 1.2505の場合、右側は1.2505です。これは、誰かが基本通貨を売り、見積もり通貨を購入する意思がある価格を表します。外国為替市場では、通貨は常に交換されます。たとえば、EUR / USDでのオファーは、EUR（ベース）を販売し、USD（見積もり通貨）を購入するオファーです。

ビッドとオファーの違いは[スプレッドと呼ばれ](/bid-and-ask)ます。ビッドは、誰かが基本通貨を購入する意思のある最高価格（見積もり通貨の観点から）であり、オファーは、誰かが基本通貨を販売する意思のある最低価格です。

ビッド/アスクスプレッドのサイズは、市場の現在の流動性の指標です。狭いスプレッドは、良好な[流動性があることを意味](/liquidity)します。これにより、取引コストが削減され、リターンが最大化されます。外国為替ブローカーは通常、ビッド/アスクのスプレッドからお金を稼ぎます。これは、ビッドとアスクの価格の差が取引での利益であるためです。これは常にそうであるとは限りません。一部の外国為替ブローカーは、人為的にスプレッドを増やしたり、スプレッドからお金を稼ごうとしたりせず、代わりに各取引に手数料を請求します。

[EUR/USD通貨ペアが1.1550/1.1560で取引されている](/eur-usd-euro-us-dollar-currency-pair)場合、RHSまたはオファー価格は1.1560です。これは、誰かがEURを販売してUSDを購入することをいとわない場所、または攻撃者（または価格取得者）がEURと米ドルを売る。スプレッドは10[ピップ](/pip)です。

ユーロが国内通貨であり、通貨ペアがEUR / USDで見積もられている場合、それは[間接見積もりと呼ばれます](/indirectquote)。一方、通貨ペアがUSD / EURとして見積もられている場合、それは[直接見積りと呼ばれます](/directquote)（ユーロが国内通貨である場合）。いずれの場合も、基本通貨は常に[左側](/left-hand-side)（LHS）になり、見積もり通貨はRHSになります。

ある人が米国にいて、1ユーロ（EUR / USD）を購入するのに1.1560ドルかかると仮定します。それが間接相場に変換される場合（米国居住者の場合は直接）、0.8650/0.8658でのUSD/EUR取引として表示され、トレーダーは1ドルを0.8658ユーロで購入できます。これらの数値を取得するには、1を直接入札（1.1550）で割り、価格を尋ねます（1.1560）。

##外国為替取引の右側の例

次のチャートは、いくつかの異なる通貨ペアのビッドとアスクの価格を示しています。ビッドは左側にあり、オファーは右側にあります。ビッド価格とアスク価格の違いはスプレッドです。

[USD/CAD](/usd-cad-us-dollar-canadian-dollar-currency-pair)見積もりの意味を分析してみましょう。左側の入札は1.30527です。つまり、誰かが1米ドルを1.30527[カナダドルで喜んで購入するということです](/cad-canadian-dollar)。これは、米ドルをすぐに売りたいと思った人が取引できる最高価格です。その価格で喜んで買い手がいるからです。

右側の質問は1.30544です。つまり、誰かが1米ドルを1.30544カナダドルで売っても構わないと思っているということです。これは、米ドルをすぐに購入したいと思った人が取引できる最低価格です。その価格で喜んで売り手がいるからです。

国内通貨がCADの場合、USD/CADは直接見積もりです。見積もりは、1米ドルを購入するのに必要なCADの量、または1米ドルを購入できるCADの量を示しています。見積もりには、1つのCADを購入するのに必要な米ドル数は示されていません。そのためには、間接見積もりが必要です（国内通貨がまだCADの場合）。

間接見積もりを取得するには、1つをビッドで割り、次に1つをオファーで割ります。これにより、間接ビッドと間接オファーが提供されます。

1 / 1.30527=0.76613および1/1.30544 = 0.7660

ビッドは常に低価格であり、オファーは常に高額です。したがって、CAD / USDのビッドは0.7660で、オファーは0.76613です。これは、1カナダドルを購入するのに何米ドルかかるかを示しています。 0.76613は見積もりの右側です。

##ハイライト

-ビッド価格は見積もりの左側に表示され、オファーは右側に表示されます。

-オファー価格は、誰かが基本通貨を販売することをいとわない最低価格、または購入者が基本通貨を即座に購入できる価格です。

-右側（RHS）は、外国為替相場のオファー価格を示します。
