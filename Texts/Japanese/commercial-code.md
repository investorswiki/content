# 商法
##商法とは何ですか？

商法は、商取引を規制および促進する一連の法律です。米国では、統一商法は、市場参加者がビジネスを行ったり紛争を解決したりするときに参照できる統一された一連の基準を提供することを定めています。

##商法を理解する

米国では、50州すべてが、[統一商法](/uniform-commercial-code)（UCC）として知られる統一された商法を採用しています。 UCCは、1951年に、American Law Institute（ALI）とNational Conference of Commissioners on Unified State Laws（NCCUSL）との協力の結果として開発されました。

商法で扱われる種類の質問の例は次のとおりです。契約はいつ法的拘束力を持つようになりますか？お金が間違った当事者に送金された場合、どのようにインスタンスを処理しますか？そして、どのようにして資産の所有権を証明しますか？

UCCの目的は、商業活動のガバナンスに関する一連の標準的な法令を全国的に提供することです。州がUCCを制定すると、その州の法律で体系化されます。州は、UCCを元の形式で採用することも、地域の利益に合わせて変更することもできます。

UCCの中心的なテーマは、契約上の紛争の解決であり、懸念事項を体系化するためのガイドラインと、契約に違反した場合の進め方に関するアドバイスを提供します。 UCCは商取引に関連する幅広い問題をカバーしていますが、[不動産ではなく、主に個人の財産に関連する取引に関係してい](/realestate)ます。そのため、その記事は、販売、リース、資金移動、銀行の預け入れと引き出し、倉庫の受け取り、および所有権の文書などの主題に焦点を当てています。

UCCは、アメリカの商取引を標準化するという目標をほぼ達成しました。 50州すべてが、グアム、コロンビア特別区、北マリアナ諸島、米国バージン諸島、プエルトリコの領土と同様に、UCCの少なくとも一部を制定しました。

ルイジアナ州、プエルトリコ、ナバホネイションなど、一部の法域ではUCCのすべての条項を採用していません。 LouisianaとPuertoRicoの場合、UCCの変更は、リースと販売に関連する従来の民法の制定に関するものでした。

##商用コードの例

あなたが倉庫保管およびトラック輸送事業を所有していると仮定します。トラック運転手の1人が顧客から[船荷証券を受け取り](/billoflading)、商品の性質と目的地を指定し、会社にそれらの輸送を許可します。しかし、ガソリンスタンドでの定期的な停車中に、彼らは、盗まれたと推定される、積荷の請求書が紛失していることを発見しました。技術的には、請求書なしで商品を輸送することは許可されていません。したがって、あなたは何をしますか？配達を完了しますか、それとも商品を倉庫に戻しますか？

第7条のセクション7-601で、UCCは、積荷の請求書が紛失、盗難、または破壊された場合の対処方法に関するガイダンスを提供しています。裁判所は、積荷の請求書を失った船会社が元の積荷の請求書をもはや所有していなくても、その会社に商品の配達を完了するように命じることができると述べています。このような状況では、配送会社は、請求書なしで商品を配送することに関連する責任から解放されます。

UCCはまた、裁判所の命令が出されない場合、積荷の請求書なしで配達を完了した会社は、配達中に発生した人的傷害に対して責任を負うと述べています。

これらの規定を念頭に置いて、配達を完了するようにドライバーに指示しますが、通常よりもさらに慎重に運転し、次にガスのために停止するときにドアをロックすることを忘れないでください。

##ハイライト

-商法は、商取引を規制するために設計された一連の法律です。

-商法は、一般的な課題や紛争を解決するためのプロトコルを提供することにより、商取引を促進することができます。

-米国では、50州すべてで統一商法（UCC）が採用されています。

-UCCの中心的なテーマは、契約上の紛争の解決であり、懸念事項を体系化するためのガイドラインと、契約に違反した場合の進め方に関するアドバイスを提供します。
