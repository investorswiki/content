# ヘッジファンドマネージャー
##ヘッジファンドマネージャーとは何ですか？

ヘッジファンドマネージャーは、ヘッジファンドの運用を管理、決定、および監督する会社または個人です。[ヘッジファンド](/hedgefund)の管理は、非常に儲かる可能性があるため、魅力的なキャリアオプションになる可能性があります。成功するためには、ヘッジファンドマネージャーは、[競争上の優位性](/competitive_advantage)、明確に定義された投資戦略、適切な資本化、マーケティングおよび販売計画、およびリスク管理戦略をどのように持つかを検討する必要があります。

##ヘッジファンドマネージャーを理解する

ヘッジファンド管理会社は、ポートフォリオを担当するマネージャーが所有することがよくあります。つまり、ヘッジファンドが生み出す利益の大部分を受け取る権利があります。ヘッジファンドを締結する際、投資家は、営業費用をカバーする[管理手数料](/managementfee)[と](/performance-fee)、通常は利益として所有者に分配される成功報酬に資金を提供します。ヘッジファンドマネージャーを他のタイプのファンド管理と区別するのは、ヘッジファンドマネージャーの個人的な価値とファンドは通常、ファンド自体に直接結びついているという事実です。

ヘッジファンドへの投資を希望する個人は、収入と[純資産の](/networth)要件を満たさなければなりません。ヘッジファンドは積極的な投資戦略を追求し、他の多くの種類の投資よりも規制が少ないため、リスクが高いと見なすことができます。

トップヘッジファンドマネージャーは、あらゆる業界で最も高給のポジションを占めており、大手企業のCEOをはるかに上回っています。売上高が最も高いマネージャーの中には、年間40億ドル近くを稼ぐ人もいます。ヘッジファンドマネージャーは、競争力を維持し、常に勝者として出てくる場合、金融業界で最も高給のエグゼクティブになる可能性がありますが、一部のヘッジファンドマネージャーは、彼らが彼らの財政的努力に全く失敗すると、彼らは支払われません。

##ヘッジファンド戦略

ヘッジファンドマネージャーは、複数の戦略を使用して、会社とクライアントのリターンを最大化できます。人気のある戦略の1つは、[グローバルマクロ投資と呼ばれるものを使用すること](/globalmacro)です。アイデアは、世界的なマクロ経済動向の予測を扱っている市場で大きなシェアまたはかなりの地位と並行して投資することです。ヘッジファンドマネージャーが使用するこのタイプの戦略は、必要な柔軟性を提供しますが、戦略は優れたタイミングに大きく依存しています。

複数のヘッジファンドマネージャーを億万長者にしたもう1つの人気のある戦術は、[イベントドリブン戦略](/eventdriven)です。これは、マネージャーが企業環境で活用するための大きな機会を探していることを意味します。この例には、M＆A、破産、株主買戻しの取り組みが含まれます。この戦略に取り組んでいるマネージャーは、[価値調査](/valueinvesting)[アプローチを使用するのと同様に、市場の不整合を利用しています](/valueinvesting)。ヘッジファンドマネージャーは通常、背後に膨大なリソースがあるため、このルートを使用します。

##ヘッジファンドマネージャーの報酬

[2と20](/two_and_twenty) （または「2と20」）は、ヘッジファンド業界で標準的であり、ベンチャーキャピタルやプライベートエクイティでも一般的な典型的な手数料の取り決めです。ヘッジファンド管理会社は通常、クライアントに管理手数料と成功報酬の両方を請求します。 「2」とは、[運用資産（AUM）の2％を意味し、資産を管理](/aum)するためにヘッジファンドが請求する年間管理手数料を指します。 「20」とは、特定の事前定義されたベンチマークを超えるファンドによって得られた利益の20％の標準的な成功報酬またはインセンティブ報酬を指します。この有利な手数料の取り決めにより、多くのヘッジファンドマネージャーは非常に裕福になりましたが、近年、さまざまな理由で投資家や政治家から[手数料体系が非難されています。](/fee-structure)

一部のヘッジファンドは、成功報酬に適用される[最高水準点と戦わなければなりません。](/highwatermark)最高水準点のポリシーは、ファンドの正味価値が以前の最高値を超えた場合にのみ、ファンドマネージャーに利益の一定の割合が支払われることを指定します。これにより、ファンドマネージャーはパフォーマンスの低下に対して多額の支払いを受けることができなくなり、成功報酬が支払われる前に損失を埋め合わせる必要があります。

##ハイライト

-ヘッジファンドマネージャーは、ヘッジファンドを設立するためにプロのポートフォリオマネージャーとアナリストを雇用する金融会社または個人です。

-ヘッジファンドマネージャーは通常、特定の投資戦略を専門としており、それをファンドポートフォリオの任務として使用します。

-ヘッジファンドマネージャーは通常、平均以上の報酬を獲得します。多くの場合、投資家からの2〜20の手数料体系から得られます。
