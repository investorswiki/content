# 労働者階級
##ワーキングクラスとは何ですか？

「ワーキングクラス」とは、低賃金、限られたスキル、または肉体労働を必要とする仕事によって特徴づけられる社会的クラスの人々を表すために使用される社会経済的用語です。通常、労働者階級の仕事は教育要件を減らしました。失業者や社会福祉プログラムの支援を受けている人は、労働者階級に含まれることがよくあります。

##ワーキングクラスを理解する

「ワーキングクラス」は通常、手作業と限られた教育に関連していますが、[ブルーカラーの](/bluecollar)労働者はすべての経済にとって不可欠です。米国のエコノミストは、一般的に「ワーキングクラス」を大学の学位を持たない成人と定義しています。ワーキングクラスの多くのメンバーもミドルクラスとして定義されています。

コーネル大学の社会学教授であり、1957年の教科書** The American Class Structure **の著者であった、DennisGilbertやJosephKahlなどの社会学者は、ワーキングクラスをアメリカで最も人口の多いクラスとして特定しました。

ウィリアム・トンプソン、ジョセフ・ヒッキー、ジェームズ・ヘンスリンなどの他の社会学者は、中低層が最大であると述べています。これらの社会学者によって考案されたクラスモデルでは、ワーキングクラスは人口の30％から35％を占め、下中級クラスとほぼ同じ数です。デニス・ギルバートによれば、労働者階級は社会の25パーセントから55パーセントの間の人々で構成されています。

[Karl Marx](/karl-marx)は、ワーキングクラスを「プロレタリア」と表現し、最終的に商品を作成し、社会の富を生み出すサービスを提供したのはワーキングクラスであると説明しました。マルクス主義者と社会主義者は、労働者階級を、彼らの労働力とスキル以外に何も売ることができない人々と定義しています。その意味で、**ワーキングクラス**には、ホワイトカラーとブルーカラーの両方の労働者、すべてのタイプの手動労働者と男性労働者が含まれます。

##労働者階級の仕事の種類

今日の労働者階級の仕事は、1950年代と1960年代の労働者階級の仕事とはかなり異なります。工場や産業の仕事で働くアメリカ人は、長年にわたって衰退しています。今日、ほとんどの労働者階級の仕事はサービス部門で見られ、通常は次のものが含まれます。

-事務職

-食品業界の立場

- 小売売上高

-低スキルの肉体労働

-低レベルのホワイトカラー労働者

多くの場合、ワーキングクラスの仕事は1時間あたり15ドル未満しか支払わず、それらの仕事のいくつかには健康上の利点が含まれていません。アメリカでは、労働者階級の人口を取り巻く人口統計はますます多様化しています。労働者階級の人口の約59％は白人アメリカ人で構成されており、1940年代の88％から減少しています。アフリカ系アメリカ人は14％を占めていますが、ヒスパニックは現在米国の労働者階級の21％を占めています

##ヨーロッパのワーキングクラスの歴史

確執的なヨーロッパでは、ほとんどが労働者階級の一部でした。さまざまな職業、職業、職業で構成されるグループ。たとえば、弁護士、職人、農民はすべてメンバーであり、貴族や宗教エリートのメンバーではありませんでした。同様の階層は、ヨーロッパ以外の産業革命以前の社会にも存在していました。

これらの労働者階級の社会的地位は、自然法と一般的な宗教的信念によって定められたものと見なされていました。農民は、ドイツ農民戦争中にこの認識に異議を唱えました。 18世紀後半、啓蒙の影響下で、変化するヨーロッパは、変化のない神が創造した社会秩序の考えと和解することができませんでした。当時の裕福な社会人は、道徳的および倫理的優位性を主張して、労働者階級を抑制しようとしました。

##ハイライト

-ワーキングクラスは、低賃金で限られたスキルしか必要としない仕事が特徴の社会クラスの人々を表す社会経済用語です。

-今日、ほとんどのワーキングクラスの仕事はサービス部門で見られ、事務、小売販売、および低スキルの手動労働の職業が含まれます。

-通常、労働者階級の仕事は教育要件を減らしました。
