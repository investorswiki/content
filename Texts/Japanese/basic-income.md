# ベーシックインカム（UBI）
##ベーシックインカム（UBI）とは何ですか？

ベーシックインカム（UBI）は、すべての成人市民が定期的に一定の金額を受け取る政府のプログラムです。ベーシックインカムシステムの目標は、貧困を緩和し、官僚的な関与を必要とする可能性のある他のニーズに基づく社会プログラムに取って代わることです。ベーシックインカムの考え方は、自動化が製造業やその他の経済部門の労働者にますます取って代わられるにつれて、米国で勢いを増しています。

##ベーシックインカム（UBI）を理解する

社会のすべてのメンバーに基本的な収入を提供するという考えは何世紀も前にさかのぼります。 16世紀の英国の哲学者で政治家のトーマス・モアは、彼の最も有名な作品である**ユートピア**でこのアイデアについて言及しています。

アイデアがアメリカ独立戦争に拍車をかけるのに役立ったパンフレット作成者のトーマス・ペインは、収入が「金持ちか貧乏人かを問わず、すべての人に」政府の収入の流れを提供する税制を提案しました。

そしてマーティン・ルーサー・キング・ジュニアは、1967年に出版された彼の著書**「ここからどこへ行くのか：混沌かコミュニティか？」**で「保証された収入」を提案しました。

[勤労所得税額控除](/earnedincomecredit)（EIC）、貧困家庭一時扶助（TANF）、およびその他のプログラムを通じて低所得のアメリカ人に財政的支援を提供していますが、米国ではベーシックインカムのシステムが定着したことはありません。

しかし、近年、国民の意識が高まっています。この新たな関心の多くは、経済の根本的な変化、つまり自動化の成長に関係しており、多くのアメリカ人が生活賃金を支払う仕事を失う恐れがあります。

>2021年3月11日にバイデン大統領によって署名されたアメリカ救助計画には低所得者と中所得者への寛大な減税が含まれています。 2021年のみ、子供がいない世帯の勤労所得税額控除の額は増加します。子供がいない人の最大クレジット額は、543ドルから1,502ドルに増加します。年齢層も拡大しました。特定のフルタイムの学生を除いて、子供がいない人は25歳ではなく19歳からクレジットを請求できます（フルタイムのコースの負荷が少なくとも半分である19〜24歳の学生は対象外です）。年齢の上限である65歳は廃止されます。シングルファイラーの場合、段階的廃止の割合（クレジットの場合）は15.3％に増加し、段階的廃止の金額は$ 11,610（最大収入）に増加します。

>>

たとえば、ブルッキングス研究所による2019年のレポートによると、米国の全雇用の4分の1が自動化の影響を受けやすいことがわかりました。研究者たちは、製造、輸送、オフィス管理、食品の準備など、より日常的なタスクを伴う役割が最も脆弱であると主張しています。

ベーシックインカムの支持者は、政府からの保証された支払いが、この経済変革によって取り残された人々が貧困を回避することを確実にするのを助けることができると信じています。政府からの収入だけでは十分ではない場合でも、理論的には、彼らがまだ得ることができる低賃金またはパートタイムの仕事からの収入を補うことができます。

支持者はまた、普遍的な支払いシステムは、必要としているが他の政府プログラムの資格を得るのに苦労している人々が援助を受けるのを容易にするだろうと信じています。たとえば、障害保険の支払いを求めている一部のアメリカ人は、医療制度へのアクセスを欠いており、それによって障害を検証する能力を妨げている可能性があります。

##UBIの政治的支援

UBIの支持者の多くは、元労働長官のロバート・ライシュや影響力のある国際サービス従業員労働組合の元首長であるアンディ・スターンなど、政治的スペクトルのより自由な端から来ています。

ただし、政府提供の収入源への支援は、右側のいくつかの著名な人物によっても支持されています。

その中には、故保守的な経済学者のミルトン・フリードマンがいます。彼は、民間の慈善寄付は、多くのアメリカ人が耐える財政的負担を軽減するのに十分ではないと示唆しました。

1962年の**資本主義と自由**で、彼は「負の所得税」（本質的にはUBI）は、他の人が追随すると信じていない場合に市民が犠牲を払う傾向がないという考え方を克服するのに役立つと主張しました。 「他のすべての人がそうするならば、私たち全員が貧困の救済に貢献することをいとわないかもしれません」と彼は書いた。

リバタリアンの哲学者チャールズ・マレーは、保証された収入が政府の官僚機構も削減すると信じています。彼は、年間10,000ドルのUBIと基本的な健康保険を提案しました。これにより、政府は社会保障やその他の再分配プログラムを削減できると彼は言います。

##UBIの勢い

ベーシックインカムは、起業家で元民主党候補のアンドリューヤンがこのアイデアをキャンペーンの基礎にした後、2020年の大統領選挙の最初の段階でかなりの注目を集めました。ヤンの「自由配当」は、彼がそれを呼んだように、18歳以上のすべてのアメリカ人に毎月1,000ドルの小切手を与えるでしょう。連邦支援プログラムに登録している人は、引き続きそれらの支払いを受け取るか、代わりに自由配当を選択することができます。

ヤン氏は、[労働力人口率](/participationrate)、つまり、仕事をしている、または仕事を探しているアメリカ人の割合は、数十年で最低だったと主張した。 「自由配当は、私たちがより良い仕事を探し、自分のビジネスを始め、学校に戻り、愛する人の世話をし、次の機会に向けて取り組むことを可能にしながら、アメリカ人の基本をカバーするためのお金を提供します」と彼のキャンペーンウェブサイト了解しました。

>元大統領候補のアンドリュー・ヤンの月額1,000ドルの「自由配当」は、2021年の連邦政府の予想予算の約50％の費用がかかります。

>>

2021年3月11日に署名されたバイデン大統領のアメリカ救助計画は1.9兆ドルのパンデミック救済パッケージでした。その利点には、米国内の資格のあるすべての成人に対する刺激の支払いの別のラウンドが含まれていました。今回の刺激の支払いは、ほとんどの受信者に対して1,400ドルの金額になります。適格納税者は、子供たちごとに同じ支払いを受け取ります。資格を得るには、1人の納税者が75,000ドル以下の調整済み総所得を持っている必要があります。共同で申請する夫婦の場合、その数は150,000ドル以下である必要があり、世帯主の場合、調整後の総所得は112,500ドル以下である必要があります。

権利を剥奪された人々の確固たる支持者である教皇フランシスコは、この問題を道徳的な観点から組み立てました。 2020年のイースターの手紙の中で、神祇官は普遍的な基本賃金について次のように書いています。

##UBIへの批判

貧困を削減し、官僚的形式主義を削減するという約束にもかかわらず、ベーシックインカムは依然として困難な戦いに直面しています。おそらく最も明白な欠点はコストです。非営利のTaxFoundationによると、Andrew Yangの成人1人あたりの月額1,000ドルの自由配当は、毎年2.8兆ドルの費用がかかります（他のプログラムの統合による相殺を差し引いたもの）。

ヤンは、他の社会的プログラムの規模を縮小し、企業に10％[の付加価値税](/valueaddedtax)（VAT）を課すことによって、そのかなりの費用を部分的にカバーすることを提案しました。彼はまた、社会保障給与税の上限を廃止し、彼の保証された所得計画に貢献するであろう炭素排出に対する税を導入することを提案している。

しかし、その一連の提案が自由配当のコストを完全に相殺するのに十分であるかどうかは、依然として論争の的となっています。税務財団による分析では、ヤンの収益を生み出すアイデアは、財務省への影響全体の約半分しかカバーしないと結論付けました。

UBIに対する他の批判の中には、雇用に依存しない収入の流れが働く意欲をそぐものになるという議論があります。それも議論の的となっています。ヤンは、年間12,000ドルを提供するという彼の計画は生き残るのに十分ではないと示唆しました。したがって、成人の大多数は他の収入で支払いを補う必要があります。

##結論

最近の研究は、UBIと失業の間の弱い関連のみを示唆しています。たとえば、MITとハーバードの研究者による2016年の分析では、発展途上国の「現金移転」プログラムは、雇用行動にほとんど認識できない影響を及ぼしていることがわかりました。

しかし、一部の支持者が示唆しているように、伝統的な福祉の支払いをベーシックインカムに置き換えることが実際に雇用を増やすことを示唆する証拠はほとんどありません。フィンランドでの最近の2年間の実験では、ベーシックインカムが失業手当に取って代わり、UBIの受給者は対照群よりも新しい雇用を見つける可能性が低くなりました。

##ハイライト

-民主党の大統領候補アンドリュー・ヤンは、ベーシックインカムを2020年のキャンペーンの重要な柱とし、この問題に全国的な注目を集めるのに役立った。

-ベーシックインカムに対する主要な批判の1つはコストであり、一部の計画は連邦予算全体の半分以上を占めています。

-必要性に関係なく、市民に定期的で保証された支払いを提供するという考えは、何世紀にもわたって存在してきました。

-UBIの提案の規模はさまざまですが、ヤンの計画では、すべてのアメリカ人の成人に連邦政府から月額1,000ドルが支給されます。
