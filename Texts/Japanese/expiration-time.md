# 有効期限
##有効期限とは何ですか？

[オプション契約またはその他のデリバティブ](/optionscontract)の有効期限は、それが無効にされた正確な日時です。満了時にマネー（OTM）で終了するデリバティブ契約は無価値になりますが、インザマネー（ITM）契約は、満了時の[決済価格に基づいて評価されます。](/settlementprice)

[有効期限](/expirationdate)よりも具体的であり、そのオプションを取引する最後の時間と混同しないでください。

##有効期限を理解する

有効期限は、オプションの有効期限が実際に満了する時期であり、後者はオプションの保有者が意図を知らせる期限であるという点で、有効期限とは異なります。ほとんどのオプショントレーダーは有効期限だけを気にする必要がありますが、有効期限も知っておくと便利です。

[NASDAQに](/nasdaq)よると、有効期限は次のとおりです。

>>

>すべての運動通知を有効期限日に受け取らなければならない時刻。技術的には、有効期限は現在、有効期限の午前11時59分[東部標準時]ですが、オプション契約の公的保有者は、有効期限の前営業日の午後5時30分[東部標準時]までに行使する意向を示さなければなりません。日にち。

>>

>>

>>

オプションの多くの公的保有者は[ブローカーと取引しているため](/broker)[、](/broker)彼らは異なる有効期限に直面しています。米国では、オプションの取引の最終日は通常、満期月の金曜日ですが、[満期日](/expirationdate)はその直後の土曜日です。金曜日が祝日の場合、最終取引日は木曜日になります。

オプションの公的保有者は通常、金曜日の午後5時30分までに行使する通知を宣言する必要があります。この時間枠により、ブローカーは土曜日の実際の満期時間までに保有者の意図の[交換を通知することができます。](/exchange)

>通知制限は、製品が取引される取引所によって異なります。たとえば、[シカゴ・ボード・オプション取引所](/cboe)（CBOE）は、有効期限が切れるオプションの取引を、最終取引日の中央時間午後3時に制限しています。

##デリバティブ契約の有効期限

[デリバティブ](/derivative)の満期日は、オプションまたは先物契約が有効になる最終日です。投資家がオプションを購入する場合、契約は、行使[価格](/strikeprice)と呼ばれる所定の価格で資産を売買する権利を与えますが、義務は与えません。

オプションの行使は、有効期限日またはそれ以前の所定の期間内でなければなりません。投資家がその権利を行使しないことを選択した場合、オプションは失効して無価値になり、投資家はそれを購入するために支払われたお金を失います。

米国で3番目に上場されているストックオプションの有効期限は、通常、契約月の金曜日です。これは、契約の有効期限が切れる月です。ただし、その金曜日が休日の場合、有効期限は第3金曜日の直前の木曜日となります。オプションまたは先物契約が有効期限を過ぎると、その契約は無効になります。[エクイティ](/equity)オプションを取引する最終日は、満期前の金曜日です。

##有効期限の警告

[オプション](/option)の大部分は有効期限に達することはありませんが、一部のオプションは実際の有効期限まで存続します。取引の最後の時間は有効期限の前になる可能性があるため、この遅延は興味深いダイナミクスを生み出す可能性があります。

[原証券](/underlying-security)も同時に取引のためにクローズする場合、この時差は問題ではありません。ただし、原証券がオプションの取引の終了を超えて取引する場合、買い手と売り手の両方が、 [ITMの場合、契約の行使が自動的に行われることに気付く可能性があります](/inthemoney)。逆に、彼らは自動行使を期待するかもしれませんが、原資産の時間外取引は彼らに[OTMをプッシュするかもしれません](/outofthemoney)。

これらの可能性をカバーするルール、特に原資産の最終価格が記録される時期は変更される可能性があります。したがって、トレーダーは、オプションが取引される取引所と、口座を処理する[証券会社の両方に確認する必要があり](/brokerage-company)ます。

##例：SPXWウィークリーオプション

SPXWは、CBOEによってリストされた[S＆P500インデックスの毎週の満期サイクルオプションです。](/sp500) SPXWウィークリーは最終取引日に決済されます。通常、SPXWEOWウィークリーの場合は金曜日です。

他の午後に決済されるインデックスオプションと同様に、行使決済値は、各構成銘柄のプライマリーマーケットで最後に報告された（終値）販売価格を使用して計算されます。最終取引日の、期限切れのSPXWウィークリーの取引は、中部標準時（CST）の午後3時に終了します。一方、期限が切れていないすべてのSPXWウィークリーは、中部標準時午後3時15分まで取引を続けます。

##ハイライト

-通常、オプションを取引する最終日は、満期月の第3金曜日です。

-デリバティブ契約では、正確な有効期限と時刻が指定されます。

-満期時間とは、デリバティブ契約が取引を停止し、義務または権利が期日または満了する正確な日時です。
