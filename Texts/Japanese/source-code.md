# ソースコード
ソースコードとは、決定論的な命令とステートメントのリストに基づいてプログラム（ソフトウェア）がどのように機能するかを定義する責任があるコンピューターコードの行のコレクションを指します。本質的に、ソースコードは、最初に作成された時点でのソフトウェアの主要な状態です。

そのため、ソースコードは、Java、JavaScript、C、Python、PHP、Go、Rusty、Solidity、Haskellなどのさまざまなコンピュータプログラミング言語で記述できる一連の命令で構成されています。これらのプログラミング言語にはすべて共通の特徴があります。特定のアクションを実行し、特定の出力を生成するためにコンピューターが従う一連の命令で構成されています。

ソースコードは、作成された後、コンピュータが理解して実行できるマシン言語に「変換」またはコンパイルされることがよくあります。ただし、状況によっては、ソースコードがコンパイルせずにすぐに実行される場合があります（これらのコンピュータープログラムはインタープリターと呼ばれます）。

コンピュータソフトウェアの開発中、ソースコードの段階は、プログラムが完成して最終的にコンピュータマシンによって実行される前に、プログラマが読み取り、コメント、編集を行うことができる最初の作業段階です。ソースコードライブラリを使用すると、コード行にコメントを含めることができるため、ソフトウェアエンジニア、コンピュータサイエンティスト、およびプログラマーの作業がはるかに簡単になります（たとえば、説明的で洞察に満ちたコメントを含む読みやすいプレーンテキストの断片）。
