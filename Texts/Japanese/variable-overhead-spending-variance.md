# 変動間接費の差異
##変動間接費の差異とは何ですか？

支出差異は、特定の費用の実際の金額と費用の予想される（または予算化された）金額との差です。可変オーバーヘッド支出の差異が何であるかを理解するには、可変オーバーヘッドが何であるかを知ることが役立ちます。変動間接費は、運用活動によって変動するビジネスの運営に関連するコストです。生産量が増減すると、変動するオーバーヘッドが連動して移動します。間接費は通常、管理費などの固定費です。一方、変動間接費は生産レベルに関係しています。

変動間接費の差異は、製造に関連する間接材料のコストに基づく実際の変動間接費と、標準の変動間接費と呼ばれる予算コストとの差です。

##変動間接費の差異を理解する

変動するオーバーヘッド支出の差異は、基本的に、変動する生産オーバーヘッドが**実際に**かかるものと、期間中の活動のレベルを考慮して**あるべき**コストとの差です。

[、機械時間数](/overhead-rate)または労働時間数で表されます。業務で手動プロセスと自動プロセスの両方を使用する場合、企業は標準（予算）レートの基礎として機械時間と労働時間の両方を使用することもあります。

間接費の変動は、塗料や石油やグリースなどの消耗品などの間接材料の実際のコストが、標準または予算の変動間接費よりも低い[場合](/rawmaterials)[に有利です。](/rawmaterials)実際のコストが予算コストよりも高い場合は不利です。

>変動する生産間接費には、特定の生産単位に直接起因することができないコストが含まれます。一方、直接材料や直接労働などのコストは、生産量の単位ごとに直接異なります。

>>

##変動間接費の差異の例

使用される実際の労働時間が140であり、標準または予算の変動間接費が直接労働時間あたり$ 8.40であり、実際の変動間接費が直接労働時間あたり$7.30であるとします。変動間接費の差異は、次のように計算されます。

標準可変間接費率$8.40-実際の可変間接費率$7.30= $ 1.10

1時間あたりの差=1.10ドル×実際の労働時間140=154ドル

変動間接費の差異=$154

この場合、実際のコストは標準コストよりも低いため、差異は良好です。

規模の経済、材料の一括割引、より安価な供給、効率的なコスト管理、または予算計画の誤りにより、有利な差異が発生する可能性があります。

間接労働のコストが増加した場合、コスト管理が効果的でない場合、または予算計画に誤りがある場合、不利な差異が発生する可能性があります。

##制限

>###速報

>変動間接費の差異は、基本的に、変動生産間接費の実際のコストと、ある期間の出力に与えられるべきコストとの差です。

>>

##ハイライト

-実際のコストが予算コストよりも高い場合、変動する間接費の差異は好ましくありません。

-間接費の実際のコストが標準または予算の変動間接費よりも低い場合、変動間接費の差異は有利です。

-変動間接費の差異は、変動生産間接費が**実際に**かかる費用と、期間中の活動のレベルを考慮して**実際に**かかる費用との差です。

-標準の可変間接費率は、通常、機械時間または労働時間で表されます。
