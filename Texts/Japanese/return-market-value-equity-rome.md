# エクイティの市場価値収益率（ROME）
##エクイティの時価総額（ROME）とは何ですか？

「エクイティの市場価値収益率」（ROME）という用語は、アナリストや投資家がさまざまな規模や価値の企業の業績と価値を比較するために使用する指標を指します。専門家は、エクイティの市場価値を使用して、他の点では低い[評価で取引しているとき](/valuation)に[簿価にプラスの](/bookvalue)[利益を生み出す企業を特定します](/return)。

会社のROMEのレベルは、会社が過小評価されているか過大評価されているかを示すことができます。さまざまな規模の企業や時価総額を比較することも役立ちます。

##エクイティの市場価値収益率（ROME）の仕組み

会社の[時価総額](/marketcapitalization)は、一般的に、その[株式の市場価値であると認められています](/market-value-of-equity)。株式または時価総額の市場価値は、会社の現在の株価に利用可能な発行済み株式数を掛けることによって計算されます。この指標は、株価が変動したり、発行済み株式数が変化したりすると、常に変化します。発行済株式数は、発行済株式数の増加や自社株[買い](/buyback)などの特別な事情により変化します。

株式の市場価値のリターンは、事実上、企業の株価の[利益率です。](/yield)この測定値は、会社の業績と[価値を決定するために使用されます](/value)。アナリスト、金融専門家、および投資家は、企業の時価総額の利益率を特定するための戦略としてこれを使用します。

株式の時価総額が高い企業は、株価に比べて収益性が高いため、[過小評価されて購入する価値がある可能性があることを示唆しています。](/undervalued)一方、企業が同様の利益を与えられてより高い株価を持っている場合、それはバリューバイほど魅力的ではないかもしれません。株式の市場価値収益率は、絶対的な尺度ではなく利回りであるため、時価総額が異なるさまざまな規模の企業間で価値を比較するための有用な尺度でもあります。

[バリュー投資家](/valueinvesting)にとっても非常に有用なツールであると考えられています。これは、この指標では、将来の成長が株式の[本源的価値](/intrinsicvalue)または資産の価値に対する投資家の認識を評価するための重要な要素であると見なしているためです。

>企業の株式の市場価値は、帳簿価額が理論的に株価に組み込まれている企業の成長の可能性を考慮していないため、帳簿価額とは異なります。

##特別な考慮事項

一部の[ヘッジファンド](/hedgefund)は、購入する過小評価された株式を特定するために、エクイティ戦略の市場価値収益率を採用しています。この戦略は、企業の本源的価値を評価し、その価値をその株式の現在の観察された市場価格と比較します。本質的価値とは、資産の価値に対する認識です。具体的には、外的要因に関係なく、内的要因に基づく株式の価値です。

[コールオプション](/calloption)の本源的価値は、次の式を使用して計算できます。

>>

>本源的価値=株価-行使価格

>>

コールオプションは、期限が切れる前に行使価格で原証券を購入する権利を付与するが、義務ではない契約です。行使[価格](/strikeprice)は、契約が行使されたときに資産を購入または売却できる価格であることを忘れないでください。したがって、この価格は[行使価格とも呼ばれ](/exerciseprice)ます。この数値にオプションの総数または取得する資格のある株式数を掛けて、購入したい株式の全額の本源的価値を取得します。

##ハイライト

-ROME戦略により、ユーザーは、他の点では低い評価で取引しながら、簿価に対してプラスのリターンを生み出す企業を特定できます。

-企業の株式の市場価値は、時価総額または発行済み株式数に関連する株価です。

-一部のヘッジファンドは、購入する過小評価された株式を特定するためにROME戦略を採用しています。

-エクイティの市場価値収益率は、さまざまな規模と価値の企業の業績と価値を比較するために使用される指標です。

-ROMEは、将来の成長が株式の本源的価値を評価する重要な要素であると考えているため、バリュー投資家にとって特に有用です。
