# ハウイーテスト
##ハウイーテストとは何ですか？

Howey Testは、取引が「投資契約」として適格であるかどうかを判断するための米国最高裁判所の訴訟に言及しているため、証券と見なされ、 [1933](/security)[年証券法](/securitiesact1933)および[1934年証券取引法に基づく](/seact1934)開示および登録要件の対象となります。 。 Howey Testの下では、「他の人の努力から得られる利益が合理的に期待される一般的な企業へのお金の投資」がある場合、投資契約が存在します。

テストは、すべての契約、スキーム、またはトランザクションに適用されます。 Howey Testは、投資家やプロジェクト支援者と一緒に[ブロックチェーンおよびデジタル通貨プロジェクトを配置するために重要です。](/blockchain)特定の暗号通貨と[イニシャルコインオファリング](/initial-coin-offering-ico)（ICO）は、テスト中の「投資契約」の定義を満たすことが判明する場合があります。

##Howeyテストを理解する

Howey Testは、1946年に最高裁判所に到達した**SECv。WJHoweyCo。**を指します。HoweyCompanyは、フロリダのバイヤーに柑橘類の果樹園を売却し、バイヤーは土地をHoweyに[リースバックしました。](/leaseback)会社のスタッフは、所有者に代わって果樹園の世話をし、果物を販売する傾向がありました。両当事者は収益を共有しました。ほとんどの購入者は農業の経験がなく、土地を自分で手入れする必要はありませんでした。

Howeyは取引の登録に失敗し、[米国証券取引委員会](/sec)（SEC）が介入しました。裁判所の最終判決は、投資契約として適格なリースバック契約を決定しました。

そうすることで、最高裁判所は投資契約が存在するかどうかを決定するための4つの基準を確立しました。投資契約は次のとおりです。

1.お金の投資

1.一般的な企業で

1.利益を期待して

1.他人の努力から導き出される

Howeyの場合、フロリダの柑橘類の果樹園の購入者は、主に労働力と専門知識が他の人から提供されたため、この取引は価値があると考えていました。買い手は、収入源にアクセスするために資本を投資するだけで済みました。これにより、このトランザクションは、現在Howey Testとして知られている投資契約として分類されたため、SECに登録する必要がありました。

##ハウイーテストと暗号通貨

ビットコインなどのデジタル通貨は、分類が難しいことで有名です。それらは分散化されているため、多くの点で規制を回避しています。それにもかかわらず、SECはデジタル資産に関心を持っており、それらの売却が投資契約の定義にいつ適合するかを明確にしようと努めてきました。

SECによると、「お金の投資」テストは、[フラットマネー](/fiatmoney)または他のデジタル資産が交換されているため、デジタル資産の販売で簡単に満足できます。同様に、「一般的な企業」のテストも簡単に満たされます。

ほとんどの場合、デジタル資産が投資契約として適格であるかどうかは、 「他者の努力から得られる利益の期待」があるかどうかに大きく影響します。

たとえば、デジタルアセットの購入者は、プロジェクトの支援者がデジタルネットワークの開発と維持を（特に初期段階で）行うことに依存している場合、これらのタスクがアフィリエイトされていないユーザー。プロジェクトの支援者が、トークンの焼き付けによって不足を生み出すなど、デジタル資産の価格をサポートするための措置を講じた場合にも、テストは満たされます。 「他者の努力」テストが満たされるもう1つの方法は、プロジェクトの支援者が引き続き管理者の役割を果たしているかどうかです。

これらは、SECによって概説されたほんの一握りの例です。プロジェクトの成功が支援者の継続的な参加に依存している場合、関連するデジタル資産の購入者は「他者の努力」に依存している可能性があります。

##特別な考慮事項

SECが[暗号通貨](/cryptocurrency)トークンがセキュリティであると判断した場合、多くの影響が生じます。事実上、これは、トークンを米国の投資家に販売できるかどうかをSECが判断し、プロジェクトにSECへの登録を強制できることを意味します。

[Ether](/ether-cryptocurrency)と引き換えにDAOトークンを販売することが連邦証券法に違反していると判断したときに行われました。 SECは、執行措置を講じる代わりに、証券法がトークンの販売に適用されることを警告し、事実上、暗号通貨業界に警告を発しました。

Howey Testのため、今日行われるほとんどのICOは、米国の投資家にとって立ち入り禁止になっている可能性があります。 2018年、当時のSEC議長のジェイクレイトンは、彼が見たすべてのICOはセキュリティとして分類される可能性があると述べました。

## Howey Test FAQs

###何かがセキュリティであるかどうかをどのように判断しますか？

米国最高裁判所は、Howey Testを使用して、特定の取引が「投資契約」として適格かどうかを判断します。 1933年証券法および1934年証券取引法に基づき、取引が「投資契約」の対象となる場合、それらの取引は証券と見なされます。

Howey Testは、「他の人の努力から得られる利益が合理的に期待される一般的な企業への投資」があるかどうかを判断しようとします。その場合、取引は、1933年証券法および1934年証券取引法に基づく開示および登録要件の対象となります。

###ビットコインがセキュリティではないのはなぜですか？

2018年6月、元SEC議長のジェイ・クレイトンは、ビットコインはセキュリティではないことを明らかにしました。「暗号通貨：これらはソブリン通貨の代替品であり、ドル、ユーロ、円をビットコインに置き換えます。このタイプの通貨はそうではありません。セキュリティ」と述べた。

技術を開発するために公的資金を求めたことのないビットコインは、証券を分類するためにSECが使用するハウイーテストに合格していません。ただし、クレイトンの定義によれば、ICOで使用されるトークンは有価証券です。

### SECはどのようにセキュリティを定義していますか？

証券は、公的および私的市場で資金を調達するために使用される代替可能で取引可能な金融商品です。証券の公売はSECによって規制されています。

セキュリティオファリングの定義は、**SECv。WJHoweyCo. **と呼ばれる1946年の訴訟で最高裁判所によって確立されました。裁判所は、その判決において、次の4つの基準に基づいてセキュリティの定義を導き出します。

-投資契約の存在

-共通企業の形成

-発行者による利益の約束

-オファリングを宣伝するためのサードパーティの使用

##ハイライト

-「他人の努力から得られる利益を合理的に期待できる一般企業への投資」がある場合、投資契約が存在します。

-Howey Testは、投資家やプロジェクト支援者と一緒にブロックチェーンおよびデジタル通貨プロジェクトを配置するために重要です。

-Howey Testは、何が「投資契約」として適格であり、したがって米国証券法の対象となるかを決定します。

-特定の暗号通貨とイニシャルコインオファリング（ICO）は、HoweyTestの下での「投資契約」の定義を満たすことが判明する場合があります。
