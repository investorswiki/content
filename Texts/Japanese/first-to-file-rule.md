# 先願主義
##先願主義とは何ですか？

先願主義は、文脈に応じて異なる意味を持つ可能性があります。連邦裁判所の訴訟の文脈では、先願主義は、訴訟を提起する最初の当事者が誰であれ、裁判または訴訟の場所について自国の裁判所に与えられると述べています。

[知的財産](/intellectualproperty)と[特許出願](/patent)の文脈では、先願主義は、発明の特許を出願した最初の人が通常、発明を販売する権利を獲得することを保証します。

##先願主義を理解する

先願主義は、文脈に応じて異なる意味を持つ可能性があります。

###先願主義の連邦訴訟

先願主義は、訴訟を提起する最初の当事者が誰であれ、裁判または訴訟の場所について自国の裁判所に与えられると述べています。これは、訴訟当事者が裁判官または裁判所に精通している可能性があり、旅費などの追加費用が発生しないという利点を提供します。訴訟当事者はまた、裁判の期間中、ホテルに滞在するために家を出る必要はありません。先願主義は単なる一般的なルールです。すべての手続きを規定する法律ではなく、例外を設けることができます。

###知的財産と特許出願

特許出願は複雑であり、多くの投資家はそのプロセスを支援するために弁護士を雇っています。同様の創造物について特許がまだ存在していないことを確認するための研究が必要です。発明の特許を最初に出願した人は、受理されたときに特許を受け取ることができます。この人が最初に発明を作成したのではなくても、彼らが最初に出願したので、彼らは特許の所有者です。

この特許により、発明者は作品を販売することができます。特許庁は、特許の存続期間内に製品を販売するための競争力をアプリケーションに持たせることを要求しています。先願主義者が最初に特許を取得する可能性がありますが、その過程で申請が数回拒否される可能性があります。これは、別の申請者が彼らをポストに打ち負かし、特許を付与されていることを意味する可能性があります。

このため、[知的財産の](/intellectualproperty)申請を遅らせることはお勧めできません。最初に提出する規則は、特定の個別の特許を取得するための申請者間の比較的短い猶予期間があり、発明者は、販売のためのアイテムの提供を含む詳細を発明に開示する前に提出する必要があります。これは、模倣者がデザインを使用しないようにするために不可欠です。

##先願主義の弁護士を雇う

法的な絡み合いを回避するため、または先願主義の特許規則を支援するために、発明者は支援のために弁理士を必要とする場合があります。法律専門家は発明者の権利を保護するために働き、弁護士は特許出願プロセスを通じて発明者を支援または支援することができます。

##ハイライト

-先願主義は特許出願にも適用され、最初に特許を出願した人に知的財産を主張する権利を与えます。

-先願主義は、訴訟を起こした最初の当事者が、裁判または訴訟手続きのために自宅の場所を授与されることを主張します。

-発明者は、模倣者が新しい発明やプロセスを奪わないように、迅速に特許を申請することをお勧めします。
