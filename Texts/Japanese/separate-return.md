# セパレートリターン
##個別返品とは何ですか？

申告書は、配偶者に別々に税金を申告している既婚の[納税者](/taxpayer)によって提出されたフォーム1040または同様の税務フォームです。別々のリターンは、一般的に離婚手続きを開始した夫婦、または別々の住居に住んでいる結婚したパートナーに関連付けられています。

##個別の返品について

個別の申告は、連邦税申告者が利用できる5つのオプションの1つです。他の4つは次のとおりです。

- 独身

-共同での結婚申告

-[世帯主](/head_of_household)

-資格のある未亡人または扶養されている子供を持つ未亡人

税務計画の分野では、個別の返品はややまれです。ほとんどの夫婦は、[共同申告書を使用して税金を](/jointreturn)申告することを選択します。そこでは、1セットのフォームに共同で記入し、それらの合計所得を一覧表示し、それらの間で納税義務を共有します。別々のアドレスを持つカップル。

彼または彼女の配偶者と比較して多数の[控除がある夫婦にも利益をもたらす可能性があります。](/deduction)個別の申告はまた、配偶者間で納税義務を分割します。これは、特定のビジネス状況で戦略的に有利な場合があります。

個別の返品を提出することには多くの潜在的な利点がありますが、考慮すべき欠点が考えられます。それらの中で最も重要なもの：別々に提出する納税者は、以下を含む多数の[税額控除](/taxcredit)と控除を放棄します。

-勤労所得税額控除（EITC）

-扶養控除（ほとんどの場合）

-養子縁組クレジット（ほとんどの場合）

-大学の授業料の控除

-高等教育費の生涯学習クレジット

-[アメリカ機会税額控除](/american-opportunity-tax-credit)（AOTC ）

-学生ローンの[利子](/interest)控除

前述の税額控除の権利を放棄することに加えて、別の申告者は、年間を通じて配偶者と同居し、10,000ドル以上を稼いだ場合、ロスIRA拠出を行うオプションも放棄します。

最後に、別々の申告書を提出する両方の配偶者は、控除項目を項目化するか、標準控除を選択することに同意する必要があります。ただし、一方の配偶者は控除項目を項目化できませんが、もう一方の配偶者は標準の控除ルートを使用します。

##個別の返品を提出するその他の理由

税務計画ソフトウェアまたは税務専門家は、夫婦が別々の申告書を提出することにより、税金の支払いが少なくなる特定の状況を特定できる場合があります。たとえば、ある配偶者は、有限責任会社または個人税に流れる別の中小企業の取り決めに関連する項目別の控除の洗濯物リストを持っている場合があります。項目別控除の数が調整された総所得によって制限されている場合、個別のリターンがカップルのお金を全体的に節約することがあります。

>納税者は、連邦税を個別に申告するかどうかを決定する際に、州の納税義務を含む全体像を考慮する必要があります。

>>

また、1人の配偶者が多額の医療費または多額の人的被害を被った場合、またはその年にかなりの慈善寄付を行った場合は、別途申請することもできます。各配偶者が別々の申告書を提出する場合、これら3つのタイプの控除すべての控除額が著しく高くなることがあります。

##ハイライト

-個別の返品には、配偶者間で納税義務を分割する権限があります。これは、特定のビジネス状況では賢明な慣行になる可能性があります。

-別の申告書は、配偶者と共同で納税する既婚納税者が提出する納税申告書です。

-別々のリターンは、離婚に向かう夫婦や、物理的に離れて暮らす夫婦によってよく使用されます。
