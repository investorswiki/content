# ラッセル3000バリューインデックス
##ラッセル3000バリューインデックスとは何ですか？

[ラッセル3000インデックス](/russell_3000)に基づいています。ラッセル3000インデックスは、[バリュー株](/valuestock)のみを含めることで、株式価値セグメントの米国株式のパフォーマンスを測定します。

ラッセル3000バリューインデックスには、株価純資産倍率が低く、予想成長率が低いラッセル3000インデックスの株式が含まれています。この価値指数は、[ラッセル3000成長指数](/russell3000-growth)と対比することができます。

##ラッセル3000バリューインデックスを理解する

ラッセル3000バリューインデックスは、ラッセル3000インデックスに基づいています。ラッセル3000インデックスは、時価総額で定義された、アメリカで設立された3,000の最大の上場企業のパフォーマンスを測定します。

サイズとスタイルで米国市場をセグメント化することは、市場パフォーマンス、ポートフォリオのエクスポージャー、および手数料の高レベルの傾向を理解したい投資家にとって有用な分析ツールです。投資家が幅広い成長と価値特性を示す株式に資金を投入したい場合、ラッセル3000バリューインデックスはこれらの市場セグメントを表示する簡単な方法を提供します。

1995年7月1日に最初に立ち上げられたラッセル3000バリューインデックスの企業は、変動調整後の時価総額に基づいて選択されます。 5月の最終取引日に、FSTE Russellは、時価総額の合計値に基づいて適格企業をランク付けします。 6月の最終金曜日に、インデックスが再構成されます。米国の株式市場指数に含める資格を得るには、証券はNYSE、NYSE MKT、NASDAQ、またはARCA取引所で取引されている必要があります。ラッセル3000バリューインデックスのパフォーマンスは、iShares Core USValueETFシンボルIUSVを介して監視できます。

バリュー株は、会社の業績が示すよりも低い価格での証券取引です。バリュー株への投資家は、原資産の価格が会社の業績と一致しない可能性があるため、市場の非効率性を利用しようとします。

バリューストックの一般的な特徴には、高い[配当利回り](/dividendyield)、低い[株価純資産倍率（P / B比）](/price-to-bookratio) 、および/または低い株価収益率（ [P / E比](/price-earningsratio)）が含まれます。投資家は、「Dogs of the Dow」投資戦略を使用して、毎年初めにダウジョーンズで配当利回りが最も高い10株を購入し、その後毎年ポートフォリオを調整することで、バリュー株を見つけることができます。

バリュー株とは対照的に、成長株は、成長の可能性が高いと予想される企業の株式です。

##ラッセルインデックス

これは、アメリカのパブリックエクイティ市場の約98％を占めています。 2021年1月31日の時点で、ラッセル3000インデックスの株式の加重平均時価総額は約1,250億ドルです。時価総額の中央値は20億ドルです。 1984年1月1日に開始されたこのインデックスは、ロンドン証券取引所グループの子会社である[FTSERussellによって維持されています。](/ftse)

ラッセル3000インデックスは、さらに[ラッセル1000](/russell_1000index)インデックスと[ラッセル2000](/russell2000)インデックスに分類されます。ラッセル3000インデックス内で成長と価値の重みを割り当てるために、ラッセルインベストメントグループは、ラッセル1000インデックスと2000インデックス内の株式をそれぞれランク付けします。株価純資産倍率（B / P）と、[機関投資家の推定システム（IBES）](/ibes)に基づく予測長期成長平均によって、株式はランク付けされます。

ランク付けされると、ラッセルインベストメントグループは非線形確率法を使用して、株式を成長スタイルと価値スタイルに分離します。一般に、ランクの高い株は価値と見なされ、ランクの低い株は成長と見なされます。中間の株には、成長と価値の両方の特徴があります。

##ハイライト
