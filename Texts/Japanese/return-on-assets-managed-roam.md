# 総資産利益率（ROAM）
##総資産利益率（ROAM）とは何ですか？

総資産利益率（ROAM）は、処理された資本のパーセンテージとして示される利益の測定値です。[総資産](/returnonassets)利益率は、営業利益を資産（売掛金や在庫など）で割って算出しています。資産回転率と営業利益率は、総資産利益率の2つの主要な要因です。

ROAMの式は、投資からの総収入x100を管理資産で割ったものです。

ROAMは、総資産利益率（ROA）や投資利益率（ROI）など、よく知られているビジネスの指標に焦点を当てていることはあまりありませんが、正しく適用すると、ビジネスの全体的な健康。

> ROAは、企業が資産に対して十分な利益を生み出すことができるかどうかを決定する優れた財務実績指標です。

>>

##総資産利益率–ROAMの説明

大まかに言えば、ROAMは企業にとって包括的な財務指標であり、市場戦略を反映し、投資家に会社の健全性への窓を提供します。この測定値の年々の変化は、管理下にある資産から利益を生み出す企業の能力の変化を示しています。

この収益を計算する別の方法は、資産回転率に営業利益率を掛けたものです。一部のアナリストは、管理さ[れた純資産の収益を](/rona)使用し、他のアナリストは、管理された総運用資産の収益を使用します。すべての企業を比較するために1つのメトリックまたはバリエーションを使用しないことが重要です。

ROAMは企業によって大きく異なる可能性があり、その企業が属する業界に大きく影響されます。そのため、ROAMを使用している企業を比較する場合は、以前の企業の数値または類似の企業と比較すると最も便利です。無関係な業界の会社とは対照的に、同じ業界。

ROAMは、総資産利益率（ROA）とは異なります。これは、投資された資本からどのような収益が得られたかを判断するために使用される、より一般的に使用される用語です。

##ハイライト

--ROAMは市場戦略を反映しており、投資家に会社の健全性への窓を提供します。

-総資産利益率は、パーセンテージで表された利益測定値です。

-ROAMの2つの主な要因は、資産回転率と営業利益です。

-これは、投資資本から生み出される収益を決定するために使用される総資産利益率とは異なります。
