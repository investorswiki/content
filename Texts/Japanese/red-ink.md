# 赤インク
##レッドインクとは何ですか？

赤インクは、経済的損失を表すビジネス用語です。[会計士が一般的な台帳](/generalledger)[に](/generalledger)物理的に入力する場合、赤のインクは負の数を示すために使用され、[黒の](/black)インクは数が正または有益であることを示すために使用されます。

##赤インクを理解する

多くの文化では、色は一種の言語として機能し、特定の感情や感情を象徴することができます。それらのいくつかは落ち着きと静けさを呼び起こしますが、他のものはより大きくて強力です。

赤は後者のカテゴリーに分類され、エネルギー、情熱、セクシュアリティから攻撃性、危険、暴力、戦争まですべてを意味すると言われています。これらの関連のいくつかはポジティブであり、他はネガティブであると説明することができますが、金融界では、この特定の色の言及または使用は、ほとんどの場合、何か悪いことを意味します。

赤インクは、「赤インクのにじみ」または「赤[で」というビジネス表現の同義語](/red)です。赤が数字とともに使用されている場合、それは一般に、会社、政府、または他のエンティティが財政難に陥っており、持ち込んでいるよりも多くのお金を費やしていることを示します[。](/overdraft)資産よりも負債があり、負債を返済することができない場合も、赤字であると見なすことができます。

赤は、ビジネスで、望ましくないことが起こっていることを示すためにもよく使用されます。たとえば、企業を管理する規制は、通常、否定的にレッドテープと呼ばれます。投資家はまた、株価や[株式市場の下落を](/stockmarket)赤字と呼ぶかもしれません。

>感謝祭の休日の翌日、小売業者が買い物客を引き付けるために商品を大幅に値下げするブラックフライデーは、「赤字」で営業している多くの小売業者が、売り上げが多かったために財政が利益（黒）に変わるのを見て、この名前が付けられました。 。

##レッドインクの歴史

、損失を表すために色を使用することを選択した[会計士](/accountant)の古い習慣に起因する可能性があります。会社の帳簿を手動または手作業で管理する場合、赤と黒のインクは、お金を失う変数や付加価値のある変数に注意を喚起するための便利な方法でした。赤、それはまさにそのように起こり、前者を意味します。

>###重要

>赤と損失の組み合わせは、会計士が赤い色のインクを使用して会社の財務諸表にマイナスの数字を入力するという伝統にまでさかのぼることができます。

>>

その習慣は生き続けています。今日、ほとんどの財務および運用元帳は電子的に保守されていますが、ソフトウェアが赤と黒の色を使用して損失と利益を強調することは珍しくありません。

##特別な考慮事項

アカウントに赤インクが多く含まれている会社は、必ずしも運命にあるとは限りません。[収益性](/profit)[の](/profit)欠如は一時的な問題であり、研究、新技術、または[負債の返済](/debt)に必要な支出に関連している可能性があります。

これらの[費用](/expense)は、[財務諸表](/financial-statements)を醜く見せます。しかし、長期的には、会社とその収益力を強化する必要があります。

投資家は、より良い成長の見通しを追いかけるために発信が適切に使用されているという保証がある場合、赤インクを見落とすことをいとわないことがよくあります。実際、最も人気のある株式の多くは、[負債資本（D / E）比率が高く、将来の](/debtequityratio)[収益](/earnings)を増やすためにリスクを冒すことを恐れない企業の好みを浮き彫りにしています。

会社が常に赤字である場合、線が引かれる可能性があります。持続的な赤字は、株主を失い、新しい[株主を引き付けることができず、](/shareholder)[資金を確保するのに苦労し](/financing)、破産への道をたどる可能性があります。

##ハイライト

-その習慣は、「赤字」という用語への道を開きました。これは、マイナスのバランスを持つあらゆる企業または個人に適用できます。

-会計士が総勘定元帳に物理的に入力する場合、赤インクは負の数を示すために使用され、黒インクは数が正または有益であることを示すために使用されます。

-赤インクは、損失またはマイナスの結果を示すために使用される会計エントリの色を説明する財務用語です。
