# ひもを押す
##ストリングを押すとは何ですか？

ひもを押すことは、金融政策の限界と経済を刺激することに関する中央銀行の無力さの比喩として造られたフレーズでした。

##文字列のプッシュを理解する

弦を押すことは、物事を別の方向よりもある方向に動かすのにより効果的な影響力のスピーチの図です。引くことはできますが、押すことはできません。[金銭的政策](/monetarypolicy)は、企業や家庭が望まない場合に強制的に支出することはできないため、一方向にしか機能しない場合があります。銀行が貸し出すのはリスクが高すぎると考え、経済の不確実性のために民間部門がより多くを節約したい場合、[金銭的基盤](/monetarybase)と[銀行準備金を](/bank-reserve)増やしても経済は刺激されません。

経済学では、ひもを押すことは、具体的には、[拡大的な金融政策](/expansionary_policy)が不況から経済を引き上げるのに効果がない状況を指します。現金を保有する需要は事実上無制限であるため[、お金](/moneysupply)とクレジットの供給への追加は、金融機関、企業、または消費者の現金残高に単純に追加され（または負債を返済するために使用され）、増加にはなりません[総需要](/aggregatedemand)または乗数効果で[。](/aggregatedemand)この状況は[流動性トラップとして知られており](/liquiditytrap)、低金利がどのように押し上げられても、市場は市場参加者の予防的現金保有に無制限の量の新しい流動性を吸収することができます。

[景気回復](/economic-recovery)を刺激するために拡張金融政策を無力にするので、ひもを押すことのアナロジーは、景気後退から経済を引き上げるための主要なツールとして引き継ぐ拡張[財政政策の議論として使用されます。](/fiscalpolicy)類推を拡張するために、資金供給が中央銀行が（失敗して）推進しているストリングである場合、新しいもので総需要を直接押し上げることによってストリングの反対側を「引っ張る」のは財政政策立案者次第です[乗数効果](/multipliereffect)を通じて信頼を回復し、現金残高の私的支出を増やすための政府支出。

「ひもを押す」というフレーズは、英国のエコノミスト、ジョン・メイナード・ケインズに起因することがよくありますが[、](/john_maynard_keynes)[彼](/john_maynard_keynes)がそれを使用したという証拠はありません。しかし、この正確な比喩は、1935年の米国議会の証言で使用されました。マリナーエクルズ連邦予備知事は、T。アランゴールズボロー議員が発した言葉を反映して、経済を刺激し、[大恐慌を終わらせるために連邦政府ができることはほとんどないと述べました](/great_depression)。

##文字列の例をプッシュする

経済を刺激するための初期の努力がほとんど成果を上げていないように見えた2007年から2008年の金融危機の間、文字列の比喩を推し進めることは適切でした。 FRBは[量的緩和](/quantitative-easing)（QE）に数兆ドルを割り当て、[フェデラルファンド金利](/federalfundsrate)をほぼゼロパーセントに引き下げました。

当初、連邦政府は、借金を抱えている世帯が[貯蓄率を高めたため、薄気味悪い空気から需要を生み出すことができないように見えまし](/savings-rate)た。金融政策は必死で無駄に見えたが、米国での資金供給の増加は、[資金速度](/velocity)の低下によって相殺された。したがって、連邦政府はひもを押していました。

家計の負債は2013年まで減少しましたが、2019年末には記録的なレベルの14.15兆ドルに回復しました。一部の人々は、量的緩和と低金利が災害を食い止めることができたと信じています。これらの努力なしで。

##ハイライト

-経済学では、ひもを押すことは、経済にすでに緩みがあり、結果がほとんどまたはまったくないときに、中央銀行が緩い金融政策を制定しようとしていることを説明するために最初に使用されました。

-文字列を押すとは、その特定のコンテキストでは役に立たない場所で努力することを指します。

-ひもを押すことは、景気後退から経済を引き上げるための主要なツールとして引き継ぐための拡大財政政策の議論として使用されるようになりました

-この用語は、1935年に議会の証言で最初に使用されたようですが、経済学者のジョン・メイナード・ケインズに起因するとされています。
