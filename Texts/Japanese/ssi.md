# 補足セキュリティ収入（SSI）
##補足セキュリティ収入（SSI）とは何ですか？

Supplemental Security Income（SSI）は、米国の連邦プログラムであり、高齢者や収入がほとんどまたはまったくない障害者に追加の収入を提供します。このプログラムは、参加者が基本的なニーズを満たすのに役立つ毎月の現金分配を提供します。 SSIは、標準の[社会保障](/social-security-benefits)退職給付とは異なります。

##補足セキュリティ収入（SSI）を理解する

SSIは、年齢や障害のために基本的な経済的ニーズを満たすことができない米国市民または国民のためのセーフティネットです。 SSIの支払いは、毎月1日に到着します。受信者は、フードスタンプと[メディケイド](/medicaid)特典の対象となる場合もあります。

SSIの資格を得るには、人が満たさなければならない正確な要件があります。まず、SSIの候補者は、65歳以上、視覚障害者、または障害を持っている必要があります。第二に、彼らは限られた収入と限られた資源を持ち、米国市民または国民でなければなりません。 2022年現在、SSIは、資産が2,000ドル以下の個人、または3,000ドル以下のカップルのみが利用できます。

最後に、50州のいずれか、コロンビア特別区、または北マリアナ諸島での居住など、他の小さな要件を満たす必要があります。

特別な場合、18歳未満の子供は障害者と見なされ、SSIの資格を得ることができます。子供が資格を得るには、障害が深刻な機能制限をもたらす必要があり、死を引き起こすか、12か月以上続くか、または続くと予想される可能性があります。

##SSIの収益制限

連邦給付率（FBR）は、適格性のSSI所得制限と最大月額SSI支払いの概要を示しています。 FBRは現在、2022年の月額支払いを個人で841ドル、カップルで1,261ドルに設定しています。FBRは、社会保障[の生活費調整と併せて毎年適度に増加し](/cola)、インフレを追跡します。現金援助を受ける人の基本的なケアに不可欠な人への奨学金として、月額421ドルの追加が「不可欠な人」に請求される場合があります。

SSIの資格を得るには、個人またはカップルの合計収入が、FBRで概説されている月額SSI支払いを超えることはできません。ただし、[社会保障局](/ssa)は、個人の収入の一部のみを収入制限にカウントします。たとえば、ある人が仕事でお金を稼いでいる場合、最初の65ドルを超えて毎月稼いだ金額の半分だけが適格性を判断するときにカウントされます。したがって、個人の特定の収入と適格性についてSSAに連絡することが不可欠です。

>子供がSSIの資格を得るには、65歳以上の個人とは異なり、障害が非常に深刻である必要があります。

##特別な考慮事項

ほとんどの州はまた、連邦SSIの支払いにお金を追加します。この追加されたお金は、適格性の許容収入レベルと毎月のSSI支払い額の両方を増加させます。サプリメントの量は州ごとに異なります。

SSAのウェブサイトによると、アリゾナ、ミシシッピ、ノースダコタ、ウェストバージニア、ノーザンマリアナ諸島は州の補足を提供していません。つまり、これらの州の人々は、FBRが概説した連邦の最低額に基づいてのみ資格と支払いを得ることができます。

##ハイライト

-連邦SSIに加えて、多くの州はまた、困窮している人々に補助的な収入を提供しています。

-2022年の場合、SSIは対象となる個人に月額最大841ドル、カップルに最大1,261ドルを支払います。

-限られた収入を持つことを含むSSIの特定の要件があります。

-SSIは、退職した成人または障害者向けの社会保障収入給付とは別のプログラムです。

-Supplemental Security Income（SSI）は、基本的なセーフティネットを提供するために、他の収入がほとんどない高齢者または障害のある市民に追加の収入を提供します。
