# 国際通貨市場
##国際通貨市場とは何ですか？

国際通貨市場は、世界中の参加者がさまざまな通貨を売買する市場です。参加者には、銀行、企業、[中央銀行](/centralbank)、投資管理会社、ヘッジファンド、[小売外国為替ブローカー](/currency-trading-forex-brokers)、および投資家が含まれます。国際通貨市場は、ローン、投資、企業[買収](/acquisition)、グローバル貿易などのグローバルな取引を促進するのに役立つため、重要です。

##国際通貨市場の仕組み

国際通貨市場は世界最大の金融市場であり、1日の平均取引額は5兆ドルです。この市場では、取引は単一の取引所ではなく、世界中の大手銀行やブローカーのグローバルなコンピューターネットワークで発生します。

通貨市場、または[外国為替市場（「外国為替」）](/foreign-exchange-markets)は、外国貿易の結果として必要な通貨の交換を容易にするために作成されました。たとえば、カナダの会社が米国の会社に製品を販売する場合、カナダドルで支払う必要があります。米国企業は、カナダ企業に支払うために、銀行を通じて外国為替転換を促進する必要があります。米国企業の銀行口座は米ドルで引き落とされます。米国の銀行は資金をカナダの会社の銀行に送金します。資金は事前に設定された為替レートでカナダドルに変換され、カナダの会社の口座に入金されます。

グローバル通貨市場は、企業が商品をグローバルに販売し、現地通貨で支払いを受けることができるため、外国貿易を促進するのに役立ちます。給与などの費用は現地通貨であるため、企業は現地通貨で支払う必要があります。

[クリアリングハウス](/clearinghouse)を含まないという点で株式市場とは異なります。取引は、各当事者がその義務を遵守することを保証するために、仲介者なしで当事者間で直接行われます。通貨には単一の価格はありませんが、他の通貨で価格設定されています。

##主要通貨ペア

以下は、相互に最も広く交換されている主要な通貨ペアです。

-EUR / USD：[ユーロ圏](/euro)の[ユーロ](/eurozone)と米ドル

-USD / JPY：米ドル対[日本円](/jpy-japanese-yen)

-GBP / USD：[グレートブリティッシュポンド](/gbp)対米ドル

-USD / CHF：米ドルとスイスのスイス[フラン](/chf)

-USD / CAD：米ドルと[カナダドル](/cad-canadian-dollar)

-AUD / USD：[オーストラリアドル](/aud-australian-dollar)と米ドル

米国は安定した経済と金融システムを持っているため、米ドルは世界の準備通貨と見なされています。多くの製品、商品、投資は米ドルで取引されているため、主要な取引や通貨交換のほとんどが関係しています。市場や為替レートが安定していない国では、投資を呼び込み、貿易を促進するためにドルで取引することを選択する可能性があります。

ただし、グローバルに取引されている他の多くの通貨ペアがあります。中国は元と人民元を通貨として持っていますが、中国との米国の貿易を含む取引のほとんどは米ドルで促進されます。

##セーフヘブン通貨

特定の通貨は、グローバル市場で特定のアイデンティティまたは役割を担っています。たとえば、スイスは政治的および経済的混乱の時代にお金を貯めるのに安全な場所と長い間考えられてきました。厄介な時期に、他のグローバル通貨からスイスフランへの外国為替換算は大幅に増加する傾向があります。

日本は安定した経済と見なされているため、投資の流れにとっても[安全な避難所と見なされています。](/safe-haven)景気後退時には、多くの投資家がドル、ユーロ、ポンド建ての投資を日本政府が保証する[日本国債（国債）と交換します。](/jgb)その結果、不況時には他の主要通貨に対して円高が進む傾向にあります。たとえば、米国の投資家は、ドル建ての[投資信託](/mutualfund)や円建ての日本国債への投資を売却し、その際、外国為替換算により円が対ドルで上昇する可能性があります。

##国際通貨市場のプレーヤー

世界の通貨市場に関与している多くの参加者がいますが、以下は外国為替市場に関与している主要なプレーヤーの一部です。

###企業

は国際的な送金と外国の利益を[ヘッジ](/hedge)するために外国為替市場に参入します。たとえば、メキシコで大規模な事業を展開している米国企業は、ドルとメキシコのペソの間の為替レートを固定するだけの[フォワード契約を結ぶ場合があります。](/forwardcontract)したがって、これらのメキシコの利益を家に持ち帰るときになると、ペソで得られた利益は予期しない為替レートの変動の影響を受けません。代わりに、ペソは事前設定された先物為替レートでドルに変換されます。企業は、為替レートが収益や利益に影響を与えるのを防ぐために、全体的なリスク管理戦略の一部としてフォワードを使用します。

###政府と中央銀行

政府は、輸出や海外売上の増加を支援するために、通貨の価値に影響を与えようとする場合があります。これは[切り下げと呼ばれます。国の資金供給を管理する](/devaluation)国の[中央銀行](/centralbank)は、国の通貨を売るために市場に参入し、価値を押し下げるのに役立つ可能性があります。為替レートが他の主要通貨と比較して下がるとき、国は為替レートだけのために安い輸出を持っていることから利益を得ます。

たとえば、米国と英国のポンド為替レートが2ドルで、投資家が英国で200,000ポンドの住宅を購入したい場合、投資家は400,000ドル（2 * 200,000ポンド）の費用がかかります。英国が為替レートを1.50ドルに引き下げた場合、米国の投資家は同じ不動産を300,000ドル（1.50 * 200,000ポンド）で購入できるようになります。

その結果、英国通貨の切り下げは、外国人投資家からの莫大な買い意欲を引き付け、英国の商品や不動産の需要を押し上げ、英国経済を強化する可能性があります。為替レートの切り下げを行う国は、「通貨[マニピュレーター」と呼ばれることがあります](/manipulation)。
