# クォンツファンド
##クォンツファンドとは何ですか？

クオンツファンドとは、クオンツアナリストでまとめた数値データをもとに有価証券を選定する[投資](/quantitativeanalysis)ファンド[です](/investment-fund)。これらのファンドは、非伝統的で受動的であると見なされます。それらは、投資を決定するためのソフトウェアプログラムを使用してカスタマイズされたモデルで構築されています。

[ファンドマネージャー](/fundmanager)による管理に関連するリスクと損失を削減するのに役立つと信じています。

##クォンツファンドの仕組み

[投資戦略](/investmentstrategy)に依存しています。そのため、彼らは投資決定を行うために人間のマネージャーの経験、判断、または意見を使用しません。[ファンダメンタル分析](/fundamentalanalysis)ではなく定量分析を使用しているため、定量ファンドとも呼ばれます。それらは、資産運用会社がサポートする多くの投資商品の1つであるだけでなく、専門の投資運用会社の中央管理の焦点の一部である可能性もあります。

[ビッグデータ](/big-data)の使用を取り巻くソリューションの数が増えていることは言うまでもなく、幅広い市場データへのアクセスが拡大したことで、クォンツファンドの成長が促進されました。[金融テクノロジーの](/fintech)開発と自動化に関するイノベーションの増加により、クォンツファンドマネージャーが使用できるデータセットが大幅に広がり、シナリオと期間の幅広い分析のためのさらに堅牢なデータフィードが提供されています。

大規模な[資産運用会社は、ファンド運用会社が時間の経過とともに市場](/assetmanagement)[のベンチマークを上回るの](/benchmark)に苦労しているため、定量的戦略への投資を増やすことを目指しています。小規模な[ヘッジファンド](/hedgefund)マネージャーも、投資市場で提供されるクォンツファンドの合計を締めくくります。全体として、クォンツファンドマネージャーは、認定された学位を持ち、数学とプログラミングの高度な技術経験を持つ才能のある個人を求めています。

>定量的戦略は、使用するアルゴリズムを取り巻く機密性が高いため、ブラックボックスと呼ばれることがよくあります。

>>

###クォンツファンドのパフォーマンス

定量的ファンドプログラミングと定量的アルゴリズムには、経済データポイントからトレンドのグローバル[資産価値](/assetvaluation)やリアルタイムの企業ニュースに至るまで、信頼できる何千もの[取引シグナルがあります。](/trade-signal) Quantファンドは、高度なソフトウェアプログラムを通じて開発された独自のアルゴリズムを使用して、勢い、品質、価値、および財務力を中心に高度なモデルを構築することでも知られています。

クォンツファンドは、長年にわたって生み出してきたリターンのために、かなりの関心と投資を集めてきました。ただし、**機関投資家**のレポートによると、 2016年以降は[パフォーマンスが低下しています。2021年までの5年間で、](/underperform) [MSCI](/msci)ワールドインデックスとエクイティクォンツインデックスは年率11.6％のリターンを生み出しました。それぞれ0.88％。

**機関投資家**は、株式クォンツ指数が2010年に10.2％、2011年に15.3％、2012年に8.8％、2013年に14.7％、2014年に10.4％、2015年に9.2％上昇したと主張しました。

###量子戦略の簡単な歴史

定量分析の基礎、したがって定量資金は、**セキュリティ分析**と呼ばれる1934年の本の出版で80年前にさかのぼる歴史があります。ベンジャミン・グラハムとデビッド・ドッドによって書かれたこの本は、特定の[株式](/stock)に関連する客観的な財務[指標](/metrics)の厳密な測定に基づいて投資することを提唱しました。

**セキュリティ分析**の後には、ジョエル・グリーンブラットの**市場を打ち負かす小さな本**やジェームズ・オショーネシーの**ウォールストリートで機能するもの**など、定量的投資戦略に関連する出版物が続きます。

##特別な考慮事項

クォンツファンドは、その管理スタイルが従来のファンドマネージャーとは異なるため、オルタナティブ投資として分類されることがよくあります。

クォンツファンドは、従来のアナリストやポートフォリオマネージャーを必要としないため、通常は低[コストで運用されます。](/costbasis)ただし、証券の回転率が高いため、取引コストは従来のファンドよりも高くなる傾向があります。それらの提供はまた、一般的に標準的なファンドよりも複雑であり、それらのいくつかは、富裕層の投資家を対象としたり、高いファンド入場要件を持っていることが一般的です。

一部の投資家は、クォンツファンドが投資業界で最も革新的で高度に技術的なサービスの1つであると考えています。それらは幅広いテーマ別投資スタイルを網羅し、業界で最も革新的なテクノロジーのいくつかを展開することがよくあります。

成功したクォンツファンドは、モデルの性質上、[リスク管理に注意を払っています。](/risk-control)ほとんどの戦略は、ユニバースまたはベンチマークから始まり、モデルでセクターと業界の重み付けを使用します。これにより、ファンドはモデル自体を損なうことなく、分散をある程度制御することができます。

###クォンツファンド戦略のリスク

[ブラックボックス](/blackbox)に投資を実行させるという概念を採用していないと主張する人もいます。そこにあるすべての成功したクォンツファンドについて、同じように多くが失敗しているようです。残念ながら、クォンツの評判のために、彼らが失敗すると、彼らはしばしば大きな失敗をします。

ロングタームキャピタルマネジメント（LTCM）は、最も尊敬されているアカデミックリーダーとノーベル記念賞を受賞した2人のエコノミスト、マイロンショールズとロバートC.マートンによって運営されていたため、最も有名なクオンツヘッジファンドの[1](/myron-scholes)[つ](/myron-scholes)でし[た](/robert-c-merton)。 1990年代、彼らのチームは平均以上のリターンを生み出し、あらゆるタイプの投資家から資本を集めました。彼らは非効率性を利用するだけでなく、資本への容易なアクセスを使用して市場の方向性に莫大な[レバレッジ](/leverage)の賭けを作成することで有名でした。

彼らの戦略の規律ある性質は、実際に彼らの崩壊につながる弱点を生み出しました。 LTCMは2000年の初めに清算され、解散しました。そのモデルには、ロシア政府が自国の債務の一部を債務[不履行に](/default2)陥れる可能性は含まれていませんでした。この1つのイベントがイベントをトリガーし、レバレッジによって拡大された連鎖反応が大混乱を引き起こしました。 LTCMは他の投資業務に深く関わっていたため、その崩壊は世界市場に影響を及ぼし、劇的な出来事を引き起こしました。結局、[連邦準備制度](/federalreservebank)（FRB）が支援に介入し、他の銀行や投資ファンドは、さらなる被害を防ぐためにLTCMを支援しました。

>クォンツファンドは、主に過去の出来事に基づいており、過去が将来繰り返されるとは限らないため、失敗する可能性があります。

>>

強力なクォンツチームが常にモデルに新しい側面を追加して将来のイベントを予測しますが、毎回将来を予測することは不可能です。経済と市場が平均以上のボラティリティを経験している場合、クォンツファンドも圧倒される可能性があります。売買[シグナル](/sell-signal)は非常に迅速に発生する可能性があるため、高い売上高は高い手数料と[課税対象のイベントを生み出す可能性があります](/taxableevent)。

クォンツファンドは、ベアプルーフとして販売されている場合や短期戦略に基づいている場合にも危険をもたらす可能性があります。[デリバティブを](/derivative)使用して景気後退を予測し、レバレッジを組み合わせるのは危険な場合があります。 1つの間違ったターンは爆縮につながる可能性があり、それはしばしばニュースになります。

##ハイライト

-クオンツファンドは最先端のテクノロジーを利用していますが、クオンツアナリストの使用は新しいものではありません。

-マネージャーは、アルゴリズムとカスタムビルドのコンピューターモデルを利用して投資を選択します。

-クォンツファンドは、高度な数学的モデルと定量分析の使用に基づいて投資決定を行います。

-市場データの入手可能性が高まっているため、投資家はファンド内の定量分析に目を向け、それに固執しています。
