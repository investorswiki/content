# デフォルト率
##デフォルトレートとは何ですか？

デフォルト率は、貸し手が長期間の不払いの後に未払いとして償却したすべての未払いのローンのパーセンテージです。デフォルト率（ペナルティ率とも呼ばれる）という用語は、ローンの定期的な支払いを逃した借り手に課せられるより高い[金利](/interestrate)を指す場合もあります。

支払いが270日遅れた場合、通常、個別のローンはデフォルトで宣言されます。デフォルトのローンは通常、発行者の財務諸表から償却され、回収機関に送金されます。

銀行のローンポートフォリオのデフォルト率は、失業率、インフレ率、消費者信頼指数、個人破産申請のレベル、株式市場のリターンなどの他の指標に加えて、次のように使用されることがあります。経済的健全性の全体的な指標。

##デフォルトレートを理解する

デフォルト率は、貸し手がリスクへのエクスポージャーを決定するために使用する重要な統計的尺度です。銀行のローンポートフォリオのデフォルト率が高いことが判明した場合、[信用リスクを軽減するために、貸付手続きの再評価を余儀なくされる](/creditrisk)可能性があります。借り手がローンの返済を怠ったり、契約を履行しなかったために損失が発生する可能性があります。義務。デフォルトのレートは、経済学者が経済の全体的な健全性を評価するためにも使用されます。

Standard＆Poor's（S＆P）と信用報告機関Experianは共同で、住宅ローン、自動車ローン、消費者など、さまざまな種類の消費者ローンのデフォルト率のレベルで、貸し手とエコノミストが時間の経過に伴う動きを追跡するのに役立つ多数のインデックスを作成します。クレジットカード。これらのインデックスを総称して、S＆P/Experian消費者信用デフォルトインデックスと呼びます。具体的には、これらはインデックスの名前です。S＆P / Experian Consumer Credit Default Composite Index; S＆P / Experian First Mortgage Default Index; S＆P / Experian Second Mortgage Default Index; S＆P / Experian Auto Default Index;およびS＆P / Experian BankcardDefaultIndex。

[住宅ローン](/mortgage)、自動車ローン、銀行のクレジットカードの両方のデータが含まれているため、これらのインデックスの中で最も包括的なものです。 2020年1月の時点で、S＆P / Experian Consumer Credit Default Composite Indexは1.02％のデフォルト率を報告しました。過去5年間で最も高い割合は、2015年2月中旬に1.12％に達したときでした。

銀行のクレジットカードはデフォルト率が最も高い傾向があり、これはS＆P / Experian BankcardDefaultIndexに反映されています。 2020年1月現在のクレジットカードのデフォルト率は3.28％でした。

>デフォルトの記録は、最終的に金額が支払われたとしても、消費者の信用報告書に6年間残ります。

貸し手は、2回目の支払い漏れ期間が経過するまで、支払い漏れについて過度に心配することはありません。借り手が2回連続してローンの支払いを逃した場合（したがって、支払いが60日遅れた場合）、アカウントは[滞納](/delinquent)していると見なされ、貸し手はそれを信用報告機関に報告します。延滞とは、ローンの支払いやその他の種類の債務など、債務に対して支払いを行う契約上の義務を負っている個人が、期限内または定期的かつタイムリーに支払いを行わない状況を表します。

その後、滞納金は借り手の信用格付けにブラックマークとして記録されます。貸し手はまた、支払い遅延のペナルティとして借り手の金利を引き上げることができます。

借り手が支払いを逃し続ける場合、貸し手はローンが償却されてデフォルトであると宣言されるまで延滞を報告し続けます。学生ローンなどの連邦政府資金によるローンの場合、デフォルトの期間は約270日です。他のすべての種類のローンのスケジュールは、州法によって定められています。

あらゆる種類の消費者債務の債務不履行は、借り手の[信用スコア](/credit_score)に損害を与え、将来、信用承認を得ることが困難または不可能になる可能性があります。

2009年のクレジットカードの説明責任、責任、および開示（CARD）法は、クレジットカード市場に新しい規則を作成しました。特に、この法律は、借り手が他の未払いの債務を滞納しているため、貸し手がカード所有者の金利を引き上げることを禁止しています。実際、貸し手は、口座の期限が60日遅れた場合にのみ、より高いデフォルト金利の請求を開始できます。
