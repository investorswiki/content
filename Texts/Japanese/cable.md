# ケーブル
##「ケーブル」とは何ですか？

、米ドル（USD）と英ポンド（GBP）の間の[為替レート](/exchangerate)の俗語です。この用語は[外国為替](/forex)トレーダーの間で使用されます。英国ポンドを指すこともあります。 「ケーブル」という言葉は、ロンドンとニューヨークの間に敷設され、通貨の見積もりやその他のデータを伝達するために使用された初期の電信ケーブルを指します。

ポンド対ドルは最も一般的に取引される通貨ペアの1つであるため、この俗語は頻繁に使用されます。

##ケーブルを理解する

「ケーブル」とは、ドルに対する取引に関連する英国ポンドのみを指し、通常はGBP/USDとして見積もられます。

現在の安値から[回復](/rebound)すると思う」のように、ポンドを英ポンド（ケーブルではない）と呼んでいます。 「」

ポンドの通貨コードはGBPで、英国ポンドを表します。外国為替市場で取引している人が「ケーブルは今日上がっている」または「ケーブルは最近下がっている」と言うのを聞くかもしれません。英国ポンドの記号は£です。

この用語のケーブルは、おそらく19世紀半ばの電信の出現に由来します。当時はポンドが支配的な通貨であり、ポンドとドルの間の取引は大西洋横断ケーブルを介して実行されていました。外国為替トレーダーは「ケーブルディーラー」と呼ばれることもありましたが、このフレーズはもはや一般的に使用されていません。

##第二次世界大戦後の期間までの支配的な通貨

英国ポンド、または英ポンドは、現在も使用されている最も古い通貨と見なされています。それは何世紀にもわたって世界の支配的な通貨であり、したがって他の国々が余剰現金を保有する主要な[準備通貨と見なされていました。](/reservecurrency)

大英帝国が世界の商取引を支配したので、ポンドは国際金融を支配しました。アフリカやアジアの大部分を含むほとんどの植民地で[法定通貨](/legal-tender)でした。第一次世界大戦後、戦争の莫大な経済的コストが経済に打撃を与えたため、帝国は衰退し始めました。

英国政府が米国に対して多額の債務を負っていたため、ドルはポンドが以前に保有していた準備通貨の地位を引き継ぎ始めました。この変更は、英国政府がポンドを30％切り下げることを余儀なくされた1949年までに完了しました。

21世紀初頭までに、ドルは世界をリードする準備通貨となり、ユーロがそれに続きました。[国際通貨基金（ＩＭＦ）に](/imf)よると、ポンドは２０２０年の終わりに４位に落ち着き、日本円に続いた。

##基本通貨

[外国為替で](/foreign-exchange)は、[基本通貨](/basecurrency)は別の通貨と比較される通貨です。ポンドが世界の主要通貨である場合、それは取引の基本通貨でもあったため、価格見積もりはポンドと交換する必要のある通貨Xの金額を示していました。

米ドル、カナダドル（CAD）、日本円（JPY）などとの取引の基本通貨です。したがって、ポンドは通常、GBP / USD、GBP / CAD、およびGBP/JPYとして見積もられます。

しかし、1999年1月1日にユーロ（EUR）が取引を開始したとき、ユーロ（EUR）は、取引されたすべての組み合わせの基本通貨ステータスを引き継ぎました。したがって、ユーロとポンドを比較すると、通常はEUR/GBPとして見積もられます。

逆レート、つまり1米ドル（USD / GBP）を購入するのに必要なGBPを見つけるには、1をGBP/USDレートで割ります。たとえば、GBP / USDレートが1.3050の場合、USD / GBPレートを取得するには、1を1.3050で割って0.76628のレートにします。

##ケーブルが歴史的にどのように動いたかの例

GBP / USDをグラフ化する場合、レートが上昇している場合は、GBPのパフォーマンスがUSDよりも優れているか、USDのパフォーマンスがGBPを下回っていることを意味します。これは、1ポンドを購入するのにますます多くの米ドルがかかるためです。

GBP / USDレートが下がっている場合、それは1つのGBPを購入するのにかかるUSDが少ないことを意味します。したがって、GBPはUSDに比べて価値が下がっています。

##ハイライト

-「ケーブル」は、米ドル（USD）と英ポンド（GBP）の間の為替レートの俗語です。

-2020年現在、GBPは、円、ユーロ、ドル（そもそも）に次ぐ世界第4位の準備通貨です。

-「ケーブル」という用語は、ポンドとドルが最も取引された通貨であった電信時代に由来します。

-米ドル（USD）と英ポンド（GBP）の間の為替レートは、正式にはGBP/USDとして見積もられています。

-英ポンドは主要な準備通貨でしたが、第一次世界大戦後、米ドルに下落し始めました。
