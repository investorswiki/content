# 平均年間収益（AAR）
##平均年間収益（AAR）とは何ですか？

平均年間収益率（AAR）は、投資信託の3年、5年、および10年の[平均収益率などの過去の収益率を報告するときに使用されるパーセンテージです。](/averagereturn)平均年間収益は、ファンドの営業費用比率を差し引いて表示されます。さらに、該当する場合は販売手数料やポートフォリオ取引仲介手数料は含まれていません。

最も簡単に言えば、平均年間収益（AAR）は、特定の期間に投資信託によって稼いだり失ったりしたお金を測定します。投資信託への投資を検討している投資家は、投資信託投資戦略の一環として、AARを確認し、他の同様の投資信託と比較することがよくあります。

##平均年間収益（AAR）を理解する

[投資信託](/mutualfund)を選択する場合、平均年間収益は、ファンドの長期的なパフォーマンスを測定するための有用なガイドです。ただし、投資家は、ファンドの年間[総収益](/totalreturn)の一貫性を十分に理解するために、ファンドの年間パフォーマンスも確認する必要があります。

たとえば、5年間の平均年間収益率10％は魅力的に見えます。ただし、年間収益（平均年間収益を生み出したもの）が+ 40％、+ 30％、-10％、+ 5％、および-15％（50/5 = 10％）の場合、過去3年間のパフォーマンスファンドの管理および投資戦略の検討を保証します。

##平均年間収益（AAR）の構成要素

エクイティミューチュアルファンドの平均年間収益率（AAR）に寄与する要素は、株価の上昇、キャピタルゲイン、配当の3つです。

###株価の上昇

[株価の上昇](/appreciation)は、ポートフォリオに保有されている原株の未実現利益または損失から生じます。株式の株価は1年以上変動するため、発行物の保有を維持しているファンドのAARに比例して貢献または減少します。

たとえば、American Funds AMCAP Fundの最大の持ち株はNetflix（NFLX）であり、これは2月現在のポートフォリオの純資産の3.7％に相当します。 2020年29日。NetflixはAMCAPファンドの199株の1つです。ファンドマネージャーは、ファンドのパフォーマンス目標を達成するために、必要に応じて、ファンドに資産を追加または減算したり、各保有の比率を変更したりできます。ファンドの結合資産は、2月までのポートフォリオの10年間のAARが11.58％に貢献しました。 2020年29日。

###キャピタルゲインの分配

投資信託から支払われる[キャピタルゲイン](/capitalgain)[の分配は、マネージャーが成長ポートフォリオで利益を実現する収入の生成または株式の売却から生じます。](/capitalgain)株主は、分配金を現金で受け取るか、ファンドに再投資するかを選択できます。キャピタルゲインは、AARの実現部分です。株価を支払った金額だけ引き下げる分配は、株主にとって課税対象の利益を表します。

ファンドは負のAARを持ち、それでも課税対象の[分配を行うことができます](/distribution)。ウェルズファーゴディスカバリーファンドは12月に2.59ドルのキャピタルゲインを支払いました。ファンドのAARがマイナス1.48％であるにもかかわらず、2015年11月。

###配当

四半期[配当](/dividend)は、投資信託のAARに貢献し、ポートフォリオの[純資産価値](/nav)（NAV）の価値も低下させます。キャピタルゲインと同様に、ポートフォリオから受け取った配当収入は、再投資するか、現金で受け取ることができます。

[の大型株ファンド](/large-cap)は通常、個人株主および機関投資家に配当を支払います。これらの四半期分配は、投資信託のAARの配当利回りコンポーネントを構成します。 T. Rowe Price Dividend Growth Fundの12か月の利回りは1.36％で、2月までの3年間のAARは15.65％でした。 2020年29日。

##特別な考慮事項

は、通常の平均の代わりに[幾何平均](/geometricmean)を使用する平均年間収益率よりもはるかに簡単です。式は次のとおりです。[（1 + r〜1〜）x（1 + r〜2〜）x（1 + r〜3〜）x ... x（1 + r〜i〜）] ^（1 / n ）^-1、ここで、rは年間収益率、nは期間の年数です。

平均年間収益は、収益が結合するのではなく[複合するため、ファンドのパフォーマンスの全体像を示すのにあまり役立たないと見なされる場合があります。](/compounding)投資家は、投資信託を見て、各ファンドの同じ種類のリターンを比較する際に注意を払う必要があります。

##ハイライト

-平均年間収益率（AAR）は、ミューチュアルファンドの過去の平均収益率を表すパーセンテージであり、通常は3年、5年、および10年にわたって示されます。

-投資信託に投資する前に、投資家は、投資信託の長期的なパフォーマンスを測定する方法として、投資信託の平均年間収益を頻繁に確認します。

-投資信託の平均年間収益に寄与する3つの要素は、株価の上昇、キャピタルゲイン、および配当です。
