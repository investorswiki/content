# 1日あたりのバレル数（B / D）
1859年にペンシルバニア州タイタスビルで石油が発見されたとき、それは利用可能な唯一の大きな容器であるウイスキー樽に積み込まれました。何年にもわたって、業界標準は42ガロンのバレルになりました。これは、石油がバレルの内部を見ることがほとんどなくなったものの、価格設定とポンプの目的で依然として標準的な紙の測定値です。しかし、今日の設備不足の奇妙な予測では、タイタスビルの1859年の石油ブームの最初の不足は、石油を貯蔵するためのバレルでした。

##ハイライト

-その他の石油生産量の測定には、1日あたり100万バレル（MMB / D）、1日あたり1,000バレル（KBPD）、その他のエネルギー製品の場合は1バレルの石油換算（BOE）が含まれます。

-1日あたりのバレル数（B / D）は、1日に生産される石油のバレル数を指します。

-国や企業は、石油の価格を管理する手段として、1日あたりのフルバレル（B / D）容量で石油を生産しないことがよくあります。

-1日あたりの石油とバレルの価格（B / D）には、通常、逆相関があります。

-1日あたりのバレル（B / D）は、グローバルな金額、単一の国、単一の生産フィールドなどを指す場合があります。
