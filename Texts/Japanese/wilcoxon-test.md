# ウィルコクソンテスト
##ウィルコクソンテストとは何ですか？

Wilcoxonテストは、ランク合計テストまたは符号付きランクテストバージョンのいずれかを参照でき、2つのペアのグループを比較するノンパラメトリック統計テストです。テストでは、基本的にペアのセット間の差を計算し、これらの差を分析して、[統計的に互いに](/statistical-significance)[有意に](/statistical-significance)異なるかどうかを確認します。

##ウィルコクソンテストを理解する

[統計](/nonparametric-statistics)家フランクウィルコクソンによって提案されました。これらの検定は、[ランク](/hypothesistesting)付けはできるが、顧客満足度や音楽レビューなどの数値。非パラメトリック分布にはパラメーターがなく、パラメトリック分布のように方程式で定義することはできません。

Wilcoxonテストが答えるのに役立つ質問の種類には、次のようなものがあります。

-同じ生徒の5年生から6年生までのテストのスコアは異なりますか？

-同じ個人でテストした場合、特定の薬は健康に影響を及ぼしますか？

これらのモデルは、データが2つの一致する、または依存する母集団からのものであり、時間または場所を通じて同じ人物または株を追跡していることを前提としています。データは、離散的ではなく連続的であると想定されています。これはノンパラメトリックテストであるため、分析で従属変数の特定の確率分布を必要としません。

##ウィルコクソン検定の種類

-Wilcoxonランク合計テストは、2つの母集団が同じ連続分布を持っている[というヌル仮説をテストするために使用できます。](/null_hypothesis)ヌル仮説は、2つの母集団または変数の間に有意差がないことを示す統計テストです。順位和検定を使用するために必要な基本的な仮定は、データが同じ母集団からのものであり、ペアになっていること、データを少なくとも間隔尺度で測定できること、およびデータがランダムに独立して選択されたことです。

-Wilcoxonの署名付きランクテストでは、ペアの観測値の違いの大きさと兆候に情報があることを前提としています。対応のあるスチューデントのt検定のノンパラメトリックな同等物として、母集団データが[正規分布](/normaldistribution)に従わない場合、符号付き順位を[t検定の代わりに使用できます](/t-test)。

##Wilcoxonテスト統計の計算

ウィルコクソンの符号付き順位検定統計量**W、**に到達するための手順は次のとおりです。

1. ** n **アイテムのサンプル内の各アイテムについて、2つの測定値間の差スコアD〜i〜を取得します（つまり、一方を他方から減算します）。

1.次に、正または負の符号を無視して、一連の** n**絶対差|D〜i〜|を取得します。

1.ゼロの差分スコアを省略し、**n**のゼロ以外の絶対差分スコアのセットを提供します。ここで**n'≤n**です。したがって、**n'**が実際のサンプルサイズになります。

1.次に、ランクR〜i〜を1から** n**までの各|D〜i〜|に割り当てます。最小の絶対差スコアがランク1になり、最大の絶対差スコアがランク**n**になります。 2つ以上の場合|D〜i〜|等しい場合、データに同点が発生しなかった場合に個別に割り当てられたであろうランクの平均ランクがそれぞれ割り当てられます。

1.ここで、D〜i〜が元々正か負かによって、記号「+」または「–」を各** n **ランクR〜i〜に再割り当てします。

1.Wilcoxonテスト統計**W**は、その後、正のランクの合計として取得されます。

実際には、このテストは統計分析ソフトウェアまたはスプレッドシートを使用して実行されます。

##ハイライト

-モデルの両方のバージョンは、データ内のペアが依存する母集団からのものであると想定しています。つまり、同じ人物または株価を時間または場所で追跡します。

-テストの目的は、2つ以上のペアのセットが統計的に有意な方法で互いに異なるかどうかを判断することです。

-Wilcoxonテストは、2つのペアのグループを比較し、ランク合計テストと署名付きランクテストの2つのバージョンがあります。
