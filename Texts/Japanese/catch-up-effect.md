# キャッチアップ効果
##キャッチアップ効果とは何ですか？

キャッチアップ効果は、発展途上の経済は裕福な経済よりも急速に成長する傾向があるという観察により、すべての経済が最終的に[一人当たりの収入の観点から収束するという理論です。](/percapita)言い換えれば、裕福でない経済は文字通り、より堅固な経済に「追いつく」でしょう。キャッチアップ効果は、[収束理論とも呼ばれ](/convergence)ます。

##キャッチアップ効果を理解する

キャッチアップ効果、つまり収束の理論は、いくつかの重要なアイデアに基づいています。

1つは、[限界利益を減少させる法則です。](/lawofdiminishingmarginalreturn)つまり、国が投資して利益を得ると、投資のレベルが上がるにつれて、投資から得られる金額は最終的には減少するという考えです。国が投資するたびに、彼らはその投資からわずかに少ない利益を得る。したがって、資本が豊富な国への[資本投資](/returnoninvestmentcapital)の収益は、発展途上国の場合ほど大きくはありません。

[発展](/developed-economy)途上国よりもゆっくりではあるが安定した速度で成長する傾向があるという経験的観察によって裏付けられています。世界銀行によると、2019年の高所得国の平均[国内総生産](/gdp)（GDP）成長率は1.6％でしたが、中所得国では3.6％、低所得国では4.0％のGDP成長率でした。

発展途上国はまた、先進国の生産方法、技術、制度を複製できるため、より急速な成長を経験できる可能性があります。これは、セカンドムーバーのアドバンテージとしても知られています。発展途上市場は先進国の技術的ノウハウにアクセスできるため、しばしば急速な成長を遂げました。

##キャッチアップ効果の制限

[速い](/economicgrowth)[経済成長](/economicgrowth)を見ることができますが、資本の不足によってもたらされる制限は、発展途上国の追いつく能力を大幅に低下させる可能性があります。歴史的に、一部の開発途上国は、経済的[生産性を効率的に高めるために資源を管理し、資本を確保することに非常に成功してきました](/productivity)。しかし、これは世界規模での標準にはなりませんでした。

エコノミストのモーゼス・アブラモウィッツは、キャッチアップ効果の限界について書いています。彼は、各国がキャッチアップ効果の恩恵を受けるためには、彼が「社会的能力」と呼んだものを開発し活用する必要があると述べた。これらには、新技術を吸収し、資本を引き付け、グローバル市場に参加する能力が含まれます。これは、技術が自由に取引されていない場合、または法外に高価である場合、キャッチアップ効果が発生しないことを意味します。

特に国際貿易に関して、質の高い制度の採用も役割を果たしています。経済学者のジェフリー・サックスとアンドリュー・ワーナーによる縦断研究によると、[自由貿易](/free-trade)と開放性に関する国家経済政策は、より急速な成長と関連しています。 1970年から1989年までの111か国を調査したところ、先進国の一人当たりの成長率は年間2.3％であり、開放貿易政策のある発展途上国の成長率は4.5％であり、[保護貿易主義](/protectionism)と閉鎖経済政策のある発展途上国の成長率は成長率はわずか2％でした。

キャッチアップ効果のもう1つの大きな障害は、1人当たりの所得がGDPの関数であるだけでなく、国の人口増加の関数でもあることです。発展途上国は、先進国よりも人口増加率が高い傾向があります。 2019年の世界銀行の数値によると、より先進国（ [OECD](/oecd)加盟国）の平均人口増加率は0.5％でしたが、国連に分類された後発開発途上国の平均人口増加率は2.3％でした。

##キャッチアップ効果の例

1911年から1940年の間に、日本は世界で最も急速に成長した[経済でした](/economy)。それは植民地化し、隣国の韓国と台湾に多額の投資を行い、彼らの経済成長にも貢献しました。しかし、第二次世界大戦後、日本経済はボロボロになりました。

国は1950年代に経済成長のための持続可能な環境を再構築し、米国から機械と技術の輸入を開始しました。それは1960年から1980年代初頭までの期間に信じられないほどの成長率を記録しました。

日本経済が前向きになったとしても、日本のインフラ・産業基盤の多くの源泉であった米国経済は活況を呈しました。その後、1970年代後半までに、日本経済が世界のトップ5にランクインしたとき、その成長率は鈍化していました。

である[アジア四小龍の経済](/four-asian-tigers)は、同様の軌道をたどり、開発の最初の数年間は急速な経済成長を示し、その後、より穏やかな（そして低下する）成長率を示しました経済が発展段階から発展段階に移行するにつれて。

##ハイライト

-開発途上国は、経済を自由貿易に開放し、「社会的能力」、つまり新技術を吸収し、資本を引き付け、世界市場に参加する能力を開発することにより、キャッチアップ効果を高めることができます。

-これは、国レベルでの投資に適用される限界収穫逓減の法則と、経済が成熟するにつれて成長率が低下する傾向があるという経験的観察に基づいています。

-キャッチアップ効果は、発展途上国が一人当たりの所得の観点からより先進国に追いつくという理論です。
