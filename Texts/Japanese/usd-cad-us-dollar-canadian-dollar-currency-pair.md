# USD / CAD（米ドル/カナダドル）
## USD / CAD（米ドル/カナダドル）とは何ですか？

USD / CADは、米ドルとカナダドル（USD / CAD）の[通貨ペアの略語です](/currencypair)。 USD / CAD通貨ペアに指定された見積もりは、1米ドル（[基本通貨）](/basecurrency)を購入するために必要なカナダドル（[見積もり通貨）の数を読者に示し](/quotecurrency)ます。

[ルーニー](/loonie)」の取引としても知られています。これは、その名を冠した鳥を表しています。 USD / CADは、外国為替市場で最も流動的で活発に取引されているペアの1つでもあります。

## USD/CAD通貨ペアを理解する

USD / CADペアの価値は、**X**カナダドルあたり1米ドルとして見積もられています。たとえば、ペアが1.20で取引されている場合、1米ドルを購入するには1.2カナダドルかかることを意味します。あるいは、1CADは0.833米ドルの価値があります。

USD / CAD通貨ペアは、歴史のさまざまな時点（つまり、1：1）で[同等に達しましたが、従来、米ドルは2つの通貨の中でより強いものでした。](/parityprice) USD / CAD通貨ペアは、両国間に重要なビジネス関係があるため、非常に活発に取引されています。

## USD/CAD通貨ペアに影響を与える要因

USD / CADは、米ドルおよび/またはカナダドルの相互および他の通貨との関係に影響を与える要因の影響を受けます。このため、連邦準備制度（Fed）と[カナダ銀行（BoC）の](/bankofcanada)[金利差は](/interest-rate-differential)、これらの通貨を相互に比較した場合の価値に影響を与えます。たとえば、FRBが米ドルを強くするために[公開市場活動に](/open-market)介入すると、より強い米ドルを購入するためにより多くのカナダドルが必要になるため、USD/CADクロスの価値が高まります。

カナダドルの価値は、商品の価格、特に原油の価格とも高い相関関係があります。カナダの経済は石油に大きく依存しているため、石油の価格が経済の状態と通貨自体を決定します。このため、カナダドルはコモディティ通貨としてラベル付けされることがよくあります。

## USD/CADとパリティ

前述のように、USD / CADペアは、その従来の関係が価格の同等性に打撃を与えてきました。たとえば、[大不況](/great-recession)とそれに続く米連邦準備制度理事会の量的緩和の余波で、カナダドルは米ドルに対して急騰し、平等を下回って取引され、最終的には0.95に達しました。実際、パリティのほとんどすべての事例は、米国の財政難または石油価格の高騰の時期に関連しており、時にはその両方に関連しています。

しかし、2016年には、石油価格は10年ぶりの安値まで下落し、1バレルあたり30ドルを下回りました。その結果、カナダドルは過去最低を記録し、1.46まで取引されました。これは、1米ドルを購入するのに1.46カナダドルが必要であることを意味しました。 2021年8月の時点で、USD/CADレートは約1.25です。

##ハイライト

-USDは、大不況の直後に1：1のパリティに達したものの、通常、時間の経過とともにCADよりも強くなっています。

-この見積もりは、1米ドルで購入できるCADの数を示しています。

-USD / CADは、米ドルとカナダドルの通貨ペアです。
