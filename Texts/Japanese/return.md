# 戻る
##リターンとは何ですか？

収益は、最も簡単な用語では、財務収益とも呼ばれ、一定期間の投資で得られた、または失われたお金です。

リターンは、名目上、投資のドル価値の経時変化として表すことができます。収益は、投資に対する利益の比率から導き出されたパーセンテージとして表すこともできます。収益は、純結果（手数料、税金、[インフレ後](/inflation)）または価格変動以外の何も考慮しない総収益として表示することもできます。 [401（k）への](/401kplan)投資も含まれています。

##返品について

賢明な投資家は、リターンの正確な定義は状況に応じて行われ、それを測定するために入力された財務データに依存することを知っています。利益のようなオムニバス用語は、グロス、営業、ネット、税引前、または税引後を意味する場合があります。投資のような総称は、選択された、平均的な、または総資産を意味する場合があります。

[保有期間収益率は、特定の投資家が所有して](/holdingperiodreturn-yield)いる期間にわたる投資の収益率です。保有期間収益率は、名目上またはパーセンテージで表すことができます。パーセンテージで表す場合、よく使用される用語は[収益率](/rateofreturn)（RoR）です。

たとえば、1か月の定期的な間隔で得られる収益は月次の収益であり、1年の収益は年次の収益です。多くの場合、人々は投資の年間収益、または今日から1年前の同じ日付までの価格変化を計算する前年比（YoY）収益に関心があります。

異なる長さの周期的な間隔での返品は、同じ長さの間隔に変換された場合にのみ比較できます。 1年の間隔で得られた収益を比較するのが通例です。より短いまたはより長い返品間隔を年間返品に変換するプロセスは、年次化と呼ばれます。

##公称リターン

[名目収益率](/nominal-rate-of-return)は、税金、手数料、配当、インフレ、またはその他の金額への影響を調整する前の、ドル（またはその他の該当する通貨）の金額で表される投資の純利益または純損失です。これは、指定された期間における投資の価値の変化に、分配から支出を差し引いたものを計算することによって計算できます。

投資家が受け取る分配は、投資またはベンチャーの種類によって異なりますが、[配当](/dividend)、利子、家賃、権利、利益、または投資家が受け取るその他のキャッシュフローが含まれる場合があります。投資家が支払う支出は、投資またはベンチャーの種類によって異なりますが、投資を取得、維持、および売却するために投資家が支払う税金、費用、手数料、または支出が含まれる場合があります。

>プラスのリターンは、投資またはベンチャーからの利益または収益です。同様に、マイナスのリターンは、損失、または投資やベンチャーで失われたお金を表します。

>>

たとえば、投資家が1,000ドル相当の上場株式を購入し、分配金を受け取らず、支出を支払わず、2年後に1,200ドルで株式を売却するとします。ドルでの名目収益は$1,200-$1,000 =$200です。

##リアルリターン

実際の収益率は、インフレやその他の外部要因による価格の変化に合わせて調整されます。この方法は、名目収益率を実質的に表現し、一定レベルの資本の[購買力](/purchasingpower)を長期にわたって一定に保ちます。

インフレなどの要因を補うために名目リターンを調整することで、名目リターンのどれだけが実際のリターンであるかを判断できます。あなたのお金を投資する前に、投資の実際の収益率を知ることは非常に重要です。それは、税金もそれに欠けるのと同じように、インフレは時間の経過とともに価値を下げる可能性があるためです。

>株式の総収益には、キャピタルゲイン/ロスと配当収入の両方が含まれますが、株式の名目収益は、その価格変動のみを示します。

>>

投資家はまた、特定の投資に伴うリスクが、実際の収益率を考慮して許容できるものであるかどうかを検討する必要があります。特に高インフレの期間中は、名目値ではなく実際の値で収益率を表すことで、投資の価値をより明確に把握できます。

##収益率

、投資がどの程度効果的に管理されているかを測定する[財務比率](/ratioanalysis)のサブセットです。それらは、投資に対して可能な限り最高の利益が生み出されているかどうかを評価するのに役立ちます。一般に、収益率は、資産や株式への投資など、利益を生み出すために利用できるツールを純利益と比較します。

収益率は、選択された資産または総資産または資本を純利益に分割することによってこの比較を行います。結果は、投資された1ドルあたりの収益率であり、類似の投資、企業、業界、または市場の収益率などのベンチマークと比較することにより、投資の強さを評価するために使用できます。たとえば[、資本利益率](/returnofcapital)（ROC）は、元の投資の回収を意味します。

###投資収益率（ROI）

収益率は、パーセンテージで表された収益率です。これは[、](/returnoninvestment)[投資](/returnoninvestment)収益率（ROI）として知られています。 ROIは、投資した1ドルあたりの収益です。 ROIは、ドルのリターンを初期のドル投資で割って計算されます。この比率に100を掛けて、パーセンテージを取得します。 1,000ドルの投資に対して200ドルの収益を想定すると、収益率またはROI =（$ 200 / $ 1,000）x 100 = 20％になります。

###自己資本利益率（ROE）

[自己資本](/returnonequity)利益率（ROE）は、純利益を平均株主資本で割って計算される収益率であり、株式投資1ドルあたりの純利益を測定します。会社がその年の純収入で$10,000を稼ぎ、同じ期間の会社の平均自己資本が$ 100,000である場合、ROEは10％です。

###総資産利益率（ROA）

[資産](/returnonassets)[利益](/returnonassets)率（ROA）は、純利益を平均総資産で割って計算される収益率であり、資産に投資された1ドルあたりの純利益がどれだけ発生するかを測定します。これは、財務レバレッジと[、資本コスト](/costofcapital)をカバーするのに十分な資産使用から得られるかどうかを決定します。純利益を平均総資産で割ると、ROAになります。

たとえば、その年の純収入が10,000ドルで、同じ期間の会社の平均資産の合計が100,000ドルに等しい場合、ROAは10,000ドルを100,000ドルで割った値（10％）になります。

##ハイライト

-株式のトータルリターンには、価格変動、配当金、利息の支払いが含まれます。

-実質収益率はインフレやその他の外部要因の影響を考慮していますが、名目収益率は価格変動のみに関心があります。

-プラスのリターンは利益を表し、マイナスのリターンは損失を表します。

-収益とは、資産、投資、またはプロジェクトの価格の経時変化であり、価格の変化またはパーセンテージの変化で表される場合があります。

-収益は、比較のために年換算されることがよくありますが、保有期間収益は、投資が保有された全期間の利益または損失を計算します。

-ファンダメンタル分析で使用するために、いくつかのリターン比率が存在します。
