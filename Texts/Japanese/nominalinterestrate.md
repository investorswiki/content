# 名目金利
##名目金利とは何ですか？

名目金利とは、インフレを考慮した前の金利のことです。[名目](/nominal)は、手数料や利息の複利を考慮せずに、ローンの宣伝または表示された金利を指すこともあります。

>###主要なTakeawys

>-名目金利とは、インフレを考慮に入れる前の金利を指します。

>-連邦準備制度によって設定された金利であるフェデラルファンド金利は、銀行や金融機関によって請求される他の金利の基礎となる短期名目金利です。

>-インフレによる購買力の低下を回避するために、投資家は名目金利ではなく実質金利を考慮します。

>-APY（年利）は実効金利であり、名目または記載された金利よりも借り手と貸し手に関連する傾向があります。

##名目金利を理解する

中央銀行は、銀行や金融機関が請求する他の金利の基礎となる短期名目金利を設定します。名目金利は、大不況後、実質金利の低さを通じて経済活動を刺激するために人為的に低水準に抑えられる可能性があり、これにより消費者はローンを組んでお金を使うようになります。しかし、そのような刺激策の必要条件は、インフレが現在または短期的な脅威であってはならないということです。米国では、連邦準備制度によって設定された金利であるフェデラルファンド金利は、名目金利と呼ばれることもあります。

逆に、インフレ期には、中央銀行は名目金利を高く設定する傾向があります。残念ながら、彼らはインフレ水準を過大評価し、名目金利を高すぎる状態に保つ可能性があります。結果として生じる金利の上昇は、支出を停滞させる傾向があるため、深刻な経済的影響を与える可能性があります。

>名目金利は、高インフレの時期に高くなる傾向があります。

>>

名目金利は、実質金利や実効金利とは対照的に存在します。実質金利は投資家と貸し手にとって重要である傾向がありますが、実効金利は借り手だけでなく投資家と貸し手にとっても重要です。

名目金利はローンに関連する表示金利ですが、通常、消費者が支払う金利ではありません。むしろ、消費者は料金と複利の効果に基づいて変化する実効レートを支払います。そのために、年利率（APR）は、手数料を考慮に入れるため、名目金利とは異なり、年利回り（APY）は、手数料と複利の両方を考慮に入れます。

実効金利がわかっている特定の期間の名目金利（n）は、次のように計算できます。

n = m×[（1 + e）^ 1 / m ^-1]

どこ：

--e=実効レート

--m=複利計算期間の数

ただし、名目金利は記載されている金利であることが多いため、ほとんどの借り手は通常、実効金利を知りたいと考えています。実効金利（e）の式は次のとおりです。

e =（1 + n / m）^ m ^-1

どこ：

--n=名目金利

--m=複利計算期間の数

たとえば、ローンの記載された（名目）利率が8％で、半年ごとに複利計算される場合、実効利率（e）は次のようになります。

e = [1 + .08 / 2] ^ 2 ^ --1 = 8.16％

> Treasury Inflation Protected Securities（TIPS）により、投資家はインフレの価値を失うことなく貯蓄を維持することができます。

>>

##名目vs。実質金利

名目金利とは異なり、実質金利はインフレ率を考慮に入れています。名目金利と実質金利を結ぶ方程式は、名目金利=実質金利+インフレ率、または名目金利-インフレ率=実質金利として概算できます。

インフレによる購買力の低下を回避するために、投資家は名目金利ではなく実質金利を考慮します。米国の実質収益率を見積もる1つの方法は、財務省インフレ保護証券（TIPS）の金利を観察することです。[国債の利回りと同じ満期のTIPSの](/treasurybond)利回りの差は、経済におけるインフレ期待の推定値を提供します。

たとえば、3年間の預金で提供される名目金利が4％で、この期間のインフレ率が3％の場合、投資家の実質収益率は1％です。一方、年率3％のインフレ環境で名目金利が2％の場合、投資家の購買力は年率1％低下します。

###なぜ投資家は実質金利をもっと気にするのですか？

インフレによる購買力の低下を回避するために、投資家は名目金利ではなく実質金利を考慮します。米国の実質収益率を見積もる1つの方法は、財務省インフレ保護証券（TIPS）の金利を観察することです。国債の利回りと同じ満期のTIPSの利回りの差は、経済におけるインフレ期待の推定値を提供します。

###名目金利がわかっている場合、実効金利をどのように計算しますか？

調合を考慮した実効利率は、名目利率から計算できます。名目利率は、多くの場合、記載されている利率です。実効金利（e）の式は次のとおりです。e=（1 + n / m）^ m -1（ここで、n =名目金利、m =複利期間の数）

###名目金利とAPYの違いは何ですか？

APY（[年利](/apy)）は実効金利であり、借り手と貸し手により関連性が高い傾向があります。消費者、通常は借り手は、手数料と複利の効果に基づいて、名目（記載された）レートとは異なる実効レートを支払います。そのために、実効金利（APY）は名目金利よりも高いことがよくあります。

###名目金利と実質金利の違いは何ですか？

名目金利はインフレを考慮していませんが、実質金利は考慮しています。たとえば、米国では、連邦準備制度によって設定された金利であるフェデラルファンド金利が、提供される名目金利の基礎を形成することができます。ただし、実際の金利は名目金利からインフレ率を差し引いたものであり、通常はCPI（消費者物価指数）で測定されます。
