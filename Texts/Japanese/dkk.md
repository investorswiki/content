# DKK（デンマーククローネ）
## DKK（デンマーククローネ）とは何ですか？

DKKは、デンマークの**クローネ**の外国為替（外国為替）市場における通貨コードです。クローネは、デンマーク、グリーンランド、フェロー諸島の公式通貨です。クローネは100鉱石に細分化され、現在はユーロに[固定されています。](/currency-peg)現地通貨は、kr100のように「kr」で表され、100クローネ（複数形）を意味します。

## DKK（デンマーククローネ）を理解する

1619年に最初に導入されたデンマーククローネは、「クラウン」がクローネの直訳であるため、デンマークの王冠に敬意を表して鋳造されました。このとき、デンマークは以前のデンマークのリグダラーを公式通貨としてデンマーククローネに置き換え、クローネを[金本位制に結び付けました](/goldstandard)。

現代の国際貿易が成長するにつれて、紙幣の需要が高まりました。デンマークは、スウェーデンおよびノルウェーとのスカンジナビア通貨連合への参加の解消の一環として、2番目のデンマーククローネを作成しました。組合は1914年に解散し、参加した3か国は通貨を維持することを選択しました。クローネは、ドイツのライヒスマークマークに簡単に固定され、次に[英国のポンド](/gbp)に固定され、その後ドイツの[ドイツマークに固定されました](/deutschmark)。

DKKは現在、1ユーロあたり7.46 DKKでユーロに固定されており、それに対して+/- 2.25％の小さな[帯域で変動することが許可されています。](/currency-band) 2009年から2019年の間、EUR / DKKレートは主に7.45から7.47の間にとどまり、移動が許可されている2.25％のしきい値内に十分収まっています。

現在流通している紙幣の種類には、50、100、200、500、1,000クローネが含まれます。コインには、ハーフクローネ、1、2、5、10、および20クローネが含まれます。

##デンマークとユーロ

[欧州連合（EU）の前身コミュニティである欧州経済コミュニティ](/europeanunion)のメンバーになることを申請しましたが、 1973年まで参加しませんでした。欧州コミュニティへのデンマークの長期参加にもかかわらず、デンマークの人々は欧州コミュニティへの統合の増加に関連するレファレンダムの失敗の結果として、現在の欧州連合に比較的懐疑的であるという評判。 2000年に、人口はクローネをユーロに置き換えないことに投票しました。

が欧州連合の憲法上の基礎を形成する2つの条約の1つである[マーストリヒト条約](/maastricht-treaty)に関する国民投票を拒否した1992年に行われました。 1992年の国民投票が失敗した後、条約の改正はデンマークの懸念を満たし、1993年に2回目の国民投票で批准されました。

しかし、ユーロがEUに加盟した後も、デンマークではユーロに対する懐疑論が続いています。 2000年9月28日、53.2％のデンマークの有権者が再びユーロを採用しないことを決定しました。

##国際経済におけるデンマーククローネ

デンマーククローネは、 EU諸国間の為替レートの変動を減らすことを目的とした[欧州為替相場メカニズムの一部です。](/exchange-rate-mechanism) 1ユーロに対して7.46クローネの比率でユーロにペッグし、そのレベルの2.25％以内にとどまる必要があります。

デンマークの通貨への愛着にもかかわらず、デンマーク国立銀行は[欧州中央銀行の](/europeancentralbank)政策を非常に綿密に追跡しているため、独立した分析では、デンマークは事実上のユーロのユーザーであると主張しています。元デンマーク首相のアンダース・フォグ・ラスムッセンは、この地位を、意思決定の席を持たずにユーロ圏の一部であると特徴づけています。

世界銀行のデータによると、デンマークは年間インフレ率が0.7％で、2019年の[国内総生産](/gdp)（GDP）は2.85％増加しました。デンマークは輸入を上回り、機械、化学薬品、医薬品などの主要な輸出を行っています。

##外国為替市場におけるDKKの例

レートが7.4725の場合、1ユーロを購入するのに7.4725krかかることを意味します。 1クローネを購入するのに必要なユーロ数を調べるには、1ユーロをEUR/DKKレートで割ります。これにより、0.1338のDKK/EURレートが提供されます。つまり、1つのクローネを購入するのに0.1338ユーロかかるということです。これらのレートは、通貨を交換する場合に役立ちます。

たとえば、1,000ユーロをクローネに換算すると、その人は7,4725クローネ（7.4725 x 1,000）を受け取ることになります。現金（デジタルまたは物理的）を変換する場合、銀行または[外貨両替所](/currency-exchange)は通常、変換に手数料を請求します。これらは1％から5％の範囲です。したがって、実際に受け取る通貨の金額は、たとえば3％削減できます。

DKKはユーロに固定されていますが、他の通貨に対して変動します。したがって、ユーロ以外の通貨ではボラティリティが高くなります。オーストラリアドル対クローネであるAUD/DKKのレートが4.47である場合、オーストラリアドルあたり4.47クローネの費用がかかることを意味します。レートが4.62に上昇した場合、1豪ドルを購入するのに4.62の費用がかかるため、クローネの価値が失われたことを意味します。レートが4.39に下がると、クローネの価値が上がります。これは、1豪ドルを購入するのにかかるクローネのコストが少なくなるためです。

##ハイライト

-クローネは1600年代初頭からデンマークに何らかの形で存在しており、今日では7.46のレートでユーロに固定されており、そのレベルの2.25％の範囲内にとどまる必要があります。

-デンマーククローネ（DKK）は、デンマーク、グリーンランド、およびフェロー諸島の公式通貨です。

-デンマークは欧州連合の一部ですが、独自の独立通貨を保持することを選択しました。
