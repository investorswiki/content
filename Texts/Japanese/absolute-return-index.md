# 絶対リターンインデックス
##絶対リターンインデックスとは何ですか？

[ヘッジファンド](/hedgefund)への投資の[絶対リターン](/absolutereturn)を測定するために設計された株価指数を指します。ヘッジファンドは、プールされたファンドを使用して投資家の利益を得る代替投資と見なされます。このインデックスは、個々のヘッジファンドのパフォーマンスをヘッジファンド市場全体と比較するために作成されました。これは、他の5つのインデックスで構成される複合インデックスです。

##絶対リターンインデックスの仕組み

ヘッジファンドは、さまざまな投資家からの資金を一緒にプールするオルタナティブ投資です。これらの投資は通常、よりリスクの高い[投資戦略](/investmentstrategy)と採用する投資により、損失の可能性が高くなります。彼らはより大きな初期投資を必要とするため、平均的な投資家ではなく、[認定投資家を対象としています。](/accreditedinvestor)これらのファンドは、投資に絶対リターンアプローチを使用しています。

一部のヘッジファンドには、成功と見なされるために達成しようとしている[ベンチマーク](/benchmark)または[パフォーマンス基準があります。](/absolute-performance-standard)他の人は彼らが提供することを目指す設定された率を持っているでしょう。たとえば、12か月間の20％のリターンは、[不動産に多額の投資を行うファンドにとって非常に成功していると見なされる場合があります](/realestate)。この同じリターンは、外貨に投資するファンドにとっては幸運ではありません。

と。絶対リターンインデックス、投資家は、特定の投資が[市場全体と](/market)重なるため、特定の投資の成功または失敗を簡単に識別でき、各トランザクションの複雑な詳細をふるいにかける必要はありません。絶対リターンインデックスは、有用なだけです。ヘッジファンド市場を扱うとき。投資信託、株式、債券などの他の市場には、商品を比較して利益とリスクを判断するための個別のメトリックシステムがあります。

投資家は、絶対リターンインデックスを使用して、あるヘッジファンドを別のヘッジファンドと比較することもできます。ヘッジファンドの性質、およびそれらが[利益を得るために使用するさまざまな戦略のために](/profit)、成功はファンドごとに異なって見える可能性があります。ヘッジファンドはほぼすべてに投資できるため、[収益](/earnings)に基づいてファンドを比較する機能はさらに複雑になります。

>ヘッジファンドの投資家は、ヘッジファンドの絶対リターンインデックスを使用して、投資のリターンを測定できます。

##絶対リターンインデックスとその他の要因

投資家は、ファンドを比較するためにインデックスの外で考慮することができる他の要因を持っています。これらには、投資に利用可能なファンド[資本の額](/capital)およびヘッジファンドの管理に関連するコストが含まれます。[ヘッジファンドマネージャー](/hedge-fund-manager)は、これらのさまざまな[指標](/metrics)を確認し、投資に最適な市場について投資家にアドバイスすることができます。長期的な目標と利用可能な資本によっては、ヘッジファンドがすべての人にとって最良の投資ではない場合があります。

[リスクプロファイルは、投資家が1つのファンドを別のファンドと比較したい場合がある別](/risk-profile)の場所です。一部の投資ファンドにはさまざまなリスクの層があります。 1つのファンドの[投資](/returnoninvestment)収益率（ROI）がより高い場合でも、リスクは高くなる可能性があります。この追加のリスクは、投資が報われると利益が高くなるため有益ですが、そうでない場合、投資家はより高い損失を被る可能性があります。

ヘッジファンドの多くの複雑さのために、投資家が考慮すべき追加のリスクがあります。これらのリスクには、資金を長期間拘束する必要があること、[分散がほとんどない単一のベンチャーに多額の資本が投入されること](/diversification)、および借入金の使用が含まれます。これらはすべてリスクを高めますが、報酬も増やす可能性があります。
