# 付加価値
##付加価値とは何ですか？

付加価値は、[債券の](/bond)現在の価値であり、多くの場合、債券が満期になるまで支払われない場合でも、未収[利息](/interestrate)を含め、[貸借対照表の目的で計算されます。](/balancesheet)

##付加価値を理解する

付加価値とは、任意の時点で、利息が発生するが満期までその利息を支払わない複数年の商品の価値[です](/accrue)。最もよく知られているアプリケーションには、[ゼロクーポン](/zero-couponbond)債または[累積優先株式が含まれます](/cumulative_preferred_stock)。

債券の付加価値は、その市場価値とは何の関係もない場合があります。たとえば、最終満期が100ドルの10年10％のゼロクーポン債は、2年目にはおそらく43.60ドルの付加価値があります。現在の市場金利が下がると、その債券の[公正市場価値](/fairmarketvalue)はその付加価値よりも高くなり、金利が上がると、その債券の価値はその付加価値よりも低くなります。

付加価値は、特定の時点で売却された場合の債券の理論価格として概念化できます（そして、市場金利は満期まで最新のレベルのままでした）。付加価値は、資本増価債の加重平均を決定する要因でもあります。

>ゼロクーポン債の付加価値は、債券の市場価値よりも高い場合も低い場合もあります。これは、付加価値が発行価格から償還可能価格への線形外挿であるためです。

##付加価値と債券価格

付加価値を評価する際には、さまざまな要素を考慮することができます。これは、債券および関連要素の新規株式公開の価格に関連しています。これには、新規株式公開が行われたときの最初の購入者の投資と、新規株式公開でのその取得に基づく最新の未収利息が含まれます。

債券の価値は、債券の期間にわたって毎日の増加が見られる線形軌道に従ってエスカレートする必要があります。ゼロクーポン債が蓄積する利息は、自動的に再投資されると見なされます。任意の日に債券に割り当てることができる数学的値があり、それがその付加価値になります。これは、累積値として表される場合もあります。

債券の市場価値と付加価値の間に差異がある場合があります。これは、償還時の価格と比較して発行されたときの価格に基づく数学的予測によるものです。

たとえば、ゼロクーポン債を90ドルで購入した場合、1,000日後に100ドルに引き換えられる可能性があります。債券が満期を迎えるにつれ、その価値は1日1セントの割合で増加します。その期間の途中で、債券の付加価値は95ドルになります。その価格は、需要と供給の変動により、当時の債券の市場価値とは相関関係がない可能性があります。債券の入手可能性は、発行者の信用力によっても影響を受ける可能性があります。

##債券増加の会計処理

債券の増加を考慮する場合、2つの主要な方法があります。定額法と一定利回り法です。

###定額法

この方法では、債券の価値の増加は、債券の期間全体に均等に分散されます。たとえば、債券の期間が10年で、会社が四半期ごとに財務報告を行う場合、満期までに40の財務期間があることを意味します。

500ドルの割引は、40の期間に分割されます。これは、四半期ごとに12.50ドルに相当します。満期までの各期間で12.50ドルの増加があり、この方法では、償還日までの各期間で債券負債残高が12.50ドル増加します。

###一定利回り法

債券の価値の増加は、一定利回り法では満期日に最も近いものです。一定歩留まり法と定額法の違いは、一定歩留まり法でも増分が均一ではないことです。一部の期間は他の期間よりも大きな利益を示し、利益は債券の寿命の最後の段階に集中します。

一定利回り法で債券の増加を考慮する場合、最初のステップは満期利回り（YTM）を決定することです。 YTMは、債券が満期日までに獲得するものです。この計算には、債券の額面価格、価格、満期までの年数、および債券の利率の3つの入力が必要です。

##付加価値に関するFAQ

###割引の増加とは何ですか？

割引の増加とは、時間の経過とともに満期日が近づくにつれて、債券などの割引商品の価値が上昇することを指します。債券の価値は、割引発行価格、満期時の価値、および満期期間によって示される金利で増加します。

###複合付加価値とは何ですか？

複合付加価値（CAV）は、ゼロクーポン債の価値の尺度を指します。これは、満期日前のゼロクーポン債の価値を計算するために使用されます。

###社債の割引をどのように記録しますか？

社債の割引は、常に社債とともに貸借対照表に記録されます。債券が固定負債である限り、社債と社債の割引の両方が固定負債として貸借対照表に報告されます。

###付加利息とは何ですか？

転換日以前の債券、特に資本増価債と転換可能な資本増価債に関連するため、付加利息とは、付加価値から額面金額を差し引いたものを指します（計算日現在）。

付加利息のより一般的な定義は、元本が発生する間に利息として支払われるのではなく、元本に追加されるローン資産に発生する利息です。

##ハイライト

-付加価値とは、任意の時点で、利息が発生するが満期までその利息を支払わない複数年の商品の価値です。

-付加価値は、特定の時点で売却された場合の債券の理論価格として概念化できます（そして、市場金利は満期まで最新のレベルのままでした）。

-債券の付加価値は、その市場価値とは何の関係もない場合があります。

-付加価値は、債券の現在の価値であり、多くの場合、貸借対照表の目的で計算されます。これには、未収利息が含まれます（通常、債券が満期になるまで支払われません）。

-付加価値の概念は、ゼロクーポン債または累積優先株式に見られます。
