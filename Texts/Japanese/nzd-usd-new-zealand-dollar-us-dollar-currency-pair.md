# NZD / USD（ニュージーランドドル/米ドル）
## NZD / USD（ニュージーランドドル/米ドル）とは何ですか？

NZD / USDは、ニュージーランドドルと米ドルの[通貨ペアの略語です](/currencypair)。この通貨ペアの見積もり価格は、1ニュージーランドドルを購入するために必要な米ドル数を読者に示します。

## NZD / USD（ニュージーランドドル/米ドル）を理解する

NZD / USDペアの値は、可変数の米ドルあたり1ニュージーランドドルとして見積もられます。たとえば、ペアが1.50で取引されている場合、1ニュージーランドドルを購入するには1.5米ドルかかることを意味します。

農業はニュージーランド経済の主要な要因です。輸出の3分の2以上は農業です。 NZDに影響を与える1つの特定の要因は、乳製品の価格です。ニュージーランドは全粉乳の世界最大の輸出国です。したがって、牛乳の価格が上昇している場合、ニュージーランド経済は改善する可能性が高く、トレーダーは予想して通貨の価格を上げる可能性があります。観光はニュージーランド経済のもう一つの定番であり、ニュージーランドへの訪問がより安価になるにつれて、経済は改善すると予想され、通貨は上昇する可能性があります。

ニュージーランドは、農業部門が国際経済に完全にさらされている（補助金や関税がない）数少ない国の1つですが、NZD / USDペアは、地域経済やその他とは関係のないさまざまな経済的理由で取引される可能性があります。それが生み出す。ニュージーランドは最初に新しい取引日を開きます。銀行やトレーダー市場は、この事実を利用して、翌日のイベントを見越して取引を位置付けることがあります。

>外国為替トレーダーの間では、NZD/USD通貨ペアは「キウイ」と呼ばれています。

>>

NZD / USDは、ニュージーランドドルおよび/または米ドルの相互および他の通貨との関係に影響を与える要因の影響も受けます。[ニュージーランド](/reserve-bank-of-new-zealand)[準備銀行](/reserve-bank-of-new-zealand)と米[連邦準備銀行（FRB）](/federalreservebank)の金利差は、これらの通貨の価値を相互に比較した場合に影響します。たとえば、FRBが公開市場活動に介入して米ドルを上昇させると、ニュージーランドドルと比較した場合の米ドル高により、NZD/USDクロスの価値が下落する可能性があります。

ニュージーランドドルは、比較的高利回りの通貨であるという点で、キャリートレード通貨と見なされています。このため、投資家はNZDを購入し、日本円やスイスフランなどの低利回り通貨で資金を調達することがよくあります。

このような取引はリスクを求める投資家向けであり、投資家がリスクを嫌うようになると閉鎖される傾向があります。この証拠は、NZDが日本円に対して50％近く下落した2008年の金融危機の際に蔓延していました。ボラティリティが上昇するにつれて、投資家はこれらのキャリートレードを解き放ち、NZDは2008年と2009年に下落した多くの高利回り通貨の1つでした。

NZD / USDは、その隣国であるオーストラリアドル（AUD / USD）と正の相関関係がある傾向があります。

##ハイライト

-NZD / USDは、ニュージーランドドルと米ドルの現在の為替レートを見積もります。

-1ニュージーランドドルを購入するのに必要な米ドル数を示します。

-NZD / USDは、トレーダーがキャリートレードを行いたい場合にも考慮されます。
