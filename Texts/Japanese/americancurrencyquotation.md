# アメリカの通貨の見積もり
##アメリカの通貨の見積もりとは何ですか？

アメリカの通貨相場は、[外国為替](/foreign-exchange-markets)[市場での相場](/foreign-exchange-markets)であり、米ドルの価値は外貨の単位当たりの尺度として表されます。このタイプの見積もりは、1単位の外貨を購入するのに必要な米国[通貨](/currency)の金額を示しています。

##アメリカの通貨の見積もりを理解する

たとえば、アメリカの通貨相場は1カナダドルあたり0.85米ドルになります。これは、カナダ通貨の1単位を購入するのに0.85米ドルかかることを示しています。 1,000カナダドルを購入するには、850米ドルかかります。関連する通貨ペアはCAD/USDです。

アメリカの通貨の見積もりの反対は、外貨が米ドルの単位当たりの測定値である[ヨーロッパの通貨の見積もりです。](/directquote)再びカナダドルを例として使用して、1米ドルあたり1.40カナダドルのレートを想定します。これは、1米ドルを購入するのに1.40カナダドルかかることを説明しています。この場合、関係するペアは[USD/CADに反転し](/usd-cad-us-dollar-canadian-dollar-currency-pair)ます。

通貨ペアでは、リストされている最初の通貨は単一の単位であり、添付の数字または見積もりは、最初の単一の単位を購入するのに必要な2番目の通貨の量を示します。

##直接および間接の見積もり

トレーダーは、すべての用語が使用されていますが、引用符をアメリカやヨーロッパではなく、直接または間接と呼ぶことがよくあります。

直接見積もりとは、1単位の外貨を購入するのに必要な国内通貨の額です。 USD / CADレート、たとえば1.35は、1米ドルを購入するのに何カナダドルかかるかを示しているため、カナダの直接見積もりレートです。米国の誰かにとって、これは間接的な見積もりになります。

##アメリカの引用の解釈

アメリカの見積もりには、 [EUR / USD](/eur-usd-euro-us-dollar-currency-pair) 、AUD / USD、GBP / USD、NZD / USDが含まれます。これらのペアは、リストされている最初の通貨を購入するのに必要な米ドルを示しているためです。

EUR/USDが1.1525で取引されていると仮定します。翌月は1.1960で取引されています。ペアの価格は上昇しました。これは、EURの価値がUSDに比べて上昇したことを意味します。現在、1ユーロを購入するにはさらに米ドルがかかります。

したがって、通貨価格チャートを見ると、ペアの最初の通貨は方向性通貨です。レートが上昇している場合、最初の通貨は2番目の通貨に比べて上昇しています。レートが下がっている場合、最初の通貨は2番目の通貨に比べて価値が下がっています。

レートが1.1525から1.1310に下がると、ユーロは米ドルに対して値が下がります。

##アメリカの通貨の見積もりと価格変更の例

ある[AUD/USD](/aud-usd-australian-dollar-us-dollar-currency-pair)が0.6845で取引されていると仮定します。これは、オーストラリアドルを購入するのに0.6845ドルかかることを意味します。このレートのヨーロッパの見積もりは、USD / AUDレートである1.4609（1 / 0.6845）になります。これは、1米ドルを購入するのに何オーストラリアドルかかるかを示しています。

AUD / USDの価格チャートでは、レートが0.70に上昇すると、AUDの値はUSDに比べて増加します。レートが0.65に下がると、AUDはUSDに対して価値を失います。

AUDが下落している間、USDは上昇しています。これは、USD/AUDチャートを見るとわかります。すべてが逆さまになります。 AUD / USDが下落すると、USD / AUDは上昇し、逆もまた同様です。

##ハイライト

-通貨ペアでは、リストされている最初の通貨は1単位であり、リストされているレートは、最初の1つのユニットを購入するのに必要な2番目の通貨の量です。

-通貨は直接または間接見積もりとも呼ばれ、直接見積もりは1単位の外貨を購入するのに必要な国内通貨の金額です。

-米国の通貨見積もりは、1単位の外貨を購入するのに必要な米国の通貨です。
