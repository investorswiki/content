# 実際の延期と実際の貢献率テスト（ADP / ACP）
##実際の延期率（ADP）および実際の貢献率（ACP）テストとは何ですか？

実際の延期率（ADP）と実際の貢献率（ACP）のテストは、401（k）プランが他の人を犠牲にして高給の従業員に不当に利益をもたらさないことを確認するために、企業が実施しなければならない2つのテストです。

401（k）プランを提供する企業は、IRS規則および[従業員退職所得保障法](/erisa)（ERISA）に基づくプランの適格なステータスを維持するために、テストを実施する必要があります。

計画がいずれかのテストに合格しなかった場合、雇用主は、監視が発生した計画年の終了後12か月間に是正措置を講じる必要があります。そうしないと、IRSが金銭的罰金を課し、計画の失格となり、雇用主の側に受託者責任を負わせる可能性があります。

##ADPおよびACPテストのしくみ

ADPテストでは、[高報酬の従業員](/highly-compensated-employee)（HCE）の平均給与繰り延べ率を非高報酬の従業員（NHCE）の平均給与繰り延べ率と比較します。 HCEとは、現在または前の計画年度中にいつでも会社の5％以上の持分を所有しているか、2020年の課税年度中に13万ドル以上を稼いだ従業員です。

ADPテストでは、税引き前の延期と税引き後のロスの延期の両方が考慮されますが、50歳以上の従業員のみが行うことができるキャッチアップの貢献は考慮されません。テストに合格するには、HCEのADPがNHCEのADPを2パーセントポイント以上超えてはなりません。さらに、すべてのHCEの合計貢献度は、NHCE貢献度の2倍を超えてはなりません。

[一致する拠出金](/matchingcontribution)または従業員[の税引き後の拠出金](/aftertaxcontribution)を使用することを除いて、ADPテストと同様の方法を使用します。

## ADP/ACPテストの失敗を修正する

雇用主がADP/ACPテストに不合格になった場合、テストに合格するために必要な金額の超過拠出金をHCEに返金することにより、不合格を是正することができます。ただし、これらの払い戻しは、HCE個人の所得税に対して責任があります。

一部の企業は、最初にADP / ACPテストに失敗する可能性を回避するために、計画ドキュメント内にバッファーゾーンを設定しています。 1つのオプションは、HCEによる貢献に上限を設定することです。もう1つのオプションは、プランがADP/ACPテストに失敗するポイントでHCEにコントリビューション制限を設定することです。計画緩衝地帯を設定すると、雇用主は、制限を適用する必要があるかどうかを判断するために、通常は計画年の半ばにADP/ACPテスト予測を実施する必要があります。

それでも、一部の企業は、ADP / ACPテストを完全に回避するために[セーフハーバー](/safeharbor)401（k）プランを使用しています。

##セーフハーバープランとは何ですか？

は、従業員に代わって適格なマッチングまたは非[選択的な拠出](/non-electivecontribution)を提供する代わりに、ADP/ACPおよびその他の差別のないテストをバイパスできます。

セーフハーバーの資格を得るには、企業は、延期された報酬の最初の3％で100％の一致、3％から5％の延期で50％の一致など、基本的な一致を提供する必要があります。また、従業員がどれだけ貢献したか、またはまったく貢献したかどうかに関係なく、各従業員に少なくとも3％の報酬の非選択的貢献を提供する場合があります。
