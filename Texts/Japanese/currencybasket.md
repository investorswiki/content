# 通貨バスケット
##通貨バスケットとは何ですか？

通貨バスケットは、異なる重みを持ついくつかの通貨のセットです。これは、別の通貨の市場価値を設定するためによく使用されます。これは、一般に[通貨ペグとして知られている方法](/currency-peg)です。外国為替トレーダーは、複数の通貨ペアを同時に取引するために[バスケット注文を入力することもできます。](/baskettrade)

口語的には、通貨バスケットは通貨カクテルと呼ばれることもあります。

##通貨バスケットを理解する

[固定通貨](/currency-peg)の場合など、独自の為替レートを設定するための参照として通貨バスケットを使用する場合があります。通貨当局は、単一通貨に固定するのではなく、外貨のバスケットを使用することで、為替レートの変動を抑えることができます。

通貨バスケットは、通貨変動のリスクを回避（または最小化）する方法として、契約でも使用されます。欧州[通貨単位](/european-currency-unit)（ユーロに置き換えられました）と[アジア通貨単位](/asian-currency-unit)は、通貨バスケットの例です。ただし、最もよく知られている通貨バスケットは[米ドルインデックス](/usdx)（USDX）です。

米ドルインデックスは1973年に開始され、現在は[ユーロ](/euro)、[日本円](/jpy-japanese-yen)、[英国ポンド](/gbp)、[カナダドル](/cad-canadian-dollar)、[スウェーデンクローナ](/sek)、[スイスフランの6つの通貨のバスケットになってい](/chf)ます。ユーロは、指数の最大の構成要素であり、バスケットのほぼ58％（公式には57.6％）を占めています。インデックスの残りの通貨のウェイトは、JPY（13.6％）、GBP（11.9％）、CAD（9.1％）、SEK（4.2％）、CHF（3.6％）です。 21世紀の間に、指数は技術ブームの間に121の最高値に達し、大不況の直前に71の最低値に達しました。

##通貨バスケットの使用

さまざまな国にエクスポージャーを持っている株式投資家は、リスクをスムーズにするために通貨バスケットを使用します。彼らのコア投資戦略は株式市場にありますが、通貨変動のために外国の株式市場に投資する際に大きな損失を被ることを望んでいません。[債券保有者](/bondholder)についても同じことが言えます。

一方、単一の通貨について幅広い見方をしている通貨トレーダーは、さまざまな異なる通貨に対してその通貨を所有することを選択します。たとえば、米ドルに対して強気なトレーダーは、USDXを使用してこの見解を表現できます。トレーダーと投資家は、戦略に応じて異なるウェイトで独自の通貨バスケットを作成できます。

バスケットトレードの通貨のウェイトは、トレーダーによって、または戦略やプログラムに従って決定されます。たとえば、トレーダーが米ドルのポジションを蓄積したい場合は、 [EUR / USD](/eur-usd-euro-us-dollar-currency-pair) 、GBP / USD、およびAUD / USDを販売し、USD / JPY、USD / CAD、 [USD/CHFを購入することができ](/usd-chf-us-dollar-swiss-franc-currency-pair)ます。彼らは資金の20％をEUR/USDとGBP/USDの両方に投入しました。資金の残りの60％は、他の4つの通貨ペアに分割され、それぞれに15％が割り当てられます。

>株式の場合と同様に、機関投資家は複数の通貨ペアで大量の取引を迅速に実行する必要がある場合があります。バスケット注文は彼らがそれをすることを可能にします。

##例：USDショーツのバスケット

USDショートのバスケットは、単一の[通貨ペア](/currencypair)ではなく、通貨のグループに対して米ドルを売ることを含む[外国為替取引戦略です](/forex)。このような状況で[米ドル](/usd)が取引される通貨グループは、[バスケットと呼ばれ](/baskettrade)ます。したがって、ドルが一度に複数の通貨に比べて価値が下がった場合、戦略は報われます。バスケットを使用すると、分散投資を通じて取引の全体的なリスクエクスポージャーが減少します。

##ハイライト

-通貨バスケットは、それぞれ異なる重みを持つ通貨の組み合わせで構成されています。

-さまざまな国にエクスポージャーを持つ外国為替トレーダーまたは株式投資家は、リスクをスムーズにするために通貨バスケットを使用できます。

-通貨バスケットは、通貨当局が通貨の価値を設定するために使用する場合があります。
