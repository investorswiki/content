# 暗号通貨セキュリティトークン
##セキュリティトークンとは何ですか？

ブロックチェーンテクノロジーはデータサイエンスの分野から生まれたため、暗号通貨とトークンで使用される用語の多くは、それらの分野で使用される用語と似ています。 「トークン」という用語はその1つです。データサイエンスでは、トークンは、元の情報をマスクするために機密データに割り当てられる、ランダムに生成された数値のような値です。したがって、ブロックチェーンでは、トークンはブロックチェーン内に保存されているデータに割り当てられた番号です。アセットにトークンを与えることを「トークン化」と呼びます。

投資資産としてのセキュリティトークンは、所有権またはその他の権利を表し、資産または資産のバンドルからトークンに価値を転送するデジタル資産です。平易な言葉で言えば、セキュリティトークンは、株式、債券、その他の証券化資産などの従来の投資のデジタル形式です。たとえば、拡張プロジェクトのために資金を調達したい会社は、株式を発行する代わりに、デジタルトークンを介して会社の分割所有権を発行することを決定できます。次に、デジタルセキュリティトークンを許可する取引所でこのトークンを投資家に提供できます。

>トークンは、Howeyテストで設定された基準を満たしている場合、規制当局によって有価証券と見なされます。つまり、お金の投資、一般的な企業、そして努力による利益の合理的な期待があります。

>>

##セキュリティトークンを理解する

トークン化を理解することは、セキュリティトークンを理解するために重要です。何でもトークン化できます。車の所有権と登録を示すトークンを作成できます。車の車両識別番号（VIN）は、所有者の名前、住所、および州が車両を登録するために必要なその他の情報とともにトークン化することができます。州の自動車部門は、ブロックチェーンインターフェースプログラムを使用して情報をブロックチェーンに入力し、車両登録と所有権トークンを生成します。

セキュリティトークンも同様に作成されます。企業はトークンが表すものを入力すると、トークンが生成されます。その後、会社はこのトークンを取引所または投資家向けの他の適切な投資プラットフォームで提供します。所有権はブロックチェーンに記録されます。

会社または資産の所有権をトークン化する背後にある考え方は、必ずしも新しいものではありません。たとえば、企業は株式を購入した投資家に紙の株券を提供していました。紙の証明書は、投資家に付与された所有権またはその他の権利を表すトークンでした。デジタルセキュリティトークンは、デジタルであり、ブロックチェーントークン化プロセスを経ていることを除いて、違いはありません。

>ほとんどのセキュリティトークンプラットフォームは、トークンの生成にイーサリアムERC2.0またはTezosのFA1.2標準を使用しています。

>>

セキュリティトークンは、識別されるいくつかの異なる形式の1つをとる場合があります。セキュリティトークンには、その値とともにデジタルウォレットに表示できる画像が割り当てられている場合があります。一方、それはあなたの財布が追跡している数だけかもしれません。ウォレットには、持ち株、価値、および配当金の分配が表示される可能性があります。お財布はまた、目論見書や年次報告書への迅速なアクセスを提供するかもしれません。

##セキュリティトークンは暗号通貨とどのように異なりますか？

セキュリティトークンと暗号通貨はほぼ同じです。それらはブロックチェーンによって作成され、ブロックチェーンに保存されます。これらは両方ともトークンですが、決定的な違いは、それらの目的、使用目的、および実際の使用にあります。暗号通貨は、通貨、お金、または支払い方法として使用されるように設計されています。セキュリティトークンは、株式、債券、証明書、またはその他の投資資産が使用されるのと同じように使用されることを目的としています。

投資手段として使用することを意図していない多くの暗号通貨が導入されています。たとえば、投資家やトレーダーは、ビットコインが暗号通貨取引所に上場されたときに、ビットコインから大きな利益を得ることができることに気づきました。その結果、投資家はビットコインがセキュリティトークンとして設計されていなくても、セキュリティトークンとして扱います。

>証券取引委員会は、ビットコインとイーサリアムを証券とは見なしていません。

>>

イーサリアムのネイティブトークンであるイーサリアムは、イーサリアムネットワーク内での取引手数料の支払いに使用するために開発されました。その点で、エーテルは暗号通貨です。ただし、取引所で取引され、その価値が高まるために保有されているため、投資家はそれをセキュリティトークンとして扱います。

ただし、BTCとETHはセキュリティトークンとして使用するように設計されておらず、開発者からの利益も期待できないため、現在、SECが証券と見なす基準を満たしていません。

##ハイライト

-セキュリティトークンは、投資基準が選択されるトークン化を使用して作成されます。情報はブロックチェーンに入力され、ブロックチェーンがトークンを作成します。

-セキュリティトークンは、ブロックチェーントークンに譲渡された所有権または資産価値を表すデジタル資産です。

-個人投資家はセキュリティトークンをまだ利用できませんが、多くの機関がトークンの開発と提供に取り組んでいます。

-証券取引委員会はセキュリティトークンを承認する必要があります。

＃＃ よくある質問

###セキュリティトークンに投資できますか？

公的株式または暗号通貨取引所の個人投資家は、セキュリティトークンをまだ利用できません。

###セキュリティトークンとは何ですか？

セキュリティトークンは、ブロックチェーンでトークン化された所有権、価値の移転、または返品の約束を表します。投資手段として扱われることを目的としています。

###イーサリアムはセキュリティトークンですか？

イーサリアムのネイティブトークンであるetherは、セキュリティトークンではありません。イーサリアム仮想マシン内で取引手数料を支払うために使用することを目的としています。所有者はまた、イーサリアム（ETH）を「賭け」て、ネットワークバリデーターになり、「マージ」が発生したときにさらに稼ぐ（ETH）チャンスを得ることができます。
