# S＆amp; P / ASX 200 VIX（A-VIX）
## S＆P / ASX 200 VIX（A-VIX）とは何ですか？

、オーストラリアのベンチマークエクイティインデックスであるS＆P / ASX 200の今後30日間の[変動性](/volatility)に関する投資家の期待を反映した、リアルタイムのインデックスです。

S＆P / ASX 200 VIXは、主に[市場感情のバロメーターとして使用され](/marketsentiment)ます。他のボラティリティ指標と同様に、比較的高いA-VIXは不確実な投資家の期待とより広い取引範囲を反映し、より低いA-VIXは投資家の信頼とより狭い取引範囲を示唆します。

## S＆P / ASX 200 VIX（A-VIX）の仕組み

S＆P / ASX 200 VIX（A-VIX）は、インデックスのプットオプションとコールオプションの中間価格を[利用](/putoption)し[て](/calloption)、これらのオプションの暗黙の変動性の加重平均を計算します。インデックスは、満期から最も遠いオプションのボラティリティと比較して、満期に最も近いオプションのボラティリティを補間し、エクイティベンチマークで予想されるボラティリティの30日間の指標を導き出します。

他のVIXインデックスと同様に、A-VIXは、基礎となるS＆P / ASX 200インデックスと強い[負の相関関係を示し](/negative-correlation)、市場参加者が予想される市場の変化に合わせてポートフォリオを配置できるようにします。 2013年10月のS＆P / ASX 200 VIX先物の発売により、トレーダーは1回の取引でオーストラリアの株式市場の変動性の予想される変化を直接推測することができました。

S＆P / ASX 200インデックスは、オーストラリアの株式市場の資本の約80％をカバーし、BHPビリトンやリオティントなどの世界的な鉱業大手、およびコモンウェルスバンクオーストラリアやANZバンキンググループなどの大手銀行の本拠地です。

基盤となるASXは、市場資本の面で世界最大の1つである垂直統合交換グループです。

A-VIXは、基礎となるインデックスの現在の変動性のレベルを反映する他のインデックスよりも前向きである傾向があります。

インデックスの各コンポーネントの暗黙の変動性を集計すると、近い将来に全体的なインデックス価格がどの程度変化すると予想するかについての市場参加者の意見が予測されます。これにより、トレーダーは、たとえば、ボラティリティが予想よりも小さいか大きいかを推測できます。また、A-VIXが極端な感情を上向きまたは下向きに反映している場合、[反対派のトレーダー](/stock-trader)はA-VIXまたはASX200のいずれかで市場が[逆転する可能性があることを示すことができます。](/reversal)

##特別な考慮事項

ただし、A-VIXは、投資家ではなく、主にトレーダーによって使用されます。[市場のタイミング](/markettiming)を組み込んだ戦略を持つオーストラリアの株式市場への投資家は、今後数週間で基礎となるインデックスに何が起こるかについての手がかりについてA-VIXを見るかもしれません。ただし、多くの人はそのような短期間の信号を無視する傾向があり、長期に焦点を当てる傾向のある基本的な分析に固執することを好みます。このため、投資アプローチにA-VIXを正式に含める人はほとんどいません。

##ハイライト

-A-VIXとして知られるS＆P / ASX 200 VIXは、オーストラリアのベンチマーク株式インデックスであるS＆P /ASX200が今後30日間でどのようになるかを投資家がどのように考えているかを測定するインデックスです。

-世界中の他のVIXインデックスと同様に、A-VIXが高いほど、投資家の不確実性が高くなり、取引が変動する可能性があります。読み取り値が低いほど、投資家の信頼が高まり、取引範囲が狭くなります。

-これはオーストラリアの市場感情の前向きな指標であり、主にトレーダー、特に市場タイミング戦略を利用するトレーダーと長期投資家によって使用されます。
