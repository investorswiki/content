# 想定値
##概念上の価値とは何ですか？

想定価値は、デリバティブ取引の原資産を評価するためによく使用される用語です。これは、ポジションの合計値、ポジションが制御する値、または契約で合意された金額にすることができます。この用語は、オプション、[先物](/futures)、および通貨市場でのデリバティブ契約を説明するときに使用されます。

##表記値を理解する

市場用語では、想定元本はデリバティブ取引の原資産の合計額です。レバレッジと呼ばれる概念により、デリバティブ契約の想定価値は市場価値よりもはるかに高くなっています。

レバレッジを使用すると、少量のお金を使用して、理論的にははるかに大きな金額を制御できます。したがって、想定価値は、取引の合計値を取引のコスト（または[市場価値）と区別するのに役立ちます](/marketvalue)。明確な区別があります。概念値はポジションの合計値を占め、市場価値は価格です。そのポジションは市場で売買することができます。利用されるレバレッジの量は、想定価値を市場価値で割ることによって計算できます。

>>

>**レバレッジ**=想定価値÷市場価値

>>

契約には、重量、量、乗数などの要素に基づくことができる、独自の標準化されたサイズがあります。たとえば、単一のCOMEXゴールド先物契約ユニット（GC）は100トロイオンスであり、E-mini S＆P500インデックス先物契約には50ドルの乗数があります。前者の想定値は市場価格の100倍です。後者の想定価値はS＆P500インデックスの市場価格の50倍です。

>>

>**想定価格**=契約サイズ*原資産価格

>>

誰かがE-miniS＆P 500契約を2,800で購入した場合、その単一[先物契約](/futurescontract)は$ 140,000（$ 50 x 2,800）の価値があります。したがって、140,000ドルは、その基礎となる先物契約の想定元本です。ただし、この契約を購入する人は、取引を行うときに$140,000を支払う必要はありません。

むしろ、彼らは[初期証拠金](/initialmargin)（市場価値）と呼ばれる金額を提示するだけでよく、これは通常、想定元本の何分の1かです。使用されるレバレッジは、想定元本を契約購入価格で割ったものになります。 1つの契約の価格（初期証拠金）が$ 10,000の場合、トレーダーは14倍のレバレッジ（140,000 / 10,000）を利用できました。

想定価値はポートフォリオリスクの評価に不可欠であり、そのリスクを相殺するためのヘッジ比率を決定する際に非常に役立ちます。たとえば、ファンドは米国の株式市場に1,000,000ドルの長期エクスポージャーを持っており、ファンドマネージャーはE-mini S＆P500先物契約を使用してそのリスクを相殺したいと考えています。彼らは、市場エクスポージャーリスクをヘッジするために、ほぼ同等のS＆P500先物契約を売却する必要があります。上記の例を使用すると、各E-mini S＆P500先物契約の想定価値は$140,000であり、市場価値は$10,000です。

>>

>**ヘッジ比率**=現金エクスポージャーリスク÷関連する原資産の想定元本

>>

>>

>**ヘッジ比率**=$1,000,000÷$140,000= 7.14

>>

したがって、ファンドマネージャーは約7つのE-mini S＆P 500契約を売却して、市場リスクに対して長期の現金ポジションを効果的にヘッジします。市場価値（コスト）は70,000ドルになります。

想定元本は、上記の方法で先物および株式（株式ポジションの合計値）で使用できますが、想定元本は、金利スワップ、トータルリターンスワップ、エクイティオプション、および外貨デリバティブにも適用されます。

##金利スワップ

[金利スワップで](/interestrateswap)は、想定元本は、金利の支払いが交換される指定された値です。金利スワップの想定元本は、支払われるべき[利息](/interest-due)の額を算出するために使用されます。通常、これらのタイプの契約の想定元本は、契約期間中に固定されます。

##トータルリターンスワップ

[トータルリターンスワップ](/totalreturnswap)には、変動金利または固定金利に想定元本額に想定元本の減少を掛けた金額を支払う当事者が関与します。これは、想定価値の上昇を支払う別の当事者による支払いと交換されます。

##エクイティオプション

の想定値は、[オプション](/option)が制御する値を指します。

たとえば、ABCは$ 20で取引されており、特定のABCコールオプションの価格は$1.50です。 1つのエクイティオプションが100の原株を管理します。トレーダーはオプションを$1.50x 100 =$150で購入します。

オプションの想定価値は$20x 100 =$2,000です。[ストックオプション契約を](/stockoption)購入すると、トレーダーは、2,000ドルで株式を完全に購入した場合と比較して、150ドルで100株の株式を管理できる可能性があります。

エクイティオプション契約の想定価値は、取引のコストではなく、管理されている株式の価値です。

##外貨両替および外貨デリバティブ

[先渡やオプション](/forwardcontract)などの外国為替デリバティブには、2つの概念上の価値があります[。](/forwardcontract)これらのトランザクションには2つの通貨が含まれるため、両方とも別々の想定値を受け取ります。たとえば、取引時に英国ポンド（GBP）と米ドル（USD）の為替レートが1.5の場合、$1,000,000USDは666,667GBPに相当します。

##ハイライト

-想定価値はポートフォリオリスクの評価に不可欠です。これは、そのリスクを相殺するためのヘッジ比率を決定する際に非常に役立ちます。

-想定価値は、デリバティブ取引の原資産を評価するためによく使用される用語です。

-デリバティブ契約の想定価値は、レバレッジと呼ばれる概念により、市場価値よりもはるかに高くなっています。
