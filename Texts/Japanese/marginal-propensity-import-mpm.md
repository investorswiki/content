# インポートする限界傾向（MPM）
##輸入の限界傾向（MPM）とは何ですか？

限界輸入傾向（MPM）は、[可処分所得](/disposableincome)の単位が増減するたびに[輸入が増減する量](/import)です。アイデアは、企業や家庭の収入の増加が海外からの商品のより大きな需要に拍車をかけ、逆もまた同様であるということです。

##限界インポート傾向（MPM）のしくみ

MPMは、[ケインジアンの](/keynesianeconomics)マクロ経済理論の構成要素です。これは、dIm / dYとして計算されます。これは、所得関数（Y）の導関数に対する輸入関数（Im）の導関数を意味します。

[所得](/income)または生産の変化の影響を受ける範囲を示します。たとえば、ある国のMPMが0.3の場合、その[経済における1ドルの追加収入は、](/economy) 30セントの輸入（$ 1 x 0.3）を誘発します。

人口の増加に伴いより多くの輸入品を消費する国は、世界の貿易に大きな影響を及ぼします。海外から大量の商品を購入する国が金融危機に陥った場合、その国の経済的苦境が[輸出](/export)国に与える影響の程度は、その国のMPMと輸入される商品の構成に依存します。

>[限界消費性向](/marginalpropensitytoconsume)（MPC）がプラスの経済は、消費される財の一部が海外から来る可能性が高いため、MPMがプラスになる可能性があります。

>>

ある国のMPMが平均的な輸入傾向よりも大きい場合、所得の減少による輸入への悪影響のレベルは大きくなります。このギャップは、輸入[に対する需要の所得弾力性を高め](/incomeelasticityofdemand)、所得の低下につながり、輸入の比例以上の低下をもたらします。

##特別な考慮事項

[先進国](/developed-economy)とその国境内に十分な天然資源がある国は、通常、MPMが低くなっています。対照的に、海外からの商品の購入に依存している国は、一般的に高いMPMを持っています。

###ケインズ経済学

MPMは、ケインジアン経済学の研究にとって重要です。まず、MPMは誘発された輸入を反映します。第二に、MPMは輸入ラインの傾きであり、これは[純輸出](/netexports)ラインの傾きの負の値であり、総支出ラインの傾きにも重要になります。

MPMは、[乗数](/multiplier)プロセスと、支出および税乗数の規模にも影響を与えます。

##限界輸入傾向（MPM）の長所と短所

MPMは測定が簡単で、出力の予想される変化に基づいてインポートの変化を予測するための便利なツールとして機能します。

問題は、国のMPMが一貫して安定している可能性が低いことです。国内外の商品の相対価格が変動し、[為替レート](/exchangerate)が変動します。これらの要因は、海外から出荷される商品の[購入力に影響を与え](/purchasingpower)、その結果、国のMPMの規模に影響を与えます。

##ハイライト

-限界輸入傾向（MPM）は、可処分所得の変化によって引き起こされる輸入の変化です。

-国境内に十分な天然資源がある先進国は、通常、これらの資源がない発展途上国よりもMPMが低くなります。

-人口の増加に伴いより多くの輸入品を消費する国々は、世界貿易に大きな影響を及ぼします。

-企業や家庭の収入が増えると、海外からの商品に対する需要が高まり、逆もまた同様です。
