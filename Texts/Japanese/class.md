# 株式のクラス
##株式のクラスとは何ですか？

株式の種類は、株主が受け取る議決権のレベルによって区別される上場会社の株式の一種です。たとえば、上場企業は、クラスAおよびクラスBとして指定された2つの株式クラス、または株式のクラスを持っている場合があります。私有で公開されている企業の所有者は、[多く](/privately-owned)[の](/privately-owned)場合、異なる議決権を持つクラスAおよびBの株式構造を作成します。支配を維持するため、および/または会社を買収のより困難な目標にするため。株式の主な種類の2つは、市場全体で利用可能な株式の大部分を表す普通株式と、通常は固定配当を保証するが議決権を持たない優先株式です。

一般的な種類の株式の1つは、アドバイザリー株式です。アドバイザーシェアとも呼ばれるこのタイプの株式は、ビジネスアドバイザーの洞察と専門知識と引き換えに提供されます。多くの場合、このタイプのストックオプション報酬を受け取るアドバイザーは、会社の創設者または上級管理職です。アドバイザの株式は通常、崖がなく、100％のシングルトリガー加速のないスケジュールで、1〜2年の期間にわたって毎月権利が確定します。

##株式のクラスを理解する

[投資信託](/mutualfund)をロードするために存在するさまざまな株式クラスを参照することもできます。 3つのシェアクラス（クラスA、クラスB、クラスC）があり、異なる[販売手数料](/salescharge)、12b-1手数料、および[営業費用](/operating_expense)構造があります。会社の株式の異なる株式クラスを参照する場合でも、アドバイザーが販売する投資信託によって提供される複数の株式クラスを参照する場合でも、どちらの場合も、各株式クラスの保有者が所有する異なる権利とコストを参照します。

##Googleの共有クラス構造

Googleのマルチクラスの株式構造は、2015年10月にAlphabet Inc.（NASDAQ：GOOG）に再編された結果として生まれました。創設者のセルゲイ・ブリンとラリー・ページは、会社の株式の過半数を所有していないことに気づきましたが、主要なビジネス上の決定に対する管理を維持したいと考えていました。その結果、同社は同社の株式の3つのシェアクラスを作成しました。クラスA株は通常の投資家によって保有されており、1株あたり1票を保有しています。主にブリンとページが[保有するクラスB株](/classbshares)は、1株あたり10票です。[クラスCの株式](/cshare)は通常、従業員が保有しており、議決権はありません。この構造は創設者にほとんどの投票権を与えますが、同様の設定は過去に平均的な株主に人気がないことが証明されています。

##投資信託シェアクラス

アドバイザーが販売する投資信託は、異なる株式クラスを持つことができ、各クラスは独自の販売手数料と手数料体系を所有しています。[クラスA](/classashares)ミューチュアルファンドの株式は、フロントエンドの負荷を請求し、12b-1の手数料が低く、営業費用のレベルが平均を下回っています。クラスBミューチュアルファンドの株式は[バックエンドの負荷](/back-end-load)を請求し、12b-1の手数料と営業費用が高くなります。クラスCの相互資金のシェアはレベル負荷と見なされます-[フロントエンドの](/front-endload)負荷はありませんが、12b-1の料金と比較的高い運用コストと同様に、低いバックエンドの負荷が適用されます。

[偶発的繰延販売手数料（CDSC）](/cdsc)として知られるバックエンドの負荷は、株式の保有期間に応じて削減または排除される場合があります。クラスB株には通常、購入日からわずか1年で消滅するCDSCがあります。クラスCのシェアは、多くの場合、5〜10年後に完全になくなるより高いCDSCで始まります。

##優先クラスの株式

投資家は、普通株と債券投資のクロスとして機能する優先株への投資を選択することがあります。普通株式と同様に、優先株式には満期日がなく、会社の所有権を表し、会社の貸借対照表に資本として計上されます。債券と比較して、優先株は固定の[分配](/distribution)率を提供し、議決権はなく、額面価格を提供します。

優先株はまた、会社の資本構成において普通株よりも上位にランクされています。したがって、企業は、普通株式のクラスに配当を支払う前に、優先株式に[配当を支払う必要があります。](/dividend)清算または破産の場合、優先株主も普通株式の保有者の前に支払いを受け取ります。

##ハイライト

-企業がさまざまな株式クラスを区別する理由の1つは、買収から身を守るためです。

-会社は、さまざまなレベルの議決権、配当へのアクセスなどを伴うさまざまな種類の株式を発行する場合があります。

-普通株式は通常、議決権を提供し、配当を含む場合があります。優先株式は通常、配当を保証しますが、議決権は含まれていません。
