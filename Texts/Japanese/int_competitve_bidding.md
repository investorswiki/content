# 国際競争入札（ICB）
##国際競争入札（ICB）とは何ですか？

国際競争入札（ICB）は、世界銀行が提供する[資金](/financing)で商品、仕事、および非コンサルティングサービスを取得する借り手に[世界銀行が要求する入札および調達プロセス](/worldbank)です。

##国際競争入札（ICB）の仕組み

世界銀行が資金提供するプロジェクトのためにサービスを調達する借り手には、国際的な競争入札プロセスが必要です。世界銀行には、借り手がビジネスパートナーを選択する際に従う必要のある詳細な要件と特定の条件があります。

[融資](/loan)で資金提供されたプロジェクトに国際競争入札プロセスを課すことの目標は、世界銀行が資金提供する経済的機会のための公正かつ健全な競争を促進することです。借入国は、プロジェクトの落札価格を選択する際にある程度の自由を維持していますが、最低価格で最も競争力のある入札を選択することが期待されています。

##世界銀行とは何ですか？

[世界銀行は](/worldbank)、ワシントンDCに本社を置く国際金融機関であり、教育、健康、行政、インフラ開発、金融、農業、環境の持続可能性に焦点を当てた経済開発プロジェクトの[貸し手として機能しています。](/lender)世界銀行は、低金利の融資、ゼロから低金利のクレジット、開発途上国への助成金の形で支援を提供しています。

世界銀行が資金提供するプロジェクトの例には、モンゴルの家畜飼育者の市場アクセスを改善するプログラム、リベリアの若者に雇用機会を提供するプロジェクト、チャコ地域に住む人々の水へのアクセスを改善するプログラムが含まれます。アルゼンチン。

##特別な考慮事項：ICBの要件とプロセス

世界銀行には、世界銀行が資金提供するプロジェクトに関連するサービスの国際競争入札プロセスで従わなければならない詳細な指示、条件、および要件があります。 ICBは、世界銀行の借り手に対して、各プロジェクトに必要なサービスを受け入れ可能な言語で国際的に宣伝することを求めています。借り手は、定性的判断のための特定の考慮事項を条件として、許容可能な最低入札額で契約を締結する必要があります。

世界銀行の資金を受け取る借り手は、国際競争入札のための銀行の調達フレームワークに従わなければなりません。フレームワークは、ポリシー、規制、指令、および手順をカバーしています。銀行は、使用する必要のある詳細なガイダンス、テンプレート、ドキュメント、およびその他の重要なリソース資料を提供しています。トレーニングも提供されます。さらに、世界銀行は借り手に彼らの進歩と活動を詳述する定期的な報告書を作成することを要求しています。

##ハイライト

-詳細な一連の手順は、世界銀行が資金提供する経済的機会をめぐる公正かつ健全な競争を確保するように設計されています。

-世界銀行は、開発途上国の教育、健康、行政、インフラ、金融、農業、環境の改善を目的としたプロジェクトに資金を提供しています。

-借り手は、調達する商品やサービスを宣伝する必要があり、許容可能な最低入札額を選択する必要があります。

-国際競争入札（ICB）とは、世界銀行が提供する資金調達を受け入れる借り手に要求する入札および調達プロセスを指します。
