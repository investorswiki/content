# 累積
##蓄積とは何ですか？

一般的に、蓄積とは、何かを収集または増加させることです。金融では、累積は、複数のトランザクションにわたって構築される資産の[ポジションサイズの](/positionsizing)増加をより狭く参照することができます。累積とは、[ポートフォリオ](/portfolio)へのポジションの全体的な追加を指す場合もあります。

テクニカル分析では、累積は資産の購入活動の一般的な増加を示します。この場合、資産は「蓄積中」または「蓄積中」と呼ばれます。

[繰延年金](/deferredannuity)は、契約の最初の年に[蓄積フェーズがあります。](/accumulationphase)このフェーズでは、セーバーが資金を提供しています。次に、蓄積フェーズの後に配布フェーズが続きます。このフェーズでは、退職者は自分の資金にアクセスして使用し始めます。

##蓄積を理解する

蓄積は、成長の概念の根底にあるため、金融および経済学の重要な概念です。企業が利益を増やす（そして株価を上げる）ためには、[資本を蓄積](/capitalaccumulation)して拡大し、新しいプロジェクトやビジネスに投資する必要があります。

トレーダーが複数のトランザクションにわたってポジションのサイズを増やすとき、彼らは株式または他の資産を蓄積しています。トレーダーは、一度にではなく、時間をかけてポジションを蓄積して、より良い平均価格を取得したり、市場への影響を少なくしたり、複数の購入から情報を取得したりすることができます。

大きなポジションをとるトレーダーは、可能な限り密かに購入することで市場への影響を制限しようとします。一度に買いすぎると価格が上がり、将来の購入コストが高くなる可能性があります。各トランザクションは、トレーダーに情報も提供します。彼らが購入を注文し、それが価格を簡単に押し上げる場合、彼らは限られた売り手がいることを知っています。彼らが[入札](/bid)し、それが即座に[満たされる](/fill)場合、彼らは売り手がいることを知っており、価格を押し上げることなくもっと買うことができるでしょう。

[ポートフォリオマネージャー](/portfoliomanager)がポートフォリオにポジションを追加することも指します。この意味で、投資家は投資を蓄積しています。投資家は時間の経過とともに退職ポートフォリオに貢献するため、その資金を使用して追加の株式、商品、およびその他の資産を購入する場合があります。

[出来高](/volume)が上昇しているとき、それは蓄積されていると言われます。これは、トレーダーと投資家が資産をまとめて購入する用意があることを意味します。資産の価値が下がり始めると、これは[分配と呼ばれ](/distribution)ます。この意味で、蓄積とは売り手よりも積極的な買い手を指し、それが価格を押し上げます。流通とは、買い手よりも積極的な売り手を指し、価格を押し下げます。

##蓄積の例

投資家が一度に複数の種類の蓄積を行う可能性があります。

投資家がPayPalHoldingsIncの購入に関心があると仮定します。 （PYPL）ポートフォリオへの長期投資として。彼らがすでに所有している他の人にこの株を追加することは、株の蓄積を表すでしょう。彼らは時間をかけてより多くを所有しています。

投資家は、**他の人**がPayPalを蓄積し始めたら、PayPalを購入したいと決めるかもしれません。これは、株価が上昇[傾向](/uptrend)にあり、価格が上昇していることを示しています。

投資家は、株価が89ドルの地域で[抵抗](/resistance)を突破し、それ以来上昇していることに注目しています。

彼らは91ドルで購入を開始します。株価は失速しますが、その後上昇を続けます。投資家は95ドルでさらに購入します。株価は引き続き好調で、101ドルでさらに購入することにしました。

複数のトランザクションで行われるこのタイプの購入は、累積と呼ばれます。彼らは一度に自分のポジションを購入しませんでした。代わりに、彼らはそれを複数の取引に広げ、時間の経過とともに株式のポジションサイズを増やしました。

##累積/分布（A / D）インジケーターの使用

[蓄積/分配（A / D）インジケーター](/accumulationdistribution)は、蓄積/分配ラインとも呼ばれ、在庫が蓄積されているか分配されているかを示すインジケーターです。 A / Dインジケーターは、資産の価格と出来高を使用して、その価格の方向を示したり、傾向を確認したりします。

指標は、資産の価格に沿っているように見える線として描かれています。しかし、それは価格よりもはるかに多くを考慮します。前期のA/D指標と資金フロー量を使用して計算されます。増加する線は、資産の蓄積を示します。逆に、減少する線は資産の分布を示します。

>累積分布（A / D）インジケーターは、期間間の価格変動を考慮しないため、他のテクニカル分析ツールで使用する必要があります。

>>

A / Dインジケーターは、強気と弱気のトレンド、および強気と弱気の発散を示すことができます。 A / Dラインが増加し、ボリュームが高い場合、これは強気の傾向を確認します。 A / Dラインが減少し、ボリュームが大きい場合、これは弱気傾向を確認します。

A / Dインジケーターは、反転が間近に迫ったときに信号を送ることもできます。 A / Dラインが下落したが、価格が弱気の価格行動に従っている場合、これは強気のA/D発散を示しています。逆に、A / Dラインが上昇するが、価格が強気の価格行動に従う場合、これは弱気のA/Dの相違を示しています。

##特別な考慮事項

###年金の累積

年金に関連するため、累積には別の定義があります。年金は、投資家に一定の支払いの流れを支払う金融商品です。年金の主な用途は、退職者の収入源としてです。年金には2つの主要なフェーズがあります。投資家が年金に資金を提供する[蓄積フェーズと](/accumulationperiod)、支払い開始後の[年金化フェーズです。](/annuitizationphase)

生命保険も累積の例として役立ちます。一定の年齢まで、その人は保険証券に毎月の保険料を寄付することができます。一定の年齢を過ぎると、彼らはお金や支払いを受け取り始めます。

##蓄積に関するFAQ

###資本蓄積とは何ですか？

[資本蓄積](/capitalaccumulation)[は](/capitalaccumulation)、投資による資本の増加です。言い換えれば、投資からの価値の累積であり、投資の現在の価値から初期投資を差し引いたものとして計算されます。

###蓄積フェーズとは何ですか？

累積フェーズは、年金などの拠出がアカウントに組み込まれる期間です。寄付と該当する収益は、分配されるまで累積されます。

###累積期間中に年金受給者が死亡した場合はどうなりますか？

年金受給者と所有者が同一人物である場合、累積額は死亡時に指定された受益者に支払われます。年金受給者が年金の所有者でない場合、所有者は年金の完全な管理を保持し、その累積価値を受け取ります。

##結論

蓄積は、何かの価値や量の増加など、何かの増加として定義されます。その具体的な定義は、その使用方法や業界によって異なります。たとえば、企業はプロジェクトに資金を提供し、事業を拡大するために資本を蓄積します。投資家は複数の取引にわたって株式やその他の資産を蓄積し、より有利な価格を獲得し、市場への影響を最小限に抑えます。年金の所有者は、時間の経過とともに拠出を行うことで年金の価値を蓄積します。

##ハイライト

-蓄積/分配（A / D）ラインは、在庫が蓄積されているか分配されているかを示すインジケーターです。

-財務では、蓄積とは、より具体的には、1つの資産のポジションサイズを増やすこと、所有する資産/ポジションの数を増やすこと、または資産の購入活動を全体的に増やすことを意味します。

-蓄積は、何かの量が時間の経過とともに追加または増加したときに発生します。

-年金の累積段階とは、保険料が支払われている期間、またはお金が投入されている期間を指します。

-価格が上昇している株式は、蓄積されていると見なされます。
