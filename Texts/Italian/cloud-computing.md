# Cloud computing
## Che cos'è il cloud computing?

Il cloud computing è la fornitura di diversi servizi tramite Internet. Queste risorse includono strumenti e applicazioni come archiviazione dati, server, database, rete e software.

Anziché conservare i file su un disco rigido proprietario o su un dispositivo di archiviazione locale, [l'archiviazione basata su cloud](/cloud-storage) consente di salvarli su un database remoto. Finché un dispositivo elettronico ha accesso al Web, ha accesso ai dati e ai programmi software per eseguirlo.

Il cloud computing è un'opzione popolare per le persone e le aziende per una serie di motivi, tra cui risparmi sui costi, maggiore produttività, velocità ed efficienza, prestazioni e sicurezza.

## Capire il cloud computing

Il cloud computing è chiamato così perché le informazioni a cui si accede si trovano in remoto nel cloud o in uno spazio virtuale. Le aziende che forniscono servizi cloud consentono agli utenti di archiviare file e applicazioni su server remoti e quindi di accedere a tutti i dati tramite Internet. Ciò significa che non è necessario che l'utente si trovi in un luogo specifico per accedervi, consentendo all'utente di lavorare in remoto.

Il cloud computing elimina tutto il lavoro pesante necessario per elaborare ed elaborare i dati dal dispositivo che porti con te o su cui ti siedi e lavori. Inoltre, sposta tutto quel lavoro su enormi cluster di computer lontani nel cyberspazio. Internet diventa il cloud e voilà: i tuoi dati, il tuo lavoro e le tue applicazioni sono disponibili da qualsiasi dispositivo con cui puoi connetterti a Internet, in qualsiasi parte del mondo.

Il cloud computing può essere sia pubblico che privato. I servizi cloud pubblici forniscono i loro servizi su Internet a pagamento. I servizi cloud privati, invece, forniscono servizi solo a un certo numero di persone. Questi servizi sono un sistema di reti che forniscono servizi in hosting. Esiste anche un'opzione ibrida, che combina elementi dei servizi pubblici e privati.

## Tipi di servizi cloud

Indipendentemente dal tipo di servizio, i servizi di cloud computing forniscono agli utenti una serie di funzioni tra cui:

- E-mail

- Archiviazione, backup e recupero dati

- Creazione e test di app

- Analisi dei dati

- Streaming audio e video

- Fornitura di software su richiesta

Il cloud computing è ancora un servizio abbastanza nuovo, ma viene utilizzato da diverse organizzazioni, dalle grandi aziende alle piccole imprese, dalle organizzazioni non profit alle agenzie governative e persino dai singoli consumatori.

## Modelli di distribuzione

Esistono vari tipi di nuvole, ognuna diversa dall'altra. I cloud pubblici forniscono i propri servizi su server e storage su Internet. Questi sono gestiti da società di terze parti, che gestiscono e controllano tutto l'hardware, il software e l'infrastruttura generale. I clienti accedono ai servizi tramite account a cui possono accedere praticamente chiunque.

I cloud privati sono riservati a una clientela specifica, solitamente un'azienda o un'organizzazione. Il centro servizi dati dell'azienda può ospitare il servizio di cloud computing. Molti servizi di cloud computing privato sono forniti su una rete privata.

I cloud ibridi sono, come suggerisce il nome, una combinazione di servizi pubblici e privati. Questo tipo di modello consente all'utente una maggiore flessibilità e aiuta a ottimizzare l'infrastruttura e la sicurezza dell'utente.

> Le nuove forme di servizi di cloud computing includono il cloud della comunità, il cloud dei big data e il multicloud.

>

## Tipi di cloud computing

Il cloud computing non è un singolo pezzo di tecnologia come un microchip o un cellulare. Piuttosto, è un sistema composto principalmente da tre servizi: [software-as-a-service](/software-as-a-service-saas) (SaaS), infrastructure-as-a-service (IaaS) e platform-as-a-service (PaaS).

1. **Software-as-a-service (SaaS)** implica la concessione in licenza di un'applicazione software ai clienti. Le licenze vengono generalmente fornite tramite un modello con pagamento in base al consumo o su richiesta. Questo tipo di sistema può essere trovato in Microsoft Office 365.

1. **Infrastructure-as-a-service (IaaS)** prevede un metodo per fornire qualsiasi cosa, dai sistemi operativi ai server e allo storage, attraverso la connettività basata su IP come parte di un servizio on-demand. I clienti possono evitare la necessità di acquistare software o server e invece procurarsi queste risorse in un servizio su richiesta in [outsourcing](/outsourcing). Esempi popolari del sistema IaaS includono IBM Cloud e Microsoft Azure.

1. **Platform-as-a-service (PaaS)** è considerato il più complesso dei tre livelli del cloud computing. PaaS condivide alcune somiglianze con SaaS, la differenza principale è che invece di fornire software online, è in realtà una piattaforma per la creazione di software che viene distribuito tramite Internet. Questo modello include piattaforme come Salesforce.com e Heroku.

## Vantaggi del Cloud Computing

Il software basato su cloud offre alle aziende di tutti i settori una serie di vantaggi, inclusa la possibilità di utilizzare il software da qualsiasi dispositivo tramite un'app nativa o un browser. Di conseguenza, gli utenti possono trasferire i propri file e le proprie impostazioni su altri dispositivi in modo completamente trasparente.

Il cloud computing è molto più che accedere a file su più dispositivi. Grazie ai servizi di cloud computing, gli utenti possono controllare la posta elettronica su qualsiasi computer e persino archiviare file utilizzando servizi come Dropbox e Google Drive. I servizi di cloud computing consentono inoltre agli utenti di [eseguire](/back-up) il backup di musica, file e foto, garantendo che tali file siano immediatamente disponibili in caso di arresto anomalo del disco rigido.

Offre inoltre alle grandi aziende un enorme potenziale di risparmio sui costi. Prima che il cloud diventasse una valida alternativa, le aziende dovevano acquistare, costruire e mantenere costose [tecnologie](/information-management-technology-imt) e infrastrutture per la gestione [della formazione.](/information-management-technology-imt) Le aziende possono scambiare costosi centri server e dipartimenti IT con connessioni Internet veloci, dove i dipendenti interagiscono con il cloud online per completare le proprie attività.

La struttura cloud consente alle persone di risparmiare spazio di archiviazione sui propri desktop o laptop. Consente inoltre agli utenti di [aggiornare](/upgrade) il software più rapidamente perché le società di software possono offrire i propri prodotti tramite il Web anziché attraverso metodi più tradizionali e tangibili che coinvolgono dischi o unità flash. Ad esempio, i clienti Adobe possono accedere alle applicazioni nel proprio Creative Cloud tramite un abbonamento basato su Internet. Ciò consente agli utenti di scaricare facilmente nuove versioni e correzioni ai propri programmi.

## Svantaggi del cloud

Con tutta la velocità, l'efficienza e le innovazioni che derivano dal cloud computing, ci sono, naturalmente, dei rischi.

La sicurezza è sempre stata una grande preoccupazione per il cloud, soprattutto quando si tratta di cartelle cliniche sensibili e informazioni finanziarie. Sebbene le normative costringano i servizi di cloud computing a rafforzare le loro misure di sicurezza e conformità, rimane un problema in corso. La crittografia protegge le informazioni vitali, ma se la chiave di crittografia viene persa, i dati scompaiono.

I server gestiti da società di cloud computing possono anche essere vittime di disastri naturali, bug interni e interruzioni di corrente. La portata geografica del cloud computing taglia in entrambi i modi: un blackout in California potrebbe paralizzare gli utenti a New York e un'azienda in Texas potrebbe perdere i propri dati se qualcosa causa il crash del suo provider con sede nel Maine.

Come con qualsiasi tecnologia, esiste una curva di apprendimento sia per i dipendenti che per i manager. Ma con molte persone che accedono e manipolano le informazioni attraverso un unico portale, gli errori involontari possono essere trasferiti in un intero sistema.

## Il mondo degli affari

Le aziende possono utilizzare il cloud computing in diversi modi. Alcuni utenti mantengono tutte le app e i dati sul cloud, mentre altri utilizzano un modello ibrido, mantenendo determinate app e dati su server privati e altri sul cloud.

Quando si tratta di fornire servizi, i grandi attori nella sfera dell'informatica aziendale includono:

- Google Cloud

- Servizi Web Amazon (AWS)

- Microsoft Azure

- IBM Cloud

- Alibaba nuvola

Amazon Web Services è pubblico al 100% e include un modello con pagamento in base al consumo in outsourcing. Una volta che sei sulla piattaforma puoi iscriverti ad app e servizi aggiuntivi. Microsoft Azure consente ai clienti di conservare alcuni dati nei propri siti. Nel frattempo, Alibaba Cloud è una filiale del gruppo Alibaba.

## Mette in risalto

- La sicurezza del cloud è diventata un campo sempre più importante nell'IT.

- Il cloud computing è la fornitura di diversi servizi tramite Internet, inclusi archiviazione dati, server, database, networking e software.

- L'archiviazione basata su cloud consente di salvare i file su un database remoto e recuperarli su richiesta.

- I servizi possono essere sia pubblici che privati: i servizi pubblici sono forniti online a pagamento mentre i servizi privati sono ospitati su una rete a clienti specifici.

- L'archiviazione su cloud è diventata sempre più popolare tra le persone che necessitano di uno spazio di archiviazione più ampio e per le aziende che cercano una soluzione efficiente per il backup dei dati fuori sede.

## FAQ

### Qual è un esempio di cloud computing?

Oggi esistono diversi esempi di applicazioni di cloud computing utilizzate sia dalle aziende che dai privati. Un tipo di servizio cloud sarebbe piattaforme di streaming per audio o video, in cui i file multimediali effettivi sono archiviati in remoto. Un altro sarebbero piattaforme di archiviazione dati come Google Drive, Dropbox, OneDrive o Box.

### Il cloud computing è sicuro?

Poiché software e dati sono archiviati in remoto nel cloud computing, la sicurezza dei dati e la sicurezza della piattaforma sono una grande preoccupazione. [La sicurezza del cloud](/cloud-security) si riferisce alle misure intraprese per proteggere le risorse digitali e i dati archiviati su servizi basati su cloud. Le misure per proteggere questi dati includono l'autorizzazione a due fattori (2FA), l'uso di VPN, token di sicurezza, crittografia dei dati e servizi firewall, tra gli altri.

### Quali sono i principali tipi di cloud computing?

I principali tipi di servizi di cloud computing includono Infrastructure-as-a-Service (IaaS), Platforms-as-a-Service (PaaS) e Software-as-a-Service (SaaS).- IaaS fornisce un'infrastruttura IT per terminare- utenti tramite Internet ed è comunemente associato all'elaborazione serverless.- PaaS offre sia software che hardware agli utenti finali, che generalmente sono sviluppatori di software. PaaS consente all'utente di sviluppare, eseguire e gestire le proprie app senza dover creare e mantenere l'infrastruttura.- SaaS è un modello di licenza software, che consente l'accesso al software in abbonamento utilizzando server esterni senza doverli scaricare e installarli localmente.
