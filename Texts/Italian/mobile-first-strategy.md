# Strategia mobile-first
## Che cos'è una strategia mobile-first?

L'espressione "strategia mobile first" si riferisce a un approccio allo sviluppo di siti Web in cui viene data la priorità alla versione mobile del sito Web rispetto alla versione desktop.

Le aziende stanno dando sempre più priorità alle versioni mobili delle loro pagine Web, a causa della crescente popolarità della navigazione in Internet tramite telefono e degli acquisti online tra i consumatori.

## Come funzionano le strategie Mobile First

Quando persegue una strategia mobile first, un'azienda rilascerà prima una versione mobile del proprio sito Web, prima di investire tempo o risorse considerevoli nello sviluppo di una versione desktop. Spesso, alcuni siti Web desktop rudimentali vengono rilasciati insieme alla versione mobile, ma questo sito Web desktop può consistere in poco più di una pagina di destinazione che si collega a un'applicazione mobile o a informazioni di base come una pagina delle domande frequenti (FAQ).

Ci sono diversi fattori alla base che guidano la popolarità delle strategie mobile-first. Per cominciare, la percentuale di traffico web attribuibile agli [smartphone](/smartphone) rispetto ai computer desktop è cresciuta costantemente nel tempo. Secondo StatCounter, la navigazione mobile era essenzialmente uguale alla navigazione desktop, ciascuna con una quota del 48% nel gennaio 2021. Perficient ha stimato che il 61% di tutto il traffico web statunitense proveniva da dispositivi mobili nel 2020.

Un altro fattore importante che incoraggia le aziende a perseguire una strategia mobile-first riguarda i dati e le funzionalità aggiuntivi che i siti Web mobili possono fornire. Poiché gli smartphone ora utilizzano in genere interfacce touch-screen, è possibile per i proprietari di siti Web abilitati ai dispositivi mobili ottenere dati dettagliati sull'esperienza utente (UX) relativi esattamente al modo in cui l'utente ha fisicamente cliccato o fatto scorrere la pagina. In alcuni casi, i proprietari di siti Web possono persino ottenere dati relativi ai movimenti oculari dell'utente, determinati utilizzando la fotocamera frontale del telefono.

Sebbene queste metriche avanzate dei dati utente siano relativamente rare, altri tipi di dati utente avanzati sono piuttosto comuni. Uno di questi esempi è il rilevamento della posizione, in cui le coordinate GPS dei telefoni degli utenti vengono automaticamente trasmesse al provider del sito Web. Questo tipo di informazioni può essere determinante per informare future [campagne di marketing](/marketing-strategy) o pianificare iniziative di sviluppo del business come l'apertura di nuovi negozi al dettaglio o magazzini di distribuzione.

## Esempio nel mondo reale di una strategia Mobile-First

Eppure, forse il fattore più significativo che guida l'adozione di strategie mobile-first è la strategia "mobile-first indexing" di Google. Questa decisione, messa in pratica nel luglio 2019, ha chiarito che, in futuro, l'algoritmo di ricerca dell'azienda avrebbe dato la priorità alle versioni mobili dei siti Web durante l'indicizzazione iniziale di un nuovo sito Web.

In pratica, ciò significa che è probabile che i nuovi siti Web sviluppati senza una strategia mobile-first producano risultati di ottimizzazione dei motori di ricerca (SEO) inferiori alla media. Considerando l'enorme influenza di cui gode Google nell'ecosistema globale dei motori di ricerca, sembra probabile che le strategie mobile-first saranno adottate come best practice per la maggior parte, se non per tutti, i nuovi progetti di sviluppo di siti Web in futuro.

## Mette in risalto

- In futuro, le strategie mobile first diventeranno probabilmente la norma, in parte a causa della decisione dei principali motori di ricerca di dare priorità alle versioni mobili dei siti web.

- Una strategia mobile first è quella in cui viene data la priorità alla versione mobile di un sito Web rispetto alla sua versione desktop.

- Questa pratica era relativamente rara in passato, ma è diventata sempre più comune.
