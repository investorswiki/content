# Tasso di esecuzione
## Qual è la velocità di esecuzione?

Il run rate si riferisce alla performance finanziaria di una società basata sull'utilizzo delle informazioni finanziarie correnti come predittore della performance futura. Il run rate funziona come un'estrapolazione della performance finanziaria corrente e presuppone che le condizioni attuali continueranno.

Il run rate può anche riferirsi alla diluizione media annua delle stock option assegnate dalle società nell'ultimo triennio registrato nel bilancio.

## Capire la velocità di esecuzione

Nel contesto dell'estrapolazione della performance futura, il run rate prende le informazioni sulla performance attuale e le estende su un periodo più lungo. Ad esempio, se un'azienda ha un [fatturato](/revenue) di 100 milioni di dollari nell'ultimo trimestre, il [CEO](/ceo) potrebbe dedurre che, in base all'ultimo trimestre, l'azienda opera a un tasso di esecuzione di 400 milioni di dollari. Quando i dati vengono utilizzati per creare una proiezione annuale per le prestazioni potenziali, il processo viene definito annualizzazione.

Un run rate può essere utile nella creazione di stime delle prestazioni per le aziende che hanno operato per brevi periodi di tempo, ad esempio meno di un anno, nonché per dipartimenti o centri di profitto di nuova creazione. Ciò può essere particolarmente vero per un'azienda che sta vivendo il suo primo trimestre redditizio. Inoltre, la velocità di esecuzione può essere utile nei casi in cui un'operazione aziendale fondamentale è stata modificata in un modo che si prevedeva avrebbe influenzato tutte le prestazioni future dell'attività associata.

## Rischi nell'utilizzo della velocità di esecuzione

La velocità di esecuzione può essere una metrica molto ingannevole, specialmente nei settori stagionali. Un ottimo esempio di ciò è un rivenditore che esamina il profitto dopo la stagione delle vacanze invernali, poiché questo è un momento in cui molti rivenditori registrano volumi di vendita più elevati. Se le informazioni basate sulle vendite delle festività natalizie sono state utilizzate per creare un tasso di esecuzione, le stime delle prestazioni future potrebbero essere accidentalmente gonfiate.

Inoltre, la velocità di esecuzione si basa generalmente solo sui dati più aggiornati e potrebbe non compensare adeguatamente le modifiche circostanziali che possono causare un quadro generale impreciso. Ad esempio, alcuni produttori di tecnologia come Apple e Microsoft registrano vendite più elevate in relazione al rilascio di un nuovo prodotto. L'utilizzo dei dati solo del periodo immediatamente successivo al rilascio di un prodotto di grandi dimensioni può causare dati distorti.

Inoltre, le tariffe di esecuzione non tengono conto delle grandi vendite una tantum. Ad esempio, se un produttore ottiene un contratto di grandi dimensioni che viene pagato in anticipo, indipendentemente dai tempi di consegna dei beni o servizi, ciò può causare un numero di vendite anormalmente elevato per un periodo di riferimento in base a questo acquisto anomalo.

### Quali sono alcuni svantaggi dell'utilizzo della velocità di esecuzione?

Il run rate può essere una metrica molto ingannevole, specialmente nei settori stagionali, dove le stime delle prestazioni future potrebbero essere gonfiate in modo errato. Inoltre, poiché generalmente si basa solo sui dati più aggiornati, potrebbe non compensare adeguatamente i cambiamenti circostanziali che possono causare un quadro generale impreciso. Inoltre, i tassi di esecuzione non tengono conto di eventi grandi e una tantum che possono distorcere le proiezioni.

### Come viene raggiunta la velocità di esecuzione?

In finanza, il run rate estrapola la performance attuale di un'impresa al fine di prevedere la performance futura partendo dal presupposto che le condizioni attuali persisteranno. Le proiezioni sono di solito per un anno intero, motivo per cui questo processo è talvolta chiamato "annualizzazione". Ad esempio, se un'azienda ha un fatturato di $ 100 milioni nell'ultimo trimestre, il CEO potrebbe dedurre che l'azienda opera a un tasso di esecuzione annualizzato di $ 400 milioni.
