# Ricerca dell'obiettivo
## Che cosa cerca l'obiettivo?

La ricerca dell'obiettivo è il processo per trovare il valore di input corretto quando si conosce solo l'output. La funzione di ricerca dell'obiettivo può essere incorporata in diversi tipi di programmi software per computer come Microsoft Excel.

## Comprendi la ricerca dell'obiettivo

La ricerca dell'obiettivo è un termine generico utilizzato per descrivere il processo coinvolto nella determinazione del valore di input sulla base di un valore di output già noto. Il processo prevede l'utilizzo di un operatore specifico in una formula, che può essere calcolata utilizzando un software per computer.

La ricerca degli obiettivi è uno degli strumenti utilizzati nell'"analisi what-if" sui programmi software per computer. Un'analisi [what -](/sensitivityanalysis) [if](/sensitivityanalysis) è un processo di modifica dei valori nelle celle (Microsoft Excel) per vedere come queste modifiche influiranno sui risultati delle formule nel foglio di lavoro. Quando si cerca un obiettivo, si esegue un'analisi di simulazione su un dato valore o sull'output. Quindi, in sostanza, creeresti uno scenario chiedendo "e se l'output fosse X" o, in pratica, una situazione di causa ed effetto.

Per alcuni dei problemi più complessi, le persone usano spesso il software del computer. Un programma di fogli di calcolo come Microsoft Excel ha uno strumento di ricerca degli obiettivi integrato. Consente all'utente di determinare il valore di input desiderato per una formula quando il valore di output è già noto. Questa funzione può aiutare l'utente a determinare cose come il [tasso di interesse per cui](/interestrate) un mutuatario deve qualificarsi (l'input) se solo sa quanto può permettersi di pagare ogni mese (l'output).

Ma c'è un avvertimento sull'utilizzo del software di ricerca degli obiettivi: funziona solo se è presente un valore di input. Se hai bisogno di capire due o più valori di input, non funzionerà. Se prendiamo l'esempio sopra, se vuoi calcolare l'importo totale del prestito e la rata mensile, il software di ricerca degli obiettivi non funzionerà. Probabilmente avresti bisogno di un componente aggiuntivo per capire più variabili.

### Come funziona la ricerca degli obiettivi in Microsoft Excel

Come accennato in precedenza, il software di ricerca degli obiettivi funzionerà solo se si conosce già il valore di output (o il risultato) ma si desidera determinare un valore di input. Di seguito sono riportati alcuni passaggi chiave che puoi seguire se desideri utilizzare la funzione di ricerca dell'obiettivo nel programma:

- Apri un nuovo foglio di calcolo

- Etichetta le tue colonne. In questo modo sarà più facile leggere tutto nel foglio di lavoro. Quindi, usando l'esempio sopra, la prima colonna sarà "Importo prestito", la seconda colonna sarebbe " [Durata](/term) in mesi", la terza sarebbe "Tasso di interesse" e l'ultima sarebbe "Pagamento".

- Digita i valori che conosci.

- Inserisci la formula per calcolare l'obiettivo, in questo caso il pagamento. Puoi ignorare il tasso di interesse per ora, che la formula presume sia 0 percento.

Ora dovrai determinare il tasso di interesse. Puoi farlo usando la funzione Goal Seek in Excel e inserisci i valori che hai già.

## Esempio di ricerca di un obiettivo

Un [imprenditore in cerca di obiettivi](/entrepreneur) è qualcuno che usa la ricerca di obiettivi per determinare come raggiungere i propri obiettivi finali. Ad esempio, un imprenditore potrebbe chiedere quanto dovrà guadagnare all'ora per incassare $ 100.000 in un anno. Conoscono il loro valore di output desiderato, $ 100.000, e dovranno, quindi, lavorare di nuovo per capire l'input ottimale. Ciò significa che dovranno determinare quante ore potranno (o sono disposti a) lavorare durante l'anno e, quindi, quanto guadagneranno per ogni ora per raggiungere il loro obiettivo.

## Mette in risalto

- Tuttavia, il software di ricerca dell'obiettivo funziona solo se è presente un solo valore di input.

- La ricerca dell'obiettivo consiste nel trovare l'input corretto quando si conosce solo l'output.

- Questo viene spesso fatto utilizzando la funzione di ricerca dell'obiettivo in Microsoft Excel o l'analisi what-if tramite programmi per computer.
