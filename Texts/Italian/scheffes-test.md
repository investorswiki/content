# Prova Scheffe
## Che cos'è un test Scheffé?

Un test di Scheffé è un test statistico che è un test post-hoc utilizzato nell'analisi statistica. Prende il nome dallo statistico americano Henry Scheffé. Il test di Scheffé viene utilizzato per effettuare confronti non pianificati, piuttosto che confronti pre-pianificati, tra le medie di gruppo in un esperimento [di analisi della varianza](/anova) (ANOVA).

Un confronto non pianificato è un confronto effettuato all'interno di un set di dati dopo che è stato eseguito un test ANOVA, quindi i parametri del confronto non sono incorporati nell'esperimento ANOVA. Il test di Scheffé può essere utilizzato per determinare se le singole medie differiscono o se una media di un gruppo di medie differisce dalla media di un altro gruppo di medie.

## Capire il test Scheffé

Mentre il test di Scheffé ha il vantaggio di dare allo sperimentatore la flessibilità per testare qualsiasi confronto che appare interessante, lo svantaggio di questa flessibilità è che il test ha una potenza statistica relativamente inferiore rispetto ai test progettati per confronti pre-pianificati.

Sebbene sia possibile effettuare confronti pre-pianificati utilizzando test come t-test o F-test, questi test non sono adatti per confronti post hoc o non pianificati. Per tali confronti, sono appropriati test di confronto multipli come il test di Scheffé, il metodo Tukey-Kramer o il [test di Bonferroni](/bonferroni-test).

## Mette in risalto

- Un test di Scheffé è una sorta di test di analisi statistica post-hoc utilizzato per effettuare confronti non pianificati.

- Il test prende il nome dallo statistico americano Henry Scheffé.

- Il test di Scheffé viene utilizzato per effettuare confronti non pianificati, piuttosto che confronti pre-pianificati, tra medie di gruppo in un esperimento di analisi della varianza (ANOVA).

- Il test di Scheffé ha il vantaggio di dare allo sperimentatore la flessibilità di testare eventuali confronti che appaiano interessanti.

- Uno svantaggio del test di Scheffé è che il test ha una potenza statistica relativamente inferiore rispetto ai test progettati per confronti pre-programmati.
