# Fascicolo del credito al consumo
## Che cos'è un file di credito al consumo?

Un file di credito al consumo è una raccolta di dati sull'attività di prestito e rimborso di un individuo. Il tuo file di credito contiene le informazioni che determinano il tuo [punteggio di credito](/credit_score). Quando richiedi un prestito automobilistico, ipotecario o di altro tipo, l'istituto finanziario controllerà il tuo file di credito per vedere se sembri avere un rischio di credito buono o cattivo. Puoi vedere cosa c'è nel tuo file di credito richiedendo il tuo rapporto di credito da ciascuna delle tre principali [agenzie di credito](/creditbureau).

## Comprensione dei file di credito al consumo

Un file di credito al consumo contiene le informazioni di identificazione di base, inclusi nome, numero di previdenza sociale, indirizzo e numero di telefono, insieme a qualsiasi altro nome, indirizzo e numero di telefono precedenti. A volte mostra anche i tuoi attuali ed ex datori di lavoro. Il file di credito mostra quali tipi di debito hai, che possono includere carte di credito, prestiti rateali, mutui e altro. Mostra chi ha chiesto informazioni sul tuo credito negli ultimi due anni e quando ha chiesto informazioni e contiene tutte le informazioni di credito negative come [fallimenti](/bankruptcy),. privilegi, sentenze e conti scaduti che sono stati inviati agli incassi.

La maggior parte del tuo file di credito al consumo è dedicato alle informazioni sui tuoi conti correnti e passati, incluso quando hai aperto ciascun conto, qual è stato il tuo saldo più alto, il tipo di conto (se si tratta di un conto individuale o cointestato), il saldo del conto, la data dell'ultimo pagamento e l'importo dell'ultimo pagamento. Per ogni account, il tuo file di credito mostra anche se hai effettuato pagamenti in tempo ogni mese, quanto sono stati ritardati i pagamenti e se il tuo account è mai stato insolvente.

I consumatori hanno tre file di credito, uno con ciascuna delle tre principali agenzie di credito: Experian, Equifax e TransUnion. A volte tutti e tre i file contengono informazioni identiche, ma a volte un file conterrà un account che un altro file non contiene. Alcuni istituti di credito e creditori non segnalano l'attività di prestito e rimborso dei propri clienti a tutti e tre gli uffici, il che crea differenze tra i file di credito dello stesso consumatore.

## File di credito determina il punteggio di credito

A volte, i consumatori confondono i termini file di credito e punteggio di credito o li usano in modo intercambiabile. Un modo per pensarci è che le informazioni nel tuo file di credito determinano il tuo punteggio di credito. Il punteggio di credito stesso è un numero statistico basato su un algoritmo che misura il tuo rischio di credito utilizzando le informazioni nel tuo file di credito. Molti consumatori sanno che i creditori e gli istituti di credito utilizzano il punteggio di credito per determinare se concedere o meno un credito al consumo. Ma è anche utile sapere che il tuo punteggio di credito viene spesso utilizzato per determinare i termini che ti vengono offerti o il tasso di interesse che pagherai per un prestito. Di solito, maggiore è il tuo punteggio di credito, minore è il tasso di interesse che dovrai pagare.

## Blocco dei file di credito al consumo

Gli hack presso le agenzie di credito al consumo hanno attirato l'attenzione sui pericoli della condivisione dei dati dal tuo file di credito. Tali hack spesso finiscono con i criminali che ottengono l'accesso a preziosi dati personali e finanziari per i clienti. Le informazioni vengono quindi vendute ad altri personaggi nefasti, che le usano per accumulare un forte debito o estraggono più denaro dalle vittime. Un esempio recente è stata la violazione dei dati presso Equifax, una delle tre agenzie di credito al consumo che conservano file di credito, che ha compromesso informazioni personali, come numeri di previdenza sociale e date di nascita, relative a 147 milioni di americani .

Uno dei modi per evitare il furto di dati personali è bloccare i file di credito. Un blocco del credito è anche noto come blocco della sicurezza e impedisce ai criminali di accedere al tuo file "bloccando" completamente l'accesso ad esso. Il blocco si estende ai nuovi creditori e ad altri agenti, rendendo loro impossibile l'accesso al tuo file a meno che tu non fornisca esplicitamente l'autorizzazione a farlo.

Tuttavia, l'accesso al tuo file di credito è disponibile per il titolare del credito, i precedenti creditori e gli esattori. Un blocco della sicurezza non impedisce ai titolari di crediti di cambiare lavoro, affittare appartamenti o acquistare assicurazioni. Sono tenuti a revocare il blocco contattando gli uffici. Se la richiesta viene effettuata per telefono o e-mail, gli uffici dovrebbero revocare il blocco entro un'ora. Se la richiesta viene effettuata per posta, le agenzie di credito devono revocare il blocco entro tre giorni lavorativi dalla ricezione della richiesta .
