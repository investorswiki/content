# Linux
## Cos'è Linux?

Linux è un popolare sistema operativo (OS) open source altamente personalizzabile. Facilita la comunicazione tra software e hardware, alimentando gran parte del mondo di oggi, dai computer alle automobili.

Creato nel 1991 da Linus Torvalds, Linux è un software open source che ora fa parte di un'ampia varietà di sistemi. In quanto open-source, Linux è disponibile gratuitamente sia per uso commerciale che non commerciale. Chiunque può utilizzare, modificare e distribuire il codice sorgente secondo i termini delle sue licenze.

Grazie alla sua flessibilità, Linux è presente in un'ampia gamma di applicazioni, inclusi computer desktop, laptop, server Web, [supercomputer](/supercomputer),. smartphone, televisori, router, fotocamere digitali, strumenti musicali e veicoli.

Tecnicamente parlando, Linux è un kernel: un programma per computer che è l'elemento centrale di un sistema operativo. In poche parole, il kernel è ciò che traduce il codice del computer in dati binari che l'hardware può comprendere.

L'uso del solo kernel Linux non assomiglia per niente all'esperienza del computer a cui la maggior parte delle persone è abituata. Quindi gli sviluppatori di tutto il mondo hanno aggiunto componenti e applicazioni aggiuntivi per renderlo più intuitivo. Pertanto, Linux è solitamente distribuito come pacchetti (raccolte di strumenti software). Queste versioni modificate di Linux sono chiamate distro (abbreviazione di Linux Distributions).

### Quali sono i tipi di distribuzioni Linux?

Nel tempo, le comunità e le aziende hanno lavorato con Linux per creare sistemi adatti alle loro preferenze o esigenze. C'è una lunga lista di diverse distribuzioni Linux, da tutto il mondo. Mentre alcuni sono interamente basati sulla comunità (come Debian o Arch Linux), altri hanno il supporto aziendale (come Fedora o openSUSE). Diverse distribuzioni mirano a fare cose diverse. Ad esempio, possono essere incentrati sulla privacy (TAILS), sull'usabilità (Ubuntu) o sulla personalizzazione (Arch).

### Perché qualcuno dovrebbe voler usare Linux?

Nel complesso, Linux è un sistema versatile, efficiente e sicuro. Può essere personalizzato in base alle preferenze dell'utente. La sua infrastruttura leggera consente velocità del computer più elevate e può anche essere utilizzata per rivitalizzare una vecchia macchina. Le numerose distribuzioni Linux consentono agli utenti di scegliere un sistema che soddisfi le loro esigenze, che si tratti di semplicità, sicurezza, privacy o personalizzazione avanzata.
