# Teoria della coda
## Che cos'è la teoria delle code?

La teoria delle code è una branca della matematica che studia come si formano le linee, come funzionano e perché non funzionano correttamente. La teoria delle code esamina ogni componente dell'attesa in fila, inclusi il processo di arrivo, il processo di servizio, il numero di server, il numero di posti nel sistema e il numero di clienti, che potrebbero essere persone, pacchetti di dati, automobili o qualsiasi altra cosa.

Le applicazioni reali della teoria delle code coprono un'ampia gamma di attività. I suoi risultati possono essere utilizzati per fornire un servizio clienti più rapido, aumentare il flusso di traffico, migliorare le spedizioni degli ordini da un magazzino o progettare reti di dati e call center.

> - Lo studio della congestione e delle sue cause in un processo viene utilizzato per contribuire a creare servizi e sistemi più efficienti ed economici.

> - Spesso utilizzata come strumento di gestione delle operazioni, la teoria delle code può affrontare le carenze di personale, pianificazione e servizio clienti.

> - La teoria delle code mira a raggiungere un equilibrio efficiente e conveniente.

## Come funziona la teoria delle code

Le code possono verificarsi ogni volta che le risorse sono limitate. Alcune code sono tollerabili in qualsiasi attività, poiché la totale assenza di una coda suggerirebbe una costosa [sovraccapacità](/excesscapacity).

La teoria delle code mira a progettare sistemi bilanciati che servono i clienti in modo rapido ed efficiente ma non costano troppo per essere sostenibili.

Al suo livello più elementare, la teoria delle code implica un'analisi degli arrivi in una struttura, come una banca o un fast-food, e un'analisi dei processi attualmente in atto per servirli. Il risultato finale è una serie di conclusioni che mirano a identificare eventuali difetti nel sistema e suggerire come possono essere migliorati.

L'origine della teoria delle code può essere fatta risalire all'inizio del 1900 in uno studio della centrale telefonica di Copenaghen di Agner Krarup Erlang, un ingegnere, statistico e matematico danese. Il suo lavoro ha portato alla teoria di Erlang delle reti efficienti e al campo dell'analisi delle reti telefoniche.

Ancora oggi, l'unità fondamentale del traffico di telecomunicazioni nei sistemi vocali è chiamata "erlang".

### I parametri di una coda

Nella teoria delle code, il processo studiato è suddiviso in sei parametri distinti. Questi includono il processo di arrivo, il servizio e il processo di partenza, il numero di server, la disciplina di accodamento (come first-in, first-out), la capacità della coda e la dimensione della popolazione dei client.

> Le code non sono necessariamente un aspetto negativo di un'azienda, in quanto la loro assenza suggerisce un eccesso di capacità.

>

## Vantaggi della teoria delle code

La teoria delle code come tecnica di gestione delle operazioni viene comunemente utilizzata per determinare e semplificare le esigenze di personale, la pianificazione e l'inventario al fine di migliorare il servizio clienti generale. Viene spesso utilizzato dai professionisti [Six Sigma](/six-sigma) per migliorare i processi.

### La psicologia della coda

La psicologia delle code è legata alla teoria delle code. Questa è la componente della fila che si occupa della naturale irritazione provata da molte persone che sono costrette a fare la fila per il servizio, sia che stiano aspettando di fare il check-out al supermercato o che stiano aspettando il caricamento di un sito web.

Un'opzione di richiamata in attesa di parlare al telefono con un rappresentante del cliente è un esempio di soluzione all'impazienza del cliente. Un esempio più antiquato è il sistema utilizzato da molte gastronomie, che emettono numeri di servizio clienti per consentire alle persone di tenere traccia dei propri progressi in prima fila.

## Esempio di teoria delle code

Un articolo del professor Lawrence Wein della Stanford Graduate School of Business et al. ha utilizzato la teoria delle code per analizzare una varietà di possibili risposte di emergenza a un attacco di bioterrorismo aereo in un luogo pubblico. Il modello ha indicato azioni specifiche che potrebbero essere intraprese per ridurre i tempi di attesa per le cure di emergenza, diminuendo così il numero potenziale di decessi.

La teoria delle code è utile, se non così urgente, per guidare la logistica di molte aziende. Ad esempio, è probabile che il reparto operazioni di una società di consegna utilizzi la teoria delle code per appianare i nodi nei suoi sistemi per spostare i pacchi da un magazzino a un cliente. In questo caso la "linea" oggetto di studio è composta da scatole di merce in attesa di essere consegnate ai clienti.

Applicando la teoria delle code, un'azienda può sviluppare sistemi, processi, meccanismi di determinazione dei prezzi, soluzioni per il personale e strategie di gestione degli arrivi più efficienti per ridurre i tempi di attesa dei clienti e aumentare il numero di clienti che possono essere serviti.

## Domande frequenti sulla teoria delle code

Ecco le risposte ad alcune domande frequenti sulla teoria delle code.

### Come usi la teoria delle code?

La teoria delle code viene utilizzata per identificare e correggere i punti di congestione in un processo. La coda può essere composta da persone, cose o informazioni. In ogni caso, sono costretti ad aspettare il servizio. Questo è inefficiente, dannoso per gli affari e fastidioso (quando la coda è composta da persone).

La teoria delle code viene utilizzata per analizzare il processo esistente e mappare le alternative con un risultato migliore.

### Chi ha inventato la teoria delle code?

Agner Krarup Erlang, un matematico, statistico e ingegnere danese, è accreditato di aver creato non solo la teoria delle code ma l'intero campo dell'ingegneria del traffico telefonico.

All'inizio del XX secolo, Erlang era a capo di un laboratorio tecnico presso la Copenhagen Telephone Co. I suoi studi approfonditi sui tempi di attesa nei servizi telefonici automatizzati e le sue proposte per reti più efficienti furono ampiamente adottati dalle compagnie telefoniche.

### Quali sono gli elementi di base della teoria delle code?

Uno studio di una linea utilizzando la teoria delle code la suddividerebbe in sei elementi: il processo di arrivo, il servizio e il processo di partenza, il numero di server disponibili, la disciplina delle code (come first-in, first-out), la capacità della coda e i numeri serviti. La creazione di un modello dell'intero processo dall'inizio alla fine consente di identificare e affrontare la causa o le cause della congestione.

### Cosa significa essere in coda?

Gli americani stanno in fila per il servizio (ad eccezione dei newyorkesi, che stanno "in linea"). Gli inglesi fanno la fila. La parola coda deriva da un antico sostantivo francese per la coda di un animale.

L'era dei computer ha introdotto un nuovo utilizzo. Un provider di posta elettronica potrebbe indicare che il tuo messaggio è stato "in coda". Ciò significa che c'è un ritardo nella consegna ma verrà inviato il prima possibile.

Accodamento e accodamento sono entrambe grafie accettabili della parola.
