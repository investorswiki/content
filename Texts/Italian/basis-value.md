# Valore di base
## Qual è il valore di base?

Il valore base è il prezzo di [un'immobilizzazione](/fixedasset) ai fini fiscali. Il valore di un'immobilizzazione può essere rettificato per aiutare le aziende a sfruttare i vantaggi fiscali come indicato [dall'Internal Revenue Service](/irs) (IRS). In altre parole, il valore di base aiuta a ridurre il carico fiscale di una società sull'attività quando l'attività viene venduta.

## Come calcolare il valore di base

Poiché le normative IRS e il codice fiscale variano da settore a settore, è meglio contattare un commercialista o l'IRS per determinare come deve essere riportato il valore base delle immobilizzazioni di una società. Potrebbero esserci normative specifiche per il settore di una società in base alle quali una società potrebbe essere ammissibile a crediti d'imposta o detrazioni fiscali che modificano il valore base di un'attività.

Di conseguenza, di tali deduzioni o crediti, l'utile o la perdita imponibile sulla vendita dell'attività sarà impattata. Ad esempio, le attività immobiliari hanno spese diverse che possono aumentare o diminuire il valore base rispetto alle immobilizzazioni come macchinari o attrezzature.

In generale, il calcolo del valore base potrebbe iniziare con il prezzo di acquisto originario dell'attività. Da lì, puoi aumentare la base aggiungendo eventuali importi spesi per migliorare il bene o eventuali spese legali o costi di vendita associati al bene. È probabile che il valore di base diminuisca se devi sottrarre importi che hai precedentemente richiesto come detrazioni fiscali, come ammortamenti, perdite per incidenti o perdite per furto.

## Cosa ti dice il valore di base?

Il valore di base è particolarmente importante quando si tratta della dismissione di un'attività poiché le plusvalenze e le eventuali imposte risultanti sono guidate dal valore di base. A parità di prezzo di vendita, maggiore è il valore base e di conseguenza il valore contabile ammortizzato, minore è la [plusvalenza imponibile](/capital_gains_tax). Il valore base viene utilizzato anche come prezzo base di un'attività su cui vengono calcolati l'ammortamento.

Quando un'azienda vende un bene fisso, idealmente vorrebbe guadagnare quanto più denaro possibile dalla vendita. Tuttavia, dal punto di vista fiscale, la società cerca modi per ridurre l'eventuale plusvalenza derivante dalla vendita del bene perché la plusvalenza è imponibile.

Il valore base costituisce il prezzo base per un cespite a cui è possibile aggiungere le spese capitalizzate. Le spese capitalizzate potrebbero includere i costi di installazione o costruzione del bene. L'aggiunta delle spese capitalizzate al valore base aumenta il valore del bene e diminuisce la plusvalenza sulla vendita del bene.

Come affermato in precedenza, diversi tipi di spese e attività nel corso della vita del bene possono aumentare o diminuire il valore base e, in definitiva, l'onere fiscale derivante dalla vendita del bene. Se una società sta costruendo un bene, i costi associati alla costruzione potrebbero essere aggiunti al valore base. I costi che aumentano il valore di base potrebbero includere la manodopera, i materiali e le spese di autorizzazione per la costruzione dell'attività. Gli articoli che potrebbero diminuire il valore base potrebbero includere eventuali detrazioni fiscali, crediti di investimento o eventuali sconti ai produttori.

## Un esempio di valore di base

La società A ha un cespite per il quale le spese capitalizzate erano di $ 50.000 e il bene ha un valore contabile (dopo l'ammortamento) di $ 100.000 dopo cinque anni.

- Il valore base è il valore contabile di $ 100.000 più $ 50.000 di spese capitalizzate o $ 150.000.

- Se l'attività viene successivamente venduta per $ 130.000, si verifica una perdita sulla vendita di $ 20.000 o ($ 150.000 - $ 130.000).

Tuttavia, una registrazione impropria delle spese può portare a inesattezze e ad un pagamento eccessivo delle tasse.

- Nel nostro esempio sopra, supponiamo che la società A non sia riuscita a registrare i $ 50.000 di spese capitalizzate per l'asset. In altre parole, il valore base è uguale al valore contabile di $ 100.000 invece di $ 150.000.

- Se l'attività viene venduta per $ 130.000, il guadagno imponibile è ora di $ 30.000 o ($ 130.000 - $ 100.000).

Poiché le spese capitalizzate non sono state registrate correttamente, la società ha pagato le tasse su una plusvalenza di $ 30.000 dalla vendita del bene invece di poter eventualmente cancellare una perdita.

## La differenza tra il valore di base e il valore di mercato

Il [valore equo di mercato](/fairmarketvalue) di un'attività non deve essere confuso con il valore base. Il valore equo di mercato di un'impresa o di un bene è la stima del prezzo che verrebbe pagato al proprietario in caso di vendita. La formula per determinare un valore equo di mercato include il valore aziendale e le attività nei mercati finanziari attuali.

Determinare il valore equo di mercato può essere difficile poiché l'unico modo per dimostrare il vero valore è vendere l'attività o l'attività. Il valore base, invece, è il prezzo base di un'immobilizzazione a cui vengono aggiunte le spese capitalizzate e fornisce il valore della plusvalenza imponibile derivante dalla vendita di un bene.

## Limitazioni dell'utilizzo del valore di base

Mentre i dipartimenti contabili delle grandi aziende seguono da vicino i valori di base delle loro attività fisse, le piccole società con risorse limitate come i contabili a tempo pieno possono affrontare difficoltà nel garantire che il valore di base delle loro attività sia accurato.

Un'altra limitazione nel calcolo del valore base può derivare dalla necessità di stare al passo con l'evoluzione delle leggi fiscali. Se i contabili di una società calcolano in modo impreciso il valore delle attività, il valore base e i calcoli fiscali risultanti saranno errati.
