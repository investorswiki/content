# Statistiche descrittive
## Cosa sono le statistiche descrittive?

Le statistiche descrittive sono brevi coefficienti descrittivi che riassumono un dato set di dati, che può essere una rappresentazione dell'intera popolazione o un campione di una popolazione. Le statistiche descrittive sono scomposte in misure di tendenza centrale e misure di variabilità (spread). Le misure della tendenza centrale includono la media, la mediana e la moda, mentre le misure della variabilità includono la deviazione standard, la varianza, le variabili minime e massime, [la curtosi](/kurtosis) e l' [asimmetria](/skewness).

## Comprendere le statistiche descrittive

Le statistiche descrittive, in breve, aiutano a descrivere e comprendere le caratteristiche di uno specifico set di dati fornendo brevi riepiloghi sul campione e sulle misure dei dati. I tipi più riconosciuti di statistica descrittiva sono le misure del centro: la [media](/mean),. la [mediana](/median) e la [moda](/mode),. che vengono utilizzate a quasi tutti i livelli di matematica e statistica. La media, o media, viene calcolata sommando tutte le cifre all'interno dell'insieme di dati e quindi dividendo per il numero di cifre all'interno dell'insieme.

Ad esempio, la somma del seguente set di dati è 20: (2, 3, 4, 5, 6). La media è 4 (20/5). La modalità di un set di dati è il valore che appare più spesso e la mediana è la cifra situata al centro del set di dati. È la cifra che separa le cifre più alte dalle cifre più basse all'interno di un set di dati. Tuttavia, ci sono tipi meno comuni di statistiche descrittive che sono ancora molto importanti.

Le persone utilizzano statistiche descrittive per riutilizzare informazioni quantitative di difficile comprensione su un ampio set di dati in descrizioni di dimensioni ridotte. La media dei voti di uno studente (GPA), ad esempio, fornisce una buona comprensione delle statistiche descrittive. L'idea di un GPA è che prende i punti dati da un'ampia gamma di esami, classi e voti e ne fa la media insieme per fornire una comprensione generale del rendimento scolastico complessivo di uno studente. Il GPA personale di uno studente riflette il suo rendimento scolastico medio.

## Tipi di statistiche descrittive

Tutte le statistiche descrittive sono misure di tendenza centrale o misure di [variabilità ,](/variability) [note](/variability) anche come misure di dispersione.

### Tendenza centrale

Le misure di tendenza centrale si concentrano sui valori medi o medi dei set di dati, mentre le misure di variabilità si concentrano sulla dispersione dei dati. Queste due misure utilizzano grafici, tabelle e discussioni generali per aiutare le persone a comprendere il significato dei dati analizzati.

Le misure di tendenza centrale descrivono la posizione centrale di una distribuzione per un set di dati. Una persona analizza la frequenza di ciascun punto dati nella distribuzione e la descrive utilizzando la media, la mediana o la modalità, che misura i modelli più comuni del set di dati analizzato.

### Misure di variabilità

Le misure di variabilità (o le misure di diffusione) aiutano ad analizzare quanto è dispersa la distribuzione per un insieme di dati. Ad esempio, mentre le misure di tendenza centrale possono fornire a una persona la media di un set di dati, non descrivono come i dati sono distribuiti all'interno dell'insieme.

Quindi, mentre la media dei dati potrebbe essere 65 su 100, possono ancora esserci punti dati sia a 1 che a 100. Le misure di variabilità aiutano a comunicarlo descrivendo la forma e la diffusione del set di dati. Intervallo, [quartili](/quartile),. deviazione assoluta e varianza sono tutti esempi di misure di variabilità.

Considera il seguente set di dati: 5, 19, 24, 62, 91, 100. L'intervallo di quel set di dati è 95, che viene calcolato sottraendo il numero più basso (5) nel set di dati dal più alto (100).

## Mette in risalto

- La statistica descrittiva è costituita da due categorie fondamentali di misure: misure di tendenza centrale e misure di variabilità (o diffusione).

- La statistica descrittiva riassume o descrive le caratteristiche di un set di dati.

- Le misure di variabilità o diffusione descrivono la dispersione dei dati all'interno dell'insieme.

- Le misure di tendenza centrale descrivono il centro di un insieme di dati.

## FAQ

### Le statistiche descrittive possono essere utilizzate per fare inferenze o previsioni?

No. Sebbene questi descrittivi aiutino a comprendere gli attributi dei dati, le tecniche statistiche inferenziali, un ramo separato della statistica, sono necessarie per comprendere come le variabili interagiscono tra loro in un set di dati.

### Cosa sono media e deviazione standard?

Queste sono due statistiche descrittive comunemente utilizzate. La media è il livello medio osservato in alcuni dati, mentre la deviazione standard descrive la varianza o la dispersione dei dati osservati in quella variabile attorno alla sua media.

### Perché abbiamo bisogno di statistiche che descrivono semplicemente i dati?

Le statistiche descrittive vengono utilizzate per descrivere o riassumere le caratteristiche di un campione o di un set di dati, come la media, la deviazione standard o la frequenza di una variabile. Le statistiche inferenziali possono aiutarci a comprendere le proprietà collettive degli elementi di un campione di dati. Conoscere la media campionaria, la varianza e la distribuzione di una variabile può aiutarci a capire il mondo che ci circonda.
