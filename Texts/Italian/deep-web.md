# Deep Web
Il deep web è la sezione del World Wide Web (WWW) che è in qualche modo nascosta. Contiene pagine che non sono indicizzate dai tradizionali motori di ricerca web, come Google, Bing e DuckDuckGo.

Il deep web è il termine opposto del "surface web". In quanto tale, può anche essere indicato come "rete nascosta" o "rete invisibile".

## Oltre la superficie

Il deep web rappresenta oltre il 99% dei contenuti del web. È una parte del WWW diversa dal cosiddetto “web di superficie”, che è la parte visibile del web. La rete di superficie è lo strato più facilmente accessibile. Include tutti gli indici (o le pagine ricercabili).

Sebbene la quantità di contenuti rilevabili attraverso i motori di ricerca sia vasta, il web di superficie contiene molte meno informazioni rispetto al deep web. Uno studio pubblicato nel 2001 ha indicato che meno dello 0,05% di tutto il contenuto è disponibile nel web di superficie, mentre l'altro ~99,95% si trova nel web profondo.

## Cos'è il Deep Web?

Per definizione, il deep web include entrambe le pagine oscure o accessibili solo tramite uno specifico metodo di autenticazione. Essere oscuri significa che richiedono un URL diretto per essere trovati (la ricerca su Google non funziona). Tuttavia, la maggior parte dei contenuti del deep web rientra in quest'ultima categoria.

In tal senso, possiamo considerare come parte del deep web qualsiasi sito Web che richiede un accesso e una password. In altre parole, molte delle pagine a cui le persone accedono ogni giorno fanno effettivamente parte del deep web. Ciò include i profili dei social media, le caselle di posta elettronica e i conti bancari.

Questo può sembrare confuso all'inizio perché la terminologia popolare dice che il deep web è il web oscuro. Ma poiché i nostri account personali non sono indicizzati nelle pagine dei motori di ricerca, sono anche considerati parte del deep web.

Tuttavia, il pezzo più grande del deep web è costituito da statistiche e database privati. Ad esempio, le agenzie governative statunitensi come la [Commodity Futures Trading Commission](/cftc) [(CFTC)](/cftc),. la National Oceanic and Atmospheric Administration (NOAA) e la [Securities and Exchange Commission (SEC)](/sec) rappresentano una quantità significativa di questi dati digitali.

## Deep web e dark web

È anche comune vedere confusione riguardo alle differenze tra il deep web e il dark web. In breve, il dark web è una porzione del deep web che è anonima.

Pertanto, è spesso associato a siti Web che facilitano attività illegali o che forniscono vantaggi in termini di privacy per attivisti e informatori (inclusi [hacker black hat](/hacker) e hacktivist). Si stima che il dark web rappresenti meno dello 0,01% del deep web ed è accessibile solo tramite piattaforme specializzate, come Tor.

## Mette in risalto

- Forse la critica più seria al deep web è che mina l'apertura e l'uguaglianza di Internet.

- Il deep web si riferisce a parti di Internet non completamente accessibili tramite motori di ricerca standard come Google, Yahoo e Bing.

- Il deep web offre agli utenti l'accesso a molte più informazioni di quante sarebbero altrimenti disponibili su Internet e aumenta anche la privacy.

- Il deep web include pagine non indicizzate, siti a pagamento, database privati e dark web.

## FAQ

### Deep Web e Dark Web sono la stessa cosa?

Il dark web è una piccola componente del deep web. Il deep web si riferisce a tutta la rete Internet che non è facilmente accessibile o indicizzata dai motori di ricerca (che è chiamata "web di superficie"). transazioni illecite online.

### È illegale accedere al Deep Web?

No. Il semplice accesso a siti non indicizzati o comunque pubblicamente disponibili non è illegale. Tuttavia, potrebbe essere illegale utilizzare in modo improprio o rubare informazioni trovate su siti Web profondi.

### Come posso trovare siti sul Deep Web?

Poiché il deep web non è completamente indicizzato con i motori di ricerca tradizionali, spesso l'unico modo per trovare tali siti è conoscere l'esatto indirizzo Internet da utilizzare. Ci sono alcuni siti web specializzati o motori di ricerca che catalogano alcuni siti web profondi. Ad esempio, le risorse accademiche sul deep web possono essere trovate utilizzando piattaforme come PubMed, LexisNexis, Web of Science o Project MUSE.
