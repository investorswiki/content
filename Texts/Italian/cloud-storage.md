# Archiviazione su cloud
## Che cos'è l'archiviazione su cloud?

Il cloud storage è un modo per le aziende e i consumatori di salvare i dati in modo sicuro online in modo che sia possibile accedervi in qualsiasi momento da qualsiasi luogo e condividerli facilmente con coloro a cui è stata concessa l'autorizzazione. L'archiviazione su cloud offre anche un modo per eseguire il backup dei dati per facilitare il ripristino fuori sede.

Oggi, le persone hanno accesso a diversi servizi di cloud computing gratuiti come Google Drive, Dropbox e Box, tutti dotati di pacchetti di abbonamento aggiornati che offrono dimensioni di archiviazione maggiori e servizi cloud aggiuntivi.

## Spiegazione dell'archiviazione nel cloud

Il cloud storage offre un modo semplice per archiviare e/o spostare i dati in modo sicuro e protetto. Consente a privati e aziende di conservare i propri file archiviati presso il provider di servizi cloud per l'accesso su richiesta su qualsiasi dispositivo. L'archiviazione cloud può essere utilizzata anche per archiviare dati che richiedono archiviazione a lungo termine ma non è necessario accedervi frequentemente, come determinati record finanziari. Sempre più spesso, i file archiviati "nel cloud" vengono utilizzati per la collaborazione di gruppo.

Il cloud storage funziona consentendo a un computer client, tablet o smartphone di inviare e recuperare file online da e verso un server di dati remoto. Gli stessi dati vengono generalmente archiviati su più server contemporaneamente in modo che i client possano sempre accedere ai propri dati anche se un server è inattivo o perde dati. Ad esempio, il proprietario di un laptop potrebbe archiviare foto personali sia sul proprio disco rigido che nel cloud in caso di furto del laptop.

Un sistema di archiviazione cloud può essere specializzato nell'archiviazione di un particolare tipo di dati come foto digitali o file musicali, oppure può fornire l'archiviazione generale di qualsiasi tipo di dati come foto, file audio, documenti di testo, presentazioni e fogli di calcolo.

Si ritiene che il cloud storage sia stato inventato dallo scienziato informatico Dr. Joseph Carl Robnett Licklider negli anni '60. Circa due decenni dopo, CompuServe iniziò a offrire ai propri clienti piccole quantità di spazio su disco per archiviare alcuni dei loro file. A metà degli anni '90, AT&T ha lanciato il primo servizio di archiviazione interamente basato sul Web per la comunicazione personale e aziendale. Da allora, una serie di servizi diversi hanno preso piede. Alcuni dei provider di archiviazione cloud più popolari sono Apple (iCloud), Amazon (Amazon Web Services), Dropbox e Google.

## In che modo il cloud storage avvantaggia le aziende

Il cloud storage aiuta le aziende con importanti esigenze di storage dei dati a risparmiare una notevole quantità di spazio e denaro, eliminando la necessità di un'infrastruttura di storage dei dati nei locali aziendali. Il provider di archiviazione cloud possiede e mantiene tutto l'hardware e il software necessari in modo che gli utenti cloud non debbano farlo. L'acquisto di spazio di archiviazione cloud in corso può costare di più a lungo termine, ma può essere significativamente meno costoso in anticipo. Inoltre, le aziende possono aumentare o diminuire quasi istantaneamente la quantità di spazio di archiviazione cloud a cui hanno accesso man mano che le loro esigenze di archiviazione cambiano.

Il cloud consente inoltre ai dipendenti di collaborare con i colleghi e di lavorare in remoto e al di fuori del normale orario lavorativo, facilitando al contempo una collaborazione agevole sui documenti consentendo ai dipendenti autorizzati un facile accesso alla versione più aggiornata di un file. A livello personale, l'archiviazione su cloud consente dati mobili e consente la vita digitale nel modo olistico in cui la viviamo oggi. Senza il cloud, gli smartphone non potrebbero essere l'interfaccia di tanti dati (foto, documenti, informazioni in movimento). L'uso del cloud per archiviare i file può anche avere un effetto positivo sull'ambiente poiché riduce il consumo di energia

## Sicurezza dell'archiviazione nel cloud

C'è così tanta attenzione sul cloud storage oggi nell'era digitale perché gran parte dei nostri dati personali sensibili sono archiviati nel cloud sia che li archiviamo volontariamente lì o se un'azienda con cui facciamo affari decide di archiviarli lì. Di conseguenza, [la sicurezza del cloud](/cloud-security) è diventata una delle principali preoccupazioni. Gli utenti si chiedono se le loro informazioni siano al sicuro e le crescenti [violazioni dei dati](/data-breach) hanno dimostrato che a volte non lo è. Gli utenti sono anche preoccupati per il fatto che i dati che hanno archiviato sul cloud saranno accessibili quando ne avranno bisogno.

Sebbene l'archiviazione su cloud possa sembrare vulnerabile a causa della prevalenza [dell'hacking ,](/cybersecurity) [anche](/cybersecurity) le alternative, come l'archiviazione in loco, presentano vulnerabilità di sicurezza. L'archiviazione cloud fornita dall'azienda può effettivamente migliorare la sicurezza offrendo ai dipendenti un'alternativa all'utilizzo dei propri account personali per eseguire il backup e trasferire i file a cui devono accedere fuori dall'ufficio.

Un buon provider di archiviazione cloud avrà la ridondanza dei dati, archiviando gli stessi file in più posizioni fisiche in modo che sopravviva a errori umani, guasti alle apparecchiature o disastri naturali. Un provider affidabile memorizzerà e trasmetterà i dati in modo sicuro in modo che nessuno possa accedervi senza autorizzazione. Alcuni utenti potrebbero anche richiedere che i dati siano archiviati in modo tale da poter essere solo letti ma non modificati; anche questa funzionalità è disponibile tramite il cloud storage.

## Mette in risalto

- L'archiviazione su cloud è diventata sempre più popolare tra le persone che necessitano di uno spazio di archiviazione più ampio e per le aziende che cercano una soluzione efficiente per il backup dei dati fuori sede.

- L'archiviazione su cloud consente a privati e aziende di archiviare e recuperare i file del computer tramite un dispositivo connesso a Internet.

- A causa della crescente popolarità e utilizzo dell'archiviazione cloud, la sicurezza del cloud è diventata una delle principali preoccupazioni per proteggere l'integrità dei dati, prevenire tentativi di hacking ed evitare il furto di file o identità.
