# Esegue il test
## Che cos'è un test di esecuzione?

Un test di esecuzione è una procedura statistica che esamina se una [stringa](/data-analytics) [di dati](/data-analytics) si verifica in modo casuale da una [distribuzione specifica](/normaldistribution). Il test di esecuzione analizza il verificarsi di eventi simili separati da eventi diversi.

Nell'investire, un test di esecuzione può essere importante per gli investitori per determinare se il set di dati che stanno utilizzando è generato casualmente o se è influenzato da una variabile sottostante. I trader che si concentrano sull'analisi tecnica possono utilizzare un run test per analizzare l'andamento dei [prezzi](/price-action) di un titolo.

## Comprensione di un test di esecuzione

Una corsa è una serie di valori crescenti o decrescenti, spesso rappresentati su un grafico da simboli più (+) o meno (-). Nelle statistiche, un test di esecuzione aiuta a determinare la [casualità dei dati](/random-factor-analysis) scoprendo eventuali variabili che potrebbero influire sui modelli di dati.

Ad esempio, un elenco di numeri a una cifra veramente casuali dovrebbe avere solo pochi casi in cui una sequenza di numeri è crescente numericamente. Tuttavia, in molti casi, è difficile affermare la casualità dei dati in cui ci sono migliaia di sequenze in una stringa di dati. Pertanto, il test delle corse è stato creato come [metodo oggettivo per](/objective-probability) determinare la casualità.

## Tipi di prove di esecuzione

Il test di corsa è una versione abbreviata del nome completo: il test di corsa Wald–Wolfowitz, così chiamato in onore dei matematici Abraham Wald e Jacob Wolfowitz. Il test di Wald-Wolfowitz è un [test statistico non parametrico](/nonparametric-statistics),. il che significa che i dati analizzati non devono soddisfare determinati presupposti o parametri. Il test di Wald-Wolfowitz può essere utilizzato per esaminare l' [ipotesi](/hypothesistesting) che le variabili nella stringa di dati siano reciprocamente indipendenti.

Alcuni statistici ritengono che un altro tipo di test di esecuzione, il test di Kolmogorov-Smirnov, sia uno strumento migliore del test di Wald-Wolfowitz per rilevare le differenze tra le distribuzioni. Il test di Kolmogorov-Smirnov è un tipo di test di [bontà di adattamento](/goodness-of-fit) che dimostra se i dati del campione sottoposti a test rappresentano [modelli di distribuzione normali](/normaldistribution) o se i dati sono in qualche modo distorti. Il test stabilisce la discrepanza tra i valori nei dati del campione e il modello di distribuzione normale.

## Vantaggi di un test di esecuzione

Il modello di test delle corse è importante per determinare se un risultato di una prova è veramente casuale, specialmente nei casi in cui i dati casuali rispetto a quelli sequenziali hanno implicazioni per le teorie e l'analisi successive. Un test di esecuzione può essere uno strumento prezioso per gli investitori che utilizzano [l'analisi tecnica](/technicalanalysis) per prendere decisioni di trading. Questi trader analizzano le tendenze statistiche, come il movimento e il [volume dei prezzi](/volume),. per individuare opportunità di trading potenzialmente redditizie. È importante che questi trader comprendano le variabili sottostanti che potrebbero influire sul movimento dei prezzi e un test di esecuzione può aiutare in questo.

Due potenti modi in cui i trader possono utilizzare un test di esecuzione includono:

1. Testare la casualità della distribuzione, prendendo i dati nell'ordine dato e contrassegnando con un più (+) i dati maggiori della [mediana](/median) e con un meno (-) i dati inferiori alla mediana (i numeri uguali alla mediana sono omesso.)

1. Verificare se una funzione si adatta bene a un set di dati, contrassegnando i dati che superano il valore della funzione con + e gli altri dati con −. Per questo uso, il test delle corse, che tiene conto della segnaletica ma non delle distanze, è complementare al [test del chi quadrato](/chi-square-statistic),. che tiene conto delle distanze ma non della segnaletica.

## Mette in risalto

- Un test di esecuzione è un'analisi statistica che aiuta a determinare la casualità dei dati rivelando eventuali variabili che potrebbero influenzare i modelli di dati.

- Un test di corsa, noto anche come test di corsa di Wald–Wolfowitz, è stato sviluppato dai matematici Abraham Wald e Jacob Wolfowitz.

- I trader tecnici possono utilizzare un test di esecuzione per analizzare le tendenze statistiche e aiutare a individuare opportunità di trading redditizie.

- Ad esempio, un investitore interessato ad analizzare il movimento dei prezzi di un determinato titolo potrebbe condurre un test di esecuzione per ottenere informazioni su una possibile azione futura del prezzo di quel titolo.
