# Linguaggio di markup ipertestuale – HTML
## Che cos'è l'HyperText Markup Language (HTML)?

HyperText Markup Language (HTML) è l'insieme di simboli o codici di markup inseriti in un file destinato alla visualizzazione su Internet. Il markup indica ai browser Web come visualizzare le parole e le immagini di una pagina Web.

Ogni codice di markup di ogni singolo pezzo (che rientrerebbe tra i caratteri "<" e ">") viene indicato come un elemento, sebbene molte persone lo facciano anche come un tag. Alcuni elementi sono disponibili in coppia che indicano quando deve iniziare un effetto di visualizzazione e quando deve finire.

## Spiegazione HTML

HyperText Markup Language è il linguaggio informatico che facilita la creazione di siti Web. Il linguaggio, che ha parole in codice e sintassi proprio come qualsiasi altro linguaggio, è relativamente facile da comprendere e, col passare del tempo, diventa sempre più potente in ciò che consente a qualcuno di creare. L'HTML continua ad evolversi per soddisfare le richieste ei requisiti di Internet sotto le spoglie del World Wide Web Consortium, l'organizzazione che progetta e mantiene il linguaggio; per esempio, con il passaggio al [Web 2.0](/web-20).

HyperText è il metodo con cui gli utenti Internet navigano nel Web. Facendo clic su un testo speciale chiamato collegamenti ipertestuali, gli utenti vengono portati a nuove pagine. L'uso di hyper significa che non è lineare, quindi gli utenti possono andare ovunque su Internet semplicemente facendo clic sui collegamenti disponibili. Il markup è ciò che i tag HTML fanno al testo al loro interno; lo contrassegnano come un tipo specifico di testo. Ad esempio, il testo di markup potrebbe essere in grassetto o in corsivo per attirare un'attenzione specifica su una parola o una frase.

## Nozioni di base sul linguaggio di markup dell'ipertesto

Al suo interno, l'HTML è una serie di codici brevi digitati in un file di testo. Questi sono i tag che alimentano le capacità di HTML. Il testo viene salvato come file HTML e visualizzato tramite un browser web. Il browser legge il file e traduce il testo in una forma visibile, come indicato dai codici che l'autore ha utilizzato per scrivere quello che diventa il rendering visibile. La scrittura di HTML richiede che i tag siano usati correttamente per creare la visione dell'autore.

I tag sono ciò che separa il testo normale dal codice HTML. I tag sono le parole tra quelle che sono note come parentesi angolari, che consentono a grafici, immagini e tabelle di apparire sulla pagina web. Tag diversi svolgono funzioni diverse. I tag più elementari applicano la formattazione al testo. Poiché le interfacce Web devono diventare più dinamiche, è possibile utilizzare Cascading Style Sheets (CSS) e applicazioni JavaScript. CSS rende le pagine web più accessibili e JavaScript aggiunge potenza all'HTML di base.

## HTML e XML

A differenza di HTML, Extensible Markup Language o XML, consente agli utenti di definire il proprio markup. Ad esempio, utilizzando XML, un utente potrebbe scegliere di denotare una nota a piè di pagina con il tag <footnote>, mentre un altro utente potrebbe optare per <fn>.

Utilizzando l'HTML, è possibile utilizzare un solo tag predeterminato per denotare un tipo specifico di informazioni. I documenti XML sono pensati per essere facili da leggere poiché contengono tag definiti dall'utente e poiché i documenti sono costituiti solo da markup e contenuto.

## Mette in risalto

- HyperText Markup Language (HTML) è il linguaggio di scripting di base utilizzato dai browser Web per eseguire il rendering di pagine sul World Wide Web.

- Le prime versioni di HTML erano statiche (Web 1.0), mentre le iterazioni più recenti offrono una grande flessibilità dinamica (Web 2.0, 3.0).

- Il markup è il testo che appare tra due parentesi appuntite (ad esempio, <footnote>), e il contenuto è tutto il resto.

- HyperText consente a un utente di fare clic su un collegamento ed essere reindirizzato a una nuova pagina referenziata da quel collegamento.
