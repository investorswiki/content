# Protocollo di trasferimento file (FTP)
## Che cos'è il protocollo FTP (File Transfer Protocol)?

Il termine FTP (File Transfer Protocol) si riferisce a un processo che prevede il trasferimento di file tra dispositivi su una rete. Il processo funziona quando una parte consente a un'altra di inviare o ricevere file su Internet. Originariamente utilizzato come mezzo per comunicare e scambiare informazioni tra due dispositivi fisici, ora è comunemente utilizzato per archiviare file nel [cloud](/cloud-computing),. che di solito è un luogo sicuro tenuto in remoto.

FTP può essere utilizzato da [un'azienda](/business) o da un individuo per trasferire file da un sistema informatico a un altro o da siti Web per caricare o scaricare file dai loro server.

## Come funziona il protocollo FTP (File Transfer Protocol).

Il protocollo di trasferimento file consente a individui e aziende di condividere file elettronici con altri senza dover essere nello stesso spazio. Questo può essere fatto utilizzando un client FTP o tramite il cloud. Indipendentemente dall'opzione, entrambe le parti richiedono una connessione Internet funzionante.

La maggior parte dei browser [Web](/web-20) viene fornita con client FTP che consentono agli utenti di trasferire file dal proprio computer a un server e viceversa. Alcuni utenti potrebbero voler utilizzare un client FTP di terze parti perché molti di loro offrono funzionalità extra. Esempi di client FTP che possono essere scaricati gratuitamente includono FileZilla Client, FTP Voyager, WinSCP, CoffeeCup Free FTP e Core FTP.

Molte persone hanno già utilizzato FTP senza nemmeno rendersene conto. Se hai mai scaricato un file da una pagina web, hai usato FTP. Il primo passo è accedere, che può avvenire automaticamente o inserendo manualmente un nome utente e una password. FTP richiederà anche l'accesso a un server FTP tramite un numero di porta specifico. Una volta che accedi al server FTP tramite il tuo client FTP, ora puoi trasferire i file. Non tutti i server FTP pubblici richiedono l'accesso perché alcuni server consentono di accedervi in modo anonimo.

Come notato sopra, FTP è stato originariamente sviluppato come un modo per inviare e ricevere file tra due computer fisici. Ma con i cambiamenti nella tecnologia, gli utenti possono eseguire trasferimenti di file attraverso il cloud. L'utilizzo del cloud consente di eseguire i trasferimenti in modo comodo e sicuro (che potrebbe proteggere individui e aziende dalle [violazioni dei dati](/data-breach) ) e con costi relativamente bassi.

> Il termine client FTP si riferisce al software che consente di trasferire file a un'altra parte.

>

## Considerazioni speciali

Il protocollo di trasferimento file è uno dei tanti protocolli diversi che determinano il comportamento dei computer e dei sistemi informatici su Internet. Altri protocolli simili includono:

- **Hypertext Transfer Protocol (HTTP):** Progettato per trasmettere dati sul web

- **Internet Message Access Protocol (IMAP):** Fornisce l'accesso alla bacheca o ai messaggi di posta elettronica da un servizio condiviso

- **Network Time Protocol (NTP):** sincronizza l'ora dei computer su una rete

FTP consente ai computer su Internet di trasferire file avanti e indietro. In quanto tale, è uno strumento essenziale per coloro che creano e mantengono siti Web oggi.

### Cosa cercare in un client FTP

I singoli client FTP forniscono diverse funzionalità che consentono agli utenti di modificare il modo in cui caricano e scaricano i file. Ad esempio, se utilizzi FileZilla, il programma ti consente di impostare limiti di [larghezza di banda per i file.](/bandwidth) Ciò ti consente di controllare le velocità di caricamento e download, il che può essere utile se gestisci più trasferimenti di file contemporaneamente.

Altre caratteristiche che potresti voler cercare in un client FTP includono l'autenticazione con chiave pubblica, la possibilità di impostare livelli di compressione dei file o strumenti che ti consentono di cercare un server utilizzando maschere di file.

## Esempio di protocollo FTP (File Transfer Protocol)

Il software FTP è relativamente semplice da configurare. FileZilla è un client FTP scaricabile gratuitamente. Digitare l'indirizzo del server a cui si desidera accedere, la porta e la password per accedere al server.

Una volta concesso l'accesso, i file dell'utente sul proprio sistema locale e sul server a cui si accede saranno visibili. L'utente può scaricare file dal server al sistema locale o caricare file dal sistema locale al server. Possono anche apportare modifiche ai file sul server, purché dispongano dell'autorizzazione adeguata per farlo.

## Mette in risalto

- FTP è uno strumento essenziale per chi costruisce e gestisce siti web.

- Il protocollo di trasferimento file è un modo per scaricare, caricare e trasferire file da una posizione all'altra su Internet e tra sistemi informatici.

- FTP consente il trasferimento di file avanti e indietro tra computer o tramite il cloud.

- Gli utenti richiedono una connessione a Internet per eseguire trasferimenti FTP.

- Molti client FTP possono essere scaricati gratuitamente, sebbene la maggior parte dei siti Web abbia già l'FTP integrato.
