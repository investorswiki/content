# ActiveX
## Che cos'è ActiveX?

ActiveX è un framework software di Microsoft (MSFT) che consente alle applicazioni di condividere funzionalità e dati tra loro tramite browser Web, indipendentemente dal linguaggio di programmazione in cui sono scritti. I componenti aggiuntivi ActiveX consentivano ai primi browser Web di incorporare file multimediali o fornire aggiornamenti software agli utenti.

Microsoft ha introdotto ActiveX nel 1996.﻿Molti controlli ActiveX vengono eseguiti solo su Windows e con prodotti Microsoft come Internet Explorer, Word ed Excel. JavaScript, un altro compilatore multilingua e altre piattaforme simili sono più ampiamente utilizzati di ActiveX.

## Capire ActiveX

I controlli ActiveX sono software precodificati simili ai plug-in del browser web. Ad esempio, una pagina Web che visualizza un file Flash potrebbe richiedere all'utente di scaricare un controllo Flash ActiveX in modo che il file possa essere riprodotto direttamente nel browser senza aprire una nuova applicazione. I controlli ActiveX estendono la funzionalità di un browser, consentendo al browser di eseguire attività che altrimenti non potrebbero essere eseguite in modo innato. È particolarmente utile per riprodurre video e altri contenuti multimediali, saltando il passaggio dell'apertura di un lettore multimediale separato.

ActiveX è ancora utilizzato in Internet Explorer 11, ma non è supportato dall'ultimo browser di Microsoft, Edge . Browser come Mozilla Firefox, Google Chrome, Safari di Apple e Opera utilizzano altri tipi di plug-in del browser, come JavaScript o linguaggi multipiattaforma simili.

ActiveX rimane utile per gli utenti Microsoft ed è incluso in Windows 10. Questo perché ActiveX consente ancora al software autonomo di ricevere aggiornamenti, interfacciarsi tra i programmi e migliorare le funzionalità. Ad esempio, ActiveX consente agli utenti di creare documenti più interattivi in Word o creare moduli compilabili in Excel.

## ActiveX e sicurezza informatica

Microsoft avverte che i controlli ActiveX a volte possono non funzionare correttamente o fornire agli utenti contenuti che non desiderano. I controlli ActiveX possono essere utilizzati anche per installare [spyware](/adware),. virus e malware o danneggiare i dati sul computer. Per questo motivo, è importante installare i controlli ActiveX solo da fonti affidabili .

In parte a causa dell'uso dannoso diffuso dei controlli ActiveX, molti browser disabilitano i controlli ActiveX per impostazione predefinita o non li supportano affatto. Ad esempio, gli utenti di Google Chrome devono abilitare ActiveX nelle impostazioni di sicurezza di Chrome o scaricare un'estensione di Chrome .

Anche Microsoft sembra allontanarsi da ActiveX. Edge, il suo nuovo browser che sta sostituendo Internet Explorer sui sistemi operativi Windows, non supporta ActiveX .

## Mette in risalto

- Microsoft avverte che i controlli ActiveX possono essere utilizzati per spiare il computer, danneggiare dati o installare software dannoso .

- ActiveX è una piattaforma per lo sviluppo di piccoli programmi o plug-in che offrono maggiori funzionalità ai browser Web e ai prodotti Microsoft.

- Molti browser non supportano ActiveX per impostazione predefinita. Edge, il nuovo browser di Microsoft, non offre supporto per i controlli ActiveX .
