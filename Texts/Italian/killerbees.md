# Api assassine
## Cosa sono le api assassine?

Le api assassine sono società o individui, come banchieri d'investimento, contabili, avvocati e specialisti fiscali, che aiutano [le aziende target a](/targetfirm) evitare di essere rilevate da un corteggiatore indesiderato. Il loro compito è ideare e attuare strategie di difesa [anti-acquisizione](/anti-takeovermeasure),. che generalmente consistono nel rendere l'obiettivo meno attraente o più difficile o costoso da acquisire.

## Capire le api assassine

Quando una società ne punta un'altra per l'acquisizione, di solito si avvicina prima al suo [consiglio di amministrazione](/boardofdirectors). Se respinto, l'acquirente potrebbe quindi tornare con un'offerta migliore, allontanarsi o cercare di aggirare la gestione avviando un'offerta [pubblica di acquisto](/tenderoffer) direttamente agli azionisti.

Se le anticipazioni di acquisizione diventano ostili o [ostili](/hostiletakeover),. le api assassine possono essere portate a bordo. Il loro compito è trovare modi fattibili per mettere la vita a disagio per il potenziale acquirente, in modo simile a come il loro omonimo punge le sue vittime quando viene provocato finché non si tira indietro e se ne va.

Le api assassine sono salite alla ribalta durante la mania di acquisizione ostile degli anni '80. All'epoca, una categoria di investitori con tasche profonde, noti come [predoni](/raider),. iniziò ad acquistare società [sottovalutate](/undervalued) e poi a smembrarle in modo controverso per ottenere un rapido profitto. Corporate America non era abituata a questo tipo di comportamento e ha chiesto l'aiuto di specialisti per difendersi da questi attacchi.

Le api assassine presenterebbero una serie di opzioni al consiglio di amministrazione del bersaglio in base alle sue circostanze individuali e alle caratteristiche dell'azienda che cerca di acquistarlo. Per sventare un tentativo di acquisizione ostile, generalmente mirano a rendere la preda troppo costosa da [acquisire](/acquisition) o così poco attraente da far perdere interesse al predatore.

## Metodi delle api assassine

Dopo gli anni '80, furono escogitate misure difensive note come [repellenti per squali](/sharkrepellent) per scoraggiare tentativi di acquisizione ostili. Le strategie popolari utilizzate dalle api assassine includono:

- [Pill-In Poison Pill](/flip-in-poison-pill) : agli azionisti esistenti viene concesso il diritto di acquistare azioni aggiuntive con uno sconto, [diluendo così](/dilution) l'interesse di proprietà della parte ostile e rendendo più difficile e costoso il suo controllo.

- [White Knight](/whiteknight) : una compagnia amichevole interviene per acquistare il bersaglio sul punto di essere preso in consegna.

- **Pac-Man:** Prende il nome dal classico gioco arcade mangia-o-essere-mangiato, la società target ribalta la situazione sull'acquirente facendo un'offerta pubblica di acquisto per esso.

- [Trappola per aragoste](/lobstertrap) : viene approvata una disposizione che vieta a qualsiasi azionista con una quota di proprietà superiore al 10 per cento di convertire [titoli](/convertible-security) [convertibili](/convertible-security) in azioni con diritto di voto, impedendo così ai grandi azionisti di ottenere voti sufficienti per costringere il consiglio di amministrazione ad accettare la fusione.

- [Poison Put](/poison-put) : viene emessa un'obbligazione che gli investitori possono rimborsare integralmente prima della data di [scadenza](/maturitydate).

le controversie, come gli [accordi di sospensione](/standstill_agreement),. potrebbero essere utilizzate per ritardare qualsiasi acquisizione.

## Critiche alle api assassine

Molte delle strategie di difesa anti-acquisizione utilizzate dalle api assassine non si adattano bene agli azionisti. Rendere l'obiettivo meno attraente o più costoso da acquistare generalmente ha l'abitudine di erodere [il valore per gli azionisti](/shareholder-value) e potenzialmente paralizzare l'azienda per gli anni a venire.

La natura drastica di alcune di queste misure e la frequente impossibilità di votare da parte degli azionisti ordinari ha portato a metterne in discussione la legittimità. Non tutti gli offerenti ostili hanno in programma di fare soldi velocemente e rovinare le aziende e, in alcuni casi, essere rilevati da uno di loro potrebbe essere più vantaggioso dal punto di vista finanziario per gli investitori esistenti.

> ### Importante

> Le strategie impiegate dalle api assassine sono spesso controverse e non sempre favorevoli agli azionisti, spingendo i tribunali a intervenire occasionalmente.

>

## Limitazioni delle api assassine

Nel corso degli anni, queste osservazioni hanno portato i tribunali a bloccare occasionalmente le imprese dall'adozione di misure anti-acquisizione, se ritenute irragionevoli. La prospettiva di un intervento da parte delle potenze superiori significa inevitabilmente che ora è molto più difficile per le api assassine adempiere ai loro mandati.

## Mette in risalto

- Le api assassine sono salite alla ribalta durante gli anni '80, quando le aziende americane sono state attaccate da investitori opportunisti noti come predoni.

- Il loro compito è ideare strategie di difesa anti-acquisizione che rendano l'obiettivo più difficile o costoso da acquisire.

- Le api assassine sono aziende o individui che aiutano le aziende target a evitare di essere rilevate.

- Le strategie impiegate dalle api assassine sono spesso controverse, spesso messe in discussione dagli azionisti e rischiano di essere ribaltate dai tribunali.
