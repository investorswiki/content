# BEP-2
BEP-2 è uno standard tecnico per l'emissione e l'implementazione di token sulla Binance Chain. Questo standard definisce un insieme di regole che i token dovrebbero seguire per funzionare nell'ecosistema Binance Chain. Uno standard di token comparabile per l'emissione di token sulla blockchain di Ethereum è [ERC-20](/erc-20).

BEP-2 non è un software, ma piuttosto un insieme di specifiche tecniche. Queste regole assicurano che i token nell'ecosistema Binance Chain possano interagire senza problemi tra loro. I token BEP-2 possono rappresentare un'ampia gamma di risorse digitali e beni negoziabili. Lo standard garantisce che tutti possano integrarsi senza problemi tra loro e con l'ecosistema circostante.

Lo standard BEP-2 definisce anche alcune delle caratteristiche di base e più essenziali dei token emessi su Binance Chain. Sebbene lo standard definisca le regole, token diversi avranno comunque implementazioni diverse per i casi d'uso specifici per cui sono stati creati. Tuttavia, la creazione di uno standard come questo riduce significativamente lo sforzo richiesto dagli sviluppatori per emettere i token.

L'asset nativo di Binance Chain è BNB, che viene utilizzato per pagare le commissioni (simile al gas). Ciò significa che quando desideri trasferire i tuoi token BEP-2, dovrai anche avere una piccola quantità di BNB nel tuo portafoglio.

I token BEP-2 possono essere scambiati su Binance DEX, uno [scambio decentralizzato](/decentralized-exchange) che funziona su Binance Chain.

Puoi archiviare i token BEP-2 in molti portafogli diversi. Trust Wallet è un'opzione eccellente se desideri utilizzare un portafoglio mobile, ma anche molti altri portafogli software supportano lo standard. Lo standard BEP-2 è supportato anche da portafogli hardware, come Ledger Nano X, Ledger Nano S, Trezor Model T o Coolwallet S.

Puoi sfogliare tutti i token che sono stati emessi su Binance Chain nell'esploratore di blocchi Binance Chain.

Puoi emettere token BEP-2 utilizzando questo portale. Si consiglia vivamente di leggere prima le specifiche tecniche più dettagliate su Binance Chain GitHub. Se vuoi giocare con le idee sui token BEP-2, puoi farlo sul testnet di Binance Chain.

Troverai guide tecniche dettagliate sull'emissione di token e molti argomenti correlati nella Binance Chain Documentation.
