# Web 2.0
## Hva er Web 2.0?

Web 2.0 beskriver den nåværende tilstanden til internett, som har mer brukergenerert innhold og brukervennlighet for sluttbrukere sammenlignet med sin tidligere inkarnasjon, Web 1.0. Generelt refererer Web 2.0 til internettapplikasjonene fra det 21. århundre som har forvandlet den digitale æraen i kjølvannet av [dotcom-boblen](/dotcom-bubble).

## Forstå Web 2.0

Web 2.0 refererer ikke til noen spesifikke tekniske oppgraderinger av internett. Det refererer ganske enkelt til et skifte i hvordan internett brukes i det 21. århundre. I den nye tiden er det et høyere nivå av informasjonsdeling og sammenkobling mellom deltakerne. Denne nye versjonen lar brukere delta aktivt i opplevelsen i stedet for bare å opptre som passive seere som tar inn informasjon.

## Web 1.0 vs. Web 2.0

Web 1.0 brukes til å beskrive den første fasen av Internett. På dette tidspunktet var det få innholdsskapere; de fleste av de som brukte Internett var forbrukere. Statiske sider var mer vanlig enn dynamisk [HTML](/html),. som inneholder interaktive og animerte nettsteder med spesifikk koding eller språk. Innhold i dette stadiet kom fra en servers filsystem i stedet for et databasebehandlingssystem. Brukere kunne signere gjestebøker på nett, og HTML-skjemaer ble sendt via e-post.

Eksempler på nettsteder som er klassifisert som Web 1.0 er Britannica Online, personlige nettsteder og mp3.com. Generelt er disse nettstedene statiske og har begrenset funksjonalitet og fleksibilitet.

Begrepet Web 2.0 kom først i bruk i 1999 da Internett dreide seg mot et system som aktivt engasjerte brukeren. Brukere ble oppfordret til å gi innhold, i stedet for bare å se det. Folk kunne nå publisere artikler og kommentarer, og det ble mulig å opprette brukerkontoer på forskjellige nettsteder, noe som økte deltakelsen. Web 2.0 ga også opphav til nettapper, selvpubliseringsplattformer som [WordPress](/wordpress-cms),. samt [sosiale medier](/social-media).

Eksempler på Web 2.0-nettsteder inkluderer Wikipedia, Facebook, Twitter og forskjellige blogger, som alle har endret måten den samme informasjonen deles og leveres på.

Det sosiale aspektet ved Internett har blitt spesielt transformert; generelt lar sosiale medier brukere engasjere seg og samhandle med hverandre ved å dele tanker, perspektiver og meninger. Brukere kan merke, dele, tweete og like.

## Fordeler og ulemper med Web 2.0

Utviklingen av teknologi har gjort det mulig for brukere å dele sine tanker og meninger med andre, noe som har skapt nye måter å organisere og knytte seg til andre mennesker på og fremmet en større grad av samarbeid.

Men det er mange ulemper ved at Internett fungerer mer som et åpent forum. Gjennom utvidelsen av sosiale medier har vi sett en økning i nettforfølgelse, nettmobbing, [doxing](/personally-identifiable-information-pii),. [identitetstyveri](/identitytheft) og andre nettforbrytelser. Det er også trusselen om at feilinformasjon spres blant brukere, enten det er gjennom åpen kildekode for informasjonsdeling eller på sosiale medier.

I likhet med Web 1.0 er Web 2.0 en annen overgangsfase i utviklingen av Internett. Web 3.0 er spådd å bli kalt Semantic Web fordi det vil bli skreddersydd for å bli mer intuitivt for hver brukers behov.

## Høydepunkter

– I internetts nye tidsalder er det et høyere nivå av informasjonsdeling og sammenkobling blant deltakerne.

- Web 2.0 refererer ikke til noen spesifikke tekniske oppgraderinger av internett; det refererer til et skifte i hvordan Internett brukes.

- Web 2.0 beskriver den nåværende tilstanden til internett, som har mer brukergenerert innhold og brukervennlighet for sluttbrukere sammenlignet med sin tidligere inkarnasjon, Web 1.0.
