# stor Data
## Hva er Big Data?

Big data refererer til store, mangfoldige sett med informasjon som vokser i stadig økende hastighet. Det omfatter volumet av informasjon, hastigheten eller hastigheten som den lages og samles inn med, og variasjonen eller omfanget av datapunktene som dekkes (kjent som de "tre v-ene" av big data). Big data kommer ofte fra [data mining](/datamining) og kommer i flere formater.

## Hvordan Big Data fungerer

Big data kan kategoriseres som ustrukturert eller strukturert. strukturerte data består av informasjon som allerede er administrert av organisasjonen i databaser og regneark; det er ofte numerisk i naturen. Ustrukturerte data er informasjon som er uorganisert og ikke faller inn i en forhåndsbestemt modell eller format. Den inkluderer data samlet inn fra [sosiale medier](/social-media),. som hjelper institusjoner med å samle informasjon om kundenes behov.

Big data kan samles inn fra offentlig delte kommentarer på sosiale nettverk og nettsteder, frivillig samlet inn fra personlig elektronikk og apper, gjennom spørreskjemaer, produktkjøp og elektroniske innsjekkinger. Tilstedeværelsen av sensorer og andre innganger i smarte enheter gjør at data kan samles over et bredt spekter av situasjoner og omstendigheter.

Big data lagres oftest i datadatabaser og analyseres ved hjelp av programvare spesielt utviklet for å håndtere store, komplekse datasett. Mange [programvare-som-en-tjeneste-](/software-as-a-service-saas) selskaper (SaaS) spesialiserer seg på å administrere denne typen komplekse data.

## Bruken av Big Data

Dataanalytikere ser på forholdet mellom ulike typer data, for eksempel [demografiske](/demographics) data og kjøpshistorikk, for å finne ut om en korrelasjon eksisterer. Slike vurderinger kan gjøres internt eller eksternt av en tredjepart som fokuserer på å behandle big data til fordøyelige formater. Bedrifter bruker ofte vurderingen av store data fra slike eksperter for å gjøre dem om til brukbar informasjon.

> Mange selskaper, som Alphabet og Meta (tidligere Facebook), bruker big data for å generere annonseinntekter ved å plassere målrettede annonser til brukere på sosiale medier og de som surfer på nettet.

>

Nesten alle avdelinger i en bedrift kan bruke funn fra dataanalyse, fra [menneskelige ressurser](/humanresources) og teknologi til markedsføring og salg. Målet med big data er å øke hastigheten som produktene kommer ut på markedet med, redusere mengden tid og ressurser som kreves for å få markedsadopsjon, målgrupper og sikre at kundene forblir fornøyde.

## Fordeler og ulemper med Big Data

Økningen i mengden tilgjengelig data byr på både muligheter og problemer. Generelt sett bør det å ha mer data om kunder (og potensielle kunder) gjøre det mulig for bedrifter å bedre skreddersy produkter og markedsføringstiltak for å skape det høyeste nivået av tilfredshet og gjenta forretninger. Bedrifter som samler inn store mengder data får muligheten til å utføre dypere og rikere analyser til fordel for alle [interessenter](/stakeholder).

> Med mengden personopplysninger tilgjengelig om enkeltpersoner i dag, er det avgjørende at selskaper tar skritt for å beskytte disse dataene; et tema som har blitt en het debatt i dagens nettverden, spesielt med de mange datainnbruddene selskaper har opplevd de siste årene.

>

bedre Selv om analyse er positivt, kan store data også skape overbelastning og støy, noe som reduserer nytten. Bedrifter må håndtere større datamengder og bestemme hvilke data som representerer signaler sammenlignet med støy. Å bestemme hva som gjør dataene relevante blir en nøkkelfaktor.

Videre kan arten og formatet til dataene kreve spesiell håndtering før det iverksettes. Strukturerte data, bestående av numeriske verdier, kan enkelt lagres og sorteres. Ustrukturerte data, som e-poster, videoer og tekstdokumenter, kan kreve at mer sofistikerte teknikker brukes før de blir nyttige.

##Høydepunkter

– Big data er en stor mengde mangfoldig informasjon som kommer i økende volum og med stadig høyere hastighet.

– Big data lagres oftest i datadatabaser og analyseres ved hjelp av programvare spesielt utviklet for å håndtere store, komplekse datasett.

- Big data kan være strukturert (ofte numerisk, enkelt formatert og lagret) eller ustrukturert (mer fri form, mindre kvantifiserbar).

– Nesten alle avdelinger i en bedrift kan bruke funn fra big data-analyse, men håndtering av rot og støy kan by på problemer.

– Big data kan samles inn fra offentlig delte kommentarer på sosiale nettverk og nettsteder, frivillig samlet inn fra personlig elektronikk og apper, gjennom spørreskjemaer, produktkjøp og elektroniske innsjekkinger.
