# Associate in Personal Insurance (API)
## Hva er Associate in Personal Insurance (API)-betegnelse?

Associate in Personal Insurance (API) er en profesjonell sertifisering i [forsikringsbransjen](/insurance). Som navnet antyder, er API-en fokusert på [personlig forsikring](/personal-insurance-lines),. som er den typen forsikring som beskytter mot personskade, død eller tap av eiendom. Det er en av mange betegnelser gitt av The Institutes, en organisasjon dedikert til å gi akkreditering og videreutdanning til forsikringsfagfolk.

## Hvordan API-designet fungerer

API-betegnelsen er populær blant fagfolk som ønsker å begynne eller avansere karrieren innen personforsikring. Ved å skaffe APIen kan fagfolk demonstrere at de har kunnskap om ikke-kommersielle forsikringsprodukter, inkludert kritiske ferdigheter som [forsikringsgaranti](/underwriting),. [premieinnstilling](/insurance-premium),. markedsføring og porteføljestyring.

For fagfolk som ønsker å oppnå mer avanserte sertifiseringer, som for eksempel [Chartered Property Casualty Underwriter (CPCU)](/chartered-property-casualty-underwriter-cpcu),. kan API være et populært alternativ. Dette er fordi mange av betingelsene som kreves for å få API-en, også teller mot kravene til CPCU. Spesielt kan to av de tre eksamenene knyttet til API-en gjenbrukes til CPCU-akkreditering, som er en av de mest prestisjefylte betegnelsene i skadeindustrien.

For å få API-en må kandidatene utvikle omfattende kunnskap om hvilke typer forsikringer som selges i personforsikringsmarkedet. Dette inkluderer en bred utdanning innen emner som kontraktspåtegninger, betingelser og garantikriterier og bransjeforskrifter. API er ofte anbefalt for forsikringsmeglere, agenter, skadejusteringsorganer, forsikringsgivere og kundeservicepersonell som betjener personlige forsikringskunder.

## Real World Eksempel på API-betegnelsen

Det er to veier som kandidater kan ta når de fullfører API. Den første av disse inkluderer kurs om [underwriting](/underwriting) og markedsføringspraksis i personforsikringsbransjen, porteføljestyring, eiendoms- og [ansvarsforsikring](/liability_insurance) og personforsikring. Den andre banen består av tre kurs, som dekker underwriting og markedsføring, [risikostyring](/riskmanagement) og eiendomsansvar og personlig økonomisk planlegging.

Vanligvis tar kandidater mellom 12 og 18 måneder på å fullføre dette kursmateriellet, som er tilgjengelig på selvbetjeningsbasis. For de som foretrekker personlig instruksjon, kan undervisning også være tilgjengelig offline i utvalgte regioner. Når de har bestått eksamenene, er ikke innehavere av API pålagt å oppfylle løpende utdanningskrav for å opprettholde betegnelsen.

##Høydepunkter

– API er en profesjonsbetegnelse som er populær i personforsikringsbransjen.

– Det administreres av The Institutes, og tar vanligvis mellom 12 og 18 måneder å fullføre.

– Mange kandidater velger å legge til API-betegnelsen ved å søke etter en mer avansert legitimasjon, som CPCU.
