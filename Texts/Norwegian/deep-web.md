# Deep Web
Det dype nettet er den delen av World Wide Web (WWW) som er noe skjult. Den inneholder sider som ikke er indeksert av tradisjonelle nettsøkemotorer - som Google, Bing og DuckDuckGo.

Det dype nettet er det motsatte uttrykket av "overflatenettet." Som sådan kan det også bli referert til som "skjult web" eller "usynlig web."

## Utover overflaten

Deep web står for over 99 % av innholdet på nettet. Det er en del av WWW som er forskjellig fra den såkalte "surface web", som er den synlige delen av nettet. Overflatebanen er det lettest tilgjengelige laget. Den inkluderer alle indekser (eller søkbare) sider.

Selv om mengden innhold som kan oppdages gjennom søkemotorer er enorm, inneholder overflatenettet mye mindre informasjon enn dypnettet. En studie publisert i 2001 indikerte at mindre enn 0,05 % av alt innholdet er tilgjengelig i overflatenettet, mens de andre ~99,95 % finnes i dypnettet.

## Hva er Deep Web?

Deep web inkluderer per definisjon begge sider som er obskure eller som bare kan nås gjennom en spesifikk autentiseringsmetode. Å være obskur betyr at de krever en direkte URL for å bli funnet (søking på Google fungerer ikke). Likevel faller størstedelen av dypnettets innhold inn under sistnevnte kategori.

På denne måten kan vi vurdere som en del av dypnettet ethvert nettsted som ber om pålogging og passord. Med andre ord, mange av sidene folk går inn på hver dag er faktisk en del av det dype nettet. Dette inkluderer profiler på sosiale medier, e-postinnbokser og bankkontoer.

Dette kan høres forvirrende ut til å begynne med fordi den populære terminologien sier at det dype nettet er det obskure nettet. Men siden våre personlige kontoer ikke er indeksert på søkemotorsider, anses de også som en del av dypnettet.

Likevel består den største delen av deep web av statistikk og private databaser. For eksempel står amerikanske offentlige etater som [Commodity Futures Trading Com](/cftc) [mission (CFTC)](/cftc),. National Oceanic and Atmospheric Administration (NOAA) og [Securities and Exchange Commission (SEC)](/sec) for en betydelig mengde av disse digitale dataene.

## Deep web vs. dark web

Det er også vanlig å se forvirring angående forskjellene mellom det dype nettet og det mørke nettet. Kort sagt er det mørke nettet en del av det dype nettet som er anonymt.

Dermed forbindes det ofte med nettsider som legger til rette for ulovlige aktiviteter eller som gir noen form for personvernfordeler for aktivister og varslere (inkludert black hat [hackere](/hacker) og hacktivister). Det mørke nettet anslås å utgjøre mindre enn 0,01 % av det dype nettet og er kun tilgjengelig via spesialiserte plattformer, som Tor.

## Høydepunkter

– Den kanskje mest alvorlige kritikken mot deep web er at det undergraver åpenheten og likestillingen på internett.

- Det dype nettet refererer til deler av internett som ikke er fullt tilgjengelig gjennom standard søkemotorer som Google, Yahoo og Bing.

– Det dype nettet gir brukerne tilgang til langt mer informasjon enn det ellers ville vært tilgjengelig på internett, og det øker også personvernet.

– Den dype nettet inkluderer sider som ikke ble indeksert, gebyr-for-tjeneste-nettsteder, private databaser og det mørke nettet.

## FAQ

### Er Deep Web og Dark Web det samme?

Det mørke nettet er en liten komponent av det dype nettet. Det dype nettet refererer til alt av internett som ikke er lett tilgjengelig eller indeksert av søkemotorer (som kalles "overflatenettet"). Det mørke nettet består av dype nettsteder som med hensikt er skjult, ofte for å utføre ulovlige eller ulovlige transaksjoner på nettet.

### Er det ulovlig å få tilgang til Deep Web?

Nei. Å bare gå inn på nettsteder som ikke er indeksert eller på annen måte offentlig tilgjengelig er ikke ulovlig. Det kan imidlertid være ulovlig å misbruke eller stjele informasjon funnet på dype nettsteder.

### Hvordan kan jeg finne nettsteder på Deep Web?

Fordi dypnettet ikke er fullstendig indeksert med vanlige søkemotorer, er ofte den eneste måten å finne slike nettsteder på å vite den nøyaktige internettadressen som skal brukes. Det er noen spesialiserte nettsteder eller søkemotorer som katalogiserer noen dype nettsteder. For eksempel kan akademiske ressurser på dypnettet bli funnet ved å bruke plattformer som PubMed, LexisNexis, Web of Science eller Project MUSE.
