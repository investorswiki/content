# EIA Petroleum Status Report
## Hva er EIA Petroleum Status Report?

EIA Petroleum Status Report publiseres hver onsdag av US [Energy Information Administration](/energy-information-admin) (EIA), et byrå innen det amerikanske energidepartementet. Den beskriver nivået på [råoljereservene](/crude-oil) som USA har, så vel som mengden råolje og relaterte produkter de produserer, både innenlands og utenlands.

## Forstå EIA Petroleum Status Report

EIA Petroleum Status Report beskriver de eksisterende amerikanske lagrene av råolje, samt lagernivåene for raffinerte petroleumsprodukter som bensin, fyringsolje og diesel. Hovedhøydepunkter-delen av rapporten bryter ned råoljeinnsatsene til raffineriene, [kapasitetsutnyttelsen](/capacityutilizationrate) til disse raffineriene, og mengden av ny bensin- og destillatproduksjon.

Vanligvis inkluderer rapporten også en tabell med nettoimport, eller total amerikansk import minus eksport, av både råolje og petroleumsprodukter. Rapporten beskriver også prisene per gallon på [West Texas](/wti) [Intermediate](/wti) (WIT) råolje, vanlig blyfri bensin, nr. 2 fyringsolje, og propan.

Prisen på råolje bestemmes av [tilbud og etterspørsel](/law-of-supply-demand) [teknisk,](/technicalanalysis) slik tilfellet er med andre varer. Når reservene faller, vil prisene stige og omvendt. Derfor er EIA Petroleum Status Report en refleksjon av gjeldende råoljepristrender. Forfatterne av rapporten prøver ikke å tolke de grunnleggende driverne for disse trendene.

Investeringsanalytikere og futureshandlere vil studere rapporten og bruke dataene til å lage anslag angående energipriser og produksjonsnivåer. På samme måte vil noen analytikere som jobber for energiselskaper bruke nyhetene til å hente data fra rapporten for å hjelpe med å informere deres langsiktige strategi og forretningsbeslutninger. Media bruker også hyppig EIA-rapporten i rapportering om energibransjen og priser, som direkte påvirker forbrukerne, som fyringsolje og bensin.

## Detaljer om EIA Petroleum Status Report

US Energy Information Administration gjør statusrapporten tilgjengelig fra sin nettside. Brukere kan se gjennom rapporten ved hjelp av nettbaserte verktøy, inkludert analyser og prognoser etter emner og data sortert etter kategori. Hele rapporten er tilgjengelig i PDF-format og som et eksporterbart Microsoft Excel- og kommadelte-verdi-regneark (CSV) for videre analyse.

- EIA Petroleum Status Report highlights-seksjonen gir produksjonsdata for de fleste artikler på en etterfølgende fireukers basis. På samme måte bruker sammenligningen for år siden og forrige uke også etterfølgende fireukerstall.

- Grunnlaget for produksjonsdata kommer fra [sluttkursene](/closingprice) for den angitte dagen.

- Tabellene i rapporten viser ukentlige lagerbeholdninger eller forsyningsestimater for råolje, motorbensin, drivstoffetanol og petroleumsprodukter. Dataene kan brytes ytterligere ned etter geografisk region, da prisene i New England kan avvike meningsfullt fra de i Alaska eller de nedre atlantiske statene.

– Lenger ned i rapporten er en tabell med varelager og priser for oljebaserte produkter, som handles sjeldnere, inkludert jetdrivstoff, parafin og ultralavsvoveldiesel.

- Prisseksjonen for råolje, bensin og fyringsolje inneholder spot- (eller kortsiktig) og futures-priser sammen med priser for flydrivstoff, propan og diesel.

- Til slutt gir rapporten månedlige priser for **detaljhandel** bensin og diesel for motorveier for USA, så vel som etter geografisk region.

## EIA-rapport vs. API-rapport

[American Petroleum Institute (API)](/american-petroleum-institute) er en organisasjon grunnlagt i 1919 med mål om å støtte en sterk amerikansk olje- og gassindustri og samtidig fremme sikkerhet på tvers av industrien. API tjener amerikanske selskaper som er involvert i produksjon, prosessering, raffinering og distribusjon av energiprodukter.

Fra sine nesten 600 medlemmer driver API-en forskning, samler inn data fra medlemmer og ikke-medlemmer, og publiserer deres statistikk og bransjeindikatorer i Weekly Statistical Bulletin. Rapporten inneholder data om tilbud og etterspørsel etter energiprodukter, import, eksport, kostnader, boreaktiviteter og brønnfullføringer.

API hjelper også med å utvikle og vedlikeholde over 700 standarder og praksis, hvorav mange er innlemmet i lokale og føderale forskrifter. API fungerer som en talsmann på vegne av olje- og naturgassindustrien overfor den føderale regjeringen, inkludert kongressen, den utøvende grenen og statlige myndigheter.

API hjelper også med å forhandle med regulatoriske byråer, inkludert å representere industrien i rettslige prosesser. APIs Weekly Statistical Bulletin utgis vanligvis på tirsdager før den ukentlige EIA-rapporten.

EIA er veldig lik API ved at det er en uavhengig organisasjon som samler inn, analyserer og publiserer informasjon om den amerikanske olje- og gassindustrien. EIA publiserer sin Weekly Petroleum Status Report på onsdager. Rapporten inneholder nivået på lagrene av råolje og raffinerte produkter samt tilbud og etterspørseldata. EIA lobbyer imidlertid ikke aktivt for endringer i politikk og reguleringer, i motsetning til API.

## Vanlige spørsmål om EIA-rapport

##Høydepunkter

– EIA Petroleum Status Report publiseres hver onsdag av US Energy Information Administration (EIA).

- EIA Petroleum Status Report beskriver de eksisterende amerikanske råoljelagrene og lagernivåene for raffinerte petroleumsprodukter som bensin, fyringsolje og diesel.

##FAQ

### Når er EIA-rapporten utgitt?

EIA-rapporten utgis etter kl. 10.30 på onsdager hver uke.

### Er EIA-rapporten pålitelig?

EIA-rapporten er allment ansett som en pålitelig indikator på tilbud og etterspørsel for olje- og gassindustrien. Rapporten garanterer eller forutsier imidlertid ikke at tilbud og etterspørsel eller pristrender vil bevege seg på en eller annen måte.

### Hvorfor er EIA-rapporten viktig?

EIA Petroleum Status Report er viktig fordi den viser pris og lagerdata for energiprodukter som råolje. EIA-rapporten er en viktig indikator på tilbud og etterspørsel i olje- og gassindustrien. For eksempel, hvis råoljelagrene stiger, kan det signalisere at oljeetterspørselen øker, noe som fører til høyere priser.
